## PictureFrame::get_ShapeType() method




```cpp
Aspose::Slides::ShapeType Aspose::Slides::PictureFrame::get_ShapeType() override
```

## Remarks


Returns the [AutoShape](../../autoshape/) type for a [PictureFrame](../). There are allowable all items of the set [Slides::ShapeType](../../shapetype/), except all sorts of lines:

[ShapeType::Line](../../shapetype/),

[ShapeType::StraightConnector1](../../shapetype/),

[ShapeType::BentConnector2](../../shapetype/),

[ShapeType::BentConnector3](../../shapetype/),

[ShapeType::BentConnector4](../../shapetype/),

[ShapeType::BentConnector5](../../shapetype/),

[ShapeType::CurvedConnector2](../../shapetype/),

[ShapeType::CurvedConnector3](../../shapetype/),

[ShapeType::CurvedConnector4](../../shapetype/),

[ShapeType::CurvedConnector5](../../shapetype/).

Read [Slides::ShapeType](../../shapetype/).

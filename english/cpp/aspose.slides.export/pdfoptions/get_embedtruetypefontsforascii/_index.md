## PdfOptions::get_EmbedTrueTypeFontsForASCII() method


Determines if [Aspose.Slides](../../../aspose.slides/) will embed common fonts for ASCII (33..127 code range) text. [Fonts](../../../aspose.slides/fonts/) for character codes greater than 127 are always embedded. Common fonts list includes PDF's base 14 fonts and additional user specified fonts. Read **bool**.

```cpp
bool Aspose::Slides::Export::PdfOptions::get_EmbedTrueTypeFontsForASCII() override
```

## Remarks


Default is **true**.

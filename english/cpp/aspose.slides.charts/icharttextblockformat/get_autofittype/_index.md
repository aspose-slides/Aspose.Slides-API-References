## IChartTextBlockFormat::get_AutofitType() method


Returns text's autofit mode. Changing of this property can produce a certain influence only for these chart parts: [DataLabel](../../datalabel/) and [DataLabelFormat](../../datalabelformat/) (full suport in PowerPoint 2013; in PowerPoint 2007 there is no effect for rendering).

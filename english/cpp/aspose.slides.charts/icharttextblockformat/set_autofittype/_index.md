## IChartTextBlockFormat::set_AutofitType(TextAutofitType) method


Sets text's autofit mode. Changing of this property can produce a certain influence only for these chart parts: [DataLabel](../../datalabel/) and [DataLabelFormat](../../datalabelformat/) (full suport in PowerPoint 2013; in PowerPoint 2007 there is no effect for rendering). Write [TextAutofitType](../../../aspose.slides/textautofittype/).

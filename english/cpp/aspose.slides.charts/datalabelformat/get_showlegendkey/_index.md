## DataLabelFormat::get_ShowLegendKey() method


Represents a specified chart's data label legend key display behavior. True if the data label legend key is visible. Read **bool**.

```cpp
bool Aspose::Slides::Charts::DataLabelFormat::get_ShowLegendKey() override
```

## Remarks


If parent of this [DataLabelFormat](../) object is a [DataLabelCollection](../../datalabelcollection/) collection of data labels then this property gets or sets the default value of the ShowLegendKey property for the new data labels in the [DataLabelCollection](../../datalabelcollection/) collection. Set this property with value also sets this value to the ShowLegendKey property for all data labels in the [DataLabelCollection](../../datalabelcollection/) collection (i.e. \"DataLabels.DefaultDataLabelFormat.ShowLegendKey = val;\" cause to all DataLabels[i].ShowLegendKey is equal to val).

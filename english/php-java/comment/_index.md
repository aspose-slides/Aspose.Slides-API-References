## Comment class

 Represents a comment on a slide.
 

## Methods

| Name | Description |
| --- | --- |
| [getAuthor](getauthor)() | Returns the author of a comment. Read-only ICommentAuthor. |
| [getCreatedTime](getcreatedtime)() | Returns or sets the time of a comment creation. Setting this property to java.util.Date(Long.MIN_VALUE) means no comment time is set. Read/write java.util.Date. Comment time is an optional parameter. |
| [getParentComment](getparentcomment)() | Gets or sets parent comment. Read/write IComment. |
| [getPosition](getposition)() | Returns or sets the position of a comment on a slide. Read/write java.awt.geom.Point2D.Float. |
| [getSlide](getslide)() | Returns or sets the parent slide of a comment. Read-only ISlide. |
| [getText](gettext)() | Returns or sets the plain text of a slide comment. Read/write String. |
| [remove](remove)() | Removes comment and all its replies from the parent collection. |
| [setCreatedTime](setcreatedtime)(Date) | Returns or sets the time of a comment creation. Setting this property to java.util.Date(Long.MIN_VALUE) means no comment time is set. Read/write java.util.Date. Comment time is an optional parameter. |
| [setParentComment](setparentcomment)(IComment) | Gets or sets parent comment. Read/write IComment. |
| [setPosition](setposition)(Point2D.Float) | Returns or sets the position of a comment on a slide. Read/write java.awt.geom.Point2D.Float. |
| [setText](settext)(String) | Returns or sets the plain text of a slide comment. Read/write String. |

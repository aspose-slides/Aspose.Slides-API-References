 Adds a copy of a specified shape to the end of the collection.
 

### Parameters

| Name | Description |
| --- | --- |
| sourceShape | Shape to clone. |
| x | X coordinate of a new shape. |
| y | Y coordinate of a new shape. |
| width | Width of a new shape. |
| height | Height of a new shape. |

### Returns
New shape.

## toArray(int, int) method

 Creates and returns an array with all shapes from the specified range in it.
 

### Parameters

| Name | Description |
| --- | --- |
| startIndex | An index of a first shape to return. |
| count | A number of shapes to return. |

### Returns
Array of Shape

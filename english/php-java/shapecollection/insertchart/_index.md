## insertChart(int type, float x, float y, float width, float height, int index)  method

 Creates a new Chart, initialize it with sample series data and settings and inserts 
 it to the specified position in the collection.
 

### Parameters

| Name | Description |
| --- | --- |
| type | Type of chart. |
| x | X coordinate of a new chart. |
| y | Y coordinate of a new chart. |
| width | Chart's width. |
| height | Chart's height. |
| index | Chart's position in the collection. |

### Returns
Created chart.


---


## insertChart(int type, float x, float y, float width, float height, int index, boolean initWithSample)  method

 Creates a new Chart and inserts it to the specified position in the collection.
 

### Parameters

| Name | Description |
| --- | --- |
| type | Type of chart. |
| x | X coordinate of a new chart. |
| y | Y coordinate of a new chart. |
| width | Chart's width. |
| height | Chart's height. |
| index | Chart's position in the collection. |
| initWithSample | If true then new chart will be initialized with sample series data and settings. If false then new chart will have no series and minimum settings. In this case chart creation will be more fast. |

### Returns
Created chart.

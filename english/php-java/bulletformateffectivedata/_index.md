## BulletFormatEffectiveData class

 Immutable object which contains effective paragraph bullet formatting properties.
 
 This class is used as a part of  ParagraphFormatEffectiveData.
 

## Methods

| Name | Description |
| --- | --- |
| [getActualBulletValue](getactualbulletvalue)() | Returns actual bullet value for parent paragraph. Read-only String. |
| [getBulletChar](getbulletchar)() | Returns the bullet string of a paragraph. Read-only String. |
| [getChar](getchar)() | Returns the bullet char of a paragraph. Read-only char. |
| [getColor](getcolor)() | Returns the color of a bullet. Read-only Color( #getColor). |
| [getFillFormat](getfillformat)() | Returns the bullet fill format of a paragraph. Read-only IFillFormatEffectiveData. |
| [getFont](getfont)() | Returns the bullet font of a paragraph. Read-only IFontData. |
| [getHeight](getheight)() | Returns the bullet height of a paragraph. Read-only float. |
| [getNumberedBulletStartWith](getnumberedbulletstartwith)() | Returns the first number which is used for group of numbered bullets. Read-only short. |
| [getNumberedBulletStyle](getnumberedbulletstyle)() | Returns the style of a numbered bullet. Read-only NumberedBulletStyle. |
| [getPicture](getpicture)() | Returns the picture used as a bullet in the paragraph. Read-only IPictureEffectiveData. |
| [getType](gettype)() | Returns the bullet type of a paragraph. Read-only BulletType. |
| [hasBullet](hasbullet)() | Determines if the paragraph has a bullet. Read-only boolean. |
| [isBulletHardColor](isbullethardcolor)() | Determines whether the bullet has own color or inherits it from the first portion in the paragraph. Returns true if bullet has own color and false if bullet inherits color from the first portion in the paragraph. Read-only boolean. |
| [isBulletHardFont](isbullethardfont)() | Determines whether the bullet has own font or inherits it from the first portion in the paragraph. Returns true if bullet has own font and true if bullet inherits font from the first portion in the paragraph. Read-only boolean. |

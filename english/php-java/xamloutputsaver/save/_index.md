## save(String path, byte[] binaryData)  method

 Saves a bytes array to a destination location
 

### Parameters

| Name | Type | Description |
| --- | --- | --- |
| path | String | The destination path |
| binaryData | byte[] | A binary data for saving to a destination location |

### Returns
void

### Exception

| Exception | Condition |
| --- | --- |
 | XamlException | Thrown when a 'path' or 'binaryData' parameter is null |

## equals(Object obj)  method

 Determines whether two TabsEx instances are equal.
 

### Parameters

| Name | Description |
| --- | --- |
| obj | The TabsEx to compare with the current TabsEx. |

### Returns
true if the specified TabsEx is equal to the current TabsEx; otherwise, false.

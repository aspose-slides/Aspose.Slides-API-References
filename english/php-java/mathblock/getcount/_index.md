## getCount()  method

 Gets the number of child math elements actually contained in the collection.
 Read-only  int.
 
Example:
 
```php
  $mathBlock = new MathBlock(new MathematicalText("x"));
  $count = $mathBlock->getCount();
```

### Returns
int

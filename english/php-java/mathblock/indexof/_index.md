## indexOf([MathLeftSubSuperscriptElement](../../mathleftsubsuperscriptelement) item)  method

 Determines the index of a specific math element in collection.
 
Example:
 
```php
  $mathBlock = new MathBlock(new MathematicalText("x"));
  $plusElement = new MathematicalText("+");
  $mathBlock->add($plusElement);
  $mathBlock->add(new MathRadical(new MathematicalText("x"), new MathematicalText("3")));
  $index = $mathBlock->indexOf($plusElement);
```

### Parameters

| Name | Type | Description |
| --- | --- | --- |
| item | [MathLeftSubSuperscriptElement](../../mathleftsubsuperscriptelement) | The element to locate in the collection. |

### Returns
int

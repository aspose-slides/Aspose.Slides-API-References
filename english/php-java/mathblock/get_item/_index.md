## get_Item(int index)  method

 Gets or sets IMathElement at the specified index.
 
Example:
 
```php
  $mathBlock = new MathBlock(new MathematicalText("x"));
  $firstElem = $mathBlock->get_Item(0);
```

### Parameters

| Name | Type | Description |
| --- | --- | --- |

# getPieSplitPosition() method

 Specifies a value that shall be used to determine which data points 
 are in the second pie or bar on a pie-of-pie or bar-of-pie chart. 
 Is used together with PieSplitBy property.
 Read/write  double.
 

##  Returns
double

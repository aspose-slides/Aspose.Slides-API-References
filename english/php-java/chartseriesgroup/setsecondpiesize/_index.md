## setSecondPieSize(int value)  method

 Specifies the size of the second pie or bar of a pie-of-pie chart or 
 a bar-of-pie chart, as a percentage of the size of the first pie (can 
 be between 5 and 200 percents).
 Read/write  int.
 

### Returns
int

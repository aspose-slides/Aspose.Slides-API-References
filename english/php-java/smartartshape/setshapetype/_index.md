# setShapeType(int) method

 Returns or sets the geometry preset type.
 Note: on value changing all adjustment values will reset to their default values.
 Read/write  ShapeType.
 

##  Returns
ShapeType

##  Exception
com.aspose.ms.System.ArgumentException Thrown when value is ShapeType.NotDefined or ShapeType.Custom

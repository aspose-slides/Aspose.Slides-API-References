## TintEffectiveData class

 Immutable object which represents a Tint effect.
 Shifts effect color values towards/away from hue by the specified amount.
 

## Methods

| Name | Description |
| --- | --- |
| [getAmount](getamount)() | Returns effect amount. Read-only float. |
| [getHue](gethue)() | Returns hue. Read-only float. |

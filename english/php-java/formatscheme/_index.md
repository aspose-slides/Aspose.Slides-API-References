# FormatScheme class

 Stores theme-defined formats for the shapes.

| name | return type | description |
| --- | --- | --- |
| [getBackgroundFillStyles](/slides/php-java/formatscheme/getbackgroundfillstyles/)() | IFillFormatCollection | Returns a collection of theme defined background fill styles. Read-only IFillFormatCollection. |
| [getEffectStyles](/slides/php-java/formatscheme/geteffectstyles/)() | IEffectStyleCollection | Returns a collection of theme defined effect styles. Read-only IEffectStyleCollection. |
| [getFillStyles](/slides/php-java/formatscheme/getfillstyles/)() | IFillFormatCollection | Returns a collection of theme defined fill styles. Read-only IFillFormatCollection. |
| [getLineStyles](/slides/php-java/formatscheme/getlinestyles/)() | ILineFormatCollection | Returns a collection of theme defined line styles. Read-only ILineFormatCollection. |
| [getPresentation](/slides/php-java/formatscheme/getpresentation/)() | IPresentation | Returns the parent presentation. Read-only IPresentation. |
| [getSlide](/slides/php-java/formatscheme/getslide/)() | IBaseSlide | Returns the parent slide. Read-only IBaseSlide. |

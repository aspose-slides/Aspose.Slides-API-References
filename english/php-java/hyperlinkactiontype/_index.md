# HyperlinkActionType class

 Represents a type of hyperlink action.
 

## Constants

| name | description |
| --- | --- |
| Hyperlink | Usual hyperlink. |
| JumpEndShow | Jump to the end of slideshow. |
| JumpFirstSlide | Jump to the first slide of the presentation. |
| JumpLastSlide | Jump to the last slide of the presentation. |
| JumpLastViewedSlide | Jump to the last viewed slide. |
| JumpNextSlide | Jump to the next slide. |
| JumpPreviousSlide | Jump to the previous slide. |
| JumpSpecificSlide | Jump to the specific slide, referred by ( IHyperlink#getTargetSlide) property. |
| NoAction | No action. |
| OpenFile | Open referenced file. |
| OpenPresentation | Open referenced presentation. |
| StartCustomSlideShow | Start custom slideshow. |
| StartMacro | Start executing macro script. |
| StartProgram | Start program. |
| StartStopMedia | Start/stop playing mediafile. |
| Unknown | Unrecognized action type. |

## removeAt(int) method

 Removes the author at the specified index of the collection.
 

### Parameters

| Name | Description |
| --- | --- |
| index | The zero-based index of the element to remove. |

### Exception

| Exception | Condition |
| --- | --- |
 | PptxEditException | Thrown if author is already removed. |

# setCharacter(char) method

 Grouping Character
 Default value: U+23DF (BOTTOM CURLY BRACKET)
 
Example:
 
```php
  $groupingCharacter = new MathGroupingCharacter(new MathematicalText("abc"));
  $groupingCharacter->setCharacter('?');// Bottom Parenthesis

```

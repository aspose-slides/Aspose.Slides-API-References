# VbaModuleCollection class

 Represents a collection of a VBA Project modules.
 

## Methods

| name | return type | description |
| --- | --- | --- |
| [addEmptyModule](/php-java/vbamodulecollection/addemptymodule/)(String) | IVbaModule | Adds a new empty module to the VBA Project. |
| [getSyncRoot](/php-java/vbamodulecollection/getsyncroot/)() | Object | Returns a synchronization root. Read-only Object. |
| [get_Item](/php-java/vbamodulecollection/get_item/)(int) | IVbaModule | Gets the element at the specified index. |
| [isSynchronized](/php-java/vbamodulecollection/issynchronized/)() | boolean | Returns a value indicating whether access to the collection is synchronized (thread-safe). Read-only boolean. |
| [iterator](/php-java/vbamodulecollection/iterator/)() | IGenericEnumerator | Returns an enumerator that iterates through the collection. |
| [iteratorJava](/php-java/vbamodulecollection/iteratorjava/)() | IGenericEnumerator | Returns a java iterator for the entire collection. |
| [remove](/php-java/vbamodulecollection/remove/)(IVbaModule) | void | Removes the first occurrence of a specific object from the collection. |
| [size](/php-java/vbamodulecollection/size/)() | int | Gets the number of elements actually contained in the collection. Read-only int. |

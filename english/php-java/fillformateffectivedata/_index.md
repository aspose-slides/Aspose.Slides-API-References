## FillFormatEffectiveData class

 Immutable object which contains effective fill formatting properties.
 
 This class is used together with the  FillFormat class to return effective formatting values with inheritance applied.
 

## Methods

| Name | Description |
| --- | --- |
| [getFillType](getfilltype)() | Returns the type of filling. Read-only FillType. |
| [getGradientFormat](getgradientformat)() | Returns the gradient fill format. Read-only IGradientFormatEffectiveData. |
| [getPatternFormat](getpatternformat)() | Returns the pattern fill format. Read-only IPatternFormatEffectiveData. |
| [getPictureFillFormat](getpicturefillformat)() | Returns the picture fill format. Read-only IPictureFillFormatEffectiveData. |
| [getRotateWithShape](getrotatewithshape)() | Determines whether the fill should be rotated with shape. Read-only boolean. |
| [getSolidFillColor](getsolidfillcolor)() | Returns the fill color. Read-only java.awt.Color. |

## TextFrameFormatEffectiveData class

 Immutable object which contains effective text frame formatting properties.
 
 This class is used together with the  TextFrameFormat class to return effective formatting values with inheritance applied.
 

## Methods

| Name | Description |
| --- | --- |
| [getAnchoringType](getanchoringtype)() | Returns vertical anchor text in a TextFrame. Read-only TextAnchorType. |
| [getAutofitType](getautofittype)() | Returns text autofit mode. Read-only TextAutofitType. |
| [getCenterText](getcentertext)() | If true then text should be centered in box horizontally. Read-only boolean. |
| [getColumnCount](getcolumncount)() | Specifies the number of columns of text in the bounding rectangle. Read-only int. |
| [getColumnSpacing](getcolumnspacing)() | Specifies the space between text columns in the text area (in points). Read-only float. |
| [getMarginBottom](getmarginbottom)() | Returns the bottom margin (points) in a TextFrame. Read-only double. |
| [getMarginLeft](getmarginleft)() | Returns the left margin (points) in a TextFrame. Read-only double. |
| [getMarginRight](getmarginright)() | Returns the right margin (points) in a TextFrame. Read-only double. |
| [getMarginTop](getmargintop)() | Returns the top margin (points) in a TextFrame. Read-only double. |
| [getTextStyle](gettextstyle)() | Returns effective text's style. Read-only ITextStyleEffectiveData. |
| [getTextVerticalType](gettextverticaltype)() | Determines text orientation. Read-only TextVerticalType. |
| [getWrapText](getwraptext)() | True or NotDefined if text is wrapped at TextFrame's margins. Read-only boolean. |

# ZoomObject class

 Represents an Zoom object in a slide.
 

## Methods

| name | return type | description |
| --- | --- | --- |
| [getImage](/php-java/zoomobject/getimage/)() | IPPImage | Gets or sets image for zoom object. Read/write IPPImage. |
| [getImageType](/php-java/zoomobject/getimagetype/)() | int | Gets or sets the image type of a zoom object. Read/write ZoomImageType. Default value: Preview Specifies whether the Zoom object is using the slide preview or a cover image. |
| [getReturnToParent](/php-java/zoomobject/getreturntoparent/)() | boolean | Gets or sets the navigation behavior in slideshow. Read/write boolean. Default value: false True value of the property specifies return to parent navigation behavior in slideshow. |
| [getShowBackground](/php-java/zoomobject/getshowbackground/)() | boolean | Gets or sets value that specifies whether the Zoom will use the background of the destination slide. Read/write boolean. Default value: true |
| [getTransitionDuration](/php-java/zoomobject/gettransitionduration/)() | float | Gets or sets the duration of the transition between Zoom and slide. Read/write float. Default value: 1.0f If not specified (TransitionDur = 0), it will use the destination slide transition and the timings associated with that transition. |
| [setImage](/php-java/zoomobject/setimage/)(IPPImage) | void | Gets or sets image for zoom object. Read/write IPPImage. |
| [setImageType](/php-java/zoomobject/setimagetype/)(int) | void | Gets or sets the image type of a zoom object. Read/write ZoomImageType. Default value: Preview Specifies whether the Zoom object is using the slide preview or a cover image. |
| [setReturnToParent](/php-java/zoomobject/setreturntoparent/)(boolean) | void | Gets or sets the navigation behavior in slideshow. Read/write boolean. Default value: false True value of the property specifies return to parent navigation behavior in slideshow. |
| [setShowBackground](/php-java/zoomobject/setshowbackground/)(boolean) | void | Gets or sets value that specifies whether the Zoom will use the background of the destination slide. Read/write boolean. Default value: true |
| [setTransitionDuration](/php-java/zoomobject/settransitionduration/)(float) | void | Gets or sets the duration of the transition between Zoom and slide. Read/write float. Default value: 1.0f If not specified (TransitionDur = 0), it will use the destination slide transition and the timings associated with that transition. |

# PictureFrame class

 Represents a frame with a picture inside.
 

## Methods

| name | return type | description |
| --- | --- | --- |
| [getPictureFormat](/php-java/pictureframe/getpictureformat/)() | IPictureFillFormat | Returns the PictureFillFormat object for a picture frame. Read-only IPictureFillFormat. |
| [getPictureFrameLock](/php-java/pictureframe/getpictureframelock/)() | IPictureFrameLock | Returns shape's locks. Read-only IPictureFrameLock. |
| [getRelativeScaleHeight](/php-java/pictureframe/getrelativescaleheight/)() | float | Returns or sets the scale of height(relative to original picture size) of the picture frame. Value 1.0 corresponds to 100%. Read/write float. |
| [getRelativeScaleWidth](/php-java/pictureframe/getrelativescalewidth/)() | float | Returns or sets the scale of width (relative to original picture size) of the picture frame. Value 1.0 corresponds to 100%. Read/write float. |

Read/write ShapeType. |

Read/write ShapeType. |

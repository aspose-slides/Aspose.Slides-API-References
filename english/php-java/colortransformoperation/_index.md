# ColorTransformOperation class

 Defines color transform operation.
 

## Constants

| name | description |
| --- | --- |
| AddAlpha | Adds a parameter's value to an alpha component of the color. Parameter is in range between -1 and 1. |
| AddBlue | Adds a parameter's value to a blue component of the color. Parameter is in range between -1 and 1. |
| AddGreen | Adds a parameter to a green component of the color. Parameter is in range between -1 and 1. |
| AddHue | Adds parameter's value to hue component of the color. Parameter is in range between -360 and 360. |
| AddLuminance | Adds a parameter's value to a luminance component of the color. Parameter is in range between -1 and 1. |
| AddRed | Adds a parameter's value to a red component of the color. Parameter is in range between -1 and 1. |
| AddSaturation | Adds a parameter's value to a saturation component of the color. Parameter is in range between -1 and 1. |
| Complement | Changes the color to a RGB complementary one. m = Max(r, g, b); r = m - r; g = m - g; b = m - b; |
| Gamma | Gamma correction. Parameter ignored. |
| Grayscale | Changes the color to a gray one with same lightness. Parameter ignored. |
| Inverse | Changes the color to an inverted color. r = 1 - r; g = 1 - g; b = 1 - b; |
| InverseGamma | Inverse gamma correction. Parameter ignored. |
| MultiplyAlpha | Multiplies an alpha component to a parameter's value. |
| MultiplyBlue | Multiplies a blue component of the color to a parameter's value. |
| MultiplyGreen | Multiplies a green component of the color to a parameter's value. |
| MultiplyHue | Multiplies a hue component to a parameter's value. |
| MultiplyLuminance | Multiplies a luminance component to a parameter's value. |
| MultiplyRed | Multiplies a red component to a parameter. |
| MultiplySaturation | Multiplies a saturation component to a parameter's value. |
| SetAlpha | Defines an alpha component of the color. Parameter is in range between 0 (transparent) and 1 (opaque). |
| SetBlue | Changes a blue component of the color to a parameter's value. Parameter is in range between 0 and 360. |
| SetGreen | Changes a green component of the color to a parameter's value value. Parameter is in range between 0 and 1. |
| SetHue | Changes a hue component of the color to a parameter's value. Parameter is in range between 0 and 360. |
| SetLuminance | Changes a luminance component of the color to a parameter's value. Parameter is in range between 0 and 1. |
| SetRed | Changes a red component of the color to a parameter's value. Parameter is in range between 0 and 1. |
| SetSaturation | Changes a saturation component of the color to a parameter's value. Parameter is in range between 0 and 1. |
| Shade | Shades the color. Parameter is in range between 0 (original color) and 1 (black). |
| Tint | Tints the color. Parameter is in range between 0 (original color) and 1 (white). |

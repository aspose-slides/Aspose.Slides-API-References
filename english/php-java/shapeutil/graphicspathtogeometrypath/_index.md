## graphicsPathToGeometryPath(java.awt.Shape) method

 Converts a  java.awt.Shape graphicsPath to the  IGeometryPath
 

### Parameters

| Name | Description |
| --- | --- |
| graphicsPath | java.awt.Shape element. Return value of the method call can be used to change the geometry of a IGeometryShape object with IGeometryShape.setGeometryPaths() method. |

### Returns
IGeometryPath

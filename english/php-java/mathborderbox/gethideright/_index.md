## getHideRight()  method

 Hide Right Edge (default is false) - specifies the hidden or shown state of the right edge of border box.
 
Example:
 
```php
  $borderBox = new MathBorderBox(new MathematicalText("x"));
  $borderBox->setHideRight(true);
```

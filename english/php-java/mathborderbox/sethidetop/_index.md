## setHideTop(boolean) method

 Hide Top Edge (default is false) - specifies the hidden or shown state of the top edge of border box.
 
Example:
 
```php
  $borderBox = new MathBorderBox(new MathematicalText("x"));
  $borderBox->setHideTop(true);
```

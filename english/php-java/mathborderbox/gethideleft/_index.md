## getHideLeft()  method

 Hide Left Edge (default is false) - specifies the hidden or shown state of the left edge of border box.
 
Example:
 
```php
  $borderBox = new MathBorderBox(new MathematicalText("x"));
  $borderBox->setHideLeft(true);
```

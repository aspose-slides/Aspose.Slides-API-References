# AdjustValue class

 Represents a geometry shape's adjustment value.
 These values affect shape's form.
 

## Methods

| --- | --- | --- |
| [getAngleValue](/slides/php-java/adjustvalue/getanglevalue/)() | float | Returns or sets value, interpreting it as angle in degrees. Read/write float. |
| [getName](/slides/php-java/adjustvalue/getname/)() | String | Returns a name of this adjustment value. Read-only String. |
| [getRawValue](/slides/php-java/adjustvalue/getrawvalue/)() | long | Returns or sets ajustment value "as is". Read/write long. |
| [setAngleValue](/slides/php-java/adjustvalue/setanglevalue/)(float) | void | Returns or sets value, interpreting it as angle in degrees. Read/write float. |

## getSuperscript()  method

 Specifies a supersript argument that, for example, in the case of an integral, sets the upper limit
 
Example:
 
```php
  $naryOperator = new MathematicalText("x")->nary(MathNaryOperatorTypes.Summation, "x=1", "100");
  $superscriptArg = $naryOperator->getSuperscript();
```

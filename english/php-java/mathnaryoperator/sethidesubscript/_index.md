# setHideSubscript(boolean) method

 Hide Subscript
 
Example:
 
```php
  $naryOperator = new MathematicalText("x")->nary(MathNaryOperatorTypes.Summation, "x=1", "100");
  $naryOperator->setHideSubscript(true);
```

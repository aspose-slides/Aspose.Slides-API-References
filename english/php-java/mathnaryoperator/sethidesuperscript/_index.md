# setHideSuperscript(boolean) method

 Hide Superscript
 
Example:
 
```php
  $naryOperator = new MathematicalText("x")->nary(MathNaryOperatorTypes.Summation, "x=1", "100");
  $naryOperator->setHideSuperscript(true);
```

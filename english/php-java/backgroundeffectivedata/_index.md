## BackgroundEffectiveData class

 Immutable object which contains effective background properties.
 
 This class is used together with the  Background class to return effective formatting values with inheritance applied.
 

## Methods

| Name | Description |
| --- | --- |
| [getEffectFormat](geteffectformat)() | Returns effective effect format. Read-only IEffectFormatEffectiveData. |
| [getFillFormat](getfillformat)() | Returns effective fill format. Read-only IFillFormatEffectiveData. |

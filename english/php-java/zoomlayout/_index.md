# ZoomLayout class

 Specifies the summary zoom layout.
 

## Constants

| name | description |
| --- | --- |
| FixedLayout | Specifies the layout with user-defined Zoom positioning. |
| GridLayout | Specifies the layout as a grid layout for a list of objects. |

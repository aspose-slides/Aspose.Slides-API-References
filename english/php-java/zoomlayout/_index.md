## ZoomLayout class

 Specifies the summary zoom layout.
 

## Constants

| Name | Value | Description |
| --- | --- | --- |
| FixedLayout | 1 | Specifies the layout with user-defined Zoom positioning. |
| GridLayout | 0 | Specifies the layout as a grid layout for a list of objects. |

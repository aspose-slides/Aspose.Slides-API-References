# ColorType class

 Represents different color modes.
 

## Constants

| name | description |
| --- | --- |
| HSL | High definition HSL color. |
| NotDefined | Color is not defined at all. |
| Preset | Preset Color. |
| RGB | Standard 24bit RGB color. |
| RGBPercentage | High definition RGB color. |
| Scheme | Scheme color. |
| System | System color. |

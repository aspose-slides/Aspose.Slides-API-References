# DataSourceType class

 Data source types.
 

## Constants

| name | description |
| --- | --- |
| DoubleLiterals | Use fast-access double variable for storing XValue, YValue, Value or BubbleSize of data point. Worksheet is NOT used for storing values. |
| StringLiterals | Use fast-access string variable for storing XValue, YValue, Value or BubbleSize of data point. Worksheet is NOT used for storing values. |
| Worksheet | Use CellXls objects for storing XValue, YValue, Value or BubbleSize of data point in worksheet. |

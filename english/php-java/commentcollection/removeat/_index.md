## removeAt(int) method

 Removes the element at the specified index in a collection.
 

### Parameters

| Name | Description |
| --- | --- |
| index | The zero-based index of the element to remove. |

### Exception

| Exception | Condition |
| --- | --- |
 | PptxEditException | Thrown if comment is already removed. |

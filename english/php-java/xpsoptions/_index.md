# XpsOptions class

 Provides options that control how a presentation is saved in XPS format.
 

## Constructors

| name | description |
| --- | --- |
| [XpsOptions](/php-java/xpsoptions/xpsoptions/)() | Default constructor. |

## Methods

| name | return type | description |
| --- | --- | --- |
| [getDrawSlidesFrame](/php-java/xpsoptions/getdrawslidesframe/)() | boolean | True to draw black frame around each slide. Read/write boolean. Default is false. |
| [getSaveMetafilesAsPng](/php-java/xpsoptions/getsavemetafilesaspng/)() | boolean | True to convert all metafiles used in a presentation to the PNG images. Read/write boolean. Default is true. |
| [getShowHiddenSlides](/php-java/xpsoptions/getshowhiddenslides/)() | boolean | Specifies whether the generated document should include hidden slides or not. Default is false. |
| [setDrawSlidesFrame](/php-java/xpsoptions/setdrawslidesframe/)(boolean) | void | True to draw black frame around each slide. Read/write boolean. Default is false. |

## ChartDataCell class

 Represents cell for chart data.
 

## Methods

| Name | Description |
| --- | --- |
| [calculate](calculate)(boolean) | If the cell contains a formula, the value will be updated base on that formula. |
| [getChartDataWorksheet](getchartdataworksheet)() | Gets the worksheet. Read-only IChartDataWorksheet. |
| [getColumn](getcolumn)() | Returns the index of the column of worksheet in which the cell is located. Read-only int. |
| [getCustomNumberFormat](getcustomnumberformat)() | Gets or sets the custom display format of numbers and dates. If value is empty will be used PresetNumberFormat value. Read/write String. |
| [getFormula](getformula)() | Gets or sets the formula in A1-style. |
| [getPresetNumberFormat](getpresetnumberformat)() | Gets or sets the built-in display format of numbers and dates. Preset number must be in [0..22] or [37..49]". Read/write byte. |
| [getR1C1Formula](getr1c1formula)() | Gets or sets the formula in R1C1-style. |
| [getRow](getrow)() | Returns the index of the row of worksheet in which the cell is located. Read-only int. |
| [getValue](getvalue)() | Gets or sets the value. Read/write Object. |
| [isHidden](ishidden)() | Determines whether the cell is hidden. Read-only boolean. |
| [setCustomNumberFormat](setcustomnumberformat)(String) | Gets or sets the custom display format of numbers and dates. If value is empty will be used PresetNumberFormat value. Read/write String. |
| [setFormula](setformula)(String) | Gets or sets the formula in A1-style. |
| [setPresetNumberFormat](setpresetnumberformat)(byte) | Gets or sets the built-in display format of numbers and dates. Preset number must be in [0..22] or [37..49]". Read/write byte. |
| [setR1C1Formula](setr1c1formula)(String) | Gets or sets the formula in R1C1-style. |
| [setValue](setvalue)(Object) | Gets or sets the value. Read/write Object. |

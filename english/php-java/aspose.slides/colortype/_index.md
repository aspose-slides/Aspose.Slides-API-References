## ColorType class

 Represents different color modes.

| RGB | 0 | Standard 24bit RGB color. |
| RGBPercentage | 1 | High definition RGB color. |
| HSL | 2 | High definition HSL color. |
| Scheme | 3 | Scheme color. |

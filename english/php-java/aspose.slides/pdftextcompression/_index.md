## PdfTextCompression class

 Constants which define the type of a compression applied to all content in the PDF file except images.
 

## Constants

| Name | Value | Description |
| --- | --- | --- |
| None | 0 | No compression. |
| Flate | 1 | Flate (ZIP) compression. |

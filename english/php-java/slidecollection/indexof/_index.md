## indexOf(com.aspose.slides.ISlide) method

 Returns an index of the specified slide in the collection.
 

### Parameters

| Name | Description |
| --- | --- |
| slide | Slide to find. |

### Returns
Index of a slide or -1 if slide not from this collection.

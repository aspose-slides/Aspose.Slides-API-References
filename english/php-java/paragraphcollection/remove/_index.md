## remove([Paragraph](../../paragraph) item)  method

| Name | Type | Description |
| --- | --- | --- |
| item | Paragraph | The object to remove from the IGenericCollection. |

### Returns
IGenericCollection

### Exception

| Exception | Condition |

 | NotSupportedException | The IGenericCollection is read-only. |

# copyTo(com.aspose.slides.IParagraph[], int) method

 Copies the elements of the  IGenericCollection to an  Array, starting at a particular  Array index.
 

##  Parameters

| name | description |
| --- | --- |
| array | The one-dimensional Array that is the destination of the elements copied from IGenericCollection. The Array must have zero-based indexing. |
| arrayIndex | The zero-based index in array at which copying begins. |

##  Returns
Array

##  Exception
ArgumentException The number of elements in the source IGenericCollection is greater than the available space from arrayIndex to the end of the destination array.

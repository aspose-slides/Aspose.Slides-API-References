# setDescription(java.lang.String) method

 Returns the text description of the Summary Zoom Section object. 
 
Example:
 
```php
  $zoomSection = $zoomFrame->getSummaryZoomCollection()->get_Item(1);
  $zoomSection->setDescription("Description");
```

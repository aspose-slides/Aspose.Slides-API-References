## setTitle(String value)  method

 Returns the text title of the Summary Zoom Section object. 
 
Example:
 
```php
  $zoomSection = $zoomFrame->getSummaryZoomCollection()->get_Item(1);
  $zoomSection->setTitle("Title");
```

### Returns
void

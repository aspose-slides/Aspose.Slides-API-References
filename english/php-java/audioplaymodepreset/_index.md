# AudioPlayModePreset class

 Constants which define how a sound is played.
 

## Constants

| name | description |
| --- | --- |
| AllSlides | Play on all slides. Please use ( IAudioFrame#getPlayAcrossSlides/ IAudioFrame#setPlayAcrossSlides(boolean)) property instead. This element is out of date since PowerPoint 2013. |
| Auto | Play automatically. |
| InClickSequence | Play automatically with a click. |
| Mixed |  |
| OnClick | Play only when the icon is clicked on. |

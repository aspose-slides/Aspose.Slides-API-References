## ZoomImageType class

 Defines whether the Zoom object is using the slide preview or a cover image.
 

## Constants

| Name | Value | Description |
| --- | --- | --- |
| Preview | 1 | Use the image of the slide or section. |
| Cover | 2 | Use a custom image. |

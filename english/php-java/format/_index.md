# Format class

 Represents chart format properties.
 

## Methods

| name | return type | description |
| --- | --- | --- |
| [getEffect](/php-java/format/geteffect/)() | IEffectFormat | Returns effects used for a chart. Read-only IEffectFormat. |
| [getEffect3D](/php-java/format/geteffect3d/)() | IThreeDFormat | Returns 3D format of a chart. Read-only IThreeDFormat. |
| [getFill](/php-java/format/getfill/)() | IFillFormat | Returns fill style properties of a chart. Read-only IFillFormat. |
| [getLine](/php-java/format/getline/)() | ILineFormat | Returns line style properties of a chart. Read-only ILineFormat. |

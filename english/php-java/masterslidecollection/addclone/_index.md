# addClone(com.aspose.slides.IMasterSlide) method

 Adds a copy of a specified master slide to the end of the collection.
 Linked layout slides will be copied too.
 

##  Parameters

| name | description |
| --- | --- |
| sourceMaster | Slide to clone. |

Added slide.

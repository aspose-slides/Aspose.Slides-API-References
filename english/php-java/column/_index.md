# Column class

 Represents a column in a table.
 

## Methods

| name | return type | description |
| --- | --- | --- |
| [getColumnFormat](/php-java/column/getcolumnformat/)() | IColumnFormat | Returns the ColumnFormat object that contains formatting properties for this column. Read-only IColumnFormat. |
| [getWidth](/php-java/column/getwidth/)() | double | Returns or sets the width of a column. Read/write double. |
| [setTextFormat](/php-java/column/settextformat/)(IPortionFormat) | void | Sets defined portion format properties to all column cells' portions. |
| [setTextFormat](/php-java/column/settextformat/)(IParagraphFormat) | void | Sets defined paragraph format properties to all column cells' paragraphs. |
| [setTextFormat](/php-java/column/settextformat/)(ITextFrameFormat) | void | Sets defined text frame format properties to all column cells' text frames. |
| [setWidth](/php-java/column/setwidth/)(double) | void | Returns or sets the width of a column. Read/write double. |

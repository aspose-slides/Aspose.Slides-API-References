## TextStyleEffectiveData class

 Immutable object which contains effective text style properties.
 
 This class is used together with the  TextStyle class to return effective formatting values with inheritance applied.
 

## Methods

| Name | Description |
| --- | --- |
| [getDefaultParagraphFormat](getdefaultparagraphformat)() | Effective default paragraph properties. Read-only IParagraphFormatEffectiveData. |
| [getLevel](getlevel)(int) | Returns level of effective style. |

## getPosition()  method

 Returns or sets zero-based position of node among sibling nodes.
 Read/write  int.
 

### Exception

| Exception | Condition |
| --- | --- |
 | ArgumentOutOfRangeException | value is less than 0. -or- value is equal to or greater than siblings count |

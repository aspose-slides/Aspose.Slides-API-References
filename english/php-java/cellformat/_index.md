 Represents format of a table cell.

| name | return type | description |
| --- | --- | --- |
| [getBorderBottom](/php-java/cellformat/getborderbottom/)() | ILineFormat | Returns a bottom border line properties object. Read-only ILineFormat. |
| [getBorderDiagonalDown](/php-java/cellformat/getborderdiagonaldown/)() | ILineFormat | Returns a top-left to bottom-right diagonal line properties object. Read-only ILineFormat. |
| [getBorderDiagonalUp](/php-java/cellformat/getborderdiagonalup/)() | ILineFormat | Returns a bottom-left to top-right diagonal line properties object. Read-only ILineFormat. |
| [getBorderLeft](/php-java/cellformat/getborderleft/)() | ILineFormat | Returns a left border line properties object. Read-only ILineFormat. |
| [getBorderRight](/php-java/cellformat/getborderright/)() | ILineFormat | Returns a right border line properties object. Read-only ILineFormat. |
| [getBorderTop](/php-java/cellformat/getbordertop/)() | ILineFormat | Returns a top border line properties object. Read-only ILineFormat. |
| [getEffective](/php-java/cellformat/geteffective/)() | ICellFormatEffectiveData | Gets effective table cell formatting properties with inheritance and table styles applied. |
| [getFillFormat](/php-java/cellformat/getfillformat/)() | IFillFormat | Returns a cell fill properties object. Read-only IFillFormat. |

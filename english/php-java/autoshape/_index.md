## AutoShape class

  Represents an AutoShape.
 

## Methods

| Name | Description |
| --- | --- |
| [addTextFrame](addtextframe)(String) | Adds a new TextFrame to a shape. If shape already has TextFrame then simply changes its text. |
| [getAutoShapeLock](getautoshapelock)() | Returns autoshape's locks. Read-only IAutoShapeLock. |
| [getShapeLock](getshapelock)() | Returns shape's locks. Read-only IAutoShapeLock. |
| [getTextFrame](gettextframe)() | Returns TextFrame object for the AutoShape. Read-only ITextFrame. |
| [getUseBackgroundFill](getusebackgroundfill)() | Determines whether this autoshape should be filled with slide's background fill instead of specified by style or fill format. Read/write boolean. |
| [isTextBox](istextbox)() | Specifies if the shape is a text box. If shape is not specified to be a text box does not mean that it cannot have text attached to it. A text box is merely a specialized shape with specific properties. |
| [setUseBackgroundFill](setusebackgroundfill)(boolean) | Determines whether this autoshape should be filled with slide's background fill instead of specified by style or fill format. Read/write boolean. |

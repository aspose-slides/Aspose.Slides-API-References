# PresentationHeaderFooterManager class

 Represents manager which holds behavior of all footer, date-time and page number placeholders of presentation.
 

## Methods

| name | return type | description |
| --- | --- | --- |
| [setAllDateTimesText](/php-java/presentationheaderfootermanager/setalldatetimestext/)(String) | void | Sets text to all date-time placeholders, including master slides, layout slides, slides, notes master, notes slides and handout master. |
| [setAllDateTimesVisibility](/php-java/presentationheaderfootermanager/setalldatetimesvisibility/)(boolean) | void | Changes all date-time placeholders visibility, including master slides, layout slides, slides, notes master, notes slides and handout master. |
| [setAllFootersText](/php-java/presentationheaderfootermanager/setallfooterstext/)(String) | void | Sets text to all footer placeholders, including master slides, layout slides, slides, notes master, notes slides and handout master. |
| [setAllFootersVisibility](/php-java/presentationheaderfootermanager/setallfootersvisibility/)(boolean) | void | Changes all footer placeholders visibility, including master slides, layout slides, slides, notes master, notes slides and handout master. |
| [setAllHeadersText](/php-java/presentationheaderfootermanager/setallheaderstext/)(String) | void | Sets text to all header placeholders, including notes master, notes slides and handout master. |
| [setAllHeadersVisibility](/php-java/presentationheaderfootermanager/setallheadersvisibility/)(boolean) | void | Changes all header placeholders visibility, including notes master, notes slides and handout master. |
| [setAllSlideNumbersVisibility](/php-java/presentationheaderfootermanager/setallslidenumbersvisibility/)(boolean) | void | Changes all page number placeholders visibility, including master slides, layout slides, slides, notes master, notes slides and handout master. |
| [setVisibilityOnAllTitleSlides](/php-java/presentationheaderfootermanager/setvisibilityonalltitleslides/)(boolean) | void | Changes the footer, date-time and page number placeholders visibility for all title slides and for first layout slide. Title slides – slides based on first layout slide (regardless of type of this first layout). |

# setAllSlideNumbersVisibility(boolean) method

 Changes all page number placeholders visibility, including master slides, layout slides, slides, 
 notes master, notes slides and handout master.
 

##  Parameters

| name | description |
| --- | --- |
| isVisible | true - makes a page number placeholders visible, otherwise - hides them. |

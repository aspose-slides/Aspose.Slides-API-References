# setAllFootersVisibility(boolean) method

 Changes all footer placeholders visibility, including master slides, layout slides, slides, 
 notes master, notes slides and handout master.
 

##  Parameters

| name | description |
| --- | --- |
| isVisible | true - makes a footer placeholders visible, otherwise - hides them. |

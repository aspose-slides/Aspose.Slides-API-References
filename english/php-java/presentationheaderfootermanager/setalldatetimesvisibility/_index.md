# setAllDateTimesVisibility(boolean) method

 Changes all date-time placeholders visibility, including master slides, layout slides, slides, 
 notes master, notes slides and handout master.
 

##  Parameters

| name | description |
| --- | --- |
| isVisible | true - makes a date-time placeholders visible, otherwise - hides them. |

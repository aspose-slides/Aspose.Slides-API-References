## PdfOptions.JpegQuality property

Returns or sets a value determining the quality of the JPEG images inside PDF document. Read/write Byte.

```csharp
public byte JpegQuality { get; set; }
```

## Remarks

Has effect only when a document contains JPEG images.Use this property to get or set the quality of the images inside a document when saving in PDF format. The value may vary from 0 to 100 where 0 means worst quality but maximum compression and 100 means best quality but minimum compression.The default value is 100.

## Rotation3D.RotationY property

Returns or sets the rotation degree around the Y-axis, i.e. in the X direction for 3D charts (between 0 and 360 degrees). The property matches with the 21.2.2.158 rotY (Y Rotation) item in ECMA-376 and with the "X Rotation" option in PowerPoint 2007+. Read/write UInt16.

```csharp
public ushort RotationY { get; set; }
```

## IDataLabelFormat.ShowLabelAsDataCallout property

确定指定图表的数据标签将显示为数据标注或数据标签。  如果此 DataLabelFormat 对象的父对象是数据标签的 DataLabelCollection 集合，则此 属性获取或设置默认值DataLabelCollection 集合中新数据 标签的 ShowLabelAsDataCallout 属性的值。 将此属性设置为 value 也将此值设置为 ShowLabelAsDataCallout 属性 用于 DataLabelCollection 集合中的所有数据标签 （即“DataLabels.DefaultDataLabelFormat.ShowLabelAsDataCallout = val;" 导致 all DataLabels[i].ShowLabelAsDataCallout 等于 val)。

public bool ShowLabelAsDataCallout { get; set; }

## DataLabelFormat.ShowBubbleSize property

表示指定图表的数据标签气泡大小值显示行为。 True 显示气泡大小值。虚假隐藏。 读/写Boolean。

```csharp
public bool ShowBubbleSize { get; set; }
```

### 评论

如果此 DataLabelFormat 对象的父对象是数据标签的 DataLabelCollection 集合，则此 属性获取或设置 DataLabelCollection 集合中新数据 标签的 ShowBubbleSize 属性的默认值。 使用值设置此属性也将此值设置为 ShowBubbleSize 属性 用于 DataLabelCollection 集合中的所有数据标签 （即“DataLabels.DefaultDataLabelFormat.ShowBubbleSize = val;" 导致 所有 DataLabels[i].ShowBubbleSize 等于 val)。

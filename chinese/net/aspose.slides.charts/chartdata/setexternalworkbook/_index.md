## SetExternalWorkbook(string) {#setexternalworkbook}


| 范围 | 类型 | 描述 |
| --- | --- | --- |

### 例外

| 例外 | （健康）状况 |
| --- | --- |

```csharp

   IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Pie, 50, 50, 400, 600, true);
```

## SetExternalWorkbook(string, bool) {#setexternalworkbook_1}

将外部工作簿设置为图表的数据源。

```csharp
public void SetExternalWorkbook(string workbookPath, bool updateChartData)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| workbookPath | String | 目标工作簿的路径 |
| updateChartData | Boolean | 如果值为 false，则只会更新工作簿路径。 图表数据不会从目标工作簿加载和更新。可以在目标工作簿不存在或不可用时使用。 如果值为 true，则图表数据将从目标工作簿更新。 |

### 例外

| 例外 | （健康）状况 |
| --- | --- |
| InvalidOperationException | 外部工作簿不可用或无法加载。 |

### 例子

```csharp
[C#]
using (Presentation pres = new Presentation())

   IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Pie, 50, 50, 400, 600, true);
   IChartData chartData = chart.ChartData;

   (chartData as ChartData).SetExternalWorkbook("http://路径/不存在/存在", false);


```

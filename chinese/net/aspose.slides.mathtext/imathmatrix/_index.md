## IMathMatrix interface

指定 Matrix 对象，由排列在一个或多个行和列中的子元素组成。 重要的是要注意矩阵没有内置分隔符。 要将矩阵放在括号中，您应该使用分隔符对象 (IMathDelimiter)。 空参数可用于在矩阵中创建间隙。

```csharp
public interface IMathMatrix : IMathElement
```

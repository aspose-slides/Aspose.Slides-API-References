## MathMatrix.MinColumnWidth property

以缇为单位的最小列宽（点的 1/20） 将间隙间距（也称为“列间隙”或“间隙宽度”）添加到 MinColumnWidth 以确定总矩阵列间距 （与不同列的相同边缘）. 默认值：0.

```csharp
public uint MinColumnWidth { get; set; }
```

## MathBox.NoBreak property

否 break 此属性指定对象框上的“不可破坏”属性。如果为 true，则框内不会出现换行符。 这对于包含多个二元运算符的运算符模拟器可能很重要。 如果未指定此元素，则框内可能会出现中断。 默认值：true

```csharp
public bool NoBreak { get; set; }
```

### 例子

示例：

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("*****"));
box.NoBreak = false;
```

## ISlideShowTransition.SoundIsBuiltIn property

指定此声音是否为内置声音。如果此属性设置为 true then 生成应用程序会收到警报，以检查其内置声音列表中为此 sound 指定的名称属性，然后可以根据需要显示自定义名称或 UI。 读写Boolean.

public bool SoundIsBuiltIn { get; set; }

## ITextFrameFormat.RotationAngle property

指定应用于边界框内文本的自定义旋转。如果未指定 ，则使用伴随形状的旋转。如果已指定，则为 独立于形状应用。也就是说，除了文本本身应用了旋转之外，形状还可以在 中应用旋转。 从该属性总结的可视文本旋转的结果值，并在属性 TextVerticalType 中预定义 垂直类型。 读/写Single。

```csharp
public float RotationAngle { get; set; }
```

### 例子

考虑形状顺时针旋转 90 度的情况。 除此之外，文本主体本身具有 -90 度的旋转 逆时针应用于它。然后生成的形状将显示为 被旋转，但其中的文本看起来好像根本没有旋转。

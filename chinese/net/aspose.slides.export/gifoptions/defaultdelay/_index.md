## GifOptions.DefaultDelay property

获取或设置默认延迟时间 [ms]。此值将在以下情况下使用[`AdvanceAfterTime`](../../../aspose.slides/islideshowtransition/advanceaftertime)未设置。 默认值为 1000。

```csharp
public int DefaultDelay { get; set; }
```

### 例子

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
	pres.Save("pres.gif", SaveFormat.Gif, new GifOptions { DefaultDelay = 2000 });
}
```

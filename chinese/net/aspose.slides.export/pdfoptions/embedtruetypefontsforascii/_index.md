## PdfOptions.EmbedTrueTypeFontsForASCII property

确定 Aspose.Slides 是否会为 ASCII（33..127 代码范围）文本嵌入常用字体。 始终嵌入大于 127 的字符代码的字体。 常用字体列表包括 PDF 的基本 14 种字体和其他用户指定的字体。 读/写Boolean.

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### 评论

默认为 **真的**.

## PdfOptions.BestImagesCompressionRatio property

Indica si se debe seleccionar la compresión más efectiva (en lugar de la predeterminada) para cada imagen automáticamente. Si se establece enBoolean.true, para cada imagen en presentación se elegirá el algoritmo de compresión más apropiado, lo que conducirá al menor tamaño del documento PDF resultante.  La selección de la mejor relación de compresión de imagen es computacionalmente costosa y requiere una cantidad adicional de RAM, y esta opción esBoolean.falso por defecto.

```csharp
public bool BestImagesCompressionRatio { get; set; }
```

### Observaciones

El valor predeterminado esBoolean .falso.

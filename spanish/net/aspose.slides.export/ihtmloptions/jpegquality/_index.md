## IHtmlOptions.JpegQuality property

Devuelve o establece un valor que determina la calidad de las imágenes JPEG dentro del documento PDF. Lectura/escrituraByte .

```csharp
public byte JpegQuality { get; set; }
```

### Observaciones

Solo tiene efecto cuando un documento contiene imágenes JPEG.

Utilice esta propiedad para obtener o establecer la calidad de las imágenes dentro de un documento al guardar en formato PDF. El valor puede variar de 0 a 100, donde 0 significa peor calidad pero máxima compresión y 100 significa mejor calidad pero mínima compresión.

El valor predeterminado es **95**.

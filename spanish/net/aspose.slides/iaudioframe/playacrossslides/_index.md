## IAudioFrame.PlayAcrossSlides property

Determina si se reproduce un audio en las diapositivas. Lectura/escrituraBoolean .

```csharp
public bool PlayAcrossSlides { get; set; }
```

### Ejemplos

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ISlide slide = pres.Slides[0];

    // Agregar cuadro de audio
    IAudioFrame audioFrame = slide.Shapes.AddAudioFrameLinked(50, 50, 100, 100, "sampleaudio.wav");

    // Establecer audio para reproducir a través de las diapositivas
    audioFrame.PlayAcrossSlides = true;

    // Configure el audio para que retroceda automáticamente para comenzar después de reproducir
    audioFrame.RewindAudio = true;

    pres.Save("AudioFrame_out.pptx", SaveFormat.Pptx);
}
```

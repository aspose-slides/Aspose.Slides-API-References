## IPresentationInfo.CheckPassword method

Comprueba si una contraseña es correcta para una presentación protegida con contraseña abierta.

```csharp
public bool CheckPassword(string password)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| password | String | La contraseña a comprobar. |

### Valor_devuelto

Verdadero si la presentación está protegida con contraseña abierta y la contraseña es correcta y falso en caso contrario.

### Observaciones

Cuando la contraseña es nula o está vacía, este método devuelve false.

### Ejemplos

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo("pres.pptx");
bool isPasswordCorrect = info.CheckPassword("my_password");
```

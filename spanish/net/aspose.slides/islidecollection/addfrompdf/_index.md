## AddFromPdf(Stream) {#addfrompdf}

Crea diapositivas a partir del documento PDF y las agrega al final de la colección.

```csharp
public ISlide[] AddFromPdf(Stream pdfStream)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| pdfStream | Stream | Una secuencia que se utilizará como fuente del documento PDF |

### Valor_devuelto

Diapositivas añadidas

### Ejemplos

Ejemplo:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    using (Stream stream = new FileStream("document.pdf", FileMode.Open, FileAccess.Read, FileShare.Read))
    {
        pres.Slides.AddFromPdf(stream);
    }
    
    pres.Save("fromPdfDocument.pptx", SaveFormat.Pptx);
}
```

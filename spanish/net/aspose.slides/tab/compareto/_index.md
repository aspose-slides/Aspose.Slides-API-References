## Tab.CompareTo method

Compara la instancia actual con otro objeto del mismo tipo.

```csharp
public int CompareTo(object obj)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| obj | Object | Un objeto para comparar con esta instancia. |

### Valor_devuelto

Un entero de 32 bits que indica el orden relativo de los comparandos. El valor de retorno tiene estos significados:  &lt; 0 - Esta instancia es menor que obj. = 0 - Esta instancia es igual a obj. &gt; 0: esta instancia es mayor que obj.

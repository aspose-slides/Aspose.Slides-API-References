## MathGroupingCharacter.VerticalJustification property

Justificación vertical del carácter de grupo. Especifica la alineación del objeto con respecto a la línea base. Por ejemplo, cuando el carácter de grupo está sobre el objeto, Justificación vertical de la parte superior significa que la parte superior del objeto cae sobre la línea base; cuando VerticalJustification se establece en Inferior, la parte inferior del objeto está en la línea base Predeterminado: Inferior para Posición=Superior y Superior para Posición=Inferior

```csharp
public MathTopBotPositions VerticalJustification { get; set; }
```

### Ejemplos

Ejemplo:

```csharp
[C#]
MathGroupingCharacter groupingCharacter = new MathGroupingCharacter(new MathematicalText("abc"));
groupingCharacter.VerticalJustification = MathTopBotPositions.Top;
```

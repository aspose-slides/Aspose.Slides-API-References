## IMathArray.RowSpacing property

Espaciado entre filas de una matriz Se usa solo cuando RowSpacingRule se establece en 3 Exactamente en cuyo caso la unidad de medida es puntos o Múltiple en cuyo caso la unidad de medida es media línea. Predeterminado: 0

```csharp
public uint RowSpacing { get; set; }
```

### Ejemplos

Ejemplo:

```csharp
[C#]
IMathArray mathArray = new MathArray(new MathematicalText("item1"));
mathArray.RowSpacingRule = MathRowSpacingRule.Exactly;
mathArray.RowSpacing = 10;
```

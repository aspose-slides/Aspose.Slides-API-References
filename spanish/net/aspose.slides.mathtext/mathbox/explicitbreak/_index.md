## MathBox.ExplicitBreak property

Salto explícito especifica si hay un salto de línea al comienzo del objeto de cuadro, tal que la línea se ajusta al comienzo del objeto de cuadro. Especifica el número del operador en la línea anterior de texto matemático que se utilizará como el punto de alineación para la línea actual de texto matemático valores posibles: 1..255 Predeterminado: 0 (sin corte explícito)

```csharp
public byte ExplicitBreak { get; set; }
```

### Ejemplos

Ejemplo:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.ExplicitBreak = 1;
```

## MathBox.AlignmentPoint property

Cuando es verdadero, este emulador de operador sirve como punto de alineación; es decir, puntos de alineación designados en otras ecuaciones se pueden alinear con él. Valor predeterminado: false

```csharp
public bool AlignmentPoint { get; set; }
```

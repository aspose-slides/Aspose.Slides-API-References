## IChartSeries.PieSplitBy property

Especifica cómo determinar qué puntos de datos se encuentran en el segundo gráfico circular o barra en un gráfico circular o de barra circular. Esta es la propiedad no solo de esta serie sino de todas las series del grupo de series principales - esta es la proyección de propiedad de grupo apropiada. Entonces, esta propiedad es de solo lectura. Use la propiedad ParentSeriesGroup para acceder al grupo de la serie principal. Use la propiedad de lectura/escritura ParentSeriesGroup.PieSplitBy para cambiar el valor. Solo lectura[`PieSplitType`](../../piesplittype) .

public PieSplitType PieSplitBy { get; }

### Observaciones

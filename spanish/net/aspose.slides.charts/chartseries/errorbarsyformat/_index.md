## ChartSeries.ErrorBarsYFormat property

Representa ErrorBars de serie con desviación Y.  ErrorBars con dirección Y están disponibles para series de tipo área, barra, línea, dispersión y burbuja. Para cualquier otro tipo de gráfico, esta propiedad devuelve un valor nulo (incluidos los gráficos 3D). En el caso de valores personalizados, use la recopilación de DataPoints para especificar value (con[`ErrorBarsCustomValues`](../../ichartdatapoint/errorbarscustomvalues) propiedad).

## ChartSeries.ErrorBarsXFormat property

Representa ErrorBars de serie con desviación X.  Las barras de error con dirección X están disponibles para series de tipo área, barra, dispersión y burbuja. Para cualquier otro tipo de gráfico, esta propiedad devuelve un valor nulo (incluidos los gráficos 3D). En el caso de valores personalizados, use la colección DataPoints para especificar value (con[`ErrorBarsCustomValues`](../../ichartdatapoint/errorbarscustomvalues) propiedad).

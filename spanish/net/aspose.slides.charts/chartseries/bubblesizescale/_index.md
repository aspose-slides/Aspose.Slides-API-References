## ChartSeries.BubbleSizeScale property

Especifica el factor de escala para el gráfico de burbujas (puede ser entre 0 y 300 por ciento del tamaño predeterminado). Esta es la propiedad no solo de esta serie sino de todas las series de la serie principal grupo: esta es la proyección del grupo apropiado propiedad. Por lo tanto, esta propiedad es de solo lectura. Use la propiedad ParentSeriesGroup para acceder al grupo de la serie principal. Use la propiedad de lectura/escritura ParentSeriesGroup.BubbleSizeScale para cambiar el valor.

```csharp
public int BubbleSizeScale { get; }
```

### Observaciones

Esta es la proyección de la propiedad ParentSeriesGroup.BubbleSizeScale.

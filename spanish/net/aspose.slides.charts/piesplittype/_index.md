## PieSplitType enumeration

Representa un tipo de puntos de división en el segundo gráfico circular o barra en un gráfico circular o de barra circular.

```csharp
public enum PieSplitType
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Default | `0` | Especifica que los puntos de datos se dividirán utilizando el mecanismo predeterminado para este tipo de gráfico. |
| Custom | `1` | Especifica que los puntos de datos se dividirán entre el gráfico circular y el segundo gráfico de acuerdo con los valores de División personalizada. |
| ByPercentage | `2` | Especifica que los puntos de datos se dividirán entre el gráfico circular y el segundo gráfico colocando los puntos con un porcentaje inferior al porcentaje de posición dividida en el segundo gráfico. |
| ByPos | `3` | Especifica que los puntos de datos se dividirán entre el gráfico circular y el segundo gráfico colocando la última posición dividida de los puntos de datos en el segundo gráfico. |
| ByValue | `4` | Especifica que los puntos de datos se dividirán entre el gráfico circular y el segundo gráfico colocando los puntos de datos con un valor inferior a Posición dividida en el segundo gráfico. |

## IDataLabel.IsVisible property

Falso significa que la etiqueta de datos no está visible (por lo que todos los indicadores Mostrar* (ShowValue, ...) son falsos). Solo lecturaBoolean .

```csharp
public bool IsVisible { get; }
```

### Observaciones

Si la etiqueta de datos está visible, puede ocultarla con el método Hide(). Pero si la etiqueta de datos no está visible (IsVisible es falso), puede hacer que la etiqueta de datos sea visible configurando Mostrar*-flags (ShowValue, ...) al estado verdadero.

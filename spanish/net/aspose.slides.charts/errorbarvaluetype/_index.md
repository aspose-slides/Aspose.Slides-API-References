Representa el tipo de barra de error value

public enum ErrorBarValueType

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Custom | `0` | Especifica que la longitud de las barras de error estará determinada por los elementos Más y Menos. |
| Fixed | `1` | Especifica que la longitud de las barras de error será el valor fijo determinado por Error Bar Value. |

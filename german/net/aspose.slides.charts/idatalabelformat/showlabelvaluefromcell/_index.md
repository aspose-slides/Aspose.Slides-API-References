## IDataLabelFormat.ShowLabelValueFromCell property

Repräsentiert das Anzeigeverhalten der Zellenwerte der Datenbeschriftung eines bestimmten Diagramms. True zeigt den Zellenwert an. Falsch zum Ausblenden. Lesen/SchreibenBoolean .

```csharp
public bool ShowLabelValueFromCell { get; set; }
```

### Bemerkungen

Wenn das übergeordnete Objekt dieses DataLabelFormat-Objekts eine DataLabelCollection-Sammlung von Datenbeschriftungen ist, dann ruft diese -Eigenschaft den Standardwert der ShowLabelValueFromCell-Eigenschaft für die neuen -Datenbeschriftungen in der DataLabelCollection-Sammlung ab oder legt ihn fest ShowLabelValueFromCell-Eigenschaft für alle Datenbeschriftungen in der DataLabelCollection-Sammlung (dh "DataLabels.DefaultDataLabelFormat.ShowLabelValueFromCell = val;" bewirkt, dass alle DataLabels[i].ShowLabelValueFromCell gleich val sind).

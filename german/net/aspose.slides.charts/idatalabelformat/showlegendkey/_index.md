## IDataLabelFormat.ShowLegendKey property

Repräsentiert das Anzeigeverhalten der Datenbeschriftungslegende eines bestimmten Diagramms. True, wenn der Legendenschlüssel der Datenbeschriftung sichtbar ist. Lesen/SchreibenBoolean .

```csharp
public bool ShowLegendKey { get; set; }
```

### Bemerkungen

Wenn das übergeordnete Objekt dieses DataLabelFormat-Objekts eine DataLabelCollection-Sammlung von Datenbeschriftungen ist, dann ruft diese -Eigenschaft den Standardwert der ShowLegendKey-Eigenschaft für die neuen -Datenbeschriftungen in der DataLabelCollection-Sammlung ab oder legt ihn fest ShowLegendKey-Eigenschaft für alle Datenbeschriftungen in der DataLabelCollection-Sammlung (dh "DataLabels.DefaultDataLabelFormat.ShowLegendKey = val;" bewirkt, dass alle DataLabels[i].ShowLegendKey gleich val sind).

## IChartDataWorkbook.CalculateFormulas method

Berechnet alle Formeln in der Arbeitsmappe und aktualisiert die entsprechenden Zellenwerte.

```csharp
public void CalculateFormulas()
```

### Ausnahmen

| Ausnahme | Bedingung |
| --- | --- |
| [CellCircularReferenceException](../../../aspose.slides.spreadsheet/cellcircularreferenceexception) | Die Arbeitsmappe enthält Formeln mit einem Zirkelbezug. |
| [CellUnsupportedDataException](../../../aspose.slides.spreadsheet/cellunsupporteddataexception) | Zelldaten werden nicht unterstützt. |

### Beispiele

Beispiel zeigt, wie man der Zelle eine Formel zuweist und einen Wert berechnet. Der Wert der Zelle "B4" wird auf 5. gesetzt.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Pie, 100, 100, 300, 400);
    IChartDataWorkbook wb = chart.ChartData.ChartDataWorkbook;
    wb.GetCell(0, "B2", 2);	
	wb.GetCell(0, "B3", 3);	
    wb.GetCell(0, "B4").Formula = "B2+B3";
    wb.CalculateFormulas();
    ...
}
```

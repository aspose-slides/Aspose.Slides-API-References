## IDataLabel.Hide method

Datenbeschriftung unsichtbar machen, indem alle Show*-Flags (ShowValue, ...) auf false gesetzt werden. IsVisible ist danach false.


### Bemerkungen

Wenn das Datenlabel nicht sichtbar ist (IsVisible ist false), können Sie das Datenlabel sichtbar machen, indem Sie die Show*-Flags (ShowValue, ...) auf true setzen.

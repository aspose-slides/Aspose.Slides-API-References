## ChartSeries.FirstSliceAngle property

Gibt den Winkel des ersten Torten- oder Ringdiagrammsegments an, in Grad (im Uhrzeigersinn von oben, von 0 bis 360 Grad). Dies ist nicht nur die Eigenschaft dieser Serie, sondern aller Serien der übergeordneten Serie Gruppe - das ist Projektion angemessenen Gruppeneigentums. Daher ist diese Eigenschaft schreibgeschützt. Verwenden Sie die Eigenschaft ParentSeriesGroup für den Zugriff auf die übergeordnete Seriengruppe. Verwenden Sie die Eigenschaft ParentSeriesGroup.FirstSliceAngle mit Lese-/Schreibzugriff, um den Wert zu ändern. Nur lesenUInt16 .

```csharp
public ushort FirstSliceAngle { get; }
```

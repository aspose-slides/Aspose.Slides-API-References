## DataLabelFormat.ShowValue property

Repräsentiert das Anzeigeverhalten des Prozentwerts der Datenbeschriftung eines bestimmten Diagramms. True zeigt den Prozentwert an. Falsch zum Ausblenden. Lesen/SchreibenBoolean .

```csharp
public bool ShowValue { get; set; }
```

### Bemerkungen

Wenn das übergeordnete Objekt dieses DataLabelFormat-Objekts eine DataLabelCollection-Sammlung von Datenbeschriftungen ist, dann ruft diese -Eigenschaft den Standardwert der ShowValue-Eigenschaft für die neuen -Datenbeschriftungen in der DataLabelCollection-Sammlung ab oder legt ihn fest ShowValue-Eigenschaft für alle Datenbeschriftungen in der DataLabelCollection-Sammlung (dh „DataLabels.DefaultDataLabelFormat.ShowValue = val;“ bewirkt, dass alle DataLabels[i].ShowValue gleich val sind).

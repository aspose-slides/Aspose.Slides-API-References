## DataLabelFormat.ShowLeaderLines property

Repräsentiert das Anzeigeverhalten der Führungslinien der Datenbeschriftung eines bestimmten Diagramms. True zeigt die Führungslinien an. Falsch zum Ausblenden. Lesen/SchreibenBoolean .

```csharp
public bool ShowLeaderLines { get; set; }
```

### Bemerkungen

Wenn das übergeordnete Objekt dieses DataLabelFormat-Objekts eine DataLabelCollection-Sammlung von Datenbeschriftungen ist, dann ruft diese -Eigenschaft den Standardwert der ShowLeaderLines-Eigenschaft für die neuen -Datenbeschriftungen in der DataLabelCollection-Sammlung ab oder legt ihn fest ShowLeaderLines-Eigenschaft für alle Datenbeschriftungen in der DataLabelCollection-Sammlung (dh "DataLabels.DefaultDataLabelFormat.ShowLeaderLines = val;" bewirkt, dass alle DataLabels[i].ShowLeaderLines gleich val sind).

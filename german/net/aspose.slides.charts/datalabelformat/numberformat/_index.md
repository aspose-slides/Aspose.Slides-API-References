## DataLabelFormat.NumberFormat property

Repräsentiert die Formatzeichenfolge für das DataLabels-Objekt. Lesen/SchreibenString .

```csharp
public string NumberFormat { get; set; }
```

### Bemerkungen

Wenn das übergeordnete Objekt dieses DataLabelFormat-Objekts eine DataLabelCollection-Sammlung von Datenbeschriftungen ist, dann ruft diese -Eigenschaft den Standardwert der NumberFormat-Eigenschaft für die neuen -Datenbeschriftungen in der DataLabelCollection-Sammlung ab oder legt ihn fest Wert wird auch für die NumberFormat-Eigenschaft für alle Datenbeschriftungen in der DataLabelCollection-Sammlung festgelegt (dh „DataLabels.DefaultDataLabelFormat.NumberFormat = val;“ bewirkt, dass alle DataLabels[i].NumberFormat gleich val sind).

### Beispiele

```csharp
[C#]
series.Labels.DefaultDataLabelFormat.ShowValue = true;
series.Labels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = false;
series.Labels.DefaultDataLabelFormat.NumberFormat = "0.0%";
```

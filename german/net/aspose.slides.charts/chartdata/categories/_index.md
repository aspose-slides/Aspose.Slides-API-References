## ChartData.Categories property

Ruft die primären Kategorien ab (oder sowohl primäre als auch sekundäre Kategorien , wenn[`UseSecondaryCategories`](../usesecondarycategories) Eigenschaft ist falsch). Schreibgeschützt[`IChartCategoryCollection`](../../ichartcategorycollection) .

```csharp
public IChartCategoryCollection Categories { get; }
```

### Bemerkungen

Wenn[`UseSecondaryCategories`](../usesecondarycategories) Eigenschaft ist dann falsch[`SecondaryCategories`](../secondarycategories) Eigenschaft gibt null und Daten darin zurück`Categories` Eigenschaft wird sowohl für primäre als auch für sekundäre Serien verwendet. Wenn[`UseSecondaryCategories`](../usesecondarycategories) Eigenschaft ist wahr, dann Daten in[`SecondaryCategories`](../secondarycategories) Die Eigenschaft wird für sekundäre Serien und Daten darin verwendet`Categories` Eigenschaft wird für Primärserien verwendet.

### Beispiele

Beispiel. Welche Kategorien beziehen sich auf Serien – ChartData.Categories oder ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // zugehörige Kategorien sind series.Chart.ChartData.SecondaryCategories
}
else
{
    // zugehörige Kategorien sind series.Chart.ChartData.Categories
}
```

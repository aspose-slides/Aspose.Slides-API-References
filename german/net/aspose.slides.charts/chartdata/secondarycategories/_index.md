## ChartData.SecondaryCategories property

Ruft die sekundären Kategorien ab, wenn[`UseSecondaryCategories`](../usesecondarycategories) Eigenschaft ist wahr. Schreibgeschützt[`IChartCategoryCollection`](../../ichartcategorycollection) .

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### Bemerkungen

Wenn[`UseSecondaryCategories`](../usesecondarycategories) Eigentum ist dann falsch`SecondaryCategories` -Eigenschaft gibt null zurück und Daten in[`Categories`](../categories) Eigenschaft wird sowohl für primäre als auch für sekundäre Serien verwendet. Wenn[`UseSecondaryCategories`](../usesecondarycategories) Eigenschaft ist wahr, dann Daten in this`SecondaryCategories` Die Eigenschaft wird für sekundäre Serien und Daten in verwendet[`Categories`](../categories) Eigenschaft wird für primäre Serien verwendet.

### Beispiele

Beispiel. Welche Kategorien beziehen sich auf Serien – ChartData.Categories oder ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // zugehörige Kategorien sind series.Chart.ChartData.SecondaryCategories
}
else
{
    // zugehörige Kategorien sind series.Chart.ChartData.Categories
}
```

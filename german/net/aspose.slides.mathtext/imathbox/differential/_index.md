## IMathBox.Differential property

Differential. Wenn wahr, fungiert das Kästchen als Differential (z. B. 𝑑𝑥 in einem Integranden) und erhält den entsprechenden horizontalen Abstand für das mathematische Differential. Standard: false

```csharp
public bool Differential { get; set; }
```

### Beispiele

Beispiel:

```csharp
[C#]
IMathBox differential = new MathematicalText("dx").ToBox();
differential.Differential = true;
```

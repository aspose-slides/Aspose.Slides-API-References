## IAlphaCeiling interface

Stellt einen Alpha-Decke-Effekt dar. Alpha-Werte (Deckkraft) größer als Null werden auf 100 % geändert. Mit anderen Worten, alles, was teilweise undurchsichtig ist, wird vollständig undurchsichtig.

## PresentationInfo.CheckWriteProtection method

Überprüft, ob ein zu änderndes Passwort für eine schreibgeschützte Präsentation korrekt ist.

```csharp
public bool CheckWriteProtection(string password)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| password | String | Das zu prüfende Passwort. |

### Rückgabewert

True, wenn die Präsentation schreibgeschützt und das Passwort korrekt ist. Andernfalls falsch.

### Ausnahmen

| Ausnahme | Bedingung |
| --- | --- |
| InvalidOperationException |  |

### Bemerkungen

1. Sie sollten das überprüfen[`IsWriteProtected`](../iswriteprotected) Eigenschaft, bevor diese Methode aufgerufen wird. 2. Wenn das Passwort null oder leer ist, gibt diese Methode false zurück.

### Beispiele

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
if (info.IsWriteProtected == NullableBool.True)
{
    bool isWriteProtectedByPassword = info.CheckWriteProtection("my_password");
}
```

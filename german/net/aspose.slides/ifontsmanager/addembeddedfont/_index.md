## AddEmbeddedFont(IFontData, EmbedFontCharacters) {#addembeddedfont}

Fügt die eingebettete Schriftart hinzu. Denken Sie beim Kopieren von Schriftarten daran, dass die meisten Schriftarten urheberrechtlich geschützt sind. Suchen Sie zunächst die Lizenz von einer Schriftart und überprüfen Sie, ob sie frei auf einen anderen Computer übertragen werden kann.Eine ArgumentException kann ausgelöst werden, wenn Schriftartdaten null sind oder diese Schriftart bereits eingebettet ist

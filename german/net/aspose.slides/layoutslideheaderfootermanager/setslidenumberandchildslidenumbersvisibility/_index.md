## LayoutSlideHeaderFooterManager.SetSlideNumberAndChildSlideNumbersVisibility method

Ändert den Platzhalter für die Seitenzahl der Layoutfolie und die Sichtbarkeit aller untergeordneten Platzhalter für die Seitenzahl. Untergeordnete Platzhalter bedeuten, dass Platzhalter auf den abhängigen Folien enthalten sind. Abhängige Folien verwenden und hängen von der Layoutfolie ab.

```csharp
public void SetSlideNumberAndChildSlideNumbersVisibility(bool isVisible)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| isVisible | Boolean | true - macht Platzhalter für Seitenzahlen sichtbar, andernfalls - blendet sie aus. |

## SplitterBarStateType enumeration

Gibt den Zustand an, in dem der Teilerbalken angezeigt werden soll. Der Teilerbalken trennt einen primären und sekundären Bereich innerhalb eines Anzeigebereichs.

```csharp
public enum SplitterBarStateType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Minimized | `0` | Die primäre Region belegt den geringsten von der Anwendung zugelassenen Anzeigebereich. |
| Restored | `1` | Die primäre Region hat eine bestimmte Zwischengröße. |
| Maximized | `2` | Die primäre Region nimmt den größten Teil des Anzeigebereichs ein, der von der Anwendung zugelassen wird. |

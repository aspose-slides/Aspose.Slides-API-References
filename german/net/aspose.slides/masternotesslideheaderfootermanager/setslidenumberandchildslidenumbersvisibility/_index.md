## MasterNotesSlideHeaderFooterManager.SetSlideNumberAndChildSlideNumbersVisibility method

Ändert die Seitenzahl-Platzhalter der Master-Folie und die Sichtbarkeit aller untergeordneten Platzhalter für Seitenzahlen. Untergeordnete Platzhalter bedeuten, dass Platzhalter auf abhängigen Notizen-Folien enthalten sind. Abhängige Notizen-Folien verwenden und hängen von der Master-Notizen-Folie ab.

```csharp
public void SetSlideNumberAndChildSlideNumbersVisibility(bool isVisible)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| isVisible | Boolean | true - macht Platzhalter für Seitenzahlen sichtbar, andernfalls - blendet sie aus. |

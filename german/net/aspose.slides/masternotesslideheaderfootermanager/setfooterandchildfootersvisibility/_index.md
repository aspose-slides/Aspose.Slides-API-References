## MasterNotesSlideHeaderFooterManager.SetFooterAndChildFootersVisibility method

Ändert die Fußzeilen-Platzhalter der Master-Folie und die Sichtbarkeit aller untergeordneten Fußzeilen-Platzhalter. Untergeordnete Platzhalter bedeuten, dass Platzhalter auf abhängigen Notizen-Folien enthalten sind. Abhängige Notizen-Folien verwenden und hängen von der Master-Notizen-Folie ab.

```csharp
public void SetFooterAndChildFootersVisibility(bool isVisible)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| isVisible | Boolean | true - macht Platzhalter in der Fußzeile sichtbar, andernfalls - blendet sie aus. |

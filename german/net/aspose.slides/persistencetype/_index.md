## PersistenceType enumeration

Gibt die Methode an, die zum Speichern von Eigenschaften des ActiveX-Steuerelements verwendet wird.

```csharp
public enum PersistenceType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| NotDefined | `-1` | Persistenz-ID nicht angegeben. |
| PersistPropertyBag | `0` | Gibt an, dass das ActiveX-Steuerelement mithilfe der auf Eigenschaftenbehältern basierenden Persistenz beibehalten wird. Property-Bag-basierte Persistenz speichert ein ActiveX-Steuerelement mithilfe einer Sammlung von Namen und Wertepaaren, die die Daten angeben, die vom ActiveX-Steuerelement beibehalten werden. |
| PersistStream | `1` | Gibt an, dass das ActiveX-Steuerelement mithilfe einer streambasierten Persistenz beibehalten wird, die die Initialisierung des ActiveX-Steuerelements in einen Standardzustand nicht unterstützt. |

| PersistStorage | `3` | Gibt an, dass das ActiveX-Steuerelement mit speicherbasierter Persistenz beibehalten wird. |

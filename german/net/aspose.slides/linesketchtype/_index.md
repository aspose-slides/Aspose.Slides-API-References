## LineSketchType enumeration

Stellt dar, welcher Skizzentyp oder Effekt einer Form zugewiesen wurde.

```csharp
public enum LineSketchType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| NotDefined | `-1` | Gibt an, dass ein Formskizzeneffekt nicht definiert ist. |
| None | `0` | Gibt an, dass eine Form keinen Skizzeneffekt hat. Dies entspricht einer leeren Eigenschaft. |

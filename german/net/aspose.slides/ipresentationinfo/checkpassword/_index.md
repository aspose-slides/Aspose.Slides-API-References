## IPresentationInfo.CheckPassword method

Überprüft, ob ein Passwort für eine mit offenem Passwort geschützte Präsentation korrekt ist.

```csharp
public bool CheckPassword(string password)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| password | String | Das zu prüfende Passwort. |

### Rückgabewert

Wahr, wenn die Präsentation mit einem offenen Passwort geschützt ist und das Passwort richtig ist, andernfalls falsch.

### Bemerkungen

Wenn das Passwort null oder leer ist, gibt diese Methode false zurück.

### Beispiele

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo("pres.pptx");
bool isPasswordCorrect = info.CheckPassword("my_password");
```

## ToArray(int, int) {#toarray_1}

Erstellt ein Array mit allen FallBack-Schriftarten aus dem angegebenen Bereich in der Liste und gibt es zurück.

```csharp
public string[] ToArray(int startIndex, int count)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| startIndex | Int32 | Ein Index einer ersten Schriftart, die hinzugefügt werden soll. |
| count | Int32 | Eine Reihe von Schriftarten zum Hinzufügen. |

### Rückgabewert

Anordnung vonString

### Beispiele

```csharp
[C#]
// Erstellen Sie eine Regel, die eine Liste von Schriftarten enthält.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

//Erhalte die letzten beiden Schriftartnamen als Array
string[] fontNames = newRule.ToArray(2,2);
```

Das nächste Beispiel zeigt die Eigenschaft using Persistence, um zu prüfen, ob Eigenschaften von ActiveX-Objekten als XML-basierte ActiveX-Eigenschaften geändert werden können:

{
    control.Properties["Value"] = value;

## PresentationLockingBehavior enumeration

Stellt das Verhalten bezüglich der Behandlung dar[`IPresentation`](../ipresentation) Quelle (Datei oder Stream ) beim Laden und Arbeiten mit einer Instanz von[`IPresentation`](../ipresentation) .

```csharp
public enum PresentationLockingBehavior
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| LoadAndRelease | `0` | Die Quelle wird nur für eine Zeit von gesperrt[`IPresentation`](../ipresentation) Konstruktorausführung.  Wenn[`IsTemporaryFilesAllowed`](../iblobmanagementoptions/istemporaryfilesallowed) auf false gesetzt ist, werden alle BLOBs in den Speicher geladen. Andernfalls könnten andere Mittel wie temporäre Dateien verwendet werden. Dieses Verhalten ist langsamer alsKeepLocked , und ob es möglich ist, den Besitz der Quelle an zu übergeben[`IPresentation`](../ipresentation) , es wird empfohlen, zu verwendenKeepLocked . |
| KeepLocked | `1` | Die Quelle wird lebenslang gesperrt[`IPresentation`](../ipresentation) Beispiel, bis es entsorgt wird. [`IsTemporaryFilesAllowed`](../iblobmanagementoptions/istemporaryfilesallowed) Dies muss auf true gesetzt werden, um dieses Verhalten zu verwenden, andernfalls wird eine Ausnahme ausgelöst. Dieses Verhalten wird empfohlen, es ist schneller und verbraucht weniger Speicher alsLoadAndRelease . |

### Bemerkungen

Die Quelle ist der Parameter, der an übergeben wird[`IPresentation`](../ipresentation) Konstrukteur. Im -Beispiel unten ist die Quelle die Datei „pres.pptx“:

```csharp
LoadOptions loadOptions = new LoadOptions { 
  BlobManagementOptions = { PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked } };
using (IPresentation pres = new Presentation("pres.pptx", loadOptions)) { }
```

In diesem Beispiel wird die Quelle ("pres.pptx"-Datei) für a gesperrt[`IPresentation`](../ipresentation) Instanzlebensdauer, dh kann nicht durch den anderen Prozess geändert oder gelöscht werden.

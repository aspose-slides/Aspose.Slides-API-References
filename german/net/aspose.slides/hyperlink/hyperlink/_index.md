## Hyperlink(ISlide) {#constructor_1}

Erstellt eine Instanz eines Hyperlinks, der auf eine bestimmte Folie verweist. Hinweis: Der erstellte Hyperlink sollte einem Objekt aus derselben Präsentation zugewiesen werden, andernfalls wird der Link als NoAction gespeichert.

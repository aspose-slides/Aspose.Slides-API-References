## GifOptions.DefaultDelay property

Ruft die Standardverzögerungszeit [ms] ab oder setzt sie. Dieser Wert wird verwendet, wenn[`AdvanceAfterTime`](../../../aspose.slides/islideshowtransition/advanceaftertime) ist nicht festgelegt. Der Standardwert ist 1000.

```csharp
public int DefaultDelay { get; set; }
```

### Beispiele

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
	pres.Save("pres.gif", SaveFormat.Gif, new GifOptions { DefaultDelay = 2000 });
}
```

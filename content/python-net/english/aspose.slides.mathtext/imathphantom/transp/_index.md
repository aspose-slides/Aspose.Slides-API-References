## transp property
Gets or sets a value indicating whether the phantom is transparent 
            for class-based spacing rules.


### Remarks

When `true`, operators and symbols inside the phantom still affect 
            mathematical spacing around the phantom (as if visible). When `false`, 
            class-based spacing is ignored. Corresponds to the OMML attribute `m:transp`.

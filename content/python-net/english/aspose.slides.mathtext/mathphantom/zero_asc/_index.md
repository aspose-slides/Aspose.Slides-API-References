## zero_asc property
Gets or sets a value indicating whether the ascent (height above baseline) 
            of the base element should be treated as zero.


### Remarks

When `true`, the phantom does not raise the baseline of the surrounding 
            math line. Corresponds to the OMML attribute `m:zeroAsc`.

## __init__ {#imathelement}
Initializes a new instance of the [`MathPhantom`](/slides/python-net/aspose.slides.mathtext/mathphantom) class 
            using the specified base math element.


```python
def __init__(self, element):
    ...
```


| Parameter | Type | Description |
| :- | :- | :- |
| element | [`IMathElement`](/slides/python-net/aspose.slides.mathtext/imathelement) | The base [`IMathElement`](/slides/python-net/aspose.slides.mathtext/imathelement) whose visibility and layout will be controlled <br/><br/>            by the phantom. This element defines the content that may be hidden or shown, <br/><br/>            while still affecting the geometric alignment of the surrounding math. |

### Remarks

The phantom element is used to reserve or suppress the visual space of its base 
            expression without necessarily displaying it. It corresponds to the OMML element 
            `<m:phant>`.

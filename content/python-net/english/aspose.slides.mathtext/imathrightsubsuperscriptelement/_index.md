## IMathRightSubSuperscriptElement class

Specifies the Sub-Superscript object, which consists of a base 
            and a subscript and superscript placed to the right of the base.

The IMathRightSubSuperscriptElement type exposes the following members:

## Properties

| Property | Description |
| :- | :- |
| [base](/slides/python-net/aspose.slides.mathtext/base) | Base argument |
| [subscript](/slides/python-net/aspose.slides.mathtext/subscript) | Subscript argument |
| [superscript](/slides/python-net/aspose.slides.mathtext/superscript) | Superscript argument |
| [align_scripts](/slides/python-net/aspose.slides.mathtext/align_scripts) | Specifies the alignment of subscript/superscript. <br/>            When true, subscript and superscript are aligned horizontally to each other.<br/>            When false, they are kerned to the shape of the base.<br/>            Default value is false. |

## index_of {#IChartCategory}
Searches for the specified [`ChartCategory`](/slides/python-net/aspose.slides.charts/chartcategory) and returns the zero-based index of the first occurrence within the entire Collection.

### Returns

The zero-based index of the first occurrence of value within the entire CollectionBase, if found; otherwise, -1.

IErrorBarsFormat class

Represents error bars of chart series. ErrorBars custom values are in IChartDataPointCollection
            (in 
[`IChartDataPoint.error_bars_custom_values`](/slides/python-net/aspose.slides.charts/ichartdatapoint#error_bars_custom_values) property).

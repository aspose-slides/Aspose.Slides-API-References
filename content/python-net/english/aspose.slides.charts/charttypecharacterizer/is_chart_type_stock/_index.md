## is_chart_type_stock {#charttype}
Return true if chartType is one of Stock subtypes.
            Subtypes set corresponds to the appropriate set in PowerPoint (see "Change Chart Type" dialog in PowerPoint):
            [`ChartType.HIGH_LOW_CLOSE`](/slides/python-net/aspose.slides.charts/charttype/HIGH_LOW_CLOSE),
            [`ChartType.OPEN_HIGH_LOW_CLOSE`](/slides/python-net/aspose.slides.charts/charttype/OPEN_HIGH_LOW_CLOSE),
            [`ChartType.VOLUME_HIGH_LOW_CLOSE`](/slides/python-net/aspose.slides.charts/charttype/VOLUME_HIGH_LOW_CLOSE),
            [`ChartType.VOLUME_OPEN_HIGH_LOW_CLOSE`](/slides/python-net/aspose.slides.charts/charttype/VOLUME_OPEN_HIGH_LOW_CLOSE).

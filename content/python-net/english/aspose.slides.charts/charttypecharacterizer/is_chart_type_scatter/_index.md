## is_chart_type_scatter {#charttype}
Return true if chartType is one of Scatter subtypes.
            Subtypes set corresponds to the appropriate set in PowerPoint (see "Change Chart Type" dialog in PowerPoint):
            [`ChartType.SCATTER_WITH_MARKERS`](/slides/python-net/aspose.slides.charts/charttype#SCATTER_WITH_MARKERS),
            [`ChartType.SCATTER_WITH_SMOOTH_LINES`](/slides/python-net/aspose.slides.charts/charttype#SCATTER_WITH_SMOOTH_LINES),
            [`ChartType.SCATTER_WITH_SMOOTH_LINES_AND_MARKERS`](/slides/python-net/aspose.slides.charts/charttype#SCATTER_WITH_SMOOTH_LINES_AND_MARKERS),
            [`ChartType.SCATTER_WITH_STRAIGHT_LINES`](/slides/python-net/aspose.slides.charts/charttype#SCATTER_WITH_STRAIGHT_LINES),
            [`ChartType.SCATTER_WITH_STRAIGHT_LINES_AND_MARKERS`](/slides/python-net/aspose.slides.charts/charttype#SCATTER_WITH_STRAIGHT_LINES_AND_MARKERS).

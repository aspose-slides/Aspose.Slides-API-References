## is_chart_type_area {#charttype}
Return true if chartType is one of Area subtypes.
            Subtypes set corresponds to the appropriate set in PowerPoint (see "Change Chart Type" dialog in PowerPoint):
            [`ChartType.AREA`](/slides/python-net/aspose.slides.charts/charttype/AREA),
            [`ChartType.PERCENTS_STACKED_AREA`](/slides/python-net/aspose.slides.charts/charttype/PERCENTS_STACKED_AREA),
            [`ChartType.PERCENTS_STACKED_AREA_3D`](/slides/python-net/aspose.slides.charts/charttype/PERCENTS_STACKED_AREA_3D),
            [`ChartType.STACKED_AREA`](/slides/python-net/aspose.slides.charts/charttype/STACKED_AREA),
            [`ChartType.STACKED_AREA_3D`](/slides/python-net/aspose.slides.charts/charttype/STACKED_AREA_3D),
            [`ChartType.AREA_3D`](/slides/python-net/aspose.slides.charts/charttype/AREA_3D).

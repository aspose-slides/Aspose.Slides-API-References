## IComment class

Represents a comment on a slide.

The IComment type exposes the following members:

## Properties

| Property | Description |
| :- | :- |
| [`text`](/slides/python-net/aspose.slides/icomment/text/) | Returns or sets the plain text of a slide comment.<br/>            Read/write **string**. |
| [`created_time`](/slides/python-net/aspose.slides/icomment/created_time/) | Returns or sets the time of a comment creation.<br/>            Setting this property to **System.DateTime** means no comment time is set.<br/>            Read/write **System.DateTime**. |
| [`slide`](/slides/python-net/aspose.slides/icomment/slide/) | Returns or sets the parent slide of a comment.<br/>            Read-only [`ISlide`](/slides/python-net/aspose.slides/islide). |
| [`author`](/slides/python-net/aspose.slides/icomment/author/) | Returns the author of a comment.<br/>            Read-only [`ICommentAuthor`](/slides/python-net/aspose.slides/icommentauthor). |
| [`position`](/slides/python-net/aspose.slides/icomment/position/) | Returns or sets the position of a comment on a slide.<br/>            Read/write **aspose.pydrawing.PointF**. |
| [`parent_comment`](/slides/python-net/aspose.slides/icomment/parent_comment/) | Gets or sets parent comment.<br/>            Read/write [`IComment`](/slides/python-net/aspose.slides/icomment). |

## Methods

| Method | Description |
| :- | :- |
| [`remove`](/slides/python-net/aspose.slides/icomment/remove/#) | Removes comment and all its replies from the parent collection. |

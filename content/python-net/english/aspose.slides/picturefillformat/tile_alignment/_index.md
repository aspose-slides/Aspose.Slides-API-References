## tile_alignment property
Returns or sets how the texture is aligned within the shape. This setting controls the starting point of the texture pattern and how it repeats across the shape.
             Read/write [`RectangleAlignment`](/slides/python-net/aspose.slides/rectanglealignment).


### Remarks

Default is [`RectangleAlignment.TOP_LEFT`](/slides/python-net/aspose.slides/rectanglealignment/TOP_LEFT).

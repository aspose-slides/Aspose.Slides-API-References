## paragraph_format property
Returns the formatting object for this paragraph.
            Read-only [`IParagraphFormat`](/slides/python-net/aspose.slides/iparagraphformat).


### Remarks

The formatting object contains the formatting parameters defined for the current paragraph only, inherited data is not applied.


In order to get the effective values including inherited ones use the [`ParagraphFormat.get_effective`](/slides/python-net/aspose.slides/paragraphformat/get_effective) method.

Comment class

| Property | Description |
| :- | :- |

| [created_time](/slides/python-net/aspose.slides/comment/created_time/) | Returns or sets the time of a comment creation.<br/>            Setting this property to :py:attr:`System.DateTime.MinValue` means no comment time is set.<br/>            Read/write :py:class:`System.DateTime`. |

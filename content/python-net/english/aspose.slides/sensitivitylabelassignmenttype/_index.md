## SensitivityLabelAssignmentType enumeration

Represents the assignment method for the sensitivity label.

The SensitivityLabelAssignmentType type exposes the following members:

## Fields

| Field | Description |
| :- | :- |
| STANDARD | Use for any sensitivity label that was not directly applied by the user. <br/>            This includes any default labels, automatically applied labels. |
| PRIVILEGED | Use for any sensitivity label that was directly applied by the user. <br/>            This includes any manually applied sensitivity labels as well as recommended<br/>            or mandatory labeling or any feature where the user decides which sensitivity label to apply. |

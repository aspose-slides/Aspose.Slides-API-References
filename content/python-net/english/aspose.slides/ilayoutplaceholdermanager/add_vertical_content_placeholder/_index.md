## add_vertical_content_placeholder {#float-float-float-float}
Adds a new placeholder shape to the layout slide to hold content, such as a picture, table, media 
            or text in a vertical direction.

### Returns

Created [`IAutoShape`](/slides/python-net/aspose.slides/iautoshape) with a Content (Vertical) placeholder.

| y | **float** | The Y coordinate of the new placeholder shape. |
| width | **float** | The width of the new placeholder shape. |

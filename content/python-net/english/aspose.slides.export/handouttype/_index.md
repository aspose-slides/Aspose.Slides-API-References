## HandoutType enumeration

Specifies how many slides and in what sequence will be placed on the page.

## Members
| Member name | Description |
| :- | :- |
|HANDOUTS1|One slide per page.|
|HANDOUTS2|Two slides per page.|
|HANDOUTS3|Three slides per page.|
|HANDOUTS_4_HORIZONTAL|Four slides per page in a horizontal sequence.|
|HANDOUTS_4_VERTICAL|Four slides per page in a vertical sequence.|
|HANDOUTS_6_HORIZONTAL|Six slides per page in a horizontal sequence.|
|HANDOUTS_6_VERTICAL|Six slides per page in a vertical sequence.|
|HANDOUTS_9_HORIZONTAL|Nine slides per page in a horizontal sequence.|
|HANDOUTS_9_VERTICAL|Nine slides per page in a vertical sequence.|

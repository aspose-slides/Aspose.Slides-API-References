## text_compression property
Specifies compression type to be used for all textual content in the document.
            Read/write [`PdfTextCompression`](/slides/python-net/aspose.slides.export/pdftextcompression).


### Remarks

Default is [`PdfTextCompression.FLATE`](/slides/python-net/aspose.slides.export/pdftextcompression#FLATE).

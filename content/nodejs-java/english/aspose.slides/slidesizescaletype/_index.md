## SlideSizeScaleType class

 Represents the scale type of slide content.
 

## Constants

| Name | Value | Description |
| --- | --- | --- |
| DoNotScale | 0 | Do not scale. |
| EnsureFit | 1 | Scale to ensure fit. |
| Maximize | 2 | Maximize size of content. |

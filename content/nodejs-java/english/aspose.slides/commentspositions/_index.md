## CommentsPositions class

 Represents the rule to render comments into exported document
 

## Constants

| Name | Value | Description |
| --- | --- | --- |
[None](#None) | 0 | Specifies that comments should not be displayed at all. |
[Bottom](#Bottom) | 1 | Specifies that comments should be displayed at the bottom of the page. |
[Right](#Right) | 2 | Specifies that comments should be displayed to the right of the page. |

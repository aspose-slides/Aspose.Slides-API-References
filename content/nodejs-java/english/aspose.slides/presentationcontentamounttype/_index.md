## PresentationContentAmountType class

 Specifies the amount of content included in the generated presentation, influencing both the number of slides and the level of detail per slide.
 

## Constants

| Name | Value | Description |
| --- | --- | --- |
[Brief](#Brief) | 0 | Short and basic content. |
[Medium](#Medium) | 1 | A moderate amount of content. |
[Detailed](#Detailed) | 2 | The most detailed option with extensive text. |

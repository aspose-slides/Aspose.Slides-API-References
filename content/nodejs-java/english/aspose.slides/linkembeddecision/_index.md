## LinkEmbedDecision class

 Determines how object will be processed during saving.
 

## Constants

| Name | Value | Description |
| --- | --- | --- |
[Link](#Link) | 0 | Object will be stored externally, referrenced by URL |
[Embed](#Embed) | 1 | Object should be embedded to a generated file if possible. If embedding is imposible, GetUrl will be called and, depending on result, object will be referrenced by URL or ignored. |
[Ignore](#Ignore) | 2 | Object will be ignored. |


---


### Link {#Link}
```
Object will be stored externally, referrenced by URL
```

---

### Embed {#Embed}
```
Object should be embedded to a generated file if possible. If embedding is imposible, GetUrl will be called and, depending on result, object will be referrenced by URL or ignored.
```

---

### Ignore {#Ignore}
```
Object will be ignored.
```

## Conformance class

 Specifies the conformance class to which the PresentationML document conforms.

[Iso29500_2008_Transitional](#Iso29500_2008_Transitional) | 1 | Specifies that the document conforms to the ISO/IEC 29500:2008 Transitional conformance class. |
[Iso29500_2008_Strict](#Iso29500_2008_Strict) | 2 | Specifies that the document conforms to the ISO/IEC 29500:2008 Strict conformance class. |

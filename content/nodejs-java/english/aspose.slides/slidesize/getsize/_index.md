## getSize()  function

 Returns or sets the size in points.
 Read/write  java.awt.geom.Dimension2D.
 Assigning any value will reset ( #getType) property to  SlideSizeType#Custom and set ( #getOrientation/ #setOrientation(int)).

### Result
Dimension2D

## getZOrderPosition()  function

 Returns the position of a shape in the z-order.
 Shapes[0] returns the shape at the back of the z-order,
 and Shapes[Shapes.Count - 1] returns the shape at the front of the z-order.
 Read-only  int.
 

### Result
int

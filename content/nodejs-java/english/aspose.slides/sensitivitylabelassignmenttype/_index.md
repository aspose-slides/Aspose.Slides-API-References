## SensitivityLabelAssignmentType class

 Represents the assignment method for the sensitivity label.
 

## Constants

| Name | Value | Description |
| --- | --- | --- |
[Standard](#Standard) | 0 | Use for any sensitivity label that was not directly applied by the user. This includes any default labels, automatically applied labels. |
[Privileged](#Privileged) | 1 | Use for any sensitivity label that was directly applied by the user. This includes any manually applied sensitivity labels as well as recommended or mandatory labeling or any feature where the user decides which sensitivity label to apply. |

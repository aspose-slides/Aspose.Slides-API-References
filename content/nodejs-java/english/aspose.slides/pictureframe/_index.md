## PictureFrame class

 Represents a frame with a picture inside.
 

## Functions

| Name | Description |
| --- | --- |
| [getPictureFormat](getpictureformat)() | Returns the PictureFillFormat object for a picture frame. Read-only IPictureFillFormat. |
| [getPictureFrameLock](getpictureframelock)() | Returns shape's locks. Read-only IPictureFrameLock. |
| [getRelativeScaleHeight](getrelativescaleheight)() | Returns or sets the scale of height(relative to original picture size) of the picture frame. Value 1.0 corresponds to 100%. Read/write float. |
| [getRelativeScaleWidth](getrelativescalewidth)() | Returns or sets the scale of width (relative to original picture size) of the picture frame. Value 1.0 corresponds to 100%. Read/write float. |
| [getShapeType](getshapetype)() | Returns or sets the AutoShape type for a PictureFrame. Read/write ShapeType. |
| [setRelativeScaleHeight](setrelativescaleheight)(float) | Returns or sets the scale of height(relative to original picture size) of the picture frame. Value 1.0 corresponds to 100%. Read/write float. |
| [setRelativeScaleWidth](setrelativescalewidth)(float) | Returns or sets the scale of width (relative to original picture size) of the picture frame. Value 1.0 corresponds to 100%. Read/write float. |

Read/write ShapeType. |

## AnimateTextType class

 Represents the animate text type of an animation effect.
 

## Constants

| Name | Value | Description |
| --- | --- | --- |
| AllAtOnce | 0 | Animate all text at once. |
| ByWord | 1 | Animate text by word. |
| ByLetter | 2 | Animate text by letter. |

## AudioPlayModePreset class

 Constants which define how a sound is played.
 

## Constants

| Name | Value | Description |
| --- | --- | --- |
[Mixed](#Mixed) | -1 |  |
[Auto](#Auto) | 0 | Play automatically. |
[OnClick](#OnClick) | 1 | Play only when the icon is clicked on. |
[AllSlides](#AllSlides) | 2 | Play on all slides. Please use ( IAudioFrame#getPlayAcrossSlides/ IAudioFrame#setPlayAcrossSlides(boolean)) property instead. This element is out of date since PowerPoint 2013. |
[InClickSequence](#InClickSequence) | 3 | Play automatically with a click. |

## LayoutTargetType class

 If layout of the plot area defined manually this property specifies whether 
 to layout the plot area by its inside (not including axis and axis labels) or outside
 (including axis and axis labels).
 

## Constants

| Name | Value | Description |
| --- | --- | --- |
[Inner](#Inner) | 0 | Specifies that the plot area size shall determine the size of the plot area, not including the tick marks and axis labels. |
[Outer](#Outer) | 1 | Specifies that the plot area size shall determine the size of the plot area, the tick marks, and the axis labels. |

## HyperlinkActionType class

 Represents a type of hyperlink action.

[StartCustomSlideShow](#StartCustomSlideShow) | 9 | Start custom slideshow. |
[OpenFile](#OpenFile) | 10 | Open referenced file. |
[OpenPresentation](#OpenPresentation) | 11 | Open referenced presentation. |

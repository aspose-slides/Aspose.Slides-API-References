## HandoutType class

 Specifies how many slides and in what sequence will be placed on the page.

[Handouts4Horizontal](#Handouts4Horizontal) | 3 | Four slides per page in a horizontal sequence. |
[Handouts4Vertical](#Handouts4Vertical) | 4 | Four slides per page in a vertical sequence. |
[Handouts6Horizontal](#Handouts6Horizontal) | 5 | Six slides per page in a horizontal sequence. |

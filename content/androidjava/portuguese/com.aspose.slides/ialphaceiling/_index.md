```
public interface IAlphaCeiling extends IImageTransformOperation, IAccessiblePVIObject<IAlphaCeilingEffectiveData>
```

Representa um efeito Alpha Ceiling. Valores de Alpha (opacidade) maiores que zero são alterados para 100%. Em outras palavras, tudo que é parcialmente opaco se torna totalmente opaco.

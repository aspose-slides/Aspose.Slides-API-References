public interface IHeadingPair
```

Representa uma propriedade 'Heading pair' do documento. Indica o nome do grupo de partes do documento e o número de partes no grupo.
## Métodos

| Método | Descrição |
| --- | --- |
| [getName()](#getName--) | Retorna o nome do grupo de partes do documento. |
| [getCount()](#getCount--) | Retorna o número de partes no grupo. |
### getName() {#getName--}
```

```

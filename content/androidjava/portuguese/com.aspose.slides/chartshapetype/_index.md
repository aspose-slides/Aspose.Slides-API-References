Representa um formato de gráfico.
## Campos

| Campo | Descrição |
| --- | --- |
| [NotDefined](#NotDefined) | Forma do gráfico não está definida. |
| [Box](#Box) | Especifica que o gráfico deve ser desenhado com um formato de caixa. |
| [Cone](#Cone) | Especifica que o gráfico deve ser desenhado como um cone, com a base do cone no chão e o ponto do cone no topo do marcador de dados. |
| [ConeToMax](#ConeToMax) | Especifica que o gráfico deve ser desenhado com cones truncados de forma que o ponto do cone seja o valor máximo dos dados. |
| [Cylinder](#Cylinder) | Especifica que o gráfico deve ser desenhado como um cilindro. |
| [Pyramid](#Pyramid) | Especifica que o gráfico deve ser desenhado como uma pirâmide retangular, com a base da pirâmide no chão e o ponto da pirâmide no topo do marcador de dados. |
| [PyramidToMaximum](#PyramidToMaximum) | Especifica que o gráfico deve ser desenhado com cones truncados de forma que o ponto do cone seja o valor máximo dos dados. |

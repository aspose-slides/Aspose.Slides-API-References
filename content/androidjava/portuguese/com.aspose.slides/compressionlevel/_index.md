Especifica os níveis de compressão ZIP para arquivos OpenXML. Níveis mais altos fornecem melhor compressão ao custo de processamento mais lento.
## Campos

| Campo | Descrição |
| --- | --- |
| [None](#None) | Nenhuma compressão é aplicada. |
| [Level1](#Level1) | Compressão mais rápida com a menor taxa de compressão. |
| [Level2](#Level2) | Compressão mais rápida com taxa de compressão ligeiramente melhor que [Level1](../../com.aspose.slides/compressionlevel\#Level1). |
| [Level3](#Level3) | Fornece melhor compressão que [Level2](../../com.aspose.slides/compressionlevel\#Level2) com impacto de desempenho moderado. |
| [Level4](#Level4) | Fornece melhor compressão que [Level3](../../com.aspose.slides/compressionlevel\#Level3). |
| [Level5](#Level5) | Fornece compressão aprimorada em relação a [Level4](../../com.aspose.slides/compressionlevel\#Level4) com tempo de processamento adicional. |
| [Level6](#Level6) | Compressão padrão, oferecendo um bom equilíbrio entre velocidade de compressão e tamanho do arquivo. |
| [Level7](#Level7) | Fornece compressão maior que [Level6](../../com.aspose.slides/compressionlevel\#Level6) com processamento mais lento. |
| [Level8](#Level8) | Fornece compressão maior que [Level7](../../com.aspose.slides/compressionlevel\#Level7). |
| [Level9](#Level9) | Compressão máxima. |

Especifica a justificação do parágrafo matemático (uma série de instâncias adjacentes de texto matemático dentro do mesmo parágrafo)
## Campos

| Campo | Descrição |
| --- | --- |
| [LeftJustified](#LeftJustified) | Alinhar o texto matemático à esquerda |
| [RightJustified](#RightJustified) | Alinhar o texto matemático à direita |
| [Centered](#Centered) | Alinhar o texto matemático ao centro |
| [CenteredAsGroup](#CenteredAsGroup) | Centralizar na página todo o grupo de blocos matemáticos como uma única unidade. |

### CenteredAsGroup {#CenteredAsGroup}
```
public static final int CenteredAsGroup
```


Centralizar na página todo o grupo de blocos matemáticos como uma única unidade. Isso significa que as instâncias de texto matemático podem ser alinhadas entre si, mas todo o grupo de texto matemático é centralizado como um todo.

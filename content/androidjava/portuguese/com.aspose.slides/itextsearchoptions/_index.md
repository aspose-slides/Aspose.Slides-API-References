public interface ITextSearchOptions
```

Representa opções que podem ser usadas para pesquisar texto em uma Presentation, Slide ou TextFrame.
## Métodos

| Método | Descrição |
| --- | --- |
| [getCaseSensitive()](#getCaseSensitive--) | Defina true para usar pesquisa que diferencia maiúsculas e minúsculas, false caso contrário. |
| [setCaseSensitive(boolean value)](#setCaseSensitive-boolean-) | Defina true para usar pesquisa que diferencia maiúsculas e minúsculas, false caso contrário. |
| [getWholeWordsOnly()](#getWholeWordsOnly--) | Defina true para combinar apenas palavras completas, false caso contrário. |
| [setWholeWordsOnly(boolean value)](#setWholeWordsOnly-boolean-) | Defina true para combinar apenas palavras completas, false caso contrário. |
| [getIncludeNotes()](#getIncludeNotes--) | Defina true para incluir texto contido nas notas do slide ao executar operações de pesquisa de texto, substituição ou destaque. |
| [setIncludeNotes(boolean value)](#setIncludeNotes-boolean-) | Defina true para incluir texto contido nas notas do slide ao executar operações de pesquisa de texto, substituição ou destaque. |

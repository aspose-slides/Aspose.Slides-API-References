Representa um tipo de pontos de divisão no segundo pedaço ou barra em um gráfico de pizza-de-pizza ou barra-de-pizza.
## Campos

| Campo | Descrição |
| --- | --- |
| [Default](#Default) | Especifica que os pontos de dados devem ser divididos usando o mecanismo padrão para este tipo de gráfico. |
| [Custom](#Custom) | Especifica que os pontos de dados devem ser divididos entre a pizza e o segundo gráfico de acordo com os valores de Divisão Personalizada. |
| [ByPercentage](#ByPercentage) | Especifica que os pontos de dados devem ser divididos entre a pizza e o segundo gráfico colocando os pontos com percentual menor que a porcentagem da Posição de Divisão no segundo gráfico. |
| [ByPos](#ByPos) | Especifica que os pontos de dados devem ser divididos entre a pizza e o segundo gráfico colocando a última Posição de Divisão dos pontos de dados no segundo gráfico. |
| [ByValue](#ByValue) | Especifica que os pontos de dados devem ser divididos entre a pizza e o segundo gráfico colocando os pontos de dados com valor menor que a Posição de Divisão no segundo gráfico. |

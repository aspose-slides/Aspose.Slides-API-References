### AccentProcess {#AccentProcess}
```
public static final int AccentProcess
```

Usar para mostrar uma progressão, uma linha do tempo ou etapas sequenciais em uma tarefa, processo ou fluxo de trabalho. Funciona bem para ilustrar textos de Nível 1 e Nível 2.

### AccentedPicture {#AccentedPicture}
```
public static final int AccentedPicture
```

Usar para mostrar uma ideia fotográfica central com ideias relacionadas ao lado. O texto de Nível 1 superior aparece sobre a imagem central. O texto correspondente para outras formas de Nível 1 aparece ao lado das pequenas imagens circulares. Este layout também funciona bem sem texto.

### AlternatingFlow {#AlternatingFlow}
```
public static final int AlternatingFlow
```

Usar para mostrar grupos de informação ou etapas sequenciais em uma tarefa, processo ou fluxo de trabalho. Enfatiza a interação ou relações entre os grupos de informação.

### AlternatingHexagons {#AlternatingHexagons}
```
public static final int AlternatingHexagons
```

Usar para representar uma série de ideias interconectadas. O texto de Nível 1 aparece dentro dos hexágonos. O texto de Nível 2 aparece fora das formas.

### AlternatingPictureBlocks {#AlternatingPictureBlocks}
```
public static final int AlternatingPictureBlocks
```

Usar para mostrar uma série de imagens de cima para baixo. O texto aparece alternadamente à direita ou à esquerda da imagem.

### AlternatingPictureCircles {#AlternatingPictureCircles}
```
public static final int AlternatingPictureCircles
```

Usar para mostrar um conjunto de imagens com texto. O texto correspondente aparece nos círculos centrais com as imagens alternando da esquerda para a direita.

### ArrowRibbon {#ArrowRibbon}
```
public static final int ArrowRibbon
```

Usar para mostrar conceitos relacionados ou contrastantes com alguma conexão, como forças opostas. As duas primeiras linhas de texto de Nível 1 são usadas para o texto nas setas. O texto não usado não aparece, mas permanece disponível se você mudar o layout.

### AscendingPictureAccentProcess {#AscendingPictureAccentProcess}
```
public static final int AscendingPictureAccentProcess
```

Usar para mostrar uma série ascendente de imagens com texto descritivo. Funciona melhor com uma pequena quantidade de texto.

### Balance {#Balance}
```
public static final int Balance
```

Usar para comparar ou mostrar a relação entre duas ideias. Cada uma das duas primeiras linhas de texto de Nível 1 corresponde ao texto na parte superior de um lado do ponto central. Enfatiza o texto de Nível 2, que é limitado a quatro formas em cada lado do ponto central. O equilíbrio inclina-se para o lado com mais formas contendo texto de Nível 2. O texto não usado não aparece, mas permanece disponível se você mudar o layout.

```
public static final int BasicBendingProcess

### BasicMatrix {#BasicMatrix}

public static final int BasicMatrix

### BasicPie {#BasicPie}
```
public static final int BasicPie

### BasicVenn {#BasicVenn}
```
public static final int BasicVenn
```

Usar para mostrar relações sobrepostas ou interconectadas. As sete primeiras linhas de texto de Nível 1 correspondem a um círculo. Se houver quatro linhas ou menos de texto de Nível 1, o texto fica dentro dos círculos; se houver mais de quatro linhas, o texto fica fora dos círculos. O texto não usado não aparece, mas permanece disponível se você mudar o layout.

Representa o tipo de valor da barra de erro
## Campos

| Campo | Descrição |
| --- | --- |
| [Custom](#Custom) | Especifica que o comprimento das barras de erro deve ser determinado pelos elementos Plus e Minus. |
| [Fixed](#Fixed) | Especifica que o comprimento das barras de erro deve ser o valor fixo determinado por Error Bar Value. |
| [Percentage](#Percentage) | Especifica que o comprimento das barras de erro deve ser a porcentagem Error Bar Value dos dados. |
| [StandardDeviation](#StandardDeviation) | Especifica que o comprimento das barras de erro deve ser Error Bar Value desvios padrão dos dados. |
| [StandardError](#StandardError) | Especifica que o comprimento das barras de erro deve ser Error Bar Value erros padrão dos dados. |

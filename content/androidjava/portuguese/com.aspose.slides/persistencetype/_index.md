### PersistPropertyBag {#PersistPropertyBag}
```
public static final int PersistPropertyBag
```

Especifica que o controle ActiveX é persistido usando persistência baseada em property-bag. A persistência baseada em property-bag armazena um controle ActiveX por meio de uma coleção de pares nome e valor que especificam os dados persistidos pelo controle ActiveX.

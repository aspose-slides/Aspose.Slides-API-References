```
public final class InkEffectType extends System.Enum
```

Especifica um conjunto de efeitos visuais predefinidos para renderização de tinta. Cada efeito corresponde a uma textura ou imagem que simula uma aparência estilizada de tinta. Esses valores podem ser usados para personalizar o estilo visual das pinceladas de tinta digital durante a renderização.
## Campos

| Campo | Descrição |
| --- | --- |
| [NotDefined](#NotDefined) | O efeito não está definido ou é desconhecido. |
| [Bronze](#Bronze) | Uma textura metálica quente e acastanhada que se assemelha a tinta de bronze. |
| [Galaxy](#Galaxy) | Uma textura colorida e cintilante que se assemelha a uma galáxia, com tons cósmicos. |
| [Gold](#Gold) | Uma textura dourada metálica brilhante que confere às pinceladas de tinta uma aparência luxuosa. |
| [Lava](#Lava) | Uma textura flamejante que se assemelha a lava fundida, com tons vermelhos e laranjas. |
| [Ocean](#Ocean) | Uma textura azul profunda, semelhante a fluido, que imita ondas oceânicas ou tinta à base de água. |
| [RainbowGlitter](#RainbowGlitter) | Um efeito cintilante e colorido de glitter arco-íris usado para pinceladas de tinta festivas ou vibrantes. |
| [RoseGold](#RoseGold) | Uma mistura suave de rosa e dourado, semelhante ao ouro rosa, para pinceladas de tinta elegantes. |
| [Silver](#Silver) | Uma textura metálica fria e prateada que simula tinta prata clássica. |

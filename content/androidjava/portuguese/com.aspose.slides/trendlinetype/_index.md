## Campos

| Campo | Descrição |
| --- | --- |
| [Exponential](#Exponential) | Especifica que a linha de tendência deve ser uma curva exponencial na forma. |
| [Linear](#Linear) | Especifica que a linha de tendência deve ser uma linha na forma. |
| [Logarithmic](#Logarithmic) | Especifica que a linha de tendência deve ser uma curva logarítmica na forma, onde log é o logaritmo natural. |
| [MovingAverage](#MovingAverage) | Especifica que a linha de tendência deve ser uma média móvel de período Period. |
| [Polynomial](#Polynomial) | Especifica que a linha de tendência deve ser uma curva polinomial de ordem Order na forma. |
| [Power](#Power) | Especifica que a linha de tendência deve ser uma curva de potência na forma. |

### Standard {#Standard}
```
public static final int Standard
```

Use para qualquer rótulo de sensibilidade que não tenha sido aplicado diretamente pelo usuário. Isso inclui quaisquer rótulos padrão, rótulos aplicados automaticamente.

### Privileged {#Privileged}
```
public static final int Privileged
```

Use para qualquer rótulo de sensibilidade que tenha sido aplicado diretamente pelo usuário. Isso inclui quaisquer rótulos de sensibilidade aplicados manualmente, bem como rotulagem recomendada ou obrigatória ou qualquer recurso onde o usuário decide qual rótulo de sensibilidade aplicar.

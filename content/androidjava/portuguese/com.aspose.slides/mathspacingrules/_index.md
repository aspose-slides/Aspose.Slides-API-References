Tipos de espaçamento (espaço horizontal) entre colunas de uma matriz
## Campos

| Campo | Descrição |
| --- | --- |
| [SingleSpacingGap](#SingleSpacingGap) | Único (1 em para colunas ou 1 linha para linhas) |
| [OneAndHalfSpacingGap](#OneAndHalfSpacingGap) | Um e meio (1.5 em para colunas ou 1.5 linhas para linhas) |
| [DoubleSpacingGap](#DoubleSpacingGap) | Espaçamento duplo (2 em para colunas ou 2 linhas para linhas) |
| [Exactly](#Exactly) | Valor exato. Para colunas, dependa do valor de ColumnGap, medido em twips. Para linhas, dependa do valor de RowGap, medido em twips. |
| [Multiple](#Multiple) | Múltiplo. Para colunas, dependa do valor de ColumnGap, medido em incrementos de 0.5 em. Para linhas, dependa do valor de RowGap, medido em linhas. |

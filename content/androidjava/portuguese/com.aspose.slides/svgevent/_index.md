## Campos

| Field | Description |
| --- | --- |
| [OnFocusIn](#OnFocusIn) | Ocorre quando um elemento recebe foco, como quando um texto é selecionado. |

| [OnActivate](#OnActivate) | Ocorre quando um elemento é ativado, por exemplo, através de um clique do mouse ou de uma tecla. |
| [OnClick](#OnClick) | Ocorre quando o botão do dispositivo apontador é clicado sobre um elemento. |
| [OnMouseDown](#OnMouseDown) | Ocorre quando o botão do dispositivo apontador é pressionado sobre um elemento. |
| [OnMouseUp](#OnMouseUp) | Ocorre quando o botão do dispositivo apontador é solto sobre um elemento. |
| [OnMouseOver](#OnMouseOver) | Ocorre quando o dispositivo apontador é movido sobre um elemento. |
| [OnMouseMove](#OnMouseMove) | Ocorre quando o dispositivo apontador é movido enquanto está sobre um elemento. |
| [OnMouseOut](#OnMouseOut) | Ocorre quando o dispositivo apontador se afasta de um elemento. |
| [OnLoad](#OnLoad) | Ocorre quando o agente de usuário analisou completamente o elemento, seus descendentes e todos os recursos referenciados necessários para renderizá-lo. |
| [OnUnload](#OnUnload) | Ocorre quando a implementação do DOM remove um documento de uma janela ou quadro. |
| [OnAbort](#OnAbort) | Ocorre quando o carregamento da página é interrompido antes que um elemento seja carregado completamente. |
| [OnError](#OnError) | Ocorre quando um elemento não carrega corretamente ou quando ocorre um erro durante a execução do script. |
| [OnResize](#OnResize) | Ocorre quando a visualização do documento está sendo redimensionada. |
| [OnScroll](#OnScroll) | Ocorre quando a visualização do documento está sendo deslocada ao longo do eixo X, Y ou ambos. |
| [OnZoom](#OnZoom) | Ocorre quando o nível de zoom da visualização do documento está sendo alterado. |
| [OnBegin](#OnBegin) | Ocorre quando um elemento de animação inicia. |
| [OnEnd](#OnEnd) | Ocorre quando um elemento de animação termina. |
| [OnRepeat](#OnRepeat) | Ocorre quando um elemento de animação se repete. |

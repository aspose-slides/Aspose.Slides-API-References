表示图表的形状。

## 字段

| 字段 | 描述 |
| --- | --- |
| [NotDefined](#NotDefined) | 图表形状未定义。 |
| [Box](#Box) | 指定图表应使用盒形绘制。 |
| [Cone](#Cone) | 指定图表应绘制为圆锥形，圆锥的底部位于地面，圆锥的顶点位于数据标记的顶部。 |
| [ConeToMax](#ConeToMax) | 指定图表应绘制为截断圆锥，使圆锥的顶点对应最大数据值。 |
| [Cylinder](#Cylinder) | 指定图表应绘制为圆柱体。 |
| [Pyramid](#Pyramid) | 指定图表应绘制为矩形金字塔，金字塔的底部位于地面，顶点位于数据标记的顶部。 |
| [PyramidToMaximum](#PyramidToMaximum) | 指定图表应绘制为截断圆锥，使圆锥的顶点对应最大数据值。 |

表示已分配给形状的草图类型或效果。

## 字段

| 字段 | 描述 |
| --- | --- |
| [NotDefined](#NotDefined) | 指定形状的 Sketch 效果未定义。 |
| [None](#None) | 指定形状没有 Sketch 效果。 |
| [Curved](#Curved) | 指定形状具有 Curved 效果，该效果将形状的每条边转换为一个大的柔和曲线。 |
| [Freehand](#Freehand) | 指定形状具有 Freehand 效果，该效果最接近不规则绘制的线条。 |
| [Scribble](#Scribble) | 指定形状具有 Scribble 效果，呈现夸张的振荡，仿佛故意绘制得凌乱。 |

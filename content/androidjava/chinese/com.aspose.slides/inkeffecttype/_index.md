指定一组预定义的墨水渲染视觉效果。每种效果对应于一种纹理或图像，用于模拟风格化的墨水外观。这些值可用于在渲染过程中自定义数字墨水笔画的视觉样式。

## 字段

| 字段 | 描述 |
| --- | --- |
| [NotDefined](#NotDefined) | 该效果未定义或未知。 |
| [Bronze](#Bronze) | 一种温暖的棕色色金属纹理，类似青铜墨水。 |
| [Galaxy](#Galaxy) | 一种多彩、闪烁的纹理，类似星系，带有宇宙色调。 |
| [Gold](#Gold) | 一种明亮的金属金色纹理，使墨水笔画呈现奢华外观。 |
| [Lava](#Lava) | 一种炽热的纹理，类似熔融的熔岩，带有红色和橙色色调。 |
| [Ocean](#Ocean) | 一种深蓝色、类似流体的纹理，模拟海浪或水基墨水。 |
| [RainbowGlitter](#RainbowGlitter) | 一种多彩、闪亮的彩虹亮片效果，用于节日或充满活力的墨水笔画。 |
| [RoseGold](#RoseGold) | 一种柔和的粉金色混合，类似玫瑰金，适用于优雅的墨水笔画。 |
| [Silver](#Silver) | 一种凉爽的金属银色纹理，模拟经典银色墨水。 |

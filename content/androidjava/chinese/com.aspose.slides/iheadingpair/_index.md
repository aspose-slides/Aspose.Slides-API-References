public interface IHeadingPair
```

表示 'Heading pair' 属性的文档。它指示文档部件的组名以及组中部件的数量。
## 方法

| 方法 | 描述 |
| --- | --- |
| [getName()](#getName--) | 返回文档部件的组名。 |
| [getCount()](#getCount--) | 返回组中部件的数量。 |

public abstract String getName()

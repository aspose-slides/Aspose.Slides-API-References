[com.aspose.slides.IImageTransformOperation](../../com.aspose.slides/iimagetransformoperation), com.aspose.slides.IAccessiblePVIObject
```
public interface IAlphaCeiling extends IImageTransformOperation, IAccessiblePVIObject<IAlphaCeilingEffectiveData>
```

表示 Alpha Ceiling 效果。Alpha（不透明度）值大于零的会被更改为 100%。换句话说，任何部分不透明的对象都会变为完全不透明。

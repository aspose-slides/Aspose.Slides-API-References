确定条形图片填充模式。

## 字段

| 字段 | 描述 |
| --- | --- |
| [NotDefined](#NotDefined) | 未定义。 |
| [Stack](#Stack) | 图片的大小设置为在最长的堆叠条形中最多重复 15 次。 |
| [StackScale](#StackScale) | 图片的大小设置为指定的单位数，并在条形的长度上重复。 |
| [Stretch](#Stretch) | 图片被拉伸至堆叠条形的整个长度。 |

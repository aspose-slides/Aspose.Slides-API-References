表示敏感度标签的分配方式。

## 字段

| 字段 | 描述 |
| --- | --- |
| [Standard](#Standard) | 用于未被用户直接应用的任何敏感度标签。 |
| [Privileged](#Privileged) | 用于已被用户直接应用的任何敏感度标签。 |
### 标准 {#Standard}
```
public static final int Standard
```

用于任何未直接由用户应用的敏感度标签。这包括任何默认标签、自动应用的标签。

### 特权 {#Privileged}
```
public static final int Privileged
```

用于任何已直接由用户应用的敏感度标签。这包括任何手动应用的敏感度标签，以及推荐的或强制的标记，或任何让用户决定使用哪个敏感度标签的功能。

指定数学段落的对齐方式（同一段落中相邻的数学文本实例的系列）

## 字段

| 字段 | 描述 |

| [RightJustified](#RightJustified) | 将数学文本右对齐 |
| [Centered](#Centered) | 将数学文本居中对齐 |
| [CenteredAsGroup](#CenteredAsGroup) | 在页面上将整个数学块组居中，作为单个单元。 |

### CenteredAsGroup {#CenteredAsGroup}
```
public static final int CenteredAsGroup
```

在页面上将整个数学块组居中，作为单个单元。这意味着数学文本的实例可以相互对齐，但整个数学文本组整体居中。

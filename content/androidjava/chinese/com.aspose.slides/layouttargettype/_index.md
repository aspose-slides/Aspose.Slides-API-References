**继承:**
java.lang.Object, com.aspose.ms.System.ValueType, com.aspose.ms.System.Enum
```
public final class LayoutTargetType extends System.Enum
```

如果手动定义了绘图区域的布局，此属性指定是按内部布局绘图区域（不包括坐标轴和坐标轴标签）还是按外部布局绘图区域（包括坐标轴和坐标轴标签）。
## 字段

| 字段 | 描述 |
| --- | --- |
| [Inner](#Inner) | 指定绘图区域的大小应决定绘图区域的尺寸，不包括刻度线和坐标轴标签。 |
| [Outer](#Outer) | 指定绘图区域的大小应决定绘图区域的尺寸、刻度线和坐标轴标签。 |
### Inner {#Inner}
```
public static final int Inner
```

### Outer {#Outer}
```
public static final int Outer
```

指定绘图区域的大小应决定绘图区域的尺寸、刻度线和坐标轴标签。

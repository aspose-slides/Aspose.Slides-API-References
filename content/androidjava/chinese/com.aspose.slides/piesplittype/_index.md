表示饼图中的饼图或柱状图中第二个饼或柱的拆分点类型。

## 字段

| 字段 | 描述 |
| --- | --- |
| [Default](#Default) | 指定数据点应使用此图表类型的默认机制进行拆分。 |
| [Custom](#Custom) | 指定数据点应根据 Custom Split 值在饼图和第二个图表之间进行拆分。 |
| [ByPercentage](#ByPercentage) | 指定数据点应通过将百分比小于 Split Position 百分比的点放入第二个图表，从而在饼图和第二个图表之间进行拆分。 |
| [ByPos](#ByPos) | 指定数据点应通过将数据点的最后 Split Position 放入第二个图表，从而在饼图和第二个图表之间进行拆分。 |
| [ByValue](#ByValue) | 指定数据点应通过将数值小于 Split Position 的数据点放入第二个图表，从而在饼图和第二个图表之间进行拆分。 |

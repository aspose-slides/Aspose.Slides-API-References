public interface IHeadingPair
```

Representerar en 'Heading pair'-egenskap i dokumentet. Den anger gruppnamnet för dokumentdelarna och antalet delar i gruppen.
## Metoder

| Metod | Beskrivning |
| --- | --- |
| [getName()](#getName--) | Returnerar gruppnamnet för dokumentdelarna. |
| [getCount()](#getCount--) | Returnerar antalet delar i gruppen. |
### getName() {#getName--}
```
public abstract String getName()
```

Returnerar gruppnamnet för dokumentdelarna. Skrivskyddad String.

**Returnerar:**
java.lang.String
### getCount() {#getCount--}
```
public abstract int getCount()
```

Returnerar antalet delar i gruppen. Skrivskyddad int.

int

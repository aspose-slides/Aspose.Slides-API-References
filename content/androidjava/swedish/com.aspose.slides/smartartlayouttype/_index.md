### AccentProcess {#AccentProcess}
```
public static final int AccentProcess
```

Använd för att visa en progression, en tidslinje eller sekventiella steg i en uppgift, process eller arbetsflöde. Fungerar bra för att illustrera både nivå-1- och nivå-2-text.

### AccentedPicture {#AccentedPicture}
```
public static final int AccentedPicture
```

Använd för att visa en central, fotografisk idé med relaterade idéer på sidan. Den översta nivå-1-texten visas över den centrala bilden. Motsvarande text för övriga nivå-1-former visas bredvid de små cirkulära bilderna. Denna layout fungerar även utan text.

```
public static final int AlternatingFlow
```

```
public static final int AlternatingHexagons
```

```
public static final int ArrowRibbon
```

### AscendingPictureAccentProcess {#AscendingPictureAccentProcess}

### Balance {#Balance}
```
public static final int Balance
```

Använd för att jämföra eller visa relationen mellan två idéer. Varje av de två första raderna av nivå-1-text motsvarar text högst upp på varsin sida om mittpunkten. Betonar nivå-2-text, som är begränsad till fyra former på varje sida om mittpunkten. Balansen tippar mot den sida med flest former som innehåller nivå-2-text. Oanvänd text visas inte, men finns kvar om du byter layout.

Använd för alla känslighetsetiketter som inte applicerades direkt av användaren. Detta inkluderar alla standardetiketter, automatiskt applicerade etiketter.

### Privilegierad {#Privileged}
```
public static final int Privileged
```

Använd för alla känslighetsetiketter som applicerades direkt av användaren. Detta inkluderar alla manuellt applicerade känslighetsetiketter samt rekommenderad eller obligatorisk märkning eller någon funktion där användaren bestämmer vilken känslighetsetikett som ska appliceras.

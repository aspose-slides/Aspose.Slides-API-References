Anger en uppsättning fördefinierade visuella effekter för bläckrendering. Varje effekt motsvarar en textur eller bild som simulerar ett stiliserat bläckutseende. Dessa värden kan användas för att anpassa den visuella stilen för digitala bläckstreck under rendering.

## Fält

| Fält | Beskrivning |
| --- | --- |
| [NotDefined](#NotDefined) | Effekten är inte definierad eller är okänd. |
| [Bronze](#Bronze) | En varm, brunaktig metallisk textur som liknar bronsbläck. |
| [Galaxy](#Galaxy) | En färgstark, skimrande textur som liknar en galax, med kosmiska toner. |
| [Gold](#Gold) | En ljus, metallisk guldtextur som ger bläckstrecken ett lyxigt utseende. |
| [Lava](#Lava) | En eldig textur som liknar smält lava, med röda och orange nyanser. |
| [Ocean](#Ocean) | En djupblå, vätskeliknande textur som efterliknar havsvågor eller vattenbaserat bläck. |
| [RainbowGlitter](#RainbowGlitter) | En färgstark, glittrande regnbågsglittereffekt som används för festliga eller livfulla bläckstreck. |
| [RoseGold](#RoseGold) | En mjuk rosa-guldblandning, liknande roséguld, för eleganta bläckstreck. |
| [Silver](#Silver) | En kall, metallisk silvertextur som simulerar klassiskt silverbläck. |

表示敏感度標籤的指派方法。
## 欄位

| 欄位 | 描述 |
| --- | --- |
| [Standard](#Standard) | 用於任何未直接由使用者套用的敏感度標籤。 |
| [Privileged](#Privileged) | 用於任何直接由使用者套用的敏感度標籤。 |
### 標準 {#Standard}
```
public static final int Standard
```

用於任何未直接由使用者套用的敏感度標籤。這包括任何預設標籤、會自動套用的標籤。

### 特權 {#Privileged}
```
public static final int Privileged
```

用於任何直接由使用者套用的敏感度標籤。這包括任何手動套用的敏感度標籤，以及建議或強制的標記，或任何讓使用者自行決定套用哪個敏感度標籤的功能。

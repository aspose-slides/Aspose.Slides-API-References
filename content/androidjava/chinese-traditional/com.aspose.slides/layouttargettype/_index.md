**繼承:**
java.lang.Object, com.aspose.ms.System.ValueType, com.aspose.ms.System.Enum
```
public final class LayoutTargetType extends System.Enum
```

如果手動定義繪圖區的版面，此屬性指定是根據內部（不包括坐標軸及坐標軸標籤）還是外部（包括坐標軸及坐標軸標籤）來佈局繪圖區。
## 欄位

| 欄位 | 說明 |
| --- | --- |
| [Inner](#Inner) | 指定繪圖區的大小應決定繪圖區的尺寸，不包括刻度線和坐標軸標籤。 |
| [Outer](#Outer) | 指定繪圖區的大小應決定繪圖區的尺寸、刻度線和坐標軸標籤。 |
### 內部 {#Inner}
```
public static final int Inner
```


指定繪圖區的大小應決定繪圖區的尺寸，不包括刻度線和坐標軸標籤。

### 外部 {#Outer}
```
public static final int Outer
```


指定繪圖區的大小應決定繪圖區的尺寸、刻度線和坐標軸標籤。

| [getWholeWordsOnly()](#getWholeWordsOnly--) | 設定為 true 以僅匹配完整單詞，否則為 false。 |
| [setWholeWordsOnly(boolean value)](#setWholeWordsOnly-boolean-) | 設定為 true 以僅匹配完整單詞，否則為 false。 |
| [getIncludeNotes()](#getIncludeNotes--) | 設定為 true 以在執行文字搜尋、取代或醒目顯示操作時包含投影片備註中的文字。 |

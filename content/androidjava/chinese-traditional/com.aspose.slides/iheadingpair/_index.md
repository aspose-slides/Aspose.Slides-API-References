public interface IHeadingPair
```

表示文件的「Heading pair」屬性。它指示文件部件的組名稱以及組中部件的數量。
## 方法

| 方法 | 描述 |
| --- | --- |
| [getName()](#getName--) | 返回文件部件的組名稱。 |
| [getCount()](#getCount--) | 返回組中部件的數量。 |
### getName() {#getName--}
```
public abstract String getName()
```


返回文件部件的組名稱。唯讀 String.

**返回:**  
java.lang.String
### getCount() {#getCount--}
```
public abstract int getCount()
```


返回組中部件的數量。唯讀 int.

**返回:**  
int

public final class InkEffectType extends System.Enum

指定一組預先定義的墨跡渲染視覺效果。每個效果對應一種紋理或圖像，用於模擬風格化的墨跡外觀。這些值可用於在渲染過程中自訂數位墨跡的視覺樣式。
## Fields

| Field | Description |
| --- | --- |
| [NotDefined](#NotDefined) | 此效果未定義或未知。 |
| [Bronze](#Bronze) | 一種溫暖、帶有棕色金屬感的紋理，類似青銅墨水。 |
| [Galaxy](#Galaxy) | 一種多彩、閃爍的紋理，類似星系，呈現宇宙色調。 |
| [Gold](#Gold) | 一種明亮的金屬金色紋理，為墨跡賦予奢華外觀。 |
| [Lava](#Lava) | 一種火焰般的紋理，類似熔岩，帶有紅色和橙色的色調。 |
| [Ocean](#Ocean) | 一種深藍色、流體般的紋理，模擬海浪或水性墨水。 |
| [RainbowGlitter](#RainbowGlitter) | 一種多彩、閃亮的彩虹亮粉效果，用於節慶或鮮豔的墨跡。 |
| [RoseGold](#RoseGold) | 一種柔和的粉金混合色，類似玫瑰金，適合優雅的墨跡。 |
| [Silver](#Silver) | 一種冷色調的金屬銀色紋理，模擬經典銀色墨水。 |

## 欄位

| 欄位 | 說明 |
| --- | --- |
| [NotDefined](#NotDefined) | 圖表形狀未定義。 |
| [Box](#Box) | 指定圖表應以盒狀形狀繪製。 |
| [Cone](#Cone) | 指定圖表應以圓錐形繪製，圓錐的底部位於地面，頂點位於資料標記的上方。 |
| [ConeToMax](#ConeToMax) | 指定圖表應以截斷的圓錐形繪製，使圓錐的頂點對應最大資料值。 |
| [Cylinder](#Cylinder) | 指定圖表應以圓柱形繪製。 |
| [Pyramid](#Pyramid) | 指定圖表應以矩形金字塔形繪製，金字塔的底部位於地面，頂點位於資料標記的上方。 |
| [PyramidToMaximum](#PyramidToMaximum) | 指定圖表應以截斷的圓錐形繪製，使圓錐的頂點對應最大資料值。 |

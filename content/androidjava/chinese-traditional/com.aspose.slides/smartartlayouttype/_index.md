### AccentProcess {#AccentProcess}
```
public static final int AccentProcess
```

用於顯示任務、流程或工作流程中的進程、時間軸或順序步驟。非常適合說明第一層與第二層文字。

### AccentedPicture {#AccentedPicture}
```
public static final int AccentedPicture
```

用於顯示中心的攝影概念，並在側邊呈現相關概念。頂部的第一層文字顯示於中心圖片上方。其他第一層形狀的相應文字顯示在小圓形圖片旁。此版面在沒有文字時亦可良好運作。

### Balance {#Balance}
```
public static final int Balance
```

用於比較或展示兩個概念之間的關係。第一層文字的前兩行分別對應於中心點兩側頂部的文字。強調第二層文字，最多在中心點兩側各有四個形狀。平衡會傾向於包含較多第二層文字形狀的一側。未使用的文字不會顯示，但在切換版面時仍可使用。

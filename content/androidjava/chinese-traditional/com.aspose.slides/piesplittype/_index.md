代表在 pie-of-pie 或 bar-of-pie 圖表中第二個餅圖或條形圖的拆分點類型。
## 欄位

| 欄位 | 說明 |
| --- | --- |
| [Default](#Default) | 指定資料點應使用此圖表類型的預設機制進行拆分。 |
| [Custom](#Custom) | 指定資料點應根據自訂拆分值在餅圖與第二個圖表之間進行拆分。 |
| [ByPercentage](#ByPercentage) | 指定資料點應通過將百分比低於 Split Position 百分比的點放入第二個圖表來在餅圖與第二個圖表之間拆分。 |
| [ByPos](#ByPos) | 指定資料點應通過將資料點的最後 Split Position 放入第二個圖表來在餅圖與第二個圖表之間拆分。 |
| [ByValue](#ByValue) | 指定資料點應通過將值低於 Split Position 的資料點放入第二個圖表來在餅圖與第二個圖表之間拆分。 |

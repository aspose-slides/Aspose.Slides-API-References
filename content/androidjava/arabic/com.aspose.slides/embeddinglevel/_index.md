### Installable {#Installable}
```
public static final int Installable
```

الخطوط التي لديها هذا الإعداد تشير إلى إمكانية تضمينها وتثبيتها دائمًا على النظام البعيد بواسطة تطبيق. يحصل مستخدم النظام البعيد على نفس الحقوق والالتزامات والرخص لهذا الخط كما يحصل المشتري الأصلي للخط، ويخضع لنفس اتفاقية ترخيص المستخدم النهائي، وحقوق الطبع والنشر، وبراءة التصميم، و/أو العلامة التجارية كما كان الحال للمشتري الأصلي.

### Restricted {#Restricted}
```
public static final int Restricted
```

الخطوط التي لديها هذا البت فقط يجب ألا تُعدل أو تُضمّن أو تُستبدل بأي طريقة دون الحصول أولاً على إذن من المالك القانوني.

### PreviewPrint {#PreviewPrint}
```
public static final int PreviewPrint
```

عند تفعيل هذا البت، يمكن تضمين الخط وتحميله مؤقتًا على النظام البعيد. يجب فتح المستندات التي تحتوي على خطوط Preview & Print في وضع "قراءة فقط"; لا يمكن تعديل المستند.

### Editable {#Editable}
```
public static final int Editable
```

عند تفعيل هذا البت، يمكن تضمين الخط لكن يجب تثبيته مؤقتًا فقط على الأنظمة الأخرى. على عكس خطوط Preview & Print، يمكن فتح المستندات التي تحتوي على خطوط Editable للقراءة، ويسمح بالتحرير، ويمكن حفظ التغييرات.

### NoSubsetting {#NoSubsetting}
```
public static final int NoSubsetting
```

عند تفعيل هذا البت، لا يجوز تقليص الخط قبل تضمينه. تنطبق القيود الأخرى لتضمين الخط المحددة في البتات 0-3 و9 أيضًا.

### BitmapOnly {#BitmapOnly}
```
public static final int BitmapOnly
```

عند تفعيل هذا البت، يمكن فقط تضمين الصور النقطية الموجودة في الخط. لا يمكن تضمين بيانات الخط الخارطي. إذا لم تتوفر أي صور نقطية في الخط، يُعتبر الخط غير قابل للتضمين وستفشل خدمات التضمين.

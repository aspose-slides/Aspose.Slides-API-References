### Εγκαταστάσιμο {#Installable}
```
public static final int Installable
```

Οι γραμματοσειρές με αυτή τη ρύθμιση υποδεικνύουν ότι μπορούν να ενσωματωθούν και να εγκατασταθούν μόνιμα στο απομακρυσμένο σύστημα από μια εφαρμογή. Ο χρήστης του απομακρυσμένου συστήματος αποκτά τα ίδια δικαιώματα, υποχρεώσεις και άδειες για αυτή τη γραμματοσειρά όπως ο αρχικός αγοραστής, και υπόκειται στην ίδια συμφωνία αδειοδότησης τελικού χρήστη, πνευματικά δικαιώματα, σχεδιαστικό δίπλωμα και/ή εμπορικό σήμα όπως ο αρχικός αγοραστής.

### Περιορισμένο {#Restricted}
```
public static final int Restricted
```

Οι γραμματοσειρές που έχουν μόνο αυτό το bit ενεργοποιημένο δεν πρέπει να τροποποιούνται, ενσωματώνονται ή ανταλλάσσονται με κανέναν τρόπο χωρίς πρώτα την άδεια του νόμιμου ιδιοκτήτη.

### Προεπισκόπηση & Εκτύπωση {#PreviewPrint}
```
public static final int PreviewPrint
```

Όταν αυτό το bit είναι ενεργοποιημένο, η γραμματοσειρά μπορεί να ενσωματωθεί και να φορτωθεί προσωρινά στο απομακρυσμένο σύστημα. Τα έγγραφα που περιέχουν γραμματοσειρές Προεπισκόπηση & Εκτύπωση πρέπει να ανοιχτούν μόνο για ανάγνωση· δεν επιτρέπεται να γίνουν αλλαγές στο έγγραφο.

### Επεξεργάσιμο {#Editable}
```
public static final int Editable
```

Όταν αυτό το bit είναι ενεργοποιημένο, η γραμματοσειρά μπορεί να ενσωματωθεί αλλά πρέπει να εγκατασταθεί προσωρινά μόνο σε άλλα συστήματα. Σε αντίθεση με τις γραμματοσειρές Προεπισκόπηση & Εκτύπωση, τα έγγραφα που περιέχουν επεξεργάσιμες γραμματοσειρές μπορούν να ανοιχτούν για ανάγνωση, επιτρέπεται η επεξεργασία και οι αλλαγές μπορούν να αποθηκευτούν.

### Χωρίς Υποσύνολο {#NoSubsetting}
```
public static final int NoSubsetting
```

Όταν αυτό το bit είναι ενεργοποιημένο, η γραμματοσειρά δεν μπορεί να υποσύνολο (subset) πριν την ενσωμάτωση. Ισχύουν επίσης άλλοι περιορισμοί ενσωμάτωσης που ορίζονται στα bits 0-3 και 9.

### Μόνο bitmap {#BitmapOnly}
```
public static final int BitmapOnly
```

Όταν αυτό το bit είναι ενεργοποιημένο, μόνο τα bitmap που περιέχονται στη γραμματοσειρά μπορούν να ενσωματωθούν. Δεν επιτρέπεται η ενσωμάτωση δεδομένων περιγράμματος. Εάν δεν υπάρχουν διαθέσιμα bitmap στη γραμματοσειρά, τότε η γραμματοσειρά θεωρείται μη ενσωματώσιμη και οι υπηρεσίες ενσωμάτωσης θα αποτύχουν.

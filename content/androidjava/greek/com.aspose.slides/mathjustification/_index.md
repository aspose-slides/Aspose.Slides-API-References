### CenteredAsGroup {#CenteredAsGroup}
```
public static final int CenteredAsGroup
```

Κεντράρει στη σελίδα ολόκληρη την ομάδα των μαθηματικών μπλοκ ως μία ενιαία μονάδα. Αυτό σημαίνει ότι οι περιπτώσεις του μαθηματικού κειμένου μπορούν να ευθυγραμμιστούν μεταξύ τους, αλλά ολόκληρη η ομάδα του μαθηματικού κειμένου κεντράρεται ως σύνολο.

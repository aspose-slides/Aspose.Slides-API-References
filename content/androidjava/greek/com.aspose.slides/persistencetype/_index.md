Καθορίζει τη μέθοδο που χρησιμοποιείται για την αποθήκευση των ιδιοτήτων του στοιχείου ελέγχου ActiveX.

| Πεδίο | Περιγραφή |

| [PersistPropertyBag](#PersistPropertyBag) | Καθορίζει ότι το στοιχείο ελέγχου ActiveX διατηρείται χρησιμοποιώντας property-bag-based persistence. |

| [PersistStreamInit](#PersistStreamInit) | Καθορίζει ότι το στοιχείο ελέγχου ActiveX διατηρείται χρησιμοποιώντας stream-based persistence που υποστηρίζει την αρχικοποίηση του στοιχείου ελέγχου ActiveX σε προεπιλεγμένη κατάσταση. |
| [PersistStorage](#PersistStorage) | Καθορίζει ότι το στοιχείο ελέγχου ActiveX διατηρείται χρησιμοποιώντας storage-based persistence. |
### NotDefined {#NotDefined}
```
public static final int NotDefined
```

Το αναγνωριστικό μόνιμης αποθήκευσης δεν έχει οριστεί.

### PersistPropertyBag {#PersistPropertyBag}
```
public static final int PersistPropertyBag
```

Καθορίζει ότι το στοιχείο ελέγχου ActiveX διατηρείται χρησιμοποιώντας property-bag-based persistence. Η property-bag-based persistence αποθηκεύει ένα στοιχείο ελέγχου ActiveX μέσω μιας συλλογής ζευγών όνομα-τιμή που καθορίζουν τα δεδομένα που αποθηκεύονται από το στοιχείο ελέγχου ActiveX.

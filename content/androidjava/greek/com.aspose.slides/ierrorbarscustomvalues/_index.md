public interface IErrorBarsCustomValues
```

Καθορίζει τις τιμές των ραβδών σφάλματος. Θα πρέπει να χρησιμοποιείται μόνο όταν ο τύπος τιμής των ραβδών σφάλματος είναι Custom.

### getXMinus() {#getXMinus--}
```
public abstract IDoubleChartValue getXMinus()
```

Καθορίζει την τιμή της ραβδου σφάλματος στη αρνητική κατεύθυνση. Διαθέσιμο εάν ο τύπος τιμής των ραβδών σφάλματος είναι Custom και το ErrorBarsXFormat είναι επιτρεπτό. Σε οποιαδήποτε άλλη περίπτωση αυτή η ιδιότητα επιστρέφει null. Μόνο για ανάγνωση [IDoubleChartValue](../../com.aspose.slides/idoublechartvalue).

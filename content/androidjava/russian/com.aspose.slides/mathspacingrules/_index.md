Типы зазоров (горизонтальное расстояние) между столбцами матрицы
## Поля

| Поле | Описание |
| --- | --- |
| [SingleSpacingGap](#SingleSpacingGap) | Одинарный (1 em для столбцов или 1 строка для строк) |
| [OneAndHalfSpacingGap](#OneAndHalfSpacingGap) | Полтора (1.5 em для столбцов или 1.5 строки для строк) |
| [DoubleSpacingGap](#DoubleSpacingGap) | Двойной зазор (2 em для столбцов или 2 строки для строк) |
| [Exactly](#Exactly) | Точное значение. Для столбцов – значение ColumnGap, измеряется в твипсах. Для строк – значение RowGap, измеряется в твипсах. |
| [Multiple](#Multiple) | Множественный. Для столбцов – значение ColumnGap, измеряется с шагом 0.5 em. Для строк – значение RowGap, измеряется в строках. |

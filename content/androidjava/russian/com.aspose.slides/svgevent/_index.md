## Поля

| Поле | Описание |

| [OnClick](#OnClick) | Происходит, когда кнопка указывающего устройства щёлкнута над элементом. |
| [OnMouseDown](#OnMouseDown) | Происходит, когда кнопка указывающего устройства нажата над элементом. |
| [OnMouseUp](#OnMouseUp) | Происходит, когда кнопка указывающего устройства отпущена над элементом. |
| [OnMouseOver](#OnMouseOver) | Происходит, когда указывающее устройство перемещено на элемент. |
| [OnMouseMove](#OnMouseMove) | Происходит, когда указывающее устройство перемещается, находясь над элементом. |
| [OnMouseOut](#OnMouseOut) | Происходит, когда указывающее устройство перемещается прочь от элемента. |
| [OnLoad](#OnLoad) | Происходит, когда пользовательский агент полностью проанализировал элемент, его потомков и все связанные ресурсы, необходимые для его отображения. |
| [OnUnload](#OnUnload) | Происходит, когда реализация DOM удаляет документ из окна или фрейма. |

| [OnError](#OnError) | Происходит, когда элемент не загружается корректно или возникает ошибка во время выполнения сценария. |

| [OnScroll](#OnScroll) | Происходит, когда представление документа смещается вдоль оси X, Y или обеих осей. |
| [OnZoom](#OnZoom) | Происходит, когда уровень масштабирования представления документа изменяется. |
| [OnBegin](#OnBegin) | Происходит, когда анимационный элемент начинается. |
| [OnEnd](#OnEnd) | Происходит, когда анимационный элемент заканчивается. |
| [OnRepeat](#OnRepeat) | Происходит, когда анимационный элемент повторяется. |

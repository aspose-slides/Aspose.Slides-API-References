public interface ITextSearchOptions
```

Представляет параметры, которые могут использоваться для поиска текста в презентации, слайде или TextFrame.
## Методы

| Метод | Описание |
| --- | --- |
| [getCaseSensitive()](#getCaseSensitive--) | Установите true для поиска с учетом регистра, false — иначе. |
| [setCaseSensitive(boolean value)](#setCaseSensitive-boolean-) | Установите true для поиска с учетом регистра, false — иначе. |
| [getWholeWordsOnly()](#getWholeWordsOnly--) | Установите true для сопоставления только полных слов, false — иначе. |
| [setWholeWordsOnly(boolean value)](#setWholeWordsOnly-boolean-) | Установите true для сопоставления только полных слов, false — иначе. |
| [getIncludeNotes()](#getIncludeNotes--) | Установите true для включения текста, содержащегося в заметках слайда, при выполнении операций поиска текста, замены или подсветки. |
| [setIncludeNotes(boolean value)](#setIncludeNotes-boolean-) | Установите true для включения текста, содержащегося в заметках слайда, при выполнении операций поиска текста, замены или подсветки. |

### AccentProcess {#AccentProcess}
```
public static final int AccentProcess
```

Используется для отображения прогрессии, временной шкалы или последовательных шагов в задаче, процессе или рабочем потоке. Хорошо подходит для иллюстрации как текста уровня 1, так и уровня 2.
### AccentedPicture {#AccentedPicture}
```
public static final int AccentedPicture
```

public interface IHeadingPair
```

Представляет свойство 'Heading pair' документа. Оно указывает название группы частей документа и количество частей в группе.
## Методы

| Метод | Описание |
| --- | --- |
| [getName()](#getName--) | Returns the group name of document parts. |
| [getCount()](#getCount--) | Returns the number of parts in group. |
### getName() {#getName--}
```
public abstract String getName()
```

Возвращает название группы частей документа. Только для чтения String.

**Возвращаемое значение:**
java.lang.String
### getCount() {#getCount--}
```
public abstract int getCount()
```

Возвращает количество частей в группе. Только для чтения int.

**Возвращаемое значение:**
int

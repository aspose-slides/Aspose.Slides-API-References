Представляет тип агрегации оси категорий. Используется только с сериями Histogram или HistogramPareto.
## Поля

| Поле | Описание |
| --- | --- |
| [ByCategory](#ByCategory) | Агрегировать по категориям |
| [Automatic](#Automatic) | Агрегировать категории автоматически. |
| [ByBinWidth](#ByBinWidth) | Агрегировать категории по ширине бина. |
| [ByNumberOfBins](#ByNumberOfBins) | Агрегировать категории по количеству бинов. |

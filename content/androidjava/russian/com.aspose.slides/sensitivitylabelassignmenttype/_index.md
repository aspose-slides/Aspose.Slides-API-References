### Стандартный {#Standard}
```

```

Используется для любой метки чувствительности, которая не была напрямую применена пользователем. Это включает любые метки по умолчанию, автоматически применяемые метки.

### Привилегированный {#Privileged}
```
public static final int Privileged
```

Используется для любой метки чувствительности, которая была напрямую применена пользователем. Это включает любые вручную применённые метки чувствительности, а также рекомендованные или обязательные метки, или любую функцию, где пользователь решает, какую метку чувствительности применить.

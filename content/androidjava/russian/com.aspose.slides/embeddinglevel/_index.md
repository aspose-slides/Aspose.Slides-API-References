### Installable {#Installable}
```
public static final int Installable
```

Шрифты с этой настройкой указывают, что они могут быть встроены и постоянно установлены на удалённой системе приложением. Пользователь удалённой системы получает такие же права, обязанности и лицензии на этот шрифт, как у первоначального покупателя шрифта, и подчиняется тому же лицензионному соглашению конечного пользователя, авторским правам, патенту на дизайн и/или товарному знаку, что и оригинальный покупатель.

### Restricted {#Restricted}
```
public static final int Restricted
```

Шрифты, у которых установлен только этот бит, не должны быть изменены, встроены или обменены каким-либо образом без предварительного получения разрешения законного владельца.

### PreviewPrint {#PreviewPrint}
```
public static final int PreviewPrint
```

Когда этот бит установлен, шрифт может быть встроен и временно загружен на удалённую систему. Документы, содержащие шрифты Preview & Print, должны открываться в режиме только для чтения; к документу нельзя применять изменения.

### Editable {#Editable}
```
public static final int Editable
```

Когда этот бит установлен, шрифт может быть встроен, но должен быть установлен только временно на других системах. В отличие от шрифтов Preview & Print, документы, содержащие шрифты Editable, могут открываться для чтения, редактирование разрешено, и изменения могут быть сохранены.

### NoSubsetting {#NoSubsetting}
```
public static final int NoSubsetting
```

Когда этот бит установлен, шрифт не может быть субсетирован перед встраиванием. Другие ограничения встраивания, указанные в битах 0-3 и 9, также применяются.

### BitmapOnly {#BitmapOnly}
```
public static final int BitmapOnly
```

Когда этот бит установлен, могут быть встроены только растровые изображения, содержащиеся в шрифте. Данные контура встраиваться не могут. Если в шрифте нет растровых изображений, шрифт считается не встраиваемым, и службы встраивания завершатся с ошибкой.

Представляет тип точек разбиения во втором круге или столбце на диаграмме типа pie-of-pie или bar-of-pie.

## Поля

| Поле | Описание |
| --- | --- |
| [Default](#Default) | Указывает, что точки данных должны быть разделены с использованием механизма по умолчанию для этого типа диаграммы. |
| [Custom](#Custom) | Указывает, что точки данных должны быть разделены между кругом и второй диаграммой в соответствии со значениями Custom Split. |
| [ByPercentage](#ByPercentage) | Указывает, что точки данных должны быть разделены между кругом и второй диаграммой, помещая точки с процентом, меньшим чем Split Position, во вторую диаграмму. |
| [ByPos](#ByPos) | Указывает, что точки данных должны быть разделены между кругом и второй диаграммой, помещая последние Split Position точек данных во вторую диаграмму. |
| [ByValue](#ByValue) | Указывает, что точки данных должны быть разделены между кругом и второй диаграммой, помещая точки данных со значением, меньшим чем Split Position, во вторую диаграмму. |

Указывает набор предопределённых визуальных эффектов для отрисовки чернил. Каждый эффект соответствует текстуре или изображению, имитирующему стилизованный вид чернил. Эти значения можно использовать для настройки визуального стиля цифровых штрихов чернил во время отрисовки.
## Поля

| Поле | Описание |
| --- | --- |
| [NotDefined](#NotDefined) | Эффект не определён или неизвестен. |
| [Bronze](#Bronze) | Тёплая, коричневато-металлическая текстура, напоминающая бронзовые чернила. |
| [Galaxy](#Galaxy) | Яркая, мерцающая текстура, напоминающая галактику, с космическими оттенками. |
| [Gold](#Gold) | Яркая, металлическая золотая текстура, придающая штрихам чернил роскошный вид. |
| [Lava](#Lava) | Огонечная текстура, напоминающая расплавленную лаву, с красными и оранжевыми оттенками. |
| [Ocean](#Ocean) | Глубокая синяя, жидкостная текстура, имитирующая океанские волны или чернила на водной основе. |
| [RainbowGlitter](#RainbowGlitter) | Яркий, искрящееся радужное сияние, используемое для праздничных или живых штрихов чернил. |
| [RoseGold](#RoseGold) | Мягкое розово-золотое сочетание, похожее на розовое золото, для элегантных штрихов чернил. |
| [Silver](#Silver) | Холодная, металлическая серебряная текстура, имитирующая классические серебряные чернила. |

### getTextFrameForOverriding() {#getTextFrameForOverriding--}
```
public final ITextFrame getTextFrameForOverriding()
```

Может содержать форматированный текст. Если это свойство не равно null, то это отформатированное значение текста переопределяет автоматически генерируемый текст. Автогенерируемый текст является неявным свойством подписи данных, подписи единиц измерения оси значений, названия оси, заголовка диаграммы, подписи линии тренда. Автогенерируемый текст формируется с помощью свойства IFormattedTextContainer.TextFormat. Только чтение [ITextFrame](../../com.aspose.slides/itextframe).

**Возвращаемое значение:**
[ITextFrame](../../com.aspose.slides/itextframe)

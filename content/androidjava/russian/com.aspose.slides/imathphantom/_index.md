Представляет объект-призрак в математике (<m:phant>), который влияет на расположение своего дочернего элемента, не обязательно отображая его. Призрак может скрывать базовое выражение, при этом сохранять его ширину, высоту или глубину для выравнивания формул или резервирования места. Видимость и геометрическое поведение контролируются свойствами, такими как Show, ZeroWid, ZeroAsc, ZeroDesc и Transp.

--------------------

> ```
> Example:
>  
>  IMathPhantom phantom = new MathPhantom(new MathematicalText("1/2"));
>  phantom.setShow(false);          // Скрыть содержимое
>  phantom.setZeroWidth(false);     // Сохранить ширину
> ```

### getTransp() {#getTransp--}
```
public abstract boolean getTransp()
```

Получает или задает значение, указывающее, является ли призрак прозрачным для правил интервалов, основанных на классах.

--------------------

Когда true, операторы и символы внутри призрака всё равно влияют на математические интервалы вокруг него (как если бы он был видим). Когда false, правила интервалов, основанные на классах, игнорируются. Соответствует атрибуту OMML m:transp.

**Возвращаемое значение:**
boolean

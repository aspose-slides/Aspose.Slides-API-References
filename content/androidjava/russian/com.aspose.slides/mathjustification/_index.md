Указывает выравнивание математического абзаца (серии соседних экземпляров математического текста внутри одного абзаца)
## Поля

| Поле | Описание |
| --- | --- |
| [LeftJustified](#LeftJustified) | Выровнять математический текст по левому краю |
| [RightJustified](#RightJustified) | Выровнять математический текст по правому краю |
| [Centered](#Centered) | Выровнять математический текст по центру |
| [CenteredAsGroup](#CenteredAsGroup) | Разместить по центру страницы всю группу математических блоков как единое целое. |

### CenteredAsGroup {#CenteredAsGroup}
```

```

Разместить по центру страницы всю группу математических блоков как единое целое. Это означает, что экземпляры математического текста могут быть выровнены друг относительно друга, но вся группа математического текста центрируется в целом.

```
public interface IAlphaCeiling extends IImageTransformOperation, IAccessiblePVIObject<IAlphaCeilingEffectiveData>
```

Представляет эффект Alpha Ceiling. Значения Alpha (непрозрачности), превышающие ноль, изменяются на 100%. Иными словами, всё частично непрозрачное становится полностью непрозрачным.

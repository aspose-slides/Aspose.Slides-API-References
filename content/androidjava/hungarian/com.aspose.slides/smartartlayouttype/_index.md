### AccentProcess {#AccentProcess}
```
public static final int AccentProcess
```

Használja haladást, idővonalat vagy szekvenciális lépéseket egy feladat, folyamat vagy munkafolyamat során. Jól működik a Level 1 és Level 2 szöveg együttes ábrázolásához.

### AccentedPicture {#AccentedPicture}
```
public static final int AccentedPicture
```

Használja központi, fényképes ötletet, a szélén kapcsolódó ötletekkel. A felső Level 1 szöveg a központi képre kerül. A többi Level 1 forma megfelelő szövege a kis kör alakú képek mellé helyeződik. Ez az elrendezés szöveg nélkül is jól működik.

### AlternatingFlow {#AlternatingFlow}
```
public static final int AlternatingFlow
```

Használja információcsoportok vagy szekvenciális lépések ábrázolására egy feladat, folyamat vagy munkafolyamat során. Hangsúlyozza a csoportok közti interakciót vagy kapcsolatot.

### AlternatingHexagons {#AlternatingHexagons}
```
public static final int AlternatingHexagons
```

Használja összekapcsolt ötletek sorozatának ábrázolására. A Level 1 szöveg a hatszögek belsejében jelenik meg. A Level 2 szöveg a formák kívül helyezkedik el.

### AlternatingPictureBlocks {#AlternatingPictureBlocks}
```
public static final int AlternatingPictureBlocks
```

Használja képek felső-alsó sorozatának megjelenítésére. A szöveg váltakozva a kép jobb vagy bal oldalán jelenik meg.

### AlternatingPictureCircles {#AlternatingPictureCircles}
```
public static final int AlternatingPictureCircles
```

Használja képek és szöveg egy csoportjának megjelenítésére. A megfelelő szöveg a központi körökben jelenik meg, a képek bal-jobbra váltakoznak.

### ArrowRibbon {#ArrowRibbon}
```
public static final int ArrowRibbon
```

Használja kapcsolódó vagy kontrasztos koncepciók, például ellentétes erők ábrázolására. Az első két Level 1 szöveg a nyilak szövegeként jelenik meg. A nem használt szöveg nem látható, de elérhető a sablonváltáskor.

### AscendingPictureAccentProcess {#AscendingPictureAccentProcess}
```
public static final int AscendingPictureAccentProcess
```

Használja a képek felfelé növekvő sorozatát leíró szöveggel. Legjobb kis mennyiségű szöveg esetén.

### Balance {#Balance}
```
public static final int Balance
```

Használja két ötlet összehasonlítására vagy kapcsolatuk ábrázolására. Az első két Level 1 szöveg a középpont két oldalának tetejére kerül. Hangsúlyozza a Level 2 szöveget, amely legfeljebb négy formán jelenik meg a középpont mindkét oldalán. A kiegyensúlyozottság a legtöbb Level 2 szöveget tartalmazó oldal felé hajlik. A nem használt szöveg nem látható, de elérhető a sablonváltáskor.

### BasicBendingProcess {#BasicBendingProcess}
```
public static final int BasicBendingProcess
```

Használja haladást vagy szekvenciális lépéseket egy feladat, folyamat vagy munkafolyamat során. Maximálja a formák vízszintes és függőleges helyét.

### BasicBlockList {#BasicBlockList}

```
public static final int BasicBlockList
```

### BasicChevronProcess {#BasicChevronProcess}

```
public static final int BasicChevronProcess
```

### BasicCycle {#BasicCycle}

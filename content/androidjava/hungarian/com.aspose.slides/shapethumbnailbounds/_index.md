### Appearance {#Appearance}
```
public static final int Appearance
```

A forma bélyegkép mérete megegyezik a forma megjelenésével (a dia határain belül). Előfordulhat, hogy a forma megjelenése nem illeszkedik a forma határaihoz. Például forgatás, éles szögű gerenda illesztése, 3D hatások stb.

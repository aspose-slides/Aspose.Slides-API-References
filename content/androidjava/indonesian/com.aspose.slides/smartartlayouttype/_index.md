### AccentProcess {#AccentProcess}
```
public static final int AccentProcess
```

Gunakan untuk menampilkan kemajuan, garis waktu, atau langkah berurutan dalam tugas, proses, atau alur kerja. Berfungsi baik untuk mengilustrasikan teks Tingkat 1 dan Tingkat 2.

### AccentedPicture {#AccentedPicture}
```
public static final int AccentedPicture
```

Gunakan untuk menampilkan ide foto pusat dengan ide terkait di sebelahnya. Teks Tingkat 1 teratas muncul di atas gambar pusat. Teks yang sesuai untuk bentuk Tingkat 1 lain muncul di samping gambar lingkaran kecil. Tata letak ini juga berfungsi baik tanpa teks.

### AlternatingFlow {#AlternatingFlow}
```
public static final int AlternatingFlow
```

Gunakan untuk menampilkan grup informasi atau langkah berurutan dalam tugas, proses, atau alur kerja. Menekankan interaksi atau hubungan di antara grup informasi.

### AlternatingHexagons {#AlternatingHexagons}
```
public static final int AlternatingHexagons
```

Gunakan untuk merepresentasikan rangkaian ide yang saling terhubung. Teks Tingkat 1 muncul di dalam heksagon. Teks Tingkat 2 muncul di luar bentuk.

### AlternatingPictureBlocks {#AlternatingPictureBlocks}
```
public static final int AlternatingPictureBlocks

```


Gunakan untuk menampilkan sekumpulan gambar dengan teks. Teks yang sesuai muncul di lingkaran pusat dengan gambar yang bergantian dari kiri ke kanan.

### ArrowRibbon {#ArrowRibbon}
```
public static final int ArrowRibbon
```

Gunakan untuk menampilkan konsep terkait atau kontras dengan beberapa keterkaitan, seperti gaya yang berlawanan. Dua baris pertama teks Tingkat 1 digunakan untuk teks pada panah. Teks yang tidak dipakai tidak muncul, tetapi tetap tersedia bila Anda mengganti tata letak.

### AscendingPictureAccentProcess {#AscendingPictureAccentProcess}
```
public static final int AscendingPictureAccentProcess
```

### Balance {#Balance}
```
public static final int Balance
```

Gunakan untuk membandingkan atau menampilkan hubungan antara dua ide. Setiap dua baris pertama teks Tingkat 1 sesuai dengan teks di atas satu sisi titik tengah. Menekankan teks Tingkat 2, yang terbatas pada empat bentuk di setiap sisi titik tengah. Keseimbangan condong ke sisi dengan bentuk paling banyak yang berisi teks Tingkat 2. Teks yang tidak dipakai tidak muncul, tetapi tetap tersedia bila Anda mengganti tata letak.

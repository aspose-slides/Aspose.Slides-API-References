## Bidang

| Bidang | Deskripsi |

| [Custom](#Custom) | Menentukan bahwa titik data akan dipisah antara pai dan diagram kedua sesuai nilai Custom Split. |
| [ByPercentage](#ByPercentage) | Menentukan bahwa titik data akan dipisah antara pai dan diagram kedua dengan menempatkan titik dengan persentase kurang dari Split Position persen di diagram kedua. |

| [ByValue](#ByValue) | Menentukan bahwa titik data akan dipisah antara pai dan diagram kedua dengan menempatkan titik data dengan nilai kurang dari Split Position di diagram kedua. |

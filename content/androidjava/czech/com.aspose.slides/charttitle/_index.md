### getTextFrameForOverriding() {#getTextFrameForOverriding--}
```
public final ITextFrame getTextFrameForOverriding()
```


Může obsahovat formátovaný text. Pokud není tato vlastnost null, tento formátovaný text přepíše automaticky generovaný text. Automaticky generovaný text je implicitní vlastností popisku dat, popisku jednotek osy hodnot, názvu osy, názvu grafu, popisku trendové čáry. Automaticky generovaný text je formátován pomocí vlastnosti IFormattedTextContainer.TextFormat. Pouze pro čtení [ITextFrame](../../com.aspose.slides/itextframe).

**Vrací:**
[ITextFrame](../../com.aspose.slides/itextframe)

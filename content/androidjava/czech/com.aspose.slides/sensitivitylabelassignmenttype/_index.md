## Pole

| Pole | Popis |

### Standardní {#Standard}
```
public static final int Standard
```

Použít pro jakýkoli citlivostní štítek, který nebyl uživatelem přímo aplikován. To zahrnuje všechny výchozí štítky, automaticky aplikované štítky.

### Privilegovaný {#Privileged}
```
public static final int Privileged
```

Použít pro jakýkoli citlivostní štítek, který byl uživatelem přímo aplikován. To zahrnuje všechny ručně aplikované citlivostní štítky, stejně jako doporučené nebo povinné označování nebo jakoukoli funkci, kde uživatel rozhoduje, který citlivostní štítek použít.

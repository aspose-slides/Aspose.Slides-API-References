### AccentProcess {#AccentProcess}
```
public static final int AccentProcess
```

Použít k zobrazení postupu, časové osy nebo sekvenčních kroků v úkolu, procesu nebo pracovním postupu. Dobře funguje pro ilustraci jak textu úrovně 1, tak úrovně 2.

### AccentedPicture {#AccentedPicture}
```
public static final int AccentedPicture
```

Použít k zobrazení centrální, fotografické myšlenky s souvisejícími myšlenkami po stranách. Vrchní text úrovně 1 je umístěn nad centrálním obrázkem. Odpovídající text pro ostatní tvary úrovně 1 se zobrazí vedle malých kulatých obrázků. Toto rozvržení také dobře funguje bez textu.

### AlternatingFlow {#AlternatingFlow}
```
public static final int AlternatingFlow
```

Použít k zobrazení skupin informací nebo sekvenčních kroků v úkolu, procesu nebo pracovním postupu. Zdůrazňuje interakci nebo vztahy mezi skupinami informací.

### AlternatingHexagons {#AlternatingHexagons}
```
public static final int AlternatingHexagons
```

Použít k zobrazení série propojených myšlenek. Text úrovně 1 se zobrazuje uvnitř hexagonů. Text úrovně 2 se zobrazuje vně tvarů.

### AlternatingPictureBlocks {#AlternatingPictureBlocks}
```
public static final int AlternatingPictureBlocks
```

Použít k zobrazení série obrázků shora dolů. Text se střídavě zobrazuje vpravo nebo vlevo od obrázku.

### AlternatingPictureCircles {#AlternatingPictureCircles}
```
public static final int AlternatingPictureCircles
```

Použít k zobrazení sady obrázků s textem. Odpovídající text se zobrazuje v centrálních kruzích s obrázky střídavě zleva doprava.

### ArrowRibbon {#ArrowRibbon}
```
public static final int ArrowRibbon
```

Použít k zobrazení souvisejících nebo kontrastních konceptů s jistým spojením, jako jsou protichůdné síly. První dva řádky textu úrovně 1 slouží jako text v šípách. Nepoužitý text se nezobrazuje, ale zůstává k dispozici při změně rozvržení.

### AscendingPictureAccentProcess {#AscendingPictureAccentProcess}
```
public static final int AscendingPictureAccentProcess
```

Použít k zobrazení vzestupné série obrázků s popisným textem. Nejlépe funguje při malém množství textu.

### Balance {#Balance}
```
public static final int Balance
```

Použít k porovnání nebo zobrazení vztahu mezi dvěma myšlenkami. Každý z prvních dvou řádků textu úrovně 1 odpovídá textu na horní části jedné strany středu. Zdůrazňuje text úrovně 2, který je omezen na čtyři tvary na každé straně středu. Vyvážení se nakloní ke straně s nejvíce tvary obsahujícími text úrovně 2. Nepoužitý text se nezobrazuje, ale zůstává k dispozici při změně rozvržení.

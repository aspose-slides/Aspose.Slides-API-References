**Dědičnost:**
java.lang.Object, com.aspose.ms.System.ValueType, com.aspose.ms.System.Enum
```
public final class InkEffectType extends System.Enum
```

Specifikuje sadu předdefinovaných vizuálních efektů pro vykreslování ink. Každý efekt odpovídá textuře nebo obrázku, který simuluje stylizovaný vzhled ink. Tyto hodnoty lze použít k přizpůsobení vizuálního stylu digitálních tahů ink během vykreslování.
## Pole

| Field | Description |
| --- | --- |
| [NotDefined](#NotDefined) | Efekt není definován nebo je neznámý. |
| [Bronze](#Bronze) | Teplá, hnědavá kovová textura připomínající bronzové ink. |
| [Galaxy](#Galaxy) | Barevná, třpytivá textura připomínající galaxii s kosmickými odstíny. |
| [Gold](#Gold) | Jasná, kovová zlatá textura, která dodává tahům ink luxusní vzhled. |
| [Lava](#Lava) | Ohnivá textura připomínající roztavenou lávu s červenými a oranžovými odstíny. |
| [Ocean](#Ocean) | Hluboká modrá, tekutá textura napodobující oceánské vlny nebo vodní ink. |
| [RainbowGlitter](#RainbowGlitter) | Barevný, zářivý duhový glitter efekt používaný pro slavnostní nebo živé tahy ink. |
| [RoseGold](#RoseGold) | Jemná růžovo-zlatá směs podobná rose gold pro elegantní tahy ink. |
| [Silver](#Silver) | Chladná, kovová stříbrná textura simulující klasický stříbrný ink. |

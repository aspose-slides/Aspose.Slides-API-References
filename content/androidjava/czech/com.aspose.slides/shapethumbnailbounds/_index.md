### Slide {#Slide}
```
public static final int Slide
```

Miniatura tvaru bude mít velikost rovnou velikosti snímku. Pozice tvaru bude uložena.

### Shape {#Shape}
```
public static final int Shape
```

Miniatura tvaru bude mít velikost rovnou obdélníku ohraničení tvaru s ohledem na nastavení obrysu tvaru.

### Appearance {#Appearance}
```
public static final int Appearance
```

Miniatura tvaru bude mít velikost rovnou vzhledu tvaru (v mezích snímku). Může se stát, že vzhled tvaru se nevejde do ohraničení tvaru. Např. rotace, šikmý spoj ostrého úhlu, 3D efekty atd.

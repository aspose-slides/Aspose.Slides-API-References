public interface INormalViewRestoredProperties
```

Gibt die Größe des Folienbereichs an ((Breite, wenn ein Kind von RestoredTop, Höhe, wenn ein Kind von RestoredLeft) der Normalansicht, wenn der Bereich eine variable wiederhergestellte Größe (weder minimiert noch maximiert) hat).

| Methode | Beschreibung |
| --- | --- |
| [getDimensionSize()](#getDimensionSize--) | Gibt die Größe des Folienbereichs an (Breite, wenn ein Kind von RestoredTop, Höhe, wenn ein Kind von RestoredLeft). |
| [setDimensionSize(float value)](#setDimensionSize-float-) | Gibt die Größe des Folienbereichs an (Breite, wenn ein Kind von RestoredTop, Höhe, wenn ein Kind von RestoredLeft). |
| [getAutoAdjust()](#getAutoAdjust--) | Gibt an, ob die Größe des Seiteninhaltsbereichs die neue Größe kompensieren soll, wenn das Fenster, das die Ansicht enthält, innerhalb der Anwendung neu dimensioniert wird Lese/Schreib boolean. |
| [setAutoAdjust(boolean value)](#setAutoAdjust-boolean-) | Gibt an, ob die Größe des Seiteninhaltsbereichs die neue Größe kompensieren soll, wenn das Fenster, das die Ansicht enthält, innerhalb der Anwendung neu dimensioniert wird Lese/Schreib boolean. |

public interface IHeadingPair
```

Stellt eine 'Heading pair'-Eigenschaft des Dokuments dar. Sie gibt den Gruppennamen der Dokumentteile und die Anzahl der Teile in der Gruppe an.
## Methoden

| Methode | Beschreibung |
| --- | --- |
| [getName()](#getName--) | Gibt den Gruppennamen der Dokumentteile zurück. |
| [getCount()](#getCount--) | Gibt die Anzahl der Teile in der Gruppe zurück. |
### getName() {#getName--}
```
public abstract String getName()
```


Gibt den Gruppennamen der Dokumentteile zurück. Nur lesbar String.

**Rückgabe:**
java.lang.String

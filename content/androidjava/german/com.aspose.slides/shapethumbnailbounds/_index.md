Aufzählung der Arten von Form-Miniaturgrenzen.
## Felder

| Feld | Beschreibung |
| --- | --- |
| [Slide](#Slide) | Die Form-Miniatur hat die Größe der Folie. |
| [Shape](#Shape) | Die Form-Miniatur hat die Größe des Begrenzungsrechtecks der Form, wobei die Umriss-Einstellungen der Form berücksichtigt werden. |

### Slide {#Slide}
```
public static final int Slide
```

Die Form-Miniatur hat die Größe der Folie. Die Position der Form wird gespeichert.

### Shape {#Shape}
```
public static final int Shape
```

Die Form-Miniatur hat die Größe des Begrenzungsrechtecks der Form, wobei die Umriss-Einstellungen der Form berücksichtigt werden.

### Appearance {#Appearance}
```
public static final int Appearance
```

Die Form-Miniatur hat die Größe des Aussehens der Form (innerhalb der Foliengrenzen). Es kann Fälle geben, in denen das Aussehen der Form nicht in die Formbegrenzung passt, z. B. Drehung, Gehrungs-Verbindung eines spitzen Winkels, 3D-Effekte usw.

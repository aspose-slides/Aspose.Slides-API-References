Stellt die Zuweisungsmethode für das Sensitivitätslabel dar.
## Felder

| Feld | Beschreibung |
| --- | --- |
| [Standard](#Standard) | Use for any sensitivity label that was not directly applied by the user. |
| [Privileged](#Privileged) | Use for any sensitivity label that was directly applied by the user. |
### Standard {#Standard}
```
public static final int Standard
```

Verwenden Sie für jedes Sensitivitätslabel, das nicht direkt vom Benutzer angewendet wurde. Dies schließt alle Standardkennzeichnungen sowie automatisch angewendete Kennzeichnungen ein.

### Privilegiert {#Privileged}
```
public static final int Privileged
```

Verwenden Sie für jedes Sensitivitätslabel, das direkt vom Benutzer angewendet wurde. Dies schließt alle manuell angewendeten Sensitivitätskennzeichnungen sowie empfohlene oder obligatorische Kennzeichnungen oder jede Funktion ein, bei der der Benutzer entscheidet, welches Sensitivitätslabel angewendet werden soll.

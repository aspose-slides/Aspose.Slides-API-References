Repräsentiert, welchen Sketch-Typ oder Effekt ein Shape zugewiesen bekommen hat.
## Felder

| Feld | Beschreibung |
| --- | --- |
| [NotDefined](#NotDefined) | Gibt an, dass ein Shape Sketch effect undefiniert ist. |
| [None](#None) | Gibt an, dass ein Shape keinen Sketch effect hat. |
| [Curved](#Curved) | Gibt an, dass ein Shape den Curved effect hat, der jede Kante des Shapes in eine große sanfte Kurve verwandelt. |
| [Freehand](#Freehand) | Gibt an, dass ein Shape den Freehand effect hat, der einer unvollkommen gezeichneten Linie am nächsten kommt. |
| [Scribble](#Scribble) | Gibt an, dass ein Shape den Scribble effect hat, der übertriebene Oszillation aufweist, als wäre er bewusst unordentlich gezeichnet. |

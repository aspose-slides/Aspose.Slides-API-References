Gibt die Ausrichtung des mathematischen Absatzes an (eine Serie benachbarter Instanzen von mathematischem Text innerhalb desselben Absatzes)
## Felder

| Feld | Beschreibung |
| --- | --- |

| [RightJustified](#RightJustified) | Mathematischen Text nach rechts ausrichten |
| [Centered](#Centered) | Mathematischen Text zentrieren |
| [CenteredAsGroup](#CenteredAsGroup) | Die gesamte Gruppe von Mathematikblöcken als Einheit auf der Seite zentrieren. |

### CenteredAsGroup {#CenteredAsGroup}
```
public static final int CenteredAsGroup
```

Die gesamte Gruppe von Mathematikblöcken als Einheit auf der Seite zentrieren. Das bedeutet, dass die Instanzen des mathematischen Textes zueinander ausgerichtet werden können, aber die gesamte Gruppe des mathematischen Textes als Ganzes zentriert ist.

Unveränderliches Objekt, das einen Alpha Floor-Effekt darstellt. Alpha (Deckkraft)-Werte, die kleiner als 100% sind, werden auf Null gesetzt. Mit anderen Worten wird alles, das teilweise transparent ist, vollständig transparent.

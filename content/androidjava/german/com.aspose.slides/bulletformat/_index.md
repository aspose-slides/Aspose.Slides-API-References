### applyDefaultParagraphIndentsShifts() {#applyDefaultParagraphIndentsShifts--}
```
public final void applyDefaultParagraphIndentsShifts()
```


Setzt Standard-Verschiebungen (ungleich Null) für den effektiven Absatz-Einzug und MarginLeft, wenn Aufzählungszeichen aktiviert sind (wie PowerPoint es tut, wenn Absatz-Aufzählungszeichen/Nummerierung aktiviert werden). Ist die Aufzählungszeichen-Funktion deaktiviert, werden Einzug und MarginLeft des Absatzes zurückgesetzt (wie PowerPoint es tut, wenn die Absatz-Aufzählungszeichen/Nummerierung deaktiviert werden). Einzugs-Verschiebungen werden in Bezug auf den aktuellen Aufzählungszeichen-Kontext angewendet – IBulletFormat.Type, .NumberedBulletStyle und FontHeight des ersten Textanteils. Nicht-null-Einzugs-Verschiebungen werden auf den effektiven Einzug und MarginLeft des aktuellen Absatzes angewendet (dadurch werden die Ergebniswerte zu lokalen Werten).
### getEffective() {#getEffective--}
```
public final IBulletFormatEffectiveData getEffective()
```


Ruft die effektiven Aufzählungszeichen-Formatierungsdaten mit angewandter Vererbung ab.

--------------------

> ```
> This example demonstrates getting some effective bullet format properties.
>  
>  Presentation pres = new Presentation("MyPresentation.pptx");
>  try {
>      IAutoShape shape = (IAutoShape)pres.getSlides().get_Item(0).getShapes().get_Item(0);
>      IBulletFormatEffectiveData effectiveBulletFormat = shape.getTextFrame().getParagraphs().get_Item(0).getParagraphFormat().getBullet().getEffective();
>      System.out.println("Bullet type: " + effectiveBulletFormat.getType());
>      if (effectiveBulletFormat.getType() == BulletType.Numbered)
>      {
>          System.out.println("Numbered style: " + effectiveBulletFormat.getNumberedBulletStyle());
>          System.out.println("Starting number: " + effectiveBulletFormat.getNumberedBulletStartWith());
>      }
>  } finally {
>      if (pres != null) pres.dispose();
>  }
> ```

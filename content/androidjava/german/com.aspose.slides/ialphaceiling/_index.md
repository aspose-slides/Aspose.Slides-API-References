```
public interface IAlphaCeiling extends IImageTransformOperation, IAccessiblePVIObject<IAlphaCeilingEffectiveData>
```

Stellt einen Alpha-Ceiling-Effekt dar. Alpha (Opazität)-Werte, die größer als null sind, werden auf 100 % geändert. Mit anderen Worten, alles, das teilweise undurchsichtig ist, wird vollständig undurchsichtig.

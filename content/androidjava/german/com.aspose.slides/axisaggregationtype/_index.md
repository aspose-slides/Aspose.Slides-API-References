Stellt den Aggregationstyp der Kategorie-Achse dar. Wird nur mit Histogram- oder HistogramPareto-Serien verwendet.
## Felder

| Feld | Beschreibung |
| --- | --- |
| [ByCategory](#ByCategory) | Nach Kategorien aggregieren |
| [Automatic](#Automatic) | Kategorien automatisch aggregieren. |
| [ByBinWidth](#ByBinWidth) | Kategorien nach Bin-Breite aggregieren. |
| [ByNumberOfBins](#ByNumberOfBins) | Kategorien nach Anzahl der Bins aggregieren. |
### ByCategory {#ByCategory}
```
public static final int ByCategory
```

Nach Kategorien aggregieren

### Automatic {#Automatic}
```
public static final int Automatic
```

Kategorien automatisch aggregieren.

### ByBinWidth {#ByBinWidth}
```
public static final int ByBinWidth
```

Kategorien nach Bin-Breite aggregieren. Die Eigenschaft IAxis.BinWidth muss gesetzt werden.

### ByNumberOfBins {#ByNumberOfBins}
```
public static final int ByNumberOfBins
```

Kategorien nach Anzahl der Bins aggregieren. Die Eigenschaft IAxis.NumberOfBins muss gesetzt werden.

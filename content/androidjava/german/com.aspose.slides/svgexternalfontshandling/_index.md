Stellt eine Möglichkeit dar, externe Schriftarten für die Textdarstellung zu verwalten.
## Felder

| Feld | Beschreibung |
| --- | --- |
| [AddLinksToFontFiles](#AddLinksToFontFiles) | Fügen Sie Links zu separaten Schriftdateien zum Style-Abschnitt der SVG-Datei hinzu. |
| [Embed](#Embed) | Speichern Sie Schriftartdaten direkt in der SVG-Datei. |
| [Vectorize](#Vectorize) | Speichern Sie den gesamten Text, der externe Schriftarten verwendet, als Grafiken. |
### AddLinksToFontFiles {#AddLinksToFontFiles}

```

```


Speichern Sie Schriftartdaten direkt in der SVG-Datei. Bitte überprüfen Sie alle Lizenzvereinbarungen der externen Schriftarten, bevor Sie diese Option verwenden.

### Vectorize {#Vectorize}
```
public static final int Vectorize
```

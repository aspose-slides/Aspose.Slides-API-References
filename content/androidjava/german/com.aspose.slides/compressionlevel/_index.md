Gibt ZIP-Komprimierungsstufen für OpenXML-Dateien an. Höhere Stufen bieten bessere Kompression auf Kosten einer langsameren Verarbeitung.
## Felder

| Feld | Beschreibung |
| --- | --- |
| [None](#None) | Keine Kompression wird angewendet. |
| [Level1](#Level1) | Schnellste Kompression mit dem niedrigsten Kompressionsverhältnis. |
| [Level2](#Level2) | Schnellere Kompression mit leicht besserem Kompressionsverhältnis als [Level1](../../com.aspose.slides/compressionlevel\#Level1). |
| [Level3](#Level3) | Bietet bessere Kompression als [Level2](../../com.aspose.slides/compressionlevel\#Level2) bei moderatem Leistungseinfluss. |
| [Level4](#Level4) | Bietet bessere Kompression als [Level3](../../com.aspose.slides/compressionlevel\#Level3). |
| [Level5](#Level5) | Bietet verbesserte Kompression gegenüber [Level4](../../com.aspose.slides/compressionlevel\#Level4) mit zusätzlicher Verarbeitungszeit. |
| [Level6](#Level6) | Standardkompression, die ein gutes Gleichgewicht zwischen Kompressionsgeschwindigkeit und Dateigröße bietet. |
| [Level7](#Level7) | Bietet höhere Kompression als [Level6](../../com.aspose.slides/compressionlevel\#Level6) bei langsamerer Verarbeitung. |
| [Level8](#Level8) | Bietet höhere Kompression als [Level7](../../com.aspose.slides/compressionlevel\#Level7). |
| [Level9](#Level9) | Maximale Kompression. |

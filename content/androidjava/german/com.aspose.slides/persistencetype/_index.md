Gibt die Methode an, die zum Speichern von Eigenschaften des ActiveX-Steuerelements verwendet wird.

## Felder

| Feld | Beschreibung |
| --- | --- |
| [NotDefined](#NotDefined) | Persistenz-ID nicht angegeben. |
| [PersistPropertyBag](#PersistPropertyBag) | Gibt an, dass das ActiveX-Steuerelement mittels property-bag-basierter Persistenz gespeichert wird. |
| [PersistStream](#PersistStream) | Gibt an, dass das ActiveX-Steuerelement mittels stream-basierter Persistenz gespeichert wird, die keine Initialisierung des ActiveX-Steuerelements in den Standardzustand unterstützt. |
| [PersistStreamInit](#PersistStreamInit) | Gibt an, dass das ActiveX-Steuerelement mittels stream-basierter Persistenz gespeichert wird, die die Initialisierung des ActiveX-Steuerelements in den Standardzustand unterstützt. |
| [PersistStorage](#PersistStorage) | Gibt an, dass das ActiveX-Steuerelement mittels storage-basierter Persistenz gespeichert wird. |

### NotDefined {#NotDefined}
```
public static final int NotDefined
```

Persistenz-ID nicht angegeben.

### PersistPropertyBag {#PersistPropertyBag}
```
public static final int PersistPropertyBag
```

Gibt an, dass das ActiveX-Steuerelement mittels property-bag-basierter Persistenz gespeichert wird. Property-bag-basierte Persistenz speichert ein ActiveX-Steuerelement mittels einer Sammlung von Namens- und Wertpaaren, die die vom ActiveX-Steuerelement persistierten Daten festlegen.

### PersistStream {#PersistStream}
```
public static final int PersistStream
```

Gibt an, dass das ActiveX-Steuerelement mittels stream-basierter Persistenz gespeichert wird, die keine Initialisierung des ActiveX-Steuerelements in den Standardzustand unterstützt.

### PersistStreamInit {#PersistStreamInit}
```
public static final int PersistStreamInit
```

Gibt an, dass das ActiveX-Steuerelement mittels stream-basierter Persistenz gespeichert wird, die die Initialisierung des ActiveX-Steuerelements in den Standardzustand unterstützt.

### PersistStorage {#PersistStorage}
```
public static final int PersistStorage
```

Gibt an, dass das ActiveX-Steuerelement mittels storage-basierter Persistenz gespeichert wird.

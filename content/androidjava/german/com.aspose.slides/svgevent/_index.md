## Felder

| Feld | Beschreibung |
| --- | --- |
| [OnFocusIn](#OnFocusIn) | Tritt auf, wenn ein Element den Fokus erhält, z. B. wenn ein Text ausgewählt wird. |
| [OnFocusOut](#OnFocusOut) | Tritt auf, wenn ein Element den Fokus verliert, z. B. wenn ein Text die Auswahl verliert. |
| [OnActivate](#OnActivate) | Tritt auf, wenn ein Element aktiviert wird, beispielsweise durch einen Mausklick oder einen Tastendruck. |
| [OnClick](#OnClick) | Tritt auf, wenn die Maustaste über ein Element geklickt wird. |
| [OnMouseDown](#OnMouseDown) | Tritt auf, wenn die Maustaste über ein Element gedrückt wird. |

| [OnMouseMove](#OnMouseMove) | Tritt auf, wenn das Zeigegerät über ein Element bewegt wird. |
| [OnMouseOut](#OnMouseOut) | Tritt auf, wenn das Zeigegerät von einem Element wegbewegt wird. |
| [OnLoad](#OnLoad) | Tritt auf, wenn der User-Agent das Element, seine Nachkommen und alle referenzierten Ressourcen, die zum Rendern erforderlich sind, vollständig geparst hat. |
| [OnUnload](#OnUnload) | Tritt auf, wenn die DOM-Implementierung ein Dokument aus einem Fenster oder Frame entfernt. |
| [OnAbort](#OnAbort) | Tritt auf, wenn das Laden der Seite gestoppt wird, bevor ein Element vollständig geladen werden konnte. |
| [OnError](#OnError) | Tritt auf, wenn ein Element nicht korrekt geladen wird oder ein Fehler während der Skriptausführung auftritt. |
| [OnResize](#OnResize) | Tritt auf, wenn die Ansicht eines Dokuments in der Größe geändert wird. |
| [OnScroll](#OnScroll) | Tritt auf, wenn die Ansicht eines Dokuments entlang der X-, Y- oder beider Achsen verschoben wird. |
| [OnZoom](#OnZoom) | Tritt auf, wenn die Zoom-Stufe einer Dokumentenansicht geändert wird. |
| [OnBegin](#OnBegin) | Tritt auf, wenn ein Animationselement startet. |
| [OnEnd](#OnEnd) | Tritt auf, wenn ein Animationselement endet. |
| [OnRepeat](#OnRepeat) | Tritt auf, wenn ein Animationselement wiederholt wird. |

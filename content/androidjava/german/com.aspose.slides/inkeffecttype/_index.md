**Vererbung:**
java.lang.Object, com.aspose.ms.System.ValueType, com.aspose.ms.System.Enum
```
public final class InkEffectType extends System.Enum
```

Gibt einen Satz vordefinierter visueller Effekte für die Tintendarstellung an. Jeder Effekt entspricht einer Textur oder einem Bild, das ein stilisiertes Tintenaussehen simuliert. Diese Werte können verwendet werden, um den visuellen Stil digitaler Tintestriche während der Darstellung anzupassen.

## Felder

| Feld | Beschreibung |
| --- | --- |
| [NotDefined](#NotDefined) | Der Effekt ist nicht definiert oder unbekannt. |
| [Bronze](#Bronze) | Eine warme, bräunlich-metallische Textur, die bronzene Tinte ähnelt. |
| [Galaxy](#Galaxy) | Eine farbenfrohe, schimmernde Textur, die einer Galaxie ähnelt, mit kosmischen Tönen. |
| [Gold](#Gold) | Eine helle, metallische Goldtextur, die Tintestrichen ein luxuriöses Aussehen verleiht. |
| [Lava](#Lava) | Eine feurige Textur, die geschmolzene Lava ähnelt, mit roten und orangefarbenen Nuancen. |
| [Ocean](#Ocean) | Eine tiefblaue, flüssigkeitsähnliche Textur, die Meereswellen oder wasserbasierte Tinte nachahmt. |
| [RainbowGlitter](#RainbowGlitter) | Ein farbenfroher, funkelnder Regenbogenglitzer-Effekt, der für festliche oder lebendige Tintestriche verwendet wird. |
| [RoseGold](#RoseGold) | Eine weiche pink-goldene Mischung, ähnlich wie Roségold, für elegante Tintestriche. |
| [Silver](#Silver) | Eine kühle, metallische Silbertextur, die klassische Silbertinte simuliert. |

public interface IErrorBarsCustomValues
```

Gibt die Fehlerbalkenwerte an. Sie darf nur verwendet werden, wenn der Fehlerbalkenwerttyp Custom ist.

## Methoden

| Method | Description |
| --- | --- |
| [getXMinus()](#getXMinus--) | Gibt den Fehlerbalkenwert in der negativen Richtung an. |
| [getYMinus()](#getYMinus--) | Gibt den Fehlerbalkenwert in der negativen Richtung an. |
| [getXPlus()](#getXPlus--) | Gibt den Fehlerbalkenwert in der positiven Richtung an. |
| [getYPlus()](#getYPlus--) | Gibt den Fehlerbalkenwert in der positiven Richtung an. |
### getXMinus() {#getXMinus--}
```
public abstract IDoubleChartValue getXMinus()
```

Gibt den Fehlerbalkenwert in der negativen Richtung an. Verfügbar, wenn der Fehlerbalkenwerttyp Custom ist und ErrorBarsXFormat zulässig ist. In allen anderen Fällen gibt diese Eigenschaft null zurück. Nur lesbar [IDoubleChartValue](../../com.aspose.slides/idoublechartvalue).

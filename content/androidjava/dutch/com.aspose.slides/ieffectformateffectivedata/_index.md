Onveranderlijk object dat effectieve effectopmaak-eigenschappen bevat.

--------------------

Deze interface wordt samen met de [IEffectFormat](../../com.aspose.slides/ieffectformat) interface gebruikt om effectieve opmaakwaarden met toegepaste overerving te retourneren.

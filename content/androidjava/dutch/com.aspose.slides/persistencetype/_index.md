Specificeert de methode die wordt gebruikt om eigenschappen van het ActiveX-control op te slaan.

## Velden

| Veld | Beschrijving |
| --- | --- |
| [NotDefined](#NotDefined) | Persistentie-id niet gespecificeerd. |
| [PersistPropertyBag](#PersistPropertyBag) | Specificeert dat het ActiveX-control wordt bewaard met property-bag-gebaseerde persistentie. |
| [PersistStream](#PersistStream) | Specificeert dat het ActiveX-control wordt bewaard met een op stream gebaseerde persistentie die geen initialisatie van het ActiveX-control naar een standaardtoestand ondersteunt. |
| [PersistStreamInit](#PersistStreamInit) | Specificeert dat het ActiveX-control wordt bewaard met een op stream gebaseerde persistentie die initialisatie van het ActiveX-control naar een standaardtoestand ondersteunt. |
| [PersistStorage](#PersistStorage) | Specificeert dat het ActiveX-control wordt bewaard met opslag-gebaseerde persistentie. |

### NotDefined {#NotDefined}
```
public static final int NotDefined
```

Persistentie-id niet gespecificeerd.

### PersistPropertyBag {#PersistPropertyBag}
```
public static final int PersistPropertyBag
```

Specificeert dat het ActiveX-control wordt bewaard met property-bag-gebaseerde persistentie. Property-bag-gebaseerde persistentie slaat een ActiveX-control op door middel van een verzameling van naam-en-waarde-paren die de door het ActiveX-control bewaarde gegevens definiëren.

### PersistStream {#PersistStream}
```
public static final int PersistStream
```

Specificeert dat het ActiveX-control wordt bewaard met een op stream gebaseerde persistentie die geen initialisatie van het ActiveX-control naar een standaardtoestand ondersteunt.

### PersistStreamInit {#PersistStreamInit}
```
public static final int PersistStreamInit
```

Specificeert dat het ActiveX-control wordt bewaard met een op stream gebaseerde persistentie die initialisatie van het ActiveX-control naar een standaardtoestand ondersteunt.

### PersistStorage {#PersistStorage}
```
public static final int PersistStorage
```

Specificeert dat het ActiveX-control wordt bewaard met opslag-gebaseerde persistentie.

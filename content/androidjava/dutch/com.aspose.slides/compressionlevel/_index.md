Specificeert ZIP-compressieniveaus voor OpenXML-bestand. Hogere niveaus bieden betere compressie ten koste van tragere verwerking.
## Velden

| Veld | Beschrijving |
| --- | --- |
| [None](#None) | Er wordt geen compressie toegepast. |
| [Level1](#Level1) | Snelste compressie met de laagste compressieverhouding. |
| [Level2](#Level2) | Snellere compressie met iets betere compressieverhouding dan [Level1](../../com.aspose.slides/compressionlevel\#Level1). |
| [Level3](#Level3) | Biedt betere compressie dan [Level2](../../com.aspose.slides/compressionlevel\#Level2) met een matige impact op de prestaties. |
| [Level4](#Level4) | Biedt betere compressie dan [Level3](../../com.aspose.slides/compressionlevel\#Level3). |
| [Level5](#Level5) | Biedt verbeterde compressie ten opzichte van [Level4](../../com.aspose.slides/compressionlevel\#Level4) met extra verwerkingstijd. |
| [Level6](#Level6) | Standaardcompressie, die een goede balans biedt tussen compressiesnelheid en bestandsgrootte. |
| [Level7](#Level7) | Biedt hogere compressie dan [Level6](../../com.aspose.slides/compressionlevel\#Level6) met tragere verwerking. |
| [Level8](#Level8) | Biedt hogere compressie dan [Level7](../../com.aspose.slides/compressionlevel\#Level7). |
| [Level9](#Level9) | Maximale compressie. |

public interface IErrorBarsCustomValues
```

Specificeert de waarden van de foutbalken. Deze mag alleen worden gebruikt wanneer het type foutbalkwaarde Custom is.

| [getXPlus()](#getXPlus--) | Specificeert de foutbalkwaarde in de positieve richting. |
| [getYPlus()](#getYPlus--) | Specificeert de foutbalkwaarde in de positieve richting. |

### getXMinus() {#getXMinus--}
```
public abstract IDoubleChartValue getXMinus()
```

Specificeert de foutbalkwaarde in de negatieve richting. Beschikbaar als het type foutbalkwaarde Custom is en ErrorBarsXFormat is toegestaan. In elk ander geval retourneert deze eigenschap null. Alleen-lezen [IDoubleChartValue](../../com.aspose.slides/idoublechartvalue).

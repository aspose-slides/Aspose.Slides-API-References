```
public final class InkEffectType extends System.Enum
```

Specificeert een reeks vooraf gedefinieerde visuele effecten voor inktweergave. Elk effect komt overeen met een textuur of afbeelding die een gestileerde inktuiterlijk simuleert. Deze waarden kunnen worden gebruikt om de visuele stijl van digitale inktstreken tijdens het renderen aan te passen.

## Velden

| Veld | Beschrijving |
| --- | --- |
| [NotDefined](#NotDefined) | Het effect is niet gedefinieerd of is onbekend. |
| [Bronze](#Bronze) | Een warme, bruinachtige metalen textuur die lijkt op bronzen inkt. |
| [Galaxy](#Galaxy) | Een kleurrijke, glinsterende textuur die lijkt op een sterrenstelsel, met kosmische tonen. |
| [Gold](#Gold) | Een heldere, metalen gouden textuur die inktstreken een luxueuze uitstraling geeft. |
| [Lava](#Lava) | Een vurig textuur die lijkt op gesmolten lava, met rode en oranje tinten. |
| [Ocean](#Ocean) | Een diepblauwe, vloeistof-achtige textuur die oceaangolven of waterachtige inkt nabootst. |
| [RainbowGlitter](#RainbowGlitter) | Een kleurrijk, sprankelend regenboogglittereffect dat wordt gebruikt voor feestelijke of levendige inktstreken. |
| [RoseGold](#RoseGold) | Een zachte roze-gouden mix, vergelijkbaar met rozen goud, voor elegante inktstreken. |
| [Silver](#Silver) | Een koele, metalen zilverkleurige textuur die klassieke zilveren inkt simuleert. |

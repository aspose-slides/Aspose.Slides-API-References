### Standaard {#Standard}
```
public static final int Standard
```


Gebruik voor elk gevoeligheidslabel dat niet rechtstreeks door de gebruiker is toegepast. Dit omvat alle standaardlabels, automatisch toegepaste labels.

### Bevoorrecht {#Privileged}
```
public static final int Privileged
```


Gebruik voor elk gevoeligheidslabel dat rechtstreeks door de gebruiker is toegepast. Dit omvat alle handmatig toegepaste gevoeligheidslabels evenals aanbevolen of verplichte labeling of elke functie waarbij de gebruiker beslist welk gevoeligheidslabel toe te passen.

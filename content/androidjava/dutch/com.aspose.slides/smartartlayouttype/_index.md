### AccentProcess {#AccentProcess}
```
public static final int AccentProcess
```

Gebruik om een voortgang, een tijdlijn of opeenvolgende stappen in een taak, proces of workflow weer te geven. Werkt goed voor het illustreren van zowel Niveau 1- als Niveau 2-tekst.

### AccentedPicture {#AccentedPicture}
```
public static final int AccentedPicture
```

Gebruik om een centraal, fotografisch idee met gerelateerde ideeën aan de zijkant weer te geven. De bovenste Niveau 1-tekst verschijnt boven de centrale afbeelding. Bijbehorende tekst voor andere Niveau 1-vormen verschijnt naast de kleine ronde afbeeldingen. Deze lay-out werkt ook goed zonder tekst.

### AlternatingFlow {#AlternatingFlow}
```
public static final int AlternatingFlow
```

Gebruik om groepen informatie of opeenvolgende stappen in een taak, proces of workflow weer te geven. Benadrukt de interactie of relaties tussen de groepen informatie.

### AlternatingHexagons {#AlternatingHexagons}
```
public static final int AlternatingHexagons
```

Gebruik om een reeks onderling verbonden ideeën weer te geven. Niveau 1-tekst verschijnt binnen de zeshoeken. Niveau 2-tekst verschijnt buiten de vormen.

### AlternatingPictureBlocks {#AlternatingPictureBlocks}
```
public static final int AlternatingPictureBlocks
```

Gebruik om een reeks afbeeldingen van boven naar beneden weer te geven. Tekst verschijnt afwisselend rechts of links van de afbeelding.

### AlternatingPictureCircles {#AlternatingPictureCircles}
```
public static final int AlternatingPictureCircles
```

Gebruik om een set afbeeldingen met tekst weer te geven. De bijbehorende tekst verschijnt in de centrale cirkels met de afbeeldingen afwisselend van links naar rechts.

### ArrowRibbon {#ArrowRibbon}
```
public static final int ArrowRibbon
```

Gebruik om gerelateerde of contrasterende concepten met enige verbinding weer te geven, zoals tegengestelde krachten. De eerste twee regels Niveau 1-tekst worden gebruikt voor tekst in de pijlen. Ongebruikte tekst verschijnt niet, maar blijft beschikbaar als u van lay-out wisselt.

### AscendingPictureAccentProcess {#AscendingPictureAccentProcess}
```
public static final int AscendingPictureAccentProcess
```

Gebruik om een oplopende reeks afbeeldingen met beschrijvende tekst weer te geven. Werkt het beste met een kleine hoeveelheid tekst.

### Balance {#Balance}
```
public static final int Balance
```

Gebruik om twee ideeën te vergelijken of de relatie tussen twee ideeën weer te geven. Elk van de eerste twee regels Niveau 1-tekst komt overeen met tekst boven één kant van het middelpunt. Benadrukt Niveau 2-tekst, die beperkt is tot vier vormen aan elke kant van het middelpunt. Het evenwicht kantelt naar de kant met de meeste vormen met Niveau 2-tekst. Ongebruikte tekst verschijnt niet, maar blijft beschikbaar als u van lay-out wisselt.

public static final int BasicBendingProcess

### BasicPie {#BasicPie}
```
public static final int BasicPie

De bovenste Niveau 1-tekstvorm verschijnt buiten de rest van de taart voor nadruk. Ongebruikte tekst verschijnt niet, maar blijft beschikbaar als u van lay-out wisselt.

public interface IHeadingPair
```

문서의 'Heading pair' 속성을 나타냅니다. 이는 문서 부분의 그룹 이름과 그룹 내 부분 수를 나타냅니다.
## 메서드

| Method | Description |
| --- | --- |
| [getName()](#getName--) | 문서 부분의 그룹 이름을 반환합니다. |
| [getCount()](#getCount--) | 그룹 내 부분 수를 반환합니다. |
### getName() {#getName--}
```
public abstract String getName()
```

문서 부분의 그룹 이름을 반환합니다. 읽기 전용 String.

**반환:**
java.lang.String
### getCount() {#getCount--}
```
public abstract int getCount()
```

그룹 내 부분 수를 반환합니다. 읽기 전용 int.

**반환:**
int

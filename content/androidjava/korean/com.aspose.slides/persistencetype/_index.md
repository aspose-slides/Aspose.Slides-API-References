ActiveX control의 속성을 저장하는 데 사용되는 방법을 지정합니다.
## 필드

| 필드 | 설명 |
| --- | --- |
| [NotDefined](#NotDefined) | 영속성 ID가 지정되지 않았습니다. |

| [PersistStream](#PersistStream) | ActiveX control이 기본 상태로 초기화하는 것을 지원하지 않는 스트림 기반 영속성을 사용하여 지속된다는 것을 지정합니다. |
| [PersistStreamInit](#PersistStreamInit) | ActiveX control이 기본 상태로 초기화하는 것을 지원하는 스트림 기반 영속성을 사용하여 지속된다는 것을 지정합니다. |
| [PersistStorage](#PersistStorage) | ActiveX control이 스토리지 기반 영속성을 사용하여 지속된다는 것을 지정합니다. |
### NotDefined {#NotDefined}
```
public static final int NotDefined
```

영속성 ID가 지정되지 않았습니다.

### PersistPropertyBag {#PersistPropertyBag}
```
public static final int PersistPropertyBag
```

ActiveX control이 property-bag 기반 영속성을 사용하여 지속된다는 것을 지정합니다. Property-bag 기반 영속성은 이름과 값 쌍의 컬렉션을 통해 ActiveX control이 유지하는 데이터를 지정하여 ActiveX control을 저장합니다.

### PersistStream {#PersistStream}
```
public static final int PersistStream
```

ActiveX control이 기본 상태로 초기화하는 것을 지원하지 않는 스트림 기반 영속성을 사용하여 지속된다는 것을 지정합니다.

### PersistStreamInit {#PersistStreamInit}
```
public static final int PersistStreamInit
```

ActiveX control이 기본 상태로 초기화하는 것을 지원하는 스트림 기반 영속성을 사용하여 지속된다는 것을 지정합니다.

### PersistStorage {#PersistStorage}
```
public static final int PersistStorage
```

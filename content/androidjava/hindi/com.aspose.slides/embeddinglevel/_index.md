### Installable {#Installable}
```
public static final int Installable
```

इस सेटिंग वाले फ़ॉन्ट दर्शाते हैं कि उन्हें एम्बेड किया जा सकता है और किसी एप्लिकेशन द्वारा दूरस्थ प्रणाली में स्थायी रूप से स्थापित किया जा सकता है। दूरस्थ प्रणाली के उपयोगकर्ता को उस फ़ॉन्ट के लिए मूल खरीदार के समान अधिकार, दायित्व और लाइसेंस प्राप्त होते हैं, और वह मूल खरीदार की तरह ही अंतिम-उपयोगकर्ता लाइसेंस समझौता, कॉपीराइट, डिज़ाइन पेटेंट, और/या ट्रेडमार्क के अधीन होते हैं।

### Restricted {#Restricted}
```
public static final int Restricted
```

जिन फ़ॉन्ट में केवल यह बिट सेट है, उन्हें कानूनन मालिक की अनुमति प्राप्त किए बिना किसी भी तरीके से संशोधित, एम्बेड या आदान-प्रदान नहीं किया जाना चाहिए।

### PreviewPrint {#PreviewPrint}
```
public static final int PreviewPrint
```

जब यह बिट सेट हो, तो फ़ॉन्ट को एम्बेड किया जा सकता है और दूरस्थ प्रणाली पर अस्थायी रूप से लोड किया जा सकता है। Preview & Print फ़ॉन्ट वाले दस्तावेज़ को केवल-पढ़ने के मोड में खोलना चाहिए; दस्तावेज़ में कोई संशोधन नहीं किया जा सकता।

### Editable {#Editable}
```
public static final int Editable
```

जब यह बिट सेट हो, तो फ़ॉन्ट को एम्बेड किया जा सकता है लेकिन इसे अन्य प्रणालियों में केवल अस्थायी रूप से स्थापित किया जाना चाहिए। Preview & Print फ़ॉन्ट के विपरीत, Editable फ़ॉन्ट वाले दस्तावेज़ पढ़ने के लिए खुले जा सकते हैं, संपादन की अनुमति है, और परिवर्तन सहेजे जा सकते हैं।

### NoSubsetting {#NoSubsetting}
```
public static final int NoSubsetting
```

जब यह बिट सेट हो, तो फ़ॉन्ट को एम्बेड करने से पहले उसे उपसमुच्चय नहीं किया जा सकता। बिट 0-3 और 9 में निर्धारित अन्य एम्बेडिंग प्रतिबंध भी लागू होते हैं।

### BitmapOnly {#BitmapOnly}
```
public static final int BitmapOnly
```

जब यह बिट सेट हो, तो केवल फ़ॉन्ट में मौजूद बिटमैप को ही एम्बेड किया जा सकता है। कोई भी आउटलाइन डेटा एम्बेड नहीं किया जा सकता। यदि फ़ॉन्ट में कोई बिटमैप उपलब्ध नहीं है, तो फ़ॉन्ट को एम्बेड करने योग्य नहीं माना जाएगा और एम्बेडिंग सेवा विफल हो जाएगी।

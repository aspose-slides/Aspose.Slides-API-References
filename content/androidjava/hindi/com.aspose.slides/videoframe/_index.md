### getCaptionTracks() {#getCaptionTracks--}
```
public final ICaptionsCollection getCaptionTracks()
```

वीडियो फ्रेम से जुड़े बंद कैप्शन के संग्रह को प्राप्त करता है। यह प्रॉपर्टी केवल पढ़ने योग्य है और एक [ICaptionsCollection](../../com.aspose.slides/icaptionscollection) लौटाता है जिसमें सभी कैप्शन ट्रैक होते हैं।

--------------------

> ```
> Example:
>   
>  Presentation pres = new Presentation("video with captions.pptx");
>  try {
>      for (IShape shape : pres.getSlides().get_Item(0).getShapes())
>      {
>          if (!(shape instanceof IVideoFrame))
>              continue;
>          IVideoFrame videoFrame = (IVideoFrame) shape;
>          for (ICaptions captionTrack : videoFrame.getCaptionTracks())
>          {
>              // कैप्शन बाइनरी डेटा निकालता है और फ़ाइल में सहेजता है
>              FileOutputStream fos = new FileOutputStream(captionTrack.getCaptionId() + ".vtt");
>              fos.write(captionTrack.getBinaryData());
>              fos.close();
>          }
>      }
>  } finally {
>      if (pres != null) pres.dispose();
>  }
> ```

**वापसी:**
[ICaptionsCollection](../../com.aspose.slides/icaptionscollection)

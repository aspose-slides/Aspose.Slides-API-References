シェイプに割り当てられたスケッチのタイプまたはエフェクトを表します。
## フィールド

| フィールド | 説明 |
| --- | --- |
| [NotDefined](#NotDefined) | シェイプの Sketch エフェクトが未定義であることを指定します。 |
| [None](#None) | シェイプに Sketch エフェクトがないことを指定します。 |
| [Curved](#Curved) | シェイプに Curved エフェクトがあることを指定します。このエフェクトはシェイプの各エッジを大きく柔らかな曲線に変えます。 |
| [Freehand](#Freehand) | シェイプに Freehand エフェクトがあることを指定します。これは最も不完全に描かれた線に近いです。 |
| [Scribble](#Scribble) | シェイプに Scribble エフェクトがあることを指定します。これは意図的に乱雑に描かれたかのように誇張された揺れがあります。 |

シェイプに Scribble エフェクトがあることを指定します。これは意図的に乱雑に描かれたかのように誇張された揺れがあります。

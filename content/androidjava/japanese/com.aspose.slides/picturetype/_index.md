## フィールド

| フィールド | 説明 |
| --- | --- |
| [NotDefined](#NotDefined) | 未定義です。 |
| [Stack](#Stack) | 画像は、最長の積み重ね棒で最大15回繰り返すようにサイズが設定されます。 |
| [StackScale](#StackScale) | 画像は指定された単位数でサイズが設定され、棒の長さに合わせて繰り返されます。 |
| [Stretch](#Stretch) | 画像は積み重ね棒全体の長さに合わせて伸ばされます。 |

インク描画用に事前定義された視覚効果のセットを指定します。各効果は、スタイライズされたインク外観をシミュレートするテクスチャまたは画像に対応しています。これらの値は、描画時にデジタルインクの筆跡の視覚スタイルをカスタマイズするために使用できます。

## フィールド

| フィールド | 説明 |
| --- | --- |
| [NotDefined](#NotDefined) | 効果は定義されていないか不明です。 |
| [Bronze](#Bronze) | 青銅インクに似た、暖かく茶色がかった金属テクスチャです。 |
| [Galaxy](#Galaxy) | 銀河に似た、カラフルで輝くテクスチャ、宇宙的な色調です。 |
| [Gold](#Gold) | 鮮やかな金属金テクスチャで、インクの筆跡に豪華な外観を与えます。 |
| [Lava](#Lava) | 溶岩に似た炎のようなテクスチャで、赤とオレンジの色合いがあります。 |
| [Ocean](#Ocean) | 深い青色で、海の波や水性インクを模した流体のようなテクスチャです。 |
| [RainbowGlitter](#RainbowGlitter) | 祝祭的または鮮やかなインク筆跡に使用される、カラフルできらめくレインボーグリッター効果です。 |
| [RoseGold](#RoseGold) | ローズゴールドに似た、柔らかなピンクとゴールドのブレンドで、エレガントなインク筆跡に適しています。 |
| [Silver](#Silver) | 冷たい金属的なシルバーテクスチャで、クラシックなシルバーインクをシミュレートします。 |

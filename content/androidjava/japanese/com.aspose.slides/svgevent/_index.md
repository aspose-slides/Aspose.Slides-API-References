SVG シェイプのオプションを表します。

## フィールド

| フィールド | 説明 |
| --- | --- |
| [OnFocusIn](#OnFocusIn) | 要素がフォーカスを受け取ったときに発生します。たとえば、テキストが選択されたときです。 |
| [OnFocusOut](#OnFocusOut) | 要素がフォーカスを失ったときに発生します。たとえば、テキストの選択が解除されたときです。 |
| [OnActivate](#OnActivate) | 要素がアクティブ化されたときに発生します。たとえば、マウスクリックやキー入力によります。 |
| [OnClick](#OnClick) | 要素上でポインティングデバイスのボタンがクリックされたときに発生します。 |
| [OnMouseDown](#OnMouseDown) | 要素上でポインティングデバイスのボタンが押されたときに発生します。 |
| [OnMouseUp](#OnMouseUp) | 要素上でポインティングデバイスのボタンが離されたときに発生します。 |
| [OnMouseOver](#OnMouseOver) | ポインティングデバイスが要素上に移動したときに発生します。 |
| [OnMouseMove](#OnMouseMove) | ポインティングデバイスが要素上で移動したときに発生します。 |
| [OnMouseOut](#OnMouseOut) | ポインティングデバイスが要素から離れたときに発生します。 |
| [OnLoad](#OnLoad) | ユーザーエージェントが要素とその子孫、そして描画に必要なすべての参照リソースを完全に解析したときに発生します。 |
| [OnUnload](#OnUnload) | DOM の実装がウィンドウまたはフレームからドキュメントを削除したときに発生します。 |
| [OnAbort](#OnAbort) | 要素が完全にロードされる前にページの読み込みが停止されたときに発生します。 |
| [OnError](#OnError) | 要素が正しくロードされなかったり、スクリプト実行中にエラーが発生したときに発生します。 |
| [OnResize](#OnResize) | ドキュメントビューがサイズ変更されているときに発生します。 |
| [OnScroll](#OnScroll) | ドキュメントビューが X 軸または Y 軸、あるいは両方の軸に沿って移動しているときに発生します。 |
| [OnZoom](#OnZoom) | ドキュメントビューのズームレベルが変更されているときに発生します。 |
| [OnBegin](#OnBegin) | アニメーション要素が開始されたときに発生します。 |
| [OnEnd](#OnEnd) | アニメーション要素が終了したときに発生します。 |
| [OnRepeat](#OnRepeat) | アニメーション要素が繰り返されたときに発生します。 |

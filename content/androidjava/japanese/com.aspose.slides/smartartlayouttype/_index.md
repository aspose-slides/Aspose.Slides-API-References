### AccentProcess {#AccentProcess}
```
public static final int AccentProcess
```

タスク、プロセス、またはワークフローにおける進行、タイムライン、または順次ステップを示すために使用します。レベル 1 とレベル 2 のテキストの両方を示すのに適しています。

### AccentedPicture {#AccentedPicture}
```
public static final int AccentedPicture
```

中心の写真的アイデアと側面の関連アイデアを示すために使用します。上部のレベル 1 テキストは中心の画像の上に表示されます。他のレベル 1 形状に対応するテキストは小さな円形画像の横に表示されます。このレイアウトはテキストがなくても適しています。

### AlternatingFlow {#AlternatingFlow}
```
public static final int AlternatingFlow
```

タスク、プロセス、またはワークフローにおける情報のグループまたは順次ステップを示すために使用します。情報のグループ間の相互作用や関係を強調します。

### AlternatingHexagons {#AlternatingHexagons}
```
public static final int AlternatingHexagons
```

相互接続されたアイデアの系列を表すために使用します。レベル 1 のテキストは六角形の内部に表示されます。レベル 2 のテキストは形状の外側に表示されます。

### AlternatingPictureBlocks {#AlternatingPictureBlocks}
```
public static final int AlternatingPictureBlocks
```

上から下へ画像の系列を示すために使用します。テキストは画像の右側または左側に交互に表示されます。

### AlternatingPictureCircles {#AlternatingPictureCircles}
```
public static final int AlternatingPictureCircles
```

テキスト付きの画像セットを示すために使用します。対応するテキストは中央の円に表示され、画像は左から右へ交互に配置されます。

### ArrowRibbon {#ArrowRibbon}
```
public static final int ArrowRibbon
```

対立する力など、何らかの接続がある関連または対照的な概念を示すために使用します。レベル 1 の最初の 2 行のテキストが矢印内のテキストとして使用されます。未使用のテキストは表示されませんが、レイアウトを切り替えると利用可能です。

### AscendingPictureAccentProcess {#AscendingPictureAccentProcess}
```
public static final int AscendingPictureAccentProcess
```

説明テキスト付きの上昇する画像シリーズを示すために使用します。少量のテキストで最適に機能します。

### Balance {#Balance}
```
public static final int Balance
```

2 つのアイデア間の比較または関係を示すために使用します。レベル 1 の最初の 2 行は中心点の各側面上部のテキストに対応します。レベル 2 のテキストは各側面の 4 つの形状に制限され、強調されます。バランスはレベル 2 テキストを含む形状が最も多い側に傾きます。未使用のテキストは表示されませんが、レイアウトを切り替えると利用可能です。

### BasicBendingProcess {#BasicBendingProcess}

```
public static final int BasicBendingProcess
```

タスク、プロセス、またはワークフローにおける進行または順次ステップを示すために使用します。形状の水平および垂直表示スペースを最大化します。

### BasicBlockList {#BasicBlockList}

```
public static final int BasicBlockList
```

この列挙型は、パイ・オブ・パイまたはバー・オブ・パイ チャートの第2のパイまたはバー上の分割点の種類を表します。
## フィールド

| フィールド | 説明 |
| --- | --- |
| [Default](#Default) | データ ポイントをこのチャート タイプの既定のメカニズムで分割することを指定します。 |
| [Custom](#Custom) | データ ポイントをカスタム分割値に従ってパイと第2チャートの間で分割することを指定します。 |
| [ByPercentage](#ByPercentage) | データ ポイントを、パーセンテージが「分割位置」パーセンテージ未満のポイントを第2チャートに置くことで、パイと第2チャートの間で分割することを指定します。 |
| [ByPos](#ByPos) | データ ポイントを、データ ポイントの最後の「分割位置」を第2チャートに置くことで、パイと第2チャートの間で分割することを指定します。 |
| [ByValue](#ByValue) | データ ポイントを、値が「分割位置」未満のデータ ポイントを第2チャートに置くことで、パイと第2チャートの間で分割することを指定します。 |

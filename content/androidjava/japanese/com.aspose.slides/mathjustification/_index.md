数式段落（同じ段落内の隣接した数式テキストの系列）の配置を指定します
## フィールド

| フィールド | 説明 |
| --- | --- |
| [LeftJustified](#LeftJustified) | 数式テキストを左揃えにします |
| [RightJustified](#RightJustified) | 数式テキストを右揃えにします |
| [Centered](#Centered) | 数式テキストを中央揃えにします |
| [CenteredAsGroup](#CenteredAsGroup) | ページ上で数式ブロック全体を単一ユニットとして中央に配置します |

### RightJustified {#RightJustified}



数式テキストを右揃えにします

### Centered {#Centered}
```
public static final int Centered
```


数式テキストを中央揃えにします

### CenteredAsGroup {#CenteredAsGroup}
```
public static final int CenteredAsGroup
```


ページ上で数式ブロック全体を単一ユニットとして中央に配置します。これにより、数式テキストのインスタンス同士は相互に配置できるが、数式テキスト全体のグループは全体として中央に配置されます。

感度ラベルの割り当て方法を表します。
## フィールド

| フィールド | 説明 |
| --- | --- |
| [Standard](#Standard) | ユーザーによって直接適用されなかった任意の感度ラベルに使用します。 |
| [Privileged](#Privileged) | ユーザーによって直接適用された任意の感度ラベルに使用します。 |
### 標準 {#Standard}
```
public static final int Standard
```

ユーザーによって直接適用されなかった任意の感度ラベルに使用します。これには、デフォルトラベルや自動的に適用されるラベルが含まれます。

### 特権 {#Privileged}
```
public static final int Privileged
```

ユーザーによって直接適用された任意の感度ラベルに使用します。これには、手動で適用された感度ラベルや、推奨または必須のラベリング、ユーザーが感度ラベルを選択する機能が含まれます。

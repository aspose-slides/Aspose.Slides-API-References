ActiveX コントロールのプロパティを保存するために使用されるメソッドを指定します。
## フィールド

| フィールド | 説明 |
| --- | --- |
| [NotDefined](#NotDefined) | 永続性 ID が指定されていません。 |
| [PersistPropertyBag](#PersistPropertyBag) | ActiveX コントロールがプロパティ バッグ ベースの永続性を使用して永続化されることを指定します。 |
| [PersistStream](#PersistStream) | ActiveX コントロールがデフォルト状態への初期化をサポートしないストリーム ベースの永続性を使用して永続化されることを指定します。 |
| [PersistStreamInit](#PersistStreamInit) | ActiveX コントロールがデフォルト状態への初期化をサポートするストリーム ベースの永続性を使用して永続化されることを指定します。 |
| [PersistStorage](#PersistStorage) | ActiveX コントロールがストレージ ベースの永続性を使用して永続化されることを指定します。 |
### NotDefined {#NotDefined}
```
public static final int NotDefined
```

永続性 ID が指定されていません。

### PersistPropertyBag {#PersistPropertyBag}
```
public static final int PersistPropertyBag
```

ActiveX コントロールがプロパティ バッグ ベースの永続性を使用して永続化されることを指定します。プロパティ バッグ ベースの永続性は、名前と値のペアのコレクションによって ActiveX コントロールを保存します。このコレクションは、ActiveX コントロールによって永続化されるデータを指定します。

### PersistStream {#PersistStream}
```
public static final int PersistStream
```

ActiveX コントロールがデフォルト状態への初期化をサポートしないストリーム ベースの永続性を使用して永続化されることを指定します。

### PersistStreamInit {#PersistStreamInit}
```
public static final int PersistStreamInit
```

ActiveX コントロールがデフォルト状態への初期化をサポートするストリーム ベースの永続性を使用して永続化されることを指定します。

### PersistStorage {#PersistStorage}
```
public static final int PersistStorage
```

ActiveX コントロールがストレージ ベースの永続性を使用して永続化されることを指定します。

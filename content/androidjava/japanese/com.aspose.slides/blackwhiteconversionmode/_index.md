スライドの画像をビトナル画像に変換する方法を制御するオプションを提供します。
## フィールド

| フィールド | 説明 |
| --- | --- |
| [Default](#Default) | 変換アルゴリズムを指定しません。 |
| [Dithering](#Dithering) | ディザリングアルゴリズム（Floyd-Steinberg）を指定します。 |
| [DitheringFloydSteinberg](#DitheringFloydSteinberg) | Floyd-Steinberg ディザリングアルゴリズムを指定します。 |
| [Auto](#Auto) | 自動計算された閾値アルゴリズム（Otsu）を指定します。 |
| [AutoOtsu](#AutoOtsu) | 自動計算された Otsu の閾値アルゴリズムを指定します。 |
| [Threshold25](#Threshold25) | 静的閾値アルゴリズム（25％）を指定します。 |
| [Threshold50](#Threshold50) | 静的閾値アルゴリズム（50％）を指定します。 |
| [Threshold75](#Threshold75) | 静的閾値アルゴリズム（75％）を指定します。 |
### Default {#Default}
```
public static final int Default
```

public interface IHeadingPair
```

Belge'nin 'Heading pair' özelliğini temsil eder. Bu özellik, belge bölümlerinin grup adını ve gruptaki bölüm sayısını gösterir.
## Yöntemler

| Yöntem | Açıklama |
| --- | --- |
| [getName()](#getName--) | Belge bölümlerinin grup adını döndürür. |
| [getCount()](#getCount--) | Gruptaki bölüm sayısını döndürür. |
### getName() {#getName--}
```
public abstract String getName()
```


Belge bölümlerinin grup adını döndürür. Salt okunur String.

**Döndürür:**
java.lang.String
### getCount() {#getCount--}
```
public abstract int getCount()
```


Gruptaki bölüm sayısını döndürür. Salt okunur int.

**Döndürür:**
int

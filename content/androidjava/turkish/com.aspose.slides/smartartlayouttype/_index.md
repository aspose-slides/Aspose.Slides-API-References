### AccentProcess {#AccentProcess}
```
public static final int AccentProcess
```

İlerlemeyi, bir zaman çizelgesini veya bir görev, süreç ya da iş akışındaki ardışık adımları göstermek için kullanılır. Hem Seviye 1 hem de Seviye 2 metni için iyi çalışır.

### AccentedPicture {#AccentedPicture}
```
public static final int AccentedPicture
```

Merkezdeki fotoğrafik bir fikri, yan taraftaki ilgili fikirlerle birlikte göstermek için kullanılır. Üst Seviye 1 metni merkez resmi üzerinde görünür. Diğer Seviye 1 şekillerin karşılık gelen metni küçük dairesel resimlerin yanında görünür. Bu düzen metin olmadan da iyi çalışır.

### AlternatingFlow {#AlternatingFlow}
```
public static final int AlternatingFlow
```

Bilgi gruplarını veya bir görev, süreç ya da iş akışındaki ardışık adımları göstermek için kullanılır. Gruplar arasındaki etkileşim ya da ilişkileri vurgular.

### AlternatingHexagons {#AlternatingHexagons}
```
public static final int AlternatingHexagons
```

Birbiriyle bağlantılı fikirlerin bir serisini temsil etmek için kullanılır. Seviye 1 metni altıgenlerin içinde görünür. Seviye 2 metni şekillerin dışında görünür.

### AlternatingPictureBlocks {#AlternatingPictureBlocks}
```
public static final int AlternatingPictureBlocks
```

Üstten alta bir dizi resmi göstermek için kullanılır. Metin, resmin sağında ya da solunda dönüşümlü olarak görünür.

### AlternatingPictureCircles {#AlternatingPictureCircles}
```
public static final int AlternatingPictureCircles
```

Metinli bir dizi resim göstermek için kullanılır. Karşılık gelen metin, resimlerin sol ve sağa dönüşümlü olarak yer aldığı merkez dairelerde görünür.

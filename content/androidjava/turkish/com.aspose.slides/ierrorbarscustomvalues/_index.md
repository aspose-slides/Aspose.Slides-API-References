public interface IErrorBarsCustomValues
```

Belirtilen hata çubuğu değerlerini tanımlar. Yalnızca Error bars value type Custom olduğunda kullanılmalıdır.

### getXMinus() {#getXMinus--}
```
public abstract IDoubleChartValue getXMinus()
```

Negatif yönde hata çubuğu değerini belirtir. Error bars value type Custom ve ErrorBarsXFormat izin verildiğinde kullanılabilir. Diğer durumlarda bu özellik null döndürür. Salt okunur [IDoubleChartValue](../../com.aspose.slides/idoublechartvalue).

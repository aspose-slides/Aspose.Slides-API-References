مجموعه‌ای از جلوه‌های بصری پیش‌تعریف‌شده برای رندر کردن جوهر را مشخص می‌کند. هر جلوه به بافت یا تصویری که ظاهر استایل‌شده‌ی جوهر را شبیه‌سازی می‌کند، متناظر است. این مقادیر می‌توانند برای سفارشی‌سازی سبک بصری خطوط جوهر دیجیتال هنگام رندر کردن استفاده شوند.

| [Bronze](#Bronze) | بافت گرم و قهوه‌ای‌رنگ فلزی که شبیه جوهر برنزی است. |
| [Galaxy](#Galaxy) | بافت رنگارنگ و درخشان که شبیه کهکشانی است، با رنگ‌های کیهانی. |
| [Gold](#Gold) | بافت درخشان و طلا رنگ فلزی که به خطوط جوهر ظاهری لوکس می‌بخشد. |
| [Lava](#Lava) | بافت آتشین که شبیه لاوان مذاب است، با رنگ‌های قرمز و نارنجی. |
| [Ocean](#Ocean) | بافت آبی عمیق، شبیه سیال که امواج اقیانوس یا جوهر آب‌پایه را شبیه‌سازی می‌کند. |
| [RainbowGlitter](#RainbowGlitter) | جلوه‌ی رنگارنگ و درخشان شاداب رنگین‌کمان که برای خطوط جوهر جشن و پر انرژی استفاده می‌شود. |
| [RoseGold](#RoseGold) | ترکیب نرم صورتی-طلایی، مشابه طلا رز، برای خطوط جوهر زیبا. |
| [Silver](#Silver) | بافت سرد و نقره‌ای فلزی که جوهر نقره کلاسیک را شبیه‌سازی می‌کند. |

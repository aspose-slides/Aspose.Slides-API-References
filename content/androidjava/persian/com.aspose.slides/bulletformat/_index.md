تغییرات پیش‌فرض غیر صفر برای تورفتگی مؤثر پاراگراف و MarginLeft را زمانی که بولت‌ها فعال هستند تنظیم می‌کند (مانند کاری که PowerPoint هنگام فعال‌سازی بولت/شماره‌گذاری پاراگراف انجام می‌دهد). اگر بولت‌ها غیرفعال شوند، فقط تورفتگی پاراگراف و MarginLeft بازنشانی می‌شود (مانند کاری که PowerPoint هنگام غیرفعال‌سازی بولت/شماره‌گذاری پاراگراف انجام می‌دهد). تغییرات تورفتگی نسبت به زمینه فعلی بولت - IBulletFormat.Type، .NumberedBulletStyle و FontHeight اولین بخش اعمال می‌شود. تغییرات تورفتگی غیر صفر بر تورفتگی مؤثر و MarginLeft پاراگراف جاری اعمال می‌شوند (نتیجه مقادیر محلی می‌شوند).

داده‌های قالب‌بندی مؤثر بولت را با اعمال ارث‌بری دریافت می‌کند.

--------------------

> ```
> This example demonstrates getting some effective bullet format properties.
>  
>  Presentation pres = new Presentation("MyPresentation.pptx");
>  try {
>      IAutoShape shape = (IAutoShape)pres.getSlides().get_Item(0).getShapes().get_Item(0);
>      IBulletFormatEffectiveData effectiveBulletFormat = shape.getTextFrame().getParagraphs().get_Item(0).getParagraphFormat().getBullet().getEffective();
>      System.out.println("Bullet type: " + effectiveBulletFormat.getType());
>      if (effectiveBulletFormat.getType() == BulletType.Numbered)
>      {
>          System.out.println("Numbered style: " + effectiveBulletFormat.getNumberedBulletStyle());
>          System.out.println("Starting number: " + effectiveBulletFormat.getNumberedBulletStartWith());
>      }
>  } finally {
>      if (pres != null) pres.dispose();
>  }
> ```

### Installable {#Installable}
```
public static final int Installable
```

فونت‌هایی که این تنظیم را دارند نشان می‌دهند که می‌توانند توسط یک برنامه در سیستم از راه دور جاسازی و به‌صورت دائمی نصب شوند. کاربر سیستم از راه دور همان حقوق، تعهدات و مجوزهای متعلق به آن فونت را که خریدار اولیه داشته است، به‌دست می‌آورد و مشمول همان توافق‌نامهٔ مجوز کاربری نهایی، کپی‌رایت، پتنت طراحی و/یا علامت تجاری است که خریدار اولیه نیز تحت آن بوده است.

### Restricted {#Restricted}
```
public static final int Restricted
```

فونت‌هایی که تنها این بیت تنظیم شده است، بدون کسب اجازه از صاحب قانونی، نمی‌توانند تحت هیچ شرایطی اصلاح، جاسازی یا تعویض شوند.

### PreviewPrint {#PreviewPrint}
```
public static final int PreviewPrint
```

هنگامی که این بیت تنظیم شده باشد، می‌توان فونت را جاسازی کرد و به‌طور موقت در سیستم از راه دور بارگذاری کرد. اسنادی که شامل فونت‌های Preview & Print هستند باید به‌صورت «فقط-خواندنی» باز شوند؛ هیچ‌گونه ویرایشی بر روی سند قابل اعمال نیست.

### Editable {#Editable}
```
public static final int Editable
```

هنگامی که این بیت تنظیم شده باشد، می‌توان فونت را جاسازی کرد اما فقط به‌صورت موقت در سایر سیستم‌ها نصب می‌شود. بر خلاف فونت‌های Preview & Print، اسنادی که شامل فونت‌های Editable هستند می‌توانند برای خواندن باز شوند، ویرایش مجاز است و تغییرات می‌توانند ذخیره شوند.

### NoSubsetting {#NoSubsetting}
```
public static final int NoSubsetting
```

هنگامی که این بیت تنظیم شده باشد، قبل از جاسازی، نمی‌توان از فونت زیرمجموعه استفاده کرد. سایر محدودیت‌های جاسازی که در بیت‌های ۰-۳ و ۹ تعیین شده‌اند نیز اعمال می‌شوند.

### BitmapOnly {#BitmapOnly}
```
public static final int BitmapOnly
```

هنگامی که این بیت تنظیم شده باشد، تنها نقشه‌های بیت موجود در فونت می‌توانند جاسازی شوند. هیچ دادهٔ خط‌کش (outline)‌ای قابل جاسازی نیست. اگر در فونت هیچ نقشهٔ بیت موجود نباشد، فونت به‌عنوان غیرقابل جاسازی در نظر گرفته می‌شود و سرویس‌های جاسازی شکست می‌خورند.

متد اختصاص برچسب حساسیت را نشان می‌دهد.
## فیلدها

| فیلد | توضیح |
| --- | --- |
| [Standard](#Standard) | برای هر برچسب حساسیتی که به‌طور مستقیم توسط کاربر اعمال نشده استفاده می‌شود. |
| [Privileged](#Privileged) | برای هر برچسب حساسیتی که به‌طور مستقیم توسط کاربر اعمال شده استفاده می‌شود. |
### استاندارد {#Standard}
```
public static final int Standard
```

برای هر برچسب حساسیتی که به‌طور مستقیم توسط کاربر اعمال نشده استفاده می‌شود. این شامل تمام برچسب‌های پیش‌فرض و برچسب‌های به‌صورت خودکار اعمال‌شده می‌باشد.

### اختصاصی {#Privileged}
```
public static final int Privileged
```

برای هر برچسب حساسیتی که به‌طور مستقیم توسط کاربر اعمال شده استفاده می‌شود. این شامل تمام برچسب‌های حساسیت اعمال‌شده به‌صورت دستی، همچنین برچسب‌گذاری پیشنهادی یا اجباری یا هر ویژگی‌ای که کاربر تصمیم می‌گیرد کدام برچسب حساسیت را اعمال کند، می‌باشد.

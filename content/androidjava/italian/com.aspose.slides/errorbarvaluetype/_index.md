Rappresenta il tipo di valore della barra di errore
## Campi

| Campo | Descrizione |
| --- | --- |
| [Custom](#Custom) | Specifica che la lunghezza delle barre di errore deve essere determinata dagli elementi Plus e Minus. |
| [Fixed](#Fixed) | Specifica che la lunghezza delle barre di errore è il valore fisso determinato da Error Bar Value. |
| [Percentage](#Percentage) | Specifica che la lunghezza delle barre di errore è la percentuale di Error Bar Value dei dati. |
| [StandardDeviation](#StandardDeviation) | Specifica che la lunghezza delle barre di errore è le deviazioni standard di Error Bar Value dei dati. |
| [StandardError](#StandardError) | Specifica che la lunghezza delle barre di errore è gli errori standard di Error Bar Value dei dati. |

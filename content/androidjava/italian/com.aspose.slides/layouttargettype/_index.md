```
public final class LayoutTargetType extends System.Enum
```

Se il layout dell'area del grafico è definito manualmente, questa proprietà specifica se posizionare l'area del grafico al suo interno (senza includere gli assi e le etichette degli assi) o all'esterno (includendo assi e etichette degli assi).
## Campi

| Campo | Descrizione |
| --- | --- |
| [Inner](#Inner) | Specifica che la dimensione dell'area del grafico determinerà la dimensione dell'area del grafico, senza includere i segni di spunta e le etichette degli assi. |
| [Outer](#Outer) | Specifica che la dimensione dell'area del grafico determinerà la dimensione dell'area del grafico, i segni di spunta e le etichette degli assi. |
### Interno {#Inner}
```
public static final int Inner
```

Specifică che la dimensione dell'area del grafico determinerà la dimensione dell'area del grafico, senza includere i segni di spunta e le etichette degli assi.

### Esterno {#Outer}
```
public static final int Outer
```

Specifică che la dimensione dell'area del grafico determinerà la dimensione dell'area del grafico, i segni di spunta e le etichette degli assi.

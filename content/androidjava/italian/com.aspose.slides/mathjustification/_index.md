Specifica l'allineamento del paragrafo matematico (una serie di istanze adiacenti di testo matematico all'interno dello stesso paragrafo)

## Campi

| Campo | Descrizione |
| --- | --- |
| [LeftJustified](#LeftJustified) | Allinea il testo matematico a sinistra |
| [RightJustified](#RightJustified) | Allinea il testo matematico a destra |
| [Centered](#Centered) | Allinea il testo matematico al centro |
| [CenteredAsGroup](#CenteredAsGroup) | Centra sull'intera pagina l'intero gruppo di blocchi matematici come una singola unità. |

### CenteredAsGroup {#CenteredAsGroup}
```
public static final int CenteredAsGroup
```

Centra sull'intera pagina l'intero gruppo di blocchi matematici come una singola unità. Ciò significa che le istanze di testo matematico possono essere allineate tra loro, ma l'intero gruppo di testo matematico è centrato nel suo complesso.

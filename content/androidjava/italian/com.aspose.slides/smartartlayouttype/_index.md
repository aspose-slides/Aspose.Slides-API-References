### AccentProcess {#AccentProcess}
```
public static final int AccentProcess
```

Utilizzare per mostrare una progressione, una linea temporale o fasi sequenziali in un'attività, processo o flusso di lavoro. Funziona bene per illustrare sia il testo di livello 1 che quello di livello 2.

### AccentedPicture {#AccentedPicture}
```
public static final int AccentedPicture
```

Utilizzare per mostrare un'idea fotografica centrale con idee correlate ai lati. Il testo di livello 1 superiore appare sopra l'immagine centrale. Il testo corrispondente per le altre forme di livello 1 appare accanto alle piccole immagini circolari. Questo layout funziona anche bene senza testo.

### AlternatingFlow {#AlternatingFlow}
```
public static final int AlternatingFlow
```

Utilizzare per mostrare gruppi di informazioni o fasi sequenziali in un'attività, processo o flusso di lavoro. Evidenzia l'interazione o le relazioni tra i gruppi di informazioni.

### AlternatingHexagons {#AlternatingHexagons}
```
public static final int AlternatingHexagons
```

Utilizzare per rappresentare una serie di idee interconnesse. Il testo di livello 1 appare all'interno degli esagoni. Il testo di livello 2 appare all'esterno delle forme.

### AlternatingPictureBlocks {#AlternatingPictureBlocks}
```
public static final int AlternatingPictureBlocks
```

Utilizzare per mostrare una serie di immagini dall'alto verso il basso. Il testo appare alternativamente a destra o a sinistra dell'immagine.

### AlternatingPictureCircles {#AlternatingPictureCircles}
```
public static final int AlternatingPictureCircles
```

Utilizzare per mostrare un insieme di immagini con testo. Il testo corrispondente appare nei cerchi centrali con le immagini che alternano da sinistra a destra.

### ArrowRibbon {#ArrowRibbon}
```
public static final int ArrowRibbon
```

Utilizzare per mostrare concetti correlati o contrastanti con qualche connessione, ad esempio forze opposte. Le prime due righe del testo di livello 1 sono usate per il testo nelle frecce. Il testo inutilizzato non appare, ma rimane disponibile se si cambiano i layout.

### AscendingPictureAccentProcess {#AscendingPictureAccentProcess}
```
public static final int AscendingPictureAccentProcess
```

Utilizzare per mostrare una serie ascendente di immagini con testo descrittivo. Funziona meglio con una piccola quantità di testo.

### Balance {#Balance}
```
public static final int Balance
```

Utilizzare per confrontare o mostrare la relazione tra due idee. Ognuna delle prime due righe del testo di livello 1 corrisponde al testo nella parte superiore di un lato del punto centrale. Evidenzia il testo di livello 2, limitato a quattro forme per ciascun lato del punto centrale. L'equilibrio si inclina verso il lato con il maggior numero di forme contenenti testo di livello 2. Il testo inutilizzato non appare, ma rimane disponibile se si cambiano i layout.

### BasicBendingProcess {#BasicBendingProcess}
```
public static final int BasicBendingProcess
```

Utilizzare per mostrare una progressione o fasi sequenziali in un'attività, processo o flusso di lavoro. Massimizza lo spazio di visualizzazione sia orizzontale che verticale per le forme.

### BasicBlockList {#BasicBlockList}
```
public static final int BasicBlockList
```

Utilizzare per mostrare blocchi di informazioni non sequenziali o raggruppati. Massimizza lo spazio di visualizzazione sia orizzontale che verticale per le forme.

### BasicChevronProcess {#BasicChevronProcess}
```
public static final int BasicChevronProcess
```

Utilizzare per mostrare una progressione; una linea temporale; fasi sequenziali in un'attività, processo o flusso di lavoro; o per enfatizzare movimento o direzione. Il testo di livello 1 appare all'interno di una forma a freccia mentre il testo di livello 2 appare sotto le forme a freccia.

### BasicCycle {#BasicCycle}
```
public static final int BasicCycle
```

Utilizzare per rappresentare una sequenza continua di fasi, compiti o eventi in un flusso circolare. Evidenzia le fasi o i passaggi piuttosto che le frecce o il flusso di collegamento. Funziona meglio solo con testo di livello 1.

### BasicMatrix {#BasicMatrix}
```
public static final int BasicMatrix
```

Utilizzare per mostrare la relazione dei componenti a un tutto nei quadranti. Le prime quattro righe del testo di livello 1 appaiono nei quadranti. Il testo inutilizzato non appare, ma rimane disponibile se si cambiano i layout.

### BasicPie {#BasicPie}
```
public static final int BasicPie
```

Utilizzare per mostrare come le parti individuali formano un tutto. Le prime sette righe del testo di livello 1 corrispondono alle forme a cuneo o torta distribuite uniformemente. La forma di testo di livello 1 superiore appare fuori dal resto della torta per evidenziare. Il testo inutilizzato non appare, ma rimane disponibile se si cambiano i layout.

### BasicProcess {#BasicProcess}
```
public static final int BasicProcess
```

Utilizzare per mostrare una progressione o fasi sequenziali in un'attività, processo o flusso di lavoro.

### BasicPyramid {#BasicPyramid}
```
public static final int BasicPyramid
```

Utilizzare per mostrare relazioni proporzionali, interconnesse o gerarchiche con il componente più grande in basso e che si restringe verso l'alto. Il testo di livello 1 appare nei segmenti della piramide e il testo di livello 2 appare nelle forme accanto a ciascun segmento.

### BasicRadial {#BasicRadial}
```
public static final int BasicRadial
```

Utilizzare per mostrare la relazione a un'idea centrale in un ciclo. La prima riga del testo di livello 1 corrisponde alla forma centrale, e il suo testo di livello 2 corrisponde alle forme circolari circostanti. Il testo inutilizzato non appare, ma rimane disponibile se si cambiano i layout.

### BasicTarget {#BasicTarget}
```
public static final int BasicTarget
```

Utilizzare per mostrare contenimento, gradazioni o relazioni gerarchiche. Le prime cinque righe del testo di livello 1 sono associate a un cerchio. Il testo inutilizzato non appare, ma rimane disponibile se si cambiano i layout.

### BasicTimeline {#BasicTimeline}
```
public static final int BasicTimeline
```

Utilizzare per mostrare fasi sequenziali in un'attività, processo o flusso di lavoro, o per mostrare informazioni sulla linea temporale. Funziona bene sia con testo di livello 1 che di livello 2.

### BasicVenn {#BasicVenn}
```
public static final int BasicVenn
```

Utilizzare per mostrare relazioni sovrapposte o interconnesse. Le prime sette righe del testo di livello 1 corrispondono a un cerchio. Se ci sono quattro o meno righe di testo di livello 1, il testo è all'interno dei cerchi; se sono più di quattro, il testo è all'esterno dei cerchi. Il testo inutilizzato non appare, ma rimane disponibile se si cambiano i layout.

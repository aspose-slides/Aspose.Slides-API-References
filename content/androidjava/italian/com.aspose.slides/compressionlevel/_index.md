Specifica i livelli di compressione ZIP per i file OpenXML. I livelli più alti offrono una compressione migliore a costo di una velocità di elaborazione più lenta.
## Campi

| Campo | Descrizione |
| --- | --- |
| [None](#None) | Non viene applicata alcuna compressione. |
| [Level1](#Level1) | Compressione più veloce con il rapporto di compressione più basso. |
| [Level2](#Level2) | Compressione più rapida con un rapporto di compressione leggermente migliore rispetto a [Level1](../../com.aspose.slides/compressionlevel\#Level1). |
| [Level3](#Level3) | Fornisce una compressione migliore rispetto a [Level2](../../com.aspose.slides/compressionlevel\#Level2) con un impatto di prestazioni moderato. |
| [Level4](#Level4) | Fornisce una compressione migliore rispetto a [Level3](../../com.aspose.slides/compressionlevel\#Level3). |
| [Level5](#Level5) | Fornisce una compressione migliorata rispetto a [Level4](../../com.aspose.slides/compressionlevel\#Level4) con tempo di elaborazione aggiuntivo. |
| [Level6](#Level6) | Compressione standard, che offre un buon equilibrio tra velocità di compressione e dimensione del file. |
| [Level7](#Level7) | Fornisce una compressione più elevata rispetto a [Level6](../../com.aspose.slides/compressionlevel\#Level6) con un'elaborazione più lenta. |
| [Level8](#Level8) | Fornisce una compressione più elevata rispetto a [Level7](../../com.aspose.slides/compressionlevel\#Level7). |
| [Level9](#Level9) | Compressione massima. |

Rappresenta le opzioni per le forme SVG.
## Campi

| Campo | Descrizione |
| --- | --- |
| [OnFocusIn](#OnFocusIn) | Si verifica quando un elemento riceve il focus, ad esempio quando del testo viene selezionato. |
| [OnFocusOut](#OnFocusOut) | Si verifica quando un elemento perde il focus, ad esempio quando del testo viene deselezionato. |
| [OnActivate](#OnActivate) | Si verifica quando un elemento viene attivato, ad esempio tramite un clic del mouse o una pressione di un tasto. |
| [OnClick](#OnClick) | Si verifica quando il pulsante del dispositivo di puntamento viene cliccato su un elemento. |
| [OnMouseDown](#OnMouseDown) | Si verifica quando il pulsante del dispositivo di puntamento viene premuto su un elemento. |
| [OnMouseUp](#OnMouseUp) | Si verifica quando il pulsante del dispositivo di puntamento viene rilasciato su un elemento. |
| [OnMouseOver](#OnMouseOver) | Si verifica quando il dispositivo di puntamento viene spostato su un elemento. |
| [OnMouseMove](#OnMouseMove) | Si verifica quando il dispositivo di puntamento viene spostato mentre è sopra un elemento. |
| [OnMouseOut](#OnMouseOut) | Si verifica quando il dispositivo di puntamento viene spostato via da un elemento. |
| [OnLoad](#OnLoad) | Si verifica quando l'agente utente ha analizzato completamente l'elemento e i suoi discendenti e tutte le risorse referenziate necessarie per renderizzarlo. |
| [OnUnload](#OnUnload) | Si verifica quando l'implementazione del DOM rimuove un documento da una finestra o da un frame. |
| [OnAbort](#OnAbort) | Si verifica quando il caricamento della pagina viene interrotto prima che un elemento sia stato caricato completamente. |
| [OnError](#OnError) | Si verifica quando un elemento non viene caricato correttamente o quando si verifica un errore durante l'esecuzione dello script. |
| [OnResize](#OnResize) | Si verifica quando la visualizzazione del documento viene ridimensionata. |
| [OnScroll](#OnScroll) | Si verifica quando la visualizzazione del documento viene spostata lungo l'asse X, Y o entrambi. |
| [OnZoom](#OnZoom) | Si verifica quando il livello di zoom della visualizzazione del documento viene modificato. |
| [OnBegin](#OnBegin) | Si verifica quando un elemento di animazione inizia. |
| [OnEnd](#OnEnd) | Si verifica quando un elemento di animazione termina. |
| [OnRepeat](#OnRepeat) | Si verifica quando un elemento di animazione si ripete. |

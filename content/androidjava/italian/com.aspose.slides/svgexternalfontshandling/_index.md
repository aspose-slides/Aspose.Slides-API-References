Rappresenta un modo per gestire i font esterni utilizzati per il disegno del testo.
## Campi

| Campo | Descrizione |
| --- | --- |
| [AddLinksToFontFiles](#AddLinksToFontFiles) | Aggiunge collegamenti a file di font separati nella sezione style del file SVG. |
| [Embed](#Embed) | Salva i dati dei font direttamente nel file SVG. |
| [Vectorize](#Vectorize) | Salva tutto il testo che utilizza font esterni come grafica. |
### AddLinksToFontFiles {#AddLinksToFontFiles}
```
public static final int AddLinksToFontFiles
```

Aggiunge collegamenti a file di font separati nella sezione style del file SVG.

### Embed {#Embed}
```
public static final int Embed
```

Salva i dati dei font direttamente nel file SVG. Verifica tutti gli accordi di licenza dei font esterni prima di utilizzare questa opzione.

### Vectorize {#Vectorize}
```
public static final int Vectorize
```

Salva tutto il testo che utilizza font esterni come grafica.

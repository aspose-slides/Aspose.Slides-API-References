Rappresenta il tipo di linea di tendenza
## Campi

| Campo | Descrizione |
| --- | --- |
| [Exponential](#Exponential) | Specifica che la linea di tendenza deve essere una curva esponenziale nella forma. |
| [Linear](#Linear) | Specifica che la linea di tendenza deve essere una linea nella forma. |
| [Logarithmic](#Logarithmic) | Specifica che la linea di tendenza deve essere una curva logaritmica nella forma , dove log è il logaritmo naturale. |
| [MovingAverage](#MovingAverage) | Specifica che la linea di tendenza deve essere una media mobile di periodo Period. |
| [Polynomial](#Polynomial) | Specifica che la linea di tendenza deve essere una curva polinomiale di ordine Order nella forma. |
| [Power](#Power) | Specifica che la linea di tendenza deve essere una curva di potenza nella forma. |

Tipi di spazio (spaziatura orizzontale) tra le colonne di una matrice
## Campi

| Campo | Descrizione |
| --- | --- |
| [SingleSpacingGap](#SingleSpacingGap) | Singolo (1 em per le colonne o 1 riga per le righe) |
| [OneAndHalfSpacingGap](#OneAndHalfSpacingGap) | Uno e mezzo (1.5 em per le colonne o 1.5 righe per le righe) |
| [DoubleSpacingGap](#DoubleSpacingGap) | Spaziatura doppia (2 em per le colonne o 2 righe per le righe) |
| [Exactly](#Exactly) | Valore esatto Per le colonne, basarsi sul valore di ColumnGap, misurato in twip Per le righe, basarsi sul valore di RowGap, misurato in twip |
| [Multiple](#Multiple) | Multiplo Per le colonne, basarsi sul valore di ColumnGap, misurato in incrementi di 0.5 em Per le righe, basarsi sul valore di RowGap, misurato in righe |

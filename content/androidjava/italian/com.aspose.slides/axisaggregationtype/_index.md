Rappresenta il tipo di aggregazione dell'asse di categoria. Utilizzato solo con le serie Histogram o HistogramPareto.
## Campi

| Campo | Descrizione |
| --- | --- |
| [ByCategory](#ByCategory) | Aggrega per categorie |
| [Automatic](#Automatic) | Aggrega le categorie automaticamente. |
| [ByBinWidth](#ByBinWidth) | Aggrega le categorie per larghezza del bin. |
| [ByNumberOfBins](#ByNumberOfBins) | Aggrega le categorie per numero di bin. |

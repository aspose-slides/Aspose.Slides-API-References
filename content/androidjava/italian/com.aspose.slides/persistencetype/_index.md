Specifica il metodo utilizzato per memorizzare le proprietà del controllo ActiveX.
## Campi

| Campo | Descrizione |
| --- | --- |
| [NotDefined](#NotDefined) | ID di persistenza non specificato. |
| [PersistPropertyBag](#PersistPropertyBag) | Specifica che il controllo ActiveX è persistito usando una persistenza basata su property-bag. |
| [PersistStream](#PersistStream) | Specifica che il controllo ActiveX è persistito usando una persistenza basata su stream che non supporta l’inizializzazione del controllo ActiveX a uno stato predefinito. |
| [PersistStreamInit](#PersistStreamInit) | Specifica che il controllo ActiveX è persistito usando una persistenza basata su stream che supporta l’inizializzazione del controllo ActiveX a uno stato predefinito. |

### NotDefined {#NotDefined}
```
public static final int NotDefined
```

ID di persistenza non specificato.

### PersistPropertyBag {#PersistPropertyBag}
```
public static final int PersistPropertyBag
```

Specifică che il controllo ActiveX è persistito usando una persistenza basata su property-bag. La persistenza basata su property-bag memorizza un controllo ActiveX mediante una raccolta di coppie nome-valore che specificano i dati persi dal controllo ActiveX.

### PersistStream {#PersistStream}
```
public static final int PersistStream
```

Specifică che il controllo ActiveX è persistito usando una persistenza basata su stream che non supporta l’inizializzazione del controllo ActiveX a uno stato predefinito.

### PersistStreamInit {#PersistStreamInit}
```
public static final int PersistStreamInit
```

Specifică che il controllo ActiveX è persistito usando una persistenza basata su stream che supporta l’inizializzazione del controllo ActiveX a uno stato predefinito.

### PersistStorage {#PersistStorage}
```
public static final int PersistStorage
```

Specifică che il controllo ActiveX è persistito usando una persistenza basata su storage.

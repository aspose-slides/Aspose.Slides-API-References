Określa metodę używaną do przechowywania właściwości kontrolki ActiveX.

## Pola

| Pole | Opis |
| --- | --- |
| [NotDefined](#NotDefined) | Identyfikator persystencji nie został określony. |
| [PersistPropertyBag](#PersistPropertyBag) | Określa, że kontrolka ActiveX jest zachowywana przy użyciu persystencji opartej na woreczku właściwości. |
| [PersistStream](#PersistStream) | Określa, że kontrolka ActiveX jest zachowywana przy użyciu persystencji opartej na strumieniu, która nie obsługuje inicjalizacji kontrolki ActiveX do stanu domyślnego. |
| [PersistStreamInit](#PersistStreamInit) | Określa, że kontrolka ActiveX jest zachowywana przy użyciu persystencji opartej na strumieniu, która obsługuje inicjalizację kontrolki ActiveX do stanu domyślnego. |
| [PersistStorage](#PersistStorage) | Określa, że kontrolka ActiveX jest zachowywana przy użyciu persystencji opartej na magazynie. |
### NotDefined {#NotDefined}
```
public static final int NotDefined
```

Identyfikator persystencji nie został określony.

### PersistPropertyBag {#PersistPropertyBag}
```
public static final int PersistPropertyBag
```

Określa, że kontrolka ActiveX jest zachowywana przy użyciu persystencji opartej na woreczku właściwości. Persystencja oparta na woreczku właściwości przechowuje kontrolkę ActiveX przy użyciu kolekcji par nazw i wartości, które określają dane zachowywane przez kontrolkę ActiveX.

### PersistStream {#PersistStream}
```
public static final int PersistStream
```

Określa, że kontrolka ActiveX jest zachowywana przy użyciu persystencji opartej na strumieniu, która nie obsługuje inicjalizacji kontrolki ActiveX do stanu domyślnego.

### PersistStreamInit {#PersistStreamInit}
```
public static final int PersistStreamInit
```

Określa, że kontrolka ActiveX jest zachowywana przy użyciu persystencji opartej na strumieniu, która obsługuje inicjalizację kontrolki ActiveX do stanu domyślnego.

### PersistStorage {#PersistStorage}
```
public static final int PersistStorage
```

Określa, że kontrolka ActiveX jest zachowywana przy użyciu persystencji opartej na magazynie.

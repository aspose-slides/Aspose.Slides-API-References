Określa poziomy kompresji ZIP dla pliku OpenXML. Wyższe poziomy zapewniają lepszą kompresję kosztem wolniejszego przetwarzania.
## Pola

| Pole | Opis |
| --- | --- |
| [None](#None) | Nie stosuje się kompresji. |
| [Level1](#Level1) | Najszybsza kompresja przy najniższym współczynniku kompresji. |
| [Level2](#Level2) | Szybsza kompresja z nieco lepszym współczynnikiem kompresji niż [Level1](../../com.aspose.slides/compressionlevel\#Level1). |
| [Level3](#Level3) | Zapewnia lepszą kompresję niż [Level2](../../com.aspose.slides/compressionlevel\#Level2) przy umiarkowanym wpływie na wydajność. |
| [Level4](#Level4) | Zapewnia lepszą kompresję niż [Level3](../../com.aspose.slides/compressionlevel\#Level3). |
| [Level5](#Level5) | Zapewnia ulepszoną kompresję w porównaniu z [Level4](../../com.aspose.slides/compressionlevel\#Level4) przy dodatkowym czasie przetwarzania. |
| [Level6](#Level6) | Standardowa kompresja, oferująca dobrą równowagę między szybkością kompresji a rozmiarem pliku. |
| [Level7](#Level7) | Zapewnia wyższą kompresję niż [Level6](../../com.aspose.slides/compressionlevel\#Level6) przy wolniejszym przetwarzaniu. |
| [Level8](#Level8) | Zapewnia wyższą kompresję niż [Level7](../../com.aspose.slides/compressionlevel\#Level7). |
| [Level9](#Level9) | Maksymalna kompresja. |

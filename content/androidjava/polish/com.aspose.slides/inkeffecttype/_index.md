Określa zestaw wstępnie zdefiniowanych efektów wizualnych dla renderowania atramentu. Każdy efekt odpowiada teksturze lub obrazowi symulującemu stylizowany wygląd atramentu. Wartości te mogą być używane do dostosowywania stylu wizualnego cyfrowych pociągnięć atramentu podczas renderowania.

## Pola

| Pole | Opis |
| --- | --- |
| [NotDefined](#NotDefined) | Efekt nie jest zdefiniowany lub jest nieznany. |
| [Bronze](#Bronze) | Ciepła, brązowawa metaliczna tekstura przypominająca atrament w kolorze brązu. |
| [Galaxy](#Galaxy) | Kolorowa, połyskująca tekstura przypominająca galaktykę, z kosmicznymi odcieniami. |
| [Gold](#Gold) | Jasna, metaliczna złota tekstura nadająca pociągnięciom atramentu luksusowy wygląd. |
| [Lava](#Lava) | Ognista tekstura przypominająca płynną lawę, z czerwonymi i pomarańczowymi odcieniami. |
| [Ocean](#Ocean) | Głęboki niebieski, płynny wygląd tekstury naśladującej fale oceanu lub atrament wodny. |
| [RainbowGlitter](#RainbowGlitter) | Kolorowy, błyszczący efekt tęczowego brokatu używany do świątecznych lub żywych pociągnięć atramentu. |
| [RoseGold](#RoseGold) | Delikatne połączenie różowego i złotego, podobne do różowego złota, dla eleganckich pociągnięć atramentu. |
| [Silver](#Silver) | Chłodna, metaliczna srebrna tekstura symulująca klasyczny srebrny atrament. |

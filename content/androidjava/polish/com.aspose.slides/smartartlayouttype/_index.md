### AccentProcess {#AccentProcess}
```
public static final int AccentProcess
```

Użyj, aby pokazać postęp, oś czasu lub sekwencyjne kroki w zadaniu, procesie lub przepływie pracy. Dobrze sprawdza się przy ilustrowaniu zarówno Level 1, jak i Level 2 tekstu.

### AccentedPicture {#AccentedPicture}
```
public static final int AccentedPicture
```

Użyj, aby pokazać centralny, fotograficzny pomysł z powiązanymi pomysłami z boku. Tekst poziomu 1 wyświetla się nad centralnym obrazem. Odpowiedni tekst dla pozostałych kształtów poziomu 1 pojawia się obok małych okrągłych obrazów. Ten układ działa również bez tekstu.

### AlternatingFlow {#AlternatingFlow}
```
public static final int AlternatingFlow
```

Użyj, aby pokazać grupy informacji lub sekwencyjne kroki w zadaniu, procesie lub przepływie pracy. Podkreśla interakcję lub relacje pomiędzy grupami informacji.

### AlternatingHexagons {#AlternatingHexagons}
```
public static final int AlternatingHexagons
```

Użyj, aby przedstawić serię powiązanych ze sobą pomysłów. Tekst poziomu 1 pojawia się wewnątrz sześciokątów. Tekst poziomu 2 pojawia się na zewnątrz kształtów.

### AlternatingPictureBlocks {#AlternatingPictureBlocks}
```
public static final int AlternatingPictureBlocks
```

Użyj, aby pokazać serię obrazów od góry do dołu. Tekst pojawia się naprzemiennie po prawej lub po lewej stronie obrazu.

### AlternatingPictureCircles {#AlternatingPictureCircles}
```
public static final int AlternatingPictureCircles
```

Użyj, aby pokazać zestaw obrazów z tekstem. Odpowiedni tekst pojawia się w centralnych okręgach, a obrazy naprzemiennie po lewej i po prawej.

### ArrowRibbon {#ArrowRibbon}
```
public static final int ArrowRibbon
```

Użyj, aby pokazać powiązane lub kontrastujące koncepcje z pewnym połączeniem, np. przeciwstawne siły. Dwie pierwsze linie tekstu poziomu 1 są używane w strzałkach. Niewykorzystany tekst nie jest wyświetlany, ale pozostaje dostępny po zmianie układu.

### AscendingPictureAccentProcess {#AscendingPictureAccentProcess}
```
public static final int AscendingPictureAccentProcess
```

Użyj, aby pokazać rosnącą serię obrazów z opisowym tekstem. Działa najlepiej przy niewielkiej ilości tekstu.

### Balance {#Balance}
```
public static final int Balance
```

Użyj, aby porównać lub pokazać zależność między dwoma pomysłami. Każda z dwóch pierwszych linii tekstu poziomu 1 odpowiada tekstowi u góry jednej strony centralnego punktu. Podkreśla tekst poziomu 2, ograniczony do czterech kształtów po każdej stronie. Waga przechyla się na stronę z większą liczbą kształtów zawierających tekst poziomu 2. Niewykorzystany tekst nie jest wyświetlany, ale pozostaje dostępny po zmianie układu.

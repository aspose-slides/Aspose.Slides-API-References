```

Reprezentuje opcje, które można wykorzystać do wyszukiwania tekstu w prezentacji, slajdzie lub ramce tekstowej.

| [setCaseSensitive(boolean value)](#setCaseSensitive-boolean-) | Ustaw true, aby używać wyszukiwania z uwzględnieniem wielkości liter, false – w przeciwnym razie. |
| [getWholeWordsOnly()](#getWholeWordsOnly--) | Ustaw true, aby dopasowywać tylko całe wyrazy, false – w przeciwnym razie. |
| [setWholeWordsOnly(boolean value)](#setWholeWordsOnly-boolean-) | Ustaw true, aby dopasowywać tylko całe wyrazy, false – w przeciwnym razie. |
| [getIncludeNotes()](#getIncludeNotes--) | Ustaw true, aby uwzględniać tekst zawarty w notatkach slajdu podczas wyszukiwania, zamiany lub podświetlania. |
| [setIncludeNotes(boolean value)](#setIncludeNotes-boolean-) | Ustaw true, aby uwzględniać tekst zawarty w notatkach slajdu podczas wyszukiwania, zamiany lub podświetlania. |

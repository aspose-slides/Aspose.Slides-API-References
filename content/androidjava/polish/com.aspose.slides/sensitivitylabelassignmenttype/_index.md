Reprezentuje metodę przypisania etykiety poufności.

## Pola

| Pole | Opis |
| --- | --- |
| [Standard](#Standard) | Użyj dla każdej etykiety poufności, która nie została bezpośrednio zastosowana przez użytkownika. |
| [Privileged](#Privileged) | Użyj dla każdej etykiety poufności, która została bezpośrednio zastosowana przez użytkownika. |

### Standardowy {#Standard}
```
public static final int Standard
```

Użyj dla każdej etykiety poufności, która nie została bezpośrednio zastosowana przez użytkownika. Obejmuje to wszystkie domyślne etykiety oraz etykiety stosowane automatycznie.

### Uprawniony {#Privileged}
```
public static final int Privileged
```

Użyj dla każdej etykiety poufności, która została bezpośrednio zastosowana przez użytkownika. Obejmuje to wszystkie ręcznie zastosowane etykiety poufności, a także rekomendowane lub obowiązkowe etykietowanie oraz wszelkie funkcje, w których użytkownik decyduje, którą etykietę poufności zastosować.

Représente la méthode d'affectation pour l'étiquette de sensibilité.
## Champs

| Champ | Description |
| --- | --- |
| [Standard](#Standard) | Use for any sensitivity label that was not directly applied by the user. |
| [Privileged](#Privileged) | Use for any sensitivity label that was directly applied by the user. |
### Standard {#Standard}
```
public static final int Standard
```

Utilisez pour toute étiquette de sensibilité qui n'a pas été appliquée directement par l'utilisateur. Cela inclut toutes les étiquettes par défaut, les étiquettes appliquées automatiquement.

### Privilégié {#Privileged}
```
public static final int Privileged
```

Utilisez pour toute étiquette de sensibilité qui a été appliquée directement par l'utilisateur. Cela inclut toutes les étiquettes de sensibilité appliquées manuellement ainsi que les étiquetages recommandés ou obligatoires ou toute fonction où l'utilisateur décide quelle étiquette de sensibilité appliquer.

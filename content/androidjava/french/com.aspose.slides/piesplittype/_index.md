Représente un type de points de division dans le deuxième secteur ou barre d’un graphique en camembert-de-camembert ou en barre-de-camembert.
## Champs

| Champ | Description |
| --- | --- |
| [Default](#Default) | Indique que les points de données seront divisés en utilisant le mécanisme par défaut pour ce type de graphique. |
| [Custom](#Custom) | Indique que les points de données seront divisés entre le camembert et le deuxième graphique selon les valeurs de division personnalisées. |
| [ByPercentage](#ByPercentage) | Indique que les points de données seront divisés entre le camembert et le deuxième graphique en plaçant les points dont le pourcentage est inférieur à Split Position % dans le deuxième graphique. |
| [ByPos](#ByPos) | Indique que les points de données seront divisés entre le camembert et le deuxième graphique en plaçant la dernière Split Position des points de données dans le deuxième graphique. |
| [ByValue](#ByValue) | Indique que les points de données seront divisés entre le camembert et le deuxième graphique en plaçant les points de données dont la valeur est inférieure à Split Position dans le deuxième graphique. |

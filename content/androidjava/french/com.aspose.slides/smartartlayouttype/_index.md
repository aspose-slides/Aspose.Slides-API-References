### AccentProcess {#AccentProcess}
```
public static final int AccentProcess
```

Utilisé pour afficher une progression, une chronologie ou des étapes séquentielles dans une tâche, un processus ou un flux de travail. Fonctionne bien pour illustrer à la fois le texte Niveau 1 et le texte Niveau 2.

### AccentedPicture {#AccentedPicture}
```
public static final int AccentedPicture
```

Utilisé pour afficher une idée centrale, photographique, avec des idées liées sur le côté. Le texte Niveau 1 supérieur apparaît au-dessus de l’image centrale. Le texte correspondant aux autres formes Niveau 1 apparaît à côté des petites images circulaires. Cette mise en page fonctionne également bien sans texte.

### AlternatingFlow {#AlternatingFlow}
```
public static final int AlternatingFlow
```

Utilisé pour afficher des groupes d’informations ou des étapes séquentielles dans une tâche, un processus ou un flux de travail. Met l’accent sur l’interaction ou les relations entre les groupes d’informations.

### AlternatingHexagons {#AlternatingHexagons}
```
public static final int AlternatingHexagons
```

Utilisé pour représenter une série d’idées interconnectées. Le texte Niveau 1 apparaît à l’intérieur des hexagones. Le texte Niveau 2 apparaît à l’extérieur des formes.

### AlternatingPictureBlocks {#AlternatingPictureBlocks}
```
public static final int AlternatingPictureBlocks
```

```
public static final int AlternatingPictureCircles
```

### ArrowRibbon {#ArrowRibbon}
```
public static final int ArrowRibbon
```

Utilisé pour afficher des concepts liés ou contrastés avec une certaine connexion, comme des forces opposées. Les deux premières lignes du texte Niveau 1 sont utilisées pour le texte dans les flèches. Le texte non utilisé n’apparaît pas, mais reste disponible si vous changez de mise en page.

### AscendingPictureAccentProcess {#AscendingPictureAccentProcess}
```
public static final int AscendingPictureAccentProcess
```

### Balance {#Balance}
```
public static final int Balance
```

Utilisé pour comparer ou afficher la relation entre deux idées. Chacune des deux premières lignes du texte Niveau 1 correspond au texte en haut d’un côté du point central. Met l’accent sur le texte Niveau 2, limité à quatre formes de chaque côté du point central. L’équilibre penche du côté qui a le plus de formes contenant du texte Niveau 2. Le texte non utilisé n’apparaît pas, mais reste disponible si vous changez de mise en page.

### BasicVenn {#BasicVenn}
```
public static final int BasicVenn
```

S’il y a quatre lignes ou moins de texte Niveau 1, le texte est à l’intérieur des cercles. S’il y a plus de quatre lignes de texte Niveau 1, le texte est à l’extérieur des cercles. Le texte non utilisé n’apparaît pas, mais reste disponible si vous changez de mise en page.

## Champs

| Champ | Description |
| --- | --- |
| [NotDefined](#NotDefined) | La forme du graphique n’est pas définie. |
| [Box](#Box) | Spécifie que le graphique doit être dessiné avec une forme de boîte. |
| [Cone](#Cone) | Spécifie que le graphique doit être dessiné en cône, la base du cône étant sur le sol et le point du cône au sommet du marqueur de données. |
| [ConeToMax](#ConeToMax) | Spécifie que le graphique doit être dessiné avec des cônes tronqués de sorte que le point du cône corresponde à la valeur maximale des données. |
| [Cylinder](#Cylinder) | Spécifie que le graphique doit être dessiné sous forme de cylindre. |
| [Pyramid](#Pyramid) | Spécifie que le graphique doit être dessiné sous forme de pyramide rectangulaire, la base de la pyramide étant sur le sol et le point de la pyramide au sommet du marqueur de données. |
| [PyramidToMaximum](#PyramidToMaximum) | Spécifie que le graphique doit être dessiné avec des cônes tronqués de sorte que le point du cône corresponde à la valeur maximale des données. |

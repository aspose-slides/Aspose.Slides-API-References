java.lang.Object, com.aspose.ms.System.ValueType, com.aspose.ms.System.Enum
```
public final class InkEffectType extends System.Enum
```

Spécifie un ensemble d'effets visuels prédéfinis pour le rendu de l'encre. Chaque effet correspond à une texture ou une image qui simule une apparence d'encre stylisée. Ces valeurs peuvent être utilisées pour personnaliser le style visuel des traits d'encre numériques lors du rendu.

## Champs

| Champ | Description |
| --- | --- |
| [NotDefined](#NotDefined) | L'effet n'est pas défini ou est inconnu. Cette valeur peut être utilisée comme valeur par défaut ou de repli. |
| [Bronze](#Bronze) | Une texture métallique chaude, brunâtre, ressemblant à de l'encre bronze. |
| [Galaxy](#Galaxy) | Une texture colorée et scintillante ressemblant à une galaxie, avec des tons cosmiques. |
| [Gold](#Gold) | Une texture métallique dorée brillante qui donne aux traits d'encre un aspect luxueux. |
| [Lava](#Lava) | Une texture ardente ressemblant à de la lave en fusion, avec des teintes rouges et oranges. |
| [Ocean](#Ocean) | Une texture bleu profond, semblable à un fluide, qui imite les vagues de l'océan ou l'encre à base d'eau. |
| [RainbowGlitter](#RainbowGlitter) | Un effet de paillettes arc-en-ciel coloré et scintillant utilisé pour des traits d'encre festifs ou vibrants. |
| [RoseGold](#RoseGold) | Un mélange doux de rose et d'or, similaire à l'or rose, pour des traits d'encre élégants. |
| [Silver](#Silver) | Une texture métallique argentée fraîche qui simule l'encre argent classique. |

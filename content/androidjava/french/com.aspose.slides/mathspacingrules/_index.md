Types d'écart (espacement horizontal) entre les colonnes d'une matrice
## Champs

| Champ | Description |
| --- | --- |
| [SingleSpacingGap](#SingleSpacingGap) | Simple (1 em pour les colonnes ou 1 ligne pour les lignes) |
| [OneAndHalfSpacingGap](#OneAndHalfSpacingGap) | Une fois et demie (1.5 ems pour les colonnes ou 1.5 lignes pour les lignes) |
| [DoubleSpacingGap](#DoubleSpacingGap) | Double espacement (2 ems pour les colonnes ou 2 lignes pour les lignes) |
| [Exactly](#Exactly) | Valeur exacte : pour les colonnes, se base sur la valeur de ColumnGap, mesurée en twips ; pour les lignes, se base sur la valeur de RowGap, mesurée en twips |
| [Multiple](#Multiple) | Multiple : pour les colonnes, se base sur la valeur de ColumnGap, mesurée par incréments de 0.5 em ; pour les lignes, se base sur la valeur de RowGap, mesurée en lignes |

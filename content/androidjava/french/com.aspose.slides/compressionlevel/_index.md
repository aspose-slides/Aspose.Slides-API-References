Spécifie les niveaux de compression ZIP pour les fichiers OpenXML. Des niveaux plus élevés offrent une meilleure compression au prix d’un traitement plus lent.

## Champs

| Champ | Description |
| --- | --- |
| [None](#None) | Aucune compression n’est appliquée. |
| [Level1](#Level1) | Compression la plus rapide avec le taux de compression le plus bas. |
| [Level2](#Level2) | Compression plus rapide avec un taux de compression légèrement meilleur que [Level1](../../com.aspose.slides/compressionlevel\#Level1). |
| [Level3](#Level3) | Offre une meilleure compression que [Level2](../../com.aspose.slides/compressionlevel\#Level2) avec un impact de performance modéré. |
| [Level4](#Level4) | Offre une meilleure compression que [Level3](../../com.aspose.slides/compressionlevel\#Level3). |
| [Level5](#Level5) | Fournit une compression améliorée par rapport à [Level4](../../com.aspose.slides/compressionlevel\#Level4) avec un temps de traitement supplémentaire. |
| [Level6](#Level6) | Compression standard, offrant un bon équilibre entre vitesse de compression et taille du fichier. |
| [Level7](#Level7) | Offre une compression plus élevée que [Level6](../../com.aspose.slides/compressionlevel\#Level6) avec un traitement plus lent. |
| [Level8](#Level8) | Offre une compression plus élevée que [Level7](../../com.aspose.slides/compressionlevel\#Level7). |
| [Level9](#Level9) | Compression maximale. |

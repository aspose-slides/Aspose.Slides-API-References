```
public final void setVisibilityOnAllTitleSlides(boolean isVisible)
```

Modifie la visibilité des espaces réservés de pied de page, de date-heure et de numéro de page pour toutes les diapositives de titre et pour la première diapositive de mise en page. Diapositives de titre \\u2013 diapositives basées sur la première diapositive de mise en page (indépendamment du type de cette première mise en page).

**Paramètres :**
| Paramètre | Type | Description |
| --- | --- | --- |
| isVisible | boolean | true - rend les espaces réservés visibles, sinon les masque. |

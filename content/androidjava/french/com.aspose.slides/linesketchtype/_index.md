Représente le type de croquis ou l'effet qui a été attribué à une forme.
## Champs

| Champ | Description |
| --- | --- |
| [NotDefined](#NotDefined) | Spécifie qu'un effet Sketch de forme est indéfini. |
| [None](#None) | Spécifie qu'une forme n'a aucun effet Sketch. |
| [Curved](#Curved) | Spécifie qu'une forme possède l'effet Curved, qui transforme chaque bord de la forme en une grande courbe douce. |
| [Freehand](#Freehand) | Spécifie qu'une forme possède l'effet Freehand, qui ressemble le plus à une ligne dessinée de manière imparfaite. |
| [Scribble](#Scribble) | Spécifie qu'une forme possède l'effet Scribble, qui présente une oscillation exagérée comme si elle était dessinée volontairement désordonnée. |
### NotDefined {#NotDefined}

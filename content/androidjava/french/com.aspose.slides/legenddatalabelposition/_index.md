Détermine la position des étiquettes de données.
## Champs

| Champ | Description |
| --- | --- |
| [NotDefined](#NotDefined) | La position n'est pas définie. |
| [Bottom](#Bottom) | Spécifie que les étiquettes de données doivent être affichées sous le repère de données. |
| [BestFit](#BestFit) | Spécifie que les étiquettes de données doivent être affichées dans la meilleure position. |
| [Center](#Center) | Spécifie que les étiquettes de données doivent être affichées centrées sur le repère de données. |
| [InsideBase](#InsideBase) | Spécifie que les étiquettes de données doivent être affichées à l'intérieur de la base du repère de données. |
| [InsideEnd](#InsideEnd) | Spécifie que les étiquettes de données doivent être affichées à l'intérieur de l'extrémité du repère de données. |
| [Left](#Left) | Spécifie que les étiquettes de données doivent être affichées à gauche du repère de données. |
| [OutsideEnd](#OutsideEnd) | Spécifie que les étiquettes de données doivent être affichées à l'extérieur de l'extrémité du repère de données. |
| [Right](#Right) | Spécifie que les étiquettes de données doivent être affichées à droite du repère de données. |
| [Top](#Top) | Spécifie que les étiquettes de données doivent être affichées au-dessus du repère de données. |

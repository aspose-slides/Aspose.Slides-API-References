public interface IHeadingPair
```

Representa una propiedad 'Heading pair' del documento. Indica el nombre del grupo de partes del documento y el número de partes en el grupo.
## Métodos

| Método | Descripción |
| --- | --- |
| [getName()](#getName--) | Devuelve el nombre del grupo de partes del documento. |
| [getCount()](#getCount--) | Devuelve el número de partes en el grupo. |
### getName() {#getName--}

```

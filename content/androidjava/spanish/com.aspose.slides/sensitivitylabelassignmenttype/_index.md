Representa el método de asignación para la etiqueta de sensibilidad.
## Campos

| Campo | Descripción |
| --- | --- |
| [Standard](#Standard) | Uso para cualquier etiqueta de sensibilidad que no fue aplicada directamente por el usuario. |
| [Privileged](#Privileged) | Uso para cualquier etiqueta de sensibilidad que fue aplicada directamente por el usuario. |
### Estándar {#Standard}
```
public static final int Standard
```

Uso para cualquier etiqueta de sensibilidad que no fue aplicada directamente por el usuario. Esto incluye cualquier etiqueta predeterminada, etiquetas aplicadas automáticamente.

### Privilegiado {#Privileged}
```
public static final int Privileged
```

Uso para cualquier etiqueta de sensibilidad que fue aplicada directamente por el usuario. Esto incluye cualquier etiqueta de sensibilidad aplicada manualmente así como etiquetado recomendado o obligatorio o cualquier función donde el usuario decide qué etiqueta de sensibilidad aplicar.

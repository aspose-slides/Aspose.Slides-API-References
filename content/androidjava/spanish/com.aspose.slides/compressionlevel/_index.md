Especifica los niveles de compresión ZIP para archivos OpenXML. Los niveles más altos proporcionan una mejor compresión a costa de un procesamiento más lento.
## Fields

| Campo | Descripción |
| --- | --- |
| [None](#None) | No se aplica compresión. |
| [Level1](#Level1) | Compresión más rápida con la relación de compresión más baja. |
| [Level2](#Level2) | Compresión más rápida con una relación de compresión ligeramente mejor que [Level1](../../com.aspose.slides/compressionlevel\#Level1). |
| [Level3](#Level3) | Proporciona una mejor compresión que [Level2](../../com.aspose.slides/compressionlevel\#Level2) con un impacto moderado en el rendimiento. |
| [Level4](#Level4) | Proporciona una mejor compresión que [Level3](../../com.aspose.slides/compressionlevel\#Level3). |
| [Level5](#Level5) | Proporciona una compresión mejorada respecto a [Level4](../../com.aspose.slides/compressionlevel\#Level4) con tiempo de procesamiento adicional. |
| [Level6](#Level6) | Compresión estándar, que ofrece un buen equilibrio entre la velocidad de compresión y el tamaño del archivo. |
| [Level7](#Level7) | Proporciona una mayor compresión que [Level6](../../com.aspose.slides/compressionlevel\#Level6) con un procesamiento más lento. |
| [Level8](#Level8) | Proporciona una mayor compresión que [Level7](../../com.aspose.slides/compressionlevel\#Level7). |
| [Level9](#Level9) | Compresión máxima. |

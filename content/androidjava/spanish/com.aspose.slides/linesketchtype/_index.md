Representa el tipo de boceto o efecto que se ha asignado a una forma.
## Campos

| Campo | Descripción |
| --- | --- |
| [NotDefined](#NotDefined) | Especifica que un efecto Sketch de una forma es indefinido. |
| [None](#None) | Especifica que una forma no tiene efecto Sketch. |
| [Curved](#Curved) | Especifica que una forma tiene el efecto Curved, que convierte cada borde de la forma en una gran curva suave. |
| [Freehand](#Freehand) | Especifica que una forma tiene el efecto Freehand, que se asemeja más a una línea dibujada de forma imperfecta. |
| [Scribble](#Scribble) | Especifica que una forma tiene el efecto Scribble, que presenta una oscilación exagerada como si se dibujara intencionalmente desordenado. |

Especifica la justificación del párrafo matemático (una serie de instancias adyacentes de texto matemático dentro del mismo párrafo)
## Campos

| Campo | Descripción |
| --- | --- |
| [LeftJustified](#LeftJustified) | Alinear el texto matemático a la izquierda |
| [RightJustified](#RightJustified) | Alinear el texto matemático a la derecha |
| [Centered](#Centered) | Alinear el texto matemático al centro |
| [CenteredAsGroup](#CenteredAsGroup) | Centrar en la página todo el grupo de bloques de matemáticas como una sola unidad. |

### CenteredAsGroup {#CenteredAsGroup}
```
public static final int CenteredAsGroup
```


Centrar en la página todo el grupo de bloques de matemáticas como una sola unidad. Esto significa que las instancias de texto matemático pueden alinearse entre sí, pero todo el grupo de texto matemático se centra como un todo.

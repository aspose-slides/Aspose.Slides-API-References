### NotDefined {#NotDefined}

### PersistPropertyBag {#PersistPropertyBag}
```
public static final int PersistPropertyBag
```

Especifica que el control ActiveX se persiste mediante persistencia basada en bolsa de propiedades. La persistencia basada en bolsa de propiedades almacena un control ActiveX mediante una colección de pares nombre y valor que especifican los datos persistidos por el control ActiveX.

### PersistStream {#PersistStream}
```
public static final int PersistStream
```

Especifica que el control ActiveX se persiste mediante una persistencia basada en flujo que no admite la inicialización del control ActiveX a un estado predeterminado.

### PersistStreamInit {#PersistStreamInit}
```
public static final int PersistStreamInit
```

Especifica que el control ActiveX se persiste mediante una persistencia basada en flujo que admite la inicialización del control ActiveX a un estado predeterminado.

### PersistStorage {#PersistStorage}
```
public static final int PersistStorage
```

Especifica que el control ActiveX se persiste mediante una persistencia basada en almacenamiento.

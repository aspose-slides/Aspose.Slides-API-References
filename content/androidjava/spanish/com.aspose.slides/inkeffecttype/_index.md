Especifica un conjunto de efectos visuales predefinidos para el renderizado de tinta. Cada efecto corresponde a una textura o imagen que simula una apariencia de tinta estilizada. Estos valores pueden usarse para personalizar el estilo visual de los trazos de tinta digital durante el renderizado.
## Campos

| Campo | Descripción |
| --- | --- |
| [NotDefined](#NotDefined) | El efecto no está definido o es desconocido. |
| [Bronze](#Bronze) | Una textura metálica cálida y marrón que se asemeja a tinta de bronce. |
| [Galaxy](#Galaxy) | Una textura colorida y brillante que se asemeja a una galaxia, con tonos cósmicos. |
| [Gold](#Gold) | Una textura metálica dorada brillante que otorga a los trazos de tinta una apariencia lujosa. |
| [Lava](#Lava) | Una textura ardiente que recuerda a lava fundida, con matices rojos y naranjas. |
| [Ocean](#Ocean) | Una textura azul profunda, similar a un fluido, que imita olas oceánicas o tinta basada en agua. |
| [RainbowGlitter](#RainbowGlitter) | Un efecto brillante y colorido de destellos arcoíris, usado para trazos de tinta festivos o vibrantes. |
| [RoseGold](#RoseGold) | Una mezcla suave de rosa y oro, similar al rosado, para trazos de tinta elegantes. |
| [Silver](#Silver) | Una textura plateada metálica fresca que simula tinta plateada clásica. |

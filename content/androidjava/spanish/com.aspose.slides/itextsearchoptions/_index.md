public interface ITextSearchOptions
```

Representa opciones que pueden usarse para buscar texto en una Presentation, Slide o TextFrame.
## Métodos

| Método | Descripción |
| --- | --- |
| [getCaseSensitive()](#getCaseSensitive--) | Establezca true para usar búsqueda sensible a mayúsculas y minúsculas, false en caso contrario. |
| [setCaseSensitive(boolean value)](#setCaseSensitive-boolean-) | Establezca true para usar búsqueda sensible a mayúsculas y minúsculas, false en caso contrario. |
| [getWholeWordsOnly()](#getWholeWordsOnly--) | Establezca true para coincidir solo palabras completas, false en caso contrario. |
| [setWholeWordsOnly(boolean value)](#setWholeWordsOnly-boolean-) | Establezca true para coincidir solo palabras completas, false en caso contrario. |
| [getIncludeNotes()](#getIncludeNotes--) | Establezca true para incluir texto contenido en notas de diapositiva al realizar búsquedas de texto, reemplazos o resaltados. |
| [setIncludeNotes(boolean value)](#setIncludeNotes-boolean-) | Establezca true para incluir texto contenido en notas de diapositiva al realizar búsquedas de texto, reemplazos o resaltados. |

```
public final class LayoutTargetType extends System.Enum
```

Si la disposición del área del gráfico se define manualmente, esta propiedad indica si el área del gráfico se dispone por su interior (sin incluir los ejes y las etiquetas de los ejes) o por su exterior (incluyendo los ejes y las etiquetas de los ejes).
## Campos

| Campo | Descripción |
| --- | --- |
| [Inner](#Inner) | Indica que el tamaño del área del gráfico determinará el tamaño del área del gráfico, sin incluir las marcas de graduación y las etiquetas de los ejes. |
| [Outer](#Outer) | Indica que el tamaño del área del gráfico determinará el tamaño del área del gráfico, las marcas de graduación y las etiquetas de los ejes. |
### Interno {#Inner}
```
public static final int Inner
```


Indica que el tamaño del área del gráfico determinará el tamaño del área del gráfico, sin incluir las marcas de graduación y las etiquetas de los ejes.

### Externo {#Outer}
```
public static final int Outer
```


Indica que el tamaño del área del gráfico determinará el tamaño del área del gráfico, las marcas de graduación y las etiquetas de los ejes.

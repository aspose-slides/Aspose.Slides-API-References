### AccentProcess {#AccentProcess}
```
public static final int AccentProcess
```

Se utiliza para mostrar una progresión, una línea de tiempo o pasos secuenciales en una tarea, proceso o flujo de trabajo. Funciona bien para ilustrar tanto texto de Nivel 1 como de Nivel 2.

### AccentedPicture {#AccentedPicture}
```
public static final int AccentedPicture
```

Se utiliza para mostrar una idea fotográfica central con ideas relacionadas a los lados. El texto de Nivel 1 superior aparece sobre la imagen central. El texto correspondiente para otras formas de Nivel 1 aparece junto a las pequeñas imágenes circulares. Este diseño también funciona bien sin texto.

### AlternatingFlow {#AlternatingFlow}
```
public static final int AlternatingFlow
```

Se utiliza para mostrar grupos de información o pasos secuenciales en una tarea, proceso o flujo de trabajo. Enfatiza la interacción o relaciones entre los grupos de información.

### AlternatingHexagons {#AlternatingHexagons}
```
public static final int AlternatingHexagons
```

Se utiliza para representar una serie de ideas interconectadas. El texto de Nivel 1 aparece dentro de los hexágonos. El texto de Nivel 2 aparece fuera de las formas.

### AlternatingPictureBlocks {#AlternatingPictureBlocks}
```
public static final int AlternatingPictureBlocks
```

Se utiliza para mostrar una serie de imágenes de arriba a abajo. El texto aparece alternadamente a la derecha o izquierda de la imagen.

### AlternatingPictureCircles {#AlternatingPictureCircles}
```
public static final int AlternatingPictureCircles
```

Se utiliza para mostrar un conjunto de imágenes con texto. El texto correspondiente aparece en los círculos centrales con las imágenes alternando de izquierda a derecha.

### ArrowRibbon {#ArrowRibbon}
```
public static final int ArrowRibbon
```

Se utiliza para mostrar conceptos relacionados o contrastantes con alguna conexión, como fuerzas opuestas. Las dos primeras líneas de texto de Nivel 1 se usan para el texto en las flechas. El texto no utilizado no aparece, pero queda disponible si se cambian los diseños.

### AscendingPictureAccentProcess {#AscendingPictureAccentProcess}
```
public static final int AscendingPictureAccentProcess
```

Se utiliza para mostrar una serie ascendente de imágenes con texto descriptivo. Funciona mejor con una pequeña cantidad de texto.

### Balance {#Balance}
```
public static final int Balance
```

Se utiliza para comparar o mostrar la relación entre dos ideas. Cada una de las dos primeras líneas de texto de Nivel 1 corresponde al texto en la parte superior de un lado del punto central. Enfatiza el texto de Nivel 2, que se limita a cuatro formas en cada lado del punto central. El equilibrio se inclina hacia el lado con más formas que contienen texto de Nivel 2. El texto no utilizado no aparece, pero queda disponible si se cambian los diseños.

### BasicBendingProcess {#BasicBendingProcess}
```
public static final int BasicBendingProcess
```

Se utiliza para mostrar una progresión o pasos secuenciales en una tarea, proceso o flujo de trabajo. Maximiza tanto el espacio horizontal como el vertical para las formas.

### BasicBlockList {#BasicBlockList}

### BasicChevronProcess {#BasicChevronProcess}

```
public static final int BasicCycle
```

```
public static final int BasicMatrix
```

### BasicPie {#BasicPie}
```
public static final int BasicPie
```

Se utiliza para mostrar cómo las partes individuales forman un todo. Las primeras siete líneas de texto de Nivel 1 corresponden a las formas de porción de pastel distribuidas uniformemente. La forma de texto de Nivel 1 superior aparece fuera del resto del pastel para enfatizar. El texto no utilizado no aparece, pero queda disponible si se cambian los diseños.

### BasicTimeline {#BasicTimeline}
```
public static final int BasicTimeline
```

### BasicVenn {#BasicVenn}
```
public static final int BasicVenn
```

Se utiliza para mostrar relaciones superpuestas o interconectadas. Las primeras siete líneas de texto de Nivel 1 corresponden a un círculo. Si hay cuatro o menos líneas de texto de Nivel 1, el texto está dentro de los círculos; si hay más de cuatro, el texto está fuera de los círculos. El texto no utilizado no aparece, pero queda disponible si se cambian los diseños.

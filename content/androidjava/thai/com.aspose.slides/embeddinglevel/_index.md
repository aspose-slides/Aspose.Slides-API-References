### Installable {#Installable}
```
public static final int Installable
```

แบบอักษรที่ตั้งค่านี้แสดงว่ามันสามารถฝังลงและติดตั้งอย่างถาวรบนระบบระยะไกลโดยแอปพลิเคชัน ผู้ใช้ของระบบระยะไกลจะได้รับสิทธิ์, ความรับผิดชอบและใบอนุญาตที่เหมือนกับผู้ซื้อเดิมของแบบอักษร และต้องปฏิบัติตามข้อตกลงการใช้สิทธิผู้ใช้งานสุดท้าย, ลิขสิทธิ์, สิทธิบัตรการออกแบบ, และ/หรือเครื่องหมายการค้าที่เหมือนกับผู้ซื้อเดิม

### Restricted {#Restricted}
```
public static final int Restricted
```

แบบอักษรที่ตั้งค่าไบต์นี้เท่านั้นต้องห้ามแก้ไข, ฝังหรือแลกเปลี่ยนในรูปแบบใด ๆ หากไม่ได้รับอนุญาตจากเจ้าของตามกฎหมายก่อน

### PreviewPrint {#PreviewPrint}
```
public static final int PreviewPrint
```

เมื่อไบต์นี้ถูกตั้งค่า, แบบอักษรสามารถฝังลงและโหลดชั่วคราวบนระบบระยะไกลได้ เอกสารที่มีแบบอักษร Preview & Print ต้องเปิดในโหมด "อ่านอย่างเดียว;" ไม่สามารถแก้ไขเอกสารได้

### Editable {#Editable}
```
public static final int Editable
```

เมื่อไบต์นี้ถูกตั้งค่า, แบบอักษรสามารถฝังลงได้แต่ต้องติดตั้งชั่วคราวบนระบบอื่นเท่านั้น ในทางตรงกันข้ามกับแบบอักษร Preview & Print, เอกสารที่มีแบบอักษร Editable สามารถเปิดเพื่ออ่านได้, การแก้ไขได้รับอนุญาต, และสามารถบันทึกการเปลี่ยนแปลงได้

### NoSubsetting {#NoSubsetting}
```
public static final int NoSubsetting
```

เมื่อไบต์นี้ถูกตั้งค่า, แบบอักษรไม่สามารถย่อยส่วนย่อยก่อนการฝังได้ ข้อจำกัดการฝังอื่นที่ระบุในไบต์ 0-3 และ 9 ยัง 적용ด้วย

### BitmapOnly {#BitmapOnly}
```
public static final int BitmapOnly
```

เมื่อไบต์นี้ถูกตั้งค่า, จะสามารถฝังเฉพาะบิตแมพที่รวมอยู่ในแบบอักษรเท่านั้น ไม่สามารถฝังข้อมูลโครงร่างได้ หากไม่มีบิตแมพในแบบอักษร, แบบอักษรจะถือว่าไม่สามารถฝังได้และบริการการฝังจะล้มเหลว

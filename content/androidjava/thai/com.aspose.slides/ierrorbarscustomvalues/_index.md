### getXMinus() {#getXMinus--}
```
public abstract IDoubleChartValue getXMinus()
```


ระบุค่าของแถบความผิดพลาดในทิศทางลบ สามารถใช้ได้หากประเภทค่าของแถบความผิดพลาดเป็น Custom และอนุญาตให้ใช้ ErrorBarsXFormat ในกรณีอื่น ๆ จะคืนค่า null เป็นคุณสมบัติแบบอ่านอย่างเดียว [IDoubleChartValue](../../com.aspose.slides/idoublechartvalue).

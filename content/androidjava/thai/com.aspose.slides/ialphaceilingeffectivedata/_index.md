**อินเทอร์เฟซทั้งหมดที่นำไปใช้:**
[com.aspose.slides.IEffectEffectiveData](../../com.aspose.slides/ieffecteffectivedata)
```
public interface IAlphaCeilingEffectiveData extends IEffectEffectiveData
```

ออบเจ็กต์ไม่เปลี่ยนแปลงที่เป็นตัวแทนของเอฟเฟกต์ Alpha Ceiling. ค่าของ Alpha (ความทึบแสง) ที่มากกว่า ศูนย์ จะถูกเปลี่ยนเป็น 100%. อีกนัยหนึ่ง ใด ๆ ที่มีความทึบแสงบางส่วนจะกลายเป็นความทึบแสงเต็มที่.

**อินเทอร์เฟซที่นำไปใช้งานทั้งหมด:**
[com.aspose.slides.IEffectEffectiveData](../../com.aspose.slides/ieffecteffectivedata)
```
public interface IAlphaFloorEffectiveData extends IEffectEffectiveData
```

ออบเจกต์ที่ไม่เปลี่ยนแปลงซึ่งเป็นตัวแทนของผลกระทบ Alpha Floor. ค่าความใส (opacity) ของ Alpha ที่น้อยกว่า 100% จะถูกเปลี่ยนเป็นศูนย์. อีกอย่างคือ สิ่งที่มีความโปร่งใสบางส่วนจะกลายเป็นโปร่งใสเต็มที่.

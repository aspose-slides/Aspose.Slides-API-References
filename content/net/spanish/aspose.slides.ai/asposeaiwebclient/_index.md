## AsposeAIWebClient clase

Una implementación [`IAIWebClient`](../iaiwebclient) incorporada que se conecta al propio LLM de Aspose. Este es el cliente predeterminado usado por el constructor sin parámetros [`SlidesAIAgent`](../slidesaiagent/slidesaiagent).

```csharp
public sealed class AsposeAIWebClient : IAIWebClient, IDisposable
```

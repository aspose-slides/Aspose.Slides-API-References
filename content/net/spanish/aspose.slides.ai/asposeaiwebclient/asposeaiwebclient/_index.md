## AsposeAIWebClient() {#constructor}

Crea una instancia del cliente web Aspose AI que se conecta al punto de enlace predeterminado de Aspose LLM. Este es el cliente usado por el constructor sin parámetros [`SlidesAIAgent`](../../slidesaiagent/slidesaiagent), por lo que crearlo explícitamente solo es necesario cuando se pasa el cliente al constructor [`SlidesAIAgent`](../../slidesaiagent/slidesaiagent) directamente.

## AsposeAIWebClient(string, HttpClient) {#constructor_3}

Crea una instancia del cliente web Aspose AI que se conecta a una URL de punto de enlace personalizada usando un HttpClient gestionado externamente.

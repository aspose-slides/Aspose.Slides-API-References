## OpenAIWebClient(string, string, string, HttpClient) {#constructor_1}

Crea una instancia del cliente web OpenAI que utiliza un `HttpClient` gestionado externamente. El `HttpClient` proporcionado no es eliminado por esta instancia y permanece bajo la responsabilidad del llamador.

```csharp
public OpenAIWebClient(string model, string apiKey, string organizationId, HttpClient httpClient)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| model | String | Modelo de lenguaje OpenAI. Valores posibles: - gpt-4o - gpt-4o-mini - o1 - o1-mini - o3 - o3-mini |
| apiKey | String | Clave API de OpenAI |
| organizationId | String | ID de organización (opcional) |
| httpClient | HttpClient | Una instancia de `HttpClient` gestionada externamente |

### Excepciones

| excepción | condición |
| --- | --- |
| ArgumentException | El valor de la clave API no puede ser nulo o vacío |
| ArgumentException | El valor del modelo de texto no puede ser nulo o vacío |
| ArgumentNullException | HttpClient no puede ser nulo |

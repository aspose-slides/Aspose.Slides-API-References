## GeneratePresentation(string, PresentationContentAmountType) {#generatepresentation}

Genera una instancia de presentación a partir de una descripción de texto. Proporcione un tema, ideas, citas o fragmentos de texto en el idioma requerido.

```csharp
public IPresentation GeneratePresentation(string description, 
    PresentationContentAmountType presentationContentAmount)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| description | String | The topic, ideas, quotes, or text snippets. |
| presentationContentAmount | PresentationContentAmountType | The amount of content in the resulting presentation. |

### Excepciones

| excepción | condición |
| --- | --- |
| ArgumentException | AI chat instruction can't be null or empty. |

### Observaciones

El ejemplo a continuación usa el [`AsposeAIWebClient`](../../asposeaiwebclient) predeterminado, que es creado por el constructor sin parámetros [`SlidesAIAgent`](../slidesaiagent) y se conecta al LLM propio de Aspose. Para usar un proveedor de IA diferente, suministre su propio LLM o personalice la conexión (por ejemplo, proporcionando su propio `HttpClient`), pase una implementación [`IAIWebClient`](../../iaiwebclient) al constructor [`SlidesAIAgent`](../slidesaiagent). Las implementaciones disponibles incluyen:

* [`AsposeAIWebClient`](../../asposeaiwebclient)
* [`OpenAIWebClient`](../../openaiwebclient)
* [`OpenAICompatibleWebClient`](../../openaicompatiblewebclient)

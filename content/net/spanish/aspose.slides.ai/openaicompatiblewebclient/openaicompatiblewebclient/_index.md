## OpenAICompatibleWebClient(string, string, string, HttpClient) {#constructor_1}

Crea una instancia del cliente web compatible con OpenAI que utiliza un HttpClient gestionado externamente. El HttpClient proporcionado no es eliminado por esta instancia y sigue siendo de propiedad del llamador.

```csharp
public OpenAICompatibleWebClient(string model, string apiKey, string baseUrl, HttpClient httpClient)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| model | String | Nombre del modelo compatible con el proveedor de LLM. |
| apiKey | String | clave API (token). |
| baseUrl | String | URL base del LLM compatible con OpenAI. |
| httpClient | HttpClient | Instancia de HttpClient gestionada externamente. |

### Excepciones

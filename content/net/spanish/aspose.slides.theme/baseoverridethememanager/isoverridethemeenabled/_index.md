## Propiedad BaseOverrideThemeManager.IsOverrideThemeEnabled

Determina si OverrideTheme reemplaza el tema efectivo heredado o no. Para habilitar OverrideTheme para la sobreescritura, utiliza los métodos OverrideTheme.Init*(). Para deshabilitar OverrideTheme de la sobreescritura, utiliza el método OverrideTheme.Clear(). Booleano de solo lectura.

```csharp
public bool IsOverrideThemeEnabled { get; }
```

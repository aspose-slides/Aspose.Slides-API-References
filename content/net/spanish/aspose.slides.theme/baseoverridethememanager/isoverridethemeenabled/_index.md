## Propiedad BaseOverrideThemeManager.IsOverrideThemeEnabled

Determina si OverrideTheme sobrescribe el tema efectivo heredado o no. Para habilitar OverrideTheme para sobrescribir, utilice los métodos OverrideTheme.Init*(). Para deshabilitar OverrideTheme de sobrescribir, utilice el método OverrideTheme.Clear(). Booleano de solo lectura.

public bool IsOverrideThemeEnabled { get; }

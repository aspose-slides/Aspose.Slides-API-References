## IOverrideThemeManager.IsOverrideThemeEnabled property

Determina si OverrideTheme anula el tema efectivo heredado o no. Para habilitar OverrideTheme para anular, use los métodos OverrideTheme.Init*(). Para deshabilitar OverrideTheme de anular, use el método OverrideTheme.Clear().Boolean .

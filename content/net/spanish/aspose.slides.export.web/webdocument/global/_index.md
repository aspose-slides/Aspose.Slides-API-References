## Propiedad WebDocument.Global

Devuelve el almacenamiento global del documento. Solo lectura [`Storage`](../../storage).


### Ejemplos

Usando esta propiedad `Global` (implementación de la interfaz [`Storage`](../../storage)), se puede utilizar una propiedad para más tarde en la plantilla:

```csharp
[C#]
var options = new WebDocumentOptions
{
    TemplateEngine = new RazorTemplateEngine(),
    OutputSaver = new FileOutputSaver(),
    EmbedImages = false
};

WebDocument document = new WebDocument(options);

// poner la propiedad "slideMargin" en uso desde las plantillas
document.Global.Put("slideMargin", 10);

// ... configurar otras opciones del documento y luego guardar el documento
document.Save();
```

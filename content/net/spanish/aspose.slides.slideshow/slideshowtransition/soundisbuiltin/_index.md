## Propiedad SlideShowTransition.SoundIsBuiltIn

Especifica si este sonido es un sonido incorporado o no. Si este atributo se establece en verdadero, la aplicación generadora se alerta para comprobar el atributo de nombre especificado para este sonido en su lista de sonidos incorporados y puede luego mostrar un nombre o interfaz de usuario personalizada según sea necesario. Booleano de lectura y escritura.

public bool SoundIsBuiltIn { get; set; }

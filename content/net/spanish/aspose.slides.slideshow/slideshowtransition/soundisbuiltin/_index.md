## SlideShowTransition.SoundIsBuiltIn property

Especifica si este sonido es un sonido incorporado o no. Si este atributo se establece en verdadero, la aplicación generadora es alertada para que verifique el atributo de nombre especificado para este sonido en su lista de sonidos incorporados y puede mostrar un nombre o interfaz personalizada según sea necesario.

## Propiedad SlideShowTransition.SoundName

Especifica un nombre legible por humanos para el sonido de la transición. La propiedad [`Sound`](../sound) debe asignarse para obtener o establecer el nombre del sonido. Cadena de lectura y escritura.

```csharp
public string SoundName { get; set; }
```

### Excepciones

| excepción | condición |
| --- | --- |
| [PptxException](../../../aspose.slides/pptxexception) | Cuando la propiedad [`Sound`](../sound) no está asignada. |

### Comentarios

Este nombre aparece en la interfaz de usuario de PowerPoint al configurar manualmente el sonido de transición.

## ExternalResourceResolver class

Clase de devolución de llamada utilizada para resolver recursos externos durante la importación de documentos Html, Svg. El uso de este solucionador podría crear una vulnerabilidad cuando el archivo HTML o SVG proporcionado por el cliente hará que el software del servidor obtenga un archivo local o de red. Utilizar con precaución. Se recomienda no especificar ExternalResourceResolver en absoluto (solo se leerán los objetos incrustados) o crear alguna subclase que verifique si el uri especificado es válido.

public class ExternalResourceResolver : IExternalResourceResolver

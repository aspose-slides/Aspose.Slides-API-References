## Método IExternalResourceResolver.ResolveUri

Resuelve la URI absoluta a partir de las URIs base y relativas.

```csharp
public string ResolveUri(string baseUri, string relativeUri)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| baseUri | String | URI base de los objetos de enlace |
| relativeUri | String | URI relativa al objeto vinculado. |

### Valor de retorno

URI absoluta o null si la URI relativa no puede ser resuelta.

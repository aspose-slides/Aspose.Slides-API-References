Contiene clases que importan datos en presentaciones de Microsoft PowerPoint.

## Clases

| Clase | Descripción |
| --- | --- |
| [ExternalResourceResolver](./externalresourceresolver) | Clase de devolución de llamada utilizada para resolver recursos externos durante la importación de documentos Html, Svg. Usar este resolutor podría crear una vulnerabilidad cuando un archivo HTML o SVG proporcionado por el cliente haga que el software del servidor obtenga un archivo local o de red. Usar con precaución. Se recomienda no especificar ExternalResourceResolver en absoluto (solo se leerán objetos incrustados) o crear alguna subclase que verifique si el uri especificado es válido. |
| [PdfImportOptions](./pdfimportoptions) | Representa las opciones de importación de PDF |

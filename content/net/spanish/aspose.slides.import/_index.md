Contiene clases que importan datos en presentaciones de Microsoft PowerPoint.

## Clases

| Clase | Descripción |
| --- | --- |
| [ExternalResourceResolver](./externalresourceresolver) | Clase de callback utilizada para resolver recursos externos durante la importación de documentos Html y Svg. El uso de este resolvedor podría crear una vulnerabilidad cuando un archivo HTML o SVG proporcionado por el cliente haga que el software del servidor obtenga un archivo local o de red. Utilizar con precaución. Se recomienda no especificar ExternalResourceResolver en absoluto (solo se leerán objetos incrustados) o crear una subclase que verifique si el URI especificado es válido. |

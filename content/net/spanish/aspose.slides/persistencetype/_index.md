## Enumeración PersistenceType

Especifica el método utilizado para almacenar las propiedades del control ActiveX.

```csharp
public enum PersistenceType
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| NotDefined | `-1` | La id de persistencia no está especificada. |
| PersistPropertyBag | `0` | Especifica que el control ActiveX se persiste utilizando una persistencia basada en un conjunto de propiedades. La persistencia basada en un conjunto de propiedades almacena un control ActiveX mediante una colección de pares de nombre y valor que especifican los datos persistidos por el control ActiveX. |
| PersistStream | `1` | Especifica que el control ActiveX se persiste utilizando una persistencia basada en flujo que no admite la inicialización del control ActiveX a un estado predeterminado. |
| PersistStreamInit | `2` | Especifica que el control ActiveX se persiste utilizando una persistencia basada en flujo que admite la inicialización del control ActiveX a un estado predeterminado. |
| PersistStorage | `3` | Especifica que el control ActiveX se persiste utilizando una persistencia basada en almacenamiento. |

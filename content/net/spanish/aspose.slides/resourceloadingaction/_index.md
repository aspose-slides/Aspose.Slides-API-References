## Enumeración ResourceLoadingAction

Especifica el modo de carga de recursos externos.

```csharp
public enum ResourceLoadingAction
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Default | `0` | Aspose.Slides cargará el recurso externo como de costumbre. |
| Skip | `1` | Aspose.Slides omitirá la carga del recurso externo. Solo se almacenará el enlace sin datos para una imagen. |
| UserProvided | `2` | Aspose.Slides utilizará el array de bytes proporcionado por el usuario en [`SetData`](../iresourceloadingargs/setdata) como datos de imagen. |

## Método IFontsManager.GetFontBytes

Recupera el arreglo de bytes que representa los datos de la fuente para un estilo de fuente y datos de fuente especificados.

```csharp
public byte[] GetFontBytes(IFontData fontData, FontStyle fontStyle)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| fontData | IFontData | El objeto de datos de fuente que contiene la información sobre la fuente [`IFontData`](../../ifontdata). |
| fontStyle | FontStyle | El estilo de la fuente para el cual se van a recuperar los datos FontStyle. |

### Valor de Retorno

Un arreglo de bytes que contiene los datos de la fuente para el estilo de fuente especificado. Si los datos de la fuente o el estilo no se encuentran, devuelve null.

### Ejemplos

```csharp
[C#]
using (Presentation pres = new Presentation ("Presentation.pptx"))
{
    // Recupera todas las fuentes utilizadas en la presentación
    IFontData[] fonts = pres.FontsManager.GetFonts();

    // Obtiene el arreglo de bytes que representa el estilo regular de la primera fuente en la presentación
    byte[] bytes = pres.FontsManager.GetFontBytes(fonts[0], FontStyle.Regular);
}
```

## Método IFontsManager.GetSubstitutions

Obtiene la información sobre las fuentes que serán reemplazadas en el renderizado de la presentación.

```csharp
public IEnumerable<FontSubstitutionInfo> GetSubstitutions()
```

### Valor de Retorno

Colección de todos los sustitutos de fuentes [`FontSubstitutionInfo`](../../fontsubstitutioninfo).

### Ejemplos

```csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    foreach (var fontSubstitution in pres.FontsManager.GetSubstitutions())
    {
        Console.WriteLine("{0} -> {1}", fontSubstitution.OriginalFontName, fontSubstitution.SubstitutedFontName);
    }
}        
```

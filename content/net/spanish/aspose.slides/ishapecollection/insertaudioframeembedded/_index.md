## InsertAudioFrameEmbedded(int, float, float, float, float, Stream) {#insertaudioframeembedded_1}

Inserta un AudioFrame con un archivo de audio incrustado. El sonido del archivo de audio incrustado puede ser solo WAV. Agrega un nuevo audio a la lista Presentation.Audios.

```csharp
public IAudioFrame InsertAudioFrameEmbedded(int index, float x, float y, float width, float height, 
    Stream audio_stream)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| index | Int32 | El índice basado en cero en el que se debe insertar el valor. |
| x | Single | Coordenada X de un nuevo marco de audio. |
| y | Single | Coordenada Y de un nuevo marco de audio. |
| width | Single | Ancho de un nuevo marco de audio. |
| height | Single | Altura de un nuevo marco de audio. |
| audio_stream | Stream | Flujo de audio. |

### Valor de retorno

Objeto AudioFrame creado.

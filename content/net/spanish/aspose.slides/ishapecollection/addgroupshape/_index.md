## AddGroupShape(ISvgImage, float, float, float, float) {#addgroupshape_1}

Crea un nuevo GroupShape, lo llena con formas convertidas de SVG y lo agrega al final de la colección.

```csharp
public IGroupShape AddGroupShape(ISvgImage svgImage, float x, float y, float width, float height)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| svgImage | ISvgImage | Objeto de imagen SVG [`ISvgImage`](../../isvgimage) |
| x | Single | La coordenada X para el lado izquierdo del marco del grupo de formas. |
| y | Single | La coordenada Y para el lado superior del marco del grupo de formas. |
| width | Single | El ancho del grupo del marco del grupo de formas. |
| height | Single | La altura de un grupo del marco del grupo de formas. |

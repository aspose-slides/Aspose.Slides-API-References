## IShapeCollection.AddSummaryZoomFrame método

Agrega un nuevo objeto de Resumen Zoom al final de una colección.

```csharp
public ISummaryZoomFrame AddSummaryZoomFrame(float x, float y, float width, float height)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| x | Single | Coordenada X de un nuevo marco de Sección Zoom Single. |
| y | Single | Coordenada Y de un nuevo marco de Sección Zoom Single. |
| width | Single | Anchura de un nuevo marco de Sección Zoom Single. |
| height | Single | Altura de un nuevo marco de Sección Zoom Single. |

### Valor de Retorno

Objeto de Resumen Zoom creado [`ISummaryZoomFrame`](../../isummaryzoomframe).

### Excepciones

| excepción | condición |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | No hay secciones en la presentación, o la diapositiva objetivo no pertenece a ninguna sección. |

### Observaciones

Este método crea un nuevo Resumen Zoom y coloca una colección de objetos en él para todas las secciones en esta presentación.

### Ejemplos

Este ejemplo demuestra cómo agregar un objeto de Resumen Zoom al final de una colección (suponga que hay al menos dos secciones en la presentación "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSummaryZoomFrame(150, 20, 500, 250);
}
```

## Enumeración EmbeddingLevel

Representa los derechos de licencia para incrustar la fuente.

```csharp
[Flags]
public enum EmbeddingLevel : ushort
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Installable | `0` | Las fuentes con esta configuración indican que pueden ser incrustadas e instaladas permanentemente en el sistema remoto por una aplicación. El usuario del sistema remoto adquiere los derechos, obligaciones y licencias idénticos para esa fuente como el comprador original de la fuente, y está sujeto al mismo acuerdo de licencia de usuario final, derechos de autor, patente de diseño y/o marca registrada que el comprador original. |
| Restricted | `2` | Las fuentes que tienen solo este bit configurado no deben ser modificadas, incrustadas o intercambiadas de ninguna manera sin primero obtener el permiso del propietario legal. |
| PreviewPrint | `4` | Cuando este bit está configurado, la fuente puede ser incrustada y cargada temporalmente en el sistema remoto. Los documentos que contienen fuentes de Vista previa e impresión deben abrirse en "solo lectura"; no se pueden aplicar ediciones al documento. |
| Editable | `8` | Cuando este bit está configurado, la fuente puede ser incrustada pero solo debe ser instalada temporalmente en otros sistemas. A diferencia de las fuentes de Vista previa e impresión, los documentos que contienen fuentes editables pueden abrirse para lectura, se permite la edición y se pueden guardar los cambios. |
| NoSubsetting | `100` | Cuando este bit está configurado, la fuente no puede ser subdividida antes de ser incrustada. Otras restricciones de incrustación especificadas en los bits 0-3 y 9 también se aplican. |
| BitmapOnly | `200` | Cuando este bit está configurado, solo se pueden incrustar los mapas de bits contenidos en la fuente. No se pueden incrustar datos de contorno. Si no hay mapas de bits disponibles en la fuente, entonces la fuente se considera no incrustable y los servicios de incrustación fallarán. |

## Enumeración PathFillModeType

Especifica la manera en que un camino debe ser llenado

```csharp
public enum PathFillModeType : byte
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| None | `0` | Sin Relleno de Camino Esto especifica que el camino correspondiente no debe tener relleno. |
| Normal | `1` | Relleno de Camino Normal Esto especifica que el camino correspondiente debe tener un color sombreado normalmente aplicado a su relleno. |
| Lighten | `2` | Relleno de Camino Aclarado Esto especifica que el camino correspondiente debe tener un color ligeramente sombreado aplicado a su relleno. |
| LightenLess | `3` | Relleno de Camino Aclarado Menos Esto especifica que el camino correspondiente debe tener un color ligeramente más claro sombreado aplicado a su relleno. |
| Darken | `4` | Relleno de Camino Oscurecido Esto especifica que el camino correspondiente debe tener un color sombreado más oscuro aplicado a su relleno. |
| DarkenLess | `5` | Relleno de Camino Oscurecido Menos Esto especifica que el camino correspondiente debe tener un color ligeramente más oscuro sombreado aplicado a su relleno. |

## LoadOptions.InterruptionToken propiedad

El token para monitorear solicitudes de interrupción. Este token gestiona toda la vida útil de la instancia [`IPresentation`](../../ipresentation). Cualquier operación de largo plazo, como cargar o guardar una presentación, será interrumpida mediante la llamada al método [`Interrupt`](../../interruptiontokensource/interrupt) de la clase [`InterruptionTokenSource`](../../interruptiontokensource).

public IInterruptionToken InterruptionToken { get; set; }

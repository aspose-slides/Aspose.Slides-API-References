## Enumeración LineSketchType

Representa qué tipo de boceto o efecto se ha asignado a una forma.

```csharp
public enum LineSketchType
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| NotDefined | `-1` | Especifica que el efecto de boceto de una forma no está definido. |
| None | `0` | Especifica que una forma no tiene efecto de boceto. Esto es equivalente a que esta propiedad esté vacía. |
| Curved | `1` | Especifica que una forma tiene el efecto Curved, que convierte cada borde de la forma en una gran curva suave. |
| Freehand | `2` | Especifica que una forma tiene el efecto Freehand, que se asemeja más a una línea dibujada de manera imperfecta. |
| Scribble | `3` | Especifica que una forma tiene el efecto Scribble, que tiene una oscilación exagerada como si se dibujara deliberadamente de manera desordenada. |

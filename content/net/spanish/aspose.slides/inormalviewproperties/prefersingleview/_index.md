## INormalViewProperties.PreferSingleView property

Especifica si el usuario prefiere ver una región de contenido único a pantalla completa sobre la vista normal estándar con tres regiones de contenido. Si está habilitado, la aplicación puede elegir mostrar una de las regiones de contenido en toda la ventana. Booleano de lectura/escritura.

```csharp
public bool PreferSingleView { get; set; }
```

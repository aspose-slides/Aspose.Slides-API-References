## TextFrameFormat.RotationAngle property

Especifica la rotación personalizada que se aplica al texto dentro del cuadro delimitador. Si no se especifica , se usa la rotación de la forma adjunta. Si se especifica, se aplica independientemente de la forma. Es decir, a la forma se le puede aplicar una rotación en además del propio texto al que se le ha aplicado una rotación. El valor resultante de la rotación de texto visual se resume a partir de esta propiedad y el tipo vertical predefinido en la propiedad TextVerticalType. Lectura/escrituraSingle .

```csharp
public float RotationAngle { get; set; }
```

### Ejemplos

Considere el caso en el que se aplica una rotación de 90 grados en el sentido de las agujas del reloj a una forma. Además de esto, el cuerpo del texto tiene aplicada una rotación de -90 grados en el sentido contrario a las agujas del reloj. Luego, la forma resultante parecería estar rotada, pero el texto dentro de ella parecería como si no hubiera sido rotado en absoluto.

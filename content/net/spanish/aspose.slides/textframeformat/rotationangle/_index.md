## Propiedad TextFrameFormat.RotationAngle

Especifica la rotación personalizada que se está aplicando al texto dentro del cuadro delimitador. Si no se especifica, se utiliza la rotación de la forma acompañante. Si se especifica, esto se aplica de forma independiente a la forma. Es decir, la forma puede tener una rotación aplicada además de que el texto en sí tenga una rotación aplicada. El valor resultante de la rotación visual del texto se resume a partir de esta propiedad y el tipo vertical predefinido en la propiedad TextVerticalType. Lectura/escritura Single.

```csharp
public float RotationAngle { get; set; }
```

### Ejemplos

Considera el caso en el que a una forma se le ha aplicado una rotación de 90 grados en el sentido de las agujas del reloj. Además de esto, el cuerpo del texto en sí tiene una rotación de -90 grados en sentido contrario a las agujas del reloj aplicada a él. Entonces, la forma resultante parecería estar rotada, pero el texto dentro de ella parecería como si no hubiera sido rotado en absoluto.

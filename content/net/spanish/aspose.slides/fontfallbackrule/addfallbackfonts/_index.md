## AddFallBackFonts(string) {#addfallbackfonts}

Agrega una nueva fuente(s) a la lista de fuentes de FallBack.

```csharp
public void AddFallBackFonts(string fontName)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| fontName | String | Nombre o nombres de la fuente (separados por comas) para FallBack |

### Ejemplos

```csharp
[C#]
// Crear nueva instancia de FontFallBackRule
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho");

//Agregar una segunda fuente a la regla 
newRule.AddFallBackFonts("MS Gothic");
//Agregar una tercera y cuarta fuentes a la regla 
newRule.AddFallBackFonts("Tahoma, Times New Roman");
```

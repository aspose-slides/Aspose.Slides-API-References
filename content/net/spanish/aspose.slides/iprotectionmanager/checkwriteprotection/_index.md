## Método IProtectionManager.CheckWriteProtection

Determina si una presentación está protegida por contraseña para modificar.

```csharp
public bool CheckWriteProtection(string password)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| password | String | La contraseña para verificar. |

### Valor de Retorno

Verdadero si la contraseña es válida; de lo contrario, falso.

### Observaciones

1. Debe verificar la propiedad [`IsWriteProtected`](../iswriteprotected) antes de llamar a este método. 2. Cuando la contraseña es nula o vacía, este método devuelve falso.

### Ejemplos

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

## IProtectionManager.CheckWriteProtection method

Determina si una presentación está protegida con contraseña para modificarla.

```csharp
public bool CheckWriteProtection(string password)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| password | String | La contraseña para comprobar. |

### Valor_devuelto

True si la contraseña es válida; en caso contrario, falso.

### Observaciones

1. Debe comprobar el[`IsWriteProtected`](../iswriteprotected) propiedad antes de llamar a este método. 2. Cuando la contraseña es nula o está vacía, este método devuelve falso.

### Ejemplos

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

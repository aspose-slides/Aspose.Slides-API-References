## IProtectionManager.EncryptDocumentProperties property

Esta propiedad tiene sentido si la presentación está protegida con contraseña. Si es verdadero, entonces las propiedades del documento están encriptadas en el archivo de presentación. Si es falso, entonces las propiedades del documento son públicas mientras la presentación está encriptada. Booleano de lectura/escritura.

public bool EncryptDocumentProperties { get; set; }

## Propiedad ILoadOptions.InterruptionToken

El token para monitorear las solicitudes de interrupción. Este token gestiona toda la vida útil de la instancia de [`IPresentation`](../../ipresentation). Cualquier operación de larga duración, como la carga o guardado de una presentación, será interrumpida mediante la llamada al método [`Interrupt`](../../iinterruptiontokensource/interrupt) de [`IInterruptionTokenSource`](../../iinterruptiontokensource).

public IInterruptionToken InterruptionToken { get; set; }

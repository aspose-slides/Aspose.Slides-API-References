## Propiedad BaseSlide.ShowMasterShapes

Especifica si las formas en la diapositiva maestra deben mostrarse en las diapositivas o no. Para la propia diapositiva maestra, esta propiedad siempre devuelve `false`. Booleano de lectura/escritura.

```csharp
public abstract bool ShowMasterShapes { get; set; }
```

### Excepciones

| excepción | condición |
| --- | --- |
| NotSupportedException | Lanzada si se establece `true` para la diapositiva maestra. |

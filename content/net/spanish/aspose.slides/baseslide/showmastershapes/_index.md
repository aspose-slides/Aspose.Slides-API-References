## BaseSlide.ShowMasterShapes property

Especifica si las formas en la diapositiva maestra deben mostrarse en las diapositivas o no. Para la diapositiva maestra en sí, esta propiedad siempre devuelve `false`. Booleano de lectura/escritura.

public abstract bool ShowMasterShapes { get; set; }

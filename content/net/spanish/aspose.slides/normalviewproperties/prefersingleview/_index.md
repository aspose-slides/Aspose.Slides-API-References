## NormalViewProperties.PreferSingleView property

Especifica si el usuario prefiere ver una región de contenido única de ventana completa sobre la vista normal estándar con tres regiones de contenido. Si se habilita, la aplicación puede elegir mostrar una de las regiones de contenido en toda la ventana. Booleano de lectura/escritura.

public bool PreferSingleView { get; set; }

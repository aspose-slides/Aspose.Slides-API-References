## Propiedad IDocumentProperties.AppVersion

Devuelve la versión de la aplicación. Solo lectura String.

```csharp
public string AppVersion { get; }
```

### Observaciones

El contenido de este elemento debe estar en la forma XX.YYYY, donde X e Y representan valores numéricos; de lo contrario, el documento se considerará no conforme. Aspose.Slides representa su versión en el formato XX.YYZZ, donde: XX - versión principal YY - versión secundaria ZZ - versión de parche Por ejemplo, el valor 23.0105 significa versión 23.1.5 de Aspose.Slides.

## Propiedad IDocumentProperties.AppVersion

Devuelve la versión de la aplicación. Cadena de solo lectura.

```csharp
public string AppVersion { get; }
```

### Observaciones

El contenido de este elemento debe tener la forma XX.YYYY, donde X e Y representan valores numéricos; de lo contrario, el documento se considerará no conforme. Aspose.Slides representa su versión en el formato XX.YYZZ, donde: XX - versión mayor YY - versión menor ZZ - versión de parche Por ejemplo, el valor 23.0105 significa la versión 23.1.5 de Aspose.Slides.

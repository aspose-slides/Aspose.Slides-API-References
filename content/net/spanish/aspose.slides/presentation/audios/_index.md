## Propiedad Presentation.Audios

Devuelve la colección de todos los archivos de audio incrustados en la presentación. Solo lectura [`IAudioCollection`](../../iaudiocollection).

```csharp
public IAudioCollection Audios { get; }
```

### Ejemplos

Los siguientes ejemplos muestran cómo agregar un hipervínculo a un archivo de audio.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    IAudio audio = pres.Audios.AddAudio(File.ReadAllBytes("audio.mp3"));
    IAudioFrame audioFrame = pres.Slides[0].Shapes.AddAudioFrameEmbedded(10, 10, 100, 100, audio);
    audioFrame.HyperlinkClick = new Hyperlink("https://www.aspose.com/");
    audioFrame.HyperlinkClick.Tooltip = "Más del 70% de las empresas de Fortune 100 confían en las API de Aspose";
    pres.Save("pres-out.pptx", SaveFormat.Pptx);
}
```

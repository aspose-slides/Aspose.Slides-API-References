## Propiedad MasterNotesSlide.ShowMasterShapes

Especifica si las formas en la diapositiva maestra deben mostrarse en las diapositivas o no. Para la diapositiva maestra en sí, esta propiedad siempre devuelve `false`. Escribible/lectura Booleano.

```csharp
public override bool ShowMasterShapes { get; set; }
```

### Excepciones

| excepción | condición |
| --- | --- |
| NotSupportedException | Se lanza si se establece en `true` para la diapositiva maestra. |

## DocumentProperties.ScaleCrop property

Indica el modo de visualización de la miniatura del documento. Establezca este elemento en **true** para habilitar el escalado de la miniatura del documento a la pantalla. Establezca este elemento en **false** para habilitar el recorte de la miniatura del documento para mostrar solo las secciones que se ajustan a la pantalla. Booleano de lectura/escritura.

public bool ScaleCrop { get; set; }

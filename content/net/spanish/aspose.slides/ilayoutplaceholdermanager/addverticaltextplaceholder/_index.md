## Método ILayoutPlaceholderManager.AddVerticalTextPlaceholder

Agrega una nueva forma de marcador de posición a la diapositiva de diseño para contener contenido de texto en dirección vertical.

```csharp
public IAutoShape AddVerticalTextPlaceholder(float x, float y, float width, float height)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| x | Single | La coordenada X de la nueva forma de marcador de posición. |
| y | Single | La coordenada Y de la nueva forma de marcador de posición. |
| width | Single | El ancho de la nueva forma de marcador de posición. |
| height | Single | La altura de la nueva forma de marcador de posición. |

### Valor de retorno

Se crea un [`IAutoShape`](../../iautoshape) con un marcador de posición de Texto (Vertical).

### Ejemplos

El siguiente ejemplo muestra cómo agregar la forma de marcador de posición de Texto (Vertical) a la diapositiva de diseño.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalTextPlaceholder(20, 20, 300, 500);
}
```

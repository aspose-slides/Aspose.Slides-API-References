## AddFromPdf(Stream, PdfImportOptions) {#addfrompdf_1}

Crea diapositivas a partir del documento PDF y las agrega al final de la colección.

```csharp
public ISlide[] AddFromPdf(Stream pdfStream, PdfImportOptions pdfImportOptions)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| pdfStream | Stream | Un stream que se utilizará como fuente del documento PDF |
| pdfImportOptions | PdfImportOptions | Opciones para la importación de PDF |

### Valor de Retorno

Diapositivas añadidas

### Ejemplos

Ejemplo:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    using (Stream stream = new FileStream("document.pdf", FileMode.Open, FileAccess.Read, FileShare.Read))
    {
        pres.Slides.AddFromPdf(stream, new PdfImportOptions { DetectTables = true });
    }
    
    pres.Save("fromPdfDocument.pptx", SaveFormat.Pptx);
}
```

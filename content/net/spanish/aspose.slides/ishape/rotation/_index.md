## Propiedad IShape.Rotation

Devuelve o establece el número de grados que la forma especificada está girada alrededor del eje z. Un valor positivo indica rotación en el sentido de las agujas del reloj; un valor negativo indica rotación en sentido contrario. Lectura/escritura Single.

```csharp
public float Rotation { get; set; }
```

### Observaciones

El valor devuelto siempre está definido (no es float.NaN). El valor asignado debe estar definido (no es float.NaN). Puedes establecer valores indefinidos para las propiedades de instancia de RawFrame.

## IShape.Rotation property

Devuelve o establece el número de grados que gira la forma especificada alrededor del eje z. Un valor positivo indica rotación en el sentido de las agujas del reloj; un valor negativo indica rotación en sentido antihorario. Lectura/escrituraSingle .

```csharp
public float Rotation { get; set; }
```

### Observaciones

El valor devuelto siempre está definido (no es float.NaN). El valor asignado debe definirse (no float.NaN). Puede establecer valores indefinidos para las propiedades de instancia de RawFrame.

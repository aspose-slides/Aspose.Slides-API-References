## FontsLoader.LoadExternalFonts method

Agrega carpetas adicionales para buscar fuentes.

```csharp
public static void LoadExternalFonts(string[] directories)
```

| Parámeter | Tipo | Descripción |
| --- | --- | --- |
| directories | String[] | Directorios para leer fuentes adicionales. |

### Ejemplos

Los siguientes ejemplos muestran cómo cargar fuentes personalizadas desde .TTF

```csharp
[C#]
// La ruta al directorio de documentos.
string dataDir = "C:\\";
// carpetas para buscar fuentes
String[] folders = new String[] { dataDir };
// Cargar las fuentes del directorio de fuentes personalizadas
FontsLoader.LoadExternalFonts(folders);
// Hacer algún trabajo y realizar la representación de la presentación/diapositivas
using (Presentation presentation = new Presentation(dataDir + "DefaultFonts.pptx"))
{
    presentation.Save(dataDir + "NewFonts_out.pptx", SaveFormat.Pptx);
	// Limpiar la caché de fuentes
	FontsLoader.ClearCache();
}
```

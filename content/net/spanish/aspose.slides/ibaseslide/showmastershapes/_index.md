## Propiedad IBaseSlide.ShowMasterShapes

Especifica si las formas en el diseño maestro deben mostrarse en las diapositivas o no. Para el diseño maestro en sí, esta propiedad siempre devuelve `false`. Booleano de lectura/escritura.

```csharp
public bool ShowMasterShapes { get; set; }
```

### Excepciones

| excepción | condición |
| --- | --- |
| NotSupportedException | Se lanza si se establece en `true` para el diseño maestro. |

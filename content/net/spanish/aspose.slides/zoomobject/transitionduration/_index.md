## ZoomObject.TransitionDuration property

Obtiene o establece la duración de la transición entre Zoom y la diapositiva. Lectura/escritura Single. Valor predeterminado: 1.0f

```csharp
public float TransitionDuration { get; set; }
```

### Remarks

Si no se especifica (TransitionDur = 0), utilizará la transición de la diapositiva de destino y los tiempos asociados con esa transición.

### Examples

El ejemplo demuestra cómo cambiar la duración de la transición entre Zoom y la diapositiva:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
    zoomFrame.TransitionDuration = 2.5f;
}
```

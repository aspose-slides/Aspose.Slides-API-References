## Propiedad BlobManagementOptions.TempFilesRootPath

La ruta raíz donde se crearán los archivos temporales. Se utilizará el directorio temporal del sistema de forma predeterminada. El proceso de alojamiento debe tener permisos para crear archivos y carpetas allí.

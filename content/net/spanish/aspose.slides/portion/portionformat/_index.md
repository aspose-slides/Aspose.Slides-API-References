## Propiedad Portion.PortionFormat

Devuelve un objeto de formato que contiene propiedades de formato explícitamente establecidas de la porción de texto sin herencia aplicada. Solo lectura [`IPortionFormat`](../../iportionformat).

```csharp
public IPortionFormat PortionFormat { get; }
```

### Observaciones

El objeto de formato contiene los parámetros de formato definidos solo para la porción actual, los datos heredados no se aplican.

Para obtener los valores efectivos, incluidos los heredados, utiliza el método [`GetEffective`](../../portionformat/geteffective).

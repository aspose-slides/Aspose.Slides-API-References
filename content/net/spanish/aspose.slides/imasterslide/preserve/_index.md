## IMasterSlide.Preserve property

Determina si el máster correspondiente se elimina cuando se eliminan todas las diapositivas que siguen a ese máster. Nota: Aspose.Slides nunca eliminará ningún máster no utilizado por sí mismo; para eliminar realmente los másteres no utilizados, llama a [`RemoveUnused`](../../imasterslidecollection/removeunused) Booleano de lectura/escritura.

```csharp
public bool Preserve { get; set; }
```

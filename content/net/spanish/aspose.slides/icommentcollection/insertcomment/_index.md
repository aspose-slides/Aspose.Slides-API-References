## Método ICommentCollection.InsertComment

Inserta un nuevo comentario en una colección en el índice especificado.

```csharp
public IComment InsertComment(int index, string text, ISlide slide, PointF position, 
    DateTime creationTime)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| index | Int32 | Índice del elemento en una colección donde se debe insertar el comentario. |
| text | String | Texto plano de un nuevo comentario. |
| slide | ISlide | Diapositiva en una presentación donde agregar un nuevo comentario. |
| position | PointF | Posición en una diapositiva donde agregar un nuevo comentario. |
| creationTime | DateTime | Hora de creación de un comentario. |

### Valor de Retorno

Comentario insertado.

## CommentCollection.AddModernComment method

Añadir un nuevo comentario moderno al final de una colección.

```csharp
public IModernComment AddModernComment(string text, ISlide slide, IShape shape, PointF position, 
    DateTime creationTime)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| text | String | Texto sin formato de un nuevo comentario moderno. |
| slide | ISlide | Deslice en una presentación donde agregar un nuevo comentario moderno. |
| shape | IShape | Forma en una diapositiva a la que se asocia un nuevo comentario moderno. |
| position | PointF | Posición en una diapositiva donde agregar un nuevo comentario moderno. |
| creationTime | DateTime | Momento de la creación de un comentario moderno. |

### Valor_devuelto

Añadido comentario moderno.

### Ejemplos

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ICommentAuthor newAuthor = pres.CommentAuthors.AddAuthor("Some Author", "SA");
    newAuthor.Comments.AddModernComment("This is modern comment", pres.Slides[0], null, new PointF(100, 100), DateTime.Now);

    pres.Save(outPptxFileName, SaveFormat.Pptx);
}
```

## FontFallBackRulesCollection.Add método

Agrega una regla de FallBack especificada al final de la colección.

```csharp
public void Add(IFontFallBackRule sourceRule)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| sourceRule | IFontFallBackRule | Regla especificada para agregar |

### Ejemplos

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    //Obteniendo la colección de reglas vacía o preinicializada de FontsManager
    IFontFallBackRulesCollection rulesList = pres.FontsManager.FontFallBackRulesCollection;

    //Agregando nueva regla a la colección
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));
}
```

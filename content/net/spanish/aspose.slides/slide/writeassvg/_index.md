## WriteAsSvg(Stream) {#writeassvg}

Guarda el contenido de la diapositiva como un archivo SVG.

```csharp
public void WriteAsSvg(Stream stream)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| stream | Stream | Flujo de destino |

### Ejemplos

El siguiente ejemplo de código muestra cómo convertir la primera diapositiva de una presentación de PowerPoint en un archivo SVG.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.svg"))
    {
        // Guarda la primera diapositiva como un archivo SVG
        pres.Slides[0].WriteAsSvg(fileStream);
    }
}
```

```csharp

{
    using (Stream fileStream = System.IO.File.Create("slide_1.svg"))

        var options = new SVGOptions() { VectorizeText = true };
```

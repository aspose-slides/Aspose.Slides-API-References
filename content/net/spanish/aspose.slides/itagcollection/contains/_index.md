## Método ITagCollection.Contains

Determina si la colección contiene un nombre específico.

```csharp
public bool Contains(string name)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| name | String | La clave a localizar. |

### Valor de Retorno

## Propiedad ProtectionManager.IsEncrypted

Obtiene un valor que indica si esta instancia está encriptada. Booleano de solo lectura.

```csharp
public bool IsEncrypted { get; }
```

### Valor de la Propiedad

`true` si la presentación se cargó desde un archivo encriptado o se llamó al método [`Encrypt`](../encrypt); de lo contrario, `false`.

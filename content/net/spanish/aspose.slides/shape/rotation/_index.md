## Shape.Rotation property

Devuelve o establece el número de grados que la forma especificada está rotada alrededor del eje z. Un valor positivo indica rotación en el sentido de las agujas del reloj; un valor negativo indica rotación en sentido antihorario. Lectura/escritura Single.

```csharp
public float Rotation { get; set; }
```

### Remarks

El valor devuelto siempre está definido (no es float.NaN). El valor asignado debe estar definido (no es float.NaN). Puedes establecer valores indefinidos para las propiedades de la instancia RawFrame.

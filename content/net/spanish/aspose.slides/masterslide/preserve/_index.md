## Propiedad MasterSlide.Preserve

Determina si el maestro correspondiente se elimina cuando se eliminan todas las diapositivas que siguen a ese maestro. Nota: Aspose.Slides nunca eliminará ningún maestro sin usar por sí mismo; para eliminar realmente maestros sin usar, llama a [`RemoveUnused`](../../masterslidecollection/removeunused) Lectura/escritura Boolean.

```csharp
public bool Preserve { get; set; }
```

## IPictureFillFormat.DeletePictureCroppedAreas método

Eliminar áreas recortadas de la imagen de relleno.

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### Valor de Retorno

Imagen recortada o imagen original si el recorte no es necesario.

### Observaciones

Este método convierte archivos metafóricos WMF/EMF a imágenes rasterizadas PNG mientras recorta.

### Ejemplos

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Obtiene el PictureFrame
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // Elimina áreas recortadas de la imagen del PictureFrame
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

## Propiedad ISlideShowTransition.SoundIsBuiltIn

Especifica si este sonido es un sonido integrado o no. Si este atributo se establece en verdadero, entonces la aplicación generadora es alertada para que verifique el atributo de nombre especificado para este sonido en su lista de sonidos integrados y puede presentar un nombre o una interfaz personalizada según sea necesario. Booleano de lectura y escritura.

```csharp
public bool SoundIsBuiltIn { get; set; }
```

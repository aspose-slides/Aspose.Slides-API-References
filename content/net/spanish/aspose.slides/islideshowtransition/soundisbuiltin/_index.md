## ISlideShowTransition.SoundIsBuiltIn property

Especifica si este sonido es un sonido incorporado o no. Si este atributo se establece en verdadero, entonces la aplicación generadora es alertada para comprobar el atributo de nombre especificado para este sonido en su lista de sonidos incorporados y puede entonces mostrar un nombre o interfaz personalizada según sea necesario.

## Propiedad ISlideShowTransition.SoundName

Especifica un nombre legible por humanos para el sonido de la transición. La propiedad [`Sound`](../sound) debe ser asignada para obtener o establecer el nombre del sonido. Cadena de lectura/escritura.

```csharp
public string SoundName { get; set; }
```

### Excepciones

| excepción | condición |
| --- | --- |
| [PptxException](../../pptxexception) | Cuando la propiedad [`Sound`](../sound) no está asignada. |

### Observaciones

Este nombre aparece en la interfaz de usuario de PowerPoint al configurar manualmente el sonido de la transición.

## Método SlideUtil.FindAndReplaceText

Encuentra y reemplaza texto en la presentación con el formato dado

```csharp
public static void FindAndReplaceText(IPresentation presentation, bool withMasters, string find, 
    string replace, PortionFormat format = null)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| presentation | IPresentation | Presentación escaneada. |
| withMasters | Boolean | Determina si las diapositivas maestras deben ser escaneadas. |
| find | String | Valor de cadena a encontrar. |
| replace | String | Valor de cadena a reemplazar. |
| format | PortionFormat | Formato para reemplazar la porción de texto. Si es null, se usará el formato del primer carácter de la cadena encontrada |

### Ejemplos

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PortionFormat format = new PortionFormat
    {
        FontHeight = 24f,
        FontItalic = NullableBool.True,
        FillFormat =
        {
            FillType = FillType.Solid,
            SolidFillColor =
            {
                Color = Color.Red
            }
        }
    };
    Aspose.Slides.Util.SlideUtil.FindAndReplaceText(pres, true, "[this block] ", "my text ", format);
    pres.Save("replaced", SaveFormat.Pptx);
}
```

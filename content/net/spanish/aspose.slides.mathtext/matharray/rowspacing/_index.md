## MathArray.RowSpacing property

Espaciado entre filas de un arreglo. Se utiliza solo cuando RowSpacingRule se establece en 3. Exactamente en cuyo caso la unidad de medida son puntos o Múltiple, en cuyo caso la unidad de medida son medios renglones. Por defecto: 0

```csharp
public uint RowSpacing { get; set; }
```

### Examples

Example:

```csharp
[C#]
IMathArray mathArray = new MathArray(new MathematicalText("item1"));
mathArray.RowSpacingRule = MathRowSpacingRule.Exactly;
mathArray.RowSpacing = 10;
```

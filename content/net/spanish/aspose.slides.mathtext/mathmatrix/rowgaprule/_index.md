## MathMatrix.RowGapRule propiedad

El tipo de espacio vertical entre filas de una matriz; las unidades de espacio vertical pueden ser líneas o puntos (almacenados como twips). Predeterminado: SingleSpacingGap (0)

```csharp
public MathSpacingRules RowGapRule { get; set; }
```

### Ejemplos

Ejemplo:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.OneAndHalfSpacingGap;
```

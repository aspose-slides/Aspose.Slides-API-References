## Propiedad MathMatrix.MinColumnWidth

Ancho mínimo de columna en twips (1/20 de un punto). El espacio entre columnas (también referido como "Column Gap" o "Gap Width") se suma al MinColumnWidth para determinar el espaciado total de columnas de la matriz (distancia entre los mismos bordes de diferentes columnas). Por defecto: 0.

```csharp
public uint MinColumnWidth { get; set; }
```

### Ejemplos

Ejemplo:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.MinColumnWidth = 20;
```

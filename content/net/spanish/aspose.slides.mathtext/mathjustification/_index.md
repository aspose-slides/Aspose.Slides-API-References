## MathJustification enumeration

Especifica la justificación del párrafo matemático (una serie de instancias adyacentes de texto matemático dentro del mismo párrafo)

```csharp
public enum MathJustification
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| LeftJustified | `1` | Alinear texto matemático a la izquierda |
| RightJustified | `2` | Alinear texto matemático a la derecha |
| Centered | `3` | Alinear texto matemático al centro |
| CenteredAsGroup | `4` | Centre en la página todo el grupo de bloques matemáticos como una sola unidad. Esto significa que las instancias de texto matemático se pueden alinear entre sí, pero todo el grupo de texto matemático se centra como un todo. |

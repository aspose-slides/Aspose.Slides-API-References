## Propiedad MathBorderBox.StrikethroughBottomLeftToTopRight

Strikethrough de Izquierda-Abajo a Derecha-Arriba (el valor predeterminado es falso). Especifica el estado oculto o visible de una línea diagonal tachada desde la esquina inferior izquierda hasta la esquina superior derecha del cuadro de borde.

```csharp
public bool StrikethroughBottomLeftToTopRight { get; set; }
```

### Ejemplos

Ejemplo:

```csharp
[C#]
MathBorderBox borderBox = new MathBorderBox(new MathematicalText("x"));
borderBox.StrikethroughBottomLeftToTopRight = true;
```

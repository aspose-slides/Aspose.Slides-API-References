## Propiedad MathBorderBox.StrikethroughBottomLeftToTopRight

Strikethrough Bottom-Left to Top-Right (el valor predeterminado es falso). Especifica el estado oculto o mostrado de una línea diagonal con tachadura desde la esquina inferior izquierda hasta la esquina superior derecha del cuadro de borde.

```csharp
public bool StrikethroughBottomLeftToTopRight { get; set; }
```

### Ejemplos

Ejemplo:

```csharp
[C#]
MathBorderBox borderBox = new MathBorderBox(new MathematicalText("x"));
borderBox.StrikethroughBottomLeftToTopRight = true;
```

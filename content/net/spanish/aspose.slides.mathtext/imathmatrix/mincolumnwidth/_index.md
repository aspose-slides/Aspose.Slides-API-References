## Propiedad IMathMatrix.MinColumnWidth

Ancho mínimo de columna en twips (1/20 de un punto). El espaciado entre columnas (también conocido como "Espacio entre Columnas" o "Ancho del Espacio") se suma al MinColumnWidth para determinar la distancia total de Espaciado de Columnas de la Matriz (distancia entre los mismos bordes de diferentes columnas). Predeterminado: 0.

```csharp
public uint MinColumnWidth { get; set; }
```

### Ejemplos

Ejemplo:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.MinColumnWidth = 20;
```

## Propiedad IMathMatrix.RowGap

El valor del espaciado vertical entre filas de una matriz; Si RowGapRule se establece en 3 ("Exactamente"), entonces la unidad se interpreta como twips (1/20 de punto). Si RowGapRule se establece en 4 ("Múltiple"), entonces la unidad se interpreta como media línea. Predeterminado: 0

```csharp
public uint RowGap { get; set; }
```

### Ejemplos

Ejemplo:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

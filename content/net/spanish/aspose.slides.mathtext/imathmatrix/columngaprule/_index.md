## Propiedad IMathMatrix.ColumnGapRule

El tipo de espaciado horizontal entre columnas de una matriz; las unidades de espaciado horizontal pueden ser ems o puntos (almacenados como twips). Por defecto: SingleSpacingGap (0)

```csharp
public MathSpacingRules ColumnGapRule { get; set; }
```

### Ejemplos

Ejemplo:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.ColumnGapRule = MathSpacingRules.OneAndHalfSpacingGap;
```

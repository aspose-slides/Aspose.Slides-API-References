## IMathDelimiter.EndingCharacter property

El delimitador Ending Character especifica el carácter delimitador de cierre o final. Los delimitadores matemáticos son caracteres de cierre como paréntesis, corchetes y llaves. El valor por defecto es ')'.

```csharp
public char EndingCharacter { get; set; }
```

### Ejemplos

Ejemplo:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Join("y").Enclose();
delimiter.EndingCharacter = ']';
```

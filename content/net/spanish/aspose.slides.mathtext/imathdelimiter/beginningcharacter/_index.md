## IMathDelimiter.BeginningCharacter propiedad

El carácter delimitador de inicio especifica el carácter delimitador inicial o de apertura. Los delimitadores matemáticos son caracteres de cierre como paréntesis, corchetes y llaves. El valor predeterminado: '('.

```csharp
public char BeginningCharacter { get; set; }
```

### Ejemplos

Ejemplo:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Join("y").Enclose();
delimiter.BeginningCharacter = '[';
```

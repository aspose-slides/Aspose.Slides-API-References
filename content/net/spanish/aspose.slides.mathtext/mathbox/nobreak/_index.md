## MathBox.NoBreak property

No break Esta propiedad especifica la propiedad "ininterrumpible" en el objeto box. Cuando es verdadero, no se pueden producir saltos de línea dentro del box. Esto puede ser importante para emuladores de operadores que consisten en más de un operador binario. Cuando este elemento no se especifica, pueden ocurrir saltos dentro del box. Predeterminado: verdadero

```csharp
public bool NoBreak { get; set; }
```

### Ejemplos

Ejemplo:

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("*****"));
box.NoBreak = false;
```

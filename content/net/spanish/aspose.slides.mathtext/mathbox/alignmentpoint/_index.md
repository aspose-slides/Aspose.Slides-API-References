## Propiedad MathBox.AlignmentPoint

Cuando es verdadero, este emulador de operador sirve como un punto de alineación; es decir, los puntos de alineación designados en otras ecuaciones se pueden alinear con él. Predeterminado: falso

```csharp
public bool AlignmentPoint { get; set; }
```

### Ejemplos

Ejemplo:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.AlignmentPoint = true;
```

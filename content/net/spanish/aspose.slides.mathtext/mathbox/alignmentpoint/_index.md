## MathBox.AlignmentPoint property  
  
Cuando es verdadero, este emulador de operador sirve como un punto de alineación; es decir, los puntos de alineación designados en otras ecuaciones pueden alinearse con él. Predeterminado: falso  
  
```csharp  
public bool AlignmentPoint { get; set; }  
```

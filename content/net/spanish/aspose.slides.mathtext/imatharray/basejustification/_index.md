## Propiedad IMathArray.BaseJustification

Especifica la alineación de la matriz en relación con el texto circundante. El texto fuera de la matriz se puede alinear con la parte inferior, la parte superior o el centro de un objeto de matriz. Valor por defecto: Centro

```csharp
public MathVerticalAlignment BaseJustification { get; set; }
```

### Ejemplos

Ejemplo:

```csharp
[C#]
IMathArray mathArray = new MathArray(new MathematicalText("item1"));
mathArray.BaseJustification = MathVerticalAlignment.Top;
```

## IMathArray.RowSpacing property

Espaciado entre filas de un array. Se utiliza únicamente cuando RowSpacingRule está configurado en 3. Exactamente en qué caso la unidad de medida es puntos o Múltiple en cuyo caso la unidad de medida es medio-líneas. Predeterminado: 0

```csharp
public uint RowSpacing { get; set; }
```

### Examples

Ejemplo:

```csharp
[C#]
IMathArray mathArray = new MathArray(new MathematicalText("item1"));
mathArray.RowSpacingRule = MathRowSpacingRule.Exactly;
mathArray.RowSpacing = 10;
```

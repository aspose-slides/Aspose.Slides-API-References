## IMathBlockCollection.Remove method

Elimina la primera ocurrencia de un objeto específico de la colección/&gt;.

```csharp
public bool Remove(IMathBlock item)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| item | IMathBlock | El objeto a eliminar de la colección. |

### Valor de Retorno

true si *item* fue eliminado con éxito de la colección; de lo contrario, false. Este método también devuelve false si *item* no se encuentra en la colección original/&gt;.

### Ejemplos

Ejemplo:

```csharp
[C#]
IMathBlockCollection blockCollection = new MathParagraph();
IMathBlock block = new MathBlock(new MathematicalText("y"));
blockCollection.Add(block);
blockCollection.Remove(block);
```

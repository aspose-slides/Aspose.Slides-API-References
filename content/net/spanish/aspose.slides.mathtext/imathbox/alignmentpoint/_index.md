## IMathBox.AlignmentPoint propiedad

Cuando es verdadero, este emulador de operador sirve como un punto de alineación; es decir, los puntos de alineación designados en otras ecuaciones se pueden alinear con él. Valor predeterminado: falso

```csharp
public bool AlignmentPoint { get; set; }
```

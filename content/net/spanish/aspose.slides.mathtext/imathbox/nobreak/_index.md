## IMathBox.NoBreak property

Sin ruptura. Esta propiedad especifica la propiedad "no rompible" en el objeto box. Cuando es verdadero, no se pueden producir rupturas de línea dentro del box. Esto puede ser importante para emuladores de operadores que consisten en más de un operador binario. Cuando este elemento no se especifica, pueden ocurrir rupturas dentro del box. Predeterminado: verdadero

```csharp
public bool NoBreak { get; set; }
```

Ejemplo:

IMathBox box = new MathematicalText("**********").ToBox();
box.NoBreak = false;

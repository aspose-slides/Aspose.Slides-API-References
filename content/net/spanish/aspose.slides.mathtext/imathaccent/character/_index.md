## Propiedad IMathAccent.Character

Carácter de Acento El valor debe estar dentro del rango de (U+0300–U+036F) o (U+20D0–U+20EF) Valor por defecto: Acento circumflejo combinante (U+0302)

```csharp
public char Character { get; set; }
```

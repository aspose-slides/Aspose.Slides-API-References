## IMathElement.ToBox method

Coloca este elemento en una caja no visual (grupo lógico) que se utiliza para agrupar componentes de una ecuación u otra instancia de texto matemático. Un objeto en una caja puede (por ejemplo) servir como un emulador de operador con o sin un punto de alineación, servir como un punto de ruptura de línea, o agruparse de tal manera que no se permitan rupturas de línea dentro.

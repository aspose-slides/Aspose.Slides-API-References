## IMathFraction interface

Especifica el objeto de fracción, que consta de un numerador y un denominador separados por una barra de fracción. La barra de fracción puede ser horizontal o diagonal, según las propiedades de la fracción. El objeto de fracción también se usa para representar la función de pila, que coloca uno elemento encima de otro, sin barra de fracción.

public interface IMathFraction : IMathElement

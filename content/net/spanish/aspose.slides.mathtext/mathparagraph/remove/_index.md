## MathParagraph.Remove method

Elimina la primera ocurrencia de un objeto específico de la colección/&gt;.

```csharp
public bool Remove(IMathBlock mathBlock)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| mathBlock | IMathBlock | El objeto que se va a eliminar de la colección. |

### Valor de retorno

true si *mathBlock* fue eliminado exitosamente de la colección; de lo contrario, false. Este método también devuelve false si *mathBlock* no se encuentra en la colección original/&gt;.

### Ejemplos

Ejemplo:

```csharp
[C#]
IAutoShape shape = slide.Shapes.AddMathShape(x, y, width, height);
IMathParagraph mathParagraph = (shape.TextFrame.Paragraphs[0].Portions[0] as MathPortion).MathParagraph;
mathParagraph.Add(new MathBlock(new MathematicalText("x")));
IMathBlock block = new MathBlock(new MathematicalText("y"));
mathParagraph.Add(block);
mathParagraph.Remove(block);
```

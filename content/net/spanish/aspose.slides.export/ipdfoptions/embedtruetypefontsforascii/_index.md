## IPdfOptions.EmbedTrueTypeFontsForASCII propiedad

Verdadero para incrustar fuentes TrueType para caracteres ASCII 32-127. Las fuentes para los códigos de caracteres superiores a 127 siempre están incrustadas. Booleano de lectura/escritura.

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### Comentarios

El valor predeterminado es **true**.

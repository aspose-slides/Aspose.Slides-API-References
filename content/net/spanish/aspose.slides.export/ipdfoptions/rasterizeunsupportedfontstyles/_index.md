## Propiedad IPdfOptions.RasterizeUnsupportedFontStyles

Indica si el texto debe ser rasterizado como un bitmap y guardado en PDF cuando la fuente no soporta el estilo en negrita. Este enfoque puede mejorar la calidad del texto en el PDF resultante para ciertas fuentes. Booleano de lectura/escritura.

```csharp
public bool RasterizeUnsupportedFontStyles { get; set; }
```

### Observaciones

El valor predeterminado es **false**.

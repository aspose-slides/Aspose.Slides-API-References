## Propiedad IRenderingOptions.DisableFontLigatures

Obtiene o establece un valor que indica si el texto se representa sin usar ligaduras. Cuando se establece en `true`, las ligaduras se desactivarán en la salida renderizada. Por defecto, esta propiedad está establecida en `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Ejemplos

Ejemplo:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    RenderingOptions options = new RenderingOptions
    {
        DisableFontLigatures = true // Desactivar ligaduras en la representación del texto
    };
    
    Bitmap[] renderedSlides = pres.GetImage(options);
    for (var index = 0; index < renderedSlides.Length; index++)
    {
        var slideImage = renderedSlides[index];
        slideImage.Save($"slide-{index}.png");
    }
}
```

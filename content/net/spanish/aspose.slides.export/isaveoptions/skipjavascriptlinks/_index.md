## ISaveOptions.SkipJavaScriptLinks propiedad  
  
Especifica si se deben omitir hipervínculos con llamadas a JavaScript al guardar la presentación. Booleano de lectura/escritura. El valor predeterminado es **false**.  
  
```csharp  
public bool SkipJavaScriptLinks { get; set; }  
```  
  
### Comentarios  
  
Cuando esta propiedad se establece en **true**, se ignorarán los hipervínculos con llamadas a JavaScript al guardar.  
  
Cuando esta propiedad se establece en **false**, todos los hipervínculos se guardarán.  
  
### Ejemplos  
  
Ejemplo:  
  
```csharp  
[C#]  
using (Presentation pres = new Presentation("demo.pptx"))  
{  
    pres.Save("result_without_JavaScript_links.html", SaveFormat.Html, new HtmlOptions()  
    {  
        SkipJavaScriptLinks = true  
    });  
}  
```

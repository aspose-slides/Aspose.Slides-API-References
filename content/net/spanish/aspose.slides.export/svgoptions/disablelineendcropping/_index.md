## SVGOptions.DisableLineEndCropping property

SVG 1.1 carece de la capacidad de definir márgenes para marcadores. El motor de escritura SVG de Aspose.Slides tiene una solución para ese problema: recorta el final de la línea con una flecha, por lo que la línea no se superpone a los marcadores. Esta opción desactiva tal comportamiento. Lectura/escritura Boolean.

```csharp
public bool DisableLineEndCropping { get; set; }
```

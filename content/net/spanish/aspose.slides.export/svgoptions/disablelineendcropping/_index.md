## Propiedad SVGOptions.DisableLineEndCropping

SVG 1.1 carece de la capacidad de definir márgenes para los marcadores. El motor de escritura SVG de Aspose.Slides tiene una solución alternativa para ese problema: recorta el extremo de la línea con una flecha, de modo que la línea no se superponga a los marcadores. Esta opción desactiva dicho comportamiento. Booleano de lectura/escritura.

```csharp
public bool DisableLineEndCropping { get; set; }
```

## Propiedad SaveOptions.SkipJavaScriptLinks

Especifica si se deben omitir los hipervínculos con llamadas de JavaScript al guardar la presentación. Booleano de lectura/escritura. El valor predeterminado es **false**.

```csharp
public bool SkipJavaScriptLinks { get; set; }
```

### Observaciones

Cuando esta propiedad se establece en **true**, los hipervínculos con llamadas de JavaScript se ignorarán al guardar.

Cuando esta propiedad se establece en **false**, se guardarán todos los hipervínculos.

### Ejemplos

Ejemplo:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("result_without_JavaScript_links.html", SaveFormat.Html, new HtmlOptions()
    {
        SkipJavaScriptLinks = true
    });
}
```

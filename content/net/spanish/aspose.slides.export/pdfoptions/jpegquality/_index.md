## PdfOptions.JpegQuality property

Devuelve o establece un valor que determina la calidad de las imágenes JPEG dentro del documento PDF. Lectura/escritura Byte.

```csharp
public byte JpegQuality { get; set; }
```

### Remarks

Affecta solo cuando un documento contiene imágenes JPEG.

Utiliza esta propiedad para obtener o establecer la calidad de las imágenes dentro de un documento al guardar en formato PDF. El valor puede variar de 0 a 100, donde 0 significa la peor calidad pero la máxima compresión y 100 significa la mejor calidad pero la mínima compresión.

El valor predeterminado es **100**.

## PdfOptions.RasterizeUnsupportedFontStyles property

Indica si el texto debe ser rasterizado como un mapa de bits y guardado en PDF cuando la fuente no admite el estilo en negrita. Este enfoque puede mejorar la calidad del texto en el PDF resultante para ciertas fuentes. Booleano de lectura/escritura.

```csharp
public bool RasterizeUnsupportedFontStyles { get; set; }
```

### Remarks

El valor predeterminado es **false**.

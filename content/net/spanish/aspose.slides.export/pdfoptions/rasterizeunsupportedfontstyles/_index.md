## Propiedad PdfOptions.RasterizeUnsupportedFontStyles

Indica si el texto debe rasterizarse como un bitmap y guardarse en PDF cuando la fuente no admite el estilo negrita. Este enfoque puede mejorar la calidad del texto en el PDF resultante para ciertas fuentes. Booleano de lectura/escritura.

```csharp
public bool RasterizeUnsupportedFontStyles { get; set; }
```

### Observaciones

El valor predeterminado es **false**.

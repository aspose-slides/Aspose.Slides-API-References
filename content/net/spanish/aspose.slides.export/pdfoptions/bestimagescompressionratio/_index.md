## PdfOptions.BestImagesCompressionRatio property

Indica si se debe seleccionar automáticamente la compresión más efectiva (en lugar de la estándar) para cada imagen. Si se establece en Boolean.true, para cada imagen en la presentación se elegirá el algoritmo de compresión más apropiado, lo que dará como resultado un tamaño menor del documento PDF resultante. La selección de la mejor relación de compresión de imágenes es costosa en términos computacionales y requiere una cantidad adicional de RAM, y esta opción es Boolean.false de forma predeterminada.

```csharp
public bool BestImagesCompressionRatio { get; set; }
```

### Remarks

El valor predeterminado es Boolean.false.

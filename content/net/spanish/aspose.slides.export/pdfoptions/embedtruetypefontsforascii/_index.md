## PdfOptions.EmbedTrueTypeFontsForASCII property

Determina si Aspose.Slides incrustará fuentes comunes para texto ASCII (rango de códigos 33..127). Las fuentes para los códigos de caracteres mayores que 127 siempre se incrustan. La lista de fuentes comunes incluye las 14 fuentes base de PDF y fuentes adicionales especificadas por el usuario. Booleano de lectura/escritura.

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### Remarks

El valor predeterminado es **true**.

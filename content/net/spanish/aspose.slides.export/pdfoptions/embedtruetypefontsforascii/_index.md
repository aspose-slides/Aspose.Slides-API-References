## PdfOptions.EmbedTrueTypeFontsForASCII property

Determina si Aspose.Slides incrustará fuentes comunes para texto ASCII (rango de código 33..127). Las fuentes para códigos de caracteres superiores a 127 siempre están incrustadas. La lista de fuentes comunes incluye fuentes base 14 de PDF y fuentes adicionales especificadas por el usuario. Leer escribirBoolean .

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### Observaciones

El valor predeterminado es **verdadero** .

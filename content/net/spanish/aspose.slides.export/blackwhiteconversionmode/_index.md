## BlackWhiteConversionMode enumeration

Proporciona opciones que controlan cómo se convertirán las imágenes de las diapositivas a imágenes bitonales.

```csharp
public enum BlackWhiteConversionMode
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Default | `0` | Especifica que no hay algoritmo de conversión. Se utilizará el algoritmo implementado en el códec TIFF. (Predeterminado) |

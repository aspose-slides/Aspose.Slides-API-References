## Enumeración SvgEvent

Representa las opciones para la forma SVG.

```csharp
public enum SvgEvent
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| OnFocusIn | `0` | Ocurre cuando un elemento recibe enfoque, como cuando un texto se selecciona. |
| OnFocusOut | `1` | Ocurre cuando un elemento pierde el enfoque, como cuando un texto se deselecciona. |
| OnActivate | `2` | Ocurre cuando un elemento es activado, por ejemplo, a través de un clic del ratón o una pulsación de tecla. |
| OnClick | `3` | Ocurre cuando el botón del dispositivo indicador es clicado sobre un elemento. |
| OnMouseDown | `4` | Ocurre cuando el botón del dispositivo indicador es presionado sobre un elemento. |
| OnMouseUp | `5` | Ocurre cuando el botón del dispositivo indicador es liberado sobre un elemento. |
| OnMouseOver | `6` | Ocurre cuando el dispositivo indicador se mueve sobre un elemento. |
| OnMouseMove | `7` | Ocurre cuando el dispositivo indicador se mueve mientras está sobre un elemento. |
| OnMouseOut | `8` | Ocurre cuando el dispositivo indicador se mueve lejos de un elemento. |
| OnLoad | `9` | Ocurre cuando el agente de usuario ha analizado completamente el elemento y sus descendientes y todos los recursos referenciados, necesarios para renderizarlo. |
| OnUnload | `10` | Ocurre cuando la implementación del DOM elimina un documento de una ventana o marco. Solo aplicable a los elementos svg más externos. |
| OnAbort | `11` | Ocurre cuando la carga de la página se detiene antes de que se permita que un elemento se cargue completamente. |
| OnError | `12` | Ocurre cuando un elemento no se carga correctamente o cuando ocurre un error durante la ejecución del script. |
| OnResize | `13` | Ocurre cuando una vista de documento está siendo redimensionada. Solo aplicable a los elementos svg más externos. |
| OnScroll | `14` | Ocurre cuando una vista de documento se desplaza a lo largo del eje X o Y o ambos. Solo aplicable a los elementos svg más externos. |
| OnZoom | `15` | Ocurre cuando el nivel de zoom de una vista de documento está siendo cambiado. Solo aplicable a los elementos svg más externos. |
| OnBegin | `16` | Ocurre cuando comienza un elemento de animación. |
| OnEnd | `17` | Ocurre cuando termina un elemento de animación. |
| OnRepeat | `18` | Ocurre cuando un elemento de animación se repite. |

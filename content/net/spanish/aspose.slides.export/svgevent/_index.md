## Enumeración SvgEvent

Representa opciones para la forma SVG.

```csharp
public enum SvgEvent
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| OnFocusIn | `0` | Ocurre cuando un elemento recibe el enfoque, como cuando un texto es seleccionado. |
| OnFocusOut | `1` | Ocurre cuando un elemento pierde el enfoque, como cuando un texto deja de estar seleccionado. |
| OnActivate | `2` | Ocurre cuando un elemento es activado, por ejemplo, a través de un clic del ratón o una pulsación de tecla. |
| OnClick | `3` | Ocurre cuando se hace clic en el botón del dispositivo de apontar sobre un elemento. |
| OnMouseDown | `4` | Ocurre cuando se presiona el botón del dispositivo de apuntar sobre un elemento. |
| OnMouseUp | `5` | Ocurre cuando se suelta el botón del dispositivo de apuntar sobre un elemento. |
| OnMouseOver | `6` | Ocurre cuando el dispositivo de apuntar se mueve sobre un elemento. |
| OnMouseMove | `7` | Ocurre cuando el dispositivo de apuntar se mueve mientras está sobre un elemento. |
| OnMouseOut | `8` | Ocurre cuando el dispositivo de apuntar se aleja de un elemento. |
| OnLoad | `9` | Ocurre cuando el agente de usuario ha analizado completamente el elemento y sus descendientes y todos los recursos referenciados, necesarios para renderizarlo. |
| OnUnload | `10` | Ocurre cuando la implementación del DOM retira un documento de una ventana o marco. Solo aplicable a los elementos SVG más externos. |
| OnAbort | `11` | Ocurre cuando la carga de la página se detiene antes de que se permita que un elemento se cargue completamente. |
| OnError | `12` | Ocurre cuando un elemento no se carga correctamente o cuando ocurre un error durante la ejecución del script. |
| OnResize | `13` | Ocurre cuando se está redimensionando una vista de documento. Solo aplicable a los elementos SVG más externos. |
| OnScroll | `14` | Ocurre cuando se desplaza una vista de documento a lo largo del eje X, Y o ambos. Solo aplicable a los elementos SVG más externos. |
| OnZoom | `15` | Ocurre cuando se está cambiando el nivel de zoom de una vista de documento. Solo aplicable a los elementos SVG más externos. |
| OnBegin | `16` | Ocurre cuando un elemento de animación comienza. |
| OnEnd | `17` | Ocurre cuando un elemento de animación finaliza. |
| OnRepeat | `18` | Ocurre cuando un elemento de animación se repite. |

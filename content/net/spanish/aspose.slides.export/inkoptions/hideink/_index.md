## Propiedad InkOptions.HideInk

Muestra u oculta elementos de tinta en el documento exportado.

```csharp
public bool HideInk { get; set; }
```

### Observaciones

El valor predeterminado es falso.

### Ejemplos

El siguiente ejemplo demuestra cómo ocultar elementos de tinta en un documento PDF exportado:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.HideInk = true;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

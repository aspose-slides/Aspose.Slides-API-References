## IGifOptions.DefaultDelay property

Obtiene o establece el tiempo de retardo predeterminado [ms]. Este valor se utilizará si[`AdvanceAfterTime`](../../../aspose.slides/islideshowtransition/advanceaftertime) no está configurado. El valor predeterminado es 1000.

```csharp
public int DefaultDelay { get; set; }
```

### Ejemplos

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
	pres.Save("pres.gif", SaveFormat.Gif, new GifOptions { DefaultDelay = 2000 });
}
```

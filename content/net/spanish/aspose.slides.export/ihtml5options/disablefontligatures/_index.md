## IHtml5Options.DisableFontLigatures property

Obtiene o establece un valor que indica si el texto se representa sin utilizar ligaduras. Cuando se establece en `true`, las ligaduras se desactivarán en la salida renderizada. Por defecto, esta propiedad se establece en `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Ejemplos

Ejemplo:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    Html5Options options = new Html5Options
    {
        DisableFontLigatures = true // Deshabilitar ligaduras en la representación de texto
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html5, options);
}
```

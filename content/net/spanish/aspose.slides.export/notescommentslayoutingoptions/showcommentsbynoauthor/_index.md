## NotesCommentsLayoutingOptions.ShowCommentsByNoAuthor property

Obtiene o establece la visibilidad de los comentarios que no tienen un autor. Si es verdadero, los comentarios se mostrarán. (Aplica solo si los comentarios se muestran).

```csharp
public bool ShowCommentsByNoAuthor { get; set; }
```

### Remarks

El valor por defecto es **false**.

## HtmlOptions.DisableFontLigatures property

Obtiene o establece un valor que indica si el texto se renderiza sin utilizar ligaduras. Cuando se establece en `true`, las ligaduras se desactivarán en la salida renderizada. Por defecto, esta propiedad se establece en `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Ejemplos

Ejemplo:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    HtmlOptions options = new HtmlOptions
    {
        DisableFontLigatures = true // Desactivar ligaduras en la representación del texto
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html, options);
}
```

## Propiedad HtmlOptions.JpegQuality

Devuelve o establece un valor que determina la calidad de las imágenes JPEG dentro del documento PDF. Lectura/escritura Byte.

```csharp
public byte JpegQuality { get; set; }
```

### Comentarios

Tiene efecto solo cuando un documento contiene imágenes JPEG.

Utilice esta propiedad para obtener o establecer la calidad de las imágenes dentro de un documento al guardarlo en formato PDF. El valor puede variar de 0 a 100, donde 0 significa peor calidad pero máxima compresión y 100 significa mejor calidad pero mínima compresión.

El valor predeterminado es **95**.

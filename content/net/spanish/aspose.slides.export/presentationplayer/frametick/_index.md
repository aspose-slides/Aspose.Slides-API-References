## Evento PresentationPlayer.FrameTick

Evento de tick de cuadro. Ocurre cuando se genera cada cuadro de la animación creada por [`PresentationAnimationsGenerator`](../../presentationanimationsgenerator) por el reproductor.

```csharp
public event FrameTickHandler FrameTick;
```

### Ejemplos

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (var animationsGenerator = new PresentationAnimationsGenerator(pres))
    using (var player = new PresentationPlayer(animationsGenerator, 33))
    {
        player.FrameTick += (sender, args) =>
        {
            args.GetFrame().Save($"frame_{sender.FrameIndex}.png");
        };
        
        animationsGenerator.Run(pres.Slides);
    }
}
```

## Propiedad TiffOptions.ImageSize

Especifica el tamaño de una imagen TIFF generada. El valor predeterminado es 0x0, lo que significa que los tamaños de imagen generados se calcularán según el valor del tamaño de la diapositiva de presentación. Propiedad de lectura/escritura Size.

```csharp
public Size ImageSize { get; set; }
```

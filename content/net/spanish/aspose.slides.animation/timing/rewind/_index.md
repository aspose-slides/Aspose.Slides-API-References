## Propiedad Timing.Rewind

Este atributo especifica si el efecto se rebobinará cuando termine de reproducirse. Booleano de lectura/escritura.

```csharp
public bool Rewind { get; set; }
```

### Ejemplos

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Obtener la secuencia de efectos para la primera diapositiva
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Obtener el primer efecto de la secuencia principal.
    IEffect effect = effectsSequence[0];

    // Activar el rebobinado del efecto Timing/Rewind.
    effect.Timing.Rewind = true;
}
```

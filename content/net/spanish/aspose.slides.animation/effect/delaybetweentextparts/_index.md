## Effect.DelayBetweenTextParts property

Define un retraso entre las partes de texto animadas (palabras o letras). Un valor positivo especifica el porcentaje de la duración del efecto. Un valor negativo especifica el retraso en segundos. Lectura/escritura Float.

```csharp
public float DelayBetweenTextParts { get; set; }
```

### Examples

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Obtener el primer efecto de la primera diapositiva.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Cambiar el tipo de texto animado del efecto a "Por palabra"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByWord;
    
    // Establecer el retraso entre las partes de texto animadas al 20% de la duración del efecto.
    firstSlideEffect.DelayBetweenTextParts = 20f;
}
```

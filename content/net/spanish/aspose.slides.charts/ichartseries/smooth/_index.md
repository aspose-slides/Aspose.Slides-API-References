## Propiedad IChartSeries.Smooth

Representa el suavizado de curvas. Verdadero si el suavizado de curvas está activado para el gráfico de líneas o gráfico de dispersión. Se aplica solo a gráficos de líneas y dispersión conectados por líneas. Booleano de lectura/escritura.

public bool Smooth { get; set; }

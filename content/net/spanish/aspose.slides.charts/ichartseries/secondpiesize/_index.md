## Propiedad IChartSeries.SecondPieSize

Especifica el tamaño del segundo gráfico circular o barra de un gráfico de gráfico circular o un gráfico de barra circular, como un porcentaje del tamaño del primer gráfico (puede estar entre el 5 y el 200 por ciento). Esta es la propiedad no solo de esta serie, sino de todas las series del grupo de series principal; esta es una proyección de la propiedad del grupo correspondiente. Y así, esta propiedad es de solo lectura. Usa la propiedad ParentSeriesGroup para acceder al grupo de series principal. Usa la propiedad de lectura/escritura ParentSeriesGroup.SecondPieSize para cambiar el valor. UInt16 de solo lectura.

```csharp
public ushort SecondPieSize { get; }
```

### Observaciones

Esta es la proyección de la propiedad ParentSeriesGroup.SecondPieSize.

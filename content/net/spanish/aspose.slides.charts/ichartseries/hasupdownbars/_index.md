## Propiedad IChartSeries.HasUpDownBars

Determina si el gráfico de Línea o el gráfico de Valores tiene barras hacia arriba/abajo. Esta es la propiedad no solo de esta serie, sino de todas las series del grupo de series padre; esta es la proyección de la propiedad del grupo correspondiente. Por lo tanto, esta propiedad es de solo lectura. Utilice la propiedad ParentSeriesGroup para acceder al grupo de series padre. Utilice la propiedad de lectura/escritura ParentSeriesGroup.UpDownBars.HasUpDownBars para cambiar el valor. Utilice la propiedad ParentSeriesGroup.UpDownBars para dar formato a las barras hacia arriba/abajo. Booleano de solo lectura.

```csharp
public bool HasUpDownBars { get; }
```

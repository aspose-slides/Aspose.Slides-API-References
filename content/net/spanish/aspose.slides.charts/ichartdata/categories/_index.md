## IChartData.Categories property

Obtiene las categorías primarias (o las categorías primaria y secundaria si[`UseSecondaryCategories`](../usesecondarycategories) propiedad es falsa). Solo lectura[`IChartCategoryCollection`](../../ichartcategorycollection) .

```csharp
public IChartCategoryCollection Categories { get; }
```

### Observaciones

Si[`UseSecondaryCategories`](../usesecondarycategories) entonces la propiedad es falsa[`SecondaryCategories`](../secondarycategories) propiedad devuelve nulo y datos en este`Categories` propiedad se usa tanto para la serie primaria como para la secundaria. Si[`UseSecondaryCategories`](../usesecondarycategories) la propiedad es verdadera, entonces los datos en[`SecondaryCategories`](../secondarycategories) La propiedad se usa para series secundarias y datos en este`Categories` la propiedad se usa para la serie primaria.

### Ejemplos

Ejemplo. ¿Qué categorías están relacionadas con la serie - ChartData.Categories o ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // las categorías relacionadas son series.Chart.ChartData.SecondaryCategories
}
else
{
    // las categorías relacionadas son series.Chart.ChartData.Categories
}
```

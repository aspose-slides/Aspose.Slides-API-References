## IChartData.GetRange method

Obtiene el rango de datos del gráfico.

```csharp
public string GetRange()
```

### Valor_devuelto

Fórmula de rango de datos de celdas. Por ejemplo: "Hoja1!$A$1:$C$4"

### Excepciones

| excepción | condición |
| --- | --- |
| InvalidOperationException | El gráfico no usa el libro de trabajo como fuente de datos |

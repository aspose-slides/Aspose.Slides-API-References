## Propiedad IDataLabelFormat.IsNumberFormatLinkedToSource

Leer/escribir Booleano.

```csharp
public bool IsNumberFormatLinkedToSource { get; set; }
```

### Comentarios

Si el padre de este objeto DataLabelFormat es una colección DataLabelCollection de etiquetas de datos, entonces esta propiedad obtiene o establece el valor predeterminado de la propiedad IsNumberFormatLinkedToSource para las nuevas etiquetas de datos en la colección DataLabelCollection. Establecer esta propiedad con un valor también establece este valor en la propiedad IsNumberFormatLinkedToSource para todas las etiquetas de datos en la colección DataLabelCollection (es decir, "DataLabels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = val;" causa que todas las DataLabels[i].IsNumberFormatLinkedToSource sean iguales a val).

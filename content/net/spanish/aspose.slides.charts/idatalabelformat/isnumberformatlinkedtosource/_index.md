## IDataLabelFormat.IsNumberFormatLinkedToSource property

Lectura/escrituraBoolean .

```csharp
public bool IsNumberFormatLinkedToSource { get; set; }
```

### Observaciones

Si el padre de este objeto DataLabelFormat es una colección de etiquetas de datos DataLabelCollection, esta propiedad obtiene o establece el valor predeterminado de la propiedad IsNumberFormatLinkedToSource para las nuevas etiquetas de datos en la colección DataLabelCollection. Establecer esta propiedad con valor también establece este valor en el IsNumberFormatLinkedToSource propiedad para todas las etiquetas de datos en la colección DataLabelCollection (es decir, "DataLabels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = val;" causa que todas las DataLabels[i].IsNumberFormatLinkedToSource sea igual a val).

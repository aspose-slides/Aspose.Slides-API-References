## Propiedad IDataLabelFormat.Separator

Establece o devuelve un Variant que representa el separador utilizado para las etiquetas de datos en un gráfico. Lectura/escritura String.

```csharp
public string Separator { get; set; }
```

### Observaciones

Si el padre de este objeto DataLabelFormat es una colección DataLabelCollection de etiquetas de datos, entonces esta propiedad obtiene o establece el valor predeterminado de la propiedad Separator para las nuevas etiquetas de datos en la colección DataLabelCollection. Establecer esta propiedad con un valor también establece este valor en la propiedad Separator para todas las etiquetas de datos en la colección DataLabelCollection (es decir, "DataLabels.DefaultDataLabelFormat.Separator = val;" hace que todos los DataLabels[i].Separator sean iguales a val).

## IDataLabelFormat.Separator property

Establece o devuelve una variante que representa el separador utilizado para las etiquetas de datos en un gráfico. Lectura/escrituraString .

```csharp
public string Separator { get; set; }
```

### Observaciones

Si el padre de este objeto DataLabelFormat es una colección de etiquetas de datos DataLabelCollection, esta propiedad obtiene o establece el valor predeterminado de la propiedad Separator para las nuevas etiquetas de datos en la colección DataLabelCollection. Establecer esta propiedad con valor también establece este valor en el Propiedad de separador para todas las etiquetas de datos en la colección DataLabelCollection (es decir, "DataLabels.DefaultDataLabelFormat.Separator = val;" causa que todas las DataLabels[i].Separator es igual a val).

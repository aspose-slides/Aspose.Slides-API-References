## Propiedad IDataLabelFormat.ShowValue

Representa un comportamiento de visualización del valor porcentual de la etiqueta de datos de un gráfico especificado. Verdadero muestra el valor porcentual. Falso para ocultar. Booleano de lectura/escritura.

```csharp
public bool ShowValue { get; set; }
```

### Comentarios

Si el padre de este objeto DataLabelFormat es una colección DataLabelCollection de etiquetas de datos, entonces esta propiedad obtiene o establece el valor predeterminado de la propiedad ShowValue para las nuevas etiquetas de datos en la colección DataLabelCollection. Establecer esta propiedad con un valor también establece este valor a la propiedad ShowValue para todas las etiquetas de datos en la colección DataLabelCollection (es decir, "DataLabels.DefaultDataLabelFormat.ShowValue = val;" hace que todas las DataLabels[i].ShowValue sean iguales a val).

## Propiedad IDataLabelCollection.IsVisible

Falso significa que la etiqueta de datos no es visible por defecto (y por lo tanto todas las banderas Show*- (ShowValue, ...) de la propiedad DefaultDataLabelFormat son falsas). Solo lectura Boolean.

```csharp
public bool IsVisible { get; }
```

### Observaciones

Si la etiqueta de datos es visible por defecto, puedes hacerla oculta por defecto con el método Hide(). Pero si la etiqueta de datos no es visible por defecto (IsVisible es falso), puedes hacer que la etiqueta de datos sea "visible por defecto" configurando las banderas Show*- (ShowValue, ...) de la propiedad DefaultDataLabelFormat en estado verdadero.

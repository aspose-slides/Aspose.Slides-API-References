## IDataLabelCollection.IsVisible property

Falso significa que la etiqueta de datos no está visible de forma predeterminada (por lo que todos los indicadores Show* (ShowValue, ...) de la propiedad DefaultDataLabelFormat son falsos). Solo lecturaBoolean .

```csharp
public bool IsVisible { get; }
```

### Observaciones

Si la etiqueta de datos está visible de forma predeterminada, puede ocultarla de forma predeterminada con el método Hide(). Pero si la etiqueta de datos no está visible de forma predeterminada (IsVisible es falso), puede hacer que la etiqueta de datos sea "visible de forma predeterminada" con la configuración Mostrar *-flags (ShowValue, ...) de la propiedad DefaultDataLabelFormat a estado verdadero.

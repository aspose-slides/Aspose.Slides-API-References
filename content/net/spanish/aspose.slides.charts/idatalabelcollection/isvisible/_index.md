## Propiedad IDataLabelCollection.IsVisible

Falso significa que la etiqueta de datos no es visible por defecto (y por lo tanto todos los indicadores Show*-flags (ShowValue, ...) de la propiedad DefaultDataLabelFormat son falsos). Solo lectura Booleano.

```csharp
public bool IsVisible { get; }
```

### Notas

Si la etiqueta de datos es visible por defecto, puedes hacer que esté oculta por defecto con el método Hide(). Pero si la etiqueta de datos no es visible por defecto (IsVisible es falso), puedes hacer que la etiqueta de datos sea "visible por defecto" configurando los indicadores Show*-flags (ShowValue, ...) de la propiedad DefaultDataLabelFormat a estado verdadero.

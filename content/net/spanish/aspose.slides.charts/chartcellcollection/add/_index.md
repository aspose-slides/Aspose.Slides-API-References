### Observaciones

Este método agrega una hoja de cálculo con el nombre AUTO_DATA y agrega todos los valores allí. Si utiliza [`ChartDataWorkbook`](../../chartdataworkbook) para agregar o editar los valores de las celdas, asegúrese de no usar esta hoja de cálculo. El número máximo de valores agregados utilizando este método no debe exceder 16711680.

## Enumeración ParentLabelLayoutType

Representa el diseño de etiquetas de datos por categoría. Se utiliza únicamente con series de Treemap.

```csharp
public enum ParentLabelLayoutType
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |

| None | `1` | Especifica que las etiquetas de las categorías padres no se dibujan |

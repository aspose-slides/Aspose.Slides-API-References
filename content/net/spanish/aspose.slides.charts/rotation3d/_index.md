```csharp
public class Rotation3D : IRotation3D
```

## Propiedades

| Nombre | Descripción |
| --- | --- |
| [DepthPercents](../../aspose.slides.charts/rotation3d/depthpercents) { get; set; } | Devuelve o establece la profundidad de un gráfico 3D como un porcentaje del ancho del gráfico (entre 20 y 2000 por ciento). Lectura/escritura UInt16. |
| [HeightPercents](../../aspose.slides.charts/rotation3d/heightpercents) { get; set; } | Especifica la altura de un gráfico 3D como un porcentaje del ancho del gráfico (entre 5 y 500 por ciento). Lectura/escritura UInt16. |
| [Perspective](../../aspose.slides.charts/rotation3d/perspective) { get; set; } | Devuelve o establece el valor de perspectiva (ángulo de campo de visión) para gráficos 3D (entre 0 y 240). Ignorado si el valor de la propiedad RightAngleAxes es verdadero. Lectura/escritura Byte. |
| [RightAngleAxes](../../aspose.slides.charts/rotation3d/rightangleaxes) { get; set; } | Determina si los ejes del gráfico están en ángulos rectos, en lugar de dibujados en perspectiva. En otras palabras, determina si los ángulos de los ejes del gráfico son independientes de la rotación o elevación del gráfico. Lectura/escritura Boolean. |
| [RotationX](../../aspose.slides.charts/rotation3d/rotationx) { get; set; } | Devuelve o establece el grado de rotación alrededor del eje X, es decir, en la dirección Y para gráficos 3D (entre -90 y 90 grados). La propiedad coincide con el elemento 21.2.2.157 rotX (Rotación X) en ECMA-376 y con la opción "Rotación Y" en PowerPoint 2007+. Lectura/escritura SByte. |
| [RotationY](../../aspose.slides.charts/rotation3d/rotationy) { get; set; } | Devuelve o establece el grado de rotación alrededor del eje Y, es decir, en la dirección X para gráficos 3D (entre 0 y 360 grados). La propiedad coincide con el elemento 21.2.2.158 rotY (Rotación Y) en ECMA-376 y con la opción "Rotación X" en PowerPoint 2007+. Lectura/escritura UInt16. |

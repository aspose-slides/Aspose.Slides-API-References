## Enumeración ErrorBarValueType

Representa el tipo de valor de barra de error

```csharp
public enum ErrorBarValueType
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Custom | `0` | Especifica que la longitud de las barras de error será determinada por los elementos Plus y Minus. |
| Fixed | `1` | Especifica que la longitud de las barras de error será el valor fijo determinado por el Valor de Barra de Error. |
| Percentage | `2` | Especifica que la longitud de las barras de error será un porcentaje del Valor de Barra de Error respecto a los datos. |
| StandardDeviation | `3` | Especifica que la longitud de las barras de error será las desviaciones estándar del Valor de Barra de Error con respecto a los datos. |
| StandardError | `4` | Especifica que la longitud de las barras de error será los errores estándar del Valor de Barra de Error con respecto a los datos. |

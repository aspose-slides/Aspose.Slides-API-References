## ChartDataPointCollection.GetOrCreateDataPointByIdx method

Si la colección ya contiene un punto de datos con índice*index* luego devuelve este punto de datos. Si la colección no contiene un punto de datos con índice*index* ==N (cuando el número de puntos de datos en esta colección es menor o igual que N) luego agrega puntos de datos deficientes y devuelve el último (que tiene el índice solicitado). Por ejemplo, los índices de colección son {0, 1, 2} y el índice solicitado es 5. Luego, el método agrega puntos de datos deficientes: {0, 1, 2, 3, 4, 5}. Y devuelve el punto de datos con el índice 5.

```csharp
public IChartDataPoint GetOrCreateDataPointByIdx(uint index)
```

| --- | --- | --- |
| index | UInt32 | Índice. |

### Valor_devuelto

Devuelve el punto de datos con el índice solicitado.

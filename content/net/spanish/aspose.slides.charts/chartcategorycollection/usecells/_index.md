## Propiedad ChartCategoryCollection.UseCells

Si es verdadero, entonces la hoja de trabajo se utiliza para almacenar categorías (este caso admite categorías de múltiples niveles). Si es falso, entonces la hoja de trabajo NO se utiliza para almacenar valores (y este caso no admite categorías de múltiples niveles). Booleano de lectura/escritura.

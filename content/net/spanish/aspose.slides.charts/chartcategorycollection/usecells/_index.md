## Propiedad ChartCategoryCollection.UseCells

Si es verdadero, entonces la hoja de cálculo se utiliza para almacenar categorías (este caso admite categorías multinivel). Si es falso, entonces la hoja de cálculo NO se utiliza para almacenar valores (y este caso no admite categorías multinivel). Booleano de lectura/escritura.

public bool UseCells { get; set; }

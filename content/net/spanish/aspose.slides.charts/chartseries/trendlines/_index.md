## Propiedad ChartSeries.TrendLines

Colección de líneas de tendencia de series. [`ITrendlineCollection`](../../itrendlinecollection) de solo lectura.

```csharp
public ITrendlineCollection TrendLines { get; }
```

### Comentarios

TrendLines están disponibles (no nulas) para series de datos en gráficos de área 2-D no apilados, barras, columnas, líneas, acciones, xy (dispersión) y burbujas. Las líneas de tendencia no están disponibles para series de datos en ningún tipo de gráfico que esté apilado o en 3-D. Las líneas de tendencia tampoco están disponibles para gráficos de radar, de pastel, de superficie o de dona.

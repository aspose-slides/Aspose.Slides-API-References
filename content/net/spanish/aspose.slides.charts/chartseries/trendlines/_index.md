## Propiedad ChartSeries.TrendLines

Colección de líneas de tendencia de serie. Solo lectura [`ITrendlineCollection`](../../itrendlinecollection).

```csharp
public ITrendlineCollection TrendLines { get; }
```

### Comentarios

Las TrendLines están disponibles (no nulas) para series de datos en gráficos de área 2-D sin apilar, barra, columna, línea, acciones, xy (dispersión) y burbuja. No hay líneas de tendencia disponibles para series de datos en ningún tipo de gráfico que esté apilado o sea 3-D. Las líneas de tendencia tampoco están disponibles para gráficos de radar, pastel, superficie o dona.

## Propiedad ChartSeries.GapWidth

Especifica el espacio entre grupos de barras o columnas, como un porcentaje del ancho de la barra o columna. Esta es la propiedad no solo de esta serie, sino de todas las series del grupo de series principal: esta es una proyección de la propiedad del grupo apropiado. Por lo tanto, esta propiedad es de solo lectura. Utilice la propiedad ParentSeriesGroup para acceder al grupo de series principal. Utilice la propiedad de lectura/escritura ParentSeriesGroup.GapWidth para cambiar el valor. Solo de lectura Int32.

```csharp
public int GapWidth { get; }
```

### Observaciones

Esta es la proyección de la propiedad ParentSeriesGroup.GapWidth.

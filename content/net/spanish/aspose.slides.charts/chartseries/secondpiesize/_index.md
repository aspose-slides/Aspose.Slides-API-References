## Propiedad ChartSeries.SecondPieSize

Especifica el tamaño del segundo pastel o barra de un gráfico de pastel de pastel o un gráfico de barra de pastel, como un porcentaje del tamaño del primer pastel (que puede estar entre 5 y 200 por ciento). Esta es la propiedad no solo de esta serie, sino de todas las series del grupo de series padre; esta es una proyección de la propiedad del grupo apropiado. Por lo tanto, esta propiedad es de solo lectura. Usa la propiedad ParentSeriesGroup para acceder al grupo de series padre. Utiliza la propiedad de lectura/escritura ParentSeriesGroup.SecondPieSize para cambiar el valor. Solo lectura UInt16.

```csharp
public ushort SecondPieSize { get; }
```

### Observaciones

Esta es la proyección de la propiedad ParentSeriesGroup.SecondPieSize.

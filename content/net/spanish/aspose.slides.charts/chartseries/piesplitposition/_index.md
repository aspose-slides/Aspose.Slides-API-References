## ChartSeries.PieSplitPosition property

Especifica un valor que se utilizará para determinar qué puntos de datos están en el segundo pastel o barra en un gráfico de pastel-de-pastel o barra-de-pastel. Se utiliza junto con la propiedad PieSplitBy. Esta es la propiedad no solo de esta serie sino de todas las series del grupo de series padre - esta es la proyección de la propiedad del grupo apropiado. Por lo tanto, esta propiedad es de solo lectura. Usa la propiedad ParentSeriesGroup para acceder al grupo de series padre. Usa la propiedad de lectura/escritura ParentSeriesGroup.PieSplitPosition para cambiar el valor. Solo lectura Double.

public double PieSplitPosition { get; }

### Comentarios

Este método agrega la hoja de cálculo con el nombre AUTO_DATA y agrega todos los valores allí. Si usas [`IChartDataWorkbook`](../../ichartdataworkbook) para agregar o editar valores de Celda, asegúrate de no usar esta hoja de cálculo. El número máximo de valores agregados usando este método no debe exceder 16711680.

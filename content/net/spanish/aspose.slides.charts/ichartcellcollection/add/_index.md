### Comentarios

Este método agrega la hoja de trabajo con el nombre AUTO_DATA y agrega todos los valores allí. Si utilizas [`IChartDataWorkbook`](../../ichartdataworkbook) para agregar o editar valores de celda, asegúrate de no utilizar esta hoja de cálculo. El número máximo de valores agregados utilizando este método no debe exceder 16711680.

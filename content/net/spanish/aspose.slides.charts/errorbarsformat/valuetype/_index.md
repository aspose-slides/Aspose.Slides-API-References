## Propiedad ErrorBarsFormat.ValueType

Representa las posibles maneras de determinar la longitud de las barras de error. En caso de un tipo de valor personalizado para especificar el valor, use la propiedad [`ErrorBarsCustomValues`](../../ichartdatapoint/errorbarscustomvalues) de un punto de datos específico en la colección DataPoints de la serie. En caso de usar el tipo de valor Fixed, Percentage o StandardDeviation, use la propiedad Value para especificar el valor. Lectura/escritura [`ErrorBarValueType`](../../errorbarvaluetype).

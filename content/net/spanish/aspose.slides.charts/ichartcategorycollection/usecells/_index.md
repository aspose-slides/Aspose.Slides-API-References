## IChartCategoryCollection.UseCells propiedad

Si es verdadero, entonces se utiliza la hoja de cálculo para almacenar categorías (en este caso soporta categorías de múltiples niveles). Si es falso, entonces la hoja de cálculo NO se utiliza para almacenar valores (y este caso no soporta categorías de múltiples niveles). Booleano de lectura/escritura.

public bool UseCells { get; set; }

## IRotation3D.RotationY property

Devuelve o establece el grado de rotación alrededor del eje Y, es decir, en la dirección X para gráficos 3D (entre 0 y 360 grados). La propiedad coincide con el elemento 21.2.2.158 rotY (Rotación Y) en ECMA-376 y con la opción "Rotación X" en PowerPoint 2007+. Lectura/escritura UInt16.

```csharp
public ushort RotationY { get; set; }
```

## IRotation3D.RotationX property

Devuelve o establece el grado de rotación alrededor del eje X, es decir, en la dirección Y para gráficos 3D (entre -90 y 90 grados). La propiedad coincide con el elemento 21.2.2.157 rotX (Rotación X) en ECMA-376 y con la opción "Rotación Y" en PowerPoint 2007+. Lectura/escritura SByte.

```csharp
public sbyte RotationX { get; set; }
```

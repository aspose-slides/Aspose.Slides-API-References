## Enumeración PieSplitType

Representa un tipo de puntos de división en el segundo gráfico circular o de barras en un gráfico de pie de pie o de barra de pie.

```csharp
public enum PieSplitType
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Default | `0` | Especifica que los puntos de datos se dividirán utilizando el mecanismo predeterminado para este tipo de gráfico. |
| Custom | `1` | Especifica que los puntos de datos se dividirán entre el gráfico circular y el segundo gráfico de acuerdo con los valores de División Personalizada. |
| ByPercentage | `2` | Especifica que los puntos de datos se dividirán entre el gráfico circular y el segundo gráfico poniendo los puntos con un porcentaje menor que el porcentaje de Posición de División en el segundo gráfico. |
| ByPos | `3` | Especifica que los puntos de datos se dividirán entre el gráfico circular y el segundo gráfico poniendo la última Posición de División de los puntos de datos en el segundo gráfico. |
| ByValue | `4` | Especifica que los puntos de datos se dividirán entre el gráfico circular y el segundo gráfico poniendo los puntos de datos con valor menor que la Posición de División en el segundo gráfico. |

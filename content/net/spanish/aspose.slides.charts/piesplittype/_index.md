## Enumeración PieSplitType

Representa un tipo de puntos de división en el segundo gráfico circular o de barras en un gráfico de pie-of-pie o bar-of-pie.

```csharp
public enum PieSplitType
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Default | `0` | Especifica que los puntos de datos se dividirán utilizando el mecanismo predeterminado para este tipo de gráfico. |
| Custom | `1` | Especifica que los puntos de datos se dividirán entre el gráfico circular y el segundo gráfico de acuerdo con los valores de Custom Split. |
| ByPercentage | `2` | Especifica que los puntos de datos se dividirán entre el gráfico circular y el segundo gráfico colocando los puntos con un porcentaje menor que el porcentaje de la posición de división en el segundo gráfico. |
| ByPos | `3` | Especifica que los puntos de datos se dividirán entre el gráfico circular y el segundo gráfico colocando la última posición de división de los puntos de datos en el segundo gráfico. |
| ByValue | `4` | Especifica que los puntos de datos se dividirán entre el gráfico circular y el segundo gráfico colocando los puntos de datos con un valor menor que la posición de división en el segundo gráfico. |

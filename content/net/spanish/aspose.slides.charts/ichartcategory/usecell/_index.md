## IChartCategory.UseCell property

Si es verdadero, la propiedad AsCell es real. En otras palabras, la hoja de trabajo se usa para la categoría de almacenamiento (este caso admite una categoría de varios niveles). Si es falso, la propiedad AsLiteral es real. En otras palabras, la hoja de trabajo NO se usa para almacenar categorías (y este caso no admite categorías de varios niveles). Solo lecturaBoolean .

```csharp
public bool UseCell { get; }
```

### Observaciones

Para cambiar el valor de esta propiedad (para todas las categorías de la colección), establezca el nuevo valor en ChartCategoryCollection.UseCells property.

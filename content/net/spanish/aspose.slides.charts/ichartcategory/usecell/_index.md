## IChartCategory.UseCell propiedad

Si es verdadero, la propiedad AsCell es actual. En otras palabras, la hoja de trabajo se utiliza para almacenar la categoría (este caso admite una categoría de múltiples niveles). Si es falso, la propiedad AsLiteral es actual. En otras palabras, la hoja de trabajo NO se utiliza para almacenar la categoría (y este caso no admite categorías de múltiples niveles). Solo lectura Booleano.

```csharp
public bool UseCell { get; }
```

### Observaciones

Para cambiar el valor de esta propiedad (para todas las categorías en la colección), establezca un nuevo valor en la propiedad ChartCategoryCollection.UseCells.

## Propiedad DataLabelFormat.ShowPercentage

Representa el comportamiento de visualización de un valor porcentual de etiqueta de datos especificada en un gráfico. Verdadero muestra el valor porcentual. Falso para ocultar. Booleano de lectura/escritura.

```csharp
public bool ShowPercentage { get; set; }
```

### Observaciones

Si el padre de este objeto DataLabelFormat es una colección DataLabelCollection de etiquetas de datos, entonces esta propiedad obtiene o establece el valor predeterminado de la propiedad ShowPercentage para las nuevas etiquetas de datos en la colección DataLabelCollection. Establecer esta propiedad con un valor también establece este valor en la propiedad ShowPercentage para todas las etiquetas de datos en la colección DataLabelCollection (es decir, "DataLabels.DefaultDataLabelFormat.ShowPercentage = val;" causa que todas DataLabels[i].ShowPercentage sea igual a val).

## Propiedad DataLabelFormat.ShowCategoryName

Representa un comportamiento de visualización del nombre de categoría de la etiqueta de datos de un gráfico específico. Verdadero para mostrar el nombre de la categoría para las etiquetas de datos en un gráfico. Falso para ocultar. Booleano de lectura/escritura.

```csharp
public bool ShowCategoryName { get; set; }
```

### Observaciones

Si el padre de este objeto DataLabelFormat es una colección DataLabelCollection de etiquetas de datos, entonces esta propiedad obtiene o establece el valor predeterminado de la propiedad ShowCategoryName para las nuevas etiquetas de datos en la colección DataLabelCollection. Establecer esta propiedad con un valor también establece este valor a la propiedad ShowCategoryName para todas las etiquetas de datos en la colección DataLabelCollection (es decir, "DataLabels.DefaultDataLabelFormat.ShowCategoryName = val;" causa que todas las DataLabels[i].ShowCategoryName sean iguales a val).

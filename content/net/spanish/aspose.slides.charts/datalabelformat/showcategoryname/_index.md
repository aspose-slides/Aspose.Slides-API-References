## Propiedad DataLabelFormat.ShowCategoryName

Representa un comportamiento de visualización del nombre de categoría de etiqueta de datos para un gráfico especificado. Verdadero para mostrar el nombre de categoría para las etiquetas de datos en un gráfico. Falso para ocultar. Booleano de lectura/escritura.

```csharp
public bool ShowCategoryName { get; set; }
```

### Notas

Si el padre de este objeto DataLabelFormat es una colección DataLabelCollection de etiquetas de datos, entonces esta propiedad obtiene o establece el valor predeterminado de la propiedad ShowCategoryName para las nuevas etiquetas de datos en la colección DataLabelCollection. Establecer esta propiedad con un valor también asigna este valor a la propiedad ShowCategoryName para todas las etiquetas de datos en la colección DataLabelCollection (es decir, "DataLabels.DefaultDataLabelFormat.ShowCategoryName = val;" causa que todas DataLabels[i].ShowCategoryName sea igual a val).

## DataLabelCollection.IsVisible property

Falso significa que la etiqueta de datos no es visible por defecto (y que todas las banderas Show*-flags (ShowValue, ...) de la propiedad DefaultDataLabelFormat son falsas). Solo lectura Boolean.

```csharp
public bool IsVisible { get; }
```

### Remarks

Si la etiqueta de datos es visible por defecto, puedes hacer que esté oculta por defecto con el método Hide(). Pero si la etiqueta de datos no es visible por defecto (IsVisible es falso), puedes hacer que la etiqueta de datos sea "visible por defecto" configurando las banderas Show*-flags (ShowValue, ...) de la propiedad DefaultDataLabelFormat a su estado verdadero.

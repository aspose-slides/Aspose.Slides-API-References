## DataLabelCollection.Hide method

Oculte la etiqueta de datos de forma predeterminada configurando todos los indicadores Mostrar* (ShowValue, ...) de la propiedad DefaultDataLabelFormat en estado falso. IsVisible será falso después de esto.

```csharp
public void Hide()
```

### Observaciones

Si la etiqueta de datos no está visible de forma predeterminada (IsVisible es falso), puede hacer que la etiqueta de datos sea "visible de forma predeterminada" configurando Mostrar*-flags (ShowValue, ...) de la propiedad DefaultDataLabelFormat en estado verdadero.

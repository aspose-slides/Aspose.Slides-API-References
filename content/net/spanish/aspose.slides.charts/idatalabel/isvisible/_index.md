## Propiedad IDataLabel.IsVisible

Falso significa que la etiqueta de datos no es visible (y por lo tanto todas las banderas Show*-flags (ShowValue, ...) son falsas). Solo de lectura Booleano.

```csharp
public bool IsVisible { get; }
```

### Observaciones

Si la etiqueta de datos es visible, puedes hacerla oculta con el método Hide(). Pero si la etiqueta de datos no es visible (IsVisible es falso), puedes hacerla visible configurando las banderas Show*-flags (ShowValue, ...) en el estado verdadero.

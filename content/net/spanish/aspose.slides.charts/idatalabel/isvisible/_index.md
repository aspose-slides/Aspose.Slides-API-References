## Propiedad IDataLabel.IsVisible

False significa que la etiqueta de datos no es visible (y, por lo tanto, todas las banderas Show*- (ShowValue, ...) son falsas). Solo lectura Boolean.

```csharp
public bool IsVisible { get; }
```

### Comentarios

Si la etiqueta de datos es visible, puedes ocultarla con el método Hide(). Pero si la etiqueta de datos no es visible (IsVisible es false) puedes hacerla visible configurando las banderas Show*- (ShowValue, ...) en verdadero.

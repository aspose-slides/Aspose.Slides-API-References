## IDataLabel.Hide method

Hacer que la etiqueta de datos esté oculta configurando todas las banderas Show*-flags (ShowValue, ...) en estado falso. IsVisible será falso después de esto.

```csharp
public void Hide()
```

### Remarks

Si la etiqueta de datos no es visible (IsVisible es falso), puedes hacer que la etiqueta de datos sea visible configurando las banderas Show*-flags (ShowValue, ...) en estado verdadero.

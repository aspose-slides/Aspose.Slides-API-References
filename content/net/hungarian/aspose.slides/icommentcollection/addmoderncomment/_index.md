## ICommentCollection.AddModernComment metódus

Új modern megjegyzés hozzáadása a gyűjtemény végére.

```csharp
public IModernComment AddModernComment(string text, ISlide slide, IShape shape, PointF position, 
    DateTime creationTime)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| text | String | Az új modern megjegyzés egyszerű szövege. |
| slide | ISlide | Az előadásban lévő dián, ahová az új modern megjegyzést hozzáadjuk. |
| shape | IShape | Az a forma a dián, amelyhez az új modern megjegyzés kapcsolódik. |
| position | PointF | A pozíció a dián, ahová az új modern megjegyzést hozzáadjuk. |
| creationTime | DateTime | A modern megjegyzés létrehozásának időpontja. |

### Visszatérési érték

Hozzáadott modern megjegyzés.

### Példák

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ICommentAuthor newAuthor = pres.CommentAuthors.AddAuthor("Some Author", "SA");
    newAuthor.Comments.AddModernComment("This is modern comment", pres.Slides[0], null, new PointF(100, 100), DateTime.Now);

    pres.Save(outPptxFileName, SaveFormat.Pptx);
}
```

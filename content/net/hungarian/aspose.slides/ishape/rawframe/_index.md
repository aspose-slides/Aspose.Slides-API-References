## IShape.RawFrame tulajdonság

Visszaadja vagy beállítja a nyers alakkeret tulajdonságait. Olvasás/írás [`IShapeFrame`](../../ishapeframe).

```csharp
public IShapeFrame RawFrame { get; set; }
```

### Példák

Az a kód, amely megpróbál egy nem definiált keretet hozzárendelni az IShape.Frame-hez, általános esetben nem értelmezhető (különösen akkor, amikor a szülő GroupShape többszörösen be van ágyazva más GroupShape-ekbe). Például:

```csharp
IShape shape = ...;
shape.Frame = new ShapeFrame(float.NaN, float.NaN, float.NaN, float.NaN, NullableBool.NotDefined, NullableBool.NotDefined, float.NaN);
```

vagy

```csharp
slide.Shapes.AddAutoShape(ShapeType.RoundCornerRectangle, float.NaN, float.NaN, float.NaN, float.NaN);
```

Az ilyen kód félreérthető helyzetekhez vezethet. Ezért korlátozások lettek bevezetve az IShape.Frame-nél nem definiált értékek használatára. Az x, y, width, height, flipH, flipV és rotationAngle értékeket definiálni kell (nem float.NaN vagy NullableBool.NotDefined). A fenti példakód most ArgumentException kivételt dob. Ez az alábbi felhasználási esetekre vonatkozik:

```csharp
IShape shape = ...;
shape.Frame = ...; // nem lehet nedefiniált

IShapeCollection shapes = ...;
// x, y, width, height paraméterek nem lehetnek float.NaN:
{
    shapes.AddAudioFrameCD(...);
    shapes.AddAudioFrameEmbedded(...);
    shapes.AddAudioFrameLinked(...);
    shapes.AddAutoShape(...);
    shapes.AddChart(...);
    shapes.AddConnector(...);
    shapes.AddOleObjectFrame(...);
    shapes.AddPictureFrame(...);
    shapes.AddSmartArt(...);
    shapes.AddTable(...);
    shapes.AddVideoFrame(...);
    shapes.InsertAudioFrameEmbedded(...);
    shapes.InsertAudioFrameLinked(...);
    shapes.InsertAutoShape(...);
    shapes.InsertChart(...);
    shapes.InsertConnector(...);
    shapes.InsertOleObjectFrame(...);
    shapes.InsertPictureFrame(...);
    shapes.InsertTable(...);
    shapes.InsertVideoFrame(...);
}
```

Az IShape.RawFrame keret tulajdonságai azonban lehetnek nedefiniáltak. Ez akkor értelmes, amikor az alakzat egy helyőrzőhöz van kapcsolva. Ebben az esetben a nedefiniált alakzatkeret értékeket a szülő helyőrző alakzat felülírja. Ha az alakzatnak nincs szülő helyőrző alakzata, akkor az alakzat alapértelmezett értékeket használ, amikor az IShape.RawFrame alapján kiszámítja a tényleges keretet. Az alapértelmezett értékek 0 és NullableBool.False az x, y, width, height, flipH, flipV és rotationAngle esetén. Például:

```csharp
IShape shape = ...; // a shape a placeholderhez van kapcsolva
shape.RawFrame = new ShapeFrame(float.NaN, float.NaN, 100, float.NaN, NullableBool.NotDefined, NullableBool.NotDefined, 0); // most a shape örökli az x, y, height, flipH, flipV értékeket a placeholderből, és felülírja a width=100 és rotationAngle=0 értékeket.
```

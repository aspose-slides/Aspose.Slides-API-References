## ISlideShowTransition.AdvanceAfter tulajdonság

Ez az attribútum meghatározza, hogy a diavetítés egy adott idő után a következő diára lép-e. Olvasás/írás Boolean.

```csharp
public bool AdvanceAfter { get; set; }
```

### Példák

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    // Az első dia átmenet lekérése
    ISlideShowTransition slideTransition = pres.Slides[0].SlideShowTransition;
    
    // Ellenőrzi, hogy az Advance Slide After jelző be van-e jelölve
    if (slideTransition.AdvanceAfter)
    {
        // Az Advance Slide After időérték lekérése
        uint advanceAfterTime = slideTransition.AdvanceAfterTime;
    }
}
```

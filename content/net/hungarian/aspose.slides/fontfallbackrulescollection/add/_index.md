## FontFallBackRulesCollection.Add metódus

Megadott FallBack szabály hozzáadása a gyűjtemény végéhez.

```csharp
public void Add(IFontFallBackRule sourceRule)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| sourceRule | IFontFallBackRule | A hozzáadandó szabály |

### Példák

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    //Üres vagy előre inicializált szabályok gyűjteményének lekérése a FontsManager-ből
```

## CommentCollection.AddModernComment metódus

Új modern kommentet ad hozzá a gyűjtemény végéhez.

```csharp
public IModernComment AddModernComment(string text, ISlide slide, IShape shape, PointF position, 
    DateTime creationTime)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| text | String | Új modern komment tiszta szövege. |
| slide | ISlide | Dia a prezentációban, ahol új modern kommentet ad hozzá. |
| shape | IShape | Diaelemet, amelyhez az új modern komment tartozik. |
| position | PointF | Pozíció a diáron, ahol új modern kommentet ad hozzá. |
| creationTime | DateTime | Modern komment létrehozásának időpontja. |

### Visszatérési érték

Hozzáadott modern komment.

### Példák

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ICommentAuthor newAuthor = pres.CommentAuthors.AddAuthor("Some Author", "SA");
    newAuthor.Comments.AddModernComment("This is modern comment", pres.Slides[0], null, new PointF(100, 100), DateTime.Now);

    pres.Save(outPptxFileName, SaveFormat.Pptx);
}
```

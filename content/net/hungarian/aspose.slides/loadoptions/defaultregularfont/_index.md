## LoadOptions.DefaultRegularFont tulajdonság

Visszaadja vagy beállítja a Regular betűtípust, amelyet akkor használnak, ha a forrásbetűtípus nem található. Olvasás/írás String.

```csharp
public string DefaultRegularFont { get; set; }
```

### Példák

Az alábbi példa bemutatja, hogyan állíthatók be az alapértelmezett betűtípusok a PowerPoint-prezentáció rendereléséhez.

```csharp
[C#]
// Használja a betöltési beállításokat az alapértelmezett reguláris és ázsiai betűtípusok meghatározásához
LoadOptions loadOptions = new LoadOptions(LoadFormat.Auto);
loadOptions.DefaultRegularFont = "Wingdings";
loadOptions.DefaultAsianFont = "Wingdings";
// Betölti a prezentációt
using (Presentation pptx = new Presentation("DefaultFonts.pptx", loadOptions))
{
    // Dia bélyegkép generálása
    pptx.Slides[0].GetThumbnail(1, 1).Save("output_out.png", ImageFormat.Png);
    // PDF generálása
    pptx.Save("output_out.pdf", SaveFormat.Pdf);
    // XPS generálása
    pptx.Save("output_out.xps", SaveFormat.Xps);
}
```

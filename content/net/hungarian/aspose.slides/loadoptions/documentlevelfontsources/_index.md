## LoadOptions.DocumentLevelFontSources tulajdonság

Megadja a külső betűtípusok forrásait, amelyeket a prezentáció használ. Ezek a betűtípusok a prezentáció teljes élettartama alatt elérhetők, és nem oszthatók meg más prezentációkkal.

```csharp
public IFontSources DocumentLevelFontSources { get; set; }
```

### Példák

A következő példa bemutatja, hogyan lehet megadni a PowerPoint-prezentációval használt egyéni betűtípusokat.

```csharp
[C#]
byte[] memoryFont1 = File.ReadAllBytes("customfonts\\CustomFont1.ttf");
byte[] memoryFont2 = File.ReadAllBytes("customfonts\\CustomFont2.ttf");
LoadOptions loadOptions = new LoadOptions();
loadOptions.DocumentLevelFontSources.FontFolders = new string[] { "assets\\fonts", "global\\fonts" };
loadOptions.DocumentLevelFontSources.MemoryFonts = new byte[][] { memoryFont1, memoryFont2 };
using (IPresentation presentation = new Presentation("MyPresentation.pptx", loadOptions))
{
    //munka a prezentációval
    //A CustomFont1, CustomFont2, valamint az assets\fonts és a global\fonts mappákból és azok alkönyvtáraiból származó betűtípusok elérhetők a prezentáció számára
}
```

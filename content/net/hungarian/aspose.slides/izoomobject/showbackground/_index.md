## IZoomObject.ShowBackground tulajdonság

Értéket ad vissza vagy állít be, amely meghatározza, hogy a Zoom használja-e a cél diára vonatkozó háttérképet. Olvasás/írás Boolean. Alapértelmezett érték: true

```csharp
public bool ShowBackground { get; set; }
```

### Példák

A példa bemutatja a Zoom objektum képének hátterének eltávolítását:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
    zoomFrame.ShowBackground = false;
}
```

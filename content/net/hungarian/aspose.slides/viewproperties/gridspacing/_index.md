## ViewProperties.GridSpacing tulajdonság

Visszaadja vagy beállítja a rácstávolságot, amelyet a prezentációs dokumentum alapjául szolgáló rácshoz kell használni, pontban megadva. Olvasás/írás Single.

```csharp
public float GridSpacing { get; set; }
```

### Megjegyzés

A rácstávolság értékének pozitív számnak kell lennie. A tipikus értéktartomány 1 mm (2.8349607 pont) és 2 hüvelyk (144 pont) között van.

### Példák

Az alábbi minta kód megmutatja, hogyan lehet megváltoztatni a rácstávolságot egy PowerPoint-prezentációban.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

## IFontFallBackRulesCollection.Add metódus

Új FallBack szabályt ad a gyűjtemény végéhez.

```csharp
public void Add(IFontFallBackRule sourceRule)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| sourceRule | IFontFallBackRule | A hozzáadáshoz megadott szabály |

### Példák

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    //A FontsManagerből az üres vagy előre inicializált szabálygyűjtemény lekérése
    IFontFallBackRulesCollection rulesList = pres.FontsManager.FontFallBackRulesCollection;

    //Új szabály hozzáadása a gyűjteményhez
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));
}
```

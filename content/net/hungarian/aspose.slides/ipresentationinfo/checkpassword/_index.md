## IPresentationInfo.CheckPassword módszer

Ellenőrzi, hogy a jelszó helyes-e egy nyílt jelszóval védett prezentáció esetén.

```csharp
public bool CheckPassword(string password)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| password | String | A vizsgálandó jelszó. |

### Visszatérési érték

True ha a prezentáció nyílt jelszóval védett és a jelszó helyes, egyébként false.

### Megjegyzés

Ha a jelszó null vagy üres, ez a módszer false értéket ad vissza.

### Példák

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo("pres.pptx");
bool isPasswordCorrect = info.CheckPassword("my_password");
```

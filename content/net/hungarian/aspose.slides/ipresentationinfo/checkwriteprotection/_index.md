## IPresentationInfo.CheckWriteProtection method

Ellenőrzi, hogy a módosításhoz szükséges jelszó helyes-e egy írásvédett bemutató esetén.

```csharp
public bool CheckWriteProtection(string password)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| password | String | A ellenőrzendő jelszó. |

### Visszatérési érték

True, ha a bemutató írásvédett és a jelszó helyes. False egyébként.

### Kivétel

| Kivétel | feltétel |
| --- | --- |
| InvalidOperationException |  |

### Megjegyzések

1. A [`IsWriteProtected`](../iswriteprotected) tulajdonságot ellenőriznie kell a metódus hívása előtt.  
2. Ha a password null vagy üres, ez a metódus false értéket ad vissza.

### Példák

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
if (info.IsWriteProtected == NullableBool.True)
{
    bool isWriteProtectedByPassword = info.CheckWriteProtection("my_password");
}
```

## ParagraphCollection.CopyTo metódus

Átmásolja az ICollection elemeit egy Array-ba, egy meghatározott Array indexnél kezdve.

```csharp
public void CopyTo(IParagraph[] array, int arrayIndex)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| array | IParagraph[] | Az egy dimenziós Array, amely a ICollection-ből másolt elemek célja. Az Array-nak nulláról induló indexeléssel kell rendelkeznie. |

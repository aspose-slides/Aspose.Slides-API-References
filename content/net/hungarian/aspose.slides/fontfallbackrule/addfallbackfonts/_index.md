## AddFallBackFonts(string) {#addfallbackfonts}

Új betűtípust (betűtípusokat) ad a FallBack betűtípusok listájához.

```csharp
public void AddFallBackFonts(string fontName)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| fontName | String | A betűtípus neve vagy nevei (vesszővel elválasztva) a FallBack számára |

### Példák

```csharp
[C#]
// Hozzon létre egy új példányt a FontFallBackRule-ból
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho");

//Adjon hozzá egy második betűtípust a szabályhoz
newRule.AddFallBackFonts("MS Gothic");
//Adjon hozzá egy harmadik és negyedik betűtípust a szabályhoz
newRule.AddFallBackFonts("Tahoma, Times New Roman");
```

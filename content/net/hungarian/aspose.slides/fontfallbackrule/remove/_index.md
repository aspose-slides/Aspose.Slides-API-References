## FontFallBackRule.Remove metódus

Eltávolítja egy adott FallBack betűtípus első előfordulását a listából.

```csharp
public void Remove(string fontName)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| fontName | String | A betűtípus neve, amelyet el kell távolítani a listából. |

### Példák

```csharp
[C#]
// Létrehoz egy szabályt, amely betűkészlet-listát tartalmaz.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// Eltávolítja a Tahoma betűtípust a listából.
newRule.Remove ("Tahoma");
```

## FontFallBackRule.IndexOf method

Visszaadja a megadott szabály indexét a gyűjteményben.

```csharp
public int IndexOf(string fontName)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| fontName | String | A keresendő betűtípus neve. |

### Visszatérési érték

A betűtípus indexe, vagy -1, ha a betűtípus nincs a listában.

### Példák

```csharp
[C#]
// Hozzon létre egy szabályt, amely betűtípusok listáját tartalmazza.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// Szerezze meg a Tahoma indexét.
int tahomaIndex = newRule.IndexOf("Tahoma");
```

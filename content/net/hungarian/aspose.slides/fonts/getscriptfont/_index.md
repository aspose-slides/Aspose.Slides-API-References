## Fonts.GetScriptFont metódus

A megadott szkriptcímkéhez kapcsolódó betűtípus nevét adja vissza a bemutató témájából.

```csharp
public string GetScriptFont(string script)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| script | String | A BCP-47 szkriptkód (pl. "Latn", "Cyrl", "Jpan"), amelyet az írásrendszer azonosítására használnak. |

### Visszatérési érték

A megadott szkripthez használt betűtípus neve, vagy `null`, ha a szkript nincs definiálva.

### Példák

Ez a példa bemutatja, hogyan lehet lekérni a bemutató témájában a cirill szkripthez rendelt betűtípust.

```csharp
[C#]
string font = presentation.MasterTheme.FontScheme.Major.GetScriptFont("Cyrl");
Console.WriteLine("Font for Cyrillic script: " + font);
```

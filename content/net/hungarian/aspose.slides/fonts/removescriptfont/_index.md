## Fonts.RemoveScriptFont metódus

Eltávolítja a betűtípus-beállítást, amely egy adott script címkéhez kapcsolódik a téma betűtípus-gyűjteményéből.

```csharp
public void RemoveScriptFont(string script)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| script | String | A BCP-47 szkript kód, amelynek a betűtípus-beállítását el kell távolítani. |

### Példák

Ez a példa bemutatja, hogyan lehet eltávolítani a héber szkript betűtípus-leképezését:

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.RemoveScriptFont("Hebr");
```

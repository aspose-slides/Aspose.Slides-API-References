## Fonts.SetScriptFont metódus

Egy betűkészlet nevet rendel egy adott írásrendszer címkéhez, amely meghatározza, hogyan lesz a szkript szövege a bemutatóban megjelenítve.

```csharp
public void SetScriptFont(string script, string fontName)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| script | String | Az írásrendszert azonosító BCP-47 szkriptkód (például "Arab", "Hebr", "Hans"). |
| fontName | String | A meghatározott szkripthez hozzárendelendő betűkészlet neve. |

### Példák

Ez a példa bemutatja, hogyan állítható be az arab írásrendszer betűkészlete "Segoe UI" értékre:

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segue UI");
```

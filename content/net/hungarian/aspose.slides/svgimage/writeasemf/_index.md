## SvgImage.WriteAsEmf metódus

Mentse az SVG képet EMF fájlként.

```csharp
public void WriteAsEmf(Stream stream)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| stream | Stream | Célfolyam |

### Kivétel

| kivétel | feltétel |
| --- | --- |
| ArgumentNullException | A célfolyam **null** |

### Példák

A következő példa bemutatja, hogyan mentse az SVG képet metafájlba.

```csharp
[C#]
// Létrehozza az új SVG képet
ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));

// Elmenti az SVG képet metafájlba
using (var fileStream = System.IO.File.OpenWrite("SvgAsEmf.emf"))
    svgImage.WriteAsEmf(fileStream);
```

Ez a példa bemutatja, hogyan adja hozzá az SVG képet metafájlként a prezentáció képgyűjteményéhez.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // Létrehozza az új SVG képet
    ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));
    using (var memStream = new MemoryStream())
    {
        // Elmenti az SVG képet metafájlba
        svgImage.WriteAsEmf(memStream);
        // Hozzáadja a metafájlt a képgyűjteményhez
        pres.Images.AddImage(memStream.ToArray());
    }
}
```

## ShapeCollection.AddMathShape metódus

Új téglalap automatikus alakzatot hoz létre a matematikai tartalom tárolására, és a forma gyűjtemény végéhez adja.

```csharp
public IAutoShape AddMathShape(float x, float y, float width, float height)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| x | Single | A forma keretének x-koordinátája pontban. |
| y | Single | A forma keretének y-koordinátája pontban. |
| width | Single | A forma keretének szélessége pontban. |
| height | Single | A forma keretének magassága pontban. |

### Visszatérési érték

Az újonnan létrehozott [`IAutoShape`](../../iautoshape).

### Példák

Az alábbi példa bemutatja, hogyan adhatunk hozzá matematikai egyenletet a PowerPoint bemutatóhoz.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
   IAutoShape mathShape = pres.Slides[0].Shapes.AddMathShape(0, 0, 720, 150);
   var mathParagraph = (mathShape.TextFrame.Paragraphs[0].Portions[0] as MathPortion).MathParagraph;
   var fraction = new MathematicalText("x").Divide("y");
   mathParagraph.Add(new MathBlock(fraction));
   var mathBlock = new MathematicalText("c")
        .SetSuperscript("2")
        .Join("=")
        .Join(new MathematicalText("a").SetSuperscript("2"))
        .Join("+")
        .Join(new MathematicalText("b").SetSuperscript("2"));
    mathParagraph.Add(mathBlock);
    pres.Save("math.pptx", SaveFormat.Pptx);
}
```

## ShapeCollection.AddSummaryZoomFrame metódus

Új Summary Zoom keretet hoz létre, és a shape gyűjtemény végéhez adja hozzá.

```csharp
public ISummaryZoomFrame AddSummaryZoomFrame(float x, float y, float width, float height)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| x | Single | Az új Summary Zoom keret x-koordinátája pontban. |
| y | Single | Az új Summary Zoom keret y-koordinátája pontban. |
| width | Single | Az új Summary Zoom keret szélessége pontban. |
| height | Single | Az új Summary Zoom keret magassága pontban. |

### Visszatérési érték

Az újonnan létrehozott [`ISummaryZoomFrame`](../../isummaryzoomframe).

### Kivételek

| kivétel | feltétel |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | Kivétel keletkezik, ha a prezentációban nincsenek szekciók, vagy ha a cél diák nem tartozik egy szekcióhoz sem. |

### Megjegyzések

Ez a metódus új Summary Zoom-ot hoz létre, és a prezentáció minden szekciójához egy objektumgyűjteményt helyez bele.

### Példák

Ez a példa bemutatja egy Summary Zoom objektum hozzáadását a gyűjtemény végéhez (feltételezve, hogy a "Presentation.pptx" prezentációban legalább két szekció van):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSummaryZoomFrame(150, 20, 500, 250);
}
```

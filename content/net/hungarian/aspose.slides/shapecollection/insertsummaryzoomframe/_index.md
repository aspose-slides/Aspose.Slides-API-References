## ShapeCollection.InsertSummaryZoomFrame method

Új Summary Zoom keretet hoz létre, és a megadott indexen beszúrja a shape gyűjteménybe.

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| index | Int32 | A nullától számított index, ahová a Summary Zoom keretet be kell szúrni. |
| x | Single | Az új Summary Zoom keret x-koordinátája pontban. |
| y | Single | Az új Summary Zoom keret y-koordinátája pontban. |
| width | Single | Az új Summary Zoom keret szélessége pontban. |
| height | Single | Az új Summary Zoom keret magassága pontban. |

### Visszatérési érték

Az újonnan létrehozott [`ISummaryZoomFrame`](../../isummaryzoomframe).

### Kivételek

| kivétel | feltétel |
| --- | --- |

### Megjegyzés

Ez a metódus egy Summary Zoom keretet hoz létre, amely az összes szekcióhoz tartozó összefoglaló hivatkozásokat aggregálja a prezentációban.

### Példák

Ez a példa bemutatja egy Summary Zoom objektum létrehozását és beszúrását a gyűjtemény megadott indexén (feltételezve, hogy a "Presentation.pptx" prezentációban legalább két szekció található):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSummaryZoomFrame(2, 150, 20, 50, 50);
}
```

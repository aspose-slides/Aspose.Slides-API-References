```

| Paraméter | Típus | Leírás |

| x | Single | Az új Zoom-keret x-koordinátája pontban. |
| y | Single | Az új Zoom-keret y-koordinátája pontban. |
| width | Single | Az új Zoom-keret szélessége pontban. |
| height | Single | Az új Zoom-keret magassága pontban. |
| slide | ISlide | A Zoom-keret által hivatkozott [`ISlide`](../../islide); a prezentációnak kell tartozzon. |

Ez a példa bemutatja, hogyan lehet egy Zoom-objektumot hozzáadni a gyűjtemény végéhez (feltételezve, hogy a „Presentation.pptx” prezentációban legalább két dia van):

```csharp

}
```

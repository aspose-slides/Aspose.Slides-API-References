## InsertAudioFrameEmbedded(int, float, float, float, float, Stream) {#insertaudioframeembedded_1}

Új hangkeretet hoz létre egy beágyazott WAV fájllal, és beszúrja a shape gyűjteménybe a megadott indexen. A beágyazott hang hozzáadódik a Presentation.Audios gyűjteményhez.

```csharp
public IAudioFrame InsertAudioFrameEmbedded(int index, float x, float y, float width, float height, 
    Stream audio_stream)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| index | Int32 | A nulla-alapú index, amelynél be kell szúrni a hangkeretet. |
| x | Single | Az új hangkeret x-koordinátája pontban. |
| y | Single | Az új hangkeret y-koordinátája pontban. |
| width | Single | Az új hangkeret szélessége pontban. |
| height | Single | Az új hangkeret magassága pontban. |
| audio_stream | Stream | Egy bemeneti adatfolyam, amely beágyazandó WAV hangadatokat tartalmaz. |

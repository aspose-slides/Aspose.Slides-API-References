## AddSectionZoomFrame(float, float, float, float, ISection) {#addsectionzoomframe}

Létrehoz egy új Section Zoom keretet, és a alakzatgyűjtemény végére adja hozzá.

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| x | Single | Az új Section Zoom keret x-koordinátája pontban. |
| y | Single | Az új Section Zoom keret y-koordinátája pontban. |
| width | Single | Az új Section Zoom keret szélessége pontban. |
| height | Single | Az új Section Zoom keret magassága pontban. |
| section | ISection | A Section Zoom keret által hivatkozott [`ISection`](../../isection); a prezentációhoz kell tartozzon, és legalább egy diát kell tartalmazzon. |

### Visszatérési érték

Az újonnan létrehozott [`ISectionZoomFrame`](../../isectionzoomframe).

### Kivételek

| kivétel | feltétel |
| --- | --- |
| ArgumentException | Akkor dobódik, ha a hivatkozott szekció nem tartozik az aktuális prezentációhoz, vagy nem tartalmaz diát. |

### Példák

Ez a példája bemutatja, hogyan lehet egy Section Zoom objektumot a gyűjtemény végére hozzáadni (feltételezve, hogy a "Presentation.pptx" prezentációban legalább két szekció van):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1]);
}
```

## AddSectionZoomFrame(float, float, float, float, ISection, IPPImage) {#addsectionzoomframe_1}

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section, IPPImage image)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| x | Single | Az új Section Zoom keret x-koordinátája pontban. |
| y | Single | Az új Section Zoom keret y-koordinátája pontban. |
| width | Single | Az új Section Zoom keret szélessége pontban. |
| height | Single | Az új Section Zoom keret magassága pontban. |
| section | ISection | A Section Zoom keret által hivatkozott [`ISection`](../../isection); a prezentációhoz kell tartozzon, és legalább egy diát kell tartalmazzon. |
| image | IPPImage | A [`IPPImage`](../../ippimage) a Section Zoom kereten belül megjelenítendő. |

### Visszatérési érték

Az újonnan létrehozott [`ISectionZoomFrame`](../../isectionzoomframe).

### Kivételek

| kivétel | feltétel |
| --- | --- |
| ArgumentException | Akkor dobódik, ha a hivatkozott szekció nem tartozik az aktuális prezentációhoz, vagy nem tartalmaz diát. |

### Példák

Ez a példája bemutatja, hogyan lehet egy Section Zoom objektumot a gyűjtemény végére hozzáadni (feltételezve, hogy a "Presentation.pptx" prezentációban legalább két szekció van):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IPPImage image = pres.Images.AddImage(Image.FromFile("image.png"));
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1], image);
}
```

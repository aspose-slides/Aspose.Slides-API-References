## AddAudioFrameEmbedded(float, float, float, float, Stream) {#addaudioframeembedded_1}}

Új hangkeretet hoz létre egy beágyazott WAV fájllal, és a forma gyűjtemény végéhez adja. A beágyazott hang hozzáadódik a Presentation.Audios gyűjteményhez.

```csharp
public IAudioFrame AddAudioFrameEmbedded(float x, float y, float width, float height, 
    Stream audio_stream)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| x | Single | Az új hangkeret x-koordinátája pontban. |
| y | Single | Az új hangkeret y-koordinátája pontban. |
| width | Single | Az új hangkeret szélessége pontban. |
| height | Single | Az új hangkeret magassága pontban. |
| audio_stream | Stream | Egy bemeneti adatfolyam, amely WAV hangadatokat tartalmaz a beágyazáshoz. |

### Visszatérési érték

Az újonnan létrehozott [`IAudioFrame`](../../iaudioframe).

### Példák

Az alábbi példák bemutatják, hogyan hozható létre hangkeret.

```csharp
[C#]
// Létrehoz egy Presentation osztályt, amely egy bemutató fájlt reprezentál
using (Presentation pres = new Presentation())
{
    // Lekéri az első diát
    ISlide sld = pres.Slides[0];
    // Betölti a wav hangfájlt az adatfolyamba
    FileStream fstr = new FileStream("sampleaudio.wav", FileMode.Open, FileAccess.Read);
    // Hozzáadja a hangkeretet
    IAudioFrame audioFrame = sld.Shapes.AddAudioFrameEmbedded(50, 150, 100, 100, fstr);
    // Beállítja a lejátszási módot és a hangerőt a hanghoz
    audioFrame.PlayMode = AudioPlayModePreset.Auto;
    audioFrame.Volume = AudioVolumeMode.Loud;
    // Kiírja a PowerPoint fájlt a lemezre
    pres.Save("AudioFrameEmbed_out.pptx", SaveFormat.Pptx);
}
```

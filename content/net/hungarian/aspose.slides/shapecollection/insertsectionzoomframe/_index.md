## InsertSectionZoomFrame(int, float, float, float, float, ISection) {#insertsectionzoomframe}

Új Section Zoom keretet hoz létre, és a megadott indexnél beilleszti a alakzatgyűjteménybe.

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| index | Int32 | A nulla-bázisú index, amelynél a Section Zoom keretet be kell illeszteni. |
| x | Single | Az új Section Zoom keret x-koordinátája pontban. |
| y | Single | Az új Section Zoom keret y-koordinátája pontban. |
| width | Single | Az új Section Zoom keret szélessége pontban. |
| height | Single | Az új Section Zoom keret magassága pontban. |
| section | ISection | A Section Zoom keret által hivatkozott [`ISection`](../../isection); a bemutatóhoz kell tartozzon, és legalább egy diát kell tartalmazzon. |

### Return Value

Az újonnan létrehozott [`ISectionZoomFrame`](../../isectionzoomframe).

### Kivételek

| kivétel | feltétel |
| --- | --- |
| ArgumentException | Ha a hivatkozott szakasz nem tartozik az aktuális bemutatóhoz, vagy nem tartalmaz diát. |

### Példák

Ez a példa bemutatja egy Section Zoom objektum létrehozását és beillesztését a gyűjtemény megadott indexén (feltételezve, hogy a "Presentation.pptx" bemutatóban legalább két szakasz van):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSectionZoomFrame(2, 150, 20, 50, 50, pres.Sections[1]);
}
```

public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width,

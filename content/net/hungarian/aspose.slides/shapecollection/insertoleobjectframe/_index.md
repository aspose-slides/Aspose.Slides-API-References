## InsertOleObjectFrame(int, float, float, float, float, IOleEmbeddedDataInfo) {#insertoleobjectframe}

Új OLE objektumkeretet hoz létre, és a megadott indexnél beilleszti azt a shape gyűjteménybe.

```csharp
public IOleObjectFrame InsertOleObjectFrame(int index, float x, float y, float width, float height, 
    IOleEmbeddedDataInfo dataInfo)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | Int32 | A nulla-alapú index, amelyen az OLE objektumkeretet be kell illeszteni. |
| x | Single | Az új OLE keret x-koordinátája pontokban. |
| y | Single | Az új OLE keret y-koordinátája pontokban. |
| width | Single | Az új OLE keret szélessége pontokban. |
| height | Single | Az új OLE keret magassága pontokban. |
| dataInfo | IOleEmbeddedDataInfo | A beágyazott OLE adat információja ([`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo)). |

### Visszatérési érték

Az újonnan létrehozott [`IOleObjectFrame`](../../ioleobjectframe).

### Példák

Ez a példa bemutatja egy OLE objektum beszúrását a második indexen:

```csharp
[C#]
byte[] fileData = File.ReadAllBytes("test.zip");
IOleDataInfo dataInfo = new OleDataInfo(fileData, "zip");
IOleObjectFrame oleObjectFrame = slidees.Shapes.AddOleObjectFrame(2, 150, 20, 50, 50, dataInfo);
```

## Presentation.HighlightRegex metódus

Kiemeli a reguláris kifejezés összes egyezését a megadott színnel.

```csharp
public void HighlightRegex(Regex regex, Color highlightColor, IFindResultCallback callback)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| regex | Regex | A kiemelendő karakterláncokat meghatározó reguláris kifejezés Regex. |
| highlightColor | Color | A szöveg kiemeléséhez használandó szín. |
| callback | IFindResultCallback | A keresési eredmények fogadására szolgáló callback objektum [`IFindResultCallback`](../../ifindresultcallback). |

### Példák

A következő kódrészlet bemutatja, hogyan lehet reguláris kifejezéssel kiemelni a szöveget egy PowerPoint Presentation-ben.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// az összes 10 vagy annál több karakteres szót kiemelve
	presentation.HighlightRegex(regex, Color.Blue, null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

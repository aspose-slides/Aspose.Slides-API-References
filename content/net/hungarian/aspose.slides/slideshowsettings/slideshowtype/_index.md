## SlideShowSettings.SlideShowType tulajdonság

Lekéri vagy beállítja a diavetítés típusát. A következő `SlideShowType` ősei képviselik: [`BrowsedAtKiosk`](../../browsedatkiosk), [`PresentedBySpeaker`](../../presentedbyspeaker) és [`BrowsedByIndividual`](../../browsedbyindividual)

```csharp
public SlideShowType SlideShowType { get; set; }
```

### Példák

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // a "Kijelző kioszk módban (teljes képernyő)" típus beállításához
    pres.SlideShowSettings.SlideShowType = new BrowsedAtKiosk();
    
    // a "Kijelző egyéni (ablak)" típus beállításához
    pres.SlideShowSettings.SlideShowType = new BrowsedByIndividual();
    
    // a "Előadó által bemutatott (teljes képernyő)" típus beállításához
    pres.SlideShowSettings.SlideShowType = new PresentedBySpeaker();
}
```

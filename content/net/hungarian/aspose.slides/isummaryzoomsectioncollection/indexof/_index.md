## ISummaryZoomSectionCollection.IndexOf módszer

Visszaadja a megadott SummaryZoomSection objektum indexét.

```csharp
public int IndexOf(ISummaryZoomSection summaryZoomSection)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| summaryZoomSection | ISummaryZoomSection | A megtalálandó SummaryZoomSection objektum [`ISummaryZoomSection`](../../isummaryzoomsection). |

### Visszatérési érték

A SummaryZoomSection objektum indexe, vagy -1, ha a SummaryZoomSection objektum nincs ebben a gyűjteményben.

### Példák

A példa azt mutatja be, hogyan lehet index alapján lekérni a Summary Zoom Section elemet:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    ISummaryZoomFrame zoomFrame = (ISummaryZoomFrame) pres.Slides[1].Shapes[0];
    ISummaryZoomSectionCollection collection = zoomFrame.SummaryZoomCollection;
    ISummaryZoomSection selectedObject = collection.GetSummarySection(pres.Sections[2]);
    int idx = collection.IndexOf(selectedObject);
}
```

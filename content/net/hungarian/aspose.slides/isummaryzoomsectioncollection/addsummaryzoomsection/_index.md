## ISummaryZoomSectionCollection.AddSummaryZoomSection metódus

Új Summary Zoom Section objektumot hoz létre, és hozzáadja a gyűjteményhez

```csharp
public ISummaryZoomSection AddSummaryZoomSection(ISection section)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| section | ISection | Szekció egy új Summary Zoom Section elemhez [`ISection`](../../isection) |

### Visszatérési érték

Hozzáadott [`ISummaryZoomFrame`](../../isummaryzoomframe) elem

### Megjegyzés

Ha egy elem ehhez a szekcióhoz már létezik a gyűjteményben, akkor a meglévő elem kerül visszaadásra.

### Példák

A példa bemutatja, hogyan lehet index alapján lekérni a Summary Zoom Section elemet:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    ISummaryZoomFrame zoomFrame = (ISummaryZoomFrame) pres.Slides[1].Shapes[0];
    ISummaryZoomSectionCollection collection = zoomFrame.SummaryZoomCollection;
    ISummaryZoomSection newZoomSection = collection.AddSummaryZoomSection(pres.Sections[3]);
}
```

## OleObjectFrame.LinkPathRelative tulajdonság

Visszaadja a kapcsolt fájl relatív elérési útját, ha létezik, különben üres karakterláncot ad vissza. Csak olvasható String.

```csharp
public string LinkPathRelative { get; }
```

### Megjegyzések

A Ppt-prezentációkban egyes Ole-objektum-linkek relatív ábrázolással rendelkezhetnek.

### Példák

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.ppt"))
{
    IOleObjectFrame oleFrame = presentation.Slides[0].Shapes[0] as IOleObjectFrame;

    if (oleFrame != null)
    {
        Console.WriteLine("The relative path: " + oleFrame.LinkPathRelative);
    } 
}
```

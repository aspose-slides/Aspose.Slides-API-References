## FontsLoader.LoadExternalFonts metódus

További mappákat ad hozzá a betűtípusok kereséséhez.

```csharp
public static void LoadExternalFonts(string[] directories)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| directories | String[] | A további betűtípusok beolvasásához használt könyvtárak. |

### Példák

A következő példák bemutatják, hogyan lehet egyedi betűtípusokat betölteni .TTF fájlokból

```csharp
[C#]
// Az dokumentumok könyvtárának az elérési útja.
string dataDir = "C:\\";
// mappák a betűtípusok kereséséhez
String[] folders = new String[] { dataDir };
// Betölti az egyéni betűtípus könyvtár betűtípusait
FontsLoader.LoadExternalFonts(folders);
// Végrehajt némi munkát, és elvégzi a prezentáció/dia renderelését
using (Presentation presentation = new Presentation(dataDir + "DefaultFonts.pptx"))
{
    presentation.Save(dataDir + "NewFonts_out.pptx", SaveFormat.Pptx);
	// Betűtípus gyorsítótár törlése
	FontsLoader.ClearCache();
}
```

## IOleObjectFrame.SetEmbeddedData metódus

Beállítja az OLE beágyazott adatokra vonatkozó információkat.

```csharp
public void SetEmbeddedData(IOleEmbeddedDataInfo embeddedData)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| embeddedData | IOleEmbeddedDataInfo | Beágyazott adat [`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo) |

### Kivételek

| Kivétel | feltétel |
| --- | --- |
| ArgumentNullException | Amikor az embeddedData paraméter null. |

### Megjegyzés

Ez a metódus megváltoztatja az objektum tulajdonságait az új adatok tükrözéséhez, és a IsObjectLink jelzőt false értékre állítja, jelezve, hogy az OLE objektum beágyazott.

### Példák

Az alábbi példa bemutatja, hogyan lehet módosítani az OLE beágyazott adatot és annak típusát egy meglévő [`IOleObjectFrame`](../../ioleobjectframe) objektumnál

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    OleObjectFrame oof = pres.Slides[0].Shapes[0] as OleObjectFrame;
    if (oof != null)
    {
        IOleEmbeddedDataInfo newData = new OleEmbeddedDataInfo(File.ReadAllBytes("Picture.png"), "png");
        oof.SetEmbeddedData(newData);
    }
}
```

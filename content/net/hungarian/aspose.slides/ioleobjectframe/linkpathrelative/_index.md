## IOleObjectFrame.LinkPathRelative tulajdonság

A relatív elérési utat adja vissza egy csatolt fájlhoz, ha létezik, egyébként üres karakterláncot ad vissza. Csak olvasható String.

```csharp
public string LinkPathRelative { get; }
```

### Megjegyzések

A Ppt bemutatókban néhány Ole objektumhivatkozás relatív ábrázolással rendelkezhet.

### Példák

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.ppt"))
{
    IOleObjectFrame oleFrame = presentation.Slides[0].Shapes[0] as IOleObjectFrame;

    if (oleFrame != null)
    {
        Console.WriteLine("The relative path: " + oleFrame.LinkPathRelative);
    } 
}
```

## AudioFrame.RewindAudio tulajdonság

Meghatározza, hogy a hang automatikusan visszatekerődik-e a lejátszás után a kezdetre. Olvasás/írás Boolean.

```csharp
public bool RewindAudio { get; set; }
```

### Példák

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ISlide slide = pres.Slides[0];

    // Hangkeret hozzáadása
    IAudioFrame audioFrame = slide.Shapes.AddAudioFrameLinked(50, 50, 100, 100, "sampleaudio.wav");

    // Beállítja, hogy a hang a diákon végig lejátszódjon
    audioFrame.PlayAcrossSlides = true;

    // Beállítja, hogy a hang automatikusan visszatekerődjön a kezdetre a lejátszás után
    audioFrame.RewindAudio = true;

    pres.Save("AudioFrame_out.pptx", SaveFormat.Pptx);
}
```

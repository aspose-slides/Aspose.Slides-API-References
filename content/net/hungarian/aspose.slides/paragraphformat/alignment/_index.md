## ParagraphFormat.Alignment tulajdonság

Visszaadja vagy beállítja a szöveg igazítását egy bekezdésben öröklődés nélkül. Olvasás/írás [`TextAlignment`](../../textalignment).

```csharp
public TextAlignment Alignment { get; set; }
```

### Példák

Az alábbi minta kód bemutatja, hogyan lehet a szöveg bekezdéseit igazítani a PowerPoint előadásban.

```csharp
[C#]
// Létrehoz egy Presentation objektumot, amely egy PPTX fájlt képvisel
using (Presentation pres = new Presentation("ParagraphsAlignment.pptx"))
{
    // Az első dia elérése
    ISlide slide = pres.Slides[0];
    // A dia első és második helyfoglalójának elérése, és AutoShape-ként való típuskonverzió
    ITextFrame tf1 = ((IAutoShape)slide.Shapes[0]).TextFrame;
    ITextFrame tf2 = ((IAutoShape)slide.Shapes[1]).TextFrame;
    // A szöveg módosítása mindkét helyfoglalóban
    tf1.Text = "Center Align by Aspose";
    tf2.Text = "Center Align by Aspose";
    // Az első bekezdés lekérése a helyfoglalókból
    IParagraph para1 = tf1.Paragraphs[0];
    IParagraph para2 = tf2.Paragraphs[0];
    // A szöveg bekezdésének középre igazítása
    para1.ParagraphFormat.Alignment = TextAlignment.Center;
    para2.ParagraphFormat.Alignment = TextAlignment.Center;
    //A bemutató mentése PPTX fájlként
    pres.Save("Centeralign_out.pptx", SaveFormat.Pptx);
}
```

## IAudioFrame.RewindAudio tulajdonság

Meghatározza, hogy a hang automatikusan visszatekerődik-e a kezdetre a lejátszás után. Olvasás/írás Boolean.

```csharp
public bool RewindAudio { get; set; }
```

### Példák

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ISlide slide = pres.Slides[0];

    // Hangkeret hozzáadása
    IAudioFrame audioFrame = slide.Shapes.AddAudioFrameLinked(50, 50, 100, 100, "sampleaudio.wav");

    // Beállítja, hogy a hang a diákon keresztül játsszon
    audioFrame.PlayAcrossSlides = true;

    // Beállítja, hogy a hang lejátszás után automatikusan visszategye a kezdőpontra
    audioFrame.RewindAudio = true;

    pres.Save("AudioFrame_out.pptx", SaveFormat.Pptx);
}
```

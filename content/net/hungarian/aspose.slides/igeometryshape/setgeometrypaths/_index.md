## IGeometryShape.SetGeometryPaths metódus

Frissíti a forma geometriáját a [`IGeometryPath`](../../igeometrypath) tömbből. A koordinátáknak a forma bal felső sarkához képest relatívnak kell lenniük. A forma típusát ([`ShapeType`](../shapetype)) Custom-ra változtatja.

```csharp
public void SetGeometryPaths(IGeometryPath[] geometryPaths)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| geometryPaths | IGeometryPath[] | Geometriai útvonalak tömbje |

### Kivételek

| kivétel | feltétel |
| --- | --- |
| ArgumentException | Útvonal nem található |
| ArgumentException | Üres útvonal |

### Példák

Példa:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    GeometryShape shape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 200, 100) as GeometryShape;

    GeometryPath geometryPath = shape.GetGeometryPaths()[0];

    geometryPath.LineTo(100, 50, 1);
    geometryPath.LineTo(100, 50, 4);

    shape.SetGeometryPath(geometryPath);

    pres.Save("output.pptx", SaveFormat.Pptx);
}
```

## IBasePortionFormat.SpellCheck tulajdonság

Lekérdezi vagy beállítja azt az értéket, amely azt jelzi, hogy a helyesírás-ellenőrzés engedélyezve van-e a szövegrészhez. Ha ez a tulajdonság false értékre van állítva, a szövegelemek helyesírás-ellenőrzése le van tiltva. Ha true értékre van állítva, a helyesírás-ellenőrzés engedélyezett. Az alapértelmezett érték a `false`.

```csharp
public bool SpellCheck { get; set; }
```

### Példák

A következő példa a SpellCheck jelző engedélyezését mutatja be a prezentáció mentése előtt:

```csharp
[C#]
using (var pres = new Presentation("input.pptx"))
{
    // Az első dia első alakzatának első szövegrésszéhez fér hozzá
    var portion = ((AutoShape)pres.Slides[0].Shapes[0]).TextFrame.Paragraphs[0].Portions[0];
    // Engedélyezi a helyesírás-ellenőrzést ennek a szövegrésznek
    portion.PortionFormat.SpellCheck = true;
    // Mentse a módosított prezentációt
    pres.Save("output-with-spellcheck.pptx", SaveFormat.Pptx);
}
```

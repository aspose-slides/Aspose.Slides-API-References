## BulletFormat.Height tulajdonság

Visszaadja vagy beállítja a felsorolás magasságát egy olyan bekezdésben, amely nem örököl. A float.NaN érték azt határozza meg, hogy a felsorolás a bekezdés első részéből örökölje a magasságot. Olvasás/írás Single.

```csharp
public float Height { get; set; }
```

### Megjegyzés

A negatív magasságérték azt jelenti, hogy a magasság pontban van megadva, míg a pozitív érték a magasságot a környező szöveg százalékában adja meg.

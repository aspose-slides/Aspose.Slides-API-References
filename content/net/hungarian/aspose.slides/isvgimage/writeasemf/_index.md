## ISvgImage.WriteAsEmf metódus

Elmenti az SVG képet EMF fájlként.

public void WriteAsEmf(Stream stream)

| Paraméter | Típus | Leírás |

### Példák

Az alábbi példa bemutatja, hogyan lehet az SVG képet meta-fájlba menteni.

```csharp
[C#]
// Létrehozza az új SVG képet
ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));

// Elmenti az SVG képet meta-fájlként
using (var fileStream = System.IO.File.OpenWrite("SvgAsEmf.emf"))
    svgImage.WriteAsEmf(fileStream);
```

Ez a mintapélda bemutatja, hogyan lehet az SVG képet meta-fájlként hozzáadni a bemutató képgyűjteményéhez.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // Létrehozza az új SVG képet
    ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));
    using (var memStream = new MemoryStream())
    {
        // Elmenti az SVG képet meta-fájlként
        svgImage.WriteAsEmf(memStream);
        // Hozzáadja a meta-fájlt a képgyűjteményhez
        pres.Images.AddImage(memStream.ToArray());
    }
}
```

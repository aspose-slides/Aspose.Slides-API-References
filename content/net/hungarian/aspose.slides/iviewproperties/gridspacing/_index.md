## IViewProperties.GridSpacing tulajdonság

Visszaadja vagy beállítja a rácstávolságot, amelyet a prezentációs dokumentum alatti rácshoz kell használni, pontokban. Olvasás/írás Single.

```csharp
public float GridSpacing { get; set; }
```

### Megjegyzés

A rácstávolság értékének pozitív számnak kell lennie. A tipikus értéktartomány 1 mm (2.8349607 pont) és 2 hüvelyk (144 pont) között.

### Példák

Az alábbi minta kód bemutatja, hogyan lehet módosítani a rácstávolságot egy PowerPoint prezentációban.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

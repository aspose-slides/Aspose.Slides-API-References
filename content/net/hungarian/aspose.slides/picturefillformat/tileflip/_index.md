## PictureFillFormat.TileFlip tulajdonság

A textúra csempét vízszintes, függőleges vagy mindkét tengelye köré forgatja. Olvasás/írás [`TileFlip`](../../tileflip).

```csharp
public TileFlip TileFlip { get; set; }
```

### Megjegyzések

Alapértelmezett érték a NoFlip.

### Példák

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Lekéri a forma képkitöltési formátumát
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Beállítja a képkitöltés módját Tile-re
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Megfordítja a textúra csempét a függőleges tengelye körül.
    pictureFillFormat.TileFlip = TileFlip.FlipY;
}
```

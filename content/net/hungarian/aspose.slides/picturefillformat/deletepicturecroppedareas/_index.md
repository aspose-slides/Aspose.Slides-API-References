## PictureFillFormat.DeletePictureCroppedAreas metódus

Törli a kitöltő Picture levágott területeit.

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### Visszatérési érték

Levágott kép vagy eredeti kép, ha a vágás nem szükséges.

### Megjegyzés

Ez a metódus WMF/EMF metafájlokat raszteres PNG képpé alakítja vágás közben.

### Példák

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // A PictureFrame lekérdezése
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // A PictureFrame kép levágott területeinek törlése
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

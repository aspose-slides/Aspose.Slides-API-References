## PictureFillFormat.TileOffsetY tulajdonság

Visszaadja vagy beállítja a textúra függőleges eltolását az alakzat kiindulási pontjától pontban. Pozitív érték esetén a textúra lefelé mozog, negatív érték esetén pedig felfelé. Olvasás/írás Single.

```csharp
public float TileOffsetY { get; set; }
```

### Példák

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // A forma kép kitöltési formátumát kapja meg
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // A kép kitöltési módot Tile-re állítja
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // A textúra függőleges eltolását -50 pont értékre állítja
    pictureFillFormat.TileOffsetY = -50;
}
```

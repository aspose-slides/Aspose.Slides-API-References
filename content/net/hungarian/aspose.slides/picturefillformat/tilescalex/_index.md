## PictureFillFormat.TileScaleX tulajdonság

Visszaadja vagy beállítja a textúra kitöltés horizontális méretarányát százalékban. Olvasás/írás Single.

```csharp
public float TileScaleX { get; set; }
```

### Példák

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Lekéri a forma képkitöltés formátumát
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Beállítja a képkitöltési módot Tile-re
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Beállítja a textúra horizontális méretarányát 120 százalékra
    pictureFillFormat.TileScaleX = 120;
}
```

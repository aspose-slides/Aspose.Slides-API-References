## PictureFillFormat.TileAlignment property

Visszaadja vagy beállítja, hogy a textúra hogyan van igazítva az alakzatban. Ez a beállítás szabályozza a textúraminta kiindulási pontját és azt, hogy hogyan ismétlődik az alakzatban. Olvasás/írás [`RectangleAlignment`](../../rectanglealignment).

```csharp
public RectangleAlignment TileAlignment { get; set; }
```

### Megjegyzés

Az alapértelmezett a TopLeft.

### Példák

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Lekéri az alakzat kép kitöltési formátumát
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Beállítja a kép kitöltési módot Tile-re
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Beállítja a csempe igazítását jobb alsó sarokba
    pictureFillFormat.TileAlignment = RectangleAlignment.BottomRight;
}
```

## MasterSlideCollection.InsertClone metódus

Beszúr egy példányt egy megadott mesterdiáról a gyűjtemény megadott pozíciójába. A kapcsolt elrendezési diák is másolásra kerül.

```csharp
public IMasterSlide InsertClone(int index, IMasterSlide sourceMaster)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| index | Int32 | Az új dia indexe. |
| sourceMaster | IMasterSlide | A klónozandó dia. |

### Visszatérési érték

A beszúrt mesterdia.

### Példák

Az alábbi példa bemutatja, hogyan klónozhat egy mesterdiát egy másik PowerPoint bemutatóban.

```csharp
[C#]
// Példányosítja a Presentation osztályt a forrás prezentáció fájl betöltéséhez
using (Presentation srcPres = new Presentation("CloneToAnotherPresentationWithMaster.pptx"))
{
    // Példányosítja a Presentation osztályt a cél prezentációhoz (ahová a dia klónozandó)
    using (Presentation destPres = new Presentation())
    {
        // Példányosítja az ISlide-t a forrás prezentáció diák gyűjteményéből együtt
        // Mesterdia
        ISlide SourceSlide = srcPres.Slides[0];
        IMasterSlide SourceMaster = SourceSlide.LayoutSlide.MasterSlide;
		// A cél prezentáció mesterdiai lekérése
        IMasterSlideCollection masters = destPres.Masters;
        // A kívánt mesterdia klónozása a forrás prezentációból a mestergyűjteménybe a
        // Cél prezentációban
        IMasterSlide iSlide = masters.AddClone(SourceMaster);
        // Diagyűjtemény a cél prezentációban
        ISlideCollection slds = destPres.Slides;
		// A forrás dia klónozása a cél diagyűjteménybe.
        slds.AddClone(SourceSlide, iSlide, true);
        // A cél prezentáció mentése lemezre
        destPres.Save("CloneToAnotherPresentationWithMaster_out.pptx", SaveFormat.Pptx);
    }
}
```

## TextFrame.SplitTextByColumns metódus

Felosztja a(z) [`ITextFrame`](../../itextframe) szövegtartalmát egy karakterlánc tömbjébe, ahol minden elem a keretben lévő különálló szövegoszlopnak felel meg.

```csharp
public string[] SplitTextByColumns()
```

### Visszatérési érték

Egy karakterlánc tömb, ahol minden karakterlánc egy adott oszlop szövegtartalmát képviseli a(z) [`ITextFrame`](../../itextframe)-ben.

### Megjegyzés

Ha a szövegkeret nem tartalmaz több oszlopot, a visszaadott tömb egyetlen elemet tartalmaz a teljes szöveggel. Az üres oszlopok a tömbben üres karakterláncokként jelennek meg.

### Példák

Az alábbi példa bemutatja, hogyan kell használni a `SplitTextByColumns` metódust:

```csharp
using (Presentation pres = new Presentation("example.pptx"))
{
    // Az első alakzatot a dián lekérjük és ITextFrame-re konvertáljuk
    ITextFrame textFrame = pres.Slides[0].Shapes[0] as ITextFrame;
    // A szövegkeret tartalmát oszlopokra osztja
    string[] columnsText = textFrame.SplitTextByColumns();
    // Kiírja minden oszlop szövegét a konzolra
    foreach (string column in columnsText)
        Console.WriteLine(column);
}
```

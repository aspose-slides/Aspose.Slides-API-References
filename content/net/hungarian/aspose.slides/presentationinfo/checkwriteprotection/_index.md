## PresentationInfo.CheckWriteProtection metódus

Ellenőrzi, hogy a módosításhoz megadott jelszó helyes-e egy írásvédett bemutató esetén.

```csharp
public bool CheckWriteProtection(string password)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| password | String | A ellenőrzendő jelszó. |

### Visszatérési érték

Igaz, ha a bemutató írásvédett és a jelszó helyes. Egyébként hamis.

### Kivételek

| kivétel | feltétel |
| --- | --- |
| InvalidOperationException |  |

### Megjegyzések

1. Ellenőriznie kell a [`IsWriteProtected`](../iswriteprotected) tulajdonságot, mielőtt meghívja ezt a metódust. 2. Ha a jelszó null vagy üres, ez a metódus hamisat ad vissza.

### Példák

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
if (info.IsWriteProtected == NullableBool.True)
{
    bool isWriteProtectedByPassword = info.CheckWriteProtection("my_password");
}
```

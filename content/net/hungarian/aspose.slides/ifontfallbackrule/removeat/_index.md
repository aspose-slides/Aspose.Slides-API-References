## IFontFallBackRule.RemoveAt metódus

Eltávolítja a FallBack betűtípust a lista megadott indexén.

```csharp
public void RemoveAt(int index)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| index | Int32 | A betűtípus eltávolításához használt nulla alapú index. |

### Példák

```csharp
[C#]
// Hozzon létre egy szabályt, amely egy betűtípus listát tartalmaz.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");
```

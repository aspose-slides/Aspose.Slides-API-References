## IFontFallBackRule.Remove metódus

Eltávolítja a listából egy adott FallBack betűtípus első előfordulását.

```csharp
public void Remove(string fontName)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| fontName | String | A betűtípus neve, amelyet el kell távolítani a listáról. |

### Példák

```csharp
[C#]
// Hozzon létre egy szabályt, amely betűkészlet-listát tartalmaz.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

//Távolítsa el a Tahoma betűtípust a listáról
newRule.Remove ("Tahoma");
```

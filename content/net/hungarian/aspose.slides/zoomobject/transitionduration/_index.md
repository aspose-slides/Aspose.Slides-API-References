## ZoomObject.TransitionDuration tulajdonság

Lekéri vagy beállítja a Zoom és a dia közötti átvitel időtartamát. Olvasás/írás Single. Alapértelmezett érték: 1.0f

```csharp
public float TransitionDuration { get; set; }
```

### Megjegyzés

Ha nincs megadva (TransitionDur = 0), akkor a cél dia átmenetét és a hozzá tartozó időzítéseket használja.

### Példák

Az alábbi példa bemutatja, hogyan lehet megváltoztatni a Zoom és a dia közötti átvitel időtartamát:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
    zoomFrame.TransitionDuration = 2.5f;
}
```

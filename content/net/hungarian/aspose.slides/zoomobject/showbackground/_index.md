## ZoomObject.ShowBackground tulajdonság

Értéket kap vagy állít be, amely meghatározza, hogy a Zoom használja-e a cél dián lévő háttérképet. Olvasás/írás Boolean. Alapértelmezett érték: true

```csharp
public bool ShowBackground { get; set; }
```

### Példák

a példa bemutatja egy Zoom objektum képének háttér eltávolítását:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
    zoomFrame.ShowBackground = false;
}
```

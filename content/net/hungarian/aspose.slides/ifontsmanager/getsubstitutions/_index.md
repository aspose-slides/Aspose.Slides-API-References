## GetSubstitutions() {#getsubstitutions}

Lekéri az információt a betűtípusokról, amelyek a bemutató renderelése során helyettesítésre kerülnek.

```csharp
public IEnumerable<FontSubstitutionInfo> GetSubstitutions()
```

### Visszatérési érték

A betűtípus helyettesítések teljes gyűjteménye [`FontSubstitutionInfo`](../../fontsubstitutioninfo).

### Példák

```csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    foreach (var fontSubstitution in pres.FontsManager.GetSubstitutions())
    {
        Console.WriteLine("{0} -> {1}", fontSubstitution.OriginalFontName, fontSubstitution.SubstitutedFontName);
    }
}        
```

## GetSubstitutions(int[]) {#getsubstitutions_1}

Lekéri az információt a betűtípusokról, amelyek a megadott diák renderelése során helyettesítésre kerülnek.

```csharp
public IEnumerable<FontSubstitutionInfo> GetSubstitutions(int[] slides)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| slides | Int32[] | Egy tömb a dia indexekkel, amelyekhez a betűtípus helyettesítési információkat le kell kérni, 1-től kezdődően. |

### Visszatérési érték

A megadott diákra vonatkozó összes betűtípus helyettesítés gyűjteménye ([`FontSubstitutionInfo`](../../fontsubstitutioninfo)).

### Példák

```csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    int[] targetSlides = { 1, 2, 5 };
    foreach (var fontSubstitution in pres.FontsManager.GetSubstitutions(targetSlides))
    {
        Console.WriteLine("{0} -> {1}", fontSubstitution.OriginalFontName, fontSubstitution.SubstitutedFontName);
    }
}
```

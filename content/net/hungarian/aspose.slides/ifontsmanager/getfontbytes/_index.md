## IFontsManager.GetFontBytes metódus

Visszaadja a bájttömböt, amely a megadott betűtípus-stílus és betűtípus-adat betűtípusadatait tartalmazza.

```csharp
public byte[] GetFontBytes(IFontData fontData, FontStyleType fontStyle)
```

| Parameter | Type | Description |
| --- | --- | --- |
| fontData | IFontData | A betűtípus-adat objektum, amely a betűtípusról tartalmaz információkat [`IFontData`](../../ifontdata). |
| fontStyle | FontStyleType | A betűtípus stílusa, amelyhez az adatot le kell kérni [`FontStyleType`](../../fontstyletype). |

### Visszatérési érték

A bájttömb, amely a megadott betűtípus-stílus betűtípusadatait tartalmazza. Ha a betűtípus-adat vagy a stílus nem található, null értékkel tér vissza.

### Példák

```csharp
[C#]
using (Presentation pres = new Presentation ("Presentation.pptx"))
{
    // Lekéri az előadásban használt összes betűtípust
    IFontData[] fonts = pres.FontsManager.GetFonts();

    // Lekéri a prezentációban szereplő első betűtípus normál stílusát reprezentáló bájttömböt
    byte[] bytes = pres.FontsManager.GetFontBytes(fonts[0], FontStyleType.Regular);
}
```

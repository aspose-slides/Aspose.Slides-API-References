## ILayoutPlaceholderManager.AddVerticalTextPlaceholder method

Új helyőrző alakzatot ad a layout diához, amely függőleges irányban tartja a szövegtartalmat.

```csharp
public IAutoShape AddVerticalTextPlaceholder(float x, float y, float width, float height)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| x | Single | Az új helyőrző alakzat X koordinátája. |
| y | Single | Az új helyőrző alakzat Y koordinátája. |
| width | Single | Az új helyőrző alakzat szélessége. |
| height | Single | Az új helyőrző alakzat magassága. |

### Visszatérési érték

Létrehozott [`IAutoShape`](../../iautoshape) egy Text (Vertical) helyőrzővel.

### Példák

A következő példa bemutatja, hogyan lehet hozzáadni a Text (Vertical) helyőrző alakzatot a layout diához.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalTextPlaceholder(20, 20, 300, 500);
}
```

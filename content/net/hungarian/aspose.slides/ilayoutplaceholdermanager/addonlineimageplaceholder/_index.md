## ILayoutPlaceholderManager.AddOnlineImagePlaceholder metódus

Új helyőrző alakzatot ad hozzá a elrendezési diára, amely online képet tartalmaz.

```csharp
public IAutoShape AddOnlineImagePlaceholder(float x, float y, float width, float height)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| x | Single | Az X koordináta az új helyőrző alakzatra. |
| y | Single | Az Y koordináta az új helyőrző alakzatra. |
| width | Single | Az új helyőrző alakzat szélessége. |
| height | Single | Az új helyőrző alakzat magassága. |

### Visszatérési érték

Létrehozva [`IAutoShape`](../../iautoshape) egy online kép helyőrzővel.

### Példák

Az alábbi példa bemutatja, hogyan adhatja hozzá az online kép helyőrző alakzatot az elrendezési diához.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddOnlineImagePlaceholder(20, 20, 200, 200);
}
```

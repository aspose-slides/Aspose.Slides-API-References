## ILayoutPlaceholderManager.AddTablePlaceholder metódus

Új helyőrző alakzatot ad a elrendezés diához, amely egy táblázatot tartalmaz.

```csharp
public IAutoShape AddTablePlaceholder(float x, float y, float width, float height)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| x | Single | Az X koordináta az új helyőrző alakzatnál. |
| y | Single | Az Y koordináta az új helyőrző alakzatnál. |
| width | Single | Az új helyőrző alakzat szélessége. |
| height | Single | Az új helyőrző alakzat magassága. |

### Visszatérési érték

Létrehozva [`IAutoShape`](../../iautoshape) egy Table helyőrzővel.

### Példák

A következő példa azt mutatja, hogyan lehet hozzáadni a Table helyőrző alakzatot a elrendezés diára.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTablePlaceholder(20, 20, 500, 200);
}
```

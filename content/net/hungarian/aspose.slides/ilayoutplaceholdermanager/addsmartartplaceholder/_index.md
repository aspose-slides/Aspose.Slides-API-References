## ILayoutPlaceholderManager.AddSmartArtPlaceholder metódus

Új helykitöltő alakzatot ad a makett diához, amely egy SmartArt diagramot tartalmaz.

```csharp
public IAutoShape AddSmartArtPlaceholder(float x, float y, float width, float height)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| x | Single | Az új helykitöltő alakzat X koordinátája. |
| y | Single | Az új helykitöltő alakzat Y koordinátája. |
| width | Single | Az új helykitöltő alakzat szélessége. |
| height | Single | Az új helykitöltő alakzat magassága. |

### Visszatérési érték

Létrehozott [`IAutoShape`](../../iautoshape) egy SmartArt helykitöltővel.

### Példák

Az alábbi példa bemutatja, hogyan adható hozzá a SmartArt helykitöltő alakzat a makett diához.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddSmartArtPlaceholder(20, 20, 200, 200);
}
```

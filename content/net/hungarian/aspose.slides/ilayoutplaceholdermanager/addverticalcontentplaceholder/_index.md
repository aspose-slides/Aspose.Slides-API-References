## ILayoutPlaceholderManager.AddVerticalContentPlaceholder metódus

Új helyfoglaló alakzatot ad az elrendezési diához, amely függőleges irányban tartalmazhat képet, táblázatot, médiát vagy szöveget.

```csharp
public IAutoShape AddVerticalContentPlaceholder(float x, float y, float width, float height)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| x | Single | Az új helyfoglaló alakzat X koordinátája. |
| y | Single | Az új helyfoglaló alakzat Y koordinátája. |
| width | Single | Az új helyfoglaló alakzat szélessége. |
| height | Single | Az új helyfoglaló alakzat magassága. |

### Visszatérési érték

Létrehozva [`IAutoShape`](../../iautoshape) egy Content (Vertical) helyfoglalóval.

### Példák

A következő példa bemutatja, hogyan lehet hozzáadni a Content (Vertical) helyfoglaló alakzatot az elrendezési diához.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalContentPlaceholder(20, 20, 300, 500);
}
```

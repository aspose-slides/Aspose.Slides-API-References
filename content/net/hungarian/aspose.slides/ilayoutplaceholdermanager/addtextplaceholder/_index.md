## ILayoutPlaceholderManager.AddTextPlaceholder metódus

Új helyőrző alakzatot ad a elrendezési diához, amely szövegtartalmat tartalmaz.

```csharp
public IAutoShape AddTextPlaceholder(float x, float y, float width, float height)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| x | Single | Az új helyőrző alakzat X koordinátája. |
| y | Single | Az új helyőrző alakzat Y koordinátája. |
| width | Single | Az új helyőrző alakzat szélessége. |
| height | Single | Az új helyőrző alakzat magassága. |

### Visszatérési érték

Létrehozott [`IAutoShape`](../../iautoshape) egy Szöveg helyőrzővel.

### Példák

A következő példa bemutatja, hogyan adhatjuk hozzá a Szöveg helyőrző alakzatot az elrendezési diához.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTextPlaceholder(20, 20, 500, 300);
}
```

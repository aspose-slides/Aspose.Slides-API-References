## ILayoutPlaceholderManager.AddChartPlaceholder metódus

Új helyőrző alakzatot ad hozzá a elrendezés diájához, hogy diagramot helyezzen el benne.

```csharp
public IAutoShape AddChartPlaceholder(float x, float y, float width, float height)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| x | Single | Az új helyőrző alakzat X koordinátája. |
| y | Single | Az új helyőrző alakzat Y koordinátája. |
| width | Single | Az új helyőrző alakzat szélessége. |
| height | Single | Az új helyőrző alakzat magassága. |

### Visszatérési érték

Létrehozva [`IAutoShape`](../../iautoshape) egy Chart helyőrzővel.

### Példák

A következő példa bemutatja, hogyan kell hozzáadni a Chart helyőrző alakzatot az elrendezés diájához.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddChartPlaceholder(20, 20, 200, 200);
}
```

## ILayoutPlaceholderManager.AddContentPlaceholder metódus

Új helykitöltő alakzatot ad hozzá az elrendezés diájához, amely tartalmat (például képet, táblázatot, médiát vagy szöveget) tárol.

```csharp
public IAutoShape AddContentPlaceholder(float x, float y, float width, float height)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| x | Single | Az új helykitöltő alakzat X koordinátája. |
| y | Single | Az új helykitöltő alakzat Y koordinátája. |
| width | Single | Az új helykitöltő alakzat szélessége. |
| height | Single | Az új helykitöltő alakzat magassága. |

### Visszatérési érték

Létrehozva [`IAutoShape`](../../iautoshape) egy Content helykitöltővel.

### Példák

Az alábbi példa bemutatja, hogyan adhatja hozzá a Content helykitöltő alakzatot az elrendezés diájához.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddContentPlaceholder(20, 20, 500, 300);
}
```

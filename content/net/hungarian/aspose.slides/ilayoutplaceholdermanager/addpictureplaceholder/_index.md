## ILayoutPlaceholderManager.AddPicturePlaceholder metódus

Új helyőrző alakzatot ad hozzá a layout diához, amely képet tartalmaz.

```csharp
public IAutoShape AddPicturePlaceholder(float x, float y, float width, float height)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| x | Single | Az új helyőrző alakzat X koordinátája. |
| y | Single | Az új helyőrző alakzat Y koordinátája. |
| width | Single | Az új helyőrző alakzat szélessége. |
| height | Single | Az új helyőrző alakzat magassága. |

### Visszatérési érték

Létrehozott [`IAutoShape`](../../iautoshape) egy Picture helyőrzővel.

### Példák

A következő példa bemutatja, hogyan lehet hozzáadni a Picture helyőrző alakzatot a layout diára.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddPicturePlaceholder(20, 20, 200, 200);
}
```

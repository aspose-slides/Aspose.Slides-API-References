## BasePortionFormat.SpellCheck tulajdonság

Értéket ad vissza vagy állít be, amely megadja, hogy a helyesírás-ellenőrzés engedélyezve van-e a szövegrészben. Ha ez a tulajdonság hamis értékre van beállítva, a szövegelemek helyesírás-ellenőrzése le van tiltva. Ha igaz értékre van beállítva, a helyesírás-ellenőrzés engedélyezett. Az alapértelmezett érték a `false`.

```csharp
public bool SpellCheck { get; set; }
```

### Példák

A következő példa bemutatja a SpellCheck jelző engedélyezését a prezentáció mentése előtt:

```csharp
[C#]
using (var pres = new Presentation("input.pptx"))
{
    // Az első dián az első alakzat első szövegdarabjának elérése
    var portion = ((AutoShape)pres.Slides[0].Shapes[0]).TextFrame.Paragraphs[0].Portions[0];
    // Engedélyezi a helyesírás-ellenőrzést ehhez a szövegrészhez
    portion.PortionFormat.SpellCheck = true;
    // Mentse a módosított prezentációt
    pres.Save("output-with-spellcheck.pptx", SaveFormat.Pptx);
}
```

## SetLicense(string) {#setlicense_1}

Licenceli a komponenst.

```csharp
public void SetLicense(string licenseName)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| licenseName | String | Lehet teljes vagy rövid fájlnév vagy egy beágyazott erőforrás neve. Üres karakterlánc használatával átvált értékelési módra. |

### Megjegyzés

A licencet a következő helyeken keresi:

1. Kifejezett útvonal.
2. A komponens összeállításának mappája.
3. A kliens hívó összeállításának mappája.
4. A belépési összeállítás mappája.
5. Egy beágyazott erőforrás a kliens hívó összeállításában.

**Megjegyzés:** A .NET Compact Framework esetén a licencet csak a következő helyeken keresi:

1. Kifejezett útvonal.
2. Egy beágyazott erőforrás a kliens hívó összeállításában.

### Példák

Ebben a példában megkísérlik megtalálni a MyLicense.lic nevű licencfájlt a komponenst tartalmazó mappában, a hívó összeállítást tartalmazó mappában, a belépési összeállítás mappájában, majd a hívó összeállítás beágyazott erőforrásai között.

```csharp
[C#]

License license = new License();
license.SetLicense("MyLicense.lic");


[Visual Basic]

Dim license As License = New License
license.SetLicense("MyLicense.lic")
```

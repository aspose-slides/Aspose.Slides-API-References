## IFonts.RemoveScriptFont metódus

Eltávolítja a betűkészlet-beállítást, amely egy adott script címkéhez kapcsolódik a téma betűkészlet-gyűjteményéből.

```csharp
public void RemoveScriptFont(string script)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| script | String | A BCP-47 script kód, amelynek a betűkészlet-beállítását el kell távolítani. |

### Példák

Ez a példa bemutatja, hogyan lehet eltávolítani a héber script betűtérképét:

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.RemoveScriptFont("Hebr");
```

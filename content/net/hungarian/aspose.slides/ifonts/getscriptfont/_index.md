## IFonts.GetScriptFont metódus

Lekéri a prezentáció témájában egy adott írásrendszer címkéhez társított betűtípus nevét.

```csharp
public string GetScriptFont(string script)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| script | String | A BCP-47 írásrendszer kód (pl. "Latn", "Cyrl", "Jpan"), amely egy írásrendszer azonosítására szolgál. |

### Visszatérési érték

A megadott írásrendszerhez használt betűtípus neve, vagy `null` ha az írásrendszer nincs definiálva.

### Példák

Ez a példa bemutatja, hogyan lehet lekérni a prezentáció témájában a cirill írásrendszerhez rendelt betűtípust.

```csharp
[C#]
string font = presentation.MasterTheme.FontScheme.Major.GetScriptFont("Cyrl");
Console.WriteLine("Font for Cyrillic script: " + font);
```

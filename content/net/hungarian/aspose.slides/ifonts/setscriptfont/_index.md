## IFonts.SetScriptFont metódus

Egy betűtípus nevet rendel egy adott script címkéhez, amely meghatározza, hogy a szkript szövege hogyan lesz megjelenítve a prezentációban.

```csharp
public void SetScriptFont(string script, string fontName)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| script | String | A BCP-47 íráskód (pl. "Arab", "Hebr", "Hans"), amely az írásrendszert azonosítja. |
| fontName | String | A megadott íráshoz hozzárendelendő betűtípus neve. |

### Példák

Ez a példa azt mutatja be, hogyan állítható be az arab írás betűtípusa "Segoe UI" értékre:

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segoe UI");
```

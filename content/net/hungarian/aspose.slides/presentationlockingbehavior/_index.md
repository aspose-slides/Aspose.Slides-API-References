## PresentationLockingBehavior enumeráció

A [`IPresentation`](../ipresentation) forrás (fájl vagy Stream) betöltése és egy [`IPresentation`](../ipresentation) példányon való munka közbeni kezelésének viselkedését jelöli.

```csharp
public enum PresentationLockingBehavior
```

### Értékek

| Név | Érték | Leírás |
| --- | --- | --- |
| LoadAndRelease | `0` | A forrás csak a [`IPresentation`](../ipresentation) konstruktor végrehajtásának ideje alatt lesz zárolva. Ha a [`IsTemporaryFilesAllowed`](../iblobmanagementoptions/istemporaryfilesallowed) értéke false, akkor az összes BLOB memóriába lesz betöltve. Ellenkező esetben más módszerek, például ideiglenes fájlok használata lehetséges. Ez a viselkedés lassabb a KeepLocked-nél, és ha lehetséges a forrás tulajdonjogát átadni a [`IPresentation`](../ipresentation)-nek, ajánlott a KeepLocked használata. |
| KeepLocked | `1` | A forrás a [`IPresentation`](../ipresentation) példány teljes élettartama során lesz zárolva, amíg el nem lesz dobva. A [`IsTemporaryFilesAllowed`](../iblobmanagementoptions/istemporaryfilesallowed) értékét true-ra kell állítani e viselkedés használatához, ellenkező esetben kivétel keletkezik. Ez a viselkedés ajánlott, gyorsabb és kevesebb memóriát használ, mint a LoadAndRelease. |

### Megjegyzések

A forrás a [`IPresentation`](../ipresentation) konstruktorának paramétere. Az alábbi példában a forrás a "pres.pptx" fájl:

```csharp
LoadOptions loadOptions = new LoadOptions { 
  BlobManagementOptions = { PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked } };
using (IPresentation pres = new Presentation("pres.pptx", loadOptions)) { }
```

Ebben a példában a forrás ("pres.pptx" fájl) a [`IPresentation`](../ipresentation) példány élettartama alatt lesz zárolva, vagyis a másik folyamat nem módosíthatja vagy törölheti.

## LayoutPlaceholderManager.AddMediaPlaceholder metódus

Új helyőrző alakzatot ad a elrendezés-diára, hogy egy médiaobjektumot tartalmazzon.

```csharp
public IAutoShape AddMediaPlaceholder(float x, float y, float width, float height)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| x | Single | Az új helyőrző alakzat X koordinátája. |
| y | Single | Az új helyőrző alakzat Y koordinátája. |
| width | Single | Az új helyőrző alakzat szélessége. |
| height | Single | Az új helyőrző alakzat magassága. |

### Visszatérési érték

Létrehozott [`IAutoShape`](../../iautoshape) egy Média helyőrzővel.

### Példák

A következő példa bemutatja, hogyan adható hozzá a Média helyőrző alakzat az elrendezés-diához.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddMediaPlaceholder(20, 20, 200, 200);
}
```

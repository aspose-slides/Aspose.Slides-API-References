## LayoutPlaceholderManager.AddTextPlaceholder metódus

Új helykitöltő alakzatot ad a elrendezésdiához, hogy szövegtartalmat tároljon.

```csharp
public IAutoShape AddTextPlaceholder(float x, float y, float width, float height)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| x | Single | Az új helykitöltő alakzat X koordinátája. |
| y | Single | Az új helykitöltő alakzat Y koordinátája. |
| width | Single | Az új helykitöltő alakzat szélessége. |
| height | Single | Az új helykitöltő alakzat magassága. |

### Visszatérési érték

Létrehozva [`IAutoShape`](../../iautoshape) egy Text placeholder-rel.

### Példák

A következő példa bemutatja, hogyan lehet hozzáadni a Text placeholder alakzatot az elrendezésdiához.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTextPlaceholder(20, 20, 500, 300);
}
```

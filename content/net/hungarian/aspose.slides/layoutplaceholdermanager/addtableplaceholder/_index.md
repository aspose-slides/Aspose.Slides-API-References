## LayoutPlaceholderManager.AddTablePlaceholder metódus

Új helyőrző alakzatot ad a layout diához, amely egy táblát tartalmaz.

```csharp
public IAutoShape AddTablePlaceholder(float x, float y, float width, float height)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| x | Single | Az új helyőrző alakzat X koordinátája. |
| y | Single | Az új helyőrző alakzat Y koordinátája. |
| width | Single | Az új helyőrző alakzat szélessége. |
| height | Single | Az új helyőrző alakzat magassága. |

### Visszatérési érték

Létrehozott [`IAutoShape`](../../iautoshape) egy Table helyőrzővel.

### Példák

Az alábbi példa bemutatja, hogyan adhatja hozzá a Table helyőrző alakzatot a layout diához.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTablePlaceholder(20, 20, 500, 200);
}
```

## LayoutPlaceholderManager.AddOnlineImagePlaceholder method

Új helyfoglaló alakzatot ad hozzá az elrendezési diára egy online kép tárolásához.

```csharp
public IAutoShape AddOnlineImagePlaceholder(float x, float y, float width, float height)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| x | Single | Az új helyfoglaló alakzat X koordinátája. |
| y | Single | Az új helyfoglaló alakzat Y koordinátája. |
| width | Single | Az új helyfoglaló alakzat szélessége. |
| height | Single | Az új helyfoglaló alakzat magassága. |

### Visszatérési érték

Létrehozott [`IAutoShape`](../../iautoshape) egy online kép helyfoglalóval.

### Példák

Az alábbi példa bemutatja, hogyan adhatja hozzá az online kép helyfoglaló alakzatot az elrendezési diához.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddOnlineImagePlaceholder(20, 20, 200, 200);
}
```

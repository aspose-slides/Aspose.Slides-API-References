## LayoutPlaceholderManager.AddSmartArtPlaceholder metódus

Új helyőrző alakzatot ad hozzá a elrendezési diára egy SmartArt diagram megjelenítéséhez.

```csharp
public IAutoShape AddSmartArtPlaceholder(float x, float y, float width, float height)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| x | Single | Az új helyőrző alakzat X koordinátája. |
| y | Single | Az új helyőrző alakzat Y koordinátája. |
| width | Single | Az új helyőrző alakzat szélessége. |
| height | Single | Az új helyőrző alakzat magassága. |

### Visszatérési érték

Létrehozva [`IAutoShape`](../../iautoshape) egy SmartArt helyőrzővel.

### Példák

A következő példa bemutatja, hogyan adhat hozzá egy SmartArt helyőrző alakzatot az elrendezési diához.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddSmartArtPlaceholder(20, 20, 200, 200);
}
```

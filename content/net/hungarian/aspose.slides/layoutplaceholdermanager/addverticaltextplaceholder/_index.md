## LayoutPlaceholderManager.AddVerticalTextPlaceholder metódus

Új helyőrző alakzatot ad hozzá az elrendezési diára, amely függőleges irányban tartalmaz szövegtartalmat.

```csharp
public IAutoShape AddVerticalTextPlaceholder(float x, float y, float width, float height)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| x | Single | Az új helyőrző alakzat X koordinátája. |
| y | Single | Az új helyőrző alakzat Y koordinátája. |
| width | Single | Az új helyőrző alakzat szélessége. |
| height | Single | Az új helyőrző alakzat magassága. |

### Visszatérési érték

Created [`IAutoShape`](../../iautoshape) with a Text (Vertical) placeholder.

### Példák

Az alábbi példa bemutatja, hogyan adhatja hozzá a Text (Vertical) helyőrző alakzatot az elrendezési diához.

```csharp
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalTextPlaceholder(20, 20, 300, 500);
}
```

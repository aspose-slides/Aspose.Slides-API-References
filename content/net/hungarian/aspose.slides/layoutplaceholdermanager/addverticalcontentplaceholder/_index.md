## LayoutPlaceholderManager.AddVerticalContentPlaceholder metódus

Új helykitöltő alakzatot ad a layout diához, hogy vertikális irányban tartalmat, például képet, táblázatot, médiát vagy szöveget tároljon.

```csharp
public IAutoShape AddVerticalContentPlaceholder(float x, float y, float width, float height)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| x | Single | Az új helykitöltő alakzat X koordinátája. |
| y | Single | Az új helykitöltő alakzat Y koordinátája. |
| width | Single | Az új helykitöltő alakzat szélessége. |
| height | Single | Az új helykitöltő alakzat magassága. |

### Visszatérési érték

Létrehozva [`IAutoShape`](../../iautoshape) egy Content (Vertical) helykitöltővel.

### Példák

Az alábbi példa bemutatja, hogyan adhatunk hozzá egy Content (Vertical) helykitöltő alakzatot a layout diához.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalContentPlaceholder(20, 20, 300, 500);
}
```

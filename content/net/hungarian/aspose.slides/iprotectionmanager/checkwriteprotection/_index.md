## IProtectionManager.CheckWriteProtection metódus

Meghatározza, hogy a prezentáció jelszóval védett-e a módosításhoz.

```csharp
public bool CheckWriteProtection(string password)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| password | String | A jelszó az ellenőrzéshez. |

### Visszatérési érték

Igaz, ha a jelszó érvényes; egyébként hamis.

### Megjegyzések

1. A módszer meghívása előtt ellenőrizni kell a [`IsWriteProtected`](../iswriteprotected) tulajdonságot. 2. Ha a jelszó null vagy üres, ez a módszer hamisat ad vissza.

### Példák

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

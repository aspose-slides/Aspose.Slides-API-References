## ProtectionManager.CheckWriteProtection metódus

Meghatározza, hogy egy prezentáció jelszóval védett-e a módosításhoz.

```csharp
public bool CheckWriteProtection(string password)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| password | String | The password for checking. |

### Visszatérési érték

True ha a jelszó érvényes; egyébként false.

### Megjegyzések

1. A [`IsWriteProtected`](../iswriteprotected) tulajdonságot ellenőriznie kell, mielőtt meghívja ezt a metódust. 2. Ha a jelszó null vagy üres, ez a metódus false értéket ad vissza.

### Példák

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

### Példák

A következő példa bemutatja, hogyan kell jegyzeteket hozzáadni egy adott PowerPoint prezentáció diájához.

```csharp
[C#]
	// Példányosít egy Presentation objektumot, amely egy prezentációs fájlt képvisel
	using(Presentation presentation = new Presentation(dataDir + "AccessSlides.pptx")) {
	  // Jegyzeteket ad hozzá az első diára
	  INotesSlideManager mgr = presentation.Slides[0].NotesSlideManager;
	  INotesSlide noteSlide = mgr.AddNotesSlide();
	  noteSlide.NotesTextFrame.Text = "Your Notes";
	  // Mentse a prezentációt lemezre
	  presentation.Save(dataDir + "RemoveNotesAtSpecificSlide_out.pptx", SaveFormat.Pptx);
	}
```

A következő példák bemutatják, hogyan kell eltávolítani a jegyzeteket egy adott PowerPoint prezentáció diájáról.

```csharp
[C#]
	// Példányosít egy Presentation objektumot, amely egy prezentációs fájlt képvisel
	using(Presentation presentation = new Presentation(dataDir + "AccessSlides.pptx")) {
	  // Az első dia jegyzeteinek eltávolítása
	  INotesSlideManager mgr = presentation.Slides[0].NotesSlideManager;
	  mgr.RemoveNotesSlide();
	  // Mentse a prezentációt lemezre
	  presentation.Save(dataDir + "RemoveNotesAtSpecificSlide_out.pptx", SaveFormat.Pptx);
	}
```

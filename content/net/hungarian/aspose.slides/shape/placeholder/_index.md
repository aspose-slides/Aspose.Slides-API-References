## Shape.Placeholder tulajdonság

Visszaadja a helyőrzőt egy alakzathoz. Visszaadja a null értéket, ha az alakzatnak nincs helyőrzője. Csak olvasható [`IPlaceholder`](../../iplaceholder).

```csharp
public IPlaceholder Placeholder { get; }
```

### Példák

Az alábbi példa bemutatja, hogyan lehet módosítani a Text-et a Placeholder-ben.

```csharp
[C#]
// Példányosít egy Presentation osztályt
using (Presentation pres = new Presentation("ReplacingText.pptx"))
{
    // Eléri az első diát
    ISlide sld = pres.Slides[0];
    // Végigiterál a formákon a helyőrző megtalálásához
    foreach (IShape shp in sld.Shapes)
        if (shp.Placeholder != null)
        {
            // Megváltoztatja a szöveget minden helyőrzőben
            ((IAutoShape)shp).TextFrame.Text = "This is a Placeholder";
        }
    // Mentés a prezentációt a lemezre
    pres.Save("output_out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
}
```

Az alábbi példa bemutatja, hogyan lehet beállítani a Prompt Text-et a Placeholder-ben.

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation2.pptx"))
{
    ISlide slide = pres.Slides[0];
    foreach (IShape shape in slide.Slide.Shapes) // Végigiterál a dián
    {
        if (shape.Placeholder != null && shape is AutoShape)
        {
            string text = "";
            if (shape.Placeholder.Type == PlaceholderType.CenteredTitle) // PowerPoint displays "Kattintson a cím hozzáadásához"
            {
                text = "Add Title";
            }
            else if (shape.Placeholder.Type == PlaceholderType.Subtitle) // Hozzáad alcímet
            {
                text = "Add Subtitle";
            }
            ((IAutoShape)shape).TextFrame.Text = text;
            Console.WriteLine($"Placeholder with text: {text}");
        }
    }
    pres.Save("Placeholders_PromptText.pptx", SaveFormat.Pptx);
}
```

## SummaryZoomSectionCollection.AddSummaryZoomSection metódus

Létrehoz egy új Summary Zoom Section objektumot, és hozzáadja a gyűjteményhez

```csharp
public ISummaryZoomSection AddSummaryZoomSection(ISection section)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| section | ISection | Az új Summary Zoom Section elemhez tartozó szakasz [`ISection`](../../isection) |

### Visszatérési érték

Hozzáadott [`ISummaryZoomFrame`](../../isummaryzoomframe) elemet

### Kivételek

| kivétel | feltétel |
| --- | --- |
| ArgumentException | A hivatkozott szakasz nem tartozik az aktuális bemutatóhoz, vagy nem tartalmaz diákot. |

### Megjegyzés

Ha egy elem már létezik a gyűjteményben ehhez a szakaszhoz, a meglévő elem kerül visszaadásra.

### Példák

A példa bemutatja, hogyan lehet index szerint lekérni a Summary Zoom Section elemet:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    ISummaryZoomFrame zoomFrame = (ISummaryZoomFrame) pres.Slides[1].Shapes[0];
    ISummaryZoomSectionCollection collection = zoomFrame.SummaryZoomCollection;
    ISummaryZoomSection newZoomSection = collection.AddSummaryZoomSection(pres.Sections[3]);
}
```

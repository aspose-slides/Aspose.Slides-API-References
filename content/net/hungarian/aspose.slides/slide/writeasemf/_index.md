### Példák

Az alábbi kódrészlet bemutatja, hogyan lehet az első diát egy PowerPoint-prezentációból metafájlba konvertálni.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.emf"))
    {
        // Elmenti az első diát metafájlként
        pres.Slides[0].WriteAsEmf(fileStream);
    }
}
```

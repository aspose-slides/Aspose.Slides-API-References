## SetLicense(string) {#setlicense_1}

A komponenst licenceli.

```csharp
public void SetLicense(string licenseName)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| licenseName | String | Lehet teljes vagy rövid fájlnév vagy egy beágyazott erőforrás neve. Üres karakterláncot használva átvált értékelési módra. |

### Megjegyzés

Megpróbálja megtalálni a licencet a következő helyeken:

1. Kifejezett útvonal.
2. A komponens összeállításának mappája.
3. Az ügyfél hívó összeállításának mappája.
4. A belépő összeállítás mappája.
5. Egy beágyazott erőforrás az ügyfél hívó összeállításában.

**Megjegyzés:** A .NET Compact Framework esetén csak a következő helyeken keresi a licencet:

1. Kifejezett útvonal.
2. Egy beágyazott erőforrás az ügyfél hívó összeállításában.

### Példák

Egy példa során megkísérli megtalálni a MyLicense.lic nevű licencfájlt a komponenst tartalmazó mappában, a hívó összeállítást tartalmazó mappában, a belépő összeállítás mappájában, majd a hívó összeállítás beágyazott erőforrásai között.

```csharp
[C#]

License license = new License();
license.SetLicense("MyLicense.lic");


[Visual Basic]

Dim license As License = New License
license.SetLicense("MyLicense.lic")
```

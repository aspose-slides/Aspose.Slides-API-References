## IPictureFillFormat.TileFlip tulajdonság

Elforgatja a textúra csempét a vízszintes, függőleges vagy mindkét tengely körül. Olvasás/írás [`TileFlip`](../../tileflip).

```csharp
public TileFlip TileFlip { get; set; }
```

### Megjegyzés

Alapértelmezett a NoFlip.

### Példák

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Lekéri a alakzat kép kitöltés formátumát
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Beállítja a kép kitöltés módját Tile-re
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Elforgatja a textúra csempét a függőleges tengely körül.
    pictureFillFormat.TileFlip = TileFlip.FlipY;
}
```

## IPictureFillFormat.DeletePictureCroppedAreas metódus

Törli a kitöltő képre alkalmazott levágott területeket.

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### Visszatérési érték

Levágott kép vagy eredeti kép, ha a levágás nem szükséges.

### Megjegyzések

Ez a metódus a WMF/EMF metafájlokat raszteres PNG képpé konvertálja a levágás során.

### Példák

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Lekéri a PictureFrame-et
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // Törli a PictureFrame kép levágott területeit
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

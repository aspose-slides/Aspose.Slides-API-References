## IPictureFillFormat.TileOffsetX tulajdonság

Visszaadja vagy beállítja a textúra vízszintes eltolását a forma kiindulási pontjától pontokban. A pozitív érték a textúrát jobbra mozgatja, míg a negatív érték balra. Olvasás/írás Single.

```csharp
public float TileOffsetX { get; set; }
```

### Példák

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Lekéri a forma képkitöltési formátumát
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Beállítja a képkitöltési módot Tile-re
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Beállítja a textúra vízszintes eltolását 20 pontra
    pictureFillFormat.TileOffsetX = 20f;
}
```

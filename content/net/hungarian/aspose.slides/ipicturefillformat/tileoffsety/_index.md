## IPictureFillFormat.TileOffsetY tulajdonság

A textúra függőleges eltolását adja vissza vagy állítja be a forma kiindulási pontjától pontokban. A pozitív érték lefelé mozgatja a textúrát, míg a negatív érték felfelé mozgatja. Olvasás/írás Single.

```csharp
public float TileOffsetY { get; set; }
```

### Példák

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Lekéri a forma kép kitöltési formátumát
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Beállítja a kép kitöltési módot a Csempézésre
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Beállítja a textúra függőleges eltolását -50 pontra
    pictureFillFormat.TileOffsetY = -50;
}
```

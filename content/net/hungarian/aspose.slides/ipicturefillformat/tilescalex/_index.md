## IPictureFillFormat.TileScaleX tulajdonság

Visszaadja vagy beállítja a textúra kitöltés vízszintes méretarányát százalékban. Olvasás/írás Single.

```csharp
public float TileScaleX { get; set; }
```

### Példák

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Lekéri a shape kép kitöltési formátumát
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Beállítja a kép kitöltési módot Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Beállítja a textúra vízszintes méretarányát 120 százalékra
    pictureFillFormat.TileScaleX = 120;
}
```

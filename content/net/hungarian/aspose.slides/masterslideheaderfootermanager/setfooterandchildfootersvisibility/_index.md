## MasterSlideHeaderFooterManager.SetFooterAndChildFootersVisibility metódus

Módosítja a mesterdia lábléc helyőrzőjének és az összes gyermek lábléc helyőrző láthatóságát. A gyermek helyőrzők olyan helyőrzőket jelentenek, amelyek a függő elrendezési diákon és a függő diákon találhatók. A függő elrendezési diákok és diákok a mesterdiát használják és tőlük függenek.

```csharp
public void SetFooterAndChildFootersVisibility(bool isVisible)
```

| Parameter | Type | Description |
| --- | --- | --- |
| isVisible | Boolean | true - lábléc helyőrzőket láthatóvá teszi, egyébként elrejti őket. |

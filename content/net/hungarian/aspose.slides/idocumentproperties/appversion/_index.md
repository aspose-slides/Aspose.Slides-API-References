## IDocumentProperties.AppVersion tulajdonság

Visszaadja az alkalmazás verzióját. Csak olvasható String.

```csharp
public string AppVersion { get; }
```

### Megjegyzések

Az elem tartalma az XX.YYYY formátumban kell legyen, ahol az X és Y numerikus értékeket jelentenek; egyébként a dokumentumot nem konformannak tekintik. Az Aspose.Slides a verzióját az XX.YYZZ formátumban jeleníti meg, ahol: XX – főverzió YY – alverzió ZZ – javítási verzió Például a 23.0105 érték az Aspose.Slides 23.1.5 verzióját jelenti.

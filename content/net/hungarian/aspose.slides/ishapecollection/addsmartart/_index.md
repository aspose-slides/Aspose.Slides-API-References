## IShapeCollection.AddSmartArt metódus

Létrehoz egy SmartArt diagramot, és a shape collection végére adja hozzá.

```csharp
public ISmartArt AddSmartArt(float x, float y, float width, float height, 
    SmartArtLayoutType layoutType)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| x | Single | A diagram keretének x-koordinátája pontban. |
| y | Single | A diagram keretének y-koordinátája pontban. |
| width | Single | A diagram keretének szélessége pontban. |
| height | Single | A diagram keretének magassága pontban. |
| layoutType | SmartArtLayoutType | A SmartArt elrendezés típusa. |

### Visszatérési érték

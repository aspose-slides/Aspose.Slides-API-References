## AddAudioFrameEmbedded(float, float, float, float, Stream) {#addaudioframeembedded_1}

Egy új audio keretet hoz létre beágyazott WAV fájllal, és a shape gyűjtemény végéhez adja hozzá. A beágyazott hang a Presentation.Audios gyűjteményhez kerül hozzáadásra.

```csharp
public IAudioFrame AddAudioFrameEmbedded(float x, float y, float width, float height, 
    Stream audio_stream)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| x | Single | Az új audio keret x-koordinátája pontban. |
| y | Single | Az új audio keret y-koordinátája pontban. |
| width | Single | Az új audio keret szélessége pontban. |
| height | Single | Az új audio keret magassága pontban. |
| audio_stream | Stream | Egy bemeneti adatfolyam, amely WAV hangadatot tartalmaz a beágyazáshoz. |

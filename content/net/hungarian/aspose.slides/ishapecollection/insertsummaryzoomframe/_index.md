## IShapeCollection.InsertSummaryZoomFrame metódus

Új Summary Zoom keretet hoz létre, és beilleszti a shape collectionbe a megadott indexnél.

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| index | Int32 | A nullálistól kezdődő index, ahol a Summary Zoom keretet be kell illeszteni. |
| x | Single | Az új Summary Zoom keret x koordinátája pontokban. |
| y | Single | Az új Summary Zoom keret y koordinátája pontokban. |
| width | Single | Az új Summary Zoom keret szélessége pontokban. |
| height | Single | Az új Summary Zoom keret magassága pontokban. |

### Visszatérési érték

Az újonnan létrehozott [`ISummaryZoomFrame`](../../isummaryzoomframe).

### Kivétel

| kivétel | feltétel |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | Dobott, ha a bemutató nem tartalmaz szekciókat, vagy ha a cél dia nem tartozik egy szekcióhoz sem. |

### Megjegyzés

Ez a metódus egy Summary Zoom keretet hoz létre, amely összegzi az összes szekció összefoglaló hivatkozásait a bemutatóban.

### Példák

Ez a példa bemutatja a Summary Zoom objektum létrehozását és a gyűjtemény megadott indexébe történő beszúrását (feltételezve, hogy a "Presentation.pptx" bemutatóban legalább két szekció van):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSummaryZoomFrame(2, 150, 20, 50, 50);
}
```

## InsertZoomFrame(int, float, float, float, float, ISlide) {#insertzoomframe}

Új Zoom keretet hoz létre, és beszúrja a forma gyűjteménybe a megadott indexnél.

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| index | Int32 | A nullártól induló index, amelyen a Zoom keretet be kell szúrni. |
| x | Single | Az új Zoom keret x-koordinátája pontban. |
| y | Single | Az új Zoom keret y-koordinátája pontban. |
| width | Single | Az új Zoom keret szélessége pontban. |
| height | Single | Az új Zoom keret magassága pontban. |
| slide | ISlide | A Zoom keret által hivatkozott [`ISlide`](../../islide). |

### Visszatérési érték

Az újonnan létrehozott [`IZoomFrame`](../../izoomframe).

### Kivétel

| kivétel | feltétel |
| --- | --- |
| ArgumentException | Akkor dobódik, ha a hivatkozott diát nem a jelenlegi bemutató tartalmazza. |

### Példák

Ez a példa bemutatja egy Zoom objektum létrehozását és beszúrását a gyűjtemény megadott indexén (tételezzük fel, hogy a "Presentation.pptx" bemutatóban legalább két dia van):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertZoomFrame(2, 150, 20, 50, 50, pres.Slides[1]);
}
```

## InsertZoomFrame(int, float, float, float, float, ISlide, IPPImage) {#insertzoomframe_1}

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide, IPPImage image)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| index | Int32 | A nullártól induló index, amelyen a Zoom keretet be kell szúrni. |
| x | Single | Az új Zoom keret x-koordinátája pontban. |
| y | Single | Az új Zoom keret y-koordinátája pontban. |
| width | Single | Az új Zoom keret szélessége pontban. |
| height | Single | Az új Zoom keret magassága pontban. |
| slide | ISlide | A Zoom kerethez hivatkozott [`ISlide`](../../islide). |
| image | IPPImage | Az a kép, amely a hivatkozott diához [`IPPImage`](../../ippimage) tartozik. |

### Visszatérési érték

### Példák

Ez a példa bemutatja egy Zoom objektum létrehozását és beszúrását a gyűjtemény megadott indexén (tételezzük fel, hogy a "Presentation.pptx" bemutatóban legalább két dia van):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{

}
```

## IShapeCollection.AddSummaryZoomFrame metódus

Új Summary Zoom keretet hoz létre, és a alakzatgyűjtemény végéhez adja hozzá.

```csharp
public ISummaryZoomFrame AddSummaryZoomFrame(float x, float y, float width, float height)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| x | Single | Az új Summary Zoom keret x koordinátája pontban. |
| y | Single | Az új Summary Zoom keret y koordinátája pontban. |
| width | Single | Az új Summary Zoom keret szélessége pontban. |
| height | Single | Az új Summary Zoom keret magassága pontban. |

### Visszatérési érték

Az újonnan létrehozott [`ISummaryZoomFrame`](../../isummaryzoomframe).

### Kivétel

| kivétel | feltétel |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | Dobás, ha a prezentációban nincsenek szekciók, vagy ha a cél dia nem tartozik egy szekcióhoz sem. |

### Megjegyzés

Ez a metódus egy Summary Zoom keretet hoz létre, amely összegzi a prezentáció összes szekciójának összefoglaló hivatkozásait.

### Példák

Ez a példa bemutatja, hogyan adhatunk egy Summary Zoom objektumot a gyűjtemény végéhez (feltételezve, hogy a "Presentation.pptx" prezentációban legalább két szekció van):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSummaryZoomFrame(150, 20, 500, 250);
}
```

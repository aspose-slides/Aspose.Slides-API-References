## InsertAudioFrameEmbedded(int, float, float, float, float, Stream) {#insertaudioframeembedded_1}

Létrehoz egy új audio keretet beágyazott WAV-fájllal, és beilleszti a shape gyűjteménybe a megadott indexen. A beágyazott hang a Presentation.Audios gyűjteményhez kerül hozzáadásra.

```csharp
public IAudioFrame InsertAudioFrameEmbedded(int index, float x, float y, float width, float height, 
    Stream audio_stream)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| index | Int32 | A nulla-alapú index, amelynél be kell szúrni az audio keretet. |
| x | Single | Az új audio keret x-koordinátája pontban. |
| y | Single | Az új audio keret y-koordinátája pontban. |
| width | Single | Az új audio keret szélessége pontban. |
| height | Single | Az új audio keret magassága pontban. |
| audio_stream | Stream | Egy bemeneti adatfolyam, amely WAV-audio adatokat tartalmaz a beágyazáshoz. |

### Visszatérési érték

## InsertAutoShape(int, ShapeType, float, float, float, float, bool) {#insertautoshape_1}

Új automatikus alakzatot hoz létre, és a megadott indexnél beilleszti az alakzatgyűjteménybe, opcionálisan az alapértelmezett sablonstílussal inicializálva.

```csharp
public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| index | Int32 | A nulla-alapú index, amelynél az automatikus alakzatot be kell szúrni. |
| shapeType | ShapeType | A beillesztendő automatikus alakzat [`ShapeType`](../../shapetype). |
| x | Single | Az alakzat keretének x-koordinátája pontban. |
| y | Single | Az alakzat keretének y-koordinátája pontban. |
| width | Single | Az alakzat keretének szélessége pontban. |
| height | Single | Az alakzat keretének magassága pontban. |
| createFromTemplate | Boolean | True, ha az alapértelmezett sablonstílust alkalmazza (beleértve a nem üres nevet, egyszerű stílust és középre igazított szöveget); false, ha az alakzatot úgy hozza létre, hogy minden tulajdonság az alapértelmezett értékekkel rendelkezik. |

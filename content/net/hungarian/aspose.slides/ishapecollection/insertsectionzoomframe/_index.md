## InsertSectionZoomFrame(int, float, float, float, float, ISection) {#insertsectionzoomframe}

Új Section Zoom keretet hoz létre, és a megadott indexnél beszúrja az alakgyűjteménybe.

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| index | Int32 | Az a nullártalapú index, amelynél be kell illeszteni a Section Zoom keretet. |
| x | Single | Az új Section Zoom keret x-koordinátája pontban. |
| y | Single | Az új Section Zoom keret y-koordinátája pontban. |
| width | Single | Az új Section Zoom keret szélessége pontban. |
| height | Single | Az új Section Zoom keret magassága pontban. |
| section | ISection | A Section Zoom keret által hivatkozott [`ISection`](../../isection); a prezentációnak kell tartoznia, és legalább egy diát kell tartalmaznia. |

### Visszatérési érték

Az újonnan létrehozott [`ISectionZoomFrame`](../../isectionzoomframe).

### Kivétel

| Kivétel | Feltétel |
| --- | --- |
| ArgumentException | Ha a hivatkozott szakasz nem tartozik az aktuális prezentációhoz, vagy nem tartalmaz diát. |

### Példák

Ez a példa bemutatja egy Section Zoom objektum létrehozását és beszúrását egy gyűjtemény megadott indexén (feltételezve, hogy a “Presentation.pptx” prezentációban legalább két szakasz van):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSectionZoomFrame(2, 150, 20, 50, 50, pres.Sections[1]);
}
```

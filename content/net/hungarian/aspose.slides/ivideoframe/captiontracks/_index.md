## IVideoFrame.CaptionTracks tulajdonság

A zárt feliratok gyűjteményét adja vissza, amely az audio kerethez kapcsolódik. Ez a tulajdonság csak olvasható, és egy [`ICaptionsCollection`](../../icaptionscollection) ad vissza, amely az összes feliratsávot tartalmaz.

```csharp
public ICaptionsCollection CaptionTracks { get; }
```

### Példák

Példa:

```csharp
[C#]
using (Presentation pres = new Presentation("video with captions.pptx"))
{
    foreach (IShape shape in pres.Slides[0].Shapes)
    {
        if (!(shape is IVideoFrame videoFrame)) continue;

        foreach (var captionTrack in videoFrame.CaptionTracks)
        {
            // Kiextrahálja a feliratok bináris adatait, és elmenti őket a fájlba
            System.IO.File.WriteAllBytes(captionTrack.CaptionId + ".vtt", captionTrack.BinaryData);
        }
    }
}
```

## IInkBrush.InkEffect tulajdonság

Lekéri a tintahatás típusát (pl. Galaxy, Gold, Silver), amely meghatározza a tintavonal vizuális stílusát. Az érték az ecset tulajdonságából, az "inkEffects" tulajdonságból kerül beolvasásra. Ha nincs felismerhető hatás megadva, a NotDefined érték kerül visszaadásra.

```csharp
public InkEffectType InkEffect { get; }
```

### Példák

Példa:

```csharp
[C#]
using (var pres = new Presentation("Presentation.pptx"))
{
    Ink ink = pres.Slides[0].Shapes[0] as Ink;
    IInkBrush brush = ink.Traces[0].Brush;
    Console.WriteLine("InkEffects = {0}", brush.InkEffect);
}
```

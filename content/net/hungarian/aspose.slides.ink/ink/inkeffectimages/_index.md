## Ink.InkEffectImages tulajdonság

A tinta ecsetek vizuális hatásainak szimulálásához használt egyéni képek gyűjteményét adja vissza. Ezeket a képeket a tinta megjelenítésekor használják a megadott [`InkEffectType`](../../inkeffecttype) értékekkel, például Galaxy, Rainbow stb. Saját képeket biztosítva szabályozhatja, hogy minden tinta hatás hogyan jelenik meg.

```csharp
public static IDictionary<InkEffectType, IImage> InkEffectImages { get; }
```

### Megjegyzések

Ez a tulajdonság lehetővé teszi az alapértelmezett tinta hatás textúrák felhasználó által meghatározottakra cserélését, ami különösen hasznos, ha az alapértelmezett eszközök licencelési okokból korlátozottak vagy futásidőben nem érhetők el. A szótár minden bejegyzésének egy [`InkEffectType`](../../inkeffecttype) értéket kell társítania egy megfelelő [`IImage`](../../../aspose.slides/iimage) objektummal (például Bitmap, vagy egy Aspose képfeldolgozó interfész).

### Példák

```csharp
[C#]
IImage image = Images.FromFile("image.png");
Ink.InkEffectImages.Add(InkEffectType.Galaxy, image);
```

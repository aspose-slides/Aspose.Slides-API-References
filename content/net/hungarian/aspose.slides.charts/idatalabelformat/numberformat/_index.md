## IDataLabelFormat.NumberFormat tulajdonság

A DataLabels objektum formátumkarakterláncát képviseli. Olvasás/írás String.

```csharp
public string NumberFormat { get; set; }
```

### Megjegyzés

Ha ennek a DataLabelFormat objektumnak a szülője egy DataLabelCollection adatcímke-gyűjtemény, akkor ez a property visszaadja vagy beállítja a NumberFormat property alapértelmezett értékét az új adatcímkékhez a DataLabelCollection gyűjteményben. Ha ezt a property-t egy értékkel állítják be, akkor az érték a NumberFormat property-re is be lesz állítva az összes adatcímkénél a DataLabelCollection gyűjteményben (azaz a "DataLabels.DefaultDataLabelFormat.NumberFormat = val;" hatására minden DataLabels[i].NumberFormat értéke val lesz).

### Példák

```csharp
[C#]
series.Labels.DefaultDataLabelFormat.ShowValue = true;
series.Labels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = false;
series.Labels.DefaultDataLabelFormat.NumberFormat = "0.0%";
```

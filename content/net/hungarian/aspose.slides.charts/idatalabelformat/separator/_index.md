## IDataLabelFormat.Separator tulajdonság

Beállítja vagy visszaadja a Variant-et, amely a diagram adatcímkéinél használt elválasztót képviseli. Olvasás/írás String.

```csharp
public string Separator { get; set; }
```

### Megjegyzés

Ha ennek a DataLabelFormat objektumnak a szülője egy DataLabelCollection adatcímke-gyűjtemény, akkor ez a tulajdonság beállítja vagy lekéri az új adatcímkék Separator tulajdonságának alapértelmezett értékét a DataLabelCollection gyűjteményben. Ennek a tulajdonságnak az értékkel való beállítása szintén beállítja ezt az értéket a Separator tulajdonságra az összes adatcímke esetén a DataLabelCollection gyűjteményben (pl. "DataLabels.DefaultDataLabelFormat.Separator = val;" okozza, hogy minden DataLabels[i].Separator egyenlő legyen a val értékkel).

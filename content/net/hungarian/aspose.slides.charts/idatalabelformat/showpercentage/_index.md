## IDataLabelFormat.ShowPercentage tulajdonság

Egy adott diagram adatcímkéjének százalékos értékvizualizációs viselkedését reprezentálja. Az igaz érték megjeleníti a százalékos értéket. A hamis elrejti. Olvasás/írás Boolean.

```csharp
public bool ShowPercentage { get; set; }
```

### Megjegyzések

Ha ennek a DataLabelFormat objektumnak a szülője egy DataLabelCollection adatcímke-gyűjtemény, akkor ez a tulajdonság lekéri vagy beállítja az új adatcímkék számára a ShowPercentage tulajdonság alapértelmezett értékét a DataLabelCollection gyűjteményben. Ennek a tulajdonságnak az értékkel való beállítása szintén beállítja ezt az értéket a ShowPercentage tulajdonságra a DataLabelCollection összes adatcímkéjénél (azaz "DataLabels.DefaultDataLabelFormat.ShowPercentage = val;" miatt minden DataLabels[i].ShowPercentage egyenlő lesz a val értékkel).

## DataLabelFormat.ShowPercentage tulajdonság

Képviseli egy adott diagram adatcímkéjének százalékos érték megjelenítési viselkedését. True megjeleníti a százalékos értéket. False elrejti. Olvasás/írás Boolean.

```csharp
public bool ShowPercentage { get; set; }
```

### Megjegyzés

Ha ennek a DataLabelFormat objektumnak a szülője egy DataLabelCollection adatcímke-gyűjtemény, akkor ez a tulajdonság visszaadja vagy beállítja az új adatcímkék ShowPercentage tulajdonságának alapértelmezett értékét a DataLabelCollection gyűjteményben. Ennek a tulajdonságnak az értékkel való beállítása ugyancsak beállítja ezt az értéket a ShowPercentage tulajdonságra minden adatcímkénél a DataLabelCollection gyűjteményben (azaz "DataLabels.DefaultDataLabelFormat.ShowPercentage = val;" miatt minden DataLabels[i].ShowPercentage egyenlő lesz a val értékkel).

## DataLabelFormat.ShowLeaderLines tulajdonság

Képviseli egy adott diagram adatcímkéjének vezetővonal megjelenítési viselkedését. True megjeleníti a vezetővonalakat. False elrejti őket. Olvasás/írás Boolean.

```csharp
public bool ShowLeaderLines { get; set; }
```

### Megjegyzés

Ha ennek a DataLabelFormat objektumnak a szülője egy DataLabelCollection adatcímke gyűjtemény, akkor ez a tulajdonság visszaadja vagy beállítja a ShowLeaderLines tulajdonság alapértelmezett értékét az új adatcímkék számára a DataLabelCollection gyűjteményben. A tulajdonság értékével történő beállítás ugyanakkor ezt az értéket a ShowLeaderLines tulajdonságra állítja az összes adatcímke esetén a DataLabelCollection gyűjteményben (azaz "DataLabels.DefaultDataLabelFormat.ShowLeaderLines = val;" miatt minden DataLabels[i].ShowLeaderLines egyenlő lesz a val értékkel).

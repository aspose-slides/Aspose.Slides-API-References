## DataLabelFormat.Separator tulajdonság

Beállítja vagy visszaadja a Variant típusú értéket, amely a diagram adatcímkéinél használt elválasztót képviseli. Olvasás/írás String.

```csharp
public string Separator { get; set; }
```

### Megjegyzések

Ha ennek a DataLabelFormat objektumnak a szülője egy DataLabelCollection adatcímkék gyűjteménye, akkor ez a tulajdonság lekéri vagy beállítja a Separator tulajdonság alapértelmezett értékét az új adatcímkékhez a DataLabelCollection gyűjteményben. Ennek a tulajdonságnak az értékkel való beállítása szintén ezt az értéket állítja be a Separator tulajdonsághoz a DataLabelCollection összes adatcímkéjében (például "DataLabels.DefaultDataLabelFormat.Separator = val;" miatt minden DataLabels[i].Separator egyenlő lesz a val értékkel).

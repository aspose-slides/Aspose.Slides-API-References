## ChartSeriesGroup.Overlap tulajdonság

Megadja, hogy a sávok és oszlopok mennyire fedik egymást a 2-D diagramokon, százalékban (-100%-tól 100%-ig). - -100%: Legnagyobb távolság (a sávok teljesen el vannak választva). - 0%: A sávok egymás mellett helyezkednek el átfedés vagy térköz nélkül. - 100%: Legnagyobb átfedés (a sávok teljesen egymást fedik). Ez a tulajdonság olvasás/írás SByte.

```csharp
public sbyte Overlap { get; set; }
```

### Kivételek

| kivétel | feltétel |
| --- | --- |
| ArgumentOutOfRangeException | Ahol az érték a -100 és 100 közötti érvényes tartományon kívül kerül beállításra. |

### Példák

Az alábbi példa bemutatja, hogyan állítható be a átfedés egy diagram sorozatcsoport esetén, és hogyan jeleníthető meg a kapott diagram egy űrlapon:

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // Állítsa be az átfedést 55%-ra

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

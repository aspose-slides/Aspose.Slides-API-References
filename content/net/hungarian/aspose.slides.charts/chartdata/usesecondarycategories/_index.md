## ChartData.UseSecondaryCategories tulajdonság

Ha false, akkor a [`SecondaryCategories`](../secondarycategories) tulajdonság null értéket ad vissza, és a [`Categories`](../categories) tulajdonságban lévő adatok mind az elsődleges, mind a másodlagos sorozatokhoz használatos. Ha true, akkor a [`SecondaryCategories`](../secondarycategories) tulajdonságban lévő adatokat a másodlagos sorozatokhoz, a [`Categories`](../categories) tulajdonságban lévő adatokat pedig az elsődleges sorozatokhoz használják. Olvasás/írás Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

### Példák

Példa. Mely kategóriák kapcsolódnak a sorozatokhoz - ChartData.Categories vagy ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // a kapcsolódó kategóriák a series.Chart.ChartData.SecondaryCategories
}
else
{
    // a kapcsolódó kategóriák a series.Chart.ChartData.Categories
}
```

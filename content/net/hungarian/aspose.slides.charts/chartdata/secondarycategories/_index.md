## ChartData.SecondaryCategories tulajdonság

Visszaadja a másodlagos kategóriákat, ha a(z) [`UseSecondaryCategories`](../usesecondarycategories) tulajdonság igaz. Csak olvasható [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### Megjegyzések

Ha a(z) [`UseSecondaryCategories`](../usesecondarycategories) tulajdonság hamis, akkor ez a `SecondaryCategories` tulajdonság null értéket ad vissza, és a [`Categories`](../categories) tulajdonság adatai mind az elsődleges, mind a másodlagos sorozatokhoz használatosak. Ha a(z) [`UseSecondaryCategories`](../usesecondarycategories) tulajdonság igaz, akkor a `SecondaryCategories` tulajdonság adatai a másodlagos sorozatokhoz, a [`Categories`](../categories) tulajdonság adatai pedig az elsődleges sorozatokhoz használatosak.

### Példák

Példa. Mely kategóriák kapcsolódnak a sorozatokhoz – ChartData.Categories vagy ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // a kapcsolódó kategóriák a series.Chart.ChartData.SecondaryCategories
}
else
{
    // a kapcsolódó kategóriák a series.Chart.ChartData.Categories
}
```

## ChartData.Categories tulajdonság

Lekéri az elsődleges kategóriákat (vagy mind az elsődleges, mind a másodlagos kategóriákat, ha [`UseSecondaryCategories`](../usesecondarycategories) tulajdonság hamis). Csak olvasható [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection Categories { get; }
```

### Megjegyzés

Ha [`UseSecondaryCategories`](../usesecondarycategories) tulajdonság hamis, akkor [`SecondaryCategories`](../secondarycategories) tulajdonság null értéket ad vissza, és a `Categories` tulajdonság adatai mind az elsődleges, mind a másodlagos sorozatokhoz használatosak. Ha [`UseSecondaryCategories`](../usesecondarycategories) tulajdonság igaz, akkor [`SecondaryCategories`](../secondarycategories) tulajdonság adatai a másodlagos sorozatokhoz, a `Categories` tulajdonság adatai pedig az elsődleges sorozatokhoz használatosak.

### Példák

Példa. Mely kategóriák kapcsolódnak a sorozatokhoz - ChartData.Categories vagy ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // a kapcsolódó kategóriák a series.Chart.ChartData.SecondaryCategories
}
else
{
    // a kapcsolódó kategóriák a series.Chart.ChartData.Categories
}
```

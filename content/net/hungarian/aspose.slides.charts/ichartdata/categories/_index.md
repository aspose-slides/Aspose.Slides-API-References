## IChartData.Categories tulajdonság

Lekéri az elsődleges kategóriákat (vagy az elsődleges és másodlagos kategóriákat, ha [`UseSecondaryCategories`](../usesecondarycategories) tulajdonság hamis). Csak olvasható [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection Categories { get; }
```

### Megjegyzés

Ha [`UseSecondaryCategories`](../usesecondarycategories) tulajdonság hamis, akkor [`SecondaryCategories`](../secondarycategories) tulajdonság null értéket ad vissza, és a `Categories` tulajdonság adatait mind az elsődleges, mind a másodlagos sorozatokhoz használják. Ha [`UseSecondaryCategories`](../usesecondarycategories) tulajdonság igaz, akkor a [`SecondaryCategories`](../secondarycategories) tulajdonság adatait a másodlagos sorozatokhoz, a `Categories` tulajdonság adatait pedig az elsődleges sorozatokhoz használják.

### Példák

Példa. Melyik kategóriák kapcsolódnak a sorozatokhoz – ChartData.Categories vagy ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // a kapcsolódó kategóriák a series.Chart.ChartData.SecondaryCategories
}
else
{
    // a kapcsolódó kategóriák a series.Chart.ChartData.Categories
}
```

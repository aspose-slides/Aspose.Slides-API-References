## IChartData.SecondaryCategories tulajdonság

A másodlagos kategóriákat adja vissza, ha a [`UseSecondaryCategories`](../usesecondarycategories) tulajdonság igaz. Csak olvasható [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### Megjegyzések

Ha a [`UseSecondaryCategories`](../usesecondarycategories) tulajdonság hamis, akkor ez a `SecondaryCategories` tulajdonság null értéket ad vissza, és a [`Categories`](../categories) tulajdonság adatai mind az elsődleges, mind a másodlagos sorozathoz használatosak. Ha a [`UseSecondaryCategories`](../usesecondarycategories) tulajdonság igaz, akkor ennek a `SecondaryCategories` tulajdonságnak az adatai a másodlagos sorozathoz, a [`Categories`](../categories) tulajdonság adatai pedig az elsődleges sorozathoz használatosak.

### Példák

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // a kapcsolódó kategóriák a series.Chart.ChartData.SecondaryCategories
}
else
{
    // a kapcsolódó kategóriák a series.Chart.ChartData.Categories
}
```

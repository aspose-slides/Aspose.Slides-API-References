## DataLabel.IsVisible property

A False azt jelenti, hogy az adatcímke nem látható (és ezáltal minden Show*-flags (ShowValue, ...) hamis). Csak olvasható Boolean.

```csharp
public bool IsVisible { get; }
```

### Megjegyzés

Ha az adatcímke látható, a Hide() metódussal elrejtheti. Ha azonban az adatcímke nem látható (IsVisible hamis), a Show*-flags (ShowValue, ...) true állapotra állításával teheti láthatóvá.

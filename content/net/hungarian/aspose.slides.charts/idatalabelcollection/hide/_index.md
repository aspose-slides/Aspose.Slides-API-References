## IDataLabelCollection.Hide method

A data label alapértelmezés szerint rejtetté tehető az összes Show*-flag (ShowValue, ...) a DefaultDataLabelFormat tulajdonságban false állapotra állításával. Az IsVisible false lesz ezután.

```csharp
public void Hide()
```

### Megjegyzés

Ha a data label alapértelmezés szerint nem látható (az IsVisible false), akkor a data label "alapértelmezés szerint látható" lesz az összes Show*-flag (ShowValue, ...) a DefaultDataLabelFormat tulajdonságban true állapotra állításával.

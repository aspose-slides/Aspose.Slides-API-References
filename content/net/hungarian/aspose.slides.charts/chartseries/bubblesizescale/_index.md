## ChartSeries.BubbleSizeScale tulajdonság

Meghatározza a buborékkart méretarányát (0-tól 300 %-ig terjedhet az alapmérettől). Ez a tulajdonság nem csak az adott sorozatra, hanem a szülő sorozatcsoport összes sorozatára vonatkozik – ez a megfelelő csoporttulajdonság projekciója. Így ez a tulajdonság csak-olvasható. A szülő sorozatcsoport eléréséhez használja a ParentSeriesGroup tulajdonságot. Az érték módosításához használja a ParentSeriesGroup.BubbleSizeScale olvasás/írás tulajdonságot.

```csharp
public int BubbleSizeScale { get; }
```

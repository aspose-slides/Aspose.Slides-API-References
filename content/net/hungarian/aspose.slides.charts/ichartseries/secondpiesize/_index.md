## IChartSeries.SecondPieSize property

Megadja a második szelet vagy sáv méretét egy pie-of-pie diagram vagy bar-of-pie diagram esetén, az első szelet méretének százalékában kifejezve (5 % és 200 % között lehet). Ez a tulajdonság nem csak az aktuális sorozatra, hanem a szülő sorozatcsoport összes sorozatára vonatkozik – ez a megfelelő csoporttulajdonság projekciója. Így ez a tulajdonság csak-olvasható. A szülő sorozatcsoport eléréséhez használja a ParentSeriesGroup tulajdonságot. Az érték módosításához használja a ParentSeriesGroup.SecondPieSize olvasás/írás tulajdonságot. Csak-olvasható UInt16.

```csharp
public ushort SecondPieSize { get; }
```

### Megjegyzés

Ez a ParentSeriesGroup.SecondPieSize tulajdonság projekciója.

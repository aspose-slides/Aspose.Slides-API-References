## IChartSeriesGroup.Overlap tulajdonság

Megadja, hogy a sávok és oszlopok mekkora mértékben fednek át egymást 2-D diagramokon, százalékban (-100% és 100% között). - -100%: Legnagyobb távolság (a sávok teljesen el vannak választva). - 0%: A sávok egymás mellett helyezkednek el átfedés vagy távolság nélkül. - 100%: Legnagyobb átfedés (a sávok teljesen egymásra fednek). Ez a tulajdonság írás/olvasás SByte.

```csharp
public sbyte Overlap { get; set; }
```

### Exceptions

| kivétel | feltétel |
| --- | --- |
| ArgumentOutOfRangeException | Kivétel, ha az érték a -100-tól 100-ig terjedő érvényes tartományon kívül van beállítva. |

### Példák

Az alábbi példa bemutatja, hogyan állítható be a átfedés egy diagram sorozatcsoport esetén, és hogyan jeleníthető meg a kapott diagram egy űrlapon:

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // Állítsa be az átfedést 55%

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

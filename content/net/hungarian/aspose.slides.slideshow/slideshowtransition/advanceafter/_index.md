## SlideShowTransition.AdvanceAfter tulajdonság

Ez a tulajdonság meghatározza, hogy a diavetítés egy bizonyos idő után áttér-e a következő diára. Olvasás/írás Boolean.

```csharp
public bool AdvanceAfter { get; set; }
```

### Példák

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    // Lekéri az első dia átmenetét
    ISlideShowTransition slideTransition = pres.Slides[0].SlideShowTransition;
    
    // Ellenőrzi, hogy az Advance Slide After jelző be van-e kapcsolva
    if (slideTransition.AdvanceAfter)
    {
        // Lekéri az Advance Slide After időértékét
        uint advanceAfterTime = slideTransition.AdvanceAfterTime;
    }
}
```

## WebDocument.Output tulajdonság

Visszaadja a dokumentum kimeneti elemeinek gyűjteményét. Csak olvasható `Output`.

```csharp
public Output Output { get; }
```

### Példák

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    var options = new WebDocumentOptions
    {
        TemplateEngine = new RazorTemplateEngine(),
        OutputSaver = new FileOutputSaver(),
        EmbedImages = false
    };
    
    WebDocument document = new WebDocument(options);

    // "index.html"-t hozzáadja a kimeneti fájlokhoz, az "index" sablont használja a generáláshoz, és a pres változót modellként
    document.Output.Add("index.html", "index", pres);

    // ... állítsa be a dokumentum egyéb beállításait, majd mentse a dokumentumot
    document.Save();
}
```

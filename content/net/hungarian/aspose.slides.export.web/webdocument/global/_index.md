## WebDocument.Global tulajdonság

Visszaadja a dokumentum globális tárolóját. Csak olvasható [`Storage`](../../storage).

```csharp
public Storage Global { get; }
```

### Példák

Ezzel a `Global` tulajdonsággal (a [`Storage`](../../storage) interfész megvalósítása) egy tulajdonság később felhasználható a sablonban:

```csharp
[C#]
var options = new WebDocumentOptions
{
    TemplateEngine = new RazorTemplateEngine(),
    OutputSaver = new FileOutputSaver(),
    EmbedImages = false
};

WebDocument document = new WebDocument(options);

// helyezze be a "slideMargin" tulajdonságot a sablonokban használatra
document.Global.Put("slideMargin", 10);

// ... állítsa be a dokumentum egyéb beállításait, majd mentse a dokumentumot
document.Save();
```

## Collect.Shapes metódus

Az összes [`Shape`](../../../aspose.slides/shape) példányt a [`Presentation`](../../../aspose.slides/presentation)-ban gyűjti össze.

```csharp
public static IEnumerable<Shape> Shapes(Presentation pres)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| pres | Presentation | Presentation a formák gyűjtéséhez |

### Visszatérési érték

A prezentációban található összes alakzat gyűjteménye.

### Példák

```csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    foreach (Shape shape in Collect.Shapes(pres))
    {
        // ha az alakzat AutoShape, adjon hozzá egy fekete szilárd keretet
        if (shape is AutoShape autoShape)
        {
            autoShape.LineFormat.Style = LineStyle.Single;
            autoShape.LineFormat.Width = 10f;
            autoShape.LineFormat.FillFormat.FillType = FillType.Solid;
            autoShape.LineFormat.FillFormat.SolidFillColor.Color = Color.Black;
        }
    }
    
    pres.Save("pres-out.pptx", SaveFormat.Pptx);
}        
```

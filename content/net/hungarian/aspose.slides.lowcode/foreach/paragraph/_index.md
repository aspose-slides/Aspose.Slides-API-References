## Paragraph(Presentation, bool, ForEachParagraphCallback) {#paragraph}

Iterálja minden [`Paragraph`](../paragraph)-t a [`Presentation`](../../../aspose.slides/presentation)-ben. A formák (Shapes) minden típusú dián iterálódnak – [`Slide`](../slide), [`MasterSlide`](../masterslide), [`LayoutSlide`](../layoutslide) és [`NotesSlide`](../../../aspose.slides/notesslide)

```csharp
public static void Paragraph(Presentation pres, bool includeNotes, 
    ForEachParagraphCallback forEachParagraph)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| pres | Presentation | Presentation az iterálandó bekezdésekhez |
| includeNotes | Boolean | Jelző, amely jelzi, hogy a NotesSlides legyen-e belefoglalva a feldolgozásba. |
| forEachParagraph | ForEachParagraphCallback | Visszahívás, amely minden bekezdésnél meghívásra kerül |

### Példák

```csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    ForEach.Paragraph(pres, true, (para, slide, index) =>
    {
        System.Console.WriteLine($"{para.Text}, index: {index}");
    });
}        
```

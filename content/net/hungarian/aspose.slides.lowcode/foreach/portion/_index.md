Iterálja a `Portion`-t a [`Presentation`](../../../aspose.slides/presentation)-ben. A `Portion`-ok minden típusú dián iterálva lesznek - [`Slide`](../slide), [`MasterSlide`](../masterslide) és [`LayoutSlide`](../layoutslide)

```csharp
public static void Portion(Presentation pres, ForEachPortionCallback forEachPortion)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| pres | Presentation | Presentation a `Portion`-ok iterálásához |
| forEachPortion | ForEachPortionCallback | Callback, amely minden `Portion`-hoz meghívásra kerül |

### Példák

```csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    ForEach.Portion(pres, (portion, para, slide, index) =>
    {
        System.Console.WriteLine($"{portion.Text}, index: {index}");
    });
}
```

```csharp
public static void Portion(Presentation pres, bool includeNotes, 
    ForEachPortionCallback forEachPortion)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| pres | Presentation | Presentation a `Portion`-ok iterálásához |
| includeNotes | Boolean | Jelző, amely azt jelzi, hogy a NotesSlides fel legyenek véve a feldolgozásba |

### Példák

```csharp

{
    ForEach.Portion(pres, true, (portion, para, slide, index) =>
    {
        System.Console.WriteLine($"{portion.Text}, index: {index}");
    });
}
```

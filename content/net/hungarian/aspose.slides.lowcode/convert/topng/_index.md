## ToPng(Presentation, string) {#topng}

Átalakítja a bemeneti prezentációt PNG formátumú képek sorozatává. Ha a kimeneti fájlnév „myPath/myFilename.png”-ként van megadva, az eredmény „myPath/myFilename_N.png” fájlok sorozataként lesz elmentve, ahol N a dia száma.

```csharp
public static void ToPng(Presentation pres, string outputFileName)
```

| Paraméter | Típus | Leírás |

| pres | Presentation | A bemeneti prezentáció. |

### Példák

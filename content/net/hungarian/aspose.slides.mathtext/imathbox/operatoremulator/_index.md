## IMathBox.OperatorEmulator tulajdonság

Operátor Emulátor. Ha igaz, a keret és tartalma egyetlen operátorként viselkedik, és örökli egy operátor tulajdonságait. Ez azt jelenti, például, hogy a karakter használható sortörés kiindulópontjának, és igazítható más operátorokhoz. Az Operátor Emulátorokat gyakran használják, amikor egy vagy több glif kombinálódik egy operátor létrehozásához, például „==”. Alapértelmezett érték: false

```csharp
public bool OperatorEmulator { get; set; }
```

### Példák

Példa:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.OperatorEmulator = true;
```

## IMathBox.ExplicitBreak tulajdonság

Az Explicit break meghatározza, hogy van-e sortörés a Box objektum kezdetén, úgy hogy a sor a doboz objektum elején törik. Meghatározza a matematikai szöveg előző sorában lévő operátor számát, amely a jelenlegi sor matematikai szövegének igazítási pontjaként szolgál. Lehetséges értékek: 1..255 Alapértelmezett: 0 (no explicit break)

```csharp
public byte ExplicitBreak { get; set; }
```

Példa:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.ExplicitBreak = 1;
```

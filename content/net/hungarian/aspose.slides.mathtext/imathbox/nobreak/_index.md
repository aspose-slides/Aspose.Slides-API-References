## IMathBox.NoBreak tulajdonság

Nincs megtörés. Ez a tulajdonság meghatározza a "unbreakable" tulajdonságot az objektumdobozon. Ha igaz, a dobozon belül nem fordulhat elő sortörés. Ez fontos lehet olyan operátor emulátoroknál, amelyek több mint egy bináris operátort tartalmaznak. Ha ez az elem nincs megadva, a doboz belsejében előfordulhatnak sortörések. Alapértelmezett: true

```csharp
public bool NoBreak { get; set; }
```

## IMathBorderBox.StrikethroughBottomLeftToTopRight tulajdonság

Áthúzás bal alsó saroktól a jobb felső sarokig (alapértelmezett érték hamis). Meghatározza egy átlós áthúzási vonal rejtett vagy látható állapotát, amely a border box bal alsó sarkától a jobb felső sarkáig húzódik.

```csharp
public bool StrikethroughBottomLeftToTopRight { get; set; }
```

Példa:


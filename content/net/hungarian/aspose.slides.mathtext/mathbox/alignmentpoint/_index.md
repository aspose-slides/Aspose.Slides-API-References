## MathBox.AlignmentPoint tulajdonság

Amikor true, ez az operátor emulátor igazítási pontként működik; vagyis a többi egyenletben kijelölt igazítási pontok ehhez igazíthatók. Alapértelmezett: false

public bool AlignmentPoint { get; set; }

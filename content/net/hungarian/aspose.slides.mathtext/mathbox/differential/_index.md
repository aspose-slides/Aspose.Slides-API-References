## MathBox.Differential tulajdonság

Differential Ha igaz, a doboz differenciálként működik (pl., 𝑑𝑥 egy integrandban), és megkapja a megfelelő vízszintes távolságot a matematikai differenciálhoz. Alapértelmezett: false

```csharp
public bool Differential { get; set; }
```

### Példák

Example:

differential.Differential = true;

## MathBox.OperatorEmulator tulajdonság

Operátor emulátor. Ha igaz, a keret és a tartalma egyetlen operátorként viselkedik, és örökli egy operátor tulajdonságait. Ez azt jelenti, például, hogy a karakter használható sorbreak pontként, és igazítható más operátorokhoz. Az operátor emulátorokat gyakran használják, amikor egy vagy több glyph egy operátort alkot, például a '=='. Alapértelmezett érték: false

```csharp
public bool OperatorEmulator { get; set; }
```

### Példák

Példa:

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("=="));
box.OperatorEmulator = true;
```

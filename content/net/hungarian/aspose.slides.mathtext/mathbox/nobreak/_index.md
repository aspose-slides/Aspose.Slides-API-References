## MathBox.NoBreak tulajdonság

Nincs törés. Ez a tulajdonság meghatározza a "unbreakable" tulajdonságot az objektumdobozon. Ha igaz, a dobozon belül nem fordulhat elő sortörés. Ez fontos lehet operátor emulátoroknál, amelyek több, mint egy bináris operátort tartalmaznak. Ha ez az elem nincs megadva, sortörések előfordulhatnak a dobozon belül. Alapértelmezett: true

```csharp
public bool NoBreak { get; set; }
```

## IMathMatrix.MinColumnWidth tulajdonság

A minimum oszlopszélesség twipben (1/20-a a pontnak). A hézag (más néven “Column Gap” vagy “Gap Width”) hozzáadódik a MinColumnWidth-hez, hogy meghatározza a teljes Matrix Column Spacing-et (a különböző oszlopok azonos széleinek távolsága). Alapértelmezett: 0.

```csharp
public uint MinColumnWidth { get; set; }
```

### Példák

Példa:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.MinColumnWidth = 20;
```

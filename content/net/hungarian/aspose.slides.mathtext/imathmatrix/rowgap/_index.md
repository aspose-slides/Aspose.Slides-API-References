## IMathMatrix.RowGap tulajdonság

A mátrix sorai közötti függőleges távolság értéke; ha a RowGapRule 3-ra („Exactly”) van állítva, akkor az egység twipként (egy pont 1/20-a) értelmezhető. Ha a RowGapRule 4-re („Multiple”) van állítva, akkor az egység fél sorokként értelmezhető. Alapértelmezett: 0

```csharp
public uint RowGap { get; set; }
```

### Példák

Példa:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

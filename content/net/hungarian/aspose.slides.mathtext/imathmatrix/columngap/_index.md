## IMathMatrix.ColumnGap tulajdonság

A mátrix oszlopai közötti vízszintes távolság értéke; ha a ColumnGapRule 3-ra ("Exactly") van beállítva, akkor az egység twips (a pont 1/20-a) értelmeződik. Ha a ColumnGapRule 4-re ("Multiple") van beállítva, akkor az egység 0.5 em lépésnek megfelelő számként értelmeződik. Más esetekben figyelmen kívül hagyja. Alapértelmezett: 0

```csharp
public uint ColumnGap { get; set; }
```

### Példák

Példa:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.ColumnGapRule = MathSpacingRules.Exactly;
matrix.ColumnGap = 20;
```

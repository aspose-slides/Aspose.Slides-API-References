## MathMatrix.ColumnGap tulajdonság

A mátrix oszlopai közötti vízszintes távolság értéke; ha a ColumnGapRule értéke 3 ("Exactly"), akkor az egység twipként (a pont 1/20-a) értelmeződik. Ha a ColumnGapRule értéke 4 ("Multiple"), akkor az egység 0,5 em inkrementek számaként értelmeződik. Más esetekben figyelmen kívül marad. Alapértelmezett: 0

```csharp
public uint ColumnGap { get; set; }
```

### Példák

Példa:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.ColumnGapRule = MathSpacingRules.Exactly;
matrix.ColumnGap = 20;
```

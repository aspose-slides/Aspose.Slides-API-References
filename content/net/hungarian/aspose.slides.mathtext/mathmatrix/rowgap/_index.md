## MathMatrix.RowGap tulajdonság

A mátrix sorai közötti függőleges távolság értéke; ha a RowGapRule 3 ("Exactly") értékre van állítva, akkor az egység twipként (a pont 1/20-a) értelmeződik. Ha a RowGapRule 4 ("Multiple") értékre van állítva, akkor az egység félvonalként értelmeződik. Alapértelmezett: 0

```csharp
public uint RowGap { get; set; }
```

### Példák

Example:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

## MathMatrix.MinColumnWidth tulajdonság

A minimum oszlopszélesség twipben (a pont 1/20-a). Az oszloptávolság (más néven “Column Gap” vagy “Gap Width”) hozzáadódik a MinColumnWidth-hez, hogy meghatározza a teljes Matrix oszloptávolságot (a különböző oszlopok ugyanazon széleinek távolsága). Alapértelmezett: 0.

```csharp
public uint MinColumnWidth { get; set; }
```

### Példák

Példa:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.MinColumnWidth = 20;
```

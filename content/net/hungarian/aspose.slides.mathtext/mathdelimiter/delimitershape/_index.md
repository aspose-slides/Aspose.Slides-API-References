## MathDelimiter.DelimiterShape tulajdonság

Megadja a delimiter objektumban a határolók alakját. Amikor a MathDelimiterShape.Centered az érték, a határolók a matematikai szöveg matematika tengelye köré vannak középre helyezve, és úgy vannak méretezve, hogy teljesen illeszkedjenek a tartalmuk magasságához. Amikor a MathDelimiterShape.Match az érték, magasságuk és alakjuk pontosan a tartalmukhoz igazodik.

```csharp
public MathDelimiterShape DelimiterShape { get; set; }
```

### Példák

Példa:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.DelimiterShape = MathDelimiterShape.Match;
```

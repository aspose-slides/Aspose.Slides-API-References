## MathDelimiter.EndingCharacter tulajdonság

A Delimiter Ending Character meghatározza a végző, vagy lezáró, határoló karaktert. A matematikai határolók olyan befoglaló karakterek, mint a zárójelek, szögletes zárójelek és kapcsos zárójelek. Alapértelmezett: ')'.

```csharp
public char EndingCharacter { get; set; }
```

### Példák

Példa:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Join("y").Enclose();
delimiter.EndingCharacter = ']';
```

## MathDelimiter.GrowToMatchOperandHeight tulajdonság

Meghatározza a BeginningCharacter, SeparatorCharacter, EndingCharacter növekedését. Amikor true, a határolók függőlegesen nőnek, hogy megegyezzenek az operandus magasságával. Az alapértelmezett érték true

```csharp
public bool GrowToMatchOperandHeight { get; set; }
```

### Példák

Példa:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.GrowToMatchOperandHeight = false;
```

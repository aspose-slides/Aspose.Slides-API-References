## MathDelimiter.BeginningCharacter tulajdonság

A Delimitátor Kezdő Karakter meghatározza a kezdő vagy nyitó delimitátor karaktert. A matematikai delimitátorok olyan befoglaló karakterek, mint a zárójelek, szögletes zárójelek és kapcsos zárójelek. Az alapértelmezett: '('.

```csharp
public char BeginningCharacter { get; set; }
```

### Példák

Példa:

```csharp

IMathDelimiter delimiter = new MathematicalText("x").Join("y").Enclose();
```

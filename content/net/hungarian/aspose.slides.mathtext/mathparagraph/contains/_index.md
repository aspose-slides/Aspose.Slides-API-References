## MathParagraph.Contains metódus

Megállapítja, hogy a gyűjtemény tartalmaz-e egy konkrét értéket.

```csharp
public bool Contains(IMathBlock mathBlock)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| mathBlock | IMathBlock | Az objektum, amelyet a gyűjteményben keresni kell. |

### Visszatérési érték

igaz, ha a *mathBlock* megtalálható a gyűjteményben; egyébként hamis.

### Példák

Példa:

```csharp
[C#]
IAutoShape shape = slide.Shapes.AddMathShape(x, y, width, height);
IMathParagraph mathParagraph = (shape.TextFrame.Paragraphs[0].Portions[0] as MathPortion).MathParagraph;
IMathBlock block = new MathBlock(new MathematicalText("y"));
mathParagraph.Add(block);
bool contains = mathParagraph.Contains(block);
```

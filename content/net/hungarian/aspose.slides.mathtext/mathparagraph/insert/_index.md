## MathParagraph.Insert metódus

Beszúr egy IMathBlock-ot a gyűjteménybe a megadott indexnél.

```csharp
public void Insert(int index, IMathBlock mathBlock)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| index | Int32 | A nulláról induló index, amelynél az elemet be kell szúrni. |
| mathBlock | IMathBlock | A beszúrandó IMathBlock. |

### Példák

Példa:

```csharp
[C#]
IAutoShape shape = slide.Shapes.AddMathShape(x, y, width, height);
IMathParagraph mathParagraph = (shape.TextFrame.Paragraphs[0].Portions[0] as MathPortion).MathParagraph;
IMathBlock block = new MathBlock(new MathematicalText("y"));
mathParagraph.Insert(0, block);
```

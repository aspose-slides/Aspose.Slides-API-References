## MathBorderBox.StrikethroughTopLeftToBottomRight tulajdonság

Strikethrough Top-Left to Bottom-Right (alapértelmezett értéke false). Megadja a rejtett vagy megjelenített állapotát egy átlós vonalkeresztő vonalnak, amely a bal felső saroktól a jobb alsó sarokig húzódik a border boxban.

```csharp
public bool StrikethroughTopLeftToBottomRight { get; set; }
```

### Példák

Példa:

```csharp
[C#]
MathBorderBox borderBox = new MathBorderBox(new MathematicalText("x"));
borderBox.StrikethroughTopLeftToBottomRight = true;
```

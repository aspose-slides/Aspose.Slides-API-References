## IMathDelimiter.GrowToMatchOperandHeight tulajdonság

Megadja a BeginningCharacter, SeparatorCharacter, EndingCharacter növekedését. Ha true, a határolók függőlegesen nőnek, hogy egyezzenek az operandus magasságával. Az alapértelmezett érték true

```csharp
public bool GrowToMatchOperandHeight { get; set; }
```

### Példák

Példa:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.GrowToMatchOperandHeight = false;
```

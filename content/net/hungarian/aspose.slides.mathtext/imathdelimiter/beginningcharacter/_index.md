## IMathDelimiter.BeginningCharacter tulajdonság

A Delimiter Beginning Character meghatározza a kezdeti vagy nyitó határoló karaktert. A matematikai határolók olyan körülvevő karakterek, mint a zárójelek, szögletes zárójelek és kapcsos zárójelek. Az alapértelmezett érték: '('.

```csharp
public char BeginningCharacter { get; set; }
```

### Példák

Példa:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Join("y").Enclose();
```

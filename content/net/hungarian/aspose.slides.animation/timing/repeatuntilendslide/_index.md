## Timing.RepeatUntilEndSlide tulajdonság

Ez az attribútum meghatározza, hogy a hatás ismétlődik-e a dia végéig. Olvasás/írás Boolean.

```csharp
public bool RepeatUntilEndSlide { get; set; }
```

### Példák

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Az első dia effektus sorozatának lekérése
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // A fő sorozat első effektusának lekérése.
    IEffect effect = effectsSequence[0];

    // Az effektus időzítés/ismétlés beállítása "A dia végéig"
    effect.Timing.RepeatUntilEndSlide = true;
}
```

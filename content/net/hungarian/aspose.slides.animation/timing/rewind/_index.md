## Timing.Rewind tulajdonság

Ez az attribútum megadja, hogy a hatás visszatekerődik-e a lejátszás befejezésekor. Olvasás/írás Bool.

```csharp
public bool Rewind { get; set; }
```

### Példák

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Lekéri az első dia effektus sorozatát
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Lekéri a fő sorozat első effektusát.
    IEffect effect = effectsSequence[0];

    // Bekapcsolja a hatás Timing/Rewind beállítását.
    effect.Timing.Rewind = true;
}
```

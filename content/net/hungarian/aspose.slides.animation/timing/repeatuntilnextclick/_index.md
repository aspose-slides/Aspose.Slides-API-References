## Timing.RepeatUntilNextClick tulajdonság

Ez az attribútum meghatározza, hogy a hatás a következő kattintásig ismétlődik-e. Olvasás/írás Boolean.

```csharp
public bool RepeatUntilNextClick { get; set; }
```

### Példák

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Az első dia hatássorozatát lekéri
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // A fő sorozat első effektusát lekéri.
    IEffect effect = effectsSequence[0];

    // A hatás Timing/Repeat beállítása "Until Next Click"-re
    effect.Timing.RepeatUntilNextClick = true;
}
```

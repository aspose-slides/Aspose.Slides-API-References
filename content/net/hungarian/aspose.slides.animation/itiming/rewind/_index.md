## ITiming.Rewind tulajdonság

Ez az attribútum meghatározza, hogy a hatás lejátszása után visszatekerődik-e. Olvasás/írás Boolean.

```csharp
public bool Rewind { get; set; }
```

### Példák

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Az első dia hatás-sorozatának lekérése
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // A fő sorozat első hatásának lekérése.
    IEffect effect = effectsSequence[0];

    // Az effetk Timing/Rewind bekapcsolása.
    effect.Timing.Rewind = true;
}
```

## ITiming.RepeatUntilEndSlide tulajdonság

Ez az attribútum meghatározza, hogy a hatás a dia végéig ismétlődik-e. Olvasás/írás Boolean.

```csharp
public bool RepeatUntilEndSlide { get; set; }
```

### Példák

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Az első dia hatásainak sorozatát kapja meg
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // A fő sorozat első hatását kapja meg.
    IEffect effect = effectsSequence[0];

    // Állítsa be a hatás Timing/Repeat értékét "Until End of Slide"-re
    effect.Timing.RepeatUntilEndSlide = true;
}
```

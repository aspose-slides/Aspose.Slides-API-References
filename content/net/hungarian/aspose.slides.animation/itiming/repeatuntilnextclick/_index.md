## ITiming.RepeatUntilNextClick tulajdonság

Ez a attribútum meghatározza, hogy a hatás a következő kattintásig ismétlődjön-e. Olvasás/írás Boolean.

```csharp
public bool RepeatUntilNextClick { get; set; }
```

### Példák

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Az első dia effektussorozatának lekérése
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // A fő sorozat első effektusának lekérése.
    IEffect effect = effectsSequence[0];

    // Az effektus időzítése/ismétlése módosítása "Until Next Click" értékre
    effect.Timing.RepeatUntilNextClick = true;
}
```

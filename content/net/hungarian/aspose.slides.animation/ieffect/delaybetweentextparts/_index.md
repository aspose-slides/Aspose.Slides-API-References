## IEffect.DelayBetweenTextParts tulajdonság

Meghatározza a késleltetést az animált szövegrészek (szavak vagy betűk) között. A pozitív érték a hatás időtartamának százalékát adja meg. A negatív érték a késleltetést másodpercben adja meg. Olvasás/írás Single.

```csharp
public float DelayBetweenTextParts { get; set; }
```

### Példák

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Az első dia első effektusát lekéri.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Megváltoztatja az effektus Animate text típusát "By word" értékre.
    firstSlideEffect.AnimateTextType = AnimateTextType.ByWord;
    
    // Beállítja a késleltetést az animált szövegrészek között a hatás időtartamának 20%-ára.
    firstSlideEffect.DelayBetweenTextParts = 20f;
}
```

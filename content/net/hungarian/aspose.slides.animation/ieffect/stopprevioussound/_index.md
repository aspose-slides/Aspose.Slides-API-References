## IEffect.StopPreviousSound tulajdonság

Ez az attribútum meghatározza, hogy az animációs effektus megállítja-e az előző hangot. Olvasás/írás Boolean.

```csharp
public bool StopPreviousSound { get; set; }
```

### Példák

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Az első dia első effektusát kapja meg.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // A második dia első effektusát kapja meg.
    IEffect secondSlideEffect = presentation.Slides[1].Timeline.MainSequence[0];
       
    if (firstSlideEffect.Sound != null)
    {
        // A második effektus Enhancements/Sound beállítása "Stop Previous Sound" értékre.
        secondSlideEffect.StopPreviousSound = true;
    }
}
```

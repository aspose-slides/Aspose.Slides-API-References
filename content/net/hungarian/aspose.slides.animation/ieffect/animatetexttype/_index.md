## IEffect.AnimateTextType tulajdonság

Az effektus animált szövegtípusa. A forma szövege betűnként, szónként vagy egyszerre animálható. Olvasás/írás `AnimateTextType`.

```csharp
public AnimateTextType AnimateTextType { get; set; }
```

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
```

## BehaviorPropertyCollection.CopyTo metódus

Átmásolja az ICollection elemeit egy Array-be, egy adott Array indexnél kezdve.

```csharp
public void CopyTo(IBehaviorProperty[] array, int arrayIndex)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| array | IBehaviorProperty[] | Az egydimenziós Array, amely a ICollection elemeiből másolt elemek célja. Az Array-nek nulláról induló indexelése kell, hogy legyen. |
| arrayIndex | Int32 | A nulláról induló index a *array*-ban, ahol a másolás kezdődik. |

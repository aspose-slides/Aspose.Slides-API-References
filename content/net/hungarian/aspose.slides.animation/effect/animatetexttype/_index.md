## Effect.AnimateTextType tulajdonság

Meghatároz egy animált szövegtípust a hatáshoz. A forma szövege betűnként, szónként vagy egyszerre animálható. Olvasás/írás `AnimateTextType`.

```csharp
public AnimateTextType AnimateTextType { get; set; }
```

### Példák

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Szerezze meg az első dia első hatását.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Módosítsa a hatás animált szövegtípusát "By letter"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByLetter;
}
```

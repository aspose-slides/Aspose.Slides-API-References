## Effect.DelayBetweenTextParts tulajdonság

Meghatároz egy késleltetést az animált szövegrészek (szavak vagy betűk) között. A pozitív érték a hatás időtartamának százalékát adja meg. A negatív érték a késleltetést másodpercben adja meg. Olvasás/írás Single.

```csharp
public float DelayBetweenTextParts { get; set; }
```

### Példák

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Az első dia első effektusát kapja.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Átállítja az effektus AnimateText típusát "By word" értékre.
    firstSlideEffect.AnimateTextType = AnimateTextType.ByWord;
    
    // Az animált szövegrészek közötti késleltetést az effektus időtartamának 20%-ára állítja.
    firstSlideEffect.DelayBetweenTextParts = 20f;
}
```

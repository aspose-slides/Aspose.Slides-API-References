## BaseOverrideThemeManager.IsOverrideThemeEnabled tulajdonság

Megállapítja, hogy az OverrideTheme felülírja-e az örökölt hatékony témát, vagy sem. Az OverrideTheme felülírásának engedélyezéséhez használja az OverrideTheme.Init*() metódusokat. Az OverrideTheme felülírásának letiltásához használja az OverrideTheme.Clear() metódust. Csak olvasható Boolean.

```csharp
public bool IsOverrideThemeEnabled { get; }
```

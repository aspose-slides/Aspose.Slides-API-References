## IInkOptions.InterpretMaskOpAsOpacity tulajdonság

ROP műveletet vagy Opacity-t használ a ecset rendereléséhez.

```csharp
public bool InterpretMaskOpAsOpacity { get; set; }
```

### Megjegyzés

Az alapértelmezett érték true.

### Példák

A következő példa bemutatja, hogyan állítható be a ROP használatával az Ink elemek exportálásához:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.InterpretMaskOpAsOpacity = false;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

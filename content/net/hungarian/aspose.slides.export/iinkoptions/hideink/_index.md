## IInkOptions.HideInk tulajdonság

Megjeleníti vagy elrejti az Ink elemeket az exportált dokumentumban.

```csharp
public bool HideInk { get; set; }
```

### Megjegyzés

Alapértelmezett érték false.

### Példák

A következő példa bemutatja, hogyan lehet elrejteni az Ink elemeket az exportált PDF dokumentumban:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.HideInk = true;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

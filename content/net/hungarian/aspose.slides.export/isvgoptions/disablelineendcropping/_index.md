## ISVGOptions.DisableLineEndCropping tulajdonság

Az SVG 1.1 nem támogatja a jelölők belső margóinak meghatározását. Az Aspose.Slides SVG író motorja rendelkezik megoldással erre a problémára: levágja a vonal végét a nyíllal, így a vonal nem fed le jelölőket. Ez a beállítás kikapcsolja ezt a viselkedést. Olvasás/írás Boolean.

```csharp
public bool DisableLineEndCropping { get; set; }
```

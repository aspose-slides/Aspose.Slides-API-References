## RenderingOptions.DisableFontLigatures tulajdonság

Értéket ad vissza vagy állít be, amely azt jelzi, hogy a szöveg ligatúrák használata nélkül van-e megjelenítve. Ha `true`-ra van állítva, a ligatúrák le lesznek tiltva a megjelenített kimenetben. Alapértelmezés szerint ez a tulajdonság `false` értékre van beállítva.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Példák

Példa:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    RenderingOptions options = new RenderingOptions
    {
        DisableFontLigatures = true // Ligatúrák letiltása a szöveg megjelenítésekor
    };
    
    Bitmap[] renderedSlides = pres.GetImage(options);
    for (var index = 0; index < renderedSlides.Length; index++)
    {
        var slideImage = renderedSlides[index];
        slideImage.Save($"slide-{index}.png");
    }
}
```

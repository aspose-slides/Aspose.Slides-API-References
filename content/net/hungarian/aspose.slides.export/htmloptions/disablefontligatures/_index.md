## HtmlOptions.DisableFontLigatures tulajdonság

Értéket ad vissza vagy állít be, amely azt jelzi, hogy a szöveg ligaturek használata nélkül kerül-e megjelenítésre. Ha `true`-ra van beállítva, a ligaturek le lesznek tiltva a megjelenített kimenetben. Alapértelmezés szerint ez a tulajdonság `false` értékre van beállítva.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Példák

Példa:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    HtmlOptions options = new HtmlOptions
    {
        DisableFontLigatures = true // Letiltja a ligatureket a szöveg megjelenítésében
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html, options);
}
```

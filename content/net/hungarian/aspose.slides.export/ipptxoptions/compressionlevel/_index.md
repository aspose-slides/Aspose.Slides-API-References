## IPptxOptions.CompressionLevel tulajdonság

Megadja a bemutató dokumentum mentésekor használt tömörítési szintet. Az alapértelmezett érték a Level6.

```csharp
public CompressionLevel CompressionLevel { get; set; }
```

### Megjegyzés

A magasabb tömörítési szintek kisebb fájlokat eredményeznek, de több feldolgozási időt igényelnek. A tényleges tömörítési arány a bemutató tartalmától függ.

### Példák

Példa:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("demo-level8.pptx", SaveFormat.Pptx, new PptxOptions()
    {
        CompressionLevel = CompressionLevel.Level8
    });
}
```

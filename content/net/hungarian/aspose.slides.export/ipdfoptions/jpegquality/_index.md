## IPdfOptions.JpegQuality tulajdonság

Visszaad vagy beállít egy értéket, amely meghatározza a JPEG képek minőségét a PDF dokumentumban. Olvasás/írás Byte.

```csharp
public byte JpegQuality { get; set; }
```

### Megjegyzés

Csak akkor van hatása, ha a dokumentum JPEG képeket tartalmaz.

Használja ezt a tulajdonságot a képek minőségének lekérdezéséhez vagy beállításához egy dokumentumban PDF formátumba mentéskor. Az érték 0 és 100 között változhat, ahol a 0 legrosszabb minőséget, de legnagyobb tömörítést jelent, a 100 pedig a legjobb minőséget, de legkisebb tömörítést.

Az alapértelmezett érték **100**.

## SVGOptions.DisableFontLigatures tulajdonság

Értéket ad vissza vagy állít be egy értéket, amely jelzi, hogy a szöveget ligatúrák használata nélkül jelenítik meg. Ha `true`-ra van beállítva, a ligatúrák le lesznek tiltva a megjelenített kimenetben. Alapértelmezés szerint ez a tulajdonság `false` értékre van beállítva.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Példák

Példa:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    SVGOptions options = new SVFOptions
    {
        DisableFontLigatures = true // Ligatúrák letiltása a szöveg renderelésében
    };
    
    using (FileStream fileStream = new FileStream("slide-0.svg", FileMode.Create, FileAccess.Write))
    {
        pres.Slides[0].WriteAsSvg(fileStream);   
    }
}
```

## SVGOptions.DisableLineEndCropping tulajdonság

Az SVG 1.1 nem rendelkezik a jelzők számára beállítható belső margók definiálásának képességével. Az Aspose.Slides SVG író motorja megoldást kínál erre a problémára: levágja a vonal végét a nyíllal, így a vonal nem fed át a jelzőket. Ez a beállítás kikapcsolja ezt a viselkedést. Olvasás/írás Boolean.

```csharp
public bool DisableLineEndCropping { get; set; }
```

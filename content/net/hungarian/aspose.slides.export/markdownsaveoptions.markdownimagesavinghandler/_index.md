## MarkdownSaveOptions.MarkdownImageSavingHandler delegate

Meghívás minden nem SVG kép (bitmap vagy metafájl) esetén a Markdown exportálása során. `true` visszaadása esetén a megadott *hivatkozást* használja, `false` esetén az alapértelmezett mentési logikát alkalmazza.

public delegate bool MarkdownImageSavingHandler(IImage image, ImageFormat format, ref string link);

| format | ImageFormat | A kép formátuma. |
| link | String& | A Markdown hivatkozás, amelyet a `true` visszaadásakor kell használni. |

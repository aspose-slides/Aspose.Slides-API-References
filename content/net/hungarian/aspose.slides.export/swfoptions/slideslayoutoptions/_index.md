## SwfOptions.SlidesLayoutOptions tulajdonság

Megkapja vagy beállítja azt a módot, ahogyan a diák az oldalon kerülnek elhelyezésre egy prezentáció exportálásakor [`ISlidesLayoutOptions`](../../islideslayoutoptions). Ez a tulajdonság nem támogatja az adott típusú objektumok hozzárendelését [`HandoutLayoutingOptions`](../../handoutlayoutingoptions)

```csharp
public ISlidesLayoutOptions SlidesLayoutOptions { get; set; }
```

### Példák

Példa:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    SwfOptions options = new SwfOptions
    {
        SlidesLayoutOptions = new NotesCommentsLayoutingOptions
        {
            CommentsPosition = CommentsPositions.Right
        }
    };
    
    pres.Save("pres.swf", SaveFormat.Swf, options);
}
```

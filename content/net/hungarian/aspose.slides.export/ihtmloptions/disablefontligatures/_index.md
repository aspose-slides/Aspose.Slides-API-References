## IHtmlOptions.DisableFontLigatures tulajdonság

Lekéri vagy beállít egy értéket, amely meghatározza, hogy a szöveg ligatúrák használata nélkül legyen renderelve. Ha `true` értékre van beállítva, a ligatúrák le lesznek tiltva a renderelt kimenetben. Alapértelmezés szerint ez a tulajdonság `false` értékre van állítva.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Példák

Példa:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    HtmlOptions options = new HtmlOptions
    {
        DisableFontLigatures = true // Ligatúrák letiltása a szöveg megjelenítésében
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html, options);
}
```

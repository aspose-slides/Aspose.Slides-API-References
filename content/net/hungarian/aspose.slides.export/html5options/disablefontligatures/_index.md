## Html5Options.DisableFontLigatures tulajdonság

Lekéri vagy beállítja azt az értéket, amely azt jelzi, hogy a szöveg ligatúrák használata nélkül jelenik-e meg. Ha `true` értékre van állítva, a ligatúrák le lesznek tiltva a megjelenített kimenetben. Alapértelmezés szerint ez a tulajdonság `false` értékre van beállítva.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Példák

Példa:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    Html5Options options = new Html5Options
    {
        DisableFontLigatures = true // Letiltja a ligatúrákat a szöveg megjelenítésében
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html5, options);
}
```

## IPptOptions.RootDirectoryClsid tulajdonság

A gyökérkönyvtár bejegyzésben tárolt objektumosztály GUID (CLSID) értékét képviseli. A dokumentum alkalmazásának COM aktiválásához használható. Az alapértelmezett érték '64818D11-4F9B-11CF-86EA-00AA00B929E8', amely a 'Microsoft Powerpoint.Slide.8' értéknek felel meg.

```csharp
public Guid RootDirectoryClsid { get; set; }
```

### Példák

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    PptOptions pptOptions = new PptOptions();
    
    /// állítsa be a CLSID-et a 'Microsoft Powerpoint.Show.8' értékre
    pptOptions.RootDirectoryClsid = new Guid("64818D10-4F9B-11CF-86EA-00AA00B929E8");
    
    pres.Save("pres.ppt", SaveFormat.Ppt, pptOptions);
}
```

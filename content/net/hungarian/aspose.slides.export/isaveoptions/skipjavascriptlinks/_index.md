## ISaveOptions.SkipJavaScriptLinks tulajdonság

Meghatározza, hogy a prezentáció mentésekor kihagyjuk-e a JavaScript hívásokat tartalmazó hiperhivatkozásokat. Olvasási/írási Boolean. Az alapértelmezett érték **false**.

```csharp
public bool SkipJavaScriptLinks { get; set; }
```

### Megjegyzések

Ha ez a tulajdonság **true**-ra van állítva, a JavaScript hívásokat tartalmazó hiperhivatkozások mentés közben figyelmen kívül lesznek hagyva.

Ha ez a tulajdonság **false**-ra van állítva, minden hiperhivatkozás el lesz mentve.

### Példák

Példa:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("result_without_JavaScript_links.html", SaveFormat.Html, new HtmlOptions()
    {
        SkipJavaScriptLinks = true
    });
}
```

## PdfOptions.EmbedTrueTypeFontsForASCII tulajdonság

Meghatározza, hogy az Aspose.Slides beágyazza-e a közös betűtípusokat az ASCII (33..127 kódtartomány) szöveghez. A 127-nél nagyobb karakterkódokhoz a betűtípusok mindig beágyazottak. A közös betűtípusok listája tartalmazza a PDF alap 14 betűtípusát és a felhasználó által megadott további betűtípusokat. Olvasás/írás Boolean.

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### Megjegyzés

Az alapértelmezett érték **true**.

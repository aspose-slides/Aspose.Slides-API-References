## SaveOptions.SkipJavaScriptLinks tulajdonság

Megadja, hogy a bemutató mentésekor kihagyja-e a JavaScript-hívásokat tartalmazó hiperhivatkozásokat. Olvasás/írás Boolean. Az alapértelmezett érték **false**.

```csharp
public bool SkipJavaScriptLinks { get; set; }
```

### Megjegyzések

Ha ez a tulajdonság **true** értékre van állítva, a JavaScript-hívásokat tartalmazó hiperhivatkozások a mentés során figyelmen kívül lesznek hagyva.

Ha ez a tulajdonság **false** értékre van állítva, minden hiperhivatkozás mentésre kerül.

### Példák

Példa:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("result_without_JavaScript_links.html", SaveFormat.Html, new HtmlOptions()
    {
        SkipJavaScriptLinks = true
    });
}
```

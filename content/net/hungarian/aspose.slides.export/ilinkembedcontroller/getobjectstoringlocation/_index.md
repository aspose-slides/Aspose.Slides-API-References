## ILinkEmbedController.GetObjectStoringLocation method

Meghatározza, hogy az objektum hol legyen tárolva. Ez a metódus minden egyes objektum-azonosítóhoz egyszer kerül meghívásra. Nem garantált, hogy nem lesz két objektum ugyanazzal a data, semanticName és contentType értékkel, de különböző azonosítóval.

```csharp
public LinkEmbedDecision GetObjectStoringLocation(int id, byte[] entityData, string semanticName, 
    string contentType, string recomendedExtension)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| id | Int32 | Objektum azonosítója. Ez az azonosító a mentési művelet során egyedi. |
| entityData | Byte[] | Objektum bináris adata. Ez a paraméter lehet null értékű, ha az objektum bináris adatai még nem kerültek előállításra. |
| semanticName | String | Rövid szöveg, amely leírja az objektum jelentését. A vezérlő használhatja ezt a külső objektumnév részeként, de a feladattételre bízzuk, hogy a nevek egyediek legyenek és csak a megengedett karaktereket tartalmazzák. |
| contentType | String | Az objektum MIME típusa. |
| recomendedExtension | String | Fájlnév kiterjesztés, amely ebben a MIME típusban ajánlott. |

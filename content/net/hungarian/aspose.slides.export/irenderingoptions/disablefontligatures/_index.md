## IRenderingOptions.DisableFontLigatures tulajdonság

Értéket ad vissza vagy állít be, amely azt jelzi, hogy a szöveg ligatúrák használata nélkül jelenik meg. `true` értékre beállítva a ligatúrák le lesznek tiltva a renderelt kimenetben. Alapértelmezés szerint ez a tulajdonság `false` értékre van állítva.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Példák

Példa:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    RenderingOptions options = new RenderingOptions
    {
        DisableFontLigatures = true // Ligatúrák letiltása a szöveg renderelésekor
    };
    
    Bitmap[] renderedSlides = pres.GetImage(options);
    for (var index = 0; index < renderedSlides.Length; index++)
    {
        var slideImage = renderedSlides[index];
        slideImage.Save($"slide-{index}.png");
    }
}
```

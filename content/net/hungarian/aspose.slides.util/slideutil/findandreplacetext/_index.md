## SlideUtil.FindAndReplaceText metódus

Megkeresi és helyettesíti a szöveget a prezentációban a megadott formátummal.

```csharp
public static void FindAndReplaceText(IPresentation presentation, bool withMasters, string find, 
    string replace, PortionFormat format = null)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| presentation | IPresentation | Beolvasott prezentáció. |
| withMasters | Boolean | Megállapítja, hogy a mesterdiák be legyenek-e olvasva. |
| find | String | A keresendő karakterlánc értéke. |
| replace | String | A helyettesítendő karakterlánc értéke. |
| format | PortionFormat | A szövegrész helyettesítéséhez használandó formátum. Ha null, akkor a megtalált karakterlánc első karakterének formátuma lesz használva. |

### Példák

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PortionFormat format = new PortionFormat
    {
        FontHeight = 24f,
        FontItalic = NullableBool.True,
        FillFormat =
        {
            FillType = FillType.Solid,
            SolidFillColor =
            {
                Color = Color.Red
            }
        }
    };
    Aspose.Slides.Util.SlideUtil.FindAndReplaceText(pres, true, "[this block] ", "my text ", format);
    pres.Save("replaced", SaveFormat.Pptx);
}
```

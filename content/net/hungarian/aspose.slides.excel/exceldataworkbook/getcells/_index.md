## ExcelDataWorkbook.GetCells metódus

A munkafüzetből a megadott képlettel egyező cellák gyűjteményét adja vissza.

```csharp
public ReadOnlyCollection<IExcelDataCell> GetCells(string formula, bool skipHiddenCells)
```

| Paraméter | Típus | Leírás |
| --- | --- | --- |
| formula | String | Képlet vagy tartománykifejezés (például "Sheet1!A1:B3"), amely a célcellákat azonosítja. |
| skipHiddenCells | Boolean | Ha `true`, a rejtett cellák (például rejtett sorokban vagy oszlopokban) kizárásra kerülnek az eredményből. |

### Visszatérési érték

A megadott képlettel egyező cellák írásvédett listája.

### Példák

Példa:

```csharp
[C#]
ExcelDataWorkbook wb = new ExcelDataWorkbook(testFile);
IReadOnlyList<IExcelDataCell> cells = wb.GetCells("Sheet1!A2:A6", false);
Console.WriteLine(cells.Count); //Kimenet: 5
```

## Свойство Timing.RepeatUntilNextClick

Этот атрибут указывает, будет ли эффект повторяться до следующего щелчка. Читаемое/Записываемое логическое значение.

```csharp
public bool RepeatUntilNextClick { get; set; }
```

### Примеры

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Получите последовательность эффектов для первого слайда
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Получите первый эффект основной последовательности.
    IEffect effect = effectsSequence[0];

    // Измените время эффекта/Повтор на "До следующего щелчка"
    effect.Timing.RepeatUntilNextClick = true;
}
```

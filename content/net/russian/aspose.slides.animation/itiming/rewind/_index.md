## ITiming.Rewind свойство

Этот атрибут указывает, будет ли эффект перематываться по завершении воспроизведения. Читаемый/записываемый логический.

```csharp
public bool Rewind { get; set; }
```

### Примеры

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Получить последовательность эффектов для первого слайда
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Получить первый эффект основной последовательности.
    IEffect effect = effectsSequence[0];

    // Включить перемотку эффекта Timing/Rewind.
    effect.Timing.Rewind = true;
}
```

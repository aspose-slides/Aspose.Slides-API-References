## Свойство Effect.AnimateTextType

Определяет тип анимации текста для эффекта. Текст формы можно анимировать по буквам, по словам или целиком. Читайте/пишите `AnimateTextType`.

```csharp
public AnimateTextType AnimateTextType { get; set; }
```

### Примеры

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Получите первый эффект первого слайда.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Измените тип анимации текста эффекта на "По буквам"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByLetter;
}
```

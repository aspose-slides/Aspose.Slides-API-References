## Свойство Effect.DelayBetweenTextParts

Определяет задержку между анимированными текстовыми частями (словами или буквами). Положительное значение указывает на процент продолжительности эффекта. Отрицательное значение указывает на задержку в секундах. Читаемое/записываемое значение типа Single.

```csharp
public float DelayBetweenTextParts { get; set; }
```

### Примеры

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Получить первый эффект первого слайда.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Изменить тип анимации текста эффекта на "По словам"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByWord;
    
    // Установить задержку между анимированными текстовыми частями на 20% от продолжительности эффекта.
    firstSlideEffect.DelayBetweenTextParts = 20f;
}
```

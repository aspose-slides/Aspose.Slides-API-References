## Свойство MathArray.BaseJustification

Указывает выравнивание массива относительно окружающего текста. Текст за пределами массива может быть выровнен по нижней, верхней части или центру объекта массива. Значение по умолчанию: Center

```csharp
public MathVerticalAlignment BaseJustification { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
IMathArray mathArray = new MathArray(new MathematicalText("item1"));
mathArray.BaseJustification = MathVerticalAlignment.Top;
```

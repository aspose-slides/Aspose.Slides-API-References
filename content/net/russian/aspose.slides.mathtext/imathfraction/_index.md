## Интерфейс IMathFraction

Указывает на объект дроби, состоящий из числителя и знаменателя, разделённых дробной чертой. Дробная черта может быть горизонтальной или диагональной в зависимости от свойств дроби. Объект дроби также используется для представления функции стек, которая размещает один элемент над другим без дробной черты.

```csharp
public interface IMathFraction : IMathElement
```

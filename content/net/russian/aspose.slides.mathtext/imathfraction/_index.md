## IMathFraction интерфейс

Определяет объект дроби, состоящий из числителя и знаменателя, разделённых чертой дроби. Черта дроби может быть горизонтальной или диагональной, в зависимости от свойств дроби. Объект дроби также используется для представления функции стэка, которая помещает один элемент над другим без черты дроби.

public interface IMathFraction : IMathElement

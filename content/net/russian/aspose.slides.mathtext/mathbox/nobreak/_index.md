## MathBox.NoBreak свойство

No break Это свойство определяет свойство "неразрывной" на объекте box. Когда это значение истинно, в пределах box не могут происходить переносы строк. Это может быть важно для эмуляторов операторов, состоящих из более чем одного двоичного оператора. Когда этот элемент не указан, внутри box могут происходить переносы. По умолчанию: true

```csharp
public bool NoBreak { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("*****"));
box.NoBreak = false;
```

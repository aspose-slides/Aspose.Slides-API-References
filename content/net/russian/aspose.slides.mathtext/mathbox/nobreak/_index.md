## MathBox.NoBreak property

No break Эта свойство указывает на "неразрывную" свойство для объекта box. Когда это значение true, разрывы строк не могут происходить внутри box. Это может быть важно для эмуляторов операторов, состоящих из более чем одного бинарного оператора. Когда этот элемент не указан, разрывы могут происходить внутри box. По умолчанию: true

```csharp
public bool NoBreak { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("*****"));
box.NoBreak = false;
```

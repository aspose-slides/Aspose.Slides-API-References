## MathBox.OperatorEmulator свойство

Эмулятор оператора. Если установлено значение true, то коробка и ее содержимое ведут себя как один оператор и унаследуют свойства оператора. Это означает, например, что символ может служить точкой для перехода на новую строку и может быть выровнен относительно других операторов. Эмуляторы операторов часто используются, когда один или несколько глифов объединяются для формирования оператора, такого как '=='. Значение по умолчанию: false

```csharp
public bool OperatorEmulator { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("=="));
box.OperatorEmulator = true;
```

## Свойство MathBox.ExplicitBreak

Явный разрыв указывает, есть ли разрыв строки в начале объекта Box, таким образом, строка оборачивается в начале объекта box. Указывает номер оператора на предыдущей строке математического текста, который будет использоваться в качестве точки выравнивания для текущей строки математического текста, возможные значения: 1..255 По умолчанию: 0 (без явного разрыва)

```csharp
public byte ExplicitBreak { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.ExplicitBreak = 1;
```

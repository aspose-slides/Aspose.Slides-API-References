## MathBox.ExplicitBreak свойство

Явный перенос определяет, есть ли перенос строки в начале объекта Box, так что строка оборачивается в начале объекта box. Указывает номер оператора на предыдущей строке математического текста, который будет использоваться как точка выравнивания для текущей строки математического текста. Возможные значения: 1..255. По умолчанию: 0 (без явного переноса)

```csharp
public byte ExplicitBreak { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.ExplicitBreak = 1;
```

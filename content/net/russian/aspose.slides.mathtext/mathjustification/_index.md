## Перечисление MathJustification

Указывает выравнивание математического абзаца (серия соседних экземпляров математического текста в одном абзаце)

```csharp
public enum MathJustification
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| LeftJustified | `1` | Выравнивание математического текста по левому краю |
| RightJustified | `2` | Выравнивание математического текста по правому краю |
| Centered | `3` | Выравнивание математического текста по центру |
| CenteredAsGroup | `4` | Центрирование всей группы математических блоков на странице как единого целого. Это означает, что экземпляры математического текста могут быть выровнены по отношению друг к другу, но вся группа математического текста центрирована целиком. |

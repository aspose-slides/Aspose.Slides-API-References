## перечисление MathJustification

Указывает выравнивание математического абзаца (серии смежных экземпляров математического текста в одном абзаце)

```csharp
public enum MathJustification
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| LeftJustified | `1` | Выровнять математический текст по левому краю |
| RightJustified | `2` | Выровнять математический текст по правому краю |
| Centered | `3` | Выровнять математический текст по центру |
| CenteredAsGroup | `4` | Разместить по центру страницы всю группу блоков с математикой как единый блок. Это означает, что отдельные фрагменты математического текста могут быть выровнены друг относительно друга, но вся группа математического текста центрируется целиком. |

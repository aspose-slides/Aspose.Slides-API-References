## Метод IMathBlockCollection.RemoveAt

Удаляет элемент по указанному индексу коллекции.

```csharp
public void RemoveAt(int index)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| index | Int32 | Индекс элемента, который нужно удалить, начиная с нуля. |


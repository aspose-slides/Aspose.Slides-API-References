## Индексатор IMathBlockCollection

Получает элемент по указанному индексу. Только для чтения [`IMathBlock`](../../imathblock).

```csharp
public IMathBlock this[int index] { get; set; }
```

| Параметр | Описание |
| --- | --- |
| index | Нулевой индекс элемента, который нужно получить |

### Возвращаемое значение

Блок математического текста.

### Примеры

Пример:

```csharp
[C#]
IMathBlockCollection blockCollection = new MathParagraph();
blockCollection.Add(new MathBlock(new MathematicalText("block1")));
blockCollection.Add(new MathBlock(new MathematicalText("block2")));
IMathBlock block = blockCollection[1];
```

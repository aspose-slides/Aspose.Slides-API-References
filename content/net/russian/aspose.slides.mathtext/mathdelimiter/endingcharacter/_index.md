## MathDelimiter.EndingCharacter свойство

Символ завершающего разделителя указывает завершающий или закрывающий символ разделителя. Математические разделители - это охватывающие символы, такие как круглые скобки, квадратные скобки и фигурные скобки. По умолчанию: ')'.

```csharp
public char EndingCharacter { get; set; }
```

Пример:

```csharp
[C#]
```

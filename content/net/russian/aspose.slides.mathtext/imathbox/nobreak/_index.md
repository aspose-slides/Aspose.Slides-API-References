## IMathBox.NoBreak свойство

Без разрыва. Это свойство указывает на "несмещаемое" свойство объекта box. Когда установлено в true, в пределах box не могут происходить разрывы строк. Это может быть важно для эмуляторов операторов, состоящих из более чем одного бинарного оператора. Когда этот элемент не указан, внутри box могут происходить разрывы. По умолчанию: true

```csharp
public bool NoBreak { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
IMathBox box = new MathematicalText("**********").ToBox();
box.NoBreak = false;
```

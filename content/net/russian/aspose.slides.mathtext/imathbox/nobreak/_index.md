## IMathBox.NoBreak property

Нет разрыва. Это свойство указывает на "нерушимое" свойство объекта box. Когда установлено в true, разрывы строк не могут произойти внутри box. Это может быть важно для эмуляторов операторов, состоящих из более чем одного двоичного оператора. Когда этот элемент не указан, разрывы могут происходить внутри box. По умолчанию: true

```csharp
public bool NoBreak { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
IMathBox box = new MathematicalText("**********").ToBox();
box.NoBreak = false;
```

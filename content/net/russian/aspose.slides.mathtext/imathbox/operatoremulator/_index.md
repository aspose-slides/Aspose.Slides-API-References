## IMathBox.OperatorEmulator свойство

Эмулятор оператора. Когда значение истинно, блок и его содержимое ведут себя как единый оператор и наследуют свойства оператора. Это означает, что, например, символ может служить точкой разрыва строки и может выравниваться по другим операторам. Эмуляторы операторов часто используются, когда один или несколько глифов объединяются, чтобы образовать оператор, такой как '=='. Значение по умолчанию: false

```csharp
public bool OperatorEmulator { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.OperatorEmulator = true;
```

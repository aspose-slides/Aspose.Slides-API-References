## IMathBox.OperatorEmulator свойство

Эмулятор оператора. Когда это значение истинно, рамка и её содержимое ведут себя как один оператор и наследуют свойства оператора. Это означает, что, например, символ может служить точкой для разрыва строки и может быть выровнен относительно других операторов. Эмуляторы операторов часто используются, когда один или несколько глифов объединяются, чтобы сформировать оператор, такой как '=='. Значение по умолчанию: false

```csharp
public bool OperatorEmulator { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.OperatorEmulator = true;
```

## MathElementBase.ToBox метод

Размещает этот элемент в невизуальной коробке (логическая группа), которая используется для группировки компонентов уравнения или другого экземпляра математического текста. Объект в коробке может (например) служить эмулятором оператора с или без точки выравнивания, служить точкой разрыва строки или быть сгруппирован так, чтобы не допускать разрывы строки внутри.

```csharp
public IMathBox ToBox()
```

### Возвращаемое значение

Логическая коробка с этим элементом внутри

### Примеры

Пример:

```csharp
[C#]
IMathBox box = new MathematicalText("x:=y").ToBox();
```

## MathGroupingCharacter.VerticalJustification свойство

Вертикальное выравнивание группового символа. Указывает выравнивание объекта относительно базовой линии. Например, когда групповой символ находится выше объекта, VerticalJustification равное Top означает, что верхняя часть объекта попадает на базовую линию; когда VerticalJustification установлено на Bottom, нижняя часть объекта находится на базовой линии. По умолчанию: Bottom для Position=Top и Top для Position=Bottom

```csharp
public MathTopBotPositions VerticalJustification { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
MathGroupingCharacter groupingCharacter = new MathGroupingCharacter(new MathematicalText("abc"));
groupingCharacter.VerticalJustification = MathTopBotPositions.Top;
```

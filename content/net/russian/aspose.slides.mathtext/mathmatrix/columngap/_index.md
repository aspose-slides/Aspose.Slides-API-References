## MathMatrix.ColumnGap свойство  

Значение горизонтального расстояния между колонками матрицы; Если ColumnGapRule установлено на 3 ("Точно"), то единица измерения интерпретируется как твипы (1/20 точки). Если ColumnGapRule установлено на 4 ("Множественное"), то единица измерения интерпретируется как количество увеличений на 0.5 em. В других случаях игнорируется. По умолчанию: 0  

```csharp  
public uint ColumnGap { get; set; }  
```  

### Примеры  

Пример:  

```csharp  
[C#]  
IMathMatrix matrix = new MathMatrix(2, 3);  
matrix.ColumnGapRule = MathSpacingRules.Exactly;  
matrix.ColumnGap = 20;  
```

## MathMatrix.MinColumnWidth свойство

Минимальная ширина колонки в твипсах (1/20 пункта). Промежуток (также известный как "Промежуток Колонки" или "Ширина Промежутка") добавляется к MinColumnWidth для определения общего расстояния Промежутка Колонки Матрицы (расстояние между одинаковыми краями разных колонок). По умолчанию: 0.

```csharp
public uint MinColumnWidth { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.MinColumnWidth = 20;
```

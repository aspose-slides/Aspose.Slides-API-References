## MathMatrix.RowGap свойство

Значение вертикального расстояния между рядами матрицы; Если RowGapRule установлено в 3 ("Точно"), то единица интерпретируется как твипы (1/20 пункта). Если RowGapRule установлено в 4 ("Множитель"), то единица интерпретируется как половина строк. По умолчанию: 0

```csharp
public uint RowGap { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

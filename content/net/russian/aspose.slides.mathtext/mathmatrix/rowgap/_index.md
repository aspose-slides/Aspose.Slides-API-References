## MathMatrix.RowGap свойство

Значение вертикального интервала между рядами матрицы; Если RowGapRule установлено на 3 ("Точно"), то единица измерения интерпретируется как твипсы (1/20 часть пункта). Если RowGapRule установлено на 4 ("Множитель"), то единица измерения интерпретируется как половина строк. По умолчанию: 0

```csharp
public uint RowGap { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

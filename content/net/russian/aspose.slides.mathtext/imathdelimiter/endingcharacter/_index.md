## IMathDelimiter.EndingCharacter property

Символ завершающего делимитера указывает завершающий или закрывающий символ делимитера. Математические делимитеры – это заключающие символы, такие как скобки, квадратные скобки и фигурные скобки. По умолчанию: ')'.

```csharp
public char EndingCharacter { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Join("y").Enclose();
delimiter.EndingCharacter = ']';
```

## Свойство IMathDelimiter.SeparatorCharacter

Символ-разделитель указывает символ, который разделяет аргументы в объекте разделителя. По умолчанию: '&#x7C;'.

```csharp
public char SeparatorCharacter { get; set; }
```

### Примеры

Пример:


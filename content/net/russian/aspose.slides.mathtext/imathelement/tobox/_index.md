## IMathElement.ToBox метод

Размещает этот элемент в невизуальной коробке (логической группе), которая используется для группировки компонентов уравнения или другого примера математического текста. Объект в коробке может (например) служить эмулятором оператора с или без точки выравнивания, служить точкой разрыва строки или быть сгруппирован таким образом, чтобы не допускать разрывов строк внутри.

```csharp
public IMathBox ToBox()
```

### Возвращаемое значение

Логическая коробка с этим элементом внутри

### Примеры

Пример:

```csharp
[C#]
IMathBox box = new MathematicalText("x:=y").ToBox();
```

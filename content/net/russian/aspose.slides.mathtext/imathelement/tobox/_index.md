## IMathElement.ToBox метод

Помещает этот элемент в невизуальную коробку (логическую группу), которая используется для группировки компонентов уравнения или другого примера математического текста. Запакованный объект может (например) служить эмулятором оператора с точкой выравнивания или без неё, быть точкой разрыва строки или быть сгруппирован таким образом, чтобы не допускать разрывов строк внутри.

```csharp
public IMathBox ToBox()
```

### Возвращаемое значение

Логическая коробка с помещённым внутри этого элемента

### Примеры

Пример:

```csharp
[C#]
IMathBox box = new MathematicalText("x:=y").ToBox();
```

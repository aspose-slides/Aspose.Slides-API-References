## IMathArray.RowSpacing свойство

Интервал между рядами массива. Используется только когда RowSpacingRule установлен в 3. Точно в этом случае единицей измерения являются пункты или Множитель, в этом случае единицей измерения являются полулинии. По умолчанию: 0

```csharp
public uint RowSpacing { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
IMathArray mathArray = new MathArray(new MathematicalText("item1"));
mathArray.RowSpacingRule = MathRowSpacingRule.Exactly;
mathArray.RowSpacing = 10;
```

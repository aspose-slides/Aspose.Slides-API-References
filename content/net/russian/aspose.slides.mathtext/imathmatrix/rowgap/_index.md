## IMathMatrix.RowGap свойство

Значение вертикального расстояния между строками матрицы; Если правило RowGapRule установлено на 3 ("Точно"), то единицей измерения является твипы (1/20 пункта). Если правило RowGapRule установлено на 4 ("Множитель"), то единицей измерения являются полустроки. По умолчанию: 0

```csharp
public uint RowGap { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

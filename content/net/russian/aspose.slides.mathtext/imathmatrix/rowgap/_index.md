## IMathMatrix.RowGap свойство

Значение вертикального расстояния между рядами матрицы; Если RowGapRule установлен на 3 ("Точно"), то единица измерения интерпретируется как твипсы (1/20 от точки) Если RowGapRule установлен на 4 ("Множитель"), то единица измерения интерпретируется как полустроки. По умолчанию: 0

```csharp
public uint RowGap { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

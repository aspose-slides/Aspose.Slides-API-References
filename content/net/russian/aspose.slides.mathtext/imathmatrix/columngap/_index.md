## Свойство IMathMatrix.ColumnGap

Значение горизонтального интервала между столбцами матрицы; Если ColumnGapRule установлено на 3 ("Exactly"), то единица измерения интерпретируется как твипсы (1/20 пункта). Если ColumnGapRule установлено на 4 ("Multiple"), то единица измерения интерпретируется как количество увеличений на 0.5 em. В других случаях игнорируется. По умолчанию: 0

```csharp
public uint ColumnGap { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.ColumnGapRule = MathSpacingRules.Exactly;
matrix.ColumnGap = 20;
```

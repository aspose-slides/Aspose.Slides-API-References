## Свойство IMathMatrix.MinColumnWidth

Минимальная ширина колонки в твипсах (1/20 пункта) Промежуток (также называемый "Промежуток колонки" или "Ширина промежутка") добавляется к MinColumnWidth для определения общего пространства между колонками матрицы (расстояние между одинаковыми краями разных колонок). По умолчанию: 0.

```csharp
public uint MinColumnWidth { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.MinColumnWidth = 20;
```

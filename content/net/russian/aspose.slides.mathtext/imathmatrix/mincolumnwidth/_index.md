## IMathMatrix.MinColumnWidth property

Минимальная ширина столбца в твипсах (1/20 пунктов) Расстояние между столбцами (также называемое “интервалом столбца” или “шириной зазора”) добавляется к MinColumnWidth для определения общего расстояния между столбцами матрицы (расстояние между одинаковыми краями разных столбцов). По умолчанию: 0.

```csharp
public uint MinColumnWidth { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.MinColumnWidth = 20;
```

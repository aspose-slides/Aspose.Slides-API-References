## IDataLabel.IsVisible свойство

Значение False означает, что метка данных не видима (и, следовательно, все флаги Show*-flags (ShowValue и др.) равны false). Только для чтения Boolean.

```csharp
public bool IsVisible { get; }
```

### Замечания

Если метка данных видима, вы можете скрыть ее с помощью метода Hide(). Но если метка данных не видима (IsVisible равно false), вы можете сделать метку данных видимой, установив флаги Show*-flags (ShowValue и др.) в состояние true.

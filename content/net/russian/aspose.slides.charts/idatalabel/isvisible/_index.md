## IDataLabel.IsVisible свойство

Ложь означает, что метка данных не видима (и все флаги Show* (ShowValue и т. д.) также ложны). Только для чтения Boolean.

```csharp
public bool IsVisible { get; }
```

### Замечания

Если метка данных видима, вы можете сделать её невидимой с помощью метода Hide(). Но если метка данных не видима (IsVisible равно ложь), вы можете сделать метку данных видимой, установив флаги Show* (ShowValue и т. д.) в состояние true.

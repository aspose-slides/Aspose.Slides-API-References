## IChartData.UseSecondaryCategories свойство

Если false, то свойство [`SecondaryCategories`](../secondarycategories) возвратит null, и данные в свойстве [`Categories`](../categories) используются как для первичных, так и для вторичных серий. Если true, то данные в свойстве [`SecondaryCategories`](../secondarycategories) используются для вторичных серий, а данные в свойстве [`Categories`](../categories) используются для первичных серий. Читаемое/записываемое логическое значение.

```csharp
public bool UseSecondaryCategories { get; set; }
```

### Примеры

Пример. Какие категории относятся к серии - ChartData.Categories или ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // связанные категории это series.Chart.ChartData.SecondaryCategories
}
else
{
    // связанные категории это series.Chart.ChartData.Categories
}
```

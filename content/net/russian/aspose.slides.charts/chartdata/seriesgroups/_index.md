## Свойство ChartData.SeriesGroups

Получает группы последовательностей. Только для чтения [`IChartSeriesGroupCollection`](../../ichartseriesgroupcollection).

```csharp
public IChartSeriesGroupCollection SeriesGroups { get; }
```

### Примечания

1) Каждая группа последовательностей содержит последовательности с комбинируемыми типами. Группы комбинируемых типов последовательностей определяются и описываются с помощью перечисления CombinableSeriesTypesGroup. Также каждая группа последовательностей содержит последовательности, которые отображаются либо на основных осях, либо на дополнительных осях (не оба случая в одной группе). Таким образом, принцип группировки последовательностей — это группировка по типам, упомянутым выше, и по типу отображения на основных/дополнительных осях. 2) Группа последовательностей содержит некоторые свойства последовательностей, которые общие для каждой последовательности в группе ("свойства группы последовательностей"). "Свойства группы последовательностей" в классе ChartSeriesGroup являются доступными для чтения и записи. Каждое из "свойств группы последовательностей" может иметь доступную только для чтения проекцию в классе ChartSeries.

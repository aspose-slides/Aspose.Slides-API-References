## IChartCategory.UseCell свойство

Если истинно, то свойство AsCell актуально. Другими словами, рабочий лист используется для хранения категории (в этом случае поддерживается многоуровневая категория). Если ложь, то свойство AsLiteral актуально. Другими словами, рабочий лист НЕ используется для хранения категории (и этот случай не поддерживает многоуровневые категории). Только для чтения Булево.

```csharp
public bool UseCell { get; }
```

### Примечания

Для изменения значения этого свойства (для всех категорий в коллекции) установите новое значение для свойства ChartCategoryCollection.UseCells.

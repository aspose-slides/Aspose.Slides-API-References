## DataLabelFormat.IsNumberFormatLinkedToSource свойство

Чтение/запись логического значения.

```csharp
public bool IsNumberFormatLinkedToSource { get; set; }
```

### Примечания

Если родительский объект этого DataLabelFormat является коллекцией DataLabelCollection объектов меток данных, тогда это свойство получает или устанавливает значение по умолчанию для свойства IsNumberFormatLinkedToSource для новых меток данных в коллекции DataLabelCollection. Установка этого свойства с определенным значением также устанавливает это значение для свойства IsNumberFormatLinkedToSource для всех меток данных в коллекции DataLabelCollection (т.е. "DataLabels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = val;" устанавливает для всех DataLabels[i].IsNumberFormatLinkedToSource значение равное val).

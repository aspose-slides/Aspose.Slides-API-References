## DataLabelFormat.Position свойство

Представляет положение метки данных. Чтение/запись [`LegendDataLabelPosition`](../../legenddatalabelposition).

```csharp
public LegendDataLabelPosition Position { get; set; }
```

### Замечания

Если родителем этого объекта DataLabelFormat является коллекция DataLabelCollection меток данных, то это свойство получает или задает значение по умолчанию для свойства Position для новых меток данных в коллекции DataLabelCollection. Представляет положение для объектов DataLabel. Установка этого свойства с заданным значением также устанавливает это значение в свойство Position для всех меток данных в коллекции DataLabelCollection (т.е. "DataLabels.DefaultDataLabelFormat.Position = val;" приводит к тому, что все DataLabels[i].Position равно val).

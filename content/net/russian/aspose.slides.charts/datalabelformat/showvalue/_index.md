## DataLabelFormat.ShowValue свойство

Представляет поведение отображения значения процента метки данных указанного графика. True отображает процентное значение. False для скрытия. Читаемое/записываемое логическое значение.

```csharp
public bool ShowValue { get; set; }
```

### Примечания

Если родителем этого объекта DataLabelFormat является коллекция меток данных DataLabelCollection, то это свойство получает или устанавливает значение по умолчанию для свойства ShowValue для новых меток данных в коллекции DataLabelCollection. Установка этого свойства с значением также устанавливает это значение для свойства ShowValue для всех меток данных в коллекции DataLabelCollection (т.е. "DataLabels.DefaultDataLabelFormat.ShowValue = val;" приводит к тому, что все DataLabels[i].ShowValue равно val).

## Свойство ITrendline.Forward

Указывает количество категорий (или единиц на рассеянном графике), на которое линия тренда расширяется после данных для серии, которая подвергается тренду. На рассеянных и нерассеянных графиках значение должно быть любым неотрицательным значением. Для чтения/записи Double.

```csharp
public double Forward { get; set; }
```

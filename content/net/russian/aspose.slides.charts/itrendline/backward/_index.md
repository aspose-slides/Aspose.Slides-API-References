## Свойство ITrendline.Backward

Указывает количество категорий (или единиц на диаграмме рассеяния), на которое линия тренда расширяется перед данными для серии, которая анализируется. На диаграммах рассеяния и нерассеяния значение должно быть любым неотрицательным значением. Чтение/запись Double.

public double Backward { get; set; }

## PieSplitType enumeration  

Представляет собой тип точек разбиения во втором круге или стержне на круговой или стержневой диаграмме.  

```csharp  
public enum PieSplitType  
```  

### Values  

| Name     | Value | Description                                                                                         |  
|----------|-------|-----------------------------------------------------------------------------------------------------|  
| Default  | `0`   | Указывает, что точки данных будут разделены с использованием механизма по умолчанию для этого типа диаграммы. |  
| Custom   | `1`   | Указывает, что точки данных будут разделены между кругом и второй диаграммой согласно значениям Кастомного Разбиения. |  
| ByPercentage | `2` | Указывает, что точки данных будут разделены между кругом и второй диаграммой, помещая точки с процентом менее чем Процент Разбиения во вторую диаграмму. |  
| ByPos    | `3`   | Указывает, что точки данных будут разделены между кругом и второй диаграммой, помещая последнее Положение Разбиения точек данных во вторую диаграмму. |  
| ByValue  | `4`   | Указывает, что точки данных будут разделены между кругом и второй диаграммой, помещая точки данных со значением меньше чем Положение Разбиения во вторую диаграмму. |

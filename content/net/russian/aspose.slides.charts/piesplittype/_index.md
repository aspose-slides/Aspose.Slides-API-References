## Перечисление PieSplitType

Представляет тип точек разделения во второй части круговой или столбчатой диаграммы в диаграмме «pie-of-pie» или «bar-of-pie».

```csharp
public enum PieSplitType
```

### Значения

| Name | Value | Description |
| --- | --- | --- |
| Default | `0` | Указывает, что точки данных должны быть разделены с использованием механизма по умолчанию для этого типа диаграммы. |
| Custom | `1` | Указывает, что точки данных должны быть разделены между кругом и второй диаграммой в соответствии со значениями Custom Split. |
| ByPercentage | `2` | Указывает, что точки данных должны быть разделены между кругом и второй диаграммой, помещая точки с процентом меньше Split Position percent во вторую диаграмму. |
| ByPos | `3` | Указывает, что точки данных должны быть разделены между кругом и второй диаграммой, помещая последнюю Split Position точек данных во вторую диаграмму. |
| ByValue | `4` | Указывает, что точки данных должны быть разделены между кругом и второй диаграммой, помещая точки данных со значением меньше Split Position во вторую диаграмму. |

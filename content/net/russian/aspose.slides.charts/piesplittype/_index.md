## Перечисление PieSplitType

Предstavляет тип разбиения точек во втором круге или столбце на диаграмме круг-круг или столбец-круг.

```csharp
public enum PieSplitType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Default | `0` | Указывает, что точки данных должны быть разделены с использованием механизма по умолчанию для этого типа диаграммы. |
| Custom | `1` | Указывает, что точки данных должны быть разделены между кругом и второй диаграммой в соответствии с пользовательскими значениями разбиения. |
| ByPercentage | `2` | Указывает, что точки данных должны быть разделены между кругом и второй диаграммой, помещая точки с процентом менее Split Position в секунду диаграмму. |
| ByPos | `3` | Указывает, что точки данных должны быть разделены между кругом и второй диаграммой, помещая последнее положение разбиения точек данных во второй диаграмме. |
| ByValue | `4` | Указывает, что точки данных должны быть разделены между кругом и второй диаграммой, помещая точки данных со значением менее Split Position во второй диаграмме. |

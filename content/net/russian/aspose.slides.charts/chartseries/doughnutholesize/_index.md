## ChartSeries.DoughnutHoleSize property

Задает размер отверстия в кольцевой диаграмме (может составлять от 10 до 90 процентов от размера области графика). Это свойство не только этой серии, но и всех серий родительской серии группа - это проекция соответствующего группового свойства. Итак, это свойство доступно только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство чтения/записи ParentSeriesGroup.DoughnutHoleSize для значения изменения. Только для чтенияByte.

```csharp
public byte DoughnutHoleSize { get; }
```

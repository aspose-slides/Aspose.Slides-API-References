## ChartSeries.DoughnutHoleSize property

Указывает размер отверстия в круговой диаграмме (может составлять от 10 до 90 процентов от размера области графика). Это свойство не только данной серии, но и всех серий родительской группы серий - это проекция соответствующего группового свойства. Итак, это свойство только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство для чтения/записи ParentSeriesGroup.DoughnutHoleSize для изменения значения. Только для чтения Byte.

public byte DoughnutHoleSize { get; }

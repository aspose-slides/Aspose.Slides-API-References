## ChartSeries.TrendLines свойство

Коллекция линий тренда серий. Только для чтения [`ITrendlineCollection`](../../itrendlinecollection).

```csharp
public ITrendlineCollection TrendLines { get; }
```

### Примечания

TrendLines доступны (не равны null) для серий данных в нестаканированных 2-D графиках: областях, столбцах, линиях, фондовых, xy (рассеяние) и пузырьковых графиках. Линии тренда недоступны для серий данных в любом типе графика, который является стаканированным или 3-D. Линии тренда также недоступны для радарных, круговых, поверхностных или пончиковых графиков.

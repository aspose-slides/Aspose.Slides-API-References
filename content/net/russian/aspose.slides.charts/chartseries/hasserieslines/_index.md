## ChartSeries.HasSeriesLines property

Определяет, существуют ли линии серий для этой серии и родственных серий. Это свойство не только этой серии, но и всех серий родительской группы серий - это проекция соответствующего группового свойства. И поэтому это свойство является только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство ParentSeriesGroup.HasSeriesLines для изменения значения. Используйте свойство ParentSeriesGroup.SeriesLinesFormat для форматирования линий серий. Boolean только для чтения.

## Свойство ChartSeries.PieSplitPosition

Указывает значение, которое будет использоваться для определения того, какие точки данных находятся во втором пироге или столбце на диаграмме пирога с пирогом или столбца с пирогом. Используется вместе с свойством PieSplitBy. Это свойство не только этой серии, но и всех серий родительской группы серий - это проекция соответствующего группового свойства. Поэтому это свойство доступно только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство ParentSeriesGroup.PieSplitPosition для изменения значения. Доступно только для чтения Double.

```csharp
public double PieSplitPosition { get; }
```

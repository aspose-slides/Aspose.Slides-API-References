## Свойство ChartSeries.FirstSliceAngle

Указывает угол первого среза круговой или донат-графика в градусах (по часовой стрелке от вверх, от 0 до 360 градусов). Это свойство не только этой серии, но и всех серий родительской группы серий - это проекция соответствующего свойства группы. Итак, это свойство доступно только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство ParentSeriesGroup.FirstSliceAngle для изменения значения. Доступно только для чтения UInt16.

```csharp
public ushort FirstSliceAngle { get; }
```

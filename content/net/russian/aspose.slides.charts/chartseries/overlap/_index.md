## ChartSeries.Overlap property

Указывает, сколько полос и столбцов должно перекрываться на двумерных диаграммах (от -100 до 100). Это свойство не только этой серии, но и всех серий родительской серии группа - это проекция соответствующего группового свойства. Итак, это свойство доступно только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство чтения/записи ParentSeriesGroup.Overlap для значения изменения. Только для чтенияSByte.

```csharp
public sbyte Overlap { get; }
```

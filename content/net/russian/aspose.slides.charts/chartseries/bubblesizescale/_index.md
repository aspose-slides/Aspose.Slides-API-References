## Свойство ChartSeries.BubbleSizeScale

Указывает коэффициент масштаба для пузырьковой диаграммы (может находиться в диапазоне от 0 до 300 процентов от стандартного размера). Это свойство не только этой серии, но и всех серий родительской группы серий - это проекция соответствующего свойства группы. Поэтому это свойство является только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство ParentSeriesGroup.BubbleSizeScale для изменения значения.

```csharp
public int BubbleSizeScale { get; }
```

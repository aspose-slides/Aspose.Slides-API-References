## ChartSeries.GapDepth property

Возвращает или устанавливает расстояние, в процентах от ширины маркера, между рядами данных в 3D-графике. Это свойство не только этого ряда, но и всех рядов родительской группы рядов - это проекция соответствующего группового свойства. Поэтому это свойство является только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе рядов. Используйте свойство ParentSeriesGroup.GapDepth для изменения значения. Доступ только для чтения Int32.

public int GapDepth { get; }

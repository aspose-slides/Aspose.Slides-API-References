## ChartSeries.IsColorVaried свойство

Указывает, что каждый маркер данных в серии имеет другой цвет. Это свойство не только этой серии, но и всех серий группы родительских серий - это проекция соответствующего группового свойства. Поэтому это свойство является только для чтения. Используйте свойство ParentSeriesGroup для доступа к группе родительских серий. Используйте свойство ParentSeriesGroup.IsColorVaried для изменения значения. Только для чтения логическое значение.

```csharp
public bool IsColorVaried { get; }
```

## ChartSeries.IsColorVaried свойство

Указывает, что каждый маркер данных в серии имеет разный цвет. Это свойство относится не только к этой серии, но и ко всем сериям родительской группы серий - это проекция соответствующего группового свойства. Таким образом, это свойство только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство ParentSeriesGroup.IsColorVaried для изменения значения. Свойство только для чтения типа Boolean.

```csharp
public bool IsColorVaried { get; }
```

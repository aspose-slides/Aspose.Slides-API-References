## ChartSeries.GapWidth property

Задает расстояние между кластерами полос или столбцов в процентах от ширины полосы или столбца. Это свойство не только этой серии, но и всех серий родительской серии группа - это проекция соответствующего группового свойства. Итак, это свойство доступно только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство чтения/записи ParentSeriesGroup.GapWidth для значения изменения. Только для чтенияInt32.

```csharp
public int GapWidth { get; }
```

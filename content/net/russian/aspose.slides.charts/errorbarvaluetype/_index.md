## Перечисление ErrorBarValueType

Представляет тип значения погрешностной линии

```csharp
public enum ErrorBarValueType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Custom | `0` | Указывает, что длина погрешностных линий должна определяться элементами Plus и Minus. |
| Fixed | `1` | Указывает, что длина погрешностных линий должна быть фиксированным значением, определяемым свойством Error Bar Value. |
| Percentage | `2` | Указывает, что длина погрешностных линий должна составлять процент свойства Error Bar Value от данных. |
| StandardDeviation | `3` | Указывает, что длина погрешностных линий должна быть стандартными отклонениями свойства Error Bar Value от данных. |
| StandardError | `4` | Указывает, что длина погрешностных линий должна быть стандартными ошибками свойства Error Bar Value от данных. |

## IOverridableText.TextFrameForOverriding property

Может содержать текст с богатым форматированием. Если это свойство не равно null, то значение этого отформатированного текста заменяет автоматически сгенерированный текст. Автоматически сгенерированный текст является неявным свойством метки данных, метки единицы отображения оси значений, заголовка оси, заголовка диаграммы, метки трендовой линии. Автоматически сгенерированный текст форматируется с помощью свойства IFormattedTextContainer.TextFormat. Только для чтения [`ITextFrame`](../../../aspose.slides/itextframe).

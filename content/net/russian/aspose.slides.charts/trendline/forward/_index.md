## Trendline.Forward свойство

Указывает количество категорий (или единиц на диаграмме рассеяния), на которое линия тренда продолжает движение после данных для ряда, который подвергается трендированиям. На диаграммах рассеяния и не рассеяния значение должно быть любым неотрицательным значением. Читаемое/записываемое Double.

public double Forward { get; set; }

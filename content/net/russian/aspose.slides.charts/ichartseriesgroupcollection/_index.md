### Замечания

1) Каждая группа серий содержит серии с объединяемыми типами. Группы объединяемых типов серий определены и описаны с помощью перечисления CombinableSeriesTypesGroup. Также каждая группа серий содержит серии, которые располагаются либо на первичных осях, либо на вторичных осях (не в обоих случаях в одной группе). Таким образом, принцип группирования серий заключается в группировке по типам, упомянутым выше, и по типу графика первичного/вторичного. 2) Группа серий содержит некоторые свойства серий, которые общие для каждой серии в группе ("свойства группы серий"). "Свойства группы серий" в классе ChartSeriesGroup являются свойствами с возможностью чтения и записи. Каждое из "свойств группы серий" может иметь доступ только для чтения в классе ChartSeries.

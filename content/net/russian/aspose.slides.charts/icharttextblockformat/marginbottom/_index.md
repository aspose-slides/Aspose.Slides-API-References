## IChartTextBlockFormat.MarginBottom свойство

Возвращает или задает нижний отступ (пункты) в TextFrame. Изменение этого свойства может оказать определенное влияние только на эти части диаграммы: DataLabel и DataLabelFormat (полная поддержка в PowerPoint 2013; в PowerPoint 2007 нет эффекта для рендеринга). Чтение/запись Double.

public double MarginBottom { get; set; }

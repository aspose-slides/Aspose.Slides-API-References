## IChartTextBlockFormat.MarginBottom property

Возвращает или задает нижнее поле (в пунктах) в TextFrame. Изменение этого свойства может оказать определенное влияние только на следующие части диаграммы: DataLabel и DataLabelFormat (полная поддержка в PowerPoint 2013; в PowerPoint 2007 нет эффекта для рендеринга). Чтение/записьDouble.

public double MarginBottom { get; set; }

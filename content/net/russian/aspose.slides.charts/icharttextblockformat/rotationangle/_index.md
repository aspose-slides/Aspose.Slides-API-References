## IChartTextBlockFormat.RotationAngle свойство  
  
Указывает пользовательское вращение, которое применяется к тексту внутри ограничивающего прямоугольника. Если оно не указано, используется вращение сопутствующей фигуры. Если оно указано, то это применяется независимо от фигуры. То есть фигура может иметь вращение, примененное дополнительно к тексту, которому также применено вращение. Полученное значение визуального вращения текста суммируется из этого свойства и предопределенного вертикального типа в свойстве TextVerticalType. Чтение/запись Single.  
  
```csharp  
public float RotationAngle { get; set; }  
```  
  
### Примеры  
  
Рассмотрим случай, когда фигура имеет вращение на 90 градусов по часовой стрелке. В дополнение к этому текст самого тела имеет вращение на -90 градусов против часовой стрелки. Тогда результирующая фигура будет казаться вращенной, но текст внутри нее будет выглядеть так, как будто он вообще не был вращен.

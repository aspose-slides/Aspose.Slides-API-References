## IChartTextBlockFormat.RotationAngle свойство

Указывает пользовательское вращение, применяемое к тексту в границах рамки. Если оно не указано, используется вращение сопутствующей фигуры. Если указано, то оно применяется независимо от фигуры. То есть к фигуре может быть применено вращение, кроме того, что текст сам по себе имеет примененное к нему вращение. Результирующее значение визуального вращения текста подводится из этого свойства и предопределенного вертикального типа в свойстве TextVerticalType. Чтение/запись Single.

```csharp
public float RotationAngle { get; set; }
```

### Примеры

Рассмотрим случай, когда фигуре применено вращение на 90 градусов по часовой стрелке. В дополнение к этому сам текст имеет вращение на -90 градусов против часовой стрелки. В результате фигура будет казаться повернутой, но текст внутри нее будет выглядеть так, будто он вовсе не был повернут.

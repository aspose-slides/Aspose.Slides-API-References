## DataLabel.IsVisible свойство

Ложь означает, что метка данных невидима (и все флаги Show*-флагов (ShowValue, ...) также ложны). Только для чтения Boolean.

```csharp
public bool IsVisible { get; }
```

### Примечания

Если метка данных видима, вы можете скрыть ее с помощью метода Hide(). Но если метка данных не видима (IsVisible равно ложь), вы можете сделать метку данных видимой, установив флаги Show*-флаги (ShowValue, ...) в состояние истинно.

## Свойство IDataLabelCollection.IsVisible

Ложное значение означает, что метка данных по умолчанию не видима (и поэтому все флаги Show* (ShowValue, ...) свойства DefaultDataLabelFormat имеют значение false). Только для чтения Boolean.

```csharp
public bool IsVisible { get; }
```

### Замечания

Если метка данных по умолчанию видима, вы можете сделать её скрытой по умолчанию с помощью метода Hide(). Но если метка данных по умолчанию не видима (IsVisible равно false), вы можете сделать метку данных "видимой по умолчанию", установив флаги Show* (ShowValue, ...) свойства DefaultDataLabelFormat в состояние true.

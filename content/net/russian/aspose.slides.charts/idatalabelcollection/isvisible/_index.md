## IDataLabelCollection.IsVisible свойство

Значение false означает, что метка данных по умолчанию не видима (и поэтому все флаги Show*- (ShowValue и т.д.) свойства DefaultDataLabelFormat равны false). Только для чтения Boolean.

```csharp
public bool IsVisible { get; }
```

### Примечания

Если метка данных по умолчанию видима, вы можете сделать её невидимой по умолчанию с помощью метода Hide(). Но если метка данных по умолчанию не видима (IsVisible равно false), вы можете сделать метку данных "видимой по умолчанию", установив состояния Show*- (ShowValue и т.д.) свойства DefaultDataLabelFormat в true.

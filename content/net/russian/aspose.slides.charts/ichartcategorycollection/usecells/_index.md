## Свойство IChartCategoryCollection.UseCells

Если true, то рабочий лист используется для хранения категорий (в этом случае поддерживаются многоуровневые категории). Если false, то рабочий лист НЕ используется для хранения значений (и этот случай не поддерживает многоуровневые категории). Чтение/запись булевый тип.

```csharp
public bool UseCells { get; set; }
```

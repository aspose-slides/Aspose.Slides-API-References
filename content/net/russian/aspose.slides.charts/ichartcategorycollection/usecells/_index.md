## IChartCategoryCollection.UseCells свойство

Если значение истинно, то рабочий лист используется для хранения категорий (это поддерживает многоуровневые категории). Если ложно, то рабочий лист НЕ используется для хранения значений (и этот случай не поддерживает многоуровневые категории). Читаемое/записываемое логическое значение.

```csharp
public bool UseCells { get; set; }
```

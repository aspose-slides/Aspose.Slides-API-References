## IDataLabelFormat.ShowValue свойство

Представляет поведение отображения значения процента метки данных указанного графика. True отображает процентное значение. False скрывает. Читаемое/записываемое логическое значение.

```csharp
public bool ShowValue { get; set; }
```

### Замечания

Если родительским объектом этого DataLabelFormat является коллекция меток данных DataLabelCollection, то это свойство получает или устанавливает значение по умолчанию для свойства ShowValue для новых меток данных в коллекции DataLabelCollection. Установка этого свойства сзначением также устанавливает это значение для свойства ShowValue для всех меток данных в коллекции DataLabelCollection (т.е. "DataLabels.DefaultDataLabelFormat.ShowValue = val;" приведет к тому, что все DataLabels[i].ShowValue будет равно val).

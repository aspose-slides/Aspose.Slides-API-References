## IDataLabelFormat.NumberFormat property

Представляет строку формата для объекта DataLabels. Запись/чтение String.

```csharp
public string NumberFormat { get; set; }
```

### Примечания

Если родительским объектом этого DataLabelFormat является коллекция DataLabelCollection, то это свойство получает или устанавливает значение по умолчанию для свойства NumberFormat для новых меток данных в коллекции DataLabelCollection. Когда это свойство установлено со значением, это значение также устанавливается для свойства NumberFormat для всех меток данных в коллекции DataLabelCollection (т.е. "DataLabels.DefaultDataLabelFormat.NumberFormat = val;" вызывает равенство всех DataLabels[i].NumberFormat значению val).

### Примеры

```csharp
[C#]
series.Labels.DefaultDataLabelFormat.ShowValue = true;
series.Labels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = false;
series.Labels.DefaultDataLabelFormat.NumberFormat = "0.0%";
```

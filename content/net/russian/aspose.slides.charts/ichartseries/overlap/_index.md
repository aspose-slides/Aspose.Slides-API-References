## IChartSeries.Overlap property

Указывает, насколько бары и столбцы перекрываются на 2-D графиках, в процентах (от -100% до 100%). Это свойство не только этой серии, но и всех серий родительской группы серий. Это проекция соответствующего свойства в родительской группе серий, и поэтому это свойство доступно только для чтения. Чтобы изменить значение, используйте свойство ParentSeriesGroup.Overlap с возможностью записи. Только для чтения SByte.

```csharp
public sbyte Overlap { get; }
```

### Remarks

Overlap определяет степень перекрытия или расстояния между барами и столбцами в процентах от их ширины: - -100%: Максимальное расстояние (бары полностью разделены). - 0%: Бары расположены рядом друг с другом без перекрытия или расстояния. - 100%: Максимальное перекрытие (бары полностью перекрывают друг друга). Это проекция свойства ParentSeriesGroup.Overlap.

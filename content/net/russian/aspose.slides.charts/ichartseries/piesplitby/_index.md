## IChartSeries.PieSplitBy property

Указывает, как определить, какие данные относятся ко второй диаграмме или столбцу на диаграмме с пирогом из пирога или столбцом из пирога. Это свойство не только этой серии, но и всех серий родительской группы серий - это проекция соответствующего свойства группы. Поэтому это свойство является доступным только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство чтения/записи ParentSeriesGroup.PieSplitBy для изменения значения. Свойство только для чтения [`PieSplitType`](../../piesplittype).

```csharp
public PieSplitType PieSplitBy { get; }
```

### Remarks

1) Это проекция свойства ParentSeriesGroup.PieSplitBy. 2) Если значение свойства равно PieSplitType.Custom, то вы можете определить индивидуальную информацию о разделении с помощью свойства ParentSeriesGroup.PieSplitCustomPoints.

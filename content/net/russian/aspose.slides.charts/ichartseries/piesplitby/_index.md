## IChartSeries.PieSplitBy свойство

Указывает, как определить, какие точки данных находятся во втором пироге или столбце на графике пирога-пирога или столбца-пирога. Это свойство не только этой серии, но и всех серий родительской группы серий - это проекция соответствующего свойства группы. И поэтому это свойство доступно только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство чтения/записи ParentSeriesGroup.PieSplitBy для изменения значения. Доступный только для чтения [`PieSplitType`](../../piesplittype).

```csharp
public PieSplitType PieSplitBy { get; }
```

### Примечания

1) Это проекция свойства ParentSeriesGroup.PieSplitBy. 2) Если значение свойства равно PieSplitType.Custom, то вы можете определить пользовательскую информацию о разделении с помощью свойства ParentSeriesGroup.PieSplitCustomPoints.

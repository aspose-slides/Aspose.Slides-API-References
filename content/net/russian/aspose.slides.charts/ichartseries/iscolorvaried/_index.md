## IChartSeries.IsColorVaried свойство

Указывает, что каждый маркер данных в серии имеет другой цвет. Это свойство относится не только к этой серии, но и ко всем сериям родительской группы серий - это проекция соответствующего группового свойства. Поэтому это свойство является только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство ParentSeriesGroup.IsColorVaried для изменения значения. Только для чтения Boolean.

```csharp
public bool IsColorVaried { get; }
```

## IChartSeries.IsColorVaried свойство

Указывает, что каждый маркер данных в серии имеет новый цвет. Это свойство не только этой серии, но и всех серий родительской группы серий - это проекция соответствующего группового свойства. Таким образом, это свойство только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство ParentSeriesGroup.IsColorVaried для изменения значения. Только для чтения логическое значение.

```csharp
public bool IsColorVaried { get; }
```

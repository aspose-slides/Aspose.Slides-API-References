## IChartSeries.DoughnutHoleSize свойство

Указывает размер отверстия в круговой диаграмме (может составлять от 10 до 90 процентов от размера области построения). Это свойство принадлежит не только этой серии, но и всем сериям родительской группы серий - это проекция соответствующего свойства группы. Поэтому это свойство является только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство ParentSeriesGroup.DoughnutHoleSize для изменения значения. Только для чтения Byte.

```csharp
public byte DoughnutHoleSize { get; }
```

Это проекция свойства ParentSeriesGroup.DoughnutHoleSize.

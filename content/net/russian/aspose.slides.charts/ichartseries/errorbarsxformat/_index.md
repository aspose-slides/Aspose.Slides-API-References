## IChartSeries.ErrorBarsXFormat property

Представляет ErrorBars серии с направлением X. ErrorBars с направлением X доступны для серий типа area, bar, scatter и bubble. Для любых других типов графиков это свойство возвращает null (включая 3D графики). В случае пользовательских значений используйте коллекцию DataPoints для указания значения (с помощью свойства [`ErrorBarsCustomValues`](../../ichartdatapoint/errorbarscustomvalues)). Только для чтения [`IErrorBarsFormat`](../../ierrorbarsformat).

public IErrorBarsFormat ErrorBarsXFormat { get; }

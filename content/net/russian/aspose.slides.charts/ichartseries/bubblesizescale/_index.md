## IChartSeries.BubbleSizeScale свойство

Указывает коэффициент масштаба для пузырьковой диаграммы (может быть в диапазоне от 0 до 300 процентов от стандартного размера). Это свойство относится не только к этой серии, но и ко всем сериям родительской группы серий - это проекция соответствующего свойства группы. Таким образом, это свойство является только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство чтения/записи ParentSeriesGroup.BubbleSizeScale для изменения значения.

```csharp
public int BubbleSizeScale { get; }
```

### Примечания

Это проекция свойства ParentSeriesGroup.BubbleSizeScale.

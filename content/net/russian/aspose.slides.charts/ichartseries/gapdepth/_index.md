## IChartSeries.GapDepth свойство

Возвращает или устанавливает расстояние, как процент от ширины маркера, между сериями данных в 3D-диаграмме. Это свойство относится не только к этой серии, но и ко всем сериям родительской группы серий - это проекция соответствующего свойства группы. Таким образом, это свойство доступно только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство ParentSeriesGroup.GapDepth читаемое/записываемое для изменения значения. Доступно только для чтения Int32.

```csharp
public int GapDepth { get; }
```

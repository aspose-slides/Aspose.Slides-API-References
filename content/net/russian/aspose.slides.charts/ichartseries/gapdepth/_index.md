## IChartSeries.GapDepth свойство

Возвращает или устанавливает расстояние, в процентах от ширины маркера, между сериями данных в 3D-графике. Это свойство не только этой серии, но и всех серий родительской группы серий - это проекция соответствующего группового свойства. Поэтому это свойство только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство ParentSeriesGroup.GapDepth для записи/чтения для изменения значения. Только для чтения Int32.

```csharp
public int GapDepth { get; }
```

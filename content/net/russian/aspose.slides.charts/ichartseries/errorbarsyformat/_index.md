## IChartSeries.ErrorBarsYFormat property

Представляет ErrorBars серии с направлением Y. ErrorBars с направлением Y доступны для серий типов area, bar, line, scatter и bubble. Для любых других типов графиков это свойство возвращает null (включая 3D графики). В случае с пользовательскими значениями используйте коллекцию DataPoints для задания значения (с помощью свойства [`ErrorBarsCustomValues`](../../ichartdatapoint/errorbarscustomvalues)). Только для чтения [`IErrorBarsFormat`](../../ierrorbarsformat).

public IErrorBarsFormat ErrorBarsYFormat { get; }

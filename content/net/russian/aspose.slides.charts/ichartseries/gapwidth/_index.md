## IChartSeries.GapWidth свойство

Указывает пространство между группами столбцов или баров, в процентном соотношении к ширине бара или столбца. Это свойство не только этой серии, но и всех серий родительской группы серий - это проекция соответствующего свойства группы. Поэтому это свойство является только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство ParentSeriesGroup.GapWidth для записи/чтения, чтобы изменить значение. Только для чтения Int32.

public int GapWidth { get; }

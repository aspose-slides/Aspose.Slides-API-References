## IChartSeries.SecondPieSize свойство

Указывает размер второго пирога или столбца в диаграмме пирог-пирог или столбец-пирог в процентах от размера первого пирога (может быть от 5 до 200 процентов). Это свойство не только этой серии, но и всех серий родительской группы серий - это проекция соответствующего группового свойства. Таким образом, это свойство доступно только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство ParentSeriesGroup.SecondPieSize для изменения значения. Доступно только для чтения UInt16.

```csharp
public ushort SecondPieSize { get; }
```

### Примечания

Это проекция свойства ParentSeriesGroup.SecondPieSize.

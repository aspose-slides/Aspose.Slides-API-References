## IChartSeries.SecondPieSize property

Задает размер второго сектора или столбца в графике "пирог из пирога" или "столбец из пирога" в процентах от размера первого сектора (может находиться в диапазоне от 5 до 200 процентов). Это свойство относится не только к этой серии, но и ко всем сериям родительской группы серий - это проекция соответствующего группового свойства. Таким образом, это свойство доступно только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство ParentSeriesGroup.SecondPieSize для изменения значения. Доступно только для чтения UInt16.

public ushort SecondPieSize { get; }

## IChartSeries.PieSplitPosition свойство

Указывает значение, которое будет использоваться для определения, какие данные находятся во втором секторе или столбце на круговой диаграмме с секторами или столбцах на круговой диаграмме. Используется вместе с свойством PieSplitBy. Это свойство не только этой серии, но и всех серий родительской группы серий - это проекция соответствующего группового свойства. Таким образом, это свойство доступно только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство ParentSeriesGroup.PieSplitPosition для чтения/записи для изменения значения. Доступно только для чтения Double.

public double PieSplitPosition { get; }

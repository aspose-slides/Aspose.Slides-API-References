## IChartSeries.PieSplitPosition property

Указывает значение, которое будет использоваться для определения, какие точки данных находятся во втором пироге или столбце на диаграмме пирога-из-пирога или бару-из-пирога. Используется вместе с свойством PieSplitBy. Это свойство принадлежит не только этой серии, но и всем сериям группы родительских серий - это проекция соответствующего свойства группы. Таким образом, это свойство является доступным только для чтения. Используйте свойство ParentSeriesGroup для доступа к группе родительских серий. Используйте свойство чтения/записи ParentSeriesGroup.PieSplitPosition для изменения значения. Доступно только для чтения Double.

public double PieSplitPosition { get; }

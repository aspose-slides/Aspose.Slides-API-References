## IChartDataPointCollection.GetOrCreateDataPointByIdx метод

Если коллекция уже содержит точку данных с индексом *index*, то возвращает эту точку данных. Если коллекция не содержит точки данных с индексом *index*==N (когда количество точек данных в этой коллекции меньше или равно N), то добавляет недостающие точки данных и возвращает последнюю (которая имеет запрашиваемый индекс). Например, индексы коллекции: {0, 1, 2}, а запрашиваемый индекс равен 5. Затем метод добавляет недостающие точки данных: {0, 1, 2, 3, 4, 5}. И возвращает точку данных с индексом 5.

```csharp
public IChartDataPoint GetOrCreateDataPointByIdx(uint index)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| index | UInt32 | Индекс. |

### Возвращаемое значение

Возвращает точку данных с запрашиваемым индексом.

## ChartCategoryCollection.UseCells свойство

Если истинно, то рабочий лист используется для хранения категорий (в этом случае поддерживаются многоуровневые категории). Если ложно, то рабочий лист НЕ используется для хранения значений (и в этом случае не поддерживаются многоуровневые категории). Читаемое/записываемое логическое значение.

public bool UseCells { get; set; }

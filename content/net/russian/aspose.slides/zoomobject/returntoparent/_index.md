## ZoomObject.ReturnToParent свойство

Получает или устанавливает поведение навигации в слайд-шоу. Чтение/запись Boolean. Значение по умолчанию: false

```csharp
public bool ReturnToParent { get; set; }
```

### Примечания

Истинное значение свойства указывает на поведение навигации «возврат к родителю» в слайд-шоу.

### Примеры

Пример:

```csharp
[C#]
IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
zoomFrame.ReturnToParent = true;
```

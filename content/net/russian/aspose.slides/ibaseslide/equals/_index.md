## Метод IBaseSlide.Equals

Определяет, равны ли два экземпляра IBaseSlide. Возвращаемое значение вычисляется на основе структуры слайда и статического контента. Два слайда равны, если все формы, стили, тексты, анимация и другие настройки и т.д. равны. Сравнение не учитывает значения уникальных идентификаторов, например, SlideId и динамический контент, например, текущее значение даты в плейсхолдере даты.

```csharp
public bool Equals(IBaseSlide slide)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| slide | IBaseSlide | IBaseSlide для сравнения с текущим IBaseSlide. |

### Возвращаемое значение

**true**, если указанный IBaseSlide равен текущему IBaseSlide; в противном случае **false**.

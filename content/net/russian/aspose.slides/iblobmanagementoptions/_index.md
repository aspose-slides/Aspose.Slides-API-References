## IBlobManagementOptions интерфейс

Binary Large Object (BLOB) — это двоичные данные, хранящиеся как единственная сущность, то есть BLOB может представлять собой аудио, видео или саму презентацию. Для оптимизации потребления памяти при работе с BLOB-ами используется ряд методов — которые могут быть уже сохранены в презентации или добавлены позже программно. При помощи [`IBlobManagementOptions`](../iblobmanagementoptions) можно изменить различные аспекты поведения, связанные с обработкой BLOB-ов для срока жизни экземпляра [`IPresentation`](../ipresentation).

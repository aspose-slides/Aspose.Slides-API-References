## Tab.CompareTo метод

Сравнивает текущий экземпляр с другим объектом того же типа.

```csharp
public int CompareTo(object obj)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| obj | Object | Объект для сравнения с этим экземпляром. |

### Возврат значения

Целое число со знаком, занимающее 32 бита, которое указывает относительный порядок сравниваемых объектов. Возвращаемое значение имеет следующие значения:  &lt; 0 - Этот экземпляр меньше, чем obj. = 0 - Этот экземпляр равен obj. &gt; 0 - Этот экземпляр больше, чем obj.

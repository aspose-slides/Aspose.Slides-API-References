## NormalViewRestoredProperties.DimensionSize свойство

Указывает размер области слайда (ширина, когда является дочерним элементом RestoredTop, высота, когда является дочерним элементом RestoredLeft). Доступно для чтения/записи Single.

```csharp
public float DimensionSize { get; set; }
```

### Исключения

| исключение | условие |
| --- | --- |
| ArgumentOutOfRangeException | Выбрасывается, когда значение меньше 0 или больше 100. |

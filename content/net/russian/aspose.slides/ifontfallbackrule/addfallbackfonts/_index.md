## AddFallBackFonts(string) {#addfallbackfonts}

Добавляет новый шрифт(ы) в список шрифтов FallBack.

```csharp
public void AddFallBackFonts(string fontName)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| fontName | String | Имя или имена шрифта (разделенные запятой) для FallBack |

### Примеры

```csharp
[C#]
//Создание нового экземпляра FontFallBackRule
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho");

//Добавление второго шрифта в правило 
newRule.AddFallBackFonts("MS Gothic");
//Добавление третьего и четвертого шрифтов в правило 
newRule.AddFallBackFonts("Tahoma, Times New Roman");
```

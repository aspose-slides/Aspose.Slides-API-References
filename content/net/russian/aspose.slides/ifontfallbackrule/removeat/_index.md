## IFontFallBackRule.RemoveAt метод

Удаляет шрифт FallBack по указанному индексу списка.

```csharp
public void RemoveAt(int index)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| index | Int32 | Индекс шрифта для удаления, основанный на нуле. |

### Примеры

```csharp
[C#]
// Создание правила, содержащего список шрифтов.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// Удаление Tahoma из списка
newRule.Remove (2);
```

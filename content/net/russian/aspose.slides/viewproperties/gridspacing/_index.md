## Свойство ViewProperties.GridSpacing

Возвращает или устанавливает расстояние между сеткой, которое должно использоваться для сетки, лежащей в основе документа презентации, в пунктах. Чтение/запись Single.

```csharp
public float GridSpacing { get; set; }
```

### Примечания

Значение расстояния между сеткой должно быть положительным числом. Типичный диапазон значений составляет от 1 мм (2.8349607 пункта) до 2 дюймов (144 пункта).

### Примеры

Следующий образец кода показывает, как изменить расстояние между сеткой в презентации PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

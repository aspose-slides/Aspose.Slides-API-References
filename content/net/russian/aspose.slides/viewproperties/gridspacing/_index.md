## ViewProperties.GridSpacing свойство

Возвращает или устанавливает расстояние между сетками, которое должно использоваться для сетки, лежащей в основе документа презентации, в пунктах. Чтение/запись Single.

```csharp
public float GridSpacing { get; set; }
```

### Примечания

Значение расстояния между сетками должно быть положительным числом. Типичный диапазон значений составляет от 1 мм (2.8349607 пункта) до 2 дюймов (144 пункта).

### Примеры

Следующий пример кода показывает, как изменить расстояние между сетками в презентации PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

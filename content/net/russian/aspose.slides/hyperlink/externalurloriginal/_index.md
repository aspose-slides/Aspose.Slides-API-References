## Hyperlink.ExternalUrlOriginal property  
  
Представляет собой гиперссылку, установленную для этой части, независимо от фактического содержания части. PowerPoint ведет себя особым образом для ссылок и соответствующего текста в части. Это позволяет создать текст для гиперссылки в виде действительного URL, отличного от реального адреса ссылки. В этом случае, когда вы просматриваете ссылку в окне редактирования, она изменится так, чтобы соответствовать текстовой части. Это свойство представляет собой исходное значение гиперссылки.  
  
```csharp  
public string ExternalUrlOriginal { get; }  
```

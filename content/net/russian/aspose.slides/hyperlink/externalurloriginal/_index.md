## Свойство Hyperlink.ExternalUrlOriginal

Представляет гиперссылку, установленную для этой части, без учета фактического содержимого части. PowerPoint ведет себя специфическим образом для ссылок и соответствующего текста в части. Это позволяет создавать текст для гиперссылки в виде действительного URL, отличного от реального адреса ссылки. В этом случае, когда вы просматриваете ссылку в окне редактирования, она будет изменена так, чтобы соответствовать текстовой части. Это свойство представляет собой первоначальное значение гиперссылки.

```csharp
public string ExternalUrlOriginal { get; }
```

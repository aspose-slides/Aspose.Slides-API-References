## PresentationInfo.CheckWriteProtection метод  

Проверяет, правильен ли пароль для изменения для защищенной от записи презентации.  

```csharp  
public bool CheckWriteProtection(string password)  
```  

| Параметр | Тип | Описание |  
| --- | --- | --- |  
| password | String | Пароль для проверки. |  

### Возвращаемое значение  

True, если презентация защищена от записи и пароль правильный. В противном случае False.  

### Исключения  

| исключение | условие |  
| --- | --- |  
| InvalidOperationException |  |  

### Примечания  

1. Вы должны проверить свойство [`IsWriteProtected`](../iswriteprotected) перед вызовом этого метода. 2. Когда пароль равен null или пустой, этот метод возвращает false.  

### Примеры  

```csharp  
[C#]  
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);  
if (info.IsWriteProtected == NullableBool.True)  
{  
    bool isWriteProtectedByPassword = info.CheckWriteProtection("my_password");  
}  
```

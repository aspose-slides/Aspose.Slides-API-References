## Control.Properties property

Возвращает коллекцию свойств ActiveX. Обратите внимание: Aspose.Slides поддерживает только свойства ActiveX на основе XML. Если свойства хранятся в двоичном формате, это свойство вернет null. Только для чтения [`IControlPropertiesCollection`](../../icontrolpropertiescollection).

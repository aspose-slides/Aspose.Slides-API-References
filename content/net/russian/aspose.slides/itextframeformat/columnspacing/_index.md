## ITextFrameFormat.ColumnSpacing property

Возвращает или задает расстояние между текстовыми столбцами в текстовой области (в пунктах). Это должно применяться только когда присутствует более 1 столбца. Это значение должно быть положительным числом. В противном случае значение будет установлено равным нулю. Чтение/записьDouble.

```csharp
public double ColumnSpacing { get; set; }
```

## ITextFrameFormat.ColumnSpacing свойство

Возвращает или задает пространство между текстовыми колонками в текстовой области (в пунктах). Это должно применяться только когда присутствует более одной колонки. Это значение должно быть положительным числом. В противном случае значение будет установлено в ноль. Чтение/запись Double.

```csharp
public double ColumnSpacing { get; set; }
```

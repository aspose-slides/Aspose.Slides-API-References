## ITextFrameFormat.RotationAngle свойство

Указывает пользовательское вращение, которое применяется к тексту внутри ограничивающего прямоугольника. Если не указано, используется вращение сопутствующей фигуры. Если указано, то это применяется независимо от фигуры. То есть фигура может иметь вращение, кроме того, что текст сам по себе имеет вращение. Результирующее значение визуального вращения текста суммируется из этого свойства и предопределенного вертикального типа в свойстве TextVerticalType. Чтение/запись тип Single.

```csharp
public float RotationAngle { get; set; }
```

### Примеры

Рассмотрим случай, когда фигура имеет вращение на 90 градусов по часовой стрелке. В дополнение к этому сам текст имеет вращение на -90 градусов против часовой стрелки. Тогда результирующая фигура будет выглядеть так, как будто она повёрнута, но текст внутри неё будет выглядеть так, будто он вовсе не был вращён.

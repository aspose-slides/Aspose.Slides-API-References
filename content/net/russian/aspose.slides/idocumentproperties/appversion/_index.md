## Свойство IDocumentProperties.AppVersion

Возвращает версию приложения. Доступно только для чтения строка.

```csharp
public string AppVersion { get; }
```

### Замечания

Содержимое этого элемента должно быть в формате XX.YYYY, где X и Y представляют числовые значения; в противном случае документ будет считаться несоответствующим. Aspose.Slides представляет свою версию в формате XX.YYZZ, где: XX - основная версия YY - второстепенная версия ZZ - версия патча Например, значение 23.0105 означает версию Aspose.Slides 23.1.5.

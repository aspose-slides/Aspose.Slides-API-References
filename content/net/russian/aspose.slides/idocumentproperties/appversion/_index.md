## IDocumentProperties.AppVersion property

Возвращает версию приложения. Свойство только для чтения String.

```csharp
public string AppVersion { get; }
```

### Remarks

Содержимое этого элемента должно быть в формате XX.YYYY, где X и Y представляют собой числовые значения; в противном случае документ будет считаться не соответствующим. Aspose.Slides представляет свою версию в формате XX.YYZZ, где: XX -.major version YY - minor version ZZ - patch version Например, значение 23.0105 означает Aspose.Slides версии 23.1.5.

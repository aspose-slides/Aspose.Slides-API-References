## Свойство IDocumentProperties.LastSavedTime

Возвращает дату последнего изменения презентации. Значения указаны в UTC. Чтение - только в случае Presentation.DocumentProperties (поскольку он будет обновляться внутренне во время процесса сохранения объекта IPresentation). Может быть изменено через экземпляр DocumentProperties, возвращаемый методом [`ReadDocumentProperties`](../../ipresentationinfo/readdocumentproperties). Пожалуйста, смотрите пример в сводке метода [`UpdateDocumentProperties`](../../ipresentationinfo/updatedocumentproperties).

public DateTime LastSavedTime { get; set; }

## Свойство IDocumentProperties.HyperlinksChanged

Указывает, что одна или несколько гиперссылок в этой части были обновлены исключительно в этой части производителем. Следующий производитель, который откроет этот документ, должен обновить отношения гиперссылок с новыми гиперссылками, указанными в этой части. Читаемое/записываемое логическое значение.

```csharp
public bool HyperlinksChanged { get; set; }
```

## Свойство IDocumentProperties.LinksUpToDate

Указывает, являются ли гиперссылки в документе актуальными. Установите этот элемент в **true**, чтобы указать, что гиперссылки обновлены. Установите этот элемент в **false**, чтобы указать, что гиперссылки устарели. Чтение/запись Boolean.

```csharp
public bool LinksUpToDate { get; set; }
```

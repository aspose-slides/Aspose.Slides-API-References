## Свойство ILoadOptions.InterruptionToken

Токен для мониторинга запросов на прерывание. Этот токен управляет всей жизнью экземпляра [`IPresentation`](../../ipresentation). Любая длительная операция, такая как загрузка или сохранение презентации, будет прервана с помощью вызова метода [`Interrupt`](../../iinterruptiontokensource/interrupt) интерфейса [`IInterruptionTokenSource`](../../iinterruptiontokensource).

```csharp
public IInterruptionToken InterruptionToken { get; set; }
```

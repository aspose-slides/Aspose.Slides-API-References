## ILoadOptions.DeleteEmbeddedBinaryObjects свойство

Определяет, будет ли Aspose.Slides удалять все встроенные бинарные объекты при загрузке презентации.

Типы встроенных бинарных объектов:

* VBA проект [`VbaProject`](../../ipresentation/vbaproject)
* Встроенные данные OLE объекта [`EmbeddedFileData`](../../ioleembeddeddatainfo/embeddedfiledata)
* Бинарные данные ActiveX контроля [`ActiveXControlBinary`](../../icontrol/activexcontrolbinary)

Чтение/запись Boolean.

```csharp
public bool DeleteEmbeddedBinaryObjects { get; set; }
```

### Примечания

По умолчанию равно **false**.

### Примеры

Следующий пример показывает, как загрузить презентацию без каких-либо встроенных бинарных объектов.

```csharp
[C#]
LoadOptions loadOptions = new LoadOptions();
loadOptions.DeleteEmbeddedBinaryObjects = true;
using (Presentation pres = new Presentation("pres.ppt", loadOptions))
{
    pres.Save("output_WithoutBinaryObjects.ppt", SaveFormat.Ppt);
}
```

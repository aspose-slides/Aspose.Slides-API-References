## ILoadOptions.OnlyLoadDocumentProperties property

Это свойство имеет смысл, если файл презентации защищен паролем. Значение true означает, что необходимо загрузить только свойства документа из зашифрованного файла презентации, и пароль должен быть проигнорирован. Значение false означает, что вся зашифрованная презентация должна быть загружена с использованием правильного пароля. Если презентация не зашифрована, то значение свойства всегда игнорируется. Если свойства документа зашифрованного файла не являются общедоступными, и значение свойства true, то свойства документа не могут быть загружены, и будет выброшено исключение. Читаемое и записываемое булево значение.

```csharp
public bool OnlyLoadDocumentProperties { get; set; }
```

## ILoadOptions.OnlyLoadDocumentProperties property

Это свойство имеет смысл, если файл презентации защищен паролем. Значение true означает, что из зашифрованного файла презентации должны быть загружены только свойства документа, а пароль должен игнорироваться. Значение false означает, что вся зашифрованная презентация должна загружаться с использованием правильного пароля. Если презентация не зашифрована, то значение свойства всегда игнорируется. Если свойства документа зашифрованного файла не являются общедоступными и значение свойства равно true, то свойства документа не могут быть загружены, и будет выдано исключение. Чтение-записьBoolean.

```csharp
public bool OnlyLoadDocumentProperties { get; set; }
```

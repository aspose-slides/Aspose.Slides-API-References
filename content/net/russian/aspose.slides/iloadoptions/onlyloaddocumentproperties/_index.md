## ILoadOptions.OnlyLoadDocumentProperties свойство

Это свойство имеет смысл, если файл презентации защищён паролем. Значение true означает, что необходимо загрузить только свойства документа из зашифрованного файла презентации, а пароль должен быть проигнорирован. Значение false означает, что вся зашифрованная презентация должна быть загружена с использованием правильного пароля. Если презентация не зашифрована, то значение свойства всегда игнорируется. Если свойства документа зашифрованного файла не являются общедоступными и значение свойства true, то свойства документа не могут быть загружены и будет выброшено исключение. Чтение-запись Boolean.

```csharp
public bool OnlyLoadDocumentProperties { get; set; }
```

## Свойство Portion.PortionFormat

Возвращает объект форматирования, который содержит явно установленные свойства форматирования текстовой порции без применения наследования. Только для чтения [`IPortionFormat`](../../iportionformat).

```csharp
public IPortionFormat PortionFormat { get; }
```

### Примечания

Объект форматирования содержит параметры форматирования, определенные только для текущей порции, данные, унаследованные от родительских объектов, не применяются.

Чтобы получить эффективные значения, включая унаследованные, используйте метод [`GetEffective`](../../portionformat/geteffective).

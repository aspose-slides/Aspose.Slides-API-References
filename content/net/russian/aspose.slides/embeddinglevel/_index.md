## Перечисление EmbeddingLevel

Описывает лицензионные права на встраивание шрифта.

```csharp
[Flags]
public enum EmbeddingLevel : ushort
```

### Значения

| Название | Значение | Описание |
| --- | --- | --- |
| Installable | `0` | Шрифты с этой настройкой указывают, что они могут быть встроены и постоянно установлены на удаленной системе приложением. Пользователь удаленной системы получает те же права, обязательства и лицензии на этот шрифт, что и первоначальный покупатель шрифта, и подчиняется тем же лицензионным соглашениям для конечного пользователя, авторским правам, патентам на дизайн и/или товарным знакам, как и первоначальный покупатель. |
| Restricted | `2` | Шрифты, у которых установлен только этот бит, не должны изменяться, встраиваться или обмениваться любым образом без предварительного получения разрешения законного владельца. |
| PreviewPrint | `4` | Когда этот бит установлен, шрифт может быть встроен и временно загружен на удаленной системе. Документы, содержащие шрифты Preview &amp; Print, должны открываться в режиме "только для чтения"; изменения в документе не могут быть применены. |
| Editable | `8` | Когда этот бит установлен, шрифт может быть встроен, но должен быть установлен только временно на других системах. В отличие от шрифтов Preview &amp; Print, документы, содержащие редактируемые шрифты, могут быть открыты для чтения, редактирование разрешено, и изменения могут быть сохранены. |
| NoSubsetting | `100` | Когда этот бит установлен, шрифт не может быть разделен на подмножества перед встраиванием. Другие ограничения на встраивание, указанные в битах 0-3 и 9, также применяются. |
| BitmapOnly | `200` | Когда этот бит установлен, могут быть встроены только битмап изображения, содержащиеся в шрифте. Данные о контурах не могут быть встроены. Если в шрифте нет доступных битмапов, то шрифт считается не подлежащим встраиванию, и услуги встраивания потерпят неудачу. |

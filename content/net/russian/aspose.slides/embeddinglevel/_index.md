## Перечисление EmbeddingLevel

Представляет права лицензирования для встраивания шрифта.

```csharp
[Flags]
public enum EmbeddingLevel : ushort
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Installable | `0` | Шрифты с этой настройкой указывают, что они могут быть встроены и постоянно установлены на удалённой системе приложением. Пользователь удалённой системы получает такие же права, обязательства и лицензии на этот шрифт, как и оригинальный покупатель шрифта, и подчиняется тем же условиям лицензионного соглашения конечного пользователя, авторскому праву, патенту на дизайн и/или торговой марке, что и оригинальный покупатель. |
| Restricted | `2` | Шрифты, у которых установлен только этот бит, не должны изменяться, встраиваться или обмениваться каким-либо образом без предварительного получения разрешения от законного владельца. |
| PreviewPrint | `4` | Когда этот бит установлен, шрифт может быть встроен и временно загружен на удалённой системе. Документы, содержащие шрифты Preview &amp; Print, должны открываться только для чтения; изменения в документ нельзя вносить. |
| Editable | `8` | Когда этот бит установлен, шрифт может быть встроен, но должен устанавливаться только временно на других системах. В отличие от шрифтов Preview &amp; Print, документы, содержащие шрифты Editable, могут открываться для чтения, редактирование разрешено, и изменения могут быть сохранены. |
| NoSubsetting | `100` | Когда этот бит установлен, шрифт не может быть субсетирован перед встраиванием. Применяются и другие ограничения встраивания, указанные в битах 0-3 и 9. |
| BitmapOnly | `200` | Когда этот бит установлен, могут быть встроены только растровые изображения, содержащиеся в шрифте. Данные контуров встраиваться не могут. Если в шрифте нет растровых изображений, шрифт считается не встраиваемым, и службы встраивания завершатся ошибкой. |

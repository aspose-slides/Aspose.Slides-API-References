## IPresentationInfo.CheckWriteProtection метод

Проверяет, верен ли пароль для модификации защищенной от записи презентации.

```csharp
public bool CheckWriteProtection(string password)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| password | String | Пароль для проверки. |

### Значение Возврата

True, если презентация защищена от записи и пароль верен. False в противном случае.

### Исключения

| исключение | условие |
| --- | --- |
| InvalidOperationException |  |

### Замечания

1. Перед вызовом этого метода следует проверить свойство [`IsWriteProtected`](../iswriteprotected). 2. Если пароль равен null или пустой, этот метод возвращает false.

### Примеры

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
if (info.IsWriteProtected == NullableBool.True)
{
    bool isWriteProtectedByPassword = info.CheckWriteProtection("my_password");
}
```

## IPresentationInfo.UpdateDocumentProperties метод  

Обновляет свойства связанной презентации.  

```csharp  
public void UpdateDocumentProperties(IDocumentProperties documentProperties)  
```  

| Параметр | Тип | Описание |  
| --- | --- | --- |  
| documentProperties | IDocumentProperties | Свойства документа [`IDocumentProperties`](../../idocumentproperties) |  

### Примеры  

Этот пример демонстрирует, как вызвать метод `UpdateDocumentProperties` для обновления свойств документа, возвращаемых при вызове метода [`ReadDocumentProperties`](../readdocumentproperties).  

```csharp  
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo("pres.pptx");  
IDocumentProperties props = info.ReadDocumentProperties();  
props.Subject = "Новый предмет";  
props.LastSavedTime = DateTime.UtcNow;  
info.UpdateDocumentProperties(props);  
info.WriteBindedPresentation("new_pres.pptx");  
```

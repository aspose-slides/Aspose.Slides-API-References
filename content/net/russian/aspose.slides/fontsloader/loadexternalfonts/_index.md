## FontsLoader.LoadExternalFonts метод

Добавляет дополнительные папки для поиска шрифтов.

```csharp
public static void LoadExternalFonts(string[] directories)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| directories | String[] | Папки для чтения дополнительных шрифтов. |

### Примеры

Следующий пример показывает, как загрузить пользовательские шрифты из .TTF

```csharp
[C#]
// Путь к директории документов.
string dataDir = "C:\\";
// папки для поиска шрифтов
String[] folders = new String[] { dataDir };
// Загрузить шрифты из директории пользовательских шрифтов
FontsLoader.LoadExternalFonts(folders);
// Выполнить несколько задач и отобразить презентацию/слайды
using (Presentation presentation = new Presentation(dataDir + "DefaultFonts.pptx"))
{
    presentation.Save(dataDir + "NewFonts_out.pptx", SaveFormat.Pptx);
	// Очистить кэш шрифтов
	FontsLoader.ClearCache();
}
```

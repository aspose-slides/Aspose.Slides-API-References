## IShapeFrame.Rotation property

Возвращает количество градусов, на которое рамка повёрнута вокруг оси z. Положительное значение указывает на вращение по часовой стрелке; отрицательное значение указывает на вращение против часовой стрелки. Только для чтения, Single.

public float Rotation { get; }

## Свойство IShapeFrame.Rotation

Возвращает количество градусов, на которое фрейм вращается вокруг оси z. Положительное значение указывает на вращение по часовой стрелке; отрицательное значение указывает на вращение против часовой стрелки. Свойство только для чтения типа Single.

public float Rotation { get; }

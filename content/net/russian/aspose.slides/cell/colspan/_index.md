## Cell.ColSpan свойство

Возвращает количество столбцов сетки в таблице родителя, которые будут охвачены текущей ячейкой. Это свойство позволяет ячейкам выглядеть так, словно они объединены, так как они охватывают вертикальные границы других ячеек в таблице. Только для чтения Int32.

```csharp
public int ColSpan { get; }
```

## BaseSlide.ShowMasterShapes property

Указывает, должны ли фигуры на основном слайде отображаться на слайдах или нет. Для самого основного слайда это свойство всегда возвращает `false`. Читаемое/записываемое булево значение.

```csharp
public abstract bool ShowMasterShapes { get; set; }
```

### Исключения

| exception | condition |
| --- | --- |
| NotSupportedException | Выбрасывается, если установить `true` для основного слайда. |

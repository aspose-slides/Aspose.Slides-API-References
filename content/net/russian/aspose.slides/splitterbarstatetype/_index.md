## Перечисление SplitterBarStateType

Указывает состояние, в котором должна отображаться разделительная полоса. Разделительная полоса разделяет первичную и вторичную области внутри области просмотра.

```csharp
public enum SplitterBarStateType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Minimized | `0` | Первичная область занимает наименьшую часть области просмотра, допускаемую приложением. |
| Restored | `1` | Первичная область имеет определённый промежуточный размер. |
| Maximized | `2` | Первичная область занимает наибольшую часть области просмотра, допускаемую приложением. |

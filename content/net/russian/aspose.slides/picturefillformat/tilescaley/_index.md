## PictureFillFormat.TileScaleY свойство  

Возвращает или устанавливает вертикальный масштаб для текстурной заливки в процентах. Чтение/запись Single.  

```csharp  
public float TileScaleY { get; set; }  
```  

### Примеры  

```csharp  
[C#]  
using (Presentation presentation = new Presentation("demo.pptx"))  
{  
    ISlide slide = presentation.Slides[0];  

    // Получает формат заливки картинки формы  
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;  

    // Устанавливает режим заливки картинки на Плитка  
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;  

    // Устанавливает вертикальный масштаб для текстуры на 120 процентов  
    pictureFillFormat.TileScaleY = 120;  
}  
```

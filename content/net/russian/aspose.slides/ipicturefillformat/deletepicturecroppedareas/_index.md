## IPictureFillFormat.DeletePictureCroppedAreas метод

Удалить обрезанные области заливки изображения.

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### Значение Возврата

Обрезанное изображение или оригинальное изображение, если обрезка не требуется.

### Замечания

Этот метод конвертирует метафайлы WMF/EMF в растровое изображение PNG во время обрезки.

### Примеры

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Получает PictureFrame
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // Удаляет обрезанные области изображения PictureFrame
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

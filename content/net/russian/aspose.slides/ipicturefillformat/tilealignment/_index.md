## IPictureFillFormat.TileAlignment свойство

Возвращает или устанавливает, как текстура выровнена внутри фигуры. Эта настройка контролирует начальную точку текстурного узора и то, как он повторяется по всей фигуре. Чтение/запись [`RectangleAlignment`](../../rectanglealignment).

```csharp
public RectangleAlignment TileAlignment { get; set; }
```

### Примечания

По умолчанию - TopLeft.

### Примеры

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Получает формат заполнения изображения фигуры
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Устанавливает режим заполнения изображения на Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Устанавливает выравнивание для повторения в правом нижнем углу
    pictureFillFormat.TileAlignment = RectangleAlignment.BottomRight;
}
```

## IPictureFillFormat.TileOffsetY свойство

Возвращает или устанавливает вертикальный сдвиг текстуры от начала формы в пунктах. Положительное значение перемещает текстуру вниз, а отрицательное значение - вверх. Чтение/запись Single.

```csharp
public float TileOffsetY { get; set; }
```

### Примеры

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Получает формат заливки изображения формы
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Устанавливает режим заливки изображения в Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Устанавливает вертикальный сдвиг текстуры в -50 пунктов
    pictureFillFormat.TileOffsetY = -50;
}
```

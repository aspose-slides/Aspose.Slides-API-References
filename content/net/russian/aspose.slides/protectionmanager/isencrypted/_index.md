## ProtectionManager.IsEncrypted свойство

Получает значение, указывающее, зашифрована ли эта instancia. Только для чтения Boolean.

```csharp
public bool IsEncrypted { get; }
```

### Значение свойства

`true`, если презентация была загружена из зашифрованного файла или была вызвана метод [`Encrypt`](../encrypt); в противном случае - `false`.

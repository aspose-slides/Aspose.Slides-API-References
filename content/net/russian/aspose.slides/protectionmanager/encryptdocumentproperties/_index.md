## ProtectionManager.EncryptDocumentProperties property

Это свойство имеет смысл, если презентация защищена паролем. Если значение истинно, тогда свойства документа зашифрованы в файле презентации. Если значение ложно, тогда свойства документа открыты, в то время как презентация зашифрована. Читаемое/записываемое логическое.

```csharp
public bool EncryptDocumentProperties { get; set; }
```

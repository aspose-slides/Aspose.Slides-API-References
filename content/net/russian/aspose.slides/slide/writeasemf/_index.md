## Метод Slide.WriteAsEmf

Сохраняет содержимое слайда в файл EMF.

```csharp
public void WriteAsEmf(Stream stream)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| stream | Stream | Целевой поток |

### Исключения

| исключение | условие |
| --- | --- |
| ArgumentNullException | Целевой поток равен `null` |

### Примеры

В следующем примере кода показано, как преобразовать первый слайд из презентации PowerPoint в метафайл.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.emf"))
    {
        // Сохраняет первый слайд как метафайл
        pres.Slides[0].WriteAsEmf(fileStream);
    }
}
```

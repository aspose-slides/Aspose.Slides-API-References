## CommentCollection.AddModernComment метод

Добавить новый современный комментарий в конце коллекции.

```csharp
public IModernComment AddModernComment(string text, ISlide slide, IShape shape, PointF position, 
    DateTime creationTime)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| text | String | Обычный текст нового современного комментария. |
| slide | ISlide | Слайд в презентации, куда будет добавлен новый современный комментарий. |
| shape | IShape | Форма на слайде, с которой будет связан новый современный комментарий. |
| position | PointF | Позиция на слайде, куда будет добавлен новый современный комментарий. |
| creationTime | DateTime | Время создания современного комментария. |

### Возвращаемое значение

Добавленный современный комментарий.

### Примеры

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ICommentAuthor newAuthor = pres.CommentAuthors.AddAuthor("Некоторый Автор", "SA");
    newAuthor.Comments.AddModernComment("Это современный комментарий", pres.Slides[0], null, new PointF(100, 100), DateTime.Now);

    pres.Save(outPptxFileName, SaveFormat.Pptx);
}
```

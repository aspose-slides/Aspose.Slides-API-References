## ILayoutPlaceholderManager.AddChartPlaceholder метод

Добавляет новую заполнительную фигуру на слайде макета для хранения диаграммы.

```csharp
public IAutoShape AddChartPlaceholder(float x, float y, float width, float height)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| x | Single | Координата X новой заполнительной фигуры. |
| y | Single | Координата Y новой заполнительной фигуры. |
| width | Single | Ширина новой заполнительной фигуры. |
| height | Single | Высота новой заполнительной фигуры. |

### Возвращаемое значение

Созданный [`IAutoShape`](../../iautoshape) с диаграммой в качестве заполнителя.

### Примеры

Следующий пример демонстрирует, как добавить заполнитель для диаграммы на слайд макета.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddChartPlaceholder(20, 20, 200, 200);
}
```

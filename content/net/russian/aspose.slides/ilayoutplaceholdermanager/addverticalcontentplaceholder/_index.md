## ILayoutPlaceholderManager.AddVerticalContentPlaceholder метод

Добавляет новую форму заполнитель в слайд макета для размещения содержимого, такого как изображение, таблица, медиа или текст в вертикальном направлении.

```csharp
public IAutoShape AddVerticalContentPlaceholder(float x, float y, float width, float height)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| x | Single | Координата X новой формы заполнителя. |
| y | Single | Координата Y новой формы заполнителя. |
| width | Single | Ширина новой формы заполнителя. |
| height | Single | Высота новой формы заполнителя. |

### Значение Возврата

Созданный [`IAutoShape`](../../iautoshape) с заполнителем Содержимое (Вертикальный).

### Примеры

Следующий пример показывает, как добавить форму заполнителя Содержимое (Вертикальный) на слайд макета.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalContentPlaceholder(20, 20, 300, 500);
}
```

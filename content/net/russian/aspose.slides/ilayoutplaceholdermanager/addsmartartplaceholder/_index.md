## Метод ILayoutPlaceholderManager.AddSmartArtPlaceholder

Добавляет новую заполненную фигуру на слайд макета для размещения диаграммы SmartArt.

```csharp
public IAutoShape AddSmartArtPlaceholder(float x, float y, float width, float height)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| x | Single | Координата X новой заполненной фигуры. |
| y | Single | Координата Y новой заполненной фигуры. |
| width | Single | Ширина новой заполненной фигуры. |
| height | Single | Высота новой заполненной фигуры. |

### Возвращаемое значение

Созданный [`IAutoShape`](../../iautoshape) с заполнителем SmartArt.

### Примеры

Следующий пример демонстрирует, как добавить фигуру-заполнитель SmartArt на слайд макета.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddSmartArtPlaceholder(20, 20, 200, 200);
}
```

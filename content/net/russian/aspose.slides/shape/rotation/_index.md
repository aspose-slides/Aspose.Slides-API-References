## Shape.Rotation property

Возвращает или устанавливает количество градусов, на которое заданная форма повёрнута вокруг оси z. Положительное значение указывает на поворот по часовой стрелке; отрицательное значение указывает на поворот против часовой стрелки. Чтение/запись Single.

```csharp
public float Rotation { get; set; }
```

### Remarks

Возвращаемое значение всегда определено (не является float.NaN). Присвоенное значение должно быть определено (не является float.NaN). Вы можете установить неопределённые значения для свойств экземпляров RawFrame.

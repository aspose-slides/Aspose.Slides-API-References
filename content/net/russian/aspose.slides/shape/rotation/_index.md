## Свойство Shape.Rotation

Возвращает или устанавливает количество градусов, на которое заданная фигура вращается вокруг оси z. Положительное значение указывает на вращение по часовой стрелке; отрицательное значение указывает на вращение против часовой стрелки. Чтение/запись Single.

```csharp
public float Rotation { get; set; }
```

### Примечания

Возвращаемое значение всегда определено (не равно float.NaN). Присвоенное значение должно быть определено (не равно float.NaN). Вы можете установить неопределенные значения для свойств экземпляров RawFrame.

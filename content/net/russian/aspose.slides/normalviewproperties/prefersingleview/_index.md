## NormalViewProperties.PreferSingleView свойство

Указывает, предпочитает ли пользователь видеть область содержимого в полном окне в сравнении со стандартным нормальным представлением с тремя областями содержимого. Если включено, приложение может выбрать отображение одной из областей содержимого на весь экран. Читаемое/записываемое логическое значение.

public bool PreferSingleView { get; set; }

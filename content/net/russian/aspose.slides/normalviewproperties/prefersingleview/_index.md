## NormalViewProperties.PreferSingleView свойство

Указывает, предпочитает ли пользователь видеть полноэкранный регион с одним содержимым вместо стандартного нормального представления с тремя регионами содержимого. Если включено, приложение может выбрать отображение одного из регионов содержимого на всем экране. Доступно для чтения/записи Boolean.

```csharp
public bool PreferSingleView { get; set; }
```

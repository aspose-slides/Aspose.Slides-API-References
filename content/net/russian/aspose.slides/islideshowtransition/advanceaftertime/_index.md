## ISlideShowTransition.AdvanceAfterTime свойство

Указывает время в миллисекундах, после которого должно начаться переключение. Этот параметр можно использовать в сочетании с атрибутом advClick. Если этот атрибут не указан, предполагается, что автопереключение не произойдет. Читаемо-записываемый UInt32.

```csharp
public uint AdvanceAfterTime { get; set; }
```

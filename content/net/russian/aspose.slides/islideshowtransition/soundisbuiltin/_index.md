## ISlideShowTransition.SoundIsBuiltIn свойство

Указывает, является ли этот звук встроенным. Если этот атрибут установлен в true, то приложение, генерирующее этот звук, получает уведомление, чтобы проверить атрибут имени, указанный для этого звука в его списке встроенных звуков, и может при необходимости предоставить пользовательское имя или интерфейс. Читаемое и записываемое булево значение.

```csharp
public bool SoundIsBuiltIn { get; set; }
```

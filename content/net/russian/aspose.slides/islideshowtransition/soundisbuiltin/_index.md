## ISlideShowTransition.SoundIsBuiltIn свойство

Указывает, является ли этот звук встроенным звуком. Если этот атрибут установлен в значение true, то генерирующее приложение получает уведомление для проверки атрибута имени, указанного для этого звука в его списке встроенных звуков, и при необходимости может отобразить настраиваемое имя или пользовательский интерфейс. Читаемое и записываемое логическое значение.

public bool SoundIsBuiltIn { get; set; }

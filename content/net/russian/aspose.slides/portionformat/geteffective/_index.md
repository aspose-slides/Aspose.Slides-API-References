## Метод PortionFormat.GetEffective

Получает данные о форматировании эффективной части с примененным наследованием.

```csharp
public IPortionFormatEffectiveData GetEffective()
```

### Возвращаемое значение

[`IPortionFormatEffectiveData`](../../iportionformateffectivedata).

### Примеры

Этот пример демонстрирует получение некоторых свойств эффективного форматирования части.

```csharp
[C#]
using (Presentation pres = new Presentation(@"MyPresentation.pptx"))
{
	IAutoShape shape = pres.Slides[0].Shapes[0] as IAutoShape;
	IPortionFormatEffectiveData effectivePortionFormat = shape.TextFrame.Paragraphs[0].Portions[0].PortionFormat.GetEffective();

	Console.WriteLine("Латинский шрифт: " + effectivePortionFormat.LatinFont.FontName);
	Console.WriteLine("Высота шрифта: " + effectivePortionFormat.FontHeight);
	Console.WriteLine("Тип заливки: " + effectivePortionFormat.FillFormat.FillType);
}
```

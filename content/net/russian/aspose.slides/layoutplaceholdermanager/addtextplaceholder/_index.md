## LayoutPlaceholderManager.AddTextPlaceholder метод

Добавляет новую заполнительную фигуру на слайд макета для удержания текстового содержимого.

```csharp
public IAutoShape AddTextPlaceholder(float x, float y, float width, float height)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| x | Single | Координата X новой заполнительной фигуры. |
| y | Single | Координата Y новой заполнительной фигуры. |
| width | Single | Ширина новой заполнительной фигуры. |
| height | Single | Высота новой заполнительной фигуры. |

### Возврат значения

Созданный [`IAutoShape`](../../iautoshape) с текстовым заполнительным элементом.

### Примеры

Следующий пример показывает, как добавить фигуру текстового заполнителя на слайд макета.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTextPlaceholder(20, 20, 500, 300);
}
```

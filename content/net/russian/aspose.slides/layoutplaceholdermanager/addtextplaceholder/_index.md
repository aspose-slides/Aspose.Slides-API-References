## LayoutPlaceholderManager.AddTextPlaceholder метод

Добавляет новую форму для заполнителя на слайд макета для хранения текстового содержимого.

```csharp
public IAutoShape AddTextPlaceholder(float x, float y, float width, float height)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| x | Single | Координата X новой формы для заполнителя. |
| y | Single | Координата Y новой формы для заполнителя. |
| width | Single | Ширина новой формы для заполнителя. |
| height | Single | Высота новой формы для заполнителя. |

### Возвращаемое значение

Созданный [`IAutoShape`](../../iautoshape) с текстовым заполнителем.

### Примеры

Следующий пример показывает, как добавить форму заполнителя текста на слайд макета.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTextPlaceholder(20, 20, 500, 300);
}
```

## LayoutPlaceholderManager.AddVerticalTextPlaceholder метод

Добавляет новую форму-заполнитель на слайд макета для размещения текстового содержимого в вертикальном направлении.

```csharp
public IAutoShape AddVerticalTextPlaceholder(float x, float y, float width, float height)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| x | Single | X координата новой формы-заполнителя. |
| y | Single | Y координата новой формы-заполнителя. |
| width | Single | Ширина новой формы-заполнителя. |
| height | Single | Высота новой формы-заполнителя. |

### Возвращаемое значение

Созданный [`IAutoShape`](../../iautoshape) с текстовым (вертикальным) заполнителем.

### Примеры

Следующий пример показывает, как добавить форму-заполнитель текста (вертикального) на слайд макета.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalTextPlaceholder(20, 20, 300, 500);
}
```

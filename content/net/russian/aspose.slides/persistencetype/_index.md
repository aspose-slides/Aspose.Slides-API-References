## Перечисление PersistenceType

Указывает метод, используемый для хранения свойств элемента управления ActiveX.

```csharp
public enum PersistenceType
```

### Значения

| Название | Значение | Описание |
| --- | --- | --- |
| NotDefined | `-1` | Идентификатор сохранения не указан. |
| PersistPropertyBag | `0` | Указывает, что элемент управления ActiveX сохраняется с использованием сохранения на основе набора свойств. Сохранение на основе набора свойств хранит элемент управления ActiveX с помощью коллекции пар имя и значение, которые указывают данные, сохраненные элементом управления ActiveX. |
| PersistStream | `1` | Указывает, что элемент управления ActiveX сохраняется с использованием сохранения на основе потока, которое не поддерживает инициализацию элемента управления ActiveX в состояние по умолчанию. |
| PersistStreamInit | `2` | Указывает, что элемент управления ActiveX сохраняется с использованием сохранения на основе потока, которое поддерживает инициализацию элемента управления ActiveX в состояние по умолчанию. |
| PersistStorage | `3` | Указывает, что элемент управления ActiveX сохраняется с использованием хранения на основе сохранения. |

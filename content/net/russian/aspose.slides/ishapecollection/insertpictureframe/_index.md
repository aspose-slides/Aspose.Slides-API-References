## IShapeCollection.InsertPictureFrame метод

```csharp
public IPictureFrame InsertPictureFrame(int index, ShapeType shapeType, float x, float y, 
    float width, float height, IPPImage image)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| index | Int32 | Индекс, начиная с нуля, по которому должно быть вставлено значение. |
| shapeType | ShapeType | Форма, содержащаяся в установленном наборе [`ShapeType`](../../shapetype) форм, кроме всех видов линий: |

| width | Single | Ширина рамки формы. |
| height | Single | Высота рамки формы. |

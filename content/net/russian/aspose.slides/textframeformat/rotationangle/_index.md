## TextFrameFormat.RotationAngle свойство

Указывает пользовательский угол поворота, который применяется к тексту в пределах ограничивающего прямоугольника. Если он не указан, используется угол поворота сопутствующей фигуры. Если он указан, то это применяется независимо от фигуры. То есть фигура может иметь примененный угол поворота в дополнение к тому, что сам текст имеет примененный угол поворота. Полученное значение визуального поворота текста суммируется из этого свойства и предопределенного вертикального типа в свойстве TextVerticalType. Чтение/запись Single.

```csharp
public float RotationAngle { get; set; }
```

### Примеры

Рассмотрим случай, когда фигура имеет угол поворота 90 градусов по часовой стрелке. В дополнение к этому, сам текст имеет угол поворота -90 градусов против часовой стрелки. Тогда результирующая фигура будет выглядеть повёрнутой, но текст внутри неё будет выглядеть так, будто он вообще не был повёрнут.

## Свойство IBackdrop3DScene.UpVector

Возвращает или устанавливает вектор, представляющий направление вверх. Более точно, этот атрибут определяет вектор, представляющий направление вверх относительно лица заднего плана. Вектор представлен массивом из 3 значений с плавающей запятой, которые определяют координаты X, Y и Z. Чтение/запись Single[].

public float[] UpVector { get; set; }

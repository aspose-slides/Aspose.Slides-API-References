## ShapeFrame.Rotation property

Возвращает количество градусов, на которое рамка повернута вокруг оси z. Положительное значение указывает на поворот по часовой стрелке; отрицательное значение указывает на поворот против часовой стрелки. Только для чтения Single.

```csharp
public float Rotation { get; }
```

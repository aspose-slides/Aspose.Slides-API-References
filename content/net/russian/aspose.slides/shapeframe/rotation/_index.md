## Свойство ShapeFrame.Rotation

Возвращает количество градусов, на которое рамка повернута вокруг оси z. Положительное значение указывает на вращение по часовой стрелке; отрицательное значение указывает на вращение против часовой стрелки. Только для чтения Single.

## LoadOptions.OnlyLoadDocumentProperties свойство

Это свойство имеет смысл, если файл презентации защищен паролем. Значение true означает, что только свойства документа должны быть загружены из зашифрованного файла презентации, и пароль должен быть проигнорирован. Значение false означает, что вся зашифрованная презентация должна быть загружена с использованием правильного пароля. Если презентация не зашифрована, то значение свойства всегда игнорируется. Если свойства документа зашифрованного файла не являются публичными и значение свойства истинно, то свойства документа не могут быть загружены, и будет вызвано исключение. Чтение/запись Boolean.

```csharp
public bool OnlyLoadDocumentProperties { get; set; }
```

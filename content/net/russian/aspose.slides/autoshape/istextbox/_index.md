## AutoShape.IsTextBox свойство

Указывает, является ли фигура текстовым полем.

```csharp
public bool IsTextBox { get; }
```

### Примечания

Если фигура не указана как текстовое поле, это не означает, что к ней не может быть прикреплен текст. Текстовое поле - это специализированная фигура с конкретными свойствами.

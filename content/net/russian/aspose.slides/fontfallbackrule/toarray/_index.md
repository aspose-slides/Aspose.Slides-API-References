
| Параметр | Тип | Описание |
| --- | --- | --- |
| startIndex | Int32 | Индекс первого добавляемого шрифта. |
| count | Int32 | Количество шрифтов для добавления. |

### Возвращаемое значение

МассивString

```csharp
[C#]
```

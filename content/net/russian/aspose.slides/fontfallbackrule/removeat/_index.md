## FontFallBackRule.RemoveAt method

Удаляет шрифт FallBack по указанному индексу списка.

```csharp
public void RemoveAt(int index)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| index | Int32 | Начинающийся с нуля индекс шрифта для удалять. |

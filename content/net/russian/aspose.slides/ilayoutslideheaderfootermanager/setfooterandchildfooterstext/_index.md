## ILayoutSlideHeaderFooterManager.SetFooterAndChildFootersText метод

Устанавливает текст для текстового поля нижнего колонтитула слайда макета и всех дочерних текстовых полей нижнего колонтитула. Дочерние текстовые поля означают, что поля содержатся на зависимых слайдах. Зависимые слайды используют и зависят от слайда макета.

```csharp
public void SetFooterAndChildFootersText(string text)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| text | String | Текст для установки. |

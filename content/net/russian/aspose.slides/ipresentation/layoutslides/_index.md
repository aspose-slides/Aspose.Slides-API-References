## IPresentation.LayoutSlides свойство  
  
Возвращает список всех макетов слайдов, определённых в презентации. Только для чтения [`IGlobalLayoutSlideCollection`](../../igloballayoutslidecollection).

public IGlobalLayoutSlideCollection LayoutSlides { get; }

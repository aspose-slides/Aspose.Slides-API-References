## MasterLayoutSlideCollection.InsertClone метод  
  
Вставляет копию указанного слайда макета в заданную позицию коллекции.  
  
```csharp  
public ILayoutSlide InsertClone(int index, ILayoutSlide sourceLayout)  
```  
  
| Параметр | Тип | Описание |  
| --- | --- | --- |  
| index | Int32 | Индекс нового слайда. |  
| sourceLayout | ILayoutSlide | Слайд для клонирования. |  
  
### Возвращаемое значение  
  
Вставленный слайд.  
  
### Замечания  
  
Новый макет будет связан с родительским слайдом-мастером для этой коллекции слайдов макета. Таким образом, это аналог функции копирования/вставки с опцией "Использовать тему назначения" в PowerPoint.

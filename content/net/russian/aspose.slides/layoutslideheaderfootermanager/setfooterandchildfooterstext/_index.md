## LayoutSlideHeaderFooterManager.SetFooterAndChildFootersText метод

Устанавливает текст для заполнитель нижнего колонтитула слайда макета и всех заполнителей нижних колонтитулов дочерних слайдов. Дочерние заполнители обозначают заполнители, которые содержатся на зависимых слайдах. Зависимые слайды используют и зависят от слайда макета.

```csharp
public void SetFooterAndChildFootersText(string text)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| text | String | Текст для установки. |

## IHyperlink.TargetSlide свойство

Если HyperlinkEx указывает на конкретный слайд, возвращается этот слайд. Если свойство становится не нулевым, то свойство ExternalUrl становится нулевым. Только для чтения [`ISlide`](../../islide).

```csharp
public ISlide TargetSlide { get; }
```

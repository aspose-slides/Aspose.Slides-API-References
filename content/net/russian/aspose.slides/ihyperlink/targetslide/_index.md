## Свойство IHyperlink.TargetSlide

Если HyperlinkEx указывает на конкретный слайд, возвращает этот слайд. Если свойство становится не равным null, то свойство ExternalUrl становится равным null. Только для чтения [`ISlide`](../../islide).

```csharp
public ISlide TargetSlide { get; }
```

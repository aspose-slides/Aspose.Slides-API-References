## Свойство ICell.ColSpan

Возвращает количество столбцов сетки в таблице родительского элемента, которые должны быть охвачены текущей ячейкой. Это свойство позволяет ячейкам иметь вид объединенных, так как они пересекают вертикальные границы других ячеек в таблице. Только для чтения Int32.

public int ColSpan { get; }

## ITextStyle.GetLevel метод

Если уровень стиля существует, возвращает его, иначе возвращает null.

```csharp
public IParagraphFormat GetLevel(int index)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| index | Int32 | Индекс уровня, нумерация начинается с нуля. Должен находиться в интервале 0..8. |

### Возвращаемое значение

Формат уровня [`IParagraphFormat`](../../iparagraphformat).

## INormalViewProperties.PreferSingleView property

Указывает, предпочитает ли пользователь видеть полноэкранный регион с одним содержимым по сравнению с обычным стандартным видом с тремя регионами содержимого. Если включено, приложение может выбрать отображение одного из регионов содержимого на весь экран. Свойство типа Boolean для чтения и записи.

```csharp
public bool PreferSingleView { get; set; }
```

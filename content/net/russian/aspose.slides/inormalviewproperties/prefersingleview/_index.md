## INormalViewProperties.PreferSingleView свойство

Указывает, предпочитает ли пользователь видеть полноэкранный регион с единственным содержимым, а не стандартный нормальный вид с тремя регионами содержимого. Если включено, приложение может выбрать отображение одного из регионов содержимого во всем окне. Чтение/запись логическое значение.

public bool PreferSingleView { get; set; }

## IShape.ZOrderPosition property

Возвращает позицию фигуры в порядке z. Shapes[0] возвращает фигуру, находящуюся на заднем плане порядка z, а Shapes[Shapes.Count - 1] возвращает фигуру на переднем плане порядка z. Только для чтения Int32.

public int ZOrderPosition { get; }

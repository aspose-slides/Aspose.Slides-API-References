## IShape.Rotation property

Возвращает или задает количество градусов, на которые заданная форма вращается вокруг оси z. Положительное значение указывает на вращение по часовой стрелке; отрицательное значение указывает на противочасовое вращение. Чтение/запись Single.

```csharp
public float Rotation { get; set; }
```

### Remarks

Возвращаемое значение всегда определено (не является float.NaN). Заданное значение должно быть определенным (не float.NaN). Вы можете устанавливать неопределенные значения для свойств экземпляров RawFrame.

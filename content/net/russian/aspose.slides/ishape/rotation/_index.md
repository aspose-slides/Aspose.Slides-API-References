## Свойство IShape.Rotation

Возвращает или устанавливает количество градусов, на которое заданная фигура поворачивается вокруг оси z. Положительное значение указывает на вращение по часовой стрелке; отрицательное значение указывает на вращение против часовой стрелки. Чтение/запись Single.

```csharp
public float Rotation { get; set; }
```

### Замечания

Возвращаемое значение всегда определено (не является float.NaN). Присвоенное значение должно быть определено (не является float.NaN). Вы можете установить неопределенные значения для свойств экземпляров RawFrame.

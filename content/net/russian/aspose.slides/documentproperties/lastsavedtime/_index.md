## DocumentProperties.LastSavedTime property

Возвращает дату последнего изменения презентации. Значения находятся в UTC. Только для чтения в случае Presentation.DocumentProperties (так как они будут обновляться внутренне в процессе сохранения объекта IPresentation). Может быть изменено через экземпляр DocumentProperties, возвращаемый методом [`ReadDocumentProperties`](../../ipresentationinfo/readdocumentproperties). Пожалуйста, смотрите пример в сводке метода [`UpdateDocumentProperties`](../../ipresentationinfo/updatedocumentproperties).

```csharp
public DateTime LastSavedTime { get; set; }
```

## Метод DocumentProperties.GetCustomPropertyName

Возвращает имя пользовательского свойства по указанному индексу.

```csharp
public string GetCustomPropertyName(int index)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| index | Int32 | Индекс, основанный на нуле, пользовательского свойства, которое нужно получить. |

### Возвращаемое значение

Имя пользовательского свойства по указанному индексу.

### Исключения

| исключение | условие |
| --- | --- |
| ArgumentOutOfRangeException | Индекс меньше нуля. Индекс равен или больше Count. |

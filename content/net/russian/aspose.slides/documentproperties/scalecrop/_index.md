## DocumentProperties.ScaleCrop свойство

Указывает режим отображения миниатюры документа. Установите этот элемент в **true**, чтобы включить масштабирование миниатюры документа по размеру экрана. Установите этот элемент в **false**, чтобы включить обрезку миниатюры документа, чтобы показывать только те части, которые соответствуют размеру экрана. Читаемое/записываемое логическое значение.

public bool ScaleCrop { get; set; }

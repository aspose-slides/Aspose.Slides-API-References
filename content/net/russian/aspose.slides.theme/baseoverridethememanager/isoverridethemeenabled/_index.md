## BaseOverrideThemeManager.IsOverrideThemeEnabled свойство

Определяет, переопределяет ли OverrideTheme унаследованную эффективную тему или нет. Чтобы включить OverrideTheme для переопределения, используйте методы OverrideTheme.Init*(). Чтобы отключить переопределение, используйте метод OverrideTheme.Clear(). Только для чтения Boolean.

public bool IsOverrideThemeEnabled { get; }

## ExternalResourceResolver класс

Callback класс, используемый для разрешения внешних ресурсов во время импорта документов Html, Svg. Использование этого резольвера может создать уязвимость, когда клиент предоставляет HTML- или SVG-файл, заставляющий серверное программное обеспечение получать локальные или сетевые файлы. Используйте с осторожностью. Рекомендуется не указывать ExternalResourceResolver вовсе (будут прочитаны только встроенные объекты) или создать подкласс, который проверяет, является ли указанный uri действительным.

```csharp
public class ExternalResourceResolver : IExternalResourceResolver
```

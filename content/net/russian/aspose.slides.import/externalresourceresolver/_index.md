## ExternalResourceResolver class

Коллбэк класс, используемый для разрешения внешних ресурсов во время импорта документов Html и Svg. Использование этого резольвера может создать уязвимость, когда предоставленный клиентом HTML или SVG файл заставит серверное программное обеспечение получить локальный или сетевой файл. Пользуйтесь осторожно. Рекомендуется вообще не указывать ExternalResourceResolver (будут считаны только встроенные объекты) или создать некоторый подкласс, который проверяет, является ли указанный uri действительным.

```csharp
public class ExternalResourceResolver : IExternalResourceResolver
```

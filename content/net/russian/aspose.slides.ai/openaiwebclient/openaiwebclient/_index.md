## OpenAIWebClient(string, string, string, HttpClient) {#constructor_1}

Создает экземпляр веб-клиента OpenAI, который использует внешне управляемый HttpClient. Переданный HttpClient не будет уничтожен этим экземпляром и останется в собственности вызывающего кода.

```csharp
public OpenAIWebClient(string model, string apiKey, string organizationId, HttpClient httpClient)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| model | String | Модель языка OpenAI. Возможные значения: - gpt-4o - gpt-4o-mini - o1 - o1-mini - o3 - o3-mini |
| apiKey | String | Ключ API OpenAI |
| organizationId | String | Идентификатор организации (необязательно) |
| httpClient | HttpClient | Внешне управляемый `HttpClient` экземпляр |

### Исключения

| исключение | условие |
| --- | --- |
| ArgumentException | Значение ключа API не может быть null или пустым |
| ArgumentException | Значение модели текста не может быть null или пустым |
| ArgumentNullException | HttpClient не может быть null |

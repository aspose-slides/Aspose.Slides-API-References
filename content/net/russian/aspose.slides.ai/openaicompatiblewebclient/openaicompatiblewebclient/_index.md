## OpenAICompatibleWebClient(string, string, string, HttpClient) {#constructor_1}


| Параметр | Тип | Описание |

| apiKey | String | API key (токен). |
| baseUrl | String | Базовый URL совместимого с OpenAI LLM. |
| httpClient | HttpClient | Внешне управляемый экземпляр HttpClient. |

### Исключения

| исключение | условие |

## AsposeAIWebClient класс

Встроенная реализация [`IAIWebClient`](../iaiwebclient), которая соединяется с собственным LLM Aspose. Это клиент по умолчанию, используемый конструктором без параметров [`SlidesAIAgent`](../slidesaiagent/slidesaiagent).

```csharp
public sealed class AsposeAIWebClient : IAIWebClient, IDisposable
```

| [AsposeAIWebClient](asposeaiwebclient#constructor)() | Создаёт экземпляр веб-клиента Aspose AI, который подключается к конечной точке LLM Aspose по умолчанию. Переданный HttpClient не будет освобождён этим экземпляром и остаётся принадлежащим вызывающему коду. |
| [AsposeAIWebClient](asposeaiwebclient#constructor_2)(string) | Создаёт экземпляр веб-клиента Aspose AI, который подключается к пользовательскому URL-адресу конечной точки. Используйте эту перегрузку, когда у вас есть URL, предоставленный командой Aspose.Slides; в противном случае используйте перегрузку [`AsposeAIWebClient`](./asposeaiwebclient) с URL-адресом по умолчанию. |
| [AsposeAIWebClient](asposeaiwebclient#constructor_3)(string, HttpClient) | Создаёт экземпляр веб-клиента Aspose AI, который подключается к пользовательскому URL-адресу конечной точки, используя внешне управляемый HttpClient. Переданный HttpClient не будет освобождён этим экземпляром и остаётся принадлежащим вызывающему коду. Используйте эту перегрузку, когда у вас есть URL, предоставленный командой Aspose.Slides, и вы хотите предоставить свой собственный HttpClient; если вам нужен только свой HttpClient с URL-адресом по умолчанию, используйте перегрузку [`AsposeAIWebClient`](./asposeaiwebclient). |

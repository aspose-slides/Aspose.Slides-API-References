Содержит классы, предоставляющие функции на основе ИИ для анализа и обработки презентаций PowerPoint.

## Классы

| Класс | Описание |
| --- | --- |
| [AsposeAIWebClient](./asposeaiwebclient) | Встроенная реализация [`IAIWebClient`](../aspose.slides.ai/iaiwebclient), которая подключается к собственному LLM Aspose. Это клиент по умолчанию, используемый конструктором без параметров [`SlidesAIAgent`](../aspose.slides.ai/slidesaiagent/slidesaiagent). |
| [OpenAICompatibleWebClient](./openaicompatiblewebclient) | Встроенная реализация [`IAIWebClient`](../aspose.slides.ai/iaiwebclient), которая подключается к совместимому с OpenAI провайдеру LLM по указанному базовому URL. |
| [OpenAIWebClient](./openaiwebclient) | Встроенная реализация [`IAIWebClient`](../aspose.slides.ai/iaiwebclient), которая подключается к API OpenAI. |
| [SlidesAIAgent](./slidesaiagent) | Предоставляет функции, основанные на ИИ, для обработки презентаций. |
| [SlidesAIAgentException](./slidesaiagentexception) | Представляет исключения, связанные с Slides AI Agent. |

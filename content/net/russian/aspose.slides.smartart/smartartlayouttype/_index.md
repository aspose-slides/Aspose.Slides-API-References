## Перечисление SmartArtLayoutType

Представляет тип макета диаграммы SmartArt.

```csharp
public enum SmartArtLayoutType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| AccentProcess | `0` | Используется для отображения прогрессии, временной шкалы или последовательных шагов задачи, процесса или рабочего процесса. Хорошо подходит для иллюстрации как текста первого, так и второго уровня. |
| AccentedPicture | `1` | Используется для отображения центральной, фотографической идеи с связанными идеями сбоку. Текст первого уровня отображается поверх центрального изображения. | AlternatingFlow | `2` | Используется для отображения групп информации или последовательных шагов задачи, процесса или рабочего процесса. Подчеркивает взаимодействие или взаимосвязи между группами информации. |
| AlternatingHexagons | `3` | Используется для представления серии взаимосвязанных идей. Текст первого уровня отображается внутри шестиугольников. Текст второго уровня – снаружи фигур. |

Неиспользованный текст не отображается, но остаётся доступным при смене макета. |
| BasicBendingProcess | `9` | Используется для отображения прогрессии или последовательных шагов задачи, процесса или рабочего процесса. Максимизирует как горизонтальное, так и вертикальное пространство для фигур. |
| BasicBlockList | `10` | Используется для отображения не последовательных или сгруппированных блоков информации. Максимизирует как горизонтальное, так и вертикальное пространство для фигур. |
| BasicChevronProcess | `11` | Используется для отображения прогрессии; временной шкалы; последовательных шагов задачи, процесса или рабочего процесса; или для подчеркивания движения или направления. Текст первого уровня отображается внутри стрелочной фигуры, а текст второго уровня – под стрелочными фигурами. |

| BasicMatrix | `13` | Используется для отображения взаимосвязи компонентов с целым в квадрантах. Первые четыре строки текста первого уровня отображаются в квадрантах. Неиспользованный текст не отображается, но остаётся доступным при смене макета. |
| BasicPie | `14` | Используется для отображения того, как отдельные части образуют целое. Верхняя фигура текста первого уровня размещается снаружи остальной части сектора для акцента. Неиспользованный текст не отображается, но остаётся доступным при смене макета. |
| BasicProcess | `15` | Используется для отображения прогрессии или последовательных шагов задачи, процесса или рабочего процесса. |

Хорошо подходит для иллюстрации как текста первого, так и второго уровня. Максимизирует как горизонтальное, так и вертикальное пространство для фигур. |
| BendingPictureBlocks | `22` | Используется для отображения серии изображений. Поле в нижнем углу может содержать небольшие количества текста. |

| BubblePictureList | `27` | Используется для отображения серии изображений. Может содержать до восьми изображений первого уровня. Неиспользованные текст и изображения не отображаются, но остаются доступными при смене макета. Лучше всего подходит для небольших объёмов текста. |

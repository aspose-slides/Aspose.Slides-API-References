## IInkOptions.HideInk свойство

Показывает или скрывает элементы Ink в экспортированном документе.

```csharp
public bool HideInk { get; set; }
```

### Примечания

Значение по умолчанию - false.

### Примеры

Следующий пример демонстрирует, как скрыть элементы Ink в экспортированном PDF документе:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.HideInk = true;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

## IHtmlOptions.JpegQuality свойство

Возвращает или задает значение, определяющее качество JPEG-изображений в PDF-документе. Чтение/запись Byte.

```csharp
public byte JpegQuality { get; set; }
```

### Замечания

Действует только в том случае, если документ содержит JPEG-изображения.

Используйте это свойство для получения или установки качества изображений в документе при сохранении в формате PDF. Значение может варьироваться от 0 до 100, где 0 означает худшее качество, но максимальное сжатие, а 100 — лучшее качество, но минимальное сжатие.

Значение по умолчанию — **95**.

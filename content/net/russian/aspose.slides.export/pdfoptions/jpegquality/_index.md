## Свойство PdfOptions.JpegQuality

Возвращает или устанавливает значение, определяющее качество изображений JPEG внутри PDF-документа. Чтение/запись Byte.

```csharp
public byte JpegQuality { get; set; }
```

### Примечания

Имеет значение только в том случае, если документ содержит изображения JPEG.

Используйте это свойство для получения или установки качества изображений внутри документа при сохранении в формате PDF. Значение может варьироваться от 0 до 100, где 0 означает худшее качество, но максимальную компрессию, а 100 означает лучшее качество, но минимальную компрессию.

Значение по умолчанию - **100**.

## PdfOptions.SaveMetafilesAsPng свойство

True для конвертации всех метафайлов, используемых в презентации, в изображения PNG. Чтение/запись логическое значение.

```csharp
public bool SaveMetafilesAsPng { get; set; }
```

### Примечания

По умолчанию **true**. PDF документ может содержать векторную графику и растровые изображения. Если SaveMetafilesAsPng установлено в true, то исходное изображение метафайла конвертируется в формат Png и сохраняется в Pdf как растровое изображение. Если SaveMetafilesAsPng установлено в false, то исходный метафайл конвертируется в векторную графику Pdf. У каждого подхода есть свои преимущества и недостатки. Например, если метафайл конвертируется в PNG, то возможна потеря качества при масштабировании созданного документа. Если метафайл конвертируется в векторную графику Pdf, то возможны проблемы с производительностью в инструменте просмотра Pdf.

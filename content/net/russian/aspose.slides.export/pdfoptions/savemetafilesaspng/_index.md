## PdfOptions.SaveMetafilesAsPng свойство

true для преобразования всех метафайлов, используемых в презентации, в изображения PNG. Чтение/запись Булевое значение.

```csharp
public bool SaveMetafilesAsPng { get; set; }
```

### Замечания

По умолчанию **true**. Pdf документ может содержать векторную графику и растровые изображения. Если SaveMetafilesAsPng установлено в true, то исходное изображение метафайла преобразуется в формат Png и сохраняется в Pdf как растровое изображение. Если SaveMetafilesAsPng установлено в false, то исходный метафайл преобразуется в векторную графику Pdf. У каждого подхода есть свои преимущества и недостатки. Например, если метафайл преобразован в PNG, то возможна потеря качества при масштабировании результирующего документа. Если метафайл преобразован в векторную графику Pdf, то возможны проблемы с производительностью в инструменте просмотра Pdf.

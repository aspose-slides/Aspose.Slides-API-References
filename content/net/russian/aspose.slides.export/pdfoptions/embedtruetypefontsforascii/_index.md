## PdfOptions.EmbedTrueTypeFontsForASCII свойство

Определяет, будет ли Aspose.Slides встраивать общие шрифты для текста в диапазоне ASCII (33..127). Шрифты для кодов символов больше 127 всегда встраиваются. Список общих шрифтов включает 14 шрифтов базовой PDF и дополнительные шрифты, указанные пользователем. Чтение/запись логическое значение.

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### Замечания

По умолчанию **true**.

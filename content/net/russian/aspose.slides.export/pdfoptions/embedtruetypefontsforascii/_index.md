## PdfOptions.EmbedTrueTypeFontsForASCII свойство

Определяет, будет ли Aspose.Slides встраивать общие шрифты для текста ASCII (диапазон кодов 33..127). Шрифты для символов с кодами больше 127 всегда встраиваются. Список общих шрифтов включает базовые 14 шрифтов PDF и дополнительные шрифты, указанные пользователем. Чтение/запись логическое значение.

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### Заметки

По умолчанию **true**.

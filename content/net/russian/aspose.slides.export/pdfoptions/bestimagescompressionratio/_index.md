## PdfOptions.BestImagesCompressionRatio свойство

Указывает, следует ли автоматически выбирать наиболее эффективное сжатие (вместо стандартного) для каждого изображения. Если установить значение Boolean.true, для каждого изображения в презентации будет выбран наиболее подходящий алгоритм сжатия, что приведет к уменьшению размера выходного PDF документа. Выбор наилучшего коэффициента сжатия изображения требует вычислительных ресурсов и занимает дополнительное количество оперативной памяти, и эта опция по умолчанию является Boolean.false.

```csharp
public bool BestImagesCompressionRatio { get; set; }
```

### Примечания

По умолчанию значение Boolean.false.

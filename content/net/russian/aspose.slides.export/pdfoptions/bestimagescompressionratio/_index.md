## PdfOptions.BestImagesCompressionRatio свойство

Указывает, должна ли автоматически выбираться наиболее эффективная компрессия (вместо стандартной) для каждого изображения. Если установлено значение Boolean.true, для каждого изображения в презентации будет выбран наиболее подходящий алгоритм сжатия, что приведет к уменьшению размера результирующего PDF-документа. Выбор наилучшего коэффициента сжатия изображения требует значительных вычислительных ресурсов и занимает дополнительное количество ОЗУ, и эта опция по умолчанию установлена на Boolean.false.

```csharp
public bool BestImagesCompressionRatio { get; set; }
```

### Замечания

По умолчанию используется значение Boolean.false.

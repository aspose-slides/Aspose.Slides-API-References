## Перечисление Zip64Mode

Указывает, когда использовать расширения формата ZIP64 для файлов OpenXML.

```csharp
public enum Zip64Mode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Never | `0` | Не использовать расширения формата ZIP64. |
| IfNecessary | `1` | Использовать расширения формата ZIP64, если это необходимо. |
| Always | `2` | Всегда использовать расширения формата ZIP64. |

### Примечания

Файл OpenXML — это ZIP-архив, который имеет ограничение в 4 ГБ (2^32 байт) на не сжатый размер файла, сжатый размер файла и общий размер архива, а также ограничение в 65,535 (2^16-1) файлов в архиве. Расширения формата ZIP64 увеличивают лимиты до 2^64.

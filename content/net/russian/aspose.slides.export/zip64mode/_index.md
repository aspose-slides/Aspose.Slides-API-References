## Zip64Mode перечисление

Указывает, когда использовать расширения формата ZIP64 для файла OpenXML.

```csharp
public enum Zip64Mode
```

### Значения

| Name | Value | Description |
| --- | --- | --- |
| Never | `0` | Не использовать расширения формата ZIP64. |
| IfNecessary | `1` | Использовать расширения формата ZIP64 при необходимости. |
| Always | `2` | Всегда использовать расширения формата ZIP64. |

### Примечания

Файл OpenXML представляет собой ZIP-архив, в котором установлен предел в 4 ГБ (2^32 байт) на несжатый размер файла, сжатый размер файла и общий размер архива, а также предел в 65 535 (2^16-1) файлов в архиве. Расширения формата ZIP64 увеличивают эти пределы до 2^64.

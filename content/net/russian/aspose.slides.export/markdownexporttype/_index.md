## MarkdownExportType перечисление  

Тип документа рендеринга.  

```csharp  
public enum MarkdownExportType  
```  

### Значения  

| Имя | Значение | Описание |  
| --- | --- | --- |  
| Sequential | `0` | Рендерить все элементы по отдельности. Один за другим. |  
| TextOnly | `1` | Рендерить только текст. |  
| Visual | `2` | Рендерить все элементы, группированные элементы - рендерить вместе. |  

### Примеры  

Пример:  

```csharp  
[C#]  
using (Presentation pres = new Presentation("pres.pptx"))  
{  
    MarkdownSaveOptions markdownSaveOptions = new MarkdownSaveOptions  
    {  
        ShowHiddenSlides = true,  
        ShowSlideNumber = true,  
        Flavor = Flavor.Github,  
        ExportType = MarkdownExportType.Sequential,  
        NewLineType = NewLineType.Windows  
    };  
     
    pres.Save("doc.md", new[] { 1, 2, 3, 4, 5, 6, 7, 8, 9 }, SaveFormat.Md, markdownSaveOptions);  
}  
```

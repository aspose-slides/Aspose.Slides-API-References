## ITiffOptions.ImageSize свойство

Указывает размер сгенерированного TIFF-изображения. Значение по умолчанию равно 0x0, что означает, что размеры сгенерированных изображений будут рассчитываться на основе значения размера слайда презентации. Чтение/запись Size.

public Size ImageSize { get; set; }

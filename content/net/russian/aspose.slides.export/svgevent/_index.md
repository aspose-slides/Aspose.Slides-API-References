## Перечисление SvgEvent

Представляет параметры для SVG-формы.

```csharp
public enum SvgEvent
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| OnFocusIn | `0` | Происходит, когда элемент получает фокус, например, когда текст становится выделенным. |
| OnFocusOut | `1` | Происходит, когда элемент теряет фокус, например, когда текст становится невыделенным. |
| OnActivate | `2` | Происходит, когда элемент активируется, например, при нажатии кнопки мыши или клавиши. |
| OnClick | `3` | Происходит, когда кнопка указательного устройства нажата над элементом. |
| OnMouseDown | `4` | Происходит, когда кнопка указательного устройства нажата над элементом. |
| OnMouseUp | `5` | Происходит, когда кнопка указательного устройства отпущена над элементом. |
| OnMouseOver | `6` | Происходит, когда указательное устройство перемещается на элемент. |
| OnMouseMove | `7` | Происходит, когда указательное устройство перемещается, находясь над элементом. |
| OnMouseOut | `8` | Происходит, когда указательное устройство перемещается от элемента. |
| OnLoad | `9` | Происходит, когда пользовательский агент полностью разобрал элемент и его потомков и все ссылочные ресурсы, необходимые для его отображения. |
| OnUnload | `10` | Происходит, когда реализация DOM удаляет документ из окна или фрейма. Применимо только к самым внешним svg-элементам. |
| OnAbort | `11` | Происходит, когда загрузка страницы останавливается до того, как элемент был загружен полностью. |
| OnError | `12` | Происходит, когда элемент не загружается должным образом или когда происходит ошибка во время выполнения скрипта. |
| OnResize | `13` | Происходит, когда представление документа изменяется. Применимо только к самым внешним svg-элементам. |
| OnScroll | `14` | Происходит, когда представление документа сдвигается вдоль оси X или Y, или обеих осей. Применимо только к самым внешним svg-элементам. |
| OnZoom | `15` | Происходит, когда уровень масштабирования представления документа изменяется. Применимо только к самым внешним svg-элементам. |
| OnBegin | `16` | Происходит, когда начинается элемент анимации. |
| OnEnd | `17` | Происходит, когда заканчивается элемент анимации. |
| OnRepeat | `18` | Происходит, когда элемент анимации повторяется. |

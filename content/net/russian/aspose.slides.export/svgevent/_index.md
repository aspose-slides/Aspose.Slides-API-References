Представляет варианты для SVG-формы.

public enum SvgEvent

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| OnFocusIn | `0` | Происходит, когда элемент получает фокус, например, когда текст выделяется. |
| OnFocusOut | `1` | Происходит, когда элемент теряет фокус, например, когда текст перестаёт быть выделенным. |
| OnActivate | `2` | Происходит, когда элемент активируется, например, нажатием мыши или клавиши. |
| OnClick | `3` | Происходит, когда кнопка указательного устройства нажата над элементом. |
| OnMouseDown | `4` | Происходит, когда кнопка указательного устройства нажата над элементом. |
| OnMouseUp | `5` | Происходит, когда кнопка указательного устройства отпущена над элементом. |
| OnMouseOver | `6` | Происходит, когда указательное устройство перемещается на элемент. |
| OnMouseMove | `7` | Происходит, когда указательное устройство перемещается, находясь над элементом. |
| OnMouseOut | `8` | Происходит, когда указательное устройство уводится от элемента. |
| OnLoad | `9` | Происходит, когда пользовательский агент полностью разобрал элемент, его потомков и все связанные ресурсы, необходимые для его рендеринга. |
| OnUnload | `10` | Происходит, когда реализация DOM удаляет документ из окна или фрейма. Применяется только к самым внешним элементам SVG. |
| OnAbort | `11` | Происходит, когда загрузка страницы останавливается до того, как элемент полностью загрузился. |
| OnError | `12` | Происходит, когда элемент загружается неправильно или при возникновении ошибки во время выполнения скрипта. |
| OnResize | `13` | Происходит, когда представление документа изменяется по размеру. Применяется только к самым внешним элементам SVG. |
| OnScroll | `14` | Происходит, когда представление документа перемещается вдоль оси X, Y или обеих осей. Применяется только к самым внешним элементам SVG. |
| OnZoom | `15` | Происходит, когда уровень масштабирования представления документа изменяется. Применяется только к самым внешним элементам SVG. |
| OnBegin | `16` | Происходит, когда элемент анимации начинается. |
| OnEnd | `17` | Происходит, когда элемент анимации заканчивается. |
| OnRepeat | `18` | Происходит, когда элемент анимации повторяется. |

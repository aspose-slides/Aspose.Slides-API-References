## HtmlOptions.JpegQuality свойство

Возвращает или устанавливает значение, определяющее качество изображений JPEG внутри PDF-документа. Чтение/запись Byte.

```csharp
public byte JpegQuality { get; set; }
```

### Замечания

Эффективно только тогда, когда документ содержит изображения JPEG.

Используйте это свойство, чтобы получить или установить качество изображений внутри документа при сохранении в формате PDF. Значение может варьироваться от 0 до 100, где 0 означает худшее качество, но максимальную компрессию, а 100 означает лучшее качество, но минимальную компрессию.

Значение по умолчанию **95**.

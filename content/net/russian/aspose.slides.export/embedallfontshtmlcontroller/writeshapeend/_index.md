## EmbedAllFontsHtmlController.WriteShapeEnd метод

Вызывается перед рендерингом фигуры. Вызывается один раз для каждой фигуры. Если эта функция что-то записывает в генератор, текущее поколение изображения слайда будет завершено, добавленный фрагмент html вставлен, и новое изображение будет начинаться поверх предыдущего.

```csharp
public virtual void WriteShapeEnd(IHtmlGenerator generator, IShape shape)
```

| --- | --- | --- |
| generator | IHtmlGenerator | Объект вывода. |
| shape | IShape | Фигура, которая рендерится последней. |

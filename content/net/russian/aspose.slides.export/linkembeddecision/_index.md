## Перечисление LinkEmbedDecision

Определяет, как объект будет обработан при сохранении.

```csharp
public enum LinkEmbedDecision
```

### Значения

| Название | Значение | Описание |
| --- | --- | --- |
| Link | `0` | Объект будет храниться вне файла и ссылаться на него по URL |
| Embed | `1` | Объект должен быть встроен в создаваемый файл, если это возможно. Если встраивание невозможно, будет вызван GetUrl, и в зависимости от результата объект будет ссылаться на URL или будет проигнорирован. |
| Ignore | `2` | Объект будет проигнорирован. |

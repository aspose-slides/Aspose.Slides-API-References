## BlackWhiteConversionMode перечисление

Предоставляет параметры, которые контролируют, как изображения слайдов будут преобразованы в битональные изображения.

```csharp
public enum BlackWhiteConversionMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Default | `0` | Указывает отсутствие алгоритма преобразования. Будет использован алгоритм, реализованный в кодеке TIFF. (Default) |
| Dithering | `1` | Указывает алгоритм дизеринга (Floyd-Steinberg). |
| DitheringFloydSteinberg | `2` | Указывает алгоритм дизеринга Floyd-Steinberg. |
| Auto | `3` | Указывает автоматически вычисляемый пороговый алгоритм (Otsu). |
| AutoOtsu | `4` | Указывает автоматически вычисляемый алгоритм порога Otsu. |
| Threshold25 | `5` | Указывает статический пороговый алгоритм (25%). |
| Threshold50 | `6` | Указывает статический пороговый алгоритм (50%). |
| Threshold75 | `7` | Указывает статический пороговый алгоритм (75%). |

## Перечисление BlackWhiteConversionMode

Предоставляет параметры, которые контролируют, как изображения слайдов будут преобразованы в битональные изображения.

```csharp
public enum BlackWhiteConversionMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Default | `0` | Не указывает алгоритм преобразования. Будет использован алгоритм, реализованный в кодеке TIFF. (По умолчанию) |
| Dithering | `1` | Указывает алгоритм дотирования (Флойд-Стейнберг). |
| DitheringFloydSteinberg | `2` | Указывает алгоритм дотирования Флойда-Стейнберга. |
| Auto | `3` | Указывает алгоритм автоматического расчета порога (Оцу). |
| AutoOtsu | `4` | Указывает алгоритм порога Оцу с автоматическим расчетом. |
| Threshold25 | `5` | Указывает статический пороговый алгоритм (25%). |
| Threshold50 | `6` | Указывает статический пороговый алгоритм (50%). |
| Threshold75 | `7` | Указывает статический пороговый алгоритм (75%). |

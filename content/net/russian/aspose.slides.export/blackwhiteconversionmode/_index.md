## Перечисление BlackWhiteConversionMode

Предоставляет параметры, которые контролируют, как изображения слайдов будут конвертированы в битоные изображения.

```csharp
public enum BlackWhiteConversionMode
```

### Значения

| Название | Значение | Описание |
| --- | --- | --- |
| Default | `0` | Указывает на отсутствие алгоритма конверсии. Будет использован алгоритм, реализованный в кодеке TIFF. (По умолчанию) |
| Dithering | `1` | Указывает алгоритм дотсирования (Floyd-Steinberg). |
| DitheringFloydSteinberg | `2` | Указывает алгоритм дотсирования Флойда-Стейнберга. |
| Auto | `3` | Указывает алгоритм автоматически рассчитанного порога (Otsu). |
| AutoOtsu | `4` | Указывает алгоритм порога Otsu с автоматическим расчетом. |
| Threshold25 | `5` | Указывает статический алгоритм порога (25%). |
| Threshold50 | `6` | Указывает статический алгоритм порога (50%). |
| Threshold75 | `7` | Указывает статический алгоритм порога (75%). |

## Перечисление CompressionLevel

Указывает уровни сжатия ZIP для файла OpenXML. Более высокие уровни обеспечивают лучшее сжатие за счёт более медленной обработки.

```csharp
public enum CompressionLevel
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | `0` | Сжатие не применяется. Файлы сохраняются без изменений. |
| Level1 | `1` | Самое быстрое сжатие с самым низким коэффициентом сжатия. |
| Level2 | `2` | Более быстрое сжатие с немного лучшим коэффициентом сжатия, чем Level1. |
| Level3 | `3` | Обеспечивает лучшее сжатие, чем Level2, с умеренным влиянием на производительность. |
| Level4 | `4` | Обеспечивает лучшее сжатие, чем Level3. |
| Level5 | `5` | Обеспечивает улучшенное сжатие по сравнению с Level4 с дополнительным временем обработки. |
| Level6 | `6` | Стандартное сжатие, предлагающее хороший баланс между скоростью сжатия и размером файла. Уровень сжатия по умолчанию. |
| Level7 | `7` | Обеспечивает более высокое сжатие, чем Level6, при более медленной обработке. |
| Level8 | `8` | Обеспечивает более высокое сжатие, чем Level7. |
| Level9 | `9` | Максимальное сжатие. Создаёт наименьший размер файла при самой медленной скорости обработки. |

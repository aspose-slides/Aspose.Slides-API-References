## ISVGOptions.DisableLineEndCropping свойство

SVG 1.1 не поддерживает определение отступов для маркеров. В движке записи SVG Aspose.Slides есть обходное решение этой проблемы: он обрезает конец линии с стрелкой, чтобы линия не перекрывала маркеры. Эта опция отключает такое поведение. Чтение/запись Boolean.

```csharp
public bool DisableLineEndCropping { get; set; }
```

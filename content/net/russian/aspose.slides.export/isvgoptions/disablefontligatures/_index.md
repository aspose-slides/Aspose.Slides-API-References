## ISVGOptions.DisableFontLigatures свойство

Получает или устанавливает значение, указывающее, выводится ли текст без использования лигатур. При установке значения `true` лигатуры будут отключены в выводе. По умолчанию это свойство установлено в `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    SVGOptions options = new SVFOptions
    {
        DisableFontLigatures = true // Отключить лигатуры при отрисовке текста
    };
    
    using (FileStream fileStream = new FileStream("slide-0.svg", FileMode.Create, FileAccess.Write))
    {
        pres.Slides[0].WriteAsSvg(fileStream);   
    }
}
```

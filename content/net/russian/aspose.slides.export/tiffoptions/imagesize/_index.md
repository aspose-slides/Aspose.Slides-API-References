## Свойство TiffOptions.ImageSize

Указывает размер создаваемого TIFF-изображения. Значение по умолчанию - 0x0, что означает, что размеры создаваемых изображений будут рассчитываться на основе значения размера слайда презентации. Чтение/запись Size.

public Size ImageSize { get; set; }

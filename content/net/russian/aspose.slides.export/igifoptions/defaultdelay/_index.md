## IGifOptions.DefaultDelay свойство  

Получает или задает время задержки по умолчанию [мс]. Это значение будет использовано, если [`AdvanceAfterTime`](../../../aspose.slides/islideshowtransition/advanceaftertime) не задано. Значение по умолчанию — 1000.  

```csharp  
public int DefaultDelay { get; set; }  
```  

### Примеры  

```csharp  
[C#]  
using (Presentation pres = new Presentation("pres.pptx"))  
{  
	pres.Save("pres.gif", SaveFormat.Gif, new GifOptions { DefaultDelay = 2000 });  
}  
```

## PdfAccessPermissions перечисление  

Содержит набор флагов, указывающих, какие разрешения доступа должны быть предоставлены при открытии документа с пользовательским доступом.  

```csharp  
[Flags]  
public enum PdfAccessPermissions  
```  

### Значения  

| Имя | Значение | Описание |  
| --- | --- | --- |  
| None | `0` | Указывает, что у пользователя нет разрешений на доступ. |  
| PrintDocument | `4` | Указывает, может ли пользователь распечатать документ (возможно, не на самом высоком уровне качества, в зависимости от того, установлен ли также флаг HighQualityPrint). |  
| ModifyContent | `8` | Указывает, может ли пользователь изменять содержимое документа с помощью операций, помимо тех, которые контролируются флагами AddOrModifyFields, FillExistingFields, AssembleDocument. |  
| CopyTextAndGraphics | `10` | Указывает, может ли пользователь копировать или иначе извлекать текст и графику из документа с помощью операций, отличных от тех, которые контролируются флагом ExtractTextAndGraphics. |  
| AddOrModifyFields | `20` | Указывает, может ли пользователь добавлять или изменять текстовые аннотации, заполнять интерактивные поля форм и, если флаг ModifyContent также установлен, создавать или изменять интерактивные поля форм (включая подписи). |  
| FillExistingFields | `100` | Указывает, может ли пользователь заполнять существующие интерактивные поля форм (включая поля для подписей), даже если флаг AddOrModifyFields не установлен. |  
| ExtractTextAndGraphics | `200` | Указывает, может ли пользователь извлекать текст и графику в поддержку доступности для пользователей с ограниченными возможностями или для других целей. |  
| AssembleDocument | `400` | Указывает, может ли пользователь собирать документ (вставлять, поворачивать или удалять страницы и создавать закладки или эскизы), даже если флаг ModifyContent не установлен. |  
| HighQualityPrint | `800` | Указывает, может ли пользователь распечатать документ в представление, из которого может быть сгенерирована точная цифровая копия содержимого PDF. Когда этот флаг не установлен (и флаг PrintDocument установлен), печать ограничена низким уровнем представления, возможно, ухудшенного качества. |

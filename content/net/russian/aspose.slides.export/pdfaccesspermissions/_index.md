## PdfAccessPermissions перечисление

Содержит набор флагов, указывающих, какие разрешения доступа должны быть предоставлены при открытии документа пользователем.

```csharp
[Flags]
public enum PdfAccessPermissions
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | `0` | Указывает, что пользователь не имеет разрешений на доступ. |
| PrintDocument | `4` | Указывает, может ли пользователь печатать документ (возможно, не в самом высоком качестве, в зависимости от того, установлен ли бит HighQualityPrint). |
| ModifyContent | `8` | Указывает, может ли пользователь изменять содержимое документа операциями, отличными от управляемых битами AddOrModifyFields, FillExistingFields, AssembleDocument. |
| CopyTextAndGraphics | `10` | Указывает, может ли пользователь копировать или иным образом извлекать текст и графику из документа операциями, отличными от управляемых битом ExtractTextAndGraphics. |
| AddOrModifyFields | `20` | Указывает, может ли пользователь добавлять или изменять текстовые аннотации, заполнять интерактивные поля формы, а если установлен бит ModifyContent, создавать или изменять интерактивные поля формы (включая поля подписи). |
| FillExistingFields | `100` | Указывает, может ли пользователь заполнять существующие интерактивные поля формы (включая поля подписи), даже если бит AddOrModifyFields не установлен. |
| ExtractTextAndGraphics | `200` | Указывает, может ли пользователь извлекать текст и графику в целях обеспечения доступности для пользователей с ограниченными возможностями или для других целей. |
| AssembleDocument | `400` | Указывает, может ли пользователь собирать документ (вставлять, вращать или удалять страницы и создавать закладки или миниатюры), даже если бит ModifyContent не установлен. |
| HighQualityPrint | `800` | Указывает, может ли пользователь печатать документ в представление, из которого можно создать точную цифровую копию содержимого PDF. Когда этот бит не установлен (а бит PrintDocument установлен), печать ограничивается низкоуровневым представлением внешнего вида, возможно, с ухудшенным качеством. |

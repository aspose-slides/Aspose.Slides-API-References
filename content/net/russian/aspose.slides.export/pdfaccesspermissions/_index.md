## Перечисление PdfAccessPermissions

Содержит набор флагов, указывающих, какие права доступа должны быть предоставлены при открытии документа с пользовательским доступом.

```csharp
[Flags]
public enum PdfAccessPermissions
```

### Значения

| Название | Значение | Описание |
| --- | --- | --- |
| None | `0` | Указывает, что у пользователя нет прав доступа. |
| PrintDocument | `4` | Указывает, может ли пользователь распечатать документ (возможно, не на самом высоком уровне качества, в зависимости от того, установлен ли также бит HighQualityPrint). |
| ModifyContent | `8` | Указывает, может ли пользователь изменять содержимое документа другими действиями, кроме тех, которые контролируются битами AddOrModifyFields, FillExistingFields, AssembleDocument. |
| CopyTextAndGraphics | `10` | Указывает, может ли пользователь копировать или иным образом извлекать текст и графику из документа другими действиями, кроме как контролируемыми битом ExtractTextAndGraphics. |
| AddOrModifyFields | `20` | Указывает, может ли пользователь добавлять или изменять текстовые аннотации, заполнять интерактивные поля форм и, если бит ModifyContent также установлен, создавать или изменять интерактивные поля форм (включая поля для подписи). |
| FillExistingFields | `100` | Указывает, может ли пользователь заполнять существующие интерактивные поля форм (включая поля для подписи), даже если бит AddOrModifyFields не установлен. |
| ExtractTextAndGraphics | `200` | Указывает, может ли пользователь извлекать текст и графику в поддержку доступности для пользователей с ограниченными возможностями или для других целей. |
| AssembleDocument | `400` | Указывает, может ли пользователь собирать документ (вставлять, вращать или удалять страницы и создавать закладки или эскизы), даже если бит ModifyContent не установлен. |
| HighQualityPrint | `800` | Указывает, может ли пользователь распечатать документ на представление, из которого может быть сгенерирована точная цифровая копия содержимого PDF. Когда этот бит не установлен (и бит PrintDocument установлен), печать ограничена низким уровнем представления внешнего вида, возможно, ухудшенного качества. |

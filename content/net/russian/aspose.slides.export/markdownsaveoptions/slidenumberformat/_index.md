## MarkdownSaveOptions.SlideNumberFormat свойство

Получает или задает строку формата, используемую для заголовков номеров слайдов в Markdown-выводе. Формат должен включать "{0}" как заполнители, который будет заменен индексом слайда во время экспорта. Пример: "# Слайд {0}" приведет к "# Слайд 1", "# Слайд 2" и т. д.

```csharp
public string SlideNumberFormat { get; set; }
```

### Исключения

| исключение | условие |
| --- | --- |
| ArgumentNullException | Генерируется, если предоставленное значение равно null или пусто. |
| ArgumentException | Генерируется, если строка формата не содержит "{0}" как заполнителя. |

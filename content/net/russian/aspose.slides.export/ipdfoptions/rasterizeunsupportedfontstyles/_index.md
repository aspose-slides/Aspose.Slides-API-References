## IPdfOptions.RasterizeUnsupportedFontStyles свойство

Указывает, следует ли растеризовать текст в виде битмапа и сохранить в PDF, когда шрифт не поддерживает жирное начертание. Этот подход может улучшить качество текста в получаемом PDF для определённых шрифтов. Чтение/запись Boolean.

```csharp
public bool RasterizeUnsupportedFontStyles { get; set; }
```

### Примечания

По умолчанию **false**.

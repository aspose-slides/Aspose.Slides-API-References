## IPdfOptions.RasterizeUnsupportedFontStyles свойство

Указывает, следует ли растрировать текст в виде битмапа и сохранять в PDF, когда шрифт не поддерживает жирное начертание. Этот подход может улучшить качество текста в результирующем PDF для определённых шрифтов. Свойство для чтения/записи Boolean.

```csharp
public bool RasterizeUnsupportedFontStyles { get; set; }
```

### Заметки

По умолчанию **false**.

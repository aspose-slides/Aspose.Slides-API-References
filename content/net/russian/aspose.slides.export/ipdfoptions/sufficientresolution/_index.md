## Свойство IPdfOptions.SufficientResolution

Возвращает или устанавливает значение, определяющее разрешение изображений внутри PDF документа.

Свойство влияет на размер файла, время экспорта и качество изображения.

Значение по умолчанию — **96**.

Эффект этого параметра зависит от нескольких факторов. Алгоритм пытается получить лучший размер выходного изображения в соответствии с значением свойства, размером исходного изображения и размером кадра изображения. Использование аналогичных значений свойств может дать одинаковый результат. Рекомендуется использовать шаг 16 или 32 для получения заметного эффекта. Чтение/запись Single.

```csharp
public float SufficientResolution { get; set; }
```

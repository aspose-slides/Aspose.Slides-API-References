## IPdfOptions.BestImagesCompressionRatio свойство

Указывает, должна ли автоматически выбираться наиболее эффективная компрессия (вместо стандартной) для каждого изображения. Если установлено значение Boolean.true, для каждого изображения в презентации будет выбран наиболее подходящий алгоритм сжатия, что приведет к уменьшению размера результирующего PDF-документа. Выбор лучшего коэффициента сжатия изображений требует значительных вычислительных ресурсов и занимает дополнительное количество оперативной памяти, и эта опция по умолчанию является Boolean.false.

```csharp
public bool BestImagesCompressionRatio { get; set; }
```

### Примечания

По умолчанию является Boolean.false.

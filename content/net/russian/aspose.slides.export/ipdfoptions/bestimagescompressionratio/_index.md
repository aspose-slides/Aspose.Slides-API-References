## Свойство IPdfOptions.BestImagesCompressionRatio

Указывает, должно ли быть автоматически выбрано наиболее эффективное сжатие (вместо значения по умолчанию) для каждого изображения. Если установлено значение Boolean.true, для каждого изображения в презентации будет выбран наиболее подходящий алгоритм сжатия, что приведет к уменьшению размера полученного PDF-документа. Выбор наилучшего коэффициента сжатия изображения является вычислительно затратным и требует дополнительного объема оперативной памяти, и эта опция по умолчанию равна Boolean.false.

```csharp
public bool BestImagesCompressionRatio { get; set; }
```

### Примечания

По умолчанию - Boolean.false.

## IPdfOptions.SaveMetafilesAsPng свойство

true для преобразования всех метафайлов, используемых в презентации, в изображения PNG. Читаемое/записываемое логическое значение.

```csharp
public bool SaveMetafilesAsPng { get; set; }
```

### Примечания

По умолчанию **true**. PDF-документ может содержать векторную графику и растровые изображения. Если SaveMetafilesAsPng установлено в true, то исходное изображение метафайла преобразуется в формат Png и сохраняется в PDF как растровое изображение. Если SaveMetafilesAsPng установлено в false, то исходный метафайл преобразуется в векторную графику PDF. У каждого подхода есть свои плюсы и минусы. Например, если метафайл преобразуется в PNG, то возможно некоторое снижение качества при масштабировании полученного документа. Если метафайл преобразуется в векторную графику PDF, то могут возникнуть проблемы с производительностью в инструменте просмотра PDF.

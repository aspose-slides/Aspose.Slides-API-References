## Свойство IPdfOptions.SaveMetafilesAsPng

true для преобразования всех метафайлов, используемых в презентации, в изображения PNG. Чтение/запись Boolean.

```csharp
public bool SaveMetafilesAsPng { get; set; }
```

### Замечания

По умолчанию **true**. Документ Pdf может содержать векторную графику и растровые изображения. Если SaveMetafilesAsPng установлено в true, то исходное изображение метафайла преобразуется в формат Png и сохраняется в Pdf как растровое изображение. Если SaveMetafilesAsPng установлено в false, то исходный метафайл преобразуется в векторную графику Pdf. Каждый подход имеет свои преимущества и недостатки. Например, если метафайл преобразуется в PNG, то возможно некоторое снижение качества при масштабировании результирующего документа. Если метафайл преобразуется в векторную графику Pdf, то возможны проблемы производительности в инструменте просмотра Pdf.

## Перечисление SvgExternalFontsHandling

Представляет способ обработки внешних шрифтов, используемых при отрисовке текста.

```csharp
public enum SvgExternalFontsHandling
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| AddLinksToFontFiles | `0` | Добавить ссылки на отдельные файлы шрифтов в секцию style SVG-файла. |
| Embed | `1` | Сохранить данные шрифтов непосредственно в SVG-файле. Перед использованием этой опции проверьте лицензионные соглашения всех внешних шрифтов. |
| Vectorize | `2` | Сохранить весь текст с внешними шрифтами в виде графики. |

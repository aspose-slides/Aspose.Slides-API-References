## SlideShowTransition.SoundIsBuiltIn свойство

Указывает, является ли этот звук встроенным. Если этот атрибут установлен в true, приложение-генератор будет уведомлено о том, чтобы проверить атрибут имени, указанный для этого звука в его списке встроенных звуков, и при необходимости может отобразить пользовательское имя или интерфейс. Тип данных - булевый (читаемый и записываемый).

```csharp
public bool SoundIsBuiltIn { get; set; }
```

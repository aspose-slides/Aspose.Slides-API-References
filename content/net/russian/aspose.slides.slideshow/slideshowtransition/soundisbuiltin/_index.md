## SlideShowTransition.SoundIsBuiltIn property

Указывает, является ли этот звук встроенным звуком. Если для этого атрибута установлено значение true, то генерирующее приложение получает предупреждение о необходимости проверить атрибут имени, указанный для этого звука в списке встроенных звуков, и затем может отображать пользовательский имя или пользовательский интерфейс по мере необходимости. Чтение-записьBoolean.

public bool SoundIsBuiltIn { get; set; }

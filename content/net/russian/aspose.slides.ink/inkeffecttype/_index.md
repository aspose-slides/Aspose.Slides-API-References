## InkEffectType перечисление

Определяет набор предопределённых визуальных эффектов для отображения чернил. Каждый эффект соответствует текстуре или изображению, имитирующему стилизованный вид чернил. Эти значения могут использоваться для настройки визуального стиля цифровых штрихов чернил при рендеринге.

```csharp
public enum InkEffectType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| NotDefined | `-1` | Эффект не определён или неизвестен. Это значение можно использовать как значение по умолчанию или резервное. |
| Bronze | `0` | Тёплая, коричневатая металлическая текстура, напоминающая бронзовые чернила. |
| Galaxy | `1` | Яркая, мерцающая текстура, напоминающая галактику, с космическими оттенками. |
| Gold | `2` | Яркая, металлическая золотая текстура, придающая штрихам чернил роскошный вид. |
| Lava | `3` | Огонёчная текстура, напоминающая расплавленную лаву, с красными и оранжевыми тонами. |
| Ocean | `4` | Глубокая синяя, жидкоподобная текстура, имитирующая океанские волны или водные чернила. |
| RainbowGlitter | `5` | Яркий, искристый радужный глиттер, используемый для праздничных или живых штрихов чернил. |
| RoseGold | `6` | Мягкое сочетание розового и золотого, похожее на розовое золото, для элегантных штрихов чернил. |
| Silver | `7` | Холодная металлическая серебряная текстура, симулирующая классические серебряные чернила. |

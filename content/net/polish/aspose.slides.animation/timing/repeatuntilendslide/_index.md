## Timing.RepeatUntilEndSlide właściwość

Ten atrybut określa, czy efekt będzie powtarzany do końca slajdu. Odczyt/zapis Boolean.

```csharp
public bool RepeatUntilEndSlide { get; set; }
```

### Przykłady

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Pobierz sekwencję efektów dla pierwszego slajdu
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Pobierz pierwszy efekt głównej sekcji.
    IEffect effect = effectsSequence[0];

    // Zmień Timing/Repeat efektu na "Do końca slajdu"
    effect.Timing.RepeatUntilEndSlide = true;
}
```

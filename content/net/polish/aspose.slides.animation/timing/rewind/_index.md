## Timing.Rewind właściwość

Ten atrybut określa, czy efekt będzie przewijany wstecz po zakończeniu odtwarzania. Odczyt/zapis Boolean.

```csharp
public bool Rewind { get; set; }
```

### Przykłady

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Pobierz sekwencję efektów dla pierwszego slajdu
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Pobierz pierwszy efekt głównej sekwencji.
    IEffect effect = effectsSequence[0];

    // Włącz Timing/Rewind efektu.
    effect.Timing.Rewind = true;
}
```

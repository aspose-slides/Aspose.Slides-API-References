## Effect.AnimateTextType właściwość

Definiuje typ animowanego tekstu dla efektu. Tekst kształtu może być animowany literą, wyrazem lub jednocześnie całość. Odczyt/zapis `AnimateTextType`.

```csharp
public AnimateTextType AnimateTextType { get; set; }
```

### Przykłady

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Pobierz pierwszy efekt pierwszego slajdu.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Zmień typ animowanego tekstu efektu na "By letter"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByLetter;
}
```

## Effect.DelayBetweenTextParts własność

Definiuje opóźnienie między animowanymi częściami tekstu (słowami lub literami). Dodatnia wartość określa procent czasu trwania efektu. Ujemna wartość określa opóźnienie w sekundach. Odczyt/zapis Single.

```csharp
public float DelayBetweenTextParts { get; set; }
```

### Przykłady

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Pobierz pierwszy efekt pierwszego slajdu.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Zmień typ animacji tekstu efektu na "By word"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByWord;
    
    // Ustaw opóźnienie między animowanymi częściami tekstu na 20% czasu trwania efektu.
    firstSlideEffect.DelayBetweenTextParts = 20f;
}
```

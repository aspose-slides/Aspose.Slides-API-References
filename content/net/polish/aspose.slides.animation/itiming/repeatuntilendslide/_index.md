## ITiming.RepeatUntilEndSlide właściwość

Ten atrybut określa, czy efekt będzie powtarzany aż do końca slajdu. Odczyt/zapis Boolean.

```csharp
public bool RepeatUntilEndSlide { get; set; }
```

### Przykłady

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Pobierz sekwencję efektów dla pierwszego slajdu
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Pobierz pierwszy efekt głównej sekwencji.
    IEffect effect = effectsSequence[0];

    // Zmień Timing/Repeat efektu na "Until End of Slide"
    effect.Timing.RepeatUntilEndSlide = true;
}
```

## ITiming.RepeatUntilNextClick właściwość

Ten atrybut określa, czy efekt będzie powtarzany do następnego kliknięcia. Odczyt/zapis Boolean.

```csharp
public bool RepeatUntilNextClick { get; set; }
```

### Przykłady

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Pobierz sekwencję efektów dla pierwszego slajdu
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Pobierz pierwszy efekt głównej sekwencji.
    IEffect effect = effectsSequence[0];

    // Zmień Timing/Repeat efektu na "Until Next Click"
    effect.Timing.RepeatUntilNextClick = true;
}
```

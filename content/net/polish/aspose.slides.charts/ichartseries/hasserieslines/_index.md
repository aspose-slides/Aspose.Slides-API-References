## IChartSeries.HasSeriesLines właściwość

Określa, czy istnieją linie serii dla tej serii i powiązanych serii. To jest właściwość nie tylko tej serii, ale wszystkich serii grupy serii nadrzędnej – jest to projekcja odpowiedniej właściwości grupy. W związku z tym ta właściwość jest tylko do odczytu. Użyj właściwości ParentSeriesGroup, aby uzyskać dostęp do grupy serii nadrzędnej. Użyj właściwości read/write ParentSeriesGroup.HasSeriesLines, aby zmienić wartość. Użyj właściwości ParentSeriesGroup.SeriesLinesFormat do formatowania linii serii. Tylko do odczytu Boolean.

```csharp
public bool HasSeriesLines { get; }
```

### Uwagi

Jest to projekcja właściwości ParentSeriesGroup.HasSeriesLines.

## IChartSeries.GapWidth właściwość

Określa odstęp między grupami słupków lub kolumn, wyrażony jako procent szerokości słupka lub kolumny. Jest to właściwość nie tylko tego szeregu, ale wszystkich szeregów w grupie nadrzędnej – jest to projekcja odpowiedniej właściwości grupy. W związku z tym ta właściwość jest tylko do odczytu. Użyj właściwości ParentSeriesGroup, aby uzyskać dostęp do grupy szeregów nadrzędnych. Użyj właściwości ParentSeriesGroup.GapWidth (odczyt/zapis), aby zmienić wartość. Tylko do odczytu Int32.

```csharp
public int GapWidth { get; }
```

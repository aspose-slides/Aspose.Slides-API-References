## IChartSeries.SecondPieSize właściwość

Określa rozmiar drugiego wycinka lub słupka wykresu typu pie-of-pie lub bar-of-pie, wyrażony w procentach rozmiaru pierwszego wycinka (może wynosić od 5 do 200 procent). Jest to właściwość nie tylko tej serii, ale wszystkich serii grupy serii nadrzędnej – jest to projekcja odpowiedniej właściwości grupy. Dlatego ta właściwość jest tylko do odczytu. Użyj właściwości ParentSeriesGroup, aby uzyskać dostęp do grupy serii nadrzędnej. Użyj właściwości ParentSeriesGroup.SecondPieSize odczyt/zapis, aby zmienić wartość. tylko do odczytu UInt16.

```csharp
public ushort SecondPieSize { get; }
```

### Uwagi

Jest to projekcja właściwości ParentSeriesGroup.SecondPieSize.

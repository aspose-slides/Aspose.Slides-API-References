## IChartSeries.PieSplitBy właściwość

Określa, w jaki sposób określić, które punkty danych znajdują się w drugim torcie lub słupku wykresu typu torczo-torcowy lub słupek-torcowy. Jest to właściwość nie tylko tej serii, ale wszystkich serii grupy serii nadrzędnej – jest to projekcja odpowiedniej właściwości grupy. Dlatego właściwość ta jest tylko do odczytu. Użyj właściwości ParentSeriesGroup, aby uzyskać dostęp do grupy serii nadrzędnej. Użyj właściwości ParentSeriesGroup.PieSplitBy odczyt/zapis, aby zmienić wartość. Tylko do odczytu [`PieSplitType`](../../piesplittype).

```csharp
public PieSplitType PieSplitBy { get; }
```

### Uwagi

1) Jest to projekcja właściwości ParentSeriesGroup.PieSplitBy.  
2) Jeśli wartość właściwości jest PieSplitType.Custom, możesz określić niestandardowe informacje podziału za pomocą właściwości ParentSeriesGroup.PieSplitCustomPoints.

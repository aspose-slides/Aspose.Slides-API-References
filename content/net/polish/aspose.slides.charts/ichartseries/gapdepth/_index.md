## IChartSeries.GapDepth właściwość

Zwraca lub ustawia odległość, wyrażoną w procentach szerokości znacznika, pomiędzy seriami danych na wykresie 3D. Jest to właściwość nie tylko tej serii, ale wszystkich serii w grupie serii nadrzędnej – jest to projekcja odpowiedniej właściwości grupy. Dlatego ta właściwość jest tylko do odczytu. Użyj właściwości ParentSeriesGroup, aby uzyskać dostęp do grupy serii nadrzędnej. Użyj właściwości ParentSeriesGroup.GapDepth (odczyt/zapis), aby zmienić wartość. tylko do odczytu Int32.

```csharp
public int GapDepth { get; }
```

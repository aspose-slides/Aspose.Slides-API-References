## IChartSeries.BubbleSizeRepresentation właściwość

Określa, w jaki sposób wartości rozmiaru bąbelka są przedstawiane na wykresie bąbelkowym. Jest to właściwość nie tylko tej serii, ale wszystkich serii grupy serii nadrzędnej – jest to projekcja odpowiedniej właściwości grupy. Dlatego ta właściwość jest tylko do odczytu. Użyj właściwości ParentSeriesGroup, aby uzyskać dostęp do grupy serii nadrzędnej. Użyj właściwości ParentSeriesGroup.BubbleSizeRepresentation do odczytu/zapisu, aby zmienić wartość.

```csharp
public BubbleSizeRepresentationType BubbleSizeRepresentation { get; }
```

### Uwagi

Jest to projekcja właściwości ParentSeriesGroup.BubbleSizeRepresentation.

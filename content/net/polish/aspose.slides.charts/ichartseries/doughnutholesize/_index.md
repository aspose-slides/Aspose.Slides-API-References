## IChartSeries.DoughnutHoleSize właściwość

Określa rozmiar otworu w wykresie pierścieniowym (może wynosić od 10 do 90 procent rozmiaru obszaru wykresu). Jest to właściwość nie tylko tej serii, ale wszystkich serii grupy serii nadrzędnej – jest to projekcja odpowiedniej właściwości grupy. Dlatego ta właściwość jest tylko do odczytu. Użyj właściwości ParentSeriesGroup, aby uzyskać dostęp do grupy serii nadrzędnej. Użyj właściwości ParentSeriesGroup.DoughnutHoleSize z dostępem odczyt/zapis, aby zmienić wartość. Tylko do odczytu Byte.

```csharp
public byte DoughnutHoleSize { get; }
```

### Uwagi

To jest projekcja właściwości ParentSeriesGroup.DoughnutHoleSize.

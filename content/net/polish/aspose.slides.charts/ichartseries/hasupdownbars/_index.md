## IChartSeries.HasUpDownBars właściwość

Określa, czy wykres liniowy lub giełdowy ma paski góra/dół. Jest to właściwość nie tylko tej serii, ale wszystkich serii grupy serii nadrzędnej – jest to projekcja odpowiedniej właściwości grupy. Dlatego ta właściwość jest tylko do odczytu. Użyj właściwości ParentSeriesGroup, aby uzyskać dostęp do grupy serii nadrzędnej. Użyj właściwości odczyt/zapis ParentSeriesGroup.UpDownBars.HasUpDownBars, aby zmienić wartość. Użyj właściwości ParentSeriesGroup.UpDownBars, aby sformatować paski góra/dół. Tylko do odczytu Boolean.

```csharp
public bool HasUpDownBars { get; }
```

### Uwagi

Jest to projekcja właściwości ParentSeriesGroup.UpDownBars.HasUpDownBars.

## IChartSeries.BubbleSizeScale właściwość

Określa współczynnik skali wykresu bąbelkowego (może być w przedziale od 0 do 300 procent domyślnego rozmiaru). Jest to właściwość nie tylko tej serii, ale wszystkich serii grupy serii nadrzędnej – jest to projekcja odpowiedniej właściwości grupy. Dlatego ta właściwość jest tylko do odczytu. Użyj właściwości ParentSeriesGroup, aby uzyskać dostęp do grupy serii nadrzędnej. Użyj właściwości odczyt/zapis ParentSeriesGroup.BubbleSizeScale, aby zmienić wartość.

```csharp
public int BubbleSizeScale { get; }
```

### Uwagi

Jest to projekcja właściwości ParentSeriesGroup.BubbleSizeScale.

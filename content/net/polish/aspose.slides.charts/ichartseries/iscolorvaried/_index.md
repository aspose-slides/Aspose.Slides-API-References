## IChartSeries.IsColorVaried właściwość

Określa, że każdy znacznik danych w serii ma inny kolor. Jest to właściwość nie tylko tej serii, ale wszystkich serii grupy serii nadrzędnej – jest to projekcja odpowiedniej właściwości grupy. Dlatego ta właściwość jest tylko do odczytu. Użyj właściwości ParentSeriesGroup, aby uzyskać dostęp do grupy serii nadrzędnej. Użyj właściwości ParentSeriesGroup.IsColorVaried do odczytu i zapisu, aby zmienić wartość. Tylko do odczytu Boolean.

```csharp
public bool IsColorVaried { get; }
```

### Uwagi

Jest to projekcja właściwości ParentSeriesGroup.IsColorVaried.

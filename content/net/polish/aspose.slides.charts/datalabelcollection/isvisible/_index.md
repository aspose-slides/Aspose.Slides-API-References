## DataLabelCollection.IsVisible właściwość

False oznacza, że etykieta danych nie jest domyślnie widoczna (a więc wszystkie flagi Show*- (ShowValue, ...) DefaultDataLabelFormat właściwość mają wartość false). Tylko do odczytu Boolean.

```csharp
public bool IsVisible { get; }
```

### Uwagi

Jeśli etykieta danych jest domyślnie widoczna, możesz ukryć ją domyślnie za pomocą metody Hide(). Jednakże jeśli etykieta danych nie jest domyślnie widoczna (IsVisible jest false), możesz ustawić etykietę danych jako „widoczną domyślnie” ustawiając flagi Show*- (ShowValue, ...) DefaultDataLabelFormat właściwość na stan true.

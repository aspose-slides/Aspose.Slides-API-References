## IDataLabelFormat.ShowSeriesName właściwość

Zwraca lub ustawia wartość Boolean określającą zachowanie wyświetlania nazwy serii dla etykiet danych na wykresie. True, aby wyświetlić nazwę serii. False, aby ukryć. Odczyt/zapis Boolean.

```csharp
public bool ShowSeriesName { get; set; }
```

### Uwagi

Jeśli rodzicem tego obiektu DataLabelFormat jest kolekcja DataLabelCollection etykiet danych, wówczas ta właściwość pobiera lub ustawia domyślną wartość właściwości ShowSeriesName dla nowych etykiet danych w kolekcji DataLabelCollection. Ustawienie tej właściwości powoduje także ustawienie tej samej wartości w właściwości ShowSeriesName dla wszystkich etykiet danych w kolekcji DataLabelCollection (np. "DataLabels.DefaultDataLabelFormat.ShowSeriesName = val;" powoduje, że wszystkie DataLabels[i].ShowSeriesName są równe val).

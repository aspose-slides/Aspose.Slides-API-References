## ChartData.UseSecondaryCategories właściwość

Jeśli wartość jest fałsz, to [`SecondaryCategories`](../secondarycategories) właściwość zwraca null, a dane w [`Categories`](../categories) właściwość są używane zarówno dla serii podstawowej, jak i pomocniczej. Jeśli wartość jest prawda, to dane w [`SecondaryCategories`](../secondarycategories) właściwość są używane dla serii pomocniczej, a dane w [`Categories`](../categories) właściwość są używane dla serii podstawowej. Odczyt/zapis Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

### Przykłady

Przykład. Które kategorie są powiązane z serią — ChartData.Categories lub ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // powiązane kategorie to series.Chart.ChartData.SecondaryCategories
}
else
{
    // powiązane kategorie to series.Chart.ChartData.Categories
}
```

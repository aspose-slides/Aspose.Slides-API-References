## ChartData.Categories właściwość

Zwraca podstawowe kategorie (lub zarówno podstawowe, jak i pomocnicze kategorie, jeśli [`UseSecondaryCategories`](../usesecondarycategories) właściwość ma wartość false). Tylko do odczytu [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection Categories { get; }
```

### Uwagi

Jeśli [`UseSecondaryCategories`](../usesecondarycategories) właściwość ma wartość false, to [`SecondaryCategories`](../secondarycategories) właściwość zwraca null i dane w tej właściwości `Categories` są używane zarówno dla serii podstawowych, jak i pomocniczych. Jeśli [`UseSecondaryCategories`](../usesecondarycategories) właściwość ma wartość true, to dane w [`SecondaryCategories`](../secondarycategories) właściwość są używane dla serii pomocniczych, a dane w tej właściwości `Categories` są używane dla serii podstawowych.

### Przykłady

Przykład. Które kategorie są związane z seriami – ChartData.Categories lub ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // powiązane kategorie to series.Chart.ChartData.SecondaryCategories
}
else
{
    // powiązane kategorie to series.Chart.ChartData.Categories
}
```

## ChartData.SecondaryCategories właściwość

Zwraca drugie kategorie, jeśli właściwość [`UseSecondaryCategories`](../usesecondarycategories) ma wartość true. Tylko do odczytu [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### Uwagi

Jeśli właściwość [`UseSecondaryCategories`](../usesecondarycategories) ma wartość false, to właściwość `SecondaryCategories` zwraca null i dane w właściwości [`Categories`](../categories) są używane zarówno dla serii podstawowej, jak i pomocniczej. Jeśli właściwość [`UseSecondaryCategories`](../usesecondarycategories) ma wartość true, to dane w tej właściwości `SecondaryCategories` są używane dla serii pomocniczej, a dane w właściwości [`Categories`](../categories) są używane dla serii podstawowej.

### Przykłady

Przykład. Jakie kategorie są powiązane z serią - ChartData.Categories czy ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // powiązane kategorie to series.Chart.ChartData.SecondaryCategories
}
else
{
    // powiązane kategorie to series.Chart.ChartData.Categories
}
```

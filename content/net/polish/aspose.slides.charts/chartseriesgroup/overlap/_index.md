## ChartSeriesGroup.Overlap właściwość

Określa, jak bardzo słupki i kolumny mają nachodzić na siebie w wykresach 2-D, wyrażone jako procent (od -100 % do 100 %). - -100 %: maksymalny odstęp (słupki są całkowicie oddzielone). - 0 %: słupki są umieszczone obok siebie bez nachodzenia ani odstępu. - 100 %: maksymalne nachodzenie (słupki całkowicie na siebie nachodzą). Ta właściwość jest odczytywana/zapisywana SByte.

```csharp
public sbyte Overlap { get; set; }
```

### Wyjątki

| wyjątek | warunek |
| --- | --- |
| ArgumentOutOfRangeException | Rzucany, gdy wartość jest ustawiona poza dopuszczalnym zakresem od -100 do 100. |

### Przykłady

The following example demonstrates how to set the overlap for a chart series group and render the resulting chart on a form:

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // Ustaw nachodzenie na 55%

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

## ChartCategory.UseCell właściwość

Jeśli wartość jest prawda, to właściwość AsCell jest aktualna. Innymi słowy, arkusz jest używany do przechowywania kategorii (ten przypadek obsługuje kategorię wielopoziomową). Jeśli wartość jest fałsz, to właściwość AsLiteral jest aktualna. Innymi słowy, arkusz NIE jest używany do przechowywania kategorii (i ten przypadek nie obsługuje kategorii wielopoziomowych). Tylko do odczytu Boolean.

```csharp
public bool UseCell { get; }
```

### Uwagi

Aby zmienić wartość tej właściwości (dla wszystkich kategorii w kolekcji) ustaw nową wartość w właściwości ChartCategoryCollection.UseCells.

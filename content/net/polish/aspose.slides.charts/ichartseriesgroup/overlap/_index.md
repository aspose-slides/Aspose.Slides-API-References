## IChartSeriesGroup.Overlap właściwość

Określa, o ile słupki i kolumny mają zachodzić na siebie na wykresach 2-D, wyrażone jako procent (od -100% do 100%). - -100%: maksymalny odstęp (słupki są całkowicie oddzielone). - 0%: słupki są ustawione obok siebie bez zachodzenia ani odstępu. - 100%: maksymalne zachodzenie (słupki całkowicie zachodzą na siebie). Ta właściwość jest odczytywalna/zapisywalna SByte.

```csharp
public sbyte Overlap { get; set; }
```

### Wyjątki

| wyjątek | warunek |
| --- | --- |
| ArgumentOutOfRangeException | Rzucany, gdy wartość jest ustawiona poza dopuszczalnym zakresem od -100 do 100. |

### Przykłady

Poniższy przykład pokazuje, jak ustawić zachodzenie dla grupy serii wykresu i wyrenderować powstały wykres na formularzu:

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // Ustaw zachodzenie na 55%

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

## IDataLabelCollection.IsVisible właściwość

False oznacza, że etykieta danych nie jest widoczna domyślnie (i dlatego wszystkie flagi Show* (ShowValue, ...) właściwości DefaultDataLabelFormat mają wartość false). tylko do odczytu Boolean.

```csharp
public bool IsVisible { get; }
```

### Uwagi

Jeśli etykieta danych jest widoczna domyślnie, możesz ją ukryć domyślnie metodą Hide(). Natomiast jeśli etykieta danych nie jest widoczna domyślnie (IsVisible jest false), możesz ustawić etykietę danych jako „widoczna domyślnie” poprzez ustawienie flag Show* (ShowValue, ...) właściwości DefaultDataLabelFormat na wartość true.

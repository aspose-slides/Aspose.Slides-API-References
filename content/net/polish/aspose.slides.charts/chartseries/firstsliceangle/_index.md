## ChartSeries.FirstSliceAngle właściwość

Określa kąt pierwszego wycinka wykresu kołowego lub pierścieniowego, w stopniach (zgodnie z ruchem wskazówek zegara od góry, od 0 do 360 stopni). Jest to właściwość nie tylko tego serii, ale wszystkich serii grupy serii nadrzędnej – jest to projekcja odpowiedniej właściwości grupy. Dlatego ta właściwość jest tylko do odczytu. Użyj właściwości ParentSeriesGroup, aby uzyskać dostęp do grupy serii nadrzędnej. Użyj właściwości ParentSeriesGroup.FirstSliceAngle do odczytu i zapisu, aby zmienić wartość. UInt16 tylko do odczytu.

```csharp
public ushort FirstSliceAngle { get; }
```

### Uwagi

Jest to projekcja właściwości ParentSeriesGroup.FirstSliceAngle.

## ChartSeries.BubbleSizeScale właściwość

Określa współczynnik skali wykresu bąbelkowego (może wynosić od 0 do 300 procent domyślnego rozmiaru). Jest to właściwość nie tylko tej serii, ale wszystkich serii grupy serii nadrzędnej – jest to projekcja odpowiedniej właściwości grupy. W związku z tym ta właściwość jest tylko do odczytu. Użyj właściwości ParentSeriesGroup, aby uzyskać dostęp do grupy serii nadrzędnej. Użyj właściwości ParentSeriesGroup.BubbleSizeScale właściwość odczyt/zapis, aby zmienić wartość.

```csharp
public int BubbleSizeScale { get; }
```

### Uwagi

Jest to projekcja właściwości ParentSeriesGroup.BubbleSizeScale.

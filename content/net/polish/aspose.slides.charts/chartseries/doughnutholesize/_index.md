## ChartSeries.DoughnutHoleSize właściwość

Określa rozmiar otworu w wykresie pierścieniowym (może wynosić od 10 do 90 procent rozmiaru obszaru wykresu). Jest to właściwość nie tylko tej serii, ale wszystkich serii grupy serii nadrzędnej – jest to projekcja odpowiedniej właściwości grupy. W związku z tym właściwość jest tylko do odczytu. Użyj właściwości ParentSeriesGroup, aby uzyskać dostęp do grupy serii nadrzędnej. Użyj właściwości ParentSeriesGroup.DoughnutHoleSize do odczytu/zapisu, aby zmienić wartość. Tylko do odczytu Byte.

```csharp
public byte DoughnutHoleSize { get; }
```

### Uwagi

Jest to projekcja właściwości ParentSeriesGroup.DoughnutHoleSize.

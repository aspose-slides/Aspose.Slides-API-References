## ChartSeries.GapDepth property

Zwraca lub ustawia odległość, wyrażoną w procentach szerokości znacznika, pomiędzy seriami danych w wykresie 3D. To właściwość nie tylko tej serii, ale wszystkich serii grupy serii nadrzędnej – jest to projekcja odpowiedniej właściwości grupy. Dlatego ta właściwość jest tylko do odczytu. Użyj właściwości ParentSeriesGroup, aby uzyskać dostęp do grupy serii nadrzędnej. Użyj właściwości ParentSeriesGroup.GapDepth do odczytu/zapisu, aby zmienić wartość. Tylko do odczytu Int32.

```csharp
public int GapDepth { get; }
```

### Uwagi

To jest projekcja właściwości ParentSeriesGroup.GapDepth.

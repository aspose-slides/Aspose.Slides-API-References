## ChartSeries.HasSeriesLines właściwość

Określa, czy istnieją linie serii dla tej serii i powiązanych serii. Jest to właściwość nie tylko tej serii, ale wszystkich serii grupy serii nadrzędnej – jest to rzutowanie odpowiedniej właściwości grupy. Dlatego właściwość jest tylko do odczytu. Użyj właściwości ParentSeriesGroup, aby uzyskać dostęp do grupy serii nadrzędnej. Użyj ParentSeriesGroup.HasSeriesLines właściwości odczyt/zapis, aby zmienić wartość. Użyj właściwości ParentSeriesGroup.SeriesLinesFormat, aby sformatować linie serii. Tylko do odczytu Boolean.

```csharp
public bool HasSeriesLines { get; }
```

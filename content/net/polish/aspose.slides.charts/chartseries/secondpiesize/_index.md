## ChartSeries.SecondPieSize właściwość

Określa rozmiar drugiego wycinka koła lub słupka wykresu koło-w-kółku lub słupek-w-kółku, jako procent rozmiaru pierwszego wycinka (może wynosić od 5 do 200 procent). Jest to właściwość nie tylko tej serii, ale wszystkich serii w grupie serii nadrzędnej – jest to projekcja odpowiedniej właściwości grupy. Dlatego ta właściwość jest tylko do odczytu. Użyj właściwości ParentSeriesGroup, aby uzyskać dostęp do grupy serii nadrzędnej. Użyj właściwości odczyt/zapis ParentSeriesGroup.SecondPieSize, aby zmienić wartość. Tylko do odczytu UInt16.

```csharp
public ushort SecondPieSize { get; }
```

### Uwagi

Jest to projekcja właściwości ParentSeriesGroup.SecondPieSize.

## ChartSeries.TrendLines właściwość

Kolekcja linii trendu serii. Tylko do odczytu [`ITrendlineCollection`](../../itrendlinecollection).

```csharp
public ITrendlineCollection TrendLines { get; }
```

### Uwagi

TrendLines są dostępne (nie null) dla serii danych w nieukładanych wykresach 2-D area, bar, column, line, stock, xy (scatter) oraz bubble. Linia trendu nie jest dostępna dla serii danych w żadnym typie wykresu, który jest stacked lub 3-D. TrendLines nie są również dostępne dla wykresów radar, pie, surface lub doughnut.

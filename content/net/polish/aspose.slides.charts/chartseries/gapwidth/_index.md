## ChartSeries.GapWidth właściwość

Określa odstęp pomiędzy grupami słupków lub kolumn, wyrażony jako procent szerokości słupka lub kolumny. To jest właściwość nie tylko tego szeregu, ale wszystkich szeregów grupy serii nadrzędnej – jest to projekcja odpowiedniej właściwości grupy. W związku z tym ta właściwość jest tylko do odczytu. Użyj właściwości ParentSeriesGroup, aby uzyskać dostęp do grupy serii nadrzędnej. Użyj właściwości ParentSeriesGroup.GapWidth do odczytu/zapisu, aby zmienić wartość. Tylko do odczytu Int32.

```csharp
public int GapWidth { get; }
```

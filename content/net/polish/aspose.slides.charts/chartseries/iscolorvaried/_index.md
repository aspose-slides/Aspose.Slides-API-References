## Właściwość ChartSeries.IsColorVaried

Określa, że każdy znacznik danych w serii ma inny kolor. Jest to własność nie tylko tej serii, ale wszystkich serii grupy serii nadrzędnej – jest to projekcja odpowiedniej własności grupy. W związku z tym własność jest tylko do odczytu. Użyj własności ParentSeriesGroup, aby uzyskać dostęp do grupy serii nadrzędnej. Użyj własności ParentSeriesGroup.IsColorVaried (odczyt/zapis), aby zmienić wartość. Boolean tylko do odczytu.

```csharp
public bool IsColorVaried { get; }
```

### Uwagi

Jest to projekcja własności ParentSeriesGroup.IsColorVaried.

## DataLabel.IsVisible właściwość

False oznacza, że etykieta danych nie jest widoczna (a więc wszystkie flagi Show*-flags (ShowValue, ...) są false). tylko do odczytu Boolean.

```csharp
public bool IsVisible { get; }
```

### Uwagi

Jeśli etykieta danych jest widoczna, możesz ją ukryć za pomocą metody Hide(). Jednak jeśli etykieta danych nie jest widoczna (IsVisible jest false), możesz ją uczynić widoczną, ustawiając flagi Show*-flags (ShowValue, ...) na stan true.

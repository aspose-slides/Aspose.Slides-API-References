## ErrorBarValueType wyliczenie

Reprezentuje typ wartości słupka błędu

```csharp
public enum ErrorBarValueType
```

### Wartości

| Nazwa | Wartość | Opis |
| --- | --- | --- |
| Custom | `0` | Określa, że długość słupków błędu ma być określona przez elementy Plus i Minus. |
| Fixed | `1` | Określa, że długość słupków błędu ma być stałą wartością określoną przez Error Bar Value. |
| Percentage | `2` | Określa, że długość słupków błędu ma być procentem Error Bar Value danych. |
| StandardDeviation | `3` | Określa, że długość słupków błędu ma być standardowymi odchyleniami Error Bar Value danych. |
| StandardError | `4` | Określa, że długość słupków błędu ma być standardowymi błędami Error Bar Value danych. |

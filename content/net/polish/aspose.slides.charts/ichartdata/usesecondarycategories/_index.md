## IChartData.UseSecondaryCategories właściwość

Jeśli false, to [`SecondaryCategories`](../secondarycategories) właściwość zwraca null i dane w [`Categories`](../categories) właściwości są używane zarówno dla serii podstawowych, jak i pomocniczych. Jeśli true, to dane w [`SecondaryCategories`](../secondarycategories) właściwości są używane dla serii pomocniczych, a dane w [`Categories`](../categories) właściwości są używane dla serii podstawowych. Odczyt/zapis Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

### Przykłady

Przykład. Jakie kategorie są powiązane z serią – ChartData.Categories lub ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // powiązane kategorie to series.Chart.ChartData.SecondaryCategories
}
else
{
    // powiązane kategorie to series.Chart.ChartData.Categories
}
```

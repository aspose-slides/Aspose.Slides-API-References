## IChartData.SecondaryCategories property

Zwraca drugorzędne kategorie, jeśli właściwość [`UseSecondaryCategories`](../usesecondarycategories) ma wartość true. Tylko do odczytu [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### Remarks

Jeśli właściwość [`UseSecondaryCategories`](../usesecondarycategories) ma wartość false, to właściwość `SecondaryCategories` zwraca null i dane w właściwości [`Categories`](../categories) są używane zarówno dla serii głównej, jak i drugorzędnej. Jeśli właściwość [`UseSecondaryCategories`](../usesecondarycategories) ma wartość true, to dane w tej właściwości `SecondaryCategories` są używane dla serii drugorzędnej, a dane w właściwości [`Categories`](../categories) są używane dla serii głównej.

### Examples

Przykład. Które kategorie są powiązane z serią — ChartData.Categories czy ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // powiązane kategorie to series.Chart.ChartData.SecondaryCategories
}
else
{
    // powiązane kategorie to series.Chart.ChartData.Categories
}
```

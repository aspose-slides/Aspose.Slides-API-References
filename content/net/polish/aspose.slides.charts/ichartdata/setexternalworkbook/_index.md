## SetExternalWorkbook(string) {#setexternalworkbook}

Ustawia zewnętrzny skoroszyt jako źródło danych wykresu. Dane wykresu będą aktualizowane z docelowego skoroszytu.

```csharp
public void SetExternalWorkbook(string workbookPath)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| workbookPath | String | Ścieżka do docelowego skoroszytu |

### Wyjątki

| wyjątek | warunek |
| --- | --- |
| InvalidOperationException | Zewnętrzny skoroszyt nie jest dostępny lub nie może zostać załadowany. |

### Przykłady

```csharp
[C#]
using (Presentation pres = new Presentation())

   IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Pie, 50, 50, 400, 600, true);
   IChartData chartData = chart.ChartData;
   (chartData as ChartData).SetExternalWorkbook("../../workbook.xlsx");

```

## SetExternalWorkbook(string, bool) {#setexternalworkbook_1}

Ustawia zewnętrzny skoroszyt jako źródło danych wykresu.

```csharp
public void SetExternalWorkbook(string workbookPath, bool updateChartData)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| workbookPath | String | Ścieżka do docelowego skoroszytu |
| updateChartData | Boolean | Jeśli wartość jest fałsz, tylko ścieżka do skoroszytu zostanie zaktualizowana. Dane wykresu nie zostaną załadowane ani zaktualizowane z docelowego skoroszytu. Może być użyte, gdy docelowy skoroszyt nie istnieje lub nie jest dostępny. Jeśli wartość jest prawda, dane wykresu zostaną zaktualizowane z docelowego skoroszytu. |

### Wyjątki

| wyjątek | warunek |
| --- | --- |
| InvalidOperationException | Zewnętrzny skoroszyt nie jest dostępny lub nie może zostać załadowany. |

### Przykłady

```csharp
using (Presentation pres = new Presentation())

   IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Pie, 50, 50, 400, 600, true);
   IChartData chartData = chart.ChartData;

   (chartData as ChartData).SetExternalWorkbook("http://path/doesnt/exists", false);


```

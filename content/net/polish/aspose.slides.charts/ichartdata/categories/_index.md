## IChartData.Categories właściwość

Pobiera podstawowe kategorie (lub zarówno podstawowe, jak i drugorzędne kategorie, jeśli właściwość [`UseSecondaryCategories`](../usesecondarycategories) ma wartość false). Tylko do odczytu [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection Categories { get; }
```

### Uwagi

Jeśli właściwość [`UseSecondaryCategories`](../usesecondarycategories) ma wartość false, to właściwość [`SecondaryCategories`](../secondarycategories) zwraca null i dane w tej właściwości `Categories` są używane zarówno dla podstawowych, jak i drugorzędnych serii. Jeśli właściwość [`UseSecondaryCategories`](../usesecondarycategories) ma wartość true, to dane w właściwości [`SecondaryCategories`](../secondarycategories) są używane dla drugorzędnych serii, a dane w tej właściwości `Categories` są używane dla podstawowych serii.

### Przykłady

Przykład. Jakie kategorie są powiązane z seriami – ChartData.Categories lub ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // powiązane kategorie to series.Chart.ChartData.SecondaryCategories
}
else
{
    // powiązane kategorie to series.Chart.ChartData.Categories
}
```

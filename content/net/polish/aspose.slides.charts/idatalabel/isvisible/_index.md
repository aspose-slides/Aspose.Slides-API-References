## IDataLabel.IsVisible właściwość

False oznacza, że etykieta danych nie jest widoczna (a więc wszystkie flagi Show*-flags (ShowValue, ...) są false). Tylko do odczytu Boolean.

```csharp
public bool IsVisible { get; }
```

### Uwagi

Jeśli etykieta danych jest widoczna, możesz ją ukryć metodą Hide(). Ale jeśli etykieta danych nie jest widoczna (IsVisible jest false), możesz uczynić etykietę danych widoczną, ustawiając flagi Show*-flags (ShowValue, ...) na stan true.

## IDataLabel.Hide metoda

Ukryj etykietę danych, ustawiając wszystkie flagi Show*- (ShowValue, ...) na stan false. IsVisible będzie false po tej operacji.

```csharp
public void Hide()
```

### Uwagi

Jeśli etykieta danych nie jest widoczna (IsVisible jest false), możesz ją uczynić widoczną, ustawiając flagi Show*- (ShowValue, ...) na stan true.

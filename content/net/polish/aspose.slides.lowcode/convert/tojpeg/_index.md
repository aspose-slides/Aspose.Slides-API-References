## ToJpeg(Presentation, string) {#tojpeg}

Konwertuje podaną prezentację na zestaw obrazów w formacie JPEG. Jeśli nazwa pliku wyjściowego zostanie podana jako "myPath/myFilename.jpeg", wynik zostanie zapisany jako zestaw plików "myPath/myFilename_N.jpeg", gdzie N oznacza numer slajdu.

```csharp
public static void ToJpeg(Presentation pres, string outputFileName)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| pres | Presentation | Wejściowa prezentacja. |
| outputFileName | String | Nazwa pliku wyjściowego. |

### Wyjątki

| wyjątek | warunek |
| --- | --- |
| ArgumentException |  |

### Przykłady

```csharp
using (var pres = new Presentation("pres.pptx"))
    Convert.ToJpeg(pres, "presImage.jpeg");
```

```csharp

    IRenderingOptions options)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| pres | Presentation | Wejściowa prezentacja. |
| outputFileName | String | Nazwa pliku wyjściowego. |
| scale | Single | Współczynnik skalowania stosowany do obrazów wyjściowych w stosunku do oryginalnego rozmiaru slajdu. |

| wyjątek | warunek |
| --- | --- |
| ArgumentException |  |

### Przykłady

```csharp

{
    SlidesLayoutOptions = new NotesCommentsLayoutingOptions()
    {
        NotesPosition = NotesPositions.BottomTruncated

    Convert.ToJpeg(pres, "presImage.jpeg", 2f, options);
```

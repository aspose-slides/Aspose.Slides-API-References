## ToPng(Presentation, string) {#topng}

Konwertuje wejściową prezentację na zestaw obrazów w formacie PNG. Jeśli nazwa pliku wyjściowego zostanie podana jako "myPath/myFilename.png", wynik zostanie zapisany jako zestaw plików "myPath/myFilename_N.png", gdzie N jest numerem slajdu.

```csharp
public static void ToPng(Presentation pres, string outputFileName)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| pres | Presentation | Wejściowa prezentacja. |
| outputFileName | String | Nazwa pliku wyjściowego. |

| --- | --- |
| ArgumentException |  |

### Przykłady

```csharp

   Convert.ToPng(pres, "presImage.png");
```

## ToPng(Presentation, string, float, IRenderingOptions) {#topng_1}

Konwertuje wejściową prezentację na zestaw obrazów w formacie PNG. Jeśli nazwa pliku wyjściowego zostanie podana jako "myPath/myFilename.png", wynik zostanie zapisany jako zestaw plików "myPath/myFilename_N.png", gdzie N jest numerem slajdu.

```csharp
public static void ToPng(Presentation pres, string outputFileName, float scale, 
    IRenderingOptions options)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| pres | Presentation | Wejściowa prezentacja. |
| outputFileName | String | Nazwa pliku wyjściowego. |
| scale | Single | Współczynnik skalowania stosowany do obrazów wyjściowych w odniesieniu do oryginalnego rozmiaru slajdu. |
| options | IRenderingOptions | Opcje renderowania. |

### Wyjątki

| Wyjątek | warunek |
| --- | --- |
| ArgumentException |  |

### Przykłady

```csharp
IRenderingOptions options = new RenderingOptions()
{
    SlidesLayoutOptions = new NotesCommentsLayoutingOptions()
    {
        NotesPosition = NotesPositions.BottomTruncated
    }
};

using (var pres = new Presentation("pres.pptx"))
   Convert.ToPng(pres, "presImage.png", 2f, options);
```

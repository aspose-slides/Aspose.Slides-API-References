## IPresentationAnimationPlayer interfejs

Reprezentuje odtwarzacz animacji. Animacje generowane przez [`PresentationAnimationsGenerator`](../presentationanimationsgenerator) za pośrednictwem jego [`NewAnimation`](../presentationanimationsgenerator/newanimation) zdarzenia.

```csharp
public interface IPresentationAnimationPlayer
```

| --- | --- |
| [Duration](../../aspose.slides.export/ipresentationanimationplayer/duration) { get; } | Pobierz czas trwania animacji [ms] |

## Metody

| --- | --- |
| [GetFrame](../../aspose.slides.export/ipresentationanimationplayer/getframe)() | Pobierz klatkę dla bieżącej pozycji czasu, wcześniej ustawionej metodą [`SetTimePosition`](./settimeposition). |
| [SetTimePosition](../../aspose.slides.export/ipresentationanimationplayer/settimeposition)(double) | Ustaw pozycję czasu animacji w obrębie [`Duration`](./duration). |

### Przykłady

```csharp
[C#]
using (Presentation presentation = new Presentation("animated.pptx"))
{
    using (var animationsGenerator = new PresentationAnimationsGenerator(presentation.SlideSize.Size.ToSize()))
    {
        animationsGenerator.NewAnimation += animationPlayer =>
        {
            Console.WriteLine($"Animation total duration: {animationPlayer.Duration}");
            
            animationPlayer.SetTimePosition(0);
            animationPlayer.GetFrame().Save("firstFrame.png");
            
            animationPlayer.SetTimePosition(animationPlayer.Duration);
            animationPlayer.GetFrame().Save("lastFrame.png");
      };
        
        animationsGenerator.Run(presentation.Slides);
    }
}
```

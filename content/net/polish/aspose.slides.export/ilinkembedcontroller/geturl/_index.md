## ILinkEmbedController.GetUrl metoda

Zwraca adres URL do zewnętrznego obiektu. Ta metoda jest zawsze wywoływana, jeśli [`GetObjectStoringLocation`](../getobjectstoringlocation) zwróciło Link i może być wywołana, jeśli [`GetObjectStoringLocation`](../getobjectstoringlocation) zwróciło Embed, ale osadzanie jest niemożliwe. Może być wywołana wiele razy dla tego samego identyfikatora obiektu.

```csharp
public string GetUrl(int id, int referrer)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| id | Int32 | Identyfikator obiektu. Ten identyfikator jest unikalny w całej operacji. |
| referrer | Int32 | Id obiektu odwołującego lub 0, jeśli obiekt jest odwoływany przez dokument główny. Może być użyte do wygenerowania względnego linku. |

### Wartość zwracana

Adres URL obiektu zewnętrznego lub null, jeśli ten obiekt powinien być zignorowany.

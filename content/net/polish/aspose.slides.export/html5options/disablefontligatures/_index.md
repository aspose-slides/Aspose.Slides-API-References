## Html5Options.DisableFontLigatures właściwość

Uzyskuje lub ustawia wartość określającą, czy tekst jest renderowany bez użycia ligatur. Gdy ustawiona jest na `true`, ligatury zostaną wyłączone w renderowanym wyjściu. Domyślnie ta właściwość jest ustawiona na `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Przykłady

Przykład:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    Html5Options options = new Html5Options
    {
        DisableFontLigatures = true // Wyłącz ligatury w renderowaniu tekstu
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html5, options);
}
```

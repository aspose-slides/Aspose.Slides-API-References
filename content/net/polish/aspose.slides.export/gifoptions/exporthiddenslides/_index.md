## GifOptions.ExportHiddenSlides właściwość

Określa, czy ukryte slajdy będą eksportowane. Domyślna wartość to false.

```csharp
public bool ExportHiddenSlides { get; set; }
```

### Przykłady

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
	pres.Save("pres.gif", SaveFormat.Gif, new GifOptions { ExportHiddenSlides = false });
}
```

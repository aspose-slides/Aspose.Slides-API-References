## GifOptions.DefaultDelay właściwość

Pobiera lub ustawia domyślny czas opóźnienia [ms]. Ta wartość zostanie użyta, jeśli [`AdvanceAfterTime`](../../../aspose.slides/islideshowtransition/advanceaftertime) nie zostanie ustawiona. Domyślna wartość wynosi 1000.

```csharp
public int DefaultDelay { get; set; }
```

### Przykłady

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
	pres.Save("pres.gif", SaveFormat.Gif, new GifOptions { DefaultDelay = 2000 });
}
```

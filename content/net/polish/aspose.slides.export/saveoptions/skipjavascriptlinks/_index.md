## Właściwość SaveOptions.SkipJavaScriptLinks

Określa, czy pomijać hiperłącza z wywołaniami JavaScript podczas zapisywania prezentacji. Odczyt/zapis Boolean. Domyślna wartość to **false**.

```csharp
public bool SkipJavaScriptLinks { get; set; }
```

### Uwagi

Gdy ta właściwość ma wartość **true**, hiperłącza z wywołaniami JavaScript będą pomijane podczas zapisywania.

Gdy ta właściwość ma wartość **false**, wszystkie hiperłącza zostaną zapisane.

### Przykłady

Przykład:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("result_without_JavaScript_links.html", SaveFormat.Html, new HtmlOptions()
    {
        SkipJavaScriptLinks = true
    });
}
```

## IHtmlOptions.DisableFontLigatures właściwość

Pobiera lub ustawia wartość określającą, czy tekst jest renderowany bez użycia ligatur. Gdy ustawione na `true`, ligatury zostaną wyłączone w renderowanym wyniku. Domyślnie ta właściwość jest ustawiona na `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Przykłady

Przykład:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    HtmlOptions options = new HtmlOptions
    {
        DisableFontLigatures = true // Wyłącz ligatury w renderowaniu tekstu
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html, options);
}
```

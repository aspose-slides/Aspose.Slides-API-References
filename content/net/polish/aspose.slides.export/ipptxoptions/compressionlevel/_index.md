## IPptxOptions.CompressionLevel właściwość

Określa poziom kompresji używany podczas zapisywania dokumentu prezentacji. Domyślna wartość to Level6.

```csharp
public CompressionLevel CompressionLevel { get; set; }
```

### Uwagi

Wyższe poziomy kompresji generują mniejsze pliki, ale wymagają więcej czasu przetwarzania. Rzeczywisty współczynnik kompresji zależy od zawartości prezentacji.

### Przykłady

Przykład:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("demo-level8.pptx", SaveFormat.Pptx, new PptxOptions()
    {
        CompressionLevel = CompressionLevel.Level8
    });
}
```

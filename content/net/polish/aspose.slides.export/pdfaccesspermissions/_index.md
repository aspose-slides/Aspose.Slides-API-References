## PdfAccessPermissions enumeracja

Zawiera zestaw flag określających, które uprawnienia dostępu powinny być przyznane, gdy dokument zostanie otwarty przez użytkownika.

```csharp
[Flags]
public enum PdfAccessPermissions
```

### Wartości

| Nazwa | Wartość | Opis |
| --- | --- | --- |
| None | `0` | Określa, że użytkownik nie posiada uprawnień dostępu. |
| PrintDocument | `4` | Określa, czy użytkownik może drukować dokument (ewentualnie nie w najwyższej jakości, w zależności od tego, czy bit HighQualityPrint jest również ustawiony). |
| ModifyContent | `8` | Określa, czy użytkownik może modyfikować zawartość dokumentu przy użyciu operacji innych niż te kontrolowane przez bity AddOrModifyFields, FillExistingFields, AssembleDocument. |
| CopyTextAndGraphics | `10` | Określa, czy użytkownik może kopiować lub w inny sposób wyodrębniać tekst i grafikę z dokumentu przy użyciu operacji innych niż te kontrolowane przez bit ExtractTextAndGraphics. |
| AddOrModifyFields | `20` | Określa, czy użytkownik może dodawać lub modyfikować adnotacje tekstowe, wypełniać interaktywne pola formularza oraz, jeśli bit ModifyContent jest również ustawiony, tworzyć lub modyfikować interaktywne pola formularza (w tym pola podpisu). |
| FillExistingFields | `100` | Określa, czy użytkownik może wypełniać istniejące interaktywne pola formularza (w tym pola podpisu), nawet jeśli bit AddOrModifyFields jest wyczyszczony. |
| ExtractTextAndGraphics | `200` | Określa, czy użytkownik może wyodrębniać tekst i grafikę w celu zapewnienia dostępności dla użytkowników niepełnosprawnych lub w innych celach. |
| AssembleDocument | `400` | Określa, czy użytkownik może tworzyć dokument (wstawiać, obracać lub usuwać strony oraz tworzyć zakładki lub obrazy miniatur), nawet jeśli bit ModifyContent jest wyczyszczony. |
| HighQualityPrint | `800` | Określa, czy użytkownik może drukować dokument do reprezentacji, z której można wygenerować wierną cyfrową kopię zawartości PDF.

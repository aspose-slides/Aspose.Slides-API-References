## PdfAccessPermissions wyliczenie

Zawiera zestaw flag określających, które uprawnienia dostępu powinny być przyznane, gdy dokument zostanie otwarty z dostępem użytkownika.

```csharp
[Flags]
public enum PdfAccessPermissions
```

### Values

| Nazwa | Wartość | Opis |
| --- | --- | --- |
| None | `0` | Określa, że użytkownik nie posiada uprawnień dostępu. |
| PrintDocument | `4` | Określa, czy użytkownik może drukować dokument (być może nie w najwyższej jakości, w zależności od tego, czy ustawiony jest również bit HighQualityPrint). |
| ModifyContent | `8` | Określa, czy użytkownik może modyfikować zawartość dokumentu przy użyciu operacji innych niż te kontrolowane przez bity AddOrModifyFields, FillExistingFields, AssembleDocument. |
| CopyTextAndGraphics | `10` | Określa, czy użytkownik może kopiować lub w inny sposób wyodrębniać tekst i grafikę z dokumentu przy użyciu operacji innych niż te kontrolowane przez bit ExtractTextAndGraphics. |
| AddOrModifyFields | `20` | Określa, czy użytkownik może dodawać lub modyfikować adnotacje tekstowe, wypełniać interaktywne pola formularzy oraz, jeśli ustawiony jest również bit ModifyContent, tworzyć lub modyfikować interaktywne pola formularzy (w tym pola podpisu). |
| FillExistingFields | `100` | Określa, czy użytkownik może wypełniać istniejące interaktywne pola formularzy (w tym pola podpisu), nawet jeśli bit AddOrModifyFields jest wyłączony. |

| AssembleDocument | `400` | Określa, czy użytkownik może składać dokument (wstawiać, obracać lub usuwać strony oraz tworzyć zakładki lub miniatury), nawet jeśli bit ModifyContent jest wyłączony. |
| HighQualityPrint | `800` | Określa, czy użytkownik może drukować dokument w reprezentacji, z której można wygenerować wierną cyfrową kopię zawartości PDF.

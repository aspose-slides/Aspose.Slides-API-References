## ISwfOptions.SlidesLayoutOptions właściwość

Pobiera lub ustawia tryb, w którym slajdy są umieszczane na stronie podczas eksportu prezentacji [`ISlidesLayoutOptions`](../../islideslayoutoptions). Ta właściwość nie obsługuje przypisywania obiektów typu `Aspose.Slides.Export.HandoutLayoutingOptions`

```csharp
public ISlidesLayoutOptions SlidesLayoutOptions { get; set; }
```

### Przykłady

Przykład:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    SwfOptions options = new SwfOptions
    {
        SlidesLayoutOptions = new NotesCommentsLayoutingOptions
        {
            CommentsPosition = CommentsPositions.Right
        }
    };
    
    pres.Save("pres.swf", SaveFormat.Swf, options);
}
```

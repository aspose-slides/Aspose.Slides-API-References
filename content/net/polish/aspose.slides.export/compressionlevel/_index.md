## enumeracja CompressionLevel

Określa poziomy kompresji ZIP dla pliku OpenXML. Wyższe poziomy zapewniają lepszą kompresję kosztem wolniejszego przetwarzania.

```csharp
public enum CompressionLevel
```

### Wartości

| Nazwa | Wartość | Opis |
| --- | --- | --- |
| None | `0` | Nie stosuje się kompresji. Pliki są przechowywane w stanie niezmienionym. |
| Level1 | `1` | Najszybsza kompresja z najniższym współczynnikiem kompresji. |
| Level2 | `2` | Szybsza kompresja z nieco lepszym współczynnikiem kompresji niż Level1. |
| Level3 | `3` | Zapewnia lepszą kompresję niż Level2 przy umiarkowanym wpływie na wydajność. |
| Level4 | `4` | Zapewnia lepszą kompresję niż Level3. |
| Level5 | `5` | Zapewnia lepszą kompresję niż Level4 przy dodatkowym czasie przetwarzania. |
| Level6 | `6` | Standardowa kompresja, zapewniająca dobrą równowagę między szybkością kompresji a rozmiarem pliku. Domyślny poziom kompresji. |
| Level7 | `7` | Zapewnia wyższą kompresję niż Level6 przy wolniejszym przetwarzaniu. |
| Level8 | `8` | Zapewnia wyższą kompresję niż Level7. |
| Level9 | `9` | Maksymalna kompresja. Tworzy najmniejszy rozmiar pliku przy najwolniejszej prędkości przetwarzania. |

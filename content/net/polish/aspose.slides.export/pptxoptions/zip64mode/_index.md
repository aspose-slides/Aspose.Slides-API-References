## PptxOptions.Zip64Mode właściwość

Określa, czy format ZIP64 jest używany dla dokumentu prezentacji. Domyślna wartość to IfNecessary

```csharp
public Zip64Mode Zip64Mode { get; set; }
```

### Przykłady

Przykład:


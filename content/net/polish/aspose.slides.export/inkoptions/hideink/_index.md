## InkOptions.HideInk właściwość

Wyświetla lub ukrywa elementy Ink w wyeksportowanym dokumencie.

```csharp
public bool HideInk { get; set; }
```

### Uwagi

Domyślna wartość to false.

### Przykłady

Poniższy przykład pokazuje, jak ukryć elementy Ink w wyeksportowanym dokumencie PDF:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.HideInk = true;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

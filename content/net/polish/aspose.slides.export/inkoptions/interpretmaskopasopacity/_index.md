## InkOptions.InterpretMaskOpAsOpacity właściwość

Używa operacji ROP lub przezroczystości do renderowania pędzla.

```csharp
public bool InterpretMaskOpAsOpacity { get; set; }
```

### Uwagi

Domyślna wartość to true.

### Przykłady

Następny przykład pokazuje, jak ustawić przy użyciu ROP do eksportowania elementów Ink:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.InterpretMaskOpAsOpacity = false;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

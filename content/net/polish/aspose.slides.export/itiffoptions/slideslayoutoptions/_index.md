## ITiffOptions.SlidesLayoutOptions właściwość

Pobiera lub ustawia tryb, w którym slajdy są umieszczane na stronie podczas eksportowania prezentacji [`ISlidesLayoutOptions`](../../islideslayoutoptions).

```csharp

```

### Przykłady

Przykład:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    TiffOptions options = new TiffOptions
    {
        SlidesLayoutOptions = new HandoutLayoutingOptions
        {
            Handout = HandoutType.Handouts4Horizontal
        }
    };
    
    pres.Save("pres.tiff", SaveFormat.Tiff, options);
}
```

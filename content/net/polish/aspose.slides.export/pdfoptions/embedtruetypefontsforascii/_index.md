## PdfOptions.EmbedTrueTypeFontsForASCII właściwość

Określa, czy Aspose.Slides będzie osadzać standardowe czcionki dla tekstu ASCII (zakres kodów 33..127). Czcionki dla kodów znaków większych niż 127 są zawsze osadzane. Lista standardowych czcionek obejmuje podstawowe 14 czcionek PDF oraz dodatkowe czcionki określone przez użytkownika. Odczyt/zapis Boolean.

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### Uwagi

Domyślnie jest **true**.

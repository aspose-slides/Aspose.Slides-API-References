## PdfOptions.IncludeOleData właściwość

True to convert all OLE data from the presentation to embedded files in the resulting PDF. Odczyt/zapis Boolean.

```csharp
public bool IncludeOleData { get; set; }
```

### Uwagi

Domyślnie **false**.

### Przykłady

Przykład:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions options = new PdfOptions { IncludeOleData = true };
    pres.Save("pres.pdf", SaveFormat.Pdf, options);
}
```

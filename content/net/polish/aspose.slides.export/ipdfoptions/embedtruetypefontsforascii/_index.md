## IPdfOptions.EmbedTrueTypeFontsForASCII właściwość

Ustaw na True, aby osadzać czcionki true type dla znaków ASCII 32-127. Czcionki dla kodów znaków większych niż 127 są zawsze osadzane. Odczyt/zapis Boolean.

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### Uwagi

Domyślnie jest **true**.

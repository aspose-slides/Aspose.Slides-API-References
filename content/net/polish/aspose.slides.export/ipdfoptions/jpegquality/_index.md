## IPdfOptions.JpegQuality właściwość

Zwraca lub ustawia wartość określającą jakość obrazów JPEG w dokumencie PDF. Odczyt/zapis Byte.

```csharp
public byte JpegQuality { get; set; }
```

### Uwagi

Ma wpływ tylko wtedy, gdy dokument zawiera obrazy JPEG.

Użyj tej właściwości, aby pobrać lub ustawić jakość obrazów w dokumencie podczas zapisywania w formacie PDF. Wartość może wynosić od 0 do 100, gdzie 0 oznacza najgorszą jakość, ale maksymalną kompresję, a 100 oznacza najlepszą jakość, ale minimalną kompresję.

Domyślna wartość to **100**.

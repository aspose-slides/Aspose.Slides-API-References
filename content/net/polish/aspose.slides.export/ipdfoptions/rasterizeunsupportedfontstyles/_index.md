## IPdfOptions.RasterizeUnsupportedFontStyles właściwość

Wskazuje, czy tekst powinien być rasteryzowany jako bitmapa i zapisywany do PDF, gdy czcionka nie obsługuje stylu pogrubionego. To podejście może poprawić jakość tekstu w wynikowym pliku PDF dla niektórych czcionek. Boolean odczyt/zapis.

```csharp
public bool RasterizeUnsupportedFontStyles { get; set; }
```

### Uwagi

Domyślnie **false**.

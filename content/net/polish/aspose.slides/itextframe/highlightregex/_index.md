## ITextFrame.HighlightRegex metoda

Wyróżnia wszystkie dopasowania wyrażenia regularnego określonym kolorem.

```csharp
public void HighlightRegex(Regex regex, Color highlightColor, IFindResultCallback callback)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| regex | Regex | Wyrażenie regularne Regex służące do pobierania łańcuchów do wyróżnienia. |
| highlightColor | Color | Kolor służący do wyróżnienia tekstu. |
| callback | IFindResultCallback | Obiekt wywołania zwrotnego służący do odbierania wyników wyszukiwania [`IFindResultCallback`](../../ifindresultcallback). |

### Przykłady

Poniższy przykład kodu pokazuje, jak wyróżnić tekst w TextFrame przy użyciu wyrażenia regularnego.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// podświetlanie wszystkich słów o długości 10 lub więcej znaków
	((AutoShape)presentation.Slides[0].Shapes[0]).TextFrame.HighlightRegex(regex, Color.Blue, null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

## ITextFrame.ReplaceRegex metoda

Zastępuje wszystkie dopasowania wyrażenia regularnego podanym ciągiem znaków.

```csharp
public void ReplaceRegex(Regex regex, string newText, IFindResultCallback callback)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| regex | Regex | Wyrażenie regularne Regex używane do pobierania ciągów do zastąpienia. |
| newText | String | Ciąg znaków służący do zastąpienia wszystkich wystąpień ciągów do zastąpienia. |
| callback | IFindResultCallback | Obiekt wywołania zwrotnego służący do odbierania wyników wyszukiwania [`IFindResultCallback`](../../ifindresultcallback). |

### Przykłady

Poniższy przykład kodu pokazuje, jak zastąpić tekst przy użyciu wyrażenia regularnego podanym ciągiem znaków.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// Zastąp wszystkie słowa mające 10 lub więcej znaków ciągiem '***'
	((AutoShape)presentation.Slides[0].Shapes[0]).TextFrame.ReplaceRegex(regex, "***", null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

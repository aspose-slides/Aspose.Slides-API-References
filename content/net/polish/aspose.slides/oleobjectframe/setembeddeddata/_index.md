## OleObjectFrame.SetEmbeddedData metoda

Ustawia informacje o osadzonych danych OLE.  Ta metoda zmienia właściwości obiektu, aby odzwierciedlić nowe dane i ustawia flagę IsObjectLink na false, wskazując, że obiekt OLE jest osadzony.

```csharp
public void SetEmbeddedData(IOleEmbeddedDataInfo embeddedData)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| embeddedData | IOleEmbeddedDataInfo | Dane osadzone [`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo) |

### Wyjątki

| Wyjątek | Warunek |
| --- | --- |
| ArgumentNullException | Gdy parametr embeddedData jest null. |

### Przykłady

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    OleObjectFrame oof = pres.Slides[0].Shapes[0] as OleObjectFrame;
    if (oof != null)
    {
        IOleEmbeddedDataInfo newData = new OleEmbeddedDataInfo(File.ReadAllBytes("Picture.png"), "png");
        oof.SetEmbeddedData(newData);
    }
}
```

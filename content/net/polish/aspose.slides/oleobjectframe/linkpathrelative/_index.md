## OleObjectFrame.LinkPathRelative właściwość

Zwraca względną ścieżkę do powiązanego pliku, jeśli istnieje; w przeciwnym razie zwraca pusty ciąg znaków. Tylko do odczytu String.

```csharp
public string LinkPathRelative { get; }
```

### Uwagi

W prezentacjach Ppt niektóre łącza obiektów Ole mogą mieć reprezentację względną.

### Przykłady

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.ppt"))
{
    IOleObjectFrame oleFrame = presentation.Slides[0].Shapes[0] as IOleObjectFrame;

    if (oleFrame != null)
    {
        Console.WriteLine("The relative path: " + oleFrame.LinkPathRelative);
    } 
}
```

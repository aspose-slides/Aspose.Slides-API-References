## LoadOptions.DefaultRegularFont właściwość

Zwraca lub ustawia czcionkę Regular używaną w przypadku, gdy nie znaleziono czcionki źródłowej. Odczyt/zapis String.

```csharp
public string DefaultRegularFont { get; set; }
```

### Przykłady

Poniższy przykład pokazuje, jak ustawić domyślne czcionki dla renderowania PowerPoint Presentation.

```csharp
[C#]
// Użyj opcji ładowania, aby określić domyślne czcionki regularne i azjatyckie
LoadOptions loadOptions = new LoadOptions(LoadFormat.Auto);
loadOptions.DefaultRegularFont = "Wingdings";
loadOptions.DefaultAsianFont = "Wingdings";
// Załaduj prezentację
using (Presentation pptx = new Presentation("DefaultFonts.pptx", loadOptions))
{
    // Generuj miniaturę slajdu
    pptx.Slides[0].GetThumbnail(1, 1).Save("output_out.png", ImageFormat.Png);
    // Generuj PDF
    pptx.Save("output_out.pdf", SaveFormat.Pdf);
    // Generuj XPS
    pptx.Save("output_out.xps", SaveFormat.Xps);
}
```

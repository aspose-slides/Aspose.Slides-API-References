## AudioFrame.TrimFromStart właściwość

Określa czas trwania, który ma zostać usunięty z początku multimediów podczas odtwarzania, w milisekundach. Odczyt/zapis Single.

```csharp
public float TrimFromStart { get; set; }
```

### Przykłady

Przykład:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    IAudio audio = pres.Audios.AddAudio(File.ReadAllBytes("sampleaudio.mp3"));
    IAudioFrame audioFrame = pres.Slides[0].Shapes.AddAudioFrameEmbedded(50, 50, 100, 100, audio);

    // Ustaw czas przycinania początkowego na 1,5 sekundy
```

## AudioFrame.TrimFromEnd właściwość

Określa czas trwania, który ma być usunięty z końca mediów podczas odtwarzania, w milisekundach. odczyt/zapis Single.

```csharp
public float TrimFromEnd { get; set; }
```

Przykład:

```csharp
[C#]
using (Presentation pres = new Presentation())
```

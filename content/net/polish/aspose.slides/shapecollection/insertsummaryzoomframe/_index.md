## ShapeCollection.InsertSummaryZoomFrame method

Tworzy nową ramkę Summary Zoom i wstawia ją do kolekcji kształtów w określonym indeksie.

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| index | Int32 | Indeks zerowy, w którym ma zostać wstawiona ramka Summary Zoom. |
| x | Single | Współrzędna x nowej ramki Summary Zoom, w punktach. |
| y | Single | Współrzędna y nowej ramki Summary Zoom, w punktach. |
| width | Single | Szerokość nowej ramki Summary Zoom, w punktach. |
| height | Single | Wysokość nowej ramki Summary Zoom, w punktach. |

### Wartość zwracana

Nowo utworzony [`ISummaryZoomFrame`](../../isummaryzoomframe).

### Wyjątki

| Wyjątek | Warunek |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | Rzucany, jeśli prezentacja nie zawiera sekcji lub jeśli docelowy slajd nie należy do żadnej sekcji. |

### Uwagi

Ta metoda tworzy ramkę Summary Zoom, która agreguje linki podsumowujące dla wszystkich sekcji w prezentacji.

### Przykłady

Ten przykład demonstruje tworzenie i wstawianie obiektu Summary Zoom w określonym indeksie kolekcji (zakładając, że w prezentacji "Presentation.pptx" znajduje się co najmniej dwie sekcje):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSummaryZoomFrame(2, 150, 20, 50, 50);
}
```

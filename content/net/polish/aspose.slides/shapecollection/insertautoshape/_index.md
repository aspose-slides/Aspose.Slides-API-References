Tworzy nowy automatyczny kształt i wstawia go do kolekcji kształtów w określonym indeksie, opcjonalnie inicjalizując go domyślnym stylem szablonu.

```csharp
public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| index | Int32 | Zero-based index, w którym ma zostać wstawiony automatyczny kształt. |
| shapeType | ShapeType | Typ [`ShapeType`](../../shapetype) automatycznego kształtu do wstawienia. |
| x | Single | Współrzędna x ramki kształtu, w punktach. |
| y | Single | Współrzędna y ramki kształtu, w punktach. |
| width | Single | Szerokość ramki kształtu, w punktach. |
| height | Single | Wysokość ramki kształtu, w punktach. |
| createFromTemplate | Boolean | True, aby zastosować domyślny styl szablonu (w tym niepustą nazwę, prosty styl i wyśrodkowany tekst); false, aby utworzyć kształt ze wszystkimi właściwościami ustawionymi na wartości domyślne. |

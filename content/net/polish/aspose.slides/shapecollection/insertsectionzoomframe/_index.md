## InsertSectionZoomFrame(int, float, float, float, float, ISection) {#insertsectionzoomframe}

Tworzy nową ramkę Section Zoom i wstawia ją do kolekcji kształtów pod określonym indeksem.

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| index | Int32 | Indeks zero-bazowy, pod którym ma zostać wstawiona ramka Section Zoom. |
| x | Single | Współrzędna x nowej ramki Section Zoom, w punktach. |
| y | Single | Współrzędna y nowej ramki Section Zoom, w punktach. |
| width | Single | Szerokość nowej ramki Section Zoom, w punktach. |
| height | Single | Wysokość nowej ramki Section Zoom, w punktach. |
| section | ISection | [`ISection`](../../isection) odwoływany przez ramkę Section Zoom; musi należeć do tej prezentacji i zawierać co najmniej jeden slajd. |

### Return Value

Nowo utworzony [`ISectionZoomFrame`](../../isectionzoomframe).

### Exceptions

| exception | condition |
| --- | --- |
| ArgumentException | Rzucany, jeśli odwoływana sekcja nie należy do bieżącej prezentacji lub nie zawiera żadnych slajdów. |

### Examples

Ten przykład demonstruje tworzenie i wstawianie obiektu Section Zoom pod określonym indeksem w kolekcji (zakładając, że w prezentacji „Presentation.pptx” istnieją co najmniej dwie sekcje):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSectionZoomFrame(2, 150, 20, 50, 50, pres.Sections[1]);
}
```

public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width,

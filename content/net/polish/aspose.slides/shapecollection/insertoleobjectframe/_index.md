## InsertOleObjectFrame(int, float, float, float, float, IOleEmbeddedDataInfo) {#insertoleobjectframe}

Tworzy nową ramkę obiektu OLE i wstawia ją do kolekcji kształtów w określonym indeksie.

```csharp
public IOleObjectFrame InsertOleObjectFrame(int index, float x, float y, float width, float height, 
    IOleEmbeddedDataInfo dataInfo)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| index | Int32 | Zero-oparty indeks, w którym wstawia się ramkę obiektu OLE. |
| x | Single | Współrzędna x nowej ramki OLE, wyrażona w punktach. |
| y | Single | Współrzędna y nowej ramki OLE, wyrażona w punktach. |
| width | Single | Szerokość nowej ramki OLE, wyrażona w punktach. |
| height | Single | Wysokość nowej ramki OLE, wyrażona w punktach. |
| dataInfo | IOleEmbeddedDataInfo | Wbudowane informacje o danych OLE ([`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo)). |

### Wartość zwracana

Nowo utworzony [`IOleObjectFrame`](../../ioleobjectframe).

### Przykłady

Ten przykład demonstruje wstawianie obiektu OLE pod drugim indeksem:

```csharp
[C#]

IOleDataInfo dataInfo = new OleDataInfo(fileData, "zip");
```

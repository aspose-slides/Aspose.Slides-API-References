## AddSectionZoomFrame(float, float, float, float, ISection) {#addsectionzoomframe}

Tworzy nową ramkę Zoom sekcji i dodaje ją na koniec kolekcji kształtów.

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| x | Single | Współrzędna x nowej ramki Zoom sekcji, w punktach. |
| y | Single | Współrzędna y nowej ramki Zoom sekcji, w punktach. |
| width | Single | Szerokość nowej ramki Zoom sekcji, w punktach. |
| height | Single | Wysokość nowej ramki Zoom sekcji, w punktach. |
| section | ISection | [`ISection`](../../isection) odwoływany przez ramkę Zoom sekcji; musi należeć do tej prezentacji i zawierać co najmniej jeden slajd. |

### Wartość zwracana

Nowo utworzony [`ISectionZoomFrame`](../../isectionzoomframe).

### Wyjątki

| wyjątek | warunek |
| --- | --- |
| ArgumentException | Rzucany, jeśli odwoływana sekcja nie należy do bieżącej prezentacji lub nie zawiera slajdów. |

### Przykłady

Ten przykład pokazuje dodawanie obiektu Section Zoom na koniec kolekcji (zakładając, że w prezentacji "Presentation.pptx" znajduje się co najmniej dwie sekcje):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1]);
}
```

## AddSectionZoomFrame(float, float, float, float, ISection, IPPImage) {#addsectionzoomframe_1}

Tworzy nową ramkę Zoom sekcji z wstępnie określonym obrazem i dodaje ją na koniec kolekcji kształtów.

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section, IPPImage image)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| x | Single | Współrzędna x nowej ramki Zoom sekcji, w punktach. |
| y | Single | Współrzędna y nowej ramki Zoom sekcji, w punktach. |
| width | Single | Szerokość nowej ramki Zoom sekcji, w punktach. |
| height | Single | Wysokość nowej ramki Zoom sekcji, w punktach. |
| section | ISection | [`ISection`](../../isection) odwoływany przez ramkę Zoom sekcji; musi należeć do tej prezentacji i zawierać co najmniej jeden slajd. |
| image | IPPImage | [`IPPImage`](../../ippimage) wyświetlany w ramce Zoom sekcji. |

### Wartość zwracana

Nowo utworzony [`ISectionZoomFrame`](../../isectionzoomframe).

### Wyjątki

| wyjątek | warunek |
| --- | --- |
| ArgumentException | Rzucany, jeśli odwoływana sekcja nie należy do bieżącej prezentacji lub nie zawiera slajdów. |

### Przykłady

Ten przykład pokazuje dodawanie obiektu Section Zoom na koniec kolekcji (zakładając, że w prezentacji "Presentation.pptx" znajduje się co najmniej dwie sekcje):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IPPImage image = pres.Images.AddImage(Image.FromFile("image.png"));
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1], image);
}
```

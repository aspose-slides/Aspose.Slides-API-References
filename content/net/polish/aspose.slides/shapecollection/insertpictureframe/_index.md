## ShapeCollection.InsertPictureFrame metoda

Tworzy nową ramkę obrazu zawierającą określony obraz i wstawia ją do kolekcji kształtów pod określonym indeksem.

```csharp
public IPictureFrame InsertPictureFrame(int index, ShapeType shapeType, float x, float y, 
    float width, float height, IPPImage image)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| index | Int32 | Indeks zerowy, pod którym zostanie wstawiona ramka obrazu. |
| shapeType | ShapeType | Określa typ kształtu zawarty w [`ShapeType`](../../shapetype), z wyjątkiem wszystkich rodzajów linii: |
| x | Single | Współrzędna x ramki obrazu, w punktach. |
| y | Single | Współrzędna y ramki obrazu, w punktach. |
| width | Single | Szerokość ramki obrazu, w punktach. |
| height | Single | Wysokość ramki obrazu, w punktach. |
| image | IPPImage | Obiekt [`IPPImage`](../../ippimage) wyświetlany w ramce obrazu. |

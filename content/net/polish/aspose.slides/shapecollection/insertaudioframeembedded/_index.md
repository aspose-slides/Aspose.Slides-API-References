## InsertAudioFrameEmbedded(int, float, float, float, float, Stream) {#insertaudioframeembedded_1}

Tworzy nową ramkę audio z osadzonym plikiem WAV i wstawia ją do kolekcji kształtów w określonym indeksie. Osadzony dźwięk jest dodawany do kolekcji Presentation.Audios.

```csharp
public IAudioFrame InsertAudioFrameEmbedded(int index, float x, float y, float width, float height, 
    Stream audio_stream)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | Int32 | Indeks zerowy, w którym należy wstawić ramkę audio. |
| x | Single | Współrzędna x nowej ramki audio, w punktach. |
| y | Single | Współrzędna y nowej ramki audio, w punktach. |
| width | Single | Szerokość nowej ramki audio, w punktach. |
| height | Single | Wysokość nowej ramki audio, w punktach. |
| audio_stream | Stream | Strumień wejściowy zawierający dane audio WAV do osadzenia. |

Nowo utworzony [`IAudioFrame`](../../iaudioframe).

| height | Single | Wysokość nowej ramki audio, w punktach. |
| audio | IAudio | Instancja [`IAudio`](../../iaudio) z kolekcji Presentation.Audios do osadzenia. |

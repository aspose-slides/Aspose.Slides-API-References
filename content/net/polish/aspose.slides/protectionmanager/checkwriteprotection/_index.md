## ProtectionManager.CheckWriteProtection metoda

Określa, czy prezentacja jest zabezpieczona hasłem przed modyfikacją.

```csharp
public bool CheckWriteProtection(string password)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| password | String | Hasło używane do sprawdzenia. |

### Wartość zwrotna

True, jeśli hasło jest prawidłowe; w przeciwnym razie false.

### Uwagi

1. Należy sprawdzić [`IsWriteProtected`](../iswriteprotected) własność przed wywołaniem tej metody. 2. Gdy hasło jest null lub puste, metoda zwraca false.

### Przykłady

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

## TextFrame.ReplaceRegex metoda

Zastępuje wszystkie dopasowania wyrażenia regularnego określonym ciągiem znaków.

```csharp
public void ReplaceRegex(Regex regex, string newText, IFindResultCallback callback)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| regex | Regex | Wyrażenie regularne Regex służące do pobierania ciągów znaków do zastąpienia. |
| newText | String | Ciąg znaków służący do zastąpienia wszystkich wystąpień ciągów znaków do zastąpienia. |
| callback | IFindResultCallback | Obiekt wywołania zwrotnego służący do zapisywania wyniku operacji zastąpienia [`IFindResultCallback`](../../ifindresultcallback). |

### Przykłady

Poniższy przykładowy kod pokazuje, jak zastąpić tekst przy użyciu wyrażenia regularnego określonym ciągiem znaków.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx")){
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// Zastąp wszystkie słowa o długości 10 lub więcej znaków ciągiem '***'
	((AutoShape)presentation.Slides[0].Shapes[0]).TextFrame.ReplaceRegex(regex, "***", null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

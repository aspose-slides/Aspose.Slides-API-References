## IAudioFrame.RewindAudio właściwość

Determines whether an audio is automatically rewinded to start after playing. Odczyt/zapis Boolean.

```csharp
public bool RewindAudio { get; set; }
```

### Przykłady

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ISlide slide = pres.Slides[0];

    // Dodaj ramkę audio
    IAudioFrame audioFrame = slide.Shapes.AddAudioFrameLinked(50, 50, 100, 100, "sampleaudio.wav");

    // Ustaw audio, aby odtwarzało się na wszystkich slajdach
    audioFrame.PlayAcrossSlides = true;

    // Ustaw audio, aby automatycznie przewijało się do początku po odtworzeniu
    audioFrame.RewindAudio = true;

    pres.Save("AudioFrame_out.pptx", SaveFormat.Pptx);
}
```

## BlobManagementOptions klasa

Reprezentuje opcje, które można używać do zarządzania regułami obsługi BLOB-ów oraz innymi ustawieniami BLOB.

```csharp
public class BlobManagementOptions : IBlobManagementOptions
```

| --- | --- |
| [BlobManagementOptions](blobmanagementoptions)() | Tworzy nowe domyślne opcje zarządzania blobami. |

## Właściwości

| Nazwa | Opis |
| --- | --- |
| [IsTemporaryFilesAllowed](../../aspose.slides/blobmanagementoptions/istemporaryfilesallowed) { get; set; } | Ta właściwość określa, czy podczas pracy z BLOB-ami mogą być tworzone pliki tymczasowe, co znacząco zmniejsza zużycie pamięci, ale wymaga uprawnień do tworzenia plików. Wszystkie pliki zostaną usunięte po zakończeniu pracy z prezentacją. |
| [MaxBlobsBytesInMemory](../../aspose.slides/blobmanagementoptions/maxblobsbytesinmemory) { get; set; } | Określa maksymalny łączny rozmiar (w bajtach), jaki wszystkie BLOB-y mogą zajmować w pamięci. Domyślnie wszystkie BLOB-y są ładowane do pamięci; dopiero po przekroczeniu tego limitu stosowane są alternatywne mechanizmy (takie jak pliki tymczasowe). Przechowywanie BLOB-ów w pamięci maksymalizuje wydajność, ale może prowadzić do wysokiego zużycia pamięci. Użyj tej właściwości, aby dostosować zachowanie do swojego środowiska lub wymagań. |
| [PresentationLockingBehavior](../../aspose.slides/blobmanagementoptions/presentationlockingbehavior) { get; set; } | Ta właściwość określa, czy instancja klasy Presentation może być właścicielem źródła – pliku lub strumienia – podczas swojego życia. Jeśli instancja jest właścicielem, blokuje źródło. Pomaga to poprawić zużycie pamięci i wydajność podczas pracy z BLOB-ami, ale źródło (strumień lub plik) nie może być zmieniane w trakcie życia instancji Presentation. |
| [TempFilesRootPath](../../aspose.slides/blobmanagementoptions/tempfilesrootpath) { get; set; } | Ścieżka główna, w której będą tworzone pliki tymczasowe. Domyślnie używany jest systemowy katalog tymczasowy. Proces hostujący powinien mieć uprawnienia do tworzenia plików i folderów w tym miejscu. |

## IPresentation.ReplaceRegex metoda

Zastępuje wszystkie dopasowania wyrażenia regularnego podanym łańcuchem znaków.

```csharp
public void ReplaceRegex(Regex regex, string newText, IFindResultCallback callback)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| regex | Regex | Wyrażenie regularne Regex określające łańcuchy do zastąpienia. |
| newText | String | Łańcuch znaków służący do zastąpienia wszystkich wystąpień łańcuchów do zamiany. |
| callback | IFindResultCallback | Obiekt wywołania zwrotnego służący do odbierania wyników wyszukiwania [`IFindResultCallback`](../../ifindresultcallback). |

### Przykłady

Poniższy przykład kodu pokazuje, jak zastąpić tekst przy użyciu wyrażenia regularnego podanym łańcuchem znaków.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// Zastąp wszystkie słowa mające 10 lub więcej znaków znakami '***'
	presentation.ReplaceRegex(regex, "***", null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

## IPresentation.ReplaceText metoda

Zamienia wszystkie wystąpienia określonego tekstu na inny określony tekst.

```csharp
public void ReplaceText(string oldText, string newText, ITextSearchOptions options, 
    IFindResultCallback callback)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| oldText | String | Ciąg znaków do zastąpienia. |
| newText | String | Ciąg znaków zastępujący wszystkie wystąpienia oldText. |
| options | ITextSearchOptions | Opcje wyszukiwania tekstu [`ITextSearchOptions`](../../itextsearchoptions). |
| callback | IFindResultCallback | Obiekt wywołania zwrotnego służący do odbierania wyników wyszukiwania [`IFindResultCallback`](../../ifindresultcallback). |

### Przykłady

Poniższy przykładowy kod pokazuje, jak zamienić jeden określony ciąg znaków na inny określony ciąg znaków.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	// Zamień wszystkie oddzielne wystąpienia 'the' na '***'
	presentation.ReplaceText("the", "***", new TextSearchOptions() {WholeWordsOnly = true}, null);
	presentation.Save("SomePresentation-out2.pptx", SaveFormat.Pptx);
}
```

## IPresentation.HighlightRegex metoda

Podświetla wszystkie dopasowania wyrażenia regularnego przy użyciu określonego koloru.

```csharp
public void HighlightRegex(Regex regex, Color highlightColor, IFindResultCallback callback)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| regex | Regex | Wyrażenie regularne Regex służące do uzyskania ciągów do podświetlenia. |
| highlightColor | Color | Kolor używany do podświetlenia tekstu. |
| callback | IFindResultCallback | Obiekt wywołania zwrotnego służący do odbierania wyników wyszukiwania [`IFindResultCallback`](../../ifindresultcallback). |

### Przykłady

Poniższy przykład kodu pokazuje, jak podświetlić tekst w prezentacji PowerPoint przy użyciu wyrażenia regularnego.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// podświetlanie wszystkich słów o długości 10 lub więcej znaków
	presentation.HighlightRegex(regex, Color.Blue, null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

## IProtectionManager.CheckWriteProtection metoda

Określa, czy prezentacja jest chroniona hasłem przed modyfikacją.

```csharp
public bool CheckWriteProtection(string password)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| password | String | Hasło do sprawdzenia. |

### Wartość zwracana

True, jeśli hasło jest prawidłowe; w przeciwnym razie false.

### Uwagi

1. Należy sprawdzić właściwość [`IsWriteProtected`](../iswriteprotected) przed wywołaniem tej metody. 2. Gdy hasło jest null lub puste, metoda zwraca false.

### Przykłady

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

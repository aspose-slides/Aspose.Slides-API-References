## IBasePortionFormat.SpellCheck właściwość

Pobiera lub ustawia wartość wskazującą, czy sprawdzanie pisowni jest włączone dla fragmentu tekstu. Gdy ta właściwość jest ustawiona na false, sprawdzanie pisowni dla elementów tekstowych jest pomijane. Gdy jest ustawiona na true, sprawdzanie pisowni jest dozwolone. Wartość domyślna to `false`.

```csharp
public bool SpellCheck { get; set; }
```

### Przykłady

Next example demonstrates enabling the SpellCheck flag before saving the presentation:

```csharp
[C#]
using (var pres = new Presentation("input.pptx"))
{
    // Uzyskaj dostęp do pierwszego fragmentu tekstu wewnątrz pierwszego kształtu na pierwszym slajdzie
    var portion = ((AutoShape)pres.Slides[0].Shapes[0]).TextFrame.Paragraphs[0].Portions[0];
    // Włącz sprawdzanie pisowni dla tego fragmentu tekstu
    portion.PortionFormat.SpellCheck = true;
    // Zapisz zmodyfikowaną prezentację
    pres.Save("output-with-spellcheck.pptx", SaveFormat.Pptx);
}
```

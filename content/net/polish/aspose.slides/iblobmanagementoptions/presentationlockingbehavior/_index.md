## IBlobManagementOptions.PresentationLockingBehavior właściwość

Ta właściwość określa, czy instancja klasy Presentation może być właścicielem źródła - pliku lub strumienia podczas czasu życia instancji. Jeśli instancja jest właścicielem, blokuje źródło. Pomaga to poprawić zużycie pamięci i wydajność podczas pracy z BLOB-ami, ale źródło (strumień lub plik) nie może być zmieniane w czasie życia instancji Presentation. Przykład:

```csharp
public PresentationLockingBehavior PresentationLockingBehavior { get; set; }
```

### Przykłady

```csharp
LoadOptions loadOptions = new LoadOptions { 
  BlobManagementOptions = {PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked} };
using (Presentation pres = new Presentation("pres.pptx", loadOptions)) {
  // IOException zostanie rzucony, ponieważ pres.pptx jest zablokowany na całym okresie życia Presentation
  // File.Delete("pres.pptx");
}
// po zwolnieniu obiektu Presentation plik jest odblokowany i może zostać usunięty
File.Delete("pres.pptx");
```

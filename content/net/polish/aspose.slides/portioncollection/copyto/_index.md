## PortionCollection.CopyTo metoda

Kopiuje elementy ICollection do Array, zaczynając od określonego indeksu Array.

```csharp
public void CopyTo(IPortion[] array, int arrayIndex)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| array | IPortion[] | Jednowymiarowa tablica Array, która jest miejscem docelowym elementów kopiowanych z ICollection. Tablica musi mieć indeksowanie zerowe. |
| arrayIndex | Int32 | Indeks zerobazowy w *array*, od którego rozpoczyna się kopiowanie. |

### Wyjątki

| wyjątek | warunek |
| --- | --- |
| ArgumentNullException | *array* jest nullem. |
| ArgumentOutOfRangeException | *arrayIndex* jest mniejsze niż 0. |
| ArgumentException | Liczba elementów w źródłowym ICollection jest większa niż dostępna przestrzeń od *arrayIndex* do końca docelowego *array*. |

## FontFallBackRulesCollection.Add metoda

Dodaj określoną regułę FallBack na koniec kolekcji.

```csharp
public void Add(IFontFallBackRule sourceRule)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| sourceRule | IFontFallBackRule | Określona reguła do dodania |

### Przykłady

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    //Pobieranie pustej lub wstępnie zainicjowanej kolekcji reguł z FontsManager
    IFontFallBackRulesCollection rulesList = pres.FontsManager.FontFallBackRulesCollection;

    //Dodawanie nowej reguły do kolekcji
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));
}
```

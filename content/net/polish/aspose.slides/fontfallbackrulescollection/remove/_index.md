## FontFallBackRulesCollection.Remove metoda

Usuwa pierwsze wystąpienie określonej reguły FallBack z kolekcji.

```csharp
public void Remove(IFontFallBackRule targetRule)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| targetRule | IFontFallBackRule | Reguła do usunięcia z kolekcji. |

### Przykłady

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    // Pobieranie pustej lub wstępnie zainicjowanej kolekcji reguł z FontsManager
    IFontFallBackRulesCollection rulesList = pres.FontsManager.FontFallBackRulesCollection;

    // Dodawanie kilku reguł do kolekcji
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));
    rulesList.Add(new FontFallBackRule(0x3040, 0x309F, "MS Mincho"));

    // Pobieranie obiektu pierwszej reguły w kolekcji
    IFontFallBackRule firstRule = rulesList[0];
    // Usuwanie
    rulesList.Remove (firstRule);
}
```

## PictureFillFormat.TileFlip właściwość

Odwraca kafelek tekstury wzdłuż jego poziomej, pionowej lub obu osi. Odczyt/zapis [`TileFlip`](../../tileflip).

```csharp
public TileFlip TileFlip { get; set; }
```

### Uwagi

Domyślnie jest NoFlip.

### Przykłady

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Pobiera format wypełnienia obrazem kształtu
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Ustawia tryb wypełnienia obrazem na Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Odwraca kafelek tekstury wzdłuż osi pionowej.
    pictureFillFormat.TileFlip = TileFlip.FlipY;
}
```

## PictureFillFormat.TileAlignment właściwość

Zwraca lub ustawia sposób, w jaki tekstura jest wyrównana w obrębie kształtu. To ustawienie kontroluje punkt początkowy wzoru tekstury oraz sposób, w jaki jest ona powtarzana w kształcie. Odczyt/zapis [`RectangleAlignment`](../../rectanglealignment).

```csharp
public RectangleAlignment TileAlignment { get; set; }
```

### Uwagi

Domyślnie jest TopLeft.

### Przykłady

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Pobiera format wypełnienia obrazu dla kształtu
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Ustawia tryb wypełnienia obrazu na Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Ustawia wyrównanie kafelkowania na prawy dolny
    pictureFillFormat.TileAlignment = RectangleAlignment.BottomRight;
}
```

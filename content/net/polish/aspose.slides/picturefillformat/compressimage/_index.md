## CompressImage(bool, PicturesCompression) {#compressimage}

Kompresuje obraz, zmniejszając jego rozmiar na podstawie rozmiaru kształtu i określonej rozdzielczości. Opcjonalnie usuwa również przycięte obszary.

```csharp
public bool CompressImage(bool deleteCroppedAreasOfImage, PicturesCompression resolution)
```

| Parameter | Type | Description |
| --- | --- | --- |
| deleteCroppedAreasOfImage | Boolean | Jeżeli true, metoda usunie przycięte obszary obrazu, co może dodatkowo zmniejszyć jego rozmiar. |
| resolution | PicturesCompression | Docelowa rozdzielczość kompresji, określona jako wartość wyliczenia [`PicturesCompression`](../../../aspose.slides.export/picturescompression). |

### Return Value

Wartość Boolean wskazująca, czy obraz został pomyślnie skompresowany. Zwraca `true`, jeśli obraz został zmieniony rozmiar lub przycięty, w przeciwnym razie `false`.

### Exceptions

| exception | condition |
| --- | --- |
| ArgumentException | Rzucany, gdy rozdzielczość nie jest prawidłową wartością. |

### Remarks

Ta metoda zmienia rozmiar i rozdzielczość obrazu podobnie jak funkcja PowerPoint „Picture Format -> Compress Pictures”.

### Examples

Poniższy przykład pokazuje, jak użyć metody `CompressImage` do zmniejszenia rozmiaru obrazu w prezentacji poprzez ustawienie docelowej rozdzielczości i usunięcie przyciętych obszarów:

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;
    // Kompresuj obraz z docelową rozdzielczością 150 DPI (rozdzielczość sieciowa) i usuń przycięte obszary
    bool result = picFrame.PictureFormat.CompressImage(true, PicturesCompression.Dpi150);
}
```

## CompressImage(bool, float) {#compressimage_1}

Kompresuje obraz, zmniejszając jego rozmiar na podstawie rozmiaru kształtu i określonej rozdzielczości. Opcjonalnie usuwa również przycięte obszary.

```csharp
public bool CompressImage(bool deleteCroppedAreasOfImage, float resolution)
```

| Parameter | Type | Description |
| --- | --- | --- |
| deleteCroppedAreasOfImage | Boolean | Jeżeli true, metoda usunie przycięte obszary obrazu, co może dodatkowo zmniejszyć jego rozmiar. |
| resolution | Single | Docelowa rozdzielczość w DPI. Ta wartość musi być dodatnia i określa, jak obraz zostanie przeskalowany. |

### Return Value

Wartość Boolean wskazująca, czy obraz został pomyślnie skompresowany. Zwraca `true`, jeśli obraz został zmieniony rozmiar lub przycięty, w przeciwnym razie `false`.

### Exceptions

| exception | condition |
| --- | --- |
| ArgumentException | Rzucany, gdy rozdzielczość nie jest dodatnią wartością. |

### Remarks

Ta metoda zmienia rozmiar i rozdzielczość obrazu podobnie jak funkcja PowerPoint „Picture Format -> Compress Pictures”.

### Examples

Poniższy przykład pokazuje, jak użyć metody `CompressImage` do zmniejszenia rozmiaru obrazu w prezentacji poprzez ustawienie docelowej rozdzielczości i usunięcie przyciętych obszarów:

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Pobiera PictureFrame
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // Kompresuj obraz z docelową rozdzielczością 150 DPI (rozdzielczość sieciowa) i usuń przycięte obszary
    bool result=picFrame.PictureFormat.CompressImage(true, 150f); // Rozdzielczość sieciowa
}
```

## PictureFillFormat.TileScaleX właściwość

Zwraca lub ustawia poziom skalowania w poziomie wypełnienia teksturą wyrażony w procentach. Odczyt/zapis Single.

```csharp
public float TileScaleX { get; set; }
```

### Przykłady

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Pobiera format wypełnienia obrazu kształtu
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Ustawia tryb wypełnienia obrazu na Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Ustawia poziome skalowanie tekstury na 120 procent
    pictureFillFormat.TileScaleX = 120;
}
```

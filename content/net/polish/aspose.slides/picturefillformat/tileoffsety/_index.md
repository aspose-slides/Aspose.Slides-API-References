## PictureFillFormat.TileOffsetY właściwość

Zwraca lub ustawia pionowy offset tekstury względem pochodzenia kształtu w punktach. Dodatnia wartość przesuwa teksturę w dół, a ujemna wartość przesuwa ją w górę. Odczyt/zapis Single.

```csharp
public float TileOffsetY { get; set; }
```

### Przykłady

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Pobiera format wypełnienia obrazem kształtu
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Ustawia tryb wypełnienia obrazem na Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Ustawia pionowy offset tekstury na -50 punktów
    pictureFillFormat.TileOffsetY = -50;
}
```

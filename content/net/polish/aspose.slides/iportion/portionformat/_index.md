## IPortion.PortionFormat właściwość

Zwraca obiekt formatowania, który zawiera wyraźnie ustawione właściwości formatowania fragmentu tekstu bez zastosowanego dziedziczenia. Tylko do odczytu [`IPortionFormat`](../../iportionformat).

```csharp
public IPortionFormat PortionFormat { get; }
```

### Uwagi

Obiekt formatowania zawiera parametry formatowania zdefiniowane wyłącznie dla bieżącego fragmentu, dziedziczone dane nie są stosowane.

Aby uzyskać wartości efektywne, w tym dziedziczone, użyj metody [`GetEffective`](../../iportionformat/geteffective).

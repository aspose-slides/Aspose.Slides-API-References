## MasterSlideCollection.InsertClone metoda

Wstawia kopię określonego slajdu nadrzędnego na określoną pozycję w kolekcji. Powiązane slajdy układu zostaną także skopiowane.

```csharp
public IMasterSlide InsertClone(int index, IMasterSlide sourceMaster)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| index | Int32 | Indeks nowego slajdu. |
| sourceMaster | IMasterSlide | Slajd do sklonowania. |

### Wartość zwracana

Wstawiony slajd nadrzędny.

### Przykłady

Poniższy przykład pokazuje, jak sklonować slajd nadrzędny w innej prezentacji PowerPoint.

```csharp
[C#]
// Utwórz instancję klasy Presentation, aby wczytać plik prezentacji źródłowej
using (Presentation srcPres = new Presentation("CloneToAnotherPresentationWithMaster.pptx"))
{
    // Utwórz instancję klasy Presentation dla prezentacji docelowej (gdzie slajd ma być sklonowany)
    using (Presentation destPres = new Presentation())
    {
        // Utwórz instancję ISlide z kolekcji slajdów w prezentacji źródłowej wraz z
        // Slajdem nadrzędnym
        ISlide SourceSlide = srcPres.Slides[0];
        IMasterSlide SourceMaster = SourceSlide.LayoutSlide.MasterSlide;
        // Pobierz slajdy nadrzędne prezentacji docelowej
        IMasterSlideCollection masters = destPres.Masters;
        // Sklonuj żądany slajd nadrzędny z prezentacji źródłowej do kolekcji slajdów nadrzędnych w
        // prezentacji docelowej
        IMasterSlide iSlide = masters.AddClone(SourceMaster);
        // Kolekcja slajdów w prezentacji docelowej
        ISlideCollection slds = destPres.Slides;
        // Sklonuj slajd źródłowy do kolekcji slajdów docelowych.
        slds.AddClone(SourceSlide, iSlide, true);
        // Zapisz prezentację docelową na dysku
        destPres.Save("CloneToAnotherPresentationWithMaster_out.pptx", SaveFormat.Pptx);
    }
}
```

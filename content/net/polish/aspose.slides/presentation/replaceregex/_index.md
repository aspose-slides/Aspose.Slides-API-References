## Presentation.ReplaceRegex metoda

Zastępuje wszystkie dopasowania wyrażenia regularnego określonym ciągiem.

```csharp
public void ReplaceRegex(Regex regex, string newText, IFindResultCallback callback)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| regex | Regex | Wyrażenie regularne Regex służące do pobrania ciągów do zastąpienia. |
| newText | String | Ciąg znaków używany do zastąpienia wszystkich wystąpień ciągów, które mają być zamienione. |
| callback | IFindResultCallback | Obiekt wywołania zwrotnego służący do odbierania wyników wyszukiwania [`IFindResultCallback`](../../ifindresultcallback). |

### Przykłady

Poniższy przykład kodu pokazuje, jak zastąpić tekst przy użyciu wyrażenia regularnego określonym ciągiem.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// Zamień wszystkie słowa o długości 10 lub więcej znaków na '***'
	presentation.ReplaceRegex(regex, "***", null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

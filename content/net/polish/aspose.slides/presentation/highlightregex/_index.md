## Presentation.HighlightRegex metoda

Wyróżnia wszystkie dopasowania wyrażenia regularnego za pomocą określonego koloru.

```csharp
public void HighlightRegex(Regex regex, Color highlightColor, IFindResultCallback callback)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| regex | Regex | Wyrażenie regularne Regex używane do pobrania ciągów do wyróżnienia. |
| highlightColor | Color | Kolor używany do wyróżnienia tekstu. |
| callback | IFindResultCallback | Obiekt wywołania zwrotnego służący do odbierania wyników wyszukiwania [`IFindResultCallback`](../../ifindresultcallback). |

### Przykłady

Poniższy przykład kodu przedstawia, jak wyróżnić tekst w prezentacji PowerPoint przy użyciu wyrażenia regularnego.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// wyróżnianie wszystkich słów składających się z 10 lub więcej znaków
	presentation.HighlightRegex(regex, Color.Blue, null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

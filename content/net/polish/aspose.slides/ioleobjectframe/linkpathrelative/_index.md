## IOleObjectFrame.LinkPathRelative property

Zwraca względną ścieżkę do pliku powiązanego, jeśli istnieje, w przeciwnym razie zwraca pusty łańcuch. Readonly String.

```csharp
public string LinkPathRelative { get; }
```

### Uwagi

W prezentacjach Ppt niektóre powiązania obiektów Ole mogą mieć reprezentację względną.

### Przykłady

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.ppt"))
{
    IOleObjectFrame oleFrame = presentation.Slides[0].Shapes[0] as IOleObjectFrame;

    if (oleFrame != null)
    {
        Console.WriteLine("The relative path: " + oleFrame.LinkPathRelative);
    } 
}
```

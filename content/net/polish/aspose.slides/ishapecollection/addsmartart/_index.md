## IShapeCollection.AddSmartArt metoda

Tworzy diagram SmartArt i dodaje go na koniec kolekcji kształtów.

```csharp
public ISmartArt AddSmartArt(float x, float y, float width, float height, 
    SmartArtLayoutType layoutType)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| x | Single | Współrzędna x ramki diagramu, w punktach. |
| y | Single | Współrzędna y ramki diagramu, w punktach. |
| width | Single | Szerokość ramki diagramu, w punktach. |
| height | Single | Wysokość ramki diagramu, w punktach. |
| layoutType | SmartArtLayoutType | Typ układu SmartArt. |

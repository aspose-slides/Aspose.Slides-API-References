## AddSectionZoomFrame(float, float, float, float, ISection) {#addsectionzoomframe}

Tworzy nową ramkę przybliżenia sekcji i dodaje ją na koniec kolekcji kształtów.

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| x | Single | Współrzędna x nowej ramki przybliżenia sekcji, w punktach. |
| y | Single | Współrzędna y nowej ramki przybliżenia sekcji, w punktach. |
| width | Single | Szerokość nowej ramki przybliżenia sekcji, w punktach. |
| height | Single | Wysokość nowej ramki przybliżenia sekcji, w punktach. |
| section | ISection | [`ISection`](../../isection) odwołujący się do ramki przybliżenia sekcji; musi należeć do tej prezentacji i zawierać co najmniej jeden slajd. |

### Wartość zwracana

Nowo utworzony [`ISectionZoomFrame`](../../isectionzoomframe).

### Wyjątki

| wyjątek | warunek |
| --- | --- |
| ArgumentException | Rzucany, jeśli odwoływana sekcja nie należy do bieżącej prezentacji lub nie zawiera slajdów. |

### Przykłady

Ten przykład demonstruje dodawanie obiektu przybliżenia sekcji na koniec kolekcji (zakładając, że w prezentacji „Presentation.pptx” istnieje co najmniej dwie sekcje):

```csharp
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1]);
}
```

## AddSectionZoomFrame(float, float, float, float, ISection, IPPImage) {#addsectionzoomframe_1}

Tworzy nową ramkę przybliżenia sekcji z wcześniej określonym obrazem i dodaje ją na koniec kolekcji kształtów.

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section, IPPImage image)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| x | Single | Współrzędna x nowej ramki przybliżenia sekcji, w punktach. |
| y | Single | Współrzędna y nowej ramki przybliżenia sekcji, w punktach. |
| width | Single | Szerokość nowej ramki przybliżenia sekcji, w punktach. |
| height | Single | Wysokość nowej ramki przybliżenia sekcji, w punktach. |
| section | ISection | [`ISection`](../../isection) odwołujący się do ramki przybliżenia sekcji; musi należeć do tej prezentacji i zawierać co najmniej jeden slajd. |
| image | IPPImage | [`IPPImage`](../../ippimage) wyświetlany w ramce przybliżenia sekcji. |

### Wartość zwracana

Nowo utworzony [`ISectionZoomFrame`](../../isectionzoomframe).

### Wyjątki

| wyjątek | warunek |
| --- | --- |
| ArgumentException | Rzucany, jeśli odwoływana sekcja nie należy do bieżącej prezentacji lub nie zawiera slajdów. |

### Przykłady

Ten przykład demonstruje dodawanie obiektu przybliżenia sekcji na koniec kolekcji (zakładając, że w prezentacji „Presentation.pptx” istnieje co najmniej dwie sekcje):

```csharp
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IPPImage image = pres.Images.AddImage(Image.FromFile("image.png"));
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1], image);
}
```

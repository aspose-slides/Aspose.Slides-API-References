## AddZoomFrame(float, float, float, float, ISlide) {#addzoomframe}

Tworzy nową ramkę Zoom i dodaje ją na koniec kolekcji kształtów.

```csharp
public IZoomFrame AddZoomFrame(float x, float y, float width, float height, ISlide slide)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| x | Single | Współrzędna x nowej ramki Zoom, wyrażona w punktach. |
| y | Single | Współrzędna y nowej ramki Zoom, wyrażona w punktach. |
| width | Single | Szerokość nowej ramki Zoom, wyrażona w punktach. |
| height | Single | Wysokość nowej ramki Zoom, wyrażona w punktach. |
| slide | ISlide | Referencja do [`ISlide`](../../islide) używana przez ramkę Zoom; musi należeć do tej prezentacji. |

### Wartość zwracana

Nowo utworzony [`IZoomFrame`](../../izoomframe).

### Wyjątki

| Wyjątek | Warunek |
| --- | --- |
| ArgumentException | Rzucany, gdy odwoływany slajd nie należy do bieżącej prezentacji. |

### Przykłady

Ten przykład demonstruje dodanie obiektu Zoom na koniec kolekcji (zakładając, że w prezentacji „Presentation.pptx” znajduje się co najmniej dwa slajdy):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
}
```

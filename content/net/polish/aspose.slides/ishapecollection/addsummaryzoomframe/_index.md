## IShapeCollection.AddSummaryZoomFrame method

Tworzy nową ramkę Summary Zoom i dodaje ją na koniec kolekcji kształtów.

```csharp
public ISummaryZoomFrame AddSummaryZoomFrame(float x, float y, float width, float height)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| x | Single | Współrzędna x nowej ramki Summary Zoom, w punktach. |
| y | Single | Współrzędna y nowej ramki Summary Zoom, w punktach. |
| width | Single | Szerokość nowej ramki Summary Zoom, w punktach. |
| height | Single | Wysokość nowej ramki Summary Zoom, w punktach. |

### Wartość zwracana

Nowo utworzony [`ISummaryZoomFrame`](../../isummaryzoomframe).

### Wyjątki

| Wyjątek | Warunek |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | Rzucany, jeśli w prezentacji nie ma sekcji lub jeśli docelowy slajd nie należy do żadnej sekcji. |

### Uwagi

Ta metoda tworzy ramkę Summary Zoom, która agreguje linki podsumowujące wszystkie sekcje w prezentacji.

### Przykłady

Ten przykład pokazuje dodawanie obiektu Summary Zoom na koniec kolekcji (zakładając, że w prezentacji „Presentation.pptx” znajdują się co najmniej dwie sekcje):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSummaryZoomFrame(150, 20, 500, 250);
}
```

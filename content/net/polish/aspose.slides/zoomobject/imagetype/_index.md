## ZoomObject.ImageType właściwość

Pobiera lub ustawia typ obrazu obiektu zoom. Odczyt/zapis [`ZoomImageType`](../../zoomimagetype). Domyślna wartość: Preview

```csharp
public ZoomImageType ImageType { get; set; }
```

### Uwagi

Określa, czy obiekt Zoom używa podglądu slajdu, czy obrazu okładki.

### Przykłady

Poniższy przykład demonstruje zmianę Image Type na wartość Preview. W tym przypadku bieżący obraz obiektu Zoom zostaje zmieniony na obraz slajdu:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    IPPImage image = pres.Images.AddImage(Image.FromFile("image.png"));
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1], image);
    zoomFrame.ImageType = ZoomImageType.Preview;
}
```

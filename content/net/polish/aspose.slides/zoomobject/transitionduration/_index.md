## ZoomObject.TransitionDuration właściwość

Pobiera lub ustawia czas trwania przejścia między zoomem a slajdem. Odczyt/zapis Single. Wartość domyślna: 1.0f

```csharp
public float TransitionDuration { get; set; }
```

### Uwagi

Jeśli nie określono (TransitionDur = 0), zostanie użyte przejście slajdu docelowego oraz czasy powiązane z tym przejściem.

### Przykłady

Przykład pokazuje zmianę czasu trwania przejścia między zoomem a slajdem:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
    zoomFrame.TransitionDuration = 2.5f;
}
```

## IDocumentProperties.AppVersion właściwość

Zwraca wersję aplikacji. Tylko do odczytu String.

```csharp
public string AppVersion { get; }
```

### Uwagi

Zawartość tego elementu musi mieć postać XX.YYYY, gdzie X i Y oznaczają wartości numeryczne; w przeciwnym razie dokument będzie uznany za niezgodny. Aspose.Slides przedstawia swoją wersję w formacie XX.YYZZ, gdzie: XX - wersja główna YY - wersja pomocnicza ZZ - wersja poprawki. Na przykład wartość 23.0105 oznacza wersję Aspose.Slides 23.1.5.

## IDocumentProperties.GetSensitivityLabels metoda

Pobiera tablicę etykiet poufności z niestandardowych własności dokumentu (Microsoft Information Protection SDK Metadata).

```csharp
public ISensitivityLabel[] GetSensitivityLabels()
```

### Przykłady

Poniższy kod pokazuje, jak przenieść informacje o etykietach poufności z niestandardowych własności dokumentu do nowoczesnej kolekcji SensitivityLabels:

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    // Pobierz etykiety poufności z niestandardowych własności dokumentu
    ISensitivityLabel[] mipSensitivityLabels = pres.DocumentProperties.GetSensitivityLabels();

    ISensitivityLabelCollection sensitivityLabels = pres.SensitivityLabels;
    foreach (var sensitivityLabel in mipSensitivityLabels)
    {
        // Dodaj etykietę do kolekcji
        // Tutaj możesz dodać sprawdzenie ważności informacji o etykiecie (etykieta jest dostępna itp.)
        sensitivityLabels.Add(sensitivityLabel);
    }

    pres.Save("SensitivityLabel.pptx", SaveFormat.Pptx);
}
```

## CommentCollection.AddModernComment metoda

Dodaj nowy nowoczesny komentarz na końcu kolekcji.

```csharp
public IModernComment AddModernComment(string text, ISlide slide, IShape shape, PointF position, 
    DateTime creationTime)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| text | String | Czysty tekst nowego nowoczesnego komentarza. |
| slide | ISlide | Slajd w prezentacji, do którego ma zostać dodany nowy nowoczesny komentarz. |
| shape | IShape | Kształt na slajdzie, z którym powiązany jest nowy nowoczesny komentarz. |
| position | PointF | Pozycja na slajdzie, w której ma zostać dodany nowy nowoczesny komentarz. |
| creationTime | DateTime | Czas utworzenia nowoczesnego komentarza. |

### Wartość zwracana

Dodany nowoczesny komentarz.

### Przykłady

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ICommentAuthor newAuthor = pres.CommentAuthors.AddAuthor("Some Author", "SA");
    newAuthor.Comments.AddModernComment("This is modern comment", pres.Slides[0], null, new PointF(100, 100), DateTime.Now);

    pres.Save(outPptxFileName, SaveFormat.Pptx);
}
```

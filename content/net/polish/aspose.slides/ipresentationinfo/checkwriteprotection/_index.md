## IPresentationInfo.CheckWriteProtection metoda

Sprawdza, czy hasło do modyfikacji jest poprawne dla prezentacji zabezpieczonej przed zapisem.

```csharp
public bool CheckWriteProtection(string password)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| password | String | Hasło do sprawdzenia. |

### Wartość zwracana

True, jeśli prezentacja jest zabezpieczona przed zapisem i podane hasło jest prawidłowe. False w przeciwnym razie.

### Wyjątki

| wyjątek | warunek |
| --- | --- |
| InvalidOperationException |  |

### Uwagi

1. Należy sprawdzić własność [`IsWriteProtected`](../iswriteprotected) przed wywołaniem tej metody. 2. Gdy hasło jest równe null lub jest puste, metoda zwraca false.

### Przykłady

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
if (info.IsWriteProtected == NullableBool.True)
{
    bool isWriteProtectedByPassword = info.CheckWriteProtection("my_password");
}
```

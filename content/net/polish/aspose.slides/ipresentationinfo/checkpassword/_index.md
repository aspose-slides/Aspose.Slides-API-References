## IPresentationInfo.CheckPassword metoda

Sprawdza, czy hasło jest poprawne dla prezentacji zabezpieczonej otwartym hasłem.

```csharp
public bool CheckPassword(string password)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| password | String | Hasło do sprawdzenia. |

### Wartość zwracana

True jeśli prezentacja jest zabezpieczona otwartym hasłem i hasło jest poprawne, w przeciwnym razie false.

### Uwagi

Gdy hasło jest równe null lub jest puste, ta metoda zwraca false.

### Przykłady

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo("pres.pptx");
bool isPasswordCorrect = info.CheckPassword("my_password");
```

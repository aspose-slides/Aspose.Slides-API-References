## Shape.Rotation property

Zwraca lub ustawia liczbę stopni, o które określony kształt jest obrócony wokół osi z. Wartość dodatnia oznacza obrót zgodnie z ruchem wskazówek zegara; wartość ujemna oznacza obrót przeciwnie do ruchu wskazówek zegara. Odczyt/zapis Single.

```csharp
public float Rotation { get; set; }
```

### Uwagi

Zwracana wartość jest zawsze określona (nie jest float.NaN). Przypisana wartość musi być określona (nie jest float.NaN). Można ustawić nieokreślone wartości dla właściwości instancji RawFrame.

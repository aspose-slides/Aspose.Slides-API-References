## ISummaryZoomSectionCollection.AddSummaryZoomSection method

Tworzy nowy obiekt Summary Zoom Section i dodaje go do kolekcji

```csharp
public ISummaryZoomSection AddSummaryZoomSection(ISection section)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| section | ISection | Sekcja dla nowego elementu Summary Zoom Section [`ISection`](../../isection) |

### Wartość zwracana

Dodano [`ISummaryZoomFrame`](../../isummaryzoomframe) element

### Uwagi

Jeśli element dla tej sekcji już istnieje w kolekcji, zwracany jest istniejący element.

### Przykłady

Przykład demonstruje pobieranie elementu Summary Zoom Section według indeksu:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    ISummaryZoomFrame zoomFrame = (ISummaryZoomFrame) pres.Slides[1].Shapes[0];
    ISummaryZoomSectionCollection collection = zoomFrame.SummaryZoomCollection;
    ISummaryZoomSection newZoomSection = collection.AddSummaryZoomSection(pres.Sections[3]);
}
```

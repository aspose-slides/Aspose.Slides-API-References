## IShape.Rotation właściwość

Zwraca lub ustawia liczbę stopni, o które określony kształt jest obrócony wokół osi z. Dodatnia wartość oznacza obrót zgodny z ruchem wskazówek zegara; ujemna wartość oznacza obrót przeciwny do ruchu wskazówek zegara. Odczyt/zapis Single.

```csharp
public float Rotation { get; set; }
```

### Uwagi

Zwracana wartość jest zawsze zdefiniowana (nie jest float.NaN). Przypisywana wartość musi być zdefiniowana (nie float.NaN). Możesz ustawić niezdefiniowane wartości dla właściwości instancji RawFrame.

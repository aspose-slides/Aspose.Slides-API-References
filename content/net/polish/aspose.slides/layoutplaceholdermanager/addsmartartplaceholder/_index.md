## LayoutPlaceholderManager.AddSmartArtPlaceholder metoda

Dodaje nowy kształt zastępujący do slajdu układu, aby przechowywać diagram SmartArt.

```csharp
public IAutoShape AddSmartArtPlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| x | Single | Współrzędna X nowego kształtu zastępującego. |
| y | Single | Współrzędna Y nowego kształtu zastępującego. |
| width | Single | Szerokość nowego kształtu zastępującego. |
| height | Single | Wysokość nowego kształtu zastępującego. |

### Wartość zwracana

Utworzono [`IAutoShape`](../../iautoshape) z elementem zastępczym SmartArt.

### Przykłady

Poniższy przykład pokazuje, jak dodać kształt zastępujący SmartArt do slajdu układu.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddSmartArtPlaceholder(20, 20, 200, 200);
}
```

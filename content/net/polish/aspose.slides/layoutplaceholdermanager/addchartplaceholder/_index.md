## LayoutPlaceholderManager.AddChartPlaceholder metoda

Dodaje nowy kształt elementu zastępczego do slajdu układu, aby pomieścić Chart.

```csharp
public IAutoShape AddChartPlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| x | Single | Współrzędna X nowego elementu zastępczego. |
| y | Single | Współrzędna Y nowego elementu zastępczego. |
| width | Single | Szerokość nowego elementu zastępczego. |
| height | Single | Wysokość nowego elementu zastępczego. |

### Wartość zwracana

Utworzono [`IAutoShape`](../../iautoshape) z elementem zastępczym Chart.

### Przykłady

Poniższy przykład pokazuje, jak dodać kształt elementu zastępczego Chart do slajdu układu.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddChartPlaceholder(20, 20, 200, 200);
}
```

## LayoutPlaceholderManager.AddTablePlaceholder metoda

Dodaje nowy kształt placeholdera do slajdu układu, aby przechowywać tabelę.

```csharp
public IAutoShape AddTablePlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| x | Single | Współrzędna X nowego kształtu placeholdera. |
| y | Single | Współrzędna Y nowego kształtu placeholdera. |
| width | Single | Szerokość nowego kształtu placeholdera. |
| height | Single | Wysokość nowego kształtu placeholdera. |

### Wartość zwracana

Utworzono [`IAutoShape`](../../iautoshape) z placeholderem Table.

### Przykłady

Poniższy przykład pokazuje, jak dodać kształt placeholdera Table do slajdu układu.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTablePlaceholder(20, 20, 500, 200);
}
```

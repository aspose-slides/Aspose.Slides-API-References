## LayoutPlaceholderManager.AddOnlineImagePlaceholder metoda

Dodaje nowy kształt zastępczy do slajdu układu, aby przechowywać obraz online.

```csharp
public IAutoShape AddOnlineImagePlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| x | Single | Współrzędna X nowego kształtu zastępczego. |
| y | Single | Współrzędna Y nowego kształtu zastępczego. |
| width | Single | Szerokość nowego kształtu zastępczego. |
| height | Single | Wysokość nowego kształtu zastępczego. |

### Wartość zwracana

Utworzono [`IAutoShape`](../../iautoshape) z zastępcą obrazu online.

### Przykłady

Poniższy przykład pokazuje, jak dodać kształt zastępczy obrazu online do slajdu układu.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddOnlineImagePlaceholder(20, 20, 200, 200);
}
```

## AddClone(ISlide) {#addclone}

Dodaje kopię określonego slajdu na koniec kolekcji.

```csharp
public ISlide AddClone(ISlide sourceSlide)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| sourceSlide | ISlide | Slide to clone. |

### Wartość zwracana

Nowy slajd.

### Uwagi

Podczas klonowania slajdu pomiędzy różnymi prezentacjami można również sklonować master slajdu. Do śledzenia automatycznie klonowanych masterów używany jest wewnętrzny rejestr, aby zapobiec tworzeniu wielu klonów tego samego mastera slajdu. Ręczne klonowanie masterów slajdów nie będzie ani zapobiegane, ani rejestrowane. Jeśli potrzebujesz większej kontroli nad procesem klonowania, użyj [`AddClone`](../addclone) lub [`AddClone`](../addclone) do klonowania slajdów, [`AddClone`](../../igloballayoutslidecollection/addclone) lub [`AddClone`](../../igloballayoutslidecollection/addclone) do klonowania układów oraz [`AddClone`](../../imasterslidecollection/addclone) do klonowania masterów.

### Przykłady

```csharp
[C#]
using (IPresentation presentation = new Presentation())
{
    presentation.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 200, 50, 300, 100);
    presentation.Sections.AddSection("Section 1", presentation.Slides[0]);
    
    ISection section2 = presentation.Sections.AppendEmptySection("Section 2");
    presentation.Slides.AddClone(presentation.Slides[0], section2);
    
    // Teraz druga sekcja zawiera kopię pierwszego slajdu.
}
```

## AddClone(ISlide, IMasterSlide, bool) {#addclone_2}

Dodaje kopię określonego źródłowego slajdu na koniec kolekcji. Odpowiedni układ zostanie wybrany automatycznie z określonego mastera (odpowiedni układ to układ o tym samym Type lub Name co układ źródłowego slajdu). Jeśli nie ma odpowiedniego układu, układ źródłowego slajdu zostanie sklonowany (jeśli allowCloneMissingLayout jest true) lub zostanie zgłoszony PptxEditException (jeśli allowCloneMissingLayout jest false).

```csharp
public ISlide AddClone(ISlide sourceSlide, IMasterSlide destMaster, bool allowCloneMissingLayout)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| sourceSlide | ISlide | Slide to clone. |
| destMaster | IMasterSlide | Master slide for a new slide. |
| allowCloneMissingLayout | Boolean | Jeśli w określonym masterze nie ma odpowiedniego układu, układ źródłowego slajdu zostanie sklonowany (jeśli allowCloneMissingLayout jest true) lub zostanie zgłoszony PptxEditException (jeśli allowCloneMissingLayout jest false). |

### Wartość zwracana

Nowy slajd.

### Wyjątki

| --- | --- |

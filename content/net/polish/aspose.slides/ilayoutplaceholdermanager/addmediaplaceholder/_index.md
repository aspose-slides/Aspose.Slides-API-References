## ILayoutPlaceholderManager.AddMediaPlaceholder metoda

Dodaje nowy kształt placeholder do slajdu układu, aby pomieścić obiekt multimedialny.

```csharp
public IAutoShape AddMediaPlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| x | Single | Współrzędna X nowego kształtu placeholder. |
| y | Single | Współrzędna Y nowego kształtu placeholder. |
| width | Single | Szerokość nowego kształtu placeholder. |
| height | Single | Wysokość nowego kształtu placeholder. |

### Wartość zwracana

Utworzono [`IAutoShape`](../../iautoshape) z Media placeholder.

### Przykłady

Poniższy przykład pokazuje, jak dodać kształt Media placeholder do slajdu układu.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddMediaPlaceholder(20, 20, 200, 200);
}
```

## ILayoutPlaceholderManager.AddVerticalContentPlaceholder metoda

Dodaje nowy kształt placeholder do slajdu układu, aby przechowywać treść, taką jak obraz, tabela, multimedia lub tekst w pionowym kierunku.

```csharp
public IAutoShape AddVerticalContentPlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| x | Single | Współrzędna X nowego kształtu placeholder. |
| y | Single | Współrzędna Y nowego kształtu placeholder. |
| width | Single | Szerokość nowego kształtu placeholder. |
| height | Single | Wysokość nowego kształtu placeholder. |

### Wartość zwracana

Utworzono [`IAutoShape`](../../iautoshape) z placeholderem Content (Vertical).

### Przykłady

Poniższy przykład pokazuje, jak dodać kształt placeholdera Content (Vertical) do slajdu układu.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalContentPlaceholder(20, 20, 300, 500);
}
```

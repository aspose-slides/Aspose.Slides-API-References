## ILayoutPlaceholderManager.AddSmartArtPlaceholder metoda

Dodaje nowy kształt symbolu zastępczego do slajdu układu, aby pomieścić diagram SmartArt.

```csharp
public IAutoShape AddSmartArtPlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| x | Single | Współrzędna X nowego kształtu symbolu zastępczego. |
| y | Single | Współrzędna Y nowego kształtu symbolu zastępczego. |
| width | Single | Szerokość nowego kształtu symbolu zastępczego. |
| height | Single | Wysokość nowego kształtu symbolu zastępczego. |

### Wartość zwracana

Utworzono [`IAutoShape`](../../iautoshape) z symbolem zastępczym SmartArt.

### Przykłady

Poniższy przykład pokazuje, jak dodać kształt symbolu zastępczego SmartArt do slajdu układu.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddSmartArtPlaceholder(20, 20, 200, 200);
}
```

## ILayoutPlaceholderManager.AddChartPlaceholder method

Dodaje nowy kształt zastępczy do slajdu układu, aby pomieścić wykres.

```csharp
public IAutoShape AddChartPlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| x | Single | Współrzędna X nowego kształtu zastępczego. |
| y | Single | Współrzędna Y nowego kształtu zastępczego. |
| width | Single | Szerokość nowego kształtu zastępczego. |
| height | Single | Wysokość nowego kształtu zastępczego. |

### Wartość zwracana

Utworzono [`IAutoShape`](../../iautoshape) z zastępcą Chart.

### Przykłady

Poniższy przykład pokazuje, jak dodać kształt zastępczy Chart do slajdu układu.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddChartPlaceholder(20, 20, 200, 200);
}
```

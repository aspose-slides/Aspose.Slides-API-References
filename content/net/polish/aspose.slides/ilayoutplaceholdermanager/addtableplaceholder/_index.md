## ILayoutPlaceholderManager.AddTablePlaceholder metoda

Dodaje nowy kształt zastępczy do slajdu układu, aby pomieścić Table.

```csharp
public IAutoShape AddTablePlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| x | Single | The X coordinate of the new placeholder shape. |
| y | Single | The Y coordinate of the new placeholder shape. |
| width | Single | The width of the new placeholder shape. |
| height | Single | The height of the new placeholder shape. |

### Wartość zwracana

Utworzono [`IAutoShape`](../../iautoshape) z zastępcą Table.

### Przykłady

Poniższy przykład pokazuje, jak dodać kształt zastępczy Table do slajdu układu.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTablePlaceholder(20, 20, 500, 200);
}
```

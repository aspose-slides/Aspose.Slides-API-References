## ILayoutPlaceholderManager.AddTextPlaceholder method

Dodaje nowy kształt placeholdera do slajdu układu, aby pomieścić treść tekstową.

```csharp
public IAutoShape AddTextPlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| x | Single | Współrzędna X nowego placeholdera. |
| y | Single | Współrzędna Y nowego placeholdera. |
| width | Single | Szerokość nowego placeholdera. |
| height | Single | Wysokość nowego placeholdera. |

### Return Value

Utworzono [`IAutoShape`](../../iautoshape) z placeholderem Text.

### Examples

Poniższy przykład pokazuje, jak dodać kształt placeholdera Text do slajdu układu.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTextPlaceholder(20, 20, 500, 300);
}
```

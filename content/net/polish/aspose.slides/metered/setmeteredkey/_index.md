## Metered.SetMeteredKey metoda

Ustawia metered publiczny i prywatny klucz. Jeśli zakupisz licencję metered, przy uruchamianiu aplikacji należy wywołać to API; zazwyczaj to wystarcza. Jednakże, jeśli ciągle nie udaje się przesłać danych zużycia i upłynie 24 godziny, licencja zostanie ustawiona w statusie ewaluacyjnym. Aby uniknąć takiej sytuacji, należy regularnie sprawdzać status licencji i w razie statusu ewaluacyjnego ponownie wywołać to API.

```csharp
public void SetMeteredKey(string publicKey, string privateKey)
```

| publicKey | String | klucz publiczny |
| privateKey | String | klucz prywatny |

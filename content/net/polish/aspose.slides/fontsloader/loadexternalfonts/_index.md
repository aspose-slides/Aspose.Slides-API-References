## FontsLoader.LoadExternalFonts metoda

Dodaje dodatkowe foldery do wyszukiwania czcionek.

```csharp
public static void LoadExternalFonts(string[] directories)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| directories | String[] | Katalogi do odczytu dodatkowych czcionek. |

### Przykłady

Poniższe przykłady pokazują, jak załadować własne czcionki z plików .TTF

```csharp
[C#]
// Ścieżka do katalogu z dokumentami.
string dataDir = "C:\\";
// foldery do wyszukiwania czcionek
String[] folders = new String[] { dataDir };
// Załaduj czcionki z własnego katalogu czcionek
FontsLoader.LoadExternalFonts(folders);
// Wykonaj pewne zadania i renderowanie prezentacji/slajdów
using (Presentation presentation = new Presentation(dataDir + "DefaultFonts.pptx"))
{
    presentation.Save(dataDir + "NewFonts_out.pptx", SaveFormat.Pptx);
	// Wyczyść pamięć podręczną czcionek
	FontsLoader.ClearCache();
}
```

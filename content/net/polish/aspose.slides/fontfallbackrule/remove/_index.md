## FontFallBackRule.Remove metoda

Usuwa pierwsze wystąpienie określonej czcionki FallBack z listy.

```csharp
public void Remove(string fontName)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| fontName | String | Nazwa czcionki do usunięcia z listy. |

### Przykłady

```csharp
[C#]
// Utwórz regułę zawierającą listę czcionek.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// Usuń Tahoma z listy.
newRule.Remove ("Tahoma");
```

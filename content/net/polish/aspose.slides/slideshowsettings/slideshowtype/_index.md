## SlideShowSettings.SlideShowType właściwość

Uzyskuje lub ustawia typ pokazu slajdów. Reprezentowane przez następujące przodki `SlideShowType`: [`BrowsedAtKiosk`](../../browsedatkiosk), [`PresentedBySpeaker`](../../presentedbyspeaker) i [`BrowsedByIndividual`](../../browsedbyindividual)

```csharp
public SlideShowType SlideShowType { get; set; }
```

### Przykłady

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // aby ustawić typ "Browsed at a kiosk (full screen)" type 
    pres.SlideShowSettings.SlideShowType = new BrowsedAtKiosk();
    
    // aby ustawić typ "Browsed by individual (window)" type
    pres.SlideShowSettings.SlideShowType = new BrowsedByIndividual();
    
    // aby ustawić typ "Presented by a speaker (full screen)" type
    pres.SlideShowSettings.SlideShowType = new PresentedBySpeaker();
}
```

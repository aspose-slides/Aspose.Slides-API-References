## IFontsManager.FontFallBackRulesCollection właściwość

Reprezentuje kolekcję reguł FontFallBack użytkownika służącą do zarządzania zbiorami czcionek w celu prawidłowych podstawień przez funkcję awaryjną Odczyt/zapis [`IFontFallBackRulesCollection`](../../ifontfallbackrulescollection).

```csharp
public IFontFallBackRulesCollection FontFallBackRulesCollection { get; set; }
```

### Przykłady

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    // Pobieranie pustej lub wstępnie zainicjowanej kolekcji reguł z FontsManager
    IFontFallBackRulesCollection rulesList = pres.FontsManager.FontFallBackRulesCollection;

    // dodawanie reguł do kolekcji
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));

    // lub 
    // inicjalizacja nowej instancji kolekcji reguł
    IFontFallBackRulesCollection rulesList = new FontFallBackRulesCollection();

    // dodawanie reguł do kolekcji
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));

    // i zastąpienie istniejącej kolekcji nową w FontsManager 
    pres.FontsManager.FontFallBackRulesCollection = rulesList;
}
```

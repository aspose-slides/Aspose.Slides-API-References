## IFontsManager.GetFontEmbeddingLevel metoda

Określa poziom osadzania czcionki na podstawie podanej tablicy bajtów i nazwy czcionki.

```csharp
public EmbeddingLevel GetFontEmbeddingLevel(byte[] fontBytes, string fontName)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| fontBytes | Byte[] | Tablica bajtów zawierająca dane czcionki. |
| fontName | String | Nazwa czcionki. |

### Wartość zwracana

Poziom osadzania określonej czcionki.

### Wyjątki

| wyjątek | warunek |
| --- | --- |
| ArgumentNullException | Rzucany, gdy *fontBytes* jest null. |

### Przykłady

```csharp
[C#]
using (Presentation pres = new Presentation(pptxFileName))
{
    // Pobierz wszystkie czcionki użyte w prezentacji
    IFontData[] fontDatas = pres.FontsManager.GetFonts();

    // Pobierz tablicę bajtów reprezentującą styl regularny pierwszej czcionki w prezentacji
    byte[] bytes = pres.FontsManager.GetFontBytes(fontDatas[0], FontStyle.Regular);

    // Określ poziom osadzania czcionki
    EmbeddingLevel embeddingLevel = pres.FontsManager.GetFontEmbeddingLevel(bytes, fontDatas[0].FontName);
}
```

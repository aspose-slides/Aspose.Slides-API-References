## ViewProperties.GridSpacing właściwość

Zwraca lub ustawia odstęp siatki, który powinien być używany dla siatki leżącej u podstaw dokumentu prezentacji, w punktach. Odczyt/Zapis Single.

```csharp
public float GridSpacing { get; set; }
```

### Uwagi

Wartość odstępu siatki musi być liczbą dodatnią. Typowy zakres wartości to od 1 mm (2.8349607 punktów) do 2 cali (144 punktów).

### Przykłady

Poniższy przykładowy kod pokazuje, jak zmienić odstęp siatki w prezentacji PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

## Fonts.GetScriptFont metoda

Pobiera nazwę czcionki powiązaną z określonym tagiem skryptu z motywu prezentacji.

```csharp
public string GetScriptFont(string script)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| script | String | Kod skryptu BCP-47 (np. "Latn", "Cyrl", "Jpan") używany do identyfikacji systemu pisma. |

### Wartość zwracana

Nazwa czcionki użytej dla określonego skryptu lub `null`, jeśli skrypt nie jest zdefiniowany.

### Przykłady

Ten przykład pokazuje, jak uzyskać czcionkę przypisaną do skryptu cyrylicy w motywie prezentacji.

```csharp
[C#]
string font = presentation.MasterTheme.FontScheme.Major.GetScriptFont("Cyrl");
Console.WriteLine("Font for Cyrillic script: " + font);
```

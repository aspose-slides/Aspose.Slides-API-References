## Fonts.SetScriptFont metoda

Przypisuje nazwę czcionki do konkretnego znacznika skryptu, który określa, jak tekst tego skryptu będzie renderowany w prezentacji.

```csharp
public void SetScriptFont(string script, string fontName)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| script | String | Kod skryptu BCP-47 (np., "Arab", "Hebr", "Hans") identyfikujący system pisma. |
| fontName | String | Nazwa czcionki, którą należy przypisać do określonego skryptu. |

### Przykłady

Ten przykład pokazuje, jak ustawić czcionkę dla skryptu arabskiego na "Segoe UI":

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segoe UI");
```

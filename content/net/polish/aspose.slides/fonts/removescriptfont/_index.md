## Fonts.RemoveScriptFont metoda

Usuwa ustawienie czcionki powiązane z określonym tagiem skryptu z kolekcji czcionek motywu.

```csharp
public void RemoveScriptFont(string script)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| script | String | Kod skryptu BCP-47, którego ustawienie czcionki powinno zostać usunięte. |

### Przykłady

Ten przykład demonstruje, jak usunąć mapowanie czcionki dla skryptu hebrajskiego:

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.RemoveScriptFont("Hebr");
```

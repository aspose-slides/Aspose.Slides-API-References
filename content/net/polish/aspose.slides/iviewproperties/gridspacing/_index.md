## IViewProperties.GridSpacing właściwość

Zwraca lub ustawia odstęp siatki, który powinien być używany dla siatki leżącej u podstaw dokumentu prezentacji, w punktach. Odczyt/zapis Single.

```csharp
public float GridSpacing { get; set; }
```

### Uwagi

Wartość odstępu siatki musi być liczbą dodatnią. Typowy zakres wartości wynosi od 1 mm (2.8349607 punktów) do 2 cali (144 punktów).

### Przykłady

Poniższy przykładowy kod pokazuje, jak zmienić odstęp siatki w prezentacji programu PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

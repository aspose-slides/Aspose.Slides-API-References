## IMasterLayoutSlideCollection.Add metoda

Dodaje nowy slajd układu na koniec kolekcji.

```csharp
public ILayoutSlide Add(SlideLayoutType layoutType, string layoutName)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| layoutType | SlideLayoutType | Typ układu dla nowego układu. Obsługiwane typy układów: Title, TitleOnly, Blank, TitleAndObject, VerticalText, VerticalTitleAndText, TwoObjects, SectionHeader, TwoTextAndTwoObjects, TitleObjectAndCaption, PictureAndCaption, Custom. Inne typy układów nie są obecnie obsługiwane: Text, TwoColumnText, Table, TextAndChart, ChartAndText, Diagram, Chart, TextAndClipArt, ClipArtAndText, TextAndObject, ObjectAndText, Object, TextAndMedia, MediaAndText, ObjectOverText, TextOverObject, TextAndTwoObjects, TwoObjectsAndText, TwoObjectsOverText, FourObjects, ClipArtAndVerticalText, VerticalTitleAndTextOverChart, ObjectAndTwoObject, TwoObjectsAndObject. |
| layoutName | String | Nazwa dla nowego układu. Jeśli podana nazwa jest już używana, zostanie wyrzucony ArgumentException. Jeśli przekazany zostanie parametr null, nazwa zostanie wygenerowana automatycznie na podstawie podanego typu układu (na przykład "Title Slide" lub "1_Title Slide", "2_..", itp.). |

### Wartość zwracana

Dodany slajd.

### Wyjątki

| wyjątek | warunek |
| --- | --- |
| NotImplementedException | Wyrzucany, jeśli przekazana zostanie nieobsługiwana wartość parametru *layoutType*. Typy układów, które nie są obecnie obsługiwane: Text, TwoColumnText, Table, TextAndChart, ChartAndText, Diagram, Chart, TextAndClipArt, ClipArtAndText, TextAndObject, ObjectAndText, Object, TextAndMedia, MediaAndText, ObjectOverText, TextOverObject, TextAndTwoObjects, TwoObjectsAndText, TwoObjectsOverText, FourObjects, ClipArtAndVerticalText, VerticalTitleAndTextOverChart, ObjectAndTwoObject, TwoObjectsAndObject. |
| ArgumentException | Wyrzucany, jeśli wartość nazwy układu *layoutName* jest już używana w tej kolekcji układów. |

### Uwagi

1) Dodany układ dla wartości SlideLayoutType.Custom parametru *layoutType* nie zawiera placeholderów ani kształtów.  
2) Analogią tej metody jest metoda [`Add`](../../igloballayoutslidecollection/add) dostępna za pośrednictwem własności [`LayoutSlides`](../../ipresentation/layoutslides).

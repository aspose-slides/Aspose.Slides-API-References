## ISvgImage.WriteAsEmf metoda

Zapisuje obraz SVG jako plik EMF.

```csharp
public void WriteAsEmf(Stream stream)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| stream | Stream | Docelowy strumień |

### Wyjątki

| wyjątek | warunek |
| --- | --- |
| ArgumentNullException | Docelowy strumień jest **null** |

### Przykłady

Poniższy przykład pokazuje, jak zapisać obraz SVG do metafila.

```csharp
[C#]
// Tworzy nowy obraz SVG
ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));

// Zapisuje obraz SVG jako plik metafilu
using (var fileStream = System.IO.File.OpenWrite("SvgAsEmf.emf"))
    svgImage.WriteAsEmf(fileStream);
```

Ten przykład demonstruje, jak dodać obraz SVG jako metafil do kolekcji obrazów prezentacji.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // Tworzy nowy obraz SVG
    ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));
    using (var memStream = new MemoryStream())
    {
        // Zapisuje obraz SVG jako plik metafilu
        svgImage.WriteAsEmf(memStream);
        // Dodaje plik metafilu do kolekcji obrazów
        pres.Images.AddImage(memStream.ToArray());
    }
}
```

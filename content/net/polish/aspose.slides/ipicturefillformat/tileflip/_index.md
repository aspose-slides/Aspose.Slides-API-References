## IPictureFillFormat.TileFlip właściwość

Flips the texture tile around its horizontal, vertical or both axis. Odczyt/zapis [`TileFlip`](../../tileflip).

```csharp
public TileFlip TileFlip { get; set; }
```

### Uwagi

Domyślnie jest NoFlip.

### Przykłady

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Pobiera format wypełnienia obrazem kształtu
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Ustawia tryb wypełnienia obrazem na Tile

    // Odwraca płytkę tekstury wokół jej osi pionowej.

}
```

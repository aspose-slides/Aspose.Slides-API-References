## IPictureFillFormat.TileOffsetY właściwość

Zwraca lub ustawia pionowy offset faktury względem pochodzenia kształtu w punktach. Dodatnia wartość przesuwa fakturę w dół, natomiast ujemna wartość przesuwa ją w górę. Odczyt/zapis Single.

```csharp
public float TileOffsetY { get; set; }
```

### Przykłady

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Pobiera format wypełnienia obrazu kształtu
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Ustawia tryb wypełnienia obrazu na Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Ustawia pionowy offset faktury na -50 punktów
    pictureFillFormat.TileOffsetY = -50;
}
```

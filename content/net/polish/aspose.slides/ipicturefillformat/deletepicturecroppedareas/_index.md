## IPictureFillFormat.DeletePictureCroppedAreas metoda

Usuwa przycięte obszary wypełnienia Picture.

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### Wartość zwracana

Obraz przycięty lub oryginalny obraz, jeśli przycinanie nie jest konieczne.

### Uwagi

Ta metoda konwertuje pliki metafile WMF/EMF na rastrowy obraz PNG podczas przycinania.

### Przykłady

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Pobiera PictureFrame
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // Usuwa przycięte obszary obrazu PictureFrame
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

## Właściwość IPictureFillFormat.TileAlignment

Zwraca lub ustawia sposób wyrównania tekstury w obrębie kształtu. To ustawienie kontroluje punkt początkowy wzoru tekstury i sposób jej powtarzania się na kształcie. Odczyt/zapis [`RectangleAlignment`](../../rectanglealignment).

```csharp
public RectangleAlignment TileAlignment { get; set; }
```

### Uwagi

Domyślnie jest TopLeft.

### Przykłady

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Pobiera format wypełnienia obrazem kształtu
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Ustawia tryb wypełnienia obrazem na Kafelkowanie
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Ustawia wyrównanie kafelkowania na prawy dolny
    pictureFillFormat.TileAlignment = RectangleAlignment.BottomRight;
}
```

## CompressImage(bool, PicturesCompression) {#compressimage}

Kompresuje obraz, zmniejszając jego rozmiar na podstawie rozmiaru kształtu i określonej rozdzielczości. Opcjonalnie usuwa także przycięte obszary.

```csharp
public bool CompressImage(bool deleteCroppedAreasOfImage, PicturesCompression resolution)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| deleteCroppedAreasOfImage | Boolean | Jeśli true, metoda usunie przycięte obszary obrazu, co może dodatkowo zmniejszyć jego rozmiar. |
| resolution | PicturesCompression | Docelowa rozdzielczość dla kompresji, określona jako wartość wyliczenia [`PicturesCompression`](../../../aspose.slides.export/picturescompression). |

### Wartość zwracana

Wartość Boolean wskazująca, czy obraz został pomyślnie skompresowany. Zwraca `true`, jeśli obraz został zmieniony rozmiar lub przycięty, w przeciwnym razie `false`.

### Wyjątki

| wyjątek | warunek |
| --- | --- |
| ArgumentException | Rzucany, gdy rozdzielczość nie jest prawidłową wartością. |

### Uwagi

Ta metoda zmienia rozmiar i rozdzielczość obrazu, podobnie jak funkcja „Picture Format -> Compress Pictures” w programie PowerPoint.

### Przykłady

Poniższy przykład pokazuje, jak użyć metody `CompressImage` do zmniejszenia rozmiaru obrazu w prezentacji, ustawiając docelową rozdzielczość i usuwając przycięte obszary:

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;
    // Skompresuj obraz z docelową rozdzielczością 150 DPI (rozdzielczość sieciowa) i usuń przycięte obszary
    bool result = picFrame.PictureFormat.CompressImage(true, PicturesCompression.Dpi150);
}
```

## CompressImage(bool, float) {#compressimage_1}

```csharp
public bool CompressImage(bool deleteCroppedAreasOfImage, float resolution)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| deleteCroppedAreasOfImage | Boolean | Jeśli true, metoda usunie przycięte obszary obrazu, co może dodatkowo zmniejszyć jego rozmiar. |
| resolution | Single | Docelowa rozdzielczość w DPI. Wartość ta musi być dodatnia i określa, jak obraz zostanie przeskalowany. |

### Wartość zwracana

Wartość Boolean wskazująca, czy obraz został pomyślnie skompresowany. Zwraca `true`, jeśli obraz został zmieniony rozmiar lub przycięty, w przeciwnym razie `false`.

| wyjątek | warunek |
| --- | --- |
| ArgumentException | Rzucany, gdy rozdzielczość nie jest wartością dodatnią. |

### Uwagi

Ta metoda zmienia rozmiar i rozdzielczość obrazu, podobnie jak funkcja „Picture Format -> Compress Pictures” w programie PowerPoint.

### Przykłady

Poniższy przykład pokazuje, jak użyć metody `CompressImage` do zmniejszenia rozmiaru obrazu w prezentacji, ustawiając docelową rozdzielczość i usuwając przycięte obszary:

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Pobiera PictureFrame
```

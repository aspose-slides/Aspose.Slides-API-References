## Właściwość DocumentProperties.HyperlinksChanged

Określa, że jeden lub więcej hiperłączy w tej części zostało zaktualizowanych wyłącznie w tej części przez producenta. Następny producent otwierający ten dokument powinien zaktualizować powiązania hiperlinków przy użyciu nowych hiperłączy określonych w tej części. Odczyt/zapis Boolean.

```csharp
public bool HyperlinksChanged { get; set; }
```

## ParagraphFormat.Alignment property

Zwraca lub ustawia wyrównanie tekstu w akapicie bez dziedziczenia. Odczyt/zapis [`TextAlignment`](../../textalignment).

```csharp
public TextAlignment Alignment { get; set; }
```

### Przykłady

Poniższy przykładowy kod pokazuje, jak wyrównać akapity tekstu w prezentacji PowerPoint.

```csharp
[C#]
// Utwórz obiekt Presentation, który reprezentuje plik PPTX
using (Presentation pres = new Presentation("ParagraphsAlignment.pptx"))
{
    // Dostęp do pierwszego slajdu
    ISlide slide = pres.Slides[0];
    // Dostęp do pierwszego i drugiego placeholdera na slajdzie oraz rzutowanie na AutoShape
    ITextFrame tf1 = ((IAutoShape)slide.Shapes[0]).TextFrame;
    ITextFrame tf2 = ((IAutoShape)slide.Shapes[1]).TextFrame;
    // Zmień tekst w obu placeholderach
    tf1.Text = "Center Align by Aspose";
    tf2.Text = "Center Align by Aspose";
    // Pobranie pierwszego akapitu z placeholderów
    IParagraph para1 = tf1.Paragraphs[0];
    IParagraph para2 = tf2.Paragraphs[0];
    // Wyrównanie akapitu tekstu do środka
    para1.ParagraphFormat.Alignment = TextAlignment.Center;
    para2.ParagraphFormat.Alignment = TextAlignment.Center;
    //Zapis prezentacji jako plik PPTX
    pres.Save("Centeralign_out.pptx", SaveFormat.Pptx);
}
```

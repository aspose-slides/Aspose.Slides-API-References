## BulletFormat.Height właściwość

Zwraca lub ustawia wysokość wypunktowania akapitu bez dziedziczenia. Wartość float.NaN określa, że wypunktowanie dziedziczy wysokość z pierwszej części akapitu. Odczyt/zapis Single.

```csharp
public float Height { get; set; }
```

### Uwagi

Ujemna wartość wysokości oznacza, że wysokość podana jest w punktach, a dodatnia wartość oznacza, że wysokość jest procentem otaczającego tekstu.

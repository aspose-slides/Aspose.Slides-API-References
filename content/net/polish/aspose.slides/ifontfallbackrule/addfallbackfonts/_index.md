## AddFallBackFonts(string) {#addfallbackfonts}

Dodaje nową czcionkę lub czcionki do listy czcionek FallBack.

```csharp
public void AddFallBackFonts(string fontName)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| fontName | String | Nazwa lub nazwy czcionki (oddzielone przecinkami) dla FallBack |

### Przykłady

```csharp
[C#]
//Utworzenie nowej instancji FantFallBackRule
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho");

//Dodaj drugą czcionkę do reguły 
newRule.AddFallBackFonts("MS Gothic");
//Dodaj trzecią i czwartą czcionkę do reguły 
newRule.AddFallBackFonts("Tahoma, Times New Roman");
```

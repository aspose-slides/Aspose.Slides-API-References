## ToArray(int, int) {#toarray_1}

Tworzy i zwraca tablicę zawierającą wszystkie czcionki FallBack z określonego zakresu na liście.

```csharp
public string[] ToArray(int startIndex, int count)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| startIndex | Int32 | Indeks pierwszej czcionki do dodania. |
| count | Int32 | Liczba czcionek do dodania. |

### Wartość zwracana

Tablica typu String

### Przykłady

```csharp
[C#]
// Utwórz regułę zawierającą listę czcionek.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

//Pobierz dwie ostatnie nazwy czcionek jako tablicę
string[] fontNames = newRule.ToArray(2,2);
```

## NotesSlideManager klasa

Menedżer slajdów notatek.

```csharp
public sealed class NotesSlideManager : DomObject<Slide>, INotesSlideManager
```

## Właściwości

### Przykłady

Poniższy przykład pokazuje, jak dodać notatki do określonego slajdu prezentacji PowerPoint.

```csharp
[C#]
	// Utwórz obiekt Presentation, który reprezentuje plik prezentacji
	using(Presentation presentation = new Presentation(dataDir + "AccessSlides.pptx")) {
	  // Dodaj notatki do pierwszego slajdu
	  INotesSlideManager mgr = presentation.Slides[0].NotesSlideManager;
	  INotesSlide noteSlide = mgr.AddNotesSlide();
	  noteSlide.NotesTextFrame.Text = "Your Notes";
	  // Zapisz prezentację na dysku
	  presentation.Save(dataDir + "RemoveNotesAtSpecificSlide_out.pptx", SaveFormat.Pptx);
	}
```

Poniższy przykład pokazuje, jak usunąć notatki z określonego slajdu prezentacji PowerPoint.

```csharp
[C#]
	// Utwórz obiekt Presentation, który reprezentuje plik prezentacji
	using(Presentation presentation = new Presentation(dataDir + "AccessSlides.pptx")) {
	  // Usuwanie notatek z pierwszego slajdu
	  INotesSlideManager mgr = presentation.Slides[0].NotesSlideManager;
	  mgr.RemoveNotesSlide();
	  // Zapisz prezentację na dysku
	  presentation.Save(dataDir + "RemoveNotesAtSpecificSlide_out.pptx", SaveFormat.Pptx);
	}
```

## ExcelDataWorkbook.GetCells method

Pobiera kolekcję komórek z skoroszytu, które pasują do określonego wzoru.

```csharp
public ReadOnlyCollection<IExcelDataCell> GetCells(string formula, bool skipHiddenCells)
```

| Parameter | Type | Description |
| --- | --- | --- |
| formula | String | Wzór lub wyrażenie zakresu (np. "Sheet1!A1:B3") używane do identyfikacji docelowych komórek. |
| skipHiddenCells | Boolean | Jeśli `true`, ukryte komórki (np. w ukrytych wierszach lub kolumnach) zostaną wykluczone z wyniku. |

### Return Value

Lista tylko do odczytu komórek, które pasują do określonego wzoru.

### Examples

Przykład:

```csharp
[C#]
ExcelDataWorkbook wb = new ExcelDataWorkbook(testFile);
IReadOnlyList<IExcelDataCell> cells = wb.GetCells("Sheet1!A2:A6", false);
Console.WriteLine(cells.Count); //Wyjście: 5
```

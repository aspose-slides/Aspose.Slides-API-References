## IMathBorderBox.StrikethroughBottomLeftToTopRight właściwość

Przekreślenie od dolnego lewego do górnego prawego (default is false). Określa ukryty lub widoczny stan przekreślającej linii przekątnej od dolnego lewego rogu do górnego prawego rogu ramki.

```csharp
public bool StrikethroughBottomLeftToTopRight { get; set; }
```

### Przykłady

Przykład:

```csharp
[C#]

borderBox.StrikethroughBottomLeftToTopRight = true;
```

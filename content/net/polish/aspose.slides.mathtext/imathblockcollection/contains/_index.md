## IMathBlockCollection.Contains metoda

Określa, czy kolekcja zawiera określoną wartość.

```csharp
public bool Contains(IMathBlock item)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| item | IMathBlock | Obiekt do znalezienia w kolekcji. |

### Wartość zwracana

true, jeśli *item* zostanie znaleziony w kolekcji; w przeciwnym razie false.

### Przykłady

Przykład:

```csharp
[C#]
IMathBlockCollection blockCollection = new MathParagraph();

blockCollection.Add(block);
bool contains = blockCollection.Contains(block);
```

## MathDelimiter.DelimiterShape właściwość

Określa kształt delimitatorów w obiekcie delimiter. Gdy jest MathDelimiterShape.Centered, delimitery są wyśrodkowane wokół osi matematycznej tekstu i nadal są dopasowywane, aby mieściły się w całej wysokości ich zawartości. Gdy jest MathDelimiterShape.Match, ich wysokość i kształt są zmieniane, aby dokładnie pasowały do ich zawartości.

```csharp
public MathDelimiterShape DelimiterShape { get; set; }
```

### Przykłady

Przykład:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.DelimiterShape = MathDelimiterShape.Match;
```

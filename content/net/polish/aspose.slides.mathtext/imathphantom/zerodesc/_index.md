## IMathPhantom.ZeroDesc właściwość

Pobiera lub ustawia wartość określającą, czy zstąpienie (głębokość poniżej linii bazowej) elementu bazowego powinno być traktowane jako zero.

```csharp
public bool ZeroDesc { get; set; }
```

### Uwagi

Gdy `true`, fantom nie obniża linii bazowej otaczającej linii matematycznej. Odpowiada atrybutowi OMML `m:zeroDesc`.

## IMathPhantom.Transp właściwość

Pobiera lub ustawia wartość określającą, czy fantom jest przezroczysty dla reguł odstępów opartych na klasach.

```csharp
public bool Transp { get; set; }
```

### Uwagi

Gdy `true`, operatory i symbole wewnątrz fantomu nadal wpływają na odstępy matematyczne wokół fantomu (tak jakby były widoczne). Gdy `false`, reguły odstępów oparte na klasach są ignorowane. Odpowiada atrybutowi OMML `m:transp`.

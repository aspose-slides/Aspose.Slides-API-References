## IMathDelimiter.DelimiterShape właściwość

Określa kształt ograniczników w obiekcie delimiter. Gdy jest MathDelimiterShape.Centered, ograniczniki są wyśrodkowane wokół osi matematycznej tekstu i nadal mogą być dopasowane do pełnej wysokości ich zawartości. Gdy jest MathDelimiterShape.Match, ich wysokość i kształt są zmieniane, aby dokładnie odpowiadały ich zawartości.

```csharp
public MathDelimiterShape DelimiterShape { get; set; }
```

### Przykłady

Przykład:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.DelimiterShape = MathDelimiterShape.Match;
```

## IMathDelimiter.BeginningCharacter właściwość

Delimiter Beginning Character określa początkowy, czyli otwierający, znak separatora. Matematyczne delimitery to znaki otaczające, takie jak nawiasy okrągłe, kwadratowe i klamrowe. Domyślna wartość: '('.

```csharp
public char BeginningCharacter { get; set; }
```

Przykład:

IMathDelimiter delimiter = new MathematicalText("x").Join("y").Enclose();

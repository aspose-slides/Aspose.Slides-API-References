## IMathDelimiter.EndingCharacter właściwość

Delimiter Ending Character określa końcowy, czyli zamykający, znak delimitera. Delimitery matematyczne to znaki otaczające, takie jak nawiasy okrągłe, nawiasy kwadratowe i klamry. Domyślnie: ')'.

```csharp
public char EndingCharacter { get; set; }
```

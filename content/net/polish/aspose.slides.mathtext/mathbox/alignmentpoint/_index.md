## MathBox.AlignmentPoint właściwość

Gdy true, ten emulator operatora służy jako punkt wyrównania; tj., wyznaczone punkty wyrównania w innych równaniach mogą być z nim wyrównane. Domyślnie: false

public bool AlignmentPoint { get; set; }

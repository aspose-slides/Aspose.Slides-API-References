## MathBox.ExplicitBreak właściwość

Explicit break określa, czy na początku obiektu Box występuje podział wiersza, tak że wiersz łamie się na początku obiektu Box. Określa numer operatora w poprzedniej linii tekstu matematycznego, który ma być użyty jako punkt wyrównania dla bieżącej linii tekstu matematycznego. możliwe wartości: 1..255 Domyślnie: 0 (brak wyraźnego podziału)

```csharp
public byte ExplicitBreak { get; set; }
```

### Przykłady

Przykład:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.ExplicitBreak = 1;
```

## MathBox.OperatorEmulator właściwość

Emulator operatora. Gdy prawda, pole i jego zawartość zachowują się jak pojedynczy operator i dziedziczą właściwości operatora. Oznacza to na przykład, że znak może służyć jako punkt przerwania linii i może być wyrównany do innych operatorów. Emulatory operatorów są często używane, gdy jeden lub więcej glifów łączy się, tworząc operator, taki jak '=='. Domyślna wartość: false

```csharp
public bool OperatorEmulator { get; set; }
```

### Przykłady

Przykład:

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("=="));
box.OperatorEmulator = true;
```

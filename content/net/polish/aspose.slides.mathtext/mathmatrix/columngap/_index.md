## MathMatrix.ColumnGap właściwość

Wartość odstępu poziomego między kolumnami macierzy; Jeśli ColumnGapRule ma wartość 3 ("Exactly"), jednostka jest interpretowana jako twips (1/20 punktu) Jeśli ColumnGapRule ma wartość 4 ("Multiple"), jednostka jest interpretowana jako liczba przyrostów 0.5 em. W innych przypadkach ignorowane. Domyślne: 0

```csharp
public uint ColumnGap { get; set; }
```

### Przykłady

Przykład:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.ColumnGapRule = MathSpacingRules.Exactly;
matrix.ColumnGap = 20;
```

## IMathBox.NoBreak właściwość

Brak przerwy. Ta właściwość określa właściwość "unbreakable" na obiekcie pudełka. Gdy ustawiona na true, w pudełku nie mogą wystąpić podziały linii. Może to być istotne dla emulatorów operatorów, które składają się z więcej niż jednego operatora binarnego. Jeśli ten element nie zostanie określony, podziały mogą wystąpić wewnątrz pudełka. Domyślnie: true

```csharp
public bool NoBreak { get; set; }
```

Przykład:

IMathBox box = new MathematicalText("**********").ToBox();
box.NoBreak = false;

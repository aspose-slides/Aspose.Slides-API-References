## IMathBox.OperatorEmulator właściwość

Emulator operatora. Gdy true, pole i jego zawartość zachowują się jako pojedynczy operator i dziedziczą właściwości operatora. Oznacza to, na przykład, że znak może służyć jako punkt podziału linii i może być wyrównany do innych operatorów. Emulatory operatorów są często używane, gdy jeden lub więcej glifów łączy się, tworząc operator, taki jak '=='. Domyślna wartość: false

```csharp
public bool OperatorEmulator { get; set; }
```

### Przykłady

Przykład:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.OperatorEmulator = true;
```

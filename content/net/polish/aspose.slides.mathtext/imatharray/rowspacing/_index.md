## IMathArray.RowSpacing właściwość

Odległość między wierszami tablicy. Jest używana tylko wtedy, gdy RowSpacingRule jest ustawione na 3 Exactly, w którym przypadku jednostką miary są punkty lub Multiple, w którym przypadku jednostką miary są pół-linii. Domyślnie: 0

```csharp
public uint RowSpacing { get; set; }
```

### Przykłady

Przykład:

```csharp
[C#]
IMathArray mathArray = new MathArray(new MathematicalText("item1"));
mathArray.RowSpacingRule = MathRowSpacingRule.Exactly;
mathArray.RowSpacing = 10;
```

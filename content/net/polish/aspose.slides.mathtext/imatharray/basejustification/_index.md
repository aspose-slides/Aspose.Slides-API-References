## IMathArray.BaseJustification właściwość

Określa wyrównanie tablicy względem otaczającego tekstu. Tekst znajdujący się poza tablicą może być wyrównany do dołu, góry lub środka obiektu tablicy. Domyślna wartość: Center

```csharp
public MathVerticalAlignment BaseJustification { get; set; }
```

### Przykłady

Przykład:

```csharp
[C#]
IMathArray mathArray = new MathArray(new MathematicalText("item1"));
mathArray.BaseJustification = MathVerticalAlignment.Top;
```

## IMathMatrix.RowGap właściwość

Wartość pionowego odstępu między wierszami macierzy; jeśli RowGapRule ma wartość 3 ("Exactly"), jednostka jest interpretowana jako twipy (1/20th of a point). Jeśli RowGapRule ma wartość 4 ("Multiple"), jednostka jest interpretowana jako półwiersze. Domyślnie: 0

```csharp
public uint RowGap { get; set; }
```

### Przykłady

Example:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

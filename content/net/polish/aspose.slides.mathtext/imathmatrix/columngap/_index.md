## IMathMatrix.ColumnGap właściwość

Wartość odstępu poziomego między kolumnami macierzy; jeżeli ColumnGapRule jest ustawione na 3 ("Exactly"), jednostka jest interpretowana jako twips (1/20 punktu) jeżeli ColumnGapRule jest ustawione na 4 ("Multiple"), jednostka jest interpretowana jako liczba przyrostów po 0,5 em. W pozostałych przypadkach ignorowane. Domyślnie: 0

```csharp
public uint ColumnGap { get; set; }
```

### Przykłady

Przykład:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.ColumnGapRule = MathSpacingRules.Exactly;
matrix.ColumnGap = 20;
```

## IMathMatrix.MinColumnWidth właściwość

Minimalna szerokość kolumny w twipach (1/20 punktu). Odstęp między kolumnami (znany również jako „Column Gap” lub „Gap Width”) jest dodawany do MinColumnWidth, aby określić całkowity odstęp kolumn w macierzy (odległość pomiędzy tymi samymi krawędziami różnych kolumn). Domyślnie: 0.

```csharp
public uint MinColumnWidth { get; set; }
```

### Przykłady

Przykład:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.MinColumnWidth = 20;
```

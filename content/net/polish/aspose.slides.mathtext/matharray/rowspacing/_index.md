## MathArray.RowSpacing właściwość

Odstęp pomiędzy wierszami tablicy. Jest używany tylko wtedy, gdy RowSpacingRule jest ustawiony na 3 Exact, w którym to przypadku jednostką miary są punkty lub Multiple, w którym to przypadku jednostką miary są półwiersze. Domyślnie: 0

```csharp
public uint RowSpacing { get; set; }
```

### Przykłady

Przykład:

```csharp
[C#]
IMathArray mathArray = new MathArray(new MathematicalText("item1"));
mathArray.RowSpacingRule = MathRowSpacingRule.Exactly;
mathArray.RowSpacing = 10;
```

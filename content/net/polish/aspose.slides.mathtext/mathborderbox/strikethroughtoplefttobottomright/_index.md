## MathBorderBox.StrikethroughTopLeftToBottomRight właściwość

Przekreślenie od lewego górnego rogu do prawego dolnego (wartość domyślna to false). Określa ukryty lub widoczny stan przekreślenia po przekątnej od lewego górnego rogu do prawego dolnego ramki.

```csharp
public bool StrikethroughTopLeftToBottomRight { get; set; }
```

### Przykłady

Przykład:

```csharp
[C#]

borderBox.StrikethroughTopLeftToBottomRight = true;
```

## MathPhantom.ZeroDesc własność

Pobiera lub ustawia wartość określającą, czy opad (głębokość poniżej linii bazowej) elementu bazowego powinien być traktowany jako zero.

```csharp
public bool ZeroDesc { get; set; }
```

### Uwagi

Gdy `true`, fantom nie obniża linii bazowej otaczającej linii matematycznej. Odpowiada atrybutowi OMML `m:zeroDesc`.

## SlideUtil.FindAndReplaceText metoda

Znajduje i zamienia tekst w prezentacji przy użyciu podanego formatu

```csharp
public static void FindAndReplaceText(IPresentation presentation, bool withMasters, string find, 
    string replace, PortionFormat format = null)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| presentation | IPresentation | Skanowana prezentacja. |
| withMasters | Boolean | Określa, czy master-slajdy powinny być skanowane. |
| find | String | Wartość łańcucha do znalezienia. |
| replace | String | Wartość łańcucha do zamiany. |
| format | PortionFormat | Format do zastąpienia fragmentu tekstu. Jeśli null, zostanie użyty format pierwszego znaku znalezionego ciągu |

### Przykłady

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PortionFormat format = new PortionFormat
    {
        FontHeight = 24f,
        FontItalic = NullableBool.True,
        FillFormat =
        {
            FillType = FillType.Solid,
            SolidFillColor =
            {
                Color = Color.Red
            }
        }
    };
    Aspose.Slides.Util.SlideUtil.FindAndReplaceText(pres, true, "[this block] ", "my text ", format);
    pres.Save("replaced", SaveFormat.Pptx);
}
```

## WebDocument.Output właściwość

Zwraca kolekcję elementów wyjściowych dokumentu. Tylko do odczytu `Output`.

```csharp
public Output Output { get; }
```

### Przykłady

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    var options = new WebDocumentOptions
    {
        TemplateEngine = new RazorTemplateEngine(),
        OutputSaver = new FileOutputSaver(),
        EmbedImages = false
    };
    
    WebDocument document = new WebDocument(options);

    // dodaj "index.html" do plików wyjściowych, używając szablonu "index" do wygenerowania go oraz zmiennej pres jako modelu
    document.Output.Add("index.html", "index", pres);

    // ... skonfiguruj inne opcje dokumentu, a następnie zapisz dokument
    document.Save();
}
```

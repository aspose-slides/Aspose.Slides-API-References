## OpenAIWebClient(string, string, string, HttpClient) {#constructor_1}

Tworzy instancję klienta internetowego OpenAI, który używa zewnętrznie zarządzanego HttpClient. Dostarczony HttpClient nie jest usuwany przez tę instancję i pozostaje własnością wywołującego.

```csharp
public OpenAIWebClient(string model, string apiKey, string organizationId, HttpClient httpClient)
```

| Parametr | Typ | Opis |
| --- | --- | --- |
| model | String | Model językowy OpenAI. Możliwe wartości: - gpt-4o - gpt-4o-mini - o1 - o1-mini - o3 - o3-mini |
| apiKey | String | Klucz API OpenAI |
| organizationId | String | Identyfikator organizacji (opcjonalny) |
| httpClient | HttpClient | Zewnętrznie zarządzana instancja `HttpClient` |

### Wyjątki

| wyjątek | warunek |
| --- | --- |
| ArgumentException | Wartość klucza API nie może być null ani pusty |
| ArgumentException | Wartość modelu tekstowego nie może być null ani pusty |
| ArgumentNullException | HttpClient nie może być null |

## Ink.InkEffectImages właściwość

Zwraca kolekcję niestandardowych obrazów używanych do symulacji efektów wizualnych pędzli atramentu. Obrazy te są używane przy renderowaniu atramentu z określonymi [`InkEffectType`](../../inkeffecttype) wartościami, takimi jak Galaxy, Rainbow itp. Dostarczając własne obrazy, możesz kontrolować wygląd każdego efektu atramentu.

```csharp
public static IDictionary<InkEffectType, IImage> InkEffectImages { get; }
```

### Uwagi

Ta właściwość umożliwia zastąpienie domyślnych tekstur efektów atramentu własnymi, co jest szczególnie przydatne, gdy domyślne zasoby są ograniczone licencją lub niedostępne w czasie wykonywania. Każdy wpis w słowniku musi łączyć wartość [`InkEffectType`](../../inkeffecttype) z odpowiednim obiektem [`IImage`](../../../aspose.slides/iimage) (np. Bitmap lub interfejs obrazu Aspose).

### Przykłady

```csharp
[C#]
IImage image = Images.FromFile("image.png");
Ink.InkEffectImages.Add(InkEffectType.Galaxy, image);
```

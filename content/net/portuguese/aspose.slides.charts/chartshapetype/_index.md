## enumeração ChartShapeType

Representa uma forma de gráfico.

```csharp
public enum ChartShapeType
```

### Valores

| Nome | Valor | Descrição |
| --- | --- | --- |
| NotDefined | `-1` | A forma do gráfico não está definida. |
| Box | `0` | Especifica que o gráfico deve ser desenhado com formato de caixa. |
| Cone | `1` | Especifica que o gráfico deve ser desenhado como um cone, com a base do cone no chão e o ponto do cone no topo do marcador de dados. |
| ConeToMax | `2` | Especifica que o gráfico deve ser desenhado com cones truncados de forma que o ponto do cone seja o valor máximo dos dados. |
| Cylinder | `3` | Especifica que o gráfico deve ser desenhado como um cilindro. |
| Pyramid | `4` | Especifica que o gráfico deve ser desenhado como uma pirâmide retangular, com a base da pirâmide no chão e o ponto da pirâmide no topo do marcador de dados. |
| PyramidToMaximum | `5` | Especifica que o gráfico deve ser desenhado com cones truncados de forma que o ponto do cone seja o valor máximo dos dados. |

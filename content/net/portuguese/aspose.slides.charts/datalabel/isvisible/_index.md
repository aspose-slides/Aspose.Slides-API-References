## DataLabel.IsVisible propriedade

False significa que o rótulo de dados não está visível (e, portanto, todas as flags Show*- (ShowValue, ...) são falsas). Somente leitura Boolean.

```csharp
public bool IsVisible { get; }
```

### Observações

Se o rótulo de dados estiver visível, você pode ocultá-lo com o método Hide(). Mas se o rótulo de dados não estiver visível (IsVisible é false), você pode torná-lo visível configurando as flags Show*- (ShowValue, ...) para o estado true.

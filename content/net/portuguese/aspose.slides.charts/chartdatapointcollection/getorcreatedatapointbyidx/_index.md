## ChartDataPointCollection.GetOrCreateDataPointByIdx método

Se a coleção já contém um ponto de dados com o índice *index*, então retorna esse ponto de dados. Se a coleção não contém um ponto de dados com o índice *index*==N (quando o número de pontos de dados nesta coleção é menor ou igual a N), então adiciona os pontos de dados deficientes e retorna o último (que possui o índice solicitado). Por exemplo, os índices da coleção são {0, 1, 2} e o índice solicitado é 5. Então o método adiciona os pontos de dados deficientes: {0, 1, 2, 3, 4, 5}. E retorna o ponto de dados com índice 5.

```csharp
public IChartDataPoint GetOrCreateDataPointByIdx(uint index)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| index | UInt32 | Índice. |

### Valor de retorno

Retorna o ponto de dados com o índice solicitado.

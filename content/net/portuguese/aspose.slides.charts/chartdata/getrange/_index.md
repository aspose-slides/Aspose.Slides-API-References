## ChartData.GetRange method

Obtém o intervalo de dados do gráfico.

```csharp
public string GetRange()
```

### Valor de Retorno

Fórmula do intervalo de dados das células. Ex.: "Sheet1!$A$1:$C$4"

### Exceções

| exceção | condição |
| --- | --- |
| InvalidOperationException | O gráfico não usa a pasta de trabalho como fonte de dados |

### Exemplos

Exemplo C#

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.PercentsStackedBar, 0, 0, 100, 100);
    string result = (chart.ChartData as ChartData).GetRange();
}
```

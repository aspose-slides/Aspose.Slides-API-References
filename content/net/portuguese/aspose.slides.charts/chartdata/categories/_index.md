## ChartData.Categories propriedade

Obtém as categorias principais (ou tanto as categorias principais quanto as secundárias se a propriedade [`UseSecondaryCategories`](../usesecondarycategories) for falsa). Somente leitura [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection Categories { get; }
```

### Observações

Se a propriedade [`UseSecondaryCategories`](../usesecondarycategories) for falsa, então a propriedade [`SecondaryCategories`](../secondarycategories) retornará nulo e os dados nesta propriedade `Categories` serão usados tanto para as séries primárias quanto para as secundárias. Se a propriedade [`UseSecondaryCategories`](../usesecondarycategories) for verdadeira, então os dados na propriedade [`SecondaryCategories`](../secondarycategories) serão usados para as séries secundárias e os dados nesta propriedade `Categories` serão usados para as séries primárias.

### Exemplos

Exemplo. Quais categorias estão relacionadas às séries - ChartData.Categories ou ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // as categorias relacionadas são series.Chart.ChartData.SecondaryCategories
}
else
{
    // as categorias relacionadas são series.Chart.ChartData.Categories
}
```

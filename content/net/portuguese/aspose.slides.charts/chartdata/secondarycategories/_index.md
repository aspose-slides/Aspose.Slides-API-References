## ChartData.SecondaryCategories propriedade

Obtém as categorias secundárias se [`UseSecondaryCategories`](../usesecondarycategories) propriedade for verdadeira. Somente leitura [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### Observações

Se [`UseSecondaryCategories`](../usesecondarycategories) propriedade for falsa, então esta `SecondaryCategories` propriedade retornará nulo e os dados na [`Categories`](../categories) propriedade são usados tanto para séries primárias quanto secundárias. Se [`UseSecondaryCategories`](../usesecondarycategories) propriedade for verdadeira, então os dados nesta `SecondaryCategories` propriedade são usados para séries secundárias e os dados na [`Categories`](../categories) propriedade são usados para séries primárias.

### Exemplos

Exemplo. Quais categorias estão relacionadas às séries - ChartData.Categories ou ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // as categorias relacionadas são series.Chart.ChartData.SecondaryCategories
}
else
{
    // as categorias relacionadas são series.Chart.ChartData.Categories
}
```

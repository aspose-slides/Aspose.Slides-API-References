## ChartData.UseSecondaryCategories propriedade

Se for falso, então [`SecondaryCategories`](../secondarycategories) property retorna null e os dados em [`Categories`](../categories) property são usados tanto para séries primárias quanto secundárias. Se for verdadeiro, então os dados em [`SecondaryCategories`](../secondarycategories) property são usados para séries secundárias e os dados em [`Categories`](../categories) property são usados para séries primárias. Leitura/Gravação Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

### Exemplos

Exemplo. Quais categorias estão relacionadas às séries - ChartData.Categories ou ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // categorias relacionadas são series.Chart.ChartData.SecondaryCategories
}
else
{
    // categorias relacionadas são series.Chart.ChartData.Categories
}
```

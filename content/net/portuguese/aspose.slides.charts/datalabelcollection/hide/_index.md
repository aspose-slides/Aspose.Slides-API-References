## DataLabelCollection.Hide método

Torne o rótulo de dados oculto por padrão definindo todos os flags Show* (ShowValue, ...) da propriedade DefaultDataLabelFormat para o estado falso. IsVisible será false após isso.

```csharp
public void Hide()
```

### Observações

Se o rótulo de dados não estiver visível por padrão (IsVisible for false) você pode tornar o rótulo de dados "visível por padrão" configurando os flags Show* (ShowValue, ...) da propriedade DefaultDataLabelFormat para o estado true.

## IChartData.SecondaryCategories propriedade

Obtém as categorias secundárias se a propriedade [`UseSecondaryCategories`](../usesecondarycategories) for verdadeira. Somente leitura [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### Observações

Se a propriedade [`UseSecondaryCategories`](../usesecondarycategories) for falsa, então esta propriedade `SecondaryCategories` retornará null e os dados na propriedade [`Categories`](../categories) serão usados tanto para a série primária quanto para a série secundária. Se a propriedade [`UseSecondaryCategories`](../usesecondarycategories) for verdadeira, então os dados nesta propriedade `SecondaryCategories` serão usados para a série secundária e os dados na propriedade [`Categories`](../categories) serão usados para a série primária.

### Exemplos

Exemplo. Quais categorias estão relacionadas a séries – ChartData.Categories ou ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // as categorias relacionadas são series.Chart.ChartData.SecondaryCategories
}
else
{
    // as categorias relacionadas são series.Chart.ChartData.Categories
}
```

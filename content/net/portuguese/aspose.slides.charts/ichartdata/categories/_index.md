## IChartData.Categories propriedade

Obtém as categorias principais (ou tanto as categorias principais quanto as secundárias se [`UseSecondaryCategories`](../usesecondarycategories) propriedade for false). Somente leitura [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection Categories { get; }
```

### Observações

Se [`UseSecondaryCategories`](../usesecondarycategories) propriedade for false então [`SecondaryCategories`](../secondarycategories) propriedade retorna null e os dados nesta `Categories` propriedade são usados tanto para as séries principais quanto para as secundárias. Se [`UseSecondaryCategories`](../usesecondarycategories) propriedade for true então os dados em [`SecondaryCategories`](../secondarycategories) propriedade são usados para séries secundárias e os dados nesta `Categories` propriedade são usados para séries principais.

### Exemplos

Exemplo. Quais categorias estão relacionadas às séries - ChartData.Categories ou ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // as categorias relacionadas são series.Chart.ChartData.SecondaryCategories
}
else
{
    // as categorias relacionadas são series.Chart.ChartData.Categories
}
```

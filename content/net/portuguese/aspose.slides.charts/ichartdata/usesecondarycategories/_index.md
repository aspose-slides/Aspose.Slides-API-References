## IChartData.UseSecondaryCategories propriedade

Se false então [`SecondaryCategories`](../secondarycategories) propriedade retorna null e os dados na [`Categories`](../categories) propriedade são usados tanto para séries primárias quanto secundárias. Se true então os dados na [`SecondaryCategories`](../secondarycategories) propriedade são usados para séries secundárias e os dados na [`Categories`](../categories) propriedade são usados para séries primárias. Leitura/gravação Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

### Exemplos

Exemplo. Quais categorias estão relacionadas às séries - ChartData.Categories ou ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // as categorias relacionadas são series.Chart.ChartData.SecondaryCategories
}
else
{
    // as categorias relacionadas são series.Chart.ChartData.Categories
}
```

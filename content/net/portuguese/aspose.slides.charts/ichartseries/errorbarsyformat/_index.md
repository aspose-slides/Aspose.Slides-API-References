## IChartSeries.ErrorBarsYFormat propriedade

Representa ErrorBars da série com direção Y. ErrorBars com direção Y estão disponíveis para séries do tipo area, bar, line, scatter e bubble. Para quaisquer outros tipos de gráfico, esta propriedade retorna null (incluindo gráficos 3D). No caso de valores personalizados, use a coleção DataPoints para especificar o valor (com a propriedade [`ErrorBarsCustomValues`](../../ichartdatapoint/errorbarscustomvalues)).

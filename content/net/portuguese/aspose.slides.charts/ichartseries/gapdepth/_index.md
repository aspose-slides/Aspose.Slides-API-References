## IChartSeries.GapDepth propriedade

Retorna ou define a distância, como porcentagem da largura do marcador, entre as séries de dados em um gráfico 3D. Esta é a propriedade não apenas desta série, mas de todas as séries do grupo de séries pai – é a projeção da propriedade do grupo apropriado. Portanto, esta propriedade é somente leitura. Use a propriedade ParentSeriesGroup para acessar o grupo de séries pai. Use a propriedade de leitura/gravação ParentSeriesGroup.GapDepth para alterar o valor. Somente leitura Int32.

```csharp
public int GapDepth { get; }
```

### Observações

Esta é a projeção da propriedade ParentSeriesGroup.GapDepth.

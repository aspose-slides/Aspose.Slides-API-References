## IChartSeries.GapWidth propriedade

Especifica o espaço entre agrupamentos de barras ou colunas, como uma porcentagem da largura da barra ou coluna. Esta é a propriedade não apenas desta série, mas de todas as séries do grupo de séries pai – é a projeção da propriedade do grupo apropriado. Portanto, esta propriedade é somente leitura. Use a propriedade ParentSeriesGroup para acessar o grupo de séries pai. Use a propriedade ParentSeriesGroup.GapWidth de leitura/gravação para mudar o valor. Somente leitura Int32.

```csharp
public int GapWidth { get; }
```

### Observações

Esta é a projeção da propriedade ParentSeriesGroup.GapWidth.

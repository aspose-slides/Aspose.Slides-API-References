## IChartSeries.BubbleSizeScale propriedade

Especifica o fator de escala para o gráfico de bolhas (pode ser entre 0 e 300 por cento do tamanho padrão). Esta é a propriedade não apenas desta série, mas de todas as séries do grupo de séries pai – é a projeção da propriedade do grupo apropriado. Portanto, esta propriedade é somente leitura. Use a propriedade ParentSeriesGroup para acessar o grupo de séries pai. Use a propriedade ParentSeriesGroup.BubbleSizeScale de leitura/gravação para alterar o valor.

```csharp
public int BubbleSizeScale { get; }
```

### Observações

Esta é a projeção da propriedade ParentSeriesGroup.BubbleSizeScale.

## IChartSeries.PieSplitBy propriedade

Especifica como determinar quais pontos de dados estão no segundo setor ou barra em um gráfico de pizza-de-pizza ou barra-de-pizza. Esta é a propriedade não apenas desta série, mas de todas as séries do grupo de séries pai – isto é a projeção da propriedade de grupo apropriada. Portanto, esta propriedade é somente leitura. Use a propriedade ParentSeriesGroup para acesso ao grupo de séries pai. Use a propriedade ParentSeriesGroup.PieSplitBy de leitura/gravação para alterar o valor. Somente leitura [`PieSplitType`](../../piesplittype).

```csharp
public PieSplitType PieSplitBy { get; }
```

### Observações

1) Esta é a projeção da propriedade ParentSeriesGroup.PieSplitBy.  
2) Se o valor da propriedade for PieSplitType.Custom, então você pode definir informações de divisão personalizadas com a propriedade ParentSeriesGroup.PieSplitCustomPoints.

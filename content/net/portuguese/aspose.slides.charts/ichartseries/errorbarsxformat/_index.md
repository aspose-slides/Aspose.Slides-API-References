## IChartSeries.ErrorBarsXFormat propriedade

Representa ErrorBars de séries com direção X.  ErrorBars com direção X são disponíveis para séries do tipo area, bar, scatter e bubble. Para qualquer outro tipo de gráfico, esta propriedade retorna nulo (incluindo gráficos 3D). No caso de valores personalizados, use a coleção DataPoints para especificar o valor (com a propriedade [`ErrorBarsCustomValues`](../../ichartdatapoint/errorbarscustomvalues)).

## IChartSeries.IsColorVaried propriedade

Especifica que cada marcador de dados na série tem uma cor diferente. Esta é a propriedade não apenas desta série, mas de todas as séries do grupo de séries pai – esta é a projeção da propriedade de grupo apropriada. Portanto, esta propriedade é somente leitura. Use a propriedade ParentSeriesGroup para acessar o grupo de séries pai. Use a propriedade ParentSeriesGroup.IsColorVaried de leitura/gravação para alterar o valor. Boolean somente leitura.

```csharp
public bool IsColorVaried { get; }
```

### Observações

Esta é a projeção da propriedade ParentSeriesGroup.IsColorVaried.

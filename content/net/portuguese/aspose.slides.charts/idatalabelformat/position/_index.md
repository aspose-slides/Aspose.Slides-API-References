## IDataLabelFormat.Position propriedade

Representa a posição do rótulo de dados. Leitura/Gravação [`LegendDataLabelPosition`](../../legenddatalabelposition).

```csharp
public LegendDataLabelPosition Position { get; set; }
```

### Observações

Se o pai deste objeto DataLabelFormat for uma coleção DataLabelCollection de rótulos de dados, então esta propriedade obtém ou define o valor padrão da propriedade Position para os novos rótulos de dados na coleção DataLabelCollection. Representa a posição para os objetos DataLabel. Definir esta propriedade com um valor também define esse valor na propriedade Position para todos os rótulos de dados na coleção DataLabelCollection (i.e. "DataLabels.DefaultDataLabelFormat.Position = val;" causa que todos DataLabels[i].Position sejam iguais a val).

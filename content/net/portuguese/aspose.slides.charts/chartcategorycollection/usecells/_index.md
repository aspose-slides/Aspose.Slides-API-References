## ChartCategoryCollection.UseCells propriedade

Se verdadeiro, a planilha é usada para armazenar categorias (este caso suporta categorias de vários níveis). Se falso, a planilha NÃO é usada para armazenar valores (e este caso não suporta categorias de vários níveis). Leitura/gravação Boolean.

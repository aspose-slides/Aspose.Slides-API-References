## Enumeração ErrorBarValueType

Representa o tipo de valor da barra de erro

```csharp
public enum ErrorBarValueType
```

### Valores

| Nome | Valor | Descrição |
| --- | --- | --- |
| Custom | `0` | Especifica que o comprimento das barras de erro deve ser determinado pelos elementos Plus e Minus. |
| Fixed | `1` | Especifica que o comprimento das barras de erro será o valor fixo determinado por Error Bar Value. |

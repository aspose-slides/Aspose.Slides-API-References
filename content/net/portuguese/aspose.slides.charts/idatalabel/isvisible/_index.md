## IDataLabel.IsVisible propriedade

False means that data label is not visible (and so all Show*-flags (ShowValue, ...) are false). Somente leitura Boolean.

```csharp
public bool IsVisible { get; }
```

### Observações

Se o rótulo de dados estiver visível, você pode torná-lo oculto com o método Hide(). Mas se o rótulo de dados não estiver visível (IsVisible é false), você pode torná-lo visível definindo as bandeiras Show*-flags (ShowValue, ...) para true.

## IDataLabel.Hide método

Torne o rótulo de dados oculto definindo todas as flags Show*- (ShowValue, ...) para o estado false. IsVisible será false após isso.

```csharp
public void Hide()
```

### Observações

Se o rótulo de dados não estiver visível (IsVisible é false) você pode tornar o rótulo de dados visível definindo as flags Show*- (ShowValue, ...) para o estado true.

## ChartSeriesGroup.Overlap propriedade

Especifica o quanto barras e colunas devem se sobrepor em gráficos 2-D, como uma porcentagem (de -100% a 100%). - -100%: Espaçamento máximo (as barras ficam completamente separadas). - 0%: As barras são colocadas lado a lado sem sobreposição nem espaçamento. - 100%: Sobreposição máxima (as barras se sobrepõem completamente). Esta propriedade é leitura/gravação SByte.

```csharp
public sbyte Overlap { get; set; }
```

### Exceções

| exceção | condição |
| --- | --- |
| ArgumentOutOfRangeException | Lançada quando o valor é definido fora do intervalo válido de -100 a 100. |

### Exemplos

O exemplo a seguir demonstra como definir a sobreposição para um grupo de séries de gráfico e renderizar o gráfico resultante em um formulário:

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // Definir sobreposição para 55%

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

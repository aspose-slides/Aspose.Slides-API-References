## IDataLabelCollection.IsVisible propriedade

False significa que o rótulo de dados não está visível por padrão (e, portanto, todas as Show*-flags (ShowValue, ...) da propriedade DefaultDataLabelFormat são false). Somente leitura Boolean.

```csharp
public bool IsVisible { get; }
```

### Observações

Se o rótulo de dados estiver visível por padrão, você pode torná-lo oculto por padrão com o método Hide(). Mas se o rótulo de dados não estiver visível por padrão (IsVisible is false) você pode tornar o rótulo de dados "visível por padrão" definindo as Show*-flags (ShowValue, ...) da propriedade DefaultDataLabelFormat para o estado true.

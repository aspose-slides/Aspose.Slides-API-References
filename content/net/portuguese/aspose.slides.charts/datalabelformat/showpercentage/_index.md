## DataLabelFormat.ShowPercentage propriedade

Representa o comportamento de exibição do valor percentual do rótulo de dados de um gráfico especificado. True exibe o valor percentual. False oculta. Leitura/gravação Boolean.

```csharp
public bool ShowPercentage { get; set; }
```

### Observações

Se o pai deste objeto DataLabelFormat for uma coleção DataLabelCollection de rótulos de dados, então esta propriedade obtém ou define o valor padrão da propriedade ShowPercentage para os novos rótulos de dados na coleção DataLabelCollection. Definir esta propriedade com um valor também define esse valor na propriedade ShowPercentage para todos os rótulos de dados na coleção DataLabelCollection (por exemplo, "DataLabels.DefaultDataLabelFormat.ShowPercentage = val;" faz com que todos DataLabels[i].ShowPercentage sejam iguais a val).

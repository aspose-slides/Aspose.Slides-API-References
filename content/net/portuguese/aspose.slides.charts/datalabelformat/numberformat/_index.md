## DataLabelFormat.NumberFormat propriedade

Representa a string de formato para o objeto DataLabels. Leitura/gravação String.

```csharp
public string NumberFormat { get; set; }
```

### Observações

Se o pai deste objeto DataLabelFormat for uma coleção DataLabelCollection de rótulos de dados, então esta propriedade obtém ou define o valor padrão da propriedade NumberFormat para os novos rótulos de dados na coleção DataLabelCollection. Quando esta propriedade é definida com um valor, esse valor também é definido para a propriedade NumberFormat de todos os rótulos de dados na coleção DataLabelCollection (por exemplo, "DataLabels.DefaultDataLabelFormat.NumberFormat = val;" faz com que todos DataLabels[i].NumberFormat sejam iguais a val).

### Exemplos

```csharp
[C#]
series.Labels.DefaultDataLabelFormat.ShowValue = true;
series.Labels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = false;
series.Labels.DefaultDataLabelFormat.NumberFormat = "0.0%";
```

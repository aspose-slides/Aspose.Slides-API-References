## ChartSeries.FirstSliceAngle propriedade

Especifica o ângulo da primeira fatia de gráfico de pizza ou rosquinha, em graus (no sentido horário a partir de cima, de 0 a 360 graus). Esta é a propriedade não apenas desta série, mas de todas as séries do grupo de séries pai – é a projeção da propriedade correspondente do grupo. Por isso, esta propriedade é somente leitura. Use a propriedade ParentSeriesGroup para acessar o grupo de séries pai. Use a propriedade de leitura/gravação ParentSeriesGroup.FirstSliceAngle para alterar o valor. Somente leitura UInt16.

```csharp
public ushort FirstSliceAngle { get; }
```

### Observações

Esta é a projeção da propriedade ParentSeriesGroup.FirstSliceAngle.

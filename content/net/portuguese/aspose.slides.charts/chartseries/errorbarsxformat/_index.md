## ChartSeries.ErrorBarsXFormat propriedade

Representa ErrorBars de séries com direção X. ErrorBars com direção X estão disponíveis para séries do tipo area, bar, scatter e bubble. Para quaisquer outros tipos de gráfico, esta propriedade retorna null (incluindo gráficos 3D). Caso haja valores personalizados, use a coleção DataPoints para especificar o valor (com a propriedade [`ErrorBarsCustomValues`](../../ichartdatapoint/errorbarscustomvalues)).

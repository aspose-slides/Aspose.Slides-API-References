## ChartSeries.PieSplitBy propriedade

Especifica como determinar quais pontos de dados estão na segunda pizza ou barra em um gráfico pie-of-pie ou bar-of-pie. Esta é a propriedade não apenas desta série, mas de todas as séries do grupo de séries pai – isso é a projeção da propriedade de grupo apropriada. Portanto, esta propriedade é somente leitura. Use a propriedade ParentSeriesGroup para acessar o grupo de séries pai. Use a propriedade read/write ParentSeriesGroup.PieSplitBy para alterar o valor. Somente leitura [`PieSplitType`](../../piesplittype).

```csharp
public PieSplitType PieSplitBy { get; }
```

### Observações

1) Esta é a projeção da propriedade ParentSeriesGroup.PieSplitBy. 2) Se o valor da propriedade for PieSplitType.Custom, então você pode definir informações de divisão personalizada com a propriedade ParentSeriesGroup.PieSplitCustomPoints.

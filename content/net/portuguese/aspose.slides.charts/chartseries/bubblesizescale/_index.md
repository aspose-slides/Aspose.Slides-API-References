## ChartSeries.BubbleSizeScale propriedade

Especifica o fator de escala para o gráfico de bolhas (pode estar entre 0 e 300 por cento do tamanho padrão). Esta é a propriedade não apenas desta série, mas de todas as séries do grupo de séries pai – é a projeção da propriedade de grupo apropriada. Portanto, esta propriedade é somente leitura. Use a propriedade ParentSeriesGroup para acessar o grupo de séries pai. Use a propriedade ParentSeriesGroup.BubbleSizeScale leitura/gravação para alterar o valor.

```csharp
public int BubbleSizeScale { get; }
```

### Observações

Esta é a projeção da propriedade ParentSeriesGroup.BubbleSizeScale.

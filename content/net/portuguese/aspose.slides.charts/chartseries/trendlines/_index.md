## ChartSeries.TrendLines propriedade

Coleção de linhas de tendência da série. Somente leitura [`ITrendlineCollection`](../../itrendlinecollection).

```csharp
public ITrendlineCollection TrendLines { get; }
```

### Observações

TrendLines estão disponíveis (não nulas) para séries de dados em gráficos de área 2-D desenfileirados, barra, coluna, linha, estoque, xy (dispersão) e bolha. Uma linha de tendência não está disponível para séries de dados em qualquer tipo de gráfico que seja empilhado ou 3-D. Linhas de tendência também não estão disponíveis para gráficos de radar, pizza, superfície ou anel.

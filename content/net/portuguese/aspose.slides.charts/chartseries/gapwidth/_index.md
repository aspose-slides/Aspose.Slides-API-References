## ChartSeries.GapWidth propriedade

Especifica o espaço entre os agrupamentos de barras ou colunas, como uma porcentagem da largura da barra ou coluna. Esta é a propriedade não apenas desta série, mas de todas as séries do grupo de séries pai – isso é a projeção da propriedade de grupo apropriada. Portanto, esta propriedade é somente leitura. Use a propriedade ParentSeriesGroup para acessar o grupo de séries pai. Use a propriedade read/write ParentSeriesGroup.GapWidth para alterar o valor. Somente leitura Int32.

```csharp
public int GapWidth { get; }
```

### Observações

Esta é a projeção da propriedade ParentSeriesGroup.GapWidth.

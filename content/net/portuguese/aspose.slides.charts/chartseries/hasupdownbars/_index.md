## ChartSeries.HasUpDownBars propriedade

Determina se um gráfico de linhas ou de ações possui barras de alta/baixa. Esta é a propriedade não apenas desta série, mas de todas as séries do grupo de séries pai – isso é a projeção da propriedade de grupo apropriada. Portanto, esta propriedade é somente leitura. Use a propriedade ParentSeriesGroup para acessar o grupo de séries pai. Use ParentSeriesGroup.UpDownBars.HasUpDownBars propriedade leitura/gravação para alterar o valor. Use a propriedade ParentSeriesGroup.UpDownBars para formatar as barras de alta/baixa. Boolean somente leitura.

```csharp
public bool HasUpDownBars { get; }
```

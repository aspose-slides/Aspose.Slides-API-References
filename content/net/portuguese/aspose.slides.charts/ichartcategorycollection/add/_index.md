## Add(object) {#add_1}

Cria um novo [`IChartCategory`](../../ichartcategory) a partir do valor e o adiciona à coleção.

```csharp
public IChartCategory Add(object value)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| value | Object | O valor. |

### Valor de Retorno

[`IChartCategory`](../../ichartcategory) adicionado.

### Exceções

| Exceção | Condição |
| --- | --- |
| InvalidOperationException | se o limite for excedido |

### Observações

Este método adiciona uma planilha com o nome AUTO_DATA e insere todos os valores nela. Se você usar [`IChartDataWorkbook`](../../ichartdataworkbook) para adicionar ou editar valores de células, certifique-se de não usar esta planilha. O número máximo de valores adicionados usando este método não deve exceder 16711680.

## IChartCategoryCollection.UseCells propriedade

Se for verdadeiro, a planilha é usada para armazenar categorias (este caso suporta categorias de vários níveis). Se for falso, a planilha NÃO é usada para armazenar valores (e este caso não suporta categorias de vários níveis). Leitura/escrita Boolean.

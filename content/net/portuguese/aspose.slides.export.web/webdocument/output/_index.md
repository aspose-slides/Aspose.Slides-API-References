## WebDocument.Output propriedade

Retorna a coleção de elementos de saída do documento. Somente leitura `Output`.

```csharp
public Output Output { get; }
```

### Exemplos

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    var options = new WebDocumentOptions
    {
        TemplateEngine = new RazorTemplateEngine(),
        OutputSaver = new FileOutputSaver(),
        EmbedImages = false
    };
    
    WebDocument document = new WebDocument(options);

    // adiciona "index.html" aos arquivos de saída, usando o modelo "index" para gerá-lo e a variável pres como modelo
    document.Output.Add("index.html", "index", pres);

    // ... configure outras opções do documento e então salve o documento
    document.Save();
}
```

## ToTiff(Presentation, string) {#totiff}

Converte a apresentação de entrada em um conjunto de imagens no formato TIFF. Se o nome do arquivo de saída for fornecido como "myPath/myFilename.tiff", o resultado será salvo como um conjunto de arquivos "myPath/myFilename_N.tiff", onde N é o número do slide.

```csharp
public static void ToTiff(Presentation pres, string outputFileName)
```

## ToTiff(Presentation, string, ITiffOptions, bool) {#totiff_1}

Converte a apresentação de entrada para o formato TIFF com opções personalizadas. Se o nome do arquivo de saída for fornecido como "myPath/myFilename.tiff" e *multipage* for `false`, o resultado será salvo como um conjunto de arquivos "myPath/myFilename_N.tiff", onde N é o número do slide. Caso contrário, se *multipage* for `true`, o resultado será um documento TIFF multipáginas "myPath/myFilename.tiff".

```csharp
public static void ToTiff(Presentation pres, string outputFileName, ITiffOptions options, 
    bool multipage)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| pres | Presentation | A apresentação de entrada. |
| outputFileName | String | O nome do arquivo de saída. |
| options | ITiffOptions | As opções de salvamento TIFF. |
| multipage | Boolean | Especifica se o documento TIFF gerado deve ser multipáginas. |

### Exceções

| Exceção | Condição |
| --- | --- |
| ArgumentException |  |

### Exemplos

```csharp
ITiffOptions options = new TiffOptions()
{
    CompressionType = TiffCompressionTypes.CCITT3,
    SlidesLayoutOptions = new NotesCommentsLayoutingOptions()
    {
        NotesPosition = NotesPositions.BottomTruncated
    }
};

using (var pres = new Presentation("pres.pptx"))
   Convert.ToTiff(pres, "pres.tiff", options, false);
```

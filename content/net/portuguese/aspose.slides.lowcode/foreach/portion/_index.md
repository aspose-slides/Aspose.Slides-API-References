## Portion(Presentation, ForEachPortionCallback) {#portion_1}

Iterar cada `Portion` no [`Presentation`](../../../aspose.slides/presentation). As Portion serão iteradas em todos os tipos de slides - [`Slide`](../slide), [`MasterSlide`](../masterslide) e [`LayoutSlide`](../layoutslide)

```csharp
public static void Portion(Presentation pres, ForEachPortionCallback forEachPortion)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| pres | Presentation | Presentation para iterar as portions |
| forEachPortion | ForEachPortionCallback | callback que será invocado para cada portion |

### Exemplos

```csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    ForEach.Portion(pres, (portion, para, slide, index) =>
    {
        System.Console.WriteLine($"{portion.Text}, index: {index}");
    });
}
```

## Portion(Presentation, bool, ForEachPortionCallback) {#portion}

Iterar cada [`Portion`](../portion) no [`Presentation`](../../../aspose.slides/presentation). As Portion serão iteradas em todos os tipos de slides - [`Slide`](../slide), [`MasterSlide`](../masterslide), [`LayoutSlide`](../layoutslide) e [`NotesSlide`](../../../aspose.slides/notesslide)

```csharp
public static void Portion(Presentation pres, bool includeNotes,

```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| pres | Presentation | Presentation para iterar as portions |
| includeNotes | Boolean | Indicador que indica se NotesSlides devem ser incluídos no processamento. |
| forEachPortion | ForEachPortionCallback | callback que será invocado para cada portion |

### Exemplos

```csharp

{
    ForEach.Portion(pres, true, (portion, para, slide, index) =>
    {
        System.Console.WriteLine($"{portion.Text}, index: {index}");
    });
} 
```

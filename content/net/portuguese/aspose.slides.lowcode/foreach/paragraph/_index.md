## Paragraph(Presentation, ForEachParagraphCallback) {#paragraph_1}

Iterar cada `Paragraph` no [`Presentation`](../../../aspose.slides/presentation). As formas serão iteradas em todos os tipos de slides - [`Slide`](../slide), [`MasterSlide`](../masterslide) e [`LayoutSlide`](../layoutslide)

```csharp
public static void Paragraph(Presentation pres, ForEachParagraphCallback forEachParagraph)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| pres | Presentation | Apresentação para iterar parágrafos |
| forEachParagraph | ForEachParagraphCallback | Callback que será invocado para cada parágrafo |

### Exemplos

```csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    ForEach.Paragraph(pres, (para, slide, index) =>
    {
        System.Console.WriteLine($"{para.Text}, index: {index}");
    });
}        
```

```csharp
public static void Paragraph(Presentation pres, bool includeNotes,

```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| pres | Presentation | Apresentação para iterar parágrafos |
| includeNotes | Boolean | Indicador que indica se NotesSlides deve ser incluído no processamento. |
| forEachParagraph | ForEachParagraphCallback | Callback que será invocado para cada parágrafo |

### Exemplos

```csharp

    ForEach.Paragraph(pres, true, (para, slide, index) =>
    {
        System.Console.WriteLine($"{para.Text}, index: {index}");
    });
}        
```

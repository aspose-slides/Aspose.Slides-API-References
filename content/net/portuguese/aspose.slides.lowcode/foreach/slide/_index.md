## Método ForEach.Slide

Itere cada `Slide` no [`Presentation`](../../../aspose.slides/presentation).

```csharp
public static void Slide(Presentation pres, ForEachSlideCallback forEachSlide)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| pres | Presentation | Apresentação para iterar slides |
| forEachSlide | ForEachSlideCallback | Callback que será invocado para cada slide |

### Exemplos

```csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    ForEach.Slide(pres, (slide, index) =>
    {
        slide.Name = $"Slide #{index}";
    });
} 
```

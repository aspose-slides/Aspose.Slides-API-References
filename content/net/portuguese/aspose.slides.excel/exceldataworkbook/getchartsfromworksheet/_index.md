## ExcelDataWorkbook.GetChartsFromWorksheet método

Recupera um dicionário contendo os índices e nomes de todos os gráficos na planilha especificada de uma pasta de trabalho do Excel.

```csharp
public IDictionary<int, string> GetChartsFromWorksheet(string worksheetName)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| worksheetName | String | O nome da planilha onde procurar gráficos. |

### Valor de Retorno

Um dicionário onde a chave é o índice do gráfico e o valor é o nome do gráfico.

### Exemplos

Exemplo:

```csharp
[C#]
ExcelDataWorkbook wb = new ExcelDataWorkbook(testFile);
var sheetCharts = wb.GetChartsFromWorksheet("worksheetName");
foreach (var chart in sheetCharts)
    Console.WriteLine(chart.Key + " : " + chart.Value);
```

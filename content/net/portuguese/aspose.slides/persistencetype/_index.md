## PersistenceType enumeração

Especifica o método usado para armazenar propriedades do controle ActiveX.

```csharp
public enum PersistenceType
```

### Valores

| Nome | Valor | Descrição |
| --- | --- | --- |
| NotDefined | `-1` | ID de persistência não especificado. |
| PersistPropertyBag | `0` | Especifica que o controle ActiveX é persistido usando persistência baseada em property-bag. A persistência baseada em property-bag armazena um controle ActiveX por meio de uma coleção de pares nome e valor que especificam os dados persistidos pelo controle ActiveX. |
| PersistStream | `1` | Especifica que o controle ActiveX é persistido usando persistência baseada em stream que não suporta a inicialização do controle ActiveX para um estado padrão. |
| PersistStreamInit | `2` | Especifica que o controle ActiveX é persistido usando persistência baseada em stream que suporta a inicialização do controle ActiveX para um estado padrão. |
| PersistStorage | `3` | Especifica que o controle ActiveX é persistido usando persistência baseada em storage. |

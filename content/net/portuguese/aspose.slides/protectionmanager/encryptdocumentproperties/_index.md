## ProtectionManager.EncryptDocumentProperties propriedade

Essa propriedade faz sentido se a apresentação estiver protegida por senha. Se true então as propriedades do documento são criptografadas no arquivo da apresentação. Se false então as propriedades do documento são públicas enquanto a apresentação está criptografada. Leitura/Gravação Booleano.

public bool EncryptDocumentProperties { get; set; }

## Propriedades

| --- | --- |
| [EncryptDocumentProperties](../../aspose.slides/protectionmanager/encryptdocumentproperties) { get; set; } | Esta propriedade faz sentido se a apresentação estiver protegida por senha. Se for true, então as propriedades do documento são criptografadas no arquivo de apresentação. Se for false, então as propriedades do documento são públicas enquanto a apresentação está criptografada. Leitura/gravação Boolean. |
| [EncryptionPassword](../../aspose.slides/protectionmanager/encryptionpassword) { get; } | Obtém a senha usada para criptografia da apresentação. Somente leitura String. |
| [IsEncrypted](../../aspose.slides/protectionmanager/isencrypted) { get; } | Obtém um valor que indica se esta instância está criptografada. Somente leitura Boolean. |
| [IsOnlyDocumentPropertiesLoaded](../../aspose.slides/protectionmanager/isonlydocumentpropertiesloaded) { get; } | Esta propriedade faz sentido se o arquivo de apresentação estiver protegido por senha e as propriedades do documento deste arquivo forem públicas. Valor true significa que apenas as propriedades do documento são carregadas de um arquivo de apresentação criptografado sem uso de senha. Valor false significa que toda a apresentação criptografada é carregada com o uso da senha correta, não apenas as propriedades do documento são carregadas. Se a apresentação não estiver criptografada, o valor da propriedade será sempre false. Se as propriedades do documento de um arquivo criptografado não forem públicas, o valor da propriedade será sempre false. Se Presentation.EncryptDocumentProperties for true, então o valor da propriedade IsOnlyDocumentPropertiesLoaded será sempre false. Somente leitura Boolean. |
| [IsWriteProtected](../../aspose.slides/protectionmanager/iswriteprotected) { get; } | Obtém um valor que indica se esta apresentação está protegida contra gravação. Somente leitura Boolean. |

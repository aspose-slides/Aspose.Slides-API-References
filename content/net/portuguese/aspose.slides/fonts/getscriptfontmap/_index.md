## Fonts.GetScriptFontMap método

Retorna um dicionário de todas as definições de fontes de script na apresentação.

```csharp
public IDictionary<string, string> GetScriptFontMap()
```

### Valor de Retorno

Um dicionário que mapeia códigos de script para nomes de fontes.

### Exemplos

```csharp
[C#]
IDictionary<string, string> map = presentation.MasterTheme.FontScheme.Major.GetScriptFontMap();
foreach (var kvp in map)
{
    Console.WriteLine(kvp.Key + " ? " + kvp.Value);
}
```

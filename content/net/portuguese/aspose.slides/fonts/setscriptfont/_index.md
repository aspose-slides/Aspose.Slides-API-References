## Fonts.SetScriptFont method

Atribui um nome de fonte a uma tag de script específica, que define como o texto desse script será renderizado na apresentação.

```csharp
public void SetScriptFont(string script, string fontName)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| script | String | O código de script BCP-47 (por exemplo, "Arab", "Hebr", "Hans") que identifica o sistema de escrita. |
| fontName | String | O nome da fonte a ser atribuído ao script especificado. |

### Exemplos

Este exemplo mostra como definir a fonte para o script Árabe como "Segoe UI":

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segoe UI");
```

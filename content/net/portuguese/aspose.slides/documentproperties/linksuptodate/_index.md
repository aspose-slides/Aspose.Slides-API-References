## DocumentProperties.LinksUpToDate propriedade

Indica se os hiperligações em um documento estão atualizadas. Defina este elemento como **true** para indicar que os hiperligações foram atualizados. Defina este elemento como **false** para indicar que os hiperligações estão desatualizados. Leitura/gravação Boolean.

public bool LinksUpToDate { get; set; }

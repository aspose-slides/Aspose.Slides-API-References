## DocumentProperties.GetSensitivityLabels método

Obtém uma matriz de rótulos de sensibilidade das propriedades personalizadas do documento (Microsoft Information Protection SDK Metadata).

```csharp
public ISensitivityLabel[] GetSensitivityLabels()
```

### Exemplos

O código a seguir mostra como mover as informações de rótulos de sensibilidade das propriedades personalizadas do documento para a coleção moderna SensitivityLabels:

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    // Obter rótulos de sensibilidade das propriedades personalizadas do documento
    ISensitivityLabel[] mipSensitivityLabels = pres.DocumentProperties.GetSensitivityLabels();

    ISensitivityLabelCollection sensitivityLabels = pres.SensitivityLabels;
    foreach (var sensitivityLabel in mipSensitivityLabels)
    {
        // Adicionar rótulo à coleção
        // Aqui você pode adicionar uma verificação para a validade das informações do rótulo (o rótulo está disponível, etc)
        sensitivityLabels.Add(sensitivityLabel);
    }

    pres.Save("SensitivityLabel.pptx", SaveFormat.Pptx);
}
```

## IFontsManager.GetFontEmbeddingLevel method

Determina o nível de incorporação de uma fonte a partir do array de bytes fornecido e do nome da fonte.

```csharp
public EmbeddingLevel GetFontEmbeddingLevel(byte[] fontBytes, string fontName)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| fontBytes | Byte[] | O array de bytes contendo os dados da fonte. |
| fontName | String | O nome da fonte. |

### Valor de retorno

O nível de incorporação da fonte especificada.

### Exceções

| exceção | condição |
| --- | --- |
| ArgumentNullException | Lançada quando *fontBytes* é nulo. |

### Exemplos

```csharp
[C#]
using (Presentation pres = new Presentation(pptxFileName))
{
    // Recupera todas as fontes usadas na apresentação
    IFontData[] fontDatas = pres.FontsManager.GetFonts();

    // Obtém o array de bytes que representa o estilo regular da primeira fonte na apresentação
    byte[] bytes = pres.FontsManager.GetFontBytes(fontDatas[0], FontStyle.Regular);

    // Determina o nível de incorporação da fonte
    EmbeddingLevel embeddingLevel = pres.FontsManager.GetFontEmbeddingLevel(bytes, fontDatas[0].FontName);
}
```

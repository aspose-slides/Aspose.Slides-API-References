## Propriedade LoadOptions.DocumentLevelFontSources

Especifica as fontes externas que serão usadas na apresentação. Essas fontes ficam disponíveis para a apresentação durante todo o seu tempo de vida e não são compartilhadas com outras apresentações

```csharp
public IFontSources DocumentLevelFontSources { get; set; }
```

### Exemplos

O exemplo a seguir demonstra como especificar fontes personalizadas usadas com o PowerPoint Presentation.

```csharp
[C#]
byte[] memoryFont1 = File.ReadAllBytes("customfonts\\CustomFont1.ttf");
byte[] memoryFont2 = File.ReadAllBytes("customfonts\\CustomFont2.ttf");
LoadOptions loadOptions = new LoadOptions();
loadOptions.DocumentLevelFontSources.FontFolders = new string[] { "assets\\fonts", "global\\fonts" };
loadOptions.DocumentLevelFontSources.MemoryFonts = new byte[][] { memoryFont1, memoryFont2 };
using (IPresentation presentation = new Presentation("MyPresentation.pptx", loadOptions))
{
    //trabalhe com a apresentação
    //CustomFont1, CustomFont2, bem como as fontes das pastas assets\fonts e global\fonts e suas subpastas, estão disponíveis para a apresentação
}
```

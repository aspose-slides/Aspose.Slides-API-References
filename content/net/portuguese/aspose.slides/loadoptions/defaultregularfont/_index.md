## LoadOptions.DefaultRegularFont propriedade

Retorna ou define a fonte Regular usada caso a fonte de origem não seja encontrada. Leitura/gravação String.

```csharp
public string DefaultRegularFont { get; set; }
```

### Exemplos

O exemplo a seguir mostra como definir fontes padrão para renderizar uma apresentação PowerPoint.

```csharp
[C#]
// Use opções de carregamento para definir as fontes padrão regular e asiática
LoadOptions loadOptions = new LoadOptions(LoadFormat.Auto);
loadOptions.DefaultRegularFont = "Wingdings";
loadOptions.DefaultAsianFont = "Wingdings";
// Carregar a apresentação
using (Presentation pptx = new Presentation("DefaultFonts.pptx", loadOptions))
{
    // Gerar miniatura do slide
    pptx.Slides[0].GetThumbnail(1, 1).Save("output_out.png", ImageFormat.Png);
    // Gerar PDF
    pptx.Save("output_out.pdf", SaveFormat.Pdf);
    // Gerar XPS
    pptx.Save("output_out.xps", SaveFormat.Xps);
}
```

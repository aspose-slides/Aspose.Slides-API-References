## Tab.CompareTo método

Compara a instância atual com outro objeto do mesmo tipo.

```csharp
public int CompareTo(object obj)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| obj | Object | Um objeto para comparar com esta instância. |

### Valor de retorno

Um inteiro de 32 bits que indica a ordem relativa dos comparandos. O valor de retorno tem os seguintes significados:  &lt; 0 - Esta instância é menor que obj. = 0 - Esta instância é igual a obj. &gt; 0 - Esta instância é maior que obj.

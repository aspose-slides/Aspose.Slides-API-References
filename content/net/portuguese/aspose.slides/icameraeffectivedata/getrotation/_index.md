## ICameraEffectiveData.GetRotation método

Uma rotação é definida através do uso de uma coordenada de latitude, uma coordenada de longitude e uma revolução em torno do eixo conforme as coordenadas de latitude e longitude. primeiro elemento no array retornado - latitude, segundo - longitude, terceiro - revolução. Retorna null se nenhuma rotação estiver definida.

```csharp
public float[] GetRotation()
```

### Valor de Retorno

Array de valores de rotação como Single[].

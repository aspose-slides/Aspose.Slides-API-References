## AudioFrame.CaptionTracks propriedade

Obtém a coleção de legendas ocultas associadas ao quadro de áudio. Esta propriedade é somente leitura e retorna um [`ICaptionsCollection`](../../icaptionscollection) contendo todas as faixas de legendas.

```csharp
public ICaptionsCollection CaptionTracks { get; }
```

### Exemplos

Exemplo:

```csharp
[C#]
using (Presentation pres = new Presentation("audio with captions.pptx"))
{
    foreach (IShape shape in pres.Slides[0].Shapes)
    {
        if (shape is IAudioFrame audioFrame)
        {
            // Salve os dados binários da faixa de legenda como um arquivo .vtt
            foreach (var captionTrack in audioFrame.CaptionTracks)
                File.WriteAllBytes(captionTrack.CaptionId + ".vtt", captionTrack.BinaryData);
        }
    }
}
```

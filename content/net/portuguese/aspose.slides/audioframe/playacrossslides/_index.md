## AudioFrame.PlayAcrossSlides propriedade

Determina se o áudio está sendo reproduzido ao longo dos slides. Leitura/gravação Boolean.

```csharp
public bool PlayAcrossSlides { get; set; }
```

### Exemplos

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ISlide slide = pres.Slides[0];

    // Adicionar quadro de áudio
    IAudioFrame audioFrame = slide.Shapes.AddAudioFrameLinked(50, 50, 100, 100, "sampleaudio.wav");

    // Definir áudio para reproduzir ao longo dos slides
    audioFrame.PlayAcrossSlides = true;

    // Definir áudio para rebobinar automaticamente ao início após a reprodução
    audioFrame.RewindAudio = true;

    pres.Save("AudioFrame_out.pptx", SaveFormat.Pptx);
}
```

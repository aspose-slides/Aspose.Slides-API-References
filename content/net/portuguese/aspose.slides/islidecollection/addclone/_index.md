## AddClone(ISlide) {#addclone}

Adiciona uma cópia de um slide especificado ao final da coleção.

```csharp
public ISlide AddClone(ISlide sourceSlide)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| sourceSlide | ISlide | Slide a ser clonado. |

### Valor de Retorno

Novo slide.

### Observações

Ao clonar um slide entre apresentações diferentes, o master do slide também pode ser clonado. Um registro interno é usado para rastrear masters clonados automaticamente e impedir a criação de múltiplos clones do mesmo master slide. A clonagem manual de masters de slide nem será impedida nem registrada. Se precisar de mais controle sobre o processo de clonagem, use [`AddClone`](../addclone) ou [`AddClone`](../addclone) para clonar slides, [`AddClone`](../../igloballayoutslidecollection/addclone) ou [`AddClone`](../../igloballayoutslidecollection/addclone) para clonar layouts e [`AddClone`](../../imasterslidecollection/addclone) para clonar masters.

### Exemplos

```csharp
[C#]
using (IPresentation presentation = new Presentation())
{
    presentation.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 200, 50, 300, 100);
    presentation.Sections.AddSection("Section 1", presentation.Slides[0]);
    
    ISection section2 = presentation.Sections.AppendEmptySection("Section 2");
    presentation.Slides.AddClone(presentation.Slides[0], section2);
    
    // Agora a segunda seção contém uma cópia do primeiro slide.
}
```

## AddClone(ISlide, IMasterSlide, bool) {#addclone_2}

Adiciona uma cópia de um slide de origem especificado ao final da coleção. O layout apropriado será selecionado automaticamente a partir do master especificado (layout apropriado é o layout com o mesmo Type ou Name do layout do slide de origem). Se não houver layout apropriado, o layout do slide de origem será clonado (se allowCloneMissingLayout for true) ou será lançada uma PptxEditException (se allowCloneMissingLayout for false).

```csharp
public ISlide AddClone(ISlide sourceSlide, IMasterSlide destMaster, bool allowCloneMissingLayout)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| sourceSlide | ISlide | Slide a ser clonado. |
| destMaster | IMasterSlide | Master slide para um novo slide. |
| allowCloneMissingLayout | Boolean | Se não houver layout apropriado no master especificado, o layout do slide de origem será clonado (se allowCloneMissingLayout for true) ou será lançada uma PptxEditException (se allowCloneMissingLayout for false). |

### Valor de Retorno

Novo slide.

### Exceções

| exceção | condição |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | Lançada se não houver layout apropriado no master especificado e allowCloneMissingLayout for false. |

## BulletFormat.Height propriedade

Retorna ou define a altura da marca de um parágrafo sem herança. O valor float.NaN determina que a marca herda a altura da primeira porção no parágrafo. Leitura/Gravação Single.

```csharp
public float Height { get; set; }
```

### Observações

Um valor de altura negativo significa que a altura é fornecida em pontos e um valor positivo significa que a altura é uma porcentagem do texto ao redor.

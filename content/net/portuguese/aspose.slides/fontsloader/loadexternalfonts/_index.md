## FontsLoader.LoadExternalFonts método

Adiciona pastas adicionais para procurar fontes.

```csharp
public static void LoadExternalFonts(string[] directories)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| directories | String[] | Diretórios para ler fontes adicionais. |

### Exemplos

O exemplo a seguir mostra como carregar fontes personalizadas a partir de arquivos .TTF

```csharp
[C#]
// O caminho para o diretório de documentos.
string dataDir = "C:\\";
// pastas para procurar fontes
String[] folders = new String[] { dataDir };
// Carregar as fontes do diretório de fontes personalizadas
FontsLoader.LoadExternalFonts(folders);
// Executar algum trabalho e renderizar a apresentação/slide
using (Presentation presentation = new Presentation(dataDir + "DefaultFonts.pptx"))
{
    presentation.Save(dataDir + "NewFonts_out.pptx", SaveFormat.Pptx);
	// Limpar cache de fontes
	FontsLoader.ClearCache();
}
```

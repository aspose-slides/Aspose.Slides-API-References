## ISummaryZoomSectionCollection.AddSummaryZoomSection method

Cria um novo objeto Summary Zoom Section e o adiciona à coleção

```csharp
public ISummaryZoomSection AddSummaryZoomSection(ISection section)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| section | ISection | Seção para um novo elemento Summary Zoom Section [`ISection`](../../isection) |

### Valor de retorno

Elemento [`ISummaryZoomFrame`](../../isummaryzoomframe) adicionado

### Observações

Se já existir um elemento para esta seção na coleção, o elemento existente será retornado.

### Exemplos

O exemplo demonstra a obtenção do elemento Summary Zoom Section por índice:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    ISummaryZoomFrame zoomFrame = (ISummaryZoomFrame) pres.Slides[1].Shapes[0];
    ISummaryZoomSectionCollection collection = zoomFrame.SummaryZoomCollection;
    ISummaryZoomSection newZoomSection = collection.AddSummaryZoomSection(pres.Sections[3]);
}
```

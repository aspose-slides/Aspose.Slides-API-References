## PictureFillFormat.DeletePictureCroppedAreas método

Exclui áreas recortadas da imagem de preenchimento.

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### Valor de Retorno

Imagem recortada ou imagem original se o recorte não for necessário.

### Observações

Este método converte arquivos metafile WMF/EMF para imagem PNG raster enquanto recorta.

### Exemplos

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Obtém o PictureFrame
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // Exclui áreas recortadas da imagem do PictureFrame
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

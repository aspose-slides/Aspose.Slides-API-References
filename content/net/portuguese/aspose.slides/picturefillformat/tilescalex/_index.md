## PictureFillFormat.TileScaleX propriedade

Retorna ou define a escala horizontal para o preenchimento de textura como uma porcentagem. Leitura/gravação Single.

```csharp
public float TileScaleX { get; set; }
```

### Exemplos

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Obtém o formato de preenchimento de imagem da forma
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Define o modo de preenchimento de imagem para Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Define a escala horizontal da textura para 120 por cento
    pictureFillFormat.TileScaleX = 120;
}
```

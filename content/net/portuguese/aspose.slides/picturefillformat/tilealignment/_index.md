## PictureFillFormat.TileAlignment propriedade

Retorna ou define como a textura é alinhada dentro da forma. Esta configuração controla o ponto de partida do padrão de textura e como ele se repete ao longo da forma. Leitura/gravação [`RectangleAlignment`](../../rectanglealignment).

```csharp
public RectangleAlignment TileAlignment { get; set; }
```

### Observações

Padrão é TopLeft.

### Exemplos

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Obtém o formato de preenchimento de imagem da forma
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Define o modo de preenchimento de imagem como Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Define o alinhamento do mosaico para a parte inferior direita
    pictureFillFormat.TileAlignment = RectangleAlignment.BottomRight;
}
```

## Portion.PortionFormat propriedade

Retorna um objeto de formatação que contém as propriedades de formatação definidas explicitamente do trecho de texto, sem herança aplicada. Somente leitura [`IPortionFormat`](../../iportionformat).

```csharp
public IPortionFormat PortionFormat { get; }
```

### Observações

O objeto de formatação contém apenas os parâmetros de formatação definidos para a porção atual, dados herdados não são aplicados.

Para obter os valores efetivos, incluindo os herdados, use o método [`GetEffective`](../../portionformat/geteffective).

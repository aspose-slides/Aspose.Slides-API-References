## ViewProperties.GridSpacing propriedade

Retorna ou define o espaçamento da grade que deve ser usado para a grade subjacente ao documento de apresentação, em pontos. Leitura/Gravação Single.

```csharp
public float GridSpacing { get; set; }
```

### Observações

O valor do espaçamento da grade deve ser um número positivo. O intervalo típico varia de 1 mm (2.8349607 pontos) a 2 polegadas (144 pontos).

### Exemplos

O código de exemplo a seguir mostra como alterar o espaçamento da grade em uma apresentação do PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

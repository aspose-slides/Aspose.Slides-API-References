## IDigitalSignature.IsValid propriedade

Se esta assinatura digital for válida e o documento não tiver sido adulterado, este valor será true. Somente leitura Boolean.

```csharp
public bool IsValid { get; }
```

### Exemplos

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentationSigned.pptx"))
{
    foreach (DigitalSignature signature in pres.DigitalSignatures)
        Console.WriteLine("Signature check: " + (signature.IsValid ? "VALID" : "INVALID"));
}
```

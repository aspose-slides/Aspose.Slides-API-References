## ParagraphCollection.CopyTo método

Copia os elementos da ICollection para um Array, iniciando em um índice específico do Array.

```csharp
public void CopyTo(IParagraph[] array, int arrayIndex)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| array | IParagraph[] | O Array unidimensional que é o destino dos elementos copiados da ICollection. O Array deve ter indexação baseada em zero. |
| arrayIndex | Int32 | O índice baseado em zero em *array* onde a cópia começa. |

### Exceções

| exceção | condição |
| --- | --- |
| ArgumentNullException | *array* é nulo. |
| ArgumentOutOfRangeException | *arrayIndex* é menor que 0. |
| ArgumentException | O número de elementos na ICollection origem é maior que o espaço disponível a partir de *arrayIndex* até o final do *array* de destino. |

## IAudioFrame.VolumeValue propriedade

Retorna ou define o volume de áudio em porcentagem. Leitura/Gravação Single.

```csharp
public float VolumeValue { get; set; }
```

### Exemplos

Exemplo:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    IAudio audio = pres.Audios.AddAudio(File.ReadAllBytes("sampleaudio.mp3"));
    IAudioFrame audioFrame = pres.Slides[0].Shapes.AddAudioFrameEmbedded(50, 50, 100, 100, audio);

    // Defina o volume do áudio para 85%
    audioFrame.VolumeValue = 85f;

    pres.Save("AudioFrameValue_out.pptx", SaveFormat.Pptx);
}
```

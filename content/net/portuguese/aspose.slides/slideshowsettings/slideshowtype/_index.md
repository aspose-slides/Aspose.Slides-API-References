## SlideShowSettings.SlideShowType propriedade

Obtém ou define o tipo de apresentação de slides. Representado pelos seguintes ancestrais `SlideShowType`: [`BrowsedAtKiosk`](../../browsedatkiosk), [`PresentedBySpeaker`](../../presentedbyspeaker) e [`BrowsedByIndividual`](../../browsedbyindividual)

```csharp
public SlideShowType SlideShowType { get; set; }
```

### Exemplos

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // para definir o tipo "Browsed at a kiosk (full screen)" type 
    // para definir o tipo "Browsed by individual (window)" type
    // para definir o tipo "Presented by a speaker (full screen)" type
    pres.SlideShowSettings.SlideShowType = new BrowsedAtKiosk();
    
    pres.SlideShowSettings.SlideShowType = new BrowsedByIndividual();
    
    pres.SlideShowSettings.SlideShowType = new PresentedBySpeaker();
}
```

## IPresentationInfo.UpdateDocumentProperties método

Atualiza as propriedades da apresentação vinculada.

```csharp
public void UpdateDocumentProperties(IDocumentProperties documentProperties)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| documentProperties | IDocumentProperties | Propriedades do documento [`IDocumentProperties`](../../idocumentproperties) |

### Exemplos

Este exemplo mostra como chamar o método `UpdateDocumentProperties` para atualizar as propriedades do documento retornadas pela chamada do método [`ReadDocumentProperties`](../readdocumentproperties).

```csharp
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo("pres.pptx");
IDocumentProperties props = info.ReadDocumentProperties();
props.Subject = "New subject";
props.LastSavedTime = DateTime.UtcNow;
info.UpdateDocumentProperties(props);
info.WriteBindedPresentation("new_pres.pptx");
```

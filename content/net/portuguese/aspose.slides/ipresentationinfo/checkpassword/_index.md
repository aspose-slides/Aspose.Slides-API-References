## IPresentationInfo.CheckPassword method

Verifica se uma senha está correta para uma apresentação protegida com senha aberta.

```csharp
public bool CheckPassword(string password)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| password | String | A senha a ser verificada. |

### Valor de Retorno

True se a apresentação estiver protegida com senha aberta e a senha estiver correta e false caso contrário.

### Observações

Quando a senha é null ou vazia, este método retorna false.

### Exemplos

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo("pres.pptx");
bool isPasswordCorrect = info.CheckPassword("my_password");
```

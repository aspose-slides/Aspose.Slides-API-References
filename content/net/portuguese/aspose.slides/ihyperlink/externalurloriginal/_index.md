## IHyperlink.ExternalUrlOriginal propriedade

Representa um hyperlink que é definido para esta porção sem considerar o conteúdo real da porção. O PowerPoint se comporta de maneira específica para links e seu texto correspondente em uma porção. Permite criar texto para o hyperlink na forma de uma URL válida, diferente do endereço real do link. Neste caso, quando você visualiza o link na janela de edição, ele será alterado para corresponder à porção de texto. Esta propriedade representa o valor original do hyperlink.

```csharp
public string ExternalUrlOriginal { get; }
```

## WriteAsSvg(Stream) {#writeassvg}

Salva o conteúdo do slide como um arquivo SVG.

```csharp
public void WriteAsSvg(Stream stream)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| stream | Stream | Fluxo de destino |

### Exemplos

O exemplo de código a seguir demonstra como converter o primeiro slide de uma apresentação PowerPoint em um arquivo SVG.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.svg"))
    {
        // Salva o primeiro slide como um arquivo SVG
        pres.Slides[0].WriteAsSvg(fileStream);
    }
}
```

| --- | --- | --- |
| stream | Stream | Fluxo de destino |
| svgOptions | ISVGOptions | opções de geração SVG |

### Exemplos

O exemplo de código a seguir demonstra como converter o primeiro slide de uma apresentação PowerPoint em um arquivo SVG com opções.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.svg"))
    {
        var options = new SVGOptions() { VectorizeText = true };
        // Salva o primeiro slide como um arquivo SVG
        pres.Slides[0].WriteAsSvg(fileStream, options);
    }
}
```

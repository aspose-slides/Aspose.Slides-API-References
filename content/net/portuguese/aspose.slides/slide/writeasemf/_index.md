## Slide.WriteAsEmf método

Salva o conteúdo do slide como um arquivo EMF.

```csharp
public void WriteAsEmf(Stream stream)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| stream | Stream | Fluxo de destino |

### Exceções

| exceção | condição |
| --- | --- |
| ArgumentNullException | O fluxo de destino é `null` |

### Exemplos

O exemplo de código a seguir demonstra como converter o primeiro slide de uma apresentação PowerPoint em um arquivo metafile.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.emf"))
    {
        // Salva o primeiro slide como um metafile
        pres.Slides[0].WriteAsEmf(fileStream);
    }
}
```

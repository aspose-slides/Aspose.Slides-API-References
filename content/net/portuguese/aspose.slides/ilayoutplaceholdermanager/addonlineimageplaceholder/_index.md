## ILayoutPlaceholderManager.AddOnlineImagePlaceholder method

Adiciona uma nova forma placeholder ao slide de layout para conter uma imagem online.

```csharp
public IAutoShape AddOnlineImagePlaceholder(float x, float y, float width, float height)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| x | Single | A coordenada X da nova forma placeholder. |
| y | Single | A coordenada Y da nova forma placeholder. |
| width | Single | A largura da nova forma placeholder. |
| height | Single | A altura da nova forma placeholder. |

### Valor de retorno

Criado [`IAutoShape`](../../iautoshape) com um placeholder de Imagem Online.

### Exemplos

O exemplo a seguir demonstra como adicionar a forma placeholder de Imagem Online ao slide de layout.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddOnlineImagePlaceholder(20, 20, 200, 200);
}
```

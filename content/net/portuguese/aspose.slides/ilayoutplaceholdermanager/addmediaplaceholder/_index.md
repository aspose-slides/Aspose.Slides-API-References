## ILayoutPlaceholderManager.AddMediaPlaceholder método

Adiciona uma nova forma de marcador de posição ao slide de layout para conter um objeto de mídia.

```csharp
public IAutoShape AddMediaPlaceholder(float x, float y, float width, float height)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| x | Single | A coordenada X da nova forma de marcador de posição. |
| y | Single | A coordenada Y da nova forma de marcador de posição. |
| width | Single | A largura da nova forma de marcador de posição. |
| height | Single | A altura da nova forma de marcador de posição. |

### Valor de Retorno

Criado [`IAutoShape`](../../iautoshape) com um marcador de posição de Mídia.

### Exemplos

O exemplo a seguir mostra como adicionar a forma de marcador de posição de Mídia ao slide de layout.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddMediaPlaceholder(20, 20, 200, 200);
}
```

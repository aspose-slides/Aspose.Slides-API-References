## ILayoutPlaceholderManager.AddVerticalTextPlaceholder método

Adiciona uma nova forma de espaço reservado ao slide de layout para conter conteúdo de texto em direção vertical.

```csharp
public IAutoShape AddVerticalTextPlaceholder(float x, float y, float width, float height)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| x | Single | A coordenada X da nova forma de espaço reservado. |
| y | Single | A coordenada Y da nova forma de espaço reservado. |
| width | Single | A largura da nova forma de espaço reservado. |
| height | Single | A altura da nova forma de espaço reservado. |

### Valor de Retorno

Created [`IAutoShape`](../../iautoshape) with a Text (Vertical) placeholder.

### Exemplos

O exemplo a seguir mostra como adicionar a forma de espaço reservado Text (Vertical) ao slide de layout.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalTextPlaceholder(20, 20, 300, 500);
}
```

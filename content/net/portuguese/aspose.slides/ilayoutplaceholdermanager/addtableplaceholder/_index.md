## ILayoutPlaceholderManager.AddTablePlaceholder method

Adiciona uma nova forma de espaço reservado ao slide de layout para conter uma tabela.

```csharp
public IAutoShape AddTablePlaceholder(float x, float y, float width, float height)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| x | Single | A coordenada X da nova forma de espaço reservado. |
| y | Single | A coordenada Y da nova forma de espaço reservado. |
| width | Single | A largura da nova forma de espaço reservado. |
| height | Single | A altura da nova forma de espaço reservado. |

### Valor de Retorno

Criado [`IAutoShape`](../../iautoshape) com um espaço reservado de Table.

### Exemplos

O exemplo a seguir mostra como adicionar a forma de espaço reservado de Table ao slide de layout.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTablePlaceholder(20, 20, 500, 200);
}
```

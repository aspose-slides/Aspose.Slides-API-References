## IProtectionManager.EncryptDocumentProperties propriedade

Esta propriedade faz sentido se a apresentação estiver protegida por senha. Se true então as propriedades do documento são criptografadas no arquivo da apresentação. Se false então as propriedades do documento são públicas enquanto a apresentação é criptografada. Leitura/Gravação Boolean.

public bool EncryptDocumentProperties { get; set; }

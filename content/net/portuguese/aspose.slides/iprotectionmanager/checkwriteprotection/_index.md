## IProtectionManager.CheckWriteProtection method

Determina se uma apresentação está protegida por senha para modificação.

```csharp
public bool CheckWriteProtection(string password)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| password | String | A senha para verificação. |

### Valor de Retorno

True se a senha for válida; caso contrário, false.

### Observações

1. Você deve verificar a propriedade [`IsWriteProtected`](../iswriteprotected) antes de chamar este método.
2. Quando a senha for nula ou vazia, este método retorna false.

### Exemplos

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

## InsertConnector(int, ShapeType, float, float, float, float, bool) {#insertconnector_1}

```csharp
public IConnector InsertConnector(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| index | Int32 | O índice baseado em zero no qual inserir a forma de conector. |
| shapeType | ShapeType | O [`ShapeType`](../../shapetype) da forma de conector a ser inserida. |
| x | Single | A coordenada x da moldura do conector, em pontos. |
| y | Single | A coordenada y da moldura do conector, em pontos. |
| width | Single | A largura da moldura do conector, em pontos. |
| height | Single | A altura da moldura do conector, em pontos. |
| createFromTemplate | Boolean | True para aplicar o estilo de modelo padrão (nome não vazio, estilo simples); false para criar o conector com valores de propriedade padrão. |

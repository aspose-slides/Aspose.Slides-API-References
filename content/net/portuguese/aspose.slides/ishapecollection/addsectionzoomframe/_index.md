## AddSectionZoomFrame(float, float, float, float, ISection) {#addsectionzoomframe}

Cria um novo quadro Section Zoom e o adiciona ao final da coleção de formas.

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| x | Single | A coordenada x do novo quadro Section Zoom, em pontos. |
| y | Single | A coordenada y do novo quadro Section Zoom, em pontos. |
| width | Single | A largura do novo quadro Section Zoom, em pontos. |
| height | Single | A altura do novo quadro Section Zoom, em pontos. |
| section | ISection | O [`ISection`](../../isection) referenciado pelo quadro Section Zoom; deve pertencer a esta apresentação e conter ao menos um slide. |

### Valor de Retorno

O [`ISectionZoomFrame`](../../isectionzoomframe) recém-criado.

### Exceções

| exceção | condição |
| --- | --- |
| ArgumentException | Lançada se a seção referenciada não pertencer à apresentação atual ou não contiver slides. |

### Exemplos

Este exemplo demonstra como adicionar um objeto Section Zoom ao final de uma coleção (suponha que existam pelo menos duas seções na apresentação "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1]);
}
```

## AddSectionZoomFrame(float, float, float, float, ISection, IPPImage) {#addsectionzoomframe_1}

Cria um novo quadro Section Zoom com uma imagem predefinida e o adiciona ao final da coleção de formas.

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section, IPPImage image)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| x | Single | A coordenada x do novo quadro Section Zoom, em pontos. |
| y | Single | A coordenada y do novo quadro Section Zoom, em pontos. |
| width | Single | A largura do novo quadro Section Zoom, em pontos. |
| height | Single | A altura do novo quadro Section Zoom, em pontos. |
| section | ISection | O [`ISection`](../../isection) referenciado pelo quadro Section Zoom; deve pertencer a esta apresentação e conter ao menos um slide. |
| image | IPPImage | O [`IPPImage`](../../ippimage) a ser exibido dentro do quadro Section Zoom. |

### Valor de Retorno

O [`ISectionZoomFrame`](../../isectionzoomframe) recém-criado.

### Exceções

| exceção | condição |
| --- | --- |
| ArgumentException | Lançada se a seção referenciada não pertencer à apresentação atual ou não contiver slides. |

### Exemplos

Este exemplo demonstra como adicionar um objeto Section Zoom ao final de uma coleção (suponha que existam pelo menos duas seções na apresentação "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IPPImage image = pres.Images.AddImage(Image.FromFile("image.png"));
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1], image);
}
```

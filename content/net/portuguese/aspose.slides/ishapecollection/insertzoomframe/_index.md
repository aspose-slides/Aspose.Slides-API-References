## InsertZoomFrame(int, float, float, float, float, ISlide) {#insertzoomframe}

Cria uma nova moldura Zoom e a insere na coleção de formas no índice especificado.

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| index | Int32 | O índice baseado em zero no qual inserir a moldura Zoom. |
| x | Single | A coordenada x da nova moldura Zoom, em pontos. |
| y | Single | A coordenada y da nova moldura Zoom, em pontos. |
| width | Single | A largura da nova moldura Zoom, em pontos. |
| height | Single | A altura da nova moldura Zoom, em pontos. |
| slide | ISlide | O [`ISlide`](../../islide) referenciado pela moldura Zoom. |

### Valor de retorno

O [`IZoomFrame`](../../izoomframe) recém-criado.

### Exceções

| Exceção | Condição |
| --- | --- |
| ArgumentException | Lançada se o slide referenciado não pertencer à apresentação atual. |

### Exemplos

Este exemplo demonstra a criação e inserção de um objeto Zoom no índice especificado de uma coleção (assuma que existam pelo menos dois slides na apresentação "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertZoomFrame(2, 150, 20, 50, 50, pres.Slides[1]);
}
```

## InsertZoomFrame(int, float, float, float, float, ISlide, IPPImage) {#insertzoomframe_1}

Cria uma nova moldura Zoom com uma imagem predefinida e a insere na coleção de formas no índice especificado.

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide, IPPImage image)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| index | Int32 | O índice baseado em zero no qual inserir a moldura Zoom. |
| x | Single | A coordenada x da nova moldura Zoom, em pontos. |
| y | Single | A coordenada y da nova moldura Zoom, em pontos. |
| width | Single | A largura da nova moldura Zoom, em pontos. |
| height | Single | A altura da nova moldura Zoom, em pontos. |
| slide | ISlide | O [`ISlide`](../../islide) referenciado pela moldura Zoom. |
| image | IPPImage | A imagem para o slide referenciado [`IPPImage`](../../ippimage). |

### Valor de retorno

O [`IZoomFrame`](../../izoomframe) recém-criado.

### Exceções

| Exceção | Condição |
| --- | --- |
| ArgumentException | Lançada se o slide referenciado não pertencer à apresentação atual. |

### Exemplos

Este exemplo demonstra a criação e inserção de um objeto Zoom no índice especificado de uma coleção (assuma que existam pelo menos dois slides na apresentação "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IPPImage image = pres.Images.AddImage(Image.FromFile("image.png"));
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertZoomFrame(2, 150, 20, 50, 50, pres.Slides[1], image);
}
```

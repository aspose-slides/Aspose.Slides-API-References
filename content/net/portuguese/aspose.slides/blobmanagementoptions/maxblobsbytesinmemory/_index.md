## BlobManagementOptions.MaxBlobsBytesInMemory propriedade

Define o tamanho total máximo (in bytes) que todos os BLOBs podem ocupar na memória. Por padrão, todos os BLOBs são carregados na memória; somente quando esse limite é atingido são empregados mecanismos alternativos (such as temporary files). Manter os BLOBs na memória maximiza o desempenho, mas pode gerar uso elevado de memória. Use esta propriedade para ajustar o comportamento ao seu ambiente ou aos requisitos.

```csharp
public ulong MaxBlobsBytesInMemory { get; set; }
```

### Observações

Esta propriedade é ignorada se [`IsTemporaryFilesAllowed`](../istemporaryfilesallowed) estiver definido como false, pois a memória será então o único local de armazenamento disponível e limitar o uso de BLOB em memória não terá efeito.

O valor padrão é 629,145,600 bytes (600 MB).

Você pode definir esta propriedade como zero, mas uma pequena quantidade mínima de memória ainda será reservada.

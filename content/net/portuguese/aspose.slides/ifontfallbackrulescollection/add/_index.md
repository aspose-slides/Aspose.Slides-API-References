## IFontFallBackRulesCollection.Add method

Adiciona uma nova regra FallBack ao final da coleção.

```csharp
public void Add(IFontFallBackRule sourceRule)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| sourceRule | IFontFallBackRule | Regra especificada para adição |

### Exemplos

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    //Obtendo a coleção de regras vazia ou pré-inicializada do FontsManager
    IFontFallBackRulesCollection rulesList = pres.FontsManager.FontFallBackRulesCollection;

    //Adicionando nova regra à coleção

}
```

## FontFallBackRulesCollection.Add método

Adiciona uma regra FallBack especificada ao final da coleção.

```csharp
public void Add(IFontFallBackRule sourceRule)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| sourceRule | IFontFallBackRule | Regra especificada para adição |

### Exemplos

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    //Obtendo a coleção de regras vazia ou pré-inicializada do FontsManager
    IFontFallBackRulesCollection rulesList = pres.FontsManager.FontFallBackRulesCollection;

    //Adicionando uma nova regra à coleção
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));
}
```

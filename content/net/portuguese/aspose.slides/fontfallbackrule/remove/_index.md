## FontFallBackRule.Remove método

Remove a primeira ocorrência de uma fonte FallBack específica da lista.

```csharp
public void Remove(string fontName)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| fontName | String | O nome da fonte a ser removido da lista. |

### Exemplos

```csharp
[C#]
// Cria uma regra que contém uma lista de fontes.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// Remove Tahoma da lista.
newRule.Remove ("Tahoma");
```

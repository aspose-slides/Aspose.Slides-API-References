## Presentation.HeaderFooterManager propriedade

Retorna o gerenciador HeaderFooter atual. Somente leitura [`IPresentationHeaderFooterManager`](../../ipresentationheaderfootermanager).

### Exemplos

O exemplo a seguir mostra como definir a visibilidade do rodapé dentro de um Slide de uma Apresentação PowerPoint.

```csharp
[C#]
using (Presentation presentation = new Presentation("presentation.ppt"))
{
    IBaseSlideHeaderFooterManager headerFooterManager = presentation.Slides[0].HeaderFooterManager;
	// Propriedade IsFooterVisible é usada para indicar que um espaço reservado de rodapé de slide não está presente.
    if (!headerFooterManager.IsFooterVisible)
    {
		// Método SetFooterVisibility é usado para tornar o espaço reservado de rodapé de slide visível.
        headerFooterManager.SetFooterVisibility(true);
    }
	// Propriedade IsSlideNumberVisible é usada para indicar que um espaço reservado de número de página de slide não está presente.
    if (!headerFooterManager.IsSlideNumberVisible)
    {
		// Método SetSlideNumberVisibility é usado para tornar o espaço reservado de número de página de slide visível.
        headerFooterManager.SetSlideNumberVisibility(true);
    }
	// Propriedade IsDateTimeVisible é usada para indicar que um espaço reservado de data e hora de slide não está presente.
    if (!headerFooterManager.IsDateTimeVisible)
    {
		// Método SetFooterVisibility é usado para tornar o espaço reservado de data e hora de slide visível.
        headerFooterManager.SetDateTimeVisibility(true);
    }
	// Método SetFooterText é usado para definir o texto no espaço reservado de rodapé de slide.
    headerFooterManager.SetFooterText("Footer text");
	// Método SetDateTimeText é usado para definir o texto no espaço reservado de data e hora de slide.
    headerFooterManager.SetDateTimeText("Date and time text");
	presentation.Save("Presentation.ppt",SaveFormat.ppt);
}
```

using (Presentation presentation = new Presentation("presentation.ppt"))

	// Método SetFooterAndChildFootersVisibility é usado para tornar visível um slide mestre e todos os espaços reservados de rodapé filhos.

	// Método SetSlideNumberAndChildSlideNumbersVisibility é usado para tornar visível um slide mestre e todos os espaços reservados de número de página filhos.
    headerFooterManager.SetSlideNumberAndChildSlideNumbersVisibility(true);
	// Método SetDateTimeAndChildDateTimesVisibility é usado para tornar visível um slide mestre e todos os espaços reservados de data e hora filhos.
    headerFooterManager.SetDateTimeAndChildDateTimesVisibility(true);
	// Método SetFooterAndChildFootersText é usado para definir o texto no slide mestre e em todos os espaços reservados de rodapé filhos.
    headerFooterManager.SetFooterAndChildFootersText("Footer text");
	// Método SetDateTimeAndChildDateTimesText é usado para definir o texto no slide mestre e em todos os espaços reservados de data e hora filhos.
    headerFooterManager.SetDateTimeAndChildDateTimesText("Date and time text");
}

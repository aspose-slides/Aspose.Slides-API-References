## IShape.Rotation propriedade

Retorna ou define o número de graus que a forma especificada está rotacionada ao redor do eixo z. Um valor positivo indica rotação no sentido horário; um valor negativo indica rotação no sentido anti-horário. Leitura/gravação Single.

```csharp
public float Rotation { get; set; }
```

### Observações

O valor retornado está sempre definido (não é float.NaN). O valor atribuído deve estar definido (não float.NaN). Você pode definir valores indefinidos para as propriedades da instância RawFrame.

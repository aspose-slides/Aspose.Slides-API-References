## PresentationInfo.CheckPassword método

Verifica se uma senha está correta para uma apresentação protegida com senha aberta.

```csharp
public bool CheckPassword(string password)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| password | String | A senha a ser verificada. |

### Valor de retorno

Verdadeiro se a apresentação estiver protegida com senha aberta e a senha estiver correta e falso caso contrário.

### Exceções

| exceção | condição |
| --- | --- |
| InvalidOperationException |  |
| NotSupportedException |  |

### Observações

Quando a senha for nula ou vazia, este método retorna falso.

### Exemplos

```csharp

bool isPasswordCorrect = info.CheckPassword("my_password");
```

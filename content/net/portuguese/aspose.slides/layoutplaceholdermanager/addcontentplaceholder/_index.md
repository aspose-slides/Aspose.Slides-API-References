## LayoutPlaceholderManager.AddContentPlaceholder método

Adiciona uma nova forma de placeholder ao slide de layout para conter conteúdo, como imagem, tabela, mídia ou texto.

```csharp
public IAutoShape AddContentPlaceholder(float x, float y, float width, float height)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| x | Single | A coordenada X da nova forma de placeholder. |
| y | Single | A coordenada Y da nova forma de placeholder. |
| width | Single | A largura da nova forma de placeholder. |
| height | Single | A altura da nova forma de placeholder. |

### Valor de Retorno

Criado [`IAutoShape`](../../iautoshape) com um placeholder de Conteúdo.

### Exemplos

O exemplo a seguir mostra como adicionar a forma de placeholder de Conteúdo ao slide de layout.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddContentPlaceholder(20, 20, 500, 300);
}
```

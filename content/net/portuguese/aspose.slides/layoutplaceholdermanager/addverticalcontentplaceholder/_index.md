## LayoutPlaceholderManager.AddVerticalContentPlaceholder method

Adiciona uma nova forma de marcador de posição ao slide de layout para conter conteúdo, como uma imagem, tabela, mídia ou texto, na direção vertical.

```csharp
public IAutoShape AddVerticalContentPlaceholder(float x, float y, float width, float height)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| x | Single | A coordenada X da nova forma de marcador de posição. |
| y | Single | A coordenada Y da nova forma de marcador de posição. |
| width | Single | A largura da nova forma de marcador de posição. |
| height | Single | A altura da nova forma de marcador de posição. |

### Valor de Retorno

Criado [`IAutoShape`](../../iautoshape) com um marcador de conteúdo (Vertical).

### Exemplos

O exemplo a seguir mostra como adicionar a forma de marcador de conteúdo (Vertical) ao slide de layout.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalContentPlaceholder(20, 20, 300, 500);
}
```

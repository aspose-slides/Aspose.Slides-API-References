## LayoutPlaceholderManager.AddSmartArtPlaceholder método

Adiciona uma nova forma de espaço reservado ao slide de layout para conter um diagrama SmartArt.

```csharp
public IAutoShape AddSmartArtPlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| x | Single | A coordenada X da nova forma de espaço reservado. |
| y | Single | A coordenada Y da nova forma de espaço reservado. |
| width | Single | A largura da nova forma de espaço reservado. |
| height | Single | A altura da nova forma de espaço reservado. |

### Valor de retorno

Criado [`IAutoShape`](../../iautoshape) com um placeholder SmartArt.

### Exemplos

O exemplo a seguir mostra como adicionar a forma de placeholder SmartArt ao slide de layout.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddSmartArtPlaceholder(20, 20, 200, 200);
}
```

## LayoutPlaceholderManager.AddTextPlaceholder método

Adiciona uma nova forma de marcador de posição ao slide de layout para conter conteúdo de texto.

```csharp
public IAutoShape AddTextPlaceholder(float x, float y, float width, float height)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| x | Single | A coordenada X da nova forma de marcador de posição. |
| y | Single | A coordenada Y da nova forma de marcador de posição. |
| width | Single | A largura da nova forma de marcador de posição. |
| height | Single | A altura da nova forma de marcador de posição. |

### Valor de Retorno

Criado [`IAutoShape`](../../iautoshape) com um marcador de posição Text.

### Exemplos

O exemplo a seguir mostra como adicionar a forma de marcador de posição Text ao slide de layout.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTextPlaceholder(20, 20, 500, 300);
}
```

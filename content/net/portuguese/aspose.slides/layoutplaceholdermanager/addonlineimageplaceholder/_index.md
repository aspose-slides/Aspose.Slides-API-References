## LayoutPlaceholderManager.AddOnlineImagePlaceholder method

Adiciona uma nova forma de espaço reservado ao slide de layout para conter uma imagem online.

```csharp
public IAutoShape AddOnlineImagePlaceholder(float x, float y, float width, float height)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| x | Single | A coordenada X da nova forma de espaço reservado. |
| y | Single | A coordenada Y da nova forma de espaço reservado. |
| width | Single | A largura da nova forma de espaço reservado. |
| height | Single | A altura da nova forma de espaço reservado. |

### Valor de retorno

Criado [`IAutoShape`](../../iautoshape) com um placeholder de Imagem Online.

### Exemplos

O exemplo a seguir mostra como adicionar a forma de placeholder de Imagem Online ao slide de layout.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddOnlineImagePlaceholder(20, 20, 200, 200);
}
```

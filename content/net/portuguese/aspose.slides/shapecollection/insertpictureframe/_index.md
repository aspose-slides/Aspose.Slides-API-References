## ShapeCollection.InsertPictureFrame método

Cria um novo quadro de imagem contendo a imagem especificada e o insere na coleção de formas no índice especificado.

```csharp
public IPictureFrame InsertPictureFrame(int index, ShapeType shapeType, float x, float y, 
    float width, float height, IPPImage image)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| index | Int32 | O índice baseado em zero no qual inserir o quadro de imagem. |
| shapeType | ShapeType | Especifica o tipo de forma contido em [`ShapeType`](../../shapetype), exceto para todos os tipos de linhas: |
| x | Single | A coordenada x do quadro de imagem, em pontos. |
| y | Single | A coordenada y do quadro de imagem, em pontos. |
| width | Single | A largura do quadro de imagem, em pontos. |
| height | Single | A altura do quadro de imagem, em pontos. |
| image | IPPImage | O [`IPPImage`](../../ippimage) a ser exibido no quadro de imagem. |

### Valor de Retorno

O [`IPictureFrame`](../../ipictureframe) recém-criado.

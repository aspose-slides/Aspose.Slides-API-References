## AddConnector(ShapeType, float, float, float, float) {#addconnector}

Cria uma nova forma de conector com estilo de modelo padrão e a adiciona ao final da coleção de formas.

```csharp
public IConnector AddConnector(ShapeType shapeType, float x, float y, float width, float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| shapeType | ShapeType | O [`ShapeType`](../../shapetype) da forma de conector a ser adicionada. |
| x | Single | A coordenada x da moldura do conector, em pontos. |
| y | Single | A coordenada y da moldura do conector, em pontos. |
| width | Single | A largura da moldura do conector, em pontos. |
| height | Single | A altura da moldura do conector, em pontos. |

### Valor de retorno

O [`IConnector`](../../iconnector) recém-criado.

### Exemplos

O exemplo a seguir mostra como adicionar um conector (um conector dobrado) entre duas formas (uma elipse e um retângulo) na Apresentação do PowerPoint.

```csharp
[C#]
// Instancia uma classe de apresentação que representa um arquivo PPTX
using (Presentation input = new Presentation())
{
    // Acessa a coleção de formas de um slide específico
    IShapeCollection shapes = input.Slides[0].Shapes;
    // Adiciona uma forma autoshape Elipse
    IAutoShape ellipse = shapes.AddAutoShape(ShapeType.Ellipse, 0, 100, 100, 100);
    // Adiciona uma forma autoshape Retângulo
    IAutoShape rectangle = shapes.AddAutoShape(ShapeType.Rectangle, 100, 300, 100, 100);
    // Adiciona uma forma de conector à coleção de formas do slide
    IConnector connector = shapes.AddConnector(ShapeType.BentConnector2, 0, 0, 10, 10);
    // Conecta as formas usando o conector
    connector.StartShapeConnectedTo = ellipse;
    connector.EndShapeConnectedTo = rectangle;
    // Chama reroute que define o caminho mais curto automático entre as formas
    connector.Reroute();
    // Salva a apresentação
    input.Save("Shapes-connector.pptx", SaveFormat.Pptx);
}
```

## AddConnector(ShapeType, float, float, float, float, bool) {#addconnector_1}

Cria uma nova forma de conector e a adiciona ao final da coleção de formas, opcionalmente aplicando o estilo de modelo padrão.

```csharp
public IConnector AddConnector(ShapeType shapeType, float x, float y, float width, float height, 
    bool createFromTemplate)
```

| Parameter | Type | Description |
| --- | --- | --- |
| shapeType | ShapeType | O [`ShapeType`](../../shapetype) da forma de conector a ser criada. |
| x | Single | A coordenada x da moldura do conector, em pontos. |
| y | Single | A coordenada y da moldura do conector, em pontos. |
| width | Single | A largura da moldura do conector, em pontos. |
| height | Single | A altura da moldura do conector, em pontos. |
| createFromTemplate | Boolean | True para aplicar o estilo de modelo padrão (nome não vazio, estilo simples); false para criar o conector com os valores padrão das propriedades. |

O [`IConnector`](../../iconnector) recém-criado.

## AddOleObjectFrame(float, float, float, float, IOleEmbeddedDataInfo) {#addoleobjectframe}

Cria um novo quadro de objeto OLE e o adiciona ao final da coleção de formas.

```csharp
public IOleObjectFrame AddOleObjectFrame(float x, float y, float width, float height, 
    IOleEmbeddedDataInfo dataInfo)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| x | Single | A coordenada x do novo quadro OLE, em pontos. |
| y | Single | A coordenada y do novo quadro OLE, em pontos. |
| width | Single | A largura do novo quadro OLE, em pontos. |
| height | Single | A altura do novo quadro OLE, em pontos. |
| dataInfo | IOleEmbeddedDataInfo | As informações sobre os dados OLE incorporados ([`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo)). |

### Valor de Retorno

O [`IOleObjectFrame`](../../ioleobjectframe) recém-criado.

### Exemplos

Os exemplos a seguir mostram como adicionar quadros de objeto OLE aos slides de uma apresentação do PowerPoint.

```csharp
[C#]
// Instancia a classe Presentation que representa o arquivo PPTX
using (Presentation pres = new Presentation())
{
    // Acessa o primeiro slide
    ISlide sld = pres.Slides[0];
    // Carrega um arquivo Excel para o fluxo
    MemoryStream mstream = new MemoryStream();
    using (FileStream fs = new FileStream("book1.xlsx", FileMode.Open, FileAccess.Read))
    {
        byte[] buf = new byte[4096];
        while (true)
        {
            int bytesRead = fs.Read(buf, 0, buf.Length);
            if (bytesRead <= 0)
                break;
            mstream.Write(buf, 0, bytesRead);
        }
    }
    // Cria um objeto de dados para incorporação
    IOleEmbeddedDataInfo dataInfo = new OleEmbeddedDataInfo(mstream.ToArray(), "xlsx");
    // Adiciona uma forma de quadro de objeto OLE
    IOleObjectFrame oleObjectFrame = sld.Shapes.AddOleObjectFrame(0, 0, pres.SlideSize.Size.Width,
        pres.SlideSize.Size.Height, dataInfo);
    // Grava o arquivo PPTX no disco
    pres.Save("OleEmbed_out.pptx", SaveFormat.Pptx);
}
```

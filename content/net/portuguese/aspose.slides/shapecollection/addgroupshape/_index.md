## AddGroupShape() {#addgroupshape}

Cria um novo **GroupShape** vazio e o adiciona ao final da coleção de shapes. A moldura do grupo será ajustada automaticamente para acomodar quaisquer shapes adicionados a ele.

```csharp
public IGroupShape AddGroupShape()
```

### Valor de retorno

O [`IGroupShape`](../../igroupshape) recém-criado.

### Exemplos

O exemplo a seguir mostra como adicionar um **GroupShape** a um slide de uma apresentação do PowerPoint.

```csharp
[C#]
// Instanciar a classe Presentation
using (Presentation pres = new Presentation())
{
    // Obter o primeiro slide
    ISlide sld = pres.Slides[0];
    // Acessando a coleção de shapes do slide
    IShapeCollection slideShapes = sld.Shapes;
    // Adicionando um group shape ao slide
    IGroupShape groupShape = slideShapes.AddGroupShape();
    // Adicionando shapes dentro do group shape adicionado
    groupShape.Shapes.AddAutoShape(ShapeType.Rectangle, 300, 100, 100, 100);
    groupShape.Shapes.AddAutoShape(ShapeType.Rectangle, 500, 100, 100, 100);
    groupShape.Shapes.AddAutoShape(ShapeType.Rectangle, 300, 300, 100, 100);
    groupShape.Shapes.AddAutoShape(ShapeType.Rectangle, 500, 300, 100, 100);
    // Definindo a moldura do group shape
    groupShape.Frame = new ShapeFrame(100, 300, 500, 40, NullableBool.False, NullableBool.False, 0);
    // Gravar o arquivo PPTX no disco
    pres.Save("GroupShape_out.pptx", SaveFormat.Pptx);
}
```

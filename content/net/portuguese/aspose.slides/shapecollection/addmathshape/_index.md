## ShapeCollection.AddMathShape método

Cria um novo shape retangular automático para hospedar conteúdo matemático e o adiciona ao final da coleção de shapes.

```csharp
public IAutoShape AddMathShape(float x, float y, float width, float height)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| x | Single | A coordenada x da moldura da forma, em pontos. |
| y | Single | A coordenada y da moldura da forma, em pontos. |
| width | Single | A largura da moldura da forma, em pontos. |
| height | Single | A altura da moldura da forma, em pontos. |

### Valor de Retorno

O [`IAutoShape`](../../iautoshape) recém-criado.

### Exemplos

O exemplo a seguir demonstra como adicionar Equação Matemática em uma Apresentação do PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
   IAutoShape mathShape = pres.Slides[0].Shapes.AddMathShape(0, 0, 720, 150);
   var mathParagraph = (mathShape.TextFrame.Paragraphs[0].Portions[0] as MathPortion).MathParagraph;
   var fraction = new MathematicalText("x").Divide("y");
   mathParagraph.Add(new MathBlock(fraction));
   var mathBlock = new MathematicalText("c")
        .SetSuperscript("2")
        .Join("=")
        .Join(new MathematicalText("a").SetSuperscript("2"))
        .Join("+")
        .Join(new MathematicalText("b").SetSuperscript("2"));
    mathParagraph.Add(mathBlock);
    pres.Save("math.pptx", SaveFormat.Pptx);
}
```

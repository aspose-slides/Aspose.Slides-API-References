## InsertOleObjectFrame(int, float, float, float, float, IOleEmbeddedDataInfo) {#insertoleobjectframe}

Cria um novo quadro de objeto OLE e o insere na coleção de formas no índice especificado.

```csharp
public IOleObjectFrame InsertOleObjectFrame(int index, float x, float y, float width, float height, 
    IOleEmbeddedDataInfo dataInfo)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| index | Int32 | O índice baseado em zero no qual inserir o quadro de objeto OLE. |
| x | Single | A coordenada x do novo quadro OLE, em pontos. |
| y | Single | A coordenada y do novo quadro OLE, em pontos. |
| width | Single | A largura do novo quadro OLE, em pontos. |
| height | Single | A altura do novo quadro OLE, em pontos. |
| dataInfo | IOleEmbeddedDataInfo | A informação de dados OLE incorporados ([`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo)). |

### Valor de Retorno

O [`IOleObjectFrame`](../../ioleobjectframe) recém-criado.

### Exemplos

Este exemplo demonstra a inserção de um objeto OLE no segundo índice:

```csharp
[C#]
byte[] fileData = File.ReadAllBytes("test.zip");
IOleDataInfo dataInfo = new OleDataInfo(fileData, "zip");
IOleObjectFrame oleObjectFrame = slidees.Shapes.AddOleObjectFrame(2, 150, 20, 50, 50, dataInfo);
```

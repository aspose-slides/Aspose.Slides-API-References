## AddZoomFrame(float, float, float, float, ISlide) {#addzoomframe}

Cria uma nova moldura de Zoom e a adiciona ao final da coleção de formas.

```csharp
public IZoomFrame AddZoomFrame(float x, float y, float width, float height, ISlide slide)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| x | Single | A coordenada x da nova moldura de Zoom, em pontos. |
| y | Single | A coordenada y da nova moldura de Zoom, em pontos. |
| width | Single | A largura da nova moldura de Zoom, em pontos. |
| height | Single | A altura da nova moldura de Zoom, em pontos. |
| slide | ISlide | O [`ISlide`](../../islide) referenciado pela moldura de Zoom; deve pertencer a esta apresentação. |

### Valor de Retorno

O [`IZoomFrame`](../../izoomframe) recém-criado.

### Exceções

| exceção | condição |
| --- | --- |
| ArgumentException | Lançada se o slide referenciado não pertencer à apresentação atual. |

### Exemplos

Este exemplo demonstra como adicionar um objeto Zoom ao final de uma coleção (assuma que existem pelo menos dois slides na apresentação "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
}
```

## AddZoomFrame(float, float, float, float, ISlide, IPPImage) {#addzoomframe_1}

Cria uma nova moldura de Zoom e a adiciona ao final da coleção de formas.

```csharp
public IZoomFrame AddZoomFrame(float x, float y, float width, float height, ISlide slide, 
    IPPImage image)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| x | Single | A coordenada x da nova moldura de Zoom, em pontos. |
| y | Single | A coordenada y da nova moldura de Zoom, em pontos. |
| width | Single | A largura da nova moldura de Zoom, em pontos. |
| height | Single | A altura da nova moldura de Zoom, em pontos. |
| slide | ISlide | O [`ISlide`](../../islide) referenciado pela moldura de Zoom; deve pertencer a esta apresentação. |
| image | IPPImage | A imagem para o slide referenciado [`IPPImage`](../../ippimage). |

### Valor de Retorno

O [`IZoomFrame`](../../izoomframe) recém-criado.

### Exceções

| exceção | condição |
| --- | --- |
| ArgumentException | Lançada se o slide referenciado não pertencer à apresentação atual. |

### Exemplos

Este exemplo demonstra como adicionar um objeto Zoom ao final de uma coleção (assuma que existem pelo menos dois slides na apresentação "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IPPImage image = pres.Images.AddImage(Image.FromFile("image.png"));
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1], image);
}
```

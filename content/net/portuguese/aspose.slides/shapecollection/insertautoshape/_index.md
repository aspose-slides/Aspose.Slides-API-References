## InsertAutoShape(int, ShapeType, float, float, float, float, bool) {#insertautoshape_1}

Cria uma nova forma automática e a insere na coleção de formas no índice especificado, opcionalmente inicializando-a com o estilo padrão do modelo.

```csharp
public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| index | Int32 | O índice baseado em zero no qual inserir a forma automática. |
| shapeType | ShapeType | O [`ShapeType`](../../shapetype) da forma automática a ser inserida. |
| x | Single | A coordenada x da moldura da forma, em pontos. |
| y | Single | A coordenada y da moldura da forma, em pontos. |
| width | Single | A largura da moldura da forma, em pontos. |
| height | Single | A altura da moldura da forma, em pontos. |
| createFromTemplate | Boolean | True para aplicar o estilo padrão do modelo (incluindo um nome não vazio, estilo simples e texto centralizado); false para criar a forma com todas as propriedades definidas para seus valores padrão. |

### Valor de retorno

A recém-criada [`IAutoShape`](../../iautoshape).

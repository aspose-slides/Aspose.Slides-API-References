## ISlideShowTransition.AdvanceAfter propriedade

Este atributo especifica se a apresentação de slides avançará para o próximo slide após um determinado tempo. Leitura/Escrita Boolean.

```csharp
public bool AdvanceAfter { get; set; }
```

### Exemplos

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    // Obtenha a primeira transição de slide
    ISlideShowTransition slideTransition = pres.Slides[0].SlideShowTransition;
    
    // Verifique se a flag Advance Slide After está marcada
    if (slideTransition.AdvanceAfter)
    {
        // Obtenha o valor Advance Slide After Time
        uint advanceAfterTime = slideTransition.AdvanceAfterTime;
    }
}
```

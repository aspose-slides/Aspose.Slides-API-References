## ISlideShowTransition.SoundIsBuiltIn propriedade

Especifica se este som é um som interno ou não. Se este atributo for definido como true, a aplicação que gera o documento é notificada para verificar o atributo name especificado para este som na sua lista de sons internos e pode então exibir um nome personalizado ou uma interface de usuário conforme necessário.

## Propriedade ISlideShowTransition.Duration

Obtém ou define a duração do efeito de transição do slide em milissegundos. Leitura/gravação Int32.

```csharp
public int Duration { get; set; }
```

### Observações

Corresponde ao atributo `p14:dur` do elemento `p:transition` no esquema PresentationML. Se não for definido, a duração é determinada automaticamente com base na propriedade [`Speed`](../speed) e no tipo de transição.

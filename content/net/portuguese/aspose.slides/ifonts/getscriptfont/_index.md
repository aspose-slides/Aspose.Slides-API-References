## IFonts.GetScriptFont método

Obtém o nome da fonte associado a uma tag de script específica do tema da apresentação.

```csharp
public string GetScriptFont(string script)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| script | String | O código de script BCP-47 (por exemplo, "Latn", "Cyrl", "Jpan") usado para identificar um sistema de escrita. |

### Valor de Retorno

O nome da fonte usada para o script especificado, ou `null` se o script não estiver definido.

### Exemplos

Este exemplo demonstra como recuperar a fonte atribuída ao script cirílico no tema da apresentação.

```csharp
[C#]
string font = presentation.MasterTheme.FontScheme.Major.GetScriptFont("Cyrl");
Console.WriteLine("Font for Cyrillic script: " + font);
```

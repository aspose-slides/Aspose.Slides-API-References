## IDocumentProperties.ScaleCrop propriedade

Indica o modo de exibição da miniatura do documento. Defina este elemento como **true** para habilitar o dimensionamento da miniatura do documento para a exibição. Defina este elemento como **false** para habilitar o recorte da miniatura do documento para mostrar apenas as seções que se ajustam à exibição. Leitura/gravação Boolean.

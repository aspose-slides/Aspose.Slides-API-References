## IDocumentProperties.LinksUpToDate propriedade

Indica se os hyperlinks em um documento estão atualizados. Defina este elemento como **true** para indicar que os hyperlinks foram atualizados. Defina este elemento como **false** para indicar que os hyperlinks estão desatualizados. Leitura/Gravação Boolean.

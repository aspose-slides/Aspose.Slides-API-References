## IDocumentProperties.AppVersion propriedade

Retorna a versão do aplicativo. Somente leitura String.

```csharp
public string AppVersion { get; }
```

### Observações

O conteúdo deste elemento deve estar no formato XX.YYYY, onde X e Y representam valores numéricos; caso contrário, o documento será considerado não conforme. Aspose.Slides representa sua versão no formato XX.YYZZ, onde: XX - versão principal YY - versão secundária ZZ - versão de correção Por exemplo, o valor 23.0105 significa a versão Aspose.Slides 23.1.5.

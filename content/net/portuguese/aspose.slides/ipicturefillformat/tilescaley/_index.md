## IPictureFillFormat.TileScaleY propriedade

Retorna ou define a escala vertical para o preenchimento de textura como uma porcentagem. Leitura/gravação Single.

```csharp
public float TileScaleY { get; set; }
```

### Exemplos

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Obtém o formato de preenchimento de imagem da forma
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Define o modo de preenchimento de imagem como Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Define a escala vertical da textura para 120 por cento
    pictureFillFormat.TileScaleY = 120;
}
```

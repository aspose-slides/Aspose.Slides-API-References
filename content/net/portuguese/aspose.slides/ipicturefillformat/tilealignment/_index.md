## IPictureFillFormat.TileAlignment propriedade

Retorna ou define como a textura está alinhada dentro da forma. Esta configuração controla o ponto inicial do padrão de textura e como ele se repete ao longo da forma. Leitura/Gravação [`RectangleAlignment`](../../rectanglealignment).

```csharp
public RectangleAlignment TileAlignment { get; set; }
```

### Observações

O padrão é TopLeft.

### Exemplos

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Obtém o formato de preenchimento de imagem da forma
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Define o modo de preenchimento de imagem como Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Define o alinhamento do ladrilho para o canto inferior direito
    pictureFillFormat.TileAlignment = RectangleAlignment.BottomRight;
}
```

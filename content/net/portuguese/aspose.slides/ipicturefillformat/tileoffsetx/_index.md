## IPictureFillFormat.TileOffsetX propriedade

Obtém ou define o deslocamento horizontal da textura a partir da origem da forma em pontos. Um valor positivo move a textura para a direita, enquanto um valor negativo a move para a esquerda. Leitura/gravação Single.

```csharp
public float TileOffsetX { get; set; }
```

### Exemplos

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Obtém o formato de preenchimento de imagem da forma
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Define o modo de preenchimento de imagem como Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Define o deslocamento horizontal da textura para 20 pontos
    pictureFillFormat.TileOffsetX = 20f;
}
```

## IPictureFillFormat.TileFlip propriedade

Inverte o ladrilho de textura ao redor de seu eixo horizontal, vertical ou ambos. Leitura/gravação [`TileFlip`](../../tileflip).

```csharp
public TileFlip TileFlip { get; set; }
```

### Observações

O padrão é NoFlip.

### Exemplos

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Obtém o formato de preenchimento de imagem da forma
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Define o modo de preenchimento de imagem como Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Inverte o ladrilho de textura ao redor de seu eixo vertical.
    pictureFillFormat.TileFlip = TileFlip.FlipY;
}
```

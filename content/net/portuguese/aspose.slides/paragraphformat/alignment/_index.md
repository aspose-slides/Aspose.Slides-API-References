## ParagraphFormat.Alignment propriedade

Retorna ou define o alinhamento de texto em um parágrafo sem herança. Leitura/Gravação [`TextAlignment`](../../textalignment).


### Exemplos

O código de exemplo a seguir mostra como Alinhar Parágrafos de Texto em uma Apresentação do PowerPoint.

```csharp
[C#]
// Instanciar um objeto Presentation que representa um arquivo PPTX
using (Presentation pres = new Presentation("ParagraphsAlignment.pptx"))
{
    // Acessando o primeiro slide
    ISlide slide = pres.Slides[0];
    // Acessando o primeiro e o segundo placeholder no slide e convertendo para AutoShape
    ITextFrame tf1 = ((IAutoShape)slide.Shapes[0]).TextFrame;
    ITextFrame tf2 = ((IAutoShape)slide.Shapes[1]).TextFrame;
    // Alterar o texto em ambos os placeholders
    tf1.Text = "Center Align by Aspose";
    tf2.Text = "Center Align by Aspose";
    // Obtendo o primeiro parágrafo dos placeholders
    IParagraph para1 = tf1.Paragraphs[0];
    IParagraph para2 = tf2.Paragraphs[0];
    // Alinhando o parágrafo de texto ao centro
    para1.ParagraphFormat.Alignment = TextAlignment.Center;
    para2.ParagraphFormat.Alignment = TextAlignment.Center;
    //Gravando a apresentação como um arquivo PPTX
    pres.Save("Centeralign_out.pptx", SaveFormat.Pptx);
}
```

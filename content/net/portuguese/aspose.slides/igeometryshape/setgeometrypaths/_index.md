## IGeometryShape.SetGeometryPaths método

Atualiza a geometria da forma a partir de um array de [`IGeometryPath`](../../igeometrypath). As coordenadas devem ser relativas ao canto superior esquerdo da forma. Altera o tipo da forma ([`ShapeType`](../shapetype)) para Custom.

```csharp
public void SetGeometryPaths(IGeometryPath[] geometryPaths)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| geometryPaths | IGeometryPath[] | Array de caminhos de geometria |

### Exceções

| exceção | condição |
| --- | --- |
| ArgumentException | Nenhum caminho encontrado |
| ArgumentException | Caminho vazio |

### Exemplos

Exemplo:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    GeometryShape shape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 200, 100) as GeometryShape;

    GeometryPath geometryPath = shape.GetGeometryPaths()[0];

    geometryPath.LineTo(100, 50, 1);
    geometryPath.LineTo(100, 50, 4);

    shape.SetGeometryPath(geometryPath);

    pres.Save("output.pptx", SaveFormat.Pptx);
}
```

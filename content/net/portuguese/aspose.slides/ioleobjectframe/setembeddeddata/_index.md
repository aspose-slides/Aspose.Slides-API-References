## IOleObjectFrame.SetEmbeddedData method

Define informações sobre dados incorporados OLE.

```csharp
public void SetEmbeddedData(IOleEmbeddedDataInfo embeddedData)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| embeddedData | IOleEmbeddedDataInfo | Dados incorporados [`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo) |

### Exceções

| Exceção | Condição |
| --- | --- |
| ArgumentNullException | Quando o parâmetro embeddedData for nulo. |

### Observações

Este método altera as propriedades do objeto para refletir os novos dados e define a bandeira IsObjectLink como false, indicando que o objeto OLE está incorporado.

### Exemplos

O exemplo a seguir demonstra como alterar os dados incorporados OLE e seu tipo para um objeto [`IOleObjectFrame`](../../ioleobjectframe) existente

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    OleObjectFrame oof = pres.Slides[0].Shapes[0] as OleObjectFrame;
    if (oof != null)
    {
        IOleEmbeddedDataInfo newData = new OleEmbeddedDataInfo(File.ReadAllBytes("Picture.png"), "png");
        oof.SetEmbeddedData(newData);
    }
}
```

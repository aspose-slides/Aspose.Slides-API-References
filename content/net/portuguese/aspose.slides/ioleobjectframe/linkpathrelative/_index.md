## IOleObjectFrame.LinkPathRelative propriedade

Retorna o caminho relativo para um arquivo vinculado, se presente, caso contrário retorna uma string vazia. Somente leitura String.

```csharp
public string LinkPathRelative { get; }
```

### Observações

Nas apresentações Ppt, alguns links de objetos Ole podem ter uma representação relativa.

### Exemplos

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.ppt"))
{
    IOleObjectFrame oleFrame = presentation.Slides[0].Shapes[0] as IOleObjectFrame;

    if (oleFrame != null)
    {
        Console.WriteLine("The relative path: " + oleFrame.LinkPathRelative);
    } 
}
```

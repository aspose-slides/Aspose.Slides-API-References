## Ink.InkEffectImages propriedade

Obtém a coleção de imagens personalizadas usadas para simular efeitos visuais para pincéis de tinta. Essas imagens são usadas ao renderizar tinta com valores específicos de [`InkEffectType`](../../inkeffecttype), como Galaxy, Rainbow, etc. Ao fornecer suas próprias imagens, você pode controlar como cada efeito de tinta aparece.

```csharp
public static IDictionary<InkEffectType, IImage> InkEffectImages { get; }
```

### Observações

Esta propriedade permite substituir as texturas padrão de efeito de tinta por definições do usuário, o que é particularmente útil quando os recursos padrão são restritos por licenciamento ou indisponíveis em tempo de execução. Cada entrada no dicionário deve associar um valor [`InkEffectType`](../../inkeffecttype) a um objeto [`IImage`](../../../aspose.slides/iimage) correspondente (por exemplo, Bitmap ou uma interface de imagem Aspose).

### Exemplos

```csharp
[C#]
IImage image = Images.FromFile("image.png");
Ink.InkEffectImages.Add(InkEffectType.Galaxy, image);
```

## IInkBrush.InkEffect propriedade

Obtém o tipo de efeito de tinta (por exemplo, Galaxy, Gold, Silver) que define o estilo visual da pincelada de tinta. O valor é analisado a partir da propriedade de pincel "inkEffects". Se nenhum efeito reconhecido for especificado, NotDefined é retornado.

```csharp
public InkEffectType InkEffect { get; }
```

### Exemplos

Exemplo:

```csharp
[C#]
using (var pres = new Presentation("Presentation.pptx"))
{
    Ink ink = pres.Slides[0].Shapes[0] as Ink;
    IInkBrush brush = ink.Traces[0].Brush;
    Console.WriteLine("InkEffects = {0}", brush.InkEffect);
}
```

## SlideUtil.FindAndReplaceText método

Localiza e substitui texto na apresentação com o formato especificado

```csharp
public static void FindAndReplaceText(IPresentation presentation, bool withMasters, string find, 
    string replace, PortionFormat format = null)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| presentation | IPresentation | Apresentação analisada. |
| withMasters | Boolean | Determina se os slides mestre devem ser analisados. |
| find | String | Valor de string a ser localizado. |
| replace | String | Valor de string a ser substituído. |
| format | PortionFormat | Formato para substituir a parte do texto. Se null, será usado o formato do primeiro caractere da string encontrada |

### Exemplos

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PortionFormat format = new PortionFormat
    {
        FontHeight = 24f,
        FontItalic = NullableBool.True,
        FillFormat =
        {
            FillType = FillType.Solid,
            SolidFillColor =
            {
                Color = Color.Red
            }
        }
    };
    Aspose.Slides.Util.SlideUtil.FindAndReplaceText(pres, true, "[this block] ", "my text ", format);
    pres.Save("replaced", SaveFormat.Pptx);
}
```

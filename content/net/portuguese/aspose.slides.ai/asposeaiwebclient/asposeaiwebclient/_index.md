## AsposeAIWebClient() {#constructor}

Cria uma instância do cliente web Aspose AI que se conecta ao endpoint padrão do Aspose LLM. Este é o cliente usado pelo construtor sem parâmetros [`SlidesAIAgent`](../../slidesaiagent/slidesaiagent), portanto criá-lo explicitamente só é necessário ao passar o cliente diretamente para o construtor [`SlidesAIAgent`](../../slidesaiagent/slidesaiagent).

## AsposeAIWebClient(string, HttpClient) {#constructor_3}

Cria uma instância do cliente web Aspose AI que se conecta a uma URL de endpoint personalizada usando um HttpClient gerenciado externamente. O HttpClient fornecido não é descartado por esta instância e continua sendo de propriedade do chamador. Use esta sobrecarga quando você tem uma URL fornecida pela equipe Aspose.Slides e deseja fornecer seu próprio HttpClient; se você só precisa do seu próprio HttpClient com a URL padrão, use a sobrecarga [`AsposeAIWebClient`](../asposeaiwebclient) em vez disso.

```csharp
public AsposeAIWebClient(string url, HttpClient httpClient)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| url | String | URL do endpoint do Aspose LLM, fornecida pela equipe Aspose.Slides. |
| httpClient | HttpClient | Uma instância de HttpClient gerenciada externamente. |

### Exceções

| Exceção | Condição |
| --- | --- |
| ArgumentNullException | URL não pode ser nula ou vazia. |
| ArgumentNullException | Instância de HttpClient não foi fornecida. |

## AsposeAIWebClient classe

Uma implementação integrada [`IAIWebClient`](../iaiwebclient) que se conecta ao próprio LLM da Aspose. Este é o cliente padrão usado pelo construtor sem parâmetros [`SlidesAIAgent`](../slidesaiagent/slidesaiagent).

```csharp
public sealed class AsposeAIWebClient : IAIWebClient, IDisposable
```

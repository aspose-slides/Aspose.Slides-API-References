## IOverrideThemeManager.IsOverrideThemeEnabled propriedade

Determina se OverrideTheme substitui o tema efetivo herdado ou não. Para habilitar OverrideTheme para substituição, use os métodos OverrideTheme.Init*(). Para desabilitar OverrideTheme da substituição, use o método OverrideTheme.Clear(). Boolean somente leitura.

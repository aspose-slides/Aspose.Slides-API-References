## BaseOverrideThemeManager.IsOverrideThemeEnabled propriedade

Determina se OverrideTheme substitui o tema efetivo herdado ou não. Para habilitar OverrideTheme para substituição, use os métodos OverrideTheme.Init*(). Para desabilitar OverrideTheme de substituir, use o método OverrideTheme.Clear(). Boolean somente leitura.

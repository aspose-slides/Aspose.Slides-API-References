## BehaviorCollection.CopyTo método

Copia os elementos do ICollection para um Array, começando em um índice de Array específico.

```csharp
public void CopyTo(IBehavior[] array, int arrayIndex)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| array | IBehavior[] | O Array unidimensional que é o destino dos elementos copiados do ICollection. O Array deve ter indexação baseada em zero. |
| arrayIndex | Int32 | O índice baseado em zero em *array* onde a cópia começa. |

### Exceções

| exceção | condição |
| --- | --- |
| ArgumentNullException | *array* é nulo. |
| ArgumentOutOfRangeException | *arrayIndex* é menor que 0. |
| ArgumentException | O número de elementos no ICollection de origem é maior que o espaço disponível a partir de *arrayIndex* até o final do *array* de destino. |

## IEffect.DelayBetweenTextParts propriedade

Define um atraso entre partes de texto animadas (palavras ou letras). Um valor positivo especifica a percentagem da duração do efeito. Um valor negativo especifica o atraso em segundos. Leitura/gravação Single.

```csharp
public float DelayBetweenTextParts { get; set; }
```

### Exemplos

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Obtém o primeiro efeito do primeiro slide.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Altera o tipo de animação de texto do efeito para "By word"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByWord;
    
    // Define o atraso entre as partes de texto animadas para 20% da duração do efeito.
    firstSlideEffect.DelayBetweenTextParts = 20f;
}
```

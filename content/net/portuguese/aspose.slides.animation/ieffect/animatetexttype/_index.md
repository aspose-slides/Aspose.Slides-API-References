## IEffect.AnimateTextType propriedade

Define um tipo de animação de texto para o efeito. O texto da forma pode ser animado por letra, por palavra ou tudo de uma vez. Leitura/Gravação `AnimateTextType`.

```csharp
public AnimateTextType AnimateTextType { get; set; }
```

### Exemplos

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Obtenha o primeiro efeito do primeiro slide.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Altere o tipo de animação de texto do efeito para "By letter"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByLetter;
}
```

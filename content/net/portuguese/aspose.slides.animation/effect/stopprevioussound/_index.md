## Effect.StopPreviousSound propriedade

Este atributo especifica se o efeito de animação interrompe o som anterior. Leitura/gravação Boolean.

```csharp
public bool StopPreviousSound { get; set; }
```

### Exemplos

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Obtenha o primeiro efeito do primeiro slide.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Obtenha o primeiro efeito do segundo slide.
    IEffect secondSlideEffect = presentation.Slides[1].Timeline.MainSequence[0];
       
    if (firstSlideEffect.Sound != null)
    {
        // Altere o segundo efeito Enhancements/Sound para "Stop Previous Sound"
        secondSlideEffect.StopPreviousSound = true;
    }
}
```

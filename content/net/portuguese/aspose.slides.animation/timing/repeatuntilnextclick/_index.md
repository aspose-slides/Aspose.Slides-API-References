## Timing.RepeatUntilNextClick propriedade

Este atributo especifica se o efeito será repetido até o próximo clique. Leitura/gravação Boolean.

```csharp
public bool RepeatUntilNextClick { get; set; }
```

### Exemplos

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Obter a sequência de efeitos do primeiro slide
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Obter o primeiro efeito da sequência principal.
    IEffect effect = effectsSequence[0];

    // Alterar o Timing/Repeat do efeito para "Until Next Click"
    effect.Timing.RepeatUntilNextClick = true;
}
```

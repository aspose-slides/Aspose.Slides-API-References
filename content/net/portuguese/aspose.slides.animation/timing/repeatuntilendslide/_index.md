## Timing.RepeatUntilEndSlide propriedade

Este atributo especifica se o efeito será repetido até o final do slide. Leitura/Gravação Boolean.

```csharp
public bool RepeatUntilEndSlide { get; set; }
```

### Exemplos

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Obtenha a sequência de efeitos do primeiro slide
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Obtenha o primeiro efeito da sequência principal.
    IEffect effect = effectsSequence[0];

    // Altere o Timing/Repeat do efeito para "Até o final do slide"
    effect.Timing.RepeatUntilEndSlide = true;
}
```

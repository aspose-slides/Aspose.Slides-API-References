## ITiming.RepeatUntilNextClick propriedade

Este atributo especifica se o efeito será repetido até o próximo clique. Boolean de leitura/gravação.

```csharp
public bool RepeatUntilNextClick { get; set; }
```

### Exemplos

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Obtém a sequência de efeitos para o primeiro slide
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Obtém o primeiro efeito da sequência principal.
    IEffect effect = effectsSequence[0];

    // Altera o Timing/Repeat do efeito para "Until Next Click"
    effect.Timing.RepeatUntilNextClick = true;
}
```

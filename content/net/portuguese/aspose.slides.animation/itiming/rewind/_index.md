## ITiming.Rewind propriedade

Este atributo especifica se o efeito será rebobinado ao terminar a reprodução. Leitura/Escrita Boolean.

```csharp
public bool Rewind { get; set; }
```

### Exemplos

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Obtém a sequência de efeitos do primeiro slide
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Obtém o primeiro efeito da sequência principal.
    IEffect effect = effectsSequence[0];

    // Ativa o Timing/Rewind do efeito.
    effect.Timing.Rewind = true;
}
```

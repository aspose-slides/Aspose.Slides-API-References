## SlideShowTransition.AdvanceAfter propriedade

Este atributo especifica se a apresentação de slides avançará para o próximo slide após um determinado tempo. Leitura/gravação Boolean.

```csharp
public bool AdvanceAfter { get; set; }
```

### Exemplos

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    // Obtém a primeira transição de slide
    ISlideShowTransition slideTransition = pres.Slides[0].SlideShowTransition;
    
    // Verifica se a bandeira Advance Slide After está marcada
    if (slideTransition.AdvanceAfter)
    {
        // Obtém o valor de Advance Slide After Time
        uint advanceAfterTime = slideTransition.AdvanceAfterTime;
    }
}
```

## IMathMatrix.RowGap propriedade

O valor do espaçamento vertical entre linhas de uma matriz; Se o RowGapRule for definido como 3 ("Exactly"), então a unidade é interpretada como twips (1/20 de ponto) Se o RowGapRule for definido como 4 ("Multiple"), então a unidade é interpretada como meia-linha. Padrão: 0

```csharp
public uint RowGap { get; set; }
```

### Exemplos

Exemplo:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

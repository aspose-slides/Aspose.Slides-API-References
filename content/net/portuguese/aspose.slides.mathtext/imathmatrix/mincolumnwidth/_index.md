## IMathMatrix.MinColumnWidth propriedade

Largura mínima da coluna em twips (1/20 de um ponto). O espaçamento de lacuna (também referido como “Column Gap” ou “Gap Width”) é adicionado ao MinColumnWidth para determinar o espaçamento total da coluna da Matrix (distância entre as mesmas bordas de colunas diferentes). Padrão: 0.

```csharp
public uint MinColumnWidth { get; set; }
```

### Exemplos

Exemplo:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.MinColumnWidth = 20;
```

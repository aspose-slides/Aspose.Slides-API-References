## MathArray.RowSpacing propriedade

Espaçamento entre linhas de um array. É usado somente quando RowSpacingRule está definido como Exactly, caso em que a unidade de medida são pontos, ou Multiple, caso em que a unidade de medida são meias linhas. Padrão: 0

```csharp
public uint RowSpacing { get; set; }
```

### Exemplos

Exemplo:

```csharp
[C#]
IMathArray mathArray = new MathArray(new MathematicalText("item1"));
mathArray.RowSpacingRule = MathRowSpacingRule.Exactly;
mathArray.RowSpacing = 10;
```

## MathBox.AlignmentPoint propriedade

Quando verdadeiro, este emulador de operador funciona como um ponto de alinhamento; ou seja, pontos de alinhamento designados em outras equações podem ser alinhados a ele. Padrão: false

```csharp
public bool AlignmentPoint { get; set; }
```

### Exemplos

Exemplo:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.AlignmentPoint = true;
```

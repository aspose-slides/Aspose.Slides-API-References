## MathBox.OperatorEmulator propriedade

Emulador de Operador. Quando verdadeiro, a caixa e seu conteúdo comportam-se como um único operador e herdam as propriedades de um operador. Isso significa, por exemplo, que o caractere pode servir como ponto para quebra de linha e pode ser alinhado a outros operadores. Emuladores de Operador são frequentemente usados quando um ou mais glifos se combinam para formar um operador, como '=='. Valor padrão: false

```csharp
public bool OperatorEmulator { get; set; }
```

### Exemplos

Exemplo:

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("=="));
box.OperatorEmulator = true;
```

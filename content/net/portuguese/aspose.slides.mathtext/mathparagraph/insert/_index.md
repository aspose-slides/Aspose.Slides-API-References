## MathParagraph.Insert método

Insere IMathBlock na coleção no índice especificado.

```csharp
public void Insert(int index, IMathBlock mathBlock)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| index | Int32 | O índice baseado em zero no qual um item deve ser inserido. |
| mathBlock | IMathBlock | O IMathBlock a ser inserido. |

### Exemplos

Exemplo:

```csharp
[C#]
IAutoShape shape = slide.Shapes.AddMathShape(x, y, width, height);
IMathParagraph mathParagraph = (shape.TextFrame.Paragraphs[0].Portions[0] as MathPortion).MathParagraph;

mathParagraph.Insert(0, block);
```

## MathParagraph.Remove método

Remove a primeira ocorrência de um objeto específico da coleção/>.

```csharp
public bool Remove(IMathBlock mathBlock)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| mathBlock | IMathBlock | O objeto a ser removido da coleção. |

### Valor de retorno

true se *mathBlock* foi removido com sucesso da coleção; caso contrário, false. Este método também retorna false se *mathBlock* não for encontrado na coleção original/>.

### Exemplos

Exemplo:

```csharp
[C#]
IAutoShape shape = slide.Shapes.AddMathShape(x, y, width, height);
IMathParagraph mathParagraph = (shape.TextFrame.Paragraphs[0].Portions[0] as MathPortion).MathParagraph;
mathParagraph.Add(new MathBlock(new MathematicalText("x")));
IMathBlock block = new MathBlock(new MathematicalText("y"));
mathParagraph.Add(block);
mathParagraph.Remove(block);
```

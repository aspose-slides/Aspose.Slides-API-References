## MathBorderBox.StrikethroughTopLeftToBottomRight propriedade

Riscado da diagonal superior esquerda para inferior direita (o padrão é false). Especifica o estado oculto ou visível de uma linha diagonal de riscado do canto superior esquerdo ao canto inferior direito da caixa de borda.

```csharp
public bool StrikethroughTopLeftToBottomRight { get; set; }
```

### Exemplos

Exemplo:

```csharp
[C#]
MathBorderBox borderBox = new MathBorderBox(new MathematicalText("x"));
borderBox.StrikethroughTopLeftToBottomRight = true;
```

## MathGroupingCharacter.VerticalJustification propriedade

Justificação vertical do caractere de agrupar. Especifica o alinhamento do objeto em relação à linha de base. Por exemplo, quando o caractere de agrupar está acima do objeto, VerticalJustification de Top indica que a parte superior do objeto fica na linha de base; quando VerticalJustification está definido como Bottom, a parte inferior do objeto está na linha de base Padrão: Bottom para Position=Top, e Top para Position=Bottom

```csharp
public MathTopBotPositions VerticalJustification { get; set; }
```

### Exemplos

Exemplo:

```csharp
[C#]
MathGroupingCharacter groupingCharacter = new MathGroupingCharacter(new MathematicalText("abc"));
groupingCharacter.VerticalJustification = MathTopBotPositions.Top;
```

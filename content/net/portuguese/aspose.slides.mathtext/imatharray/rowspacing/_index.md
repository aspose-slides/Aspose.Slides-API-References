## IMathArray.RowSpacing propriedade

Espaçamento entre linhas de uma matriz. É usado apenas quando RowSpacingRule está definido como 3 Exactly, caso em que a unidade de medida é pontos, ou Multiple, caso em que a unidade de medida é meia linha. Padrão: 0

```csharp
public uint RowSpacing { get; set; }
```

### Exemplos

Exemplo:

```csharp
[C#]
IMathArray mathArray = new MathArray(new MathematicalText("item1"));
mathArray.RowSpacingRule = MathRowSpacingRule.Exactly;
mathArray.RowSpacing = 10;
```

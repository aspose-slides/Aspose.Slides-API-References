## IMathElementCollection.Remove método

Remove the first occurrence of a specific object from the collection.

```csharp
public bool Remove(IMathElement item)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| item | IMathElement | O objeto a ser removido da coleção. |

### Valor de retorno

true se *item* for removido com sucesso da coleção; caso contrário, false. Este método também retorna false se *item* não for encontrado na coleção original.

### Exemplos

Exemplo:

```csharp
[C#]
IMathElementCollection collection = new MathBlock(new MathematicalText("x"));
MathematicalText plusElement = new MathematicalText("+");
collection.Add(plusElement);
collection.Add(new MathRadical(new MathematicalText("x"), new MathematicalText("3")));
collection.Remove(plusElement);
```

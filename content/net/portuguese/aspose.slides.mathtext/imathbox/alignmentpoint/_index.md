## IMathBox.AlignmentPoint propriedade

Quando true, este emulador de operador atua como um ponto de alinhamento; ou seja, pontos de alinhamento designados em outras equações podem ser alinhados com ele. Padrão: false

```csharp
public bool AlignmentPoint { get; set; }
```

### Exemplos

Exemplo:

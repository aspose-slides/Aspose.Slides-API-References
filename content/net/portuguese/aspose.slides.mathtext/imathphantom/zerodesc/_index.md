## IMathPhantom.ZeroDesc propriedade

Obtém ou define um valor que indica se a descida (profundidade abaixo da linha de base) do elemento base deve ser tratada como zero.

```csharp
public bool ZeroDesc { get; set; }
```

### Observações

When `true`, the phantom does not lower the baseline of the surrounding math line. Corresponds to the OMML attribute `m:zeroDesc`.

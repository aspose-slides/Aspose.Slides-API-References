## IMathPhantom.Transp propriedade

Obtém ou define um valor que indica se o phantom é transparente para regras de espaçamento baseadas em classe.

```csharp
public bool Transp { get; set; }
```

### Observações

Quando `true`, operadores e símbolos dentro do phantom ainda afetam o espaçamento matemático ao redor do phantom (como se fosse visível). Quando `false`, o espaçamento baseado em classe é ignorado. Corresponde ao atributo OMML `m:transp`.

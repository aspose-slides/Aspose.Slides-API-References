## MathDelimiter.EndingCharacter propriedade

Delimiter Ending Character especifica o caractere delimitador de término, ou de fechamento. Delimitadores matemáticos são caracteres de envolvimento, como parênteses, colchetes e chaves. O padrão: ')'.

```csharp
public char EndingCharacter { get; set; }
```

### Exemplos

Exemplo:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Join("y").Enclose();
delimiter.EndingCharacter = ']';
```

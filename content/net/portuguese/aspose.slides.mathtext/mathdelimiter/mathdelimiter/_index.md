## Construtor MathDelimiter

Inicializa MathDelimiter com o elemento especificado como argumento base único

```csharp
public MathDelimiter(IMathElement element)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| element | IMathElement | O elemento base ao qual o delimitador é aplicado. Pode ser nulo. |

### Exceções

| exceção | condição |
| --- | --- |
| InvalidOperationException | Lança quando *element* é um contêiner para outros elementos, como MathBlock. Nesse caso, você precisa chamar um construtor diferente com argumento IEnumerable. |

### Exemplos

Exemplo:

```csharp
[C#]
IMathElement element = new MathematicalText("x");
MathDelimiter delimiter = new MathDelimiter(element);
```

## MathPhantom.ZeroDesc propriedade

Obtém ou define um valor que indica se a descida (profundidade abaixo da linha de base) do elemento base deve ser tratada como zero.

```csharp
public bool ZeroDesc { get; set; }
```

### Observações

Quando `true`, o fantasma não abaixa a linha de base da linha matemática ao redor. Corresponde ao atributo OMML `m:zeroDesc`.

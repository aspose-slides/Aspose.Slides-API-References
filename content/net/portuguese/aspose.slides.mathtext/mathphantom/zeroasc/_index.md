## MathPhantom.ZeroAsc propriedade

Obtém ou define um valor que indica se a ascendente (altura acima da linha de base) do elemento base deve ser tratada como zero.

```csharp
public bool ZeroAsc { get; set; }
```

### Observações

Quando `true`, o fantasma não eleva a linha de base da linha matemática circundante. Corresponde ao atributo OMML `m:zeroAsc`.

## IMathDelimiter.EndingCharacter propriedade

Delimiter Ending Character especifica o caractere delimitador de finalização, ou de fechamento. Delimitadores matemáticos são caracteres de fechamento, como parênteses, colchetes e chaves. O padrão: ')'.

```csharp
public char EndingCharacter { get; set; }
```

### Exemplos

Exemplo:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Join("y").Enclose();
delimiter.EndingCharacter = ']';
```

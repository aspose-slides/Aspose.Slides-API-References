## IMathDelimiter.DelimiterShape propriedade

Especifica a forma dos delimitadores no objeto delimitador. Quando é MathDelimiterShape.Centered, os delimitadores são centralizados ao redor do eixo matemático do texto matemático e ainda são feitos para se ajustar à altura total de seu conteúdo. Quando é MathDelimiterShape.Match, sua altura e forma são alteradas para corresponder exatamente ao seu conteúdo.

```csharp
public MathDelimiterShape DelimiterShape { get; set; }
```

### Exemplos

Exemplo:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.DelimiterShape = MathDelimiterShape.Match;
```

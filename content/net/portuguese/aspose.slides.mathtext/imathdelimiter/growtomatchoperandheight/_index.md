## IMathDelimiter.GrowToMatchOperandHeight propriedade

Especifica o crescimento de BeginningCharacter, SeparatorCharacter, EndingCharacter. Quando true, os delimitadores crescem verticalmente para corresponder à altura do operando. O valor padrão é true

```csharp
public bool GrowToMatchOperandHeight { get; set; }
```

### Exemplos

Exemplo:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.GrowToMatchOperandHeight = false;
```

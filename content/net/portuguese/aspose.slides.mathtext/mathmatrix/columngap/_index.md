## MathMatrix.ColumnGap propriedade

O valor do espaçamento horizontal entre as colunas de uma matriz; se a ColumnGapRule for definida como 3 ("Exactly"), a unidade é interpretada como twips (1/20 de um ponto). Se a ColumnGapRule for definida como 4 ("Multiple"), a unidade é interpretada como número de incrementos de 0.5 em. Em outros casos é ignorado. Padrão: 0

```csharp
public uint ColumnGap { get; set; }
```

### Exemplos

Exemplo:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.ColumnGapRule = MathSpacingRules.Exactly;
matrix.ColumnGap = 20;
```

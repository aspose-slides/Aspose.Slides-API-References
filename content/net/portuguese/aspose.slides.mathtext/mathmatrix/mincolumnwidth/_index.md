## MathMatrix.MinColumnWidth propriedade

Largura mínima da coluna em twips (1/20 de ponto). O espaçamento entre colunas (também chamado de “Column Gap” ou “Gap Width”) é adicionado ao MinColumnWidth para determinar o espaçamento total das colunas da Matriz (distância entre as mesmas bordas de colunas diferentes). Padrão: 0.

```csharp
public uint MinColumnWidth { get; set; }
```

### Exemplos

Exemplo:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.MinColumnWidth = 20;
```

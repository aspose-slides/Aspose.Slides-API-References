## MathMatrix.RowGap propriedade

O valor do espaçamento vertical entre as linhas de uma matriz; Se o RowGapRule estiver definido como 3 ("Exactly"), então a unidade é interpretada como twips (1/20 de um ponto) Se o RowGapRule estiver definido como 4 ("Multiple"), então a unidade é interpretada como meia-linha. Padrão: 0

```csharp
public uint RowGap { get; set; }
```

### Exemplos

Exemplo:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

## IMathBorderBox.StrikethroughTopLeftToBottomRight propriedade

Riscado superior esquerdo para inferior direito (padrão é false). Especifica o estado oculto ou exibido de uma linha diagonal de riscamento do canto superior esquerdo ao canto inferior direito da caixa de borda.

```csharp
public bool StrikethroughTopLeftToBottomRight { get; set; }
```

### Exemplos

Exemplo:

```csharp
[C#]
IMathBorderBox borderBox = new MathematicalText("x+y+z").ToBorderBox();
borderBox.StrikethroughTopLeftToBottomRight = true;
```

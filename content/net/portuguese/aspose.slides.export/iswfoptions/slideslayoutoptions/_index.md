## ISwfOptions.SlidesLayoutOptions propriedade

Obtém ou define o modo em que os slides são colocados na página ao exportar uma apresentação [`ISlidesLayoutOptions`](../../islideslayoutoptions). Esta propriedade não suporta a atribuição de objetos do tipo `Aspose.Slides.Export.HandoutLayoutingOptions`

```csharp
public ISlidesLayoutOptions SlidesLayoutOptions { get; set; }
```

### Exemplos

Exemplo:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    SwfOptions options = new SwfOptions
    {
        SlidesLayoutOptions = new NotesCommentsLayoutingOptions
        {
            CommentsPosition = CommentsPositions.Right
        }
    };
    
    pres.Save("pres.swf", SaveFormat.Swf, options);
}
```

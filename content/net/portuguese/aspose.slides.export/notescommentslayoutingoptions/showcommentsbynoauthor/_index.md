## NotesCommentsLayoutingOptions.ShowCommentsByNoAuthor propriedade

Obtém ou define a visibilidade dos comentários que não têm autor. Se verdadeiro, os comentários serão exibidos. (Aplica-se apenas se os comentários forem exibidos).

```csharp
public bool ShowCommentsByNoAuthor { get; set; }
```

### Observações

O valor padrão é **false**.

## PptxOptions.Zip64Mode propriedade

Especifica se o formato ZIP64 é usado para o documento Presentation. O valor padrão é IfNecessary

```csharp
public Zip64Mode Zip64Mode { get; set; }
```

### Exemplos

Exemplo:

```csharp

{
    pres.Save("demo-zip64.pptx", SaveFormat.Pptx, new PptxOptions()
```

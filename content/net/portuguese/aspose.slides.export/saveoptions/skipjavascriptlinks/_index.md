## SaveOptions.SkipJavaScriptLinks propriedade

Especifica se deve pular hyperlinks com chamadas JavaScript ao salvar a apresentação. Leitura/Gravação Boolean. O valor padrão é **false**.

```csharp
public bool SkipJavaScriptLinks { get; set; }
```

### Observações

Quando esta propriedade está definida como **true**, hyperlinks com chamadas JavaScript serão ignorados ao salvar.

Quando esta propriedade está definida como **false**, todos os hyperlinks serão salvos.

### Exemplos

Exemplo:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("result_without_JavaScript_links.html", SaveFormat.Html, new HtmlOptions()
    {
        SkipJavaScriptLinks = true
    });
}
```

## RenderingOptions.DisableFontLigatures propriedade

Obtém ou define um valor que indica se o texto é renderizado sem usar ligaduras. Quando definido como `true`, as ligaduras serão desativadas na saída renderizada. Por padrão, esta propriedade é definida como `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Exemplos

Exemplo:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    RenderingOptions options = new RenderingOptions
    {
        DisableFontLigatures = true // Desativar ligaduras na renderização de texto
    };
    
    Bitmap[] renderedSlides = pres.GetImage(options);
    for (var index = 0; index < renderedSlides.Length; index++)
    {
        var slideImage = renderedSlides[index];
        slideImage.Save($"slide-{index}.png");
    }
}
```

## Enumeração Zip64Mode

Especifica quando usar as extensões de formato ZIP64 para arquivos OpenXML.

```csharp
public enum Zip64Mode
```

### Valores

| Nome | Valor | Descrição |
| --- | --- | --- |
| Never | `0` | Não use extensões de formato ZIP64. |
| IfNecessary | `1` | Use extensões de formato ZIP64 se necessário. |
| Always | `2` | Sempre use extensões de formato ZIP64. |

### Observações

Um arquivo OpenXML é um arquivo ZIP que possui limite de 4 GB (2^32 bytes) para o tamanho descompactado de um arquivo, o tamanho compactado de um arquivo e o tamanho total do arquivamento, além de um limite de 65 535 (2^16-1) arquivos no arquivamento. As extensões de formato ZIP64 aumentam os limites para 2^64.

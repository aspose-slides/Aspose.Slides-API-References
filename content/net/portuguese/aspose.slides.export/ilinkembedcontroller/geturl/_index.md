## ILinkEmbedController.GetUrl método

Retorna um URL para um objeto externo. Este método sempre é chamado se [`GetObjectStoringLocation`](../getobjectstoringlocation) retornou Link e pode ser chamado se [`GetObjectStoringLocation`](../getobjectstoringlocation) retornou Embed mas a incorporação é impossível. Pode ser chamado várias vezes para o mesmo id de objeto.

```csharp
public string GetUrl(int id, int referrer)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| id | Int32 | Id do objeto. Esse id é exclusivo em toda a operação. |
| referrer | Int32 | Id do objeto referenciador ou 0, se o objeto for referenciado pelo documento raiz. Pode ser usado para gerar link relativo. |

### Valor de retorno

Url do objeto externo ou null se este objeto deve ser ignorado.

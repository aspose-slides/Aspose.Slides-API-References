### Exemplos

O exemplo a seguir mostra como converter apresentações para GIF animado usando configurações personalizadas.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    pres.Save("pres.gif", SaveFormat.Gif, new GifOptions
    {
        FrameSize = new Size(960, 720), // o tamanho do GIF resultante
        DefaultDelay = 2000, // por quanto tempo cada slide será exibido até ser alterado para o próximo
        TransitionFps = 35 // aumente o FPS para melhorar a qualidade da animação de transição
    });
}
```

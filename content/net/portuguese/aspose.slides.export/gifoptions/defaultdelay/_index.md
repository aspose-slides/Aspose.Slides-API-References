## GifOptions.DefaultDelay propriedade

Obtém ou define o tempo de atraso padrão [ms]. Este valor será usado se [`AdvanceAfterTime`](../../../aspose.slides/islideshowtransition/advanceaftertime) não for definido. O valor padrão é 1000.

```csharp
public int DefaultDelay { get; set; }
```

### Exemplos

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
	pres.Save("pres.gif", SaveFormat.Gif, new GifOptions { DefaultDelay = 2000 });
}
```

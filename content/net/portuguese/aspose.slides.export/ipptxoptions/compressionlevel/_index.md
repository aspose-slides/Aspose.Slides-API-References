## IPptxOptions.CompressionLevel propriedade

Especifica o nível de compressão usado ao salvar o documento de apresentação. O valor padrão é Level6.

```csharp
public CompressionLevel CompressionLevel { get; set; }
```

### Observações

Níveis de compressão mais altos produzem arquivos menores, mas requerem mais tempo de processamento. A taxa de compressão real depende do conteúdo da apresentação.

### Exemplos

Exemplo:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("demo-level8.pptx", SaveFormat.Pptx, new PptxOptions()
    {
        CompressionLevel = CompressionLevel.Level8
    });
}
```

## IInkOptions.HideInk propriedade

Exibe ou oculta elementos Ink no documento exportado.

```csharp
public bool HideInk { get; set; }
```

### Observações

O valor padrão é false.

### Exemplos

O próximo exemplo demonstra como ocultar elementos Ink em um documento PDF exportado:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.HideInk = true;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

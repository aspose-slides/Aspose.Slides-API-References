## Enumeração PdfAccessPermissions

Contém um conjunto de sinalizadores que especificam quais permissões de acesso devem ser concedidas quando o documento é aberto com acesso de usuário.

```csharp
[Flags]
public enum PdfAccessPermissions
```

### Valores

| Nome | Valor | Descrição |
| --- | --- | --- |
| None | `0` | Especifica que um usuário não tem permissões de acesso. |
| PrintDocument | `4` | Especifica se um usuário pode imprimir o documento (possivelmente não no nível de maior qualidade, dependendo se o sinalizador HighQualityPrint também está definido). |
| ModifyContent | `8` | Especifica se um usuário pode modificar o conteúdo do documento por operações diferentes daquelas controladas pelos sinalizadores AddOrModifyFields, FillExistingFields, AssembleDocument. |
| CopyTextAndGraphics | `10` | Especifica se um usuário pode copiar ou de outra forma extrair texto e gráficos do documento por operações diferentes da controlada pelo sinalizador ExtractTextAndGraphics. |
| AddOrModifyFields | `20` | Especifica se um usuário pode adicionar ou modificar anotações de texto, preencher campos de formulário interativos e, se o sinalizador ModifyContent também estiver definido, criar ou modificar campos de formulário interativos (incluindo campos de assinatura). |
| FillExistingFields | `100` | Especifica se um usuário pode preencher campos de formulário interativos existentes (incluindo campos de assinatura), mesmo se o sinalizador AddOrModifyFields estiver desativado. |
| ExtractTextAndGraphics | `200` | Especifica se um usuário pode extrair texto e gráficos em apoio à acessibilidade para usuários com deficiência ou para outros fins. |
| AssembleDocument | `400` | Especifica se um usuário pode montar o documento (inserir, rotacionar ou excluir páginas e criar marcadores ou imagens em miniatura), mesmo se o sinalizador ModifyContent estiver desativado. |
| HighQualityPrint | `800` | Especifica se um usuário pode imprimir o documento para uma representação a partir da qual uma cópia digital fiel do conteúdo PDF possa ser gerada. Quando este sinalizador está desativado (e o sinalizador PrintDocument está definido), a impressão fica limitada a uma representação de baixo nível da aparência, possivelmente de qualidade degradada. |

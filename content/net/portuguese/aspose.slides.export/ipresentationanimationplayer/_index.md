## IPresentationAnimationPlayer interface

Representa um reprodutor da animação. Animações geradas por [`PresentationAnimationsGenerator`](../presentationanimationsgenerator) via seu [`NewAnimation`](../presentationanimationsgenerator/newanimation) event.

```csharp
public interface IPresentationAnimationPlayer
```

| Nome | Descrição |
| --- | --- |
| [Duration](../../aspose.slides.export/ipresentationanimationplayer/duration) { get; } | Obtém a duração da animação [ms] |

## Métodos

| Nome | Descrição |
| --- | --- |
| [GetFrame](../../aspose.slides.export/ipresentationanimationplayer/getframe)() | Obtém o quadro para a posição de tempo atual previamente definida com o método [`SetTimePosition`](./settimeposition). |
| [SetTimePosition](../../aspose.slides.export/ipresentationanimationplayer/settimeposition)(double) | Define a posição de tempo da animação dentro do [`Duration`](./duration). |

### Exemplos

```csharp
[C#]
using (Presentation presentation = new Presentation("animated.pptx"))
{
    using (var animationsGenerator = new PresentationAnimationsGenerator(presentation.SlideSize.Size.ToSize()))
    {
        animationsGenerator.NewAnimation += animationPlayer =>
        {
            Console.WriteLine($"Animation total duration: {animationPlayer.Duration}");
            
            animationPlayer.SetTimePosition(0);
            animationPlayer.GetFrame().Save("firstFrame.png");
            
            animationPlayer.SetTimePosition(animationPlayer.Duration);
            animationPlayer.GetFrame().Save("lastFrame.png");
      };
        
        animationsGenerator.Run(presentation.Slides);
    }
}
```

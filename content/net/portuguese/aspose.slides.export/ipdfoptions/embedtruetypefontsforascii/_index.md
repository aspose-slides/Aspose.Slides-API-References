## IPdfOptions.EmbedTrueTypeFontsForASCII propriedade

True para incorporar fontes TrueType para caracteres ASCII 32-127. Fontes para códigos de caracteres maiores que 127 são sempre incorporadas. Leitura/Gravação Boolean.

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### Observações

O padrão é **true**.

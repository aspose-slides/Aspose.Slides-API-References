## IPdfOptions.RasterizeUnsupportedFontStyles propriedade

Indica se o texto deve ser rasterizado como bitmap e salvo em PDF quando a fonte não suporta estilo negrito. Essa abordagem pode melhorar a qualidade do texto no PDF resultante para certas fontes. Leitura/gravação Boolean.

```csharp
public bool RasterizeUnsupportedFontStyles { get; set; }
```

### Observações

Default is **false**.

## PdfOptions.EmbedTrueTypeFontsForASCII propriedade

Determina se Aspose.Slides incorporará fontes comuns para texto ASCII (faixa de códigos 33..127). Fontes para códigos de caracteres superiores a 127 são sempre incorporadas. A lista de fontes comuns inclui as 14 fontes base do PDF e fontes adicionais especificadas pelo usuário. Leitura/gravação Boolean.

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### Observações

O padrão é **true**.

## PdfOptions.BestImagesCompressionRatio propriedade

Indica se a compressão mais eficaz (em vez da padrão) para cada imagem deve ser selecionada automaticamente. Se definido como Boolean.true, para cada imagem na apresentação o algoritmo de compressão mais apropriado será escolhido, o que levará a um tamanho menor do documento PDF resultante. A seleção da melhor taxa de compressão de imagem é computacionalmente cara e consome uma quantidade adicional de RAM, e esta opção é Boolean.false por padrão.

```csharp
public bool BestImagesCompressionRatio { get; set; }
```

### Observações

O padrão é Boolean.false.

## PdfOptions.RasterizeUnsupportedFontStyles propriedade

Indica se o texto deve ser rasterizado como bitmap e salvo em PDF quando a fonte não oferece suporte a estilo negrito. Esta abordagem pode melhorar a qualidade do texto no PDF resultante para determinadas fontes. Leitura/Gravação Boolean.

```csharp
public bool RasterizeUnsupportedFontStyles { get; set; }
```

### Observações

O padrão é **false**.

## InkOptions.InterpretMaskOpAsOpacity propriedade

Usa operação ROP ou Opacity para renderizar o pincel.

```csharp
public bool InterpretMaskOpAsOpacity { get; set; }
```

### Observações

O valor padrão é true.

### Exemplos

O próximo exemplo demonstra como definir usando ROP para exportar elementos Ink:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.InterpretMaskOpAsOpacity = false;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

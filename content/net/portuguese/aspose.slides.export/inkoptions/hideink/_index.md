## InkOptions.HideInk propriedade

Exibe ou oculta elementos de Ink no documento exportado.

```csharp
public bool HideInk { get; set; }
```

### Observações

Valor padrão é falso.

### Exemplos

O próximo exemplo demonstra como ocultar elementos de Ink no documento PDF exportado:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.HideInk = true;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

## MarkdownSaveOptions.ImageSaving evento

Ocorre para cada imagem que não seja SVG (bitmap ou metafile) durante a exportação para Markdown. Permite personalizar como a imagem é salva e referenciada. Se não for tratada, a imagem será salva localmente com um link relativo.

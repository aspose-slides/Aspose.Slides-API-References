## MarkdownSaveOptions.SlideNumberFormat property

Obtém ou define a cadeia de formato usada para cabeçalhos de número de slide na saída Markdown. O formato deve incluir o placeholder "{0}", que será substituído pelo índice do slide durante a exportação. Exemplo: "# Slide {0}" produzirá "# Slide 1", "# Slide 2", etc.

```csharp
public string SlideNumberFormat { get; set; }
```

### Exceções

| exceção | condição |
| --- | --- |
| ArgumentNullException | Lançada se o valor fornecido for nulo ou vazio. |
| ArgumentException | Lançada se a cadeia de formato não contiver o placeholder "{0}". |

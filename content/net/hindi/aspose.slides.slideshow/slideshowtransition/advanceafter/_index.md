## SlideShowTransition.AdvanceAfter गुण

यह विशेषता निर्धारित करती है कि स्लाइडशो निश्चित समय के बाद अगली स्लाइड पर जाएगा या नहीं। पढ़ने/लिखने Boolean.

```csharp
public bool AdvanceAfter { get; set; }
```

### उदाहरण

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    // पहली स्लाइड ट्रांज़िशन प्राप्त करें
    ISlideShowTransition slideTransition = pres.Slides[0].SlideShowTransition;
    
    // जांचें कि Advance Slide After फ़्लैग चयनित है या नहीं
    if (slideTransition.AdvanceAfter)
    {
        // Advance Slide After Time मान प्राप्त करें
        uint advanceAfterTime = slideTransition.AdvanceAfterTime;
    }
}
```

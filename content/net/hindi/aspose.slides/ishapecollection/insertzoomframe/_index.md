## InsertZoomFrame(int, float, float, float, float, ISlide) {#insertzoomframe}

नया Zoom फ्रेम बनाता है और उसे निर्धारित सूचकांक पर shape संग्रह में सम्मिलित करता है।

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide)
```

| पैरामीटर | टाइप | विवरण |
| --- | --- | --- |
| index | Int32 | वह शून्य-आधारित सूचकांक जहाँ Zoom फ्रेम सम्मिलित किया जाएगा। |
| x | Single | नए Zoom फ्रेम का x-निर्देशांक, पॉइंट में। |
| y | Single | नए Zoom फ्रेम का y-निर्देशांक, पॉइंट में। |
| width | Single | नए Zoom फ्रेम की चौड़ाई, पॉइंट में। |
| height | Single | नए Zoom फ्रेम की ऊँचाई, पॉइंट में। |
| slide | ISlide | Zoom फ्रेम द्वारा संदर्भित [`ISlide`](../../islide)। |

### रिटर्न वैल्यू

नया बनाया गया [`IZoomFrame`](../../izoomframe)।

### अपवाद

| अपवाद | स्थिति |
| --- | --- |
| ArgumentException | यदि संदर्भित स्लाइड वर्तमान प्रस्तुति का भाग नहीं है तो उत्पन्न होता है। |

### उदाहरण

यह उदाहरण, एक संग्रह में निर्दिष्ट सूचकांक पर Zoom ऑब्जेक्ट बनाना और सम्मिलित करना दर्शाता है (मान लें कि "Presentation.pptx" प्रस्तुति में कम से कम दो स्लाइड हैं):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertZoomFrame(2, 150, 20, 50, 50, pres.Slides[1]);
}
```

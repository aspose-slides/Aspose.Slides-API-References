## AddSectionZoomFrame(float, float, float, float, ISection) {#addsectionzoomframe}

नया Section Zoom फ्रेम बनाता है और उसे shape collection के अंत में जोड़ता है।

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section)
```

| Parameter | Type | Description |
| --- | --- | --- |
| x | Single | नया Section Zoom फ्रेम का x-निर्देशांक, पॉइंट में। |
| y | Single | नया Section Zoom फ्रेम का y-निर्देशांक, पॉइंट में। |
| width | Single | नया Section Zoom फ्रेम की चौड़ाई, पॉइंट में। |
| height | Single | नया Section Zoom फ्रेम की ऊँचाई, पॉइंट में। |
| section | ISection | [`ISection`](../../isection) को Section Zoom फ्रेम द्वारा संदर्भित किया गया है; यह इस प्रस्तुति से संबंधित होना चाहिए और कम से कम एक स्लाइड होना चाहिए। |

### Return Value

नया बनाया गया [`ISectionZoomFrame`](../../isectionzoomframe)।

### Exceptions

| exception | condition |
| --- | --- |
| ArgumentException | तब फेंका जाता है जब संदर्भित सेक्शन वर्तमान प्रस्तुति से संबंधित नहीं है या उसमें कोई स्लाइड नहीं है। |

### Examples

यह उदाहरण संग्रह के अंत में Section Zoom ऑब्जेक्ट जोड़ने को दर्शाता है (मान लीजिए "Presentation.pptx" प्रस्तुति में कम से कम दो सेक्शन हैं):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1]);
}
```

## AddAudioFrameEmbedded(float, float, float, float, Stream) {#addaudioframeembedded_1}

नया ऑडियो फ्रेम एक एम्बेडेड WAV फ़ाइल के साथ बनाता है और इसे शेप कलेक्शन के अंत में जोड़ता है। एम्बेडेड ऑडियो को Presentation.Audios कलेक्शन में जोड़ा जाता है।

```csharp
public IAudioFrame AddAudioFrameEmbedded(float x, float y, float width, float height, 
    Stream audio_stream)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| x | Single | नए ऑडियो फ्रेम का x-निर्देशांक, पॉइंट में। |
| y | Single | नए ऑडियो फ्रेम का y-निर्देशांक, पॉइंट में। |
| width | Single | नए ऑडियो फ्रेम की चौड़ाई, पॉइंट में। |
| height | Single | नए ऑडियो फ्रेम की ऊंचाई, पॉइंट में। |
| audio_stream | Stream | एम्बेड करने के लिए WAV ऑडियो डेटा वाला इनपुट स्ट्रीम। |

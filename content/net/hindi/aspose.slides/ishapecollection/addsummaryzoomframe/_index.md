## IShapeCollection.AddSummaryZoomFrame मेथड

नया Summary Zoom फ्रेम बनाता है और उसे shape collection के अंत में जोड़ता है।

```csharp
public ISummaryZoomFrame AddSummaryZoomFrame(float x, float y, float width, float height)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| x | Single | नए Summary Zoom फ्रेम का x-कोऑर्डिनेट, पॉइंट्स में। |
| y | Single | नए Summary Zoom फ्रेम का y-कोऑर्डिनेट, पॉइंट्स में। |
| width | Single | नए Summary Zoom फ्रेम की चौड़ाई, पॉइंट्स में। |
| height | Single | नए Summary Zoom फ्रेम की ऊँचाई, पॉइंट्स में। |

### रिटर्न वैल्यू

नया बनाया गया [`ISummaryZoomFrame`](../../isummaryzoomframe)।

### अपवाद

| अपवाद | शर्त |
| --- | --- |

### टिप्पणियाँ

यह मेथड एक Summary Zoom फ्रेम बनाता है जो प्रस्तुति के सभी सेक्शन के सारांश लिंक को एकत्रित करता है।

### उदाहरण

यह उदाहरण एक Summary Zoom ऑब्जेक्ट को संग्रह के अंत में जोड़ने को दर्शाता है (मान लीजिए "Presentation.pptx" प्रस्तुति में कम से कम दो सेक्शन हैं):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSummaryZoomFrame(150, 20, 500, 250);
}
```

## IShapeCollection.InsertSummaryZoomFrame मेथड

एक नया Summary Zoom फ्रेम बनाता है और निर्दिष्ट अनुक्रमांक पर इसे shape संग्रह में सम्मिलित करता है।

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| index | Int32 | Summary Zoom फ्रेम को सम्मिलित करने के लिए शून्य-आधारित अनुक्रमांक। |
| x | Single | नए Summary Zoom फ्रेम का x-निर्देशांक, बिंदुओं में। |
| y | Single | नए Summary Zoom फ्रेम का y-निर्देशांक, बिंदुओं में। |
| width | Single | नए Summary Zoom फ्रेम की चौड़ाई, बिंदुओं में। |
| height | Single | नए Summary Zoom फ्रेम की ऊँचाई, बिंदुओं में। |

### वापसी मान

नव निर्मित [`ISummaryZoomFrame`](../../isummaryzoomframe)।

### टिप्पणियाँ

यह मेथड एक Summary Zoom फ्रेम बनाता है जो प्रस्तुति के सभी सेक्शनों के लिए सारांश लिंक को संकलित करता है।

### उदाहरण

यह उदाहरण एक Summary Zoom ऑब्जेक्ट को संग्रह के निर्दिष्ट अनुक्रमांक पर बनाने और सम्मिलित करने को दर्शाता है (मान लें कि "Presentation.pptx" प्रस्तुति में कम से कम दो सेक्शन हैं):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSummaryZoomFrame(2, 150, 20, 50, 50);
}
```

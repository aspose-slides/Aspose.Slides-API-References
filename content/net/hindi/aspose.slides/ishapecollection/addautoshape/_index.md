## AddAutoShape(ShapeType, float, float, float, float, bool) {#addautoshape_1}

एक नया ऑटो शेप बनाता है और इसे शेप कलेक्शन के अंत में जोड़ता है, वैकल्पिक रूप से इसे डिफ़ॉल्ट टेम्प्लेट फॉर्मेटिंग के साथ प्रारंभ करता है।

```csharp
public IAutoShape AddAutoShape(ShapeType shapeType, float x, float y, float width, float height, 
    bool createFromTemplate)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| shapeType | ShapeType | जोड़े जाने वाले ऑटो शेप का [`ShapeType`](../../shapetype)। |
| x | Single | शेप के फ्रेम का x-निर्देशांक, पॉइंट्स में। |
| y | Single | शेप के फ्रेम का y-निर्देशांक, पॉइंट्स में। |
| width | Single | शेप के फ्रेम की चौड़ाई, पॉइंट्स में। |
| height | Single | शेप के फ्रेम की ऊँचाई, पॉइंट्स में। |
| createFromTemplate | Boolean | true होने पर नई शेप पर डिफ़ॉल्ट टेम्प्लेट स्टाइल (सादा स्टाइल, केंद्रित टेक्स्ट, और खाली नहीं नाम) लागू किया जाता है; false होने पर सभी प्रॉपर्टी डिफ़ॉल्ट मानों के साथ बनाई जाती है। |

## InsertSectionZoomFrame(int, float, float, float, float, ISection) {#insertsectionzoomframe}

एक नया Section Zoom फ्रेम बनाता है और इसे निर्दिष्ट इंडेक्स पर shape संग्रह में सम्मिलित करता है।

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| index | Int32 | शून्य-आधारित इंडेक्स जहाँ Section Zoom फ्रेम सम्मिलित किया जाना है। |
| x | Single | नए Section Zoom फ्रेम का x-निर्देशांक, पॉइंट्स में। |
| y | Single | नए Section Zoom फ्रेम का y-निर्देशांक, पॉइंट्स में। |
| width | Single | नए Section Zoom फ्रेम की चौड़ाई, पॉइंट्स में। |
| height | Single | नए Section Zoom फ्रेम की ऊँचाई, पॉइंट्स में। |
| section | ISection | [`ISection`](../../isection) जिसे Section Zoom फ्रेम द्वारा संदर्भित किया गया है; यह वर्तमान प्रस्तुति से जुड़ा होना चाहिए और कम से कम एक स्लाइड होना चाहिए। |

### वापसी मान

नव निर्मित [`ISectionZoomFrame`](../../isectionzoomframe)।

### अपवाद

| अपवाद | शर्त |
| --- | --- |
| ArgumentException | यदि संदर्भित सेक्शन वर्तमान प्रस्तुति से संबंधित नहीं है या उसमें कोई स्लाइड नहीं है तो उत्पन्न होता है। |

### उदाहरण

यह उदाहरण एक Section Zoom ऑब्जेक्ट को संग्रह के निर्दिष्ट इंडेक्स पर बनाने और सम्मिलित करने को दर्शाता है (मान लीजिए कि "Presentation.pptx" प्रस्तुति में कम से कम दो सेक्शन हैं):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSectionZoomFrame(2, 150, 20, 50, 50, pres.Sections[1]);
}
```

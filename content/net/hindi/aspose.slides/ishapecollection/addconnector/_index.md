## AddConnector(ShapeType, float, float, float, float, bool) {#addconnector_1}

एक नया कनेक्टर आकार बनाता है और इसे शेप संग्रह के अंत में जोड़ता है, वैकल्पिक रूप से डिफ़ॉल्ट टेम्प्लेट स्टाइलिंग लागू करता है।

```csharp
public IConnector AddConnector(ShapeType shapeType, float x, float y, float width, float height, 
    bool createFromTemplate)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| shapeType | ShapeType | बनाने के लिए कनेक्टर आकार का [`ShapeType`](../../shapetype)। |
| x | Single | कनेक्टर के फ्रेम का x-निर्देशांक, पॉइंट्स में। |
| y | Single | कनेक्टर के फ्रेम का y-निर्देशांक, पॉइंट्स में। |
| width | Single | कनेक्टर के फ्रेम की चौड़ाई, पॉइंट्स में। |
| height | Single | कनेक्टर के फ्रेम की ऊँचाई, पॉइंट्स में। |
| createFromTemplate | Boolean | डिफ़ॉल्ट टेम्प्लेट स्टाइलिंग लागू करने के लिए true (खाली नहीं नाम, साधारण शैली); कनेक्टर को डिफ़ॉल्ट प्रॉपर्टी मूल्यों के साथ बनाने के लिए false। |

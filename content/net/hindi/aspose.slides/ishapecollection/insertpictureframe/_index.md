```csharp
public IPictureFrame InsertPictureFrame(int index, ShapeType shapeType, float x, float y, 
    float width, float height, IPPImage image)
```

| --- | --- | --- |
| index | Int32 | picture frame को सम्मिलित करने के लिए शून्य-आधारित सूचकांक। |

| width | Single | picture frame की चौड़ाई, पॉइंट्स में। |
| height | Single | picture frame की ऊँचाई, पॉइंट्स में। |

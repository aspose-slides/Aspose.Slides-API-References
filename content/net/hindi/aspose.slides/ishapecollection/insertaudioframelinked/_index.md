```csharp
public IAudioFrame InsertAudioFrameLinked(int index, float x, float y, float width, float height, 
    string fname)
```

| --- | --- | --- |
| index | Int32 | शून्य-आधारित इंडेक्स जहाँ ऑडियो फ्रेम को डालना है। |
| x | Single | नए ऑडियो फ्रेम का x-निर्देशांक, बिंदुओं में। |
| y | Single | नए ऑडियो फ्रेम का y-निर्देशांक, बिंदुओं में। |
| width | Single | नए ऑडियो फ्रेम की चौड़ाई, बिंदुओं में। |
| height | Single | नए ऑडियो फ्रेम की ऊँचाई, बिंदुओं में। |

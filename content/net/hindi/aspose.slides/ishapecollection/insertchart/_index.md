## InsertChart(ChartType, float, float, float, float, int, bool) {#insertchart_1}

नया चार्ट बनाता है, नमूना श्रृंखला डेटा और सेटिंग्स के साथ प्रारंभ करता है, और निर्दिष्ट इंडेक्स पर शेप संग्रह में इसे सम्मिलित करता है।

```csharp
public IChart InsertChart(ChartType type, float x, float y, float width, float height, int index, 
    bool initWithSample)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| type | ChartType | बनाने के लिए चार्ट का प्रकार। |
| x | Single | नई चार्ट का x-निर्देशांक, पॉइंट्स में। |
| y | Single | नई चार्ट का y-निर्देशांक, पॉइंट्स में। |
| width | Single | नई चार्ट की चौड़ाई, पॉइंट्स में। |
| height | Single | नई चार्ट की ऊँचाई, पॉइंट्स में। |
| index | Int32 | शेप संग्रह में नई चार्ट को सम्मिलित करने के लिए शून्य-आधारित इंडेक्स। |
| initWithSample | Boolean | true होने पर नई चार्ट को नमूना श्रृंखला डेटा और सेटिंग्स के साथ प्रारंभ करता है; false होने पर कोई श्रृंखला नहीं और केवल न्यूनतम सेटिंग्स के साथ चार्ट बनाता है, जिससे निर्माण तेज़ होता है। |

## AddChart(ChartType, float, float, float, float) {#addchart}

एक नया चार्ट बनाता है, इसे नमूना सीरीज़ डेटा और सेटिंग्स के साथ प्रारंभ करता है, और इसे शेप संग्रह के अंत में जोड़ता है।

```csharp
public IChart AddChart(ChartType type, float x, float y, float width, float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| type | ChartType | जोड़ने के लिए चार्ट का प्रकार। |
| x | Single | नए चार्ट का x-कोऑर्डिनेट, पॉइंट्स में। |
| y | Single | नए चार्ट का y-कोऑर्डिनेट, पॉइंट्स में। |
| width | Single | चार्ट की चौड़ाई, पॉइंट्स में। |
| height | Single | चार्ट की ऊँचाई, पॉइंट्स में। |

### Return Value

नया बनाया गया [`IChart`](../../../aspose.slides.charts/ichart)।

## AddChart(ChartType, float, float, float, float, bool) {#addchart_1}

एक नया चार्ट बनाता है, इसे नमूना सीरीज़ डेटा और सेटिंग्स के साथ प्रारंभ करता है, और इसे शेप संग्रह के अंत में जोड़ता है।

```csharp
public IChart AddChart(ChartType type, float x, float y, float width, float height, 
    bool initWithSample)
```

| Parameter | Type | Description |
| --- | --- | --- |
| type | ChartType | जोड़ने के लिए चार्ट का प्रकार। |
| x | Single | नए चार्ट का x-कोऑर्डिनेट, पॉइंट्स में। |
| y | Single | नए चार्ट का y-कोऑर्डिनेट, पॉइंट्स में। |
| width | Single | चार्ट की चौड़ाई, पॉइंट्स में। |
| height | Single | चार्ट की ऊँचाई, पॉइंट्स में। |
| initWithSample | Boolean | नया चार्ट नमूना सीरीज़ डेटा और सेटिंग्स के साथ प्रारंभ करने के लिए true; कोई सीरीज़ न रख कर केवल न्यूनतम सेटिंग्स के साथ चार्ट बनाने के लिए false, जिससे निर्माण तेज़ हो जाता है। |

### Return Value

नया बनाया गया [`IChart`](../../../aspose.slides.charts/ichart)।

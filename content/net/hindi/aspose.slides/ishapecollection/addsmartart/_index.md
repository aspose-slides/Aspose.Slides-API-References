## IShapeCollection.AddSmartArt विधि

SmartArt डायग्राम बनाता है और उसे शैप कलेक्शन के अंत में जोड़ता है।

```csharp
public ISmartArt AddSmartArt(float x, float y, float width, float height, 
    SmartArtLayoutType layoutType)
```

| पैरामीटर | टाइप | विवरण |
| --- | --- | --- |
| x | Single | डायग्राम के फ्रेम का x-कोऑर्डिनेट, पॉइंट्स में. |
| y | Single | डायग्राम के फ्रेम का y-कोऑर्डिनेट, पॉइंट्स में. |
| width | Single | डायग्राम के फ्रेम की चौड़ाई, पॉइंट्स में. |
| height | Single | डायग्राम के फ्रेम की ऊंचाई, पॉइंट्स में. |
| layoutType | SmartArtLayoutType | SmartArt लेआउट प्रकार. |

### रिटर्न मान

नया निर्मित [`ISmartArt`](../../../aspose.slides.smartart/ismartart)।

### उदाहरण

उदाहरण:

```csharp
[C#]
Presentation pres = new Presentation();
Slide slide = pres.Slides[0];
SmartArt smart = slide.Shapes.AddSmartArt(0, 0, 400, 400, SmartArtLayoutType.BasicBlockList);
```

[Visual Basic]

## IPresentation.HighlightRegex मेथड

विशिष्ट रंग के साथ नियमित अभिव्यक्ति के सभी मेल को हाइलाइट करता है।

```csharp
public void HighlightRegex(Regex regex, Color highlightColor, IFindResultCallback callback)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| regex | Regex | हाइलाइट करने के लिए स्ट्रिंग्स प्राप्त करने वाली नियमित अभिव्यक्ति Regex। |
| highlightColor | Color | टेक्स्ट को हाइलाइट करने के लिए रंग। |
| callback | IFindResultCallback | खोज परिणाम प्राप्त करने के लिए कॉलबैक ऑब्जेक्ट [`IFindResultCallback`](../../ifindresultcallback)। |

### उदाहरण

नियमित अभिव्यक्ति का उपयोग करके PowerPoint Presentation में टेक्स्ट को हाइलाइट करने के लिए निम्नलिखित कोड नमूना दिखाया गया है।

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// 10 या अधिक अक्षरों वाले सभी शब्दों को हाइलाइट करना
	presentation.HighlightRegex(regex, Color.Blue, null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

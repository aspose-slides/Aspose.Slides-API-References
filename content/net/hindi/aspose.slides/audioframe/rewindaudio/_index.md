## AudioFrame.RewindAudio प्रॉपर्टी

निर्धारित करता है कि चलाने के बाद ऑडियो स्वचालित रूप से प्रारंभ पर पुनः चलाया जाता है या नहीं। पढ़ें/लिखें Boolean.

```csharp
public bool RewindAudio { get; set; }
```

### उदाहरण

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ISlide slide = pres.Slides[0];

    // ऑडियो फ्रेम जोड़ें
    IAudioFrame audioFrame = slide.Shapes.AddAudioFrameLinked(50, 50, 100, 100, "sampleaudio.wav");

    // ऑडियो को सभी स्लाइड्स पर चलाने के लिये सेट करें
    audioFrame.PlayAcrossSlides = true;

    // चलाने के बाद ऑडियो को स्वचालित रूप से शुरू से रीवाइंड करने के लिए सेट करें
    audioFrame.RewindAudio = true;

    pres.Save("AudioFrame_out.pptx", SaveFormat.Pptx);
}
```


| Parameter | Type | Description |
| --- | --- | --- |

| offset | Int32 | बफ़र में शून्य-आधारित बाइट ऑफसेट जहाँ से बाइट्स को वर्तमान स्ट्रीम में कॉपी करना शुरू किया जाता है Int32 |
| count | Int32 | वर्तमान स्ट्रीम में लिखे जाने वाले बाइट्स की संख्या Int32 |

## AddFromHtml(Stream) {#addfromhtml}

HTML पाठ से स्लाइड बनाता है और उन्हें संग्रह के अंत में जोड़ता है।

```csharp
public ISlide[] AddFromHtml(Stream htmlStream)
```

| परामीटर | प्रकार | विवरण |
| --- | --- | --- |
| htmlStream | Stream | HTML फ़ाइल के स्रोत के रूप में उपयोग किया जाएगा ऐसा Stream ऑब्जेक्ट। |

### वापसी मान

जोड़ दी गई स्लाइड्स।

### उदाहरण

```csharp
[C#]
// Presentation क्लास का एक उदाहरण बनाएं।
using (var presentation = new Presentation())
{
    using (var htmlStream = File.OpenRead("page.html"))
    {
        // AddFromHtml मेथड को कॉल करें और HTML फ़ाइल पास करें।
        presentation.Slides.AddFromHtml(htmlStream);
    }
    // फ़ाइल को PowerPoint दस्तावेज़ के रूप में सहेजने के लिए Save मेथड का उपयोग करें।
    presentation.Save("MyPresentation.pptx", SaveFormat.Pptx);
}
```

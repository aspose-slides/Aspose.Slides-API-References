## IFontsManager.GetFontEmbeddingLevel मेथड

दिए गए बाइट ऐरे और फ़ॉन्ट नाम से फ़ॉन्ट के एम्बेडिंग स्तर को निर्धारित करता है।

```csharp
public EmbeddingLevel GetFontEmbeddingLevel(byte[] fontBytes, string fontName)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| fontBytes | Byte[] | फ़ॉन्ट डेटा रखने वाला बाइट ऐरे। |
| fontName | String | फ़ॉन्ट का नाम। |

### रिटर्न वैल्यू

निर्दिष्ट फ़ॉन्ट का एम्बेडिंग स्तर।

### अपवाद

| अपवाद | स्थिति |
| --- | --- |
| ArgumentNullException | जब *fontBytes* null हो तो फेंका जाता है। |

### उदाहरण

```csharp
[C#]
using (Presentation pres = new Presentation(pptxFileName))
{
    // प्रेजेंटेशन में उपयोग किए गए सभी फ़ॉन्ट प्राप्त करें
    IFontData[] fontDatas = pres.FontsManager.GetFonts();

    // प्रेजेंटेशन में पहले फ़ॉन्ट की नियमित शैली का प्रतिनिधित्व करने वाला बाइट ऐरे प्राप्त करें
    byte[] bytes = pres.FontsManager.GetFontBytes(fontDatas[0], FontStyle.Regular);

    // फ़ॉन्ट के एम्बेडिंग स्तर को निर्धारित करें
    EmbeddingLevel embeddingLevel = pres.FontsManager.GetFontEmbeddingLevel(bytes, fontDatas[0].FontName);
}
```

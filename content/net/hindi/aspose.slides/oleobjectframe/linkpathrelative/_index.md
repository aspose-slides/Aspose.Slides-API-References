## OleObjectFrame.LinkPathRelative प्रॉपर्टी

यदि मौजूद हो तो लिंक की गई फ़ाइल का सापेक्ष पथ लौटाता है, अन्यथा एक खाली स्ट्रिंग लौटाता है। केवल पढ़ने योग्य String.

```csharp
public string LinkPathRelative { get; }
```

### टिप्पणी

Ppt प्रस्तुतियों में, कुछ Ole ऑब्जेक्ट लिंक की सापेक्ष प्रतिनिधित्व हो सकता है।

### उदाहरण

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.ppt"))
{
    IOleObjectFrame oleFrame = presentation.Slides[0].Shapes[0] as IOleObjectFrame;

    if (oleFrame != null)
    {
        Console.WriteLine("The relative path: " + oleFrame.LinkPathRelative);
    } 
}
```

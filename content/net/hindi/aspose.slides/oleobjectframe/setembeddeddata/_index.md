## OleObjectFrame.SetEmbeddedData मेथड

OLE एम्बेडेड डेटा के बारे में जानकारी सेट करता है। यह मेथड ऑब्जेक्ट की प्रॉपर्टीज़ को नए डेटा को प्रतिबिंबित करने के लिए बदलता है और IsObjectLink फ़्लैग को false सेट करता है, यह संकेत देते हुए कि OLE ऑब्जेक्ट एम्बेडेड है।

```csharp
public void SetEmbeddedData(IOleEmbeddedDataInfo embeddedData)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| embeddedData | IOleEmbeddedDataInfo | एम्बेडेड डेटा [`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo) |

### अपवाद

| अपवाद | शर्त |
| --- | --- |
| ArgumentNullException | जब embeddedData पैरामीटर null हो। |

### उदाहरण

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    OleObjectFrame oof = pres.Slides[0].Shapes[0] as OleObjectFrame;
    if (oof != null)
    {
        IOleEmbeddedDataInfo newData = new OleEmbeddedDataInfo(File.ReadAllBytes("Picture.png"), "png");
        oof.SetEmbeddedData(newData);
    }
}
```

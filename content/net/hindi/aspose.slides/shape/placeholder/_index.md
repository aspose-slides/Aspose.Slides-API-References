## Shape.Placeholder प्रॉपर्टी

शेप के लिए प्लेसहोल्डर लौटाता है। यदि शेप में कोई प्लेसहोल्डर नहीं है तो null लौटाता है। केवल-पढ़ने योग्य [`IPlaceholder`](../../iplaceholder).

```csharp
public IPlaceholder Placeholder { get; }
```

### उदाहरण

निम्न उदाहरण दिखाता है कि प्लेसहोल्डर में Text को कैसे बदलें।

```csharp
[C#]
// एक Presentation क्लास का उदाहरण बनाता है
using (Presentation pres = new Presentation("ReplacingText.pptx"))
{
    // पहले स्लाइड तक पहुंचता है
    ISlide sld = pres.Slides[0];
    // प्लेसहोल्डर खोजने के लिए शैप्स पर इटरैट करता है
    foreach (IShape shp in sld.Shapes)
        if (shp.Placeholder != null)
        {
            // प्रत्येक प्लेसहोल्डर में टेक्स्ट बदलता है
            ((IAutoShape)shp).TextFrame.Text = "This is a Placeholder";
        }
    // प्रेजेंटेशन को डिस्क पर सेव करता है
    pres.Save("output_out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
}
```

निम्न उदाहरण दिखाता है कि प्लेसहोल्डर में Prompt Text को कैसे सेट करें।

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation2.pptx"))
{
    ISlide slide = pres.Slides[0];
    foreach (IShape shape in slide.Slide.Shapes) // स्लाइड के माध्यम से इटरैट करता है
    {
        if (shape.Placeholder != null && shape is AutoShape)
        {
            string text = "";
            if (shape.Placeholder.Type == PlaceholderType.CenteredTitle) // PowerPoint "Click to add title" दिखाता है
            {
                text = "Add Title";
            }
            else if (shape.Placeholder.Type == PlaceholderType.Subtitle) // सबटाइटल जोड़ता है
            {
                text = "Add Subtitle";
            }
            ((IAutoShape)shape).TextFrame.Text = text;
            Console.WriteLine($"Placeholder with text: {text}");
        }
    }
    pres.Save("Placeholders_PromptText.pptx", SaveFormat.Pptx);
}
```

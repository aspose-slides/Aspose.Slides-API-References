## IBasePortionFormat.SpellCheck प्रॉपर्टी

टेक्स्ट भाग के लिए वर्तनी जाँच सक्षम है या नहीं, यह दर्शाने वाला मान प्राप्त करती या सेट करती है। जब इस प्रॉपर्टी को false सेट किया जाता है, तो टेक्स्ट तत्वों के लिए वर्तनी जाँच दबी रहती है। जब इसे true सेट किया जाता है, तो वर्तनी जाँच की अनुमति होती है। डिफ़ॉल्ट मान `false` है।

```csharp
public bool SpellCheck { get; set; }
```

### उदाहरण

Next example demonstrates enabling the SpellCheck flag before saving the presentation:

```csharp
[C#]
using (var pres = new Presentation("input.pptx"))
{
    // पहले स्लाइड पर पहले आकार के भीतर पाठ का पहला भाग एक्सेस करें
    var portion = ((AutoShape)pres.Slides[0].Shapes[0]).TextFrame.Paragraphs[0].Portions[0];
    // इस पाठ भाग के लिए वर्तनी जांच सक्षम करें
    portion.PortionFormat.SpellCheck = true;
    // संशोधित प्रस्तुति को सहेजें
    pres.Save("output-with-spellcheck.pptx", SaveFormat.Pptx);
}
```

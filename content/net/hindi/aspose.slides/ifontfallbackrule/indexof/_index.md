## IFontFallBackRule.IndexOf विधि

दिए गए नियम का अनुक्रमांक संग्रह में लौटाता है।

```csharp
public int IndexOf(string fontName)
```

| परामीटर | प्रकार | विवरण |
| --- | --- | --- |
| fontName | String | फ़ॉन्ट का नाम जिसे खोजा जाना है। |

### रिटर्न मान

फ़ॉन्ट का अनुक्रमांक, या यदि सूची में फ़ॉन्ट न मिला हो तो -1।

### उदाहरण

```csharp
[C#]
// फ़ॉन्टों की सूची वाला एक नियम बनाएं।
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

//Tahoma का अनुक्रमांक प्राप्त करें।
int tahomaIndex = newRule.IndexOf("Tahoma");
```

## PictureFillFormat.TileAlignment प्रॉपर्टी

टेक्सचर को आकार के भीतर कैसे संरेखित किया जाता है, इसे लौटाता या सेट करता है। यह सेटिंग टेक्सचर पैटर्न की शुरुआती बिंदु और आकार के पार इसकी दोहराव को नियंत्रित करती है। पढ़ें/लिखें [`RectangleAlignment`](../../rectanglealignment).

```csharp
public RectangleAlignment TileAlignment { get; set; }
```

### टिप्पणियाँ

डिफ़ॉल्ट TopLeft है।

### उदाहरण

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // आकृति के चित्र भराव प्रारूप को प्राप्त करता है
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // चित्र भराव मोड को टाइल पर सेट करता है
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // टाइलिंग के संरेखण को दाएँ नीचे सेट करता है
    pictureFillFormat.TileAlignment = RectangleAlignment.BottomRight;
}
```

## CompressImage(bool, PicturesCompression) {#compressimage}

चित्र को आकार को घटाकर और निर्धारित रिज़ॉल्यूशन के आधार पर उसके आकार को कम करता है। वैकल्पिक रूप से, यह क्रॉप किए गए क्षेत्रों को भी हटा देता है।

```csharp
public bool CompressImage(bool deleteCroppedAreasOfImage, PicturesCompression resolution)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| deleteCroppedAreasOfImage | Boolean | यदि true है, तो यह मेथड चित्र के क्रॉप किए गए क्षेत्रों को हटा देगा, जिससे उसका आकार और भी घट सकता है। |
| resolution | PicturesCompression | संपीड़न के लिए लक्ष्य रिज़ॉल्यूशन, जो [`PicturesCompression`](../../../aspose.slides.export/picturescompression) enum के मान के रूप में निर्दिष्ट किया गया है। |

### वापसी मान

एक Boolean जो दर्शाता है कि चित्र सफलतापूर्वक संकुचित हुआ या नहीं। यदि चित्र का आकार बदला या क्रॉप किया गया हो तो `true` लौटाता है, अन्यथा `false`।

### अपवाद

| अपवाद | शर्त |
| --- | --- |
| ArgumentException | जब resolution मान मान्य नहीं हो। |

### टिप्पणियाँ

यह मेथड चित्र के आकार और रिज़ॉल्यूशन को PowerPoint की "Picture Format → Compress Pictures" सुविधा के समान बदलता है।

### उदाहरण

निम्न उदाहरण दर्शाता है कि कैसे `CompressImage` मेथड का उपयोग करके प्रस्तुति में किसी चित्र का आकार लक्ष्य रिज़ॉल्यूशन निर्धारित करके और क्रॉप किए गए क्षेत्रों को हटाकर घटाया जा सकता है:

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;
    // छवि को 150 DPI (वेब रिज़ॉल्यूशन) के लक्ष्य रिज़ॉल्यूशन के साथ संकुचित करें और क्रॉप किए गए क्षेत्रों को हटाएँ
    bool result = picFrame.PictureFormat.CompressImage(true, PicturesCompression.Dpi150);
}
```

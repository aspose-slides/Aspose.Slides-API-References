## PictureFillFormat.TileScaleY प्रॉपर्टी

वर्टिकल स्केल को टेक्टचर फ़िल के लिए प्रतिशत के रूप में प्राप्त करता है या सेट करता है। पढ़ें/लिखें Single.

```csharp
public float TileScaleY { get; set; }
```

### उदाहरण

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // शेप का पिक्चर फ़िल फ़ॉर्मेट प्राप्त करता है
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // पिक्चर फ़िल मोड को टाइल पर सेट करता है
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // टेक्सचर के लिए वर्टिकल स्केल को 120 प्रतिशत पर सेट करता है
    pictureFillFormat.TileScaleY = 120;
}
```

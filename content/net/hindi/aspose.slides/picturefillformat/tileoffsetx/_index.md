## PictureFillFormat.TileOffsetX गुण

टेक्सचर को आकार की मूल बिंदु से क्षैतिज ऑफ़सेट को पॉइंट्स में लौटाता है या सेट करता है। एक सकारात्मक मान टेक्सचर को दाईं ओर ले जाता है, जबकि नकारात्मक मान इसे बाईं ओर ले जाता है। पढ़ें/लिखें Single.

```csharp
public float TileOffsetX { get; set; }
```

### उदाहरण

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // शेप का चित्र भरने का प्रारूप प्राप्त करता है
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // चित्र भरने के मोड को टाइल पर सेट करता है
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // टेक्सचर का क्षैतिज ऑफ़सेट 20 पॉइंट्स पर सेट करता है
    pictureFillFormat.TileOffsetX = 20f;
}
```

## PictureFillFormat.DeletePictureCroppedAreas मेथड

फ़िल चित्र के क्रॉप किए गए क्षेत्रों को हटाता है।

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### रिटर्न मान

यदि क्रॉपिंग आवश्यक नहीं है तो क्रॉप किया गया चित्र या मूल चित्र लौटाता है।

### टिप्पणियाँ

यह मेथड WMF/EMF मेटा फ़ाइलों को रास्टर PNG चित्र में बदलता है तथा क्रॉपिंग करता है।

### उदाहरण

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // PictureFrame प्राप्त करता है
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // PictureFrame छवि के क्रॉप किए गए क्षेत्रों को हटाता है
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

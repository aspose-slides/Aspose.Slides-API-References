## PictureFillFormat.TileOffsetY प्रॉपर्टी

टेक्सचर को शैप की मूल स्थिति से बिंदुओं में ऊर्ध्वाधर ऑफ़सेट लौटाता है या सेट करता है। एक सकारात्मक मान टेक्सचर को नीचे ले जाता है, जबकि नकारात्मक मान इसे ऊपर ले जाता है। पढ़ें/लिखें Single.

```csharp
public float TileOffsetY { get; set; }
```

### उदाहरण

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // शैप के चित्र भरने के फ़ॉर्मेट को प्राप्त करता है
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // चित्र भरने के मोड को टाइल पर सेट करता है
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // टेक्सचर के ऊर्ध्वाधर ऑफ़सेट को -50 पॉइंट्स पर सेट करता है
    pictureFillFormat.TileOffsetY = -50;
}
```

## IZoomObject.ShowBackground प्रॉपर्टी

Zoom यह निर्धारित करने के लिए मान प्राप्त करता है या सेट करता है कि क्या गंतव्य स्लाइड की पृष्ठभूमि का उपयोग किया जाएगा। पढ़ने/लिखने योग्य बूलियन। डिफ़ॉल्ट मान: true

```csharp
public bool ShowBackground { get; set; }
```

### उदाहरण

यह उदाहरण Zoom ऑब्जेक्ट की छवि की पृष्ठभूमि हटाने का प्रदर्शन करता है:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
    zoomFrame.ShowBackground = false;
}
```

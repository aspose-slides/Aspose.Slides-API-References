## IZoomObject.TransitionDuration प्रॉपर्टी

Zoom और स्लाइड के बीच ट्रांज़िशन की अवधि प्राप्त करता है या सेट करता है। पढ़ें/लिखें Single। डिफ़ॉल्ट मान: 1.0f

```csharp
public float TransitionDuration { get; set; }
```

### टिप्पणियाँ

यदि निर्दिष्ट नहीं किया गया है (TransitionDur = 0), तो यह गंतव्य स्लाइड ट्रांज़िशन और उस ट्रांज़िशन से जुड़े टाइमिंग्स का उपयोग करेगा।

### उदाहरण

उदाहरण दिखाता है कि Zoom और स्लाइड के बीच ट्रांज़िशन की अवधि को कैसे बदलें:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
    zoomFrame.TransitionDuration = 2.5f;
}
```

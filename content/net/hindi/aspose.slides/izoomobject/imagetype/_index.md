## IZoomObject.ImageType प्रॉपर्टी

ज़ूम ऑब्जेक्ट का इमेज टाइप प्राप्त करता है या सेट करता है। पढ़ें/लिखें [`ZoomImageType`](../../zoomimagetype). डिफ़ॉल्ट मान: Preview

### टिप्पणियाँ

निर्दिष्ट करता है कि ज़ूम ऑब्जेक्ट स्लाइड प्रीव्यू या कवर इमेज का उपयोग कर रहा है।

### उदाहरण

यह उदाहरण इमेज टाइप को Preview मान में बदलने को दर्शाता है। इस मामले में ज़ूम ऑब्जेक्ट की वर्तमान इमेज स्लाइड इमेज में बदल जाती है:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    IPPImage image = pres.Images.AddImage(Image.FromFile("image.png"));
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1], image);
    zoomFrame.ImageType = ZoomImageType.Preview;
}
```

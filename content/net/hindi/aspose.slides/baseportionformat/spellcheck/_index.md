## BasePortionFormat.SpellCheck प्रॉपर्टी

टेक्स्ट हिस्से के लिए स्पेल चेकिंग सक्षम है या नहीं, यह दर्शाने वाला मान प्राप्त करता है या सेट करता है। जब यह प्रॉपर्टी false पर सेट की जाती है, तो टेक्स्ट तत्वों के लिए स्पेलिंग जाँच दबा दी जाती है। जब इसे true पर सेट किया जाता है, तो स्पेल चेकिंग की अनुमति होती है। डिफ़ॉल्ट मान `false` है।

```csharp
public bool SpellCheck { get; set; }
```

### उदाहरण

अगला उदाहरण प्रस्तुति को सहेजने से पहले SpellCheck फ़्लैग को सक्षम करने को दर्शाता है:

```csharp
[C#]
using (var pres = new Presentation("input.pptx"))
{
    // पहली स्लाइड पर पहले शपे के अंदर पहले टेक्स्ट भाग तक पहुँचें
    var portion = ((AutoShape)pres.Slides[0].Shapes[0]).TextFrame.Paragraphs[0].Portions[0];
    // इस टेक्स्ट भाग के लिए स्पेल चेकिंग सक्षम करें
    portion.PortionFormat.SpellCheck = true;
    // संशोधित प्रस्तुति को सहेजें
    pres.Save("output-with-spellcheck.pptx", SaveFormat.Pptx);
}
```

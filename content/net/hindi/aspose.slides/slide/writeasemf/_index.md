## Slide.WriteAsEmf विधि

स्लाइड सामग्री को EMF फ़ाइल के रूप में सहेजता है।

```csharp
public void WriteAsEmf(Stream stream)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| stream | Stream | लक्षित स्ट्रीम |

### अपवाद

| अपवाद | स्थिति |
| --- | --- |
| ArgumentNullException | लक्षित स्ट्रीम `null` है |

### उदाहरण

निम्नलिखित कोड उदाहरण दर्शाता है कि कैसे PowerPoint प्रस्तुति की पहली स्लाइड को एक मेटाफाइल में परिवर्तित किया जाए।

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.emf"))
    {
        // पहले स्लाइड को एक मेटाफाइल के रूप में सहेजता है
        pres.Slides[0].WriteAsEmf(fileStream);
    }
}
```

## WriteAsSvg(Stream, ISVGOptions) {#writeassvg_1}

स्लाइड सामग्री को SVG फ़ाइल के रूप में सहेजता है।

```csharp
public void WriteAsSvg(Stream stream, ISVGOptions svgOptions)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| stream | Stream | लक्ष्य stream |
| svgOptions | ISVGOptions | SVG जनरेशन विकल्प |

### उदाहरण

निम्नलिखित कोड उदाहरण दर्शाता है कि कैसे PowerPoint प्रस्तुति की पहली स्लाइड को विकल्पों के साथ SVG फ़ाइल में परिवर्तित किया जा सकता है।

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.svg"))
    {
        var options = new SVGOptions() { VectorizeText = true };
        // पहला स्लाइड को SVG फ़ाइल के रूप में सहेजता है
        pres.Slides[0].WriteAsSvg(fileStream, options);
    }
}
```

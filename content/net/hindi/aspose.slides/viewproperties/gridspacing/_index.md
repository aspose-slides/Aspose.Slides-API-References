## ViewProperties.GridSpacing प्रॉपर्टी

प्रस्तुति दस्तावेज़ के नीचे मौजूद ग्रिड के लिए उपयोग किए जाने वाले ग्रिड स्पेसिंग को लौटाता है या सेट करता है, पॉइंट्स में। पढ़ें/लिखें Single.

```csharp
public float GridSpacing { get; set; }
```

### टिप्पणी

ग्रिड स्पेसिंग मान एक सकारात्मक संख्या होनी चाहिए। सामान्य मान रेंज 1 मिमी (2.8349607 पॉइंट्स) से 2 इंच (144 पॉइंट्स) तक है।

### उदाहरण

निम्नलिखित नमूना कोड दिखाता है कि PowerPoint प्रस्तुति में ग्रिड स्पेसिंग को कैसे बदलें।

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

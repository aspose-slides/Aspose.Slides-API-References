## TextFrame.ReplaceText मेथड

निर्दिष्ट पाठ की सभी घटनाओं को दूसरे निर्दिष्ट पाठ से बदलता है।

```csharp
public void ReplaceText(string oldText, string newText, ITextSearchOptions options, 
    IFindResultCallback callback)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| oldText | String | बदलने के लिए स्ट्रिंग। |
| newText | String | oldText की सभी घटनाओं को बदलने के लिए स्ट्रिंग। |
| options | ITextSearchOptions | पाठ खोज विकल्प [`ITextSearchOptions`](../../itextsearchoptions)। |
| callback | IFindResultCallback | प्रतिस्थापन ऑपरेशन परिणाम को सहेजने के लिए कॉलबैक ऑब्जेक्ट [`IFindResultCallback`](../../ifindresultcallback)। |

### उदाहरण

निम्नलिखित नमूना कोड दर्शाता है कि कैसे एक निर्दिष्ट स्ट्रिंग को दूसरे निर्दिष्ट स्ट्रिंग से बदला जाए।

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx")){
	// सभी अलग-अलग 'the' घटनाओं को '***' से बदलें
	((AutoShape)presentation.Slides[0].Shapes[0]).TextFrame.ReplaceText("the", "***", new TextSearchOptions()
	{ WholeWordsOnly = true }, null);
	presentation.Save("SomePresentation-out2.pptx", SaveFormat.Pptx);
}
```

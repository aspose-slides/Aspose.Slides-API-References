## TextFrame.HighlightRegex मेथड

निर्दिष्ट रंग के साथ नियमित अभिव्यक्ति की सभी मैचों को उजागर करता है।

```csharp
public void HighlightRegex(Regex regex, Color highlightColor, IFindResultCallback callback)
```

| Parameter | Type | Description |
| --- | --- | --- |
| regex | Regex | उन स्ट्रिंग्स को प्राप्त करने के लिए नियमित अभिव्यक्ति Regex जो उजागर की जाएँ। |
| highlightColor | Color | पाठ को उजागर करने के लिए रंग। |
| callback | IFindResultCallback | खोज परिणाम प्राप्त करने के लिए कॉलबैक ऑब्जेक्ट [`IFindResultCallback`](../../ifindresultcallback)। |

### उदाहरण

निम्न कोड नमूना दिखाता है कि कैसे नियमित अभिव्यक्ति का उपयोग करके TextFrame में पाठ को उजागर किया जाए।

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// 10 या अधिक अक्षर वाले सभी शब्दों को उजागर करना
	((AutoShape)presentation.Slides[0].Shapes[0]).TextFrame.HighlightRegex(regex, Color.Blue, null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

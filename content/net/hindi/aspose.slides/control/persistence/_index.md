## Control.Persistence गुण

ActiveX नियंत्रण के गुणों को संग्रहीत करने के लिए उपयोग की जाने वाली विधि को प्राप्त करता है। केवल पढ़ने योग्य [`PersistenceType`](../../persistencetype).

```csharp
public PersistenceType Persistence { get; }
```

### उदाहरण

अगला उदाहरण Persistence गुण का उपयोग दिखाता है जिससे जांचा जा सके कि ActiveX ऑब्जेक्ट के गुण XML-आधारित ActiveX गुणों के रूप में बदले जा सकते हैं या नहीं:

```csharp
[C#]
if (control.Persistence == PersistenceType.PersistPropertyBag)
{
    control.Properties["Value"] = value;
}
else
{
    YourMethodHere(control.ActiveXControlBinary); //ActiveX गुणों को उसकी बाइनरी फ़ाइल में संग्रहीत करने के लिए अपनी स्वयं की विधि का उपयोग करें
}
```

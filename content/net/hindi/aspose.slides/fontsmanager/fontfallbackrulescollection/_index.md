## FontsManager.FontFallBackRulesCollection प्रॉपर्टी

FontFallBack नियमों के उपयोगकर्ता संग्रह को दर्शाता है, फॉंट्स के संग्रह को उचित प्रतिस्थापन के लिए फॉलबैक कार्यक्षमता द्वारा प्रबंधित करने हेतु पढ़ें/लिखें [`IFontFallBackRulesCollection`](../../ifontfallbackrulescollection).

```csharp
public IFontFallBackRulesCollection FontFallBackRulesCollection { get; set; }
```

### उदाहरण

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    // FontsManager से खाली या पहले से प्रारम्भिकीकृत नियमों का संग्रह प्राप्त करना
    IFontFallBackRulesCollection rulesList = pres.FontsManager.FontFallBackRulesCollection;

    // संग्रह में नियम जोड़ना
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));

    // या 
    // नियम संग्रह की नई इंस्टेंस का प्रारम्भिकीकरण
    IFontFallBackRulesCollection rulesList = new FontFallBackRulesCollection();

    // संग्रह में नियम जोड़ना
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));

    // FontsManager में नई संग्रह द्वारा मौजूदा संग्रह को बदलना 
    pres.FontsManager.FontFallBackRulesCollection = rulesList;
}
```

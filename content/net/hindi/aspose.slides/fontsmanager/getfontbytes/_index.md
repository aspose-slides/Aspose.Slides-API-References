## FontsManager.GetFontBytes मेथड

निर्दिष्ट फ़ॉन्ट स्टाइल और फ़ॉन्ट डेटा के लिए फ़ॉन्ट डेटा का प्रतिनिधित्व करने वाला बाइट एरे प्राप्त करता है।

```csharp
public byte[] GetFontBytes(IFontData fontData, FontStyleType fontStyle)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| fontData | IFontData | फ़ॉन्ट डेटा ऑब्जेक्ट जिसमें फ़ॉन्ट की जानकारी है [`IFontData`](../../ifontdata). |
| fontStyle | FontStyleType | फ़ॉन्ट का वह शैली जिसके लिए डेटा प्राप्त किया जाना है [`FontStyleType`](../../fontstyletype). |

### वापसी मान

निर्दिष्ट फ़ॉन्ट स्टाइल के लिए फ़ॉन्ट डेटा वाला बाइट एरे। यदि फ़ॉन्ट डेटा या स्टाइल नहीं मिलता है, तो null लौटाता है।

### उदाहरण

```csharp
[C#]
using (Presentation pres = new Presentation ("Presentation.pptx"))
{
    // प्रस्तुति में उपयोग किए गए सभी फ़ॉन्ट प्राप्त करें
    IFontData[] fonts = pres.FontsManager.GetFonts();

    // प्रस्तुति में पहले फ़ॉन्ट की नियमित शैली का प्रतिनिधित्व करने वाले बाइट एरे को प्राप्त करें
    byte[] bytes = pres.FontsManager.GetFontBytes(fonts[0], FontStyleType.Regular);
}
```

## IBlobManagementOptions.PresentationLockingBehavior प्रॉपर्टी

यह प्रॉपर्टी निर्धारित करता है कि क्या Presentation क्लास की एक उदाहरण स्रोत - फ़ाइल या स्ट्रीम का मालिक हो सकता है, उदाहरण के जीवनकाल के दौरान। यदि उदाहरण मालिक है, तो यह स्रोत को लॉक कर देता है। यह BLOBs के साथ काम करते समय स्मृति उपयोग और प्रदर्शन को सुधारने में मदद करता है, लेकिन स्रोत (स्ट्रीम या फ़ाइल) को Presentation के उदाहरण जीवनकाल के दौरान बदला नहीं जा सकता। यह एक उदाहरण है:

```csharp
public PresentationLockingBehavior PresentationLockingBehavior { get; set; }
```

### उदाहरण

```csharp
LoadOptions loadOptions = new LoadOptions { 
  BlobManagementOptions = {PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked} };
using (Presentation pres = new Presentation("pres.pptx", loadOptions)) {
  // IOException फेंका जाएगा क्योंकि pres.pptx को Presentation के जीवनकाल के लिए लॉक किया गया है
  // File.Delete("pres.pptx");
}
// after Presentation ऑब्जेक्ट के निपटाए जाने के बाद, फ़ाइल अनलॉक हो जाती है और इसे हटाया जा सकता है
File.Delete("pres.pptx");
```

## IGeometryShape.SetGeometryPaths पद्धति

एक [`IGeometryPath`](../../igeometrypath) की array से आकृति ज्यामिति को अद्यतन करता है। निर्देशांक आकृति के बाएँ शीर्ष कोने के सापेक्ष होने चाहिए। आकृति ([`ShapeType`](../shapetype)) के प्रकार को कस्टम में बदलता है।

```csharp
public void SetGeometryPaths(IGeometryPath[] geometryPaths)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| geometryPaths | IGeometryPath[] | ज्यामिति पथों की array |

### अपवाद

| अपवाद | शर्त |
| --- | --- |
| ArgumentException | कोई पथ नहीं मिला |
| ArgumentException | खाली पथ |

### उदाहरण

उदाहरण:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    GeometryShape shape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 200, 100) as GeometryShape;

    GeometryPath geometryPath = shape.GetGeometryPaths()[0];

    geometryPath.LineTo(100, 50, 1);
    geometryPath.LineTo(100, 50, 4);

    shape.SetGeometryPath(geometryPath);

    pres.Save("output.pptx", SaveFormat.Pptx);
}
```

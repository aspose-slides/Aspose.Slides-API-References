## AddConnector(ShapeType, float, float, float, float) {#addconnector}

डिफ़ॉल्ट टेम्प्लेट शैली के साथ एक नया कनेक्टर शेप बनाता है और इसे शेप कलेक्शन के अंत में जोड़ता है।

```csharp
public IConnector AddConnector(ShapeType shapeType, float x, float y, float width, float height)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| shapeType | ShapeType | कनेक्टर शेप का [`ShapeType`](../../shapetype) जिसे जोड़ना है। |
| x | Single | कनेक्टर के फ्रेम का x-निर्देशांक, पॉइंट्स में। |
| y | Single | कनेक्टर के फ्रेम का y-निर्देशांक, पॉइंट्स में। |
| width | Single | कनेक्टर के फ्रेम की चौड़ाई, पॉइंट्स में। |

नया बनाया गया [`IConnector`](../../iconnector)।

### उदाहरण

निम्नलिखित उदाहरण दिखाता है कि PowerPoint प्रस्तुति में दो शेप्स (एक अंडाकार और आयत) के बीच एक कनेक्टर (वक्र कनेक्टर) कैसे जोड़ा जाए।

```csharp
[C#]
// एक प्रस्तुति क्लास का उदाहरण बनाता है जो PPTX फ़ाइल का प्रतिनिधित्व करता है
using (Presentation input = new Presentation())
{
    // एक विशिष्ट स्लाइड के लिए शैप्स कलेक्शन तक पहुंचता है
    IShapeCollection shapes = input.Slides[0].Shapes;
    // एक Ellipse ऑटोशेप जोड़ता है
    IAutoShape ellipse = shapes.AddAutoShape(ShapeType.Ellipse, 0, 100, 100, 100);
    // एक Rectangle ऑटोशेप जोड़ता है
    IAutoShape rectangle = shapes.AddAutoShape(ShapeType.Rectangle, 100, 300, 100, 100);
    // स्लाइड शैप कलेक्शन में एक कनेक्टर शेप जोड़ता है
    IConnector connector = shapes.AddConnector(ShapeType.BentConnector2, 0, 0, 10, 10);
    // कनेक्टर का उपयोग कर शैप्स को कनेक्ट करता है
    connector.StartShapeConnectedTo = ellipse;
    connector.EndShapeConnectedTo = rectangle;
    // reroute को कॉल करता है जो शैप्स के बीच स्वचालित सबसे छोटा पथ सेट करता है
    connector.Reroute();
    // प्रेजेंटेशन को सहेजता है
    input.Save("Shapes-connector.pptx", SaveFormat.Pptx);
}
```

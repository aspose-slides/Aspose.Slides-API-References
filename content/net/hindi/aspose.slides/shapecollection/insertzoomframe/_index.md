## InsertZoomFrame(int, float, float, float, float, ISlide) {#insertzoomframe}

नया Zoom फ्रेम बनाता है और निर्दिष्ट अनुक्रमणिका पर इसे shape संग्रह में सम्मिलित करता है।

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| index | Int32 | Zoom फ्रेम को सम्मिलित करने के लिए शून्य-आधारित अनुक्रमणिका। |
| x | Single | नए Zoom फ्रेम का x-निर्देशांक, पॉइंट्स में। |
| y | Single | नए Zoom फ्रेम का y-निर्देशांक, पॉइंट्स में। |
| width | Single | नए Zoom फ्रेम की चौड़ाई, पॉइंट्स में। |
| height | Single | नए Zoom फ्रेम की ऊँचाई, पॉइंट्स में। |
| slide | ISlide | Zoom फ्रेम द्वारा संदर्भित [`ISlide`](../../islide)। |

### रिटर्न वैल्यू

नया निर्मित [`IZoomFrame`](../../izoomframe)।

### अपवाद

| exception | condition |
| --- | --- |
| ArgumentException | यदि संदर्भित स्लाइड वर्तमान प्रस्तुति का हिस्सा नहीं है तो फेंका जाता है। |

### उदाहरण

यह उदाहरण एक संग्रह में निर्दिष्ट अनुक्रमणिका पर Zoom ऑब्जेक्ट के निर्माण और सम्मिलन को दर्शाता है (मान लें कि "Presentation.pptx" प्रस्तुति में कम से कम दो स्लाइड हैं):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertZoomFrame(2, 150, 20, 50, 50, pres.Slides[1]);
}
```

## InsertAudioFrameEmbedded(int, float, float, float, float, Stream) {#insertaudioframeembedded_1}

एक नया ऑडियो फ्रेम बनाता है जिसमें एक एम्बेडेड WAV फ़ाइल होती है और इसे निर्दिष्ट अनुक्रमणिका पर shape संग्रह में सम्मिलित करता है। एम्बेडेड ऑडियो को Presentation.Audios संग्रह में जोड़ा जाता है।

```csharp
public IAudioFrame InsertAudioFrameEmbedded(int index, float x, float y, float width, float height, 
    Stream audio_stream)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| index | Int32 | ऑडियो फ्रेम को सम्मिलित करने के लिए शून्य-आधारित इंडेक्स। |
| x | Single | नए ऑडियो फ्रेम का x-निर्देशांक, पॉइंट्स में। |
| y | Single | नए ऑडियो फ्रेम का y-निर्देशांक, पॉइंट्स में। |
| width | Single | नए ऑडियो फ्रेम की चौड़ाई, पॉइंट्स में। |
| height | Single | नए ऑडियो फ्रेम की ऊँचाई, पॉइंट्स में। |
| audio_stream | Stream | WAV ऑडियो डेटा को एम्बेड करने वाला एक इनपुट स्ट्रीम। |

## ShapeCollection.InsertSummaryZoomFrame विधि

एक नया Summary Zoom फ्रेम बनाता है और इसे निर्दिष्ट इंडेक्स पर shape collection में सम्मिलित करता है।

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| पैरामीटर | टाइप | विवरण |
| --- | --- | --- |
| index | Int32 | वह शून्य-आधारित इंडेक्स जहाँ Summary Zoom फ्रेम को सम्मिलित किया जाना है। |
| x | Single | नए Summary Zoom फ्रेम का x-निर्देशांक, पॉइंट्स में। |
| y | Single | नए Summary Zoom फ्रेम का y-निर्देशांक, पॉइंट्स में। |
| width | Single | नए Summary Zoom फ्रेम की चौड़ाई, पॉइंट्स में। |
| height | Single | नए Summary Zoom फ्रेम की ऊँचाई, पॉइंट्स में। |

### रिटर्न मान

नया बनाया गया [`ISummaryZoomFrame`](../../isummaryzoomframe)।

| अपवाद | स्थिति |

### टिप्पणियाँ

यह विधि एक Summary Zoom फ्रेम बनाती है जो प्रस्तुति में सभी सेक्शन के सार लिंक को एकत्रित करता है।

### उदाहरण

यह उदाहरण एक Summary Zoom ऑब्जेक्ट को कलेक्शन के निर्दिष्ट इंडेक्स पर बनाने और सम्मिलित करने को दर्शाता है (मान लीजिए "Presentation.pptx" प्रस्तुति में कम से कम दो सेक्शन हैं):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSummaryZoomFrame(2, 150, 20, 50, 50);
}
```

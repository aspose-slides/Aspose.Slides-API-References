## InsertAutoShape(int, ShapeType, float, float, float, float) {#insertautoshape}

public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width,

## InsertAutoShape(int, ShapeType, float, float, float, float, bool) {#insertautoshape_1}

एक नया ऑटो शैप बनाता है और इसे निर्दिष्ट इंडेक्स पर शैप संग्रह में सम्मिलित करता है, वैकल्पिक रूप से इसे डिफ़ॉल्ट टेम्प्लेट स्टाइलिंग के साथ प्रारंभ करता है।

```csharp
public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| index | Int32 | ऑटो शैप को सम्मिलित करने के लिए शून्य-आधारित इंडेक्स। |
| shapeType | ShapeType | सम्मिलित करने वाले ऑटो शैप का [`ShapeType`](../../shapetype)। |
| x | Single | शैप के फ्रेम का x-निर्देशांक, पॉइंट्स में। |
| y | Single | शैप के फ्रेम का y-निर्देशांक, पॉइंट्स में। |
| width | Single | शैप के फ्रेम की चौड़ाई, पॉइंट्स में। |
| height | Single | शैप के फ्रेम की ऊँचाई, पॉइंट्स में। |
| createFromTemplate | Boolean | डिफ़ॉल्ट टेम्प्लेट स्टाइलिंग लागू करने के लिए true (जिसमें गैर-खाली नाम, साधारण शैली, और केंद्रित टेक्स्ट शामिल है); false होने पर सभी गुणों को उनके डिफ़ॉल्ट मानों पर सेट कर शैप बनाया जाता है। |

## InsertChart(ChartType, float, float, float, float, int) {#insertchart}

एक नया चार्ट बनाता है, इसे नमूना सीरीज़ डेटा और सेटिंग्स के साथ प्रारंभ करता है, और इसे निर्दिष्ट इंडेक्स पर shape collection में सम्मिलित करता है।

| y | Single | नए चार्ट का y-निर्देशांक, पॉइंट्स में। |
| width | Single | नए चार्ट की चौड़ाई, पॉइंट्स में। |
| height | Single | नए चार्ट की ऊँचाई, पॉइंट्स में। |
| index | Int32 | शून्य-आधारित इंडेक्स, जिस पर नया चार्ट shape collection में सम्मिलित किया जाएगा। |

## InsertChart(ChartType, float, float, float, float, int, bool) {#insertchart_1}

एक नया चार्ट बनाता है, इसे नमूना सीरीज़ डेटा और सेटिंग्स के साथ प्रारंभ करता है, और इसे निर्दिष्ट इंडेक्स पर shape collection में सम्मिलित करता है।

```csharp
public IChart InsertChart(ChartType type, float x, float y, float width, float height, int index, 
    bool initWithSample)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| type | ChartType | बनाने के लिए चार्ट का प्रकार। |
| x | Single | नए चार्ट का x-निर्देशांक, पॉइंट्स में। |
| y | Single | नए चार्ट का y-निर्देशांक, पॉइंट्स में। |
| width | Single | नए चार्ट की चौड़ाई, पॉइंट्स में। |
| height | Single | नए चार्ट की ऊँचाई, पॉइंट्स में। |
| index | Int32 | शून्य-आधारित इंडेक्स, जिस पर नया चार्ट shape collection में सम्मिलित किया जाएगा। |
| initWithSample | Boolean | सही (true) होने पर नया चार्ट नमूना सीरीज़ डेटा और सेटिंग्स के साथ प्रारंभ किया जाता है; गलत (false) होने पर चार्ट को बिना सीरीज़ और केवल न्यूनतम सेटिंग्स के साथ बनाया जाता है, जिससे निर्माण तेज़ होता है। |

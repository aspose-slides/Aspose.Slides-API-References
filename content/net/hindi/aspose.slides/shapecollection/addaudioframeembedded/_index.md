## AddAudioFrameEmbedded(float, float, float, float, Stream) {#addaudioframeembedded_1}

नया ऑडियो फ़्रेम बनाता है जिसमें एम्बेडेड WAV फ़ाइल होती है और इसे शेप कलेक्शन के अंत में जोड़ता है। एम्बेडेड ऑडियो को Presentation.Audios संग्रह में जोड़ा जाता है।

```csharp
public IAudioFrame AddAudioFrameEmbedded(float x, float y, float width, float height, 
    Stream audio_stream)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| x | Single | नए ऑडियो फ़्रेम का x-कोऑर्डिनेट, पॉइंट्स में। |
| y | Single | नए ऑडियो फ़्रेम का y-कोऑर्डिनेट, पॉइंट्स में। |
| width | Single | नए ऑडियो फ़्रेम की चौड़ाई, पॉइंट्स में। |
| height | Single | नए ऑडियो फ़्रेम की ऊँचाई, पॉइंट्स में। |
| audio_stream | Stream | एम्बेड करने के लिए WAV ऑडियो डेटा वाला इनपुट स्ट्रीम। |

### वापसी मान

नया बनाया गया [`IAudioFrame`](../../iaudioframe)।

### उदाहरण

निम्नलिखित उदाहरण दर्शाते हैं कि ऑडियो फ़्रेम कैसे बनाया जाए।

```csharp
[C#]
// एक प्रेजेंटेशन फ़ाइल का प्रतिनिधित्व करने वाली प्रेजेंटेशन क्लास को इंस्टैंशिएट करता है
using (Presentation pres = new Presentation())
{
    // पहला स्लाइड प्राप्त करता है
    ISlide sld = pres.Slides[0];
    // wav साउंड फ़ाइल को स्ट्रीम में लोड करता है
    FileStream fstr = new FileStream("sampleaudio.wav", FileMode.Open, FileAccess.Read);
    // ऑडियो फ़्रेम जोड़ता है
    IAudioFrame audioFrame = sld.Shapes.AddAudioFrameEmbedded(50, 150, 100, 100, fstr);
    // ऑडियो की प्ले मोड और वॉल्यूम सेट करता है
    audioFrame.PlayMode = AudioPlayModePreset.Auto;
    audioFrame.Volume = AudioVolumeMode.Loud;
    // PowerPoint फ़ाइल को डिस्क पर लिखता है
    pres.Save("AudioFrameEmbed_out.pptx", SaveFormat.Pptx);
}
```

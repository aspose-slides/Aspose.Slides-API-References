## ShapeCollection.InsertAudioFrameCD विधि

एक नया ऑडियो फ़्रेम जो CD ट्रैक से जुड़ा है, बनाता है और इसे निर्दिष्ट सूचकांक पर shape collection में सम्मिलित करता है।

```csharp
public IAudioFrame InsertAudioFrameCD(int index, float x, float y, float width, float height)
```

| परामीटर | प्रकार | विवरण |
| --- | --- | --- |
| index | Int32 | ऑडियो फ़्रेम को सम्मिलित करने के लिए शून्य-आधारित सूचकांक। |
| x | Single | नए ऑडियो फ़्रेम का x-निर्देशांक, पॉइंट्स में। |
| y | Single | नए ऑडियो फ़्रेम का y-निर्देशांक, पॉइंट्स में। |
| width | Single | नए ऑडियो फ़्रेम की चौड़ाई, पॉइंट्स में। |
| height | Single | नए ऑडियो फ़्रेम की ऊँचाई, पॉइंट्स में। |

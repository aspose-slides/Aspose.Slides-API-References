## AddZoomFrame(float, float, float, float, ISlide) {#addzoomframe}

एक नया Zoom फ्रेम बनाता है और इसे शेप संग्रह के अंत में जोड़ता है।

```csharp
public IZoomFrame AddZoomFrame(float x, float y, float width, float height, ISlide slide)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| x | Single | नए Zoom फ्रेम का x-निर्देशांक, बिंदुओं में। |
| y | Single | नए Zoom फ्रेम का y-निर्देशांक, बिंदुओं में। |
| width | Single | नए Zoom फ्रेम की चौड़ाई, बिंदुओं में। |
| height | Single | नए Zoom फ्रेम की ऊंचाई, बिंदुओं में। |
| slide | ISlide | Zoom फ्रेम द्वारा संदर्भित [`ISlide`](../../islide); यह इस प्रस्तुति का हिस्सा होना चाहिए। |

### वापसी मान

नव निर्मित [`IZoomFrame`](../../izoomframe)।

### अपवाद

| अपवाद | स्थिति |
| --- | --- |
| ArgumentException | यदि संदर्भित स्लाइड वर्तमान प्रस्तुति का हिस्सा नहीं है तो उत्पन्न होता है। |

### उदाहरण

यह उदाहरण एक Zoom ऑब्जेक्ट को संग्रह के अंत में जोड़ने को दर्शाता है (मान लें कि "Presentation.pptx" प्रस्तुति में कम से कम दो स्लाइड हैं):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
}
```

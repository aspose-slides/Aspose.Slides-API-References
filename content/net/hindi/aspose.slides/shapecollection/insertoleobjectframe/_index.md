## InsertOleObjectFrame(int, float, float, float, float, IOleEmbeddedDataInfo) {#insertoleobjectframe}

नया OLE ऑब्जेक्ट फ्रेम बनाता है और इसे निर्दिष्ट इंडेक्स पर shape collection में डालता है।

```csharp
public IOleObjectFrame InsertOleObjectFrame(int index, float x, float y, float width, float height, 
    IOleEmbeddedDataInfo dataInfo)
```

| पैरामीटर | Type | विवरण |
| --- | --- | --- |
| index | Int32 | वह शून्य-आधारित इंडेक्स जहाँ OLE ऑब्जेक्ट फ्रेम को डालना है। |
| x | Single | नई OLE फ्रेम का x-निर्देशांक, पॉइंट्स में। |
| y | Single | नई OLE फ्रेम का y-निर्देशांक, पॉइंट्स में। |
| width | Single | नई OLE फ्रेम की चौड़ाई, पॉइंट्स में। |
| height | Single | नई OLE फ्रेम की ऊँचाई, पॉइंट्स में। |
| dataInfo | IOleEmbeddedDataInfo | एम्बेडेड OLE डेटा जानकारी ([`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo))। |

### Return Value

नया बनाया गया [`IOleObjectFrame`](../../ioleobjectframe)।

### Examples

यह उदाहरण दूसरे इंडेक्स पर OLE ऑब्जेक्ट डालने को दर्शाता है:

```csharp
[C#]
byte[] fileData = File.ReadAllBytes("test.zip");
IOleDataInfo dataInfo = new OleDataInfo(fileData, "zip");
IOleObjectFrame oleObjectFrame = slidees.Shapes.AddOleObjectFrame(2, 150, 20, 50, 50, dataInfo);
```

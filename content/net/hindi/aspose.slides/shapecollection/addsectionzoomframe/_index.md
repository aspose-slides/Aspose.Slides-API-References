## AddSectionZoomFrame(float, float, float, float, ISection) {#addsectionzoomframe}

एक नया Section Zoom फ्रेम बनाता है और इसे shape संग्रह के अंत में जोड़ता है।

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| x | Single | नए Section Zoom फ्रेम का x-निर्देशांक, बिंदु में। |
| y | Single | नए Section Zoom फ्रेम का y-निर्देशांक, बिंदु में। |
| width | Single | नए Section Zoom फ्रेम की चौड़ाई, बिंदु में। |
| height | Single | नए Section Zoom फ्रेम की ऊँचाई, बिंदु में। |
| section | ISection | Section Zoom फ्रेम द्वारा संदर्भित [`ISection`](../../isection); यह इस प्रस्तुति से संबंधित होना चाहिए और कम से कम एक स्लाइड शामिल करे। |

### रिटर्न मान

नया बनाया गया [`ISectionZoomFrame`](../../isectionzoomframe)।

### अपवाद

| अपवाद | शर्त |
| --- | --- |
| ArgumentException | यदि संदर्भित सेक्शन वर्तमान प्रस्तुतीकरण से संबंधित नहीं है या इसमें कोई स्लाइड नहीं है तो फेंका जाता है। |

### उदाहरण

यह उदाहरण एक Section Zoom ऑब्जेक्ट को संग्रह के अंत में जोड़ने को दर्शाता है (मान लें कि "Presentation.pptx" प्रस्तुति में कम से कम दो सेक्शन हैं):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1]);
}
```

## ShapeCollection.InsertPictureFrame मेथड

निर्दिष्ट छवि को शामिल करने वाला नया पिक्चर फ्रेम बनाता है और इसे निर्दिष्ट सूचकांक पर आकार संग्रह में सम्मिलित करता है।

```csharp
public IPictureFrame InsertPictureFrame(int index, ShapeType shapeType, float x, float y, 
    float width, float height, IPPImage image)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| index | Int32 | वह शून्य-आधारित सूचकांक जहाँ पिक्चर फ्रेम को सम्मिलित किया जाना है। |
| shapeType | ShapeType | [`ShapeType`](../../shapetype) में निहित शेप प्रकार को निर्दिष्ट करता है, सभी प्रकार की रेखाओं को छोड़कर: |
| x | Single | पिक्चर फ्रेम का x-निर्देशांक, पॉइंट्स में। |
| y | Single | पिक्चर फ्रेम का y-निर्देशांक, पॉइंट्स में। |
| width | Single | पिक्चर फ्रेम की चौड़ाई, पॉइंट्स में। |
| height | Single | पिक्चर फ्रेम की ऊँचाई, पॉइंट्स में। |
| image | IPPImage | [`IPPImage`](../../ippimage) जिसे पिक्चर फ्रेम में प्रदर्शित किया जाएगा। |

## ShapeCollection.AddSummaryZoomFrame method

नया Summary Zoom फ्रेम बनाता है और उसे शेप कलेक्शन के अंत में जोड़ता है।

```csharp
public ISummaryZoomFrame AddSummaryZoomFrame(float x, float y, float width, float height)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| x | Single | नए Summary Zoom फ्रेम का x-निर्देशांक, पॉइंट्स में। |
| y | Single | नए Summary Zoom फ्रेम का y-निर्देशांक, पॉइंट्स में। |
| width | Single | नए Summary Zoom फ्रेम की चौड़ाई, पॉइंट्स में। |
| height | Single | नए Summary Zoom फ्रेम की ऊँचाई, पॉइंट्स में। |

### Return Value

नया बनाया गया [`ISummaryZoomFrame`](../../isummaryzoomframe)।

### Exceptions

| exception | condition |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | यदि प्रस्तुति में कोई सेक्शन नहीं है, या लक्ष्य स्लाइड किसी सेक्शन से संबंधित नहीं है, तो थ्रो किया जाता है। |

### Remarks

यह मेथड नया Summary Zoom बनाता है और इस प्रस्तुति के सभी सेक्शनों के लिए उसमे ऑब्जेक्ट्स का कलेक्शन रखता है।

### Examples

यह उदाहरण एक Summary Zoom ऑब्जेक्ट को कलेक्शन के अंत में जोड़ने को दर्शाता है (मान लें कि "Presentation.pptx" प्रस्तुति में कम से कम दो सेक्शन हैं):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSummaryZoomFrame(150, 20, 500, 250);
}
```

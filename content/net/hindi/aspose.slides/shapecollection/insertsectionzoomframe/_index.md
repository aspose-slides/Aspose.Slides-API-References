## InsertSectionZoomFrame(int, float, float, float, float, ISection) {#insertsectionzoomframe}

एक नया Section Zoom फ्रेम बनाता है और इसे निर्दिष्ट अनुक्रमांक पर shape collection में डालता है।

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| index | Int32 | Section Zoom फ्रेम डालने के लिए शून्य-आधारित अनुक्रमांक। |
| x | Single | नए Section Zoom फ्रेम का x-निर्देशांक, पॉइंट्स में। |
| y | Single | नए Section Zoom फ्रेम का y-निर्देशांक, पॉइंट्स में। |
| width | Single | नए Section Zoom फ्रेम की चौड़ाई, पॉइंट्स में। |
| height | Single | नए Section Zoom फ्रेम की ऊँचाई, पॉइंट्स में। |
| section | ISection | Section Zoom फ्रेम द्वारा संदर्भित [`ISection`](../../isection); यह इस प्रस्तुति का हिस्सा होना चाहिए और कम से कम एक स्लाइड शामिल करनी चाहिए। |

### रिटर्न मान

नया बनाया गया [`ISectionZoomFrame`](../../isectionzoomframe)।

### अपवाद

| अपवाद | शर्त |
| --- | --- |
| ArgumentException | यदि संदर्भित सेक्शन वर्तमान प्रस्तुति का हिस्सा नहीं है या उसमें कोई स्लाइड नहीं है तो फेंका जाता है। |

### उदाहरण

यह उदाहरण एक Section Zoom ऑब्जेक्ट को संग्रह में निर्दिष्ट अनुक्रमांक पर बनाने और डालने को दर्शाता है (मान लें कि "Presentation.pptx" प्रस्तुति में कम से कम दो सेक्शन हैं):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSectionZoomFrame(2, 150, 20, 50, 50, pres.Sections[1]);
}
```

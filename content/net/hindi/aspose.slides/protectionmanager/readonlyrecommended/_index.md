## ProtectionManager.ReadOnlyRecommended प्रॉपर्टी

पढ़ने-केवल अनुशंसा प्राप्त या सेट करता है। Read/write Boolean.

```csharp
public bool ReadOnlyRecommended { get; set; }
```

### उदाहरण

निम्नलिखित नमूना कोड दिखाता है कि Aspose.Slides का उपयोग करके C# में PowerPoint प्रस्तुति को पढ़ने-केवल कैसे सेट किया जाए।

```csharp

{
	pres.ProtectionManager.ReadOnlyRecommended = true;
	pres.Save("ReadOnlyPresentation.pptx", SaveFormat.Pptx);
}
```

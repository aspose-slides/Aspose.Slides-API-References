## ProtectionManager.Encrypt विधि

निर्दिष्ट पासवर्ड के साथ प्रस्तुति को एन्क्रिप्ट करता है।

```csharp
public void Encrypt(string encryptionPassword)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| encryptionPassword | String | पासवर्ड। |

### उदाहरण

निम्नलिखित नमूना कोड दिखाता है कि आप PowerPoint प्रस्तुति को कैसे एन्क्रिप्ट कर सकते हैं।

```csharp
[C#]
using (Presentation presentation = new Presentation("pres.pptx"))
{
    presentation.ProtectionManager.Encrypt("123123");
    presentation.Save("encrypted-pres.pptx", SaveFormat.Pptx);
}
```

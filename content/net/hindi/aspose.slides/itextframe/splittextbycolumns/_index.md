## ITextFrame.SplitTextByColumns विधि

[`ITextFrame`](../../itextframe) की टेक्स्ट सामग्री को स्ट्रिंग्स की एक सरणी में विभाजित करता है, जहाँ प्रत्येक तत्व फ्रेम में एक अलग टेक्स्ट कॉलम के अनुरूप होता है।

```csharp
public string[] SplitTextByColumns()
```

### रिटर्न मान

एक स्ट्रिंग्स की सरणी, जहाँ प्रत्येक स्ट्रिंग [`ITextFrame`](../../itextframe) में एक विशिष्ट कॉलम की टेक्स्ट सामग्री का प्रतिनिधित्व करती है।

### टिप्पणियाँ

यदि टेक्स्ट फ्रेम में कई कॉलम नहीं हैं, तो लौटाई गई सरणी में पूर्ण टेक्स्ट वाला एकल तत्व होगा। खाली कॉलम सरणी में खाली स्ट्रिंग्स के रूप में दर्शाए जाएंगे।

### उदाहरण

निम्नलिखित उदाहरण दर्शाता है कि `SplitTextByColumns` का उपयोग कैसे किया जाता है:

```csharp
using (Presentation pres = new Presentation("example.pptx"))
{
    // स्लाइड पर पहला शेप प्राप्त करें और इसे ITextFrame में कास्ट करें
    ITextFrame textFrame = pres.Slides[0].Shapes[0] as ITextFrame;
    // टेक्स्ट फ्रेम की सामग्री को कॉलम में विभाजित करें
    string[] columnsText = textFrame.SplitTextByColumns();
    // प्रत्येक कॉलम का टेक्स्ट कंसोल में प्रिंट करें
    foreach (string column in columnsText)
        Console.WriteLine(column);
}
```

## ITextFrame.ReplaceRegex विधि

नियमित अभिव्यक्ति के सभी मिलान को निर्दिष्ट स्ट्रिंग से बदलता है।

```csharp
public void ReplaceRegex(Regex regex, string newText, IFindResultCallback callback)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| regex | Regex | वह नियमित अभिव्यक्ति Regex जिससे बदलने के लिए स्ट्रिंग प्राप्त की जाती है। |
| newText | String | सभी बदली जाने वाली स्ट्रिंग्स की घटनाओं को प्रतिस्थापित करने वाली स्ट्रिंग। |
| callback | IFindResultCallback | खोज परिणाम प्राप्त करने के लिए कॉलबैक ऑब्जेक्ट [`IFindResultCallback`](../../ifindresultcallback)। |

### उदाहरण

निम्नलिखित कोड नमूना दिखाता है कि कैसे नियमित अभिव्यक्ति का उपयोग करके निर्दिष्ट स्ट्रिंग से टेक्स्ट को बदलें।

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// 10 या अधिक अक्षरों वाले सभी शब्दों को '***' से बदलें
	((AutoShape)presentation.Slides[0].Shapes[0]).TextFrame.ReplaceRegex(regex, "***", null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

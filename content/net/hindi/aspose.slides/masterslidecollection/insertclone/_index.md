## MasterSlideCollection.InsertClone मेथड

एक निर्दिष्ट मास्टर स्लाइड की प्रति को संग्रह के निर्दिष्ट स्थान पर सम्मिलित करता है। जुड़े लेआउट स्लाइड भी कॉपी की जाएँगी।

```csharp
public IMasterSlide InsertClone(int index, IMasterSlide sourceMaster)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| index | Int32 | नए स्लाइड का सूचकांक। |
| sourceMaster | IMasterSlide | क्लोन करने के लिए स्लाइड। |

### रिटर्न वैल्यू

डाली गई मास्टर स्लाइड।

### उदाहरण

निम्न उदाहरण दर्शाता है कि कैसे एक अन्य PowerPoint प्रस्तुति में मास्टर स्लाइड को क्लोन किया जाए।

```csharp
[C#]
// स्रोत प्रस्तुति फ़ाइल को लोड करने के लिये Presentation क्लास का उदाहरण बनायें
using (Presentation srcPres = new Presentation("CloneToAnotherPresentationWithMaster.pptx"))
{
    // गंतव्य प्रस्तुति (जहाँ स्लाइड को क्लोन किया जाएगा) के लिये Presentation क्लास का उदाहरण बनायें
    using (Presentation destPres = new Presentation())
    {
        // स्रोत प्रस्तुति में स्लाइड्स के संग्रह से ISlide का उदाहरण बनायें, साथ में
        // मास्टर स्लाइड
        ISlide SourceSlide = srcPres.Slides[0];
        IMasterSlide SourceMaster = SourceSlide.LayoutSlide.MasterSlide;
		// गंतव्य प्रस्तुति के मास्टर स्लाइड्स प्राप्त करें
        IMasterSlideCollection masters = destPres.Masters;
        // स्रोत प्रस्तुति से वांछित मास्टर स्लाइड को मास्टर संग्रह में क्लोन करें
        // गंतव्य प्रस्तुति
        IMasterSlide iSlide = masters.AddClone(SourceMaster);
        // गंतव्य प्रस्तुति में स्लाइड्स का संग्रह
        ISlideCollection slds = destPres.Slides;
		// स्रोत स्लाइड को गंतव्य स्लाइड संग्रह में क्लोन करें।
        slds.AddClone(SourceSlide, iSlide, true);
        // गंतव्य प्रस्तुति को डिस्क पर सहेजें
        destPres.Save("CloneToAnotherPresentationWithMaster_out.pptx", SaveFormat.Pptx);
    }
}
```

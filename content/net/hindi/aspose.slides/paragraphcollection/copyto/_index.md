## ParagraphCollection.CopyTo मेथड

ICollection के तत्वों को एक Array में कॉपी करता है, जो एक विशिष्ट Array index से शुरू होता है।

```csharp
public void CopyTo(IParagraph[] array, int arrayIndex)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| array | IParagraph[] | एक-आयामी Array जो ICollection से कॉपी किए गए तत्वों का गंतव्य है। Array में शून्य-आधारित अनुक्रमण होना चाहिए। |
| arrayIndex | Int32 | शून्य-आधारित अनुक्रमण *array* में जहाँ कॉपी करना शुरू होता है। |

### अपवाद

| अपवाद | शर्त |
| --- | --- |
| ArgumentNullException | *array* शून्य है। |
| ArgumentOutOfRangeException | *arrayIndex* 0 से कम है। |
| ArgumentException | स्रोत ICollection में तत्वों की संख्या उपलब्ध स्थान से अधिक है, जो *arrayIndex* से गंतव्य *array* के अंत तक है। |

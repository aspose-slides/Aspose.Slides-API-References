## PortionCollection.CopyTo विधि

ICollection के तत्वों को एक Array में कॉपी करता है, जो एक विशेष Array इंडेक्स से शुरू होता है।

```csharp
public void CopyTo(IPortion[] array, int arrayIndex)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| array | IPortion[] | ICollection से कॉपी किए गए तत्वों का गंतव्य होने वाला एक-आयामी Array। Array में शून्य-आधारित अनुक्रमणिका होनी चाहिए। |
| arrayIndex | Int32 | *array* में वह शून्य-आधारित अनुक्रमणिका जहाँ से कॉपी शुरू होती है। |

### अपवाद

| अपवाद | स्थिति |
| --- | --- |
| ArgumentNullException | *array* शून्य है। |
| ArgumentOutOfRangeException | *arrayIndex* 0 से कम है। |
| ArgumentException | स्रोत ICollection में तत्वों की संख्या उपलब्ध स्थान से अधिक है जो *arrayIndex* से लेकर गंतव्य *array* के अंत तक है। |

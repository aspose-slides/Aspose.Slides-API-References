## ILayoutPlaceholderManager.AddTextPlaceholder मेथड

एक नया प्लेसहोल्डर आकार लेआउट स्लाइड में जोड़ता है ताकि टेक्स्ट सामग्री रखी जा सके।

```csharp
public IAutoShape AddTextPlaceholder(float x, float y, float width, float height)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| x | Single | नए प्लेसहोल्डर आकार का X निर्देशांक। |
| y | Single | नए प्लेसहोल्डर आकार का Y निर्देशांक। |
| width | Single | नए प्लेसहोल्डर आकार की चौड़ाई। |
| height | Single | नए प्लेसहोल्डर आकार की ऊँचाई। |

### रिटर्न मान

[`IAutoShape`](../../iautoshape) के साथ एक टेक्स्ट प्लेसहोल्डर बनाया गया।

### उदाहरण

निम्न उदाहरण दिखाता है कि लेआउट स्लाइड में टेक्स्ट प्लेसहोल्डर आकार कैसे जोड़ें।

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTextPlaceholder(20, 20, 500, 300);
}
```

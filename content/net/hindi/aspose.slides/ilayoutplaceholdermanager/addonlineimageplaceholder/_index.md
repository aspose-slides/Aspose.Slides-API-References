## ILayoutPlaceholderManager.AddOnlineImagePlaceholder मेथड

लेआउट स्लाइड में एक नया प्लेसहोल्डर आकार जोड़ता है जो ऑनलाइन छवि को धारण करता है।

### वापसी मान

एक ऑनलाइन इमेज प्लेसहोल्डर के साथ [`IAutoShape`](../../iautoshape) बनाया गया।

### उदाहरण

निम्नलिखित उदाहरण दिखाता है कि लेआउट स्लाइड में ऑनलाइन इमेज प्लेसहोल्डर आकार कैसे जोड़ें।

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddOnlineImagePlaceholder(20, 20, 200, 200);
}
```

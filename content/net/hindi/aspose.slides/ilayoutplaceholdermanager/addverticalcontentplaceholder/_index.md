## ILayoutPlaceholderManager.AddVerticalContentPlaceholder मेथड

नए प्लेसहोल्डर आकार को लेआउट स्लाइड में जोड़ता है ताकि सामग्री, जैसे चित्र, तालिका, मीडिया या टेक्स्ट को ऊर्ध्वाधर दिशा में रखा जा सके।

```csharp
public IAutoShape AddVerticalContentPlaceholder(float x, float y, float width, float height)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| x | Single | नए प्लेसहोल्डर आकार का X निर्देशांक। |
| y | Single | नए प्लेसहोल्डर आकार का Y निर्देशांक। |
| width | Single | नए प्लेसहोल्डर आकार की चौड़ाई। |
| height | Single | नए प्लेसहोल्डर आकार की ऊँचाई। |

### रिटर्न वैल्यू

[`IAutoShape`](../../iautoshape) को एक सामग्री (ऊर्ध्वाधर) प्लेसहोल्डर के साथ बनाया गया।

### उदाहरण

निम्न उदाहरण दर्शाता है कि लेआउट स्लाइड में सामग्री (ऊर्ध्वाधर) प्लेसहोल्डर आकार कैसे जोड़ें।

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalContentPlaceholder(20, 20, 300, 500);
}
```

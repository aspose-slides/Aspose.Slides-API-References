## ILayoutPlaceholderManager.AddSmartArtPlaceholder मेथड

लेआउट स्लाइड में एक नया प्लेसहोल्डर शैप जोड़ता है जो एक SmartArt डायग्राम को रखता है।

```csharp
public IAutoShape AddSmartArtPlaceholder(float x, float y, float width, float height)
```

| परमीटर | प्रकार | विवरण |
| --- | --- | --- |
| x | Single | नए प्लेसहोल्डर शैप का X निर्देशांक। |
| y | Single | नए प्लेसहोल्डर शैप का Y निर्देशांक। |
| width | Single | नए प्लेसहोल्डर शैप की चौड़ाई। |
| height | Single | नए प्लेसहोल्डर शैप की ऊँचाई। |

### Return Value

[`IAutoShape`](../../iautoshape) एक SmartArt प्लेसहोल्डर के साथ बनाया गया।

### Examples

निम्न उदाहरण दिखाता है कि लेआउट स्लाइड में SmartArt प्लेसहोल्डर शैप कैसे जोड़ा जाए।

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddSmartArtPlaceholder(20, 20, 200, 200);
}
```

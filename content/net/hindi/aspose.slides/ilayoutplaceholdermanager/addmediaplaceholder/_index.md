## ILayoutPlaceholderManager.AddMediaPlaceholder मेथड

लेआउट स्लाइड में एक मीडिया ऑब्जेक्ट रखने के लिए नया प्लेसहोल्डर आकार जोड़ता है।

```csharp
public IAutoShape AddMediaPlaceholder(float x, float y, float width, float height)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| x | Single | नए प्लेसहोल्डर आकार का X निर्देशांक। |
| y | Single | नए प्लेसहोल्डर आकार का Y निर्देशांक। |
| width | Single | नए प्लेसहोल्डर आकार की चौड़ाई। |
| height | Single | नए प्लेसहोल्डर आकार की ऊँचाई। |

### रिटर्न वैल्यू

एक Media प्लेसहोल्डर के साथ [`IAutoShape`](../../iautoshape) बनाया गया।

### उदाहरण

निम्नलिखित उदाहरण दर्शाता है कि लेआउट स्लाइड में Media प्लेसहोल्डर आकार कैसे जोड़ें।

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddMediaPlaceholder(20, 20, 200, 200);
}
```

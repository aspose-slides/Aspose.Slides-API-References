## ILayoutPlaceholderManager.AddContentPlaceholder विधि

नए placeholder shape को layout slide पर जोड़ता है ताकि सामग्री जैसे चित्र, तालिका, मीडिया या पाठ रख सके।

```csharp
public IAutoShape AddContentPlaceholder(float x, float y, float width, float height)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| x | Single | नए placeholder shape का X निर्देशांक। |
| y | Single | नए placeholder shape का Y निर्देशांक। |
| width | Single | नए placeholder shape की चौड़ाई। |
| height | Single | नए placeholder shape की ऊँचाई। |

### रिटर्न वैल्यू

[`IAutoShape`](../../iautoshape) को एक Content placeholder के साथ बनाया गया।

### उदाहरण

निचे दिया गया उदाहरण दिखाता है कि कैसे Content placeholder shape को layout slide में जोड़ा जा सकता है।

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddContentPlaceholder(20, 20, 500, 300);
}
```

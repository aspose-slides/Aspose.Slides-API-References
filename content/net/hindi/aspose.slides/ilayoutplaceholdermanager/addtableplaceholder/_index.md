## ILayoutPlaceholderManager.AddTablePlaceholder मेथड

एक नया प्लेसहोल्डर आकार लेआउट स्लाइड में जोड़ता है ताकि एक Table रखी जा सके।

```csharp
public IAutoShape AddTablePlaceholder(float x, float y, float width, float height)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| x | Single | नए प्लेसहोल्डर आकार का X निर्देशांक। |
| y | Single | नए प्लेसहोल्डर आकार का Y निर्देशांक। |
| width | Single | नए प्लेसहोल्डर आकार की चौड़ाई। |
| height | Single | नए प्लेसहोल्डर आकार की ऊंचाई। |

### वापसी मान

एक Table प्लेसहोल्डर के साथ [`IAutoShape`](../../iautoshape) बनाया गया।

### उदाहरण

निम्न उदाहरण दिखाता है कि लेआउट स्लाइड में Table प्लेसहोल्डर आकार कैसे जोड़ें।

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTablePlaceholder(20, 20, 500, 200);
}
```

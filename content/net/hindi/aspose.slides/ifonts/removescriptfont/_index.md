## IFonts.RemoveScriptFont विधि

थीम के फ़ॉन्ट संग्रह से किसी विशिष्ट स्क्रिप्ट टैग से जुड़ी फ़ॉन्ट सेटिंग को हटाता है।

```csharp
public void RemoveScriptFont(string script)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| script | String | उस BCP-47 स्क्रिप्ट कोड को जिसकी फ़ॉन्ट सेटिंग को हटाया जाना चाहिए। |

### उदाहरण

यह उदाहरण यह दर्शाता है कि हिब्रू स्क्रिप्ट के लिए फ़ॉन्ट मैपिंग को कैसे हटाया जाए:

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.RemoveScriptFont("Hebr");
```

## IFonts.GetScriptFont मेथड

प्रस्तुति थीम से एक विशिष्ट स्क्रिप्ट टैग से संबंधित फ़ॉन्ट नाम प्राप्त करता है।

```csharp
public string GetScriptFont(string script)
```

| परामीटर | प्रकार | विवरण |
| --- | --- | --- |
| script | String | BCP-47 स्क्रिप्ट कोड (उदाहरण के लिए, "Latn", "Cyrl", "Jpan") जो लेखन प्रणाली की पहचान के लिए उपयोग किया जाता है। |

### Return Value

निर्दिष्ट स्क्रिप्ट के लिए प्रयुक्त फ़ॉन्ट का नाम, या यदि स्क्रिप्ट परिभाषित नहीं है तो `null`।

### Examples

यह उदाहरण दर्शाता है कि प्रस्तुति थीम में सिरिलिक स्क्रिप्ट के लिए निर्धारित फ़ॉन्ट को कैसे प्राप्त किया जाए।

```csharp
[C#]
string font = presentation.MasterTheme.FontScheme.Major.GetScriptFont("Cyrl");
Console.WriteLine("Font for Cyrillic script: " + font);
```

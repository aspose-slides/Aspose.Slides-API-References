## IFonts.SetScriptFont मेथड

विशिष्ट स्क्रिप्ट टैग को एक फ़ॉन्ट नाम असाइन करता है, जो निर्धारित करता है कि उस स्क्रिप्ट का पाठ प्रस्तुति में कैसे प्रदर्शित होगा।

```csharp
public void SetScriptFont(string script, string fontName)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| script | String | BCP-47 स्क्रिप्ट कोड (जैसे, "Arab", "Hebr", "Hans") जो लेखन प्रणाली की पहचान करता है। |
| fontName | String | निर्दिष्ट स्क्रिप्ट के लिए असाइन किए जाने वाले फ़ॉन्ट का नाम। |

### उदाहरण

यह उदाहरण दिखाता है कि अरबी स्क्रिप्ट के लिए फ़ॉन्ट को "Segoe UI" पर कैसे सेट किया जाए:

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segoe UI");
```

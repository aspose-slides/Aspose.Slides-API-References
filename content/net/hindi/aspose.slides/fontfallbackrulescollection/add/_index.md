## FontFallBackRulesCollection.Add मेथड

संकलन के अंत में एक निर्दिष्ट FallBack नियम जोड़ता है।

```csharp
public void Add(IFontFallBackRule sourceRule)
```

| पैरामीटर | टाइप | विवरण |
| --- | --- | --- |
| sourceRule | IFontFallBackRule | जोड़ने के लिए निर्दिष्ट नियम |

### उदाहरण

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    //FontsManager से खाली या पूर्व-प्रारम्भित नियम संग्रह प्राप्त कर रहा है
    IFontFallBackRulesCollection rulesList = pres.FontsManager.FontFallBackRulesCollection;

    //संग्रह में नया नियम जोड़ रहा है
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));
}
```

## SummaryZoomFrame.Layout गुण

फ़्रेम में Summary Zoom Sections का लेआउट प्राप्त करता है। डिफ़ॉल्ट मान GridLayout है।

```csharp
public ZoomLayout Layout { get; }
```

### उदाहरण

यह उदाहरण इंडेक्स द्वारा Summary Zoom Section तत्व प्राप्त करने को दर्शाता है:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[1].Shapes[0] as ISummaryZoomFrame;
    ZoomLayout layout = zoomFrame.Layout;
}
```

## InsertClone(int, ISlide) {#insertclone}

एक निर्दिष्ट स्लाइड की प्रति को संग्रह के निर्दिष्ट स्थिति में सम्मिलित करता है।

```csharp
public ISlide InsertClone(int index, ISlide sourceSlide)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | Int32 | नए स्लाइड का सूचकांक। |
| sourceSlide | ISlide | क्लोन करने के लिए स्लाइड। |

### Return Value

डाली गई स्लाइड।

### Remarks

जब विभिन्न प्रस्तुतियों के बीच स्लाइड को क्लोन किया जाता है तो स्लाइड का मास्टर भी क्लोन हो सकता है। आंतरिक रजिस्ट्री का उपयोग स्वतः क्लोन किए गए मास्टर को ट्रैक करने के लिए किया जाता है ताकि एक ही मास्टर स्लाइड के कई क्लोन बनाने से बचा जा सके। मास्टर स्लाइड्स के मैन्युअल क्लोनिंग को न तो रोका जाएगा और न ही रजिस्टर्ड किया जाएगा। यदि आपको क्लोनिंग प्रक्रिया पर अधिक नियंत्रण चाहिए तो क्लोनिंग स्लाइड्स के लिए [`InsertClone`](../insertclone) या [`InsertClone`](../insertclone) और मास्टर क्लोनिंग के लिए [`AddClone`](../../imasterslidecollection/addclone) का उपयोग करें।

## InsertClone(int, ISlide, IMasterSlide, bool) {#insertclone_2}

एक निर्दिष्ट स्रोत स्लाइड की प्रति को संग्रह के निर्दिष्ट स्थिति में सम्मिलित करता है। निर्दिष्ट मास्टर से उपयुक्त लेआउट स्वचालित रूप से चयनित किया जाएगा (उपयुक्त लेआउट वह लेआउट है जिसका Type या Name स्रोत स्लाइड के लेआउट के समान हो)। यदि उपयुक्त लेआउट नहीं है तो स्रोत स्लाइड का लेआउट क्लोन किया जाएगा (यदि allowCloneMissingLayout true है) या PptxEditException फेंका जाएगा (यदि allowCloneMissingLayout false है)।

```csharp
public ISlide InsertClone(int index, ISlide sourceSlide, IMasterSlide destMaster, 
    bool allowCloneMissingLayout)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | Int32 | नए स्लाइड का सूचकांक। |
| sourceSlide | ISlide | क्लोन करने के लिए स्लाइड। |
| destMaster | IMasterSlide | नए स्लाइड के लिए मास्टर स्लाइड। |
| allowCloneMissingLayout | Boolean | यदि निर्दिष्ट मास्टर में उपयुक्त लेआउट नहीं है तो स्रोत स्लाइड का लेआउट क्लोन किया जाएगा (यदि allowCloneMissingLayout true है) या PptxEditException फेंका जाएगा (यदि allowCloneMissingLayout false है)। |

### Return Value

डाली गई स्लाइड।

### Exceptions

| अपवाद | शर्त |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | यदि निर्दिष्ट मास्टर में उपयुक्त लेआउट नहीं है और allowCloneMissingLayout false है तो यह फेंका जाता है। |

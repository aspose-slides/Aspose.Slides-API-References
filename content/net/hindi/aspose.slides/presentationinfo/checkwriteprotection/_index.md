## PresentationInfo.CheckWriteProtection मेथड

एक लिखित संरक्षित प्रस्तुति के लिए संशोधन पासवर्ड सही है या नहीं यह जाँचता है।

```csharp
public bool CheckWriteProtection(string password)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| password | String | जाँचने के लिए पासवर्ड। |

### वापसी मान

यदि प्रस्तुति लिखने के लिए संरक्षित है और पासवर्ड सही है तो True, अन्यथा False।

### अपवाद

| अपवाद | शर्त |
| --- | --- |
| InvalidOperationException |  |

### टिप्पणियाँ

1. इस मेथड को कॉल करने से पहले आपको [`IsWriteProtected`](../iswriteprotected) प्रॉपर्टी को जाँचना चाहिए। 2. जब पासवर्ड null या empty हो, तो यह मेथड false लौटाता है।

### उदाहरण

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
if (info.IsWriteProtected == NullableBool.True)
{
    bool isWriteProtectedByPassword = info.CheckWriteProtection("my_password");
}
```

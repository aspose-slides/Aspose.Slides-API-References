## IDocumentProperties.GetSensitivityLabels मेथड

कस्टम दस्तावेज़ गुणों से संवेदनशीलता लेबल्स की एक एरे प्राप्त करता है (Microsoft Information Protection SDK Metadata).

```csharp
public ISensitivityLabel[] GetSensitivityLabels()
```

### उदाहरण

निम्नलिखित कोड दिखाता है कि कैसे कस्टम दस्तावेज़ गुणों से संवेदनशीलता लेबल जानकारी को आधुनिक SensitivityLabels संग्रह में ले जाया जाए:

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    // कस्टम दस्तावेज़ गुणों से संवेदनशीलता लेबल प्राप्त करें
    ISensitivityLabel[] mipSensitivityLabels = pres.DocumentProperties.GetSensitivityLabels();

    ISensitivityLabelCollection sensitivityLabels = pres.SensitivityLabels;
    foreach (var sensitivityLabel in mipSensitivityLabels)
    {
        // लेबल को संग्रह में जोड़ें
        // यहाँ आप लेबल जानकारी की वैधता (लेबल उपलब्ध है आदि) की जाँच जोड़ सकते हैं
        sensitivityLabels.Add(sensitivityLabel);
    }

    pres.Save("SensitivityLabel.pptx", SaveFormat.Pptx);
}
```

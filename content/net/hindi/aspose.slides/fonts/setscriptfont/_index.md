## Fonts.SetScriptFont विधि

फ़ॉन्ट नाम को एक विशिष्ट स्क्रिप्ट टैग को आवंटित करता है, जो निर्धारित करता है कि उस स्क्रिप्ट के पाठ को प्रस्तुति में कैसे प्रदर्शित किया जाएगा।

```csharp
public void SetScriptFont(string script, string fontName)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| script | String | BCP-47 स्क्रिप्ट कोड (उदा., "Arab", "Hebr", "Hans") जो लेखन प्रणाली की पहचान करता है। |
| fontName | String | निर्दिष्ट स्क्रिप्ट के लिए निर्धारित करने हेतु फ़ॉन्ट का नाम। |

### उदाहरण

यह उदाहरण दर्शाता है कि अरबी स्क्रिप्ट के फ़ॉन्ट को "Segoe UI" में कैसे सेट किया जाए:

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segoe UI");
```

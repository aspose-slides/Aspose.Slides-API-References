## Fonts.RemoveScriptFont मेथड

फ़ॉन्ट सेटिंग को हटाता है जो विशिष्ट स्क्रिप्ट टैग से जुड़ी होती है थीम के फ़ॉन्ट संग्रह से।

```csharp
public void RemoveScriptFont(string script)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| script | String | वह BCP-47 स्क्रिप्ट कोड जिसका फ़ॉन्ट सेटिंग हटाया जाना चाहिए। |

### उदाहरण

यह उदाहरण दिखाता है कि हिब्रू स्क्रिप्ट के लिए फ़ॉन्ट मैपिंग को कैसे हटाया जाए:

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.RemoveScriptFont("Hebr");
```

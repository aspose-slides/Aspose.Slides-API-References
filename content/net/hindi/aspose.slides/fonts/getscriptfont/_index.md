## Fonts.GetScriptFont method

प्रेज़ेंटेशन थीम से एक विशिष्ट स्क्रिप्ट टैग से संबंधित फ़ॉन्ट नाम प्राप्त करता है।

```csharp
public string GetScriptFont(string script)
```

| परिमाण | प्रकार | विवरण |
| --- | --- | --- |
| script | String | BCP-47 स्क्रिप्ट कोड (जैसे, "Latn", "Cyrl", "Jpan") जिसका उपयोग लेखन प्रणाली की पहचान के लिए किया जाता है। |

### वापसी मान

निर्दिष्ट स्क्रिप्ट के लिए उपयोग किए गए फ़ॉन्ट का नाम, या यदि स्क्रिप्ट परिभाषित नहीं है तो `null`।

### उदाहरण

यह उदाहरण प्रदर्शित करता है कि प्रेज़ेंटेशन थीम में सायरिलिक स्क्रिप्‍ट को सौंपा गया फ़ॉन्ट कैसे प्राप्त किया जाए।

```csharp
[C#]
string font = presentation.MasterTheme.FontScheme.Major.GetScriptFont("Cyrl");
Console.WriteLine("Font for Cyrillic script: " + font);
```

## Metered.SetMeteredKey मेथड

मीटर आधारित सार्वजनिक और निजी कुंजी सेट करता है। यदि आप मीटर आधारित लाइसेंस खरीदते हैं, तो अनुप्रयोग प्रारंभ करने पर इस API को कॉल किया जाना चाहिए, सामान्यतः यह पर्याप्त है। हालांकि, यदि हमेशा खपत डेटा अपलोड करने में विफल रहता है और 24 घंटे से अधिक हो जाता है, तो लाइसेंस को मूल्यांकन स्थिति में सेट किया जाएगा। ऐसे मामले से बचने के लिए, आपको नियमित रूप से लाइसेंस स्थिति जाँचनी चाहिए, यदि यह मूल्यांकन स्थिति है, तो इस API को फिर से कॉल करें।

```csharp
public void SetMeteredKey(string publicKey, string privateKey)
```

| privateKey | String | निजी कुंजी |

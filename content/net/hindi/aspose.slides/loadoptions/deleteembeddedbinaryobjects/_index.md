## LoadOptions.DeleteEmbeddedBinaryObjects प्रॉपर्टी

निर्धारित करता है कि Aspose.Slides प्रेज़ेंटेशन लोड करते समय सभी एम्बेडेड बाइनरी ऑब्जेक्ट्स को हटाएगा या नहीं।

एम्बेडेड बाइनरी ऑब्जेक्ट्स के प्रकार:

* VBA प्रोजेक्ट [`VbaProject`](../../ipresentation/vbaproject)
* OLE ऑब्जेक्ट एम्बेडेड डेटा [`EmbeddedFileData`](../../ioleembeddeddatainfo/embeddedfiledata)
* ActiveX कंट्रोल बाइनरी डेटा [`ActiveXControlBinary`](../../icontrol/activexcontrolbinary)

पढ़ें/लिखें Boolean.

```csharp
public bool DeleteEmbeddedBinaryObjects { get; set; }
```

### टिप्पणियाँ

डिफ़ॉल्ट **false** है।

### उदाहरण

निम्नलिखित उदाहरण दर्शाता है कि सभी एम्बेडेड बाइनरी ऑब्जेक्ट्स के बिना प्रेज़ेंटेशन को कैसे लोड किया जाए।

```csharp
[C#]
LoadOptions loadOptions = new LoadOptions();
loadOptions.DeleteEmbeddedBinaryObjects = true;
using (Presentation pres = new Presentation("pres.ppt", loadOptions))
{
    pres.Save("output_WithoutBinaryObjects.ppt", SaveFormat.Ppt);
}
```

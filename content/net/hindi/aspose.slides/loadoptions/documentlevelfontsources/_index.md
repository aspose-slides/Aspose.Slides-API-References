## LoadOptions.DocumentLevelFontSources प्रॉपर्टी

प्रस्तुति द्वारा उपयोग किए जाने वाले बाहरी फ़ॉन्ट्स के स्रोत निर्दिष्ट करता है। ये फ़ॉन्ट्स पूरी प्रस्तुति अवधि के दौरान उपलब्ध होते हैं और अन्य प्रस्तुतियों के साथ साझा नहीं किए जाते।

```csharp
public IFontSources DocumentLevelFontSources { get; set; }
```

### उदाहरण

निम्नलिखित उदाहरण दिखाता है कि PowerPoint प्रस्तुति के साथ उपयोग किए जाने वाले कस्टम फ़ॉन्ट्स को कैसे निर्धारित किया जाए।

```csharp
[C#]
byte[] memoryFont1 = File.ReadAllBytes("customfonts\\CustomFont1.ttf");
byte[] memoryFont2 = File.ReadAllBytes("customfonts\\CustomFont2.ttf");
LoadOptions loadOptions = new LoadOptions();
loadOptions.DocumentLevelFontSources.FontFolders = new string[] { "assets\\fonts", "global\\fonts" };
loadOptions.DocumentLevelFontSources.MemoryFonts = new byte[][] { memoryFont1, memoryFont2 };
using (IPresentation presentation = new Presentation("MyPresentation.pptx", loadOptions))
{
    //प्रस्तुति के साथ काम करें
    //CustomFont1, CustomFont2 के साथ-साथ assets\fonts और global\fonts फोल्डर तथा उनके सबफ़ोल्डर में मौजूद फ़ॉन्ट्स प्रस्तुति के लिए उपलब्ध हैं
}
```

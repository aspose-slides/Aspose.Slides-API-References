## ZoomObject.ShowBackground गुण

जाँचता है या सेट करता है वह मान जो यह निर्धारित करता है कि Zoom गंतव्य स्लाइड की पृष्ठभूमि का उपयोग करेगा या नहीं। Read/write Boolean. डिफ़ॉल्ट मान: true

```csharp
public bool ShowBackground { get; set; }
```

### उदाहरण

उदाहरण दिखाता है कि एक Zoom ऑब्जेक्ट की छवि की पृष्ठभूमि को कैसे हटाया जाए:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
    zoomFrame.ShowBackground = false;
}
```

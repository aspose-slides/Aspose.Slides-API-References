## FontsLoader.LoadExternalFonts मेथड

फ़ॉन्ट्स खोजने के लिए अतिरिक्त फ़ोल्डर्स जोड़ता है।

```csharp
public static void LoadExternalFonts(string[] directories)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| directories | String[] | अतिरिक्त फ़ॉन्ट्स पढ़ने के लिए डायरेक्टरीज़। |

### उदाहरण

नीचे दिया गया उदाहरण दिखाता है कि .TTF से कस्टम फ़ॉन्ट्स कैसे लोड करें।

```csharp
[C#]
// दस्तावेज़ निर्देशिका का पथ।
string dataDir = "C:\\";
// फ़ॉन्ट खोजने के लिए फ़ोल्डर
String[] folders = new String[] { dataDir };
// कस्टम फ़ॉन्ट निर्देशिका फ़ॉन्ट लोड करें
FontsLoader.LoadExternalFonts(folders);
// कुछ कार्य करें और प्रस्तुति/स्लाइड रेंडरिंग निष्पादित करें
using (Presentation presentation = new Presentation(dataDir + "DefaultFonts.pptx"))
{
    presentation.Save(dataDir + "NewFonts_out.pptx", SaveFormat.Pptx);
	// फ़ॉन्ट कैश साफ़ करें
	FontsLoader.ClearCache();
}
```

## SummaryZoomSectionCollection.AddSummaryZoomSection मेथड

नया Summary Zoom Section ऑब्जेक्ट बनाता है और इसे संग्रह में जोड़ता है

```csharp
public ISummaryZoomSection AddSummaryZoomSection(ISection section)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| section | ISection | नए Summary Zoom Section तत्व के लिए अनुभाग [`ISection`](../../isection) |

### वापसी मान

जोड़ा गया [`ISummaryZoomFrame`](../../isummaryzoomframe) तत्व

### अपवाद

| अपवाद | शर्त |
| --- | --- |
| ArgumentException | संदर्भित अनुभाग वर्तमान प्रस्तुति से संबंधित नहीं है या इसमें कोई स्लाइड नहीं है। |

### टिप्पणी

यदि इस अनुभाग के लिए कोई तत्व पहले से संग्रह में मौजूद है, तो मौजूदा तत्व वापस किया जाता है।

### उदाहरण

उदाहरण इंडेक्स द्वारा Summary Zoom Section तत्व प्राप्त करने को दर्शाता है:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    ISummaryZoomFrame zoomFrame = (ISummaryZoomFrame) pres.Slides[1].Shapes[0];
    ISummaryZoomSectionCollection collection = zoomFrame.SummaryZoomCollection;
    ISummaryZoomSection newZoomSection = collection.AddSummaryZoomSection(pres.Sections[3]);
}
```

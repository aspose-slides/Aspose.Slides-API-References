## IViewProperties.GridSpacing प्रॉपर्टी

प्रस्तुति दस्तावेज़ के आधारभूत ग्रिड के लिए उपयोग की जाने वाली ग्रिड स्पेसिंग को पॉइंट्स में लौटाता या सेट करता है। पढ़ें/लिखें Single.

```csharp
public float GridSpacing { get; set; }
```

### टिप्पणियाँ

ग्रिड स्पेसिंग मान एक सकारात्मक संख्या होना चाहिए। सामान्य मान सीमा 1 मिमी (2.8349607 पॉइंट्स) से 2 इंच (144 पॉइंट्स) तक है।

### उदाहरण

निम्नलिखित नमूना कोड दिखाता है कि PowerPoint प्रस्तुति में ग्रिड स्पेसिंग को कैसे बदलें।

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

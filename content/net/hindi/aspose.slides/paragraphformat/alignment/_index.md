## ParagraphFormat.Alignment गुण

किसी पैराग्राफ में बिना विरासत के पाठ संरेखण को प्राप्त करता है या सेट करता है। पढ़ें/लिखें [`TextAlignment`](../../textalignment).

```csharp
public TextAlignment Alignment { get; set; }
```

### उदाहरण

निम्नलिखित नमूना कोड दिखाता है कि PowerPoint प्रस्तुति में टेक्स्ट पैराग्राफ कैसे संरेखित करें।

```csharp
[C#]
// PPTX फ़ाइल को दर्शाने वाला Presentation ऑब्जेक्ट बनाता है
using (Presentation pres = new Presentation("ParagraphsAlignment.pptx"))
{
    // पहली स्लाइड तक पहुँच रहा है
    ISlide slide = pres.Slides[0];
    // स्लाइड में पहले और दूसरे प्लेसहोल्डर तक पहुँच रहा है और इसे AutoShape में टाइपकास्ट कर रहा है
    ITextFrame tf1 = ((IAutoShape)slide.Shapes[0]).TextFrame;
    ITextFrame tf2 = ((IAutoShape)slide.Shapes[1]).TextFrame;
    // दोनों प्लेसहोल्डरों में टेक्स्ट बदल रहा है
    tf1.Text = "Center Align by Aspose";
    tf2.Text = "Center Align by Aspose";
    // प्लेसहोल्डरों का पहला पैराग्राफ प्राप्त कर रहा है
    IParagraph para1 = tf1.Paragraphs[0];
    IParagraph para2 = tf2.Paragraphs[0];
    // टेक्स्ट पैराग्राफ को सेंटर में संरेखित कर रहा है
    para1.ParagraphFormat.Alignment = TextAlignment.Center;
    para2.ParagraphFormat.Alignment = TextAlignment.Center;
    // प्रेजेंटेशन को PPTX फ़ाइल के रूप में लिख रहा है
    pres.Save("Centeralign_out.pptx", SaveFormat.Pptx);
}
```

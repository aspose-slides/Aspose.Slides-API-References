## IPictureFillFormat.DeletePictureCroppedAreas method

फ़िल Picture के क्रॉप किए गए क्षेत्रों को हटाता है।

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### Return Value

क्रॉप किया गया इमेज या मूल इमेज यदि क्रॉपिंग आवश्यक नहीं है।

### Remarks

यह मेथड WMF/EMF मेटाफाइल को रास्टर PNG इमेज में बदलता है जबकि क्रॉप किया जाता है।

### Examples

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // PictureFrame प्राप्त करता है
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // PictureFrame छवि के क्रॉप किए गए क्षेत्रों को हटाता है
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

## IPictureFillFormat.TileOffsetY प्रॉपर्टी

टेक्सचर का ऊर्ध्वाधर ऑफसेट, आकार की उत्पत्ति से पॉइंट्स में, लौटाता है या सेट करता है। सकारात्मक मान टेक्सचर को नीचे ले जाता है, जबकि नकारात्मक मान इसे ऊपर ले जाता है। पढ़ने/लिखने Single.

```csharp
public float TileOffsetY { get; set; }
```

### उदाहरण

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // शेप का पिक्चर फ़िल फ़ॉर्मेट प्राप्त करता है
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // पिक्चर फ़िल मोड को टाइल पर सेट करता है
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // टेक्सचर का ऊर्ध्वाधर ऑफसेट -50 पॉइंट्स पर सेट करता है
    pictureFillFormat.TileOffsetY = -50;
}
```

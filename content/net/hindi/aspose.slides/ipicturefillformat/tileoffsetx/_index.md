## IPictureFillFormat.TileOffsetX प्रॉपर्टी

टेक्सचर का क्षैतिज ऑफसेट, आकार की मूल बिंदु से, बिंदुओं (points) में, लौटाता है या सेट करता है। एक सकारात्मक मान टेक्सचर को दाईं ओर ले जाता है, जबकि नकारात्मक मान इसे बाईं ओर ले जाता है। पढ़ें/लिखें Single.

```csharp
public float TileOffsetX { get; set; }
```

### उदाहरण

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // आकार का पिक्चर फ़िल फ़ॉर्मेट प्राप्त करता है
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // पिक्चर फ़िल मोड को टाइल पर सेट करता है
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // टेक्सचर का क्षैतिज ऑफसेट 20 पॉइंट्स पर सेट करता है
    pictureFillFormat.TileOffsetX = 20f;
}
```

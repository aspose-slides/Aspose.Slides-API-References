## CompressImage(bool, PicturesCompression) {#compressimage}

छवि को आकार और निर्धारित रिज़ॉल्यूशन के आधार पर आकार घटाकर संकुचित करता है। वैकल्पिक रूप से, यह क्रॉप किए गए क्षेत्रों को भी हटा देता है।

```csharp
public bool CompressImage(bool deleteCroppedAreasOfImage, PicturesCompression resolution)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| deleteCroppedAreasOfImage | Boolean | यदि true हो, तो यह मेथड छवि के क्रॉप किए गए क्षेत्रों को हटा देगा, जिससे उसका आकार और अधिक घट सकता है। |
| resolution | PicturesCompression | संकुचन के लिए लक्ष्य रिज़ॉल्यूशन, जिसे [`PicturesCompression`](../../../aspose.slides.export/picturescompression) enum के मान के रूप में निर्दिष्ट किया गया है। |

### Return Value

एक Boolean जो दर्शाता है कि छवि सफलतापूर्वक संकुचित हुई या नहीं। यदि छवि का आकार बदल दिया गया या क्रॉप किया गया हो तो `true` लौटाता है, अन्यथा `false`।

### Exceptions

| अपवाद | स्थिति |
| --- | --- |
| ArgumentException | तब फेंका जाता है जब रिज़ॉल्यूशन मान्य मान नहीं है। |

### Remarks

यह मेथड PowerPoint की "Picture Format -> Compress Pictures" फीचर के समान छवि का आकार और रिज़ॉल्यूशन बदलता है।

### Examples

निम्नलिखित उदाहरण दर्शाता है कि `CompressImage` मेथड का उपयोग करके प्रस्तुति में छवि का आकार लक्ष्य रिज़ॉल्यूशन सेट करके और क्रॉप किए गए क्षेत्रों को हटाकर कैसे कम किया जा सकता है:

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;
    // 150 DPI (वेब रिज़ॉल्यूशन) के लक्ष्य रिज़ॉल्यूशन के साथ छवि को संकुचित करें और क्रॉप किए गए क्षेत्रों को हटाएँ
    bool result = picFrame.PictureFormat.CompressImage(true, PicturesCompression.Dpi150);
}
```

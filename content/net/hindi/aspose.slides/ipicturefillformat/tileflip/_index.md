## IPictureFillFormat.TileFlip गुण

टेक्सचर टाइल को इसके क्षैतिज, लंबवत या दोनों अक्षों के चारों ओर उलटता है। पढ़ें/लिखें [`TileFlip`](../../tileflip).

```csharp
public TileFlip TileFlip { get; set; }
```

### टिप्पणी

डिफ़ॉल्ट NoFlip है।

### उदाहरण

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // शेप के पिक्चर फ़िल फ़ॉर्मेट को प्राप्त करता है
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // पिक्चर फ़िल मोड को टाइल पर सेट करता है
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // टेक्सचर टाइल को उसके लंबवत अक्ष के चारों ओर उलटता है।
    pictureFillFormat.TileFlip = TileFlip.FlipY;
}
```

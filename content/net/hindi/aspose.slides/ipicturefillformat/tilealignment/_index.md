## IPictureFillFormat.TileAlignment प्रॉपर्टी

टेक्सचर को आकार के भीतर कैसे संरेखित किया जाता है, इसे प्राप्त करता है या सेट करता है। यह सेटिंग टेक्सचर पैटर्न के प्रारंभिक बिंदु और आकार के भीतर इसका दोहराव कैसे होता है, इसे नियंत्रित करती है। पढ़ें/लिखें [`RectangleAlignment`](../../rectanglealignment).

```csharp
public RectangleAlignment TileAlignment { get; set; }
```

### टिप्पणियाँ

डिफ़ॉल्ट TopLeft है।

### उदाहरण

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // आकार का चित्र फ़िल फ़ॉर्मेट प्राप्त करता है
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // चित्र फ़िल मोड को टाइल पर सेट करता है
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // टाइलिंग के संरेखण को दाएँ नीचे पर सेट करता है
    pictureFillFormat.TileAlignment = RectangleAlignment.BottomRight;
}
```

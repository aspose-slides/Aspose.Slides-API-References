## IPresentationInfo.CheckPassword मेथड

जाँचता है कि खुला पासवर्ड से सुरक्षित प्रस्तुति के लिए पासवर्ड सही है या नहीं।

```csharp
public bool CheckPassword(string password)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| password | String | जाँचने के लिये पासवर्ड। |

### रिटर्न मान

यदि प्रस्तुति खुला पासवर्ड से सुरक्षित है और पासवर्ड सही है तो True और अन्यथा false।

### टिप्पणी

जब पासवर्ड null या खाली हो, तो यह मेथड false लौटाता है।

### उदाहरण

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo("pres.pptx");
bool isPasswordCorrect = info.CheckPassword("my_password");
```

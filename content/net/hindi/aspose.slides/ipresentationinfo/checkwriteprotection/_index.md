## IPresentationInfo.CheckWriteProtection विधि

जाँचता है कि क्या संशोधन के लिए पासवर्ड लिखित रूप से संरक्षित प्रस्तुति के लिए सही है।

```csharp
public bool CheckWriteProtection(string password)
```

| Parameter | Type | Description |
| --- | --- | --- |
| password | String | जाँचने के लिए पासवर्ड। |

### रिटर्न मान

यदि प्रस्तुति लिखित रूप से संरक्षित है और पासवर्ड सही है तो True। अन्यथा False।

### अपवाद

| exception | condition |
| --- | --- |
| InvalidOperationException |  |

### टिप्पणियाँ

1. आपको इस विधि को कॉल करने से पहले [`IsWriteProtected`](../iswriteprotected) प्रॉपर्टी की जाँच करनी चाहिए। 2. जब पासवर्ड null या खाली हो, तो यह विधि false लौटाती है।

### उदाहरण

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
if (info.IsWriteProtected == NullableBool.True)
{
    bool isWriteProtectedByPassword = info.CheckWriteProtection("my_password");
}
```

## IProtectionManager.CheckWriteProtection मेथड

निर्धारित करता है कि क्या प्रस्तुति संशोधित करने के लिए पासवर्ड से संरक्षित है।

```csharp
public bool CheckWriteProtection(string password)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| password | String | जाँच के लिए पासवर्ड। |

### वापसी मान

यदि पासवर्ड मान्य है तो true; अन्यथा false।

### टिप्पणियाँ

1. इस मेथड को कॉल करने से पहले आपको [`IsWriteProtected`](../iswriteprotected) प्रॉपर्टी को जांचना चाहिए।  
2. जब पासवर्ड null या खाली हो, तो यह मेथड false लौटाता है।

### उदाहरण

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

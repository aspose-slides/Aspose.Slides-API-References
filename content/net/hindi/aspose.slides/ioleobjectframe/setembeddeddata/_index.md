## IOleObjectFrame.SetEmbeddedData मेथड

OLE एंबेडेड डेटा के बारे में जानकारी सेट करता है।

```csharp
public void SetEmbeddedData(IOleEmbeddedDataInfo embeddedData)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| embeddedData | IOleEmbeddedDataInfo | एंबेडेड डेटा [`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo) |

### अपवाद

| अपवाद | स्थिति |
| --- | --- |
| ArgumentNullException | जब embeddedData पैरामीटर null हो। |

### टिप्पणियाँ

यह मेथड ऑब्जेक्ट की प्रॉपर्टीज़ को नए डेटा को दर्शाने के लिए बदलता है और IsObjectLink फ्लैग को false सेट करता है, जिससे संकेत मिलता है कि OLE ऑब्जेक्ट एंबेडेड है।

### उदाहरण

निम्न उदाहरण दिखाता है कि मौजूदा [`IOleObjectFrame`](../../ioleobjectframe) ऑब्जेक्ट के लिए OLE एंबेडेड डेटा और उसके प्रकार को कैसे बदलें।

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    OleObjectFrame oof = pres.Slides[0].Shapes[0] as OleObjectFrame;
    if (oof != null)
    {
        IOleEmbeddedDataInfo newData = new OleEmbeddedDataInfo(File.ReadAllBytes("Picture.png"), "png");
        oof.SetEmbeddedData(newData);
    }
}
```

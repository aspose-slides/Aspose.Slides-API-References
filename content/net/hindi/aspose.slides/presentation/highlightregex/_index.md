## Presentation.HighlightRegex मेथड

नियमित अभिव्यक्ति के सभी मिलानों को निर्दिष्ट रंग के साथ हाइलाइट करता है।

```csharp
public void HighlightRegex(Regex regex, Color highlightColor, IFindResultCallback callback)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| regex | Regex | हाइलाइट करने के लिए स्ट्रिंग्स प्राप्त करने हेतु नियमित अभिव्यक्ति Regex। |
| highlightColor | Color | पाठ को हाइलाइट करने का रंग। |
| callback | IFindResultCallback | सर्च परिणाम प्राप्त करने के लिए कॉलबैक ऑब्जेक्ट [`IFindResultCallback`](../../ifindresultcallback)। |

### उदाहरण

निम्नलिखित कोड उदाहरण दिखाता है कि कैसे नियमित अभिव्यक्ति का उपयोग करके PowerPoint Presentation में टेक्स्ट को हाइलाइट किया जाए।

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// 10 या अधिक अक्षरों वाले सभी शब्दों को हाइलाइट कर रहा है
	presentation.HighlightRegex(regex, Color.Blue, null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

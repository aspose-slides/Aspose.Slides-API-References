## Presentation.HeaderFooterManager प्रॉपर्टी

वास्तविक HeaderFooter मैनेजर लौटाता है। केवल पढ़ने योग्य [`IPresentationHeaderFooterManager`](../../ipresentationheaderfootermanager).

```csharp
public IPresentationHeaderFooterManager HeaderFooterManager { get; }
```

### उदाहरण

निम्न उदाहरण दिखाता है कि PowerPoint Presentation के Slide के भीतर फुटर दृश्यता कैसे सेट करें।

```csharp
[C#]
using (Presentation presentation = new Presentation("presentation.ppt"))
{
    IBaseSlideHeaderFooterManager headerFooterManager = presentation.Slides[0].HeaderFooterManager;
	// प्रॉपर्टी IsFooterVisible का उपयोग यह संकेत देने के लिए किया जाता है कि स्लाइड फुटर प्लेसहोल्डर मौजूद नहीं है।
    if (!headerFooterManager.IsFooterVisible)
    {
		// मेथड SetFooterVisibility का उपयोग स्लाइड फुटर प्लेसहोल्डर को दृश्यमान बनाने के लिए किया जाता है।
        headerFooterManager.SetFooterVisibility(true);
    }
	// प्रॉपर्टी IsSlideNumberVisible का उपयोग यह संकेत देने के लिए किया जाता है कि स्लाइड पेज नंबर प्लेसहोल्डर मौजूद नहीं है।
    if (!headerFooterManager.IsSlideNumberVisible)
    {
		// मेथड SetSlideNumberVisibility का उपयोग स्लाइड पेज नंबर प्लेसहोल्डर को दृश्यमान बनाने के लिए किया जाता है।
        headerFooterManager.SetSlideNumberVisibility(true);
    }
	// प्रॉपर्टी IsDateTimeVisible का उपयोग यह संकेत देने के लिए किया जाता है कि स्लाइड डेट-टाइम प्लेसहोल्डर मौजूद नहीं है।
    if (!headerFooterManager.IsDateTimeVisible)
    {
		// मेथड SetFooterVisibility का उपयोग स्लाइड डेट-टाइम प्लेसहोल्डर को दृश्यमान बनाने के लिए किया जाता है।
        headerFooterManager.SetDateTimeVisibility(true);
    }
	// मेथड SetFooterText का उपयोग स्लाइड फुटर प्लेसहोल्डर में टेक्स्ट सेट करने के लिए किया जाता है।
    headerFooterManager.SetFooterText("Footer text");
	// मेथड SetDateTimeText का उपयोग स्लाइड डेट-टाइम प्लेसहोल्डर में टेक्स्ट सेट करने के लिए किया जाता है।
    headerFooterManager.SetDateTimeText("Date and time text");
	presentation.Save("Presentation.ppt",SaveFormat.ppt);
}
```

निम्न उदाहरण दिखाता है कि Slide के भीतर चाइल्ड फुटर दृश्यता कैसे सेट करें।

```csharp
[C#]
using (Presentation presentation = new Presentation("presentation.ppt"))
{
    IMasterSlideHeaderFooterManager headerFooterManager = presentation.Masters[0].HeaderFooterManager;
	// मेथड SetFooterAndChildFootersVisibility का उपयोग मास्टर स्लाइड और सभी चाइल्ड फुटर प्लेसहोल्डर को दृश्यमान बनाने के लिए किया जाता है।
    headerFooterManager.SetFooterAndChildFootersVisibility(true);
	// मेथड SetSlideNumberAndChildSlideNumbersVisibility का उपयोग मास्टर स्लाइड और सभी चाइल्ड पेज नंबर प्लेसहोल्डर को दृश्यमान बनाने के लिए किया जाता है।
    headerFooterManager.SetSlideNumberAndChildSlideNumbersVisibility(true);
	// मेथड SetDateTimeAndChildDateTimesVisibility का उपयोग मास्टर स्लाइड और सभी चाइल्ड डेट-टाइम प्लेसहोल्डर को दृश्यमान बनाने के लिए किया जाता है।
    headerFooterManager.SetDateTimeAndChildDateTimesVisibility(true);
	// मेथड SetFooterAndChildFootersText का उपयोग मास्टर स्लाइड और सभी चाइल्ड फुटर प्लेसहोल्डर में टेक्स्ट सेट करने के लिए किया जाता है।
    headerFooterManager.SetFooterAndChildFootersText("Footer text");
	// मेथड SetDateTimeAndChildDateTimesText का उपयोग मास्टर स्लाइड और सभी चाइल्ड डेट-टाइम प्लेसहोल्डर में टेक्स्ट सेट करने के लिए किया जाता है।
    headerFooterManager.SetDateTimeAndChildDateTimesText("Date and time text");
}
```

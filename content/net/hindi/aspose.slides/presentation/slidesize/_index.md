## Presentation.SlideSize प्रॉपर्टी

स्लाइड आकार ऑब्जेक्ट लौटाता है। केवल-पढ़ने योग्य [`ISlideSize`](../../islidesize).

```csharp
public ISlideSize SlideSize { get; }
```

### उदाहरण

निम्न उदाहरण दिखाता है कि PowerPoint Presentation में स्लाइड आकार कैसे बदलें।

```csharp
[C#]
using (Presentation pres = new Presentation("pres-4x3-aspect-ratio.pptx"))
{
    pres.SlideSize.SetSize(SlideSizeType.OnScreen16x9, SlideSizeScaleType.DoNotScale);
    pres.Save("pres-4x3-aspect-ratio.pptx", SaveFormat.Pptx);
}
```

निम्न उदाहरण दिखाता है कि PowerPoint Presentation के लिए कंटेंट स्केलिंग के अनुसार स्लाइड आकार कैसे सेट करें।

```csharp
[C#]
// एक Presentation ऑब्जेक्ट को इंस्टैंटिएट करता है जो प्रस्तुति फ़ाइल का प्रतिनिधित्व करता है
using(Presentation presentation = new Presentation("AccessSlides.pptx")) {
  using(Presentation auxPresentation = new Presentation()) {
    ISlide slide = presentation.Slides[0];
    // उत्पन्न प्रस्तुतियों के स्लाइड आकार को स्रोत के समान सेट करता है
    presentation.SlideSize.SetSize(540, 720, SlideSizeScaleType.EnsureFit); // मेथड SetSize का उपयोग स्केल सामग्री के साथ स्लाइड आकार सेट करने के लिए किया जाता है ताकि फिट सुनिश्चित हो
    presentation.SlideSize.SetSize(SlideSizeType.A4Paper, SlideSizeScaleType.Maximize); // मेथड SetSize का उपयोग सामग्री के आकार को अधिकतम करने के साथ स्लाइड आकार सेट करने के लिए किया जाता है
    // प्रस्तुति को डिस्क पर सहेजें
    auxPresentation.Save("Set_Size&Type_out.pptx", SaveFormat.Pptx);
  }
}
```

निम्न उदाहरण दिखाता है कि PowerPoint Presentation में कस्टम स्लाइड आकार कैसे निर्दिष्ट करें।

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    pres.SlideSize.SetSize(780, 540, SlideSizeScaleType.DoNotScale); // A4 कागज़ आकार
    pres.Save("pres-a4-slide-size.pptx", SaveFormat.Pptx);
}
```

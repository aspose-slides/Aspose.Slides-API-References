## HighlightText(string, Color) {#highlighttext}

नमूना पाठ के सभी मिलानों को निर्दिष्ट रंग से हाइलाइट करता है।

```csharp
public void HighlightText(string text, Color highlightColor)
```

| परिचर | प्रकार | विवरण |
| --- | --- | --- |
| text | String | हाइलाइट करने के लिए पाठ। |
| highlightColor | Color | पाठ को हाइलाइट करने के लिए रंग। |

### उदाहरण

PowerPoint प्रस्तुति में पाठ को हाइलाइट करने के लिए निम्न कोड उदाहरण दिखाया गया है।

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	// सभी अलग 'the' उपस्थितियों को हाइलाइट करना
	presentation.HighlightText("the", Color.Violet);
	presentation.Save("SomePresentation-out2.pptx", SaveFormat.Pptx);
}
```

```csharp
public void HighlightText(string text, Color highlightColor, ITextSearchOptions options, 
    IFindResultCallback callback)
```

| परिचर | प्रकार | विवरण |
| --- | --- | --- |
| text | String | हाइलाइट करने के लिए पाठ। |
| highlightColor | Color | पाठ को हाइलाइट करने के लिए रंग। |
| options | ITextSearchOptions | टेक्स्ट खोज विकल्प [`ITextSearchOptions`](../../itextsearchoptions)। |
| callback | IFindResultCallback | खोज परिणाम प्राप्त करने के लिए कॉलबैक ऑब्जेक्ट [`IFindResultCallback`](../../ifindresultcallback)। |

### उदाहरण

PowerPoint प्रस्तुति में पाठ को हाइलाइट करने के लिए निम्न कोड उदाहरण दिखाया गया है।

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	// सभी अलग 'the' उपस्थितियों को हाइलाइट करना
	presentation.HighlightText("the", Color.Violet, new TextSearchOptions() {WholeWordsOnly = true}, null);
	presentation.Save("SomePresentation-out2.pptx", SaveFormat.Pptx);
}
```

## CellFormat.GetEffective मेथड

विरासत और टेबल शैलियों को लागू करके प्रभावी टेबल सेल फ़ॉर्मेटिंग गुण प्राप्त करता है।

```csharp
public ICellFormatEffectiveData GetEffective()
```

### रिटर्न मान

एक [`ICellFormatEffectiveData`](../../icellformateffectivedata)।

### उदाहरण

यह उदाहरण विभिन्न टेबल लॉजिक भागों के लिए प्रभावी फ़िल फ़ॉर्मेट प्राप्त करने को दर्शाता है। कृपया ध्यान दें कि सेल फ़ॉर्मेटिंग का हमेशा रो फ़ॉर्मेटिंग पर अधिक प्राथमिकता रहती है, रो का कॉलम पर अधिक, कॉलम का पूरी टेबल पर अधिक। इसलिए अंततः CellFormatEffectiveData गुण हमेशा टेबल को ड्रॉ करने के लिए उपयोग किए जाते हैं। निम्नलिखित कोड केवल API का एक उदाहरण है।

```csharp
[C#]
using (Presentation pres = new Presentation(@"MyPresentation.pptx"))
{
    ITable tbl = pres.Slides[0].Shapes[0] as ITable;
    IFillFormatEffectiveData tableFillFormatEffective = tbl.TableFormat.GetEffective().FillFormat;
    IFillFormatEffectiveData rowFillFormatEffective = tbl.Rows[0].RowFormat.GetEffective().FillFormat;
    IFillFormatEffectiveData columnFillFormatEffective = tbl.Columns[0].ColumnFormat.GetEffective().FillFormat;
    IFillFormatEffectiveData cellFillFormatEffective = tbl[0, 0].CellFormat.GetEffective().FillFormat;
    /* आउटपुट और तुलना */
}
```

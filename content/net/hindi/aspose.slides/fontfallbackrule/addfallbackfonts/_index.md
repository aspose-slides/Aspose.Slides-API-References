## AddFallBackFonts(string) {#addfallbackfonts}

FallBack फ़ॉन्ट की सूची में नया फ़ॉन्ट(स) जोड़ता है।

```csharp
public void AddFallBackFonts(string fontName)
```

| पैरामीटर | टाइप | विवरण |
| --- | --- | --- |
| fontName | String | FallBack के लिए Font का नाम या नाम (कॉमा द्वारा अलग किया गया) |

### उदाहरण

```csharp
[C#]
// FontFallBackRule की नई instance बनाएँ
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho");

//दूसरा फ़ॉन्ट नियम में जोड़ें 
newRule.AddFallBackFonts("MS Gothic");
//तीसरा और चौथा फ़ॉन्ट नियम में जोड़ें 
newRule.AddFallBackFonts("Tahoma, Times New Roman");
```

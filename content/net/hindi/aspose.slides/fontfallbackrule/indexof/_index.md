## FontFallBackRule.IndexOf विधि

निर्दिष्ट नियम का अनुक्रमणिका संग्रह में लौटाता है।

```csharp
public int IndexOf(string fontName)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| fontName | String | खोजने के लिए फ़ॉन्ट का नाम। |

### रिटर्न मान

फ़ॉन्ट का अनुक्रमणिका, या यदि सूची में फ़ॉन्ट नहीं मिला तो -1।

### उदाहरण

```csharp
[C#]
// फ़ॉन्ट्स की सूची वाली एक नियम बनाएं।
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// Tahoma का अनुक्रमणिका प्राप्त करें।
int tahomaIndex = newRule.IndexOf("Tahoma");
```

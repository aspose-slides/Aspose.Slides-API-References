```csharp
public string[] ToArray()
```

## ToArray(int, int) {#toarray_1}

सूची में निर्दिष्ट रेंज से सभी FallBack फ़ॉन्ट्स के साथ एक एरे बनाता है और लौटाता है।

```csharp
public string[] ToArray(int startIndex, int count)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| startIndex | Int32 | पहले फ़ॉन्ट को जोड़ने के लिए इंडेक्स। |
| count | Int32 | जोड़ने के लिए फ़ॉन्ट्स की संख्या। |

### वापसी मान

String की एरे

### उदाहरण

```csharp
[C#]
// एक नियम बनाता है जिसमें फ़ॉन्ट्स की सूची शामिल है।
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// आखिरी दो फ़ॉन्ट नामों को एरे के रूप में प्राप्त करता है।
```

## FontFallBackRule.Remove मेथड

सूची से एक विशिष्ट FallBack फ़ॉन्ट की पहली उपस्थिति को हटाता है।

```csharp
public void Remove(string fontName)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| fontName | String | सूची से हटाने हेतु फ़ॉन्ट का नाम। |

### उदाहरण

```csharp
[C#]
// नियम बनाता है जिसमें फोंट की सूची होती है।
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// सूची से Tahoma हटाएँ।
newRule.Remove ("Tahoma");
```

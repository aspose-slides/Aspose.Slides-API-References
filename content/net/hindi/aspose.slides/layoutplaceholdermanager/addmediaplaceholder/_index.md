## LayoutPlaceholderManager.AddMediaPlaceholder मेथड

लेआउट स्लाइड में एक नया प्लेसहोल्डर शेप जोड़ता है ताकि एक मीडिया ऑब्जेक्ट रख सके।

```csharp
public IAutoShape AddMediaPlaceholder(float x, float y, float width, float height)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| x | Single | नए प्लेसहोल्डर शेप का X निर्देशांक। |
| y | Single | नए प्लेसहोल्डर शेप का Y निर्देशांक। |
| width | Single | नए प्लेसहोल्डर शेप की चौड़ाई। |
| height | Single | नए प्लेसहोल्डर शेप की ऊँचाई। |

### रिटर्न मान

[`IAutoShape`](../../iautoshape) को एक Media प्लेसहोल्डर के साथ बनाया गया।

### उदाहरण

निम्न उदाहरण दर्शाता है कि लेआउट स्लाइड में Media प्लेसहोल्डर शेप कैसे जोड़ें।

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddMediaPlaceholder(20, 20, 200, 200);
}
```

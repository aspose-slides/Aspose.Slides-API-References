## LayoutPlaceholderManager.AddSmartArtPlaceholder मेथड

नए प्लेसहोल्डर आकार को लेआउट स्लाइड में जोड़ता है ताकि SmartArt आरेख को रखा जा सके।

```csharp
public IAutoShape AddSmartArtPlaceholder(float x, float y, float width, float height)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| x | Single | नए प्लेसहोल्डर आकार का X निर्देशांक। |
| y | Single | नए प्लेसहोल्डर आकार का Y निर्देशांक। |
| width | Single | नए प्लेसहोल्डर आकार की चौड़ाई। |
| height | Single | नए प्लेसहोल्डर आकार की ऊँचाई। |

### रिटर्न वैल्यू

[`IAutoShape`](../../iautoshape) को SmartArt प्लेसहोल्डर के साथ बनाया गया।

### उदाहरण

निम्न उदाहरण दिखाता है कि कैसे SmartArt प्लेसहोल्डर आकार को लेआउट स्लाइड में जोड़ा जाए।

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddSmartArtPlaceholder(20, 20, 200, 200);
}
```

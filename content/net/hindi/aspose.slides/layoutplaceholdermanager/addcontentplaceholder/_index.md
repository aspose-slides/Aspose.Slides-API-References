## LayoutPlaceholderManager.AddContentPlaceholder विधि

लेआउट स्लाइड में एक नया प्लेसहोल्डर आकृति जोड़ता है जो सामग्री, जैसे चित्र, तालिका, मीडिया या पाठ, को रखने के लिए उपयोग होता है।

```csharp
public IAutoShape AddContentPlaceholder(float x, float y, float width, float height)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| x | Single | नए प्लेसहोल्डर आकृति का X समन्वय। |
| y | Single | नए प्लेसहोल्डर आकृति का Y समन्वय। |
| width | Single | नए प्लेसहोल्डर आकृति की चौड़ाई। |
| height | Single | नए प्लेसहोल्डर आकृति की ऊँचाई। |

### रिटर्न मान

[`IAutoShape`](../../iautoshape) को एक सामग्री प्लेसहोल्डर के साथ बनाया गया।

### उदाहरण

निम्न उदाहरण दर्शाता है कि लेआउट स्लाइड में सामग्री प्लेसहोल्डर आकार कैसे जोड़ें।

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddContentPlaceholder(20, 20, 500, 300);
}
```

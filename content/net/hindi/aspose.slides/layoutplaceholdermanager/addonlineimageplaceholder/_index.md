## LayoutPlaceholderManager.AddOnlineImagePlaceholder विधि

लेआउट स्लाइड में एक नया प्लेसहोल्डर आकार जोड़ता है जो ऑनलाइन छवि को रखता है।

```csharp
public IAutoShape AddOnlineImagePlaceholder(float x, float y, float width, float height)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| x | Single | नए प्लेसहोल्डर आकार का X समन्वय। |
| y | Single | नए प्लेसहोल्डर आकार का Y समन्वय। |
| width | Single | नए प्लेसहोल्डर आकार की चौड़ाई। |
| height | Single | नए प्लेसहोल्डर आकार की ऊँचाई। |

### वापसी मान

एक ऑनलाइन छवि प्लेसहोल्डर के साथ [`IAutoShape`](../../iautoshape) बनाया गया।

### उदाहरण

निम्नलिखित उदाहरण दिखाता है कि लेआउट स्लाइड में ऑनलाइन छवि प्लेसहोल्डर आकार कैसे जोड़ें।

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddOnlineImagePlaceholder(20, 20, 200, 200);
}
```

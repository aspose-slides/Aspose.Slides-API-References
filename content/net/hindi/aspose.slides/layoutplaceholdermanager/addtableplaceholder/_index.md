## LayoutPlaceholderManager.AddTablePlaceholder मेथड

लेआउट स्लाइड में एक नई प्लेसहोल्डर आकृति जोड़ता है जो एक तालिका को धारण करता है।

```csharp
public IAutoShape AddTablePlaceholder(float x, float y, float width, float height)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| x | Single | नए प्लेसहोल्डर आकृति का X निर्देशांक। |
| y | Single | नए प्लेसहोल्डर आकृति का Y निर्देशांक। |
| width | Single | नए प्लेसहोल्डर आकृति की चौड़ाई। |
| height | Single | नए प्लेसहोल्डर आकृति की ऊँचाई। |

### रिटर्न वैल्यू

[`IAutoShape`](../../iautoshape) को Table प्लेसहोल्डर के साथ बनाया गया।

### उदाहरण

निम्न उदाहरण दिखाता है कि लेआउट स्लाइड में Table प्लेसहोल्डर आकृति कैसे जोड़ें।

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTablePlaceholder(20, 20, 500, 200);
}
```

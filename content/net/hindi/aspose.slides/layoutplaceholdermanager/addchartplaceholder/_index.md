## LayoutPlaceholderManager.AddChartPlaceholder विधि

लेआउट स्लाइड में एक Chart रखने के लिए नया placeholder shape जोड़ता है।

```csharp
public IAutoShape AddChartPlaceholder(float x, float y, float width, float height)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| x | Single | नए placeholder shape का X निर्देशांक। |
| y | Single | नए placeholder shape का Y निर्देशांक। |
| width | Single | नए placeholder shape की चौड़ाई। |
| height | Single | नए placeholder shape की ऊँचाई। |

### रिटर्न मान

[`IAutoShape`](../../iautoshape) को एक Chart placeholder के साथ बनाया गया।

### उदाहरण

लेआउट स्लाइड में Chart placeholder shape जोड़ने का नीचे दिया गया उदाहरण दिखाता है।

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddChartPlaceholder(20, 20, 200, 200);
}
```

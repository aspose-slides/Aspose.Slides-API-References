## LayoutPlaceholderManager.AddVerticalTextPlaceholder विधि

लेआउट स्लाइड में टेक्स्ट सामग्री को लंबवत दिशा में रखने के लिए एक नया प्लेसहोल्डर आकार जोड़ता है।

```csharp
public IAutoShape AddVerticalTextPlaceholder(float x, float y, float width, float height)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| x | Single | नए प्लेसहोल्डर आकार का X निर्देशांक। |
| y | Single | नए प्लेसहोल्डर आकार का Y निर्देशांक। |
| width | Single | नए प्लेसहोल्डर आकार की चौड़ाई। |
| height | Single | नए प्लेसहोल्डर आकार की ऊँचाई। |

### रिटर्न वैल्यू

एक Text (Vertical) प्लेसहोल्डर के साथ बनाया गया [`IAutoShape`](../../iautoshape)।

### उदाहरण

निम्न उदाहरण दिखाता है कि लेआउट स्लाइड में Text (Vertical) प्लेसहोल्डर आकार कैसे जोड़ा जाए।

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalTextPlaceholder(20, 20, 300, 500);
}
```

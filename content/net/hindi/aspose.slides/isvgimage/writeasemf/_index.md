## ISvgImage.WriteAsEmf मेथड

SVG छवि को EMF फ़ाइल के रूप में सहेजता है।

```csharp
public void WriteAsEmf(Stream stream)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| stream | Stream | लक्ष्य स्ट्रीम |

### अपवाद

| अपवाद | शर्त |
| --- | --- |
| ArgumentNullException | लक्ष्य स्ट्रीम **null** है |

### उदाहरण

निम्न उदाहरण दर्शाता है कि SVG छवि को एक मेटाफाइल में कैसे सहेजा जाए।

```csharp
[C#]
// नई SVG छवि बनाता है
ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));

// SVG छवि को एक मेटाफाइल के रूप में सहेजता है
using (var fileStream = System.IO.File.OpenWrite("SvgAsEmf.emf"))
    svgImage.WriteAsEmf(fileStream);
```

यह नमूना दिखाता है कि प्रस्तुति छवि संग्रह में SVG छवि को मेटाफाइल के रूप में कैसे जोड़ा जाए।

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // नई SVG छवि बनाता है
    ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));
    using (var memStream = new MemoryStream())
    {
        // SVG छवि को एक मेटाफाइल के रूप में सहेजता है
        svgImage.WriteAsEmf(memStream);
        // इमेज संग्रह में मेटाफाइल जोड़ता है
        pres.Images.AddImage(memStream.ToArray());
    }
}
```

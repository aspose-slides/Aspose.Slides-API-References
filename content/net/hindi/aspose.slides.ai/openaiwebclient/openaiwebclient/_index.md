## OpenAIWebClient(string, string, string, HttpClient) {#constructor_1}

एक ऐसा OpenAI वेब क्लाइंट बनाता है जो बाहरी रूप से प्रबंधित HttpClient का उपयोग करता है। प्रदान किया गया HttpClient इस उदाहरण द्वारा नष्ट नहीं किया जाता और कॉलर द्वारा ही स्वामित्व में रहता है।

```csharp
public OpenAIWebClient(string model, string apiKey, string organizationId, HttpClient httpClient)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| model | String | OpenAI भाषा मॉडल। उपलब्ध मान: - gpt-4o - gpt-4o-mini - o1 - o1-mini - o3 - o3-mini |
| apiKey | String | OpenAI API कुंजी |
| organizationId | String | संगठन ID (वैकल्पिक) |
| httpClient | HttpClient | एक बाहरी रूप से प्रबंधित `HttpClient` उदाहरण |

### अपवाद

| अपवाद | शर्त |
| --- | --- |
| ArgumentException | API कुंजी मान null या खाली नहीं हो सकता |

| ArgumentNullException | HttpClient null नहीं हो सकता |

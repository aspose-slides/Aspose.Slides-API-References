## GeneratePresentationAsync(string, PresentationContentAmountType) {#generatepresentationasync}

एक पाठ विवरण से प्रस्तुति इंस्टेंस उत्पन्न करता है। आवश्यक भाषा में विषय, विचार, उद्धरण या पाठ अंश प्रदान करें।

```csharp
public Task<IPresentation> GeneratePresentationAsync(string description, 
    PresentationContentAmountType presentationContentAmount)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| description | String | विषय, विचार, उद्धरण या पाठ अंश। |
| presentationContentAmount | PresentationContentAmountType | परिणामी प्रस्तुति में सामग्री की मात्रा। |

### अपवाद

| अपवाद | शर्त |
| --- | --- |
| ArgumentException | AI चैट निर्देश null या खाली नहीं हो सकता। |

### टिप्पणियाँ

नीचे दिया गया उदाहरण डिफ़ॉल्ट [`AsposeAIWebClient`](../../asposeaiwebclient) का उपयोग करता है, जो पैरामीटरलेस [`SlidesAIAgent`](../slidesaiagent) कंस्ट्रक्टर द्वारा बनाया गया है और Aspose के अपने LLM से जुड़ता है। अलग AI प्रदाता उपयोग करने के लिए, अपना स्वयं का LLM प्रदान करें, या कनेक्शन को अनुकूलित करें (उदाहरण के लिए, अपना `HttpClient` प्रदान करके), [`IAIWebClient`](../../iaiwebclient) कार्यान्वयन को [`SlidesAIAgent`](../slidesaiagent) कंस्ट्रक्टर में पास करें। उपलब्ध कार्यान्वयन शामिल हैं:

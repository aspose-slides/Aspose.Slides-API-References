## SlidesAIAgent.TranslateAsync मेथड

AI का उपयोग करके एक प्रस्तुति को निर्दिष्ट भाषा में अनुवाद करता है।

```csharp
public Task TranslateAsync(IPresentation presentation, string language)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| presentation | IPresentation | लक्षित प्रस्तुति |
| language | String | लक्षित भाषा |

### अपवाद

| अपवाद | स्थिति |
| --- | --- |
| ArgumentNullException | प्रेजेंटेशन इंस्टेंस प्रदान नहीं किया गया है |
| ArgumentException | भाषा मान null या खाली नहीं हो सकता |

### टिप्पणियाँ

नीचे दिया गया उदाहरण डिफ़ॉल्ट [`AsposeAIWebClient`](../../asposeaiwebclient) का उपयोग करता है, जो पैरामीटर रहित [`SlidesAIAgent`](../slidesaiagent) कंस्ट्रक्टर द्वारा बनाया जाता है और Aspose के स्वयं के LLM से कनेक्ट होता है। अलग AI प्रदाता का उपयोग करने के लिए, अपना स्वयं का LLM उपलब्ध कराएँ, या कनेक्शन को कस्टमाइज़ करें (उदाहरण के लिए, अपना `HttpClient` प्रदान करके), [`IAIWebClient`](../../iaiwebclient) कार्यान्वयन को [`SlidesAIAgent`](../slidesaiagent) कंस्ट्रक्टर में पास करें। उपलब्ध कार्यान्वयन में शामिल हैं:

* [`AsposeAIWebClient`](../../asposeaiwebclient)
* [`OpenAIWebClient`](../../openaiwebclient)
* [`OpenAICompatibleWebClient`](../../openaicompatiblewebclient)

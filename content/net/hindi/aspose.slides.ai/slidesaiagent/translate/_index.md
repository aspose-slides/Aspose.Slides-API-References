## SlidesAIAgent.Translate विधि

AI का उपयोग करके किसी प्रस्तुति को निर्दिष्ट भाषा में अनुवादित करता है (सिंक्रोनस संस्करण)।

```csharp
public void Translate(IPresentation presentation, string language)
```

| परामीटर | प्रकार | विवरण |
| --- | --- | --- |
| presentation | IPresentation | लक्ष्य प्रस्तुति |
| language | String | लक्ष्य भाषा |

### अपवाद

| अपवाद | शर्त |
| --- | --- |
| ArgumentNullException | प्रस्तुति उदाहरण प्रदान नहीं किया गया है |
| ArgumentException | भाषा मान शून्य या खाली नहीं हो सकता |

### टिप्पणियाँ

नीचे दिया गया उदाहरण डिफ़ॉल्ट [`AsposeAIWebClient`](../../asposeaiwebclient), जो पैरामीटर-रहित [`SlidesAIAgent`](../slidesaiagent) कंस्ट्रक्टर द्वारा बनाया गया है और Aspose के अपने LLM से जुड़ता है। अलग AI प्रदाता का उपयोग करने के लिए, अपना स्वयं का LLM प्रदान करें, या कनेक्शन को अनुकूलित करें (उदाहरण के लिए, अपना स्वयं का `HttpClient` प्रदान करके), [`SlidesAIAgent`](../slidesaiagent) कंस्ट्रक्टर को [`IAIWebClient`](../../iaiwebclient) कार्यान्वयन पास करें। उपलब्ध कार्यान्वयन में शामिल हैं:

* [`AsposeAIWebClient`](../../asposeaiwebclient)

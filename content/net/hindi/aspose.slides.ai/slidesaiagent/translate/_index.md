## SlidesAIAgent.Translate मेथड

AI (समकालिक संस्करण) का उपयोग करके प्रस्तुति को निर्दिष्ट भाषा में अनुवाद करता है।

```csharp
public void Translate(IPresentation presentation, string language)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| presentation | IPresentation | लक्षित प्रस्तुति |
| language | String | लक्षित भाषा |

### अपवाद

| अपवाद | स्थिति |
| --- | --- |
| ArgumentNullException | प्रेजेंटेशन इंस्टेंस प्रदान नहीं किया गया है |
| ArgumentException | भाषा मान null या खाली नहीं हो सकता |

### टिप्पणियाँ

नीचे दिया गया उदाहरण डिफ़ॉल्ट [`AsposeAIWebClient`](../../asposeaiwebclient) का उपयोग करता है, जिसे पैरामीटररहित [`SlidesAIAgent`](../slidesaiagent) कंस्ट्रक्टर द्वारा बनाया गया है और Aspose के अपने LLM से जुड़ता है। अलग AI प्रदाता का उपयोग करने के लिए, अपना स्वयं का LLM प्रदान करें, या कनेक्शन को अनुकूलित करें (उदाहरण के लिए, अपना `HttpClient` प्रदान करके), [`SlidesAIAgent`](../slidesaiagent) कंस्ट्रक्टर में [`IAIWebClient`](../../iaiwebclient) कार्यान्वयन पास करें। उपलब्ध कार्यान्वयन शामिल हैं:

* [`AsposeAIWebClient`](../../asposeaiwebclient)
* [`OpenAIWebClient`](../../openaiwebclient)

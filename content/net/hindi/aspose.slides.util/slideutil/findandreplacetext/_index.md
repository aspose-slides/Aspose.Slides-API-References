## SlideUtil.FindAndReplaceText विधि

प्रस्तुति में दिए गए स्वरूप के साथ टेक्स्ट को खोजता है और बदलता है

```csharp
public static void FindAndReplaceText(IPresentation presentation, bool withMasters, string find, 
    string replace, PortionFormat format = null)
```

| पैरामीटर | टाइप | विवरण |
| --- | --- | --- |
| presentation | IPresentation | स्कैन किया गया प्रस्तुति। |
| withMasters | Boolean | निर्धारित करता है कि मास्टर स्लाइड्स को स्कैन किया जाना चाहिए या नहीं। |
| find | String | खोजने के लिए स्ट्रिंग मान। |
| replace | String | बदलने के लिए स्ट्रिंग मान। |
| format | PortionFormat | टेक्स्ट भाग को बदलने के लिए स्वरूप। यदि null हो तो खोजे गए स्ट्रिंग के पहले अक्षर के स्वरूप का उपयोग किया जाएगा। |

### उदाहरण

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PortionFormat format = new PortionFormat
    {
        FontHeight = 24f,
        FontItalic = NullableBool.True,
        FillFormat =
        {
            FillType = FillType.Solid,
            SolidFillColor =
            {
                Color = Color.Red
            }
        }
    };
    Aspose.Slides.Util.SlideUtil.FindAndReplaceText(pres, true, "[this block] ", "my text ", format);
    pres.Save("replaced", SaveFormat.Pptx);
}
```

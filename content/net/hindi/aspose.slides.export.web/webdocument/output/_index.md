## WebDocument.Output प्रॉपर्टी

दस्तावेज़ के आउटपुट तत्वों का संग्रह लौटाता है। केवल-पढ़ने योग्य `Output`.

```csharp
public Output Output { get; }
```

### उदाहरण

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    var options = new WebDocumentOptions
    {
        TemplateEngine = new RazorTemplateEngine(),
        OutputSaver = new FileOutputSaver(),
        EmbedImages = false
    };
    
    WebDocument document = new WebDocument(options);

    // output फ़ाइलों में "index.html" जोड़ें, इसे बनाने के लिए "index" टेम्पलेट का उपयोग करें और pres वेरिएबल को मॉडल के रूप में उपयोग करें
    document.Output.Add("index.html", "index", pres);

    // ... दस्तावेज़ के अन्य विकल्प सेट करें और फिर दस्तावेज़ को सहेजें
    document.Save();
}
```

## WebDocument.Global प्रॉपर्टी

दस्तावेज़ का ग्लोबल स्टोरेज लौटाता है। केवल-पढ़ने योग्य [`Storage`](../../storage)।

```csharp
public Storage Global { get; }
```

### उदाहरण

इस `Global` प्रॉपर्टी का उपयोग करके ([`Storage`](../../storage) इंटरफ़ेस का कार्यान्वयन) प्रॉपर्टी को बाद में टेम्प्लेट में उपयोग के लिए रखा जा सकता है:

```csharp
[C#]
var options = new WebDocumentOptions
{
    TemplateEngine = new RazorTemplateEngine(),
    OutputSaver = new FileOutputSaver(),
    EmbedImages = false
};

WebDocument document = new WebDocument(options);

// put "slideMargin" property to use from templates
document.Global.Put("slideMargin", 10);

// ... set up other options of the document and then save the document
document.Save();
```

## PdfOptions.IncludeOleData प्रॉपर्टी

सभी OLE डेटा को प्रस्तुति से उत्पन्न PDF में एम्बेडेड फ़ाइलों में बदलने के लिए True. पढ़ने/लिखने योग्य बूलियन.

```csharp
public bool IncludeOleData { get; set; }
```

### टिप्पणी

डिफ़ॉल्ट **false**।

### उदाहरण

उदाहरण:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions options = new PdfOptions { IncludeOleData = true };
    pres.Save("pres.pdf", SaveFormat.Pdf, options);
}
```

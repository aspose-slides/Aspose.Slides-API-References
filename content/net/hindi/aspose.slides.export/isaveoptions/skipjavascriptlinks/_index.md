## ISaveOptions.SkipJavaScriptLinks प्रॉपर्टी

प्रेज़ेंटेशन सहेजते समय JavaScript कॉल वाले हाइपरलिंक्स को छोड़ें या नहीं, यह निर्धारित करता है। पढ़ने/लिखने योग्य Boolean। डिफ़ॉल्ट मान **false** है।

```csharp
public bool SkipJavaScriptLinks { get; set; }
```

### टिप्पणी

जब यह प्रॉपर्टी **true** पर सेट होती है, तो सहेजते समय JavaScript कॉल वाले हाइपरलिंक्स को अनदेखा किया जाएगा।

जब यह प्रॉपर्टी **false** पर सेट होती है, तो सभी हाइपरलिंक्स सहेजे जाएंगे।

### उदाहरण

उदाहरण:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("result_without_JavaScript_links.html", SaveFormat.Html, new HtmlOptions()
    {
        SkipJavaScriptLinks = true
    });
}
```

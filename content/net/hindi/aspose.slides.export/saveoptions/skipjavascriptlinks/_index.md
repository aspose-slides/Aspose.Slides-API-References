## SaveOptions.SkipJavaScriptLinks प्रॉपर्टी

निर्दिष्ट करता है कि प्रस्तुति को सहेजते समय JavaScript कॉल वाले हाइपरलिंक को छोड़ना है या नहीं। पढ़ें/लिखें Boolean। डिफ़ॉल्ट मान **false** है।

```csharp
public bool SkipJavaScriptLinks { get; set; }
```

### टिप्पणियाँ

जब इस प्रॉपर्टी को **true** पर सेट किया जाता है, तो सहेजते समय JavaScript कॉल वाले हाइपरलिंक को अनदेखा किया जाएगा।

जब इस प्रॉपर्टी को **false** पर सेट किया जाता है, तो सभी हाइपरलिंक सहेजे जाएंगे।

### उदाहरण

उदाहरण:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("result_without_JavaScript_links.html", SaveFormat.Html, new HtmlOptions()
    {
        SkipJavaScriptLinks = true
    });
}
```

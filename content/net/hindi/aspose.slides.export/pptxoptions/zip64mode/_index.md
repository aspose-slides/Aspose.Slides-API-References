## PptxOptions.Zip64Mode प्रॉपर्टी

निर्दिष्ट करता है कि प्रस्तुति दस्तावेज़ के लिए ZIP64 फ़ॉर्मेट प्रयोग किया जाता है या नहीं। डिफ़ॉल्ट मान IfNecessary है।

```csharp
public Zip64Mode Zip64Mode { get; set; }
```

### उदाहरण

उदाहरण:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("demo-zip64.pptx", SaveFormat.Pptx, new PptxOptions()
    {
        Zip64Mode = Zip64Mode.Always
    });
}
```

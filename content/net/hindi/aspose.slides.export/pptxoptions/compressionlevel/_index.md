## PptxOptions.CompressionLevel प्रॉपर्टी

प्रस्तुति दस्तावेज़ को सहेजते समय उपयोग किए जाने वाले संपीड़न स्तर को निर्दिष्ट करता है। डिफ़ॉल्ट मान Level6 है।

```csharp
public CompressionLevel CompressionLevel { get; set; }
```

### टिप्पणियाँ

उच्च संपीड़न स्तर छोटे फ़ाइलें उत्पन्न करते हैं लेकिन अधिक प्रसंस्करण समय की आवश्यकता होती है। वास्तविक संपीड़न अनुपात प्रस्तुति की सामग्री पर निर्भर करता है।

### उदाहरण

उदाहरण:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("demo-level8.pptx", SaveFormat.Pptx, new PptxOptions()
    {
        CompressionLevel = CompressionLevel.Level8
    });
}
```

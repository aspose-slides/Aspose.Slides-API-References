## ISSVGOptions.DisableFontLigatures प्रॉपर्टी

एक मान प्राप्त करता है या सेट करता है जो यह दर्शाता है कि क्या टेक्स्ट लिगेचर का उपयोग किए बिना रेंडर किया जाता है। जब इसे `true` पर सेट किया जाता है, तो रेंडर आउटपुट में लिगेचर निष्क्रिय हो जाएंगे। डिफ़ॉल्ट रूप से, इस प्रॉपर्टी को `false` पर सेट किया जाता है।

```csharp
public bool DisableFontLigatures { get; set; }
```

### उदाहरण

उदाहरण:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    SVGOptions options = new SVFOptions
    {
        DisableFontLigatures = true // टेक्स्ट रेंडरिंग में लिगेचर को निष्क्रिय करें
    };
    
    using (FileStream fileStream = new FileStream("slide-0.svg", FileMode.Create, FileAccess.Write))
    {
        pres.Slides[0].WriteAsSvg(fileStream);   
    }
}
```

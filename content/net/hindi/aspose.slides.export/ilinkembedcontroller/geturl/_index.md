## ILinkEmbedController.GetUrl विधि

किसी बाहरी वस्तु के लिए एक URL लौटाता है। यह विधि हमेशा तब कॉल की जाती है जब [`GetObjectStoringLocation`](../getobjectstoringlocation) ने Link लौटाया हो और इसे कॉल किया जा सकता है जब [`GetObjectStoringLocation`](../getobjectstoringlocation) ने Embed लौटाया हो लेकिन एम्बेडिंग असम्भव हो। इसे समान ऑब्जेक्ट आईडी के लिए कई बार कॉल किया जा सकता है।

```csharp
public string GetUrl(int id, int referrer)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| id | Int32 | ऑब्जेक्ट आईडी। यह आईडी ऑपरेशन-व्यापी अद्वितीय है। |
| referrer | Int32 | संदर्भित करने वाले ऑब्जेक्ट की आईडी या 0, यदि ऑब्जेक्ट मूल दस्तावेज़ द्वारा संदर्भित है। यह सापेक्ष लिंक उत्पन्न करने के लिए उपयोग किया जा सकता है। |

### रिटर्न मान

बाहरी ऑब्जेक्ट का URL या null यदि इस ऑब्जेक्ट को अनदेखा किया जाना चाहिए।

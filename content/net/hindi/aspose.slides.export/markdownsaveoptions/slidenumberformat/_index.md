## MarkdownSaveOptions.SlideNumberFormat प्रॉपर्टी

Markdown आउटपुट में स्लाइड नंबर हेडर के लिए उपयोग की जाने वाली फ़ॉर्मेट स्ट्रिंग को प्राप्त करता है या सेट करता है। फ़ॉर्मेट में “{0}” प्लेसहोल्डर शामिल होना आवश्यक है, जिसे निर्यात के दौरान स्लाइड इंडेक्स से बदल दिया जाएगा। उदाहरण: “# Slide {0}” परिणामस्वरूप “# Slide 1”, “# Slide 2”, आदि उत्पन्न करेगा।

```csharp
public string SlideNumberFormat { get; set; }
```

## InkOptions.HideInk प्रॉपर्टी

निर्यात किए दस्तावेज़ में Ink तत्वों को दिखाता या छुपाता है।

```csharp
public bool HideInk { get; set; }
```

### टिप्पणियाँ

डिफ़ॉल्ट मान false है।

### उदाहरण

अगला उदाहरण दिखाता है कि निर्यात किए गए PDF दस्तावेज़ में Ink तत्वों को कैसे छिपाया जाए:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.HideInk = true;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

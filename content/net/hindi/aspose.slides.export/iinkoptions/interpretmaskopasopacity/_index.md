## IInkOptions.InterpretMaskOpAsOpacity प्रॉपर्टी

ब्रश को रेंडर करने के लिए ROP ऑपरेशन या अपारदर्शिता का उपयोग करता है।

```csharp
public bool InterpretMaskOpAsOpacity { get; set; }
```

### टिप्पणी

डिफ़ॉल्ट मान true है।

### उदाहरण

अगला उदाहरण ROP का उपयोग करके Ink तत्वों को निर्यात करने के लिए सेट करने का प्रदर्शन करता है:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.InterpretMaskOpAsOpacity = false;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

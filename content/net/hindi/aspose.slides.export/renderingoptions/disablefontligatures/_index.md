## RenderingOptions.DisableFontLigatures प्रॉपर्टी

एक मान प्राप्त करता है या सेट करता है जो यह दर्शाता है कि टेक्स्ट को लिगेचर का उपयोग किए बिना रेंडर किया जाता है। जब `true` पर सेट किया जाता है, तो रेंडर किए गए आउटपुट में लिगेचर निष्क्रिय हो जाएंगे। डिफ़ॉल्ट रूप से, यह प्रॉपर्टी `false` पर सेट होती है।

```csharp
public bool DisableFontLigatures { get; set; }
```

### उदाहरण

उदाहरण:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    RenderingOptions options = new RenderingOptions
    {
        DisableFontLigatures = true // टेक्स्ट रेंडरिंग में लिगेचर को निष्क्रिय करें
    };
    
    Bitmap[] renderedSlides = pres.GetImage(options);
    for (var index = 0; index < renderedSlides.Length; index++)
    {
        var slideImage = renderedSlides[index];
        slideImage.Save($"slide-{index}.png");
    }
}
```

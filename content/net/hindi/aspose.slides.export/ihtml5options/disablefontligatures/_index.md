## IHtml5Options.DisableFontLigatures प्रॉपर्टी

यह मान प्राप्त करता है या सेट करता है जो दर्शाता है कि क्या टेक्स्ट को लिगेचर का उपयोग किए बिना रेंडर किया गया है। जब इसे `true` पर सेट किया जाता है, तो रेंडर आउटपुट में लिगेचर निष्क्रिय हो जाएंगे। डिफ़ॉल्ट रूप से, यह प्रॉपर्टी `false` पर सेट है।

```csharp
public bool DisableFontLigatures { get; set; }
```

### उदाहरण

उदाहरण:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    Html5Options options = new Html5Options
    {
        DisableFontLigatures = true // टेक्स्ट रेंडरिंग में लिगेचर निष्क्रिय करें
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html5, options);
}
```

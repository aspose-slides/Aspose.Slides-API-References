## IHtmlOptions.DisableFontLigatures प्रॉपर्टी

पाठ को लिगेचर का उपयोग किए बिना रेंडर किया जाता है या नहीं, यह दर्शाने वाला मान प्राप्त या निर्धारित करता है। जब इसे `true` पर सेट किया जाता है, तो रेंडर किए गए आउटपुट में लिगेचर निष्क्रिय हो जाएंगे। डिफ़ॉल्ट रूप से, यह प्रॉपर्टी `false` पर सेट होती है।

```csharp
public bool DisableFontLigatures { get; set; }
```

### उदाहरण

उदाहरण:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    HtmlOptions options = new HtmlOptions
    {
        DisableFontLigatures = true // पाठ रेंडरिंग में लिगेचर को निष्क्रिय करें
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html, options);
}
```

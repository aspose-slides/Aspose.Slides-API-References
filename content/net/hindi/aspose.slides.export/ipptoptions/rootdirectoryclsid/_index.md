## IPptOptions.RootDirectoryClsid प्रॉपर्टी

ऑब्जेक्ट क्लास GUID (CLSID) को दर्शाता है जो रूट डायरेक्टरी एंट्री में संग्रहीत होता है। दस्तावेज़ के एप्लीकेशन के COM सक्रियण के लिए इसका उपयोग किया जा सकता है। डिफ़ॉल्ट मान '64818D11-4F9B-11CF-86EA-00AA00B929E8' है जो 'Microsoft Powerpoint.Slide.8' के अनुरूप है।

```csharp
public Guid RootDirectoryClsid { get; set; }
```

### उदाहरण

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    PptOptions pptOptions = new PptOptions();
    
    /// CLSID को 'Microsoft Powerpoint.Show.8' पर सेट करें
    pptOptions.RootDirectoryClsid = new Guid("64818D10-4F9B-11CF-86EA-00AA00B929E8");
    
    pres.Save("pres.ppt", SaveFormat.Ppt, pptOptions);
}
```

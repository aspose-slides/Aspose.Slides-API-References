## HtmlOptions.JpegQuality गुण

PDF दस्तावेज़ के भीतर JPEG छवियों की गुणवत्ता निर्धारित करने वाला मान लौटाता है या सेट करता है। पढ़ें/लिखें Byte.

```csharp
public byte JpegQuality { get; set; }
```

### टिप्पणी

केवल तब प्रभावी होता है जब दस्तावेज़ में JPEG छवियां हों।

दस्तावेज़ को PDF प्रारूप में सहेजते समय छवियों की गुणवत्ता प्राप्त करने या सेट करने के लिए इस गुण का उपयोग करें। मान 0 से 100 के बीच बदल सकता है जहाँ 0 सबसे ख़राब गुणवत्ता लेकिन अधिकतम संपीड़न दर्शाता है और 100 सबसे अच्छी गुणवत्ता लेकिन न्यूनतम संपीड़न।

डिफ़ॉल्ट मान **95** है।

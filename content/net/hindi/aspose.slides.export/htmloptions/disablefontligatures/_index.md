## HtmlOptions.DisableFontLigatures प्रॉपर्टी

एक मान प्राप्त करता है या सेट करता है जो संकेत करता है कि क्या पाठ लिगेचर का उपयोग किए बिना रेंडर किया जाता है। जब इसे `true` पर सेट किया जाता है, तो रेंडर किए गए आउटपुट में लिगेचर अक्षम हो जाएंगे। डिफ़ॉल्ट रूप से, यह प्रॉपर्टी `false` पर सेट होती है।

```csharp
public bool DisableFontLigatures { get; set; }
```

### उदाहरण

उदाहरण:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    HtmlOptions options = new HtmlOptions
    {
        DisableFontLigatures = true // पाठ रेंडरिंग में लिगेचर को अक्षम करें
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html, options);
}
```

## GifOptions.DefaultDelay प्रॉपर्टी

डिफ़ॉल्ट देरी समय [ms] को प्राप्त करता है या सेट करता है। यदि [`AdvanceAfterTime`](../../../aspose.slides/islideshowtransition/advanceaftertime) सेट नहीं है तो यह मान उपयोग किया जाएगा। डिफ़ॉल्ट मान 1000 है।

```csharp
public int DefaultDelay { get; set; }
```

```csharp
[C#]

}
```

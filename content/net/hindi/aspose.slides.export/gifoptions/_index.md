### उदाहरण

निम्नलिखित उदाहरण दिखाता है कि कस्टम सेटिंग्स का उपयोग करके प्रेजेंटेशन को एनिमेटेड GIF में कैसे रूपांतरित किया जाए।

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    pres.Save("pres.gif", SaveFormat.Gif, new GifOptions
    {
        FrameSize = new Size(960, 720), // उत्पन्न GIF का आकार
        DefaultDelay = 2000, // प्रत्येक स्लाइड कितने समय तक प्रदर्शित होगी जब तक वह अगली में बदल न जाए
        TransitionFps = 35 // बेहतर ट्रांज़िशन एनीमेशन गुणवत्ता के लिए FPS बढ़ाएँ
    });
}
```

## ImageTransformOperationCollection.CopyTo मेथड

ICollection के तत्वों को एक Array में कॉपी करता है, जो किसी विशिष्ट Array इंडेक्स से शुरू होता है।

```csharp
public void CopyTo(IImageTransformOperation[] array, int arrayIndex)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| array | IImageTransformOperation[] | एक-आयामी Array जो ICollection से कॉपी किए गए तत्वों का गंतव्य है। Array को शून्य-आधारित अनुक्रमणिका होनी चाहिए। |
| arrayIndex | Int32 | *array* में वह शून्य-आधारित इंडेक्स जहाँ कॉपी शुरू होता है। |

### अपवाद

| अपवाद | शर्त |
| --- | --- |
| ArgumentNullException | *array* null है। |
| ArgumentOutOfRangeException | *arrayIndex* 0 से कम है। |
| ArgumentException | स्रोत ICollection में तत्वों की संख्या उपलब्ध स्थान से अधिक है जो *arrayIndex* से लेकर गंतव्य *array* के अंत तक है। |

विभिन्न प्रभावों के साथ काम करने के लिए Microsoft PowerPoint प्रस्तुतियों में क्लासेस सम्मिलित करता है।

## क्लासेस

| क्लास | विवरण |
| --- | --- |
| [AlphaBiLevel](./alphabilevel) | Alpha Bi-Level प्रभाव का प्रतिनिधित्व करता है। Alpha (Opacity) मान जो थ्रेशहोल्ड से कम हैं, उन्हें 0 (पूर्ण पारदर्शी) कर दिया जाता है और थ्रेशहोल्ड से बराबर या अधिक मानों को 100% (पूर्ण अपारदर्शी) कर दिया जाता है। |
| [AlphaCeiling](./alphaceiling) | Alpha Ceiling प्रभाव का प्रतिनिधित्व करता है। Alpha (opacity) मान जो शून्य से अधिक हैं, उन्हें 100% कर दिया जाता है। अर्थात्, आंशिक अपारदर्शी सभी वस्तुएँ पूर्ण अपारदर्शी बन जाती हैं। |
| [AlphaFloor](./alphafloor) | Alpha Floor प्रभाव का प्रतिनिधित्व करता है। Alpha (opacity) मान जो 100% से कम हैं, उन्हें शून्य कर दिया जाता है। अर्थात्, आंशिक पारदर्शी सभी वस्तुएँ पूर्ण पारदर्शी बन जाती हैं। |
| [AlphaInverse](./alphainverse) | Alpha Inverse प्रभाव का प्रतिनिधित्व करता है। Alpha (opacity) मानों को 100% से घटाकर उलटा किया जाता है। |
| [AlphaModulate](./alphamodulate) | Alpha Modulate प्रभाव का प्रतिनिधित्व करता है। प्रभाव की Alpha (opacity) मान को स्थिर प्रतिशत से गुणा किया जाता है। प्रभाव कंटेनर वह प्रभाव निर्धारित करता है जिसमें मोड्यूलेट करने हेतु Alpha मान होते हैं। |
| [AlphaModulateFixed](./alphamodulatefixed) | Alpha Modulate Fixed प्रभाव का प्रतिनिधित्व करता है। प्रभाव की Alpha (opacity) मान को स्थिर प्रतिशत से गुणा किया जाता है। |
| [AlphaReplace](./alphareplace) | Alpha Replace प्रभाव का प्रतिनिधित्व करता है। प्रभाव की Alpha (opacity) मान को एक निश्चित Alpha से बदल दिया जाता है। |
| [BiLevel](./bilevel) | Bi-Level (काला/सफ़ेद) प्रभाव का प्रतिनिधित्व करता है। इनपुट रंग जिनकी ल्यूमिनेंस निर्दिष्ट थ्रेशहोल्ड से कम है, उन्हें काला कर दिया जाता है। इनपुट रंग जिनकी ल्यूमिनेंस बराबर या अधिक है, उन्हें सफ़ेद कर दिया जाता है। Alpha प्रभाव मान इस प्रभाव से अपरिवर्तित रहते हैं। |
| [Blur](./blur) | Blur प्रभाव का प्रतिनिधित्व करता है जो पूरे आकार पर लागू होता है, जिसमें उसका भराव भी शामिल है। सभी रंग चैनल, Alpha सहित, प्रभावित होते हैं। |
| [BrightnessContrast](./brightnesscontrast) | BrightnessContrast प्रभाव का प्रतिनिधित्व करता है। चमक और कंट्रास्ट को समायोजित करता है। |
| [ColorChange](./colorchange) | Color Change प्रभाव का प्रतिनिधित्व करता है। FromColor के उदाहरणों को ToColor के उदाहरणों से बदल दिया जाता है। |
| [ColorReplace](./colorreplace) | Color Replacement प्रभाव का प्रतिनिधित्व करता है। सभी प्रभाव रंग को एक निश्चित रंग में बदल दिया जाता है। Alpha मान अपरिवर्तित रहते हैं। |
| [Duotone](./duotone) | Duotone प्रभाव का प्रतिनिधित्व करता है। प्रत्येक पिक्सेल के लिए, Color1 और Color2 को रैखिक इंटरपोलेशन द्वारा मिलाकर नया रंग निर्धारित किया जाता है। |
| [EffectFactory](./effectfactory) | प्रभाव बनाने की अनुमति देता है। |
| [FillOverlay](./filloverlay) | Fill Overlay प्रभाव का प्रतिनिधित्व करता है। Fill overlay का उपयोग वस्तु के लिए अतिरिक्त भराव निर्दिष्ट करने और दो भरावों को मिलाने के लिए किया जा सकता है। |
| [Glow](./glow) | Glow प्रभाव का प्रतिनिधित्व करता है, जिसमें वस्तु के किनारों के बाहर रंगीन धुंधला रूपरेखा जोड़ी जाती है। |
| [GrayScale](./grayscale) | Gray Scale प्रभाव का प्रतिनिधित्व करता है। सभी प्रभाव रंग मानों को उनकी ल्यूमिनेंस के अनुसार एक ग्रे शेड में परिवर्तित करता है। Alpha (opacity) मान अपरिवर्तित रहते हैं। |
| [HSL](./hsl) | Hue/Saturation/Luminance प्रभाव का प्रतिनिधित्व करता है। hue, saturation, और luminance को प्रत्येक को वर्तमान मान के सापेक्ष समायोजित किया जा सकता है। |
| [ImageTransformOCollectionEffectiveData](./imagetransformocollectioneffectivedata) | अपरिवर्तनीय वस्तु जो प्रभावी छवि रूपांतरण प्रभावों का केवल-पढ़ने योग्य संग्रह प्रतिनिधित्व करती है। |
| [ImageTransformOperation](./imagetransformoperation) | अमूर्त छवि रूपांतरण प्रभाव का प्रतिनिधित्व करता है। |
| [ImageTransformOperationCollection](./imagetransformoperationcollection) | छवि पर लागू प्रभावों का संग्रह प्रतिनिधित्व करता है। |
| [ImageTransformOperationFactory](./imagetransformoperationfactory) | छवि रूपांतरण संचालन बनाने की अनुमति देता है। |
| [InnerShadow](./innershadow) | Inner Shadow प्रभाव का प्रतिनिधित्व करता है। |
| [Luminance](./luminance) | Luminance प्रभाव का प्रतिनिधित्व करता है। चमक सभी रंगों को रैखिक रूप से सफ़ेद या काले की ओर शिफ्ट करती है। कंट्रास्ट सभी रंगों को एक-दूसरे के करीब या दूर स्केल करता है। |
| [OuterShadow](./outershadow) | Outer Shadow प्रभाव का प्रतिनिधित्व करता है। |
| [PresetShadow](./presetshadow) | Preset Shadow प्रभाव का प्रतिनिधित्व करता है। |
| [Reflection](./reflection) | Reflection प्रभाव का प्रतिनिधित्व करता है। |
| [SoftEdge](./softedge) | Soft Edge प्रभाव का प्रतिनिधित्व करता है। आकार के किनारे धुंधले होते हैं, जबकि भराव अप्रभावित रहता है। |
| [Tint](./tint) | Tint प्रभाव का प्रतिनिधित्व करता है। प्रभाव रंग मानों को निर्दिष्ट मात्रा में hue की ओर/से दूर शिफ्ट करता है। |

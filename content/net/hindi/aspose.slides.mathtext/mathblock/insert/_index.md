## MathBlock.Insert विधि

निर्दिष्ट इंडेक्स पर संग्रह में एक MathElement सम्मिलित करता है।

```csharp
public void Insert(int index, IMathElement item)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| index | Int32 | शून्य-आधारित इंडेक्स जहाँ MathElement सम्मिलित किया जाना चाहिए। |
| item | IMathElement | डालने के लिए MathElement। |

### उदाहरण

उदाहरण:

```csharp
[C#]
MathBlock mathBlock = new MathBlock(new MathematicalText("x"));
MathematicalText plusElement = new MathematicalText("+");
mathBlock.Add(plusElement);
mathBlock.Insert(0, new MathRadical(new MathematicalText("x"), new MathematicalText("3")));
```

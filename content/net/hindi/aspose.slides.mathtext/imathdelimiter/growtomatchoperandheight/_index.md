## IMathDelimiter.GrowToMatchOperandHeight प्रॉपर्टी

BeginningCharacter, SeparatorCharacter, EndingCharacter की वृद्धि को निर्दिष्ट करता है। जब true हो, तो डिलिमिटर ऊर्ध्वाधर रूप से बढ़कर अपने ऑपरेण्ड की ऊँचाई से मेल खाते हैं। डिफ़ॉल्ट मान true है।

```csharp
public bool GrowToMatchOperandHeight { get; set; }
```

### उदाहरण

उदाहरण:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.GrowToMatchOperandHeight = false;
```

## MathJustification एन्यूमरेशन

गणित पैराग्राफ (एक ही पैराग्राफ में निकटवर्ती गणितीय पाठ की श्रृंखला) की संरेखण को निर्दिष्ट करता है।

public enum MathJustification

### मान

| Name | Value | Description |
| --- | --- | --- |
| LeftJustified | `1` | गणितीय पाठ को बाएँ संरेखित करें |
| RightJustified | `2` | गणितीय पाठ को दाएँ संरेखित करें |
| Centered | `3` | गणितीय पाठ को केंद्र में संरेखित करें |
| CenteredAsGroup | `4` | पूरे गणितीय ब्लॉकों के समूह को पृष्ठ पर एक इकाई के रूप में केंद्रित करें। इसका अर्थ है कि गणितीय पाठ के उदाहरण एक दूसरे के संबंध में संरेखित किए जा सकते हैं, लेकिन सम्पूर्ण गणितीय पाठ का समूह संपूर्ण रूप में केंद्रित रहता है। |

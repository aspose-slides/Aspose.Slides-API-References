## IMathMatrix.RowGap प्रॉपर्टी

मैट्रिक्स की पंक्तियों के बीच लंबवत दूरी का मान; यदि RowGapRule को 3 ("Exactly") पर सेट किया जाता है, तो इकाई को ट्विप्स (1/20वाँ भाग पॉइंट) के रूप में व्याख्यायित किया जाता है। यदि RowGapRule को 4 ("Multiple") पर सेट किया जाता है, तो इकाई को आधी-लाइन के रूप में व्याख्यायित किया जाता है। Default: 0

```csharp
public uint RowGap { get; set; }
```

### उदाहरण

Example:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

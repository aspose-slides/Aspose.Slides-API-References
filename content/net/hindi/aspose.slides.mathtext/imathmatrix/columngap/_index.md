## IMathMatrix.ColumnGap प्रॉपर्टी

मैट्रिक्स की कॉलमों के बीच क्षैतिज अंतराल का मान; यदि ColumnGapRule को 3 ("Exactly") पर सेट किया जाता है, तो इकाई को टविप्स (पॉइंट का 1/20 भाग) के रूप में व्याख्यायित किया जाता है। यदि ColumnGapRule को 4 ("Multiple") पर सेट किया जाता है, तो इकाई को 0.5 इम् वृद्धि की संख्या के रूप में व्याख्यायित किया जाता है। अन्य मामलों में अनदेखा किया जाता है। Default: 0

```csharp
public uint ColumnGap { get; set; }
```

### उदाहरण

Example:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.ColumnGapRule = MathSpacingRules.Exactly;
matrix.ColumnGap = 20;
```

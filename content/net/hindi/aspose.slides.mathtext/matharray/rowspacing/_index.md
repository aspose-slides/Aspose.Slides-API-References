## MathArray.RowSpacing प्रॉपर्टी

ऐरे की पंक्तियों के बीच की स्पेसिंग। यह केवल तब उपयोग किया जाता है जब RowSpacingRule को 3 पर सेट किया गया हो, ठीक उसी स्थिति में माप की इकाई पॉइंट्स होती है, या Multiple जब माप की इकाई आधी-लाइन होती है। Default: 0

```csharp
public uint RowSpacing { get; set; }
```

### उदाहरण

Example:

```csharp
[C#]
IMathArray mathArray = new MathArray(new MathematicalText("item1"));
mathArray.RowSpacingRule = MathRowSpacingRule.Exactly;
mathArray.RowSpacing = 10;
```

## IMathArray.RowSpacing प्रॉपर्टी

एक array की पंक्तियों के बीच की दूरी। यह केवल तब उपयोग किया जाता है जब RowSpacingRule को 3 पर सेट किया जाता है। ठीक उसी स्थिति में माप की इकाई points होती है या Multiple में माप की इकाई आधी-लाइनें होती हैं। डिफ़ॉल्ट: 0

```csharp
public uint RowSpacing { get; set; }
```

### उदाहरण

उदाहरण:

```csharp
[C#]
IMathArray mathArray = new MathArray(new MathematicalText("item1"));
mathArray.RowSpacingRule = MathRowSpacingRule.Exactly;
mathArray.RowSpacing = 10;
```

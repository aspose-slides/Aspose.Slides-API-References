## MathBox.ExplicitBreak प्रॉपर्टी

Explicit break यह निर्धारित करता है कि Box ऑब्जेक्ट की शुरुआत में लाइन ब्रेक है या नहीं, जिससे लाइन Box ऑब्जेक्ट की शुरुआत पर ही मोड़ लेती है। यह गणितीय पाठ की पिछली पंक्ति में ऑपरेटर की संख्या निर्धारित करता है जिसे वर्तमान पंक्ति के गणितीय पाठ के संरेखण बिंदु के रूप में उपयोग किया जाएगा। संभावित मान: 1..255 डिफ़ॉल्ट: 0 (कोई explicit break नहीं)

```csharp
public byte ExplicitBreak { get; set; }
```

### उदाहरण

उदाहरण:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.ExplicitBreak = 1;
```

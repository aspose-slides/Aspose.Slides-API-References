## MathBorderBox.StrikethroughBottomLeftToTopRight प्रॉपर्टी

Strikethrough Bottom-Left to Top-Right (डिफ़ॉल्ट है false). बॉर्डर बॉक्स के निचले-बाएँ कोने से ऊपर-दाएँ कोने तक की स्ट्राइकथ्रू विकर्ण रेखा की छिपी या दृश्यमान स्थिति निर्दिष्ट करता है।

```csharp
public bool StrikethroughBottomLeftToTopRight { get; set; }
```

### उदाहरण

उदाहरण:

```csharp
[C#]
MathBorderBox borderBox = new MathBorderBox(new MathematicalText("x"));
borderBox.StrikethroughBottomLeftToTopRight = true;
```

## MathBorderBox.StrikethroughTopLeftToBottomRight प्रॉपर्टी

Strikethrough Top-Left to Bottom-Right (default is false). बॉर्डर बॉक्स के बाएँ-ऊपर कोने से दाएँ-नीचे कोने तक एक विकर्ण स्ट्राइकथ्रू रेखा की छिपी या दिखाई देने वाली स्थिति को निर्दिष्ट करता है।

```csharp
public bool StrikethroughTopLeftToBottomRight { get; set; }
```

### उदाहरण

उदाहरण:

```csharp
[C#]
MathBorderBox borderBox = new MathBorderBox(new MathematicalText("x"));
borderBox.StrikethroughTopLeftToBottomRight = true;
```

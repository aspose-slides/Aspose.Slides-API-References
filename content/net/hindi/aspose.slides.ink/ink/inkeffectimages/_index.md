## Ink.InkEffectImages प्रॉपर्टी

इंक ब्रश के लिए दृश्य प्रभावों का अनुकरण करने हेतु उपयोग की जाने कस्टम छवियों का संग्रह प्राप्त करता है। इन छवियों का उपयोग विशिष्ट [`InkEffectType`](../../inkeffecttype) मानों के साथ इंक रेंडर करने पर किया जाता है, जैसे Galaxy, Rainbow आदि। अपनी खुद की छवियों को प्रदान करके आप प्रत्येक इंक प्रभाव को कैसे प्रदर्शित किया जाए, इसे नियंत्रित कर सकते हैं।

```csharp
public static IDictionary<InkEffectType, IImage> InkEffectImages { get; }
```

### टिप्पणियाँ

यह प्रॉपर्टी डिफ़ॉल्ट इंक इफ़ेक्ट बनावटों को उपयोगकर्ता-परिभाषित बनावटों से बदलने की अनुमति देती है, जो विशेष रूप से तब उपयोगी होती है जब लाइसेंसिंग के कारण डिफ़ॉल्ट एसेट प्रतिबंधित हों या रन-टाइम पर उपलब्ध न हों। शब्दकोश में प्रत्येक प्रविष्टि को एक [`InkEffectType`](../../inkeffecttype) मान को संबंधित [`IImage`](../../../aspose.slides/iimage) ऑब्जेक्ट (जैसे Bitmap, या कोई Aspose इमेज इंटरफ़ेस) के साथ जोड़ना आवश्यक है।

### उदाहरण

```csharp
[C#]
IImage image = Images.FromFile("image.png");
Ink.InkEffectImages.Add(InkEffectType.Galaxy, image);
```

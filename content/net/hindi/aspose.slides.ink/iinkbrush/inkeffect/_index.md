## IInkBrush.InkEffect प्रॉपर्टी

इंक स्ट्रोक की दृश्य शैली को परिभाषित करने वाले इंक इफ़ेक्ट प्रकार को प्राप्त करता है (जैसे, Galaxy, Gold, Silver). यह मान ब्रश प्रॉपर्टी "inkEffects" से पार्स किया जाता है। यदि कोई मान्य इफ़ेक्ट निर्दिष्ट नहीं किया गया है, तो NotDefined लौटाया जाता है।

```csharp
public InkEffectType InkEffect { get; }
```

### उदाहरण

उदाहरण:

```csharp
[C#]
using (var pres = new Presentation("Presentation.pptx"))
{
    Ink ink = pres.Slides[0].Shapes[0] as Ink;
    IInkBrush brush = ink.Traces[0].Brush;
    Console.WriteLine("InkEffects = {0}", brush.InkEffect);
}
```

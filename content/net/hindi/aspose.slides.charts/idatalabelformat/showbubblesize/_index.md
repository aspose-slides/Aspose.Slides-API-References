## IDataLabelFormat.ShowBubbleSize प्रॉपर्टी

निर्दिष्ट चार्ट के डेटा लेबल बबल साइज मान प्रदर्शन व्यवहार को दर्शाता है। True बबल साइज मान दिखाता है। False छुपाने के लिए। पढ़ने/लिखने योग्य Boolean.

```csharp
public bool ShowBubbleSize { get; set; }
```

### टिप्पणियाँ

यदि इस DataLabelFormat ऑब्जेक्ट का पैरेंट एक DataLabelCollection डेटा लेबल्स का संग्रह है, तो यह प्रॉपर्टी DataLabelCollection संग्रह में νέे डेटा लेबल्स के लिए ShowBubbleSize प्रॉपर्टी का डिफ़ॉल्ट मान प्राप्त या सेट करती है। इस प्रॉपर्टी को मान के साथ सेट करने से यह मान DataLabelCollection संग्रह में सभी डेटा लेबल्स की ShowBubbleSize प्रॉपर्टी पर भी लागू हो जाता है (उदा. "DataLabels.DefaultDataLabelFormat.ShowBubbleSize = val;" जिससे सभी DataLabels[i].ShowBubbleSize बराबर val हो जाता है)।

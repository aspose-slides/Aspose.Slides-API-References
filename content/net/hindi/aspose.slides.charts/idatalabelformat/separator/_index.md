## IDataLabelFormat.Separator प्रॉपर्टी

डेटा लेबल्स पर चार्ट में उपयोग किए जाने वाले Separator को दर्शाने वाला Variant सेट या लौटाता है। पढ़ने/लिखने योग्य String.

```csharp
public string Separator { get; set; }
```

### टिप्पणी

यदि इस DataLabelFormat ऑब्जेक्ट का पैरेंट एक DataLabelCollection डेटा लेबल्स का संग्रह है, तो यह प्रॉपर्टी DataLabelCollection संग्रह में नए डेटा लेबल्स के लिए Separator प्रॉपर्टी का डिफ़ॉल्ट मान प्राप्त या सेट करती है। इस प्रॉपर्टी को मान के साथ सेट करने से यह मान DataLabelCollection संग्रह में सभी डेटा लेबल्स की Separator प्रॉपर्टी पर भी लागू हो जाता है (जैसे "DataLabels.DefaultDataLabelFormat.Separator = val;" कारण कि सभी DataLabels[i].Separator बराबर val हो जाता है)।

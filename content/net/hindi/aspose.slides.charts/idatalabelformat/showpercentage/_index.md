## IDataLabelFormat.ShowPercentage प्रॉपर्टी

निर्दिष्ट चार्ट के डेटा लेबल प्रतिशत मान प्रदर्शन व्यवहार को दर्शाता है। True प्रतिशत मान को प्रदर्शित करता है। False छुपाने के लिए। पढ़ें/लिखें Boolean.

```csharp
public bool ShowPercentage { get; set; }
```

### टिप्पणियाँ

यदि इस DataLabelFormat वस्तु का पैरेंट एक DataLabelCollection डेटा लेबल संग्रह है तो यह प्रॉपर्टी DataLabelCollection संग्रह में नए डेटा लेबल्स के लिए ShowPercentage प्रॉपर्टी का डिफ़ॉल्ट मान प्राप्त करती या सेट करती है। इस प्रॉपर्टी को मान के साथ सेट करने से यह मान DataLabelCollection संग्रह में सभी डेटा लेबल्स के ShowPercentage प्रॉपर्टी पर भी सेट हो जाता है (i.e. "DataLabels.DefaultDataLabelFormat.ShowPercentage = val;" जिससे सभी DataLabels[i].ShowPercentage बराबर val हो जाता है)।

## IDataLabelFormat.NumberFormat प्रॉपर्टी

DataLabels ऑब्जेक्ट के लिए फॉर्मेट स्ट्रिंग का प्रतिनिधित्व करता है। पढ़ें/लिखें स्ट्रिंग।

```csharp
public string NumberFormat { get; set; }
```

### टिप्पणी

यदि इस DataLabelFormat ऑब्जेक्ट का पैरेंट एक DataLabelCollection डेटा लेबल संग्रह है, तो यह प्रॉपर्टी DataLabelCollection संग्रह में नए डेटा लेबल्स के लिए NumberFormat प्रॉपर्टी का डिफ़ॉल्ट मान प्राप्त करती या सेट करती है। जब इस प्रॉपर्टी को किसी मान से सेट किया जाता है, तो वही मान DataLabelCollection संग्रह के सभी डेटा लेबल्स की NumberFormat प्रॉपर्टी के लिए भी सेट हो जाता है (उदा. "DataLabels.DefaultDataLabelFormat.NumberFormat = val;" सभी DataLabels[i].NumberFormat को val के बराबर कर देता है)।

### उदाहरण

```csharp
[C#]
series.Labels.DefaultDataLabelFormat.ShowValue = true;
series.Labels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = false;
series.Labels.DefaultDataLabelFormat.NumberFormat = "0.0%";
```

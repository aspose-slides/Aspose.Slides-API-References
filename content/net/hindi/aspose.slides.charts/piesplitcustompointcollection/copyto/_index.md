## PieSplitCustomPointCollection.CopyTo मेथड

ICollection के तत्वों को एक Array में कॉपी करता है, जो एक विशिष्ट Array अनुक्रमण से शुरू होता है।

```csharp
public void CopyTo(IChartDataPoint[] array, int arrayIndex)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| array | IChartDataPoint[] | ICollection से कॉपी किए गए तत्वों का गंतव्य एक-आयामी Array है। Array में शून्य-आधारित अनुक्रमण होना चाहिए। |
| arrayIndex | Int32 | *array* में वह शून्य-आधारित अनुक्रमण जहाँ से कॉपी शुरू होती है। |

### अपवाद

| अपवाद | शर्त |
| --- | --- |
| ArgumentNullException | *array* null है। |
| ArgumentOutOfRangeException | *arrayIndex* 0 से कम है। |
| ArgumentException | स्रोत ICollection में तत्वों की संख्या *arrayIndex* से लेकर गंतव्य *array* के अंत तक उपलब्ध स्थान से अधिक है। |

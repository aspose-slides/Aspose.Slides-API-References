## IChartData.Categories प्रॉपर्टी

प्राथमिक श्रेणियों को प्राप्त करता है (या यदि [`UseSecondaryCategories`](../usesecondarycategories) प्रॉपर्टी false है तो दोनों प्राथमिक और द्वितीयक श्रेणियों को)। केवल-पढ़ने योग्य [`IChartCategoryCollection`](../../ichartcategorycollection)।

```csharp
public IChartCategoryCollection Categories { get; }
```

### टिप्पणियाँ

यदि [`UseSecondaryCategories`](../usesecondarycategories) प्रॉपर्टी false है तो [`SecondaryCategories`](../secondarycategories) प्रॉपर्टी null लौटाती है और इस `Categories` प्रॉपर्टी में डेटा दोनों प्राथमिक और द्वितीयक श्रृंखलाओं के लिए उपयोग किया जाता है। यदि [`UseSecondaryCategories`](../usesecondarycategories) प्रॉपर्टी true है तो [`SecondaryCategories`](../secondarycategories) प्रॉपर्टी में डेटा द्वितीयक श्रृंखलाओं के लिए उपयोग किया जाता है और इस `Categories` प्रॉपर्टी में डेटा प्राथमिक श्रृंखलाओं के लिए उपयोग किया जाता है।

### उदाहरण

उदाहरण। कौन सी श्रेणियाँ श्रृंखला से संबंधित हैं - ChartData.Categories या ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // संबंधित श्रेणियाँ हैं series.Chart.ChartData.SecondaryCategories
}
else
{
    // संबंधित श्रेणियाँ हैं series.Chart.ChartData.Categories
}
```

## IDataLabelCollection.Hide विधि

सभी Show*-फ़्लैग (ShowValue, ...) को false स्थिति पर सेट करके डेटा लेबल को डिफ़ॉल्ट रूप से छिपा दें। इसके बाद IsVisible false होगा।

```csharp
public void Hide()
```

### टिप्पणियाँ

यदि डेटा लेबल डिफ़ॉल्ट रूप से दिखाई नहीं दे रहा है (IsVisible false है) तो आप Show*-फ़्लैग (ShowValue, ...) को true स्थिति पर सेट करके डेटा लेबल को “डिफ़ॉल्ट रूप से दिखाने योग्य” बना सकते हैं।

## IDataLabelCollection.IsVisible प्रॉपर्टी

False का अर्थ है कि डेटा लेबल डिफॉल्ट रूप से दृश्यमान नहीं है (और इसलिए DefaultDataLabelFormat प्रॉपर्टी के सभी Show*-flags (ShowValue, ...) false होते हैं)। केवल-पढ़ने योग्य बूलियन।

```csharp
public bool IsVisible { get; }
```

### टिप्पणियां

यदि डेटा लेबल डिफॉल्ट रूप से दृश्यमान है तो आप इसे Hide() मेथड से डिफॉल्ट रूप से छिपा सकते हैं। लेकिन यदि डेटा लेबल डिफॉल्ट रूप से दृश्यमान नहीं है (IsVisible false है) तो आप DefaultDataLabelFormat प्रॉपर्टी के Show*-flags (ShowValue, ...) को true स्थिति में सेट करके डेटा लेबल को “डिफॉल्ट रूप से दृश्यमान” बना सकते हैं।

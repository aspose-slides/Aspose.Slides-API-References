## IChartSeriesGroup.Overlap प्रॉपर्टी

यह निर्दिष्ट करता है कि 2-D चार्ट में बार और कॉलम कितनी प्रतिशत ओवरलैप करेंगे ( -100% से 100% तक)।

- -100%: अधिकतम अंतर (बार पूरी तरह से अलग हैं)।
- 0%: बार बगल-बगल रखे जाते हैं बिना ओवरलैप या अंतर के।
- 100%: अधिकतम ओवरलैप (बार पूरी तरह से एक-दूसरे पर ओवरलैप होते हैं)।

यह प्रॉपर्टी पढ़ें/लिखें SByte है।

```csharp
public sbyte Overlap { get; set; }
```

### अपवाद

| अपवाद | शर्त |
| --- | --- |
| ArgumentOutOfRangeException | जब मान को -100 से 100 के वैध सीमा के बाहर सेट किया जाता है तो यह फेंका जाता है। |

### उदाहरण

निम्नलिखित उदाहरण दर्शाता है कि चार्ट सीरीज़ ग्रुप के लिए ओवरलैप कैसे सेट करें और फ़ॉर्म पर परिणामी चार्ट को रेंडर करें:

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // ओवरलैप को 55% पर सेट करें

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

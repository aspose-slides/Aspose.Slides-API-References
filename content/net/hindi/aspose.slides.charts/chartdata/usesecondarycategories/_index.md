## ChartData.UseSecondaryCategories प्रॉपर्टी

यदि false हो तो [`SecondaryCategories`](../secondarycategories) प्रॉपर्टी null लौटाएगी और [`Categories`](../categories) प्रॉपर्टी में डेटा प्राथमिक और द्वितीयक श्रृंखला दोनों के लिए उपयोग किया जाता है। यदि true हो तो [`SecondaryCategories`](../secondarycategories) प्रॉपर्टी में डेटा द्वितीयक श्रृंखला के लिए और [`Categories`](../categories) प्रॉपर्टी में डेटा प्राथमिक श्रृंखला के लिए उपयोग किया जाता है। पढ़ें/लिखें Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

### उदाहरण

उदाहरण। श्रृंखला से संबंधित श्रेणियां कौन सी हैं - ChartData.Categories या ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // संबंधित श्रेणियां हैं series.Chart.ChartData.SecondaryCategories
}
else
{
    // संबंधित श्रेणियां हैं series.Chart.ChartData.Categories
}
```

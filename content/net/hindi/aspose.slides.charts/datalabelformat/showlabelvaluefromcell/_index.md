## DataLabelFormat.ShowLabelValueFromCell प्रॉपर्टी

निर्दिष्ट चार्ट के डेटा लेबल सेल मान प्रदर्शन व्यवहार का प्रतिनिधित्व करता है। True सेल मान दिखाता है। False इसे छिपाता है। पढ़ने/लिखने Boolean।

```csharp
public bool ShowLabelValueFromCell { get; set; }
```

### टिप्पणियाँ

यदि इस DataLabelFormat ऑब्जेक्ट का पैरेंट डेटा लेबल्स का DataLabelCollection संग्रह है, तो यह प्रॉपर्टी नई डेटा लेबल्स के लिए DataLabelCollection संग्रह में ShowLabelValueFromCell प्रॉपर्टी का डिफ़ॉल्ट मान प्राप्त करती या सेट करती है। यह प्रॉपर्टी मान के साथ सेट करने पर सभी डेटा लेबल्स के DataLabelCollection संग्रह में ShowLabelValueFromCell प्रॉपर्टी के लिए भी यह मान सेट करती है (उदाहरण के लिए "DataLabels.DefaultDataLabelFormat.ShowLabelValueFromCell = val;" जिससे सभी DataLabels[i].ShowLabelValueFromCell बराबर val हो जाता है)।

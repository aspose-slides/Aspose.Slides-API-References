## DataLabelFormat.ShowBubbleSize प्रॉपर्टी

निर्दिष्ट चार्ट के डेटा लेबल बबल आकार मूल्य प्रदर्शन व्यवहार को दर्शाता है। True बबल आकार मूल्य प्रदर्शित करता है। False छिपाने के लिए। पढ़ें/लिखें Boolean.

```csharp
public bool ShowBubbleSize { get; set; }
```

### टिप्पणी

यदि इस DataLabelFormat ऑब्जेक्ट का पैरेंट DataLabelCollection डेटा लेबल्स का संग्रह है तो यह प्रॉपर्टी DataLabelCollection संग्रह में नए डेटा लेबल्स के लिए ShowBubbleSize प्रॉपर्टी का डिफ़ॉल्ट मान प्राप्त या सेट करती है। इस प्रॉपर्टी को मान से सेट करना सभी डेटा लेबल्स में ShowBubbleSize प्रॉपर्टी को उसी मान पर सेट करता है (उदा. "DataLabels.DefaultDataLabelFormat.ShowBubbleSize = val;" जिसके कारण सभी DataLabels[i].ShowBubbleSize बराबर val हो जाता है)।

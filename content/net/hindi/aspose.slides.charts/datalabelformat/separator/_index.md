## DataLabelFormat.Separator प्रॉपर्टी

डेटा लेबल्स पर चार्ट में उपयोग किए जाने वाले विभाजक को दर्शाने वाला Variant सेट करता है या लौटाता है। पढ़ें/लिखें String.

```csharp
public string Separator { get; set; }
```

### टिप्पणियाँ

यदि इस DataLabelFormat वस्तु का पैरेंट एक DataLabelCollection डेटा लेबल्स का कलेक्शन है, तो यह प्रॉपर्टी नए डेटा लेबल्स के लिए DataLabelCollection कलेक्शन में Separator प्रॉपर्टी का डिफ़ॉल्ट मान प्राप्त करती है या सेट करती है। इस प्रॉपर्टी को मान के साथ सेट करने से यह मान सभी डेटा लेबल्स के लिए DataLabelCollection कलेक्शन में Separator प्रॉपर्टी पर भी सेट हो जाता है (उदाहरण के लिए "DataLabels.DefaultDataLabelFormat.Separator = val;" जिससे सभी DataLabels[i].Separator बराबर val हो जाता है)।

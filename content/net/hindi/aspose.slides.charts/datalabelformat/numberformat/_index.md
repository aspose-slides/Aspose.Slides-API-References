## DataLabelFormat.NumberFormat प्रॉपर्टी

डेटा लेबल्स ऑब्जेक्ट के लिए फ़ॉर्मेट स्ट्रिंग का प्रतिनिधित्व करता है। पढ़ें/लिखें String.

```csharp
public string NumberFormat { get; set; }
```

### टिप्पणियाँ

यदि इस DataLabelFormat ऑब्जेक्ट का पैरेंट DataLabelCollection डेटा लेबल्स का संग्रह है, तो यह प्रॉपर्टी DataLabelCollection संग्रह में नए डेटा लेबल्स के लिए NumberFormat प्रॉपर्टी का डिफ़ॉल्ट मान प्राप्त करती है या सेट करती है। जब इस प्रॉपर्टी को किसी मान से सेट किया जाता है, तो वह मान DataLabelCollection संग्रह में सभी डेटा लेबल्स के लिए NumberFormat प्रॉपर्टी पर भी सेट हो जाता है (उदाहरण के लिए "DataLabels.DefaultDataLabelFormat.NumberFormat = val;" सभी DataLabels[i].NumberFormat को val के बराबर कर देता है)।

### उदाहरण

```csharp
[C#]
series.Labels.DefaultDataLabelFormat.ShowValue = true;
series.Labels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = false;
series.Labels.DefaultDataLabelFormat.NumberFormat = "0.0%";
```

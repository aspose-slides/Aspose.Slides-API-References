## DataLabel.IsVisible प्रॉपर्टी

False का अर्थ है कि डेटा लेबल दिखाई नहीं देता (और इसलिए सभी Show*-flags (ShowValue, ...) false होते हैं)। केवल-पढ़ने-योग्य Boolean।

```csharp
public bool IsVisible { get; }
```

### टिप्पणियाँ

यदि डेटा लेबल दिखाई देता है तो आप इसे Hide() विधि से छिपा सकते हैं। लेकिन यदि डेटा लेबल नहीं दिखता (IsVisible false है) तो आप Show*-flags (ShowValue, ...) को true स्थिति में सेट करके डेटा लेबल को दिखाई दे सके हैं।

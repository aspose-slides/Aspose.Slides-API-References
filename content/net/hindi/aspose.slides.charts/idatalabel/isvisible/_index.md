## IDataLabel.IsVisible प्रॉपर्टी

False का अर्थ है कि डेटा लेबल दृश्यमान नहीं है (और इस प्रकार सभी Show*-flags (ShowValue, ...) false होते हैं)। केवल-पढ़ने योग्य Boolean.

```csharp
public bool IsVisible { get; }
```

### टिप्पणी

यदि डेटा लेबल दृश्यमान है तो आप इसे Hide() मेथड से छुपा सकते हैं। लेकिन यदि डेटा लेबल दृश्यमान नहीं है (IsVisible false है) तो आप Show*-flags (ShowValue, ...) को true स्थिति में सेट करके डेटा लेबल को दृश्यमान बना सकते हैं।

## BehaviorCollection.CopyTo विधि

ICollection के तत्वों को एक Array में कॉपी करता है, जो एक विशिष्ट Array इंडेक्स से शुरू होता है।

```csharp
public void CopyTo(IBehavior[] array, int arrayIndex)
```

| पैरामीटर | टाइप | विवरण |
| --- | --- | --- |
| array | IBehavior[] | वह एक-आयामी Array जो ICollection से कॉपी किए गए तत्वों का गंतव्य है। Array में शून्य-आधारित अनुक्रमण होना चाहिए। |
| arrayIndex | Int32 | *array* में वह शून्य-आधारित अनुक्रमण जहाँ कॉपी करना शुरू होता है। |

### अपवाद

| exception | condition |
| --- | --- |
| ArgumentNullException | *array* null है। |
| ArgumentOutOfRangeException | *arrayIndex* 0 से कम है। |
| ArgumentException | स्रोत ICollection में तत्वों की संख्या *arrayIndex* से लक्ष्य *array* के अंत तक उपलब्ध स्थान से अधिक है। |

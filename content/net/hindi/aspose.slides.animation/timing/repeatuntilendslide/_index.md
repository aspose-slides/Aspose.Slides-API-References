## Timing.RepeatUntilEndSlide प्रॉपर्टी

यह गुण विशेषता निर्धारित करती है कि प्रभाव स्लाइड के अंत तक दोहराया जाएगा या नहीं। पढ़ें/लिखें Boolean.

```csharp
public bool RepeatUntilEndSlide { get; set; }
```

### उदाहरण

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // पहले स्लाइड के लिए इफ़ेक्ट्स सीक्वेंस प्राप्त करें
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // मुख्य क्रम के पहले इफ़ेक्ट को प्राप्त करें।
    IEffect effect = effectsSequence[0];

    // इफ़ेक्ट की टाइमिंग/रिपीट को "स्लाइड के अंत तक" बदलें
    effect.Timing.RepeatUntilEndSlide = true;
}
```

## Timing.Rewind प्रॉपर्टी

यह विशेषता निर्दिष्ट करती है कि प्रभाव समाप्त होने पर पुनः चलाया जाएगा या नहीं। पढ़ें/लिखें Boolean.

```csharp
public bool Rewind { get; set; }
```

### उदाहरण

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // पहले स्लाइड के लिए इफ़ेक्ट्स क्रम प्राप्त करें
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // मुख्य क्रम का पहला इफ़ेक्ट प्राप्त करें।
    IEffect effect = effectsSequence[0];

    // इफ़ेक्ट का Timing/Rewind सक्रिय करें।
    effect.Timing.Rewind = true;
}
```

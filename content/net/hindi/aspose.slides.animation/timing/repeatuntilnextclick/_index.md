## Timing.RepeatUntilNextClick प्रॉपर्टी

This attribute specifies if the effect will repeat until the next click. पढ़ें/लिखें Boolean.

```csharp
public bool RepeatUntilNextClick { get; set; }
```

### उदाहरण

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // पहले स्लाइड के लिए इफ़ेक्ट्स अनुक्रम प्राप्त करें
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // मुख्य अनुक्रम का पहला इफ़ेक्ट प्राप्त करें।
    IEffect effect = effectsSequence[0];

    // प्रभाव Timing/Repeat को "Until Next Click" पर बदलें
    effect.Timing.RepeatUntilNextClick = true;
}
```

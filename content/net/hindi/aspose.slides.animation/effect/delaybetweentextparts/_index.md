## Effect.DelayBetweenTextParts प्रॉपर्टी

एनिमेटेड टेक्स्ट भागों (words or letters) के बीच देरी को परिभाषित करता है। एक सकारात्मक मान प्रभाव की अवधि का प्रतिशत निर्दिष्ट करता है। एक नकारात्मक मान सेकंड में देरी निर्दिष्ट करता है। पढ़ें/लिखें Single.

```csharp
public float DelayBetweenTextParts { get; set; }
```

### उदाहरण

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // पहले स्लाइड के पहले इफ़ेक्ट को प्राप्त करें।
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // इफ़ेक्ट Animate टेक्स्ट प्रकार को "By word" में बदलें
    firstSlideEffect.AnimateTextType = AnimateTextType.ByWord;
    
    // एनिमेटेड टेक्स्ट भागों के बीच देरी को प्रभाव अवधि के 20% पर सेट करें।
    firstSlideEffect.DelayBetweenTextParts = 20f;
}
```

## ITiming.RepeatUntilEndSlide गुण

यह गुण निर्धारित करता है कि प्रभाव स्लाइड के अंत तक दोहराया जाएगा या नहीं। पढ़ें/लिखें Boolean.

```csharp
public bool RepeatUntilEndSlide { get; set; }
```

### उदाहरण

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // पहले स्लाइड के लिए इफ़ेक्ट्स क्रम प्राप्त करें
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // मुख्य क्रम का पहला इफ़ेक्ट प्राप्त करें।
    IEffect effect = effectsSequence[0];

    // इफ़ेक्ट टाइमिंग/रीपीट को "स्लाइड के अंत तक" बदलें
    effect.Timing.RepeatUntilEndSlide = true;
}
```

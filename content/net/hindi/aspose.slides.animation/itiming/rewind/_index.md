## ITiming.Rewind प्रॉपर्टी

यह एट्रिब्यूट निर्दिष्ट करता है कि प्रभाव समाप्त होने पर रीवाइंड होगा या नहीं। पढ़ें/लिखें Boolean.

```csharp
public bool Rewind { get; set; }
```

### उदाहरण

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // पहले स्लाइड के लिए इफ़ेक्ट अनुक्रम प्राप्त करें
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // मुख्य अनुक्रम का पहला इफ़ेक्ट प्राप्त करें।
    IEffect effect = effectsSequence[0];

    // इफ़ेक्ट Timing/Rewind को चालू करें।
    effect.Timing.Rewind = true;
}
```

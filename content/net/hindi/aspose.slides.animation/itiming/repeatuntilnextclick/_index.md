## ITTiming.RepeatUntilNextClick गुण

यह विशेषता निर्धारित करती है कि प्रभाव अगली क्लिक तक दोहराया जाएगा या नहीं। पढ़ें/लिखें Boolean.

```csharp
public bool RepeatUntilNextClick { get; set; }
```

### उदाहरण

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // पहले स्लाइड के लिए इफ़ेक्ट सीक्वेंस प्राप्त करें
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // मुख्य सीक्वेंस का पहला इफ़ेक्ट प्राप्त करें।
    IEffect effect = effectsSequence[0];

    // इफ़ेक्ट के Timing/Repeat को "Until Next Click" में बदलें
    effect.Timing.RepeatUntilNextClick = true;
}
```

## IEffect.DelayBetweenTextParts प्रॉपर्टी

एनिमेटेड टेक्स्ट हिस्सों (शब्द या अक्षर) के बीच विलंब को परिभाषित करता है। एक सकारात्मक मान प्रभाव की अवधि का प्रतिशत निर्दिष्ट करता है। एक नकारात्मक मान सेकंड में विलंब निर्दिष्ट करता है। रीड/राइट Single.

```csharp
public float DelayBetweenTextParts { get; set; }
```

### उदाहरण

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // पहले स्लाइड का पहला इफ़ेक्ट प्राप्त करें।
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // इफ़ेक्ट के एनीमेट टेक्स्ट प्रकार को "By word" में बदलें
    firstSlideEffect.AnimateTextType = AnimateTextType.ByWord;
    
    // एनिमेटेड टेक्स्ट भागों के बीच विलंब को प्रभाव की अवधि के 20% पर सेट करें।
    firstSlideEffect.DelayBetweenTextParts = 20f;
}
```

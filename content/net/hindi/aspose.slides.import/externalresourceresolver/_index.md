## ExternalResourceResolver क्लास

Callback क्लास का उपयोग Html, Svg दस्तावेज़ आयात के दौरान बाहरी संसाधनों को हल करने के लिए किया जाता है। इस रेज़ॉल्वर का उपयोग करने से सुरक्षा जोखिम पैदा हो सकता है जब क्लाइंट द्वारा प्रदान किया गया HTML या SVG फ़ाइल सर्वर सॉफ़्टवेयर को स्थानीय या नेटवर्क फ़ाइल प्राप्त करने के लिए प्रेरित करती है। सावधानी से उपयोग करें। यह अनुशंसा की जाती है कि ExternalResourceResolver को बिल्कुल भी निर्दिष्ट न किया जाए (केवल एंबेडेड ऑब्जेक्ट पढ़े जाएंगे) या कोई उपवर्ग बनाया जाए जो जाँचता हो कि निर्दिष्ट uri वैध है या नहीं।

```csharp
public class ExternalResourceResolver : IExternalResourceResolver
```

## Shape.Rotation eigenschap

Retourneert of stelt het aantal graden in waarop de opgegeven vorm rond de z-as is geroteerd. Een positieve waarde geeft een rotatie met de klok mee aan; een negatieve waarde geeft een rotatie tegen de klok in aan. Lezen/schrijven Single.

```csharp
public float Rotation { get; set; }
```

### Opmerkingen

De geretourneerde waarde is altijd gedefinieerd (is niet float.NaN). Toegewezen waarde moet gedefinieerd zijn (niet float.NaN). U kunt ongedefinieerde waarden instellen voor RawFrame instance properties.

## ILoadOptions.DeleteEmbeddedBinaryObjects eigenschap

Bepaalt of Aspose.Slides alle ingesloten binaire objecten verwijdert tijdens het laden van een presentatie.

De types van de ingesloten binaire objecten:

* VBA Project [`VbaProject`](../../ipresentation/vbaproject)
* OLE Object ingesloten gegevens [`EmbeddedFileData`](../../ioleembeddeddatainfo/embeddedfiledata)
* ActiveX Control binaire gegevens [`ActiveXControlBinary`](../../icontrol/activexcontrolbinary)

Lezen/Schrijven Boolean.

```csharp
public bool DeleteEmbeddedBinaryObjects { get; set; }
```

### Opmerkingen

Standaard is **false**.

### Voorbeelden

Het volgende voorbeeld toont hoe de presentatie kan worden geladen zonder enige ingesloten binaire objecten.

```csharp
[C#]
LoadOptions loadOptions = new LoadOptions();
loadOptions.DeleteEmbeddedBinaryObjects = true;
using (Presentation pres = new Presentation("pres.ppt", loadOptions))
{
    pres.Save("output_WithoutBinaryObjects.ppt", SaveFormat.Ppt);
}
```

## ICommentCollection.AddModernComment method

Voeg een nieuwe moderne opmerking toe aan het einde van een collectie.

```csharp
public IModernComment AddModernComment(string text, ISlide slide, IShape shape, PointF position, 
    DateTime creationTime)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| text | String | Platte tekst van een nieuwe moderne opmerking. |
| slide | ISlide | Dia in een presentatie waarin een nieuwe moderne opmerking wordt toegevoegd. |
| shape | IShape | Vorm op een dia waaraan een nieuwe moderne opmerking is gekoppeld. |
| position | PointF | Positie op een dia waar een nieuwe moderne opmerking moet worden toegevoegd. |
| creationTime | DateTime | Tijdstip van het aanmaken van de moderne opmerking. |

### Retourwaarde

Toegevoegde moderne opmerking.

### Voorbeelden

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ICommentAuthor newAuthor = pres.CommentAuthors.AddAuthor("Some Author", "SA");
    newAuthor.Comments.AddModernComment("This is modern comment", pres.Slides[0], null, new PointF(100, 100), DateTime.Now);

    pres.Save(outPptxFileName, SaveFormat.Pptx);
}
```

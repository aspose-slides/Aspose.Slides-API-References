## IBlobManagementOptions.PresentationLockingBehavior eigenschap

Deze eigenschap definieert of een exemplaar van de Presentation-klasse een eigenaar kan zijn van de bron – bestand of stream gedurende de levensduur van het exemplaar. Als het exemplaar een eigenaar is, vergrendelt het de bron. Dit helpt het geheugenverbruik en de prestaties te verbeteren bij het werken met BLOB's, maar de bron (stream of bestand) kan niet worden gewijzigd tijdens de levensduur van het Presentation-exemplaar. Dit is een voorbeeld:

```csharp
public PresentationLockingBehavior PresentationLockingBehavior { get; set; }
```

### Voorbeelden

```csharp
LoadOptions loadOptions = new LoadOptions { 
  BlobManagementOptions = {PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked} };
using (Presentation pres = new Presentation("pres.pptx", loadOptions)) {
  // IOException zal worden gegooid omdat pres.pptx vergrendeld is voor de levensduur van een Presentation
  // File.Delete("pres.pptx");
}
 // nadat het Presentation-object is verwijderd, is het bestand ontgrendeld en kan het worden verwijderd
File.Delete("pres.pptx");
```

## ViewProperties.GridSpacing eigenschap

Geeft de rasterafstand terug of stelt deze in die moet worden gebruikt voor het raster onderliggend aan het presentatiedocument, in punten. Lezen/Schrijven Single.

```csharp
public float GridSpacing { get; set; }
```

### Opmerkingen

De rasterafstand moet een positief getal zijn . Het typische waardebereik is van 1 mm (2.8349607 points) tot 2 inch (144 points).

### Voorbeelden

De volgende voorbeeldcode laat zien hoe je de rasterafstand in een PowerPoint-presentatie wijzigt.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

## PresentationLockingBehavior enumeratie

Geeft het gedrag weer met betrekking tot de behandeling van de [`IPresentation`](../ipresentation) bron (bestand of Stream) tijdens het laden en werken met een exemplaar van [`IPresentation`](../ipresentation).

```csharp
public enum PresentationLockingBehavior
```

### Waarden

| Naam | Waarde | Beschrijving |
| --- | --- | --- |
| LoadAndRelease | `0` | De bron wordt alleen vergrendeld gedurende de uitvoering van de [`IPresentation`](../ipresentation) constructor. Als [`IsTemporaryFilesAllowed`](../iblobmanagementoptions/istemporaryfilesallowed) onwaar (false) is, worden alle BLOB's in het geheugen geladen. Anders kunnen andere middelen, zoals tijdelijke bestanden, worden gebruikt. Dit gedrag is langzamer dan KeepLocked, en als het mogelijk is de eigendom van de bron over te dragen aan [`IPresentation`](../ipresentation), wordt aanbevolen KeepLocked te gebruiken. |
| KeepLocked | `1` | De bron wordt vergrendeld voor de volledige levensduur van het [`IPresentation`](../ipresentation) exemplaar, totdat het wordt verwijderd. [`IsTemporaryFilesAllowed`](../iblobmanagementoptions/istemporaryfilesallowed) moet op true worden gezet om dit gedrag te gebruiken, anders wordt er een uitzondering gegooid. Dit gedrag wordt aanbevolen, het is sneller en verbruikt minder geheugen dan LoadAndRelease. |

### Opmerkingen

De bron is de parameter die wordt doorgegeven aan de [`IPresentation`](../ipresentation) constructor. In het onderstaande voorbeeld is de bron het bestand "pres.pptx":

```csharp
LoadOptions loadOptions = new LoadOptions { 
  BlobManagementOptions = { PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked } };
using (IPresentation pres = new Presentation("pres.pptx", loadOptions)) { }
```

Voor dit voorbeeld wordt de bron (bestand "pres.pptx") vergrendeld voor de levensduur van een [`IPresentation`](../ipresentation) exemplaar, d.w.z. kan niet worden gewijzigd of verwijderd door een ander proces.

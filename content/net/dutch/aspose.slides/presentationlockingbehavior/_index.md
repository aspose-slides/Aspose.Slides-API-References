## PresentationLockingBehavior enumeratie

Stelt het gedrag voor met betrekking tot het behandelen van de [`IPresentation`](../ipresentation) bron (file of Stream) tijdens het laden en werken met een instantie van [`IPresentation`](../ipresentation).

```csharp
public enum PresentationLockingBehavior
```

### Waarden

| Naam | Waarde | Beschrijving |
| --- | --- | --- |
| LoadAndRelease | `0` | De bron wordt alleen vergrendeld gedurende de uitvoering van de [`IPresentation`](../ipresentation) constructor. Als [`IsTemporaryFilesAllowed`](../iblobmanagementoptions/istemporaryfilesallowed) op false is ingesteld, worden alle BLOB's in het geheugen geladen. Anders kunnen andere middelen, zoals tijdelijke bestanden, worden gebruikt. Dit gedrag is langzamer dan KeepLocked, en als het mogelijk is de eigendom van de bron over te dragen aan [`IPresentation`](../ipresentation), wordt aanbevolen KeepLocked te gebruiken. |
| KeepLocked | `1` | De bron wordt vergrendeld voor de volledige levensduur van de [`IPresentation`](../ipresentation) instantie, totdat deze wordt vrijgegeven. [`IsTemporaryFilesAllowed`](../iblobmanagementoptions/istemporaryfilesallowed) moet op true worden ingesteld om dit gedrag te gebruiken, anders wordt een uitzondering gegooid. Dit gedrag wordt aanbevolen, het is sneller en verbruikt minder geheugen dan LoadAndRelease. |

### Opmerkingen

De bron is de parameter die wordt doorgegeven aan de [`IPresentation`](../ipresentation) constructor. In het onderstaande voorbeeld is de bron het bestand "pres.pptx":

```csharp
LoadOptions loadOptions = new LoadOptions { 
  BlobManagementOptions = { PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked } };
using (IPresentation pres = new Presentation("pres.pptx", loadOptions)) { }
```

Voor dit voorbeeld wordt de bron ("pres.pptx" bestand) vergrendeld voor de levensduur van de [`IPresentation`](../ipresentation) instantie, d.w.z. kan deze niet worden gewijzigd of verwijderd door een ander proces.

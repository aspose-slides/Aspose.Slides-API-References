## ISvgImage.WriteAsEmf methode

Slaat de SVG-afbeelding op als een EMF-bestand.


| Parameter | Type | Beschrijving |

### Voorbeelden

Het volgende voorbeeld toont hoe u de SVG-afbeelding opslaat in een metafile.

```csharp
[C#]
// Maakt de nieuwe SVG-afbeelding
ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));

// Slaat de SVG-afbeelding op als een metafile
using (var fileStream = System.IO.File.OpenWrite("SvgAsEmf.emf"))
    svgImage.WriteAsEmf(fileStream);
```

Dit voorbeeld toont hoe u de SVG-afbeelding als een metafile toevoegt aan de afbeeldingverzameling van de presentatie.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // Maakt de nieuwe SVG-afbeelding
    ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));
    using (var memStream = new MemoryStream())
    {
        // Slaat de SVG-afbeelding op als een metafile
        svgImage.WriteAsEmf(memStream);
        // Voegt metafile toe aan de afbeeldingverzameling
        pres.Images.AddImage(memStream.ToArray());
    }
}
```

Kopieert de elementen van de ICollection naar een Array, beginnend op een bepaalde Array-index.


| Parameter | Type | Beschrijving |
| --- | --- | --- |
| array | IPortion[] | De één-dimensionale Array die de bestemming is van de elementen die gekopieerd zijn vanuit ICollection. De Array moet nul-gebaseerde indexering hebben. |

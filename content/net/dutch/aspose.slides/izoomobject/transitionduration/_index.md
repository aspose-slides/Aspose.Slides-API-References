## IZoomObject.TransitionDuration eigenschap

Haalt of stelt de duur van de overgang tussen Zoom en dia in. Lezen/schrijven Single. Standaardwaarde: 1.0f

```csharp
public float TransitionDuration { get; set; }
```

### Opmerkingen

Indien niet gespecificeerd (TransitionDur = 0), wordt de overgang van de bestemmingsdia en de bijbehorende timing gebruikt.

### Voorbeelden

Het voorbeeld toont hoe de duur van de overgang tussen Zoom en dia wordt aangepast:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
    zoomFrame.TransitionDuration = 2.5f;
}
```

## TextFrame.SplitTextByColumns methode

Splitst de tekstinhoud van de [`ITextFrame`](../../itextframe) in een array van strings, waarbij elk element overeenkomt met een aparte tekstkolom binnen het frame.

```csharp
public string[] SplitTextByColumns()
```

### Retourwaarde

Een array van strings, waarbij elke string de tekstinhoud van een specifieke kolom in de [`ITextFrame`](../../itextframe) vertegenwoordigt.

### Opmerkingen

Als het tekstraam geen meerdere kolommen bevat, zal de geretourneerde array één element bevatten met de volledige tekst. Lege kolommen worden in de array weergegeven als lege strings.

### Voorbeelden

Het volgende voorbeeld laat zien hoe `SplitTextByColumns` te gebruiken:

```csharp
using (Presentation pres = new Presentation("example.pptx"))
{
    // Haal de eerste vorm op de dia op en cast deze naar ITextFrame
    ITextFrame textFrame = pres.Slides[0].Shapes[0] as ITextFrame;
    // Splits de tekstframe-inhoud in kolommen
    string[] columnsText = textFrame.SplitTextByColumns();
    // Print de tekst van elke kolom naar de console
    foreach (string column in columnsText)
        Console.WriteLine(column);
}
```

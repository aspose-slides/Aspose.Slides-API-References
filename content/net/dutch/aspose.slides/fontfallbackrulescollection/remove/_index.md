## FontFallBackRulesCollection.Remove methode

Verwijdert de eerste voorkoming van een specifieke FallBack-regel uit de collectie.

```csharp
public void Remove(IFontFallBackRule targetRule)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| targetRule | IFontFallBackRule | De regel die uit de collectie moet worden verwijderd. |

### Voorbeelden

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    //Lezen van lege of vooraf geïnitialiseerde regelscollectie van FontsManager
    IFontFallBackRulesCollection rulesList = pres.FontsManager.FontFallBackRulesCollection;

    //Toevoegen van meerdere regels aan de collectie
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));
    rulesList.Add(new FontFallBackRule(0x3040, 0x309F, "MS Mincho"));

    //Ophalen van object van de eerste regel in de collectie
    IFontFallBackRule firstRule = rulesList[0];
    //Verwijderen 
    rulesList.Remove (firstRule);
}
```

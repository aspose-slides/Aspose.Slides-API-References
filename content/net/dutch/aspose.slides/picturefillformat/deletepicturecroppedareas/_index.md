## PictureFillFormat.DeletePictureCroppedAreas methode

Verwijder bijgesneden gebieden van de opvullende Picture.

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### Retourwaarde

Bijgesneden afbeelding of oorspronkelijke afbeelding als bijsnijden niet nodig is.

### Opmerkingen

Deze methode zet WMF/EMF-metabestanden om naar een raster-PNG-afbeelding tijdens het bijsnijden.

### Voorbeelden

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Haalt het PictureFrame op
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // Verwijdert bijgesneden gebieden van de PictureFrame-afbeelding
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

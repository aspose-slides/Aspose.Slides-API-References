## PictureFillFormat.TileOffsetX eigenschap

Retourneert of stelt de horizontale offset van de texture ten opzichte van de oorsprong van de vorm in punten in. Een positieve waarde verplaatst de texture naar rechts, terwijl een negatieve waarde deze naar links verplaatst. Lezen/Schrijven Single.

```csharp
public float TileOffsetX { get; set; }
```

### Voorbeelden

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Haal het picture fill format van de shape op
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Stel de picture fill mode in op Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Stel de horizontale offset van de texture in op 20 punten
    pictureFillFormat.TileOffsetX = 20f;
}
```

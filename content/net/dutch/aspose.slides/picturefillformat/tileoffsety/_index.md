## PictureFillFormat.TileOffsetY eigenschap

Geeft de verticale offset van de textuur ten opzichte van de oorsprong van de vorm terug, of stelt deze in punten in. Een positieve waarde beweegt de textuur naar beneden, terwijl een negatieve waarde deze naar boven beweegt. Lezen/Schrijven Single.

```csharp
public float TileOffsetY { get; set; }
```

### Voorbeelden

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Haalt het picture fill format van de vorm op
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Stelt de picture fill modus in op Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Stelt de verticale offset van de textuur in op -50 punten
    pictureFillFormat.TileOffsetY = -50;
}
```

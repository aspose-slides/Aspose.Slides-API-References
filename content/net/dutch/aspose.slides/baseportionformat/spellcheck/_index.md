## BasePortionFormat.SpellCheck eigenschap

Haalt een waarde op of stelt een waarde in die aangeeft of spellingscontrole is ingeschakeld voor het tekstgedeelte. Wanneer deze eigenschap is ingesteld op false, worden spellingcontroles voor tekstelementen onderdrukt. Wanneer deze op true is ingesteld, is spellingscontrole toegestaan. Standaardwaarde is `false`.

```csharp
public bool SpellCheck { get; set; }
```

### Voorbeelden

Het volgende voorbeeld laat zien hoe de SpellCheck-vlag wordt ingeschakeld vóór het opslaan van de presentatie:

```csharp
[C#]
using (var pres = new Presentation("input.pptx"))
{
    // Toegang tot het eerste tekstgedeelte binnen de eerste vorm op de eerste dia
    var portion = ((AutoShape)pres.Slides[0].Shapes[0]).TextFrame.Paragraphs[0].Portions[0];
    // Schakel spellingscontrole in voor dit tekstgedeelte
    portion.PortionFormat.SpellCheck = true;
    // Sla de gewijzigde presentatie op
    pres.Save("output-with-spellcheck.pptx", SaveFormat.Pptx);
}
```

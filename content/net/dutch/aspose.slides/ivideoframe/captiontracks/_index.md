## IVideoFrame.CaptionTracks eigenschap

Haalt de verzameling van gesloten ondertitels op die aan het audioframe zijn gekoppeld. Deze eigenschap is alleen-lezen en retourneert een [`ICaptionsCollection`](../../icaptionscollection) die alle ondertitelsporen bevat.

```csharp
public ICaptionsCollection CaptionTracks { get; }
```

### Voorbeelden

Voorbeeld:

```csharp
[C#]
using (Presentation pres = new Presentation("video with captions.pptx"))
{
    foreach (IShape shape in pres.Slides[0].Shapes)
    {
        if (!(shape is IVideoFrame videoFrame)) continue;

        foreach (var captionTrack in videoFrame.CaptionTracks)
        {
            // Haalt de binaire ondertitelgegevens op en slaat ze op in het bestand
            System.IO.File.WriteAllBytes(captionTrack.CaptionId + ".vtt", captionTrack.BinaryData);
        }
    }
}
```

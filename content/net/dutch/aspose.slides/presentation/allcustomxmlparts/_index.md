## Presentation.AllCustomXmlParts eigenschap

Retourneert alle aangepaste gegevensonderdelen in de presentatie. Alleen-lezen [`ICustomXmlPart`](../../icustomxmlpart)[].

```csharp
public ICustomXmlPart[] AllCustomXmlParts { get; }
```

### Voorbeelden

De volgende voorbeelden laten zien hoe alle aangepaste xml-onderdelen uit een PowerPoint-presentatie kunnen worden gewist.

```csharp
[C#]
using (Presentation pres = new Presentation("PresentationWithCustomXml.pptx"))
{
	// Itereer over alle aangepaste XML-onderdelen
    foreach (ICustomXmlPart item in pres.AllCustomXmlParts)
    {
        item.Remove();
    }
    pres.Save("out.pptx", SaveFormat.Pptx);
}
```

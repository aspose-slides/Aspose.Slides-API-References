## Presentation.Masters eigenschap

Retourneert een lijst van alle masterslides die zijn gedefinieerd in de presentatie. Alleen-lezen [`IMasterSlideCollection`](../../imasterslidecollection).

### Voorbeelden

De volgende voorbeelden laten zien hoe u afbeeldingen kunt toevoegen aan masterslides van een PowerPoint-presentatie.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ISlide slide = pres.Slides[0];
    IMasterSlide masterSlide = slide.LayoutSlide.MasterSlide;
    IPPImage image = pres.Images.AddImage(File.ReadAllBytes("image.png"));
    masterSlide.Shapes.AddPictureFrame(ShapeType.Rectangle, 10, 10, 100, 100, image);
    pres.Save("pres.pptx", SaveFormat.Pptx);
}
```

De volgende voorbeelden laten zien hoe u de achtergrondkleur van de masterslide van een PowerPoint-presentatie kunt wijzigen.

```csharp
[C#]
// Instantieer de Presentation-klasse die het presentatiebestand vertegenwoordigt
using (Presentation pres = new Presentation())
{
    // Stel de achtergrondkleur van de Master ISlide in op Bosgroen
    pres.Masters[0].Background.Type = BackgroundType.OwnBackground;
    pres.Masters[0].Background.FillFormat.FillType = FillType.Solid;
    pres.Masters[0].Background.FillFormat.SolidFillColor.Color = Color.ForestGreen;
    // Schrijf de presentatie naar schijf
    pres.Save("SetSlideBackgroundMaster_out.pptx", SaveFormat.Pptx);
}
```

De volgende voorbeelden laten zien hoe u een slide-indeling aan een PowerPoint-presentatie kunt toevoegen.

```csharp
[C#]
// Instantieer de Presentation-klasse die het presentatiebestand vertegenwoordigt
using (Presentation presentation = new Presentation("AccessSlides.pptx"))
{
    // Probeer te zoeken op lay-out slidetype
    IMasterLayoutSlideCollection layoutSlides = presentation.Masters[0].LayoutSlides;
    ILayoutSlide layoutSlide = layoutSlides.GetByType(SlideLayoutType.TitleAndObject) ?? layoutSlides.GetByType(SlideLayoutType.Title);
    if (layoutSlide == null)
    {
        // De situatie waarin een presentatie niet bepaalde type lay-outs bevat.
        // presentation bestand bevat alleen Blank en Custom lay-outtypen.
        // Maar lay-outslides met Custom-typen hebben verschillende slide-namen,
        // zoals "Title", "Title and Content", enz. En het is mogelijk deze te gebruiken
        // namen voor lay-outslide-selectie.
        // Ook is het mogelijk de set van placeholder-vormtypen te gebruiken. Bijvoorbeeld,
        // Title-slide zou alleen het Title-placeholder-type moeten hebben, enz.
        foreach (ILayoutSlide titleAndObjectLayoutSlide in layoutSlides)
        {
            if (titleAndObjectLayoutSlide.Name == "Title and Object")
            {
                layoutSlide = titleAndObjectLayoutSlide;
                break;
            }
        }
        if (layoutSlide == null)
        {
            foreach (ILayoutSlide titleLayoutSlide in layoutSlides)
            {
                if (titleLayoutSlide.Name == "Title")
                {
                    layoutSlide = titleLayoutSlide;
                    break;
                }
            }
            if (layoutSlide == null)
            {
                layoutSlide = layoutSlides.GetByType(SlideLayoutType.Blank);
                if (layoutSlide == null)
                {
                    layoutSlide = layoutSlides.Add(SlideLayoutType.TitleAndObject, "Title and Object");
                }
            }
        }
    }
    // Voeg een lege slide toe met de toegevoegde lay-outslide
    presentation.Slides.InsertEmptySlide(0, layoutSlide);
    // Sla de presentatie op
    presentation.Save("AddLayoutSlides_out.pptx", SaveFormat.Pptx);
}
```

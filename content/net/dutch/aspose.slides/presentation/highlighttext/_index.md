## HighlightText(string, Color) {#highlighttext}

Markeert alle overeenkomsten van de voorbeeldtekst met de opgegeven kleur.

```csharp
public void HighlightText(string text, Color highlightColor)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| text | String | De tekst die gemarkeerd moet worden. |
| highlightColor | Color | De kleur waarmee de tekst gemarkeerd wordt. |

### Voorbeelden

De volgende code-voorbeeld laat zien hoe tekst in een PowerPoint-presentatie gemarkeerd kan worden.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	// markeren van alle afzonderlijke 'the'-voorkomens
	presentation.HighlightText("the", Color.Violet);
	presentation.Save("SomePresentation-out2.pptx", SaveFormat.Pptx);
}
```

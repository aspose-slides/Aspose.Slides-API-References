## LoadOptions.DeleteEmbeddedBinaryObjects eigenschap

Bepaalt of Aspose.Slides alle ingebedde binaire objecten zal verwijderen tijdens het laden van de presentatie.

De typen van de ingebedde binaire objecten:

* VBA-project [`VbaProject`](../../ipresentation/vbaproject)
* OLE-object ingebedde gegevens [`EmbeddedFileData`](../../ioleembeddeddatainfo/embeddedfiledata)
* ActiveX-controle binaire gegevens [`ActiveXControlBinary`](../../icontrol/activexcontrolbinary)

Lezen/schrijven Boolean.

```csharp
public bool DeleteEmbeddedBinaryObjects { get; set; }
```

### Opmerkingen

Standaard is **false**.

### Voorbeelden

Het volgende voorbeeld laat zien hoe de presentatie kan worden geladen zonder enige ingebedde binaire objecten.

```csharp
[C#]
LoadOptions loadOptions = new LoadOptions();
loadOptions.DeleteEmbeddedBinaryObjects = true;
using (Presentation pres = new Presentation("pres.ppt", loadOptions))
{
    pres.Save("output_WithoutBinaryObjects.ppt", SaveFormat.Ppt);
}
```

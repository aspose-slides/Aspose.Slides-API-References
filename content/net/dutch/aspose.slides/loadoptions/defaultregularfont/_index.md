## LoadOptions.DefaultRegularFont eigenschap

Geeft of stelt het standaardlettertype in dat wordt gebruikt wanneer het bronlettertype niet wordt gevonden. Lezen/Schrijven String.

```csharp
public string DefaultRegularFont { get; set; }
```

### Voorbeelden

Het volgende voorbeeld toont hoe standaardlettertypen in te stellen voor het renderen van een PowerPoint-presentatie.

```csharp
[C#]
// Gebruik loadopties om de standaard reguliere en Aziatische lettertypen te definiëren
LoadOptions loadOptions = new LoadOptions(LoadFormat.Auto);
loadOptions.DefaultRegularFont = "Wingdings";
loadOptions.DefaultAsianFont = "Wingdings";
// Genereer slide-miniatuur
pptx.Slides[0].GetThumbnail(1, 1).Save("output_out.png", ImageFormat.Png);
// Genereer PDF
pptx.Save("output_out.pdf", SaveFormat.Pdf);
// Genereer XPS
pptx.Save("output_out.xps", SaveFormat.Xps);
```

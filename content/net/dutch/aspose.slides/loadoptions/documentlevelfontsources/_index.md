## LoadOptions.DocumentLevelFontSources eigenschap

Specificeert de bronnen voor externe lettertypen die door de presentatie worden gebruikt. Deze lettertypen zijn gedurende de levensduur van de presentatie beschikbaar en worden niet gedeeld met andere presentaties

```csharp
public IFontSources DocumentLevelFontSources { get; set; }
```

### Voorbeelden

Het volgende voorbeeld laat zien hoe u aangepaste lettertypen kunt opgeven die worden gebruikt met PowerPoint Presentation.

```csharp
[C#]
byte[] memoryFont1 = File.ReadAllBytes("customfonts\\CustomFont1.ttf");
byte[] memoryFont2 = File.ReadAllBytes("customfonts\\CustomFont2.ttf");
LoadOptions loadOptions = new LoadOptions();
loadOptions.DocumentLevelFontSources.FontFolders = new string[] { "assets\\fonts", "global\\fonts" };
loadOptions.DocumentLevelFontSources.MemoryFonts = new byte[][] { memoryFont1, memoryFont2 };
using (IPresentation presentation = new Presentation("MyPresentation.pptx", loadOptions))
{
    //werk met de presentatie
    //CustomFont1, CustomFont2 evenals lettertypen uit de mappen assets\fonts en global\fonts en hun submappen zijn beschikbaar voor de presentatie
}
```

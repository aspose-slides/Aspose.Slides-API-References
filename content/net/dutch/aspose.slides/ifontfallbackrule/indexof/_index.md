## IFontFallBackRule.IndexOf methode

Retourneert een index van de opgegeven regel in de collectie.

```csharp
public int IndexOf(string fontName)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| fontName | String | Naam van het lettertype om te vinden. |

### Retourwaarde

Index van een lettertype of -1 als het lettertype niet in de lijst staat.

### Voorbeelden

```csharp
[C#]
// Maak een regel die een lijst van lettertypen bevat.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

//Haalt index van Tahoma op
int tahomaIndex = newRule.IndexOf("Tahoma");
```

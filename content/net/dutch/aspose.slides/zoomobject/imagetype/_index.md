## ZoomObject.ImageType eigenschap

Krijgt of stelt het afbeeldingstype van een zoomobject in. Lezen/schrijven [`ZoomImageType`](../../zoomimagetype). Standaardwaarde: Preview

```csharp
public ZoomImageType ImageType { get; set; }
```

### Opmerkingen

Specificeert of het Zoom-object de dia-preview gebruikt of een omslagafbeelding.

### Voorbeelden

Het volgende voorbeeld laat zien hoe de Afbeeldingstype wordt gewijzigd naar de waarde Preview. In dit geval verandert de huidige afbeelding van een Zoom-object naar een dia-afbeelding:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    IPPImage image = pres.Images.AddImage(Image.FromFile("image.png"));
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1], image);
    zoomFrame.ImageType = ZoomImageType.Preview;
}
```

## ShapeCollection.InsertPictureFrame methode

Maakt een nieuw afbeeldingsframe dat de opgegeven afbeelding bevat en voegt het in de shape-collectie in op de opgegeven index.

```csharp
public IPictureFrame InsertPictureFrame(int index, ShapeType shapeType, float x, float y, 
    float width, float height, IPPImage image)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| index | Int32 | De nulgebaseerde index waarop het afbeeldingsframe moet worden ingevoegd. |
| shapeType | ShapeType | Specificeert het shape-type dat in [`ShapeType`](../../shapetype) zit, behalve voor alle soorten lijnen: |
| x | Single | De x-coördinaat van het afbeeldingsframe, in points. |
| y | Single | De y-coördinaat van het afbeeldingsframe, in points. |
| width | Single | De breedte van het afbeeldingsframe, in points. |
| height | Single | De hoogte van het afbeeldingsframe, in points. |
| image | IPPImage | De [`IPPImage`](../../ippimage) die in het afbeeldingsframe moet worden weergegeven. |

### Returnwaarde

De nieuw aangemaakte [`IPictureFrame`](../../ipictureframe).

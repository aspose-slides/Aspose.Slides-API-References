## ShapeCollection.InsertSummaryZoomFrame methode

Maakt een nieuw Summary Zoom frame en voegt het in de shape collection in op de opgegeven index.

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | Int32 | De nulgebaseerde index waarop het Summary Zoom frame moet worden ingevoegd. |
| x | Single | De x-coördinaat van het nieuwe Summary Zoom frame, in punten. |
| y | Single | De y-coördinaat van het nieuwe Summary Zoom frame, in punten. |
| width | Single | De breedte van het nieuwe Summary Zoom frame, in punten. |
| height | Single | De hoogte van het nieuwe Summary Zoom frame, in punten. |

### Retourwaarde

De nieuw gemaakte [`ISummaryZoomFrame`](../../isummaryzoomframe).

### Uitzonderingen

| uitzondering | conditie |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | Wordt gegooid als de presentatie geen secties bevat, of als de doel-slide niet tot een sectie behoort. |

### Opmerkingen

Deze methode maakt een Summary Zoom frame dat samenvattings-links voor alle secties in de presentatie aggregeert.

### Voorbeelden

Dit voorbeeld toont het maken en invoegen van een Summary Zoom object op de opgegeven index van een collectie (ervan uitgaande dat er minstens twee secties in de presentatie "Presentation.pptx" zitten):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSummaryZoomFrame(2, 150, 20, 50, 50);
}
```

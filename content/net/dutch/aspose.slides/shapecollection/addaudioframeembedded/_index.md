## AddAudioFrameEmbedded(float, float, float, float, Stream) {#addaudioframeembedded_1}

Maakt een nieuw audioframe met een ingesloten WAV-bestand en voegt het toe aan het einde van de vormcollectie. De ingesloten audio wordt toegevoegd aan de Presentation.Audios-collectie.

```csharp
public IAudioFrame AddAudioFrameEmbedded(float x, float y, float width, float height, 
    Stream audio_stream)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| x | Single | De x-coördinaat van het nieuwe audioframe, in punten. |
| y | Single | De y-coördinaat van het nieuwe audioframe, in punten. |
| width | Single | De breedte van het nieuwe audioframe, in punten. |
| height | Single | De hoogte van het nieuwe audioframe, in punten. |
| audio_stream | Stream | Een invoerstroom met WAV-audiogegevens om in te sluiten. |

### Retourwaarde

Het nieuw aangemaakte [`IAudioFrame`](../../iaudioframe).

### Voorbeelden

De volgende voorbeelden laten zien hoe een Audio Frame te maken.

```csharp
[C#]
// Instantieert een presentatieklasse die een presentatiebestand voorstelt
using (Presentation pres = new Presentation())
{
    // Haalt de eerste dia op
    ISlide sld = pres.Slides[0];
    // Laadt het wav-geluidbestand naar een stream
    FileStream fstr = new FileStream("sampleaudio.wav", FileMode.Open, FileAccess.Read);
    // Voegt het audioframe toe
    IAudioFrame audioFrame = sld.Shapes.AddAudioFrameEmbedded(50, 150, 100, 100, fstr);
    // Stelt de afspeelmodus en het volume van de audio in
    audioFrame.PlayMode = AudioPlayModePreset.Auto;
    audioFrame.Volume = AudioVolumeMode.Loud;
    // Schrijft het PowerPoint-bestand naar schijf
    pres.Save("AudioFrameEmbed_out.pptx", SaveFormat.Pptx);
}
```

## InsertSectionZoomFrame(int, float, float, float, float, ISection) {#insertsectionzoomframe}

Maakt een nieuw Section Zoom-frame en voegt het toe aan de vormverzameling op de opgegeven index.

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| index | Int32 | De nulgebaseerde index waarop het Section Zoom-frame ingevoegd moet worden. |
| x | Single | Het x-coördinaat van het nieuwe Section Zoom-frame, in punten. |
| y | Single | Het y-coördinaat van het nieuwe Section Zoom-frame, in punten. |
| width | Single | De breedte van het nieuwe Section Zoom-frame, in punten. |
| height | Single | De hoogte van het nieuwe Section Zoom-frame, in punten. |
| section | ISection | De [`ISection`](../../isection) waarnaar het Section Zoom-frame verwijst; moet tot deze presentatie behoren en ten minste één dia bevatten. |

### Return Value

De nieuw aangemaakte [`ISectionZoomFrame`](../../isectionzoomframe).

### Exceptions

| exception | condition |
| --- | --- |
| ArgumentException | Wordt opgegooid wanneer de verwezen sectie niet tot de huidige presentatie behoort of geen dia’s bevat. |

### Examples

Dit voorbeeld laat zien hoe een Section Zoom-object wordt aangemaakt en ingevoegd op de opgegeven index van een verzameling (ervan uitgaande dat er minstens twee secties zijn in de presentatie “Presentation.pptx”):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSectionZoomFrame(2, 150, 20, 50, 50, pres.Sections[1]);
}
```

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section, IPPImage image)
```

| Parameter | Type | Beschrijving |

| index | Int32 | De nulgebaseerde index waarop het Section Zoom-frame ingevoegd moet worden. |
| x | Single | Het x-coördinaat van het nieuwe Section Zoom-frame, in punten. |
| y | Single | Het y-coördinaat van het nieuwe Section Zoom-frame, in punten. |
| width | Single | De breedte van het nieuwe Section Zoom-frame, in punten. |
| height | Single | De hoogte van het nieuwe Section Zoom-frame, in punten. |
| section | ISection | De [`ISection`](../../isection) waarnaar het Section Zoom-frame verwijst; moet tot deze presentatie behoren en ten minste één dia bevatten. |
| image | IPPImage | De afbeelding die in het Section Zoom-frame moet worden weergegeven. |

### Return Value

De nieuw aangemaakte [`ISectionZoomFrame`](../../isectionzoomframe).

| exception | condition |

| ArgumentException | Wordt opgegooid wanneer de verwezen sectie niet tot de huidige presentatie behoort of geen dia’s bevat. |

### Examples

Dit voorbeeld laat zien hoe een Section Zoom-object wordt aangemaakt en ingevoegd op de opgegeven index van een verzameling (ervan uitgaande dat er minstens twee secties zijn in de presentatie “Presentation.pptx”):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IPPImage image = pres.Images.AddImage(Image.FromFile("image.png"));

}
```

## AddConnector(ShapeType, float, float, float, float) {#addconnector}

Maakt een nieuwe connectorvorm met standaard sjabloonopmaak en voegt deze toe aan het einde van de vormverzameling.

```csharp
public IConnector AddConnector(ShapeType shapeType, float x, float y, float width, float height)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| shapeType | ShapeType | De [`ShapeType`](../../shapetype) van de connectorvorm die moet worden toegevoegd. |
| x | Single | De x-coördinaat van het kader van de connector, in punten. |
| y | Single | De y-coördinaat van het kader van de connector, in punten. |
| width | Single | De breedte van het kader van de connector, in punten. |
| height | Single | De hoogte van het kader van de connector, in punten. |

### Return Value

De nieuw aangemaakte [`IConnector`](../../iconnector).

### Examples

Het volgende voorbeeld toont hoe een connector (een gebogen connector) tussen twee vormen (een ellips en een rechthoek) kan worden toegevoegd in een PowerPoint-presentatie.

```csharp
[C#]
// Instantieert een presentatieklasse die een PPTX-bestand vertegenwoordigt
using (Presentation input = new Presentation())
{
    // Benadert de vormverzameling voor een specifieke dia
    IShapeCollection shapes = input.Slides[0].Shapes;
    // Voegt een Ellipse-autoshape toe
    IAutoShape ellipse = shapes.AddAutoShape(ShapeType.Ellipse, 0, 100, 100, 100);
    // Voegt een Rectangle-autoshape toe
    IAutoShape rectangle = shapes.AddAutoShape(ShapeType.Rectangle, 100, 300, 100, 100);
    // Voegt een connectorvorm toe aan de vormverzameling van de dia
    IConnector connector = shapes.AddConnector(ShapeType.BentConnector2, 0, 0, 10, 10);
    // Verbindt de vormen met de connector
    connector.StartShapeConnectedTo = ellipse;
    connector.EndShapeConnectedTo = rectangle;
    // Roept reroute aan dat het automatische kortste pad tussen vormen instelt
    connector.Reroute();
    // Slaat de presentatie op
    input.Save("Shapes-connector.pptx", SaveFormat.Pptx);
}
```

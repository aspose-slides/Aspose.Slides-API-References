## InsertZoomFrame(int, float, float, float, float, ISlide) {#insertzoomframe}

Maakt een nieuw Zoom-frame en voegt dit toe aan de vormcollectie op de opgegeven index.

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| index | Int32 | De nul-gebaseerde index waarop het Zoom-frame moet worden ingevoegd. |
| x | Single | De x-coördinaat van het nieuwe Zoom-frame, in punten. |
| y | Single | De y-coördinaat van het nieuwe Zoom-frame, in punten. |
| width | Single | De breedte van het nieuwe Zoom-frame, in punten. |
| height | Single | De hoogte van het nieuwe Zoom-frame, in punten. |
| slide | ISlide | De [`ISlide`](../../islide) waarnaar het Zoom-frame verwijst. |

### Retourwaarde

De nieuw gemaakte [`IZoomFrame`](../../izoomframe).

### Uitzonderingen

| Uitzondering | conditie |
| --- | --- |
| ArgumentException | Wordt gegooid als de verwijzende dia niet tot de huidige presentatie behoort. |

### Voorbeelden

Dit voorbeeld toont het maken en invoegen van een Zoom-object op de opgegeven index van een collectie (neem aan dat er minstens twee dia's in de presentatie "Presentation.pptx" aanwezig zijn):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertZoomFrame(2, 150, 20, 50, 50, pres.Slides[1]);
}
```

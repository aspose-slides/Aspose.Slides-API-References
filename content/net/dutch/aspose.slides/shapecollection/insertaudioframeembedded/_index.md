## InsertAudioFrameEmbedded(int, float, float, float, float, Stream) {#insertaudioframeembedded_1}

Maakt een nieuw audioframe met een ingesloten WAV-bestand en voegt het in de shape-verzameling in op de opgegeven index. Het ingesloten audio wordt toegevoegd aan de Presentation.Audios-collectie.

```csharp
public IAudioFrame InsertAudioFrameEmbedded(int index, float x, float y, float width, float height, 
    Stream audio_stream)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| index | Int32 | De nulgebaseerde index waarop het audioframe moet worden ingevoegd. |
| x | Single | Het x-coördinaat van het nieuwe audioframe, in punten. |
| y | Single | Het y-coördinaat van het nieuwe audioframe, in punten. |
| width | Single | De breedte van het nieuwe audioframe, in punten. |
| height | Single | De hoogte van het nieuwe audioframe, in punten. |
| audio_stream | Stream | Een invoerstroom die WAV-audiogegevens bevat om in te sluiten. |

### Return Value

De nieuw aangemaakte [`IAudioFrame`](../../iaudioframe).

## InsertAutoShape(int, ShapeType, float, float, float, float) {#insertautoshape}

## InsertAutoShape(int, ShapeType, float, float, float, float, bool) {#insertautoshape_1}

Maakt een nieuw autoshape en voegt het in de vormverzameling in op de opgegeven index, waarbij het desgewenst wordt geïnitialiseerd met de standaard sjabloonstijl.

```csharp
public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| index | Int32 | De nulgebaseerde index waarop de autoshape moet worden ingevoegd. |
| shapeType | ShapeType | De [`ShapeType`](../../shapetype) van de in te voegen autoshape. |
| x | Single | De x-coördinaat van het frame van de vorm, in punten. |
| y | Single | De y-coördinaat van het frame van de vorm, in punten. |
| width | Single | De breedte van het frame van de vorm, in punten. |
| height | Single | De hoogte van het frame van de vorm, in punten. |
| createFromTemplate | Boolean | True om de standaard sjabloonstijl toe te passen (inclusief een niet-lege naam, eenvoudige stijl en gecentreerde tekst); false om de vorm te maken met alle eigenschappen ingesteld op hun standaardwaarden. |

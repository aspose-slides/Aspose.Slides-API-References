## InsertOleObjectFrame(int, float, float, float, float, IOleEmbeddedDataInfo) {#insertoleobjectframe}

Creëert een nieuw OLE-objectframe en voegt het toe aan de vormverzameling op de opgegeven index.

```csharp
public IOleObjectFrame InsertOleObjectFrame(int index, float x, float y, float width, float height, 
    IOleEmbeddedDataInfo dataInfo)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| index | Int32 | De nul-gebaseerde index waarop het OLE-objectframe moet worden ingevoegd. |
| x | Single | Het x-coördinaat van het nieuwe OLE-frame, in punten. |
| y | Single | Het y-coördinaat van het nieuwe OLE-frame, in punten. |
| width | Single | De breedte van het nieuwe OLE-frame, in punten. |
| height | Single | De hoogte van het nieuwe OLE-frame, in punten. |
| dataInfo | IOleEmbeddedDataInfo | De embedded OLE-datainformatie ([`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo)). |

### Retourwaarde

De nieuw aangemaakte [`IOleObjectFrame`](../../ioleobjectframe).

### Voorbeelden

Dit voorbeeld laat zien hoe een OLE-object op de tweede index wordt ingevoegd:

```csharp
[C#]
byte[] fileData = File.ReadAllBytes("test.zip");
IOleDataInfo dataInfo = new OleDataInfo(fileData, "zip");
IOleObjectFrame oleObjectFrame = slidees.Shapes.AddOleObjectFrame(2, 150, 20, 50, 50, dataInfo);
```

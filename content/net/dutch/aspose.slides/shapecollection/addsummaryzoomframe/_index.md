## ShapeCollection.AddSummaryZoomFrame methode

Maakt een nieuw Summary Zoom-frame en voegt het toe aan het einde van de vormverzameling.

```csharp
public ISummaryZoomFrame AddSummaryZoomFrame(float x, float y, float width, float height)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| x | Single | De x-coördinaat van het nieuwe Summary Zoom-frame, in punten. |
| y | Single | De y-coördinaat van het nieuwe Summary Zoom-frame, in punten. |
| width | Single | De breedte van het nieuwe Summary Zoom-frame, in punten. |
| height | Single | De hoogte van het nieuwe Summary Zoom-frame, in punten. |

### Retourwaarde

Het nieuw aangemaakte [`ISummaryZoomFrame`](../../isummaryzoomframe).

### Uitzonderingen

| uitzondering | voorwaarde |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | Wordt gegooid als er geen secties in de presentatie zijn, of als de doeldia niet tot een sectie behoort. |

### Opmerkingen

Deze methode maakt een nieuw Summary Zoom en plaatst er een verzameling objecten in voor alle secties in deze presentatie.

### Voorbeelden

Dit voorbeeld toont hoe een Summary Zoom-object aan het einde van een verzameling wordt toegevoegd (ervar dat er ten minste twee secties in de presentatie "Presentation.pptx" zitten):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSummaryZoomFrame(150, 20, 500, 250);
}
```

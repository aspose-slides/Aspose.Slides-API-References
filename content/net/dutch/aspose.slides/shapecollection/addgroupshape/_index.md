## AddGroupShape() {#addgroupshape}

Maakt een nieuwe lege GroupShape aan en voegt deze toe aan het einde van de vormverzameling. Het frame van de groep wordt automatisch aangepast om alle toegevoegde vormen te bevatten.

```csharp
public IGroupShape AddGroupShape()
```

### Retourwaarde

De nieuw aangemaakte [`IGroupShape`](../../igroupshape).

### Voorbeelden

Het volgende voorbeeld toont hoe een GroupShape kan worden toegevoegd aan een dia van een PowerPoint-presentatie.

```csharp
[C#]
// Instantieer Presentation-klasse
using (Presentation pres = new Presentation())
{
    // Haal de eerste dia op
    ISlide sld = pres.Slides[0];
    // Toegang tot de vormverzameling van dia's
    IShapeCollection slideShapes = sld.Shapes;
    // Groepsvorm aan de dia toevoegen
    IGroupShape groupShape = slideShapes.AddGroupShape();
    // Vormen toevoegen binnen de toegevoegde groepsvorm
    groupShape.Shapes.AddAutoShape(ShapeType.Rectangle, 300, 100, 100, 100);
    groupShape.Shapes.AddAutoShape(ShapeType.Rectangle, 500, 100, 100, 100);
    groupShape.Shapes.AddAutoShape(ShapeType.Rectangle, 300, 300, 100, 100);
    groupShape.Shapes.AddAutoShape(ShapeType.Rectangle, 500, 300, 100, 100);
    // Frame van de groepsvorm toevoegen
    groupShape.Frame = new ShapeFrame(100, 300, 500, 40, NullableBool.False, NullableBool.False, 0);
    // Schrijf het PPTX-bestand naar schijf
    pres.Save("GroupShape_out.pptx", SaveFormat.Pptx);
}
```

## ShapeCollection.AddMathShape methode

Maakt een nieuw rechthoekig auto shape aan om wiskundige inhoud te hosten en voegt het toe aan het einde van de shape-collectie.

```csharp
public IAutoShape AddMathShape(float x, float y, float width, float height)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| x | Single | De x-coördinaat van het shape-frame, in punten. |
| y | Single | De y-coördinaat van het shape-frame, in punten. |
| width | Single | De breedte van het shape-frame, in punten. |
| height | Single | De hoogte van het shape-frame, in punten. |

### Retourwaarde

The newly created [`IAutoShape`](../../iautoshape).

### Voorbeelden

Het volgende voorbeeld toont hoe u een wiskundige vergelijking aan een PowerPoint-presentatie kunt toevoegen.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
   IAutoShape mathShape = pres.Slides[0].Shapes.AddMathShape(0, 0, 720, 150);
   var mathParagraph = (mathShape.TextFrame.Paragraphs[0].Portions[0] as MathPortion).MathParagraph;
   var fraction = new MathematicalText("x").Divide("y");
   mathParagraph.Add(new MathBlock(fraction));
   var mathBlock = new MathematicalText("c")
        .SetSuperscript("2")
        .Join("=")
        .Join(new MathematicalText("a").SetSuperscript("2"))
        .Join("+")
        .Join(new MathematicalText("b").SetSuperscript("2"));
    mathParagraph.Add(mathBlock);
    pres.Save("math.pptx", SaveFormat.Pptx);
}
```

## MasterSlideCollection.InsertClone methode

Voegt een kopie van een opgegeven masterslide toe op de opgegeven positie in de collectie. Gekoppelde layout-slides worden ook gekopieerd.

```csharp
public IMasterSlide InsertClone(int index, IMasterSlide sourceMaster)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| index | Int32 | Index van de nieuwe slide. |
| sourceMaster | IMasterSlide | Slide om te klonen. |

### Retourwaarde

Toegevoegde masterslide.

### Voorbeelden

Het volgende voorbeeld toont hoe een masterslide te klonen in een andere PowerPoint-presentatie.

```csharp
[C#]
// Instantieer de Presentation-klasse om het bronpresentatie-bestand te laden
using (Presentation srcPres = new Presentation("CloneToAnotherPresentationWithMaster.pptx"))
{
    // Instantieer de Presentation-klasse voor de doelpresentatie (waar de slide moet worden gekloond)
    using (Presentation destPres = new Presentation())
    {
        // Instantieer ISlide uit de collectie slides in de bronpresentatie samen met
        // Master-slide
        ISlide SourceSlide = srcPres.Slides[0];
        IMasterSlide SourceMaster = SourceSlide.LayoutSlide.MasterSlide;
		// Haal Master-slides op van de doelpresentatie
        IMasterSlideCollection masters = destPres.Masters;
        // Kopieer de gewenste master-slide van de bronpresentatie naar de collectie masters in de
        // Doelpresentatie
        IMasterSlide iSlide = masters.AddClone(SourceMaster);
        // Collectie van slides in de doelpresentatie
        ISlideCollection slds = destPres.Slides;
		// Kopieer de bron-slide naar de slides-collectie van de doelpresentatie.
        slds.AddClone(SourceSlide, iSlide, true);
        // Sla de doelpresentatie op naar schijf
        destPres.Save("CloneToAnotherPresentationWithMaster_out.pptx", SaveFormat.Pptx);
    }
}
```

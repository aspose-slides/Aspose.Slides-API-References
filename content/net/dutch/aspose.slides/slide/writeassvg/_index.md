Slaat de slide-inhoud op als een SVG-bestand.


| Parameter | Type | Beschrijving |
| --- | --- | --- |
| stream | Stream | Doelstream |

### Voorbeelden

De volgende code-voorbeeld toont hoe de eerste slide van een PowerPoint-presentatie kan worden geconverteerd naar een SVG-bestand.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.svg"))
    {
        // Slaat de eerste slide op als een SVG-bestand
        pres.Slides[0].WriteAsSvg(fileStream);
    }
}
```

### Voorbeelden

De volgende code-voorbeeld toont hoe de eerste slide van een PowerPoint-presentatie kan worden geconverteerd naar een SVG-bestand met opties.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.svg"))
    {
        var options = new SVGOptions() { VectorizeText = true };
        // Slaat de eerste slide op als een SVG-bestand
        pres.Slides[0].WriteAsSvg(fileStream, options);
    }
}
```

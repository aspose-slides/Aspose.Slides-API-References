### Voorbeelden

Het volgende codevoorbeeld laat zien hoe de eerste dia van een PowerPoint-presentatie kan worden geconverteerd naar een metafile.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.emf"))
    {
        // Slaat de eerste dia op als een metafile
        pres.Slides[0].WriteAsEmf(fileStream);
    }
}
```

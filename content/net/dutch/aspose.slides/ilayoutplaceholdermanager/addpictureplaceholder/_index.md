## ILayoutPlaceholderManager.AddPicturePlaceholder methode

Voegt een nieuw placeholder-vorm toe aan de layoutdia om een afbeelding vast te houden.

```csharp
public IAutoShape AddPicturePlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| x | Single | The X coordinate of the new placeholder shape. |
| y | Single | The Y coordinate of the new placeholder shape. |
| width | Single | The width of the new placeholder shape. |
| height | Single | The height of the new placeholder shape. |

### Retourwaarde

Gemaakt [`IAutoShape`](../../iautoshape) met een Picture-placeholder.

### Voorbeelden

Het volgende voorbeeld toont hoe u de Picture-placeholder vorm aan de layoutdia toevoegt.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddPicturePlaceholder(20, 20, 200, 200);
}
```

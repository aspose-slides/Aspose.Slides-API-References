## ILayoutPlaceholderManager.AddSmartArtPlaceholder methode

Voegt een nieuw plaatshoudervorm toe aan de lay-outdia om een SmartArt-diagram te bevatten.

```csharp
public IAutoShape AddSmartArtPlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| x | Single | De X-coördinaat van de nieuwe plaatshoudervorm. |
| y | Single | De Y-coördinaat van de nieuwe plaatshoudervorm. |
| width | Single | De breedte van de nieuwe plaatshoudervorm. |
| height | Single | De hoogte van de nieuwe plaatshoudervorm. |

### Retourwaarde

Gemaakt [`IAutoShape`](../../iautoshape) met een SmartArt-plaatshouder.

### Voorbeelden

Het volgende voorbeeld laat zien hoe u de SmartArt-plaatshoudervorm aan de lay-outdia kunt toevoegen.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddSmartArtPlaceholder(20, 20, 200, 200);
}
```

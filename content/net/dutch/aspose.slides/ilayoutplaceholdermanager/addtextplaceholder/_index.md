## ILayoutPlaceholderManager.AddTextPlaceholder methode

Voegt een nieuw placeholder-vorm toe aan de lay-out-dia om tekstinhoud vast te houden.

```csharp
public IAutoShape AddTextPlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| x | Single | De X-coördinaat van de nieuwe placeholder-vorm. |
| y | Single | De Y-coördinaat van de nieuwe placeholder-vorm. |
| width | Single | De breedte van de nieuwe placeholder-vorm. |
| height | Single | De hoogte van de nieuwe placeholder-vorm. |

### Retourwaarde

Gemaakt [`IAutoShape`](../../iautoshape) met een Text placeholder.

### Voorbeelden

Het volgende voorbeeld laat zien hoe je de Text-placeholder-vorm toevoegt aan de lay-out-dia.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTextPlaceholder(20, 20, 500, 300);
}
```

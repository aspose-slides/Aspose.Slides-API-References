## ILayoutPlaceholderManager.AddMediaPlaceholder-methode

Voegt een nieuw placeholder-vorm toe aan de layout-dia om een media-object te bevatten.

```csharp
public IAutoShape AddMediaPlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| x | Single | De X-coördinaat van de nieuwe placeholder-vorm. |
| y | Single | De Y-coördinaat van de nieuwe placeholder-vorm. |
| width | Single | De breedte van de nieuwe placeholder-vorm. |
| height | Single | De hoogte van de nieuwe placeholder-vorm. |

### Retourwaarde

Gemaakt [`IAutoShape`](../../iautoshape) met een Media-placeholder.

### Voorbeelden

Het volgende voorbeeld toont hoe de Media-placeholder-vorm kan worden toegevoegd aan de layout-dia.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddMediaPlaceholder(20, 20, 200, 200);
}
```

## ILayoutPlaceholderManager.AddTablePlaceholder method

Voegt een nieuw placeholder-shape toe aan de layout-slide om een tabel te bevatten.

```csharp
public IAutoShape AddTablePlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| x | Single | De X-coördinaat van het nieuwe placeholder-shape. |
| y | Single | De Y-coördinaat van het nieuwe placeholder-shape. |
| width | Single | De breedte van het nieuwe placeholder-shape. |
| height | Single | De hoogte van het nieuwe placeholder-shape. |

### Retourwaarde

Gemaakt [`IAutoShape`](../../iautoshape) met een Table placeholder.

### Voorbeelden

Het volgende voorbeeld laat zien hoe je de Table placeholder-shape toevoegt aan de layout-slide.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTablePlaceholder(20, 20, 500, 200);
}
```

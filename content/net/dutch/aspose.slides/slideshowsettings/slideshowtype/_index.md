## SlideShowSettings.SlideShowType eigenschap

Haalt of stelt het type diavoorstelling in. Wordt vertegenwoordigd door de volgende `SlideShowType` voorouders: [`BrowsedAtKiosk`](../../browsedatkiosk), [`PresentedBySpeaker`](../../presentedbyspeaker) en [`BrowsedByIndividual`](../../browsedbyindividual)

```csharp
public SlideShowType SlideShowType { get; set; }
```

### Voorbeelden

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // om het type "Browsed at a kiosk (full screen)" in te stellen 
    pres.SlideShowSettings.SlideShowType = new BrowsedAtKiosk();
    
    // om het type "Browsed by individual (window)" in te stellen
    pres.SlideShowSettings.SlideShowType = new BrowsedByIndividual();
    
    // om het type "Presented by a speaker (full screen)" in te stellen
    pres.SlideShowSettings.SlideShowType = new PresentedBySpeaker();
}
```

## IPresentation.HighlightRegex methode

Markeert alle overeenkomsten van de reguliere expressie met de opgegeven kleur.

```csharp
public void HighlightRegex(Regex regex, Color highlightColor, IFindResultCallback callback)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| regex | Regex | De reguliere expressie Regex om de tekenreeksen te markeren. |
| highlightColor | Color | De kleur om de tekst te markeren. |
| callback | IFindResultCallback | Het callback-object voor het ontvangen van zoekresultaten [`IFindResultCallback`](../../ifindresultcallback). |

### Voorbeelden

Het volgende codevoorbeeld toont hoe tekst in een PowerPoint-presentatie kan worden gemarkeerd met behulp van een reguliere expressie.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// markeren van alle woorden met 10 of meer tekens
	presentation.HighlightRegex(regex, Color.Blue, null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

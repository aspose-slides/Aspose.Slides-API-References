## IDocumentProperties.AppVersion eigenschap

Retourneert de app-versie. Alleen-lezen String.

```csharp
public string AppVersion { get; }
```

### Opmerkingen

De inhoud van dit element moet de vorm XX.YYYY hebben, waarbij X en Y numerieke waarden vertegenwoordigen; anders wordt het document als niet-conform beschouwd. Aspose.Slides geeft zijn versie weer in het formaat XX.YYZZ, waarbij:  
XX - major version  
YY - minor version  
ZZ - patch version  
Bijvoorbeeld, de waarde 23.0105 betekent Aspose.Slides versie 23.1.5.

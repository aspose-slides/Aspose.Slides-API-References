## PresentationInfo.CheckWriteProtection methode

Controleert of een wachtwoord om te wijzigen correct is voor een schrijfbeschermde presentatie.

```csharp
public bool CheckWriteProtection(string password)
```

| Parameter | Type | Description |
| --- | --- | --- |
| password | String | Het wachtwoord om te controleren. |

### Retourwaarde

True als de presentatie schrijfbeschermd is en het wachtwoord correct is. False anders.

### Uitzonderingen

| uitzondering | conditie |
| --- | --- |
| InvalidOperationException |  |

### Opmerkingen

1. U moet de [`IsWriteProtected`](../iswriteprotected) eigenschap controleren voordat u deze methode aanroept. 2. Wanneer wachtwoord null of leeg is, retourneert deze methode false.

### Voorbeelden

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
if (info.IsWriteProtected == NullableBool.True)
{
    bool isWriteProtectedByPassword = info.CheckWriteProtection("my_password");
}
```

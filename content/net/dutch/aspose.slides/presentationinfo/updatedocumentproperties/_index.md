## PresentationInfo.UpdateDocumentProperties methode

Werkt de eigenschappen van de gekoppelde presentatie bij.

```csharp
public void UpdateDocumentProperties(IDocumentProperties documentProperties)
```

### Voorbeelden

Dit voorbeeld toont hoe de `UpdateDocumentProperties` methode aangeroepen kan worden om de documenteigenschappen bij te werken die worden geretourneerd door een aanroep van de [`ReadDocumentProperties`](../readdocumentproperties) methode.

```csharp
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo("pres.pptx");
IDocumentProperties props = info.ReadDocumentProperties();
props.Subject = "New subject";
props.LastSavedTime = DateTime.UtcNow;
info.UpdateDocumentProperties(props);
info.WriteBindedPresentation("new_pres.pptx");
```

## IOleObjectFrame.LinkPathRelative eigenschap

Geeft het relatieve pad naar een gekoppeld bestand terug indien aanwezig, anders wordt een lege string geretourneerd. Alleen-lezen String.

```csharp
public string LinkPathRelative { get; }
```

### Opmerkingen

In de Ppt-presentaties kunnen sommige Ole-objectkoppelingen een relatieve weergave hebben.

### Voorbeelden

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.ppt"))
{
    IOleObjectFrame oleFrame = presentation.Slides[0].Shapes[0] as IOleObjectFrame;

    if (oleFrame != null)
    {
        Console.WriteLine("The relative path: " + oleFrame.LinkPathRelative);
    } 
}
```

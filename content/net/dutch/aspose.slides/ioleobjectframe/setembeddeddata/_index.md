## IOleObjectFrame.SetEmbeddedData methode

Stelt informatie in over OLE ingesloten gegevens.

```csharp
public void SetEmbeddedData(IOleEmbeddedDataInfo embeddedData)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| embeddedData | IOleEmbeddedDataInfo | Embedded data [`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo) |

### Uitzonderingen

| Uitzondering | Voorwaarde |
| --- | --- |
| ArgumentNullException | Wanneer de parameter embeddedData null is. |

### Opmerkingen

Deze methode wijzigt de eigenschappen van het object om de nieuwe gegevens weer te geven en zet de IsObjectLink-vlag op false, wat aangeeft dat het OLE-object is ingesloten.

### Voorbeelden

Het volgende voorbeeld laat zien hoe OLE ingesloten gegevens en het type ervan te wijzigen voor een bestaand [`IOleObjectFrame`](../../ioleobjectframe) object

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    OleObjectFrame oof = pres.Slides[0].Shapes[0] as OleObjectFrame;
    if (oof != null)
    {
        IOleEmbeddedDataInfo newData = new OleEmbeddedDataInfo(File.ReadAllBytes("Picture.png"), "png");
        oof.SetEmbeddedData(newData);
    }
}
```

## IPresentationInfo.IsWriteProtected eigenschap

Geeft een waarde terug die aangeeft of een gekoppelde presentatie schrijfbeveiligd is.

```csharp
public NullableBool IsWriteProtected { get; }
```

### Opmerkingen

Als de presentatie is beveiligd met een wachtwoord om te openen, is de eigenschapswaarde gelijk aan NotDefined. Zie [`NullableBool`](../../nullablebool) enumeratie.

### Voorbeelden

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
if (info.IsWriteProtected == NullableBool.True)
{
    Console.WriteLine("The presentation '" + presentationFilePath + "' is write protected by a password.");
}
```

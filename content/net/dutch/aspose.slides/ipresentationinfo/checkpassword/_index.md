## IPresentationInfo.CheckPassword methode

Controleert of een wachtwoord correct is voor een presentatie die beschermd is met een open wachtwoord.

```csharp
public bool CheckPassword(string password)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| password | String | Het wachtwoord om te controleren. |

### Retourwaarde

True als de presentatie beschermd is met een open wachtwoord en het wachtwoord correct is, en false anders.

### Opmerkingen

Wanneer het wachtwoord null of leeg is, geeft deze methode false terug.

### Voorbeelden

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo("pres.pptx");
bool isPasswordCorrect = info.CheckPassword("my_password");
```

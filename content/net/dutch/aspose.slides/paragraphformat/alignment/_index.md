## ParagraphFormat.Alignment eigenschap

Geeft de uitlijning van tekst in een alinea terug of stelt deze in zonder overerving. Lezen/schrijven [`TextAlignment`](../../textalignment).

public TextAlignment Alignment { get; set; }

### Voorbeelden

De volgende voorbeeldcode laat zien hoe tekstalinea's uitgelijnd kunnen worden in een PowerPoint-presentatie.

```csharp
[C#]
// Maak een Presentation-object aan dat een PPTX-bestand vertegenwoordigt
using (Presentation pres = new Presentation("ParagraphsAlignment.pptx"))
{
    // Toegang tot de eerste dia
    ISlide slide = pres.Slides[0];
    // Toegang tot de eerste en tweede placeholder op de dia en deze casten naar AutoShape
    ITextFrame tf1 = ((IAutoShape)slide.Shapes[0]).TextFrame;
    ITextFrame tf2 = ((IAutoShape)slide.Shapes[1]).TextFrame;
    // Verander de tekst in beide placeholders
    tf1.Text = "Center Align by Aspose";
    tf2.Text = "Center Align by Aspose";
    // Haalt de eerste alinea van de placeholders op
    IParagraph para1 = tf1.Paragraphs[0];
    IParagraph para2 = tf2.Paragraphs[0];
    // Lijnt de tekstaandelen uit naar het midden
    para1.ParagraphFormat.Alignment = TextAlignment.Center;
    para2.ParagraphFormat.Alignment = TextAlignment.Center;
    // Schrijft de presentatie weg als een PPTX-bestand
    pres.Save("Centeralign_out.pptx", SaveFormat.Pptx);
}
```

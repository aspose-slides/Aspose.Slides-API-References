## Fonts.SetScriptFont-methode

Wijst een lettertype toe aan een specifiek script-tag, dat bepaalt hoe tekst van dat script wordt weergegeven in de presentatie.

```csharp
public void SetScriptFont(string script, string fontName)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| script | String | De BCP-47 scriptcode (bijv. "Arab", "Hebr", "Hans") die het schrift identificeert. |
| fontName | String | De naam van het lettertype dat aan het opgegeven script moet worden toegewezen. |

### Voorbeelden

Dit voorbeeld laat zien hoe u het lettertype voor het Arabische script instelt op "Segoe UI":

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segoe UI");
```

## Fonts.GetScriptFont methode

Haalt de naam van het lettertype op die is gekoppeld aan een specifiek script-tag uit het presentatiethema.

```csharp
public string GetScriptFont(string script)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| script | String | De BCP-47 scriptcode (bijv. "Latn", "Cyrl", "Jpan") die wordt gebruikt om een schrijfsysteem te identificeren. |

### Retourwaarde

De naam van het lettertype dat wordt gebruikt voor het opgegeven script, of `null` als het script niet is gedefinieerd.

### Voorbeelden

Dit voorbeeld laat zien hoe je het lettertype kunt ophalen dat is toegewezen aan het Cyrillische script in het presentatiethema.

```csharp
[C#]
string font = presentation.MasterTheme.FontScheme.Major.GetScriptFont("Cyrl");
Console.WriteLine("Font for Cyrillic script: " + font);
```

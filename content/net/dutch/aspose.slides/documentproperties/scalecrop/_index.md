## DocumentProperties.ScaleCrop eigenschap

Geeft de weergavemodus van de documentminiatuur aan. Stel dit element in op **true** om schalen van de documentminiatuur naar het scherm mogelijk te maken. Stel dit element in op **false** om bijsnijden van de documentminiatuur mogelijk te maken, zodat alleen secties die op het scherm passen worden getoond. Lezen/schrijven Boolean.

public bool ScaleCrop { get; set; }

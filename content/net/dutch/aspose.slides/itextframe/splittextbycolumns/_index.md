## ITextFrame.SplitTextByColumns methode

Splitst de tekstinhoud van de [`ITextFrame`](../../itextframe) in een array van strings, waarbij elk element overeenkomt met een afzonderlijke tekstkolom binnen het frame.

```csharp
public string[] SplitTextByColumns()
```

### Retourwaarde

Een array van strings, waarbij elke string de tekstinhoud van een specifieke kolom in de [`ITextFrame`](../../itextframe) weergeeft.

### Opmerkingen

Als het tekstframe geen meerdere kolommen bevat, zal de geretourneerde array één element hebben dat de volledige tekst bevat. Lege kolommen worden in de array weergegeven als lege strings.

### Voorbeelden

Het volgende voorbeeld laat zien hoe `SplitTextByColumns` gebruikt kan worden:

```csharp
using (Presentation pres = new Presentation("example.pptx"))
{
    // Haal de eerste vorm op de dia en cast deze naar ITextFrame
    ITextFrame textFrame = pres.Slides[0].Shapes[0] as ITextFrame;
    // Splits de inhoud van het tekstframe in kolommen
    string[] columnsText = textFrame.SplitTextByColumns();
    // Print de tekst van elke kolom naar de console
    foreach (string column in columnsText)
        Console.WriteLine(column);
}
```

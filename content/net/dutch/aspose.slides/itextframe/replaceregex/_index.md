## ITextFrame.ReplaceRegex-methode

Vervangt alle overeenkomsten van de reguliere expressie door de opgegeven tekenreeks.

```csharp
public void ReplaceRegex(Regex regex, string newText, IFindResultCallback callback)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| regex | Regex | De reguliere expressie Regex om de te vervangen tekenreeksen op te halen. |
| newText | String | De tekenreeks om alle voorkomens van de te vervangen tekenreeksen te vervangen. |
| callback | IFindResultCallback | Het callback-object voor het ontvangen van zoekresultaten [`IFindResultCallback`](../../ifindresultcallback). |

### Voorbeelden

De volgende code-voorbeeld toont hoe tekst te vervangen met behulp van een reguliere expressie en de opgegeven tekenreeks.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// Vervang alle woorden met 10 of meer tekens door '***'
	((AutoShape)presentation.Slides[0].Shapes[0]).TextFrame.ReplaceRegex(regex, "***", null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

## IFonts.GetScriptFont methode

Haalt de naam van het lettertype op dat aan een specifiek script-tag is gekoppeld in het presentatiethema.

```csharp
public string GetScriptFont(string script)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| script | String | De BCP-47 scriptcode (bijv. "Latn", "Cyrl", "Jpan") die wordt gebruikt om een schrijfsysteem te identificeren. |

### Retourwaarde

De naam van het lettertype dat voor het opgegeven script wordt gebruikt, of `null` als het script niet is gedefinieerd.

### Voorbeelden

Dit voorbeeld laat zien hoe het aan het Cyrilisch script toegewezen lettertype in het presentatiethema kan worden opgehaald.

```csharp
[C#]
string font = presentation.MasterTheme.FontScheme.Major.GetScriptFont("Cyrl");
Console.WriteLine("Font for Cyrillic script: " + font);
```

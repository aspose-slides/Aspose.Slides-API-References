## IPictureFillFormat.TileOffsetY eigenschap

Geeft de verticale offset van de textuur vanaf de oorsprong van de vorm in punten terug of stelt deze in. Een positieve waarde verplaatst de textuur naar beneden, terwijl een negatieve waarde deze naar boven verplaatst. Lezen/schrijven Single.

```csharp
public float TileOffsetY { get; set; }
```

### Voorbeelden

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Haalt het picture fill format van de shape op
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Stelt de picture fill mode in op Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Stelt de verticale offset van de texture in op -50 punten
    pictureFillFormat.TileOffsetY = -50;
}
```

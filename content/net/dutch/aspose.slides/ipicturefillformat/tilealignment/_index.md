## IPictureFillFormat.TileAlignment eigenschap

Retourneert of stelt in hoe de textuur is uitgelijnd binnen de vorm. Deze instelling bepaalt het startpunt van het textuurpatroon en hoe het zich herhaalt over de vorm. Lezen/Schrijven [`RectangleAlignment`](../../rectanglealignment).

```csharp
public RectangleAlignment TileAlignment { get; set; }
```

### Opmerkingen

Standaard is TopLeft.

### Voorbeelden

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Haalt het picture fill-formaat van de vorm op
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Stelt de picture fill-modus in op Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Stelt de uitlijning voor het tegelpatroon in op rechtsonder
    pictureFillFormat.TileAlignment = RectangleAlignment.BottomRight;
}
```

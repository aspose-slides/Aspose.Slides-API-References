## IPictureFillFormat.DeletePictureCroppedAreas method

Verwijder bijgesneden gebieden van de gevulde Picture.

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### Retourwaarde

Bijgesneden afbeelding of originele afbeelding als bijsnijden niet nodig is.

### Opmerkingen

Deze methode converteert WMF/EMF-metabestanden naar raster PNG-afbeelding tijdens het bijsnijden.

### Voorbeelden

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Haalt het PictureFrame op
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // Verwijdert bijgesneden gebieden van de afbeelding in het PictureFrame
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

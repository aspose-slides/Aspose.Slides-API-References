## GetSubstitutions(int[]) {#getsubstitutions_1}

Haalt de informatie op over lettertypen die tijdens het weergeven van de opgegeven dia's worden vervangen.

```csharp
public IEnumerable<FontSubstitutionInfo> GetSubstitutions(int[] slides)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| slides | Int32[] | Een array van dia-indexen waarvoor de informatie over lettertype-substitutie moet worden opgehaald, beginnend bij 1. |

### Retourwaarde

Een collectie van alle lettertype-substituties ([`FontSubstitutionInfo`](../../fontsubstitutioninfo)) voor de opgegeven dia's.

### Voorbeelden

```csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    int[] targetSlides = { 1, 2, 5 };
    foreach (var fontSubstitution in pres.FontsManager.GetSubstitutions(targetSlides))
    {
        Console.WriteLine("{0} -> {1}", fontSubstitution.OriginalFontName, fontSubstitution.SubstitutedFontName);
    }
}
```

## BaseSlide.Equals methode

Bepaalt of de twee IBaseSlide-instanties gelijk zijn. De geretourneerde waarde wordt berekend op basis van de structuur en statische inhoud van de dia. Twee dia's zijn gelijk als alle vormen, stijlen, teksten, animaties en andere instellingen, enz., gelijk zijn. Bij de vergelijking worden unieke identificatiewaarden, zoals SlideId, en dynamische inhoud, zoals de huidige datumwaarde in Datum Placeholder, niet in aanmerking genomen.

```csharp
public bool Equals(IBaseSlide slide)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| slide | IBaseSlide | De IBaseSlide om te vergelijken met de huidige IBaseSlide. |

### Retourwaarde

**true** if de opgegeven IBaseSlide gelijk is aan de huidige IBaseSlide; anders, **false**.

### Voorbeelden

Het volgende voorbeeld toont hoe twee dia's te vergelijken.

```csharp
[C#]
using (Presentation presentation1 = new Presentation("AccessSlides.pptx"))
{
	using (Presentation presentation2 = new Presentation("HelloWorld.pptx"))
	{
		for (int i = 0; i < presentation1.Masters.Count; i++)
		{
			for (int j = 0; j < presentation2.Masters.Count; j++)
			{
				if (presentation1.Masters[i].Equals(presentation2.Masters[j]))
					Console.WriteLine(string.Format("SomePresentation1 MasterSlide#{0} is equal to SomePresentation2 MasterSlide#{1}", i, j));
			}
		}
	}
}
```

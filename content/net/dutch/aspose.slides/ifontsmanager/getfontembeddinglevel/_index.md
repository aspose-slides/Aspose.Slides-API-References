## IFontsManager.GetFontEmbeddingLevel method

Bepaalt het insluitingsniveau van een lettertype op basis van de opgegeven byte-array en de naam van het lettertype.

```csharp
public EmbeddingLevel GetFontEmbeddingLevel(byte[] fontBytes, string fontName)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| fontBytes | Byte[] | De byte-array die de lettertype-gegevens bevat. |
| fontName | String | De naam van het lettertype. |

### Retourwaarde

Het insluitingsniveau van het opgegeven lettertype.

### Uitzonderingen

| Uitzondering | conditie |
| --- | --- |
| ArgumentNullException | Wordt gegooid wanneer *fontBytes* null is. |

### Voorbeelden

```csharp
[C#]
using (Presentation pres = new Presentation(pptxFileName))
{
    // Haal alle lettertypen op die in de presentatie worden gebruikt
    IFontData[] fontDatas = pres.FontsManager.GetFonts();

    // Haal de byte-array op die de reguliere stijl van het eerste lettertype in de presentatie weergeeft
    byte[] bytes = pres.FontsManager.GetFontBytes(fontDatas[0], FontStyle.Regular);

    // Bepaal het insluitingsniveau van het lettertype
    EmbeddingLevel embeddingLevel = pres.FontsManager.GetFontEmbeddingLevel(bytes, fontDatas[0].FontName);
}
```

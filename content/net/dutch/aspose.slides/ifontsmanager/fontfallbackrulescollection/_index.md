## IFontsManager.FontFallBackRulesCollection eigenschap

Stelt de collectie van FontFallBack-regels van een gebruiker voor voor het beheren van lettertypecollecties voor juiste substituties via fallback-functionaliteit Lezen/Schrijven [`IFontFallBackRulesCollection`](../../ifontfallbackrulescollection).

```csharp
public IFontFallBackRulesCollection FontFallBackRulesCollection { get; set; }
```

### Voorbeelden

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    // Ophalen van een lege of voorgeïnitialiseerde regelsverzameling van FontsManager
    IFontFallBackRulesCollection rulesList = pres.FontsManager.FontFallBackRulesCollection;

    // toevoegen van regels aan de verzameling
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));

    // of 
    // initialisatie van een nieuwe instantie van de regelsverzameling
    IFontFallBackRulesCollection rulesList = new FontFallBackRulesCollection();

    // toevoegen van regels aan de verzameling
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));

    // en vervangen van de bestaande verzameling door de nieuwe in FontsManager 
    pres.FontsManager.FontFallBackRulesCollection = rulesList;
}
```

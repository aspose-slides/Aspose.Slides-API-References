## IFontsManager.GetFontBytes methode

Haalt de byte-array op die de lettertypegegevens vertegenwoordigt voor een opgegeven lettertype-stijl en lettertypegegevens.

```csharp
public byte[] GetFontBytes(IFontData fontData, FontStyleType fontStyle)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| fontData | IFontData | Het lettertype-gegevensobject dat de informatie over het lettertype [`IFontData`](../../ifontdata) bevat. |
| fontStyle | FontStyleType | De stijl van het lettertype waarvoor de gegevens moeten worden opgehaald [`FontStyleType`](../../fontstyletype). |

### Returnwaarde

Een byte-array die de lettertypegegevens bevat voor de opgegeven lettertype-stijl. Als de lettertype-gegevens of -stijl niet worden gevonden, wordt null geretourneerd.

### Voorbeelden

```csharp
[C#]
using (Presentation pres = new Presentation ("Presentation.pptx"))
{
    // Alle lettertypen ophalen die in de presentatie worden gebruikt
    IFontData[] fonts = pres.FontsManager.GetFonts();

    // De byte-array ophalen die de reguliere stijl van het eerste lettertype in de presentatie weergeeft
    byte[] bytes = pres.FontsManager.GetFontBytes(fonts[0], FontStyleType.Regular);
}
```

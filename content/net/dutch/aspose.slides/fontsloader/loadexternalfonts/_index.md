## FontsLoader.LoadExternalFonts methode

Voegt extra mappen toe om naar lettertypen te zoeken.

```csharp
public static void LoadExternalFonts(string[] directories)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| directories | String[] | Mappen om extra lettertypen te lezen. |

### Voorbeelden

De volgende voorbeelden tonen hoe aangepaste lettertypen uit .TTF te laden.

```csharp
[C#]
// Het pad naar de documentenmap.
string dataDir = "C:\\";
// mappen om lettertypen te zoeken
String[] folders = new String[] { dataDir };
// Laad de aangepaste lettertypemap
FontsLoader.LoadExternalFonts(folders);
// Doe wat werk en render de presentatie/diavoorstellingen
using (Presentation presentation = new Presentation(dataDir + "DefaultFonts.pptx"))
{
    presentation.Save(dataDir + "NewFonts_out.pptx", SaveFormat.Pptx);
	// Wis Lettertypecache
	FontsLoader.ClearCache();
}
```

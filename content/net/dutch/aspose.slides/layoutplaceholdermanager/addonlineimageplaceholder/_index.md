## LayoutPlaceholderManager.AddOnlineImagePlaceholder methode

Voegt een nieuw placeholder-vorm toe aan de layout-dia om een online-afbeelding vast te houden.

```csharp
public IAutoShape AddOnlineImagePlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| x | Single | De X-coördinaat van de nieuwe placeholder-vorm. |
| y | Single | De Y-coördinaat van de nieuwe placeholder-vorm. |
| width | Single | De breedte van de nieuwe placeholder-vorm. |
| height | Single | De hoogte van de nieuwe placeholder-vorm. |

### Retourwaarde

Aangemaakt [`IAutoShape`](../../iautoshape) met een Online Image-placeholder.

### Voorbeelden

Het volgende voorbeeld laat zien hoe de Online Image-placeholder-vorm aan de layout-dia kan worden toegevoegd.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddOnlineImagePlaceholder(20, 20, 200, 200);
}
```

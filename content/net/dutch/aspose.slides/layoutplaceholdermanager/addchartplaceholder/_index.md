## LayoutPlaceholderManager.AddChartPlaceholder methode

Voegt een nieuw placeholder-vorm toe aan de layout-dia om een grafiek te bevatten.

```csharp
public IAutoShape AddChartPlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| x | Single | De X-coördinaat van de nieuwe placeholder-vorm. |
| y | Single | De Y-coördinaat van de nieuwe placeholder-vorm. |
| width | Single | De breedte van de nieuwe placeholder-vorm. |
| height | Single | De hoogte van de nieuwe placeholder-vorm. |

### Retourwaarde

Aangemaakt [`IAutoShape`](../../iautoshape) met een Chart placeholder.

### Voorbeelden

Het volgende voorbeeld laat zien hoe u de Chart-placeholder-vorm toevoegt aan de layout-dia.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddChartPlaceholder(20, 20, 200, 200);
}
```

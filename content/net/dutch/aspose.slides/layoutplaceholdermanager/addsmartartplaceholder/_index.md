## LayoutPlaceholderManager.AddSmartArtPlaceholder method

Voegt een nieuw placeholder-vorm toe aan de lay-outdia om een SmartArt-diagram te bevatten.

```csharp
public IAutoShape AddSmartArtPlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| x | Single | De X-coördinaat van de nieuwe placeholder-vorm. |
| y | Single | De Y-coördinaat van de nieuwe placeholder-vorm. |
| width | Single | De breedte van de nieuwe placeholder-vorm. |
| height | Single | De hoogte van de nieuwe placeholder-vorm. |

### Retourwaarde

Gemaakt [`IAutoShape`](../../iautoshape) met een SmartArt-placeholder.

### Voorbeelden

Het volgende voorbeeld laat zien hoe u de SmartArt-placeholder-vorm toevoegt aan de lay-outdia.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddSmartArtPlaceholder(20, 20, 200, 200);
}
```

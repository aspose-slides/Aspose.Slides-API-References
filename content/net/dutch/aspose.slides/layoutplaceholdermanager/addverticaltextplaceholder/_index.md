## LayoutPlaceholderManager.AddVerticalTextPlaceholder method

Voegt een nieuw placeholder-vorm toe aan de layout-slide om tekstinhoud in verticale richting vast te houden.

```csharp
public IAutoShape AddVerticalTextPlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| x | Single | De X-coördinaat van de nieuwe placeholder-vorm. |
| y | Single | De Y-coördinaat van de nieuwe placeholder-vorm. |
| width | Single | De breedte van de nieuwe placeholder-vorm. |
| height | Single | De hoogte van de nieuwe placeholder-vorm. |

### Retourwaarde

Aangemaakt [`IAutoShape`](../../iautoshape) met een Text (Vertical) placeholder.

### Voorbeelden

Het volgende voorbeeld toont hoe een Text (Vertical) placeholder-vorm aan de layout-slide kan worden toegevoegd.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalTextPlaceholder(20, 20, 300, 500);
}
```

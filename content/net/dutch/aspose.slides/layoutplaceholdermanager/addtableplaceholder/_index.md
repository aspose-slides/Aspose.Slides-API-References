## LayoutPlaceholderManager.AddTablePlaceholder methode

Voegt een nieuwe placeholdervorm toe aan de layout slide om een tabel te bevatten.

```csharp
public IAutoShape AddTablePlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| x | Single | De X-coördinaat van de nieuwe placeholdervorm. |
| y | Single | De Y-coördinaat van de nieuwe placeholdervorm. |
| width | Single | De breedte van de nieuwe placeholdervorm. |
| height | Single | De hoogte van de nieuwe placeholdervorm. |

### Retourwaarde

Aangemaakt [`IAutoShape`](../../iautoshape) met een Table placeholder.

### Voorbeelden

Het volgende voorbeeld toont hoe de Table placeholdervorm aan de layout slide kan worden toegevoegd.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTablePlaceholder(20, 20, 500, 200);
}
```

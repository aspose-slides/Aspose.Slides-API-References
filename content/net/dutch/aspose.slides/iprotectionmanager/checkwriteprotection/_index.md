## IProtectionManager.CheckWriteProtection methode

Bepaalt of een presentatie met een wachtwoord beveiligd is voor bewerking.

```csharp
public bool CheckWriteProtection(string password)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| password | String | Het wachtwoord voor controle. |

### Retourwaarde

True als het wachtwoord geldig is; anders false.

### Opmerkingen

1. U moet de [`IsWriteProtected`](../iswriteprotected) eigenschap controleren voordat u deze methode aanroept. 2. Wanneer het wachtwoord null of leeg is, retourneert deze methode false.

### Voorbeelden

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

## IViewProperties.GridSpacing eigenschap

Geeft de rasterafstand terug of stelt deze in die moet worden gebruikt voor het raster dat ten grondslag ligt aan het presentatiedocument, in punten. Lezen/Schrijven Single.

```csharp
public float GridSpacing { get; set; }
```

### Opmerkingen

De rasterafstandwaarde moet een positief getal zijn. Het typische bereik is van 1 mm (2.8349607 punten) tot 2 inch (144 punten).

### Voorbeelden

De volgende voorbeeldcode toont hoe u de rasterafstand kunt wijzigen in een PowerPoint-presentatie.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

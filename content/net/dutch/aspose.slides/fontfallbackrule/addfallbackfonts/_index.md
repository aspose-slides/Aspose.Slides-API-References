## AddFallBackFonts(string) {#addfallbackfonts}

Voegt een nieuw lettertype(s) toe aan de lijst met FallBack-lettertypen.

```csharp
public void AddFallBackFonts(string fontName)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| fontName | String | Font's name or names (delimited by comma) for FallBack |

### Voorbeelden

```csharp
[C#]
// Maak een nieuw exemplaar van FontFallBackRule
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho");

//Voeg een tweede lettertype toe aan de regel 
newRule.AddFallBackFonts("MS Gothic");
//Voeg een derde en vierde lettertype toe aan de regel 
newRule.AddFallBackFonts("Tahoma, Times New Roman");
```

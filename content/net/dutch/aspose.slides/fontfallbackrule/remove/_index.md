## FontFallBackRule.Remove methode

Verwijdert de eerste overeenkomst van een specifiek fallback-lettertype uit de lijst.

```csharp
public void Remove(string fontName)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| fontName | String | De naam van het lettertype dat uit de lijst moet worden verwijderd. |

### Voorbeelden

```csharp
[C#]
// Maak een regel die een lijst met lettertypen bevat.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// Verwijder Tahoma uit de lijst.
newRule.Remove ("Tahoma");
```

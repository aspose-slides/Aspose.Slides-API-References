## IShapeCollection.InsertSummaryZoomFrame methode

Maakt een nieuw Summary Zoom-frame en voegt het toe aan de shape-collectie op de opgegeven index.

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| index | Int32 | De nulgebaseerde index waarop het Summary Zoom-frame moet worden ingevoegd. |
| x | Single | De x-coördinaat van het nieuwe Summary Zoom-frame, in punten. |
| y | Single | De y-coördinaat van het nieuwe Summary Zoom-frame, in punten. |
| width | Single | De breedte van het nieuwe Summary Zoom-frame, in punten. |
| height | Single | De hoogte van het nieuwe Summary Zoom-frame, in punten. |

### Retourwaarde

Het nieuw aangemaakte [`ISummaryZoomFrame`](../../isummaryzoomframe).

### Uitzonderingen

| uitzondering | conditie |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | Wordt gegooid als de presentatie geen secties bevat, of als de doeldia niet tot een sectie behoort. |

### Opmerkingen

Deze methode maakt een Summary Zoom-frame dat samenvattingskoppelingen voor alle secties in de presentatie aggregeert.

### Voorbeelden

Dit voorbeeld laat zien hoe een Summary Zoom-object wordt aangemaakt en ingevoegd op de opgegeven index van een collectie (veronderstel dat er minimaal twee secties in de "Presentation.pptx"-presentatie aanwezig zijn):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSummaryZoomFrame(2, 150, 20, 50, 50);
}
```

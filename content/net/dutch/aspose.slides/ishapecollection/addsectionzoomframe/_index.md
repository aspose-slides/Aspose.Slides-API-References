## AddSectionZoomFrame(float, float, float, float, ISection) {#addsectionzoomframe}

Maakt een nieuw Section Zoom frame en voegt het toe aan het einde van de vormverzameling.

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| x | Single | De x-coördinaat van het nieuwe Section Zoom frame, in punten. |
| y | Single | De y-coördinaat van het nieuwe Section Zoom frame, in punten. |
| width | Single | De breedte van het nieuwe Section Zoom frame, in punten. |
| height | Single | De hoogte van het nieuwe Section Zoom frame, in punten. |
| section | ISection | De [`ISection`](../../isection) waarnaar verwezen wordt door het Section Zoom frame; moet tot deze presentatie behoren en minstens één dia bevatten. |

### Retourwaarde

Het nieuw aangemaakte [`ISectionZoomFrame`](../../isectionzoomframe).

### Uitzonderingen

| Uitzondering | Voorwaarde |
| --- | --- |
| ArgumentException | Wordt opgegooid wanneer de verwezen sectie niet tot de huidige presentatie behoort of geen dia's bevat. |

### Voorbeelden

Dit voorbeeld laat zien hoe een Section Zoom object wordt toegevoegd aan het einde van een collectie (ga uit van ten minste twee secties in de presentatie "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1]);
}
```

## AddSectionZoomFrame(float, float, float, float, ISection, IPPImage) {#addsectionzoomframe_1}

Maakt een nieuw Section Zoom frame met een vooraf gedefinieerde afbeelding en voegt het toe aan het einde van de vormverzameling.

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section, IPPImage image)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| x | Single | De x-coördinaat van het nieuwe Section Zoom frame, in punten. |
| y | Single | De y-coördinaat van het nieuwe Section Zoom frame, in punten. |
| width | Single | De breedte van het nieuwe Section Zoom frame, in punten. |
| height | Single | De hoogte van het nieuwe Section Zoom frame, in punten. |
| section | ISection | De [`ISection`](../../isection) waarnaar verwezen wordt door het Section Zoom frame; moet tot deze presentatie behoren en minstens één dia bevatten. |
| image | IPPImage | De [`IPPImage`](../../ippimage) die binnen het Section Zoom frame moet worden weergegeven. |

### Retourwaarde

Het nieuw aangemaakte [`ISectionZoomFrame`](../../isectionzoomframe).

### Uitzonderingen

| Uitzondering | Voorwaarde |
| --- | --- |
| ArgumentException | Wordt opgegooid wanneer de verwezen sectie niet tot de huidige presentatie behoort of geen dia's bevat. |

### Voorbeelden

Dit voorbeeld laat zien hoe een Section Zoom object wordt toegevoegd aan het einde van een collectie (ga uit van ten minste twee secties in de presentatie "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IPPImage image = pres.Images.AddImage(Image.FromFile("image.png"));
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1], image);
}
```

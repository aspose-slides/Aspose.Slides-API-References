## InsertAudioFrameEmbedded(int, float, float, float, float, Stream) {#insertaudioframeembedded_1}

Maakt een nieuw audio-frame met een ingesloten WAV-bestand en voegt het in de vormverzameling in op de opgegeven index. Het ingesloten audio-bestand wordt toegevoegd aan de Presentation.Audios-verzameling.

```csharp
public IAudioFrame InsertAudioFrameEmbedded(int index, float x, float y, float width, float height, 
    Stream audio_stream)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| index | Int32 | De nulgebaseerde index waarop het audio-frame moet worden ingevoegd. |
| x | Single | De x-coördinaat van het nieuwe audio-frame, in punten. |
| y | Single | De y-coördinaat van het nieuwe audio-frame, in punten. |
| width | Single | De breedte van het nieuwe audio-frame, in punten. |
| height | Single | De hoogte van het nieuwe audio-frame, in punten. |
| audio_stream | Stream | Een invoer-stream die WAV-audiodata bevat om in te sluiten. |

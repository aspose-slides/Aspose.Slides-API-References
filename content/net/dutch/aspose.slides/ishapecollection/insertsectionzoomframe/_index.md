## InsertSectionZoomFrame(int, float, float, float, float, ISection) {#insertsectionzoomframe}

Maakt een nieuw Section Zoom frame en voegt het in de shape collection in op de opgegeven index.

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| index | Int32 | De nulgebaseerde index waarop de Section Zoom frame moet worden ingevoegd. |
| x | Single | De x-coördinaat van het nieuwe Section Zoom frame, in points. |
| y | Single | De y-coördinaat van het nieuwe Section Zoom frame, in points. |
| width | Single | De breedte van het nieuwe Section Zoom frame, in points. |
| height | Single | De hoogte van het nieuwe Section Zoom frame, in points. |
| section | ISection | De [`ISection`](../../isection) waarnaar wordt verwezen door het Section Zoom frame; moet tot deze presentatie behoren en ten minste één dia bevatten. |

### Return Value

Het nieuw aangemaakte [`ISectionZoomFrame`](../../isectionzoomframe).

### Exceptions

| Uitzondering | Conditie |
| --- | --- |
| ArgumentException | Wordt opgegooid als de gerefereerde sectie niet tot de huidige presentatie behoort of geen dia's bevat. |

### Examples

Dit voorbeeld toont het maken en invoegen van een Section Zoom object op de opgegeven index van een collectie (ga er vanuit dat er minstens twee secties zijn in de presentatie "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSectionZoomFrame(2, 150, 20, 50, 50, pres.Sections[1]);
}
```

public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width,

| Parameter | Type | Beschrijving |

### Return Value

}

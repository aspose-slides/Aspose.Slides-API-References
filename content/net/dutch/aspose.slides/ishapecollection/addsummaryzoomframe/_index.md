## IShapeCollection.AddSummaryZoomFrame methode

Creëert een nieuw Summary Zoom frame en voegt het toe aan het einde van de vormverzameling.

```csharp
public ISummaryZoomFrame AddSummaryZoomFrame(float x, float y, float width, float height)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| x | Single | De x-coördinaat van het nieuwe Summary Zoom frame, in punten. |
| y | Single | De y-coördinaat van het nieuwe Summary Zoom frame, in punten. |
| width | Single | De breedte van het nieuwe Summary Zoom frame, in punten. |
| height | Single | De hoogte van het nieuwe Summary Zoom frame, in punten. |

### Retourwaarde

Het nieuw aangemaakte [`ISummaryZoomFrame`](../../isummaryzoomframe).

### Uitzonderingen

| Uitzondering | Voorwaarde |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | Wordt gegooid als er geen secties in de presentatie zijn, of als de doel-dia niet tot een sectie behoort. |

### Opmerkingen

Deze methode maakt een Summary Zoom frame dat samenvattingslinks voor alle secties in de presentatie verzamelt.

### Voorbeelden

Dit voorbeeld laat zien hoe een Summary Zoom object aan het einde van een verzameling wordt toegevoegd (ga uit van ten minste twee secties in de presentatie "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSummaryZoomFrame(150, 20, 500, 250);
}
```

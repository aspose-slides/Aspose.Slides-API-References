## InsertAutoShape(int, ShapeType, float, float, float, float, bool) {#insertautoshape_1}

Maakt een nieuwe auto-shape en voegt deze in de shape-collectie in op de opgegeven index, met de mogelijkheid om deze te initialiseren met de standaard template-styling.

```csharp
public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| index | Int32 | De nulgebaseerde index waarop de auto-shape moet worden ingevoegd. |
| shapeType | ShapeType | De [`ShapeType`](../../shapetype) van de auto-shape die moet worden ingevoegd. |
| x | Single | De x-coördinaat van het frame van de shape, in punten. |
| y | Single | De y-coördinaat van het frame van de shape, in punten. |
| width | Single | De breedte van het frame van de shape, in punten. |
| height | Single | De hoogte van het frame van de shape, in punten. |
| createFromTemplate | Boolean | True om de standaard template-styling toe te passen (inclusief een niet-lege naam, eenvoudige stijl en gecentreerde tekst); false om de shape te maken met alle eigenschappen ingesteld op hun standaardwaarden. |

## AddZoomFrame(float, float, float, float, ISlide) {#addzoomframe}

Maakt een nieuw Zoom-frame en voegt het toe aan het einde van de vormverzameling.

```csharp
public IZoomFrame AddZoomFrame(float x, float y, float width, float height, ISlide slide)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| x | Single | De x-coördinaat van het nieuwe Zoom-frame, in punten. |
| y | Single | De y-coördinaat van het nieuwe Zoom-frame, in punten. |
| width | Single | De breedte van het nieuwe Zoom-frame, in punten. |
| height | Single | De hoogte van het nieuwe Zoom-frame, in punten. |
| slide | ISlide | De [`ISlide`](../../islide) waarnaar verwezen wordt door het Zoom-frame; moet tot deze presentatie behoren. |

### Retourwaarde

Het nieuw aangemaakte [`IZoomFrame`](../../izoomframe).

### Excepties

| exception | condition |
| --- | --- |
| ArgumentException | Wordt gegooid als de aangewezen slide niet tot de huidige presentatie behoort. |

### Voorbeelden

Dit voorbeeld toont het toevoegen van een Zoom-object aan het einde van een collectie (ga uit van ten minste twee dia's in de presentatie "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
}
```

## OleObjectFrame.SetEmbeddedData methode

Stelt informatie over OLE ingesloten gegevens in. Deze methode wijzigt de eigenschappen van het object zodat ze overeenkomen met de nieuwe gegevens en zet de IsObjectLink-vlag op false, wat aangeeft dat het OLE-object is ingesloten.

```csharp
public void SetEmbeddedData(IOleEmbeddedDataInfo embeddedData)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| embeddedData | IOleEmbeddedDataInfo | Ingesloten gegevens [`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo) |

### Uitzonderingen

| uitzondering | conditie |
| --- | --- |
| ArgumentNullException | Wanneer de embeddedData-parameter null is. |

### Voorbeelden

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    OleObjectFrame oof = pres.Slides[0].Shapes[0] as OleObjectFrame;
    if (oof != null)
    {
        IOleEmbeddedDataInfo newData = new OleEmbeddedDataInfo(File.ReadAllBytes("Picture.png"), "png");
        oof.SetEmbeddedData(newData);
    }
}
```

## IInkBrush.InkEffect eigenschap

Haalt het type inkeffect op (bijvoorbeeld Galaxy, Gold, Silver) dat de visuele stijl van de inktstreep definieert. De waarde wordt geparseerd uit de brush-eigenschap "inkEffects". Als er geen herkend effect is opgegeven, wordt NotDefined geretourneerd.

```csharp
public InkEffectType InkEffect { get; }
```

### Voorbeelden

Voorbeeld:

```csharp
[C#]
using (var pres = new Presentation("Presentation.pptx"))
{
    Ink ink = pres.Slides[0].Shapes[0] as Ink;
    IInkBrush brush = ink.Traces[0].Brush;
    Console.WriteLine("InkEffects = {0}", brush.InkEffect);
}
```

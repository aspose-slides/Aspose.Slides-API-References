## Ink.InkEffectImages eigenschap

Haalt de collectie van aangepaste afbeeldingen op die worden gebruikt om visuele effecten voor inktpenseel te simuleren. Deze afbeeldingen worden gebruikt bij het renderen van inkt met specifieke [`InkEffectType`](../../inkeffecttype) waarden, zoals Galaxy, Rainbow, enz. Door uw eigen afbeeldingen te leveren, kunt u regelen hoe elk inkt-effect verschijnt.

```csharp
public static IDictionary<InkEffectType, IImage> InkEffectImages { get; }
```

### Opmerkingen

Deze eigenschap maakt het mogelijk om de standaard inkt-effect texturen te vervangen door door de gebruiker gedefinieerde, wat vooral nuttig is wanneer standaard assets beperkt zijn door licenties of niet beschikbaar zijn tijdens runtime. Elke invoer in de dictionary moet een [`InkEffectType`](../../inkeffecttype) waarde koppelen aan een overeenkomstig [`IImage`](../../../aspose.slides/iimage) object (bijv. Bitmap, of een Aspose-afbeelding-interface).

### Voorbeelden

```csharp
[IImage image = Images.FromFile("image.png");
Ink.InkEffectImages.Add(InkEffectType.Galaxy, image);
```

## PresentationPlayer.FrameTick event

Frame-tick-event. Vindt plaats wanneer elk frame van de animatie die door [`PresentationAnimationsGenerator`](../../presentationanimationsgenerator) is aangemaakt, door de speler wordt gegenereerd.

```csharp
public event FrameTickHandler FrameTick;
```

### Voorbeelden

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (var animationsGenerator = new PresentationAnimationsGenerator(pres))
    using (var player = new PresentationPlayer(animationsGenerator, 33))
    {
        player.FrameTick += (sender, args) =>
        {
            args.GetFrame().Save($"frame_{sender.FrameIndex}.png");
        };
        
        animationsGenerator.Run(pres.Slides);
    }
}
```

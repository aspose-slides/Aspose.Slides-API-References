## MarkdownExportType enumeratie

Type van documentweergave.

```csharp
public enum MarkdownExportType
```

### Waarden

| Naam | Waarde | Beschrijving |
| --- | --- | --- |
| Sequential | `0` | Render alle items afzonderlijk. Eén voor één. |
| TextOnly | `1` | Render alleen tekst. |
| Visual | `2` | Render alle items, items die gegroepeerd zijn - samen renderen. |

### Voorbeelden

Voorbeeld:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    MarkdownSaveOptions markdownSaveOptions = new MarkdownSaveOptions
    {
        ShowHiddenSlides = true,
        ShowSlideNumber = true,
        Flavor = Flavor.Github,
        ExportType = MarkdownExportType.Sequential,
        NewLineType = NewLineType.Windows
    };
    
    pres.Save("doc.md", new[] { 1, 2, 3, 4, 5, 6, 7, 8, 9 }, SaveFormat.Md, markdownSaveOptions);
```

## RenderingOptions.DisableFontLigatures eigenschap

Geeft of stelt een waarde in die aangeeft of tekst wordt gerenderd zonder ligaturen te gebruiken. Wanneer ingesteld op `true`, worden ligaturen uitgeschakeld in de gerenderde uitvoer. Standaard is deze eigenschap ingesteld op `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Voorbeelden

Voorbeeld:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    RenderingOptions options = new RenderingOptions
    {
        DisableFontLigatures = true // Schakel ligaturen uit bij het renderen van tekst
    };
    
    Bitmap[] renderedSlides = pres.GetImage(options);
    for (var index = 0; index < renderedSlides.Length; index++)
    {
        var slideImage = renderedSlides[index];
        slideImage.Save($"slide-{index}.png");
    }
}
```

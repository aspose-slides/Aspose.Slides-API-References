## IPptxOptions.CompressionLevel eigenschap

Specificeert het compressieniveau dat wordt gebruikt bij het opslaan van het presentatiedocument. De standaardwaarde is Level6.

```csharp
public CompressionLevel CompressionLevel { get; set; }
```

### Opmerkingen

Hogere compressieniveaus produceren kleinere bestanden, maar vereisen meer verwerkingstijd. De werkelijke compressieverhouding hangt af van de inhoud van de presentatie.

### Voorbeelden

Voorbeeld:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("demo-level8.pptx", SaveFormat.Pptx, new PptxOptions()
    {
        CompressionLevel = CompressionLevel.Level8
    });
}
```

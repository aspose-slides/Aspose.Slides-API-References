## ISwfOptions.SlidesLayoutOptions eigenschap

Geeft de modus terug of stelt deze in waarin dia's op de pagina worden geplaatst bij het exporteren van een presentatie [`ISlidesLayoutOptions`](../../islideslayoutoptions). Deze eigenschap ondersteunt het toewijzen van objecten van het type `Aspose.Slides.Export.HandoutLayoutingOptions` niet.

```csharp
public ISlidesLayoutOptions SlidesLayoutOptions { get; set; }
```

### Voorbeelden

Example:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    SwfOptions options = new SwfOptions
    {
        SlidesLayoutOptions = new NotesCommentsLayoutingOptions
        {
            CommentsPosition = CommentsPositions.Right
        }
    };
    
    pres.Save("pres.swf", SaveFormat.Swf, options);
}
```

## HtmlOptions.DisableFontLigatures eigenschap

Geeft een waarde terug of stelt een waarde in die aangeeft of tekst wordt gerenderd zonder ligaturen te gebruiken. Wanneer ingesteld op `true`, worden ligaturen uitgeschakeld in de gerenderde uitvoer. Standaard is deze eigenschap ingesteld op `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Voorbeelden

Voorbeeld:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    HtmlOptions options = new HtmlOptions
    {
        DisableFontLigatures = true // Schakel ligaturen uit bij het renderen van tekst
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html, options);
}
```

## SwfOptions.SlidesLayoutOptions eigenschap

Haalt op of stelt de modus in waarin dia's op de pagina worden geplaatst bij het exporteren van een presentatie [`ISlidesLayoutOptions`](../../islideslayoutoptions). Deze eigenschap ondersteunt het toewijzen van objecten van type [`HandoutLayoutingOptions`](../../handoutlayoutingoptions) niet.

```csharp
public ISlidesLayoutOptions SlidesLayoutOptions { get; set; }
```

### Voorbeelden

Voorbeeld:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    SwfOptions options = new SwfOptions
    {
        SlidesLayoutOptions = new NotesCommentsLayoutingOptions
        {
            CommentsPosition = CommentsPositions.Right
        }
    };
    
    pres.Save("pres.swf", SaveFormat.Swf, options);
}
```

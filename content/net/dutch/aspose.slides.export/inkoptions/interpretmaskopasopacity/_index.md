## InkOptions.InterpretMaskOpAsOpacity eigenschap

Gebruikt ROP-bewerking of Opacity voor het renderen van de penseel.

```csharp
public bool InterpretMaskOpAsOpacity { get; set; }
```

### Opmerkingen

Standaardwaarde is true.

### Voorbeelden

Het volgende voorbeeld laat zien hoe je ROP instelt voor het exporteren van Ink-elementen:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.InterpretMaskOpAsOpacity = false;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

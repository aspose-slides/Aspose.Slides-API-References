## IInkOptions.HideInk eigenschap

Toont of verbergt Ink-elementen in het geëxporteerde document.

```csharp
public bool HideInk { get; set; }
```

### Opmerkingen

Standaardwaarde is false.

### Voorbeelden

Het volgende voorbeeld laat zien hoe Ink-elementen in een geëxporteerd PDF-document verborgen kunnen worden:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.HideInk = true;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

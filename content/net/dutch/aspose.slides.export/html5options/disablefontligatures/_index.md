## Html5Options.DisableFontLigatures eigenschap

Haalt een waarde op of stelt een waarde in die aangeeft of tekst wordt weergegeven zonder ligaturen te gebruiken. Als dit op `true` is ingesteld, worden ligaturen uitgeschakeld in de gerenderde output. Standaard is deze eigenschap ingesteld op `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Voorbeelden

Voorbeeld:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    Html5Options options = new Html5Options
    {
        DisableFontLigatures = true // Schakel ligaturen uit bij weergave van tekst
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html5, options);
}
```

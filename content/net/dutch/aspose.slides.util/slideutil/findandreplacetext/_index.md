## SlideUtil.FindAndReplaceText methode

Zoekt en vervangt tekst in de presentatie met het opgegeven formaat

```csharp
public static void FindAndReplaceText(IPresentation presentation, bool withMasters, string find, 
    string replace, PortionFormat format = null)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| presentation | IPresentation | Doorzochte presentatie. |
| withMasters | Boolean | Bepaalt of masterdia's moeten worden doorzocht. |
| find | String | String-waarde om te vinden. |
| replace | String | String-waarde om te vervangen. |
| format | PortionFormat | Formaat voor het vervangen van een tekstdelen. Als null, wordt het formaat van het eerste teken van de gevonden tekenreeks gebruikt. |

### Voorbeelden

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PortionFormat format = new PortionFormat
    {
        FontHeight = 24f,
        FontItalic = NullableBool.True,
        FillFormat =
        {
            FillType = FillType.Solid,
            SolidFillColor =
            {
                Color = Color.Red
            }
        }
    };
    Aspose.Slides.Util.SlideUtil.FindAndReplaceText(pres, true, "[this block] ", "my text ", format);
    pres.Save("replaced", SaveFormat.Pptx);
}
```

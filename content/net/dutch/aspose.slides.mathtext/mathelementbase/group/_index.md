## Group(char, MathTopBotPositions, MathTopBotPositions) {#group_1}

Plaatst dit element in een groep met een groepeerteken, zoals een onderste accolade of een ander teken

```csharp
public IMathGroupingCharacter Group(char character, MathTopBotPositions position, 
    MathTopBotPositions verticalJustification)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| character | Char | Groeperend teken, zoals BOTTOM CURLY BRACKET (U+23DF) of een ander |
| position | MathTopBotPositions | Positie van het groepeerteken |
| verticalJustification | MathTopBotPositions | Vertical justification van het groepeerteken. Geeft de uitlijning van het object ten opzichte van de basislijn aan. Bijvoorbeeld, wanneer het groepeerteken zich boven het object bevindt, VerticalJustification of Top betekent dat de bovenkant van het object op de basislijn ligt; wanneer VerticalJustification op Bottom is ingesteld, ligt de onderkant van het object op de basislijn |

### Retourwaarde

New instance of type [`IMathGroupingCharacter`](../../imathgroupingcharacter)

### Voorbeelden

Voorbeeld:

```csharp
[C#]
```

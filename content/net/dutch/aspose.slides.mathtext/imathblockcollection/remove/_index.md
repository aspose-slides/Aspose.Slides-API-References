## IMathBlockCollection.Remove methode

Verwijdert de eerste instantie van een specifiek object uit de collectie/&gt;.

```csharp
public bool Remove(IMathBlock item)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| item | IMathBlock | Het object dat uit de collectie moet worden verwijderd. |

### Retourwaarde

true als *item* succesvol is verwijderd uit de collectie; anders false. Deze methode geeft ook false terug als *item* niet wordt gevonden in de oorspronkelijke collectie/&gt;.

### Voorbeelden

Voorbeeld:

```csharp
[C#]
IMathBlockCollection blockCollection = new MathParagraph();
IMathBlock block = new MathBlock(new MathematicalText("y"));
blockCollection.Add(block);
blockCollection.Remove(block);
```

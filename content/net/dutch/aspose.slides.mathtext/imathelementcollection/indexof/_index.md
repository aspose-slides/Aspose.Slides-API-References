## IMathElementCollection.IndexOf method

Bepaalt de index van een specifiek wiskundig element in de verzameling.

```csharp
public int IndexOf(IMathElement item)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| item | IMathElement | Het element dat moet worden opgezocht in de verzameling. |

### Retourwaarde

De index van *item* als deze in de verzameling wordt gevonden; anders -1.

Voorbeeld:


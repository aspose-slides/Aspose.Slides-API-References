## IMathElementCollection.Remove methode

Verwijdert de eerste voorkomen van een specifiek object uit de collectie.

```csharp
public bool Remove(IMathElement item)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| item | IMathElement | Het object dat uit de collectie moet worden verwijderd. |

### Retourwaarde

true als *item* met succes uit de collectie is verwijderd; anders false. Deze methode geeft ook false terug als *item* niet wordt gevonden in de oorspronkelijke collectie.

### Voorbeelden

Voorbeeld:

```csharp
[C#]
IMathElementCollection collection = new MathBlock(new MathematicalText("x"));
MathematicalText plusElement = new MathematicalText("+");
collection.Add(plusElement);
collection.Add(new MathRadical(new MathematicalText("x"), new MathematicalText("3")));
collection.Remove(plusElement);
```

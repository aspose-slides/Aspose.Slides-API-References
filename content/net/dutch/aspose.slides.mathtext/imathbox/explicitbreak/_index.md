## IMathBox.ExplicitBreak eigenschap

Expliciete onderbreking geeft aan of er een regeleinde is aan het begin van het Box-object, zodat de regel wordt afgebroken aan het begin van het box-object. Geeft het nummer van de operator op de vorige regel van wiskundige tekst op die moet worden gebruikt als uitlijningspunt voor de huidige regel van wiskundige tekst. Mogelijke waarden: 1..255 Standaard: 0 (geen expliciete onderbreking)

```csharp
public byte ExplicitBreak { get; set; }
```

Voorbeeld:

[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.ExplicitBreak = 1;

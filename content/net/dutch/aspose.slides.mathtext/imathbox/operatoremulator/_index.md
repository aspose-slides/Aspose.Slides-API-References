## IMathBox.OperatorEmulator eigenschap

Operator Emulator. Wanneer true, gedraagt de box en de inhoud ervan zich als één enkele operator en erven ze de eigenschappen van een operator. Dit betekent bijvoorbeeld dat het teken kan dienen als een punt voor een regelonderbreking en kan worden uitgelijnd met andere operators. Operator Emulators worden vaak gebruikt wanneer een of meer glyphs combineren om een operator te vormen, zoals '=='. Standaardwaarde: false

```csharp
public bool OperatorEmulator { get; set; }
```

### Voorbeelden

Voorbeeld:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.OperatorEmulator = true;
```

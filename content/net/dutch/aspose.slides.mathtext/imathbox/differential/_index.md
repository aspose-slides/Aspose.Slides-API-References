## IMathBox.Differential eigenschap

Differential. Wanneer true, functioneert de box als een differentiaal (bijv., 𝑑𝑥 in een integrand), en krijgt de passende horizontale spatiëring voor het wiskundige differentiaal. Standaard: false

public bool Differential { get; set; }

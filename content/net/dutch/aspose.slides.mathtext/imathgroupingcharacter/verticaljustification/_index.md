## IMathGroupingCharacter.VerticalJustification eigenschap

Verticale uitlijning van groepskarakter. Specificeert de uitlijning van het object ten opzichte van de basislijn. Bijvoorbeeld, wanneer het groepskarakter boven het object staat, betekent VerticalJustification van Top dat de bovenkant van het object op de basislijn ligt; wanneer VerticalJustification is ingesteld op Bottom, ligt de onderkant van het object op de basislijn. Standaard: Bottom voor Position=Top, en Top voor Position=Bottom

```csharp
public MathTopBotPositions VerticalJustification { get; set; }
```

### Voorbeelden

Voorbeeld:

```csharp
[C#]
MathGroupingCharacter groupingCharacter = new MathGroupingCharacter(new MathematicalText("abc"));
groupingCharacter.VerticalJustification = MathTopBotPositions.Top;
```

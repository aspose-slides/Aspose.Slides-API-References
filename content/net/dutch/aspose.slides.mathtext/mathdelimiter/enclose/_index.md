## MathDelimiter.Enclose methode

Omvat een wiskundig element in opgegeven tekens, zoals haakjes of andere tekens als kader

```csharp
public override IMathDelimiter Enclose(char beginningCharacter, char endingCharacter)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| beginningCharacter | Char | Beginteken (meestal linker haakje) |
| endingCharacter | Char | Eindteken (meestal rechter haakje) |

### Retourwaarde

Als *beginningCharacter* en *endingCharacter* null zijn, krijgen de overeenkomende eigenschappen alleen waarden toegewezen en wordt er geen nieuw object aangemaakt (retourneert deze instantie). Anders wordt een nieuw wiskundig element van type Delimiter geretourneerd dat de opgegeven tekens als kader bevat en deze instantie van [`MathDelimiter`](../../mathdelimiter) ingesloten.

### Voorbeelden

Voorbeeld:

```csharp
[C#]
IMathDelimiter innerDelimiter = new MathematicalText("x").Join(",y").Enclose('{', '}');
IMathDelimiter outerDelimiter = innerDelimiter.Enclose('[', ']');
```

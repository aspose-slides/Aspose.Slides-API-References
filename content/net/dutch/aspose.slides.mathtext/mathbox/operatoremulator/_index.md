## MathBox.OperatorEmulator eigenschap

Operator Emulator. Wanneer true, vertonen de doos en de inhoud zich als één operator en erven ze de eigenschappen van een operator. Dit betekent bijvoorbeeld dat het teken kan dienen als een punt voor een regeleinde en kan worden uitgelijnd met andere operatoren. Operator Emulators worden vaak gebruikt wanneer één of meer glyphs combineren tot een operator, zoals '=='. Standaardwaarde: false

```csharp
public bool OperatorEmulator { get; set; }
```

### Voorbeelden

Voorbeeld:

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("=="));
box.OperatorEmulator = true;
```

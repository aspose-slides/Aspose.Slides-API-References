## MathBox.NoBreak eigenschap

No break Deze eigenschap specificeert de "unbreakable" eigenschap op de objectdoos. Wanneer true, kunnen er geen regelafbrekingen binnen de doos optreden. Dit kan belangrijk zijn voor operator-emulators die uit meer dan één binaire operator bestaan. Wanneer dit element niet gespecificeerd is, kunnen er afbrekingen binnen de doos optreden. Default: true

```csharp
public bool NoBreak { get; set; }
```

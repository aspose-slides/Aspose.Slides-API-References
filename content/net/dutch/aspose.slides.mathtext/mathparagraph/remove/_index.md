## MathParagraph.Remove methode

Verwijdert het eerste voorkomen van een specifiek object uit de collectie/>.

```csharp
public bool Remove(IMathBlock mathBlock)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| mathBlock | IMathBlock | Het object dat uit de collectie moet worden verwijderd. |

### Returnwaarde

true als *mathBlock* succesvol uit de collectie is verwijderd; anders false. Deze methode geeft ook false terug als *mathBlock* niet wordt gevonden in de oorspronkelijke collectie/>.

### Voorbeelden

Voorbeeld:

```csharp
[C#]
IAutoShape shape = slide.Shapes.AddMathShape(x, y, width, height);
IMathParagraph mathParagraph = (shape.TextFrame.Paragraphs[0].Portions[0] as MathPortion).MathParagraph;
mathParagraph.Add(new MathBlock(new MathematicalText("x")));
IMathBlock block = new MathBlock(new MathematicalText("y"));
mathParagraph.Add(block);
mathParagraph.Remove(block);
```

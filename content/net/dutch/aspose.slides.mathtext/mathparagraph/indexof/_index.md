## MathParagraph.IndexOf methode

Bepaalt de index van een specifiek IMathBlock in de verzameling.

```csharp
public int IndexOf(IMathBlock mathBlock)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| mathBlock | IMathBlock | Het item dat moet worden gevonden in de verzameling. |

### Retourwaarde

De index van *mathBlock* indien gevonden in de verzameling; anders -1.

### Voorbeelden

Voorbeeld:

```csharp
[C#]
IAutoShape shape = slide.Shapes.AddMathShape(x, y, width, height);
IMathParagraph mathParagraph = (shape.TextFrame.Paragraphs[0].Portions[0] as MathPortion).MathParagraph;
IMathBlock block = new MathBlock(new MathematicalText("y"));
mathParagraph.Add(block);
int index = mathParagraph.IndexOf(block);
```

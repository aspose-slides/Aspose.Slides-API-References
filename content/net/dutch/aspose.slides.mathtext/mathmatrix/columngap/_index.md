## MathMatrix.ColumnGap eigenschap

De waarde van de horizontale afstand tussen kolommen van een matrix; als de ColumnGapRule is ingesteld op 3 ("Exactly"), dan wordt de eenheid geïnterpreteerd als twips (1/20e van een punt) Als de ColumnGapRule is ingesteld op 4 ("Multiple"), dan wordt de eenheid geïnterpreteerd als aantal van 0.5 em-increments. In andere gevallen genegeerd. Standaard: 0

```csharp
public uint ColumnGap { get; set; }
```

### Voorbeelden

Voorbeeld:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.ColumnGapRule = MathSpacingRules.Exactly;
matrix.ColumnGap = 20;
```

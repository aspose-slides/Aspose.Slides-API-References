## IMathMatrix.RowGap eigenschap

De waarde van de verticale tussenruimte tussen rijen van een matrix; Als de RowGapRule is ingesteld op 3 ("Exactly"), wordt de eenheid geïnterpreteerd als twips (1/20e van een punt) Als de RowGapRule is ingesteld op 4 ("Multiple"), wordt de eenheid geïnterpreteerd als halve regels. Standaard: 0

```csharp
public uint RowGap { get; set; }
```

### Voorbeelden

Voorbeeld:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

## Effect.StopPreviousSound eigenschap

Deze eigenschap geeft aan of het animatie-effect het vorige geluid stopt. Lezen/schrijven Boolean.

```csharp
public bool StopPreviousSound { get; set; }
```

### Voorbeelden

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Haal het eerste effect van de eerste dia op.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Haal het eerste effect van de tweede dia op.
    IEffect secondSlideEffect = presentation.Slides[1].Timeline.MainSequence[0];
       
    if (firstSlideEffect.Sound != null)
    {
        // Wijzig het tweede effect Enhancements/Sound naar "Stop Previous Sound"
        secondSlideEffect.StopPreviousSound = true;
    }
}
```

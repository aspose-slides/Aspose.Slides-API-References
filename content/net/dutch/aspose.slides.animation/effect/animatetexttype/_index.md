## Effect.AnimateTextType eigenschap

Definieert een animatieteksttype voor effect. De vormtekst kan per letter, per woord of in één keer worden geanimeerd. Lezen/schrijven `AnimateTextType`.

```csharp
public AnimateTextType AnimateTextType { get; set; }
```

### Voorbeelden

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Haal het eerste effect van de eerste dia op.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Wijzig het effect AnimateTextType naar "By letter"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByLetter;
}
```

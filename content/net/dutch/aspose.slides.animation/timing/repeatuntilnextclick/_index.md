## Timing.RepeatUntilNextClick eigenschap

Dit attribuut geeft aan of het effect zich herhaalt tot de volgende klik. Lezen/schrijven Boolean.

```csharp
public bool RepeatUntilNextClick { get; set; }
```

### Voorbeelden

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Haal de effectensequentie op voor de eerste dia
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Haal het eerste effect van de hoofdsequentie op.
    IEffect effect = effectsSequence[0];

    // Verander de Timing/Repeat van het effect naar "Until Next Click"
    effect.Timing.RepeatUntilNextClick = true;
}
```

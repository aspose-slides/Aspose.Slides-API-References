## Timing.RepeatUntilEndSlide eigenschap

Dit attribuut geeft aan of het effect wordt herhaald tot het einde van de dia. Lezen/Schrijven Boolean.

```csharp
public bool RepeatUntilEndSlide { get; set; }
```

### Voorbeelden

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Haal de effectensequentie op voor de eerste dia
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Haal het eerste effect van de hoofdsequente op.
    IEffect effect = effectsSequence[0];

    // Verander de Timing/Repeat van het effect naar "Until End of Slide"
    effect.Timing.RepeatUntilEndSlide = true;
}
```

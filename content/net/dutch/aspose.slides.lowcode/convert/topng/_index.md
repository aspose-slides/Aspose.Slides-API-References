## ToPng(Presentation, string) {#topng}

Converteert de ingevoerde presentatie naar een set PNG-afbeeldingen. Als de uitvoerbestandsnaam wordt opgegeven als "myPath/myFilename.png", wordt het resultaat opgeslagen als een set "myPath/myFilename_N.png"-bestanden, waarbij N een slide-nummer is.

```csharp
public static void ToPng(Presentation pres, string outputFileName)
```

| pres | Presentation | De ingevoerde presentatie. |

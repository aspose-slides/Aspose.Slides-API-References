## ToJpeg(Presentation, string) {#tojpeg}

Converteert de invoer-presentatie naar een reeks afbeeldingen in JPEG-formaat. Als de uitvoer-bestandsnaam wordt opgegeven als "myPath/myFilename.jpeg", wordt het resultaat opgeslagen als een reeks bestanden "myPath/myFilename_N.jpeg", waarbij N een dia-nummer is.

```csharp
public static void ToJpeg(Presentation pres, string outputFileName)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| pres | Presentation | De invoer-presentatie. |
| outputFileName | String | De naam van het uitvoer-bestand. |

### Voorbeelden


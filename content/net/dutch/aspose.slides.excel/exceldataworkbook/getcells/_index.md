## ExcelDataWorkbook.GetCells methode

Haalt een verzameling cellen op uit de werkmap die overeenkomt met de opgegeven formule.

```csharp
public ReadOnlyCollection<IExcelDataCell> GetCells(string formula, bool skipHiddenCells)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| formula | String | Een formule of bereikexpressie (bijv. "Sheet1!A1:B3") die wordt gebruikt om doelcellen te identificeren. |
| skipHiddenCells | Boolean | Als `true`, worden verborgen cellen (bijv. in verborgen rijen of kolommen) uitgesloten van het resultaat. |

### Retourwaarde

Een alleen-lezen lijst van cellen die overeenkomt met de opgegeven formule.

### Voorbeelden

Voorbeeld:

```csharp
[C#]
ExcelDataWorkbook wb = new ExcelDataWorkbook(testFile);
IReadOnlyList<IExcelDataCell> cells = wb.GetCells("Sheet1!A2:A6", false);
Console.WriteLine(cells.Count); //Uitvoer: 5
```

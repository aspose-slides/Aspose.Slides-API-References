## ExcelDataWorkbook.GetChartsFromWorksheet methode

Haalt een dictionary op die de indexen en namen van alle grafieken in het opgegeven werkblad van een Excel-werkmap bevat.

```csharp
public IDictionary<int, string> GetChartsFromWorksheet(string worksheetName)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| worksheetName | String | De naam van het werkblad waarin gezocht moet worden naar grafieken. |

### Retourwaarde

Een dictionary waarbij de sleutel de grafiekindex is en de waarde de grafieknaam.

### Voorbeelden

Voorbeeld:

```csharp
[C#]
ExcelDataWorkbook wb = new ExcelDataWorkbook(testFile);
var sheetCharts = wb.GetChartsFromWorksheet("worksheetName");
foreach (var chart in sheetCharts)
    Console.WriteLine(chart.Key + " : " + chart.Value);
```

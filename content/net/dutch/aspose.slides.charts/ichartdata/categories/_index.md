## IChartData.Categories eigenschap

Haalt de primaire categorieën op (of zowel primaire als secundaire categorieën als [`UseSecondaryCategories`](../usesecondarycategories) eigenschap false is). Alleen-lezen [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection Categories { get; }
```

### Opmerkingen

Als [`UseSecondaryCategories`](../usesecondarycategories) eigenschap false is, dan retourneert [`SecondaryCategories`](../secondarycategories) eigenschap null en worden de gegevens in deze `Categories` eigenschap zowel voor primaire als secundaire series gebruikt. Als [`UseSecondaryCategories`](../usesecondarycategories) eigenschap true is, dan worden de gegevens in [`SecondaryCategories`](../secondarycategories) eigenschap gebruikt voor de secundaire series en worden de gegevens in deze `Categories` eigenschap gebruikt voor de primaire series.

### Voorbeelden

Voorbeeld. Welke categorieën zijn gerelateerd aan de reeks - ChartData.Categories of ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // gerelateerde categorieën zijn series.Chart.ChartData.SecondaryCategories
}
else
{
    // gerelateerde categorieën zijn series.Chart.ChartData.Categories
}
```

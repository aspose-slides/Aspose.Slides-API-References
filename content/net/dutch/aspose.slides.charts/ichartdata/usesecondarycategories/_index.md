## IChartData.UseSecondaryCategories eigenschap

Als false dan [`SecondaryCategories`](../secondarycategories) eigenschap retourneert null en de gegevens in [`Categories`](../categories) eigenschap worden zowel voor primaire als voor secundaire series gebruikt. Als true dan worden de gegevens in [`SecondaryCategories`](../secondarycategories) eigenschap gebruikt voor secundaire series en de gegevens in [`Categories`](../categories) eigenschap gebruikt voor primaire series. Lezen/schrijven Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

Voorbeeld. Welke categorieën zijn gerelateerd aan series - ChartData.Categories of ChartData.SecondaryCategories?

{
    // gerelateerde categorieën zijn series.Chart.ChartData.SecondaryCategories

    // gerelateerde categorieën zijn series.Chart.ChartData.Categories
}

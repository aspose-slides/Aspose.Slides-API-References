## ChartSeriesGroup.Overlap eigenschap

Specificeert hoeveel balken en kolommen elkaar overlappen op 2-D grafieken, als een percentage (van -100% tot 100%). - -100%: Maximale tussenruimte (balken zijn volledig gescheiden). - 0%: Balken worden naast elkaar geplaatst zonder overlap of ruimte. - 100%: Maximale overlap (balken overlappen elkaar volledig). Deze eigenschap is lezen/schrijven SByte.

```csharp
public sbyte Overlap { get; set; }
```

### Uitzonderingen

| uitzondering | conditie |
| --- | --- |
| ArgumentOutOfRangeException | Wordt gegooid wanneer de waarde wordt ingesteld buiten het geldige bereik van -100 tot 100. |

### Voorbeelden

Het volgende voorbeeld laat zien hoe de overlap voor een ChartSeriesGroup kan worden ingesteld en de resulterende grafiek op een formulier kan worden gerenderd:

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // Stel overlap in op 55%

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

## DataLabelCollection.IsVisible eigenschap

False betekent dat het gegevenslabel standaard niet zichtbaar is (en dus alle Show*-flags (ShowValue, ...) van de DefaultDataLabelFormat eigenschap zijn false). Alleen-lezen Boolean.

```csharp
public bool IsVisible { get; }
```

### Opmerkingen

Als het gegevenslabel standaard zichtbaar is, kun je het standaard verbergen met de Hide() methode. Maar als het gegevenslabel standaard niet zichtbaar is (IsVisible is false) kun je het gegevenslabel “standaard zichtbaar” maken door de Show*-flags (ShowValue, ...) van de DefaultDataLabelFormat eigenschap op true te zetten.

## DataLabelFormat.ShowValue eigenschap

Stelt het weergavegedrag van het percentage van een opgegeven diagram-gegevenslabel voor. True geeft de percentagewaarde weer. False verbergt deze. Lezen/schrijven Boolean.

```csharp
public bool ShowValue { get; set; }
```

### Opmerkingen

Als de ouder van dit DataLabelFormat-object een DataLabelCollection-collectie van gegevenslabels is, krijgt of stelt deze eigenschap de standaardwaarde van de ShowValue-eigenschap in voor de nieuwe gegevenslabels in de DataLabelCollection-collectie. Het instellen van deze eigenschap met een waarde stelt deze waarde ook in op de ShowValue-eigenschap voor alle gegevenslabels in de DataLabelCollection-collectie (i.e. "DataLabels.DefaultDataLabelFormat.ShowValue = val;" cause to all DataLabels[i].ShowValue is equal to val).

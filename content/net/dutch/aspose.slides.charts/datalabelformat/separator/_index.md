## DataLabelFormat.Separator eigenschap

Stelt in of retourneert een Variant die het scheidingsteken vertegenwoordigt dat wordt gebruikt voor de gegevenslabels op een diagram. Lezen/Schrijven String.

```csharp
public string Separator { get; set; }
```

### Opmerkingen

Als de ouder van dit DataLabelFormat-object een DataLabelCollection-collectie van gegevenslabels is, dan haalt deze eigenschap of stelt de standaardwaarde van de Separator eigenschap in voor de nieuwe gegevenslabels in de DataLabelCollection-collectie. Het instellen van deze eigenschap met een waarde stelt deze waarde ook in op de Separator eigenschap voor alle gegevenslabels in de DataLabelCollection-collectie (bijv. "DataLabels.DefaultDataLabelFormat.Separator = val;" waardoor alle DataLabels[i].Separator gelijk is aan val).

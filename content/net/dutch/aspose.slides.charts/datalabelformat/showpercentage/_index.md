## DataLabelFormat.ShowPercentage eigenschap

Geeft het weergavegedrag van het percentage van de gegevenslabels van een opgegeven diagram weer. True toont de percentagewaarde. False om te verbergen. Lezen/Schrijven Boolean.

```csharp
public bool ShowPercentage { get; set; }
```

### Opmerkingen

Als de ouder van dit DataLabelFormat-object een DataLabelCollection-verzameling van gegevenslabels is, dan krijgt of stelt deze eigenschap de standaardwaarde van de ShowPercentage-eigenschap in voor de nieuwe gegevenslabels in de DataLabelCollection-verzameling. Het instellen van deze eigenschap met een waarde stelt deze waarde ook in op de ShowPercentage-eigenschap voor alle gegevenslabels in de DataLabelCollection-verzameling (bijv. "DataLabels.DefaultDataLabelFormat.ShowPercentage = val;" zorgt ervoor dat alle DataLabels[i].ShowPercentage gelijk is aan val).

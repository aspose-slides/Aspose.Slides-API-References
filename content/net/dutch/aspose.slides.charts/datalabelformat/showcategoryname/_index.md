## DataLabelFormat.ShowCategoryName eigenschap

Geeft het weergavegedrag van de categorienaam van de gegevensetiketten van een opgegeven diagram weer. True om de categorienaam voor de gegevensetiketten op een diagram weer te geven. False om te verbergen. Lezen/Schrijven Boolean.

```csharp
public bool ShowCategoryName { get; set; }
```

### Opmerkingen

Als de bovenliggende van dit DataLabelFormat-object een DataLabelCollection-verzameling van gegevensetiketten is, dan krijgt of stelt deze eigenschap de standaardwaarde van de ShowCategoryName-eigenschap in voor de nieuwe gegevensetiketten in de DataLabelCollection-verzameling. Het instellen van deze eigenschap met een waarde stelt deze waarde ook in voor de ShowCategoryName-eigenschap van alle gegevensetiketten in de DataLabelCollection-verzameling (bijv. "DataLabels.DefaultDataLabelFormat.ShowCategoryName = val;" cause to all DataLabels[i].ShowCategoryName is equal to val).

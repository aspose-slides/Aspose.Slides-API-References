## DataLabelFormat.IsNumberFormatLinkedToSource property

Lezen/schrijven Boolean.

```csharp
public bool IsNumberFormatLinkedToSource { get; set; }
```

### Opmerkingen

Als de ouder van dit DataLabelFormat-object een DataLabelCollection-verzameling van datalabels is, dan haalt deze property de standaardwaarde van de IsNumberFormatLinkedToSource-property op of stelt deze in voor de nieuwe datalabels in de DataLabelCollection-verzameling. Het instellen van deze property met een waarde stelt deze waarde ook in op de IsNumberFormatLinkedToSource-property voor alle datalabels in de DataLabelCollection-verzameling (i.e. "DataLabels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = val;" cause to all DataLabels[i].IsNumberFormatLinkedToSource is equal to val).

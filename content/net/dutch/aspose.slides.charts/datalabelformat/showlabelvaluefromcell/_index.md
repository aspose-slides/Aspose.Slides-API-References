## DataLabelFormat.ShowLabelValueFromCell eigenschap

Geeft het weergegeven gedrag van de celwaarde van een datalabel van een opgegeven diagram weer. True toont de celwaarde. False verbergt deze. Lezen/schrijven Boolean.

```csharp
public bool ShowLabelValueFromCell { get; set; }
```

### Opmerkingen

Als de ouder van dit DataLabelFormat-object een DataLabelCollection-collection van datalabels is, krijgt of stelt deze eigenschap de standaardwaarde van de ShowLabelValueFromCell eigenschap in voor de nieuwe datalabels in de DataLabelCollection-collection. Het instellen van deze eigenschap met een waarde stelt deze waarde ook in voor de ShowLabelValueFromCell eigenschap van alle datalabels in de DataLabelCollection-collection (bijv. "DataLabels.DefaultDataLabelFormat.ShowLabelValueFromCell = val;" zorgt ervoor dat alle DataLabels[i].ShowLabelValueFromCell gelijk is aan val).

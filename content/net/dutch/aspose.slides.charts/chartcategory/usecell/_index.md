## ChartCategory.UseCell eigenschap

If true then AsCell eigenschap is actual. In other words, worksheet is used for storing category (this case supports a multi-level category). If false then AsLiteral eigenschap is actual. In other words, worksheet is NOT used for storing category (and this case doesn't support a multi-level categories). Alleen-lezen Boolean.

```csharp
public bool UseCell { get; }
```

### Opmerkingen

Om de waarde van deze eigenschap te wijzigen (voor alle categorieën in de collectie) stelt u de nieuwe waarde in op ChartCategoryCollection.UseCells eigenschap.

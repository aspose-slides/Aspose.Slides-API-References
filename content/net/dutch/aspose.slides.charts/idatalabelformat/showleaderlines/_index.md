## IDataLabelFormat.ShowLeaderLines eigenschap

Geeft het weergeven van leader-lijnen van gegevenslabels voor een opgegeven diagram weer. True geeft de leader-lijnen weer. False verbergt ze. Lezen/Schrijven Boolean.

```csharp
public bool ShowLeaderLines { get; set; }
```

### Opmerkingen

Als de ouder van dit DataLabelFormat-object een DataLabelCollection-collectie van gegevenslabels is, dan krijgt of stelt deze eigenschap de standaardwaarde van de ShowLeaderLines-eigenschap in voor de nieuwe gegevenslabels in de DataLabelCollection-collectie. Het instellen van deze eigenschap met een waarde stelt deze waarde ook in voor de ShowLeaderLines-eigenschap van alle gegevenslabels in de DataLabelCollection-collectie (bijv. "DataLabels.DefaultDataLabelFormat.ShowLeaderLines = val;" waardoor alle DataLabels[i].ShowLeaderLines gelijk is aan val).

## IDataLabelFormat.ShowPercentage eigenschap

Geeft het weergavegedrag weer van de percentage-waarde van data-labels voor een opgegeven diagram. True toont de percentage-waarde. False om te verbergen. Lezen/schrijven Boolean.

```csharp
public bool ShowPercentage { get; set; }
```

### Opmerkingen

Als de bovenliggende instantie van dit DataLabelFormat-object een DataLabelCollection-verzameling van data-labels is, dan haalt deze eigenschap de standaardwaarde op of stelt deze in voor de ShowPercentage-eigenschap van de nieuwe data-labels in de DataLabelCollection-verzameling. Het instellen van deze eigenschap met een waarde stelt deze waarde ook in voor de ShowPercentage-eigenschap van alle data-labels in de DataLabelCollection-verzameling (bijv. "DataLabels.DefaultDataLabelFormat.ShowPercentage = val;" waardoor alle DataLabels[i].ShowPercentage gelijk is aan val).

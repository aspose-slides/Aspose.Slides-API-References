## IDataLabelFormat.ShowBubbleSize eigenschap

Geeft het weergavegedrag van de bubbelgroottewaarde van een opgegeven diagram-databelabel weer. True toont de bubbelgroottewaarde. False verbergt deze. Lezen/schrijven Boolean.

```csharp
public bool ShowBubbleSize { get; set; }
```

### Opmerkingen

Als de bovenliggende van dit DataLabelFormat-object een DataLabelCollection-verzameling van datalabels is, dan krijgt of stelt deze eigenschap de standaardwaarde van de ShowBubbleSize-eigenschap in voor de nieuwe datalabels in de DataLabelCollection-verzameling. Deze eigenschap instellen met een waarde stelt deze waarde ook in voor de ShowBubbleSize-eigenschap van alle datalabels in de DataLabelCollection-verzameling (bijv. "DataLabels.DefaultDataLabelFormat.ShowBubbleSize = val;" waardoor alle DataLabels[i].ShowBubbleSize gelijk is aan val).

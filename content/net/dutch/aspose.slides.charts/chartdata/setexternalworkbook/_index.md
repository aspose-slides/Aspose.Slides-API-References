## SetExternalWorkbook(string) {#setexternalworkbook}

Stelt de externe werkmap in als gegevensbron voor de grafiek. Grafiekgegevens worden bijgewerkt vanuit de doelwerkmap.

```csharp
public void SetExternalWorkbook(string workbookPath)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| workbookPath | String | Pad naar de doelwerkmap |

### Uitzonderingen

| uitzondering | conditie |
| --- | --- |
| InvalidOperationException | Externe werkmap is niet beschikbaar of kan niet worden geladen. |

### Voorbeelden

```csharp
[C#]
using (Presentation pres = new Presentation())

   IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Pie, 50, 50, 400, 600, true);
   IChartData chartData = chart.ChartData;
   (chartData as ChartData).SetExternalWorkbook("../../workbook.xlsx");

```

```csharp

```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| workbookPath | String | Pad naar de doelwerkmap |
| updateChartData | Boolean | Als de waarde false is, wordt alleen het pad naar de werkmap bijgewerkt. Grafiekgegevens worden niet geladen en bijgewerkt vanuit de doelwerkmap.

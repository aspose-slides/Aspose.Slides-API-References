## ChartData.UseSecondaryCategories eigenschap

If false then [`SecondaryCategories`](../secondarycategories) eigenschap return null and data in [`Categories`](../categories) eigenschap is used both for primary and secondary series. If true then data in [`SecondaryCategories`](../secondarycategories) eigenschap is used for secondary series and data in [`Categories`](../categories) eigenschap is used for primary series. Lezen/Schrijven Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

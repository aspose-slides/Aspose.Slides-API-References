## IChartSeriesGroup.Overlap eigenschap

Specificeert hoeveel balken en kolommen op 2-D-grafieken overlappen, als een percentage (van -100% tot 100%).  
- -100%: maximale afstand (balken zijn volledig gescheiden).  
- 0%: balken staan naast elkaar zonder overlap of afstand.  
- 100%: maximale overlap (balken overlappen volledig).  
Deze eigenschap is lees/schrijf SByte.

```csharp
public sbyte Overlap { get; set; }
```

### Uitzonderingen

| uitzondering | voorwaarde |
| --- | --- |
| ArgumentOutOfRangeException | Wordt gegooid wanneer de waarde buiten het geldige bereik van -100 tot 100 wordt ingesteld. |

### Voorbeelden

Het volgende voorbeeld toont hoe je de overlap voor een grafiekreeks-groep instelt en de resulterende grafiek op een formulier rendert:

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // Stel overlap in op 55%

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

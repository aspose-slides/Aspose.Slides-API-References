## IDataLabel.IsVisible eigenschap

False betekent dat het gegevenslabel niet zichtbaar is (en dat alle Show*-vlaggen (ShowValue, ...) false zijn). Alleen-lezen Boolean.

```csharp
public bool IsVisible { get; }
```

### Opmerkingen

Als het gegevenslabel zichtbaar is, kun je het verbergen met de Hide()-methode. Maar als het gegevenslabel niet zichtbaar is (IsVisible is false), kun je het gegevenslabel zichtbaar maken door de Show*-vlaggen (ShowValue, ...) op de waarde true te zetten.

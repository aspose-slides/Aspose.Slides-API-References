## MathSpacingRules enumeration

Types d'écart (espacement horizontal) entre les colonnes d'une matrice

```csharp
public enum MathSpacingRules
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| SingleSpacingGap | `0` | Unique (1 em pour les colonnes ou 1 ligne pour les lignes) |
| OneAndHalfSpacingGap | `1` | Un an et demi (1,5 ems pour les colonnes ou 1,5 lignes pour les lignes) |
| DoubleSpacingGap | `2` | Double interligne (2 ems pour les colonnes ou 2 lignes pour les lignes) |

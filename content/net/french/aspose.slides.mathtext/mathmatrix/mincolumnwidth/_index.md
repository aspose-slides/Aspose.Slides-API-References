## Propriété MathMatrix.MinColumnWidth

Largeur minimale de colonne en twips (1/20 d'un point) L'espacement des colonnes (également appelé « Écart de colonne » ou « Largeur d'écart ») est ajouté à la MinColumnWidth pour déterminer l'espacement total des colonnes de la matrice (distance entre les mêmes bords de différentes colonnes). Par défaut : 0.

```csharp
public uint MinColumnWidth { get; set; }
```

### Exemples

Exemple :

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.MinColumnWidth = 20;
```

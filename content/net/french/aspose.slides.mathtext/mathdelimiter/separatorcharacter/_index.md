## Propriété MathDelimiter.SeparatorCharacter

Le caractère séparateur de délimiteur spécifie le caractère qui sépare les arguments dans l'objet délimiteur. La valeur par défaut : '&#x7C;'.

```csharp
public char SeparatorCharacter { get; set; }
```

### Exemples

Exemple:


## Propriété MathDelimiter.EndingCharacter

Le caractère de délimiteur de fin spécifie le caractère délimiteur de fin ou de fermeture. Les délimiteurs mathématiques sont des caractères englobants tels que des parenthèses, des crochets et des accolades. La valeur par défaut : ')'.

```csharp
public char EndingCharacter { get; set; }
```

### Exemples

Exemple :

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Join("y").Enclose();
delimiter.EndingCharacter = ']';
```

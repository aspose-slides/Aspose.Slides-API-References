## Propriété MathDelimiter.BeginningCharacter

Le caractère de début de délimiteur spécifie le caractère délimiteur de début ou d'ouverture. Les délimiteurs mathématiques sont des caractères d'encadrement tels que des parenthèses, des crochets et des accolades. Par défaut : '('.

```csharp
public char BeginningCharacter { get; set; }
```

### Exemples

Exemple :

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Join("y").Enclose();
delimiter.BeginningCharacter = '[';
```

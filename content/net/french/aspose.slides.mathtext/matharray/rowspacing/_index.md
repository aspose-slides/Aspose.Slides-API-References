## MathArray.RowSpacing property

Espacement entre les lignes d'un tableau Il est utilisé uniquement lorsque RowSpacingRule est défini sur 3 Exactement auquel cas l'unité de mesure est des points ou Multiple auquel cas l'unité de mesure est des demi-lignes. Par défaut : 0

public uint RowSpacing { get; set; }

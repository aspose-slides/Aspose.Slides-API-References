## MathArray.RowSpacing property

Espacement entre les lignes d'un tableau. Il est utilisé uniquement lorsque RowSpacingRule est défini sur 3. Dans ce cas, l'unité de mesure est des points ou Multiple, auquel cas l'unité de mesure est des demi-lignes. Par défaut : 0

```csharp
public uint RowSpacing { get; set; }
```

### Exemples

Exemple :

```csharp
[C#]
IMathArray mathArray = new MathArray(new MathematicalText("item1"));
mathArray.RowSpacingRule = MathRowSpacingRule.Exactly;
mathArray.RowSpacing = 10;
```

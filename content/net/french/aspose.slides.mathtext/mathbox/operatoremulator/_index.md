## Propriété MathBox.OperatorEmulator

Émulateur d'opérateur. Lorsque c'est vrai, la boîte et son contenu se comportent comme un seul opérateur et héritent des propriétés d'un opérateur. Cela signifie, par exemple, que le caractère peut servir de point pour un saut de ligne et peut être aligné avec d'autres opérateurs. Les émulateurs d'opérateurs sont souvent utilisés lorsqu'un ou plusieurs glyphes se combinent pour former un opérateur, tel que '=='. Valeur par défaut : false

```csharp
public bool OperatorEmulator { get; set; }
```

### Exemples

Exemple :

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("=="));
box.OperatorEmulator = true;
```

## MathBox.NoBreak property

Pas de coupure Cette propriété spécifie la propriété "non cassable" sur la boîte d'objet. Lorsque c'est vrai, aucune coupure de ligne ne peut se produire à l'intérieur de la boîte. Cela peut être important pour les émulateurs d'opérateurs qui consistent en plus d'un opérateur binaire. Lorsque cet élément n'est pas spécifié, des coupures peuvent se produire à l'intérieur de la boîte. Par défaut : vrai

```csharp
public bool NoBreak { get; set; }
```

### Examples

Exemple:

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("*****"));
box.NoBreak = false;
```

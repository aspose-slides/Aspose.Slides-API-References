## Propriété MathBox.AlignmentPoint

Lorsqu'il est vrai, cet émulateur d'opérateur sert de point d'alignement ; c'est-à-dire que les points d'alignement désignés dans d'autres équations peuvent être alignés avec lui. Par défaut : faux

public bool AlignmentPoint { get; set; }

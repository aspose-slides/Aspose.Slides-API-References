## IMathGroupingCharacter interface

Spécifie un symbole de groupement au-dessus ou en dessous d'une expression, généralement pour souligner la relation entre les éléments

```csharp
public interface IMathGroupingCharacter : IMathElement
```

Spécifie l'alignement de l'objet par rapport à la ligne de base. Par exemple, lorsque le caractère de groupe est au-dessus de l'objet, VerticalJustification à Top indique que le haut de l'objet se trouve sur la ligne de base ; lorsque VerticalJustification est réglé sur Bottom, le bas de l'objet est sur la ligne de base Valeur par défaut : Bottom pour Position=Top, et Top pour Position=Bottom |

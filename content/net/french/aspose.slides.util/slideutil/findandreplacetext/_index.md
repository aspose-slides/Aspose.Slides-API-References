## Méthode SlideUtil.FindAndReplaceText

Trouve et remplace du texte dans la présentation avec le format donné

```csharp
public static void FindAndReplaceText(IPresentation presentation, bool withMasters, string find, 
    string replace, PortionFormat format = null)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| presentation | IPresentation | Présentation scannée. |
| withMasters | Boolean | Détermine si les diapositives maîtresses doivent être scannées. |
| find | String | Valeur de chaîne à trouver. |
| replace | String | Valeur de chaîne à remplacer. |
| format | PortionFormat | Format pour remplacer la portion de texte. Si null, le format du premier caractère de la chaîne trouvée sera utilisé |

### Exemples

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PortionFormat format = new PortionFormat
    {
        FontHeight = 24f,
        FontItalic = NullableBool.True,
        FillFormat =
        {
            FillType = FillType.Solid,
            SolidFillColor =
            {
                Color = Color.Red
            }
        }
    };
    Aspose.Slides.Util.SlideUtil.FindAndReplaceText(pres, true, "[this block] ", "my text ", format);
    pres.Save("replaced", SaveFormat.Pptx);
}
```

## Méthode WebDocument.Save

Enregistre la sortie du document.

```csharp
public void Save()
```

### Exemples

```csharp
[C#]        
using (Presentation pres = new Presentation("pres.pptx"))
{
    var options = new WebDocumentOptions
    {
        TemplateEngine = new RazorTemplateEngine(),
        OutputSaver = new FileOutputSaver(),
        EmbedImages = false
    };
    
    WebDocument document = new WebDocument(options);

    // ajouter le modèle "index-template.html" avec la clé de modèle "index" pour l'utiliser plus tard (pour la sortie)
    document.Input.AddTemplate<Aspose.Slides.Presentation>("index", "index-template.html");

    // ajouter "index.html" aux fichiers de sortie, en utilisant le modèle "index" pour le générer et la variable pres comme modèle
    document.Output.Add("index.html", "index", pres);
    
    // ... configurer d'autres options du document puis enregistrer le document
    document.Global.Put("slideMargin", 10);
    document.Global.Put("imagesPath", "root/site/images");
    // ...
 
    document.Save();
}
```

### Exemples

En utilisant ceci`Global` immobilier (mise en place de[`Storage`](../../storage) interface) une propriété peut être utilisée ultérieurement dans le modèle :

```csharp
[C#]
var options = new WebDocumentOptions
{
    TemplateEngine = new RazorTemplateEngine(),
    OutputSaver = new FileOutputSaver(),
    EmbedImages = false
};

WebDocument document = new WebDocument(options);

// mettre la propriété "slideMargin" à utiliser à partir des modèles
document.Global.Put("slideMargin", 10);

// ... configurez les autres options du document puis enregistrez le document
document.Save();
```

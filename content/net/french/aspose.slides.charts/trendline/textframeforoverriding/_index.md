## Propriété Trendline.TextFrameForOverriding

Peut contenir un texte richement formaté. Si cette propriété n'est pas nulle, alors cette valeur de texte formaté remplace le texte généré automatiquement de l'étiquette de données. Le texte généré automatiquement de l'étiquette de données signifie le texte qui est géré par les propriétés ShowSeriesName, ShowValue, ... et est formaté avec la propriété TextFormatManager.TextFormat.

## Propriété Trendline.TextFrameForOverriding

Peut contenir un texte riche formaté. Si cette propriété n'est pas nulle, alors cette valeur de texte formatée remplace le texte généré automatiquement de l'étiquette de données. Le texte généré automatiquement de l'étiquette de données signifie le texte qui est géré par les propriétés ShowSeriesName, ShowValue, ... et est formaté avec la propriété TextFormatManager.TextFormat.

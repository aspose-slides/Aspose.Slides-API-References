## IChartDataPointCollection.GetOrCreateDataPointByIdx method

Si la collection contient déjà un point de données avec index*index* puis renvoie ce point de données. Si la collection ne contient pas de point de données avec index*index* ==N (lorsque le nombre de points de données dans cette collection est inférieur ou égal à N) ajoute alors les points de données déficients et renvoie le dernier (qui a demandé l'index). Par exemple, les index de collection sont {0, 1, 2} et l'index demandé est 5. Ensuite, la méthode ajoute des points de données déficients : {0, 1, 2, 3, 4, 5}. Et renvoie le point de données avec l'index 5.

## ChartSeries.ErrorBarsXFormat property

Représente les ErrorBars de la série avec déréction X.  Les barres d'erreur avec la direction X sont disponibles pour les séries de type zone, barre, dispersion et bulle. Pour tout autre type de graphique, cette propriété renvoie null (y compris les graphiques 3D). En cas de valeurs personnalisées, utilisez la collection DataPoints pour spécifier value (avec[`ErrorBarsCustomValues`](../../ichartdatapoint/errorbarscustomvalues) propriété).

## ChartSeries.ErrorBarsXFormat propriété

Représente les ErrorBars des séries avec direction X. Les ErrorBars avec direction X sont disponibles pour les séries de type aire, barres, nuages de points et bulles. Pour tout autre type de graphique, cette propriété renvoie null (y compris pour les graphiques 3D). En cas de valeurs personnalisées, utilisez la collection DataPoints pour spécifier la valeur (avec la propriété [`ErrorBarsCustomValues`](../../ichartdatapoint/errorbarscustomvalues)).

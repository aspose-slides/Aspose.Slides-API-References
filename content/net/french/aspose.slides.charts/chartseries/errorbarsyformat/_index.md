## ChartSeries.ErrorBarsYFormat property

Représente les ErrorBars des séries avec direction Y. Les ErrorBars avec direction Y sont disponibles pour les séries de type area, bar, line, scatter et bubble. Pour d'autres types de graphiques, cette propriété renvoie null (y compris les graphiques 3D). En cas de valeurs personnalisées, utilisez la collection DataPoints pour spécifier la valeur (avec la propriété [`ErrorBarsCustomValues`](../../ichartdatapoint/errorbarscustomvalues)).

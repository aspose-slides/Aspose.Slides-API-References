## Propriété IDataLabelFormat.ShowLabelValueFromCell

Représente un comportement d'affichage de la valeur de cellule d'étiquette de données d'un graphique spécifié. True affiche la valeur de cellule. False pour masquer. Booléen en lecture/écriture.

```csharp
public bool ShowLabelValueFromCell { get; set; }
```

### Remarques

Si le parent de cet objet DataLabelFormat est une collection DataLabelCollection d'étiquettes de données, alors cette propriété obtient ou définit la valeur par défaut de la propriété ShowLabelValueFromCell pour les nouvelles étiquettes de données dans la collection DataLabelCollection. Définir cette propriété avec une valeur fixe également cette valeur à la propriété ShowLabelValueFromCell pour toutes les étiquettes de données dans la collection DataLabelCollection (c'est-à-dire que "DataLabels.DefaultDataLabelFormat.ShowLabelValueFromCell = val;" provoque que toutes DataLabels[i].ShowLabelValueFromCell soit égal à val).

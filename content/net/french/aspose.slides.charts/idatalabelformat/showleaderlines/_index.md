## Propriété IDataLabelFormat.ShowLeaderLines

Représente le comportement d'affichage des lignes de leader du label de données d'un graphique spécifié. True affiche les lignes de leader. False pour cacher. Booléen en lecture/écriture.

```csharp
public bool ShowLeaderLines { get; set; }
```

### Remarques

Si le parent de cet objet DataLabelFormat est une collection DataLabelCollection de labels de données, alors cette propriété obtient ou définit la valeur par défaut de la propriété ShowLeaderLines pour les nouveaux labels de données dans la collection DataLabelCollection. Définir cette propriété avec une valeur définit également cette valeur pour la propriété ShowLeaderLines de tous les labels de données dans la collection DataLabelCollection (c'est-à-dire que "DataLabels.DefaultDataLabelFormat.ShowLeaderLines = val;" entraîne que tous les DataLabels[i].ShowLeaderLines sont égaux à val).

## IChartData.SecondaryCategories property

Obtient les catégories secondaires si[`UseSecondaryCategories`](../usesecondarycategories) propriété est true. Lecture seule[`IChartCategoryCollection`](../../ichartcategorycollection) .

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### Remarques

Si[`UseSecondaryCategories`](../usesecondarycategories) la propriété est fausse alors ceci`SecondaryCategories` La propriété renvoie null et des données dans[`Categories`](../categories) La propriété est utilisée à la fois pour les séries primaires et secondaires. Si[`UseSecondaryCategories`](../usesecondarycategories) la propriété est vraie alors les données dans this`SecondaryCategories` la propriété est utilisée pour les séries secondaires et les données dans[`Categories`](../categories) la propriété est utilisée pour la série primaire.

```csharp
{
    // les catégories associées sont series.Chart.ChartData.SecondaryCategories

else
{
    // les catégories associées sont series.Chart.ChartData.Categories
}
```

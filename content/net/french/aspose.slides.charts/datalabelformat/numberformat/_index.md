## Propriété DataLabelFormat.NumberFormat

Représente la chaîne de format pour l'objet DataLabels. Chaîne en lecture/écriture.

```csharp
public string NumberFormat { get; set; }
```

### Remarques

Si le parent de cet objet DataLabelFormat est une collection DataLabelCollection de données étiquetées, alors cette propriété obtient ou définit la valeur par défaut de la propriété NumberFormat pour les nouvelles données étiquetées dans la collection DataLabelCollection. Lorsque cette propriété est définie avec une valeur, cette valeur est également définie pour la propriété NumberFormat de toutes les données étiquetées dans la collection DataLabelCollection (c'est-à-dire que "DataLabels.DefaultDataLabelFormat.NumberFormat = val;" entraîne que toutes les DataLabels[i].NumberFormat soient égales à val).

### Exemples

```csharp
[C#]
series.Labels.DefaultDataLabelFormat.ShowValue = true;
series.Labels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = false;
series.Labels.DefaultDataLabelFormat.NumberFormat = "0.0%";
```

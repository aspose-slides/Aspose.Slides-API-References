## Énumération TypeAggregationAxe

Représente le type d'agrégation de l'axe des catégories. Utilisé uniquement avec les séries Histogramme ou HistogrammePareto.

```csharp
public enum AxisAggregationType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| ParCatégorie | `0` | Agréger par catégories |
| Automatique | `1` | Agréger les catégories automatiquement. |
| ParLargeurDeBin | `2` | Agréger les catégories par largeur de bin. La propriété IAxis.BinWidth doit être définie. |
| ParNombreDeBins | `3` | Agréger les catégories par nombre de bins. La propriété IAxis.NumberOfBins doit être définie. |

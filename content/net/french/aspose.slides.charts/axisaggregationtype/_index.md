## Enumeration AxisAggregationType

Représente le type d'agrégation de l'axe des catégories. Utilisé uniquement avec les séries Histogram ou HistogramPareto.

```csharp
public enum AxisAggregationType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| ByCategory | `0` | Agréger par catégories |
| Automatic | `1` | Agréger les catégories automatiquement. |
| ByBinWidth | `2` | Agréger les catégories par largeur de classe. La propriété IAxis.BinWidth doit être définie. |
| ByNumberOfBins | `3` | Agréger les catégories par nombre de classes. La propriété IAxis.NumberOfBins doit être définie. |

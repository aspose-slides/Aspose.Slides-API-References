## Méthode IDataLabel.Hide

Rendre l'étiquette de données cachée en réglant tous les drapeaux Show*-flags (ShowValue, ...) sur l'état faux. IsVisible sera faux après cela.

```csharp
public void Hide()
```

### Remarques

Si l'étiquette de données n'est pas visible (IsVisible est faux), vous pouvez rendre l'étiquette de données visible en réglant les drapeaux Show*-flags (ShowValue, ...) sur l'état vrai.

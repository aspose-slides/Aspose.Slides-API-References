## IDataLabel.Hide méthode

Rendre l'étiquette de données cachée en définissant tous les drapeaux Show* (ShowValue, ...) sur l'état faux. IsVisible sera faux après cela.

```csharp
public void Hide()
```

### Remarques

Si l'étiquette de données n'est pas visible (IsVisible est faux), vous pouvez rendre l'étiquette de données visible en définissant les drapeaux Show* (ShowValue, ...) sur l'état vrai.

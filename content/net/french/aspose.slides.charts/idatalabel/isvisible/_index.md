## IDataLabel.IsVisible property

False signifie que l'étiquette de données n'est pas visible (et donc tous les drapeaux Show* (ShowValue, ...) sont faux). Lecture seuleBoolean .

```csharp
public bool IsVisible { get; }
```

### Remarques

Si l'étiquette de données est visible, vous pouvez la masquer avec la méthode Hide(). Mais si l'étiquette de données n'est pas visible (IsVisible est faux), vous pouvez rendre l'étiquette de données visible avec le paramètre Show*-flags (ShowValue, ...) à l'état vrai.

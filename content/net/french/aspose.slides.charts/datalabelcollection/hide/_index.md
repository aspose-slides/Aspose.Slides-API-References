## DataLabelCollection.Hide méthode

Rendez l'étiquette de données cachée par défaut en définissant tous les drapeaux Show* (ShowValue, ...) de la propriété DefaultDataLabelFormat sur l'état false. IsVisible sera faux après cela.

```csharp
public void Hide()
```

### Remarques

Si l'étiquette de données n'est pas visible par défaut (IsVisible est faux), vous pouvez rendre l'étiquette de données "visible par défaut" en réglant les drapeaux Show* (ShowValue, ...) de la propriété DefaultDataLabelFormat sur l'état true.

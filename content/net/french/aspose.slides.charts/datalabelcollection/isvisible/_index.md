## DataLabelCollection.IsVisible property

Faux signifie que l'étiquette de données n'est pas visible par défaut (et donc tous les drapeaux Show*- (ShowValue, ...) de la propriété DefaultDataLabelFormat sont faux). Booléen en lecture seule.

```csharp
public bool IsVisible { get; }
```

### Remarques

Si l'étiquette de données est visible par défaut, vous pouvez la rendre cachée par défaut avec la méthode Hide(). Mais si l'étiquette de données n'est pas visible par défaut (IsVisible est faux), vous pouvez rendre l'étiquette de données "visible par défaut" en définissant les drapeaux Show*- (ShowValue, ...) de la propriété DefaultDataLabelFormat à l'état vrai.

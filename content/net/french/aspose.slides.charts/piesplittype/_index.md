## Énumération PieSplitType

Représente un type de points de séparation dans le deuxième camembert ou barre d'un graphique camembert-à-camembert ou barre-à-barre.

```csharp
public enum PieSplitType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Default | `0` | Spécifie que les points de données seront séparés en utilisant le mécanisme par défaut pour ce type de graphique. |
| Custom | `1` | Spécifie que les points de données seront séparés entre le camembert et le deuxième graphique selon les valeurs de séparation personnalisées. |
| ByPercentage | `2` | Spécifie que les points de données seront séparés entre le camembert et le deuxième graphique en plaçant les points avec un pourcentage inférieur à celui de la Position de Séparation dans le deuxième graphique. |
| ByPos | `3` | Spécifie que les points de données seront séparés entre le camembert et le deuxième graphique en plaçant la dernière Position de Séparation des points de données dans le deuxième graphique. |
| ByValue | `4` | Spécifie que les points de données seront séparés entre le camembert et le deuxième graphique en plaçant les points de données avec une valeur inférieure à celle de la Position de Séparation dans le deuxième graphique. |

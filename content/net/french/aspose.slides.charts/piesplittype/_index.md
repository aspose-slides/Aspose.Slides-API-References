## Énumération PieSplitType

Représente un type de points de division dans le deuxième secteur ou barre d’un graphique secteur-dans-secteur ou barre-dans-secteur.

```csharp
public enum PieSplitType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Default | `0` | Spécifie que les points de données doivent être divisés en utilisant le mécanisme par défaut pour ce type de graphique. |
| Custom | `1` | Spécifie que les points de données doivent être divisés entre le secteur et le deuxième graphique selon les valeurs de division personnalisées. |
| ByPercentage | `2` | Spécifie que les points de données doivent être divisés entre le secteur et le deuxième graphique en plaçant les points dont le pourcentage est inférieur à la position de division dans le deuxième graphique. |
| ByPos | `3` | Spécifie que les points de données doivent être divisés entre le secteur et le deuxième graphique en plaçant la dernière position de division des points de données dans le deuxième graphique. |
| ByValue | `4` | Spécifie que les points de données doivent être divisés entre le secteur et le deuxième graphique en plaçant les points de données dont la valeur est inférieure à la position de division dans le deuxième graphique. |

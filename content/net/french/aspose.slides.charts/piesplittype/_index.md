## Énumération PieSplitType

Représente un type de points de séparation dans le deuxième graphique en secteur ou en barre sur un graphique en secteur de secteur ou un graphique en barre de barre.

```csharp
public enum PieSplitType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Default | `0` | Spécifie que les points de données doivent être séparés en utilisant le mécanisme par défaut pour ce type de graphique. |
| Custom | `1` | Spécifie que les points de données doivent être séparés entre le graphique en secteur et le deuxième graphique selon les valeurs de séparation personnalisées. |
| ByPercentage | `2` | Spécifie que les points de données doivent être séparés entre le graphique en secteur et le deuxième graphique en plaçant les points avec un pourcentage inférieur à la position de séparation dans le deuxième graphique. |
| ByPos | `3` | Spécifie que les points de données doivent être séparés entre le graphique en secteur et le deuxième graphique en plaçant la dernière position de séparation des points de données dans le deuxième graphique. |
| ByValue | `4` | Spécifie que les points de données doivent être séparés entre le graphique en secteur et le deuxième graphique en plaçant les points de données avec une valeur inférieure à la position de séparation dans le deuxième graphique. |

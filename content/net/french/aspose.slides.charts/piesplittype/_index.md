## PieSplitType enumeration

Représente un type de points de fractionnement dans le deuxième secteur ou barre sur un graphique en secteurs ou en barres.

```csharp
public enum PieSplitType
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| Default | `0` | Spécifie que les points de données doivent être divisés à l'aide du mécanisme par défaut pour ce type de graphique. |
| Custom | `1` | Spécifie que les points de données doivent être répartis entre le secteur et le deuxième graphique en fonction des valeurs de fractionnement personnalisé. |
| ByPercentage | `2` | Spécifie que les points de données doivent être répartis entre le secteur et le deuxième graphique en plaçant les points avec un pourcentage inférieur à Split Position pour cent dans le deuxième graphique. |
| ByPos | `3` | Spécifie que les points de données doivent être divisés entre le camembert et le deuxième graphique en plaçant la dernière position de fractionnement des points de données dans le deuxième graphique. |
| ByValue | `4` | Spécifie que les points de données doivent être divisés entre le camembert et le deuxième graphique en plaçant les points de données avec une valeur inférieure à Split Position dans le deuxième graphique. |

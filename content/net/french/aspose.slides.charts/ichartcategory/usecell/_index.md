## IChartCategory.UseCell property

Si vrai, la propriété AsCell est réelle. En d'autres termes, la feuille de calcul est utilisée pour la catégorie de stockage (ce cas prend en charge une catégorie à plusieurs niveaux). Si false, la propriété AsLiteral est réelle. En d'autres termes, la feuille de calcul n'est PAS utilisée pour stocker la catégorie (et ce cas ne prend pas en charge les catégories à plusieurs niveaux). Lecture seuleBoolean .

```csharp
public bool UseCell { get; }
```

### Remarques

Pour modifier la valeur de cette propriété (pour toutes les catégories de la collection), définissez la nouvelle valeur sur la propriété ChartCategoryCollection.UseCells.

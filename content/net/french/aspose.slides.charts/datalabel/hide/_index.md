## DataLabel.Hide method

Rendez l'étiquette de données invisible en définissant tous les indicateurs Show*- (ShowValue, ...) sur l'état false. IsVisible sera false après cela.

```csharp
public void Hide()
```

### Remarks

Si l'étiquette de données n'est pas visible (IsVisible est false), vous pouvez rendre l'étiquette de données visible en définissant les indicateurs Show*- (ShowValue, ...) sur l'état true.

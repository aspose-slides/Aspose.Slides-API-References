### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| SourceFileCorruption | `0` | Un problème a été détecté dans le document source, ce qui rend très probable que le document ne pourra pas être ouvert s'il est enregistré dans son format original. |

## SVGOptions.DisableLineEndCropping property

SVG 1.1 ne permet pas de définir des marges pour les marqueurs. Le moteur d'écriture SVG d'Aspose.Slides a une solution de contournement pour ce problème : il coupe la fin de la ligne avec une flèche, de sorte que la ligne ne chevauche pas les marqueurs. Cette option désactive ce comportement. Lecteur/écrivain booléen.

```csharp
public bool DisableLineEndCropping { get; set; }
```

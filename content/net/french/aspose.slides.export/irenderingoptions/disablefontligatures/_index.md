## Propriété IRenderingOptions.DisableFontLigatures

Obtient ou définit une valeur indiquant si le texte est rendu sans utiliser de ligatures. Lorsqu'il est défini sur `true`, les ligatures seront désactivées dans le rendu. Par défaut, cette propriété est définie sur `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Exemples

Exemple:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    RenderingOptions options = new RenderingOptions
    {
        DisableFontLigatures = true // Désactiver les ligatures dans le rendu du texte
    };
    
    Bitmap[] renderedSlides = pres.GetImage(options);
    for (var index = 0; index < renderedSlides.Length; index++)
    {
        var slideImage = renderedSlides[index];
        slideImage.Save($"slide-{index}.png");
    }
}
```

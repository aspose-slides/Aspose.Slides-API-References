## Énumération SvgEvent

Représente les options pour la forme SVG.

```csharp
public enum SvgEvent
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| OnFocusIn | `0` | Se produit lorsqu'un élément reçoit le focus, par exemple, lorsque du texte est sélectionné. |
| OnFocusOut | `1` | Se produit lorsqu'un élément perd le focus, par exemple, lorsque du texte est désélectionné. |
| OnActivate | `2` | Se produit lorsqu'un élément est activé, par exemple, par un clic de souris ou une pression de touche. |
| OnClick | `3` | Se produit lorsque le bouton de l'appareil de pointage est cliqué sur un élément. |
| OnMouseDown | `4` | Se produit lorsque le bouton de l'appareil de pointage est enfoncé sur un élément. |
| OnMouseUp | `5` | Se produit lorsque le bouton de l'appareil de pointage est relâché sur un élément. |
| OnMouseOver | `6` | Se produit lorsque l'appareil de pointage est déplacé sur un élément. |
| OnMouseMove | `7` | Se produit lorsque l'appareil de pointage est déplacé alors qu'il est au-dessus d'un élément. |
| OnMouseOut | `8` | Se produit lorsque l'appareil de pointage est déplacé à l'écart d'un élément. |
| OnLoad | `9` | Se produit lorsque l'agent utilisateur a entièrement analysé l'élément et ses descendants ainsi que toutes les ressources référencées nécessaires à son rendu. |
| OnUnload | `10` | Se produit lorsque l'implémentation DOM supprime un document d'une fenêtre ou d'un cadre. Applicable uniquement aux éléments svg les plus externes. |
| OnAbort | `11` | Se produit lorsque le chargement de la page est arrêté avant qu'un élément n'ait été entièrement chargé. |
| OnError | `12` | Se produit lorsqu'un élément ne se charge pas correctement ou lorsqu'une erreur se produit lors de l'exécution du script. |
| OnResize | `13` | Se produit lorsqu'une vue de document est redimensionnée. Applicable uniquement aux éléments svg les plus externes. |
| OnScroll | `14` | Se produit lorsqu'une vue de document est décalée le long de l'axe X ou Y ou les deux. Applicable uniquement aux éléments svg les plus externes. |
| OnZoom | `15` | Se produit lorsque le niveau de zoom d'une vue de document est modifié. Applicable uniquement aux éléments svg les plus externes. |
| OnBegin | `16` | Se produit lorsqu'un élément d'animation commence. |
| OnEnd | `17` | Se produit lorsqu'un élément d'animation se termine. |
| OnRepeat | `18` | Se produit lorsqu'un élément d'animation se répète. |

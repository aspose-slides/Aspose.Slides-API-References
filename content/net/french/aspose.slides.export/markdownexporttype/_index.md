## Énumération MarkdownExportType

Type de document de rendu.

```csharp
public enum MarkdownExportType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Séquentiel | `0` | Rendre tous les éléments séparément. Un par un. |
| TexteUniquement | `1` | Rendre uniquement le texte. |
| Visuel | `2` | Rendre tous les éléments, les éléments qui sont groupés - rendre ensemble. |

### Exemples

Exemple:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    MarkdownSaveOptions markdownSaveOptions = new MarkdownSaveOptions
    {
        ShowHiddenSlides = true,
        ShowSlideNumber = true,
        Flavor = Flavor.Github,
        ExportType = MarkdownExportType.Sequential,
        NewLineType = NewLineType.Windows
    };
    
    pres.Save("doc.md", new[] { 1, 2, 3, 4, 5, 6, 7, 8, 9 }, SaveFormat.Md, markdownSaveOptions);
}
```

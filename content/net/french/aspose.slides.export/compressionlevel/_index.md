## CompressionLevel énumération

Spécifie les niveaux de compression ZIP pour le fichier OpenXML. Des niveaux plus élevés offrent une meilleure compression au détriment d'un traitement plus lent.

```csharp
public enum CompressionLevel
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| None | `0` | Aucune compression n'est appliquée. Les fichiers sont stockés tels quels. |
| Level1 | `1` | Compression la plus rapide avec le taux de compression le plus bas. |
| Level2 | `2` | Compression plus rapide avec un taux de compression légèrement meilleur que Level1. |
| Level3 | `3` | Fournit une meilleure compression que Level2 avec un impact de performance modéré. |
| Level4 | `4` | Fournit une meilleure compression que Level3. |
| Level5 | `5` | Fournit une compression améliorée par rapport à Level4 avec un temps de traitement supplémentaire. |
| Level6 | `6` | Compression standard, offrant un bon équilibre entre la vitesse de compression et la taille du fichier. Le niveau de compression par défaut. |
| Level7 | `7` | Fournit une compression supérieure à Level6 avec un traitement plus lent. |
| Level8 | `8` | Fournit une compression supérieure à Level7. |
| Level9 | `9` | Compression maximale. Produit la plus petite taille de fichier avec la vitesse de traitement la plus lente. |

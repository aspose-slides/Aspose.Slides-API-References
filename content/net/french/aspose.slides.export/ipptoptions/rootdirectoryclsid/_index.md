## IPptOptions.RootDirectoryClsid property

Représente le GUID de la classe d'objets (CLSID) qui est stocké dans l'entrée du répertoire racine. Peut être utilisé pour COM activation de l'application du document. La valeur par défaut est '64818D11-4F9B-11CF-86EA-00AA00B929E8' qui correspond à 'Microsoft Powerpoint.Slide.8'.

```csharp
public Guid RootDirectoryClsid { get; set; }
```

### Exemples

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    PptOptions pptOptions = new PptOptions();
    
    /// définir le CLSID sur 'Microsoft Powerpoint.Show.8'
    pptOptions.RootDirectoryClsid = new Guid("64818D10-4F9B-11CF-86EA-00AA00B929E8");
    
    pres.Save("pres.ppt", SaveFormat.Ppt, pptOptions);
}
```

## Méthode ILinkEmbedController.GetUrl

Renvoie une URL à un objet externe. Cette méthode est toujours appelée si [`GetObjectStoringLocation`](../getobjectstoringlocation) a renvoyé Link et peut être appelée si [`GetObjectStoringLocation`](../getobjectstoringlocation) a renvoyé Embed mais que l'intégration est impossible. Peut être appelée plusieurs fois pour le même identifiant d'objet.

```csharp
public string GetUrl(int id, int referrer)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| id | Int32 | Identifiant de l'objet. Cet identifiant est unique à l'opération dans son ensemble. |
| referrer | Int32 | identifiant de l'objet référent ou 0 si l'objet est référencé par le document racine. Peut être utilisé pour générer un lien relatif. |

### Valeur de retour

URL de l'objet externe ou null si cet objet doit être ignoré.

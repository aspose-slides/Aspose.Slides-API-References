## ILinkEmbedController.GetUrl méthode

Retourne une URL vers un objet externe. Cette méthode est toujours appelée si [`GetObjectStoringLocation`](../getobjectstoringlocation) a retourné Link et peut être appelée si [`GetObjectStoringLocation`](../getobjectstoringlocation) a retourné Embed mais que l'incorporation est impossible. Peut être appelée plusieurs fois pour le même identifiant d'objet.

```csharp
public string GetUrl(int id, int referrer)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| id | Int32 | Identifiant de l'objet. Cet identifiant est unique sur l'ensemble des opérations. |
| referrer | Int32 | identifiant de l'objet référent ou 0, si l'objet est référencé par le document racine. Peut être utilisé pour générer un lien relatif. |

### Valeur de retour

URL de l'objet externe ou null si cet objet doit être ignoré.

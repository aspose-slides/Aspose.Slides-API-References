## PdfOptions.BestImagesCompressionRatio propriété

Indique si la compression la plus efficace (au lieu de celle par défaut) pour chaque image doit être sélectionnée automatiquement. Si réglé sur Boolean.true, pour chaque image dans la présentation, l'algorithme de compression le plus approprié sera choisi, ce qui conduira à une taille plus petite du document PDF résultant. La sélection du meilleur ratio de compression d'image est gourmande en ressources informatiques et nécessite une quantité supplémentaire de RAM, et cette option est Boolean.false par défaut.

```csharp
public bool BestImagesCompressionRatio { get; set; }
```

### Remarques

Par défaut, c'est Boolean.false.

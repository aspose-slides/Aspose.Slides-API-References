## PdfOptions.EmbedTrueTypeFontsForASCII property

Détermine si Aspose.Slides incorporera les polices communes pour le texte ASCII (plage de codes 33..127). Les polices pour les codes de caractères supérieurs à 127 sont toujours incorporées. La liste des polices communes inclut les 14 polices de base du PDF et les polices supplémentaires spécifiées par l'utilisateur. Lire écrireBoolean .

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### Remarques

La valeur par défaut est **vrai** .

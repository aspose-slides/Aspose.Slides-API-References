## PdfOptions.EmbedTrueTypeFontsForASCII property

Détermine si Aspose.Slides doit incorporer des polices communes pour le texte ASCII (plage de codes 33..127). Les polices pour les codes de caractère supérieurs à 127 sont toujours intégrées. La liste des polices communes comprend les 14 polices de base de PDF et les polices supplémentaires spécifiées par l'utilisateur. Booléen en lecture/écriture.

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### Remarks

La valeur par défaut est **true**.

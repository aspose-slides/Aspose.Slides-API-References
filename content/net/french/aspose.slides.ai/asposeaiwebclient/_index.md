## AsposeAIWebClient classe

Une implémentation [`IAIWebClient`](../iaiwebclient) intégrée qui se connecte au LLM d'Aspose. Ceci est le client par défaut utilisé par le constructeur [`SlidesAIAgent`](../slidesaiagent/slidesaiagent) sans parametres.

```csharp
public sealed class AsposeAIWebClient : IAIWebClient, IDisposable
```

## Constructeurs

| Nom | Description |
| --- | --- |
| [AsposeAIWebClient](asposeaiwebclient#constructor)() | Crée une instance du client web Aspose AI qui se connecte au point de terminaison LLM Aspose par défaut. C’est le client utilisé par le constructeur [`SlidesAIAgent`](../slidesaiagent/slidesaiagent) sans parametres, donc le créer explicitement n’est nécessaire que lorsqu’on transmet le client directement au constructeur [`SlidesAIAgent`](../slidesaiagent/slidesaiagent). |
| [AsposeAIWebClient](asposeaiwebclient#constructor_1)(HttpClient) | Crée une instance du client web Aspose AI qui se connecte au point de terminaison LLM Aspose par défaut en utilisant un HttpClient géré à l’extérieur. Le HttpClient fourni n’est pas libéré par cette instance et reste la propriété du code appelant. |
| [AsposeAIWebClient](asposeaiwebclient#constructor_2)(string) | Crée une instance du client web Aspose AI qui se connecte à une URL de point de terminaison personnalisée. Utilisez cette surcharge lorsque vous avez une URL fournie par l’équipe Aspose.Slides; sinon, utilisez la surcharge [`AsposeAIWebClient`](./asposeaiwebclient) avec l’URL par défaut. |
| [AsposeAIWebClient](asposeaiwebclient#constructor_3)(string, HttpClient) | Crée une instance du client web Aspose AI qui se connecte à une URL de point de terminaison personnalisée en utilisant un HttpClient géré à l’extérieur. Le HttpClient fourni n’est pas libéré par cette instance et reste la propriété du code appelant. Utilisez cette surcharge lorsque vous disposez d’une URL fournie par l’équipe Aspose.Slides et que vous souhaitez fournir votre propre HttpClient; si vous avez seulement besoin de votre propre HttpClient avec l’URL par défaut, utilisez plutôt la surcharge [`AsposeAIWebClient`](./asposeaiwebclient). |

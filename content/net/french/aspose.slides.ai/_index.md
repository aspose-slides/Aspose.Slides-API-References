Contient des classes qui fournissent des fonctionnalités basées sur l'IA pour analyser et traiter les présentations PowerPoint.

## Classes

| Classe | Description |
| --- | --- |
| [AsposeAIWebClient](./asposeaiwebclient) | Une implémentation intégrée [`IAIWebClient`](../aspose.slides.ai/iaiwebclient) qui se connecte au LLM propriétaire d'Aspose. C’est le client par défaut utilisé par le constructeur sans paramètre [`SlidesAIAgent`](../aspose.slides.ai/slidesaiagent/slidesaiagent). |
| [OpenAICompatibleWebClient](./openaicompatiblewebclient) | Une implémentation intégrée [`IAIWebClient`](../aspose.slides.ai/iaiwebclient) qui se connecte à un fournisseur LLM compatible OpenAI à une URL de base spécifiée. |
| [OpenAIWebClient](./openaiwebclient) | Une implémentation intégrée [`IAIWebClient`](../aspose.slides.ai/iaiwebclient) qui se connecte à l'API OpenAI. |
| [SlidesAIAgent](./slidesaiagent) | Fournit des fonctionnalités alimentées par l'IA pour le traitement des présentations. |
| [SlidesAIAgentException](./slidesaiagentexception) | Représente les exceptions liées à Slides AI Agent. |

## Interfaces

| Interface | Description |

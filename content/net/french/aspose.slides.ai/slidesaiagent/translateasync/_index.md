```csharp
public Task TranslateAsync(IPresentation presentation, string language)
```

| Paramètre | Type | Description |

### Remarques

L'exemple ci-dessous utilise le [`AsposeAIWebClient`](../../asposeaiwebclient) par défaut, qui est créé par le constructeur sans paramètre [`SlidesAIAgent`](../slidesaiagent) et se connecte au LLM d'Aspose. Pour utiliser un fournisseur d'IA différent, fournissez votre propre LLM, ou personnalisez la connexion (par exemple, en fournissant votre propre `HttpClient`), passez une implémentation [`IAIWebClient`](../../iaiwebclient) au constructeur [`SlidesAIAgent`](../slidesaiagent). Les implémentations disponibles incluent :

* [`AsposeAIWebClient`](../../asposeaiwebclient)
* [`OpenAIWebClient`](../../openaiwebclient)
* [`OpenAICompatibleWebClient`](../../openaicompatiblewebclient)

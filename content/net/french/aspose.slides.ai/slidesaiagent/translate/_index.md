## SlidesAIAgent.Translate méthode

Traduit une présentation vers la langue spécifiée en utilisant l'IA (version synchrone).

```csharp
public void Translate(IPresentation presentation, string language)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| presentation | IPresentation | Présentation cible |
| language | String | Langue cible |

### Exceptions

| exception | condition |
| --- | --- |
| ArgumentNullException | L'instance de présentation n'est pas fournie |
| ArgumentException | La valeur de langue ne peut pas être nulle ou vide |

### Remarques

L'exemple ci-dessous utilise le [`AsposeAIWebClient`](../../asposeaiwebclient) par défaut, qui est créé par le constructeur [`SlidesAIAgent`](../slidesaiagent) sans paramètres et se connecte au LLM propriétaire d'Aspose. Pour utiliser un fournisseur d'IA différent, fournissez votre propre LLM, ou personnalisez la connexion (par exemple, en fournissant votre propre `HttpClient`), passez une implémentation [`IAIWebClient`](../../iaiwebclient) au constructeur [`SlidesAIAgent`](../slidesaiagent). Les implémentations disponibles incluent :

* [`AsposeAIWebClient`](../../asposeaiwebclient)
* [`OpenAIWebClient`](../../openaiwebclient)
* [`OpenAICompatibleWebClient`](../../openaicompatiblewebclient)

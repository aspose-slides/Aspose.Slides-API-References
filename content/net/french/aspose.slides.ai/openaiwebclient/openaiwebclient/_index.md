## OpenAIWebClient(string, string, string, HttpClient) {#constructor_1}

Crée une instance du client Web OpenAI qui utilise un HttpClient géré à l'extérieur. Le HttpClient fourni n’est pas libéré par cette instance et reste la responsabilité de l’appelant.

```csharp
public OpenAIWebClient(string model, string apiKey, string organizationId, HttpClient httpClient)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| model | String | OpenAI language model. Possible values: - gpt-4o - gpt-4o-mini - o1 - o1-mini - o3 - o3-mini |
| apiKey | String | OpenAI API key |
| organizationId | String | Organization ID (optional) |
| httpClient | HttpClient | Une instance `HttpClient` gérée à l'extérieur |

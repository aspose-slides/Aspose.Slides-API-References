## Propriété IEffect.AnimateTextType

Définit un type d'animation de texte pour l'effet. Le texte de la forme peut être animé lettre par lettre, mot par mot ou en une seule fois. Lecture/écriture `AnimateTextType`.

```csharp
public AnimateTextType AnimateTextType { get; set; }
```

### Exemples

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Obtenir le premier effet de la première diapositive.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Changer le type d'animation de texte de l'effet en "Par lettre"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByLetter;
}
```

## Propriété IEffect.AnimateTextType  
  
Définit un type de texte animé pour l'effet. Le texte de la forme peut être animé par lettre, par mot ou tout d'un coup. Lecture/écriture `AnimateTextType`.  
  
```csharp  
public AnimateTextType AnimateTextType { get; set; }  
```  
  
### Exemples  
  
```csharp  
[C#]  
using (Presentation presentation = new Presentation("demo.pptx"))  
{  
    // Obtenir le premier effet de la première diapositive.  
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];  
    
    // Changer le type de texte animé de l'effet en "Par lettre"  
    firstSlideEffect.AnimateTextType = AnimateTextType.ByLetter;  
}  
```

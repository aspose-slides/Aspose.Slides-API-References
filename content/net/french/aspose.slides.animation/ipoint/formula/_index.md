## IPoint.Formula property

Les formules dans les valeurs, from, to, by peuvent être composées de ce qui suit : Opérateurs arithmétiques standards : ‘+’, ‘-‘, ‘*’, ‘/’, ‘^’, ‘%’ (mod) Constantes : ‘pi’ ‘e’ Opérateurs conditionnels : ‘abs’, ‘min’, ‘max’, ‘?’ (if) Opérateurs de comparaison : '==', '&gt;=', '', '!=', '!' Opérateurs trigonométriques : ‘sin()’, ‘cos()’, ‘tan()’, ‘asin()’, ‘acos()’, ‘atan()’ Logarithme naturel ‘ln()’ Références de propriété (propriétés prises en charge par l'hôte) par exemple : "#ppt_x+(cos(-2*pi*(1-$))*-#ppt_x-sin(-2*pi*(1-$))*(1-#ppt_y))*(1-$)" Lire/écrire String.

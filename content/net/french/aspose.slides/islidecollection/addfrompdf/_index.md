## AddFromPdf(Stream) {#addfrompdf}

Crée des diapositives à partir du document PDF et les ajoute à la fin de la collection.

```csharp
public ISlide[] AddFromPdf(Stream pdfStream)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| pdfStream | Stream | Un flux qui sera utilisé comme source du document PDF |

### Return_Value

Diapositives ajoutées

### Exemples

Exemple :

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    using (Stream stream = new FileStream("document.pdf", FileMode.Open, FileAccess.Read, FileShare.Read))
    {
        pres.Slides.AddFromPdf(stream);
    }
    
    pres.Save("fromPdfDocument.pptx", SaveFormat.Pptx);
}
```

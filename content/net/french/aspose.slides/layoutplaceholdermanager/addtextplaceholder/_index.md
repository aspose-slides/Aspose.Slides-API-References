## LayoutPlaceholderManager.AddTextPlaceholder méthode

Ajoute une nouvelle forme de zone réservée à la diapositive de mise en page pour contenir du texte.

```csharp
public IAutoShape AddTextPlaceholder(float x, float y, float width, float height)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| x | Single | La coordonnée X de la nouvelle forme de zone réservée. |
| y | Single | La coordonnée Y de la nouvelle forme de zone réservée. |
| width | Single | La largeur de la nouvelle forme de zone réservée. |
| height | Single | La hauteur de la nouvelle forme de zone réservée. |

### Valeur de retour

[`IAutoShape`](../../iautoshape) créée avec une zone réservée de texte.

### Exemples

L'exemple suivant montre comment ajouter la forme de zone réservée de texte à la diapositive de mise en page.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTextPlaceholder(20, 20, 500, 300);
}
```

## Méthode IPresentationInfo.CheckPassword

Vérifie si un mot de passe est correct pour une présentation protégée par un mot de passe ouvert.

```csharp
public bool CheckPassword(string password)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| password | String | Le mot de passe à vérifier. |

### Valeur de retour

Vrai si la présentation est protégée par un mot de passe ouvert et que le mot de passe est correct, faux sinon.

### Remarques

Lorsque le mot de passe est null ou vide, cette méthode renvoie faux.

### Exemples

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo("pres.pptx");
bool isPasswordCorrect = info.CheckPassword("my_password");
```

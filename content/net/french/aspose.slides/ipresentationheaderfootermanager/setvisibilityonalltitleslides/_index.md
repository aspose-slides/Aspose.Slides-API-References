## IPresentationHeaderFooterManager.SetVisibilityOnAllTitleSlides méthode

Modifie la visibilité des espaces réservés pour le pied de page, la date-heure et le numéro de page pour toutes les diapositives de titre et pour la première diapositive de mise en page. Les diapositives de titre – diapositives basées sur la première diapositive de mise en page (quel que soit le type de cette première mise en page).

```csharp
public void SetVisibilityOnAllTitleSlides(bool isVisible)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| isVisible | Boolean | true - rend les espaces réservés visibles, sinon - les masque. |

## Méthode Portion.GetRect

Obtenez les coordonnées du rectangle qui délimite la portion. Le rectangle inclut toutes les lignes de texte de la portion, y compris celles vides.

```csharp
public RectangleF GetRect()
```

### Exemples

Exemple:

```csharp

using (Presentation pres = new Presentation())

   ISlide slide = pres.Slides[0];
   IAutoShape shape = slide.Shapes.AddAutoShape(ShapeType.Rectangle, 50, 50, 200, 50);

   shape.TextFrame.Paragraphs[0].Portions.Clear();
   var portion0 = new Portion("Some text");
   var portion1 = new Portion("GetRect text");
```

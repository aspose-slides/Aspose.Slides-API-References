## Propriété MasterSlide.ShowMasterShapes  

Spécifie si les formes sur le diapositif maître doivent être affichées sur les diapositives ou non. Pour le diapositif maître lui-même, cette propriété renvoie toujours `false`. Lecture/écriture Booléen.  

```csharp  
public override bool ShowMasterShapes { get; set; }  
```

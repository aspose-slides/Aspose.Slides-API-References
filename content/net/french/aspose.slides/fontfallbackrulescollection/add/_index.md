## Méthode FontFallBackRulesCollection.Add

Ajoutez une règle FallBack spécifiée à la fin de la collection.

```csharp
public void Add(IFontFallBackRule sourceRule)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| sourceRule | IFontFallBackRule | Règle spécifiée à ajouter |

### Exemples

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    //Récupération de la collection de règles vide ou préinitialisée depuis FontsManager
    IFontFallBackRulesCollection rulesList = pres.FontsManager.FontFallBackRulesCollection;

    //Ajout d'une nouvelle règle à la collection
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));
}
```

## Énumération LineSketchType

Représente le type de croquis ou l'effet qu'une forme a été assigné.

```csharp
public enum LineSketchType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| NotDefined | `-1` | Spécifie qu'un effet de croquis d'une forme est indéfini. |
| None | `0` | Spécifie qu'une forme n'a aucun effet de croquis. Cela équivaut à ce que cette propriété soit vide. |
| Curved | `1` | Spécifie qu'une forme a l'effet Courbé, qui transforme chaque bord de la forme en une grande courbe douce. |
| Freehand | `2` | Spécifie qu'une forme a l'effet Dessin à main levée, qui ressemble le plus à une ligne dessinée de manière imparfaite. |
| Scribble | `3` | Spécifie qu'une forme a l'effet Gribouillis, qui présente une oscillation exagérée comme si elle était dessinée de manière intentionnellement désordonnée. |

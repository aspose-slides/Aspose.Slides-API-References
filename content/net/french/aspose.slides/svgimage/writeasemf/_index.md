## Méthode SvgImage.WriteAsEmf

Sauvegarde l'image SVG en tant que fichier EMF.

```csharp
public void WriteAsEmf(Stream stream)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| stream | Stream | Flux cible |

### Exceptions

| exception | condition |
| --- | --- |
| ArgumentNullException | Le flux cible est **null** |

### Exemples

L'exemple suivant montre comment sauvegarder l'image SVG dans le fichier métropolitain.

```csharp
[C#]
// Crée une nouvelle image SVG
ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));

// Sauvegarde l'image SVG en tant que fichier métropolitain
using (var fileStream = System.IO.File.OpenWrite("SvgAsEmf.emf"))
    svgImage.WriteAsEmf(fileStream);
```

Cet échantillon démontre comment ajouter l'image SVG en tant que fichier métropolitain à la collection d'images de la présentation.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // Crée une nouvelle image SVG
    ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));
    using (var memStream = new MemoryStream())
    {
        // Sauvegarde l'image SVG en tant que fichier métropolitain
        svgImage.WriteAsEmf(memStream);
        // Ajoute le fichier métropolitain à la collection d'images
        pres.Images.AddImage(memStream.ToArray());
    }
}
```

## Méthode IShapeCollection.AddSummaryZoomFrame

Ajoute un nouvel objet Summary Zoom à la fin d'une collection.

```csharp
public ISummaryZoomFrame AddSummaryZoomFrame(float x, float y, float width, float height)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| x | Single | Coordonnée X d'un nouveau cadre de zoom de section Single. |
| y | Single | Coordonnée Y d'un nouveau cadre de zoom de section Single. |
| width | Single | Largeur d'un nouveau cadre de zoom de section Single. |
| height | Single | Hauteur d'un nouveau cadre de zoom de section Single. |

### Valeur de Retour

Objet Summary Zoom créé [`ISummaryZoomFrame`](../../isummaryzoomframe).

### Exceptions

| exception | condition |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | Il n'y a pas de sections dans la présentation, ou la diapositive cible n'appartient à aucune section. |

### Remarques

Cette méthode crée un nouveau Summary Zoom et y met une collection d'objets pour toutes les sections de cette présentation.

### Exemples

Cet exemple démontre comment ajouter un objet Summary Zoom à la fin d'une collection (supposez qu'il y ait au moins deux sections dans la présentation "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSummaryZoomFrame(150, 20, 500, 250);
}
```

### Exemples

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // pour définir le type "Navigué dans un kiosque (plein écran)" 
    pres.SlideShowSettings.SlideShowType = new BrowsedAtKiosk();
    
    // pour définir le type "Parcouru par individu (fenêtre)"
```

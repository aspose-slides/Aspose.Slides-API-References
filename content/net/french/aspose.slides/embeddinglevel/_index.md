## Énumération EmbeddingLevel

Représente les droits de licence pour l'incorporation de la police.

```csharp
[Flags]
public enum EmbeddingLevel : ushort
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Installable | `0` | Les polices avec ce paramètre indiquent qu'elles peuvent être incorporées et installées de manière permanente sur le système distant par une application. L'utilisateur du système distant acquiert les droits, obligations et licences identiques pour cette police que l'acheteur d'origine de la police, et est soumis au même contrat de licence utilisateur final, droits d'auteur, brevet de design et/ou marque déposée que l'était l'acheteur d'origine. |
| Restricted | `2` | Les polices qui ont uniquement ce bit activé ne doivent pas être modifiées, incorporées ou échangées de quelque manière que ce soit sans avoir d'abord obtenu la permission du propriétaire légal. |
| PreviewPrint | `4` | Lorsque ce bit est activé, la police peut être incorporée et temporairement chargée sur le système distant. Les documents contenant des polices Preview &amp; Print doivent être ouverts en mode "lecture seule;" aucune modification ne peut être appliquée au document. |
| Editable | `8` | Lorsque ce bit est activé, la police peut être incorporée mais doit uniquement être installée temporairement sur d'autres systèmes. Contrairement aux polices Preview &amp; Print, les documents contenant des polices Editables peuvent être ouverts pour la lecture, la modification est autorisée et les modifications peuvent être enregistrées. |
| NoSubsetting | `100` | Lorsque ce bit est activé, la police ne peut pas être sous-ensemencée avant l'incorporation. D'autres restrictions d'incorporation spécifiées dans les bits 0-3 et 9 s'appliquent également. |
| BitmapOnly | `200` | Lorsque ce bit est activé, seules les bitmaps contenues dans la police peuvent être incorporées. Aucune donnée de contour ne peut être incorporée. S'il n'y a pas de bitmaps disponibles dans la police, la police est considérée comme non incorporable et les services d'incorporation échoueront. |

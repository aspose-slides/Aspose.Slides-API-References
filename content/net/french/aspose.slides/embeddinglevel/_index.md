## Énumération Level d'intégration

Représente les droits de licence pour l'intégration de la police.

```csharp
[Flags]
public enum EmbeddingLevel : ushort
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Installable | `0` | Les polices avec ce paramètre indiquent qu'elles peuvent être intégrées et installées de manière permanente sur le système distant par une application. L'utilisateur du système distant acquiert les mêmes droits, obligations et licences pour cette police que l'acheteur original de la police, et est soumis au même contrat de licence utilisateur final, droit d'auteur, brevet de design et/ou marque déposée que l'acheteur original. |
| Restricted | `2` | Les polices qui n'ont que ce bit activé ne doivent pas être modifiées, intégrées ou échangées de quelque manière que ce soit sans avoir préalablement obtenu l'autorisation du propriétaire légal. |
| PreviewPrint | `4` | Lorsque ce bit est activé, la police peut être intégrée et temporairement chargée sur le système distant. Les documents contenant des polices Preview &amp; Print doivent être ouverts en mode "lecture seule" ; aucune modification ne peut être appliquée au document. |
| Editable | `8` | Lorsque ce bit est activé, la police peut être intégrée mais doit uniquement être installée temporairement sur d'autres systèmes. Contrairement aux polices Preview &amp; Print, les documents contenant des polices Editables peuvent être ouverts pour la lecture, les modifications sont autorisées et les changements peuvent être sauvegardés. |
| NoSubsetting | `100` | Lorsque ce bit est activé, la police ne peut pas être réduite en sous-ensemble avant l'intégration. D'autres restrictions d'intégration spécifiées dans les bits 0-3 et 9 s'appliquent également. |
| BitmapOnly | `200` | Lorsque ce bit est activé, seules les images bitmap contenues dans la police peuvent être intégrées. Aucune donnée d'outline ne peut être intégrée. S'il n'y a pas d'images bitmap disponibles dans la police, alors la police est considérée comme non intégrable et les services d'intégration échoueront. |

## Méthode PresentationInfo.CheckWriteProtection

Vérifie si un mot de passe pour modifier est correct pour une présentation protégée en écriture.

```csharp
public bool CheckWriteProtection(string password)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| password | String | Le mot de passe à vérifier. |

### Valeur de retour

Vrai si la présentation est protégée en écriture et que le mot de passe est correct. Faux sinon.

### Exceptions

| exception | condition |
| --- | --- |
| InvalidOperationException |  |

### Remarques

1. Vous devriez vérifier la propriété [`IsWriteProtected`](../iswriteprotected) avant d'appeler cette méthode. 2. Lorsque le mot de passe est null ou vide, cette méthode retourne faux.

### Exemples

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
if (info.IsWriteProtected == NullableBool.True)
{
    bool isWriteProtectedByPassword = info.CheckWriteProtection("my_password");
}
```

## Propriété LoadOptions.DefaultRegularFont

Renvoie ou définit la police régulière utilisée en cas de non disponibilité de la police source. Chaîne en lecture/écriture.

```csharp
public string DefaultRegularFont { get; set; }
```

### Exemples

L'exemple suivant montre comment définir des polices par défaut pour le rendu de présentations PowerPoint.

```csharp
[C#]
// Utilisez les options de chargement pour définir les polices régulières et asiatiques par défaut
LoadOptions loadOptions = new LoadOptions(LoadFormat.Auto);
loadOptions.DefaultRegularFont = "Wingdings";
loadOptions.DefaultAsianFont = "Wingdings";
// Charger la présentation
using (Presentation pptx = new Presentation("DefaultFonts.pptx", loadOptions))
{
    // Générer une miniature de la diapositive
    pptx.Slides[0].GetThumbnail(1, 1).Save("output_out.png", ImageFormat.Png);
    // Générer un PDF
    pptx.Save("output_out.pdf", SaveFormat.Pdf);
    // Générer un XPS
    pptx.Save("output_out.xps", SaveFormat.Xps);
}
```

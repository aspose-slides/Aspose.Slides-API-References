## IFontFallBackRule.IndexOf méthode  
  
Renvoie un index de la règle spécifiée dans la collection.  
  
```csharp  
public int IndexOf(string fontName)  
```  
  
| Paramètre | Type | Description |  
| --- | --- | --- |  
| fontName | String | Nom de la police à chercher. |  
  
### Valeur de retour  
  
Index d'une police ou -1 si la police n'est pas trouvée dans la liste.  
  
### Exemples  
  
```csharp  
[C#]  
// Créer une règle contenant une liste de polices.  
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");  
  
//Obtenir l'index de Tahoma  
int tahomaIndex = newRule.IndexOf("Tahoma");  
```

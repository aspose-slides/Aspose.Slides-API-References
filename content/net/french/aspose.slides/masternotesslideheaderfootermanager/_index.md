## MasterNotesSlideHeaderFooterManager class

Représente le gestionnaire qui contient le comportement du pied de page de la diapositive de notes maître, des espaces réservés pour la date-heure, le numéro de page et tous les espaces réservés enfants. Les espaces réservés enfants désignent les espaces réservés qui se trouvent sur les diapositives de notes dépendantes. Les diapositives de notes dépendantes utilisent et dépendent de la diapositive de notes maître.

```csharp
public sealed class MasterNotesSlideHeaderFooterManager : BaseHandoutNotesSlideHeaderFooterManager, 
    IMasterNotesSlideHeaderFooterManager
```

## MasterNotesSlideHeaderFooterManager.SetDateTimeAndChildDateTimesVisibility méthode

Modifie la visibilité de l'espace réservé pour la date-heure de la diapositive maître et de tous les espaces réservés de date-heure enfants. Les espaces réservés enfants désignent les espaces réservés contenus dans les diapositives de notes dépendantes. Les diapositives de notes dépendantes utilisent et dépendent de la diapositive de notes maître.

```csharp
public void SetDateTimeAndChildDateTimesVisibility(bool isVisible)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| isVisible | Boolean | true - rend les espaces réservés de date-heure visibles, sinon - les masque. |

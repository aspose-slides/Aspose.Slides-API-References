## ISlideShowTransition.AdvanceAfter propriété

Cet attribut spécifie si le diaporama passera à la diapositive suivante après un certain temps. Boolean en lecture et écriture.

```csharp
public bool AdvanceAfter { get; set; }
```

### Exemples

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    // Obtenir la transition de la première diapositive
    ISlideShowTransition slideTransition = pres.Slides[0].SlideShowTransition;
    
    // Vérifier si le flag Avancer la diapositive après est coché
    if (slideTransition.AdvanceAfter)
    {
        // Obtenir la valeur du temps d'avance après
        uint advanceAfterTime = slideTransition.AdvanceAfterTime;
    }
}
```

## Propriété ISlideShowTransition.AdvanceAfter

Cet attribut spécifie si le diaporama se déplacera vers la diapositive suivante après un certain temps. Lecture/écriture Booléen.

```csharp
public bool AdvanceAfter { get; set; }
```

### Exemples

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    // Obtenir la transition de la première diapositive
    ISlideShowTransition slideTransition = pres.Slides[0].SlideShowTransition;
    
    // Vérifier si le flag de passage à la diapositive suivante est cochée
    if (slideTransition.AdvanceAfter)
    {
        // Obtenir la valeur du temps de passage à la diapositive suivante
        uint advanceAfterTime = slideTransition.AdvanceAfterTime;
    }
}
```

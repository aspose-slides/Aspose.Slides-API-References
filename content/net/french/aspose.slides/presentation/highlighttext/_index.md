## HighlightText(string, Color) {#highlighttext}

Met en surbrillance toutes les occurrences du texte sample avec la couleur spécifiée.

```csharp
public void HighlightText(string text, Color highlightColor)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| text | String | Le texte à mettre en surbrillance. |
| highlightColor | Color | La couleur pour mettre le texte en surbrillance. |

### Exemples

L'exemple de code suivant montre comment mettre en surbrillance le texte dans une présentation PowerPoint.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	// mise en surbrillance de toutes les occurrences séparées de 'the'
	presentation.HighlightText("the", Color.Violet);
	presentation.Save("SomePresentation-out2.pptx", SaveFormat.Pptx);
}
```

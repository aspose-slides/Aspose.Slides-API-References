## ZoomObject.ShowBackground property

Obtient ou définit la valeur spécifiant si le Zoom utilisera l'arrière-plan de la diapositive de destination. Valeur booléenne en lecture/écriture. Valeur par défaut : true

```csharp
public bool ShowBackground { get; set; }
```

### Exemples

L'exemple démontre la suppression de l'arrière-plan d'une image d'un objet Zoom:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
    zoomFrame.ShowBackground = false;
}
```

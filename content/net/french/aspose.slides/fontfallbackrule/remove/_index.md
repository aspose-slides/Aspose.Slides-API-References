## FontFallBackRule.Remove méthode  

Supprime la première occurrence d'une police FallBack spécifique de la liste.  

```csharp  
public void Remove(string fontName)  
```  

| Paramètre | Type | Description |  
| --- | --- | --- |  
| fontName | String | Le nom de la police à supprimer de la liste. |  

### Exemples  

```csharp  
[C#]  
// Crée une règle contenant une liste de polices.  
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");  

// Supprimer Tahoma de la liste.  
newRule.Remove ("Tahoma");  
```

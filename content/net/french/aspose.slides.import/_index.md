Contient des classes pour importer des données dans des présentations Microsoft PowerPoint.

## Classes

| Classe | Description |
| --- | --- |
| [ExternalResourceResolver](./externalresourceresolver) | Classe de rappel utilisée pour résoudre les ressources externes lors de l'importation de documents Html et Svg. L'utilisation de ce résolveur pourrait créer une vulnérabilité lorsque le fichier HTML ou SVG fourni par le client poussera le logiciel serveur à obtenir un fichier local ou réseau. À utiliser avec précaution. Il est recommandé de ne pas spécifier le ExternalResourceResolver du tout (seuls les objets intégrés seront lus) ou de créer une sous-classe qui vérifie si l'uri spécifié est valide. |
| [PdfImportOptions](./pdfimportoptions) | Représente les options d'importation PDF |

Contient des classes pour importer des données dans des présentations Microsoft PowerPoint.

## Classes

| Classe | Description |
| --- | --- |
| [ExternalResourceResolver](./externalresourceresolver) | Classe de rappel utilisée pour résoudre des ressources externes lors de l'importation de documents Html, Svg. L'utilisation de ce résolveur pourrait créer une vulnérabilité lorsque le fichier HTML ou SVG fourni par le client amène le logiciel serveur à obtenir un fichier local ou réseau. Utilisez avec prudence. Il est recommandé de ne pas spécifier du tout l'ExternalResourceResolver (seuls les objets intégrés seront lus) ou de créer une sous-classe qui vérifie si l'URI spécifié est valide. |

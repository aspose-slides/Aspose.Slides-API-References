## Méthode Theme.GetEffective

Obtient les données de thème effectif avec l'héritage appliqué.

```csharp
public IThemeEffectiveData GetEffective()
```

### Valeur de retour

Un [`IThemeEffectiveData`](../../ithemeeffectivedata).

### Exemples

Cet exemple démontre comment obtenir les propriétés de thème effectif.

```csharp
[C#]
using (Presentation pres = new Presentation(@"MyPresentation.pptx"))
{
	IThemeEffectiveData effectiveTheme  = pres.Slides[0].ThemeManager.OverrideTheme.GetEffective();

	Console.WriteLine("Nom du schéma de police : " + effectiveTheme.FontScheme.Name);
	Console.WriteLine("Police latine majeure : " + effectiveTheme.FontScheme.Major.LatinFont.FontName);
	Console.WriteLine("Police latine mineure : " + effectiveTheme.FontScheme.Minor.LatinFont.FontName);
}
```

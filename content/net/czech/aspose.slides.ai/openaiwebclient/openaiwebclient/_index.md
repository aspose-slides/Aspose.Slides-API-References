## OpenAIWebClient(string, string, string, HttpClient) {#constructor_1}

Vytvoří instanci webového klienta OpenAI, který používá externě spravovaný HttpClient. Poskytnutý HttpClient není touto instancí uvolněn a zůstává vlastněn volajícím.

```csharp
public OpenAIWebClient(string model, string apiKey, string organizationId, HttpClient httpClient)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| model | String | OpenAI jazykový model. Možné hodnoty: - gpt-4o - gpt-4o-mini - o1 - o1-mini - o3 - o3-mini |
| apiKey | String | OpenAI API klíč |
| organizationId | String | ID organizace (volitelné) |
| httpClient | HttpClient | Externě spravovaná instance `HttpClient` |

### Výjimky

| výjimka | podmínka |
| --- | --- |
| ArgumentException | Hodnota klíče API nesmí být null nebo prázdná |
| ArgumentException | Hodnota textového modelu nesmí být null nebo prázdná |
| ArgumentNullException | HttpClient nesmí být null |

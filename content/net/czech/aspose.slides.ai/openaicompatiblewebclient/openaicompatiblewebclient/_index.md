## OpenAICompatibleWebClient(string, string, string, HttpClient) {#constructor_1}

Vytvoří instanci webového klienta kompatibilního s OpenAI, který používá externě spravovaný HttpClient. Poskytnutý HttpClient není touto instancí uvolněn a zůstává vlastněn volajícím.

```csharp

```

| Parametr | Typ | Popis |

| apiKey | String | API klíč (token). |

## MathBorderBox.StrikethroughBottomLeftToTopRight vlastnost

Přeškrtnutí z levého dolního rohu do pravého horního rohu (výchozí je false). Určuje, zda je přeškrtnutá úhlopříčná čára z levého dolního rohu do pravého horního rohu rámečku skrytá či zobrazena.

```csharp
public bool StrikethroughBottomLeftToTopRight { get; set; }
```

### Příklady

Příklad:

```csharp
[C#]
MathBorderBox borderBox = new MathBorderBox(new MathematicalText("x"));
borderBox.StrikethroughBottomLeftToTopRight = true;
```

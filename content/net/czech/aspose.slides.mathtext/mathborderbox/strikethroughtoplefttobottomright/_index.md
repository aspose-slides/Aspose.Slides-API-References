## MathBorderBox.StrikethroughTopLeftToBottomRight vlastnost

Strikethrough Top-Left to Bottom-Right (výchozí je false). Udává, zda je skrytý nebo zobrazený stav úhlopříčného přeškrtnutí od levého horního rohu po pravý dolní roh rámečku.

```csharp
public bool StrikethroughTopLeftToBottomRight { get; set; }
```

### Příklady

Example:

```csharp
[C#]
MathBorderBox borderBox = new MathBorderBox(new MathematicalText("x"));
borderBox.StrikethroughTopLeftToBottomRight = true;
```

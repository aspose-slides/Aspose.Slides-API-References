## MathElementBase.ToBox metoda

Umístí tento prvek do neviditelného boxu (logické seskupení), který se používá ke skupinování součástí rovnice nebo jiného výskytu matematického textu. Objekty v boxu mohou (například) sloužit jako emulátor operátoru s nebo bez zarovnávacího bodu, sloužit jako bod zalomení řádku nebo být seskupeny tak, aby neumožňovaly vnitřní zalomení řádku.

```csharp
public IMathBox ToBox()
```

### Návratová hodnota

Logický box s tímto prvkem umístěným uvnitř

### Příklady

Příklad:

```csharp
[C#]
IMathBox box = new MathematicalText("x:=y").ToBox();
```

## IMathPhantom.Transp vlastnost

Získá nebo nastaví hodnotu určující, zda je phantom průhledný pro pravidla mezery založená na třídě.

```csharp
public bool Transp { get; set; }
```

### Poznámky

Když je `true`, operátory a symboly uvnitř phantomu stále ovlivňují matematické mezery kolem phantomu (jako kdyby byl viditelný). Když je `false`, třídně založené mezery jsou ignorovány. Odpovídá atributu OMML `m:transp`.

## IMathPhantom.ZeroAsc vlastnost

Získá nebo nastaví hodnotu, která určuje, zda má být vzestup (výška nad základní čárou) základního prvku považován za nulu.

```csharp
public bool ZeroAsc { get; set; }
```

### Poznámky

Když je `true`, phantom nezvedá základní čáru okolní matematické řádky. Odpovídá atributu OMML `m:zeroAsc`.

## IMathPhantom.Show vlastnost

Získá nebo nastaví hodnotu určující, zda je základní prvek zobrazen.

```csharp
public bool Show { get; set; }
```

### Poznámky

Když je `false`, základní prvek je skrytý, ale může stále zabírat místo v závislosti na dalších nastaveních phantom. Odpovídá atributu OMML `m:show`.

## MathMatrix.ColumnGap vlastnost

Hodnota vodorovného odsazení mezi sloupci matice; pokud je ColumnGapRule nastavena na 3 („Exactly“), jednotka se interpretuje jako twip (1/20 bodu). Pokud je ColumnGapRule nastavena na 4 („Multiple“), jednotka se interpretuje jako počet 0,5 em kroků. V ostatních případech je ignorována. Výchozí hodnota: 0

```csharp
public uint ColumnGap { get; set; }
```

### Příklady

Example:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.ColumnGapRule = MathSpacingRules.Exactly;
matrix.ColumnGap = 20;
```

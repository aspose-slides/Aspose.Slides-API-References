## MathMatrix.RowGap vlastnost

Hodnota svislé mezery mezi řádky matice; pokud je RowGapRule nastaveno na 3 („Exactly“), jednotka se interpretuje jako twips (1/20 bodu). Pokud je RowGapRule nastaveno na 4 („Multiple“), jednotka se interpretuje jako půl-řádky. Výchozí: 0

```csharp
public uint RowGap { get; set; }
```

### Příklady

Příklad:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

## IMathBox.OperatorEmulator vlastnost

Operator Emulator. Když je nastaveno na true, krabice a její obsah se chovají jako jeden operátor a dědí vlastnosti operátoru. To znamená například, že znak může sloužit jako místo pro zalomení řádku a může být zarovnán k dalším operátorům. Emulátory operátorů se často používají, když se jeden nebo více glyfů spojí a vytvoří operátor, například '=='. Výchozí hodnota: false

```csharp
public bool OperatorEmulator { get; set; }
```

### Příklady

Příklad:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.OperatorEmulator = true;
```

## IMathBox.ExplicitBreak vlastnost

ExplicitBreak určuje, zda na začátku objektu Box existuje zalomení řádku, takže řádek zalomí na začátku objektu Box. Určuje číslo operátoru na předchozím řádku matematického textu, které bude použito jako zarovnávací bod pro aktuální řádek matematického textu. Možné hodnoty: 1..255 Výchozí: 0 (žádné explicitní zalomení)

```csharp
public byte ExplicitBreak { get; set; }
```

### Příklady

Příklad:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.ExplicitBreak = 1;
```

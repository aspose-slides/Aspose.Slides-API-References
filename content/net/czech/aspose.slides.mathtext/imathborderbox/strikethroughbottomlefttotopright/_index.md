## IMathBorderBox.StrikethroughBottomLeftToTopRight vlastnost

Strikethrough Bottom-Left to Top-Right (výchozí hodnota je false). Určuje skrytý či zobrazený stav přeškrtnuté úhlopříčky od levého dolního rohu k pravému hornímu rohu okrajového pole.

```csharp
public bool StrikethroughBottomLeftToTopRight { get; set; }
```

### Příklady

Example:

```csharp
[C#]
IMathBorderBox borderBox = new MathematicalText("x+y+z").ToBorderBox();
borderBox.StrikethroughBottomLeftToTopRight = true;
```

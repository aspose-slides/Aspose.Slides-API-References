## IMathElementCollection.IndexOf metoda

Určuje index konkrétního matematického elementu ve sbírce.

```csharp
public int IndexOf(IMathElement item)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| item | IMathElement | Element, který se má ve sbírce vyhledat. |

### Návratová hodnota

Index *item*, pokud je ve sbírce nalezen; jinak -1.

### Příklady

Příklad:

```csharp
[C#]
IMathElementCollection collection = new MathBlock(new MathematicalText("x"));
MathematicalText plusElement = new MathematicalText("+");
collection.Add(plusElement);
collection.Add(new MathRadical(new MathematicalText("x"), new MathematicalText("3")));
int index = collection.IndexOf(plusElement);
```

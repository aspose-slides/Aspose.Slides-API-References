## MathArray.RowSpacing vlastnost

Mezery mezi řádky pole. Používá se pouze když je RowSpacingRule nastaven na 3 Exactly, v takovém případě je jednotkou míry body nebo Multiple, v takovém případě je jednotkou míry půlřádky. Default: 0

```csharp
public uint RowSpacing { get; set; }
```

### Příklady

Příklad:

```csharp
[C#]
IMathArray mathArray = new MathArray(new MathematicalText("item1"));
mathArray.RowSpacingRule = MathRowSpacingRule.Exactly;
mathArray.RowSpacing = 10;
```

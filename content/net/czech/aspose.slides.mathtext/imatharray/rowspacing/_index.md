## IMathArray.RowSpacing vlastnost

Mezery mezi řádky pole. Používá se pouze když je RowSpacingRule nastaven na 3, přesně tehdy, když je jednotkou měření body, nebo Multiple, když je jednotkou měření půlřádky. Výchozí: 0

public uint RowSpacing { get; set; }

## IMathMatrix.RowGap vlastnost

Hodnota vertikálního rozestupu mezi řádky matice; pokud je RowGapRule nastaveno na 3 ("Exactly"), pak je jednotka interpretována jako twips (1/20 bodu) pokud je RowGapRule nastaveno na 4 ("Multiple"), pak je jednotka interpretována jako půlřádky. Výchozí: 0

```csharp
public uint RowGap { get; set; }
```

### Příklady

Příklad:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

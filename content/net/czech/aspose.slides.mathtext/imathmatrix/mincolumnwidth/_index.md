## IMathMatrix.MinColumnWidth vlastnost

Minimální šířka sloupce v jednotkách twips (1/20 bodu). Rozestup (také označovaný jako „Column Gap“ nebo „Gap Width“) se přičítá k MinColumnWidth, aby se určil celkový Matrix Column Spacing (vzdálenost mezi stejnými okraji různých sloupců). Výchozí: 0.

```csharp
public uint MinColumnWidth { get; set; }
```

### Příklady

Příklad:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.MinColumnWidth = 20;
```

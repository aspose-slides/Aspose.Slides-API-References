## IMathMatrix.ColumnGap vlastnost

Hodnota vodorovného rozestupu mezi sloupci matice; pokud je ColumnGapRule nastaven na 3 ("Exactly"), jednotka se interpretuje jako twips (1/20 bodu) pokud je ColumnGapRule nastaven na 4 ("Multiple"), jednotka se interpretuje jako počet 0,5 em kroků. V ostatních případech se ignoruje. Výchozí: 0

```csharp
public uint ColumnGap { get; set; }
```

### Příklady

Příklad:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.ColumnGapRule = MathSpacingRules.Exactly;
matrix.ColumnGap = 20;
```

## IMathGroupingCharacter.VerticalJustification vlastnost

Vertikální zarovnání skupinového znaku. Určuje zarovnání objektu vzhledem k základní linii. Například když je skupinový znak nad objektem, VerticalJustification hodnoty Top znamená, že horní část objektu leží na základní linii; když je VerticalJustification nastaven na Bottom, spodní část objektu leží na základní linii. Výchozí: Bottom pro Position=Top a Top pro Position=Bottom

```csharp
public MathTopBotPositions VerticalJustification { get; set; }
```

### Příklady

Příklad:

```csharp
[C#]
MathGroupingCharacter groupingCharacter = new MathGroupingCharacter(new MathematicalText("abc"));
groupingCharacter.VerticalJustification = MathTopBotPositions.Top;
```

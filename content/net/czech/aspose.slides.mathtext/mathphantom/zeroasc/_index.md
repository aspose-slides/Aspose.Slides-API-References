## MathPhantom.ZeroAsc vlastnost

Získá nebo nastaví hodnotu určující, zda má být výška (vzdálenost nad základní čarou) základního prvku považována za nulu.

```csharp
public bool ZeroAsc { get; set; }
```

### Poznámky

Když je `true`, phantom nezvedá základní čáru okolní řádky rovnice. Odpovídá atributu OMML `m:zeroAsc`.

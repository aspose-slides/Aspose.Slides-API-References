## IMathBlockCollection.Contains metoda

Určuje, zda kolekce obsahuje konkrétní hodnotu.

```csharp
public bool Contains(IMathBlock item)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| item | IMathBlock | Objekt, který má být v kolekci nalezen. |

### Návratová hodnota

true, pokud je *item* v kolekci nalezen; jinak false.

### Příklady

Příklad:

```csharp
[C#]
IMathBlockCollection blockCollection = new MathParagraph();
IMathBlock block = new MathBlock(new MathematicalText("y"));
blockCollection.Add(block);
bool contains = blockCollection.Contains(block);
```

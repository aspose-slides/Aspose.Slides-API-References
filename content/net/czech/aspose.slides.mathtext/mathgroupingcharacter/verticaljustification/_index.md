## MathGroupingCharacter.VerticalJustification vlastnost

Vertikální zarovnání skupinového znaku. Určuje zarovnání objektu vzhledem k základní linii. Například když je skupinový znak nad objektem, VerticalJustification Top znamená, že horní část objektu leží na základní linii; když je VerticalJustification nastaveno na Bottom, spodní část objektu je na základní linii. Výchozí: Bottom pro Position=Top a Top pro Position=Bottom

```csharp
public MathTopBotPositions VerticalJustification { get; set; }
```

### Příklady

Příklad:

```csharp
[C#]
MathGroupingCharacter groupingCharacter = new MathGroupingCharacter(new MathematicalText("abc"));
groupingCharacter.VerticalJustification = MathTopBotPositions.Top;
```

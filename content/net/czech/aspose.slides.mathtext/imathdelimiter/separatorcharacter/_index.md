## IMathDelimiter.SeparatorCharacter vlastnost

Delimiter Separator Character určuje znak, který odděluje argumenty v objektu oddělovače. Výchozí hodnota: '&#x7C;'.

```csharp
public char SeparatorCharacter { get; set; }
```

### Příklady

Příklad:


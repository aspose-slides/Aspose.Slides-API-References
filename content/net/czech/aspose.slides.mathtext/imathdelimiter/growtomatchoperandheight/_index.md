## IMathDelimiter.GrowToMatchOperandHeight vlastnost

Určuje růst **BeginningCharacter**, **SeparatorCharacter**, **EndingCharacter**. Když je true, oddělovače rostou vertikálně, aby odpovídaly výšce jejich operandu. Výchozí hodnota je true

```csharp
public bool GrowToMatchOperandHeight { get; set; }
```

### Příklad

Příklad:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.GrowToMatchOperandHeight = false;
```

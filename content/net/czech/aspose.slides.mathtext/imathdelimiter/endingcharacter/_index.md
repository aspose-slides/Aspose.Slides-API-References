## IMathDelimiter.EndingCharacter vlastnost

Delimiter Ending Character určuje koncový, nebo uzavírací, znak oddělovače. Matematické oddělovače jsou obklopující znaky, jako jsou závorky, hranaté závorky a složené závorky. Výchozí hodnota: ')'.

```csharp
public char EndingCharacter { get; set; }
```

### Příklady

Příklad:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Join("y").Enclose();
delimiter.EndingCharacter = ']';
```

## MathDelimiter.DelimiterShape vlastnost

Určuje tvar ohraničovačů v objektu delimiter. Když je MathDelimiterShape.Centered, jsou ohraničovače centrovány kolem matematické osy matematického textu a zároveň jsou upraveny tak, aby zaplnily celou výšku jejich obsahu. Když je MathDelimiterShape.Match, jejich výška a tvar jsou změněny tak, aby přesně odpovídaly jejich obsahu.

```csharp
public MathDelimiterShape DelimiterShape { get; set; }
```

### Příklady

Příklad:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.DelimiterShape = MathDelimiterShape.Match;
```

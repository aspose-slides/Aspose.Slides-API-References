## MathDelimiter.EndingCharacter vlastnost

Koncový znak oddělovače určuje ukončovací nebo zavírací znak oddělovače. Matematické oddělovače jsou uzavírací znaky, jako jsou kulaté závorky, hranaté závorky a složené závorky. Výchozí hodnota: ')'.

```csharp
public char EndingCharacter { get; set; }
```

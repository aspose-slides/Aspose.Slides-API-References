## MathDelimiter.GrowToMatchOperandHeight vlastnost

Určuje růst znaků BeginningCharacter, SeparatorCharacter, EndingCharacter. Pokud je true, oddělovače se vertikálně roztahují tak, aby odpovídaly výšce jejich operandu. Výchozí hodnota je true

```csharp
public bool GrowToMatchOperandHeight { get; set; }
```

### Příklady

Příklad:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.GrowToMatchOperandHeight = false;
```

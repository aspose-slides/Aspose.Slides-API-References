## MathBox.NoBreak vlastnost

Bez zalomení Tato vlastnost určuje vlastnost "unbreakable" na objektové krabici. Když je true, v krabici nemohou nastat žádné zalomení řádku. To může být důležité pro emulátory operátorů, které se skládají z více než jednoho binárního operátoru. Když tento prvek není specifikován, mohou v krabici nastat zalomení. Výchozí: true

```csharp
public bool NoBreak { get; set; }
```

### Příklady

Příklad:

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("*****"));
box.NoBreak = false;
```

## MathBox.OperatorEmulator vlastnost

Emulátor operátoru. Když je hodnota true, krabice a její obsah se chovají jako jedinečný operátor a dědí vlastnosti operátoru. To znamená například, že znak může sloužit jako místo pro zalomení řádku a může být zarovnán k dalším operátorům. Emulátory operátorů se často používají, když jeden nebo více glyfů tvoří operátor, například '=='. Výchozí hodnota: false

```csharp
public bool OperatorEmulator { get; set; }
```

### Příklady

Příklad:

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("=="));
box.OperatorEmulator = true;
```

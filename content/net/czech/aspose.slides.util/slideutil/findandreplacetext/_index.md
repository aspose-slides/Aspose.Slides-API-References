## SlideUtil.FindAndReplaceText metoda

Vyhledá a nahradí text v prezentaci s daným formátem

```csharp
public static void FindAndReplaceText(IPresentation presentation, bool withMasters, string find, 
    string replace, PortionFormat format = null)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| presentation | IPresentation | Prohledávaná prezentace. |
| withMasters | Boolean | Určuje, zda mají být skenovány hlavní snímky. |
| find | String | Hodnota řetězce k vyhledání. |
| replace | String | Hodnota řetězce k nahrazení. |
| format | PortionFormat | Formát pro nahrazení části textu. Pokud je null, bude použit formát prvního znaku nalezeného řetězce. |

### Příklady

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PortionFormat format = new PortionFormat
    {
        FontHeight = 24f,
        FontItalic = NullableBool.True,
        FillFormat =
        {
            FillType = FillType.Solid,
            SolidFillColor =
            {
                Color = Color.Red
            }
        }
    };
    Aspose.Slides.Util.SlideUtil.FindAndReplaceText(pres, true, "[this block] ", "my text ", format);
    pres.Save("replaced", SaveFormat.Pptx);
}
```

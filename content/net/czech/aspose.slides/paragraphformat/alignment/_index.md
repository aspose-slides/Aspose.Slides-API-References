## Vlastnost ParagraphFormat.Alignment

Vrací nebo nastavuje zarovnání textu v odstavci bez dědičnosti. Čtení/zápis [`TextAlignment`](../../textalignment).

```csharp
public TextAlignment Alignment { get; set; }
```

### Příklady

Následující ukázkový kód ukazuje, jak zarovnat textové odstavce v prezentaci PowerPoint.

```csharp
[C#]
// Vytvořte objekt Presentation, který představuje soubor PPTX
using (Presentation pres = new Presentation("ParagraphsAlignment.pptx"))
{
    // Přístup k prvnímu snímku
    ISlide slide = pres.Slides[0];
    // Přístup k prvnímu a druhému zástupnému objektu (placeholder) na snímku a přetypování na AutoShape
    ITextFrame tf1 = ((IAutoShape)slide.Shapes[0]).TextFrame;
    ITextFrame tf2 = ((IAutoShape)slide.Shapes[1]).TextFrame;
    // Změňte text v obou zástupných objektech
    tf1.Text = "Center Align by Aspose";
    tf2.Text = "Center Align by Aspose";
    // Získání prvního odstavce ze zástupných objektů
    IParagraph para1 = tf1.Paragraphs[0];
    IParagraph para2 = tf2.Paragraphs[0];
    // Zarovnání odstavce textu na střed
    para1.ParagraphFormat.Alignment = TextAlignment.Center;
    para2.ParagraphFormat.Alignment = TextAlignment.Center;
    //Zapis prezentace jako soubor PPTX
    pres.Save("Centeralign_out.pptx", SaveFormat.Pptx);
}
```

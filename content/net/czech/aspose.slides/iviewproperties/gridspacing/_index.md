## IViewProperties.GridSpacing vlastnost

Vrací nebo nastavuje rozestup mřížky, který se má použít pro mřížku podkladového dokumentu prezentace, v bodech. Číst/psát Single.

```csharp
public float GridSpacing { get; set; }
```

### Poznámky

Hodnota rozestupu mřížky musí být kladné číslo. Typický rozsah hodnot je od 1 mm (2.8349607 points) do 2 palců (144 points).

### Příklady

Následující ukázkový kód ukazuje, jak změnit rozestup mřížky v prezentaci PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

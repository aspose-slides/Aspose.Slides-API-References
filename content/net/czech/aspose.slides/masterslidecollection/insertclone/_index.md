## MasterSlideCollection.InsertClone metoda

Vloží kopii určeného master snímku na zadanou pozici v kolekci. Propojené snímky rozvržení budou také zkopírovány.

```csharp
public IMasterSlide InsertClone(int index, IMasterSlide sourceMaster)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| index | Int32 | Index nového snímku. |
| sourceMaster | IMasterSlide | Snímek, který se má klonovat. |

### Návratová hodnota

Vložený master snímek.

### Příklady

Následující příklad ukazuje, jak klonovat master snímek v jiné prezentaci PowerPoint.

```csharp
[C#]
// Vytvořte instanci třídy Presentation k načtení zdrojového souboru prezentace
using (Presentation srcPres = new Presentation("CloneToAnotherPresentationWithMaster.pptx"))
{
    // Vytvořte instanci třídy Presentation pro cílovou prezentaci (kde bude snímek klonován)
    using (Presentation destPres = new Presentation())
    {
        // Vytvořte instanci ISlide ze sbírky snímků ve zdrojové prezentaci spolu s
        // master snímkem
        ISlide SourceSlide = srcPres.Slides[0];
        IMasterSlide SourceMaster = SourceSlide.LayoutSlide.MasterSlide;
		// Získejte master snímky cílové prezentace
        IMasterSlideCollection masters = destPres.Masters;
        // Zkopírujte požadovaný master snímek ze zdrojové prezentace do kolekce master snímků v
        // cílové prezentaci
        IMasterSlide iSlide = masters.AddClone(SourceMaster);
        // Kolekce snímků v cílové prezentaci
        ISlideCollection slds = destPres.Slides;
		// Klonujte zdrojový snímek do kolekce snímků cílové prezentace.
        slds.AddClone(SourceSlide, iSlide, true);
        // Uložte cílovou prezentaci na disk
        destPres.Save("CloneToAnotherPresentationWithMaster_out.pptx", SaveFormat.Pptx);
    }
}
```

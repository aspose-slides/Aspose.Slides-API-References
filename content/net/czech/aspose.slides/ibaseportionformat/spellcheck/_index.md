## IBasePortionFormat.SpellCheck vlastnost

Získá nebo nastaví hodnotu, která určuje, zda je pro textový úsek povolena kontrola pravopisu. Když je tato vlastnost nastavena na false, kontroly pravopisu pro textové prvky jsou potlačeny. Když je nastavena na true, kontrola pravopisu je povolena. Výchozí hodnota je `false`.

```csharp
public bool SpellCheck { get; set; }
```

### Příklady

Next example demonstrates enabling the SpellCheck flag before saving the presentation:

```csharp
[C#]
using (var pres = new Presentation("input.pptx"))
{
    // Přístup k prvnímu úseku textu uvnitř první tvary na první snímek
    var portion = ((AutoShape)pres.Slides[0].Shapes[0]).TextFrame.Paragraphs[0].Portions[0];
    // Povolit kontrolu pravopisu pro tento úsek textu
    portion.PortionFormat.SpellCheck = true;
    // Uložit upravenou prezentaci
    pres.Save("output-with-spellcheck.pptx", SaveFormat.Pptx);
}
```

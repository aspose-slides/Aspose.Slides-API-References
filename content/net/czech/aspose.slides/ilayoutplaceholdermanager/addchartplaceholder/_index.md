## ILayoutPlaceholderManager.AddChartPlaceholder metoda

Přidá nový tvar placeholderu do snímku rozvržení, aby obsahoval Chart.

```csharp
public IAutoShape AddChartPlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| x | Single | Souřadnice X nového tvaru placeholderu. |
| y | Single | Souřadnice Y nového tvaru placeholderu. |
| width | Single | Šířka nového tvaru placeholderu. |
| height | Single | Výška nového tvaru placeholderu. |

### Vrácená hodnota

Vytvořeno [`IAutoShape`](../../iautoshape) se zástupcem Chart.

### Příklady

Následující příklad ukazuje, jak přidat tvar placeholderu Chart do snímku rozvržení.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
    IAutoShape placeholder = layout.PlaceholderManager.AddChartPlaceholder(20, 20, 200, 200);
}
```

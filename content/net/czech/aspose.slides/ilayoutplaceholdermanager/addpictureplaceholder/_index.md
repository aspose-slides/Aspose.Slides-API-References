## ILayoutPlaceholderManager.AddPicturePlaceholder method

Přidá nový tvar zástupce do rozložení snímku pro umístění obrázku.

```csharp
public IAutoShape AddPicturePlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| x | Single | Souřadnice X nového tvaru zástupce. |
| y | Single | Souřadnice Y nového tvaru zástupce. |
| width | Single | Šířka nového tvaru zástupce. |
| height | Single | Výška nového tvaru zástupce. |

### Návratová hodnota

Vytvořen [`IAutoShape`](../../iautoshape) s placeholderem obrázku.

### Příklady

Následující příklad ukazuje, jak přidat tvar placeholderu obrázku do rozložení snímku.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddPicturePlaceholder(20, 20, 200, 200);
}
```

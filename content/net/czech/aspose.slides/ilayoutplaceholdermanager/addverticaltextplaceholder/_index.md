## ILayoutPlaceholderManager.AddVerticalTextPlaceholder metoda

Přidá nový placeholder shape do rozvržení snímku, aby držel textový obsah ve svislém směru.

```csharp
public IAutoShape AddVerticalTextPlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| x | Single | Souřadnice X nového placeholder shape. |
| y | Single | Souřadnice Y nového placeholder shape. |
| width | Single | Šířka nového placeholder shape. |
| height | Single | Výška nového placeholder shape. |

### Návratová hodnota

Vytvořen [`IAutoShape`](../../iautoshape) s Text (Vertical) placeholder.

### Příklady

Následující příklad ukazuje, jak přidat Text (Vertical) placeholder shape do rozvržení snímku.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalTextPlaceholder(20, 20, 300, 500);
}
```

## ILayoutPlaceholderManager.AddTextPlaceholder method

Přidá nový placeholder shape do layout slide, aby obsahoval textový obsah.

```csharp
public IAutoShape AddTextPlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| x | Single | Souřadnice X nového placeholder shape. |
| y | Single | Souřadnice Y nového placeholder shape. |
| width | Single | Šířka nového placeholder shape. |
| height | Single | Výška nového placeholder shape. |

### Návratová hodnota

Created [`IAutoShape`](../../iautoshape) with a Text placeholder.

### Příklady

Následující příklad ukazuje, jak přidat Text placeholder shape do layout slide.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTextPlaceholder(20, 20, 500, 300);
}
```

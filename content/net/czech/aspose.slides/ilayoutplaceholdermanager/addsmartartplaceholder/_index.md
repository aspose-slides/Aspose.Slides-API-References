## Metoda ILayoutPlaceholderManager.AddSmartArtPlaceholder

Přidá nový tvar zástupce do snímku rozvržení, aby obsahoval diagram SmartArt.

```csharp
public IAutoShape AddSmartArtPlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| x | Single | Souřadnice X nového tvaru zástupce. |
| y | Single | Souřadnice Y nového tvaru zástupce. |
| width | Single | Šířka nového tvaru zástupce. |
| height | Single | Výška nového tvaru zástupce. |

### Návratová hodnota

Vytvořeno [`IAutoShape`](../../iautoshape) se zástupcem SmartArt.

### Příklady

Následující příklad ukazuje, jak přidat tvar zástupce SmartArt do snímku rozvržení.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddSmartArtPlaceholder(20, 20, 200, 200);
}
```

## ILayoutPlaceholderManager.AddMediaPlaceholder metoda

Přidá nový tvar zástupného elementu do rozložení snímku, který bude obsahovat mediální objekt.

```csharp
public IAutoShape AddMediaPlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| x | Single | Souřadnice X nového tvaru zástupného elementu. |
| y | Single | Souřadnice Y nového tvaru zástupného elementu. |
| width | Single | Šířka nového tvaru zástupného elementu. |
| height | Single | Výška nového tvaru zástupného elementu. |

### Návratová hodnota

Vytvořeno [`IAutoShape`](../../iautoshape) s Media placeholder.

### Příklady

Následující příklad ukazuje, jak přidat tvar Media placeholder do rozložení snímku.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddMediaPlaceholder(20, 20, 200, 200);
}
```

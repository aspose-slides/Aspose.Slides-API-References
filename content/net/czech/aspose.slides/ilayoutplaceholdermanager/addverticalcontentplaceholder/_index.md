## ILayoutPlaceholderManager.AddVerticalContentPlaceholder method

Přidá nový tvar zástupného prvku do snímku rozvržení, který bude obsahovat obsah, například obrázek, tabulku, média nebo text ve svislém směru.

```csharp
public IAutoShape AddVerticalContentPlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| x | Single | Souřadnice X nového tvaru zástupného prvku. |
| y | Single | Souřadnice Y nového tvaru zástupného prvku. |
| width | Single | Šířka nového tvaru zástupného prvku. |
| height | Single | Výška nového tvaru zástupného prvku. |

### Návratová hodnota

Vytvořeno [`IAutoShape`](../../iautoshape) s Content (Vertical) placeholderem.

### Příklady

Následující příklad ukazuje, jak přidat tvar Content (Vertical) placeholder do snímku rozvržení.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalContentPlaceholder(20, 20, 300, 500);
}
```

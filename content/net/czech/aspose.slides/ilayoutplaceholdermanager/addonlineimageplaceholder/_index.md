## ILayoutPlaceholderManager.AddOnlineImagePlaceholder metoda

Přidá nový tvar zástupného objektu do rozvržení snímku, aby uchovával online obrázek.

```csharp
public IAutoShape AddOnlineImagePlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| x | Single | Souřadnice X nového tvaru zástupného objektu. |
| y | Single | Souřadnice Y nového tvaru zástupného objektu. |
| width | Single | Šířka nového tvaru zástupného objektu. |
| height | Single | Výška nového tvaru zástupného objektu. |

### Návratová hodnota

Vytvořeno [`IAutoShape`](../../iautoshape) s online obrázkovým zástupným objektem.

### Příklady

Následující příklad ukazuje, jak přidat tvar online obrázkového zástupného objektu do rozvržení snímku.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddOnlineImagePlaceholder(20, 20, 200, 200);
}
```

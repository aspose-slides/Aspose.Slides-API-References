## ILayoutPlaceholderManager.AddContentPlaceholder metoda

Přidá nový tvar placeholderu Content do rozvržení snímku, který bude sloužit k obsahu, například obrázku, tabulky, médií nebo textu.

```csharp
public IAutoShape AddContentPlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| x | Single | Souřadnice X nového tvaru placeholderu. |
| y | Single | Souřadnice Y nového tvaru placeholderu. |
| width | Single | Šířka nového tvaru placeholderu. |
| height | Single | Výška nového tvaru placeholderu. |

### Návratová hodnota

Vytvořeno [`IAutoShape`](../../iautoshape) s placeholderem typu Content.

### Příklady

Následující příklad ukazuje, jak přidat tvar placeholderu Content do rozvržení snímku.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddContentPlaceholder(20, 20, 500, 300);
}
```

## ILayoutPlaceholderManager.AddTablePlaceholder metoda

Přidá nový placeholder tvar do snímku rozvržení, aby uchovával tabulku.

```csharp
public IAutoShape AddTablePlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| x | Single | Souřadnice X nového placeholder tvaru. |
| y | Single | Souřadnice Y nového placeholder tvaru. |
| width | Single | Šířka nového placeholder tvaru. |
| height | Single | Výška nového placeholder tvaru. |

### Návratová hodnota

Vytvořený [`IAutoShape`](../../iautoshape) s placeholderem Table.

### Příklady

Následující příklad ukazuje, jak přidat tvar placeholderu Table do snímku rozvržení.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTablePlaceholder(20, 20, 500, 200);
}
```

## Portion.PortionFormat vlastnost

Vrací objekt formátování, který obsahuje výslovně nastavené vlastnosti formátování textové části bez aplikovaného dědění. Pouze pro čtení [`IPortionFormat`](../../iportionformat).

```csharp
public IPortionFormat PortionFormat { get; }
```

### Poznámky

Objekt formátování obsahuje parametry formátování definované pouze pro aktuální část, zděděná data nejsou použita.

Pro získání účinných hodnot včetně zděděných použijte metodu [`GetEffective`](../../portionformat/geteffective).

## AudioFrame.RewindAudio vlastnost

Určuje, zda je audio po přehrání automaticky přetočeno na začátek. Čtení/zápis Boolean.

```csharp
public bool RewindAudio { get; set; }
```

### Příklady

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ISlide slide = pres.Slides[0];

    // Přidat Audio Frame
    IAudioFrame audioFrame = slide.Shapes.AddAudioFrameLinked(50, 50, 100, 100, "sampleaudio.wav");

    // Nastavit Audio, aby se přehrávalo napříč snímky
    audioFrame.PlayAcrossSlides = true;

    // Nastavit Audio tak, aby se po přehrání automaticky přetočilo na začátek
    audioFrame.RewindAudio = true;

    pres.Save("AudioFrame_out.pptx", SaveFormat.Pptx);
}
```

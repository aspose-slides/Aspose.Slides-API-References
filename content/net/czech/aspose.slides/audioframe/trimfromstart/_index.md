## AudioFrame.TrimFromStart vlastnost

Specifies the time duration to be removed from the beginning of the media during playback, in milliseconds. Číst/zapisovat Single.

```csharp
public float TrimFromStart { get; set; }
```

### Příklady

Příklad:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    IAudio audio = pres.Audios.AddAudio(File.ReadAllBytes("sampleaudio.mp3"));
    IAudioFrame audioFrame = pres.Slides[0].Shapes.AddAudioFrameEmbedded(50, 50, 100, 100, audio);

    // Nastavte čas ořezání od začátku 1,5 sekundy
```

## PortionCollection.CopyTo metoda

Zkopíruje prvky rozhraní ICollection do pole Array, počínaje konkrétním indexem pole Array.

```csharp
public void CopyTo(IPortion[] array, int arrayIndex)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| array | IPortion[] | Jednorozměrné pole Array, které je cílem prvků zkopírovaných z ICollection. Pole Array musí mít nulové indexování. |
| arrayIndex | Int32 | Nulový index v *array*, kde začíná kopírování. |

### Výjimky

| výjimka | podmínka |
| --- | --- |
| ArgumentNullException | *array* je null. |
| ArgumentOutOfRangeException | *arrayIndex* je menší než 0. |
| ArgumentException | Počet prvků ve zdrojovém ICollection je větší než dostupné místo od *arrayIndex* do konce cílového *array*. |

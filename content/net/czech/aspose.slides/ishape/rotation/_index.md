## IShape.Rotation vlastnost

Vrací nebo nastavuje počet stupňů, o které je daný tvar otočen kolem osy z. Kladná hodnota označuje otáčení po směru hodinových ručiček; záporná hodnota označuje otáčení proti směru hodinových ručiček. Čtení/zápis Single.

```csharp
public float Rotation { get; set; }
```

### Poznámky

Vrácená hodnota je vždy definována (není float.NaN). Přidělená hodnota musí být definována (není float.NaN). Můžete nastavit nedefinované hodnoty pro vlastnosti instance RawFrame.

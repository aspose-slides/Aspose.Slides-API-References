## BulletFormat.Height vlastnost

Vrací nebo nastavuje výšku odrážky odstavce bez dědičnosti. Hodnota float.NaN určuje, že odrážka dědí výšku z první části odstavce. Čtení/Zápis Single.

```csharp
public float Height { get; set; }
```

### Poznámky

Záporná hodnota výšky znamená, že výška je uvedena v bodech, a kladná hodnota znamená, že výška je procentuální podíl na okolním textu.

## ViewProperties.GridSpacing vlastnost

Vrací nebo nastavuje rozestup mřížky, který by měl být použit pro mřížku podkládající dokument prezentace, v bodech. Čtení/Zápis Single.

```csharp
public float GridSpacing { get; set; }
```

### Poznámky

Hodnota rozestupu mřížky musí být kladné číslo. Typický rozsah hodnot je od 1 mm (2.8349607 bodů) do 2 inches (144 bodů).

### Příklady

Následující ukázkový kód ukazuje, jak změnit rozestup mřížky v prezentaci PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

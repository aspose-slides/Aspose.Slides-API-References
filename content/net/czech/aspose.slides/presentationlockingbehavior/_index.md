## PresentationLockingBehavior enumerace

Represents the behavior regarding treating the [`IPresentation`](../ipresentation) source (file or Stream) while loading and working with an instance of [`IPresentation`](../ipresentation).

```csharp
public enum PresentationLockingBehavior
```

### Hodnoty

| Název | Hodnota | Popis |
| --- | --- | --- |
| LoadAndRelease | `0` | Zdroj bude zamčen pouze po dobu vykonávání konstruktoru [`IPresentation`](../ipresentation). Pokud je [`IsTemporaryFilesAllowed`](../iblobmanagementoptions/istemporaryfilesallowed) nastaveno na false, všechny BLOBy budou načteny do paměti. V opačném případě mohou být použity jiné prostředky, například dočasné soubory. Toto chování je pomalejší než KeepLocked a pokud je možné předat vlastnictví zdroje [`IPresentation`](../ipresentation), doporučuje se použít KeepLocked. |
| KeepLocked | `1` | Zdroj bude zamčen po celou dobu životnosti instance [`IPresentation`](../ipresentation), dokud nebude uvolněn. [`IsTemporaryFilesAllowed`](../iblobmanagementoptions/istemporaryfilesallowed) musí být nastaveno na true pro použití tohoto chování, jinak bude vyvolána výjimka. Toto chování je doporučeno, je rychlejší a spotřebuje méně paměti než LoadAndRelease. |

### Poznámky

Zdroj je parametr předaný konstruktoru [`IPresentation`](../ipresentation). V níže uvedeném příkladu je zdroj soubor "pres.pptx":

```csharp
LoadOptions loadOptions = new LoadOptions { 
  BlobManagementOptions = { PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked } };
using (IPresentation pres = new Presentation("pres.pptx", loadOptions)) { }
```

Pro tento příklad bude zdroj (soubor "pres.pptx") zamčen po dobu životnosti instance [`IPresentation`](../ipresentation), tj. nemůže být jiným procesem změněn ani smazán.

## PresentationLockingBehavior výčet

Representuje chování při nakládání se zdrojem [`IPresentation`](../ipresentation) (soubor nebo Stream) během načítání a práce s instancí [`IPresentation`](../ipresentation).

```csharp
public enum PresentationLockingBehavior
```

### Values

| Název | Hodnota | Popis |
| --- | --- | --- |
| LoadAndRelease | `0` | Zdroj bude uzamčen pouze během provádění konstruktoru [`IPresentation`](../ipresentation). Pokud je [`IsTemporaryFilesAllowed`](../iblobmanagementoptions/istemporaryfilesallowed) nastaveno na false, všechny BLOBy budou načteny do paměti. V opačném případě mohou být použity jiné prostředky, například dočasné soubory. Toto chování je pomalejší než KeepLocked a pokud je možné předat vlastnictví zdroje [`IPresentation`](../ipresentation), doporučuje se použít KeepLocked. |
| KeepLocked | `1` | Zdroj bude uzamčen po celou životnost instance [`IPresentation`](../ipresentation), dokud nebude uvolněn. [`IsTemporaryFilesAllowed`](../iblobmanagementoptions/istemporaryfilesallowed) musí být nastaveno na true pro použití tohoto chování, jinak bude vyvolána výjimka. Toto chování se doporučuje, je rychlejší a spotřebuje méně paměti než LoadAndRelease. |

### Poznámky

Zdroj je parametr předávaný konstruktoru [`IPresentation`](../ipresentation). V níže uvedeném příkladu je zdrojem soubor "pres.pptx":

```csharp
LoadOptions loadOptions = new LoadOptions { 
  BlobManagementOptions = { PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked } };
using (IPresentation pres = new Presentation("pres.pptx", loadOptions)) { }
```

V tomto příkladu bude zdroj (soubor "pres.pptx") uzamčen po dobu životnosti instance [`IPresentation`](../ipresentation), tj. nebude jej moci změnit nebo smazat jiný proces.

## IBlobManagementOptions.PresentationLockingBehavior vlastnost

Tato vlastnost určuje, zda může být instance třídy Presentation vlastníkem zdroje – souboru nebo proudu během životnosti instance. Pokud je instance vlastníkem, uzamkne zdroj. To pomáhá zlepšit spotřebu paměti a výkon při práci s BLOBy, ale zdroj (proud nebo soubor) nemůže být během životnosti instance Presentation změněn. Toto je příklad:

```csharp
public PresentationLockingBehavior PresentationLockingBehavior { get; set; }
```

### Příklady

```csharp
LoadOptions loadOptions = new LoadOptions { 
  BlobManagementOptions = {PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked} };
using (Presentation pres = new Presentation("pres.pptx", loadOptions)) {
  // IOException bude vyvolána, protože soubor pres.pptx je uzamčen po celou dobu životnosti Presentation
  // File.Delete("pres.pptx");
}
// po uvolnění objektu Presentation je soubor odemčen a může být smazán
File.Delete("pres.pptx");
```

## LoadOptions.DeleteEmbeddedBinaryObjects vlastnost

Určuje, zda Aspose.Slides při načítání prezentace smaže všechny vložené binární objekty.

Typy vložených binárních objektů:

* Projekt VBA [`VbaProject`](../../ipresentation/vbaproject)
* Vložená data objektu OLE [`EmbeddedFileData`](../../ioleembeddeddatainfo/embeddedfiledata)
* Binární data ActiveX ovládacího prvku [`ActiveXControlBinary`](../../icontrol/activexcontrolbinary)

Číst/zapisovat Boolean.

```csharp
public bool DeleteEmbeddedBinaryObjects { get; set; }
```

### Poznámky

Výchozí hodnota je **false**.

### Příklady

Následující příklad ukazuje, jak načíst prezentaci bez jakýchkoli vložených binárních objektů.

```csharp
[C#]
LoadOptions loadOptions = new LoadOptions();
loadOptions.DeleteEmbeddedBinaryObjects = true;
using (Presentation pres = new Presentation("pres.ppt", loadOptions))
{
    pres.Save("output_WithoutBinaryObjects.ppt", SaveFormat.Ppt);
}
```

## LoadOptions.DocumentLevelFontSources vlastnost

Určuje zdroje externích písem, které budou v prezentaci použity. Tato písma jsou v prezentaci k dispozici po celou dobu její existence a nejsou sdílena s jinými prezentacemi.

```csharp
public IFontSources DocumentLevelFontSources { get; set; }
```

### Příklady

Následující příklad ukazuje, jak zadat vlastní písma používaná v prezentaci PowerPoint.

```csharp
[C#]
byte[] memoryFont1 = File.ReadAllBytes("customfonts\\CustomFont1.ttf");
byte[] memoryFont2 = File.ReadAllBytes("customfonts\\CustomFont2.ttf");
LoadOptions loadOptions = new LoadOptions();
loadOptions.DocumentLevelFontSources.FontFolders = new string[] { "assets\\fonts", "global\\fonts" };
loadOptions.DocumentLevelFontSources.MemoryFonts = new byte[][] { memoryFont1, memoryFont2 };
using (IPresentation presentation = new Presentation("MyPresentation.pptx", loadOptions))
{
    //pracujte s prezentací
    //CustomFont1, CustomFont2 stejně jako písma z složek assets\fonts & global\fonts a jejich podadresářů jsou k dispozici prezentaci
}
```

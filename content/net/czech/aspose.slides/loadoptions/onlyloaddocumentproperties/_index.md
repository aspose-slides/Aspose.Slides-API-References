## LoadOptions.OnlyLoadDocumentProperties vlastnost

Tato vlastnost má smysl, pokud je soubor prezentace chráněn heslem. Hodnota true znamená, že z šifrovaného souboru prezentace musí být načteny pouze vlastnosti dokumentu a heslo se má ignorovat. Hodnota false znamená, že celá šifrovaná prezentace musí být načtena s použitím správného hesla. Pokud prezentace není šifrována, hodnota vlastnosti se vždy ignoruje. Pokud nejsou veřejné vlastnosti dokumentu šifrovaného souboru a hodnota vlastnosti je true, pak vlastnosti dokumentu nelze načíst a bude vyvolána výjimka. Čtení/Zápis Boolean.

public bool OnlyLoadDocumentProperties { get; set; }

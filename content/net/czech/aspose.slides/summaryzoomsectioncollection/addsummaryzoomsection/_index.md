## SummaryZoomSectionCollection.AddSummaryZoomSection method

Vytváří nový objekt Summary Zoom Section a přidá jej do kolekce

```csharp
public ISummaryZoomSection AddSummaryZoomSection(ISection section)
```

| Parameter | Type | Description |
| --- | --- | --- |
| section | ISection | Sekce pro nový prvek Summary Zoom Section [`ISection`](../../isection) |

### Return Value

Přidaný [`ISummaryZoomFrame`](../../isummaryzoomframe) prvek

### Exceptions

| exception | condition |
| --- | --- |
| ArgumentException | Odkazovaná sekce nepatří k aktuální prezentaci nebo neobsahuje žádné snímky. |

### Remarks

Pokud již v kolekci existuje prvek pro tuto sekci, je vrácen existující prvek.

### Examples

Příklad ukazuje získání prvku Summary Zoom Section podle indexu:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    ISummaryZoomFrame zoomFrame = (ISummaryZoomFrame) pres.Slides[1].Shapes[0];
    ISummaryZoomSectionCollection collection = zoomFrame.SummaryZoomCollection;
    ISummaryZoomSection newZoomSection = collection.AddSummaryZoomSection(pres.Sections[3]);
}
```

## BasePortionFormat.SpellCheck vlastnost

Získá nebo nastaví hodnotu, která určuje, zda je kontrola pravopisu povolena pro část textu. Když je tato vlastnost nastavena na false, kontrola pravopisu pro textové prvky je potlačena. Když je nastavena na true, kontrola pravopisu je povolena. Výchozí hodnota je `false`.

```csharp
public bool SpellCheck { get; set; }
```

### Příklady

Následující příklad ukazuje povolení příznaku SpellCheck před uložením prezentace:

```csharp
[C#]
using (var pres = new Presentation("input.pptx"))
{
    // Přístup k prvnímu úseku textu uvnitř prvního tvaru na prvním snímku
    var portion = ((AutoShape)pres.Slides[0].Shapes[0]).TextFrame.Paragraphs[0].Portions[0];
    // Povolení kontroly pravopisu pro tento úsek textu
    portion.PortionFormat.SpellCheck = true;
    // Uložení upravené prezentace
    pres.Save("output-with-spellcheck.pptx", SaveFormat.Pptx);
}
```

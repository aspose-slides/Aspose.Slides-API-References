## MasterLayoutSlideCollection.Insert metoda

Vloží nový snímek rozložení na určenou pozici v kolekci.

```csharp
public ILayoutSlide Insert(int index, SlideLayoutType layoutType, string layoutName)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| index | Int32 | Index nového snímku. |
| layoutType | SlideLayoutType | Typ rozložení pro nové rozložení. Podporované typy rozložení: Title, TitleOnly, Blank, TitleAndObject, VerticalText, VerticalTitleAndText, TwoObjects, SectionHeader, TwoTextAndTwoObjects, TitleObjectAndCaption, PictureAndCaption, Custom. Ostatní typy rozložení nyní nejsou podporovány: Text, TwoColumnText, Table, TextAndChart, ChartAndText, Diagram, Chart, TextAndClipArt, ClipArtAndText, TextAndObject, ObjectAndText, Object, TextAndMedia, MediaAndText, ObjectOverText, TextOverObject, TextAndTwoObjects, TwoObjectsAndText, TwoObjectsOverText, FourObjects, ClipArtAndVerticalText, VerticalTitleAndTextOverChart, ObjectAndTwoObject, TwoObjectsAndObject. |
| layoutName | String | Jméno pro nové rozložení. Pokud je předáno jméno, které již je použito, bude vyvolána výjimka ArgumentException. Pokud je předán parametr null, bude jméno vygenerováno automaticky vzhledem k předanému typu rozložení (např. "Title Slide" nebo "1_Title Slide", "2_..", atd.). |

### Návratová hodnota

Vložený snímek.

| výjimka | podmínka |

## ISummaryZoomSectionCollection.AddSummaryZoomSection metoda

Vytvoří nový objekt Summary Zoom Section a přidá jej do kolekce

```csharp
public ISummaryZoomSection AddSummaryZoomSection(ISection section)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| section | ISection | Sekce pro nový prvek Summary Zoom Section [`ISection`](../../isection) |

### Návratová hodnota

Přidán [`ISummaryZoomFrame`](../../isummaryzoomframe) prvek

### Poznámky

Pokud prvek pro tuto sekci v kolekci již existuje, vrátí se existující prvek.

### Příklady

Příklad ukazuje získání prvku Summary Zoom Section podle indexu:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    ISummaryZoomFrame zoomFrame = (ISummaryZoomFrame) pres.Slides[1].Shapes[0];
    ISummaryZoomSectionCollection collection = zoomFrame.SummaryZoomCollection;
    ISummaryZoomSection newZoomSection = collection.AddSummaryZoomSection(pres.Sections[3]);
}
```

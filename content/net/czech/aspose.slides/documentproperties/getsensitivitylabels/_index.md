## DocumentProperties.GetSensitivityLabels metoda

Získá pole značek citlivosti ze vlastních vlastností dokumentu (Microsoft Information Protection SDK Metadata).

```csharp
public ISensitivityLabel[] GetSensitivityLabels()
```

### Příklady

Následující kód ukazuje, jak přesunout informace o značkách citlivosti ze vlastních vlastností dokumentu do moderní kolekce SensitivityLabels:

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    // Získá značky citlivosti ze vlastních vlastností dokumentu
    ISensitivityLabel[] mipSensitivityLabels = pres.DocumentProperties.GetSensitivityLabels();

    ISensitivityLabelCollection sensitivityLabels = pres.SensitivityLabels;
    foreach (var sensitivityLabel in mipSensitivityLabels)
    {
        // Přidá značku do kolekce
        // Zde můžete přidat kontrolu platnosti informací o značce (značka je dostupná atd.)
        sensitivityLabels.Add(sensitivityLabel);
    }

    pres.Save("SensitivityLabel.pptx", SaveFormat.Pptx);
}
```

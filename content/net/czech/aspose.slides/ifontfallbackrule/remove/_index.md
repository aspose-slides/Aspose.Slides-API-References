## IFontFallBackRule.Remove metoda

Odstraní první výskyt konkrétního fontu FallBack ze seznamu.

```csharp
public void Remove(string fontName)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| fontName | String | Název fontu, který se má odstranit ze seznamu. |

### Příklady

```csharp
[C#]
// Vytvoří pravidlo, které obsahuje seznam fontů.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// Odstranění Tahoma ze seznamu
newRule.Remove ("Tahoma");
```

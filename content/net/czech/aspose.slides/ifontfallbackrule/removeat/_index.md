## IFontFallBackRule.RemoveAt metoda

Odstraňuje písmo FallBack na určeném indexu seznamu.

```csharp
public void RemoveAt(int index)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| index | Int32 | Index písma, který se má odstranit, počítaný od nuly. |

### Příklady

```csharp
[C#]
// Vytvořte pravidlo obsahující seznam písem.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// Odstranění Tahoma ze seznamu
newRule.Remove (2);
```

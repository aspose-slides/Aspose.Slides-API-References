## IFontFallBackRule.IndexOf metoda

Vrací index určeného pravidla ve sbírce.

```csharp
public int IndexOf(string fontName)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| fontName | String | Název písma k vyhledání. |

### Návratová hodnota

Index písma nebo -1, pokud písmo není v seznamu nalezeno.

### Příklady

```csharp
[C#]
// Vytvoří pravidlo obsahující seznam písem.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

//Get index of Tahoma
int tahomaIndex = newRule.IndexOf("Tahoma");
```

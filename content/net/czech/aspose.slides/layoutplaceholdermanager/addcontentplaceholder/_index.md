## LayoutPlaceholderManager.AddContentPlaceholder metoda

Přidá nový placeholder tvar do rozvržení snímku, který bude obsahovat obsah, například obrázek, tabulku, média nebo text.

```csharp
public IAutoShape AddContentPlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| x | Single | Souřadnice X nového placeholder tvaru. |
| y | Single | Souřadnice Y nového placeholder tvaru. |
| width | Single | Šířka nového placeholder tvaru. |
| height | Single | Výška nového placeholder tvaru. |

### Návratová hodnota

Vytvořeno [`IAutoShape`](../../iautoshape) s placeholderem Content.

### Příklady

Následující příklad ukazuje, jak přidat tvar placeholderu Content do rozvržení snímku.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddContentPlaceholder(20, 20, 500, 300);
}
```

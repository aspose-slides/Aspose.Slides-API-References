## LayoutPlaceholderManager.AddVerticalTextPlaceholder metoda

Přidá nový tvar zástupce do snímku rozvržení, aby uchovával textový obsah ve svislém směru.

```csharp
public IAutoShape AddVerticalTextPlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| x | Single | Souřadnice X nového tvaru zástupce. |
| y | Single | Souřadnice Y nového tvaru zástupce. |
| width | Single | Šířka nového tvaru zástupce. |
| height | Single | Výška nového tvaru zástupce. |

### Návratová hodnota

Vytvořen [`IAutoShape`](../../iautoshape) s zástupcem Text (Vertical).

### Příklady

Následující příklad ukazuje, jak přidat tvar zástupce Text (Vertical) do snímku rozvržení.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalTextPlaceholder(20, 20, 300, 500);
}
```

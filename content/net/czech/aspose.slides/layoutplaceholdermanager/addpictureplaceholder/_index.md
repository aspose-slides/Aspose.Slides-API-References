## LayoutPlaceholderManager.AddPicturePlaceholder method

Přidá nový placeholder shape do rozložení snímku, který bude obsahovat obrázek.

```csharp
public IAutoShape AddPicturePlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| x | Single | Souřadnice X nového placeholder tvaru. |
| y | Single | Souřadnice Y nového placeholder tvaru. |
| width | Single | Šířka nového placeholder tvaru. |
| height | Single | Výška nového placeholder tvaru. |

### Návratová hodnota

Vytvořeno [`IAutoShape`](../../iautoshape) s placeholderem obrázku.

### Příklady

Následující příklad ukazuje, jak přidat placeholder tvaru obrázku do rozložení snímku.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddPicturePlaceholder(20, 20, 200, 200);
}
```

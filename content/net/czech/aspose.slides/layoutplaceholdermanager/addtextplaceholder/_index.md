## LayoutPlaceholderManager.AddTextPlaceholder metoda

Přidá nový tvar zástupného symbolu do rozvržení snímku, aby obsahoval text.

```csharp
public IAutoShape AddTextPlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| x | Single | X-souřadnice nového tvaru zástupného symbolu. |
| y | Single | Y-souřadnice nového tvaru zástupného symbolu. |
| width | Single | Šířka nového tvaru zástupného symbolu. |
| height | Single | Výška nového tvaru zástupného symbolu. |

### Návratová hodnota

Vytvořeno [`IAutoShape`](../../iautoshape) s textovým zástupným symbolem.

### Příklady

Následující příklad ukazuje, jak přidat tvar textového zástupného symbolu do rozvržení snímku.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTextPlaceholder(20, 20, 500, 300);
}
```

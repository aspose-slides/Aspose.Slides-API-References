## LayoutPlaceholderManager.AddOnlineImagePlaceholder metoda

Přidá nový tvar placeholderu do snímku rozložení, který bude držet online obrázek.

```csharp
public IAutoShape AddOnlineImagePlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| x | Single | Souřadnice X nového tvaru placeholderu. |
| y | Single | Souřadnice Y nového tvaru placeholderu. |
| width | Single | Šířka nového tvaru placeholderu. |
| height | Single | Výška nového tvaru placeholderu. |

### Návratová hodnota

Vytvořen [`IAutoShape`](../../iautoshape) s placeholderem online obrázku.

### Příklady

Následující příklad ukazuje, jak přidat tvar placeholderu Online Image do rozložení snímku.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddOnlineImagePlaceholder(20, 20, 200, 200);
}
```

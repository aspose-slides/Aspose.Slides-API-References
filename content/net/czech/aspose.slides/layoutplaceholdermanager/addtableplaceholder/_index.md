## LayoutPlaceholderManager.AddTablePlaceholder metoda

Přidá nový tvar placeholder do snímku rozvržení, aby obsahoval tabulku.

```csharp
public IAutoShape AddTablePlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| x | Single | X-souřadnice nového tvaru placeholder. |
| y | Single | Y-souřadnice nového tvaru placeholder. |
| width | Single | Šířka nového tvaru placeholder. |
| height | Single | Výška nového tvaru placeholder. |

### Návratová hodnota

Vytvořeno [`IAutoShape`](../../iautoshape) s Table placeholder.

### Příklady

Následující příklad ukazuje, jak přidat tvar Table placeholder do snímku rozvržení.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTablePlaceholder(20, 20, 500, 200);
}
```

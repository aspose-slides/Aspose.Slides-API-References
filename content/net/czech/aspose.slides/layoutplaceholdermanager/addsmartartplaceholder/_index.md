## LayoutPlaceholderManager.AddSmartArtPlaceholder metoda

Přidá nový placeholder tvar do rozložení snímku, který bude obsahovat diagram SmartArt.

```csharp
public IAutoShape AddSmartArtPlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| x | Single | Souřadnice X nového placeholder tvaru. |
| y | Single | Souřadnice Y nového placeholder tvaru. |
| width | Single | Šířka nového placeholder tvaru. |
| height | Single | Výška nového placeholder tvaru. |

### Návratová hodnota

Vytvořeno [`IAutoShape`](../../iautoshape) s SmartArt placeholderem.

### Příklady

Následující příklad ukazuje, jak přidat SmartArt placeholder tvar do rozložení snímku.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddSmartArtPlaceholder(20, 20, 200, 200);
}
```

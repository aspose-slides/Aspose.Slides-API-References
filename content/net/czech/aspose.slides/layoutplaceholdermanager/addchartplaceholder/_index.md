## LayoutPlaceholderManager.AddChartPlaceholder metoda

Přidá nový placeholder tvar na rozložení snímku pro umístění grafu.

```csharp
public IAutoShape AddChartPlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| x | Single | Souřadnice X nového placeholder tvaru. |
| y | Single | Souřadnice Y nového placeholder tvaru. |
| width | Single | Šířka nového placeholder tvaru. |
| height | Single | Výška nového placeholder tvaru. |

### Návratová hodnota

Vytvořeno [`IAutoShape`](../../iautoshape) s placeholderem Chart.

### Příklady

Následující příklad ukazuje, jak přidat placeholder tvar Chart do rozložení snímku.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddChartPlaceholder(20, 20, 200, 200);
}
```

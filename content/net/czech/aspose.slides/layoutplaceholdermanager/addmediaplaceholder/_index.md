## LayoutPlaceholderManager.AddMediaPlaceholder metoda

Přidá nový tvar zástupce do snímku rozvržení, který bude držet mediální objekt.

```csharp
public IAutoShape AddMediaPlaceholder(float x, float y, float width, float height)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| x | Single | X-souřadnice nového tvaru zástupce. |
| y | Single | Y-souřadnice nového tvaru zástupce. |
| width | Single | Šířka nového tvaru zástupce. |
| height | Single | Výška nového tvaru zástupce. |

### Návratová hodnota

Vytvořeno [`IAutoShape`](../../iautoshape) s mediálním zástupcem.

### Příklady

Následující příklad ukazuje, jak přidat tvar mediálního zástupce do snímku rozvržení.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddMediaPlaceholder(20, 20, 200, 200);
}
```

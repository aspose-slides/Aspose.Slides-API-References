## IShapeCollection.InsertSummaryZoomFrame metoda

Vytvoří nový rámec Summary Zoom a vloží jej do kolekce tvarů na zadaném indexu.

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| index | Int32 | Nulově založený index, na který se vloží rámec Summary Zoom. |
| x | Single | Souřadnice x nového rámce Summary Zoom v bodech. |
| y | Single | Souřadnice y nového rámce Summary Zoom v bodech. |
| width | Single | Šířka nového rámce Summary Zoom v bodech. |
| height | Single | Výška nového rámce Summary Zoom v bodech. |

### Návratová hodnota

Nově vytvořený [`ISummaryZoomFrame`](../../isummaryzoomframe).

### Výjimky

| výjimka | podmínka |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | Vyvolána, pokud prezentace neobsahuje žádné sekce, nebo pokud cílový snímek nepatří do žádné sekce. |

### Poznámky

Tato metoda vytvoří rámec Summary Zoom, který shromažďuje souhrnné odkazy pro všechny sekce v prezentaci.

### Příklady

Tento příklad ukazuje vytvoření a vložení objektu Summary Zoom na určený index kolekce (předpokládejte, že v prezentaci "Presentation.pptx" jsou alespoň dvě sekce):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSummaryZoomFrame(2, 150, 20, 50, 50);
}
```

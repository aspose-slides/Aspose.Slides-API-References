## AddZoomFrame(float, float, float, float, ISlide) {#addzoomframe}

Vytvoří nový Zoom rámec a přidá jej na konec kolekce tvarů.

```csharp
public IZoomFrame AddZoomFrame(float x, float y, float width, float height, ISlide slide)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| x | Single | X-souřadnice nového Zoom rámce v bodech. |
| y | Single | Y-souřadnice nového Zoom rámce v bodech. |
| width | Single | Šířka nového Zoom rámce v bodech. |
| height | Single | Výška nového Zoom rámce v bodech. |
| slide | ISlide | Slide [`ISlide`](../../islide) odkazovaný Zoom rámcem; musí patřit k této prezentaci. |

### Návratová hodnota

Nově vytvořený [`IZoomFrame`](../../izoomframe).

### Výjimky

| Výjimka | Podmínka |
| --- | --- |
| ArgumentException | Vyvolána, pokud odkazovaný slide nepatří do aktuální prezentace. |

### Příklady

Tento příklad ukazuje přidání objektu Zoom na konec kolekce (předpokládejme, že v prezentaci "Presentation.pptx" jsou alespoň dva slidy):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
}
```

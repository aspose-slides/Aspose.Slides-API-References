## IShapeCollection.AddSummaryZoomFrame metoda

Vytvoří nový rámec Summary Zoom a přidá jej na konec kolekce tvarů.

```csharp
public ISummaryZoomFrame AddSummaryZoomFrame(float x, float y, float width, float height)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| x | Single | Souřadnice x nového rámce Summary Zoom v bodech. |
| y | Single | Souřadnice y nového rámce Summary Zoom v bodech. |
| width | Single | Šířka nového rámce Summary Zoom v bodech. |
| height | Single | Výška nového rámce Summary Zoom v bodech. |

### Návratová hodnota

Nově vytvořený [`ISummaryZoomFrame`](../../isummaryzoomframe).

### Výjimky

| výjimka | podmínka |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | Vyvolána, pokud v prezentaci nejsou žádné sekce nebo pokud cílový snímek nepatří do žádné sekce. |

### Poznámky

Tato metoda vytvoří rámec Summary Zoom, který agreguje odkazy na souhrny pro všechny sekce v prezentaci.

### Příklady

Tento příklad ukazuje, jak přidat objekt Summary Zoom na konec kolekce (předpokládejte, že v prezentaci „Presentation.pptx“ jsou alespoň dvě sekce):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSummaryZoomFrame(150, 20, 500, 250);
}
```

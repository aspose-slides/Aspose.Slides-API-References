## IPresentationInfo.CheckWriteProtection metoda

Kontroluje, zda je heslo pro úpravy správné u chráněné prezentace proti zápisu.

```csharp
public bool CheckWriteProtection(string password)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| password | String | Heslo, které se má zkontrolovat. |

### Návratová hodnota

True, pokud je prezentace chráněna proti zápisu a heslo je správné. False v opačném případě.

### Výjimky

| výjimka | podmínka |
| --- | --- |
| InvalidOperationException |  |

### Poznámky

1. Před voláním této metody byste měli zkontrolovat vlastnost [`IsWriteProtected`](../iswriteprotected). 2. Když je password null nebo prázdný, tato metoda vrací false.

### Příklady

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
if (info.IsWriteProtected == NullableBool.True)
{
    bool isWriteProtectedByPassword = info.CheckWriteProtection("my_password");
}
```

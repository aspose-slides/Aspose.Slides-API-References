## TextFrame.SplitTextByColumns metoda

Rozděluje textový obsah [`ITextFrame`](../../itextframe) do pole řetězců, kde každý prvek odpovídá samostatnému sloupci textu v rámci rámečku.

```csharp
public string[] SplitTextByColumns()
```

### Návratová hodnota

Pole řetězců, kde každý řetězec představuje textový obsah konkrétního sloupce v [`ITextFrame`](../../itextframe).

### Poznámky

Pokud textový rámeček neobsahuje více sloupců, vrácené pole bude mít jediný prvek obsahující celý text. Prázdné sloupce budou v poli reprezentovány prázdnými řetězci.

### Příklady

Následující příklad ukazuje, jak použít `SplitTextByColumns`:

```csharp
using (Presentation pres = new Presentation("example.pptx"))
{
    // Získá první tvar na snímku a přetypeuje jej na ITextFrame
    ITextFrame textFrame = pres.Slides[0].Shapes[0] as ITextFrame;
    // Rozdělí obsah textového rámce do sloupců
    string[] columnsText = textFrame.SplitTextByColumns();
    // Vytiskne text každého sloupce na konzoli
    foreach (string column in columnsText)
        Console.WriteLine(column);
}
```

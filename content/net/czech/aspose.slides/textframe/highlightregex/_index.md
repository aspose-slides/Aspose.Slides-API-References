## TextFrame.HighlightRegex metoda

Zvýrazní všechny shody regulárního výrazu zadanou barvou.

```csharp
public void HighlightRegex(Regex regex, Color highlightColor, IFindResultCallback callback)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| regex | Regex | Regulární výraz Regex, který se používá k získání řetězců ke zvýraznění. |
| highlightColor | Color | Barva použité k zvýraznění textu. |
| callback | IFindResultCallback | Objekt zpětného volání pro přijímání výsledků vyhledávání [`IFindResultCallback`](../../ifindresultcallback). |

### Příklady

Následující ukázka kódu ukazuje, jak zvýraznit text v TextFrame pomocí regulárního výrazu.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// zvýraznění všech slov s 10 a více znaky
	((AutoShape)presentation.Slides[0].Shapes[0]).TextFrame.HighlightRegex(regex, Color.Blue, null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

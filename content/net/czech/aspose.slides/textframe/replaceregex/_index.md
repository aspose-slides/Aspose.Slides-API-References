## TextFrame.ReplaceRegex metoda

Nahrazuje všechny shody regulárního výrazu zadaným řetězcem.

```csharp
public void ReplaceRegex(Regex regex, string newText, IFindResultCallback callback)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| regex | Regex | Regulární výraz Regex, který získá řetězce k nahrazení. |
| newText | String | Řetězec, který nahradí všechny výskyty řetězců k nahrazení. |
| callback | IFindResultCallback | Objekt zpětného volání pro uložení výsledku operace nahrazení [`IFindResultCallback`](../../ifindresultcallback). |

### Příklady

Následující ukázkový kód ukazuje, jak nahradit text pomocí regulárního výrazu zadaným řetězcem.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx")){
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// Nahradí všechna slova s 10 nebo více znaky řetězcem '***'
	((AutoShape)presentation.Slides[0].Shapes[0]).TextFrame.ReplaceRegex(regex, "***", null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

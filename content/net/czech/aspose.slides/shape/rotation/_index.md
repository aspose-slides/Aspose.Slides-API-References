## Shape.Rotation vlastnost

Vrací nebo nastavuje počet stupňů, o které je daný tvar otočen kolem osy z. Kladná hodnota značí otáčení po směru hodinových ručiček; záporná hodnota značí otáčení proti směru hodinových ručiček. Čtení/Zápis Single.

```csharp
public float Rotation { get; set; }
```

### Poznámky

Vrácená hodnota je vždy definována (není float.NaN). Přiřazená hodnota musí být definována (ne float.NaN). Můžete nastavit nedefinované hodnoty pro vlastnosti instance RawFrame.

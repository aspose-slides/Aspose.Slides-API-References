## Shape.RawFrame vlastnost

Vrací nebo nastavuje vlastnosti surového rámečku tvaru. Čtení/zápis [`IShapeFrame`](../../ishapeframe).

```csharp
public IShapeFrame RawFrame { get; set; }
```

### Příklady

Kód, který se pokouší přiřadit nedefinovaný rámeček k IShape.Frame, nedává smysl v obecném případě (zejména v situaci, kdy je nadřazený GroupShape několikrát vnořen do dalších GroupShape). Například:

```csharp
IShape shape = ...;
shape.Frame = new ShapeFrame(float.NaN, float.NaN, float.NaN, float.NaN, NullableBool.NotDefined, NullableBool.NotDefined, float.NaN);
```

nebo

```csharp
slide.Shapes.AddAutoShape(ShapeType.RoundCornerRectangle, float.NaN, float.NaN, float.NaN, float.NaN);
```

Takový kód může vést k nejasným situacím. Proto byla přidána omezení pro použití nedefinovaných hodnot u IShape.Frame. Hodnoty x, y, šířka, výška, flipH, flipV a rotationAngle musí být definovány (ne float.NaN ani NullableBool.NotDefined). Výše uvedený ukázkový kód nyní vyvolá výjimku ArgumentException. Toto se vztahuje na následující případy použití:

```csharp
IShape shape = ...;
shape.Frame = ...; // nemůže být nedefinováno

IShapeCollection shapes = ...;
// x, y, width, height parametry nemohou být float.NaN:
{
    shapes.AddAudioFrameCD(...);
    shapes.AddAudioFrameEmbedded(...);
    shapes.AddAudioFrameLinked(...);
    shapes.AddAutoShape(...);
    shapes.AddChart(...);
    shapes.AddConnector(...);
    shapes.AddOleObjectFrame(...);
    shapes.AddPictureFrame(...);
    shapes.AddSmartArt(...);
    shapes.AddTable(...);
    shapes.AddVideoFrame(...);
    shapes.InsertAudioFrameEmbedded(...);
    shapes.InsertAudioFrameLinked(...);
    shapes.InsertAutoShape(...);
    shapes.InsertChart(...);
    shapes.InsertConnector(...);
    shapes.InsertOleObjectFrame(...);
    shapes.InsertPictureFrame(...);
    shapes.InsertTable(...);
    shapes.InsertVideoFrame(...);
}
```

Ale IShape.RawFrame vlastnosti rámce mohou být nedefinovány. To dává smysl, když je tvar propojen s placeholderem. Pak jsou nedefinované hodnoty rámečku tvaru přepsány z nadřazeného placeholderu. Pokud pro tento tvar neexistuje nadřazený placeholder, tvar použije výchozí hodnoty při výpočtu efektivního rámečku na základě svého IShape.RawFrame. Výchozí hodnoty jsou 0 a NullableBool.False pro x, y, šířku, výšku, flipH, flipV a rotationAngle. Například:

```csharp
IShape shape = ...; // tvar je propojen s placeholderem
shape.RawFrame = new ShapeFrame(float.NaN, float.NaN, 100, float.NaN, NullableBool.NotDefined, NullableBool.NotDefined, 0); // nyní tvar dědí hodnoty x, y, výšky, flipH, flipV z placeholderu a přepisuje šířku=100 a rotationAngle=0.
```

## Fonts.SetScriptFont metoda

Přiřadí název písma ke konkrétnímu značkovému skriptu, což určuje, jak bude text tohoto skriptu vykreslen v prezentaci.

```csharp
public void SetScriptFont(string script, string fontName)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| script | String | Kód skriptu BCP-47 (např. "Arab", "Hebr", "Hans") určující psaný systém. |
| fontName | String | Název písma, který se přiřadí ke zvolenému skriptu. |

### Příklady

Tento příklad ukazuje, jak nastavit písmo pro arabské písmo na "Segoe UI":

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segoe UI");
```

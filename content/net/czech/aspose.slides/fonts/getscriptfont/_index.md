## Fonts.GetScriptFont metoda

Získá název písma spojený s konkrétní značkou skriptu z motivu prezentace.

```csharp
public string GetScriptFont(string script)
```

| Parameter | Type | Description |
| --- | --- | --- |
| script | String | Kód skriptu BCP-47 (např. "Latn", "Cyrl", "Jpan") používaný k identifikaci psacího systému. |

### Návratová hodnota

Název písma použitého pro zadaný skript, nebo `null`, pokud skript není definován.

### Příklady

Tento příklad ukazuje, jak získat písmo přiřazené k cyrilickému skriptu v motivu prezentace.

```csharp
[C#]
string font = presentation.MasterTheme.FontScheme.Major.GetScriptFont("Cyrl");
Console.WriteLine("Font for Cyrillic script: " + font);
```

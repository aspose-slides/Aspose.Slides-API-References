## IVideoFrame.CaptionTracks vlastnost

Získá kolekci uzavřených titulků spojených s audio rámcem. Tato vlastnost je jen pro čtení a vrací [`ICaptionsCollection`](../../icaptionscollection) obsahující všechny stopy titulků.

```csharp
public ICaptionsCollection CaptionTracks { get; }
```

### Příklady

Příklad:

```csharp
[C#]
using (Presentation pres = new Presentation("video with captions.pptx"))
{
    foreach (IShape shape in pres.Slides[0].Shapes)
    {
        if (!(shape is IVideoFrame videoFrame)) continue;

        foreach (var captionTrack in videoFrame.CaptionTracks)
        {
            // Extrahuje binární data titulků a uloží je do souboru
            System.IO.File.WriteAllBytes(captionTrack.CaptionId + ".vtt", captionTrack.BinaryData);
        }
    }
}
```

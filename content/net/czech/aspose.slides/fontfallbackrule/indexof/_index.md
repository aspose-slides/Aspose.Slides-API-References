## FontFallBackRule.IndexOf metoda

Vrací index zadaného pravidla ve sbírce.

```csharp
public int IndexOf(string fontName)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| fontName | String | Název písma, které se má najít. |

### Návratová hodnota

Index písma nebo -1, pokud písmo v seznamu není nalezeno.

### Příklady

```csharp
[C#]
// Vytvoří pravidlo obsahující seznam písem.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// Získá index Tahoma.
int tahomaIndex = newRule.IndexOf("Tahoma");
```

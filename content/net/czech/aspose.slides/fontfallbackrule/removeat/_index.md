## FontFallBackRule.RemoveAt metoda

Odstraní font FallBack na zadaném indexu v seznamu.

```csharp
public void RemoveAt(int index)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| index | Int32 | Nulově-založený index fontu, který se má odstranit. |

### Příklady

```csharp
[C#]
// Vytvoří pravidlo obsahující seznam fontů.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// Odstraňuje Tahoma ze seznamu.
newRule.Remove (2);
```

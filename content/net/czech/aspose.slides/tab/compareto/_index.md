## Tab.CompareTo metoda

Porovnává aktuální instanci s jiným objektem stejného typu.

```csharp
public int CompareTo(object obj)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| obj | Object | Objekt, který se má porovnat s touto instancí. |

### Návratová hodnota

32bitové celé číslo, které udává relativní pořadí porovnávaných hodnot. Návratová hodnota má tyto významy:  &lt; 0 - Tato instance je menší než obj. = 0 - Tato instance je rovna obj. &gt; 0 - Tato instance je větší než obj.

### Příklady

Následující ukázkový kód demonstruje, jak převést první snímek z prezentace PowerPoint do metafily.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.emf"))
    {
        // Uloží první snímek jako metafilu
        pres.Slides[0].WriteAsEmf(fileStream);
    }
}
```

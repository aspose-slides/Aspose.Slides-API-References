## InsertSectionZoomFrame(int, float, float, float, float, ISection) {#insertsectionzoomframe}

Vytvoří nový rámeček Section Zoom a vloží jej do kolekce tvarů na určeném indexu.

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| index | Int32 | Nulovým indexem, na který se má vložit rámeček Section Zoom. |
| x | Single | Souřadnice x nového rámečku Section Zoom v bodech. |
| y | Single | Souřadnice y nového rámečku Section Zoom v bodech. |
| width | Single | Šířka nového rámečku Section Zoom v bodech. |
| height | Single | Výška nového rámečku Section Zoom v bodech. |
| section | ISection | [`ISection`](../../isection) odkazovaný rámečkem Section Zoom; musí patřit této prezentaci a obsahovat alespoň jeden snímek. |

### Návratová hodnota

Nově vytvořený [`ISectionZoomFrame`](../../isectionzoomframe).

### Výjimky

| Výjimka | Podmínka |
| --- | --- |
| ArgumentException | Vyvolána, pokud odkazovaná sekce nepatří do aktuální prezentace nebo neobsahuje žádné snímky. |

### Příklady

Tento příklad ukazuje vytvoření a vložení objektu Section Zoom na určený index v kolekci (předpokládejme, že v prezentaci „Presentation.pptx“ existují alespoň dvě sekce):

```csharp
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSectionZoomFrame(2, 150, 20, 50, 50, pres.Sections[1]);
}
```

## InsertSectionZoomFrame(int, float, float, float, float, ISection, IPPImage) {#insertsectionzoomframe_1}

Vytvoří nový rámeček Section Zoom s předdefinovaným obrázkem a vloží jej do kolekce tvarů na určeném indexu.

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section, IPPImage image)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| index | Int32 | Nulovým indexem, na který se má vložit rámeček Section Zoom. |
| x | Single | Souřadnice x nového rámečku Section Zoom v bodech. |
| y | Single | Souřadnice y nového rámečku Section Zoom v bodech. |
| width | Single | Šířka nového rámečku Section Zoom v bodech. |
| height | Single | Výška nového rámečku Section Zoom v bodech. |
| section | ISection | [`ISection`](../../isection) odkazovaný rámečkem Section Zoom; musí patřit této prezentaci a obsahovat alespoň jeden snímek. |
| image | IPPImage | Obrázek, který se má zobrazit v rámečku Section Zoom. |

### Návratová hodnota

Nově vytvořený [`ISectionZoomFrame`](../../isectionzoomframe).

### Výjimky

| Výjimka | Podmínka |
| --- | --- |
| ArgumentException | Vyvolána, pokud odkazovaná sekce nepatří do aktuální prezentace nebo neobsahuje žádné snímky. |

### Příklady

Tento příklad ukazuje vytvoření a vložení objektu Section Zoom na určený index v kolekci (předpokládejme, že v prezentaci „Presentation.pptx“ existují alespoň dvě sekce):

```csharp
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IPPImage image = pres.Images.AddImage(Image.FromFile("image.png"));
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSectionZoomFrame(2, 150, 20, 50, 50, pres.Sections[1], image);
}
```

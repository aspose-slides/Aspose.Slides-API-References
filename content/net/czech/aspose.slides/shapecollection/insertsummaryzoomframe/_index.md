## ShapeCollection.InsertSummaryZoomFrame metoda

Vytvoří nový rámec Summary Zoom a vloží jej do kolekce tvarů na zadaném indexu.

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| index | Int32 | Nulový index, na který se má vložit rámec Summary Zoom. |
| x | Single | X-souřadnice nového rámce Summary Zoom, v bodech. |
| y | Single | Y-souřadnice nového rámce Summary Zoom, v bodech. |
| width | Single | Šířka nového rámce Summary Zoom, v bodech. |
| height | Single | Výška nového rámce Summary Zoom, v bodech. |

### Návratová hodnota

Nově vytvořený [`ISummaryZoomFrame`](../../isummaryzoomframe).

### Výjimky

| výjimka | podmínka |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | Vyvolána, pokud prezentace neobsahuje žádné sekce, nebo pokud cílový snímek nepatří do žádné sekce. |

### Poznámky

Tato metoda vytváří rámec Summary Zoom, který agreguje odkazy na souhrny pro všechny sekce v prezentaci.

### Příklady

Tento příklad ukazuje vytvoření a vložení objektu Summary Zoom na zadaný index kolekce (předpokládejte, že v prezentaci "Presentation.pptx" jsou alespoň dvě sekce):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSummaryZoomFrame(2, 150, 20, 50, 50);
}
```

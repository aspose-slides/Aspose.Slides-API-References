## AddAudioFrameEmbedded(float, float, float, float, Stream) {#addaudioframeembedded_1}

Vytvoří nový zvukový rámec s vloženým souborem WAV a přidá jej na konec kolekce tvarů. Vložený zvuk je přidán do kolekce Presentation.Audios.

```csharp
public IAudioFrame AddAudioFrameEmbedded(float x, float y, float width, float height, 
    Stream audio_stream)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| x | Single | X-souřadnice nového zvukového rámce v bodech. |
| y | Single | Y-souřadnice nového zvukového rámce v bodech. |
| width | Single | Šířka nového zvukového rámce v bodech. |
| height | Single | Výška nového zvukového rámce v bodech. |
| audio_stream | Stream | Vstupní proud obsahující data WAV audio k vložení. |

### Návratová hodnota

Nově vytvořený [`IAudioFrame`](../../iaudioframe).

### Příklady

Následující příklady ukazují, jak vytvořit Audio Frame.

```csharp
[C#]
// Vytvoří instanci třídy Presentation, která reprezentuje soubor prezentace
using (Presentation pres = new Presentation())
{
    // Získá první snímek
    ISlide sld = pres.Slides[0];
    // Načte soubor wav zvuku do proudu
    FileStream fstr = new FileStream("sampleaudio.wav", FileMode.Open, FileAccess.Read);
    // Přidá zvukový rámec
    IAudioFrame audioFrame = sld.Shapes.AddAudioFrameEmbedded(50, 150, 100, 100, fstr);
    // Nastaví režim přehrávání a hlasitost zvuku
    audioFrame.PlayMode = AudioPlayModePreset.Auto;
    audioFrame.Volume = AudioVolumeMode.Loud;
    // Zapíše soubor PowerPoint na disk
    pres.Save("AudioFrameEmbed_out.pptx", SaveFormat.Pptx);
}
```

## AddSectionZoomFrame(float, float, float, float, ISection) {#addsectionzoomframe}

Vytvoří nový rámec Section Zoom a přidá jej na konec kolekce tvarů.

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| x | Single | Souřadnice x nového rámce Section Zoom, v bodech. |
| y | Single | Souřadnice y nového rámce Section Zoom, v bodech. |
| width | Single | Šířka nového rámce Section Zoom, v bodech. |
| height | Single | Výška nového rámce Section Zoom, v bodech. |
| section | ISection | [`ISection`](../../isection) odkazovaný rámcem Section Zoom; musí patřit k této prezentaci a obsahovat alespoň jeden snímek. |

### Return Value

Nově vytvořený [`ISectionZoomFrame`](../../isectionzoomframe).

### Exceptions

| exception | condition |
| --- | --- |
| ArgumentException | Vyvoláno, pokud odkazovaná sekce nepatří k aktuální prezentaci nebo neobsahuje žádné snímky. |

### Examples

Tento příklad ukazuje přidání objektu Section Zoom na konec kolekce (předpokládejme, že v prezentaci "Presentation.pptx" jsou alespoň dvě sekce):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1]);
}
```

## AddSectionZoomFrame(float, float, float, float, ISection, IPPImage) {#addsectionzoomframe_1}

Vytvoří nový rámec Section Zoom s předdefinovaným obrázkem a přidá jej na konec kolekce tvarů.

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section, IPPImage image)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| x | Single | Souřadnice x nového rámce Section Zoom, v bodech. |
| y | Single | Souřadnice y nového rámce Section Zoom, v bodech. |
| width | Single | Šířka nového rámce Section Zoom, v bodech. |
| height | Single | Výška nového rámce Section Zoom, v bodech. |
| section | ISection | [`ISection`](../../isection) odkazovaný rámcem Section Zoom; musí patřit k této prezentaci a obsahovat alespoň jeden snímek. |
| image | IPPImage | [`IPPImage`](../../ippimage) zobrazovaný v rámci Section Zoom. |

### Return Value

Nově vytvořený [`ISectionZoomFrame`](../../isectionzoomframe).

### Exceptions

| exception | condition |
| --- | --- |
| ArgumentException | Vyvoláno, pokud odkazovaná sekce nepatří k aktuální prezentaci nebo neobsahuje žádné snímky. |

### Examples

Tento příklad ukazuje přidání objektu Section Zoom na konec kolekce (předpokládejme, že v prezentaci "Presentation.pptx" jsou alespoň dvě sekce):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IPPImage image = pres.Images.AddImage(Image.FromFile("image.png"));
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1], image);
}
```

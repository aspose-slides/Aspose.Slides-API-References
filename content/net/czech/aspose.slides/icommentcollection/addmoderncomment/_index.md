## ICommentCollection.AddModernComment metoda

Přidá nový moderní komentář na konec kolekce.

```csharp
public IModernComment AddModernComment(string text, ISlide slide, IShape shape, PointF position, 
    DateTime creationTime)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| text | String | Prostý text nového moderního komentáře. |
| slide | ISlide | Snímek v prezentaci, do kterého se má přidat nový moderní komentář. |
| shape | IShape | Tvar na snímku, ke kterému je nový moderní komentář přiřazen. |
| position | PointF | Pozice na snímku, kde se má přidat nový moderní komentář. |
| creationTime | DateTime | Čas vytvoření moderního komentáře. |

### Návratová hodnota

Přidaný moderní komentář.

### Příklady

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ICommentAuthor newAuthor = pres.CommentAuthors.AddAuthor("Some Author", "SA");
    newAuthor.Comments.AddModernComment("This is modern comment", pres.Slides[0], null, new PointF(100, 100), DateTime.Now);

    pres.Save(outPptxFileName, SaveFormat.Pptx);
}
```

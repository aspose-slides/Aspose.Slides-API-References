## IFontFallBackRulesCollection.Remove metoda

Odstraní první výskyt konkrétního pravidla FallBack ze sbírky.

```csharp
public void Remove(IFontFallBackRule targetRule)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| targetRule | IFontFallBackRule | Pravidlo, které má být odstraněno ze sbírky. |

### Příklady

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    //Získání prázdné nebo předinicializované kolekce pravidel z FontsManageru
    IFontFallBackRulesCollection rulesList = pres.FontsManager.FontFallBackRulesCollection;

    //Přidání několika pravidel do kolekce
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));
    rulesList.Add(new FontFallBackRule(0x3040, 0x309F, "MS Mincho"));

    //Získání objektu prvního pravidla v kolekci
    IFontFallBackRule firstRule = rulesList[0];
    //Odstranění 
    rulesList.Remove (firstRule);
}
```

## IOleObjectFrame.SetEmbeddedData metoda

Nastavuje informace o OLE vložených datech.

```csharp
public void SetEmbeddedData(IOleEmbeddedDataInfo embeddedData)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| embeddedData | IOleEmbeddedDataInfo | Vložená data [`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo) |

### Výjimky

| výjimka | podmínka |
| --- | --- |
| ArgumentNullException | Když je parametr embeddedData null. |

### Poznámky

Tato metoda mění vlastnosti objektu tak, aby odrážely nová data, a nastaví příznak IsObjectLink na false, což naznačuje, že OLE objekt je vložen.

### Příklady

Následující příklad ukazuje, jak změnit OLE vložená data a jejich typ pro existující objekt [`IOleObjectFrame`](../../ioleobjectframe)

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    OleObjectFrame oof = pres.Slides[0].Shapes[0] as OleObjectFrame;
    if (oof != null)
    {
        IOleEmbeddedDataInfo newData = new OleEmbeddedDataInfo(File.ReadAllBytes("Picture.png"), "png");
        oof.SetEmbeddedData(newData);
    }
}
```

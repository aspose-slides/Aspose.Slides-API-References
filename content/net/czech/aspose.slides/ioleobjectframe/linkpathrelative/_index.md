## IOleObjectFrame.LinkPathRelative vlastnost

Vrací relativní cestu k propojenému souboru, pokud je přítomen, jinak vrací prázdný řetězec. Readonly String.

```csharp
public string LinkPathRelative { get; }
```

### Poznámky

V prezentacích Ppt mohou mít některé odkazy Ole objektů relativní reprezentaci.

### Příklady

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.ppt"))
{
    IOleObjectFrame oleFrame = presentation.Slides[0].Shapes[0] as IOleObjectFrame;

    if (oleFrame != null)
    {
        Console.WriteLine("The relative path: " + oleFrame.LinkPathRelative);
    } 
}
```

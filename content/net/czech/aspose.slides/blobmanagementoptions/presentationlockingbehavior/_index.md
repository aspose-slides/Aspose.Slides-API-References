## BlobManagementOptions.PresentationLockingBehavior vlastnost

Tato vlastnost určuje, zda může být instance třídy Presentation vlastníkem zdroje – souboru nebo proudu během životnosti instance. Pokud je instance vlastníkem, zamkne zdroj. To pomáhá zlepšit spotřebu paměti a výkon při práci s BLOBy, ale zdroj (proud nebo soubor) nemůže být během životnosti instance Presentation změněn.

```csharp
public PresentationLockingBehavior PresentationLockingBehavior { get; set; }
```

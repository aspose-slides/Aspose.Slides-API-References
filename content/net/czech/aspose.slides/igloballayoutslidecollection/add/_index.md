## IGlobalLayoutSlideCollection.Add metoda

Přidá nový snímek rozložení do prezentace.

```csharp
public ILayoutSlide Add(IMasterSlide master, SlideLayoutType layoutType, string layoutName)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| master | IMasterSlide | Mistrový snímek pro nové rozložení. |
| layoutType | SlideLayoutType | Typ rozložení pro nové rozložení. Podporované typy rozložení: Title, TitleOnly, Blank, TitleAndObject, VerticalText, VerticalTitleAndText, TwoObjects, SectionHeader, TwoTextAndTwoObjects, TitleObjectAndCaption, PictureAndCaption, Custom. Další typy rozložení nejsou nyní podporovány: Text, TwoColumnText, Table, TextAndChart, ChartAndText, Diagram, Chart, TextAndClipArt, ClipArtAndText, TextAndObject, ObjectAndText, Object, TextAndMedia, MediaAndText, ObjectOverText, TextOverObject, TextAndTwoObjects, TwoObjectsAndText, TwoObjectsOverText, FourObjects, ClipArtAndVerticalText, VerticalTitleAndTextOverChart, ObjectAndTwoObject, TwoObjectsAndObject. |
| layoutName | String | Název pro nové rozložení. Pokud je předán název, který již existuje, bude vyvolána výjimka ArgumentException. Pokud je předán parametr null, název je automaticky vygenerován s ohledem na předaný typ rozložení (například "Title Slide" nebo "1_Title Slide", "2_..", atd.). |

### Návratová hodnota

Přidaný snímek.

### Výjimky

| výjimka | podmínka |
| --- | --- |
| NotImplementedException | Vyvolána, pokud je předána nepodporovaná hodnota parametru *layoutType*. Typy rozložení, které nyní nejsou podporovány: Text, TwoColumnText, Table, TextAndChart, ChartAndText, Diagram, Chart, TextAndClipArt, ClipArtAndText, TextAndObject, ObjectAndText, Object, TextAndMedia, MediaAndText, ObjectOverText, TextOverObject, TextAndTwoObjects, TwoObjectsAndText, TwoObjectsOverText, FourObjects, ClipArtAndVerticalText, VerticalTitleAndTextOverChart, ObjectAndTwoObject, TwoObjectsAndObject. |
| ArgumentNullException | Vyvolána, pokud je *master* null. |
| ArgumentException | Vyvolána, pokud *master* patří jiné prezentaci. |
| ArgumentException | Vyvolána, pokud je hodnota názvu rozložení *layoutName* již použita ve sbírce rozložení *master*. |

### Poznámky

1) Přidané rozložení pro hodnotu SlideLayoutType.Custom parametru *layoutType* neobsahuje žádné zástupné řetězce ani tvary. 2) Analogie této metody je metoda [`Add`](../../imasterlayoutslidecollection/add) přístupná pomocí vlastnosti [`LayoutSlides`](../../imasterslide/layoutslides).

## FontsManager.GetFontEmbeddingLevel metoda

Určuje úroveň vkládání fontu z daného pole bajtů a názvu fontu.

```csharp
public EmbeddingLevel GetFontEmbeddingLevel(byte[] fontBytes, string fontName)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| fontBytes | Byte[] | Pole bajtů obsahující data fontu. |
| fontName | String | Název fontu. |

### Návratová hodnota

Úroveň vkládání zadaného fontu.

### Výjimky

| výjimka | podmínka |
| --- | --- |
| ArgumentNullException | Vyvolána, když je *fontBytes* null. |

### Příklady

```csharp
[C#]
using (Presentation pres = new Presentation(pptxFileName))
{
    // Načte všechny písma použité v prezentaci
    IFontData[] fontDatas = pres.FontsManager.GetFonts();

    // Získá pole bajtů představující běžný styl prvního písma v prezentaci
    byte[] bytes = pres.FontsManager.GetFontBytes(fontDatas[0], FontStyle.Regular);

    // Určuje úroveň vkládání písma
    EmbeddingLevel embeddingLevel = pres.FontsManager.GetFontEmbeddingLevel(bytes, fontDatas[0].FontName);
}
```

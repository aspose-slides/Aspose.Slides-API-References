## PictureFillFormat.TileFlip vlastnost

Otočí dlaždici textury kolem její horizontální, vertikální nebo obou os. Číst/Zapisovat [`TileFlip`](../../tileflip).

```csharp
public TileFlip TileFlip { get; set; }
```

### Poznámky

Výchozí hodnota je NoFlip.

### Příklady

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Získá formát vyplnění obrázkem tvaru
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Nastaví režim vyplnění obrázkem na Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Otočí dlaždici textury kolem její vertikální osy.
    pictureFillFormat.TileFlip = TileFlip.FlipY;
}
```

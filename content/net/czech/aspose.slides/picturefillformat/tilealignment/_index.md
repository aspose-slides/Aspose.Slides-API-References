## PictureFillFormat.TileAlignment vlastnost

Vrací nebo nastavuje, jak je textura zarovnána uvnitř tvaru. Toto nastavení určuje počáteční bod vzoru textury a to, jak se opakuje po celé ploše tvaru. Číst/Zapisovat [`RectangleAlignment`](../../rectanglealignment).

```csharp
public RectangleAlignment TileAlignment { get; set; }
```

### Poznámky

Výchozí hodnota je TopLeft.

### Příklady

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Získá formát výplně obrázkem tvaru
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Nastaví režim výplně obrázkem na Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Nastaví zarovnání dlaždic na pravý dolní roh
    pictureFillFormat.TileAlignment = RectangleAlignment.BottomRight;
}
```

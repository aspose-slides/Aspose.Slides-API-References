## PictureFillFormat.TileOffsetX vlastnost

Vrací nebo nastavuje vodorovný posun textury od počátku tvaru v bodech. Kladná hodnota posouvá texturu doprava, zatímco záporná hodnota ji posouvá doleva. Číst/zapisovat Single.

```csharp
public float TileOffsetX { get; set; }
```

### Příklady

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Získá formát výplně obrázkem tvaru
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Nastaví režim výplně obrázkem na Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Nastaví vodorovný posun textury na 20 bodů
    pictureFillFormat.TileOffsetX = 20f;
}
```

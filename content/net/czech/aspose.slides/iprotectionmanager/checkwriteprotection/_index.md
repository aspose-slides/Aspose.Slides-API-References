## IProtectionManager.CheckWriteProtection metoda

Určuje, zda je prezentace chráněna heslem pro úpravy.

```csharp
public bool CheckWriteProtection(string password)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| password | String | Heslo pro kontrolu. |

### Návratová hodnota

True, pokud je heslo platné; jinak false.

### Poznámky

1. Měli byste zkontrolovat vlastnost [`IsWriteProtected`](../iswriteprotected) před voláním této metody. 2. Když je heslo null nebo prázdné, tato metoda vrací false.

### Příklady

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

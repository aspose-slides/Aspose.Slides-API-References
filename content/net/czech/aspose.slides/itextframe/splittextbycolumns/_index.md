## ITextFrame.SplitTextByColumns metoda

Rozdělí textový obsah [`ITextFrame`](../../itextframe) na pole řetězců, kde každý prvek odpovídá samostatnému sloupci textu v rámci rámce.

```csharp
public string[] SplitTextByColumns()
```

### Návratová hodnota

Pole řetězců, kde každý řetězec představuje textový obsah konkrétního sloupce v [`ITextFrame`](../../itextframe).

### Poznámky

Pokud textový rámec neobsahuje více sloupců, vrácené pole bude mít jeden prvek obsahující celý text. Prázdné sloupce budou v poli reprezentovány prázdnými řetězci.

### Příklady

Následující příklad ukazuje, jak použít `SplitTextByColumns`:

```csharp
using (Presentation pres = new Presentation("example.pptx"))
{
    // Získejte první tvar na snímku a přetypujte jej na ITextFrame
    ITextFrame textFrame = pres.Slides[0].Shapes[0] as ITextFrame;
    // Rozdělte obsah textového rámce do sloupců
    string[] columnsText = textFrame.SplitTextByColumns();
    // Vytiskněte text každého sloupce do konzole
    foreach (string column in columnsText)
        Console.WriteLine(column);
}
```

## IPictureFillFormat.TileOffsetY vlastnost

Vrací nebo nastavuje vertikální posun textury od počátku tvaru v bodech. Kladná hodnota posouvá texturu dolů, zatímco záporná hodnota ji posouvá nahoru. Čtení/zápis Single.

```csharp
public float TileOffsetY { get; set; }
```

### Příklady

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Získá formát výplně obrázkem tvaru
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Nastaví režim výplně obrázkem na Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Nastaví vertikální posun textury na -50 bodů
    pictureFillFormat.TileOffsetY = -50;
}
```

## IPictureFillFormat.DeletePictureCroppedAreas metoda

Odstraní oříznuté oblasti výplně obrázku.

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### Návratová hodnota

Oříznutý obrázek nebo původní obrázek, pokud ořez není nutný.

### Poznámky

Tato metoda převádí metafily WMF/EMF na rastrový PNG obrázek během ořezávání.

### Příklady

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Získá PictureFrame
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // Odstraní oříznuté oblasti obrázku PictureFrame
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

## CompressImage(bool, PicturesCompression) {#compressimage}

Komprimuje obrázek snížením jeho velikosti na základě velikosti tvaru a zadaného rozlišení. Volitelně také odstraňuje oříznuté oblasti.

```csharp
public bool CompressImage(bool deleteCroppedAreasOfImage, PicturesCompression resolution)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| deleteCroppedAreasOfImage | Boolean | Pokud je true, metoda odstraní oříznuté oblasti obrázku, což může dále snížit jeho velikost. |
| resolution | PicturesCompression | Cílové rozlišení pro kompresi, určené hodnotou výčtu [`PicturesCompression`](../../../aspose.slides.export/picturescompression). |

### Návratová hodnota

Boolean hodnota indikující, zda byl obrázek úspěšně komprimován. Vrací `true`, pokud byl obrázek změněn velikostně nebo oříznut, jinak `false`.

### Výjimky

| výjimka | podmínka |
| --- | --- |
| ArgumentException | Vyvolána, když rozlišení není platná hodnota. |

### Poznámky

Tato metoda mění velikost a rozlišení obrázku podobně jako funkce PowerPointu „Formát obrázku → Komprimovat obrázky“.

### Příklady

Následující příklad ukazuje, jak použít metodu `CompressImage` ke snížení velikosti obrázku v prezentaci nastavením cílového rozlišení a odstraněním oříznutých oblastí:

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;
    // Komprimujte obrázek s cílovým rozlišením 150 DPI (webové rozlišení) a odstraňte oříznuté oblasti
    bool result = picFrame.PictureFormat.CompressImage(true, PicturesCompression.Dpi150);
}
```

public bool CompressImage(bool deleteCroppedAreasOfImage, float resolution)

| Parametr | Typ | Popis |

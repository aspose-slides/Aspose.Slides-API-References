## IPictureFillFormat.TileAlignment vlastnost

Vrací nebo nastavuje, jak je textura zarovnána uvnitř útvaru. Toto nastavení řídí výchozí bod vzoru textury a to, jak se opakuje po celém útvaru. Číst/Zapisovat [`RectangleAlignment`](../../rectanglealignment).

```csharp
public RectangleAlignment TileAlignment { get; set; }
```

### Poznámky

Výchozí hodnota je TopLeft.

### Příklady

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Získá formát výplně obrázku útvaru
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Nastaví režim výplně obrázku na Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Nastaví zarovnání dlaždicování na pravý dolní roh
    pictureFillFormat.TileAlignment = RectangleAlignment.BottomRight;
}
```

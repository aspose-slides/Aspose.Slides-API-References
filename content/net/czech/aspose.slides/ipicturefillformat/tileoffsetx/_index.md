## IPictureFillFormat.TileOffsetX vlastnost

Vrací nebo nastavuje horizontální posun textury od počátku tvaru v bodech. Kladná hodnota posune texturu doprava, záporná hodnota ji posune doleva. Čtení/Zápis Single.

```csharp
public float TileOffsetX { get; set; }
```

### Příklady

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Získá formát výplně obrázku tvaru
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Nastaví režim výplně obrázku na dlaždice
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Nastaví horizontální posun textury na 20 bodů
    pictureFillFormat.TileOffsetX = 20f;
}
```

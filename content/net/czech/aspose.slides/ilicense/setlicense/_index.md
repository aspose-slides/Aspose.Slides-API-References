## SetLicense(string) {#setlicense_1}

Licencuje komponentu.

```csharp
public void SetLicense(string licenseName)
```

| Parameter | Type | Description |
| --- | --- | --- |
| licenseName | String | Může být úplný nebo zkrácený název souboru nebo název vloženého zdroje. Použijte prázdný řetězec pro přepnutí do evaluačního režimu. |

### Poznámky

Snaží se najít licenci na následujících místech:

1. Explicitní cesta.

2. Složka sestavy komponenty.

3. Složka sestavy volajícího klienta.

4. Složka vstupní sestavy.

5. Vložený zdroj v sestavě volajícího klienta.

**Poznámka:**Na .NET Compact Framework se licence hledá pouze na těchto místech:

1. Explicitní cesta.

2. Vložený zdroj v sestavě volajícího klienta.

### Příklady

V tomto příkladu se pokusí najít licenční soubor s názvem MyLicense.lic ve složce, která obsahuje komponentu, ve složce, která obsahuje volající sestavu, ve složce vstupní sestavy a poté ve vložených zdrojích volající sestavy.

```csharp
[C#]

License license = new License();
license.SetLicense("MyLicense.lic");


[Visual Basic]

Dim license As License = New License
license.SetLicense("MyLicense.lic")
```

## FontsLoader.LoadExternalFonts metoda

Přidává další složky pro vyhledávání fontů.

```csharp
public static void LoadExternalFonts(string[] directories)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| directories | String[] | Adresáře pro načtení dalších fontů. |

### Příklady

Následující příklady ukazují, jak načíst vlastní fonty z formátu .TTF

```csharp
[C#]
// Cesta k adresáři dokumentů.
string dataDir = "C:\\";
// složky pro vyhledávání fontů
String[] folders = new String[] { dataDir };
// Načíst vlastní fonty ze složky
FontsLoader.LoadExternalFonts(folders);
// Proveďte nějakou práci a vykreslete prezentaci/slides rendering
using (Presentation presentation = new Presentation(dataDir + "DefaultFonts.pptx"))
{
    presentation.Save(dataDir + "NewFonts_out.pptx", SaveFormat.Pptx);
	// Vymazat mezipaměť fontů
	FontsLoader.ClearCache();
}
```

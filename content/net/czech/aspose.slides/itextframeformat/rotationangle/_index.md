## ITextFrameFormat.RotationAngle vlastnost

Určuje vlastní rotaci, která je aplikována na text uvnitř ohraničujícího rámečku. Pokud není uvedena, použije se rotace přidruženého tvaru. Pokud je uvedena, pak se použije nezávisle na tvaru. To znamená, že tvar může mít aplikovanou rotaci navíc k rotaci, která je aplikována přímo na text. Výsledná hodnota vizuální rotace textu je souhrnem této vlastnosti a předdefinovaného svislého typu v vlastnosti TextVerticalType. Čtení/Zápis Single.

```csharp
public float RotationAngle { get; set; }
```

### Příklady

Chtěte-li případ, kdy tvar má aplikovanou rotaci 90 stupňů po směru hodinových ručiček. K tomu je tělu textu také aplikována rotace -90 stupňů proti směru hodinových ručiček. Výsledný tvar se pak jeví jako natočený, ale text uvnitř se zdá, jako by nebyl vůbec natočen.

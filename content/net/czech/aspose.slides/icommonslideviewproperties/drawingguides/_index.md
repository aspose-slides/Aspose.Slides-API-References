## ICommonSlideViewProperties.DrawingGuides vlastnost

Vrací kolekci vodítek kreslení. Pouze ke čtení [`IDrawingGuidesCollection`](../../idrawingguidescollection)

```csharp
public IDrawingGuidesCollection DrawingGuides { get; }
```

### Příklady

Následující ukázkový kód ukazuje, jak přidat nová vodítka kreslení do prezentace PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    var slideSize = pres.SlideSize.Size;
    
    IDrawingGuidesCollection guides = pres.ViewProperties.SlideViewProperties.DrawingGuides;
    // Přidání nového svislého vodítka kreslení vpravo od středu snímku
    guides.Add(Orientation.Vertical, slideSize.Width / 2 + 12.5f);
    // Přidání nového vodorovného vodítka kreslení pod střed snímku
    guides.Add(Orientation.Horizontal, slideSize.Height / 2 + 12.5f);
    
    pres.Save("DrawingGuides_out.pptx", SaveFormat.Pptx);
}
```

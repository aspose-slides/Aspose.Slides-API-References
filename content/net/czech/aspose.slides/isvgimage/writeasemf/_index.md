## ISvgImage.WriteAsEmf metoda

Uloží SVG obrázek jako soubor EMF.

```csharp
public void WriteAsEmf(Stream stream)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| stream | Stream | Cílový stream |

### Výjimky

| Výjimka | podmínka |
| --- | --- |
| ArgumentNullException | Cílový stream je **null** |

### Příklady

Následující příklad ukazuje, jak uložit SVG obrázek do metafile.

```csharp
[C#]
// Vytvoří nový SVG obrázek
ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));

// Uloží SVG obrázek jako metafájl
using (var fileStream = System.IO.File.OpenWrite("SvgAsEmf.emf"))
    svgImage.WriteAsEmf(fileStream);
```

Tento příklad ukazuje, jak přidat SVG obrázek jako metafile do kolekce obrázků prezentace.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // Vytvoří nový SVG obrázek
    ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));
    using (var memStream = new MemoryStream())
    {
        // Uloží SVG obrázek jako metafájl
        svgImage.WriteAsEmf(memStream);
        // Přidá metafájl do kolekce obrázků
        pres.Images.AddImage(memStream.ToArray());
    }
}
```

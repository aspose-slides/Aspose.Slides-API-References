## Presentation.AllCustomXmlParts vlastnost

Vrací všechny vlastní datové části v prezentaci. Pouze pro čtení [`ICustomXmlPart`](../../icustomxmlpart)[].

```csharp
public ICustomXmlPart[] AllCustomXmlParts { get; }
```

### Příklady

Následující příklady ukazují, jak vymazat všechny vlastní XML části z PowerPoint prezentaci.

```csharp
[C#]
using (Presentation pres = new Presentation("PresentationWithCustomXml.pptx"))
{
	// Procházej všechny vlastní XML části
    foreach (ICustomXmlPart item in pres.AllCustomXmlParts)
    {
        item.Remove();
    }
    pres.Save("out.pptx", SaveFormat.Pptx);
}
```

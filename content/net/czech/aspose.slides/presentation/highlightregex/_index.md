## Presentation.HighlightRegex metoda

Zvýrazní všechny shody regulárního výrazu zadanou barvou.

```csharp
public void HighlightRegex(Regex regex, Color highlightColor, IFindResultCallback callback)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| regex | Regex | Regulární výraz Regex pro získání řetězců k zvýraznění. |
| highlightColor | Color | Barva pro zvýraznění textu. |
| callback | IFindResultCallback | Objekt zpětného volání pro přijímání výsledků vyhledávání [`IFindResultCallback`](../../ifindresultcallback). |

### Příklady

Následující ukázkový kód ukazuje, jak zvýraznit text v prezentaci PowerPoint pomocí regulárního výrazu.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// zvýraznění všech slov s 10 a více znaky
	presentation.HighlightRegex(regex, Color.Blue, null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

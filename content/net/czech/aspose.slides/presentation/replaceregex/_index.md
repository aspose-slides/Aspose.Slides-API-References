## Presentation.ReplaceRegex metoda

Nahrazuje všechny shody regulárního výrazu zadaným řetězcem.

```csharp
public void ReplaceRegex(Regex regex, string newText, IFindResultCallback callback)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| regex | Regex | Regulární výraz Regex sloužící k získání řetězců, které mají být nahrazeny. |
| newText | String | Řetězec, který nahradí všechny výskyty řetězců, jež mají být nahrazeny. |
| callback | IFindResultCallback | Objekt zpětného volání pro získání výsledků vyhledávání [`IFindResultCallback`](../../ifindresultcallback). |

### Příklady

Následující ukázka kódu ukazuje, jak nahradit text pomocí regulárního výrazu zadaným řetězcem.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// Nahradí všechna slova s 10 nebo více znaky řetězcem '***'
	presentation.ReplaceRegex(regex, "***", null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

## FontFallBackRulesCollection.Remove metoda

Odstraňuje první výskyt konkrétního pravidla FallBack ze sbírky.

```csharp
public void Remove(IFontFallBackRule targetRule)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| targetRule | IFontFallBackRule | Pravidlo, které se má odstranit ze sbírky. |

### Příklady

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    //Získání prázdné nebo předinicializované sbírky pravidel z FontsManager
    IFontFallBackRulesCollection rulesList = pres.FontsManager.FontFallBackRulesCollection;

    //Přidání několika pravidel do sbírky
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));
    rulesList.Add(new FontFallBackRule(0x3040, 0x309F, "MS Mincho"));

    //Získání objektu prvního pravidla ve sbírce
    IFontFallBackRule firstRule = rulesList[0];
    //Odstraňování 
    rulesList.Remove (firstRule);
}
```

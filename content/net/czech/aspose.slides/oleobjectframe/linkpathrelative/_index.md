## OleObjectFrame.LinkPathRelative vlastnost

Vrací relativní cestu k propojenému souboru, pokud existuje, jinak vrací prázdný řetězec. Pouze pro čtení String.

```csharp
public string LinkPathRelative { get; }
```

### Poznámky

V prezentacích Ppt mohou některé odkazy Ole objektů mít relativní reprezentaci.

### Příklady

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.ppt"))
{
    IOleObjectFrame oleFrame = presentation.Slides[0].Shapes[0] as IOleObjectFrame;

    if (oleFrame != null)
    {
        Console.WriteLine("The relative path: " + oleFrame.LinkPathRelative);
    } 
}
```

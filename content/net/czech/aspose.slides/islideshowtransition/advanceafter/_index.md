## ISlideShowTransition.AdvanceAfter vlastnost

Tento atribut určuje, zda se prezentace přesune na další snímek po uplynutí určitého času. Číst/Zapisovat Boolean.

```csharp
public bool AdvanceAfter { get; set; }
```

### Příklady

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    // Získat první přechod snímku
    ISlideShowTransition slideTransition = pres.Slides[0].SlideShowTransition;
    
    // Zkontrolovat, zda je nastaven příznak Advance Slide After
    if (slideTransition.AdvanceAfter)
    {
        // Získat hodnotu Advance Slide After Time
        uint advanceAfterTime = slideTransition.AdvanceAfterTime;
```

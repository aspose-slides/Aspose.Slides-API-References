## SvgImage.WriteAsEmf metoda

Uloží SVG obrázek jako soubor EMF.

```csharp
public void WriteAsEmf(Stream stream)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| stream | Stream | Cílový proud |

### Výjimky

| výjimka | podmínka |
| --- | --- |
| ArgumentNullException | Cílový proud je **null** |

### Příklady

Následující příklad ukazuje, jak uložit SVG obrázek do metafilu.

```csharp
[C#]
// Vytvoří nový SVG obrázek
ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));

// Uloží SVG obrázek jako metafil
using (var fileStream = System.IO.File.OpenWrite("SvgAsEmf.emf"))
    svgImage.WriteAsEmf(fileStream);
```

Tento vzor ukazuje, jak přidat SVG obrázek jako metafil do kolekce obrázků prezentace.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // Vytvoří nový SVG obrázek
    ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));
    using (var memStream = new MemoryStream())
    {
        // Uloží SVG obrázek jako metafil
        svgImage.WriteAsEmf(memStream);
        // Přidá metafil do kolekce obrázků
        pres.Images.AddImage(memStream.ToArray());
    }
}
```

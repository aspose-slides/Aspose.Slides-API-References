## LineSketchType výčet

Represents which sketch type or effect a shape has been assigned.

```csharp
public enum LineSketchType
```

### Hodnoty

| Název | Hodnota | Popis |
| --- | --- | --- |
| NotDefined | `-1` | Určuje, že efekt Sketch tvaru není definován. |
| None | `0` | Určuje, že tvar nemá žádný efekt Sketch. To je ekvivalentní tomu, že je tato vlastnost prázdná. |
| Curved | `1` | Určuje, že tvar má efekt Curved, který změní každou hranu tvaru na jednu velkou jemnou křivku. |
| Freehand | `2` | Určuje, že tvar má efekt Freehand, který nejvíce připomíná nedokonalou ručně kreslenou čáru. |
| Scribble | `3` | Určuje, že tvar má efekt Scribble, který má přehnané oscilace, jako by byl záměrně načmárán. |

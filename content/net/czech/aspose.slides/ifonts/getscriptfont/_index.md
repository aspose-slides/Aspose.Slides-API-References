## IFonts.GetScriptFont metoda

Získá název písma přiřazeného k určitému skriptovému tagu v motivu prezentace.

```csharp
public string GetScriptFont(string script)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| script | String | Kód skriptu BCP-47 (např. "Latn", "Cyrl", "Jpan") používaný k identifikaci psacího systému. |

### Návratová hodnota

Název písma používaného pro zadaný skript, nebo `null`, pokud skript není definován.

### Příklady

Tento příklad ukazuje, jak získat písmo přiřazené k cyrilskému skriptu v motivu prezentace.

```csharp
[C#]
string font = presentation.MasterTheme.FontScheme.Major.GetScriptFont("Cyrl");
Console.WriteLine("Font for Cyrillic script: " + font);
```

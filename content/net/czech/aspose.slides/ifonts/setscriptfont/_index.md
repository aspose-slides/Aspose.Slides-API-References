## IFonts.SetScriptFont metoda

Přiřadí název písma ke konkrétnímu štítku skriptu, který určuje, jak bude text tohoto skriptu v prezentaci vykreslen.

```csharp
public void SetScriptFont(string script, string fontName)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| script | String | Kód skriptu podle BCP-47 (např. "Arab", "Hebr", "Hans") identifikující písmo. |
| fontName | String | Název písma, které se má přiřadit ke specifikovanému skriptu. |

### Příklady

Tento příklad ukazuje, jak nastavit písmo pro arabský skript na "Segoe UI":

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segoe UI");
```

## ImageTransformOperationCollection.CopyTo metoda

Zkopíruje prvky rozhraní ICollection do pole Array, počínaje konkrétním indexem v Array.

```csharp
public void CopyTo(IImageTransformOperation[] array, int arrayIndex)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| array | IImageTransformOperation[] | Jednorozměrné pole Array, které je cílem prvků zkopírovaných z ICollection. Pole Array musí mít nulové indexování. |
| arrayIndex | Int32 | Nulový index v *array*, od kterého kopírování začíná. |

### Výjimky

| výjimka | podmínka |
| --- | --- |
| ArgumentNullException | *array* je null. |
| ArgumentOutOfRangeException | *arrayIndex* je menší než 0. |
| ArgumentException | Počet prvků ve zdrojovém ICollection je větší než dostupný prostor od *arrayIndex* až do konce cílového *array*. |

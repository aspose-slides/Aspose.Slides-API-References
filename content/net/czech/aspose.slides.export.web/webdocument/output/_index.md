## WebDocument.Output vlastnost

Vrací kolekci výstupních prvků dokumentu. Pouze pro čtení `Output`.

```csharp

```

### Příklady

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    var options = new WebDocumentOptions
    {
        TemplateEngine = new RazorTemplateEngine(),
        OutputSaver = new FileOutputSaver(),
        EmbedImages = false
    };
    
    WebDocument document = new WebDocument(options);

    // přidá "index.html" do výstupních souborů, používá šablonu "index" k jejímu vygenerování a proměnnou pres jako model
    document.Output.Add("index.html", "index", pres);

    // ... nastaví další možnosti dokumentu a poté dokument uloží
    document.Save();
}
```

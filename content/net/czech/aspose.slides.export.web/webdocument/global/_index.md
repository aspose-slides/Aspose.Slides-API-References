### Příklady

Použitím této `Global` vlastnosti (implementace rozhraní [`Storage`](../../storage)) lze vlastnost použít později v šabloně:

```csharp
[C#]
var options = new WebDocumentOptions
{
    TemplateEngine = new RazorTemplateEngine(),
    OutputSaver = new FileOutputSaver(),
    EmbedImages = false
};

WebDocument document = new WebDocument(options);

// vložte vlastnost "slideMargin" k použití ze šablon
document.Global.Put("slideMargin", 10);

// ... nastavte další možnosti dokumentu a pak dokument uložte
document.Save();
```

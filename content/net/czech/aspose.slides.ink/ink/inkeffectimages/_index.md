## Ink.InkEffectImages property

Získá kolekci vlastních obrázků používaných k simulaci vizuálních efektů pro inkové štětce. Tyto obrázky se používají při vykreslování inku s konkrétními [`InkEffectType`](../../inkeffecttype) hodnotami, jako jsou Galaxy, Rainbow atd. Poskytnutím vlastních obrázků můžete ovlivnit, jak se každý inkový efekt zobrazí.

```csharp
public static IDictionary<InkEffectType, IImage> InkEffectImages { get; }
```

### Remarks

Tato vlastnost umožňuje nahradit výchozí textury inkových efektů uživatelskými, což je zvláště užitečné, když jsou výchozí prostředky omezeny licencí nebo nejsou během běhu k dispozici. Každý záznam ve slovníku musí přiřadit [`InkEffectType`](../../inkeffecttype) hodnotu k odpovídajícímu [`IImage`](../../../aspose.slides/iimage) objektu (např. Bitmap nebo rozhraní Aspose image).

### Examples

```csharp
[C#]
IImage image = Images.FromFile("image.png");
Ink.InkEffectImages.Add(InkEffectType.Galaxy, image);
```

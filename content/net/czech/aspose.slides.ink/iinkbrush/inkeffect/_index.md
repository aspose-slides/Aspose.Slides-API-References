## IInkBrush.InkEffect vlastnost

Vrací typ ink efektu (např. Galaxy, Gold, Silver), který určuje vizuální styl ink tahu. Hodnota je načtena z vlastnosti štětce "inkEffects". Pokud není zadán žádný rozpoznaný efekt, vrací se NotDefined.

```csharp
public InkEffectType InkEffect { get; }
```

### Příklady

Příklad:

```csharp
[C#]
using (var pres = new Presentation("Presentation.pptx"))
{
    Ink ink = pres.Slides[0].Shapes[0] as Ink;
    IInkBrush brush = ink.Traces[0].Brush;
    Console.WriteLine("InkEffects = {0}", brush.InkEffect);
}
```

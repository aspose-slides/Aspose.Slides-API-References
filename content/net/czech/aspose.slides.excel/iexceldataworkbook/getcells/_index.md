## IExcelDataWorkbook.GetCells metoda

Vrátí kolekci buněk ze sešitu, které odpovídají zadanému vzorci.

```csharp
public ReadOnlyCollection<IExcelDataCell> GetCells(string formula, bool skipHiddenCells)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| formula | String | Vzorec nebo výraz rozsahu (např. "Sheet1!A1:B3"), který slouží k určení cílových buněk. |
| skipHiddenCells | Boolean | Pokud je `true`, skryté buňky (např. ve skrytých řádcích nebo sloupcích) budou z výsledku vyloučeny. |

### Návratová hodnota

Seznam buněk pouze pro čtení, které odpovídají zadanému vzorci.

### Příklady

Příklad:

```csharp
[C#]
ExcelDataWorkbook wb = new ExcelDataWorkbook(testFile);
IReadOnlyList<IExcelDataCell> cells = wb.GetCells("Sheet1!A2:A6", false);
Console.WriteLine(cells.Count); //Výstup: 5
```

## ToJpeg(Presentation, string) {#tojpeg}

Převádí vstupní prezentaci na sadu obrázků ve formátu JPEG. Pokud je zadán výstupní soubor jako „myPath/myFilename.jpeg“, výsledek bude uložen jako sada souborů „myPath/myFilename_N.jpeg“, kde N je číslo snímku.

```csharp
public static void ToJpeg(Presentation pres, string outputFileName)
```

| pres | Presentation | Vstupní prezentace. |

```csharp
public static void ToJpeg(Presentation pres, string outputFileName, float scale, 
    IRenderingOptions options)
```

| Parametr | Typ | Popis |

| outputFileName | String | Název výstupního souboru. |

    Convert.ToJpeg(pres, "presImage.jpeg", 2f, options);

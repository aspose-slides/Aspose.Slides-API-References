## ToTiff(Presentation, string) {#totiff}

Převádí vstupní prezentaci na sadu obrázků ve formátu TIFF. Pokud je výstupní soubor zadán jako "myPath/myFilename.tiff", výsledek bude uložen jako sada souborů "myPath/myFilename_N.tiff", kde N je číslo snímku.

public static void ToTiff(Presentation pres, string outputFileName)

## ToTiff(Presentation, string, ITiffOptions, bool) {#totiff_1}

Převádí vstupní prezentaci do formátu TIFF s vlastními možnostmi. Pokud je výstupní soubor zadán jako "myPath/myFilename.tiff" a *multipage* je `false`, výsledek bude uložen jako sada souborů "myPath/myFilename_N.tiff", kde N je číslo snímku. V opačném případě, pokud je *multipage* `true`, výsledek bude vícestránkový dokument "myPath/myFilename.tiff".

```csharp
public static void ToTiff(Presentation pres, string outputFileName, ITiffOptions options, 
    bool multipage)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| pres | Presentation | Vstupní prezentace. |
| outputFileName | String | Název výstupního souboru. |
| options | ITiffOptions | Možnosti ukládání TIFF. |
| multipage | Boolean | Určuje, zda má být generovaný dokument TIFF vícestránkový. |

### Výjimky

| výjimka | podmínka |
| --- | --- |
| ArgumentException |  |

### Příklady

```csharp
ITiffOptions options = new TiffOptions()
{
    CompressionType = TiffCompressionTypes.CCITT3,
    SlidesLayoutOptions = new NotesCommentsLayoutingOptions()
    {
        NotesPosition = NotesPositions.BottomTruncated
    }
};

using (var pres = new Presentation("pres.pptx"))
   Convert.ToTiff(pres, "pres.tiff", options, false);
```

## ToPng(Presentation, string) {#topng}

Převádí vstupní prezentaci na sadu obrázků ve formátu PNG. Pokud je zadáno výstupní jméno souboru jako "myPath/myFilename.png", výsledek bude uložen jako sada souborů "myPath/myFilename_N.png", kde N je číslo snímku.

```csharp
public static void ToPng(Presentation pres, string outputFileName)
```

## ToPng(Presentation, string, float, IRenderingOptions) {#topng_1}

Převádí vstupní prezentaci na sadu obrázků ve formátu PNG. Pokud je zadáno výstupní jméno souboru jako "myPath/myFilename.png", výsledek bude uložen jako sada souborů "myPath/myFilename_N.png", kde N je číslo snímku.

```csharp
public static void ToPng(Presentation pres, string outputFileName, float scale, 
    IRenderingOptions options)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| pres | Presentation | Vstupní prezentace. |
| outputFileName | String | Výstupní jméno souboru. |
| scale | Single | Měřítko aplikované na výstupní obrázky vzhledem k původní velikosti snímku. |
| options | IRenderingOptions | Možnosti vykreslování. |

### Výjimky

| Výjimka | Podmínka |
| --- | --- |
| ArgumentException |  |

### Příklady

```csharp
IRenderingOptions options = new RenderingOptions()
{
    SlidesLayoutOptions = new NotesCommentsLayoutingOptions()
    {
        NotesPosition = NotesPositions.BottomTruncated
    }
};

using (var pres = new Presentation("pres.pptx"))
   Convert.ToPng(pres, "presImage.png", 2f, options);
```

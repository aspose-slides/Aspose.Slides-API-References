## SlideShowTransition.SoundIsBuiltIn vlastnost

Specifikuje, zda je tento zvuk vestavěný zvuk či nikoli. Pokud je tento atribut nastaven na true, aplikace generující dokument je upozorněna, aby zkontrolovala atribut name uvedený pro tento zvuk v jejím seznamu vestavěných zvuků a může poté zobrazit vlastní název nebo uživatelské rozhraní podle potřeby. Čtení-zápis Boolean.

public bool SoundIsBuiltIn { get; set; }

## DataLabelFormat.Separator vlastnost

Nastavuje nebo vrací Variant představující oddělovač použitý pro datové popisky v grafu. Čtení/Zápis String.

```csharp
public string Separator { get; set; }
```

### Poznámky

Pokud je nadřazeným objektem tohoto DataLabelFormat objekt DataLabelCollection kolekce datových popisků, pak tato vlastnost získává nebo nastavuje výchozí hodnotu vlastnosti Separator pro nové datové popisky v kolekci DataLabelCollection. Nastavení této vlastnosti s hodnotou také nastaví tuto hodnotu do vlastnosti Separator pro všechny datové popisky v kolekci DataLabelCollection (např. "DataLabels.DefaultDataLabelFormat.Separator = val;" způsobí, že všechny DataLabels[i].Separator jsou rovnaky val).

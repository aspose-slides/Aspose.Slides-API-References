## DataLabelFormat.ShowCategoryName vlastnost

Reprezentuje chování zobrazování názvu kategorie popisku dat pro specifikovaný graf. True pro zobrazení názvu kategorie pro popisky dat v grafu. False pro skrytí. Read/write Boolean.

```csharp
public bool ShowCategoryName { get; set; }
```

### Poznámky

Pokud je nadřazený objekt DataLabelFormat kolekcí DataLabelCollection popisků dat, potom tato vlastnost získává nebo nastavuje výchozí hodnotu vlastnosti ShowCategoryName pro nové popisky dat v kolekci DataLabelCollection. Nastavením této vlastnosti na hodnotu se tato hodnota také nastaví pro vlastnost ShowCategoryName všech popisků dat v kolekci DataLabelCollection (i.e. "DataLabels.DefaultDataLabelFormat.ShowCategoryName = val;" způsobí, že všechny DataLabels[i].ShowCategoryName jsou rovny val).

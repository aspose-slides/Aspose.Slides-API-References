## DataLabelFormat.Position vlastnost

Představuje pozici datové popisky. Číst/Zapisovat [`LegendDataLabelPosition`](../../legenddatalabelposition).

```csharp
public LegendDataLabelPosition Position { get; set; }
```

### Poznámky

Pokud je nadřazený prvek tohoto objektu DataLabelFormat kolekcí DataLabelCollection datových popisek, pak tato vlastnost získá nebo nastaví výchozí hodnotu vlastnosti Position pro nové datové popisky v kolekci DataLabelCollection. Zastupuje pozici pro objekty DataLabel. Nastavením této vlastnosti na hodnotu se tato hodnota také nastaví na vlastnost Position pro všechny datové popisky v kolekci DataLabelCollection (např. "DataLabels.DefaultDataLabelFormat.Position = val;" způsobí, že všechny DataLabels[i].Position budou rovny val).

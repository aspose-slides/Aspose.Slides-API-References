## DataLabelFormat.ShowLeaderLines vlastnost

Representuje chování zobrazení čar vůdců popisků dat v určeném grafu. True zobrazuje čáry vůdců. False skryje. Čtení/zápis Boolean.

```csharp
public bool ShowLeaderLines { get; set; }
```

### Poznámky

Pokud je rodičem tohoto objektu DataLabelFormat kolekce DataLabelCollection popisků dat, pak tato vlastnost získá nebo nastaví výchozí hodnotu vlastnosti ShowLeaderLines pro nové popisky dat v kolekci DataLabelCollection. Nastavením této vlastnosti na hodnotu se také nastaví tato hodnota pro vlastnost ShowLeaderLines všech popisků dat v kolekci DataLabelCollection (např. "DataLabels.DefaultDataLabelFormat.ShowLeaderLines = val;" způsobí, že všechny DataLabels[i].ShowLeaderLines jsou rovny val).

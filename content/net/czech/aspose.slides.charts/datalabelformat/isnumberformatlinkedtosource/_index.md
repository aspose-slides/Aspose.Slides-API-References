## DataLabelFormat.IsNumberFormatLinkedToSource vlastnost

Čtení/zápis Boolean.

```csharp
public bool IsNumberFormatLinkedToSource { get; set; }
```

### Poznámky

Pokud je rodičem tohoto objektu DataLabelFormat kolekce DataLabelCollection datových popisků, pak tato vlastnost získá nebo nastaví výchozí hodnotu vlastnosti IsNumberFormatLinkedToSource pro nové datové popisky v kolekci DataLabelCollection. Nastavení této vlastnosti s hodnotou také nastaví tuto hodnotu vlastnosti IsNumberFormatLinkedToSource pro všechny datové popisky v kolekci DataLabelCollection (např. "DataLabels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = val;" způsobí, že všechny DataLabels[i].IsNumberFormatLinkedToSource jsou rovny val).

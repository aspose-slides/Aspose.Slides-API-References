## DataLabelFormat.ShowLabelValueFromCell vlastnost

Reprezentuje chování zobrazování hodnoty buňky popisku dat pro daný graf. True zobrazuje hodnotu buňky. False pro skrytí. Čtení/zápis Boolean.

```csharp
public bool ShowLabelValueFromCell { get; set; }
```

### Poznámky

Pokud je nadřazeným objektem DataLabelFormat kolekce DataLabelCollection popisků dat, pak tato vlastnost získává nebo nastavuje výchozí hodnotu vlastnosti ShowLabelValueFromCell pro nové popisky dat v kolekci DataLabelCollection. Nastavením této vlastnosti s hodnotou se také nastaví tato hodnota vlastnosti ShowLabelValueFromCell pro všechny popisky dat v kolekci DataLabelCollection (např. "DataLabels.DefaultDataLabelFormat.ShowLabelValueFromCell = val;" způsobí, že všechny DataLabels[i].ShowLabelValueFromCell budou rovny val).

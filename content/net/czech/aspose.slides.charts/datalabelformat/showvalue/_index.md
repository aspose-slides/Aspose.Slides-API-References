## DataLabelFormat.ShowValue vlastnost

Representuje chování zobrazování hodnoty procenta popisku dat pro daný graf. True zobrazí hodnotu procenta. False ji skryje. Čtení/zápis Boolean.

```csharp
public bool ShowValue { get; set; }
```

### Poznámky

Pokud je nadřazený objekt DataLabelFormat součástí kolekce DataLabelCollection popisků dat, pak tato vlastnost získá nebo nastaví výchozí hodnotu vlastnosti ShowValue pro nové popisky dat v kolekci DataLabelCollection. Nastavením této vlastnosti na hodnotu se také nastaví tato hodnota vlastnosti ShowValue pro všechny popisky dat v kolekci DataLabelCollection (tj. "DataLabels.DefaultDataLabelFormat.ShowValue = val;" způsobí, že všechna DataLabels[i].ShowValue bude rovna val).

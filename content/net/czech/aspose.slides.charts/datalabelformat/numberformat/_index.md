## Vlastnost DataLabelFormat.NumberFormat

Reprezentuje formátovací řetězec pro objekt DataLabels. Čtení/zápis String.

```csharp
public string NumberFormat { get; set; }
```

### Poznámky

Pokud je rodičem tohoto objektu DataLabelFormat kolekce DataLabelCollection obsahující datové popisky, pak tato vlastnost získá nebo nastaví výchozí hodnotu vlastnosti NumberFormat pro nové datové popisky v kolekci DataLabelCollection. Když je tato vlastnost nastavena na hodnotu, tato hodnota je také nastavena pro vlastnost NumberFormat všech datových popisků v kolekci DataLabelCollection (např. "DataLabels.DefaultDataLabelFormat.NumberFormat = val;" způsobí, že všechny DataLabels[i].NumberFormat mají hodnotu val).

### Příklady

```csharp
[C#]
series.Labels.DefaultDataLabelFormat.ShowValue = true;
series.Labels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = false;
series.Labels.DefaultDataLabelFormat.NumberFormat = "0.0%";
```

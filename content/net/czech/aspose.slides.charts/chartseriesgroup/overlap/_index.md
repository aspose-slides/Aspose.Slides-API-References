## ChartSeriesGroup.Overlap vlastnost

Specifikuje, jak moc se pruhy a sloupce překrývají v 2-D grafech, jako procento (od –100 % do 100 %).  
- –100 %: Maximální mezera (pruhy jsou zcela oddělené).  
- 0 %: Pruhy jsou umístěny vedle sebe bez překrytí nebo mezery.  
- 100 %: Maximální překrytí (pruhy se zcela překrývají).  
Tato vlastnost je čtení/zápis SByte.

```csharp
public sbyte Overlap { get; set; }
```

### Výjimky

| výjimka | podmínka |
| --- | --- |
| ArgumentOutOfRangeException | Vyvolána, když je hodnota nastavena mimo platný rozsah –100 až 100. |

### Příklady

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // Nastavit překrytí na 55%

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

## IDataLabelFormat.ShowLabelValueFromCell property

Zastupuje chování zobrazení hodnoty buňky popisku dat určeného grafu. True zobrazuje hodnotu buňky. False ji skryje. Číst/Zapisovat Boolean.

```csharp
public bool ShowLabelValueFromCell { get; set; }
```

### Poznámky

Pokud je rodičem tohoto objektu DataLabelFormat kolekce DataLabelCollection popisků dat, pak tato vlastnost získá nebo nastaví výchozí hodnotu vlastnosti ShowLabelValueFromCell pro nové popisky dat v kolekci DataLabelCollection. Nastavením této vlastnosti na hodnotu se tato hodnota také přiřadí vlastnosti ShowLabelValueFromCell pro všechny popisky dat v kolekci DataLabelCollection (např. "DataLabels.DefaultDataLabelFormat.ShowLabelValueFromCell = val;" způsobí, že všechna DataLabels[i].ShowLabelValueFromCell jsou rovna val).

## IDataLabelFormat.ShowValue vlastnost

Representuje chování zobrazení procentuální hodnoty popisku dat určeného grafu. True zobrazuje procentuální hodnotu. False skrývá. Číst/zapisovat Boolean.

```csharp
public bool ShowValue { get; set; }
```

### Poznámky

Pokud je rodičem tohoto objektu DataLabelFormat kolekce DataLabelCollection obsahující popisky dat, pak tato vlastnost získá nebo nastaví výchozí hodnotu vlastnosti ShowValue pro nové popisky dat v kolekci DataLabelCollection. Nastavením této vlastnosti na hodnotu se tato hodnota také nastaví na vlastnost ShowValue pro všechny popisky dat v kolekci DataLabelCollection (např. "DataLabels.DefaultDataLabelFormat.ShowValue = val;" způsobí, že všechny DataLabels[i].ShowValue jsou rovny val).

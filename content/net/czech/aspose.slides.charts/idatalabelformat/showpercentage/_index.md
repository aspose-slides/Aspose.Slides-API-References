## IDataLabelFormat.ShowPercentage vlastnost

Reprezentuje chování zobrazování hodnoty procenta popisku dat v určeném grafu. True zobrazí hodnotu procenta. False ji skryje. Čtení/zápis Boolean.

```csharp
public bool ShowPercentage { get; set; }
```

### Poznámky

Pokud je nadřazeným objektem tohoto DataLabelFormat objekt DataLabelCollection, tedy sbírka popisků dat, pak tato vlastnost získá nebo nastaví výchozí hodnotu vlastnosti ShowPercentage pro nové popisky dat v kolekci DataLabelCollection. Nastavením této vlastnosti na hodnotu se také nastaví tato hodnota vlastnosti ShowPercentage pro všechny popisky dat v kolekci DataLabelCollection (např. "DataLabels.DefaultDataLabelFormat.ShowPercentage = val;" způsobí, že všechny DataLabels[i].ShowPercentage jsou rovny val).

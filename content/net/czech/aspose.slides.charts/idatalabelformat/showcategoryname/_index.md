## IDataLabelFormat.ShowCategoryName vlastnost

Reprezentuje chování zobrazování názvu kategorie popisku dat určeného grafu. True pro zobrazení názvu kategorie pro popisky dat v grafu. False pro skrytí. Číst/zapisovat Boolean.

```csharp
public bool ShowCategoryName { get; set; }
```

### Poznámky

Pokud je rodičem tohoto objektu DataLabelFormat kolekce DataLabelCollection popisků dat, pak tato vlastnost získává nebo nastavuje výchozí hodnotu vlastnosti ShowCategoryName pro nové popisky dat v kolekci DataLabelCollection. Nastavením této vlastnosti na hodnotu se také tato hodnota nastaví pro vlastnost ShowCategoryName všech popisků dat v kolekci DataLabelCollection (i.e. "DataLabels.DefaultDataLabelFormat.ShowCategoryName = val;" způsobí, že všechny DataLabels[i].ShowCategoryName jsou rovny val).

## IDataLabelFormat.ShowLeaderLines vlastnost

Reprezentuje chování zobrazování čar vodicích linií popisků dat určeného grafu. True zobrazuje čáry vodicích linií. False skryje. Čtení/zápis Boolean.

```csharp
public bool ShowLeaderLines { get; set; }
```

### Poznámky

Pokud je rodičem tohoto objektu DataLabelFormat kolekce DataLabelCollection obsahující popisky dat, pak tato vlastnost získává nebo nastavuje výchozí hodnotu vlastnosti ShowLeaderLines pro nové popisky dat v kolekci DataLabelCollection. Nastavení této vlastnosti s hodnotou také nastaví tuto hodnotu pro vlastnost ShowLeaderLines u všech popisků dat v kolekci DataLabelCollection (např. "DataLabels.DefaultDataLabelFormat.ShowLeaderLines = val;" způsobí, že všechny DataLabels[i].ShowLeaderLines budou rovny val).

## IDataLabel.IsVisible vlastnost

False znamená, že popisek dat není viditelný (a tak všechny příznaky Show*-flags (ShowValue, ...) jsou false). Pouze pro čtení Boolean.

```csharp
public bool IsVisible { get; }
```

### Poznámky

Pokud je popisek dat viditelný, můžete jej skrýt pomocí metody Hide(). Ale pokud popisek dat není viditelný (IsVisible je false), můžete jej zobrazit nastavením příznaků Show*-flags (ShowValue, ...) do stavu true.

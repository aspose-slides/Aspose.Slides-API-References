## IChartPlotArea.LayoutTargetType vlastnost

Pokud je rozložení vykreslovací oblasti definováno ručně, tato vlastnost určuje, zda má být rozložení vykreslovací oblasti provedeno podle jejího vnitřku (ne včetně os a popisků os) nebo podle vnějšího okraje (včetně os a popisků os). Čtení/zápis `LayoutTargetType`.

```csharp
public LayoutTargetType LayoutTargetType { get; set; }
```

## IChartTextBlockFormat.RotationAngle vlastnost

Specifikuje vlastní rotaci aplikovanou na text uvnitř ohraničujícího rámečku. Pokud není uvedena, použije se rotace odpovídajícího tvaru. Pokud je uvedena, použije se nezávisle na tvaru. To znamená, že tvar může mít aplikovanou rotaci navíc k rotaci samotného textu. Výsledná hodnota vizuální rotace textu je souhrnem této vlastnosti a předdefinovaného svislého typu ve vlastnosti TextVerticalType. Čtení/Zápis Single.

```csharp
public float RotationAngle { get; set; }
```

### Příklady

Uvažujme případ, kdy má tvar rotaci 90 stupňů po směru hodinových ručiček. Navíc má samotné tělo textu rotaci -90 stupňů proti směru hodinových ručiček. Pak se výsledný tvar jeví jako otočený, ale text uvnitř něj se jeví, jako by vůbec nebyl otočen.

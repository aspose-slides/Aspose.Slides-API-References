## Výčet LayoutTargetType

Pokud je rozvržení oblasti grafu definováno ručně, tato vlastnost určuje, zda rozvrhnout oblast grafu podle jejího vnitřku (bez os a popisků os) nebo podle vnějšího okraje (včetně os a popisků os).

```csharp
public enum LayoutTargetType
```

### Hodnoty

| Name | Value | Description |
| --- | --- | --- |
| Inner | `0` | Určuje, že velikost oblasti grafu stanoví velikost oblasti grafu, aniž by zahrnovala značky os a popisky os. |
| Outer | `1` | Určuje, že velikost oblasti grafu stanoví velikost oblasti grafu, značky os a popisky os. |

## IChartSeries.IsColorVaried vlastnost

Určuje, že každý datový marker v sérii má jinou barvu. Tato vlastnost není jen pro tuto sérii, ale pro všechny série nadřazené skupiny sérií – jedná se o projekci příslušné vlastnosti skupiny. Proto je tato vlastnost pouze pro čtení. Použijte vlastnost ParentSeriesGroup pro přístup k nadřazené skupině sérií. Použijte vlastnost ParentSeriesGroup.IsColorVaried pro čtení a zápis pro změnu hodnoty. Pouze pro čtení Boolean.

```csharp
public bool IsColorVaried { get; }
```

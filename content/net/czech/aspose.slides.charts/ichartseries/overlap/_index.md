## IChartSeries.Overlap vlastnost

Specifikuje, jak moc se pruhy a sloupce překrývají v 2-D grafech, jako procento (od -100 % do 100 %). Toto není vlastnost jen tohoto řady, ale všech řad nadřazené skupiny řad. Jedná se o projekci odpovídající vlastnosti v nadřazené skupině řad, a proto je tato vlastnost jen pro čtení. Pro změnu hodnoty použijte vlastnost ParentSeriesGroup.Overlap pro čtení/zápis. Pouze pro čtení SByte.

```csharp
public sbyte Overlap { get; }
```

### Poznámky

Overlap určuje míru překrytí nebo mezery mezi pruhy a sloupci jako procento jejich šířky: - -100 %: Maximální mezera (pruhy jsou zcela odděleny). - 0 %: Pruhy jsou umístěny vedle sebe bez překrytí nebo mezery. - 100 %: Maximální překrytí (pruhy se zcela překrývají). Jedná se o projekci vlastnosti ParentSeriesGroup.Overlap.

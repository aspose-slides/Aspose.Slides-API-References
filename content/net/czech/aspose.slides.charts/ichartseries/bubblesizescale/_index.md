## IChartSeries.BubbleSizeScale vlastnost

Určuje měřítkový faktor pro bublinový graf (může být mezi 0 a 300 procenty výchozí velikosti). Toto není vlastnost jen této řady, ale všech řad nadřazené skupiny řad – jedná se o projekci odpovídající skupinové vlastnosti. A tak je tato vlastnost pouze pro čtení. Použijte vlastnost ParentSeriesGroup pro přístup k nadřazené skupině řad. Použijte vlastnost ParentSeriesGroup.BubbleSizeScale pro čtení a zápis k změně hodnoty.

```csharp
public int BubbleSizeScale { get; }
```

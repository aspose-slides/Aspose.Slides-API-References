## ChartSeries.IsColorVaried vlastnost

Určuje, že každý datový marker v sérii má jinou barvu. Tato vlastnost platí nejen pro tuto sérii, ale i pro všechny série nadřazené skupiny sérií – jedná se o projekci příslušné skupinové vlastnosti. Proto je tato vlastnost pouze pro čtení. Pro přístup k nadřazené skupině sérií použijte vlastnost ParentSeriesGroup. Pro změnu hodnoty použijte vlastnost ParentSeriesGroup.IsColorVaried s přístupem čtení a zápisu. Pouze pro čtení Boolean.

```csharp
public bool IsColorVaried { get; }
```

## ChartSeries.GapWidth vlastnost

Určuje prostor mezi shluky pruhů nebo sloupců jako procento šířky pruhu nebo sloupce. Tato vlastnost není jen pro tuto řadu, ale pro všechny řady nadřazené skupiny řad – jedná se o projekci odpovídající vlastnosti skupiny. Proto je tato vlastnost pouze pro čtení. Použijte vlastnost ParentSeriesGroup pro přístup k nadřazené skupině řad. Použijte vlastnost ParentSeriesGroup.GapWidth čtení/zápis pro změnu hodnoty. Pouze pro čtení Int32.

```csharp
public int GapWidth { get; }
```

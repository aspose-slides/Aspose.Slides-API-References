## ChartSeries.Overlap vlastnost

Určuje, jak moc se pruhy a sloupce překrývají v 2-D grafech, jako procento (od -100 % do 100 %). Jedná se o vlastnost nejen tohoto řady, ale o všechny řady v nadřazené skupině řad. Je to projekce odpovídající vlastnosti v nadřazené skupině řad, a proto je tato vlastnost jen pro čtení. Hodnotu lze změnit pomocí vlastnosti !:ParentSeriesGroup.Overlap, která má režim čtení/zápisu. Pouze pro čtení SByte.

```csharp
public sbyte Overlap { get; }
```

### Poznámky

Překrytí určuje míru překrytí nebo mezery mezi pruhy a sloupci jako procento jejich šířky: - -100 %: Maximální mezera (pruhy jsou zcela odděleny). - 0 %: Pruhy jsou umístěny vedle sebe bez překrytí či mezery. - 100 %: Maximální překrytí (pruhy se úplně překrývají). Jedná se o projekci vlastnosti !:ParentSeriesGroup.Overlap.

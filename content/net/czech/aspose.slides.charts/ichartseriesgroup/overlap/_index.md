## IChartSeriesGroup.Overlap vlastnost

Určuje, jak moc se mají sloupce a pruhy překrývat na 2-D grafech, jako procento (od –100 % do 100 %). – –100 %: Maximální mezera (pruhy jsou zcela odděleny). – 0 %: Pruhy jsou umístěny vedle sebe bez překrytí i mezery. – 100 %: Maximální překrytí (pruhy se zcela překrývají). Tato vlastnost má čtení/zápis a je typu SByte.

```csharp
public sbyte Overlap { get; set; }
```

### Výjimky

| výjimka | podmínka |
| --- | --- |
| ArgumentOutOfRangeException | Vyvolána, když je hodnota nastavena mimo platný rozsah -100 až 100. |

### Příklady

Následující příklad ukazuje, jak nastavit překrytí pro skupinu řad grafu a vykreslit výsledný graf ve formuláři:

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // Nastavte překrytí na 55%

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

## ChartCategory.UseCell vlastnost

If true then AsCell vlastnost is actual. In other words, worksheet is used for storing category (this case supports a multi-level category). If false then AsLiteral vlastnost is actual. In other words, worksheet is NOT used for storing category (and this case doesn't support a multi-level categories). Pouze pro čtení Boolean.

```csharp
public bool UseCell { get; }
```

### Poznámky

Pro změnu hodnoty této vlastnosti (pro všechny kategorie ve sbírce) nastavte novou hodnotu vlastnosti ChartCategoryCollection.UseCells.

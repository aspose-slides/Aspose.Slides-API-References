## IChartData.SecondaryCategories vlastnost

Získá sekundární kategorie, pokud je vlastnost [`UseSecondaryCategories`](../usesecondarycategories) nastavena na true. Pouze pro čtení [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### Poznámky

Pokud je vlastnost [`UseSecondaryCategories`](../usesecondarycategories) nastavená na false, pak tato vlastnost `SecondaryCategories` vrátí null a data ve vlastnosti [`Categories`](../categories) se používají jak pro primární, tak pro sekundární řady. Pokud je vlastnost [`UseSecondaryCategories`](../usesecondarycategories) nastavena na true, pak data v této vlastnosti `SecondaryCategories` se používají pro sekundární řady a data ve vlastnosti [`Categories`](../categories) se používají pro primární řady.

### Příklady

Příklad. Které kategorie jsou přiřazeny k řadám – ChartData.Categories nebo ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // související kategorie jsou series.Chart.ChartData.SecondaryCategories
}
else
{
    // související kategorie jsou series.Chart.ChartData.Categories
}
```

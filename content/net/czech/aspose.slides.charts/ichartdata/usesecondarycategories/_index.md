## IChartData.UseSecondaryCategories vlastnost

Pokud je false, pak [`SecondaryCategories`](../secondarycategories) vlastnost vrátí null a data v [`Categories`](../categories) vlastnost se používají jak pro primární, tak pro sekundární řady. Pokud je true, pak data v [`SecondaryCategories`](../secondarycategories) vlastnost se používají pro sekundární řady a data v [`Categories`](../categories) vlastnost se používají pro primární řady. Čtení/Zápis Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

### Příklady

Příklad. Jaké kategorie jsou související s řadou – ChartData.Categories nebo ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // související kategorie jsou series.Chart.ChartData.SecondaryCategories
}
else
{
    // související kategorie jsou series.Chart.ChartData.Categories
}
```

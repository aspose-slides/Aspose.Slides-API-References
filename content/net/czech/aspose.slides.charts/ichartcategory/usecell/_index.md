## IChartCategory.UseCell vlastnost

Pokud je true, pak je vlastnost AsCell aktuální. Jinými slovy, list se používá k ukládání kategorie (tento případ podporuje víceúrovňovou kategorii). Pokud je false, pak je vlastnost AsLiteral aktuální. Jinými slovy, list se NEpoužívá k ukládání kategorie (a tento případ nepodporuje víceúrovňové kategorie). Pouze pro čtení Boolean.

```csharp
public bool UseCell { get; }
```

### Poznámky

Pro změnu hodnoty této vlastnosti (pro všechny kategorie ve sbírce) nastavte novou hodnotu vlastnosti ChartCategoryCollection.UseCells.

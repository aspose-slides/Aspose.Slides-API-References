## DataLabel.IsVisible vlastnost

False znamená, že popisek dat není viditelný (a tedy všechny příznaky Show*-flags (ShowValue, ...) jsou false). Pouze ke čtení Boolean.

```csharp
public bool IsVisible { get; }
```

### Poznámky

Pokud je popisek dat viditelný, můžete jej skrýt pomocí metody Hide(). Pokud však popisek dat není viditelný (IsVisible je false), můžete popisek dat zobrazit nastavením příznaků Show*-flags (ShowValue, ...) do stavu true.

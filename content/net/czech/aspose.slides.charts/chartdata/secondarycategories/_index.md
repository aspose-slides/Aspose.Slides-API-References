## ChartData.SecondaryCategories vlastnost

Získá sekundární kategorie, pokud je vlastnost [`UseSecondaryCategories`](../usesecondarycategories) nastavena na true. Pouze pro čtení [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### Poznámky

Pokud je vlastnost [`UseSecondaryCategories`](../usesecondarycategories) nastavena na false, pak tato `SecondaryCategories` vlastnost vrací null a data ve vlastnosti [`Categories`](../categories) jsou použita jak pro primární, tak pro sekundární řady. Pokud je vlastnost [`UseSecondaryCategories`](../usesecondarycategories) nastavena na true, pak data v této `SecondaryCategories` vlastnosti jsou použita pro sekundární řady a data ve vlastnosti [`Categories`](../categories) jsou použita pro primární řady.

### Příklady

Příklad. Které kategorie jsou související se sériemi – ChartData.Categories nebo ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // související kategorie jsou series.Chart.ChartData.SecondaryCategories
}
else
{
    // související kategorie jsou series.Chart.ChartData.Categories
}
```

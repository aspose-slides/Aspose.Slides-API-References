## ChartData.Categories property

Získá primární kategorie (nebo jak primární, tak sekundární kategorie, pokud je [`UseSecondaryCategories`](../usesecondarycategories) vlastnost false). Pouze pro čtení [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection Categories { get; }
```

### Poznámky

Pokud je [`UseSecondaryCategories`](../usesecondarycategories) vlastnost false, pak [`SecondaryCategories`](../secondarycategories) vlastnost vrátí null a data v této `Categories` vlastnosti jsou použita jak pro primární, tak pro sekundární sérii. Pokud je [`UseSecondaryCategories`](../usesecondarycategories) vlastnost true, pak data v [`SecondaryCategories`](../secondarycategories) vlastnosti jsou použita pro sekundární sérii a data v této `Categories` vlastnosti jsou použita pro primární sérii.

### Příklady

Příklad. Jaké kategorie jsou přiřazeny k sériím – ChartData.Categories nebo ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // související kategorie jsou series.Chart.ChartData.SecondaryCategories
}
else
{
    // související kategorie jsou series.Chart.ChartData.Categories
}
```

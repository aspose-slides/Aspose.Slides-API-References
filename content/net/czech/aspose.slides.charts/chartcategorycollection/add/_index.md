## Add(object) {#add_1}

Vytvoří nový [`ChartCategory`](../../chartcategory) z hodnoty a přidá jej do kolekce.

```csharp
public IChartCategory Add(object value)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| value | Object | Hodnota. |

### Návratová hodnota

Přidáno [`IChartCategory`](../../ichartcategory).

### Výjimky

| výjimka | podmínka |
| --- | --- |
| InvalidOperationException | pokud je překročeno omezení |

### Poznámky

Tato metoda přidá list s názvem AUTO_DATA a přidá do něj všechny hodnoty. Pokud používáte [`ChartDataWorkbook`](../../chartdataworkbook) k přidání nebo úpravě hodnot buněk, ujistěte se, že tento list nepoužíváte. Maximální počet hodnot přidaných touto metodou nesmí překročit 16711680.

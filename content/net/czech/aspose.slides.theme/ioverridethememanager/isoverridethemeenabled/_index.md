## Vlastnost IOverrideThemeManager.IsOverrideThemeEnabled

Určuje, zda OverrideTheme přepisuje zděděné efektivní téma, nebo ne. Pro povolení OverrideTheme k přepisování použijte metody OverrideTheme.Init*(). Pro zakázání přepisování OverrideTheme použijte metodu OverrideTheme.Clear(). Pouze ke čtení Boolean.

```csharp
public bool IsOverrideThemeEnabled { get; }
```

## BehaviorPropertyCollection.CopyTo metoda

Kopíruje prvky rozhraní ICollection do pole Array, počínaje konkrétním indexem pole.

```csharp
public void CopyTo(IBehaviorProperty[] array, int arrayIndex)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| array | IBehaviorProperty[] | Jednorozměrné pole Array, které je cílem prvků kopírovaných z ICollection. Pole Array musí mít nulové indexování. |
| arrayIndex | Int32 | Nulový index v *array*, od kterého začíná kopírování. |

### Výjimky

| výjimka | podmínka |
| --- | --- |
| ArgumentNullException | *array* je null. |
| ArgumentOutOfRangeException | *arrayIndex* je menší než 0. |
| ArgumentException | Počet prvků ve zdrojovém rozhraní ICollection je větší než dostupný prostor od *arrayIndex* až po konec cílového *array*. |

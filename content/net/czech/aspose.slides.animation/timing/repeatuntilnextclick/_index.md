## Timing.RepeatUntilNextClick vlastnost

Tento atribut určuje, zda se efekt opakuje až do dalšího kliknutí. Číst/Zapisovat Boolean.

```csharp
public bool RepeatUntilNextClick { get; set; }
```

### Příklady

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Získá sekvenci efektů pro první snímek
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Získá první efekt hlavní sekvence.
    IEffect effect = effectsSequence[0];

    // Změní nastavení Timing/Repeat efektu na "Do dalšího kliknutí"

}
```

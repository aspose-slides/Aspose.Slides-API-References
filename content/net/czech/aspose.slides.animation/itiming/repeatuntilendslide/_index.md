## ITiming.RepeatUntilEndSlide vlastnost

Tento atribut určuje, zda se efekt bude opakovat až do konce snímku. Čtení/Zápis Boolean.

```csharp
public bool RepeatUntilEndSlide { get; set; }
```

### Příklady

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Získá sekvenci efektů pro první snímek
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Získá první efekt hlavní sekvence.
    IEffect effect = effectsSequence[0];

    // Změní časování/opakování efektu na "Until End of Slide"
    effect.Timing.RepeatUntilEndSlide = true;
}
```

## ITiming.RepeatUntilNextClick vlastnost

Tento atribut určuje, zda se efekt bude opakovat až do dalšího kliknutí. Čtení/Zápis Boolean.

```csharp
public bool RepeatUntilNextClick { get; set; }
```

### Příklady

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Získá sekvenci efektů pro první snímek
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Získá první efekt hlavní sekvence.
    IEffect effect = effectsSequence[0];

    // Změní Timing/Repeat efektu na "Until Next Click"
    effect.Timing.RepeatUntilNextClick = true;
}
```

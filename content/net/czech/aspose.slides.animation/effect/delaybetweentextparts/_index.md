## Effect.DelayBetweenTextParts vlastnost

Definuje prodlevu mezi animovanými částmi textu (slovy nebo písmeny). Kladná hodnota určuje procento trvání efektu. Záporná hodnota určuje prodlevu v sekundách. Číst/Zapisovat Single.

```csharp
public float DelayBetweenTextParts { get; set; }
```

### Příklady

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Získá první efekt první snímky.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Změní typ animace textu na „By word“
    firstSlideEffect.AnimateTextType = AnimateTextType.ByWord;
    
    // Nastaví prodlevu mezi animovanými částmi textu na 20% trvání efektu.
    firstSlideEffect.DelayBetweenTextParts = 20f;
}
```

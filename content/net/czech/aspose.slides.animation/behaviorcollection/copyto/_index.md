## BehaviorCollection.CopyTo metoda

Kopíruje prvky ICollection do pole Array, počínaje konkrétním indexem pole Array.

```csharp
public void CopyTo(IBehavior[] array, int arrayIndex)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| array | IBehavior[] | Jednorozměrné pole Array, které je cílem prvků zkopírovaných z ICollection. Pole Array musí mít indexování od nuly. |
| arrayIndex | Int32 | Nulový index v *array*, kde začíná kopírování. |

### Výjimky

| výjimka | podmínka |
| --- | --- |
| ArgumentNullException | *array* je null. |
| ArgumentOutOfRangeException | *arrayIndex* je menší než 0. |
| ArgumentException | Počet prvků ve zdrojovém ICollection je větší než dostupné místo od *arrayIndex* do konce cílového *array*. |

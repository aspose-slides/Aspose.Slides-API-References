## IEffect.DelayBetweenTextParts vlastnost

Definuje zpoždění mezi animovanými částmi textu (slova nebo písmena). Kladná hodnota určuje procento trvání efektu. Záporná hodnota určuje zpoždění v sekundách. Čtení/Zápis Single.

```csharp
public float DelayBetweenTextParts { get; set; }
```

### Příklady

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Získá první efekt z první snímku.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Změní typ animace textu efektu na "By word"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByWord;
    
    // Nastaví zpoždění mezi animovanými částmi textu na 20 % trvání efektu.
    firstSlideEffect.DelayBetweenTextParts = 20f;
}
```

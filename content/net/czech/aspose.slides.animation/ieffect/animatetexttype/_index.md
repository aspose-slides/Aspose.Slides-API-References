## IEffect.AnimateTextType vlastnost

Definuje typ animovaného textu pro efekt. Text tvaru může být animován po písmenu, po slově nebo najednou. Čtení/zápis `AnimateTextType`.

```csharp
public AnimateTextType AnimateTextType { get; set; }
```

### Příklady

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Získá první efekt prvního snímku.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Změní typ animovaného textu efektu na "By letter"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByLetter;
}
```

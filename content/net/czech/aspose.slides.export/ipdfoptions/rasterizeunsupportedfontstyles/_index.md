## IPdfOptions.RasterizeUnsupportedFontStyles vlastnost

Určuje, zda by měl být text rastrován jako bitmapa a uložen do PDF, pokud písmo nepodporuje tučné formátování. Tento přístup může zlepšit kvalitu textu ve výsledném PDF u některých písem. Čtení/zápis Boolean.

```csharp
public bool RasterizeUnsupportedFontStyles { get; set; }
```

### Poznámky

Výchozí hodnota je **false**.

## MarkdownSaveOptions.SlideNumberFormat vlastnost

Získává nebo nastavuje řetězec formátu používaný pro záhlaví číslování snímků v Markdown výstupu. Formát musí obsahovat zástupný znak "{0}", který bude během exportu nahrazen indexem snímku. Příklad: "# Slide {0}" vygeneruje "# Slide 1", "# Slide 2", atd.

```csharp
public string SlideNumberFormat { get; set; }
```

### Výjimky

| výjimka | podmínka |
| --- | --- |
| ArgumentNullException | Vyvolána, pokud je poskytnutá hodnota null nebo prázdná. |
| ArgumentException | Vyvolána, pokud řetězec formátu neobsahuje zástupný znak "{0}". |

## IPptxOptions.CompressionLevel vlastnost

Určuje úroveň komprese používanou při ukládání dokumentu prezentace. Výchozí hodnota je Level6.

```csharp
public CompressionLevel CompressionLevel { get; set; }
```

### Poznámky

Vyšší úrovně komprese produkují menší soubory, ale vyžadují více času na zpracování. Skutečný kompresní poměr závisí na obsahu prezentace.

### Příklady

Příklad:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("demo-level8.pptx", SaveFormat.Pptx, new PptxOptions()
    {
        CompressionLevel = CompressionLevel.Level8
    });
}
```

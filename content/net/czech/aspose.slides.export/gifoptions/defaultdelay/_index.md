## GifOptions.DefaultDelay vlastnost

Získá nebo nastaví výchozí dobu zpoždění [ms]. Tato hodnota bude použita, pokud není [`AdvanceAfterTime`](../../../aspose.slides/islideshowtransition/advanceaftertime) nastavena. Výchozí hodnota je 1000.

```csharp
public int DefaultDelay { get; set; }
```

### Příklady

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
	pres.Save("pres.gif", SaveFormat.Gif, new GifOptions { DefaultDelay = 2000 });
}
```

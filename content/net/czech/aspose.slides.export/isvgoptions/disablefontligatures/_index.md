## ISVGOptions.DisableFontLigatures vlastnost

Získá nebo nastaví hodnotu, která určuje, zda je text vykreslován bez použití ligatur. Pokud je nastavena na `true`, budou ligatury v renderovaném výstupu zakázány. Ve výchozím nastavení je tato vlastnost nastavena na `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Příklady

Příklad:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    SVGOptions options = new SVFOptions
    {
        DisableFontLigatures = true // Zakázat ligatury při vykreslování textu
    };
    
    using (FileStream fileStream = new FileStream("slide-0.svg", FileMode.Create, FileAccess.Write))
    {
        pres.Slides[0].WriteAsSvg(fileStream);   
    }
}
```

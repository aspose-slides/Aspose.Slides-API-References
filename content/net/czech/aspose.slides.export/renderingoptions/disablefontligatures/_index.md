## RenderingOptions.DisableFontLigatures vlastnost

Získá nebo nastaví hodnotu určující, zda je text vykreslován bez použití ligatur. Pokud je nastaveno na `true`, ligatury budou vygenerovaném výstupu zakázány. Ve výchozím nastavení je tato vlastnost nastavena na `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Příklady

Příklad:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    RenderingOptions options = new RenderingOptions
    {
        DisableFontLigatures = true // Zakázat ligatury při vykreslování textu
    };
    
    Bitmap[] renderedSlides = pres.GetImage(options);
    for (var index = 0; index < renderedSlides.Length; index++)
    {
        var slideImage = renderedSlides[index];
        slideImage.Save($"slide-{index}.png");
    }
}
```

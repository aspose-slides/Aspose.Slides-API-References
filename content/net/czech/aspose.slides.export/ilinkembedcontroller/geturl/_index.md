## ILinkEmbedController.GetUrl metoda

Vrací URL na externí objekt. Tato metoda je vždy volána, pokud [`GetObjectStoringLocation`](../getobjectstoringlocation) vrátil Link a může být volána, pokud [`GetObjectStoringLocation`](../getobjectstoringlocation) vrátil Embed, ale vložení není možné. Může být volána vícekrát pro stejný identifikátor objektu.

```csharp
public string GetUrl(int id, int referrer)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| id | Int32 | Identifikátor objektu. Tento identifikátor je během operace jedinečný. |
| referrer | Int32 | Identifikátor odkazujícího objektu nebo 0, pokud je objekt odkazován kořenovým dokumentem. Může být použito k vytvoření relativního odkazu. |

### Návratová hodnota

URL externího objektu nebo null, pokud má být tento objekt ignorován.

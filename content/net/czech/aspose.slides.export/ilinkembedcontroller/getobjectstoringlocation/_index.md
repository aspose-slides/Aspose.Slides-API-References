## ILinkEmbedController.GetObjectStoringLocation metoda

Určuje, kde má být objekt uložen. Tato metoda je volána jednou pro každé ID objektu. Není zaručeno, že nebudou existovat dva objekty se stejnými daty, semanticName a contentType, ale s odlišným ID.

```csharp
public LinkEmbedDecision GetObjectStoringLocation(int id, byte[] entityData, string semanticName, 
    string contentType, string recomendedExtension)
```

| Parametr | Typ | Popis |
| --- | --- | --- |
| id | Int32 | ID objektu. Toto ID je jedinečné v rámci celé operace ukládání. |
| entityData | Byte[] | Binární data objektu. Tento parametr může být null, pokud binární data objektu ještě nebyla vygenerována. |
| semanticName | String | Krátký text popisující význam objektu. Kontrolér může toto použít jako součást externího názvu objektu, ale je na dispečerovi, aby zajistil, že názvy budou jedinečné a budou obsahovat pouze povolené znaky. |
| contentType | String | MIME typ objektu. |
| recomendedExtension | String | Přípona souboru doporučená pro tento MIME typ. |

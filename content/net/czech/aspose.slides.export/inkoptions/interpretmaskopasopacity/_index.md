## InkOptions.InterpretMaskOpAsOpacity vlastnost

Používá operaci ROP nebo neprůhlednost pro vykreslování štětce.

```csharp
public bool InterpretMaskOpAsOpacity { get; set; }
```

### Poznámky

Výchozí hodnota je true.

### Příklady

Následující příklad ukazuje, jak nastavit pomocí ROP pro exportování Ink prvků:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.InterpretMaskOpAsOpacity = false;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

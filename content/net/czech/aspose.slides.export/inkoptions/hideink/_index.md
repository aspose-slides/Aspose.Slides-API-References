## InkOptions.HideInk vlastnost

Zobrazuje nebo skrývá Ink prvky v exportovaném dokumentu.

```csharp
public bool HideInk { get; set; }
```

### Poznámky

Výchozí hodnota je false.

### Příklady

Následující příklad ukazuje, jak skrýt Ink prvky v exportovaném PDF dokumentu:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.HideInk = true;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

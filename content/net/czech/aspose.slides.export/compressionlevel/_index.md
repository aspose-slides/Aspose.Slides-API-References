## CompressionLevel enumerace

Specifikuje úrovně komprese ZIP pro soubor OpenXML. Vyšší úrovně poskytují lepší kompresi za cenu pomalejšího zpracování.

```csharp
public enum CompressionLevel
```

### Hodnoty

| Název | Hodnota | Popis |
| --- | --- | --- |
| None | `0` | Komprese se nepoužije. Soubory jsou uloženy v původní podobě. |
| Level1 | `1` | Nejrychlejší komprese s nejnižším kompresním poměrem. |
| Level2 | `2` | Rychlejší komprese se mírně lepším kompresním poměrem než Level1. |
| Level3 | `3` | Poskytuje lepší kompresi než Level2 s mírně sníženým výkonem. |
| Level4 | `4` | Poskytuje lepší kompresi než Level3. |
| Level5 | `5` | Poskytuje vylepšenou kompresi oproti Level4 s dodatečným časem zpracování. |
| Level6 | `6` | Standardní komprese, nabízející dobrý poměr mezi rychlostí komprese a velikostí souboru. Výchozí úroveň komprese. |
| Level7 | `7` | Poskytuje vyšší kompresi než Level6 při pomalejším zpracování. |
| Level8 | `8` | Poskytuje vyšší kompresi než Level7. |
| Level9 | `9` | Maximální komprese. Produkuje nejmenší velikost souboru při nejpomalejším zpracování. |

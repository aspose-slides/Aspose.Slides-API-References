## enumerace CompressionLevel

Určuje úrovně komprese ZIP pro soubor OpenXML. Vyšší úrovně poskytují lepší kompresi za cenu pomalejšího zpracování.

```csharp
public enum CompressionLevel
```

### Hodnoty

| Název | Hodnota | Popis |
| --- | --- | --- |
| None | `0` | Žádná komprese není použita. Soubory jsou uloženy v původním stavu. |
| Level1 | `1` | Nejschůdnější komprese s nejnižším kompresním poměrem. |
| Level2 | `2` | Rychlejší komprese s mírně lepším kompresním poměrem než Level1. |
| Level3 | `3` | Poskytuje lepší kompresi než Level2 s mírnějším dopadem na výkon. |
| Level4 | `4` | Poskytuje lepší kompresi než Level3. |
| Level5 | `5` | Poskytuje vylepšenou kompresi oproti Level4 s dalším časem zpracování. |
| Level6 | `6` | Standardní komprese, nabízející dobrý poměr mezi rychlostí komprese a velikostí souboru. Výchozí úroveň komprese. |
| Level7 | `7` | Poskytuje vyšší kompresi než Level6 při pomalejším zpracování. |
| Level8 | `8` | Poskytuje vyšší kompresi než Level7. |
| Level9 | `9` | Maximální komprese. Produkuje nejmenší velikost souboru při nejpomalejším zpracování. |

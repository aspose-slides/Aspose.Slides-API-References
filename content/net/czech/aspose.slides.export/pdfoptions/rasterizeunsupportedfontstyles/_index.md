## PdfOptions.RasterizeUnsupportedFontStyles vlastnost

Určuje, zda má být text rasterizován jako bitmapa a uložen do PDF, když font nepodporuje tučné písmo. Tento přístup může zlepšit kvalitu textu ve výsledném PDF u určitých fontů. Čtení/zápis Boolean.

```csharp
public bool RasterizeUnsupportedFontStyles { get; set; }
```

### Poznámky

Výchozí hodnota je **false**.

## PdfOptions.EmbedTrueTypeFontsForASCII vlastnost

Určuje, zda Aspose.Slides vloží běžná písma pro text ASCII (rozsah kódů 33..127). Písma pro znaky s kódy vyššími než 127 jsou vždy vložena. Seznam běžných písem zahrnuje základních 14 písem PDF a další uživatelem určená písma. Čtení/zápis Boolean.

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### Poznámky

Výchozí hodnota je **true**.

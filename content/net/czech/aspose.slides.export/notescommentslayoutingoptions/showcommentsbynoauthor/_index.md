## NotesCommentsLayoutingOptions.ShowCommentsByNoAuthor vlastnost

Získává nebo nastavuje viditelnost komentářů, které nemají autora. Pokud je true, komentáře budou zobrazeny. (Platí jen pokud jsou komentáře zobrazeny).

```csharp
public bool ShowCommentsByNoAuthor { get; set; }
```

### Poznámky

Výchozí hodnota je **false**.

## ISaveOptions.SkipJavaScriptLinks vlastnost

Určuje, zda při ukládání prezentace přeskočit hyperlinky s voláním JavaScriptu. Čtení/Zápis Boolean. Výchozí hodnota je **false**.

```csharp
public bool SkipJavaScriptLinks { get; set; }
```

### Poznámky

Když je tato vlastnost nastavena na **true**, budou při ukládání hyperlanky s voláním JavaScriptu ignorovány.

Když je tato vlastnost nastavena na **false**, všechny hyperlanky budou uloženy.

### Příklady

Příklad:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("result_without_JavaScript_links.html", SaveFormat.Html, new HtmlOptions()
    {
        SkipJavaScriptLinks = true
    });
}
```

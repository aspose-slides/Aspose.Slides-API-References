```csharp
public ReadOnlyCollection<IExcelDataCell> GetCells(string formula, bool skipHiddenCells)
```

| المعامل | النوع | الوصف |
| --- | --- | --- |
| formula | String | صيغة أو تعبير نطاق (مثل "Sheet1!A1:B3") يُستخدم لتحديد الخلايا المستهدفة. |
| skipHiddenCells | Boolean | إذا `true`، سيتم استبعاد الخلايا المخفية (مثل الخلايا في الصفوف أو الأعمدة المخفية) من النتيجة. |

### قيمة الإرجاع

### أمثلة

مثال:

```csharp
[C#]
ExcelDataWorkbook wb = new ExcelDataWorkbook(testFile);
IReadOnlyList<IExcelDataCell> cells = wb.GetCells("Sheet1!A2:A6", false);
Console.WriteLine(cells.Count); //الإخراج: 5
```

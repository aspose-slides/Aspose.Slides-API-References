## MathDelimiter.GrowToMatchOperandHeight خاصية

يحدد نمو BeginningCharacter و SeparatorCharacter و EndingCharacter. عندما تكون true، تنمو الفواصل عموديًا لتطابق ارتفاع المعامل. القيمة الافتراضية هي true

```csharp
public bool GrowToMatchOperandHeight { get; set; }
```

### أمثلة

مثال:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.GrowToMatchOperandHeight = false;
```

## MathDelimiter.DelimiterShape خاصية

تحدد شكل الفواصل في كائن الفاصل. عندما يكون MathDelimiterShape.Centered، يتم توسيط الفواصل حول محور الرياضيات للنص الرياضي وتظل مُصممة لتتناسب مع الارتفاع الكامل لمحتوياتها. عندما يكون MathDelimiterShape.Match، يتم تعديل ارتفاعها وشكلها لتطابق محتوياتها تمامًا.

```csharp
public MathDelimiterShape DelimiterShape { get; set; }
```

### أمثلة

مثال:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.DelimiterShape = MathDelimiterShape.Match;
```

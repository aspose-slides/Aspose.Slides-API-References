## IMathBox.ExplicitBreak خاصية

Explicit break يحدد ما إذا كان هناك فاصل سطر في بداية كائن Box، بحيث يلتف السطر عند بداية كائن Box. يحدد عدد المشغل في السطر السابق للنص الرياضي الذي سيُستخدم كنقطة محاذاة للسطر الحالي من النص الرياضي. القيم الممكنة: 1..255 الافتراضي: 0 (لا فاصل صريح)

```csharp
public byte ExplicitBreak { get; set; }
```

### أمثلة

مثال:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.ExplicitBreak = 1;
```

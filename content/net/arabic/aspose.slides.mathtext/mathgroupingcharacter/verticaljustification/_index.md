## MathGroupingCharacter.VerticalJustification الخاصية

محاذاة عمودية لحرف المجموعة. يحدد محاذاة الكائن بالنسبة إلى خط الأساس. على سبيل المثال، عندما يكون حرف المجموعة فوق الكائن، يشير VerticalJustification بقيمة Top إلى أن أعلى الكائن يقع على خط الأساس؛ عندما يتم تعيين VerticalJustification إلى Bottom، يكون أسفل الكائن على خط الأساس. الافتراضي: Bottom لـ Position=Top، و Top لـ Position=Bottom

```csharp
public MathTopBotPositions VerticalJustification { get; set; }
```

### أمثلة

مثال:

```csharp
[C#]
MathGroupingCharacter groupingCharacter = new MathGroupingCharacter(new MathematicalText("abc"));
groupingCharacter.VerticalJustification = MathTopBotPositions.Top;
```

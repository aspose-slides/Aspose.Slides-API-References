## IMathDelimiter.DelimiterShape خاصية

يحدد شكل الفواصل في كائن الفاصل. عندما تكون MathDelimiterShape.Centered، تكون الفواصل متمركزة حول محور الرياضيات للنص الرياضي ولا تزال تُصمم لتتناسب مع الارتفاع الكامل لمحتوياتها. عندما تكون MathDelimiterShape.Match، يتم تعديل ارتفاعها وشكلها لتطابق تمامًا محتوياتها.

```csharp
public MathDelimiterShape DelimiterShape { get; set; }
```

### أمثلة

مثال:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.DelimiterShape = MathDelimiterShape.Match;
```

## MathPhantom.Transp خاصية

يحصل على أو يضبط قيمة تشير إلى ما إذا كان الفانتوم شفافًا لقواعد التباعد المستندة إلى الفئة.

```csharp
public bool Transp { get; set; }
```

### ملاحظات

عند `true`، لا يزال المشغلات والرموز داخل الفانتوم تؤثر على التباعد الرياضي حول الفانتوم (كما لو كان مرئيًا). عند `false`، يتم تجاهل التباعد المستند إلى الفئة. يتطابق مع سمة OMML `m:transp`.

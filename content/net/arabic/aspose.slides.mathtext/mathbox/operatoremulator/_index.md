## خاصية MathBox.OperatorEmulator

محاكي المشغل. عندما تكون true، الصندوق ومحتوياته يتصرفان كمشغل واحد وتورثان خصائص المشغل. هذا يعني، على سبيل المثال، أن الحرف يمكن أن يكون نقطة لفاصل سطر ويمكن محاذاته إلى المشغلات الأخرى. غالبًا ما تُستخدم محاكيات المشغل عندما يجتمع حرف أو أكثر لتكوين مشغل، مثل '=='. القيمة الافتراضية: false

```csharp
public bool OperatorEmulator { get; set; }
```

### أمثلة

مثال:

[C#]
MathBox box = new MathBox(new MathematicalText("=="));
box.OperatorEmulator = true;

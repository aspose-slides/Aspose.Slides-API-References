## MathBox.NoBreak خاصية

بدون فاصل هذه الخاصية تحدد خاصية "غير قابل للكسر" على صندوق الكائن. عندما تكون true، لا يمكن حدوث فواصل أسطر داخل الصندوق. قد يكون هذا مهمًا لمحاكيات المشغل التي تتكون من أكثر من مشغل ثنائي. عندما لا يتم تحديد هذا العنصر، يمكن حدوث فواصل داخل الصندوق. الافتراضي: true

```csharp
public bool NoBreak { get; set; }
```

### أمثلة

مثال:

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("*****"));
box.NoBreak = false;
```

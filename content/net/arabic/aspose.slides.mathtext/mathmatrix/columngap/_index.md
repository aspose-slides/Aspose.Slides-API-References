## MathMatrix.ColumnGap خاصية

قيمة التباعد الأفقي بين أعمدة المصفوفة؛ إذا تم تعيين ColumnGapRule إلى 3 ("Exactly")، فسيتم تفسير الوحدة على أنها twips (1/20 من النقطة). إذا تم تعيين ColumnGapRule إلى 4 ("Multiple")، فسيتم تفسير الوحدة على أنها عدد من الزيادات 0.5 em. في الحالات الأخرى يتم تجاهلها. الافتراضي: 0

```csharp
public uint ColumnGap { get; set; }
```

### أمثلة

مثال:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.ColumnGapRule = MathSpacingRules.Exactly;
matrix.ColumnGap = 20;
```

## IMathMatrix.RowGap الخاصية

القيمة التي تمثل المسافة العمودية بين صفوف المصفوفة؛ إذا تم ضبط RowGapRule على 3 ("Exactly")، فسيتم تفسير الوحدة كـ twips (1/20 من نقطة). إذا تم ضبط RowGapRule على 4 ("Multiple")، فسيتم تفسير الوحدة كخطوط نصفية. الافتراضي: 0

```csharp
public uint RowGap { get; set; }
```

### أمثلة

مثال:

```csharp

matrix.RowGapRule = MathSpacingRules.Exactly;
```

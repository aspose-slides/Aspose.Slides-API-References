## IMathMatrix.ColumnGap الخاصية

قيمة التباعد الأفقي بين أعمدة المصفوفة؛ إذا تم تعيين ColumnGapRule إلى 3 ("Exactly"), فتُفسَّر الوحدة كـ twips (1/20 من النقطة) إذا تم تعيين ColumnGapRule إلى 4 ("Multiple"), فتُفسَّر الوحدة كعدد من الزيادات 0.5 em. في الحالات الأخرى يتم تجاهلها. الافتراضي: 0

```csharp
public uint ColumnGap { get; set; }
```

### أمثلة

مثال:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.ColumnGapRule = MathSpacingRules.Exactly;
```

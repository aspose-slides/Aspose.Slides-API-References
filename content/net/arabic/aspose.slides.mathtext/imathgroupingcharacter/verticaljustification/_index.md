## IMathGroupingCharacter.VerticalJustification خاصية

محاذاة رأسية لحرف المجموعة. يحدد محاذاة الكائن بالنسبة إلى خط الأساس. على سبيل المثال، عندما يكون حرف المجموعة فوق الكائن، فإن VerticalJustification بقيمة Top يعني أن أعلى الكائن يقع على خط الأساس؛ وعندما يتم تعيين VerticalJustification إلى Bottom، يكون أسفل الكائن على خط الأساس. الافتراضي: Bottom عندما Position=Top، وTop عندما Position=Bottom

```csharp
public MathTopBotPositions VerticalJustification { get; set; }
```

### أمثلة

مثال:

```csharp
[C#]
MathGroupingCharacter groupingCharacter = new MathGroupingCharacter(new MathematicalText("abc"));
groupingCharacter.VerticalJustification = MathTopBotPositions.Top;
```

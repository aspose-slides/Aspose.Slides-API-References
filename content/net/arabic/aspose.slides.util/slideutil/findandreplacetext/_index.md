## SlideUtil.FindAndReplaceText طريقة

يقوم بالبحث واستبدال النص في العرض التقديمي بالتنسيق المحدد

```csharp
public static void FindAndReplaceText(IPresentation presentation, bool withMasters, string find, 
    string replace, PortionFormat format = null)
```

| المعامل | النوع | الوصف |
| --- | --- | --- |
| presentation | IPresentation | العرض الممسوح. |
| withMasters | Boolean | يحدد ما إذا كان يجب مسح الشرائح الرئيسة. |
| find | String | قيمة النص للبحث عنها. |
| replace | String | قيمة النص للاستبدال. |
| format | PortionFormat | التنسيق لاستبدال الجزء النصي. إذا كان null فسيتم استخدام تنسيق الحرف الأول من السلسلة التي تم العثور عليها |

### أمثلة

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PortionFormat format = new PortionFormat
    {
        FontHeight = 24f,
        FontItalic = NullableBool.True,
        FillFormat =
        {
            FillType = FillType.Solid,
            SolidFillColor =
            {
                Color = Color.Red
            }
        }
    };
    Aspose.Slides.Util.SlideUtil.FindAndReplaceText(pres, true, "[this block] ", "my text ", format);
    pres.Save("replaced", SaveFormat.Pptx);
}
```

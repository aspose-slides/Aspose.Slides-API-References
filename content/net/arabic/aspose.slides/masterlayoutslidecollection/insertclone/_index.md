## MasterLayoutSlideCollection.InsertClone طريقة

يقوم بإدراج نسخة من شريحة تخطيط محددة إلى الموضع المحدد في المجموعة.

```csharp
public ILayoutSlide InsertClone(int index, ILayoutSlide sourceLayout)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | Int32 | فهرس الشريحة الجديدة. |
| sourceLayout | ILayoutSlide | الشريحة التي سيتم استنساخها. |

### Return Value

الشريحة التي تم إدراجها.

### Remarks

سيتم ربط التخطيط الجديد بالشريحة الرئيسة الأصلية لهذا التجميع من شرائح التخطيط. لذا فهذا يعادل النسخ/اللصق مع خيار "Use Destination Theme" في PowerPoint.

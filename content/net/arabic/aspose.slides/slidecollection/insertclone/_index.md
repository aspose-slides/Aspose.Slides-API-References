### قيمة الإرجاع

الشريحة التي تم إدراجها.

### ملاحظات

عند استنساخ شريحة بين عروض تقديمية مختلفة يمكن أيضًا استنساخ القالب الرئيسي للشريحة. يُستخدم سجل داخلي لتتبع القوالب الرئيسية المستنسخة تلقائيًا لمنع إنشاء نسخ متعددة من نفس القالب الرئيسي. لن يتم منع أو تسجيل الاستنساخ اليدوي لقوالب الشرائح الرئيسية. إذا كنت بحاجة إلى مزيد من التحكم في عملية الاستنساخ، استخدم [`InsertClone`](../insertclone) أو [`InsertClone`](../insertclone) لاستنساخ الشرائح و[`AddClone`](../../imasterslidecollection/addclone) لاستنساخ القوالب الرئيسية.

### أمثلة

يوضح المثال التالي كيفية استنساخ شريحة في موضع آخر داخل العرض التقديمي.

```csharp
[C#]
// إنشاء كائن من فئة Presentation التي تمثل ملف عرض تقديمي
using (Presentation pres = new Presentation("CloneWithInSamePresentation.pptx"))
{
    // استنساخ الشريحة المطلوبة إلى نهاية مجموعة الشرائح في نفس العرض التقديمي
    ISlideCollection slds = pres.Slides;
    // استنساخ الشريحة المطلوبة إلى الفهرس المحدد في نفس العرض التقديمي
    slds.InsertClone(2, pres.Slides[1]);
    // حفظ العرض التقديمي المعدل إلى القرص
    pres.Save("Aspose_CloneWithInSamePresentation_out.pptx", SaveFormat.Pptx);
}
```

يوضح المثال التالي كيفية استنساخ شريحة في موضع آخر داخل العرض التقديمي.

```csharp
[C#]
// إنشاء كائن من فئة Presentation لتحميل ملف العرض التقديمي المصدر
using (Presentation srcPres = new Presentation("CloneAtEndOfAnother.pptx"))
{
    // إنشاء كائن من فئة Presentation للملف PPTX الوجهة (حيث ستُستنسخ الشريحة)
    using (Presentation destPres = new Presentation())
    {
        ISlideCollection slds = destPres.Slides;
        slds.InsertClone(2, srcPres.Slides[0]);
        // حفظ العرض التقديمي الوجهة إلى القرص
        destPres.Save("Aspose2_out.pptx", SaveFormat.Pptx);
    }
}
```

## InsertClone(int, ISlide, IMasterSlide, bool) {#insertclone_2}

يدرج نسخة من الشريحة المصدر المحددة في الموضع المحدد داخل المجموعة. سيتم اختيار التخطيط المناسب تلقائيًا من القالب المحدد (التخطيط المناسب هو التخطيط الذي يحمل نفس النوع أو الاسم لتخطيط الشريحة المصدر). إذا لم يكن هناك تخطيط مناسب، فسيتم استنساخ تخطيط الشريحة المصدر (إذا كان allowCloneMissingLayout true) أو سيتم إطلاق استثناء PptxEditException (إذا كان allowCloneMissingLayout false).

public ISlide InsertClone(int index, ISlide sourceSlide, IMasterSlide destMaster,

| allowCloneMissingLayout | Boolean | إذا لم يكن هناك تخطيط مناسب في القالب المحدد، فسيتم استنساخ تخطيط الشريحة المصدر (إذا كان allowCloneMissingLayout true) أو سيتم إطلاق استثناء PptxEditException (إذا كان allowCloneMissingLayout false). |

## DocumentProperties.GetSensitivityLabels طريقة

يحصل على مصفوفة من تسميات الحساسية من خصائص المستند المخصصة (Microsoft Information Protection SDK Metadata).

```csharp
public ISensitivityLabel[] GetSensitivityLabels()
```

### أمثلة

يعرض الكود التالي كيفية نقل معلومات تسميات الحساسية من خصائص المستند المخصصة إلى مجموعة SensitivityLabels الحديثة:

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    // الحصول على تسميات الحساسية من خصائص المستند المخصصة
    ISensitivityLabel[] mipSensitivityLabels = pres.DocumentProperties.GetSensitivityLabels();

    ISensitivityLabelCollection sensitivityLabels = pres.SensitivityLabels;
    foreach (var sensitivityLabel in mipSensitivityLabels)
    {
        // إضافة التسمية إلى المجموعة
        // هنا يمكنك إضافة تحقق من صحة معلومات التسمية (التسمية متاحة، إلخ)
        sensitivityLabels.Add(sensitivityLabel);
    }

    pres.Save("SensitivityLabel.pptx", SaveFormat.Pptx);
}
```

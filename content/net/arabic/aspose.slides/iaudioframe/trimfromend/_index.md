## IAudioFrame.TrimFromEnd خاصية

يحدد مدة الوقت التي سيتم إزالتها من نهاية الوسائط أثناء التشغيل، بالمللي ثانية. قراءة/كتابة Single.

```csharp
public float TrimFromEnd { get; set; }
```

### أمثلة

مثال:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    IAudio audio = pres.Audios.AddAudio(File.ReadAllBytes("sampleaudio.mp3"));
    IAudioFrame audioFrame = pres.Slides[0].Shapes.AddAudioFrameEmbedded(50, 50, 100, 100, audio);

    // تعيين مدة القطع من النهاية 2 ثانية
    audioFrame.TrimFromEnd = 2000f;
}
```


### أمثلة

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    var slideSize = pres.SlideSize.Size;
    
    IDrawingGuidesCollection guides = pres.LayoutSlides[0].DrawingGuides;
    // إضافة دليل الرسم العمودي الجديد إلى يسار مركز الشريحة

}
```

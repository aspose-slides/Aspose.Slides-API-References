## خاصية IBlobManagementOptions.PresentationLockingBehavior

تحدد هذه الخاصية ما إذا كان يمكن لكائن من الفئة Presentation أن يكون مالكًا للمصدر - ملف أو تدفق - طوال عمر الكائن. إذا كان الكائن مالكًا، فإنه يقوم بقفل المصدر. يساعد ذلك على تحسين استهلاك الذاكرة والأداء أثناء العمل مع BLOBs، لكن لا يمكن تغيير المصدر (التدفق أو الملف) طوال عمر كائن Presentation. هذا مثال:

```csharp
public PresentationLockingBehavior PresentationLockingBehavior { get; set; }
```

### أمثلة

```csharp
LoadOptions loadOptions = new LoadOptions { 
  BlobManagementOptions = {PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked} };
using (Presentation pres = new Presentation("pres.pptx", loadOptions)) {
  // سيتم رمي IOException لأن pres.pptx مقفل طوال عمر Presentation
  // File.Delete("pres.pptx");
}
// بعد تحرير كائن Presentation، يصبح الملف غير مقفل ويمكن حذفه
File.Delete("pres.pptx");
```

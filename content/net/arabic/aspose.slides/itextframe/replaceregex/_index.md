## ITextFrame.ReplaceRegex طريقة

يستبدل جميع التطابقات للعبارة النمطية بالنص المحدد.

```csharp
public void ReplaceRegex(Regex regex, string newText, IFindResultCallback callback)
```

| المعامل | النوع | الوصف |
| --- | --- | --- |
| regex | Regex | العبارة النمطية Regex للحصول على السلاسل التي سيتم استبدالها. |
| newText | String | النص لاستبدال جميع تكرارات السلاسل التي سيتم استبدالها. |
| callback | IFindResultCallback | كائن رد الاتصال لتلقي نتائج البحث [`IFindResultCallback`](../../ifindresultcallback). |

### أمثلة

يعرض مثال الشيفرة التالي كيفية استبدال النص باستخدام العبارة النمطية مع النص المحدد.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// استبدل جميع الكلمات التي تحتوي على 10 أحرف أو أكثر بـ '***'
	((AutoShape)presentation.Slides[0].Shapes[0]).TextFrame.ReplaceRegex(regex, "***", null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

## ITextFrame.HighlightRegex طريقة

يقوم بتمييز جميع التطابقات للتعبير النمطي المحدد باللون المحدد.

```csharp
public void HighlightRegex(Regex regex, Color highlightColor, IFindResultCallback callback)
```

| Parameter | Type | Description |
| --- | --- | --- |
| regex | Regex | التعبير النمطي Regex للحصول على السلاسل التي سيتم تمييزها. |
| highlightColor | Color | اللون المستخدم لتسليط الضوء على النص. |
| callback | IFindResultCallback | كائن رد النداء للحصول على نتائج البحث [`IFindResultCallback`](../../ifindresultcallback). |

### أمثلة

الشفرة البرمجية التالية توضح كيفية تمييز النص داخل TextFrame باستخدام تعبير نمطي.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// تمييز جميع الكلمات التي تحتوي على 10 أحرف أو أكثر
	((AutoShape)presentation.Slides[0].Shapes[0]).TextFrame.HighlightRegex(regex, Color.Blue, null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

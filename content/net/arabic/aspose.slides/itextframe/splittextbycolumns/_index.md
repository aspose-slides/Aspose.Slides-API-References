## ITextFrame.SplitTextByColumns طريقة

يقوم بتقسيم محتوى النص الخاص بـ [`ITextFrame`](../../itextframe) إلى مصفوفة من السلاسل، حيث يتطابق كل عنصر مع عمود نصي منفصل داخل الإطار.

```csharp
public string[] SplitTextByColumns()
```

### قيمة الإرجاع

مصفوفة من السلاسل، حيث تمثل كل سلسلة محتوى النص لعمود محدد في [`ITextFrame`](../../itextframe).

### ملاحظات

إذا لم يحتوي إطار النص على أعمدة متعددة، فإن المصفوفة المرجعة ستحوي عنصرًا واحدًا يحتوي على النص الكامل. ستُمثل الأعمدة الفارغة كسلاسل فارغة في المصفوفة.

### أمثلة

المثال التالي يوضح كيفية استخدام `SplitTextByColumns`:

```csharp
using (Presentation pres = new Presentation("example.pptx"))
{
    // احصل على الشكل الأول في الشريحة وحوله إلى ITextFrame
    ITextFrame textFrame = pres.Slides[0].Shapes[0] as ITextFrame;
    // قسم محتوى إطار النص إلى أعمدة
    string[] columnsText = textFrame.SplitTextByColumns();
    // اطبع نص كل عمود إلى وحدة التحكم
    foreach (string column in columnsText)
        Console.WriteLine(column);
}
```

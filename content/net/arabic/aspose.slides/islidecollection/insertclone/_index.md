| Parameter | Type | Description |
| --- | --- | --- |

### Return Value

### Remarks

عند استنساخ شريحة بين عروض تقديمية مختلفة يمكن استنساخ القالب الرئيسي للشريحة أيضًا. يُستخدم سجل داخلي لتتبع القوالب المستنسخة تلقائيًا لمنع إنشاء نسخ متعددة من نفس القالب الرئيسي. لن يتم منع أو تسجيل الاستنساخ اليدوي للقوالب الرئيسية. إذا كنت تحتاج إلى مزيد من التحكم في عملية الاستنساخ، استخدم [`InsertClone`](../insertclone) أو [`InsertClone`](../insertclone) لاستنساخ الشرائح و[`AddClone`](../../imasterslidecollection/addclone) لاستنساخ القوالب الرئيسية.

## InsertClone(int, ISlide, IMasterSlide, bool) {#insertclone_2}

يدخل نسخة من الشريحة المصدر المحددة إلى الموضع المحدد في المجموعة. سيتم اختيار التخطيط المناسب تلقائيًا من القالب المحدد (التخطيط المناسب هو التخطيط الذي له نفس النوع أو الاسم مثل تخطيط الشريحة المصدر). إذا لم يوجد تخطيط مناسب، سيتم استنساخ تخطيط الشريحة المصدر (إذا كان allowCloneMissingLayout true) أو سيتم رمي PptxEditException (إذا كان allowCloneMissingLayout false).

```csharp
public ISlide InsertClone(int index, ISlide sourceSlide, IMasterSlide destMaster, 
    bool allowCloneMissingLayout)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | Int32 | فهرس الشريحة الجديدة. |
| sourceSlide | ISlide | الشريحة التي سيتم استنساخها. |
| destMaster | IMasterSlide | القالب الرئيسي للشريحة الجديدة. |
| allowCloneMissingLayout | Boolean | إذا لم يوجد تخطيط مناسب في القالب المحدد، سيتم استنساخ تخطيط الشريحة المصدر (إذا كان allowCloneMissingLayout true) أو سيتم رمي PptxEditException (إذا كان allowCloneMissingLayout false). |

### Return Value

الشريحة المُدرجة.

### Exceptions

| exception | condition |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | يتم رميه إذا لم يكن هناك تخطيط مناسب في القالب المحدد وكان allowCloneMissingLayout false. |

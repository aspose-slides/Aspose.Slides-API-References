## طريقة Presentation.ReplaceText

يستبدل جميع مرات ظهور النص المحدد بنص محدد آخر.

```csharp
public void ReplaceText(string oldText, string newText, ITextSearchOptions options, 
    IFindResultCallback callback)
```

| المعامل | النوع | الوصف |
| --- | --- | --- |
| oldText | String | السلسلة التي سيتم استبدالها. |
| newText | String | السلسلة التي ستحل محل جميع مرات ظهور oldText. |
| options | ITextSearchOptions | خيارات بحث النص [`ITextSearchOptions`](../../itextsearchoptions). |
| callback | IFindResultCallback | كائن الاسترجاع لتلقي نتائج البحث [`IFindResultCallback`](../../ifindresultcallback). |

### أمثلة

يعرض الكود العيني التالي كيفية استبدال سلسلة محددة بأخرى محددة.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	// استبدال جميع حدوثات كلمة 'the' المنفصلة بـ '***'
	presentation.ReplaceText("the", "***", new TextSearchOptions() {WholeWordsOnly = true}, null);
	presentation.Save("SomePresentation-out2.pptx", SaveFormat.Pptx);
}
```

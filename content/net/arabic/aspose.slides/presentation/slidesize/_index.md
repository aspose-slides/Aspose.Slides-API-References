## Presentation.SlideSize property

يُرجع كائن حجم الشريحة. للقراءة فقط [`ISlideSize`](../../islidesize).

```csharp
public ISlideSize SlideSize { get; }
```

### أمثلة

يعرض المثال التالي كيفية تغيير حجم الشريحة في عرض PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation("pres-4x3-aspect-ratio.pptx"))
{
    pres.SlideSize.SetSize(SlideSizeType.OnScreen16x9, SlideSizeScaleType.DoNotScale);
    pres.Save("pres-4x3-aspect-ratio.pptx", SaveFormat.Pptx);
}
```

يعرض المثال التالي كيفية تعيين حجم الشريحة مع مراعاة مقياس المحتوى لعرض PowerPoint.

```csharp
[C#]
// إنشاء كائن Presentation يمثل ملف عرض تقديمي
using(Presentation presentation = new Presentation("AccessSlides.pptx")) {
  using(Presentation auxPresentation = new Presentation()) {
    ISlide slide = presentation.Slides[0];
    // ضبط حجم الشريحة للعروض المُنشأة ليكون مثل حجم المصدر
    presentation.SlideSize.SetSize(540, 720, SlideSizeScaleType.EnsureFit); // طريقة SetSize تُستخدم لتعيين حجم الشريحة مع مقياس المحتوى لضمان التناسب
    presentation.SlideSize.SetSize(SlideSizeType.A4Paper, SlideSizeScaleType.Maximize); // طريقة SetSize تُستخدم لتعيين حجم الشريحة مع تكبير حجم المحتوى
    // حفظ العرض التقديمي على القرص
    auxPresentation.Save("Set_Size&Type_out.pptx", SaveFormat.Pptx);
  }
}
```

يعرض المثال التالي كيفية تحديد أحجام شرائح مخصصة في عرض PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    pres.SlideSize.SetSize(780, 540, SlideSizeScaleType.DoNotScale); // حجم ورق A4
    pres.Save("pres-a4-slide-size.pptx", SaveFormat.Pptx);
}
```

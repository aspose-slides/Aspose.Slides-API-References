## طريقة Presentation.ReplaceRegex

يستبدل جميع التطابقات للتعبير النمطي بالسلسلة المحددة.

```csharp
public void ReplaceRegex(Regex regex, string newText, IFindResultCallback callback)
```

| المعلمة | النوع | الوصف |
| --- | --- | --- |
| regex | Regex | التعبير النمطي Regex للحصول على السلاسل التي سيتم استبدالها. |
| newText | String | السلسلة التي تستبدل جميع مرات ظهور السلاسل التي ستُستبدل. |
| callback | IFindResultCallback | كائن الاستدعاء الراجع لتلقي نتائج البحث [`IFindResultCallback`](../../ifindresultcallback). |

### أمثلة

يعرض عينة الشيفرة التالية كيفية استبدال النص باستخدام التعبير النمطي مع السلسلة المحددة.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// استبدال جميع الكلمات التي تحتوي على 10 أحرف أو أكثر بـ '***'
	presentation.ReplaceRegex(regex, "***", null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

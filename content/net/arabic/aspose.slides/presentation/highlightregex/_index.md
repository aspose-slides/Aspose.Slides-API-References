## Presentation.HighlightRegex طريقة

يقوم بتمييز جميع التطابقات للتعبير المنتظم باللون المحدد.

```csharp
public void HighlightRegex(Regex regex, Color highlightColor, IFindResultCallback callback)
```

| المعامل | النوع | الوصف |
| --- | --- | --- |
| regex | Regex | التعبير المنتظم Regex للحصول على السلاسل المراد تمييزها. |
| highlightColor | Color | اللون لتمييز النص. |
| callback | IFindResultCallback | كائن الاسترجاع لتلقي نتائج البحث [`IFindResultCallback`](../../ifindresultcallback). |

### أمثلة

يعرض عينة الكود التالية كيفية تمييز النص في عرض PowerPoint باستخدام تعبير منتظم.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// تمييز جميع الكلمات التي تحتوي على 10 أحرف أو أكثر
	presentation.HighlightRegex(regex, Color.Blue, null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

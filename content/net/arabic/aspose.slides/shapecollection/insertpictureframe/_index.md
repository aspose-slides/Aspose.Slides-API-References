```csharp
public IPictureFrame InsertPictureFrame(int index, ShapeType shapeType, float x, float y, 
    float width, float height, IPPImage image)
```

| index | Int32 | المؤشر الصفري الذي يتم عنده إدراج إطار الصورة. |
| shapeType | ShapeType | يحدد نوع الشكل الموجود في [`ShapeType`](../../shapetype)، باستثناء جميع أنواع الخطوط: |

| y | Single | الإحداثي الصادي لإطار الصورة، بوحدات النقاط. |
| width | Single | العرض لإطار الصورة، بوحدات النقاط. |
| height | Single | الارتفاع لإطار الصورة، بوحدات النقاط. |

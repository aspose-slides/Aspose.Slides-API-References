## InsertSectionZoomFrame(int, float, float, float, float, ISection) {#insertsectionzoomframe}

يقوم بإنشاء إطار Section Zoom جديد ويُدرجه في مجموعة الأشكال عند الفهرس المحدد.

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section)
```

| المعامل | النوع | الوصف |
| --- | --- | --- |
| index | Int32 | الفهرس الصفري الذي يُدرج فيه إطار Section Zoom. |
| x | Single | إحداثي x لإطار Section Zoom الجديد، بالوحدات النقطية. |
| y | Single | إحداثي y لإطار Section Zoom الجديد، بالوحدات النقطية. |
| width | Single | عرض إطار Section Zoom الجديد، بالوحدات النقطية. |
| height | Single | ارتفاع إطار Section Zoom الجديد، بالوحدات النقطية. |
| section | ISection | الـ [`ISection`](../../isection) المشار إليه من قبل إطار Section Zoom؛ يجب أن يكون جزءًا من هذا العرض التقديمي ويحتوي على شريحة واحدة على الأقل. |

### قيمة الإرجاع

الكائن [`ISectionZoomFrame`](../../isectionzoomframe) الذي تم إنشاؤه حديثًا.

### الاستثناءات

| استثناء | الشرط |
| --- | --- |
| ArgumentException | يُطرح إذا كان القسم المشار إليه لا ينتمي إلى العرض التقديمي الحالي أو لا يحتوي على أي شرائح. |

### أمثلة

هذا المثال يوضح إنشاء وإدراج كائن Section Zoom في الفهرس المحدد لمجموعة (افترض أن هناك قسمين على الأقل في عرض "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSectionZoomFrame(2, 150, 20, 50, 50, pres.Sections[1]);
}
```

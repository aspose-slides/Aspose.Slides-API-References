## ShapeCollection.InsertSummaryZoomFrame الطريقة

ينشئ إطار Summary Zoom جديد ويدخله في مجموعة الأشكال عند الفهرس المحدد.

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | Int32 | الفهرس القائم على الصفر الذي يتم عنده إدراج إطار Summary Zoom. |
| x | Single | إحداثي x لإطار Summary Zoom الجديد، بالنقاط. |
| y | Single | إحداثي y لإطار Summary Zoom الجديد، بالنقاط. |
| width | Single | عرض إطار Summary Zoom الجديد، بالنقاط. |
| height | Single | ارتفاع إطار Summary Zoom الجديد، بالنقاط. |

### قيمة الإرجاع

الكائن الذي تم إنشاؤه حديثًا [`ISummaryZoomFrame`](../../isummaryzoomframe).

### ملاحظات

تنشئ هذه الطريقة إطار Summary Zoom يجمع روابط الملخص لجميع الأقسام في العرض.

### أمثلة

يوضح هذا المثال إنشاء وإدراج كائن Summary Zoom عند الفهرس المحدد لمجموعة (افترض أن هناك قسمين على الأقل في عرض "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSummaryZoomFrame(2, 150, 20, 50, 50);
}
```

## طريقة ShapeCollection.AddSummaryZoomFrame

ينشئ إطار ملخص تكبير جديد ويضيفه إلى نهاية مجموعة الأشكال.

```csharp
public ISummaryZoomFrame AddSummaryZoomFrame(float x, float y, float width, float height)
```

| معامل | نوع | الوصف |
| --- | --- | --- |
| x | Single | إحداثي x لإطار ملخص التكبير الجديد، بوحدات النقاط. |
| y | Single | إحداثي y لإطار ملخص التكبير الجديد، بوحدات النقاط. |
| width | Single | عرض إطار ملخص التكبير الجديد، بوحدات النقاط. |
| height | Single | ارتفاع إطار ملخص التكبير الجديد، بوحدات النقاط. |

}

## InsertAudioFrameEmbedded(int, float, float, float, float, Stream) {#insertaudioframeembedded_1}

يقوم بإنشاء إطار صوت جديد يحتوي على ملف WAV مدمج ويُدرجه في مجموعة الأشكال في الفهرس المحدد. يتم إضافة الصوت المدمج إلى مجموعة Presentation.Audios.

```csharp
public IAudioFrame InsertAudioFrameEmbedded(int index, float x, float y, float width, float height, 
    Stream audio_stream)
```

| المعامل | النوع | الوصف |
| --- | --- | --- |
| index | Int32 | الفهرس الصفري الذي يُستخدم لإدراج إطار الصوت. |
| x | Single | إحداثي x لإطار الصوت الجديد، بوحدة النقاط. |
| y | Single | إحداثي y لإطار الصوت الجديد، بوحدة النقاط. |
| width | Single | عرض إطار الصوت الجديد، بوحدة النقاط. |
| height | Single | ارتفاع إطار الصوت الجديد، بوحدة النقاط. |
| audio_stream | Stream | تيار إدخال يحتوي على بيانات صوتية بصيغة WAV لتضمينه. |

## AddAudioFrameEmbedded(float, float, float, float, Stream) {#addaudioframeembedded_1}

ينشئ إطار صوت جديد يحتوي على ملف WAV مضمّن ويضيفه إلى نهاية مجموعة الأشكال. يتم إضافة الصوت المضمّن إلى مجموعة Presentation.Audios.

```csharp
public IAudioFrame AddAudioFrameEmbedded(float x, float y, float width, float height, 
    Stream audio_stream)
```

| Parameter | Type | Description |
| --- | --- | --- |
| x | Single | إحداثي x لإطار الصوت الجديد، بوحدات النقاط. |
| y | Single | إحداثي y لإطار الصوت الجديد، بوحدات النقاط. |
| width | Single | عرض إطار الصوت الجديد، بوحدات النقاط. |
| height | Single | ارتفاع إطار الصوت الجديد، بوحدات النقاط. |
| audio_stream | Stream | دفق إدخال يحتوي على بيانات صوتية بصيغة WAV للتضمين. |

### القيمة المرجعة

الإطار الصوتي الجديد الذي تم إنشاؤه [`IAudioFrame`](../../iaudioframe).

### أمثلة

يعرض المثال التالي كيفية إنشاء إطار صوت.

```csharp
[C#]
// ينشئ كائن من فئة العرض التي تمثل ملف عرض تقديمي
using (Presentation pres = new Presentation())
{
    // يحصل على الشريحة الأولى
    ISlide sld = pres.Slides[0];
    // يحمل ملف الصوت wav إلى تدفق
    FileStream fstr = new FileStream("sampleaudio.wav", FileMode.Open, FileAccess.Read);
    // يضيف إطار الصوت
    IAudioFrame audioFrame = sld.Shapes.AddAudioFrameEmbedded(50, 150, 100, 100, fstr);
    // يحدد وضع التشغيل وحجم الصوت
    audioFrame.PlayMode = AudioPlayModePreset.Auto;
    audioFrame.Volume = AudioVolumeMode.Loud;
    // يكتب ملف PowerPoint إلى القرص
    pres.Save("AudioFrameEmbed_out.pptx", SaveFormat.Pptx);
}
```

## طريقة IFontsManager.GetFontEmbeddingLevel

يحدد مستوى التضمين للخط من مصفوفة البايتات المعطاة واسم الخط.

```csharp
public EmbeddingLevel GetFontEmbeddingLevel(byte[] fontBytes, string fontName)
```

| المعامل | النوع | الوصف |
| --- | --- | --- |
| fontBytes | Byte[] | مصفوفة البايتات التي تحتوي على بيانات الخط. |
| fontName | String | اسم الخط. |

### قيمة الإرجاع

مستوى التضمين للخط المحدد.

### الاستثناءات

| الاستثناء | الشرط |
| --- | --- |
| ArgumentNullException | يتم طرحه عندما يكون *fontBytes* غير موجود. |

### أمثلة

```csharp
[C#]
using (Presentation pres = new Presentation(pptxFileName))
{
    // استرجاع جميع الخطوط المستخدمة في العرض التقديمي
    IFontData[] fontDatas = pres.FontsManager.GetFonts();

    // الحصول على مصفوفة البايتات التي تمثل النمط العادي لأول خط في العرض التقديمي
    byte[] bytes = pres.FontsManager.GetFontBytes(fontDatas[0], FontStyle.Regular);

    // تحديد مستوى تضمين الخط
    EmbeddingLevel embeddingLevel = pres.FontsManager.GetFontEmbeddingLevel(bytes, fontDatas[0].FontName);
}
```

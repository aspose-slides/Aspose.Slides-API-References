## ILoadOptions.DeleteEmbeddedBinaryObjects خاصية

يحدد ما إذا كان Aspose.Slides سيحذف جميع الكائنات الثنائية المضمنة أثناء تحميل العرض التقديمي.

أنواع الكائنات الثنائية المضمنة:

* مشروع VBA [`VbaProject`](../../ipresentation/vbaproject)
* بيانات OLE Object المضمنة [`EmbeddedFileData`](../../ioleembeddeddatainfo/embeddedfiledata)
* البيانات الثنائية لـ ActiveX Control [`ActiveXControlBinary`](../../icontrol/activexcontrolbinary)

قابل للقراءة والكتابة Boolean.

```csharp
public bool DeleteEmbeddedBinaryObjects { get; set; }
```

### ملاحظات

القيمة الافتراضية هي **false**.

### أمثلة

يوضح المثال التالي كيفية تحميل العرض التقديمي دون أي كائنات ثنائية مضمنة.

```csharp
[C#]
LoadOptions loadOptions = new LoadOptions();
loadOptions.DeleteEmbeddedBinaryObjects = true;
using (Presentation pres = new Presentation("pres.ppt", loadOptions))
{
    pres.Save("output_WithoutBinaryObjects.ppt", SaveFormat.Ppt);
}
```

```csharp
public void HighlightText(string text, Color highlightColor, ITextSearchOptions options, 
    IFindResultCallback callback)
```

| Parameter | Type | Description |
| --- | --- | --- |
| text | String | النص المراد تمييزه. |
| highlightColor | Color | اللون المستخدم لتمييز النص. |
| options | ITextSearchOptions | خيارات بحث النص [`ITextSearchOptions`](../../itextsearchoptions). |
| callback | IFindResultCallback | كائن الاستدعاء لتلقي نتائج البحث [`IFindResultCallback`](../../ifindresultcallback). |

### أمثلة

يعرض المثال البرمجي التالي كيفية تمييز النص في عرض تقديمي من PowerPoint.

```csharp

using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	// تمييز جميع ظهورات 'the' المنفصلة
```

## IPresentation.HighlightRegex طريقة

يُبرز جميع التطابقات للتعبير النمطي باللون المحدد.

```csharp
public void HighlightRegex(Regex regex, Color highlightColor, IFindResultCallback callback)
```

| المعامل | النوع | الوصف |
| --- | --- | --- |
| regex | Regex | التعبير النمطي Regex للحصول على السلاسل لتسليط الضوء عليها. |
| highlightColor | Color | اللون لتسليط الضوء على النص. |
| callback | IFindResultCallback | كائن رد النداء لتلقي نتائج البحث [`IFindResultCallback`](../../ifindresultcallback). |

### أمثلة

يعرض عينة الشيفرة التالية كيفية تسليط الضوء على النص في عرض تقديمي PowerPoint باستخدام تعبير نمطي.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// تسليط الضوء على جميع الكلمات التي تتكون من 10 أحرف أو أكثر
	presentation.HighlightRegex(regex, Color.Blue, null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

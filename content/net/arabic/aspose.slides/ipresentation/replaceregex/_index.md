## IPresentation.ReplaceRegex الطريقة

يستبدل جميع المتطابقات في التعبير العادي بالنص المحدد.

```csharp
public void ReplaceRegex(Regex regex, string newText, IFindResultCallback callback)
```

| معامل | نوع | الوصف |
| --- | --- | --- |
| regex | Regex | التعبير العادي Regex للحصول على السلاسل التي سيتم استبدالها. |
| newText | String | السلسلة التي تستبدل جميع حدوث السلاسل المراد استبدالها. |
| callback | IFindResultCallback | كائن النداء العكسي لتلقي نتائج البحث [`IFindResultCallback`](../../ifindresultcallback). |

### أمثلة

العينة التالية من الشيفرة توضح كيفية استبدال النص باستخدام التعبير العادي مع النص المحدد.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// استبدال جميع الكلمات التي تتكون من 10 أحرف أو أكثر بـ '***'
	presentation.ReplaceRegex(regex, "***", null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

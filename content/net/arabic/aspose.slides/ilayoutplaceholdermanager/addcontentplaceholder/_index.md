## ILayoutPlaceholderManager.AddContentPlaceholder طريقة

يضيف شكلاً نائبًا جديدًا إلى شريحة التخطيط لحمل المحتوى، مثل صورة أو جدول أو وسائط أو نص.

```csharp
public IAutoShape AddContentPlaceholder(float x, float y, float width, float height)
```

| المعامل | النوع | الوصف |
| --- | --- | --- |
| x | Single | الإحداثي X للشكل النائب الجديد. |
| y | Single | الإحداثي Y للشكل النائب الجديد. |
| width | Single | عرض الشكل النائب الجديد. |
| height | Single | ارتفاع الشكل النائب الجديد. |

### قيمة الإرجاع

تم إنشاء [`IAutoShape`](../../iautoshape) مع عنصر نائب من نوع Content.

### أمثلة

يوضح المثال التالي كيفية إضافة شكل عنصر نائب Content إلى شريحة التخطيط.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddContentPlaceholder(20, 20, 500, 300);
}
```

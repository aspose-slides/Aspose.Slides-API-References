## ILayoutPlaceholderManager.AddChartPlaceholder طريقة

يضيف شكل نائب جديد إلى شريحة التخطيط لاحتواء Chart.

```csharp
public IAutoShape AddChartPlaceholder(float x, float y, float width, float height)
```

| المعامل | النوع | الوصف |
| --- | --- | --- |
| x | Single | الإحداثي X للشكل النائب الجديد. |
| y | Single | الإحداثي Y للشكل النائب الجديد. |
| width | Single | عرض الشكل النائب الجديد. |
| height | Single | ارتفاع الشكل النائب الجديد. |

### قيمة الإرجاع

تم إنشاء [`IAutoShape`](../../iautoshape) مع عنصر نائب للـ Chart.

### أمثلة

المثال التالي يوضح كيفية إضافة شكل نائب للـ Chart إلى شريحة التخطيط.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddChartPlaceholder(20, 20, 200, 200);
}
```

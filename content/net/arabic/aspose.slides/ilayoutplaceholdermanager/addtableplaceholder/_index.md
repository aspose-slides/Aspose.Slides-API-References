## ILayoutPlaceholderManager.AddTablePlaceholder طريقة

يضيف شكل عنصر نائب جديد إلى شريحة التخطيط لاحتواء جدول.

```csharp
public IAutoShape AddTablePlaceholder(float x, float y, float width, float height)
```

| المعامل | النوع | الوصف |
| --- | --- | --- |
| x | Single | الإحداثي X للشكل العنصر النائب الجديد. |
| y | Single | الإحداثي Y للشكل العنصر النائب الجديد. |
| width | Single | عرض العنصر النائب الجديد. |
| height | Single | ارتفاع العنصر النائب الجديد. |

### قيمة الإرجاع

تم إنشاء [`IAutoShape`](../../iautoshape) مع عنصر نائب Table.

### أمثلة

المثال التالي يوضح كيفية إضافة شكل عنصر نائب Table إلى شريحة التخطيط.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTablePlaceholder(20, 20, 500, 200);
}
```

## طريقة ILayoutPlaceholderManager.AddTextPlaceholder

يضيف شكلاً نائبًا جديدًا إلى شريحة التخطيط لاحتواء محتوى النص.

```csharp
public IAutoShape AddTextPlaceholder(float x, float y, float width, float height)
```

| المعامل | النوع | الوصف |
| --- | --- | --- |
| x | Single | الإحداثي X للشكلة النائبة الجديدة. |
| y | Single | الإحداثي Y للشكلة النائبة الجديدة. |
| width | Single | العرض للشكلة النائبة الجديدة. |
| height | Single | الارتفاع للشكلة النائبة الجديدة. |

### قيمة الإرجاع

تم إنشاء [`IAutoShape`](../../iautoshape) مع عنصر نائب نص.

### أمثلة

المثال التالي يوضح كيفية إضافة شكل العنصر النائب للنص إلى شريحة التخطيط.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTextPlaceholder(20, 20, 500, 300);
}
```

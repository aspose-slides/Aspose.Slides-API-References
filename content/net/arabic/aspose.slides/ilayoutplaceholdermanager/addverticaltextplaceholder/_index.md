## ILayoutPlaceholderManager.AddVerticalTextPlaceholder طريقة

يضيف شكلاً نائباً جديداً إلى شريحة التخطيط لاحتواء محتوى النص في اتجاه عمودي.

```csharp
public IAutoShape AddVerticalTextPlaceholder(float x, float y, float width, float height)
```

| المعامل | النوع | الوصف |
| --- | --- | --- |
| x | Single | إحداثي X للشكل النائب الجديد. |
| y | Single | إحداثي Y للشكل النائب الجديد. |
| width | Single | عرض الشكل النائب الجديد. |
| height | Single | ارتفاع الشكل النائب الجديد. |

### قيمة الإرجاع

تم إنشاء [`IAutoShape`](../../iautoshape) مع نائب نص (عمودي).

### أمثلة

يوضح المثال التالي كيفية إضافة شكل نائب النص (عمودي) إلى شريحة التخطيط.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalTextPlaceholder(20, 20, 300, 500);
}
```

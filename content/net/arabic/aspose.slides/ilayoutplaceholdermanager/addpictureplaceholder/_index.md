## ILayoutPlaceholderManager.AddPicturePlaceholder طريقة

يضيف شكلاً عنصرًا نائبًا جديدًا إلى شريحة التخطيط ليحتوي على صورة.

```csharp
public IAutoShape AddPicturePlaceholder(float x, float y, float width, float height)
```

| معامل | نوع | وصف |
| --- | --- | --- |
| x | Single | إحداثي X للشكل العنصر النايب الجديد. |
| y | Single | إحداثي Y للشكل العنصر النايب الجديد. |
| width | Single | عرض الشكل العنصر النايب الجديد. |
| height | Single | ارتفاع الشكل العنصر النايب الجديد. |

### قيمة الإرجاع

تم إنشاء [`IAutoShape`](../../iautoshape) مع عنصر نائب صورة.

### أمثلة

يظهر المثال التالي كيفية إضافة شكل عنصر نائب صورة إلى شريحة التخطيط.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddPicturePlaceholder(20, 20, 200, 200);
}
```

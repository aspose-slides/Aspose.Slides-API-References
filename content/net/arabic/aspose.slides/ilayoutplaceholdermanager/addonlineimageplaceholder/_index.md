## ILayoutPlaceholderManager.AddOnlineImagePlaceholder method

يضيف شكلاً نائبًا جديدًا إلى شريحة التخطيط لحمل صورة عبر الإنترنت.

```csharp
public IAutoShape AddOnlineImagePlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| x | Single | إحداثي X للشكلة النائبة الجديدة. |
| y | Single | إحداثي Y للشكلة النائبة الجديدة. |
| width | Single | عرض الشكلة النائبة الجديدة. |
| height | Single | ارتفاع الشكلة النائبة الجديدة. |

### Return Value

تم إنشاء [`IAutoShape`](../../iautoshape) مع نائب صورة عبر الإنترنت.

### Examples

يوضح المثال التالي كيفية إضافة شكلة نائب صورة عبر الإنترنت إلى شريحة التخطيط.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddOnlineImagePlaceholder(20, 20, 200, 200);
}
```

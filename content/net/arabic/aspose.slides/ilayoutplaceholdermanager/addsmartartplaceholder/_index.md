## ILayoutPlaceholderManager.AddSmartArtPlaceholder طريقة

يضيف شكل عنصر نائب جديد إلى شريحة التخطيط لحمل مخطط SmartArt.

```csharp
public IAutoShape AddSmartArtPlaceholder(float x, float y, float width, float height)
```

| المعامل | النوع | الوصف |
| --- | --- | --- |
| x | Single | إحداثي X للعنصر النائب الجديد. |
| y | Single | إحداثي Y للعنصر النائب الجديد. |
| width | Single | عرض العنصر النائب الجديد. |
| height | Single | ارتفاع العنصر النائب الجديد. |

### قيمة الإرجاع

تم إنشاء [`IAutoShape`](../../iautoshape) مع عنصر نائب SmartArt.

### أمثلة

يوضح المثال التالي كيفية إضافة شكل عنصر نائب SmartArt إلى شريحة التخطيط.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddSmartArtPlaceholder(20, 20, 200, 200);
}
```

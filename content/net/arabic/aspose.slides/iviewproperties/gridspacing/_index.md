## IViewProperties.GridSpacing خاصية

تُرجِع أو تعيّن تباعد الشبكة الذي يجب استخدامه للشبكة الأساسية لوثيقة العرض التقديمي، بوحدات النقاط. قراءة/كتابة Single.

```csharp
public float GridSpacing { get; set; }
```

### ملاحظات

يجب أن تكون قيمة تباعد الشبكة رقمًا موجبًا. النطاق القياسي للقيمة يتراوح من 1 مم (2.8349607 نقطة) إلى 2 بوصة (144 نقطة).

### أمثلة

يعرض الكود النموذجي التالي كيفية تغيير تباعد الشبكة في عرض PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

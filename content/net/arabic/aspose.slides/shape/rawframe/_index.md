## Shape.RawFrame خاصية

تُعيد أو تُعين خصائص إطار الشكل الخام. قراءة/كتابة [`IShapeFrame`](../../ishapeframe).

```csharp
public IShapeFrame RawFrame { get; set; }
```

### أمثلة

الكود الذي يحاول تعيين إطار غير معرف إلى IShape.Frame لا معنى له في الحالة العامة (وخاصةً في الحالة التي تكون فيها مجموعة الأشكال الأصلية GroupShape متداخلة متعددة داخل مجموعات أشكال أخرى). على سبيل المثال:

```csharp
IShape shape = ...;
shape.Frame = new ShapeFrame(float.NaN, float.NaN, float.NaN, float.NaN, NullableBool.NotDefined, NullableBool.NotDefined, float.NaN);
```

أو

```csharp
slide.Shapes.AddAutoShape(ShapeType.RoundCornerRectangle, float.NaN, float.NaN, float.NaN, float.NaN);
```

يمكن أن يؤدي مثل هذا الكود إلى مواقف غير واضحة. لذلك تم إضافة قيود لاستخدام القيم غير المعرفة لـ IShape.Frame. يجب تعريف قيم x و y والعرض والارتفاع و flipH و flipV و rotationAngle (ليس float.NaN أو NullableBool.NotDefined). الآن يرمي الكود في المثال السابق استثناء ArgumentException. ينطبق هذا على حالات الاستخدام التالية:

```csharp
IShape shape = ...;
shape.Frame = ...; // لا يمكن أن تكون غير معرفة

IShapeCollection shapes = ...;
// المعلمات x ، y ، العرض ، الارتفاع لا يمكن أن تكون float.NaN:
{
    shapes.AddAudioFrameCD(...);
    shapes.AddAudioFrameEmbedded(...);
    shapes.AddAudioFrameLinked(...);
    shapes.AddAutoShape(...);
    shapes.AddChart(...);
    shapes.AddConnector(...);
    shapes.AddOleObjectFrame(...);
    shapes.AddPictureFrame(...);
    shapes.AddSmartArt(...);
    shapes.AddTable(...);
    shapes.AddVideoFrame(...);
    shapes.InsertAudioFrameEmbedded(...);
    shapes.InsertAudioFrameLinked(...);
    shapes.InsertAutoShape(...);
    shapes.InsertChart(...);
    shapes.InsertConnector(...);
    shapes.InsertOleObjectFrame(...);
    shapes.InsertPictureFrame(...);
    shapes.InsertTable(...);
    shapes.InsertVideoFrame(...);
}
```

لكن خصائص إطار IShape.RawFrame يمكن أن تكون غير معرفة. هذا منطقي عندما يكون الشكل مرتبطًا بعنصر نائب. عندها يتم استبدال قيم إطار الشكل غير المعرفة من قبل شكل العنصر النائب الأصلي. إذا لم يكن هناك شكل عنصر نائب أصلي لذلك الشكل، فإن ذلك الشكل يستخدم القيم الافتراضية عند تقييم الإطار الفعّال بناءً على IShape.RawFrame الخاص به. القيم الافتراضية هي 0 و NullableBool.False للمتغيرات x و y والعرض والارتفاع و flipH و flipV و rotationAngle. على سبيل المثال:

```csharp
IShape shape = ...; // shape مرتبط بالعنصر النائب
shape.RawFrame = new ShapeFrame(float.NaN, float.NaN, 100, float.NaN, NullableBool.NotDefined, NullableBool.NotDefined, 0); // الآن shape يُورّث القيم x, y, height, flipH, flipV من العنصر النائب ويستبدل width=100 و rotationAngle=0.
```

## ProtectionManager.IsOnlyDocumentPropertiesLoaded خاصية

هذه الخاصية منطقية إذا كان ملف العرض محميًا بكلمة مرور وكانت خصائص المستند لهذا الملف عامة. تعني قيمة true أن خصائص المستند فقط يتم تحميلها من ملف عرض مشفر دون استخدام كلمة المرور. تعني قيمة false أن العرض المشفر بالكامل يتم تحميله باستخدام كلمة المرور الصحيحة، وليس فقط خصائص المستند. إذا لم يكن العرض مشفرًا فإن قيمة الخاصية تكون دائمًا false. إذا لم تكن خصائص المستند لملف مشفر عامة فإن قيمة الخاصية تكون دائمًا false. إذا كان Presentation.EncryptDocumentProperties هو true فإن قيمة خاصية IsOnlyDocumentPropertiesLoaded تكون دائمًا false. قراءة فقط Boolean.

```csharp
public bool IsOnlyDocumentPropertiesLoaded { get; }
```

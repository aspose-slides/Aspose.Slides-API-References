## ProtectionManager.SetWriteProtection طريقة

قم بتعيين حماية الكتابة لهذا العرض التقديمي باستخدام كلمة المرور المحددة.

```csharp
public void SetWriteProtection(string password)
```

| المعامل | النوع | الوصف |
| --- | --- | --- |
| password | String | كلمة المرور. |

### أمثلة

يعرض لك الشيفرة النموذجية التالية كيفية تعيين حماية كتابة للعرض التقديمي.

```csharp
[C#]
using (Presentation presentation = new Presentation("pres.pptx"))
{
    presentation.ProtectionManager.SetWriteProtection("123123");
    presentation.Save("write-protected-pres.pptx", SaveFormat.Pptx);
}
```

## ProtectionManager.CheckWriteProtection method

يحدد ما إذا كان العرض محميًا بكلمة مرور للتعديل.

```csharp
public bool CheckWriteProtection(string password)
```

| معامل | نوع | وصف |
| --- | --- | --- |
| password | String | كلمة المرور للتحقق. |

### قيمة الإرجاع

صحيح إذا كانت كلمة المرور صالحة؛ وإلا خطأ.

### ملاحظات

1. يجب عليك فحص الخاصية [`IsWriteProtected`](../iswriteprotected) قبل استدعاء هذه الدالة. 2. عندما تكون كلمة المرور null أو فارغة، تُعيد هذه الدالة خطأ.

### أمثلة

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

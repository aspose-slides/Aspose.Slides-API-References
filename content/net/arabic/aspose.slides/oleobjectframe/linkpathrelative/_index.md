## OleObjectFrame.LinkPathRelative خاصية

يعيد المسار النسبي إلى الملف المرتبط إذا كان موجودًا، وإلا يعيد سلسلة فارغة. للقراءة فقط String.

```csharp
public string LinkPathRelative { get; }
```

### ملاحظات

في عروض تقديمية Ppt، قد تكون بعض روابط كائنات Ole لها تمثيل نسبي.

### أمثلة

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.ppt"))
{
    IOleObjectFrame oleFrame = presentation.Slides[0].Shapes[0] as IOleObjectFrame;

    if (oleFrame != null)
    {
        Console.WriteLine("The relative path: " + oleFrame.LinkPathRelative);
    } 
}
```

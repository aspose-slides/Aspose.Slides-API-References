## OleObjectFrame.SetEmbeddedData طريقة

يضبط معلومات حول بيانات OLE المدمجة. تقوم هذه الطريقة بتغيير خصائص الكائن لتعكس البيانات الجديدة وتضبط علامة IsObjectLink إلى false، مما يشير إلى أن كائن OLE مدمج.

```csharp
public void SetEmbeddedData(IOleEmbeddedDataInfo embeddedData)
```

| معاملة | نوع | وصف |
| --- | --- | --- |
| embeddedData | IOleEmbeddedDataInfo | بيانات مدمجة [`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo) |

### الاستثناءات

| استثناء | شرط |
| --- | --- |
| ArgumentNullException | عند كون معامل embeddedData يساوي null. |

### أمثلة

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    OleObjectFrame oof = pres.Slides[0].Shapes[0] as OleObjectFrame;
    if (oof != null)
    {
        IOleEmbeddedDataInfo newData = new OleEmbeddedDataInfo(File.ReadAllBytes("Picture.png"), "png");
        oof.SetEmbeddedData(newData);
    }
}
```

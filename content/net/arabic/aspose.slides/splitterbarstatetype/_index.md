## SplitterBarStateType تعداد

يحدد الحالة التي يجب عرض شريط القسّام فيها. يفصل شريط القسّام بين منطقة أساسية ومنطقة ثانوية داخل منطقة العرض.

```csharp
public enum SplitterBarStateType
```

### القيم

| Name | Value | Description |
| --- | --- | --- |

| Restored | `1` | للمنطقة الأساسية حجم متوسط محدد. |
| Maximized | `2` | تشغل المنطقة الأساسية أكبر مساحة ممكنة من مساحة العرض المسموح بها من قبل التطبيق. |

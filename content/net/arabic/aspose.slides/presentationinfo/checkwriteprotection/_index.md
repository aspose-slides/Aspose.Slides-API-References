## PresentationInfo.CheckWriteProtection method

يتحقق مما إذا كانت كلمة المرور للتعديل صحيحة للعرض التقديمي المحمي ضد الكتابة.

```csharp
public bool CheckWriteProtection(string password)
```

| المعامل | النوع | الوصف |
| --- | --- | --- |
| password | String | كلمة المرور للتحقق منها. |

### قيمة الإرجاع

True إذا كان العرض التقديمي محميًا ضد الكتابة وكانت كلمة المرور صحيحة. False خلاف ذلك.

### الاستثناءات

| الاستثناء | الشرط |
| --- | --- |
| InvalidOperationException |  |

### ملاحظات

1. يجب عليك التحقق من الخاصية [`IsWriteProtected`](../iswriteprotected) قبل استدعاء هذه الطريقة. 2. عندما تكون كلمة المرور null أو فارغة، تُعيد هذه الطريقة false.

### أمثلة

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
if (info.IsWriteProtected == NullableBool.True)
{
    bool isWriteProtectedByPassword = info.CheckWriteProtection("my_password");
}
```

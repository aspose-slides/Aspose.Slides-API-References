## CommonSlideViewProperties.DrawingGuides خاصية

يعيد مجموعة أدلة الرسم. للقراءة فقط [`IDrawingGuidesCollection`](../../idrawingguidescollection)

```csharp
public IDrawingGuidesCollection DrawingGuides { get; }
```

### أمثلة

تُظهر الشفرة النموذجية التالية كيفية إضافة أدلة الرسم الجديدة في عرض تقديمي لبرنامج PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    var slideSize = pres.SlideSize.Size;
    
    IDrawingGuidesCollection guides = pres.ViewProperties.SlideViewProperties.DrawingGuides;
    // إضافة دليل الرسم العمودي الجديد إلى يمين مركز الشريحة
    guides.Add(Orientation.Vertical, slideSize.Width / 2 + 12.5f);
    // إضافة دليل الرسم الأفقي الجديد أسفل مركز الشريحة
    guides.Add(Orientation.Horizontal, slideSize.Height / 2 + 12.5f);
    
    pres.Save("DrawingGuides_out.pptx", SaveFormat.Pptx);
}
```

إرجاع مجموعة من إرشادات الرسم للشرائح الرئيسية. للقراءة فقط [`IDrawingGuidesCollection`](../../idrawingguidescollection)


### أمثلة

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    var slideSize = pres.SlideSize.Size;
    
    IDrawingGuidesCollection guides = pres.Masters[0].DrawingGuides;
    // إضافة دليل الرسم العمودي الجديد إلى يمين مركز الشريحة
    guides.Add(Orientation.Vertical, slideSize.Width / 2 + 20f);

}
```

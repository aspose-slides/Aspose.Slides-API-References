## طريقة MasterSlideCollection.InsertClone

يدخل نسخة من شريحة رئيسية محددة في الموضع المحدد في المجموعة. سيتم نسخ شرائح التخطيط المرتبطة أيضًا.

```csharp
public IMasterSlide InsertClone(int index, IMasterSlide sourceMaster)
```

| معامل | نوع | وصف |
| --- | --- | --- |
| index | Int32 | فهرس الشريحة الجديدة. |
| sourceMaster | IMasterSlide | الشريحة المراد استنساخها. |

### قيمة الإرجاع

الشريحة الرئيسية المُدخلة.

### أمثلة

يظهر المثال التالي كيفية استنساخ شريحة رئيسية في عرض تقديمي آخر لبرنامج PowerPoint.

```csharp
[C#]
// إنشاء كائن من فئة Presentation لتحميل ملف العرض التقديمي المصدر
using (Presentation srcPres = new Presentation("CloneToAnotherPresentationWithMaster.pptx"))
{
    // إنشاء كائن من فئة Presentation للعرض التقديمي الوجهة (حيث سيتم استنساخ الشريحة)
    using (Presentation destPres = new Presentation())
    {
        // إنشاء كائن ISlide من مجموعة الشرائح في العرض التقديمي المصدر مع
        // الشريحة الرئيسية
        ISlide SourceSlide = srcPres.Slides[0];
        IMasterSlide SourceMaster = SourceSlide.LayoutSlide.MasterSlide;
		// جلب الشرائح الرئيسية للعرض التقديمي الوجهة
        IMasterSlideCollection masters = destPres.Masters;
        // استنساخ الشريحة الرئيسية المطلوبة من العرض التقديمي المصدر إلى مجموعة الشرائح الرئيسية في
        // العرض التقديمي الوجهة
        IMasterSlide iSlide = masters.AddClone(SourceMaster);
        // مجموعة الشرائح في العرض التقديمي الوجهة
        ISlideCollection slds = destPres.Slides;
		// استنساخ الشريحة المصدر إلى مجموعة شرائح العرض التقديمي الوجهة.
        slds.AddClone(SourceSlide, iSlide, true);
        // Save the destination presentation to disk
        destPres.Save("CloneToAnotherPresentationWithMaster_out.pptx", SaveFormat.Pptx);
    }
}
```

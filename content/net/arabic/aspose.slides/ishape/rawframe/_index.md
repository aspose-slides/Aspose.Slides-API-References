## IShape.RawFrame خاصية

يُعيد أو يعيّن خصائص إطار الشكل الخام. قراءة/كتابة [`IShapeFrame`](../../ishapeframe).

```csharp
public IShapeFrame RawFrame { get; set; }
```

### أمثلة

الكود الذي يحاول تعيين إطار غير معرف إلى IShape.Frame لا معنى له في الحالة العامة (وخاصةً عندما يكون GroupShape الأب متداخلًا عدة مرات داخل GroupShape أخرى). على سبيل المثال:

```csharp
IShape shape = ...;
shape.Frame = new ShapeFrame(float.NaN, float.NaN, float.NaN, float.NaN, NullableBool.NotDefined, NullableBool.NotDefined, float.NaN);
```

أو

```csharp
slide.Shapes.AddAutoShape(ShapeType.RoundCornerRectangle, float.NaN, float.NaN, float.NaN, float.NaN);
```

مثل هذا الكود يمكن أن يؤدي إلى مواقف غير واضحة. لذلك تم إضافة قيود لاستخدام القيم غير المعرفة لـ IShape.Frame. يجب تعريف قيم x و y والعرض والارتفاع و flipH و flipV و rotationAngle (ليس float.NaN أو NullableBool.NotDefined). الآن يرمي الكود المثال أعلاه استثناء ArgumentException. هذا ينطبق على حالات الاستخدام التالية:

```csharp
IShape shape = ...;
shape.Frame = ...; // لا يمكن أن تكون غير معرفة

IShapeCollection shapes = ...;
// المعاملات x, y, width, height لا يمكن أن تكون float.NaN:
{
    shapes.AddAudioFrameCD(...);
    shapes.AddAudioFrameEmbedded(...);
    shapes.AddAudioFrameLinked(...);
    shapes.AddAutoShape(...);
    shapes.AddChart(...);
    shapes.AddConnector(...);
    shapes.AddOleObjectFrame(...);
    shapes.AddPictureFrame(...);
    shapes.AddSmartArt(...);
    shapes.AddTable(...);
    shapes.AddVideoFrame(...);
    shapes.InsertAudioFrameEmbedded(...);
    shapes.InsertAudioFrameLinked(...);
    shapes.InsertAutoShape(...);
    shapes.InsertChart(...);
    shapes.InsertConnector(...);
    shapes.InsertOleObjectFrame(...);
    shapes.InsertPictureFrame(...);
    shapes.InsertTable(...);
    shapes.InsertVideoFrame(...);
}
```

لكن خصائص إطار IShape.RawFrame يمكن أن تكون غير معرفة. هذا منطقي عندما يكون الشكل مرتبطًا بعنصر نائب. ثم يتم استبدال قيم إطار الشكل غير المعرفة من عنصر نائب الشكل الأب. إذا لم يكن هناك عنصر نائب أب لهذا الشكل فإن هذا الشكل يستخدم القيم الافتراضية عند تقييم الإطار الفعلي بناءً على IShape.RawFrame الخاص به. القيم الافتراضية هي 0 و NullableBool.False لـ x و y والعرض والارتفاع و flipH و flipV و rotationAngle. على سبيل المثال:

```csharp
IShape shape = ...; // الشكل مرتبط بالعنصر النائب
shape.RawFrame = new ShapeFrame(float.NaN, float.NaN, 100, float.NaN, NullableBool.NotDefined, NullableBool.NotDefined, 0); // الآن يرث الشكل قيم x, y, height, flipH, flipV من العنصر النائب ويتجاوز عرض=100 وزاوية الدوران=0.
```

## IProtectionManager.CheckWriteProtection الطريقة

يحدد ما إذا كان العرض التقديمي محميًا بكلمة مرور للتعديل.

```csharp
public bool CheckWriteProtection(string password)
```

| المعامل | النوع | الوصف |
| --- | --- | --- |
| password | String | كلمة المرور للتحقق. |

### قيمة الإرجاع

True إذا كانت كلمة المرور صالحة؛ وإلا، false.

### ملاحظات

1. يجب عليك فحص خاصية [`IsWriteProtected`](../iswriteprotected) قبل استدعاء هذه الطريقة. 2. عندما تكون كلمة المرور فارغة أو لا قيمة لها، تُعيد هذه الطريقة false.

### أمثلة

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

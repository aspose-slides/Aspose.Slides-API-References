## IProtectionManager.IsOnlyDocumentPropertiesLoaded خاصية

هذا الخاصية تكون ذات معنى عندما يكون ملف العرض محميًا بكلمة مرور وتكون خصائص المستند لهذا الملف عامة. قيمة true تعني أنه يتم تحميل خصائص المستند فقط من ملف عرض مشفر دون استخدام كلمة المرور. قيمة false تعني أنه يتم تحميل العرض المشفر بالكامل باستخدام كلمة المرور الصحيحة، وليس فقط خصائص المستند. إذا لم يكن العرض مشفرًا فإن قيمة الخاصية تكون دائمًا false. إذا لم تكن خصائص المستند لملف مشفر عامة فإن قيمة الخاصية تكون دائمًا false. إذا كان PresentationEx.EncryptDocumentProperties صحيحًا فإن قيمة IsOnlyDocumentPropertiesLoaded تكون دائمًا false. قيمة منطقية للقراءة فقط.

public bool IsOnlyDocumentPropertiesLoaded { get; }

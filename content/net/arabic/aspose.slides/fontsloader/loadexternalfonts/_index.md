## FontsLoader.LoadExternalFonts طريقة

يضيف مجلدات إضافية للبحث عن الخطوط.

```csharp
public static void LoadExternalFonts(string[] directories)
```

| معلمة | نوع | وصف |
| --- | --- | --- |
| directories | String[] | المجلدات لقراءة الخطوط الإضافية. |

### أمثلة

الأمثلة التالية توضح كيفية تحميل خطوط مخصصة من .TTF

```csharp
[C#]
// مسار دليل المستندات.
string dataDir = "C:\\";
// مجلدات للبحث عن الخطوط
String[] folders = new String[] { dataDir };
// تحميل خطوط دليل الخطوط المخصص
FontsLoader.LoadExternalFonts(folders);
// قم ببعض العمل وأجرِ عرض تقديمي/شرائح
using (Presentation presentation = new Presentation(dataDir + "DefaultFonts.pptx"))
{
    presentation.Save(dataDir + "NewFonts_out.pptx", SaveFormat.Pptx);
	// مسح ذاكرة الخط المؤقتة
	FontsLoader.ClearCache();
}
```

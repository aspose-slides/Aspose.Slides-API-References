## Fonts.SetScriptFont طريقة

يُعيّن اسم الخط إلى علامة كتابة محددة، والتي تحدد كيفية عرض نص تلك العلامة في العرض التقديمي.

```csharp
public void SetScriptFont(string script, string fontName)
```

| المعامل | النوع | الوصف |
| --- | --- | --- |
| script | String | رمز الكتابة وفق BCP-47 (مثل "Arab", "Hebr", "Hans") الذي يحدد نظام الكتابة. |
| fontName | String | اسم الخط الذي سيتم تعيينه إلى العلامة المحددة. |

### أمثلة

هذا المثال يوضح كيفية ضبط الخط للكتابة العربية إلى "Segoe UI":

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segoe UI");
```

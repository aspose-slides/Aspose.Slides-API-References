## طريقة Fonts.GetScriptFont

يحصل على اسم الخط المرتبط بعلامة كتابة معينة من سمة العرض التقديمي.

```csharp
public string GetScriptFont(string script)
```

| معلمة | نوع | الوصف |
| --- | --- | --- |
| script | String | رمز الكتابة BCP-47 (مثلاً "Latn", "Cyrl", "Jpan") المستخدم لتحديد نظام كتابة. |

### قيمة الإرجاع

اسم الخط المستخدم للكتابة المحددة، أو `null` إذا لم يتم تعريف الكتابة.

### أمثلة

يوضح هذا المثال كيفية استرجاع الخط المعين للكتابة السرييلية في سمة العرض التقديمي.

```csharp
[C#]
string font = presentation.MasterTheme.FontScheme.Major.GetScriptFont("Cyrl");
Console.WriteLine("Font for Cyrillic script: " + font);
```

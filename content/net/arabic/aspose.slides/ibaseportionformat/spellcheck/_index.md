## IBasePortionFormat.SpellCheck الخاصية

الحصول على أو تعيين قيمة تشير إلى ما إذا كان تدقيق الإملاء مفعّلاً لجزء النص. عندما يتم تعيين هذه الخاصية إلى false، يتم قمع الفحوص الإملائية لعناصر النص. عندما يتم تعيينها إلى true، يُسمح بتدقيق الإملاء. القيمة الافتراضية هي `false`.

```csharp
public bool SpellCheck { get; set; }
```

### أمثلة

المثال التالي يوضح تمكين علامة SpellCheck قبل حفظ العرض التقديمي:

```csharp
[C#]
using (var pres = new Presentation("input.pptx"))
{
    // الوصول إلى الجزء الأول من النص داخل الشكل الأول على الشريحة الأولى
    var portion = ((AutoShape)pres.Slides[0].Shapes[0]).TextFrame.Paragraphs[0].Portions[0];
    // تمكين تدقيق الإملاء لهذا الجزء النصي
    portion.PortionFormat.SpellCheck = true;
    // حفظ العرض التقديمي المعدل
    pres.Save("output-with-spellcheck.pptx", SaveFormat.Pptx);
}
```

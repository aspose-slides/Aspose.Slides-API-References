## طريقة TextFrame.ReplaceText

يستبدل جميع مرات ظهور النص المحدد بنص آخر محدد.

```csharp
public void ReplaceText(string oldText, string newText, ITextSearchOptions options, 
    IFindResultCallback callback)
```

| المعامل | النوع | الوصف |
| --- | --- | --- |
| oldText | String | السلسلة التي سيتم استبدالها. |
| newText | String | السلسلة التي ستستبدل جميع مرات ظهور oldText. |
| options | ITextSearchOptions | خيارات البحث عن النص [`ITextSearchOptions`](../../itextsearchoptions). |
| callback | IFindResultCallback | كائن رد النداء لحفظ نتيجة عملية الاستبدال [`IFindResultCallback`](../../ifindresultcallback). |

### أمثلة

يعرض الشيفرة النموذجية التالية كيفية استبدال سلسلة محددة بأخرى محددة.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx")){
	// استبدال جميع مرات ظهور كلمة 'the' المنفصلة بـ '***'
	((AutoShape)presentation.Slides[0].Shapes[0]).TextFrame.ReplaceText("the", "***", new TextSearchOptions()
	{ WholeWordsOnly = true }, null);
	presentation.Save("SomePresentation-out2.pptx", SaveFormat.Pptx);
}
```

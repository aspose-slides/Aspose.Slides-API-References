## TextFrame.HighlightRegex طريقة

يتم تمييز جميع التطابقات للتعبير النمطي باللون المحدد.

```csharp
public void HighlightRegex(Regex regex, Color highlightColor, IFindResultCallback callback)
```

| المعامل | النوع | الوصف |
| --- | --- | --- |
| regex | Regex | التعبير النمطي Regex للحصول على السلاسل لتمييزها. |
| highlightColor | Color | اللون المستخدم لتمييز النص. |
| callback | IFindResultCallback | كائن رد النداء لاستلام نتائج البحث [`IFindResultCallback`](../../ifindresultcallback). |

### أمثلة

يوضح مثال الشيفرة التالي كيفية تمييز النص في TextFrame باستخدام تعبير نمطي.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// تمييز جميع الكلمات التي تحتوي على 10 أحرف أو أكثر
	((AutoShape)presentation.Slides[0].Shapes[0]).TextFrame.HighlightRegex(regex, Color.Blue, null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

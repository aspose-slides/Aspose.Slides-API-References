## TextFrame.SplitTextByColumns طريقة

يقسم محتوى النص داخل [`ITextFrame`](../../itextframe) إلى مصفوفة من السلاسل النصية، بحيث يُطابق كل عنصر عمود نصي منفصل داخل الإطار.

```csharp
public string[] SplitTextByColumns()
```

### قيمة الإرجاع

مصفوفة من السلاسل النصية، حيث تمثل كل سلسلة محتوى النص لعمود محدد في [`ITextFrame`](../../itextframe).

### ملاحظات

إذا لم يحتوي إطار النص على أعمدة متعددة، فإن المصفوفة المُرجعة ستحمل عنصرًا واحدًا يحتوي على النص الكامل. سيتم تمثيل الأعمدة الفارغة كسلاسل نصية فارغة في المصفوفة.

### أمثلة

يوضح المثال التالي كيفية استخدام `SplitTextByColumns`:

```csharp
using (Presentation pres = new Presentation("example.pptx"))
{
    // احصل على الشكل الأول في الشريحة وقم بتحويله إلى ITextFrame
    ITextFrame textFrame = pres.Slides[0].Shapes[0] as ITextFrame;
    // قسّم محتوى إطار النص إلى أعمدة
    string[] columnsText = textFrame.SplitTextByColumns();
    // اطبع نص كل عمود في وحدة التحكم
    foreach (string column in columnsText)
        Console.WriteLine(column);
}
```

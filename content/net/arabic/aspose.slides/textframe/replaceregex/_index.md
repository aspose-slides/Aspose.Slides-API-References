## TextFrame.ReplaceRegex method

يستبدل جميع التطابقات للتعبير النمطي بالسلسلة المحددة.

```csharp
public void ReplaceRegex(Regex regex, string newText, IFindResultCallback callback)
```

| المعلمة | النوع | الوصف |
| --- | --- | --- |
| regex | Regex | التعبير النمطي Regex للحصول على السلاسل التي سيتم استبدالها. |
| newText | String | السلسلة التي ستحل محل جميع مرات ظهور السلاسل التي سيتم استبدالها. |
| callback | IFindResultCallback | كائن رد النداء لحفظ نتيجة عملية الاستبدال [`IFindResultCallback`](../../ifindresultcallback). |

### أمثلة

يوضح الكود النموذجي التالي كيفية استبدال النص باستخدام تعبير نمطي مع سلسلة محددة.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx")){
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// استبدال جميع الكلمات التي تتكون من 10 أحرف أو أكثر بـ '***'
	((AutoShape)presentation.Slides[0].Shapes[0]).TextFrame.ReplaceRegex(regex, "***", null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

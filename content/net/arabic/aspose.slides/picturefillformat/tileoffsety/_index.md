## PictureFillFormat.TileOffsetY خاصية

تُرجِع أو تحدد الإزاحة الرأسية للنقش من أصل الشكل بالنقاط. القيمة الموجبة تحرك النقش إلى الأسفل، بينما القيمة السالبة تحركه إلى الأعلى. قراءة/كتابة Single.

```csharp
public float TileOffsetY { get; set; }
```

### أمثلة

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // يحصل على تنسيق تعبئة الصورة للشكل
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // يضبط وضع تعبئة الصورة إلى Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // يضبط الإزاحة الرأسية للنقش إلى -50 نقطة
    pictureFillFormat.TileOffsetY = -50;
}
```

## CompressImage(bool, PicturesCompression) {#compressimage}

يقوم بضغط الصورة عن طريق تقليل حجمها بناءً على حجم الشكل والدقة المحددة. بشكل اختياري، يقوم أيضًا بحذف المناطق المقتطعة.

```csharp
public bool CompressImage(bool deleteCroppedAreasOfImage, PicturesCompression resolution)
```

| Parameter | Type | Description |
| --- | --- | --- |
| deleteCroppedAreasOfImage | Boolean | إذا كان صحيحًا، سيقوم الأسلوب بإزالة المناطق المقصوصة من الصورة، مما قد يقلل حجمها أكثر. |
| resolution | PicturesCompression | الدقة المستهدفة للضغط، المحددة كقيمة من تعداد [`PicturesCompression`](../../../aspose.slides.export/picturescompression). |

### قيمة الإرجاع

قيمة منطقية تشير إلى ما إذا تم ضغط الصورة بنجاح. تُرجع `true` إذا تم تغيير حجم الصورة أو قصها، وإلا تُرجع `false`.

### الاستثناءات

| exception | condition |
| --- | --- |
| ArgumentException | يُرمى عندما لا تكون الدقة قيمة صالحة. |

### ملاحظات

تغيّر هذه الطريقة حجم الصورة ودقتها بطريقة مماثلة لميزة "Picture Format -> Compress Pictures" في PowerPoint.

### أمثلة

يوضح المثال التالي كيفية استخدام طريقة `CompressImage` لتقليل حجم صورة في عرض تقديمي عبر تعيين دقة مستهدفة وإزالة المناطق المقصوصة:

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;
    // ضغط الصورة بدقة مستهدفة 150 DPI (دقة الويب) وإزالة المناطق المقصوصة
    bool result = picFrame.PictureFormat.CompressImage(true, PicturesCompression.Dpi150);
}
```

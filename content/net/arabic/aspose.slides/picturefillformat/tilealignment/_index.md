## خاصية PictureFillFormat.TileAlignment

ترجع أو تعين كيفية محاذاة النسيج داخل الشكل. تتحكم هذه الإعدادات في نقطة بدء نمط النسيج وكيفية تكراره عبر الشكل. قراءة/كتابة [`RectangleAlignment`](../../rectanglealignment).

```csharp
public RectangleAlignment TileAlignment { get; set; }
```

### ملاحظات

القيمة الافتراضية هي أعلى اليسار.

### أمثلة

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // يحصل على تنسيق ملء الصورة للشكل
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // يضبط وضع ملء الصورة إلى التكرار
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // يضبط محاذاة التكرار إلى الزاوية السفلية اليمنى
    pictureFillFormat.TileAlignment = RectangleAlignment.BottomRight;
}
```

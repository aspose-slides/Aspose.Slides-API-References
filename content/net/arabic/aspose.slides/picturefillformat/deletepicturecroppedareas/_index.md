## PictureFillFormat.DeletePictureCroppedAreas طريقة

حذف المناطق المقصوصة من الصورة.

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### قيمة الإرجاع

صورة مقصوصة أو الصورة الأصلية إذا لم يكن القطع ضروريًا.

### ملاحظات

تحول هذه الطريقة ملفات WMF/EMF الوصفية إلى صورة PNG نقطية أثناء القطع.

### أمثلة

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // يحصل على PictureFrame
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // يحذف المناطق المقصوصة من صورة PictureFrame
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

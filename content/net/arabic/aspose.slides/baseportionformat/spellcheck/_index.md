## BasePortionFormat.SpellCheck الخاصية

يحصل أو يضبط قيمة تشير إلى ما إذا كان التدقيق الإملائي مفعلاً لجزء النص. عندما يتم تعيين هذه الخاصية إلى false، تُقمع فحوصات الإملاء لعناصر النص. عند تعيينها إلى true، يُسمح بالتدقيق الإملائي. القيمة الافتراضية هي `false`.

```csharp
public bool SpellCheck { get; set; }
```

### أمثلة

المثال التالي يوضح تمكين علم SpellCheck قبل حفظ العرض التقديمي:

```csharp
[C#]
using (var pres = new Presentation("input.pptx"))
{
    // الوصول إلى الجزء الأول من النص داخل الشكل الأول في الشريحة الأولى
    var portion = ((AutoShape)pres.Slides[0].Shapes[0]).TextFrame.Paragraphs[0].Portions[0];
    // تمكين التدقيق الإملائي لهذا الجزء من النص
    portion.PortionFormat.SpellCheck = true;
    // حفظ العرض التقديمي المعدل
    pres.Save("output-with-spellcheck.pptx", SaveFormat.Pptx);
}
```

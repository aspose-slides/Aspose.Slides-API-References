## IFonts.GetScriptFont الطريقة

يجلب اسم الخط المرتبط بوسم script محدد من سمة العرض التقديمي.

```csharp
public string GetScriptFont(string script)
```

| معامل | النوع | الوصف |
| --- | --- | --- |
| script | String | رمز script وفق BCP-47 (مثال، "Latn"، "Cyrl"، "Jpan") المستخدم لتحديد نظام كتابة. |

### قيمة الإرجاع

اسم الخط المستخدم للسكريبت المحدد، أو `null` إذا لم يتم تعريف السكريبت.

### أمثلة

يوضح هذا المثال كيفية استرجاع الخط المعين للسكريبت السيريلي في سمة العرض التقديمي.

```csharp
[C#]
string font = presentation.MasterTheme.FontScheme.Major.GetScriptFont("Cyrl");
Console.WriteLine("Font for Cyrillic script: " + font);
```

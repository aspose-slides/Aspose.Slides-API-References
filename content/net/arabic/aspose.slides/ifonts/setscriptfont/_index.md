## طريقة IFonts.SetScriptFont

يُعيّن اسم الخط إلى علامة نصية محددة، والتي تحدد كيفية عرض نص ذلك النص في العرض التقديمي.

```csharp
public void SetScriptFont(string script, string fontName)
```

| المعامل | النوع | الوصف |
| --- | --- | --- |
| script | String | رمز النص BCP-47 (مثل "Arab"، "Hebr"، "Hans") الذي يحدد نظام الكتابة. |
| fontName | String | اسم الخط الذي سيتم تعيينه للنص المحدد. |

### أمثلة

يوضح هذا المثال كيفية تعيين الخط للنص العربي إلى "Segoe UI":

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segoe UI");
```

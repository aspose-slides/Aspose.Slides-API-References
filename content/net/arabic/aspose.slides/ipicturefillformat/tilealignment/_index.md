## IPictureFillFormat.TileAlignment property

تُرجِع أو تُعيِّن كيفية محاذاة النسيج داخل الشكل. يتحكم هذه الإعدادات في نقطة بدء نمط النسيج وكيفية تكراره عبر الشكل. قراءة/كتابة [`RectangleAlignment`](../../rectanglealignment).

```csharp
public RectangleAlignment TileAlignment { get; set; }
```

### ملاحظة

القيمة الافتراضية هي TopLeft.

### أمثلة

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // يحصل على تنسيق تعبئة الصورة للشكل
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // يضبط وضع تعبئة الصورة إلى Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // يضبط محاذاة التببيط إلى الزاوية السفلية اليمنى
    pictureFillFormat.TileAlignment = RectangleAlignment.BottomRight;
}
```

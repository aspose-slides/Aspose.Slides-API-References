## IPictureFillFormat.TileOffsetX خاصية

إرجاع أو تعيين الإزاحة الأفقية للنقش من أصل الشكل بالنقاط. القيمة الموجبة تحرك النقش إلى اليمين، بينما القيمة السالبة تحركه إلى اليسار. قراءة/كتابة Single.

```csharp
public float TileOffsetX { get; set; }
```

### أمثلة

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // يحصل على تنسيق تعبئة الصورة للشكل
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // يحدد وضع تعبئة الصورة إلى Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // يضبط الإزاحة الأفقية للنقش إلى 20 نقطة
    pictureFillFormat.TileOffsetX = 20f;
}
```

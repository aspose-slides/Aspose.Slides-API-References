## GeometryShape.SetGeometryPaths طريقة

يقوم بتحديث هندسة الشكل من مصفوفة من [`IGeometryPath`](../../igeometrypath). يجب أن تكون الإحداثيات نسبة إلى الزاوية اليسرى العلوية للشكل. يغيّر نوع الشكل ([`ShapeType`](../shapetype)) إلى Custom.

```csharp
public void SetGeometryPaths(IGeometryPath[] geometryPaths)
```

| المعلمة | النوع | الوصف |
| --- | --- | --- |
| geometryPaths | IGeometryPath[] | مصفوفة مسارات الهندسة |

### الاستثناءات

| الاستثناء | الشرط |
| --- | --- |
| ArgumentException | لم يتم العثور على مسار |
| ArgumentException | مسار فارغ |

### أمثلة

مثال:

```csharp
using (Presentation pres = new Presentation())
{
    GeometryShape shape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 200, 100) as GeometryShape;

    GeometryPath geometryPath = shape.GetGeometryPaths()[0];

    geometryPath.LineTo(100, 50, 1);
    geometryPath.LineTo(100, 50, 4);

    shape.SetGeometryPath(geometryPath);

    pres.Save("output.pptx", SaveFormat.Pptx);
}
```

## IDocumentProperties.GetSensitivityLabels طريقة

يجلب مصفوفة من أوسمة الحساسية من خصائص المستند المخصصة (Microsoft Information Protection SDK Metadata).

```csharp
public ISensitivityLabel[] GetSensitivityLabels()
```

### أمثلة

يوضح الكود التالي كيفية نقل معلومات أوسمة الحساسية من خصائص المستند المخصصة إلى مجموعة SensitivityLabels الحديثة:

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    // احصل على أوسمة الحساسية من خصائص المستند المخصصة
    ISensitivityLabel[] mipSensitivityLabels = pres.DocumentProperties.GetSensitivityLabels();

    ISensitivityLabelCollection sensitivityLabels = pres.SensitivityLabels;
    foreach (var sensitivityLabel in mipSensitivityLabels)
    {
        // إضافة الوسم إلى المجموعة
        // هنا يمكنك إضافة فحص لصحة معلومات الوسم (الوسم متاح، إلخ)
        sensitivityLabels.Add(sensitivityLabel);
    }

    pres.Save("SensitivityLabel.pptx", SaveFormat.Pptx);
}
```

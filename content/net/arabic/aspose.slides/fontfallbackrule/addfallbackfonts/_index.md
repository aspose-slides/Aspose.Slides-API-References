## AddFallBackFonts(string) {#addfallbackfonts}

يقوم بإضافة خط أو خطوط جديدة إلى قائمة خطوط FallBack.

```csharp
public void AddFallBackFonts(string fontName)
```

| معامل | نوع | وصف |
| --- | --- | --- |
| fontName | String | اسم الخط أو الأسماء (مفصولة بفواصل) لـ FallBack |

### أمثلة

```csharp
[C#]
// إنشاء نسخة جديدة من FontFallBackRule
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho");

//إضافة خط ثانٍ إلى القاعدة 
newRule.AddFallBackFonts("MS Gothic");
//إضافة الخط الثالث والرابع إلى القاعدة 
newRule.AddFallBackFonts("Tahoma, Times New Roman");
```

## FontFallBackRule.IndexOf طريقة

يرجع فهرس القاعدة المحددة في المجموعة.

```csharp
public int IndexOf(string fontName)
```

| معامل | نوع | الوصف |
| --- | --- | --- |
| fontName | String | اسم الخط للبحث عنه. |

### قيمة الإرجاع

فهرس الخط أو -1 إذا لم يتم العثور على الخط في القائمة.

### أمثلة

```csharp
[C#]
// إنشاء قاعدة تحتوي على قائمة من الخطوط.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// الحصول على فهرس Tahoma.
int tahomaIndex = newRule.IndexOf("Tahoma");
```

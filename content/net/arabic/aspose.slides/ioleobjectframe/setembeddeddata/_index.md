| الاستثناء | الشرط |

### ملاحظات

تقوم هذه الطريقة بتغيير خصائص الكائن لتنعكس البيانات الجديدة وتضبط العلم IsObjectLink على false، مشيرة إلى أن كائن OLE مضمَّن.

### أمثلة

يوضح المثال التالي كيفية تغيير بيانات OLE المضمَّنة ونوعها لكائن [`IOleObjectFrame`](../../ioleobjectframe) الحالي

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    OleObjectFrame oof = pres.Slides[0].Shapes[0] as OleObjectFrame;
    if (oof != null)
    {
        IOleEmbeddedDataInfo newData = new OleEmbeddedDataInfo(File.ReadAllBytes("Picture.png"), "png");
        oof.SetEmbeddedData(newData);
    }
}
```

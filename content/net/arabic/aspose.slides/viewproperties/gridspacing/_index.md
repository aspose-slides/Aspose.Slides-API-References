## ViewProperties.GridSpacing خاصية

يعيد أو يضبط تباعد الشبكة الذي يجب استخدامه للشبكة الأساسية لمستند العرض التقديمي، بوحدات النقاط. قراءة/كتابة Single.

```csharp
public float GridSpacing { get; set; }
```

### ملاحظات

يجب أن تكون قيمة تباعد الشبكة عددًا موجبًا. النطاق النموذجي يتراوح من 1 مم (2.8349607 نقطة) إلى 2 بوصة (144 نقطة).

### أمثلة

يعرض الشيفرة النموذجية التالية طريقة تغيير تباعد الشبكة في عرض PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

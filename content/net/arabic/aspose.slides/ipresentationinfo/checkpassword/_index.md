## IPresentationInfo.CheckPassword طريقة

يتحقق مما إذا كانت كلمة المرور صحيحة لعروض تقديمية محمية بكلمة مرور مفتوحة.

```csharp
public bool CheckPassword(string password)
```

| المعامل | النوع | الوصف |
| --- | --- | --- |
| password | String | كلمة المرور المراد التحقق منها. |

### قيمة الإرجاع

True إذا كانت العروض التقديمية محمية بكلمة مرور مفتوحة وكانت كلمة المرور صحيحة، وإلا false.

### ملاحظات

عند كون كلمة المرور null أو فارغة، تُعيد هذه الطريقة false.

### أمثلة

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo("pres.pptx");
bool isPasswordCorrect = info.CheckPassword("my_password");
```

## طريقة LayoutPlaceholderManager.AddSmartArtPlaceholder

يقوم بإضافة شكل نائب جديد إلى شريحة التخطيط لاحتواء مخطط SmartArt.

```csharp
public IAutoShape AddSmartArtPlaceholder(float x, float y, float width, float height)
```

| المُعامل | النوع | الوصف |
| --- | --- | --- |
| x | Single | إحداثي X لشكل النائب الجديد. |
| y | Single | إحداثي Y لشكل النائب الجديد. |
| width | Single | عرض شكل النائب الجديد. |
| height | Single | ارتفاع شكل النائب الجديد. |

### قيمة الإرجاع

تم إنشاء [`IAutoShape`](../../iautoshape) مع نائب SmartArt.

### الأمثلة

يوضح المثال التالي كيفية إضافة شكل نائب SmartArt إلى شريحة التخطيط.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddSmartArtPlaceholder(20, 20, 200, 200);
}
```

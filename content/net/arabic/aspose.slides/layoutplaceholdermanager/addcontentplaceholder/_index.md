## LayoutPlaceholderManager.AddContentPlaceholder الطريقة

يضيف شكلاً جديدًا من نوع العنصر النائب إلى شريحة التخطيط لتخزين المحتوى، مثل صورة أو جدول أو وسائط أو نص.

```csharp
public IAutoShape AddContentPlaceholder(float x, float y, float width, float height)
```

| معامل | نوع | الوصف |
| --- | --- | --- |
| x | Single | الإحداثي X لشكل العنصر النائب الجديد. |
| y | Single | الإحداثي Y لشكل العنصر النائب الجديد. |
| width | Single | عرض شكل العنصر النائب الجديد. |
| height | Single | ارتفاع شكل العنصر النائب الجديد. |

### قيمة الإرجاع

تم إنشاء [`IAutoShape`](../../iautoshape) مع عنصر نائب من نوع Content.

### أمثلة

المثال التالي يوضح كيفية إضافة شكل العنصر النائب من نوع Content إلى شريحة التخطيط.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddContentPlaceholder(20, 20, 500, 300);
}
```

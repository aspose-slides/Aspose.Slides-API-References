## LayoutPlaceholderManager.AddOnlineImagePlaceholder طريقة

يضيف شكلًا نائبًا جديدًا إلى شريحة التخطيط لحمل صورة عبر الإنترنت.

```csharp
public IAutoShape AddOnlineImagePlaceholder(float x, float y, float width, float height)
```

| المعلمة | النوع | الوصف |
| --- | --- | --- |
| x | Single | الإحداثي X للشكل العنصر النائب الجديد. |
| y | Single | الإحداثي Y للشكل العنصر النائب الجديد. |
| width | Single | عرض الشكل العنصر النائب الجديد. |
| height | Single | ارتفاع الشكل العنصر النائب الجديد. |

### قيمة الإرجاع

تم إنشاء [`IAutoShape`](../../iautoshape) مع عنصر نائب صورة عبر الإنترنت.

### أمثلة

يوضح المثال التالي كيفية إضافة شكل عنصر نائب صورة عبر الإنترنت إلى شريحة التخطيط.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddOnlineImagePlaceholder(20, 20, 200, 200);
}
```

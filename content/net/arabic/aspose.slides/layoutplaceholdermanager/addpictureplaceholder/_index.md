## LayoutPlaceholderManager.AddPicturePlaceholder طريقة

يضيف شكل عنصر نائب جديد إلى شريحة التخطيط لحمل صورة.

```csharp
public IAutoShape AddPicturePlaceholder(float x, float y, float width, float height)
```

| معامل | نوع | الوصف |
| --- | --- | --- |
| x | Single | الإحداثي X للشكل العنصر النائب الجديد. |
| y | Single | الإحداثي Y للشكل العنصر النائب الجديد. |
| width | Single | عرض الشكل العنصر النائب الجديد. |
| height | Single | ارتفاع الشكل العنصر النائب الجديد. |

### قيمة الإرجاع

تم إنشاء [`IAutoShape`](../../iautoshape) مع عنصر نائب للصور.

### الأمثلة

يوضح المثال التالي كيفية إضافة شكل عنصر نائب للصور إلى شريحة التخطيط.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddPicturePlaceholder(20, 20, 200, 200);
}
```

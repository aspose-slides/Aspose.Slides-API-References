## LayoutPlaceholderManager.AddVerticalContentPlaceholder طريقة

يقوم بإضافة شكل عنصر نائب جديد إلى شريحة التخطيط لحفظ المحتوى، مثل صورة أو جدول أو وسائط أو نص في الاتجاه العمودي.

```csharp
public IAutoShape AddVerticalContentPlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| x | Single | إحداثي X للشكل العنصر النائب الجديد. |
| y | Single | إحداثي Y للشكل العنصر النائب الجديد. |
| width | Single | عرض الشكل العنصر النائب الجديد. |
| height | Single | ارتفاع الشكل العنصر النائب الجديد. |

### قيمة الإرجاع

تم إنشاء [`IAutoShape`](../../iautoshape) مع عنصر نائب للمحتوى (عمودي).

### أمثلة

يوضح المثال التالي كيفية إضافة شكل عنصر نائب المحتوى (عمودي) إلى شريحة التخطيط.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalContentPlaceholder(20, 20, 300, 500);
}
```

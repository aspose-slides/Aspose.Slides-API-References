## طريقة LayoutPlaceholderManager.AddVerticalTextPlaceholder

يقوم بإضافة شكل عنصر نائب جديد إلى شريحة التخطيط لحفظ محتوى النص في اتجاه عمودي.

```csharp
public IAutoShape AddVerticalTextPlaceholder(float x, float y, float width, float height)
```

| معامل | نوع | وصف |
| --- | --- | --- |
| x | Single | إحداثي X لشكل العنصر النائب الجديد. |
| y | Single | إحداثي Y لشكل العنصر النائب الجديد. |
| width | Single | عرض شكل العنصر النائب الجديد. |
| height | Single | ارتفاع شكل العنصر النائب الجديد. |

### قيمة الإرجاع

تم إنشاء [`IAutoShape`](../../iautoshape) مع عنصر نائب من نوع Text (Vertical).

### أمثلة

يوضح المثال التالي كيفية إضافة شكل عنصر نائب Text (Vertical) إلى شريحة التخطيط.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalTextPlaceholder(20, 20, 300, 500);
}
```

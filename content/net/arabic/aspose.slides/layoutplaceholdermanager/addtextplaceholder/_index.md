## LayoutPlaceholderManager.AddTextPlaceholder method

يقوم بإضافة شكل عنصر نائب جديد إلى شريحة التخطيط لاحتواء محتوى النص.

```csharp
public IAutoShape AddTextPlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| x | Single | إحداثي X لعنصر العنصر النائب الجديد. |
| y | Single | إحداثي Y لعنصر العنصر النائب الجديد. |
| width | Single | عرض عنصر العنصر النائب الجديد. |
| height | Single | ارتفاع عنصر العنصر النائب الجديد. |

### Return Value

تم إنشاء [`IAutoShape`](../../iautoshape) مع عنصر نائب Text.

### Examples

المثال التالي يوضح كيفية إضافة شكل عنصر نائب Text إلى شريحة التخطيط.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTextPlaceholder(20, 20, 500, 300);
}
```

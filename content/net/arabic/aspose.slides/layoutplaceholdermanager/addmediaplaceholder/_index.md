## LayoutPlaceholderManager.AddMediaPlaceholder طريقة

يضيف شكلاً نائبًا جديدًا إلى شريحة التخطيط لاحتواء كائن وسائط.

```csharp
public IAutoShape AddMediaPlaceholder(float x, float y, float width, float height)
```

| معامل | نوع | وصف |
| --- | --- | --- |
| x | Single | إحداثي X للشكل النائب الجديد. |
| y | Single | إحداثي Y للشكل النائب الجديد. |
| width | Single | عرض الشكل النائب الجديد. |
| height | Single | ارتفاع الشكل النائب الجديد. |

### قيمة الإرجاع

تم إنشاء [`IAutoShape`](../../iautoshape) مع عنصر نائب للوسائط.

### أمثلة

يوضح المثال التالي كيفية إضافة شكل نائب للوسائط إلى شريحة التخطيط.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddMediaPlaceholder(20, 20, 200, 200);
}
```

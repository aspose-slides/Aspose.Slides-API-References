## LayoutPlaceholderManager.AddChartPlaceholder طريقة

يضيف شكلاً نائماً جديداً إلى شريحة التخطيط لتضمين مخطط Chart.

```csharp
public IAutoShape AddChartPlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| x | Single | إحداثي X للشكلة النائبة الجديدة. |
| y | Single | إحداثي Y للشكلة النائبة الجديدة. |
| width | Single | عرض الشكلة النائبة الجديدة. |
| height | Single | ارتفاع الشكلة النائبة الجديدة. |

### قيمة الإرجاع

تم إنشاء [`IAutoShape`](../../iautoshape) مع عنصر نائب من نوع Chart.

### أمثلة

يوضح المثال التالي كيفية إضافة شكل نائب مخطط Chart إلى شريحة التخطيط.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddChartPlaceholder(20, 20, 200, 200);
}
```

## InsertZoomFrame(int, float, float, float, float, ISlide) {#insertzoomframe}

يقوم بإنشاء إطار تكبير جديد وإدراجه في مجموعة الأشكال عند الفهرس المحدد.

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | Int32 | المؤشر الصفري الذي سيتم إدراج إطار التكبير عنده. |
| x | Single | الإحداثي السيني لإطار التكبير الجديد، بالنقاط. |
| y | Single | الإحداثي الصادي لإطار التكبير الجديد، بالنقاط. |
| width | Single | عرض إطار التكبير الجديد، بالنقاط. |
| height | Single | ارتفاع إطار التكبير الجديد، بالنقاط. |
| slide | ISlide | ال[`ISlide`](../../islide) المشار إليه بواسطة إطار التكبير. |

### قيمة الإرجاع

ال[`IZoomFrame`](../../izoomframe) الذي تم إنشاؤه حديثًا.

| --- | --- |
| ArgumentException | يتم إلقاؤه إذا لم تكن الشريحة المشار إليها تنتمي إلى العرض التقديمي الحالي. |

### أمثلة

يوضح هذا المثال إنشاء وإدراج كائن Zoom عند الفهرس المحدد في مجموعة (افترض وجود شريحتين على الأقل في عرض تقديمي "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertZoomFrame(2, 150, 20, 50, 50, pres.Slides[1]);
}
```

## InsertAudioFrameEmbedded(int, float, float, float, float, Stream) {#insertaudioframeembedded_1}

إنشاء إطار صوتي جديد بملف WAV مدمج وإدراجه في مجموعة الأشكال عند الفهرس المحدد. يتم إضافة الصوت المدمج إلى مجموعة Presentation.Audios.

```csharp
public IAudioFrame InsertAudioFrameEmbedded(int index, float x, float y, float width, float height, 
    Stream audio_stream)
```

| معامل | نوع | الوصف |
| --- | --- | --- |
| index | Int32 | الفهرس الصفري الذي يتم عنده إدراج إطار الصوت. |
| x | Single | الإحداثي السيني لإطار الصوت الجديد، بالنقاط. |
| y | Single | الإحداثي الصادي لإطار الصوت الجديد، بالنقاط. |
| width | Single | عرض إطار الصوت الجديد، بالنقاط. |
| height | Single | ارتفاع إطار الصوت الجديد، بالنقاط. |
| audio_stream | Stream | دفق الإدخال الذي يحتوي على بيانات صوتية بتنسيق WAV لتضمينه. |

### Return Value

الكائن [`IAudioFrame`](../../iaudioframe) الذي تم إنشاؤه حديثًا.

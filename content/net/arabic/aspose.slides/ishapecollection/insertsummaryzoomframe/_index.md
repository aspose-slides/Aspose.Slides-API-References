## IShapeCollection.InsertSummaryZoomFrame طريقة

ينشئ إطار ملخص تكبير جديد ويُدرجه في مجموعة الأشكال في الموضع المحدد.

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| معامل | نوع | الوصف |
| --- | --- | --- |
| index | Int32 | الفهرس الصفري-المستند الذي تُدرج فيه إطار ملخص التكبير. |
| x | Single | إحداثي x لإطار ملخص التكبير الجديد، بالنقاط. |
| y | Single | إحداثي y لإطار ملخص التكبير الجديد، بالنقاط. |
| width | Single | عرض إطار ملخص التكبير الجديد، بالنقاط. |
| height | Single | ارتفاع إطار ملخص التكبير الجديد، بالنقاط. |

### أمثلة

المثال التالي يوضح كيفية حفظ صورة SVG إلى ملف الميتا.

```csharp
[C#]
// ينشئ صورة SVG جديدة
ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));

// يحفظ صورة SVG كملف ميتا
using (var fileStream = System.IO.File.OpenWrite("SvgAsEmf.emf"))
    svgImage.WriteAsEmf(fileStream);
```

هذا المثال يوضح كيفية إضافة صورة SVG كملف ميتا إلى مجموعة صور العرض التقديمي.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // ينشئ صورة SVG جديدة
    ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));
    using (var memStream = new MemoryStream())
    {
        // يحفظ صورة SVG كملف ميتا
        svgImage.WriteAsEmf(memStream);
        // يضيف ملف ميتا إلى مجموعة الصور
        pres.Images.AddImage(memStream.ToArray());
    }
}
```

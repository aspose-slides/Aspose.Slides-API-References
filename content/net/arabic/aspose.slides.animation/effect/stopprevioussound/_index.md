## خاصية Effect.StopPreviousSound

تحدد هذه السمة ما إذا كان تأثير الرسوم المتحركة يتوقف عن الصوت السابق. قراءة/كتابة Boolean.

```csharp
public bool StopPreviousSound { get; set; }
```

### أمثلة

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // احصل على التأثير الأول للشريحة الأولى.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // احصل على التأثير الأول للشريحة الثانية.
    IEffect secondSlideEffect = presentation.Slides[1].Timeline.MainSequence[0];
       
    if (firstSlideEffect.Sound != null)
    {
        // غيّر تحسينات/صوت التأثير الثاني إلى "Stop Previous Sound"
        secondSlideEffect.StopPreviousSound = true;
    }
}
```

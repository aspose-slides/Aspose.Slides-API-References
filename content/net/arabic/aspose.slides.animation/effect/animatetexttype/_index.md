## Effect.AnimateTextType الخاصية

يحدد نوع تحريك النص للتأثير. يمكن تحريك نص الشكل بالحرف أو بالكلمة أو كله مرة واحدة. قراءة/كتابة `AnimateTextType`.

```csharp
public AnimateTextType AnimateTextType { get; set; }
```

### أمثلة

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // الحصول على التأثير الأول للشريحة الأولى.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // تغيير نوع تحريك النص للتأثير إلى "حسب الحرف"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByLetter;
```

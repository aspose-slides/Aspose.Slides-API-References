## Effect.DelayBetweenTextParts الخاصية

يحدد تأخيرًا بين أجزاء النص المتحركة (كلمات أو أحرف). القيمة الموجبة تحدد نسبة مدة التأثير. القيمة السالبة تحدد التأخير بالثواني. قراءة/كتابة Single.

```csharp
public float DelayBetweenTextParts { get; set; }
```

### أمثلة

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // احصل على التأثير الأول للشريحة الأولى.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // غيّر نوع نص التأثير إلى "حسب الكلمة"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByWord;
    
    // حدد التأخير بين أجزاء النص المتحركة إلى 20٪ من مدة التأثير.
    firstSlideEffect.DelayBetweenTextParts = 20f;
}
```

## IEffect.AnimateTextType الخاصية

يعرف نوع نص متحرك للتأثير. يمكن تحريك نص الشكل بحرف، بكلمة أو جميعًا مرة واحدة. قابل للقراءة/الكتابة `AnimateTextType`.

```csharp
public AnimateTextType AnimateTextType { get; set; }
```

### أمثلة

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // الحصول على أول تأثير في الشريحة الأولى.
    // تغيير نوع النص المتحرك للتأثير إلى "حسب الحرف"
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Change the effect Animate text type to "By letter"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByLetter;
}
```

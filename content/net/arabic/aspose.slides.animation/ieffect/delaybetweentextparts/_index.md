## IEffect.DelayBetweenTextParts الخاصية

يعّرف تأخيرًا بين أجزاء النص المتحركة (كلمات أو أحرف). القيمة الإيجابية تحدد نسبة مدة التأثير. القيمة السلبية تحدد التأخير بالثواني. قراءة/كتابة Single.

```csharp
public float DelayBetweenTextParts { get; set; }
```

### أمثلة

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // الحصول على أول تأثير في الشريحة الأولى.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // تغيير نوع تأثير نص التحريك إلى "By word"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByWord;
    
    // ضبط التأخير بين أجزاء النص المتحركة إلى 20% من مدة التأثير.
    firstSlideEffect.DelayBetweenTextParts = 20f;
}
```

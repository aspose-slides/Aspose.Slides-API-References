## ITiming.RepeatUntilEndSlide خاصية

This attribute specifies if the effect will repeat until the end of the slide. قابل للقراءة والكتابة Boolean.

```csharp
public bool RepeatUntilEndSlide { get; set; }
```

### أمثلة

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // احصل على تسلسل التأثيرات للشريحة الأولى
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // احصل على أول تأثير في التسلسل الرئيسي.
    IEffect effect = effectsSequence[0];

    // غيّر توقيت/تكرار التأثير إلى "حتى نهاية الشريحة"
    effect.Timing.RepeatUntilEndSlide = true;
}
```

## ITiming.RepeatUntilNextClick خاصية

تحدد هذه الخاصية ما إذا كان التأثير سيتكرر حتى النقر التالي. قراءة/كتابة Boolean.

```csharp
public bool RepeatUntilNextClick { get; set; }
```

### أمثلة

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // الحصول على تسلسل التأثيرات للشريحة الأولى
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // الحصول على التأثير الأول من التسلسل الرئيسي.
    IEffect effect = effectsSequence[0];

    // تغيير توقيت/تكرار التأثير إلى "حتى النقر التالي"
    effect.Timing.RepeatUntilNextClick = true;
}
```

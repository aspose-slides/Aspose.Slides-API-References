## ITiming.Rewind خاصية

تحدد هذه السمة ما إذا كان التأثير سيعيد التشغيل عند الانتهاء من التشغيل. قراءة/كتابة Boolean.

```csharp
public bool Rewind { get; set; }
```

### أمثلة

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // الحصول على تسلسل التأثيرات للشريحة الأولى
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // الحصول على أول تأثير في التسلسل الرئيسي.
    IEffect effect = effectsSequence[0];

    // تفعيل خاصية Timing/Rewind للتأثير.
    effect.Timing.Rewind = true;
}
```

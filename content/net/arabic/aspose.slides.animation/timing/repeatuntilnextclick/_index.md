## خاصية Timing.RepeatUntilNextClick

هذا السمة تحدد ما إذا كان التأثير سيُعاد حتى النقر التالي. قابل للقراءة والكتابة Boolean.

```csharp
public bool RepeatUntilNextClick { get; set; }
```

### أمثلة

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // الحصول على تسلسل التأثيرات للشريحة الأولى
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // الحصول على التأثير الأول في التسلسل الرئيسي.
    IEffect effect = effectsSequence[0];

    // تغيير توقيت/إعادة التكرار للتأثير إلى "حتى النقر التالي"
    effect.Timing.RepeatUntilNextClick = true;
}
```

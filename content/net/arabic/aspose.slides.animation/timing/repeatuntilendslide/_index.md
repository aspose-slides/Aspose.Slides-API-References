## Timing.RepeatUntilEndSlide خاصية

هذه السمة تحدد ما إذا كان التأثير سيتكرر حتى نهاية الشريحة. قراءة/كتابة Boolean.

```csharp
public bool RepeatUntilEndSlide { get; set; }
```

### أمثلة

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // احصل على تسلسل التأثيرات للشريحة الأولى
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // احصل على أول تأثير في التسلسل الرئيسي.
    IEffect effect = effectsSequence[0];

    // غيّر توقيت/تكرار التأثير إلى "Until End of Slide"
    effect.Timing.RepeatUntilEndSlide = true;
}
```

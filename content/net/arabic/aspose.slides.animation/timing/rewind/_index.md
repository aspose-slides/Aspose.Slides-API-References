## خاصية Timing.Rewind

هذه الخاصية تحدد ما إذا كان التأثير سيعود إلى البداية عند الانتهاء من التشغيل. قابل للقراءة/الكتابة Boolean.

```csharp
public bool Rewind { get; set; }
```

### أمثلة

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // احصل على تسلسل التأثيرات للشريحة الأولى
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // احصل على التأثير الأول في التسلسل الرئيسي.
    IEffect effect = effectsSequence[0];

    // فعّل توقيت/إعادة تشغيل التأثير.
    effect.Timing.Rewind = true;
}
```

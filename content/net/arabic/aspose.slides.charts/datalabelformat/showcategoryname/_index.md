## DataLabelFormat.ShowCategoryName خاصية

يمثل سلوك عرض اسم فئة تسمية البيانات لمخطط محدد. True لعرض اسم الفئة لتسميات البيانات على المخطط. False لإخفائه. قراءة/كتابة Boolean.

```csharp
public bool ShowCategoryName { get; set; }
```

### ملاحظات

إذا كان أصل كائن DataLabelFormat هذا هو مجموعة DataLabelCollection من تسميات البيانات، فإن هذه الخاصية تحصل أو تعيين القيمة الافتراضية لخاصية ShowCategoryName للتسميات البيانات الجديدة في مجموعة DataLabelCollection. تعيين هذه الخاصية بقيمة يضع أيضًا هذه القيمة لخاصية ShowCategoryName لجميع تسميات البيانات في مجموعة DataLabelCollection (أي "DataLabels.DefaultDataLabelFormat.ShowCategoryName = val;" يؤدي إلى أن جميع DataLabels[i].ShowCategoryName يساوي val).

## DataLabelFormat.ShowLegendKey خاصية

يمثّل سلوك عرض مفتاح وسيلة الإيضاح لتسمية البيانات للمخطط المحدد. True إذا كان مفتاح وسيلة الإيضاح لتسمية البيانات مرئياً. قراءة/كتابة Boolean.

```csharp
public bool ShowLegendKey { get; set; }
```

### ملاحظات

إذا كان أب هذا الكائن DataLabelFormat هو مجموعة DataLabelCollection من تسميات البيانات، فإن هذه الخاصية تحصل على أو تعيين القيمة الافتراضية لخاصية ShowLegendKey للتسميات الجديدة في مجموعة DataLabelCollection. تعيين هذه الخاصية بقيمة يؤدي أيضاً إلى تعيين هذه القيمة لخاصية ShowLegendKey لجميع تسميات البيانات في مجموعة DataLabelCollection (i.e. "DataLabels.DefaultDataLabelFormat.ShowLegendKey = val;" cause to all DataLabels[i].ShowLegendKey is equal to val).

## DataLabelFormat.ShowBubbleSize property

يمثل سلوك عرض قيمة حجم الفقاعة لتسمية البيانات في مخطط محدد. True يعرض قيمة حجم الفقاعة. False لإخفائها. قراءة/كتابة Boolean.

```csharp
public bool ShowBubbleSize { get; set; }
```

### ملاحظات

إذا كان الأصل لهذا الكائن DataLabelFormat هو مجموعة DataLabelCollection من تسمية البيانات، فإن هذه الخاصية تحصل أو تُعيّن القيمة الافتراضية لخاصية ShowBubbleSize لتسمية البيانات الجديدة في مجموعة DataLabelCollection. تعيين هذه الخاصية بالقيمة سيؤدي أيضًا إلى تعيين هذه القيمة لخاصية ShowBubbleSize لجميع تسمية البيانات في مجموعة DataLabelCollection (مثلاً "DataLabels.DefaultDataLabelFormat.ShowBubbleSize = val;" يتسبب في أن تكون جميع DataLabels[i].ShowBubbleSize مساوية للقيمة).

## خاصية DataLabelFormat.ShowPercentage

يمثل سلوك عرض قيمة النسبة المئوية لتسمية البيانات في المخطط المحدد. True تُظهر قيمة النسبة المئوية. False لإخفائها. قراءة/كتابة Boolean.

```csharp
public bool ShowPercentage { get; set; }
```

### ملاحظات

إذا كان العنصر الأب لهذا الكائن DataLabelFormat هو مجموعة DataLabelCollection من تسميات البيانات، فإن هذه الخاصية تحصل أو تعين القيمة الافتراضية لخاصية ShowPercentage لتسميات البيانات الجديدة في مجموعة DataLabelCollection. ضبط هذه الخاصية بالقيمة يضبط أيضاً هذه القيمة لخاصية ShowPercentage لجميع تسميات البيانات في مجموعة DataLabelCollection (مثال: "DataLabels.DefaultDataLabelFormat.ShowPercentage = val;" يؤدي إلى أن جميع DataLabels[i].ShowPercentage تكون مساوية للقيمة).

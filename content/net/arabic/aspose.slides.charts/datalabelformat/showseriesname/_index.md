## DataLabelFormat.ShowSeriesName الخاصية

تُرجع أو تعيين Boolean لتحديد سلوك عرض اسم السلسلة لتسميات البيانات في رسم بياني. True لإظهار اسم السلسلة. False لإخفائه. قابل للقراءة والكتابة Boolean.

```csharp
public bool ShowSeriesName { get; set; }
```

### ملاحظات

إذا كان الكائن الأب لهذا DataLabelFormat هو مجموعة DataLabelCollection من تسميات البيانات، فإن هذه الخاصية تحصل أو تعيين القيمة الافتراضية لخاصية ShowSeriesName لتسميات البيانات الجديدة في مجموعة DataLabelCollection. تعيين هذه الخاصية بالقيمة أيضاً يعين هذه القيمة لخاصية ShowSeriesName لجميع تسميات البيانات في مجموعة DataLabelCollection (أي "DataLabels.DefaultDataLabelFormat.ShowSeriesName = val;" يتسبب في أن تكون جميع DataLabels[i].ShowSeriesName مساوية للقيمة).

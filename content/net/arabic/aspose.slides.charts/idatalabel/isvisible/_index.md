## IDataLabel.IsVisible خاصية

False يعني أن تسمية البيانات غير مرئية (وبالتالي جميع Show*-flags (ShowValue, ...) تكون false). Boolean للقراءة فقط.

```csharp
public bool IsVisible { get; }
```

### ملاحظات

إذا كانت تسمية البيانات مرئية يمكنك إخفاؤها باستخدام الطريقة Hide(). ولكن إذا كانت تسمية البيانات غير مرئية (IsVisible هو false) يمكنك جعلها مرئية عن طريق ضبط Show*-flags (ShowValue, ...) إلى الحالة true.

## IChartData.SecondaryCategories الخاصية

يحصل على الفئات الثانوية إذا كانت خاصية [`UseSecondaryCategories`](../usesecondarycategories) صحيحة. للقراءة فقط [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### ملاحظات

إذا كانت خاصية [`UseSecondaryCategories`](../usesecondarycategories) خاطئة فإن خاصية `SecondaryCategories` هذه تُعيد null والبيانات في خاصية [`Categories`](../categories) تُستخدم لكل من السلاسل الأساسية والثانوية. إذا كانت خاصية [`UseSecondaryCategories`](../usesecondarycategories) صحيحة فإن البيانات في خاصية `SecondaryCategories` هذه تُستخدم للسلاسل الثانوية والبيانات في خاصية [`Categories`](../categories) تُستخدم للسلاسل الأساسية.

### أمثلة

مثال. ما الفئات المرتبطة بالسلاسل - ChartData.Categories أو ChartData.SecondaryCategories؟

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // الفئات المرتبطة هي series.Chart.ChartData.SecondaryCategories
}
else
{
    // الفئات المرتبطة هي series.Chart.ChartData.Categories
}
```

## IChartData.UseSecondaryCategories خاصية

إذا كانت false فخاصية [`SecondaryCategories`](../secondarycategories) تُعيد null والبيانات في خاصية [`Categories`](../categories) تُستخدم لكل من السلاسل الأساسية والثانوية. إذا كانت true فالبيانات في خاصية [`SecondaryCategories`](../secondarycategories) تُستخدم للسلاسل الثانوية والبيانات في خاصية [`Categories`](../categories) تُستخدم للسلاسل الأساسية. قراءة/كتابة Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

### أمثلة

مثال. ما الفئات المتعلقة بالسلسلة - ChartData.Categories أو ChartData.SecondaryCategories؟

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // الفئات المرتبطة هي series.Chart.ChartData.SecondaryCategories
}
else
{
    // الفئات المرتبطة هي series.Chart.ChartData.Categories
}
```

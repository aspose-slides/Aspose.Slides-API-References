## IChartData.Categories property

يحصل على الفئات الأساسية (أو كل من الفئات الأساسية والثانوية إذا كانت خاصية [`UseSecondaryCategories`](../usesecondarycategories) قيمتها false). للقراءة فقط [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection Categories { get; }
```

### ملاحظات

إذا كانت خاصية [`UseSecondaryCategories`](../usesecondarycategories) قيمتها false فإن خاصية [`SecondaryCategories`](../secondarycategories) تُرجِع null والبيانات في خاصية `Categories` تُستخدم لكل من السلاسل الأساسية والثانوية. إذا كانت خاصية [`UseSecondaryCategories`](../usesecondarycategories) قيمتها true فإن البيانات في خاصية [`SecondaryCategories`](../secondarycategories) تُستخدم للسلاسل الثانوية والبيانات في خاصية `Categories` تُستخدم للسلاسل الأساسية.

### أمثلة

مثال. ما هي الفئات المرتبطة بالسلسلة - ChartData.Categories أم ChartData.SecondaryCategories؟

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // الفئات المرتبطة هي series.Chart.ChartData.SecondaryCategories
}
else
{
    // الفئات المرتبطة هي series.Chart.ChartData.Categories
}
```

## ChartData.Categories الخاصية

يجلب الفئات الأساسية (أو كلاً من الفئات الأساسية والثانوية إذا كانت خاصية [`UseSecondaryCategories`](../usesecondarycategories) خاطئة). قراءة فقط [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection Categories { get; }
```

### ملاحظات

إذا كانت الخاصية [`UseSecondaryCategories`](../usesecondarycategories) خاطئة فإن الخاصية [`SecondaryCategories`](../secondarycategories) تُعيد null وتُستخدم البيانات في الخاصية `Categories` هذه لكل من السلاسل الأساسية والثانوية. إذا كانت الخاصية [`UseSecondaryCategories`](../usesecondarycategories) صحيحة فإن البيانات في الخاصية [`SecondaryCategories`](../secondarycategories) تُستخدم للسلاسل الثانوية والبيانات في الخاصية `Categories` هذه تُستخدم للسلاسل الأساسية.

### أمثلة

مثال. ما هي الفئات المرتبطة بالسلسلة - ChartData.Categories أم ChartData.SecondaryCategories؟

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // الفئات المرتبطة هي series.Chart.ChartData.SecondaryCategories
}
else
{
    // الفئات المرتبطة هي series.Chart.ChartData.Categories
}
```

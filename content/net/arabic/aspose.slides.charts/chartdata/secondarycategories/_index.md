## خاصية ChartData.SecondaryCategories

يحصل على الفئات الثانوية إذا كانت الخاصية [`UseSecondaryCategories`](../usesecondarycategories) صحيحة. للقراءة فقط [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### ملاحظات

إذا كانت خاصية [`UseSecondaryCategories`](../usesecondarycategories) خاطئة فإن خاصية `SecondaryCategories` هذه تُعيد null وتُستخدم البيانات في خاصية [`Categories`](../categories) لكل من السلاسل الأولية والثانوية. إذا كانت خاصية [`UseSecondaryCategories`](../usesecondarycategories) صحيحة فإن البيانات في خاصية `SecondaryCategories` هذه تُستخدم للسلاسل الثانوية وتُستخدم البيانات في خاصية [`Categories`](../categories) للسلاسل الأولية.

### أمثلة

مثال. ما هي الفئات المرتبطة بالسلاسل - ChartData.Categories أو ChartData.SecondaryCategories؟

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // الفئات المرتبطة هي series.Chart.ChartData.SecondaryCategories
}
else
{
    // الفئات المرتبطة هي series.Chart.ChartData.Categories
}
```

## ChartData.UseSecondaryCategories خاصية

إذا كان false فإن خاصية [`SecondaryCategories`](../secondarycategories) تُعيد null وتُستخدم البيانات في خاصية [`Categories`](../categories) لكل من السلسلة الأولية والثانوية. إذا كان true فإن البيانات في خاصية [`SecondaryCategories`](../secondarycategories) تُستخدم للسلسلة الثانوية والبيانات في خاصية [`Categories`](../categories) تُستخدم للسلسلة الأولية. قراءة/كتابة Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

### أمثلة

مثال. ما الفئات المرتبطة بالسلسلة - ChartData.Categories أو ChartData.SecondaryCategories؟

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // الفئات المرتبطة هي series.Chart.ChartData.SecondaryCategories
}
else
{
    // الفئات المرتبطة هي series.Chart.ChartData.Categories
}
```

## IDataLabelCollection.IsVisible خاصية

False يعني أن تسمية البيانات غير مرئية بشكل افتراضي (وبالتالي جميع العلامات Show*-flags (ShowValue, ...) للخاصية DefaultDataLabelFormat هي false). قراءة فقط Boolean.

```csharp
public bool IsVisible { get; }
```

### ملاحظات

إذا كانت تسمية البيانات مرئية بشكل افتراضي يمكنك إخفاؤها بشكل افتراضي باستخدام طريقة Hide(). ولكن إذا كانت تسمية البيانات غير مرئية بشكل افتراضي (IsVisible هو false) يمكنك جعل تسمية البيانات "مرئية بشكل افتراضي" عن طريق تعيين العلامات Show*-flags (ShowValue, ...) للخاصية DefaultDataLabelFormat إلى الحالة true.

## IChartCategory.UseCell الخاصية

إذا كان true فإن خاصية AsCell هي الفعلية. بمعنى آخر، يتم استخدام ورقة العمل لتخزين الفئة (هذا الوضع يدعم فئة متعددة المستويات). إذا كان false فإن خاصية AsLiteral هي الفعلية. بمعنى آخر، لا يتم استخدام ورقة العمل لتخزين الفئة (وهذا الوضع لا يدعم الفئات متعددة المستويات). Boolean للقراءة فقط.

```csharp
public bool UseCell { get; }
```

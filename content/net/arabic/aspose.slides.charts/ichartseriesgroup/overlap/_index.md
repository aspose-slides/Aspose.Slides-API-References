## IChartSeriesGroup.خاصية

يحدد مقدار تداخل الأعمدة والشرائط في المخططات الثنائية الأبعاد كنسبة مئوية (من -100% إلى 100%). - -100%: أقصى مسافة (الشرائط منفصلة تمامًا). - 0%: يتم وضع الشرائط جنبًا إلى جنب بدون تداخل أو مسافة. - 100%: أقصى تداخل (الشرائط تتداخل تمامًا مع بعضها البعض). هذه الخاصية قابلة للقراءة والكتابة بنوع SByte.

```csharp
public sbyte Overlap { get; set; }
```

### الاستثناءات

| الاستثناء | الشرط |
| --- | --- |
| ArgumentOutOfRangeException | يتم رمي الاستثناء عندما يتم تعيين القيمة خارج النطاق الصالح من -100 إلى 100. |

### أمثلة

يوضح المثال التالي كيفية تعيين التداخل لمجموعة سلسلة مخطط وعرض المخطط الناتج على نموذج:

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // تعيين التداخل إلى 55%

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

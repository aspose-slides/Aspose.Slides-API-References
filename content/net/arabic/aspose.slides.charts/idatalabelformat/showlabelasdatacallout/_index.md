## IDataLabelFormat.ShowLabelAsDataCallout خاصية

يحدد ما إذا كانت تسمية البيانات للـ chart المحدد ستُعرض كإشارة بيانات أو كتسمية بيانات. إذا كان عنصر الـ DataLabelFormat parent لهذا الكائن هو مجموعة DataLabelCollection لتسميات البيانات، فإن هذه الخاصية تحصل على أو تعيين القيمة الافتراضية للخاصية ShowLabelAsDataCallout لتسميات البيانات الجديدة في مجموعة DataLabelCollection. تعيين هذه الخاصية بقيمة يؤدي أيضًا إلى تعيين هذه القيمة للخاصية ShowLabelAsDataCallout لجميع تسميات البيانات في مجموعة DataLabelCollection (مثال: "DataLabels.DefaultDataLabelFormat.ShowLabelAsDataCallout = val;" يؤدي إلى أن جميع DataLabels[i].ShowLabelAsDataCallout تكون مساوية للقيمة).

```csharp
public bool ShowLabelAsDataCallout { get; set; }
```

## IDataLabelFormat.NumberFormat خاصية

يمثل سلسلة التنسيق ل كائن DataLabels. قراءة/كتابة String.

```csharp
public string NumberFormat { get; set; }
```

### ملاحظات

إذا كان العنصر الأب لهذا الكائن DataLabelFormat هو مجموعة DataLabelCollection من تسميات البيانات، فإن هذه الخاصية تحصل أو تعين القيمة الافتراضية للخاصية NumberFormat لتسميات البيانات الجديدة في مجموعة DataLabelCollection. عندما يتم تعيين هذه الخاصية بقيمة، يتم أيضًا تعيين تلك القيمة للخاصية NumberFormat لجميع تسميات البيانات في مجموعة DataLabelCollection (أي "DataLabels.DefaultDataLabelFormat.NumberFormat = val;" يجعل جميع DataLabels[i].NumberFormat تساوي val).

### أمثلة

```csharp
[C#]
series.Labels.DefaultDataLabelFormat.ShowValue = true;
series.Labels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = false;
series.Labels.DefaultDataLabelFormat.NumberFormat = "0.0%";
```

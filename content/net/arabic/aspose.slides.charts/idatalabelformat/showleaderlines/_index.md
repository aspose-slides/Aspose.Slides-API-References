## IDataLabelFormat.ShowLeaderLines خاصية

يمثل سلوك عرض خطوط القائد لتسمية البيانات لمخطط محدد. True يعرض خطوط القائد. False لإخفائها. قراءة/كتابة Boolean.

```csharp
public bool ShowLeaderLines { get; set; }
```

### ملاحظات

إذا كان الأصل لكائن DataLabelFormat هو مجموعة DataLabelCollection من تسميات البيانات، فإن هذه الخاصية تحصل أو تعيين القيمة الافتراضية لخاصية ShowLeaderLines لتسميات البيانات الجديدة في مجموعة DataLabelCollection. تعيين هذه الخاصية بقيمة يضبط أيضًا هذه القيمة إلى خاصية ShowLeaderLines لجميع تسميات البيانات في مجموعة DataLabelCollection (أي "DataLabels.DefaultDataLabelFormat.ShowLeaderLines = val;" يجعل جميع DataLabels[i].ShowLeaderLines يساوي val).

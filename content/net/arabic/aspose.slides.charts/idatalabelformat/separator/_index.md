## IDataLabelFormat.Separator خاصية

يضبط أو يرجع Variant يمثل الفاصل المستخدم لتسميات البيانات في المخطط. قراءة/كتابة String.

```csharp
public string Separator { get; set; }
```

### ملاحظات

إذا كان العنصر الأب لهذا الكائن DataLabelFormat هو مجموعة DataLabelCollection من تسميات البيانات، فإن هذه الخاصية تحصل أو تعين القيمة الافتراضية لخاصية Separator لتسميات البيانات الجديدة في مجموعة DataLabelCollection. تعيين هذه الخاصية بقيمة يعني أيضًا تعيين هذه القيمة لخاصية Separator لجميع تسميات البيانات في مجموعة DataLabelCollection (على سبيل المثال "DataLabels.DefaultDataLabelFormat.Separator = val;" يؤدي إلى أن جميع DataLabels[i].Separator تساوي val).

## IDataLabelFormat.ShowLabelValueFromCell خاصية

يمثل سلوك عرض قيمة خلية تسمية البيانات لمخطط محدد. True يعرض قيمة الخلية. False لإخفائها. قراءة/كتابة Boolean.

```csharp
public bool ShowLabelValueFromCell { get; set; }
```

### ملاحظات

إذا كان الوالد لهذا الكائن DataLabelFormat هو مجموعة DataLabelCollection من تسميات البيانات، فإن هذه الخاصية تحصل أو تضبط القيمة الافتراضية لخاصية ShowLabelValueFromCell للتسميات الجديدة في مجموعة DataLabelCollection. ضبط هذه الخاصية بالقيمة أيضاً يضبط هذه القيمة لخاصية ShowLabelValueFromCell لجميع تسميات البيانات في مجموعة DataLabelCollection (أي "DataLabels.DefaultDataLabelFormat.ShowLabelValueFromCell = val;" يؤدي إلى أن جميع DataLabels[i].ShowLabelValueFromCell يساوي val).

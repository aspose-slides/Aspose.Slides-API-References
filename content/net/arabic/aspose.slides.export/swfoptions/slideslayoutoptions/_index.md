يحصل أو يحدد الوضع الذي يتم فيه وضع الشرائح على الصفحة عند تصدير عرض تقديمي [`ISlidesLayoutOptions`](../../islideslayoutoptions). هذه الخاصية لا تدعم تعيين كائنات من النوع [`HandoutLayoutingOptions`](../../handoutlayoutingoptions)

### أمثلة

مثال:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    SwfOptions options = new SwfOptions
    {
        SlidesLayoutOptions = new NotesCommentsLayoutingOptions
        {
            CommentsPosition = CommentsPositions.Right
        }
    };
    
    pres.Save("pres.swf", SaveFormat.Swf, options);
}
```

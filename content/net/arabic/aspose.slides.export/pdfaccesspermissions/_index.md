## PdfAccessPermissions تعداد

Contains a set of flags specifying which access permissions should be granted when the document is opened with user access.

```csharp
[Flags]
public enum PdfAccessPermissions
```

| الاسم | القيمة | الوصف |
| --- | --- | --- |
| None | `0` | يحدد أن المستخدم لا يملك أذونات الوصول. |
| PrintDocument | `4` | يحدد ما إذا كان بمقدور المستخدم طباعة المستند (قد لا يكون بأعلى مستوى جودة، وفقًا لما إذا كانت البت HighQualityPrint مُعينة أيضًا). |
| ModifyContent | `8` | يحدد ما إذا كان بإمكان المستخدم تعديل محتويات المستند عبر عمليات غير تلك التي تتحكم بها البتات AddOrModifyFields و FillExistingFields و AssembleDocument. |
| CopyTextAndGraphics | `10` | يحدد ما إذا كان بإمكان المستخدم نسخ أو استخراج النص والرسومات من المستند عبر عمليات غير تلك التي تتحكم بها البت ExtractTextAndGraphics. |
| AddOrModifyFields | `20` | يحدد ما إذا كان بإمكان المستخدم إضافة أو تعديل التعليقات النصية، تعبئة حقول النماذج التفاعلية، وإذا كانت البت ModifyContent مُعينة أيضًا، إنشاء أو تعديل حقول النماذج التفاعلية (بما في ذلك حقول التوقيع). |

| ExtractTextAndGraphics | `200` | يحدد ما إذا كان بإمكان المستخدم استخراج النص والرسومات لدعم إمكانية الوصول للمستخدمين ذوي الإعاقات أو لأغراض أخرى. |
| AssembleDocument | `400` | يحدد ما إذا كان بإمكان المستخدم تجميع المستند (إدراج أو تدوير أو حذف الصفحات وإنشاء إشارات مرجعية أو صور مصغرة)، حتى وإن كانت البت ModifyContent غير مفعلة. |
| HighQualityPrint | `800` | يحدد ما إذا كان بإمكان المستخدم طباعة المستند إلى تمثيل يمكن من خلاله إنشاء نسخة رقمية دقيقة لمحتوى PDF. عندما تكون هذه البت غير مفعلة (وتكون البت PrintDocument مفعلة)، يقتصر الطباعة على تمثيل منخفض المستوى للمظهر، قد يكون بجودة منخفضة. |

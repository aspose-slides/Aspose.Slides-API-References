## SVGOptions.DisableFontLigatures خاصية

يحصل أو يعيّن قيمة تشير إلى ما إذا كان النص يُعرض دون استخدام الروابط الحرفية. عند تعيينه إلى `true`، سيتم تعطيل الروابط الحرفية في الإخراج المرسوم. بشكل افتراضي، يتم تعيين هذه الخاصية إلى `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### أمثلة

مثال:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    SVGOptions options = new SVFOptions
    {
        DisableFontLigatures = true // تعطيل الروابط الحرفية في عرض النص
    };
    
    using (FileStream fileStream = new FileStream("slide-0.svg", FileMode.Create, FileAccess.Write))
    {
        pres.Slides[0].WriteAsSvg(fileStream);   
    }
}
```

## خاصية RenderingOptions.DisableFontLigatures

يتم الحصول على قيمة أو تعيينها لتحديد ما إذا كان يتم عرض النص دون استخدام الروابط الأحرف. عند تعيينها إلى `true`، سيتم تعطيل الروابط الأحرف في المخرجات المعروضة. بشكل افتراضي، يتم تعيين هذه الخاصية إلى `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### أمثلة

مثال:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    RenderingOptions options = new RenderingOptions
    {
        DisableFontLigatures = true // تعطيل الروابط الأحرف في عرض النص
    };
    
    Bitmap[] renderedSlides = pres.GetImage(options);
    for (var index = 0; index < renderedSlides.Length; index++)
    {
        var slideImage = renderedSlides[index];
        slideImage.Save($"slide-{index}.png");
    }
}
```

## IPptxOptions.CompressionLevel property

يحدد مستوى الضغط المستخدم عند حفظ مستند العرض التقديمي. القيمة الافتراضية هي Level6.

```csharp
public CompressionLevel CompressionLevel { get; set; }
```

### ملاحظات

تنتج مستويات الضغط الأعلى ملفات أصغر ولكنها تتطلب وقت معالجة أطول. يعتمد معدل الضغط الفعلي على محتوى العرض التقديمي.

### أمثلة

مثال:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("demo-level8.pptx", SaveFormat.Pptx, new PptxOptions()
    {
        CompressionLevel = CompressionLevel.Level8
    });
}
```

## Html5Options.DisableFontLigatures الخاصية

يُحصل على أو يحدد قيمة تشير إلى ما إذا كان النص يُعرض دون استخدام الأحرف المتصلة. عندما يتم تعيينه إلى `true`، سيتم تعطيل الأحرف المتصلة في ناتج العرض. بشكل افتراضي، يتم تعيين هذه الخاصية إلى `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### الأمثلة

مثال:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    Html5Options options = new Html5Options
    {
        DisableFontLigatures = true // تعطيل الأحرف المتصلة في عرض النص
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html5, options);
}
```

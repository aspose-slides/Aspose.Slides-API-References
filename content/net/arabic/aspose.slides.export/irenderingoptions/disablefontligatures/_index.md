## IRenderingOptions.DisableFontLigatures الخاصية

تُعيد أو تُعيّن قيمة تُشير إلى ما إذا كان النص يُعرض دون استخدام الأحرف المتصلة. عندما تُعيّن إلى `true`، سيتم تعطيل الأحرف المتصلة في الإخراج المعروض. بشكل افتراضي، تُعيّن هذه الخاصية إلى `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### أمثلة

مثال:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    RenderingOptions options = new RenderingOptions
    {
        DisableFontLigatures = true // تعطيل الأحرف المتصلة في عرض النص
    };
    
    Bitmap[] renderedSlides = pres.GetImage(options);
    for (var index = 0; index < renderedSlides.Length; index++)
    {
        var slideImage = renderedSlides[index];
        slideImage.Save($"slide-{index}.png");
    }
}
```

## IHtmlOptions.DisableFontLigatures خاصية

يحصل أو يعيّن قيمة تشير إلى ما إذا كان النص يُعرض دون استخدام الروابط الأحرفية. عند ضبطها على `true`، سيتم تعطيل الروابط الأحرفية في الناتج المعروض. بشكل افتراضي، يتم ضبط هذه الخاصية على `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### أمثلة

مثال:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    HtmlOptions options = new HtmlOptions
    {
        DisableFontLigatures = true // تعطيل الروابط الأحرفية في عرض النص
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html, options);
}
```

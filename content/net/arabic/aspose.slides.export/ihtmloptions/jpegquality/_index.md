## IHtmlOptions.JpegQuality خاصية

تُعيد أو تعين قيمة تحدد جودة صور JPEG داخل مستند PDF. قراءة/كتابة Byte.

```csharp
public byte JpegQuality { get; set; }
```

### ملاحظات

يُطبق فقط عندما يحتوي المستند على صور JPEG.

استخدم هذه الخاصية للحصول على أو تعيين جودة الصور داخل المستند عند حفظه بصيغة PDF. قد تتراوح القيمة من 0 إلى 100 حيث يعني 0 أسوأ جودة ولكن أقصى ضغط و 100 يعني أفضل جودة ولكن أقل ضغط.

القيمة الافتراضية هي **95**.

## IPdfOptions.JpegQuality خاصية

إرجاع أو تعيين قيمة تحدد جودة صور JPEG داخل مستند PDF. قراءة/كتابة Byte.

```csharp
public byte JpegQuality { get; set; }
```

### ملاحظات

يؤثر فقط عندما يحتوي المستند على صور JPEG.

استخدم هذه الخاصية للحصول على أو تعيين جودة الصور داخل المستند عند الحفظ بصيغة PDF. قد تتراوح القيمة من 0 إلى 100 حيث يعني 0 أسوأ جودة مع أقصى ضغط و100 يعني أفضل جودة مع أقل ضغط.

القيمة الافتراضية هي **100**.

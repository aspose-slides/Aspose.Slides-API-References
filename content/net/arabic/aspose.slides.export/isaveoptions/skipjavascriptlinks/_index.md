## ISaveOptions.SkipJavaScriptLinks خاصية

يحدد ما إذا كان يجب تخطي الروابط التشعبية التي تستدعي JavaScript عند حفظ العرض التقديمي. قراءة/كتابة Boolean. القيمة الافتراضية هي **false**.

```csharp
public bool SkipJavaScriptLinks { get; set; }
```

### ملاحظات

عند ضبط هذه الخاصية على **true**, سيتم تجاهل الروابط التشعبية التي تستدعي JavaScript أثناء الحفظ.

عند ضبط هذه الخاصية على **false**, سيتم حفظ جميع الروابط التشعبية.

### أمثلة

مثال:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("result_without_JavaScript_links.html", SaveFormat.Html, new HtmlOptions()
    {
        SkipJavaScriptLinks = true
    });
}
```

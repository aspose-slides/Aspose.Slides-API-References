## IGifOptions.DefaultDelay الخاصية

يحصل أو يضبط وقت التأخير الافتراضي [ms]. ستُستخدم هذه القيمة إذا لم يتم تعيين [`AdvanceAfterTime`](../../../aspose.slides/islideshowtransition/advanceaftertime). القيمة الافتراضية هي 1000.

```csharp
public int DefaultDelay { get; set; }
```

### أمثلة

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
	pres.Save("pres.gif", SaveFormat.Gif, new GifOptions { DefaultDelay = 2000 });
}
```

## ILinkEmbedController.GetUrl طريقة

يرجع عنوان URL إلى كائن خارجي. هذه الطريقة تُستدعى دائمًا إذا كان [`GetObjectStoringLocation`](../getobjectstoringlocation) قد أعاد Link وقد تُستدعى إذا كان [`GetObjectStoringLocation`](../getobjectstoringlocation) قد أعاد Embed لكن التضمين غير ممكن. يمكن استدعاؤها عدة مرات لنفس معرف الكائن.

```csharp
public string GetUrl(int id, int referrer)
```

| المعامل | النوع | الوصف |
| --- | --- | --- |
| id | Int32 | معرف الكائن. هذا المعرف فريد على مستوى العملية. |
| referrer | Int32 | معرف الكائن المرجعي أو 0، إذا كان الكائن مراجعًا من المستند الجذر. قد يُستخدم لتوليد رابط نسبي. |

### قيمة الإرجاع

عنوان URL للكائن الخارجي أو null إذا كان يجب تجاهل هذا الكائن.

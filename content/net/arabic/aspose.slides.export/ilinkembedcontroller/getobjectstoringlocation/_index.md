## ILinkEmbedController.GetObjectStoringLocation طريقة

يحدد أين يجب تخزين الكائن. تُستدعى هذه الطريقة مرة واحدة لكل معرف كائن. لا يُضمن عدم وجود كائنين يحملان نفس البيانات وsemanticName وcontentType لكن بمعرف مختلف.

```csharp
public LinkEmbedDecision GetObjectStoringLocation(int id, byte[] entityData, string semanticName, 
    string contentType, string recomendedExtension)
```

| المعامل | النوع | الوصف |
| --- | --- | --- |
| id | Int32 | معرف الكائن. هذا المعرف فريد على مستوى العملية. |
| entityData | Byte[] | بيانات الكائن الثنائية. يمكن أن تكون هذه المعلمة فارغة إذا لم يتم إنشاء البيانات الثنائية للكائن بعد. |
| semanticName | String | نص قصير يصف معنى الكائن. قد يستخدم المتحكم هذا كجزء من اسم الكائن الخارجي، ولكن الأمر متروك للناشر لضمان أن الأسماء فريدة وتحتوي فقط على الأحرف المسموح بها. |
| contentType | String | نوع MIME للكائن. |
| recomendedExtension | String | امتداد اسم الملف الموصى به لهذا النوع من MIME. |

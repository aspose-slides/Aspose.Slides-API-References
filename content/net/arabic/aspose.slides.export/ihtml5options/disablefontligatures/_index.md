## IHtml5Options.DisableFontLigatures خاصية

يُعيد أو يضبط قيمة تشير إلى ما إذا كان النص يُعرض دون استخدام الروابط الأحرفية. عند ضبطه على `true`، سيتم إلغاء الروابط الأحرفية في المخرجات المُعروضة. بشكل افتراضي، يتم ضبط هذه الخاصية على `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### أمثلة

مثال:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    Html5Options options = new Html5Options
    {
        DisableFontLigatures = true // تعطيل الروابط الأحرفية في عرض النص
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html5, options);
}
```

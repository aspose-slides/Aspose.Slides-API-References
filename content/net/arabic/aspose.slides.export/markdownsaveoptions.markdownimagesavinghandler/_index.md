## MarkdownSaveOptions.MarkdownImageSavingHandler المندوب

يتم استدعاؤه لكل صورة غير SVG (bitmap أو metafile) أثناء تصدير Markdown. أعد `true` لاستخدام *الرابط* المحدد، أو `false` لتطبيق منطق الحفظ الافتراضي.

public delegate bool MarkdownImageSavingHandler(IImage image, ImageFormat format, ref string link);

| format | ImageFormat | تنسيق الصورة. |
| link | String& | رابط Markdown لاستخدامه عند إرجاع `true`. |

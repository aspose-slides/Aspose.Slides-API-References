## HtmlOptions.DisableFontLigatures خاصية

يحصل أو يعيّن قيمة تشير إلى ما إذا كان النص يُعرض دون استخدام الروابط الأحرفية. عند تعيينه إلى `true`، سيتم تعطيل الروابط الأحرفية في الناتج المُعرض. بشكل افتراضي، يتم تعيين هذه الخاصية إلى `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### أمثلة

مثال:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    HtmlOptions options = new HtmlOptions
    {
        DisableFontLigatures = true // تعطيل الروابط الأحرفية في عرض النص
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html, options);
}
```

## OpenAIWebClient(string, string, string, HttpClient) {#constructor_1}

```csharp
public OpenAIWebClient(string model, string apiKey, string organizationId, HttpClient httpClient)
```

| المعامل | النوع | الوصف |
| --- | --- | --- |
| model | String | نموذج لغة OpenAI. القيم المحتملة: - gpt-4o - gpt-4o-mini - o1 - o1-mini - o3 - o3-mini |
| apiKey | String | مفتاح API الخاص بـ OpenAI |
| organizationId | String | معرف المؤسسة (اختياري) |
| httpClient | HttpClient | مثيل `HttpClient` مُدار خارجيًا |

### الاستثناءات

| الاستثناء | الشرط |
| --- | --- |
| ArgumentException | قيمة مفتاح API لا يمكن أن تكون فارغة أو null |

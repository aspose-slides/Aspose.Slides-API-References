## Ink.InkEffectImages الخاصية

يحصل على مجموعة الصور المخصَّصة المستخدمة لمحاكاة التأثيرات البصرية لفرش الحبر. تُستخدم هذه الصور عند عرض الحبر باستخدام قيم [`InkEffectType`](../../inkeffecttype) محددة، مثل Galaxy و Rainbow وغيرها. من خلال توفير صورك الخاصة، يمكنك التحكم في كيفية ظهور كل تأثير حبر.

```csharp
public static IDictionary<InkEffectType, IImage> InkEffectImages { get; }
```

### ملاحظات

تسمح هذه الخاصية باستبدال القوام الافتراضية لتأثيرات الحبر بملفات تعريفية يحددها المستخدم، وهو أمر مفيد بشكل خاص عندما تكون الأصول الافتراضية مقيدة بترخيص أو غير متاحة أثناء التشغيل. يجب أن يرتبط كل إدخال في القاموس بقيمة [`InkEffectType`](../../inkeffecttype) مع كائن [`IImage`](../../../aspose.slides/iimage) مقابل (مثال: Bitmap أو واجهة صورة Aspose).

### أمثلة

```csharp
[C#]
IImage image = Images.FromFile("image.png");
Ink.InkEffectImages.Add(InkEffectType.Galaxy, image);
```

## IInkBrush.InkEffect خاصية

يتم الحصول على نوع تأثير الحبر (مثلاً Galaxy، Gold، Silver) الذي يحدد النمط البصري لضربة الحبر. تُستخلص القيمة من خاصية الفرشاة "inkEffects". إذا لم يُحدَّد أي تأثير معترف به، يتم إرجاع NotDefined.

```csharp
public InkEffectType InkEffect { get; }
```

### أمثلة

مثال:

```csharp
[C#]
using (var pres = new Presentation("Presentation.pptx"))
{
    Ink ink = pres.Slides[0].Shapes[0] as Ink;
    IInkBrush brush = ink.Traces[0].Brush;
    Console.WriteLine("InkEffects = {0}", brush.InkEffect);
}
```

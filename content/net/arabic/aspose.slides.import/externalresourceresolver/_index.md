## ExternalResourceResolver الفئة

فئة رد نداء تُستخدم لحل الموارد الخارجية أثناء استيراد مستندات Html وSvg. قد يؤدي استخدام هذا المحلل إلى خلق ثغرة عندما يقدم العميل ملف HTML أو SVG يجعل برنامج الخادم يحصل على ملف محلي أو شبكي. استخدمه بحذر. يُوصى بعدم تحديد ExternalResourceResolver على الإطلاق (سيتم قراءة الكائنات المضمنة فقط) أو إنشاء فئة فرعية تتحقق مما إذا كان الـ uri المحدد صالحًا.

```csharp
public class ExternalResourceResolver : IExternalResourceResolver
```

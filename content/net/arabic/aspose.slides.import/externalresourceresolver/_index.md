## ExternalResourceResolver فئة

فئة رد نداء تُستَخدم لحل الموارد الخارجية أثناء استيراد مستندات Html و Svg. قد يؤدي استخدام هذا المحلّل إلى خلق ثغرة عندما يُزود العميل بملف HTML أو SVG يجعل برنامج الخادم يحصل على ملف محلي أو عبر الشبكة. استخدمه بحذر. يُنصَح بعدم تحديد ExternalResourceResolver على الإطلاق (ستُقراً فقط الكائنات المدمجة) أو إنشاء فئة فرعية تتحقق مما إذا كان الـ uri المحدد صالحًا.

```csharp
public class ExternalResourceResolver : IExternalResourceResolver
```

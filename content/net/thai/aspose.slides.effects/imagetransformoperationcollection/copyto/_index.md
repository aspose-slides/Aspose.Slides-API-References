```csharp
public void CopyTo(IImageTransformOperation[] array, int arrayIndex)
```

| Parameter | Type | Description |
| --- | --- | --- |
| array | IImageTransformOperation[] | Array แบบหนึ่งมิติที่เป็นปลายทางขององค์ประกอบที่คัดลอกจาก ICollection. Array ต้องมีการจัดทำดัชนีแบบเริ่มจากศูนย์. |
| arrayIndex | Int32 | ดัชนีที่เริ่มจากศูนย์ใน *array* ที่การคัดลอกเริ่มต้น. |

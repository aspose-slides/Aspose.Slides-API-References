มีคลาสสำหรับทำงานกับเอฟเฟกต์ต่าง ๆ ในการนำเสนอ Microsoft PowerPoint

## คลาส

| คลาส | คำอธิบาย |
| --- | --- |
| [AlphaBiLevel](./alphabilevel) | แสดงถึงเอฟเฟกต์ Alpha Bi-Level. ค่าความทึบ (Opacity) ที่น้อยกว่าค่าธรณีจะถูกเปลี่ยนเป็น 0 (โปร่งใสเต็มที่) และค่าความทึบที่เท่ากับหรือมากกว่าค่าธรณีจะถูกเปลี่ยนเป็น 100% (ทึบเต็มที่). |
| [AlphaCeiling](./alphaceiling) | แสดงถึงเอฟเฟกต์ Alpha Ceiling. ค่าความทึบที่มากกว่า 0 จะถูกเปลี่ยนเป็น 100%. กล่าวคือ สิ่งที่มีความทึบบางส่วนจะกลายเป็นทึบเต็มที่. |
| [AlphaFloor](./alphafloor) | แสดงถึงเอฟเฟกต์ Alpha Floor. ค่าความทึบที่น้อยกว่า 100% จะถูกเปลี่ยนเป็นศูนย์. กล่าวคือ สิ่งที่มีความโปร่งใสบางส่วนจะกลายเป็นโปร่งใสเต็มที่. |
| [AlphaInverse](./alphainverse) | แสดงถึงเอฟเฟกต์ Alpha Inverse. ค่าความทึบจะถูกกลับโดยการลบจาก 100%. |
| [AlphaModulate](./alphamodulate) | แสดงถึงเอฟเฟกต์ Alpha Modulate. ค่าความทึบของเอฟเฟกต์จะถูกคูณด้วยเปอร์เซ็นต์คงที่. คอนเทนเนอร์ของเอฟเฟกต์ระบุเอฟเฟกต์ที่มีค่าความทึบเพื่อทำการปรับเปลี่ยน. |
| [AlphaModulateFixed](./alphamodulatefixed) | แสดงถึงเอฟเฟกต์ Alpha Modulate Fixed. ค่าความทึบของเอฟเฟกต์จะถูกคูณด้วยเปอร์เซ็นต์คงที่. |
| [AlphaReplace](./alphareplace) | แสดงถึงเอฟเฟกต์ Alpha Replace. ค่าความทึบของเอฟเฟกต์จะถูกแทนที่ด้วยความทึบคงที่. |
| [BiLevel](./bilevel) | แสดงถึงเอฟเฟกต์ Bi-Level (สีดำ/สีขาว). สีอินพุตที่มีค่าความสว่างน้อยกว่าค่าธรณีที่กำหนดจะถูกเปลี่ยนเป็นสีดำ. สีอินพุตที่มีค่าความสว่างเท่ากับหรือมากกว่าค่าธรณีจะถูกตั้งเป็นสีขาว. ค่าความทึบของเอฟเฟกต์ไม่ได้รับผลกระทบจากเอฟเฟกต์นี้. |
| [Blur](./blur) | แสดงถึงเอฟเฟกต์ Blur ที่ใช้กับรูปทรงทั้งหมดรวมถึงการเติม. ทุกช่องสีรวมถึงความทึบก็จะได้รับผลกระทบ. |
| [BrightnessContrast](./brightnesscontrast) | แสดงถึงเอฟเฟกต์ BrightnessContrast. ปรับความสว่างและความคอนทราสต์ |
| [ColorChange](./colorchange) | แสดงถึงเอฟเฟกต์ Color Change. ตัวอย่างของ FromColor จะถูกแทนที่ด้วยตัวอย่างของ ToColor. |
| [ColorReplace](./colorreplace) | แสดงถึงเอฟเฟกต์ Color Replacement. สีทั้งหมดของเอฟเฟกต์จะถูกเปลี่ยนเป็นสีคงที่. ค่าความทึบไม่ได้รับผลกระทบ. |
| [Duotone](./duotone) | แสดงถึงเอฟเฟกต์ Duotone. สำหรับพิกเซลแต่ละพิกเซล จะผสม Color1 และ Color2 ด้วยการเชิงเส้นเพื่อกำหนดสีใหม่ของพิกเซลนั้น. |
| [EffectFactory](./effectfactory) | อนุญาตให้สร้างเอฟเฟกต์ |
| [FillOverlay](./filloverlay) | แสดงถึงเอฟเฟกต์ Fill Overlay. Fill overlay สามารถใช้ระบุการเติมเพิ่มเติมสำหรับวัตถุและผสานการเติมทั้งสองเข้าด้วยกัน. |
| [Glow](./glow) | แสดงถึงเอฟเฟกต์ Glow ซึ่งเป็นการเพิ่มเส้นขอบสีที่เบลออยู่ด้านนอกของวัตถุ. |
| [GrayScale](./grayscale) | แสดงถึงเอฟเฟกต์ Gray Scale. แปลงค่าสีทั้งหมดของเอฟเฟกต์เป็นเฉดสีเทาตามค่าความสว่างของมัน. ค่าความทึบของเอฟเฟกต์ไม่ได้รับผลกระทบ. |
| [HSL](./hsl) | แสดงถึงเอฟเฟกต์ Hue/Saturation/Luminance. ค่าความสี (hue), ความอิ่ม (saturation) และความสว่าง (luminance) สามารถปรับได้โดยอิงจากค่าปัจจุบันของแต่ละอย่าง. |
| [ImageTransformOCollectionEffectiveData](./imagetransformocollectioneffectivedata) | ออบเจกต์ที่ไม่เปลี่ยนแปลงซึ่งแสดงถึงคอลเลกชันแบบอ่านอย่างเดียวของเอฟเฟกต์การแปลงภาพที่มีผล. |
| [ImageTransformOperation](./imagetransformoperation) | แสดงถึงเอฟเฟกต์การแปลงภาพแบบนามธรรม. |
| [ImageTransformOperationCollection](./imagetransformoperationcollection) | แสดงถึงคอลเลกชันของเอฟเฟกต์ที่ใช้กับภาพ. |
| [ImageTransformOperationFactory](./imagetransformoperationfactory) | อนุญาตให้สร้างการดำเนินการแปลงภาพ |
| [InnerShadow](./innershadow) | แสดงถึงเอฟเฟกต์ Inner Shadow. |
| [Luminance](./luminance) | แสดงถึงเอฟเฟกต์ Luminance. ความสว่างจะทำการเปลี่ยนสีทั้งหมดให้ใกล้สีขาวหรือสีดำอย่างเชิงเส้น. ความคอนทราสต์จะปรับสเกลสีทั้งหมดให้ใกล้หรือห่างกันมากขึ้น. |
| [OuterShadow](./outershadow) | แสดงถึงเอฟเฟกต์ Outer Shadow. |
| [PresetShadow](./presetshadow) | แสดงถึงเอฟเฟกต์ Preset Shadow. |
| [Reflection](./reflection) | แสดงถึงเอฟเฟกต์ Reflection. |
| [SoftEdge](./softedge) | แสดงถึงเอฟเฟกต์ Soft Edge. ขอบของรูปทรงจะเบลอในขณะที่การเติมไม่ได้รับผลกระทบ. |
| [Tint](./tint) | แสดงถึงเอฟเฟกต์ Tint. ปรับค่าสีของเอฟเฟกต์ให้เคลื่อนเข้าหรือออกจาก hue ตามจำนวนที่ระบุ. |

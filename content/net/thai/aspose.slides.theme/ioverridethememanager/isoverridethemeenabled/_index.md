## IOverrideThemeManager.IsOverrideThemeEnabled คุณสมบัติ

กำหนดว่า OverrideTheme จะทำการแทนที่ธีมที่มีประสิทธิภาพที่สืบทอดหรือไม่. เพื่อเปิดใช้งาน OverrideTheme สำหรับการแทนที่ให้ใช้เมธอด OverrideTheme.Init*() เมธอด. เพื่อปิดการแทนที่ OverrideTheme ให้ใช้เมธอด OverrideTheme.Clear() เมธอด. อ่านอย่างเดียว Boolean.

```csharp
public bool IsOverrideThemeEnabled { get; }
```

## SlideUtil.FindAndReplaceText เมธอด

ค้นหาและแทนที่ข้อความในงานนำเสนอด้วยรูปแบบที่กำหนด

```csharp
public static void FindAndReplaceText(IPresentation presentation, bool withMasters, string find, 
    string replace, PortionFormat format = null)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| presentation | IPresentation | งานนำเสนอที่สแกน |
| withMasters | Boolean | กำหนดว่าควรสแกนสไลด์แม่หรือไม่ |
| find | String | ค่าข้อความที่จะค้นหา |
| replace | String | ค่าข้อความที่จะแทนที่ |
| format | PortionFormat | รูปแบบสำหรับการแทนที่ส่วนของข้อความ หากเป็น null จะใช้รูปแบบของอักขระตัวแรกของสตริงที่พบ |

### ตัวอย่าง

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PortionFormat format = new PortionFormat
    {
        FontHeight = 24f,
        FontItalic = NullableBool.True,
        FillFormat =
        {
            FillType = FillType.Solid,
            SolidFillColor =
            {
                Color = Color.Red
            }
        }
    };
    Aspose.Slides.Util.SlideUtil.FindAndReplaceText(pres, true, "[this block] ", "my text ", format);
    pres.Save("replaced", SaveFormat.Pptx);
}
```

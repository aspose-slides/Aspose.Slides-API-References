## IExcelDataWorkbook.GetCells เมธอด

ดึงคอลเลกชันของเซลล์จากเวิร์กบุ๊คที่ตรงกับสูตรที่ระบุ

```csharp
public ReadOnlyCollection<IExcelDataCell> GetCells(string formula, bool skipHiddenCells)
```

| พารามิเตอร์ | ชนิด | คำอธิบาย |
| --- | --- | --- |
| formula | String | สูตรหรือการระบุช่วง (เช่น "Sheet1!A1:B3") ที่ใช้เพื่อระบุเซลล์เป้าหมาย |
| skipHiddenCells | Boolean | หาก `true` เซลล์ที่ซ่อนอยู่ (เช่น ในแถวหรือคอลัมน์ที่ซ่อน) จะถูกตัดออกจากผลลัพธ์ |

### Return Value

รายการเซลล์แบบอ่านอย่างเดียวที่ตรงกับสูตรที่ระบุ

### Examples

ตัวอย่าง:

```csharp
[C#]
ExcelDataWorkbook wb = new ExcelDataWorkbook(testFile);
IReadOnlyList<IExcelDataCell> cells = wb.GetCells("Sheet1!A2:A6", false);
Console.WriteLine(cells.Count); //ผลลัพธ์: 5
```

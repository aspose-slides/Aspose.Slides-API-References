## Ink.InkEffectImages คุณสมบัติ

รับคอลเลกชันของภาพที่กำหนดเองที่ใช้เพื่อจำลองเอฟเฟกต์ภาพสำหรับแปรงหมึก ภาพเหล่านี้จะถูกใช้เมื่อเรนเดอร์หมึกด้วยค่า [`InkEffectType`](../../inkeffecttype) เฉพาะ เช่น Galaxy, Rainbow เป็นต้น โดยการจัดหาภาพของคุณเอง คุณสามารถควบคุมลักษณะของเอฟเฟกต์หมึกแต่ละแบบได้

```csharp
public static IDictionary<InkEffectType, IImage> InkEffectImages { get; }
```

### หมายเหตุ

คุณสมบัตินี้ทำให้สามารถแทนที่พื้นผิวเอฟเฟกต์หมึกเริ่มต้นด้วยภาพที่กำหนดโดยผู้ใช้ ซึ่งเป็นประโยชน์อย่างยิ่งเมื่อทรัพยากรเริ่มต้นถูกจำกัดโดยลิขสิทธิ์หรือไม่มีในเวลารันไทม์ แต่ละรายการในพจนานุกรมต้องเชื่อมโยงค่า [`InkEffectType`](../../inkeffecttype) กับอ็อบเจ็กต์ [`IImage`](../../../aspose.slides/iimage) ที่สอดคล้องกัน (เช่น Bitmap หรืออินเทอร์เฟซภาพของ Aspose)

```csharp
[C#]
IImage image = Images.FromFile("image.png");
Ink.InkEffectImages.Add(InkEffectType.Galaxy, image);
```

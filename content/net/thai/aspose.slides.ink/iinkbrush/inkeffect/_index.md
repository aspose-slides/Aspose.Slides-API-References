## IInkBrush.InkEffect คุณสมบัติ

รับประเภทเอฟเฟกต์สีหมึก (เช่น Galaxy, Gold, Silver) ที่กำหนดรูปแบบการแสดงผลของเส้นหมึก ค่าจะถูกแยกวิเคราะห์จากคุณสมบัติ brush "inkEffects". หากไม่มีเอฟเฟกต์ที่รู้จักระบุไว้ จะคืนค่า NotDefined.

```csharp
public InkEffectType InkEffect { get; }
```

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]
using (var pres = new Presentation("Presentation.pptx"))
{
    Ink ink = pres.Slides[0].Shapes[0] as Ink;
    IInkBrush brush = ink.Traces[0].Brush;
    Console.WriteLine("InkEffects = {0}", brush.InkEffect);
}
```

## ILinkEmbedController.GetUrl เมธอด

คืน URL ไปยังอ็อบเจ็กต์ภายนอก. เมธอดนี้จะถูกเรียกเสมอเมื่อ [`GetObjectStoringLocation`](../getobjectstoringlocation) คืนค่า Link และอาจถูกเรียกเมื่อ [`GetObjectStoringLocation`](../getobjectstoringlocation) คืนค่า Embed แต่ไม่สามารถฝังได้. สามารถเรียกหลายครั้งสำหรับรหัสอ็อบเจ็กต์เดียวกัน.

```csharp
public string GetUrl(int id, int referrer)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| id | Int32 | รหัสอ็อบเจ็กต์. รหัสนี้ถูกบันทึกให้เป็นเอกลักษณ์ทั่วทั้งการดำเนินการ. |
| referrer | Int32 | รหัสของอ็อบเจ็กต์อ้างอิงหรือ 0 หากอ็อบเจ็กต์ถูกอ้างอิงโดยเอกสารราก. อาจใช้เพื่อสร้างลิงก์สัมพันธ์. |

### ค่าที่ส่งกลับ

URL ของอ็อบเจ็กต์ภายนอกหรือ null หากอ็อบเจ็กต์นี้ควรถูกละเว้น.

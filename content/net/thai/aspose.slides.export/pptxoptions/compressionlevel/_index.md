## PptxOptions.CompressionLevel คุณสมบัติ

ระบุระดับการบีบอัดที่ใช้เมื่อบันทึกเอกสารงานนำเสนอ ค่าปริยายคือ Level6.

```csharp
public CompressionLevel CompressionLevel { get; set; }
```

### หมายเหตุ

ระดับการบีบอัดที่สูงกว่าจะทำให้ไฟล์มีขนาดเล็กลงแต่ต้องใช้เวลาประมวลผลมากขึ้น อัตราส่วนการบีบอัดจริงขึ้นอยู่กับเนื้อหาของงานนำเสนอ.

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("demo-level8.pptx", SaveFormat.Pptx, new PptxOptions()
    {
        CompressionLevel = CompressionLevel.Level8
    });
}
```

## PptxOptions.RefreshThumbnail คุณสมบัติ

ระบุว่าภาพย่อของการนำเสนอจะถูกรีเฟรชหรือไม่ อ่าน/เขียน Boolean ค่าเริ่มต้นคือ **true**.

```csharp
public bool RefreshThumbnail { get; set; }
```

### หมายเหตุ

เมื่อค่าตัวเลือกเป็น **true** จะสร้างภาพย่อใหม่

เมื่อค่าตัวเลือกเป็น **false** ภาพย่อปัจจุบันจะถูกบันทึกตามเดิม

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("result_with_old_thumbnail.pptx", SaveFormat.Pptx, new PptxOptions()
    {
        RefreshThumbnail = false
    });
}
```

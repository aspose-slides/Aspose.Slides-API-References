## IHtmlOptions.JpegQuality คุณสมบัติ

คืนหรือกำหนดค่าที่ระบุคุณภาพของภาพ JPEG ภายในเอกสาร PDF. อ่าน/เขียน Byte.

```csharp
public byte JpegQuality { get; set; }
```

### หมายเหตุ

มีผลเฉพาะเมื่อเอกสารมีภาพ JPEG.

ใช้คุณสมบัตินี้เพื่อรับหรือกำหนดคุณภาพของภาพภายในเอกสารเมื่อต้องการบันทึกเป็นรูปแบบ PDF. ค่าที่กำหนดอาจอยู่ระหว่าง 0 ถึง 100 โดยที่ 0 แสดงถึงคุณภาพแย่ที่สุดแต่การบีบอัดสูงสุดและ 100 แสดงถึงคุณภาพดีที่สุดแต่การบีบอัดต่ำสุด.

ค่าเริ่มต้นคือ **95**.

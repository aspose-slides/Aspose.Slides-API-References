## IHtmlOptions.DisableFontLigatures คุณสมบัติ

รับหรือกำหนดค่าที่บ่งชี้ว่าข้อความถูกเรนเดอร์โดยไม่ใช้ลิเกเจอร์ เมื่อกำหนดเป็น `true` ลิเกเจอร์จะถูกปิดในผลลัพธ์ที่แสดงโดยค่าเริ่มต้น คุณสมบัตินี้ตั้งค่าเป็น `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    HtmlOptions options = new HtmlOptions
    {
        DisableFontLigatures = true // ปิดลิเกเจอร์ในการเรนเดอร์ข้อความ
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html, options);
}
```

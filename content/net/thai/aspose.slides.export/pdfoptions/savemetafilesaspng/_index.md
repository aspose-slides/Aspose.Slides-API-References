## PdfOptions.SaveMetafilesAsPng คุณสมบัติ

True เพื่อแปลงเมตาฟายล์ทั้งหมดที่ใช้ในงานนำเสนอเป็นภาพ PNG. อ่าน/เขียน Boolean.

```csharp
public bool SaveMetafilesAsPng { get; set; }
```

### หมายเหตุ

ค่าเริ่มต้นคือ **true**. เอกสาร Pdf สามารถมีกราฟิกเวคเตอร์และภาพแรสเตอร์.  
หาก SaveMetafilesAsPng ถูกตั้งค่าเป็น true จากนั้นภาพ Metafile ต้นฉบับจะถูกแปลงเป็นรูปแบบ Png และบันทึกลงใน Pdf เป็นภาพแรสเตอร์.  
หาก SaveMetafilesAsPng ถูกตั้งค่าเป็น false จากนั้น Metafile ต้นฉบับจะถูกแปลงเป็นกราฟิกเวคเตอร์ของ Pdf.  
แต่ละวิธีมีข้อดีและข้อเสีย. ตัวอย่างเช่น หาก Metafile ถูกแปลงเป็น PNG อาจเกิดการสูญเสียคุณภาพบางส่วนระหว่างการปรับขนาดของเอกสารที่ได้. หาก Metafile ถูกแปลงเป็นกราฟิกเวคเตอร์ของ Pdf อาจเกิดปัญหาด้านประสิทธิภาพในเครื่องมือดู Pdf.

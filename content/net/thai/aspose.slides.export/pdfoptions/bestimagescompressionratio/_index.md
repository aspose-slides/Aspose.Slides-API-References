## PdfOptions.BestImagesCompressionRatio คุณสมบัติ

ระบุว่าควรเลือกการบีบอัดที่มีประสิทธิภาพสูงสุด (แทนค่าปริยาย) สำหรับแต่ละรูปภาพโดยอัตโนมัติหรือไม่ หากกำหนดเป็น Boolean.true จะเลือกอัลกอริธึมการบีบอัดที่เหมาะสมที่สุดสำหรับรูปภาพแต่ละภาพในงานนำเสนอ ซึ่งจะทำให้ขนาดของเอกสาร PDF ที่ได้มีขนาดเล็กลง การเลือกอัตราการบีบอัดรูปภาพที่ดีที่สุดใช้ทรัพยากรคำนวณมากและต้องการ RAM เพิ่มเติม และโดยค่าเริ่มต้นตัวเลือกนี้เป็น Boolean.false

```csharp
public bool BestImagesCompressionRatio { get; set; }
```

### หมายเหตุ

ค่าเริ่มต้นคือ Boolean.false.

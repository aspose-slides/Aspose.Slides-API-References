## PdfOptions.EmbedTrueTypeFontsForASCII คุณสมบัติ

กำหนดว่า Aspose.Slides จะฝังแบบอักษรทั่วไปสำหรับข้อความ ASCII (ช่วงรหัส 33..127) หรือไม่ แบบอักษรสำหรับรหัสอักขระที่มากกว่า 127 จะถูกฝังเสมอ รายการแบบอักษรทั่วไปรวมถึงแบบอักษรฐาน 14 ของ PDF และแบบอักษรเพิ่มเติมที่ผู้ใช้ระบุ. อ่าน/เขียน Boolean.

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### หมายเหตุ

ค่าเริ่มต้นคือ **true**.

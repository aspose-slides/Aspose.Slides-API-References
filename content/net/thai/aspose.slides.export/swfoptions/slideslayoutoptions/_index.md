## SwfOptions.SlidesLayoutOptions คุณสมบัติ

รับหรือกำหนดโหมดที่สไลด์ถูกวางบนหน้าเมื่อส่งออกงานนำเสนอ [`ISlidesLayoutOptions`](../../islideslayoutoptions). คุณสมบัตินี้ไม่สนับสนุนการกำหนดค่าอ็อบเจกต์ประเภท [`HandoutLayoutingOptions`](../../handoutlayoutingoptions)

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    SwfOptions options = new SwfOptions
    {
        SlidesLayoutOptions = new NotesCommentsLayoutingOptions
        {
            CommentsPosition = CommentsPositions.Right
        }
    };
    
    pres.Save("pres.swf", SaveFormat.Swf, options);
}
```

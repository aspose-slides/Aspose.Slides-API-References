## IPptOptions.RootDirectoryClsid คุณสมบัติ

แทนคลาส GUID ของอ็อบเจกต์ (CLSID) ที่ถูกจัดเก็บในรายการไดเรกทอรีราก สามารถใช้สำหรับการเรียกใช้งาน COM ของแอปพลิเคชันเอกสาร ค่าดีฟอลต์คือ '64818D11-4F9B-11CF-86EA-00AA00B929E8' ซึ่งตรงกับ 'Microsoft Powerpoint.Slide.8'

```csharp
public Guid RootDirectoryClsid { get; set; }
```

### ตัวอย่าง

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    PptOptions pptOptions = new PptOptions();
    
    /// ตั้งค่า CLSID เป็น 'Microsoft Powerpoint.Show.8'
    pptOptions.RootDirectoryClsid = new Guid("64818D10-4F9B-11CF-86EA-00AA00B929E8");
    
    pres.Save("pres.ppt", SaveFormat.Ppt, pptOptions);
}
```

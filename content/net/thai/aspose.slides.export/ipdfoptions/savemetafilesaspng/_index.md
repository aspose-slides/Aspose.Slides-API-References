## IPdfOptions.SaveMetafilesAsPng property

True เพื่อแปลงเมตาฟายล์ทั้งหมดที่ใช้ในงานนำเสนอเป็นภาพ PNG. อ่าน/เขียน Boolean.

```csharp
public bool SaveMetafilesAsPng { get; set; }
```

### หมายเหตุ

ค่าเริ่มต้นคือ **true**. เอกสาร Pdf สามารถมีกราฟิกเวกเตอร์และภาพแรสเตอร์ได้. หาก SaveMetafilesAsPng ถูกตั้งค่าเป็น true แล้วภาพ Metafile ต้นฉบับจะถูกแปลงเป็นรูปแบบ Png และบันทึกลงใน Pdf เป็นภาพแรสเตอร์. หาก SaveMetafilesAsPng ถูกตั้งค่าเป็น false แล้ว Metafile ต้นฉบับจะถูกแปลงเป็นกราฟิกเวกเตอร์ของ Pdf. วิธีการแต่ละอย่างมีข้อดีและข้อเสีย. ตัวอย่างเช่น หาก Metafile ถูกแปลงเป็น PNG แล้วอาจเกิดการสูญเสียคุณภาพบางส่วนระหว่างการปรับสเกลของเอกสารที่ได้. หาก Metafile ถูกแปลงเป็นกราฟิกเวกเตอร์ของ Pdf แล้วอาจเกิดปัญหาด้านประสิทธิภาพในเครื่องมือดู Pdf.

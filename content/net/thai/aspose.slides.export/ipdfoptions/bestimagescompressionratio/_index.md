## IPdfOptions.BestImagesCompressionRatio คุณสมบัติ

ระบุว่าควรเลือกการบีบอัดที่มีประสิทธิภาพสูงสุด (แทนการบีบอัดค่าเริ่มต้น) สำหรับแต่ละภาพโดยอัตโนมัติหรือไม่ หากตั้งค่าเป็น Boolean.true, สำหรับแต่ละภาพในงานนำเสนอจะเลือกอัลกอริทึมการบีบอัดที่เหมาะสมที่สุด ซึ่งจะนำไปสู่ขนาดที่เล็กลงของเอกสาร PDF ที่ได้ การเลือกอัตราส่วนการบีบอัดภาพที่ดีที่สุดใช้ทรัพยากรการคำนวณอย่างมากและต้องใช้หน่วยความจำเพิ่มเติม และตัวเลือกนี้มีค่าเริ่มต้นเป็น Boolean.false

```csharp
public bool BestImagesCompressionRatio { get; set; }
```

### หมายเหตุ

ค่าเริ่มต้นคือ Boolean.false.

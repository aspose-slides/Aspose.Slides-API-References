## ISVGOptions.DisableLineEndCropping คุณสมบัติ

SVG 1.1 ขาดความสามารถในการกำหนดระยะห่างสำหรับ markers. Aspose.Slides SVG writing engine มีวิธีแก้ปัญหาดังกล่าว: มันจะทำการครอปส่วนท้ายของเส้นที่มีลูกศร, ดังนั้นเส้นจะไม่ทับซ้อนกับ markers. ตัวเลือกนี้จะปิดการทำงานดังกล่าว. อ่าน/เขียน Boolean.

```csharp
public bool DisableLineEndCropping { get; set; }
```

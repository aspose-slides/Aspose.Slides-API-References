## ISVGOptions.DisableFontLigatures คุณสมบัติ

รับหรือกำหนดค่าแจ้งว่า ข้อความจะถูกเรนเดอร์โดยไม่ใช้ลิเกเจอร์ หากตั้งค่าเป็น `true` ลิเกเจอร์จะถูกปิดในผลลัพธ์ที่เรนเดอร์ โดยค่าเริ่มต้น คุณสมบัตินี้ถูกตั้งค่าเป็น `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    SVGOptions options = new SVFOptions
    {
        DisableFontLigatures = true // ปิดลิเกเจอร์ในการเรนเดอร์ข้อความ
    };
    
    using (FileStream fileStream = new FileStream("slide-0.svg", FileMode.Create, FileAccess.Write))
    {
        pres.Slides[0].WriteAsSvg(fileStream);   
    }
}
```

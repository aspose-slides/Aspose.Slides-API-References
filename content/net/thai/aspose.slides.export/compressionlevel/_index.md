## enumeration CompressionLevel

กำหนดระดับการบีบอัด ZIP สำหรับไฟล์ OpenXML. ระดับที่สูงกว่าจะให้การบีบอัดที่ดีกว่าแต่ต้องแลกกับการประมวลผลที่ช้าลง.

```csharp
public enum CompressionLevel
```

### ค่า

| ชื่อ | ค่า | คำอธิบาย |
| --- | --- | --- |
| None | `0` | ไม่มีการบีบอัดใด ๆ ถูกนำมาใช้. ไฟล์จะถูกเก็บไว้ตามเดิม. |
| Level1 | `1` | การบีบอัดที่เร็วที่สุดโดยมีอัตราการบีบอัดต่ำที่สุด. |
| Level2 | `2` | การบีบอัดที่เร็วขึ้นโดยมีอัตราการบีบอัดที่ดีขึ้นเล็กน้อยเมื่อเทียบกับ Level1. |
| Level3 | `3` | ให้การบีบอัดที่ดีกว่า Level2 โดยมีผลต่อประสิทธิภาพระดับปานกลาง. |
| Level4 | `4` | ให้การบีบอัดที่ดีกว่า Level3. |
| Level5 | `5` | ให้การบีบอัดที่ดีขึ้นเมื่อเทียบกับ Level4 โดยใช้เวลาประมวลผลเพิ่มขึ้น. |
| Level6 | `6` | การบีบอัดมาตรฐานที่ให้สมดุลที่ดีระหว่างความเร็วในการบีบอัดและขนาดไฟล์. เป็นระดับการบีบอัดค่าเริ่มต้น. |
| Level7 | `7` | ให้การบีบอัดที่สูงกว่า Level6 โดยมีการประมวลผลที่ช้าลง. |
| Level8 | `8` | ให้การบีบอัดที่สูงกว่า Level7. |
| Level9 | `9` | การบีบอัดสูงสุด. ผลลัพธ์คือไฟล์ที่มีขนาดเล็กที่สุดด้วยความเร็วการประมวลผลที่ช้าที่สุด. |

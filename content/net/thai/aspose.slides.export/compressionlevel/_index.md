## CompressionLevel การนับ

ระบุระดับการบีบอัด ZIP สำหรับไฟล์ OpenXML. ระดับที่สูงกว่าจะให้การบีบอัดที่ดีกว่าแต่มีค่าใช้จ่ายของการประมวลผลที่ช้ากว่า.

```csharp
public enum CompressionLevel
```

### ค่า

| ชื่อ | ค่า | คำอธิบาย |
| --- | --- | --- |
| None | `0` | ไม่มีการบีบอัดใดๆ ถูกนำไปใช้. ไฟล์จะถูกเก็บไว้ตามต้นฉบับ. |
| Level1 | `1` | การบีบอัดที่เร็วที่สุดโดยมีอัตราการบีบอัดที่ต่ำที่สุด. |
| Level2 | `2` | การบีบอัดที่เร็วกว่าโดยมีอัตราการบีบอัดที่ดีกว่าเล็กน้อยเมื่อเทียบกับ Level1. |
| Level3 | `3` | ให้การบีบอัดที่ดีกว่า Level2 พร้อมผลกระทบต่อประสิทธิภาพในระดับปานกลาง. |
| Level4 | `4` | ให้การบีบอัดที่ดีกว่า Level3. |
| Level5 | `5` | ให้การบีบอัดที่พัฒนาขึ้นเมื่อเทียบกับ Level4 พร้อมเวลาการประมวลผลเพิ่มเติม. |
| Level6 | `6` | การบีบอัดมาตรฐาน ให้ความสมดุลที่ดีระหว่างความเร็วในการบีบอัดและขนาดไฟล์. ระดับการบีบอัดเริ่มต้น. |
| Level7 | `7` | ให้การบีบอัดที่สูงกว่า Level6 พร้อมการประมวลผลที่ช้าลง. |
| Level8 | `8` | ให้การบีบอัดที่สูงกว่า Level7. |
| Level9 | `9` | การบีบอัดสูงสุด. ผลลัพธ์เป็นไฟล์ขนาดเล็กที่สุดโดยมีความเร็วการประมวลผลที่ช้าที่สุด. |

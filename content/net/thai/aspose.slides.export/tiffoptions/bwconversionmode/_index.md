## TiffOptions.BwConversionMode คุณสมบัติ

ระบุอัลกอริทึมสำหรับการแปลงภาพสีเป็นภาพสีขาวและดำ ตัวเลือกนี้จะใช้ก็ต่อเมื่อตัวแปร [`CompressionType`](../compressiontype) ถูกตั้งค่าเป็น CCITT4 หรือ CCITT3 อ่าน/เขียน [`BlackWhiteConversionMode`](../../blackwhiteconversionmode) ค่าเริ่มต้นคือ Default.

```csharp
public BlackWhiteConversionMode BwConversionMode { get; set; }
```

```csharp
[C#]
TiffOptions tiffOptions = new TiffOptions();

    presentation.Save(tiffFilePath, SaveFormat.Tiff, tiffOptions);
```

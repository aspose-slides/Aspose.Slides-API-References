## IPptxOptions.CompressionLevel คุณสมบัติ

กำหนดระดับการบีบอัดที่ใช้เมื่อต้องการบันทึกเอกสารการนำเสนอ. ค่าเริ่มต้นคือ Level6.

```csharp
public CompressionLevel CompressionLevel { get; set; }
```

### หมายเหตุ

ระดับการบีบอัดที่สูงกว่าจะทำให้ไฟล์มีขนาดเล็กลงแต่ต้องใช้เวลาประมวลผลมากขึ้น. อัตราการบีบอัดจริงขึ้นอยู่กับเนื้อหาของการนำเสนอ.

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("demo-level8.pptx", SaveFormat.Pptx, new PptxOptions()
    {
        CompressionLevel = CompressionLevel.Level8
    });
}
```

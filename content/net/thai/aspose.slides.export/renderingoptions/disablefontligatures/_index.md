## RenderingOptions.DisableFontLigatures คุณสมบัติ

รับหรือกำหนดค่าที่บ่งชี้ว่าข้อความจะถูกแสดงโดยไม่ได้ใช้ลิการเจอร์ เมื่อกำหนดเป็น `true` ลิการเจอร์จะถูกปิดใช้งานในผลลัพธ์ที่แสดง โดยค่าเริ่มต้น คุณสมบัตินี้จะถูกตั้งค่าเป็น `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    RenderingOptions options = new RenderingOptions
    {
        DisableFontLigatures = true // ปิดการใช้ลิการเจอร์ในการแสดงข้อความ
    };
    
    Bitmap[] renderedSlides = pres.GetImage(options);
    for (var index = 0; index < renderedSlides.Length; index++)
    {
        var slideImage = renderedSlides[index];
        slideImage.Save($"slide-{index}.png");
    }
}
```

## IRenderingOptions.DisableFontLigatures คุณสมบัติ

รับหรือกำหนดค่าแสดงว่าข้อความจะถูกเรนเดอร์โดยไม่ใช้ลิแกเชอร์ หากตั้งค่าเป็น `true` ลิแกเชอร์จะถูกปิดใช้งานในผลลัพธ์ที่เรนเดอร์ โดยค่าเริ่มต้นคุณสมบัตินี้จะถูกตั้งค่าเป็น `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### ตัวอย่าง

Example:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    RenderingOptions options = new RenderingOptions
    {
        DisableFontLigatures = true // ปิดการใช้ลิแกเชอร์ในการเรนเดอร์ข้อความ
    };
    
    Bitmap[] renderedSlides = pres.GetImage(options);
    for (var index = 0; index < renderedSlides.Length; index++)
    {
        var slideImage = renderedSlides[index];
        slideImage.Save($"slide-{index}.png");
    }
}
```

## SVGOptions.DisableLineEndCropping คุณสมบัติ

SVG 1.1 ขาดความสามารถในการกำหนดระยะห่างสำหรับมาร์กเกอร์. Aspose.Slides SVG writing engine มีวิธีแก้ไขสำหรับปัญหานั้น: มันตัดส่วนท้ายของเส้นด้วยลูกศร ดังนั้น เส้นจะไม่ทับซ้อนกับมาร์กเกอร์. ตัวเลือกนี้จะปิดการทำงานดังกล่าว. อ่าน/เขียน Boolean.

public bool DisableLineEndCropping { get; set; }

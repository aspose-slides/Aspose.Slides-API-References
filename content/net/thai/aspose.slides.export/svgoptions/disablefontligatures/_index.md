## SVGOptions.DisableFontLigatures คุณสมบัติ

รับหรือกำหนดค่าที่บ่งบอกว่าข้อความจะถูกแสดงโดยไม่ใช้ลิกาเจอร์ เมื่อกำหนดเป็น `true` ลิกาเจอร์จะถูกปิดใช้งานในผลลัพธ์ที่แสดงโดยค่าเริ่มต้น คุณสมบัตินี้ถูกตั้งค่าเป็น `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    SVGOptions options = new SVFOptions
    {
        DisableFontLigatures = true // ปิดการใช้งานลิกาเจอร์ในการแสดงข้อความ
    };
    
    using (FileStream fileStream = new FileStream("slide-0.svg", FileMode.Create, FileAccess.Write))
    {
        pres.Slides[0].WriteAsSvg(fileStream);   
    }
}
```

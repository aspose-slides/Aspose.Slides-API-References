## IInkOptions.InterpretMaskOpAsOpacity คุณสมบัติ

ใช้การดำเนินการ ROP หรือ Opacity สำหรับการเรนเดอร์แปรง.

```csharp
public bool InterpretMaskOpAsOpacity { get; set; }
```

### หมายเหตุ

ค่าเริ่มต้นคือ true.

### ตัวอย่าง

ตัวอย่างต่อไปแสดงวิธีตั้งค่าโดยใช้ ROP เพื่อส่งออกองค์ประกอบ Ink:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.InterpretMaskOpAsOpacity = false;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

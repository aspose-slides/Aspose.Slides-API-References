## ISaveOptions.SkipJavaScriptLinks คุณสมบัติ

ระบุว่าจะข้ามลิงก์ไฮเปอร์ที่มีการเรียกใช้ JavaScript หรือไม่เมื่อบันทึกงานนำเสนอ. อ่าน/เขียน Boolean. ค่าเริ่มต้นคือ **false**.

```csharp
public bool SkipJavaScriptLinks { get; set; }
```

### หมายเหตุ

เมื่อกำหนดคุณสมบัตินี้เป็น **true**, ลิงก์ไฮเปอร์ที่มีการเรียกใช้ JavaScript จะถูกละเว้นขณะบันทึก.

เมื่อกำหนดคุณสมบัตินี้เป็น **false**, ลิงก์ไฮเปอร์ทั้งหมดจะถูกบันทึก.

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("result_without_JavaScript_links.html", SaveFormat.Html, new HtmlOptions()
    {
        SkipJavaScriptLinks = true
    });
}
```

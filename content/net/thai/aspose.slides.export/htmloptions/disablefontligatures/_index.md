## HtmlOptions.DisableFontLigatures คุณสมบัติ

รับหรือกำหนดค่าที่บ่งชี้ว่าข้อความถูกแสดงโดยไม่ใช้ลิแกเจอร์ เมื่อกำหนดเป็น `true` ลิแกเจอร์จะถูกปิดในผลลัพธ์ที่แสดง โดยค่าเริ่มต้นคุณสมบัตินี้ถูกตั้งค่าเป็น `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### ตัวอย่าง

Example:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    HtmlOptions options = new HtmlOptions
    {
        DisableFontLigatures = true // ปิดการใช้ลิแกเจอร์ในการแสดงข้อความ
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html, options);
}
```

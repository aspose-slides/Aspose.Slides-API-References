## HtmlOptions.JpegQuality คุณสมบัติ

คืนค่า หรือ ตั้งค่าค่าที่กำหนดคุณภาพของภาพ JPEG ภายในเอกสาร PDF. อ่าน/เขียน Byte.

```csharp
public byte JpegQuality { get; set; }
```

### หมายเหตุ

มีผลเฉพาะเมื่อเอกสารมีภาพ JPEG.

ใช้คุณสมบัตินี้เพื่อรับหรือกำหนดคุณภาพของภาพภายในเอกสารเมื่อบันทึกเป็นรูปแบบ PDF. ค่าจะอยู่ระหว่าง 0 ถึง 100 โดยที่ 0 หมายถึงคุณภาพต่ำสุดแต่บีบอัดสูงสุดและ 100 หมายถึงคุณภาพดีที่สุดแต่บีบอัดต่ำสุด.

ค่าปริยายคือ **95**.

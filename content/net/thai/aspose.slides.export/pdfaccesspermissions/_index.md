## การระบุค่า PdfAccessPermissions

ประกอบด้วยชุดของแฟล็กที่ระบุว่าสิทธิ์การเข้าถึงใดควรได้รับมอบเมื่อเอกสารถูกเปิดด้วยการเข้าถึงของผู้ใช้.

```csharp
[Flags]
public enum PdfAccessPermissions
```

### ค่าต่างๆ

| ชื่อ | ค่า | คำอธิบาย |
| --- | --- | --- |
| None | `0` | ระบุว่าผู้ใช้ไม่มีสิทธิ์การเข้าถึง. |
| PrintDocument | `4` | ระบุว่าผู้ใช้สามารถพิมพ์เอกสารได้หรือไม่ (อาจไม่อยู่ในระดับคุณภาพสูงสุด ขึ้นอยู่กับว่าบิต HighQualityPrint ถูกตั้งค่าด้วยหรือไม่). |
| ModifyContent | `8` | ระบุว่าผู้ใช้สามารถแก้ไขเนื้อหาในเอกสารโดยการดำเนินการอื่น ๆ นอกเหนือจากที่ควบคุมโดยบิต AddOrModifyFields, FillExistingFields, AssembleDocument หรือไม่. |
| CopyTextAndGraphics | `10` | ระบุว่าผู้ใช้สามารถคัดลอกหรือดึงข้อความและกราฟิกจากเอกสารได้โดยการดำเนินการอื่น ๆ ที่ไม่ใช่บิต ExtractTextAndGraphics หรือไม่. |
| AddOrModifyFields | `20` | ระบุว่าผู้ใช้สามารถเพิ่มหรือแก้ไขคำอธิบายข้อความ, เติมฟิลด์ฟอร์มโต้ตอบ, และหากบิต ModifyContent ถูกตั้งค่าไว้ด้วย, สร้างหรือแก้ไขฟิลด์ฟอร์มโต้ตอบ (รวมถึงฟิลด์ลายเซ็น) ได้หรือไม่. |
| FillExistingFields | `100` | ระบุว่าผู้ใช้สามารถเติมฟิลด์ฟอร์มโต้ตอบที่มีอยู่ (รวมถึงฟิลด์ลายเซ็น) ได้หรือไม่ แม้ว่าบิต AddOrModifyFields จะไม่ได้ตั้งค่า. |
| ExtractTextAndGraphics | `200` | ระบุว่าผู้ใช้สามารถดึงข้อความและกราฟิกเพื่อสนับสนุนการเข้าถึงสำหรับผู้ใช้ที่มีความพิการหรือเพื่อวัตถุประสงค์อื่น ๆ ได้หรือไม่. |
| AssembleDocument | `400` | ระบุว่าผู้ใช้สามารถประกอบเอกสาร (แทรก, หมุน, หรือลบหน้าและสร้างบุ๊กมาร์คหรือภาพย่อ) ได้หรือไม่ แม้ว่าบิต ModifyContent จะไม่ได้ตั้งค่า. |
| HighQualityPrint | `800` | ระบุว่าผู้ใช้สามารถพิมพ์เอกสารเป็นรูปแบบที่สามารถสร้างสำเนาดิจิทัลที่ตรงกับเนื้อหา PDF ได้หรือไม่ เมื่อบิตนี้ไม่ได้ตั้งค่า (และบิต PrintDocument ตั้งค่า) การพิมพ์จะถูกจำกัดอยู่ที่การแสดงผลระดับต่ำ ซึ่งอาจมีคุณภาพลดต่ำลง. |

## PdfAccessPermissions การกำหนดค่า

Contains a set of flags specifying which access permissions should be granted when the document is opened with user access.

```csharp
[Flags]
public enum PdfAccessPermissions
```

| ชื่อ | ค่า | คำอธิบาย |
| --- | --- | --- |
| None | `0` | ระบุว่าผู้ใช้ไม่มีการอนุญาตการเข้าถึง |
| PrintDocument | `4` | ระบุว่าผู้ใช้สามารถพิมพ์เอกสารได้หรือไม่ (อาจไม่อยู่ในระดับคุณภาพสูงสุด ขึ้นอยู่กับว่าบิต HighQualityPrint ถูกตั้งค่าอยู่หรือไม่) |

| CopyTextAndGraphics | `10` | ระบุว่าผู้ใช้สามารถคัดลอกหรือสกัดข้อความและกราฟิกจากเอกสารได้หรือไม่โดยการดำเนินการที่ไม่ได้ควบคุมโดยบิต ExtractTextAndGraphics |

| HighQualityPrint | `800` | ระบุว่าผู้ใช้สามารถพิมพ์เอกสารเป็นการแสดงผลที่สามารถสร้างสำเนาดิจิทัลที่แม่นยำของเนื้อหา PDF ได้หรือไม่ เมื่อบิตนี้ไม่ได้ตั้งค่า (และบิต PrintDocument ถูกตั้งค่า) การพิมพ์จะจำกัดอยู่ที่การแสดงผลระดับต่ำซึ่งอาจมีคุณภาพลดลง |

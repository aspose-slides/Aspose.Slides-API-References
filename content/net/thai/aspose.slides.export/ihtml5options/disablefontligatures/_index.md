## IHtml5Options.DisableFontLigatures คุณสมบัติ

รับหรือกำหนดค่าที่บ่งบอกว่าข้อความจะถูกเรนเดอร์โดยไม่ใช้ลิจูรย์ เมื่อกำหนดเป็น `true` ลิจูรย์จะถูกปิดใช้ในผลลัพธ์ที่แสดง ตามค่าเริ่มต้น คุณสมบัตินี้จะถูกตั้งค่าเป็น `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    Html5Options options = new Html5Options
    {
        DisableFontLigatures = true // ปิดการใช้ลิจูรณ์ในการเรนเดอร์ข้อความ
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html5, options);
}
```

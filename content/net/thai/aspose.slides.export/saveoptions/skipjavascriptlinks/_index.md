## SaveOptions.SkipJavaScriptLinks คุณสมบัติ

Specifies whether to skip hyperlinks with JavaScript calls when saving the presentation. อ่าน/เขียน Boolean. ค่าเริ่มต้นคือ **false**.

```csharp
public bool SkipJavaScriptLinks { get; set; }
```

### หมายเหตุ

เมื่อคุณสมบัตินี้ถูกตั้งค่าเป็น **true**, hyperlinks with JavaScript calls จะถูกละเว้นขณะบันทึก

เมื่อคุณสมบัตินี้ถูกตั้งค่าเป็น **false**, hyperlinks ทั้งหมดจะถูกบันทึก

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("result_without_JavaScript_links.html", SaveFormat.Html, new HtmlOptions()
    {
        SkipJavaScriptLinks = true
    });
}
```

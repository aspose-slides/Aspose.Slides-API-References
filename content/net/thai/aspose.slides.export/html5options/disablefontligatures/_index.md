## Html5Options.DisableFontLigatures คุณสมบัติ

รับหรือกำหนดค่าที่บ่งบอกว่าข้อความจะถูกเรนเดอร์โดยไม่ใช้ลิการ์เจอร์ เมื่อกำหนดเป็น `true` ลิการ์เจอร์จะถูกปิดใช้งานในผลลัพธ์ที่เรนเดอร์ โดยค่าเริ่มต้น คุณสมบัตินี้ถูกตั้งค่าเป็น `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    Html5Options options = new Html5Options
    {
        DisableFontLigatures = true // ปิดลิการ์เจอร์ในการเรนเดอร์ข้อความ
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html5, options);
}
```

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
  pres.Save("demo-animate-shapes-and-transitions.html", SaveFormat.Html5, new Html5Options()

      AnimateTransitions = true
  });
}
```

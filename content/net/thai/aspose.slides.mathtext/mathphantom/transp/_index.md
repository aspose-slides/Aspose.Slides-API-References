## MathPhantom.Transp คุณสมบัติ

รับหรือกำหนดค่าที่บ่งบอกว่า phantom มีความโปร่งใสสำหรับกฎการเว้นระยะตามคลาสหรือไม่

```csharp
public bool Transp { get; set; }
```

### Remarks

เมื่อ `true` ตัวดำเนินการและสัญลักษณ์ภายใน phantom ยังมีผลต่อการเว้นระยะคณิตศาสตร์รอบ phantom (เหมือนเป็นที่มองเห็น) เมื่อ `false` การเว้นระยะตามคลาสจะถูกละเลย สอดคล้องกับแอตทริบิวต์ OMML `m:transp`

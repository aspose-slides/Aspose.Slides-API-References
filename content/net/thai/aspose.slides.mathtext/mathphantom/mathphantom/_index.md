## คอนสตรัคเตอร์ MathPhantom

สร้างอินสแตนซ์ใหม่ของคลาสต[`MathPhantom`](../../mathphantom) โดยใช้ base math element ที่ระบุ

```csharp
public MathPhantom(IMathElement element)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| element | IMathElement | ฐาน[`IMathElement`](../../imathelement)ที่การมองเห็นและการจัดวางจะถูกควบคุมโดย phantom. องค์ประกอบนี้กำหนดเนื้อหาที่อาจถูกซ่อนหรือแสดง, แต่ยังมีผลต่อการจัดแนวเชิงเรขาคณิตของคณิตศาสตร์รอบข้าง |

### หมายเหตุ

phantom element ถูกใช้เพื่อสำรองหรือกดทับพื้นที่การแสดงผลของนิพจน์ฐานโดยไม่จำเป็นต้องแสดงผลออกมา มันสอดคล้องกับ OMML element `<m:phant>`.

### ตัวอย่าง

```csharp
[C#]
IMathElement fraction = new MathFraction(
    new MathematicalText("1"),
    new MathematicalText("2"));
```

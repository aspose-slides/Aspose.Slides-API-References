## IMathBox.ExplicitBreak คุณสมบัติ

Explicit break ระบุว่ามีการขึ้นบรรทัดใหม่ที่จุดเริ่มต้นของวัตถุ Box หรือไม่ เพื่อให้ข้อความตัดบรรทัดที่จุดเริ่มต้นของวัตถุ Box. ระบุจำนวนของตัวดำเนินการในบรรทัดก่อนหน้าของข้อความคณิตศาสตร์ซึ่งจะใช้เป็นจุดจัดตำแหน่งสำหรับบรรทัดปัจจุบันของข้อความคณิตศาสตร์ ค่าที่เป็นไปได้: 1..255 ค่าเริ่มต้น: 0 (no explicit break)

```csharp
public byte ExplicitBreak { get; set; }
```

### ตัวอย่าง

Example:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.ExplicitBreak = 1;
```

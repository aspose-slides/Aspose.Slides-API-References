## IMathBox.NoBreak คุณสมบัติ

ไม่มีการตัดบรรทัด คุณสมบัตินี้ระบุคุณสมบัติ "unbreakable" ของกล่องอ็อบเจกต์ เมื่อเป็น true จะไม่มีการตัดบรรทัดภายในกล่อง ซึ่งอาจสำคัญสำหรับตัวจำลองตัวดำเนินการที่ประกอบด้วยตัวดำเนินการไบนารีมากกว่าหนึ่งตัว เมื่อไม่ได้ระบุองค์ประกอบนี้ การตัดบรรทัดสามารถเกิดขึ้นภายในกล่องได้ ค่าเริ่มต้น: true

```csharp
public bool NoBreak { get; set; }
```

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]
IMathBox box = new MathematicalText("**********").ToBox();
box.NoBreak = false;
```

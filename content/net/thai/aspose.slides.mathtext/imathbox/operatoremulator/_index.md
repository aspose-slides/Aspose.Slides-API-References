## IMathBox.OperatorEmulator คุณสมบัติ

อิมูเลเตอร์ของผู้ดำเนินการ. เมื่อเป็นค่า true, กล่องและเนื้อหาภายในทำงานเป็นผู้ดำเนินการเดียวและสืบทอดคุณสมบัติของผู้ดำเนินการ. สิ่งนี้หมายความว่า, ตัวอย่างเช่น, ตัวอักษรสามารถทำหน้าที่เป็นจุดตัดบรรทัดและสามารถจัดแนวกับผู้ดำเนินการอื่นๆ ได้. อิมูเลเตอร์ของผู้ดำเนินการมักใช้เมื่อ glyph หนึ่งหรือหลายตัวรวมกันเป็นผู้ดำเนินการ, เช่น '=='.

ค่าเริ่มต้น: false

```csharp
public bool OperatorEmulator { get; set; }
```

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.OperatorEmulator = true;
```

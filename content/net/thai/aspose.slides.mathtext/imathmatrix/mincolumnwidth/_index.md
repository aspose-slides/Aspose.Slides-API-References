## IMathMatrix.MinColumnWidth คุณสมบัติ

ความกว้างคอลัมน์ต่ำสุดใน twips (1/20 ของจุด) ระยะห่างช่องว่าง (ที่เรียกว่า “Column Gap” หรือ “Gap Width”) จะถูกเพิ่มเข้าไปใน MinColumnWidth เพื่อกำหนด Matrix Column Spacing รวมทั้งหมด (ระยะห่างระหว่างขอบเดียวกันของคอลัมน์ที่ต่างกัน) ค่าเริ่มต้น: 0.

```csharp
public uint MinColumnWidth { get; set; }
```

### ตัวอย่าง

Example:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.MinColumnWidth = 20;
```

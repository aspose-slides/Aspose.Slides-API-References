## IMathMatrix.ColumnGapRule คุณสมบัติ

ประเภทของระยะห่างแนวนอนระหว่างคอลัมน์ของเมทริกซ์; หน่วยระยะห่างแนวนอนสามารถเป็น ems หรือ points (จัดเก็บเป็น twips) ได้. ค่าเริ่มต้น: SingleSpacingGap (0)

```csharp
public MathSpacingRules ColumnGapRule { get; set; }
```

### ตัวอย่าง

ตัวอย่าง:

```csharp

matrix.ColumnGapRule = MathSpacingRules.OneAndHalfSpacingGap;
```

## IMathPhantom.ZeroAsc คุณสมบัติ

รับหรือกำหนดค่าที่บ่งชี้ว่าการยกส่วน (height above baseline) ขององค์ประกอบฐานควรถือเป็นศูนย์หรือไม่.

```csharp
public bool ZeroAsc { get; set; }
```

### หมายเหตุ

เมื่อ `true` phantom จะไม่ยกเส้นฐานของบรรทัดคณิตศาสตร์ที่อยู่รอบ ๆ. สอดคล้องกับแอตทริบิวต์ OMML `m:zeroAsc`.

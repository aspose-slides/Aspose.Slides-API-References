## IMathPhantom.Transp คุณสมบัติ

รับหรือกำหนดค่าที่บ่งบอกว่า phantom มีความโปร่งใสสำหรับกฎการเว้นระยะตามคลาสหรือไม่.

```csharp
public bool Transp { get; set; }
```

### หมายเหตุ

เมื่อ `true` ตัวดำเนินการและสัญลักษณ์ภายใน phantom ยังคงมีผลต่อการเว้นระยะทางคณิตศาสตร์รอบ phantom (เหมือนมองเห็นได้) เมื่อ `false` การเว้นระยะตามคลาสจะถูกละเลย ตรงกับแอตทริบิวต์ OMML `m:transp`.

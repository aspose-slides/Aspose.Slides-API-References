## MathDelimiter.EndingCharacter คุณสมบัติ

Delimiter Ending Character กำหนดตัวอักษรสิ้นสุดหรือปิดของตัวคั่น. ตัวคั่นคณิตศาสตร์เป็นอักขระที่ล้อมรอบเช่นวงเล็บ, วงกลมเหลี่ยม, และวงเล็บเคอร์ลี่. ค่าเริ่มต้นคือ ')'.

```csharp
public char EndingCharacter { get; set; }
```

### ตัวอย่าง

Example:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Join("y").Enclose();
delimiter.EndingCharacter = ']';
```

## IMathGroupingCharacter.VerticalJustification คุณสมบัติ

การจัดแนวดิ่งของอักขระกลุ่ม กำหนดการจัดตำแหน่งของวัตถุตามเส้นฐาน ตัวอย่างเช่น เมื่ออักขระกลุ่มอยู่เหนือวัตถุ VerticalJustification ของ Top แสดงว่าด้านบนของวัตถุอยู่บนเส้นฐาน; เมื่อกำหนด VerticalJustification เป็น Bottom ด้านล่างของวัตถุอยู่บนเส้นฐาน ค่าเริ่มต้น: Bottom สำหรับ Position=Top และ Top สำหรับ Position=Bottom

```csharp
public MathTopBotPositions VerticalJustification { get; set; }
```

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]
MathGroupingCharacter groupingCharacter = new MathGroupingCharacter(new MathematicalText("abc"));
groupingCharacter.VerticalJustification = MathTopBotPositions.Top;
```

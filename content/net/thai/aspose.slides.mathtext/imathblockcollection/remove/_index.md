## IMathBlockCollection.Remove เมธอด

ลบการปรากฏครั้งแรกของอ็อบเจ็กต์เฉพาะจากคอลเลกชัน/&gt;.

```csharp
public bool Remove(IMathBlock item)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| item | IMathBlock | อ็อบเจ็กต์ที่จะลบออกจากคอลเลกชัน |

### ค่าที่ส่งกลับ

true ถ้า *item* ถูกลบออกจากคอลเลกชันสำเร็จ; มิฉะนั้น false. เมธอดนี้ยังคืนค่า false หากไม่พบ *item* ในคอลเลกชันต้นฉบับ/&gt;.

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]
IMathBlockCollection blockCollection = new MathParagraph();
IMathBlock block = new MathBlock(new MathematicalText("y"));
blockCollection.Add(block);
blockCollection.Remove(block);
```

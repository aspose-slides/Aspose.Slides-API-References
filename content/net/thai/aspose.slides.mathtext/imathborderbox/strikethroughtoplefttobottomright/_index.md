## IMathBorderBox.StrikethroughTopLeftToBottomRight คุณสมบัติ

Strikethrough Top-Left to Bottom-Right (default is false). ระบุสถานะซ่อนหรือแสดงของเส้นขีดทับแนวทแยงจากมุมบนซ้ายไปมุมล่างขวาของกล่องขอบ.

```csharp
public bool StrikethroughTopLeftToBottomRight { get; set; }
```

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]

borderBox.StrikethroughTopLeftToBottomRight = true;
```

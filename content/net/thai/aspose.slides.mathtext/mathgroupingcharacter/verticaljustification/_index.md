## MathGroupingCharacter.VerticalJustification คุณสมบัติ

การจัดแนวแนวตั้งของตัวอักขระกลุ่ม. ระบุการจัดตำแหน่งของอ็อบเจ็กต์สัมพันธ์กับเส้นฐาน. ตัวอย่างเช่น เมื่อ ตัวอักขระกลุ่ม อยู่เหนืออ็อบเจ็กต์, VerticalJustification ของ Top แสดงว่าด้านบนของอ็อบเจ็กต์อยู่บนเส้นฐาน; เมื่อ VerticalJustification ถูกตั้งค่าเป็น Bottom, ด้านล่างของอ็อบเจ็กต์อยู่บนเส้นฐาน ค่าเริ่มต้น: Bottom สำหรับ Position=Top, และ Top สำหรับ Position=Bottom

```csharp
public MathTopBotPositions VerticalJustification { get; set; }
```

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]
MathGroupingCharacter groupingCharacter = new MathGroupingCharacter(new MathematicalText("abc"));
groupingCharacter.VerticalJustification = MathTopBotPositions.Top;
```

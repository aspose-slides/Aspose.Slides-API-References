## IMMathDelimiter.DelimiterShape คุณสมบัติ

ระบุรูปร่างของตัวคั่นในอ็อบเจกต์ตัวคั่น เมื่อเป็น MathDelimiterShape.Centered ตัวคั่นจะถูกจัดกึ่งกลางรอบแกนคณิตศาสตร์ของข้อความคณิตศาสตร์และยังสามารถปรับให้พอดีกับความสูงทั้งหมดของเนื้อหาภายในได้ เมื่อเป็น MathDelimiterShape.Match ความสูงและรูปร่างของมันจะถูกปรับให้ตรงกับเนื้อหาอย่างแม่นยำ

```csharp
public MathDelimiterShape DelimiterShape { get; set; }
```

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.DelimiterShape = MathDelimiterShape.Match;
```

## IMathElementCollection.Remove เมธอด

ลบการปรากฏครั้งแรกของวัตถุที่กำหนดจากคอลเลกชัน

```csharp
public bool Remove(IMathElement item)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| item | IMathElement | วัตถุที่จะลบออกจากคอลเลกชัน |

### ค่าที่ส่งกลับ

true หาก *item* ถูกลบออกจากคอลเลกชันสำเร็จ; มิฉะนั้นจะเป็น false. เมธอดนี้ยังคืนค่า false หากไม่พบ *item* ในคอลเลกชันต้นฉบับ

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]
IMathElementCollection collection = new MathBlock(new MathematicalText("x"));
MathematicalText plusElement = new MathematicalText("+");
collection.Add(plusElement);
collection.Add(new MathRadical(new MathematicalText("x"), new MathematicalText("3")));
collection.Remove(plusElement);
```

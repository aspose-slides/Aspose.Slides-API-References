## MathMatrix.RowGap คุณสมบัติ

ค่าของการเว้นระยะแนวตั้งระหว่างแถวของเมทริกซ์; หาก RowGapRule ถูกตั้งค่าเป็น 3 ("Exactly") แล้วหน่วยจะถูกตีความเป็น twips (1/20 ของจุด) หาก RowGapRule ถูกตั้งค่าเป็น 4 ("Multiple") แล้วหน่วยจะถูกตีความเป็น half-lines. ค่าเริ่มต้น: 0

```csharp
public uint RowGap { get; set; }
```

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

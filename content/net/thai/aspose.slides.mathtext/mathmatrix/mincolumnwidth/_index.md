## MathMatrix.MinColumnWidth คุณสมบัติ

ความกว้างคอลัมน์ขั้นต่ำในหน่วย twips (1/20 ของจุด) ช่องว่างระหว่างคอลัมน์ (ที่เรียกว่า “Column Gap” หรือ “Gap Width”) จะถูกเพิ่มไปยัง MinColumnWidth เพื่อกำหนด Matrix Column Spacing ทั้งหมด (ระยะห่างระหว่างขอบเดียวกันของคอลัมน์ต่าง ๆ) ค่าเริ่มต้น: 0.

```csharp
public uint MinColumnWidth { get; set; }
```

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.MinColumnWidth = 20;
```

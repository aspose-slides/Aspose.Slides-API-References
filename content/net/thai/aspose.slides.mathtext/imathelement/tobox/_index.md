## IMathElement.ToBox เมธอด

วางองค์ประกอบนี้ไว้ในกล่องที่ไม่ปรากฏ (การจัดกลุ่มเชิงตรรกะ) ซึ่งใช้เพื่อจัดกลุ่มส่วนประกอบของสมการหรือข้อความคณิตศาสตร์อื่น ๆ วัตถุที่อยู่ในกล่องสามารถ (เช่น) ทำหน้าที่เป็นอีมูเลเตอร์ของตัวดำเนินการพร้อมหรือไม่มีจุดจัดตำแหน่ง ทำหน้าที่เป็นจุดขึ้นบรรทัดใหม่ หรือถูกจัดกลุ่มเพื่อไม่ให้มีการขึ้นบรรทัดใหม่ภายใน

```csharp
public IMathBox ToBox()
```

### ค่าที่ส่งคืน

กล่องเชิงตรรกะที่มีองค์ประกอบนี้วางอยู่ภายใน

### ตัวอย่าง

Example:

```csharp
[C#]
IMathBox box = new MathematicalText("x:=y").ToBox();
```

## MathBox.OperatorEmulator คุณสมบัติ

Operator Emulator. เมื่อเป็น true กล่องและเนื้อหาในกล่องจะทำงานเป็นตัวดำเนินการเดียวและสืบทอดคุณสมบัติของตัวดำเนินการ ซึ่งหมายความว่า ตัวอักษรสามารถทำหน้าที่เป็นจุดสำหรับการหยุดบรรทัดและสามารถจัดตำแหน่งกับตัวดำเนินการอื่นๆ ได้ ตัวจำลองตัวดำเนินการมักใช้เมื่อ glyph หนึ่งหรือหลายตัวรวมกันเป็นตัวดำเนินการ เช่น '==' ค่าเริ่มต้น: false

```csharp
public bool OperatorEmulator { get; set; }
```

### ตัวอย่าง

Example:

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("=="));
box.OperatorEmulator = true;
```

## MathBox.NoBreak คุณสมบัติ

ไม่มีการตัดบรรทัด คุณสมบัตินี้ระบุคุณสมบัติ "unbreakable" บนกล่องวัตถุ เมื่อเป็นจริง จะไม่มีการตัดบรรทัดภายในกล่อง สิ่งนี้อาจมีความสำคัญสำหรับอีมูเลเตอร์ผู้ดำเนินการที่ประกอบด้วยผู้ดำเนินการไบนารีมากกว่าหนึ่งตัว เมื่อไม่ระบุองค์ประกอบนี้ การตัดบรรทัดอาจเกิดขึ้นภายในกล่อง ค่าเริ่มต้น: true

```csharp
public bool NoBreak { get; set; }
```

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("*****"));
box.NoBreak = false;
```

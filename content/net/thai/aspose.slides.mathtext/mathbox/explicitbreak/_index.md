## MathBox.ExplicitBreak คุณสมบัติ

Explicit break ระบุว่ามีการหยุดบรรทัดที่จุดเริ่มต้นของอ็อบเจ็กต์ Box หรือไม่ ซึ่งบรรทัดจะตัดต่อที่จุดเริ่มต้นของอ็อบเจ็กต์ Box ระบุจำนวนของตัวดำเนินการในบรรทัดก่อนหน้าของข้อความคณิตศาสตร์ที่ใช้เป็นจุดจัดตำแหน่งสำหรับบรรทัดปัจจุบันของข้อความคณิตศาสตร์ ค่าที่เป็นไปได้: 1..255 ค่าเริ่มต้น: 0 (ไม่มีการหยุดบรรทัดโดยชัดเจน)

```csharp
public byte ExplicitBreak { get; set; }
```

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.ExplicitBreak = 1;
```

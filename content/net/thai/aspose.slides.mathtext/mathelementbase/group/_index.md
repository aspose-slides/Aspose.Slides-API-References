## Group(char, MathTopBotPositions, MathTopBotPositions) {#group_1}

วางองค์ประกอบนี้ในกลุ่มโดยใช้อักขระจัดกลุ่มเช่นวงเล็บปีกกาแบบล่างหรืออักขระอื่น

```csharp
public IMathGroupingCharacter Group(char character, MathTopBotPositions position, 
    MathTopBotPositions verticalJustification)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| character | Char | อักขระจัดกลุ่มเช่น BOTTOM CURLY BRACKET (U+23DF) หรืออักขระอื่น |
| position | MathTopBotPositions | ตำแหน่งของอักขระจัดกลุ่ม |
| verticalJustification | MathTopBotPositions | การจัดแนวแนวตั้งของอักขระกลุ่ม ระบุการจัดตำแหน่งของอ็อบเจกต์สัมพันธ์กับบรรทัดฐาน ตัวอย่างเช่น เมื่ออักขระกลุ่มอยู่เหนืออ็อบเจกต์ การจัดแนวแนวตั้งเป็น Top หมายถึงด้านบนของอ็อบเจกต์อยู่บนบรรทัดฐาน; เมื่อการจัดแนวแนวตั้งตั้งเป็น Bottom ด้านล่างของอ็อบเจกต์อยู่บนบรรทัดฐาน |

### ค่าที่ส่งกลับ

อินสแตนซ์ใหม่ของประเภท [`IMathGroupingCharacter`](../../imathgroupingcharacter)

### ตัวอย่าง

Example:

```csharp
[C#]
IMathGroupingCharacter groupingElement = new MathematicalText("x;y;z").Group('\u23E1', MathTopBotPositions.Bottom, MathTopBotPositions.Top);
```

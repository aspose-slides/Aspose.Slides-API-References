## MathElementBase.ToBox เมธอด

วางอีลีเมนต์นี้ในกล่องที่ไม่แสดงผล (การจัดกลุ่มเชิงตรรกะ) ซึ่งใช้เพื่อจัดกลุ่มส่วนประกอบของสมการหรือข้อความคณิตศาสตร์อื่น ๆ วัตถุที่อยู่ในกล่องสามารถ (เช่น) ทำหน้าที่เป็นอิมูเลเตอร์ของโอเปอร์เรเตอร์กับหรือไม่มีจุดจัดตำแหน่ง, ทำหน้าที่เป็นจุดตัดบรรทัด, หรือถูกจัดกลุ่มเพื่อไม่ให้เกิดการตัดบรรทัดภายใน

```csharp
public IMathBox ToBox()
```

### ค่าที่ส่งคืน

กล่องตรรกะที่มีอีลีเมนต์นี้อยู่ภายใน

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]
IMathBox box = new MathematicalText("x:=y").ToBox();
```

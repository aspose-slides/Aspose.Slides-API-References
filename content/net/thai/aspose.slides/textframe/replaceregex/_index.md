## TextFrame.ReplaceRegex เมธอด

แทนที่การจับคู่ทั้งหมดของ regular expression ด้วยสตริงที่ระบุ.

```csharp
public void ReplaceRegex(Regex regex, string newText, IFindResultCallback callback)
```

| พารามิเตอร์ | ชนิด | คำอธิบาย |
| --- | --- | --- |
| regex | Regex | นิพจน์ regular expression Regex เพื่อรับสตริงที่จะต้องแทนที่ |
| newText | String | สตริงที่จะใช้แทนที่ทุก occurrence ของสตริงที่ต้องการแทนที่ |
| callback | IFindResultCallback | อ็อบเจ็กต์ Callback สำหรับบันทึกผลการดำเนินการแทนที่ [`IFindResultCallback`](../../ifindresultcallback). |

### ตัวอย่าง

ตัวอย่างโค้ดต่อไปนี้แสดงวิธีการแทนที่ข้อความโดยใช้ regular expression กับสตริงที่ระบุ.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx")){
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// แทนที่คำทั้งหมดที่มี 10 ตัวอักษรหรือมากกว่าด้วย '***'
	((AutoShape)presentation.Slides[0].Shapes[0]).TextFrame.ReplaceRegex(regex, "***", null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

## ProtectionManager.CheckWriteProtection เมธอด

กำหนดว่าการนำเสนอถูกป้องกันด้วยรหัสผ่านเพื่อแก้ไขหรือไม่

```csharp
public bool CheckWriteProtection(string password)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| password | String | รหัสผ่านสำหรับการตรวจสอบ |

### ค่าที่ส่งกลับ

True หากรหัสผ่านเป็นค่าที่ถูกต้อง; หากไม่เป็นเช่นนั้น จะเป็น false.

### หมายเหตุ

1. คุณควรตรวจสอบคุณสมบัติ [`IsWriteProtected`](../iswriteprotected) ก่อนเรียกเมธอดนี้.  
2. เมื่อรหัสผ่านเป็น null หรือว่างเปล่า เมธอดนี้จะคืนค่า false.

### ตัวอย่าง

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

## ProtectionManager.IsOnlyDocumentPropertiesLoaded คุณสมบัติ

คุณสมบัตินี้มีความหมาย หากไฟล์พรีเซนเทชันถูกป้องกันด้วยรหัสผ่านและคุณสมบัติของเอกสารของไฟล์นี้เป็นสาธารณะ ค่า true หมายถึงมีการโหลดเฉพาะคุณสมบัติของเอกสารจากไฟล์พรีเซนเทชันที่เข้ารหัสโดยไม่ใช้รหัสผ่าน ค่า false หมายถึงมีการโหลดพรีเซนเทชันที่เข้ารหัสทั้งหมดโดยใช้รหัสผ่านที่ถูกต้อง ไม่ได้โหลดเฉพาะคุณสมบัติของเอกสาร หากพรีเซนเทชันไม่ได้ถูกเข้ารหัส ค่าคุณสมบัติก็จะเป็น false เสมอ หากคุณสมบัติของเอกสารของไฟล์ที่เข้ารหัสไม่ได้เป็นสาธารณะ ค่าคุณสมบัติก็จะเป็น false เสมอ หาก Presentation.EncryptDocumentProperties เป็น true แล้วค่าคุณสมบัติ IsOnlyDocumentPropertiesLoaded จะเป็น false เสมอ อ่านอย่างเดียว Boolean.

public bool IsOnlyDocumentPropertiesLoaded { get; }

## ProtectionManager.ReadOnlyRecommended คุณสมบัติ

รับหรือกำหนดคำแนะนำการอ่านอย่างเดียว. อ่าน/เขียน Boolean.

```csharp
public bool ReadOnlyRecommended { get; set; }
```

### ตัวอย่าง

โค้ดตัวอย่างต่อไปนี้แสดงวิธีตั้งค่า PowerPoint Presentation ให้เป็นแบบอ่านอย่างเดียวใน C# โดยใช้ Aspose.Slides.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	pres.ProtectionManager.ReadOnlyRecommended = true;
	pres.Save("ReadOnlyPresentation.pptx", SaveFormat.Pptx);
}
```

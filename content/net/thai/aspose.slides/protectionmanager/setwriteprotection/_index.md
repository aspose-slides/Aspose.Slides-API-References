## ProtectionManager.SetWriteProtection เมธอด

ตั้งการป้องกันการเขียนสำหรับพรีเซนเทชันนี้ด้วยรหัสผ่านที่ระบุ

```csharp
public void SetWriteProtection(string password)
```

| Parameter | Type | Description |
| --- | --- | --- |
| password | String | รหัสผ่าน |

### ตัวอย่าง

โค้ดตัวอย่างต่อไปนี้แสดงวิธีตั้งการป้องกันการเขียนให้กับพรีเซนเทชัน

```csharp
[C#]
using (Presentation presentation = new Presentation("pres.pptx"))
{
    presentation.ProtectionManager.SetWriteProtection("123123");
    presentation.Save("write-protected-pres.pptx", SaveFormat.Pptx);
}
```

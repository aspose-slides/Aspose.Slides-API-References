## ViewProperties.GridSpacing คุณสมบัติ

Returns or sets the grid spacing that should be used for the grid underlying the presentation document, in points. อ่าน/เขียน Single.

```csharp
public float GridSpacing { get; set; }
```

### หมายเหตุ

ค่าการห่างของกริดต้องเป็นจำนวนเป็นบวก . ช่วงค่าทั่วไปอยู่ระหว่าง 1 mm (2.8349607 points) ถึง 2 inches (144 points).

### ตัวอย่าง

โค้ดตัวอย่างต่อไปนี้แสดงวิธีการเปลี่ยนค่าการห่างของกริดในงานนำเสนอ PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

## LoadOptions.OnlyLoadDocumentProperties คุณสมบัติ

คุณสมบัตินี้มีความหมายเมื่อไฟล์การนำเสนอถูกป้องกันด้วยรหัสผ่าน. ค่าของ true หมายถึงว่าจะโหลดเฉพาะคุณสมบัติของเอกสารจากไฟล์การนำเสนอที่เข้ารหัสและละเว้นรหัสผ่าน. ค่าของ false หมายถึงว่าจะต้องโหลดการนำเสนอที่เข้ารหัสทั้งหมดโดยใช้รหัสผ่านที่ถูกต้อง. หากการนำเสนอไม่ได้ถูกเข้ารหัสค่าของคุณสมบัติก็จะถูกละเว้นเสมอ. หากคุณสมบัติของเอกสารในไฟล์ที่เข้ารหัสไม่เป็นสาธารณะและค่าของคุณสมบัติเป็น true จะไม่สามารถโหลดคุณสมบัติของเอกสารได้และจะเกิดข้อยกเว้น. อ่าน/เขียน Boolean.

```csharp
public bool OnlyLoadDocumentProperties { get; set; }
```

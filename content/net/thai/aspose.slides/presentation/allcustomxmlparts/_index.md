## Presentation.AllCustomXmlParts คุณสมบัติ

ส่งคืนส่วนข้อมูลแบบกำหนดเองทั้งหมดในงานนำเสนอ. อ่านอย่างเดียว [`ICustomXmlPart`](../../icustomxmlpart)[].

```csharp
public ICustomXmlPart[] AllCustomXmlParts { get; }
```

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีการลบส่วน XML ที่กำหนดเองทั้งหมดจากงานนำเสนอ PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation("PresentationWithCustomXml.pptx"))
{
	// วนลูปส่วน XML แบบกำหนดเองทั้งหมด
    foreach (ICustomXmlPart item in pres.AllCustomXmlParts)
    {
        item.Remove();
    }
    pres.Save("out.pptx", SaveFormat.Pptx);
}
```

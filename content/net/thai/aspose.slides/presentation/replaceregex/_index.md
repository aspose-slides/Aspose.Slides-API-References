## Presentation.ReplaceRegex เมธอด

แทนที่การจับคู่ทั้งหมดของ regular expression ด้วยสตริงที่ระบุ

```csharp
public void ReplaceRegex(Regex regex, string newText, IFindResultCallback callback)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| regex | Regex | regular expression Regex ที่ใช้เพื่อรับสตริงที่จะทำการแทนที่ |
| newText | String | สตริงที่จะใช้แทนที่การเกิดขึ้นทั้งหมดของสตริงที่ต้องการแทนที่ |
| callback | IFindResultCallback | อ็อบเจกต์ callback สำหรับรับผลการค้นหา [`IFindResultCallback`](../../ifindresultcallback) |

### ตัวอย่าง

ตัวอย่างโค้ดต่อไปนี้แสดงวิธีการแทนที่ข้อความโดยใช้ regular expression พร้อมสตริงที่ระบุ

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// แทนที่คำทั้งหมดที่มีความยาว 10 ตัวอักษรหรือมากกว่า ด้วย '***'
	presentation.ReplaceRegex(regex, "***", null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

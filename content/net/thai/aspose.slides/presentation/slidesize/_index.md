## Presentation.SlideSize คุณสมบัติ

ส่งคืนอ็อบเจกต์ขนาดสไลด์ อ่านอย่างเดียว [`ISlideSize`](../../islidesize).

```csharp
public ISlideSize SlideSize { get; }
```

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีเปลี่ยนขนาดสไลด์ใน PowerPoint Presentation.

```csharp
[C#]
using (Presentation pres = new Presentation("pres-4x3-aspect-ratio.pptx"))
{
    pres.SlideSize.SetSize(SlideSizeType.OnScreen16x9, SlideSizeScaleType.DoNotScale);
    pres.Save("pres-4x3-aspect-ratio.pptx", SaveFormat.Pptx);
}
```

ตัวอย่างต่อไปนี้แสดงวิธีตั้งค่าขนาดสไลด์โดยคำนึงถึงการสเกลเนื้อหาใน PowerPoint Presentation.

```csharp
[C#]
// สร้างอ็อบเจกต์ Presentation ที่แสดงถึงไฟล์การนำเสนอ
using(Presentation presentation = new Presentation("AccessSlides.pptx")) {
  using(Presentation auxPresentation = new Presentation()) {
    ISlide slide = presentation.Slides[0];
    // ตั้งค่าขนาดสไลด์ของการนำเสนอที่สร้างขึ้นให้เหมือนกับของต้นฉบับ
    presentation.SlideSize.SetSize(540, 720, SlideSizeScaleType.EnsureFit); // เมธอด SetSize ใช้สำหรับตั้งค่าขนาดสไลด์โดยสเกลเนื้อหาเพื่อให้พอดี
    presentation.SlideSize.SetSize(SlideSizeType.A4Paper, SlideSizeScaleType.Maximize); // เมธอด SetSize ใช้สำหรับตั้งค่าขนาดสไลด์โดยขยายขนาดเนื้อหาให้เต็ม
    // บันทึก Presentation ลงดิสก์
    auxPresentation.Save("Set_Size&Type_out.pptx", SaveFormat.Pptx);
  }
}
```

ตัวอย่างต่อไปนี้แสดงวิธีกำหนดขนาดสไลด์แบบกำหนดเองใน PowerPoint Presentation.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    pres.SlideSize.SetSize(780, 540, SlideSizeScaleType.DoNotScale); // ขนาดกระดาษ A4
    pres.Save("pres-a4-slide-size.pptx", SaveFormat.Pptx);
}
```

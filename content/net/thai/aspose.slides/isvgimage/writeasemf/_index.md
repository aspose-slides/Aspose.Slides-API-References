## ISvgImage.WriteAsEmf เมธอด

บันทึกภาพ SVG เป็นไฟล์ EMF.

public void WriteAsEmf(Stream stream)

| --- | --- |

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีบันทึกภาพ SVG เป็นไฟล์เมต้า.

```csharp
[C#]
// สร้างภาพ SVG ใหม่
ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));

// บันทึกภาพ SVG เป็นเมทาฟิลล์
using (var fileStream = System.IO.File.OpenWrite("SvgAsEmf.emf"))
    svgImage.WriteAsEmf(fileStream);
```

ตัวอย่างนี้แสดงวิธีเพิ่มภาพ SVG เป็นไฟล์เมต้าเข้าสู่คอลเลกชันภาพการนำเสนอ.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // สร้างภาพ SVG ใหม่
    ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));
    using (var memStream = new MemoryStream())
    {
        // บันทึกภาพ SVG เป็นเมทาฟิล์ม
        svgImage.WriteAsEmf(memStream);
        // เพิ่มเมทาฟิล์มลงในคอลเลกชันภาพ
        pres.Images.AddImage(memStream.ToArray());
    }
}
```

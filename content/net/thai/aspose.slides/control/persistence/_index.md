## Control.Persistence คุณสมบัติ

รับวิธีการที่ใช้ในการจัดเก็บคุณสมบัติของคอนโทรล ActiveX. อ่านอย่างเดียว [`PersistenceType`](../../persistencetype).

```csharp
public PersistenceType Persistence { get; }
```

### ตัวอย่าง

ตัวอย่างต่อไปแสดงการใช้คุณสมบัติ Persistence เพื่อตรวจสอบว่าคุณสมบัติของอ็อบเจกต์ ActiveX สามารถถูกเปลี่ยนแปลงเป็นคุณสมบัติ ActiveX ที่อิง XML ได้หรือไม่:

```csharp
[C#]
if (control.Persistence == PersistenceType.PersistPropertyBag)
{
    control.Properties["Value"] = value;
}
else
{
    YourMethodHere(control.ActiveXControlBinary); //ใช้เมธอดของคุณเองเพื่อจัดการคุณสมบัติ ActiveX ที่เก็บไว้ในไฟล์ไบนารีของมัน
}
```

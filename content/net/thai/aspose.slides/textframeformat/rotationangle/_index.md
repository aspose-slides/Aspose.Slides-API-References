## TextFrameFormat.RotationAngle คุณสมบัติ

ระบุการหมุนที่กำหนดเองซึ่งถูกนำไปใช้กับข้อความภายในกล่องกรอบ. หากไม่ได้ระบุ จะใช้การหมุนของรูปร่างที่แนบมาด้วย. หากมีการระบุ การหมุนนี้จะถูกนำไปใช้แยกจากรูปร่าง. นั่นคือรูปร่างอาจมีการหมุนโดยเพิ่มจากการหมุนของข้อความเอง. ค่าการหมุนของข้อความที่แสดงผลสรุปจากคุณสมบัตินี้และประเภทแนวตั้งที่กำหนดล่วงหน้าในคุณสมบัติ TextVerticalType. อ่าน/เขียน Single.

```csharp
public float RotationAngle { get; set; }
```

### ตัวอย่าง

พิจารณากรณีที่รูปร่างมีการหมุน 90 องศาตามเข็มนาฬิกาที่ถูกนำไปใช้. นอกจากนี้ ข้อความภายในเองยังมีการหมุน -90 องศาตามทิศทางตรงกันข้ามของเข็มนาฬิกาที่ถูกนำไปใช้. ดังนั้นรูปร่างที่ได้จะดูเหมือนถูกหมุน แต่ข้อความภายในจะดูเหมือนไม่ได้ถูกหมุนเลย.

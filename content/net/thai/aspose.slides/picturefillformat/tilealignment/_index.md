## PictureFillFormat.TileAlignment คุณสมบัติ

คืนค่า หรือ ตั้งค่า วิธีที่พื้นผิวจัดตำแหน่งภายในรูปร่าง การตั้งค่านี้ควบคุมจุดเริ่มต้นของรูปแบบพื้นผิวและวิธีที่มันทำซ้ำทั่วรูปร่าง อ่าน/เขียน [`RectangleAlignment`](../../rectanglealignment).

```csharp
public RectangleAlignment TileAlignment { get; set; }
```

### หมายเหตุ

ค่าเริ่มต้นคือ TopLeft.

### ตัวอย่าง

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // รับรูปแบบการเติมภาพของรูปร่าง
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // ตั้งค่าโหมดการเติมภาพเป็น Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // ตั้งค่าการจัดตำแหน่งการทำแผ่นกระเบื้องเป็นด้านล่างขวา
    pictureFillFormat.TileAlignment = RectangleAlignment.BottomRight;
}
```

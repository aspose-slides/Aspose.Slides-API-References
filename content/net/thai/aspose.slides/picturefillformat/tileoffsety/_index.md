## PictureFillFormat.TileOffsetY คุณสมบัติ

คืนค่า หรือกำหนดการเลื่อนแนวตั้งของพื้นผิวจากจุดกำเนิดของรูปร่างเป็นหน่วยจุด ค่าเป็นบวกจะทำให้พื้นผิวเลื่อนลง ส่วนค่าเป็นลบจะทำให้พื้นผิวเลื่อนขึ้น อ่าน/เขียน Single.

```csharp
public float TileOffsetY { get; set; }
```

### ตัวอย่าง

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // รับรูปแบบการเติมภาพของรูปร่าง
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // ตั้งโหมดการเติมภาพเป็น Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // ตั้งค่าออฟเซ็ตแนวตั้งของพื้นผิวเป็น -50 จุด
    pictureFillFormat.TileOffsetY = -50;
}
```

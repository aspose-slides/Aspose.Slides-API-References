## PictureFillFormat.TileOffsetX คุณสมบัติ

คืนค่า หรือ ตั้งค่าการเยื้องแนวนอนของพื้นผิวจากจุดกำเนิดของรูปร่างในหน่วยพิกัด จุดค่าเป็นบวกจะทำให้พื้นผิวเคลื่อนไปทางขวา ในขณะที่ค่าลบจะทำให้เคลื่อนไปทางซ้าย อ่าน/เขียน Single.

```csharp
public float TileOffsetX { get; set; }
```

### ตัวอย่าง

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // รับรูปแบบการเติมรูปภาพของรูปร่าง
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // ตั้งค่าโหมดการเติมรูปภาพเป็น Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // ตั้งค่าการเยื้องแนวนอนของพื้นผิวเป็น 20 จุด
    pictureFillFormat.TileOffsetX = 20f;
}
```

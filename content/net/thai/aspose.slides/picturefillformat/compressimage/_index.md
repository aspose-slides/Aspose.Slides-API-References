## CompressImage(bool, PicturesCompression) {#compressimage}

ทำการบีบอัดภาพโดยลดขนาดตามขนาดของรูปร่างและความละเอียดที่ระบุ โดยสามารถเลือกให้ลบพื้นที่ที่ถูกตัดออกได้ด้วย

```csharp
public bool CompressImage(bool deleteCroppedAreasOfImage, PicturesCompression resolution)
```

| พารามิเตอร์ | ชนิด | คำอธิบาย |
| --- | --- | --- |
| deleteCroppedAreasOfImage | Boolean | หากเป็น true เมธอดจะลบพื้นที่ที่ถูกตัดออกของภาพ ซึ่งอาจช่วยลดขนาดของภาพได้เพิ่มเติม |
| resolution | PicturesCompression | ความละเอียดเป้าหมายสำหรับการบีบอัด ระบุเป็นค่าของ enum [`PicturesCompression`](../../../aspose.slides.export/picturescompression) |

### ค่าที่ส่งคืน

Boolean ที่ระบุว่าภาพถูกบีบอัดสำเร็จหรือไม่ จะคืนค่า `true` หากภาพถูกปรับขนาดหรือถูกตัดออก มิฉะนั้นจะคืนค่า `false`

### ข้อยกเว้น

| ข้อยกเว้น | เงื่อนไข |
| --- | --- |
| ArgumentException | ถูกใส่เมื่อความละเอียดไม่เป็นค่าที่ถูกต้อง |

### หมายเหตุ

เมธอดนี้เปลี่ยนขนาดและความละเอียดของภาพเช่นเดียวกับคุณลักษณะ "Picture Format -> Compress Pictures" ของ PowerPoint

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีใช้เมธอด `CompressImage` เพื่อลดขนาดของภาพในงานนำเสนอโดยกำหนดความละเอียดเป้าหมายและลบพื้นที่ที่ถูกตัดออก:

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;
    // บีบอัดภาพด้วยความละเอียดเป้าหมาย 150 DPI (ความละเอียดเว็บ) และลบพื้นที่ที่ถูกครอปออก
    bool result = picFrame.PictureFormat.CompressImage(true, PicturesCompression.Dpi150);
}
```

## PictureFillFormat.DeletePictureCroppedAreas เมธอด

ลบพื้นที่ที่ถูกครอปของรูปภาพที่เติม.

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### ค่าที่ส่งกลับ

ภาพที่ถูกครอปหรือภาพต้นฉบับหากไม่จำเป็นต้องครอป.

### หมายเหตุ

เมธอดนี้แปลงไฟล์เมตาฟาย WMF/EMF เป็นภาพ PNG แบบแรสเตอร์พร้อมการครอป.

### ตัวอย่าง

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // ดึง PictureFrame
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // ลบพื้นที่ครอปของรูปภาพ PictureFrame
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

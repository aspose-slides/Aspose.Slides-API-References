## LayoutPlaceholderManager.AddMediaPlaceholder เมธอด

เพิ่มรูปทรง placeholder ใหม่ลงในสไลด์เลเอาต์เพื่อเก็บอ็อบเจ็กต์สื่อ

```csharp
public IAutoShape AddMediaPlaceholder(float x, float y, float width, float height)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| x | Single | พิกัด X ของรูปทรง placeholder ใหม่ |
| y | Single | พิกัด Y ของรูปทรง placeholder ใหม่ |
| width | Single | ความกว้างของรูปทรง placeholder ใหม่ |
| height | Single | ความสูงของรูปทรง placeholder ใหม่ |

### ค่าที่ส่งกลับ

สร้าง [`IAutoShape`](../../iautoshape) พร้อม placeholder Media

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีการเพิ่มรูปทรง placeholder Media ลงในสไลด์เลเอาต์

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddMediaPlaceholder(20, 20, 200, 200);
}
```

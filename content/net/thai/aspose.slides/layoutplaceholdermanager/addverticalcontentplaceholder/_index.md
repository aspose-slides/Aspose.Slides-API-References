## LayoutPlaceholderManager.AddVerticalContentPlaceholder เมธอด

เพิ่มรูปทรง placeholder ใหม่ลงในสไลด์เค้าโครงเพื่อเก็บเนื้อหา เช่น รูปภาพ, ตาราง, สื่อ หรือข้อความในแนวแนวตั้ง

```csharp
public IAutoShape AddVerticalContentPlaceholder(float x, float y, float width, float height)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| x | Single | พิกัด X ของรูปทรง placeholder ใหม่ |
| y | Single | พิกัด Y ของรูปทรง placeholder ใหม่ |
| width | Single | ความกว้างของรูปทรง placeholder ใหม่ |
| height | Single | ความสูงของรูปทรง placeholder ใหม่ |

### ค่าที่ส่งกลับ

Created [`IAutoShape`](../../iautoshape) with a Content (Vertical) placeholder.

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีการเพิ่มรูปทรง placeholder Content (Vertical) ลงในสไลด์เค้าโครง

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalContentPlaceholder(20, 20, 300, 500);
}
```

## LayoutPlaceholderManager.AddSmartArtPlaceholder เมธอด

เพิ่มรูปทรงตัวคั่นใหม่ลงในสไลด์เค้าโครงเพื่อเก็บภาพแบบ SmartArt.

```csharp
public IAutoShape AddSmartArtPlaceholder(float x, float y, float width, float height)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| x | Single | The X coordinate of the new placeholder shape. |
| y | Single | The Y coordinate of the new placeholder shape. |
| width | Single | The width of the new placeholder shape. |
| height | Single | The height of the new placeholder shape. |

### ค่าที่ส่งคืน

สร้าง [`IAutoShape`](../../iautoshape) ด้วยตัวคั่น SmartArt.

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีการเพิ่มรูปทรงตัวคั่น SmartArt ลงในสไลด์เค้าโครง.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddSmartArtPlaceholder(20, 20, 200, 200);
}
```

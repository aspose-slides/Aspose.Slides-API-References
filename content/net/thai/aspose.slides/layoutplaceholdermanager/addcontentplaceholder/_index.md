## LayoutPlaceholderManager.AddContentPlaceholder เมธอด

เพิ่ม placeholder shape ใหม่ไปยังสไลด์เค้าโครงเพื่อเก็บเนื้อหา เช่น รูปภาพ ตาราง สื่อ หรือข้อความ

```csharp
public IAutoShape AddContentPlaceholder(float x, float y, float width, float height)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| x | Single | พิกัด X ของรูปทรง placeholder ใหม่. |
| y | Single | พิกัด Y ของรูปทรง placeholder ใหม่. |
| width | Single | ความกว้างของรูปทรง placeholder ใหม่. |
| height | Single | ความสูงของรูปทรง placeholder ใหม่. |

### ค่าที่ส่งกลับ

สร้าง [`IAutoShape`](../../iautoshape) ด้วย placeholder ชนิด Content.

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีการเพิ่มรูปทรง Content placeholder ไปยังสไลด์เค้าโครง

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddContentPlaceholder(20, 20, 500, 300);
}
```

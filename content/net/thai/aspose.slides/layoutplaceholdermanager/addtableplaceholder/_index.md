## LayoutPlaceholderManager.AddTablePlaceholder เมธอด

เพิ่มรูปทรง placeholder ใหม่ลงในสไลด์เลย์เอาต์เพื่อเก็บ Table

```csharp
public IAutoShape AddTablePlaceholder(float x, float y, float width, float height)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| x | Single | พิกัด X ของรูปทรง placeholder ใหม่ |
| y | Single | พิกัด Y ของรูปทรง placeholder ใหม่ |
| width | Single | ความกว้างของรูปทรง placeholder ใหม่ |
| height | Single | ความสูงของรูปทรง placeholder ใหม่ |

### ค่าที่ส่งคืน

สร้าง [`IAutoShape`](../../iautoshape) ด้วย placeholder ของ Table

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีเพิ่มรูปทรง placeholder ของ Table ลงในสไลด์เลย์เอาต์

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTablePlaceholder(20, 20, 500, 200);
}
```

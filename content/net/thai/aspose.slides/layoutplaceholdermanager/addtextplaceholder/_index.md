## LayoutPlaceholderManager.AddTextPlaceholder เมธอด

เพิ่มรูปทรงตัวแทนใหม่ลงในสไลด์เลย์เอาต์เพื่อเก็บเนื้อหาแบบข้อความ

```csharp
public IAutoShape AddTextPlaceholder(float x, float y, float width, float height)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| x | Single | พิกัด X ของรูปทรงตัวแทนใหม่ |
| y | Single | พิกัด Y ของรูปทรงตัวแทนใหม่ |
| width | Single | ความกว้างของรูปทรงตัวแทนใหม่ |
| height | Single | ความสูงของรูปทรงตัวแทนใหม่ |

### ผลลัพธ์

สร้าง [`IAutoShape`](../../iautoshape) ด้วยตัวแทน Text

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีเพิ่มรูปทรงตัวแทน Text ลงในสไลด์เลย์เอาต์

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTextPlaceholder(20, 20, 500, 300);
}
```

## LayoutPlaceholderManager.AddVerticalTextPlaceholder วิธีการ

เพิ่มรูปแบบ placeholder ใหม่ลงในสไลด์เทมเพลตเพื่อเก็บเนื้อหาข้อความในแนวตั้ง

```csharp
public IAutoShape AddVerticalTextPlaceholder(float x, float y, float width, float height)
```

| พารามิเตอร์ | ชนิด | คำอธิบาย |
| --- | --- | --- |
| x | Single | พิกัด X ของรูปร่าง placeholder ใหม่ |
| y | Single | พิกัด Y ของรูปร่าง placeholder ใหม่ |
| width | Single | ความกว้างของรูปร่าง placeholder ใหม่ |
| height | Single | ความสูงของรูปร่าง placeholder ใหม่ |

### ค่าที่ส่งกลับ

สร้าง [`IAutoShape`](../../iautoshape) ด้วย placeholder ข้อความ (แนวตั้ง)

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีเพิ่มรูปร่าง placeholder ข้อความ (แนวตั้ง) ลงในสไลด์เทมเพลต

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalTextPlaceholder(20, 20, 300, 500);
}
```

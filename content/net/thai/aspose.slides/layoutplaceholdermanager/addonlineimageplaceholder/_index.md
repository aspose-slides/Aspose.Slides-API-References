## LayoutPlaceholderManager.AddOnlineImagePlaceholder เมธอด

เพิ่มรูปทรง placeholder ใหม่ลงในสไลด์เลย์เอาต์เพื่อเก็บรูปภาพออนไลน์

```csharp
public IAutoShape AddOnlineImagePlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| x | Single | พิกัด X ของรูปทรง placeholder ใหม่ |
| y | Single | พิกัด Y ของรูปทรง placeholder ใหม่ |
| width | Single | ความกว้างของรูปทรง placeholder ใหม่ |
| height | Single | ความสูงของรูปทรง placeholder ใหม่ |

### ค่าที่ส่งคืน

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีเพิ่มรูปทรงตำแหน่งที่เก็บรูปออนไลน์ลงในสไลด์เลย์เอาต์

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
    IAutoShape placeholder = layout.PlaceholderManager.AddOnlineImagePlaceholder(20, 20, 200, 200);
}
```

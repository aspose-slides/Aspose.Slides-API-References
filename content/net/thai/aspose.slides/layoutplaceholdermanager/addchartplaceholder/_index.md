## LayoutPlaceholderManager.AddChartPlaceholder เมธอด

เพิ่มรูปทรงตัวเติมที่ใหม่ลงในสไลด์เค้าโครงเพื่อบรรจุแผนภูมิ.

```csharp
public IAutoShape AddChartPlaceholder(float x, float y, float width, float height)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| x | Single | พิกัด X ของรูปทรงตัวเติมใหม่ |
| y | Single | พิกัด Y ของรูปทรงตัวเติมใหม่ |
| width | Single | ความกว้างของรูปทรงตัวเติมใหม่ |
| height | Single | ความสูงของรูปทรงตัวเติมใหม่ |

### ค่าที่ส่งคืน

สร้าง [`IAutoShape`](../../iautoshape) พร้อมตัวเติม Chart.

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีการเพิ่มรูปทรงตัวเติม Chart ลงในสไลด์เค้าโครง.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddChartPlaceholder(20, 20, 200, 200);
}
```

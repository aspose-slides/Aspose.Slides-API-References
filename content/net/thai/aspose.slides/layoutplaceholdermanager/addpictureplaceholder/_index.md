## LayoutPlaceholderManager.AddPicturePlaceholder เมธอด

เพิ่มรูปทรงตำแหน่งเก็บชั่วคราวใหม่ลงในสไลด์เค้าโครงเพื่อเก็บรูปภาพ

```csharp
public IAutoShape AddPicturePlaceholder(float x, float y, float width, float height)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| x | Single | พิกัด X ของรูปทรงตำแหน่งเก็บชั่วคราวใหม่ |
| y | Single | พิกัด Y ของรูปทรงตำแหน่งเก็บชั่วคราวใหม่ |
| width | Single | ความกว้างของรูปทรงตำแหน่งเก็บชั่วคราวใหม่ |
| height | Single | ความสูงของรูปทรงตำแหน่งเก็บชั่วคราวใหม่ |

### ค่าที่ส่งคืน

สร้าง [`IAutoShape`](../../iautoshape) พร้อมตำแหน่งเก็บรูปภาพ

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีเพิ่มรูปทรงตำแหน่งเก็บรูปภาพลงในสไลด์เค้าโครง

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddPicturePlaceholder(20, 20, 200, 200);
}
```

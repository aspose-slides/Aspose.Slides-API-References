## IFonts.GetScriptFont เมธอด

รับชื่อแบบอักษรที่เชื่อมโยงกับแท็กสคริปต์เฉพาะจากธีมการนำเสนอ

```csharp
public string GetScriptFont(string script)
```

| พารามิเตอร์ | ประเภท | รายละเอียด |
| --- | --- | --- |
| script | String | โค้ดสคริปต์ BCP-47 (เช่น "Latn", "Cyrl", "Jpan") ที่ใช้เพื่อระบุระบบการเขียน |

### ค่าที่ส่งคืน

ชื่อแบบอักษรที่ใช้สำหรับสคริปต์ที่ระบุ, หรือ `null` หากสคริปต์ไม่ได้กำหนด

### ตัวอย่าง

ตัวอย่างนี้แสดงวิธีการดึงแบบอักษรที่กำหนดให้กับสคริปต์ Cyrillic ในธีมการนำเสนอ

```csharp
[C#]
string font = presentation.MasterTheme.FontScheme.Major.GetScriptFont("Cyrl");
Console.WriteLine("Font for Cyrillic script: " + font);
```

## IFonts.SetScriptFont เมธอด

กำหนดชื่อฟอนต์ให้กับแท็กสคริปต์เฉพาะ ซึ่งกำหนดวิธีการเรนเดอร์ข้อความของสคริปต์นั้นในงานนำเสนอ

```csharp
public void SetScriptFont(string script, string fontName)
```

| Parameter | Type | Description |
| --- | --- | --- |
| script | String | รหัสสคริปต์ BCP-47 (เช่น "Arab", "Hebr", "Hans") ที่ระบุระบบการเขียน |
| fontName | String | ชื่อฟอนต์ที่จะกำหนดให้สคริปต์ที่ระบุ |

### ตัวอย่าง

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segoe UI");
```

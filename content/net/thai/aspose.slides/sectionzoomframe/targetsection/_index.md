## SectionZoomFrame.TargetSection คุณสมบัติ

รับหรือกำหนดอ็อบเจกต์ส่วนที่วัตถุ Section Zoom เชื่อมโยงไปยัง. อ่าน/เขียน [`ISection`](../../isection).

```csharp
public ISection TargetSection { get; set; }
```

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงการเปลี่ยนส่วนเป้าหมายและสร้างรูปภาพใหม่สำหรับวัตถุ section zoom:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
  ISectionZoomFrame sectionZoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1]);
  sectionZoomFrame.TargetSection = pres.Sections[2];
}
```

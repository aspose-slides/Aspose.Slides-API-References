## ISlideShowTransition.AdvanceAfter คุณสมบัติ

คุณลักษณะนี้ระบุว่าการนำเสนอภาพสไลด์จะย้ายไปสไลด์ต่อไปหลังจากระยะเวลาที่กำหนดหรือไม่. อ่าน/เขียน Boolean.

```csharp
public bool AdvanceAfter { get; set; }
```

### ตัวอย่าง

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    // ดึงการเปลี่ยนสไลด์แรก
    ISlideShowTransition slideTransition = pres.Slides[0].SlideShowTransition;
    
    // ตรวจสอบว่าธง Advance Slide After ถูกตั้งหรือไม่
    if (slideTransition.AdvanceAfter)
    {
        // รับค่าเวลา Advance Slide After
        uint advanceAfterTime = slideTransition.AdvanceAfterTime;
    }
}
```

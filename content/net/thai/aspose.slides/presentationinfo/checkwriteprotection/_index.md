## PresentationInfo.CheckWriteProtection เมธอด

ตรวจสอบว่ารหัสผ่านสำหรับการแก้ไขถูกต้องสำหรับงานนำเสนอที่ป้องกันการเขียนหรือไม่

```csharp
public bool CheckWriteProtection(string password)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| password | String | รหัสผ่านที่ต้องตรวจสอบ. |

### ค่าที่คืน

ส่งกลับ true หากงานนำเสนอถูกป้องกันการเขียนและ password ถูกต้อง มิฉะนั้นส่งกลับ false.

### ข้อยกเว้น

| ข้อยกเว้น | เงื่อนไข |
| --- | --- |
| InvalidOperationException |  |

### หมายเหตุ

1. คุณควรตรวจสอบคุณสมบัติ [`IsWriteProtected`](../iswriteprotected) ก่อนเรียกเมธอดนี้. 2. เมื่อ password มีค่า null หรือว่างเปล่า เมธอดนี้จะส่งกลับ false.

### ตัวอย่าง

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
if (info.IsWriteProtected == NullableBool.True)
{
    bool isWriteProtectedByPassword = info.CheckWriteProtection("my_password");
}
```

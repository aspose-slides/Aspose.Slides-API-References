## PresentationInfo.CheckPassword เมธอด

ตรวจสอบว่ารหัสผ่านที่ให้มาถูกต้องสำหรับพรีเซนเทชันที่มีการป้องกันด้วยรหัสผ่านเปิดหรือไม่

```csharp
public bool CheckPassword(string password)
```

| พารามิเตอร์ | ชนิด | คำอธิบาย |
| --- | --- | --- |
| password | String | รหัสผ่านที่ต้องตรวจสอบ |

### ค่าที่คืนกลับ

True หากพรีเซนเทชันถูกป้องกันด้วยรหัสผ่านเปิดและรหัสผ่านถูกต้อง และ false ในกรณีอื่น

### ข้อยกเว้น

| ข้อยกเว้น | เงื่อนไข |
| --- | --- |
| InvalidOperationException |  |
| NotSupportedException |  |

### หมายเหตุ

เมื่อรหัสผ่านเป็น null หรือว่างเปล่า เมธอดนี้จะคืนค่า false

### ตัวอย่าง

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
bool isPasswordCorrect = info.CheckPassword("my_password");
```

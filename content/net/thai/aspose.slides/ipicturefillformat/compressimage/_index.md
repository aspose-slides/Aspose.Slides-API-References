## CompressImage(bool, PicturesCompression) {#compressimage}

ทำการบีบอัดภาพโดยลดขนาดตามขนาดของรูปร่างและความละเอียดที่ระบุ โดยอาจลบส่วนที่ถูกตัดออกได้เช่นกัน.

```csharp
public bool CompressImage(bool deleteCroppedAreasOfImage, PicturesCompression resolution)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| deleteCroppedAreasOfImage | Boolean | หากเป็น true เมธอดจะลบส่วนที่ถูกตัดออกของภาพ ซึ่งอาจทำให้ขนาดลดลงเพิ่มเติม |
| resolution | PicturesCompression | ความละเอียดเป้าหมายสำหรับการบีบอัด ระบุเป็นค่าของ enum [`PicturesCompression`](../../../aspose.slides.export/picturescompression) |

### ค่าที่ส่งคืน

Boolean ที่บ่งชี้ว่าภาพถูกบีบอัดสำเร็จหรือไม่ คืนค่า `true` หากภาพถูกปรับขนาดหรือถูกตัดออก มิฉะนั้นคืนค่า `false`.

### ข้อยกเว้น

| ข้อยกเว้น | เงื่อนไข |
| --- | --- |
| ArgumentException | เกิดเมื่อความละเอียดไม่เป็นค่าที่ถูกต้อง |

### หมายเหตุ

เมธอดนี้เปลี่ยนขนาดและความละเอียดของภาพคล้ายกับคุณลักษณะ "Picture Format -> Compress Pictures" ของ PowerPoint.

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีการใช้เมธอด `CompressImage` เพื่อลดขนาดของภาพในงานนำเสนอโดยตั้งค่าความละเอียดเป้าหมายและลบส่วนที่ถูกตัดออก:

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;
    // บีบอัดภาพด้วยความละเอียดเป้าหมาย 150 DPI (ความละเอียดเว็บ) และลบส่วนที่ถูกตัดออก
    bool result = picFrame.PictureFormat.CompressImage(true, PicturesCompression.Dpi150);
}
```

## IPictureFillFormat.DeletePictureCroppedAreas เมธอด

ลบพื้นที่ที่ถูกครอปของ Picture ที่เติม

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### ค่าที่ส่งคืน

ภาพที่ถูกครอปหรือภาพต้นทางหากไม่จำเป็นต้องครอป

### หมายเหตุ

เมธอดนี้แปลงไฟล์เมต้าไฟล์ WMF/EMF เป็นภาพ PNG แบบแรสเตอร์พร้อมทำการครอป

### ตัวอย่าง

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // รับ PictureFrame
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // ลบพื้นที่ที่ถูกครอปของรูปภาพ PictureFrame
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

## IPictureFillFormat.TileAlignment คุณสมบัติ

คืนค่า หรือ กำหนดว่าลวดลายถูกจัดตำแหน่งอย่างไรภายในรูปทรง การตั้งค่านี้ควบคุมจุดเริ่มต้นของแบบลวดลายและวิธีที่มันทำซ้ำทั่วรูปทรง อ่าน/เขียน [`RectangleAlignment`](../../rectanglealignment).

```csharp
public RectangleAlignment TileAlignment { get; set; }
```

### หมายเหตุ

ค่าเริ่มต้นคือ TopLeft.

### ตัวอย่าง

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // รับรูปแบบการเติมภาพของรูปร่าง
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // ตั้งโหมดการเติมภาพเป็น Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // ตั้งการจัดตำแหน่งของการเรียงรูปแบบเป็นด้านล่างขวา
    pictureFillFormat.TileAlignment = RectangleAlignment.BottomRight;
}
```

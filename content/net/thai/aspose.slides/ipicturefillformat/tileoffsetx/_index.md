## คุณลักษณะ IPictureFillFormat.TileOffsetX

คืนค่า หรือ กำหนด การจัดตำแหน่งแนวนอนของพื้นผิวจากจุดกำเนิดของรูปร่างเป็นหน่วยจุด. ค่าบวกจะย้ายพื้นผิวไปทางขวา, ส่วนค่าลบจะย้ายไปทางซ้าย. อ่าน/เขียน Single.

```csharp
public float TileOffsetX { get; set; }
```

### ตัวอย่าง

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // รับรูปแบบการเติมรูปภาพของรูปร่าง
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // ตั้งโหมดการเติมรูปภาพเป็น Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // ตั้งค่าการจัดตำแหน่งแนวนอนของพื้นผิวเป็น 20 จุด
    pictureFillFormat.TileOffsetX = 20f;
}
```

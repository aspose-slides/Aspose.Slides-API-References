## IPictureFillFormat.TileOffsetY คุณสมบัติ

คืนค่า หรือ ตั้งค่าการเยื้องแนวตั้งของพื้นผิวจากจุดกำเนิดของรูปร่างเป็นหน่วยจุด. ค่าเป็นบวกจะทำให้พื้นผิวเลื่อนลง, ในขณะที่ค่าลบจะทำให้พื้นผิวเลื่อนขึ้น. อ่าน/เขียน Single.

```csharp
public float TileOffsetY { get; set; }
```

### ตัวอย่าง

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // รับรูปแบบการเติมรูปภาพของรูปร่าง
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // ตั้งค่าโหมดการเติมรูปภาพเป็น Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // ตั้งค่าการเยื้องแนวตั้งของพื้นผิวเป็น -50 จุด
    pictureFillFormat.TileOffsetY = -50;
}
```

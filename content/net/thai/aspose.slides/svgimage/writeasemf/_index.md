## SvgImage.WriteAsEmf เมธอด

บันทึกภาพ SVG เป็นไฟล์ EMF

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีการบันทึกภาพ SVG เป็นเมตาไฟล์.

```csharp
[C#]
// สร้างภาพ SVG ใหม่
ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));

// บันทึกภาพ SVG เป็นไฟล์เมตา
using (var fileStream = System.IO.File.OpenWrite("SvgAsEmf.emf"))
    svgImage.WriteAsEmf(fileStream);
```

ตัวอย่างนี้แสดงวิธีการเพิ่มภาพ SVG เป็นเมตาไฟล์เข้าไปในคอลเลกชันภาพของการนำเสนอ.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // สร้างภาพ SVG ใหม่
    ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));
    using (var memStream = new MemoryStream())
    {
        // บันทึกภาพ SVG เป็นไฟล์เมตา
        svgImage.WriteAsEmf(memStream);
        // เพิ่มไฟล์เมตาไปยังคอลเลกชันภาพ
        pres.Images.AddImage(memStream.ToArray());
    }
}
```

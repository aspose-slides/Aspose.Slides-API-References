## ILoadOptions.OnlyLoadDocumentProperties คุณสมบัติ

คุณสมบัตินี้มีความหมายเมื่อไฟล์พรีเซนเทชันถูกป้องกันด้วยรหัสผ่าน. ค่า true หมายความว่าจะโหลดเฉพาะคุณสมบัติของเอกสารจากไฟล์พรีเซนเทชันที่เข้ารหัสและจะละเว้นรหัสผ่าน. ค่า false หมายความว่าจะโหลดพรีเซนเทชันที่เข้ารหัสทั้งหมดโดยใช้รหัสผ่านที่ถูกต้อง. หากพรีเซนเทชันไม่ได้เข้ารหัสค่า คุณสมบัติ จะถูกละเว้นเสมอ. หากคุณสมบัติของเอกสารในไฟล์ที่เข้ารหัสไม่เป็นสาธารณะและค่า คุณสมบัติ เป็น true แล้วคุณสมบัติของเอกสารจะไม่สามารถโหลดได้และจะเกิดข้อยกเว้น. Boolean แบบอ่าน-เขียน.

```csharp
public bool OnlyLoadDocumentProperties { get; set; }
```

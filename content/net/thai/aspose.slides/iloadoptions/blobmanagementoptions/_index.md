## ILoadOptions.BlobManagementOptions คุณสมบัติ

แสดงถึงตัวเลือกที่สามารถใช้ในการจัดการพฤติกรรมการจัดการ Binary Large Objects (BLOBs) เช่น การใช้ไฟล์ชั่วคราวหรือจำนวนไบต์สูงสุดของ BLOBs ในหน่วยความจำ ตัวเลือกเหล่านี้มีวัตถุประสงค์เพื่อกำหนดอัตราส่วนระหว่างประสิทธิภาพและการใช้หน่วยความจำที่ดีที่สุดสำหรับสภาพแวดล้อมหรือความต้องการเฉพาะ A Binary Large Object (BLOB) คือข้อมูลไบเนอร์รีที่เก็บเป็นเอนทิตีเดียว - เช่น BLOB สามารถเป็นไฟล์เสียง วิดีโอ หรือการนำเสนอเอง

public IBlobManagementOptions BlobManagementOptions { get; set; }

## MasterSlideCollection.InsertClone เมธอด

แทรกสำเนาของสไลด์มาสเตอร์ที่ระบุไปยังตำแหน่งที่กำหนดของคอลเลกชัน สไลด์เลย์เอาต์ที่เชื่อมโยงจะถูกคัดลอกด้วย

```csharp
public IMasterSlide InsertClone(int index, IMasterSlide sourceMaster)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| index | Int32 | ดัชนีของสไลด์ใหม่ |
| sourceMaster | IMasterSlide | สไลด์ที่ต้องการคัดลอก |

### ค่าที่ส่งคืน

สไลด์มาสเตอร์ที่แทรก

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีการคัดลอกสไลด์มาสเตอร์ใน PowerPoint Presentation อื่น

```csharp
[C#]
// สร้างอินสแตนซ์ของคลาส Presentation เพื่อโหลดไฟล์การนำเสนอต้นฉบับ
using (Presentation srcPres = new Presentation("CloneToAnotherPresentationWithMaster.pptx"))
{
    // สร้างอินสแตนซ์ของคลาส Presentation สำหรับการนำเสนอปลายทาง (ที่สไลด์จะถูกคัดลอก)
    using (Presentation destPres = new Presentation())
    {
        // สร้างอินสแตนซ์ของ ISlide จากคอลเลกชันสไลด์ในการนำเสนอต้นฉบับพร้อมกับ
        // สไลด์มาสเตอร์
        ISlide SourceSlide = srcPres.Slides[0];
        IMasterSlide SourceMaster = SourceSlide.LayoutSlide.MasterSlide;
		// ดึงสไลด์มาสเตอร์ของการนำเสนอปลายทาง
        IMasterSlideCollection masters = destPres.Masters;
        // คัดลอกสไลด์มาสเตอร์ที่ต้องการจากการนำเสนอต้นฉบับไปยังคอลเลกชันมาสเตอร์ใน
        // การนำเสนอปลายทาง
        IMasterSlide iSlide = masters.AddClone(SourceMaster);
        // คอลเลกชันสไลด์ในการนำเสนอปลายทาง
        ISlideCollection slds = destPres.Slides;
		// คัดลอกสไลด์ต้นฉบับไปยังคอลเลกชันสไลด์ปลายทาง.
        slds.AddClone(SourceSlide, iSlide, true);
        // บันทึกการนำเสนอปลายทางลงดิสก์
        destPres.Save("CloneToAnotherPresentationWithMaster_out.pptx", SaveFormat.Pptx);
    }
}
```

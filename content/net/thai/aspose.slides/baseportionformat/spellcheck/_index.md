## BasePortionFormat.SpellCheck คุณสมบัติ

รับหรือกำหนดค่าที่บ่งชี้ว่าการตรวจสอบการสะกดคำเปิดใช้งานสำหรับส่วนข้อความหรือไม่ เมื่อคุณสมบัตินี้ถูกตั้งค่าเป็น `false` การตรวจสอบการสะกดสำหรับองค์ประกอบข้อความจะถูกระงับ เมื่อถูกตั้งค่าเป็น `true` การตรวจสอบการสะกดจะได้รับอนุญาต ค่าเริ่มต้นคือ `false`.

```csharp
public bool SpellCheck { get; set; }
```

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงการเปิดใช้แฟล็ก SpellCheck ก่อนบันทึกพรีเซนเทชัน:

```csharp
[C#]
using (var pres = new Presentation("input.pptx"))
{
    // เข้าถึงส่วนข้อความแรกภายในรูปร่างแรกบนสไลด์แรก
    var portion = ((AutoShape)pres.Slides[0].Shapes[0]).TextFrame.Paragraphs[0].Portions[0];
    // เปิดการตรวจสอบการสะกดสำหรับส่วนข้อความนี้
    portion.PortionFormat.SpellCheck = true;
    // บันทึกพรีเซนเทชันที่แก้ไขแล้ว
    pres.Save("output-with-spellcheck.pptx", SaveFormat.Pptx);
}
```

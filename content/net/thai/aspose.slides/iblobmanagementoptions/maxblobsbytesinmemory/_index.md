## IBlobManagementOptions.MaxBlobsBytesInMemory คุณสมบัติ

กำหนดขนาดรวมสูงสุด (เป็นไบต์) ที่ BLOB ทั้งหมดอาจใช้ในหน่วยความจำ โดยค่าเริ่มต้น BLOB ทั้งหมดจะถูกโหลดเข้าสู่หน่วยความจำ; เมื่อถึงขีดจำกัดนี้แล้วจึงใช้กลไกทางเลือก (เช่น ไฟล์ชั่วคราว) การเก็บ BLOB ในหน่วยความจำจะเพิ่มประสิทธิภาพสูงสุดแต่สามารถทำให้การใช้หน่วยความจำสูงขึ้น ใช้คุณสมบัตินี้เพื่อปรับพฤติกรรมให้เหมาะกับสภาพแวดล้อมหรือความต้องการของคุณ

```csharp
public ulong MaxBlobsBytesInMemory { get; set; }
```

### หมายเหตุ

คุณสมบัตินี้จะถูกละเลย หาก [`IsTemporaryFilesAllowed`](../istemporaryfilesallowed) ถูกตั้งค่าเป็น false เนื่องจากหน่วยความจำจะเป็นที่จัดเก็บเดียวที่มีอยู่และการจำกัดการใช้ BLOB ในหน่วยความจำจะไม่มีผล

ค่าเริ่มต้นคือ 629,145,600 ไบต์ (600 MB)

คุณสามารถตั้งคุณสมบัตินี้เป็นศูนย์ได้ แต่ยังคงมีการจองหน่วยความจำขั้นต่ำจำนวนเล็กน้อย

## IBlobManagementOptions.PresentationLockingBehavior คุณสมบัติ

คุณสมบัตินี้กำหนดว่าตัวอย่างของคลาส Presentation สามารถเป็นเจ้าของแหล่งข้อมูล - ไฟล์หรือสตรีม ในช่วงอายุของตัวอย่างได้หรือไม่ หากตัวอย่างเป็นเจ้าของ จะทำการล็อกแหล่งข้อมูล สิ่งนี้ช่วยปรับปรุงการใช้หน่วยความจำและประสิทธิภาพเมื่อทำงานกับ BLOBs แต่แหล่งข้อมูล (สตรีมหรือไฟล์) ไม่สามารถเปลี่ยนแปลงได้ในช่วงอายุของตัวอย่าง Presentation นี้เป็นตัวอย่าง:

```csharp
public PresentationLockingBehavior PresentationLockingBehavior { get; set; }
```

### ตัวอย่าง

```csharp
LoadOptions loadOptions = new LoadOptions { 
  BlobManagementOptions = {PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked} };
using (Presentation pres = new Presentation("pres.pptx", loadOptions)) {
  // IOException จะถูกโยนเนื่องจาก pres.pptx ถูกล็อกตลอดอายุการใช้งานของ Presentation
  // File.Delete("pres.pptx");
}
// หลังจากวัตถุ Presentation ถูกทำลาย ไฟล์จะถูกปลดล็อกและสามารถลบได้
File.Delete("pres.pptx");
```

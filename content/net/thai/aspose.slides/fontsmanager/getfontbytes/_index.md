## FontsManager.GetFontBytes เมธอด

ดึงอาร์เรย์ของไบต์ที่แทนข้อมูลฟอนต์สำหรับสไตล์ฟอนต์ที่ระบุและข้อมูลฟอนต์

```csharp
public byte[] GetFontBytes(IFontData fontData, FontStyleType fontStyle)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| fontData | IFontData | วัตถุข้อมูลฟอนต์ที่บรรจุข้อมูลเกี่ยวกับฟอนต์ [`IFontData`](../../ifontdata) |
| fontStyle | FontStyleType | สไตล์ของฟอนต์ที่ต้องการดึงข้อมูล [`FontStyleType`](../../fontstyletype) |

### ค่าที่คืนกลับ

อาร์เรย์ของไบต์ที่บรรจุข้อมูลฟอนต์สำหรับสไตล์ฟอนต์ที่ระบุ หากไม่พบข้อมูลฟอนต์หรือสไตล์จะคืนค่า null

### ตัวอย่าง

```csharp
[C#]
using (Presentation pres = new Presentation ("Presentation.pptx"))
{
    // ดึงฟอนต์ทั้งหมดที่ใช้ในงานนำเสนอ
    IFontData[] fonts = pres.FontsManager.GetFonts();

    // รับอาร์เรย์ไบต์ที่แสดงสไตล์ปกติของฟอนต์แรกในงานนำเสนอ
    byte[] bytes = pres.FontsManager.GetFontBytes(fonts[0], FontStyleType.Regular);
}
```

## Fonts.GetScriptFont เมธอด

ดึงชื่อแบบอักษรที่เชื่อมโยงกับแท็กสคริปต์เฉพาะจากธีมของงานนำเสนอ

```csharp
public string GetScriptFont(string script)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| script | String | รหัสสคริปต์ BCP-47 (เช่น "Latn", "Cyrl", "Jpan") ที่ใช้เพื่อระบุระบบการเขียน |

### ค่าที่ส่งกลับ

ชื่อแบบอักษรที่ใช้สำหรับสคริปต์ที่ระบุ, หรือ `null` หากสคริปต์ไม่ได้กำหนด

### ตัวอย่าง

ตัวอย่างนี้สาธิตวิธีการดึงแบบอักษรที่กำหนดให้กับสคริปต์ Cyrillic ในธีมของงานนำเสนอ

```csharp
[C#]
string font = presentation.MasterTheme.FontScheme.Major.GetScriptFont("Cyrl");
Console.WriteLine("Font for Cyrillic script: " + font);
```

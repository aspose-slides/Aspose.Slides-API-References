## Fonts.SetScriptFont เมธอด

กำหนดชื่อแบบอักษรให้กับแท็ก script เฉพาะ ซึ่งกำหนดวิธีการแสดงข้อความของ script นั้นในงานนำเสนอ.

```csharp
public void SetScriptFont(string script, string fontName)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| script | String | รหัส script BCP-47 (เช่น "Arab", "Hebr", "Hans") ที่ระบุระบบการเขียน |
| fontName | String | ชื่อแบบอักษรที่ต้องการกำหนดให้กับ script ที่ระบุ |

### ตัวอย่าง

ตัวอย่างนี้แสดงวิธีตั้งค่าแบบอักษรสำหรับ script Arabic เป็น "Segoe UI":

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segoe UI");
```

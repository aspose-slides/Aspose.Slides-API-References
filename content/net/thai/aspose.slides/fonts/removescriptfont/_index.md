## Fonts.RemoveScriptFont เมธอด

ลบการตั้งค่าแบบอักษรที่เชื่อมโยงกับแท็กสคริปต์เฉพาะออกจากคอลเลกชันแบบอักษรของธีม

```csharp
public void RemoveScriptFont(string script)
```

| พารามิเตอร์ | ชนิด | คำอธิบาย |
| --- | --- | --- |
| script | String | รหัสสคริปต์ BCP-47 ที่การตั้งค่าแบบอักษรควรถูกลบออก |

### ตัวอย่าง

ตัวอย่างนี้สาธิตวิธีการลบการแมปแบบอักษรสำหรับสคริปต์ฮีบรู:

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.RemoveScriptFont("Hebr");
```

## FontFallBackRulesCollection.Add เมธอด

เพิ่มกฎ FallBack ที่ระบุไปยังส่วนท้ายของคอลเลกชัน

```csharp
public void Add(IFontFallBackRule sourceRule)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| sourceRule | IFontFallBackRule | กฎที่ระบุสำหรับการเพิ่ม |

### ตัวอย่าง

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    //การดึงคอลเลกชันกฎที่ว่างหรือกำหนดล่วงหน้าจาก FontsManager
    IFontFallBackRulesCollection rulesList = pres.FontsManager.FontFallBackRulesCollection;

    //การเพิ่มกฎใหม่เข้าไปในคอลเลกชัน
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));
}
```

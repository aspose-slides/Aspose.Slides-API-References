## IViewProperties.GridSpacing คุณสมบัติ

ส่งคืนหรือกำหนดการเว้นระยะของกริดที่ควรใช้สำหรับกริดที่อยู่ภายใต้เอกสารการนำเสนอ ในหน่วยจุด. อ่าน/เขียน Single.

```csharp
public float GridSpacing { get; set; }
```

### หมายเหตุ

ค่าการเว้นระยะของกริดต้องเป็นจำนวนบวก. ช่วงค่าที่พบบ่อยอยู่ระหว่าง 1 มม. (2.8349607 จุด) ถึง 2 นิ้ว (144 จุด).

### ตัวอย่าง

โค้ดตัวอย่างต่อไปนี้แสดงวิธีเปลี่ยนการเว้นระยะของกริดในงานนำเสนอ PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

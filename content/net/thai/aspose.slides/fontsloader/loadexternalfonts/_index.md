## FontsLoader.LoadExternalFonts method

เพิ่มโฟลเดอร์เพิ่มเติมเพื่อค้นหาแบบอักษร.

```csharp
public static void LoadExternalFonts(string[] directories)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| directories | String[] | ไดเรกทอรีสำหรับอ่านแบบอักษรเพิ่มเติม. |

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีโหลดแบบอักษรแบบกำหนดเองจากไฟล์ .TTF

```csharp
[C#]
// เส้นทางไปยังไดเรกทอรีเอกสาร.
string dataDir = "C:\\";
// โฟลเดอร์เพื่อค้นหาแบบอักษร
String[] folders = new String[] { dataDir };
// โหลดแบบอักษรจากไดเรกทอรีแบบอักษรที่กำหนดเอง
FontsLoader.LoadExternalFonts(folders);
// ทำงานบางอย่างและเรนเดอร์การนำเสนอ/สไลด์
using (Presentation presentation = new Presentation(dataDir + "DefaultFonts.pptx"))
{
    presentation.Save(dataDir + "NewFonts_out.pptx", SaveFormat.Pptx);
	// ลบแคชแบบอักษร
	FontsLoader.ClearCache();
}
```

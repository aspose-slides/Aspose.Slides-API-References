## ILayoutPlaceholderManager.AddOnlineImagePlaceholder เมธอด

เพิ่มรูปร่าง placeholder ใหม่ลงในสไลด์เค้าโครงเพื่อเก็บภาพออนไลน์.

```csharp
public IAutoShape AddOnlineImagePlaceholder(float x, float y, float width, float height)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| x | Single | พิกัด X ของรูปร่าง placeholder ใหม่ |
| y | Single | พิกัด Y ของรูปร่าง placeholder ใหม่ |
| width | Single | ความกว้างของรูปร่าง placeholder ใหม่ |
| height | Single | ความสูงของรูปร่าง placeholder ใหม่ |

### ค่าที่คืนกลับ

สร้าง [`IAutoShape`](../../iautoshape) พร้อม placeholder ภาพออนไลน์.

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีเพิ่มรูปร่าง placeholder ภาพออนไลน์ลงในสไลด์เค้าโครง.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddOnlineImagePlaceholder(20, 20, 200, 200);
}
```

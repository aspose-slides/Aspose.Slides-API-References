## ILayoutPlaceholderManager.AddSmartArtPlaceholder method

เพิ่มรูปร่าง placeholder ใหม่ไปยังสไลด์เลเอาท์เพื่อใช้เป็นที่เก็บแผนภาพ SmartArt.

```csharp
public IAutoShape AddSmartArtPlaceholder(float x, float y, float width, float height)
```

| พารามิเตอร์ | ชนิด | คำอธิบาย |
| --- | --- | --- |
| x | Single | พิกัด X ของรูปร่าง placeholder ใหม่ |
| y | Single | พิกัด Y ของรูปร่าง placeholder ใหม่ |
| width | Single | ความกว้างของรูปร่าง placeholder ใหม่ |
| height | Single | ความสูงของรูปร่าง placeholder ใหม่ |

### ค่าที่คืนกลับ

สร้าง [`IAutoShape`](../../iautoshape) ด้วย placeholder ของ SmartArt.

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีเพิ่มรูปร่าง placeholder ของ SmartArt ไปยังสไลด์เลเอาท์.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddSmartArtPlaceholder(20, 20, 200, 200);
}
```

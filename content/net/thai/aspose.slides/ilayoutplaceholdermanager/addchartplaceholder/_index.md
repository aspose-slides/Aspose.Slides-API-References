## ILayoutPlaceholderManager.AddChartPlaceholder เมธอด

เพิ่มรูปร่าง placeholder ใหม่ลงในสไลด์เค้าโครงเพื่อเก็บ Chart.

```csharp
public IAutoShape AddChartPlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| x | Single | พิกัด X ของรูปร่าง placeholder ใหม่. |
| y | Single | พิกัด Y ของรูปร่าง placeholder ใหม่. |
| width | Single | ความกว้างของรูปร่าง placeholder ใหม่. |
| height | Single | ความสูงของรูปร่าง placeholder ใหม่. |

### ค่าที่ส่งคืน

สร้าง [`IAutoShape`](../../iautoshape) พร้อมกับ placeholder ของ Chart.

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีการเพิ่มรูปร่าง placeholder ของ Chart ไปยังสไลด์เค้าโครง.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddChartPlaceholder(20, 20, 200, 200);
}
```

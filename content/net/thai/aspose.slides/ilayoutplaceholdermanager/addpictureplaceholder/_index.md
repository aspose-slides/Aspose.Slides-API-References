## ILayoutPlaceholderManager.AddPicturePlaceholder เมธอด

เพิ่มรูปทรง placeholder ใหม่ลงในสไลด์เลเอาท์เพื่อใช้เก็บรูปภาพ.

```csharp
public IAutoShape AddPicturePlaceholder(float x, float y, float width, float height)
```

| พารามิเตอร์ | ชนิด | คำอธิบาย |
| --- | --- | --- |
| x | Single | พิกัด X ของรูปทรง placeholder ใหม่ |
| y | Single | พิกัด Y ของรูปทรง placeholder ใหม่ |
| width | Single | ความกว้างของรูปทรง placeholder ใหม่ |
| height | Single | ความสูงของรูปทรง placeholder ใหม่ |

### ค่าที่คืนกลับ

สร้าง [`IAutoShape`](../../iautoshape) พร้อมกับ placeholder Picture.

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีการเพิ่มรูปทรง placeholder Picture ลงในสไลด์เลเอาท์

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddPicturePlaceholder(20, 20, 200, 200);
}
```

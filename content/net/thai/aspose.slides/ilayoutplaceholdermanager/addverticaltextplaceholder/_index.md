## ILayoutPlaceholderManager.AddVerticalTextPlaceholder เมธอด

เพิ่มรูปแบบตัวเติมใหม่ลงในสไลด์เค้าโครงเพื่อใช้เป็นที่เก็บเนื้อหาข้อความในแนวตั้ง

```csharp
public IAutoShape AddVerticalTextPlaceholder(float x, float y, float width, float height)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| x | Single | พิกัด X ของรูปแบบตัวเติมใหม่ |
| y | Single | พิกัด Y ของรูปแบบตัวเติมใหม่ |
| width | Single | ความกว้างของรูปแบบตัวเติมใหม่ |
| height | Single | ความสูงของรูปแบบตัวเติมใหม่ |

### ค่าที่ส่งกลับ

สร้าง [`IAutoShape`](../../iautoshape) ด้วยตัวเติมข้อความ (แนวตั้ง)

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีการเพิ่มรูปแบบตัวเติมข้อความ (แนวตั้ง) ลงในสไลด์เค้าโครง

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalTextPlaceholder(20, 20, 300, 500);
}
```

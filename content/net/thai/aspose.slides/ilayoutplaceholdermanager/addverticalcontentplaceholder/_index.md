## ILayoutPlaceholderManager.AddVerticalContentPlaceholder method

เพิ่มรูปทรงตัวแทนใหม่ลงในสไลด์เค้าโครงเพื่อเก็บเนื้อหา เช่น รูปภาพ ตาราง สื่อ หรือข้อความในแนวตั้ง

```csharp
public IAutoShape AddVerticalContentPlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| x | Single | พิกัด X ของรูปทรงตัวแทนใหม่ |
| y | Single | พิกัด Y ของรูปทรงตัวแทนใหม่ |
| width | Single | ความกว้างของรูปทรงตัวแทนใหม่ |
| height | Single | ความสูงของรูปทรงตัวแทนใหม่ |

### Return Value

สร้าง [`IAutoShape`](../../iautoshape) พร้อมกับตัวสำรอง Content (Vertical)

### Examples

ตัวอย่างต่อไปนี้แสดงวิธีเพิ่มรูปทรงตัวแทน Content (Vertical) ลงในสไลด์เค้าโครง

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalContentPlaceholder(20, 20, 300, 500);
}
```

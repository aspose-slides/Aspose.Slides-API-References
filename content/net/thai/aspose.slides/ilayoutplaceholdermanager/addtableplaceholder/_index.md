## ILayoutPlaceholderManager.AddTablePlaceholder เมธอด

เพิ่มรูปทรงพื้นที่ตัวยึดใหม่ลงในสไลด์เค้าโครงเพื่อเก็บตาราง.

```csharp
public IAutoShape AddTablePlaceholder(float x, float y, float width, float height)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| x | Single | พิกัด X ของรูปทรงพื้นที่ตัวยึดใหม่ |
| y | Single | พิกัด Y ของรูปทรงพื้นที่ตัวยึดใหม่ |
| width | Single | ความกว้างของรูปทรงพื้นที่ตัวยึดใหม่ |
| height | Single | ความสูงของรูปทรงพื้นที่ตัวยึดใหม่ |

### Return Value

สร้าง [`IAutoShape`](../../iautoshape) พร้อมพื้นที่ตัวยึด Table.

### Examples

ตัวอย่างต่อไปนี้แสดงวิธีเพิ่มรูปทรงพื้นที่ตัวยึด Table ลงในสไลด์เค้าโครง.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTablePlaceholder(20, 20, 500, 200);
}
```

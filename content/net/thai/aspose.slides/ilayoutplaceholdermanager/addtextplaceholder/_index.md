## ILayoutPlaceholderManager.AddTextPlaceholder เมธอด

เพิ่มรูปร่าง placeholder ใหม่ในสไลด์เค้าโครงเพื่อเก็บเนื้อหาข้อความ

```csharp
public IAutoShape AddTextPlaceholder(float x, float y, float width, float height)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| x | Single | พิกัด X ของรูปร่าง placeholder ใหม่ |
| y | Single | พิกัด Y ของรูปร่าง placeholder ใหม่ |
| width | Single | ความกว้างของรูปร่าง placeholder ใหม่ |
| height | Single | ความสูงของรูปร่าง placeholder ใหม่ |

### ค่าในการคืนค่า

สร้าง [`IAutoShape`](../../iautoshape) พร้อมกับ placeholder ประเภท Text

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีเพิ่มรูปร่าง placeholder ประเภท Text ไปยังสไลด์เค้าโครง

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTextPlaceholder(20, 20, 500, 300);
}
```

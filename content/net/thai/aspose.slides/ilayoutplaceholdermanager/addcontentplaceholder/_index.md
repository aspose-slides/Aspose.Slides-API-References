## ILayoutPlaceholderManager.AddContentPlaceholder เมธอด

เพิ่มรูปทรงตัวแทนใหม่ลงในสไลด์เลย์เอาต์เพื่อเก็บเนื้อหา เช่น รูปภาพ ตาราง สื่อ หรือข้อความ

```csharp
public IAutoShape AddContentPlaceholder(float x, float y, float width, float height)
```

| พารามิเตอร์ | ชนิด | คำอธิบาย |
| --- | --- | --- |
| x | Single | พิกัด X ของรูปทรงตัวแทนใหม่ |
| y | Single | พิกัด Y ของรูปทรงตัวแทนใหม่ |
| width | Single | ความกว้างของรูปทรงตัวแทนใหม่ |
| height | Single | ความสูงของรูปทรงตัวแทนใหม่ |

### ค่าที่ส่งคืน

สร้าง [`IAutoShape`](../../iautoshape) พร้อมกับตัวแทนเนื้อหา

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีการเพิ่มรูปทรงตัวแทนเนื้อหาไปยังสไลด์เลย์เอาต์

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddContentPlaceholder(20, 20, 500, 300);
}
```

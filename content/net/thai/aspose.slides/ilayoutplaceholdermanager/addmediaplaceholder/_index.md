## ILayoutPlaceholderManager.AddMediaPlaceholder เมธอด

เพิ่มรูปทรงตัวตำแหน่งใหม่ไปยังสไลด์เค้าโครงเพื่อเก็บวัตถุสื่อ

```csharp
public IAutoShape AddMediaPlaceholder(float x, float y, float width, float height)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| x | Single | พิกัด X ของรูปทรงตัวตำแหน่งใหม่ |
| y | Single | พิกัด Y ของรูปทรงตัวตำแหน่งใหม่ |
| width | Single | ความกว้างของรูปทรงตัวตำแหน่งใหม่ |
| height | Single | ความสูงของรูปทรงตัวตำแหน่งใหม่ |

### ค่าที่ส่งกลับ

สร้าง [`IAutoShape`](../../iautoshape) พร้อมตัวตำแหน่ง Media

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีการเพิ่มรูปทรงตัวตำแหน่ง Media ไปยังสไลด์เค้าโครง

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddMediaPlaceholder(20, 20, 200, 200);
}
```

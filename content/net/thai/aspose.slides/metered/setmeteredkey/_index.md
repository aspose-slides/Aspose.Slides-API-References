## Metered.SetMeteredKey เมธอด

กำหนดคีย์สาธารณะและคีย์ส่วนตัวแบบมีการวัดค่า หากคุณซื้อใบอนุญาตแบบมีการวัดค่า เมื่อเริ่มแอปพลิเคชัน ควรเรียกใช้ API นี้โดยปกติแล้วก็เพียงพอ อย่างไรก็ตาม หากการอัปโหลดข้อมูลการใช้งานล้มเหลวตลอดเวลาและเกิน 24 ชั่วโมง ใบอนุญาตจะถูกตั้งเป็นสถานะประเมินค่า เพื่อหลีกเลี่ยงกรณีดังกล่าว คุณควรตรวจสอบสถานะใบอนุญาตเป็นประจำ หากเป็นสถานะประเมินค่า ให้เรียกใช้ API นี้อีกครั้ง

```csharp
public void SetMeteredKey(string publicKey, string privateKey)
```

| พารามิเตอร์ | ชนิด | คำอธิบาย |
| --- | --- | --- |
| publicKey | String | คีย์สาธารณะ |
| privateKey | String | คีย์ส่วนตัว |

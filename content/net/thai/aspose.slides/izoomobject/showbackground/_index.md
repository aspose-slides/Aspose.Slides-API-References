## IZoomObject.ShowBackground คุณสมบัติ

รับหรือกำหนดค่าที่ระบุว่า Zoom จะใช้พื้นหลังของสไลด์เป้าหมายหรือไม่. อ่าน/เขียน Boolean. ค่าเริ่มต้น: true

```csharp
public bool ShowBackground { get; set; }
```

### ตัวอย่าง

ตัวอย่างแสดงการลบพื้นหลังของภาพของออบเจ็กต์ Zoom:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
    zoomFrame.ShowBackground = false;
}
```

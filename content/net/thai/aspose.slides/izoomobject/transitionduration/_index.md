## IZoomObject.TransitionDuration property

รับหรือกำหนดค่าระยะเวลาการเปลี่ยนระหว่าง Zoom และสไลด์. อ่าน/เขียน Single. ค่าตั้งต้น: 1.0f

```csharp
public float TransitionDuration { get; set; }
```

### หมายเหตุ

หากไม่ได้ระบุ (TransitionDur = 0) จะใช้การเปลี่ยนสไลด์ของสไลด์ปลายทางและเวลาที่สัมพันธ์กับการเปลี่ยนดังกล่าว.

### ตัวอย่าง

ตัวอย่างนี้แสดงการเปลี่ยนแปลงระยะเวลาการเปลี่ยนระหว่าง Zoom และสไลด์:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
    zoomFrame.TransitionDuration = 2.5f;
}
```

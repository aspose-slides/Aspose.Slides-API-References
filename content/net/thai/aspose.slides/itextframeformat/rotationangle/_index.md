## ITextFrameFormat.RotationAngle คุณสมบัติ

ระบุการหมุนแบบกำหนดเองที่ถูกนำไปใช้กับข้อความภายในกล่องขอบเขต หากไม่ได้ระบุ การหมุนของรูปร่างที่เกี่ยวข้องจะถูกใช้ หากระบุแล้ว การหมุนนี้จะถูกนำไปใช้แยกจากรูปร่าง คือรูปร่างสามารถมีการหมุนเพิ่มเติมได้ในขณะที่ข้อความเองก็มีการหมุนที่ถูกนำไปใช้ ค่าการหมุนของข้อความที่แสดงผลที่ได้สรุปจากคุณสมบัตินี้และประเภทแนวตั้งที่กำหนดล่วงหน้าในคุณสมบัติ TextVerticalType อ่าน/เขียน Single.

```csharp
public float RotationAngle { get; set; }
```

### ตัวอย่าง

พิจารณากรณีที่รูปร่างมีการหมุน 90 องศาตามเข็มนาฬิกาที่นำไปใช้กับมัน นอกจากนี้ เนื้อความเองยังมีการหมุน -90 องศาตรงข้ามเข็มนาฬิกาที่นำไปใช้กับมัน ดังนั้นรูปร่างที่ได้จะแสดงว่าถูกหมุน แต่ข้อความภายในจะดูเหมือนว่าไม่ได้ถูกหมุนเลย

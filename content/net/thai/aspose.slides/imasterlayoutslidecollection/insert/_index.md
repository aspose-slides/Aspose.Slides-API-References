## IMasterLayoutSlideCollection.Insert เมธอด

แทรกสไลด์เค้าโครงใหม่ลงในตำแหน่งที่ระบุของคอลเลกชัน

```csharp
public ILayoutSlide Insert(int index, SlideLayoutType layoutType, string layoutName)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| index | Int32 | ดัชนีของสไลด์ใหม่ |
| layoutType | SlideLayoutType | ประเภทเค้าโครงสำหรับเค้าโครงใหม่. ประเภทเค้าโครงที่รองรับ: Title, TitleOnly, Blank, TitleAndObject, VerticalText, VerticalTitleAndText, TwoObjects, SectionHeader, TwoTextAndTwoObjects, TitleObjectAndCaption, PictureAndCaption, Custom. ประเภทเค้าโครงอื่นๆ ที่ยังไม่รองรับในขณะนี้: Text, TwoColumnText, Table, TextAndChart, ChartAndText, Diagram, Chart, TextAndClipArt, ClipArtAndText, TextAndObject, ObjectAndText, Object, TextAndMedia, MediaAndText, ObjectOverText, TextOverObject, TextAndTwoObjects, TwoObjectsAndText, TwoObjectsOverText, FourObjects, ClipArtAndVerticalText, VerticalTitleAndTextOverChart, ObjectAndTwoObject, TwoObjectsAndObject |
| layoutName | String | ชื่อสำหรับเค้าโครงใหม่ หากชื่อที่ส่งเข้ามาใช้งานอยู่แล้วจะเกิด ArgumentException. หากส่งพารามิเตอร์เป็น null ชื่อจะถูกสร้างโดยอัตโนมัติเพื่อสอดคล้องกับประเภทเค้าโครงที่ส่ง (เช่น "Title Slide" หรือ "1_Title Slide", "2_..", เป็นต้น) |

### ค่าที่ส่งกลับ

สไลด์ที่แทรก

### ข้อยกเว้น

| ข้อยกเว้น | เงื่อนไข |
| --- | --- |
| NotImplementedException | เกิดขึ้นหากค่า *layoutType* ที่ไม่รองรับถูกส่งเข้ามา. ประเภทเค้าโครงที่ยังไม่รองรับในขณะนี้: Text, TwoColumnText, Table, TextAndChart, ChartAndText, Diagram, Chart, TextAndClipArt, ClipArtAndText, TextAndObject, ObjectAndText, Object, TextAndMedia, MediaAndText, ObjectOverText, TextOverObject, TextAndTwoObjects, TwoObjectsAndText, TwoObjectsOverText, FourObjects, ClipArtAndVerticalText, VerticalTitleAndTextOverChart, ObjectAndTwoObject, TwoObjectsAndObject |
| ArgumentException | เกิดขึ้นหากค่าชื่อเค้าโครง *layoutName* ถูกใช้แล้วในคอลเลกชันเค้าโครงนี้ |

### หมายเหตุ

เค้าโครงที่แทรกสำหรับค่า SlideLayoutType.Custom ของ *layoutType* ไม่มีตำแหน่งเก็บข้อมูลชั่วคราวและไม่มีรูปร่างใดๆ

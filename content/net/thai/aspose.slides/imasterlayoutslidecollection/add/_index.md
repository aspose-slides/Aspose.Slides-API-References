## IMasterLayoutSlideCollection.Add เมธอด

เพิ่มสไลด์เค้าโครงใหม่ที่ส่วนท้ายของคอลเลกชัน

```csharp
public ILayoutSlide Add(SlideLayoutType layoutType, string layoutName)
```

| พารามิเตอร์ | ชนิด | คำอธิบาย |
| --- | --- | --- |
| layoutType | SlideLayoutType | ประเภทเค้าโครงสำหรับเค้าโครงใหม่. ประเภทเค้าโครงที่รองรับ: Title, TitleOnly, Blank, TitleAndObject, VerticalText, VerticalTitleAndText, TwoObjects, SectionHeader, TwoTextAndTwoObjects, TitleObjectAndCaption, PictureAndCaption, Custom. ประเภทเค้าโครงอื่น ๆ ไม่ได้รับการสนับสนุนในขณะนี้: Text, TwoColumnText, Table, TextAndChart, ChartAndText, Diagram, Chart, TextAndClipArt, ClipArtAndText, TextAndObject, ObjectAndText, Object, TextAndMedia, MediaAndText, ObjectOverText, TextOverObject, TextAndTwoObjects, TwoObjectsAndText, TwoObjectsOverText, FourObjects, ClipArtAndVerticalText, VerticalTitleAndTextOverChart, ObjectAndTwoObject, TwoObjectsAndObject. |
| layoutName | String | ชื่อสำหรับเค้าโครงใหม่. หากชื่อที่ส่งเข้ามาใช้งานอยู่แล้วจะเกิด ArgumentException. หากพารามิเตอร์เป็น null จะสร้างชื่อโดยอัตโนมัติตามประเภทเค้าโครงที่ส่งเข้ามา (เช่น "Title Slide" หรือ "1_Title Slide", "2_..", เป็นต้น). |

| ข้อยกเว้น | เงื่อนไข |

| NotImplementedException | โยน หากค่า *layoutType* ที่ไม่สนับสนุนถูกส่งเข้ามา. ประเภทเค้าโครงที่ไม่รองรับในขณะนี้: Text, TwoColumnText, Table, TextAndChart, ChartAndText, Diagram, Chart, TextAndClipArt, ClipArtAndText, TextAndObject, ObjectAndText, Object, TextAndMedia, MediaAndText, ObjectOverText, TextOverObject, TextAndTwoObjects, TwoObjectsAndText, TwoObjectsOverText, FourObjects, ClipArtAndVerticalText, VerticalTitleAndTextOverChart, ObjectAndTwoObject, TwoObjectsAndObject. |
| ArgumentException | โยน หากค่าชื่อเค้าโครง *layoutName* ถูกใช้แล้วในคอลเลกชันนี้. |

### หมายเหตุ

1) เค้าโครงที่เพิ่มสำหรับค่า SlideLayoutType.Custom ของ *layoutType* ไม่มีตัวแทนและไม่มีรูปทรง. 2) ตรงข้ามของเมธอดนี้คือเมธอด [`Add`](../../igloballayoutslidecollection/add) ที่เข้าถึงได้ด้วยคุณสมบัติ [`LayoutSlides`](../../ipresentation/layoutslides)

## License ตัวสร้าง

เริ่มต้นอินสแตนซ์ใหม่ของคลาสนี้.

```csharp
public License()
```

### ตัวอย่าง

ในตัวอย่างนี้จะพยายามค้นหาไฟล์ใบอนุญาตชื่อ MyLicense.lic ในโฟลเดอร์ที่บรรจุคอมโพเนนท์, ในโฟลเดอร์ที่บรรจุแอสเซมบลีที่เรียก, ในโฟลเดอร์ของแอสเซมบลีเริ่มต้นและจากนั้นในทรัพยากรฝังของแอสเซมบลีที่เรียก.

```csharp
[C#]

License license = new License();
license.SetLicense("MyLicense.lic");


[Visual Basic]

Dim license As license = New license
License.SetLicense("MyLicense.lic")
```

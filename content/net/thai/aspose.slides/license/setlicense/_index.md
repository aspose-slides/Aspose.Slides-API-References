## SetLicense(string) {#setlicense_1}

ให้ไลเซนส์กับคอมโพเนนต์.

```csharp
public void SetLicense(string licenseName)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| licenseName | String | อาจเป็นชื่อไฟล์เต็มหรือสั้น หรือชื่อของทรัพยากรที่ฝังอยู่ ใช้สตริงว่างเพื่อสลับเป็นโหมดประเมินผล |

### หมายเหตุ

พยายามค้นหาไลเซนส์ในตำแหน่งต่อไปนี้:

1. เส้นทางที่ระบุโดยตรง

2. โฟลเดอร์ของแอสเซมบลีคอมโพเนนต์

3. โฟลเดอร์ของแอสเซมบลีที่เรียกใช้ของไคลเอนต์

4. โฟลเดอร์ของแอสเซมบลีเริ่มต้น

5. ทรัพยากรที่ฝังอยู่ในแอสเซมบลีที่เรียกใช้ของไคลเอนต์

**หมายเหตุ:**บน .NET Compact Framework จะพยายามค้นหาไลเซนส์เฉพาะในตำแหน่งต่อไปนี้:

1. เส้นทางที่ระบุโดยตรง

2. ทรัพยากรที่ฝังอยู่ในแอสเซมบลีที่เรียกใช้ของไคลเอนต์

### ตัวอย่าง

ในตัวอย่างนี้ จะพยายามค้นหาไฟล์ไลเซนส์ชื่อ MyLicense.lic ในโฟลเดอร์ที่บรรจุคอมโพเนนต์ โฟลเดอร์ที่บรรจุแอสเซมบลีที่เรียกใช้ โฟลเดอร์ของแอสเซมบลีเริ่มต้น และสุดท้ายในทรัพยากรที่ฝังอยู่ของแอสเซมบลีที่เรียกใช้

```csharp
[C#]

License license = new License();
license.SetLicense("MyLicense.lic");


[Visual Basic]

Dim license As License = New License
license.SetLicense("MyLicense.lic")
```

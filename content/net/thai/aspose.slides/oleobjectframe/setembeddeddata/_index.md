## OleObjectFrame.SetEmbeddedData เมธอด

ตั้งค่าข้อมูลเกี่ยวกับข้อมูลฝัง OLE. วิธีนี้เปลี่ยนคุณสมบัติของอ็อบเจกต์เพื่อสะท้อนข้อมูลใหม่และตั้งค่าแฟล็ก IsObjectLink เป็น false ซึ่งบ่งบอกว่าอ็อบเจกต์ OLE ถูกฝังไว้.

```csharp
public void SetEmbeddedData(IOleEmbeddedDataInfo embeddedData)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| embeddedData | IOleEmbeddedDataInfo | ข้อมูลฝัง [`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo) |

### ข้อยกเว้น

| ข้อยกเว้น | เงื่อนไข |
| --- | --- |
| ArgumentNullException | เมื่อพารามิเตอร์ embeddedData มีค่า null. |

### ตัวอย่าง

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    OleObjectFrame oof = pres.Slides[0].Shapes[0] as OleObjectFrame;
    if (oof != null)
    {
        IOleEmbeddedDataInfo newData = new OleEmbeddedDataInfo(File.ReadAllBytes("Picture.png"), "png");
        oof.SetEmbeddedData(newData);
    }
}
```

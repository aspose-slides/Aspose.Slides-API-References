## OleObjectFrame.LinkPathRelative คุณสมบัติ

คืนเส้นทางสัมพัทธ์ไปยังไฟล์ที่เชื่อมโยงหากมี, หากไม่มีจะคืนสตริงว่าง. อ่านอย่างเดียว String.

```csharp
public string LinkPathRelative { get; }
```

### หมายเหตุ

ในงานนำเสนอ Ppt, ลิงก์อ็อบเจ็กต์ Ole บางอย่างอาจมีการแสดงผลแบบสัมพัทธ์.

### ตัวอย่าง

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.ppt"))
{
    IOleObjectFrame oleFrame = presentation.Slides[0].Shapes[0] as IOleObjectFrame;

    if (oleFrame != null)
    {
        Console.WriteLine("The relative path: " + oleFrame.LinkPathRelative);
    } 
}
```

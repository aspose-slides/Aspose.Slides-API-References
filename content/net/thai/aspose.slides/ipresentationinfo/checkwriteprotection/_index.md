## IPresentationInfo.CheckWriteProtection method

ตรวจสอบว่ารหัสผ่านสำหรับการแก้ไขนั้นถูกต้องสำหรับงานนำเสนอที่มีการป้องกันการเขียนหรือไม่.

```csharp
public bool CheckWriteProtection(string password)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| password | String | รหัสผ่านที่ต้องการตรวจสอบ. |

### ค่าที่ส่งคืน

True หากงานนำเสนอถูกป้องกันการเขียนและรหัสผ่านถูกต้อง. False ในกรณีอื่น.

### ข้อยกเว้น

| ข้อยกเว้น | เงื่อนไข |
| --- | --- |
| InvalidOperationException |  |

### หมายเหตุ

1. คุณควรตรวจสอบคุณสมบัติ [`IsWriteProtected`](../iswriteprotected) ก่อนเรียกใช้เมธอดนี้. 2. เมื่อ password เป็น null หรือเป็นสตริงว่าง เมธอดนี้จะคืนค่า false.

### ตัวอย่าง

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
if (info.IsWriteProtected == NullableBool.True)
{
    bool isWriteProtectedByPassword = info.CheckWriteProtection("my_password");
}
```

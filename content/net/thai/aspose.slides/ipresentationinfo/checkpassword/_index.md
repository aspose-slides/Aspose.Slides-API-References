## IPresentationInfo.CheckPassword เมธอด

ตรวจสอบว่ารหัสผ่านถูกต้องสำหรับการนำเสนอที่ได้รับการปกป้องด้วยรหัสผ่านเปิดหรือไม่.

```csharp
public bool CheckPassword(string password)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| password | String | รหัสผ่านที่ต้องตรวจสอบ. |

### ค่าที่ส่งคืน

True หากการนำเสนอได้รับการปกป้องด้วยรหัสผ่านเปิดและรหัสผ่านถูกต้อง และ false หากไม่เป็นเช่นนั้น.

### หมายเหตุ

เมื่อรหัสผ่านเป็น null หรือว่าง, เมธอดนี้จะคืนค่า false.

### ตัวอย่าง

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo("pres.pptx");
bool isPasswordCorrect = info.CheckPassword("my_password");
```

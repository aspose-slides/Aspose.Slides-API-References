## IProtectionManager.CheckWriteProtection เมธอด

กำหนดว่าการนำเสนอได้รับการป้องกันด้วยรหัสผ่านเพื่อการแก้ไขหรือไม่

```csharp
public bool CheckWriteProtection(string password)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| password | String | รหัสผ่านสำหรับการตรวจสอบ |

### ค่าที่ส่งคืน

True หากรหัสผ่านถูกต้อง; มิฉะนั้น false.

### หมายเหตุ

1. คุณควรตรวจสอบคุณสมบัติ [`IsWriteProtected`](../iswriteprotected) ก่อนเรียกใช้เมธอดนี้.  
2. เมื่อรหัสผ่านเป็นค่า null หรือว่างเปล่า เมธอดนี้จะคืนค่า false.

### ตัวอย่าง

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

## IProtectionManager.IsOnlyDocumentPropertiesLoaded คุณสมบัติ

คุณสมบัตินี้มีประโยชน์เมื่อไฟล์งานนำเสนอถูกป้องกันด้วยรหัสผ่านและคุณสมบัติของเอกสารของไฟล์นั้นเป็นสาธารณะ. ค่า true หมายความว่าเฉพาะคุณสมบัติของเอกสารเท่านั้นที่ถูกโหลดจากไฟล์งานนำเสนอที่เข้ารหัสโดยไม่ได้ใช้รหัสผ่าน. ค่า false หมายความว่าไฟล์งานนำเสนอที่เข้ารหัสทั้งหมดจะถูกโหลดโดยใช้รหัสผ่านที่ถูกต้อง ไม่ใช่แค่คุณสมบัติของเอกสารเท่านั้นที่ถูกโหลด. หากไฟล์งานนำเสนอไม่ได้ถูกเข้ารหัส ค่าของคุณสมบัติจึงจะเป็น false เสมอ. หากคุณสมบัติของเอกสารของไฟล์ที่เข้ารหัสไม่เป็นสาธารณะ ค่าของคุณสมบัติก็จะเป็น false เสมอ. หาก PresentationEx.EncryptDocumentProperties มีค่าเป็น true แล้วค่าของคุณสมบัติ IsOnlyDocumentPropertiesLoaded จะเป็น false เสมอ. อ่านอย่างเดียว Boolean.

```csharp
public bool IsOnlyDocumentPropertiesLoaded { get; }
```

## FontFallBackRule.IndexOf เมธอด

คืนดัชนีของกฎที่ระบุในคอลเลกชัน.

```csharp
public int IndexOf(string fontName)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| fontName | String | ชื่อแบบอักษรที่ต้องการค้นหา |

### ค่าที่คืน

ดัชนีของแบบอักษร หรือ -1 หากไม่พบแบบอักษรในรายการ.

### ตัวอย่าง

```csharp
[C#]
// สร้างกฎที่มีรายการแบบอักษร.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// รับดัชนีของ Tahoma.
int tahomaIndex = newRule.IndexOf("Tahoma");
```

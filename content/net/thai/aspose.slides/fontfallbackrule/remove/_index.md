## FontFallBackRule.Remove เมธอด

ลบการปรากฏครั้งแรกของฟอนต์ FallBack ที่ระบุจากรายการ

```csharp
public void Remove(string fontName)
```

| Parameter | Type | Description |
| --- | --- | --- |
| fontName | String | ชื่อฟอนต์ที่จะลบจากรายการ |

### ตัวอย่าง

```csharp
[C#]
// สร้างกฎที่มีรายการฟอนต์.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// ลบ Tahoma ออกจากรายการ.
newRule.Remove ("Tahoma");
```

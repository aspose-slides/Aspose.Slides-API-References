## AddFallBackFonts(string) {#addfallbackfonts}

เพิ่มฟอนต์ใหม่(s) ไปยังรายการฟอนต์ FallBack

```csharp
public void AddFallBackFonts(string fontName)
```

| พารามิเตอร์ | ชนิด | รายละเอียด |
| --- | --- | --- |
| fontName | String | ชื่อของฟอนต์หรือหลายชื่อ (คั่นด้วยเครื่องหมายจุลภาค) สำหรับ FallBack |

### ตัวอย่าง

```csharp
[C#]
// สร้างอินสแตนซ์ใหม่ของ FontFallBackRule
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho");

//Add a second font to the rule 
//เพิ่มฟอนต์ที่สองให้กับกฎ 
newRule.AddFallBackFonts("MS Gothic");
//Add a third and fourth fonts to the rule 
//เพิ่มฟอนต์ที่สามและสี่ให้กับกฎ 
newRule.AddFallBackFonts("Tahoma, Times New Roman");
```

## AddFallBackFonts(string[]) {#addfallbackfonts_1}

เพิ่มฟอนต์ใหม่หลายตัวไปยังรายการฟอนต์ FallBack

```csharp
public void AddFallBackFonts(string[] fontNames)
```

| พารามิเตอร์ | ชนิด | รายละเอียด |
| --- | --- | --- |
| fontNames | String[] | ชื่อของฟอนต์หรือหลายชื่อ (คั่นด้วยเครื่องหมายจุลภาค) สำหรับ FallBack |


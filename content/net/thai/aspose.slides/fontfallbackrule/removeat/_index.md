## FontFallBackRule.RemoveAt เมธอด

ลบแบบอักษร FallBack ที่ตำแหน่งที่กำหนดของรายการ.

```csharp
public void RemoveAt(int index)
```

| พารามิเตอร์ | ประเภท | รายละเอียด |
| --- | --- | --- |
| index | Int32 | ตำแหน่งดัชนีเริ่มจากศูนย์ของแบบอักษรที่ต้องการลบ |

### ตัวอย่าง

```csharp
[C#]
// สร้างกฎที่มีรายการของฟอนต์.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// กำลังลบ Tahoma ออกจากรายการ.
newRule.Remove (2);
```

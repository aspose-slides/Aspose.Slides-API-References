## MasterLayoutSlideCollection.Insert เมธอด

แทรกสไลด์เค้าโครงใหม่ในตำแหน่งที่ระบุของคอลเลกชัน.

```csharp
public ILayoutSlide Insert(int index, SlideLayoutType layoutType, string layoutName)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| index | Int32 | ดัชนีของสไลด์ใหม่. |
| layoutType | SlideLayoutType | ประเภทเค้าโครงสำหรับเค้าโครงใหม่. ประเภทเค้าโครงที่รองรับ: Title, TitleOnly, Blank, TitleAndObject, VerticalText, VerticalTitleAndText, TwoObjects, SectionHeader, TwoTextAndTwoObjects, TitleObjectAndCaption, PictureAndCaption, Custom. ประเภทเค้าโครงอื่นที่ไม่ได้รองรับในขณะนี้: Text, TwoColumnText, Table, TextAndChart, ChartAndText, Diagram, Chart, TextAndClipArt, ClipArtAndText, TextAndObject, ObjectAndText, Object, TextAndMedia, MediaAndText, ObjectOverText, TextOverObject, TextAndTwoObjects, TwoObjectsAndText, TwoObjectsOverText, FourObjects, ClipArtAndVerticalText, VerticalTitleAndTextOverChart, ObjectAndTwoObject, TwoObjectsAndObject. |
| layoutName | String | ชื่อสำหรับเค้าโครงใหม่. หากชื่อที่ส่งเข้ามาใช้งานอยู่แล้วจะเกิด ArgumentException. หากส่งค่าพารามิเตอร์เป็น null จะสร้างชื่อโดยอัตโนมัติตามประเภทเค้าโครงที่ส่งเข้ามา (เช่น "Title Slide" หรือ "1_Title Slide", "2_..", เป็นต้น). |

### ค่าที่ส่งคืน

สไลด์ที่แทรกแล้ว.

### ข้อยกเว้น

| ข้อยกเว้น | เงื่อนไข |
| --- | --- |
| NotImplementedException | เกิดข้อผิดพลาดเมื่อส่งค่า *layoutType* ที่ไม่รองรับ. ประเภทเค้าโครงที่ไม่ได้รองรับในขณะนี้: Text, TwoColumnText, Table, TextAndChart, ChartAndText, Diagram, Chart, TextAndClipArt, ClipArtAndText, TextAndObject, ObjectAndText, Object, TextAndMedia, MediaAndText, ObjectOverText, TextOverObject, TextAndTwoObjects, TwoObjectsAndText, TwoObjectsOverText, FourObjects, ClipArtAndVerticalText, VerticalTitleAndTextOverChart, ObjectAndTwoObject, TwoObjectsAndObject. |
| ArgumentException | เกิดข้อผิดพลาดเมื่อค่า layout name *layoutName* ใช้แล้วในคอลเลกชันของเค้าโครงนี้. |

### หมายเหตุ

เค้าโครงที่แทรกสำหรับค่า SlideLayoutType.Custom ของ *layoutType* ไม่มีตัวยึดตำแหน่งและไม่มีรูปร่าง.

## SlideShowSettings.SlideShowType คุณสมบัติ

รับหรือกำหนดประเภทการแสดงสไลด์. แสดงโดยบรรพบุรุษ `SlideShowType` ต่อไปนี้: [`BrowsedAtKiosk`](../../browsedatkiosk), [`PresentedBySpeaker`](../../presentedbyspeaker) and [`BrowsedByIndividual`](../../browsedbyindividual)

```csharp
public SlideShowType SlideShowType { get; set; }
```

### ตัวอย่าง

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // เพื่อกำหนดประเภท "Browsed at a kiosk (full screen)" 
    // เพื่อกำหนดประเภท "Browsed by individual (window)"
    // เพื่อกำหนดประเภท "Presented by a speaker (full screen)"
    pres.SlideShowSettings.SlideShowType = new BrowsedAtKiosk();
    
    pres.SlideShowSettings.SlideShowType = new BrowsedByIndividual();
    
    pres.SlideShowSettings.SlideShowType = new PresentedBySpeaker();
}
```

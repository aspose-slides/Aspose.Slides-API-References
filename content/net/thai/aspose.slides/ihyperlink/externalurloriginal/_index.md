## IHyperlink.ExternalUrlOriginal คุณสมบัติ

แสดงถึงไฮเปอร์ลิงก์ที่ตั้งค่าสำหรับส่วนนี้โดยไม่คำนึงถึงเนื้อหาจริงของส่วนนั้น PowerPoint ทำงานเฉพาะกับลิงก์และข้อความที่สอดคล้องกันในส่วนนั้น มันอนุญาตให้สร้างข้อความสำหรับไฮเปอร์ลิงก์ในรูปแบบของ URL ที่ถูกต้อง ซึ่งแตกต่างจากที่อยู่จริงของลิงก์ ในกรณีนี้เมื่อคุณดูลิงก์ในหน้าต่างการแก้ไข จะถูกเปลี่ยนให้ตรงกับข้อความของส่วนนี้ คุณสมบัตินี้แสดงถึงค่าต้นฉบับของไฮเปอร์ลิงก์。

public string ExternalUrlOriginal { get; }

## IPresentation.HighlightRegex เมธอด

ทำไฮไลต์ทุกการจับคู่ของนิพจน์ปกติด้วยสีที่ระบุ

```csharp
public void HighlightRegex(Regex regex, Color highlightColor, IFindResultCallback callback)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| regex | Regex | นิพจน์ปกติ Regex ที่ใช้เพื่อดึงสตริงที่ต้องการทำไฮไลต์ |
| highlightColor | Color | สีที่ใช้ทำไฮไลต์ข้อความ |
| callback | IFindResultCallback | อ็อบเจ็กต์ callback สำหรับรับผลการค้นหา [`IFindResultCallback`](../../ifindresultcallback) |

### ตัวอย่าง

ตัวอย่างโค้ดต่อไปนี้แสดงวิธีทำไฮไลต์ข้อความใน PowerPoint Presentation ด้วยการใช้นิพจน์ปกติ

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// ทำไฮไลต์คำทั้งหมดที่มีความยาว 10 ตัวอักษรหรือมากกว่า
	presentation.HighlightRegex(regex, Color.Blue, null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

## IFontsManager.GetFontBytes เมธอด

ดึงอาร์เรย์ของไบต์ที่แสดงข้อมูลแบบอักษรสำหรับสไตล์แบบอักษรและข้อมูลแบบอักษรที่ระบุ

```csharp
public byte[] GetFontBytes(IFontData fontData, FontStyleType fontStyle)
```

| Parameter | Type | Description |
| --- | --- | --- |
| fontData | IFontData | อ็อบเจ็กต์ข้อมูลแบบอักษรที่มีข้อมูลเกี่ยวกับแบบอักษร [`IFontData`](../../ifontdata). |
| fontStyle | FontStyleType | สไตล์ของแบบอักษรที่ต้องการดึงข้อมูล [`FontStyleType`](../../fontstyletype). |

### ค่าที่ส่งคืน

อาร์เรย์ของไบต์ที่มีข้อมูลแบบอักษรสำหรับสไตล์แบบอักษรที่ระบุ หากไม่พบข้อมูลแบบอักษรหรือสไตล์ จะส่งคืนค่า null.

### ตัวอย่าง

```csharp
[C#]
using (Presentation pres = new Presentation ("Presentation.pptx"))
{
    // ดึงแบบอักษรทั้งหมดที่ใช้ในงานนำเสนอ
    IFontData[] fonts = pres.FontsManager.GetFonts();

    // ดึงอาร์เรย์ของไบต์ที่แสดงสไตล์ Regular ของแบบอักษรแรกในงานนำเสนอ
    byte[] bytes = pres.FontsManager.GetFontBytes(fonts[0], FontStyleType.Regular);
}
```

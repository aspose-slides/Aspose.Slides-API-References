## IOleObjectFrame.SetEmbeddedData เมธอด

ตั้งค่าข้อมูลเกี่ยวกับข้อมูล OLE ที่ฝังอยู่

```csharp
public void SetEmbeddedData(IOleEmbeddedDataInfo embeddedData)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| embeddedData | IOleEmbeddedDataInfo | ข้อมูลที่ฝัง [`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo) |

### ข้อยกเว้น

| ข้อยกเว้น | เงื่อนไข |
| --- | --- |
| ArgumentNullException | เมื่อพารามิเตอร์ embeddedData เป็น null. |

### หมายเหตุ

เมธอดนี้จะเปลี่ยนแปลงคุณสมบัติของอ็อบเจ็กต์เพื่อสะท้อนข้อมูลใหม่และตั้งค่าแฟล็ก IsObjectLink เป็น false ซึ่งระบุว่าอ็อบเจ็กต์ OLE ถูกฝังอยู่

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีการเปลี่ยนข้อมูล OLE ที่ฝังและประเภทของมันสำหรับอ็อบเจ็กต์ [`IOleObjectFrame`](../../ioleobjectframe) ที่มีอยู่

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    OleObjectFrame oof = pres.Slides[0].Shapes[0] as OleObjectFrame;
    if (oof != null)
    {
        IOleEmbeddedDataInfo newData = new OleEmbeddedDataInfo(File.ReadAllBytes("Picture.png"), "png");
        oof.SetEmbeddedData(newData);
    }
}
```

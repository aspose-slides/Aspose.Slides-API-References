## IOleObjectFrame.LinkPathRelative คุณสมบัติ

ส่งกลับเส้นทางแบบสัมพัทธ์ไปยังไฟล์ที่เชื่อมโยงหากมีอยู่, หากไม่มีจะส่งกลับสตริงว่าง. Readonly String.

```csharp
public string LinkPathRelative { get; }
```

### หมายเหตุ

ในงานนำเสนอ Ppt, ลิงก์ออบเจ็กต์ Ole บางส่วนอาจมีการแสดงเป็นแบบสัมพัทธ์.

### ตัวอย่าง

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.ppt"))
{
    IOleObjectFrame oleFrame = presentation.Slides[0].Shapes[0] as IOleObjectFrame;

    if (oleFrame != null)
    {
        Console.WriteLine("The relative path: " + oleFrame.LinkPathRelative);
    } 
}
```

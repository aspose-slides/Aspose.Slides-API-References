ส่งคืนคอลเลกชันของคู่มือการวาดสำหรับสไลด์เลย์เอาต์. อ่านอย่างเดียว [`IDrawingGuidesCollection`](../../idrawingguidescollection)

### ตัวอย่าง

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    var slideSize = pres.SlideSize.Size;
    
    IDrawingGuidesCollection guides = pres.LayoutSlides[0].DrawingGuides;
    // เพิ่มคู่มือการวาดแนวตั้งใหม่ทางซ้ายของจุดกึ่งกลางสไลด์

    pres.Save("LayoutDrawingGuides_out.pptx", SaveFormat.Pptx);
}
```

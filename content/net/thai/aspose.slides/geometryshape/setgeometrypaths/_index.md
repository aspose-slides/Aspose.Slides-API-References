## GeometryShape.SetGeometryPaths เมธอด

อัปเดตรูปทรงเรขาคณิตของรูปจากอาเรย์ของ [`IGeometryPath`](../../igeometrypath). พิกัดต้องสัมพันธ์กับมุมซ้ายบนของรูป. เปลี่ยนประเภทของรูป ([`ShapeType`](../shapetype)) เป็น Custom.

```csharp
public void SetGeometryPaths(IGeometryPath[] geometryPaths)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| geometryPaths | IGeometryPath[] | อาร์เรย์ของเส้นทางเรขาคณิต |

### ข้อยกเว้น

| ข้อยกเว้น | เงื่อนไข |
| --- | --- |
| ArgumentException | ไม่พบเส้นทาง |
| ArgumentException | เส้นทางว่าง |

### ตัวอย่าง

ตัวอย่าง:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    GeometryShape shape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 200, 100) as GeometryShape;

    GeometryPath geometryPath = shape.GetGeometryPaths()[0];

    geometryPath.LineTo(100, 50, 1);
    geometryPath.LineTo(100, 50, 4);

    shape.SetGeometryPath(geometryPath);

    pres.Save("output.pptx", SaveFormat.Pptx);
}
```

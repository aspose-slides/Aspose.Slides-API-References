## AddChart(ChartType, float, float, float, float) {#addchart}

สร้างแผนภูมิใหม่, เริ่มต้นด้วยข้อมูลชุดตัวอย่างและการตั้งค่า, และเพิ่มไปยังส่วนท้ายของคอลเลกชันรูปทรง

```csharp
public IChart AddChart(ChartType type, float x, float y, float width, float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| type | ChartType | ประเภทของแผนภูมิที่จะเพิ่ม |
| x | Single | พิกัด x ของแผนภูมิใหม่, หน่วยเป็นพอยต์ |
| y | Single | พิกัด y ของแผนภูมิใหม่, หน่วยเป็นพอยต์ |
| width | Single | ความกว้างของแผนภูมิ, หน่วยเป็นพอยต์ |
| height | Single | ความสูงของแผนภูมิ, หน่วยเป็นพอยต์ |

### Return Value

อ็อบเจ็กต์ที่สร้างขึ้นใหม่ [`IChart`](../../../aspose.slides.charts/ichart).

## AddChart(ChartType, float, float, float, float, bool) {#addchart_1}

สร้างแผนภูมิใหม่, เริ่มต้นด้วยข้อมูลชุดตัวอย่างและการตั้งค่า, และเพิ่มไปยังส่วนท้ายของคอลเลกชันรูปทรง

```csharp
public IChart AddChart(ChartType type, float x, float y, float width, float height, 
    bool initWithSample)
```

| Parameter | Type | Description |
| --- | --- | --- |
| type | ChartType | ประเภทของแผนภูมิที่จะเพิ่ม |
| x | Single | พิกัด x ของแผนภูมิใหม่, หน่วยเป็นพอยต์ |

| height | Single | ความสูงของแผนภูมิ, หน่วยเป็นพอยต์ |
| initWithSample | Boolean | true เพื่อเริ่มต้นแผนภูมิใหม่ด้วยข้อมูลชุดตัวอย่างและการตั้งค่า; false เพื่อสร้างแผนภูมิโดยไม่มีชุดข้อมูลและตั้งค่าเพียงพื้นฐาน, ทำให้การสร้างเร็วขึ้น |

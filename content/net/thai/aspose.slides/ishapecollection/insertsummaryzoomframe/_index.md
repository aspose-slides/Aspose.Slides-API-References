## IShapeCollection.InsertSummaryZoomFrame เมธอด

สร้างกรอบ Summary Zoom ใหม่และแทรกเข้าไปในคอลเลกชันรูปร่างที่ตำแหน่ง index ที่ระบุ

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| พารามิเตอร์ | ชนิด | คำอธิบาย |
| --- | --- | --- |
| index | Int32 | ตำแหน่ง index ที่เริ่มจากศูนย์ที่จะใช้แทรกกรอบ Summary Zoom |
| x | Single | พิกัด x ของกรอบ Summary Zoom ใหม่, หน่วยเป็น points |
| y | Single | พิกัด y ของกรอบ Summary Zoom ใหม่, หน่วยเป็น points |
| width | Single | ความกว้างของกรอบ Summary Zoom ใหม่, หน่วยเป็น points |
| height | Single | ความสูงของกรอบ Summary Zoom ใหม่, หน่วยเป็น points |

### ค่าที่ส่งกลับ

อ็อบเจกต์ที่สร้างใหม่ [`ISummaryZoomFrame`](../../isummaryzoomframe).

### หมายเหตุ

เมธอดนี้สร้างกรอบ Summary Zoom ที่รวบรวมลิงก์สรุปสำหรับทุก section ในการนำเสนอ

### ตัวอย่าง

ตัวอย่างนี้แสดงการสร้างและแทรกอ็อบเจกต์ Summary Zoom ที่ตำแหน่ง index ที่ระบุของคอลเลกชัน (สมมติว่ามีอย่างน้อยสอง section ในการนำเสนอ "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSummaryZoomFrame(2, 150, 20, 50, 50);
}
```

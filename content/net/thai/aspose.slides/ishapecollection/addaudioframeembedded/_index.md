## AddAudioFrameEmbedded(float, float, float, float, Stream) {#addaudioframeembedded_1}

```csharp
    Stream audio_stream)
```

| --- | --- | --- |
| x | Single | ค่าพิกัด x ของเฟรมเสียงใหม่ หน่วยเป็นจุด |
| y | Single | ค่าพิกัด y ของเฟรมเสียงใหม่ หน่วยเป็นจุด |
| width | Single | ความกว้างของเฟรมเสียงใหม่ หน่วยเป็นจุด |
| height | Single | ความสูงของเฟรมเสียงใหม่ หน่วยเป็นจุด |
| audio_stream | Stream | สตรีมอินพุตที่มีข้อมูลเสียง WAV เพื่อฝัง |

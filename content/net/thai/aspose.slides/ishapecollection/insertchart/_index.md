## InsertChart(ChartType, float, float, float, float, int) {#insertchart}


| x | Single | พิกัด x ของแผนภูมิใหม่ หน่วยเป็นพอยต์ |
| y | Single | พิกัด y ของแผนภูมิใหม่ หน่วยเป็นพอยต์ |
| width | Single | ความกว้างของแผนภูมิใหม่ หน่วยเป็นพอยต์ |
| height | Single | ความสูงของแผนภูมิใหม่ หน่วยเป็นพอยต์ |
| index | Int32 | ดัชนีเริ่มจากศูนย์ที่จะแทรกแผนภูมิใหม่ลงในคอลเลกชันรูปร่าง |

## InsertChart(ChartType, float, float, float, float, int, bool) {#insertchart_1}

สร้างแผนภูมิใหม่ เริ่มต้นด้วยข้อมูลซีรีส์ตัวอย่างและการตั้งค่า แล้วแทรกลงในคอลเลกชันรูปร่างที่ตำแหน่งที่ระบุ

```csharp
public IChart InsertChart(ChartType type, float x, float y, float width, float height, int index, 
    bool initWithSample)
```

| Parameter | Type | Description |
| --- | --- | --- |
| type | ChartType | ประเภทของแผนภูมิที่จะสร้าง |
| x | Single | พิกัด x ของแผนภูมิใหม่ หน่วยเป็นพอยต์ |
| y | Single | พิกัด y ของแผนภูมิใหม่ หน่วยเป็นพอยต์ |
| width | Single | ความกว้างของแผนภูมิใหม่ หน่วยเป็นพอยต์ |
| height | Single | ความสูงของแผนภูมิใหม่ หน่วยเป็นพอยต์ |
| index | Int32 | ดัชนีเริ่มจากศูนย์ที่จะแทรกแผนภูมิใหม่ลงในคอลเลกชันรูปร่าง |
| initWithSample | Boolean | True เพื่อเริ่มต้นแผนภูมิใหม่ด้วยข้อมูลซีรีส์ตัวอย่างและการตั้งค่า; false เพื่อสร้างแผนภูมิโดยไม่มีซีรีส์และมีการตั้งค่าน้อยที่สุด ซึ่งทำให้การสร้างเร็วขึ้น |

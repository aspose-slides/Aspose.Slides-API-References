## InsertSectionZoomFrame(int, float, float, float, float, ISection) {#insertsectionzoomframe}

สร้าง Section Zoom frame ใหม่และแทรกลงในคอลเลกชันรูปทรงที่ตำแหน่งที่กำหนด

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| index | Int32 | ดัชนีที่เริ่มจากศูนย์ที่ใช้เพื่อแทรก Section Zoom frame. |
| x | Single | พิกัด x ของ Section Zoom frame ใหม่, หน่วยเป็น point. |
| y | Single | พิกัด y ของ Section Zoom frame ใหม่, หน่วยเป็น point. |
| width | Single | ความกว้างของ Section Zoom frame ใหม่, หน่วยเป็น point. |
| height | Single | ความสูงของ Section Zoom frame ใหม่, หน่วยเป็น point. |
| section | ISection | The [`ISection`](../../isection) ที่อ้างอิงโดย Section Zoom frame; ต้องเป็นของงานนำเสนอนี้และต้องมีสไลด์อย่างน้อยหนึ่งสไลด์. |

### ข้อยกเว้น

### ตัวอย่าง

ตัวอย่างนี้สาธิตการสร้างและการแทรกอ็อบเจ็กต์ Section Zoom ที่ตำแหน่งที่ระบุของคอลเลกชัน (สมมติว่ามีอย่างน้อยสองส่วนในงานนำเสนอ "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSectionZoomFrame(2, 150, 20, 50, 50, pres.Sections[1]);
}
```

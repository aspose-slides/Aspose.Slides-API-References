## InsertAutoShape(int, ShapeType, float, float, float, float, bool) {#insertautoshape_1}

สร้างรูปร่างอัตโนมัติใหม่และแทรกลงในคอลเลกชันรูปร่างตามดัชนีที่ระบุ โดยอาจกำหนดค่าเริ่มต้นด้วยการจัดรูปแบบเทมเพลต

```csharp
public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)
```

| พารามิเตอร์ | ชนิด | คำอธิบาย |
| --- | --- | --- |
| index | Int32 | ดัชนีที่เริ่มจากศูนย์ที่ใช้แทรกรูปร่างอัตโนมัติ |
| shapeType | ShapeType | [`ShapeType`](../../shapetype) ของรูปร่างอัตโนมัติที่จะแทรก |
| x | Single | พิกัด x ของกรอบรูปร่าง หน่วยเป็นจุด |
| y | Single | พิกัด y ของกรอบรูปร่าง หน่วยเป็นจุด |
| width | Single | ความกว้างของกรอบรูปร่าง หน่วยเป็นจุด |
| height | Single | ความสูงของกรอบรูปร่าง หน่วยเป็นจุด |

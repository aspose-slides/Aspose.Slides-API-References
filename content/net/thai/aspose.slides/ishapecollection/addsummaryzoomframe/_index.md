## IShapeCollection.AddSummaryZoomFrame เมธอด

สร้าง Summary Zoom frame ใหม่และเพิ่มไปที่ส่วนท้ายของคอลเลกชันรูปทรง

```csharp
public ISummaryZoomFrame AddSummaryZoomFrame(float x, float y, float width, float height)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| x | Single | พิกัด x ของ Summary Zoom frame ใหม่, หน่วยเป็นจุด |
| y | Single | พิกัด y ของ Summary Zoom frame ใหม่, หน่วยเป็นจุด |
| width | Single | ความกว้างของ Summary Zoom frame ใหม่, หน่วยเป็นจุด |
| height | Single | ความสูงของ Summary Zoom frame ใหม่, หน่วยเป็นจุด |

### หมายเหตุ

เมธอดนี้สร้าง Summary Zoom frame ที่รวบรวมลิงก์สรุปสำหรับทุกส่วนในงานนำเสนอ

### ตัวอย่าง

ตัวอย่างนี้แสดงการเพิ่มวัตถุ Summary Zoom ไปที่ส่วนท้ายของคอลเลกชัน (สมมติว่ามีอย่างน้อยสองส่วนในงานนำเสนอ “Presentation.pptx”):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSummaryZoomFrame(150, 20, 500, 250);
}
```

## InsertZoomFrame(int, float, float, float, float, ISlide) {#insertzoomframe}

สร้าง Zoom frame ใหม่และแทรกลงในคอลเลกชันรูปร่างที่ตำแหน่งที่ระบุ

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide)
```

| พารามิเตอร์ | ชนิด | คำอธิบาย |
| --- | --- | --- |
| index | Int32 | ดัชนีที่เริ่มจากศูนย์ที่ใช้แทรก Zoom frame |
| x | Single | พิกัด x ของ Zoom frame ใหม่, หน่วยเป็นพอยต์ |
| y | Single | พิกัด y ของ Zoom frame ใหม่, หน่วยเป็นพอยต์ |
| width | Single | ความกว้างของ Zoom frame ใหม่, หน่วยเป็นพอยต์ |
| height | Single | ความสูงของ Zoom frame ใหม่, หน่วยเป็นพอยต์ |
| slide | ISlide | The [`ISlide`](../../islide) ที่อ้างอิงโดย Zoom frame |

### ตัวอย่าง

```csharp

}
```

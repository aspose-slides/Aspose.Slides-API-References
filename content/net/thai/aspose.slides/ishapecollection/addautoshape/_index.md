## AddAutoShape(ShapeType, float, float, float, float) {#addautoshape}

| width | Single | ความกว้างของกรอบรูป, หน่วยเป็น points |

## AddAutoShape(ShapeType, float, float, float, float, bool) {#addautoshape_1}

สร้างรูปอัตโนมัติใหม่และเพิ่มลงในส่วนท้ายของคอลเลกชันรูป, โดยอาจกำหนดค่าเริ่มต้นด้วยการจัดรูปแบบเทมเพลตเริ่มต้น

```csharp
public IAutoShape AddAutoShape(ShapeType shapeType, float x, float y, float width, float height, 
    bool createFromTemplate)
```

| Parameter | Type | Description |
| --- | --- | --- |
| shapeType | ShapeType | [`ShapeType`](../../shapetype) ของรูปอัตโนมัติที่จะเพิ่ม |
| x | Single | พิกัด x ของกรอบรูป, หน่วยเป็น points |
| y | Single | พิกัด y ของกรอบรูป, หน่วยเป็น points |
| width | Single | ความกว้างของกรอบรูป, หน่วยเป็น points |
| height | Single | ความสูงของกรอบรูป, หน่วยเป็น points |
| createFromTemplate | Boolean | True เพื่อใช้การจัดรูปแบบเทมเพลตเริ่มต้น (สไตล์เรียบง่าย, ข้อความกึ่งกลาง, และชื่อที่ไม่ว่าง) กับรูปใหม่; false เพื่อสร้างรูปโดยกำหนดค่าทุกคุณสมบัติให้เป็นค่าตั้งต้น |

### Return Value

อ็อบเจกต์ [`IAutoShape`](../../iautoshape) ที่สร้างใหม่

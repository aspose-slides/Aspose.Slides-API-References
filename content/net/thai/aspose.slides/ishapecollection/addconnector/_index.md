## AddConnector(ShapeType, float, float, float, float, bool) {#addconnector_1}

สร้างรูปแบบคอนเนคเตอร์ใหม่และเพิ่มเข้าไปในตำแหน่งสุดท้ายของคอลเลกชันรูปแบบ โดยสามารถเลือกใช้สไตล์เทมเพลตเริ่มต้นได้

```csharp
public IConnector AddConnector(ShapeType shapeType, float x, float y, float width, float height, 
    bool createFromTemplate)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| shapeType | ShapeType | [`ShapeType`](../../shapetype) ของรูปแบบคอนเนคเตอร์ที่ต้องการสร้าง |
| x | Single | พิกัด x ของเฟรมคอนเนคเตอร์ หน่วยเป็นจุด |
| y | Single | พิกัด y ของเฟรมคอนเนคเตอร์ หน่วยเป็นจุด |
| width | Single | ความกว้างของเฟรมคอนเนคเตอร์ หน่วยเป็นจุด |
| height | Single | ความสูงของเฟรมคอนเนคเตอร์ หน่วยเป็นจุด |
| createFromTemplate | Boolean | true หากต้องการใช้สไตล์เทมเพลตเริ่มต้น (ชื่อไม่ว่าง, สไตล์ง่าย); false หากต้องการสร้างคอนเนคเตอร์ด้วยค่าเริ่มต้นของคุณสมบัติ |

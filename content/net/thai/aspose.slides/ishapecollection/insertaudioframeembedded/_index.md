## InsertAudioFrameEmbedded(int, float, float, float, float, Stream) {#insertaudioframeembedded_1}

    Stream audio_stream)

| --- | --- | --- |
| index | Int32 | ดัชนีแบบศูนย์ฐานที่ใช้ในการแทรกเฟรมเสียง |
| x | Single | พิกัด x ของเฟรมเสียงใหม่ หน่วยเป็นพอยท์ |
| y | Single | พิกัด y ของเฟรมเสียงใหม่ หน่วยเป็นพอยท์ |
| width | Single | ความกว้างของเฟรมเสียงใหม่ หน่วยเป็นพอยท์ |
| height | Single | ความสูงของเฟรมเสียงใหม่ หน่วยเป็นพอยท์ |
| audio_stream | Stream | สตรีมอินพุตที่มีข้อมูลเสียง WAV เพื่อฝัง |

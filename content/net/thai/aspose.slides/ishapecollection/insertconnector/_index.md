## InsertConnector(int, ShapeType, float, float, float, float, bool) {#insertconnector_1}

สร้างรูปร่างตัวเชื่อมใหม่และแทรกลงในคอลเลกชันของรูปร่างที่ตำแหน่งดัชนีที่ระบุ โดยอาจใช้สไตล์เทมเพลตเริ่มต้นตามต้องการ

```csharp
public IConnector InsertConnector(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | Int32 | ดัชนีที่เริ่มจากศูนย์ซึ่งใช้สำหรับแทรกรูปร่างตัวเชื่อม |
| shapeType | ShapeType | [`ShapeType`](../../shapetype) ของรูปร่างตัวเชื่อมที่จะทำการแทรก |
| x | Single | พิกัด x ของกรอบตัวเชื่อม หน่วยเป็นพอยต์ |
| y | Single | พิกัด y ของกรอบตัวเชื่อม หน่วยเป็นพอยต์ |
| width | Single | ความกว้างของกรอบตัวเชื่อม หน่วยเป็นพอยต์ |
| height | Single | ความสูงของกรอบตัวเชื่อม หน่วยเป็นพอยต์ |
| createFromTemplate | Boolean | true เพื่อใช้สไตล์เทมเพลตเริ่มต้น (ชื่อไม่ว่างเปล่า, สไตล์แบบง่าย); false เพื่อสร้างตัวเชื่อมด้วยค่าคุณสมบัติเบื้องต้น |

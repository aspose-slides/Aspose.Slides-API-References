## AddZoomFrame(float, float, float, float, ISlide) {#addzoomframe}

สร้าง Zoom frame ใหม่และเพิ่มเข้าไปในตำแหน่งสุดท้ายของชุด shape

```csharp
public IZoomFrame AddZoomFrame(float x, float y, float width, float height, ISlide slide)
```

| พารามิเตอร์ | ชนิด | คำอธิบาย |
| --- | --- | --- |
| x | Single | พิกัด X ของ Zoom frame ใหม่, หน่วยเป็นพอยต์. |
| y | Single | พิกัด Y ของ Zoom frame ใหม่, หน่วยเป็นพอยต์. |
| width | Single | ความกว้างของ Zoom frame ใหม่, หน่วยเป็นพอยต์. |
| height | Single | ความสูงของ Zoom frame ใหม่, หน่วยเป็นพอยต์. |
| slide | ISlide | [`ISlide`](../../islide) ที่อ้างอิงโดย Zoom frame; ต้องเป็นของงานนำเสนอชุดนี้. |

### ค่าที่ส่งกลับ

[`IZoomFrame`](../../izoomframe) ที่สร้างใหม่

### ข้อยกเว้น

| ข้อยกเว้น | เงื่อนไข |
| --- | --- |
| ArgumentException | เกิดขึ้นเมื่อ slide ที่อ้างอิงไม่ได้เป็นของงานนำเสนอปัจจุบัน. |

### ตัวอย่าง

ตัวอย่างนี้แสดงการเพิ่มออบเจ็กต์ Zoom เข้าไปที่ตำแหน่งสุดท้ายของคอลเลกชัน (สมมติว่ามีสไลด์อย่างน้อยสองสไลด์ในงานนำเสนอ "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
}
```

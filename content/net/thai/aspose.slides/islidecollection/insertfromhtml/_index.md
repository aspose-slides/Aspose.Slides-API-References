## InsertFromHtml(int, string, IExternalResourceResolver, string) {#insertfromhtml_7}

สร้างสไลด์จากข้อความ HTML และแทรกลงในคอลเลกชันที่ตำแหน่งที่ระบุไว้

```csharp
public ISlide[] InsertFromHtml(int index, string htmlText, IExternalResourceResolver resolver, 
    string uri)
```

| พารามิเตอร์ | ชนิด | คำอธิบาย |
| --- | --- | --- |
| index | Int32 | ตำแหน่งที่จะทำการแทรก |
| htmlText | String | HTML ที่จะเพิ่ม |
| resolver | IExternalResourceResolver | ออบเจ็กต์ callback ที่ใช้ดึงวัตถุภายนอก หากพารามิเตอร์นี้เป็น null จะละเว้นวัตถุภายนอกทั้งหมด |
| uri | String | URI ของ HTML ที่ระบุ ใช้สำหรับแก้ไขลิงก์สัมพันธ์ |

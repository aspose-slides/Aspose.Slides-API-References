## InsertClone(int, ISlide) {#insertclone}

แทรกสำเนาของสไลด์ที่ระบุไปยังตำแหน่งที่กำหนดในคอลเลกชัน

| sourceSlide | ISlide | สไลด์ที่จะโคลน. |

### ค่าที่คืนกลับ

สไลด์ที่แทรก

### หมายเหตุ

เมื่อทำการโคลนสไลด์ระหว่างการนำเสนอที่ต่างกัน มาสเตอร์ของสไลด์สามารถถูกโคลนได้เช่นกัน  
มีการใช้เรจิสทรีภายในเพื่อติดตามมาสเตอร์ที่ถูกโคลนอัตโนมัติ เพื่อป้องกันการสร้างสำเนาหลายชุดของมาสเตอร์สไลด์เดียวกัน  
การโคลนมาสเตอร์สไลด์ด้วยตนเองจะไม่ถูกป้องกันหรือบันทึก  
หากคุณต้องการควบคุมการโคลนขั้นสูง ให้ใช้ [`InsertClone`](../insertclone) หรือ [`InsertClone`](../insertclone) สำหรับการโคลนสไลด์และ [`AddClone`](../../imasterslidecollection/addclone) สำหรับการโคลนมาสเตอร์

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีการโคลนที่ตำแหน่งอื่นภายใน Presentation.

```csharp
[C#]
// สร้างอินสแตนซ์ของคลาส Presentation ที่เป็นตัวแทนของไฟล์การนำเสนอ
using (Presentation pres = new Presentation("CloneWithInSamePresentation.pptx"))
{
    // โคลนสไลด์ที่ต้องการไปยังตำแหน่งสุดท้ายของคอลเลกชันสไลด์ในการนำเสนอเดียวกัน
    ISlideCollection slds = pres.Slides;
    // โคลนสไลด์ที่ต้องการไปยังดัชนีที่ระบุในการนำเสนอเดียวกัน
    slds.InsertClone(2, pres.Slides[1]);
    // บันทึกการนำเสนอที่แก้ไขลงดิสก์
    pres.Save("Aspose_CloneWithInSamePresentation_out.pptx", SaveFormat.Pptx);
}
```

ตัวอย่างต่อไปนี้แสดงวิธีการโคลนที่ตำแหน่งอื่นภายใน Presentation.

```csharp
[C#]
// สร้างอินสแตนซ์ของคลาส Presentation เพื่อโหลดไฟล์การนำเสนอต้นฉบับ
using (Presentation srcPres = new Presentation("CloneAtEndOfAnother.pptx"))
{
    // สร้างอินสแตนซ์ของคลาส Presentation สำหรับไฟล์ PPTX ปลายทาง (ที่สไลด์จะถูกโคลน)
    using (Presentation destPres = new Presentation())
    {
        ISlideCollection slds = destPres.Slides;
        slds.InsertClone(2, srcPres.Slides[0]);
        // บันทึกการนำเสนอปลายทางลงดิสก์
        destPres.Save("Aspose2_out.pptx", SaveFormat.Pptx);
    }
}
```

## InsertClone(int, ISlide, IMasterSlide, bool) {#insertclone_2}

แทรกสำเนาของสไลด์ต้นฉบับที่ระบุไปยังตำแหน่งที่กำหนดในคอลเลกชัน รูปแบบ (layout) ที่เหมาะสมจะถูกเลือกโดยอัตโนมัติจากมาสเตอร์ที่ระบุ (รูปแบบที่เหมาะสมคือรูปแบบที่มี Type หรือ Name เหมือนกับรูปแบบของสไลด์ต้นฉบับ) หากไม่มีรูปแบบที่เหมาะสม รูปแบบของสไลด์ต้นฉบับจะถูกโคลน (หาก allowCloneMissingLayout เป็น true) หรือจะเกิด PptxEditException (หาก allowCloneMissingLayout เป็น false)

public ISlide InsertClone(int index, ISlide sourceSlide, IMasterSlide destMaster,

## AddFromHtml(Stream) {#addfromhtml}

สร้างสไลด์จากข้อความ HTML และเพิ่มไปยังส่วนท้ายของคอลเลกชัน

```csharp
public ISlide[] AddFromHtml(Stream htmlStream)
```

| พารามิเตอร์ | ชนิด | คำอธิบาย |
| --- | --- | --- |
| htmlStream | Stream | อ็อบเจ็กต์ Stream ที่จะใช้เป็นแหล่งของไฟล์ HTML. |

### ค่าที่คืนกลับ

สไลด์ที่เพิ่มแล้ว

### ตัวอย่าง

```csharp
[C#]
// สร้างอินสแตนซ์ของคลาส Presentation.
using (var presentation = new Presentation())
{
    using (var htmlStream = File.OpenRead("page.html"))
    {
		// เรียกใช้เมธอด AddFromHtml และส่งไฟล์ HTML.
        presentation.Slides.AddFromHtml(htmlStream);
    }
	// ใช้เมธอด Save เพื่อบันทึกไฟล์เป็นเอกสาร PowerPoint.
    presentation.Save("MyPresentation.pptx", SaveFormat.Pptx);
}
```

## IBasePortionFormat.SpellCheck คุณสมบัติ

รับหรือกำหนดค่าเพื่อระบุว่าการตรวจสอบการสะกดคำถูกเปิดใช้งานสำหรับส่วนของข้อความหรือไม่ เมื่อคุณสมบัตินี้ตั้งค่าเป็น false การตรวจสอบการสะกดคำสำหรับองค์ประกอบข้อความจะถูกระงับ เมื่อตั้งค่าเป็น true การตรวจสอบการสะกดคำจะได้รับอนุญาต ค่าเริ่มต้นคือ `false`.

```csharp
public bool SpellCheck { get; set; }
```

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงการเปิดใช้งานแฟล็ก SpellCheck ก่อนบันทึกงานนำเสนอ:

```csharp
[C#]
using (var pres = new Presentation("input.pptx"))
{
    // เข้าถึงส่วนข้อความแรกภายในรูปร่างแรกบนสไลด์แรก
    var portion = ((AutoShape)pres.Slides[0].Shapes[0]).TextFrame.Paragraphs[0].Portions[0];
    // เปิดการตรวจสอบการสะกดสำหรับส่วนข้อความนี้
    portion.PortionFormat.SpellCheck = true;
    // บันทึกการนำเสนอที่แก้ไขแล้ว
    pres.Save("output-with-spellcheck.pptx", SaveFormat.Pptx);
}
```

## PortionCollection.CopyTo เมธอด

คัดลอกองค์ประกอบของ ICollection ไปยัง Array เริ่มที่ตำแหน่งดัชนีของ Array ที่ระบุ

```csharp
public void CopyTo(IPortion[] array, int arrayIndex)
```

| พารามิเตอร์ | ชนิด | คำอธิบาย |
| --- | --- | --- |
| array | IPortion[] | Array แบบมิติเดียวที่เป็นปลายทางขององค์ประกอบที่คัดลอกจาก ICollection. Array ต้องมีการจัดทำดัชนีแบบศูนย์ฐาน |
| arrayIndex | Int32 | ดัชนีที่เริ่มจากศูนย์ใน *array* ที่การคัดลอกเริ่มต้น |

### ข้อยกเว้น

| ข้อยกเว้น | เงื่อนไข |
| --- | --- |
| ArgumentNullException | *array* เป็น null |
| ArgumentOutOfRangeException | *arrayIndex* น้อยกว่า 0 |
| ArgumentException | จำนวนองค์ประกอบใน ICollection ต้นทาง มากกว่าพื้นที่ที่มีอยู่ตั้งแต่ *arrayIndex* ถึงจุดสิ้นสุดของ *array* ปลายทาง |

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีเพิ่มโน้ตไปยังสไลด์เฉพาะของ PowerPoint Presentation.

```csharp
[C#]
	// สร้างออบเจ็กต์ Presentation ซึ่งแสดงถึงไฟล์งานนำเสนอ
	using(Presentation presentation = new Presentation(dataDir + "AccessSlides.pptx")) {
	  // เพิ่มโน้ตในสไลด์แรก
	  INotesSlideManager mgr = presentation.Slides[0].NotesSlideManager;
	  INotesSlide noteSlide = mgr.AddNotesSlide();
	  noteSlide.NotesTextFrame.Text = "Your Notes";
	  // บันทึกงานนำเสนอไปยังดิสก์
	  presentation.Save(dataDir + "RemoveNotesAtSpecificSlide_out.pptx", SaveFormat.Pptx);
	}
```

ตัวอย่างต่อไปนี้แสดงวิธีลบโน้ตจากสไลด์เฉพาะของ PowerPoint Presentation.

```csharp
[C#]
	// สร้างออบเจ็กต์ Presentation ซึ่งแสดงถึงไฟล์งานนำเสนอ
	using(Presentation presentation = new Presentation(dataDir + "AccessSlides.pptx")) {
	  // ลบโน้ตของสไลด์แรก
	  INotesSlideManager mgr = presentation.Slides[0].NotesSlideManager;
	  mgr.RemoveNotesSlide();
	  // บันทึกงานนำเสนอไปยังดิสก์
	  presentation.Save(dataDir + "RemoveNotesAtSpecificSlide_out.pptx", SaveFormat.Pptx);
	}
```

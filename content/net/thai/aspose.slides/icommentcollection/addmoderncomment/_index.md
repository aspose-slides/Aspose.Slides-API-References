## ICommentCollection.AddModernComment เมธอด

เพิ่มความคิดเห็นสมัยใหม่ใหม่ที่ส่วนท้ายของคอลเลกชัน

```csharp
public IModernComment AddModernComment(string text, ISlide slide, IShape shape, PointF position, 
    DateTime creationTime)
```

| พารามิเตอร์ | ชนิด | คำอธิบาย |
| --- | --- | --- |
| text | String | ข้อความธรรมดาของความคิดเห็นสมัยใหม่ใหม่ |
| slide | ISlide | สไลด์ในงานนำเสนอที่ต้องการเพิ่มความคิดเห็นสมัยใหม่ใหม่ |
| shape | IShape | รูปทรงบนสไลด์ที่เชื่อมโยงกับความคิดเห็นสมัยใหม่ใหม่ |
| position | PointF | ตำแหน่งบนสไลด์ที่ต้องการเพิ่มความคิดเห็นสมัยใหม่ใหม่ |
| creationTime | DateTime | เวลาในการสร้างความคิดเห็นสมัยใหม่ |

### ค่าที่ส่งคืน

ความคิดเห็นสมัยใหม่ที่เพิ่มเข้ามา

### ตัวอย่าง

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ICommentAuthor newAuthor = pres.CommentAuthors.AddAuthor("Some Author", "SA");
    newAuthor.Comments.AddModernComment("This is modern comment", pres.Slides[0], null, new PointF(100, 100), DateTime.Now);

    pres.Save(outPptxFileName, SaveFormat.Pptx);
}
```

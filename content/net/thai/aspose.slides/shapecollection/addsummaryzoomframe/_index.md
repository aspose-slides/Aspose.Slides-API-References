## ShapeCollection.AddSummaryZoomFrame เมธอด

สร้าง Summary Zoom frame ใหม่และเพิ่มไปยังส่วนท้ายของ shape collection.

```csharp
public ISummaryZoomFrame AddSummaryZoomFrame(float x, float y, float width, float height)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| x | Single | พิกัด x ของ Summary Zoom frame ใหม่, หน่วยเป็น point. |
| y | Single | พิกัด y ของ Summary Zoom frame ใหม่, หน่วยเป็น point. |
| width | Single | ความกว้างของ Summary Zoom frame ใหม่, หน่วยเป็น point. |
| height | Single | ความสูงของ Summary Zoom frame ใหม่, หน่วยเป็น point. |

### ค่าที่ส่งกลับ

อ็อบเจ็กต์ที่สร้างใหม่ [`ISummaryZoomFrame`](../../isummaryzoomframe).

### หมายเหตุ

เมธอดนี้สร้าง Summary Zoom ใหม่และใส่คอลเลกชันของอ็อบเจ็กต์ลงไปสำหรับทุก section ใน presentation นี้.

### ตัวอย่าง

ตัวอย่างนี้แสดงการเพิ่มอ็อบเจ็กต์ Summary Zoom ไปยังส่วนท้ายของคอลเลกชัน (สมมติว่ามีอย่างน้อยสอง section ใน presentation "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSummaryZoomFrame(150, 20, 500, 250);
}
```

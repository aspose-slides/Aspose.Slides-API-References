## InsertAutoShape(int, ShapeType, float, float, float, float, bool) {#insertautoshape_1}

```csharp
public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| index | Int32 | ดัชนีที่เริ่มจากศูนย์ที่ใช้ในการแทรกรูปร่างอัตโนมัติ |

| x | Single | พิกัด x ของกรอบของรูปร่าง, หน่วยเป็นพอยต์ |
| y | Single | พิกัด y ของกรอบของรูปร่าง, หน่วยเป็นพอยต์ |
| width | Single | ความกว้างของกรอบของรูปร่าง, หน่วยเป็นพอยต์ |
| height | Single | ความสูงของกรอบของรูปร่าง, หน่วยเป็นพอยต์ |

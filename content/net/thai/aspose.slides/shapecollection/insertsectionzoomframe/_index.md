## InsertSectionZoomFrame(int, float, float, float, float, ISection) {#insertsectionzoomframe}

สร้างกรอบ Section Zoom ใหม่และแทรกลงในคอลเลกชัน shape ที่ตำแหน่งที่ระบุ

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | Int32 | ดัชนีที่เริ่มจากศูนย์ที่ใช้เพื่อแทรกกรอบ Section Zoom |
| x | Single | พิกัด x ของกรอบ Section Zoom ใหม่ หน่วยเป็นจุด |
| y | Single | พิกัด y ของกรอบ Section Zoom ใหม่ หน่วยเป็นจุด |
| width | Single | ความกว้างของกรอบ Section Zoom ใหม่ หน่วยเป็นจุด |
| height | Single | ความสูงของกรอบ Section Zoom ใหม่ หน่วยเป็นจุด |
| section | ISection | [`ISection`](../../isection) ที่อ้างอิงโดยกรอบ Section Zoom; ต้องเป็นของงานนำเสนอปัจจุบันและต้องมีสไลด์อย่างน้อยหนึ่งแผ่น |

### ตัวอย่าง

ตัวอย่างนี้สาธิตการสร้างและแทรกวัตถุ Section Zoom ที่ตำแหน่งที่ระบุของคอลเลกชัน (สมมติว่ามีอย่างน้อยสองส่วนในงานนำเสนอ “Presentation.pptx”):

```csharp

    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSectionZoomFrame(2, 150, 20, 50, 50, pres.Sections[1]);
}
```

public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width,

| x | Single | พิกัด x ของกรอบ Section Zoom ใหม่ หน่วยเป็นจุด |
| y | Single | พิกัด y ของกรอบ Section Zoom ใหม่ หน่วยเป็นจุด |
| width | Single | ความกว้างของกรอบ Section Zoom ใหม่ หน่วยเป็นจุด |

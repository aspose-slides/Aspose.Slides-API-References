## InsertChart(ChartType, float, float, float, float, int) {#insertchart}

```csharp
public IChart InsertChart(ChartType type, float x, float y, float width, float height, int index)
```

| --- | --- | --- |
| type | ChartType | ประเภทของแผนภูมิที่ต้องการสร้าง. |
| x | Single | พิกัด x ของแผนภูมิใหม่, หน่วยเป็นจุด. |
| y | Single | พิกัด y ของแผนภูมิใหม่, หน่วยเป็นจุด. |
| width | Single | ความกว้างของแผนภูมิใหม่, หน่วยเป็นจุด. |
| height | Single | ความสูงของแผนภูมิใหม่, หน่วยเป็นจุด. |
| index | Int32 | ดัชนีเริ่มจากศูนย์ที่ต้องการแทรกแผนภูมิใหม่ในคอลเลกชันรูปทรง. |

## InsertChart(ChartType, float, float, float, float, int, bool) {#insertchart_1}

สร้างแผนภูมิใหม่, กำหนดค่าเริ่มต้นด้วยข้อมูลชุดตัวอย่างและการตั้งค่า, แล้วแทรกลงในคอลเลกชันรูปทรงที่ตำแหน่งที่ระบุ

```csharp
public IChart InsertChart(ChartType type, float x, float y, float width, float height, int index, 
    bool initWithSample)
```

| Parameter | Type | Description |
| --- | --- | --- |
| type | ChartType | ประเภทของแผนภูมิที่ต้องการสร้าง. |
| x | Single | พิกัด x ของแผนภูมิใหม่, หน่วยเป็นจุด. |
| y | Single | พิกัด y ของแผนภูมิใหม่, หน่วยเป็นจุด. |
| width | Single | ความกว้างของแผนภูมิใหม่, หน่วยเป็นจุด. |
| height | Single | ความสูงของแผนภูมิใหม่, หน่วยเป็นจุด. |
| index | Int32 | ดัชนีเริ่มจากศูนย์ที่ต้องการแทรกแผนภูมิใหม่ในคอลเลกชันรูปทรง. |
| initWithSample | Boolean | true เพื่อกำหนดค่าเริ่มต้นด้วยข้อมูลชุดตัวอย่าง; false เพื่อสร้างแผนภูมิที่ไม่มีชุดข้อมูลและมีการตั้งค่าน้อยที่สุด, ซึ่งทำให้การสร้างเร็วขึ้น. |

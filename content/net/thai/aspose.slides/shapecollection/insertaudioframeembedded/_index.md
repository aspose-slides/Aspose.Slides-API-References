## InsertAudioFrameEmbedded(int, float, float, float, float, Stream) {#insertaudioframeembedded_1}

```csharp
public IAudioFrame InsertAudioFrameEmbedded(int index, float x, float y, float width, float height, 
    Stream audio_stream)
```

| --- | --- | --- |
| index | Int32 | ดัชนีที่เริ่มจากศูนย์ที่ต้องการแทรกเฟรมเสียง |
| x | Single | พิกัด x ของเฟรมเสียงใหม่ หน่วยเป็นพอยต์ |
| y | Single | พิกัด y ของเฟรมเสียงใหม่ หน่วยเป็นพอยต์ |
| width | Single | ความกว้างของเฟรมเสียงใหม่ หน่วยเป็นพอยต์ |
| height | Single | ความสูงของเฟรมเสียงใหม่ หน่วยเป็นพอยต์ |
| audio_stream | Stream | สตรีมข้อมูลเข้าที่มีข้อมูลเสียง WAV เพื่อฝัง |

## AddConnector(ShapeType, float, float, float, float) {#addconnector}

สร้างรูปเชื่อมต่อใหม่โดยใช้สไตล์แม่แบบเริ่มต้นและเพิ่มไปยังตำแหน่งสุดท้ายของคอลเลกชันรูป

```csharp
public IConnector AddConnector(ShapeType shapeType, float x, float y, float width, float height)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| shapeType | ShapeType | ค่า [`ShapeType`](../../shapetype) ของรูปเชื่อมต่อที่ต้องการเพิ่ม |
| x | Single | พิกัด x ของเฟรมของรูปเชื่อมต่อ, มีหน่วยเป็นจุด |
| y | Single | พิกัด y ของเฟรมของรูปเชื่อมต่อ, มีหน่วยเป็นจุด |
| width | Single | ความกว้างของเฟรมของรูปเชื่อมต่อ, มีหน่วยเป็นจุด |
| height | Single | ความสูงของเฟรมของรูปเชื่อมต่อ, มีหน่วยเป็นจุด |

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีการเพิ่มรูปเชื่อมต่อ (รูปเชื่อมต่อแบบโค้ง) ระหว่างสองรูป (วงรีและสี่เหลี่ยม) ใน PowerPoint Presentation

```csharp
[C#]
// สร้างอินสแตนซ์ของคลาส Presentation ที่แทนไฟล์ PPTX
using (Presentation input = new Presentation())
{
    // เข้าถึงคอลเลกชันรูปสำหรับสไลด์เฉพาะ
    IShapeCollection shapes = input.Slides[0].Shapes;
    // เพิ่ม autoshape รูปวงรี
    IAutoShape ellipse = shapes.AddAutoShape(ShapeType.Ellipse, 0, 100, 100, 100);
    // เพิ่ม autoshape สี่เหลี่ยมผืนผ้า
    IAutoShape rectangle = shapes.AddAutoShape(ShapeType.Rectangle, 100, 300, 100, 100);
    // เพิ่มรูปเชื่อมต่อลงในคอลเลกชันรูปของสไลด์
    IConnector connector = shapes.AddConnector(ShapeType.BentConnector2, 0, 0, 10, 10);
    // เชื่อมต่อรูปโดยใช้ตัวเชื่อมต่อ
    connector.StartShapeConnectedTo = ellipse;
    connector.EndShapeConnectedTo = rectangle;
    // เรียกใช้ reroute ที่ตั้งค่าเส้นทางสั้นที่สุดแบบอัตโนมัติระหว่างรูป
    connector.Reroute();
    // บันทึกงานนำเสนอ
    input.Save("Shapes-connector.pptx", SaveFormat.Pptx);
}
```

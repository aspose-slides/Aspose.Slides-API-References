## AddSectionZoomFrame(float, float, float, float, ISection) {#addsectionzoomframe}

สร้างกรอบ Section Zoom ใหม่และเพิ่มไปยังส่วนท้ายของคอลเลกชันรูปทรง

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| x | Single | พิกัด x ของกรอบ Section Zoom ใหม่, หน่วยเป็นจุด. |
| y | Single | พิกัด y ของกรอบ Section Zoom ใหม่, หน่วยเป็นจุด. |
| width | Single | ความกว้างของกรอบ Section Zoom ใหม่, หน่วยเป็นจุด. |
| height | Single | ความสูงของกรอบ Section Zoom ใหม่, หน่วยเป็นจุด. |
| section | ISection | [`ISection`](../../isection) ที่อ้างอิงโดยกรอบ Section Zoom; ต้องเป็นของงานนำเสนอเดียวกันและต้องมีสไลด์อย่างน้อยหนึ่งสไลด์. |

### ค่าที่ส่งกลับ

[`ISectionZoomFrame`](../../isectionzoomframe) ที่สร้างใหม่.

### ข้อยกเว้น

| ข้อยกเว้น | เงื่อนไข |
| --- | --- |
| ArgumentException | เกิดขึ้นหากส่วนที่อ้างอิงไม่เป็นของงานนำเสนอปัจจุบันหรือไม่มีสไลด์ใดๆ. |

### ตัวอย่าง

ตัวอย่างนี้แสดงการเพิ่มออบเจกต์ Section Zoom ไปยังส่วนท้ายของคอลเลกชัน (สมมติว่ามีอย่างน้อยสองส่วนในงานนำเสนอ "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1]);
}
```

| y | Single | พิกัด y ของกรอบ Section Zoom ใหม่, หน่วยเป็นจุด. |
| width | Single | ความกว้างของกรอบ Section Zoom ใหม่, หน่วยเป็นจุด. |

}

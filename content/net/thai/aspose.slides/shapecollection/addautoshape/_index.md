## AddAutoShape(ShapeType, float, float, float, float, bool) {#addautoshape_1}

สร้างรูปอัตโนมัติใหม่และเพิ่มลงในส่วนท้ายของคอลเลกชันรูป, โดยอาจกำหนดค่าเริ่มต้นด้วยรูปแบบเทมเพลตเริ่มต้น

```csharp
public IAutoShape AddAutoShape(ShapeType shapeType, float x, float y, float width, float height, 
    bool createFromTemplate)
```

| พารามิเตอร์ | ชนิด | คำอธิบาย |
| --- | --- | --- |
| shapeType | ShapeType | [`ShapeType`](../../shapetype) ของรูปอัตโนมัติที่ต้องการเพิ่ม. |
| x | Single | พิกัด x ของกรอบรูป, หน่วยเป็น point. |
| y | Single | พิกัด y ของกรอบรูป, หน่วยเป็น point. |
| width | Single | ความกว้างของกรอบรูป, หน่วยเป็น point. |
| height | Single | ความสูงของกรอบรูป, หน่วยเป็น point. |
| createFromTemplate | Boolean | True เพื่อใช้สไตล์เทมเพลตเริ่มต้น (สไตล์ง่าย, ข้อความกึ่งกลาง, และชื่อไม่ว่าง) กับรูปใหม่; false เพื่อสร้างรูปโดยกำหนดค่าทุกคุณสมบัติเป็นค่าดีฟอลต์. |

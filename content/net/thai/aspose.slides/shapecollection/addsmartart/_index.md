## ShapeCollection.AddSmartArt เมธอด

สร้างแผนภาพ SmartArt และเพิ่มไปยังส่วนท้ายของคอลเลกชันของรูปร่าง

```csharp
public ISmartArt AddSmartArt(float x, float y, float width, float height, 
    SmartArtLayoutType layoutType)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| x | Single | พิกัด x ของกรอบแผนภาพ, หน่วยเป็นจุด |
| y | Single | พิกัด y ของกรอบแผนภาพ, หน่วยเป็นจุด |
| width | Single | ความกว้างของกรอบแผนภาพ, หน่วยเป็นจุด |
| height | Single | ความสูงของกรอบแผนภาพ, หน่วยเป็นจุด |
| layoutType | SmartArtLayoutType | ประเภทเค้าโครง SmartArt |

### ค่าที่ส่งคืน

อ็อบเจกต์ที่สร้างใหม่ [`ISmartArt`](../../../aspose.slides.smartart/ismartart).

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีเพิ่ม smart shape ใน PowerPoint Presentation

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	Slide slide = pres.Slides[0];
	SmartArt smart = slide.Shapes.AddSmartArt(0, 0, 400, 400, SmartArtLayoutType.BasicBlockList);
}
```

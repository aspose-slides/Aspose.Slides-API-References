## InsertConnector(int, ShapeType, float, float, float, float, bool) {#insertconnector_1}

สร้างรูปเชื่อมต่อใหม่และใส่ลงในคอลเลกชันรูปที่ตำแหน่งดัชนีที่ระบุ, โดยอาจใช้การจัดรูปแบบเทมเพลตเริ่มต้น

```csharp
public IConnector InsertConnector(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | Int32 | ดัชนีที่เริ่มจากศูนย์ที่ใช้ในการแทรกรูปเชื่อมต่อ |
| shapeType | ShapeType | The [`ShapeType`](../../shapetype) ของรูปเชื่อมต่อที่จะแทรก |
| x | Single | พิกัด x ของกรอบของรูปเชื่อมต่อ, หน่วยเป็นจุด |
| y | Single | พิกัด y ของกรอบของรูปเชื่อมต่อ, หน่วยเป็นจุด |
| width | Single | ความกว้างของกรอบรูปเชื่อมต่อ, หน่วยเป็นจุด |
| height | Single | ความสูงของกรอบรูปเชื่อมต่อ, หน่วยเป็นจุด |
| createFromTemplate | Boolean | True เพื่อใช้การจัดรูปแบบเทมเพลตเริ่มต้น (ชื่อไม่ว่างเปล่า, รูปแบบง่าย); false เพื่อสร้างรูปเชื่อมต่อด้วยค่าคุณสมบัติมาตรฐาน |

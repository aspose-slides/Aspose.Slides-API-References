## AddOleObjectFrame(float, float, float, float, IOleEmbeddedDataInfo) {#addoleobjectframe}

สร้างกรอบอ็อบเจกต์ OLE ใหม่และเพิ่มเข้าไปที่ส่วนท้ายของคอลเลกชันรูปทรง

```csharp
public IOleObjectFrame AddOleObjectFrame(float x, float y, float width, float height, 
    IOleEmbeddedDataInfo dataInfo)
```

| Parameter | Type | Description |
| --- | --- | --- |
| x | Single | พิกัด x ของกรอบ OLE ใหม่, หน่วยเป็นจุด. |
| y | Single | พิกัด y ของกรอบ OLE ใหม่, หน่วยเป็นจุด. |
| width | Single | ความกว้างของกรอบ OLE ใหม่, หน่วยเป็นจุด. |
| height | Single | ความสูงของกรอบ OLE ใหม่, หน่วยเป็นจุด. |
| dataInfo | IOleEmbeddedDataInfo | ข้อมูลเกี่ยวกับข้อมูล OLE ที่ฝังอยู่ ([`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo)). |

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีการเพิ่มกรอบอ็อบเจกต์ OLE ไปยังสไลด์ของการนำเสนอ PowerPoint

```csharp
[C#]
    // สร้างอินสแตนซ์ของคลาส Presentation ที่แทนไฟล์ PPTX
    using (Presentation pres = new Presentation())
    {
        // เข้าถึงสไลด์แรก
        ISlide sld = pres.Slides[0];
        // โหลดไฟล์ Excel ไปยังสตรีม
        MemoryStream mstream = new MemoryStream();
        using (FileStream fs = new FileStream("book1.xlsx", FileMode.Open, FileAccess.Read))
        {
            byte[] buf = new byte[4096];
            while (true)
            {
                int bytesRead = fs.Read(buf, 0, buf.Length);
                if (bytesRead <= 0)
                    break;
                mstream.Write(buf, 0, bytesRead);
            }
        }
        // สร้างอ็อบเจกต์ข้อมูลสำหรับฝัง
        IOleEmbeddedDataInfo dataInfo = new OleEmbeddedDataInfo(mstream.ToArray(), "xlsx");
        // เพิ่มรูปแบบกรอบอ็อบเจกต์ Ole
        IOleObjectFrame oleObjectFrame = sld.Shapes.AddOleObjectFrame(0, 0, pres.SlideSize.Size.Width,
            pres.SlideSize.Size.Height, dataInfo);
        // เขียนไฟล์ PPTX ไปยังดิสก์
        pres.Save("OleEmbed_out.pptx", SaveFormat.Pptx);
    }
```

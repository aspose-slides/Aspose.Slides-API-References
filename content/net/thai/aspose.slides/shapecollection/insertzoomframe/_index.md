## InsertZoomFrame(int, float, float, float, float, ISlide) {#insertzoomframe}

สร้างเฟรม Zoom ใหม่และแทรกลงในคอลเลกชันรูปทรงที่ตำแหน่งที่ระบุ

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide)
```

| พารามิเตอร์ | ชนิด | คำอธิบาย |
| --- | --- | --- |
| index | Int32 | ดัชนีที่เริ่มจาก 0 ที่จะใช้แทรกเฟรม Zoom |
| x | Single | พิกัด x ของเฟรม Zoom ใหม่ หน่วยเป็นพอยต์ |
| y | Single | พิกัด y ของเฟรม Zoom ใหม่ หน่วยเป็นพอยต์ |
| width | Single | ความกว้างของเฟรม Zoom ใหม่ หน่วยเป็นพอยต์ |
| height | Single | ความสูงของเฟรม Zoom ใหม่ หน่วยเป็นพอยต์ |

### Examples

ตัวอย่างนี้สาธิตการสร้างและแทรกอ็อบเจ็กต์ Zoom ที่ตำแหน่งที่ระบุของคอลเลกชัน (สมมติว่ามีอย่างน้อยสองสไลด์ในงานนำเสนอ "Presentation.pptx"):

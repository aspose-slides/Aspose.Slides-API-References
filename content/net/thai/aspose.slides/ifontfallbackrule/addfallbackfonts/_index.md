## AddFallBackFonts(string) {#addfallbackfonts}

เพิ่มฟอนต์ใหม่ (หรือหลายฟอนต์) ไปยังรายการฟอนต์ FallBack.

```csharp
public void AddFallBackFonts(string fontName)
```

| Parameter | Type | Description |
| --- | --- | --- |
| fontName | String | ชื่อหรือหลายชื่อของฟอนต์ (คั่นด้วยเครื่องหมายคอมม่า) สำหรับ FallBack |

### ตัวอย่าง

```csharp
[C#]
//สร้างอินสแตนซ์ใหม่ของ FantFallBackRule
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho");

//เพิ่มฟอนต์ที่สองลงในกฎ 
newRule.AddFallBackFonts("MS Gothic");
//เพิ่มฟอนต์ที่สามและสี่ลงในกฎ 
newRule.AddFallBackFonts("Tahoma, Times New Roman");
```

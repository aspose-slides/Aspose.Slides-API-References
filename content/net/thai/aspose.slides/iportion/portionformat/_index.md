## IPortion.PortionFormat คุณสมบัติ

ส่งคืนอ็อบเจ็กต์การจัดรูปแบบที่มีคุณสมบัติการจัดรูปแบบที่กำหนดอย่างชัดเจนของส่วนข้อความโดยไม่มีการสืบทอดใดๆ อ่านอย่างเดียว [`IPortionFormat`](../../iportionformat).

```csharp
public IPortionFormat PortionFormat { get; }
```

### หมายเหตุ

อ็อบเจ็กต์การจัดรูปแบบนี้มีพารามิเตอร์การจัดรูปแบบที่กำหนดไว้สำหรับส่วนปัจจุบันเท่านั้น ไม่ได้นำข้อมูลที่สืบทอดมาประยุกต์ใช้.

หากต้องการรับค่าอย่างมีประสิทธิภาพรวมถึงค่าที่สืบทอดด้วย ให้ใช้เมธอด [`GetEffective`](../../iportionformat/geteffective).

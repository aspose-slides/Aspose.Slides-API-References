## SetLicense(string) {#setlicense_1}

ให้สิทธิการใช้งานกับคอมโพเนนต์

```csharp
public void SetLicense(string licenseName)
```

| Parameter | Type | Description |
| --- | --- | --- |
| licenseName | String | สามารถเป็นชื่อไฟล์เต็มหรือสั้น หรือชื่อของทรัพยากรฝังตัว ใช้สตริงว่างเพื่อสลับเป็นโหมดประเมินผล |

### Remarks

พยายามค้นหาไลเซนส์ในตำแหน่งต่อไปนี้

1. เส้นทางที่ระบุโดยชัดเจน
2. โฟลเดอร์ของแอสเซมบลีคอมโพเนนต์
3. โฟลเดอร์ของแอสเซมบลีที่เรียกใช้ของไคลเอนท์
4. โฟลเดอร์ของแอสเซมบลีเริ่มต้น
5. ทรัพยากรฝังตัวในแอสเซมบลีที่เรียกใช้ของไคลเอนท์

**หมายเหตุ:**บน .NET Compact Framework จะพยายามค้นหาไลเซนส์เฉพาะในตำแหน่งเหล่านี้

1. เส้นทางที่ระบุโดยชัดเจน
2. ทรัพยากรฝังตัวในแอสเซมบลีที่เรียกใช้ของไคลเอนท์

### Examples

ในตัวอย่างนี้ จะพยายามค้นหาไฟล์ไลเซนส์ชื่อ MyLicense.lic ในโฟลเดอร์ที่ประกอบคอมโพเนนต์, โฟลเดอร์ที่ประกอบแอสเซมบลีที่เรียกใช้, โฟลเดอร์ของแอสเซมบลีเริ่มต้น และจากทรัพยากรฝังตัวของแอสเซมบลีที่เรียกใช้

```csharp
[C#]

License license = new License();
license.SetLicense("MyLicense.lic");


[Visual Basic]

Dim license As License = New License
license.SetLicense("MyLicense.lic")
```

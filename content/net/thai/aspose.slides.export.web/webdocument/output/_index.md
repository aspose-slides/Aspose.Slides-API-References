## WebDocument.Output คุณสมบัติ

ส่งคืนคอลเลกชันขององค์ประกอบผลลัพธ์ของเอกสาร. อ่านอย่างเดียว `Output`.

```csharp
public Output Output { get; }
```

### ตัวอย่าง

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    var options = new WebDocumentOptions
    {
        TemplateEngine = new RazorTemplateEngine(),
        OutputSaver = new FileOutputSaver(),
        EmbedImages = false
    };
    
    WebDocument document = new WebDocument(options);

    // เพิ่ม "index.html" ไปยังไฟล์ผลลัพธ์, ใช้เทมเพลต "index" เพื่อสร้างและตัวแปร pres เป็นโมเดล
    document.Output.Add("index.html", "index", pres);

    // ... ตั้งค่า ตัวเลือกอื่น ๆ ของเอกสารแล้วบันทึกเอกสาร
    document.Save();
}
```

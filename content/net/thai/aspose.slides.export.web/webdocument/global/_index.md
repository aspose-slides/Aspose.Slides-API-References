## WebDocument.Global คุณสมบัติ

ส่งคืนการจัดเก็บแบบทั่วโลกของเอกสาร อ่านเท่านั้น [`Storage`](../../storage).

```csharp
public Storage Global { get; }
```

### ตัวอย่าง

โดยใช้ `Global` คุณสมบัตินี้ (การดำเนินการของอินเทอร์เฟซ [`Storage`](../../storage)) คุณสมบัติสามารถนำไปใช้ต่อในเทมเพลตได้:

```csharp
[C#]
var options = new WebDocumentOptions
{
    TemplateEngine = new RazorTemplateEngine(),
    OutputSaver = new FileOutputSaver(),
    EmbedImages = false
};

WebDocument document = new WebDocument(options);

// ใส่คุณสมบัติ "slideMargin" เพื่อใช้งานจากเทมเพลต
document.Global.Put("slideMargin", 10);

// ... set up other options of the document and then save the document
document.Save();
```

## IDataLabel.IsVisible คุณสมบัติ

False หมายถึง data label ไม่แสดง (และดังนั้นทุก Show*-flags (ShowValue, ...) จะเป็น false). อ่านอย่างเดียว Boolean.

```csharp
public bool IsVisible { get; }
```

### หมายเหตุ

หาก data label แสดงคุณสามารถทำให้มันซ่อนได้โดยใช้เมธอด Hide(). แต่หาก data label ไม่แสดง (IsVisible เป็น false) คุณสามารถทำให้ data label แสดงได้โดยตั้งค่า Show*-flags (ShowValue, ...) ให้เป็นสถานะ true.

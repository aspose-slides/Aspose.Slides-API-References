## IOverridableText.TextFrameForOverriding คุณสมบัติ

สามารถบรรจุข้อความที่จัดรูปแบบอย่างหลากหลายได้ หากคุณสมบัตินี้ไม่เป็น null แล้วค่าข้อความที่จัดรูปแบบนี้จะทับข้อความที่สร้างโดยอัตโนมัติ ข้อความที่สร้างโดยอัตโนมัติเป็นคุณสมบัติโดยอ้อมของป้ายข้อมูล, ป้ายหน่วยแสดงของแกนค่า, ชื่อแกน, ชื่อแผนภูมิ, ป้ายของเส้นแนวโน้ม ข้อความที่สร้างโดยอัตโนมัติถูกจัดรูปแบบด้วยคุณสมบัติ IFormattedTextContainer.TextFormat อ่านอย่างเดียว [`ITextFrame`](../../../aspose.slides/itextframe).

```csharp
public ITextFrame TextFrameForOverriding { get; }
```

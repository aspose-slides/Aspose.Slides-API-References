## ChartTitle.TextFrameForOverriding คุณสมบัติ

สามารถบรรจุข้อความที่จัดรูปแบบอย่างรัดกุมได้. ถ้าคุณสมบัตินี้ไม่เป็น null แล้วค่าข้อความที่จัดรูปแบบนี้จะทับข้อความที่สร้างอัตโนมัติ. ข้อความที่สร้างอัตโนมัติเป็นคุณสมบัติโดยอ้อมของป้ายข้อมูล, ป้ายหน่วยแสดงผลของแกนค่า, ชื่อแกน, ชื่อแผนภูมิ, ป้ายของเส้นแนวโน้ม. ข้อความที่สร้างอัตโนมัติถูกจัดรูปแบบด้วย IFormattedTextContainer.TextFormat คุณสมบัติ. อ่านอย่างเดียว [`ITextFrame`](../../../aspose.slides/itextframe).

```csharp
public ITextFrame TextFrameForOverriding { get; }
```

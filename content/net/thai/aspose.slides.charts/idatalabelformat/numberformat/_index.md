## IDataLabelFormat.NumberFormat คุณสมบัติ

เป็นสตริงฟอร์แมตสำหรับวัตถุ DataLabels. อ่าน/เขียน String.

```csharp
public string NumberFormat { get; set; }
```

### หมายเหตุ

หากพาเรนต์ของวัตถุ DataLabelFormat นี้เป็นคอลเลกชัน DataLabelCollection ของป้ายข้อมูล, แล้วคุณสมบัตินี้จะรับหรือกำหนดค่าเริ่มต้นของคุณสมบัติ NumberFormat สำหรับป้ายข้อมูลใหม่ในคอลเลกชัน DataLabelCollection. เมื่อคุณสมบัตินี้ถูกตั้งค่าเป็นค่าหนึ่ง ค่านั้นจะถูกตั้งค่าให้กับคุณสมบัติ NumberFormat สำหรับป้ายข้อมูลทั้งหมดในคอลเลกชัน DataLabelCollection ด้วย (เช่น "DataLabels.DefaultDataLabelFormat.NumberFormat = val;" ทำให้ DataLabels[i].NumberFormat ทั้งหมดเท่ากับ val).

### ตัวอย่าง

```csharp
[C#]
series.Labels.DefaultDataLabelFormat.ShowValue = true;
series.Labels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = false;
series.Labels.DefaultDataLabelFormat.NumberFormat = "0.0%";
```

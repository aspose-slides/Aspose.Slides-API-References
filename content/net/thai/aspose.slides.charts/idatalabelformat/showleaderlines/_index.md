## IDataLabelFormat.ShowLeaderLines คุณสมบัติ

แสดงพฤติกรรมการแสดงเส้นนำของป้ายข้อมูลในแผนภูมาที่ระบุ. True แสดงเส้นนำ. False เพื่อซ่อน. อ่าน/เขียน Boolean.

```csharp
public bool ShowLeaderLines { get; set; }
```

### หมายเหตุ

หากพาเรนท์ของอ็อบเจ็กต์ DataLabelFormat นี้เป็นคอลเลกชัน DataLabelCollection ของป้ายข้อมูลแล้วคุณสมบัตินี้จะดึงหรือกำหนดค่าตั้งต้นของคุณสมบัติ ShowLeaderLines สำหรับป้ายข้อมูลใหม่ในคอลเลกชัน DataLabelCollection. ตั้งค่าคุณสมบัตินี้ด้วยค่าใดค่าหนึ่งจะทำให้ค่าดังกล่าวถูกกำหนดให้กับคุณสมบัติ ShowLeaderLines ของป้ายข้อมูลทั้งหมดในคอลเลกชัน DataLabelCollection (เช่น "DataLabels.DefaultDataLabelFormat.ShowLeaderLines = val;" ทำให้ DataLabels[i].ShowLeaderLines มีค่าเท่ากับ val).

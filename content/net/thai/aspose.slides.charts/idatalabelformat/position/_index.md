## IDataLabelFormat.Position คุณลักษณะ

แสดงตำแหน่งของป้ายข้อมูล. อ่าน/เขียน [`LegendDataLabelPosition`](../../legenddatalabelposition).

```csharp
public LegendDataLabelPosition Position { get; set; }
```

### หมายเหตุ

หากพาเรนต์ของวัตถุ DataLabelFormat นี้เป็นคอลเลกชัน DataLabelCollection ของป้ายข้อมูลแล้ว คุณลักษณะนี้จะรับหรือกำหนดค่าที่เป็นค่าเริ่มต้นของคุณลักษณะ Position สำหรับป้ายข้อมูลใหม่ในคอลเลกชัน DataLabelCollection. แสดงตำแหน่งสำหรับอ็อบเจ็กต์ DataLabel. ตั้งค่าคุณลักษณะนี้ด้วยค่า จะกำหนดค่านั้นให้กับคุณลักษณะ Position ของป้ายข้อมูลทั้งหมดในคอลเลกชัน DataLabelCollection (เช่น "DataLabels.DefaultDataLabelFormat.Position = val;" ทำให้ DataLabels[i].Position มีค่าเท่ากับ val).

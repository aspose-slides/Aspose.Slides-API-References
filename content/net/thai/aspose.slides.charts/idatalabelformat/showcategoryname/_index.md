## IDataLabelFormat.ShowCategoryName คุณสมบัติ

แสดงพฤติกรรมการแสดงชื่อหมวดหมู่ของป้ายข้อมูลสำหรับแผนภูมิที่ระบุ. True เพื่อแสดงชื่อหมวดหมู่สำหรับป้ายข้อมูลบนแผนภูมิ. False เพื่อซ่อน. อ่าน/เขียน Boolean.

```csharp
public bool ShowCategoryName { get; set; }
```

### หมายเหตุ

หากพาเรนท์ของวัตถุ DataLabelFormat นี้เป็นคอลเลกชัน DataLabelCollection ของป้ายข้อมูล แล้วคุณสมบัตินี้จะรับหรือกำหนดค่าเริ่มต้นของคุณสมบัติ ShowCategoryName สำหรับป้ายข้อมูลใหม่ในคอลเลกชัน DataLabelCollection. การกำหนดค่าคุณสมบัตินี้ด้วยค่าใดค่าหนึ่งยังกำหนดค่านั้นให้กับคุณสมบัติ ShowCategoryName ของป้ายข้อมูลทั้งหมดในคอลเลกชัน DataLabelCollection (เช่น "DataLabels.DefaultDataLabelFormat.ShowCategoryName = val;" ทำให้ DataLabels[i].ShowCategoryName มีค่าเท่ากับ val).

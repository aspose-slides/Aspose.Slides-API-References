## IDataLabelFormat.ShowPercentage คุณสมบัติ

อธิบายพฤติกรรมการแสดงค่าร้อยละของป้ายข้อมูลของแผนภูมิที่ระบุ. True แสดงค่าร้อยละ. False ซ่อน. อ่าน/เขียน Boolean.

```csharp
public bool ShowPercentage { get; set; }
```

### หมายเหตุ

หากพาเรนต์ของวัตถุ DataLabelFormat นี้เป็นคอลเลกชัน DataLabelCollection ของป้ายข้อมูลแล้วคุณสมบัตินี้จะรับหรือกำหนดค่าเริ่มต้นของคุณสมบัติ ShowPercentage สำหรับป้ายข้อมูลใหม่ในคอลเลกชัน DataLabelCollection การตั้งค่าคุณสมบัตินี้ด้วยค่า จะกำหนดค่านี้ให้กับคุณสมบัติ ShowPercentage สำหรับทุกป้ายข้อมูลในคอลเลกชัน DataLabelCollection (i.e. "DataLabels.DefaultDataLabelFormat.ShowPercentage = val;" ทำให้ DataLabels[i].ShowPercentage มีค่าเท่ากับ val).

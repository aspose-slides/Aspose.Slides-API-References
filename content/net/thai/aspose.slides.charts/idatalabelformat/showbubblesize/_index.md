## IDataLabelFormat.ShowBubbleSize คุณสมบัติ

แทนค่าพฤติกรรมการแสดงผลขนาดฟองของป้ายข้อมูลในแผนภูมิที่ระบุ ค่า True จะทำการแสดงขนาดฟอง ค่า False จะซ่อน อ่าน/เขียน Boolean.

```csharp
public bool ShowBubbleSize { get; set; }
```

### หมายเหตุ

หากพาเรนต์ของวัตถุ DataLabelFormat นี้เป็นคอลเลกชัน DataLabelCollection ของป้ายข้อมูล จะทำให้คุณสมบัตินี้รับหรือกำหนดค่าตั้งต้นของคุณสมบัติ ShowBubbleSize สำหรับป้ายข้อมูลใหม่ในคอลเลกชัน DataLabelCollection การตั้งค่าคุณสมบัตินี้ด้วยค่าใดค่าหนึ่งก็จะกำหนดค่านั้นให้กับคุณสมบัติ ShowBubbleSize ของป้ายข้อมูลทั้งหมดในคอลเลกชัน DataLabelCollection (เช่น "DataLabels.DefaultDataLabelFormat.ShowBubbleSize = val;" ทำให้ DataLabels[i].ShowBubbleSize ทั้งหมดเท่ากับ val)

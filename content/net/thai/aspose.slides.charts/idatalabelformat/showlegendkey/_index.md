## IDataLabelFormat.ShowLegendKey สมบัติ

แสดงพฤติกรรมการแสดงคีย์คำอธิบายข้อมูลของแผนภูมิที่ระบุ. True หากคีย์คำอธิบายข้อมูลมองเห็นได้. อ่าน/เขียน Boolean.

```csharp
public bool ShowLegendKey { get; set; }
```

### หมายเหตุ

หากพาเรนต์ของวัตถุ DataLabelFormat นี้เป็นคอลเลกชัน DataLabelCollection ของป้ายข้อมูลแล้วสมบัตินี้จะดึงหรือกำหนดค่าตั้งต้นของสมบัติ ShowLegendKey สำหรับป้ายข้อมูลใหม่ในคอลเลกชัน DataLabelCollection. การกำหนดค่าสมบัตินี้ด้วยค่าใดค่าหนึ่งยังทำให้ค่าดังกล่าวถูกตั้งให้กับสมบัติ ShowLegendKey ของป้ายข้อมูลทั้งหมดในคอลเลกชัน DataLabelCollection (เช่น "DataLabels.DefaultDataLabelFormat.ShowLegendKey = val;" ทำให้ DataLabels[i].ShowLegendKey มีค่าเท่ากับ val).

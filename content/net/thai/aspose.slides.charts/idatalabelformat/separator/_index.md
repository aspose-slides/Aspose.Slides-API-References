## IDataLabelFormat.Separator คุณสมบัติ

ตั้งค่า หรือคืนค่า Variant ที่แสดงถึงตัวคั่นที่ใช้สำหรับป้ายข้อมูลบนแผนภูมิ อ่าน/เขียน String.

```csharp
public string Separator { get; set; }
```

### หมายเหตุ

หากพาเรนต์ของออบเจ็กต์ DataLabelFormat นี้เป็นคอลเลกชัน DataLabelCollection ของป้ายข้อมูลแล้วคุณสมบัตินี้จะรับหรือกำหนดค่าเริ่มต้นของคุณสมบัติ Separator สำหรับป้ายข้อมูลใหม่ในคอลเลกชัน DataLabelCollection การตั้งค่าคุณสมบัตินี้ด้วยค่าใดค่าหนึ่งจะกำหนดค่าดังกล่าวให้กับคุณสมบัติ Separator สำหรับป้ายข้อมูลทั้งหมดในคอลเลกชัน DataLabelCollection (เช่น "DataLabels.DefaultDataLabelFormat.Separator = val;" ทำให้ DataLabels[i].Separator มีค่าเท่ากับ val).

## IDataLabelFormat.ShowSeriesName คุณสมบัติ

ส่งคืนหรือกำหนดค่า Boolean เพื่อระบุพฤติกรรมการแสดงชื่อซีรีส์สำหรับป้ายข้อมูลบนแผนภูมิ ค่า true เพื่อแสดงชื่อซีรีส์ ค่า false เพื่อซ่อน อ่าน/เขียน Boolean.

```csharp
public bool ShowSeriesName { get; set; }
```

### หมายเหตุ

หากพาเรนท์ของวัตถุ DataLabelFormat นี้เป็นคอลเลกชัน DataLabelCollection ของป้ายข้อมูลแล้วคุณสมบัตินี้จะรับหรือกำหนดค่าตั้งต้นของคุณสมบัติ ShowSeriesName สำหรับป้ายข้อมูลใหม่ในคอลเลกชัน DataLabelCollection การตั้งค่าคุณสมบัตินี้ด้วยค่าใดค่าหนึ่งจะกำหนดค่านี้ให้กับคุณสมบัติ ShowSeriesName ของป้ายข้อมูลทุกรายการในคอลเลกชัน DataLabelCollection (เช่น "DataLabels.DefaultDataLabelFormat.ShowSeriesName = val;" ทำให้ DataLabels[i].ShowSeriesName ทั้งหมดเท่ากับ val).

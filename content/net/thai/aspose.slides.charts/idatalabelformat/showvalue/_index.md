## IDataLabelFormat.ShowValue property

แสดงพฤติกรรมการแสดงค่าร้อยละของป้ายข้อมูลในแผนภูมิที่ระบุ. True แสดงค่าร้อยละ. False เพื่อซ่อน. อ่าน/เขียน Boolean.

```csharp
public bool ShowValue { get; set; }
```

### หมายเหตุ

หากพาเรนต์ของอ็อบเจ็กต์ DataLabelFormat นี้เป็นคอลเลกชัน DataLabelCollection ของป้ายข้อมูลแล้วคุณสมบัตินี้จะรับหรือกำหนดค่าปริยายของคุณสมบัติ ShowValue สำหรับป้ายข้อมูลใหม่ในคอลเลกชัน DataLabelCollection. การตั้งค่าคุณสมบัตินี้ด้วยค่าใดค่าหนึ่งจะตั้งค่าค่านั้นให้กับคุณสมบัติ ShowValue ของป้ายข้อมูลทั้งหมดในคอลเลกชัน DataLabelCollection (เช่น "DataLabels.DefaultDataLabelFormat.ShowValue = val;" ทำให้ DataLabels[i].ShowValue เท่ากับ val).

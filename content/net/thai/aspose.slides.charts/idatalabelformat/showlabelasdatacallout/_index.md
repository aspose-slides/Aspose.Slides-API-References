## IDataLabelFormat.ShowLabelAsDataCallout คุณสมบัติ

กำหนดว่าป้ายข้อมูลของแผนภูมิที่ระบุจะถูกแสดงเป็นการอ้างอิงข้อมูลหรือเป็นป้ายข้อมูล

ถ้าพาเรนท์ของอ็อบเจกต์ DataLabelFormat นี้เป็นคอลเลกชัน DataLabelCollection ของป้ายข้อมูลแล้วคุณสมบัตินี้จะรับหรือกำหนดค่าเริ่มต้นของคุณสมบัติ ShowLabelAsDataCallout สำหรับป้ายข้อมูลใหม่ในคอลเลกชัน DataLabelCollection ค่าที่ตั้งในคุณสมบัตินี้จะถูกตั้งให้กับคุณสมบัติ ShowLabelAsDataCallout ของป้ายข้อมูลทั้งหมดในคอลเลกชัน DataLabelCollection (เช่น "DataLabels.DefaultDataLabelFormat.ShowLabelAsDataCallout = val;" ทำให้ DataLabels[i].ShowLabelAsDataCallout ทั้งหมดมีค่าเท่ากับ val).

```csharp
public bool ShowLabelAsDataCallout { get; set; }
```

## IChartTextBlockFormat.RotationAngle คุณสมบัติ

ระบุการหมุนแบบกำหนดเองที่ถูกนำไปใช้กับข้อความภายในกล่องขอบเขต หากไม่ได้ระบุ การหมุนของรูปทรงที่แนบมาจะถูกใช้ หากระบุแล้ว จะถูกนำไปใช้แยกจากรูปทรง นั่นหมายความว่ารูปทรงสามารถมีการหมุนที่นำไปใช้เพิ่มเติมจากข้อความเองที่มีการหมุนถูกนำไปใช้ด้วย ค่าที่ได้จากการหมุนของข้อความเชิงภาพสรุปจากคุณสมบัตินี้และประเภทแนวตั้งที่กำหนดไว้ล่วงหน้าในคุณสมบัติ TextVerticalType. อ่าน/เขียน Single.

```csharp
public float RotationAngle { get; set; }
```

### ตัวอย่าง

พิจารณากรณีที่รูปทรงมีการหมุน 90 องศา ตามเข็มนาฬิกา นอกจากนี้ เนื้อความเองยังมีการหมุน -90 องศา ทวนเข็มนาฬิกา แล้วผลลัพธ์คือรูปทรงดูเหมือนถูกหมุน แต่ข้อความภายในดูเหมือนว่าไม่ได้ถูกหมุนเลย.

## IChartSeriesGroup.Overlap คุณสมบัติ

ระบุว่ากราฟแท่งและคอลัมน์ในแผนภูมิ 2-D จะซ้อนทับกันมากแค่ไหนเป็นเปอร์เซ็นต์ (จาก -100% ถึง 100%)  
- -100%: ระยะห่างสูงสุด (แถบแยกจากกันโดยสมบูรณ์)  
- 0%: แถวางเคียงข้างกันโดยไม่มีการซ้อนทับหรือระยะห่าง  
- 100%: การซ้อนทับสูงสุด (แถบซ้อนทับกันอย่างสมบูรณ์)  

คุณสมบัตินี้เป็นแบบอ่าน/เขียน SByte.

```csharp
public sbyte Overlap { get; set; }
```

### ข้อยกเว้น

| ข้อยกเว้น | เงื่อนไข |
| --- | --- |
| ArgumentOutOfRangeException | ถูกโยนเมื่อค่าถูกตั้งอยู่นอกช่วงที่ถูกต้องของ -100 ถึง 100 |

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีตั้งค่าการซ้อนทับสำหรับกลุ่มซีรีส์ของแผนภูมิและแสดงผลแผนภูมิที่ได้บนฟอร์ม:

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // กำหนดการซ้อนทับเป็น 55%

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

## ChartSeriesGroup.Overlap คุณสมบัติ

ระบุว่าความกว้างของแถบและคอลัมน์จะทับซ้อนกันเท่าใดในแผนภูมิ 2-D โดยเป็นเปอร์เซ็นต์ (จาก -100% ถึง 100%) - -100%: ระยะห่างสูงสุด (แถบแยกจากกันอย่างสมบูรณ์) - 0%: แถบวางเคียงกันโดยไม่มีการทับซ้อนหรือช่องว่าง - 100%: การทับซ้อนสูงสุด (แถบทับซ้อนกันอย่างสมบูรณ์) คุณสมบัตินี้เป็นการอ่าน/เขียน SByte.

```csharp
public sbyte Overlap { get; set; }
```

### ข้อยกเว้น

| ข้อยกเว้น | เงื่อนไข |
| --- | --- |
| ArgumentOutOfRangeException | โยนข้อยกเว้นเมื่อค่าถูกตั้งอยู่นอกช่วงที่ถูกต้องระหว่าง -100 ถึง 100. |

### ตัวอย่าง

ตัวอย่างต่อไปนี้แสดงวิธีตั้งค่าการทับซ้อนสำหรับกลุ่มซีรีส์ของแผนภูมิและเรนเดอร์แผนภูมิที่ได้บนฟอร์ม:

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // ตั้งค่า overlap เป็น 55%

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

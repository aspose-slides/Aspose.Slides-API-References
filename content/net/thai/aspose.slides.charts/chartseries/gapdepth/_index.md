## ChartSeries.GapDepth คุณสมบัติ

คืนค่า หรือกำหนดระยะทางเป็นเปอร์เซ็นต์ของความกว้างของมาร์คเกอร์ ระหว่างชุดข้อมูลในแผนภูมิ 3D. นี่คือคุณสมบัติไม่เพียงของชุดข้อมูลนี้เท่านั้น แต่ของชุดข้อมูลทั้งหมดในกลุ่มชุดข้อมูลแม่ — เป็นการสะท้อนของคุณสมบัติของกลุ่มที่เหมาะสม. ดังนั้นคุณสมบัตินี้เป็นอ่านอย่างเดียว. ใช้คุณสมบัติ ParentSeriesGroup เพื่อเข้าถึงกลุ่มชุดข้อมูลแม่. ใช้คุณสมบัติอ่าน/เขียน ParentSeriesGroup.GapDepth เพื่อเปลี่ยนค่า. อ่านอย่างเดียว Int32.

```csharp
public int GapDepth { get; }
```

## ChartSeries.TrendLines คุณสมบัติ

คอลเลกชันของเส้นแนวโน้มของซีรีส์. อ่านอย่างเดียว [`ITrendlineCollection`](../../itrendlinecollection).

```csharp
public ITrendlineCollection TrendLines { get; }
```

### หมายเหตุ

TrendLines มีอยู่ (ไม่เป็นค่า null) สำหรับชุดข้อมูลในแผนภูมิพื้นที่ 2-D ที่ไม่ได้ซ้อน, แถบ, คอลัมน์, เส้น, หุ้น, xy (กระจาย) และฟองอากาศ. เส้นแนวโน้มไม่สามารถใช้ได้กับชุดข้อมูลในประเภทแผนภูมิที่ซ้อนกันหรือแบบ 3-D. TrendLines ก็ไม่สามารถใช้ได้กับแผนภูมิเรดาร์, พาย, พื้นผิว หรือโดนัท.

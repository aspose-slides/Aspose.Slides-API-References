## ChartSeries.HasSeriesLines คุณสมบัติ

กำหนดว่ามีเส้นซีรีส์สำหรับซีรีส์นี้และซีรีส์ที่เกี่ยวข้องหรือไม่. นี่เป็นคุณสมบัติไม่เพียงของซีรีส์นี้เท่านั้น แต่ของทุกซีรีส์ในกลุ่มซีรีส์แม่ - นี่เป็นการฉายของคุณสมบัติของกลุ่มที่เหมาะสม. ดังนั้นคุณสมบัตินี้เป็นอ่านอย่างเดียว. ใช้คุณสมบัติ ParentSeriesGroup เพื่อเข้าถึงกลุ่มซีรีส์แม่. ใช้คุณสมบัติอ่าน/เขียน ParentSeriesGroup.HasSeriesLines เพื่อเปลี่ยนค่า. ใช้คุณสมบัติ ParentSeriesGroup.SeriesLinesFormat เพื่อจัดรูปแบบเส้นซีรีส์. อ่านอย่างเดียว Boolean.

```csharp
public bool HasSeriesLines { get; }
```

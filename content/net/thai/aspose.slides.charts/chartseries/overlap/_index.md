## ChartSeries.Overlap คุณสมบัติ

ระบุว่าบาร์และคอลัมน์ทับซ้อนกันเท่าใดบนแผนภูมิ 2-D โดยเป็นเปอร์เซ็นต์ (จาก -100% ถึง 100%) คุณสมบัตินี้ไม่ใช่ของซีรีส์นี้เท่านั้น แต่เป็นของซีรีส์ทั้งหมดในกลุ่มซีรีส์แม่ เป็นการฉายของคุณสมบัติเชิงสัมพันธ์ในกลุ่มซีรีส์แม่ ดังนั้นคุณสมบัตินี้เป็นแบบอ่านอย่างเดียว เพื่อเปลี่ยนค่าให้ใช้คุณสมบัติ !:ParentSeriesGroup.Overlap แบบอ่าน/เขียน อ่านอย่างเดียว SByte.

```csharp
public sbyte Overlap { get; }
```

### หมายเหตุ

Overlap ระบุระดับของการทับซ้อนหรือการเว้นระยะระหว่างบาร์และคอลัมน์เป็นเปอร์เซ็นต์ของความกว้างของพวกมัน:
- -100%: ระยะห่างสูงสุด (แท่งแยกออกจากกันอย่างสมบูรณ์)
- 0%: แท่งถูกจัดวางเคียงข้างกันโดยไม่มีการทับซ้อนหรือระยะห่าง
- 100%: การทับซ้อนสูงสุด (แท่งทับซ้อนกันอย่างสมบูรณ์) นี้เป็นการฉายของคุณสมบัติ !:ParentSeriesGroup.Overlap.

## DataLabel.IsVisible คุณสมบัติ

False หมายถึงป้ายข้อมูลไม่แสดง (และทั้งหมดของ Show*-flags (ShowValue, ...) จะเป็น false). อ่านอย่างเดียว Boolean.

```csharp
public bool IsVisible { get; }
```

### หมายเหตุ

หากป้ายข้อมูลแสดงอยู่ คุณสามารถซ่อนมันด้วยเมธอด Hide(). แต่ถ้าป้ายข้อมูลไม่แสดง (IsVisible เป็น false) คุณสามารถทำให้ป้ายข้อมูลแสดงได้โดยตั้งค่า Show*-flags (ShowValue, ...) ให้เป็นสถานะ true.

## IChartData.UseSecondaryCategories คุณสมบัติ

ถ้าเป็น false แล้ว [`SecondaryCategories`](../secondarycategories) คุณสมบัติ คืนค่า null และข้อมูลใน [`Categories`](../categories) คุณสมบัติ ถูกใช้ทั้งสำหรับซีรีส์หลักและซีรีส์รอง. ถ้าเป็น true แล้วข้อมูลใน [`SecondaryCategories`](../secondarycategories) คุณสมบัติ ถูกใช้สำหรับซีรีส์รองและข้อมูลใน [`Categories`](../categories) คุณสมบัติ ถูกใช้สำหรับซีรีส์หลัก. อ่าน/เขียน Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

### ตัวอย่าง

ตัวอย่าง. หมวดหมู่ใดที่สัมพันธ์กับซีรีส์ - ChartData.Categories หรือ ChartData.SecondaryCategories?

if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)

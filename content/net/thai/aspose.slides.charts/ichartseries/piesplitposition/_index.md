## IChartSeries.PieSplitPosition คุณสมบัติ

กำหนดค่าที่จะใช้เพื่อระบุว่าจุดข้อมูลใดอยู่ในพายหรือแถบที่สองของแผนภูมิโปรด-ของ-พายหรือแผนภูมิเส้น-ของ-พาย ใช้ร่วมกับคุณสมบัติ PieSplitBy นี้เป็นคุณสมบัติไม่เพียงของซีรีส์นี้ แต่ของทุกซีรีส์ในกลุ่มซีรีส์แม่ - เป็นการฉายของคุณสมบัติของกลุ่มที่เกี่ยวข้อง ดังนั้นคุณสมบัตินี้เป็นอ่านอย่างเดียว ใช้คุณสมบัติ ParentSeriesGroup เพื่อเข้าถึงกลุ่มซีรีส์แม่ ใช้คุณสมบัติ ParentSeriesGroup.PieSplitPosition คุณสมบัติอ่าน/เขียนเพื่อเปลี่ยนค่า อ่านอย่างเดียว Double.

```csharp
public double PieSplitPosition { get; }
```

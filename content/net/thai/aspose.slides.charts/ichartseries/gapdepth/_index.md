## IChartSeries.GapDepth คุณสมบัติ

คืนค่า หรือกำหนดระยะห่างเป็นเปอร์เซ็นต์ของความกว้างของมาร์คเกอร์ ระหว่างชุดข้อมูลในแผนภูมิ 3 มิติ. นี่เป็นคุณสมบัติของไม่เพียงชุดข้อมูลนี้เท่านั้น แต่ยังรวมถึงชุดข้อมูลทั้งหมดของกลุ่มชุดข้อมูลแม่ด้วย – เป็นการฉายของคุณสมบัติของกลุ่มที่เหมาะสม. ดังนั้นคุณสมบัตินี้เป็นแบบอ่านอย่างเดียว. ใช้คุณสมบัติ ParentSeriesGroup เพื่อเข้าถึงกลุ่มชุดข้อมูลแม่. ใช้คุณสมบัติ ParentSeriesGroup.GapDepth แบบอ่าน/เขียน เพื่อเปลี่ยนค่า. อ่านอย่างเดียว Int32.

```csharp
public int GapDepth { get; }
```

## IChartSeries.ErrorBarsXFormat คุณสมบัติ

แสดง ErrorBars ของซีรีส์ที่มีทิศทาง X. ErrorBars ที่มีทิศทาง X มีให้ใช้สำหรับซีรีส์ประเภท area, bar, scatter และ bubble. สำหรับประเภทแผนภูมิอื่น ๆ คุณสมบัตินี้จะคืนค่า null (รวมถึงแผนภูมิดูมิติ 3D). ในกรณีที่ต้องการค่าที่กำหนดเอง ให้ใช้คอลเลกชัน DataPoints เพื่อระบุค่า (พร้อมคุณสมบัติ [`ErrorBarsCustomValues`](../../ichartdatapoint/errorbarscustomvalues)). อ่านอย่างเดียว [`IErrorBarsFormat`](../../ierrorbarsformat).

public IErrorBarsFormat ErrorBarsXFormat { get; }

## IChartSeries.PieSplitBy คุณสมบัติ

ระบุวิธีการกำหนดว่าจุดข้อมูลใดอยู่ในพายหรือแถบที่สองบนแผนภูมิ pie-of-pie หรือ bar-of-pie นี่เป็นคุณสมบัติไม่เพียงของชุดข้อมูลนี้แต่ของทุกชุดข้อมูลในกลุ่มชุดข้อมูลพาเรนต์ — เป็นการฉายของคุณสมบัติของกลุ่มที่เหมาะสม ดังนั้นคุณสมบัตินี้เป็นแบบอ่านอย่างเดียว ใช้คุณสมบัติ ParentSeriesGroup เพื่อเข้าถึงกลุ่มชุดข้อมูลพาเรนต์ ใช้คุณสมบัติ ParentSeriesGroup.PieSplitBy แบบอ่าน/เขียนเพื่อเปลี่ยนค่า อ่านอย่างเดียว [`PieSplitType`](../../piesplittype).

```csharp
public PieSplitType PieSplitBy { get; }
```

### หมายเหตุ

1) นี่เป็นการฉายของคุณสมบัติ ParentSeriesGroup.PieSplitBy. 2) หากค่าคุณสมบัติคือ PieSplitType.Custom คุณสามารถกำหนดข้อมูลการแยกแบบกำหนดเองด้วยคุณสมบัติ ParentSeriesGroup.PieSplitCustomPoints.

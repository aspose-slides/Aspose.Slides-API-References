## IChartSeries.IsColorVaried คุณสมบัติ

ระบุว่าตัวทำเครื่องหมายข้อมูลแต่ละจุดในชุดข้อมูลมีสีที่แตกต่างกัน. นี้เป็นคุณสมบัติไม่เพียงของชุดข้อมูลนี้เท่านั้นแต่ของทุกชุดข้อมูลในกลุ่มชุดข้อมูลแม่ - นี้เป็นการฉายของคุณสมบัติของกลุ่มที่เหมาะสม. ดังนั้นคุณสมบัตินี้เป็นแบบอ่านอย่างเดียว. ใช้คุณสมบัติ ParentSeriesGroup เพื่อเข้าถึงกลุ่มชุดข้อมูลแม่. ใช้คุณสมบัติ ParentSeriesGroup.IsColorVaried อ่าน/เขียน เพื่อเปลี่ยนค่า. อ่านอย่างเดียว Boolean.

```csharp
public bool IsColorVaried { get; }
```

### หมายเหตุ

นี้เป็นการฉายของคุณสมบัติ ParentSeriesGroup.IsColorVaried.

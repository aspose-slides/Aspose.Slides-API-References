## IChartSeries.Overlap คุณสมบัติ

ระบุว่าความซ้อนทับของแท่งและคอลัมน์ในแผนภูมิ 2 มิติเป็นเท่าใดในรูปของเปอร์เซ็นต์ (ตั้งแต่ -100% ถึง 100%). นี่เป็นคุณสมบัติไม่เพียงของชุดข้อมูลนี้เท่านั้น แต่ของชุดข้อมูลทั้งหมดในกลุ่มชุดข้อมูลพาเรนต์. มันเป็นการฉายของคุณสมบัติที่สอดคล้องในกลุ่มชุดข้อมูลพาเรนต์ ดังนั้นคุณสมบัตินี้จึงเป็นแบบอ่านอย่างเดียว. หากต้องการเปลี่ยนค่า ให้ใช้คุณสมบัติอ่าน/เขียน ParentSeriesGroup.Overlap. อ่านอย่างเดียว SByte.

```csharp
public sbyte Overlap { get; }
```

### Remarks

Overlap ระบุระดับของการซ้อนทับหรือช่องว่างระหว่างแท่งและคอลัมน์เป็นเปอร์เซ็นต์ของความกว้างของพวกมัน: - -100%: ช่องว่างสูงสุด (แท่งแยกจากกันอย่างสมบูรณ์) - 0%: แท่งถูกจัดวางเคียงข้างกันโดยไม่มีการซ้อนทับหรือช่องว่าง - 100%: การซ้อนทับสูงสุด (แท่งซ้อนกันอย่างสมบูรณ์) นี่เป็นการฉายของคุณสมบัติ ParentSeriesGroup.Overlap.

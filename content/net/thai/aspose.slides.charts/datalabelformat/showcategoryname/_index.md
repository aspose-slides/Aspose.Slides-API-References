## DataLabelFormat.ShowCategoryName คุณสมบัติ

แทนพฤติกรรมการแสดงชื่อหมวดของป้ายข้อมูลในแผนภูมิที่ระบุ. True เพื่อแสดงชื่อหมวดสำหรับป้ายข้อมูลบนแผนภูมิ. False เพื่อซ่อน. อ่าน/เขียน Boolean.

```csharp
public bool ShowCategoryName { get; set; }
```

### หมายเหตุ

หากพาเรนต์ของอ็อบเจ็กต์ DataLabelFormat นี้เป็นคอลเลกชัน DataLabelCollection ของป้ายข้อมูลแล้ว คุณสมบัตินี้จะรับหรือกำหนดค่าตั้งต้นของคุณสมบัติ ShowCategoryName สำหรับป้ายข้อมูลใหม่ในคอลเลกชัน DataLabelCollection. การตั้งค่าคุณสมบัตินี้ด้วยค่าใดค่าหนึ่งจะกำหนดค่านั้นให้กับคุณสมบัติ ShowCategoryName ของป้ายข้อมูลทั้งหมดในคอลเลกชัน DataLabelCollection (เช่น "DataLabels.DefaultDataLabelFormat.ShowCategoryName = val;" ทำให้ DataLabels[i].ShowCategoryName มีค่าเท่ากับ val).

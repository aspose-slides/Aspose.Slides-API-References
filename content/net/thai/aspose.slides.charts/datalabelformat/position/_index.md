## DataLabelFormat.Position คุณสมบัติ

แสดงตำแหน่งของป้ายข้อมูล. อ่าน/เขียน [`LegendDataLabelPosition`](../../legenddatalabelposition).

```csharp
public LegendDataLabelPosition Position { get; set; }
```

### หมายเหตุ

หากพาเรนต์ของวัตถุ DataLabelFormat นี้เป็นคอลเลกชัน DataLabelCollection ของป้ายข้อมูลแล้วคุณสมบัตินี้จะได้หรือกำหนดค่าตั้งต้นของคุณสมบัติ Position สำหรับป้ายข้อมูลใหม่ในคอลเลกชัน DataLabelCollection. แสดงตำแหน่งสำหรับอ็อบเจ็กต์ DataLabel. การตั้งค่าคุณสมบัตินี้ด้วยค่า จะตั้งค่าดังกล่าวให้กับคุณสมบัติ Position สำหรับป้ายข้อมูลทั้งหมดในคอลเลกชัน DataLabelCollection (เช่น "DataLabels.DefaultDataLabelFormat.Position = val;" ทำให้ DataLabels[i].Position ทั้งหมดเท่ากับค่า val).

## DataLabelFormat.ShowPercentage คุณสมบัติ

แสดงพฤติกรรมการแสดงค่าร้อยละของป้ายข้อมูลในแผนภูมิที่ระบุ. True แสดงค่าร้อยละ. False เพื่อซ่อน. อ่าน/เขียน Boolean.

```csharp
public bool ShowPercentage { get; set; }
```

### หมายเหตุ

หากพาเรนท์ของอ็อบเจกต์ DataLabelFormat นี้เป็นคอลเลกชัน DataLabelCollection ของป้ายข้อมูลแล้วคุณสมบัตินี้จะรับหรือกำหนดค่าปริยายของคุณสมบัติ ShowPercentage สำหรับป้ายข้อมูลใหม่ในคอลเลกชัน DataLabelCollection. การตั้งค่าคุณสมบัตินี้ด้วยค่าใดค่าหนึ่งจะกำหนดค่านี้ให้กับคุณสมบัติ ShowPercentage ของป้ายข้อมูลทั้งหมดในคอลเลกชัน DataLabelCollection (เช่น "DataLabels.DefaultDataLabelFormat.ShowPercentage = val;" ทำให้ DataLabels[i].ShowPercentage มีค่าเท่ากับ val).

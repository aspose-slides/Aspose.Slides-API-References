## DataLabelFormat.ShowSeriesName คุณสมบัติ

คืนค่า หรือ กำหนดค่า Boolean เพื่อบ่งชี้พฤติกรรมการแสดงชื่อซีรีส์สำหรับป้ายข้อมูลบนแผนภูมิ True เพื่อแสดงชื่อซีรีส์ False เพื่อซ่อน อ่าน/เขียน Boolean.

```csharp
public bool ShowSeriesName { get; set; }
```

### หมายเหตุ

หากผู้ปกครองของอ็อบเจ็กต์ DataLabelFormat นี้เป็นคอลเลกชัน DataLabelCollection ของป้ายข้อมูลแล้ว property นี้จะรับหรือกำหนดค่าพื้นฐานของ property ShowSeriesName สำหรับป้ายข้อมูลใหม่ในคอลเลกชัน DataLabelCollection การตั้งค่า property นี้ด้วยค่าจะทำให้ค่าดังกล่าวถูกกำหนดให้กับ property ShowSeriesName สำหรับป้ายข้อมูลทั้งหมดในคอลเลกชัน DataLabelCollection อีกด้วย (เช่น "DataLabels.DefaultDataLabelFormat.ShowSeriesName = val;" ทำให้ทุก DataLabels[i].ShowSeriesName มีค่าเท่ากับ val).

## DataLabelFormat.ShowLabelAsDataCallout คุณสมบัติ

กำหนดว่าป้ายข้อมูลของแผนภูมิที่ระบุจะถูกแสดงเป็นการชี้เรียกข้อมูลหรือเป็นป้ายข้อมูล หากพาเรนต์ของวัตถุ DataLabelFormat นี้เป็นคอลเลกชัน DataLabelCollection ของป้ายข้อมูลแล้วคุณสมบัตินี้จะรับหรือกำหนดค่าปริยายของคุณสมบัติ ShowLabelAsDataCallout สำหรับป้ายข้อมูลใหม่ในคอลเลกชัน DataLabelCollection การตั้งค่าคุณสมบัตินี้ด้วยค่าจะตั้งค่าดังกล่าวให้กับคุณสมบัติ ShowLabelAsDataCallout สำหรับป้ายข้อมูลทั้งหมดในคอลเลกชัน DataLabelCollection (เช่น "DataLabels.DefaultDataLabelFormat.ShowLabelAsDataCallout = val;" ทำให้ทุก DataLabels[i].ShowLabelAsDataCallout มีค่าเท่ากับ val).

```csharp
public bool ShowLabelAsDataCallout { get; set; }
```

## DataLabelFormat.ShowValue คุณสมบัติ

แสดงพฤติกรรมการแสดงค่าร้อยละของป้ายข้อมูลในแผนภูมิที่ระบุ ค่าจริงจะแสดงค่าร้อยละ ค่าผิดจะซ่อนค่า อ่าน/เขียน Boolean.

```csharp
public bool ShowValue { get; set; }
```

### หมายเหตุ

หากพาเรนต์ของวัตถุ DataLabelFormat นี้เป็นคอลเลกชัน DataLabelCollection ของป้ายข้อมูลแล้วคุณสมบัตินี้จะรับหรือกำหนดค่าเริ่มต้นของคุณสมบัติ ShowValue สำหรับป้ายข้อมูลใหม่ในคอลเลกชัน DataLabelCollection การตั้งค่าคุณสมบัตินี้พร้อมค่าจะทำให้ค่านี้ถูกตั้งค่าให้กับคุณสมบัติ ShowValue สำหรับป้ายข้อมูลทั้งหมดในคอลเลกชัน DataLabelCollection (เช่น "DataLabels.DefaultDataLabelFormat.ShowValue = val;" ทำให้ DataLabels[i].ShowValue มีค่าเท่ากับ val).

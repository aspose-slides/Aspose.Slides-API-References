## DataLabelFormat.ShowLegendKey คุณสมบัติ

แสดงพฤติกรรมการแสดงคีย์คำอธิบายข้อมูลของแผนภูมิที่ระบุไว้เป็นจริงหากคีย์คำอธิบายข้อมูลมองเห็นได้ อ่าน/เขียน Boolean.

```csharp
public bool ShowLegendKey { get; set; }
```

### หมายเหตุ

หากพาเรนท์ของออบเจกต์ DataLabelFormat นี้เป็นคอลเลกชัน DataLabelCollection ของป้ายกำกับข้อมูลแล้วคุณสมบัตินี้จะรับหรือกำหนดค่าปริยายของคุณสมบัติ ShowLegendKey สำหรับป้ายกำกับข้อมูลใหม่ในคอลเลกชัน DataLabelCollection การตั้งค่าคุณสมบัตินี้ด้วยค่าใดค่าหนึ่งจะตั้งค่าค่านั้นให้กับคุณสมบัติ ShowLegendKey ของป้ายกำกับข้อมูลทั้งหมดในคอลเลกชัน DataLabelCollection (เช่น "DataLabels.DefaultDataLabelFormat.ShowLegendKey = val;" ทำให้ DataLabels[i].ShowLegendKey มีค่าเท่ากับ val)

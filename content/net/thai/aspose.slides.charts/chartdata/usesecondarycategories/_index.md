## ChartData.UseSecondaryCategories คุณสมบัติ

หากเป็น false แล้ว [`SecondaryCategories`](../secondarycategories) คุณสมบัติจะคืนค่า null และข้อมูลใน [`Categories`](../categories) คุณสมบัติจะถูกใช้ทั้งสำหรับชุดหลักและชุดรอง หากเป็น true แล้วข้อมูลใน [`SecondaryCategories`](../secondarycategories) คุณสมบัติจะถูกใช้สำหรับชุดรองและข้อมูลใน [`Categories`](../categories) คุณสมบัติจะถูกใช้สำหรับชุดหลัก อ่าน/เขียน Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

### ตัวอย่าง

Example. หมวดหมู่ใดที่เกี่ยวข้องกับชุดข้อมูล - ChartData.Categories หรือ ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // หมวดหมูที่เกี่ยวข้องคือ series.Chart.ChartData.SecondaryCategories
}
else
{
    // หมวดหมูที่เกี่ยวข้องคือ series.Chart.ChartData.Categories
}
```

## ChartData.Categories คุณสมบัติ

รับประเภทหลัก (หรือทั้งประเภทหลักและรองหาก [`UseSecondaryCategories`](../usesecondarycategories) คุณสมบัติเป็น false). อ่านอย่างเดียว [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection Categories { get; }
```

### หมายเหตุ

หาก [`UseSecondaryCategories`](../usesecondarycategories) คุณสมบัติเป็น false แล้ว [`SecondaryCategories`](../secondarycategories) คุณสมบัติจะคืนค่า null และข้อมูลในคุณสมบัตินี้ `Categories` จะถูกใช้ทั้งสำหรับชุดข้อมูลหลักและรอง. หาก [`UseSecondaryCategories`](../usesecondarycategories) คุณสมบัติเป็น true แล้วข้อมูลใน [`SecondaryCategories`](../secondarycategories) คุณสมบัติจะถูกใช้สำหรับชุดข้อมูลรองและข้อมูลในคุณสมบัตินี้ `Categories` จะถูกใช้สำหรับชุดข้อมูลหลัก.

### ตัวอย่าง

ตัวอย่าง. ประเภทใดที่เกี่ยวข้องกับชุดข้อมูล - ChartData.Categories หรือ ChartData.SecondaryCategories?

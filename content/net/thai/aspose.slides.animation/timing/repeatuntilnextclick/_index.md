## Timing.RepeatUntilNextClick คุณสมบัติ

คุณลักษณะนี้ระบุว่าเอฟเฟกต์จะทำซ้ำจนกว่าจะคลิกครั้งต่อไปหรือไม่. อ่าน/เขียน Boolean.

```csharp
public bool RepeatUntilNextClick { get; set; }
```

### ตัวอย่าง

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // รับลำดับของเอฟเฟกต์สำหรับสไลด์แรก
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // รับเอฟเฟกต์แรกของลำดับหลัก.
    IEffect effect = effectsSequence[0];

    // เปลี่ยน Timing/Repeat ของเอฟเฟกต์เป็น "Until Next Click"
    effect.Timing.RepeatUntilNextClick = true;
}
```

## Timing.Rewind คุณสมบัติ

แอตทริบิวต์นี้ระบุว่าผลกระทบจะรีวินด์เมื่อเล่นเสร็จแล้วหรือไม่. อ่าน/เขียน Boolean.

```csharp
public bool Rewind { get; set; }
```

### ตัวอย่าง

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // รับลำดับเอฟเฟกต์สำหรับสไลด์แรก
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // รับเอฟเฟกต์แรกของลำดับหลัก.
    IEffect effect = effectsSequence[0];

    // เปิดการทำงานของ Timing/Rewind ของเอฟเฟกต์.
    effect.Timing.Rewind = true;
}
```

## IEffect.AnimateTextType คุณสมบัติ

กำหนดประเภทการเคลื่อนที่ของข้อความสำหรับเอฟเฟกต์ ข้อความของรูปร่างสามารถเคลื่อนที่ตามตัวอักษร, ตามคำ หรือทั้งหมดพร้อมกัน อ่าน/เขียน `AnimateTextType`.

```csharp
public AnimateTextType AnimateTextType { get; set; }
```

### ตัวอย่าง

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // รับเอฟเฟกต์แรกของสไลด์แรก.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // เปลี่ยนประเภทการเคลื่อนที่ของข้อความเอฟเฟกต์เป็น "By letter"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByLetter;
}
```

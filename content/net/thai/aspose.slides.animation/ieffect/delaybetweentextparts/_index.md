## IEffect.DelayBetweenTextParts คุณสมบัติ

กำหนดการหน่วงเวลาระหว่างส่วนของข้อความที่ทำงานแบบเคลื่อนที่ (คำหรืออักษร) ค่าบวกระบุเปอร์เซ็นต์ของระยะเวลาผลกระทบ ค่าลบระบุการหน่วงเวลาเป็นวินาที อ่าน/เขียน Single.

```csharp
public float DelayBetweenTextParts { get; set; }
```

### ตัวอย่าง

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // รับเอฟเฟกต์แรกของสไลด์แรก.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // เปลี่ยนประเภทของเอฟเฟกต์ Animate text ให้เป็น "By word"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByWord;
    
    // ตั้งค่าการหน่วงเวลาระหว่างส่วนของข้อความที่ทำแอนิเมชันเป็น 20% ของระยะเวลาเอฟเฟกต์.
    firstSlideEffect.DelayBetweenTextParts = 20f;
}
```

## ITiming.RepeatUntilNextClick คุณสมบัติ

คุณลักษณะนี้ระบุว่าผลกระทบจะทำซ้ำจนกว่าจะมีการคลิกครั้งถัดไป. อ่าน/เขียน Boolean.

```csharp
public bool RepeatUntilNextClick { get; set; }
```

### ตัวอย่าง

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // ดึงลำดับเอฟเฟ็กต์สำหรับสไลด์แรก
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // ดึงเอฟเฟ็กต์แรกของลำดับหลัก.
    IEffect effect = effectsSequence[0];

    // เปลี่ยน Timing/Repeat ของเอฟเฟ็กต์เป็น "Until Next Click"
    effect.Timing.RepeatUntilNextClick = true;
}
```

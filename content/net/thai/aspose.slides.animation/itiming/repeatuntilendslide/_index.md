## ITiming.RepeatUntilEndSlide คุณสมบัติ

แอตทริบิวต์นี้ระบุว่าผลกระทบจะทำซ้ำจนถึงจุดสิ้นสุดของสไลด์หรือไม่. อ่าน/เขียน Boolean.

```csharp
public bool RepeatUntilEndSlide { get; set; }
```

### ตัวอย่าง

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // ดึงลำดับเอฟเฟกต์สำหรับสไลด์แรก
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // ดึงเอฟเฟกต์แรกของลำดับหลัก.
    IEffect effect = effectsSequence[0];

    // เปลี่ยน Timing/Repeat ของเอฟเฟกต์เป็น "Until End of Slide"
    effect.Timing.RepeatUntilEndSlide = true;
}
```

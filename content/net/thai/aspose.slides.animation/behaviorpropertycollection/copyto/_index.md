## BehaviorPropertyCollection.CopyTo เมธอด

คัดลอกองค์ประกอบของ ICollection ไปยัง Array โดยเริ่มที่ตำแหน่งดัชนีของ Array ที่ระบุ

```csharp
public void CopyTo(IBehaviorProperty[] array, int arrayIndex)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| array | IBehaviorProperty[] | Array มิติเดียวที่เป็นตำแหน่งปลายทางขององค์ประกอบที่คัดลอกจาก ICollection. Array ต้องมีการจัดทำดัชนีเริ่มจากศูนย์. |
| arrayIndex | Int32 | ดัชนีเริ่มจากศูนย์ใน *array* ที่การคัดลอกรเริ่มต้น. |

### ข้อยกเว้น

| ข้อยกเว้น | เงื่อนไข |
| --- | --- |
| ArgumentNullException | *array* มีค่าเป็น null. |
| ArgumentOutOfRangeException | *arrayIndex* มีค่าน้อยกว่า 0. |
| ArgumentException | จำนวนขององค์ประกอบใน ICollection แหล่งที่มามากกว่าพื้นที่ที่มีอยู่จาก *arrayIndex* ถึงจุดสิ้นสุดของ *array* ปลายทาง. |

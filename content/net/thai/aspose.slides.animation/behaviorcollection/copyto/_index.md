## BehaviorCollection.CopyTo เมธอด

คัดลอกองค์ประกอบของ ICollection ไปยัง Array โดยเริ่มที่ดัชนี Array เฉพาะ

```csharp
public void CopyTo(IBehavior[] array, int arrayIndex)
```

| พารามิเตอร์ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| array | IBehavior[] | Array แบบหนึ่งมิติที่เป็นจุดหมายขององค์ประกอบที่คัดลอกจาก ICollection. Array ต้องมีการจัดทำดัชนีเริ่มที่ศูนย์ |
| arrayIndex | Int32 | ดัชนีแบบศูนย์เริ่มต้นใน *array* ที่การคัดลอกเริ่มต้น |

### ข้อยกเว้น

| ข้อยกเว้น | เงื่อนไข |
| --- | --- |
| ArgumentNullException | *array* เป็น null |
| ArgumentOutOfRangeException | *arrayIndex* น้อยกว่า 0 |
| ArgumentException | จำนวนขององค์ประกอบใน ICollection แหล่งที่มามีค่ามากกว่าพื้นที่ที่ว่างจาก *arrayIndex* ถึงจบของ *array* ปลายทาง |

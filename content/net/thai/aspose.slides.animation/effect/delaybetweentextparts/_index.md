## Effect.DelayBetweenTextParts คุณสมบัติ

กำหนดการหน่วงระหว่างส่วนข้อความที่ทำให้เคลื่อนไหว (คำหรืออักษร) ค่าบวกระบุเปอร์เซ็นต์ของระยะเวลาของเอฟเฟกต์ ค่าลบระบุการหน่วงเป็นวินาที อ่าน/เขียน Single.

```csharp
public float DelayBetweenTextParts { get; set; }
```

### ตัวอย่าง

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // รับเอฟเฟกต์แรกของสไลด์แรก.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // เปลี่ยนประเภทการแอนิเมตข้อความของเอฟเฟกต์เป็น "By word"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByWord;
    
    // ตั้งค่าการหน่วงระหว่างส่วนข้อความที่เคลื่อนไหวเป็น 20% ของระยะเวลาเอฟเฟกต์.
    firstSlideEffect.DelayBetweenTextParts = 20f;
}
```

## BehaviorCollection.CopyTo 方法

将ICollection的元素复制到数组中，从特定数组索引开始。

```csharp
public void CopyTo(IBehavior[] array, int arrayIndex)
```

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| array | IBehavior[] | 一维数组，是从ICollection复制的元素的目标。数组必须具有零基索引。 |

## IDataLabelFormat.ShowBubbleSize 属性

表示指定图表的数据标签气泡大小值的显示行为。True 显示气泡大小值。False 隐藏。读/写布尔值。

```csharp
public bool ShowBubbleSize { get; set; }
```

### 备注

如果此 DataLabelFormat 对象的父级是数据标签集合 DataLabelCollection，则此属性获取或设置 DataLabelCollection 中新数据标签的 ShowBubbleSize 属性的默认值。设置此属性的值也会将此值设置为 DataLabelCollection 中所有数据标签的 ShowBubbleSize 属性（即 "DataLabels.DefaultDataLabelFormat.ShowBubbleSize = val;" 导致所有 DataLabels[i].ShowBubbleSize 等于 val）。

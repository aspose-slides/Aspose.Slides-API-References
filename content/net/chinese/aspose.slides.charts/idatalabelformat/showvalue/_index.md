## IDataLabelFormat.ShowValue 属性

表示指定图表的数据标签百分比值显示行为。True 显示百分比值。False 隐藏。可读/可写布尔值。

```csharp
public bool ShowValue { get; set; }
```

### 注释

如果此 DataLabelFormat 对象的父级是数据标签的 DataLabelCollection 集合，则此属性用于获取或设置 DataLabelCollection 集合中新数据标签的 ShowValue 属性的默认值。设置该属性的值也会将此值设置为 DataLabelCollection 集合中所有数据标签的 ShowValue 属性（即 "DataLabels.DefaultDataLabelFormat.ShowValue = val;" 会导致所有 DataLabels[i].ShowValue 等于 val）。

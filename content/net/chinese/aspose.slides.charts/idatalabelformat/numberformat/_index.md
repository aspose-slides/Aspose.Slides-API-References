## IDataLabelFormat.NumberFormat 属性

表示 DataLabels 对象的格式字符串。可读写字符串。

```csharp
public string NumberFormat { get; set; }
```

### 备注

如果这个 DataLabelFormat 对象的父对象是一个 DataLabelCollection 数据标签集合，则该属性获取或设置 DataLabelCollection 集合中新数据标签的 NumberFormat 属性的默认值。当该属性被设置为一个值时，该值也会被设置为 DataLabelCollection 集合中所有数据标签的 NumberFormat 属性（即 "DataLabels.DefaultDataLabelFormat.NumberFormat = val;" 会导致所有 DataLabels[i].NumberFormat 等于 val）。

### 示例

```csharp
[C#]
series.Labels.DefaultDataLabelFormat.ShowValue = true;
series.Labels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = false;
series.Labels.DefaultDataLabelFormat.NumberFormat = "0.0%";
```

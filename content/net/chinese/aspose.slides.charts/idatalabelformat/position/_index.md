## IDataLabelFormat.Position 属性

表示数据标签的位置。可读写 [`LegendDataLabelPosition`](../../legenddatalabelposition)。

```csharp
public LegendDataLabelPosition Position { get; set; }
```

### 备注

如果此 DataLabelFormat 对象的父对象是数据标签的 DataLabelCollection 集合，则此属性获取或设置 DataLabelCollection 集合中新数据标签的 Position 属性的默认值。表示 DataLabel 对象的位置。设置此属性时，值也会设置为 DataLabelCollection 集合中所有数据标签的 Position 属性（即 "DataLabels.DefaultDataLabelFormat.Position = val;" 使得所有 DataLabels[i].Position 等于 val）。

## IDataLabelFormat.IsNumberFormatLinkedToSource 属性

读/写布尔值。

```csharp
public bool IsNumberFormatLinkedToSource { get; set; }
```

### 备注

如果此 DataLabelFormat 对象的父级是一个 DataLabelCollection 数据标签集合，则此属性用于获取或设置 DataLabelCollection 中新数据标签的 IsNumberFormatLinkedToSource 属性的默认值。设置此属性的值也会将此值设置为 DataLabelCollection 中所有数据标签的 IsNumberFormatLinkedToSource 属性（即 "DataLabels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = val;" 导致所有 DataLabels[i].IsNumberFormatLinkedToSource 等于 val）。

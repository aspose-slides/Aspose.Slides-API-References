## ChartData.SecondaryCategories 属性

如果 [`UseSecondaryCategories`](../usesecondarycategories) 属性为 true，则获取次级类别。只读 [`IChartCategoryCollection`](../../ichartcategorycollection)。

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### 备注

如果 [`UseSecondaryCategories`](../usesecondarycategories) 属性为 false， 则 `SecondaryCategories` 属性返回 null，并且 [`Categories`](../categories) 属性中的数据同时用于主要和次级系列。如果 [`UseSecondaryCategories`](../usesecondarycategories) 属性为 true，则 `SecondaryCategories` 属性中的数据用于次级系列，而 [`Categories`](../categories) 属性中的数据用于主要系列。

### 示例

示例。与系列相关的类别是 - ChartData.Categories 还是 ChartData.SecondaryCategories？

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // 相关类别是 series.Chart.ChartData.SecondaryCategories
}
else
{
    // 相关类别是 series.Chart.ChartData.Categories
}
```

## ChartData.Categories property

获取主要类别（如果 [`UseSecondaryCategories`](../usesecondarycategories) 属性为 false，则同时获取主要和次要类别）。只读 [`IChartCategoryCollection`](../../ichartcategorycollection)。

```csharp
public IChartCategoryCollection Categories { get; }
```

### Remarks

如果 [`UseSecondaryCategories`](../usesecondarycategories) 属性为 false，则 [`SecondaryCategories`](../secondarycategories) 属性返回 null，并且此 `Categories` 属性中的数据同时用于主要和次要系列。如果 [`UseSecondaryCategories`](../usesecondarycategories) 属性为 true，则 [`SecondaryCategories`](../secondarycategories) 属性中的数据用于次要系列，而此 `Categories` 属性中的数据用于主要系列。

### Examples

示例。与系列相关的类别是 - ChartData.Categories 还是 ChartData.SecondaryCategories？

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // 相关类别是 series.Chart.ChartData.SecondaryCategories
}
else
{
    // 相关类别是 series.Chart.ChartData.Categories
}
```

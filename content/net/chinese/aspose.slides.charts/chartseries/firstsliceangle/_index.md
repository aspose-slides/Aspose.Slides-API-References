## ChartSeries.FirstSliceAngle property

指定第一个饼状图或甜甜圈图切片的角度，以度为单位（顺时针方向，从上方0到360度）。这是该系列的属性，不仅适用于该系列，还适用于父系列组的所有系列 - 这是适当组属性的投影。因此，此属性是只读的。使用 ParentSeriesGroup 属性访问父系列组。使用 ParentSeriesGroup.FirstSliceAngle 读/写属性更改值。只读 UInt16。

```csharp
public ushort FirstSliceAngle { get; }
```

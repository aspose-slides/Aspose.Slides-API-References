## ChartSeries.Overlap 属性

指定 2D 图表中条形和柱形图重叠的程度，以百分比表示（从 -100% 到 100%）。这不仅是此系列的属性，也是父系列组中所有系列的属性。它是父系列组中相应属性的投影，因此该属性是只读的。要更改值，请使用 !:ParentSeriesGroup.Overlap 读写属性。只读 SByte。

```csharp
public sbyte Overlap { get; }
```

### 备注

Overlap 指定条形和柱形图之间重叠或间距的程度，作为其宽度的百分比： - -100%：最大间隔（条形图完全分开）。 - 0%：条形图并排放置，没有重叠或间距。 - 100%：最大重叠（条形图完全重叠）。这是属性 !:ParentSeriesGroup.Overlap 的投影。

## ChartSeries.Overlap property

指定条形和柱状图在二维图表上重叠的百分比（从 -100% 到 100%）。这个属性不仅属于这一系列，还属于父系列组的所有系列。它是父系列组中相应属性的投影，因此这个属性是只读的。要更改值，请使用 !:ParentSeriesGroup.Overlap 读/写属性。只读 SByte。

```csharp
public sbyte Overlap { get; }
```

### Remarks

Overlap 指定条形和柱状图之间的重叠程度或间距，作为其宽度的百分比：- -100%：最大间距（条形完全分开）。- 0%：条形并排放置，没有重叠或间距。- 100%：最大重叠（条形完全重叠）。这是属性 !:ParentSeriesGroup.Overlap 的投影。

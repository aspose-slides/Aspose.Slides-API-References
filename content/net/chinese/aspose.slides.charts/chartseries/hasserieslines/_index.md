## ChartSeries.HasSeriesLines property

该属性用于确定此系列及相关系列是否存在系列线。此属性不仅适用于该系列，还适用于父系列组的所有系列——这是相应组属性的投影。因此，此属性是只读的。使用 ParentSeriesGroup 属性访问父系列组。使用 ParentSeriesGroup.HasSeriesLines 读/写属性更改值。使用 ParentSeriesGroup.SeriesLinesFormat 属性设置系列线的格式。只读布尔值。

```csharp
public bool HasSeriesLines { get; }
```

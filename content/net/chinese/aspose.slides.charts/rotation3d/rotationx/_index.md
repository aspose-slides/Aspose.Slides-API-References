## Rotation3D.RotationX 属性

返回或设置关于 X 轴的旋转角度，即在 Y 方向上的 3D 图表的旋转角度（范围为 -90 到 90 度）。该属性与 ECMA-376 中的 21.2.2.157 rotX（X 旋转）项以及 PowerPoint 2007+ 中的“Y 旋转”选项匹配。可读写 SByte。

```csharp
public sbyte RotationX { get; set; }
```

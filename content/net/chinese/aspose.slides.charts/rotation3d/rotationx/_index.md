## Rotation3D.RotationX 属性

返回或设置围绕X轴的旋转角度，即3D图表在Y方向上的旋转角度（范围为-90到90度）。该属性与ECMA-376中的21.2.2.157 rotX（X Rotation）项以及PowerPoint 2007+中的“Y Rotation”选项相匹配。可读/可写SByte。

```csharp
public sbyte RotationX { get; set; }
```

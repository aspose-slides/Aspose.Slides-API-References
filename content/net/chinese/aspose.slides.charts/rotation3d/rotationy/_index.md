## Rotation3D.RotationY 属性

返回或设置围绕 Y 轴的旋转角度，即在 X 方向上对于 3D 图表的旋转度数（范围从 0 到 360 度）。该属性对应于 ECMA-376 中的 21.2.2.158 rotY（Y 旋转）项以及 PowerPoint 2007+ 中的“X 旋转”选项。可读写 UInt16。

```csharp
public ushort RotationY { get; set; }
```

## Rotation3D.RotationY属性

返回或设置围绕Y轴的旋转角度，即3D图表在X方向上的旋转度（介于0到360度之间）。该属性与ECMA-376中的21.2.2.158 rotY（Y旋转）项相匹配，并且与PowerPoint 2007+中的“X旋转”选项相匹配。可读写UInt16。

```csharp
public ushort RotationY { get; set; }
```

## IDataLabel.IsVisible property

False 表示数据标签不可见（因此所有 Show*-flags (ShowValue, ...) 都是 false）。 只读Boolean。

```csharp
public bool IsVisible { get; }
```

### 评论

如果数据标签可见，您可以使用 Hide() 方法将其隐藏。 但如果数据标签不可见（IsVisible 为假），您可以通过将 Show*-flags (ShowValue, ...) 设置为 true 状态使数据标签可见 。

## IDataLabel.IsVisible 属性

False表示数据标签不可见（因此所有Show*-flags（ShowValue，...）都是false）。只读布尔值。

```csharp
public bool IsVisible { get; }
```

### 备注

如果数据标签可见，可以使用Hide()方法使其隐藏。但是，如果数据标签不可见（IsVisible为false），可以通过将Show*-flags（ShowValue，...）设置为true来使数据标签可见。

## IChartSeries.DoughnutHoleSize 属性

指定 doughnut 图表中孔的大小（可以在绘图区大小的 10% 到 90% 之间）。此属性不仅属于此系列，还属于父系列组的所有系列 - 这是相应组属性的投影。因此，此属性是只读的。使用 ParentSeriesGroup 属性访问父系列组。使用 ParentSeriesGroup.DoughnutHoleSize 读/写属性更改值。只读字节。

```csharp
public byte DoughnutHoleSize { get; }
```

### 备注

这是属性 ParentSeriesGroup.DoughnutHoleSize 的投影。

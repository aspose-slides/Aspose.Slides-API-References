## IChartSeries.PieSplitPosition property

指定一个值，用于确定哪个数据点在饼图或柱状图的第二个饼图或柱状图中。与 PieSplitBy 属性一起使用。这个属性不仅属于该系列，也属于父系列组的所有系列——这是相应组属性的投影。因此，这个属性是只读的。使用 ParentSeriesGroup 属性访问父系列组。使用 ParentSeriesGroup.PieSplitPosition 读/写属性更改值。只读 Double。

```csharp
public double PieSplitPosition { get; }
```

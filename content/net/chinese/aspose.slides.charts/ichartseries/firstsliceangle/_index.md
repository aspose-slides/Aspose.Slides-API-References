## IChartSeries.FirstSliceAngle 属性

指定第一个饼图或甜甜圈图切片的角度，单位为度（顺时针从上方开始，从0至360度）。此属性不仅属于该系列，还属于父系列组的所有系列——这是相应组属性的投影。因此，此属性是只读的。使用 ParentSeriesGroup 属性访问父系列组。使用 ParentSeriesGroup.FirstSliceAngle 读写属性更改值。只读 UInt16。

```csharp
public ushort FirstSliceAngle { get; }
```

### 备注

这是属性 ParentSeriesGroup.FirstSliceAngle 的投影。

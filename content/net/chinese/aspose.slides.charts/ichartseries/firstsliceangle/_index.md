## IChartSeries.FirstSliceAngle property

指定第一个饼图或甜甜圈图切片的角度，以度为单位（顺时针方向从上方0到360度）。这个属性不仅属于该系列，也属于所有父系列组的系列 - 这是相应组属性的投影。因此，这个属性是只读的。使用 ParentSeriesGroup 属性访问父系列组。使用 ParentSeriesGroup.FirstSliceAngle 读/写属性更改值。只读 UInt16。

```csharp
public ushort FirstSliceAngle { get; }
```

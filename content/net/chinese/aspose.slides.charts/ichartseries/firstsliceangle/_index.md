## IChartSeries.FirstSliceAngle property

指定第一个饼图或圆环图切片的角度， 以度为单位（从上顺时针，从 0 到 360 度）。 这不仅是这个系列的属性，而且是所有父系列的属性 group - 这是适当组属性的投影。所以这个属性 是只读的。 使用 ParentSeriesGroup 属性访问父系列组。 使用 ParentSeriesGroup.FirstSliceAngle 读/写属性来更改值。 只读UInt16。

public ushort FirstSliceAngle { get; }

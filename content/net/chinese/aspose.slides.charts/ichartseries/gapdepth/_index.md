## IChartSeries.GapDepth 属性

返回或设置3D图表中数据系列之间的距离，以标记宽度的百分比表示。该属性不仅适用于该系列，还适用于父系列组的所有系列 - 这是适当组属性的投影。因此，该属性是只读的。使用 ParentSeriesGroup 属性访问父系列组。使用 ParentSeriesGroup.GapDepth 读/写属性更改值。只读 Int32。

```csharp
public int GapDepth { get; }
```

## IChartSeries.HasSeriesLines property

确定此系列及相关系列是否存在系列线。这不仅是此系列的属性，也是父系列组所有系列的属性 - 这是适当组属性的投影。因此，此属性是只读的。使用 ParentSeriesGroup 属性访问父系列组。使用 ParentSeriesGroup.HasSeriesLines 读/写属性更改值。使用 ParentSeriesGroup.SeriesLinesFormat 属性格式化系列线。只读布尔值。

```csharp
public bool HasSeriesLines { get; }
```

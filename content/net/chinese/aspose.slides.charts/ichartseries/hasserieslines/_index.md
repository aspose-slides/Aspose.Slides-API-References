## IChartSeries.HasSeriesLines 属性

确定此系列及相关系列是否有系列线。此属性不仅属于此系列，还属于父系列组的所有系列——这是相应组属性的投影。因此，此属性是只读的。使用 ParentSeriesGroup 属性访问父系列组。使用 ParentSeriesGroup.HasSeriesLines 读/写属性更改值。使用 ParentSeriesGroup.SeriesLinesFormat 属性格式化系列线。只读布尔值。

```csharp
public bool HasSeriesLines { get; }
```

这是 ParentSeriesGroup.HasSeriesLines 属性的投影。

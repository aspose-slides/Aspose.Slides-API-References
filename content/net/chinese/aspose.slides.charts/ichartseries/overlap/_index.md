## IChartSeries.Overlap property

指定了条形图和柱形图在二维图表上的重叠程度，作为一个百分比（从 -100% 到 100%）。这不仅是该系列的属性，也是父系列组中所有系列的属性。它是父系列组中相应属性的投影，因此该属性是只读的。要更改该值，请使用 ParentSeriesGroup.Overlap 读/写属性。只读 SByte。

```csharp
public sbyte Overlap { get; }
```

### Remarks

Overlap 指定了条形图和柱形图之间的重叠或间距程度，作为它们宽度的百分比： - -100%：最大间距（条形图完全分开）。 - 0%：条形图并排放置，没有重叠或间距。 - 100%：最大重叠（条形图完全重叠）。这是属性 ParentSeriesGroup.Overlap 的投影。

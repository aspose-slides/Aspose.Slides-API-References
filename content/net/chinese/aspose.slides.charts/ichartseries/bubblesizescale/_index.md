## IChartSeries.BubbleSizeScale 属性

指定气泡图的比例因子（可以在默认大小的0到300百分比之间）。这个属性不仅属于这一系列，也属于父系列组的所有系列 - 这是相应组属性的投影。因此，这个属性是只读的。使用 ParentSeriesGroup 属性访问父系列组。使用 ParentSeriesGroup.BubbleSizeScale 可读/写属性更改值。

```csharp
public int BubbleSizeScale { get; }
```

### 备注

这是属性 ParentSeriesGroup.BubbleSizeScale 的投影。

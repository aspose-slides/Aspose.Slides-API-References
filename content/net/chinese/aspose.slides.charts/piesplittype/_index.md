## PieSplitType 枚举

表示在饼图或条形图中第二个饼图或条形的分割点类型。

```csharp
public enum PieSplitType
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Default | `0` | 指定数据点应使用该图表类型的默认机制进行分割。 |
| Custom | `1` | 指定数据点应根据自定义分割值在饼图和第二个图表之间进行分割。 |
| ByPercentage | `2` | 指定数据点应通过将百分比小于分割位置百分比的点放入第二个图表中，在饼图和第二个图表之间进行分割。 |
| ByPos | `3` | 指定数据点应通过将数据点的最后分割位置放入第二个图表中，在饼图和第二个图表之间进行分割。 |
| ByValue | `4` | 指定数据点应通过将值小于分割位置的数据点放入第二个图表中，在饼图和第二个图表之间进行分割。 |

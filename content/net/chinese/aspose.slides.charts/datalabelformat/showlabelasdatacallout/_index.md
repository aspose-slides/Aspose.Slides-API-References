## DataLabelFormat.ShowLabelAsDataCallout property

确定指定图表的数据标签是显示为数据调用提示还是作为数据标签。如果该 DataLabelFormat 对象的父对象是数据标签的 DataLabelCollection 集合，则此属性获取或设置 DataLabelCollection 集合中新数据标签的 ShowLabelAsDataCallout 属性的默认值。使用该属性的值设置也会将此值设置为 DataLabelCollection 集合中所有数据标签的 ShowLabelAsDataCallout 属性（即 "DataLabels.DefaultDataLabelFormat.ShowLabelAsDataCallout = val;" 导致所有 DataLabels[i].ShowLabelAsDataCallout 等于 val）。

```csharp
public bool ShowLabelAsDataCallout { get; set; }
```

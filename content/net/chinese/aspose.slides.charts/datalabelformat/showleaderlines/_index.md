## DataLabelFormat.ShowLeaderLines property

表示指定图表的数据标签前导线显示行为。 True 显示引导线。虚假隐藏。 读/写Boolean。

```csharp
public bool ShowLeaderLines { get; set; }
```

### 评论

如果此 DataLabelFormat 对象的父对象是数据标签的 DataLabelCollection 集合，则此 属性获取或设置 DataLabelCollection 集合中新数据 标签的 ShowLeaderLines 属性的默认值。 设置此属性的值也将此值设置为 ShowLeaderLines 属性 用于 DataLabelCollection 集合 中的所有数据标签（即“DataLabels.DefaultDataLabelFormat.ShowLeaderLines = val;" 导致 all DataLabels[i].ShowLeaderLines 等于 val)。

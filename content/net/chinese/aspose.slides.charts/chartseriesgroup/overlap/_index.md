## ChartSeriesGroup.Overlap 属性

指定在 2-D 图表上，条形和列的重叠程度，作为百分比（从 -100% 到 100%）。 - -100%：最大间隔（条形完全分开）。 - 0%：条形并排放置，没有重叠或间隔。 - 100%：最大重叠（条形完全重叠）。此属性为可读写 SByte。

```csharp
public sbyte Overlap { get; set; }
```

### 异常

| 异常 | 条件 |
| --- | --- |
| ArgumentOutOfRangeException | 当值设置在 -100 到 100 的有效范围之外时抛出。 |

### 示例

以下示例演示如何为图表系列组设置重叠，并在窗体上渲染结果图表：

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // 设置重叠为 55%

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

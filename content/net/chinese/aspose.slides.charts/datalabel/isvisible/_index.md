## DataLabel.IsVisible 属性

False意味着数据标签不可见（因此所有Show*-标志（ShowValue，...）都是false）。只读布尔值。

```csharp
public bool IsVisible { get; }
```

### 备注

如果数据标签可见，您可以使用Hide()方法将其隐藏。但是，如果数据标签不可见（IsVisible为false），您可以通过将Show*-标志（ShowValue，...）设置为true来使数据标签可见。

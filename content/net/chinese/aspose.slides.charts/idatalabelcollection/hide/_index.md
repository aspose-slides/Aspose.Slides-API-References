## IDataLabelCollection.Hide 方法

通过将 DefaultDataLabelFormat 属性的所有 Show*-标志 (ShowValue, ...) 设置为 false 状态，使数据标签默认隐藏。此时 IsVisible 将为 false。

```csharp
public void Hide()
```

### 注释

如果数据标签默认情况下不可见 (IsVisible 为 false)，您可以通过将 DefaultDataLabelFormat 属性的 Show*-标志 (ShowValue, ...) 设置为 true 状态，使数据标签“默认可见”。

## IDataLabelCollection.IsVisible 属性

false 表示数据标签默认情况下不可见（因此 DefaultDataLabelFormat 属性的所有 Show*-flags（ShowValue，...）都是 false）。只读布尔值。

```csharp
public bool IsVisible { get; }
```

### 备注

如果数据标签默认可见，您可以使用 Hide() 方法将其默认隐藏。但是如果数据标签默认不可见（IsVisible 为 false），则可以通过将 DefaultDataLabelFormat 属性的 Show*-flags（ShowValue，...）设置为 true 状态，使数据标签“默认可见”。

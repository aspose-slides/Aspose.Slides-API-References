## IRotation3D.RotationY 属性

返回或设置围绕 Y 轴的旋转角度，即在 3D 图表中的 X 方向（范围为 0 到 360 度）。该属性与 ECMA-376 中的 21.2.2.158 rotY（Y 旋转）项以及 PowerPoint 2007+ 中的 "X 旋转" 选项匹配。可读/可写 UInt16。

public ushort RotationY { get; set; }

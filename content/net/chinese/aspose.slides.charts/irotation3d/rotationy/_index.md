## IRotation3D.RotationY 属性

返回或设置围绕 Y 轴的旋转角度，即 3D 图表中 X 方向的旋转角度（在 0 到 360 度之间）。该属性与 ECMA-376 中的 21.2.2.158 rotY（Y 旋转）项和 PowerPoint 2007+ 中的 "X Rotation" 选项匹配。可读写 UInt16。

```csharp
public ushort RotationY { get; set; }
```

## IRotation3D.RotationX 属性

返回或设置围绕 X 轴的旋转度数，即在 Y 方向上对 3D 图表的旋转度（在 -90 到 90 度之间）。该属性与 ECMA-376 中的 21.2.2.157 rotX（X 旋转）项相匹配，并与 PowerPoint 2007+ 中的 "Y 旋转" 选项相匹配。可读/写 SByte。

```csharp
public sbyte RotationX { get; set; }
```

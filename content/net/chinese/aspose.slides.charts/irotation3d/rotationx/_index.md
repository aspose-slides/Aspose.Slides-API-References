## IRotation3D.RotationX 属性

返回或设置围绕 X 轴的旋转角度，即 3D 图表在 Y 方向上的旋转（在 -90 到 90 度之间）。该属性与 ECMA-376 中的 21.2.2.157 rotX（X 旋转）项以及 PowerPoint 2007+ 中的 "Y 旋转" 选项相匹配。读/写 SByte。

public sbyte RotationX { get; set; }

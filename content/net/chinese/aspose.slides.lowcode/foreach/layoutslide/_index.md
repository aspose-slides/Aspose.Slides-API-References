## ForEach.LayoutSlide 方法

迭代 [`Presentation`](../../../aspose.slides/presentation) 中的每个 `LayoutSlide`。

```csharp
public static void LayoutSlide(Presentation pres, ForEachLayoutSlideCallback forEachLayoutSlide)
```

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| pres | Presentation | 要迭代布局幻灯片的演示文稿 |
| forEachLayoutSlide | ForEachLayoutSlideCallback | 将为每个布局幻灯片调用的回调 |

### 示例

```csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    ForEach.LayoutSlide(pres, (layoutSlide, index) =>
    {
        layoutSlide.Name = $"LayoutSlide #{index}";
    });
} 
```

## ForEach.Slide 方法

遍历 [`Presentation`](../../../aspose.slides/presentation) 中的每个 `Slide`。

| 参数 | 类型 | 描述 |

| pres | Presentation | 用于遍历幻灯片的演示文稿 |
| forEachSlide | ForEachSlideCallback | 将为每个幻灯片调用的回调 |

### 示例

```csharp

{

}
```

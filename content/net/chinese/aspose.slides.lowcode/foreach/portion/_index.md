## Portion(Presentation, ForEachPortionCallback) {#portion_1}

遍历 [`Presentation`](../../../aspose.slides/presentation) 中的每个 `Portion`。所有类型的幻灯片 - [`Slide`](../slide)、[`MasterSlide`](../masterslide) 和 [`LayoutSlide`](../layoutslide) 中的部分将被遍历。

```csharp
public static void Portion(Presentation pres, ForEachPortionCallback forEachPortion)
```

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| pres | Presentation | 要遍历部分的演示文稿 |
| forEachPortion | ForEachPortionCallback | 对每个部分调用的回调 |

### 示例

```csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    ForEach.Portion(pres, (portion, para, slide, index) =>
    {
        System.Console.WriteLine($"{portion.Text}, index: {index}");
    });
} 
```

## Portion(Presentation, bool, ForEachPortionCallback) {#portion}

遍历 [`Presentation`](../../../aspose.slides/presentation) 中的每个 [`Portion`](../portion)。所有类型的幻灯片 - [`Slide`](../slide)、[`MasterSlide`](../masterslide)、[`LayoutSlide`](../layoutslide) 和 [`NotesSlide`](../../../aspose.slides/notesslide) 中的部分将被遍历。

```csharp
public static void Portion(Presentation pres, bool includeNotes, 
    ForEachPortionCallback forEachPortion)
```

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| pres | Presentation | 要遍历部分的演示文稿 |
| includeNotes | Boolean | 指示是否应在处理过程中包括 NotesSlides 的标志。 |
| forEachPortion | ForEachPortionCallback | 对每个部分调用的回调 |

### 示例

```csharp

{
    ForEach.Portion(pres, true, (portion, para, slide, index) =>
    {
        System.Console.WriteLine($"{portion.Text}, index: {index}");
    });
}
```

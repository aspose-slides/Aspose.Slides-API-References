## IMathBox.NoBreak property

没有中断。 此属性指定对象框上的“牢不可破”属性。如果为 true，则框内不会出现换行符。 这对于由多个二元运算符组成的运算符模拟器可能很重要。 如果未指定此元素，则框内可能会出现中断。 默认值:true

```csharp
public bool NoBreak { get; set; }
```

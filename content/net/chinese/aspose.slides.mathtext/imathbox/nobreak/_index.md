## IMathBox.NoBreak 属性

不可断裂。此属性指定对象框上的“不可断裂”属性。当为真时，框内无法发生换行。这对于由多个二进制运算符组成的运算符模拟器可能很重要。当未指定此元素时，框内可以发生换行。默认值：true

```csharp
public bool NoBreak { get; set; }
```

### 示例

示例：

```csharp
[C#]
IMathBox box = new MathematicalText("**********").ToBox();
box.NoBreak = false;
```

## IMathBox.NoBreak property

不断行。此属性指定对象框上的“不可分割”属性。当为真时，框内不能出现换行。这对于由多个二元运算符组成的运算符模拟器可能很重要。当不指定此元素时，框内可能会出现换行。默认值：真

```csharp
public bool NoBreak { get; set; }
```

示例：

box.NoBreak = false;

## IMathBox interface

指定数学元素的逻辑装箱（包装）。 例如，装箱的对象可以用作带有或不带有对齐点的操作员模拟器， 用作换行点，或者被分组以不允许换行内。 例如，“==”运算符应该被装箱以防止换行。

```csharp
public interface IMathBox : IMathElement
```

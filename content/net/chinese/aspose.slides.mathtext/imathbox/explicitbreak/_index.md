## IMathBox.ExplicitBreak property

显式换行指定 Box 对象的开头是否有换行符， 使得换行在盒子的开头目的。 指定前一行数学文本上的运算符编号，该编号应 用作当前数学文本行的对齐点 可能值:1..255 默认值:0（无显式中断）

```csharp
public byte ExplicitBreak { get; set; }
```

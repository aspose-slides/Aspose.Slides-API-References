## IMathBox.Differential property

Differential. 当为真时，盒子作为微分（例如，积分中的 𝑑𝑥），并获得适当的水平间距以用于数学微分。 默认值：假

```csharp
public bool Differential { get; set; }
```

### Examples

示例：

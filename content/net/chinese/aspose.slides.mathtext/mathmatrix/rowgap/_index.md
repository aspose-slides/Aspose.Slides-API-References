## MathMatrix.RowGap 属性

矩阵行之间的垂直间距值；如果 RowGapRule 设置为 3 (“Exactly”)，那么单位被解释为 twips（1/20 点）。如果 RowGapRule 设置为 4 (“Multiple”)，那么单位被解释为半行。默认值：0

```csharp
public uint RowGap { get; set; }
```

### 示例

示例：

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

## MathMatrix.MinColumnWidth 属性

最小列宽以twips表示（1/20点）间隙间距（也称为“列间距”或“间隙宽度”）会添加到MinColumnWidth中，以确定总的矩阵列间距（不同列相同边缘之间的距离）。默认值：0。

```csharp
public uint MinColumnWidth { get; set; }
```

### 示例

示例：

```csharp
[C#]

matrix.MinColumnWidth = 20;
```

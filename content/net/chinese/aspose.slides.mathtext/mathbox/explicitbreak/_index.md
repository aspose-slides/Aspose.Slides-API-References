## MathBox.ExplicitBreak 属性

显式换行指定 Box 对象开始处是否有换行，这样该行会在 Box 对象开始处换行。指定在前一行数学文本中的操作符号码，该号码将用作当前行数学文本的对齐点，可能的值为：1..255 默认值：0（无显式换行）

```csharp
public byte ExplicitBreak { get; set; }
```

### 示例

示例：

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.ExplicitBreak = 1;
```

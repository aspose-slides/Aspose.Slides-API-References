## MathBox.ExplicitBreak 属性

显式换行指定了在 Box 对象的开始是否存在换行，使得行在 Box 对象的开始处换行。指定了在数学文本前一行上使用的操作符的编号，该编号将作为当前数学行的对齐点，可能的值：1..255 默认：0（没有显式换行）

```csharp
public byte ExplicitBreak { get; set; }
```

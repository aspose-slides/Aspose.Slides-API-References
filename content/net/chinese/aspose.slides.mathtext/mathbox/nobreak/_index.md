## MathBox.NoBreak 属性

不可断裂 该属性指定对象框上的 "不可断裂" 属性。当为 true 时，不可以在框内产生换行。这对于由多个二元操作符组成的操作符仿真器可能很重要。当未指定该元素时，可以在框内出现断裂。默认值：true

```csharp
public bool NoBreak { get; set; }
```

### 示例

示例：

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("*****"));
box.NoBreak = false;
```

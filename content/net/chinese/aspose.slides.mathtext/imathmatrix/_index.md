## IMathMatrix 接口

指定矩阵对象，由一个或多个行和列排列的子元素组成。重要的是要注意，矩阵没有内置的分隔符。要将矩阵放在括号中，您应使用分隔符对象 (IMathDelimiter)。可以使用 Null 参数在矩阵中创建间隙。

```csharp
public interface IMathMatrix : IMathElement
```

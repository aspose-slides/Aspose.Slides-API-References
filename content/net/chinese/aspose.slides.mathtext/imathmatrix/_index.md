## IMathMatrix 接口

指定 Matrix 对象，由一个或多个行列排列的子元素组成。需要注意的是矩阵没有内建分隔符。要在括号中放置矩阵，应使用分隔符对象 (IMathDelimiter)。可使用空参数在矩阵中创建空格。

```csharp
public interface IMathMatrix : IMathElement
```

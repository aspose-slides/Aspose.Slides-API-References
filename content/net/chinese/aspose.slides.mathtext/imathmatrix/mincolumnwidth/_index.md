## IMathMatrix.MinColumnWidth 属性

列的最小宽度以 twips 表示（1/20 点）。间隔距（也称为“列间距”或“间隔宽度”）加到 MinColumnWidth 上以确定不同列相同边缘之间的总矩阵列间距。默认值：0。

```csharp
public uint MinColumnWidth { get; set; }
```

### 示例

示例：

```csharp

matrix.MinColumnWidth = 20;
```

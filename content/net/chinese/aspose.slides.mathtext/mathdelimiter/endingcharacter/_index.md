## MathDelimiter.EndingCharacter 属性

分隔符结束字符指定结束或闭合的分隔符字符。数学分隔符是括号、方括号和大括号等包围字符。默认值为 ')'。

```csharp
public char EndingCharacter { get; set; }
```

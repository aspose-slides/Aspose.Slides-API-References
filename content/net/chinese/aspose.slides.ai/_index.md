包含提供用于分析和处理 PowerPoint 演示文稿的 AI 功能的类。

## 类

| 类 | 描述 |
| --- | --- |
| [AsposeAIWebClient](./asposeaiwebclient) | 一个内置的 [`IAIWebClient`](../aspose.slides.ai/iaiwebclient) 实现，连接到 Aspose 的自有 LLM。这是无参 [`SlidesAIAgent`](../aspose.slides.ai/slidesaiagent/slidesaiagent) 构造函数使用的默认客户端。 |
| [OpenAICompatibleWebClient](./openaicompatiblewebclient) | 一个内置的 [`IAIWebClient`](../aspose.slides.ai/iaiwebclient) 实现，连接到指定基 URL 的兼容 OpenAI 的 LLM 提供商。 |

| [SlidesAIAgent](./slidesaiagent) | 提供用于处理演示文稿的 AI 功能。 |
| [SlidesAIAgentException](./slidesaiagentexception) | 表示 Slides AI Agent 相关的异常。 |

| 接口 | 描述 |

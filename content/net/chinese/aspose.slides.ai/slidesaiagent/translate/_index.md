## SlidesAIAgent.Translate 方法

Translates a presentation to the specified language using AI (synchronous version).

```csharp
public void Translate(IPresentation presentation, string language)
```

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| presentation | IPresentation | 目标演示文稿 |
| language | String | 目标语言 |

### 异常

| 异常 | 条件 |
| --- | --- |
| ArgumentNullException | 未提供 Presentation 实例 |
| ArgumentException | Language 值不能为空或为空 |

### 备注

下面的示例使用默认的 [`AsposeAIWebClient`](../../asposeaiwebclient)，该实例由无参数的 [`SlidesAIAgent`](../slidesaiagent) 构造函数创建，并连接到 Aspose 自己的 LLM。要使用不同的 AI 提供商，提供您自己的 LLM，或自定义连接（例如，通过提供您自己的 `HttpClient`），将 [`IAIWebClient`](../../iaiwebclient) 实现传递给 [`SlidesAIAgent`](../slidesaiagent) 构造函数。可用的实现包括：

* [`AsposeAIWebClient`](../../asposeaiwebclient)
* [`OpenAIWebClient`](../../openaiwebclient)
* [`OpenAICompatibleWebClient`](../../openaicompatiblewebclient)

## IHtmlOptions.JpegQuality 属性

返回或设置一个值，决定 PDF 文档中 JPEG 图像的质量。可读/写字节。

```csharp
public byte JpegQuality { get; set; }
```

### 备注

仅当文档包含 JPEG 图像时才有效。

使用此属性在以 PDF 格式保存时获取或设置文档中图像的质量。值的范围从 0 到 100，其中 0 表示最差质量但最大压缩，100 表示最佳质量但最小压缩。

默认值为 **95**。

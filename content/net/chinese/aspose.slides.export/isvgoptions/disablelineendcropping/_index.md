## ISVGOptions.DisableLineEndCropping 属性

SVG 1.1 缺乏定义标记插图的能力。Aspose.Slides SVG 写入引擎对此问题有解决方法：它裁剪带箭头的线段末端，以防止线段与标记重叠。此选项关闭此行为。可读/可写布尔值。

```csharp
public bool DisableLineEndCropping { get; set; }
```

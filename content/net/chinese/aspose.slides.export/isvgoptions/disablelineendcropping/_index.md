## ISVGOptions.DisableLineEndCropping property  

SVG 1.1 缺乏定义标记内边距的能力。Aspose.Slides SVG 写入引擎为该问题提供了变通方案：它裁剪带箭头的线的末端，以便线不会与标记重叠。此选项关闭此类行为。读/写布尔值。

public bool DisableLineEndCropping { get; set; }

## CompressionLevel 枚举

指定 ZIP 对 OpenXML 文件的压缩级别。更高的级别在压缩效果更好时会导致处理速度更慢。

```csharp
public enum CompressionLevel
```

### Values

| 名称 | 值 | 描述 |
| --- | --- | --- |
| None | `0` | 未应用压缩。文件将按原样存储。 |
| Level1 | `1` | 最快的压缩，压缩率最低。 |
| Level2 | `2` | 比 Level1 稍好压缩率的更快压缩。 |
| Level3 | `3` | 提供比 Level2 更好的压缩，性能影响适中。 |
| Level4 | `4` | 提供比 Level3 更好的压缩。 |
| Level5 | `5` | 在 Level4 基础上提供改进的压缩，但需要额外的处理时间。 |
| Level6 | `6` | 标准压缩，在压缩速度和文件大小之间提供良好平衡。默认的压缩级别。 |
| Level7 | `7` | 提供比 Level6 更高的压缩，但处理速度较慢。 |
| Level8 | `8` | 提供比 Level7 更高的压缩。 |
| Level9 | `9` | 最大压缩。产生最小的文件大小，但处理速度最慢。 |

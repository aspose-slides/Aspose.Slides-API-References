## IPdfOptions.RasterizeUnsupportedFontStyles 属性

当字体不支持粗体样式时，指示文本是否应被栅格化为位图并保存为PDF。此方法可以提高某些字体在生成的PDF中的文本质量。可读/可写布尔值。

```csharp
public bool RasterizeUnsupportedFontStyles { get; set; }
```

### 备注

默认值为 **false**。

## IPdfOptions.EmbedTrueTypeFontsForASCII 属性

设置为 true 以嵌入 ASCII 字符 32-127 的 TrueType 字体。字符代码大于 127 的字体始终会被嵌入。可读/写布尔值。

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### 备注

默认值为 **true**。

## IPdfOptions.JpegQuality property

返回或设置一个值，该值决定PDF文档中JPEG图像的质量。可读/写字节。

```csharp
public byte JpegQuality { get; set; }
```

### Remarks

仅当文档包含JPEG图像时才有效。

使用此属性在以PDF格式保存文档时获取或设置图像的质量。值可以从0到100变化，其中0表示质量最差但压缩最大，100表示质量最佳但压缩最小。

默认值为**100**。

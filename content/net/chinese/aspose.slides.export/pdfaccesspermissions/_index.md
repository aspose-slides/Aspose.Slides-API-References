## PdfAccessPermissions 枚举

包含一组标志，用于指定在使用用户访问打开文档时应授予哪些访问权限。

```csharp
[Flags]
public enum PdfAccessPermissions
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| None | `0` | 指定用户没有访问权限。 |
| PrintDocument | `4` | 指定用户是否可以打印文档（可能不是最高质量级别，这取决于是否同时设置了 HighQualityPrint 位）。 |
| ModifyContent | `8` | 指定用户是否可以通过除 AddOrModifyFields、FillExistingFields、AssembleDocument 位控制的操作之外的方式修改文档内容。 |
| CopyTextAndGraphics | `10` | 指定用户是否可以通过除 ExtractTextAndGraphics 位控制的操作之外的方式复制或以其他方式提取文档中的文本和图形。 |
| AddOrModifyFields | `20` | 指定用户是否可以添加或修改文本批注、填写交互式表单字段，并且在同时设置了 ModifyContent 位时，创建或修改交互式表单字段（包括签名字段）。 |
| FillExistingFields | `100` | 指定用户是否可以填写已有的交互式表单字段（包括签名字段），即使 AddOrModifyFields 位被清除。 |
| ExtractTextAndGraphics | `200` | 指定用户是否可以提取文本和图形，以支持残障用户的可访问性或出于其他目的。 |
| AssembleDocument | `400` | 指定用户是否可以组装文档（插入、旋转或删除页面以及创建书签或缩略图），即使 ModifyContent 位被清除。 |
| HighQualityPrint | `800` | 指定用户是否可以将文档打印为可生成 PDF 内容忠实数字副本的表示。当此位被清除且 PrintDocument 位被设置时，打印将限制为外观的低层次表示，可能质量下降。 |

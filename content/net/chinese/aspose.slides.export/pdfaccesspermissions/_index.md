## PdfAccessPermissions 枚举

包含一组标志，指定在以用户访问权限打开文档时应授予哪些访问权限。

```csharp
[Flags]
public enum PdfAccessPermissions
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| None | `0` | 指定用户没有访问权限。 |
| PrintDocument | `4` | 指定用户是否可以打印文档（可能不是以最高质量级别打印，具体取决于是否设置了位 HighQualityPrint）。 |
| ModifyContent | `8` | 指定用户是否可以通过其他操作修改文档内容，而不是受位 AddOrModifyFields、FillExistingFields、AssembleDocument 控制的操作。 |
| CopyTextAndGraphics | `10` | 指定用户是否可以通过其他操作复制或提取文档中的文本和图形，而不是受位 ExtractTextAndGraphics 控制的操作。 |
| AddOrModifyFields | `20` | 指定用户是否可以添加或修改文本注释，填写交互式表单字段，并且如果位 ModifyContent 也被设置，则可以创建或修改交互式表单字段（包括签名字段）。 |
| FillExistingFields | `100` | 指定用户是否可以填写现有的交互式表单字段（包括签名字段），即使位 AddOrModifyFields 清除。 |
| ExtractTextAndGraphics | `200` | 指定用户是否可以提取文本和图形，以支持残障用户的可访问性或其他目的。 |
| AssembleDocument | `400` | 指定用户是否可以组装文档（插入、旋转或删除页面并创建书签或缩略图），即使位 ModifyContent 清除。 |
| HighQualityPrint | `800` | 指定用户是否可以打印文档，以生成可以生成 PDF 内容忠实数字副本的表示形式。当此位清除（且位 PrintDocument 被设置）时，打印限制为外观的低级表示，可能质量下降。 |

## MarkdownSaveOptions.SlideNumberFormat 属性

获取或设置用于 Markdown 输出中的幻灯片编号标题的格式字符串。该格式必须包含 "{0}" 占位符，该占位符将在导出期间替换为幻灯片索引。例如："# Slide {0}" 将生成 "# Slide 1"、"# Slide 2" 等。

public string SlideNumberFormat { get; set; }

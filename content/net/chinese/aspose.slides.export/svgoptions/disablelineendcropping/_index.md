## SVGOptions.DisableLineEndCropping 属性

SVG 1.1 缺乏定义标记内边距的能力。Aspose.Slides SVG 写入引擎为该问题提供了一个解决方法：它裁剪带箭头的行尾，以便行与标记不重叠。此选项关闭此行为。可读/可写布尔值。

```csharp
public bool DisableLineEndCropping { get; set; }
```

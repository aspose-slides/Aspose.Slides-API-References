## SVGOptions.DisableLineEndCropping 属性

SVG 1.1 缺乏定义标记插入的能力。Aspose.Slides SVG 写入引擎有一个解决此问题的变通方案：它裁剪带箭头的线段末端，以便线段不会与标记重叠。此选项关闭此行为。读/写布尔值。

public bool DisableLineEndCropping { get; set; }

## PdfOptions.SaveMetafilesAsPng 属性

将演示文稿中使用的所有元文件转换为PNG图像。可读/写布尔值。

```csharp
public bool SaveMetafilesAsPng { get; set; }
```

### 备注

默认值为**true**。Pdf文档可以包含矢量图形和栅格图像。如果将SaveMetafilesAsPng设置为true，则源元文件图像将转换为Png格式并以栅格图像的形式保存到Pdf中。如果将SaveMetafilesAsPng设置为false，则源元文件将转换为Pdf矢量图形。每种方法都有其优点和缺点。例如，如果元文件转换为PNG，则在生成的文档缩放时可能会有质量损失。如果元文件转换为Pdf矢量图形，则在Pdf查看工具中可能会出现性能问题。

## PdfOptions.EmbedTrueTypeFontsForASCII 属性

确定 Aspose.Slides 是否会为 ASCII (33..127 代码范围) 文本嵌入常用字体。字符代码大于 127 的字体始终被嵌入。常用字体列表包括 PDF 的基础 14 种字体和额外的用户指定字体。可读写布尔值。

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### 备注

默认值为 **true**。

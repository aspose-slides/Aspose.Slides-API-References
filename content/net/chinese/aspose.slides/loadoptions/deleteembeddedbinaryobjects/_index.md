## LoadOptions.DeleteEmbeddedBinaryObjects 属性

确定 Aspose.Slides 在加载演示文稿时是否会删除所有嵌入的二进制对象。

嵌入的二进制对象类型：

* VBA 项目 [`VbaProject`](../../ipresentation/vbaproject)
* OLE 对象嵌入数据 [`EmbeddedFileData`](../../ioleembeddeddatainfo/embeddedfiledata)
* ActiveX 控件二进制数据 [`ActiveXControlBinary`](../../icontrol/activexcontrolbinary)

可读/写布尔值。

```csharp
public bool DeleteEmbeddedBinaryObjects { get; set; }
```

### 备注

默认值为 **false**。

### 示例

以下示例演示如何加载演示文稿而不包含任何嵌入的二进制对象。

```csharp
[C#]
LoadOptions loadOptions = new LoadOptions();
loadOptions.DeleteEmbeddedBinaryObjects = true;
using (Presentation pres = new Presentation("pres.ppt", loadOptions))
{
    pres.Save("output_WithoutBinaryObjects.ppt", SaveFormat.Ppt);
}
```

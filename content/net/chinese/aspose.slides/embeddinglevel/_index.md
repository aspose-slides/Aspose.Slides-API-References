## EmbeddingLevel 枚举

表示嵌入字体的许可权利。

```csharp
[Flags]
public enum EmbeddingLevel : ushort
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Installable | `0` | 具有此设置的字体表示它们可以被嵌入并通过应用程序永久安装在远程系统上。远程系统的用户获得与该字体的原始购买者相同的权利、义务和许可，并受与原始购买者相同的最终用户许可协议、版权、外观专利和/或商标约束。 |
| Restricted | `2` | 仅设置了此位的字体不得在未事先获得合法所有者许可的情况下以任何方式修改、嵌入或交换。 |
| PreviewPrint | `4` | 当设置此位时，字体可以嵌入并暂时加载到远程系统。包含 Preview &amp; Print 字体的文档必须以只读方式打开；不允许对文档进行编辑。 |
| Editable | `8` | 当设置此位时，字体可以嵌入，但只能临时安装在其他系统上。与 Preview &amp; Print 字体相比，包含 Editable 字体的文档可以打开读取，允许编辑，并且可以保存更改。 |
| NoSubsetting | `100` | 当设置此位时，字体在嵌入前不得进行子集化。位 0-3 和 9 中指定的其他嵌入限制同样适用。 |
| BitmapOnly | `200` | 当设置此位时，仅可以嵌入字体中包含的位图。轮廓数据不能被嵌入。如果字体中没有可用的位图，则该字体被视为不可嵌入，嵌入服务将失败。 |

## 嵌入级别枚举

表示嵌入字体的许可权利。

```csharp
[Flags]
public enum EmbeddingLevel : ushort
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Installable | `0` | 具有此设置的字体表示它们可以被嵌入并由应用程序永久安装在远程系统上。远程系统的用户获得与原字体购买者相同的权利、义务和许可，并受到与原购买者相同的最终用户许可协议、版权、设计专利和/或商标的约束。 |
| Restricted | `2` | 仅设置此位的字体在未经合法所有者许可的情况下不得以任何方式进行修改、嵌入或交换。 |
| PreviewPrint | `4` | 当设置此位时，字体可以被嵌入，并临时加载在远程系统上。包含预览和打印字体的文档必须以“只读”方式打开；文档不得进行编辑。 |
| Editable | `8` | 当设置此位时，字体可以被嵌入，但只能在其他系统上临时安装。与预览和打印字体不同，包含可编辑字体的文档可以被打开以供阅读，并允许进行编辑，且可以保存更改。 |
| NoSubsetting | `100` | 当设置此位时，字体在嵌入前不得被子集化。位 0-3 和 9 中指定的其他嵌入限制也适用。 |
| BitmapOnly | `200` | 当设置此位时，仅可以嵌入字体中包含的位图。不得嵌入轮廓数据。如果字体中没有可用的位图，则该字体被视为不可嵌入，嵌入服务将失败。 |

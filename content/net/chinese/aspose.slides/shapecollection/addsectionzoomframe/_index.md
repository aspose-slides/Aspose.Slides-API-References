## 添加节缩放框(float, float, float, float, ISection) {#addsectionzoomframe}

将一个新的节缩放对象添加到集合的末尾。

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section)
```

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| x | Single | 新节缩放框的 X 坐标。 |
| y | Single | 新节缩放框的 Y 坐标。 |
| width | Single | 新节缩放框的宽度。 |
| height | Single | 新节缩放框的高度。 |
| section | ISection | 节缩放框所引用的节对象 [`ISection`](../../isection)。 |

### 返回值

创建的节缩放对象 [`ISectionZoomFrame`](../../isectionzoomframe)。

### 异常

| 异常 | 条件 |
| --- | --- |
| ArgumentException | 引用的节不属于当前演示文稿或不包含任何幻灯片。 |

### 示例

此示例演示如何将节缩放对象添加到集合的末尾（假设“Presentation.pptx”演示文稿中至少有两个节）：

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1]);
}
```

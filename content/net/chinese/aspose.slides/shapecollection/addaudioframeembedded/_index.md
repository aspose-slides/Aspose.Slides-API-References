## AddAudioFrameEmbedded(float, float, float, float, Stream) {#addaudioframeembedded_1}

将带有嵌入音频文件的新音频帧添加到集合的末尾。 嵌入的音频文件只能是 WAV。 它将新音频添加到 Presentation.Audios 列表中。

```csharp
public IAudioFrame AddAudioFrameEmbedded(float x, float y, float width, float height, 
    Stream audio_stream)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| x | Single | 新音频帧的 X 坐标。 |
| y | Single | 新音频帧的 Y 坐标。 |
| width | Single | 新音频帧的宽度。 |
| height | Single | 新音频帧的高度。 |
| audio_stream | Stream | 带有音频数据的输入流。 |

### 返回值

创建了 AudioFrame 对象。

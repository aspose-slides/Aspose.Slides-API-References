## IBaseSlide.ShowMasterShapes 属性

指定母版幻灯片上的形状是否应在幻灯片上显示。对于母版幻灯片本身，此属性始终返回 `false`。可读写布尔值。

```csharp
public bool ShowMasterShapes { get; set; }
```

### 异常

| exception | condition |
| --- | --- |
| NotSupportedException | 如果为母版幻灯片设置为 `true`，则抛出此异常。 |

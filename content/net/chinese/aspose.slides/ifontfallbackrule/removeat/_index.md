## IFontFallBackRule.RemoveAt 方法

删除指定索引处的 FallBack 字体。

```csharp
public void RemoveAt(int index)
```

| 参数 | 类型 | 描述 |

| index | Int32 | 要删除的字体的零基索引。 |

### 示例

```csharp
[C#]
```

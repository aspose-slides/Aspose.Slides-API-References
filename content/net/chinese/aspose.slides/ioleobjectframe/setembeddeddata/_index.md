## IOleObjectFrame.SetEmbeddedData 方法

设置有关OLE嵌入数据的信息。

```csharp
public void SetEmbeddedData(IOleEmbeddedDataInfo embeddedData)
```

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| embeddedData | IOleEmbeddedDataInfo | 嵌入数据 [`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo) |

### 异常

| 异常 | 条件 |
| --- | --- |
| ArgumentNullException | 当embeddedData参数为null时。 |

### 备注

此方法更改对象的属性以反映新数据，并将IsObjectLink标志设置为false，表示OLE对象是嵌入的。

### 示例

以下示例演示了如何为现有的[`IOleObjectFrame`](../../ioleobjectframe)对象更改OLE嵌入数据及其类型

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    OleObjectFrame oof = pres.Slides[0].Shapes[0] as OleObjectFrame;
    if (oof != null)
    {
        IOleEmbeddedDataInfo newData = new OleEmbeddedDataInfo(File.ReadAllBytes("Picture.png"), "png");
        oof.SetEmbeddedData(newData);
    }
}
```

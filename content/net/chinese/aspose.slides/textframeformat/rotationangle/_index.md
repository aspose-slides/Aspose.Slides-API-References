## TextFrameFormat.RotationAngle property

指定应用于边界框内文本的自定义旋转。如果未指定，则使用相应形状的旋转。如果指定，则与形状独立应用。也就是说，形状可以在文本本身应用旋转的基础上另行应用旋转。从该属性和属性 TextVerticalType 中的预定义垂直类型总结出的可视文本旋转的结果值。可读/可写单精度浮点数。

```csharp
public float RotationAngle { get; set; }
```

### Examples

考虑一个形状应用了顺时针旋转90度的情况。除了这个，文本主体本身应用了逆时针旋转-90度。然后，结果的形状看起来像是旋转了，但其中的文本看起来像是没有旋转过一样。

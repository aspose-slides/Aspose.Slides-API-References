## INormalViewRestoredProperties.DimensionSize 属性

指定幻灯片区域的大小（当为 RestoredTop 的子项时为宽度，当为 RestoredLeft 的子项时为高度）。可读/可写的 Single。在值小于 0 或大于 100 时抛出异常。

public float DimensionSize { get; set; }

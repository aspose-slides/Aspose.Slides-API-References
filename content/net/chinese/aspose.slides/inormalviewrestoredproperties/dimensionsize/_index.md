## INormalViewRestoredProperties.DimensionSize property

指定幻灯片区域的大小（当是RestoredTop的子项时为宽度，当是RestoredLeft的子项时为高度）。可读写单精度浮点数。当值小于0或大于100时会抛出异常。

public float DimensionSize { get; set; }

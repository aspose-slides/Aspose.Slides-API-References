## IProtectionManager.IsOnlyDocumentPropertiesLoaded 属性

如果演示文稿文件是密码保护的，并且该文件的文档属性是公开的，这个属性才有意义。值为 true 意味着仅从加密的演示文件中加载文档属性，而不使用密码。值为 false 意味着加载整个加密的演示文件，使用正确的密码，而不仅仅加载文档属性。如果演示文稿未加密，则属性值始终为 false。如果加密文件的文档属性不是公开的，则属性值始终为 false。如果 PresentationEx.EncryptDocumentProperties 为 true，则 IsOnlyDocumentPropertiesLoaded 属性值始终为 false。只读布尔值。

```csharp
public bool IsOnlyDocumentPropertiesLoaded { get; }
```

## BaseSlide.ShowMasterShapes 属性

指定主幻灯片上的形状是否应在幻灯片上显示。对于主幻灯片本身，该属性始终返回 `false`。可读写布尔值。

```csharp
public abstract bool ShowMasterShapes { get; set; }
```

## IDocumentProperties.AppVersion 属性

返回应用程序版本。只读字符串。

```csharp
public string AppVersion { get; }
```

### 备注

该元素的内容应为 XX.YYYY 形式，其中 X 和 Y 表示数值；否则，文档将被视为不合规。Aspose.Slides 将其版本表示为 XX.YYZZ 格式，其中： XX - 主版本 YY - 次版本 ZZ - 修补版本 例如，值 23.0105 表示 Aspose.Slides 版本 23.1.5。

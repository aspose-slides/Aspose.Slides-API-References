## IViewProperties.GridSpacing 属性

返回或设置应用于演示文档底层网格的网格间距，以点为单位。可读/写的单精度浮点数。

```csharp
public float GridSpacing { get; set; }
```

### 备注

网格间距值必须是正数。典型的值范围是从 1 毫米（2.8349607 点）到 2 英寸（144 点）。

### 示例

以下示例代码演示如何更改 PowerPoint 演示文稿中的网格间距。

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

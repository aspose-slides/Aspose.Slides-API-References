## IViewProperties.GridSpacing 属性

返回或设置应为演示文稿文档底层网格使用的网格间距，以点为单位。可读/可写单精度浮点型。

```csharp
public float GridSpacing { get; set; }
```

### 注意事项

网格间距值必须是正数。典型的值范围从 1 毫米（2.8349607 点）到 2 英寸（144 点）。

### 示例

以下示例代码演示了如何在 PowerPoint 演示文稿中更改网格间距。

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

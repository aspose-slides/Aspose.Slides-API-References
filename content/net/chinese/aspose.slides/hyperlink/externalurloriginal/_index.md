## Hyperlink.ExternalUrlOriginal 属性

表示为这一部分设置的超链接，与该部分的实际内容无关。 PowerPoint 对链接及其相应文本的行为是特定的。它允许以有效的 URL 形式创建超链接文本，这与链接的真实地址不同。在这种情况下，当您在编辑窗口查看链接时，它将更改为与文本部分匹配。此属性表示超链接的原始值。

```csharp
public string ExternalUrlOriginal { get; }
```

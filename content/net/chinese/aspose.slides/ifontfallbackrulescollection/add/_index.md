## IFontFallBackRulesCollection.Add method

将新的 FallBack 规则添加到集合末尾。


### 示例

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    // 从 FontsManager 获取空或预初始化的规则集合

    // 将新规则添加到集合
```

## MasterHandoutSlide.ShowMasterShapes property

指定母版幻灯片上的形状是否应在幻灯片上显示。对于母版幻灯片本身，这个属性始终返回 `false`。可读/写布尔值。

```csharp
public override bool ShowMasterShapes { get; set; }
```

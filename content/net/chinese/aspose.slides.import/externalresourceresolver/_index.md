## ExternalResourceResolver 类

用于在导入 Html、Svg 文档时解析外部资源的回调类。使用此解析器可能会产生漏洞，当客户提供的 HTML 或 SVG 文件使服务器软件获取本地或网络文件时。请谨慎使用。建议根本不指定 ExternalResourceResolver（仅会读取嵌入对象），或创建一些子类以检查指定的 URI 是否有效。

public class ExternalResourceResolver : IExternalResourceResolver

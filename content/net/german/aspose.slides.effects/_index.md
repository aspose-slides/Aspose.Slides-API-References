Enthält Klassen für die Arbeit mit verschiedenen Effekten in Microsoft PowerPoint-Präsentationen.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [AlphaBiLevel](./alphabilevel) | Stellt einen Alpha Bi-Level-Effekt dar. Alpha (Opazität) Werte unterhalb des Schwellenwerts werden auf 0 (vollständig transparent) geändert und Alpha-Werte, die größer oder gleich dem Schwellenwert sind, werden auf 100% (vollständig undurchsichtig) geändert. |
| [AlphaCeiling](./alphaceiling) | Stellt einen Alpha Ceiling-Effekt dar. Alpha (Opazität) Werte größer als null werden auf 100% geändert. Mit anderen Worten, alles teilweise undurchsichtig wird vollständig undurchsichtig. |
| [AlphaFloor](./alphafloor) | Stellt einen Alpha Floor-Effekt dar. Alpha (Opazität) Werte unter 100% werden auf null geändert. Mit anderen Worten, alles teilweise transparent wird vollständig transparent. |
| [AlphaInverse](./alphainverse) | Stellt einen Alpha Inverse-Effekt dar. Alpha (Opazität) Werte werden durch Subtrahieren von 100% invertiert. |
| [AlphaModulate](./alphamodulate) | Stellt einen Alpha Modulate-Effekt dar. Effekt-Alpha (Opazität) Werte werden mit einem festen Prozentsatz multipliziert. Der Effektcontainer gibt einen Effekt an, der Alpha-Werte zur Modulation enthält. |
| [AlphaModulateFixed](./alphamodulatefixed) | Stellt einen Alpha Modulate Fixed-Effekt dar. Effekt-Alpha (Opazität) Werte werden mit einem festen Prozentsatz multipliziert. |
| [AlphaReplace](./alphareplace) | Stellt einen Alpha Replace-Effekt dar. Effekt-Alpha (Opazität) Werte werden durch einen festen Alpha ersetzt. |
| [BiLevel](./bilevel) | Stellt einen Bi-Level (schwarz/weiß) Effekt dar. Eingabefarben, deren Helligkeit unter dem angegebenen Schwellenwert liegt, werden auf schwarz geändert. Eingabefarben, deren Helligkeit größer oder gleich dem angegebenen Wert sind, werden auf weiß gesetzt. Die Alpha-Effektwerte werden von diesem Effekt nicht beeinflusst. |
| [Blur](./blur) | Stellt einen Blur-Effekt dar, der auf die gesamte Form angewendet wird, einschließlich ihrer Füllung. Alle Farbkanäle, einschließlich Alpha, sind betroffen. |
| [ColorChange](./colorchange) | Stellt einen Color Change-Effekt dar. Instanzen von FromColor werden durch Instanzen von ToColor ersetzt. |
| [ColorReplace](./colorreplace) | Stellt einen Color Replacement-Effekt dar. Alle Effektfarben werden auf eine feste Farbe geändert. Alpha-Werte sind nicht betroffen. |
| [Duotone](./duotone) | Stellt einen Duotone-Effekt dar. Für jeden Pixel kombiniert er Color1 und Color2 durch eine lineare Interpolation, um die neue Farbe für diesen Pixel zu bestimmen. |
| [EffectFactory](./effectfactory) | Ermöglicht das Erstellen von Effekten |
| [FillOverlay](./filloverlay) | Stellt einen Fill Overlay-Effekt dar. Ein Füllüberlagerung kann verwendet werden, um eine zusätzliche Füllung für ein Objekt anzugeben und die beiden Füllungen zu mischen. |
| [Glow](./glow) | Stellt einen Glow-Effekt dar, bei dem ein farbiger, unscharfer Rand außerhalb der Kanten des Objekts hinzugefügt wird. |
| [GrayScale](./grayscale) | Stellt einen Gray Scale-Effekt dar. Konvertiert alle Effektfarben in einen Grauton, der ihrer Helligkeit entspricht. Effekt-Alpha (Opazität) Werte sind nicht betroffen. |
| [HSL](./hsl) | Stellt einen Hue/Saturation/Luminance-Effekt dar. Der Farbton, die Sättigung und die Helligkeit können jeweils relativ zu ihrem aktuellen Wert angepasst werden. |
| [ImageTransformOCollectionEffectiveData](./imagetransformocollectioneffectivedata) | Unveränderliches Objekt, das eine schreibgeschützte Sammlung von effektiven Bildtransformations-Effekten darstellt. |
| [ImageTransformOperation](./imagetransformoperation) | Stellt einen abstrakten Bildtransformationseffekt dar. |
| [ImageTransformOperationCollection](./imagetransformoperationcollection) | Stellt eine Sammlung von auf ein Bild angewendeten Effekten dar. |

| [OuterShadow](./outershadow) | Stellt einen Outer Shadow-Effekt dar. |
| [PresetShadow](./presetshadow) | Stellt einen Preset Shadow-Effekt dar. |
| [Reflection](./reflection) | Stellt einen Reflection-Effekt dar. |
| [SoftEdge](./softedge) | Stellt einen Soft Edge-Effekt dar. Die Kanten der Form sind unscharf, während die Füllung nicht betroffen ist. |
| [Tint](./tint) | Stellt einen Tint-Effekt dar. Verschiebt Effektfarbwerte in Richtung/Ferne von Hue um den angegebenen Betrag. |

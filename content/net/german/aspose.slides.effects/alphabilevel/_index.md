## AlphaBiLevel class

Stellt einen Alpha Bi-Level-Effekt dar. Alpha (Transparenz) Werte, die kleiner als der Schwellenwert sind, werden auf 0 (vollständig transparent) geändert und Alpha-Werte, die größer oder gleich dem Schwellenwert sind, werden auf 100% (vollständig undurchsichtig) geändert.

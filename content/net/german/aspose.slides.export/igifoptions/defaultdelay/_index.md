## IGifOptions.DefaultDelay-Eigenschaft

Ruft die Standardverzögerungszeit [ms] ab oder setzt sie. Dieser Wert wird verwendet, wenn [`AdvanceAfterTime`](../../../aspose.slides/islideshowtransition/advanceaftertime) nicht gesetzt ist. Der Standardwert beträgt 1000.

```csharp
public int DefaultDelay { get; set; }
```

### Beispiele

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
	pres.Save("pres.gif", SaveFormat.Gif, new GifOptions { DefaultDelay = 2000 });
}
```

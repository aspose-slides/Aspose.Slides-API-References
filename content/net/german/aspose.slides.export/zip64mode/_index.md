## Zip64Mode Aufzählung

Gibt an, wann ZIP64-Format-Erweiterungen für OpenXML-Dateien verwendet werden sollen.

```csharp
public enum Zip64Mode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Nie | `0` | Verwenden Sie keine ZIP64-Format-Erweiterungen. |
| Bei Bedarf | `1` | Verwenden Sie ZIP64-Format-Erweiterungen, wenn dies erforderlich ist. |
| Immer | `2` | Verwenden Sie immer ZIP64-Format-Erweiterungen. |

### Bemerkungen

Eine OpenXML-Datei ist ein ZIP-Archiv, das eine Begrenzung von 4 GB (2^32 Bytes) für die unkomprimierte Größe einer Datei, die komprimierte Größe einer Datei und die Gesamtgröße des Archivs hat, sowie eine Begrenzung von 65.535 (2^16-1) Dateien im Archiv. ZIP64-Format-Erweiterungen erhöhen die Grenzen auf 2^64.

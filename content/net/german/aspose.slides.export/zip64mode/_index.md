## Zip64Mode Aufzählung

Gibt an, wann ZIP64-Format-Erweiterungen für eine OpenXML-Datei verwendet werden sollen.

```csharp
public enum Zip64Mode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Never | `0` | Verwenden Sie die ZIP64-Format-Erweiterungen nicht. |
| IfNecessary | `1` | Verwenden Sie die ZIP64-Format-Erweiterungen, falls erforderlich. |
| Always | `2` | Verwenden Sie immer die ZIP64-Format-Erweiterungen. |

### Hinweise

Eine OpenXML-Datei ist ein ZIP-Archiv, das eine Grenze von 4 GB (2^32 Bytes) für die unkomprimierte Dateigröße, die komprimierte Dateigröße und die Gesamtegröße des Archivs hat, sowie eine Grenze von 65 535 (2^16-1) Dateien im Archiv. ZIP64-Format-Erweiterungen erhöhen die Grenzen auf 2^64.

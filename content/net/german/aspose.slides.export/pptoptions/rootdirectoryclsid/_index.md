## PptOptions.RootDirectoryClsid property

Stellt die Objektklassen-GUID (CLSID) dar, die im Stammverzeichniseintrag gespeichert ist. Kann für die COM -Aktivierung der Anwendung des Dokuments verwendet werden. Der Standardwert ist „64818D11-4F9B-11CF-86EA-00AA00B929E8“, was „Microsoft Powerpoint.Slide.8“ entspricht.

```csharp
public Guid RootDirectoryClsid { get; set; }
```

### Beispiele

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    PptOptions pptOptions = new PptOptions();
    
    /// CLSID auf 'Microsoft Powerpoint.Show.8' setzen
    pptOptions.RootDirectoryClsid = new Guid("64818D10-4F9B-11CF-86EA-00AA00B929E8");
    
    pres.Save("pres.ppt", SaveFormat.Ppt, pptOptions);
}
```

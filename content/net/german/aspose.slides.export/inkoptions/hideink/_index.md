## InkOptions.HideInk property

Zeigt oder verbirgt Tintelemente im exportierten Dokument.

```csharp
public bool HideInk { get; set; }
```

### Bemerkungen

Der Standardwert ist false.

### Beispiele

Das nächste Beispiel demonstriert, wie man Tintelemente im exportierten PDF-Dokument verbirgt:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.HideInk = true;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

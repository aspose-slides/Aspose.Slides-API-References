## SVGOptions.DisableLineEndCropping property

SVG 1.1 kann keine Einfügungen für Markierungen definieren. Die SVG-Schreibmaschine Aspose.Slides hat eine Problemumgehung für dieses Problem: Das Zeilenende wird mit einem Pfeil abgeschnitten, sodass die Linie die Markierungen nicht überlappt. Diese Option schaltet dieses Verhalten aus. Lesen/SchreibenBoolean .

public bool DisableLineEndCropping { get; set; }

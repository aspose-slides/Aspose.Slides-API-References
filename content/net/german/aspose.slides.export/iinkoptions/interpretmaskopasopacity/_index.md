## IInkOptions.InterpretMaskOpAsOpacity-Eigenschaft

Verwendet ROP-Operation oder Opazität zum Rendern von Pinseln.

```csharp
public bool InterpretMaskOpAsOpacity { get; set; }
```

### Bemerkungen

Standardwert ist true.

### Beispiele

Das nächste Beispiel demonstriert, wie man ROP für den Export von Tintelementen verwendet:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.InterpretMaskOpAsOpacity = false;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

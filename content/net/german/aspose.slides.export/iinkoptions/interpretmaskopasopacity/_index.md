## IInkOptions.InterpretMaskOpAsOpacity Eigenschaft

Verwendet ROP-Operation oder Opazität für die Rendering-Pinsel.

```csharp
public bool InterpretMaskOpAsOpacity { get; set; }
```

### Bemerkungen

Der Standardwert ist true.

### Beispiele

Das nächste Beispiel zeigt, wie man ROP für den Export von Ink-Elementen verwendet:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.InterpretMaskOpAsOpacity = false;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

## PdfOptions.EmbedTrueTypeFontsForASCII-Eigenschaft

Bestimmt, ob Aspose.Slides gängige Schriftarten für ASCII (33..127 Codebereich) Texte einbetten wird. Schriftarten für Zeichencodes größer als 127 werden immer eingebettet. Die Liste der gängigen Schriftarten umfasst die Basis-14-Schriftarten von PDF und zusätzlich vom Benutzer angegebene Schriftarten. Lese-/Schreib-Boolean.

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### Bemerkungen

Standardwert ist **true**.

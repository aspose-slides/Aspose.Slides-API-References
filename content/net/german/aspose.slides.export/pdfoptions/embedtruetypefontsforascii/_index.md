## PdfOptions.EmbedTrueTypeFontsForASCII-Eigenschaft

Bestimmt, ob Aspose.Slides gängige Schriftarten für ASCII (33..127 Codebereich) Text einbetten wird. Schriftarten für Zeichencodes größer als 127 werden immer eingebettet. Die Liste der gängigen Schriftarten umfasst die 14 Basis-Schriftarten von PDF und zusätzliche benutzerspezifizierte Schriftarten. Lese-/Schreib-Boolean.

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### Anmerkungen

Standardwert ist **true**.

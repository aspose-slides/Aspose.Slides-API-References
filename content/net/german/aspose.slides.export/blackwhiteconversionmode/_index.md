## BlackWhiteConversionMode Aufzählung

Stellt Optionen bereit, die steuern, wie Bilder von Folien in bitonale Bilder konvertiert werden.

```csharp
public enum BlackWhiteConversionMode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Default | `0` | Gibt keinen Konversionsalgorithmus an. Der im TIFF-Codec implementierte Algorithmus wird verwendet. (Standard) |
| Dithering | `1` | Gibt den Dithering-Algorithmus (Floyd-Steinberg) an. |
| DitheringFloydSteinberg | `2` | Gibt den Floyd-Steinberg-Dithering-Algorithmus an. |

| AutoOtsu | `4` | Gibt den automatisch berechneten Otsu-Schwellwertalgorithmus an. |

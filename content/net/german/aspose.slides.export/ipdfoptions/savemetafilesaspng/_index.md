## IPdfOptions.SaveMetafilesAsPng property

True, um alle in einer Präsentation verwendeten Metadateien in PNG-Bilder zu konvertieren. Lesen/SchreibenBoolean .

```csharp
public bool SaveMetafilesAsPng { get; set; }
```

### Bemerkungen

Standard ist **Stimmt**. PDF-Dokument kann Vektorgrafiken und Rasterbilder enthalten. Wenn SaveMetafilesAsPng auf „true“ gesetzt ist, wird das Metafile-Quellbild in das Png-Format konvertiert und als -Rasterbild im PDF-Format gespeichert. Wenn SaveMetafilesAsPng auf false gesetzt ist, wird die Quell-Metadatei in Pdf-Vektorgrafiken konvertiert. Jeder Ansatz hat Vorteile und Nachteile. Wenn beispielsweise Metafile in PNG, konvertiert wird, ist ein gewisser Qualitätsverlust während der resultierenden -Dokumentskalierung möglich. Wenn Metafile in PDF-Vektorgrafiken konvertiert wird, dann sind Leistungsprobleme im PDF-Anzeigetool möglich.

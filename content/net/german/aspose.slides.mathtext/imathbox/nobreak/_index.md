## IMathBox.NoBreak Eigenschaft

Kein Umbruch. Diese Eigenschaft gibt die "unbrechbare" Eigenschaft des Objektkastens an. Wenn true, können innerhalb des Kastens keine Zeilenumbrüche auftreten. Dies kann wichtig für Operator-Emulatoren sein, die aus mehr als einem binären Operator bestehen. Wenn dieses Element nicht spezifiziert ist, können innerhalb des Kastens Umbrüche auftreten. Standard: true

```csharp
public bool NoBreak { get; set; }
```

### Beispiele

Beispiel:

```csharp
[C#]
IMathBox box = new MathematicalText("**********").ToBox();
box.NoBreak = false;
```

## IMathBox.NoBreak Eigenschaft

Kein Umbruch. Diese Eigenschaft gibt die "unbrechbare" Eigenschaft des Objektfelds an. Wenn wahr, können innerhalb des Felds keine Zeilenumbrüche erfolgen. Dies kann wichtig für Operator-Emulatoren sein, die aus mehr als einem binären Operator bestehen. Wenn dieses Element nicht angegeben ist, können Umbrüche im Feld auftreten. Standard: wahr

```csharp
public bool NoBreak { get; set; }
```

### Beispiele

Beispiel:

```csharp
[C#]
IMathBox box = new MathematicalText("**********").ToBox();
box.NoBreak = false;
```

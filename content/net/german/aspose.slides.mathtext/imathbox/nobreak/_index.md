## IMathBox.NoBreak property

Kein Umbruch. Diese Eigenschaft gibt die Eigenschaft "unzerbrechlich" für die Objektbox an. Wenn wahr, können innerhalb der Box keine Zeilenumbrüche auftreten. Dies kann für Operatoremulatoren wichtig sein, die aus mehr als einem binären Operator bestehen. Wenn dieses Element nicht angegeben ist, können Umbrüche innerhalb der Box auftreten. Standard: true

```csharp
public bool NoBreak { get; set; }
```

### Beispiele

Beispiel:

```csharp
[C#]
IMathBox box = new MathematicalText("**********").ToBox();
box.NoBreak = false;
```

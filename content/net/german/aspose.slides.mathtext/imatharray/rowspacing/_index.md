## IMathArray.RowSpacing-Eigenschaft

Abstand zwischen den Zeilen eines Arrays. Er wird nur verwendet, wenn RowSpacingRule auf 3 gesetzt ist. Genau in diesem Fall ist die Einheit Maßpunkte oder Multiple, in welchem Fall die Einheit halbe Zeilen beträgt. Standard: 0

```csharp
public uint RowSpacing { get; set; }
```

### Beispiele

Beispiel:

```csharp
[C#]
IMathArray mathArray = new MathArray(new MathematicalText("item1"));
mathArray.RowSpacingRule = MathRowSpacingRule.Exactly;
mathArray.RowSpacing = 10;
```

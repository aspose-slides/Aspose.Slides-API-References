## IMathBorderBox.StrikethroughBottomLeftToTopRight property

Von unten links nach oben rechts durchgestrichen (Standardwert ist „false“). Gibt den ausgeblendeten oder angezeigten Zustand einer durchgestrichenen diagonalen Linie von der unteren linken Ecke zur oberen rechten Ecke des Rahmenfelds an.

```csharp
public bool StrikethroughBottomLeftToTopRight { get; set; }
```

### Beispiele

Beispiel:

```csharp
[C#]
```

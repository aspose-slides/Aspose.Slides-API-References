## MathDelimiter.EndingCharacter-Eigenschaft

Der abschließende Delimiter gibt das abschließende oder schließende Delimiter-Zeichen an. Mathematische Delimiter sind umschließende Zeichen wie Klammern, eckige Klammern und geschweifte Klammern. Der Standardwert ist ')'.

```csharp
public char EndingCharacter { get; set; }
```

Beispiel:

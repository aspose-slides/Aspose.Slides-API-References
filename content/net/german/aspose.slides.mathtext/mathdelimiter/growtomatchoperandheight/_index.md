## MathDelimiter.GrowToMatchOperandHeight-Eigenschaft

Gibt das Wachstum von BeginningCharacter, SeparatorCharacter, EndingCharacter an. Wenn true, wachsen die Trennzeichen vertikal, um der Höhe ihres Operanden zu entsprechen. Der Standardwert ist true.

```csharp
public bool GrowToMatchOperandHeight { get; set; }
```

### Beispiele

Beispiel:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.GrowToMatchOperandHeight = false;
```

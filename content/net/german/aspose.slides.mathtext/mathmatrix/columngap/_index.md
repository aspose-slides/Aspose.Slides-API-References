## MathMatrix.ColumnGap-Eigenschaft

Der Wert des horizontalen Abstands zwischen Spalten einer Matrix; Wenn die ColumnGapRule auf 3 ("Genau") gesetzt ist, wird die Einheit als Twips (1/20 eines Punkts) interpretiert. Wenn die ColumnGapRule auf 4 ("Multiplikation") gesetzt ist, wird die Einheit als Anzahl der 0,5 em-Inkremente interpretiert. In anderen Fällen wird sie ignoriert. Standard: 0

```csharp
public uint ColumnGap { get; set; }
```

### Beispiele

Beispiel:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.ColumnGapRule = MathSpacingRules.Exactly;
matrix.ColumnGap = 20;
```

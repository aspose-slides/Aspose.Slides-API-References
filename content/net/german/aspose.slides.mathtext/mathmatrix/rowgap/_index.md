## MathMatrix.RowGap-Eigenschaft

Der Wert des vertikalen Abstands zwischen den Zeilen einer Matrix; Wenn die RowGapRule auf 3 ("Genau") gesetzt ist, wird die Einheit als Twips (1/20 eines Punkts) interpretiert. Wenn die RowGapRule auf 4 ("Mehrfach") gesetzt ist, wird die Einheit als Halblinien interpretiert. Standard: 0

```csharp
public uint RowGap { get; set; }
```

### Beispiele

Beispiel:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

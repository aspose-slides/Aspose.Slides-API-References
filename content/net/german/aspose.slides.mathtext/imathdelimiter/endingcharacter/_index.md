## IMathDelimiter.EndingCharacter-Eigenschaft

Delimiter Ending Character gibt das End- oder schließende Trennzeichen an. Mathematische Trennzeichen sind umschließende Zeichen wie Klammern, eckige Klammern und geschweifte Klammern. Der Standardwert: ')'.

```csharp
public char EndingCharacter { get; set; }
```

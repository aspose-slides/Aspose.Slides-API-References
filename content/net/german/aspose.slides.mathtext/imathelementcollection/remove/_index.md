## IMathElementCollection.Remove Methode  

Entfernt das erste Vorkommen eines bestimmten Objekts aus der Sammlung.  

```csharp  
public bool Remove(IMathElement item)  
```  

| Parameter | Typ | Beschreibung |  
| --- | --- | --- |  
| item | IMathElement | Das Objekt, das aus der Sammlung entfernt werden soll. |  

### Rückgabewert  

true, wenn *item* erfolgreich aus der Sammlung entfernt wurde; andernfalls false. Diese Methode gibt auch false zurück, wenn *item* in der ursprünglichen Sammlung nicht gefunden wird.  

### Beispiele  

Beispiel:  

```csharp  
[C#]  
IMathElementCollection collection = new MathBlock(new MathematicalText("x"));  
MathematicalText plusElement = new MathematicalText("+");  
collection.Add(plusElement);  
collection.Add(new MathRadical(new MathematicalText("x"), new MathematicalText("3")));  
collection.Remove(plusElement);  
```

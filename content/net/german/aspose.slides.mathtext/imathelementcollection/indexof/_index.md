## IMathElementCollection.IndexOf-Methode

Bestimmt den Index eines bestimmten mathematischen Elements in der Sammlung.

```csharp
public int IndexOf(IMathElement item)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| item | IMathElement | Das Element, das in der Sammlung gesucht wird. |

### Rückgabewert

Der Index von *item*, falls es in der Sammlung gefunden wird; andernfalls -1.

### Beispiele

Beispiel:

```csharp
[C#]
IMathElementCollection collection = new MathBlock(new MathematicalText("x"));
MathematicalText plusElement = new MathematicalText("+");
collection.Add(plusElement);
collection.Add(new MathRadical(new MathematicalText("x"), new MathematicalText("3")));
int index = collection.IndexOf(plusElement);
```

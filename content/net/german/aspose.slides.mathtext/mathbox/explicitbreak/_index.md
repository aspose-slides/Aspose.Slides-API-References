## MathBox.ExplicitBreak property

Explicit break gibt an, ob es einen Zeilenumbruch am Anfang des Box-Objekts gibt, sodass die Zeile am Anfang des Box-Objekts umbricht. Legt die Nummer des Operators in der vorherigen Zeile des mathematischen Textes fest, die als Ausrichtungspunkt für die aktuelle Zeile des mathematischen Textes verwendet werden soll. Mögliche Werte: 1..255 Standard: 0 (kein expliziter Umbruch)

```csharp
public byte ExplicitBreak { get; set; }
```

### Beispiele

Beispiel:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.ExplicitBreak = 1;
```

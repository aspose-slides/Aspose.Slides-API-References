## MathBox.NoBreak-Eigenschaft

Kein Umbruch Diese Eigenschaft gibt die "unzerbrechliche" Eigenschaft des Objektkästchens an. Wenn sie wahr ist, können keine Zeilenumbrüche innerhalb des Kästchens auftreten. Dies kann wichtig für Operator-Emulatoren sein, die aus mehr als einem binären Operator bestehen. Wenn dieses Element nicht angegeben ist, können innerhalb des Kästchens Umbrüche auftreten. Standard: wahr

```csharp
public bool NoBreak { get; set; }
```

### Beispiele

Beispiel:

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("*****"));
box.NoBreak = false;
```

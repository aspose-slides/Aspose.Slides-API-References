## MathBox.NoBreak-Eigenschaft

Kein Umbruch Diese Eigenschaft gibt die "unbrechbare" Eigenschaft des Objektfelds an. Wenn true, können innerhalb des Feldes keine Zeilenumbrüche auftreten. Dies kann wichtig für Operatorenemulatoren sein, die aus mehr als einem binären Operator bestehen. Wenn dieses Element nicht angegeben ist, können innerhalb des Feldes Umbrüche auftreten. Standard: true

```csharp
public bool NoBreak { get; set; }
```

### Beispiele

Beispiel:

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("*****"));
box.NoBreak = false;
```

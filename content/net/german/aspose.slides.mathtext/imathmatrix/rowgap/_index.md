## IMathMatrix.RowGap-Eigenschaft

Der Wert des vertikalen Abstands zwischen den Reihen einer Matrix; Wenn die RowGapRule auf 3 ("Genau") gesetzt ist, wird die Einheit als Twips (1/20 eines Punktes) interpretiert. Wenn die RowGapRule auf 4 ("Multipel") gesetzt ist, wird die Einheit als halbe Zeilen interpretiert. Standard: 0

```csharp
public uint RowGap { get; set; }
```

### Beispiele

Beispiel:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

## MathJustification-Aufzählung

Gibt die Rechtfertigung des mathematischen Absatzes an (eine Reihe benachbarter Instanzen von mathematischem Text innerhalb desselben Absatzes)

```csharp
public enum MathJustification
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| LeftJustified | `1` | Richtet mathematischen Text nach links aus |
| RightJustified | `2` | Richtet mathematischen Text nach rechts aus |
| Centered | `3` | Richtet mathematischen Text in der Mitte aus |
| CenteredAsGroup | `4` | Zentriert auf der Seite die gesamte Gruppe von mathematischen Blöcken als eine Einheit. Das bedeutet, dass die Instanzen von mathematischem Text in Bezug zueinander ausgerichtet werden können, aber die gesamte Gruppe von mathematischem Text als Ganze zentriert wird. |

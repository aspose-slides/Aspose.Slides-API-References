## IMathFraction-Schnittstelle

Spezifiziert das Bruchobjekt, das aus einem Zähler und einem Nenner besteht, die durch eine Bruchlinie getrennt sind. Die Bruchlinie kann horizontal oder diagonal sein, je nach den Bruch Eigenschaften. Das Bruchobjekt wird auch verwendet, um die Stapelfunktion darzustellen, die ein Element über ein anderes stellt, ohne eine Bruchlinie.

public interface IMathFraction : IMathElement

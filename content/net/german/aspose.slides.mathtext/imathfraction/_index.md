## IMathFraction-Schnittstelle  

Gibt das Bruchobjekt an, das aus einem Zähler und einem Nenner besteht, die durch eine Bruchlinie getrennt sind. Die Bruchlinie kann horizontal oder diagonal sein, abhängig von den Bruchteileigenschaften. Das Bruchobjekt wird auch verwendet, um die Stapelfunktion darzustellen, die ein Element über ein anderes ohne Bruchlinie legt.

```csharp  
public interface IMathFraction : IMathElement  
```

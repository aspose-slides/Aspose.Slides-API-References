## IEffect.StopPreviousSound-Eigenschaft

Dieses Attribut gibt an, ob der Animationseffekt den vorherigen Ton stoppt. Lese-/Schreib-Boolean.

```csharp
public bool StopPreviousSound { get; set; }
```

### Beispiele

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Holen Sie sich den ersten Effekt der ersten Folie.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Holen Sie sich den ersten Effekt der zweiten Folie.
    IEffect secondSlideEffect = presentation.Slides[1].Timeline.MainSequence[0];
       
    if (firstSlideEffect.Sound != null)
    {
        // Ändern Sie den zweiten Effekt Enhancements/Sound in "Stop Previous Sound"
        secondSlideEffect.StopPreviousSound = true;
    }
}
```

## Timing.RepeatUntilNextClick-Eigenschaft

Dieses Attribut gibt an, ob der Effekt bis zum nächsten Klick wiederholt wird. Lese-/Schreib-Boolean.

### Beispiele

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Holen Sie sich die Effekte-Sequenz für die erste Folie
```

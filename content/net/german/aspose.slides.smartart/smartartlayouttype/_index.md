## SmartArtLayoutType Enumeration

Repräsentiert den Layouttyp eines SmartArt-Diagramms.

```csharp
public enum SmartArtLayoutType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| AccentProcess | `0` | Verwendung, um eine Progression, einen Zeitplan oder sequenzielle Schritte in einer Aufgabe, einem Prozess oder einem Workflow darzustellen. Eignet sich gut zur Veranschaulichung sowohl von Text der Ebene 1 als auch der Ebene 2. |
| AccentedPicture | `1` | Verwendung, um eine zentrale, fotografische Idee mit verwandten Ideen an der Seite darzustellen. Der obere Ebene-1-Text erscheint über dem zentralen Bild. Der entsprechende Text für andere Ebene-1-Formen erscheint neben den kleinen, runden Bildern. Dieses Layout funktioniert auch gut ohne Text. |
| AlternatingFlow | `2` | Verwendung, um Gruppen von Informationen oder sequenzielle Schritte in einer Aufgabe, einem Prozess oder einem Workflow zu zeigen. Betont die Interaktion oder die Beziehungen zwischen den Gruppen von Informationen. |
| AlternatingHexagons | `3` | Verwendung, um eine Reihe von miteinander verbundenen Ideen darzustellen. Der Ebene-1-Text erscheint in den Hexagonen. Der Ebene-2-Text erscheint außerhalb der Formen. |
| AlternatingPictureBlocks | `4` | Verwendung, um eine Reihe von Bildern von oben nach unten darzustellen. Text erscheint abwechselnd rechts oder links vom Bild. |
| AlternatingPictureCircles | `5` | Verwendung, um eine Reihe von Bildern mit Text darzustellen. Der entsprechende Text erscheint in den zentralen Kreisen, und die Bilder wechseln von links nach rechts. |

Ungenutzter Text erscheint nicht, bleibt aber verfügbar, wenn Sie die Layouts wechseln. |

| BasicMatrix | `13` | Verwendung, um die Beziehung der Komponenten zu einem Ganzen in Quadranten darzustellen. Die ersten vier Zeilen des Ebene-1-Texts erscheinen in den Quadranten. Ungenutzter Text erscheint nicht, bleibt aber verfügbar, wenn Sie die Layouts wechseln. |
| BasicPie | `14` | Verwendung, um zu zeigen, wie Einzelteile ein Ganzes bilden. Die ersten sieben Zeilen des Ebene-1-Texts entsprechen den gleichmäßig verteilten Keil- oder Kuchengestalten. | BasicProcess | `15` | Verwendung, um eine Progression oder sequenzielle Schritte in einer Aufgabe, einem Prozess oder einem Workflow darzustellen. |

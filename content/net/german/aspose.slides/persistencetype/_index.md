## PersistenceType Aufzählung

Gibt die Methode an, die verwendet wird, um Eigenschaften des ActiveX-Steuerelements zu speichern.

```csharp
public enum PersistenceType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| NotDefined | `-1` | Persistenzid nicht spezifiziert. |
| PersistPropertyBag | `0` | Gibt an, dass das ActiveX-Steuerelement unter Verwendung einer eigenschaftsbasierenden Persistenz gespeichert wird. Die eigenschaftsbasierte Persistenz speichert ein ActiveX-Steuerelement mittels einer Sammlung von Namen- und Werte-Paaren, die die vom ActiveX-Steuerelement gespeicherten Daten angeben. |
| PersistStream | `1` | Gibt an, dass das ActiveX-Steuerelement unter Verwendung einer stream-basierten Persistenz gespeichert wird, die keine Initialisierung des ActiveX-Steuerelements in einen Standardzustand unterstützt. |
| PersistStreamInit | `2` | Gibt an, dass das ActiveX-Steuerelement unter Verwendung einer stream-basierten Persistenz gespeichert wird, die die Initialisierung des ActiveX-Steuerelements in einen Standardzustand unterstützt. |
| PersistStorage | `3` | Gibt an, dass das ActiveX-Steuerelement unter Verwendung einer speicherbasierten Persistenz gespeichert wird. |

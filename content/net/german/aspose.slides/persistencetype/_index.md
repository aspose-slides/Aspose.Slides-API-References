## PersistenceType Aufzählung

Gibt die Methode an, die verwendet wird, um die Eigenschaften des ActiveX-Steuerelements zu speichern.

```csharp
public enum PersistenceType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| NotDefined | `-1` | Persistenz-ID nicht angegeben. |
| PersistPropertyBag | `0` | Gibt an, dass das ActiveX-Steuerelement mit einer auf dem Property-Bag basierenden Persistenz gespeichert wird. Die auf dem Property-Bag basierende Persistenz speichert ein ActiveX-Steuerelement durch eine Sammlung von Namens- und Wertpaaren, die die vom ActiveX-Steuerelement gespeicherten Daten angeben. |
| PersistStream | `1` | Gibt an, dass das ActiveX-Steuerelement mit einer stream-basierten Persistenz gespeichert wird, die die Initialisierung des ActiveX-Steuerelements in einen Standardstatus nicht unterstützt. |
| PersistStreamInit | `2` | Gibt an, dass das ActiveX-Steuerelement mit einer stream-basierten Persistenz gespeichert wird, die die Initialisierung des ActiveX-Steuerelements in einen Standardstatus unterstützt. |
| PersistStorage | `3` | Gibt an, dass das ActiveX-Steuerelement mit einer speicherbasierten Persistenz gespeichert wird. |

## PresentationLockingBehavior Aufzählung

Stellt das Verhalten dar, das die Behandlung der [`IPresentation`](../ipresentation) Quelle (Datei oder Stream) beim Laden und Arbeiten mit einer Instanz von [`IPresentation`](../ipresentation) betrifft.

```csharp
public enum PresentationLockingBehavior
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| LoadAndRelease | `0` | Die Quelle wird nur für die Dauer der Ausführung des [`IPresentation`](../ipresentation) Konstruktors gesperrt. Wenn [`IsTemporaryFilesAllowed`](../iblobmanagementoptions/istemporaryfilesallowed) auf false gesetzt ist, werden alle BLOBs in den Speicher geladen. Andernfalls können andere Mittel wie temporäre Dateien verwendet werden. Dieses Verhalten ist langsamer als KeepLocked, und wenn es möglich ist, das Eigentum an der Quelle an [`IPresentation`](../ipresentation) zu übergeben, wird empfohlen, KeepLocked zu verwenden. |
| KeepLocked | `1` | Die Quelle wird für die gesamte Lebensdauer der [`IPresentation`](../ipresentation) Instanz gesperrt, bis sie verworfen wird. [`IsTemporaryFilesAllowed`](../iblobmanagementoptions/istemporaryfilesallowed) muss auf true gesetzt werden, um dieses Verhalten zu nutzen, andernfalls wird eine Ausnahme ausgelöst. Dieses Verhalten wird empfohlen, es ist schneller und verbraucht weniger Speicher als LoadAndRelease. |

### Bemerkungen

Die Quelle ist der Parameter, der an den [`IPresentation`](../ipresentation) Konstruktor übergeben wird. Im nachfolgenden Beispiel ist die Quelle die Datei „pres.pptx“:

```csharp
LoadOptions loadOptions = new LoadOptions { 
  BlobManagementOptions = { PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked } };
using (IPresentation pres = new Presentation("pres.pptx", loadOptions)) { }
```

Für dieses Beispiel wird die Quelle (Datei „pres.pptx“) für die Lebensdauer einer [`IPresentation`](../ipresentation) Instanz gesperrt, d. h. sie kann von einem anderen Prozess nicht geändert oder gelöscht werden.

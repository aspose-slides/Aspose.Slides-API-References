## IViewProperties.GridSpacing-Eigenschaft

Gibt den Rasterabstand zurück oder legt ihn fest, der für das Raster unter dem Präsentationsdokument in Punkten verwendet werden soll. Lese-/Schreibzugriff auf Single.

```csharp
public float GridSpacing { get; set; }
```

### Anmerkungen

Der Wert für den Rasterabstand muss eine positive Zahl sein. Der typische Wertebereich reicht von 1 mm (2,8349607 Punkte) bis 2 Zoll (144 Punkte).

### Beispiele

Der folgende Beispielcode zeigt, wie man den Rasterabstand in einer PowerPoint-Präsentation ändert.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

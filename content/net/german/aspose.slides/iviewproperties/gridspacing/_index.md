## IViewProperties.GridSpacing-Eigenschaft  

Gibt den Gitterabstand zurück oder setzt ihn, der für das Gitter unter dem Präsentationsdokument verwendet werden soll, in Punkten. Lesen/Schreiben Einzel.  

```csharp  
public float GridSpacing { get; set; }  
```  

### Hinweise  

Der Wert für den Gitterabstand muss eine positive Zahl sein. Der typische Wertebereich liegt zwischen 1 mm (2.8349607 Punkte) und 2 Zoll (144 Punkte).  

### Beispiele  

Der folgende Beispielcode zeigt, wie der Gitterabstand in einer PowerPoint-Präsentation geändert werden kann.  

```csharp  
[C#]  
using (Presentation pres = new Presentation())  
{  
    pres.ViewProperties.GridSpacing = 72f;  
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);  
}  
```

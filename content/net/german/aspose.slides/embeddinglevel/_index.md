## EmbeddingLevel-Aufzählung

Represents the licensing rights for embedding the font.

```csharp
[Flags]
public enum EmbeddingLevel : ushort
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Installable | `0` | Schriften mit dieser Einstellung zeigen an, dass sie von einer Anwendung in das entfernte System eingebettet und dauerhaft installiert werden dürfen. Der Benutzer des entfernten Systems erwirbt die gleichen Rechte, Pflichten und Lizenzen für diese Schrift wie der ursprüngliche Käufer der Schrift und unterliegt derselben Endbenutzer-Lizenzvereinbarung, Urheberrecht, Designpatent und/oder Markenregistrierung wie der ursprüngliche Käufer. |
| Restricted | `2` | Schriften, bei denen nur dieses Bit gesetzt ist, dürfen nicht geändert, eingebettet oder in irgendeiner Weise ausgetauscht werden, ohne vorher die Erlaubnis des Rechtsinhabers einzuholen. |
| PreviewPrint | `4` | Wenn dieses Bit gesetzt ist, darf die Schrift eingebettet und vorübergehend auf dem entfernten System geladen werden. Dokumente, die Preview &amp; Print-Schriften enthalten, müssen im schreibgeschützten Modus geöffnet werden; es können keine Änderungen am Dokument vorgenommen werden. |
| Editable | `8` | Wenn dieses Bit gesetzt ist, darf die Schrift eingebettet werden, darf jedoch nur temporär auf anderen Systemen installiert werden. Im Gegensatz zu Preview &amp; Print-Schriften dürfen Dokumente, die Editable-Schriften enthalten, zum Lesen geöffnet werden, das Bearbeiten ist erlaubt und Änderungen können gespeichert werden. |
| NoSubsetting | `100` | Wenn dieses Bit gesetzt ist, darf die Schrift vor dem Einbetten nicht unterteilt (subsetting) werden. Andere Einbettungsbeschränkungen, die in den Bits 0-3 und 9 angegeben sind, gelten ebenfalls. |
| BitmapOnly | `200` | Wenn dieses Bit gesetzt ist, dürfen nur Bitmaps, die in der Schrift enthalten sind, eingebettet werden. Konturinformationen dürfen nicht eingebettet werden. Gibt es keine Bitmaps in der Schrift, wird die Schrift als nicht einbettbar betrachtet und die Einbettungsdienste schlagen fehl. |

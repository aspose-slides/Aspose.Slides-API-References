## ShapeCollection.InsertSummaryZoomFrame method

Erstellt ein neues Zusammenfassungs-Zoom-Objekt und fügt es am angegebenen Index in eine Sammlung ein.

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| index | Int32 | Der nullbasierte Index, an dem der Schnittzoomrahmen eingefügt werden soll. |
| x | Single | X-Koordinate eines neuen Schnitt-ZoomrahmensSingle. |
| y | Single | Y-Koordinate eines neuen Schnitt-ZoomrahmensSingle. |
| width | Single | Breite eines neuen Schnitt-Zoom-RahmensSingle. |
| height | Single | Höhe eines neuen Schnitt-ZoomrahmensSingle. |

### Rückgabewert

Zusammenfassungs-Zoom-Objekt erstellt[`ISummaryZoomFrame`](../../isummaryzoomframe).

### Ausnahmen

| Ausnahme | Bedingung |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | Die Präsentation enthält keine Abschnitte oder die Zielfolie gehört zu keinem Abschnitt. |

### Bemerkungen

Diese Methode erstellt einen neuen Zusammenfassungszoom und fügt ihm eine Sammlung von Objekten für alle Abschnitte in dieser Präsentation hinzu.

### Beispiele

Dieses Beispiel demonstriert das Erstellen und Einfügen eines Zusammenfassungs-Zoom-Objekts am angegebenen Index einer Sammlung (vorausgesetzt, dass die Präsentation „Presentation.pptx“ mindestens zwei Abschnitte enthält):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSummaryZoomFrame(2, 150, 20, 50, 50);
}
```

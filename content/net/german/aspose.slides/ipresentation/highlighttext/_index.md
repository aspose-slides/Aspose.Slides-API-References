## HighlightText(string, Color) {#highlighttext}

Hebt alle Übereinstimmungen des Beispieltextes mit der angegebenen Farbe hervor.

```csharp
public void HighlightText(string text, Color highlightColor)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| text | String | Der hervorzuhebende Text. |
| highlightColor | Color | Die Farbe, um den Text hervorzuheben. |

### Beispiele

Das folgende Codebeispiel zeigt, wie man Text in einer PowerPoint-Präsentation hervorheben kann.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	// Alle getrennten 'the' Vorkommen hervorheben
	presentation.HighlightText("the", Color.Violet);
	presentation.Save("SomePresentation-out2.pptx", SaveFormat.Pptx);
}
```

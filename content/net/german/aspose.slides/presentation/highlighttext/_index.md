## HighlightText(string, Color) {#highlighttext}

Hebt alle Übereinstimmungen des Beispiels mit der angegebenen Farbe hervor.

```csharp
public void HighlightText(string text, Color highlightColor)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| text | String | Der Text, der hervorgehoben werden soll. |
| highlightColor | Color | Die Farbe, mit der der Text hervorgehoben werden soll. |

### Beispiele

Das folgende Codebeispiel zeigt, wie man Text in einer PowerPoint-Präsentation hervorhebt.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	// Hervorhebung aller separaten Vorkommen von 'the'
	presentation.HighlightText("the", Color.Violet);
	presentation.Save("SomePresentation-out2.pptx", SaveFormat.Pptx);
}
```

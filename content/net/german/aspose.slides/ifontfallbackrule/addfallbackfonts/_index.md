## AddFallBackFonts(string) {#addfallbackfonts}

Fügt eine neue(n) Schriftart(en) zur Liste der FallBack-Schriftarten hinzu.

```csharp
public void AddFallBackFonts(string fontName)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| fontName | String | Name oder Namen der Schriftart(en) (durch Kommas getrennt) für FallBack |

### Beispiele

```csharp
[C#]
//Erstellen einer neuen Instanz von FontFallBackRule
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho");

//Hinzufügen einer zweiten Schriftart zur Regel
newRule.AddFallBackFonts("MS Gothic");
//Hinzufügen einer dritten und vierten Schriftart zur Regel
newRule.AddFallBackFonts("Tahoma, Times New Roman");
```


| Parameter  | Typ    | Beschreibung                           |
|------------|--------|----------------------------------------|
| startIndex | Int32  | Ein Index der ersten zu hinzufügenden Schriftart. |
| count      | Int32  | Anzahl der hinzuzufügenden Schriftarten. |

### Rückgabewert

Array von String

### Beispiele

```csharp
[C#]
```

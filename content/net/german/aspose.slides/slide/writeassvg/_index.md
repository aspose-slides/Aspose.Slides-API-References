
| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| stream | Stream | Zielstream |

### Beispiele

Das folgende Codebeispiel zeigt, wie man die erste Folie aus einer PowerPoint-Präsentation in eine SVG-Datei konvertiert.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.svg"))
    {
        // Speichert die erste Folie als SVG-Datei
        pres.Slides[0].WriteAsSvg(fileStream);
    }
}
```

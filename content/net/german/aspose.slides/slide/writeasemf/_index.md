## Slide.WriteAsEmf-Methode

| Ausnahme | Bedingung |
| --- | --- |

### Beispiele

Das folgende Codebeispiel zeigt, wie die erste Folie einer PowerPoint-Präsentation in eine Metadatei konvertiert wird.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.emf"))
    {
        // Speichert die erste Folie als Metadatei
        pres.Slides[0].WriteAsEmf(fileStream);
    }
}
```

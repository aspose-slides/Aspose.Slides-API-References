## ICommonSlideViewProperties.DrawingGuides-Eigenschaft

Gibt die Sammlung der Zeichnungsrichtlinien zurück. Nur lesezugängliche [`IDrawingGuidesCollection`](../../idrawingguidescollection)

### Beispiele

Der folgende Beispielcode zeigt, wie man die neuen Zeichnungsrichtlinien in eine PowerPoint-Präsentation hinzufügt.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    var slideSize = pres.SlideSize.Size;
    
    IDrawingGuidesCollection guides = pres.ViewProperties.SlideViewProperties.DrawingGuides;
    // Hinzufügen der neuen vertikalen Zeichnungsrichtlinie rechts vom Folienzentrum
    guides.Add(Orientation.Vertical, slideSize.Width / 2 + 12.5f);
    // Hinzufügen der neuen horizontalen Zeichnungsrichtlinie unterhalb des Folienzentrums
    guides.Add(Orientation.Horizontal, slideSize.Height / 2 + 12.5f);
    
    pres.Save("DrawingGuides_out.pptx", SaveFormat.Pptx);
}
```

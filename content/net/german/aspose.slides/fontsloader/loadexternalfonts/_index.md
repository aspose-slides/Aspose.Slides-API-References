## FontsLoader.LoadExternalFonts-Methode

Fügt zusätzliche Ordner hinzu, um Schriftarten zu suchen.

```csharp
public static void LoadExternalFonts(string[] directories)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| directories | String[] | Verzeichnisse zum Lesen zusätzlicher Schriftarten. |

### Beispiele

Die folgenden Beispiele zeigen, wie man benutzerdefinierte Schriftarten von .TTF lädt.

```csharp
[C#]
// Der Pfad zum Dokumentenverzeichnis.
string dataDir = "C:\\";
// Ordner, um Schriftarten zu suchen
String[] folders = new String[] { dataDir };
// Lade die benutzerdefinierten Schriftartenverzeichnis-Schriftarten
FontsLoader.LoadExternalFonts(folders);
// Führen Sie einige Arbeiten aus und rendern Sie die Präsentation/Slides
using (Presentation presentation = new Presentation(dataDir + "DefaultFonts.pptx"))
{
    presentation.Save(dataDir + "NewFonts_out.pptx", SaveFormat.Pptx);
	// Cache der Schriftarten leeren
	FontsLoader.ClearCache();
}
```

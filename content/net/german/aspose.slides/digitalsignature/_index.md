### Beispiele

Das folgende Beispiel demonstriert, wie man eine digitale Signatur aus einem PFX-Zertifikat in einer PowerPoint-Präsentation hinzufügt.

```csharp
[C#]
// Präsentationsinstanz initialisieren
using (Presentation pres = new Presentation())
{
    // DigitalSignature-Objekt mit PFX-Datei und PFX-Passwort erstellen
    DigitalSignature signature = new DigitalSignature("testsignature1.pfx", @"testpass1");
    // Kommentar zur neuen digitalen Signatur hinzufügen
    signature.Comments = "Aspose.Slides digitale Signatur-Test.";
    // Digitale Signatur zur Präsentation hinzufügen
    pres.DigitalSignatures.Add(signature);
    // Präsentation speichern
    pres.Save("SomePresentationSigned.pptx", SaveFormat.Pptx);
}
```

Der folgende Beispielcode demonstriert, wie man die digitale Signatur einer PowerPoint-Präsentation validiert.

```csharp
[C#]
// Präsentationsinstanz initialisieren
using (Presentation pres = new Presentation("SomePresentationSigned.pptx"))
{
    if (pres.DigitalSignatures.Count < 0)
    {
        bool allSignaturesAreValid = true;
        Console.WriteLine("Signaturen, die zur Signierung der Präsentation verwendet wurden: ");
        // Überprüfen, ob alle digitalen Signaturen gültig sind
        foreach (DigitalSignature signature in pres.DigitalSignatures)
        {
            Console.WriteLine(signature.Certificate.SubjectName.Name + ", "
                    + signature.SignTime.ToString("yyyy-MM-dd HH:mm") + " -- " + (signature.IsValid ? "GÜLTIG" : "UNGÜLTIG"));
            allSignaturesAreValid &= signature.IsValid;
        }
        if (allSignaturesAreValid)
            Console.WriteLine("Präsentation ist echt, alle Signaturen sind gültig.");
        else
            Console.WriteLine("Präsentation wurde seit der Signierung geändert.");
    }
}
```

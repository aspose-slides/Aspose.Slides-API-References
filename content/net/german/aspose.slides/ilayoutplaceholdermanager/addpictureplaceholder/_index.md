## ILayoutPlaceholderManager.AddPicturePlaceholder-Methode  
  
Fügt der Layout-Folie eine neue Platzhalterform hinzu, um ein Bild zu halten.  
  
```csharp  
public IAutoShape AddPicturePlaceholder(float x, float y, float width, float height)  
```  
  
| Parameter | Typ | Beschreibung |  
| --- | --- | --- |  
| x | Einzel | Die X-Koordinate der neuen Platzhalterform. |  
| y | Einzel | Die Y-Koordinate der neuen Platzhalterform. |  
| width | Einzel | Die Breite der neuen Platzhalterform. |  
| height | Einzel | Die Höhe der neuen Platzhalterform. |  
  
### Rückgabewert  
  
Erstellt [`IAutoShape`](../../iautoshape) mit einem Bild-Platzhalter.  
  
### Beispiele  
  
Das folgende Beispiel zeigt, wie man die Bild-Platzhalterform zur Layout-Folie hinzufügt.  
  
```csharp  
[C#]  
using (Presentation pres = new Presentation())  
{  
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);  
	IAutoShape placeholder = layout.PlaceholderManager.AddPicturePlaceholder(20, 20, 200, 200);  
}  
```

## ShapeThumbnailBounds enumeration

Aufzählung der Arten von Begrenzungen für Miniaturansichten von Formen.

```csharp
public enum ShapeThumbnailBounds
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Slide | `0` | Form-Miniaturansicht hat die gleiche Größe wie die Foliengröße. Formposition wird gespeichert. |
| Shape | `1` | Form-Miniaturansichten haben eine Größe, die gleich dem Formbegrenzungsrechteck ist, wobei die Formumrisseinstellungen berücksichtigt werden. |

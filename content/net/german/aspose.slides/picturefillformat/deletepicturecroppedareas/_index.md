## PictureFillFormat.DeletePictureCroppedAreas-Methode

Löscht die zugeschnittenen Bereiche des Füllbildes.

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### Rückgabewert

Zugeschnittenes Bild oder Originalbild, wenn das Beschneiden nicht erforderlich ist.

### Anmerkungen

Diese Methode konvertiert WMF/EMF-Metadateien in Raster-PNG-Bilder während des Beschneidens.

### Beispiele

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Erhält das PictureFrame
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // Löscht die zugeschnittenen Bereiche des PictureFrame-Bildes
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

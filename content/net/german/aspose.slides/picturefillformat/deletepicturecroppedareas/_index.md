## PictureFillFormat.DeletePictureCroppedAreas Methode

Löscht die beschnittenen Bereiche des Bildes.

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### Rückgabewert

Beschnittenes Bild oder Originalbild, wenn das Zuschneiden nicht erforderlich ist.

### Bemerkungen

Diese Methode konvertiert WMF/EMF Metadateien in ein Raster-PNG-Bild während des Zuschneidens.

### Beispiele

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Erhält den PictureFrame
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // Löscht die beschnittenen Bereiche des PictureFrame-Bildes
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

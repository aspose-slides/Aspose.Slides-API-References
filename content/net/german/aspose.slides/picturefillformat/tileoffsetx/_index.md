## PictureFillFormat.TileOffsetX-Eigenschaft

Gibt den horizontalen Offset der Textur vom Ursprung der Form in Punkten zurück oder setzt ihn. Ein positiver Wert bewegt die Textur nach rechts, während ein negativer Wert sie nach links bewegt. Lesen/Schreiben Einzelgänger.

```csharp
public float TileOffsetX { get; set; }
```

### Beispiele

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Ruft das Bildfüllformat der Form ab
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Setzt den Bildfüllmodus auf Kacheln
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Setzt den horizontalen Offset der Textur auf 20 Punkte
    pictureFillFormat.TileOffsetX = 20f;
}
```

## PictureFillFormat.TileScaleX-Eigenschaft

Gibt den horizontalen Maßstab für die Texturfüllung als Prozentsatz zurück oder legt ihn fest. Lese-/Schreibzugriff auf Single.

```csharp
public float TileScaleX { get; set; }
```

### Beispiele

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Holt das Picture Fill Format der Form
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Setzt den Picture Fill Modus auf Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Legt den horizontalen Maßstab für die Textur auf 120 Prozent fest
    pictureFillFormat.TileScaleX = 120;
}
```

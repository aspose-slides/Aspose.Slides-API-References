## IPictureFillFormat.DeletePictureCroppedAreas-Methode

Löscht die beschnittenen Bereiche des Bildfüllung.

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### Rückgabewert

Beschnittenes Bild oder Ursprungsbild, wenn kein Beschnitt erforderlich ist.

### Bemerkungen

Diese Methode konvertiert WMF/EMF-Metadateien in ein Raster-PNG-Bild, während sie beschnitten werden.

### Beispiele

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Ruft den PictureFrame ab
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // Löscht die beschnittenen Bereiche des Bildes im PictureFrame
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

## IPictureFillFormat.DeletePictureCroppedAreas-Methode  
  
Löscht zugeschnittene Bereiche des Füllbilds.  
  
```csharp  
public IPPImage DeletePictureCroppedAreas()  
```  
  
### Rückgabewert  
  
Zugeschnittenes Bild oder Ursprungsbild, wenn kein Zuschnitt erforderlich ist.  
  
### Bemerkungen  
  
Diese Methode konvertiert WMF/EMF-Metadateien in ein Raster-PNG-Bild und schneidet dabei zu.  
  
### Beispiele  
  
```csharp  
[C#]  
using (Presentation presentation = new Presentation("demo.pptx"))  
{  
    ISlide slide = presentation.Slides[0];  
  
    // Holt den PictureFrame  
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;  
  
    // Löscht die zugeschnittenen Bereiche des PictureFrame-Bildes  
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();  
}  
```

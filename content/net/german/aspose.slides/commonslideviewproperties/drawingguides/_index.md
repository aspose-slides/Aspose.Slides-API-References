## CommonSlideViewProperties.DrawingGuides-Eigenschaft

Gibt die Sammlung der Zeichnungsleitfäden zurück. Nur lesbar [`IDrawingGuidesCollection`](../../idrawingguidescollection)

### Beispiele

Der folgende Beispielcode zeigt, wie man die neuen Zeichnungsleitfäden in einer PowerPoint-Präsentation hinzufügt.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    var slideSize = pres.SlideSize.Size;
    
    IDrawingGuidesCollection guides = pres.ViewProperties.SlideViewProperties.DrawingGuides;
    // Hinzufügen des neuen vertikalen Zeichnungsleitfadens rechts von der Folienmitte
    guides.Add(Orientation.Vertical, slideSize.Width / 2 + 12.5f);
    // Hinzufügen des neuen horizontalen Zeichnungsleitfadens unterhalb der Folienmitte
    guides.Add(Orientation.Horizontal, slideSize.Height / 2 + 12.5f);
    
    pres.Save("DrawingGuides_out.pptx", SaveFormat.Pptx);
}
```

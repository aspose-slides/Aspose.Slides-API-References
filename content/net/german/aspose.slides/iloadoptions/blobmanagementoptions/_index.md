## ILoadOptions.BlobManagementOptions-Eigenschaft

Stellt die Optionen dar, die verwendet werden können, um das Verhalten der Verwaltung von Binary Large Objects (BLOBs) zu steuern, wie z.B. die Verwendung von temporären Dateien oder das maximale BLOB-Bytes im Speicher. Diese Optionen sind vorgesehen, um das beste Verhältnis von Leistung zu Speicherverbrauch für eine bestimmte Umgebung oder Anforderungen einzurichten. Ein Binary Large Object (BLOB) ist ein binärer Datensatz, der als eine einzige Einheit gespeichert wird - d.h. ein BLOB kann ein Audio, ein Video oder die Präsentation selbst sein.

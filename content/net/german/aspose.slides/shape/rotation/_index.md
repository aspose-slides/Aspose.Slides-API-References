## Shape.Rotation property

Gibt die Gradzahl zurück, um die die angegebene Form um die z-Achse gedreht wird, oder legt sie fest. Ein positiver Wert zeigt eine Drehung im Uhrzeigersinn an; ein negativer Wert zeigt eine Drehung gegen den Uhrzeigersinn an. Lesen/SchreibenSingle .

```csharp
public float Rotation { get; set; }
```

### Bemerkungen

Rückgabewert ist immer definiert (ist nicht float.NaN). Zugewiesener Wert muss definiert sein (nicht float.NaN). Sie können undefinierte Werte für RawFrame-Instanzeigenschaften festlegen.

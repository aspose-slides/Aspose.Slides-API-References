## OleObjectFrame.LinkPathRelative-Eigenschaft

Gibt den relativen Pfad zu einer verlinkten Datei zurück, wenn vorhanden, andernfalls einen leeren String. Schreibgeschützter String.

```csharp
public string LinkPathRelative { get; }
```

### Bemerkungen

In den Ppt-Präsentationen können einige Ole-Objektlinks eine relative Darstellung haben.

### Beispiele

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.ppt"))
{
    IOleObjectFrame oleFrame = presentation.Slides[0].Shapes[0] as IOleObjectFrame;

    if (oleFrame != null)
    {
        Console.WriteLine("Der relative Pfad: " + oleFrame.LinkPathRelative);
    } 
}
```

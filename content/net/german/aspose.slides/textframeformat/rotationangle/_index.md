## TextFrameFormat.RotationAngle-Eigenschaft

Gibt die benutzerdefinierte Drehung an, die auf den Text innerhalb des Begrenzungsrahmens angewendet wird. Wenn sie nicht angegeben ist, wird die Drehung der begleitenden Form verwendet. Wenn sie angegeben ist, wird sie unabhängig von der Form angewendet. Das bedeutet, dass die Form eine Drehung haben kann, zusätzlich zu der Anwendung einer Drehung auf den Text selbst. Der resultierende Wert der visuellen Textdrehung wird aus dieser Eigenschaft und dem vordefinierten vertikalen Typ in der Eigenschaft TextVerticalType zusammengefasst. Lese-/Schreibzugriff: Single.

```csharp
public float RotationAngle { get; set; }
```

### Beispiele

Betrachten Sie den Fall, in dem eine Form eine Drehung von 90 Grad im Uhrzeigersinn angewendet hat. Darüber hinaus hat der Textkörper selbst eine Drehung von -90 Grad gegen den Uhrzeigersinn angewendet. Dann würde die resultierende Form scheinbar gedreht erscheinen, aber der Text darin würde erscheinen, als wäre er nicht gedreht worden.

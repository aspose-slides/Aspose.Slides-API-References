## IZoomObject.ShowBackground property

Erhält oder legt einen Wert fest, der angibt, ob der Zoom den Hintergrund der Zielfolie verwendet. Lesen/SchreibenBoolean . Standardwert: true

```csharp
public bool ShowBackground { get; set; }
```

### Beispiele

Das Beispiel zeigt das Entfernen des Hintergrunds eines Bilds eines Zoom-Objekts:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
    zoomFrame.ShowBackground = false;
}
```

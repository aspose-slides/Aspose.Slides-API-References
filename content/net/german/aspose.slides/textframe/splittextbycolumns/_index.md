## TextFrame.SplitTextByColumns-Methode

Teilt den Textinhalt des [`ITextFrame`](../../itextframe) in ein Array von Zeichenfolgen, wobei jedes Element einer separaten Textspalte innerhalb des Rahmens entspricht.

```csharp
public string[] SplitTextByColumns()
```

### Rückgabewert

Ein Array von Zeichenfolgen, wobei jede Zeichenfolge den Textinhalt einer bestimmten Spalte im [`ITextFrame`](../../itextframe) repräsentiert.

### Hinweise

Wenn der Textrahmen keine mehreren Spalten enthält, hat das zurückgegebene Array ein einzelnes Element, das den gesamten Text enthält. Leere Spalten werden im Array als leere Zeichenfolgen dargestellt.

### Beispiele

Das folgende Beispiel zeigt, wie man `SplitTextByColumns` verwendet:

```csharp
using (Presentation pres = new Presentation("example.pptx"))
{
    // Holen Sie sich die erste Form auf der Folie und casten Sie sie zu ITextFrame
    ITextFrame textFrame = pres.Slides[0].Shapes[0] as ITextFrame;
    // Teilen Sie den Inhalt des Textrahmens in Spalten
    string[] columnsText = textFrame.SplitTextByColumns();
    // Drucken Sie den Text jeder Spalte in die Konsole
    foreach (string column in columnsText)
        Console.WriteLine(column);
}
```

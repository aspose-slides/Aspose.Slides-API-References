## IProtectionManager.CheckWriteProtection method

Legt fest, ob eine Präsentation passwortgeschützt ist und geändert werden kann.

```csharp
public bool CheckWriteProtection(string password)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| password | String | Das Passwort zur Überprüfung. |

### Rückgabewert

True, wenn das Passwort gültig ist; andernfalls falsch.

### Bemerkungen

1. Sie sollten das überprüfen[`IsWriteProtected`](../iswriteprotected) Eigenschaft, bevor diese Methode aufgerufen wird. 2. Wenn das Passwort null oder leer ist, gibt diese Methode false zurück.

### Beispiele

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

## LayoutPlaceholderManager.AddTablePlaceholder-Methode

Fügt ein neues Platzhalter-Shape zur Layout-Folie hinzu, um eine Tabelle zu halten.

```csharp
public IAutoShape AddTablePlaceholder(float x, float y, float width, float height)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| x | Single | Die X-Koordinate des neuen Platzhalter-Shapes. |
| y | Single | Die Y-Koordinate des neuen Platzhalter-Shapes. |
| width | Single | Die Breite des neuen Platzhalter-Shapes. |
| height | Single | Die Höhe des neuen Platzhalter-Shapes. |

### Rückgabewert

Erstellt [`IAutoShape`](../../iautoshape) mit einem Tabellen-Platzhalter.

### Beispiele

Das folgende Beispiel zeigt, wie man das Tabellen-Platzhalter-Shape zur Layout-Folie hinzufügt.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTablePlaceholder(20, 20, 500, 200);
}
```

## LayoutPlaceholderManager.AddPicturePlaceholder-Methode

Fügt der Layout-Folie eine neue Platzhalterform hinzu, um ein Bild zu halten.

```csharp
public IAutoShape AddPicturePlaceholder(float x, float y, float width, float height)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| x | Single | Die X-Koordinate der neuen Platzhalterform. |
| y | Single | Die Y-Koordinate der neuen Platzhalterform. |
| width | Single | Die Breite der neuen Platzhalterform. |
| height | Single | Die Höhe der neuen Platzhalterform. |

### Rückgabewert

Erstellt ein [`IAutoShape`](../../iautoshape) mit einem Bildplatzhalter.

### Beispiele

Das folgende Beispiel zeigt, wie man die Platzhalterform für das Bild zur Layout-Folie hinzufügt.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddPicturePlaceholder(20, 20, 200, 200);
}
```

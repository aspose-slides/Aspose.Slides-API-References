## ZoomObject.ReturnToParent property

Ruft das Navigationsverhalten in der Diashow ab oder legt es fest. Lesen/SchreibenBoolean . Standardwert: false

```csharp
public bool ReturnToParent { get; set; }
```

### Bemerkungen

True-Wert der Eigenschaft gibt die Rückkehr zum übergeordneten Navigationsverhalten in der Diashow an.

### Beispiele

Beispiel:

```csharp
[C#]
```

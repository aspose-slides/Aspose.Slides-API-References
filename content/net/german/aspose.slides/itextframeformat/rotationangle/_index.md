## ITextFrameFormat.RotationAngle-Eigenschaft

Gibt die benutzerdefinierte Drehung an, die auf den Text innerhalb des Begrenzungsrahmens angewendet wird. Wenn sie nicht angegeben ist, wird die Drehung der begleitenden Form verwendet. Wenn sie angegeben ist, wird sie unabhängig von der Form angewendet. Das bedeutet, dass der Form eine Drehung zusätzlich zur Drehung des Textes selbst angewendet werden kann. Der resultierende Wert der visuellen Textdrehung wird aus dieser Eigenschaft und dem vordefinierten vertikalen Typ in der Eigenschaft TextVerticalType zusammengefasst. Lese-/Schreibzugriff Single.

```csharp
public float RotationAngle { get; set; }
```

### Beispiele

Betrachten Sie den Fall, in dem einer Form eine Drehung von 90 Grad im Uhrzeigersinn zugewiesen wird. Zusätzlich hat der Textkörper selbst eine Drehung von -90 Grad gegen den Uhrzeigersinn. Dann würde die resultierende Form gedreht erscheinen, während der Text darin so aussehen würde, als wäre er gar nicht gedreht worden.

## LoadOptions.DeleteEmbeddedBinaryObjects-Eigenschaft  

Bestimmt, ob Aspose.Slides alle eingebetteten binären Objekte beim Laden der Präsentation löschen wird.  

Die Arten der eingebetteten binären Objekte:  

* VBA-Projekt [`VbaProject`](../../ipresentation/vbaproject)  
* OLE-Objekt eingebettete Daten [`EmbeddedFileData`](../../ioleembeddeddatainfo/embeddedfiledata)  
* ActiveX-Steuerelement binäre Daten [`ActiveXControlBinary`](../../icontrol/activexcontrolbinary)  

Lese-/Schreib-Boolean.  

```csharp  
public bool DeleteEmbeddedBinaryObjects { get; set; }  
```  

### Anmerkungen  

Standard ist **false**.  

### Beispiele  

Das folgende Beispiel zeigt, wie die Präsentation ohne eingebettete binäre Objekte geladen werden kann.  

```csharp  
[C#]  
LoadOptions loadOptions = new LoadOptions();  
loadOptions.DeleteEmbeddedBinaryObjects = true;  
using (Presentation pres = new Presentation("pres.ppt", loadOptions))  
{  
    pres.Save("output_WithoutBinaryObjects.ppt", SaveFormat.Ppt);  
}  
```

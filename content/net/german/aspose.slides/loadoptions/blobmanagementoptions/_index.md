## LoadOptions.BlobManagementOptions-Eigenschaft

Stellt die Optionen dar, die verwendet werden können, um das Verhalten der Verwaltung von Binary Large Objects (BLOBs) zu steuern, wie z. B. die Verwendung von temporären Dateien oder die maximalen BLOBs-Bytes im Speicher. Diese Optionen sollen das beste Verhältnis von Leistung zu Speichernutzung für eine bestimmte Umgebung oder Anforderungen festlegen. Ein Binary Large Object (BLOB) ist eine binäre Daten, die als eine einzige Entität gespeichert ist - d.h. ein BLOB kann ein Audio, Video oder selbst eine Präsentation sein.

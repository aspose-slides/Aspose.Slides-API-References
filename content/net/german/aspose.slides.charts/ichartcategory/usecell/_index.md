## IChartCategory.UseCell-Eigenschaft

Wenn wahr, ist die AsCell-Eigenschaft aktuell. Mit anderen Worten, das Arbeitsblatt wird zur Speicherung der Kategorie verwendet (dieser Fall unterstützt eine mehrstufige Kategorie). Wenn falsch, ist die AsLiteral-Eigenschaft aktuell. Mit anderen Worten, das Arbeitsblatt wird NICHT zur Speicherung der Kategorie verwendet (und dieser Fall unterstützt keine mehrstufigen Kategorien). Schreibgeschützter Boolean.

```csharp
public bool UseCell { get; }
```

### Bemerkungen

Um den Wert dieser Eigenschaft (für alle Kategorien in der Sammlung) zu ändern, setzen Sie einen neuen Wert für die ChartCategoryCollection.UseCells-Eigenschaft.

## IChartCategory.UseCell-Eigenschaft  

Wenn wahr, ist die AsCell-Eigenschaft aktuell. Mit anderen Worten, das Arbeitsblatt wird zum Speichern der Kategorie verwendet (dieser Fall unterstützt eine mehrstufige Kategorie). Wenn falsch, ist die AsLiteral-Eigenschaft aktuell. Mit anderen Worten, das Arbeitsblatt wird NICHT zum Speichern der Kategorie verwendet (und dieser Fall unterstützt keine mehrstufigen Kategorien). Nur lesbarer Boolean.  

```csharp  
public bool UseCell { get; }  
```  

### Hinweise  

Um den Wert dieser Eigenschaft (für alle Kategorien in der Sammlung) zu ändern, setzen Sie einen neuen Wert für die ChartCategoryCollection.UseCells-Eigenschaft.

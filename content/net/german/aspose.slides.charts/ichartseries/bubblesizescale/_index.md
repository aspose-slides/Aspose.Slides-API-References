## IChartSeries.BubbleSizeScale-Eigenschaft

Gibt den Maßstabfaktor für das Blasendiagramm an (kann zwischen 0 und 300 Prozent der Standardgröße liegen). Dies ist die Eigenschaft nicht nur dieser Serie, sondern aller Serien der übergeordneten Serienegruppe - dies ist eine Projektion der entsprechenden Gruppeneigenschaft. Daher ist diese Eigenschaft schreibgeschützt. Verwenden Sie die Eigenschaft ParentSeriesGroup, um auf die übergeordnete Serienegruppe zuzugreifen. Verwenden Sie die lese/schreibbare Eigenschaft ParentSeriesGroup.BubbleSizeScale, um den Wert zu ändern.

```csharp
public int BubbleSizeScale { get; }
```

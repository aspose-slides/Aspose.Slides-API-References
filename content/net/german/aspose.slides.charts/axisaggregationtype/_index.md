## AxisAggregationType Aufzählung

Stellt den Aggregationstyp der Kategorienachse dar. Wird nur mit Histogramm- oder HistogramPareto-Serien verwendet.

```csharp
public enum AxisAggregationType
```

### Werte

| Name | Value | Description |
| --- | --- | --- |
| ByCategory | `0` | Nach Kategorien aggregieren |
| Automatic | `1` | Kategorien automatisch aggregieren. |
| ByBinWidth | `2` | Kategorien nach Bin-Breite aggregieren. Die Eigenschaft IAxis.BinWidth muss gesetzt werden. |
| ByNumberOfBins | `3` | Kategorien nach Anzahl der Bins aggregieren. Die Eigenschaft IAxis.NumberOfBins muss gesetzt werden. |

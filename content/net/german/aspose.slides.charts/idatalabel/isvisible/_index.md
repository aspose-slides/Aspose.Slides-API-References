## IDataLabel.IsVisible Eigenschaft

Falsch bedeutet, dass das Datenlabel nicht sichtbar ist (und somit alle Show*-Flags (ShowValue, ...) falsch sind). Nur-lesbarer Boolean.

```csharp
public bool IsVisible { get; }
```

### Anmerkungen

Wenn das Datenlabel sichtbar ist, können Sie es mit der Hide()-Methode ausblenden. Wenn das Datenlabel jedoch nicht sichtbar ist (IsVisible ist falsch), können Sie das Datenlabel sichtbar machen, indem Sie die Show*-Flags (ShowValue, ...) auf den Wert wahr setzen.

## IDataLabel.IsVisible property

False bedeutet, dass das Datenlabel nicht sichtbar ist (und daher sind alle Show*-Flags (ShowValue, ...) falsch). Read-onlyBoolean .

```csharp
public bool IsVisible { get; }
```

### Bemerkungen

Wenn die Datenbeschriftung sichtbar ist, können Sie sie mit der Methode Hide() ausblenden. Wenn die Datenbeschriftung jedoch nicht sichtbar ist (IsVisible ist falsch), können Sie die Datenbeschriftung sichtbar machen, indem Sie Show*-Flags setzen (ShowValue, ...) in den wahren Zustand.

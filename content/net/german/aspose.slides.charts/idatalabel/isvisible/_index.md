## IDataLabel.IsVisible-Eigenschaft

False bedeutet, dass das Datenetikett nicht sichtbar ist (und somit alle Show*-Flags (ShowValue, ...) false sind). Nur lesbare Boolean.

```csharp
public bool IsVisible { get; }
```

### Anmerkungen

Wenn das Datenetikett sichtbar ist, können Sie es mit der Hide()-Methode ausblenden. Wenn das Datenetikett jedoch nicht sichtbar ist (IsVisible ist false), können Sie das Datenetikett sichtbar machen, indem Sie die Show*-Flags (ShowValue, ...) auf den Wert true setzen.

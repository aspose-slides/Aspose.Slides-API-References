## IDataLabel.Hide-Methode

Machen Sie das Datenlabel unsichtbar, indem Sie alle Show*-Flags (ShowValue, ...) auf den falschen Zustand setzen. IsVisible wird danach false sein.

```csharp
public void Hide()
```

### Bemerkungen

Wenn das Datenlabel nicht sichtbar ist (IsVisible ist false), können Sie das Datenlabel sichtbar machen, indem Sie die Show*-Flags (ShowValue, ...) auf den wahren Zustand setzen.

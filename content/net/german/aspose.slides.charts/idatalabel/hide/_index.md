## IDataLabel.Hide-Methode

Machen Sie die Datenbeschriftung unsichtbar, indem Sie alle Show*-Flags (ShowValue, ...) auf den false-Zustand setzen. IsVisible wird danach false sein.

```csharp
public void Hide()
```

### Anmerkungen

Wenn die Datenbeschriftung nicht sichtbar ist (IsVisible ist false), können Sie die Datenbeschriftung sichtbar machen, indem Sie die Show*-Flags (ShowValue, ...) auf den true-Zustand setzen.

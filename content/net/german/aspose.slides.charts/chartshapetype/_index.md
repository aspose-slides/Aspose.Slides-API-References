## ChartShapeType Aufzählung

Stellt eine Form eines Diagramms dar.


### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| NotDefined | `-1` | Die Diagrammform ist nicht definiert. |
| Box | `0` | Gibt an, dass das Diagramm mit einer Boxform dargestellt werden soll. |
| Cone | `1` | Gibt an, dass das Diagramm als Kegel dargestellt werden soll, wobei die Basis des Kegels auf dem Boden und die Spitze des Kegels an der Oberseite des Datenmarkers liegt. |
| ConeToMax | `2` | Gibt an, dass das Diagramm mit abgeschnittenen Kegeln dargestellt werden soll, so dass die Spitze des Kegels den maximalen Datenwert darstellt. |
| Cylinder | `3` | Gibt an, dass das Diagramm als Zylinder dargestellt werden soll. |
| Pyramid | `4` | Gibt an, dass das Diagramm als rechteckige Pyramide dargestellt werden soll, wobei die Basis der Pyramide auf dem Boden und die Spitze der Pyramide an der Oberseite des Datenmarkers liegt. |
| PyramidToMaximum | `5` | Gibt an, dass das Diagramm mit abgeschnittenen Kegeln dargestellt werden soll, so dass die Spitze des Kegels den maximalen Datenwert darstellt. |

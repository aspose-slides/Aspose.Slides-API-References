## IChartTextBlockFormat.RotationAngle-Eigenschaft

Gibt die benutzerdefinierte Rotation an, die auf den Text innerhalb des Begrenzungsrahmens angewendet wird. Wenn dies nicht spezifiziert ist, wird die Rotation der begleitenden Form verwendet. Wenn es spezifiziert ist, wird dies unabhängig von der Form angewendet. Das heißt, die Form kann eine zusätzliche Rotation haben, während der Text selbst eine eigene Rotation hat. Der resultierende Wert der visuellen Textrotation wird aus dieser Eigenschaft und dem vordefinierten vertikalen Typ in der Eigenschaft TextVerticalType zusammengefasst. Lese-/schreibbar Einzelwert.

```csharp
public float RotationAngle { get; set; }
```

### Beispiele

Betrachten Sie den Fall, in dem eine Form eine Rotation von 90 Grad im Uhrzeigersinn hat. Darüber hinaus hat der Textkörper selbst eine Rotation von -90 Grad gegen den Uhrzeigersinn. Dann würde die resultierende Form gedreht erscheinen, aber der Text darin würde so erscheinen, als wäre er überhaupt nicht gedreht worden.

## IChartTextBlockFormat.RotationAngle-Eigenschaft

Gibt die benutzerdefinierte Rotation an, die auf den Text innerhalb des Begrenzungsrahmens angewendet wird. Wenn sie nicht angegeben ist, wird die Rotation der begleitenden Form verwendet. Wenn sie angegeben ist, wird dies unabhängig von der Form angewendet. Das bedeutet, dass die Form eine Rotation zusätzlich zur Rotation des Textes selbst haben kann. Der resultierende Wert der visuellen Textrotation wird aus dieser Eigenschaft und dem vordefinierten vertikalen Typ in der Eigenschaft TextVerticalType zusammengefasst. Lese-/Schreibzugriff Single.

```csharp
public float RotationAngle { get; set; }
```

### Beispiele

Betrachten Sie den Fall, in dem eine Form eine Rotation von 90 Grad im Uhrzeigersinn aufweist. Zusätzlich dazu hat der Textkörper selbst eine Rotation von -90 Grad gegen den Uhrzeigersinn. Dann würde die resultierende Form rotiert erscheinen, aber der Text darin würde so aussehen, als ob er überhaupt nicht rotiert wurde.

## IDataLabelCollection.IsVisible Eigenschaft

False bedeutet, dass das Datenlabel standardmäßig nicht sichtbar ist (und somit alle Show*-Flags (ShowValue, ...) der Eigenschaft DefaultDataLabelFormat false sind). Nur lesbarer Boolean.

```csharp
public bool IsVisible { get; }
```

### Bemerkungen

Wenn das Datenlabel standardmäßig sichtbar ist, können Sie es mit der Methode Hide() standardmäßig ausblenden. Wenn das Datenlabel jedoch standardmäßig nicht sichtbar ist (IsVisible ist false), können Sie das Datenlabel "standardmäßig sichtbar" machen, indem Sie die Show*-Flags (ShowValue, ...) der Eigenschaft DefaultDataLabelFormat auf den Status true setzen.

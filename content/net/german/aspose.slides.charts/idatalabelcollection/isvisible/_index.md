## IDataLabelCollection.IsVisible property

False bedeutet, dass die Datenbeschriftung standardmäßig nicht sichtbar ist (und daher sind alle Show*-Flags (ShowValue, ...) der DefaultDataLabelFormat-Eigenschaft falsch). Read-onlyBoolean .

```csharp
public bool IsVisible { get; }
```

### Bemerkungen

Wenn die Datenbeschriftung standardmäßig sichtbar ist, können Sie sie standardmäßig mit der Methode Hide() ausblenden. Wenn die Datenbeschriftung jedoch standardmäßig nicht sichtbar ist (IsVisible ist falsch), können Sie die Datenbeschriftung mit der Einstellung Show "sichtbar standardmäßig" machen *-Flags (ShowValue, ...) der DefaultDataLabelFormat-Eigenschaft auf True-Zustand.

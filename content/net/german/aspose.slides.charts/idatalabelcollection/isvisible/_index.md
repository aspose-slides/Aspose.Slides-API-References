## IDataLabelCollection.IsVisible-Eigenschaft

False bedeutet, dass das Datenlabel standardmäßig nicht sichtbar ist (und somit alle Show*-Flags (ShowValue, ...) der DefaultDataLabelFormat-Eigenschaft false sind). Schreibgeschützter Boolean.

```csharp
public bool IsVisible { get; }
```

### Hinweise

Wenn das Datenlabel standardmäßig sichtbar ist, können Sie es standardmäßig mit der Methode Hide() ausblenden. Wenn das Datenlabel jedoch standardmäßig nicht sichtbar ist (IsVisible ist false), können Sie das Datenlabel "standardmäßig sichtbar" machen, indem Sie die Show*-Flags (ShowValue, ...) der DefaultDataLabelFormat-Eigenschaft auf den Wert true setzen.

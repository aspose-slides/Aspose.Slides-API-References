## IChartData.SeriesGroups-Eigenschaft

Erhält die Gruppen von Serien. Schreibgeschützte [`IChartSeriesGroupCollection`](../../ichartseriesgroupcollection).

public IChartSeriesGroupCollection SeriesGroups { get; }

### Anmerkungen

1) Jede Gruppe von Serien enthält Serien mit kombinierbaren Typen. Gruppen von kombinierbaren Serientypen sind durch das Enum CombinableSeriesTypesGroup definiert und beschrieben. Außerdem enthält jede Gruppe von Serien Serien, die entweder auf den primären Achsen oder auf den sekundären Achsen (nicht in beiden Fällen in einer Gruppe) geplottet werden. Daher besteht das Prinzip der Seriengruppierung aus einer Gruppierung nach den oben genannten Typgruppen und nach dem Typ der primären/seconderen Darstellung. 2) Eine Gruppe von Serien enthält einige Eigenschaften, die für jede Serie in der Gruppe gemeinsam sind ("Seriengruppeneigenschaften"). "Seriengruppeneigenschaften" in der Klasse ChartSeriesGroup sind schreibbar. Jede der "Seriengruppeneigenschaften" kann eine schreibgeschützte Projektion in der Klasse ChartSeries haben.

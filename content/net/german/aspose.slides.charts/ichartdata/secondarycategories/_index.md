## IChartData.SecondaryCategories-Eigenschaft

Gibt die sekundären Kategorien zurück, wenn die Eigenschaft [`UseSecondaryCategories`](../usesecondarycategories) wahr ist. Schreibgeschützte [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### Bemerkungen

Wenn die Eigenschaft [`UseSecondaryCategories`](../usesecondarycategories) falsch ist, gibt die `SecondaryCategories`-Eigenschaft null zurück, und die Daten in der [`Categories`](../categories) Eigenschaft werden sowohl für primäre als auch für sekundäre Serien verwendet. Wenn die Eigenschaft [`UseSecondaryCategories`](../usesecondarycategories) wahr ist, werden die Daten in dieser `SecondaryCategories`-Eigenschaft für sekundäre Serien und die Daten in der [`Categories`](../categories) Eigenschaft für primäre Serien verwendet.

### Beispiele

Beispiel. Welche Kategorien sind mit der Serie verbunden - ChartData.Categories oder ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // zugehörige Kategorien sind series.Chart.ChartData.SecondaryCategories
}
else
{
    // zugehörige Kategorien sind series.Chart.ChartData.Categories
}
```

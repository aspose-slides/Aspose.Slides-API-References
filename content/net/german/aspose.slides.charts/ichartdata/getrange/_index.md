## IChartData.GetRange Methode

Gibt den Datenbereich des Diagramms zurück.

```csharp
public string GetRange()
```

### Rückgabewert

Formel des Zellen-Datenbereichs. Z.B: "Sheet1!$A$1:$C$4"

### Ausnahmen

| Ausnahme | Bedingung |
| --- | --- |
| InvalidOperationException | Diagramm verwendet kein Arbeitsbuch als Datenquelle |

### Beispiele

Beispiel C#

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.PercentsStackedBar, 100, 100, 500, 400);
    string result = (chart.ChartData as ChartData).GetRange();
}
```

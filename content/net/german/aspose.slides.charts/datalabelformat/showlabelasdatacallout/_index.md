## DataLabelFormat.ShowLabelAsDataCallout-Eigenschaft

Bestimmt, ob das Datenlabel des angegebenen Diagramms als Datenhinweis oder als Datenlabel angezeigt wird. Wenn der übergeordnete Teil dieses DataLabelFormat-Objekts eine DataLabelCollection-Sammlung von Datenlabels ist, wird dieser Wert für die neuen Datenlabels in der DataLabelCollection-Sammlung abgerufen oder gesetzt. Setzen Sie diesen Wert, wird dieser Wert auch für die ShowLabelAsDataCallout-Eigenschaft aller Datenlabels in der DataLabelCollection-Sammlung gesetzt (d.h. "DataLabels.DefaultDataLabelFormat.ShowLabelAsDataCallout = val;" bewirkt, dass alle DataLabels[i].ShowLabelAsDataCallout gleich val sind).

public bool ShowLabelAsDataCallout { get; set; }

## DataLabelFormat.NumberFormat-Eigenschaft  

Stellt den Format-String für das DataLabels-Objekt dar. Lese-/schreibbarer String.  

```csharp  
public string NumberFormat { get; set; }  
```  

### Bemerkungen  

Wenn der übergeordnete DataLabelFormat-Objekt eine DataLabelCollection-Sammlung von Datenbeschriftungen ist, dann ruft diese Eigenschaft den Standardwert der NumberFormat-Eigenschaft für die neuen Datenbeschriftungen in der DataLabelCollection-Sammlung ab oder legt ihn fest. Wenn diese Eigenschaft mit einem Wert festgelegt wird, wird dieser Wert auch für die NumberFormat-Eigenschaft für alle Datenbeschriftungen in der DataLabelCollection-Sammlung festgelegt (d.h. "DataLabels.DefaultDataLabelFormat.NumberFormat = val;" bewirkt, dass alle DataLabels[i].NumberFormat gleich val sind).  

### Beispiele  

```csharp  
[C#]  
series.Labels.DefaultDataLabelFormat.ShowValue = true;  
series.Labels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = false;  
series.Labels.DefaultDataLabelFormat.NumberFormat = "0.0%";  
```

## DataLabelFormat.ShowLeaderLines-Eigenschaft  
  
Stellt das Anzeigeverhalten der Führungsleitungen für die Datenbeschriftung eines bestimmten Diagramms dar. True zeigt die Führungsleitungen an. False, um sie auszublenden. Lese-/schreibbarer Boolean.  
  
```csharp  
public bool ShowLeaderLines { get; set; }  
```  
  
### Bemerkungen  
  
Wenn der übergeordnete Block dieses DataLabelFormat-Objekts eine DataLabelCollection von Datenbeschriftungen ist, dann erhält oder setzt diese Eigenschaft den Standardwert der ShowLeaderLines-Eigenschaft für die neuen Datenbeschriftungen in der DataLabelCollection. Wenn Sie diese Eigenschaft mit einem Wert festlegen, wird dieser Wert auch auf die ShowLeaderLines-Eigenschaft für alle Datenbeschriftungen in der DataLabelCollection gesetzt (d.h. "DataLabels.DefaultDataLabelFormat.ShowLeaderLines = val;" bewirkt, dass alle DataLabels[i].ShowLeaderLines gleich val ist).

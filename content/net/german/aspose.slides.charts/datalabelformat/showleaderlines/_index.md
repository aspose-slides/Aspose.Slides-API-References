## DataLabelFormat.ShowLeaderLines-Eigenschaft

Stellt das Verhalten der Anzeige von Führungsleitungen für die Datentextbeschriftung eines bestimmten Diagramms dar. True zeigt die Führungsleitungen an. False um sie auszublenden. Schreib-/Lese-Boolean.

```csharp
public bool ShowLeaderLines { get; set; }
```

### Hinweise

Wenn der Elternteil dieses DataLabelFormat-Objekts eine DataLabelCollection-Sammlung von Datentextbeschriftungen ist, wird dieser Eigenschaft der Standardwert der ShowLeaderLines-Eigenschaft für die neuen Datentextbeschriftungen in der DataLabelCollection-Sammlung zugewiesen oder gesetzt. Wenn Sie diese Eigenschaft mit einem Wert setzen, wird dieser Wert auch für die ShowLeaderLines-Eigenschaft aller Datentextbeschriftungen in der DataLabelCollection-Sammlung gesetzt (d.h. "DataLabels.DefaultDataLabelFormat.ShowLeaderLines = val;" bewirkt, dass alle DataLabels[i].ShowLeaderLines gleich val sind).

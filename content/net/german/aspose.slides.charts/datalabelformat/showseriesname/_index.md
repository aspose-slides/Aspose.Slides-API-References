## DataLabelFormat.ShowSeriesName property

Gibt einen booleschen Wert zurück oder legt ihn fest, um das Verhalten der Reihennamenanzeige für die Datenbeschriftungen in einem Diagramm anzugeben. True, um den Seriennamen anzuzeigen. Falsch zum Ausblenden. Lesen/SchreibenBoolean .

```csharp
public bool ShowSeriesName { get; set; }
```

### Bemerkungen

Wenn das übergeordnete Objekt dieses DataLabelFormat-Objekts eine DataLabelCollection-Sammlung von Datenbeschriftungen ist, dann ruft diese -Eigenschaft den Standardwert der ShowSeriesName-Eigenschaft für die neuen -Datenbeschriftungen in der DataLabelCollection-Sammlung ab oder legt ihn fest ShowSeriesName-Eigenschaft für alle Datenlabels in der DataLabelCollection-Sammlung (dh "DataLabels.DefaultDataLabelFormat.ShowSeriesName = val;" bewirkt, dass alle DataLabels[i].ShowSeriesName gleich val sind).

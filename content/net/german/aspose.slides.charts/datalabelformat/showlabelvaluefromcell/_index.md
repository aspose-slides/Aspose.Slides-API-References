## DataLabelFormat.ShowLabelValueFromCell-Eigenschaft

Stellt das Anzeigeverhalten des Zellwerts eines bestimmten Diagramms dar. Wahr zeigt den Zellwert an. Falsch, um ihn auszublenden. Lese-/Schreib-Boolean.

```csharp
public bool ShowLabelValueFromCell { get; set; }
```

### Anmerkungen

Wenn der übergeordnete DataLabelFormat-Objekt eine DataLabelCollection-Sammlung von Datenlabels ist, wird dieser Wert der Standardwert der ShowLabelValueFromCell-Eigenschaft für die neuen Datenlabels in der DataLabelCollection-Sammlung festgelegt oder abgerufen. Wenn Sie diese Eigenschaft mit einem Wert festlegen, wird dieser Wert auch für die ShowLabelValueFromCell-Eigenschaft aller Datenlabels in der DataLabelCollection-Sammlung festgelegt (d.h. "DataLabels.DefaultDataLabelFormat.ShowLabelValueFromCell = val;" bewirkt, dass alle DataLabels[i].ShowLabelValueFromCell gleich val sind).

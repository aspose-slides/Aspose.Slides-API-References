## DataLabelFormat.ShowLabelValueFromCell-Eigenschaft  

Stellt das Anzeigeverhalten des Zellenwerts des Datenlabels eines bestimmten Diagramms dar. Wahr zeigt den Zellenwert an. Falsch, um ihn auszublenden. Lese-/Schreib-Boolean.  

```csharp  
public bool ShowLabelValueFromCell { get; set; }  
```  

### Hinweise  

Wenn der übergeordnete Objekt dieser DataLabelFormat-Instanz eine DataLabelCollection-Sammlung von Datenlabels ist, dann wird dieser Eigenschaft der Standardwert für die ShowLabelValueFromCell-Eigenschaft der neuen Datenlabels in der DataLabelCollection-Sammlung zugewiesen. Wenn Sie diesen Wert setzen, wird dieser Wert ebenfalls der ShowLabelValueFromCell-Eigenschaft für alle Datenlabels in der DataLabelCollection-Sammlung zugewiesen (d.h. "DataLabels.DefaultDataLabelFormat.ShowLabelValueFromCell = val;" bewirkt, dass für alle DataLabels[i].ShowLabelValueFromCell der Wert gleich val ist).

## DataLabelCollection.Hide-Methode

Machen Sie das Datenlabel standardmäßig unsichtbar, indem Sie alle Show*-Flags (ShowValue, ...) der DefaultDataLabelFormat-Eigenschaft auf den falschen Zustand setzen. IsVisible wird danach falsch sein.

```csharp
public void Hide()
```

### Anmerkungen

Wenn das Datenlabel standardmäßig nicht sichtbar ist (IsVisible ist falsch), können Sie das Datenlabel "standardmäßig sichtbar" machen, indem Sie die Show*-Flags (ShowValue, ...) der DefaultDataLabelFormat-Eigenschaft auf den wahren Zustand setzen.

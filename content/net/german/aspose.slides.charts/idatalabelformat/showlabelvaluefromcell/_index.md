## IDataLabelFormat.ShowLabelValueFromCell-Eigenschaft  

Stellt das Anzeigeverhalten des Zellwertes des Datenlabels eines bestimmten Diagramms dar. True zeigt den Zellwert an. False, um ihn auszublenden. Lese-/Schreib-Boolean.  

```csharp  
public bool ShowLabelValueFromCell { get; set; }  
```  

### Bemerkungen  

Wenn der übergeordnete Objekt dieser DataLabelFormat-Instanz eine DataLabelCollection von Datenlabels ist, dann wird dieses Property verwendet, um den Standardwert der ShowLabelValueFromCell-Eigenschaft für die neuen Datenlabels in der DataLabelCollection festzulegen. Das Setzen dieser Eigenschaft mit einem Wert setzt auch diesen Wert für die ShowLabelValueFromCell-Eigenschaft für alle Datenlabels in der DataLabelCollection (d.h. "DataLabels.DefaultDataLabelFormat.ShowLabelValueFromCell = val;" bewirkt, dass alle DataLabels[i].ShowLabelValueFromCell gleich val sind).

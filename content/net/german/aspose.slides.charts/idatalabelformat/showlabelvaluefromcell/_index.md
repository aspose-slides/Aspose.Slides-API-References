## IDataLabelFormat.ShowLabelValueFromCell-Eigenschaft

Repräsentiert das Verhalten der Anzeige von Zellenwerten von Datenbeschriftungen eines bestimmten Diagramms. Wahr zeigt den Zellenwert an. Falsch, um ihn zu verbergen. Lese-/Schreib-Boolean.

```csharp
public bool ShowLabelValueFromCell { get; set; }
```

### Anmerkungen

Wenn der übergeordnete DataLabelFormat-Objekt eine DataLabelCollection-Sammlung von Datenbeschriftungen ist, dann erhält oder setzt diese Eigenschaft den Standardwert der ShowLabelValueFromCell-Eigenschaft für die neuen Datenbeschriftungen in der DataLabelCollection-Sammlung. Das Setzen dieser Eigenschaft mit einem Wert setzt auch diesen Wert auf die ShowLabelValueFromCell-Eigenschaft für alle Datenbeschriftungen in der DataLabelCollection-Sammlung (d.h. "DataLabels.DefaultDataLabelFormat.ShowLabelValueFromCell = val;" sorgt dafür, dass alle DataLabels[i].ShowLabelValueFromCell gleich val sind).

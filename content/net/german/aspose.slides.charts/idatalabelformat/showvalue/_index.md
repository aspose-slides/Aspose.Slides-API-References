## IDataLabelFormat.ShowValue-Eigenschaft

Stellt das angegebene Verhalten zur Anzeige des Prozentwerts von Diagrammdatenetiketten dar. True zeigt den Prozentwert an. False, um ihn auszublenden. Lese-/Schreib-Boolean.

```csharp
public bool ShowValue { get; set; }
```

### Anmerkungen

Wenn der übergeordnete DataLabelFormat-Objekt eine DataLabelCollection von Datenetiketten ist, dann wird diese Eigenschaft verwendet, um den Standardwert der ShowValue-Eigenschaft für die neuen Datenetiketten in der DataLabelCollection festzulegen oder abzurufen. Das Setzen dieser Eigenschaft mit einem Wert setzt diesen Wert auch für die ShowValue-Eigenschaft aller Datenetiketten in der DataLabelCollection (d.h. "DataLabels.DefaultDataLabelFormat.ShowValue = val;" führt dazu, dass alle DataLabels[i].ShowValue gleich val sind).

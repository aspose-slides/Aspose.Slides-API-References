## IDataLabelFormat.ShowLabelAsDataCallout-Eigenschaft

Bestimmt, ob das angegebene Diagrammdatenetikett als Datenaufruf oder als Datenetikett angezeigt wird. Wenn der übergeordnete Teil dieses DataLabelFormat-Objekts eine DataLabelCollection von Datenetiketten ist, erhält oder setzt diese Eigenschaft den Standardwert der ShowLabelAsDataCallout-Eigenschaft für die neuen Datenetiketten in der DataLabelCollection (d.h. "DataLabels.DefaultDataLabelFormat.ShowLabelAsDataCallout = val;" bewirkt, dass alle DataLabels[i].ShowLabelAsDataCallout gleich val ist).

public bool ShowLabelAsDataCallout { get; set; }

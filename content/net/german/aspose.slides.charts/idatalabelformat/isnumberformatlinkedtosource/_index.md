## IDataLabelFormat.IsNumberFormatLinkedToSource-Eigenschaft

Lesen/ Schreiben Boolean.

```csharp
public bool IsNumberFormatLinkedToSource { get; set; }
```

### Bemerkungen

Wenn das übergeordnete Objekt dieses DataLabelFormat eine DataLabelCollection von Datenbeschriftungen ist, wird mit dieser Eigenschaft der Standardwert der IsNumberFormatLinkedToSource-Eigenschaft für die neuen Datenbeschriftungen in der DataLabelCollection festgelegt oder abgerufen. Das Setzen dieser Eigenschaft mit einem Wert setzt diesen Wert auch für die IsNumberFormatLinkedToSource-Eigenschaft für alle Datenbeschriftungen in der DataLabelCollection (d.h. "DataLabels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = val;" bewirkt, dass alle DataLabels[i].IsNumberFormatLinkedToSource gleich val sind).

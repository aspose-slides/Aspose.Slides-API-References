## IDataLabelFormat.ShowLeaderLines-Eigenschaft

Stellt das Verhalten der Anzeige von Führungsleitungen für die Datenbeschriftung eines bestimmten Diagramms dar. True zeigt die Führungsleitungen an. False zum Ausblenden. Lese-/Schreib-Boolean.

```csharp
public bool ShowLeaderLines { get; set; }
```

### Anmerkungen

Wenn der Elternteil dieses DataLabelFormat-Objekts eine DataLabelCollection-Sammlung von Datenbeschriftungen ist, dann erhält oder setzt diese Eigenschaft den Standardwert der ShowLeaderLines-Eigenschaft für die neuen Datenbeschriftungen in der DataLabelCollection-Sammlung. Das Setzen dieser Eigenschaft mit einem Wert setzt diesen Wert auch für die ShowLeaderLines-Eigenschaft für alle Datenbeschriftungen in der DataLabelCollection-Sammlung (d.h. "DataLabels.DefaultDataLabelFormat.ShowLeaderLines = val;" führt dazu, dass alle DataLabels[i].ShowLeaderLines gleich val ist).

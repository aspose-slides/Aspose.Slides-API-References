## IDataLabelFormat.ShowBubbleSize property

Repräsentiert das Anzeigeverhalten des Datenbeschriftungsblasengrößenwerts eines bestimmten Diagramms. True zeigt den Wert der Blasengröße an. Falsch zum Ausblenden. Lesen/SchreibenBoolean .

```csharp
public bool ShowBubbleSize { get; set; }
```

### Bemerkungen

Wenn das übergeordnete Objekt dieses DataLabelFormat-Objekts eine DataLabelCollection-Sammlung von Datenbeschriftungen ist, dann ruft diese -Eigenschaft den Standardwert der ShowBubbleSize-Eigenschaft für die neuen -Datenbeschriftungen in der DataLabelCollection-Sammlung ab oder legt ihn fest ShowBubbleSize-Eigenschaft für alle Datenbeschriftungen in der DataLabelCollection-Sammlung (dh "DataLabels.DefaultDataLabelFormat.ShowBubbleSize = val;" bewirkt, dass alle DataLabels[i].ShowBubbleSize gleich val sind).

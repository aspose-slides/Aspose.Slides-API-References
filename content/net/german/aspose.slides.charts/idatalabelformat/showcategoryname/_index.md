## IDataLabelFormat.ShowCategoryName-Eigenschaft

Stellt das Anzeigeverhalten des Kategorienamens für die Datenbeschriftungen eines bestimmten Diagramms dar. Wahr, um den Kategorienamen für die Datenbeschriftungen in einem Diagramm anzuzeigen. Falsch, um ihn auszublenden. Lese-/Schreib-Boolean.

```csharp
public bool ShowCategoryName { get; set; }
```

### Bemerkungen

Wenn der Übergeordnete dieses DataLabelFormat-Objekts eine DataLabelCollection-Sammlung von Datenbeschriftungen ist, erhält oder setzt diese Eigenschaft den Standardwert der ShowCategoryName-Eigenschaft für die neuen Datenbeschriftungen in der DataLabelCollection-Sammlung. Das Setzen dieser Eigenschaft mit einem Wert setzt auch diesen Wert für die ShowCategoryName-Eigenschaft für alle Datenbeschriftungen in der DataLabelCollection-Sammlung (d.h. "DataLabels.DefaultDataLabelFormat.ShowCategoryName = val;" bewirkt, dass alle DataLabels[i].ShowCategoryName gleich val sind).

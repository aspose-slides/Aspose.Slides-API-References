## IDataLabelFormat.ShowCategoryName-Eigenschaft

Stellt das Anzeigeverhalten des Kategorienamens der Datenbeschriftung eines bestimmten Diagramms dar. Wahr, um den Kategorienamen für die Datenbeschriftungen in einem Diagramm anzuzeigen. Falsch, um ihn auszublenden. Lese-/schreibbare Boolean.

```csharp
public bool ShowCategoryName { get; set; }
```

### Hinweise

Wenn der übergeordnete Teil dieses DataLabelFormat-Objekts eine DataLabelCollection-Sammlung von Datenbeschriftungen ist, dann wird diese Eigenschaft durch den Standardwert der ShowCategoryName-Eigenschaft für die neuen Datenbeschriftungen in der DataLabelCollection-Sammlung festgelegt oder abgerufen. Setzen Sie diese Eigenschaft mit einem Wert, setzt dies auch diesen Wert für die ShowCategoryName-Eigenschaft aller Datenbeschriftungen in der DataLabelCollection-Sammlung (d.h. "DataLabels.DefaultDataLabelFormat.ShowCategoryName = val;" bewirkt, dass alle DataLabels[i].ShowCategoryName gleich val sind).

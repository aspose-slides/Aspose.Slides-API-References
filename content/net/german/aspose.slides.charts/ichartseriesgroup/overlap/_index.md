## IChartSeriesGroup.Overlap-Eigenschaft

Gibt an, wie stark Balken und Säulen in 2-D-Diagrammen als Prozentsatz (von -100% bis 100%) überlappen sollen. - -100%: Maximale Abstände (Balken sind vollständig getrennt). - 0%: Balken werden nebeneinander ohne Überlappung oder Abstand platziert. - 100%: Maximale Überlappung (Balken überlappen sich vollständig). Diese Eigenschaft ist beschreibbar und lesbar SByte.

```csharp
public sbyte Overlap { get; set; }
```

### Ausnahmen

| Ausnahme | Bedingung |
| --- | --- |
| ArgumentOutOfRangeException | Wird ausgelöst, wenn der Wert außerhalb des gültigen Bereichs von -100 bis 100 gesetzt wird. |

### Beispiele

Das folgende Beispiel zeigt, wie man die Überlappung für eine Diagrammreihe festlegt und das resultierende Diagramm in einem Formular rendert:

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // Setze die Überlappung auf 55%

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

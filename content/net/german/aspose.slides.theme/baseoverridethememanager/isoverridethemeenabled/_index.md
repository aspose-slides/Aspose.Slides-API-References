## BaseOverrideThemeManager.IsOverrideThemeEnabled-Eigenschaft  
  
Bestimmt, ob OverrideTheme das vererbte effektive Thema überschreibt oder nicht. Um OverrideTheme zum Überschreiben zu aktivieren, verwenden Sie die Methoden OverrideTheme.Init*(). Um OverrideTheme vom Überschreiben abzuhalten, verwenden Sie die Methode OverrideTheme.Clear(). Nur lesbarer Boolean.

public bool IsOverrideThemeEnabled { get; }

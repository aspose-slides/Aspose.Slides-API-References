## IOverrideThemeManager.IsOverrideThemeEnabled Eigenschaft

Bestimmt, ob OverrideTheme das geerbte effektive Thema überschreibt oder nicht. Um OverrideTheme für das Überschreiben zu aktivieren, verwenden Sie die OverrideTheme.Init*() Methoden. Um OverrideTheme vom Überschreiben abzuhalten, verwenden Sie die OverrideTheme.Clear() Methode. Nur-lesbares Boolean.

public bool IsOverrideThemeEnabled { get; }

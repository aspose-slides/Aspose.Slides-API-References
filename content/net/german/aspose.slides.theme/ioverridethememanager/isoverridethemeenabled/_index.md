## IOverrideThemeManager.IsOverrideThemeEnabled property  

Bestimmt, ob OverrideTheme das geerbte effektive Design überschreibt oder nicht. Um OverrideTheme zum Überschreiben zu aktivieren, verwenden Sie die OverrideTheme.Init*() Methoden. Um OverrideTheme vom Überschreiben abzuhalten, verwenden Sie die Methode OverrideTheme.Clear(). Nur-Lese Boolean.

public bool IsOverrideThemeEnabled { get; }

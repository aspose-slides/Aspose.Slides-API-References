## ExternalResourceResolver-Klasse

Callback-Klasse, die verwendet wird, um externe Ressourcen während des Imports von Html- und Svg-Dokumenten aufzulösen. Die Verwendung dieses Resolvers kann eine Sicherheitsanfälligkeit schaffen, wenn die vom Client bereitgestellte HTML- oder SVG-Datei dazu führt, dass die Server-Software auf lokale oder Netzwerkdateien zugreift. Vorsicht geboten. Es wird empfohlen, den ExternalResourceResolver überhaupt nicht anzugeben (es werden nur eingebettete Objekte gelesen) oder eine Unterklasse zu erstellen, die überprüft, ob die angegebene URI gültig ist.

public class ExternalResourceResolver : IExternalResourceResolver

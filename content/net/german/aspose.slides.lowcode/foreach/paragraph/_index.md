## Absatz(Presentation, ForEachParagraphCallback) {#paragraph_1}

Durchlaufen Sie jeden `Paragraph` in der [`Presentation`](../../../aspose.slides/presentation). Formen werden in allen Arten von Folien durchlaufen - [`Slide`](../slide), [`MasterSlide`](../masterslide) und [`LayoutSlide`](../layoutslide)

```csharp
public static void Paragraph(Presentation pres, ForEachParagraphCallback forEachParagraph)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| pres | Presentation | Präsentation zum Durchlaufen der Absätze |
| forEachParagraph | ForEachParagraphCallback | Callback, der für jeden Absatz aufgerufen wird |

### Beispiele

```csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    ForEach.Paragraph(pres, (para, slide, index) =>
    {
        System.Console.WriteLine($"{para.Text}, index: {index}");
    });
}
```

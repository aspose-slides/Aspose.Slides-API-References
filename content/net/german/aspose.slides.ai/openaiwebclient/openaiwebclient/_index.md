## OpenAIWebClient(string, string, string, HttpClient) {#constructor_1}

Erstellt eine Instanz des OpenAI-Webclients, der einen extern verwalteten HttpClient verwendet. Der bereitgestellte HttpClient wird von dieser Instanz nicht freigegeben und bleibt im Besitz des Aufrufers.

```csharp
public OpenAIWebClient(string model, string apiKey, string organizationId, HttpClient httpClient)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| model | String | OpenAI-Sprachmodell. Mögliche Werte: - gpt-4o - gpt-4o-mini - o1 - o1-mini - o3 - o3-mini |
| apiKey | String | OpenAI-API-Schlüssel |
| organizationId | String | Organisations-ID (optional) |
| httpClient | HttpClient | Eine extern verwaltete `HttpClient`-Instanz |

### Ausnahmen

| Ausnahme | Bedingung |
| --- | --- |
| ArgumentException | API-Schlüsselwert darf nicht null oder leer sein |
| ArgumentException | Textmodellwert darf nicht null oder leer sein |
| ArgumentNullException | HttpClient darf nicht null sein |

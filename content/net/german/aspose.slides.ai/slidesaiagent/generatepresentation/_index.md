## GeneratePresentation(string, PresentationContentAmountType) {#generatepresentation}

Erzeugt eine Präsentationsinstanz aus einer Textbeschreibung. Geben Sie ein Thema, Ideen, Zitate oder Textausschnitte in der gewünschten Sprache an.

```csharp
public IPresentation GeneratePresentation(string description, 
    PresentationContentAmountType presentationContentAmount)
```

| Parameter | Type | Beschreibung |
| --- | --- | --- |
| description | String | Das Thema, Ideen, Zitate oder Textausschnitte. |
| presentationContentAmount | PresentationContentAmountType | Die Menge des Inhalts in der resultierenden Präsentation. |

### Exceptions

| Ausnahme | Bedingung |
| --- | --- |
| ArgumentException | AI-Chat-Anweisung darf nicht null oder leer sein. |

### Remarks

Das nachstehende Beispiel verwendet das Standard-[`AsposeAIWebClient`](../../asposeaiwebclient), das durch den parameterlosen [`SlidesAIAgent`](../slidesaiagent)-Konstruktor erstellt wird und eine Verbindung zu Asposes eigenem LLM herstellt. Um einen anderen KI-Anbieter zu verwenden, stellen Sie Ihr eigenes LLM bereit oder passen die Verbindung an (z. B. indem Sie Ihren eigenen `HttpClient` bereitstellen), und übergeben eine [`IAIWebClient`](../../iaiwebclient)-Implementierung an den [`SlidesAIAgent`](../slidesaiagent)-Konstruktor. Verfügbare Implementierungen umfassen:

* [`AsposeAIWebClient`](../../asposeaiwebclient)
* [`OpenAIWebClient`](../../openaiwebclient)
* [`OpenAICompatibleWebClient`](../../openaicompatiblewebclient)

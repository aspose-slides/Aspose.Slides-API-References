## SlidesAIAgent(IAIWebClient) {#constructor_1}

Initialisiert eine neue Instanz von [`SlidesAIAgent`](../../slidesaiagent) mit einem benutzerdefinierten KI-Client. Verwenden Sie diese Überladung, um den KI-Anbieter anzugeben, Ihr eigenes LLM bereitzustellen oder die Verbindung anzupassen (zum Beispiel, indem Sie Ihren eigenen `HttpClient` bereitstellen). Jede Implementierung von [`IAIWebClient`](../../iaiwebclient) kann verwendet werden, einschließlich:

* [`AsposeAIWebClient`](../../asposeaiwebclient)
* [`OpenAIWebClient`](../../openaiwebclient)
* [`OpenAICompatibleWebClient`](../../openaicompatiblewebclient)

Um das integrierte [`AsposeAIWebClient`](../../asposeaiwebclient) mit seiner Standardkonfiguration zu verwenden, benutzen Sie stattdessen die [`SlidesAIAgent`](../slidesaiagent)-Überladung.

```csharp
public SlidesAIAgent(IAIWebClient aiClient)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| aiClient | IAIWebClient | AI-Client-Instanz. Jede Implementierung von [`IAIWebClient`](../../iaiwebclient) kann verwendet werden. |

### Ausnahmen

| Ausnahme | Bedingung |
| --- | --- |
| ArgumentNullException | AI-Client-Instanz ist nicht angegeben. |

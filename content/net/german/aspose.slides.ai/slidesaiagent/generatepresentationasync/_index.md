## GeneratePresentationAsync(string, PresentationContentAmountType) {#generatepresentationasync}

Erzeugt eine Präsentationsinstanz aus einer Textbeschreibung. Geben Sie ein Thema, Ideen, Zitate oder Textausschnitte in der gewünschten Sprache an.

```csharp
public Task<IPresentation> GeneratePresentationAsync(string description, 
    PresentationContentAmountType presentationContentAmount)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| description | String | Das Thema, Ideen, Zitate oder Textausschnitte. |
| presentationContentAmount | PresentationContentAmountType | Die Menge des Inhalts in der resultierenden Präsentation. |

### Ausnahmen

| exception | condition |
| --- | --- |
| ArgumentException | AI chat instruction can't be null or empty. |

### Bemerkungen

Das nachstehende Beispiel verwendet das Standard-[`AsposeAIWebClient`](../../asposeaiwebclient), das durch den parameterlosen [`SlidesAIAgent`](../slidesaiagent)-Konstruktor erstellt wird und eine Verbindung zum eigenen LLM von Aspose herstellt. Um einen anderen KI-Anbieter zu verwenden, stellen Sie Ihr eigenes LLM bereit oder passen Sie die Verbindung an (z. B. indem Sie Ihren eigenen `HttpClient` bereitstellen), übergeben Sie eine [`IAIWebClient`](../../iaiwebclient)-Implementierung an den [`SlidesAIAgent`](../slidesaiagent)-Konstruktor. Verfügbare Implementierungen sind:

* [`AsposeAIWebClient`](../../asposeaiwebclient)
* [`OpenAIWebClient`](../../openaiwebclient)
* [`OpenAICompatibleWebClient`](../../openaicompatiblewebclient)

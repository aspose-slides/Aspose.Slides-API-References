## AsposeAIWebClient() {#constructor}

Erstellt eine Instanz des Aspose AI Web-Clients, die eine Verbindung zum standardmäßigen Aspose-LLM-Endpunkt herstellt. Dies ist der Client, der vom parameterlosen [`SlidesAIAgent`](../../slidesaiagent/slidesaiagent) Konstruktor verwendet wird, daher ist das explizite Erstellen nur erforderlich, wenn der Client direkt an den [`SlidesAIAgent`](../../slidesaiagent/slidesaiagent) Konstruktor übergeben wird.

```csharp
public AsposeAIWebClient()
```

## AsposeAIWebClient(HttpClient) {#constructor_1}

Erstellt eine Instanz des Aspose AI Web-Clients, die über einen extern verwalteten HttpClient eine Verbindung zum standardmäßigen Aspose-LLM-Endpunkt herstellt. Der bereitgestellte HttpClient wird von dieser Instanz nicht entsorgt und bleibt im Besitz des Aufrufers.

```csharp
public AsposeAIWebClient(HttpClient httpClient)
```

## AsposeAIWebClient(string, HttpClient) {#constructor_3}

Erstellt eine Instanz des Aspose AI Web-Clients, die eine benutzerdefinierte Endpunkt-URL unter Verwendung eines extern verwalteten HttpClient verwendet. Der bereitgestellte HttpClient wird von dieser Instanz nicht entsorgt und bleibt im Besitz des Aufrufers. Verwenden Sie diese Überladung, wenn Sie eine von dem Aspose.Slides-Team bereitgestellte URL haben und Ihren eigenen HttpClient bereitstellen möchten; benötigen Sie nur Ihren eigenen HttpClient mit der Standard-URL, verwenden Sie stattdessen die [`AsposeAIWebClient`](../asposeaiwebclient) Überladung.

```csharp
public AsposeAIWebClient(string url, HttpClient httpClient)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| url | String | Endpunkt-URL des Aspose LLM, bereitgestellt vom Aspose.Slides-Team. |
| httpClient | HttpClient | Eine extern verwaltete HttpClient-Instanz. |

### Ausnahmen

| Ausnahme | Bedingung |
| --- | --- |
| ArgumentNullException | URL darf nicht null oder leer sein. |
| ArgumentNullException | HttpClient-Instanz wurde nicht bereitgestellt. |

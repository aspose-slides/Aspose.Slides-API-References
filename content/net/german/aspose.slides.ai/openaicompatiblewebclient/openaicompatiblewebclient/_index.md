## OpenAICompatibleWebClient(string, string, string, HttpClient) {#constructor_1}

Erstellt eine Instanz des OpenAI-kompatiblen Web-Clients, die einen extern verwalteten HttpClient verwendet. Der bereitgestellte HttpClient wird von dieser Instanz nicht entsorgt und bleibt im Besitz des Aufrufenden.

```csharp
public OpenAICompatibleWebClient(string model, string apiKey, string baseUrl, HttpClient httpClient)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| model | String | Modellname, der vom LLM-Anbieter unterstützt wird. |
| apiKey | String | API-Schlüssel (Token). |
| baseUrl | String | Basis-URL des OpenAI-kompatiblen LLM. |
| httpClient | HttpClient | Eine extern verwaltete HttpClient-Instanz. |

### Ausnahmen

| Ausnahme | Bedingung |
| --- | --- |
| ArgumentException | API-Schlüsselwert darf nicht null oder leer sein. |
| ArgumentException | Text-Modellwert darf nicht null oder leer sein. |
| ArgumentException | Basis-URL-Wert darf nicht null oder leer sein. |
| ArgumentNullException | HttpClient darf nicht null sein. |

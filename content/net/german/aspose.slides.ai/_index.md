Enthält Klassen, die KI-basierte Funktionen zum Analysieren und Verarbeiten von PowerPoint-Präsentationen bereitstellen.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [AsposeAIWebClient](./asposeaiwebclient) | Eine integrierte [`IAIWebClient`](../aspose.slides.ai/iaiwebclient)-Implementierung, die eine Verbindung zu Asposes eigener LLM herstellt. Dies ist der Standard-Client, der vom parameterlosen [`SlidesAIAgent`](../aspose.slides.ai/slidesaiagent/slidesaiagent)-Konstruktor verwendet wird. |
| [OpenAICompatibleWebClient](./openaicompatiblewebclient) | Eine integrierte [`IAIWebClient`](../aspose.slides.ai/iaiwebclient)-Implementierung, die eine Verbindung zu einem OpenAI-kompatiblen LLM-Anbieter unter einer angegebenen Basis-URL herstellt. |
| [OpenAIWebClient](./openaiwebclient) | Eine integrierte [`IAIWebClient`](../aspose.slides.ai/iaiwebclient)-Implementierung, die eine Verbindung zur OpenAI-API herstellt. |
| [SlidesAIAgent](./slidesaiagent) | Bietet KI-gestützte Funktionen zur Verarbeitung von Präsentationen. |
| [SlidesAIAgentException](./slidesaiagentexception) | Stellt mit Slides AI Agent zusammenhängende Ausnahmen dar. |

Περιέχει κλάσεις που εισάγουν δεδομένα σε παρουσιάσεις Microsoft PowerPoint.

## Κλάσεις

| Κλάση | Περιγραφή |
| --- | --- |
| [ExcelWorkbookImporter](./excelworkbookimporter) | Παρέχει λειτουργικότητα για εισαγωγή περιεχομένου από βιβλίο εργασίας Excel σε μια παρουσίαση. |
| [ExternalResourceResolver](./externalresourceresolver) | Κλάση callback που χρησιμοποιείται για την επίλυση εξωτερικών πόρων κατά την εισαγωγή εγγράφων Html, Svg. Η χρήση αυτού του resolver μπορεί να δημιουργήσει ευπάθεια όταν το αρχείο HTML ή SVG που παρέχεται από τον πελάτη επιτρέπει στο λογισμικό του διακομιστή να αποκτήσει τοπικά ή δικτυακά αρχεία. Χρησιμοποιήστε το με προσοχή. Συνιστάται να μην καθορίζετε το ExternalResourceResolver καθόλου (θα διαβαστούν μόνο τα ενσωματωμένα αντικείμενα) ή να δημιουργήσετε κάποιο subclass που ελέγχει αν το καθορισμένο uri είναι έγκυρο. |
| [PdfImportOptions](./pdfimportoptions) | Αντιπροσωπεύει τις επιλογές εισαγωγής PDF |

## ITTiming.RepeatUntilEndSlide ιδιότητα

Αυτό το χαρακτηριστικό καθορίζει αν το εφέ θα επαναλαμβάνεται μέχρι το τέλος της διαφάνειας. Ανάγνωση/Εγγραφή Boolean.

```csharp
public bool RepeatUntilEndSlide { get; set; }
```

### Παραδείγματα

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Αποκτήστε τη σειρά των εφέ για την πρώτη διαφάνεια
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Αποκτήστε το πρώτο εφέ της κύριας σειράς.
    IEffect effect = effectsSequence[0];

    // Αλλάξτε το Timing/Repeat του εφέ σε "μέχρι το τέλος της διαφάνειας"
    effect.Timing.RepeatUntilEndSlide = true;
}
```

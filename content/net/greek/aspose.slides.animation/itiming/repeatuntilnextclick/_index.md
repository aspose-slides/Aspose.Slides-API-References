## ITTiming.RepeatUntilNextClick ιδιότητα

Αυτή η ιδιότητα καθορίζει εάν το εφέ θα επαναληφθεί μέχρι το επόμενο κλικ. Ανάγνωση/εγγραφή Boolean.

```csharp
public bool RepeatUntilNextClick { get; set; }
```

### Παραδείγματα

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Λάβετε τη σειρά εφέ για την πρώτη διαφάνεια
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Λάβετε το πρώτο εφέ της κύριας σειράς.
    IEffect effect = effectsSequence[0];

    // Αλλάξτε το Timing/Repeat του εφέ σε "Until Next Click"
    effect.Timing.RepeatUntilNextClick = true;
}
```

## ITiming.Rewind ιδιότητα

Αυτό το χαρακτηριστικό καθορίζει εάν το εφέ θα επαναγυρίσει όταν ολοκληρωθεί η αναπαραγωγή. Ανάγνωση/εγγραφή Boolean.

```csharp
public bool Rewind { get; set; }
```

### Παραδείγματα

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Λάβετε τη σειρά εφέ για την πρώτη διαφάνεια
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Λάβετε το πρώτο εφέ της κύριας σειράς.
    IEffect effect = effectsSequence[0];

    // Ενεργοποιήστε το Timing/Rewind του εφέ.
    effect.Timing.Rewind = true;
}
```

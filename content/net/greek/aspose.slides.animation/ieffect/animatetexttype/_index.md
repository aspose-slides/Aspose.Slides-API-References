## IEffect.AnimateTextType ιδιότητα

Ορίζει έναν τύπο κειμένου κίνησης για το εφέ. Το κείμενο του σχήματος μπορεί να κίνηθεί ανά γράμμα, ανά λέξη ή ολόκληρο ταυτόχρονα. Ανάγνωση/εγγραφή `AnimateTextType`.

```csharp
public AnimateTextType AnimateTextType { get; set; }
```

### Παραδείγματα

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Λάβετε το πρώτο εφέ της πρώτης διαφάνειας.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Αλλάξτε το Animate text type του εφέ σε "By letter"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByLetter;
}
```

## Effect.DelayBetweenTextParts ιδιότητα

Καθορίζει μια καθυστέρηση μεταξύ των ανατεθειμένων τμημάτων κειμένου (λέξεις ή γράμματα). Μια θετική τιμή καθορίζει το ποσοστό της διάρκειας του εφέ. Μια αρνητική τιμή καθορίζει την καθυστέρηση σε δευτερόλεπτα. Ανάγνωση/εγγραφή Single.

```csharp
public float DelayBetweenTextParts { get; set; }
```

### Παραδείγματα

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Λάβετε το πρώτο εφέ της πρώτης διαφάνειας.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Αλλάξτε τον τύπο κειμένου του εφέ Animate σε "By word"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByWord;
    
    // Ορίστε την καθυστέρηση μεταξύ των ανατεθειμένων τμημάτων κειμένου σε 20% της διάρκειας του εφέ.
    firstSlideEffect.DelayBetweenTextParts = 20f;
}
```

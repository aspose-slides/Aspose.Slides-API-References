## Timing.RepeatUntilNextClick ιδιότητα

Αυτό το χαρακτηριστικό καθορίζει αν το εφέ θα επαναληφθεί μέχρι το επόμενο κλικ. Ανάγνωση/Εγγραφή Boolean.

```csharp
public bool RepeatUntilNextClick { get; set; }
```

### Παραδείγματα

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Αποκτά τη σειρά εφέ για την πρώτη διαφάνεια
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Αποκτά το πρώτο εφέ της κύριας σειράς.
    IEffect effect = effectsSequence[0];

    // Αλλάξτε το Timing/Repeat του εφέ σε "Until Next Click"
    effect.Timing.RepeatUntilNextClick = true;
}
```

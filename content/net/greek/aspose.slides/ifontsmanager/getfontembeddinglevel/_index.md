## IFontsManager.GetFontEmbeddingLevel μέθοδος

Καθορίζει το επίπεδο ενσωμάτωσης μιας γραμματοσειράς από τον δοθέντα πίνακα bytes και το όνομα της γραμματοσειράς.

```csharp
public EmbeddingLevel GetFontEmbeddingLevel(byte[] fontBytes, string fontName)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| fontBytes | Byte[] | Ο πίνακας bytes που περιέχει τα δεδομένα της γραμματοσειράς. |
| fontName | String | Το όνομα της γραμματοσειράς. |

### Τιμή Επιστροφής

Το επίπεδο ενσωμάτωσης της καθορισμένης γραμματοσειράς.

### Εξαιρέσεις

| εξαίρεση | συνθήκη |
| --- | --- |
| ArgumentNullException | Εγείρεται όταν το *fontBytes* είναι null. |

### Παραδείγματα

```csharp
[C#]
using (Presentation pres = new Presentation(pptxFileName))
{
    // Ανάκτηση όλων των γραμματοσειρών που χρησιμοποιούνται στην παρουσίαση
    IFontData[] fontDatas = pres.FontsManager.GetFonts();

    // Λήψη του πίνακα byte που αντιπροσωπεύει το κανονικό στυλ της πρώτης γραμματοσειράς στην παρουσίαση
    byte[] bytes = pres.FontsManager.GetFontBytes(fontDatas[0], FontStyle.Regular);

    // Προσδιορισμός του επιπέδου ενσωμάτωσης της γραμματοσειράς
    EmbeddingLevel embeddingLevel = pres.FontsManager.GetFontEmbeddingLevel(bytes, fontDatas[0].FontName);
}
```

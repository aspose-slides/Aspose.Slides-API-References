## GetSubstitutions() {#getsubstitutions}

Ανακτά τις πληροφορίες σχετικά με τις γραμματοσειρές που θα αντικατασταθούν κατά την απόδοση της παρουσίασης.

```csharp
public IEnumerable<FontSubstitutionInfo> GetSubstitutions()
```

### Τιμή επιστροφής

Συλλογή όλων των αντικαταστάσεων γραμματοσειρών [`FontSubstitutionInfo`](../../fontsubstitutioninfo).

### Παραδείγματα

```csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    foreach (var fontSubstitution in pres.FontsManager.GetSubstitutions())
    {
        Console.WriteLine("{0} -> {1}", fontSubstitution.OriginalFontName, fontSubstitution.SubstitutedFontName);
    }
}        
```

## GetSubstitutions(int[]) {#getsubstitutions_1}

Ανακτά τις πληροφορίες σχετικά με τις γραμματοσειρές που θα αντικατασταθούν κατά την απόδοση των συγκεκριμένων διαφανειών.

```csharp
public IEnumerable<FontSubstitutionInfo> GetSubstitutions(int[] slides)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| slides | Int32[] | Ένας πίνακας δεικτών διαφανειών για τους οποίους θα ανακτηθούν οι πληροφορίες αντικατάστασης γραμματοσειρών, αρχίζοντας από το 1. |

### Τιμή επιστροφής

Μια συλλογή όλων των αντικαταστάσεων γραμματοσειρών ([`FontSubstitutionInfo`](../../fontsubstitutioninfo)) για τις συγκεκριμένες διαφάνειες.

### Παραδείγματα

```csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    int[] targetSlides = { 1, 2, 5 };
    foreach (var fontSubstitution in pres.FontsManager.GetSubstitutions(targetSlides))
    {
        Console.WriteLine("{0} -> {1}", fontSubstitution.OriginalFontName, fontSubstitution.SubstitutedFontName);
    }
}
```

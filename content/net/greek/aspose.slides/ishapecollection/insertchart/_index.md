## InsertChart(ChartType, float, float, float, float, int) {#insertchart}

Δημιουργεί ένα νέο διάγραμμα, το αρχικοποιεί με δείγμα δεδομένων σειράς και ρυθμίσεις και το εισάγει στη συλλογή σχημάτων στο συγκεκριμένο δείκτη.

```csharp
public IChart InsertChart(ChartType type, float x, float y, float width, float height, int index)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| type | ChartType | Ο τύπος του διαγράμματος προς δημιουργία. |
| x | Single | Η x-συντεταγμένη του νέου διαγράμματος, σε points. |
| y | Single | Η y-συντεταγμένη του νέου διαγράμματος, σε points. |
| width | Single | Το πλάτος του νέου διαγράμματος, σε points. |
| height | Single | Το ύψος του νέου διαγράμματος, σε points. |
| index | Int32 | Ο μηδενικός δείκτης στον οποίο θα εισαχθεί το νέο διάγραμμα στη συλλογή σχημάτων. |

## InsertChart(ChartType, float, float, float, float, int, bool) {#insertchart_1}

public IChart InsertChart(ChartType type, float x, float y, float width, float height, int index, 
    bool initWithSample)

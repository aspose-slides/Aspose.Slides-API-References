## IShapeCollection.InsertSummaryZoomFrame μέθοδος

Δημιουργεί ένα νέο πλαίσιο Summary Zoom και το εισάγει στη συλλογή σχημάτων στη συγκεκριμένη θέση.

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| index | Int32 | Ο δείκτης μηδενικής βάσης στον οποίο θα εισαχθεί το πλαίσιο Summary Zoom. |
| x | Single | Η x-συντεταγμένη του νέου πλαισίου Summary Zoom, σε μονάδες σημείου. |
| y | Single | Η y-συντεταγμένη του νέου πλαισίου Summary Zoom, σε μονάδες σημείου. |
| width | Single | Το πλάτος του νέου πλαισίου Summary Zoom, σε μονάδες σημείου. |
| height | Single | Το ύψος του νέου πλαισίου Summary Zoom, σε μονάδες σημείου. |

### Τιμή Επιστροφής

Το νεοδημιουργημένο [`ISummaryZoomFrame`](../../isummaryzoomframe).

### Εξαιρέσεις

| exception | condition |

### Παρατηρήσεις

Αυτή η μέθοδος δημιουργεί ένα πλαίσιο Summary Zoom που συγκεντρώνει συνδέσμους σύνοψης για όλες τις ενότητες της παρουσίασης.

### Παραδείγματα

Αυτό το παράδειγμα επιδεικνύει τη δημιουργία και την εισαγωγή ενός αντικειμένου Summary Zoom στη συγκεκριμένη θέση μιας συλλογής (υποθέστε ότι υπάρχουν τουλάχιστον δύο ενότητες στην παρουσίαση "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSummaryZoomFrame(2, 150, 20, 50, 50);
}
```

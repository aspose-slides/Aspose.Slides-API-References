## AddConnector(ShapeType, float, float, float, float, bool) {#addconnector_1}

Δημιουργεί ένα νέο σχήμα σύνδεσης και το προσθέτει στο τέλος της συλλογής σχημάτων, εφαρμόζοντας προαιρετικά το προεπιλεγμένο στυλ προτύπου.

```csharp
public IConnector AddConnector(ShapeType shapeType, float x, float y, float width, float height, 
    bool createFromTemplate)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| shapeType | ShapeType | Το [`ShapeType`](../../shapetype) του σχήματος σύνδεσης προς δημιουργία. |
| x | Single | Η συντεταγμένη x του πλαισίου του σύνδεσμου, σε σημεία. |
| y | Single | Η συντεταγμένη y του πλαισίου του σύνδεσμου, σε σημεία. |
| width | Single | Το πλάτος του πλαισίου του σύνδεσμου, σε σημεία. |
| height | Single | Το ύψος του πλαισίου του σύνδεσμου, σε σημεία. |
| createFromTemplate | Boolean | True για να εφαρμοστεί το προεπιλεγμένο στυλ προτύπου (μη κενό όνομα, απλό στυλ); false για να δημιουργηθεί ο σύνδεσμος με προεπιλεγμένες τιμές ιδιοτήτων. |

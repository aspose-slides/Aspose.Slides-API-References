## IShapeCollection.AddSummaryZoomFrame method

Δημιουργεί ένα νέο πλαίσιο Summary Zoom και το προσθέτει στο τέλος της συλλογής σχημάτων.

public ISummaryZoomFrame AddSummaryZoomFrame(float x, float y, float width, float height)

| width | Single | Το πλάτος του νέου πλαισίου Summary Zoom, σε σημεία. |

### Παρατηρήσεις

Αυτή η μέθοδος δημιουργεί ένα πλαίσιο Summary Zoom που συγκεντρώνει συνδέσμους περίληψης για όλες τις ενότητες στην παρουσίαση.

### Παραδείγματα

Αυτό το παράδειγμα δείχνει την προσθήκη ενός αντικειμένου Summary Zoom στο τέλος μιας συλλογής (υποθέστε ότι υπάρχουν τουλάχιστον δύο ενότητες στην παρουσίαση "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSummaryZoomFrame(150, 20, 500, 250);
}
```

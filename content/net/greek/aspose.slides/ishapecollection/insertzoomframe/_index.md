## InsertZoomFrame(int, float, float, float, float, ISlide) {#insertzoomframe}

Δημιουργεί ένα νέο πλαίσιο Zoom και το εισάγει στη συλλογή σχημάτων στο καθορισμένο ευρετήριο.

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| index | Int32 | Ο δείκτης μηδενικής βάσης στο οποίο θα εισαχθεί το πλαίσιο Zoom. |
| x | Single | Η συντεταγμένη x του νέου πλαισίου Zoom, σε μονάδες σημείων. |
| y | Single | Η συντεταγμένη y του νέου πλαισίου Zoom, σε μονάδες σημείων. |
| width | Single | Το πλάτος του νέου πλαισίου Zoom, σε μονάδες σημείων. |
| height | Single | Το ύψος του νέου πλαισίου Zoom, σε μονάδες σημείων. |
| slide | ISlide | Η [`ISlide`](../../islide) που αναφέρεται από το πλαίσιο Zoom. |

Αυτό το παράδειγμα δείχνει τη δημιουργία και την εισαγωγή ενός αντικειμένου Zoom στο καθορισμένο ευρετήριο μιας συλλογής (υποθέστε ότι υπάρχουν τουλάχιστον δύο διαφάνειες στην παρουσίαση "Presentation.pptx"):

## IShapeCollection.InsertAudioFrameLinked μέθοδος

Δημιουργεί ένα νέο audio frame που συνδέεται με εξωτερικό αρχείο ήχου και το εισάγει στη συλλογή σχήματος στη συγκεκριμένη θέση.

```csharp
public IAudioFrame InsertAudioFrameLinked(int index, float x, float y, float width, float height, 
    string fname)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| index | Int32 | Ο μηδενικός δείκτης (zero-based index) στο οποίο θα εισαχθεί το audio frame. |
| x | Single | Η συντεταγμένη x του νέου audio frame, σε σημεία. |
| y | Single | Η συντεταγμένη y του νέου audio frame, σε σημεία. |
| width | Single | Το πλάτος του νέου audio frame, σε σημεία. |
| height | Single | Το ύψος του νέου audio frame, σε σημεία. |
| fname | String | Η διαδρομή ή το όνομα του εξωτερικού αρχείου ήχου για σύνδεση. |

### Τιμή Επιστροφής

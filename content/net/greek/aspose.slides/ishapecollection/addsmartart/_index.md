## IShapeCollection.AddSmartArt μέθοδος

Δημιουργεί ένα διάγραμμα SmartArt και το προσθέτει στο τέλος της συλλογής σχημάτων.

```csharp
public ISmartArt AddSmartArt(float x, float y, float width, float height, 
    SmartArtLayoutType layoutType)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| x | Single | Η συντεταγμένη x του πλαισίου του διαγράμματος, σε σημεία. |
| y | Single | Η συντεταγμένη y του πλαισίου του διαγράμματος, σε σημεία. |
| width | Single | Το πλάτος του πλαισίου του διαγράμματος, σε σημεία. |
| height | Single | Το ύψος του πλαισίου του διαγράμματος, σε σημεία. |
| layoutType | SmartArtLayoutType | Ο τύπος διάταξης SmartArt. |

### Τιμή Επιστροφής

Το νέο δημιουργημένο [`ISmartArt`](../../../aspose.slides.smartart/ismartart).

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
Presentation pres = new Presentation();
Slide slide = pres.Slides[0];
SmartArt smart = slide.Shapes.AddSmartArt(0, 0, 400, 400, SmartArtLayoutType.BasicBlockList);
```

[Visual Basic]
Private pres As New Presentation()

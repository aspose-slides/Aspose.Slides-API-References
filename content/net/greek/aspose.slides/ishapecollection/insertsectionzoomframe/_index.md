## InsertSectionZoomFrame(int, float, float, float, float, ISection) {#insertsectionzoomframe}

Δημιουργεί ένα νέο πλαίσιο Section Zoom και το εισάγει στη συλλογή σχημάτων στη συγκεκριμένη θέση.

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| index | Int32 | Ο δείκτης αρχής μηδέν στο οποίο θα εισαχθεί το πλαίσιο Section Zoom. |
| x | Single | Η συντεταγμένη x του νέου πλαισίου Section Zoom, σε μονάδες σημείου. |
| y | Single | Η συντεταγμένη y του νέου πλαισίου Section Zoom, σε μονάδες σημείου. |
| width | Single | Το πλάτος του νέου πλαισίου Section Zoom, σε μονάδες σημείου. |
| height | Single | Το ύψος του νέου πλαισίου Section Zoom, σε μονάδες σημείου. |
| section | ISection | Το [`ISection`](../../isection) που αναφέρεται από το πλαίσιο Section Zoom· πρέπει να ανήκει σε αυτήν την παρουσίαση και να περιέχει τουλάχιστον μία διαφάνεια. |

### Τιμή Επιστροφής

Το νεοδημιουργημένο [`ISectionZoomFrame`](../../isectionzoomframe).

### Εξαιρέσεις

| Εξαίρεση | Συνθήκη |
| --- | --- |
| ArgumentException | Εκκρίνεται εάν η αναφερόμενη ενότητα δεν ανήκει στην τρέχουσα παρουσίαση ή δεν περιέχει διαφάνειες. |

### Παραδείγματα

Αυτό το παράδειγμα δείχνει τη δημιουργία και την εισαγωγή ενός αντικειμένου Section Zoom στη συγκεκριμένη θέση μιας συλλογής (υποθέστε ότι υπάρχουν τουλάχιστον δύο ενότητες στην παρουσίαση "Presentation.pptx"):

```csharp
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSectionZoomFrame(2, 150, 20, 50, 50, pres.Sections[1]);
}
```

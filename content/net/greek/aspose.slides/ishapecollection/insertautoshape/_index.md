## InsertAutoShape(int, ShapeType, float, float, float, float) {#insertautoshape}

Δημιουργεί ένα νέο αυτόματο σχήμα και το εισάγει στη συλλογή σχημάτων στη συγκεκριμένη θέση, εφαρμόζοντας την προεπιλεγμένη μορφοποίηση προτύπου.

```csharp
public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width, 
    float height)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| index | Int32 | Ο μηδενικός δείκτης στο οποίο θα εισαχθεί το νέο αυτόματο σχήμα. |
| shapeType | ShapeType | Η [`ShapeType`](../../shapetype) του αυτόματου σχήματος που θα εισαχθεί. |
| x | Single | Η συντεταγμένη x του πλαισίου του σχήματος, σε σημεία. |
| y | Single | Η συντεταγμένη y του πλαισίου του σχήματος, σε σημεία. |
| width | Single | Το πλάτος του πλαισίου του σχήματος, σε σημεία. |
| height | Single | Το ύψος του πλαισίου του σχήματος, σε σημεία. |

## InsertAutoShape(int, ShapeType, float, float, float, float, bool) {#insertautoshape_1}

Δημιουργεί ένα νέο αυτόματο σχήμα και το εισάγει στη συλλογή σχημάτων στη συγκεκριμένη θέση, προαιρετικά αρχικοποιώντας το με το προεπιλεγμένο στυλ προτύπου.

```csharp
public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| index | Int32 | Ο μηδενικός δείκτης στο οποίο θα εισαχθεί το αυτόματο σχήμα. |
| shapeType | ShapeType | Η [`ShapeType`](../../shapetype) του αυτόματου σχήματος που θα εισαχθεί. |
| x | Single | Η συντεταγμένη x του πλαισίου του σχήματος, σε σημεία. |
| y | Single | Η συντεταγμένη y του πλαισίου του σχήματος, σε σημεία. |
| width | Single | Το πλάτος του πλαισίου του σχήματος, σε σημεία. |
| height | Single | Το ύψος του πλαισίου του σχήματος, σε σημεία. |
| createFromTemplate | Boolean | True για εφαρμογή του προεπιλεγμένου στυλ προτύπου (συμπεριλαμβανομένου μη κενής ονομασίας, απλού στυλ και κεντρασμένου κειμένου); false για δημιουργία του σχήματος με όλες τις ιδιότητες στην προεπιλογή. |

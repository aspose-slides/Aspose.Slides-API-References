## AddSectionZoomFrame(float, float, float, float, ISection) {#addsectionzoomframe}

Δημιουργεί ένα νέο πλαίσιο Section Zoom και το προσθέτει στο τέλος της συλλογής σχήματος.

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| x | Single | Η συντεταγμένη x του νέου πλαισίου Section Zoom, σε μονάδες σημείου. |
| y | Single | Η συντεταγμένη y του νέου πλαισίου Section Zoom, σε μονάδες σημείου. |
| width | Single | Το πλάτος του νέου πλαισίου Section Zoom, σε μονάδες σημείου. |
| height | Single | Το ύψος του νέου πλαισίου Section Zoom, σε μονάδες σημείου. |
| section | ISection | Το [`ISection`](../../isection) που αναφέρεται από το πλαίσιο Section Zoom· πρέπει να ανήκει σε αυτήν την παρουσίαση και να περιέχει τουλάχιστον μία διαφάνεια. |

### Τιμή Επιστροφής

Το πρόσφατα δημιουργημένο [`ISectionZoomFrame`](../../isectionzoomframe).

### Παραδείγματα

Αυτό το παράδειγμα δείχνει την προσθήκη ενός αντικειμένου Section Zoom στο τέλος μιας συλλογής (υποθέστε ότι υπάρχουν τουλάχιστον δύο ενότητες στην παρουσίαση "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1]);
}
```

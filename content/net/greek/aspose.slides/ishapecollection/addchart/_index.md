## AddChart(ChartType, float, float, float, float) {#addchart}

Δημιουργεί ένα νέο διάγραμμα, το αρχικοποιεί με δείγμα δεδομένων σειράς και ρυθμίσεων, και το προσθέτει στο τέλος της συλλογής σχήματος.

```csharp
public IChart AddChart(ChartType type, float x, float y, float width, float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| type | ChartType | Ο τύπος του διαγράμματος που θα προστεθεί. |
| x | Single | Η συντεταγμένη x του νέου διαγράμματος, σε μονάδες σημείου. |
| y | Single | Η συντεταγμένη y του νέου διαγράμματος, σε μονάδες σημείου. |
| width | Single | Το πλάτος του διαγράμματος, σε μονάδες σημείου. |
| height | Single | Το ύψος του διαγράμματος, σε μονάδες σημείου. |

### Return Value

Το νέο δημιουργημένο [`IChart`](../../../aspose.slides.charts/ichart).

## AddChart(ChartType, float, float, float, float, bool) {#addchart_1}

Δημιουργεί ένα νέο διάγραμμα, το αρχικοποιεί με δείγμα δεδομένων σειράς και ρυθμίσεων, και το προσθέτει στο τέλος της συλλογής σχήματος.

```csharp
public IChart AddChart(ChartType type, float x, float y, float width, float height, 
    bool initWithSample)
```

| Parameter | Type | Description |
| --- | --- | --- |
| type | ChartType | Ο τύπος του διαγράμματος που θα προστεθεί. |
| x | Single | Η συντεταγμένη x του νέου διαγράμματος, σε μονάδες σημείου. |
| y | Single | Η συντεταγμένη y του νέου διαγράμματος, σε μονάδες σημείου. |
| width | Single | Το πλάτος του διαγράμματος, σε μονάδες σημείου. |
| height | Single | Το ύψος του διαγράμματος, σε μονάδες σημείου. |
| initWithSample | Boolean | True για να αρχικοποιήσετε το νέο διάγραμμα με δείγμα δεδομένων σειράς και ρυθμίσεων· false για να δημιουργήσετε το διάγραμμα χωρίς σειρά και μόνο με ελάχιστες ρυθμίσεις, κάτι που κάνει τη δημιουργία πιο γρήγορη. |

### Return Value

Το νέο δημιουργημένο [`IChart`](../../../aspose.slides.charts/ichart).

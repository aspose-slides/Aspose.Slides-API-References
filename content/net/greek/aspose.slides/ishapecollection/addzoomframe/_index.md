## AddZoomFrame(float, float, float, float, ISlide) {#addzoomframe}

Δημιουργεί ένα νέο πλαίσιο Zoom και το προσθέτει στο τέλος της συλλογής σχήματος.

```csharp
public IZoomFrame AddZoomFrame(float x, float y, float width, float height, ISlide slide)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| x | Single | Η συντεταγμένη x του νέου πλαισίου Zoom, σε σημεία. |
| y | Single | Η συντεταγμένη y του νέου πλαισίου Zoom, σε σημεία. |
| width | Single | Το πλάτος του νέου πλαισίου Zoom, σε σημεία. |
| height | Single | Το ύψος του νέου πλαισίου Zoom, σε σημεία. |
| slide | ISlide | Το [`ISlide`](../../islide) που αναφέρεται από το πλαίσιο Zoom· πρέπει να ανήκει σε αυτήν την παρουσίαση. |

### Παραδείγματα

Αυτό το παράδειγμα δείχνει την προσθήκη ενός αντικειμένου Zoom στο τέλος μιας συλλογής (υποθέστε ότι υπάρχουν τουλάχιστον δύο διαφάνειες στην παρουσίαση "Presentation.pptx"):

```csharp

}
```

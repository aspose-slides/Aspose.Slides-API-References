## AddGroupShape() {#addgroupshape}

Δημιουργεί ένα νέο κενό σχήμα ομάδας και το προσθέτει στο τέλος της συλλογής σχήματος. Το πλαίσιο της ομάδας θα προσαρμοστεί αυτόματα ώστε να χωράει οποιαδήποτε σχήματα προστεθούν σε αυτό.

public IGroupShape AddGroupShape()

## AddGroupShape(ISvgImage, float, float, float, float) {#addgroupshape_1}

Δημιουργεί ένα νέο σχήμα ομάδας, μετατρέπει την καθορισμένη εικόνα SVG σε μεμονωμένα σχήματα και προσθέτει την προκύπτουσα ομάδα στο τέλος της συλλογής σχήματος.

```csharp
public IGroupShape AddGroupShape(ISvgImage svgImage, float x, float y, float width, float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| svgImage | ISvgImage | Το [`ISvgImage`](../../isvgimage) που περιέχει διανυσματικό περιεχόμενο για μετατροπή σε σχήματα. |
| x | Single | Η x-συντεταγμένη του πλαισίου της ομάδας, σε σημεία. |
| y | Single | Η y-συντεταγμένη του πλαισίου της ομάδας, σε σημεία. |
| width | Single | Το πλάτος του πλαισίου της ομάδας, σε σημεία. |
| height | Single | Το ύψος του πλαισίου της ομάδας, σε σημεία. |

### Return Value

Το νέο δημιουργημένο [`IGroupShape`](../../igroupshape).

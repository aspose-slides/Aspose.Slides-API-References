## AddAudioFrameEmbedded(float, float, float, float, Stream) {#addaudioframeembedded_1}

Δημιουργεί ένα νέο πλαίσιο ήχου με ενσωματωμένο αρχείο WAV και το προσθέτει στο τέλος της συλλογής σχήματος. Ο ενσωματωμένος ήχος προστίθεται στη συλλογή Presentation.Audios.

```csharp
public IAudioFrame AddAudioFrameEmbedded(float x, float y, float width, float height, 
    Stream audio_stream)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| x | Single | Η συντεταγμένη x του νέου πλαισίου ήχου, σε μονάδες σημείου. |
| y | Single | Η συντεταγμένη y του νέου πλαισίου ήχου, σε μονάδες σημείου. |
| width | Single | Το πλάτος του νέου πλαισίου ήχου, σε μονάδες σημείου. |
| height | Single | Το ύψος του νέου πλαισίου ήχου, σε μονάδες σημείου. |
| audio_stream | Stream | Ένα ροή εισόδου που περιέχει δεδομένα ήχου WAV για ενσωμάτωση. |

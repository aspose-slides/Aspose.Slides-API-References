## IShapeCollection.AddPictureFrame μέθοδος

Δημιουργεί ένα νέο πλαίσιο εικόνας που περιέχει την καθορισμένη εικόνα και το προσθέτει στο τέλος της συλλογής σχήματος.

```csharp
public IPictureFrame AddPictureFrame(ShapeType shapeType, float x, float y, float width, 
    float height, IPPImage image)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| shapeType | ShapeType | Καθορίζει τον τύπο σχήματος που περιέχεται στο [`ShapeType`](../../shapetype), εκτός από όλα τα είδη γραμμών: |
| x | Single | Η συντεταγμένη x του πλαισίου εικόνας, σε μονάδες πόντων. |
| y | Single | Η συντεταγμένη y του πλαισίου εικόνας, σε μονάδες πόντων. |
| width | Single | Το πλάτος του πλαισίου εικόνας, σε μονάδες πόντων. |
| height | Single | Το ύψος του πλαισίου εικόνας, σε μονάδες πόντων. |
| image | IPPImage | Το [`IPPImage`](../../ippimage) που θα εμφανιστεί στο πλαίσιο εικόνας. |

### Τιμή Επιστροφής

## InsertAudioFrameEmbedded(int, float, float, float, float, Stream) {#insertaudioframeembedded_1}

Δημιουργεί ένα νέο πλαίσιο ήχου με ενσωματωμένο αρχείο WAV και το εισάγει στη συλλογή σχημάτων στο καθορισμένο δείκτη. Ο ενσωματωμένος ήχος προστίθεται στη συλλογή Presentation.Audios.

```csharp
public IAudioFrame InsertAudioFrameEmbedded(int index, float x, float y, float width, float height, 
    Stream audio_stream)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | Int32 | Ο δείκτης μηδενικής βάσης στο οποίο θα εισαχθεί το πλαίσιο ήχου. |
| x | Single | Η συντεταγμένη x του νέου πλαισίου ήχου, σε σημεία. |
| y | Single | Η συντεταγμένη y του νέου πλαισίου ήχου, σε σημεία. |
| width | Single | Το πλάτος του νέου πλαισίου ήχου, σε σημεία. |
| height | Single | Το ύψος του νέου πλαισίου ήχου, σε σημεία. |
| audio_stream | Stream | Ένα ρεύμα εισόδου που περιέχει δεδομένα ήχου WAV για ενσωμάτωση. |

### Return Value

Το νεοδημιουργημένο [`IAudioFrame`](../../iaudioframe).

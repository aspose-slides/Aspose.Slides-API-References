## IPresentationInfo.CheckPassword μέθοδος

Ελέγχει εάν ένας κωδικός πρόσβασης είναι σωστός για μια παρουσίαση που είναι προστατευμένη με ανοιχτό κωδικό.

```csharp
public bool CheckPassword(string password)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| password | String | Ο κωδικός πρόσβασης προς έλεγχο. |

### Τιμή Επιστροφής

True εάν η παρουσίαση είναι προστατευμένη με ανοιχτό κωδικό και ο κωδικός πρόσβασης είναι σωστός, διαφορετικά false.

### Παρατηρήσεις

Όταν ο κωδικός πρόσβασης είναι null ή κενός, αυτή η μέθοδος επιστρέφει false.

### Παραδείγματα

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo("pres.pptx");
bool isPasswordCorrect = info.CheckPassword("my_password");
```

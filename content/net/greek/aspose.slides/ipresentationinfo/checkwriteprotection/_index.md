## IPresentationInfo.CheckWriteProtection μέθοδος

Ελέγχει εάν ο κωδικός για τροποποίηση είναι σωστός για μια παρουσίαση που προστατεύεται από εγγραφή.

```csharp
public bool CheckWriteProtection(string password)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| password | String | Ο κωδικός που θα ελεγχθεί. |

### Return Value

Αληθές εάν η παρουσίαση είναι προστατευμένη από εγγραφή και ο κωδικός είναι σωστός. Ψευδές διαφορετικά.

### Exceptions

| εξαίρεση | συνθήκη |
| --- | --- |
| InvalidOperationException |  |

### Remarks

1. Θα πρέπει να ελέγξετε την [`IsWriteProtected`](../iswriteprotected) ιδιότητα πριν καλέσετε αυτή τη μέθοδο. 2. Όταν ο κωδικός είναι null ή κενός, αυτή η μέθοδος επιστρέφει ψευδές.

### Examples

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
if (info.IsWriteProtected == NullableBool.True)
{
    bool isWriteProtectedByPassword = info.CheckWriteProtection("my_password");
}
```

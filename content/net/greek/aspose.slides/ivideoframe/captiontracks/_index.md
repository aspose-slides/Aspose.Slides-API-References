## IVideoFrame.CaptionTracks ιδιότητα

Ανακτά τη συλλογή των κλειστών υποτίτλων που σχετίζονται με το πλαίσιο ήχου. Αυτή η ιδιότητα είναι μόνο για ανάγνωση και επιστρέφει ένα [`ICaptionsCollection`](../../icaptionscollection) που περιέχει όλες τις γραμμές υποτίτλων.

```csharp
public ICaptionsCollection CaptionTracks { get; }
```

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
using (Presentation pres = new Presentation("video with captions.pptx"))
{
    foreach (IShape shape in pres.Slides[0].Shapes)
    {
        if (!(shape is IVideoFrame videoFrame)) continue;

        foreach (var captionTrack in videoFrame.CaptionTracks)
        {
            // Αποκτά τα δυαδικά δεδομένα των υποτίτλων και τα αποθηκεύει στο αρχείο
            System.IO.File.WriteAllBytes(captionTrack.CaptionId + ".vtt", captionTrack.BinaryData);
        }
    }
}
```

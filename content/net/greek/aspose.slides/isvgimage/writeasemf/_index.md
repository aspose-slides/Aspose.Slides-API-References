### Παραδείγματα

Το παρακάτω παράδειγμα δείχνει πώς να αποθηκεύσετε την εικόνα SVG σε ένα metafile.

```csharp
[C#]
// Δημιουργεί τη νέα εικόνα SVG
ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));

// Αποθηκεύει την εικόνα SVG ως metafile
using (var fileStream = System.IO.File.OpenWrite("SvgAsEmf.emf"))
    svgImage.WriteAsEmf(fileStream);
```

Αυτό το παράδειγμα δείχνει πώς να προσθέσετε την εικόνα SVG ως metafile στη συλλογή εικόνων της παρουσίασης.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // Δημιουργεί τη νέα εικόνα SVG
    ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));
    using (var memStream = new MemoryStream())
    {
        // Αποθηκεύει την εικόνα SVG ως metafile
        svgImage.WriteAsEmf(memStream);
        // Προσθέτει το metafile στη συλλογή εικόνων
        pres.Images.AddImage(memStream.ToArray());
    }
}
```

## TextFrame.HighlightRegex μέθοδος

Επισημαίνει όλες τις αντιστοιχίες της κανονικής έκφρασης με το καθορισμένο χρώμα.

```csharp
public void HighlightRegex(Regex regex, Color highlightColor, IFindResultCallback callback)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| regex | Regex | Η κανονική έκφραση Regex για λήψη των συμβολοσειρών που θα επισημανθούν. |
| highlightColor | Color | Το χρώμα για την επισήμανση του κειμένου. |
| callback | IFindResultCallback | Το αντικείμενο callback για τη λήψη των αποτελεσμάτων αναζήτησης [`IFindResultCallback`](../../ifindresultcallback). |

### Παραδείγματα

Το παρακάτω παράδειγμα κώδικα δείχνει πώς να επισημάνετε κείμενο σε ένα TextFrame χρησιμοποιώντας κανονική έκφραση.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// επισημαίνοντας όλες τις λέξεις με 10 ή περισσότερους χαρακτήρες
	((AutoShape)presentation.Slides[0].Shapes[0]).TextFrame.HighlightRegex(regex, Color.Blue, null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

## PictureFillFormat.TileAlignment ιδιότητα

Επιστρέφει ή ορίζει πώς ευθυγραμμίζεται η υφή εντός του σχήματος. Αυτή η ρύθμιση ελέγχει το σημείο έναρξης του μοτίβου υφής και πώς επαναλαμβάνεται σε όλο το σχήμα. Ανάγνωση/εγγραφή [`RectangleAlignment`](../../rectanglealignment).

```csharp
public RectangleAlignment TileAlignment { get; set; }
```

### Παρατηρήσεις

Η προεπιλογή είναι TopLeft.

### Παραδείγματα

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Λαμβάνει τη μορφή γεμίσματος εικόνας του σχήματος
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Ορίζει τη λειτουργία γεμίσματος εικόνας σε Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Ορίζει την ευθυγράμμιση του πλακιδίου στο κάτω δεξιό
    pictureFillFormat.TileAlignment = RectangleAlignment.BottomRight;
}
```

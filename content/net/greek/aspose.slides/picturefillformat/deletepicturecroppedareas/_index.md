## PictureFillFormat.DeletePictureCroppedAreas μέθοδος

Διαγράφει τις περικομμένες περιοχές του γεμίσματος Picture.

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### Τιμή επιστροφής

Περικομμένη εικόνα ή αρχική εικόνα εάν η περικοπή δεν είναι απαραίτητη.

### Παρατηρήσεις

Αυτή η μέθοδος μετατρέπει αρχεία metafile WMF/EMF σε εικόνα raster PNG ενώ κάνει περικοπή.

### Παραδείγματα

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Παίρνει το PictureFrame
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // Διαγράφει τις περικομμένες περιοχές της εικόνας PictureFrame
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

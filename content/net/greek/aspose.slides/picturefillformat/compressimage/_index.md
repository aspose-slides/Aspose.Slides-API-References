## CompressImage(bool, PicturesCompression) {#compressimage}

Συμπιέζει την εικόνα μειώνοντας το μέγεθός της βάσει του μεγέθους του σχήματος και της καθορισμένης ανάλυσης. Προαιρετικά, διαγράφει επίσης τις περικομμένες περιοχές.

```csharp
public bool CompressImage(bool deleteCroppedAreasOfImage, PicturesCompression resolution)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| deleteCroppedAreasOfImage | Boolean | Αν είναι true, η μέθοδος θα αφαιρέσει τις περικομμένες περιοχές της εικόνας, μειώνοντας ενδέχεται περαιτέρω το μέγεθός της. |
| resolution | PicturesCompression | Η επιθυμητή ανάλυση για τη συμπίεση, καθοριζόμενη ως τιμή του [`PicturesCompression`](../../../aspose.slides.export/picturescompression) enum. |

### Τιμή Επιστροφής

Ένα Boolean που υποδεικνύει αν η εικόνα συμπιέστηκε επιτυχώς. Επιστρέφει `true` εάν η εικόνα αυξήθηκε ή περικόπηκε, διαφορετικά `false`.

### Εξαιρέσεις

| εξαίρεση | κατάσταση |
| --- | --- |
| ArgumentException | Εμφανίζεται όταν η ανάλυση δεν είναι έγκυρη τιμή. |

### Παρατηρήσεις

Αυτή η μέθοδος αλλάζει το μέγεθος και την ανάλυση της εικόνας παρόμοια με τη λειτουργία «Μορφή Εικόνας → Συμπίεση Εικόνων» του PowerPoint.

### Παραδείγματα

Το παρακάτω παράδειγμα δείχνει πώς να χρησιμοποιήσετε τη μέθοδο `CompressImage` για να μειώσετε το μέγεθος μιας εικόνας σε μια παρουσίαση ορίζοντας μια στόχο ανάλυση και αφαιρώντας τις περικομμένες περιοχές:

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;
    // Συμπιέστε την εικόνα με στόχο ανάλυση 150 DPI (ανάλυση ιστού) και αφαιρέστε τις περικομμένες περιοχές
    bool result = picFrame.PictureFormat.CompressImage(true, PicturesCompression.Dpi150);
}
```

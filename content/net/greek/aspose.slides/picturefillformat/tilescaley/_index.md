## PictureFillFormat.TileScaleY ιδιότητα

Επιστρέφει ή ορίζει την κάθετη κλίμακα για τη γεμίση υφής ως ποσοστό. Ανάγνωση/εγγραφή Single.

```csharp
public float TileScaleY { get; set; }
```

### Παραδείγματα

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Λαμβάνει τη μορφή γεμίσματος εικόνας του σχήματος
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Ορίζει τη λειτουργία γεμίσματος εικόνας σε Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Ορίζει την κάθετη κλίμακα για την υφή σε 120 τοις εκατό
    pictureFillFormat.TileScaleY = 120;
}
```

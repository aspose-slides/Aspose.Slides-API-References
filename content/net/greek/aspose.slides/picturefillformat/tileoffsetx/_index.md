## PictureFillFormat.TileOffsetX ιδιότητα

Επιστρέφει ή ορίζει την οριζόντια απόσταση της υφής από το αρχικό σημείο του σχήματος σε points. Μια θετική τιμή μετακινεί την υφή προς τα δεξιά, ενώ μια αρνητική τιμή τη μετακινεί προς τα αριστερά. Ανάγνωση/εγγραφή Single.

```csharp
public float TileOffsetX { get; set; }
```

### Παραδείγματα

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Λαμβάνει τη μορφή γεμίσματος εικόνας του σχήματος
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Ορίζει τη λειτουργία γεμίσματος εικόνας σε Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Ορίζει την οριζόντια απόσταση της υφής σε 20 points
    pictureFillFormat.TileOffsetX = 20f;
}
```

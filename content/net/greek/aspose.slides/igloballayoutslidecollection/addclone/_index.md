## AddClone(ILayoutSlide) {#addclone}

Προσθέτει ένα αντίγραφο μιας συγκεκριμένης διαφάνειας διάταξης στην παρουσίαση.

```csharp
public ILayoutSlide AddClone(ILayoutSlide sourceLayout)
```

| Parameter | Type | Description |
| --- | --- | --- |
| sourceLayout | ILayoutSlide | Διαφάνεια προς κλωνοποίηση. |

### Return Value

Η προστιθέμενη διαφάνεια.

### Remarks

Κατά την κλωνοποίηση μιας διάταξης μεταξύ διαφορετικών παρουσιάσεων, ο master της διάταξης μπορεί επίσης να κλωνοποιηθεί ώστε να διατηρηθεί η μορφοποίηση της πηγής. Χρησιμοποιείται εσωτερικό μητρώο για την παρακολούθηση αυτόματα κλωνοποιημένων master ώστε να αποτραπεί η δημιουργία πολλαπλών κλωνοποιήσεων του ίδιου master slide. Η μη αυτόματη κλωνοποίηση master slides δεν θα εμποδίζεται ούτε θα καταγράφεται.

## AddClone(ILayoutSlide, IMasterSlide) {#addclone_1}

Προσθέτει ένα αντίγραφο μιας συγκεκριμένης διαφάνειας διάταξης στην παρουσίαση.

```csharp
public ILayoutSlide AddClone(ILayoutSlide sourceLayout, IMasterSlide destMaster)
```

| Parameter | Type | Description |
| --- | --- | --- |
| sourceLayout | ILayoutSlide | Διαφάνεια προς κλωνοποίηση. |
| destMaster | IMasterSlide | Master διαφάνεια για τη νέα διάταξη. |

### Return Value

Η προστιθέμενη διαφάνεια.

### Remarks

Η νέα διάταξη θα συνδεθεί με τον καθορισμένο master στην προοριστική παρουσίαση. Έτσι, αυτό είναι ανάλογο της εντολής αντιγραφή/επικόλληση με την επιλογή "Use Destination Theme" στο PowerPoint.

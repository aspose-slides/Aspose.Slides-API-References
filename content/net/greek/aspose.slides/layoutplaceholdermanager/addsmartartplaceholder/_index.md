## LayoutPlaceholderManager.AddSmartArtPlaceholder μέθοδος

Προσθέτει ένα νέο σχήμα placeholder στη διαφάνεια διάταξης για να φιλοξενήσει ένα διάγραμμα SmartArt.

```csharp
public IAutoShape AddSmartArtPlaceholder(float x, float y, float width, float height)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| x | Single | Η συντεταγμένη X του νέου σχήματος placeholder. |
| y | Single | Η συντεταγμένη Y του νέου σχήματος placeholder. |
| width | Single | Το πλάτος του νέου σχήματος placeholder. |
| height | Single | Το ύψος του νέου σχήματος placeholder. |

### Τιμή Επιστροφής

Δημιουργήθηκε [`IAutoShape`](../../iautoshape) με ένα placeholder SmartArt.

### Παραδείγματα

Το παρακάτω παράδειγμα δείχνει πώς να προσθέσετε το σχήμα placeholder SmartArt στη διαφάνεια διάταξης.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddSmartArtPlaceholder(20, 20, 200, 200);
}
```

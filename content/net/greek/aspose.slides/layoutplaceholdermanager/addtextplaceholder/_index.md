## LayoutPlaceholderManager.AddTextPlaceholder μέθοδος

Προσθέτει ένα νέο σχήμα placeholder στη διαφάνεια διάταξης για να περιέχει κείμενο.

```csharp
public IAutoShape AddTextPlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| x | Single | Η συντεταγμένη X του νέου σχήματος placeholder. |
| y | Single | Η συντεταγμένη Y του νέου σχήματος placeholder. |
| width | Single | Το πλάτος του νέου σχήματος placeholder. |
| height | Single | Το ύψος του νέου σχήματος placeholder. |

### Τιμή Επιστροφής

Δημιουργήθηκε [`IAutoShape`](../../iautoshape) με ένα placeholder κειμένου.

### Παραδείγματα

Το παρακάτω παράδειγμα δείχνει πώς να προσθέσετε το σχήμα Text placeholder στη διαφάνεια διάταξης.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTextPlaceholder(20, 20, 500, 300);
}
```

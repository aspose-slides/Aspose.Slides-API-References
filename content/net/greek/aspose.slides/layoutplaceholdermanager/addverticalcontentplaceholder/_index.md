## LayoutPlaceholderManager.AddVerticalContentPlaceholder μέθοδος

Προσθέτει ένα νέο σχήμα κράτησης θέσης στη διαφάνεια διάταξης για να φιλοξενήσει περιεχόμενο, όπως εικόνα, πίνακα, πολυμέσα ή κείμενο σε κάθετη κατεύθυνση.

```csharp
public IAutoShape AddVerticalContentPlaceholder(float x, float y, float width, float height)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| x | Single | Η συντεταγμένη X του νέου σχήματος κράτησης θέσης. |
| y | Single | Η συντεταγμένη Y του νέου σχήματος κράτησης θέσης. |
| width | Single | Το πλάτος του νέου σχήματος κράτησης θέσης. |
| height | Single | Το ύψος του νέου σχήματος κράτησης θέσης. |

### Τιμή Επιστροφής

Δημιουργήθηκε [`IAutoShape`](../../iautoshape) με μια θέση περιεχομένου (κάθετη).

### Παραδείγματα

Το παρακάτω παράδειγμα δείχνει πώς να προσθέσετε το σχήμα θέση περιεχομένου (κάθετη) στη διαφάνεια διάταξης.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalContentPlaceholder(20, 20, 300, 500);
}
```

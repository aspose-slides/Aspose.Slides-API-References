## LayoutPlaceholderManager.AddChartPlaceholder μέθοδος

Προσθέτει ένα νέο σχήμα κράτησης θέσης στη διάταξη διαφάνειας για να φιλοξενήσει ένα γράφημα.

```csharp
public IAutoShape AddChartPlaceholder(float x, float y, float width, float height)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| x | Single | Η συντεταγμένη X του νέου σχήματος κράτησης θέσης. |
| y | Single | Η συντεταγμένη Y του νέου σχήματος κράτησης θέσης. |
| width | Single | Το πλάτος του νέου σχήματος κράτησης θέσης. |
| height | Single | Το ύψος του νέου σχήματος κράτησης θέσης. |

### Τιμή Επιστροφής

Δημιουργήθηκε [`IAutoShape`](../../iautoshape) με ένα Chart placeholder.

### Παραδείγματα

Το παρακάτω παράδειγμα δείχνει πώς να προσθέσετε το σχήμα Chart placeholder στη διάταξη διαφάνειας.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
    IAutoShape placeholder = layout.PlaceholderManager.AddChartPlaceholder(20, 20, 200, 200);
}
```

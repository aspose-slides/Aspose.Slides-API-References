## LayoutPlaceholderManager.AddVerticalTextPlaceholder μέθοδος

Προσθέτει ένα νέο σχήμα placeholder στη διαφάνεια διάταξης για να κρατήσει το κείμενο σε κάθετη κατεύθυνση.

```csharp
public IAutoShape AddVerticalTextPlaceholder(float x, float y, float width, float height)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| x | Single | Η συντεταγμένη X του νέου σχήματος placeholder. |
| y | Single | Η συντεταγμένη Y του νέου σχήματος placeholder. |
| width | Single | Το πλάτος του νέου σχήματος placeholder. |
| height | Single | Το ύψος του νέου σχήματος placeholder. |

### Τιμή Επιστροφής

Δημιουργήθηκε [`IAutoShape`](../../iautoshape) με ένα Text (Vertical) placeholder.

### Παραδείγματα

Το παρακάτω παράδειγμα δείχνει πώς να προσθέσετε το σχήμα Text (Vertical) placeholder στη διαφάνεια διάταξης.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalTextPlaceholder(20, 20, 300, 500);
}
```

## ILayoutPlaceholderManager.AddVerticalContentPlaceholder method

Προσθέτει ένα νέο σχήμα placeholder στη διαφάνεια διάταξης για να περιέχει περιεχόμενο, όπως εικόνα, πίνακα, πολυμέσα ή κείμενο σε κατακόρυφη κατεύθυνση.

```csharp
public IAutoShape AddVerticalContentPlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| x | Single | Η συντεταγμένη X του νέου σχήματος placeholder. |
| y | Single | Η συντεταγμένη Y του νέου σχήματος placeholder. |
| width | Single | Το πλάτος του νέου σχήματος placeholder. |
| height | Single | Το ύψος του νέου σχήματος placeholder. |

### Return Value

Δημιουργήθηκε [`IAutoShape`](../../iautoshape) με έναν Content (Vertical) placeholder.

### Examples

Το παρακάτω παράδειγμα δείχνει πώς να προσθέσετε το σχήμα Content (Vertical) placeholder στη διαφάνεια διάταξης.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalContentPlaceholder(20, 20, 300, 500);
}
```

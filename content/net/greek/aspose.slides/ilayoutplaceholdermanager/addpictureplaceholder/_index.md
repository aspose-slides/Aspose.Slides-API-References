## ILayoutPlaceholderManager.AddPicturePlaceholder μέθοδος

Προσθέτει ένα νέο σχήμα αντικαταστήματος στη διαφάνεια διάταξης για να κρατήσει μια εικόνα.

```csharp
public IAutoShape AddPicturePlaceholder(float x, float y, float width, float height)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| x | Single | Η συντεταγμένη X του νέου σχήματος αντικαταστήματος. |
| y | Single | Η συντεταγμένη Y του νέου σχήματος αντικαταστήματος. |
| width | Single | Το πλάτος του νέου σχήματος αντικαταστήματος. |
| height | Single | Το ύψος του νέου σχήματος αντικαταστήματος. |

### Τιμή Επιστροφής

Δημιουργήθηκε [`IAutoShape`](../../iautoshape) με ένα υποκατάστατο Picture.

### Παραδείγματα

Το ακόλουθο παράδειγμα δείχνει πώς να προσθέσετε το σχήμα αντικαταστήματος Picture στη διαφάνεια διάταξης.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddPicturePlaceholder(20, 20, 200, 200);
}
```

## ILayoutPlaceholderManager.AddOnlineImagePlaceholder μέθοδος

Προσθέτει ένα νέο σχήμα placeholder στη διαφάνεια διάταξης για να φιλοξενήσει μια online εικόνα.

```csharp
public IAutoShape AddOnlineImagePlaceholder(float x, float y, float width, float height)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| x | Single | Η συντεταγμένη X του νέου σχήματος σύμβολο κράτησης. |
| y | Single | Η συντεταγμένη Y του νέου σχήματος σύμβολο κράτησης. |
| width | Single | Το πλάτος του νέου σχήματος σύμβολο κράτησης. |
| height | Single | Το ύψος του νέου σχήματος σύμβολο κράτησης. |

### Τιμή επιστροφής

Δημιουργήθηκε [`IAutoShape`](../../iautoshape) με ένα placeholder online εικόνας.

### Παραδείγματα

Το παρακάτω παράδειγμα δείχνει πώς να προσθέσετε το σχήμα placeholder online εικόνας στη διαφάνεια διάταξης.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddOnlineImagePlaceholder(20, 20, 200, 200);
}
```

## ILayoutPlaceholderManager.AddChartPlaceholder μέθοδος

Προσθέτει ένα νέο σύμβολο κράτησης σχήματος στη διαφάνεια διάταξης για να φιλοξενήσει ένα chart.

```csharp
public IAutoShape AddChartPlaceholder(float x, float y, float width, float height)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| x | Single | Η συντεταγμένη X του νέου σύμβολου κράτησης σχήματος. |
| y | Single | Η συντεταγμένη Y του νέου σύμβολου κράτησης σχήματος. |
| width | Single | Το πλάτος του νέου σύμβολου κράτησης σχήματος. |
| height | Single | Το ύψος του νέου σύμβολου κράτησης σχήματος. |

### Τιμή Επιστροφής

Δημιουργήθηκε [`IAutoShape`](../../iautoshape) με ένα placeholder Chart.

### Παραδείγματα

Το παρακάτω παράδειγμα δείχνει πώς να προσθέσετε το σχήμα σύμβολο κράτησης Chart στη διαφάνεια διάταξης.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddChartPlaceholder(20, 20, 200, 200);
}
```

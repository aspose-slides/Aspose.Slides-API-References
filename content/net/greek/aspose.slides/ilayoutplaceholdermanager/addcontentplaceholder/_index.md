## ILayoutPlaceholderManager.AddContentPlaceholder μέθοδος

Προσθέτει ένα νέο σχήμα κράτησης θέσης στη διαφάνεια διάταξης για την αποθήκευση περιεχομένου, όπως εικόνα, πίνακας, πολυμέσα ή κείμενο.

```csharp
public IAutoShape AddContentPlaceholder(float x, float y, float width, float height)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| x | Single | Η συντεταγμένη X του νέου σχήματος κράτησης θέσης. |
| y | Single | Η συντεταγμένη Y του νέου σχήματος κράτησης θέσης. |
| width | Single | Το πλάτος του νέου σχήματος κράτησης θέσης. |
| height | Single | Το ύψος του νέου σχήματος κράτησης θέσης. |

### Τιμή Επιστροφής

Δημιουργήθηκε [`IAutoShape`](../../iautoshape) με ένα σύμβολο κράτησης θέσης Content.

### Παραδείγματα

Το παρακάτω παράδειγμα δείχνει πώς να προσθέσετε το σχήμα Content placeholder στη διαφάνεια διάταξης.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
    IAutoShape placeholder = layout.PlaceholderManager.AddContentPlaceholder(20, 20, 500, 300);
}
```

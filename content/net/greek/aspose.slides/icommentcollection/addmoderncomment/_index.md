## ICommentCollection.AddModernComment μέθοδος

Προσθέστε νέο σύγχρονο σχόλιο στο τέλος μιας συλλογής.

```csharp
public IModernComment AddModernComment(string text, ISlide slide, IShape shape, PointF position, 
    DateTime creationTime)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| text | String | Απλό κείμενο ενός νέου σύγχρονου σχολίου. |
| slide | ISlide | Διαφάνεια σε μια παρουσίαση στην οποία θα προστεθεί ένα νέο σύγχρονο σχόλιο. |
| shape | IShape | Σχήμα σε μια διαφάνεια στο οποίο συνδέεται ένα νέο σύγχρονο σχόλιο. |
| position | PointF | Θέση σε μια διαφάνεια στην οποία θα προστεθεί ένα νέο σύγχρονο σχόλιο. |
| creationTime | DateTime | Ώρα δημιουργίας ενός σύγχρονου σχολίου. |

### Τιμή Επιστροφής

Το προστιθέμενο σύγχρονο σχόλιο.

### Παραδείγματα

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ICommentAuthor newAuthor = pres.CommentAuthors.AddAuthor("Some Author", "SA");
    newAuthor.Comments.AddModernComment("This is modern comment", pres.Slides[0], null, new PointF(100, 100), DateTime.Now);

    pres.Save(outPptxFileName, SaveFormat.Pptx);
}
```

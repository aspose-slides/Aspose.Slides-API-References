## ILayoutSlide.DrawingGuides ιδιότητα

Επιστρέφει μια συλλογή από οδηγούς σχεδίασης για τη διαφάνεια διάταξης. Μόνο ανάγνωση [`IDrawingGuidesCollection`](../../idrawingguidescollection)

```csharp
public IDrawingGuidesCollection DrawingGuides { get; }
```

### Παραδείγματα

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    var slideSize = pres.SlideSize.Size;
    
    IDrawingGuidesCollection guides = pres.LayoutSlides[0].DrawingGuides;
    // Προσθήκη του νέου κάθετου οδηγού σχεδίασης στα αριστερά του κέντρου της διαφάνειας
    guides.Add(Orientation.Vertical, slideSize.Width / 2 - 20f);
    
    pres.Save("LayoutDrawingGuides_out.pptx", SaveFormat.Pptx);
}
```

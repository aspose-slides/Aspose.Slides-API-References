## PresentationLockingBehavior απαρίθμηση

Αντιπροσωπεύει τη συμπεριφορά όσον αφορά τη διαχείριση της [`IPresentation`](../ipresentation) πηγής (αρχείο ή Stream) κατά τη φόρτωση και εργασία με μια παρουσία του [`IPresentation`](../ipresentation).

```csharp
public enum PresentationLockingBehavior
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| LoadAndRelease | `0` | Η πηγή θα κλειδωθεί μόνο για τη διάρκεια της εκτέλεσης του κατασκευαστή [`IPresentation`](../ipresentation). Αν το [`IsTemporaryFilesAllowed`](../iblobmanagementoptions/istemporaryfilesallowed) είναι ορισμένο σε false, όλα τα BLOB θα φορτωθούν στη μνήμη. Διαφορετικά, μπορεί να χρησιμοποιηθούν άλλοι μέθοδοι όπως προσωρινά αρχεία. Αυτή η συμπεριφορά είναι πιο αργή από το KeepLocked και αν είναι δυνατόν να μεταβιβαστεί η ιδιοκτησία της πηγής στο [`IPresentation`](../ipresentation), συνιστάται η χρήση του KeepLocked. |
| KeepLocked | `1` | Η πηγή θα κλειδωθεί για ολόκληρη τη διάρκεια ζωής της παρουσίασης [`IPresentation`](../ipresentation), μέχρι να απορριφθεί. Το [`IsTemporaryFilesAllowed`](../iblobmanagementoptions/istemporaryfilesallowed) πρέπει να οριστεί σε true για χρήση αυτής της συμπεριφοράς, αλλιώς θα προκληθεί εξαίρεση. Αυτή η συμπεριφορά συνιστάται, είναι ταχύτερη και καταναλώνει λιγότερη μνήμη από το LoadAndRelease. |

### Σχόλια

Η πηγή είναι η παράμετρος που περνάται στον κατασκευαστή [`IPresentation`](../ipresentation). Στο παρακάτω παράδειγμα, η πηγή είναι το αρχείο «pres.pptx»:

```csharp
LoadOptions loadOptions = new LoadOptions { 
  BlobManagementOptions = { PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked } };
using (IPresentation pres = new Presentation("pres.pptx", loadOptions)) { }
```

Για αυτό το παράδειγμα, η πηγή (αρχείο «pres.pptx») θα κλειδωθεί για τη διάρκεια μιας [`IPresentation`](../ipresentation) παρουσίασης, δηλαδή δεν μπορεί να τροποποιηθεί ή διαγραφεί από άλλη διαδικασία.

## PresentationLockingBehavior απαρίθμηση

Αντιπροσωπεί τη συμπεριφορά σχετικά με την επεξεργασία της πηγής [`IPresentation`](../ipresentation) (αρχείο ή Stream) κατά τη φόρτωση και τη χρήση μιας παρουσίας του [`IPresentation`](../ipresentation).

```csharp
public enum PresentationLockingBehavior
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| LoadAndRelease | `0` | Η πηγή θα κλειδωθεί μόνο για τη διάρκεια της εκτέλεσης του κατασκευαστή [`IPresentation`](../ipresentation). Εάν το [`IsTemporaryFilesAllowed`](../iblobmanagementoptions/istemporaryfilesallowed) οριστεί σε false, όλα τα BLOBs θα φορτωθούν στη μνήμη. Διαφορετικά, ενδέχεται να χρησιμοποιηθούν άλλα μέσα, όπως προσωρινά αρχεία. Αυτή η συμπεριφορά είναι πιο αργή από το KeepLocked και, εάν είναι δυνατόν να μεταβιβαστεί η ιδιοκτησία της πηγής στο [`IPresentation`](../ipresentation), συνιστάται η χρήση του KeepLocked. |
| KeepLocked | `1` | Η πηγή θα κλειδωθεί για ολόκληρη τη διάρκεια ζωής του [`IPresentation`](../ipresentation) αντίτυπου, μέχρι να καταργηθεί. Το [`IsTemporaryFilesAllowed`](../iblobmanagementoptions/istemporaryfilesallowed) πρέπει να οριστεί σε true για τη χρήση αυτής της συμπεριφοράς, διαφορετικά θα ριφθεί εξαίρεση. Αυτή η συμπεριφορά συνιστάται, είναι ταχύτερη και καταναλώνει λιγότερη μνήμη από το LoadAndRelease. |

### Παρατηρήσεις

Η πηγή είναι η παράμετρος που περνάται στον κατασκευαστή [`IPresentation`](../ipresentation). Στο παρακάτω παράδειγμα, η πηγή είναι το αρχείο "pres.pptx":

```csharp
LoadOptions loadOptions = new LoadOptions { 
  BlobManagementOptions = { PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked } };
using (IPresentation pres = new Presentation("pres.pptx", loadOptions)) { }
```

Για αυτό το παράδειγμα, η πηγή (αρχείο "pres.pptx") θα κλειδωθεί για τη διάρκεια ζωής του [`IPresentation`](../ipresentation) αντίτυπου, δηλαδή δεν μπορεί να αλλάξει ή να διαγραφεί από άλλη διεργασία.

## IDocumentProperties.LinksUpToDate ιδιότητα

Δηλώνει εάν οι υπερσύνδεσμοι σε ένα έγγραφο είναι ενημερωμένοι. Ορίστε αυτό το στοιχείο σε **true** για να δηλώσετε ότι οι υπερσύνδεσμοι είναι ενημερωμένοι. Ορίστε αυτό το στοιχείο σε **false** για να δηλώσετε ότι οι υπερσύνδεσμοι είναι μη ενημερωμένοι. Ανάγνωση/εγγραφή Boolean.

```csharp
public bool LinksUpToDate { get; set; }
```

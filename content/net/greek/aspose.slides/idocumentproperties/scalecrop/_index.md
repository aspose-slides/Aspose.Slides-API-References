## IDocumentProperties.ScaleCrop ιδιότητα

Δείχνει τη λειτουργία εμφάνισης της μικρογραφίας του εγγράφου. Ορίστε αυτό το στοιχείο σε **true** για να ενεργοποιήσετε την κλιμάκωση της μικρογραφίας του εγγράφου στην οθόνη. Ορίστε αυτό το στοιχείο σε **false** για να ενεργοποιήσετε την περικοπή της μικρογραφίας του εγγράφου ώστε να εμφανίζονται μόνο τα τμήματα που ταιριάζουν στην οθόνη. Ανάγνωση/εγγραφή Boolean.

```csharp
public bool ScaleCrop { get; set; }
```

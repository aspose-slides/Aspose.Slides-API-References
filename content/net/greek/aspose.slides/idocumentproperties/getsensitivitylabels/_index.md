## IDocumentProperties.GetSensitivityLabels μέθοδος

Ανακτά έναν πίνακα ευαισθητοποιημένων ετικετών από τις προσαρμοσμένες ιδιότητες του εγγράφου (Microsoft Information Protection SDK Metadata).

```csharp
public ISensitivityLabel[] GetSensitivityLabels()
```

### Παραδείγματα

Ο παρακάτω κώδικας δείχνει πώς να μετακινήσετε τις πληροφορίες των ευαίσθητων ετικετών από τις προσαρμοσμένες ιδιότητες του εγγράφου στη σύγχρονη συλλογή SensitivityLabels:

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    // Ανάκτηση ετικετών ευαισθησίας από τις προσαρμοσμένες ιδιότητες του εγγράφου
    ISensitivityLabel[] mipSensitivityLabels = pres.DocumentProperties.GetSensitivityLabels();

    ISensitivityLabelCollection sensitivityLabels = pres.SensitivityLabels;
    foreach (var sensitivityLabel in mipSensitivityLabels)
    {
        // Προσθήκη ετικέτας στη συλλογή
        // Εδώ μπορείτε να προσθέσετε έναν έλεγχο για την εγκυρότητα των πληροφοριών της ετικέτας (η ετικέτα είναι διαθέσιμη κλπ.)
        sensitivityLabels.Add(sensitivityLabel);
    }

    pres.Save("SensitivityLabel.pptx", SaveFormat.Pptx);
}
```

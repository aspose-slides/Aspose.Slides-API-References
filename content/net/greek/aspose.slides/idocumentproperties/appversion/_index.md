## IDocumentProperties.AppVersion ιδιότητα

Επιστρέφει την έκδοση της εφαρμογής. Μόνο ανάγνωση String.

public string AppVersion { get; }

### Παρατηρήσεις

Το περιεχόμενο αυτού του στοιχείου πρέπει να είναι της μορφής XX.YYYY, όπου τα X και Y αντιπροσωπεύουν αριθμητικές τιμές· διαφορετικά, το έγγραφο θεωρείται μη συμμορφωμένο. Το Aspose.Slides αντιπροσωπεύει την έκδοσή του στη μορφή XX.YYZZ, όπου: XX - κύρια έκδοση YY - δευτερεύουσα έκδοση ZZ - έκδοση διόρθωσης. Για παράδειγμα, η τιμή 23.0105 σημαίνει έκδοση Aspose.Slides 23.1.5.

## IPortion.PortionFormat ιδιότητα

Επιστρέφει το αντικείμενο μορφοποίησης που περιέχει ρητά ορισμένες ιδιότητες μορφοποίησης του τμήματος κειμένου χωρίς εφαρμογή κληρονομικότητας. Μόνο για ανάγνωση [`IPortionFormat`](../../iportionformat).

```csharp
public IPortionFormat PortionFormat { get; }
```

### Παρατηρήσεις

Το αντικείμενο μορφοποίησης περιέχει τις παραμέτρους μορφοποίησης που ορίζονται μόνο για το τρέχον τμήμα, τα κληρονομικά δεδομένα δεν εφαρμόζονται.

Για να λάβετε τις αποτελεσματικές τιμές, συμπεριλαμβανομένων των κληρονομικών, χρησιμοποιήστε τη μέθοδο [`GetEffective`](../../iportionformat/geteffective).

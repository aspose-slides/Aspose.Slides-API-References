## IViewProperties.GridSpacing ιδιότητα

Επιστρέφει ή ορίζει τη απόσταση του πλέγματος που πρέπει να χρησιμοποιείται για το πλέγμα που υποστηρίζει το έγγραφο παρουσίασης, σε σημεία. Ανάγνωση/Εγγραφή Single.

```csharp
public float GridSpacing { get; set; }
```

### Σχόλια

Η τιμή απόστασης πλέγματος πρέπει να είναι θετικός αριθμός. Η τυπική εμβέλεια τιμών είναι από 1 mm (2.8349607 σημεία) έως 2 ίντσες (144 σημεία).

### Παραδείγματα

Ο παρακάτω κώδικας παραδείγματος δείχνει πώς να αλλάξετε την απόσταση του πλέγματος σε παρουσίαση PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

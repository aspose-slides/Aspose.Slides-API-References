## AudioFrame.CaptionTracks ιδιότητα

Λαμβάνει τη συλλογή των κλειστών υποτίτλων που σχετίζονται με το πλαίσιο ήχου. Αυτή η ιδιότητα είναι μόνο για ανάγνωση και επιστρέφει ένα [`ICaptionsCollection`](../../icaptionscollection) που περιέχει όλα τα μονοπάτια υποτίτλων.

```csharp
public ICaptionsCollection CaptionTracks { get; }
```

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
using (Presentation pres = new Presentation("audio with captions.pptx"))
{
    foreach (IShape shape in pres.Slides[0].Shapes)
    {
        if (shape is IAudioFrame audioFrame)
        {
            // Αποθηκεύστε τα δυαδικά δεδομένα του κομματιού υπότιτλου ως αρχείο .vtt
            foreach (var captionTrack in audioFrame.CaptionTracks)
                File.WriteAllBytes(captionTrack.CaptionId + ".vtt", captionTrack.BinaryData);
        }
    }
}
```

## BasePortionFormat.SpellCheck ιδιότητα

Λαμβάνει ή ορίζει μια τιμή που υποδεικνύει εάν η ορθογραφική διόρθωση είναι ενεργοποιημένη για το τμήμα κειμένου. Όταν αυτή η ιδιότητα οριστεί σε false, οι ορθογραφικοί έλεγχοι για στοιχεία κειμένου καταστέλλονται. Όταν οριστεί σε true, η ορθογραφική διόρθωση επιτρέπεται. Η προεπιλεγμένη τιμή είναι `false`.

```csharp
public bool SpellCheck { get; set; }
```

### Παραδείγματα

Το επόμενο παράδειγμα δείχνει την ενεργοποίηση της σημαίας SpellCheck πριν από την αποθήκευση της παρουσίασης:

```csharp
[C#]
using (var pres = new Presentation("input.pptx"))
{
    // Πρόσβαση στο πρώτο τμήμα κειμένου μέσα στο πρώτο σχήμα στην πρώτη διαφάνεια
    var portion = ((AutoShape)pres.Slides[0].Shapes[0]).TextFrame.Paragraphs[0].Portions[0];
    // Ενεργοποίηση ορθογραφικού ελέγχου για αυτό το τμήμα κειμένου
    portion.PortionFormat.SpellCheck = true;
    // Αποθήκευση της τροποποιημένης παρουσίασης
    pres.Save("output-with-spellcheck.pptx", SaveFormat.Pptx);
}
```

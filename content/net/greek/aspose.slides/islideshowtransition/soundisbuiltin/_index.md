## ISlideShowTransition.SoundIsBuiltIn ιδιότητα

Καθορίζει αν αυτό το ήχος είναι ενσωματωμένος ή όχι. Εάν αυτό το χαρακτηριστικό οριστεί σε true, η εφαρμογή που δημιουργεί το αρχείο ειδοποιείται να ελέγξει το χαρακτηριστικό name που έχει οριστεί για αυτόν τον ήχο στη λίστα των ενσωματωμένων ήχων και μπορεί, εφόσον χρειαστεί, να εμφανίσει ένα προσαρμοσμένο όνομα ή διεπαφή χρήστη. Αναγνώσιμη-εγγράψιμη Boolean.

```csharp
public bool SoundIsBuiltIn { get; set; }
```

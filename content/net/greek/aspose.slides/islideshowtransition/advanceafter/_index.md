## ISlideShowTransition.AdvanceAfter ιδιότητα

Αυτό το χαρακτηριστικό καθορίζει αν η παρουσίαση θα μεταβεί στην επόμενη διαφάνεια μετά από κάποιο χρονικό διάστημα. Ανάγνωση/εγγραφή Boolean.

```csharp
public bool AdvanceAfter { get; set; }
```

### Παραδείγματα

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    // Λάβετε την πρώτη μετάβαση διαφάνειας
    ISlideShowTransition slideTransition = pres.Slides[0].SlideShowTransition;
    
    // Ελέγξτε αν η σημαία Advance Slide After είναι ενεργοποιημένη
    if (slideTransition.AdvanceAfter)
    {
        // Λάβετε την τιμή χρόνου Advance Slide After
        uint advanceAfterTime = slideTransition.AdvanceAfterTime;
    }
}
```

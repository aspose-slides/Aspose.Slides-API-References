## ITextFrame.ReplaceRegex μέθοδος

Αντικαθιστά όλα τα αποτελέσματα της κανονικής έκφρασης με το συγκεκριμένο κείμενο.

```csharp
public void ReplaceRegex(Regex regex, string newText, IFindResultCallback callback)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| regex | Regex | Η κανονική έκφραση Regex για την εύρεση των συμβολοσειρών που θα αντικατασταθούν. |
| newText | String | Η συμβολοσειρά που θα αντικαταστήσει όλες τις εμφανίσεις των συμβολοσειρών που πρόκειται να αντικατασταθούν. |
| callback | IFindResultCallback | Το αντικείμενο callback για τη λήψη των αποτελεσμάτων αναζήτησης [`IFindResultCallback`](../../ifindresultcallback). |

### Παραδείγματα

Το παρακάτω δείγμα κώδικα δείχνει πώς να αντικαταστήσετε κείμενο χρησιμοποιώντας κανονική έκφραση με το συγκεκριμένο κείμενο.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// Αντικαθιστά όλες τις λέξεις με 10 ή περισσότερους χαρακτήρες με '***'
	((AutoShape)presentation.Slides[0].Shapes[0]).TextFrame.ReplaceRegex(regex, "***", null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

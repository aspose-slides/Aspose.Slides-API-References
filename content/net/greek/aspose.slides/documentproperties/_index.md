### Παραδείγματα

Το παρακάτω παράδειγμα δείχνει πώς να αποκτήσετε πρόσβαση στις ενσωματωμένες Ιδιότητες μιας παρουσίασης PowerPoint.

```csharp
[C#]
// Δημιουργία της κλάσης Presentation που αντιπροσωπεύει την παρουσίαση
using (Presentation pres = new Presentation(dataDir + "AccessBuiltin Properties.pptx"))
{
	// Δημιουργία αναφοράς στο αντικείμενο IDocumentProperties που σχετίζεται με την Presentation
	IDocumentProperties documentProperties = pres.DocumentProperties;
	// Εμφάνιση των ενσωματωμένων ιδιοτήτων
	Console.WriteLine("Category : " + documentProperties.Category);
	Console.WriteLine("Current Status : " + documentProperties.ContentStatus);
	Console.WriteLine("Creation Date : " + documentProperties.CreatedTime);
	Console.WriteLine("Author : " + documentProperties.Author);
	Console.WriteLine("Description : " + documentProperties.Comments);
}
```

Το παρακάτω παράδειγμα δείχνει πώς να τροποποιήσετε τις ενσωματωμένες Ιδιότητες μιας παρουσίασης PowerPoint.

```csharp
[C#]
// Δημιουργία της κλάσης Presentation που αντιπροσωπεύει την Presentation
using (Presentation presentation = new Presentation(dataDir + "ModifyBuiltinProperties.pptx"))
{
	// Δημιουργία αναφοράς στο αντικείμενο IDocumentProperties που σχετίζεται με την Presentation
	IDocumentProperties documentProperties = presentation.DocumentProperties;
	// Ορισμός των ενσωματωμένων ιδιοτήτων
	documentProperties.Author = "Aspose.Slides for .NET";
	documentProperties.Title = "Modifying Presentation Properties";
	documentProperties.Subject = "Aspose Subject";
	// Αποθήκευση της παρουσίασής σας σε αρχείο
	presentation.Save(dataDir + "DocumentProperties_out.pptx", SaveFormat.Pptx);
}
```

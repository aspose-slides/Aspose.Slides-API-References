## DocumentProperties.ScaleCrop ιδιότητα

Δείχνει τη λειτουργία εμφάνισης του μικρογραφίας του εγγράφου. Ορίστε αυτό το στοιχείο σε **true** για να ενεργοποιήσετε την κλιμάκωση της μικρογραφίας του εγγράφου στην οθόνη. Ορίστε αυτό το στοιχείο σε **false** για να ενεργοποιήσετε τη περικοπή της μικρογραφίας του εγγράφου ώστε να εμφανίζει μόνο τμήματα που ταιριάζουν στην οθόνη. Ανάγνωση/Εγγραφή Boolean.

```csharp
public bool ScaleCrop { get; set; }
```

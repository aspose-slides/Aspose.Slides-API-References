## DocumentProperties.LinksUpToDate ιδιότητα

Δηλώνει εάν οι υπερσύνδεσμοι σε ένα έγγραφο είναι ενημερωμένοι. Ορίστε αυτό το στοιχείο σε **true** για να υποδείξετε ότι οι υπερσύνδεσμοι είναι ενημερωμένοι. Ορίστε αυτό το στοιχείο σε **false** για να υποδείξετε ότι οι υπερσύνδεσμοι είναι ξεπερασμένοι. Ανάγνωση/Εγγραφή Boolean.

public bool LinksUpToDate { get; set; }

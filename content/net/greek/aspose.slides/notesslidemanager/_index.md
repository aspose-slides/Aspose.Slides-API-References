### Παραδείγματα

Το παρακάτω παράδειγμα δείχνει πώς να προσθέσετε σημειώσεις σε συγκεκριμένη διαφάνεια παρουσίασης PowerPoint.

```csharp
[C#]
	// Δημιουργεί ένα αντικείμενο Presentation που αντιπροσωπεύει ένα αρχείο παρουσίασης
	using(Presentation presentation = new Presentation(dataDir + "AccessSlides.pptx")) {
	  // Προσθέτει σημειώσεις στην πρώτη διαφάνεια
	  INotesSlideManager mgr = presentation.Slides[0].NotesSlideManager;
	  INotesSlide noteSlide = mgr.AddNotesSlide();
	  noteSlide.NotesTextFrame.Text = "Your Notes";
	  // Αποθηκεύει την παρουσίαση στο δίσκο
	  presentation.Save(dataDir + "RemoveNotesAtSpecificSlide_out.pptx", SaveFormat.Pptx);
	}
```

Το παρακάτω παράδειγμα δείχνει πώς να καταργήσετε τις σημειώσεις από τη συγκεκριμένη διαφάνεια μιας παρουσίασης PowerPoint.

```csharp
[C#]
	// Δημιουργεί ένα αντικείμενο Presentation που αντιπροσωπεύει ένα αρχείο παρουσίασης
	using(Presentation presentation = new Presentation(dataDir + "AccessSlides.pptx")) {
	  // Κατάργηση σημειώσεων της πρώτης διαφάνειας
	  INotesSlideManager mgr = presentation.Slides[0].NotesSlideManager;
	  mgr.RemoveNotesSlide();
	  // Αποθήκευση παρουσίασης στο δίσκο
	  presentation.Save(dataDir + "RemoveNotesAtSpecificSlide_out.pptx", SaveFormat.Pptx);
	}
```

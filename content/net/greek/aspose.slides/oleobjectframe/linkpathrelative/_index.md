## OleObjectFrame.LinkPathRelative ιδιότητα

Επιστρέφει τη σχετική διαδρομή σε ένα συνδεδεμένο αρχείο αν υπάρχει, αλλιώς επιστρέφει μια κενή συμβολοσειρά. Μόνο για ανάγνωση String.

```csharp
public string LinkPathRelative { get; }
```

### Παρατηρήσεις

Στις παρουσιάσεις Ppt, ορισμένοι σύνδεσμοι Ole object ενδέχεται να έχουν σχετική αναπαράσταση.

### Παραδείγματα

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.ppt"))
{
    IOleObjectFrame oleFrame = presentation.Slides[0].Shapes[0] as IOleObjectFrame;

    if (oleFrame != null)
    {
        Console.WriteLine("The relative path: " + oleFrame.LinkPathRelative);
    } 
}
```

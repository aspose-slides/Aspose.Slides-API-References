## IBasePortionFormat.SpellCheck ιδιότητα

Λαμβάνει ή ορίζει μια τιμή που υποδεικνύει εάν ο ορθογραφικός έλεγχος είναι ενεργοποιημένος για το τμήμα κειμένου. Όταν αυτή η ιδιότητα ορίζεται σε false, οι ορθογραφικοί έλεγχοι για στοιχεία κειμένου καταστέλλονται. Όταν ορίζεται σε true, επιτρέπεται ο ορθογραφικός έλεγχος. Η προεπιλεγμένη τιμή είναι `false`.

```csharp
public bool SpellCheck { get; set; }
```

### Παραδείγματα

Το παρακάτω παράδειγμα δείχνει την ενεργοποίηση της σημαίας SpellCheck πριν από την αποθήκευση της παρουσίασης:

```csharp
[C#]
using (var pres = new Presentation("input.pptx"))
{
    // Πρόσβαση στο πρώτο τμήμα κειμένου μέσα στο πρώτο σχήμα της πρώτης διαφάνειας
    var portion = ((AutoShape)pres.Slides[0].Shapes[0]).TextFrame.Paragraphs[0].Portions[0];
    // Ενεργοποίηση του ορθογραφικού ελέγχου για αυτό το τμήμα κειμένου
    portion.PortionFormat.SpellCheck = true;
    // Αποθήκευση της τροποποιημένης παρουσίασης
    pres.Save("output-with-spellcheck.pptx", SaveFormat.Pptx);
}
```

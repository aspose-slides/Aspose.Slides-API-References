## WriteAsSvg(Stream) {#writeassvg}

Αποθηκεύει το περιεχόμενο της διαφάνειας ως αρχείο SVG.


| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |

### Παραδείγματα

Το ακόλουθο παράδειγμα κώδικα δείχνει πώς να μετατρέψετε την πρώτη διαφάνεια από μια παρουσίαση PowerPoint σε αρχείο SVG.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.svg"))
    {
        // Αποθηκεύει την πρώτη διαφάνεια ως αρχείο SVG
        pres.Slides[0].WriteAsSvg(fileStream);
    }
}
```

```csharp
public void WriteAsSvg(Stream stream, ISVGOptions svgOptions)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| stream | Stream | Target stream |
| svgOptions | ISVGOptions | SVG generation options |

### Παραδείγματα

Το ακόλουθο παράδειγμα κώδικα δείχνει πώς να μετατρέψετε την πρώτη διαφάνεια από μια παρουσίαση PowerPoint σε αρχείο SVG με επιλογές.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.svg"))
    {
        var options = new SVGOptions() { VectorizeText = true };
        // Αποθηκεύει την πρώτη διαφάνεια ως αρχείο SVG
        pres.Slides[0].WriteAsSvg(fileStream, options);
    }
}
```

## Slide.WriteAsEmf μέθοδος

Αποθηκεύει το περιεχόμενο της διαφάνειας ως αρχείο EMF.

```csharp
public void WriteAsEmf(Stream stream)
```

| Parameter | Type | Description |

| stream | Stream | Ροή προορισμού |

### Exceptions

| exception | condition |
| --- | --- |
| ArgumentNullException | Η ροή προορισμού είναι `null` |

### Examples

Το παρακάτω παράδειγμα κώδικα δείχνει πώς να μετατρέψετε την πρώτη διαφάνεια από μια παρουσίαση PowerPoint σε μετααρχείο.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.emf"))
    {
        // Αποθηκεύει την πρώτη διαφάνεια ως μετααρχείο
        pres.Slides[0].WriteAsEmf(fileStream);
    }
}
```

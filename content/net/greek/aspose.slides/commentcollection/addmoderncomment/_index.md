## CommentCollection.AddModernComment μέθοδος

Προσθέτει νέο μοντέρνο σχόλιο στο τέλος μιας συλλογής.

```csharp
public IModernComment AddModernComment(string text, ISlide slide, IShape shape, PointF position, 
    DateTime creationTime)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| text | String | Απλό κείμενο ενός νέου μοντέρνου σχολίου. |
| slide | ISlide | Διαφάνεια σε μια παρουσίαση στην οποία προστίθεται νέο μοντέρνο σχόλιο. |
| shape | IShape | Σχήμα σε μια διαφάνεια στο οποίο συσχετίζεται ένα νέο μοντέρνο σχόλιο. |
| position | PointF | Θέση σε μια διαφάνεια όπου θα προστεθεί νέο μοντέρνο σχόλιο. |
| creationTime | DateTime | Ώρα δημιουργίας ενός μοντέρνου σχολίου. |

### Τιμή επιστροφής

Το προστεθέν μοντέρνο σχόλιο.

### Παραδείγματα

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ICommentAuthor newAuthor = pres.CommentAuthors.AddAuthor("Some Author", "SA");
    newAuthor.Comments.AddModernComment("This is modern comment", pres.Slides[0], null, new PointF(100, 100), DateTime.Now);

    pres.Save(outPptxFileName, SaveFormat.Pptx);
}
```

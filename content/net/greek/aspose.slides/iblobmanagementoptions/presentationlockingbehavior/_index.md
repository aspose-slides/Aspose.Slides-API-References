## IBlobManagementOptions.PresentationLockingBehavior ιδιότητα

Αυτή η ιδιότητα ορίζει εάν ένα αντίγραφο της κλάσης Presentation μπορεί να είναι ιδιοκτήτης της πηγής – αρχείου ή ροής – κατά τη διάρκεια της ζωής του αντίγραφου. Εάν το αντίγραφο είναι ιδιοκτήτης, κλειδώνει την πηγή. Αυτό βοηθά στη βελτίωση της κατανάλωσης μνήμης και της απόδοσης κατά την εργασία με BLOBs, αλλά η πηγή (ροή ή αρχείο) δεν μπορεί να αλλάξει κατά τη διάρκεια της ζωής του αντίγραφου Presentation. Αυτό είναι ένα παράδειγμα:

```csharp
public PresentationLockingBehavior PresentationLockingBehavior { get; set; }
```

### Παραδείγματα

```csharp
LoadOptions loadOptions = new LoadOptions { 
  BlobManagementOptions = {PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked} };
using (Presentation pres = new Presentation("pres.pptx", loadOptions)) {
  // Θα προκληθεί IOException επειδή το pres.pptx είναι κλειδωμένο για τη διάρκεια ζωής ενός Presentation
  // File.Delete("pres.pptx");
}
// μετά τη διάσωση του αντικειμένου Presentation, το αρχείο ξεκλειδώνεται και μπορεί να διαγραφεί
File.Delete("pres.pptx");
```

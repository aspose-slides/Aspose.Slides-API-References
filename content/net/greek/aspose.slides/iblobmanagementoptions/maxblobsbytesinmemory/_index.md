## IBlobManagementOptions.MaxBlobsBytesInMemory ιδιότητα

Ορίζει το μέγιστο συνολικό μέγεθος (σε bytes) που μπορούν να καταλάβουν όλα τα BLOBs στη μνήμη. Από προεπιλογή, όλα τα BLOBs φορτώνονται στη μνήμη· μόνο όταν αυτό το όριο φτάσει χρησιμοποιούνται εναλλακτικοί μηχανισμοί (όπως προσωρινά αρχεία). Η διατήρηση των BLOBs στη μνήμη μεγιστοποιεί την απόδοση, αλλά μπορεί να οδηγήσει σε υψηλή χρήση μνήμης. Χρησιμοποιήστε αυτήν την ιδιότητα για να προσαρμόσετε τη συμπεριφορά στο περιβάλλον ή τις απαιτήσεις σας.

```csharp
public ulong MaxBlobsBytesInMemory { get; set; }
```

### Παρατηρήσεις

Αυτή η ιδιότητα αγνοείται εάν το [`IsTemporaryFilesAllowed`](../istemporaryfilesallowed) έχει οριστεί σε false, επειδή τότε η μνήμη είναι η μόνη διαθέσιμη τοποθεσία αποθήκευσης και ο περιορισμός της χρήσης BLOB στη μνήμη δεν έχει καμία επίδραση.

Η προεπιλεγμένη τιμή είναι 629,145,600 bytes (600 MB).

Μπορείτε να ορίσετε αυτήν την ιδιότητα στο μηδέν, αλλά θα κρατηθεί ακόμα ένα μικρό ελάχιστο ποσό μνήμης.

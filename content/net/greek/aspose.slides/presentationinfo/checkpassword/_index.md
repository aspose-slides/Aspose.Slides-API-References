## PresentationInfo.CheckPassword μέθοδος

Ελέγχει εάν ένας κωδικός πρόσβασης είναι σωστός για μια παρουσίαση που προστατεύεται με ανοιχτό κωδικό.

```csharp
public bool CheckPassword(string password)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| password | String | Ο κωδικός πρόσβασης για έλεγχο. |

### Τιμή Επιστροφής

True εάν η παρουσίαση προστατεύεται με ανοιχτό κωδικό και ο κωδικός πρόσβασης είναι σωστός, και false σε άλλη περίπτωση.

### Εξαιρέσεις

| εξαίρεση | συνθήκη |
| --- | --- |
| InvalidOperationException |  |
| NotSupportedException |  |

### Παρατηρήσεις

Όταν ο κωδικός πρόσβασης είναι null ή κενό, αυτή η μέθοδος επιστρέφει false.

### Παραδείγματα

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
bool isPasswordCorrect = info.CheckPassword("my_password");
```

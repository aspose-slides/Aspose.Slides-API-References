## IAudioFrame.TrimFromEnd ιδιότητα

Καθορίζει τη χρονική διάρκεια που θα αφαιρεθεί από το τέλος του μέσου κατά την αναπαραγωγή, σε χιλιοστά του δευτερολέπτου. Ανάγνωση/εγγραφή Single.

```csharp
public float TrimFromEnd { get; set; }
```

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    IAudio audio = pres.Audios.AddAudio(File.ReadAllBytes("sampleaudio.mp3"));
    IAudioFrame audioFrame = pres.Slides[0].Shapes.AddAudioFrameEmbedded(50, 50, 100, 100, audio);

    // Ορίστε τη διάρκεια αποκοπής από το τέλος 2 δευτερόλεπτα
    audioFrame.TrimFromEnd = 2000f;
}
```

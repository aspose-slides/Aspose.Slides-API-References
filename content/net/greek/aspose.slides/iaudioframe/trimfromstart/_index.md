## IAudioFrame.TrimFromStart ιδιότητα

Καθορίζει τη χρονική διάρκεια που πρέπει να αφαιρεθεί από την αρχή του μέσου κατά την αναπαραγωγή, σε χιλιοστά του δευτερολέπτου. Read/write Single.

```csharp
public float TrimFromStart { get; set; }
```

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    IAudio audio = pres.Audios.AddAudio(File.ReadAllBytes("sampleaudio.mp3"));
    IAudioFrame audioFrame = pres.Slides[0].Shapes.AddAudioFrameEmbedded(50, 50, 100, 100, audio);

    // Ορίστε το χρόνο περικοπής από την αρχή 1,5 δευτερόλεπτα
    audioFrame.TrimFromStart = 1500f;
}
```

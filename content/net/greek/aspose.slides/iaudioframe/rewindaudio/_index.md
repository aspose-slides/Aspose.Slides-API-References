## IAudioFrame.RewindAudio ιδιότητα

Καθορίζει εάν ένα αρχείο ήχου επαναφέρεται αυτόματα στην αρχή μετά την αναπαραγωγή. Ανάγνωση/εγγραφή Boolean.

```csharp
public bool RewindAudio { get; set; }
```

### Παραδείγματα

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ISlide slide = pres.Slides[0];

    // Προσθήκη πλαισίου ήχου
    IAudioFrame audioFrame = slide.Shapes.AddAudioFrameLinked(50, 50, 100, 100, "sampleaudio.wav");

    // Ορίστε τον ήχο να παίζει σε όλες τις διαφάνειες
    audioFrame.PlayAcrossSlides = true;

    // Ορίστε τον ήχο να επανέρχεται αυτόματα στην αρχή μετά την αναπαραγωγή
    audioFrame.RewindAudio = true;

    pres.Save("AudioFrame_out.pptx", SaveFormat.Pptx);
}
```

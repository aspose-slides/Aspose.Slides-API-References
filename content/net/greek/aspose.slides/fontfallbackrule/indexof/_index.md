## FontFallBackRule.IndexOf μέθοδος

Επιστρέφει το δείκτη του καθορισμένου κανόνα στη συλλογή.

```csharp
public int IndexOf(string fontName)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| fontName | String | Όνομα γραμματοσειράς που θα βρεθεί. |

### Τιμή Επιστροφής

Δείκτης μιας γραμματοσειράς ή -1 εάν η γραμματοσειρά δεν βρεθεί στη λίστα.

### Παραδείγματα

```csharp
[C#]
// Δημιουργήστε έναν κανόνα που περιέχει μια λίστα γραμματοσειρών.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// Πάρτε το δείκτη του Tahoma.
int tahomaIndex = newRule.IndexOf("Tahoma");
```

## FontFallBackRule.Remove μέθοδος

Αφαιρεί την πρώτη εμφάνιση μιας συγκεκριμένης γραμματοσειράς FallBack από τη λίστα.

```csharp
public void Remove(string fontName)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| fontName | String | Το όνομα της γραμματοσειράς που θα αφαιρεθεί από τη λίστα. |

### Παραδείγματα

```csharp
[C#]
// Δημιουργεί έναν κανόνα που περιέχει μια λίστα γραμματοσειρών.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// Αφαιρεί το Tahoma από τη λίστα.
newRule.Remove ("Tahoma");
```

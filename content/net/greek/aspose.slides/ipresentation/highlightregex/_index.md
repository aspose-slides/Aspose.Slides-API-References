## IPresentation.HighlightRegex method

Επισημαίνει όλα τα ταιριάσματα της κανονικής έκφρασης με το καθορισμένο χρώμα.

```csharp
public void HighlightRegex(Regex regex, Color highlightColor, IFindResultCallback callback)
```

| Parameter | Type | Description |
| --- | --- | --- |
| regex | Regex | Η κανονική έκφραση Regex για λήψη των συμβολοσειρών προς επισήμανση. |
| highlightColor | Color | Το χρώμα για επισήμανση του κειμένου. |
| callback | IFindResultCallback | Το αντικείμενο callback για λήψη των αποτελεσμάτων αναζήτησης [`IFindResultCallback`](../../ifindresultcallback). |

### Παραδείγματα

Το παρακάτω δείγμα κώδικα δείχνει πώς να επισημάνετε κείμενο σε μια παρουσίαση PowerPoint χρησιμοποιώντας μια κανονική έκφραση.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// επισημαίνοντας όλες τις λέξεις με 10 ή περισσότερους χαρακτήρες
	presentation.HighlightRegex(regex, Color.Blue, null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

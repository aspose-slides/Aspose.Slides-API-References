## ILoadOptions.BlobManagementOptions ιδιότητα

Αντιπροσωπεύει τις επιλογές που μπορούν να χρησιμοποιηθούν για τη διαχείριση της συμπεριφοράς επεξεργασίας Μεγάλων Δυαδικών Αντικειμένων (BLOBs), όπως η χρήση προσωρινών αρχείων ή το μέγιστο αριθμό bytes BLOB στη μνήμη. Αυτές οι επιλογές προορίζονται να ρυθμίσουν τον καλύτερο λόγο απόδοσης/κατανάλωσης μνήμης για ένα συγκεκριμένο περιβάλλον ή απαιτήσεις. Ένα Μεγάλο Δυαδικό Αντικείμενο (BLOB) είναι δυαδικά δεδομένα αποθηκευμένα ως μία ενιαία οντότητα· δηλαδή, ένα BLOB μπορεί να είναι ήχος, βίντεο ή η παρουσίαση αυτή καθ' αυτή.

```csharp
public IBlobManagementOptions BlobManagementOptions { get; set; }
```

## ILoadOptions.InterruptionToken ιδιότητα

Το token για την παρακολούθηση αιτήσεων διακοπής. Αυτό το token διαχειρίζεται ολόκληρη τη διάρκεια ζωής του [`IPresentation`](../../ipresentation). Οποιαδήποτε λειτουργία μεγάλης διάρκειας, όπως η φόρτωση ή η αποθήκευση παρουσίασης, θα διακοπεί με την κλήση της [`Interrupt`](../../iinterruptiontokensource/interrupt) μεθόδου του [`IInterruptionTokenSource`](../../iinterruptiontokensource).

public IInterruptionToken InterruptionToken { get; set; }

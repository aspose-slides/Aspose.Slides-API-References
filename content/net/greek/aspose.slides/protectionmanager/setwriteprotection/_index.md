## ProtectionManager.SetWriteProtection μέθοδος

Ορίζει την προστασία εγγραφής για αυτήν την παρουσίαση με τον καθορισμένο κωδικό πρόσβασης.

```csharp
public void SetWriteProtection(string password)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| password | String | Ο κωδικός πρόσβασης. |

### Παραδείγματα

Ο παρακάτω κώδικας δείγματος δείχνει πώς να ορίσετε προστασία εγγραφής σε μια παρουσίαση.

```csharp
[C#]
using (Presentation presentation = new Presentation("pres.pptx"))
{
    presentation.ProtectionManager.SetWriteProtection("123123");
    presentation.Save("write-protected-pres.pptx", SaveFormat.Pptx);
}
```

## Portion.PortionFormat ιδιότητα

Επιστρέφει αντικείμενο μορφοποίησης που περιέχει ρητά ορισμένες ιδιότητες μορφοποίησης του τμήματος κειμένου χωρίς να έχει εφαρμοστεί κληρονομικότητα. Μόνο για ανάγνωση [`IPortionFormat`](../../iportionformat).

```csharp
public IPortionFormat PortionFormat { get; }
```

### Σχόλια

Το αντικείμενο μορφοποίησης περιέχει τις παραμέτρους μορφοποίησης που ορίζονται μόνο για το τρέχον τμήμα, χωρίς να εφαρμοστούν κληρονομημένα δεδομένα.

Για να λάβετε τις αποτελεσματικές τιμές, συμπεριλαμβανομένων των κληρονομημένων, χρησιμοποιήστε τη μέθοδο [`GetEffective`](../../portionformat/geteffective).

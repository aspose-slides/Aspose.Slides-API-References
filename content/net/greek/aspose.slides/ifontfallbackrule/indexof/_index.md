## IFontFallBackRule.IndexOf μέθοδος

Επιστρέφει έναν δείκτη του καθορισμένου κανόνα στη συλλογή.

```csharp
public int IndexOf(string fontName)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| fontName | String | Το όνομα της γραμματοσειράς που θα βρεθεί. |

### Τιμή Επιστροφής

Δείκτης μιας γραμματοσειράς ή -1 εάν η γραμματοσειρά δεν βρέθηκε στη λίστα.

### Παραδείγματα

```csharp
[C#]
// Δημιουργεί έναν κανόνα που περιέχει μια λίστα γραμματοσειρών.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

//Λάβετε δείκτη του Tahoma
int tahomaIndex = newRule.IndexOf("Tahoma");
```

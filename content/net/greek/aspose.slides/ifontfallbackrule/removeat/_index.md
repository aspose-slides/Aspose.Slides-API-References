## IFontFallBackRule.RemoveAt μέθοδος

Αφαιρεί τη γραμματοσειρά FallBack στο δείκτη που καθορίζεται στη λίστα.

```csharp
public void RemoveAt(int index)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| index | Int32 | Ο δείκτης που αρχίζει από το μηδέν της γραμματοσειράς προς διαγραφή. |

### Παραδείγματα

```csharp
[C#]
// Δημιουργήστε έναν κανόνα που περιέχει μια λίστα γραμματοσειρών.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

//Αφαίρεση του Tahoma από τη λίστα
newRule.Remove (2);
```

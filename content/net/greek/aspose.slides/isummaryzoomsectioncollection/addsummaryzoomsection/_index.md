## ISummaryZoomSectionCollection.AddSummaryZoomSection μέθοδος

Δημιουργεί νέο αντικείμενο Summary Zoom Section και το προσθέτει στη συλλογή

```csharp
public ISummaryZoomSection AddSummaryZoomSection(ISection section)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| section | ISection | Τμήμα για ένα νέο Summary Zoom Section στοιχείο [`ISection`](../../isection) |

### Τιμή επιστροφής

Προστέθηκε [`ISummaryZoomFrame`](../../isummaryzoomframe) στοιχείο

### Παρατηρήσεις

Εάν υπάρχει ήδη ένα στοιχείο για αυτό το τμήμα στη συλλογή, επιστρέφεται το υπάρχον στοιχείο.

### Παραδείγματα

Το παράδειγμα δείχνει την λήψη του στοιχείου Summary Zoom Section με βάση το ευρετήριο:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    ISummaryZoomFrame zoomFrame = (ISummaryZoomFrame) pres.Slides[1].Shapes[0];
    ISummaryZoomSectionCollection collection = zoomFrame.SummaryZoomCollection;
    ISummaryZoomSection newZoomSection = collection.AddSummaryZoomSection(pres.Sections[3]);
}
```

## Metered.SetMeteredKey μέθοδος

Ορίζει το δημόσιο και ιδιωτικό κλειδί με μέτρηση. Εάν αγοράσετε άδεια με μέτρηση, κατά την εκκίνηση της εφαρμογής, θα πρέπει να κληθεί αυτό το API· συνήθως, αυτό αρκεί. Ωστόσο, εάν αποτύχει συνεχώς η αποστολή των δεδομένων κατανάλωσης και υπερβεί τις 24 ώρες, η άδεια θα μετατραπεί σε κατάσταση αξιολόγησης· για να αποφύγετε αυτή την περίπτωση, θα πρέπει να ελέγχετε τακτικά την κατάσταση της άδειας· εάν είναι σε κατάσταση αξιολόγησης, καλέστε πάλι αυτό το API.

```csharp
public void SetMeteredKey(string publicKey, string privateKey)
```

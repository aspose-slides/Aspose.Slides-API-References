## AddFromPdf(Stream, PdfImportOptions) {#addfrompdf_1}

Δημιουργεί διαφάνειες από το έγγραφο PDF και τις προσθέτει στο τέλος της συλλογής.

```csharp
public ISlide[] AddFromPdf(Stream pdfStream, PdfImportOptions pdfImportOptions)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| pdfStream | Stream | Ροή που θα χρησιμοποιηθεί ως πηγή του εγγράφου PDF |

### Return Value

Διαφάνειες που προστέθηκαν

### Examples

Παράδειγμα:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    using (Stream stream = new FileStream("document.pdf", FileMode.Open, FileAccess.Read, FileShare.Read))
```

## InsertConnector(int, ShapeType, float, float, float, float) {#insertconnector}

```csharp
public IConnector InsertConnector(int index, ShapeType shapeType, float x, float y, float width, 
    float height)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| index | Int32 | Ο δείκτης μηδενικής βάσης στη θέση όπου θα εισαχθεί το σχήμα σύνδεσμου. |
| shapeType | ShapeType | Το [`ShapeType`](../../shapetype) του σχήματος σύνδεσμου που θα εισαχθεί. |
| x | Single | Η συντεταγμένη x του πλαισίου του σύνδεσμου, σε σημεία. |
| y | Single | Η συντεταγμένη y του πλαισίου του σύνδεσμου, σε σημεία. |
| width | Single | Το πλάτος του πλαισίου του σύνδεσμου, σε σημεία. |
| height | Single | Το ύψος του πλαισίου του σύνδεσμου, σε σημεία. |

## InsertConnector(int, ShapeType, float, float, float, float, bool) {#insertconnector_1}

public IConnector InsertConnector(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)

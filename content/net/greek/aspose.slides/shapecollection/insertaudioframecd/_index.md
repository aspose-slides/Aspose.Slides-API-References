## ShapeCollection.InsertAudioFrameCD μέθοδος

Δημιουργεί ένα νέο πλαίσιο ήχου συνδεδεμένο με ένα κομμάτι CD και το εισάγει στη συλλογή σχημάτων στον καθορισμένο δείκτη.

```csharp
public IAudioFrame InsertAudioFrameCD(int index, float x, float y, float width, float height)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| index | Int32 | Ο δείκτης που ξεκινά από το μηδέν, στον οποίο θα εισαχθεί το πλαίσιο ήχου. |
| x | Single | Η συντεταγμένη x του νέου πλαισίου ήχου, σε σημεία. |
| y | Single | Η συντεταγμένη y του νέου πλαισίου ήχου, σε σημεία. |
| width | Single | Το πλάτος του νέου πλαισίου ήχου, σε σημεία. |
| height | Single | Το ύψος του νέου πλαισίου ήχου, σε σημεία. |

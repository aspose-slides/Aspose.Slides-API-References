## AddAudioFrameEmbedded(float, float, float, float, Stream) {#addaudioframeembedded_1}

Δημιουργεί ένα νέο πλαίσιο ήχου με ενσωματωμένο αρχείο WAV και το προσθέτει στο τέλος της συλλογής σχήματος. Ο ενσωματωμένος ήχος προστίθεται στη συλλογή Presentation.Audios.

```csharp
public IAudioFrame AddAudioFrameEmbedded(float x, float y, float width, float height, 
    Stream audio_stream)
```

| Parameter | Type | Description |
| --- | --- | --- |
| x | Single | Η συντεταγμένη x του νέου πλαισίου ήχου, σε σημεία. |
| y | Single | Η συντεταγμένη y του νέου πλαισίου ήχου, σε σημεία. |
| width | Single | Το πλάτος του νέου πλαισίου ήχου, σε σημεία. |
| height | Single | Το ύψος του νέου πλαισίου ήχου, σε σημεία. |
| audio_stream | Stream | Μία ροή εισόδου που περιέχει δεδομένα ήχου WAV για ενσωμάτωση. |

### Τιμή Επιστροφής

Το νεοδημιουργημένο [`IAudioFrame`](../../iaudioframe).

### Παραδείγματα

Τα ακόλουθα παραδείγματα δείχνουν πώς να δημιουργήσετε πλαίσιο ήχου.

```csharp
[C#]
// Δημιουργεί μια κλάση παρουσίασης που αντιπροσωπεύει ένα αρχείο παρουσίασης
using (Presentation pres = new Presentation())
{
    // Λαμβάνει την πρώτη διαφάνεια
    ISlide sld = pres.Slides[0];
    // Φορτώνει το αρχείο ήχου wav σε ροή
    FileStream fstr = new FileStream("sampleaudio.wav", FileMode.Open, FileAccess.Read);
    // Προσθέτει το πλαίσιο ήχου
    IAudioFrame audioFrame = sld.Shapes.AddAudioFrameEmbedded(50, 150, 100, 100, fstr);
    // Ορίζει τη λειτουργία αναπαραγώγισης και την ένταση του ήχου
    audioFrame.PlayMode = AudioPlayModePreset.Auto;
    audioFrame.Volume = AudioVolumeMode.Loud;
    // Γράφει το αρχείο PowerPoint στο δίσκο
    pres.Save("AudioFrameEmbed_out.pptx", SaveFormat.Pptx);
}
```

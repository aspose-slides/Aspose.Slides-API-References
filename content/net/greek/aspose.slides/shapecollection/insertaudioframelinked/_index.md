## ShapeCollection.InsertAudioFrameLinked μέθοδος

Δημιουργεί ένα νέο πλαίσιο ήχου που συνδέεται με ένα εξωτερικό αρχείο ήχου και το εισάγει στη συλλογή σχημάτων στην καθορισμένη θέση.

```csharp
public IAudioFrame InsertAudioFrameLinked(int index, float x, float y, float width, float height, 
    string fname)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| index | Int32 | Ο μηδενικός δείκτης στο οποίο θα εισαχθεί το πλαίσιο ήχου. |
| x | Single | Η συντεταγμένη x του νέου πλαισίου ήχου, σε μονάδες σημείων. |
| y | Single | Η συντεταγμένη y του νέου πλαισίου ήχου, σε μονάδες σημείων. |
| width | Single | Το πλάτος του νέου πλαισίου ήχου, σε μονάδες σημείων. |
| height | Single | Το ύψος του νέου πλαισίου ήχου, σε μονάδες σημείων. |
| fname | String | Η διαδρομή ή το όνομα του εξωτερικού αρχείου ήχου για σύνδεση. |

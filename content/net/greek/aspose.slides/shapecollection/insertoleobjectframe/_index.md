## InsertOleObjectFrame(int, float, float, float, float, IOleEmbeddedDataInfo) {#insertoleobjectframe}

Δημιουργεί ένα νέο πλαίσιο αντικειμένου OLE και το εισάγει στη συλλογή σχημάτων στην καθορισμένη θέση.

```csharp
public IOleObjectFrame InsertOleObjectFrame(int index, float x, float y, float width, float height, 
    IOleEmbeddedDataInfo dataInfo)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| index | Int32 | Ο μηδενικής βάσης δείκτης στη θέση που θα εισαχθεί το πλαίσιο αντικειμένου OLE. |
| x | Single | Η συντεταγμένη x του νέου πλαισίου OLE, σε σημεία. |
| y | Single | Η συντεταγμένη y του νέου πλαισίου OLE, σε σημεία. |
| width | Single | Το πλάτος του νέου πλαισίου OLE, σε σημεία. |
| height | Single | Το ύψος του νέου πλαισίου OLE, σε σημεία. |
| dataInfo | IOleEmbeddedDataInfo | Οι ενσωματωμένες πληροφορίες δεδομένων OLE ([`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo)). |

### Τιμή Επιστροφής

Το νεοδημιουργημένο [`IOleObjectFrame`](../../ioleobjectframe).

Αυτό το παράδειγμα δείχνει την εισαγωγή ενός αντικειμένου OLE στη δεύτερη θέση:

```csharp
[C#]
```

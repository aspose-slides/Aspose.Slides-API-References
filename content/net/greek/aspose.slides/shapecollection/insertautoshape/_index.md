## InsertAutoShape(int, ShapeType, float, float, float, float) {#insertautoshape}

Δημιουργεί ένα νέο αυτόματο σχήμα και το εισάγει στη συλλογή σχημάτων στον καθορισμένο δείκτη, εφαρμόζοντας την προεπιλεγμένη μορφοποίηση προτύπου.

```csharp
public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width, 
    float height)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| index | Int32 | Ο μηδενικός δείκτης στον οποίο θα εισαχθεί το νέο αυτόματο σχήμα. |
| shapeType | ShapeType | Το [`ShapeType`](../../shapetype) του αυτόματου σχήματος που θα εισαχθεί. |
| x | Single | Η x-συντεταγμένη του πλαισίου του σχήματος, σε points. |
| y | Single | Η y-συντεταγμένη του πλαισίου του σχήματος, σε points. |
| width | Single | Το πλάτος του πλαισίου του σχήματος, σε points. |
| height | Single | Το ύψος του πλαισίου του σχήματος, σε points. |

## InsertAutoShape(int, ShapeType, float, float, float, float, bool) {#insertautoshape_1}

Δημιουργεί ένα νέο αυτόματο σχήμα και το εισάγει στη συλλογή σχημάτων στον καθορισμένο δείκτη, προαιρετικά αρχικοποιώντας το με προεπιλεγμένη μορφοποίηση προτύπου.

```csharp
public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| index | Int32 | Ο μηδενικός δείκτης στον οποίο θα εισαχθεί το αυτόματο σχήμα. |
| shapeType | ShapeType | Το [`ShapeType`](../../shapetype) του αυτόματου σχήματος που θα εισαχθεί. |
| x | Single | Η x-συντεταγμένη του πλαισίου του σχήματος, σε points. |
| y | Single | Η y-συντεταγμένη του πλαισίου του σχήματος, σε points. |
| width | Single | Το πλάτος του πλαισίου του σχήματος, σε points. |
| height | Single | Το ύψος του πλαισίου του σχήματος, σε points. |
| createFromTemplate | Boolean | True για την εφαρμογή προεπιλεγμένης μορφοποίησης προτύπου (συμπεριλαμβανομένου ενός μη κενό ονόματος, απλού στυλ και κεντραρισμένου κειμένου); false για τη δημιουργία του σχήματος με όλες τις ιδιότητες να έχουν τις προεπιλεγμένες τιμές τους. |

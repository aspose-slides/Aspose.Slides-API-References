## ShapeCollection.InsertSummaryZoomFrame μέθοδος

Δημιουργεί ένα νέο Summary Zoom πλαίσιο και το εισάγει στη συλλογή σχημάτων στο καθορισμένο ευρετήριο.

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| index | Int32 | Ο δείκτης μηδενικής βάσης στο οποίο θα εισαχθεί το Summary Zoom πλαίσιο. |
| x | Single | Η συντεταγμένη x του νέου Summary Zoom πλαίσιο, σε μονάδες points. |
| y | Single | Η συντεταγμένη y του νέου Summary Zoom πλαίσιο, σε μονάδες points. |
| width | Single | Το πλάτος του νέου Summary Zoom πλαίσιο, σε μονάδες points. |
| height | Single | Το ύψος του νέου Summary Zoom πλαίσιο, σε μονάδες points. |

### Τιμή Επιστροφής

Το νέο δημιουργημένο [`ISummaryZoomFrame`](../../isummaryzoomframe).

| Εξαίρεση | Συνθήκη |

### Παρατηρήσεις

Αυτή η μέθοδος δημιουργεί ένα Summary Zoom πλαίσιο που συγκεντρώνει συνδέσμους σύνοψης για όλες τις ενότητες στην παρουσίαση.

### Παραδείγματα

Αυτό το παράδειγμα δείχνει τη δημιουργία και την εισαγωγή ενός αντικειμένου Summary Zoom στο καθορισμένο ευρετήριο μιας συλλογής (υποθέτουμε ότι υπάρχουν τουλάχιστον δύο ενότητες στην παρουσίαση "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSummaryZoomFrame(2, 150, 20, 50, 50);
}
```

## InsertChart(ChartType, float, float, float, float, int) {#insertchart}

Δημιουργεί ένα νέο διάγραμμα, το αρχικοποιεί με δείγμα δεδομένων σειράς και ρυθμίσεων, και το εισάγει στη συλλογή σχημάτων στον καθορισμένο δείκτη.

```csharp
public IChart InsertChart(ChartType type, float x, float y, float width, float height, int index)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| type | ChartType | Ο τύπος του διαγράμματος που θα δημιουργηθεί. |
| x | Single | Η συντεταγμένη x του νέου διαγράμματος, σε μονάδες σημείου. |
| y | Single | Η συντεταγμένη y του νέου διαγράμματος, σε μονάδες σημείου. |
| width | Single | Το πλάτος του νέου διαγράμματος, σε μονάδες σημείου. |
| height | Single | Το ύψος του νέου διαγράμματος, σε μονάδες σημείου. |
| index | Int32 | Ο μηδενικός δείκτης στο οποίο θα εισαχθεί το νέο διάγραμμα στη συλλογή σχημάτων. |

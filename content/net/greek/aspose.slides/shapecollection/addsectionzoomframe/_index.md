## AddSectionZoomFrame(float, float, float, float, ISection) {#addsectionzoomframe}

Δημιουργεί ένα νέο πλαίσιο Section Zoom και το προσθέτει στο τέλος της συλλογής σχήματος.

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| x | Single | Η συντεταγμένη x του νέου πλαισίου Section Zoom, σε μονάδες point. |
| y | Single | Η συντεταγμένη y του νέου πλαισίου Section Zoom, σε μονάδες point. |
| width | Single | Το πλάτος του νέου πλαισίου Section Zoom, σε μονάδες point. |
| height | Single | Το ύψος του νέου πλαισίου Section Zoom, σε μονάδες point. |
| section | ISection | Η [`ISection`](../../isection) που αναφέρεται από το πλαίσιο Section Zoom· πρέπει να ανήκει σε αυτήν την παρουσίαση και να περιέχει τουλάχιστον μία διαφάνεια. |

### Τιμή Επιστροφής

Το νεοδημιουργημένο [`ISectionZoomFrame`](../../isectionzoomframe).

### Εξαιρέσεις

| εξαίρεση | συνθήκη |
| --- | --- |
| ArgumentException | Επαναβλήθηκε αν η αναφερόμενη ενότητα δεν ανήκει στην τρέχουσα παρουσίαση ή δεν περιέχει διαφάνειες. |

### Παραδείγματα

Αυτό το παράδειγμα δείχνει την προσθήκη ενός αντικειμένου Section Zoom στο τέλος μιας συλλογής (υποθέστε ότι υπάρχουν τουλάχιστον δύο ενότητες στην παρουσίαση "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1]);
}
```

public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height,

Αυτό το παράδειγμα δείχνει την προσθήκη ενός αντικειμένου Section Zoom στο τέλος μιας συλλογής (υποθέστε ότι υπάρχουν τουλάχιστον δύο ενότητες στην παρουσίαση "Presentation.pptx"):

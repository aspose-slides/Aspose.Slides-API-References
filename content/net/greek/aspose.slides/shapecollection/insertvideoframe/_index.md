## ShapeCollection.InsertVideoFrame μέθοδος

Δημιουργεί ένα νέο πλαίσιο βίντεο και το εισάγει στη συλλογή σχημάτων στο καθορισμένο δείκτη.

```csharp
public IVideoFrame InsertVideoFrame(int index, float x, float y, float width, float height, 
    string fname)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| index | Int32 | Ο δείκτης μηδενικής βάσης στον οποίο θα εισαχθεί το πλαίσιο βίντεο. |
| x | Single | Η συντεταγμένη x του νέου πλαισίου βίντεο, σε μονάδες σημείου. |
| y | Single | Η συντεταγμένη y του νέου πλαισίου βίντεο, σε μονάδες σημείου. |
| width | Single | Το πλάτος του νέου πλαισίου βίντεο, σε μονάδες σημείου. |
| height | Single | Το ύψος του νέου πλαισίου βίντεο, σε μονάδες σημείου. |
| fname | String | Η διαδρομή ή το όνομα του αρχείου βίντεο προς ενσωμάτωση. |

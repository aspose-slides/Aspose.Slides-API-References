## ShapeCollection.AddSmartArt μέθοδος

Δημιουργεί ένα SmartArt διάγραμμα και το προσθέτει στο τέλος της συλλογής σχημάτων.

```csharp
public ISmartArt AddSmartArt(float x, float y, float width, float height, 
    SmartArtLayoutType layoutType)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| x | Single | Η συντεταγμένη x του πλαισίου του διαγράμματος, σε μονάδες σημείων. |
| y | Single | Η συντεταγμένη y του πλαισίου του διαγράμματος, σε μονάδες σημείων. |
| width | Single | Το πλάτος του πλαισίου του διαγράμματος, σε μονάδες σημείων. |
| height | Single | Το ύψος του πλαισίου του διαγράμματος, σε μονάδες σημείων. |
| layoutType | SmartArtLayoutType | Ο τύπος διάταξης SmartArt. |

### Τιμή Επιστροφής

Το νέο δημιουργημένο [`ISmartArt`](../../../aspose.slides.smartart/ismartart).

### Παραδείγματα

Το παρακάτω παράδειγμα δείχνει πώς να προσθέσετε έξυπνο σχήμα σε παρουσίαση PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	Slide slide = pres.Slides[0];
	SmartArt smart = slide.Shapes.AddSmartArt(0, 0, 400, 400, SmartArtLayoutType.BasicBlockList);
}
```

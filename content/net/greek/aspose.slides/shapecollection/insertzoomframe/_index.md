## InsertZoomFrame(int, float, float, float, float, ISlide) {#insertzoomframe}

Δημιουργεί ένα νέο πλαίσιο Zoom και το εισάγει στη συλλογή σχημάτων στον καθορισμένο δείκτη.

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| index | Int32 | Ο δείκτης μηδενικής βάσης στον οποίο θα εισαχθεί το πλαίσιο Zoom. |
| x | Single | Η x-συντεταγμένη του νέου πλαισίου Zoom, σε σημεία. |
| y | Single | Η y-συντεταγμένη του νέου πλαισίου Zoom, σε σημεία. |
| width | Single | Το πλάτος του νέου πλαισίου Zoom, σε σημεία. |
| height | Single | Το ύψος του νέου πλαισίου Zoom, σε σημεία. |
| slide | ISlide | Το [`ISlide`](../../islide) που αναφέρεται από το πλαίσιο Zoom. |

Το νεοδημιουργημένο [`IZoomFrame`](../../izoomframe).

### Παραδείγματα

Αυτό το παράδειγμα δείχνει τη δημιουργία και την εισαγωγή ενός αντικειμένου Zoom στον καθορισμένο δείκτη μιας συλλογής (υποθέστε ότι υπάρχουν τουλάχιστον δύο διαφάνειες στην παρουσίαση "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))

}
```

## ShapeCollection.AddSummaryZoomFrame μέθοδος

Δημιουργεί ένα νέο Summary Zoom πλαίσιο και το προσθέτει στο τέλος της συλλογής σχήματος.

```csharp
public ISummaryZoomFrame AddSummaryZoomFrame(float x, float y, float width, float height)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| x | Single | Η συντεταγμένη x του νέου Summary Zoom πλαισίου, σε σημεία. |
| y | Single | Η συντεταγμένη y του νέου Summary Zoom πλαισίου, σε σημεία. |
| width | Single | Το πλάτος του νέου Summary Zoom πλαισίου, σε σημεία. |
| height | Single | Το ύψος του νέου Summary Zoom πλαισίου, σε σημεία. |

### Τιμή Επιστροφής

Το νέο δημιουργημένο [`ISummaryZoomFrame`](../../isummaryzoomframe).

### Παρατηρήσεις

Αυτή η μέθοδος δημιουργεί ένα νέο Summary Zoom και τοποθετεί μια συλλογή αντικειμένων σε αυτό για όλες τις ενότητες σε αυτή την παρουσίαση.

### Παραδείγματα

Αυτό το παράδειγμα δείχνει την προσθήκη ενός αντικειμένου Summary Zoom στο τέλος μιας συλλογής (υποθέτουμε ότι υπάρχουν τουλάχιστον δύο ενότητες στην παρουσίαση "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSummaryZoomFrame(150, 20, 500, 250);
}
```

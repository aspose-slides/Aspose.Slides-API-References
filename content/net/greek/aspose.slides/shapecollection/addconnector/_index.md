## AddConnector(ShapeType, float, float, float, float) {#addconnector}

Δημιουργεί ένα νέο σχήμα συνδέσμου με προεπιλεγμένη μορφοποίηση προτύπου και το προσθέτει στο τέλος της συλλογής σχημάτων.

```csharp
public IConnector AddConnector(ShapeType shapeType, float x, float y, float width, float height)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| shapeType | ShapeType | Το [`ShapeType`](../../shapetype) του σχήματος συνδέσμου που θα προστεθεί. |
| x | Single | Η συντεταγμένη x του πλαισίου του συνδέσμου, σε μονάδες σημείου. |
| y | Single | Η συντεταγμένη y του πλαισίου του συνδέσμου, σε μονάδες σημείου. |
| width | Single | Το πλάτος του πλαισίου του συνδέσμου, σε μονάδες σημείου. |
| height | Single | Το ύψος του πλαισίου του συνδέσμου, σε μονάδες σημείου. |

### Τιμή Επιστροφής

Το νεοδημιουργημένο [`IConnector`](../../iconnector).

### Παραδείγματα

Το παρακάτω παράδειγμα δείχνει πώς να προσθέσετε ένα σύνδεσμο (ένα λυγό σύνδεσμο) μεταξύ δύο σχημάτων (έλλειψη και ορθογώνιο) σε παρουσίαση PowerPoint.

```csharp
[C#]
// Δημιουργεί μια κλάση παρουσίασης που αντιπροσωπεύει ένα αρχείο PPTX
using (Presentation input = new Presentation())
{
    // Προσπελαύνει τη συλλογή σχημάτων για μια συγκεκριμένη διαφάνεια
    IShapeCollection shapes = input.Slides[0].Shapes;
    // Προσθέτει ένα αυτοσχήμα Ellipse
    IAutoShape ellipse = shapes.AddAutoShape(ShapeType.Ellipse, 0, 100, 100, 100);
    // Προσθέτει ένα αυτοσχήμα Rectangle
    IAutoShape rectangle = shapes.AddAutoShape(ShapeType.Rectangle, 100, 300, 100, 100);
    // Προσθέτει ένα σχήμα συνδέσμου στη συλλογή σχημάτων της διαφάνειας
    IConnector connector = shapes.AddConnector(ShapeType.BentConnector2, 0, 0, 10, 10);
    // Συνδέει τα σχήματα χρησιμοποιώντας το σύνδεσμο
    connector.StartShapeConnectedTo = ellipse;
    connector.EndShapeConnectedTo = rectangle;
    // Καλεί τη μέθοδο reroute που ορίζει την αυτόματη μικρότερη διαδρομή μεταξύ των σχημάτων
    connector.Reroute();
    // Αποθηκεύει την παρουσίαση
    input.Save("Shapes-connector.pptx", SaveFormat.Pptx);
}
```

## AddConnector(ShapeType, float, float, float, float, bool) {#addconnector_1}

Δημιουργεί ένα νέο σχήμα συνδέσμου και το προσθέτει στο τέλος της συλλογής σχημάτων, προαιρετικά εφαρμόζοντας προεπιλεγμένη μορφοποίηση προτύπου.

```csharp
public IConnector AddConnector(ShapeType shapeType, float x, float y, float width, float height, 
    bool createFromTemplate)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| shapeType | ShapeType | Το [`ShapeType`](../../shapetype) του σχήματος συνδέσμου που θα δημιουργηθεί. |
| x | Single | Η συντεταγμένη x του πλαισίου του συνδέσμου, σε μονάδες σημείου. |
| y | Single | Η συντεταγμένη y του πλαισίου του συνδέσμου, σε μονάδες σημείου. |
| width | Single | Το πλάτος του πλαισίου του συνδέσμου, σε μονάδες σημείου. |
| height | Single | Το ύψος του πλαισίου του συνδέσμου, σε μονάδες σημείου. |
| createFromTemplate | Boolean | True για να εφαρμοστεί η προεπιλεγμένη μορφοποίηση προτύπου (μη κενό όνομα, απλό στυλ); false για να δημιουργηθεί ο σύνδεσμος με προεπιλεγμένες τιμές ιδιοτήτων. |

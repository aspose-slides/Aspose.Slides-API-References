## InsertAudioFrameEmbedded(int, float, float, float, float, Stream) {#insertaudioframeembedded_1}

Δημιουργεί ένα νέο πλαίσιο ήχου με ενσωματωμένο αρχείο WAV και το εισάγει στη συλλογή σχήματος στον καθορισμένο δείκτη. Ο ενσωματωμένος ήχος προστίθεται στη συλλογή Presentation.Audios.

```csharp
public IAudioFrame InsertAudioFrameEmbedded(int index, float x, float y, float width, float height, 
    Stream audio_stream)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| index | Int32 | Ο δείκτης μηδενικής βάσης στον οποίο θα εισαχθεί το πλαίσιο ήχου. |
| x | Single | Η συντεταγμένη x του νέου πλαισίου ήχου, σε μονάδες σημείου. |
| y | Single | Η συντεταγμένη y του νέου πλαισίου ήχου, σε μονάδες σημείου. |
| width | Single | Το πλάτος του νέου πλαισίου ήχου, σε μονάδες σημείου. |
| height | Single | Το ύψος του νέου πλαισίου ήχου, σε μονάδες σημείου. |
| audio_stream | Stream | Μια ροή εισόδου που περιέχει δεδομένα ήχου WAV για ενσωμάτωση. |

### Τιμή Επιστροφής

Το νέο δημιουργημένο [`IAudioFrame`](../../iaudioframe).

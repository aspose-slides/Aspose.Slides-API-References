## EmbeddingLevel απαρίθμηση

Αναπαριστά τα δικαιώματα αδειοδότησης για την ενσωμάτωση της γραμματοσειράς.

```csharp
[Flags]
public enum EmbeddingLevel : ushort
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Installable | `0` | Οι γραμματοσειρές με αυτή τη ρύθμιση υποδεικνύουν ότι μπορούν να ενσωματωθούν και να εγκατασταθούν μόνιμα στο απομακρυσμένο σύστημα από μια εφαρμογή. Ο χρήστης του απομακρυσμένου συστήματος αποκτά τα ίδια δικαιώματα, υποχρεώσεις και άδειες για αυτή τη γραμματοσειρά όπως ο αρχικός αγοραστής της γραμματοσειράς και υπόκειται στην ίδια συμφωνία τελικού χρήστη, πνευματικά δικαιώματα, ευριστική πατέντα και/ή εμπορικό σήμα όπως ο αρχικός αγοραστής. |
| Restricted | `2` | Οι γραμματοσειρές που έχουν μόνο αυτό το bit ενεργοποιημένο δεν πρέπει να τροποποιηθούν, ενσωματωθούν ή ανταλλαγούν με οποιονδήποτε τρόπο χωρίς πρώτα την άδεια του νομικού κατόχου. |
| PreviewPrint | `4` | Όταν αυτό το bit είναι ενεργοποιημένο, η γραμματοσειρά μπορεί να ενσωματωθεί και να φορτωθεί προσωρινά στο απομακρυσμένο σύστημα. Τα έγγραφα που περιέχουν γραμματοσειρές Preview &amp; Print πρέπει να ανοίγονται "μόνο για ανάγνωση"· δεν μπορούν να εφαρμοστούν επεξεργασίες στο έγγραφο. |
| Editable | `8` | Όταν αυτό το bit είναι ενεργοποιημένο, η γραμματοσειρά μπορεί να ενσωματωθεί αλλά πρέπει να εγκατασταθεί προσωρινά μόνο σε άλλα συστήματα. Σε αντίθεση με τις γραμματοσειρές Preview &amp; Print, τα έγγραφα που περιέχουν Editable γραμματοσειρές μπορούν να ανοίγονται για ανάγνωση, επιτρέπεται η επεξεργασία και οι αλλαγές μπορούν να αποθηκευτούν. |
| NoSubsetting | `100` | Όταν αυτό το bit είναι ενεργοποιημένο, η γραμματοσειρά δεν μπορεί να υποσύνολο (subsetting) πριν την ενσωμάτωση. Άλλοι περιορισμοί ενσωμάτωσης που καθορίζονται στα bits 0-3 και 9 ισχύουν επίσης. |
| BitmapOnly | `200` | Όταν αυτό το bit είναι ενεργοποιημένο, μόνο τα bitmap που περιέχονται στη γραμματοσειρά μπορούν να ενσωματωθούν. Δεν μπορούν να ενσωματωθούν δεδομένα περιγράμματος. Εάν δεν υπάρχουν διαθέσιμα bitmap στη γραμματοσειρά, τότε η γραμματοσειρά θεωρείται μη ενσωματώνιμη και οι υπηρεσίες ενσωμάτωσης θα αποτύχουν. |

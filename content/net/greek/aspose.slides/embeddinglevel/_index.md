## EmbeddingLevel απαρίθμηση

Αντιπροσωπεύει τα δικαιώματα αδειοδότησης για την ενσωμάτωση της γραμματοσειράς.

```csharp
[Flags]
public enum EmbeddingLevel : ushort
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Installable | `0` | Οι γραμματοσειρές με αυτήν τη ρύθμιση υποδεικνύουν ότι μπορούν να ενσωματωθούν και να εγκατασταθούν μόνιμα στο απομακρυσμένο σύστημα από μια εφαρμογή. Ο χρήστης του απομακρυσμένου συστήματος αποκτά τα ίδια δικαιώματα, υποχρεώσεις και άδειες για αυτήν τη γραμματοσειρά όπως ο αρχικός αγοραστής της, και υπόκειται στην ίδια συμφωνία τελικού χρήστη, τα πνευματικά δικαιώματα, την πατέντα σχεδίασης και/ή το εμπορικό σήμα όπως ο αρχικός αγοραστής. |
| Restricted | `2` | Οι γραμματοσειρές που έχουν οριστεί μόνο αυτό το δυαδικό ψηφίο δεν πρέπει να τροποποιηθούν, ενσωματωθούν ή ανταλλαχθούν με οποιονδήποτε τρόπο χωρίς πρώτα την άδεια του νόμιμου κατόχου. |
| PreviewPrint | `4` | Όταν αυτό το δυαδικό ψηφίο είναι ενεργό, η γραμματοσειρά μπορεί να ενσωματωθεί και να φορτωθεί προσωρινά στο απομακρυσμένο σύστημα. Τα έγγραφα που περιέχουν γραμματοσειρές Προεπισκόπηση &amp; Εκτύπωση πρέπει να ανοίγονται "μόνο για ανάγνωση;" χωρίς δυνατότητα επεξεργασίας. |
| Editable | `8` | Όταν αυτό το δυαδικό ψηφίο είναι ενεργό, η γραμματοσειρά μπορεί να ενσωματωθεί αλλά πρέπει να εγκατασταθεί προσωρινά σε άλλα συστήματα. Σε αντίθεση με τις γραμματοσειρές Προεπισκόπηση &amp; Εκτύπωση, τα έγγραφα που περιέχουν γραμματοσειρές Editable μπορούν να ανοίγουν για ανάγνωση, η επεξεργασία επιτρέπεται και οι αλλαγές μπορούν να αποθηκευτούν. |
| NoSubsetting | `100` | Όταν αυτό το δυαδικό ψηφίο είναι ενεργό, η γραμματοσειρά δεν μπορεί να υποσυντομευθεί πριν την ενσωμάτωση. Εφαρμόζονται επίσης άλλοι περιορισμοί ενσωμάτωσης που καθορίζονται στα δυαδικά ψηφία 0-3 και 9. |
| BitmapOnly | `200` | Όταν αυτό το δυαδικό ψηφίο είναι ενεργό, μόνο τα bitmap που περιλαμβάνονται στη γραμματοσειρά μπορούν να ενσωματωθούν. Δεν μπορούν να ενσωματωθούν δεδομένα περιγράμματος. Εάν δεν υπάρχουν διαθέσιμα bitmap στη γραμματοσειρά, τότε η γραμματοσειρά θεωρείται μη ενσωματώσιμη και οι υπηρεσίες ενσωμάτωσης θα αποτύχουν. |

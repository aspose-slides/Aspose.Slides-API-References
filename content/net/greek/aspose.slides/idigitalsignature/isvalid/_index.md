## IDigitalSignature.IsValid ιδιότητα

Εάν αυτή η ψηφιακή υπογραφή είναι έγκυρη και το έγγραφο δεν έχει υποστεί αλλοίωση, αυτή η τιμή θα είναι true. Μόνο για ανάγνωση Boolean.

```csharp
public bool IsValid { get; }
```

}

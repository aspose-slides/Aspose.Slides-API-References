## IGeometryShape.GetGeometryPaths μέθοδος

Επιστρέφει το αντίγραφο της διαδρομής του γεωμετρικού σχήματος. Οι συντεταγμένες είναι σχετικές με την πάνω αριστερή γωνία του σχήματος.

```csharp
public IGeometryPath[] GetGeometryPaths()
```

### Τιμή Επιστροφής

Πίνακας του [`IGeometryPath`](../../igeometrypath)

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    GeometryShape shape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 200, 100) as GeometryShape;

    IGeometryPath geometryPath = shape.GetGeometryPaths()[0];

    geometryPath.LineTo(100, 50, 1);
    geometryPath.LineTo(100, 50, 4);

    shape.SetGeometryPath(geometryPath);

    pres.Save("output.pptx", SaveFormat.Pptx);
}
```

## Fonts.SetScriptFont μέθοδος

Αναθέτει ένα όνομα γραμματοσειράς σε μια συγκεκριμένη ετικέτα script, η οποία καθορίζει πώς θα αποτυπωθεί το κείμενο αυτής της γραφής στην παρουσίαση.

```csharp
public void SetScriptFont(string script, string fontName)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| script | String | Ο κωδικός script BCP-47 (π.χ., "Arab", "Hebr", "Hans") που ταυτοποιεί το σύστημα γραφής. |
| fontName | String | Το όνομα της γραμματοσειράς που θα ανατεθεί στο συγκεκριμένο script. |

### Παραδείγματα

Αυτό το παράδειγμα δείχνει πώς να ορίσετε τη γραμματοσειρά για το αραβικό script σε "Segoe UI":

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segoe UI");
```

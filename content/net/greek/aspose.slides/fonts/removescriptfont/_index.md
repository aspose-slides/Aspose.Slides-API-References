## Μέθοδος Fonts.RemoveScriptFont

Αφαιρεί τη ρύθμιση γραμματοσειράς που σχετίζεται με μια συγκεκριμένη ετικέτα script από τη συλλογή γραμματοσειρών του θέματος.

```csharp
public void RemoveScriptFont(string script)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| script | String | Ο κώδικας script BCP-47 της γραμματοσειράς που πρέπει να αφαιρεθεί. |

### Παραδείγματα

Αυτό το παράδειγμα δείχνει πώς να αφαιρέσετε τη αντιστοίχηση γραμματοσειράς για το εβραϊκό script:

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.RemoveScriptFont("Hebr");
```

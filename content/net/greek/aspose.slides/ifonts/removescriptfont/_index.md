## IFonts.RemoveScriptFont μέθοδος

Αφαιρεί τη ρύθμιση γραμματοσειράς που σχετίζεται με μια συγκεκριμένη ετικέτα script από τη συλλογή γραμματοσειρών του θέματος.

```csharp
public void RemoveScriptFont(string script)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| script | String | Ο κωδικός script BCP-47 της γραμματοσειράς που πρέπει να αφαιρεθεί. |

### Παραδείγματα

Αυτό το παράδειγμα δείχνει πώς να αφαιρέσετε τη χαρτογράφηση γραμματοσειράς για τη εβραϊκή γραφή:

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.RemoveScriptFont("Hebr");
```

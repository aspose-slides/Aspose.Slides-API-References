## IFonts.SetScriptFont μέθοδος

Αναθέτει ένα όνομα γραμματοσειράς σε μια συγκεκριμένη ετικέτα script, η οποία ορίζει πώς το κείμενο αυτής της γραφής θα αποτυπώνεται στην παρουσίαση.

```csharp
public void SetScriptFont(string script, string fontName)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| script | String | Ο κωδικός script BCP-47 (π.χ., "Arab", "Hebr", "Hans") που αναγνωρίζει το σύστημα γραφής. |
| fontName | String | Το όνομα της γραμματοσειράς που θα ανατεθεί στο καθορισμένο script. |

### Παραδείγματα

Αυτό το παράδειγμα δείχνει πώς να ορίσετε τη γραμματοσειρά για το Αραβικό script σε "Segoe UI":

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segue UI");
```

## BlobManagementOptions.MaxBlobsBytesInMemory ιδιότητα

Ορίζει το μέγιστο συνολικό μέγεθος (σε byte) που μπορούν να καταλαμβάνουν όλα τα BLOBs στη μνήμη. Από προεπιλογή, όλα τα BLOB φορτώνονται στη μνήμη· μόνο όταν αυτή η όριο επιτευχθεί χρησιμοποιούνται εναλλακτικοί μηχανισμοί (όπως προσωρινά αρχεία). Η διατήρηση των BLOB στη μνήμη μεγιστοποιεί την απόδοση αλλά μπορεί να οδηγήσει σε υψηλή χρήση μνήμης. Χρησιμοποιήστε αυτή την ιδιότητα για να προσαρμόσετε τη συμπεριφορά στο περιβάλλον ή τις απαιτήσεις σας.

```csharp
public ulong MaxBlobsBytesInMemory { get; set; }
```

### Παρατηρήσεις

Αυτή η ιδιότητα αγνοείται εάν [`IsTemporaryFilesAllowed`](../istemporaryfilesallowed) οριστεί σε false, επειδή η μνήμη είναι τότε η μοναδική διαθέσιμη θέση αποθήκευσης και ο περιορισμός της χρήσης BLOB στη μνήμη δεν έχει καμία επίδραση.

Η προεπιλεγμένη τιμή είναι 629 145 600 byte (600 MB).

Μπορείτε να ορίσετε αυτή την ιδιότητα σε μηδέν, αλλά θα διατηρηθεί ένα μικρό ελάχιστο ποσό μνήμης.

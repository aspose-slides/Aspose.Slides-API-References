## ZoomObject.ShowBackground ιδιότητα

Λαμβάνει ή ορίζει την τιμή που καθορίζει αν το Zoom θα χρησιμοποιήσει το φόντο της διαφάνειας προορισμού. Ανάγνωση/εγγραφή Boolean. Προεπιλεγμένη τιμή: true

```csharp
public bool ShowBackground { get; set; }
```

### Παραδείγματα

Το παράδειγμα δείχνει την αφαίρεση του φόντου μιας εικόνας ενός αντικειμένου Zoom:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
    zoomFrame.ShowBackground = false;
}
```

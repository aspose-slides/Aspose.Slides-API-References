## SlideShowSettings.SlideShowType ιδιότητα

Λαμβάνει ή ορίζει τον τύπο παρουσίασης διαφανειών. Αναπαρίσταται από τις ακόλουθες `SlideShowType` προγόνους: [`BrowsedAtKiosk`](../../browsedatkiosk), [`PresentedBySpeaker`](../../presentedbyspeaker) και [`BrowsedByIndividual`](../../browsedbyindividual)

```csharp
public SlideShowType SlideShowType { get; set; }
```

### Παραδείγματα

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // για να ορίσετε τύπο "Browsed at a kiosk (full screen)" 
    pres.SlideShowSettings.SlideShowType = new BrowsedAtKiosk();
    
    // για να ορίσετε τύπο "Browsed by individual (window)" 
    pres.SlideShowSettings.SlideShowType = new BrowsedByIndividual();
    
    // για να ορίσετε τύπο "Presented by a speaker (full screen)" 
    pres.SlideShowSettings.SlideShowType = new PresentedBySpeaker();
}
```

## IProtectionManager.CheckWriteProtection μέθοδος

Καθορίζει εάν μια παρουσίαση είναι προστατευμένη με κωδικό πρόσβασης για τροποποίηση.

```csharp
public bool CheckWriteProtection(string password)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| password | String | Ο κωδικός πρόσβασης για έλεγχο. |

### Τιμή Επιστροφής

True εάν ο κωδικός πρόσβασης είναι έγκυρος· διαφορετικά, false.

### Παρατηρήσεις

1. Θα πρέπει να ελέγξετε την ιδιότητα [`IsWriteProtected`](../iswriteprotected) πριν καλέσετε αυτή τη μέθοδο. 2. Όταν ο κωδικός πρόσβασης είναι null ή κενός, αυτή η μέθοδος επιστρέφει false.

### Παραδείγματα

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

## NormalViewProperties.PreferSingleView ιδιότητα

Καθορίζει αν ο χρήστης προτιμά να βλέπει μια περιοχή περιεχομένου πλήρους παραθύρου αντί για την τυπική κανονική προβολή με τρεις περιοχές περιεχομένου. Εάν είναι ενεργοποιημένη, η εφαρμογή μπορεί να επιλέξει να εμφανίσει μία από τις περιοχές περιεχομένου σε ολόκληρο το παράθυρο. Ανάγνωση/εγγραφή Boolean.

```csharp
public bool PreferSingleView { get; set; }
```

## LoadOptions.DefaultRegularFont ιδιότητα

Επιστρέφει ή ορίζει τη γραμματοσειρά Regular που χρησιμοποιείται σε περίπτωση που δεν βρεθεί η πηγή γραμματοσειράς. Ανάγνωση/εγγραφή String.

```csharp
public string DefaultRegularFont { get; set; }
```

### Παραδείγματα

Το παρακάτω παράδειγμα δείχνει πώς να ορίσετε τις προεπιλεγμένες γραμματοσειρές για την απόδοση μιας παρουσίασης PowerPoint Presentation.

```csharp
[C#]
// Χρησιμοποιήστε τις επιλογές φόρτωσης για να ορίσετε τις προεπιλεγμένες κανονικές και ασιατικές γραμματοσειρές
LoadOptions loadOptions = new LoadOptions(LoadFormat.Auto);
loadOptions.DefaultRegularFont = "Wingdings";
loadOptions.DefaultAsianFont = "Wingdings";
// Φορτώστε την παρουσίαση
using (Presentation pptx = new Presentation("DefaultFonts.pptx", loadOptions))
{
    // Δημιουργήστε μικρογραφία διαφάνειας
    pptx.Slides[0].GetThumbnail(1, 1).Save("output_out.png", ImageFormat.Png);
    // Δημιουργήστε PDF
    pptx.Save("output_out.pdf", SaveFormat.Pdf);
    // Δημιουργήστε XPS
    pptx.Save("output_out.xps", SaveFormat.Xps);
}
```

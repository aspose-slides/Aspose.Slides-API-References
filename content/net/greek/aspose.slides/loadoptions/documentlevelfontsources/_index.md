## LoadOptions.DocumentLevelFontSources ιδιότητα

Καθορίζει τις πηγές για εξωτερικές γραμματοσειρές που θα χρησιμοποιηθούν από την παρουσίαση. Αυτές οι γραμματοσειρές είναι διαθέσιμες στην παρουσίαση καθ' όλη τη διάρκεια της ζωής της και δεν μοιράζονται με άλλες παρουσιάσεις

```csharp
public IFontSources DocumentLevelFontSources { get; set; }
```

### Παραδείγματα

Το παρακάτω παράδειγμα δείχνει πώς να καθορίσετε προσαρμοσμένες γραμματοσειρές που χρησιμοποιούνται με την παρουσίαση PowerPoint.

```csharp
[C#]
byte[] memoryFont1 = File.ReadAllBytes("customfonts\\CustomFont1.ttf");
byte[] memoryFont2 = File.ReadAllBytes("customfonts\\CustomFont2.ttf");
LoadOptions loadOptions = new LoadOptions();
loadOptions.DocumentLevelFontSources.FontFolders = new string[] { "assets\\fonts", "global\\fonts" };
loadOptions.DocumentLevelFontSources.MemoryFonts = new byte[][] { memoryFont1, memoryFont2 };
using (IPresentation presentation = new Presentation("MyPresentation.pptx", loadOptions))
{
    //εργαστείτε με την παρουσίαση
    //Τα CustomFont1, CustomFont2 καθώς και οι γραμματοσειρές από τους φακέλους assets\fonts & global\fonts και τους υποφακέλους τους είναι διαθέσιμες στην παρουσίαση
}
```

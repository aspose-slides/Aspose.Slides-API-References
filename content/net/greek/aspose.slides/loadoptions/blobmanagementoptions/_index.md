## LoadOptions.BlobManagementOptions ιδιότητα

Αντιπροσωπεύει τις επιλογές που μπορούν να χρησιμοποιηθούν για τη διαχείριση της συμπεριφοράς Binary Large Objects (BLOBs), όπως η χρήση προσωρινών αρχείων ή το μέγιστο αριθμό bytes BLOBs στη μνήμη. Αυτές οι επιλογές προορίζονται να θέσουν το καλύτερο λόγο απόδοσης/κατανάλωσης μνήμης για ένα συγκεκριμένο περιβάλλον ή απαιτήσεις. Ένα Binary Large Object (BLOB) είναι δυαδικά δεδομένα αποθηκευμένα ως μία ενιαία οντότητα – π.χ. το BLOB μπορεί να είναι ήχος, βίντεο ή η παρουσίαση αυτή καθαυτή.

public IBlobManagementOptions BlobManagementOptions { get; set; }

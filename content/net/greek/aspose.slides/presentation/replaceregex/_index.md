## Presentation.ReplaceRegex μέθοδος

Αντικαθιστά όλες τις αντιστοιχίες της κανονικής έκφρασης με το καθορισμένο κείμενο.

```csharp
public void ReplaceRegex(Regex regex, string newText, IFindResultCallback callback)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| regex | Regex | Η κανονική έκφραση Regex για την απόκτηση των συμβολοσειρών προς αντικατάσταση. |
| newText | String | Η συμβολοσειρά για την αντικατάσταση όλων των εμφανίσεων των συμβολοσειρών που θα αντικατασταθούν. |
| callback | IFindResultCallback | Το αντικείμενο callback για τη λήψη αποτελεσμάτων αναζήτησης [`IFindResultCallback`](../../ifindresultcallback). |

### Παραδείγματα

Το παρακάτω δείγμα κώδικα δείχνει πώς να αντικαταστήσετε κείμενο χρησιμοποιώντας κανονική έκφραση με το καθορισμένο κείμενο.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// Αντικαθιστά όλες τις λέξεις με 10 ή περισσότερους χαρακτήρες με '***'
	presentation.ReplaceRegex(regex, "***", null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

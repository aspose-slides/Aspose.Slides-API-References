## Presentation.SlideSize ιδιότητα

Επιστρέφει αντικείμενο slide size. Μόνο για ανάγνωση [`ISlideSize`](../../islidesize).

```csharp
public ISlideSize SlideSize { get; }
```

### Παραδείγματα

Το παρακάτω παράδειγμα δείχνει πώς να αλλάξετε το slide size σε μια PowerPoint Presentation.

```csharp
[C#]
using (Presentation pres = new Presentation("pres-4x3-aspect-ratio.pptx"))
{
    pres.SlideSize.SetSize(SlideSizeType.OnScreen16x9, SlideSizeScaleType.DoNotScale);
    pres.Save("pres-4x3-aspect-ratio.pptx", SaveFormat.Pptx);
}
```

Το παρακάτω παράδειγμα δείχνει πώς να ορίσετε το slide size ως προς κλιμάκωση περιεχομένου για μια PowerPoint Presentation.

```csharp
[C#]
// Δημιουργεί ένα αντικείμενο Presentation που αντιπροσωπεύει ένα αρχείο παρουσίασης
using(Presentation presentation = new Presentation("AccessSlides.pptx")) {
  using(Presentation auxPresentation = new Presentation()) {
    ISlide slide = presentation.Slides[0];
    // Ορίζει το μέγεθος διαφάνειας των δημιουργημένων παρουσιάσεων στο μέγεθος της πηγής
    presentation.SlideSize.SetSize(540, 720, SlideSizeScaleType.EnsureFit); // Η μέθοδος SetSize χρησιμοποιείται για να ορίσει το μέγεθος διαφάνειας με κλιμάκωση του περιεχομένου ώστε να διασφαλιστεί η προσαρμογή
    presentation.SlideSize.SetSize(SlideSizeType.A4Paper, SlideSizeScaleType.Maximize); // Η μέθοδος SetSize χρησιμοποιείται για να ορίσει το μέγεθος διαφάνειας με μεγιστοποίηση του περιεχομένου
    // Αποθηκεύει την Παρουσίαση στο δίσκο
    auxPresentation.Save("Set_Size&Type_out.pptx", SaveFormat.Pptx);
  }
}
```

Το παρακάτω παράδειγμα δείχνει πώς να προσδιορίσετε προσαρμοσμένα slide sizes σε μια PowerPoint Presentation.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    pres.SlideSize.SetSize(780, 540, SlideSizeScaleType.DoNotScale); // μέγεθος χαρτιού A4
    pres.Save("pres-a4-slide-size.pptx", SaveFormat.Pptx);
}
```

## Control.Persistence ιδιότητα

Λαμβάνει τη μέθοδο που χρησιμοποιείται για την αποθήκευση των ιδιοτήτων του ελέγχου ActiveX. Μόνο για ανάγνωση [`PersistenceType`](../../persistencetype).

```csharp
public PersistenceType Persistence { get; }
```

### Παραδείγματα

Το επόμενο παράδειγμα δείχνει τη χρήση της ιδιότητας Persistence για τον έλεγχο εάν οι ιδιότητες του αντικειμένου ActiveX μπορούν να αλλάξουν ως ιδιότητες ActiveX βασισμένες σε XML:

```csharp
[C#]
if (control.Persistence == PersistenceType.PersistPropertyBag)
{
    control.Properties["Value"] = value;
}
else
{
    YourMethodHere(control.ActiveXControlBinary); //Χρησιμοποιήστε τη δική σας μέθοδο για τη διαχείριση των ιδιοτήτων ActiveX που αποθηκεύονται στο δυαδικό της αρχείο
}
```

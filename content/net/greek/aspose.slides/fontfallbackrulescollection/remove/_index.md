## FontFallBackRulesCollection.Remove μέθοδος

Αφαιρεί την πρώτη εμφάνιση ενός συγκεκριμένου κανόνα FallBack από τη συλλογή.

```csharp
public void Remove(IFontFallBackRule targetRule)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| targetRule | IFontFallBackRule | Ο κανόνας που θα αφαιρεθεί από τη συλλογή. |

### Παραδείγματα

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    //Λήψη μιας κενής ή προαρχικοποιημένης συλλογής κανόνων από το FontsManager
    IFontFallBackRulesCollection rulesList = pres.FontsManager.FontFallBackRulesCollection;

    //Προσθήκη πολλών κανόνων στη συλλογή
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));
    rulesList.Add(new FontFallBackRule(0x3040, 0x309F, "MS Mincho"));

    //Ανάκτηση του αντικειμένου του πρώτου κανόνα στη συλλογή
    IFontFallBackRule firstRule = rulesList[0];
    //Αφαίρεση 
    rulesList.Remove (firstRule);
}
```

## FontFallBackRulesCollection.Add μέθοδος

Προσθέτει έναν καθορισμένο κανόνα FallBack στο τέλος της συλλογής.

```csharp
public void Add(IFontFallBackRule sourceRule)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| sourceRule | IFontFallBackRule | Καθορισμένος κανόνας για προσθήκη |

### Παραδείγματα

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    //Ανάκτηση κενής ή προαρχικοποιημένης συλλογής κανόνων από το FontsManager
    IFontFallBackRulesCollection rulesList = pres.FontsManager.FontFallBackRulesCollection;

    //Προσθήκη νέου κανόνα στη συλλογή
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));
}
```

## FontsLoader.LoadExternalFonts μέθοδος

Προσθέτει πρόσθετους φακέλους για την ανεύρεση γραμματοσειρών.

```csharp
public static void LoadExternalFonts(string[] directories)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| directories | String[] | Κατάλογοι για την ανάγνωση πρόσθετων γραμματοσειρών. |

### Παραδείγματα

Τα παρακάτω παραδείγματα δείχνουν πώς να φορτώσετε προσαρμοσμένες γραμματοσειρές από .TTF

```csharp
[C#]
// Η διαδρομή προς το φάκελο εγγράφων.
string dataDir = "C:\\";
// Φάκελοι για την αναζήτηση γραμματοσειρών
String[] folders = new String[] { dataDir };
// Φόρτωση των γραμματοσειρών από προσαρμοσμένο φάκελο.
FontsLoader.LoadExternalFonts(folders);
// Κάντε κάποια εργασία και εκτελέστε την απόδοση παρουσίασης/διαφάνειας
using (Presentation presentation = new Presentation(dataDir + "DefaultFonts.pptx"))
{
    presentation.Save(dataDir + "NewFonts_out.pptx", SaveFormat.Pptx);
	// Καθαρισμός κρυφής μνήμης γραμματοσειρών
	FontsLoader.ClearCache();
}
```

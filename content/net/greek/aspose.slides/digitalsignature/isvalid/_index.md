## DigitalSignature.IsValid ιδιότητα

Εάν αυτή η ψηφιακή υπογραφή είναι έγκυρη και το έγγραφο δεν έχει αλλοιωθεί, αυτή η τιμή θα είναι true. Μόνο-ανάγνωση Boolean.

```csharp
public bool IsValid { get; }
```

{

}

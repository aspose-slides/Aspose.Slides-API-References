## IOleObjectFrame.LinkPathRelative ιδιότητα

Επιστρέφει τη σχετική διαδρομή σε ένα συνδεδεμένο αρχείο εάν υπάρχει, διαφορετικά επιστρέφει κενή συμβολοσειρά. Μόνο ανάγνωση String.

```csharp
public string LinkPathRelative { get; }
```

### Σχόλια

Σ στις παρουσιάσεις Ppt, ορισμένες συνδέσεις αντικειμένων Ole ενδέχεται να έχουν σχετική αναπαράσταση.

### Παραδείγματα

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.ppt"))
{
    IOleObjectFrame oleFrame = presentation.Slides[0].Shapes[0] as IOleObjectFrame;

    if (oleFrame != null)
    {
        Console.WriteLine("The relative path: " + oleFrame.LinkPathRelative);
    } 
}
```

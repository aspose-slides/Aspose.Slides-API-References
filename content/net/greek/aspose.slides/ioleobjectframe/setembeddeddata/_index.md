## IOleObjectFrame.SetEmbeddedData method

Ορίζει πληροφορίες σχετικά με τα ενσωματωμένα δεδομένα OLE.

```csharp
public void SetEmbeddedData(IOleEmbeddedDataInfo embeddedData)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| embeddedData | IOleEmbeddedDataInfo | Ενσωματωμένα δεδομένα [`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo) |

### Εξαιρέσεις

| Εξαίρεση | Συνθήκη |
| --- | --- |
| ArgumentNullException | Όταν η παράμετρος embeddedData είναι null. |

### Παρατηρήσεις

Αυτή η μέθοδος αλλάζει τις ιδιότητες του αντικειμένου ώστε να αντανακλούν τα νέα δεδομένα και θέτει τη σημαία IsObjectLink σε false, υποδεικνύοντας ότι το αντικείμενο OLE είναι ενσωματωμένο.

### Παραδείγματα

Το παρακάτω παράδειγμα δείχνει πώς να αλλάξετε τα ενσωματωμένα δεδομένα OLE και τον τύπο τους για ένα υπάρχον [`IOleObjectFrame`](../../ioleobjectframe) αντικείμενο

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    OleObjectFrame oof = pres.Slides[0].Shapes[0] as OleObjectFrame;
    if (oof != null)
    {
        IOleEmbeddedDataInfo newData = new OleEmbeddedDataInfo(File.ReadAllBytes("Picture.png"), "png");
        oof.SetEmbeddedData(newData);
    }
}
```

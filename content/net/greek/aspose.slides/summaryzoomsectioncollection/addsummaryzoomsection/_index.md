## SummaryZoomSectionCollection.AddSummaryZoomSection μέθοδος

Δημιουργεί νέο αντικείμενο Summary Zoom Section και το προσθέτει στη συλλογή

```csharp
public ISummaryZoomSection AddSummaryZoomSection(ISection section)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| section | ISection | Τμήμα για ένα νέο στοιχείο Summary Zoom Section [`ISection`](../../isection) |

### Τιμή Επιστροφής

Το στοιχείο [`ISummaryZoomFrame`](../../isummaryzoomframe) προστέθηκε

### Εξαιρέσεις

| Εξαίρεση | Συνθήκη |
| --- | --- |
| ArgumentException | Η αναφερόμενη ενότητα δεν ανήκει στην τρέχουσα παρουσίαση ή δεν περιέχει διαφάνειες. |

### Παρατηρήσεις

Εάν ένα στοιχείο για αυτήν την ενότητα υπάρχει ήδη στη συλλογή, επιστρέφεται το υπάρχον στοιχείο.

### Παραδείγματα

Το παράδειγμα δείχνει τη λήψη του στοιχείου Summary Zoom Section με βάση το ευρετήριο:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    ISummaryZoomFrame zoomFrame = (ISummaryZoomFrame) pres.Slides[1].Shapes[0];
    ISummaryZoomSectionCollection collection = zoomFrame.SummaryZoomCollection;
    ISummaryZoomSection newZoomSection = collection.AddSummaryZoomSection(pres.Sections[3]);
}
```

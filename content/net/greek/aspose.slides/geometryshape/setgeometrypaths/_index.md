## GeometryShape.SetGeometryPaths μέθοδος

Ενημερώνει τη γεωμετρία του σχήματος από έναν πίνακα των [`IGeometryPath`](../../igeometrypath). Οι συντεταγμένες πρέπει να είναι σχετικές με την αριστερή ανώτερη γωνία του σχήματος. Αλλάζει τον τύπο του σχήματος ([`ShapeType`](../shapetype)) σε Custom.

```csharp
public void SetGeometryPaths(IGeometryPath[] geometryPaths)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| geometryPaths | IGeometryPath[] | Πίνακας γεωμετρικών διαδρομών |

### Εξαιρέσεις

| εξαίρεση | συνθήκη |
| --- | --- |
| ArgumentException | Δεν βρέθηκε διαδρομή |
| ArgumentException | Κενή διαδρομή |

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    GeometryShape shape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 200, 100) as GeometryShape;

    GeometryPath geometryPath = shape.GetGeometryPaths()[0];

    geometryPath.LineTo(100, 50, 1);
    geometryPath.LineTo(100, 50, 4);

    shape.SetGeometryPath(geometryPath);

    pres.Save("output.pptx", SaveFormat.Pptx);
}
```

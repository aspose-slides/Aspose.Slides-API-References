## SetLicense(string) {#setlicense_1}

Παρέχει άδεια στο στοιχείο.

```csharp
public void SetLicense(string licenseName)
```

| Parameter | Type | Description |
| --- | --- | --- |
| licenseName | String | Μπορεί να είναι το πλήρες ή σύντομο όνομα αρχείου ή το όνομα ενός ενσωματωμένου πόρου. Χρησιμοποιήστε μια κενή συμβολοσειρά για να μεταβείτε σε λειτουργία αξιολόγησης. |

### Σχόλια

Προσπαθεί να βρει την άδεια στις ακόλουθες τοποθεσίες:

1. Ρητή διαδρομή.
2. Ο φάκελος του assembly του στοιχείου.
3. Ο φάκελος του assembly κλήσης του πελάτη.
4. Ο φάκελος του entry assembly.
5. Ένας ενσωματωμένος πόρος στο assembly κλήσης του πελάτη.

**Σημείωση:**Στο .NET Compact Framework, προσπαθεί να βρει την άδεια μόνο σε αυτές τις τοποθεσίες:

1. Ρητή διαδρομή.
2. Ένας ενσωματωμένος πόρος στο assembly κλήσης του πελάτη.

### Παραδείγματα

Σε αυτό το παράδειγμα, θα γίνει προσπάθεια να βρεθεί ένα αρχείο άδειας με όνομα MyLicense.lic στον φάκελο που περιέχει το στοιχείο, στον φάκελο που περιέχει το assembly κλήσης, στον φάκελο του entry assembly και, τέλος, στους ενσωματωμένους πόρους του assembly κλήσης.

```csharp
[C#]

License license = new License();
license.SetLicense("MyLicense.lic");


[Visual Basic]

Dim license As License = New License
license.SetLicense("MyLicense.lic")
```

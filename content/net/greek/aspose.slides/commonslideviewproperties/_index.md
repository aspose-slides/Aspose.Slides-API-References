### Παραδείγματα

Το παρακάτω παράδειγμα δείχνει πώς να ορίσετε την τιμή μεγέθυνσης για τη διαφάνεια μιας παρουσίασης PowerPoint.

```csharp
[C#]
// Δημιουργήστε ένα αντικείμενο Presentation που αντιπροσωπεύει ένα αρχείο παρουσίασης
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Ορισμός ιδιοτήτων προβολής της παρουσίασης
    presentation.ViewProperties.SlideViewProperties.Scale = 100; // Τιμή μεγέθυνσης σε ποσοστά για την προβολή διαφάνειας
	presentation.ViewProperties.NotesViewProperties.Scale = 100; // Τιμή μεγέθυνσης σε ποσοστά για την προβολή σημειώσεων
    presentation.Save("Zoom_out.pptx", SaveFormat.Pptx);
}
```

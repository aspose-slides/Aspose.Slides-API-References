## IPictureFillFormat.TileOffsetY ιδιότητα

Επιστρέφει ή ορίζει την κάθετη μετατόπιση της υφής από το αρχικό σημείο του σχήματος σε μονάδες σημείου. Μια θετική τιμή μετακινεί την υφή προς τα κάτω, ενώ μια αρνητική τιμή τη μετακινεί προς τα πάνω. Ανάγνωση/εγγραφή Single.

```csharp
public float TileOffsetY { get; set; }
```

### Παραδείγματα

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Λαμβάνει τη μορφή γεμίσματος εικόνας του σχήματος
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Ορίζει τη λειτουργία γεμίσματος εικόνας σε Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Ορίζει την κάθετη μετατόπιση της υφής σε -50 σημεία
    pictureFillFormat.TileOffsetY = -50;
}
```

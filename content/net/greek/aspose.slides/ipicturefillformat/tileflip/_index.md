## IPictureFillFormat.TileFlip ιδιότητα

Περιέχει την αντιστροφή του πλακιδίου υφής γύρω από τον οριζόντιο, κάθετο ή και τους δύο άξονες. Ανάγνωση/εγγραφή [`TileFlip`](../../tileflip).

```csharp
public TileFlip TileFlip { get; set; }
```

### Παρατηρήσεις

Η προεπιλογή είναι NoFlip.

### Παραδείγματα

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Λαμβάνει τη μορφή γεμίσματος εικόνας του σχήματος
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Ορίζει τη λειτουργία γεμίσματος εικόνας σε Πλακίδιο
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Αναστρέφει το πλακίδιο υφής γύρω από τον κάθετο άξονα.
    pictureFillFormat.TileFlip = TileFlip.FlipY;
}
```

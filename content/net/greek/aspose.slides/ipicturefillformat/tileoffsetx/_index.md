## IPictureFillFormat.TileOffsetX ιδιότητα

Επιστρέφει ή ορίζει την οριζόντια μετατόπιση της υφής από το αρχικό σημείο του σχήματος σε μονάδες μέτρησης (points). Μια θετική τιμή μετακινεί την υφή προς τα δεξιά, ενώ μια αρνητική τιμή τη μετακινεί προς τα αριστερά. Ανάγνωση/Εγγραφή Single.

```csharp
public float TileOffsetX { get; set; }
```

### Παραδείγματα

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Αποκτά τη μορφή γέμισης εικόνας του σχήματος
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Ορίζει τη λειτουργία γέμισης εικόνας σε Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Ορίζει την οριζόντια μετατόπιση της υφής στα 20 points
    pictureFillFormat.TileOffsetX = 20f;
}
```

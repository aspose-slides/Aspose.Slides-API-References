## IPictureFillFormat.DeletePictureCroppedAreas μέθοδος

Διαγράφει τις περικομμένες περιοχές του fill Picture.

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### Return Value

Η περικομμένη εικόνα ή η αρχική εικόνα εάν η περικοπή δεν είναι απαραίτητη.

### Remarks

Αυτή η μέθοδος μετατρέπει τα metafiles WMF/EMF σε raster PNG εικόνα ενώ περικόπτε.

### Examples

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Λαμβάνει το PictureFrame
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // Διαγράφει τις περικομμένες περιοχές της εικόνας του PictureFrame
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

## CompressImage(bool, PicturesCompression) {#compressimage}

Συμπιέζει την εικόνα μειώνοντας το μέγεθός της βάσει του μεγέθους του σχήματος και της καθορισμένης ανάλυσης. Προαιρετικά, διαγράφει επίσης τις περικομμένες περιοχές.

```csharp
public bool CompressImage(bool deleteCroppedAreasOfImage, PicturesCompression resolution)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| deleteCroppedAreasOfImage | Boolean | Εάν είναι true, η μέθοδος θα καταργήσει τις περικομμένες περιοχές της εικόνας, ενδεχομένως μειώνοντας περαιτέρω το μέγεθός της. |
| resolution | PicturesCompression | Η στοχευμένη ανάλυση για συμπίεση, καθορίζεται ως τιμή του [`PicturesCompression`](../../../aspose.slides.export/picturescompression) enum. |

### Τιμή επιστροφής

Ένα Boolean που δείχνει αν η εικόνα συμπιέστηκε επιτυχώς. Επιστρέφει `true` εάν η εικόνα μεγέθυνε ή περικόπηκε, διαφορετικά `false`.

### Εξαιρέσεις

| εξαίρεση | συνθήκη |
| --- | --- |
| ArgumentException | Ρίχνεται όταν η ανάλυση δεν είναι έγκυρη τιμή. |

### Σχόλια

Αυτή η μέθοδος αλλάζει το μέγεθος και την ανάλυση της εικόνας παρόμοια με τη λειτουργία του PowerPoint «Μορφή εικόνας → Συμπίεση εικόνων».

### Παραδείγματα

Το παρακάτω παράδειγμα δείχνει πώς να χρησιμοποιήσετε τη μέθοδο `CompressImage` για να μειώσετε το μέγεθος μιας εικόνας σε μια παρουσίαση ορίζοντας μια στοχευμένη ανάλυση και αφαιρώντας τις περικομμένες περιοχές:

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;
    // Συμπιέζει την εικόνα με στοχευμένη ανάλυση 150 DPI (ανάλυση ιστού) και αφαιρεί τις περικομμένες περιοχές
    bool result = picFrame.PictureFormat.CompressImage(true, PicturesCompression.Dpi150);
}
```

## WebDocument.Save μέθοδος

Αποθηκεύει την έξοδο του εγγράφου.

```csharp
public void Save()
```

### Παραδείγματα

```csharp
[C#]        
using (Presentation pres = new Presentation("pres.pptx"))
{
    var options = new WebDocumentOptions
    {
        TemplateEngine = new RazorTemplateEngine(),
        OutputSaver = new FileOutputSaver(),
        EmbedImages = false
    };
    
    WebDocument document = new WebDocument(options);

    // προσθέστε το πρότυπο "index-template.html" με το κλειδί προτύπου "index" για να το χρησιμοποιήσετε αργότερα (για έξοδο)
    document.Input.AddTemplate<Aspose.Slides.Presentation>("index", "index-template.html");

    // προσθέστε το "index.html" στα αρχεία εξόδου, χρησιμοποιώντας το πρότυπο "index" για να το δημιουργήσετε και τη μεταβλητή pres ως μοντέλο
    document.Output.Add("index.html", "index", pres);
    
    // ... ρυθμίστε άλλες επιλογές του εγγράφου και έπειτα αποθηκεύστε το έγγραφο
    document.Global.Put("slideMargin", 10);
    document.Global.Put("imagesPath", "root/site/images");
    // ...
 
    document.Save();
}
```

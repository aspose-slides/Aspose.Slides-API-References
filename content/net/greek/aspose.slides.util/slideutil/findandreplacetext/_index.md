## SlideUtil.FindAndReplaceText μέθοδος

Εντοπίζει και αντικαθιστά κείμενο στην παρουσίαση με δεδομένη μορφή

```csharp
public static void FindAndReplaceText(IPresentation presentation, bool withMasters, string find, 
    string replace, PortionFormat format = null)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| presentation | IPresentation | Σαρωμένη παρουσίαση. |
| withMasters | Boolean | Καθορίζει εάν οι κύριες διαφάνειες πρέπει να σαρωθούν. |
| find | String | Τιμή String για εύρεση. |
| replace | String | Τιμή String για αντικατάσταση. |
| format | PortionFormat | Μορφή για την αντικατάσταση τμήματος κειμένου. Εάν είναι null, θα χρησιμοποιηθεί η μορφή του πρώτου χαρακτήρα της αναζητούμενης συμβολοσειράς |

### Παραδείγματα

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PortionFormat format = new PortionFormat
    {
        FontHeight = 24f,
        FontItalic = NullableBool.True,
        FillFormat =
        {
            FillType = FillType.Solid,
            SolidFillColor =
            {
                Color = Color.Red
            }
        }
    };
    Aspose.Slides.Util.SlideUtil.FindAndReplaceText(pres, true, "[this block] ", "my text ", format);
    pres.Save("replaced", SaveFormat.Pptx);
}
```

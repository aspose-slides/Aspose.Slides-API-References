## IExcelDataWorkbook.GetChartsFromWorksheet μέθοδος

Ανακτά ένα λεξικό που περιέχει τα ευρετήρια και τα ονόματα όλων των γραφημάτων στο καθορισμένο φύλλο εργασίας ενός βιβλίου εργασίας Excel.

```csharp
public IDictionary<int, string> GetChartsFromWorksheet(string worksheetName)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| worksheetName | String | Το όνομα του φύλλου εργασίας για αναζήτηση γραφημάτων. |

### Τιμή Επιστροφής

Ένα λεξικό όπου το κλειδί είναι το ευρετήριο του γραφήματος και η τιμή είναι το όνομα του γραφήματος.

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
IExcelDataWorkbook wb = new ExcelDataWorkbook(testFile);
var sheetCharts = wb.GetChartsFromWorksheet("worksheetName");
foreach (var chart in sheetCharts)
    Console.WriteLine(chart.Key + " : " + chart.Value);
```

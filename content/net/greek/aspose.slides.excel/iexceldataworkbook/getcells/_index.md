## IExcelDataWorkbook.GetCells μέθοδος

Ανακτά μια συλλογή από κελιά από το βιβλίο εργασίας που ταιριάζει με τη συγκεκριμένη φόρμουλα.

```csharp
public ReadOnlyCollection<IExcelDataCell> GetCells(string formula, bool skipHiddenCells)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| formula | String | Μια φόρμουλα ή έκφραση περιοχής (π.χ., "Sheet1!A1:B3") που χρησιμοποιείται για τον προσδιορισμό των κελιών στόχου. |
| skipHiddenCells | Boolean | Εάν `true`, τα κρυφά κελιά (π.χ., σε κρυφές γραμμές ή στήλες) θα εξαιρεθούν από το αποτέλεσμα. |

### Τιμή Επιστροφής

Μια λίστα μόνο για ανάγνωση από κελιά που ταιριάζει με τη συγκεκριμένη φόρμουλα.

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
ExcelDataWorkbook wb = new ExcelDataWorkbook(testFile);
IReadOnlyList<IExcelDataCell> cells = wb.GetCells("Sheet1!A2:A6", false);
Console.WriteLine(cells.Count); //Έξοδος: 5
```

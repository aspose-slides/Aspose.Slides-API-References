## Μέθοδος ExcelDataWorkbook.GetCells

Ανακτά μια συλλογή κελιών από το βιβλίο εργασίας που ταιριάζουν με τον καθορισμένο τύπο.

```csharp
public ReadOnlyCollection<IExcelDataCell> GetCells(string formula, bool skipHiddenCells)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| formula | String | Ένας τύπος ή έκφραση περιοχής (π.χ., "Sheet1!A1:B3") που χρησιμοποιείται για την αναγνώριση των κελιών-στόχων. |
| skipHiddenCells | Boolean | Εάν `true`, τα κρυμμένα κελιά (π.χ., σε κρυμμένες γραμμές ή στήλες) θα εξαιρεθούν από το αποτέλεσμα. |

### Τιμή Επιστροφής

Μια λίστα μόνο για ανάγνωση των κελιών που ταιριάζουν με τον καθορισμένο τύπο.

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
ExcelDataWorkbook wb = new ExcelDataWorkbook(testFile);
IReadOnlyList<IExcelDataCell> cells = wb.GetCells("Sheet1!A2:A6", false);
Console.WriteLine(cells.Count); //Έξοδος: 5
```

## BaseOverrideThemeManager.IsOverrideThemeEnabled ιδιότητα

Καθορίζει εάν το OverrideTheme αντικαθιστά το κληρονομικό αποτελεσματικό θέμα ή όχι. Για να ενεργοποιήσετε το OverrideTheme για αντικατάσταση χρησιμοποιήστε τις μεθόδους OverrideTheme.Init*(). Για να απενεργοποιήσετε το OverrideTheme από την αντικατάσταση χρησιμοποιήστε τη μέθοδο OverrideTheme.Clear(). Μόνο για ανάγνωση Boolean.

```csharp
public bool IsOverrideThemeEnabled { get; }
```

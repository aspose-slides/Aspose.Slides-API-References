## MathMatrix.MinColumnWidth ιδιότητα

Ελάχιστο πλάτος στήλης σε twips (1/20 του σημείου) Το κενό διάστημα (επίσης αναφέρεται ως “Column Gap” ή “Gap Width”) προστίθεται στο MinColumnWidth για να προσδιορίσει το συνολικό Matrix Column Spacing (απόσταση μεταξύ των ίδιων άκρων διαφορετικών στηλών). Προεπιλογή: 0.

```csharp
public uint MinColumnWidth { get; set; }
```

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]

matrix.MinColumnWidth = 20;
```

## MathMatrix.RowGap ιδιότητα

Η τιμή του κάθετου διαστήματος μεταξύ των γραμμών ενός πίνακα· Αν το RowGapRule είναι ορισμένο σε 3 ("Exactly"), τότε η μονάδα ερμηνεύεται ως twips (1/20 του σημείου). Αν το RowGapRule είναι ορισμένο σε 4 ("Multiple"), τότε η μονάδα ερμηνεύεται ως μισές γραμμές. Προεπιλογή: 0

```csharp
public uint RowGap { get; set; }
```

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

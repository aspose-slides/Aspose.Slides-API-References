## MathSpacingRules απαρίθμηση

Τύποι κενού (οριζόντιου διαστήματος) μεταξύ των στηλών ενός πίνακα

```csharp
public enum MathSpacingRules
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| SingleSpacingGap | `0` | Μονό (1 em για στήλες ή 1 γραμμή για σειρές) |
| OneAndHalfSpacingGap | `1` | Μία και μισή (1.5 ems για στήλες ή 1.5 γραμμές για σειρές) |
| DoubleSpacingGap | `2` | Διπλό διάστημα (2 ems για στήλες ή 2 γραμμές για σειρές) |
| Exactly | `3` | Ακριβής τιμή. Για στήλες, βασίζεται στην τιμή του ColumnGap, μετρημένη σε twips. Για σειρές, βασίζεται στην τιμή του RowGap, μετρημένη σε twips |
| Multiple | `4` | Πολλαπλό. Για στήλες, βασίζεται στην τιμή του ColumnGap, μετρημένη σε βήματα 0.5 em. Για σειρές, βασίζεται στην τιμή του RowGap, μετρημένη σε γραμμές |

## IMathMatrix.ColumnGap ιδιότητα

Η τιμή του οριζόντιου διαστήματος μεταξύ των στηλών ενός πίνακα· Αν το ColumnGapRule οριστεί σε 3 ("Exactly"), τότε η μονάδα ερμηνεύεται ως twips (1/20 του σημείου). Αν το ColumnGapRule οριστεί σε 4 ("Multiple"), τότε η μονάδα ερμηνεύεται ως αριθμός 0.5 em βημάτων. Σε άλλες περιπτώσεις αγνοείται. Προεπιλογή: 0

```csharp
public uint ColumnGap { get; set; }
```

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.ColumnGapRule = MathSpacingRules.Exactly;
matrix.ColumnGap = 20;
```

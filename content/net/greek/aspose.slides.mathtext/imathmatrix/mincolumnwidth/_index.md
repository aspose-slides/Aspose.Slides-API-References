## IMathMatrix.MinColumnWidth ιδιότητα

Ελάχιστο πλάτος στήλης σε twips (1/20 του σημείου) Η απόσταση του κενού (επίσης αναφέρεται ως “Column Gap” ή “Gap Width”) προστίθεται στο MinColumnWidth για να καθοριστεί η συνολική απόσταση Matrix Column Spacing (απόσταση μεταξύ των ίδιων άκρων διαφορετικών στηλών). Προεπιλογή: 0.

```csharp
public uint MinColumnWidth { get; set; }
```

## MathParagraph.Remove μέθοδος

Αφαιρεί την πρώτη εμφάνιση ενός συγκεκριμένου αντικειμένου από τη συλλογή/>.

```csharp
public bool Remove(IMathBlock mathBlock)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| mathBlock | IMathBlock | Το αντικείμενο που θα αφαιρεθεί από τη συλλογή. |

### Τιμή επιστροφής

true αν *mathBlock* καταργήθηκε επιτυχώς από τη συλλογή· διαφορετικά, false. Αυτή η μέθοδος επίσης επιστρέφει false αν *mathBlock* δεν βρεθεί στην αρχική συλλογή/>.

### Παραδείγματα

Example:

```csharp
[C#]
IAutoShape shape = slide.Shapes.AddMathShape(x, y, width, height);
IMathParagraph mathParagraph = (shape.TextFrame.Paragraphs[0].Portions[0] as MathPortion).MathParagraph;
mathParagraph.Add(new MathBlock(new MathematicalText("x")));
IMathBlock block = new MathBlock(new MathematicalText("y"));
mathParagraph.Add(block);
mathParagraph.Remove(block);
```

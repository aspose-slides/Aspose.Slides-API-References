## MathBox.NoBreak ιδιότητα

Καμία διάσπαση Αυτή η ιδιότητα καθορίζει την ιδιότητα "αδιαπρίσιμη" στο πλαίσιο αντικειμένου. Όταν είναι true, δεν μπορούν να εμφανιστούν αλλαγές γραμμής μέσα στο πλαίσιο. Αυτό μπορεί να είναι σημαντικό για τις προσομοιώσεις χειριστών που αποτελούνται από περισσότερους από ένα δυαδικούς χειριστές. Όταν αυτό το στοιχείο δεν έχει οριστεί, μπορούν να εμφανιστούν διακοπές μέσα στο πλαίσιο. Default: true

```csharp
public bool NoBreak { get; set; }
```

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("*****"));
box.NoBreak = false;
```

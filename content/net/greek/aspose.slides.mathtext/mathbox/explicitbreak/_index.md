## MathBox.ExplicitBreak ιδιότητα

Explicit break καθορίζει αν υπάρχει αλλαγή γραμμής στην αρχή του αντικειμένου Box, έτσι ώστε η γραμμή να τυλίγεται στην αρχή του αντικειμένου box. Καθορίζει τον αριθμό του τελεστή στη προηγούμενη γραμμή του μαθηματικού κειμένου που θα χρησιμοποιηθεί ως σημείο στοίχισης για την τρέχουσα γραμμή του μαθηματικού κειμένου δυνατές τιμές: 1..255 Προεπιλογή: 0 (no explicit break)

```csharp
public byte ExplicitBreak { get; set; }
```

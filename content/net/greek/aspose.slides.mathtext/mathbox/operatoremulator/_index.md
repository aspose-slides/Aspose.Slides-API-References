## MathBox.OperatorEmulator ιδιότητα

Operator Emulator. Όταν είναι true, το πλαίσιο και τα περιεχόμενά του συμπεριφέρονται ως ένας ενιαίος τελεστής και κληρονομούν τις ιδιότητες ενός τελεστή. Αυτό σημαίνει, για παράδειγμα, ότι ο χαρακτήρας μπορεί να λειτουργήσει ως σημείο για αλλαγή γραμμής και μπορεί να ευθυγραμμιστεί με άλλους τελεστές. Operator Emulators χρησιμοποιούνται συχνά όταν ένα ή περισσότερα σύμβολα συνδυάζονται για να σχηματίσουν έναν τελεστή, όπως το '=='. Προεπιλεγμένη τιμή: false

```csharp
public bool OperatorEmulator { get; set; }
```

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("=="));
box.OperatorEmulator = true;
```

## IMathDelimiter.GrowToMatchOperandHeight ιδιότητα

Καθορίζει την αύξηση του BeginningCharacter, SeparatorCharacter, EndingCharacter. Όταν είναι true, τα διαχωριστικά αυξάνονται κάθετα ώστε να ταιριάζουν με το ύψος του τελεστέου. Η προεπιλεγμένη τιμή είναι true

```csharp
public bool GrowToMatchOperandHeight { get; set; }
```

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.GrowToMatchOperandHeight = false;
```

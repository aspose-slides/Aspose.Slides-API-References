## IMathPhantom.ZeroDesc ιδιότητα

Λαμβάνει ή ορίζει μια τιμή που υποδεικνύει εάν η κατάπτωση (βάθος κάτω από τη γραμμή αναφοράς) του στοιχείου βάσης πρέπει να θεωρηθεί μηδέν.

```csharp
public bool ZeroDesc { get; set; }
```

### Παρατηρήσεις

Όταν `true`, το φάντασμα δεν μειώνει τη γραμμή αναφοράς της γύρω μαθηματικής γραμμής. Αντιστοιχεί στο χαρακτηριστικό OMML `m:zeroDesc`.

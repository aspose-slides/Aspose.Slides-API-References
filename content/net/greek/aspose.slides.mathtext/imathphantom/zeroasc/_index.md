## IMMathPhantom.ZeroAsc ιδιότητα

Λαμβάνει ή ορίζει μια τιμή που υποδεικνύει εάν η άνοδος (υψόμετρο πάνω από τη γραμμή βάσης) του βασικού στοιχείου πρέπει να θεωρείται μηδέν.

```csharp
public bool ZeroAsc { get; set; }
```

### Παρατηρήσεις

Όταν `true`, το φάντασμα δεν ανεβάζει τη γραμμή βάσης της περιβάλλουσας μαθηματικής γραμμής. Αντιστοιχεί στο χαρακτηριστικό OMML `m:zeroAsc`.

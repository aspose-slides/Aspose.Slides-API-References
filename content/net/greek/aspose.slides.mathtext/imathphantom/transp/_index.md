## IMathPhantom.Transp ιδιότητα

Λαμβάνει ή ορίζει μια τιμή που υποδεικνύει αν το φάντασμα είναι διαφανές για τους κανόνες διαστήματος βάσει κλάσης.

```csharp
public bool Transp { get; set; }
```

### Σχόλια

Όταν `true`, οι τελεστές και τα σύμβολα μέσα στο φάντασμα εξακολουθούν να επηρεάζουν τη μαθηματική διάσταση γύρω από το φάντασμα (σαν να είναι ορατά). Όταν `false`, η διάσταση βάσει κλάσης αγνοείται. Αντιστοιχεί στο χαρακτηριστικό OMML `m:transp`.

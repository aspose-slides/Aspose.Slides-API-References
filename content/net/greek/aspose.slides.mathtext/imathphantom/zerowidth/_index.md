## IMathPhantom.ZeroWidth ιδιότητα

Λαμβάνει ή ορίζει μια τιμή που υποδεικνύει εάν το πλάτος του στοιχείου βάσης θα πρέπει να θεωρηθεί μηδενικό.

```csharp
public bool ZeroWidth { get; set; }
```

### Παρατηρήσεις

Όταν `true`, το φάντασμα δεν διατηρεί οριζόντιο χώρο για τη βάση του. Αντιστοιχεί στο χαρακτηριστικό OMML `m:zeroWid`.

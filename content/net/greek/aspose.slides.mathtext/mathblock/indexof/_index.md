## MathBlock.IndexOf μέθοδος

Καθορίζει τη θέση ενός συγκεκριμένου μαθηματικού στοιχείου στη συλλογή.

```csharp
public int IndexOf(IMathElement item)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| item | IMathElement | Το στοιχείο που θα εντοπιστεί στη συλλογή. |

### Τιμή Επιστροφής

Η θέση του *item* αν βρεθεί στη συλλογή· διαφορετικά, -1.

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
MathBlock mathBlock = new MathBlock(new MathematicalText("x"));
MathematicalText plusElement = new MathematicalText("+");
mathBlock.Add(plusElement);
mathBlock.Add(new MathRadical(new MathematicalText("x"), new MathematicalText("3")));
int index = mathBlock.IndexOf(plusElement);
```

## MathDelimiter.GrowToMatchOperandHeight ιδιότητα

Καθορίζει την αύξηση του BeginningCharacter, SeparatorCharacter, EndingCharacter. Όταν είναι true, τα διαχωριστικά μεγαλώνουν κατακόρυφα ώστε να ταιριάζουν με το ύψος του τελεστή τους. Η προεπιλεγμένη τιμή είναι true

```csharp
public bool GrowToMatchOperandHeight { get; set; }
```

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.GrowToMatchOperandHeight = false;
```

## MathBorderBox.StrikethroughBottomLeftToTopRight ιδιότητα

Διαγράμμισμα Κάτω-Αριστερά προς Πάνω-Δεξιά (προεπιλογή είναι false). Καθορίζει την κατάσταση κρυμμένης ή εμφανιζόμενης διαγώνιας γραμμής διαγράμμισματος από την κάτω-αριστερή γωνία έως την πάνω-δεξιά γωνία του κουτιού περιγράμματος.

```csharp
public bool StrikethroughBottomLeftToTopRight { get; set; }
```

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]

borderBox.StrikethroughBottomLeftToTopRight = true;
```

## IMathBox.OperatorEmulator ιδιότητα

Εξομοιωτής τελεστή. Όταν είναι αληθές, το κουτί και το περιεχόμενό του συμπεριφέρονται ως ένας ενιαίος τελεστής και κληρονομούν τις ιδιότητες ενός τελεστή. Αυτό σημαίνει, για παράδειγμα, ότι ο χαρακτήρας μπορεί να λειτουργήσει ως σημείο διακοπής γραμμής και να ευθυγραμμιστεί με άλλους τελεστές. Οι Εξομοιωτές τελεστή χρησιμοποιούνται συχνά όταν ένα ή περισσότερα σύμβολα συνδυάζονται για να σχηματίσουν έναν τελεστή, όπως το '=='. Προεπιλεγμένη τιμή: false

```csharp
public bool OperatorEmulator { get; set; }
```

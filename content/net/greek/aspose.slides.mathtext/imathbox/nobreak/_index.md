## IMathBox.NoBreak ιδιότητα

Χωρίς διακοπή. Αυτή η ιδιότητα καθορίζει την ιδιότητα "αδιάσπαστη" στο κουτί αντικειμένου. Όταν είναι true, δεν μπορούν να εμφανιστούν αλλαγές γραμμής μέσα στο κουτί. Αυτό μπορεί να είναι σημαντικό για τους προσομοιωτές τελεστών που περιέχουν περισσότερους από έναν δυαδικό τελεστή. Όταν αυτό το στοιχείο δεν καθορίζεται, μπορούν να συμβούν διακοπές μέσα στο κουτί. Προεπιλογή: true

```csharp
public bool NoBreak { get; set; }
```

box.NoBreak = false;

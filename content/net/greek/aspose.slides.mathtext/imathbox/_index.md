## Διασύνδεση IMathBox

Καθορίζει τη λογική συσκευασία (packaging) μαθηματικού στοιχείου. Για παράδειγμα, ένα αντικείμενο σε κουτί μπορεί να λειτουργήσει ως προσομοιωτής τελεστή με ή χωρίς σημείο στοίχισης, να λειτουργήσει ως σημείο αλλαγής γραμμής ή να ομαδοποιηθεί ώστε να μη επιτρέπονται αλλαγές γραμμής εντός του. Για παράδειγμα, ο τελεστής "==" θα πρέπει να τοποθετηθεί σε κουτί για να αποτραπούν οι αλλαγές γραμμής.

```csharp
public interface IMathBox : IMathElement
```

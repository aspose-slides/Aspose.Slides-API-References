## IMathBox.ExplicitBreak ιδιότητα

Η ρητή διακοπή καθορίζει εάν υπάρχει αλλαγή γραμμής στην αρχή του αντικειμένου Box, ώστε η γραμμή να τυλίγεται στην αρχή του αντικειμένου Box. Καθορίζει τον αριθμό του τελεστή στη προηγούμενη γραμμή μαθηματικού κειμένου που θα χρησιμοποιηθεί ως σημείο ευθυγράμμισης για την τρέχουσα γραμμή μαθηματικού κειμένου. Επιτρεπτές τιμές: 1..255 Προεπιλογή: 0 (χωρίς ρητή διακοπή)

```csharp
public byte ExplicitBreak { get; set; }
```

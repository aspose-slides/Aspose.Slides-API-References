## MathPhantom.ZeroAsc ιδιότητα

Λαμβάνει ή ορίζει μια τιμή που υποδεικνύει εάν η ανύψωση (ύψος πάνω από τη γραμμή βάσης) του βασικού στοιχείου πρέπει να αντιμετωπίζεται ως μηδέν.

```csharp
public bool ZeroAsc { get; set; }
```

### Σχόλια

Όταν `true`, το phantom δεν ανυψώνει τη γραμμή βάσης της περιβάλλουσας μαθηματικής γραμμής. Αντιστοιχεί στο χαρακτηριστικό OMML `m:zeroAsc`.

## MathPhantom.ZeroDesc ιδιότητα

Λαμβάνει ή ορίζει μια τιμή που υποδεικνύει εάν η κατάπτωση (βάθος κάτω από τη γραμμή βάσης) του βασικού στοιχείου πρέπει να αντιμετωπίζεται ως μηδέν.

```csharp
public bool ZeroDesc { get; set; }
```

### Παρατηρήσεις

Όταν `true`, το phantom δεν χαμηλώνει τη γραμμή βάσης της περιβάλλουσας μαθηματικής γραμμής. Συμπίπτει με το χαρακτηριστικό OMML `m:zeroDesc`.

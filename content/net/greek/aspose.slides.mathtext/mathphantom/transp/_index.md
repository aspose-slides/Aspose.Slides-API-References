## MathPhantom.Transp ιδιότητα

Λαμβάνει ή ορίζει μια τιμή που υποδεικνύει εάν το φάντασμα είναι διαφανές για κανόνες απόστασης που βασίζονται σε κλάση.

```csharp
public bool Transp { get; set; }
```

### Σχόλια

Όταν `true`, οι τελεστές και τα σύμβολα μέσα στο φάντασμα εξακολουθούν να επηρεάζουν τη μαθηματική απόσταση γύρω από το φάντασμα (σαν να είναι ορατό). Όταν `false`, η απόσταση βάσει κλάσης παραβλέπεται. Αντιστοιχεί στο χαρακτηριστικό OMML `m:transp`.

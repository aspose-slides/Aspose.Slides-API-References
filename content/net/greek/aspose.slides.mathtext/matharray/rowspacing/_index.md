## MathArray.RowSpacing ιδιότητα

Απόσταση μεταξύ των σειρών ενός πίνακα. Χρησιμοποιείται μόνο όταν το RowSpacingRule έχει οριστεί σε 3 Exact, στην περίπτωση αυτή η μονάδα μέτρησης είναι σημεία, ή Multiple, στην περίπτωση αυτή η μονάδα μέτρησης είναι μισές γραμμές. Προεπιλογή: 0

```csharp
public uint RowSpacing { get; set; }
```

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
IMathArray mathArray = new MathArray(new MathematicalText("item1"));
mathArray.RowSpacingRule = MathRowSpacingRule.Exactly;
mathArray.RowSpacing = 10;
```

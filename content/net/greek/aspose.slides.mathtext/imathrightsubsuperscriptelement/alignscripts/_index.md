## IMathRightSubSuperscriptElement.AlignScripts ιδιότητα

Καθορίζει την ευθυγράμμιση του subscript/superscript. Όταν είναι true, το subscript και το superscript ευθυγραμμίζονται οριζόντια μεταξύ τους. Όταν είναι false, προσαρμόζονται (kerned) στο σχήμα της βάσης. Η προεπιλεγμένη τιμή είναι false.

```csharp
public bool AlignScripts { get; set; }
```

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
IMathElement baseElement = new MathematicalText("X");
IMathElement subscript = new MathematicalText("i");
IMathElement superscript = new MathematicalText("j");
MathRightSubSuperscriptElement subsuperscript = new MathRightSubSuperscriptElement(baseElement, subscript, superscript);
subsuperscript.AlignScripts = true;
```

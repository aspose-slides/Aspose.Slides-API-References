## IMathElementCollection.IndexOf μέθοδος

Καθορίζει το ευρετήριο ενός συγκεκριμένου μαθηματικού στοιχείου στη συλλογή.

```csharp
public int IndexOf(IMathElement item)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| item | IMathElement | Το στοιχείο που πρέπει να εντοπιστεί στη συλλογή. |

### Τιμή Επιστροφής

Το ευρετήριο του *item* εάν βρεθεί στη συλλογή· διαφορετικά, -1.

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
IMathElementCollection collection = new MathBlock(new MathematicalText("x"));
MathematicalText plusElement = new MathematicalText("+");
collection.Add(plusElement);
collection.Add(new MathRadical(new MathematicalText("x"), new MathematicalText("3")));
int index = collection.IndexOf(plusElement);
```

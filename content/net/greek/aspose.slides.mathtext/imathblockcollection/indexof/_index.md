## IMathBlockCollection.IndexOf μέθοδος

Καθορίζει το ευρετήριο ενός συγκεκριμένου IMathBlock στη συλλογή.

```csharp
public int IndexOf(IMathBlock item)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| item | IMathBlock | Το στοιχείο που θα εντοπιστεί στη συλλογή. |

### Τιμή Επιστροφής

Το ευρετήριο του *item* εάν βρεθεί στη συλλογή; διαφορετικά, -1.

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
IMathBlockCollection blockCollection = new MathParagraph();
IMathBlock block = new MathBlock(new MathematicalText("y"));
blockCollection.Add(block);
int index = blockCollection.IndexOf(block);
```

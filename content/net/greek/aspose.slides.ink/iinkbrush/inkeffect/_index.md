## IInkBrush.InkEffect ιδιότητα

Λαμβάνει τον τύπο του εφέ μελάνης (π.χ., Galaxy, Gold, Silver) που ορίζει το οπτικό στυλ της πινελιάς μελάνης. Η τιμή αναλύεται από την ιδιότητα της πινέλου "inkEffects". Εάν δεν καθοριστεί αναγνωρισμένο εφέ, επιστρέφεται το NotDefined.

```csharp
public InkEffectType InkEffect { get; }
```

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
using (var pres = new Presentation("Presentation.pptx"))
{
    Ink ink = pres.Slides[0].Shapes[0] as Ink;
    IInkBrush brush = ink.Traces[0].Brush;
    Console.WriteLine("InkEffects = {0}", brush.InkEffect);
}
```

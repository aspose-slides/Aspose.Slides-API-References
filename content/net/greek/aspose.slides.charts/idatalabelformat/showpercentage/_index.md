## IDataLabelFormat.ShowPercentage ιδιότητα

Αναπαριστά τη συμπεριφορά εμφάνισης της τιμής ποσοστού ετικέτας δεδομένων ενός συγκεκριμένου γραφήματος. True εμφανίζει την τιμή του ποσοστού. False για απόκρυψη. Ανάγνωση/εγγραφή Boolean.

```csharp
public bool ShowPercentage { get; set; }
```

### Παρατηρήσεις

Εάν το γονικό αντικείμενο του DataLabelFormat είναι μια συλλογή DataLabelCollection ετικετών δεδομένων, τότε αυτή η ιδιότητα λαμβάνει ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας ShowPercentage για τις νέες ετικέτες δεδομένων στη συλλογή DataLabelCollection. Ορίζοντας αυτή την ιδιότητα με τιμή, θέτει επίσης αυτήν την τιμή στην ιδιότητα ShowPercentage για όλες τις ετικέτες δεδομένων στη συλλογή DataLabelCollection (π.χ. "DataLabels.DefaultDataLabelFormat.ShowPercentage = val;" προκαλεί όλες οι DataLabels[i].ShowPercentage να είναι ίσες με val).

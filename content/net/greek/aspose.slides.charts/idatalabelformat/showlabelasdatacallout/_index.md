## IDataLabelFormat.ShowLabelAsDataCallout ιδιότητα

Καθορίζει εάν η ετικέτα δεδομένων του καθορισμένου διαγράμμου θα εμφανιστεί ως κλήση δεδομένων ή ως ετικέτα δεδομένων. Εάν ο γονέας αυτού του αντικειμένου DataLabelFormat είναι μια συλλογή DataLabelCollection ετικετών δεδομένων, τότε αυτή η ιδιότητα λαμβάνει ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας ShowLabelAsDataCallout για τις νέες ετικέτες δεδομένων στη συλλογή DataLabelCollection. Ο καθορισμός αυτής της ιδιότητας με τιμή θέτει επίσης αυτήν την τιμή στην ιδιότητα ShowLabelAsDataCallout για όλες τις ετικέτες δεδομένων στη συλλογή DataLabelCollection (π.χ. "DataLabels.DefaultDataLabelFormat.ShowLabelAsDataCallout = val;" προκαλεί το ότι όλες οι DataLabels[i].ShowLabelAsDataCallout είναι ίσες με val).

```csharp
public bool ShowLabelAsDataCallout { get; set; }
```

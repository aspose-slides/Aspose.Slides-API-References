## IDataLabelFormat.ShowSeriesName ιδιότητα

Επιστρέφει ή ορίζει ένα Boolean που δείχνει τη συμπεριφορά εμφάνισης του ονόματος σειράς για τις ετικέτες δεδομένων σε ένα διάγραμμα. True για εμφάνιση του ονόματος σειράς. False για απόκρυψη. Αναγνώσιμη/Εγγράψιμη Boolean.

```csharp
public bool ShowSeriesName { get; set; }
```

### Παρατηρήσεις

Αν ο γονέας αυτού του αντικειμένου DataLabelFormat είναι μια συλλογή DataLabelCollection ετικετών δεδομένων, τότε αυτή η ιδιότητα παίρνει ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας ShowSeriesName για τις νέες ετικέτες δεδομένων στη συλλογή DataLabelCollection. Ορίστε αυτή την ιδιότητα με τιμή ώστε επίσης να ορίσει αυτή την τιμή στην ιδιότητα ShowSeriesName για όλες τις ετικέτες δεδομένων στη συλλογή DataLabelCollection (π.χ. "DataLabels.DefaultDataLabelFormat.ShowSeriesName = val;" προκαλεί όλες οι DataLabels[i].ShowSeriesName να είναι ίσες με val).

## IDataLabel.IsVisible ιδιότητα

False σημαίνει ότι η ετικέτα δεδομένων δεν είναι ορατή (και έτσι όλες οι Show*-flags (ShowValue, ...) είναι false). Μόνο για ανάγνωση Boolean.

```csharp
public bool IsVisible { get; }
```

### Παρατηρήσεις

Αν η ετικέτα δεδομένων είναι ορατή, μπορείτε να την κρύψετε με τη μέθοδο Hide(). Αλλά αν η ετικέτα δεδομένων δεν είναι ορατή (IsVisible είναι false), μπορείτε να την κάνετε ορατή ορίζοντας τις Show*-flags (ShowValue, ...) στην κατάσταση true.

## IDataLabel.Hide μέθοδος

Κάντε την ετικέτα δεδομένων κρυφή ορίζοντας όλες τις σημαίες Show*-flags (ShowValue, ...) στην κατάσταση false. Το IsVisible θα είναι false μετά από αυτό.

```csharp
public void Hide()
```

### Παρατηρήσεις

Εάν η ετικέτα δεδομένων δεν είναι ορατή (IsVisible είναι false) μπορείτε να κάνετε την ετικέτα δεδομένων ορατή ορίζοντας τις σημαίες Show*-flags (ShowValue, ...) στην κατάσταση true.

## IChartSeries.HasSeriesLines ιδιότητα

Καθορίζει εάν υπάρχουν γραμμές σειράς για αυτή τη σειρά και συναφείς σειρές. Αυτή είναι η ιδιότητα όχι μόνο για αυτή τη σειρά αλλά και για όλες τις σειρές της γονικής ομάδας σειρών – αυτή είναι η προβολή της αντίστοιχης ιδιότητας της ομάδας. Καθώς έτσι αυτή η ιδιότητα είναι μόνο για ανάγνωση. Χρησιμοποιήστε την ιδιότητα ParentSeriesGroup για πρόσβασης στη γονική ομάδα σειρών. Χρησιμοποιήστε την ιδιότητα ParentSeriesGroup.HasSeriesLines ανάγνωση/εγγραφή για αλλαγή τιμής. Χρησιμοποιήστε την ιδιότητα ParentSeriesGroup.SeriesLinesFormat για μορφοποίηση γραμμών σειράς. Μόνο για ανάγνωση Boolean.

```csharp
public bool HasSeriesLines { get; }
```

## IChartSeries.PieSplitPosition ιδιότητα

Καθορίζει μια τιμή που θα χρησιμοποιηθεί για τον προσδιορισμό του ποιων δείκτων δεδομένων βρίσκονται στο δεύτερο κέικ ή ράβδο σε γράφημα τύπου pie-of-pie ή bar-of-pie. Χρησιμοποιείται μαζί με την ιδιότητα PieSplitBy. Αυτή η ιδιότητα δεν είναι μόνο για αυτή τη σειρά αλλά για όλες τις σειρές της γονικής ομάδας σειρών – πρόκειται για προβολή της αντίστοιχης ιδιότητας ομάδας. Συνεπώς αυτή η ιδιότητα είναι μόνο για ανάγνωση. Χρησιμοποιήστε την ιδιότητα ParentSeriesGroup για πρόσβαση στη γονική ομάδα σειρών. Χρησιμοποιήστε την ιδιότητα ParentSeriesGroup.PieSplitPosition ιδιότητα ανάγνωση/εγγραφή για αλλαγή τιμής. Μόνο για ανάγνωση Double.

public double PieSplitPosition { get; }

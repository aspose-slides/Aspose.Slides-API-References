## IChartSeries.GapWidth ιδιότητα

Καθορίζει το διάστημα μεταξύ των ομάδων ράβδων ή στηλών, ως ποσοστό του πλάτους της ράβδους ή της στήλης. Αυτή η ιδιότητα δεν αφορά μόνο αυτή τη σειρά, αλλά και όλες τις σειρές της γονικής ομάδας σειρών – είναι η προβολή της αντίστοιχης ιδιότητας της ομάδας. Συνεπώς, αυτή η ιδιότητα είναι μόνο για ανάγνωση. Χρησιμοποιήστε την ιδιότητα ParentSeriesGroup για πρόσβαση στην γονική ομάδα σειρών. Χρησιμοποιήστε την ιδιότητα ParentSeriesGroup.GapWidth ανάγνωση/εγγραφή για αλλαγή της τιμής. Μόνο για ανάγνωση Int32.

```csharp
public int GapWidth { get; }
```

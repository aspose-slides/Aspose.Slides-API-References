## IChartSeries.FirstSliceAngle ιδιότητα

Καθορίζει τη γωνία του πρώτου κομματιού πίτας ή δακτυλίου, σε μοίρες (δεξιόστροφα από την κορυφή, από 0 έως 360 μοίρες). Αυτή είναι η ιδιότητα όχι μόνο για αυτή τη σειρά, αλλά και για όλες τις σειρές της γονικής ομάδας σειρών – πρόκειται για προβολή της αντίστοιχης ιδιότητας της ομάδας. Επομένως, αυτή η ιδιότητα είναι μόνο για ανάγνωση. Χρησιμοποιήστε την ιδιότητα ParentSeriesGroup για πρόσβαση στη γονική ομάδα σειρών. Χρησιμοποιήστε την ιδιότητα ParentSeriesGroup.FirstSliceAngle ανάγνωση/εγγραφή για να αλλάξετε την τιμή. Μόνο για ανάγνωση UInt16.

```csharp
public ushort FirstSliceAngle { get; }
```

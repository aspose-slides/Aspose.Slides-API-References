## IChartSeries.SecondPieSize ιδιότητα

Καθορίζει το μέγεθος του δεύτερου πίτας ή ράβδου ενός διαγράμματος πίτας-με-πίτα ή διαγράμματος ράβδου-με-πίτα, ως ποσοστό του μεγέθους της πρώτης πίτας (μπορεί να είναι μεταξύ 5 και 200 τοις εκατό). Αυτή η ιδιότητα αφορά όχι μόνο αυτή τη σειρά αλλά και όλες τις σειρές της γονικής ομάδας σειρών – πρόκειται για προβολή της αντίστοιχης ιδιότητας της ομάδας. Συνεπώς αυτή η ιδιότητα είναι μόνο για ανάγνωση. Χρησιμοποιήστε την ιδιότητα ParentSeriesGroup για πρόσβαση στην γονική ομάδα σειρών. Χρησιμοποιήστε την ιδιότητα ParentSeriesGroup.SecondPieSize ανάγνωση/εγγραφή για αλλαγή της τιμής. Μόνο για ανάγνωση UInt16.

```csharp
public ushort SecondPieSize { get; }
```

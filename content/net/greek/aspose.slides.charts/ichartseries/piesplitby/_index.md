## IChartSeries.PieSplitBy ιδιότητα

Καθορίζει πώς να προσδιοριστεί ποια σημεία δεδομένων βρίσκονται στο δεύτερο πίτα ή ράβδο σε γράφημα πίτα-επί-πίτα ή ράβδο-επί-πίτα. Αυτή είναι η ιδιότητα όχι μόνο αυτής της σειράς αλλά και όλων των σειρών της ομάδας γονικής σειράς – αυτή είναι η προβολή της αντίστοιχης ιδιότητας της ομάδας. Και έτσι αυτή η ιδιότητα είναι μόνο για ανάγνωση. Χρησιμοποιήστε την ιδιότητα ParentSeriesGroup για πρόσβαση στην ομάδα γονικής σειράς. Χρησιμοποιήστε την ιδιότητα ανάγνωση/εγγραφή ParentSeriesGroup.PieSplitBy για αλλαγή της τιμής. Μόνο για ανάγνωση [`PieSplitType`](../../piesplittype).

```csharp
public PieSplitType PieSplitBy { get; }
```

### Σχόλια

1) Αυτή είναι η προβολή της ιδιότητας ParentSeriesGroup.PieSplitBy. 2) Εάν η τιμή της ιδιότητας είναι PieSplitType.Custom, τότε μπορείτε να ορίσετε προσαρμοσμένες πληροφορίες διαχωρισμού με την ιδιότητα ParentSeriesGroup.PieSplitCustomPoints.

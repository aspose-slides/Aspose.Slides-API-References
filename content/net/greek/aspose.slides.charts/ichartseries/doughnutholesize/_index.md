## IChartSeries.DoughnutHoleSize ιδιότητα

Καθορίζει το μέγεθος της τρύπας σε διάγραμμα δακτυλίου (μπορεί να είναι μεταξύ 10 και 90 τοις εκατό του μεγέθους της περιοχής σχεδίασης). Αυτή είναι η ιδιότητα όχι μόνο για αυτή τη σειρά αλλά και για όλες τις σειρές της γονικής ομάδας σειρών – πρόκειται για προβολή της αντίστοιχης ιδιότητας της ομάδας. Και έτσι αυτή η ιδιότητα είναι μόνο για ανάγνωση. Χρησιμοποιήστε την ιδιότητα ParentSeriesGroup για πρόσβαση στην γονική ομάδα σειρών. Χρησιμοποιήστε την ιδιότητα ParentSeriesGroup.DoughnutHoleSize ανάγνωση/εγγραφή για να αλλάξετε την τιμή. Μόνο για ανάγνωση Byte.

```csharp
public byte DoughnutHoleSize { get; }
```

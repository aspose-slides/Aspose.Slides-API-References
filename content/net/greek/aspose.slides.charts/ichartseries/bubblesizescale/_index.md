## IChartSeries.BubbleSizeScale ιδιότητα

Καθορίζει τον συντελεστή κλίμακας για το διάγραμμα φυσαλίδων (μπορεί να είναι μεταξύ 0 και 300 τοις εκατό του προεπιλεγμένου μεγέθους). Αυτή η ιδιότητα δεν ισχύει μόνο για αυτή τη σειρά, αλλά και για όλες τις σειρές της ομάδας γονικών σειρών - αυτή είναι η προβολή της αντίστοιχης ιδιότητας της ομάδας. Έτσι αυτή η ιδιότητα είναι μόνο για ανάγνωση. Χρησιμοποιήστε την ιδιότητα ParentSeriesGroup για πρόσβαση στην ομάδα γονικών σειρών. Χρησιμοποιήστε την ιδιότητα ParentSeriesGroup.BubbleSizeScale για ανάγνωση/εγγραφή για την αλλαγή της τιμής.

```csharp
public int BubbleSizeScale { get; }
```

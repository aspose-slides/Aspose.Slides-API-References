## DataLabelFormat.ShowBubbleSize ιδιότητα

Αναπαριστά τη συμπεριφορά εμφάνισης της τιμής μεγέθους φυσαλίδας ετικέτας δεδομένων ενός συγκεκριμένου διαγράμματος. True εμφανίζει την τιμή μεγέθους φυσαλίδας. False για απόκρυψη. Ανάγνωση/εγγραφή Boolean.

```csharp
public bool ShowBubbleSize { get; set; }
```

### Παρατηρήσεις

Αν το γονικό αντικείμενο αυτού του DataLabelFormat είναι μια συλλογή DataLabelCollection ετικετών δεδομένων, τότε αυτή η ιδιότητα λαμβάνει ή ορίζει την προεπιλεγμένη τιμή της ShowBubbleSize ιδιότητα για τις νέες ετικέτες δεδομένων στη DataLabelCollection συλλογή. Ορίζοντας αυτήν την ιδιότητα με τιμή, επίσης ορίζεται αυτή η τιμή στην ShowBubbleSize ιδιότητα για όλες τις ετικέτες δεδομένων στη DataLabelCollection συλλογή (π.χ. "DataLabels.DefaultDataLabelFormat.ShowBubbleSize = val;" προκαλεί όλες οι DataLabels[i].ShowBubbleSize να είναι ίσες με val).

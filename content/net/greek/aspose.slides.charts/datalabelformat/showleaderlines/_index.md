## DataLabelFormat.ShowLeaderLines ιδιότητα

Αντιπροσωπεύει τη συμπεριφορά εμφάνισης των γραμμών οδηγού ετικετών δεδομένων ενός καθορισμένου γραφήματος. True εμφανίζει τις γραμμές οδηγού. False για απόκρυψη. Read/write Boolean.

```csharp
public bool ShowLeaderLines { get; set; }
```

### Παρατηρήσεις

Εάν ο γονέας αυτού του αντικειμένου DataLabelFormat είναι μια συλλογή DataLabelCollection ετικετών δεδομένων, τότε αυτή η ιδιότητα λαμβάνει ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας ShowLeaderLines για τις νέες ετικέτες δεδομένων στη συλλογή DataLabelCollection. Ορισμός αυτής της ιδιότητας με τιμή ορίζει επίσης αυτή την τιμή στην ιδιότητα ShowLeaderLines για όλες τις ετικέτες δεδομένων στη συλλογή DataLabelCollection (π.χ. "DataLabels.DefaultDataLabelFormat.ShowLeaderLines = val;" προκαλεί όλα τα DataLabels[i].ShowLeaderLines να είναι ίσα με val).

## DataLabelFormat.ShowLabelValueFromCell ιδιότητα

Αναπαριστά τη συμπεριφορά εμφάνισης τιμής κελιού ετικέτας δεδομένων ενός συγκεκριμένου διαγράμματος. True εμφανίζει την τιμή του κελιού. False κρύβει. Ανάγνωση/γραφή Boolean.

```csharp
public bool ShowLabelValueFromCell { get; set; }
```

### Παρατηρήσεις

Εάν ο γονέας αυτού του αντικειμένου DataLabelFormat είναι μια συλλογή DataLabelCollection ετικετών δεδομένων, τότε αυτή η ιδιότητα επιστρέφει ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας ShowLabelValueFromCell για τις νέες ετικέτες δεδομένων στη συλλογή DataLabelCollection. Ορισμός αυτής της ιδιότητας με τιμή θέτει επίσης αυτήν την τιμή στην ιδιότητα ShowLabelValueFromCell για όλες τις ετικέτες δεδομένων στη συλλογή DataLabelCollection (π.χ. "DataLabels.DefaultDataLabelFormat.ShowLabelValueFromCell = val;" προκαλεί όλες οι DataLabels[i].ShowLabelValueFromCell να είναι ίσες με val).

## DataLabelFormat.NumberFormat ιδιότητα

Αντιπροσωπεύει τη συμβολοσειρά μορφής για το αντικείμενο DataLabels. Ανάγνωση/εγγραφή String.

```csharp
public string NumberFormat { get; set; }
```

### Παρατηρήσεις

Αν ο γονέας αυτού του αντικειμένου DataLabelFormat είναι μια συλλογή DataLabelCollection ετικετών δεδομένων, τότε αυτή η ιδιότητα ανακτά ή ορίζει την προεπιλεγμένη τιμή της NumberFormat ιδιότητα για τις νέες ετικέτες δεδομένων στη συλλογή DataLabelCollection. Όταν αυτή η ιδιότητα ορίζεται με μια τιμή, η τιμή αυτή ορίζεται επίσης για την NumberFormat ιδιότητα για όλες τις ετικέτες δεδομένων στη συλλογή DataLabelCollection (π.χ. "DataLabels.DefaultDataLabelFormat.NumberFormat = val;" προκαλεί όλα τα DataLabels[i].NumberFormat να ισούν με val).

### Παραδείγματα

```csharp
[C#]
series.Labels.DefaultDataLabelFormat.ShowValue = true;
series.Labels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = false;
series.Labels.DefaultDataLabelFormat.NumberFormat = "0.0%";
```

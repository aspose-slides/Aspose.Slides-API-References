## ChartSeriesGroup.Overlap ιδιότητα

Καθορίζει πόσο πρέπει να επικάλλονται οι μπάρες και οι στήλες σε διαγράμματα 2-Δ, ως ποσοστό (από -100% έως 100%). - -100%: Μέγιστο διάστημα (οι μπάρες είναι εντελώς χωρισμένες). - 0%: Οι μπάρες τοποθετούνται δίπλα-δίπλα χωρίς επικάλυψη ή διάστημα. - 100%: Μέγιστη επικάλυψη (οι μπάρες επικάλλονται πλήρως η μία την άλλη). Αυτή η ιδιότητα είναι ανάγνωση/εγγραφή SByte.

```csharp
public sbyte Overlap { get; set; }
```

### Εξαιρέσεις

| εξαίρεση | συνθήκη |
| --- | --- |
| ArgumentOutOfRangeException | Ενίχνευση όταν η τιμή ορίζεται εκτός του έγκυρου εύρους από -100 έως 100. |

### Παραδείγματα

Το παρακάτω παράδειγμα δείχνει πώς να ορίσετε την επικάλυψη για μια ομάδα σειράς διαγράμματος και να αποδώσετε το προκύπτον διάγραμμα σε μια φόρμα:

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // Ορίστε την επικάλυψη στο 55%
    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

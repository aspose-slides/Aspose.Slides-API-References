## Ιδιότητα DataLabel.IsVisible

False σημαίνει ότι η ετικέτα δεδομένων δεν είναι ορατή (και έτσι όλες οι σημαίες Show*-flags (ShowValue, ...) είναι false). Μόνο ανάγνωση Boolean.

```csharp
public bool IsVisible { get; }
```

### Παρατηρήσεις

Εάν η ετικέτα δεδομένων είναι ορατή, μπορείτε να την κρύψετε με τη μέθοδο Hide(). Αλλά εάν η ετικέτα δεδομένων δεν είναι ορατή (IsVisible είναι false), μπορείτε να κάνετε την ετικέτα δεδομένων ορατή ορίζοντας τις σημαίες Show*-flags (ShowValue, ...) σε κατάσταση true.

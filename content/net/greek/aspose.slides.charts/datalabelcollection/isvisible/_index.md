## DataLabelCollection.IsVisible ιδιότητα

False σημαίνει ότι η ετικέτα δεδομένων δεν είναι ορατή από προεπιλογή (και έτσι όλες οι σημαίες Show*-flags (ShowValue, ...) της ιδιότητας DefaultDataLabelFormat είναι false). Μόνο ανάγνωση Boolean.

```csharp
public bool IsVisible { get; }
```

### Παρατηρήσεις

Αν η ετικέτα δεδομένων είναι ορατή από προεπιλογή, μπορείτε να την κρύψετε από προεπιλογή με τη μέθοδο Hide(). Ωστόσο, εάν η ετικέτα δεδομένων δεν είναι ορατή από προεπιλογή (IsVisible είναι false), μπορείτε να κάνετε την ετικέτα δεδομένων «ορατή από προεπιλογή» ορίζοντας τις σημαίες Show*-flags (ShowValue, ...) της ιδιότητας DefaultDataLabelFormat σε κατάσταση true.

## IChartData.UseSecondaryCategories ιδιότητα

Εάν είναι false τότε η ιδιότητα [`SecondaryCategories`](../secondarycategories) επιστρέφει null και τα δεδομένα στην ιδιότητα [`Categories`](../categories) χρησιμοποιούνται και για τις κύριες και για τις δευτερεύουσες σειρές. Εάν είναι true τότε τα δεδομένα στην ιδιότητα [`SecondaryCategories`](../secondarycategories) χρησιμοποιούνται για τις δευτερεύουσες σειρές και τα δεδομένα στην ιδιότητα [`Categories`](../categories) χρησιμοποιούνται για τις κύριες σειρές. Ανάγνωση/Εγγραφή Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

### Παραδείγματα

Παράδειγμα. Ποιες κατηγορίες σχετίζονται με τις σειρές - ChartData.Categories ή ChartData.SecondaryCategories;

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // οι σχετικές κατηγορίες είναι series.Chart.ChartData.SecondaryCategories
}
else
{
    // οι σχετικές κατηγορίες είναι series.Chart.ChartData.Categories
}
```

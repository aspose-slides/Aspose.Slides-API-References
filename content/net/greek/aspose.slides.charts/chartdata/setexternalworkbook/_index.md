## SetExternalWorkbook(string) {#setexternalworkbook}

Ορίζει το εξωτερικό αρχείο εργασίας ως πηγή δεδομένων για το γράφημα. Τα δεδομένα του γραφήματος θα ενημερωθούν από το αρχείο εργασίας προορισμού.

```csharp
public void SetExternalWorkbook(string workbookPath)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| workbookPath | String | Διαδρομή προς το αρχείο εργασίας προορισμού |

### Εξαιρέσεις

| εξαίρεση | συνθήκη |
| --- | --- |
| InvalidOperationException | Το εξωτερικό αρχείο εργασίας δεν είναι διαθέσιμο ή δεν μπορεί να φορτωθεί. |

### Παραδείγματα

```csharp
[C#]
using (Presentation pres = new Presentation())

   IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Pie, 50, 50, 400, 600, true);
   IChartData chartData = chart.ChartData;
   (chartData as ChartData).SetExternalWorkbook("../../workbook.xlsx");

```

## SetExternalWorkbook(string, bool) {#setexternalworkbook_1}

Ορίζει το εξωτερικό αρχείο εργασίας ως πηγή δεδομένων για το γράφημα.

```csharp
public void SetExternalWorkbook(string workbookPath, bool updateChartData)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| workbookPath | String | Διαδρομή προς το αρχείο εργασίας προορισμού |
| updateChartData | Boolean | Εάν η τιμή είναι false, θα ενημερωθεί μόνο η διαδρομή του αρχείου εργασίας. Τα δεδομένα του γραφήματος δεν θα φορτωθούν και δεν θα ενημερωθούν από το αρχείο εργασίας προορισμού. Μπορεί να χρησιμοποιηθεί όταν το αρχείο εργασίας προορισμού δεν υπάρχει ή δεν είναι διαθέσιμο. Εάν η τιμή είναι true, τα δεδομένα του γραφήματος θα ενημερωθούν από το αρχείο εργασίας προορισμού. |

### Εξαιρέσεις

| εξαίρεση | συνθήκη |
| --- | --- |
| InvalidOperationException | Το εξωτερικό αρχείο εργασίας δεν είναι διαθέσιμο ή δεν μπορεί να φορτωθεί. |

### Παραδείγματα

```csharp
[C#]
using (Presentation pres = new Presentation())

   IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Pie, 50, 50, 400, 600, true);
   IChartData chartData = chart.ChartData;

   (chartData as ChartData).SetExternalWorkbook("http://path/doesnt/exists", false);


```

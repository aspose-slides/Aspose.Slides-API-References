## ChartData.UseSecondaryCategories ιδιότητα

Εάν false, τότε η ιδιότητα [`SecondaryCategories`](../secondarycategories) επιστρέφει null και τα δεδομένα στην ιδιότητα [`Categories`](../categories) χρησιμοποιούνται τόσο για τις πρωτεύουσες όσο και για τις δευτερεύουσες σειρές. Εάν true, τότε τα δεδομένα στην ιδιότητα [`SecondaryCategories`](../secondarycategories) χρησιμοποιούνται για τις δευτερεύουσες σειρές και τα δεδομένα στην ιδιότητα [`Categories`](../categories) χρησιμοποιούνται για τις πρωτεύουσες σειρές. Ανάγνωση/εγγραφή Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

### Παραδείγματα

Παράδειγμα. Ποιες κατηγορίες σχετίζονται με τις σειρές - ChartData.Categories ή ChartData.SecondaryCategories;

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // σχετικές κατηγορίες είναι series.Chart.ChartData.SecondaryCategories
}
else
{
    // σχετικές κατηγορίες είναι series.Chart.ChartData.Categories
}
```

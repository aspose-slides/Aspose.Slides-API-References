## ChartData.SecondaryCategories ιδιότητα

Αποκτά τις δευτερεύουσες κατηγορίες εάν η ιδιότητα [`UseSecondaryCategories`](../usesecondarycategories) είναι αληθής. Μόνο για ανάγνωση [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### Σχόλια

Εάν η ιδιότητα [`UseSecondaryCategories`](../usesecondarycategories) είναι ψευδής, τότε η ιδιότητα `SecondaryCategories` επιστρέφει null και τα δεδομένα στην ιδιότητα [`Categories`](../categories) χρησιμοποιούνται τόσο για τις πρωτεύουσες όσο και για τις δευτερεύουσες σειρές. Εάν η ιδιότητα [`UseSecondaryCategories`](../usesecondarycategories) είναι αληθής, τότε τα δεδομένα σε αυτήν την ιδιότητα `SecondaryCategories` χρησιμοποιούνται για τις δευτερεύουσες σειρές και τα δεδομένα στην ιδιότητα [`Categories`](../categories) χρησιμοποιούνται για τις πρωτεύουσες σειρές.

### Παραδείγματα

Παράδειγμα. Ποιες κατηγορίες σχετίζονται με τις σειρές - ChartData.Categories ή ChartData.SecondaryCategories;

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // σχετικές κατηγορίες είναι series.Chart.ChartData.SecondaryCategories
}
else
{
    // σχετικές κατηγορίες είναι series.Chart.ChartData.Categories
}
```

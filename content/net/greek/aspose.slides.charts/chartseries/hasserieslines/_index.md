## ChartSeries.HasSeriesLines ιδιότητα

Καθορίζει εάν υπάρχουν γραμμές σειράς για αυτή τη σειρά και σχετικές σειρές. Αυτή είναι η ιδιότητα όχι μόνο αυτής της σειράς αλλά και όλων των σειρών της ομάδας γονικής σειράς – πρόκειται για προβολή της αντίστοιχης ιδιότητας της ομάδας. Συνεπώς αυτή η ιδιότητα είναι μόνο για ανάγνωση. Χρησιμοποιήστε την ιδιότητα ParentSeriesGroup για πρόσβαση στην ομάδα γονικής σειράς. Χρησιμοποιήστε την ιδιότητα ParentSeriesGroup.HasSeriesLines ιδιότητα ανάγνωση/εγγραφή για αλλαγή της τιμής. Χρησιμοποιήστε την ιδιότητα ParentSeriesGroup.SeriesLinesFormat ιδιότητα για μορφοποίηση των γραμμών σειράς. Μόνο για ανάγνωση Boolean.

```csharp
public bool HasSeriesLines { get; }
```

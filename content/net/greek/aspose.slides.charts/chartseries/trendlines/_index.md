## ChartSeries.TrendLines ιδιότητα

Συλλογή των γραμμών τάσεων των σειρών. Μόνο για ανάγνωση [`ITrendlineCollection`](../../itrendlinecollection).

```csharp
public ITrendlineCollection TrendLines { get; }
```

### Παρατηρήσεις

Οι TrendLines είναι διαθέσιμες (μη μηδενικές) για τις σειρές δεδομένων σε μη στοιβαγμένες 2-Δ περιοχή, ραβδόγραμμα, στήλη, γραμμή, απόθεσης, xy (διάσπαρτα) και διάγραμμα φυσαλίδων. Μια γραμμή τάσης δεν είναι διαθέσιμη για τις σειρές δεδομένων σε οποιονδήποτε τύπο γραφήματος που είναι στοιβαγμένος ή 3-Δ. Οι TrendLines επίσης δεν είναι διαθέσιμες για ραντάρ, πίτα, επιφάνεια ή δακτυλίδι.

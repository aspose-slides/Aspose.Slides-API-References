## ChartSeries.HasUpDownBars ιδιότητα

Καθορίζει εάν ένα γραφικό τύπου Line ή Stock διαθέτει μπάρες ανοδικής/καθοδικής κίνησης. Αυτή η ιδιότητα δεν αφορά μόνο αυτή τη σειρά αλλά όλες τις σειρές της γονικής ομάδας σειρών — αυτή είναι η προβολή της αντίστοιχης ιδιότητας της ομάδας. Συνεπώς αυτή η ιδιότητα είναι μόνο για ανάγνωση. Χρησιμοποιήστε την ιδιότητα ParentSeriesGroup για πρόσβαση στην γονική ομάδα σειρών. Χρησιμοποιήστε την ιδιότητα ParentSeriesGroup.UpDownBars.HasUpDownBars ανάγνωση/εγγραφή για αλλαγή τιμής. Χρησιμοποιήστε την ιδιότητα ParentSeriesGroup.UpDownBars για μορφοποίηση των μπάρων ανοδικής/καθοδικής κίνησης. Μόνο για ανάγνωση Boolean.

public bool HasUpDownBars { get; }

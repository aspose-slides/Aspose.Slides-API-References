## ChartSeries.PieSplitPosition ιδιότητα

Καθορίζει μια τιμή που θα χρησιμοποιηθεί για τον προσδιορισμό του ποιων σημείων δεδομένων βρίσκονται στη δεύτερη πίτα ή μπάρα σε ένα γράφημα τύπου pie-of-pie ή bar-of-pie. Χρησιμοποιείται μαζί με την ιδιότητα PieSplitBy. Αυτή η ιδιότητα αφορά όχι μόνο αυτή τη σειρά αλλά και όλες τις σειρές της γονικής ομάδας σειρών – πρόκειται για προβολή της αντίστοιχης ιδιότητας της ομάδας. Επομένως αυτή η ιδιότητα είναι μόνο για ανάγνωση. Χρησιμοποιήστε την ιδιότητα ParentSeriesGroup για πρόσβαση στην γονική ομάδα σειρών. Χρησιμοποιήστε την ιδιότητα ParentSeriesGroup.PieSplitPosition για ανάγνωση/εγγραφή προκειμένου να αλλάξετε την τιμή. Μόνο για ανάγνωση Double.

```csharp
public double PieSplitPosition { get; }
```

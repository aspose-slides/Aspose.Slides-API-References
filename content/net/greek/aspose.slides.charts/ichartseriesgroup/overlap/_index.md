## IChartSeriesGroup.Overlap ιδιότητα

Καθορίζει το πόσο οι μπάρες και οι στήλες πρέπει να επικαλύπτονται σε 2-Δ διαγράμματα, ως ποσοστό (από -100% έως 100%). - -100%: Μέγιστη απόσταση (οι μπάρες είναι εντελώς χωρισμένες). - 0%: Οι μπάρες τοποθετούνται πλάι-πλάι χωρίς επικάλυψη ή απόσταση. - 100%: Μέγιστη επικάλυψη (οι μπάρες επικαλύπτονται πλήρως μεταξύ τους). Αυτή η ιδιότητα είναι ανάγνωση/εγγραφή SByte.

```csharp
public sbyte Overlap { get; set; }
```

### Εξαιρέσεις

| εξαίρεση | συνθήκη |
| --- | --- |
| ArgumentOutOfRangeException | Αποβάλλεται όταν η τιμή ορίζεται εκτός του έγκυρου εύρους -100 έως 100. |

### Παραδείγματα

Το παρακάτω παράδειγμα δείχνει πώς να ορίσετε την επικάλυψη για μια ομάδα σειράς διαγράμματος και να αποδώσετε το προκύπτο διάγραμμα σε μια φόρμα:

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // Ορίστε την επικάλυψη στο 55%

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

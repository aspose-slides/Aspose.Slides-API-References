## IChartCategoryCollection.UseCells ιδιότητα

Εάν true τότε το worksheet χρησιμοποιείται για την αποθήκευση των κατηγοριών (αυτή η περίπτωση υποστηρίζει πολυεπίπεδες κατηγορίες). Εάν false τότε το worksheet ΔΕΝ χρησιμοποιείται για την αποθήκευση τιμών (και αυτή η περίπτωση δεν υποστηρίζει πολυεπίπεδες κατηγορίες). Ανάγνωση/εγγραφή Boolean.

public bool UseCells { get; set; }

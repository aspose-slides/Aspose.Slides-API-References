## ChartCategory.UseCell ιδιότητα

Αν είναι true τότε η ιδιότητα AsCell είναι ενεργή. Με άλλα λόγια, το worksheet χρησιμοποιείται για την αποθήκευση της κατηγορίας (αυτή η περίπτωση υποστηρίζει πολυεπίπεδη κατηγορία). Αν είναι false τότε η ιδιότητα AsLiteral είναι ενεργή. Με άλλα λόγια, το worksheet ΔΕΝ χρησιμοποιείται για την αποθήκευση της κατηγορίας (και αυτή η περίπτωση δεν υποστηρίζει πολυεπίπεδες κατηγορίες). Μόνο για ανάγνωση Boolean.

```csharp
public bool UseCell { get; }
```

### Παρατηρήσεις

Για την αλλαγή της τιμής αυτής της ιδιότητας (για όλες τις κατηγορίες στη συλλογή) ορίστε τη νέα τιμή στην ιδιότητα ChartCategoryCollection.UseCells.

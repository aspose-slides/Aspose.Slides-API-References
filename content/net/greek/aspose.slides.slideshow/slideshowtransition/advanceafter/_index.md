## SlideShowTransition.AdvanceAfter ιδιότητα

Αυτό το χαρακτηριστικό καθορίζει εάν η παρουσίαση θα προχωρήσει στην επόμενη διαφάνεια μετά από συγκεκριμένο χρόνο. Ανάγνωση/εγγραφή Boolean.

```csharp
public bool AdvanceAfter { get; set; }
```

### Παραδείγματα

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    // Λάβετε την πρώτη μετάβαση διαφάνειας
    ISlideShowTransition slideTransition = pres.Slides[0].SlideShowTransition;
    
    // Ελέγξτε αν η σημαία Advance Slide After είναι επιλεγμένη
    if (slideTransition.AdvanceAfter)
    {
        // Λάβετε την τιμή του Advance Slide After Time
        uint advanceAfterTime = slideTransition.AdvanceAfterTime;
    }
}
```

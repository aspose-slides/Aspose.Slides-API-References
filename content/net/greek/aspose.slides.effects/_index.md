Περιέχει κλάσεις για εργασία με διάφορα εφέ σε παρουσιάσεις του Microsoft PowerPoint.

## Κλάσεις

| Κλάση | Περιγραφή |
| --- | --- |
| [AlphaBiLevel](./alphabilevel) | Αναπαριστά ένα εφέ Alpha Bi-Level. Οι τιμές Alpha (Opacity) μικρότερες από το όριο μετατρέπονται σε 0 (πλήρως διαφανές) και οι τιμές Alpha μεγαλύτερες ή ίσες με το όριο μετατρέπονται στο 100 % (πλήρως αδιαφανές). |
| [AlphaCeiling](./alphaceiling) | Αναπαριστά ένα εφέ Alpha Ceiling. Οι τιμές Alpha (opacity) μεγαλύτερες του μηδενός μετατρέπονται στο 100 %. Με άλλα λόγια, ό,τι είναι μερικά αδιαφανές γίνεται πλήρως αδιαφανές. |
| [AlphaFloor](./alphafloor) | Αναπαριστά ένα εφέ Alpha Floor. Οι τιμές Alpha (opacity) μικρότερες του 100 % μετατρέπονται σε μηδέν. Με άλλα λόγια, ό,τι είναι μερικά διαφανές γίνεται πλήρως διαφανές. |
| [AlphaInverse](./alphainverse) | Αναπαριστά ένα εφέ Alpha Inverse. Οι τιμές Alpha (opacity) αντιστρέφονται αφαιρώντας τες από το 100 %. |
| [AlphaModulate](./alphamodulate) | Αναπαριστά ένα εφέ Alpha Modulate. Οι τιμές Alpha (opacity) του εφέ πολλαπλασιάζονται με ένα σταθερό ποσοστό. Το περιέκτη εφέ καθορίζει ένα εφέ που περιέχει τιμές άλφα για τη διαμόρφωση. |
| [AlphaModulateFixed](./alphamodulatefixed) | Αναπαριστά ένα εφέ Alpha Modulate Fixed. Οι τιμές Alpha (opacity) του εφέ πολλαπλασιάζονται με ένα σταθερό ποσοστό. |
| [AlphaReplace](./alphareplace) | Αναπαριστά ένα εφέ Alpha Replace. Οι τιμές Alpha (opacity) του εφέ αντικαθίστανται με μια σταθερή τιμή άλφα. |
| [BiLevel](./bilevel) | Αναπαριστά ένα εφέ Bi-Level (μαύρο/άσπρο). Τα χρώματα εισόδου των οποίων η φωτεινότητα είναι μικρότερη από την καθορισμένη τιμή ορίου μετατρέπονται σε μαύρο. Τα χρώματα εισόδου των οποίων η φωτεινότητα είναι μεγαλύτερη ή ίση με την καθορισμένη τιμή ορίζονται σε λευκό. Οι τιμές άλφα του εφέ δεν επηρεάζονται. |
| [Blur](./blur) | Αναπαριστά ένα εφέ Blur που εφαρμόζεται σε ολόκληρο το σχήμα, συμπεριλαμβανομένου του γεμίσματος. Όλα τα κανάλια χρώματος, συμπεριλαμβανομένου του άλφα, επηρεάζονται. |
| [BrightnessContrast](./brightnesscontrast) | Αναπαριστά ένα εφέ BrightnessContrast. Ρυθμίζει τη φωτεινότητα και την αντίθεση. |
| [ColorChange](./colorchange) | Αναπαριστά ένα εφέ Color Change. Οι εμφανίσεις του FromColor αντικαθίστανται με εμφανίσεις του ToColor. |
| [ColorReplace](./colorreplace) | Αναπαριστά ένα εφέ Color Replacement. Όλα τα χρώματα του εφέ μετατρέπονται σε ένα σταθερό χρώμα. Οι τιμές άλφα δεν επηρεάζονται. |
| [Duotone](./duotone) | Αναπαριστά ένα εφέ Duotone. Για κάθε pixel, συνδυάζει τα Color1 και Color2 με γραμμική παρεμβολή για τον καθορισμό του νέου χρώματος του pixel. |
| [EffectFactory](./effectfactory) | Επιτρέπει τη δημιουργία εφέ. |
| [FillOverlay](./filloverlay) | Αναπαριστά ένα εφέ Fill Overlay. Ένα fill overlay μπορεί να χρησιμοποιηθεί για να καθορίσει ένα πρόσθετο γέμισμα για ένα αντικείμενο και να συγχωνεύσει τα δύο γεμίσματα. |
| [Glow](./glow) | Αναπαριστά ένα εφέ Glow, στο οποίο προστίθεται ένα αχνό περίγραμμα χρώματος έξω από τις άκρες του αντικειμένου. |
| [GrayScale](./grayscale) | Αναπαριστά ένα εφέ Gray Scale. Μετατρέπει όλες τις τιμές χρώματος του εφέ σε απόχρωση του γκρι, ανάλογα με τη φωτεινότητά τους. Οι τιμές άλφα (opacity) του εφέ δεν επηρεάζονται. |
| [HSL](./hsl) | Αναπαριστά ένα εφέ Hue/Saturation/Luminance. Η απόχρωση, ο κορεσμός και η φωτεινότητα μπορούν να προσαρμοστούν ανεξάρτητα σε σχέση με την τρέχουσα τιμή τους. |
| [ImageTransformOCollectionEffectiveData](./imagetransformocollectioneffectivedata) | Αμετάβλητο αντικείμενο που αντιπροσωπεύει μια μόνο προς ανάγνωση συλλογή αποτελεσματικών μετασχηματισμών εικόνας. |
| [ImageTransformOperation](./imagetransformoperation) | Αναπαριστά ένα αφηρημένο εφέ μετασχηματισμού εικόνας. |
| [ImageTransformOperationCollection](./imagetransformoperationcollection) | Αναπαριστά μια συλλογή εφέ που εφαρμόζονται σε μια εικόνα. |
| [ImageTransformOperationFactory](./imagetransformoperationfactory) | Επιτρέπει τη δημιουργία λειτουργιών μετασχηματισμού εικόνας. |
| [InnerShadow](./innershadow) | Αναπαριστά ένα εφέ Inner Shadow. |
| [Luminance](./luminance) | Αναπαριστά ένα εφέ Luminance. Η φωτεινότητα μετατοπίζει γραμμικά όλα τα χρώματα πιο κοντά στο λευκό ή το μαύρο. Η αντίθεση κλιμακώνει όλα τα χρώματα ώστε να είναι πιο κοντά ή πιο μακριά μεταξύ τους. |
| [OuterShadow](./outershadow) | Αναπαριστά ένα εφέ Outer Shadow. |
| [PresetShadow](./presetshadow) | Αναπαριστά ένα εφέ Preset Shadow. |
| [Reflection](./reflection) | Αναπαριστά ένα εφέ Reflection. |
| [SoftEdge](./softedge) | Αναπαριστά ένα εφέ soft edge. Οι άκρες του σχήματος θολώνουν, ενώ το γέμισμα δεν επηρεάζεται. |
| [Tint](./tint) | Αναπαριστά ένα εφέ Tint. Μετατοπίζει τις τιμές χρώματος του εφέ προς ή μακριά από την απόχρωση κατά το καθορισμένο ποσό. |

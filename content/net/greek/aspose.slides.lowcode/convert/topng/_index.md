## ToPng(Presentation, string) {#topng}

Μετατρέπει την εισερχόμενη παρουσίαση σε σύνολο εικόνων μορφής PNG. Εάν το όνομα του αρχείου εξόδου δοθεί ως "myPath/myFilename.png", το αποτέλεσμα θα αποθηκευτεί ως σύνολο αρχείων "myPath/myFilename_N.png", όπου N είναι ο αριθμός της διαφάνειας.

```csharp
public static void ToPng(Presentation pres, string outputFileName)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| pres | Presentation | Η εισερχόμενη παρουσίαση. |
| outputFileName | String | Το όνομα του αρχείου εξόδου. |

| ArgumentException |  |

### Παραδείγματα

```csharp

   Convert.ToPng(pres, "presImage.png");
```

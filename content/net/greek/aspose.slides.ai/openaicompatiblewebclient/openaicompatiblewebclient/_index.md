## OpenAICompatibleWebClient(string, string, string, HttpClient) {#constructor_1}

Δημιουργεί μια παρουσία του web client συμβατού με το OpenAI που χρησιμοποιεί ένα εξωτερικά διαχειριζόμενο HttpClient. Το παρεχόμενο HttpClient δεν διαγράφεται από αυτή την παρουσία και παραμένει στην ιδιοκτησία του καλούντος.

```csharp
public OpenAICompatibleWebClient(string model, string apiKey, string baseUrl, HttpClient httpClient)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| model | String | Όνομα μοντέλου που υποστηρίζεται από τον πάροχο LLM. |
| apiKey | String | Κλειδί API (token). |
| baseUrl | String | Βασική διεύθυνση URL του LLM συμβατού με το OpenAI. |
| httpClient | HttpClient | Ένα εξωτερικά διαχειριζόμενο αντικείμενο HttpClient. |

### Εξαιρέσεις

| εξαίρεση | συνθήκη |
| --- | --- |
| ArgumentException | Η τιμή του κλειδιού API δεν μπορεί να είναι null ή κενή. |
| ArgumentException | Η τιμή του μοντέλου κειμένου δεν μπορεί να είναι null ή κενή. |
| ArgumentException | Η τιμή του Base URL δεν μπορεί να είναι null ή κενή. |
| ArgumentNullException | Το HttpClient δεν μπορεί να είναι null. |

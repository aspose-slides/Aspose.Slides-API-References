## OpenAIWebClient(string, string, string, HttpClient) {#constructor_1}

Δημιουργεί μια παρουσία του πελάτη ιστού OpenAI που χρησιμοποιεί ένα εξωτερικά διαχειριζόμενο HttpClient. Το δοθέν HttpClient δεν διαγράφεται από αυτήν την παρουσία και παραμένει στην ιδιοκτησία του καλούντος.

```csharp
public OpenAIWebClient(string model, string apiKey, string organizationId, HttpClient httpClient)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| model | String | Μοντέλο γλώσσας OpenAI. Πιθανές τιμές: - gpt-4o - gpt-4o-mini - o1 - o1-mini - o3 - o3-mini |
| apiKey | String | Κλειδί API του OpenAI |
| organizationId | String | Αναγνωριστικό οργάνωσης (προαιρετικό) |
| httpClient | HttpClient | Μια εξωτερικά διαχειριζόμενη παρουσία `HttpClient` |

### Εξαιρέσεις

| εξαίρεση | συνθήκη |
| --- | --- |
| ArgumentException | Η τιμή του κλειδιού API δεν μπορεί να είναι null ή κενή |
| ArgumentException | Η τιμή του μοντέλου κειμένου δεν μπορεί να είναι null ή κενή |
| ArgumentNullException | Το HttpClient δεν μπορεί να είναι null |

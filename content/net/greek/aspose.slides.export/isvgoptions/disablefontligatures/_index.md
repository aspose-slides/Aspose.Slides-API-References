## ISVGOptions.DisableFontLigatures ιδιότητα

Λαμβάνει ή ορίζει μια τιμή που υποδεικνύει εάν το κείμενο αποδίδεται χωρίς τη χρήση συνδέσμων. Όταν οριστεί σε `true`, οι συνδέσμοι θα απενεργοποιηθούν στην αποδομένη έξοδο. Από προεπιλογή, αυτή η ιδιότητα είναι ορισμένη σε `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    SVGOptions options = new SVFOptions
    {
        DisableFontLigatures = true // Απενεργοποίηση συνδέσμων στην απόδοση κειμένου
    };
    
    using (FileStream fileStream = new FileStream("slide-0.svg", FileMode.Create, FileAccess.Write))
    {
        pres.Slides[0].WriteAsSvg(fileStream);   
    }
}
```

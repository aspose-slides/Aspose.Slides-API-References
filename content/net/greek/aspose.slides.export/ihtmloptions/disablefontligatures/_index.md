## IHtmlOptions.DisableFontLigatures ιδιότητα

Λαμβάνει ή ορίζει μια τιμή που υποδεικνύει εάν το κείμενο αποδίδεται χωρίς χρήση συνόλων. Όταν οριστεί στο `true`, τα συνόλα θα απενεργοποιηθούν στην παραγόμενη έξοδο. Από προεπιλογή, αυτή η ιδιότητα ορίζεται σε `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    HtmlOptions options = new HtmlOptions
    {
        DisableFontLigatures = true // Απενεργοποιήστε τα συνόλα στην απόδοση κειμένου
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html, options);
}
```

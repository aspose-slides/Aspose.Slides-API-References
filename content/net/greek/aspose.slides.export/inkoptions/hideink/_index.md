## InkOptions.HideInk property

Εμφανίζει ή κρύβει τα στοιχεία Ink στο εξαγόμενο έγγραφο.

```csharp
public bool HideInk { get; set; }
```

### Παρατηρήσεις

Η προεπιλεγμένη τιμή είναι false.

### Παραδείγματα

Το επόμενο παράδειγμα δείχνει πώς να κρύψετε τα στοιχεία Ink στο εξαγόμενο έγγραφο PDF:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.HideInk = true;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

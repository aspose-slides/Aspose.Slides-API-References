## MarkdownSaveOptions.SlideNumberFormat ιδιότητα

Λαμβάνει ή ορίζει τη συμβολοσειρά μορφής που χρησιμοποιείται για τις επικεφαλίδες αριθμού διαφάνειας στην έξοδο Markdown. Η μορφή πρέπει να περιλαμβάνει το σύμβολο κράτησης θέσης "{0}", το οποίο θα αντικατασταθεί με τον δείκτη της διαφάνειας κατά την εξαγωγή. Παράδειγμα: "# Slide {0}" θα παράγει "# Slide 1", "# Slide 2", κ.λπ.

```csharp
public string SlideNumberFormat { get; set; }
```

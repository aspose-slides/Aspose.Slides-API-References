## PptxOptions.Zip64Mode ιδιότητα

Καθορίζει αν χρησιμοποιείται η μορφή ZIP64 για το έγγραφο Presentation. Η προεπιλεγμένη τιμή είναι IfNecessary

```csharp
public Zip64Mode Zip64Mode { get; set; }
```

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]

{
    pres.Save("demo-zip64.pptx", SaveFormat.Pptx, new PptxOptions()

}
```

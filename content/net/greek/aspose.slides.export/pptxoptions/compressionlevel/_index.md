## PptxOptions.CompressionLevel ιδιότητα

Καθορίζει το επίπεδο συμπίεσης που χρησιμοποιείται κατά την αποθήκευση του εγγράφου παρουσίασης. Η προεπιλεγμένη τιμή είναι Level6.

```csharp
public CompressionLevel CompressionLevel { get; set; }
```

### Παρατηρήσεις

Τα υψηλότερα επίπεδα συμπίεσης παράγουν μικρότερα αρχεία, αλλά απαιτούν περισσότερο χρόνο επεξεργασίας. Η πραγματική αναλογία συμπίεσης εξαρτάται από το περιεχόμενο της παρουσίασης.

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("demo-level8.pptx", SaveFormat.Pptx, new PptxOptions()
    {
        CompressionLevel = CompressionLevel.Level8
    });
}
```

## IHtml5Options.DisableFontLigatures ιδιότητα

Λαμβάνει ή ορίζει μια τιμή που υποδεικνύει εάν το κείμενο αποδίδεται χωρίς τη χρήση συνδυασμών χαρακτήρων. Όταν οριστεί σε `true`, οι συνδυασμοί χαρακτήρων θα απενεργοποιηθούν στην αποδιδόμενη έξοδο. Από προεπιλογή, αυτή η ιδιότητα είναι ορισμένη σε `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    Html5Options options = new Html5Options
    {
        DisableFontLigatures = true // Απενεργοποίηση συνδυασμών χαρακτήρων στην απόδοση κειμένου
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html5, options);
}
```

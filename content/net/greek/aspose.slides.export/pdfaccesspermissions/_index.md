## PdfAccessPermissions απαρίθμηση

Περιέχει ένα σύνολο σημαιών που καθορίζουν ποιες δικαιοδοσίες πρόσβασης πρέπει να χορηγηθούν όταν το έγγραφο ανοίγει με πρόσβαση χρήστη.

```csharp
[Flags]
public enum PdfAccessPermissions
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| None | `0` | Καθορίζει ότι ένας χρήστης δεν έχει δικαιοδοσίες πρόσβασης. |
| PrintDocument | `4` | Καθορίζει εάν ένας χρήστης μπορεί να εκτυπώσει το έγγραφο (ενδέχεται να μην είναι στο υψηλότερο επίπεδο ποιότητας, ανάλογα με το αν το bit HighQualityPrint είναι επίσης ορισμένο). |
| ModifyContent | `8` | Καθορίζει εάν ένας χρήστης μπορεί να τροποποιήσει το περιεχόμενο του εγγράφου μέσω λειτουργιών διαφορετικών από αυτές που ελέγχονται από τα bits AddOrModifyFields, FillExistingFields, AssembleDocument. |
| CopyTextAndGraphics | `10` | Καθορίζει εάν ένας χρήστης μπορεί να αντιγράψει ή να εξάγει με άλλο τρόπο κείμενο και γραφικά από το έγγραφο μέσω λειτουργιών διαφορετικών από αυτή που ελέγχεται από το bit ExtractTextAndGraphics. |
| AddOrModifyFields | `20` | Καθορίζει εάν ένας χρήστης μπορεί να προσθέσει ή να τροποποιήσει επεξηγήσεις κειμένου, να συμπληρώσει διαδραστικά πεδία φόρμας και, εάν το bit ModifyContent είναι επίσης ορισμένο, να δημιουργήσει ή να τροποποιήσει διαδραστικά πεδία φόρμας (συμπεριλαμβανομένων των πεδίων υπογραφής). |
| FillExistingFields | `100` | Καθορίζει εάν ένας χρήστης μπορεί να συμπληρώσει υπάρχοντα διαδραστικά πεδία φόρμας (συμπεριλαμβανομένων των πεδίων υπογραφής), ακόμη και αν το bit AddOrModifyFields δεν είναι ορισμένο. |
| ExtractTextAndGraphics | `200` | Καθορίζει εάν ένας χρήστης μπορεί να εξάγει κείμενο και γραφικά για υποστήριξη προσβασιμότητας σε χρήστες με αναπηρίες ή για άλλους σκοπούς. |
| AssembleDocument | `400` | Καθορίζει εάν ένας χρήστης μπορεί να συναρμολογήσει το έγγραφο (εισαγωγή, περιστροφή ή διαγραφή σελίδων και δημιουργία σελιδοδεικτών ή μικρογραφιών), ακόμη και αν το bit ModifyContent δεν είναι ορισμένο. |
| HighQualityPrint | `800` | Καθορίζει εάν ένας χρήστης μπορεί να εκτυπώσει το έγγραφο σε μια αναπαράσταση από την οποία μπορεί να παραχθεί πιστό ψηφιακό αντίγραφο του περιεχομένου PDF. Όταν αυτό το bit δεν είναι ορισμένο (και το bit PrintDocument είναι ορισμένο), η εκτύπωση περιορίζεται σε μια χαμηλού επιπέδου αναπαράσταση της εμφάνισης, πιθανόν με μειωμένη ποιότητα. |

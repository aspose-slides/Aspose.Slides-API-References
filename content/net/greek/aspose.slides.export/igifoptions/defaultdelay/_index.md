## IGifOptions.DefaultDelay ιδιότητα

Παίρνει ή ορίζει τον προεπιλεγμένο χρόνο καθυστέρησης [ms]. Αυτή η τιμή θα χρησιμοποιηθεί εάν [`AdvanceAfterTime`](../../../aspose.slides/islideshowtransition/advanceaftertime) δεν έχει οριστεί. Η προεπιλεγμένη τιμή είναι 1000.

```csharp
public int DefaultDelay { get; set; }
```

### Παραδείγματα

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
	pres.Save("pres.gif", SaveFormat.Gif, new GifOptions { DefaultDelay = 2000 });
}
```

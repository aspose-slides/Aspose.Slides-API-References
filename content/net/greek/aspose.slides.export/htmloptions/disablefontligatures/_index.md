## HtmlOptions.DisableFontLigatures ιδιότητα

Λαμβάνει ή ορίζει μια τιμή που υποδεικνύει εάν το κείμενο αποδίδεται χωρίς χρήση λιγκατούρων. Όταν οριστεί σε `true`, οι λιγκατούρες θα απενεργοποιηθούν στην παραγόμενη έξοδο. Από προεπιλογή, αυτή η ιδιότητα είναι ορισμένη σε `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    HtmlOptions options = new HtmlOptions
    {
        DisableFontLigatures = true // Απενεργοποίηση λιγκατούρων στην απόδοση κειμένου
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html, options);
}
```

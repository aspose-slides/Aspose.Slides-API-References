## HtmlOptions.JpegQuality ιδιότητα

Επιστρέφει ή ορίζει μια τιμή που καθορίζει την ποιότητα των εικόνων JPEG μέσα σε έγγραφο PDF. Ανάγνωση/Εγγραφή Byte.

```csharp
public byte JpegQuality { get; set; }
```

### Παρατηρήσεις

Έχει αποτέλεσμα μόνο όταν ένα έγγραφο περιέχει εικόνες JPEG.

Χρησιμοποιήστε αυτήν την ιδιότητα για να λάβετε ή να ορίσετε την ποιότητα των εικόνων μέσα σε ένα έγγραφο κατά την αποθήκευση σε μορφή PDF. Η τιμή μπορεί να κυμαίνεται από 0 έως 100, όπου το 0 σημαίνει τη χειρότερη ποιότητα αλλά μέγιστη συμπίεση και το 100 σημαίνει την καλύτερη ποιότητα αλλά ελάχιστη συμπίεση.

Η προεπιλεγμένη τιμή είναι **95**.

## SVGOptions.DisableLineEndCropping ιδιότητα

Το SVG 1.1 δεν διαθέτει δυνατότητα ορισμού εσωτερικών αποστάσεων για δείκτες. Η μηχανή εγγραφής SVG του Aspose.Slides διαθέτει λύση για αυτό το πρόβλημα: περικόπτει το τέλος της γραμμής με το βέλος, ώστε η γραμμή να μην επικαλύπτει τους δείκτες. Αυτή η επιλογή απενεργοποιεί αυτή τη συμπεριφορά. Ανάγνωση/εγγραφή Boolean.

```csharp
public bool DisableLineEndCropping { get; set; }
```

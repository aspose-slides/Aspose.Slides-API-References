## SaveOptions.SkipJavaScriptLinks ιδιότητα

Καθορίζει εάν πρέπει να παραληφθούν οι υπερσυνδέσεις με κλήσεις JavaScript κατά την αποθήκευση της παρουσίασης. Ανάγνωση/εγγραφή Boolean. Η προεπιλεγμένη τιμή είναι **false**.

```csharp
public bool SkipJavaScriptLinks { get; set; }
```

### Παρατηρήσεις

Όταν αυτή η ιδιότητα έχει τιμή **true**, οι υπερσυνδέσεις με κλήσεις JavaScript θα αγνοούνται κατά την αποθήκευση.

Όταν αυτή η ιδιότητα έχει τιμή **false**, όλες οι υπερσυνδέσεις θα αποθηκευτούν.

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("result_without_JavaScript_links.html", SaveFormat.Html, new HtmlOptions()
    {
        SkipJavaScriptLinks = true
    });
}
```

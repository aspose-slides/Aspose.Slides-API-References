## SaveOptions.DefaultRegularFont ιδιότητα

Επιστρέφει ή ορίζει τη γραμματοσειρά που χρησιμοποιείται όταν δεν βρεθεί η γραμματοσειρά προέλευσης. Ανάγνωση-Εγγραφή String.

```csharp
public string DefaultRegularFont { get; set; }
```

### Παραδείγματα

```csharp
[C#]

using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
	HtmlOptions htmlOpts = new HtmlOptions();
	htmlOpts.DefaultRegularFont = "Arial Black";
	pres.Save(@"SomePresentation-out-ArialBlack.html", SaveFormat.Html, htmlOpts);
	htmlOpts.DefaultRegularFont = "Lucida Console";
	pres.Save(@"Somepresentation-out-LucidaConsole.html", SaveFormat.Html, htmlOpts);

	PdfOptions pdfOpts = new PdfOptions();
	pdfOpts.DefaultRegularFont = "Arial Black";
	pres.Save(@"SomePresentation-out-ArialBlack.pdf", SaveFormat.Pdf, pdfOpts);
}
```

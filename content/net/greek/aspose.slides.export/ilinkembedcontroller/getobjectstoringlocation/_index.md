## ILinkEmbedController.GetObjectStoringLocation μέθοδος

Καθορίζει πού πρέπει να αποθηκευθεί το αντικείμενο. Αυτή η μέθοδος καλείται μία φορά για κάθε αναγνωριστικό αντικειμένου. Δεν υπάρχει εγγύηση ότι δεν θα υπάρξουν δύο αντικείμενα με τα ίδια δεδομένα, semanticName και contentType αλλά με διαφορετικό αναγνωριστικό.

```csharp
public LinkEmbedDecision GetObjectStoringLocation(int id, byte[] entityData, string semanticName, 
    string contentType, string recomendedExtension)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| id | Int32 | Αναγνωριστικό αντικειμένου. Αυτό το αναγνωριστικό είναι μοναδικό για ολόκληρη τη λειτουργία αποθήκευσης. |
| entityData | Byte[] | Δυαδικά δεδομένα αντικειμένου. Αυτή η παράμετρος μπορεί να είναι null, εάν τα δυαδικά δεδομένα του αντικειμένου δεν έχουν παραχθεί ακόμα. |
| semanticName | String | Κάποιο σύντομο κείμενο που περιγράφει τη σημασία του αντικειμένου. Ο ελεγκτής μπορεί να το χρησιμοποιήσει ως μέρος του εξωτερικού ονόματος του αντικειμένου, αλλά είναι στους διαχειριστές να εξασφαλίσουν ότι τα ονόματα θα είναι μοναδικά και θα περιέχουν μόνο τους επιτρεπτούς χαρακτήρες. |
| contentType | String | Τύπος MIME του αντικειμένου. |
| recomendedExtension | String | Επέκταση ονόματος αρχείου, συνιστώμενη για αυτόν τον τύπο MIME. |

## ILinkEmbedController.GetUrl μέθοδος

Επιστρέφει ένα URL σε ένα εξωτερικό αντικείμενο. Αυτή η μέθοδος πάντα καλείται εάν [`GetObjectStoringLocation`](../getobjectstoringlocation) επέστρεψε Link και μπορεί να κληθεί εάν [`GetObjectStoringLocation`](../getobjectstoringlocation) επέστρεψε Embed αλλά η ενσωμάτωση είναι αδύνατη. Μπορεί να κληθεί πολλές φορές για το ίδιο αναγνωριστικό αντικειμένου.

```csharp
public string GetUrl(int id, int referrer)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| id | Int32 | Object id. This id is saving operation-wide unique. |
| referrer | Int32 | id of referrencing object or 0, if object is referrenced by the root document. May be used to generate relative link. |

### Τιμή Επιστροφής

URL του εξωτερικού αντικειμένου ή null εάν αυτό το αντικείμενο πρέπει να αγνοηθεί.

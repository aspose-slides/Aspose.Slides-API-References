## PdfOptions.EmbedTrueTypeFontsForASCII ιδιότητα

Καθορίζει αν το Aspose.Slides θα ενσωματώσει κοινές γραμματοσειρές για κείμενο ASCII (διάστημα κωδίκων 33..127). Οι γραμματοσειρές για κωδικούς χαρακτήρων μεγαλύτερους του 127 ενσωματώνονται πάντα. Η λίστα κοινών γραμματοσειρών περιλαμβάνει τις βασικές 14 γραμματοσειρές του PDF και επιπλέον γραμματοσειρές που καθορίζονται από τον χρήστη. Boolean ανάγνωση/εγγραφή.

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### Παρατηρήσεις

Η προεπιλογή είναι **true**.

## PdfOptions.JpegQuality ιδιότητα

Επιστρέφει ή ορίζει μια τιμή που καθορίζει την ποιότητα των εικόνων JPEG εντός εγγράφου PDF. Ανάγνωση/εγγραφή Byte.

```csharp
public byte JpegQuality { get; set; }
```

### Σχόλια

Έχει αποτέλεσμα μόνο όταν ένα έγγραφο περιέχει εικόνες JPEG.

Χρησιμοποιήστε αυτήν την ιδιότητα για να αποκτήσετε ή να ορίσετε την ποιότητα των εικόνων μέσα σε ένα έγγραφο κατά την αποθήκευση σε μορφή PDF. Η τιμή μπορεί να κυμαίνεται από 0 έως 100, όπου το 0 σημαίνει χαμηλότερη ποιότητα αλλά μέγιστη συμπίεση και το 100 σημαίνει υψηλότερη ποιότητα αλλά ελάχιστη συμπίεση.

Η προεπιλεγμένη τιμή είναι **100**.

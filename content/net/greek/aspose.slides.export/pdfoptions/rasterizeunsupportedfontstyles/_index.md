## PdfOptions.RasterizeUnsupportedFontStyles ιδιότητα

Δείχνει αν το κείμενο πρέπει να rasterize ως bitmap και να σωθεί στο PDF όταν η γραμματοσειρά δεν υποστηρίζει έντονη μορφοποίηση. Αυτή η προσέγγιση μπορεί να βελτιώσει την ποιότητα του κειμένου στο προκύπτον PDF για ορισμένες γραμματοσειρές. Ανάγνωση/εγγραφή Boolean.

```csharp
public bool RasterizeUnsupportedFontStyles { get; set; }
```

### Παρατηρήσεις

Η προεπιλογή είναι **false**.

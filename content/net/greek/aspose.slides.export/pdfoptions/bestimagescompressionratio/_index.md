## PdfOptions.BestImagesCompressionRatio ιδιότητα

Υποδεικνύει εάν πρέπει να επιλεγεί αυτόματα η πιο αποτελεσματική συμπίεση (αντί για τη προεπιλεγμένη) για κάθε εικόνα. Εάν οριστεί σε Boolean.true, για κάθε εικόνα στην παρουσίαση θα επιλεγεί ο πιο κατάλληλος αλγόριθμος συμπίεσης, κάτι που θα οδηγήσει σε μικρότερο μέγεθος του τελικού εγγράφου PDF. Η επιλογή του βέλτιστου λόγου συμπίεσης εικόνας είναι υπολογιστικά δαπανηρή και απαιτεί πρόσθετη μνήμη RAM, και αυτή η επιλογή είναι Boolean.false εξ ορισμού.

```csharp
public bool BestImagesCompressionRatio { get; set; }
```

### Παρατηρήσεις

Η προεπιλογή είναι Boolean.false.

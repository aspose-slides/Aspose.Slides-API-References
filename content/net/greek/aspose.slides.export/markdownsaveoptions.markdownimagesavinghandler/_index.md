## MarkdownSaveOptions.MarkdownImageSavingHandler delegate

Καλείται για κάθε εικόνα μη-SVG (bitmap ή metafile) κατά την εξαγωγή σε Markdown. Επιστρέψτε `true` για να χρησιμοποιήσετε το καθορισμένο *link*, ή `false` για να εφαρμόσετε την προεπιλεγμένη λογική αποθήκευσης.

```csharp
public delegate bool MarkdownImageSavingHandler(IImage image, ImageFormat format, ref string link);
```

| Parameter | Type | Description |
| --- | --- | --- |
| image | IImage | Η εικόνα που εξάγεται (bitmap ή metafile). |
| format | ImageFormat | Η μορφή της εικόνας. |
| link | String& | Ο σύνδεσμος Markdown που θα χρησιμοποιηθεί όταν επιστρέφεται `true`. |

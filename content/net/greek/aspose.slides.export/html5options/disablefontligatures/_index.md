## Html5Options.DisableFontLigatures ιδιότητα

Λαμβάνει ή ορίζει μια τιμή που υποδεικνύει εάν το κείμενο αποδίδεται χωρίς τη χρήση συνόρων. Όταν οριστεί σε `true`, τα σύναψη θα απενεργοποιηθούν στο παραγόμενο αποτέλεσμα. Από προεπιλογή, αυτή η ιδιότητα είναι ορισμένη σε `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    Html5Options options = new Html5Options
    {
        DisableFontLigatures = true // Απενεργοποιεί τις συνάψεις στην απόδοση κειμένου
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html5, options);
}
```

## SwfOptions.SlidesLayoutOptions ιδιότητα

Αποκτά ή ορίζει τη λειτουργία με την οποία οι διαφάνειες τοποθετούνται στη σελίδα κατά την εξαγωγή παρουσίασης [`ISlidesLayoutOptions`](../../islideslayoutoptions). Αυτή η ιδιότητα δεν υποστηρίζει την εκχώρηση αντικειμένων τύπου [`HandoutLayoutingOptions`](../../handoutlayoutingoptions)

```csharp
public ISlidesLayoutOptions SlidesLayoutOptions { get; set; }
```

### Παραδείγματα

Παράδειγμα:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    SwfOptions options = new SwfOptions
    {
        SlidesLayoutOptions = new NotesCommentsLayoutingOptions
        {
            CommentsPosition = CommentsPositions.Right
        }
    };
    
    pres.Save("pres.swf", SaveFormat.Swf, options);
}
```

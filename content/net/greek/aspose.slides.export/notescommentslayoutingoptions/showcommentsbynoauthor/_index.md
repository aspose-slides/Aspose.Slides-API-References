## NotesCommentsLayoutingOptions.ShowCommentsByNoAuthor ιδιότητα

Λαμβάνει ή ορίζει την ορατότητα των σχολίων που δεν έχουν συγγραφέα. Εάν είναι true, τα σχόλια θα εμφανιστούν. (Ισχύει μόνο εάν εμφανίζονται τα σχόλια).

```csharp
public bool ShowCommentsByNoAuthor { get; set; }
```

### Παρατηρήσεις

Η προεπιλεγμένη τιμή είναι **false**.

## ITiffOptions.BwConversionMode ιδιότητα

Καθορίζει τον αλγόριθμο για τη μετατροπή μιας έγχρωμης εικόνας σε ασπρόμαυρη εικόνα. Αυτή η επιλογή θα εφαρμοστεί μόνο εάν [`CompressionType`](../compressiontype) έχει οριστεί σε CCITT4 ή CCITT3 Ανάγνωση/Γραφή [`BlackWhiteConversionMode`](../../blackwhiteconversionmode). Η προεπιλογή είναι Default.

```csharp
public BlackWhiteConversionMode BwConversionMode { get; set; }
```

### Παραδείγματα

```csharp
[C#]
TiffOptions tiffOptions = new TiffOptions();
tiffOptions.CompressionType = TiffCompressionTypes.CCITT4;
tiffOptions.BwConversionMode = BlackWhiteConversionMode.Dithering;

using (var presentation = new Presentation())
{
    presentation.Save(tiffFilePath, SaveFormat.Tiff, tiffOptions);
}
```

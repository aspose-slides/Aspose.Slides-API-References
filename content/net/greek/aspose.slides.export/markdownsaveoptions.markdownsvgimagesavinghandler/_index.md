## MarkdownSaveOptions.MarkdownSvgImageSavingHandler delegate

Καλείται για κάθε εικόνα SVG κατά την εξαγωγή Markdown. Επιστρέψτε `true` για να χρησιμοποιήσετε το καθορισμένο *σύνδεσμος*, ή `false` για να εφαρμόσετε την προεπιλεγμένη λογική αποθήκευσης.

```csharp
public delegate bool MarkdownSvgImageSavingHandler(ISvgImage svgImage, ref string link);
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| svgImage | ISvgImage | Η εικόνα SVG που εξάγεται. |
| link | String& | Ο σύνδεσμος Markdown που θα χρησιμοποιηθεί όταν επιστρέφεται `true`. |

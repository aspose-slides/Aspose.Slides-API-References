## IInkOptions.InterpretMaskOpAsOpacity ιδιότητα

Χρησιμοποιεί λειτουργία ROP ή Opacity για την απόδοση του πινέλου.

```csharp
public bool InterpretMaskOpAsOpacity { get; set; }
```

### Σχόλια

Η προεπιλεγμένη τιμή είναι true.

### Παραδείγματα

Το παρακάτω παράδειγμα δείχνει πώς να ορίσετε τη χρήση του ROP για εξαγωγή στοιχείων Ink:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.InterpretMaskOpAsOpacity = false;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

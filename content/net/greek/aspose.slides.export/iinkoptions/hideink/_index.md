## IInkOptions.HideInk ιδιότητα

Εμφανίζει ή αποκρύπτει στοιχεία Ink σε εξαχθέν έγγραφο.

```csharp
public bool HideInk { get; set; }
```

### Σχόλια

Η προεπιλεγμένη τιμή είναι false.

### Παραδείγματα

Το παρακάτω παράδειγμα δείχνει πώς να αποκρύψετε στοιχεία Ink σε εξαχθέν έγγραφο PDF:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.HideInk = true;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

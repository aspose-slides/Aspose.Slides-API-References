## IPdfOptions.SaveMetafilesAsPng ιδιότητα

True για τη μετατροπή όλων των μεταφαίρων που χρησιμοποιούνται σε μία παρουσίαση σε εικόνες PNG. Ανάγνωση/εγγραφή Boolean.

```csharp
public bool SaveMetafilesAsPng { get; set; }
```

### Σχόλια

Η προεπιλογή είναι **true**. Το έγγραφο Pdf μπορεί να περιέχει διανυσματικά γραφικά και ράστερ εικόνες. Εάν το SaveMetafilesAsPng είναι ορισμένο σε true, τότε η εικόνα Metafile μετατρέπεται σε μορφή Png και αποθηκεύεται στο Pdf ως ράστερ εικόνα. Εάν το SaveMetafilesAsPng είναι ορισμένο σε false, τότε το Metafile μετατρέπεται σε διανυσματικά γραφικά Pdf. Κάθε προσέγγιση έχει πλεονεκτήματα και μειονεκτήματα. Για παράδειγμα, εάν το Metafile μετατραπεί σε PNG, τότε μπορεί να υπάρξει κάποια απώλεια ποιότητας κατά την κλιμάκωση του αποτελικού εγγράφου. Εάν το Metafile μετατραπεί σε διανυσματικά γραφικά Pdf, τότε ενδέχεται να προκύψουν προβλήματα απόδοσης στο εργαλείο προβολής Pdf.

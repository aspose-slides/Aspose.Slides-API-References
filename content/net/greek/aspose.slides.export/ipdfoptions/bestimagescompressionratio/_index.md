## IPdfOptions.BestImagesCompressionRatio ιδιότητα

Υποδεικνύει εάν η πιο αποδοτική συμπίεση (αντί της προεπιλεγμένης) για κάθε εικόνα πρέπει να επιλέγεται αυτόματα. Εάν οριστεί σε Boolean.true, για κάθε εικόνα στην παρουσίαση θα επιλεγεί ο πιο κατάλληλος αλγόριθμος συμπίεσης, κάτι που θα οδηγήσει σε μικρότερο μέγεθος του τελικού εγγράφου PDF. Η επιλογή του βέλτιστου λόγου συμπίεσης εικόνας είναι υπολογιστικά απαιτητική και καταναλώνει πρόσθετη μνήμη RAM, και αυτή η επιλογή είναι Boolean.false από προεπιλογή.

```csharp
public bool BestImagesCompressionRatio { get; set; }
```

### Παρατηρήσεις

Η προεπιλογή είναι Boolean.false.

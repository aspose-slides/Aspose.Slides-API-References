## ToJpeg(Presentation, string) {#tojpeg}

將輸入的 Presentation 轉換為一組 JPEG 格式的圖像。若將輸出檔案名稱指定為「myPath/myFilename.jpeg」，結果將儲存為一組「myPath/myFilename_N.jpeg」檔案，其中 N 為投影片編號。

```csharp
public static void ToJpeg(Presentation pres, string outputFileName)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| pres | Presentation | 輸入的 Presentation。 |
| outputFileName | String | 輸出檔案名稱。 |

### 例外

| 例外 | 條件 |
| --- | --- |
| ArgumentException |  |

### 範例

```csharp
using (var pres = new Presentation("pres.pptx"))
    Convert.ToJpeg(pres, "presImage.jpeg");
```

## ToPng(Presentation, string) {#topng}

將輸入的簡報轉換為一組 PNG 格式的影像。若將輸出檔案名稱指定為 "myPath/myFilename.png"，結果將儲存為一組 "myPath/myFilename_N.png" 檔案，其中 N 為投影片編號。

```csharp
public static void ToPng(Presentation pres, string outputFileName)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| pres | Presentation | 輸入的簡報。 |
| outputFileName | String | 輸出檔案名稱。 |

### 例外狀況

| 例外 | 條件 |
| --- | --- |
| ArgumentException |  |

### 範例

```csharp
using (var pres = new Presentation("pres.pptx"))
   Convert.ToPng(pres, "presImage.png");
```

## IBlobManagementOptions.PresentationLockingBehavior 屬性

此屬性定義 Presentation 類別的實例在其生命週期內是否可以成為來源（檔案或資料流）的擁有者。若實例為擁有者，則會鎖定來源。此舉有助於在處理 BLOB 時改善記憶體使用量與效能，但在 Presentation 實例的生命週期內，來源（資料流或檔案）無法變更。以下是一個範例：

```csharp
public PresentationLockingBehavior PresentationLockingBehavior { get; set; }
```

### 範例

```csharp
LoadOptions loadOptions = new LoadOptions { 
  BlobManagementOptions = {PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked} };
using (Presentation pres = new Presentation("pres.pptx", loadOptions)) {
  // 會拋出 IOException，因為 pres.pptx 在 Presentation 的生命週期內被鎖定
  // File.Delete("pres.pptx");
}
// 在 Presentation 物件釋放後，檔案會解鎖並可被刪除
File.Delete("pres.pptx");
```

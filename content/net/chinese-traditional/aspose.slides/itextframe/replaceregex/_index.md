## ITextFrame.ReplaceRegex 方法

將正則表達式的所有匹配項替換為指定的字串。

```csharp
public void ReplaceRegex(Regex regex, string newText, IFindResultCallback callback)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| regex | Regex | 用於取得要替換之字串的正則表達式。 |
| newText | String | 用於取代所有要被替換之字串的字串。 |
| callback | IFindResultCallback | 接收搜尋結果的回呼物件 [`IFindResultCallback`](../../ifindresultcallback)。 |

### 範例

以下程式碼示範如何使用正則表達式將文字替換為指定的字串。

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// 將所有 10 個字元或以上的單詞替換為 '***'
	((AutoShape)presentation.Slides[0].Shapes[0]).TextFrame.ReplaceRegex(regex, "***", null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

## IFonts.SetScriptFont 方法

將字型名稱指派給特定的腳本標籤，該標籤定義了此腳本的文字在簡報中的呈現方式。

```csharp
public void SetScriptFont(string script, string fontName)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| script | String | BCP-47 腳本代碼（例如 "Arab", "Hebr", "Hans"）用於識別書寫系統。 |
| fontName | String | 要指派給指定腳本的字型名稱。 |

### 範例

此範例說明如何將阿拉伯語腳本的字型設定為 "Segoe UI"：

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segoe UI");
```

## IFonts.GetScriptFont 方法

取得與簡報主題中特定腳本標記相關聯的字型名稱。

```csharp
public string GetScriptFont(string script)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| script | String | 用於識別書寫系統的 BCP-47 腳本代碼（例如 "Latn"、"Cyrl"、"Jpan"）。 |

### 返回值

指定腳本使用的字型名稱；如果未定義該腳本，則為 `null`。

### 範例

此範例示範如何從簡報主題中取得指派給西里爾字母腳本的字型。

```csharp
[C#]
string font = presentation.MasterTheme.FontScheme.Major.GetScriptFont("Cyrl");
Console.WriteLine("Font for Cyrillic script: " + font);
```

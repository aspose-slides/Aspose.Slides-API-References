## SummaryZoomSectionCollection.AddSummaryZoomSection 方法

建立新的 Summary Zoom Section 物件並將其加入集合

```csharp
public ISummaryZoomSection AddSummaryZoomSection(ISection section)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| section | ISection | 用於新 Summary Zoom Section 元素的 Section [`ISection`](../../isection) |

### 返回值

已加入 [`ISummaryZoomFrame`](../../isummaryzoomframe) 元素

### 例外狀況

| 例外 | 條件 |
| --- | --- |
| ArgumentException | 參考的 section 不屬於目前的簡報或不包含任何投影片。 |

### 備註

如果此 section 已有對應的元素存在於集合中，將返回現有的元素。

### 範例

此範例示範如何依索引取得 Summary Zoom Section 元素：

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    ISummaryZoomFrame zoomFrame = (ISummaryZoomFrame) pres.Slides[1].Shapes[0];
    ISummaryZoomSectionCollection collection = zoomFrame.SummaryZoomCollection;
    ISummaryZoomSection newZoomSection = collection.AddSummaryZoomSection(pres.Sections[3]);
}
```

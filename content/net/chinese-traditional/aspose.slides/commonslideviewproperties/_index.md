### 範例

以下範例說明如何設定 PowerPoint 簡報投影片的縮放值。

```csharp
[C#]
// 實例化一個表示簡報檔案的 Presentation 物件
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // 設定簡報的檢視屬性
    presentation.ViewProperties.SlideViewProperties.Scale = 100; // 投影片檢視的縮放值（百分比）
	presentation.ViewProperties.NotesViewProperties.Scale = 100; // 備註檢視的縮放值（百分比）
    presentation.Save("Zoom_out.pptx", SaveFormat.Pptx);
}
```

## AddClone(ILayoutSlide) {#addclone}

將指定版面投影片的副本新增至簡報。

```csharp
public ILayoutSlide AddClone(ILayoutSlide sourceLayout)
```

| 參數 | 類型 | 描述 |
| --- | --- | --- |
| sourceLayout | ILayoutSlide | 要克隆的投影片。 |

### Return Value

已新增的投影片。

### Remarks

在不同簡報之間克隆版面時，版面的母片也可能被克隆，以保留來源的格式。內部註冊表用於追蹤自動克隆的母片，以防止同一母片投影片產生多個克隆。手動克隆母片投影片既不會被阻止，也不會被註冊。

## AddClone(ILayoutSlide, IMasterSlide) {#addclone_1}

將指定版面投影片的副本新增至簡報。

```csharp
public ILayoutSlide AddClone(ILayoutSlide sourceLayout, IMasterSlide destMaster)
```

| 參數 | 類型 | 描述 |
| --- | --- | --- |
| sourceLayout | ILayoutSlide | 要克隆的投影片。 |
| destMaster | IMasterSlide | 新版面的母片投影片。 |

### Return Value

已新增的投影片。

### Remarks

新版面將與目標簡報中指定的母片連結。因此這相當於在 PowerPoint 中使用「使用目標主題」選項的複製/貼上。

## IGlobalLayoutSlideCollection.Add 方法

將新的版面投影片新增至簡報。

```csharp
public ILayoutSlide Add(IMasterSlide master, SlideLayoutType layoutType, string layoutName)
```

| 參數 | 型別 | 說明 |
| --- | --- | --- |
| master | IMasterSlide | 新版面配置的母投影片。 |
| layoutType | SlideLayoutType | 支援的版面配置類型：Title, TitleOnly, Blank, TitleAndObject, VerticalText, VerticalTitleAndText, TwoObjects, SectionHeader, TwoTextAndTwoObjects, TitleObjectAndCaption, PictureAndCaption, Custom。其他版面配置目前不支援：Text, TwoColumnText, Table, TextAndChart, ChartAndText, Diagram, Chart, TextAndClipArt, ClipArtAndText, TextAndObject, ObjectAndText, Object, TextAndMedia, MediaAndText, ObjectOverText, TextOverObject, TextAndTwoObjects, TwoObjectsAndText, TwoObjectsOverText, FourObjects, ClipArtAndVerticalText, VerticalTitleAndTextOverChart, ObjectAndTwoObject, TwoObjectsAndObject。 |
| layoutName | String | 新版面配置的名稱。若傳入的名稱已被使用，將拋出 ArgumentException。若傳入 null 參數，則根據傳入的版面類型自動產生名稱（例如「Title Slide」或「1_Title Slide」、「2_..」等）。 |

### 傳回值

已新增的投影片。

### 例外狀況

| 例外 | 條件 |
| --- | --- |
| NotImplementedException | 如果傳入 *layoutType* 參數的不支援值，則拋出。目前不支援的版面類型：Text, TwoColumnText, Table, TextAndChart, ChartAndText, Diagram, Chart, TextAndClipArt, ClipArtAndText, TextAndObject, ObjectAndText, Object, TextAndMedia, MediaAndText, ObjectOverText, TextOverObject, TextAndTwoObjects, TwoObjectsAndText, TwoObjectsOverText, FourObjects, ClipArtAndVerticalText, VerticalTitleAndTextOverChart, ObjectAndTwoObject, TwoObjectsAndObject。 |
| ArgumentNullException | 如果 *master* 為 null，則拋出。 |
| ArgumentException | 如果 *master* 屬於其他簡報，則拋出。 |
| ArgumentException | 如果 *layoutName* 在 *master* 的版面集合中已被使用，則拋出。 |

### 備註

1) 為 *layoutType* 的 SlideLayoutType.Custom 值新增的版面配置不包含佔位符和圖形。 2) 此方法的對應方法是 [`Add`](../../imasterlayoutslidecollection/add)，透過 [`LayoutSlides`](../../imasterslide/layoutslides) 屬性存取。

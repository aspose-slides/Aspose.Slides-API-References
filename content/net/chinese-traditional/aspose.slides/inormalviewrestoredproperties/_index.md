## INormalViewRestoredProperties 介面

指定普通檢視中投影片區域的尺寸（作為 restoredTop 的子項時為寬度，作為 restoredLeft 的子項時為高度），當區域為可變的還原尺寸（既非最小化亦非最大化）時。

```csharp
public interface INormalViewRestoredProperties
```

| 名稱 | 說明 |
| --- | --- |
| [AutoAdjust](../../aspose.slides/inormalviewrestoredproperties/autoadjust) { get; set; } | 指定在重新調整包含視圖的視窗大小時，側邊內容區域的大小是否應補償新的尺寸。讀寫 Boolean。 |
| [DimensionSize](../../aspose.slides/inormalviewrestoredproperties/dimensionsize) { get; set; } | 指定投影片區域的大小（作為 RestoredTop 的子項時為寬度，作為 RestoredLeft 的子項時為高度）。讀寫 Single。當值小於 0 或大於 100 時拋出例外。 |

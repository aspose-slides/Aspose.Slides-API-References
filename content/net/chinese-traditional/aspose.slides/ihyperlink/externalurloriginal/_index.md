## IHyperlink.ExternalUrlOriginal 屬性

表示設定於此區段的超連結，而不考慮該區段的實際內容。PowerPoint 會針對區段中的連結及其相應文字有特別的行為。它允許以有效的 URL 形式建立超連結的文字，該文字可與連結的實際位址不同。在此情況下，當您在編輯視窗中檢視連結時，連結會被更改以符合文字區段。此屬性表示超連結的原始值。

```csharp
public string ExternalUrlOriginal { get; }
```

## SetLicense(string) {#setlicense_1}

授權此元件。

```csharp
public void SetLicense(string licenseName)
```

| 參數 | 類型 | 描述 |
| --- | --- | --- |
| licenseName | String | 可以是完整或簡短的檔案名稱，或嵌入式資源的名稱。使用空字串可切換為評估模式。 |

### Remarks

嘗試在以下位置尋找授權：

1. 明確路徑。

2. 元件組件所在的資料夾。

3. 客戶端呼叫組件所在的資料夾。

4. 入口組件所在的資料夾。

5. 客戶端呼叫組件中的嵌入式資源。

**注意:** 在 .NET Compact Framework 上，只在以下位置尋找授權：

1. 明確路徑。

2. 客戶端呼叫組件中的嵌入式資源。

### Examples

在此範例中，系統會嘗試在包含元件的資料夾、呼叫組件的資料夾、入口組件的資料夾，以及呼叫組件的嵌入式資源中，尋找名為 MyLicense.lic 的授權檔案。

```csharp
[C#]

License license = new License();
license.SetLicense("MyLicense.lic");


[Visual Basic]

Dim license As License = New License
license.SetLicense("MyLicense.lic")
```

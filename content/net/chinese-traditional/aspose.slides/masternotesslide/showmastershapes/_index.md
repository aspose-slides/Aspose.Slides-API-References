## MasterNotesSlide.ShowMasterShapes property

指定是否在投影片上顯示母版投影片上的圖形。若為母版投影片本身，此屬性始終傳回 `false`。可讀寫 Boolean。

```csharp
public override bool ShowMasterShapes { get; set; }
```

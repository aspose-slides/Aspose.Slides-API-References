## TextFrame.SplitTextByColumns 方法

將 [`ITextFrame`](../../itextframe) 的文字內容拆分為字串陣列，每個元素對應框架內的單獨文字欄位。

```csharp
public string[] SplitTextByColumns()
```

### 返回值

字串陣列，其中每個字串代表 [`ITextFrame`](../../itextframe) 中特定欄位的文字內容。

### 備註

如果文字框不含多個欄位，返回的陣列將僅有一個元素，包含完整文字。空的欄位將以空字串表示於陣列中。

### 範例

以下範例示範如何使用 `SplitTextByColumns`：

```csharp
using (Presentation pres = new Presentation("example.pptx"))
{
    // 取得投影片上的第一個圖形並將其轉型為 ITextFrame
    ITextFrame textFrame = pres.Slides[0].Shapes[0] as ITextFrame;
    // 將文字框的內容拆分為多個欄位
    string[] columnsText = textFrame.SplitTextByColumns();
    // 將每個欄位的文字印出到主控台
    foreach (string column in columnsText)
        Console.WriteLine(column);
}
```

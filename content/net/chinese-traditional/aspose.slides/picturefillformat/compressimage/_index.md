## CompressImage(bool, PicturesCompression) {#compressimage}

根據形狀大小和指定的解析度壓縮圖像以減少其大小。可選地，它還會刪除裁剪區域。

```csharp
public bool CompressImage(bool deleteCroppedAreasOfImage, PicturesCompression resolution)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| deleteCroppedAreasOfImage | Boolean | 若為 true，則方法會移除圖像的裁剪區域，可能會進一步減少其大小。 |
| resolution | PicturesCompression | 壓縮的目標解析度，以 [`PicturesCompression`](../../../aspose.slides.export/picturescompression) enum 的值指定。 |

### 回傳值

布林值，表示圖像是否成功壓縮。若圖像已重新調整大小或被裁剪則回傳 `true`，否則回傳 `false`。

### 例外

| 例外 | 條件 |
| --- | --- |
| ArgumentException | 當解析度不是有效值時拋出。 |

### 備註

此方法會變更圖像的大小和解析度，類似於 PowerPoint 的「圖片格式 → 壓縮圖片」功能。

### 範例

以下範例示範如何使用 `CompressImage` 方法，透過設定目標解析度與移除裁剪區域來減少簡報中圖像的大小：

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;
    // 使用目標解析度 150 DPI（網路解析度）壓縮圖像並移除裁剪區域
    bool result = picFrame.PictureFormat.CompressImage(true, PicturesCompression.Dpi150);
}
```

## PortionCollection.CopyTo 方法

將 ICollection 的元素複製到 Array，從特定的 Array 索引開始。

```csharp
public void CopyTo(IPortion[] array, int arrayIndex)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| array | IPortion[] | 一維 Array，作為從 ICollection 複製的元素的目的地。Array 必須使用零基礎索引。 |
| arrayIndex | Int32 | 在 *array* 中開始複製的零基礎索引。 |

### 例外狀況

| 例外 | 條件 |
| --- | --- |
| ArgumentNullException | *array* 為 null。 |
| ArgumentOutOfRangeException | *arrayIndex* 小於 0。 |
| ArgumentException | 來源 ICollection 中的元素數量大於從 *arrayIndex* 到目的地 *array* 結尾的可用空間。 |

## SetLicense(string) {#setlicense_1}

為元件授權。

```csharp
public void SetLicense(string licenseName)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| licenseName | String | 可以是完整或簡短的檔案名稱，或嵌入式資源的名稱。使用空字串可切換為評估模式。 |

### 備註

嘗試在以下位置尋找授權：

1. 明確路徑。

2. 元件組件的資料夾。

3. 用戶端呼叫組件的資料夾。

4. 入口組件的資料夾。

5. 用戶端呼叫組件中的嵌入式資源。

**注意：** 在 .NET Compact Framework 上，只會在以下位置尋找授權：

1. 明確路徑。

2. 用戶端呼叫組件中的嵌入式資源。

### 範例

在此範例中，將嘗試在包含元件的資料夾、包含呼叫組件的資料夾、入口組件的資料夾，以及呼叫組件的嵌入式資源中尋找名為 MyLicense.lic 的授權檔案。

```csharp
[C#]

License license = new License();
license.SetLicense("MyLicense.lic");


[Visual Basic]

Dim license As License = New License
license.SetLicense("MyLicense.lic")
```

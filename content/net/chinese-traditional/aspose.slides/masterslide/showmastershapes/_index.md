## MasterSlide.ShowMasterShapes 屬性

指定是否在投影片上顯示母片上的形狀。對於母片本身，此屬性總是傳回 `false`。可讀寫 Boolean。

```csharp
public override bool ShowMasterShapes { get; set; }
```

### 例外狀況

| 例外 | 條件 |
| --- | --- |
| NotSupportedException | 如果為母片設定 `true`，則拋出此例外。 |

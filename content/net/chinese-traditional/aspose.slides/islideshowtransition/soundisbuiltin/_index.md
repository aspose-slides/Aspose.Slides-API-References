## ISlideShowTransition.SoundIsBuiltIn 屬性

指定此聲音是否為內建聲音。如果此屬性設為 true，則產生此文件的應用程式會被提醒檢查此聲音在內建聲音清單中指定的 name 屬性，並可依需要顯示自訂名稱或 UI。讀寫 Boolean。

```csharp
public bool SoundIsBuiltIn { get; set; }
```

## ISlideShowTransition.SoundName property

指定一個人類可讀的名稱，用於轉場的音效。[`Sound`](../sound) 屬性必須被指派以取得或設定音效名稱。可讀寫 String.

```csharp
public string SoundName { get; set; }
```

## ZoomObject.ReturnToParent 屬性

取得或設定幻燈片放映中的導航行為。讀寫 Boolean。預設值：false

```csharp
public bool ReturnToParent { get; set; }
```

### 備註

屬性的 true 值表示在幻燈片放映中返回父項的導航行為。

### 範例

範例：


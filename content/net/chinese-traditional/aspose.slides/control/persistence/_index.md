## Control.Persistence 屬性

取得用於儲存 ActiveX 控制項屬性的方法。唯讀 [`PersistenceType`](../../persistencetype).

```csharp
public PersistenceType Persistence { get; }
```

### 範例

以下範例示範如何使用 Persistence 屬性來檢查 ActiveX 物件的屬性是否可以以 XML 基礎的 ActiveX 屬性進行變更：

```csharp
[C#]
if (control.Persistence == PersistenceType.PersistPropertyBag)
{
    control.Properties["Value"] = value;
}
else
{
    YourMethodHere(control.ActiveXControlBinary); //使用您自己的方法來管理儲存在其二進位檔案中的 ActiveX 屬性
}
```

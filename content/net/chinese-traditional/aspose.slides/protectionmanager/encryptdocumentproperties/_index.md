## ProtectionManager.EncryptDocumentProperties 屬性

此屬性在簡報受密碼保護時才有意義。若為 true，則文件屬性會在簡報檔案中加密。若為 false，則文件屬性為公開，而簡報則被加密。可讀寫 Boolean.

```csharp
public bool EncryptDocumentProperties { get; set; }
```

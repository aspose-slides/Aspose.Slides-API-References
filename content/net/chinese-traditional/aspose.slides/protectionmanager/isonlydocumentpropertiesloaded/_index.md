## ProtectionManager.IsOnlyDocumentPropertiesLoaded 屬性

若投影片檔案受密碼保護且該檔案的文件屬性為公開，則此屬性有意義。值為 true 表示僅從加密的投影片檔案中載入文件屬性，而不使用密碼。值為 false 表示使用正確的密碼載入整個加密的投影片，而不僅載入文件屬性。若投影片未加密，則屬性值始終為 false。若加密檔案的文件屬性不是公開的，則屬性值始終為 false。若 Presentation.EncryptDocumentProperties 為 true，則 IsOnlyDocumentPropertiesLoaded 屬性值始終為 false。唯讀 Boolean.

```csharp
public bool IsOnlyDocumentPropertiesLoaded { get; }
```

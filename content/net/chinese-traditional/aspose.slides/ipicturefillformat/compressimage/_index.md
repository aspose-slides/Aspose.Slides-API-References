## CompressImage(bool, PicturesCompression) {#compressimage}

根據形狀大小和指定的解析度縮小圖像的尺寸以壓縮圖像。可選地，它也會刪除已裁剪的區域。

```csharp
public bool CompressImage(bool deleteCroppedAreasOfImage, PicturesCompression resolution)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| deleteCroppedAreasOfImage | Boolean | 如果為 true，方法將移除圖像的裁剪區域，可能進一步減小其大小。 |
| resolution | PicturesCompression | 壓縮的目標解析度，指定為 [`PicturesCompression`](../../../aspose.slides.export/picturescompression) 列舉的值。 |

### 回傳值

指示圖像是否成功壓縮的 Boolean。若圖像已重新調整大小或已裁剪則回傳 `true`，否則回傳 `false`。

### 例外

| 例外 | 條件 |
| --- | --- |
| ArgumentException | 當解析度不是有效值時拋出此例外。 |

### 備註

此方法會更改圖像的大小與解析度，類似於 PowerPoint 的「Picture Format -&gt; Compress Pictures」功能。

### 範例

以下範例說明如何使用 `CompressImage` 方法，透過設定目標解析度並移除裁剪區域，以減少簡報中圖像的大小：

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;
    // 以 150 DPI（網路解析度）為目標解析度壓縮圖像，並移除裁剪區域
    bool result = picFrame.PictureFormat.CompressImage(true, PicturesCompression.Dpi150);
}
```

## CompressImage(bool, float) {#compressimage_1}

根據形狀大小和指定的解析度縮小圖像的尺寸以壓縮圖像。可選地，它也會刪除已裁剪的區域。

```csharp
public bool CompressImage(bool deleteCroppedAreasOfImage, float resolution)
```

| 參數 | 類型 | 說明 |

| resolution | Single | 目標解析度（DPI）。此值必須為正值，決定圖像將如何重新調整大小。 |

### 回傳值

指示圖像是否成功壓縮的 Boolean。若圖像已重新調整大小或已裁剪則回傳 `true`，否則回傳 `false`。

### 範例

以下範例說明如何使用 `CompressImage` 方法，透過設定目標解析度並移除裁剪區域，以減少簡報中圖像的大小：


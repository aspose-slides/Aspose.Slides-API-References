## BaseSlide.ShowMasterShapes 屬性

指定在母片投影片上的圖形是否應在投影片上顯示。對於母片投影片本身，此屬性永遠回傳 `false`。可讀寫 Boolean。

```csharp
public abstract bool ShowMasterShapes { get; set; }
```

### 例外

| 例外 | 條件 |
| --- | --- |
| NotSupportedException | 若在母片投影片上設定為 `true`，則拋出。 |

## InsertClone(int, ISlide) {#insertclone}

將指定投影片的副本插入至集合中的指定位置。

```csharp
public ISlide InsertClone(int index, ISlide sourceSlide)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| index | Int32 | 新投影片的索引。 |
| sourceSlide | ISlide | 要複製的投影片。 |

### 傳回值

已插入的投影片。

### 備註

在不同簡報之間複製投影片時，投影片的母片也可能被複製。內部登錄用於自動追蹤已複製的母片，以防止建立同一母片的多個複本。手動複製母片既不會被阻止，也不會被登錄。如果需要更精細的複製控制，請使用 [`InsertClone`](../insertclone) 或 [`InsertClone`](../insertclone) 來複製投影片，並使用 [`AddClone`](../../imasterslidecollection/addclone) 來複製母片。

### 範例

以下範例說明如何在同一個 Presentation 中於其他位置複製投影片。

```csharp
[C#]
// 建立代表簡報檔案的 Presentation 類別實例
using (Presentation pres = new Presentation("CloneWithInSamePresentation.pptx"))
{
    // 將指定的投影片複製到同一簡報中投影片集合的末端
    ISlideCollection slds = pres.Slides;
    // 將指定的投影片複製到同一簡報中指定的索引位置
    slds.InsertClone(2, pres.Slides[1]);
    // 將已修改的簡報寫入磁碟
    pres.Save("Aspose_CloneWithInSamePresentation_out.pptx", SaveFormat.Pptx);
}
```

以下範例說明如何在同一個 Presentation 中於其他位置複製投影片。

```csharp
[C#]
// 建立 Presentation 類別以載入來源簡報檔案
using (Presentation srcPres = new Presentation("CloneAtEndOfAnother.pptx"))
{
    // 建立目標 PPTX 的 Presentation 類別（投影片將被複製至此）
    using (Presentation destPres = new Presentation())
    {
        ISlideCollection slds = destPres.Slides;
        slds.InsertClone(2, srcPres.Slides[0]);
        // 將目標簡報寫入磁碟
        destPres.Save("Aspose2_out.pptx", SaveFormat.Pptx);
    }
}
```

## InsertClone(int, ISlide, IMasterSlide, bool) {#insertclone_2}

將指定來源投影片的副本插入至集合中的指定位置。系統會自動從指定的母片中選取相應的版面配置（相同類型或名稱的版面配置即為相應的版面配置）。如果在指定的母片中沒有相應的版面配置，則會根據 allowCloneMissingLayout 的值決定：為 true 時會複製來源投影片的版面配置，為 false 時會拋出 PptxEditException。

```csharp
public ISlide InsertClone(int index, ISlide sourceSlide, IMasterSlide destMaster, 
    bool allowCloneMissingLayout)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| index | Int32 | 新投影片的索引。 |
| sourceSlide | ISlide | 要複製的投影片。 |
| destMaster | IMasterSlide | 新投影片的母片。 |
| allowCloneMissingLayout | Boolean | 如果在指定的母片中沒有相應的版面配置，則根據此參數決定：為 true 時會複製來源投影片的版面配置，為 false 時會拋出 PptxEditException。 |

### 傳回值

已插入的投影片。

### 例外狀況

| 例外 | 條件 |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | 當在指定的母片中沒有相應的版面配置且 allowCloneMissingLayout 為 false 時拋出。 |

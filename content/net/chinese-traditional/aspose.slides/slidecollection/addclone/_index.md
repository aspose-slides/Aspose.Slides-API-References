## AddClone(ISlide) {#addclone}

將指定投影片的副本新增至集合的末端。

```csharp
public ISlide AddClone(ISlide sourceSlide)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| sourceSlide | ISlide | 要複製的投影片。 |

### 傳回值

新投影片。

### 備註

在不同簡報之間複製投影片時，也可能會複製投影片的母片。系統會使用內部登錄表來追蹤自動複製的母片，以防止相同母片產生多個複本。手動複製母片既不會被阻止，也不會被登錄。如果需要對複製過程有更多控制，請使用 [`AddClone`](../addclone) 或 [`AddClone`](../addclone) 來複製投影片，[`AddClone`](../../igloballayoutslidecollection/addclone) 或 [`AddClone`](../../igloballayoutslidecollection/addclone) 來複製版面配置，並使用 [`AddClone`](../../imasterslidecollection/addclone) 來複製母片。

### 範例

```csharp
[C#]
using (IPresentation presentation = new Presentation())
{
    presentation.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 200, 50, 300, 100);
    presentation.Sections.AddSection("Section 1", presentation.Slides[0]);
    
    ISection section2 = presentation.Sections.AppendEmptySection("Section 2");
    presentation.Slides.AddClone(presentation.Slides[0], section2);
    
    // 現在第二個章節包含了第一張投影片的副本。
}
```

## AddClone(ISlide, IMasterSlide, bool) {#addclone_2}

將指定來源投影片的副本新增至集合的末端。系統會自動從指定的母片中選取相應的版面配置（相應的版面配置是與來源投影片的版面配置類型或名稱相同的版面配置）。如果在指定的母片中找不到相應的版面配置，則會複製來源投影片的版面配置（若 allowCloneMissingLayout 為 true）或拋出 PptxEditException（若 allowCloneMissingLayout 為 false）。

```csharp
public ISlide AddClone(ISlide sourceSlide, IMasterSlide destMaster, bool allowCloneMissingLayout)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| sourceSlide | ISlide | 要複製的投影片。 |
| destMaster | IMasterSlide | 新投影片的母片。 |
| allowCloneMissingLayout | Boolean | 若在指定的母片中找不到相應的版面配置，則會複製來源投影片的版面配置（若 allowCloneMissingLayout 為 true）或拋出 PptxEditException（若 allowCloneMissingLayout 為 false）。 |

### 傳回值

新投影片。

### 例外

| 例外 | 條件 |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | 當在指定的母片中找不到相應的版面配置且 allowCloneMissingLayout 為 false 時拋出。 |

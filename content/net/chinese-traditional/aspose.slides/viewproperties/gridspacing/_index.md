## ViewProperties.GridSpacing 屬性

傳回或設定在簡報文件下方格線所使用的格線間距，單位為點。讀寫 Single.

```csharp
public float GridSpacing { get; set; }
```

### 備註

格線間距值必須為正數。典型的值範圍為 1 毫米 (2.8349607 點) 到 2 吋 (144 點)。

### 範例

以下範例程式碼示範如何在 PowerPoint 簡報中變更格線間距。

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

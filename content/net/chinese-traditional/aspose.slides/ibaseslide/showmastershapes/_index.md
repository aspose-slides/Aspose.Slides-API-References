## IBaseSlide.ShowMasterShapes 屬性

指定在母投影片上的圖形是否應顯示在投影片上。對於母投影片本身，此屬性永遠返回 `false`。讀寫 Boolean.

```csharp
public bool ShowMasterShapes { get; set; }
```

### 例外

| 例外 | 條件 |
| --- | --- |
| NotSupportedException | 如果對母投影片設定 `true`，則會拋出此例外。 |

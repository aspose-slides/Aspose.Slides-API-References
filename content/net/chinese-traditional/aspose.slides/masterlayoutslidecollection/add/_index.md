## MasterLayoutSlideCollection.Add 方法

將新的版面投影片新增至集合的末端。

```csharp
public ILayoutSlide Add(SlideLayoutType layoutType, string layoutName)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| layoutType | SlideLayoutType | 為新佈局提供的版面類型。支援的版面類型：Title、TitleOnly、Blank、TitleAndObject、VerticalText、VerticalTitleAndText、TwoObjects、SectionHeader、TwoTextAndTwoObjects、TitleObjectAndCaption、PictureAndCaption、Custom。現在不支援的其他版面類型：Text、TwoColumnText、Table、TextAndChart、ChartAndText、Diagram、Chart、TextAndClipArt、ClipArtAndText、TextAndObject、ObjectAndText、Object、TextAndMedia、MediaAndText、ObjectOverText、TextOverObject、TextAndTwoObjects、TwoObjectsAndText、TwoObjectsOverText、FourObjects、ClipArtAndVerticalText、VerticalTitleAndTextOverChart、ObjectAndTwoObject、TwoObjectsAndObject。 |
| layoutName | String | 新佈局的名稱。如果傳入的名稱已被使用，將拋出 ArgumentException。如果傳入 null 參數，則會根據傳入的版面類型自動產生名稱（例如 “Title Slide” 或 “1_Title Slide”、 “2_..” 等）。 |

### 傳回值

已新增的投影片。

### 例外狀況

| 例外 | 條件 |
| --- | --- |
| NotImplementedException | 如果傳入不支援的 *layoutType* 參數值，則會拋出 NotImplementedException。現在不支援的版面類型：Text、TwoColumnText、Table、TextAndChart、ChartAndText、Diagram、Chart、TextAndClipArt、ClipArtAndText、TextAndObject、ObjectAndText、Object、TextAndMedia、MediaAndText、ObjectOverText、TextOverObject、TextAndTwoObjects、TwoObjectsAndText、TwoObjectsOverText、FourObjects、ClipArtAndVerticalText、VerticalTitleAndTextOverChart、ObjectAndTwoObject、TwoObjectsAndObject。 |
| ArgumentException | 如果此佈局集合中已經使用 *layoutName* 的佈局名稱值，則會拋出 ArgumentException。 |

### 備註

1) 為 *layoutType* 為 SlideLayoutType.Custom 的值所新增的佈局不包含任何佔位符與圖形。 2) 此方法的類似方法是 [`Add`](../../igloballayoutslidecollection/add)，可透過 [`LayoutSlides`](../../ipresentation/layoutslides) 屬性存取。

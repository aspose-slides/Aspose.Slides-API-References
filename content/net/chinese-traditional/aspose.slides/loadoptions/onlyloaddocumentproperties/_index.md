## LoadOptions.OnlyLoadDocumentProperties 屬性

此屬性在簡報檔案受密碼保護時才有意義。值為 true 表示只需要從加密的簡報檔案載入文件屬性，且忽略密碼。值為 false 表示必須使用正確的密碼載入整個加密的簡報。如果簡報未加密，則此屬性值始終會被忽略。如果加密檔案的文件屬性不是公開的且屬性值為 true，則無法載入文件屬性，並會拋出例外。讀/寫 Boolean.

```csharp
public bool OnlyLoadDocumentProperties { get; set; }
```

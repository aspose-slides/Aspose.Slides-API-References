## LoadOptions.DeleteEmbeddedBinaryObjects 屬性

決定 Aspose.Slides 在載入簡報時是否會刪除所有嵌入的二進位物件。

嵌入的二進位物件類型如下：

* VBA 專案 [`VbaProject`](../../ipresentation/vbaproject)
* OLE 物件嵌入資料 [`EmbeddedFileData`](../../ioleembeddeddatainfo/embeddedfiledata)
* ActiveX 控制項二進位資料 [`ActiveXControlBinary`](../../icontrol/activexcontrolbinary)

讀寫 Boolean。

```csharp
public bool DeleteEmbeddedBinaryObjects { get; set; }
```

### 備註

預設為 **false**。

### 範例

以下範例說明如何在不含任何嵌入二進位物件的情況下載入簡報。

```csharp
[C#]
LoadOptions loadOptions = new LoadOptions();
loadOptions.DeleteEmbeddedBinaryObjects = true;
using (Presentation pres = new Presentation("pres.ppt", loadOptions))
{
    pres.Save("output_WithoutBinaryObjects.ppt", SaveFormat.Ppt);
}
```

## Hyperlink.ExternalUrlOriginal 屬性

表示為此區段設定的超連結，與該區段的實際內容無關。 PowerPoint 針對區段中的連結及其對應文字具有特定的行為。 它允許以有效的 URL 形式為超連結建立文字，該文字與連結的實際位址不同。 在此情況下，當您在編輯視窗中檢視連結時，連結會被更改為與文字區段相符。 此屬性代表超連結的原始值。

```csharp
public string ExternalUrlOriginal { get; }
```

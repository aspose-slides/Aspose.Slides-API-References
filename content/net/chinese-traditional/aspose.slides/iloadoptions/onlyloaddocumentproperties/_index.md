## ILoadOptions.OnlyLoadDocumentProperties 屬性

此屬性在簡報檔案受到密碼保護時才有意義。值為 true 表示只能從加密的簡報檔案中載入文件屬性，且密碼將被忽略。值為 false 表示必須使用正確的密碼載入整個加密的簡報。若簡報未加密，則屬性值一律會被忽略。若加密檔案的文件屬性未公開且屬性值為 true，則無法載入文件屬性，並會拋出例外。讀寫 Boolean.

```csharp
public bool OnlyLoadDocumentProperties { get; set; }
```

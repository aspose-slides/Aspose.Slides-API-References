## EmbeddingLevel 列舉

表示嵌入字型的授權權利。

```csharp
[Flags]
public enum EmbeddingLevel : ushort
```

### 值

| 名稱 | 值 | 說明 |
| --- | --- | --- |
| Installable | `0` | 具備此設定的字型表示可由應用程式嵌入並永久安裝於遠端系統。遠端系統的使用者取得與字型原始購買者相同的權利、義務和授權，且受相同的最終使用者授權協議、版權、設計專利以及/或商標限制。 |
| Restricted | `2` | 只設定此位元的字型不得在未取得合法所有者許可的情況下進行修改、嵌入或交換。 |
| PreviewPrint | `4` | 設定此位元時，字型可被嵌入，並暫時載入遠端系統。包含 Preview &amp; Print 字型的文件必須以「唯讀」模式開啟，文件內容不得被編輯。 |
| Editable | `8` | 設定此位元時，字型可被嵌入，但只能暫時安裝於其他系統。與 Preview &amp; Print 字型不同，包含 Editable 字型的文件可以開啟閱讀、允許編輯，且變更可以儲存。 |
| NoSubsetting | `100` | 設定此位元時，嵌入前不得對字型執行子集化。位元 0-3 與 9 中指定的其他嵌入限制亦同時適用。 |
| BitmapOnly | `200` | 設定此位元時，只能嵌入字型中包含的點陣圖，不能嵌入輪廓資料。若字型中沒有可用的點陣圖，則視為不可嵌入，嵌入服務將失敗。 |

## IMasterLayoutSlideCollection.Insert 方法

將新版面投影片插入至集合的指定位置。

```csharp
public ILayoutSlide Insert(int index, SlideLayoutType layoutType, string layoutName)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| index | Int32 | 新投影片的索引。 |
| layoutType | SlideLayoutType | 新版面的版面配置類型。支援的版面配置類型：Title, TitleOnly, Blank, TitleAndObject, VerticalText, VerticalTitleAndText, TwoObjects, SectionHeader, TwoTextAndTwoObjects, TitleObjectAndCaption, PictureAndCaption, Custom。其他版面配置類型目前不支援：Text, TwoColumnText, Table, TextAndChart, ChartAndText, Diagram, Chart, TextAndClipArt, ClipArtAndText, TextAndObject, ObjectAndText, Object, TextAndMedia, MediaAndText, ObjectOverText, TextOverObject, TextAndTwoObjects, TwoObjectsAndText, TwoObjectsOverText, FourObjects, ClipArtAndVerticalText, VerticalTitleAndTextOverChart, ObjectAndTwoObject, TwoObjectsAndObject。 |
| layoutName | String | 新版面的名稱。如果傳入的名稱已被使用，將拋出 ArgumentException。如果傳入 null 參數，則會根據傳入的版面配置類型自動產生名稱（例如「Title Slide」或「1_Title Slide」、 「2_..」等）。 |

### 返回值

已插入的投影片。

### 例外

| 例外 | 條件 |
| --- | --- |
| NotImplementedException | 如果傳入不支援的參數 *layoutType* 值，則拋出此例外。現在不支援的版面配置類型：Text, TwoColumnText, Table, TextAndChart, ChartAndText, Diagram, Chart, TextAndClipArt, ClipArtAndText, TextAndObject, ObjectAndText, Object, TextAndMedia, MediaAndText, ObjectOverText, TextOverObject, TextAndTwoObjects, TwoObjectsAndText, TwoObjectsOverText, FourObjects, ClipArtAndVerticalText, VerticalTitleAndTextOverChart, ObjectAndTwoObject, TwoObjectsAndObject。 |
| ArgumentException | 如果版面名稱值 *layoutName* 已在此版面集合中使用，則拋出此例外。 |

### 備註

對於 *layoutType* 為 SlideLayoutType.Custom 的已插入版面不含任何佔位符和形狀。

## Presentation.HighlightRegex 方法

以指定的顏色突顯正則表達式的所有匹配項。

```csharp
public void HighlightRegex(Regex regex, Color highlightColor, IFindResultCallback callback)
```

| Parameter | Type | Description |
| --- | --- | --- |
| regex | Regex | 正則表達式 Regex 用於取得要突顯的字串。 |
| highlightColor | Color | 用於突顯文字的顏色。 |
| callback | IFindResultCallback | 用於接收搜尋結果的 callback 物件 [`IFindResultCallback`](../../ifindresultcallback)。 |

### 範例

以下程式碼範例示範如何在 PowerPoint Presentation 中使用正則表達式突顯文字。

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// 突顯所有長度為 10 個字元或以上的詞
	presentation.HighlightRegex(regex, Color.Blue, null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

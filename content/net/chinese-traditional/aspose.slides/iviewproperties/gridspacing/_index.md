## IViewProperties.GridSpacing 屬性

返回或設定應用於簡報文件底層格線的間距（以點為單位）。讀寫 Single.

```csharp
public float GridSpacing { get; set; }
```

### 備註

格線間距值必須為正數。典型的取值範圍為 1 mm（2.8349607 點）至 2 英吋（144 點）。

### 範例

以下範例程式碼示範如何在 PowerPoint 簡報中變更格線間距。

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

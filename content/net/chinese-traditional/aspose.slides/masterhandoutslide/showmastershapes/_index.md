## MasterHandoutSlide.ShowMasterShapes 屬性

指定是否在投影片上顯示母投影片上的形狀。對於母投影片本身，此屬性永遠回傳 `false`。可讀寫 Boolean。

```csharp
public override bool ShowMasterShapes { get; set; }
```

### 例外

| 例外 | 條件 |
| --- | --- |
| NotSupportedException | 如果在母投影片上設為 `true`，則拋出此例外。 |

## AddClone(ISlide) {#addclone}

將指定投影片的副本新增至集合的末端。

```csharp
public ISlide AddClone(ISlide sourceSlide)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| sourceSlide | ISlide | 要複製的投影片。 |

### 回傳值

新投影片。

### 備註

在不同簡報之間複製投影片時，投影片的母片也可能被複製。內部註冊表用於自動追蹤已複製的母片，以防止同一母片產生多個副本。手動複製母片既不會被阻止也不會被註冊。若需要更精細的複製控制，請使用 [`AddClone`](../addclone) 或 [`AddClone`](../addclone) 進行投影片複製，[`AddClone`](../../igloballayoutslidecollection/addclone) 或 [`AddClone`](../../igloballayoutslidecollection/addclone) 進行版面配置複製，[`AddClone`](../../imasterslidecollection/addclone) 進行母片複製。

### 範例

```csharp
[C#]
using (IPresentation presentation = new Presentation())
{
    presentation.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 200, 50, 300, 100);
    presentation.Sections.AddSection("Section 1", presentation.Slides[0]);
    
    ISection section2 = presentation.Sections.AppendEmptySection("Section 2");
    presentation.Slides.AddClone(presentation.Slides[0], section2);
    
    // 現在第二個分段包含第一張投影片的副本。
}
```

## AddClone(ISlide, IMasterSlide, bool) {#addclone_2}

將指定來源投影片的副本新增至集合的末端。系統會自動從指定的母片中選取適當的版面配置（適當的版面配置是與來源投影片版面配置具有相同 Type 或 Name 的版面配置）。如果指定的母片中沒有適當的版面配置，則會根據 allowCloneMissingLayout 的值決定：為 true 時複製來源投影片的版面配置，為 false 時拋出 PptxEditException。

```csharp
public ISlide AddClone(ISlide sourceSlide, IMasterSlide destMaster, bool allowCloneMissingLayout)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| sourceSlide | ISlide | 要複製的投影片。 |
| destMaster | IMasterSlide | 新投影片的母片。 |
| allowCloneMissingLayout | Boolean | 若指定的母片中沒有適當的版面配置，則根據此參數決定是複製來源投影片的版面配置（為 true 時）或拋出 PptxEditException（為 false 時）。 |

### 回傳值

新投影片。

### 例外情況

| 例外 | 條件 |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | 若在指定的母片中沒有適當的版面配置且 allowCloneMissingLayout 為 false 時拋出此例外。 |

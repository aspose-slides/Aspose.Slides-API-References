## InsertClone(int, ISlide) {#insertclone}

在集合的指定位置插入指定投影片的複本。

```csharp
public ISlide InsertClone(int index, ISlide sourceSlide)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| index | Int32 | 新投影片的索引。 |
| sourceSlide | ISlide | 要克隆的投影片。 |

### 傳回值

已插入的投影片。

### 備註

在不同簡報之間複製投影片時，投影片的母片也可能被複製。內部登錄用於追蹤自動複製的母片，以防止同一母片產生多個複本。手動複製母片既不會被阻止，也不會被登錄。如果需要對複製過程有更多控制，請使用 [`InsertClone`](../insertclone) 或 [`InsertClone`](../insertclone) 來複製投影片，並使用 [`AddClone`](../../imasterslidecollection/addclone) 來複製母片。

## InsertClone(int, ISlide, IMasterSlide, bool) {#insertclone_2}

在集合的指定位置插入指定來源投影片的複本。系統會從指定的母片自動選取適當的版面配置（適當的版面配置是與來源投影片的版面配置在 Type 或 Name 上相同的版面配置）。如果在指定的母片中找不到適當的版面配置，則會克隆來源投影片的版面配置（若 allowCloneMissingLayout 為 true），或拋出 PptxEditException（若 allowCloneMissingLayout 為 false）。

```csharp
public ISlide InsertClone(int index, ISlide sourceSlide, IMasterSlide destMaster, 
    bool allowCloneMissingLayout)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| index | Int32 | 新投影片的索引。 |
| sourceSlide | ISlide | 要克隆的投影片。 |
| destMaster | IMasterSlide | 新投影片的母片。 |
| allowCloneMissingLayout | Boolean | 若在指定的母片中找不到適當的版面配置，則會克隆來源投影片的版面配置（若 allowCloneMissingLayout 為 true），或拋出 PptxEditException（若 allowCloneMissingLayout 為 false）。 |

### 傳回值

已插入的投影片。

### 例外

| 例外 | 條件 |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | 若在指定的母片中找不到適當的版面配置且 allowCloneMissingLayout 為 false，則拋出此例外。 |

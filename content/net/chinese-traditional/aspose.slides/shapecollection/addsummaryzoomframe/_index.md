## ShapeCollection.AddSummaryZoomFrame 方法

建立一個新的 Summary Zoom 框架，並將其新增至形狀集合的末端。

```csharp
public ISummaryZoomFrame AddSummaryZoomFrame(float x, float y, float width, float height)
```

| 參數 | 類型 | 描述 |
| --- | --- | --- |
| x | Single | 新 Summary Zoom 框架的 x 座標（單位為點）。 |
| y | Single | 新 Summary Zoom 框架的 y 座標（單位為點）。 |
| width | Single | 新 Summary Zoom 框架的寬度（單位為點）。 |
| height | Single | 新 Summary Zoom 框架的高度（單位為點）。 |

### 返回值

新建立的 [`ISummaryZoomFrame`](../../isummaryzoomframe)。

### 例外

| 例外 | 條件 |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | 如果簡報中沒有任何節，或目標投影片不屬於任何節，則拋出此例外。 |

### 備註

此方法會建立一個新的 Summary Zoom，並將所有節的物件集合放入其中。

### 範例

此範例示範將 Summary Zoom 物件新增至集合的末端（假設在 "Presentation.pptx" 簡報中至少有兩個節）：

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSummaryZoomFrame(150, 20, 500, 250);
}
```

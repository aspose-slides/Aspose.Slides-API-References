## InsertAudioFrameEmbedded(int, float, float, float, float, Stream) {#insertaudioframeembedded_1}

建立一個包含嵌入式 WAV 檔案的新音訊框架，並將其插入至指定索引的圖形集合中。嵌入的音訊會加入 Presentation.Audios 集合。

```csharp
public IAudioFrame InsertAudioFrameEmbedded(int index, float x, float y, float width, float height, 
    Stream audio_stream)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| index | Int32 | 插入音訊框架的零基索引。 |
| x | Single | 新音訊框架的 X 座標（單位：點）。 |
| y | Single | 新音訊框架的 Y 座標（單位：點）。 |
| width | Single | 新音訊框架的寬度（單位：點）。 |
| height | Single | 新音訊框架的高度（單位：點）。 |
| audio_stream | Stream | 包含要嵌入的 WAV 音訊資料的輸入串流。 |

### 返回值

新建立的 [`IAudioFrame`](../../iaudioframe)。

## InsertAudioFrameEmbedded(int, float, float, float, float, IAudio) {#insertaudioframeembedded}

建立新的音訊框架，並使用 Presentation.Audios 清單中的現有音訊物件，將其插入至指定索引的圖形集合中。

```csharp
public IAudioFrame InsertAudioFrameEmbedded(int index, float x, float y, float width, float height, 
    IAudio audio)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| index | Int32 | 插入音訊框架的零基索引。 |
| x | Single | 新音訊框架的 X 座標（單位：點）。 |
| y | Single | 新音訊框架的 Y 座標（單位：點）。 |
| width | Single | 新音訊框架的寬度（單位：點）。 |
| height | Single | 新音訊框架的高度（單位：點）。 |
| audio | IAudio | 從 Presentation.Audios 集合中取得的 [`IAudio`](../../iaudio) 例項，用於嵌入。 |

### 返回值

新建立的 [`IAudioFrame`](../../iaudioframe)。

## InsertZoomFrame(int, float, float, float, float, ISlide) {#insertzoomframe}

建立新的 Zoom 框並將其插入指定索引處的形狀集合。

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| index | Int32 | 要插入 Zoom 框的零基索引。 |
| x | Single | 新 Zoom 框的 X 坐標（單位為點）。 |
| y | Single | 新 Zoom 框的 Y 坐標（單位為點）。 |
| width | Single | 新 Zoom 框的寬度（單位為點）。 |
| height | Single | 新 Zoom 框的高度（單位為點）。 |
| slide | ISlide | Zoom 框所參照的 [`ISlide`](../../islide)。 |

### 返回值

新建立的 [`IZoomFrame`](../../izoomframe)。

### 例外狀況

| 例外 | 條件 |
| --- | --- |
| ArgumentException | 如果參照的投影片不屬於目前的簡報，則會拋出此例外。 |

### 範例

此範例示範在集合的指定索引處建立並插入 Zoom 物件（假設「Presentation.pptx」簡報中至少有兩張投影片）：

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertZoomFrame(2, 150, 20, 50, 50, pres.Slides[1]);
}
```

public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide, IPPImage image)

| width | Single | 新 Zoom 框的寬度（單位為點）。 |

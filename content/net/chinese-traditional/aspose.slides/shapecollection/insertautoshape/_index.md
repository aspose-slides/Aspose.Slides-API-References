## InsertAutoShape(int, ShapeType, float, float, float, float) {#insertautoshape}

建立一個新的自動形狀，並將其插入至指定索引的形狀集合中，套用預設範本格式。

```csharp
public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width, 
    float height)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| index | Int32 | 要插入新自動形狀的零基索引。 |
| shapeType | ShapeType | 要插入的自動形狀的 [`ShapeType`](../../shapetype)。 |
| x | Single | 形狀框架的 X 座標（單位：點）。 |
| y | Single | 形狀框架的 Y 座標（單位：點）。 |
| width | Single | 形狀框架的寬度（單位：點）。 |
| height | Single | 形狀框架的高度（單位：點）。 |

### 返回值

新建立的 [`IAutoShape`](../../iautoshape)。

## InsertAutoShape(int, ShapeType, float, float, float, float, bool) {#insertautoshape_1}

建立一個新的自動形狀，並將其插入至指定索引的形狀集合中，亦可選擇性地以預設範本樣式初始化。

```csharp
public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| index | Int32 | 要插入自動形狀的零基索引。 |
| shapeType | ShapeType | 要插入的自動形狀的 [`ShapeType`](../../shapetype)。 |
| x | Single | 形狀框架的 X 座標（單位：點）。 |
| y | Single | 形狀框架的 Y 座標（單位：點）。 |
| width | Single | 形狀框架的寬度（單位：點）。 |
| height | Single | 形狀框架的高度（單位：點）。 |
| createFromTemplate | Boolean | True 表示套用預設範本樣式（包括非空名稱、簡易樣式與置中文字）；false 表示建立形狀時所有屬性均使用預設值。 |

### 返回值

新建立的 [`IAutoShape`](../../iautoshape)。

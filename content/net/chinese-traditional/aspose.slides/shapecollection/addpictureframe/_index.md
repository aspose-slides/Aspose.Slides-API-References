## ShapeCollection.AddPictureFrame 方法

建立一個包含指定影像的新圖片框，並將其新增至形狀集合的末端。

```csharp
public IPictureFrame AddPictureFrame(ShapeType shapeType, float x, float y, float width, 
    float height, IPPImage image)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| shapeType | ShapeType | 指定 [`ShapeType`](../../shapetype) 中所含的形狀類型，除了所有線條類型外： |
| x | Single | 圖片框的 x 座標，以點為單位。 |
| y | Single | 圖片框的 y 座標，以點為單位。 |
| width | Single | 圖片框的寬度，以點為單位。 |
| height | Single | 圖片框的高度，以點為單位。 |
| image | IPPImage | 要在圖片框中顯示的 [`IPPImage`](../../ippimage)。 |

### 傳回值

新建立的 [`IPictureFrame`](../../ipictureframe)。

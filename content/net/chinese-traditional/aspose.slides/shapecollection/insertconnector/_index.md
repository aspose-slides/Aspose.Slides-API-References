## InsertConnector(int, ShapeType, float, float, float, float) {#insertconnector}

建立一個新的連接器形狀，並在指定的索引處將其插入到形狀集合中，套用預設範本樣式。

```csharp
public IConnector InsertConnector(int index, ShapeType shapeType, float x, float y, float width, 
    float height)
```

| 參數 | 型別 | 說明 |
| --- | --- | --- |
| index | Int32 | 要插入連接器形狀的零基索引。 |
| shapeType | ShapeType | 要插入的連接器形狀的 [`ShapeType`](../../shapetype)。 |
| x | Single | 連接器框架的 x 座標（以點為單位）。 |
| y | Single | 連接器框架的 y 座標（以點為單位）。 |
| width | Single | 連接器框架的寬度（以點為單位）。 |
| height | Single | 連接器框架的高度（以點為單位）。 |

### 傳回值

新建立的 [`IConnector`](../../iconnector)。

## InsertConnector(int, ShapeType, float, float, float, float, bool) {#insertconnector_1}

建立一個新的連接器形狀，並在指定的索引處將其插入到形狀集合中，可選擇套用預設範本樣式。

```csharp
public IConnector InsertConnector(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)
```

| 參數 | 型別 | 說明 |
| --- | --- | --- |
| index | Int32 | 要插入連接器形狀的零基索引。 |
| shapeType | ShapeType | 要插入的連接器形狀的 [`ShapeType`](../../shapetype)。 |
| x | Single | 連接器框架的 x 座標（以點為單位）。 |
| y | Single | 連接器框架的 y 座標（以點為單位）。 |
| width | Single | 連接器框架的寬度（以點為單位）。 |
| height | Single | 連接器框架的高度（以點為單位）。 |
| createFromTemplate | Boolean | True 為套用預設範本樣式（非空名稱、簡易樣式）；false 為以預設屬性值建立連接器。 |

### 傳回值

新建立的 [`IConnector`](../../iconnector)。

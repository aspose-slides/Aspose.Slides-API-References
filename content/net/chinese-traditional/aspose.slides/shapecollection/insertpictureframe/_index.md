## ShapeCollection.InsertPictureFrame 方法

建立一個包含指定圖像的新圖片框，並將其插入到指定索引處的形狀集合中。

```csharp
public IPictureFrame InsertPictureFrame(int index, ShapeType shapeType, float x, float y, 
    float width, float height, IPPImage image)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| index | Int32 | 要插入圖片框的零基索引。 |
| shapeType | ShapeType | 指定 [`ShapeType`](../../shapetype) 中包含的形狀類型，除所有類型的線條外： |
| x | Single | 圖片框的 x 座標，單位為點。 |
| y | Single | 圖片框的 y 座標，單位為點。 |
| width | Single | 圖片框的寬度，單位為點。 |
| height | Single | 圖片框的高度，單位為點。 |
| image | IPPImage | 在圖片框中顯示的 [`IPPImage`](../../ippimage)。 |

### 回傳值

新建立的 [`IPictureFrame`](../../ipictureframe)。

## AddSectionZoomFrame(float, float, float, float, ISection) {#addsectionzoomframe}

建立一個新的 Section Zoom 框架，並將其添加到形狀集合的末端。

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| x | Single | 新 Section Zoom 框架的 x 座標，單位為點。 |
| y | Single | 新 Section Zoom 框架的 y 座標，單位為點。 |
| width | Single | 新 Section Zoom 框架的寬度，單位為點。 |
| height | Single | 新 Section Zoom 框架的高度，單位為點。 |
| section | ISection | 被 Section Zoom 框架引用的 [`ISection`](../../isection)；必須屬於此簡報且至少包含一張投影片。 |

### 返回值

新建立的 [`ISectionZoomFrame`](../../isectionzoomframe)。

### 例外

| 例外 | 條件 |
| --- | --- |
| ArgumentException | 若引用的區段不屬於目前的簡報或不包含任何投影片，則拋出此例外。 |

### 範例

此範例示範如何將 Section Zoom 物件加入集合的末端（假設在「Presentation.pptx」簡報中至少有兩個區段）：

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1]);
}
```

## AddSectionZoomFrame(float, float, float, float, ISection, IPPImage) {#addsectionzoomframe_1}

建立一個具有預定義影像的新 Section Zoom 框架，並將其添加到形狀集合的末端。

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section, IPPImage image)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| x | Single | 新 Section Zoom 框架的 x 座標，單位為點。 |
| y | Single | 新 Section Zoom 框架的 y 座標，單位為點。 |
| width | Single | 新 Section Zoom 框架的寬度，單位為點。 |
| height | Single | 新 Section Zoom 框架的高度，單位為點。 |
| section | ISection | 被 Section Zoom 框架引用的 [`ISection`](../../isection)；必須屬於此簡報且至少包含一張投影片。 |
| image | IPPImage | 在 Section Zoom 框架中顯示的 [`IPPImage`](../../ippimage)。 |

### 返回值

新建立的 [`ISectionZoomFrame`](../../isectionzoomframe)。

### 例外

| 例外 | 條件 |
| --- | --- |
| ArgumentException | 若引用的區段不屬於目前的簡報或不包含任何投影片，則拋出此例外。 |

### 範例

此範例示範如何將 Section Zoom 物件加入集合的末端（假設在「Presentation.pptx」簡報中至少有兩個區段）：

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IPPImage image = pres.Images.AddImage(Image.FromFile("image.png"));
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1], image);
}
```

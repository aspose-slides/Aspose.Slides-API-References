## ShapeCollection.InsertSummaryZoomFrame method

建立新的 Summary Zoom 框架，並將其插入至形狀集合中指定的索引位置。

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| index | Int32 | 要插入 Summary Zoom 框架的零基索引。 |
| x | Single | 新 Summary Zoom 框架的 x 坐標（以點為單位）。 |
| y | Single | 新 Summary Zoom 框架的 y 坐標（以點為單位）。 |
| width | Single | 新 Summary Zoom 框架的寬度（以點為單位）。 |
| height | Single | 新 Summary Zoom 框架的高度（以點為單位）。 |

### 返回值

新建立的 [`ISummaryZoomFrame`](../../isummaryzoomframe)。

### 例外

| 例外 | 條件 |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | 如果簡報不包含任何節，或目標投影片不屬於任何節，則會拋出此例外。 |

### 備註

此方法會建立一個 Summary Zoom 框架，彙總簡報中所有節的摘要連結。

### 範例

此範例說明如何在集合的指定索引處建立並插入 Summary Zoom 物件（假設在 "Presentation.pptx" 簡報中至少有兩個節）：

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSummaryZoomFrame(2, 150, 20, 50, 50);
}
```

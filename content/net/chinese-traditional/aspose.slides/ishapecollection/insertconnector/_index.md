## InsertConnector(int, ShapeType, float, float, float, float) {#insertconnector}

建立一個新的連接線形狀，並將其插入至形狀集合中指定的索引位置，套用預設範本樣式。

```csharp
public IConnector InsertConnector(int index, ShapeType shapeType, float x, float y, float width, 
    float height)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| index | Int32 | 要插入連接線形狀的零基索引。 |
| shapeType | ShapeType | 要插入的連接線形狀的 [`ShapeType`](../../shapetype)。 |
| x | Single | 連接線框架的 x 坐標，單位為點。 |
| y | Single | 連接線框架的 y 坐標，單位為點。 |
| width | Single | 連接線框架的寬度，單位為點。 |
| height | Single | 連接線框架的高度，單位為點。 |

### 傳回值

新建立的 [`IConnector`](../../iconnector)。

## InsertConnector(int, ShapeType, float, float, float, float, bool) {#insertconnector_1}

建立一個新的連接線形狀，並將其插入至形狀集合中指定的索引位置，可選擇性套用預設範本樣式。

```csharp
public IConnector InsertConnector(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| index | Int32 | 要插入連接線形狀的零基索引。 |
| shapeType | ShapeType | 要插入的連接線形狀的 [`ShapeType`](../../shapetype)。 |
| x | Single | 連接線框架的 x 坐標，單位為點。 |

| width | Single | 連接線框架的寬度，單位為點。 |
| height | Single | 連接線框架的高度，單位為點。 |
| createFromTemplate | Boolean | True 表示套用預設範本樣式（非空名稱、簡易樣式）；false 表示以預設屬性值建立連接線。 |

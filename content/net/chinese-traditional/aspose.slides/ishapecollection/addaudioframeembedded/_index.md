## AddAudioFrameEmbedded(float, float, float, float, Stream) {#addaudioframeembedded_1}

建立一個嵌入 WAV 檔案的新音訊框，並將其加入形狀集合的末端。嵌入的音訊會加入 Presentation.Audios 集合中。

```csharp
public IAudioFrame AddAudioFrameEmbedded(float x, float y, float width, float height, 
    Stream audio_stream)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| x | Single | 新音訊框的 x 坐標，單位為點。 |
| y | Single | 新音訊框的 y 坐標，單位為點。 |
| width | Single | 新音訊框的寬度，單位為點。 |
| height | Single | 新音訊框的高度，單位為點。 |
| audio_stream | Stream | 包含要嵌入之 WAV 音訊資料的輸入串流。 |

### 回傳值

新建立的 [`IAudioFrame`](../../iaudioframe)。

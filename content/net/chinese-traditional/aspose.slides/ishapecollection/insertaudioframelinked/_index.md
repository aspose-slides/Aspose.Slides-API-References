## IShapeCollection.InsertAudioFrameLinked 方法

建立一個連結至外部音訊檔案的新音訊框架，並將其插入至指定索引位置的形狀集合中。

```csharp
public IAudioFrame InsertAudioFrameLinked(int index, float x, float y, float width, float height, 
    string fname)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| index | Int32 | 要插入音訊框架的零基索引。 |
| x | Single | 新音訊框架的 x 座標，單位為點。 |
| y | Single | 新音訊框架的 y 座標，單位為點。 |
| width | Single | 新音訊框架的寬度，單位為點。 |
| height | Single | 新音訊框架的高度，單位為點。 |
| fname | String | 要連結的外部音訊檔案的路徑或名稱。 |

## AddGroupShape() {#addgroupshape}

建立一個新的空白群組形狀，並將其加入形狀集合的末端。群組的框架會自動調整以適應加入的任何形狀。

```csharp
public IGroupShape AddGroupShape()
```

### 返回值

新建立的 [`IGroupShape`](../../igroupshape)。

## AddGroupShape(ISvgImage, float, float, float, float) {#addgroupshape_1}

建立一個新的群組形狀，將指定的 SVG 圖像轉換為個別形狀，並將產生的群組加入形狀集合的末端。

```csharp
public IGroupShape AddGroupShape(ISvgImage svgImage, float x, float y, float width, float height)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| svgImage | ISvgImage | 包含向量內容以轉換為形狀的 [`ISvgImage`](../../isvgimage)。 |
| x | Single | 群組框架的 x 座標，單位為點。 |
| y | Single | 群組框架的 y 座標，單位為點。 |
| width | Single | 群組框架的寬度，單位為點。 |
| height | Single | 群組框架的高度，單位為點。 |

### 返回值

新建立的 [`IGroupShape`](../../igroupshape)。

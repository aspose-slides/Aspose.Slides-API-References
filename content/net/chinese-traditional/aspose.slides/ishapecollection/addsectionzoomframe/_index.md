## AddSectionZoomFrame(float, float, float, float, ISection) {#addsectionzoomframe}

建立新的 Section Zoom 框架，並將其加入形狀集合的末端。

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| x | Single | 新 Section Zoom 框架的 x 坐標，以點為單位。 |
| y | Single | 新 Section Zoom 框架的 y 坐標，以點為單位。 |
| width | Single | 新 Section Zoom 框架的寬度，以點為單位。 |
| height | Single | 新 Section Zoom 框架的高度，以點為單位。 |
| section | ISection | Section Zoom 框架所參照的 [`ISection`](../../isection)；必須屬於此簡報且至少包含一張投影片。 |

### 回傳值

新建立的 [`ISectionZoomFrame`](../../isectionzoomframe)。

### 例外狀況

| 例外 | 條件 |
| --- | --- |
| ArgumentException | 如果參照的節不屬於目前的簡報或不包含投影片，則拋出此例外。 |

### 範例

此範例示範將 Section Zoom 物件新增至集合的末端（假設 "Presentation.pptx" 簡報中至少有兩個節）：

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1]);
}
```

## AddSectionZoomFrame(float, float, float, float, ISection, IPPImage) {#addsectionzoomframe_1}

建立帶有預先定義圖像的新 Section Zoom 框架，並將其加入形狀集合的末端。

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section, IPPImage image)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| x | Single | 新 Section Zoom 框架的 x 坐標，以點為單位。 |
| y | Single | 新 Section Zoom 框架的 y 坐標，以點為單位。 |
| width | Single | 新 Section Zoom 框架的寬度，以點為單位。 |
| height | Single | 新 Section Zoom 框架的高度，以點為單位。 |
| section | ISection | Section Zoom 框架所參照的 [`ISection`](../../isection)；必須屬於此簡報且至少包含一張投影片。 |
| image | IPPImage | 在 Section Zoom 框架內顯示的 [`IPPImage`](../../ippimage)。 |

### 回傳值

| 例外 | 條件 |

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
```

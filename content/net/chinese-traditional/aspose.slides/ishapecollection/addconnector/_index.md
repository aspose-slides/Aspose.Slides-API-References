```csharp

    bool createFromTemplate)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| shapeType | ShapeType | 要建立的連接線形狀的 [`ShapeType`](../../shapetype)。 |
| x | Single | 連接線框架的 x 座標（以點為單位）。 |
| y | Single | 連接線框架的 y 座標（以點為單位）。 |

| height | Single | 連接線框架的高度（以點為單位）。 |

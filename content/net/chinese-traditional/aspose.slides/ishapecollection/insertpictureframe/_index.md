## IShapeCollection.InsertPictureFrame 方法

建立包含指定影像的新圖片框，並將其插入至圖形集合中指定的索引位置。

```csharp
public IPictureFrame InsertPictureFrame(int index, ShapeType shapeType, float x, float y, 
    float width, float height, IPPImage image)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | Int32 | 要插入圖片框的零基索引。 |
| shapeType | ShapeType | 指定包含於 [`ShapeType`](../../shapetype) 中的圖形類型，除所有類型的線條外： |
| x | Single | 圖片框的 x 座標，以點為單位。 |
| y | Single | 圖片框的 y 座標，以點為單位。 |
| width | Single | 圖片框的寬度，以點為單位。 |
| height | Single | 圖片框的高度，以點為單位。 |
| image | IPPImage | 要在圖片框中顯示的 [`IPPImage`](../../ippimage)。 |

### 返回值

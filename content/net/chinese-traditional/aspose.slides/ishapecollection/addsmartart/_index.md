## IShapeCollection.AddSmartArt 方法

建立 SmartArt 圖表，並將其新增至形狀集合的末端。

```csharp
public ISmartArt AddSmartArt(float x, float y, float width, float height, 
    SmartArtLayoutType layoutType)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| x | Single | 圖表框架的 x 坐標（以點為單位）。 |
| y | Single | 圖表框架的 y 坐標（以點為單位）。 |
| width | Single | 圖表框架的寬度（以點為單位）。 |
| height | Single | 圖表框架的高度（以點為單位）。 |
| layoutType | SmartArtLayoutType | SmartArt 版面配置類型。 |

### 返回值

新建立的 [`ISmartArt`](../../../aspose.slides.smartart/ismartart)。

### 範例

範例：

```csharp
[C#]
Presentation pres = new Presentation();
Slide slide = pres.Slides[0];
SmartArt smart = slide.Shapes.AddSmartArt(0, 0, 400, 400, SmartArtLayoutType.BasicBlockList);
```

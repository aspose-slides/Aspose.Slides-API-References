## AddZoomFrame(float, float, float, float, ISlide) {#addzoomframe}

建立新的 Zoom 框架，並將其新增至形狀集合的末端。

```csharp
public IZoomFrame AddZoomFrame(float x, float y, float width, float height, ISlide slide)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| x | Single | 新 Zoom 框架的 x 座標（單位：點）。 |
| y | Single | 新 Zoom 框架的 y 座標（單位：點）。 |
| width | Single | 新 Zoom 框架的寬度（單位：點）。 |
| height | Single | 新 Zoom 框架的高度（單位：點）。 |
| slide | ISlide | Zoom 框架所參照的 [`ISlide`](../../islide)；必須屬於此簡報。 |

### 傳回值

新建立的 [`IZoomFrame`](../../izoomframe)。

### 例外狀況

| 例外 | 條件 |
| --- | --- |
| ArgumentException | 若參照的投影片不屬於目前的簡報，則拋出例外。 |

### 範例

以下範例示範如何將 Zoom 物件新增至集合的末端（假設在「Presentation.pptx」簡報中至少有兩張投影片）：

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
}
```

```csharp
public IZoomFrame AddZoomFrame(float x, float y, float width, float height, ISlide slide, 
    IPPImage image)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| x | Single | 新 Zoom 框架的 x 座標（單位：點）。 |
| y | Single | 新 Zoom 框架的 y 座標（單位：點）。 |
| width | Single | 新 Zoom 框架的寬度（單位：點）。 |
| height | Single | 新 Zoom 框架的高度（單位：點）。 |
| slide | ISlide | Zoom 框架所參照的 [`ISlide`](../../islide)；必須屬於此簡報。 |
| image | IPPImage | 參考投影片 [`IPPImage`](../../ippimage) 的圖片。 |

### 傳回值

以下範例示範如何將 Zoom 物件新增至集合的末端（假設在「Presentation.pptx」簡報中至少有兩張投影片）：

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))

}
```

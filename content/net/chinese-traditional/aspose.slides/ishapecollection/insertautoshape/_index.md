| x | Single | 圖形框架的 x 座標（單位：點）。 |

## InsertAutoShape(int, ShapeType, float, float, float, float, bool) {#insertautoshape_1}

建立一個新的自動圖形，並將其插入至指定索引的圖形集合中，亦可選擇性地以預設範本樣式初始化。

```csharp
public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| index | Int32 | 要插入自動圖形的零基索引。 |
| shapeType | ShapeType | 要插入的自動圖形的 [`ShapeType`](../../shapetype)。 |
| x | Single | 圖形框架的 x 座標（單位：點）。 |
| y | Single | 圖形框架的 y 座標（單位：點）。 |
| width | Single | 圖形框架的寬度（單位：點）。 |
| height | Single | 圖形框架的高度（單位：點）。 |
| createFromTemplate | Boolean | True 代表套用預設範本樣式（包括非空白名稱、簡易樣式與置中對齊的文字）；false 代表建立圖形時所有屬性皆設定為預設值。 |

### 返回值

新建立的 [`IAutoShape`](../../iautoshape)。

## IShapeCollection.InsertSummaryZoomFrame 方法

建立一個新的 Summary Zoom 框架，並將其插入到指定索引處的形狀集合中。

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| 參數 | 類型 | 描述 |
| --- | --- | --- |
| index | Int32 | 插入 Summary Zoom 框架的零基索引。 |
| x | Single | 新 Summary Zoom 框架的 x 坐標（單位為點）。 |
| y | Single | 新 Summary Zoom 框架的 y 坐標（單位為點）。 |
| width | Single | 新 Summary Zoom 框架的寬度（單位為點）。 |
| height | Single | 新 Summary Zoom 框架的高度（單位為點）。 |

### 返回值

新建立的 [`ISummaryZoomFrame`](../../isummaryzoomframe)。

### 例外

| 例外 | 條件 |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | 如果簡報不包含任何章節，或目標投影片不屬於任何章節，則拋出此例外。 |

### 備註

此方法建立一個 Summary Zoom 框架，彙總簡報中所有章節的摘要連結。

### 示例

此範例示範在集合的指定索引處建立並插入 Summary Zoom 物件（假設在 "Presentation.pptx" 簡報中至少有兩個章節）：

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSummaryZoomFrame(2, 150, 20, 50, 50);
}
```

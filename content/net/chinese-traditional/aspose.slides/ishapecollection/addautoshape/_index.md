## AddAutoShape(ShapeType, float, float, float, float, bool) {#addautoshape_1}

建立新的自動圖形並將其加入圖形集合的末端，亦可選擇以預設範本格式初始化。

```csharp
public IAutoShape AddAutoShape(ShapeType shapeType, float x, float y, float width, float height, 
    bool createFromTemplate)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| shapeType | ShapeType | 要新增的自動圖形的[`ShapeType`](../../shapetype)。 |
| x | Single | 形狀框架的 x 座標（單位：點）。 |
| y | Single | 形狀框架的 y 座標（單位：點）。 |
| width | Single | 形狀框架的寬度（單位：點）。 |
| height | Single | 形狀框架的高度（單位：點）。 |
| createFromTemplate | Boolean | True 表示將預設範本樣式（簡單樣式、置中文字且名稱非空）套用至新圖形；false 表示以全部屬性預設值建立圖形。 |

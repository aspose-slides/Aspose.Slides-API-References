## InsertAudioFrameEmbedded(int, float, float, float, float, Stream) {#insertaudioframeembedded_1}

建立一個嵌入 WAV 檔案的新音訊框架，並將它插入至形狀集合中指定的索引位置。嵌入的音訊會加入到 Presentation.Audios 集合。

```csharp
public IAudioFrame InsertAudioFrameEmbedded(int index, float x, float y, float width, float height, 
    Stream audio_stream)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| index | Int32 | 要插入音訊框架的零基索引。 |
| x | Single | 新音訊框架的 X 座標（以點為單位）。 |
| y | Single | 新音訊框架的 Y 座標（以點為單位）。 |
| width | Single | 新音訊框架的寬度（以點為單位）。 |
| height | Single | 新音訊框架的高度（以點為單位）。 |
| audio_stream | Stream | 包含要嵌入之 WAV 音訊資料的輸入串流。 |

### 返回值

新建立的 [`IAudioFrame`](../../iaudioframe)。

## InsertAudioFrameEmbedded(int, float, float, float, float, IAudio) {#insertaudioframeembedded}

使用 Presentation.Audios 清單中的現有音訊物件，建立一個新音訊框架，並將它插入至形狀集合中指定的索引位置。

```csharp
public IAudioFrame InsertAudioFrameEmbedded(int index, float x, float y, float width, float height, 
    IAudio audio)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| index | Int32 | 要插入音訊框架的零基索引。 |
| x | Single | 新音訊框架的 X 座標（以點為單位）。 |
| y | Single | 新音訊框架的 Y 座標（以點為單位）。 |
| width | Single | 新音訊框架的寬度（以點為單位）。 |
| height | Single | 新音訊框架的高度（以點為單位）。 |
| audio | IAudio | 從 Presentation.Audios 集合中取得的 [`IAudio`](../../iaudio) 實例，用於嵌入。 |

### 返回值

新建立的 [`IAudioFrame`](../../iaudioframe)。

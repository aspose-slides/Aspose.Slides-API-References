## Ink.InkEffectImages 屬性

取得用於模擬墨水筆刷視覺效果的自訂圖像集合。這些圖像在使用特定 [`InkEffectType`](../../inkeffecttype) 值（例如 Galaxy、Rainbow 等）渲染墨水時使用。透過提供自己的圖像，您可以控制每個墨水效果的外觀。

```csharp
public static IDictionary<InkEffectType, IImage> InkEffectImages { get; }
```

### 備註

此屬性允許將預設的墨水效果紋理替換為使用者自訂的紋理，當預設資產受到授權限制或在執行階段不可用時，特別有用。字典中的每個條目必須將 [`InkEffectType`](../../inkeffecttype) 值與相對應的 [`IImage`](../../../aspose.slides/iimage) 物件（例如 Bitmap，或 Aspose 圖像介面）關聯起來。

### 範例

```csharp
[C#]
IImage image = Images.FromFile("image.png");
Ink.InkEffectImages.Add(InkEffectType.Galaxy, image);
```

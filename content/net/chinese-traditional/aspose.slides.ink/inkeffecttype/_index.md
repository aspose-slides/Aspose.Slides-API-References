## InkEffectType 列舉

指定一組預定義的墨跡渲染視覺效果。每種效果對應一種紋理或圖像，用以模擬風格化的墨跡外觀。這些值可用於在渲染過程中自訂數位墨跡筆畫的視覺樣式。

```csharp
public enum InkEffectType
```

### 值

| 名稱 | 值 | 說明 |
| --- | --- | --- |
| NotDefined | `-1` | 效果未定義或未知。此值可作為預設或備援使用。 |
| Bronze | `0` | 類似青銅墨水的暖色棕色金屬紋理。 |
| Galaxy | `1` | 類似星系的多彩閃爍紋理，帶有宇宙色調。 |
| Gold | `2` | 明亮的金屬金色紋理，使墨跡筆畫呈現奢華外觀。 |
| Lava | `3` | 類似熔岩的炙熱紋理，呈現紅色與橙色調。 |
| Ocean | `4` | 深藍色、類似液體的紋理，模擬海浪或水基墨水。 |
| RainbowGlitter | `5` | 多彩閃爍的彩虹亮粉效果，適用於節慶或鮮豔的墨跡筆畫。 |
| RoseGold | `6` | 柔和的粉金色混合，類似玫瑰金，適合優雅的墨跡筆畫。 |
| Silver | `7` | 冷色金屬銀色紋理，模擬傳統銀色墨水。 |

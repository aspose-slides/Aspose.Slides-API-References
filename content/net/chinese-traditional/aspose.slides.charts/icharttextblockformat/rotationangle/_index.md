## IChartTextBlockFormat.RotationAngle 屬性

指定套用於邊框內文字的自訂旋轉角度。若未指定，則使用伴隨圖形的旋轉。若已指定，則會獨立於圖形套用。也就是說，圖形本身可以有旋轉，而文字本身也可以有旋轉。此屬性與屬性 TextVerticalType 中的預定義垂直類型共同彙總出的視覺文字旋轉值。讀寫 Single。

```csharp
public float RotationAngle { get; set; }
```

### 範例

考慮一個情況，圖形被套用了順時針 90 度的旋轉。除此之外，文字本身又被套用了逆時針 -90 度的旋轉。如此一來，最終的圖形看起來仍是旋轉的，但其中的文字看起來彷彿根本沒有被旋轉。

## ChartSeries.FirstSliceAngle 屬性

指定第一個餅圖或環形圖切片的角度，單位為度（從上方順時針，範圍 0 到 360 度）。此屬性不僅屬於此系列，亦屬於父系列組的所有系列──它是相應組屬性的投影。因此此屬性為唯讀。使用 ParentSeriesGroup 屬性存取父系列組。使用 ParentSeriesGroup.FirstSliceAngle 讀寫屬性以更改值。唯讀 UInt16。

```csharp
public ushort FirstSliceAngle { get; }
```

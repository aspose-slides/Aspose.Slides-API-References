## ChartSeries.TrendLines 屬性

系列趨勢線的集合。唯讀 [`ITrendlineCollection`](../../itrendlinecollection).

```csharp
public ITrendlineCollection TrendLines { get; }
```

### 備註

TrendLines 在未堆疊的 2-D 面積圖、條形圖、柱狀圖、折線圖、股票圖、xy（散點圖）和氣泡圖中的資料系列可用（非 null）。在任何堆疊或 3-D 的圖表類型中，資料系列均不可使用趨勢線。雷達圖、圓餅圖、表面圖或環形圖亦不支援趨勢線。

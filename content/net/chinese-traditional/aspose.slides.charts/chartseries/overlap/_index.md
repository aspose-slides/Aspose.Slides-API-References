## ChartSeries.Overlap 屬性

指定 2D 圖表中長條與柱形的重疊程度，以百分比表示（從 -100% 到 100%）。此屬性不僅屬於此系列，還屬於父系列組的所有系列。它是父系列組中相應屬性的投影，因而此屬性為唯讀。若要變更數值，請使用 !:ParentSeriesGroup.Overlap 可讀寫屬性。唯讀 SByte。

```csharp
public sbyte Overlap { get; }
```

### 備註

Overlap 指定長條與柱形之間的重疊或間距程度，以其寬度的百分比表示：- -100%：最大間距（長條完全分離）。- 0%：長條並排放置，沒有重疊或間距。- 100%：最大重疊（長條彼此完全重疊）。此為屬性 !:ParentSeriesGroup.Overlap 的投影。

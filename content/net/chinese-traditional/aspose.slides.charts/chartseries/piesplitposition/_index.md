## ChartSeries.PieSplitPosition 屬性

指定用於確定哪些資料點位於 pie-of-pie 或 bar-of-pie 圖表的第二個餅圖或條形圖的值。此屬性與 PieSplitBy 屬性一起使用。此屬性不僅屬於此序列，亦屬於父序列群組的所有序列──它是相應群組屬性的投影。因此此屬性為唯讀。使用 ParentSeriesGroup 屬性存取父序列群組。使用 ParentSeriesGroup.PieSplitPosition 可讀寫屬性來變更值。唯讀 Double。

```csharp
public double PieSplitPosition { get; }
```

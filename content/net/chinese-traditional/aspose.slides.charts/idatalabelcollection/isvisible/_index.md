## IDataLabelCollection.IsVisible 屬性

False 表示預設情況下資料標籤不可見（因此 DefaultDataLabelFormat 屬性的所有 Show*-flags（如 ShowValue 等）皆為 false）。唯讀 Boolean。

```csharp
public bool IsVisible { get; }
```

### 備註

如果資料標籤預設可見，您可以使用 Hide() 方法將其預設隱藏。但如果資料標籤預設不可見（IsVisible 為 false），則可以透過將 DefaultDataLabelFormat 屬性的 Show*-flags（如 ShowValue 等）設為 true，使資料標籤「預設可見」。

## IRotation3D.RotationY 屬性

傳回或設定圍繞 Y 軸的旋轉角度，即 3D 圖表中 X 方向的旋轉角度（介於 0 到 360 度之間）。 此屬性符合 ECMA-376 中的 21.2.2.158 rotY（Y Rotation）項目，且與 PowerPoint 2007+ 中的「X Rotation」選項相對應。 可讀寫 UInt16。

```csharp
public ushort RotationY { get; set; }
```

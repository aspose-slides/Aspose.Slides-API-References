## IChartData.UseSecondaryCategories 屬性

如果 false，則 [`SecondaryCategories`](../secondarycategories) 屬性返回 null，並且在 [`Categories`](../categories) 屬性中的資料同時用於主要和次要系列。若 true，則 [`SecondaryCategories`](../secondarycategories) 屬性中的資料用於次要系列，而 [`Categories`](../categories) 屬性中的資料用於主要系列。可讀寫 Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

### 範例

範例。哪個類別與系列相關 - ChartData.Categories 或 ChartData.SecondaryCategories？

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // 相關類別是 series.Chart.ChartData.SecondaryCategories
}
else
{
    // 相關類別是 series.Chart.ChartData.Categories
}
```

## IChartData.SecondaryCategories 屬性

如果 [`UseSecondaryCategories`](../usesecondarycategories) 屬性為 true，取得次要類別。唯讀 [`IChartCategoryCollection`](../../ichartcategorycollection)。

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### 備註

如果 [`UseSecondaryCategories`](../usesecondarycategories) 屬性為 false，則此 `SecondaryCategories` 屬性回傳 null，且 [`Categories`](../categories) 屬性中的資料同時用於主要和次要系列。若 [`UseSecondaryCategories`](../usesecondarycategories) 屬性為 true，則此 `SecondaryCategories` 屬性中的資料用於次要系列，而 [`Categories`](../categories) 屬性中的資料用於主要系列。

### 範例

範例。哪個類別與系列相關 - ChartData.Categories 或 ChartData.SecondaryCategories？

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // 相關類別為 series.Chart.ChartData.SecondaryCategories
}
else
{
    // 相關類別為 series.Chart.ChartData.Categories
}
```

## IChartSeriesGroup.Overlap 屬性

指定在 2-D 圖表上長條與柱形的重疊比例（從 -100% 到 100%）。  
- -100%：最大間距（長條完全分開）。  
- 0%：長條並排放置，沒有重疊也沒有間距。  
- 100%：最大重疊（長條彼此完全重疊）。此屬性為讀寫 SByte。

```csharp
public sbyte Overlap { get; set; }
```

### 例外狀況

| 例外 | 條件 |
| --- | --- |
| ArgumentOutOfRangeException | 當值設定在 -100 到 100 的有效範圍之外時拋出此例外。 |

### 範例

以下範例示範如何設定圖表系列群組的重疊程度，並在表單上呈現產生的圖表：

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // 設定重疊為 55%

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

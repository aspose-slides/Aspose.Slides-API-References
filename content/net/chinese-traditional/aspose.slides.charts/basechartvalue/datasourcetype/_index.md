## BaseChartValue.DataSourceType 屬性

指定在衍生類別中實際使用 AsCell、AsCells、AsLiteralString 或 AsLiteralDouble 屬性。換句話說，它指定 Data 屬性之值的類型。可讀寫 [`DataSourceType`](../../datasourcetype)。

```csharp
public DataSourceType DataSourceType { get; set; }
```

### 備註

對於 ChartDataPointCollection 中的點，此屬性為唯讀。在此情況下，要變更此屬性的值，您可以使用 ChartDataPointCollection.DataSourceTypeFor&lt;...&gt; 之一的屬性。

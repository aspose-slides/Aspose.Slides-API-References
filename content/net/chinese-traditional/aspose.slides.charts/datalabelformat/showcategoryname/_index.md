## DataLabelFormat.ShowCategoryName 屬性

表示指定圖表的資料標籤類別名稱顯示行為。True 代表在圖表上的資料標籤顯示類別名稱。False 代表隱藏。可讀寫 Boolean。

```csharp
public bool ShowCategoryName { get; set; }
```

### 備註

如果此 DataLabelFormat 物件的父項是 DataLabelCollection（資料標籤的集合），則此屬性取得或設定 DataLabelCollection 中新資料標籤的 ShowCategoryName 屬性的預設值。將此屬性設定為某個值時，亦會將該值設定給 DataLabelCollection 中所有資料標籤的 ShowCategoryName 屬性（即「DataLabels.DefaultDataLabelFormat.ShowCategoryName = val;」導致所有 DataLabels[i].ShowCategoryName 等於 val）。

## DataLabelFormat.ShowPercentage 屬性

表示指定圖表的資料標籤百分比值顯示行為。True 會顯示百分比值。False 會隱藏。讀寫 Boolean.

```csharp
public bool ShowPercentage { get; set; }
```

### 備註

如果此 DataLabelFormat 物件的父項是 DataLabelCollection 資料標籤集合，則此屬性會取得或設定 DataLabelCollection 集合中新資料標籤的 ShowPercentage 屬性的預設值。以值設定此屬性也會將此值設定給 DataLabelCollection 集合中所有資料標籤的 ShowPercentage 屬性 (例如 "DataLabels.DefaultDataLabelFormat.ShowPercentage = val;" 會使所有 DataLabels[i].ShowPercentage 等於 val)。

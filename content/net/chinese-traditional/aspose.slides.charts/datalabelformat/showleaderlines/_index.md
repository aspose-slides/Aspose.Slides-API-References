## DataLabelFormat.ShowLeaderLines property

表示指定圖表的資料標籤領線顯示行為。True 表示顯示領線。False 表示隱藏。可讀寫 Boolean。

```csharp
public bool ShowLeaderLines { get; set; }
```

### 備註

如果此 DataLabelFormat 物件的父項是 DataLabelCollection 資料標籤集合，則此屬性會取得或設定 DataLabelCollection 集合中新資料標籤的 ShowLeaderLines 屬性的預設值。將此屬性設定為某個值時，也會將該值設定給 DataLabelCollection 集合中所有資料標籤的 ShowLeaderLines 屬性 (i.e. "DataLabels.DefaultDataLabelFormat.ShowLeaderLines = val;" 會導致所有 DataLabels[i].ShowLeaderLines 等於 val)。

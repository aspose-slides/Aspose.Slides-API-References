## DataLabelFormat.Separator 屬性

設定或傳回代表圖表中資料標籤所使用之分隔符的 Variant。可讀寫 String。

```csharp
public string Separator { get; set; }
```

### 備註

如果此 DataLabelFormat 物件的父項是 DataLabelCollection（資料標籤的集合），則此屬性會取得或設定 DataLabelCollection 中新資料標籤之 Separator 屬性的預設值。以某個值設定此屬性，同時也會將該值設定到 DataLabelCollection 中所有資料標籤的 Separator 屬性（例如 "DataLabels.DefaultDataLabelFormat.Separator = val;" 會使所有 DataLabels[i].Separator 等於 val）。

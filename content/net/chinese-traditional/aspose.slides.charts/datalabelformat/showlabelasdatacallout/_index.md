## DataLabelFormat.ShowLabelAsDataCallout 屬性

判斷指定圖表的資料標籤將以資料標註框或資料標籤顯示。若此 DataLabelFormat 物件的父項是 DataLabelCollection（資料標籤集合），則此屬性取得或設定 DataLabelCollection（資料標籤集合）中新資料標籤的 ShowLabelAsDataCallout 屬性的預設值。將此屬性設定為某個值時，同時也會將該值設定給 DataLabelCollection（資料標籤集合）中所有資料標籤的 ShowLabelAsDataCallout 屬性（即 "DataLabels.DefaultDataLabelFormat.ShowLabelAsDataCallout = val;" 會導致所有 DataLabels[i].ShowLabelAsDataCallout 等於 val）。

```csharp
public bool ShowLabelAsDataCallout { get; set; }
```

## DataLabelFormat.Position 屬性

表示資料標籤的位置。可讀寫 [`LegendDataLabelPosition`](../../legenddatalabelposition).

```csharp
public LegendDataLabelPosition Position { get; set; }
```

### 備註

如果此 DataLabelFormat 物件的父項是 DataLabelCollection（資料標籤的集合），則此屬性會取得或設定 DataLabelCollection 集合中新資料標籤的 Position 屬性的預設值。表示 DataLabel 物件的位置。為此屬性設定值時，同時會將此值設定為 DataLabelCollection 集合中所有資料標籤的 Position 屬性 (例如 "DataLabels.DefaultDataLabelFormat.Position = val;" 會使所有 DataLabels[i].Position 等於 val)。

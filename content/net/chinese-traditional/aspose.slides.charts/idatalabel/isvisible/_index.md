## IDataLabel.IsVisible 屬性

False 表示資料標籤不可見（因此所有 Show*-flags（ShowValue，…）皆為 false）。唯讀 Boolean.

```csharp
public bool IsVisible { get; }
```

### 備註

如果資料標籤可見，您可以使用 Hide() 方法將其隱藏。但如果資料標籤不可見（IsVisible 為 false），您可以透過將 Show*-flags（ShowValue，…）設定為 true 來使資料標籤顯示。

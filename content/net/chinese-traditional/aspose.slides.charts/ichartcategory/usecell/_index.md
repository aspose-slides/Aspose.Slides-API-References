## IChartCategory.UseCell 屬性

如果為 true，則 AsCell 屬性為實際值。換句話說，工作表用於儲存類別（此情況支援多層類別）。如果為 false，則 AsLiteral 屬性為實際值。換句話說，工作表不會用於儲存類別（此情況不支援多層類別）。唯讀 Boolean。

```csharp
public bool UseCell { get; }
```

### 備註

若要變更此屬性的值（針對集合中的所有類別），將新值設定至 ChartCategoryCollection.UseCells 屬性。

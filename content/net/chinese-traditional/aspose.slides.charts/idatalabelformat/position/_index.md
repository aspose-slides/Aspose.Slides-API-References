## IDataLabelFormat.Position property

表示資料標籤的位置。可讀寫 [`LegendDataLabelPosition`](../../legenddatalabelposition)。

```csharp
public LegendDataLabelPosition Position { get; set; }
```

### Remarks

如果此 DataLabelFormat 物件的父項是 DataLabelCollection（資料標籤集合），則此屬性取得或設定 DataLabelCollection 中新資料標籤的 Position 屬性的預設值。表示 DataLabel 物件的位置。將此屬性設定為某個值時，也會將此值設定給 DataLabelCollection 中所有資料標籤的 Position 屬性（例如 "DataLabels.DefaultDataLabelFormat.Position = val;" 會導致所有 DataLabels[i].Position 等於 val）。

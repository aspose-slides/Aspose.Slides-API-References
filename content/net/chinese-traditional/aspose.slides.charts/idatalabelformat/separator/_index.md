## IDataLabelFormat.Separator 屬性

設定或傳回一個 Variant，表示圖表上資料標籤所使用的分隔符號。可讀寫 String.

```csharp
public string Separator { get; set; }
```

### 備註

如果此 DataLabelFormat 物件的父物件是 DataLabelCollection（資料標籤的集合），則此屬性會取得或設定新資料標籤在 DataLabelCollection 中的 Separator 屬性的預設值。將此屬性設定為某個值時，也會將此值設定給 DataLabelCollection 中所有資料標籤的 Separator 屬性（例如 "DataLabels.DefaultDataLabelFormat.Separator = val;" 會導致所有 DataLabels[i].Separator 等於 val）。

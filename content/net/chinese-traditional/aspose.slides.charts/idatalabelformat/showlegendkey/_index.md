## IDataLabelFormat.ShowLegendKey 屬性

表示指定圖表的資料標籤圖例鍵的顯示行為。若資料標籤圖例鍵可見則為 True。可讀寫 Boolean.

```csharp
public bool ShowLegendKey { get; set; }
```

### 備註

如果此 DataLabelFormat 物件的父項是 DataLabelCollection（資料標籤的集合），則此屬性會取得或設定 DataLabelCollection 中新資料標籤的 ShowLegendKey 屬性的預設值。將此屬性設為某個值時，也會將該值設定為 DataLabelCollection 中所有資料標籤的 ShowLegendKey 屬性（例如 "DataLabels.DefaultDataLabelFormat.ShowLegendKey = val;" 會使所有 DataLabels[i].ShowLegendKey 等於 val）。

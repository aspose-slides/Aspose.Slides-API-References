## IDataLabelFormat.ShowBubbleSize 屬性

表示指定圖表的資料標籤氣泡大小值的顯示行為。True 會顯示氣泡大小值；False 會隱藏。可讀寫 Boolean.

```csharp
public bool ShowBubbleSize { get; set; }
```

### 備註

如果此 DataLabelFormat 物件的父項是 DataLabelCollection（資料標籤集合），則此屬性會取得或設定新資料標籤在 DataLabelCollection 中的 ShowBubbleSize 屬性的預設值。將此屬性設為某值時，也會將該值套用至 DataLabelCollection 中所有資料標籤的 ShowBubbleSize 屬性（例如 `DataLabels.DefaultDataLabelFormat.ShowBubbleSize = val;` 會使所有 `DataLabels[i].ShowBubbleSize` 等於 val）。

## IDataLabelFormat.NumberFormat屬性

表示 DataLabels 物件的格式字串。 可讀寫 String。

```csharp
public string NumberFormat { get; set; }
```

### 備註

如果此 DataLabelFormat 物件的父物件是 DataLabelCollection（資料標籤集合），則此屬性會取得或設定 DataLabelCollection 中新資料標籤的 NumberFormat 屬性的預設值。當此屬性以某個值設定時，該值也會同時設定到 DataLabelCollection 中所有資料標籤的 NumberFormat 屬性（例如 "DataLabels.DefaultDataLabelFormat.NumberFormat = val;" 會使所有 DataLabels[i].NumberFormat 等於 val）。

### 範例

```csharp
[C#]
series.Labels.DefaultDataLabelFormat.ShowValue = true;
series.Labels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = false;
series.Labels.DefaultDataLabelFormat.NumberFormat = "0.0%";
```

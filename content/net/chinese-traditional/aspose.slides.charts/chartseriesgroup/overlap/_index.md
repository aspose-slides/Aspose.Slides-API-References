## ChartSeriesGroup.Overlap 屬性

指定在 2-D 圖表上條形圖和柱狀圖的重疊程度，以百分比表示（從 -100% 到 100%）。- -100%：最大間距（條形圖完全分離）。- 0%：條形圖並排放置，無重疊或間距。- 100%：最大重疊（條形圖彼此完全重疊）。此屬性為可讀寫 SByte。

```csharp
public sbyte Overlap { get; set; }
```

### 例外

| 例外 | 條件 |
| --- | --- |
| ArgumentOutOfRangeException | 當值設定超出 -100 到 100 的有效範圍時拋出。 |

### 範例

以下範例說明如何設定圖表系列群組的重疊，並在表單上呈現產生的圖表：

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // 設定重疊為 55%

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

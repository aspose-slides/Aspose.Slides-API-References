## IOverridableText.TextFrameForOverriding 屬性

可以包含富格式文字。若此屬性不為 null，則此格式化文字值會覆寫自動產生的文字。自動產生的文字是資料標籤、值軸的顯示單位標籤、軸標題、圖表標題、趨勢線標籤的隱含屬性。自動產生的文字會使用 IFormattedTextContainer.TextFormat 屬性進行格式化。唯讀 [`ITextFrame`](../../../aspose.slides/itextframe).

```csharp
public ITextFrame TextFrameForOverriding { get; }
```

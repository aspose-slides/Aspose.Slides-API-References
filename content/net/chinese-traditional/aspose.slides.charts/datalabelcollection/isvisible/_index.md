## DataLabelCollection.IsVisible 屬性

False 表示預設情況下資料標籤不可見（因此 DefaultDataLabelFormat 屬性的所有 Show*-flags (ShowValue, ...) 為 false）。唯讀 Boolean.

```csharp
public bool IsVisible { get; }
```

### 備註

如果資料標籤預設為可見，您可以使用 Hide() 方法將其預設設為隱藏。但如果資料標籤預設不可見 (IsVisible 為 false)，您可以透過將 DefaultDataLabelFormat 屬性的 Show*-flags (ShowValue, ...) 設為 true，將資料標籤設為「預設可見」狀態。

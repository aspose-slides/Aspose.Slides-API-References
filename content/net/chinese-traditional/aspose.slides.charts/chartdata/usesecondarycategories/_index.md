## ChartData.UseSecondaryCategories 屬性

如果為 false，則 [`SecondaryCategories`](../secondarycategories) 屬性傳回 null，且 [`Categories`](../categories) 屬性中的資料同時用於主要和次要系列。若為 true，則 [`SecondaryCategories`](../secondarycategories) 屬性中的資料用於次要系列，而 [`Categories`](../categories) 屬性中的資料用於主要系列。可讀寫 Boolean。

```csharp
public bool UseSecondaryCategories { get; set; }
```

### 範例

範例。哪個類別與系列相關 - ChartData.Categories 或 ChartData.SecondaryCategories？

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // 相關類別為 series.Chart.ChartData.SecondaryCategories
}
else
{
    // 相關類別為 series.Chart.ChartData.Categories
}
```

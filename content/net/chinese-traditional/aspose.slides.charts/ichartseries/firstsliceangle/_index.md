## IChartSeries.FirstSliceAngle 屬性

指定第一個圓餅或環形圖切片的角度，單位為度（從上方順時針，範圍 0 到 360 度）。此屬性不僅屬於此序列，也屬於父序列群組的所有序列——這是相應群組屬性的投影。因此此屬性為唯讀。使用 ParentSeriesGroup 屬性以存取父序列群組。使用 ParentSeriesGroup.FirstSliceAngle 讀寫屬性以變更值。唯讀 UInt16。

```csharp
public ushort FirstSliceAngle { get; }
```

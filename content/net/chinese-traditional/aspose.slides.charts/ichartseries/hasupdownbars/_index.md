## IChartSeries.HasUpDownBars property

判斷 Line- 或 Stock-chart 是否具有上/下長條。此屬性不僅屬於此系列，同時屬於父系列群組的所有系列——它是相應群組屬性的投射。因此此屬性為唯讀。使用 ParentSeriesGroup 屬性可存取父系列群組。使用 ParentSeriesGroup.UpDownBars.HasUpDownBars 可讀寫屬性來變更數值。使用 ParentSeriesGroup.UpDownBars 屬性來設定上/下長條的格式。唯讀 Boolean。

```csharp
public bool HasUpDownBars { get; }
```

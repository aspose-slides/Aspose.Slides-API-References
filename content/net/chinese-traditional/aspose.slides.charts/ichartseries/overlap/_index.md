## IChartSeries.Overlap 屬性

指定在 2-D 圖表中條形和柱形的重疊程度，以百分比表示（從 -100% 到 100%）。此屬性不僅屬於此系列，還屬於父系列組的所有系列。它是父系列組中相應屬性的投影，因此此屬性為唯讀。要變更數值，請使用 ParentSeriesGroup.Overlap 可讀寫屬性。唯讀 SByte。

```csharp
public sbyte Overlap { get; }
```

### 備註

Overlap 指定條形和柱形之間的重疊或間距程度，以其寬度的百分比表示：- -100%：最大間距（條形完全分開）。- 0%：條形並排放置，沒有重疊或間距。- 100%：最大重疊（條形完全互相重疊）。此為屬性 ParentSeriesGroup.Overlap 的投影。

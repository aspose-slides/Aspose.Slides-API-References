## IChartSeries.HasSeriesLines 屬性

判斷此系列及相關系列是否具有系列線。此屬性不僅屬於此系列，而是屬於父系列群組的所有系列 —— 這是相應群組屬性的投射。因此此屬性為唯讀。使用 ParentSeriesGroup 屬性以存取父系列群組。使用 ParentSeriesGroup.HasSeriesLines 讀寫屬性以變更其值。使用 ParentSeriesGroup.SeriesLinesFormat 屬性來設定系列線的格式。唯讀 Boolean.

```csharp
public bool HasSeriesLines { get; }
```

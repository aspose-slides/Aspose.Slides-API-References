## IChartSeries.BubbleSizeScale 屬性

指定氣泡圖的比例因子（可為預設大小的 0% 到 300% 之間）。此屬性不僅屬於此系列，亦屬於父系列組的所有系列——它是相應組屬性的投射。因此此屬性為唯讀。使用 ParentSeriesGroup 屬性取得父系列組。使用 ParentSeriesGroup.BubbleSizeScale 可讀寫屬性來變更值。

public int BubbleSizeScale { get; }

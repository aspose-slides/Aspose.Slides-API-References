## IChartSeries.PieSplitBy 屬性

指定如何決定哪些資料點位於 pie-of-pie 或 bar-of-pie 圖表的第二個餅圖或長條圖中。此屬性不僅屬於此系列，亦屬於父系列群組的所有系列——此為相應群組屬性的投影。因此此屬性為唯讀。使用 ParentSeriesGroup 屬性以存取父系列群組。使用 ParentSeriesGroup.PieSplitBy 讀寫屬性以變更值。唯讀 [`PieSplitType`](../../piesplittype).

```csharp
public PieSplitType PieSplitBy { get; }
```

### 備註

1) 這是屬性 ParentSeriesGroup.PieSplitBy 的投影。2) 若屬性值為 PieSplitType.Custom，則可使用 ParentSeriesGroup.PieSplitCustomPoints 屬性定義自訂分割資訊。

## IPdfOptions.SaveMetafilesAsPng 屬性

True 以將簡報中使用的所有 metafile 轉換為 PNG 圖像。讀/寫 Boolean。

```csharp
public bool SaveMetafilesAsPng { get; set; }
```

### 備註

預設為 **true**。Pdf 文件可以包含向量圖形和點陣圖像。若 SaveMetafilesAsPng 設為 true，則來源 Metafile 圖像會被轉換為 Png 格式，並以點陣圖像儲存至 Pdf。若 SaveMetafilesAsPng 設為 false，則來源 Metafile 會被轉換為 Pdf 向量圖形。每種方法都有優點與缺點。例如，若 Metafile 轉換為 PNG，則在最終文件縮放過程中可能會有品質損失。若 Metafile 轉換為 Pdf 向量圖形，則在 Pdf 檢視工具中可能出現效能問題。

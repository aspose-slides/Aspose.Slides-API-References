## IPdfOptions.EmbedTrueTypeFontsForASCII 屬性

True 代表將 ASCII 32-127 的字元嵌入 TrueType 字型。字元代碼大於 127 的字型始終會被嵌入。讀/寫 Boolean.

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### 備註

預設為 **true**。

## IPdfOptions.RasterizeUnsupportedFontStyles 屬性

指示當字型不支援粗體樣式時，是否將文字光柵化為位圖並儲存為 PDF。此作法可提升特定字型在產生的 PDF 中文字的品質。可讀寫 Boolean.

```csharp
public bool RasterizeUnsupportedFontStyles { get; set; }
```

### 備註

預設為 **false**。

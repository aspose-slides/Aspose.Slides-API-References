## IPdfOptions.JpegQuality 屬性

傳回或設定決定 PDF 文件中 JPEG 圖像品質的值。讀/寫 Byte.

```csharp
public byte JpegQuality { get; set; }
```

### 備註

僅當文件包含 JPEG 圖像時才會產生效應。

使用此屬性在將文件儲存為 PDF 格式時取得或設定文件中圖像的品質。該值可以在 0 到 100 之間變化，其中 0 表示最差的品質但最大壓縮，100 表示最佳品質但最小壓縮。

預設值為 **100**。

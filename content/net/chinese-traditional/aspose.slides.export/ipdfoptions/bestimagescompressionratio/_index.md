## IPdfOptions.BestImagesCompressionRatio 屬性

指示是否必須自動為每張圖像選擇最有效的壓縮（而非預設的壓縮）。如果設定為 Boolean.true，則會為簡報中的每張圖像選擇最合適的壓縮演算法，從而使產生的 PDF 文件大小更小。最佳圖像壓縮比的選擇計算量大且會佔用額外的 RAM，而此選項預設為 Boolean.false。

```csharp
public bool BestImagesCompressionRatio { get; set; }
```

### 備註

預設為 Boolean.false。

## IHtmlOptions.JpegQuality 屬性

傳回或設定決定 PDF 文件內 JPEG 圖片品質的值。讀/寫 Byte.

```csharp
public byte JpegQuality { get; set; }
```

### 備註

僅在文件包含 JPEG 圖片時才會生效。

使用此屬性在以 PDF 格式儲存文件時取得或設定文件內圖像的品質。值的範圍為 0 到 100，0 表示品質最差但壓縮率最高，100 表示品質最佳但壓縮率最低。

預設值為 **95**。

## PdfAccessPermissions 列舉

包含一組旗標，用於指定在使用者開啟文件時應授予的存取權限。

```csharp
[Flags]
public enum PdfAccessPermissions
```

### 值

| 名稱 | 值 | 說明 |
| --- | --- | --- |
| None | `0` | 指定使用者未擁有存取權限。 |
| PrintDocument | `4` | 指定使用者是否可以列印文件（可能無法以最高品質列印，取決於位元 HighQualityPrint 是否也被設定）。 |
| ModifyContent | `8` | 指定使用者是否可以透過非由位元 AddOrModifyFields、FillExistingFields、AssembleDocument 控制的操作來修改文件內容。 |
| CopyTextAndGraphics | `10` | 指定使用者是否可以透過非由位元 ExtractTextAndGraphics 控制的操作，複製或以其他方式提取文件中的文字與圖形。 |
| AddOrModifyFields | `20` | 指定使用者是否可以新增或修改文字註解、填寫互動式表單欄位，且當位元 ModifyContent 亦被設定時，建立或修改互動式表單欄位（包含簽名欄位）。 |
| FillExistingFields | `100` | 指定使用者是否可以填寫現有的互動式表單欄位（包含簽名欄位），即使位元 AddOrModifyFields 未被設定。 |
| ExtractTextAndGraphics | `200` | 指定使用者是否可以提取文字與圖形，以支援對有障礙的使用者的可及性或用於其他目的。 |
| AssembleDocument | `400` | 指定使用者是否可以組合文件（插入、旋轉或刪除頁面以及建立書籤或縮圖），即使位元 ModifyContent 未被設定。 |
| HighQualityPrint | `800` | 指定使用者是否可以列印文件至一種可生成 PDF 內容忠實數位副本的表示形式。當此位元未被設定（且位元 PrintDocument 已被設定）時，列印僅限於外觀的低階表示，可能品質下降。 |

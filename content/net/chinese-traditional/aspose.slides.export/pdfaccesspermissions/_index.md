## PdfAccessPermissions 列舉

包含一組旗標，用於指定在使用者存取文件時應授予的存取權限。

```csharp
[Flags]
public enum PdfAccessPermissions
```

### 列舉值

| Name | Value | Description |
| --- | --- | --- |
| None | `0` | 指定使用者沒有任何存取權限。 |
| PrintDocument | `4` | 指定使用者是否可以列印文件（可能不是最高品質等級，取決於是否同時設定了 HighQualityPrint 位元）。 |
| ModifyContent | `8` | 指定使用者是否可以透過除受位元 AddOrModifyFields、FillExistingFields、AssembleDocument 控制之外的操作來修改文件內容。 |
| CopyTextAndGraphics | `10` | 指定使用者是否可以複製或以其他方式從文件中取出文字與圖形，前提是此操作未受位元 ExtractTextAndGraphics 的控制。 |
| AddOrModifyFields | `20` | 指定使用者是否可以新增或修改文字註釋、填寫互動式表單欄位，且當同時設定了 ModifyContent 位元時，亦可建立或修改互動式表單欄位（包括簽章欄位）。 |
| FillExistingFields | `100` | 指定使用者是否可以填寫現有的互動式表單欄位（包括簽章欄位），即使位元 AddOrModifyFields 未設定。 |
| ExtractTextAndGraphics | `200` | 指定使用者是否可以提取文字與圖形，以支援對有障礙的使用者的可及性或用於其他目的。 |
| AssembleDocument | `400` | 指定使用者是否可以組合文件（插入、旋轉或刪除頁面，以及建立書籤或縮圖），即使位元 ModifyContent 未設定。 |
| HighQualityPrint | `800` | 指定使用者是否可以將文件列印為可從中產生 PDF 內容忠實數位副本的表示形式。當此位元未設定（且同時設定了 PrintDocument 位元）時，列印將受限於外觀的低階表示，可能品質較差。 |

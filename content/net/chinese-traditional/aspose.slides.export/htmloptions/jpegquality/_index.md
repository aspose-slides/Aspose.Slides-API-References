## HtmlOptions.JpegQuality 屬性

返回或設定一個值，以決定 PDF 文件中 JPEG 圖像的品質。讀/寫 Byte.

```csharp
public byte JpegQuality { get; set; }
```

### 備註

僅在文件包含 JPEG 圖像時才會產生作用。

使用此屬性在以 PDF 格式儲存文件時取得或設定文件內圖像的品質。該值可在 0 到 100 之間變化，其中 0 代表最低品質但最高壓縮，100 代表最高品質但最低壓縮。

預設值為 **95**。

## ILinkEmbedController.GetUrl 方法

返回指向外部物件的 URL。若 [`GetObjectStoringLocation`](../getobjectstoringlocation) 回傳 Link，則此方法必定被呼叫；若 [`GetObjectStoringLocation`](../getobjectstoringlocation) 回傳 Embed，但無法嵌入時亦可能被呼叫。可對同一物件 ID 多次呼叫。

```csharp
public string GetUrl(int id, int referrer)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| id | Int32 | 物件 ID。此 ID 在整個操作中唯一保存。 |
| referrer | Int32 | 參照物件的 ID，或 0（如果該物件由根文件參照）。可用於產生相對連結。 |

### 返回值

外部物件的 URL，若此物件應被忽略則為 null。

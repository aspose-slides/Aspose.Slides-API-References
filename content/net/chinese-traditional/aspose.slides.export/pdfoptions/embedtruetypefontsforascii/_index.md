## PdfOptions.EmbedTrueTypeFontsForASCII 屬性

判斷 Aspose.Slides 是否會嵌入常用字型以支援 ASCII (33..127 代碼範圍) 文字。字元代碼大於 127 的字型將永遠嵌入。常用字型清單包括 PDF 的基本 14 種字型以及其他使用者指定的字型。讀/寫 Boolean。

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### 備註

預設為 **true**。

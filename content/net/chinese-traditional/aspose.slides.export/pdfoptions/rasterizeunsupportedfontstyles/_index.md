## PdfOptions.RasterizeUnsupportedFontStyles 屬性

指示當字型不支援粗體樣式時，文字是否應以點陣圖形式光柵化並儲存為 PDF。此方法可提升某些字型於產生的 PDF 中文字的品質。讀寫 Boolean.

```csharp
public bool RasterizeUnsupportedFontStyles { get; set; }
```

### 備註

預設值為 **false**。

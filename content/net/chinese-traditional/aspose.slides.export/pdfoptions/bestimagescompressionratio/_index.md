## PdfOptions.BestImagesCompressionRatio 屬性

指示是否必須自動為每個影像選擇最有效的壓縮（而非預設的壓縮）。如果設定為 Boolean.true，則簡報中的每個影像都會選擇最適合的壓縮演算法，從而使產生的 PDF 文件體積更小。最佳影像壓縮比的選擇計算成本較高且會佔用額外的記憶體，且此選項預設為 Boolean.false。

```csharp
public bool BestImagesCompressionRatio { get; set; }
```

### 備註

預設為 Boolean.false。

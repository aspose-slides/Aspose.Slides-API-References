## MarkdownSaveOptions.SlideNumberFormat 屬性

取得或設定用於 Markdown 輸出中投影片編號標題的格式字串。格式必須包含「{0}」佔位符，匯出時會以投影片索引取代它。範例：「# Slide {0}」會產生「# Slide 1」、「# Slide 2」等。

```csharp
public string SlideNumberFormat { get; set; }
```

### 例外

| 例外 | 條件 |
| --- | --- |
| ArgumentNullException | 如果提供的值為 null 或空，則拋出。 |
| ArgumentException | 如果格式字串未包含「{0}」佔位符，則拋出。 |

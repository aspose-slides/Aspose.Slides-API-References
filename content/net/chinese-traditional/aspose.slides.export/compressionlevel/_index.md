## CompressionLevel 列舉

指定 OpenXML 檔案的 ZIP 壓縮等級。較高的等級會提供更佳的壓縮效果，但會導致處理速度較慢。

```csharp
public enum CompressionLevel
```

### 值

| 名稱 | 值 | 描述 |
| --- | --- | --- |
| None | `0` | 未套用任何壓縮。檔案以原始方式儲存。 |
| Level1 | `1` | 最快的壓縮，壓縮比例最低。 |
| Level2 | `2` | 較 Level1 稍快的壓縮，且壓縮比例略佳。 |
| Level3 | `3` | 提供比 Level2 更佳的壓縮，且對效能的影響適中。 |
| Level4 | `4` | 提供比 Level3 更佳的壓縮。 |
| Level5 | `5` | 在 Level4 基礎上提供更佳的壓縮，但需要額外的處理時間。 |
| Level6 | `6` | 標準壓縮，在壓縮速度與檔案大小之間取得良好平衡。預設的壓縮等級。 |
| Level7 | `7` | 提供比 Level6 更高的壓縮，但處理速度較慢。 |
| Level8 | `8` | 提供比 Level7 更高的壓縮。 |
| Level9 | `9` | 最大壓縮。產生最小的檔案大小，但處理速度最慢。 |

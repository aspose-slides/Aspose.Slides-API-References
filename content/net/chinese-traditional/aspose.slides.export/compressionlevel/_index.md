## CompressionLevel 列舉

指定 OpenXML 檔案的 ZIP 壓縮等級。較高的等級可提供更好的壓縮，但會以較慢的處理速度為代價。

```csharp
public enum CompressionLevel
```

### 值

| 名稱 | 值 | 說明 |
| --- | --- | --- |
| None | `0` | 不進行壓縮。檔案會原樣儲存。 |
| Level1 | `1` | 最快的壓縮，但壓縮比最低。 |
| Level2 | `2` | 較快速的壓縮，壓縮比略好於 Level1。 |
| Level3 | `3` | 提供比 Level2 更佳的壓縮，對效能的影響為中等。 |
| Level4 | `4` | 提供比 Level3 更佳的壓縮。 |
| Level5 | `5` | 在 Level4 基礎上提供更好的壓縮，但需要額外的處理時間。 |
| Level6 | `6` | 標準壓縮，提供壓縮速度與檔案大小之間的良好平衡。預設的壓縮等級。 |
| Level7 | `7` | 提供比 Level6 更高的壓縮，但處理速度較慢。 |
| Level8 | `8` | 提供比 Level7 更高的壓縮。 |
| Level9 | `9` | 最大壓縮。產生最小的檔案大小，但處理速度最慢。 |

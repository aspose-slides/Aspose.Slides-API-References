## ISVGOptions.DisableLineEndCropping 屬性

SVG 1.1 缺乏為標記定義內縮的功能。Aspose.Slides SVG 寫入引擎針對此問題提供了解決方案：它會裁切帶箭頭的線段末端，從而避免線段與標記重疊。此選項可關閉此行為。可讀寫 Boolean.

```csharp
public bool DisableLineEndCropping { get; set; }
```

## SVGOptions.DisableLineEndCropping 屬性

SVG 1.1 沒有定義標記插入點的能力。Aspose.Slides SVG 寫入引擎為此問題提供了解決方法：它會裁剪帶有箭頭的線段末端，使線段不會與標記重疊。此選項可關閉此行為。讀寫 Boolean.

```csharp
public bool DisableLineEndCropping { get; set; }
```

包含提供 AI 功能以分析與處理 PowerPoint 簡報的類別。

## 類別

| 類別 | 說明 |
| --- | --- |
| [AsposeAIWebClient](./asposeaiwebclient) | 內建的 [`IAIWebClient`](../aspose.slides.ai/iaiwebclient) 實作，可連接至 Aspose 自有的 LLM。這是無參數 [`SlidesAIAgent`](../aspose.slides.ai/slidesaiagent/slidesaiagent) 建構函式所使用的預設客戶端。 |
| [OpenAICompatibleWebClient](./openaicompatiblewebclient) | 內建的 [`IAIWebClient`](../aspose.slides.ai/iaiwebclient) 實作，可連接至指定基礎 URL 的 OpenAI 相容 LLM 供應者。 |
| [OpenAIWebClient](./openaiwebclient) | 內建的 [`IAIWebClient`](../aspose.slides.ai/iaiwebclient) 實作，可連接至 OpenAI API。 |
| [SlidesAIAgent](./slidesaiagent) | 提供用於處理簡報的 AI 驅動功能。 |
| [SlidesAIAgentException](./slidesaiagentexception) | 代表與 Slides AI Agent 相關的例外情況。 |

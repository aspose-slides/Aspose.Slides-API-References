## AsposeAIWebClient 類別

內建的 [`IAIWebClient`](../iaiwebclient)實作，可連接至 Aspose 自家的 LLM。此為無參數 [`SlidesAIAgent`](../slidesaiagent/slidesaiagent) 建構式所使用的預設用戶端。

```csharp
public sealed class AsposeAIWebClient : IAIWebClient, IDisposable
```

## 建構函式

| 名稱 | 說明 |
| --- | --- |
| [AsposeAIWebClient](asposeaiwebclient#constructor)() | 建立一個連接至預設 Aspose LLM 端點的 Aspose AI 網路用戶端實例。此客戶端由無參數 [`SlidesAIAgent`](../slidesaiagent/slidesaiagent) 建構式使用，因此僅在直接將客戶端傳遞給 [`SlidesAIAgent`](../slidesaiagent/slidesaiagent) 建構式時才需要顯式建立它。 |
| [AsposeAIWebClient](asposeaiwebclient#constructor_1)(HttpClient) | 建立一個使用外部管理的 HttpClient，連接至預設 Aspose LLM 端點的 Aspose AI 網路用戶端實例。此實例不會處置所提供的 HttpClient，該物件仍由呼叫端負責管理。 |
| [AsposeAIWebClient](asposeaiwebclient#constructor_2)(string) | 建立一個連接至自訂端點 URL 的 Aspose AI 網路用戶端實例。當您擁有由 Aspose.Slides 團隊提供的 URL 時，請使用此多載；否則，請使用帶有預設 URL 的 [`AsposeAIWebClient`](./asposeaiwebclient) 多載。 |
| [AsposeAIWebClient](asposeaiwebclient#constructor_3)(string, HttpClient) | 建立一個使用外部管理的 HttpClient，連接至自訂端點 URL 的 Aspose AI 網路用戶端實例。此實例不會處置所提供的 HttpClient，該物件仍由呼叫端負責管理。當您擁有由 Aspose.Slides 團隊提供的 URL 且想自行提供 HttpClient 時，請使用此多載；若僅需在預設 URL 下使用自訂 HttpClient，請改用 [`AsposeAIWebClient`](./asposeaiwebclient) 多載。 |

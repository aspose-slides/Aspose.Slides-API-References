### 範例

將檔案儲存至 FileStream 實作範例：

```csharp
[C#]
public void Save(string path, IOutputFile outputFile)
{
    using (FileStream stream = new FileStream(path, FileMode.Create))

        outputFile.Write(stream);
    }
}
```

## IMathBox.AlignmentPoint 屬性

當為 true 時，此運算子模擬器充當對齊點；也就是說，其他方程式中指定的對齊點可以與之對齊。預設值：false

```csharp
public bool AlignmentPoint { get; set; }
```

範例：

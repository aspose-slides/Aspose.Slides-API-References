## IMathBox.NoBreak 屬性

無斷行。此屬性指定物件方框的「不可斷行」屬性。當設為 true 時，方框內不能換行。對於由多個二元運算子組成的運算子模擬器而言，這可能很重要。若未指定此元素，方框內可以換行。預設值：true

```csharp
public bool NoBreak { get; set; }
```

### 範例

範例：

```csharp
[C#]
IMathBox box = new MathematicalText("**********").ToBox();
box.NoBreak = false;
```

## IMathBox.OperatorEmulator 屬性

Operator Emulator. 當設定為 true 時，方框及其內容會視為單一運算子，並繼承運算子的屬性。這表示，例如，該字元可以作為換行點，並且可以與其他運算子對齊。當一或多個字形組合形成運算子（如 '=='）時，通常會使用 Operator Emulators。預設值: false

```csharp
public bool OperatorEmulator { get; set; }
```

### 範例

範例: ```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.OperatorEmulator = true;
```

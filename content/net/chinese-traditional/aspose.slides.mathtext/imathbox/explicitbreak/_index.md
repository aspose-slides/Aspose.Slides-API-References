## IMathBox.ExplicitBreak 屬性

ExplicitBreak 指定 Box 物件的起始位置是否有換行，使得在 Box 物件起始處換行。指定前一行數學文字中運算子之編號，該編號將用作目前行數學文字的對齊點。可能的值：1..255 預設：0（無明確換行）

```csharp
public byte ExplicitBreak { get; set; }
```

### 範例

範例：

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.ExplicitBreak = 1;
```

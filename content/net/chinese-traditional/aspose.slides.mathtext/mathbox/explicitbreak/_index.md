## MathBox.ExplicitBreak 屬性

Explicit break 指定在 Box 物件的起始處是否有換行，使得行在 Box 物件的起始處換行。指定在先前數學文字行上的運算子編號，該編號將用作目前數學文字行的對齊點。可能的值：1..255 預設：0（無顯式換行）

```csharp
public byte ExplicitBreak { get; set; }
```

### 範例

Example:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.ExplicitBreak = 1;
```

## MathBox.AlignmentPoint 屬性

當為 true 時，此運算子模擬器充當對齊點；也就是說，其他等式中指定的對齊點可以與它對齊。預設值： false

```csharp
public bool AlignmentPoint { get; set; }
```

### 範例

範例：

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.AlignmentPoint = true;
```

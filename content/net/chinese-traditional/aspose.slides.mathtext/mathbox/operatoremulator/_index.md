## MathBox.OperatorEmulator 屬性

Operator Emulator。當為 true 時，方框及其內容會作為單一運算子行為，並繼承運算子的屬性。這意味著，例如，該字元可以作為換行點，並且可以與其他運算子對齊。當一個或多個字形組合成運算子（如 '=='）時，常會使用 Operator Emulator。預設值：false

```csharp
public bool OperatorEmulator { get; set; }
```

### 範例

範例：

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("=="));
box.OperatorEmulator = true;
```

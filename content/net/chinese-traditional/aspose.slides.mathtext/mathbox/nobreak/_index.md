## MathBox.NoBreak 屬性

不分行 此屬性指定物件盒的「不可斷行」屬性。當值為 true 時，盒內不得出現換行。這對於由多個二元運算子組成的運算子模擬器非常重要。若未指定此元素，則允許在盒內換行。預設值： true

```csharp
public bool NoBreak { get; set; }
```

### 範例

範例：

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("*****"));
box.NoBreak = false;
```

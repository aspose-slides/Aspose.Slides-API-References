## MathMatrix.RowGap 屬性

矩陣行之間垂直間距的值；如果 RowGapRule 設為 3（"Exactly"），則單位解釋為 twips（1/20 點）；如果 RowGapRule 設為 4（"Multiple"），則單位解釋為半行。預設：0

```csharp
public uint RowGap { get; set; }
```

### 範例

範例：

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

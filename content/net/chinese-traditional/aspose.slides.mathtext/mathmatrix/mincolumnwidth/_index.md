## MathMatrix.MinColumnWidth 屬性

以 twips（每點的 1/20）為單位的最小列寬。間隙間距（亦稱為“Column Gap”或“Gap Width”）會加到 MinColumnWidth，以確定整體 Matrix Column Spacing（不同列之相同邊緣之間的距離）。預設值：0。

```csharp
public uint MinColumnWidth { get; set; }
```

### 範例

範例：

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.MinColumnWidth = 20;
```

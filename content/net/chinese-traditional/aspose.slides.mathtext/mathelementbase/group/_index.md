## Group(char, MathTopBotPositions, MathTopBotPositions) {#group_1}

使用分組字元（例如底部大括號或其他）將此元素放入群組

```csharp
public IMathGroupingCharacter Group(char character, MathTopBotPositions position, 
    MathTopBotPositions verticalJustification)
```

| 參數 | 類型 | 說明 |
| --- | --- | --- |
| character | Char | 用於分組的字元，例如 BOTTOM CURLY BRACKET (U+23DF) 或其他任意字元 |
| position | MathTopBotPositions | 分組字元的位置 |
| verticalJustification | MathTopBotPositions | 群組字元的垂直對齊方式。指定物件相對於基線的對齊方式。例如，當群組字元位於物件之上時，VerticalJustification 為 Top 表示物件的上方落在基線上；當 VerticalJustification 設為 Bottom 時，物件的下方落在基線上 |

### 返回值

型別 [`IMathGroupingCharacter`](../../imathgroupingcharacter) 的新執行個體

### 範例

範例：

```csharp
[C#]
IMathGroupingCharacter groupingElement = new MathematicalText("x;y;z").Group('\u23E1', MathTopBotPositions.Bottom, MathTopBotPositions.Top);
```

## MathDelimiter.GrowToMatchOperandHeight 屬性

指定 BeginningCharacter、SeparatorCharacter、EndingCharacter 的增長。當為 true 時，分隔符會垂直增長以匹配其操作數高度。預設值為 true

```csharp
public bool GrowToMatchOperandHeight { get; set; }
```

### 範例

範例：

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.GrowToMatchOperandHeight = false;
```

## MathDelimiter.DelimiterShape 屬性

指定定界符物件中定界符的形狀。當為 MathDelimiterShape.Centered 時，定界符會圍繞數學文字的數學軸置中，並且仍會調整以符合其內容的整個高度。當為 MathDelimiterShape.Match 時，會將它們的高度和形狀更改為完全匹配其內容。

```csharp
public MathDelimiterShape DelimiterShape { get; set; }
```

### 範例

範例：

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.DelimiterShape = MathDelimiterShape.Match;
```

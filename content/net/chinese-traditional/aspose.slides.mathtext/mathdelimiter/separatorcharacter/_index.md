## MathDelimiter.SeparatorCharacter 屬性

Delimiter Separator Character specifies the character that separates arguments in the delimiter object. 指定在 delimiter 物件中用來分隔參數的字元。預設值：'&#x7C;'.

```csharp
public char SeparatorCharacter { get; set; }
```

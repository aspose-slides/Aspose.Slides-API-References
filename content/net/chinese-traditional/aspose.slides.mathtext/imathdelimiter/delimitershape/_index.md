## IMathDelimiter.DelimiterShape 屬性

指定分隔符物件中分隔符的形狀。當為 MathDelimiterShape.Centered 時，分隔符會以數學文字的數學軸為中心，且仍會調整以符合其內容的整體高度。當為 MathDelimiterShape.Match 時，分隔符的高度和形狀會被改變，以精確匹配其內容。

```csharp
public MathDelimiterShape DelimiterShape { get; set; }
```

### 範例

範例：

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.DelimiterShape = MathDelimiterShape.Match;
```

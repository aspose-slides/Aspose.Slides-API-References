## IMathMatrix.RowGap 屬性

矩陣行之間的垂直間距值；如果 RowGapRule 設為 3（「精確」），則單位解讀為 twips（1/20 點）；如果 RowGapRule 設為 4（「多重」），則單位解讀為半行。預設值: 0

```csharp
public uint RowGap { get; set; }
```

### 範例

範例：

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

## IMathMatrix.MinColumnWidth 屬性

最小欄寬，以 twip 為單位 (1/20 點)。間距 (亦稱「Column Gap」或「Gap Width」) 會加到 MinColumnWidth 上，以決定整體矩陣欄間距（不同欄位相同邊緣之間的距離）。預設值：0。

```csharp
public uint MinColumnWidth { get; set; }
```

### 範例

範例：

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.MinColumnWidth = 20;
```

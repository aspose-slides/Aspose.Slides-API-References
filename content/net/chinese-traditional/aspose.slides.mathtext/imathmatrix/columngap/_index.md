## IMathMatrix.ColumnGap 屬性

矩陣中欄位之間的水平間距的值；如果 ColumnGapRule 設為 3（「Exactly」），則單位會被解釋為 twips（1/20 點）如果 ColumnGapRule 設為 4（「Multiple」），則單位會被解釋為 0.5 em 增量的數量。在其他情況下會被忽略。預設值：0

```csharp
public uint ColumnGap { get; set; }
```

### 範例

範例：

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.ColumnGapRule = MathSpacingRules.Exactly;
matrix.ColumnGap = 20;
```

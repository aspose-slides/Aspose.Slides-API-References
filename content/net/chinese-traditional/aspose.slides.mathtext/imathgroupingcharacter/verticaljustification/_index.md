## IMathGroupingCharacter.VerticalJustification 屬性

組合字元的垂直對齊方式。指定物件相對於基線的對齊方式。例如，當組合字元位於物件之上時，VerticalJustification 設為 Top 表示物件的頂部位於基線上；當 VerticalJustification 設為 Bottom 時，物件的底部位於基線上。預設值：Position=Top 時為 Bottom，Position=Bottom 時為 Top

```csharp
public MathTopBotPositions VerticalJustification { get; set; }
```

### 範例

範例：

```csharp
[C#]
MathGroupingCharacter groupingCharacter = new MathGroupingCharacter(new MathematicalText("abc"));
groupingCharacter.VerticalJustification = MathTopBotPositions.Top;
```

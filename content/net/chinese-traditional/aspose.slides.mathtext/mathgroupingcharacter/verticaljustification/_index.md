## MathGroupingCharacter.VerticalJustification 屬性

群組字元的垂直對齊方式。指定物件相對於基線的對齊方式。例如，當群組字元位於物件之上時，VerticalJustification 為 Top 表示物件的頂部位於基線上；當 VerticalJustification 設為 Bottom 時，物件的底部位於基線上。預設值：Bottom 用於 Position=Top，Top 用於 Position=Bottom

```csharp
public MathTopBotPositions VerticalJustification { get; set; }
```

### 範例

範例：

```csharp
[C#]
MathGroupingCharacter groupingCharacter = new MathGroupingCharacter(new MathematicalText("abc"));
groupingCharacter.VerticalJustification = MathTopBotPositions.Top;
```

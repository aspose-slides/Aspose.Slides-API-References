## PptOptions.RootDirectoryClsid 속성

개체 클래스 GUID(CLSID)를 나타내며, 루트 디렉터리 항목에 저장됩니다. 문서 애플리케이션의 COM 활성화에 사용할 수 있습니다. 기본 값은 '64818D11-4F9B-11CF-86EA-00AA00B929E8'이며, 이는 'Microsoft Powerpoint.Slide.8'에 해당합니다.

```csharp
public Guid RootDirectoryClsid { get; set; }
```

### 예제

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    PptOptions pptOptions = new PptOptions();
    
    /// CLSID를 'Microsoft Powerpoint.Show.8'으로 설정합니다
    pptOptions.RootDirectoryClsid = new Guid("64818D10-4F9B-11CF-86EA-00AA00B929E8");
    
    pres.Save("pres.ppt", SaveFormat.Ppt, pptOptions);
}
```

## IHtml5Options.DisableFontLigatures 속성

텍스트를 합자(ligature)를 사용하지 않고 표시할지 여부를 나타내는 값을 가져오거나 설정합니다. `true`로 설정하면 렌더링된 출력에서 합자가 비활성화됩니다. 기본적으로 이 속성은 `false`로 설정됩니다.

```csharp
public bool DisableFontLigatures { get; set; }
```

### 예제

예제:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    Html5Options options = new Html5Options
    {
        DisableFontLigatures = true // 텍스트 렌더링에서 합자 비활성화
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html5, options);
}
```

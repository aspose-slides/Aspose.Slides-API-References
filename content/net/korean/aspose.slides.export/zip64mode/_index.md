## Zip64Mode 열거형

OpenXML 파일에 ZIP64 형식 확장을 사용할 시기를 지정합니다.

```csharp
public enum Zip64Mode
```

### 값

| Name | Value | Description |
| --- | --- | --- |
| Never | `0` | ZIP64 형식 확장을 사용하지 않습니다. |
| IfNecessary | `1` | 필요할 경우 ZIP64 형식 확장을 사용합니다. |
| Always | `2` | 항상 ZIP64 형식 확장을 사용합니다. |

### 참고

OpenXML 파일은 ZIP 아카이브이며 파일의 압축되지 않은 크기, 압축된 크기 및 아카이브 전체 크기에 4 GB(2^32 바이트) 제한과 아카이브 내 파일 수가 65,535(2^16-1)개로 제한됩니다. ZIP64 형식 확장은 제한을 2^64까지 늘립니다.

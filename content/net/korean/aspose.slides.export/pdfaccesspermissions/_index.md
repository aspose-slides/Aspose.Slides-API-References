## PdfAccessPermissions 열거형

Contains a set of flags specifying which access permissions should be granted when the document is opened with user access.

```csharp
[Flags]
public enum PdfAccessPermissions
```

### 값

| 이름 | 값 | 설명 |
| --- | --- | --- |
| None | `0` | 사용자가 접근 권한을 가지고 있지 않음을 지정합니다. |
| PrintDocument | `4` | 사용자가 문서를 인쇄할 수 있는지 지정합니다(HighQualityPrint 비트가 설정되어 있는지에 따라 최고 품질이 아닐 수도 있습니다). |
| ModifyContent | `8` | 사용자가 AddOrModifyFields, FillExistingFields, AssembleDocument 비트가 제어하는 작업이 아닌 다른 작업으로 문서의 내용을 수정할 수 있는지 지정합니다. |
| CopyTextAndGraphics | `10` | 사용자가 ExtractTextAndGraphics 비트가 제어하는 작업이 아닌 다른 작업으로 문서에서 텍스트와 그래픽을 복사하거나 추출할 수 있는지 지정합니다. |
| AddOrModifyFields | `20` | 사용자가 텍스트 주석을 추가하거나 수정하고, 대화형 양식 필드를 채우며, ModifyContent 비트가 설정된 경우 대화형 양식 필드(서명 필드 포함)를 만들거나 수정할 수 있는지 지정합니다. |
| FillExistingFields | `100` | 사용자가 AddOrModifyFields 비트가 비활성화되어 있어도 기존 대화형 양식 필드(서명 필드 포함)를 채울 수 있는지 지정합니다. |
| ExtractTextAndGraphics | `200` | 사용자가 장애가 있는 사용자에 대한 접근성 지원 또는 기타 목적을 위해 텍스트와 그래픽을 추출할 수 있는지 지정합니다. |
| AssembleDocument | `400` | 사용자가 ModifyContent 비트가 비활성화된 경우에도 문서를 조합(페이지 삽입, 회전, 삭제 및 북마크나 썸네일 이미지 생성)할 수 있는지 지정합니다. |
| HighQualityPrint | `800` | 사용자가 PDF 콘텐츠의 정확한 디지털 사본을 생성할 수 있는 표현으로 문서를 인쇄할 수 있는지 지정합니다. 이 비트가 비활성화되고 PrintDocument 비트가 설정된 경우 인쇄는 외관의 저수준 표현으로 제한되며, 품질이 저하될 수 있습니다. |

## IHtmlOptions.DisableFontLigatures 속성

텍스트가 합자를 사용하지 않고 렌더링되는지 여부를 나타내는 값을 가져오거나 설정합니다. `true` 로 설정하면 렌더링된 출력에서 합자가 비활성화됩니다. 기본값은 `false` 입니다.

```csharp
public bool DisableFontLigatures { get; set; }
```

### 예제

예제:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    HtmlOptions options = new HtmlOptions
    {
        DisableFontLigatures = true // 텍스트 렌더링에서 합자를 비활성화합니다
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html, options);
}
```

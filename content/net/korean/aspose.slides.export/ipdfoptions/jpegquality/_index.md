## IPdfOptions.JpegQuality 속성

PDF 문서 내 JPEG 이미지의 품질을 결정하는 값을 반환하거나 설정합니다. Read/write Byte.

```csharp
public byte JpegQuality { get; set; }
```

### 참고

문서에 JPEG 이미지가 포함된 경우에만 영향을 미칩니다.

이 속성을 사용하여 PDF 형식으로 저장할 때 문서 내 이미지의 품질을 가져오거나 설정합니다. 값은 0부터 100까지이며, 0은 최악의 품질이지만 최대 압축을 의미하고 100은 최고의 품질이지만 최소 압축을 의미합니다.

기본값은 **100**입니다.

## SaveOptions.SkipJavaScriptLinks 속성

프레젠테이션을 저장할 때 JavaScript 호출이 포함된 하이퍼링크를 건너뛸지 여부를 지정합니다. 읽기/쓰기 Boolean. 기본값은 **false**.

```csharp
public bool SkipJavaScriptLinks { get; set; }
```

### 비고

이 속성이 **true** 로 설정되면 JavaScript 호출이 포함된 하이퍼링크는 저장 중에 무시됩니다.

이 속성이 **false** 로 설정되면 모든 하이퍼링크가 저장됩니다.

### 예제

예시:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("result_without_JavaScript_links.html", SaveFormat.Html, new HtmlOptions()
    {
        SkipJavaScriptLinks = true
    });
}
```

## MarkdownSaveOptions.SlideNumberFormat 속성

슬라이드 번호 헤더를 Markdown 출력에 사용할 형식 문자열을 가져오거나 설정합니다. 형식 문자열에는 "{0}" 자리 표시자가 포함되어야 하며, 내보내기 시 슬라이드 인덱스로 대체됩니다. 예: "# Slide {0}"은 "# Slide 1", "# Slide 2" 등으로 출력됩니다.

```csharp
public string SlideNumberFormat { get; set; }
```

### 예외

| 예외 | 조건 |

| ArgumentException | 형식 문자열에 "{0}" 자리 표시자가 포함되지 않은 경우 발생합니다. |

## PdfOptions.EmbedTrueTypeFontsForASCII 속성

이 속성은 Aspose.Slides가 ASCII(33~127 코드 범위) 텍스트에 대해 일반 글꼴을 포함시킬지 여부를 결정합니다. 127보다 큰 문자 코드에 대한 글꼴은 항상 포함됩니다. 일반 글꼴 목록에는 PDF의 기본 14글꼴과 추가 사용자 지정 글꼴이 포함됩니다. 읽기/쓰기 Boolean.

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### 비고

Default is **true**.

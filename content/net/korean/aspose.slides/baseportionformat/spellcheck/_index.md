## BasePortionFormat.SpellCheck 속성

텍스트 부분에 대해 맞춤법 검사가 활성화되어 있는지를 나타내는 값을 가져오거나 설정합니다. 이 속성이 false로 설정되면 텍스트 요소에 대한 맞춤법 검사가 억제됩니다. true로 설정하면 맞춤법 검사가 허용됩니다. 기본값은 `false` 입니다.

```csharp
public bool SpellCheck { get; set; }
```

### 예제

다음 예제에서는 프레젠테이션을 저장하기 전에 SpellCheck 플래그를 활성화하는 방법을 보여줍니다:

```csharp
[C#]
using (var pres = new Presentation("input.pptx"))
{
    // 첫 번째 슬라이드의 첫 번째 셰이프 안에 있는 첫 번째 텍스트 부분에 접근합니다
    var portion = ((AutoShape)pres.Slides[0].Shapes[0]).TextFrame.Paragraphs[0].Portions[0];
    // 이 텍스트 부분에 대해 맞춤법 검사를 활성화합니다
    portion.PortionFormat.SpellCheck = true;
    // 수정된 프레젠테이션을 저장합니다
    pres.Save("output-with-spellcheck.pptx", SaveFormat.Pptx);
}
```

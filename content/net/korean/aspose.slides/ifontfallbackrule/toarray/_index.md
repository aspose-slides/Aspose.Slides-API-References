## ToArray(int, int) {#toarray_1}

목록의 지정된 범위에서 모든 FallBack 글꼴을 포함하는 배열을 생성하고 반환합니다.

```csharp
public string[] ToArray(int startIndex, int count)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| startIndex | Int32 | 추가할 첫 번째 글꼴의 인덱스입니다. |
| count | Int32 | 추가할 글꼴의 개수입니다. |

### 반환 값

String 배열

### 예제

```csharp
[C#]
// 글꼴 목록을 포함하는 규칙을 생성합니다.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

//마지막 두 글꼴 이름을 배열로 가져옵니다
```

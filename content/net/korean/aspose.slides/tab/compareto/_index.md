## Tab.CompareTo 메서드

현재 인스턴스를 동일한 유형의 다른 객체와 비교합니다.

```csharp
public int CompareTo(object obj)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| obj | Object | 이 인스턴스와 비교할 객체. |

### 반환값

비교 대상의 상대적인 순서를 나타내는 32비트 정수. 반환값은 다음과 같은 의미를 가집니다:  
< 0 - 이 인스턴스는 obj보다 작습니다.  
= 0 - 이 인스턴스는 obj와 같습니다.  
> 0 - 이 인스턴스는 obj보다 큽니다.

## IFontsManager.GetFontBytes 메서드

지정된 글꼴 스타일 및 글꼴 데이터를 위한 글꼴 데이터를 나타내는 바이트 배열을 반환합니다.

```csharp
public byte[] GetFontBytes(IFontData fontData, FontStyleType fontStyle)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| fontData | IFontData | 폰트 데이터에 대한 정보를 포함하는 폰트 데이터 객체 [`IFontData`](../../ifontdata). |
| fontStyle | FontStyleType | 데이터를 가져올 글꼴의 스타일 [`FontStyleType`](../../fontstyletype). |

### 반환 값

지정된 글꼴 스타일에 대한 글꼴 데이터를 포함하는 바이트 배열입니다. 글꼴 데이터 또는 스타일을 찾을 수 없으면 null을 반환합니다.

### 예제

```csharp
[C#]
using (Presentation pres = new Presentation ("Presentation.pptx"))
{
    // 프레젠테이션에 사용된 모든 글꼴을 가져옵니다
    IFontData[] fonts = pres.FontsManager.GetFonts();

    // 프레젠테이션에서 첫 번째 글꼴의 일반 스타일을 나타내는 바이트 배열을 가져옵니다
    byte[] bytes = pres.FontsManager.GetFontBytes(fonts[0], FontStyleType.Regular);
}
```

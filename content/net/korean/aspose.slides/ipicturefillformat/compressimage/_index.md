## CompressImage(bool, PicturesCompression) {#compressimage}

이미지를 모양 크기와 지정된 해상도를 기준으로 크기를 줄여 압축합니다. 선택적으로, 잘린 영역도 삭제합니다.

```csharp
public bool CompressImage(bool deleteCroppedAreasOfImage, PicturesCompression resolution)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| deleteCroppedAreasOfImage | Boolean | true이면, 메서드는 이미지의 잘린 영역을 제거하여 크기를 추가로 줄일 수 있습니다. |
| resolution | PicturesCompression | 압축을 위한 목표 해상도이며, [`PicturesCompression`](../../../aspose.slides.export/picturescompression) 열거형 값으로 지정됩니다. |

### 반환 값

이미지가 성공적으로 압축되었는지 여부를 나타내는 Boolean 값입니다. 이미지가 크기가 조정되었거나 잘린 경우 `true`를 반환하고, 그렇지 않으면 `false`를 반환합니다.

### 예외

| 예외 | 조건 |
| --- | --- |
| ArgumentException | 해상도가 유효한 값이 아닌 경우 발생합니다. |

### 비고

이 메서드는 PowerPoint의 "Picture Format → Compress Pictures" 기능과 유사하게 이미지의 크기와 해상도를 변경합니다.

### 예제

다음 예제는 목표 해상도를 설정하고 잘린 영역을 제거하여 프레젠테이션의 이미지 크기를 줄이는 `CompressImage` 메서드 사용 방법을 보여줍니다:

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;
    // 목표 해상도 150 DPI (웹 해상도)로 이미지를 압축하고 잘린 영역을 제거합니다.
    bool result = picFrame.PictureFormat.CompressImage(true, PicturesCompression.Dpi150);
}
```

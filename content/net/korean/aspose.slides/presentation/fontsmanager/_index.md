## Presentation.FontsManager 속성

글꼴 관리자를 반환합니다. 읽기 전용 [`IFontsManager`](../../ifontsmanager).

```csharp
public IFontsManager FontsManager { get; }
```

### 예제

다음 예제는 PowerPoint 프레젠테이션에 임베디드 글꼴을 추가하는 방법을 보여줍니다.

```csharp
[C#]
// 프레젠테이션 로드
using (Presentation presentation = new Presentation("Fonts.pptx"))
{
	// 교체할 원본 글꼴 로드
	IFontData sourceFont = new FontData("Arial");
	IFontData[] allFonts = presentation.FontsManager.GetFonts();
	IFontData[] embeddedFonts = presentation.FontsManager.GetEmbeddedFonts();
	foreach (IFontData font in allFonts)
	{
		if (!embeddedFonts.Contains(font))
		{
			presentation.FontsManager.AddEmbeddedFont(font, EmbedFontCharacters.All);
		}
	}
	// 프레젠테이션 저장
	presentation.Save("AddEmbeddedFont_out.pptx", SaveFormat.Pptx);
}
```

## Presentation.AllCustomXmlParts 속성

프레젠테이션의 모든 사용자 정의 데이터 파트를 반환합니다. 읽기 전용 [`ICustomXmlPart`](../../icustomxmlpart)[].

```csharp
public ICustomXmlPart[] AllCustomXmlParts { get; }
```

### 예제

다음 예제는 PowerPoint 프레젠테이션에서 모든 사용자 정의 XML 파트를 지우는 방법을 보여줍니다.

```csharp
[C#]
using (Presentation pres = new Presentation("PresentationWithCustomXml.pptx"))
{
	// 모든 사용자 정의 XML 파트를 순회합니다
    foreach (ICustomXmlPart item in pres.AllCustomXmlParts)
    {
        item.Remove();
    }
    pres.Save("out.pptx", SaveFormat.Pptx);
}
```

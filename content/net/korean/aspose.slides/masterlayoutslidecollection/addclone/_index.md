## MasterLayoutSlideCollection.AddClone 메서드

지정된 레이아웃 슬라이드의 복사본을 컬렉션의 끝에 추가합니다.

```csharp
public ILayoutSlide AddClone(ILayoutSlide sourceLayout)
```

| Parameter | Type | Description |
| --- | --- | --- |
| sourceLayout | ILayoutSlide | 복제할 슬라이드. |

### 반환 값

추가된 슬라이드.

### 비고

1) 새 레이아웃은 이 레이아웃 슬라이드 컬렉션의 상위 마스터 슬라이드와 연결됩니다. 따라서 PowerPoint의 “Use Destination Theme” 옵션을 사용한 복사/붙여넣기와 동일한 동작입니다. 2) 이 메서드와 동일한 기능은 [`AddClone`](../../igloballayoutslidecollection/addclone) 메서드를 [`LayoutSlides`](../../ipresentation/layoutslides) 속성을 통해 액세스하는 것입니다.

## MasterLayoutSlideCollection.InsertClone 메서드

지정된 레이아웃 슬라이드의 복사본을 컬렉션의 지정된 위치에 삽입합니다.

```csharp
public ILayoutSlide InsertClone(int index, ILayoutSlide sourceLayout)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | Int32 | 새 슬라이드의 인덱스입니다. |
| sourceLayout | ILayoutSlide | 복제할 슬라이드입니다. |

### Return Value

삽입된 슬라이드.

### Remarks

새 레이아웃은 이 레이아웃 슬라이드 컬렉션의 상위 마스터 슬라이드와 연결됩니다. 따라서 PowerPoint의 "Use Destination Theme" 옵션을 사용한 복사/붙여넣기와 동일합니다.

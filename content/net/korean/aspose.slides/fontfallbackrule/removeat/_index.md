## FontFallBackRule.RemoveAt 메서드

리스트에서 지정된 인덱스에 있는 FallBack 폰트를 제거합니다.

```csharp
public void RemoveAt(int index)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| index | Int32 | 제거할 폰트의 0부터 시작하는 인덱스입니다. |

### 예제

```csharp
[C#]
// 글꼴 목록을 포함하는 규칙을 생성합니다.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

//목록에서 Tahoma를 제거합니다.
newRule.Remove (2);
```

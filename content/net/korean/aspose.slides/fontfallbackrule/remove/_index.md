## FontFallBackRule.Remove 메서드

목록에서 특정 FallBack 폰트의 첫 번째 발생을 제거합니다.

```csharp

```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| fontName | String | 목록에서 제거할 폰트의 이름 |

### 예제

```csharp
[C#]
// 폰트 목록을 포함하는 규칙을 생성합니다.
```

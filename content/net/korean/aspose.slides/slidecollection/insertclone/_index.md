## InsertClone(int, ISlide) {#insertclone}

지정된 슬라이드의 복사본을 컬렉션의 지정된 위치에 삽입합니다.

```csharp
public ISlide InsertClone(int index, ISlide sourceSlide)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| index | Int32 | 새 슬라이드의 인덱스. |
| sourceSlide | ISlide | 복제할 슬라이드. |

### 반환값

삽입된 슬라이드.

### 비고

다른 프레젠테이션 간에 슬라이드를 복제할 때 슬라이드의 마스터도 복제될 수 있습니다. 내부 레지스트리를 사용하여 자동으로 복제된 마스터를 추적하고 동일한 마스터 슬라이드의 복제본이 여러 개 생성되는 것을 방지합니다. 마스터 슬라이드의 수동 복제는 방지되지 않으며 등록되지도 않습니다. 복제 프로세스를 보다 정밀하게 제어하려면 슬라이드 복제에 [`InsertClone`](../insertclone) 또는 [`InsertClone`](../insertclone) 를, 마스터 복제에 [`AddClone`](../../imasterslidecollection/addclone) 를 사용하십시오.

### 예제

```csharp
[C#]
// 프레젠테이션 파일을 나타내는 Presentation 클래스를 인스턴스화합니다
using (Presentation pres = new Presentation("CloneWithInSamePresentation.pptx"))
{
    // 같은 프레젠테이션의 슬라이드 컬렉션 끝에 원하는 슬라이드를 복제합니다
    ISlideCollection slds = pres.Slides;
    // 같은 프레젠테이션에서 지정된 인덱스로 원하는 슬라이드를 복제합니다
    slds.InsertClone(2, pres.Slides[1]);
    // 수정된 프레젠테이션을 디스크에 저장합니다
    pres.Save("Aspose_CloneWithInSamePresentation_out.pptx", SaveFormat.Pptx);
}
```

다음 예제는 프레젠테이션 내에서 다른 위치에 복제하는 방법을 보여줍니다.

```csharp
[C#]
// 소스 프레젠테이션 파일을 로드하기 위해 Presentation 클래스를 인스턴스화합니다
using (Presentation srcPres = new Presentation("CloneAtEndOfAnother.pptx"))
{
    // 슬라이드가 복제될 대상 PPTX를 위해 Presentation 클래스를 인스턴스화합니다
    using (Presentation destPres = new Presentation())
    {
        ISlideCollection slds = destPres.Slides;
        slds.InsertClone(2, srcPres.Slides[0]);
        // 대상 프레젠테이션을 디스크에 저장합니다
        destPres.Save("Aspose2_out.pptx", SaveFormat.Pptx);
    }
}
```

다음 예제는 프레젠테이션 내에서 다른 위치에 복제하는 방법을 보여줍니다.

## InsertClone(int, ISlide, IMasterSlide, bool) {#insertclone_2}

지정된 원본 슬라이드의 복사본을 컬렉션의 지정된 위치에 삽입합니다. 지정된 마스터에서 적절한 레이아웃이 자동으로 선택됩니다(적절한 레이아웃은 원본 슬라이드 레이아웃과 동일한 Type 또는 Name을 가진 레이아웃입니다). 적절한 레이아웃이 없으면 원본 슬라이드의 레이아웃이 복제됩니다(allowCloneMissingLayout가 true인 경우) 또는 allowCloneMissingLayout가 false인 경우 PptxEditException이 발생합니다.

```csharp
public ISlide InsertClone(int index, ISlide sourceSlide, IMasterSlide destMaster, 
    bool allowCloneMissingLayout)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| index | Int32 | 새 슬라이드의 인덱스. |
| sourceSlide | ISlide | 복제할 슬라이드. |
| destMaster | IMasterSlide | 새 슬라이드용 마스터 슬라이드. |
| allowCloneMissingLayout | Boolean | 지정된 마스터에 적절한 레이아웃이 없으면 원본 슬라이드의 레이아웃이 복제됩니다(allowCloneMissingLayout가 true인 경우) 또는 allowCloneMissingLayout가 false인 경우 PptxEditException이 발생합니다. |

### 반환값

삽입된 슬라이드.

### 예외

| 예외 | 조건 |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | 지정된 마스터에 적절한 레이아웃이 없고 allowCloneMissingLayout가 false인 경우 발생합니다. |

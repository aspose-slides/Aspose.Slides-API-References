
| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| sourceSlide | ISlide | 복제할 슬라이드. |

### 반환 값

### 비고

다른 프레젠테이션 간에 슬라이드를 복제할 때 슬라이드의 마스터도 복제될 수 있습니다. 내부 레지스트리는 자동으로 복제된 마스터를 추적하여 동일한 마스터 슬라이드의 복제본이 여러 개 생성되는 것을 방지합니다. 마스터 슬라이드의 수동 복제는 방지되거나 등록되지 않습니다. 복제 프로세스를 더 세밀하게 제어해야 하는 경우 슬라이드 복제에는 [`AddClone`](../addclone) 또는 [`AddClone`](../addclone)을, 레이아웃 복제에는 [`AddClone`](../../igloballayoutslidecollection/addclone) 또는 [`AddClone`](../../igloballayoutslidecollection/addclone)을, 마스터 복제에는 [`AddClone`](../../imasterslidecollection/addclone)을 사용하십시오.

### 예외

| 예외 | 조건 |
| --- | --- |
| ArgumentNullException |  |
| [PptxEditException](../../pptxeditexception) |  |

### 예제

```csharp
[C#]
using (IPresentation presentation = new Presentation())
{
    presentation.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 200, 50, 300, 100);
    presentation.Sections.AddSection("Section 1", presentation.Slides[0]);
    
    ISection section2 = presentation.Sections.AppendEmptySection("Section 2");
    presentation.Slides.AddClone(presentation.Slides[0], section2);
    
    // 이제 두 번째 섹션에 첫 번째 슬라이드의 복사본이 포함됩니다.
}
```

## AddClone(ISlide, IMasterSlide, bool) {#addclone_2}

지정된 원본 슬라이드의 복사본을 컬렉션의 끝에 추가합니다. 지정된 마스터에서 적절한 레이아웃이 자동으로 선택됩니다(적절한 레이아웃은 원본 슬라이드 레이아웃과 동일한 Type 또는 Name을 가진 레이아웃입니다). 적절한 레이아웃이 없는 경우 원본 슬라이드의 레이아웃이 복제됩니다(allowCloneMissingLayout가 true인 경우) 또는 allowCloneMissingLayout가 false인 경우 PptxEditException이 발생합니다.

```csharp
public ISlide AddClone(ISlide sourceSlide, IMasterSlide destMaster, bool allowCloneMissingLayout)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| sourceSlide | ISlide | 복제할 슬라이드. |
| destMaster | IMasterSlide | 새 슬라이드의 마스터 슬라이드. |
| allowCloneMissingLayout | Boolean | 지정된 마스터에 적절한 레이아웃이 없을 경우 원본 슬라이드 레이아웃이 복제됩니다(allowCloneMissingLayout가 true인 경우) 또는 allowCloneMissingLayout가 false인 경우 PptxEditException이 발생합니다. |

### 반환 값

새 슬라이드.

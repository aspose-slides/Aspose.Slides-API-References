## EmbeddingLevel 열거형

글꼴을 임베드하기 위한 라이선스 권한을 나타냅니다.

```csharp
[Flags]
public enum EmbeddingLevel : ushort
```

### 값

| Name | Value | Description |
| --- | --- | --- |
| Installable | `0` | 이 설정이 지정된 글꼴은 애플리케이션에 의해 원격 시스템에 임베드되고 영구적으로 설치될 수 있음을 나타냅니다. 원격 시스템의 사용자는 해당 글꼴에 대해 원본 구매자와 동일한 권리, 의무 및 라이선스를 획득하며, 원본 구매자와 동일한 최종 사용자 라이선스 계약, 저작권, 디자인 특허 및/또는 상표에 적용됩니다. |
| Restricted | `2` | 이 비트만 설정된 글꼴은 법적 소유자의 허가를 먼저 얻지 않고는 어떠한 방식으로도 수정, 임베드 또는 교환해서는 안 됩니다. |
| PreviewPrint | `4` | 이 비트가 설정되면 글꼴을 임베드할 수 있으며 원격 시스템에 일시적으로 로드됩니다. Preview &amp; Print 글꼴을 포함한 문서는 "read-only"로 열어야 하며 문서에 편집을 적용할 수 없습니다. |
| Editable | `8` | 이 비트가 설정되면 글꼴을 임베드할 수 있지만 다른 시스템에 일시적으로만 설치해야 합니다. Preview &amp; Print 글꼴과 달리 Editable 글꼴을 포함한 문서는 읽기 위해 열 수 있으며 편집이 허용되고 변경 사항을 저장할 수 있습니다. |
| NoSubsetting | `100` | 이 비트가 설정되면 글꼴을 임베드하기 전에 서브셋팅할 수 없습니다. 비트 0-3 및 9에 지정된 다른 임베드 제한도 적용됩니다. |
| BitmapOnly | `200` | 이 비트가 설정되면 글꼴에 포함된 비트맵만 임베드할 수 있습니다. 아웃라인 데이터는 임베드할 수 없습니다. 글꼴에 비트맵이 전혀 없으면 해당 글꼴은 임베드 불가능한 것으로 간주되며 임베드 서비스가 실패합니다. |

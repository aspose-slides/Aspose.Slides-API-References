## TextFrameFormat.RotationAngle 속성

바운딩 박스 내부 텍스트에 적용되는 사용자 지정 회전을 지정합니다. 지정되지 않은 경우, 함께 있는 도형의 회전이 사용됩니다. 지정된 경우에는 도형과 독립적으로 적용됩니다. 즉, 도형에 회전이 적용될 수 있으며 텍스트 자체에도 회전이 별도로 적용될 수 있습니다. 이 속성 및 속성 TextVerticalType에 정의된 사전 정의 수직 유형에서 요약된 시각적 텍스트 회전 값이 결과로 제공됩니다. 읽기/쓰기 Single.

```csharp
public float RotationAngle { get; set; }
```

### 예제

도형에 시계 방향으로 90도 회전이 적용된 경우를 고려해 보세요. 여기에 텍스트 본문 자체에도 반시계 방향으로 -90도 회전이 적용됩니다. 이렇게 되면 결과 도형은 회전된 것처럼 보이지만, 그 안의 텍스트는 전혀 회전되지 않은 것처럼 보입니다.

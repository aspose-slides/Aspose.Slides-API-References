## IDocumentProperties.AppVersion 속성

앱 버전을 반환합니다. 읽기 전용 String.

```csharp
public string AppVersion { get; }
```

### 비고

이 요소의 내용은 X와 Y가 숫자 값을 나타내는 형식 XX.YYYY이어야 하며, 그렇지 않을 경우 문서는 비준수로 간주됩니다. Aspose.Slides는 버전을 XX.YYZZ 형식으로 나타내며, 여기서:  
XX - 주 버전  
YY - 부 버전  
ZZ - 패치 버전  
예를 들어, 값 23.0105는 Aspose.Slides 버전 23.1.5를 의미합니다.

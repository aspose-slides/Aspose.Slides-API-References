## Portion.PortionFormat 속성

텍스트 부분에 대해 명시적으로 설정된 서식 속성을 포함하고 상속이 적용되지 않은 서식 개체를 반환합니다. 읽기 전용 [`IPortionFormat`](../../iportionformat).

```csharp
public IPortionFormat PortionFormat { get; }
```

### 비고

서식 개체는 현재 부분에 대해 정의된 서식 매개변수만 포함하며, 상속된 데이터는 적용되지 않습니다.

상속된 값을 포함한 실제 값을 얻으려면 [`GetEffective`](../../portionformat/geteffective) 메서드를 사용하십시오.

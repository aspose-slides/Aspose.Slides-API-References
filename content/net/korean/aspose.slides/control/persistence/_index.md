ActiveX 컨트롤의 속성을 저장하는 데 사용되는 메서드를 가져옵니다. 읽기 전용 [`PersistenceType`](../../persistencetype).

```csharp
public PersistenceType Persistence { get; }
```

### 예제

다음 예제는 Persistence 속성을 사용하여 ActiveX 객체의 속성을 XML 기반 ActiveX 속성으로 변경할 수 있는지 확인하는 방법을 보여줍니다:

```csharp
[C#]
if (control.Persistence == PersistenceType.PersistPropertyBag)
{
    control.Properties["Value"] = value;
}
else
{
    YourMethodHere(control.ActiveXControlBinary); //ActiveX 속성을 바이너리 파일에 저장된 상태로 관리하기 위해 자신의 메서드를 사용하십시오
}
```

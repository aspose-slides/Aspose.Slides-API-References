## Slide.WriteAsEmf 메서드

슬라이드 콘텐츠를 EMF 파일로 저장합니다.

```csharp
public void WriteAsEmf(Stream stream)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| stream | Stream | 대상 스트림 |

### 예외

| 예외 | 조건 |
| --- | --- |
| ArgumentNullException | 대상 스트림이 `null`입니다 |

### 예시

다음 코드 예제는 PowerPoint 프레젠테이션의 첫 번째 슬라이드를 메타파일로 변환하는 방법을 보여줍니다.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.emf"))
    {
        // 첫 번째 슬라이드를 메타파일로 저장합니다
        pres.Slides[0].WriteAsEmf(fileStream);
    }
}
```

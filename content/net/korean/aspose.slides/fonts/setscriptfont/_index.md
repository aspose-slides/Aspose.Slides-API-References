## Fonts.SetScriptFont 메서드

특정 스크립트 태그에 폰트 이름을 할당하여 해당 스크립트의 텍스트가 프레젠테이션에서 렌더링되는 방식을 정의합니다.

```csharp
public void SetScriptFont(string script, string fontName)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| script | String | BCP-47 스크립트 코드(예: "Arab", "Hebr", "Hans")로, 쓰기 시스템을 식별합니다. |
| fontName | String | 지정된 스크립트에 할당할 폰트 이름입니다. |

### 예제

이 예제는 아랍어 스크립트의 폰트를 "Segoe UI"로 설정하는 방법을 보여줍니다:

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segoe UI");
```

## ILoadOptions.DeleteEmbeddedBinaryObjects 속성

프레젠테이션을 로드하는 동안 Aspose.Slides가 모든 임베디드 바이너리 객체를 삭제할지 여부를 결정합니다.

임베디드 바이너리 객체의 유형:

* VBA Project [`VbaProject`](../../ipresentation/vbaproject)
* OLE Object 임베디드 데이터 [`EmbeddedFileData`](../../ioleembeddeddatainfo/embeddedfiledata)
* ActiveX Control 바이너리 데이터 [`ActiveXControlBinary`](../../icontrol/activexcontrolbinary)

읽기/쓰기 Boolean.

```csharp
public bool DeleteEmbeddedBinaryObjects { get; set; }
```

### 비고

기본값은 **false**입니다.

### 예제

다음 예제는 임베디드 바이너리 객체 없이 프레젠테이션을 로드하는 방법을 보여줍니다.

```csharp
[C#]
LoadOptions loadOptions = new LoadOptions();
loadOptions.DeleteEmbeddedBinaryObjects = true;
using (Presentation pres = new Presentation("pres.ppt", loadOptions))
{
    pres.Save("output_WithoutBinaryObjects.ppt", SaveFormat.Ppt);
}
```

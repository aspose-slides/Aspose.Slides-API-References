## ILoadOptions.OnlyLoadDocumentProperties 속성

프레젠테이션 파일이 비밀번호로 보호된 경우에 이 속성이 의미가 있습니다. true 값을 지정하면 암호화된 프레젠테이션 파일에서 문서 속성만 로드되고 비밀번호는 무시됩니다. false 값을 지정하면 올바른 비밀번호를 사용하여 전체 암호화된 프레젠테이션을 로드해야 합니다. 프레젠테이션이 암호화되지 않은 경우 이 속성 값은 항상 무시됩니다. 암호화된 파일의 문서 속성이 공개되지 않았고 속성 값이 true인 경우 문서 속성을 로드할 수 없으며 예외가 발생합니다. 읽기-쓰기 Boolean.

```csharp
public bool OnlyLoadDocumentProperties { get; set; }
```

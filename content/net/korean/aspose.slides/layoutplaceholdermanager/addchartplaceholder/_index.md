## LayoutPlaceholderManager.AddChartPlaceholder 메서드

새 차트를 포함하기 위해 레이아웃 슬라이드에 새 자리 표시자 모양을 추가합니다.

```csharp
public IAutoShape AddChartPlaceholder(float x, float y, float width, float height)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| x | Single | 새 자리 표시자 모양의 X 좌표입니다. |
| y | Single | 새 자리 표시자 모양의 Y 좌표입니다. |
| width | Single | 새 자리 표시자 모양의 너비입니다. |
| height | Single | 새 자리 표시자 모양의 높이입니다. |

### 반환값

[`IAutoShape`](../../iautoshape)을(를) Chart 자리 표시자와 함께 생성했습니다.

### 예제

다음 예제는 레이아웃 슬라이드에 Chart 자리 표시자 모양을 추가하는 방법을 보여줍니다.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddChartPlaceholder(20, 20, 200, 200);
}
```

## LayoutPlaceholderManager.AddSmartArtPlaceholder 메서드

레이아웃 슬라이드에 SmartArt 다이어그램을 담을 새로운 플레이스홀더 모양을 추가합니다.

```csharp
public IAutoShape AddSmartArtPlaceholder(float x, float y, float width, float height)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| x | Single | 새 플레이스홀더 모양의 X 좌표. |
| y | Single | 새 플레이스홀더 모양의 Y 좌표. |
| width | Single | 새 플레이스홀더 모양의 너비. |
| height | Single | 새 플레이스홀더 모양의 높이. |

### 반환값

[`IAutoShape`](../../iautoshape)를 SmartArt 플레이스홀더와 함께 생성했습니다.

### 예제

다음 예제는 SmartArt 플레이스홀더 모양을 레이아웃 슬라이드에 추가하는 방법을 보여줍니다.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddSmartArtPlaceholder(20, 20, 200, 200);
}
```

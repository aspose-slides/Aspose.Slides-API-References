## LayoutPlaceholderManager.AddVerticalTextPlaceholder method

새 레이아웃 슬라이드에 텍스트 내용을 수직 방향으로 보관하기 위한 새 플레이스홀더 도형을 추가합니다.

```csharp
public IAutoShape AddVerticalTextPlaceholder(float x, float y, float width, float height)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| x | Single | 새 플레이스홀더 도형의 X 좌표입니다. |
| y | Single | 새 플레이스홀더 도형의 Y 좌표입니다. |
| width | Single | 새 플레이스홀더 도형의 너비입니다. |
| height | Single | 새 플레이스홀더 도형의 높이입니다. |

### 반환값

[`IAutoShape`](../../iautoshape)를 사용하여 Text (Vertical) 플레이스홀더를 생성했습니다.

### 예제

다음 예제는 레이아웃 슬라이드에 Text (Vertical) 플레이스홀더 도형을 추가하는 방법을 보여줍니다.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalTextPlaceholder(20, 20, 300, 500);
}
```

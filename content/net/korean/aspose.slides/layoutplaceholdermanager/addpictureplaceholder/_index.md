## LayoutPlaceholderManager.AddPicturePlaceholder 메서드

레이아웃 슬라이드에 그림을 보관할 새 자리표시자 도형을 추가합니다.

```csharp
public IAutoShape AddPicturePlaceholder(float x, float y, float width, float height)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| x | Single | 새 자리표시자 도형의 X 좌표. |
| y | Single | 새 자리표시자 도형의 Y 좌표. |
| width | Single | 새 자리표시자 도형의 너비. |
| height | Single | 새 자리표시자 도형의 높이. |

### 반환 값

[`IAutoShape`](../../iautoshape)와 함께 Picture 자리표시자를 생성했습니다.

### 예제

다음 예제는 레이아웃 슬라이드에 Picture 자리표시자 도형을 추가하는 방법을 보여줍니다.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddPicturePlaceholder(20, 20, 200, 200);
}
```

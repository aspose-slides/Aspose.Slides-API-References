## LoadOptions.DeleteEmbeddedBinaryObjects 속성

프레젠테이션을 로드하는 동안 Aspose.Slides가 모든 포함된 바이너리 개체를 삭제할지 여부를 결정합니다.

포함된 바이너리 개체의 유형:

* VBA 프로젝트 [`VbaProject`](../../ipresentation/vbaproject)
* OLE 객체 포함 데이터 [`EmbeddedFileData`](../../ioleembeddeddatainfo/embeddedfiledata)
* ActiveX 컨트롤 바이너리 데이터 [`ActiveXControlBinary`](../../icontrol/activexcontrolbinary)

읽기/쓰기 Boolean.

```csharp
public bool DeleteEmbeddedBinaryObjects { get; set; }
```

### 비고

기본값은 **false**입니다.

### 예제

다음 예제는 포함된 바이너리 개체 없이 프레젠테이션을 로드하는 방법을 보여줍니다.

```csharp
[C#]
LoadOptions loadOptions = new LoadOptions();
loadOptions.DeleteEmbeddedBinaryObjects = true;
using (Presentation pres = new Presentation("pres.ppt", loadOptions))
{
    pres.Save("output_WithoutBinaryObjects.ppt", SaveFormat.Ppt);
}
```

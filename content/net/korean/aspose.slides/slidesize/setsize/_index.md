## SetSize(SlideSizeType, SlideSizeScaleType) {#setsize}

유형에 따라 슬라이드 크기를 설정하고 기존 콘텐츠를 확대/축소합니다.

```csharp
public void SetSize(SlideSizeType type, SlideSizeScaleType scaleType)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| type | SlideSizeType | 적용할 미리 정의된 슬라이드 크기입니다. |
| scaleType | SlideSizeScaleType | 사용할 콘텐츠 확대/축소 모드입니다. |

## SetSize(float, float, SlideSizeScaleType) {#setsize_1}

슬라이드 크기를 명시적으로 설정하고 기존 콘텐츠를 확대/축소합니다.

```csharp
public void SetSize(float width, float height, SlideSizeScaleType scaleType)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| width | Single | 새 슬라이드 너비(포인트 단위)입니다. |
| height | Single | 새 슬라이드 높이(포인트 단위)입니다. |
| scaleType | SlideSizeScaleType | 사용할 콘텐츠 확대/축소 모드입니다. |

### 비고

이것은 [`Type`](../type) 속성을 Custom으로 재설정하고 [`Orientation`](../orientation)를 설정합니다.

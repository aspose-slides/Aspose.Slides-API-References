## AddConnector(ShapeType, float, float, float, float, bool) {#addconnector_1}

새 연결자 모양을 생성하고 기본 템플릿 스타일링을 선택적으로 적용하여 모양 컬렉션의 끝에 추가합니다.

```csharp
public IConnector AddConnector(ShapeType shapeType, float x, float y, float width, float height, 
    bool createFromTemplate)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| shapeType | ShapeType | 생성할 연결자 모양의 [`ShapeType`](../../shapetype). |
| x | Single | 연결자 프레임의 x 좌표(포인트 단위). |
| y | Single | 연결자 프레임의 y 좌표(포인트 단위). |
| width | Single | 연결자 프레임의 너비(포인트 단위). |
| height | Single | 연결자 프레임의 높이(포인트 단위). |
| createFromTemplate | Boolean | True이면 기본 템플릿 스타일링(이름 비어 있지 않음, 간단한 스타일)을 적용하고, false이면 기본 속성 값으로 연결자를 생성합니다. |

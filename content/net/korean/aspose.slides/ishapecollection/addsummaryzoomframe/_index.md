## IShapeCollection.AddSummaryZoomFrame 메서드

새 Summary Zoom 프레임을 생성하고 shape 컬렉션의 끝에 추가합니다.

```csharp
public ISummaryZoomFrame AddSummaryZoomFrame(float x, float y, float width, float height)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| x | Single | 새 Summary Zoom 프레임의 x 좌표(포인트) |
| y | Single | 새 Summary Zoom 프레임의 y 좌표(포인트) |
| width | Single | 새 Summary Zoom 프레임의 너비(포인트) |
| height | Single | 새 Summary Zoom 프레임의 높이(포인트) |

### 반환값

새로 생성된 [`ISummaryZoomFrame`](../../isummaryzoomframe).

### 예외

| 예외 | 조건 |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | 프레젠테이션에 섹션이 없거나 대상 슬라이드가 어떤 섹션에도 속하지 않을 경우 발생합니다. |

### 비고

이 메서드는 프레젠테이션의 모든 섹션에 대한 요약 링크를 집계하는 Summary Zoom 프레임을 생성합니다.

### 예제

이 예제는 컬렉션 끝에 Summary Zoom 객체를 추가하는 방법을 보여줍니다(예: "Presentation.pptx" 프레젠테이션에 최소 두 개의 섹션이 있다고 가정).

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSummaryZoomFrame(150, 20, 500, 250);
}
```

## AddAutoShape(ShapeType, float, float, float, float, bool) {#addautoshape_1}

새 자동 도형을 생성하고 도형 컬렉션 끝에 추가합니다. 선택적으로 기본 템플릿 서식을 적용하여 초기화할 수 있습니다.

```csharp
public IAutoShape AddAutoShape(ShapeType shapeType, float x, float y, float width, float height, 
    bool createFromTemplate)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| shapeType | ShapeType | 추가할 자동 도형의 [`ShapeType`](../../shapetype). |
| x | Single | 도형 프레임의 x 좌표(포인트 단위). |
| y | Single | 도형 프레임의 y 좌표(포인트 단위). |
| width | Single | 도형 프레임의 너비(포인트 단위). |
| height | Single | 도형 프레임의 높이(포인트 단위). |
| createFromTemplate | Boolean | true이면 새 도형에 기본 템플릿 스타일(단순 스타일, 텍스트 가운데 정렬, 비어 있지 않은 이름)을 적용합니다; false이면 모든 속성이 기본값으로 설정된 상태로 도형을 생성합니다. |

### 반환값

새로 생성된 [`IAutoShape`](../../iautoshape).

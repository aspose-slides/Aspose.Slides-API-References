## IShapeCollection.InsertPictureFrame 메서드

지정된 이미지를 포함하는 새 그림 프레임을 생성하고, 지정된 인덱스에 쉐이프 컬렉션에 삽입합니다.

```csharp
public IPictureFrame InsertPictureFrame(int index, ShapeType shapeType, float x, float y, 
    float width, float height, IPPImage image)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| index | Int32 | 그림 프레임을 삽입할 0부터 시작하는 인덱스. |
| shapeType | ShapeType | [`ShapeType`](../../shapetype)에 포함된 쉐이프 유형을 지정합니다. 단, 모든 종류의 선은 제외합니다: |
| x | Single | 그림 프레임의 x 좌표(단위: 포인트). |
| y | Single | 그림 프레임의 y 좌표(단위: 포인트). |
| width | Single | 그림 프레임의 너비(단위: 포인트). |
| height | Single | 그림 프레임의 높이(단위: 포인트). |
| image | IPPImage | 그림 프레임에 표시할 [`IPPImage`](../../ippimage). |

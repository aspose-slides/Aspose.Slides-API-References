## IShapeCollection.InsertSummaryZoomFrame 메서드

새로운 Summary Zoom 프레임을 생성하고 지정된 인덱스에서 shape collection에 삽입합니다.

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| index | Int32 | 요약 확대 프레임을 삽입할 0부터 시작하는 인덱스입니다. |
| x | Single | 새 요약 확대 프레임의 x 좌표(포인트 단위)입니다. |
| y | Single | 새 요약 확대 프레임의 y 좌표(포인트 단위)입니다. |
| width | Single | 새 요약 확대 프레임의 너비(포인트 단위)입니다. |
| height | Single | 새 요약 확대 프레임의 높이(포인트 단위)입니다. |

### 반환값

새로 생성된 [`ISummaryZoomFrame`](../../isummaryzoomframe).

### 예외

| 예외 | 조건 |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | 프레젠테이션에 섹션이 없거나 대상 슬라이드가 어떤 섹션에도 속하지 않을 경우 발생합니다. |

### 비고

이 메서드는 프레젠테이션의 모든 섹션에 대한 요약 링크를 집계하는 Summary Zoom 프레임을 생성합니다.

### 예제

이 예제는 컬렉션의 지정된 인덱스에 Summary Zoom 개체를 생성하고 삽입하는 방법을 보여줍니다(“Presentation.pptx” 프레젠테이션에 섹션이 두 개 이상 있다고 가정합니다):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSummaryZoomFrame(2, 150, 20, 50, 50);
}
```

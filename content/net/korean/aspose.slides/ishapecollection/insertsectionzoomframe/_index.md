## InsertSectionZoomFrame(int, float, float, float, float, ISection) {#insertsectionzoomframe}

새로운 Section Zoom 프레임을 생성하고 지정된 인덱스에 shape 컬렉션에 삽입합니다.

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| index | Int32 | Section Zoom 프레임을 삽입할 0부터 시작하는 인덱스입니다. |
| x | Single | 새 Section Zoom 프레임의 x 좌표이며, 단위는 포인트입니다. |
| y | Single | 새 Section Zoom 프레임의 y 좌표이며, 단위는 포인트입니다. |
| width | Single | 새 Section Zoom 프레임의 너비이며, 단위는 포인트입니다. |
| height | Single | 새 Section Zoom 프레임의 높이이며, 단위는 포인트입니다. |
| section | ISection | Section Zoom 프레임이 참조하는 [`ISection`](../../isection)이며, 현재 프레젠테이션에 속하고 최소 한 개의 슬라이드를 포함해야 합니다. |

### 반환값

새로 생성된 [`ISectionZoomFrame`](../../isectionzoomframe)입니다.

### 예외

| 예외 | 조건 |
| --- | --- |
| ArgumentException | 참조된 섹션이 현재 프레젠테이션에 속하지 않거나 슬라이드가 없는 경우 발생합니다. |

### 예제

이 예제는 컬렉션의 지정된 인덱스에 Section Zoom 객체를 생성하고 삽입하는 방법을 보여줍니다 ("Presentation.pptx" 프레젠테이션에 섹션이 최소 두 개 있다고 가정).

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSectionZoomFrame(2, 150, 20, 50, 50, pres.Sections[1]);
}
```

## InsertSectionZoomFrame(int, float, float, float, float, ISection, IPPImage) {#insertsectionzoomframe_1}

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section, IPPImage image)

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| index | Int32 | Section Zoom 프레임을 삽입할 0부터 시작하는 인덱스입니다. |
| x | Single | 새 Section Zoom 프레임의 x 좌표이며, 단위는 포인트입니다. |
| y | Single | 새 Section Zoom 프레임의 y 좌표이며, 단위는 포인트입니다. |
| width | Single | 새 Section Zoom 프레임의 너비이며, 단위는 포인트입니다. |
| height | Single | 새 Section Zoom 프레임의 높이이며, 단위는 포인트입니다. |

### 반환값

}
```

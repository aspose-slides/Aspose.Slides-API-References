## InsertAudioFrameEmbedded(int, float, float, float, float, Stream) {#insertaudioframeembedded_1}

새 오디오 프레임을 생성하고, 포함된 WAV 파일을 내장한 후 지정된 인덱스에 shape 컬렉션에 삽입합니다. 내장된 오디오는 Presentation.Audios 컬렉션에 추가됩니다.

```csharp
public IAudioFrame InsertAudioFrameEmbedded(int index, float x, float y, float width, float height, 
    Stream audio_stream)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| index | Int32 | 삽입할 오디오 프레임의 0부터 시작하는 인덱스입니다. |
| x | Single | 새 오디오 프레임의 x좌표(포인트)입니다. |
| y | Single | 새 오디오 프레임의 y좌표(포인트)입니다. |
| width | Single | 새 오디오 프레임의 너비(포인트)입니다. |
| height | Single | 새 오디오 프레임의 높이(포인트)입니다. |
| audio_stream | Stream | 삽입할 WAV 오디오 데이터를 포함하는 입력 스트림입니다. |

### 반환값

새로 생성된 [`IAudioFrame`](../../iaudioframe).

    IAudio audio)

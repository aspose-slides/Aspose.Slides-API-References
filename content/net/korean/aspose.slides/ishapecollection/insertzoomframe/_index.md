## InsertZoomFrame(int, float, float, float, float, ISlide) {#insertzoomframe}

새로운 Zoom 프레임을 생성하고 지정된 인덱스에 shape 컬렉션에 삽입합니다.

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| index | Int32 | Zoom 프레임을 삽입할 0부터 시작하는 인덱스입니다. |
| x | Single | 새로운 Zoom 프레임의 x 좌표(포인트 단위)입니다. |
| y | Single | 새로운 Zoom 프레임의 y 좌표(포인트 단위)입니다. |
| width | Single | 새로운 Zoom 프레임의 너비(포인트 단위)입니다. |
| height | Single | 새로운 Zoom 프레임의 높이(포인트 단위)입니다. |
| slide | ISlide | Zoom 프레임이 참조하는 [`ISlide`](../../islide). |

### 반환값

새로 생성된 [`IZoomFrame`](../../izoomframe).

### 예외

| 예외 | 조건 |
| --- | --- |
| ArgumentException | 참조된 슬라이드가 현재 프레젠테이션에 속하지 않을 경우 발생합니다. |

### 예제

이 예제는 컬렉션의 지정된 인덱스에 Zoom 객체를 생성하고 삽입하는 방법을 보여줍니다(예시 프레젠테이션인 "Presentation.pptx"에 슬라이드가 최소 두 개 있다고 가정합니다):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertZoomFrame(2, 150, 20, 50, 50, pres.Slides[1]);
}
```

## InsertZoomFrame(int, float, float, float, float, ISlide, IPPImage) {#insertzoomframe_1}

미리 정의된 이미지를 사용하여 새로운 Zoom 프레임을 생성하고 지정된 인덱스에 shape 컬렉션에 삽입합니다.

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide, IPPImage image)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| index | Int32 | Zoom 프레임을 삽입할 0부터 시작하는 인덱스입니다. |
| x | Single | 새로운 Zoom 프레임의 x 좌표(포인트 단위)입니다. |
| y | Single | 새로운 Zoom 프레임의 y 좌표(포인트 단위)입니다. |
| width | Single | 새로운 Zoom 프레임의 너비(포인트 단위)입니다. |
| height | Single | 새로운 Zoom 프레임의 높이(포인트 단위)입니다. |
| slide | ISlide | Zoom 프레임이 참조하는 [`ISlide`](../../islide). |
| image | IPPImage | 참조된 슬라이드 [`IPPImage`](../../ippimage)에 대한 이미지입니다. |

### 반환값

새로 생성된 [`IZoomFrame`](../../izoomframe).

### 예외

| 예외 | 조건 |
| --- | --- |
| ArgumentException | 참조된 슬라이드가 현재 프레젠테이션에 속하지 않을 경우 발생합니다. |

### 예제

이 예제는 컬렉션의 지정된 인덱스에 Zoom 객체를 생성하고 삽입하는 방법을 보여줍니다(예시 프레젠테이션인 "Presentation.pptx"에 슬라이드가 최소 두 개 있다고 가정합니다):

```csharp

using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IPPImage image = pres.Images.AddImage(Image.FromFile("image.png"));
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertZoomFrame(2, 150, 20, 50, 50, pres.Slides[1], image);
}
```

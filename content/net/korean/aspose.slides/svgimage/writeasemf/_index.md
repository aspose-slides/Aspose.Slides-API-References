## SvgImage.WriteAsEmf 메서드

SVG 이미지를 EMF 파일로 저장합니다.

```csharp
public void WriteAsEmf(Stream stream)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| stream | Stream | 대상 stream |

### 예외

| 예외 | 조건 |
| --- | --- |
| ArgumentNullException | 대상 stream은 **null**입니다 |

### 예제

다음 예제는 SVG 이미지를 메타파일에 저장하는 방법을 보여줍니다.

```csharp
[C#]
// 새 SVG 이미지를 생성합니다
ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));

// SVG 이미지를 메타파일로 저장합니다
using (var fileStream = System.IO.File.OpenWrite("SvgAsEmf.emf"))
    svgImage.WriteAsEmf(fileStream);
```

이 샘플은 SVG 이미지를 메타파일로 프레젠테이션 이미지 컬렉션에 추가하는 방법을 보여줍니다.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // 새 SVG 이미지를 생성합니다
    ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));
    using (var memStream = new MemoryStream())
    {
        // SVG 이미지를 메타파일로 저장합니다
        svgImage.WriteAsEmf(memStream);
        // 메타파일을 이미지 컬렉션에 추가합니다
        pres.Images.AddImage(memStream.ToArray());
    }
}
```

## ILayoutPlaceholderManager.AddVerticalTextPlaceholder 메서드

새로운 자리 표시자 모양을 레이아웃 슬라이드에 추가하여 텍스트 내용을 세로 방향으로 보관합니다.

```csharp
public IAutoShape AddVerticalTextPlaceholder(float x, float y, float width, float height)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| x | Single | 새 자리 표시자 모양의 X 좌표입니다. |
| y | Single | 새 자리 표시자 모양의 Y 좌표입니다. |
| width | Single | 새 자리 표시자 모양의 너비입니다. |
| height | Single | 새 자리 표시자 모양의 높이입니다. |

### 반환값

[`IAutoShape`](../../iautoshape)가 텍스트(Vertical) 자리 표시자를 사용하여 생성되었습니다.

### 예제

다음 예제는 레이아웃 슬라이드에 텍스트(Vertical) 자리 표시자 모양을 추가하는 방법을 보여줍니다.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalTextPlaceholder(20, 20, 300, 500);
}
```

## ILayoutPlaceholderManager.AddVerticalContentPlaceholder 메서드

레이아웃 슬라이드에 새 자리 표시자 도형을 추가하여 그림, 표, 미디어 또는 텍스트와 같은 내용을 세로 방향으로 보관합니다.

```csharp
public IAutoShape AddVerticalContentPlaceholder(float x, float y, float width, float height)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| x | Single | 새 자리 표시자 도형의 X 좌표입니다. |
| y | Single | 새 자리 표시자 도형의 Y 좌표입니다. |
| width | Single | 새 자리 표시자 도형의 너비입니다. |
| height | Single | 새 자리 표시자 도형의 높이입니다. |

### 반환 값

[`IAutoShape`](../../iautoshape)와 함께 Content (Vertical) placeholder를 생성했습니다.

### 예제

다음 예제는 레이아웃 슬라이드에 Content (Vertical) placeholder 도형을 추가하는 방법을 보여줍니다.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalContentPlaceholder(20, 20, 300, 500);
}
```

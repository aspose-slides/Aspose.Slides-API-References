## ILayoutPlaceholderManager.AddChartPlaceholder 메서드

차트를 포함하기 위해 레이아웃 슬라이드에 새 자리표시자 모양을 추가합니다.

```csharp
public IAutoShape AddChartPlaceholder(float x, float y, float width, float height)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| x | Single | 새 자리표시자 모양의 X 좌표 |
| y | Single | 새 자리표시자 모양의 Y 좌표 |
| width | Single | 새 자리표시자 모양의 너비 |
| height | Single | 새 자리표시자 모양의 높이 |

### 반환 값

Chart 자리표시자를 사용하여 [`IAutoShape`](../../iautoshape)를 생성했습니다.

### 예제

다음 예제는 레이아웃 슬라이드에 Chart 자리표시자 모양을 추가하는 방법을 보여줍니다.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddChartPlaceholder(20, 20, 200, 200);
}
```

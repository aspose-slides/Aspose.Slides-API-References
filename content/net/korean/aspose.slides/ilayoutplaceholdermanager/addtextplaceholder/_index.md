## ILayoutPlaceholderManager.AddTextPlaceholder 메서드

레이아웃 슬라이드에 텍스트 내용을 담는 새 자리 표시자 도형을 추가합니다.

```csharp
public IAutoShape AddTextPlaceholder(float x, float y, float width, float height)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| x | Single | 새 자리 표시자 도형의 X 좌표. |
| y | Single | 새 자리 표시자 도형의 Y 좌표. |
| width | Single | 새 자리 표시자 도형의 너비. |
| height | Single | 새 자리 표시자 도형의 높이. |

### 반환값

[`IAutoShape`](../../iautoshape)를 Text 자리 표시자와 함께 생성했습니다.

### 예제

다음 예제는 Text 자리 표시자 도형을 레이아웃 슬라이드에 추가하는 방법을 보여줍니다.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTextPlaceholder(20, 20, 500, 300);
}
```

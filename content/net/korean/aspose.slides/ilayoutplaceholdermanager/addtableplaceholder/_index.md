## ILayoutPlaceholderManager.AddTablePlaceholder 메서드

레이아웃 슬라이드에 테이블을 보관할 새 플레이스홀더 도형을 추가합니다.

```csharp
public IAutoShape AddTablePlaceholder(float x, float y, float width, float height)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| x | Single | 새 플레이스홀더 도형의 X 좌표입니다. |
| y | Single | 새 플레이스홀더 도형의 Y 좌표입니다. |
| width | Single | 새 플레이스홀더 도형의 너비입니다. |
| height | Single | 새 플레이스홀더 도형의 높이입니다. |

### 반환 값

[`IAutoShape`](../../iautoshape)와 함께 테이블 플레이스홀더가 생성되었습니다.

### 예제

다음 예제는 레이아웃 슬라이드에 테이블 플레이스홀더 도형을 추가하는 방법을 보여줍니다.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTablePlaceholder(20, 20, 500, 200);
}
```

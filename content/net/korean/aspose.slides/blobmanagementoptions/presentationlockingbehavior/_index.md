## BlobManagementOptions.PresentationLockingBehavior 속성

이 속성은 Presentation 클래스의 인스턴스가 인스턴스 수명 동안 소스 - 파일 또는 스트림의 소유자가 될 수 있는지를 정의합니다. 인스턴스가 소유자인 경우 소스를 잠급니다. 이는 BLOB 작업 시 메모리 사용량 및 성능을 향상시키는 데 도움이 되지만, Presentation 인스턴스 수명 동안 소스(스트림 또는 파일)를 변경할 수 없습니다.

```csharp
public PresentationLockingBehavior PresentationLockingBehavior { get; set; }
```

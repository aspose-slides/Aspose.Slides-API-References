## AddClone(ILayoutSlide) {#addclone}

지정된 레이아웃 슬라이드의 복사본을 프레젠테이션에 추가합니다.

```csharp
public ILayoutSlide AddClone(ILayoutSlide sourceLayout)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| sourceLayout | ILayoutSlide | 복제할 슬라이드. |

### 반환값

추가된 슬라이드.

### 비고

레이아웃을 다른 프레젠테이션 사이에서 복제할 경우 레이아웃의 마스터도 복제되어 원본 서식을 유지할 수 있습니다. 내부 레지스트리는 자동으로 복제된 마스터를 추적하여 동일한 마스터 슬라이드의 복제본이 여러 개 생성되는 것을 방지합니다. 마스터 슬라이드의 수동 복제는 방지되지도 않고 등록되지도 않습니다.

## AddClone(ILayoutSlide, IMasterSlide) {#addclone_1}

지정된 레이아웃 슬라이드의 복사본을 프레젠테이션에 추가합니다.

```csharp
public ILayoutSlide AddClone(ILayoutSlide sourceLayout, IMasterSlide destMaster)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| sourceLayout | ILayoutSlide | 복제할 슬라이드. |
| destMaster | IMasterSlide | 새 레이아웃의 마스터 슬라이드. |

### 반환값

추가된 슬라이드.

### 비고

1) 새로운 레이아웃은 대상 프레젠테이션의 정의된 마스터와 연결됩니다. 따라서 이는 PowerPoint에서 “Use Destination Theme” 옵션을 사용하는 복사/붙여넣기와 동일합니다.  
2) 이 메서드와 유사한 메서드는 [`AddClone`](../../imasterlayoutslidecollection/addclone)이며, [`LayoutSlides`](../../imasterslide/layoutslides) 속성을 통해 액세스합니다.

## IPresentationInfo.CheckWriteProtection 메서드

쓰기 보호된 프레젠테이션에 대한 수정 비밀번호가 올바른지 확인합니다.

```csharp
public bool CheckWriteProtection(string password)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| password | String | 확인할 password. |

### 반환 값

프레젠테이션이 쓰기 보호되어 있고 비밀번호가 올바른 경우 True, 그렇지 않으면 False.

### 예외

| exception | condition |
| --- | --- |
| InvalidOperationException |  |

### 비고

1. 이 메서드를 호출하기 전에 [`IsWriteProtected`](../iswriteprotected) 속성을 확인해야 합니다. 2. password가 null 이거나 비어 있는 경우, 이 메서드는 false를 반환합니다.

### 예제

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
if (info.IsWriteProtected == NullableBool.True)
{
    bool isWriteProtectedByPassword = info.CheckWriteProtection("my_password");
}
```

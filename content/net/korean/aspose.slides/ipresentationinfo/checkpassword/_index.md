## IPresentationInfo.CheckPassword 메서드

비밀번호가 개방형 암호로 보호된 프레젠테이션에 대해 올바른지 여부를 확인합니다.

```csharp
public bool CheckPassword(string password)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| password | String | 확인할 비밀번호입니다. |

### 반환값

프레젠테이션이 개방형 암호로 보호되어 있고 비밀번호가 올바른 경우 True이며, 그렇지 않은 경우 false입니다.

### 비고

비밀번호가 null이거나 비어 있으면 이 메서드는 false를 반환합니다.

### 예제

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo("pres.pptx");
bool isPasswordCorrect = info.CheckPassword("my_password");
```

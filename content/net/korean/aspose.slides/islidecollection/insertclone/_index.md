## InsertClone(int, ISlide) {#insertclone}

지정된 슬라이드의 복사본을 컬렉션의 지정된 위치에 삽입합니다.

```csharp
public ISlide InsertClone(int index, ISlide sourceSlide)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| index | Int32 | 새 슬라이드의 인덱스. |
| sourceSlide | ISlide | 복제할 슬라이드. |

### 반환값

삽입된 슬라이드.

### 비고

다른 프레젠테이션 간에 슬라이드를 복제할 때 슬라이드 마스터도 복제될 수 있습니다. 내부 레지스트리는 자동으로 복제된 마스터를 추적하여 동일한 마스터 슬라이드의 복제본이 여러 개 생성되는 것을 방지합니다. 마스터 슬라이드의 수동 복제는 방지되거나 등록되지 않습니다. 복제 프로세스를 보다 세밀하게 제어하려면 슬라이드 복제에 [`InsertClone`](../insertclone) 또는 [`InsertClone`](../insertclone)를 사용하고 마스터 복제에는 [`AddClone`](../../imasterslidecollection/addclone)를 사용하십시오.

## InsertClone(int, ISlide, IMasterSlide, bool) {#insertclone_2}

지정된 원본 슬라이드의 복사본을 컬렉션의 지정된 위치에 삽입합니다. 적절한 레이아웃은 지정된 마스터에서 자동으로 선택됩니다(적절한 레이아웃은 원본 슬라이드와 동일한 Type 또는 Name을 가진 레이아웃). 적절한 레이아웃이 없으면 원본 슬라이드의 레이아웃이 복제됩니다(allowCloneMissingLayout이 true인 경우) 또는 allowCloneMissingLayout이 false인 경우 PptxEditException이 발생합니다.

```csharp
public ISlide InsertClone(int index, ISlide sourceSlide, IMasterSlide destMaster, 
    bool allowCloneMissingLayout)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| index | Int32 | 새 슬라이드의 인덱스. |
| sourceSlide | ISlide | 복제할 슬라이드. |
| destMaster | IMasterSlide | 새 슬라이드에 대한 마스터 슬라이드. |
| allowCloneMissingLayout | Boolean | 지정된 마스터에 적절한 레이아웃이 없을 경우, allowCloneMissingLayout이 true이면 원본 슬라이드의 레이아웃이 복제되고, false이면 PptxEditException이 발생합니다. |

### 반환값

삽입된 슬라이드.

### 예외

| 예외 | 조건 |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | 지정된 마스터에 적절한 레이아웃이 없고 allowCloneMissingLayout이 false인 경우 발생합니다. |

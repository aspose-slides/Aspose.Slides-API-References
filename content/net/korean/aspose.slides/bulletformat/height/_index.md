## BulletFormat.Height property

상속이 없는 단락의 글머리 높이를 반환하거나 설정합니다. 값 float.NaN 은 글머리가 단락의 첫 번째 부분에서 높이를 상속함을 의미합니다. 읽기/쓰기 Single.

```csharp
public float Height { get; set; }
```

### 비고

음수 높이 값은 높이가 포인트 단위로 지정됨을 의미하고, 양수 값은 높이가 주변 텍스트의 백분율임을 의미합니다.

## PresentationInfo.CheckPassword 메서드

Checks whether a password is correct for a presentation protected with open password.

```csharp
public bool CheckPassword(string password)
```

| 매개변수 | Type | 설명 |
| --- | --- | --- |
| password | String | The password to check. |

### 반환 값

프레젠테이션이 오픈 비밀번호로 보호되어 있고 비밀번호가 올바른 경우 true를, 그렇지 않으면 false를 반환합니다.

| exception | condition |

| InvalidOperationException |  |

### 비고

비밀번호가 null이거나 빈 문자열인 경우, 이 메서드는 false를 반환합니다.

### 예제

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
bool isPasswordCorrect = info.CheckPassword("my_password");
```

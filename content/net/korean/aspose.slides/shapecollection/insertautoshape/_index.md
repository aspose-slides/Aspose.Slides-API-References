## InsertAutoShape(int, ShapeType, float, float, float, float) {#insertautoshape}

```csharp
public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width, 
    float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | Int32 | 새 자동 도형을 삽입할 0 기반 인덱스입니다. |
| shapeType | ShapeType | 삽입할 자동 도형의 [`ShapeType`](../../shapetype)입니다. |
| x | Single | 도형 프레임의 x 좌표(포인트 단위)입니다. |
| y | Single | 도형 프레임의 y 좌표(포인트 단위)입니다. |
| width | Single | 도형 프레임의 너비(포인트 단위)입니다. |
| height | Single | 도형 프레임의 높이(포인트 단위)입니다. |

## InsertAutoShape(int, ShapeType, float, float, float, float, bool) {#insertautoshape_1}

새 자동 도형을 생성하고 지정된 인덱스에 도형 컬렉션에 삽입하며, 선택적으로 기본 템플릿 스타일을 초기화합니다.

```csharp
public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | Int32 | 자동 도형을 삽입할 0 기반 인덱스입니다. |
| shapeType | ShapeType | 삽입할 자동 도형의 [`ShapeType`](../../shapetype)입니다. |
| x | Single | 도형 프레임의 x 좌표(포인트 단위)입니다. |
| y | Single | 도형 프레임의 y 좌표(포인트 단위)입니다. |
| width | Single | 도형 프레임의 너비(포인트 단위)입니다. |
| height | Single | 도형 프레임의 높이(포인트 단위)입니다. |
| createFromTemplate | Boolean | true이면 기본 템플릿 스타일을 적용합니다(비어 있지 않은 이름, 간단한 스타일, 가운데 정렬 텍스트 포함). false이면 모든 속성을 기본값으로 설정하여 도형을 생성합니다. |

### 반환 값

새로 생성된 [`IAutoShape`](../../iautoshape).

## ShapeCollection.InsertPictureFrame 메서드

지정된 이미지를 포함하는 새 picture frame을 만들고, 지정된 인덱스에 shape collection에 삽입합니다.

```csharp
public IPictureFrame InsertPictureFrame(int index, ShapeType shapeType, float x, float y, 
    float width, float height, IPPImage image)
```

| 파라미터 | 유형 | 설명 |
| --- | --- | --- |
| index | Int32 | 그림 프레임을 삽입할 0부터 시작하는 인덱스입니다. |
| shapeType | ShapeType | [`ShapeType`](../../shapetype)에 포함된 shape type을 지정합니다. 단, 모든 종류의 선은 제외합니다. |
| x | Single | picture frame의 x 좌표(포인트 단위)입니다. |
| y | Single | picture frame의 y 좌표(포인트 단위)입니다. |
| width | Single | picture frame의 너비(포인트 단위)입니다. |
| height | Single | picture frame의 높이(포인트 단위)입니다. |
| image | IPPImage | picture frame에 표시할 [`IPPImage`](../../ippimage)입니다. |

### 반환 값

새로 만든 [`IPictureFrame`](../../ipictureframe).

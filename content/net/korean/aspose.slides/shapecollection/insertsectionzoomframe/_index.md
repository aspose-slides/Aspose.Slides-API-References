## InsertSectionZoomFrame(int, float, float, float, float, ISection) {#insertsectionzoomframe}

새 섹션 확대 프레임을 생성하고 지정된 인덱스에 있는 shape 컬렉션에 삽입합니다.

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | Int32 | 섹션 확대 프레임을 삽입할 0 기반 인덱스입니다. |
| x | Single | 새 섹션 확대 프레임의 x 좌표(포인트 단위)입니다. |
| y | Single | 새 섹션 확대 프레임의 y 좌표(포인트 단위)입니다. |
| width | Single | 새 섹션 확대 프레임의 너비(포인트 단위)입니다. |
| height | Single | 새 섹션 확대 프레임의 높이(포인트 단위)입니다. |
| section | ISection | 섹션 확대 프레임이 참조하는 [`ISection`](../../isection)이며, 현재 프레젠테이션에 속하고 최소 하나의 슬라이드를 포함해야 합니다. |

### Return Value

새로 만든 [`ISectionZoomFrame`](../../isectionzoomframe)입니다.

### Exceptions

| exception | condition |
| --- | --- |
| ArgumentException | 참조된 섹션이 현재 프레젠테이션에 속하지 않거나 슬라이드가 없을 경우 발생합니다. |

### Examples

이 예제는 컬렉션의 지정된 인덱스에 섹션 확대 객체를 생성하고 삽입하는 방법을 보여줍니다(예: "Presentation.pptx" 프레젠테이션에 섹션이 최소 두 개 있다고 가정).

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSectionZoomFrame(2, 150, 20, 50, 50, pres.Sections[1]);
}
```

이 예제는 컬렉션의 지정된 인덱스에 섹션 확대 객체를 생성하고 삽입하는 방법을 보여줍니다(예: "Presentation.pptx" 프레젠테이션에 섹션이 최소 두 개 있다고 가정).

}

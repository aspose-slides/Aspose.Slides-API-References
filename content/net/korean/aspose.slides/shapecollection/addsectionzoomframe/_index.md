## AddSectionZoomFrame(float, float, float, float, ISection) {#addsectionzoomframe}

새 Section Zoom 프레임을 생성하고 shape 컬렉션의 끝에 추가합니다.

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| x | Single | 새 Section Zoom 프레임의 x 좌표(포인트 단위). |
| y | Single | 새 Section Zoom 프레임의 y 좌표(포인트 단위). |
| width | Single | 새 Section Zoom 프레임의 너비(포인트 단위). |
| height | Single | 새 Section Zoom 프레임의 높이(포인트 단위). |
| section | ISection | [`ISection`](../../isection)를 참조하는 Section Zoom 프레임; 이 프레젠테이션에 속하고 최소 하나의 슬라이드를 포함해야 합니다. |

### 반환 값

새로 생성된 [`ISectionZoomFrame`](../../isectionzoomframe).

### 예외

| 예외 | 조건 |
| --- | --- |
| ArgumentException | 참조된 섹션이 현재 프레젠테이션에 속하지 않거나 슬라이드가 없는 경우 발생합니다. |

### 예제

이 예제는 컬렉션 끝에 Section Zoom 개체를 추가하는 방법을 보여줍니다(예: "Presentation.pptx" 프레젠테이션에 최소 두 개의 섹션이 있다고 가정).

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1]);
}
```

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height,
```


## ShapeCollection.AddSummaryZoomFrame method

새 Summary Zoom 프레임을 생성하고 shape 컬렉션의 끝에 추가합니다.

```csharp
public ISummaryZoomFrame AddSummaryZoomFrame(float x, float y, float width, float height)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| x | Single | 새 Summary Zoom 프레임의 x 좌표(단위: 포인트). |
| y | Single | 새 Summary Zoom 프레임의 y 좌표(단위: 포인트). |
| width | Single | 새 Summary Zoom 프레임의 너비(단위: 포인트). |
| height | Single | 새 Summary Zoom 프레임의 높이(단위: 포인트). |

### 반환 값

새로 만든 [`ISummaryZoomFrame`](../../isummaryzoomframe).

| 예외 | 조건 |

### 비고

이 메서드는 새로운 Summary Zoom을 생성하고 이 프레젠테이션의 모든 섹션에 대한 객체 컬렉션을 넣습니다.

### 예제

이 예제는 컬렉션의 끝에 Summary Zoom 객체를 추가하는 방법을 보여줍니다(예: "Presentation.pptx" 프레젠테이션에 섹션이 최소 두 개 이상 있다고 가정).

```csharp
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSummaryZoomFrame(150, 20, 500, 250);
}
```

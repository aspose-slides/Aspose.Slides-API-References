## InsertAudioFrameEmbedded(int, float, float, float, float, Stream) {#insertaudioframeembedded_1}

임베드된 WAV 파일을 사용하여 새 오디오 프레임을 만들고 지정된 인덱스에 shape 컬렉션에 삽입합니다. 임베드된 오디오는 Presentation.Audios 컬렉션에 추가됩니다.

```csharp
public IAudioFrame InsertAudioFrameEmbedded(int index, float x, float y, float width, float height, 
    Stream audio_stream)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| index | Int32 | 삽입할 오디오 프레임의 0부터 시작하는 인덱스입니다. |
| x | Single | 새 오디오 프레임의 x좌표이며, 포인트 단위입니다. |
| y | Single | 새 오디오 프레임의 y좌표이며, 포인트 단위입니다. |
| width | Single | 새 오디오 프레임의 너비이며, 포인트 단위입니다. |
| height | Single | 새 오디오 프레임의 높이이며, 포인트 단위입니다. |
| audio_stream | Stream | 임베드할 WAV 오디오 데이터를 포함하는 입력 Stream입니다. |

### 반환값

새로 만든 [`IAudioFrame`](../../iaudioframe).

## InsertAudioFrameEmbedded(int, float, float, float, float, IAudio) {#insertaudioframeembedded}

```csharp
public IAudioFrame InsertAudioFrameEmbedded(int index, float x, float y, float width, float height, 
    IAudio audio)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| index | Int32 | 삽입할 오디오 프레임의 0부터 시작하는 인덱스입니다. |
| x | Single | 새 오디오 프레임의 x좌표이며, 포인트 단위입니다. |
| y | Single | 새 오디오 프레임의 y좌표이며, 포인트 단위입니다. |
| width | Single | 새 오디오 프레임의 너비이며, 포인트 단위입니다. |
| height | Single | 새 오디오 프레임의 높이이며, 포인트 단위입니다. |
| audio | IAudio | Presentation.Audios 컬렉션에서 가져온 [`IAudio`](../../iaudio) 인스턴스를 임베드합니다. |

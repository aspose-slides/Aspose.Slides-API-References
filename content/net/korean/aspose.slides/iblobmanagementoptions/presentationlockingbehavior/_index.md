## IBlobManagementOptions.PresentationLockingBehavior 속성

이 속성은 Presentation 클래스의 인스턴스가 인스턴스 수명 동안 소스(파일 또는 스트림)의 소유자가 될 수 있는지를 정의합니다. 인스턴스가 소유자인 경우 소스를 잠급니다. 이는 BLOB를 사용할 때 메모리 사용량과 성능을 향상시키는 데 도움이 되지만, 소스(스트림 또는 파일)는 Presentation 인스턴스 수명 동안 변경할 수 없습니다. 예제는 다음과 같습니다:

```csharp
public PresentationLockingBehavior PresentationLockingBehavior { get; set; }
```

### 예제

```csharp
LoadOptions loadOptions = new LoadOptions { 
  BlobManagementOptions = {PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked} };
using (Presentation pres = new Presentation("pres.pptx", loadOptions)) {
  // IOException이 발생합니다. pres.pptx가 Presentation 수명 동안 잠겨 있기 때문입니다.
  // File.Delete("pres.pptx");
}
// Presentation 객체가 해제된 후, 파일의 잠금이 해제되어 삭제할 수 있습니다.
File.Delete("pres.pptx");
```

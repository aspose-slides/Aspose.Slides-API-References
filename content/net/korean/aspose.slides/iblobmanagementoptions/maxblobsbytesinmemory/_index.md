## IBlobManagementOptions.MaxBlobsBytesInMemory 속성

모든 BLOB이 메모리에서 차지할 수 있는 최대 총 크기(바이트)를 정의합니다. 기본적으로 모든 BLOB은 메모리로 로드됩니다; 이 제한에 도달했을 때에만 임시 파일과 같은 대체 메커니즘이 사용됩니다. BLOB을 메모리에 유지하면 성능이 극대화되지만 메모리 사용량이 크게 증가할 수 있습니다. 이 속성을 사용하여 환경이나 요구 사항에 맞게 동작을 조정하십시오.

```csharp
public ulong MaxBlobsBytesInMemory { get; set; }
```

### 비고

이 속성은 [`IsTemporaryFilesAllowed`](../istemporaryfilesallowed) 가 false 로 설정된 경우 무시됩니다. 이 경우 메모리가 유일한 저장 위치가 되므로 인메모리 BLOB 사용량을 제한해도 효과가 없습니다.

기본값은 629,145,600 바이트(600 MB)입니다.

이 속성을 0으로 설정할 수 있지만, 최소한의 메모리는 여전히 예약됩니다.

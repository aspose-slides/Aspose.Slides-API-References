## PortionCollection.CopyTo 메서드

ICollection의 요소를 Array에 복사하고, 지정된 Array 인덱스에서 시작합니다.

```csharp
public void CopyTo(IPortion[] array, int arrayIndex)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| array | IPortion[] | ICollection에서 복사된 요소의 대상이 되는 1차원 Array입니다. Array는 0부터 시작하는 인덱스를 가져야 합니다. |
| arrayIndex | Int32 | *array*에서 복사가 시작되는 0 기반 인덱스입니다. |

### 예외

| 예외 | 조건 |
| --- | --- |
| ArgumentNullException | *array*가 null입니다. |
| ArgumentOutOfRangeException | *arrayIndex*가 0보다 작습니다. |
| ArgumentException | 소스 ICollection의 요소 수가 *arrayIndex*부터 대상 *array*의 끝까지 사용 가능한 공간보다 많습니다. |

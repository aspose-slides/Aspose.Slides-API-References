## IProtectionManager.CheckWriteProtection 메서드

프레젠테이션이 수정하기 위해 비밀번호로 보호되어 있는지 여부를 결정합니다.

```csharp
public bool CheckWriteProtection(string password)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| password | String | 확인용 비밀번호. |

### 반환값

true이면 비밀번호가 유효합니다; 그렇지 않으면 false.

### 비고

1. 이 메서드를 호출하기 전에 [`IsWriteProtected`](../iswriteprotected) 속성을 확인해야 합니다.  
2. 비밀번호가 null이거나 비어 있을 경우, 이 메서드는 false를 반환합니다.

### 예제

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

## ProtectionManager.CheckWriteProtection method

프레젠테이션이 수정에 대해 비밀번호로 보호되어 있는지 여부를 확인합니다.

```csharp
public bool CheckWriteProtection(string password)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| password | String | 검사용 password. |

### 반환 값

비밀번호가 유효하면 true; 그렇지 않으면 false.

### 주석

1. 이 메서드를 호출하기 전에 [`IsWriteProtected`](../iswriteprotected) 속성을 확인해야 합니다. 2. password가 null이거나 빈 문자열이면, 이 메서드는 false를 반환합니다.

### 예제

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

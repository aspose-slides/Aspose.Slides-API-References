## IFonts.GetScriptFont method

프레젠테이션 테마에서 특정 스크립트 태그와 연관된 글꼴 이름을 가져옵니다.

```csharp
public string GetScriptFont(string script)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| script | String | BCP-47 스크립트 코드(예: "Latn", "Cyrl", "Jpan")는 작문 시스템을 식별하는 데 사용됩니다. |

### 반환 값

지정된 스크립트에 사용되는 글꼴 이름이며, 스크립트가 정의되지 않은 경우 `null`을 반환합니다.

### 예제

이 예제는 프레젠테이션 테마에서 Cyrillic 스크립트에 할당된 글꼴을 검색하는 방법을 보여줍니다.

```csharp
[C#]
string font = presentation.MasterTheme.FontScheme.Major.GetScriptFont("Cyrl");
Console.WriteLine("Font for Cyrillic script: " + font);
```

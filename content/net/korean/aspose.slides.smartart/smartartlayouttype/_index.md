## SmartArtLayoutType 열거형

SmartArt 다이어그램의 레이아웃 유형을 나타냅니다.

```csharp
public enum SmartArtLayoutType
```

### 값

| 이름 | 값 | 설명 |
| --- | --- | --- |
| AccentProcess | `0` | 작업, 프로세스 또는 워크플로에서 진행, 타임라인 또는 순차적인 단계를 표시하는 데 사용합니다. Level 1 및 Level 2 텍스트를 모두 설명하는 데 적합합니다. |

| AscendingPictureAccentProcess | `7` | 설명 텍스트와 함께 상승하는 사진 시리즈를 표시하는 데 사용합니다. 텍스트 양이 적을수록 좋습니다. |
| Balance | `8` | 두 아이디어를 비교하거나 관계를 보여주는 데 사용합니다. 첫 번째 두 줄의 Level 1 텍스트가 중심점 양쪽 상단에 배치됩니다. Level 2 텍스트는 각 측면에 최대 네 개의 도형에 제한됩니다. 균형은 Level 2 텍스트가 가장 많은 쪽으로 기울어집니다. 사용되지 않은 텍스트는 보이지 않지만 레이아웃을 전환하면 사용할 수 있습니다. |

| BasicBlockList | `10` | 순차적이지 않거나 그룹화된 정보 블록을 표시하는 데 사용합니다. 도형의 가로·세로 표시 공간을 최대화합니다. |
| BasicChevronProcess | `11` | 진행, 타임라인, 순차적인 단계 또는 움직임·방향을 강조하는 데 사용합니다. Level 1 텍스트는 화살표 도형 내부에, Level 2 텍스트는 화살표 도형 아래에 표시됩니다. |
| BasicCycle | `12` | 원형 흐름에서 단계, 작업 또는 이벤트의 지속적인 순서를 나타냅니다. 연결 화살표보다 단계 자체를 강조합니다. Level 1 텍스트만 사용할 때 가장 좋습니다. |
| BasicMatrix | `13` | 사분면에 전체와의 구성 요소 관계를 표시하는 데 사용합니다. 첫 네 줄의 Level 1 텍스트가 사분면에 표시됩니다. 사용되지 않은 텍스트는 보이지 않지만 레이아웃을 전환하면 사용할 수 있습니다. |
| BasicPie | `14` | 개별 부분이 전체를 형성하는 방식을 보여줍니다. 첫 일곱 줄의 Level 1 텍스트가 골고루 배분된 파이 도형에 해당합니다. 상위 Level 1 텍스트 도형은 파이 나머지 부분 밖에 표시되어 강조됩니다. 사용되지 않은 텍스트는 보이지 않지만 레이아웃을 전환하면 사용할 수 있습니다. |
| BasicProcess | `15` | 작업, 프로세스 또는 워크플로에서 진행 또는 순차적인 단계를 표시하는 데 사용합니다. |
| BasicPyramid | `16` | 비례, 상호 연결 또는 계층 관계를 표시합니다. 가장 큰 구성 요소가 하단에 있고 위로 좁아집니다. Level 1 텍스트는 피라미드 구간에, Level 2 텍스트는 각 구간 옆 도형에 표시됩니다. |
| BasicRadial | `17` | 순환에서 중심 아이디어와의 관계를 표시합니다. 첫 번째 줄의 Level 1 텍스트가 중심 도형에, 해당 Level 2 텍스트가 주변 원형 도형에 대응합니다. 사용되지 않은 텍스트는 보이지 않지만 레이아웃을 전환하면 사용할 수 있습니다. |
| BasicTarget | `18` | 포함, 등급 구분 또는 계층 관계를 표시합니다. 첫 다섯 줄의 Level 1 텍스트가 원에 연결됩니다. 사용되지 않은 텍스트는 보이지 않지만 레이아웃을 전환하면 사용할 수 있습니다. |
| BasicTimeline | `19` | 작업, 프로세스 또는 워크플로의 순차적인 단계 또는 타임라인 정보를 표시하는 데 사용합니다. Level 1 및 Level 2 텍스트 모두에 잘 어울립니다. |
| BasicVenn | `20` | 겹치거나 상호 연결된 관계를 표시합니다. 첫 일곱 줄의 Level 1 텍스트가 원에 대응합니다. 텍스트가 네 줄 이하이면 원 내부에, 네 줄 초과이면 원 외부에 표시됩니다. 사용되지 않은 텍스트는 보이지 않지만 레이아웃을 전환하면 사용할 수 있습니다. |

| BendingPictureBlocks | `22` | 일련의 사진을 표시합니다. 하단 모서리를 덮는 상자는 소량의 텍스트를 포함할 수 있습니다. |
| BendingPictureCaption | `23` | 순차적인 사진 시리즈를 표시합니다. 하단 모서리를 덮는 상자는 소량의 텍스트를 포함할 수 있습니다. |
| BendingPictureCaptionList | `24` | 일련의 사진을 표시합니다. 제목과 설명이 각 사진 아래의 말풍선 도형에 나타납니다. |
| BendingPictureSemiTransparentText | `25` | 일련의 사진을 표시합니다. 반투명 상자가 사진 하단을 덮고 모든 수준의 텍스트를 포함합니다. |
| BlockCycle | `26` | 원형 흐름에서 단계, 작업 또는 이벤트의 지속적인 순서를 나타냅니다. 연결 화살표보다 단계 자체를 강조합니다. |
| BubblePictureList | `27` | 일련의 사진을 표시합니다. 최대 여덟 개의 Level 1 사진을 포함할 수 있습니다. 사용되지 않은 텍스트와 사진은 보이지 않지만 레이아웃을 전환하면 사용할 수 있습니다. 텍스트가 적을 때 가장 좋습니다. |
| CaptionedPictures | `28` | 여러 수준의 텍스트가 포함된 사진을 표시합니다. 소량의 Level 1 텍스트와 중간 양의 Level 2 텍스트에 가장 적합합니다. |
| ChevronList | `29` | 전체 워크플로를 구성하는 여러 프로세스를 통한 진행을 표시합니다. 또한 대조적인 프로세스를 설명하는 데도 사용됩니다. Level 1 텍스트는 왼쪽 첫 화살표 도형에, Level 2 텍스트는 각 Level 1 텍스트가 포함된 도형의 수평 하위 단계에 해당합니다. |

| CircleArrowProcess | `31` | 각 항목에 대한 지원 텍스트와 함께 순차적인 항목을 표시합니다. 이 다이어그램은 소량의 Level 1 텍스트에 가장 적합합니다. |

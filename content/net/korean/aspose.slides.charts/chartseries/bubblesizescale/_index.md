## ChartSeries.BubbleSizeScale property

버블 차트의 스케일 팩터를 지정합니다(기본 크기의 0%에서 300% 사이가 될 수 있습니다). 이 속성은 이 시리즈뿐만 아니라 상위 시리즈 그룹의 모든 시리즈에 적용되는 속성의 투영입니다. 따라서 이 속성은 읽기 전용입니다. 상위 시리즈 그룹에 접근하려면 ParentSeriesGroup 속성을 사용하십시오. 값을 변경하려면 ParentSeriesGroup.BubbleSizeScale 읽기/쓰기 속성을 사용하십시오.

public int BubbleSizeScale { get; }

## ChartSeries.FirstSliceAngle 속성

첫 번째 파이 또는 도넛 차트 조각의 각도를 도(degree) 단위로 지정합니다(위쪽을 기준으로 시계 방향, 0에서 360도까지). 이 속성은 이 시리즈뿐만 아니라 상위 시리즈 그룹의 모든 시리즈에 적용됩니다 - 해당 그룹 속성의 투영입니다. 따라서 이 속성은 읽기 전용입니다. 상위 시리즈 그룹에 접근하려면 ParentSeriesGroup 속성을 사용하십시오. 값을 변경하려면 ParentSeriesGroup.FirstSliceAngle 읽기/쓰기 속성을 사용하십시오. 읽기 전용 UInt16.

```csharp
public ushort FirstSliceAngle { get; }
```

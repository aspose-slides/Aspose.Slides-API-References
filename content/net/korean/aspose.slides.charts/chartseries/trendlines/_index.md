## ChartSeries.TrendLines 속성

시리즈 추세선의 컬렉션. 읽기 전용 [`ITrendlineCollection`](../../itrendlinecollection).

```csharp
public ITrendlineCollection TrendLines { get; }
```

### 비고

TrendLines는 스택되지 않은 2-D 영역, 막대, 열, 선, 주식, xy(분산) 및 버블 차트의 데이터 시리즈에 대해 사용 가능(널이 아님)합니다. 추세선은 스택되었거나 3-D인 차트 유형의 데이터 시리즈에 대해 사용할 수 없습니다. 또한 레이더, 파이, 서페이스 또는 도넛 차트에서는 추세선을 사용할 수 없습니다.

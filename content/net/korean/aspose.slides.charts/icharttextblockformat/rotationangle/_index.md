## IChartTextBlockFormat.RotationAngle 속성

텍스트가 경계 상자 안에 적용되는 사용자 정의 회전을 지정합니다. 지정되지 않은 경우, 해당 모양의 회전이 사용됩니다. 지정된 경우, 이는 모양과 독립적으로 적용됩니다. 즉, 모양에 회전이 적용될 수 있으며 텍스트 자체에도 별도로 회전이 적용될 수 있습니다. 이 속성 및 속성 TextVerticalType의 미리 정의된 수직 유형으로부터 요약된 시각적 텍스트 회전값이 결과가 됩니다. 읽기/쓰기 Single.

```csharp
public float RotationAngle { get; set; }
```

### 예제

쉐이프에 시계 방향으로 90도 회전이 적용된 경우를 고려하십시오. 여기에 추가로 텍스트 본문 자체에 반시계 방향으로 -90도 회전이 적용됩니다. 그러면 결과 쉐이프는 회전된 것처럼 보이지만, 그 안의 텍스트는 전혀 회전되지 않은 것처럼 보입니다.

## IChartSeries.ErrorBarsXFormat 속성

시리즈의 X 방향 ErrorBars를 나타냅니다. X 방향의 ErrorBars는 area, bar, scatter, bubble 유형의 시리즈에 사용할 수 있습니다. 다른 모든 차트 유형에 대해서는 이 속성이 null을 반환합니다(3D 차트 포함). 사용자 지정 값의 경우 DataPoints 컬렉션을 사용하여 값을 지정합니다([`ErrorBarsCustomValues`](../../ichartdatapoint/errorbarscustomvalues) 속성을 사용). 읽기 전용 [`IErrorBarsFormat`](../../ierrorbarsformat).

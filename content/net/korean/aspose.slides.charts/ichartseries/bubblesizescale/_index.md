## IChartSeries.BubbleSizeScale 속성

버블 차트에 대한 배율 계수를 지정합니다 (기본 크기의 0%에서 300% 사이일 수 있습니다). 이 속성은 이 시리즈에만 해당되는 것이 아니라 상위 시리즈 그룹의 모든 시리즈에 적용되는 속성의 투영입니다. 따라서 이 속성은 읽기 전용입니다. 상위 시리즈 그룹에 액세스하려면 ParentSeriesGroup 속성을 사용하십시오. 값을 변경하려면 ParentSeriesGroup.BubbleSizeScale 읽기/쓰기 속성을 사용하십시오.

```csharp
public int BubbleSizeScale { get; }
```

## IChartSeries.SecondPieSize 속성

두 번째 파이 또는 파이-오브-파이 차트 또는 바-오브-파이 차트의 두 번째 파이(또는 바)의 크기를 첫 번째 파이의 크기에 대한 백분율로 지정합니다(5%에서 200% 사이 가능). 이 속성은 해당 시리즈뿐만 아니라 상위 시리즈 그룹의 모든 시리즈에 적용되는 속성으로, 해당 그룹 속성의 투영입니다. 따라서 이 속성은 읽기 전용입니다. 상위 시리즈 그룹에 접근하려면 ParentSeriesGroup 속성을 사용하십시오. 값을 변경하려면 ParentSeriesGroup.SecondPieSize 읽기/쓰기 속성을 사용하십시오. 읽기 전용 UInt16.

```csharp
public ushort SecondPieSize { get; }
```

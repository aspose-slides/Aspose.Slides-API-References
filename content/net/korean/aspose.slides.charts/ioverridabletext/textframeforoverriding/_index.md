## IOverridableText.TextFrameForOverriding 속성

서식이 적용된 풍부한 텍스트를 포함할 수 있습니다. 이 속성이 null이 아니면 해당 서식 텍스트 값이 자동 생성된 텍스트를 대체합니다. 자동 생성된 텍스트는 데이터 레이블, 값 축의 표시 단위 레이블, 축 제목, 차트 제목, 추세선 레이블의 암시적 속성입니다. 자동 생성된 텍스트는 IFormattedTextContainer.TextFormat 속성을 사용하여 서식이 지정됩니다. 읽기 전용 [`ITextFrame`](../../../aspose.slides/itextframe).

```csharp
public ITextFrame TextFrameForOverriding { get; }
```

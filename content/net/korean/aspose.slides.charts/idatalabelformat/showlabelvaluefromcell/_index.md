## IDataLabelFormat.ShowLabelValueFromCell 속성

지정된 차트의 데이터 레이블 셀 값 표시 동작을 나타냅니다. True이면 셀 값을 표시합니다. False이면 숨깁니다. 읽기/쓰기 Boolean.

```csharp
public bool ShowLabelValueFromCell { get; set; }
```

### 참고

만약 이 DataLabelFormat 객체의 부모가 DataLabelCollection 데이터 레이블 컬렉션이면, 이 속성은 DataLabelCollection 컬렉션의 새 데이터 레이블에 대한 ShowLabelValueFromCell 속성의 기본값을 가져오거나 설정합니다. 이 속성을 값과 함께 설정하면 DataLabelCollection 컬렉션의 모든 데이터 레이블에 대한 ShowLabelValueFromCell 속성에도 해당 값이 설정됩니다(예: "DataLabels.DefaultDataLabelFormat.ShowLabelValueFromCell = val;" 은 모든 DataLabels[i].ShowLabelValueFromCell 가 val와 같게 됩니다).

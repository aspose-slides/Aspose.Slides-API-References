## IDataLabelFormat.ShowValue 속성

지정된 차트의 데이터 레이블 백분율 값 표시 동작을 나타냅니다. True이면 백분율 값을 표시하고, False이면 숨깁니다. 읽기/쓰기 Boolean.

```csharp
public bool ShowValue { get; set; }
```

### 비고

이 DataLabelFormat 객체의 부모가 DataLabelCollection 데이터 레이블 컬렉션인 경우, 이 속성은 DataLabelCollection 컬렉션에 있는 새 데이터 레이블의 ShowValue 속성에 대한 기본값을 가져오거나 설정합니다. 이 속성에 값을 설정하면 DataLabelCollection 컬렉션에 있는 모든 데이터 레이블의 ShowValue 속성에도 이 값이 설정됩니다(예: "DataLabels.DefaultDataLabelFormat.ShowValue = val;"는 모든 DataLabels[i].ShowValue가 val와 같아짐을 의미합니다).

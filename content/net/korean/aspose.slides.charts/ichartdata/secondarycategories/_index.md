## IChartData.SecondaryCategories 속성

[`UseSecondaryCategories`](../usesecondarycategories) 속성이 true인 경우 보조 범주를 가져옵니다. 읽기 전용 [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### 비고

[`UseSecondaryCategories`](../usesecondarycategories) 속성이 false이면 이 `SecondaryCategories` 속성이 null을 반환하고 [`Categories`](../categories) 속성의 데이터가 기본 및 보조 시리즈 모두에 사용됩니다. [`UseSecondaryCategories`](../usesecondarycategories) 속성이 true이면 이 `SecondaryCategories` 속성의 데이터는 보조 시리즈에 사용되고 [`Categories`](../categories) 속성의 데이터는 기본 시리즈에 사용됩니다.

### 예제

예제. 시리즈와 관련된 범주는 ChartData.Categories 또는 ChartData.SecondaryCategories인가요?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // 관련 카테고리는 series.Chart.ChartData.SecondaryCategories
}
else
{
    // 관련 카테고리는 series.Chart.ChartData.Categories
}
```

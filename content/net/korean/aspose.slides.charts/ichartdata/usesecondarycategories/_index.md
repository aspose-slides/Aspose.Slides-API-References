## IChartData.UseSecondaryCategories 속성

false인 경우 [`SecondaryCategories`](../secondarycategories) 속성은 null을 반환하고 [`Categories`](../categories) 속성의 데이터는 기본 시리즈와 보조 시리즈 모두에 사용됩니다. true인 경우 [`SecondaryCategories`](../secondarycategories) 속성의 데이터는 보조 시리즈에 사용되고 [`Categories`](../categories) 속성의 데이터는 기본 시리즈에 사용됩니다. 읽기/쓰기 Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

### 예제

예시. 시리즈와 관련된 카테고리는 ChartData.Categories 또는 ChartData.SecondaryCategories인가요?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // 관련 카테고리는 series.Chart.ChartData.SecondaryCategories
}
else
{
    // 관련 카테고리는 series.Chart.ChartData.Categories
}
```

## IChartData.Categories 속성

기본 카테고리를 가져옵니다(또는 [`UseSecondaryCategories`](../usesecondarycategories) 속성이 false인 경우 기본 및 보조 카테고리를 모두 가져옵니다). 읽기 전용 [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection Categories { get; }
```

### 비고

[`UseSecondaryCategories`](../usesecondarycategories) 속성이 false이면 [`SecondaryCategories`](../secondarycategories) 속성이 null을 반환하고 이 `Categories` 속성의 데이터가 기본 및 보조 시리즈 모두에 사용됩니다. [`UseSecondaryCategories`](../usesecondarycategories) 속성이 true이면 [`SecondaryCategories`](../secondarycategories) 속성의 데이터가 보조 시리즈에 사용되고 이 `Categories` 속성의 데이터가 기본 시리즈에 사용됩니다.

### 예제

예제. 어떤 카테고리가 시리즈와 관련이 있는가 - ChartData.Categories 또는 ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // 관련 카테고리는 series.Chart.ChartData.SecondaryCategories
}
else
{
    // 관련 카테고리는 series.Chart.ChartData.Categories
}
```

## DataLabelFormat.ShowLabelValueFromCell 속성

지정된 차트의 데이터 레이블 셀 값 표시 동작을 나타냅니다. True이면 셀 값을 표시하고, False이면 숨깁니다. 읽기/쓰기 Boolean.

```csharp
public bool ShowLabelValueFromCell { get; set; }
```

### 비고

이 DataLabelFormat 객체의 부모가 DataLabelCollection 컬렉션인 경우, 이 속성은 DataLabelCollection 컬렉션에 있는 새 데이터 레이블에 대한 ShowLabelValueFromCell 속성의 기본값을 가져오거나 설정합니다. 값을 사용하여 이 속성을 설정하면 DataLabelCollection 컬렉션에 있는 모든 데이터 레이블의 ShowLabelValueFromCell 속성에도 해당 값이 설정됩니다(예: "DataLabels.DefaultDataLabelFormat.ShowLabelValueFromCell = val;"는 모든 DataLabels[i].ShowLabelValueFromCell이 val과 동일하도록 합니다).

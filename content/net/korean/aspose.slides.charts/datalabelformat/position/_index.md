## DataLabelFormat.Position 속성

데이터 레이블의 위치를 나타냅니다. 읽기/쓰기 [`LegendDataLabelPosition`](../../legenddatalabelposition).

```csharp
public LegendDataLabelPosition Position { get; set; }
```

### 비고

이 DataLabelFormat 객체의 상위가 DataLabelCollection 데이터 레이블 컬렉션인 경우, 이 속성은 DataLabelCollection 컬렉션에 있는 새 데이터 레이블의 Position 속성 기본값을 가져오거나 설정합니다. DataLabel 객체의 위치를 나타냅니다. 이 속성에 값을 설정하면 해당 값이 DataLabelCollection 컬렉션에 포함된 모든 데이터 레이블의 Position 속성에도 설정됩니다(예: "DataLabels.DefaultDataLabelFormat.Position = val;" 은 모든 DataLabels[i].Position이 val과 동일하게 됩니다).

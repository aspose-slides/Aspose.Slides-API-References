## DataLabelFormat.ShowBubbleSize 속성

지정된 차트의 데이터 레이블 버블 크기 값 표시 동작을 나타냅니다. True는 버블 크기 값을 표시하고, False는 숨깁니다. 읽기/쓰기 Boolean.

```csharp
public bool ShowBubbleSize { get; set; }
```

### 비고

이 DataLabelFormat 객체의 상위 개체가 DataLabelCollection 컬렉션인 경우, 이 속성은 DataLabelCollection 컬렉션의 새 데이터 레이블에 대한 ShowBubbleSize 속성의 기본값을 가져오거나 설정합니다. 값을 사용하여 이 속성을 설정하면 해당 값이 DataLabelCollection 컬렉션에 포함된 모든 데이터 레이블의 ShowBubbleSize 속성에도 적용됩니다(예: "DataLabels.DefaultDataLabelFormat.ShowBubbleSize = val;" 로 설정하면 모든 DataLabels[i].ShowBubbleSize가 val과 같아집니다).

## DataLabelFormat.Separator 속성

차트의 데이터 레이블에 사용되는 구분자를 나타내는 Variant를 설정하거나 반환합니다. 읽기/쓰기 String.

```csharp
public string Separator { get; set; }
```

### Remarks

이 DataLabelFormat 객체의 부모가 DataLabelCollection 컬렉션인 경우, 이 속성은 DataLabelCollection 컬렉션의 새 데이터 레이블에 대해 Separator 속성의 기본값을 가져오거나 설정합니다. 이 속성을 값으로 설정하면 해당 값이 DataLabelCollection 컬렉션의 모든 데이터 레이블에 대한 Separator 속성에도 설정됩니다(예: "DataLabels.DefaultDataLabelFormat.Separator = val;"는 모든 DataLabels[i].Separator가 val와 동일하게 됩니다).

## IDataLabelCollection.Hide 메서드

데이터 레이블을 기본적으로 숨기려면 DefaultDataLabelFormat 속성의 모든 Show*-flags(ShowValue, ...)를 false 상태로 설정합니다. IsVisible는 이 후 false가 됩니다.

```csharp
public void Hide()
```

### 비고

데이터 레이블이 기본적으로 보이지 않는 경우(IsVisible가 false) DefaultDataLabelFormat 속성의 Show*-flags(ShowValue, ...)를 true 상태로 설정하면 데이터 레이블을 기본적으로 보이게 만들 수 있습니다.

## ChartData.UseSecondaryCategories 속성

false이면 [`SecondaryCategories`](../secondarycategories) 속성이 null을 반환하고 [`Categories`](../categories) 속성의 데이터가 기본 및 보조 시리즈 모두에 사용됩니다. true이면 [`SecondaryCategories`](../secondarycategories) 속성의 데이터가 보조 시리즈에 사용되고 [`Categories`](../categories) 속성의 데이터가 기본 시리즈에 사용됩니다. 읽기/쓰기 Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

### 예제

예시. 시리즈와 관련된 카테고리는 ChartData.Categories 또는 ChartData.SecondaryCategories 중 어느 것입니까?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // 관련 카테고리는 series.Chart.ChartData.SecondaryCategories
}
else
{
    // 관련 카테고리는 series.Chart.ChartData.Categories
}
```

## ChartData.SetRange 메서드

차트 데이터 범위를 설정합니다. 새로운 데이터 범위를 기준으로 시리즈와 카테고리가 업데이트됩니다. 데이터 범위에 있는 시리즈 수가 차트 데이터의 시리즈 수보다 많을 경우, 현재 컬렉션의 마지막 시리즈와 동일한 유형의 추가 시리즈가 컬렉션 끝에 추가됩니다.

```csharp
public void SetRange(string formula)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| formula | String | 셀 데이터 범위 수식. 예: "Sheet1!$A$1:$C$4" , "SomeSheetName!A1:B100", "Sheet1!$A$1:$B$5;Sheet1!$D$1:$D$5". |

### 예외

| 예외 | 조건 |
| --- | --- |
| ArgumentNullException | formula가 null입니다. |
| InvalidOperationException | 지원되지 않는 차트 유형 |
| ArgumentException | formula의 형식이 올바르지 않습니다. |

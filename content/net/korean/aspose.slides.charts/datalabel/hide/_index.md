## DataLabel.Hide 메서드

Show*-플래그(ShowValue, ...)를 모두 false 상태로 설정하여 데이터 레이블을 숨깁니다. IsVisible는 이 후 false가 됩니다.

```csharp
public void Hide()
```

### 비고

데이터 레이블이 보이지 않는 경우(IsVisible가 false) Show*-플래그(ShowValue, ...)를 true 상태로 설정하여 데이터 레이블을 보이게 할 수 있습니다.

## DataLabel.TextFrameForOverriding 속성

리치 형식 텍스트를 포함할 수 있습니다. 이 속성이 null이 아니면 이 형식화된 텍스트 값이 데이터 레이블의 자동 생성 텍스트를 대체합니다. 데이터 레이블의 자동 생성 텍스트는 ShowSeriesName, ShowValue, ... 속성에 의해 관리되며 TextFormatManager.TextFormat 속성을 사용해 형식이 지정된 텍스트를 의미합니다. 읽기 전용 [`ITextFrame`](../../../aspose.slides/itextframe).

## DataLabel.IsVisible 속성

False는 데이터 레이블이 표시되지 않음을 의미합니다(따라서 모든 Show*-플래그(ShowValue, ...)는 false입니다). 읽기 전용 Boolean.

```csharp
public bool IsVisible { get; }
```

### 비고

데이터 레이블이 표시되어 있는 경우 Hide() 메서드로 숨길 수 있습니다. 그러나 데이터 레이블이 표시되지 않은 경우(IsVisible가 false) Show*-플래그(ShowValue, ...)를 true 상태로 설정하여 데이터 레이블을 표시할 수 있습니다.

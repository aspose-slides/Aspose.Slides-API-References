## IChartSeriesGroup.Overlap 속성

2-D 차트에서 막대와 열이 겹치는 정도를 백분율(-100% ~ 100%)로 지정합니다.  
- -100%: 최대 간격(막대가 완전히 분리됨).  
- 0%: 막대가 겹치거나 간격 없이 나란히 배치됨.  
- 100%: 최대 겹침(막대가 서로 완전히 겹침).  
이 속성은 읽기/쓰기 SByte입니다.

```csharp
public sbyte Overlap { get; set; }
```

### 예외

| 예외 | 조건 |
| --- | --- |
| ArgumentOutOfRangeException | 값이 -100에서 100 사이의 유효 범위를 벗어나도록 설정될 때 발생합니다. |

### 예제

다음 예제는 차트 시리즈 그룹에 대한 겹침을 설정하고 결과 차트를 폼에 렌더링하는 방법을 보여줍니다:

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // 겹침을 55%로 설정합니다.

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

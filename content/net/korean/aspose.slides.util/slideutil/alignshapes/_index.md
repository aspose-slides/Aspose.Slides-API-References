## AlignShapes(ShapesAlignmentType, bool, IBaseSlide) {#alignshapes}

```csharp
public static void AlignShapes(ShapesAlignmentType alignmentType, bool alignToSlide, 
    IBaseSlide slide)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| alignmentType | ShapesAlignmentType | 적용될 정렬 유형을 결정합니다. |
| alignToSlide | Boolean | true인 경우 도형을 슬라이드 가장자리를 기준으로 정렬합니다. |
| slide | IBaseSlide | 상위 슬라이드. |

### 예제

Example:

```csharp
[C#]

}
```

| slide | IBaseSlide | 상위 슬라이드. |
| shapeIndexes | Int32[] | 정렬할 도형의 인덱스. |

### 예제

Example:

```csharp
[C#]

{
   ISlide slide = pres.Slides[0];
   IShape shape1 = slide.Shapes[0];
   IShape shape2 = slide.Shapes[1]; 

   SlideUtil.AlignShapes(ShapesAlignmentType.AlignBottom, false, pres.Slides[0], new int[]
   {
      slide.Shapes.IndexOf(shape1),
      slide.Shapes.IndexOf(shape2)
   });
}
```

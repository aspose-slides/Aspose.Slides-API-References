## AddTableFromWorkbook(IShapeCollection, float, float, IExcelDataWorkbook, string, string) {#addtablefromworkbook}

지정된 Excel 워크북에서 테이블을 가져와 지정된 좌표에 있는 주어진 shape 컬렉션의 끝에 추가합니다.

```csharp
public static ITable AddTableFromWorkbook(IShapeCollection shapes, float x, float y, 
    IExcelDataWorkbook workbook, string worksheetName, string cellRange)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| shapes | IShapeCollection | 테이블을 추가할 shape 컬렉션입니다. |
| x | Single | 테이블 위치 지정에 사용되는 X 좌표입니다. |
| y | Single | 테이블 위치 지정에 사용되는 Y 좌표입니다. |
| workbook | IExcelDataWorkbook | Excel 워크북입니다. |
| worksheetName | String | 테이블이 포함된 워크시트 이름입니다. |
| cellRange | String | 테이블을 정의하는 셀 범위(예: "A1:D10")입니다. |

### 반환값

shape 컬렉션에 추가된 테이블입니다.

### 예외

| 예외 | 조건 |
| --- | --- |
| ArgumentException | 필수 매개변수가 null이거나 비어 있거나 지정된 워크시트 또는 셀 범위가 잘못된 경우 발생합니다. |
| InvalidOperationException | 입력 데이터가 지원되지 않는 형식인 경우 발생합니다. |

### 예제

```csharp
[C#]
IExcelDataWorkbook workbook = new ExcelDataWorkbook(testFile);
using (var pres = new Presentation())
{
    ExcelWorkbookImporter.AddTableFromWorkbook(pres.Slides[0].Shapes, 10, 10, workbook, worksheetName, "A1:D10");
    pres.Save("result.pptx", SaveFormat.Pptx);
}
```

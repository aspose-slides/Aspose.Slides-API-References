## MathPhantom.Transp 속성

phantom이 클래스 기반 간격 규칙에 대해 투명한지 여부를 나타내는 값을 가져오거나 설정합니다.

```csharp
public bool Transp { get; set; }
```

### 비고

`true`인 경우, phantom 내부의 연산자와 기호는 phantom 주변의 수학적 간격에 영향을 미칩니다(보이는 것처럼). `false`인 경우, 클래스 기반 간격이 무시됩니다. OMML 특성 `m:transp`에 해당합니다.

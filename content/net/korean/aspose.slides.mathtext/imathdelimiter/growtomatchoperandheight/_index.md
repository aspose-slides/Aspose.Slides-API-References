## IMathDelimiter.GrowToMatchOperandHeight 속성

BeginningCharacter, SeparatorCharacter, EndingCharacter의 성장을 지정합니다. true인 경우 구분자는 피연산자 높이에 맞게 수직으로 성장합니다. 기본값은 true입니다.

```csharp
public bool GrowToMatchOperandHeight { get; set; }
```

### 예제

예시:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.GrowToMatchOperandHeight = false;
```

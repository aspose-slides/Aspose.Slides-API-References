## IMathMatrix 인터페이스

행과 열 하나 이상에 배치된 자식 요소들로 구성된 Matrix 객체를 지정합니다. 행렬에는 내장 구분 기호가 없다는 점에 유의해야 합니다. 행렬을 괄호로 묶으려면 구분 기호 객체(IMathDelimiter)를 사용해야 합니다. Null 인수를 사용하여 행렬에 빈 공간을 만들 수 있습니다.

```csharp
public interface IMathMatrix : IMathElement
```

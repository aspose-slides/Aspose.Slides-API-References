## IMathMatrix.RowGap 속성

행렬 행 사이의 수직 간격 값입니다; RowGapRule이 3 ("Exactly")으로 설정된 경우 단위는 트윕(포인트의 1/20)으로 해석됩니다. RowGapRule이 4 ("Multiple")으로 설정된 경우 단위는 반 라인으로 해석됩니다. 기본값: 0

public uint RowGap { get; set; }

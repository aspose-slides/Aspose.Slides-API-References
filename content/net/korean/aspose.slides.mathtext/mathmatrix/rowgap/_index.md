## MathMatrix.RowGap 속성

행렬의 행 사이의 수직 간격 값입니다. RowGapRule이 3("Exactly")으로 설정된 경우 단위는 트윕(1/20th of a point)으로 해석됩니다. RowGapRule이 4("Multiple")으로 설정된 경우 단위는 반줄로 해석됩니다. 기본값: 0

```csharp
public uint RowGap { get; set; }
```

### 예제

Example:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

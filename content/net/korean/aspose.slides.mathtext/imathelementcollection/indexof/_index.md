## IMathElementCollection.IndexOf 메서드

컬렉션에서 특정 수학 요소의 인덱스를 결정합니다.

```csharp
public int IndexOf(IMathElement item)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| item | IMathElement | 컬렉션에서 찾을 요소입니다. |

### 반환 값

컬렉션에서 찾은 경우 *item*의 인덱스, 찾지 못한 경우 -1.

### 예제

예:

```csharp

int index = collection.IndexOf(plusElement);
```

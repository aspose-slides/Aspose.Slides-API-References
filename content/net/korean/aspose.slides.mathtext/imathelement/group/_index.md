## Group(char, MathTopBotPositions, MathTopBotPositions) {#group_1}

이 요소를 하단 중괄호와 같은 그룹화 문자 또는 다른 문자를 사용하여 그룹에 배치합니다.

```csharp
public IMathGroupingCharacter Group(char character, MathTopBotPositions position, 
    MathTopBotPositions verticalJustification)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| character | Char | BOTTOM CURLY BRACKET (U+23DF)와 같은 그룹화 문자 또는 기타 문자 |
| position | MathTopBotPositions | 그룹화 문자의 위치 |
| verticalJustification | MathTopBotPositions | 그룹 문자에 대한 수직 정렬. 객체가 기준선에 대해 어떻게 정렬되는지를 지정합니다. 예를 들어, 그룹 문자가 객체 위에 있을 때 VerticalJustification이 Top이면 객체의 위쪽이 기준선에 맞춰짐을 의미하고, VerticalJustification이 Bottom으로 설정되면 객체의 아래쪽이 기준선에 맞춰집니다. |

### 반환값

새 인스턴스 유형 [`IMathGroupingCharacter`](../../imathgroupingcharacter)

### 예제

예제:

```csharp
[C#]
IMathGroupingCharacter groupingElement = new MathematicalText("x;y;z").Group('\u23E1', MathTopBotPositions.Bottom, MathTopBotPositions.Top);
```

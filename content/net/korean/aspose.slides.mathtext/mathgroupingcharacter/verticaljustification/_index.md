## MathGroupingCharacter.VerticalJustification 속성

그룹 문자에 대한 수직 정렬입니다. 객체와 기준선 사이의 정렬을 지정합니다. 예를 들어, 그룹 문자가 객체 위에 있을 때, Top값의 VerticalJustification은 객체의 상단이 기준선에 놓임을 의미합니다; Bottom값으로 설정하면 객체의 하단이 기준선에 놓입니다. 기본값: Bottom for Position=Top, and Top for Position=Bottom

```csharp
public MathTopBotPositions VerticalJustification { get; set; }
```

### 예제

예제:

```csharp
[C#]
MathGroupingCharacter groupingCharacter = new MathGroupingCharacter(new MathematicalText("abc"));
groupingCharacter.VerticalJustification = MathTopBotPositions.Top;
```

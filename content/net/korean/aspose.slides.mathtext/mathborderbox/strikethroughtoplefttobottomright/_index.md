## MathBorderBox.StrikethroughTopLeftToBottomRight 속성

Strikethrough Top-Left to Bottom-Right (기본값은 false). 숨김 또는 표시 상태를 지정합니다. 이는 테두리 상자의 왼쪽 상단 모서리에서 오른쪽 하단 모서리까지 그려지는 취소선 대각선을 의미합니다.

```csharp
public bool StrikethroughTopLeftToBottomRight { get; set; }
```

### 예제

예시:

```csharp
[C#]
MathBorderBox borderBox = new MathBorderBox(new MathematicalText("x"));
borderBox.StrikethroughTopLeftToBottomRight = true;
```

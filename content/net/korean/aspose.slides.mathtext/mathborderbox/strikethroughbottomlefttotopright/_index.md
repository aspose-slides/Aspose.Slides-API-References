## MathBorderBox.StrikethroughBottomLeftToTopRight 속성

Strikethrough Bottom-Left to Top-Right(기본값은 false). 테두리 상자에서 왼쪽 하단 모서리에서 오른쪽 상단 모서리까지 대각선으로 그어지는 취소선의 표시 여부(숨김 또는 표시)를 지정합니다.

```csharp
public bool StrikethroughBottomLeftToTopRight { get; set; }
```

### 예제

예시:

```csharp
[C#]
MathBorderBox borderBox = new MathBorderBox(new MathematicalText("x"));
borderBox.StrikethroughBottomLeftToTopRight = true;
```

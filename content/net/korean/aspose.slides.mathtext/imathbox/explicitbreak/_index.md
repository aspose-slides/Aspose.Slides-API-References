## IMathBox.ExplicitBreak 속성

Explicit break는 Box 객체의 시작 부분에 줄 바꿈이 있는지 여부를 지정하며, 따라서 줄이 Box 객체의 시작 부분에서 래핑됩니다. 이전 수학 텍스트 줄에 있는 연산자의 번호를 지정하며, 이는 현재 수학 텍스트 줄의 정렬 지점으로 사용됩니다. 가능한 값: 1..255 기본값: 0 (명시적 줄 바꿈 없음)

```csharp
public byte ExplicitBreak { get; set; }
```

### 예제

예시:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.ExplicitBreak = 1;
```

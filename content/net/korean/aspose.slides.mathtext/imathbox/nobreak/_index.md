## IMathBox.NoBreak 속성

줄 바꿈 없음. 이 속성은 객체 상자에 대한 "unbreakable" 속성을 지정합니다. true일 경우, 상자 내부에서 줄 바꿈이 발생하지 않습니다. 이는 둘 이상의 이진 연산자로 구성된 연산자 에뮬레이터에 중요할 수 있습니다. 이 요소가 지정되지 않으면, 상자 내부에서 줄 바꿈이 발생할 수 있습니다. 기본값: true

```csharp
public bool NoBreak { get; set; }
```

### 예제

예시:

```csharp
[C#]
IMathBox box = new MathematicalText("**********").ToBox();
box.NoBreak = false;
```

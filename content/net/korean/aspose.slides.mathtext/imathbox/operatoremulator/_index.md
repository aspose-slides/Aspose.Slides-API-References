## IMathBox.OperatorEmulator 속성

연산자 에뮬레이터. true이면, 상자와 그 내용물이 단일 연산자처럼 동작하고 연산자의 속성을 상속합니다. 이는 예를 들어, 해당 문자가 줄 바꿈 지점으로 사용할 수 있고 다른 연산자와 정렬될 수 있음을 의미합니다. 연산자 에뮬레이터는 ‘==’와 같이 하나 이상의 글리프가 결합되어 연산자를 형성할 때 흔히 사용됩니다. 기본값: false

```csharp
public bool OperatorEmulator { get; set; }
```

### 예제

예제:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.OperatorEmulator = true;
```

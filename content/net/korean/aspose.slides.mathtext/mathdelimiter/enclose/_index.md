## MathDelimiter.Enclose 메서드

수학 요소를 괄호와 같은 지정된 문자 또는 다른 문자로 둘러싸서 프레이밍합니다.

```csharp
public override IMathDelimiter Enclose(char beginningCharacter, char endingCharacter)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| beginningCharacter | Char | 시작 문자 (보통 왼쪽 괄호) |
| endingCharacter | Char | 끝 문자 (보통 오른쪽 괄호) |

### 반환 값

만약 *beginningCharacter*와 *endingCharacter*가 null이면, 해당 속성에 값만 할당되고 새 객체가 생성되지 않습니다 (이 인스턴스를 반환합니다). 그렇지 않으면, 지정된 문자를 프레이밍으로 포함하고 이 인스턴스 [`MathDelimiter`](../../mathdelimiter)가 내부에 프레임된 Delimiter 유형의 새로운 수학 요소를 반환합니다.

### 예제

예시:

```csharp
[C#]
IMathDelimiter innerDelimiter = new MathematicalText("x").Join(",y").Enclose('{', '}');
IMathDelimiter outerDelimiter = innerDelimiter.Enclose('[', ']');
```

## MathDelimiter.GrowToMatchOperandHeight 속성

BeginningCharacter, SeparatorCharacter, EndingCharacter 의 성장 방식을 지정합니다. true인 경우 구분 기호가 그 피연산자 높이에 맞게 수직으로 확대됩니다. 기본값은 true입니다.

```csharp
public bool GrowToMatchOperandHeight { get; set; }
```

### 예제

예시:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.GrowToMatchOperandHeight = false;
```

## MathDelimiter 생성자

지정된 요소를 단일 기본 인수로 사용하여 MathDelimiter를 초기화합니다.

```csharp
public MathDelimiter(IMathElement element)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| element | IMathElement | 구분기가 적용되는 기본 요소입니다. null일 수 있습니다. |

### 예외

| 예외 | 조건 |
| --- | --- |
| InvalidOperationException | *element*이 MathBlock과 같은 다른 요소들의 컨테이너인 경우 예외가 발생합니다. 이 경우 IEnumerable 인수를 사용하는 다른 생성자를 호출해야 합니다. |

### 예제

예:

```csharp
[C#]
IMathElement element = new MathematicalText("x");
MathDelimiter delimiter = new MathDelimiter(element);
```

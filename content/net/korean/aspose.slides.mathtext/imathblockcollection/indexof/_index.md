## IMathBlockCollection.IndexOf 메서드

컬렉션에서 특정 IMathBlock의 인덱스를 결정합니다.

```csharp
public int IndexOf(IMathBlock item)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| item | IMathBlock | 컬렉션에서 찾을 항목. |

### 반환값

컬렉션에서 *item*을(를) 찾은 경우 해당 인덱스, 찾지 못한 경우 -1.

예제:


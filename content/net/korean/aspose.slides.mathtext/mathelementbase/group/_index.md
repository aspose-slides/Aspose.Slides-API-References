## Group(char, MathTopBotPositions, MathTopBotPositions) {#group_1}

그룹화 문자(예: 하단 중괄호) 또는 다른 문자를 사용하여 이 요소를 그룹에 배치합니다

```csharp
public IMathGroupingCharacter Group(char character, MathTopBotPositions position, 
    MathTopBotPositions verticalJustification)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| character | Char | BOTTOM CURLY BRACKET (U+23DF)와 같은 그룹화 문자 또는 기타 |
| position | MathTopBotPositions | 그룹화 문의 위치 |
| verticalJustification | MathTopBotPositions | 그룹 문자에 대한 수직 정렬. 객체를 기준선에 맞추는 정렬을 지정합니다. 예를 들어, 그룹 문자가 객체 위에 있을 때, VerticalJustification이 Top이면 객체의 상단이 기준선에 맞춰짐을 의미합니다; VerticalJustification이 Bottom으로 설정되면 객체의 하단이 기준선에 맞춰집니다 |

### 반환 값

형식 [`IMathGroupingCharacter`](../../imathgroupingcharacter)의 새 인스턴스

### 예제

예시:

```csharp
[C#]
IMathGroupingCharacter groupingElement = new MathematicalText("x;y;z").Group('\u23E1', MathTopBotPositions.Bottom, MathTopBotPositions.Top);
```

```csharp
public IMathBorderBox ToBorderBox(bool hideTop, bool hideBottom, bool hideLeft, bool hideRight, 
    bool strikethroughHorizontal, bool strikethroughVertical, 
    bool strikethroughBottomLeftToTopRight, bool strikethroughTopLeftToBottomRight)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| hideTop | Boolean | 상단 가장자리 숨기기 |
| hideBottom | Boolean | 하단 가장자리 숨기기 |
| hideLeft | Boolean | 왼쪽 가장자리 숨기기 |
| hideRight | Boolean | 오른쪽 가장자리 숨기기 |
| strikethroughHorizontal | Boolean | 보더 박스 가로 취소선 |
| strikethroughVertical | Boolean | 보더 박스 세로 취소선 |
| strikethroughBottomLeftToTopRight | Boolean | 보더 박스 왼쪽 아래에서 오른쪽 위로 취소선 |
| strikethroughTopLeftToBottomRight | Boolean | 보더 박스 왼쪽 위에서 오른쪽 아래로 취소선 |

### 반환값

이 요소가 배치된 border-box

### 예제

예:

```csharp
[C#]
```

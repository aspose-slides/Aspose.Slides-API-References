## ToTiff(Presentation, string) {#totiff}

입력 프레젠테이션을 TIFF 형식 이미지 집합으로 변환합니다. 출력 파일 이름을 "myPath/myFilename.tiff"로 지정하면 결과는 "myPath/myFilename_N.tiff" 파일 집합으로 저장되며, N은 슬라이드 번호입니다.

public static void ToTiff(Presentation pres, string outputFileName)

## ToTiff(Presentation, string, ITiffOptions, bool) {#totiff_1}

입력 프레젠테이션을 사용자 정의 옵션으로 TIFF 형식으로 변환합니다. 출력 파일 이름을 "myPath/myFilename.tiff"로 지정하고 *multipage*가 `false`인 경우 결과는 "myPath/myFilename_N.tiff" 파일 집합으로 저장되며, N은 슬라이드 번호입니다. *multipage*가 `true`인 경우 결과는 다중 페이지 "myPath/myFilename.tiff" 문서가 됩니다.

```csharp
public static void ToTiff(Presentation pres, string outputFileName, ITiffOptions options, 
    bool multipage)
```

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| pres | Presentation | 입력 프레젠테이션입니다. |
| outputFileName | String | 출력 파일 이름입니다. |
| options | ITiffOptions | TIFF 저장 옵션입니다. |
| multipage | Boolean | 생성된 TIFF 문서가 다중 페이지여야 하는지 지정합니다. |

### 예외

| 예외 | 조건 |
| --- | --- |
| ArgumentException |  |

### 예제

```csharp
ITiffOptions options = new TiffOptions()
{
    CompressionType = TiffCompressionTypes.CCITT3,
    SlidesLayoutOptions = new NotesCommentsLayoutingOptions()
    {
        NotesPosition = NotesPositions.BottomTruncated
    }
};

using (var pres = new Presentation("pres.pptx"))
   Convert.ToTiff(pres, "pres.tiff", options, false);
```

## BaseOverrideThemeManager.IsOverrideThemeEnabled 속성

OverrideTheme이 상속된 유효 테마를 재정의하는지 여부를 결정합니다. OverrideTheme을 재정의하려면 OverrideTheme.Init*() 메서드를 사용하십시오. OverrideTheme의 재정을 비활성화하려면 OverrideTheme.Clear() 메서드를 사용하십시오. 읽기 전용 Boolean.

public bool IsOverrideThemeEnabled { get; }

## IOverrideThemeManager.IsOverrideThemeEnabled 속성

OverrideTheme가 상속된 효과적인 테마를 재정의하는지 여부를 결정합니다. OverrideTheme를 재정의하도록 활성화하려면 OverrideTheme.Init*() 메서드를 사용하십시오. OverrideTheme가 재정의하지 않도록 비활성화하려면 OverrideTheme.Clear() 메서드를 사용하십시오. 읽기 전용 Boolean.

public bool IsOverrideThemeEnabled { get; }

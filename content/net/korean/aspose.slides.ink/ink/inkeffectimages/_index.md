## Ink.InkEffectImages 속성

잉크 브러시의 시각 효과를 시뮬레이션하기 위해 사용되는 사용자 정의 이미지 컬렉션을 가져옵니다. 이러한 이미지는 Galaxy, Rainbow 등과 같은 특정 [`InkEffectType`](../../inkeffecttype) 값으로 잉크를 렌더링할 때 사용됩니다. 직접 이미지를 제공함으로써 각 잉크 효과가 어떻게 표시되는지를 제어할 수 있습니다.

```csharp
public static IDictionary<InkEffectType, IImage> InkEffectImages { get; }
```

### 비고

이 속성을 사용하면 기본 잉크 효과 텍스처를 사용자 정의 텍스처로 교체할 수 있으며, 이는 기본 에셋이 라이선스로 제한되거나 런타임에 사용할 수 없을 때 특히 유용합니다. 사전의 각 항목은 [`InkEffectType`](../../inkeffecttype) 값을 해당 [`IImage`](../../../aspose.slides/iimage) 객체(예: Bitmap 또는 Aspose 이미지 인터페이스)와 연결해야 합니다.

### 예제

```csharp
[C#]
IImage image = Images.FromFile("image.png");
Ink.InkEffectImages.Add(InkEffectType.Galaxy, image);
```

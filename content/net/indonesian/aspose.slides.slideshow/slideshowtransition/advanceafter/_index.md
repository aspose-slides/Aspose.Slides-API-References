## SlideShowTransition.AdvanceAfter properti

Atribut ini menentukan apakah slideshow akan beralih ke slide berikutnya setelah waktu tertentu. Baca/tulis Boolean.

```csharp
public bool AdvanceAfter { get; set; }
```

### Contoh

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    // Dapatkan Transisi slide pertama
    ISlideShowTransition slideTransition = pres.Slides[0].SlideShowTransition;
    
    // Periksa apakah flag Advance Slide After sudah diaktifkan
    if (slideTransition.AdvanceAfter)
    {
        // Dapatkan nilai Advance Slide After Time
        uint advanceAfterTime = slideTransition.AdvanceAfterTime;
    }
}
```

## IChartData.SecondaryCategories properti

Mendapatkan kategori sekunder jika properti [`UseSecondaryCategories`](../usesecondarycategories) bernilai true. Hanya-baca [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### Catatan

Jika properti [`UseSecondaryCategories`](../usesecondarycategories) bernilai false maka properti `SecondaryCategories` ini mengembalikan null dan data dalam properti [`Categories`](../categories) digunakan baik untuk seri primer maupun sekunder. Jika properti [`UseSecondaryCategories`](../usesecondarycategories) bernilai true maka data dalam properti `SecondaryCategories` ini digunakan untuk seri sekunder dan data dalam properti [`Categories`](../categories) digunakan untuk seri primer.

### Contoh

Contoh. Kategori apa yang terkait dengan seri - ChartData.Categories atau ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // kategori terkait adalah series.Chart.ChartData.SecondaryCategories
}
else
{
    // kategori terkait adalah series.Chart.ChartData.Categories
}
```

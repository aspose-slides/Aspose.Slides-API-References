## IChartData.UseSecondaryCategories properti

Jika false maka [`SecondaryCategories`](../secondarycategories) properti mengembalikan null dan data di [`Categories`](../categories) properti digunakan untuk seri utama dan sekunder. Jika true maka data di [`SecondaryCategories`](../secondarycategories) properti digunakan untuk seri sekunder dan data di [`Categories`](../categories) properti digunakan untuk seri utama. Baca/tulis Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

### Contoh

Contoh. Kategori apa yang terkait dengan seri - ChartData.Categories atau ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // kategori terkait adalah series.Chart.ChartData.SecondaryCategories
}
else
{
    // kategori terkait adalah series.Chart.ChartData.Categories
}
```

## IChartCategory.UseCell property

Jika true maka properti AsCell berlaku. Dengan kata lain, worksheet digunakan untuk menyimpan kategori (kasus ini mendukung kategori multi-level). Jika false maka properti AsLiteral berlaku. Dengan kata lain, worksheet TIDAK digunakan untuk menyimpan kategori (dan kasus ini tidak mendukung kategori multi-level). Hanya baca Boolean.

```csharp
public bool UseCell { get; }
```

### Catatan

Untuk mengubah nilai properti ini (untuk semua kategori dalam koleksi), set nilai baru ke properti ChartCategoryCollection.UseCells.

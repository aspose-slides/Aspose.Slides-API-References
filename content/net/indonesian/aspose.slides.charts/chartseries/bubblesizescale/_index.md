## ChartSeries.BubbleSizeScale properti

Menentukan faktor skala untuk diagram gelembung (dapat berada di antara 0 dan 300 persen dari ukuran default). Ini adalah properti bukan hanya dari seri ini tetapi dari semua seri dalam grup seri induk - ini merupakan proyeksi properti grup yang sesuai. Karena itu properti ini hanya-baca. Gunakan properti ParentSeriesGroup untuk mengakses grup seri induk. Gunakan properti read/write ParentSeriesGroup.BubbleSizeScale untuk mengubah nilai.

```csharp
public int BubbleSizeScale { get; }
```

Ini merupakan proyeksi dari properti ParentSeriesGroup.BubbleSizeScale.

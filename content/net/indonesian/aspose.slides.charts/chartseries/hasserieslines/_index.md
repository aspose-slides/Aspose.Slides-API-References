## ChartSeries.HasSeriesLines properti

Menentukan apakah ada garis seri untuk seri ini dan seri sejenis. Ini adalah properti bukan hanya untuk seri ini tetapi untuk semua seri dalam grup seri induk – ini adalah proyeksi properti grup yang sesuai. Oleh karena itu properti ini hanya-baca. Gunakan properti ParentSeriesGroup untuk mengakses kelompok seri induk. Gunakan properti baca/tulis ParentSeriesGroup.HasSeriesLines untuk mengubah nilai. Gunakan properti ParentSeriesGroup.SeriesLinesFormat untuk memformat garis seri. Hanya-baca Boolean.

public bool HasSeriesLines { get; }

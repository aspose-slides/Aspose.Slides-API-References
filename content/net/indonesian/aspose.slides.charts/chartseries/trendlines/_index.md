## ChartSeries.TrendLines properti

Koleksi garis tren seri. Hanya baca [`ITrendlineCollection`](../../itrendlinecollection).

```csharp
public ITrendlineCollection TrendLines { get; }
```

### Catatan

TrendLines tersedia (tidak null) untuk seri data dalam diagram area 2-D tidak ditumpuk, batang, kolom, garis, saham, xy (sebar), dan gelembung. Garis tren tidak tersedia untuk seri data dalam tipe diagram apa pun yang ditumpuk atau 3-D. Garis tren juga tidak tersedia untuk diagram radar, pai, permukaan, atau donat.

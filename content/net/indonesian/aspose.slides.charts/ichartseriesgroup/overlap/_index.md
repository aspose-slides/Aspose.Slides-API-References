## Properti IChartSeriesGroup.Overlap

Menentukan berapa banyak batang dan kolom yang harus saling tumpang pada grafik 2-D, sebagai persentase (dari -100% hingga 100%). - -100%: Jarak maksimum (batang sepenuhnya terpisah). - 0%: Batang ditempatkan berdampingan tanpa tumpang tindih atau jarak. - 100%: Tumpang tindih maksimum (batang sepenuhnya tumpang tindih satu sama lain). Properti ini dapat dibaca/ditulis SByte.

```csharp
public sbyte Overlap { get; set; }
```

### Pengecualian

| pengecualian | kondisi |
| --- | --- |
| ArgumentOutOfRangeException | Dilempar ketika nilai diatur di luar rentang valid -100 hingga 100. |

### Contoh

Contoh berikut menunjukkan cara mengatur tumpang tindih untuk grup seri diagram dan merender diagram yang dihasilkan pada sebuah formulir:

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // Atur tumpang tindih menjadi 55%

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

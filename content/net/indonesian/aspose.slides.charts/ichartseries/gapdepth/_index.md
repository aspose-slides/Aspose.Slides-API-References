## IChartSeries.GapDepth properti

Mengembalikan atau mengatur jarak, sebagai persentase lebar penanda, antara seri data dalam grafik 3D. Ini bukan properti hanya untuk seri ini tetapi untuk semua seri dalam grup seri induk — ini merupakan proyeksi properti grup yang sesuai. Oleh karena itu properti ini hanya-baca. Gunakan properti ParentSeriesGroup untuk mengakses grup seri induk. Gunakan properti ParentSeriesGroup.GapDepth baca/tulis untuk mengubah nilai. Hanya-baca Int32.

```csharp
public int GapDepth { get; }
```

### Catatan

Ini adalah proyeksi properti ParentSeriesGroup.GapDepth.

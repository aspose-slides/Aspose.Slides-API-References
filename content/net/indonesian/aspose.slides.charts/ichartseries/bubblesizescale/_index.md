## IChartSeries.BubbleSizeScale properti

Menentukan faktor skala untuk diagram gelembung (dapat berada antara 0 dan 300 persen dari ukuran default). Ini merupakan properti tidak hanya dari seri ini tetapi juga dari semua seri dalam grup seri induk – ini adalah proyeksi dari properti grup yang sesuai. Karena itu properti ini bersifat baca-saja. Gunakan properti ParentSeriesGroup untuk mengakses grup seri induk. Gunakan properti ParentSeriesGroup.BubbleSizeScale baca/tulis untuk mengubah nilai.

```csharp
public int BubbleSizeScale { get; }
```

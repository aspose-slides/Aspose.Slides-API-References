## IChartSeries.GapWidth properti

Menentukan ruang antara klaster batang atau kolom, sebagai persentase dari lebar batang atau kolom. Properti ini tidak hanya berlaku untuk seri ini tetapi juga untuk semua seri dalam grup seri induk – ini merupakan proyeksi properti grup yang bersangkutan. Oleh karena itu properti ini bersifat read-only. Gunakan properti ParentSeriesGroup untuk mengakses grup seri induk. Gunakan properti ParentSeriesGroup.GapWidth read/write untuk mengubah nilai. Read-only Int32.

```csharp
public int GapWidth { get; }
```

### Catatan

Ini adalah proyeksi properti ParentSeriesGroup.GapWidth.

## IChartSeries.PieSplitBy properti

Menentukan cara menentukan titik data mana yang berada di pai atau batang kedua pada diagram pai-dan-pai atau batang-dan-pai. Ini adalah properti tidak hanya untuk seri ini tetapi untuk semua seri dalam grup seri induk — ini adalah proyeksi properti grup yang sesuai. Karena itu properti ini hanya-baca. Gunakan properti ParentSeriesGroup untuk mengakses grup seri induk. Gunakan properti baca/tulis ParentSeriesGroup.PieSplitBy untuk mengubah nilai. Hanya-baca [`PieSplitType`](../../piesplittype).

```csharp
public PieSplitType PieSplitBy { get; }
```

### Catatan

1) Ini adalah proyeksi dari properti ParentSeriesGroup.PieSplitBy. 2) Jika nilai properti adalah PieSplitType.Custom maka Anda dapat menentukan informasi pembagian khusus dengan properti ParentSeriesGroup.PieSplitCustomPoints.

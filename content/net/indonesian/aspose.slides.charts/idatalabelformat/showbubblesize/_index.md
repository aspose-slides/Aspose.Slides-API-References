## IDataLabelFormat.ShowBubbleSize properti

Mewakili perilaku tampilan nilai ukuran gelembung label data pada diagram tertentu. True menampilkan nilai ukuran gelembung. False untuk menyembunyikannya. Baca/tulis Boolean.

```csharp
public bool ShowBubbleSize { get; set; }
```

### Catatan

Jika induk dari objek DataLabelFormat ini adalah kumpulan DataLabelCollection berisi label data, maka properti ini memperoleh atau menetapkan nilai default properti ShowBubbleSize untuk label data baru dalam kumpulan DataLabelCollection. Menetapkan properti ini dengan nilai juga menetapkan nilai tersebut ke properti ShowBubbleSize untuk semua label data dalam kumpulan DataLabelCollection (misalnya "DataLabels.DefaultDataLabelFormat.ShowBubbleSize = val;" menyebabkan semua DataLabels[i].ShowBubbleSize menjadi sama dengan val).

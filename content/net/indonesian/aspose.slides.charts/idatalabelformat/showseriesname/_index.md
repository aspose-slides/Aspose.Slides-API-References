## IDataLabelFormat.ShowSeriesName properti

Mengembalikan atau mengatur Boolean untuk menunjukkan perilaku tampilan nama seri pada label data di bagan. True untuk menampilkan nama seri. False untuk menyembunyikannya. Baca/tulis Boolean.

```csharp
public bool ShowSeriesName { get; set; }
```

### Catatan

Jika induk dari objek DataLabelFormat ini adalah koleksi DataLabelCollection dari label data maka properti ini mendapatkan atau mengatur nilai default properti ShowSeriesName untuk label data baru dalam koleksi DataLabelCollection. Mengatur properti ini dengan nilai juga mengatur nilai ini ke properti ShowSeriesName untuk semua label data dalam koleksi DataLabelCollection (mis. "DataLabels.DefaultDataLabelFormat.ShowSeriesName = val;" menyebabkan semua DataLabels[i].ShowSeriesName menjadi sama dengan val).

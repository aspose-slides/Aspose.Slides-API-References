## IDataLabelFormat.ShowPercentage properti

Mewakili perilaku tampilan nilai persentase label data pada diagram yang ditentukan. True menampilkan nilai persentase. False untuk menyembunyikan. Baca/tulis Boolean.

```csharp
public bool ShowPercentage { get; set; }
```

### Catatan

Jika induk objek DataLabelFormat ini adalah koleksi DataLabelCollection dari label data, maka properti ini mendapatkan atau mengatur nilai default properti ShowPercentage untuk label data baru dalam koleksi DataLabelCollection. Mengatur properti ini dengan nilai juga mengatur nilai ini ke properti ShowPercentage untuk semua label data dalam koleksi DataLabelCollection (misalnya "DataLabels.DefaultDataLabelFormat.ShowPercentage = val;" menyebabkan semua DataLabels[i].ShowPercentage menjadi sama dengan val).

## ChartData.UseSecondaryCategories properti

Jika false maka properti [`SecondaryCategories`](../secondarycategories) mengembalikan null dan data pada properti [`Categories`](../categories) digunakan untuk seri utama dan sekunder. Jika true maka data pada properti [`SecondaryCategories`](../secondarycategories) digunakan untuk seri sekunder dan data pada properti [`Categories`](../categories) digunakan untuk seri utama. Baca/tulis Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

### Contoh

Contoh. Kategori apa yang terkait dengan seri - ChartData.Categories atau ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // kategori terkait adalah series.Chart.ChartData.SecondaryCategories
}
else
{
    // kategori terkait adalah series.Chart.ChartData.Categories
}
```

## ChartData.SecondaryCategories properti

Mendapatkan kategori sekunder jika [`UseSecondaryCategories`](../usesecondarycategories) properti bernilai true. Baca-saja [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### Catatan

Jika [`UseSecondaryCategories`](../usesecondarycategories) properti bernilai false maka `SecondaryCategories` properti ini mengembalikan null dan data dalam [`Categories`](../categories) properti digunakan untuk seri primer dan sekunder. Jika [`UseSecondaryCategories`](../usesecondarycategories) properti bernilai true maka data dalam `SecondaryCategories` properti ini digunakan untuk seri sekunder dan data dalam [`Categories`](../categories) properti digunakan untuk seri primer.

### Contoh

Contoh. Kategori mana yang terkait dengan seri - ChartData.Categories atau ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // kategori terkait adalah series.Chart.ChartData.SecondaryCategories
}
else
{
    // kategori terkait adalah series.Chart.ChartData.Categories
}
```

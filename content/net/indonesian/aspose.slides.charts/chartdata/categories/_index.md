## ChartData.Categories properti

Mengambil kategori utama (atau kategori utama dan sekunder jika properti [`UseSecondaryCategories`](../usesecondarycategories) bernilai false). Hanya baca [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection Categories { get; }
```

### Catatan

Jika properti [`UseSecondaryCategories`](../usesecondarycategories) bernilai false maka properti [`SecondaryCategories`](../secondarycategories) mengembalikan null dan data pada properti `Categories` ini digunakan baik untuk seri utama maupun sekunder. Jika properti [`UseSecondaryCategories`](../usesecondarycategories) bernilai true maka data pada properti [`SecondaryCategories`](../secondarycategories) digunakan untuk seri sekunder dan data pada properti `Categories` ini digunakan untuk seri utama.

### Contoh

Contoh. Kategori mana yang terkait dengan seri - ChartData.Categories atau ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // kategori terkait adalah series.Chart.ChartData.SecondaryCategories
}
else
{
    // kategori terkait adalah series.Chart.ChartData.Categories
}
```

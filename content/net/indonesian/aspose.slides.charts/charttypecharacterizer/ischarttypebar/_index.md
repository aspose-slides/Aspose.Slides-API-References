## ChartTypeCharacterizer.IsChartTypeBar metode

Mengembalikan true jika chartType adalah salah satu subtipe Bar. Set subtipe sesuai dengan set yang tepat di PowerPoint (lihat dialog "Change Chart Type" di PowerPoint): ClusteredBar, ClusteredBar3D, PercentsStackedBar, PercentsStackedBar3D, StackedBar, StackedBar3D, ClusteredHorizontalCone, ClusteredHorizontalCylinder, ClusteredHorizontalPyramid, StackedHorizontalCone, StackedHorizontalCylinder, StackedHorizontalPyramid, PercentsStackedHorizontalCone, PercentsStackedHorizontalCylinder, PercentsStackedHorizontalPyramid.

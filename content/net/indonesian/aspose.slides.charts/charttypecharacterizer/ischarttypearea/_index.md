## ChartTypeCharacterizer.IsChartTypeArea metode

Mengembalikan true jika chartType adalah salah satu subtipe Area. Set subtipe sesuai dengan set yang tepat di PowerPoint (lihat dialog "Change Chart Type" di PowerPoint): Area, PercentsStackedArea, PercentsStackedArea3D, StackedArea, StackedArea3D, Area3D.

## ChartTypeCharacterizer.IsChartTypeScatter method

Mengembalikan true jika chartType adalah salah satu subtipe Scatter. Set subtipe sesuai dengan set yang tepat di PowerPoint (lihat dialog "Change Chart Type" di PowerPoint): ScatterWithMarkers, ScatterWithSmoothLines, ScatterWithSmoothLinesAndMarkers, ScatterWithStraightLines, ScatterWithStraightLinesAndMarkers.

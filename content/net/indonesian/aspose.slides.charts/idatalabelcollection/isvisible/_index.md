## IDataLabelCollection.IsVisible properti

False berarti label data tidak terlihat secara default (dan sehingga semua Show*-flags (ShowValue, ...) dari properti DefaultDataLabelFormat adalah false). Baca-saja Boolean.

```csharp
public bool IsVisible { get; }
```

### Keterangan

Jika label data terlihat secara default Anda dapat menyembunyikannya secara default dengan metode Hide(). Namun jika label data tidak terlihat secara default (IsVisible adalah false) Anda dapat membuat label data "terlihat secara default" dengan mengatur Show*-flags (ShowValue, ...) dari properti DefaultDataLabelFormat ke keadaan true.

## DataLabelCollection.Hide metode

Sembunyikan label data secara default dengan mengatur semua flag Show*- (ShowValue, ...) pada properti DefaultDataLabelFormat menjadi false. IsVisible akan menjadi false setelahnya.

```csharp
public void Hide()
```

### Catatan

Jika label data tidak terlihat secara default (IsVisible false) Anda dapat membuat label data “terlihat secara default” dengan mengatur flag Show*- (ShowValue, ...) pada properti DefaultDataLabelFormat menjadi true.

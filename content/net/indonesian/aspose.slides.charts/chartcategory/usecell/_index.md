## ChartCategory.UseCell properti

Jika true maka properti AsCell yang berlaku. Dengan kata lain, worksheet digunakan untuk menyimpan kategori (kasus ini mendukung kategori bertingkat multi). Jika false maka properti AsLiteral yang berlaku. Dengan kata lain, worksheet TIDAK digunakan untuk menyimpan kategori (dan kasus ini tidak mendukung kategori bertingkat multi). Baca-saja Boolean.

```csharp
public bool UseCell { get; }
```

### Catatan

Untuk mengubah nilai properti ini (untuk semua kategori dalam koleksi) atur nilai baru ke properti ChartCategoryCollection.UseCells.

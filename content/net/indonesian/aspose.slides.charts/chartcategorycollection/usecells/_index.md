## ChartCategoryCollection.UseCells properti

Jika true maka worksheet digunakan untuk menyimpan kategori (kasus ini mendukung kategori multi-level). Jika false maka worksheet TIDAK digunakan untuk menyimpan nilai (dan kasus ini tidak mendukung kategori multi-level). Baca/tulis Boolean.

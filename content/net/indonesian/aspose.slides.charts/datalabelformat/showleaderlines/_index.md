## DataLabelFormat.ShowLeaderLines properti

Mewakili perilaku tampilan garis pemimpin label data pada diagram yang ditentukan. True menampilkan garis pemimpin. False untuk menyembunyikannya. Baca/tulis Boolean.

```csharp
public bool ShowLeaderLines { get; set; }
```

### Keterangan

Jika induk dari objek DataLabelFormat ini adalah koleksi DataLabelCollection dari label data maka properti ini mendapatkan atau mengatur nilai default properti ShowLeaderLines untuk label data baru dalam koleksi DataLabelCollection. Mengatur properti ini dengan nilai juga mengatur nilai ini ke properti ShowLeaderLines untuk semua label data dalam koleksi DataLabelCollection (misalnya "DataLabels.DefaultDataLabelFormat.ShowLeaderLines = val;" menyebabkan semua DataLabels[i].ShowLeaderLines sama dengan val).

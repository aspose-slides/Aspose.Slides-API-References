## DataLabelFormat.ShowValue properti

Mewakili perilaku tampilan nilai persentase label data pada diagram yang ditentukan. True menampilkan nilai persentase. False untuk menyembunyikannya. Baca/tulis Boolean.

```csharp
public bool ShowValue { get; set; }
```

### Catatan

Jika parent dari objek DataLabelFormat ini adalah koleksi DataLabelCollection dari label data maka properti ini memperoleh atau mengatur nilai default dari properti ShowValue untuk label data baru dalam koleksi DataLabelCollection. Menetapkan properti ini dengan nilai juga mengatur nilai ini ke properti ShowValue untuk semua label data dalam koleksi DataLabelCollection (misalnya "DataLabels.DefaultDataLabelFormat.ShowValue = val;" menyebabkan semua DataLabels[i].ShowValue menjadi sama dengan val).

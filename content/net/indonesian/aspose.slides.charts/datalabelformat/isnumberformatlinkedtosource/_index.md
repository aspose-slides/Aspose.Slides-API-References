## DataLabelFormat.IsNumberFormatLinkedToSource properti

Baca/tulis Boolean.

```csharp
public bool IsNumberFormatLinkedToSource { get; set; }
```

### Catatan

Jika induk objek DataLabelFormat ini adalah koleksi DataLabelCollection dari label data, maka properti ini mendapatkan atau menetapkan nilai default dari properti IsNumberFormatLinkedToSource untuk label data baru dalam koleksi DataLabelCollection. Menetapkan properti ini dengan nilai juga menetapkan nilai tersebut ke properti IsNumberFormatLinkedToSource untuk semua label data dalam koleksi DataLabelCollection (misalnya "DataLabels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = val;" menyebabkan semua DataLabels[i].IsNumberFormatLinkedToSource menjadi sama dengan val).

## DataLabelFormat.Separator properti

Mengatur atau mengembalikan Variant yang mewakili pemisah yang digunakan untuk label data pada grafik. Baca/tulis String.

```csharp
public string Separator { get; set; }
```

### Keterangan

Jika induk dari objek DataLabelFormat ini adalah koleksi DataLabelCollection dari label data, maka properti ini mendapatkan atau mengatur nilai default dari properti Separator untuk label data baru dalam koleksi DataLabelCollection. Mengatur properti ini dengan nilai juga mengatur nilai tersebut ke properti Separator untuk semua label data dalam koleksi DataLabelCollection (misalnya "DataLabels.DefaultDataLabelFormat.Separator = val;" menyebabkan semua DataLabels[i].Separator menjadi sama dengan val).

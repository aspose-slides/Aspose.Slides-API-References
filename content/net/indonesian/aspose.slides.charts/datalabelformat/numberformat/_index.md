## DataLabelFormat.NumberFormat properti

Mewakili string format untuk objek DataLabels. Baca/tulis String.

```csharp
public string NumberFormat { get; set; }
```

### Keterangan

Jika induk dari objek DataLabelFormat ini adalah koleksi DataLabelCollection berisi label data, maka properti ini mengambil atau mengatur nilai default dari properti NumberFormat untuk label data baru dalam koleksi DataLabelCollection. Ketika properti ini diatur dengan sebuah nilai, nilai tersebut juga diatur untuk properti NumberFormat untuk semua label data dalam koleksi DataLabelCollection (misalnya "DataLabels.DefaultDataLabelFormat.NumberFormat = val;" menyebabkan semua DataLabels[i].NumberFormat menjadi val).

### Contoh

```csharp
[C#]
series.Labels.DefaultDataLabelFormat.ShowValue = true;
series.Labels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = false;
series.Labels.DefaultDataLabelFormat.NumberFormat = "0.0%";
```

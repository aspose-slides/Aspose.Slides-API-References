## IChartCategoryCollection.UseCells properti

Jika bernilai true maka worksheet digunakan untuk menyimpan kategori (kasus ini mendukung kategori berlapis). Jika bernilai false maka worksheet TIDAK digunakan untuk menyimpan nilai (dan kasus ini tidak mendukung kategori berlapis). Baca/tulis Boolean.

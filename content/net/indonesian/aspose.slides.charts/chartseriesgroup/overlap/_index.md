## ChartSeriesGroup.Overlap properti

Menentukan sejauh mana batang dan kolom akan tumpang tindih pada diagram 2-D, sebagai persentase (dari -100% hingga 100%). - -100%: Jarak maksimum (batang benar-benar terpisah). - 0%: Batang ditempatkan berdampingan tanpa tumpang tindih atau jarak. - 100%: Tumpang tindih maksimum (batang sepenuhnya menumpuk satu sama lain). Properti ini adalah baca/tulis SByte.

```csharp
public sbyte Overlap { get; set; }
```

### Exceptions

| eksepsi | kondisi |
| --- | --- |
| ArgumentOutOfRangeException | Dilemparkan ketika nilai diatur di luar rentang yang valid antara -100 hingga 100. |

### Examples

Contoh berikut menunjukkan cara mengatur tumpang tindih untuk grup seri diagram dan merender diagram yang dihasilkan pada sebuah formulir:

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // Atur tumpang tindih menjadi 55%

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

## DataLabel.IsVisible properti

False berarti label data tidak terlihat (dan sehingga semua Show*-flags (ShowValue, ...) adalah false). Hanya-baca Boolean.

```csharp
public bool IsVisible { get; }
```

### Catatan

Jika label data terlihat Anda dapat menyembunyikannya dengan metode Hide(). Namun jika label data tidak terlihat (IsVisible is false) Anda dapat membuat label data terlihat dengan mengatur Show*-flags (ShowValue, ...) ke keadaan true.

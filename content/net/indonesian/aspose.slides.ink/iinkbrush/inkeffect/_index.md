## IInkBrush.InkEffect properti

Mendapatkan jenis efek tinta (mis., Galaxy, Gold, Silver) yang mendefinisikan gaya visual goresan tinta. Nilai diparsing dari brush properti "inkEffects". Jika tidak ada efek yang dikenali yang ditentukan, NotDefined dikembalikan.

```csharp
public InkEffectType InkEffect { get; }
```

### Contoh

Contoh:

```csharp
[C#]
using (var pres = new Presentation("Presentation.pptx"))
{
    Ink ink = pres.Slides[0].Shapes[0] as Ink;
    IInkBrush brush = ink.Traces[0].Brush;
    Console.WriteLine("InkEffects = {0}", brush.InkEffect);
}
```

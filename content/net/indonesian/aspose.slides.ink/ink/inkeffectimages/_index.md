## Ink.InkEffectImages properti

Mendapatkan koleksi gambar khusus yang digunakan untuk mensimulasikan efek visual untuk kuas tinta. Gambar-gambar ini digunakan saat merender tinta dengan nilai [`InkEffectType`](../../inkeffecttype) tertentu, seperti Galaxy, Rainbow, dll. Dengan menyediakan gambar Anda sendiri, Anda dapat mengontrol bagaimana setiap efek tinta muncul.

```csharp
public static IDictionary<InkEffectType, IImage> InkEffectImages { get; }
```

### Catatan

Properti ini memungkinkan mengganti tekstur efek tinta default dengan yang didefinisikan pengguna, yang sangat berguna ketika aset default dibatasi oleh lisensi atau tidak tersedia saat runtime. Setiap entri dalam kamus harus mengaitkan nilai [`InkEffectType`](../../inkeffecttype) dengan objek [`IImage`](../../../aspose.slides/iimage) yang sesuai (mis., Bitmap, atau antarmuka gambar Aspose).

### Contoh

```csharp
[C#]
IImage image = Images.FromFile("image.png");
Ink.InkEffectImages.Add(InkEffectType.Galaxy, image);
```

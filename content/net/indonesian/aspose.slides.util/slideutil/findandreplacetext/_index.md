## SlideUtil.FindAndReplaceText metode

Menemukan dan menggantikan teks dalam presentasi dengan format yang diberikan

```csharp
public static void FindAndReplaceText(IPresentation presentation, bool withMasters, string find, 
    string replace, PortionFormat format = null)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| presentation | IPresentation | Presentasi yang dipindai. |
| withMasters | Boolean | Menentukan apakah slide master harus dipindai. |
| find | String | Nilai string untuk dicari. |
| replace | String | Nilai string untuk diganti. |
| format | PortionFormat | Format untuk menggantikan bagian teks. Jika null maka akan digunakan format dari karakter pertama dari string yang ditemukan. |

### Contoh

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PortionFormat format = new PortionFormat
    {
        FontHeight = 24f,
        FontItalic = NullableBool.True,
        FillFormat =
        {
            FillType = FillType.Solid,
            SolidFillColor =
            {
                Color = Color.Red
            }
        }
    };
    Aspose.Slides.Util.SlideUtil.FindAndReplaceText(pres, true, "[this block] ", "my text ", format);
    pres.Save("replaced", SaveFormat.Pptx);
}
```

## Effect.AnimateTextType properti

Mendefinisikan tipe animasi teks untuk efek. Teks bentuk dapat dianimasikan per huruf, per kata, atau sekaligus. Baca/tulis `AnimateTextType`.

```csharp
public AnimateTextType AnimateTextType { get; set; }
```

### Contoh

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Dapatkan efek pertama dari slide pertama.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Ubah tipe animasi teks efek menjadi "By letter"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByLetter;
}
```

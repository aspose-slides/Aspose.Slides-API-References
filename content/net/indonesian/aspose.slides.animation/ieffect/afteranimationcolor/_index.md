## IEffect.AfterAnimationColor properti

Mendefinisikan warna animasi setelah untuk efek. Baca/tulis [`IColorFormat`](../../../aspose.slides/icolorformat).

```csharp
public IColorFormat AfterAnimationColor { get; set; }
```

### Contoh

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Dapatkan efek pertama dari slide pertama.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Ubah tipe animasi setelah efek menjadi "Color"
    firstSlideEffect.AfterAnimationType = AfterAnimationType.Color;
    
    // Atur warna animasi setelah efek.
    firstSlideEffect.AfterAnimationColor.Color = Color.Green;
}
```

## Timing.RepeatUntilNextClick properti

Atribut ini menentukan apakah efek akan diulang hingga klik berikutnya. Baca/tulis Boolean.

```csharp
public bool RepeatUntilNextClick { get; set; }
```

### Contoh

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Dapatkan urutan efek untuk slide pertama
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Dapatkan efek pertama dari urutan utama.
    IEffect effect = effectsSequence[0];

    // Ubah Timing/Repeat menjadi "Until Next Click"
    effect.Timing.RepeatUntilNextClick = true;
}
```

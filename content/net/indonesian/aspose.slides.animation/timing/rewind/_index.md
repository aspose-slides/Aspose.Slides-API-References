## Timing.Rewind properti

Atribut ini menentukan apakah efek akan diputar kembali setelah selesai diputar. Baca/tulis Boolean.

```csharp
public bool Rewind { get; set; }
```

### Contoh

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Dapatkan urutan efek untuk slide pertama
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Dapatkan efek pertama dari urutan utama.
    IEffect effect = effectsSequence[0];

    // Aktifkan Timing/Rewind efek.
    effect.Timing.Rewind = true;
}
```

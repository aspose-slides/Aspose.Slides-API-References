## Metode BehaviorPropertyCollection.CopyTo

Menyalin elemen ICollection ke Array, dimulai pada indeks Array tertentu.

```csharp
public void CopyTo(IBehaviorProperty[] array, int arrayIndex)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| array | IBehaviorProperty[] | Array satu dimensi yang menjadi tujuan elemen yang disalin dari ICollection. Array harus memiliki indeks berbasis nol. |
| arrayIndex | Int32 | Indeks berbasis nol dalam *array* tempat penyalinan dimulai. |

### Pengecualian

| pengecualian | kondisi |
| --- | --- |
| ArgumentNullException | *array* bernilai null. |
| ArgumentOutOfRangeException | *arrayIndex* kurang dari 0. |
| ArgumentException | Jumlah elemen dalam ICollection sumber lebih besar daripada ruang yang tersedia dari *arrayIndex* hingga akhir *array* tujuan. |

## ITiming.RepeatUntilEndSlide properti

Atribut ini menentukan apakah efek akan diulang sampai akhir slide. Baca/tulis Boolean.

```csharp
public bool RepeatUntilEndSlide { get; set; }
```

### Contoh

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Dapatkan urutan efek untuk slide pertama
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Dapatkan efek pertama dari urutan utama.
    IEffect effect = effectsSequence[0];

    // Ubah Timing/Ulangi efek menjadi "Sampai Akhir Slide"
    effect.Timing.RepeatUntilEndSlide = true;
}
```

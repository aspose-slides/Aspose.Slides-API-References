## WebDocument.Output properti

Mengembalikan koleksi elemen output dari dokumen. Hanya-baca `Output`.

```csharp
public Output Output { get; }
```

### Contoh

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    var options = new WebDocumentOptions
    {
        TemplateEngine = new RazorTemplateEngine(),
        OutputSaver = new FileOutputSaver(),
        EmbedImages = false
    };
    
    WebDocument document = new WebDocument(options);

    // tambahkan "index.html" ke file output, menggunakan templat "index" untuk menghasilkan file tersebut dan variabel pres sebagai model
    document.Output.Add("index.html", "index", pres);

    // ... atur opsi lain dari dokumen dan kemudian simpan dokumen
    document.Save();
}
```

## WebDocument.Global properti

Mengembalikan penyimpanan global dokumen. Hanya-baca [`Storage`](../../storage).

```csharp
public Storage Global { get; }
```

### Contoh

Menggunakan properti `Global` ini (implementasi antarmuka [`Storage`](../../storage)) sebuah properti dapat digunakan nanti dalam templat:

```csharp
[C#]
var options = new WebDocumentOptions
{
    TemplateEngine = new RazorTemplateEngine(),
    OutputSaver = new FileOutputSaver(),
    EmbedImages = false
};

WebDocument document = new WebDocument(options);

// letakkan properti "slideMargin" untuk digunakan dari templat
document.Global.Put("slideMargin", 10);

// ... atur opsi lain dokumen dan kemudian simpan dokumen
document.Save();
```

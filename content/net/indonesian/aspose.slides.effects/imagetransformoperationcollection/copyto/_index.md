## ImageTransformOperationCollection.CopyTo metode

Menyalin elemen ICollection ke Array, mulai pada indeks Array tertentu.

```csharp
public void CopyTo(IImageTransformOperation[] array, int arrayIndex)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| array | IImageTransformOperation[] | Array satu dimensi yang menjadi tujuan elemen yang disalin dari ICollection. Array harus memiliki pengindeksan berbasis nol. |
| arrayIndex | Int32 | Indeks berbasis nol dalam *array* tempat penyalinan dimulai. |

### Pengecualian

| eksepsi | kondisi |

## IViewProperties.GridSpacing properti

Mengembalikan atau mengatur jarak kisi yang harus digunakan untuk kisi yang mendasari dokumen presentasi, dalam poin. Baca/tulis Single.

```csharp
public float GridSpacing { get; set; }
```

### Keterangan

Nilai jarak kisi harus berupa angka positif. Kisaran nilai tipikal adalah dari 1 mm (2.8349607 poin) hingga 2 inci (144 poin).

### Contoh

Contoh kode berikut menunjukkan cara mengubah jarak kisi dalam presentasi PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

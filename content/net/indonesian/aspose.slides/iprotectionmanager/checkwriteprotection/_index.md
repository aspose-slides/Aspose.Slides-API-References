## IProtectionManager.CheckWriteProtection metode

Menentukan apakah presentasi dilindungi kata sandi untuk dimodifikasi.

```csharp
public bool CheckWriteProtection(string password)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| password | String | Kata sandi untuk pengecekan. |

### Nilai Kembali

True jika kata sandi valid; jika tidak, false.

### Catatan

1. Anda harus memeriksa properti [`IsWriteProtected`](../iswriteprotected) sebelum memanggil metode ini. 2. Ketika kata sandi null atau kosong, metode ini mengembalikan false.

### Contoh

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

## Slide.WriteAsEmf metode

Menyimpan konten slide sebagai file EMF.

```csharp
public void WriteAsEmf(Stream stream)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| stream | Stream | Alur target |

### Pengecualian

| Pengecualian | kondisi |
| --- | --- |
| ArgumentNullException | Alur target adalah `null` |

### Contoh

Contoh kode berikut menunjukkan cara mengonversi slide pertama dari presentasi PowerPoint menjadi metafile.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.emf"))
    {
        // Menyimpan slide pertama sebagai metafile
        pres.Slides[0].WriteAsEmf(fileStream);
    }
}
```

## IPresentationInfo.CheckPassword method

Memeriksa apakah password benar untuk presentasi yang dilindungi dengan password terbuka.

```csharp
public bool CheckPassword(string password)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| password | String | Password yang akan diperiksa. |

### Nilai Kembali

True jika presentasi dilindungi dengan password terbuka dan passwordnya benar dan false sebaliknya.

### Catatan

Ketika password bernilai null atau kosong, metode ini mengembalikan false.

### Contoh

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo("pres.pptx");
bool isPasswordCorrect = info.CheckPassword("my_password");
```

## IPresentationInfo.CheckWriteProtection metode

Memeriksa apakah kata sandi untuk memodifikasi benar untuk presentasi yang dilindungi penulisan.

```csharp
public bool CheckWriteProtection(string password)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| password | String | Kata sandi yang akan diperiksa. |

### Nilai Kembalian

True if the presentation is write protected and the password is correct. False otherwise.

### Pengecualian

| exception | condition |
| --- | --- |
| InvalidOperationException |  |

### Catatan

1. Anda harus memeriksa properti [`IsWriteProtected`](../iswriteprotected) sebelum memanggil metode ini. 2. Ketika password bernilai null atau kosong, metode ini mengembalikan false.

### Contoh

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
if (info.IsWriteProtected == NullableBool.True)
{
    bool isWriteProtectedByPassword = info.CheckWriteProtection("my_password");
}
```

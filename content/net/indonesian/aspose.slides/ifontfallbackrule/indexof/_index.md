## IFontFallBackRule.IndexOf metode

Mengembalikan indeks dari aturan yang ditentukan dalam koleksi.

```csharp
public int IndexOf(string fontName)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| fontName | String | Nama font yang akan dicari. |

### Nilai Kembali

Indeks font atau -1 jika font tidak ditemukan dalam daftar.

### Contoh

```csharp
[C#]
// Buat sebuah aturan yang berisi daftar font.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

//Get index of Tahoma
//Dapatkan indeks Tahoma
int tahomaIndex = newRule.IndexOf("Tahoma");
```

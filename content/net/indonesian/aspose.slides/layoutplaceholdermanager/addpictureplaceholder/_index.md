## LayoutPlaceholderManager.AddPicturePlaceholder method

Menambahkan bentuk placeholder baru ke slide tata letak untuk menampung gambar.

```csharp
public IAutoShape AddPicturePlaceholder(float x, float y, float width, float height)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| x | Single | Koordinat X dari bentuk placeholder baru. |
| y | Single | Koordinat Y dari bentuk placeholder baru. |
| width | Single | Lebar dari bentuk placeholder baru. |
| height | Single | Tinggi dari bentuk placeholder baru. |

### Nilai Kembali

Dibuat [`IAutoShape`](../../iautoshape) dengan placeholder gambar.

### Contoh

Contoh berikut menunjukkan cara menambahkan bentuk placeholder Gambar ke slide tata letak.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddPicturePlaceholder(20, 20, 200, 200);
}
```

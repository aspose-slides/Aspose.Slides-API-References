## LayoutPlaceholderManager.AddVerticalTextPlaceholder method

Menambahkan shape placeholder baru ke slide tata letak untuk menampung konten teks secara vertikal.

```csharp
public IAutoShape AddVerticalTextPlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| x | Single | Koordinat X dari shape placeholder baru. |
| y | Single | Koordinat Y dari shape placeholder baru. |
| width | Single | Lebar dari shape placeholder baru. |
| height | Single | Tinggi dari shape placeholder baru. |

### Nilai Kembalian

Dibuat [`IAutoShape`](../../iautoshape) dengan placeholder Text (Vertical).

### Contoh

Contoh berikut menunjukkan cara menambahkan shape placeholder Text (Vertical) ke slide tata letak.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalTextPlaceholder(20, 20, 300, 500);
}
```

## LayoutPlaceholderManager.AddChartPlaceholder metode

Menambahkan bentuk placeholder baru ke slide tata letak untuk menampung sebuah Chart.

```csharp
public IAutoShape AddChartPlaceholder(float x, float y, float width, float height)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| x | Single | Koordinat X dari bentuk placeholder baru. |
| y | Single | Koordinat Y dari bentuk placeholder baru. |
| width | Single | Lebar dari bentuk placeholder baru. |
| height | Single | Tinggi dari bentuk placeholder baru. |

### Nilai Kembalian

Created [`IAutoShape`](../../iautoshape) with a Chart placeholder.

### Contoh

Contoh berikut menunjukkan cara menambahkan bentuk placeholder Chart ke slide tata letak.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddChartPlaceholder(20, 20, 200, 200);
}
```

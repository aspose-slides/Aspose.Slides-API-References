## LayoutPlaceholderManager.AddTablePlaceholder metode

Menambahkan shape placeholder baru ke slide tata letak untuk menampung tabel.

```csharp
public IAutoShape AddTablePlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| x | Single | Koordinat X dari shape placeholder baru. |
| y | Single | Koordinat Y dari shape placeholder baru. |
| width | Single | Lebar shape placeholder baru. |
| height | Single | Tinggi shape placeholder baru. |

### Nilai Kembalian

Dibuat [`IAutoShape`](../../iautoshape) dengan placeholder Table.

### Contoh

Contoh berikut menunjukkan cara menambahkan shape placeholder Table ke slide tata letak.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTablePlaceholder(20, 20, 500, 200);
}
```

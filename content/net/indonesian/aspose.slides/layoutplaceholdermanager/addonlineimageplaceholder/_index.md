## LayoutPlaceholderManager.AddOnlineImagePlaceholder metode

Menambahkan bentuk placeholder baru ke slide tata letak untuk menampung gambar daring.

```csharp
public IAutoShape AddOnlineImagePlaceholder(float x, float y, float width, float height)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| x | Single | Koordinat X dari bentuk placeholder baru. |
| y | Single | Koordinat Y dari bentuk placeholder baru. |
| width | Single | Lebar bentuk placeholder baru. |
| height | Single | Tinggi bentuk placeholder baru. |

### Nilai Kembalian

Dibuat [`IAutoShape`](../../iautoshape) dengan placeholder Gambar Daring.

### Contoh

Contoh berikut menunjukkan cara menambahkan bentuk placeholder Gambar Daring ke slide tata letak.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddOnlineImagePlaceholder(20, 20, 200, 200);
}
```

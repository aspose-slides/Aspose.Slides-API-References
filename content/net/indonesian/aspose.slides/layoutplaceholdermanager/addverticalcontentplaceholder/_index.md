## LayoutPlaceholderManager.AddVerticalContentPlaceholder metode

Menambahkan bentuk placeholder baru ke slide tata letak untuk menampung konten, seperti gambar, tabel, media, atau teks dalam arah vertikal.

```csharp
public IAutoShape AddVerticalContentPlaceholder(float x, float y, float width, float height)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| x | Single | Koordinat X dari bentuk placeholder baru. |
| y | Single | Koordinat Y dari bentuk placeholder baru. |
| width | Single | Lebar dari bentuk placeholder baru. |
| height | Single | Tinggi dari bentuk placeholder baru. |

### Nilai Kembalian

Membuat [`IAutoShape`](../../iautoshape) dengan placeholder Konten (Vertikal).

### Contoh

Contoh berikut menunjukkan cara menambahkan bentuk placeholder Konten (Vertikal) ke slide tata letak.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalContentPlaceholder(20, 20, 300, 500);
}
```

## Metode Fonts.SetScriptFont

Menetapkan nama font ke tag skrip tertentu, yang menentukan bagaimana teks dari skrip tersebut akan ditampilkan dalam presentasi.

```csharp
public void SetScriptFont(string script, string fontName)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| script | String | Kode skrip BCP-47 (mis., "Arab", "Hebr", "Hans") yang mengidentifikasi sistem penulisan. |
| fontName | String | Nama font yang akan ditetapkan ke skrip yang ditentukan. |

### Contoh

Contoh ini menunjukkan cara mengatur font untuk skrip Arab menjadi "Segoe UI":

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segoe UI");
```

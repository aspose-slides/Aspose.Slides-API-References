## Fonts.GetScriptFont metode

Mendapatkan nama font yang terkait dengan tag skrip tertentu dari tema presentasi.

```csharp
public string GetScriptFont(string script)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| script | String | Kode skrip BCP-47 (misalnya, "Latn", "Cyrl", "Jpan") yang digunakan untuk mengidentifikasi sistem penulisan. |

### Nilai Kembali

Nama font yang digunakan untuk skrip yang ditentukan, atau `null` jika skrip tidak didefinisikan.

### Contoh

Contoh ini menunjukkan cara mengambil font yang ditetapkan untuk skrip Sirilik dalam tema presentasi.

```csharp
[C#]
string font = presentation.MasterTheme.FontScheme.Major.GetScriptFont("Cyrl");
Console.WriteLine("Font for Cyrillic script: " + font);
```

## Properti Shape.Placeholder

Mengembalikan placeholder untuk sebuah shape. Mengembalikan null jika shape tidak memiliki placeholder. Hanya-baca [`IPlaceholder`](../../iplaceholder).

```csharp
public IPlaceholder Placeholder { get; }
```

### Contoh

Contoh berikut menunjukkan cara mengubah Text dalam Placeholder.

```csharp
[C#]
// Menginstansiasi kelas Presentation
using (Presentation pres = new Presentation("ReplacingText.pptx"))
{
    // Mengakses slide pertama
    ISlide sld = pres.Slides[0];
    // Mengulang melalui shapes untuk menemukan placeholder
    foreach (IShape shp in sld.Shapes)
        if (shp.Placeholder != null)
        {
            // Mengubah teks di setiap placeholder
            ((IAutoShape)shp).TextFrame.Text = "This is a Placeholder";
        }
    // Menyimpan presentasi ke disk
    pres.Save("output_out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
}
```

Contoh berikut menunjukkan cara mengatur Prompt Text dalam Placeholder.

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation2.pptx"))
{
    ISlide slide = pres.Slides[0];
    foreach (IShape shape in slide.Slide.Shapes) // Mengulang melalui slide
    {
        if (shape.Placeholder != null && shape is AutoShape)
        {
            string text = "";
            if (shape.Placeholder.Type == PlaceholderType.CenteredTitle) // PowerPoint menampilkan "Klik untuk menambah judul"
            {
                text = "Add Title";
            }
            else if (shape.Placeholder.Type == PlaceholderType.Subtitle) // Menambahkan subtitle
            {
                text = "Add Subtitle";
            }
            ((IAutoShape)shape).TextFrame.Text = text;
            Console.WriteLine($"Placeholder with text: {text}");
        }
    }
    pres.Save("Placeholders_PromptText.pptx", SaveFormat.Pptx);
}
```

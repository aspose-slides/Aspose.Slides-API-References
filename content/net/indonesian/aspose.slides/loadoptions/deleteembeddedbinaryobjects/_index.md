## LoadOptions.DeleteEmbeddedBinaryObjects properti

Menentukan apakah Aspose.Slides akan menghapus semua objek biner yang disematkan saat memuat presentasi.

Jenis-jenis objek biner yang disematkan:

* Proyek VBA [`VbaProject`](../../ipresentation/vbaproject)
* Data yang disematkan OLE Object [`EmbeddedFileData`](../../ioleembeddeddatainfo/embeddedfiledata)
* Data biner ActiveX Control [`ActiveXControlBinary`](../../icontrol/activexcontrolbinary)

Baca/tulis Boolean.

```csharp
public bool DeleteEmbeddedBinaryObjects { get; set; }
```

### Catatan

Nilai default **false**.

### Contoh

Contoh berikut menunjukkan cara memuat presentasi tanpa objek biner yang disematkan.

```csharp
[C#]
LoadOptions loadOptions = new LoadOptions();
loadOptions.DeleteEmbeddedBinaryObjects = true;
using (Presentation pres = new Presentation("pres.ppt", loadOptions))
{
    pres.Save("output_WithoutBinaryObjects.ppt", SaveFormat.Ppt);
}
```

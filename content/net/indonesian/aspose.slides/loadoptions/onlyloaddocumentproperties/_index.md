## LoadOptions.OnlyLoadDocumentProperties properti

Properti ini masuk akal jika berkas presentasi dilindungi kata sandi. Nilai true berarti hanya properti dokumen yang harus dimuat dari berkas presentasi terenkripsi dan kata sandi harus diabaikan. Nilai false berarti seluruh presentasi terenkripsi harus dimuat dengan menggunakan kata sandi yang benar. Jika presentasi tidak terenkripsi, maka nilai properti selalu diabaikan. Jika properti dokumen dari berkas terenkripsi tidak bersifat publik dan nilai properti adalah true, maka properti dokumen tidak dapat dimuat dan pengecualian akan dilempar. Baca/tulis Boolean.

```csharp
public bool OnlyLoadDocumentProperties { get; set; }
```

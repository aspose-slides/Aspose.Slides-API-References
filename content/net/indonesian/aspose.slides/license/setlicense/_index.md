## SetLicense(string) {#setlicense_1}

Memberi lisensi pada komponen.

```csharp
public void SetLicense(string licenseName)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| licenseName | String | Dapat berupa nama berkas lengkap atau singkat, atau nama sumber daya tertanam. Gunakan string kosong untuk beralih ke mode evaluasi. |

### Catatan

Mencari lisensi di lokasi berikut:

1. Jalur eksplisit.
2. Folder assembly komponen.
3. Folder assembly pemanggil klien.
4. Folder assembly entri.
5. Sumber daya tertanam di assembly pemanggil klien.

**Catatan:** Pada .NET Compact Framework, hanya mencari lisensi di lokasi berikut:

1. Jalur eksplisit.
2. Sumber daya tertanam di assembly pemanggil klien.

### Contoh

Dalam contoh ini, sistem akan mencoba menemukan berkas lisensi bernama MyLicense.lic di folder yang berisi komponen, di folder yang berisi assembly pemanggil, di folder assembly entri, dan kemudian di sumber daya tertanam pada assembly pemanggil.

```csharp
[C#]

License license = new License();
license.SetLicense("MyLicense.lic");


[Visual Basic]

Dim license As License = New License
license.SetLicense("MyLicense.lic")
```

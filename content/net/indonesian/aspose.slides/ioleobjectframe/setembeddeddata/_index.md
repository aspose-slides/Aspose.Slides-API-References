## IOleObjectFrame.SetEmbeddedData metode

Sets information about OLE embedded data.

```csharp
public void SetEmbeddedData(IOleEmbeddedDataInfo embeddedData)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| embeddedData | IOleEmbeddedDataInfo | Data tersemat [`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo) |

### Eksepsi

| eksepsi | kondisi |
| --- | --- |
| ArgumentNullException | Ketika parameter embeddedData bernilai null. |

### Catatan

Metode ini mengubah properti objek untuk mencerminkan data baru dan mengatur flag IsObjectLink menjadi false, menunjukkan bahwa objek OLE tersemat.

### Contoh

Contoh berikut menunjukkan cara mengubah data tersemat OLE dan tipenya untuk objek [`IOleObjectFrame`](../../ioleobjectframe) yang sudah ada

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    OleObjectFrame oof = pres.Slides[0].Shapes[0] as OleObjectFrame;
    if (oof != null)
    {
        IOleEmbeddedDataInfo newData = new OleEmbeddedDataInfo(File.ReadAllBytes("Picture.png"), "png");
        oof.SetEmbeddedData(newData);
    }
}
```

## IPresentation.ReplaceRegex metode

Mengganti semua kecocokan dari ekspresi reguler dengan string yang ditentukan.

```csharp
public void ReplaceRegex(Regex regex, string newText, IFindResultCallback callback)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| regex | Regex | Ekspresi reguler Regex untuk mendapatkan string yang akan diganti. |
| newText | String | String yang akan menggantikan semua kemunculan string yang akan diganti. |
| callback | IFindResultCallback | Objek callback untuk menerima hasil pencarian [`IFindResultCallback`](../../ifindresultcallback). |

### Contoh

Contoh kode berikut menunjukkan cara mengganti teks menggunakan ekspresi reguler dengan string yang ditentukan.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// Ganti semua kata dengan 10 atau lebih karakter dengan '***'
	presentation.ReplaceRegex(regex, "***", null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

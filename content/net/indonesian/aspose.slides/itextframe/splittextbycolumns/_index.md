## ITextFrame.SplitTextByColumns metode

Membagi konten teks dari [`ITextFrame`](../../itextframe) menjadi sebuah array string, dimana setiap elemen sesuai dengan sebuah kolom teks terpisah di dalam frame.

```csharp
public string[] SplitTextByColumns()
```

### Nilai Kembalian

Sebuah array string, dimana setiap string mewakili konten teks dari kolom tertentu di dalam [`ITextFrame`](../../itextframe).

### Catatan

Jika frame teks tidak berisi beberapa kolom, array yang dikembalikan akan memiliki satu elemen yang berisi seluruh teks. Kolom kosong akan direpresentasikan sebagai string kosong dalam array.

### Contoh

Contoh berikut menunjukkan cara menggunakan `SplitTextByColumns`:

```csharp
using (Presentation pres = new Presentation("example.pptx"))
{
    // Dapatkan bentuk pertama pada slide dan cast ke ITextFrame
    ITextFrame textFrame = pres.Slides[0].Shapes[0] as ITextFrame;
    // Bagi konten frame teks menjadi kolom
    string[] columnsText = textFrame.SplitTextByColumns();
    // Cetak teks setiap kolom ke konsol
    foreach (string column in columnsText)
        Console.WriteLine(column);
}
```

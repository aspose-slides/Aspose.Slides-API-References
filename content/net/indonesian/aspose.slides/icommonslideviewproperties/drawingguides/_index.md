## ICommonSlideViewProperties.DrawingGuides property

Mengembalikan koleksi panduan gambar. Hanya-baca [`IDrawingGuidesCollection`](../../idrawingguidescollection)

```csharp
public IDrawingGuidesCollection DrawingGuides { get; }
```

### Contoh

Kode contoh berikut menunjukkan cara menambahkan panduan gambar baru dalam presentasi PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    var slideSize = pres.SlideSize.Size;
    
    IDrawingGuidesCollection guides = pres.ViewProperties.SlideViewProperties.DrawingGuides;
    // Menambahkan panduan gambar vertikal baru di sebelah kanan pusat slide
    guides.Add(Orientation.Vertical, slideSize.Width / 2 + 12.5f);
    // Menambahkan panduan gambar horizontal baru di bawah pusat slide
    guides.Add(Orientation.Horizontal, slideSize.Height / 2 + 12.5f);
    
    pres.Save("DrawingGuides_out.pptx", SaveFormat.Pptx);
}
```

## HighlightText(string, Color) {#highlighttext}

Menyorot semua kecocokan teks contoh dengan warna yang ditentukan.

```csharp
public void HighlightText(string text, Color highlightColor)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| text | String | Teks yang akan disorot. |
| highlightColor | Color | Warna untuk menyorot teks. |

### Contoh

Contoh kode berikut menunjukkan cara menyorot teks dalam presentasi PowerPoint.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	// menyorot semua kemunculan 'the' terpisah
	presentation.HighlightText("the", Color.Violet);
	presentation.Save("SomePresentation-out2.pptx", SaveFormat.Pptx);
}
```


| Parameter | Tipe | Deskripsi |
| --- | --- | --- |

| highlightColor | Color | Warna untuk menyorot teks. |
| options | ITextSearchOptions | Opsi pencarian teks [`ITextSearchOptions`](../../itextsearchoptions). |

### Contoh

Contoh kode berikut menunjukkan cara menyorot teks dalam presentasi PowerPoint.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	// menyorot semua kemunculan 'the' terpisah
```

## Presentation.SlideSize properti

Mengembalikan objek ukuran slide. Baca-saja [`ISlideSize`](../../islidesize).

```csharp
public ISlideSize SlideSize { get; }
```

### Contoh

Contoh berikut menunjukkan cara mengubah ukuran slide dalam sebuah Presentasi PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation("pres-4x3-aspect-ratio.pptx"))
{
    pres.SlideSize.SetSize(SlideSizeType.OnScreen16x9, SlideSizeScaleType.DoNotScale);
    pres.Save("pres-4x3-aspect-ratio.pptx", SaveFormat.Pptx);
}
```

Contoh berikut menunjukkan cara mengatur ukuran slide dengan memperhatikan skala konten untuk sebuah Presentasi PowerPoint.

```csharp
[C#]
// Membuat objek Presentation yang mewakili file presentasi
using(Presentation presentation = new Presentation("AccessSlides.pptx")) {
  using(Presentation auxPresentation = new Presentation()) {
    ISlide slide = presentation.Slides[0];
    // Atur ukuran slide presentasi yang dihasilkan ke ukuran sumber
    presentation.SlideSize.SetSize(540, 720, SlideSizeScaleType.EnsureFit); // Metode SetSize digunakan untuk mengatur ukuran slide dengan menskalakan konten agar pas
    presentation.SlideSize.SetSize(SlideSizeType.A4Paper, SlideSizeScaleType.Maximize); // Metode SetSize digunakan untuk mengatur ukuran slide dengan memaksimalkan ukuran konten
    // Simpan Presentasi ke disk
    auxPresentation.Save("Set_Size&Type_out.pptx", SaveFormat.Pptx);
  }
}
```

Contoh berikut menunjukkan cara menentukan ukuran slide khusus dalam sebuah Presentasi PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    pres.SlideSize.SetSize(780, 540, SlideSizeScaleType.DoNotScale); // ukuran kertas A4
    pres.Save("pres-a4-slide-size.pptx", SaveFormat.Pptx);
}
```

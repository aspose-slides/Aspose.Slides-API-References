## IFontsManager.FontFallBackRulesCollection properti

Mewakili koleksi aturan FontFallBack milik pengguna untuk mengelola koleksi font guna substitusi yang tepat melalui fungsi fallback Baca/tulis [`IFontFallBackRulesCollection`](../../ifontfallbackrulescollection).

```csharp
public IFontFallBackRulesCollection FontFallBackRulesCollection { get; set; }
```

### Contoh

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    // Mengambil koleksi aturan yang kosong atau telah diinisialisasi sebelumnya dari FontsManager
    IFontFallBackRulesCollection rulesList = pres.FontsManager.FontFallBackRulesCollection;

    // menambahkan aturan ke koleksi
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));

    // atau 
    // inisialisasi instance baru dari koleksi aturan
    IFontFallBackRulesCollection rulesList = new FontFallBackRulesCollection();

    // menambahkan aturan ke koleksi
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));

    // dan mengganti koleksi yang ada dengan yang baru di FontsManager 
    pres.FontsManager.FontFallBackRulesCollection = rulesList;
}
```

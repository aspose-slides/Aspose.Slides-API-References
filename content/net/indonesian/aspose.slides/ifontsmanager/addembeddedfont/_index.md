## AddEmbeddedFont(IFontData, EmbedFontCharacters) {#addembeddedfont}

Menambahkan font yang tertanam. Perlu diingat saat menyalin font apa pun bahwa kebanyakan font memiliki hak cipta. Pertama, temukan lisensi font tersebut terlebih dahulu dan pastikan font tersebut dapat dipindahkan secara bebas ke mesin lain. ArgumentException dapat dilemparkan jika data font bernilai null atau font ini sudah tertanam.

public void AddEmbeddedFont(IFontData fontData, EmbedFontCharacters embedFontRule)

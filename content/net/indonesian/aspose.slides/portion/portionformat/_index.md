## Portion.PortionFormat properti

Mengembalikan objek format yang berisi properti format yang ditetapkan secara eksplisit pada bagian teks tanpa pewarisan diterapkan. Hanya-baca [`IPortionFormat`](../../iportionformat).

```csharp
public IPortionFormat PortionFormat { get; }
```

### Catatan

Objek format berisi parameter format yang didefinisikan hanya untuk portion saat ini, data yang diwariskan tidak diterapkan.

Untuk mendapatkan nilai efektif termasuk yang diwariskan, gunakan metode [`GetEffective`](../../portionformat/geteffective).

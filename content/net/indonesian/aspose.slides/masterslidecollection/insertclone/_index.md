## MasterSlideCollection.InsertClone metode

Menyisipkan salinan slide master yang ditentukan ke posisi yang ditentukan dalam koleksi. Slide tata letak yang ditautkan juga akan disalin.

```csharp
public IMasterSlide InsertClone(int index, IMasterSlide sourceMaster)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| index | Int32 | Indeks slide baru. |
| sourceMaster | IMasterSlide | Slide yang akan digandakan. |

### Nilai Kembalian

Slide master yang disisipkan.

### Contoh

Contoh berikut menunjukkan cara menggandakan slide master dalam Presentasi PowerPoint lain.

```csharp
[C#]
// Membuat instance kelas Presentation untuk memuat file presentasi sumber
using (Presentation srcPres = new Presentation("CloneToAnotherPresentationWithMaster.pptx"))
{
    // Membuat instance kelas Presentation untuk presentasi tujuan (tempat slide akan digandakan)
    using (Presentation destPres = new Presentation())
    {
        // Membuat instance ISlide dari koleksi slide dalam presentasi sumber bersama
        // Slide master
        ISlide SourceSlide = srcPres.Slides[0];
        IMasterSlide SourceMaster = SourceSlide.LayoutSlide.MasterSlide;
		// Dapatkan Master Slides dari presentasi tujuan
        IMasterSlideCollection masters = destPres.Masters;
        // Gandakan slide master yang diinginkan dari presentasi sumber ke koleksi master di
        // Presentasi tujuan
        IMasterSlide iSlide = masters.AddClone(SourceMaster);
        // Koleksi slide dalam presentasi tujuan
        ISlideCollection slds = destPres.Slides;
		// Gandakan slide sumber ke koleksi slide tujuan.
        slds.AddClone(SourceSlide, iSlide, true);
        // Simpan presentasi tujuan ke disk
        destPres.Save("CloneToAnotherPresentationWithMaster_out.pptx", SaveFormat.Pptx);
    }
}
```

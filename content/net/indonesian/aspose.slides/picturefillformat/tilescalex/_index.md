## PictureFillFormat.TileScaleX properti

Mengembalikan atau mengatur skala horizontal untuk isian tekstur sebagai persentase. Baca/tulis Single.

```csharp
public float TileScaleX { get; set; }
```

### Contoh

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Mendapatkan format pengisian gambar dari shape
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Mengatur mode pengisian gambar ke Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Mengatur skala horizontal untuk tekstur menjadi 120 persen
    pictureFillFormat.TileScaleX = 120;
}
```

## PictureFillFormat.TileOffsetY properti

Mengembalikan atau menetapkan offset vertikal tekstur dari asal shape dalam point. Nilai positif memindahkan tekstur ke bawah, sedangkan nilai negatif memindahkan tekstur ke atas. Baca/tulis Single.

```csharp
public float TileOffsetY { get; set; }
```

### Contoh

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Mendapatkan format pengisian gambar dari shape
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Mengatur mode pengisian gambar menjadi Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Mengatur offset vertikal tekstur menjadi -50 point
    pictureFillFormat.TileOffsetY = -50;
}
```

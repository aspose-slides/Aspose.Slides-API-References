## PictureFillFormat.TileFlip properti

Membalik ubin tekstur di sekitar sumbu horizontal, vertikal, atau keduanya. Baca/tulis [`TileFlip`](../../tileflip).

```csharp
public TileFlip TileFlip { get; set; }
```

### Catatan

Default adalah NoFlip.

### Contoh

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Mendapatkan PictureFillFormat dari shape
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Mengatur PictureFillMode ke Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Membalik ubin tekstur di sekitar sumbu vertikalnya.
    pictureFillFormat.TileFlip = TileFlip.FlipY;
}
```

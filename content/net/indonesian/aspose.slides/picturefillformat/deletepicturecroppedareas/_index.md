## PictureFillFormat.DeletePictureCroppedAreas metode

Hapus area terpangkas dari Picture pengisian.

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### Nilai Kembalian

Gambar terpangkas atau gambar asli jika pemangkasan tidak diperlukan.

### Catatan

Metode ini mengonversi file metafile WMF/EMF menjadi gambar PNG raster saat memotong.

### Contoh

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Mendapatkan PictureFrame
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // Menghapus area terpangkas dari gambar PictureFrame
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

## PictureFillFormat.TileOffsetX properti

Mengembalikan atau mengatur offset horizontal tekstur dari asal bentuk dalam poin. Nilai positif memindahkan tekstur ke kanan, sedangkan nilai negatif memindahkannya ke kiri. Baca/tulis Single.

```csharp
public float TileOffsetX { get; set; }
```

### Contoh

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Mendapatkan format pengisian gambar dari bentuk
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Mengatur mode pengisian gambar menjadi Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Mengatur offset horizontal tekstur menjadi 20 poin
    pictureFillFormat.TileOffsetX = 20f;
}
```

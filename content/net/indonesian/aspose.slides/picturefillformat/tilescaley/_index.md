## PictureFillFormat.TileScaleY properti

Mengembalikan atau mengatur skala vertikal untuk isian tekstur sebagai persentase. Baca/tulis Single.

```csharp
public float TileScaleY { get; set; }
```

### Contoh

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Mendapatkan format pengisian gambar dari shape
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Mengatur mode pengisian gambar menjadi Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Mengatur skala vertikal untuk tekstur menjadi 120 persen
    pictureFillFormat.TileScaleY = 120;
}
```

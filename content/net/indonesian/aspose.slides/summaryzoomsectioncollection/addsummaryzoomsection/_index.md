## SummaryZoomSectionCollection.AddSummaryZoomSection metode

Membuat objek Summary Zoom Section baru dan menambahkannya ke dalam koleksi

```csharp
public ISummaryZoomSection AddSummaryZoomSection(ISection section)
```

| Parameter | Type | Description |
| --- | --- | --- |
| section | ISection | Bagian untuk elemen Summary Zoom Section baru [`ISection`](../../isection) |

### Nilai Kembalian

Elemen [`ISummaryZoomFrame`](../../isummaryzoomframe) yang ditambahkan

### Pengecualian

| Pengecualian | Kondisi |
| --- | --- |
| ArgumentException | Bagian yang direferensikan tidak termasuk dalam presentasi saat ini atau tidak berisi slide apa pun. |

### Catatan

Jika elemen untuk bagian ini sudah ada dalam koleksi, elemen yang ada akan dikembalikan.

### Contoh

Contoh ini menunjukkan cara mendapatkan elemen Summary Zoom Section berdasarkan indeks:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    ISummaryZoomFrame zoomFrame = (ISummaryZoomFrame) pres.Slides[1].Shapes[0];
    ISummaryZoomSectionCollection collection = zoomFrame.SummaryZoomCollection;
    ISummaryZoomSection newZoomSection = collection.AddSummaryZoomSection(pres.Sections[3]);
}
```

## IGeometryShape.SetGeometryPaths metode

Memperbarui geometri bentuk dari array [`IGeometryPath`](../../igeometrypath). Koordinat harus relatif terhadap sudut kiri atas bentuk. Mengubah tipe bentuk ([`ShapeType`](../shapetype)) menjadi Custom.

```csharp
public void SetGeometryPaths(IGeometryPath[] geometryPaths)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| geometryPaths | IGeometryPath[] | Array jalur geometri |

### Pengecualian

| eksepsi | kondisi |
| --- | --- |
| ArgumentException | Tidak ada jalur ditemukan |
| ArgumentException | Jalur kosong |

### Contoh

Contoh:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    GeometryShape shape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 200, 100) as GeometryShape;

    GeometryPath geometryPath = shape.GetGeometryPaths()[0];

    geometryPath.LineTo(100, 50, 1);
    geometryPath.LineTo(100, 50, 4);

    shape.SetGeometryPath(geometryPath);

    pres.Save("output.pptx", SaveFormat.Pptx);
}
```

## InsertAutoShape(int, ShapeType, float, float, float, float, bool) {#insertautoshape_1}

Membuat auto shape baru dan menyisipkannya ke dalam koleksi shape pada indeks yang ditentukan, secara opsional menginisialisasinya dengan gaya templat default.

```csharp
public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| index | Int32 | Indeks berbasis nol tempat menyisipkan auto shape. |
| shapeType | ShapeType | [`ShapeType`](../../shapetype) dari auto shape yang akan disisipkan. |
| x | Single | Koordinat x dari bingkai shape, dalam poin. |
| y | Single | Koordinat y dari bingkai shape, dalam poin. |
| width | Single | Lebar bingkai shape, dalam poin. |
| height | Single | Tinggi bingkai shape, dalam poin. |
| createFromTemplate | Boolean | True untuk menerapkan gaya templat default (termasuk nama yang tidak kosong, gaya sederhana, dan teks terpusat); false untuk membuat shape dengan semua properti diatur ke nilai default mereka. |

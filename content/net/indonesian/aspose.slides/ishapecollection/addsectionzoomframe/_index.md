## AddSectionZoomFrame(float, float, float, float, ISection) {#addsectionzoomframe}

```csharp
    ISection section)
```

| Parameter | Tipe | Deskripsi |

| y | Single | Koordinat y dari bingkai Section Zoom baru, dalam poin. |
| width | Single | Lebar bingkai Section Zoom baru, dalam poin. |
| height | Single | Tinggi bingkai Section Zoom baru, dalam poin. |
| section | ISection | [`ISection`](../../isection) yang dirujuk oleh bingkai Section Zoom; harus menjadi milik presentasi ini dan berisi setidaknya satu slide. |

### Contoh

Contoh ini menunjukkan cara menambahkan objek Section Zoom ke akhir koleksi (asumsikan ada setidaknya dua bagian dalam presentasi "Presentation.pptx"):

```csharp

{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1]);
```

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section, IPPImage image)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| x | Single | Koordinat x dari bingkai Section Zoom baru, dalam poin. |
| y | Single | Koordinat y dari bingkai Section Zoom baru, dalam poin. |
| width | Single | Lebar bingkai Section Zoom baru, dalam poin. |
| height | Single | Tinggi bingkai Section Zoom baru, dalam poin. |
| section | ISection | [`ISection`](../../isection) yang dirujuk oleh bingkai Section Zoom; harus menjadi milik presentasi ini dan berisi setidaknya satu slide. |
| image | IPPImage | [`IPPImage`](../../ippimage) untuk ditampilkan dalam bingkai Section Zoom. |

### Nilai Kembalian

[`ISectionZoomFrame`](../../isectionzoomframe) yang baru dibuat.

### Pengecualian

| eksepsi | kondisi |
| --- | --- |
| ArgumentException | Dilempar jika bagian yang dirujuk tidak termasuk dalam presentasi saat ini atau tidak berisi slide. |

### Contoh

Contoh ini menunjukkan cara menambahkan objek Section Zoom ke akhir koleksi (asumsikan ada setidaknya dua bagian dalam presentasi "Presentation.pptx"):

```csharp

}
```

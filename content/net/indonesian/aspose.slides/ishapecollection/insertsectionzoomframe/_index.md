## InsertSectionZoomFrame(int, float, float, float, float, ISection) {#insertsectionzoomframe}

Membuat bingkai Section Zoom baru dan menyisipkannya ke dalam koleksi shape pada indeks yang ditentukan.

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| index | Int32 | Indeks berbasis nol di mana bingkai Section Zoom akan disisipkan. |
| x | Single | Koordinat x dari bingkai Section Zoom baru, dalam titik. |
| y | Single | Koordinat y dari bingkai Section Zoom baru, dalam titik. |
| width | Single | Lebar bingkai Section Zoom baru, dalam titik. |
| height | Single | Tinggi bingkai Section Zoom baru, dalam titik. |
| section | ISection | [`ISection`](../../isection) yang direferensikan oleh bingkai Section Zoom; harus termasuk dalam presentasi ini dan berisi setidaknya satu slide. |

### Nilai Kembali

[`ISectionZoomFrame`](../../isectionzoomframe) yang baru dibuat.

### Pengecualian

| Pengecualian | Kondisi |
| --- | --- |
| ArgumentException | Dilempar jika section yang direferensikan tidak termasuk dalam presentasi saat ini atau tidak berisi slide. |

### Contoh

Contoh ini menunjukkan pembuatan dan penyisipan objek Section Zoom pada indeks yang ditentukan dalam sebuah koleksi (asumsikan terdapat setidaknya dua section dalam presentasi "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSectionZoomFrame(2, 150, 20, 50, 50, pres.Sections[1]);
}
```

## InsertSectionZoomFrame(int, float, float, float, float, ISection, IPPImage) {#insertsectionzoomframe_1}

Membuat bingkai Section Zoom baru dengan gambar yang telah ditentukan dan menyisipkannya ke dalam koleksi shape pada indeks yang ditentukan.

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section, IPPImage image)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| index | Int32 | Indeks berbasis nol di mana bingkai Section Zoom akan disisipkan. |
| x | Single | Koordinat x dari bingkai Section Zoom baru, dalam titik. |
| y | Single | Koordinat y dari bingkai Section Zoom baru, dalam titik. |
| width | Single | Lebar bingkai Section Zoom baru, dalam titik. |
| height | Single | Tinggi bingkai Section Zoom baru, dalam titik. |
| section | ISection | [`ISection`](../../isection) yang direferensikan oleh bingkai Section Zoom; harus termasuk dalam presentasi ini dan berisi setidaknya satu slide. |
| image | IPPImage | Gambar yang akan ditampilkan dalam bingkai Section Zoom. |

### Nilai Kembali

[`ISectionZoomFrame`](../../isectionzoomframe) yang baru dibuat.

### Pengecualian

| Pengecualian | Kondisi |
| --- | --- |
| ArgumentException | Dilempar jika section yang direferensikan tidak termasuk dalam presentasi saat ini atau tidak berisi slide. |

### Contoh

Contoh ini menunjukkan pembuatan dan penyisipan objek Section Zoom pada indeks yang ditentukan dalam sebuah koleksi (asumsikan terdapat setidaknya dua section dalam presentasi "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IPPImage image = pres.Images.AddImage(Image.FromFile("image.png"));

}
```

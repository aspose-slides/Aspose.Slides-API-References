## InsertZoomFrame(int, float, float, float, float, ISlide) {#insertzoomframe}

Membuat frame Zoom baru dan menyisipkannya ke dalam koleksi shape pada indeks yang ditentukan.

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| index | Int32 | Indeks berbasis nol tempat menyisipkan frame Zoom. |
| x | Single | Koordinat x dari frame Zoom baru, dalam poin. |
| y | Single | Koordinat y dari frame Zoom baru, dalam poin. |
| width | Single | Lebar frame Zoom baru, dalam poin. |
| height | Single | Tinggi frame Zoom baru, dalam poin. |
| slide | ISlide | [`ISlide`](../../islide) yang direferensikan oleh frame Zoom. |

### Nilai Kembalian

[`IZoomFrame`](../../izoomframe) yang baru dibuat.

### Pengecualian

| Pengecualian | Kondisi |
| --- | --- |
| ArgumentException | Dibuang jika slide yang direferensikan tidak termasuk dalam presentasi saat ini. |

### Contoh

Contoh ini menunjukkan cara membuat dan menyisipkan objek Zoom pada indeks yang ditentukan dalam sebuah koleksi (asumsikan ada setidaknya dua slide dalam presentasi "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertZoomFrame(2, 150, 20, 50, 50, pres.Slides[1]);
}
```

## InsertZoomFrame(int, float, float, float, float, ISlide, IPPImage) {#insertzoomframe_1}

Membuat frame Zoom baru dengan gambar pra-definisi dan menyisipkannya ke dalam koleksi shape pada indeks yang ditentukan.

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide, IPPImage image)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| index | Int32 | Indeks berbasis nol tempat menyisipkan frame Zoom. |
| x | Single | Koordinat x dari frame Zoom baru, dalam poin. |
| y | Single | Koordinat y dari frame Zoom baru, dalam poin. |
| width | Single | Lebar frame Zoom baru, dalam poin. |
| height | Single | Tinggi frame Zoom baru, dalam poin. |
| slide | ISlide | [`ISlide`](../../islide) yang direferensikan oleh frame Zoom. |
| image | IPPImage | Gambar untuk slide yang direferensikan [`IPPImage`](../../ippimage). |

### Nilai Kembalian

[`IZoomFrame`](../../izoomframe) yang baru dibuat.

### Pengecualian

| Pengecualian | Kondisi |
| --- | --- |
| ArgumentException | Dibuang jika slide yang direferensikan tidak termasuk dalam presentasi saat ini. |

### Contoh

Contoh ini menunjukkan cara membuat dan menyisipkan objek Zoom pada indeks yang ditentukan dalam sebuah koleksi (asumsikan ada setidaknya dua slide dalam presentasi "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IPPImage image = pres.Images.AddImage(Image.FromFile("image.png"));
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertZoomFrame(2, 150, 20, 50, 50, pres.Slides[1], image);
}
```

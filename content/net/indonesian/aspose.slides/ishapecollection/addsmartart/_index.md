## IShapeCollection.AddSmartArt method

Membuat diagram SmartArt dan menambahkannya ke akhir koleksi shape.

```csharp
    SmartArtLayoutType layoutType)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| x | Single | Koordinat x dari frame diagram, dalam poin. |
| y | Single | Koordinat y dari frame diagram, dalam poin. |
| width | Single | Lebar frame diagram, dalam poin. |
| height | Single | Tinggi frame diagram, dalam poin. |

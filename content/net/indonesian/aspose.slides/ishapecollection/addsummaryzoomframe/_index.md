## IShapeCollection.AddSummaryZoomFrame metode

Membuat frame Summary Zoom baru dan menambahkannya ke akhir koleksi shape.

```csharp
public ISummaryZoomFrame AddSummaryZoomFrame(float x, float y, float width, float height)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| x | Single | Koordinat x dari frame Summary Zoom baru, dalam poin. |
| y | Single | Koordinat y dari frame Summary Zoom baru, dalam poin. |
| width | Single | Lebar frame Summary Zoom baru, dalam poin. |
| height | Single | Tinggi frame Summary Zoom baru, dalam poin. |

### Nilai Kembalian

[`ISummaryZoomFrame`](../../isummaryzoomframe) yang baru dibuat.

### Pengecualian

| pengecualian | kondisi |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | Dilemparkan jika tidak ada bagian dalam presentasi, atau jika slide target tidak termasuk dalam bagian manapun. |

### Catatan

Metode ini membuat frame Summary Zoom yang menggabungkan tautan ringkasan untuk semua bagian dalam presentasi.

### Contoh

Contoh ini menunjukkan penambahan objek Summary Zoom ke akhir sebuah koleksi (asumsikan ada setidaknya dua bagian dalam presentasi "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSummaryZoomFrame(150, 20, 500, 250);
}
```

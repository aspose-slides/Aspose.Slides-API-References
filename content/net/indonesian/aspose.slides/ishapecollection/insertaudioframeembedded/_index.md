## InsertAudioFrameEmbedded(int, float, float, float, float, Stream) {#insertaudioframeembedded_1}

Membuat sebuah frame audio baru dengan file WAV tersemat dan menyisipkannya ke dalam koleksi shape pada indeks yang ditentukan. Audio yang tersemat ditambahkan ke koleksi Presentation.Audios.

```csharp
public IAudioFrame InsertAudioFrameEmbedded(int index, float x, float y, float width, float height, 
    Stream audio_stream)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| index | Int32 | Indeks berbasis nol tempat frame audio akan disisipkan. |
| x | Single | Koordinat x dari frame audio baru, dalam poin. |
| y | Single | Koordinat y dari frame audio baru, dalam poin. |
| width | Single | Lebar dari frame audio baru, dalam poin. |
| height | Single | Tinggi dari frame audio baru, dalam poin. |
| audio_stream | Stream | Aliran input yang berisi data audio WAV untuk disematkan. |

### Nilai Kembali

The newly created [`IAudioFrame`](../../iaudioframe).

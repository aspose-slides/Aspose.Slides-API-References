## CommentCollection.AddModernComment metode

Menambahkan komentar modern baru di akhir koleksi.

```csharp
public IModernComment AddModernComment(string text, ISlide slide, IShape shape, PointF position, 
    DateTime creationTime)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| text | String | Teks biasa dari komentar modern baru. |
| slide | ISlide | Slide dalam presentasi tempat menambahkan komentar modern baru. |
| shape | IShape | Bentuk pada slide yang dikaitkan dengan komentar modern baru. |
| position | PointF | Posisi pada slide tempat menambahkan komentar modern baru. |
| creationTime | DateTime | Waktu pembuatan komentar modern. |

### Nilai Kembali

Komentar modern yang ditambahkan.

### Contoh

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ICommentAuthor newAuthor = pres.CommentAuthors.AddAuthor("Some Author", "SA");
    newAuthor.Comments.AddModernComment("This is modern comment", pres.Slides[0], null, new PointF(100, 100), DateTime.Now);

    pres.Save(outPptxFileName, SaveFormat.Pptx);
}
```

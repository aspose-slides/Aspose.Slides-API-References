## PresentationLockingBehavior enumerasi

Mewakili perilaku terkait perlakuan sumber [`IPresentation`](../ipresentation) (file atau Stream) saat memuat dan bekerja dengan contoh [`IPresentation`](../ipresentation).

```csharp
public enum PresentationLockingBehavior
```

### Nilai

| Nama | Nilai | Deskripsi |
| --- | --- | --- |
| LoadAndRelease | `0` | Sumber akan terkunci hanya selama eksekusi konstruktor [`IPresentation`](../ipresentation). Jika [`IsTemporaryFilesAllowed`](../iblobmanagementoptions/istemporaryfilesallowed) diatur ke false, semua BLOB akan dimuat ke memori. Jika tidak, cara lain seperti file sementara mungkin akan digunakan. Perilaku ini lebih lambat daripada KeepLocked, dan jika memungkinkan untuk memindahkan kepemilikan sumber ke [`IPresentation`](../ipresentation), disarankan menggunakan KeepLocked. |
| KeepLocked | `1` | Sumber akan terkunci selama seluruh masa hidup instance [`IPresentation`](../ipresentation), sampai objek tersebut dibuang. [`IsTemporaryFilesAllowed`](../iblobmanagementoptions/istemporaryfilesallowed) harus diatur ke true untuk menggunakan perilaku ini, jika tidak akan terjadi pengecualian. Perilaku ini disarankan, karena lebih cepat dan menggunakan memori lebih sedikit daripada LoadAndRelease. |

### Catatan

Sumber adalah parameter yang diteruskan ke konstruktor [`IPresentation`](../ipresentation). Pada contoh di bawah, sumber adalah file "pres.pptx":

```csharp
LoadOptions loadOptions = new LoadOptions { 
  BlobManagementOptions = { PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked } };
using (IPresentation pres = new Presentation("pres.pptx", loadOptions)) { }
```

Untuk contoh ini, sumber (file "pres.pptx") akan terkunci selama masa hidup instance [`IPresentation`](../ipresentation), yaitu tidak dapat diubah atau dihapus oleh proses lain.

## PresentationLockingBehavior enumerasi

Mewakili perilaku dalam memperlakukan sumber [`IPresentation`](../ipresentation) (file atau Stream) saat memuat dan bekerja dengan instance [`IPresentation`](../ipresentation).

```csharp
public enum PresentationLockingBehavior
```

### Nilai

| Nama | Nilai | Deskripsi |
| --- | --- | --- |
| LoadAndRelease | `0` | Sumber akan dikunci hanya selama eksekusi konstruktor [`IPresentation`](../ipresentation). Jika [`IsTemporaryFilesAllowed`](../iblobmanagementoptions/istemporaryfilesallowed) diatur ke false, semua BLOB akan dimuat ke memori. Jika tidak, cara lain seperti file sementara mungkin digunakan. Perilaku ini lebih lambat daripada KeepLocked, dan jika memungkinkan untuk memindahkan kepemilikan sumber ke [`IPresentation`](../ipresentation), disarankan untuk menggunakan KeepLocked. |
| KeepLocked | `1` | Sumber akan dikunci selama seluruh masa hidup instance [`IPresentation`](../ipresentation), hingga objek tersebut dibuang. [`IsTemporaryFilesAllowed`](../iblobmanagementoptions/istemporaryfilesallowed) harus diatur ke true untuk menggunakan perilaku ini, jika tidak, pengecualian akan dilempar. Perilaku ini disarankan, lebih cepat dan menggunakan lebih sedikit memori dibandingkan LoadAndRelease. |

### Catatan

Sumber adalah parameter yang diberikan ke konstruktor [`IPresentation`](../ipresentation). Pada contoh di bawah, sumber adalah file "pres.pptx":

```csharp
LoadOptions loadOptions = new LoadOptions { 
  BlobManagementOptions = { PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked } };
using (IPresentation pres = new Presentation("pres.pptx", loadOptions)) { }
```

Untuk contoh ini, sumber (file "pres.pptx") akan dikunci selama masa hidup instance [`IPresentation`](../ipresentation), yaitu tidak dapat diubah atau dihapus oleh proses lain.

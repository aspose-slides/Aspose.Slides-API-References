## PresentationInfo.CheckWriteProtection metode

Memeriksa apakah kata sandi untuk mengubah benar untuk presentasi yang dilindungi penulisan.

```csharp
public bool CheckWriteProtection(string password)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| password | String | Kata sandi yang akan diperiksa. |

### Nilai Kembalian

True jika presentasi dilindungi penulisan dan kata sandi benar. False jika tidak.

### Pengecualian

| exception | condition |
| --- | --- |
| InvalidOperationException |  |

### Catatan

1. Anda harus memeriksa properti [`IsWriteProtected`](../iswriteprotected) sebelum memanggil metode ini. 2. Ketika password null atau kosong, metode ini mengembalikan false.

### Contoh

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
if (info.IsWriteProtected == NullableBool.True)
{
    bool isWriteProtectedByPassword = info.CheckWriteProtection("my_password");
}
```

## PresentationInfo.CheckPassword method

Memeriksa apakah kata sandi benar untuk presentasi yang dilindungi dengan kata sandi terbuka.

```csharp
public bool CheckPassword(string password)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| password | String | Kata sandi yang akan diperiksa. |

### Nilai Kembalian

True jika presentasi dilindungi dengan kata sandi terbuka dan kata sandi tersebut benar, serta false jika tidak.

| Pengecualian | Kondisi |

### Catatan

Ketika kata sandi bernilai null atau kosong, metode ini mengembalikan false.

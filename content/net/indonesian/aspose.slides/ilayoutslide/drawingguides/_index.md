## ILayoutSlide.DrawingGuides properti

Mengembalikan koleksi panduan gambar untuk slide tata letak. Hanya baca [`IDrawingGuidesCollection`](../../idrawingguidescollection)

```csharp
public IDrawingGuidesCollection DrawingGuides { get; }
```

### Contoh

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    var slideSize = pres.SlideSize.Size;
    
    IDrawingGuidesCollection guides = pres.LayoutSlides[0].DrawingGuides;
    // Menambahkan panduan gambar vertikal baru ke kiri pusat slide
    guides.Add(Orientation.Vertical, slideSize.Width / 2 - 20f);
    
    pres.Save("LayoutDrawingGuides_out.pptx", SaveFormat.Pptx);
}
```

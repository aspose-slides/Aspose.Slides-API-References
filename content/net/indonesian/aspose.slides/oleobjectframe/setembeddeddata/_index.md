## OleObjectFrame.SetEmbeddedData method

Mengatur informasi tentang data ter-embed OLE. Metode ini mengubah properti objek untuk mencerminkan data baru dan mengatur flag IsObjectLink menjadi false, menunjukkan bahwa objek OLE ter-embed.

```csharp
public void SetEmbeddedData(IOleEmbeddedDataInfo embeddedData)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| embeddedData | IOleEmbeddedDataInfo | Data ter-embed [`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo) |

### Pengecualian

| Exception | Kondisi |
| --- | --- |
| ArgumentNullException | Ketika parameter embeddedData bernilai null. |

### Contoh

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    OleObjectFrame oof = pres.Slides[0].Shapes[0] as OleObjectFrame;
    if (oof != null)
    {
        IOleEmbeddedDataInfo newData = new OleEmbeddedDataInfo(File.ReadAllBytes("Picture.png"), "png");
        oof.SetEmbeddedData(newData);
    }
}
```

## ISvgImage.WriteAsEmf metode

Menyimpan gambar SVG sebagai file EMF.

```csharp
public void WriteAsEmf(Stream stream)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| stream | Stream | Stream target |

### Pengecualian

| Pengecualian | kondisi |
| --- | --- |

### Contoh

Contoh berikut menunjukkan cara menyimpan gambar SVG ke dalam metafile.

```csharp
[C#]
// Membuat gambar SVG baru
ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));

// Menyimpan gambar SVG sebagai metafile
using (var fileStream = System.IO.File.OpenWrite("SvgAsEmf.emf"))
    svgImage.WriteAsEmf(fileStream);
```

Contoh ini menunjukkan cara menambahkan gambar SVG sebagai metafile ke koleksi gambar presentasi.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // Membuat gambar SVG baru
    ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));
    using (var memStream = new MemoryStream())
    {
        // Menyimpan gambar SVG sebagai metafile
        svgImage.WriteAsEmf(memStream);
        // Menambahkan metafile ke koleksi gambar
        pres.Images.AddImage(memStream.ToArray());
    }
}
```

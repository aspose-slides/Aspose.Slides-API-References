## Metode ProtectionManager.CheckWriteProtection

Menentukan apakah presentasi dilindungi sandi untuk dimodifikasi.

```csharp
public bool CheckWriteProtection(string password)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| password | String | Sandi yang akan diperiksa. |

### Nilai Kembalian

True jika sandi valid; otherwise, false.

### Catatan

1. Anda harus memeriksa properti [`IsWriteProtected`](../iswriteprotected) sebelum memanggil metode ini. 2. Ketika sandi bernilai null atau kosong, metode ini mengembalikan false.

### Contoh

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

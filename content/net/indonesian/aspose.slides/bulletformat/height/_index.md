## BulletFormat.Height properti

Mengembalikan atau mengatur tinggi bullet sebuah paragraf tanpa pewarisan. Nilai float.NaN menentukan bahwa bullet mewarisi tinggi dari bagian pertama dalam paragraf. Baca/tulis Single.

```csharp
public float Height { get; set; }
```

### Keterangan

Nilai tinggi negatif berarti tinggi diberikan dalam poin dan nilai positif berarti tinggi merupakan persentase dari teks di sekitarnya.

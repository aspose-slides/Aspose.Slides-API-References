## FontsManager.GetFontEmbeddingLevel metode

Menentukan tingkat penyematan font dari array byte yang diberikan dan nama font.

```csharp
public EmbeddingLevel GetFontEmbeddingLevel(byte[] fontBytes, string fontName)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| fontBytes | Byte[] | Array byte yang berisi data font. |
| fontName | String | Nama font. |

### Nilai Kembalian

Tingkat penyematan font yang ditentukan.

### Pengecualian

| pengecualian | kondisi |
| --- | --- |
| ArgumentNullException | Dilemparkan ketika *fontBytes* bernilai null. |

### Contoh

```csharp
[C#]
using (Presentation pres = new Presentation(pptxFileName))
{
    // Mengambil semua font yang digunakan dalam presentasi
    IFontData[] fontDatas = pres.FontsManager.GetFonts();

    // Mendapatkan array byte yang mewakili gaya reguler dari font pertama dalam presentasi

    // Menentukan tingkat penyematan font
    EmbeddingLevel embeddingLevel = pres.FontsManager.GetFontEmbeddingLevel(bytes, fontDatas[0].FontName);
}
```

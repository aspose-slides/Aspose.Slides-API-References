## AddEmbeddedFont(IFontData, EmbedFontCharacters) {#addembeddedfont}

Menambahkan font yang disematkan. Perlu diingat saat menyalin font apa pun bahwa sebagian besar font memiliki hak cipta. Pertama, temukan lisensi font terlebih dahulu dan verifikasi bahwa font tersebut dapat dipindahkan secara bebas ke mesin lain. ArgumentException dapat dilempar jika data font null atau font ini sudah disematkan.

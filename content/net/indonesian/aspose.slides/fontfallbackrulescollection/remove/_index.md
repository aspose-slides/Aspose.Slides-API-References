## FontFallBackRulesCollection.Remove metode

Menghapus kemunculan pertama dari aturan FallBack tertentu dari koleksi.

```csharp
public void Remove(IFontFallBackRule targetRule)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| targetRule | IFontFallBackRule | Aturan yang akan dihapus dari koleksi. |

### Contoh

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    //Mendapatkan koleksi aturan yang kosong atau telah diprainisialisasi dari FontsManager
    IFontFallBackRulesCollection rulesList = pres.FontsManager.FontFallBackRulesCollection;

    //Menambahkan beberapa aturan ke koleksi
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));
    rulesList.Add(new FontFallBackRule(0x3040, 0x309F, "MS Mincho"));

    //Mengambil objek aturan pertama dalam koleksi
    IFontFallBackRule firstRule = rulesList[0];
    //Menghapus 
    rulesList.Remove (firstRule);
}
```

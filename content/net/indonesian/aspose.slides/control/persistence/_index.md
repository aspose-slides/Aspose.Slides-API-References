## Control.Persistence properti

Mendapatkan metode yang digunakan untuk menyimpan properti kontrol ActiveX. Hanya baca [`PersistenceType`](../../persistencetype).

```csharp
public PersistenceType Persistence { get; }
```

### Contoh

Contoh berikut menunjukkan penggunaan properti Persistence untuk memeriksa apakah properti objek ActiveX dapat diubah sebagai properti ActiveX berbasis XML:

```csharp
[C#]
if (control.Persistence == PersistenceType.PersistPropertyBag)
{
    control.Properties["Value"] = value;
}
else
{
    YourMethodHere(control.ActiveXControlBinary); //Gunakan metode Anda sendiri untuk mengelola properti ActiveX yang disimpan dalam file biner-nya
}
```

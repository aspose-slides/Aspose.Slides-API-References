## FontsLoader.LoadExternalFonts metode

Menambahkan folder tambahan untuk mencari font.

```csharp
public static void LoadExternalFonts(string[] directories)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| directories | String[] | Direktori untuk membaca font tambahan. |

### Contoh

Contoh berikut menunjukkan cara memuat font khusus dari .TTF

```csharp
[C#]
// Jalur ke direktori dokumen.
string dataDir = "C:\\";
// folder untuk mencari font
String[] folders = new String[] { dataDir };
// Muat font direktori font khusus
FontsLoader.LoadExternalFonts(folders);
// Lakukan beberapa pekerjaan dan render presentasi/slide
using (Presentation presentation = new Presentation(dataDir + "DefaultFonts.pptx"))
{
    presentation.Save(dataDir + "NewFonts_out.pptx", SaveFormat.Pptx);
	// Bersihkan Cache Font
	FontsLoader.ClearCache();
}
```

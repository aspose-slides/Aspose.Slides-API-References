## ParagraphFormat.Alignment properti

Mengembalikan atau mengatur perataan teks dalam paragraf tanpa pewarisan. Baca/tulis [`TextAlignment`](../../textalignment).

```csharp
public TextAlignment Alignment { get; set; }
```

### Contoh

Kode contoh berikut menunjukkan cara Menyelaraskan Paragraf Teks dalam Presentasi PowerPoint.

```csharp
[C#]
// Membuat objek Presentation yang mewakili file PPTX
using (Presentation pres = new Presentation("ParagraphsAlignment.pptx"))
{
    // Mengakses slide pertama
    ISlide slide = pres.Slides[0];
    // Mengakses placeholder pertama dan kedua di slide dan melakukan typecasting menjadi AutoShape
    ITextFrame tf1 = ((IAutoShape)slide.Shapes[0]).TextFrame;
    ITextFrame tf2 = ((IAutoShape)slide.Shapes[1]).TextFrame;
    // Mengubah teks di kedua placeholder
    tf1.Text = "Center Align by Aspose";
    tf2.Text = "Center Align by Aspose";
    // Mendapatkan paragraf pertama dari placeholder
    IParagraph para1 = tf1.Paragraphs[0];
    IParagraph para2 = tf2.Paragraphs[0];
    // Menyelaraskan paragraf teks ke tengah
    para1.ParagraphFormat.Alignment = TextAlignment.Center;
    para2.ParagraphFormat.Alignment = TextAlignment.Center;
    //Menulis presentasi sebagai file PPTX
    pres.Save("Centeralign_out.pptx", SaveFormat.Pptx);
}
```

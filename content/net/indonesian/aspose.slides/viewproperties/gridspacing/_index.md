## ViewProperties.GridSpacing properti

Mengembalikan atau mengatur jarak kisi yang harus digunakan untuk kisi yang mendasari dokumen presentasi, dalam poin. Read/write Single.

```csharp
public float GridSpacing { get; set; }
```

### Catatan

Nilai jarak kisi harus berupa angka positif. Rentang nilai tipikal adalah dari 1 mm (2.8349607 poin) hingga 2 inci (144 poin).

### Contoh

Kode contoh berikut menunjukkan cara mengubah jarak kisi dalam presentasi PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

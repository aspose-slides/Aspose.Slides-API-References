## IDocumentProperties.GetSensitivityLabels metode

Mendapatkan sebuah array label sensitif dari properti dokumen khusus (Microsoft Information Protection SDK Metadata).

```csharp
public ISensitivityLabel[] GetSensitivityLabels()
```

### Contoh

Kode berikut menunjukkan cara memindahkan informasi label sensitif dari properti dokumen khusus ke koleksi SensitivityLabels modern:

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    // Dapatkan label sensitif dari properti dokumen khusus
    ISensitivityLabel[] mipSensitivityLabels = pres.DocumentProperties.GetSensitivityLabels();

    ISensitivityLabelCollection sensitivityLabels = pres.SensitivityLabels;
    foreach (var sensitivityLabel in mipSensitivityLabels)
    {
        // Tambahkan label ke koleksi
        // Di sini Anda dapat menambahkan pemeriksaan validitas informasi label (label tersedia, dll)
        sensitivityLabels.Add(sensitivityLabel);
    }

    pres.Save("SensitivityLabel.pptx", SaveFormat.Pptx);
}
```

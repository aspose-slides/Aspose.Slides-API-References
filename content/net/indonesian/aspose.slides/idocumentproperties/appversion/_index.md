## IDocumentProperties.AppVersion properti

Mengembalikan versi aplikasi. Hanya baca String.

```csharp
public string AppVersion { get; }
```

### Catatan

Konten elemen ini harus dalam bentuk XX.YYYY, di mana X dan Y mewakili nilai numerik; jika tidak, dokumen akan dianggap tidak sesuai. Aspose.Slides merepresentasikan versinya dalam format XX.YYZZ, di mana: XX - versi utama YY - versi minor ZZ - versi patch. Sebagai contoh, nilai 23.0105 berarti versi Aspose.Slides 23.1.5.

## PortionCollection.CopyTo method

Menyalin elemen ICollection ke Array, dimulai pada indeks Array tertentu.


| Parameter | Type | Description |
| --- | --- | --- |
| array | IPortion[] | Array satu dimensi yang menjadi tujuan elemen yang disalin dari ICollection. Array harus menggunakan indeks berbasis nol. |

## InsertOleObjectFrame(int, float, float, float, float, IOleEmbeddedDataInfo) {#insertoleobjectframe}

Membuat frame objek OLE baru dan menyisipkannya ke dalam koleksi shape pada indeks yang ditentukan.

```csharp
public IOleObjectFrame InsertOleObjectFrame(int index, float x, float y, float width, float height, 
    IOleEmbeddedDataInfo dataInfo)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | Int32 | Indeks berbasis nol tempat menyisipkan frame objek OLE. |
| x | Single | Koordinat x dari frame OLE baru, dalam poin. |
| y | Single | Koordinat y dari frame OLE baru, dalam poin. |
| width | Single | Lebar frame OLE baru, dalam poin. |
| height | Single | Tinggi frame OLE baru, dalam poin. |
| dataInfo | IOleEmbeddedDataInfo | Informasi data OLE yang disematkan ([`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo)). |

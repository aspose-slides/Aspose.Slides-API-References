## ShapeCollection.AddSummaryZoomFrame metode

Membuat bingkai Summary Zoom baru dan menambahkannya ke akhir koleksi shape.

```csharp
public ISummaryZoomFrame AddSummaryZoomFrame(float x, float y, float width, float height)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| x | Single | Koordinat x dari bingkai Summary Zoom baru, dalam poin. |
| y | Single | Koordinat y dari bingkai Summary Zoom baru, dalam poin. |
| width | Single | Lebar bingkai Summary Zoom baru, dalam poin. |
| height | Single | Tinggi bingkai Summary Zoom baru, dalam poin. |

### Nilai Kembali

[`ISummaryZoomFrame`](../../isummaryzoomframe) yang baru dibuat.

### Pengecualian

| eksepsi | kondisi |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | Dilemparkan jika tidak ada bagian dalam presentasi, atau jika slide target tidak termasuk dalam bagian mana pun. |

### Catatan

Metode ini membuat Summary Zoom baru dan menempatkan sekumpulan objek ke dalamnya untuk semua bagian dalam presentasi ini.

### Contoh

Contoh ini menunjukkan cara menambahkan objek Summary Zoom ke akhir sebuah koleksi (asumsikan terdapat setidaknya dua bagian dalam presentasi "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSummaryZoomFrame(150, 20, 500, 250);
}
```

## AddAudioFrameEmbedded(float, float, float, float, Stream) {#addaudioframeembedded_1}

Membuat frame audio baru dengan file WAV tersemat dan menambahkannya ke akhir koleksi shape. Audio yang tersemat ditambahkan ke koleksi Presentation.Audios.

```csharp
public IAudioFrame AddAudioFrameEmbedded(float x, float y, float width, float height, 
    Stream audio_stream)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| x | Single | Koordinat x dari frame audio baru, dalam point. |
| y | Single | Koordinat y dari frame audio baru, dalam point. |
| width | Single | Lebar frame audio baru, dalam point. |
| height | Single | Tinggi frame audio baru, dalam point. |
| audio_stream | Stream | Aliran masuk yang berisi data audio WAV untuk disematkan. |

### Nilai Kembali

Objek [`IAudioFrame`](../../iaudioframe) yang baru dibuat.

### Contoh

Contoh berikut menunjukkan cara membuat frame audio.

```csharp
[C#]
// Membuat instance kelas presentasi yang mewakili file presentasi
using (Presentation pres = new Presentation())
{
    // Mendapatkan slide pertama
    ISlide sld = pres.Slides[0];
    // Muat file suara wav ke dalam stream
    FileStream fstr = new FileStream("sampleaudio.wav", FileMode.Open, FileAccess.Read);
    // Menambahkan Audio Frame
    IAudioFrame audioFrame = sld.Shapes.AddAudioFrameEmbedded(50, 150, 100, 100, fstr);
    // Mengatur Mode Putar dan Volume Audio
    audioFrame.PlayMode = AudioPlayModePreset.Auto;
    audioFrame.Volume = AudioVolumeMode.Loud;
    // Menulis file PowerPoint ke disk
    pres.Save("AudioFrameEmbed_out.pptx", SaveFormat.Pptx);
}
```

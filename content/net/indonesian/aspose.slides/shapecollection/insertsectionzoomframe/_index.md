## InsertSectionZoomFrame(int, float, float, float, float, ISection) {#insertsectionzoomframe}

Membuat frame Section Zoom baru dan menyisipkannya ke dalam koleksi shape pada indeks yang ditentukan.

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| index | Int32 | Indeks berbasis nol tempat menyisipkan frame Section Zoom. |
| x | Single | Koordinat x dari frame Section Zoom baru, dalam poin. |
| y | Single | Koordinat y dari frame Section Zoom baru, dalam poin. |
| width | Single | Lebar frame Section Zoom baru, dalam poin. |
| height | Single | Tinggi frame Section Zoom baru, dalam poin. |
| section | ISection | [`ISection`](../../isection) yang direferensikan oleh frame Section Zoom; harus menjadi bagian dari presentasi ini dan berisi setidaknya satu slide. |

### Nilai Kembalian

[`ISectionZoomFrame`](../../isectionzoomframe) yang baru dibuat.

### Pengecualian

| pengecualian | kondisi |
| --- | --- |
| ArgumentException | Dilemparkan jika section yang direferensikan tidak termasuk dalam presentasi saat ini atau tidak berisi slide. |

### Contoh

Contoh ini menunjukkan pembuatan dan penyisipan objek Section Zoom pada indeks yang ditentukan dalam sebuah koleksi (asumsikan terdapat setidaknya dua section dalam presentasi "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSectionZoomFrame(2, 150, 20, 50, 50, pres.Sections[1]);
}
```

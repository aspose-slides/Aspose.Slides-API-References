## InsertAudioFrameEmbedded(int, float, float, float, float, Stream) {#insertaudioframeembedded_1}

Membuat bingkai audio baru dengan file WAV yang disematkan dan menyisipkannya ke dalam koleksi shape pada indeks yang ditentukan. Audio yang disematkan ditambahkan ke koleksi Presentation.Audios.

```csharp
public IAudioFrame InsertAudioFrameEmbedded(int index, float x, float y, float width, float height, 
    Stream audio_stream)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| index | Int32 | Indeks berbasis nol tempat menyisipkan bingkai audio. |
| x | Single | Koordinat x dari bingkai audio baru, dalam poin. |
| y | Single | Koordinat y dari bingkai audio baru, dalam poin. |
| width | Single | Lebar bingkai audio baru, dalam poin. |
| height | Single | Tinggi bingkai audio baru, dalam poin. |
| audio_stream | Stream | Aliran masukan yang berisi data audio WAV untuk disematkan. |

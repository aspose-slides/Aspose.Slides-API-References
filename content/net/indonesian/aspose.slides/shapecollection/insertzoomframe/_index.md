## InsertZoomFrame(int, float, float, float, float, ISlide) {#insertzoomframe}

Membuat bingkai Zoom baru dan menyisipkannya ke dalam koleksi shape pada indeks yang ditentukan.

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | Int32 | Indeks berbasis nol tempat menyisipkan Zoom frame. |
| x | Single | Koordinat x dari Zoom frame baru, dalam poin. |
| y | Single | Koordinat y dari Zoom frame baru, dalam poin. |
| width | Single | Lebar Zoom frame baru, dalam poin. |
| height | Single | Tinggi Zoom frame baru, dalam poin. |
| slide | ISlide | [`ISlide`](../../islide) yang dirujuk oleh Zoom frame. |

### Return Value

[`IZoomFrame`](../../izoomframe) yang baru dibuat.

### Exceptions

| eksepsi | kondisi |
| --- | --- |
| ArgumentException | Dilempar jika slide yang dirujuk tidak termasuk dalam presentasi saat ini. |

### Examples

Contoh ini menunjukkan pembuatan dan penyisipan objek Zoom pada indeks yang ditentukan dalam sebuah koleksi (asumsikan ada setidaknya dua slide dalam presentasi "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertZoomFrame(2, 150, 20, 50, 50, pres.Slides[1]);
}
```

## InsertZoomFrame(int, float, float, float, float, ISlide, IPPImage) {#insertzoomframe_1}

Membuat bingkai Zoom baru dengan gambar yang telah ditentukan dan menyisipkannya ke dalam koleksi shape pada indeks yang ditentukan.

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide, IPPImage image)
```

| Parameter | Type | Description |

| index | Int32 | Indeks berbasis nol tempat menyisipkan Zoom frame. |
| x | Single | Koordinat x dari Zoom frame baru, dalam poin. |
| y | Single | Koordinat y dari Zoom frame baru, dalam poin. |
| width | Single | Lebar Zoom frame baru, dalam poin. |
| height | Single | Tinggi Zoom frame baru, dalam poin. |
| slide | ISlide | [`ISlide`](../../islide) yang dirujuk oleh Zoom frame. |
| image | IPPImage | Gambar untuk slide yang dirujuk [`IPPImage`](../../ippimage). |

### Return Value

[`IZoomFrame`](../../izoomframe) yang baru dibuat.

### Exceptions

| eksepsi | kondisi |
| --- | --- |
| ArgumentException | Dilempar jika slide yang dirujuk tidak termasuk dalam presentasi saat ini. |

### Examples

Contoh ini menunjukkan pembuatan dan penyisipan objek Zoom pada indeks yang ditentukan dalam sebuah koleksi (asumsikan ada setidaknya dua slide dalam presentasi "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))

}
```

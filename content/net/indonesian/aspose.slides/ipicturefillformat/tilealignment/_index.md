## IPictureFillFormat.TileAlignment property

Mengembalikan atau mengatur cara tekstur disejajarkan dalam bentuk. Pengaturan ini mengontrol titik mulai pola tekstur dan cara pola tersebut diulang di seluruh bentuk. Baca/tulis [`RectangleAlignment`](../../rectanglealignment).

```csharp
public RectangleAlignment TileAlignment { get; set; }
```

### Catatan

Bawaan adalah TopLeft.

### Contoh

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Mendapatkan format isi gambar dari bentuk
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Mengatur mode isi gambar menjadi Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Mengatur perataan untuk pengulangan ke kanan bawah
    pictureFillFormat.TileAlignment = RectangleAlignment.BottomRight;
}
```

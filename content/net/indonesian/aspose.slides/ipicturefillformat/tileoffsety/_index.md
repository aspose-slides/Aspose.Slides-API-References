## IPictureFillFormat.TileOffsetY properti

Mengembalikan atau mengatur offset vertikal tekstur dari asal bentuk dalam satuan poin. Nilai positif memindahkan tekstur ke bawah, sementara nilai negatif memindahkannya ke atas. Baca/tulis Single.

```csharp
public float TileOffsetY { get; set; }
```

### Contoh

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Mendapatkan format isi gambar dari shape
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Mengatur mode isi gambar menjadi Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Mengatur offset vertikal tekstur menjadi -50 poin
    pictureFillFormat.TileOffsetY = -50;
}
```

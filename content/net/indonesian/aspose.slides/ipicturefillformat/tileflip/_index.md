## IPictureFillFormat.TileFlip properti

Membalik ubin tekstur di sepanjang sumbu horizontal, vertikal, atau keduanya. Baca/tulis [`TileFlip`](../../tileflip).

```csharp
public TileFlip TileFlip { get; set; }
```

### Keterangan

Default adalah NoFlip.

### Contoh

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Mendapatkan format isi gambar dari shape
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Mengatur mode isi gambar menjadi Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Membalik ubin tekstur di sepanjang sumbu vertikal.
    pictureFillFormat.TileFlip = TileFlip.FlipY;
}
```

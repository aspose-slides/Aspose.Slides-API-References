## IPictureFillFormat.DeletePictureCroppedAreas method

Hapus area terpotong pada Picture yang diisi.

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### Nilai Kembalian

Gambar terpotong atau gambar asli jika pemotongan tidak diperlukan.

### Catatan

Metode ini mengonversi file metafile WMF/EMF menjadi gambar raster PNG saat memotong.

### Contoh

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Mendapatkan PictureFrame
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // Menghapus area terpotong pada gambar PictureFrame
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

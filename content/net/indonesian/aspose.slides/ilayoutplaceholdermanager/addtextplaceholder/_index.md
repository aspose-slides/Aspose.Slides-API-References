## ILayoutPlaceholderManager.AddTextPlaceholder metode

Menambahkan shape placeholder baru ke slide tata letak untuk menampung konten teks.

```csharp
public IAutoShape AddTextPlaceholder(float x, float y, float width, float height)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| x | Single | Koordinat X dari shape placeholder baru. |
| y | Single | Koordinat Y dari shape placeholder baru. |
| width | Single | Lebar shape placeholder baru. |
| height | Single | Tinggi shape placeholder baru. |

### Nilai Kembali

Membuat [`IAutoShape`](../../iautoshape) dengan placeholder Text.

### Contoh

Contoh berikut menunjukkan cara menambahkan shape placeholder Text ke slide tata letak.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTextPlaceholder(20, 20, 500, 300);
}
```

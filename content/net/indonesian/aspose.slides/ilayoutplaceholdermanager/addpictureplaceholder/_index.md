## ILayoutPlaceholderManager.AddPicturePlaceholder metode

Menambahkan shape placeholder baru ke slide tata letak untuk menampung Picture.

```csharp
public IAutoShape AddPicturePlaceholder(float x, float y, float width, float height)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| x | Single | Koordinat X dari shape placeholder baru. |
| y | Single | Koordinat Y dari shape placeholder baru. |
| width | Single | Lebar shape placeholder baru. |
| height | Single | Tinggi shape placeholder baru. |

### Nilai Kembali

Dibuat [`IAutoShape`](../../iautoshape) dengan placeholder Picture.

### Contoh

Contoh berikut menunjukkan cara menambahkan shape placeholder Picture ke slide tata letak.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddPicturePlaceholder(20, 20, 200, 200);
}
```

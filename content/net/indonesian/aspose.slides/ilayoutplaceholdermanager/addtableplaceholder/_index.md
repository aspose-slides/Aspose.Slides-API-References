## ILayoutPlaceholderManager.AddTablePlaceholder metode

Menambahkan bentuk placeholder baru ke slide tata letak untuk menampung sebuah Table.

```csharp
public IAutoShape AddTablePlaceholder(float x, float y, float width, float height)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| x | Single | Koordinat X dari bentuk placeholder baru. |
| y | Single | Koordinat Y dari bentuk placeholder baru. |
| width | Single | Lebar bentuk placeholder baru. |
| height | Single | Tinggi bentuk placeholder baru. |

### Nilai Kembalian

Membuat [`IAutoShape`](../../iautoshape) dengan placeholder Table.

### Contoh

Contoh berikut menunjukkan cara menambahkan bentuk placeholder Table ke slide tata letak.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTablePlaceholder(20, 20, 500, 200);
}
```

## ILayoutPlaceholderManager.AddChartPlaceholder method

Menambahkan shape placeholder baru ke slide tata letak untuk menampung diagram.

```csharp
public IAutoShape AddChartPlaceholder(float x, float y, float width, float height)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| x | Single | Koordinat X dari shape placeholder baru. |
| y | Single | Koordinat Y dari shape placeholder baru. |
| width | Single | Lebar shape placeholder baru. |
| height | Single | Tinggi shape placeholder baru. |

### Nilai Kembali

Membuat [`IAutoShape`](../../iautoshape) dengan placeholder Chart.

### Contoh

Contoh berikut menunjukkan cara menambahkan shape placeholder Chart ke slide tata letak.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddChartPlaceholder(20, 20, 200, 200);
}
```

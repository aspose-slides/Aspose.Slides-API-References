## ILayoutPlaceholderManager.AddVerticalTextPlaceholder metode

Menambahkan bentuk placeholder baru ke slide tata letak untuk menampung konten teks dalam arah vertikal.

```csharp
public IAutoShape AddVerticalTextPlaceholder(float x, float y, float width, float height)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| x | Single | Koordinat X dari bentuk placeholder baru. |
| y | Single | Koordinat Y dari bentuk placeholder baru. |
| width | Single | Lebar dari bentuk placeholder baru. |
| height | Single | Tinggi dari bentuk placeholder baru. |

### Nilai Kembali

Dibuat [`IAutoShape`](../../iautoshape) dengan placeholder Teks (Vertikal).

### Contoh

Contoh berikut menunjukkan cara menambahkan bentuk placeholder Teks (Vertikal) ke slide tata letak.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalTextPlaceholder(20, 20, 300, 500);
}
```

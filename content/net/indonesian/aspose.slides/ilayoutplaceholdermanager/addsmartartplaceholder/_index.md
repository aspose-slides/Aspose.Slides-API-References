## ILayoutPlaceholderManager.AddSmartArtPlaceholder method

Menambahkan shape placeholder baru ke slide tata letak untuk menampung diagram SmartArt.

```csharp
public IAutoShape AddSmartArtPlaceholder(float x, float y, float width, float height)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| x | Single | Koordinat X dari shape placeholder baru. |
| y | Single | Koordinat Y dari shape placeholder baru. |
| width | Single | Lebar dari shape placeholder baru. |
| height | Single | Tinggi dari shape placeholder baru. |

### Nilai Kembali

Membuat [`IAutoShape`](../../iautoshape) dengan placeholder SmartArt.

### Contoh

Contoh berikut menunjukkan cara menambahkan shape placeholder SmartArt ke slide tata letak.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddSmartArtPlaceholder(20, 20, 200, 200);
}
```

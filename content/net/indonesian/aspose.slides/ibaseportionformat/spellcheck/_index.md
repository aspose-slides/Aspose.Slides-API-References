## IBasePortionFormat.SpellCheck properti

Mendapatkan atau menetapkan nilai yang menunjukkan apakah pemeriksaan ejaan diaktifkan untuk bagian teks. Ketika properti ini diatur ke false, pemeriksaan ejaan untuk elemen teks tidak dilakukan. Ketika diatur ke true, pemeriksaan ejaan diizinkan. Nilai default adalah `false`.

```csharp
public bool SpellCheck { get; set; }
```

### Contoh

Contoh berikut menunjukkan cara mengaktifkan flag SpellCheck sebelum menyimpan presentasi:

```csharp
[C#]
using (var pres = new Presentation("input.pptx"))
{
    // Akses bagian teks pertama di dalam shape pertama pada slide pertama
    var portion = ((AutoShape)pres.Slides[0].Shapes[0]).TextFrame.Paragraphs[0].Portions[0];
    // Aktifkan pemeriksaan ejaan untuk bagian teks ini
    portion.PortionFormat.SpellCheck = true;
    // Simpan presentasi yang telah dimodifikasi
    pres.Save("output-with-spellcheck.pptx", SaveFormat.Pptx);
}
```

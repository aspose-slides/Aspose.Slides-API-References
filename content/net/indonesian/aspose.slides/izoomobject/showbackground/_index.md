## IZoomObject.ShowBackground properti

Mendapatkan atau menetapkan nilai yang menentukan apakah Zoom akan menggunakan latar belakang slide tujuan. Baca/tulis Boolean. Nilai default: true

```csharp
public bool ShowBackground { get; set; }
```

### Contoh

Contoh ini menunjukkan cara menghapus latar belakang gambar objek Zoom:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
    zoomFrame.ShowBackground = false;
}
```

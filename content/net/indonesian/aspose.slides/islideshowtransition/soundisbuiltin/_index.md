## Properti ISlideShowTransition.SoundIsBuiltIn

Menentukan apakah suara ini merupakan suara bawaan atau tidak. Jika atribut ini diatur ke true, maka aplikasi yang menghasilkan akan diberi peringatan untuk memeriksa atribut name yang ditentukan untuk suara ini dalam daftar suara bawaan dan kemudian dapat menampilkan nama khusus atau UI sesuai kebutuhan. Boolean baca-tulis.

```csharp
public bool SoundIsBuiltIn { get; set; }
```

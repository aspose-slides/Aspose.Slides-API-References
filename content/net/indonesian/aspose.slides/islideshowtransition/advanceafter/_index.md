## ISlideShowTransition.AdvanceAfter properti

This attribute specifies if the slideshow will move to the next slide after a certain time. Baca/tulis Boolean.

```csharp
public bool AdvanceAfter { get; set; }
```

### Contoh

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    // Dapatkan transisi slide pertama
    ISlideShowTransition slideTransition = pres.Slides[0].SlideShowTransition;
    
    // Periksa apakah flag Advance Slide After diaktifkan
    if (slideTransition.AdvanceAfter)
    {
        // Dapatkan nilai waktu Advance Slide After
        uint advanceAfterTime = slideTransition.AdvanceAfterTime;
    }
}
```

## DocumentProperties.ScaleCrop property

Menunjukkan mode tampilan thumbnail dokumen. Setel elemen ini ke **true** untuk mengaktifkan skala thumbnail dokumen ke tampilan. Setel elemen ini ke **false** untuk mengaktifkan pemotongan thumbnail dokumen sehingga hanya menampilkan bagian yang sesuai dengan tampilan. Baca/tulis Boolean.

public bool ScaleCrop { get; set; }

## SVGOptions.DisableLineEndCropping properti

SVG 1.1 tidak memiliki kemampuan untuk mendefinisikan inset untuk penanda. Mesin penulisan SVG Aspose.Slides memiliki solusi untuk masalah itu: ia memotong ujung garis dengan panah, sehingga garis tidak tumpang tindih dengan penanda. Opsi ini menonaktifkan perilaku tersebut. Baca/tulis Boolean.

```csharp
public bool DisableLineEndCropping { get; set; }
```

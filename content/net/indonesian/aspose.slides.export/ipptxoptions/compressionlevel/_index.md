## IPptxOptions.CompressionLevel properti

Menentukan tingkat kompresi yang digunakan saat menyimpan dokumen presentasi. Nilai default adalah Level6.

```csharp
public CompressionLevel CompressionLevel { get; set; }
```

### Catatan

Tingkat kompresi yang lebih tinggi menghasilkan file yang lebih kecil tetapi memerlukan waktu pemrosesan yang lebih lama. Rasio kompresi aktual bergantung pada konten presentasi.

### Contoh

Contoh:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("demo-level8.pptx", SaveFormat.Pptx, new PptxOptions()
    {
        CompressionLevel = CompressionLevel.Level8
    });
}
```

## Zip64Mode enumerasi

Menentukan kapan harus menggunakan ekstensi format ZIP64 untuk file OpenXML.

```csharp
public enum Zip64Mode
```

### Nilai

| Nama | Nilai | Deskripsi |
| --- | --- | --- |
| Never | `0` | Jangan gunakan ekstensi format ZIP64. |
| IfNecessary | `1` | Gunakan ekstensi format ZIP64 jika diperlukan. |
| Always | `2` | Selalu gunakan ekstensi format ZIP64. |

### Keterangan

File OpenXML adalah arsip ZIP yang memiliki batas 4 GB (2^32 byte) pada ukuran file yang tidak terkompresi, ukuran file terkompresi, dan total ukuran arsip, serta batas 65.535 (2^16-1) file dalam arsip. Ekstensi format ZIP64 meningkatkan batas tersebut menjadi 2^64.

## ISaveOptions.SkipJavaScriptLinks properti

Menentukan apakah akan melewatkan hyperlink dengan pemanggilan JavaScript saat menyimpan presentasi. Baca/tulis Boolean. Nilai default adalah **false**.

```csharp
public bool SkipJavaScriptLinks { get; set; }
```

### Catatan

Ketika properti ini diatur ke **true**, tautan hiper dengan pemanggilan JavaScript akan diabaikan saat menyimpan.

Ketika properti ini diatur ke **false**, semua tautan hiper akan disimpan.

### Contoh

Contoh:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("result_without_JavaScript_links.html", SaveFormat.Html, new HtmlOptions()
    {
        SkipJavaScriptLinks = true
    });
}
```

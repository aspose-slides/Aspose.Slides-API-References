### Nilai

| Nama | Nilai | Deskripsi |
| --- | --- | --- |
| OnFocusIn | `0` | Terjadi ketika sebuah elemen menerima fokus, seperti ketika teks menjadi terpilih. |
| OnFocusOut | `1` | Terjadi ketika sebuah elemen kehilangan fokus, seperti ketika teks menjadi tidak terpilih. |
| OnActivate | `2` | Terjadi ketika sebuah elemen diaktifkan, misalnya melalui klik mouse atau penekanan tombol. |
| OnClick | `3` | Terjadi ketika tombol perangkat penunjuk diklik di atas sebuah elemen. |
| OnMouseDown | `4` | Terjadi ketika tombol perangkat penunjuk ditekan di atas sebuah elemen. |
| OnMouseUp | `5` | Terjadi ketika tombol perangkat penunjuk dilepaskan di atas sebuah elemen. |
| OnMouseOver | `6` | Terjadi ketika perangkat penunjuk dipindahkan ke sebuah elemen. |
| OnMouseMove | `7` | Terjadi ketika perangkat penunjuk dipindahkan saat berada di atas sebuah elemen. |
| OnMouseOut | `8` | Terjadi ketika perangkat penunjuk dipindahkan menjauh dari sebuah elemen. |
| OnLoad | `9` | Terjadi ketika agen pengguna telah sepenuhnya mengurai elemen serta turunannya dan semua sumber daya yang dirujuk, yang diperlukan untuk merendernya. |

| OnAbort | `11` | Terjadi ketika pemuatan halaman dihentikan sebelum sebuah elemen selesai dimuat sepenuhnya. |
| OnError | `12` | Terjadi ketika sebuah elemen tidak dimuat dengan benar atau ketika terjadi kesalahan selama eksekusi skrip. |
| OnResize | `13` | Terjadi ketika tampilan dokumen diubah ukurannya. Hanya berlaku untuk elemen svg terluar. |
| OnScroll | `14` | Terjadi ketika tampilan dokumen digeser sepanjang sumbu X atau Y atau keduanya. Hanya berlaku untuk elemen svg terluar. |
| OnZoom | `15` | Terjadi ketika tingkat zoom tampilan dokumen diubah. Hanya berlaku untuk elemen svg terluar. |
| OnBegin | `16` | Terjadi ketika elemen animasi dimulai. |
| OnEnd | `17` | Terjadi ketika elemen animasi berakhir. |
| OnRepeat | `18` | Terjadi ketika elemen animasi diulang. |

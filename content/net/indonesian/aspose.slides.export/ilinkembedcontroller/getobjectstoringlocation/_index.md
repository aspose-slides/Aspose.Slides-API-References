## ILinkEmbedController.GetObjectStoringLocation metode

Menentukan di mana objek harus disimpan. Metode ini dipanggil sekali untuk setiap id objek. Tidak dijamin tidak akan ada dua objek dengan data, semanticName, dan contentType yang sama namun dengan id yang berbeda.

```csharp
public LinkEmbedDecision GetObjectStoringLocation(int id, byte[] entityData, string semanticName, 
    string contentType, string recomendedExtension)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| id | Int32 | Id objek. Id ini unik untuk seluruh operasi penyimpanan. |
| entityData | Byte[] | Data biner objek. Parameter ini dapat bernilai null, jika data biner objek belum dihasilkan. |
| semanticName | String | Beberapa teks pendek yang menjelaskan arti objek. Controller dapat menggunakan ini sebagai bagian dari nama objek eksternal, tetapi terserah dispatcher untuk memastikan bahwa nama tersebut unik dan hanya berisi karakter yang diizinkan. |
| contentType | String | Tipe MIME objek. |
| recomendedExtension | String | Ekstensi nama file, yang direkomendasikan untuk tipe MIME ini. |

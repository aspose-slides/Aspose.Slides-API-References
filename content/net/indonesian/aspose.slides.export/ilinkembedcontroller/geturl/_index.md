## ILinkEmbedController.GetUrl metode

Mengembalikan URL ke objek eksternal. Metode ini selalu dipanggil jika [`GetObjectStoringLocation`](../getobjectstoringlocation) mengembalikan Link dan dapat dipanggil jika [`GetObjectStoringLocation`](../getobjectstoringlocation) mengembalikan Embed tetapi penyematan tidak memungkinkan. Dapat dipanggil berkali-kali untuk ID objek yang sama.

```csharp
public string GetUrl(int id, int referrer)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| id | Int32 | ID objek. ID ini bersifat unik untuk seluruh operasi. |
| referrer | Int32 | ID objek yang merujuk atau 0, jika objek dirujuk oleh dokumen root. Dapat digunakan untuk menghasilkan tautan relatif. |

## ISwfOptions.SlidesLayoutOptions properti

Mendapatkan atau mengatur mode di mana slide ditempatkan pada halaman saat mengekspor presentasi [`ISlidesLayoutOptions`](../../islideslayoutoptions). Properti ini tidak mendukung penugasan objek dengan tipe `Aspose.Slides.Export.HandoutLayoutingOptions`

```csharp
public ISlidesLayoutOptions SlidesLayoutOptions { get; set; }
```

### Contoh

Contoh:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    SwfOptions options = new SwfOptions
    {
        SlidesLayoutOptions = new NotesCommentsLayoutingOptions
        {
            CommentsPosition = CommentsPositions.Right
        }
    };
    
    pres.Save("pres.swf", SaveFormat.Swf, options);
}
```

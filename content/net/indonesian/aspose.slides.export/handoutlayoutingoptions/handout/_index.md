## HandoutLayoutingOptions.Handout properti

Menentukan berapa banyak slide dan dalam urutan apa yang akan ditempatkan pada halaman [`HandoutType`](../../handouttype).

```csharp
public HandoutType Handout { get; set; }
```

### Catatan

Nilai default adalah **HandoutType.Handouts6Horizontal**.

### Contoh

Contoh:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    RenderingOptions options = new RenderingOptions
    {
        SlidesLayoutOptions = new HandoutLayoutingOptions
        {
            Handout = HandoutType.Handouts4Horizontal
        }
    };
    
    pres.Slides[0].GetThumbnail(options, new Size(1920, 1080)).Save("pres-handout.png");
}
```

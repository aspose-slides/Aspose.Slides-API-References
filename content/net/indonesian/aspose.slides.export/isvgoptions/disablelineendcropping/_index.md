## ISVGOptions.DisableLineEndCropping properti

SVG 1.1 tidak memiliki kemampuan untuk mendefinisikan inset pada penanda. Mesin penulisan SVG Aspose.Slides memiliki solusi untuk masalah tersebut: ia memotong ujung garis dengan panah, sehingga garis tidak tumpang tindih dengan penanda. Opsi ini mematikan perilaku tersebut. Baca/tulis Boolean.

public bool DisableLineEndCropping { get; set; }

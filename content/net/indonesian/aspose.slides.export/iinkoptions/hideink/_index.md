## IInkOptions.HideInk properti

Menampilkan atau menyembunyikan elemen Ink dalam dokumen yang diekspor.

```csharp
public bool HideInk { get; set; }
```

### Catatan

Nilai default adalah false.

### Contoh

Contoh berikut menunjukkan cara menyembunyikan elemen Ink dalam dokumen PDF yang diekspor:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.HideInk = true;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

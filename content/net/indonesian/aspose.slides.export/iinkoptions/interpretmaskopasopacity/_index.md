## IInkOptions.InterpretMaskOpAsOpacity properti

Menggunakan operasi ROP atau Opacity untuk merender kuas.

```csharp
public bool InterpretMaskOpAsOpacity { get; set; }
```

### Keterangan

Nilai default adalah true.

### Contoh

Contoh berikut menunjukkan cara mengatur menggunakan ROP untuk mengekspor elemen Ink:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.InterpretMaskOpAsOpacity = false;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

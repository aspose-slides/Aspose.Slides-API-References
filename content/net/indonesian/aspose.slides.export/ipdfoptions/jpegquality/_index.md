## IPdfOptions.JpegQuality properti

Mengembalikan atau menetapkan nilai yang menentukan kualitas gambar JPEG di dalam dokumen PDF. Baca/tulis Byte.

```csharp
public byte JpegQuality { get; set; }
```

### Catatan

Berpengaruh hanya ketika dokumen berisi gambar JPEG.

Gunakan properti ini untuk mendapatkan atau menetapkan kualitas gambar di dalam dokumen saat menyimpan dalam format PDF. Nilai dapat bervariasi dari 0 sampai 100 dimana 0 berarti kualitas terburuk tetapi kompresi maksimum dan 100 berarti kualitas terbaik tetapi kompresi minimum.

Nilai default adalah **100**.

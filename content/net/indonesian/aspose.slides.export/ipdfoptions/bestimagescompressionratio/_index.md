## IPdfOptions.BestImagesCompressionRatio properti

Menunjukkan apakah kompresi paling efektif (alih-alih yang default) untuk setiap gambar harus dipilih secara otomatis. Jika disetel ke Boolean.true, untuk setiap gambar dalam presentasi algoritma kompresi yang paling tepat akan dipilih, yang akan menghasilkan ukuran dokumen PDF yang lebih kecil. Pemilihan rasio kompresi gambar terbaik membutuhkan komputasi yang berat dan memerlukan tambahan RAM, dan opsi ini secara default adalah Boolean.false.

```csharp
public bool BestImagesCompressionRatio { get; set; }
```

### Catatan

Default adalah Boolean.false.

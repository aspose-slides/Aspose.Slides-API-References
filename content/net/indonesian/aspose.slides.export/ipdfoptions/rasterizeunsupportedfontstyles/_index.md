## IPdfOptions.RasterizeUnsupportedFontStyles properti

Menunjukkan apakah teks harus dirasterkan sebagai bitmap dan disimpan ke PDF ketika font tidak mendukung gaya tebal. Pendekatan ini dapat meningkatkan kualitas teks dalam PDF yang dihasilkan untuk font tertentu. Baca/tulis Boolean.

```csharp
public bool RasterizeUnsupportedFontStyles { get; set; }
```

### Keterangan

Nilai default adalah **false**.

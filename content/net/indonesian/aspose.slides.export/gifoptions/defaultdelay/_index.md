## GifOptions.DefaultDelay properti

Mendapatkan atau mengatur waktu tunda default [ms]. Nilai ini akan digunakan jika [`AdvanceAfterTime`](../../../aspose.slides/islideshowtransition/advanceaftertime) tidak diatur. Nilai defaultnya adalah 1000.

```csharp
public int DefaultDelay { get; set; }
```

### Contoh

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
	pres.Save("pres.gif", SaveFormat.Gif, new GifOptions { DefaultDelay = 2000 });
}
```

## PdfOptions.BestImagesCompressionRatio properti

Menunjukkan apakah kompresi paling efektif (bukan yang default) untuk setiap gambar harus dipilih secara otomatis. Jika diatur ke Boolean.true, untuk setiap gambar dalam presentasi algoritma kompresi yang paling tepat akan dipilih, yang akan menghasilkan ukuran dokumen PDF yang lebih kecil. Pemilihan rasio kompresi gambar terbaik memerlukan biaya komputasi tinggi dan membutuhkan tambahan RAM, dan opsi ini bernilai Boolean.false secara default.

```csharp
public bool BestImagesCompressionRatio { get; set; }
```

### Catatan

Nilai default adalah Boolean.false.

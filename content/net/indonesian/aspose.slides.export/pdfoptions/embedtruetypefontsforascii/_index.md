## PdfOptions.EmbedTrueTypeFontsForASCII properti

Menentukan apakah Aspose.Slides akan menyematkan font umum untuk teks ASCII (rentang kode 33..127). Font untuk kode karakter lebih besar dari 127 selalu disematkan. Daftar font umum mencakup 14 font dasar PDF dan font tambahan yang ditentukan pengguna. Baca/tulis Boolean.

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### Catatan

Default adalah **true**.

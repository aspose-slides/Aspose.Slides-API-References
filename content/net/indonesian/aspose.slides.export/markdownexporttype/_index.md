## MarkdownExportType enumeration

Jenis rendering dokumen.

```csharp
public enum MarkdownExportType
```

### Nilai

| Nama | Nilai | Deskripsi |
| --- | --- | --- |
| Sequential | `0` | Render semua item secara terpisah. Satu per satu. |
| TextOnly | `1` | Render hanya teks. |
| Visual | `2` | Render semua item, item yang dikelompokkan - render bersama-sama. |

### Contoh

Contoh:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    MarkdownSaveOptions markdownSaveOptions = new MarkdownSaveOptions
    {
        ShowHiddenSlides = true,
        ShowSlideNumber = true,
        Flavor = Flavor.Github,
        ExportType = MarkdownExportType.Sequential,
        NewLineType = NewLineType.Windows
    };
    
    pres.Save("doc.md", new[] { 1, 2, 3, 4, 5, 6, 7, 8, 9 }, SaveFormat.Md, markdownSaveOptions);
```

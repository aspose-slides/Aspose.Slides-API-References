## RenderingOptions.DisableFontLigatures properti

Mendapatkan atau mengatur nilai yang menunjukkan apakah teks di-render tanpa menggunakan ligatur. Ketika diatur ke `true`, ligatur akan dinonaktifkan dalam output yang di-render. Secara default, properti ini diatur ke `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Contoh

Contoh:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    RenderingOptions options = new RenderingOptions
    {
        DisableFontLigatures = true // Nonaktifkan ligatur dalam rendering teks
    };
    
    Bitmap[] renderedSlides = pres.GetImage(options);
    for (var index = 0; index < renderedSlides.Length; index++)
    {
        var slideImage = renderedSlides[index];
        slideImage.Save($"slide-{index}.png");
    }
}
```

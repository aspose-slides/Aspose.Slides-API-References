## Properti IMathGroupingCharacter.VerticalJustification

Justifikasi vertikal karakter grup. Menentukan perataan objek relatif terhadap garis dasar. Misalnya, ketika karakter grup berada di atas objek, VerticalJustification dengan nilai Top menandakan bahwa bagian atas objek berada pada garis dasar; ketika VerticalJustification diatur ke Bottom, bagian bawah objek berada pada garis dasar Bawaan: Bottom untuk Position=Top, dan Top untuk Position=Bottom

```csharp
public MathTopBotPositions VerticalJustification { get; set; }
```

### Contoh

Contoh:

```csharp
[C#]
MathGroupingCharacter groupingCharacter = new MathGroupingCharacter(new MathematicalText("abc"));
groupingCharacter.VerticalJustification = MathTopBotPositions.Top;
```

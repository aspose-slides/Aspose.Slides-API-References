## IMathArray.RowSpacing property

Jarak antar baris array Hanya digunakan ketika RowSpacingRule diatur ke 3 Jika tepat, satuan pengukuran adalah poin atau Multiple, satuan pengukuran adalah setengah baris. Default: 0

```csharp
public uint RowSpacing { get; set; }
```

### Contoh

Contoh:

```csharp
[C#]
IMathArray mathArray = new MathArray(new MathematicalText("item1"));
mathArray.RowSpacingRule = MathRowSpacingRule.Exactly;
mathArray.RowSpacing = 10;
```

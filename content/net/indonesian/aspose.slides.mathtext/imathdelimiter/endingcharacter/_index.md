## IMathDelimiter.EndingCharacter properti

Delimiter Ending Character menentukan karakter delimiter penutup, atau akhir. Delimiter matematis adalah karakter yang mengelilingi seperti tanda kurung, tanda kurung siku, dan kurung kurawal. Defaultnya: ')'.

```csharp
public char EndingCharacter { get; set; }
```

### Contoh

Contoh:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Join("y").Enclose();
delimiter.EndingCharacter = ']';
```

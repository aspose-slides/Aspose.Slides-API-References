## IMathDelimiter.DelimiterShape properti

Menentukan bentuk pemisah dalam objek delimiter. Ketika nilai adalah MathDelimiterShape.Centered, delimiter diposisikan di tengah sekitar sumbu matematika dari teks matematis dan masih dapat dibuat agar sesuai dengan seluruh tinggi kontennya. Ketika nilai adalah MathDelimiterShape.Match, tinggi dan bentuknya diubah agar tepat cocok dengan kontennya.

```csharp
public MathDelimiterShape DelimiterShape { get; set; }
```

### Contoh

Contoh:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.DelimiterShape = MathDelimiterShape.Match;
```

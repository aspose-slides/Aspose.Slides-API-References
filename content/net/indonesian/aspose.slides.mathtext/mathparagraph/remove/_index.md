## MathParagraph.Remove metode

Menghapus kemunculan pertama dari objek tertentu dari koleksi/>.

```csharp
public bool Remove(IMathBlock mathBlock)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| mathBlock | IMathBlock | Objek yang akan dihapus dari koleksi. |

### Nilai Kembali

true jika *mathBlock* berhasil dihapus dari koleksi; selain itu, false. Metode ini juga mengembalikan false jika *mathBlock* tidak ditemukan dalam koleksi asli/>.

### Contoh

Contoh:

```csharp
[C#]
IAutoShape shape = slide.Shapes.AddMathShape(x, y, width, height);
IMathParagraph mathParagraph = (shape.TextFrame.Paragraphs[0].Portions[0] as MathPortion).MathParagraph;
mathParagraph.Add(new MathBlock(new MathematicalText("x")));
IMathBlock block = new MathBlock(new MathematicalText("y"));
mathParagraph.Add(block);
mathParagraph.Remove(block);
```

## IMathBorderBox.StrikethroughTopLeftToBottomRight properti

Strikethrough Top-Left to Bottom-Right (default adalah false). Menentukan keadaan tersembunyi atau ditampilkan dari garis diagonal coret yang melintang dari sudut kiri atas ke sudut kanan bawah dari kotak batas.

```csharp
public bool StrikethroughTopLeftToBottomRight { get; set; }
```

### Contoh

Contoh:

```csharp
[C#]
IMathBorderBox borderBox = new MathematicalText("x+y+z").ToBorderBox();
borderBox.StrikethroughTopLeftToBottomRight = true;
```

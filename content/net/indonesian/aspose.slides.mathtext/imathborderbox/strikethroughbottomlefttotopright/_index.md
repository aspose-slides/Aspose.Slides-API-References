## IMathBorderBox.StrikethroughBottomLeftToTopRight properti

Strikethrough Bottom-Left to Top-Right (default is false). Menentukan keadaan tersembunyi atau terlihat dari garis diagonal strikethrough dari sudut kiri bawah ke sudut kanan atas kotak batas.

```csharp
public bool StrikethroughBottomLeftToTopRight { get; set; }
```

### Contoh

Contoh:

```csharp
[C#]
IMathBorderBox borderBox = new MathematicalText("x+y+z").ToBorderBox();
borderBox.StrikethroughBottomLeftToTopRight = true;
```

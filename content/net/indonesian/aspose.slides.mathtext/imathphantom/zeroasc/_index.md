## IMathPhantom.ZeroAsc properti

Mendapatkan atau menetapkan nilai yang menunjukkan apakah kenaikan (tinggi di atas garis dasar) elemen dasar harus diperlakukan sebagai nol.

```csharp
public bool ZeroAsc { get; set; }
```

### Catatan

Ketika `true`, phantom tidak menaikkan garis dasar dari baris matematika di sekitarnya. Berkorespondensi dengan atribut OMML `m:zeroAsc`.

## MathPhantom.Show properti

Mendapat atau mengatur nilai yang menunjukkan apakah elemen dasar ditampilkan.

```csharp
public bool Show { get; set; }
```

### Catatan

Ketika `false`, elemen dasar disembunyikan tetapi masih dapat menempati ruang tergantung pada pengaturan phantom lainnya. Berkorespondensi dengan atribut OMML `m:show`.

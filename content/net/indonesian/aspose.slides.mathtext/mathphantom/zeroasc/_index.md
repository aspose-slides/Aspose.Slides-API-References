## MathPhantom.ZeroAsc properti

Mendapatkan atau menetapkan nilai yang menunjukkan apakah ascent (height above baseline) elemen dasar harus diperlakukan sebagai nol.

```csharp
public bool ZeroAsc { get; set; }
```

### Catatan

Ketika `true`, phantom tidak menaikkan baseline dari baris matematika di sekitarnya. Sesuai dengan atribut OMML `m:zeroAsc`.

## MathArray.RowSpacing property

Jarak antar baris dalam sebuah array. Hanya digunakan ketika RowSpacingRule diatur ke 3 Exactly, dalam kasus ini satuan ukur adalah poin, atau Multiple, dalam kasus ini satuan ukur adalah setengah baris. Default: 0

```csharp
public uint RowSpacing { get; set; }
```

### Contoh

Contoh:

```csharp
[C#]
IMathArray mathArray = new MathArray(new MathematicalText("item1"));
mathArray.RowSpacingRule = MathRowSpacingRule.Exactly;
mathArray.RowSpacing = 10;
```

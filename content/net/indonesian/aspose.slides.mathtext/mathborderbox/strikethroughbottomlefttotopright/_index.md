## MathBorderBox.StrikethroughBottomLeftToTopRight properti

Coretan diagonal dari kiri bawah ke kanan atas (default adalah false). Menentukan apakah garis diagonal coretan terlihat atau disembunyikan dari sudut kiri bawah ke sudut kanan atas kotak batas.

```csharp
public bool StrikethroughBottomLeftToTopRight { get; set; }
```

### Contoh

Contoh:

```csharp
[C#]
MathBorderBox borderBox = new MathBorderBox(new MathematicalText("x"));
borderBox.StrikethroughBottomLeftToTopRight = true;
```

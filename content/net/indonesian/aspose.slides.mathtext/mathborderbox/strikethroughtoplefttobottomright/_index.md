## MathBorderBox.StrikethroughTopLeftToBottomRight properti

Strikethrough Top-Left ke Bottom-Right (default is false). Menentukan keadaan tersembunyi atau terlihat dari garis diagonal strikethrough dari sudut kiri atas ke sudut kanan bawah kotak batas.

```csharp
public bool StrikethroughTopLeftToBottomRight { get; set; }
```

### Contoh

Contoh:

```csharp
[C#]
MathBorderBox borderBox = new MathBorderBox(new MathematicalText("x"));
borderBox.StrikethroughTopLeftToBottomRight = true;
```

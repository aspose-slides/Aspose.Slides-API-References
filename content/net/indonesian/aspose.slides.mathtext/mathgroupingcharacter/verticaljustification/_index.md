## MathGroupingCharacter.VerticalJustification properti

Justifikasi vertikal karakter grup. Menentukan perataan objek relatif terhadap baseline. Sebagai contoh, ketika karakter grup berada di atas objek, VerticalJustification dari Top menandakan bahwa bagian atas objek berada pada baseline; ketika VerticalJustification diatur ke Bottom, bagian bawah objek berada pada baseline Default: Bottom for Position=Top, and Top for Position=Bottom

```csharp
public MathTopBotPositions VerticalJustification { get; set; }
```

### Contoh

Contoh:

```csharp
[C#]
MathGroupingCharacter groupingCharacter = new MathGroupingCharacter(new MathematicalText("abc"));
groupingCharacter.VerticalJustification = MathTopBotPositions.Top;
```

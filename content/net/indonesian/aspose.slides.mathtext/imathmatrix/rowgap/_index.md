## IMathMatrix.RowGap properti

Nilai jarak vertikal antara baris-baris matriks; Jika RowGapRule diatur ke 3 ("Exactly"), maka satuannya diartikan sebagai twip (1/20 poin) Jika RowGapRule diatur ke 4 ("Multiple"), maka satuannya diartikan sebagai setengah baris. Default: 0

```csharp
public uint RowGap { get; set; }
```

### Contoh

Contoh:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

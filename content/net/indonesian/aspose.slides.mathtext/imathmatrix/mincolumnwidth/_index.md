## IMathMatrix.MinColumnWidth properti

Lebar kolom minimum dalam twips (1/20th of a point) Jarak celah (also referred to as “Column Gap” or “Gap Width”) ditambahkan ke MinColumnWidth untuk menentukan total Matrix Column Spacing (distance between the same edges of different columns). Default: 0.

```csharp
public uint MinColumnWidth { get; set; }
```

### Contoh

Contoh:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.MinColumnWidth = 20;
```

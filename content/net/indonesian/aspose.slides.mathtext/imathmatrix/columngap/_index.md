## IMathMatrix.ColumnGap properti

Nilai jarak horizontal antar kolom dari sebuah matriks; Jika ColumnGapRule diatur ke 3 ("Exactly"), maka satuannya diinterpretasikan sebagai twip (1/20 poin) Jika ColumnGapRule diatur ke 4 ("Multiple"), maka satuannya diinterpretasikan sebagai jumlah kenaikan 0.5 em. Dalam kasus lain diabaikan. Default: 0

```csharp
public uint ColumnGap { get; set; }
```

### Contoh

Contoh:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.ColumnGapRule = MathSpacingRules.Exactly;
matrix.ColumnGap = 20;
```

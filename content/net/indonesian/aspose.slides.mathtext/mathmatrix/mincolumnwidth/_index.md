## MathMatrix.MinColumnWidth properti

Lebar kolom minimum dalam twip (1/20 poin). Jarak celah (juga disebut “Column Gap” atau “Gap Width”) ditambahkan ke MinColumnWidth untuk menentukan total Spacing Kolom Matriks (jarak antara tepi yang sama dari kolom yang berbeda). Default: 0.

```csharp
public uint MinColumnWidth { get; set; }
```

### Contoh

Contoh:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.MinColumnWidth = 20;
```

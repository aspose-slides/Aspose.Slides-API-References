## MathDelimiter.BeginningCharacter properti

Delimiter Beginning Character menentukan karakter delimiter awal, atau pembuka. Delimiter matematis adalah karakter yang mengelilingi seperti kurung, tanda kurung siku, dan kurung kurawal. Nilai default: '('.

```csharp
public char BeginningCharacter { get; set; }
```

### Contoh

Contoh:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Join("y").Enclose();
delimiter.BeginningCharacter = '[';
```

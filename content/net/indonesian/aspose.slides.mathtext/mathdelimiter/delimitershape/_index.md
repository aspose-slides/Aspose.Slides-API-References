## MathDelimiter.DelimiterShape properti

Menentukan bentuk delimiter dalam objek delimiter. Ketika nilai adalah MathDelimiterShape.Centered, delimiter diposisikan di tengah sumbu matematika dari teks matematis dan tetap dibuat agar cocok dengan seluruh tinggi kontennya. Ketika nilai adalah MathDelimiterShape.Match, tinggi dan bentuknya diubah agar persis cocok dengan kontennya.

```csharp
public MathDelimiterShape DelimiterShape { get; set; }
```

### Contoh

Contoh:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.DelimiterShape = MathDelimiterShape.Match;
```

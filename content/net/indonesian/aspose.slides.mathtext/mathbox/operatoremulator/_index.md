## MathBox.OperatorEmulator properti

Operator Emulator. Ketika true, kotak dan isinya berperilaku sebagai satu operator tunggal dan mewarisi properti dari sebuah operator. Ini berarti, misalnya, bahwa karakter dapat berfungsi sebagai titik pemutusan baris dan dapat disejajarkan dengan operator lain. Operator Emulator sering digunakan ketika satu atau lebih glyph digabungkan menjadi sebuah operator, seperti '=='. Nilai default: false

```csharp
public bool OperatorEmulator { get; set; }
```

### Contoh

Contoh:

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("=="));
box.OperatorEmulator = true;
```

## MathBox.ExplicitBreak properti

Explicit break menentukan apakah ada jeda baris di awal objek Box, sehingga baris membungkus di awal objek box. Menentukan nomor operator pada baris sebelumnya dari teks matematis yang akan digunakan sebagai titik penjajaran untuk baris teks matematis saat ini nilai yang mungkin: 1..255 Default: 0 (tidak ada explicit break)

```csharp
public byte ExplicitBreak { get; set; }
```

### Contoh

Contoh:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.ExplicitBreak = 1;
```

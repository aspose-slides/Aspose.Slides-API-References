## MathBox.AlignmentPoint properti

Ketika bernilai true, emulator operator ini berfungsi sebagai titik penyelarasan; yaitu, titik penyelarasan yang ditentukan dalam persamaan lain dapat diselaraskan dengannya. Default: false

```csharp
public bool AlignmentPoint { get; set; }
```

### Contoh

Contoh:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.AlignmentPoint = true;
```

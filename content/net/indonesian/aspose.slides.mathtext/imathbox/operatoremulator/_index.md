## IMathBox.OperatorEmulator properti

Emulator Operator. Ketika true, kotak dan isinya berperilaku sebagai satu operator tunggal dan mewarisi properti dari sebuah operator. Ini berarti, misalnya, bahwa karakter dapat berfungsi sebagai titik untuk pemutusan baris dan dapat disejajarkan dengan operator lain. Emulator Operator sering digunakan ketika satu atau lebih glyph digabungkan menjadi sebuah operator, seperti '=='. Nilai default: false

```csharp
public bool OperatorEmulator { get; set; }
```

### Contoh

Contoh:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.OperatorEmulator = true;
```

## IMathBox.AlignmentPoint properti

Ketika bernilai true, emulator operator ini berfungsi sebagai titik kesejajaran; yaitu, titik kesejajaran yang ditetapkan dalam persamaan lain dapat disejajarkan dengannya. Default: false

```csharp
public bool AlignmentPoint { get; set; }
```

### Contoh

Contoh:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.AlignmentPoint = true;
```

## IMathBox antarmuka

Menentukan pengemasan (boxing) logis dari elemen matematika. Misalnya, objek yang dikemas dapat berfungsi sebagai emulator operator dengan atau tanpa titik penyelarasan, berfungsi sebagai titik pemutusan baris, atau dikelompokkan sehingga tidak mengizinkan pemutusan baris di dalamnya. Sebagai contoh, operator "==" harus dikemas untuk mencegah pemutusan baris.

```csharp
public interface IMathBox : IMathElement
```

Emulator Operator sering digunakan ketika satu atau lebih glyph digabungkan menjadi sebuah operator, seperti '=='. Nilai default: false |

## IMathBox.NoBreak properti

Tidak ada pemutusan. Properti ini menentukan properti "unbreakable" pada kotak objek. Jika bernilai true, tidak ada pemutusan baris yang dapat terjadi di dalam kotak. Hal ini dapat penting untuk emulator operator yang terdiri dari lebih dari satu operator biner. Jika elemen ini tidak ditentukan, pemutusan dapat terjadi di dalam kotak. Bawaan: true

```csharp
public bool NoBreak { get; set; }
```

### Contoh

Contoh:

```csharp
[C#]
IMathBox box = new MathematicalText("**********").ToBox();
box.NoBreak = false;
```

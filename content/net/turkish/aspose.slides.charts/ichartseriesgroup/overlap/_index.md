## IChartSeriesGroup.Overlap özelliği

Belirtilen yüzde olarak (-%100'den %100'e) 2-B grafiklerde çubukların ve sütunların ne kadar üst üste geleceğini tanımlar.  
- -100%: En büyük boşluk (çubuklar tamamen ayrılmıştır).  
- 0%: Çubuklar yan yana yerleştirilir, üst üste gelme veya boşluk yoktur.  
- 100%: En büyük üst üste gelme (çubuklar birbirini tamamen kaplar).  
Bu özellik okuma/yazma SByte.

```csharp
public sbyte Overlap { get; set; }
```

### İstisnalar

| istisna | koşul |
| --- | --- |
| ArgumentOutOfRangeException | Değer geçerli -100 ile 100 aralığının dışına ayarlandığında fırlatılır. |

### Örnekler

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // Üst üste gelmeyi %55 olarak ayarla

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

## IChartData.Categories özelliği

Birincil kategorileri alır (veya [`UseSecondaryCategories`](../usesecondarycategories) özelliği false ise birincil ve ikincil kategorileri alır). Salt okunur [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection Categories { get; }
```

### Açıklamalar

Eğer [`UseSecondaryCategories`](../usesecondarycategories) özelliği false ise [`SecondaryCategories`](../secondarycategories) özelliği null döndürür ve bu `Categories` özelliğindeki veriler birincil ve ikincil seriler için kullanılır. Eğer [`UseSecondaryCategories`](../usesecondarycategories) özelliği true ise [`SecondaryCategories`](../secondarycategories) özelliğindeki veri ikincil seriler için, bu `Categories` özelliğindeki veri ise birincil seriler için kullanılır.

### Örnekler

Örnek. Hangi kategoriler serilerle ilişkilidir - ChartData.Categories veya ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // ilgili kategoriler series.Chart.ChartData.SecondaryCategories
}
else
{
    // ilgili kategoriler series.Chart.ChartData.Categories
}
```

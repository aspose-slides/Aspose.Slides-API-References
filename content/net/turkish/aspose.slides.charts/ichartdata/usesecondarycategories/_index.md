## IChartData.UseSecondaryCategories özelliği

Yanlış ise [`SecondaryCategories`](../secondarycategories) özelliği null döndürür ve [`Categories`](../categories) özelliğindeki veri birincil ve ikincil seriler için kullanılır. Doğru ise [`SecondaryCategories`](../secondarycategories) özelliğindeki veri ikincil seriler için, [`Categories`](../categories) özelliğindeki veri ise birincil seriler için kullanılır. Okuma/yazma Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

### Örnekler

Örnek. Hangi kategoriler serilere ilişkilidir - ChartData.Categories veya ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // ilgili kategoriler series.Chart.ChartData.SecondaryCategories
}
else
{
    // ilgili kategoriler series.Chart.ChartData.Categories
}
```

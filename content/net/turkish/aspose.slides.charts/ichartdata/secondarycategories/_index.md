## IChartData.SecondaryCategories özelliği

Eğer [`UseSecondaryCategories`](../usesecondarycategories) özelliği doğru ise ikincil kategorileri alır. Salt okunur [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### Açıklamalar

Eğer [`UseSecondaryCategories`](../usesecondarycategories) özelliği yanlış ise bu `SecondaryCategories` özelliği null döner ve [`Categories`](../categories) özelliğindeki veri hem birincil hem ikincil seriler için kullanılır. Eğer [`UseSecondaryCategories`](../usesecondarycategories) özelliği doğru ise bu `SecondaryCategories` özelliğindeki veri ikincil seriler için, [`Categories`](../categories) özelliğindeki veri ise birincil seriler için kullanılır.

### Örnekler

Örnek. Serilere hangi kategoriler ilişkilidir - ChartData.Categories veya ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // ilgili kategoriler series.Chart.ChartData.SecondaryCategories
}
else
{
    // ilgili kategoriler series.Chart.ChartData.Categories
}
```

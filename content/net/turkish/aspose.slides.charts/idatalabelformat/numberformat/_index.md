## IDataLabelFormat.NumberFormat özelliği

DataLabels nesnesi için biçim dizesini temsil eder. Okunur/Yazılır String.

```csharp
public string NumberFormat { get; set; }
```

### Açıklamalar

Bu DataLabelFormat nesnesinin üst öğesi bir DataLabelCollection veri etiketleri koleksiyonuysa, bu özellik yeni veri etiketleri için NumberFormat özelliğinin varsayılan değerini alır veya ayarlar. Bu özellik bir değerle ayarlandığında, aynı değer DataLabelCollection koleksiyonundaki tüm veri etiketleri için NumberFormat özelliğine de ayarlanır (ör. "DataLabels.DefaultDataLabelFormat.NumberFormat = val;" tüm DataLabels[i].NumberFormat değerinin val olmasına neden olur).

### Örnekler

```csharp
[C#]
series.Labels.DefaultDataLabelFormat.ShowValue = true;
series.Labels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = false;
series.Labels.DefaultDataLabelFormat.NumberFormat = "0.0%";
```

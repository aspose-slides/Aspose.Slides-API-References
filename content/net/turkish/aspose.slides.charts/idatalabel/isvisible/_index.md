## IDataLabel.IsVisible özelliği

False, veri etiketinin görünür olmadığını (ve bu yüzden tüm Show*-bayraklarının (ShowValue, ...) false olduğunu) gösterir. Salt Okunur Boolean.

```csharp
public bool IsVisible { get; }
```

### Açıklamalar

Veri etiketi görünürse Hide() yöntemiyle gizleyebilirsiniz. Ancak veri etiketi görünür değilse (IsVisible false ise) Show*-bayraklarını (ShowValue, ...) true duruma getirerek veri etiketini görünür hâle getirebilirsiniz.

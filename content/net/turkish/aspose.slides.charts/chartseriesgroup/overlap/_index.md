## ChartSeriesGroup.Overlap özelliği

2-D grafiklerde çubukların ve sütunların ne kadar üst üste geleceğini yüzde olarak belirtir (-100% ile 100% arasında). - -100%: Azami boşluk (çubuklar tamamen ayrılmıştır). - 0%: Çubuklar üst üste gelmeden yan yana konumlandırılır. - 100%: Azami üst üste geçme (çubuklar tamamen birbirinin üzerine gelir). Bu özellik okuma/yazma SByte tipindedir.

```csharp
public sbyte Overlap { get; set; }
```

### İstisnalar

| istisna | koşul |
| --- | --- |
| ArgumentOutOfRangeException | Değer -100 ile 100 arasındaki geçerli aralığın dışına ayarlandığında fırlatılır. |

### Örnekler

Aşağıdaki örnek, bir grafik serisi grubu için üst üste geçmeyi ayarlamayı ve oluşan grafiği bir formda görüntülemeyi gösterir:

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // Üst üste geçmeyi %55 olarak ayarla

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

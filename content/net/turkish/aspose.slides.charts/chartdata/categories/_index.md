## ChartData.Categories özelliği

Birincil kategorileri alır (veya [`UseSecondaryCategories`](../usesecondarycategories) özelliği false ise hem birincil hem de ikincil kategorileri). Yalnızca okuma [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection Categories { get; }
```

### Açıklamalar

Eğer [`UseSecondaryCategories`](../usesecondarycategories) özelliği false ise [`SecondaryCategories`](../secondarycategories) özelliği null döndürür ve bu `Categories` özelliğindeki veri hem birincil hem de ikincil seriler için kullanılır. Eğer [`UseSecondaryCategories`](../usesecondarycategories) özelliği true ise [`SecondaryCategories`](../secondarycategories) özelliğindeki veri ikincil seriler için, bu `Categories` özelliğindeki veri ise birincil seriler için kullanılır.

### Örnekler

Örnek. Serilere hangi kategorilerin bağlı olduğunu - ChartData.Categories veya ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // ilgili kategoriler series.Chart.ChartData.SecondaryCategories
}
else
{
    // ilgili kategoriler series.Chart.ChartData.Categories
}
```

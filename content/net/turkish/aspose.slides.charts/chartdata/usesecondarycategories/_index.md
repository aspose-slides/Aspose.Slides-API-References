## ChartData.UseSecondaryCategories özellik

Yanlış ise [`SecondaryCategories`](../secondarycategories) özelliği null döndürür ve [`Categories`](../categories) özelliğindeki veri birincil ve ikincil seriler için birlikte kullanılır. Doğru ise [`SecondaryCategories`](../secondarycategories) özelliğindeki veri ikincil seriler için, [`Categories`](../categories) özelliğindeki veri ise birincil seriler için kullanılır. Okunur/Yazılabilir Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

### Örnekler

Örnek. Seriyle ilişkili kategoriler nelerdir - ChartData.Categories veya ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // ilgili kategoriler series.Chart.ChartData.SecondaryCategories
}
else
{
    // ilgili kategoriler series.Chart.ChartData.Categories
}
```

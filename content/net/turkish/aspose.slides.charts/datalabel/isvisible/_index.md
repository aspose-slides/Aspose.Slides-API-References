## DataLabel.IsVisible özelliği

False, veri etiketinin görünür olmadığı (ve bu yüzden tüm Show*-flags (ShowValue, ...) false olduğu) anlamına gelir. Yalnızca okuma Boolean.

```csharp
public bool IsVisible { get; }
```

### Açıklamalar

Veri etiketi görünürse Hide() yöntemiyle gizleyebilirsiniz. Ancak veri etiketi görünür değilse (IsVisible false) Show*-flags (ShowValue, ...) true durumuna ayarlanarak veri etiketi görünür hâle getirilebilir.

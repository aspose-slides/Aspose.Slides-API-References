## WebDocument.Output özelliği

Belgenin çıktı öğelerinin koleksiyonunu döndürür. Salt okunur `Output`.

```csharp
public Output Output { get; }
```

### Örnekler

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    var options = new WebDocumentOptions
    {
        TemplateEngine = new RazorTemplateEngine(),
        OutputSaver = new FileOutputSaver(),
        EmbedImages = false
    };
    
    WebDocument document = new WebDocument(options);

    // "index.html" dosyasını çıkış dosyalarına ekle, onu oluşturmak için "index" şablonunu kullan ve pres değişkenini model olarak geçir
    document.Output.Add("index.html", "index", pres);

    // ... belgenin diğer seçeneklerini ayarla ve ardından belgeyi kaydet
    document.Save();
}
```

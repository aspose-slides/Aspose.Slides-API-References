## WebDocument.Global özellik

Belgenin global depolamasını döndürür. Yalnızca okuma [`Storage`](../../storage).

```csharp
public Storage Global { get; }
```

### Örnekler

Bu `Global` özelliğini ([`Storage`](../../storage) arayüzünün uygulaması) kullanarak bir özellik şablonda daha sonra kullanılmak üzere yerleştirilebilir:

```csharp
[C#]
var options = new WebDocumentOptions
{
    TemplateEngine = new RazorTemplateEngine(),
    OutputSaver = new FileOutputSaver(),
    EmbedImages = false
};

WebDocument document = new WebDocument(options);

// "slideMargin" özelliğini şablonlarda kullanmak için ekle
document.Global.Put("slideMargin", 10);

// ... belgenin diğer seçeneklerini ayarla ve ardından belgeyi kaydet
document.Save();
```

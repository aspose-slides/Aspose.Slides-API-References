## IInkBrush.InkEffect özelliği

Mürekkep efekti türünü alır (ör. Galaxy, Gold, Silver) ve mürekkep darbesinin görsel stilini tanımlar. Değer, fırça özelliği "inkEffects" üzerinden ayrıştırılır. Tanınan bir efekt belirtilmezse, NotDefined döndürülür.

```csharp
public InkEffectType InkEffect { get; }
```

### Örnekler

Example:

```csharp
[C#]
using (var pres = new Presentation("Presentation.pptx"))
{
    Ink ink = pres.Slides[0].Shapes[0] as Ink;
    IInkBrush brush = ink.Traces[0].Brush;
    Console.WriteLine("InkEffects = {0}", brush.InkEffect);
}
```

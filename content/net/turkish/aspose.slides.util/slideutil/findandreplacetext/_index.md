## SlideUtil.FindAndReplaceText yöntemi

Sunumdaki metni verilen biçimle bulur ve değiştirir

```csharp
public static void FindAndReplaceText(IPresentation presentation, bool withMasters, string find, 
    string replace, PortionFormat format = null)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| presentation | IPresentation | Taranmış sunum. |
| withMasters | Boolean | Ana slaytların taranıp taranmayacağını belirler. |
| find | String | Bulunacak dize değeri. |
| replace | String | Değiştirilecek dize değeri. |
| format | PortionFormat | Metin bölümü değiştirmek için biçim. Null ise bulunan dizenin ilk karakterinin biçimi kullanılır. |

### Örnekler

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PortionFormat format = new PortionFormat
    {
        FontHeight = 24f,
        FontItalic = NullableBool.True,
        FillFormat =
        {
            FillType = FillType.Solid,
            SolidFillColor =
            {
                Color = Color.Red
            }
        }
    };
    Aspose.Slides.Util.SlideUtil.FindAndReplaceText(pres, true, "[this block] ", "my text ", format);
    pres.Save("replaced", SaveFormat.Pptx);
}
```

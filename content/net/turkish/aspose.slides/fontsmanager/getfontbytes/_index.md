## FontsManager.GetFontBytes yöntemi

Belirtilen bir yazı tipi stili ve yazı tipi verisi için yazı tipi verisini temsil eden bayt dizisini alır.

```csharp
public byte[] GetFontBytes(IFontData fontData, FontStyleType fontStyle)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| fontData | IFontData | Yazı tipi hakkında bilgi içeren font data nesnesi [`IFontData`](../../ifontdata). |
| fontStyle | FontStyleType | Verinin alınacağı yazı tipinin stili [`FontStyleType`](../../fontstyletype). |

### Dönüş Değeri

Belirtilen yazı tipi stili için yazı tipi verisini içeren bir bayt dizisi. Yazı tipi verisi veya stil bulunamazsa null döndürür.

### Örnekler

```csharp
[C#]
using (Presentation pres = new Presentation ("Presentation.pptx"))
{
    // Sunumda kullanılan tüm yazı tiplerini al
    IFontData[] fonts = pres.FontsManager.GetFonts();

    // Sunumdaki ilk yazı tipinin regular stilini temsil eden bayt dizisini al
    byte[] bytes = pres.FontsManager.GetFontBytes(fonts[0], FontStyleType.Regular);
}
```

## IGeometryShape.SetGeometryPaths metodu

Şekil geometrisini [`IGeometryPath`](../../igeometrypath) dizisinden günceller. Koordinatlar, şeklin sol üst köşesine göre göreceli olmalıdır. Şeklin türünü ([`ShapeType`](../shapetype)) Custom'a değiştirir.

```csharp
public void SetGeometryPaths(IGeometryPath[] geometryPaths)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| geometryPaths | IGeometryPath[] | Geometri yolları dizisi |

### İstisnalar

| istisna | koşul |
| --- | --- |
| ArgumentException | Yol bulunamadı |
| ArgumentException | Boş yol |

### Örnekler

Örnek:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    GeometryShape shape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 200, 100) as GeometryShape;

    GeometryPath geometryPath = shape.GetGeometryPaths()[0];

    geometryPath.LineTo(100, 50, 1);
    geometryPath.LineTo(100, 50, 4);

    shape.SetGeometryPath(geometryPath);

    pres.Save("output.pptx", SaveFormat.Pptx);
}
```

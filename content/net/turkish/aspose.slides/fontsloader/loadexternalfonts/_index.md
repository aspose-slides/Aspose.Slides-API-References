## FontsLoader.LoadExternalFonts yöntemi

Yazı tiplerini aramak için ek klasörler ekler.

```csharp
public static void LoadExternalFonts(string[] directories)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| directories | String[] | Ek yazı tiplerini okumak için dizinler. |

### Örnekler

Aşağıdaki örnekler, .TTF dosyalarından özel yazı tiplerini nasıl yükleyeceğinizi gösterir.

```csharp
[C#]
// Belgeler dizininin yolu.
string dataDir = "C:\\";
// yazı tiplerini aramak için klasörler
String[] folders = new String[] { dataDir };
// Özel yazı tipi dizini yazı tiplerini yükle
FontsLoader.LoadExternalFonts(folders);
// Bazı işlemler yap ve sunum/slayt oluşturmayı gerçekleştir
using (Presentation presentation = new Presentation(dataDir + "DefaultFonts.pptx"))
{
    presentation.Save(dataDir + "NewFonts_out.pptx", SaveFormat.Pptx);
	// Yazı tipi önbelleğini temizle
	FontsLoader.ClearCache();
}
```

## IViewProperties.GridSpacing özelliği

Sunum belgesinin altında bulunan ızgara için kullanılacak ızgara aralığını döndürür veya ayarlar, puan cinsinden. Okunur/Yazılır Single.

```csharp
public float GridSpacing { get; set; }
```

### Açıklamalar

Izgara aralığı değeri pozitif bir sayı olmalıdır. Tipik değer aralığı 1 mm (2.8349607 puan) ile 2 inç (144 puan) arasındadır.

### Örnekler

Aşağıdaki örnek kod, PowerPoint sunumunda ızgara aralığını nasıl değiştireceğinizi gösterir.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

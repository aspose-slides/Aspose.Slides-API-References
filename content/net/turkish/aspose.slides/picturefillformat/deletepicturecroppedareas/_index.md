## PictureFillFormat.DeletePictureCroppedAreas method

Doldurulan Resmin kırpılmış alanlarını siler.

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### Return Value

Kırpılmış görüntü veya kırpma gerekmiyorsa orijinal görüntü.

### Remarks

Bu yöntem, WMF/EMF metafilelerini kırparak raster PNG görüntüsüne dönüştürür.

### Examples

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // PictureFrame'i alır

    // PictureFrame görüntüsünün kırpılmış alanlarını siler
```

## PictureFillFormat.TileOffsetX özelliği

Dokuun şeklin orijinalinden yatay offsetini nokta cinsinden döndürür veya ayarlar. Pozitif bir değer dokuyu sağa, negatif bir değer ise sola kaydırır. Okuma/yazma Single.

```csharp
public float TileOffsetX { get; set; }
```

### Örnekler

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Şeklin resim dolgu formatını alır
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Resim dolgu modunu Döşeme olarak ayarlar
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Dokuun yatay offsetini 20 puana ayarlar
    pictureFillFormat.TileOffsetX = 20f;
}
```

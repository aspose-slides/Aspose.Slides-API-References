## CompressImage(bool, PicturesCompression) {#compressimage}

Görüntüyü, şekil boyutuna ve belirtilen çözünürlüğe göre boyutunu küçülterek sıkıştırır. İsteğe bağlı olarak, kırpılmış alanları da siler.

```csharp
public bool CompressImage(bool deleteCroppedAreasOfImage, PicturesCompression resolution)
```

| Parameter | Type | Description |
| --- | --- | --- |
| deleteCroppedAreasOfImage | Boolean | Doğru ise, yöntem görüntünün kırpılmış alanlarını kaldıracak ve bu da boyutunun daha da küçülmesine neden olabilir. |
| resolution | PicturesCompression | Sıkıştırma için hedef çözünürlük, [`PicturesCompression`](../../../aspose.slides.export/picturescompression) enum değerı olarak belirtilir. |

### Return Value

Görüntünün başarıyla sıkıştırılıp sıkıştırılmadığını gösteren bir Boolean. Görüntü yeniden boyutlandırıldıysa veya kırpıldıysa `true`, aksi takdirde `false` döner.

### Exceptions

| exception | condition |
| --- | --- |
| ArgumentException | Çözünürlük geçerli bir değer olmadığında fırlatılır. |

### Remarks

Bu yöntem, PowerPoint'in "Picture Format -> Compress Pictures" özelliğine benzer şekilde görüntünün boyutunu ve çözünürlüğünü değiştirir.

### Examples

Aşağıdaki örnek, `CompressImage` yöntemini kullanarak bir sunumdaki görüntünün boyutunu hedef çözünürlüğü ayarlayarak ve kırpılmış alanları kaldırarak nasıl küçülteceğinizi gösterir:

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;
    // Görüntüyü 150 DPI (Web çözünürlüğü) hedef çözünürlükle sıkıştır ve kırpılmış alanları kaldır
    bool result = picFrame.PictureFormat.CompressImage(true, PicturesCompression.Dpi150);
}
```

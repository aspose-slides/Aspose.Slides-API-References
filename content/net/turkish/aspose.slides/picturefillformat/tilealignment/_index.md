## PictureFillFormat.TileAlignment özelliği

Doku şekil içinde nasıl hizalanacağını döndürür veya ayarlar. Bu ayar, doku deseninin başlangıç noktasını ve şekil boyunca nasıl tekrarlandığını kontrol eder. Okuma/Yazma [`RectangleAlignment`](../../rectanglealignment).

```csharp
public RectangleAlignment TileAlignment { get; set; }
```

### Açıklamalar

Varsayılan TopLeft'tir.

### Örnekler

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Şeklin resim dolgu biçimini alır
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Resim dolgu modunu Döşeme olarak ayarlar
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Döşemenin hizalamasını sağ alt köşeye ayarlar
    pictureFillFormat.TileAlignment = RectangleAlignment.BottomRight;
}
```

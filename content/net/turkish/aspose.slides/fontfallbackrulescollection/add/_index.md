## FontFallBackRulesCollection.Add metodu

Belirtilen FallBack kuralını koleksiyonun sonuna ekler.

```csharp
public void Add(IFontFallBackRule sourceRule)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| sourceRule | IFontFallBackRule | Eklemek için belirtilen kural |

### Örnekler

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    //FontsManager'dan boş ya da önceden başlatılmış kurallar koleksiyonunu alıyor
    IFontFallBackRulesCollection rulesList = pres.FontsManager.FontFallBackRulesCollection;

    //Yeni kuralı koleksiyona ekliyor
```

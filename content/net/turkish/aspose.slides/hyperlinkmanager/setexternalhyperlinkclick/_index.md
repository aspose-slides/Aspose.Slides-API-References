## HyperlinkManager.SetExternalHyperlinkClick metodu

Tıklandığında dış bağlantı ayarlar.

```csharp
public IHyperlink SetExternalHyperlinkClick(string url)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| url | String | Hipermetin Bağlantısı URL'si. |

### Örnekler

Aşağıdaki örnek kod, Hipermetin Bağlantılı bir Metin Kutusu eklemeyi gösterir.

```csharp
[C#]
// PPTX'i temsil eden bir Presentation sınıfını örnekler
using(Presentation pptxPresentation = new Presentation()) {
  // Sunumdaki ilk slaytı alır
  ISlide slide = pptxPresentation.Slides[0];
  // Tipi Rectangle olarak ayarlanmış bir AutoShape nesnesi ekler
  IShape pptxShape = slide.Shapes.AddAutoShape(ShapeType.Rectangle, 150, 150, 150, 50);
  // Şekli AutoShape'e dönüştürür
  IAutoShape pptxAutoShape = (IAutoShape) pptxShape;
  // AutoShape ile ilişkili ITextFrame özelliğine erişir
  pptxAutoShape.AddTextFrame("");
  ITextFrame textFrame = pptxAutoShape.TextFrame;
  // Çerçeveye bazı metinler ekler
  textFrame.Paragraphs[0].Portions[0].Text = "Aspose.Slides";
  // Bölüm metni için Hipermetin Bağlantısını ayarlar
  IHyperlinkManager HypMan = textFrame.Paragraphs[0].Portions[0].PortionFormat.HyperlinkManager;
  HypMan.SetExternalHyperlinkClick("http://www.aspose.com");
  // PPTX Sunumunu kaydeder
  pptxPresentation.Save("hLinkPPTX_out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
}
```

### Örnekler

Aşağıdaki kod örneği, bir PowerPoint sunumundaki ilk slaytı bir metafile'a dönüştürmenin nasıl yapılacağını gösterir.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.emf"))
    {
        // İlk slaytı bir metafile olarak kaydeder
        pres.Slides[0].WriteAsEmf(fileStream);
    }
}
```

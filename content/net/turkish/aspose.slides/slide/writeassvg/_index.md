### Örnekler

Aşağıdaki kod örneği, bir PowerPoint sunumunun ilk slaytını bir SVG dosyasına dönüştürmeyi gösterir.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.svg"))
    {
        // İlk slaytı bir SVG dosyası olarak kaydeder
        pres.Slides[0].WriteAsSvg(fileStream);
    }
}
```

## WriteAsSvg(Stream, ISVGOptions) {#writeassvg_1}

Slayt içeriğini SVG dosyası olarak kaydeder.

```csharp
public void WriteAsSvg(Stream stream, ISVGOptions svgOptions)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| stream | Stream | Hedef akış |
| svgOptions | ISVGOptions | SVG oluşturma seçenekleri |

### Örnekler

Aşağıdaki kod örneği, bir PowerPoint sunumunun ilk slaytını seçeneklerle bir SVG dosyasına dönüştürmeyi gösterir.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.svg"))
    {
        var options = new SVGOptions() { VectorizeText = true };
        // İlk slaytı bir SVG dosyası olarak kaydeder
        pres.Slides[0].WriteAsSvg(fileStream, options);
    }
}
```

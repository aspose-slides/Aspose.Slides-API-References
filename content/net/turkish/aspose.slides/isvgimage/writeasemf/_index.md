## ISvgImage.WriteAsEmf metodu

SVG görüntüyü bir EMF dosyası olarak kaydeder.

```csharp
public void WriteAsEmf(Stream stream)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| stream | Stream | Hedef akış |

### İstisnalar

| istisna | koşul |
| --- | --- |
| ArgumentNullException | Hedef akış **null** |

### Örnekler

Aşağıdaki örnek, SVG görüntüyü bir metafile kaydetmenin nasıl yapılacağını gösterir.

```csharp
[C#]
// Yeni SVG görüntüsünü oluşturur
ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));

// SVG görüntüyü bir metafile olarak kaydeder
using (var fileStream = System.IO.File.OpenWrite("SvgAsEmf.emf"))
    svgImage.WriteAsEmf(fileStream);
```

Bu örnek, SVG görüntüyü bir metafile olarak sunum görüntü koleksiyonuna eklemenin nasıl yapılacağını gösterir.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // Yeni SVG görüntüsünü oluşturur
    ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));
    using (var memStream = new MemoryStream())
    {
        // SVG görüntüyü bir metafile olarak kaydeder
        svgImage.WriteAsEmf(memStream);
        // Metafile'i görüntü koleksiyonuna ekler
        pres.Images.AddImage(memStream.ToArray());
    }
}
```

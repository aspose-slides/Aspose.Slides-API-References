## IPresentationInfo.CheckPassword method

Açık şifre ile korunan bir sunum için şifrenin doğru olup olmadığını kontrol eder.

```csharp
public bool CheckPassword(string password)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| password | String | Kontrol edilecek şifre. |

### Dönüş Değeri

Sunum açık şifre ile korunuyorsa ve şifre doğruysa True, aksi takdirde false döner.

### Açıklamalar

Şifre null veya boş olduğunda bu yöntem false döner.

### Örnekler

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo("pres.pptx");
bool isPasswordCorrect = info.CheckPassword("my_password");
```

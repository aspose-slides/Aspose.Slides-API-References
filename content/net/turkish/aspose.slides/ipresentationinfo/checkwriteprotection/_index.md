## IPresentationInfo.CheckWriteProtection yöntemi

Yazma korumalı bir sunum için değiştirme parolasının doğru olup olmadığını kontrol eder.

```csharp
public bool CheckWriteProtection(string password)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| password | String | Kontrol edilecek parola. |

### Dönüş Değeri

Sunum yazma korumalıysa ve parola doğruysa **True** döner. Aksi takdirde **False** döner.

### İstisnalar

| İstisna | Şart |
| --- | --- |
| InvalidOperationException |  |

### Açıklamalar

1. Bu yöntemi çağırmadan önce [`IsWriteProtected`](../iswriteprotected) özelliğini kontrol etmelisiniz.  
2. Parola **null** ya da boş olduğunda bu yöntem **false** döner.

### Örnekler

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
if (info.IsWriteProtected == NullableBool.True)
{
    bool isWriteProtectedByPassword = info.CheckWriteProtection("my_password");
}
```

## IPresentationInfo.UpdateDocumentProperties yöntemi

Bağlı sunumun özelliklerini günceller.

```csharp
public void UpdateDocumentProperties(IDocumentProperties documentProperties)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| documentProperties | IDocumentProperties | Belge özellikleri [`IDocumentProperties`](../../idocumentproperties) |

### Örnekler

Bu örnek, `UpdateDocumentProperties` yöntemini çağırarak, [`ReadDocumentProperties`](../readdocumentproperties) yönteminin çağrısı tarafından döndürülen belge özelliklerini nasıl güncelleyeceğinizi gösterir.

```csharp
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo("pres.pptx");
IDocumentProperties props = info.ReadDocumentProperties();
props.Subject = "New subject";
props.LastSavedTime = DateTime.UtcNow;
info.UpdateDocumentProperties(props);
info.WriteBindedPresentation("new_pres.pptx");
```

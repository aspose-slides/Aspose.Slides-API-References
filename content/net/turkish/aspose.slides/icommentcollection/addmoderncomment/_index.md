## ICommentCollection.AddModernComment metodu

Yeni modern yorumu koleksiyonun sonuna ekleyin.

```csharp
public IModernComment AddModernComment(string text, ISlide slide, IShape shape, PointF position, 
    DateTime creationTime)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| text | String | Yeni modern yorumun düz metni. |
| slide | ISlide | Yeni modern yorumun ekleneceği sunumdaki slayt. |
| shape | IShape | Yeni modern yorumun ilişkili olduğu slayttaki şekil. |
| position | PointF | Yeni modern yorumun ekleneceği slayttaki konum. |
| creationTime | DateTime | Modern yorumun oluşturulma zamanı. |

### Dönüş Değeri

Eklenen modern yorum.

### Örnekler

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ICommentAuthor newAuthor = pres.CommentAuthors.AddAuthor("Some Author", "SA");
    newAuthor.Comments.AddModernComment("This is modern comment", pres.Slides[0], null, new PointF(100, 100), DateTime.Now);

    pres.Save(outPptxFileName, SaveFormat.Pptx);
}
```

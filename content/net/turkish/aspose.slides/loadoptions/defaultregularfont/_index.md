## LoadOptions.DefaultRegularFont özelliği

Kaynak yazı tipi bulunamadığında kullanılan Regular yazı tipini döndürür veya ayarlar. Okuma/yazma String.

```csharp
public string DefaultRegularFont { get; set; }
```

### Örnekler

Aşağıdaki örnek, PowerPoint Sunumu oluştururken varsayılan yazı tiplerini nasıl ayarlayacağınızı gösterir.

```csharp
[C#]
// Yükleme seçeneklerini kullanarak varsayılan normal ve Asya yazı tiplerini tanımla
LoadOptions loadOptions = new LoadOptions(LoadFormat.Auto);
loadOptions.DefaultRegularFont = "Wingdings";
loadOptions.DefaultAsianFont = "Wingdings";
// Sunumu yükle
using (Presentation pptx = new Presentation("DefaultFonts.pptx", loadOptions))
{
    // Slayt küçük resmi oluştur
    pptx.Slides[0].GetThumbnail(1, 1).Save("output_out.png", ImageFormat.Png);
    // PDF oluştur
    pptx.Save("output_out.pdf", SaveFormat.Pdf);
    // XPS oluştur
    pptx.Save("output_out.xps", SaveFormat.Xps);
}
```

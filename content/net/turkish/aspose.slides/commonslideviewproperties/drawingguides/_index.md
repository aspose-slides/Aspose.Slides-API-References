## CommonSlideViewProperties.DrawingGuides özelliği

Çizim kılavuzlarının koleksiyonunu döndürür. Salt okunur [`IDrawingGuidesCollection`](../../idrawingguidescollection)

```csharp
public IDrawingGuidesCollection DrawingGuides { get; }
```

### Örnekler

Aşağıdaki örnek kod, yeni çizim kılavuzlarının bir PowerPoint sunumuna nasıl ekleneceğini gösterir.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    var slideSize = pres.SlideSize.Size;
    
    IDrawingGuidesCollection guides = pres.ViewProperties.SlideViewProperties.DrawingGuides;
    // Yeni dikey çizim kılavuzunu slayt merkezinin sağına ekleme
    guides.Add(Orientation.Vertical, slideSize.Width / 2 + 12.5f);
    // Yeni yatay çizim kılavuzunu slayt merkezinin altına ekleme
    guides.Add(Orientation.Horizontal, slideSize.Height / 2 + 12.5f);
    
    pres.Save("DrawingGuides_out.pptx", SaveFormat.Pptx);
}
```

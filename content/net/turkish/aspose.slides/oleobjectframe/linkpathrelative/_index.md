## OleObjectFrame.LinkPathRelative özelliği

Bağlantılı bir dosya mevcutsa, ilgili dosyanın göreli yolunu döndürür; aksi takdirde boş bir dize döndürür. Salt okunur String.

```csharp
public string LinkPathRelative { get; }
```

### Açıklama

Ppt sunumlarında, bazı Ole nesne bağlantılarının göreli bir gösterimi olabilir.

### Örnekler

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.ppt"))
{
    IOleObjectFrame oleFrame = presentation.Slides[0].Shapes[0] as IOleObjectFrame;

    if (oleFrame != null)
    {
        Console.WriteLine("The relative path: " + oleFrame.LinkPathRelative);
    } 
}
```

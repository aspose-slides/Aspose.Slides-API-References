## Presentation.SlideSize özelliği

Slayt boyutu nesnesini döndürür. Sadece okuma [`ISlideSize`](../../islidesize).

```csharp
public ISlideSize SlideSize { get; }
```

### Örnekler

Aşağıdaki örnek, bir PowerPoint Sunumu'nda slayt boyutunu nasıl değiştireceğinizi gösterir.

```csharp
[C#]
using (Presentation pres = new Presentation("pres-4x3-aspect-ratio.pptx"))
{
    pres.SlideSize.SetSize(SlideSizeType.OnScreen16x9, SlideSizeScaleType.DoNotScale);
    pres.Save("pres-4x3-aspect-ratio.pptx", SaveFormat.Pptx);
}
```

Aşağıdaki örnek, bir PowerPoint Sunumu için içerik ölçeklemesiyle slayt boyutunu nasıl ayarlayacağınızı gösterir.

```csharp
[C#]
// Bir sunum dosyasını temsil eden Presentation nesnesi oluşturur
using(Presentation presentation = new Presentation("AccessSlides.pptx")) {
  using(Presentation auxPresentation = new Presentation()) {
    ISlide slide = presentation.Slides[0];
    // Oluşturulan sunumların slayt boyutunu kaynağın boyutuna ayarlar
    presentation.SlideSize.SetSize(540, 720, SlideSizeScaleType.EnsureFit); // Method SetSize, slayt boyutunu içerik ölçeklemesiyle ayarlamak ve sığdırmak için kullanılır
    presentation.SlideSize.SetSize(SlideSizeType.A4Paper, SlideSizeScaleType.Maximize); // Method SetSize, slayt boyutunu içeriğin boyutunu maksimize ederek ayarlamak için kullanılır
    // Sunumu diske kaydeder
    auxPresentation.Save("Set_Size&Type_out.pptx", SaveFormat.Pptx);
  }
}
```

Aşağıdaki örnek, bir PowerPoint Sunumu'nda özel slayt boyutları nasıl belirtileceğini gösterir.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    pres.SlideSize.SetSize(780, 540, SlideSizeScaleType.DoNotScale); // A4 kağıt boyutu
    pres.Save("pres-a4-slide-size.pptx", SaveFormat.Pptx);
}
```

## Presentation.MasterTheme özelliği

Ana temayı döndürür. Yalnızca okunur [`IMasterTheme`](../../../aspose.slides.theme/imastertheme).

```csharp
public IMasterTheme MasterTheme { get; }
```

### Örnekler

Aşağıdaki örnekler, PowerPoint Sunumu öğelerinin bölümlerini değiştirerek bir tema efektini nasıl değiştireceğinizi gösterir.

```csharp
[C#]
//Sunum dosyasını temsil eden bir sunum nesnesi oluşturur
using (Presentation pres = new Presentation("Subtle_Moderate_Intense.pptx"))
{
    pres.MasterTheme.FormatScheme.LineStyles[0].FillFormat.SolidFillColor.Color = Color.Red;
    pres.MasterTheme.FormatScheme.FillStyles[2].FillType = FillType.Solid;
    pres.MasterTheme.FormatScheme.FillStyles[2].SolidFillColor.Color = Color.ForestGreen;
    pres.MasterTheme.FormatScheme.EffectStyles[2].EffectFormat.OuterShadowEffect.Distance = 10f;
    pres.Save("Design_04_Subtle_Moderate_Intense-out.pptx", SaveFormat.Pptx);
}
```

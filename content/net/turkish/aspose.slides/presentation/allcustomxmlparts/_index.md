## Presentation.AllCustomXmlParts özelliği

Sunumda bulunan tüm özel veri bölümlerini döndürür. Salt okunur [`ICustomXmlPart`](../../icustomxmlpart)[].

```csharp
public ICustomXmlPart[] AllCustomXmlParts { get; }
```

### Örnekler

Aşağıdaki örnekler, PowerPoint Sunumundan tüm özel xml bölümlerini nasıl temizleyeceğinizi gösterir.

```csharp
[C#]
using (Presentation pres = new Presentation("PresentationWithCustomXml.pptx"))
{
	// Tüm özel XML bölümlerini yinele
    foreach (ICustomXmlPart item in pres.AllCustomXmlParts)
    {
        item.Remove();
    }
    pres.Save("out.pptx", SaveFormat.Pptx);
}
```

## Presentation.HeaderFooterManager özelliği

Gerçek HeaderFooter yöneticisini döndürür. Yalnızca okunabilir [`IPresentationHeaderFooterManager`](../../ipresentationheaderfootermanager).

```csharp
public IPresentationHeaderFooterManager HeaderFooterManager { get; }
```

### Örnekler

Aşağıdaki örnek, PowerPoint Sunumunun Slide içinde altbilgi görünürlüğünün nasıl ayarlanacağını gösterir.

```csharp
[C#]
using (Presentation presentation = new Presentation("presentation.ppt"))
{
    IBaseSlideHeaderFooterManager headerFooterManager = presentation.Slides[0].HeaderFooterManager;
	// Property IsFooterVisible, bir slayt altbilgi yer tutucusunun mevcut olmadığını göstermek için kullanılır.
    if (!headerFooterManager.IsFooterVisible)
    {
		// Method SetFooterVisibility, bir slayt altbilgi yer tutucusunu görünür kılmak için kullanılır.
        headerFooterManager.SetFooterVisibility(true);
    }
	// Property IsSlideNumberVisible, bir slayt sayfa numarası yer tutucusunun mevcut olmadığını göstermek için kullanılır.
    if (!headerFooterManager.IsSlideNumberVisible)
    {
		// Method SetSlideNumberVisibility, bir slayt sayfa numarası yer tutucusunu görünür kılmak için kullanılır.
        headerFooterManager.SetSlideNumberVisibility(true);
    }
	// Property IsDateTimeVisible, bir slayt tarih-saat yer tutucusunun mevcut olmadığını göstermek için kullanılır.
    if (!headerFooterManager.IsDateTimeVisible)
    {
		// Method SetFooterVisibility, bir slayt tarih-saat yer tutucusunu görünür kılmak için kullanılır.
        headerFooterManager.SetDateTimeVisibility(true);
    }
	// Method SetFooterText, slayt altbilgi yer tutucusuna metin ayarlamak için kullanılır.
    headerFooterManager.SetFooterText("Footer text");
	// Method SetDateTimeText, slayt tarih-saat yer tutucusuna metin ayarlamak için kullanılır.
    headerFooterManager.SetDateTimeText("Date and time text");
	presentation.Save("Presentation.ppt",SaveFormat.ppt);
}
```

	// Method SetDateTimeAndChildDateTimesVisibility, bir ana slayt ve tüm alt slayt tarih-saat yer tutucularını görünür kılmak için kullanılır.
    headerFooterManager.SetDateTimeAndChildDateTimesVisibility(true);
	// Method SetFooterAndChildFootersText, bir ana slayt ve tüm alt slayt altbilgi yer tutucularına metin ayarlamak için kullanılır.
    headerFooterManager.SetFooterAndChildFootersText("Footer text");
	// Method SetDateTimeAndChildDateTimesText, bir ana slayt ve tüm alt slayt tarih-saat yer tutucularına metin ayarlamak için kullanılır.

}

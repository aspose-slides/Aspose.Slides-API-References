## Presentation.HighlightRegex method

Belirtilen renk ile düzenli ifadeye uyan tüm eşleşmeleri vurgular.

```csharp
public void HighlightRegex(Regex regex, Color highlightColor, IFindResultCallback callback)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| regex | Regex | Vurgulanacak dizeleri almak için kullanılan düzenli ifade Regex. |
| highlightColor | Color | Metni vurgulamak için kullanılan renk. |
| callback | IFindResultCallback | Arama sonuçlarını almak için kullanılan geri çağırma nesnesi [`IFindResultCallback`](../../ifindresultcallback). |

### Örnekler

Aşağıdaki kod örneği, bir PowerPoint Sunumunda düzenli ifade kullanarak metni nasıl vurgulayacağınızı gösterir.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// 10 veya daha fazla karaktere sahip tüm kelimeleri vurgulama
	presentation.HighlightRegex(regex, Color.Blue, null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

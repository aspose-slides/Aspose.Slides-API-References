## DocumentProperties.ScaleCrop özelliği

Belge küçük resminin görüntüleme modunu gösterir. **true** değerine ayarlayarak belge küçük resminin ekrana ölçeklenmesini etkinleştirir. **false** değerine ayarlayarak belge küçük resminin kırpılmasını etkinleştirir; yalnızca ekrana uyan bölümleri gösterir. Okunur/yazılır Boolean.

public bool ScaleCrop { get; set; }

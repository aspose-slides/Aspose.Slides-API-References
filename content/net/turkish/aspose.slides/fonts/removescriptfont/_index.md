## Fonts.RemoveScriptFont yöntemi

Belirli bir script etiketine ilişkin yazı tipi ayarını temanın yazı tipi koleksiyonundan kaldırır.

```csharp
public void RemoveScriptFont(string script)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| script | String | Yazı tipi ayarının kaldırılması gereken BCP-47 script kodu. |

Bu örnek, İbranice scripti için yazı tipi eşlemesini nasıl kaldıracağınızı gösterir:

## Fonts.GetScriptFont metodu

Sunum temasındaki belirli bir script etiketine ilişkin yazı tipinin adını alır.

```csharp
public string GetScriptFont(string script)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| script | String | BCP-47 script kodu (örn., "Latn", "Cyrl", "Jpan") bir yazı sistemini tanımlamak için kullanılır. |

### Dönüş Değeri

Belirtilen script için kullanılan yazı tipinin adı, script tanımlı değilse `null` döner.

### Örnekler

Bu örnek, sunum temasında Kiril scriptine atanmış yazı tipini nasıl alacağınızı gösterir.

```csharp
[C#]
string font = presentation.MasterTheme.FontScheme.Major.GetScriptFont("Cyrl");
Console.WriteLine("Font for Cyrillic script: " + font);
```

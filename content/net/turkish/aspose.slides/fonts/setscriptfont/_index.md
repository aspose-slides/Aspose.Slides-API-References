## Fonts.SetScriptFont metodu

Belirli bir script etiketine bir yazı tipi adı atar; bu, sunumda o scriptin metninin nasıl görüntüleneceğini tanımlar.

```csharp
public void SetScriptFont(string script, string fontName)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| script | String | Yazı sistemini tanımlayan BCP-47 script kodu (örnek: "Arab", "Hebr", "Hans"). |
| fontName | String | Belirtilen script için atanacak yazı tipinin adı. |

### Örnekler

Bu örnek, Arapça scripti için yazı tipini "Segoe UI" olarak nasıl ayarlayacağınızı gösterir:

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segoe UI");
```

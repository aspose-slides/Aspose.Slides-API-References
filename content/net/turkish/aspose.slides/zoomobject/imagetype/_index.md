## ZoomObject.ImageType özelliği

Bir zoom nesnesinin görüntü türünü alır veya ayarlar. Okuma/yazma [`ZoomImageType`](../../zoomimagetype). Varsayılan değer: Preview

```csharp
public ZoomImageType ImageType { get; set; }
```

### Açıklamalar

Zoom nesnesinin slayt önizlemesini mi yoksa bir kapak görüntüsü mü kullandığını belirtir.

### Örnekler

Aşağıdaki örnek, Image Type değerinin Preview olarak değiştirilmesini gösterir. Bu durumda bir Zoom nesnesinin mevcut görüntüsü slayt görüntüsüne değişir:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    IPPImage image = pres.Images.AddImage(Image.FromFile("image.png"));
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1], image);
    zoomFrame.ImageType = ZoomImageType.Preview;
}
```

## LayoutPlaceholderManager.AddSmartArtPlaceholder metod

Yeni bir SmartArt diyagramını tutmak için düzen slaytına yeni bir yer tutucu şekil ekler.

```csharp
public IAutoShape AddSmartArtPlaceholder(float x, float y, float width, float height)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| x | Single | Yeni yer tutucu şeklin X koordinatı. |
| y | Single | Yeni yer tutucu şeklin Y koordinatı. |
| width | Single | Yeni yer tutucu şeklin genişliği. |
| height | Single | Yeni yer tutucu şeklin yüksekliği. |

### Dönüş Değeri

SmartArt yer tutucusuyla [`IAutoShape`](../../iautoshape) oluşturuldu.

### Örnekler

Aşağıdaki örnek, SmartArt yer tutucu şeklini düzen slaytına nasıl ekleyeceğinizi gösterir.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddSmartArtPlaceholder(20, 20, 200, 200);
}
```

## LayoutPlaceholderManager.AddContentPlaceholder yöntemi

Yeni bir yer tutucu şekli ekler; bu şekil resim, tablo, medya veya metin gibi içerik tutmak için düzen slaytına eklenir.

```csharp
public IAutoShape AddContentPlaceholder(float x, float y, float width, float height)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| x | Single | Yeni yer tutucu şeklin X koordinatı. |
| y | Single | Yeni yer tutucu şeklin Y koordinatı. |
| width | Single | Yeni yer tutucu şeklin genişliği. |
| height | Single | Yeni yer tutucu şeklin yüksekliği. |

### Dönüş Değeri

[`IAutoShape`](../../iautoshape) bir İçerik yer tutucusuyla oluşturuldu.

### Örnekler

Aşağıdaki örnek, İçerik yer tutucu şeklinin düzen slaytına nasıl ekleneceğini gösterir.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddContentPlaceholder(20, 20, 500, 300);
}
```

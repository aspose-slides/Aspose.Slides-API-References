## LayoutPlaceholderManager.AddTablePlaceholder metodu

Yeni bir yer tutucu şekli ekleyerek tabloyu tutmak için düzen kaydırmasına ekler.

```csharp
public IAutoShape AddTablePlaceholder(float x, float y, float width, float height)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| x | Single | Yeni yer tutucu şeklin X koordinatı. |
| y | Single | Yeni yer tutucu şeklin Y koordinatı. |
| width | Single | Yeni yer tutucu şeklin genişliği. |
| height | Single | Yeni yer tutucu şeklin yüksekliği. |

### Dönüş Değeri

[`IAutoShape`](../../iautoshape) bir Tablo yer tutucusu ile oluşturuldu.

### Örnekler

Aşağıdaki örnek, Tablo yer tutucu şeklini düzen kaydırmasına nasıl ekleyeceğinizi gösterir.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTablePlaceholder(20, 20, 500, 200);
}
```

## LayoutPlaceholderManager.AddVerticalContentPlaceholder yöntemi

Dikey yönde bir resim, tablo, medya veya metin gibi içeriği tutmak için düzen slaytına yeni bir yer tutucu şekli ekler.

```csharp
public IAutoShape AddVerticalContentPlaceholder(float x, float y, float width, float height)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| x | Single | Yeni yer tutucu şeklin X koordinatı. |
| y | Single | Yeni yer tutucu şeklin Y koordinatı. |
| width | Single | Yeni yer tutucu şeklin genişliği. |
| height | Single | Yeni yer tutucu şeklin yüksekliği. |

### Dönüş Değeri

[`IAutoShape`](../../iautoshape) bir Content (Vertical) yer tutucu ile oluşturuldu.

### Örnekler

Aşağıdaki örnek, Content (Vertical) yer tutucu şeklini düzen slaytına nasıl ekleyeceğinizi gösterir.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalContentPlaceholder(20, 20, 300, 500);
}
```

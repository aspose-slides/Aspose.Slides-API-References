## LayoutPlaceholderManager.AddTextPlaceholder metodu

Düzen slaytına metin içeriği tutmak için yeni bir yer tutucu şekil ekler.

```csharp
public IAutoShape AddTextPlaceholder(float x, float y, float width, float height)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| x | Single | Yeni yer tutucu şeklin X koordinatı. |
| y | Single | Yeni yer tutucu şeklin Y koordinatı. |
| width | Single | Yeni yer tutucu şeklin genişliği. |
| height | Single | Yeni yer tutucu şeklin yüksekliği. |

### Dönüş Değeri

Oluşturuldu [`IAutoShape`](../../iautoshape) bir Text yer tutucusuyla.

### Örnekler

Aşağıdaki örnek, Text yer tutucu şeklinin düzen slaytına nasıl ekleneceğini gösterir.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTextPlaceholder(20, 20, 500, 300);
}
```

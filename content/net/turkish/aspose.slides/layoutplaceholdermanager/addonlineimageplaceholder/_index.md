## LayoutPlaceholderManager.AddOnlineImagePlaceholder yöntemi

Yeni bir yer tutucu şekli ekler; bu şekil, çevrimiçi bir resmi tutmak için düzen slaytına eklenir.

```csharp
public IAutoShape AddOnlineImagePlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| x | Single | Yeni yer tutucu şeklin X koordinatı. |
| y | Single | Yeni yer tutucu şeklin Y koordinatı. |
| width | Single | Yeni yer tutucu şeklin genişliği. |
| height | Single | Yeni yer tutucu şeklin yüksekliği. |

### Dönüş Değeri

[`IAutoShape`](../../iautoshape) bir Çevrimiçi Görüntü yer tutucusuyla oluşturuldu.

### Örnekler

Aşağıdaki örnek, Çevrimiçi Görüntü yer tutucu şeklini düzen slaytına nasıl ekleyeceğinizi gösterir.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddOnlineImagePlaceholder(20, 20, 200, 200);
}
```

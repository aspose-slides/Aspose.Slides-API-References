## LayoutPlaceholderManager.AddPicturePlaceholder yöntemi

Yeni bir resim tutmak için düzen slaytına yeni bir yer tutucu şekil ekler.

```csharp
public IAutoShape AddPicturePlaceholder(float x, float y, float width, float height)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| x | Single | Yeni yer tutucu şeklin X koordinatı. |
| y | Single | Yeni yer tutucu şeklin Y koordinatı. |
| width | Single | Yeni yer tutucu şeklin genişliği. |
| height | Single | Yeni yer tutucu şeklin yüksekliği. |

### Dönüş Değeri

[`IAutoShape`](../../iautoshape) bir Picture placeholder ile oluşturuldu.

### Örnekler

Aşağıdaki örnek, Picture yer tutucu şeklinin düzen slaytına nasıl ekleneceğini gösterir.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddPicturePlaceholder(20, 20, 200, 200);
}
```

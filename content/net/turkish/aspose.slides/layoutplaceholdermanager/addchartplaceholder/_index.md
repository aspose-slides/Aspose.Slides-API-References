## LayoutPlaceholderManager.AddChartPlaceholder yöntemi

Yeni bir placeholder şekli, bir Chart tutmak için yerleşim slaytına ekler.

```csharp
public IAutoShape AddChartPlaceholder(float x, float y, float width, float height)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| x | Single | Yeni placeholder şeklinin X koordinatı. |
| y | Single | Yeni placeholder şeklinin Y koordinatı. |
| width | Single | Yeni placeholder şeklinin genişliği. |
| height | Single | Yeni placeholder şeklinin yüksekliği. |

### Dönüş Değeri

[`IAutoShape`](../../iautoshape) bir Chart yer tutucusuyla oluşturuldu.

### Örnekler

Aşağıdaki örnek, Chart yer tutucu şeklinin yerleşim slaytına nasıl ekleneceğini gösterir.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddChartPlaceholder(20, 20, 200, 200);
}
```

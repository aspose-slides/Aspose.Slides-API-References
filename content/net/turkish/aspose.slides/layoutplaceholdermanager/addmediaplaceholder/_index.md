## LayoutPlaceholderManager.AddMediaPlaceholder metodu

Medyayı tutmak için yer tutucu slayta yeni bir yer tutucu şekil ekler.

```csharp
public IAutoShape AddMediaPlaceholder(float x, float y, float width, float height)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| x | Single | Yeni yer tutucu şeklin X koordinatı. |
| y | Single | Yeni yer tutucu şeklin Y koordinatı. |
| width | Single | Yeni yer tutucu şeklin genişliği. |
| height | Single | Yeni yer tutucu şeklin yüksekliği. |

### Dönüş Değeri

Media yer tutucusuyla [`IAutoShape`](../../iautoshape) oluşturuldu.

### Örnekler

Aşağıdaki örnek, Media yer tutucu şeklini yer tutucu slayta nasıl ekleyeceğinizi gösterir.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddMediaPlaceholder(20, 20, 200, 200);
}
```

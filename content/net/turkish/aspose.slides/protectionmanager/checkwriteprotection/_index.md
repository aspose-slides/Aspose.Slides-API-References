## ProtectionManager.CheckWriteProtection yöntemi

Bir sunumun değiştirilmek üzere parola korumalı olup olmadığını belirler.

```csharp
public bool CheckWriteProtection(string password)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| password | String | Kontrol için kullanılan parola. |

### Dönüş Değeri

Parola geçerli ise true; aksi takdirde false.

### Açıklamalar

1. Bu yöntemi çağırmadan önce [`IsWriteProtected`](../iswriteprotected) özelliğini kontrol etmelisiniz.  
2. Parola null veya boş olduğunda, bu yöntem false döner.

### Örnekler

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

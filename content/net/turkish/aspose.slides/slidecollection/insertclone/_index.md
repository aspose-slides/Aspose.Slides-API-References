## InsertClone(int, ISlide) {#insertclone}

Belirtilen slaytın bir kopyasını koleksiyonun belirtilen konumuna ekler.

```csharp
public ISlide InsertClone(int index, ISlide sourceSlide)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| index | Int32 | Yeni slaydın indeksi. |
| sourceSlide | ISlide | Kopyalanacak slayt. |

### Dönüş Değeri

Eklenen slayt.

### Açıklamalar

Farklı sunumlar arasında bir slaytı kopyalarken slaytın master'ı da kopyalanabilir. Aynı master slaydının birden fazla kopyasının oluşturulmasını önlemek için otomatik olarak kopyalanan master'lar dahili bir kayıt defteri ile izlenir. Master slaytlarının elle kopyalanması ne önlenmez ne de kaydedilir. Kopyalama işlemi üzerinde daha fazla kontrol ihtiyacınız varsa slaytları kopyalamak için [`InsertClone`](../insertclone) veya [`InsertClone`](../insertclone) ve masterları kopyalamak için [`AddClone`](../../imasterslidecollection/addclone) kullanın.

### Örnekler

Aşağıdaki örnek, sunum içinde farklı bir konuma nasıl kopyalanacağını gösterir.

```csharp
[C#]
// Sunum dosyasını temsil eden Presentation sınıfını örnekle
using (Presentation pres = new Presentation("CloneWithInSamePresentation.pptx"))
{
    // İstenen slaytı aynı sunumdaki slayt koleksiyonunun sonuna kopyala
    ISlideCollection slds = pres.Slides;
    // İstenen slaytı aynı sunumda belirtilen indeks'e kopyala
    slds.InsertClone(2, pres.Slides[1]);
    // Değiştirilen sunumu diske kaydet
    pres.Save("Aspose_CloneWithInSamePresentation_out.pptx", SaveFormat.Pptx);
}
```

Aşağıdaki örnek, sunum içinde farklı bir konuma nasıl kopyalanacağını gösterir.

```csharp
[C#]
// Kaynak sunum dosyasını yüklemek için Presentation sınıfını örnekle
using (Presentation srcPres = new Presentation("CloneAtEndOfAnother.pptx"))
{
    // Hedef PPTX (slaytın kopyalanacağı) için Presentation sınıfını örnekle
    using (Presentation destPres = new Presentation())
    {
        ISlideCollection slds = destPres.Slides;
        slds.InsertClone(2, srcPres.Slides[0]);
        // Hedef sunumu diske kaydet
        destPres.Save("Aspose2_out.pptx", SaveFormat.Pptx);
    }
}
```

## InsertClone(int, ISlide, IMasterSlide, bool) {#insertclone_2}

Belirtilen kaynak slaytın bir kopyasını koleksiyonun belirtilen konumuna ekler. Uygun düzen, belirtilen master’dan otomatik olarak seçilir (uygun düzen, kaynak slaydın düzeniyle aynı Tür veya Ad’a sahip düzen olur). Uygun bir düzen yoksa kaynak slaydın düzeni kopyalanır (allowCloneMissingLayout true ise) veya allowCloneMissingLayout false ise PptxEditException fırlatılır.

```csharp
public ISlide InsertClone(int index, ISlide sourceSlide, IMasterSlide destMaster, 
    bool allowCloneMissingLayout)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| index | Int32 | Yeni slaydın indeksi. |
| sourceSlide | ISlide | Kopyalanacak slayt. |
| destMaster | IMasterSlide | Yeni slayt için master slayt. |
| allowCloneMissingLayout | Boolean | Belirtilen master içinde uygun bir düzen yoksa kaynak slaydın düzeni kopyalanır (allowCloneMissingLayout true ise) veya allowCloneMissingLayout false ise PptxEditException fırlatılır. |

### Dönüş Değeri

Eklenen slayt.

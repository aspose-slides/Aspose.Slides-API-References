## PresentationInfo.CheckWriteProtection yöntemi

Yazma korumalı bir sunum için değiştirme şifresinin doğru olup olmadığını denetler.

```csharp
public bool CheckWriteProtection(string password)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| password | String | Denenecek şifre. |

### Dönüş Değeri

Sunum yazma korumalı ve şifre doğru ise True. Aksi takdirde False.

### İstisnalar

| exception | condition |
| --- | --- |
| InvalidOperationException |  |

### Açıklamalar

1. Bu yöntemi çağırmadan önce [`IsWriteProtected`](../iswriteprotected) özelliğini kontrol etmelisiniz. 2. Şifre null veya boş olduğunda bu yöntem false döner.

### Örnekler

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
if (info.IsWriteProtected == NullableBool.True)
{
    bool isWriteProtectedByPassword = info.CheckWriteProtection("my_password");
}
```

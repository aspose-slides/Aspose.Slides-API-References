## IShapeCollection.InsertSummaryZoomFrame method

Yeni bir Summary Zoom çerçevesi oluşturur ve belirtilen dizine şekil koleksiyonuna ekler.

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| index | Int32 | Summary Zoom çerçevesinin ekleneceği sıfır tabanlı dizin. |
| x | Single | Yeni Summary Zoom çerçevesinin x koordinatı, nokta cinsinden. |
| y | Single | Yeni Summary Zoom çerçevesinin y koordinatı, nokta cinsinden. |
| width | Single | Yeni Summary Zoom çerçevesinin genişliği, nokta cinsinden. |
| height | Single | Yeni Summary Zoom çerçevesinin yüksekliği, nokta cinsinden. |

### Açıklamalar

Bu yöntem, sunumdaki tüm bölümler için özet bağlantılarını toplayan bir Summary Zoom çerçevesi oluşturur.

### Örnekler

Bu örnek, bir koleksiyonun belirtilen dizinine Summary Zoom nesnesi oluşturup eklemeyi gösterir ("Presentation.pptx" sunumunda en az iki bölüm olduğunu varsayın):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSummaryZoomFrame(2, 150, 20, 50, 50);
}
```

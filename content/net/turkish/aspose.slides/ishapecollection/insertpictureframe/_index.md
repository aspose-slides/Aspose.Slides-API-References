## IShapeCollection.InsertPictureFrame yöntemi

Belirtilen görüntüyü içeren yeni bir resim çerçevesi oluşturur ve bunu belirtilen indeksde şekil koleksiyonuna ekler.

```csharp
public IPictureFrame InsertPictureFrame(int index, ShapeType shapeType, float x, float y, 
    float width, float height, IPPImage image)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| index | Int32 | Resim çerçevesinin ekleneceği sıfır tabanlı indeks. |
| shapeType | ShapeType | [`ShapeType`](../../shapetype) içinde bulunan şekil türünü belirtir, tüm çizgi türleri hariç: |
| x | Single | Resim çerçevesinin x koordinatı, puan cinsinden. |
| y | Single | Resim çerçevesinin y koordinatı, puan cinsinden. |
| width | Single | Resim çerçevesinin genişliği, puan cinsinden. |
| height | Single | Resim çerçevesinin yüksekliği, puan cinsinden. |
| image | IPPImage | Resim çerçevesinde görüntülenecek [`IPPImage`](../../ippimage). |

## InsertZoomFrame(int, float, float, float, float, ISlide) {#insertzoomframe}

Yeni bir Zoom çerçevesi oluşturur ve belirtilen dizine şekil koleksiyonuna ekler.

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| index | Int32 | Zoom çerçevesinin ekleneceği sıfır tabanlı dizin. |
| x | Single | Yeni Zoom çerçevesinin x koordinatı, nokta cinsinden. |
| y | Single | Yeni Zoom çerçevesinin y koordinatı, nokta cinsinden. |
| width | Single | Yeni Zoom çerçevesinin genişliği, nokta cinsinden. |
| height | Single | Yeni Zoom çerçevesinin yüksekliği, nokta cinsinden. |
| slide | ISlide | Zoom çerçevesi tarafından referans alınan [`ISlide`](../../islide). |

### Dönüş Değeri

Yeni oluşturulan [`IZoomFrame`](../../izoomframe).

### İstisnalar

| İstisna | koşul |
| --- | --- |
| ArgumentException | Referans alınan slayt mevcut sunuma ait değilse fırlatılır. |

### Örnekler

Bu örnek, bir koleksiyonun belirtilen dizinine bir Zoom nesnesi oluşturup eklemeyi gösterir ("Presentation.pptx" sunumunda en az iki slayt olduğunu varsayın):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertZoomFrame(2, 150, 20, 50, 50, pres.Slides[1]);
}
```

## InsertZoomFrame(int, float, float, float, float, ISlide, IPPImage) {#insertzoomframe_1}

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide, IPPImage image)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| index | Int32 | Zoom çerçevesinin ekleneceği sıfır tabanlı dizin. |
| x | Single | Yeni Zoom çerçevesinin x koordinatı, nokta cinsinden. |
| y | Single | Yeni Zoom çerçevesinin y koordinatı, nokta cinsinden. |
| width | Single | Yeni Zoom çerçevesinin genişliği, nokta cinsinden. |
| height | Single | Yeni Zoom çerçevesinin yüksekliği, nokta cinsinden. |
| slide | ISlide | Zoom çerçevesi tarafından referans alınan [`ISlide`](../../islide). |
| image | IPPImage | Referans alınan slayt [`IPPImage`](../../ippimage) için görüntü. |

### Dönüş Değeri

Yeni oluşturulan [`IZoomFrame`](../../izoomframe).

| İstisna | koşul |

### Örnekler

Bu örnek, bir koleksiyonun belirtilen dizinine bir Zoom nesnesi oluşturup eklemeyi gösterir ("Presentation.pptx" sunumunda en az iki slayt olduğunu varsayın):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IPPImage image = pres.Images.AddImage(Image.FromFile("image.png"));
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertZoomFrame(2, 150, 20, 50, 50, pres.Slides[1], image);
}
```

## InsertSectionZoomFrame(int, float, float, float, float, ISection) {#insertsectionzoomframe}

Yeni bir Section Zoom çerçevesi oluşturur ve belirtilen dizinde şekil koleksiyonuna ekler.

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| index | Int32 | Section Zoom çerçevesinin ekleneceği sıfır bazlı indeks. |
| x | Single | Yeni Section Zoom çerçevesinin x-koordinatı, nokta cinsinden. |
| y | Single | Yeni Section Zoom çerçevesinin y-koordinatı, nokta cinsinden. |
| width | Single | Yeni Section Zoom çerçevesinin genişliği, nokta cinsinden. |
| height | Single | Yeni Section Zoom çerçevesinin yüksekliği, nokta cinsinden. |
| section | ISection | Section Zoom çerçevesi tarafından referans verilen [`ISection`](../../isection); bu sunuma ait olmalı ve en az bir slayt içermelidir. |

### Dönüş Değeri

Yeni oluşturulan [`ISectionZoomFrame`](../../isectionzoomframe).

### İstisnalar

| İstisna | koşul |
| --- | --- |
| ArgumentException | Referans verilen bölüm mevcut sunuma ait değilse veya slayt içermiyorsa fırlatılır. |

### Örnekler

Bu örnek, bir koleksiyonun belirtilen dizinine bir Section Zoom nesnesi oluşturmayı ve eklemeyi gösterir (\"Presentation.pptx\" sunumunda en az iki bölüm olduğunu varsayın):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSectionZoomFrame(2, 150, 20, 50, 50, pres.Sections[1]);
}
```

## AddSectionZoomFrame(float, float, float, float, ISection) {#addsectionzoomframe}

Yeni bir Bölüm Yakınlaştırma çerçevesi oluşturur ve şekil koleksiyonunun sonuna ekler.

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section)
```

| Parametre | Tip | Açıklama |
| --- | --- | --- |
| x | Single | Yeni Bölüm Yakınlaştırma çerçevesinin x koordinatı, nokta biriminde. |
| y | Single | Yeni Bölüm Yakınlaştırma çerçevesinin y koordinatı, nokta biriminde. |
| width | Single | Yeni Bölüm Yakınlaştırma çerçevesinin genişliği, nokta biriminde. |
| height | Single | Yeni Bölüm Yakınlaştırma çerçevesinin yüksekliği, nokta biriminde. |
| section | ISection | Bölüm Yakınlaştırma çerçevesi tarafından referans verilen [`ISection`](../../isection); bu sunuma ait olmalı ve en az bir slayt içermelidir. |

### Dönüş Değeri

Yeni oluşturulan [`ISectionZoomFrame`](../../isectionzoomframe).

### İstisnalar

| İstisna | Koşul |
| --- | --- |
| ArgumentException | Referans verilen bölüm mevcut sunuma ait değilse veya slayt içermiyorsa fırlatılır. |

### Örnekler

Bu örnek, bir Bölüm Yakınlaştırma nesnesinin bir koleksiyonun sonuna eklenmesini gösterir (\"Presentation.pptx\" sunumunda en az iki bölüm olduğunu varsayın):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1]);
}
```

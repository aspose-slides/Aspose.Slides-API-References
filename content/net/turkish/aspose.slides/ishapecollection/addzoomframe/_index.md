## AddZoomFrame(float, float, float, float, ISlide) {#addzoomframe}

```csharp
public IZoomFrame AddZoomFrame(float x, float y, float width, float height, ISlide slide)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| x | Single | Yeni Zoom çerçevesinin x koordinatı, puan cinsinden. |
| y | Single | Yeni Zoom çerçevesinin y koordinatı, puan cinsinden. |
| width | Single | Yeni Zoom çerçevesinin genişliği, puan cinsinden. |
| height | Single | Yeni Zoom çerçevesinin yüksekliği, puan cinsinden. |
| slide | ISlide | Zoom çerçevesi tarafından başvurulan [`ISlide`](../../islide); bu sunuma ait olmalıdır. |

### Dönüş Değeri

Yeni oluşturulan [`IZoomFrame`](../../izoomframe).

### İstisnalar

| İstisna | Koşul |
| --- | --- |
| ArgumentException | Başvurulan slayt mevcut sunuma ait değilse fırlatılır. |

### Örnekler

Bu örnek, bir Zoom nesnesinin bir koleksiyonun sonuna eklenmesini gösterir (\"Presentation.pptx\" sunumunda en az iki slayt olduğunu varsayın):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
}
```

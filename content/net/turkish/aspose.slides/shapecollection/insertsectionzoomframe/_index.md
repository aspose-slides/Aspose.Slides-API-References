## InsertSectionZoomFrame(int, float, float, float, float, ISection) {#insertsectionzoomframe}

Belirtilen dizinde yeni bir Section Zoom çerçevesi oluşturur ve şekil koleksiyonuna ekler.

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| index | Int32 | Section Zoom çerçevesinin ekleneceği sıfır tabanlı indeks. |
| x | Single | Yeni Section Zoom çerçevesinin x koordinatı, puan cinsinden. |
| y | Single | Yeni Section Zoom çerçevesinin y koordinatı, puan cinsinden. |
| width | Single | Yeni Section Zoom çerçevesinin genişliği, puan cinsinden. |
| height | Single | Yeni Section Zoom çerçevesinin yüksekliği, puan cinsinden. |
| section | ISection | Section Zoom çerçevesi tarafından başvurulan [`ISection`](../../isection); bu sunuma ait olmalı ve en az bir slayt içermelidir. |

### Dönüş Değeri

Yeni oluşturulan [`ISectionZoomFrame`](../../isectionzoomframe).

### İstisnalar

| İstisna | Koşul |
| --- | --- |
| ArgumentException | Başvurulan bölüm mevcut sunuma ait değilse veya slayt içermiyorsa fırlatılır. |

### Örnekler

Bu örnek, bir koleksiyonun belirtilen dizininde bir Section Zoom nesnesi oluşturmayı ve eklemeyi gösterir ("Presentation.pptx" sunumunda en az iki bölüm olduğunu varsayın):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSectionZoomFrame(2, 150, 20, 50, 50, pres.Sections[1]);
}
```

public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section, IPPImage image)

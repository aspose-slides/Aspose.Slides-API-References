## AddAudioFrameEmbedded(float, float, float, float, Stream) {#addaudioframeembedded_1}

Gömülü bir WAV dosyasıyla yeni bir ses çerçevesi oluşturur ve şekil koleksiyonunun sonuna ekler. Gömülü ses, Presentation.Audios koleksiyonuna eklenir.

```csharp
public IAudioFrame AddAudioFrameEmbedded(float x, float y, float width, float height, 
    Stream audio_stream)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| x | Single | Yeni ses çerçevesinin x koordinatı, puan cinsinden. |
| y | Single | Yeni ses çerçevesinin y koordinatı, puan cinsinden. |
| width | Single | Yeni ses çerçevesinin genişliği, puan cinsinden. |
| height | Single | Yeni ses çerçevesinin yüksekliği, puan cinsinden. |
| audio_stream | Stream | Gömülecek WAV ses verilerini içeren bir giriş akışı. |

### Dönüş Değeri

Yeni oluşturulan [`IAudioFrame`](../../iaudioframe).

### Örnekler

Aşağıdaki örnekler, Audio Frame'in nasıl oluşturulacağını gösterir.

```csharp
[C#]
// Sunum dosyasını temsil eden bir Presentation sınıfını oluşturur
using (Presentation pres = new Presentation())
{
    // İlk slaytı alır
    ISlide sld = pres.Slides[0];
    // wav ses dosyasını akısa yükler
    FileStream fstr = new FileStream("sampleaudio.wav", FileMode.Open, FileAccess.Read);
    // Audio Frame'i ekler
    IAudioFrame audioFrame = sld.Shapes.AddAudioFrameEmbedded(50, 150, 100, 100, fstr);
    // Sesin Oynatma Modunu ve Ses Seviyesini ayarlar
    audioFrame.PlayMode = AudioPlayModePreset.Auto;
    audioFrame.Volume = AudioVolumeMode.Loud;
    // PowerPoint dosyasını diske yazar
    pres.Save("AudioFrameEmbed_out.pptx", SaveFormat.Pptx);
}
```

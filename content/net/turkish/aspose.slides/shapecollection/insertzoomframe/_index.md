## InsertZoomFrame(int, float, float, float, float, ISlide) {#insertzoomframe}

Yeni bir Zoom çerçevesi oluşturur ve belirtilen dizinde şekil koleksiyonuna ekler.

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| index | Int32 | Zoom çerçevesinin ekleneceği sıfır tabanlı dizin. |
| x | Single | Yeni Zoom çerçevesinin x koordinatı, puan cinsinden. |
| y | Single | Yeni Zoom çerçevesinin y koordinatı, puan cinsinden. |
| width | Single | Yeni Zoom çerçevesinin genişliği, puan cinsinden. |
| height | Single | Yeni Zoom çerçevesinin yüksekliği, puan cinsinden. |
| slide | ISlide | Zoom çerçevesi tarafından referans edilen [`ISlide`](../../islide). |

### Dönüş Değeri

Yeni oluşturulan [`IZoomFrame`](../../izoomframe).

### İstisnalar

| İstisna | koşul |
| --- | --- |
| ArgumentException | Referans verilen slayt mevcut sunuma ait değilse oluşturulur. |

### Örnekler

Bu örnek, \"Presentation.pptx\" sunumunda en az iki slayt olduğunu varsayarak, bir koleksiyonun belirtilen dizininde bir Zoom nesnesi oluşturulmasını ve eklenmesini gösterir:

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertZoomFrame(2, 150, 20, 50, 50, pres.Slides[1]);
}
```

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height,
```

| height | Single | Yeni Zoom çerçevesinin yüksekliği, puan cinsinden. |

### Örnekler

Bu örnek, \"Presentation.pptx\" sunumunda en az iki slayt olduğunu varsayarak, bir koleksiyonun belirtilen dizininde bir Zoom nesnesi oluşturulmasını ve eklenmesini gösterir:

```csharp

using (Presentation pres = new Presentation("Presentation.pptx"))

}
```

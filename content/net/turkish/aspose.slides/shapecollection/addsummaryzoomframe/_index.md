## ShapeCollection.AddSummaryZoomFrame yöntemi

Yeni bir Summary Zoom çerçevesi oluşturur ve şekil koleksiyonunun sonuna ekler.

```csharp
public ISummaryZoomFrame AddSummaryZoomFrame(float x, float y, float width, float height)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| x | Single | Yeni Summary Zoom çerçevesinin x koordinatı, nokta cinsinden. |
| y | Single | Yeni Summary Zoom çerçevesinin y koordinatı, nokta cinsinden. |
| width | Single | Yeni Summary Zoom çerçevesinin genişliği, nokta cinsinden. |
| height | Single | Yeni Summary Zoom çerçevesinin yüksekliği, nokta cinsinden. |

### Dönüş Değeri

Yeni oluşturulan [`ISummaryZoomFrame`](../../isummaryzoomframe).

### İstisnalar

| İstisna | koşul |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | Sunumda bölüm bulunmadığında veya hedef slayt herhangi bir bölüme ait olmadığında fırlatılır. |

### Açıklamalar

Bu yöntem yeni bir Summary Zoom oluşturur ve bu sunumdaki tüm bölümler için bir nesne koleksiyonunu içine yerleştirir.

### Örnekler

Bu örnek, bir koleksiyonun sonuna Summary Zoom nesnesi eklemeyi gösterir (\"Presentation.pptx\" sunumunda en az iki bölüm olduğunu varsayalım):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSummaryZoomFrame(150, 20, 500, 250);
}
```

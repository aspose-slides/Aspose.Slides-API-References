## InsertAudioFrameEmbedded(int, float, float, float, float, Stream) {#insertaudioframeembedded_1}

Yeni bir ses çerçevesi oluşturur ve gömülü bir WAV dosyasıyla birlikte belirtilen dizine şekil koleksiyonuna ekler. Gömülü ses, Presentation.Audios koleksiyonuna eklenir.

```csharp
public IAudioFrame InsertAudioFrameEmbedded(int index, float x, float y, float width, float height, 
    Stream audio_stream)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| index | Int32 | Ses çerçevesinin ekleneceği sıfır tabanlı dizin. |
| x | Single | Yeni ses çerçevesinin x koordinatı, puan cinsinden. |
| y | Single | Yeni ses çerçevesinin y koordinatı, puan cinsinden. |
| width | Single | Yeni ses çerçevesinin genişliği, puan cinsinden. |
| height | Single | Yeni ses çerçevesinin yüksekliği, puan cinsinden. |
| audio_stream | Stream | Gömülecek WAV ses verilerini içeren bir giriş akışı. |

### Dönüş Değeri

Yeni oluşturulan [`IAudioFrame`](../../iaudioframe).

## InsertAudioFrameEmbedded(int, float, float, float, float, IAudio) {#insertaudioframeembedded}

```csharp
public IAudioFrame InsertAudioFrameEmbedded(int index, float x, float y, float width, float height, 
    IAudio audio)
```

| Parametre | Tür | Açıklama |

| index | Int32 | Ses çerçevesinin ekleneceği sıfır tabanlı dizin. |
| x | Single | Yeni ses çerçevesinin x koordinatı, puan cinsinden. |
| y | Single | Yeni ses çerçevesinin y koordinatı, puan cinsinden. |
| width | Single | Yeni ses çerçevesinin genişliği, puan cinsinden. |
| height | Single | Yeni ses çerçevesinin yüksekliği, puan cinsinden. |
| audio | IAudio | Gömmek için Presentation.Audios koleksiyonundan bir [`IAudio`](../../iaudio) örneği. |

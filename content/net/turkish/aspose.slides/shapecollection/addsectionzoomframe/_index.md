## AddSectionZoomFrame(float, float, float, float, ISection) {#addsectionzoomframe}

Yeni bir Section Zoom çerçevesi oluşturur ve şekil koleksiyonunun sonuna ekler.

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| x | Single | Yeni Section Zoom çerçevesinin x koordinatı, nokta cinsinden. |
| y | Single | Yeni Section Zoom çerçevesinin y koordinatı, nokta cinsinden. |
| width | Single | Yeni Section Zoom çerçevesinin genişliği, nokta cinsinden. |
| height | Single | Yeni Section Zoom çerçevesinin yüksekliği, nokta cinsinden. |
| section | ISection | [`ISection`](../../isection) referans verilen Section Zoom çerçevesi; bu sunuma ait olmalı ve en az bir slayt içermelidir. |

### Dönüş Değeri

Yeni oluşturulan [`ISectionZoomFrame`](../../isectionzoomframe).

### İstisnalar

| istisna | koşul |
| --- | --- |
| ArgumentException | Referans verilen bölüm mevcut sunuma ait değilse veya slayt içermiyorsa fırlatılır. |

### Örnekler

Bu örnek, bir koleksiyonun sonuna bir Section Zoom nesnesi eklemeyi gösterir (\"Presentation.pptx\" sunumunda en az iki bölüm olduğunu varsayın):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1]);
}
```

## AddSectionZoomFrame(float, float, float, float, ISection, IPPImage) {#addsectionzoomframe_1}

Önceden tanımlı bir resimle yeni bir Section Zoom çerçevesi oluşturur ve şekil koleksiyonunun sonuna ekler.

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section, IPPImage image)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| x | Single | Yeni Section Zoom çerçevesinin x koordinatı, nokta cinsinden. |
| y | Single | Yeni Section Zoom çerçevesinin y koordinatı, nokta cinsinden. |
| width | Single | Yeni Section Zoom çerçevesinin genişliği, nokta cinsinden. |
| height | Single | Yeni Section Zoom çerçevesinin yüksekliği, nokta cinsinden. |
| section | ISection | [`ISection`](../../isection) referans verilen Section Zoom çerçevesi; bu sunuma ait olmalı ve en az bir slayt içermelidir. |
| image | IPPImage | Section Zoom çerçevesi içinde görüntülenecek [`IPPImage`](../../ippimage). |

### Dönüş Değeri

Yeni oluşturulan [`ISectionZoomFrame`](../../isectionzoomframe).

### İstisnalar

| istisna | koşul |
| --- | --- |
| ArgumentException | Referans verilen bölüm mevcut sunuma ait değilse veya slayt içermiyorsa fırlatılır. |

### Örnekler

Bu örnek, bir koleksiyonun sonuna bir Section Zoom nesnesi eklemeyi gösterir (\"Presentation.pptx\" sunumunda en az iki bölüm olduğunu varsayın):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IPPImage image = pres.Images.AddImage(Image.FromFile("image.png"));
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1], image);
}
```

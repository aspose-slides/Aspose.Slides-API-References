## ShapeCollection.InsertSummaryZoomFrame yöntemi

Belirtilen dizinde yeni bir Summary Zoom çerçevesi oluşturur ve şekil koleksiyonuna ekler.

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| index | Int32 | Summary Zoom çerçevesinin ekleneceği sıfırdan başlayan indeks. |
| x | Single | Yeni Summary Zoom çerçevesinin x koordinatı, nokta biriminde. |
| y | Single | Yeni Summary Zoom çerçevesinin y koordinatı, nokta biriminde. |
| width | Single | Yeni Summary Zoom çerçevesinin genişliği, nokta biriminde. |
| height | Single | Yeni Summary Zoom çerçevesinin yüksekliği, nokta biriminde. |

### Dönüş Değeri

The newly created [`ISummaryZoomFrame`](../../isummaryzoomframe).

### İstisnalar

| istisna | koşul |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | Sunumda hiçbir bölüm yoksa veya hedef slayt herhangi bir bölüme ait değilse fırlatılır. |

### Açıklamalar

Bu yöntem, sunumdaki tüm bölümler için özet bağlantılarını toplayan bir Summary Zoom çerçevesi oluşturur.

### Örnekler

Bu örnek, bir koleksiyonun belirtilen dizinine bir Summary Zoom nesnesi oluşturup eklemeyi gösterir (\"Presentation.pptx\" sunumunda en az iki bölüm olduğunu varsayın):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSummaryZoomFrame(2, 150, 20, 50, 50);
}
```

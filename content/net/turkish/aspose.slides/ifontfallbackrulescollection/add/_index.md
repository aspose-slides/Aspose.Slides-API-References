## IFontFallBackRulesCollection.Add yöntemi

Koleksiyonun sonuna yeni bir FallBack kuralı ekler.

```csharp
public void Add(IFontFallBackRule sourceRule)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| sourceRule | IFontFallBackRule | Eklemek için belirtilen kural |

### Örnekler

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    //FontsManager'dan boş veya önceden başlatılmış kurallar koleksiyonu alınıyor
    IFontFallBackRulesCollection rulesList = pres.FontsManager.FontFallBackRulesCollection;
```

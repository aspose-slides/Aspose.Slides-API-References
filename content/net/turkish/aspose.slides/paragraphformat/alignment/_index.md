## ParagraphFormat.Alignment özelliği

Metin hizalamasını miras alınmadan bir paragrafta getirir veya ayarlar. Okunabilir/Yazılabilir [`TextAlignment`](../../textalignment).

```csharp
public TextAlignment Alignment { get; set; }
```

### Örnekler

Aşağıdaki örnek kod, PowerPoint Sunumunda Metin Paragraflarını Hizalamayı gösterir.

```csharp
[C#]
// PPTX dosyasını temsil eden bir Presentation nesnesi oluşturur
using (Presentation pres = new Presentation("ParagraphsAlignment.pptx"))
{
    // İlk slayta erişim
    ISlide slide = pres.Slides[0];
    // Slayttaki birinci ve ikinci yer tutucuya erişilir ve AutoShape olarak tip dönüştürülür
    ITextFrame tf1 = ((IAutoShape)slide.Shapes[0]).TextFrame;
    ITextFrame tf2 = ((IAutoShape)slide.Shapes[1]).TextFrame;
    // Her iki yer tutucudaki metin değiştirilir
    tf1.Text = "Center Align by Aspose";
    tf2.Text = "Center Align by Aspose";
    // Yer tutucuların ilk paragrafı alınır
    IParagraph para1 = tf1.Paragraphs[0];
    IParagraph para2 = tf2.Paragraphs[0];
    // Metin paragrafı ortaya hizalanır
    para1.ParagraphFormat.Alignment = TextAlignment.Center;
    para2.ParagraphFormat.Alignment = TextAlignment.Center;
    //Sunumu PPTX dosyası olarak kaydeder
    pres.Save("Centeralign_out.pptx", SaveFormat.Pptx);
}
```

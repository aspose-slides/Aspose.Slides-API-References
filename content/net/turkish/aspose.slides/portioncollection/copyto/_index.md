## PortionCollection.CopyTo yöntemi

ICollection öğelerinin bir Array'e kopyalanmasını sağlar, belirli bir Array diziniyle başlayarak.

```csharp
public void CopyTo(IPortion[] array, int arrayIndex)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| array | IPortion[] | ICollection'dan kopyalanan öğelerin hedefi olan tek boyutlu Array. Array sıfır tabanlı indekslemeye sahip olmalıdır. |
| arrayIndex | Int32 | *array* içinde kopyalamanın başladığı sıfır tabanlı indeks. |

### İstisnalar

| istisna | koşul |
| --- | --- |
| ArgumentNullException | *array* null'dur. |
| ArgumentOutOfRangeException | *arrayIndex* 0'dan küçüktür. |
| ArgumentException | Kaynak ICollection'daki öğe sayısı, hedef *array*'ın sonuna kadar *arrayIndex*'den kalan kullanılabilir alandan daha fazladır. |

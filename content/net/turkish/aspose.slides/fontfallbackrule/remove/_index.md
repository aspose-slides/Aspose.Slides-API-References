## FontFallBackRule.Remove yöntemi

Listeden belirli bir FallBack yazı tipinin ilk oluşumunu kaldırır.

```csharp
public void Remove(string fontName)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| fontName | String | Listeden kaldırılacak yazı tipinin adı. |

### Örnekler

```csharp
[C#]

IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// Listeden Tahoma'yı kaldır.
newRule.Remove ("Tahoma");
```

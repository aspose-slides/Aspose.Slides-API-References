## AddFallBackFonts(string) {#addfallbackfonts}

FallBack yazı tipleri listesine yeni bir yazı tipi ekler.

```csharp
public void AddFallBackFonts(string fontName)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| fontName | String | FallBack için virgülle ayrılmış yazı tipi adı veya adları |

### Örnekler

```csharp
[C#]
// FontFallBackRule sınıfının yeni bir örneğini oluştur
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho");

// Kurala ikinci bir yazı tipi ekle
newRule.AddFallBackFonts("MS Gothic");
// Kurala üçüncü ve dördüncü yazı tiplerini ekle
newRule.AddFallBackFonts("Tahoma, Times New Roman");
```

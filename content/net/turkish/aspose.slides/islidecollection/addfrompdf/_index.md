## AddFromPdf(Stream, PdfImportOptions) {#addfrompdf_1}

PDF belgesinden slaytlar oluşturur ve bunları koleksiyonun sonuna ekler.

```csharp
public ISlide[] AddFromPdf(Stream pdfStream, PdfImportOptions pdfImportOptions)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| pdfStream | Stream | PDF belgesinin kaynağı olarak kullanılacak akış |
| pdfImportOptions | PdfImportOptions | pdf içe aktarma seçenekleri |

### Dönüş Değeri

Eklenen slaytlar

### Örnekler

Örnek:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    using (Stream stream = new FileStream("document.pdf", FileMode.Open, FileAccess.Read, FileShare.Read))
```

## ILayoutPlaceholderManager.AddSmartArtPlaceholder yöntemi

Düzen slaytına bir SmartArt diyagramı tutmak için yeni bir yer tutucu şekil ekler.

```csharp
public IAutoShape AddSmartArtPlaceholder(float x, float y, float width, float height)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| x | Single | Yeni yer tutucu şeklin X koordinatı. |
| y | Single | Yeni yer tutucu şeklin Y koordinatı. |
| width | Single | Yeni yer tutucu şeklin genişliği. |
| height | Single | Yeni yer tutucu şeklin yüksekliği. |

### Dönüş Değeri

SmartArt yer tutucusu ile [`IAutoShape`](../../iautoshape) oluşturuldu.

### Örnekler

Aşağıdaki örnek, SmartArt yer tutucu şeklinin düzen slaytına nasıl ekleneceğini gösterir.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddSmartArtPlaceholder(20, 20, 200, 200);
}
```

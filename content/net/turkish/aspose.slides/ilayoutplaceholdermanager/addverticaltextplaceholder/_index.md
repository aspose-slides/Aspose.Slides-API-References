## ILayoutPlaceholderManager.AddVerticalTextPlaceholder metot

Dikey yönde metin içeriğini tutmak için yeni bir yer tutucu şekil ekler.

```csharp
public IAutoShape AddVerticalTextPlaceholder(float x, float y, float width, float height)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| x | Single | Yeni yer tutucu şeklin X koordinatı. |
| y | Single | Yeni yer tutucu şeklin Y koordinatı. |
| width | Single | Yeni yer tutucu şeklin genişliği. |
| height | Single | Yeni yer tutucu şeklin yüksekliği. |

### Dönüş Değeri

Metin (Dikey) yer tutucu ile [`IAutoShape`](../../iautoshape) oluşturuldu.

### Örnekler

Aşağıdaki örnek, Metin (Dikey) yer tutucu şeklinin düzen slaytına nasıl ekleneceğini gösterir.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalTextPlaceholder(20, 20, 300, 500);
}
```

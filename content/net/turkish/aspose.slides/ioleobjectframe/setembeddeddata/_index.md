## IOleObjectFrame.SetEmbeddedData metodu

OLE gömülü veri hakkında bilgi ayarlar.

```csharp
public void SetEmbeddedData(IOleEmbeddedDataInfo embeddedData)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| embeddedData | IOleEmbeddedDataInfo | Gömülü veri [`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo) |

### İstisnalar

| istisna | koşul |
| --- | --- |
| ArgumentNullException | embeddedData parametresi null olduğunda. |

### Açıklamalar

Bu metod, nesnenin özelliklerini yeni veriyi yansıtacak şekilde değiştirir ve IsObjectLink bayrağını false olarak ayarlar, OLE nesnesinin gömülü olduğunu gösterir.

### Örnekler

Aşağıdaki örnek, mevcut [`IOleObjectFrame`](../../ioleobjectframe) nesnesi için OLE gömülü veriyi ve türünü nasıl değiştireceğinizi gösterir.

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    OleObjectFrame oof = pres.Slides[0].Shapes[0] as OleObjectFrame;
    if (oof != null)
    {
        IOleEmbeddedDataInfo newData = new OleEmbeddedDataInfo(File.ReadAllBytes("Picture.png"), "png");
        oof.SetEmbeddedData(newData);
    }
}
```

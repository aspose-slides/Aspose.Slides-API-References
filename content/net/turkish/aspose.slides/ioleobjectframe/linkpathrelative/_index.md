## IOleObjectFrame.LinkPathRelative özelliği

Bağlantılı bir dosya mevcutsa göreli yolu döndürür, aksi takdirde boş bir dize döndürür. Yalnızca Okunur String.

```csharp
public string LinkPathRelative { get; }
```

### Açıklamalar

Ppt sunumlarında, bazı Ole nesne bağlantıları göreli bir temsil içerebilir.

### Örnekler

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.ppt"))
{
    IOleObjectFrame oleFrame = presentation.Slides[0].Shapes[0] as IOleObjectFrame;

    if (oleFrame != null)
    {
        Console.WriteLine("The relative path: " + oleFrame.LinkPathRelative);
    } 
}
```

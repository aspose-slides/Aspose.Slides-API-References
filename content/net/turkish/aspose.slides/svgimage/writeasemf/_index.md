## SvgImage.WriteAsEmf metodu

SVG görüntüsünü EMF dosyası olarak kaydeder.

```csharp
public void WriteAsEmf(Stream stream)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| stream | Stream | Hedef akış |

### İstisnalar

| İstisna | Koşul |
| --- | --- |
| ArgumentNullException | Hedef akış **null**'dır |

### Örnekler

Aşağıdaki örnek, SVG görüntüsünü metafile olarak kaydetmeyi gösterir.

```csharp
[C#]
// Yeni SVG görüntüsü oluşturur
ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));

// SVG görüntüsünü bir metafile olarak kaydeder
using (var fileStream = System.IO.File.OpenWrite("SvgAsEmf.emf"))
    svgImage.WriteAsEmf(fileStream);
```

Bu örnek, SVG görüntüsünü bir metafile olarak sunum görüntü koleksiyonuna eklemeyi gösterir.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // Yeni SVG görüntüsü oluşturur
    ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));
    using (var memStream = new MemoryStream())
    {
        // SVG görüntüsünü bir metafile olarak kaydeder
        svgImage.WriteAsEmf(memStream);
        // Metafile'i görüntü koleksiyonuna ekler
        pres.Images.AddImage(memStream.ToArray());
    }
}
```

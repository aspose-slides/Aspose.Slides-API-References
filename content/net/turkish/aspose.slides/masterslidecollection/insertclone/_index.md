## MasterSlideCollection.InsertClone yöntemi

Belirtilen bir ana slaytın bir kopyasını koleksiyonun belirtilen konumuna ekler. Bağlı düzen slaytları da kopyalanacaktır.

```csharp
public IMasterSlide InsertClone(int index, IMasterSlide sourceMaster)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| index | Int32 | Yeni slaydın indeksi. |
| sourceMaster | IMasterSlide | Klonlanacak slayt. |

### Dönüş Değeri

Eklenen master slayt.

### Örnekler

Aşağıdaki örnek, master slaytı başka bir PowerPoint Sunumu içinde nasıl klonlayacağını gösterir.

```csharp
[C#]
// Kaynak sunum dosyasını yüklemek için Presentation sınıfını örnekleyin
using (Presentation srcPres = new Presentation("CloneToAnotherPresentationWithMaster.pptx"))
{
    // Hedef sunum (slayın klonlanacağı) için Presentation sınıfını örnekleyin
    using (Presentation destPres = new Presentation())
    {
        // Kaynak sunumdaki slaytlar koleksiyonundan ISlide'ı örnekleyin ve
        // Master slaytı
        ISlide SourceSlide = srcPres.Slides[0];
        IMasterSlide SourceMaster = SourceSlide.LayoutSlide.MasterSlide;
		// Hedef sunumun Master Slaytlarını al
        IMasterSlideCollection masters = destPres.Masters;
        // İstenen master slaytı kaynak sunumdan alıp, master koleksiyonuna
        // Hedef sunuma klonlayın
        IMasterSlide iSlide = masters.AddClone(SourceMaster);
        // Hedef sunumdaki slaytların koleksiyonu
        ISlideCollection slds = destPres.Slides;
		// Kaynak slaytı hedef slayt koleksiyonuna klonlayın.
        slds.AddClone(SourceSlide, iSlide, true);
        // Save the destination presentation to disk
        destPres.Save("CloneToAnotherPresentationWithMaster_out.pptx", SaveFormat.Pptx);
    }
}
```

## TableFormat.GetEffective yöntemi

Miras ve tablo stilleri uygulanmış etkili tablo biçimlendirme özelliklerini alır.

```csharp
public ITableFormatEffectiveData GetEffective()
```

### Dönüş Değeri

Bir [`ITableFormatEffectiveData`](../../itableformateffectivedata).

### Örnekler

Bu örnek, farklı tablo mantık bölümleri için etkili dolgu formatının alınmasını gösterir. Lütfen hücre biçimlendirmenin her zaman satır biçimlendirmeden daha yüksek önceliğe sahip olduğunu, satırın sütundan, sütunun ise tüm tablodan daha yüksek olduğunu unutmayın. Böylece sonunda CellFormatEffectiveData özellikleri tabloyu çizerken her zaman kullanılır. Aşağıdaki kod sadece bir API örneğidir.

```csharp
[C#]
using (Presentation pres = new Presentation(@"MyPresentation.pptx"))
{
    ITable tbl = pres.Slides[0].Shapes[0] as Table;
    IFillFormatEffectiveData tableFillFormatEffective = tbl.TableFormat.GetEffective().FillFormat;
    IFillFormatEffectiveData rowFillFormatEffective = tbl.Rows[0].RowFormat.GetEffective().FillFormat;
    IFillFormatEffectiveData columnFillFormatEffective = tbl.Columns[0].ColumnFormat.GetEffective().FillFormat;
    IFillFormatEffectiveData cellFillFormatEffective = tbl[0, 0].CellFormat.GetEffective().FillFormat;
    /* Çıktı ve karşılaştırma */
}
```

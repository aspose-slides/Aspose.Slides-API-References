## IPictureFillFormat.DeletePictureCroppedAreas metodu

Dolgu Picture nesnesinin kırpılmış alanlarını siler.

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### Dönüş Değeri

Kırpılmış görüntü veya kırpma gerekli değilse özgün görüntü.

### Açıklamalar

Bu metot, WMF/EMF metafile'larını kırpma sırasında raster PNG görüntüsüne dönüştürür.

### Örnekler

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // PictureFrame'i alır
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // PictureFrame görüntüsünün kırpılmış alanlarını siler
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

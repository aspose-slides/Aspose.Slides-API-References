## IPictureFillFormat.TileFlip özelliği

Doku döşemesini yatay, dikey veya her iki eksen etrafında çevirir. Okunur/Yazılabilir [`TileFlip`](../../tileflip).

```csharp
public TileFlip TileFlip { get; set; }
```

### Açıklamalar

Varsayılan NoFlip.

### Örnekler

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Şeklin resim doldurma biçimini alır
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Resim doldurma modunu Döşeme (Tile) olarak ayarlar
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Doku döşemesini dikey eksen etrafında çevirir.
    pictureFillFormat.TileFlip = TileFlip.FlipY;
}
```

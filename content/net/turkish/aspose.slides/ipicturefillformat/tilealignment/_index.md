## IPictureFillFormat.TileAlignment özelliği

Doku şekil içinde nasıl hizalanacağını döndürür veya ayarlar. Bu ayar, doku deseninin başlangıç noktasını ve şekil boyunca nasıl tekrarlandığını kontrol eder. Okunur/yazılabilir [`RectangleAlignment`](../../rectanglealignment).

```csharp
public RectangleAlignment TileAlignment { get; set; }
```

### Açıklamalar

Varsayılan değer TopLeft'dir.

### Örnekler

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Şeklin resim doldurma formatını alır
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Resim doldurma modunu Döşeme olarak ayarlar
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Döşemenin hizalamasını sağ alt köşeye ayarlar
    pictureFillFormat.TileAlignment = RectangleAlignment.BottomRight;
}
```

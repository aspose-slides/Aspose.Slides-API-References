## IPictureFillFormat.TileOffsetY özelliği

Doku'nun şeklin kökeninden dikey ofsetini puan cinsinden döndürür veya ayarlar. Pozitif bir değer dokuyu aşağı kaydırırken, negatif bir değer yukarı kaydırır. Okunur/Yazılır Single.

```csharp
public float TileOffsetY { get; set; }
```

### Örnekler

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Şeklin resim doldurma formatını alır
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Resim doldurma modunu Tile olarak ayarlar
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Doku'nun dikey ofsetini -50 puan olarak ayarlar
    pictureFillFormat.TileOffsetY = -50;
}
```

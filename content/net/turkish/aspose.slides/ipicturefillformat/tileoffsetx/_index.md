## IPictureFillFormat.TileOffsetX özellik

Doku'nun şeklin orijinalinden yatay ofsetini nokta cinsinden döndürür veya ayarlar. Pozitif bir değer doku'yu sağa, negatif bir değer ise sola kaydırır. Okuma/Yazma Single.

```csharp
public float TileOffsetX { get; set; }
```

### Örnekler

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Şeklin resim dolgu formatını alır
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Resim dolgu modunu Tile olarak ayarlar
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Doku'nun yatay ofsetini 20 noktaya ayarlar
    pictureFillFormat.TileOffsetX = 20f;
}
```

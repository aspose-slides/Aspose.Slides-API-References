## IBlobManagementOptions.PresentationLockingBehavior özelliği

Bu özellik, Presentation sınıfının bir örneğinin ömrü boyunca kaynağın – dosyanın veya akışın – sahibi olup olamayacağını tanımlar. Örnek bir sahibi ise kaynağı kilitler. Bu, BLOB'larla çalışırken bellek tüketimini ve performansı artırmaya yardımcı olur, ancak kaynak (akış veya dosya) Presentation örneğinin ömrü boyunca değiştirilemez. İşte bir örnek:

```csharp
public PresentationLockingBehavior PresentationLockingBehavior { get; set; }
```

### Örnekler

```csharp
LoadOptions loadOptions = new LoadOptions { 
  BlobManagementOptions = {PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked} };
using (Presentation pres = new Presentation("pres.pptx", loadOptions)) {
  // IOException, pres.pptx dosyası Presentation ömrü boyunca kilitli olduğu için fırlatılacaktır
  // File.Delete("pres.pptx");
}
// Presentation nesnesi yok edildiğinden sonra dosya kilidi kaldırılır ve silinebilir
File.Delete("pres.pptx");
```

## ISummaryZoomSectionCollection.IndexOf metodu

Belirtilen SummaryZoomSection nesnesinin indeksini döndürür.

```csharp
public int IndexOf(ISummaryZoomSection summaryZoomSection)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| summaryZoomSection | ISummaryZoomSection | Bulunacak SummaryZoomSection nesnesi [`ISummaryZoomSection`](../../isummaryzoomsection). |

### Dönüş Değeri

SummaryZoomSection nesnesinin indeksi veya -1, eğer SummaryZoomSection nesnesi bu koleksiyonda değilse.

### Örnekler

Örnek, Summary Zoom Section öğesini indeksle almayı gösterir:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    ISummaryZoomFrame zoomFrame = (ISummaryZoomFrame) pres.Slides[1].Shapes[0];
    ISummaryZoomSectionCollection collection = zoomFrame.SummaryZoomCollection;
    ISummaryZoomSection selectedObject = collection.GetSummarySection(pres.Sections[2]);
    int idx = collection.IndexOf(selectedObject);
}
```

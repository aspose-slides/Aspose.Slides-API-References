## IFontFallBackRule.Remove yöntemi

Belirli bir FallBack yazı tipinin listedeki ilk oluşumunu kaldırır.

```csharp
public void Remove(string fontName)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| fontName | String | Listeden kaldırılacak yazı tipinin adı. |

### Örnekler

```csharp
[C#]
// Yazı tiplerinin bir listesini içeren bir kural oluşturur.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

//Listeden Tahoma'yı kaldırma
newRule.Remove ("Tahoma");
```

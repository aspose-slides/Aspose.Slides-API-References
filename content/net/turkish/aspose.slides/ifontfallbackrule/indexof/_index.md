## IFontFallBackRule.IndexOf yöntemi

Koleksiyondaki belirtilen kuralın dizinini döndürür.

```csharp
public int IndexOf(string fontName)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| fontName | String | Bulunması gereken fontun adı. |

### Dönüş Değeri

Bir fontun dizini ya da listede font bulunamazsa -1.

### Örnekler

```csharp
[C#]
// Bir kural oluşturur ve bir font listesi içerir.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

//Tahoma indeksini al
int tahomaIndex = newRule.IndexOf("Tahoma");
```

### Örnekler

Aşağıdaki örnek, belirli bir PowerPoint sunum slaytına Not eklemenin nasıl yapılacağını gösterir.

```csharp
[C#]
	// Bir sunum dosyasını temsil eden Presentation nesnesini oluşturun
	using(Presentation presentation = new Presentation(dataDir + "AccessSlides.pptx")) {
	  // İlk slayta not ekle
	  INotesSlideManager mgr = presentation.Slides[0].NotesSlideManager;
	  INotesSlide noteSlide = mgr.AddNotesSlide();
	  noteSlide.NotesTextFrame.Text = "Your Notes";
	  // Sunumu diske kaydet
	  presentation.Save(dataDir + "RemoveNotesAtSpecificSlide_out.pptx", SaveFormat.Pptx);
	}
```

Aşağıdaki örnek, PowerPoint sunumunun belirli bir slaytından Notların nasıl kaldırılacağını gösterir.

```csharp
[C#]
	// Bir sunum dosyasını temsil eden Presentation nesnesini oluşturun
	using(Presentation presentation = new Presentation(dataDir + "AccessSlides.pptx")) {
	  // İlk slaytin notlarını kaldırma
	  INotesSlideManager mgr = presentation.Slides[0].NotesSlideManager;
	  mgr.RemoveNotesSlide();
	  // Sunumu diske kaydet
	  presentation.Save(dataDir + "RemoveNotesAtSpecificSlide_out.pptx", SaveFormat.Pptx);
	}
```

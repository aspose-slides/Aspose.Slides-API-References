## ISlideShowTransition.AdvanceAfter özelliği

Bu öznitelik, slayt gösterisinin belirli bir süreden sonra bir sonraki slayta geçip geçmeyeceğini belirler. Okunabilir/Yazılabilir Boolean.

```csharp
public bool AdvanceAfter { get; set; }
```

### Örnekler

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    // İlk slayt geçişini al
    ISlideShowTransition slideTransition = pres.Slides[0].SlideShowTransition;
    
    // Advance Slide After bayrağının işaretli olup olmadığını kontrol et
    if (slideTransition.AdvanceAfter)
    {
        // Advance Slide After Time değerini al
        uint advanceAfterTime = slideTransition.AdvanceAfterTime;
    }
}
```

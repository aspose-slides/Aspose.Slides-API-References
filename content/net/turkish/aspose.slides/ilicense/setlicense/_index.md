## SetLicense(string) {#setlicense_1}

Bileşeni lisanslar.

```csharp
public void SetLicense(string licenseName)
```

| Parameter | Type | Description |
| --- | --- | --- |
| licenseName | String | Tam veya kısa bir dosya adı ya da gömülü kaynağın adı olabilir. Değerlendirme moduna geçmek için boş bir dize kullanın. |

### Açıklamalar

Lisansı aşağıdaki konumlarda bulmaya çalışır:

1. Belirtilen yol.
2. Bileşen derlemesinin klasörü.
3. İstemcinin çağıran derlemesinin klasörü.
4. Giriş derlemesinin klasörü.
5. İstemcinin çağıran derlemesindeki gömülü kaynak.

**Not:** .NET Compact Framework'te, lisansı yalnızca şu konumlardan bulmaya çalışır:

1. Belirtilen yol.
2. İstemcinin çağıran derlemesindeki gömülü kaynak.

### Örnekler

Bu örnekte, MyLicense.lic adlı lisans dosyasını bulmak için bileşeni içeren klasörde, çağıran derlemeyi içeren klasörde, giriş derlemesinin klasöründe ve ardından çağıran derlemedeki gömülü kaynaklarda bir deneme yapılacaktır.

```csharp
[C#]

License license = new License();
license.SetLicense("MyLicense.lic");


[Visual Basic]

Dim license As License = New License
license.SetLicense("MyLicense.lic")
```

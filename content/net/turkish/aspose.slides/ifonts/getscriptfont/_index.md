## IFonts.GetScriptFont metodu

Belirli bir script etiketine ilişkin yazı tipinin adını sunum temasından alır.

```csharp
public string GetScriptFont(string script)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| script | String | Yazı sistemini tanımlamak için kullanılan BCP-47 script kodu (ör. "Latn", "Cyrl", "Jpan"). |

### Dönüş Değeri

Belirtilen script için kullanılan yazı tipinin adı, script tanımlı değilse `null`.

### Örnekler

Bu örnek, sunum temasında Kiril alfabesine (Cyrillic) atanmış yazı tipini nasıl alacağınızı gösterir.

```csharp
[C#]
string font = presentation.MasterTheme.FontScheme.Major.GetScriptFont("Cyrl");
Console.WriteLine("Font for Cyrillic script: " + font);
```

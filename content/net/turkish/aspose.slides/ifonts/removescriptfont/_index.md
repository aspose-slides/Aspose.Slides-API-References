## IFonts.RemoveScriptFont yöntemi

Temanın yazı tipi koleksiyonundan belirli bir script etiketine ilişkin yazı tipi ayarını kaldırır.

```csharp
public void RemoveScriptFont(string script)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| script | String | Yazı tipi ayarının kaldırılması gereken BCP-47 script kodu. |

### Örnekler

Bu örnek, İbranice betiği için yazı tipi eşlemesini nasıl kaldıracağınızı gösterir:

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.RemoveScriptFont("Hebr");
```

## IFonts.SetScriptFont metodu

Belirli bir script etiketine bir yazı tipi adı atar; bu, o scriptin metninin sunumda nasıl görüntüleneceğini tanımlar.

```csharp
public void SetScriptFont(string script, string fontName)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| script | String | BCP-47 script kodu (örn., "Arab", "Hebr", "Hans") yazı sistemini tanımlar. |
| fontName | String | Belirtilen script için atanacak yazı tipinin adı. |

### Örnekler

Bu örnek, Arapça scripti için yazı tipini "Segoe UI" olarak ayarlamayı gösterir:

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segoe UI");
```

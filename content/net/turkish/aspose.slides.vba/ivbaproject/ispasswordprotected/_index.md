## IVbaProject.IsPasswordProtected özelliği

VBAProject'in proje özelliklerini görüntülemek için bir şifreyle korunduğunu gösterir. Salt okunur Boolean.

```csharp
public bool IsPasswordProtected { get; }
```

### Örnekler

```csharp
[C#]

{
    if (presentation.VbaProject.IsPasswordProtected)
        Console.WriteLine("The VBAProject '" + presentation.VbaProject.Name +

}
```

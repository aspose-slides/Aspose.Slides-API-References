## VbaProject.IsPasswordProtected özellik

VBAProject'in proje özelliklerini görüntülemek için bir parola ile korumalı olup olmadığını gösterir. Salt-okunur Boolean.

```csharp
public bool IsPasswordProtected { get; }
```

### Örnekler

```csharp
[C#]

{
    if (presentation.VbaProject.IsPasswordProtected)
        Console.WriteLine("The VBAProject '" + presentation.VbaProject.Name + 
            "' is protected by password to view project properties.");
}
```

## Timing.RepeatUntilNextClick özelliği

Bu öznitelik, etkinin bir sonraki tıklamaya kadar tekrar edip etmeyeceğini belirtir. Okunur/yazılır Boolean.

```csharp
public bool RepeatUntilNextClick { get; set; }
```

### Örnekler

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // İlk slayt için efektler sırasını alın
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Ana sıranın ilk efektini alın.
    IEffect effect = effectsSequence[0];

    // Efektin Timing/Repeat özelliğini "Bir Sonraki Tıklamaya Kadar" olarak değiştir
    effect.Timing.RepeatUntilNextClick = true;
}
```

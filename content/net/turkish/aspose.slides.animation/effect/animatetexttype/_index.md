## Effect.AnimateTextType özelliği

Efekt için bir animasyon metni türü tanımlar. Şekil metni harf, kelime bazında veya tüm olarak animasyonlandırılabilir. Okunur/Yazılır `AnimateTextType`.

```csharp
public AnimateTextType AnimateTextType { get; set; }
```

### Örnekler

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // İlk slaytın ilk efektini al.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Efektin AnimateTextType değerini "Harfe göre" olarak değiştir.
    firstSlideEffect.AnimateTextType = AnimateTextType.ByLetter;
}
```

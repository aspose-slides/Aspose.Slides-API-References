## Effect.DelayBetweenTextParts özelliği

Animasyonlu metin parçaları (kelimeler veya harfler) arasında bir gecikme tanımlar. Pozitif bir değer, etki süresinin yüzdesini belirtir. Negatif bir değer, gecikmeyi saniye cinsinden belirtir. Okuma/yazma Single.

```csharp
public float DelayBetweenTextParts { get; set; }
```

### Örnekler

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // İlk slaydın ilk efektini al.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Efektin AnimateTextType özelliğini "By word" olarak değiştir.
    firstSlideEffect.AnimateTextType = AnimateTextType.ByWord;
    
    // Animasyonlu metin parçaları arasındaki gecikmeyi efekt süresinin %20'si olarak ayarla.
    firstSlideEffect.DelayBetweenTextParts = 20f;
}
```

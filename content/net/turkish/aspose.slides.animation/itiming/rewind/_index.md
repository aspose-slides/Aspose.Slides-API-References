## ITiming.Rewind özelliği

Bu öznitelik, efektin oynatılması tamamlandığında yeniden sarmalanıp sarmalanmayacağını belirler. Okunur/yazılabilir Boolean.

```csharp
public bool Rewind { get; set; }
```

### Örnekler

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // İlk slayt için efekt dizisini al
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Ana dizinin ilk efektini al.
    IEffect effect = effectsSequence[0];

    // Efektin Timing/Rewind özelliğini etkinleştir.
    effect.Timing.Rewind = true;
}
```

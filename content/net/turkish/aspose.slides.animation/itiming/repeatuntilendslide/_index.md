## ITiming.RepeatUntilEndSlide özelliği

Bu öznitelik, etkinin slaytın sonuna kadar tekrar edip etmeyeceğini belirtir. Okunur/Yazılabilir Boolean.

```csharp
public bool RepeatUntilEndSlide { get; set; }
```

### Örnekler

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // İlk slayt için efekt dizisini al
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Ana sekansın ilk efektini al.
    IEffect effect = effectsSequence[0];

    // Efektin Timing/Repeat özelliğini "Slayt Sonuna Kadar" olarak değiştir
    effect.Timing.RepeatUntilEndSlide = true;
}
```

## BehaviorPropertyCollection.CopyTo yöntemi

ICollection öğelerinin bir Array'e kopyalanmasını sağlar, belirli bir Array dizininden başlayarak.

```csharp
public void CopyTo(IBehaviorProperty[] array, int arrayIndex)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| array | IBehaviorProperty[] | ICollection'dan kopyalanan öğelerin hedefi olan tek boyutlu Array. Array'in sıfır tabanlı indekslemesi olmalıdır. |
| arrayIndex | Int32 | Kopyalamanın başladığı *array* içindeki sıfır tabanlı indeks. |

## ToPng(Presentation, string) {#topng}

Giriş sunumunu PNG formatında bir dizi görüntüye dönüştürür. Çıktı dosya adı "myPath/myFilename.png" olarak verilirse, sonuç "myPath/myFilename_N.png" şeklinde bir dizi dosyaya kaydedilir; burada N slayt numarasıdır.

```csharp
public static void ToPng(Presentation pres, string outputFileName)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| pres | Presentation | Giriş sunumu. |
| outputFileName | String | Çıktı dosya adı. |

### İstisnalar

| İstisna | koşul |
| --- | --- |
| ArgumentException |  |

### Örnekler

```csharp

   Convert.ToPng(pres, "presImage.png");
```

```csharp
public static void ToPng(Presentation pres, string outputFileName, float scale, 
    IRenderingOptions options)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| pres | Presentation | Giriş sunumu. |
| outputFileName | String | Çıktı dosya adı. |

```csharp

   Convert.ToPng(pres, "presImage.png", 2f, options);
```

## Paragraph(Presentation, ForEachParagraphCallback) {#paragraph_1}

Her bir `Paragraph`'ı [`Presentation`](../../../aspose.slides/presentation) içinde yineleyin. Şekiller tüm slayt türlerinde yineleyecek - [`Slide`](../slide), [`MasterSlide`](../masterslide) ve [`LayoutSlide`](../layoutslide)

```csharp
public static void Paragraph(Presentation pres, ForEachParagraphCallback forEachParagraph)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| pres | Presentation | Paragrafları yinelemek için Presentation |
| forEachParagraph | ForEachParagraphCallback | Her paragraf için çağrılacak geri çağırma |

### Örnekler

```csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    ForEach.Paragraph(pres, (para, slide, index) =>
    {
        System.Console.WriteLine($"{para.Text}, index: {index}");
    });
}        
```

## ForEach.Slide yöntemi

Belirtilen [`Presentation`](../../../aspose.slides/presentation) içinde her bir `Slide` öğesini yineleyin.

```csharp
public static void Slide(Presentation pres, ForEachSlideCallback forEachSlide)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| pres | Presentation | Slaytları yinelemek için sunum |
| forEachSlide | ForEachSlideCallback | Her slayt için çağrılacak geri arama |

### Örnekler

```csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    ForEach.Slide(pres, (slide, index) =>
    {
        slide.Name = $"Slide #{index}";
    });
} 
```

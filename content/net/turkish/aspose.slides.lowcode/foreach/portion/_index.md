## Portion(Presentation, ForEachPortionCallback) {#portion_1}

Belirtilen [`Presentation`](../../../aspose.slides/presentation) içinde her `Portion` öğesini yineleyin. Portions tüm slayt türlerinde yineleyecektir - [`Slide`](../slide), [`MasterSlide`](../masterslide) ve [`LayoutSlide`](../layoutslide)

```csharp
public static void Portion(Presentation pres, ForEachPortionCallback forEachPortion)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| pres | Presentation | Portionları yinelemek için Presentation |
| forEachPortion | ForEachPortionCallback | Her portion için çağrılacak geri çağırma |

### Örnekler

```csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    ForEach.Portion(pres, (portion, para, slide, index) =>
    {
        System.Console.WriteLine($"{portion.Text}, index: {index}");
    });
```

## Portion(Presentation, bool, ForEachPortionCallback) {#portion}

Portions tüm slayt türlerinde yineleyecektir - [`Slide`](../slide), [`MasterSlide`](../masterslide), [`LayoutSlide`](../layoutslide) ve [`NotesSlide`](../../../aspose.slides/notesslide)

```csharp
public static void Portion(Presentation pres, bool includeNotes, 
    ForEachPortionCallback forEachPortion)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| pres | Presentation | Portionları yinelemek için Presentation |
| includeNotes | Boolean | İşleme sırasında NotesSlides'ın dahil edilip edilmeyeceğini gösteren işaret. |
| forEachPortion | ForEachPortionCallback | Her portion için çağrılacak geri çağırma |

### Örnekler

```csharp

    ForEach.Portion(pres, true, (portion, para, slide, index) =>
```

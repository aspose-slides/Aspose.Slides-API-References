## IPptxOptions.CompressionLevel özelliği

Sunum belgesini kaydederken kullanılan sıkıştırma seviyesini belirtir. Varsayılan değer Level6'dır.

```csharp
public CompressionLevel CompressionLevel { get; set; }
```

### Açıklamalar

Daha yüksek sıkıştırma seviyeleri daha küçük dosyalar üretir ancak daha fazla işleme süresi gerektirir. Gerçek sıkıştırma oranı sunumun içeriğine bağlıdır.

### Örnekler

Örnek:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("demo-level8.pptx", SaveFormat.Pptx, new PptxOptions()
    {
        CompressionLevel = CompressionLevel.Level8
    });
}
```

## PptxOptions.Zip64Mode özelliği

Presentation belgesi için ZIP64 formatının kullanılıp kullanılmadığını belirtir. Varsayılan değer IfNecessary'dir.

```csharp
public Zip64Mode Zip64Mode { get; set; }
```

### Örnekler

Örnek:

```csharp

{
    pres.Save("demo-zip64.pptx", SaveFormat.Pptx, new PptxOptions()
```

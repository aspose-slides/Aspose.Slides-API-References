## SwfOptions.SlidesLayoutOptions özelliği

Sunumu dışa aktarırken slaytların sayfaya yerleştirildiği modu alır veya ayarlar [`ISlidesLayoutOptions`](../../islideslayoutoptions). Bu özellik [`HandoutLayoutingOptions`](../../handoutlayoutingoptions) türündeki nesnelerin atanmasını desteklemez.

```csharp
public ISlidesLayoutOptions SlidesLayoutOptions { get; set; }
```

### Örnekler

Örnek:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    SwfOptions options = new SwfOptions
    {
        SlidesLayoutOptions = new NotesCommentsLayoutingOptions
        {
            CommentsPosition = CommentsPositions.Right
        }
    };
    
    pres.Save("pres.swf", SaveFormat.Swf, options);
}
```

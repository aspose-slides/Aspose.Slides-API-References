## SVGOptions.DisableFontLigatures özelliği

Metnin ligatürler kullanılmadan render edilip edilmediğini gösteren bir değeri alır veya ayarlar. `true` olarak ayarlandığında, ligatürler render edilmiş çıktıda devre dışı bırakılır. Varsayılan olarak bu özellik `false` olarak ayarlanmıştır.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Örnekler

Örnek:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    SVGOptions options = new SVFOptions
    {
        DisableFontLigatures = true // Metin render'ında ligatürleri devre dışı bırak
    };
    
    using (FileStream fileStream = new FileStream("slide-0.svg", FileMode.Create, FileAccess.Write))
    {
        pres.Slides[0].WriteAsSvg(fileStream);   
    }
}
```

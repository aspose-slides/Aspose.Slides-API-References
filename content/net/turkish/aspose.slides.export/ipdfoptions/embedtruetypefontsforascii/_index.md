## IPdfOptions.EmbedTrueTypeFontsForASCII özelliği

ASCII karakterleri 32-127 için True Type yazı tiplerini gömmek amacıyla True. 127'den büyük karakter kodları için yazı tipleri her zaman gömülür. Okunur/Yazılabilir Boolean.

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### Açıklamalar

Varsayılan **true**.

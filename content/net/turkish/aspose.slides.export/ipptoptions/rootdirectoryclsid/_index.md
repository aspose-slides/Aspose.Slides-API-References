## IPptOptions.RootDirectoryClsid özelliği

Kök dizin girdisinde saklanan nesne sınıfı GUID'sini (CLSID) temsil eder. Belgenin uygulamasının COM etkinleştirmesi için kullanılabilir. Varsayılan değer, '64818D11-4F9B-11CF-86EA-00AA00B929E8' olup 'Microsoft Powerpoint.Slide.8' değerine karşılık gelir.

```csharp
public Guid RootDirectoryClsid { get; set; }
```

### Örnekler

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    PptOptions pptOptions = new PptOptions();
    
    /// CLSID'i 'Microsoft Powerpoint.Show.8' olarak ayarla
    pptOptions.RootDirectoryClsid = new Guid("64818D10-4F9B-11CF-86EA-00AA00B929E8");
    
    pres.Save("pres.ppt", SaveFormat.Ppt, pptOptions);
}
```

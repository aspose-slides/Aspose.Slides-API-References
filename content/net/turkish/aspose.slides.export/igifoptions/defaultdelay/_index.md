## IGifOptions.DefaultDelay özelliği

Varsayılan gecikme süresini [ms] alır veya ayarlar. Bu değer, [`AdvanceAfterTime`](../../../aspose.slides/islideshowtransition/advanceaftertime) ayarlanmamışsa kullanılır. Varsayılan değer 1000'dir.

```csharp
public int DefaultDelay { get; set; }
```

### Örnekler

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
	pres.Save("pres.gif", SaveFormat.Gif, new GifOptions { DefaultDelay = 2000 });
}
```

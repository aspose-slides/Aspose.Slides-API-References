## HandoutLayoutingOptions.PrintFrameSlide özelliği

Belirtilen slaytlar etrafında çerçeve çizilip çizilmeyeceğini belirtir.

```csharp
public bool PrintFrameSlide { get; set; }
```

### Açıklamalar

Varsayılan değer **true**.

### Örnekler

Örnek:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    RenderingOptions options = new RenderingOptions
    {
        SlidesLayoutOptions = new HandoutLayoutingOptions
        {
            Handout = HandoutType.Handouts4Horizontal,
            PrintFrameSlide = false
        }
    };
    
    pres.Slides[0].GetThumbnail(options, new Size(1920, 1080)).Save("pres-handout.png");
}
```

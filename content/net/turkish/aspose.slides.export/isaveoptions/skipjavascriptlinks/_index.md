## ISaveOptions.SkipJavaScriptLinks özelliği

Sunumu kaydederken JavaScript çağrıları içeren köprüleri atlayıp atlamayacağını belirtir. Read/write Boolean. Varsayılan değer **false**'dur.

```csharp
public bool SkipJavaScriptLinks { get; set; }
```

### Açıklamalar

Bu özellik **true** olarak ayarlandığında, JavaScript çağrıları içeren köprüler kaydetme sırasında yok sayılacaktır.

Bu özellik **false** olarak ayarlandığında, tüm köprüler kaydedilecektir.

### Örnekler

Example:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("result_without_JavaScript_links.html", SaveFormat.Html, new HtmlOptions()
    {
        SkipJavaScriptLinks = true
    });
}
```

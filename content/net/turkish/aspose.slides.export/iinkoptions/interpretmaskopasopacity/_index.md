## IInkOptions.InterpretMaskOpAsOpacity özelliği

Fırça işleme için ROP işlemi veya opaklık kullanır.

```csharp
public bool InterpretMaskOpAsOpacity { get; set; }
```

### Açıklamalar

Varsayılan değer true'dur.

### Örnekler

Aşağıdaki örnek, Ink öğelerini dışa aktarırken ROP kullanarak nasıl ayarlanacağını gösterir:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.InterpretMaskOpAsOpacity = false;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

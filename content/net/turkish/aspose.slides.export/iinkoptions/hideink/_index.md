## IInkOptions.HideInk özelliği

İhracat edilen belgede Ink öğelerini gösterir veya gizler.

```csharp
public bool HideInk { get; set; }
```

### Açıklamalar

Varsayılan değer false'tur.

### Örnekler

Aşağıdaki örnek, ihraç edilen PDF belgesinde Ink öğelerinin nasıl gizleneceğini göstermektedir:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.HideInk = true;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

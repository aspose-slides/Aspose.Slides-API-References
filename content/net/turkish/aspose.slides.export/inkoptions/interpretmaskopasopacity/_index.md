## InkOptions.InterpretMaskOpAsOpacity özelliği

Fırça oluşturulması için ROP işlemini veya Opacity'yi kullanır.

```csharp
public bool InterpretMaskOpAsOpacity { get; set; }
```

### Açıklama

Varsayılan değer true'tir.

### Örnekler

Aşağıdaki örnek, Ink öğelerini dışa aktarmak için ROP kullanarak nasıl ayarlanacağını gösterir:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.InterpretMaskOpAsOpacity = false;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

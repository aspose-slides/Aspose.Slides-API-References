## PresentationPlayer.FrameTick olayı

Frame tick olayı. Bu olay, [`PresentationAnimationsGenerator`](../../presentationanimationsgenerator) tarafından oluşturulan animasyonun her çerçevesi oynatıcı tarafından üretildiğinde meydana gelir.

```csharp
public event FrameTickHandler FrameTick;
```

### Örnekler

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (var animationsGenerator = new PresentationAnimationsGenerator(pres))
    using (var player = new PresentationPlayer(animationsGenerator, 33))
    {
        player.FrameTick += (sender, args) =>
        {
            args.GetFrame().Save($"frame_{sender.FrameIndex}.png");
        };
        
        animationsGenerator.Run(pres.Slides);
    }
}
```

## HtmlOptions.DisableFontLigatures özelliği

Metnin ligatürler kullanılmadan render edilip edilmediğini gösteren bir değeri alır veya ayarlar. `true` olarak ayarlandığında, ligatürler render edilmiş çıktıda devre dışı bırakılacaktır. Varsayılan olarak, bu özellik `false` olarak ayarlanmıştır.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Örnekler

Örnek:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    HtmlOptions options = new HtmlOptions
    {
        DisableFontLigatures = true // Metin render'ında ligatürleri devre dışı bırak
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html, options);
}
```

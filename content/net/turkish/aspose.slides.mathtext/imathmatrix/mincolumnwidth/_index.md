## IMathMatrix.MinColumnWidth özelliği

Twip cinsinden minimum sütun genişliği (bir puanın 1/20'si). Boşluk aralığı (“Column Gap” veya “Gap Width” olarak da adlandırılır), MinColumnWidth'e eklenerek toplam Matrix Column Spacing'i (farklı sütunların aynı kenarları arasındaki mesafe) belirler. Varsayılan: 0.

```csharp
public uint MinColumnWidth { get; set; }
```

### Örnekler

Örnek:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.MinColumnWidth = 20;
```

## IMathBorderBox.StrikethroughTopLeftToBottomRight özelliği

Üst Sol'dan Alt Sağ'a Çizgi Üstü (varsayılan false). Kenar kutusunun üst sol köşesinden alt sağ köşesine doğru çapraz bir çizgi üstü çizgisinin gizli ya da gösterilen durumunu belirtir.

```csharp
public bool StrikethroughTopLeftToBottomRight { get; set; }
```

### Örnekler

Örnek:

```csharp
[C#]
IMathBorderBox borderBox = new MathematicalText("x+y+z").ToBorderBox();
borderBox.StrikethroughTopLeftToBottomRight = true;
```

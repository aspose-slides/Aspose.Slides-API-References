## IMathBorderBox.StrikethroughBottomLeftToTopRight özelliği

Alt Sol'dan Üst Sağ'a üzeri çizili (varsayılan false). Kenarlık kutusunun alt-sol köşesinden üst-sağ köşesine doğru çapraz bir üzeri çizili satırın gizli veya gösterilir durumunu belirtir.

```csharp
public bool StrikethroughBottomLeftToTopRight { get; set; }
```

### Örnekler

Örnek:

```csharp
[C#]
IMathBorderBox borderBox = new MathematicalText("x+y+z").ToBorderBox();
borderBox.StrikethroughBottomLeftToTopRight = true;
```

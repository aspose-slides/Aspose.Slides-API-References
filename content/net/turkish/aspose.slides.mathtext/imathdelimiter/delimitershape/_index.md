## IMathDelimiter.DelimiterShape özelliği

Delimitör nesnesindeki sınırlayıcıların şeklini belirtir. MathDelimiterShape.Centered olduğunda, sınırlayıcılar matematik metninin matematik ekseni etrafında ortalanır ve içeriklerinin tüm yüksekliğine sığacak şekilde yapılabilir. MathDelimiterShape.Match olduğunda ise, yüksekliği ve şekli içeriklerine tam olarak uyması için değiştirilir.

```csharp
public MathDelimiterShape DelimiterShape { get; set; }
```

### Örnekler

Örnek:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.DelimiterShape = MathDelimiterShape.Match;
```

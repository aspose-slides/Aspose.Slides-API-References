## IMathDelimiter.GrowToMatchOperandHeight özelliği

BeginningCharacter, SeparatorCharacter, EndingCharacter'in büyümesini belirler. true olduğunda, sınırlayıcılar yüksekliğine uyacak şekilde dikey olarak büyür. Varsayılan değer true'dır.

```csharp
public bool GrowToMatchOperandHeight { get; set; }
```

### Örnekler

Example:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.GrowToMatchOperandHeight = false;
```

## Group(char, MathTopBotPositions, MathTopBotPositions) {#group_1}

Bu öğeyi grup karakteri (ör. alt kıvrım parantezi veya başka bir karakter) kullanarak bir gruba yerleştirir

```csharp
public IMathGroupingCharacter Group(char character, MathTopBotPositions position, 
    MathTopBotPositions verticalJustification)
```

| Parameter | Type | Description |
| --- | --- | --- |
| character | Char | BOTTOM CURLY BRACKET (U+23DF) gibi bir Gruplama Karakteri veya başka bir karakter |
| position | MathTopBotPositions | Gruplama karakterinin konumu |
| verticalJustification | MathTopBotPositions | Grup karakterinin dikey hizalaması. Nesnenin taban çizgisine göre hizalanmasını belirtir. Örneğin, grup karakteri nesnenin üstünde olduğunda, VerticalJustification değerinin Top olması nesnenin üst kısmının taban çizgisinde olacağını, VerticalJustification değerinin Bottom olması nesnenin alt kısmının taban çizgisinde olacağını gösterir. |

### Dönüş Değeri

[`IMathGroupingCharacter`](../../imathgroupingcharacter) tipinde yeni bir örnek

### Örnekler

Örnek:

```csharp
[C#]
IMathGroupingCharacter groupingElement = new MathematicalText("x;y;z").Group('\u23E1', MathTopBotPositions.Bottom, MathTopBotPositions.Top);
```

## IMathElementCollection.Remove metot

Koleksiyondan belirli bir nesnenin ilk örneğini kaldırır.

```csharp
public bool Remove(IMathElement item)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| item | IMathElement | Koleksiyondan kaldırılacak nesne. |

### Dönüş Değeri

true if *item* was successfully removed from the collection; otherwise, false. Bu metot ayrıca *item* orijinal koleksiyonda bulunamazsa false döndürür.

### Örnekler

Örnek:

```csharp
[C#]
IMathElementCollection collection = new MathBlock(new MathematicalText("x"));
MathematicalText plusElement = new MathematicalText("+");
collection.Add(plusElement);
collection.Add(new MathRadical(new MathematicalText("x"), new MathematicalText("3")));
collection.Remove(plusElement);
```

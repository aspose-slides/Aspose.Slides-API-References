## IMathBox.ExplicitBreak özelliği

Açık kesme, Box nesnesinin başlangıcında bir satır sonu olup olmadığını belirler; böylece satır, kutu nesnesinin başlangıcında sarılır. Matematiksel metnin bir önceki satırındaki operatörün sayısını belirler; bu sayı, mevcut matematiksel metin satırı için hizalama noktası olarak kullanılacaktır. olası değerler: 1..255 Varsayılan: 0 (açık kesme yok)

```csharp
public byte ExplicitBreak { get; set; }
```

### Örnekler

Örnek:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.ExplicitBreak = 1;
```

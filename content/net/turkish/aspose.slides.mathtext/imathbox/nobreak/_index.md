## IMathBox.NoBreak özelliği

Kırılma yok. Bu özellik, nesne kutusundaki "kırılmaz" özelliğini belirtir. true olduğunda, kutu içinde satır sonları oluşamaz. Bu, birden fazla ikili operatörden oluşan operatör öykünücülerinde önemli olabilir. Bu öğe belirtilmediğinde, kutu içinde satır sonları oluşabilir. Varsayılan: true

```csharp
public bool NoBreak { get; set; }
```

## IMathArray.RowSpacing özelliği

Bir dizinin satırları arasındaki boşluk. Yalnızca RowSpacingRule 3 Exactly olarak ayarlandığında kullanılır; bu durumda ölçü birimi nokta olur veya Multiple olarak ayarlandığında ölçü birimi yarı satır olur. Default: 0

```csharp
public uint RowSpacing { get; set; }
```

### Örnekler

Örnek:

```csharp
[C#]
IMathArray mathArray = new MathArray(new MathematicalText("item1"));
mathArray.RowSpacingRule = MathRowSpacingRule.Exactly;
mathArray.RowSpacing = 10;
```

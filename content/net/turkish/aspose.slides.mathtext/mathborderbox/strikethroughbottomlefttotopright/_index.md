## MathBorderBox.StrikethroughBottomLeftToTopRight özelliği

Alt Sol'dan Üst Sağ'a Çizgili (varsayılan olarak false). Kenar kutusunun alt sol köşesinden üst sağ köşesine uzanan çapraz bir çizgili satırın gizli ya da gösterilen durumunu belirtir.

```csharp
public bool StrikethroughBottomLeftToTopRight { get; set; }
```

### Örnekler

Örnek:

```csharp
[C#]
MathBorderBox borderBox = new MathBorderBox(new MathematicalText("x"));
borderBox.StrikethroughBottomLeftToTopRight = true;
```

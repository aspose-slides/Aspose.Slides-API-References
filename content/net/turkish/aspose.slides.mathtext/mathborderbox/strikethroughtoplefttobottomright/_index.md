## MathBorderBox.StrikethroughTopLeftToBottomRight özelliği

Üst Sol'dan Alt Sağ'a Çizgi (varsayılan false). Kenar kutusunun üst sol köşesinden alt sağ köşesine giden çapraz çizginin gizli veya gösterilen durumunu belirtir.

```csharp
public bool StrikethroughTopLeftToBottomRight { get; set; }
```

### Örnekler

Örnek:

```csharp
[C#]
MathBorderBox borderBox = new MathBorderBox(new MathematicalText("x"));
borderBox.StrikethroughTopLeftToBottomRight = true;
```

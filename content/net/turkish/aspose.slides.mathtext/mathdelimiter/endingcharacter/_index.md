## MathDelimiter.EndingCharacter özelliği

Delimiter Ending Character sonlandırma veya kapanış sınırlama karakterini belirtir. Matematiksel sınırlayıcılar, parantezler, köşeli parantezler ve süslü parantezler gibi çevreleyen karakterlerdir. Varsayılan: ')'.

```csharp
public char EndingCharacter { get; set; }
```

Örnek:

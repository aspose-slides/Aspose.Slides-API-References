## MathDelimiter.GrowToMatchOperandHeight özelliği

BeginningCharacter, SeparatorCharacter, EndingCharacter öğelerinin büyümesini belirtir. True olduğunda, sınırlayıcılar yüksekliğine göre dikey olarak büyür. Varsayılan değer true’tur.

```csharp
public bool GrowToMatchOperandHeight { get; set; }
```

### Örnekler

Örnek:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.GrowToMatchOperandHeight = false;
```

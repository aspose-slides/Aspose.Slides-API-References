## MathMatrix.MinColumnWidth özelliği

Twip cinsinden minimum sütun genişliği (1 puanın 1/20'si). Boşluk aralığı (“Column Gap” veya “Gap Width” olarak da adlandırılır), toplam Matrix Column Spacing'i belirlemek için MinColumnWidth'e eklenir (farklı sütunların aynı kenarları arasındaki mesafe). Varsayılan: 0.

```csharp
public uint MinColumnWidth { get; set; }
```

### Örnekler

Example:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.MinColumnWidth = 20;
```

## MathMatrix.ColumnGap özelliği

Bir matrisin sütunları arasındaki yatay boşluğun değeridir; ColumnGapRule 3 ("Exactly") olarak ayarlanmışsa birim twip (bir noktanın 1/20'i) olarak yorumlanır. ColumnGapRule 4 ("Multiple") olarak ayarlanmışsa birim 0.5 em artışının sayısı olarak yorumlanır. Diğer durumlarda yok sayılır. Varsayılan: 0

```csharp
public uint ColumnGap { get; set; }
```

### Örnekler

Örnek:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.ColumnGapRule = MathSpacingRules.Exactly;
matrix.ColumnGap = 20;
```

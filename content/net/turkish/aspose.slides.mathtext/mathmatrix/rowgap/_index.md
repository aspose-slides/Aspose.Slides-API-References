## MathMatrix.RowGap özelliği

Bir matrisin satırları arasındaki dikey boşluk değeri; RowGapRule 3 ("Exactly") olarak ayarlanmışsa, birim twip (bir noktanın 1/20'si) olarak yorumlanır. RowGapRule 4 ("Multiple") olarak ayarlanmışsa, birim yarım satır olarak yorumlanır. Varsayılan: 0

```csharp
public uint RowGap { get; set; }
```

### Örnekler

Örnek:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
```

## MathArray.RowSpacing özelliği

Bir dizinin satırları arasındaki boşluk. Bu yalnızca RowSpacingRule 3 olarak ayarlandığında kullanılır. Tam olarak bu durumda ölçü birimi nokta olur veya Multiple olduğunda ölçü birimi yarı satır olur. Varsayılan: 0

```csharp
public uint RowSpacing { get; set; }
```

### Örnekler

Example:

```csharp
[C#]
IMathArray mathArray = new MathArray(new MathematicalText("item1"));
mathArray.RowSpacingRule = MathRowSpacingRule.Exactly;
mathArray.RowSpacing = 10;
```

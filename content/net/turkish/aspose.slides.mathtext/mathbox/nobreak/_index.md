## MathBox.NoBreak özelliği

Kesinti yok Bu özellik, nesne kutusundaki "unbreakable" özelliğini belirler. true olduğunda, kutunun içinde satır sonları oluşamaz. Bu, birden fazla ikili operatörden oluşan operatör emülatörleri için önemli olabilir. Bu öğe belirtilmediğinde, kutunun içinde satır sonları oluşabilir. Varsayılan: true

```csharp
public bool NoBreak { get; set; }
```

[C#]

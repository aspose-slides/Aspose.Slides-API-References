## MathBox.ExplicitBreak özelliği

Explicit break, Box nesnesinin başlangıcında bir satır sonu olup olmadığını belirler, böylece satır Box nesnesinin başlangıcında kayar. Önceki matematiksel metin satırındaki operatör sayısını belirtir; bu sayı, mevcut matematiksel metin satırı için hizalama noktası olarak kullanılacaktır. Olası değerler: 1..255 Varsayılan: 0 (no explicit break)

```csharp
public byte ExplicitBreak { get; set; }
```

### Örnekler

Örnek:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.ExplicitBreak = 1;
```

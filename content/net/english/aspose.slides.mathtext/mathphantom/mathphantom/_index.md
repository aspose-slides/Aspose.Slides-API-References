| Parameter | Type | Description |
| --- | --- | --- |
| element | IMathElement | The base [`IMathElement`](../../imathelement) whose visibility and layout will be controlled by the phantom. This element defines the content that may be hidden or shown, while still affecting the geometric alignment of the surrounding math. |

### Remarks

The phantom element is used to reserve or suppress the visual space of its base expression without necessarily displaying it. It corresponds to the OMML element `<m:phant>`.

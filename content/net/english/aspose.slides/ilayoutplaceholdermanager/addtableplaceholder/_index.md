## ILayoutPlaceholderManager.AddTablePlaceholder method

Adds a new placeholder shape to the layout slide to hold a table.

```csharp
public IAutoShape AddTablePlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| x | Single | The X coordinate of the new placeholder shape. |
| y | Single | The Y coordinate of the new placeholder shape. |
| width | Single | The width of the new placeholder shape. |
| height | Single | The height of the new placeholder shape. |

### Return Value

Created [`IAutoShape`](../../iautoshape) with a Table placeholder.

### Examples

The following example shows how to add the Table placeholder shape to the layout slide.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTablePlaceholder(20, 20, 500, 200);
}
```

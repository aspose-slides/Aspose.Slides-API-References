## IFonts.GetScriptFont method

Gets the font name associated with a specific script tag from the presentation theme.

```csharp
public string GetScriptFont(string script)
```

| Parameter | Type | Description |
| --- | --- | --- |
| script | String | The BCP-47 script code (e.g., "Latn", "Cyrl", "Jpan") used to identify a writing system. |

### Return Value

The name of the font used for the specified script, or `null` if the script is not defined.

### Examples

This example demonstrates how to retrieve the font assigned to the Cyrillic script in the presentation theme.

```csharp
[C#]
string font = presentation.MasterTheme.FontScheme.Major.GetScriptFont("Cyrl");
Console.WriteLine("Font for Cyrillic script: " + font);
```

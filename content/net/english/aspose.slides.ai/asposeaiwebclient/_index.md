## AsposeAIWebClient class

A built-in [`IAIWebClient`](../iaiwebclient) implementation that connects to Aspose's own LLM. This is the default client used by the parameterless [`SlidesAIAgent`](../slidesaiagent/slidesaiagent) constructor.

```csharp
public sealed class AsposeAIWebClient : IAIWebClient, IDisposable
```

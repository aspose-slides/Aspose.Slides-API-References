## Thuộc tính WebDocument.Output

Trả về tập hợp các phần tử output của tài liệu. Chỉ đọc `Output`.

```csharp
public Output Output { get; }
```

### Ví dụ

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    var options = new WebDocumentOptions
    {
        TemplateEngine = new RazorTemplateEngine(),
        OutputSaver = new FileOutputSaver(),
        EmbedImages = false
    };
    
    WebDocument document = new WebDocument(options);

    // thêm "index.html" vào các tệp output, sử dụng mẫu "index" để tạo nó và biến pres làm mô hình
    document.Output.Add("index.html", "index", pres);

    // ... thiết lập các tùy chọn khác của tài liệu và sau đó lưu tài liệu
    document.Save();
}
```

## WebDocument.Global thuộc tính

Trả về bộ lưu trữ toàn cục của tài liệu. Chỉ đọc [`Storage`](../../storage).

```csharp
public Storage Global { get; }
```

### Ví dụ

Sử dụng thuộc tính `Global` này (cài đặt của [`Storage`](../../storage) giao diện) có thể đặt một thuộc tính để sử dụng sau này trong mẫu:

```csharp
[C#]
var options = new WebDocumentOptions
{
    TemplateEngine = new RazorTemplateEngine(),
    OutputSaver = new FileOutputSaver(),
    EmbedImages = false
};

WebDocument document = new WebDocument(options);

// đặt thuộc tính "slideMargin" để sử dụng từ mẫu
document.Global.Put("slideMargin", 10);

// ... thiết lập các tùy chọn khác của tài liệu và sau đó lưu tài liệu
document.Save();
```

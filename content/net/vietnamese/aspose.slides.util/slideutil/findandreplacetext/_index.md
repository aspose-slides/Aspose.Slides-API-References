## SlideUtil.FindAndReplaceText method

Tìm và thay thế văn bản trong bản trình chiếu với định dạng cho trước

```csharp
public static void FindAndReplaceText(IPresentation presentation, bool withMasters, string find, 
    string replace, PortionFormat format = null)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| presentation | IPresentation | Bản trình chiếu được quét. |
| withMasters | Boolean | Xác định liệu các slide chủ đề có nên được quét hay không. |
| find | String | Giá trị chuỗi cần tìm. |
| replace | String | Giá trị chuỗi thay thế. |
| format | PortionFormat | Định dạng cho phần văn bản được thay thế. Nếu null thì sẽ sử dụng định dạng của ký tự đầu tiên trong chuỗi được tìm thấy |

### Ví dụ

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PortionFormat format = new PortionFormat
    {
        FontHeight = 24f,
        FontItalic = NullableBool.True,
        FillFormat =
        {
            FillType = FillType.Solid,
            SolidFillColor =
            {
                Color = Color.Red
            }
        }
    };
    Aspose.Slides.Util.SlideUtil.FindAndReplaceText(pres, true, "[this block] ", "my text ", format);
    pres.Save("replaced", SaveFormat.Pptx);
}
```

## Fonts.GetScriptFont method

Lấy tên phông chữ liên kết với thẻ script cụ thể từ chủ đề của bản trình chiếu.

```csharp
public string GetScriptFont(string script)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| script | String | Mã script BCP-47 (ví dụ: "Latn", "Cyrl", "Jpan") được dùng để xác định một hệ thống viết. |

### Giá trị trả về

Tên phông chữ được sử dụng cho script đã chỉ định, hoặc `null` nếu script không được định nghĩa.

### Ví dụ

Ví dụ này minh họa cách lấy phông chữ được gán cho script Cyrillic trong chủ đề bản trình chiếu.

```csharp
[C#]
string font = presentation.MasterTheme.FontScheme.Major.GetScriptFont("Cyrl");
Console.WriteLine("Font for Cyrillic script: " + font);
```

## Fonts.SetScriptFont phương thức

Gán tên phông chữ cho một thẻ script cụ thể, xác định cách văn bản của script đó sẽ được hiển thị trong bản trình chiếu.

```csharp
public void SetScriptFont(string script, string fontName)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| script | String | Mã script BCP-47 (ví dụ: "Arab", "Hebr", "Hans") xác định hệ thống viết. |
| fontName | String | Tên phông chữ sẽ được gán cho script đã chỉ định. |

### Ví dụ

Ví dụ này cho thấy cách đặt phông chữ cho script Arabic thành "Segoe UI":

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segoe UI");
```

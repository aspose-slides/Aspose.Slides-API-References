## Fonts.RemoveScriptFont phương thức

Xóa cài đặt phông chữ liên quan đến một thẻ script cụ thể khỏi bộ sưu tập phông chữ của theme.

```csharp
public void RemoveScriptFont(string script)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| script | String | Mã script BCP-47 mà cài đặt phông chữ cần được xóa. |

### Ví dụ

Ví dụ này minh họa cách xóa ánh xạ phông chữ cho script Hebrew:

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.RemoveScriptFont("Hebr");
```

### Ví dụ

Ví dụ sau đây cho thấy cách thêm một thẻ vào PowerPoint Presentation.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
   ITagCollection tags = pres.CustomData.Tags;
   pres.CustomData.Tags["MyTag"] = "My Tag Value";
}
```

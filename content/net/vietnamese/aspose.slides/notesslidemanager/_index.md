### Ví dụ

Ví dụ sau đây cho thấy cách Thêm ghi chú vào slide cụ thể của bản trình chiếu ProwerPoint.

```csharp
[C#]
	// Tạo một đối tượng Presentation đại diện cho tệp trình chiếu
	using(Presentation presentation = new Presentation(dataDir + "AccessSlides.pptx")) {
	  // Thêm ghi chú vào slide đầu tiên
	  INotesSlideManager mgr = presentation.Slides[0].NotesSlideManager;
	  INotesSlide noteSlide = mgr.AddNotesSlide();
	  noteSlide.NotesTextFrame.Text = "Your Notes";
	  // Lưu trình chiếu vào đĩa
	  presentation.Save(dataDir + "RemoveNotesAtSpecificSlide_out.pptx", SaveFormat.Pptx);
	}
```

Ví dụ sau đây cho thấy cách xóa ghi chú khỏi slide cụ thể của bản trình chiếu PowerPoint.

```csharp
[C#]
	// Tạo một đối tượng Presentation đại diện cho tệp trình chiếu
	using(Presentation presentation = new Presentation(dataDir + "AccessSlides.pptx")) {
	  // Xóa ghi chú của slide đầu tiên
	  INotesSlideManager mgr = presentation.Slides[0].NotesSlideManager;
	  mgr.RemoveNotesSlide();
	  // Lưu trình chiếu vào đĩa
	  presentation.Save(dataDir + "RemoveNotesAtSpecificSlide_out.pptx", SaveFormat.Pptx);
	}
```

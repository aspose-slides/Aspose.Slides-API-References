## IProtectionManager.CheckWriteProtection method

Xác định liệu bài thuyết trình có được bảo vệ bằng mật khẩu để sửa đổi hay không.

```csharp
public bool CheckWriteProtection(string password)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| password | String | Mật khẩu để kiểm tra. |

### Giá trị trả về

Đúng nếu mật khẩu hợp lệ; ngược lại, sai.

### Ghi chú

1. Bạn nên kiểm tra thuộc tính [`IsWriteProtected`](../iswriteprotected) trước khi gọi phương thức này. 2. Khi mật khẩu là null hoặc rỗng, phương thức này trả về sai.

### Ví dụ

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

## Phương thức GeometryShape.SetGeometryPaths

Cập nhật hình học của hình dạng từ mảng [`IGeometryPath`](../../igeometrypath). Các tọa độ phải tương đối với góc trên trái của hình dạng. Thay đổi loại của hình dạng ([`ShapeType`](../shapetype)) thành Custom.

```csharp
public void SetGeometryPaths(IGeometryPath[] geometryPaths)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| geometryPaths | IGeometryPath[] | Mảng các đường hình học |

### Ngoại lệ

| ngoại lệ | điều kiện |
| --- | --- |
| ArgumentException | Không tìm thấy đường dẫn |
| ArgumentException | Đường dẫn rỗng |

### Ví dụ

Ví dụ:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    GeometryShape shape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 200, 100) as GeometryShape;

    GeometryPath geometryPath = shape.GetGeometryPaths()[0];

    geometryPath.LineTo(100, 50, 1);
    geometryPath.LineTo(100, 50, 4);

    shape.SetGeometryPath(geometryPath);

    pres.Save("output.pptx", SaveFormat.Pptx);
}
```

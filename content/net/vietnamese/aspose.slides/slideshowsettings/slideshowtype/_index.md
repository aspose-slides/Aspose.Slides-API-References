## SlideShowSettings.SlideShowType thuộc tính

Lấy hoặc thiết lập kiểu trình chiếu. Được biểu diễn bởi các tổ tiên `SlideShowType` sau: [`BrowsedAtKiosk`](../../browsedatkiosk), [`PresentedBySpeaker`](../../presentedbyspeaker) và [`BrowsedByIndividual`](../../browsedbyindividual)

```csharp
public SlideShowType SlideShowType { get; set; }
```

### Ví dụ

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // để đặt loại "Browsed at a kiosk (full screen)" 
    pres.SlideShowSettings.SlideShowType = new BrowsedAtKiosk();
    
    // để đặt loại "Browsed by individual (window)" 
    pres.SlideShowSettings.SlideShowType = new BrowsedByIndividual();
    
    // để đặt loại "Presented by a speaker (full screen)" 
    pres.SlideShowSettings.SlideShowType = new PresentedBySpeaker();
}
```

## Slide.WriteAsEmf phương thức

Lưu nội dung slide dưới dạng tệp EMF.

```csharp
public void WriteAsEmf(Stream stream)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| stream | Stream | Luồng đích |

### Ngoại lệ

| Ngoại lệ | Điều kiện |
| --- | --- |
| ArgumentNullException | Luồng đích là `null` |

### Ví dụ

Ví dụ mã sau minh họa cách chuyển đổi slide đầu tiên trong một bản trình bày PowerPoint thành một metafile.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.emf"))
    {
        // Lưu slide đầu tiên dưới dạng metafile
        pres.Slides[0].WriteAsEmf(fileStream);
    }
}
```

## Control.Persistence thuộc tính

Lấy phương thức được sử dụng để lưu các thuộc tính của điều khiển ActiveX. Chỉ đọc [`PersistenceType`](../../persistencetype).

```csharp
public PersistenceType Persistence { get; }
```

### Ví dụ

Ví dụ tiếp theo cho thấy việc sử dụng Persistence thuộc tính để kiểm tra xem các thuộc tính của đối tượng ActiveX có thể được thay đổi dưới dạng các thuộc tính ActiveX dựa trên XML hay không:

```csharp
[C#]
if (control.Persistence == PersistenceType.PersistPropertyBag)
{
    control.Properties["Value"] = value;
}
else
{
    YourMethodHere(control.ActiveXControlBinary); //Sử dụng phương thức của riêng bạn để quản lý các thuộc tính ActiveX được lưu trong tệp nhị phân của nó
}
```

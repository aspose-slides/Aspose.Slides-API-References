## IShapeCollection.AddSummaryZoomFrame method

Tạo một khung Summary Zoom mới và thêm nó vào cuối bộ sưu tập shape.

```csharp
public ISummaryZoomFrame AddSummaryZoomFrame(float x, float y, float width, float height)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| x | Single | Tọa độ x của khung Summary Zoom mới, tính bằng point. |
| y | Single | Tọa độ y của khung Summary Zoom mới, tính bằng point. |
| width | Single | Chiều rộng của khung Summary Zoom mới, tính bằng point. |
| height | Single | Chiều cao của khung Summary Zoom mới, tính bằng point. |

### Ghi chú

Phương thức này tạo một khung Summary Zoom tổng hợp các liên kết tóm tắt cho tất cả các phần trong bản trình chiếu.

### Ví dụ

Ví dụ này minh họa cách thêm một đối tượng Summary Zoom vào cuối bộ sưu tập (giả sử bản trình chiếu "Presentation.pptx" có ít nhất hai phần):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSummaryZoomFrame(150, 20, 500, 250);
}
```

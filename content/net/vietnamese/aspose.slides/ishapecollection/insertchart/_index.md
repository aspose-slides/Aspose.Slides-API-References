## InsertChart(ChartType, float, float, float, float, int) {#insertchart}

Tạo một biểu đồ mới, khởi tạo nó với dữ liệu và cài đặt mẫu, và chèn nó vào bộ sưu tập shape tại chỉ mục đã chỉ định.

```csharp
public IChart InsertChart(ChartType type, float x, float y, float width, float height, int index)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| type | ChartType | Kiểu biểu đồ cần tạo. |
| x | Single | Tọa độ x của biểu đồ mới, tính bằng points. |
| y | Single | Tọa độ y của biểu đồ mới, tính bằng points. |
| width | Single | Độ rộng của biểu đồ mới, tính bằng points. |
| height | Single | Độ cao của biểu đồ mới, tính bằng points. |
| index | Int32 | Chỉ mục dựa trên số 0 tại vị trí chèn biểu đồ mới trong bộ sưu tập shape. |

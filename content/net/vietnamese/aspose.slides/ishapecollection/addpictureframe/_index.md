## IShapeCollection.AddPictureFrame phương thức

Tạo một khung hình ảnh mới chứa hình ảnh đã chỉ định và thêm nó vào cuối bộ sưu tập hình dạng.

```csharp
public IPictureFrame AddPictureFrame(ShapeType shapeType, float x, float y, float width, 
    float height, IPPImage image)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| shapeType | ShapeType | Xác định loại hình dạng được chứa trong [`ShapeType`](../../shapetype), ngoại trừ mọi loại đường thẳng: |
| x | Single | Tọa độ x của khung hình ảnh, tính bằng điểm. |
| y | Single | Tọa độ y của khung hình ảnh, tính bằng điểm. |
| width | Single | Chiều rộng của khung hình ảnh, tính bằng điểm. |
| height | Single | Chiều cao của khung hình ảnh, tính bằng điểm. |
| image | IPPImage | [`IPPImage`](../../ippimage) để hiển thị trong khung hình ảnh. |

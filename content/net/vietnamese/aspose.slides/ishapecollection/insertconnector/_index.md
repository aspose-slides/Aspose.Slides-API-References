## InsertConnector(int, ShapeType, float, float, float, float) {#insertconnector}

public IConnector InsertConnector(int index, ShapeType shapeType, float x, float y, float width,

| y | Single | Tọa độ y của khung connector, tính bằng điểm. |
| width | Single | Chiều rộng của khung connector, tính bằng điểm. |

## InsertConnector(int, ShapeType, float, float, float, float, bool) {#insertconnector_1}

Tạo một hình connector mới và chèn nó vào bộ sưu tập hình tại chỉ mục được chỉ định, tùy chọn áp dụng kiểu mẫu mặc định.

```csharp
public IConnector InsertConnector(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| index | Int32 | Chỉ mục bắt đầu từ 0 tại vị trí sẽ chèn hình connector. |
| shapeType | ShapeType | [`ShapeType`](../../shapetype) của hình connector sẽ chèn. |
| x | Single | Tọa độ x của khung connector, tính bằng điểm. |
| y | Single | Tọa độ y của khung connector, tính bằng điểm. |
| width | Single | Chiều rộng của khung connector, tính bằng điểm. |
| height | Single | Chiều cao của khung connector, tính bằng điểm. |
| createFromTemplate | Boolean | True để áp dụng kiểu mẫu mặc định (tên không rỗng, kiểu đơn giản); false để tạo connector với các giá trị thuộc tính mặc định. |

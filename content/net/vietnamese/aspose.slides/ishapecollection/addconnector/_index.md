```csharp
public IConnector AddConnector(ShapeType shapeType, float x, float y, float width, float height, 
    bool createFromTemplate)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| shapeType | ShapeType | [`ShapeType`](../../shapetype) của hình connector cần tạo. |
| x | Single | Tọa độ x của khung connector, tính bằng điểm. |
| y | Single | Tọa độ y của khung connector, tính bằng điểm. |
| width | Single | Chiều rộng của khung connector, tính bằng điểm. |
| height | Single | Chiều cao của khung connector, tính bằng điểm. |

## IShapeCollection.InsertSummaryZoomFrame method

Tạo một khung Summary Zoom mới và chèn nó vào bộ sưu tập shape tại chỉ mục được chỉ định.

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| index | Int32 | Chỉ mục bắt đầu từ 0 tại vị trí sẽ chèn khung Summary Zoom. |
| x | Single | Tọa độ x của khung Summary Zoom mới, tính bằng điểm. |
| y | Single | Tọa độ y của khung Summary Zoom mới, tính bằng điểm. |
| width | Single | Chiều rộng của khung Summary Zoom mới, tính bằng điểm. |
| height | Single | Chiều cao của khung Summary Zoom mới, tính bằng điểm. |

### Ghi chú

Phương thức này tạo một khung Summary Zoom tổng hợp các liên kết tóm tắt cho tất cả các phần trong bản trình chiếu.

### Ví dụ

Ví dụ này minh họa cách tạo và chèn một đối tượng Summary Zoom vào chỉ mục được chỉ định của một bộ sưu tập (giả sử có ít nhất hai phần trong bản trình chiếu "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSummaryZoomFrame(2, 150, 20, 50, 50);
}
```

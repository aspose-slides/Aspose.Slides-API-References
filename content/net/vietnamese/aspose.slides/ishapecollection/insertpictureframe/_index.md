## IShapeCollection.InsertPictureFrame method

Tạo một khung ảnh mới chứa hình ảnh được chỉ định và chèn nó vào bộ sưu tập hình dạng tại vị trí chỉ mục được chỉ định.

```csharp
public IPictureFrame InsertPictureFrame(int index, ShapeType shapeType, float x, float y, 
    float width, float height, IPPImage image)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| index | Int32 | Chỉ mục bắt đầu từ 0 tại vị trí sẽ chèn khung ảnh. |
| shapeType | ShapeType | Xác định kiểu hình dạng chứa trong [`ShapeType`](../../shapetype), ngoại trừ mọi loại đường thẳng: |
| x | Single | Tọa độ x của khung ảnh, tính bằng điểm. |
| y | Single | Tọa độ y của khung ảnh, tính bằng điểm. |
| width | Single | Chiều rộng của khung ảnh, tính bằng điểm. |
| height | Single | Chiều cao của khung ảnh, tính bằng điểm. |
| image | IPPImage | [`IPPImage`](../../ippimage) sẽ hiển thị trong khung ảnh. |

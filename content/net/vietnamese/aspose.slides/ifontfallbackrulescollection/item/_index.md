## IFontFallBackRulesCollection bộ chỉ mục

Lấy quy tắc tại chỉ mục được chỉ định. Chỉ đọc [`IFontFallBackRule`](../../ifontfallbackrule).

```csharp
public IFontFallBackRule this[int index] { get; }
```

### Ví dụ

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    //Lấy bộ sưu tập quy tắc trống hoặc đã khởi tạo trước từ FontsManager
    IFontFallBackRulesCollection rulesList = pres.FontsManager.FontFallBackRulesCollection;

    //Thêm một số quy tắc vào bộ sưu tập
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));
    rulesList.Add(new FontFallBackRule(0x3040, 0x309F, "MS Mincho"));

    //Lấy đối tượng của quy tắc đầu tiên trong bộ sưu tập
    IFontFallBackRule firstRule = rulesList[0];
}
```

## IPresentationInfo.CheckWriteProtection phương thức

Kiểm tra xem mật khẩu để chỉnh sửa có đúng cho một bản trình chiếu được bảo vệ ghi không.

```csharp
public bool CheckWriteProtection(string password)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| password | String | Mật khẩu cần kiểm tra. |

### Giá trị trả về

True nếu bản trình chiếu được bảo vệ ghi và mật khẩu đúng. False nếu không.

### Ngoại lệ

| ngoại lệ | điều kiện |
| --- | --- |
| InvalidOperationException |  |

### Ghi chú

1. Bạn nên kiểm tra thuộc tính [`IsWriteProtected`](../iswriteprotected) trước khi gọi phương thức này. 2. Khi password là null hoặc rỗng, phương thức này trả về false.

### Ví dụ

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
if (info.IsWriteProtected == NullableBool.True)
{
    bool isWriteProtectedByPassword = info.CheckWriteProtection("my_password");
}
```

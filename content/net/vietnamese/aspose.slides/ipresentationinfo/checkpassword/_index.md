## IPresentationInfo.CheckPassword phương thức

Kiểm tra xem mật khẩu có đúng cho bản trình chiếu được bảo vệ bằng mật khẩu mở hay không.

```csharp
public bool CheckPassword(string password)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| password | String | Mật khẩu cần kiểm tra. |

### Giá trị trả về

True nếu bản trình chiếu được bảo vệ bằng mật khẩu mở và mật khẩu đúng và false trong các trường hợp khác.

### Ghi chú

Khi mật khẩu là null hoặc rỗng, phương thức này trả về false.

### Ví dụ

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo("pres.pptx");
bool isPasswordCorrect = info.CheckPassword("my_password");
```

## ITextFrame.HighlightRegex phương thức

Làm nổi bật tất cả các khớp của biểu thức chính quy với màu đã chỉ định.

```csharp
public void HighlightRegex(Regex regex, Color highlightColor, IFindResultCallback callback)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| regex | Regex | Biểu thức chính quy Regex để lấy các chuỗi cần làm nổi bật. |
| highlightColor | Color | Màu để làm nổi bật văn bản. |
| callback | IFindResultCallback | Đối tượng callback để nhận kết quả tìm kiếm [`IFindResultCallback`](../../ifindresultcallback). |

### Ví dụ

Mã mẫu sau cho thấy cách làm nổi bật văn bản trong TextFrame bằng biểu thức chính quy.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// làm nổi bật tất cả các từ có 10 ký tự trở lên
	((AutoShape)presentation.Slides[0].Shapes[0]).TextFrame.HighlightRegex(regex, Color.Blue, null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

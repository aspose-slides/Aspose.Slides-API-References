## IAudioFrame.TrimFromStart thuộc tính

Xác định khoảng thời gian cần loại bỏ khỏi phần đầu của media khi phát, tính bằng mili giây. Đọc/ghi Single.

```csharp
public float TrimFromStart { get; set; }
```

### Ví dụ

Ví dụ:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    IAudio audio = pres.Audios.AddAudio(File.ReadAllBytes("sampleaudio.mp3"));
    IAudioFrame audioFrame = pres.Slides[0].Shapes.AddAudioFrameEmbedded(50, 50, 100, 100, audio);

    // Đặt thời gian cắt đầu 1.5 giây
    audioFrame.TrimFromStart = 1500f;
}
```

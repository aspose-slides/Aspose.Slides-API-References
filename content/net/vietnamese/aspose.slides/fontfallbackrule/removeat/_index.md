## FontFallBackRule.RemoveAt phương thức

Xóa phông chữ FallBack tại chỉ mục được chỉ định trong danh sách.

```csharp
public void RemoveAt(int index)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| index | Int32 | Chỉ mục không bắt đầu từ 0 của phông chữ cần xóa. |

### Ví dụ

```csharp
[C#]
// Tạo một quy tắc chứa danh sách phông chữ.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// Xóa Tahoma khỏi danh sách.
newRule.Remove (2);
```

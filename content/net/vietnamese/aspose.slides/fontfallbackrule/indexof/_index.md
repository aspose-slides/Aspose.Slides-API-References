## FontFallBackRule.IndexOf phương thức

Trả về chỉ mục của quy tắc được chỉ định trong bộ sưu tập.

```csharp
public int IndexOf(string fontName)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| fontName | String | Tên phông chữ cần tìm. |

### Giá trị trả về

Chỉ mục của phông chữ hoặc -1 nếu không tìm thấy phông chữ trong danh sách.

### Ví dụ

```csharp
[C#]
// Tạo một quy tắc chứa danh sách các phông chữ.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// Lấy chỉ mục của Tahoma.
int tahomaIndex = newRule.IndexOf("Tahoma");
```

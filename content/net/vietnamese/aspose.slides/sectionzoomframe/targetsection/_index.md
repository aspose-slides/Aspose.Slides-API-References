## SectionZoomFrame.TargetSection thuộc tính

Lấy hoặc đặt đối tượng phần mà đối tượng Section Zoom liên kết tới. Đọc/ghi [`ISection`](../../isection).

```csharp
public ISection TargetSection { get; set; }
```

### Ví dụ

Ví dụ tiếp theo trình bày cách thay đổi phần mục tiêu và tạo ảnh mới cho đối tượng section zoom:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
  ISectionZoomFrame sectionZoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1]);
  sectionZoomFrame.TargetSection = pres.Sections[2];
}
```

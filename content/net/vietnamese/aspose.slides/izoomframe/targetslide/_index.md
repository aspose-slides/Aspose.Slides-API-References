## IZoomFrame.TargetSlide thuộc tính

Lấy hoặc đặt đối tượng slide mà đối tượng Slide Zoom liên kết tới. Đọc/ghi [`ISlide`](../../islide).

```csharp
public ISlide TargetSlide { get; set; }
```

### Ví dụ

Ví dụ tiếp theo minh họa việc thay đổi slide mục tiêu và tạo hình ảnh mới cho đối tượng Slide Zoom:

```csharp
[C#]
IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
zoomFrame.TargetSlide = pres.Slides[2];
```

## SvgImage.WriteAsEmf phương thức

Lưu ảnh SVG dưới dạng tệp EMF.

```csharp
public void WriteAsEmf(Stream stream)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| stream | Stream | luồng đích |

### Ngoại lệ

| ngoại lệ | điều kiện |
| --- | --- |
| ArgumentNullException | Luồng đích là **null** |

### Ví dụ

Ví dụ sau cho thấy cách lưu ảnh SVG vào tệp metafile.

```csharp
[C#]
// Tạo ảnh SVG mới
ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));

// Lưu ảnh SVG dưới dạng tệp metafile
using (var fileStream = System.IO.File.OpenWrite("SvgAsEmf.emf"))
    svgImage.WriteAsEmf(fileStream);
```

Mẫu này minh họa cách thêm ảnh SVG dưới dạng metafile vào bộ sưu tập ảnh của bản trình chiếu.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // Tạo ảnh SVG mới
    ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));
    using (var memStream = new MemoryStream())
    {
        // Lưu ảnh SVG dưới dạng tệp metafile
        svgImage.WriteAsEmf(memStream);
        // Thêm metafile vào bộ sưu tập ảnh
        pres.Images.AddImage(memStream.ToArray());
    }
}
```

## ICommentCollection.AddModernComment phương thức

Thêm bình luận hiện đại mới vào cuối bộ sưu tập.

```csharp
public IModernComment AddModernComment(string text, ISlide slide, IShape shape, PointF position, 
    DateTime creationTime)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| text | String | Văn bản thuần của một bình luận hiện đại mới. |
| slide | ISlide | Slide trong bản trình bày nơi sẽ thêm một bình luận hiện đại mới. |
| shape | IShape | Hình trên slide mà bình luận hiện đại mới được gắn kết. |
| position | PointF | Vị trí trên slide nơi sẽ thêm một bình luận hiện đại mới. |
| creationTime | DateTime | Thời gian tạo bình luận hiện đại. |

### Giá trị trả về

Bình luận hiện đại đã được thêm.

### Ví dụ

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ICommentAuthor newAuthor = pres.CommentAuthors.AddAuthor("Some Author", "SA");
    newAuthor.Comments.AddModernComment("This is modern comment", pres.Slides[0], null, new PointF(100, 100), DateTime.Now);

    pres.Save(outPptxFileName, SaveFormat.Pptx);
}
```

## Hyperlink.Sound thuộc tính

Biểu diễn âm thanh đang phát của siêu liên kết. Đọc/ghi [`IAudio`](../../iaudio).

```csharp
public IAudio Sound { get; set; }
```

### Ví dụ

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];
    
    // Lấy siêu liên kết của hình dạng đầu tiên
    IHyperlink link = presentation.Slides[0].Shapes[0].HyperlinkClick;
       
    if (link.Sound != null)
    {
        // Trích xuất âm thanh siêu liên kết dưới dạng mảng byte
        byte[] audioData = link.Sound.BinaryData;
    }
}
```

## IViewProperties.GridSpacing thuộc tính

Trả về hoặc thiết lập khoảng cách lưới sẽ được sử dụng cho lưới nền của tài liệu trình chiếu, tính bằng điểm. Đọc/ghi Single.

```csharp
public float GridSpacing { get; set; }
```

### Ghi chú

Giá trị khoảng cách lưới phải là số dương. Dải giá trị thường nằm trong khoảng từ 1 mm (2.8349607 điểm) đến 2 inch (144 điểm).

### Ví dụ

Đoạn mã mẫu dưới đây cho thấy cách thay đổi khoảng cách lưới trong bản trình bày PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

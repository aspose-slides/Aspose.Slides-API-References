## ISlideShowTransition.AdvanceAfter thuộc tính

Thuộc tính này xác định liệu bản trình chiếu có chuyển sang slide tiếp theo sau một khoảng thời gian nhất định hay không. Đọc/ghi Boolean.

```csharp
public bool AdvanceAfter { get; set; }
```

### Ví dụ

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    // Lấy Transition của slide đầu tiên
    ISlideShowTransition slideTransition = pres.Slides[0].SlideShowTransition;
    
    // Kiểm tra xem cờ Advance Slide After có được bật không
    if (slideTransition.AdvanceAfter)
    {
        // Lấy giá trị thời gian Advance Slide After
        uint advanceAfterTime = slideTransition.AdvanceAfterTime;
    }
}
```

## ISlideShowTransition.Duration thuộc tính

Lấy hoặc đặt thời lượng của hiệu ứng chuyển slide tính bằng mili giây. Đọc/ghi Int32.

```csharp
public int Duration { get; set; }
```

### Ghi chú

Corresponds to the `p14:dur` attribute of the `p:transition` element in the PresentationML schema. If not set, the duration is determined automatically based on the [`Speed`](../speed) thuộc tính và kiểu chuyển.

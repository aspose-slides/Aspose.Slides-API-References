## IFontsManager.GetFontBytes phương thức

Truy xuất mảng byte đại diện cho dữ liệu phông chữ cho một kiểu phông chữ và dữ liệu phông chữ được chỉ định.

```csharp
public byte[] GetFontBytes(IFontData fontData, FontStyleType fontStyle)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| fontData | IFontData | Đối tượng dữ liệu phông chữ chứa thông tin về phông chữ [`IFontData`](../../ifontdata). |
| fontStyle | FontStyleType | Kiểu phông chữ mà dữ liệu sẽ được truy xuất [`FontStyleType`](../../fontstyletype). |

### Giá trị trả về

Một mảng byte chứa dữ liệu phông chữ cho kiểu phông chữ đã chỉ định. Nếu dữ liệu phông chữ hoặc kiểu không được tìm thấy, trả về null.

### Ví dụ

```csharp
[C#]
using (Presentation pres = new Presentation ("Presentation.pptx"))
{
    // Lấy tất cả phông chữ được sử dụng trong bản trình chiếu
    IFontData[] fonts = pres.FontsManager.GetFonts();

    // Lấy mảng byte đại diện cho kiểu thường của phông chữ đầu tiên trong bản trình chiếu
    byte[] bytes = pres.FontsManager.GetFontBytes(fonts[0], FontStyleType.Regular);
}
```

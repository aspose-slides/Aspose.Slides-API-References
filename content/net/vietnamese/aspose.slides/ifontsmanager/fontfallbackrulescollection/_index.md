## IFontsManager.FontFallBackRulesCollection thuộc tính

Đại diện cho bộ sưu tập các quy tắc FontFallBack của người dùng để quản lý các bộ sưu tập phông chữ nhằm thực hiện việc thay thế thích hợp thông qua chức năng fallback. Đọc/ghi [`IFontFallBackRulesCollection`](../../ifontfallbackrulescollection).

```csharp
public IFontFallBackRulesCollection FontFallBackRulesCollection { get; set; }
```

### Ví dụ

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    // Lấy bộ sưu tập quy tắc trống hoặc đã được khởi tạo trước từ FontsManager
    IFontFallBackRulesCollection rulesList = pres.FontsManager.FontFallBackRulesCollection;

    // Thêm quy tắc vào bộ sưu tập
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));

    // hoặc 
    // khởi tạo một thực thể mới của bộ sưu tập quy tắc
    IFontFallBackRulesCollection rulesList = new FontFallBackRulesCollection();

    // Thêm quy tắc vào bộ sưu tập
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));

    // và thay thế bộ sưu tập hiện có bằng bộ mới trong FontsManager 
    pres.FontsManager.FontFallBackRulesCollection = rulesList;
}
```

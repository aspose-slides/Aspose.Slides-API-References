## IPictureFillFormat.TileOffsetY thuộc tính

Trả về hoặc thiết lập độ dịch dọc của texture so với gốc của shape bằng điểm. Giá trị dương di chuyển texture xuống, trong khi giá trị âm di chuyển nó lên. Đọc/ghi Single.

```csharp
public float TileOffsetY { get; set; }
```

### Ví dụ

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Lấy định dạng lấp đầy ảnh của shape
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Đặt chế độ lấp đầy ảnh thành Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Đặt độ dịch dọc của texture thành -50 điểm
    pictureFillFormat.TileOffsetY = -50;
}
```

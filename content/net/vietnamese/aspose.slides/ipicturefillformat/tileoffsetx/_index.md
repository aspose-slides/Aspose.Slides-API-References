## IPictureFillFormat.TileOffsetX thuộc tính

Trả về hoặc thiết lập độ lệch ngang của hoa văn so với gốc của hình dạng tính bằng điểm. Giá trị dương di chuyển hoa văn sang bên phải, trong khi giá trị âm di chuyển nó sang bên trái. Đọc/ghi Single.

```csharp
public float TileOffsetX { get; set; }
```

### Ví dụ

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Lấy định dạng lấp đầy ảnh của hình
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Đặt chế độ lấp đầy ảnh thành Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Đặt độ lệch ngang của hoa văn thành 20 điểm
    pictureFillFormat.TileOffsetX = 20f;
}
```

## IPictureFillFormat.DeletePictureCroppedAreas phương thức

Xóa các khu vực đã cắt của Picture được điền.

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### Giá trị trả về

Ảnh đã cắt hoặc ảnh gốc nếu không cần cắt.

### Ghi chú

Phương thức này chuyển các metafile WMF/EMF sang ảnh PNG raster trong khi cắt.

### Ví dụ

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Lấy PictureFrame
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // Xóa các khu vực đã cắt của hình ảnh PictureFrame
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

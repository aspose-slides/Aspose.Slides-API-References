## LoadOptions.DocumentLevelFontSources thuộc tính

Xác định các nguồn cho phông chữ bên ngoài sẽ được sử dụng bởi bản trình chiếu. Những phông chữ này có sẵn cho bản trình chiếu trong suốt thời gian tồn tại và không được chia sẻ với các bản trình chiếu khác.

```csharp
public IFontSources DocumentLevelFontSources { get; set; }
```

### Ví dụ

Ví dụ sau đây cho thấy cách chỉ định phông chữ tùy chỉnh được sử dụng với PowerPoint Presentation.

```csharp
[C#]
byte[] memoryFont1 = File.ReadAllBytes("customfonts\\CustomFont1.ttf");
byte[] memoryFont2 = File.ReadAllBytes("customfonts\\CustomFont2.ttf");
LoadOptions loadOptions = new LoadOptions();
loadOptions.DocumentLevelFontSources.FontFolders = new string[] { "assets\\fonts", "global\\fonts" };
loadOptions.DocumentLevelFontSources.MemoryFonts = new byte[][] { memoryFont1, memoryFont2 };
using (IPresentation presentation = new Presentation("MyPresentation.pptx", loadOptions))
{
    //làm việc với bản trình chiếu
    //CustomFont1, CustomFont2 cũng như các phông chữ từ thư mục assets\fonts & global\fonts và các thư mục con của chúng đều khả dụng cho bản trình chiếu
}
```

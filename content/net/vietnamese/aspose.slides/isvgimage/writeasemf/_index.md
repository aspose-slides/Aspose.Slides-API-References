## ISvgImage.WriteAsEmf phương thức

Lưu ảnh SVG dưới dạng tệp EMF.

```csharp
public void WriteAsEmf(Stream stream)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| stream | Stream | Luồng mục tiêu |

### Ngoại lệ

| ngoại lệ | điều kiện |
| --- | --- |
| ArgumentNullException | Luồng mục tiêu là **null** |

### Ví dụ

Ví dụ sau minh họa cách lưu ảnh SVG vào tệp meta.

```csharp
[C#]
// Tạo ảnh SVG mới
ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));

// Lưu ảnh SVG dưới dạng metafille
using (var fileStream = System.IO.File.OpenWrite("SvgAsEmf.emf"))
    svgImage.WriteAsEmf(fileStream);
```

Mẫu này minh họa cách thêm ảnh SVG dưới dạng tệp meta vào bộ sưu tập ảnh của bản trình chiếu.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // Tạo ảnh SVG mới
    ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));
    using (var memStream = new MemoryStream())
    {
        // Lưu ảnh SVG dưới dạng metafille
        svgImage.WriteAsEmf(memStream);
        // Thêm metafile vào bộ sưu tập ảnh
        pres.Images.AddImage(memStream.ToArray());
    }
}
```

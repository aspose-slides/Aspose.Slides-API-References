## IZoomObject.ImageType property

Lấy hoặc thiết lập kiểu hình ảnh của đối tượng Zoom. Đọc/ghi [`ZoomImageType`](../../zoomimagetype). Giá trị mặc định: Preview

```csharp
public ZoomImageType ImageType { get; set; }
```

### Ghi chú

Xác định xem đối tượng Zoom đang sử dụng bản xem trước của slide hay hình ảnh bìa.

### Ví dụ

Ví dụ này minh họa việc thay đổi Image Type thành giá trị Preview. Trong trường hợp này, hình ảnh hiện tại của một đối tượng Zoom sẽ thay đổi thành hình ảnh slide:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    IPPImage image = pres.Images.AddImage(Image.FromFile("image.png"));
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1], image);
    zoomFrame.ImageType = ZoomImageType.Preview;
}
```

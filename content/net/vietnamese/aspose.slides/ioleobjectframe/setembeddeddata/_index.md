## IOleObjectFrame.SetEmbeddedData phương thức

| ngoại lệ | điều kiện |
| --- | --- |

### Ghi chú

Phương thức này thay đổi các thuộc tính của đối tượng để phản ánh dữ liệu mới và đặt cờ IsObjectLink thành false, cho biết rằng đối tượng OLE được nhúng.

### Ví dụ

Ví dụ sau minh họa cách thay đổi dữ liệu nhúng OLE và kiểu của nó cho đối tượng [`IOleObjectFrame`](../../ioleobjectframe) hiện có

```csharp
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    OleObjectFrame oof = pres.Slides[0].Shapes[0] as OleObjectFrame;
    if (oof != null)
    {
        IOleEmbeddedDataInfo newData = new OleEmbeddedDataInfo(File.ReadAllBytes("Picture.png"), "png");
        oof.SetEmbeddedData(newData);
    }
}
```

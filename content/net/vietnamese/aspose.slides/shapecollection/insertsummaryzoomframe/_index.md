## ShapeCollection.InsertSummaryZoomFrame phương thức

Tạo một Summary Zoom frame mới và chèn nó vào bộ sưu tập shape tại chỉ mục được chỉ định.

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| index | Int32 | Chỉ mục bắt đầu từ 0 mà tại đó chèn Summary Zoom frame. |
| x | Single | Tọa độ x của Summary Zoom frame mới, tính bằng điểm. |
| y | Single | Tọa độ y của Summary Zoom frame mới, tính bằng điểm. |
| width | Single | Chiều rộng của Summary Zoom frame mới, tính bằng điểm. |
| height | Single | Chiều cao của Summary Zoom frame mới, tính bằng điểm. |

### Ghi chú

Phương thức này tạo một Summary Zoom frame tổng hợp các liên kết tóm tắt cho tất cả các phần trong bản trình chiếu.

### Ví dụ

Ví dụ này minh họa cách tạo và chèn một đối tượng Summary Zoom tại chỉ mục được chỉ định của một bộ sưu tập (giả sử có ít nhất hai phần trong bản trình chiếu "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSummaryZoomFrame(2, 150, 20, 50, 50);
}
```

## InsertOleObjectFrame(int, float, float, float, float, IOleEmbeddedDataInfo) {#insertoleobjectframe}

Tạo một khung đối tượng OLE mới và chèn nó vào bộ sưu tập hình dạng tại chỉ số được chỉ định.

```csharp
public IOleObjectFrame InsertOleObjectFrame(int index, float x, float y, float width, float height, 
    IOleEmbeddedDataInfo dataInfo)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| index | Int32 | Chỉ số bắt đầu từ 0 mà sẽ chèn khung đối tượng OLE. |
| x | Single | Tọa độ x của khung OLE mới, tính bằng điểm. |
| y | Single | Tọa độ y của khung OLE mới, tính bằng điểm. |
| width | Single | Chiều rộng của khung OLE mới, tính bằng điểm. |
| height | Single | Chiều cao của khung OLE mới, tính bằng điểm. |
| dataInfo | IOleEmbeddedDataInfo | Thông tin dữ liệu OLE được nhúng ([`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo)). |

### Giá trị trả về

Đối tượng [`IOleObjectFrame`](../../ioleobjectframe) mới tạo.

### Ví dụ

Ví dụ này minh họa cách chèn một đối tượng OLE vào chỉ số thứ hai:

```csharp
[C#]
byte[] fileData = File.ReadAllBytes("test.zip");
IOleDataInfo dataInfo = new OleDataInfo(fileData, "zip");
IOleObjectFrame oleObjectFrame = slidees.Shapes.AddOleObjectFrame(2, 150, 20, 50, 50, dataInfo);
```

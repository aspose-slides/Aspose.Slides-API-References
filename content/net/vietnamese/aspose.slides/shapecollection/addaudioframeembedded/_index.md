## AddAudioFrameEmbedded(float, float, float, float, Stream) {#addaudioframeembedded_1}

Tạo một khung âm thanh mới có tệp WAV được nhúng và thêm nó vào cuối bộ sưu tập hình dạng. Âm thanh được nhúng sẽ được thêm vào bộ sưu tập Presentation.Audios.

```csharp
public IAudioFrame AddAudioFrameEmbedded(float x, float y, float width, float height, 
    Stream audio_stream)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| x | Single | Tọa độ x của khung âm thanh mới, tính bằng điểm. |
| y | Single | Tọa độ y của khung âm thanh mới, tính bằng điểm. |
| width | Single | Chiều rộng của khung âm thanh mới, tính bằng điểm. |
| height | Single | Chiều cao của khung âm thanh mới, tính bằng điểm. |
| audio_stream | Stream | Luồng đầu vào chứa dữ liệu âm thanh WAV để nhúng. |

### Giá trị trả về

Đối tượng [`IAudioFrame`](../../iaudioframe) mới được tạo.

### Ví dụ

Các ví dụ sau đây cho thấy cách tạo khung âm thanh.

```csharp
[C#]
// Khởi tạo một lớp presentation đại diện cho tệp presentation
using (Presentation pres = new Presentation())
{
    // Lấy slide đầu tiên
    ISlide sld = pres.Slides[0];
    // Tải tệp âm thanh wav vào luồng
    FileStream fstr = new FileStream("sampleaudio.wav", FileMode.Open, FileAccess.Read);
    // Thêm khung âm thanh
    IAudioFrame audioFrame = sld.Shapes.AddAudioFrameEmbedded(50, 150, 100, 100, fstr);
    // Đặt chế độ phát và âm lượng cho âm thanh
    audioFrame.PlayMode = AudioPlayModePreset.Auto;
    audioFrame.Volume = AudioVolumeMode.Loud;
    // Ghi tệp PowerPoint ra đĩa
    pres.Save("AudioFrameEmbed_out.pptx", SaveFormat.Pptx);
}
```

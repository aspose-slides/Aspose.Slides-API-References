## InsertAutoShape(int, ShapeType, float, float, float, float) {#insertautoshape}

public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width, 
    float height)

| shapeType | ShapeType | [`ShapeType`](../../shapetype) của hình tự động cần chèn. |
| x | Single | Tọa độ x của khung hình, tính bằng điểm. |
| y | Single | Tọa độ y của khung hình, tính bằng điểm. |
| width | Single | Chiều rộng của khung hình, tính bằng điểm. |
| height | Single | Chiều cao của khung hình, tính bằng điểm. |

## InsertAutoShape(int, ShapeType, float, float, float, float, bool) {#insertautoshape_1}

Tạo một hình tự động mới và chèn nó vào bộ sưu tập hình tại chỉ mục được chỉ định, tùy chọn khởi tạo nó với kiểu dáng mẫu mặc định.

```csharp
public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| index | Int32 | Chỉ mục bắt đầu từ 0 tại vị trí chèn hình tự động. |
| shapeType | ShapeType | [`ShapeType`](../../shapetype) của hình tự động cần chèn. |
| x | Single | Tọa độ x của khung hình, tính bằng điểm. |
| y | Single | Tọa độ y của khung hình, tính bằng điểm. |
| width | Single | Chiều rộng của khung hình, tính bằng điểm. |
| height | Single | Chiều cao của khung hình, tính bằng điểm. |
| createFromTemplate | Boolean | True để áp dụng kiểu dáng mẫu mặc định (bao gồm tên không rỗng, kiểu đơn giản và văn bản căn giữa); false để tạo hình với mọi thuộc tính được đặt về mặc định. |

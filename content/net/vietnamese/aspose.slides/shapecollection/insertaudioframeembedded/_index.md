## InsertAudioFrameEmbedded(int, float, float, float, float, Stream) {#insertaudioframeembedded_1}

Tạo một khung âm thanh mới với tệp WAV được nhúng và chèn nó vào bộ sưu tập shape tại chỉ mục đã chỉ định. Âm thanh được nhúng sẽ được thêm vào bộ sưu tập Presentation.Audios.

```csharp
public IAudioFrame InsertAudioFrameEmbedded(int index, float x, float y, float width, float height, 
    Stream audio_stream)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | Int32 | Chỉ mục bắt đầu từ 0 tại vị trí sẽ chèn khung âm thanh. |
| x | Single | Tọa độ x của khung âm thanh mới, tính bằng điểm. |
| y | Single | Tọa độ y của khung âm thanh mới, tính bằng điểm. |
| width | Single | Chiều rộng của khung âm thanh mới, tính bằng điểm. |
| height | Single | Chiều cao của khung âm thanh mới, tính bằng điểm. |
| audio_stream | Stream | Luồng đầu vào chứa dữ liệu âm thanh WAV để nhúng. |

## Shape.Placeholder thuộc tính

Trả về placeholder cho một shape. Trả về null nếu shape không có placeholder. Chỉ đọc [`IPlaceholder`](../../iplaceholder).

```csharp
public IPlaceholder Placeholder { get; }
```

### Ví dụ

Ví dụ sau cho thấy cách thay đổi Text trong Placeholder.

```csharp
[C#]
// Tạo một đối tượng lớp Presentation
using (Presentation pres = new Presentation("ReplacingText.pptx"))
{
    // Truy cập slide đầu tiên
    ISlide sld = pres.Slides[0];
    // Lặp qua các shape để tìm placeholder
    foreach (IShape shp in sld.Shapes)
        if (shp.Placeholder != null)
        {
            // Thay đổi văn bản trong mỗi placeholder
            ((IAutoShape)shp).TextFrame.Text = "This is a Placeholder";
        }
    // Lưu bản trình chiếu vào đĩền
    pres.Save("output_out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
}
```

Ví dụ sau cho thấy cách đặt Prompt Text trong Placeholder.

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation2.pptx"))
{
    ISlide slide = pres.Slides[0];
    foreach (IShape shape in slide.Slide.Shapes) // Duyệt qua slide
    {
        if (shape.Placeholder != null && shape is AutoShape)
        {
            string text = "";
            if (shape.Placeholder.Type == PlaceholderType.CenteredTitle) // PowerPoint hiển thị "Click to add title"
            {
                text = "Add Title";
            }
            else if (shape.Placeholder.Type == PlaceholderType.Subtitle) // Thêm phụ đề
            {
                text = "Add Subtitle";
            }
            ((IAutoShape)shape).TextFrame.Text = text;
            Console.WriteLine($"Placeholder with text: {text}");
        }
    }
    pres.Save("Placeholders_PromptText.pptx", SaveFormat.Pptx);
}
```

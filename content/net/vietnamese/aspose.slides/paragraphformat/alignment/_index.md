## ParagraphFormat.Alignment thuộc tính

Trả về hoặc đặt căn chỉnh văn bản trong một đoạn không kế thừa. Đọc/ghi [`TextAlignment`](../../textalignment).

```csharp
public TextAlignment Alignment { get; set; }
```

### Ví dụ

Mã mẫu sau cho thấy cách căn chỉnh các đoạn văn bản trong bản trình chiếu PowerPoint.

```csharp
[C#]
// Tạo một đối tượng Presentation đại diện cho tệp PPTX
using (Presentation pres = new Presentation("ParagraphsAlignment.pptx"))
{
    // Truy cập slide đầu tiên
    ISlide slide = pres.Slides[0];
    // Truy cập placeholder thứ nhất và thứ hai trong slide và ép kiểu thành AutoShape
    ITextFrame tf1 = ((IAutoShape)slide.Shapes[0]).TextFrame;
    ITextFrame tf2 = ((IAutoShape)slide.Shapes[1]).TextFrame;
    // Thay đổi văn bản trong cả hai placeholder
    tf1.Text = "Center Align by Aspose";
    tf2.Text = "Center Align by Aspose";
    // Lấy đoạn văn bản đầu tiên của các placeholder
    IParagraph para1 = tf1.Paragraphs[0];
    IParagraph para2 = tf2.Paragraphs[0];
    // Căn chỉnh đoạn văn bản sang giữa
    para1.ParagraphFormat.Alignment = TextAlignment.Center;
    para2.ParagraphFormat.Alignment = TextAlignment.Center;
    // Ghi bản trình chiếu dưới dạng tệp PPTX
    pres.Save("Centeralign_out.pptx", SaveFormat.Pptx);
}
```

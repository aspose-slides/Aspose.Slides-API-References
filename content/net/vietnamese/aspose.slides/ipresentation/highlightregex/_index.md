## IPresentation.HighlightRegex phương thức

Tô sáng tất cả các kết quả khớp của biểu thức chính quy với màu được chỉ định.

```csharp
public void HighlightRegex(Regex regex, Color highlightColor, IFindResultCallback callback)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| regex | Regex | Biểu thức chính quy Regex để lấy các chuỗi cần tô màu. |
| highlightColor | Color | Màu sắc để tô màu văn bản. |
| callback | IFindResultCallback | Đối tượng callback để nhận kết quả tìm kiếm [`IFindResultCallback`](../../ifindresultcallback). |

### Ví dụ

Đoạn mã mẫu dưới đây cho thấy cách tô sáng văn bản trong PowerPoint Presentation bằng cách sử dụng biểu thức chính quy.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// đánh dấu tất cả các từ có độ dài 10 ký tự trở lên
	presentation.HighlightRegex(regex, Color.Blue, null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

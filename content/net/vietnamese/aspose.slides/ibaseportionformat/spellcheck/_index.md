## IBasePortionFormat.SpellCheck thuộc tính

Lấy hoặc đặt một giá trị cho biết liệu việc kiểm tra chính tả có được bật cho phần văn bản hay không. Khi thuộc tính này được đặt thành false, việc kiểm tra chính tả cho các phần tử văn bản sẽ bị bỏ qua. Khi được đặt thành true, việc kiểm tra chính tả được cho phép. Giá trị mặc định là `false`.

```csharp
public bool SpellCheck { get; set; }
```

### Ví dụ

Ví dụ tiếp theo minh họa cách bật cờ SpellCheck trước khi lưu bản trình chiếu:

```csharp
[C#]
using (var pres = new Presentation("input.pptx"))
{
    // Truy cập phần văn bản đầu tiên bên trong hình dạng đầu tiên trên slide đầu tiên
    var portion = ((AutoShape)pres.Slides[0].Shapes[0]).TextFrame.Paragraphs[0].Portions[0];
    // Bật kiểm tra chính tả cho phần văn bản này
    portion.PortionFormat.SpellCheck = true;
    // Lưu bản trình chiếu đã chỉnh sửa
    pres.Save("output-with-spellcheck.pptx", SaveFormat.Pptx);
}
```

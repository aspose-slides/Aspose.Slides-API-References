## ILayoutPlaceholderManager.AddTablePlaceholder method

Thêm một hình dạng placeholder mới vào slide bố cục để chứa một bảng.

```csharp
public IAutoShape AddTablePlaceholder(float x, float y, float width, float height)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| x | Single | Tọa độ X của hình dạng placeholder mới. |
| y | Single | Tọa độ Y của hình dạng placeholder mới. |
| width | Single | Chiều rộng của hình dạng placeholder mới. |
| height | Single | Chiều cao của hình dạng placeholder mới. |

### Return Value

Đã tạo [`IAutoShape`](../../iautoshape) với một placeholder Table.

### Examples

Ví dụ sau đây cho thấy cách thêm hình dạng placeholder Table vào slide bố cục.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTablePlaceholder(20, 20, 500, 200);
}
```

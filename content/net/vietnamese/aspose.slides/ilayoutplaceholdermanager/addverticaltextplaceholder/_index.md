## ILayoutPlaceholderManager.AddVerticalTextPlaceholder method

Thêm một hình placeholder mới vào slide bố cục để chứa nội dung văn bản theo hướng dọc.

```csharp
public IAutoShape AddVerticalTextPlaceholder(float x, float y, float width, float height)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| x | Single | Tọa độ X của hình placeholder mới. |
| y | Single | Tọa độ Y của hình placeholder mới. |
| width | Single | Chiều rộng của hình placeholder mới. |
| height | Single | Chiều cao của hình placeholder mới. |

### Giá trị trả về

Created [`IAutoShape`](../../iautoshape) with a Text (Vertical) placeholder.

### Ví dụ

The following example shows how to add the Text (Vertical) placeholder shape to the layout slide.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
    IAutoShape placeholder = layout.PlaceholderManager.AddVerticalTextPlaceholder(20, 20, 300, 500);
}
```

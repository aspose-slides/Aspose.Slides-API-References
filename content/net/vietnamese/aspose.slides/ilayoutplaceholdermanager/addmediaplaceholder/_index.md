## ILayoutPlaceholderManager.AddMediaPlaceholder phương thức

Thêm một hình giữ chỗ mới vào slide bố cục để chứa một đối tượng phương tiện.

```csharp
public IAutoShape AddMediaPlaceholder(float x, float y, float width, float height)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| x | Single | Tọa độ X của hình giữ chỗ mới. |
| y | Single | Tọa độ Y của hình giữ chỗ mới. |
| width | Single | Chiều rộng của hình giữ chỗ mới. |
| height | Single | Chiều cao của hình giữ chỗ mới. |

### Giá trị trả về

Đã tạo [`IAutoShape`](../../iautoshape) với một trình giữ chỗ Media.

### Ví dụ

Ví dụ sau đây cho thấy cách thêm hình giữ chỗ Media vào slide bố cục.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddMediaPlaceholder(20, 20, 200, 200);
}
```

## ILayoutPlaceholderManager.AddContentPlaceholder phương thức

Thêm một hình placeholder mới vào slide bố cục để chứa nội dung, chẳng hạn như hình ảnh, bảng, phương tiện hoặc văn bản.

```csharp
public IAutoShape AddContentPlaceholder(float x, float y, float width, float height)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| x | Single | Tọa độ X của hình placeholder mới. |
| y | Single | Tọa độ Y của hình placeholder mới. |
| width | Single | Chiều rộng của hình placeholder mới. |
| height | Single | Chiều cao của hình placeholder mới. |

### Giá trị trả về

Đã tạo [`IAutoShape`](../../iautoshape) với một placeholder Content.

### Ví dụ

Ví dụ sau đây cho thấy cách thêm hình placeholder Content vào slide bố cục.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddContentPlaceholder(20, 20, 500, 300);
}
```

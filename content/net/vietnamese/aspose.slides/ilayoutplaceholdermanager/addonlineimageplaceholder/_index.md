## ILayoutPlaceholderManager.AddOnlineImagePlaceholder phương thức

Thêm một hình dạng placeholder mới vào slide bố cục để chứa một hình ảnh trực tuyến.

```csharp
public IAutoShape AddOnlineImagePlaceholder(float x, float y, float width, float height)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| x | Single | Tọa độ X của hình dạng placeholder mới. |
| y | Single | Tọa độ Y của hình dạng placeholder mới. |
| width | Single | Chiều rộng của hình dạng placeholder mới. |
| height | Single | Chiều cao của hình dạng placeholder mới. |

### Giá trị trả về

Đã tạo [`IAutoShape`](../../iautoshape) với một placeholder Hình ảnh trực tuyến.

### Ví dụ

Ví dụ sau cho thấy cách thêm placeholder Hình ảnh trực tuyến vào slide bố cục.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddOnlineImagePlaceholder(20, 20, 200, 200);
}
```

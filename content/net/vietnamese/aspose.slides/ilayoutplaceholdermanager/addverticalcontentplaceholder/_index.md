## ILayoutPlaceholderManager.AddVerticalContentPlaceholder method

Thêm một hình giữ chỗ mới vào slide bố cục để chứa nội dung, chẳng hạn như hình ảnh, bảng, phương tiện hoặc văn bản theo hướng dọc.

```csharp
public IAutoShape AddVerticalContentPlaceholder(float x, float y, float width, float height)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| x | Single | Tọa độ X của hình giữ chỗ mới. |
| y | Single | Tọa độ Y của hình giữ chỗ mới. |
| width | Single | Chiều rộng của hình giữ chỗ mới. |
| height | Single | Chiều cao của hình giữ chỗ mới. |

### Giá trị trả về

[`IAutoShape`](../../iautoshape) đã được tạo với một placeholder Content (Vertical).

### Ví dụ

Ví dụ sau cho thấy cách thêm hình placeholder Content (Vertical) vào slide bố cục.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalContentPlaceholder(20, 20, 300, 500);
}
```

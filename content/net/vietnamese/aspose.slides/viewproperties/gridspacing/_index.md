## ViewProperties.GridSpacing thuộc tính

Trả về hoặc đặt khoảng cách lưới sẽ được sử dụng cho lưới nền của tài liệu bản trình chiếu, tính bằng điểm. Đọc/ghi Single.

```csharp
public float GridSpacing { get; set; }
```

### Ghi chú

Giá trị khoảng cách lưới phải là một số dương. Khoảng giá trị điển hình là từ 1 mm (2.8349607 điểm) đến 2 inch (144 điểm).

### Ví dụ

Mã mẫu dưới đây cho thấy cách thay đổi khoảng cách lưới trong một bài thuyết trình PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

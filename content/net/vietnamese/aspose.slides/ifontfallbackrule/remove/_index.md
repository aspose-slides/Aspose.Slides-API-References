## IFontFallBackRule.Remove phương thức

Xóa lần xuất hiện đầu tiên của một phông chữ FallBack cụ thể khỏi danh sách.

```csharp
public void Remove(string fontName)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| fontName | String | Tên phông chữ để xóa khỏi danh sách. |

### Ví dụ

```csharp
[C#]
// Tạo một quy tắc chứa danh sách các phông chữ.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// Xóa Tahoma khỏi danh sách
newRule.Remove ("Tahoma");
```

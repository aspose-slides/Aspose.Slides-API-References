## MasterSlideCollection.InsertClone phương thức

Chèn một bản sao của slide mẫu được chỉ định vào vị trí xác định trong bộ sưu tập. Các slide bố cục được liên kết cũng sẽ được sao chép.

```csharp
public IMasterSlide InsertClone(int index, IMasterSlide sourceMaster)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| index | Int32 | Chỉ mục của slide mới. |
| sourceMaster | IMasterSlide | Slide để sao chép. |

### Giá trị trả về

Slide mẫu đã chèn.

### Ví dụ

Ví dụ sau cho thấy cách sao chép slide mẫu trong một bản trình bày PowerPoint khác.

```csharp
[C#]
// Tạo đối tượng lớp Presentation để tải tệp trình chiếu nguồn
using (Presentation srcPres = new Presentation("CloneToAnotherPresentationWithMaster.pptx"))
{
    // Tạo đối tượng lớp Presentation cho trình chiếu đích (nơi slide sẽ được sao chép)
    using (Presentation destPres = new Presentation())
    {
        // Tạo đối tượng ISlide từ bộ sưu tập các slide trong trình chiếu nguồn cùng với
        // Slide mẫu
        ISlide SourceSlide = srcPres.Slides[0];
        IMasterSlide SourceMaster = SourceSlide.LayoutSlide.MasterSlide;
		// Lấy các Slide mẫu của trình chiếu đích
        IMasterSlideCollection masters = destPres.Masters;
        // Sao chép slide mẫu mong muốn từ trình chiếu nguồn vào bộ sưu tập các slide mẫu trong
        // Trình chiếu đích
        IMasterSlide iSlide = masters.AddClone(SourceMaster);
        // Bộ sưu tập các slide trong trình chiếu đích
        ISlideCollection slds = destPres.Slides;
		// Sao chép slide nguồn vào bộ sưu tập slide đích.
        slds.AddClone(SourceSlide, iSlide, true);
        // Lưu trình chiếu đích vào đĩa
        destPres.Save("CloneToAnotherPresentationWithMaster_out.pptx", SaveFormat.Pptx);
    }
}
```

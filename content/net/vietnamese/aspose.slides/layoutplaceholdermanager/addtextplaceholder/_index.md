## LayoutPlaceholderManager.AddTextPlaceholder phương thức

Thêm một hình dạng placeholder mới vào slide bố cục để chứa nội dung văn bản.

```csharp
public IAutoShape AddTextPlaceholder(float x, float y, float width, float height)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| x | Single | Tọa độ X của hình dạng placeholder mới. |
| y | Single | Tọa độ Y của hình dạng placeholder mới. |
| width | Single | Độ rộng của hình dạng placeholder mới. |
| height | Single | Độ cao của hình dạng placeholder mới. |

### Giá trị trả về

Đã tạo [`IAutoShape`](../../iautoshape) với một placeholder Văn bản.

### Ví dụ

Ví dụ sau đây cho thấy cách thêm hình dạng placeholder Văn bản vào slide bố cục.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTextPlaceholder(20, 20, 500, 300);
}
```

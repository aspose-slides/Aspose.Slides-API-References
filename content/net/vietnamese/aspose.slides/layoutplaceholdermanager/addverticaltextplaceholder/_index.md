## LayoutPlaceholderManager.AddVerticalTextPlaceholder phương thức

Thêm một hình giữ chỗ mới vào slide bố cục để chứa nội dung văn bản theo chiều dọc.

```csharp
public IAutoShape AddVerticalTextPlaceholder(float x, float y, float width, float height)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| x | Single | Tọa độ X của hình giữ chỗ mới. |
| y | Single | Tọa độ Y của hình giữ chỗ mới. |
| width | Single | Độ rộng của hình giữ chỗ mới. |
| height | Single | Độ cao của hình giữ chỗ mới. |

### Giá trị trả về

Đã tạo [`IAutoShape`](../../iautoshape) với một giữ chỗ Văn bản (Dọc).

### Ví dụ

Ví dụ sau đây cho thấy cách thêm hình giữ chỗ Văn bản (Dọc) vào slide bố cục.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalTextPlaceholder(20, 20, 300, 500);
}
```

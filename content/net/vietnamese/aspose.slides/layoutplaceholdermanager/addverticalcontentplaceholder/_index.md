## LayoutPlaceholderManager.AddVerticalContentPlaceholder phương thức

Thêm một hình dạng placeholder mới vào slide bố cục để chứa nội dung, chẳng hạn như hình ảnh, bảng, phương tiện hoặc văn bản theo hướng dọc.

```csharp
public IAutoShape AddVerticalContentPlaceholder(float x, float y, float width, float height)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| x | Single | Tọa độ X của hình dạng placeholder mới. |
| y | Single | Tọa độ Y của hình dạng placeholder mới. |
| width | Single | Độ rộng của hình dạng placeholder mới. |
| height | Single | Độ cao của hình dạng placeholder mới. |

### Giá trị trả về

Đã tạo [`IAutoShape`](../../iautoshape) với một placeholder Nội dung (Dọc).

### Ví dụ

Ví dụ sau minh họa cách thêm hình dạng placeholder Nội dung (Dọc) vào slide bố cục.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalContentPlaceholder(20, 20, 300, 500);
}
```

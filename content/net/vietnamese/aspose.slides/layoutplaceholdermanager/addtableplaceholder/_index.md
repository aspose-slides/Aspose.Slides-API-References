## LayoutPlaceholderManager.AddTablePlaceholder phương thức

Thêm một hình dạng giữ chỗ mới vào slide bố cục để chứa một bảng.

```csharp
public IAutoShape AddTablePlaceholder(float x, float y, float width, float height)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| x | Single | Tọa độ X của hình dạng giữ chỗ mới. |
| y | Single | Tọa độ Y của hình dạng giữ chỗ mới. |
| width | Single | Độ rộng của hình dạng giữ chỗ mới. |
| height | Single | Độ cao của hình dạng giữ chỗ mới. |

### Giá trị trả về

Đã tạo [`IAutoShape`](../../iautoshape) với một giữ chỗ Table.

### Ví dụ

Ví dụ sau đây cho thấy cách thêm hình dạng giữ chỗ Table vào slide bố cục.

```csharp
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTablePlaceholder(20, 20, 500, 200);
}
```

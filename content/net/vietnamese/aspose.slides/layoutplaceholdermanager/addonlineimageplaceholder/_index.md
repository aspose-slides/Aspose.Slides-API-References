## LayoutPlaceholderManager.AddOnlineImagePlaceholder phương thức

Thêm một hình dạng giữ chỗ mới vào slide bố cục để chứa hình ảnh trực tuyến.

```csharp
public IAutoShape AddOnlineImagePlaceholder(float x, float y, float width, float height)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| x | Single | Tọa độ X của hình dạng giữ chỗ mới. |
| y | Single | Tọa độ Y của hình dạng giữ chỗ mới. |
| width | Single | Chiều rộng của hình dạng giữ chỗ mới. |
| height | Single | Chiều cao của hình dạng giữ chỗ mới. |

### Giá trị trả về

Được tạo [`IAutoShape`](../../iautoshape) với một vị trí giữ chỗ hình ảnh trực tuyến.

### Ví dụ

Ví dụ sau đây cho thấy cách thêm hình dạng giữ chỗ Hình ảnh Trực tuyến vào slide bố cục.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddOnlineImagePlaceholder(20, 20, 200, 200);
}
```

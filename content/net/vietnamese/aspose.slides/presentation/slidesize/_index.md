## Presentation.SlideSize thuộc tính

Trả về đối tượng kích thước slide. chỉ đọc [`ISlideSize`](../../islidesize).

```csharp
public ISlideSize SlideSize { get; }
```

### Ví dụ

Ví dụ sau cho thấy cách thay đổi kích thước slide trong một PowerPoint Presentation.

```csharp
[C#]
using (Presentation pres = new Presentation("pres-4x3-aspect-ratio.pptx"))
{
    pres.SlideSize.SetSize(SlideSizeType.OnScreen16x9, SlideSizeScaleType.DoNotScale);
    pres.Save("pres-4x3-aspect-ratio.pptx", SaveFormat.Pptx);
}
```

Ví dụ sau cho thấy cách đặt kích thước slide với việc co giãn nội dung cho một PowerPoint Presentation.

```csharp
[C#]
// Khởi tạo một đối tượng Presentation đại diện cho tệp bài thuyết trình
using(Presentation presentation = new Presentation("AccessSlides.pptx")) {
  using(Presentation auxPresentation = new Presentation()) {
    ISlide slide = presentation.Slides[0];
    // Đặt kích thước slide cho các bài thuyết trình được tạo thành kích thước của nguồn
    presentation.SlideSize.SetSize(540, 720, SlideSizeScaleType.EnsureFit); // Phương thức SetSize được sử dụng để đặt kích thước slide với nội dung được co giãn để đảm bảo vừa
    presentation.SlideSize.SetSize(SlideSizeType.A4Paper, SlideSizeScaleType.Maximize); // Phương thức SetSize được sử dụng để đặt kích thước slide với kích thước nội dung tối đa
    // Lưu Presentation vào đĩu
    auxPresentation.Save("Set_Size&Type_out.pptx", SaveFormat.Pptx);
  }
}
```

Ví dụ sau cho thấy cách chỉ định kích thước slide tùy chỉnh trong một PowerPoint Presentation.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    pres.SlideSize.SetSize(780, 540, SlideSizeScaleType.DoNotScale); // Kích thước giấy A4
    pres.Save("pres-a4-slide-size.pptx", SaveFormat.Pptx);
}
```

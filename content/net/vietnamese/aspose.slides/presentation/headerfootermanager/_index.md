## Presentation.HeaderFooterManager thuộc tính

Trả về trình quản lý HeaderFooter thực tế. Chỉ đọc [`IPresentationHeaderFooterManager`](../../ipresentationheaderfootermanager).

### Ví dụ

Ví dụ sau cho thấy cách thiết lập hiển thị chân trang trong Slide của PowerPoint Presentation.

```csharp
[C#]
using (Presentation presentation = new Presentation("presentation.ppt"))
{
    IBaseSlideHeaderFooterManager headerFooterManager = presentation.Slides[0].HeaderFooterManager;
	// Thuộc tính IsFooterVisible được sử dụng để chỉ ra rằng trình giữ chỗ chân trang của slide không tồn tại.
    if (!headerFooterManager.IsFooterVisible)
    {
		// Phương thức SetFooterVisibility được sử dụng để làm cho trình giữ chỗ chân trang của slide hiển thị.
        headerFooterManager.SetFooterVisibility(true);
    }
	// Thuộc tính IsSlideNumberVisible được sử dụng để chỉ ra rằng trình giữ chỗ số trang của slide không tồn tại.
    if (!headerFooterManager.IsSlideNumberVisible)
    {
		// Phương thức SetSlideNumberVisibility được sử dụng để làm cho trình giữ chỗ số trang của slide hiển thị.
        headerFooterManager.SetSlideNumberVisibility(true);
    }
	// Thuộc tính IsDateTimeVisible được sử dụng để chỉ ra rằng trình giữ chỗ ngày-giờ của slide không tồn tại.
    if (!headerFooterManager.IsDateTimeVisible)
    {
		// Phương thức SetFooterVisibility được sử dụng để làm cho trình giữ chỗ ngày-giờ của slide hiển thị.
        headerFooterManager.SetDateTimeVisibility(true);
    }
	// Phương thức SetFooterText được sử dụng để đặt văn bản cho trình giữ chỗ chân trang của slide.
    headerFooterManager.SetFooterText("Footer text");
	// Phương thức SetDateTimeText được sử dụng để đặt văn bản cho trình giữ chỗ ngày-giờ của slide.
    headerFooterManager.SetDateTimeText("Date and time text");
	presentation.Save("Presentation.ppt",SaveFormat.ppt);
}
```

	// Phương thức SetDateTimeAndChildDateTimesVisibility được sử dụng để làm cho slide master và tất cả các trình giữ chỗ ngày-giờ con hiển thị.

	// Phương thức SetFooterAndChildFootersText được sử dụng để đặt văn bản cho slide master và tất cả các trình giữ chỗ chân trang con.

}

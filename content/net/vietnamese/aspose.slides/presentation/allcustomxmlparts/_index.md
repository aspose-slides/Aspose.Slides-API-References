## Presentation.AllCustomXmlParts thuộc tính

Trả về tất cả các phần dữ liệu tùy chỉnh trong presentaion. Chỉ đọc [`ICustomXmlPart`](../../icustomxmlpart)[].

```csharp
public ICustomXmlPart[] AllCustomXmlParts { get; }
```

### Ví dụ

Các ví dụ sau cho thấy cách xóa tất cả các phần xml tùy chỉnh khỏi PowerPoint Presentation.

```csharp
[C#]
using (Presentation pres = new Presentation("PresentationWithCustomXml.pptx"))
{
	// Duyệt qua tất cả các phần XML tùy chỉnh
    foreach (ICustomXmlPart item in pres.AllCustomXmlParts)
    {
        item.Remove();
    }
    pres.Save("out.pptx", SaveFormat.Pptx);
}
```

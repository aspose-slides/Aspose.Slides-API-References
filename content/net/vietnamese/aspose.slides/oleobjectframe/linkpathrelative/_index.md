## OleObjectFrame.LinkPathRelative thuộc tính

Trả về đường dẫn tương đối tới tệp được liên kết nếu có, nếu không trả về chuỗi rỗng. Readonly String.

```csharp
public string LinkPathRelative { get; }
```

### Ghi chú

Trong các bản trình chiếu Ppt, một số liên kết đối tượng Ole có thể có biểu diễn tương đối.

### Ví dụ

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.ppt"))
{
    IOleObjectFrame oleFrame = presentation.Slides[0].Shapes[0] as IOleObjectFrame;

    if (oleFrame != null)
    {
        Console.WriteLine("The relative path: " + oleFrame.LinkPathRelative);
    } 
}
```

## OleObjectFrame.SetEmbeddedData phương thức

Đặt thông tin về dữ liệu OLE được nhúng. Phương thức này thay đổi các thuộc tính của đối tượng để phản ánh dữ liệu mới và đặt cờ IsObjectLink thành false, cho biết đối tượng OLE được nhúng.

```csharp
public void SetEmbeddedData(IOleEmbeddedDataInfo embeddedData)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| embeddedData | IOleEmbeddedDataInfo | Embedded data [`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo) |

### Ngoại lệ

| Ngoại lệ | Điều kiện |
| --- | --- |
| ArgumentNullException | Khi tham số embeddedData là null. |

### Ví dụ

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    OleObjectFrame oof = pres.Slides[0].Shapes[0] as OleObjectFrame;
    if (oof != null)
    {
        IOleEmbeddedDataInfo newData = new OleEmbeddedDataInfo(File.ReadAllBytes("Picture.png"), "png");
        oof.SetEmbeddedData(newData);
    }
}
```

## GetSubstitutions() {#getsubstitutions}

Lấy thông tin về các phông chữ sẽ được thay thế khi hiển thị bản trình chiếu.

```csharp
public IEnumerable<FontSubstitutionInfo> GetSubstitutions()
```

### Giá trị trả về

Bộ sưu tập tất cả các thay thế phông chữ [`FontSubstitutionInfo`](../../fontsubstitutioninfo).

### Ví dụ

```csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    foreach (FontSubstitutionInfo fontSubstitution in pres.FontsManager.GetSubstitutions())
    {
        Console.WriteLine("{0} -> {1}", fontSubstitution.OriginalFontName, fontSubstitution.SubstitutedFontName);
    }
}        
```

## GetSubstitutions(int[]) {#getsubstitutions_1}

Lấy thông tin về các phông chữ sẽ được thay thế trong quá trình hiển thị các slide được chỉ định.

```csharp
public IEnumerable<FontSubstitutionInfo> GetSubstitutions(int[] slides)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| slides | Int32[] | Mảng các chỉ số slide để lấy thông tin thay thế phông chữ, bắt đầu từ 1. |

### Giá trị trả về

Một bộ sưu tập tất cả các thay thế phông chữ ([`FontSubstitutionInfo`](../../fontsubstitutioninfo)) cho các slide được chỉ định.

### Ví dụ

```csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    int[] targetSlides = { 1, 2, 5 };
    foreach (var fontSubstitution in pres.FontsManager.GetSubstitutions(targetSlides))
    {
        Console.WriteLine("{0} -> {1}", fontSubstitution.OriginalFontName, fontSubstitution.SubstitutedFontName);
    }
}
```

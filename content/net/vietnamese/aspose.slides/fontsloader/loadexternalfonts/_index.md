## FontsLoader.LoadExternalFonts phương thức

Thêm các thư mục bổ sung để tìm phông chữ.

```csharp
public static void LoadExternalFonts(string[] directories)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| directories | String[] | Các thư mục để đọc phông chữ bổ sung. |

### Ví dụ

Các ví dụ sau cho thấy cách tải phông chữ tùy chỉnh từ .TTF

```csharp
[C#]
// Đường dẫn tới thư mục tài liệu.
string dataDir = "C:\\";
// các thư mục để tìm phông chữ
String[] folders = new String[] { dataDir };
// Tải phông chữ từ thư mục phông chữ tùy chỉnh
FontsLoader.LoadExternalFonts(folders);
// Thực hiện một số công việc và thực hiện việc render bản trình bày/slide
using (Presentation presentation = new Presentation(dataDir + "DefaultFonts.pptx"))
{
    presentation.Save(dataDir + "NewFonts_out.pptx", SaveFormat.Pptx);
	// Xóa bộ nhớ đệm phông chữ
	FontsLoader.ClearCache();
}
```

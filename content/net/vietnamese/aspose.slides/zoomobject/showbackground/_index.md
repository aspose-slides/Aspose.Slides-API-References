## ZoomObject.ShowBackground thuộc tính

Lấy hoặc đặt giá trị chỉ định liệu Zoom có sử dụng nền của slide đích hay không. Đọc/ghi Boolean. Giá trị mặc định: true

```csharp
public bool ShowBackground { get; set; }
```

### Ví dụ

ví dụ minh họa cách loại bỏ nền của hình ảnh của một đối tượng Zoom:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
    zoomFrame.ShowBackground = false;
}
```

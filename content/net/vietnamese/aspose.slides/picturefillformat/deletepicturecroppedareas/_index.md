## PictureFillFormat.DeletePictureCroppedAreas phương thức

Xóa các vùng đã cắt của Picture được dùng để tô.

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### Giá trị trả về

Hình ảnh đã cắt hoặc hình ảnh gốc nếu không cần cắt.

### Ghi chú

Phương thức này chuyển các tập tin metafile WMF/EMF sang ảnh PNG raster trong khi cắt.

### Ví dụ

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Lấy PictureFrame
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // Xóa các vùng đã cắt của hình ảnh PictureFrame
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

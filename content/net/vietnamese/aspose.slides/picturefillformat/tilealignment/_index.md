## PictureFillFormat.TileAlignment thuộc tính

Trả về hoặc đặt cách mà kết cấu được căn chỉnh trong hình dạng. Cài đặt này kiểm soát điểm bắt đầu của mẫu kết cấu và cách nó lặp lại trên hình dạng. Đọc/ghi [`RectangleAlignment`](../../rectanglealignment).

```csharp
public RectangleAlignment TileAlignment { get; set; }
```

### Ghi chú

Mặc định là TopLeft.

### Ví dụ

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Lấy định dạng tô hình ảnh của hình dạng
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Đặt chế độ tô hình ảnh thành Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Đặt căn chỉnh cho việc lát thành góc dưới bên phải
    pictureFillFormat.TileAlignment = RectangleAlignment.BottomRight;
}
```

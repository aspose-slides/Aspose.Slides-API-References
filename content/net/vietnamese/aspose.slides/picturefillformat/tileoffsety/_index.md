## PictureFillFormat.TileOffsetY thuộc tính

Trả về hoặc thiết lập độ lệch dọc của họa tiết so với gốc của hình dạng tính bằng điểm. Giá trị dương di chuyển họa tiết xuống dưới, trong khi giá trị âm di chuyển lên trên. Đọc/ghi Single.

```csharp
public float TileOffsetY { get; set; }
```

### Ví dụ

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Lấy định dạng tô hình ảnh của shape
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Đặt chế độ tô hình ảnh thành Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Đặt độ lệch dọc của họa tiết thành -50 điểm
    pictureFillFormat.TileOffsetY = -50;
}
```

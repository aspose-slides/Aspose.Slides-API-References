## PictureFillFormat.TileOffsetX thuộc tính

Trả về hoặc đặt độ lệch ngang của texture so với gốc của shape tính bằng điểm. Giá trị dương di chuyển texture sang phải, trong khi giá trị âm di chuyển nó sang trái. Đọc/ghi Single.

```csharp
public float TileOffsetX { get; set; }
```

### Ví dụ

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Lấy định dạng tô ảnh của shape
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Đặt chế độ tô ảnh thành Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Đặt độ lệch ngang của texture thành 20 điểm
    pictureFillFormat.TileOffsetX = 20f;
}
```

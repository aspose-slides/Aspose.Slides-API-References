## IDocumentProperties.GetSensitivityLabels phương thức

Lấy một mảng các nhãn độ nhạy từ các thuộc tính tài liệu tùy chỉnh (Microsoft Information Protection SDK Metadata).

```csharp
public ISensitivityLabel[] GetSensitivityLabels()
```

### Ví dụ

Mã sau đây cho thấy cách di chuyển thông tin nhãn độ nhạy từ các thuộc tính tài liệu tùy chỉnh sang bộ sưu tập SensitivityLabels hiện đại:

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    // Lấy nhãn độ nhạy từ các thuộc tính tài liệu tùy chỉnh
    ISensitivityLabel[] mipSensitivityLabels = pres.DocumentProperties.GetSensitivityLabels();

    ISensitivityLabelCollection sensitivityLabels = pres.SensitivityLabels;
    foreach (var sensitivityLabel in mipSensitivityLabels)
    {
        // Thêm nhãn vào bộ sưu tập
        // Ở đây bạn có thể thêm kiểm tra tính hợp lệ của thông tin nhãn (nhãn có sẵn, v.v.)
        sensitivityLabels.Add(sensitivityLabel);
    }

    pres.Save("SensitivityLabel.pptx", SaveFormat.Pptx);
}
```

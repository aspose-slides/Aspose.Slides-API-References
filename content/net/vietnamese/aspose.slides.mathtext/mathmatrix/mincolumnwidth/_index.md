## Thuộc tính MathMatrix.MinColumnWidth

Chiều rộng cột tối thiểu tính bằng twip (1/20 một điểm). Khoảng cách giữa các cột (cũng được gọi là “Column Gap” hoặc “Gap Width”) được cộng vào MinColumnWidth để xác định tổng Matrix Column Spacing (khoảng cách giữa các cạnh tương đồng của các cột khác nhau). Mặc định: 0.

```csharp
public uint MinColumnWidth { get; set; }
```

### Ví dụ

Ví dụ:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.MinColumnWidth = 20;
```

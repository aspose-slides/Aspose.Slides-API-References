## MathMatrix.RowGap thuộc tính

Giá trị của khoảng cách dọc giữa các hàng của ma trận; nếu RowGapRule được đặt thành 3 ("Exactly"), thì đơn vị được hiểu là twips (1/20 điểm); nếu RowGapRule được đặt thành 4 ("Multiple"), thì đơn vị được hiểu là nửa dòng. Mặc định: 0

```csharp
public uint RowGap { get; set; }
```

### Ví dụ

Ví dụ:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

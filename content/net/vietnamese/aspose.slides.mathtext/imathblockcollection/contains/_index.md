## IMathBlockCollection.Contains phương thức

Xác định xem bộ sưu tập có chứa một giá trị cụ thể hay không.

```csharp
public bool Contains(IMathBlock item)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| item | IMathBlock | Đối tượng cần tìm trong bộ sưu tập. |

### Giá trị trả về

true nếu *item* được tìm thấy trong bộ sưu tập; nếu không, false.

### Ví dụ

Ví dụ:

```csharp
[C#]
IMathBlockCollection blockCollection = new MathParagraph();
IMathBlock block = new MathBlock(new MathematicalText("y"));
blockCollection.Add(block);
bool contains = blockCollection.Contains(block);
```

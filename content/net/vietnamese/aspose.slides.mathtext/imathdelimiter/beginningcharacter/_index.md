## IMathDelimiter.BeginningCharacter thuộc tính

Ký tự bắt đầu dấu phân tách chỉ định ký tự đầu, hoặc ký tự mở, của dấu phân tách. Dấu phân tách toán học là các ký tự bao quanh như ngoặc tròn, ngoặc vuông và ngoặc nhọn. Giá trị mặc định: '('.

```csharp
public char BeginningCharacter { get; set; }
```

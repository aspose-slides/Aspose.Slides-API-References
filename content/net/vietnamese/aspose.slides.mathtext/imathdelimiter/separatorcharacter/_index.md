## IMathDelimiter.SeparatorCharacter thuộc tính

Delimiter Separator Character chỉ định ký tự phân tách các đối số trong đối tượng delimiter. Mặc định: '&#x7C;'.

```csharp
public char SeparatorCharacter { get; set; }
```

### Ví dụ

Ví dụ:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Join("y").Enclose();
delimiter.SeparatorCharacter = '$';
```

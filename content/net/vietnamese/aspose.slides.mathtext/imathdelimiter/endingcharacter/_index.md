## IMathDelimiter.EndingCharacter thuộc tính

Delimiter Ending Character chỉ định ký tự phân cách kết thúc, hoặc ký tự đóng. Các ký tự phân cách toán học là các ký tự bao quanh như dấu ngoặc tròn, dấu ngoặc vuông và dấu ngoặc nhọn. Mặc định: ')'.

```csharp
public char EndingCharacter { get; set; }
```

### Ví dụ

Ví dụ:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Join("y").Enclose();
delimiter.EndingCharacter = ']';
```

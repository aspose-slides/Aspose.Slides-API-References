## Enclose(char, char) {#enclose_1}

Bao quanh một phần tử toán học bằng các ký tự được chỉ định như dấu ngoặc hoặc các ký tự khác dưới dạng khung

```csharp
public virtual IMathDelimiter Enclose(char beginningCharacter, char endingCharacter)
```

| Parameter | Type | Description |
| --- | --- | --- |
| beginningCharacter | Char | Ký tự bắt đầu (thường là dấu ngoặc trái) |
| endingCharacter | Char | Ký tự kết thúc (thường là dấu ngoặc phải) |

### Return Value

Phần tử toán học kiểu [`IMathDelimiter`](../../imathdelimiter) bao gồm các ký tự được chỉ định làm khung

### Examples

Ví dụ:

```csharp
[C#]

IMathDelimiter delimiter = element.Enclose('[', ']');
```

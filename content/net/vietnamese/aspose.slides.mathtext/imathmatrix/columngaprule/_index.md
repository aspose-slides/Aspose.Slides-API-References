## IMathMatrix.ColumnGapRule thuộc tính

Kiểu của khoảng cách ngang giữa các cột của ma trận; Đơn vị khoảng cách ngang có thể là ems hoặc points (được lưu dưới dạng twips). Mặc định: SingleSpacingGap (0)

```csharp
public MathSpacingRules ColumnGapRule { get; set; }
```

### Ví dụ

Ví dụ:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.ColumnGapRule = MathSpacingRules.OneAndHalfSpacingGap;
```

## IMathBorderBox.StrikethroughBottomLeftToTopRight thuộc tính

Gạch ngang từ góc dưới trái lên góc trên phải (mặc định là false). Xác định trạng thái ẩn hoặc hiển thị của một đường gạch ngang chéo từ góc dưới trái đến góc trên phải của hộp viền.

```csharp
public bool StrikethroughBottomLeftToTopRight { get; set; }
```

### Ví dụ

Ví dụ:

```csharp
[C#]
IMathBorderBox borderBox = new MathematicalText("x+y+z").ToBorderBox();
borderBox.StrikethroughBottomLeftToTopRight = true;
```

## IMathArray.RowSpacing thuộc tính

Khoảng cách giữa các hàng của một mảng. Nó chỉ được sử dụng khi RowSpacingRule được đặt thành 3. Chính xác trong trường hợp này đơn vị đo là điểm hoặc Multiple trong trường hợp này đơn vị đo là nửa dòng. Mặc định: 0

```csharp
public uint RowSpacing { get; set; }
```

### Ví dụ

Example:

```csharp
[C#]
IMathArray mathArray = new MathArray(new MathematicalText("item1"));
mathArray.RowSpacingRule = MathRowSpacingRule.Exactly;
mathArray.RowSpacing = 10;
```

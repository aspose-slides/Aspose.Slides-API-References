## MathDelimiter.EndingCharacter thuộc tính

Delimiter Ending Character chỉ định ký tự delimiter kết thúc, hoặc đóng. Các delimiter toán học là các ký tự bao quanh như ngoặc tròn, ngoặc vuông và ngoặc nhọn. Mặc định: ')'.

```csharp
public char EndingCharacter { get; set; }
```

### Ví dụ

Ví dụ:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Join("y").Enclose();
delimiter.EndingCharacter = ']';
```

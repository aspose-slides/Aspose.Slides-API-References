## MathDelimiter.GrowToMatchOperandHeight thuộc tính

Xác định sự mở rộng của BeginningCharacter, SeparatorCharacter, EndingCharacter. Khi true, các dấu phân cách sẽ mở rộng theo chiều dọc để phù hợp với chiều cao toán hạng của chúng. Giá trị mặc định là true

```csharp
public bool GrowToMatchOperandHeight { get; set; }
```

### Ví dụ

Ví dụ:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.GrowToMatchOperandHeight = false;
```

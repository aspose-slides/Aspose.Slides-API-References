## MathDelimiter.BeginningCharacter thuộc tính

Delimiter Beginning Character chỉ định ký tự dấu phân cách bắt đầu, hay ký tự mở. Các dấu phân cách toán học là các ký tự bao quanh như ngoặc tròn, ngoặc vuông và ngoặc nhọn. Mặc định: '('.

```csharp
public char BeginningCharacter { get; set; }
```

### Ví dụ

Ví dụ:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Join("y").Enclose();
delimiter.BeginningCharacter = '[';
```

## ToTiff(Presentation, string, ITiffOptions, bool) {#totiff_1}

Chuyển đổi bản trình bày đầu vào sang định dạng TIFF với các tùy chọn tùy chỉnh. Nếu tên tệp đầu ra được cung cấp là "myPath/myFilename.tiff" và *multipage* là `false`, kết quả sẽ được lưu dưới dạng tập hợp các tệp "myPath/myFilename_N.tiff", trong đó N là số slide. Ngược lại, nếu *multipage* là `true`, kết quả sẽ là tài liệu đa trang "myPath/myFilename.tiff".

```csharp
public static void ToTiff(Presentation pres, string outputFileName, ITiffOptions options, 
    bool multipage)
```

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| pres | Presentation | Bản trình bày đầu vào. |
| outputFileName | String | Tên tệp đầu ra. |
| options | ITiffOptions | Các tùy chọn lưu TIFF. |
| multipage | Boolean | Xác định xem tài liệu TIFF được tạo ra có nên là đa trang hay không. |

### Ngoại lệ

| ngoại lệ | điều kiện |
| --- | --- |
| ArgumentException |  |

### Ví dụ

```csharp
ITiffOptions options = new TiffOptions()
{
    CompressionType = TiffCompressionTypes.CCITT3,
    SlidesLayoutOptions = new NotesCommentsLayoutingOptions()
    {
        NotesPosition = NotesPositions.BottomTruncated
    }
};

using (var pres = new Presentation("pres.pptx"))
   Convert.ToTiff(pres, "pres.tiff", options, false);
```

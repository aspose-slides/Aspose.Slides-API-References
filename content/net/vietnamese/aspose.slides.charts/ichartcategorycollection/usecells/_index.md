## IChartCategoryCollection.UseCells thuộc tính

Nếu true thì bảng tính được sử dụng để lưu trữ các danh mục (trường hợp này hỗ trợ danh mục đa cấp). Nếu false thì bảng tính KHÔNG được sử dụng để lưu trữ các giá trị (và trường hợp này không hỗ trợ danh mục đa cấp). Đọc/ghi Boolean.

```csharp
public bool UseCells { get; set; }
```

## IChartSeriesGroup.Overlap property

Xác định mức độ chồng chập của các thanh và cột trên biểu đồ 2-D, tính bằng phần trăm (từ -100% đến 100%). - -100%: Khoảng cách tối đa (các thanh hoàn toàn tách rời). - 0%: Các thanh được đặt cạnh nhau mà không chồng lẫn nhau hoặc có khoảng cách. - 100%: Chồng lẫn tối đa (các thanh hoàn toàn chồng lên nhau). Thuộc tính này là đọc/ghi SByte.

```csharp
public sbyte Overlap { get; set; }
```

### Ngoại lệ

| ngoại lệ | điều kiện |
| --- | --- |
| ArgumentOutOfRangeException | Được ném khi giá trị được đặt ngoài phạm vi hợp lệ từ -100 đến 100. |

### Ví dụ

Ví dụ sau minh họa cách đặt giá trị overlap cho một nhóm chuỗi biểu đồ và hiển thị biểu đồ kết quả trên một form:

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // Đặt overlap thành 55%

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

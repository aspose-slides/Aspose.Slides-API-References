## DataLabelFormat.NumberFormat property

Biểu diễn chuỗi định dạng cho đối tượng DataLabels. Đọc/ghi String.

```csharp
public string NumberFormat { get; set; }
```

### Ghi chú

Nếu lớp cha của đối tượng DataLabelFormat này là một bộ sưu tập DataLabelCollection chứa các nhãn dữ liệu, thì thuộc tính này lấy hoặc đặt giá trị mặc định của thuộc tính NumberFormat cho các nhãn dữ liệu mới trong bộ sưu tập DataLabelCollection. Khi thuộc tính này được đặt một giá trị, giá trị đó cũng được đặt cho thuộc tính NumberFormat của tất cả các nhãn dữ liệu trong bộ sưu tập DataLabelCollection (i.e. "DataLabels.DefaultDataLabelFormat.NumberFormat = val;" khiến tất cả DataLabels[i].NumberFormat bằng val).

### Ví dụ

```csharp
[C#]
series.Labels.DefaultDataLabelFormat.ShowValue = true;
series.Labels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = false;
series.Labels.DefaultDataLabelFormat.NumberFormat = "0.0%";
```

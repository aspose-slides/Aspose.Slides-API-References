## DataLabel.IsVisible thuộc tính

False có nghĩa là nhãn dữ liệu không hiển thị (và do đó tất cả các cờ Show*-flags (ShowValue, ...) đều sai). Boolean chỉ đọc.

```csharp
public bool IsVisible { get; }
```

### Ghi chú

Nếu nhãn dữ liệu hiển thị, bạn có thể ẩn nó bằng phương thức Hide(). Nhưng nếu nhãn dữ liệu không hiển thị (IsVisible là false) bạn có thể làm nhãn dữ liệu hiển thị bằng cách thiết lập các cờ Show*-flags (ShowValue, ...) thành trạng thái true.

## IDataLabelFormat.NumberFormat thuộc tính

Represents the format string for the DataLabels object. Đọc/ghi String.

```csharp
public string NumberFormat { get; set; }
```

### Ghi chú

Nếu cha mẹ của đối tượng DataLabelFormat này là một DataLabelCollection collection of data labels, thì thuộc tính này sẽ lấy hoặc đặt giá trị mặc định của thuộc tính NumberFormat cho các nhãn dữ liệu mới trong DataLabelCollection collection. Khi thuộc tính này được đặt bằng một giá trị, giá trị đó cũng được đặt cho thuộc tính NumberFormat của tất cả các nhãn dữ liệu trong DataLabelCollection collection (i.e. "DataLabels.DefaultDataLabelFormat.NumberFormat = val;" gây ra tất cả DataLabels[i].NumberFormat bằng val).

### Ví dụ

```csharp
[C#]
series.Labels.DefaultDataLabelFormat.ShowValue = true;
series.Labels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = false;
series.Labels.DefaultDataLabelFormat.NumberFormat = "0.0%";
```

## ChartData.UseSecondaryCategories thuộc tính

Nếu false thì [`SecondaryCategories`](../secondarycategories) thuộc tính trả về null và dữ liệu trong [`Categories`](../categories) thuộc tính được sử dụng cho cả chuỗi chính và chuỗi phụ. Nếu true thì dữ liệu trong [`SecondaryCategories`](../secondarycategories) thuộc tính được sử dụng cho chuỗi phụ và dữ liệu trong [`Categories`](../categories) thuộc tính được sử dụng cho chuỗi chính. Đọc/ghi Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

### Ví dụ

Ví dụ. Các danh mục nào liên quan đến chuỗi - ChartData.Categories hoặc ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // các danh mục liên quan là series.Chart.ChartData.SecondaryCategories
}
else
{
    // các danh mục liên quan là series.Chart.ChartData.Categories
}
```

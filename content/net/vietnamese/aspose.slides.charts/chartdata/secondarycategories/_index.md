## ChartData.SecondaryCategories thuộc tính

Lấy các danh mục phụ nếu [`UseSecondaryCategories`](../usesecondarycategories) thuộc tính là true. Chỉ đọc [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### Ghi chú

Nếu [`UseSecondaryCategories`](../usesecondarycategories) thuộc tính là false thì `SecondaryCategories` thuộc tính này trả về null và dữ liệu trong [`Categories`](../categories) thuộc tính được sử dụng cho cả chuỗi chính và chuỗi phụ. Nếu [`UseSecondaryCategories`](../usesecondarycategories) thuộc tính là true thì dữ liệu trong `SecondaryCategories` thuộc tính này được sử dụng cho chuỗi phụ và dữ liệu trong [`Categories`](../categories) thuộc tính được sử dụng cho chuỗi chính.

### Ví dụ

Ví dụ. Các danh mục nào liên quan đến chuỗi - ChartData.Categories hoặc ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // các danh mục liên quan là series.Chart.ChartData.SecondaryCategories
}
else
{
    // các danh mục liên quan là series.Chart.ChartData.Categories
}
```

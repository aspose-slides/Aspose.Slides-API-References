## Phương thức ChartData.GetRange

Lấy phạm vi dữ liệu biểu đồ.

```csharp
public string GetRange()
```

### Giá trị trả về

Công thức phạm vi dữ liệu của các ô. E.g: "Sheet1!$A$1:$C$4"

### Ngoại lệ

| ngoại lệ | điều kiện |
| --- | --- |
| InvalidOperationException | Biểu đồ không sử dụng workbook làm nguồn dữ liệu |

### Ví dụ

Ví dụ C#

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.PercentsStackedBar, 0, 0, 100, 100);
    string result = (chart.ChartData as ChartData).GetRange();
}
```

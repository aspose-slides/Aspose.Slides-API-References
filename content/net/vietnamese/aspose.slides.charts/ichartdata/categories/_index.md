## IChartData.Categories property

Lấy các danh mục chính (hoặc cả danh mục chính và phụ nếu thuộc tính [`UseSecondaryCategories`](../usesecondarycategories) là false). Chỉ đọc [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection Categories { get; }
```

### Ghi chú

Nếu thuộc tính [`UseSecondaryCategories`](../usesecondarycategories) là false thì thuộc tính [`SecondaryCategories`](../secondarycategories) trả về null và dữ liệu trong thuộc tính `Categories` này được sử dụng cho cả series chính và phụ. Nếu thuộc tính [`UseSecondaryCategories`](../usesecondarycategories) là true thì dữ liệu trong thuộc tính [`SecondaryCategories`](../secondarycategories) được sử dụng cho series phụ và dữ liệu trong thuộc tính `Categories` này được sử dụng cho series chính.

### Ví dụ

Ví dụ. Các danh mục nào liên quan đến series - ChartData.Categories hay ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // các danh mục liên quan là series.Chart.ChartData.SecondaryCategories
}
else
{
    // các danh mục liên quan là series.Chart.ChartData.Categories
}
```

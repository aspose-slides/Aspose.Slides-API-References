## IChartData.SecondaryCategories thuộc tính

Lấy các danh mục phụ nếu thuộc tính [`UseSecondaryCategories`](../usesecondarycategories) là true. Chỉ-đọc [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### Ghi chú

Nếu thuộc tính [`UseSecondaryCategories`](../usesecondarycategories) là false thì thuộc tính `SecondaryCategories` này trả về null và dữ liệu trong thuộc tính [`Categories`](../categories) được sử dụng cho cả chuỗi chính và chuỗi phụ. Nếu thuộc tính [`UseSecondaryCategories`](../usesecondarycategories) là true thì dữ liệu trong thuộc tính `SecondaryCategories` này được sử dụng cho chuỗi phụ và dữ liệu trong thuộc tính [`Categories`](../categories) được sử dụng cho chuỗi chính.

### Ví dụ

Ví dụ. Các danh mục nào liên quan đến chuỗi - ChartData.Categories hoặc ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // các danh mục liên quan là series.Chart.ChartData.SecondaryCategories
}
else
{
    // các danh mục liên quan là series.Chart.ChartData.Categories
}
```

## IInkBrush.InkEffect thuộc tính

Lấy loại hiệu ứng mực (ví dụ: Galaxy, Gold, Silver) xác định phong cách hiển thị của nét mực. Giá trị được phân tích từ thuộc tính brush "inkEffects". Nếu không có hiệu ứng nào được nhận dạng, NotDefined được trả về.

```csharp
public InkEffectType InkEffect { get; }
```

### Ví dụ

Ví dụ:

```csharp
[C#]
using (var pres = new Presentation("Presentation.pptx"))
{
    Ink ink = pres.Slides[0].Shapes[0] as Ink;
    IInkBrush brush = ink.Traces[0].Brush;
    Console.WriteLine("InkEffects = {0}", brush.InkEffect);
}
```

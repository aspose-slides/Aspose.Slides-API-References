## IRenderingOptions.DisableFontLigatures thuộc tính

Nhận hoặc đặt một giá trị cho biết liệu văn bản có được hiển thị mà không sử dụng ligature hay không. Khi được đặt thành `true`, các ligature sẽ bị tắt trong đầu ra đã được hiển thị. Mặc định, thuộc tính này được đặt là `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Ví dụ

Ví dụ:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    RenderingOptions options = new RenderingOptions
    {
        DisableFontLigatures = true // Vô hiệu hoá ligature trong việc hiển thị văn bản
    };
    
    Bitmap[] renderedSlides = pres.GetImage(options);
    for (var index = 0; index < renderedSlides.Length; index++)
    {
        var slideImage = renderedSlides[index];
        slideImage.Save($"slide-{index}.png");
    }
}
```

## Html5Options.DisableFontLigatures thuộc tính

Lấy hoặc đặt một giá trị cho biết liệu văn bản có được hiển thị mà không sử dụng các ligature hay không. Khi được đặt thành `true`, các ligature sẽ bị tắt trong đầu ra đã render. Mặc định, thuộc tính này được đặt là `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Ví dụ

Example:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    Html5Options options = new Html5Options
    {
        DisableFontLigatures = true // Vô hiệu hoá ligature trong việc render văn bản
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html5, options);
}
```

## IPptxOptions.CompressionLevel thuộc tính

Xác định mức nén được sử dụng khi lưu tài liệu bài thuyết trình. Giá trị mặc định là Level6.

```csharp
public CompressionLevel CompressionLevel { get; set; }
```

### Ghi chú

Mức nén cao hơn tạo ra các tệp có kích thước nhỏ hơn nhưng yêu cầu thời gian xử lý lâu hơn. Tỷ lệ nén thực tế phụ thuộc vào nội dung của bài thuyết trình.

### Ví dụ

Ví dụ:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("demo-level8.pptx", SaveFormat.Pptx, new PptxOptions()
    {
        CompressionLevel = CompressionLevel.Level8
    });
}
```

## ISwfOptions.SlidesLayoutOptions thuộc tính

Lấy hoặc đặt chế độ mà các slide được đặt trên trang khi xuất bản một bài thuyết trình [`ISlidesLayoutOptions`](../../islideslayoutoptions). Thuộc tính này không hỗ trợ gán các đối tượng kiểu `Aspose.Slides.Export.HandoutLayoutingOptions`

```csharp
public ISlidesLayoutOptions SlidesLayoutOptions { get; set; }
```

### Ví dụ

Ví dụ:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    SwfOptions options = new SwfOptions
    {
        SlidesLayoutOptions = new NotesCommentsLayoutingOptions
        {
            CommentsPosition = CommentsPositions.Right
        }
    };
    
    pres.Save("pres.swf", SaveFormat.Swf, options);
}
```

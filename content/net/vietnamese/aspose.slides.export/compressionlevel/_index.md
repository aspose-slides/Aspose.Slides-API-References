## CompressionLevel liệt kê

Xác định mức nén ZIP cho tệp OpenXML. Các mức cao hơn cung cấp mức nén tốt hơn nhưng làm chậm quá trình xử lý.

```csharp
public enum CompressionLevel
```

### Giá trị

| Tên | Giá trị | Mô tả |
| --- | --- | --- |
| None | `0` | Không áp dụng nén. Các tệp được lưu nguyên. |
| Level1 | `1` | Nén nhanh nhất với tỷ lệ nén thấp nhất. |
| Level2 | `2` | Nén nhanh hơn với tỷ lệ nén hơi tốt hơn so với Level1. |
| Level3 | `3` | Cung cấp mức nén tốt hơn Level2 với tác động hiệu năng vừa phải. |
| Level4 | `4` | Cung cấp mức nén tốt hơn Level3. |
| Level5 | `5` | Cung cấp mức nén cải thiện so với Level4 với thời gian xử lý bổ sung. |
| Level6 | `6` | Nén chuẩn, cung cấp sự cân bằng tốt giữa tốc độ nén và kích thước tệp. Mức nén mặc định. |
| Level7 | `7` | Cung cấp mức nén cao hơn Level6 nhưng xử lý chậm hơn. |
| Level8 | `8` | Cung cấp mức nén cao hơn Level7. |
| Level9 | `9` | Nén tối đa. Tạo kích thước tệp nhỏ nhất với tốc độ xử lý chậm nhất. |

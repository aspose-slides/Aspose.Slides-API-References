## SaveOptions.SkipJavaScriptLinks thuộc tính

Xác định liệu có bỏ qua các siêu liên kết có lời gọi JavaScript khi lưu bản trình chiếu hay không. Đọc/ghi Boolean. Giá trị mặc định là **false**.

```csharp
public bool SkipJavaScriptLinks { get; set; }
```

### Ghi chú

Khi thuộc tính này được đặt là **true**, các siêu liên kết có lời gọi JavaScript sẽ bị bỏ qua khi lưu.

Khi thuộc tính này được đặt là **false**, tất cả các siêu liên kết sẽ được lưu.

### Ví dụ

Ví dụ:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("result_without_JavaScript_links.html", SaveFormat.Html, new HtmlOptions()
    {
        SkipJavaScriptLinks = true
    });
}
```

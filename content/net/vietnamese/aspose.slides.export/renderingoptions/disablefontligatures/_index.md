## Thuộc tính RenderingOptions.DisableFontLigatures

Lấy hoặc thiết lập giá trị cho biết văn bản có được hiển thị mà không sử dụng các ligature hay không. Khi được đặt thành `true`, các ligature sẽ bị tắt trong kết quả hiển thị. Mặc định, thuộc tính này được đặt thành `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Ví dụ

Ví dụ:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    RenderingOptions options = new RenderingOptions
    {
        DisableFontLigatures = true // Vô hiệu hoá ligature trong việc render văn bản
    };
    
    Bitmap[] renderedSlides = pres.GetImage(options);
    for (var index = 0; index < renderedSlides.Length; index++)
    {
        var slideImage = renderedSlides[index];
        slideImage.Save($"slide-{index}.png");
    }
}
```

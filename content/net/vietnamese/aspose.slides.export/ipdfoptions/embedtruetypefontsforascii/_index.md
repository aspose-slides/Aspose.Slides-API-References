## IPdfOptions.EmbedTrueTypeFontsForASCII thuộc tính

Đúng để nhúng phông chữ TrueType cho các ký tự ASCII 32-127. Các phông chữ cho mã ký tự lớn hơn 127 luôn được nhúng. Đọc/ghi Boolean.

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### Ghi chú

Mặc định là **true**.

## PptxOptions.Zip64Mode thuộc tính

Xác định xem định dạng ZIP64 có được sử dụng cho tài liệu Presentation hay không. Giá trị mặc định là IfNecessary

```csharp
public Zip64Mode Zip64Mode { get; set; }
```

### Ví dụ

Ví dụ:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("demo-zip64.pptx", SaveFormat.Pptx, new PptxOptions()
    {
        Zip64Mode = Zip64Mode.Always
    });
}
```

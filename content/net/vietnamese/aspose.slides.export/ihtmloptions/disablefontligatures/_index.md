## IHtmlOptions.DisableFontLigatures thuộc tính

Lấy hoặc đặt giá trị cho biết liệu văn bản có được hiển thị mà không sử dụng ligature hay không. Khi được đặt thành `true`, các ligature sẽ bị tắt trong đầu ra đã render. Mặc định, thuộc tính này được đặt thành `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Ví dụ

Ví dụ:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    HtmlOptions options = new HtmlOptions
    {
        DisableFontLigatures = true // Vô hiệu hoá ligature trong việc render văn bản
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html, options);
}
```

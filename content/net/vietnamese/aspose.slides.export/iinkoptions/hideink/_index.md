## IInkOptions.HideInk thuộc tính

Hiển thị hoặc ẩn các phần tử Ink trong tài liệu đã xuất.

```csharp
public bool HideInk { get; set; }
```

### Ghi chú

Giá trị mặc định là false.

### Ví dụ

Ví dụ tiếp theo minh họa cách ẩn các phần tử Ink trong tài liệu PDF đã xuất:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.HideInk = true;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

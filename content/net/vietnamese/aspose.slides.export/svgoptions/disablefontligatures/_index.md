## SVGOptions.DisableFontLigatures thuộc tính

Lấy hoặc đặt một giá trị cho biết liệu văn bản có được hiển thị mà không sử dụng các ligature hay không. Khi đặt thành `true`, các ligature sẽ bị tắt trong đầu ra đã hiển thị. Mặc định, thuộc tính này được đặt thành `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Ví dụ

Ví dụ:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    SVGOptions options = new SVFOptions
    {
        DisableFontLigatures = true // Tắt các ligature trong việc hiển thị văn bản
    };
    
    using (FileStream fileStream = new FileStream("slide-0.svg", FileMode.Create, FileAccess.Write))
    {
        pres.Slides[0].WriteAsSvg(fileStream);   
    }
}
```

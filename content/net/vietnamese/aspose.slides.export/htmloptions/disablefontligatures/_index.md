## HtmlOptions.DisableFontLigatures thuộc tính

Lấy hoặc đặt một giá trị cho biết liệu văn bản có được hiển thị mà không sử dụng ligatures hay không. Khi được đặt thành `true`, các ligatures sẽ bị tắt trong đầu ra đã kết xuất. Mặc định, thuộc tính này được đặt thành `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Ví dụ

Ví dụ:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    HtmlOptions options = new HtmlOptions
    {
        DisableFontLigatures = true // Vô hiệu hoá ligatures trong việc hiển thị văn bản
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html, options);
}
```

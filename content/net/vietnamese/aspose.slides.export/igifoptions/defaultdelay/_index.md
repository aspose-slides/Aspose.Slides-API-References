## IGifOptions.DefaultDelay thuộc tính

Lấy hoặc đặt thời gian trễ mặc định [ms]. Giá trị này sẽ được sử dụng nếu [`AdvanceAfterTime`](../../../aspose.slides/islideshowtransition/advanceaftertime) không được đặt. Giá trị mặc định là 1000.

```csharp
public int DefaultDelay { get; set; }
```

### Ví dụ

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
	pres.Save("pres.gif", SaveFormat.Gif, new GifOptions { DefaultDelay = 2000 });
}
```

## IGifOptions.ExportHiddenSlides thuộc tính

Xác định liệu các slide ẩn có được xuất hay không. Giá trị mặc định là false.

```csharp
public bool ExportHiddenSlides { get; set; }
```

### Ví dụ

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
	pres.Save("pres.gif", SaveFormat.Gif, new GifOptions { ExportHiddenSlides = false });
}
```

## IHtml5Options.DisableFontLigatures thuộc tính

Lấy hoặc đặt một giá trị chỉ ra liệu văn bản có được hiển thị mà không sử dụng ligature hay không. Khi đặt thành `true`, ligature sẽ bị tắt trong đầu ra được hiển thị. Mặc định, thuộc tính này được đặt thành `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Ví dụ

Ví dụ:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    Html5Options options = new Html5Options
    {
        DisableFontLigatures = true // Tắt ligature trong việc hiển thị văn bản
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html5, options);
}
```

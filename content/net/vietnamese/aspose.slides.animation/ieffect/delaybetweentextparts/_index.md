## IEffect.DelayBetweenTextParts thuộc tính

Xác định độ trễ giữa các phần văn bản được hoạt hình (từ hoặc ký tự). Giá trị dương chỉ tỷ lệ phần trăm của thời lượng hiệu ứng. Giá trị âm chỉ độ trễ tính bằng giây. Đọc/ghi Single.

```csharp
public float DelayBetweenTextParts { get; set; }
```

### Ví dụ

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Lấy hiệu ứng đầu tiên của slide đầu tiên.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Thay đổi kiểu Animate text của hiệu ứng thành "By word"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByWord;
    
    // Đặt độ trễ giữa các phần văn bản được hoạt hình thành 20% thời lượng hiệu ứng.
    firstSlideEffect.DelayBetweenTextParts = 20f;
}
```

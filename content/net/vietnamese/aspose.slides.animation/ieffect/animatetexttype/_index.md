## IEffect.AnimateTextType thuộc tính

Xác định một kiểu văn bản hoạt hình cho hiệu ứng. Văn bản hình dạng có thể được hoạt hình theo chữ, theo từ hoặc toàn bộ cùng một lúc. Đọc/ghi `AnimateTextType`.

```csharp
public AnimateTextType AnimateTextType { get; set; }
```

### Ví dụ

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Lấy hiệu ứng đầu tiên của slide đầu tiên.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Thay đổi kiểu Animate text type của hiệu ứng thành "By letter"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByLetter;
}
```

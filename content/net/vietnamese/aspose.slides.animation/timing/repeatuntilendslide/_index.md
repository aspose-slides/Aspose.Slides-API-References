## Timing.RepeatUntilEndSlide thuộc tính

Thuộc tính này xác định liệu hiệu ứng có lặp lại cho đến cuối slide hay không. Đọc/ghi Boolean.

```csharp
public bool RepeatUntilEndSlide { get; set; }
```

### Ví dụ

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Lấy chuỗi hiệu ứng cho slide đầu tiên
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Lấy hiệu ứng đầu tiên của chuỗi chính.
    IEffect effect = effectsSequence[0];

    // Thay đổi Timing/Repeat của hiệu ứng thành "Cho đến cuối slide"
    effect.Timing.RepeatUntilEndSlide = true;
}
```

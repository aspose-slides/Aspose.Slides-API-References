## Effect.AnimateTextType thuộc tính

Xác định loại hoạt ảnh văn bản cho hiệu ứng. Văn bản hình dạng có thể được hoạt ảnh theo ký tự, theo từ hoặc đồng thời. Đọc/ghi `AnimateTextType`.

```csharp
public AnimateTextType AnimateTextType { get; set; }
```

### Ví dụ

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Lấy hiệu ứng đầu tiên của slide đầu tiên.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Thay đổi loại hoạt ảnh văn bản của hiệu ứng thành "Theo ký tự"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByLetter;
}
```

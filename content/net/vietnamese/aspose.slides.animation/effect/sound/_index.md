## Effect.Sound thuộc tính

Định nghĩa âm thanh nhúng cho hiệu ứng. Đọc/ghi [`IAudio`](../../../aspose.slides/iaudio).

```csharp
public IAudio Sound { get; set; }
```

### Ví dụ

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];
    
    // Lấy chuỗi hiệu ứng cho slide
    ISequence effectsSequence = slide.Timeline.MainSequence;
       
    foreach (IEffect effect in effectsSequence)
    {
        if (effect.Sound == null)
            continue;
        
        // Trích xuất âm thanh của hiệu ứng thành mảng byte
        byte[] audio = effect.Sound.BinaryData;
    }
}
```

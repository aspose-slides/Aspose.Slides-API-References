## ITiming.Rewind thuộc tính

Thuộc tính này chỉ định liệu hiệu ứng có được tua lại khi phát xong hay không. Đọc/ghi Boolean.

```csharp
public bool Rewind { get; set; }
```

### Ví dụ

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Lấy chuỗi hiệu ứng cho slide đầu tiên
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Lấy hiệu ứng đầu tiên của chuỗi chính.
    IEffect effect = effectsSequence[0];

    // Bật tính năng Timing/Rewind cho hiệu ứng.
    effect.Timing.Rewind = true;
}
```

## ITiming.RepeatUntilEndSlide thuộc tính

Thuộc tính này chỉ định liệu hiệu ứng có lặp lại cho đến khi kết thúc slide hay không. Đọc/ghi Boolean.

```csharp
public bool RepeatUntilEndSlide { get; set; }
```

### Ví dụ

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Lấy chuỗi hiệu ứng cho slide đầu tiên
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Lấy hiệu ứng đầu tiên của chuỗi chính.
    IEffect effect = effectsSequence[0];

    // Thay đổi Timing/Lặp lại của hiệu ứng thành "Until End of Slide"
    effect.Timing.RepeatUntilEndSlide = true;
}
```

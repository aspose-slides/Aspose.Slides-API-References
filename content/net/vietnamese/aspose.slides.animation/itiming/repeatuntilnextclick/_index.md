## ITiming.RepeatUntilNextClick thuộc tính

Thuộc tính này xác định liệu hiệu ứng có lặp lại cho đến lần nhấp tiếp theo hay không. Đọc/ghi Boolean.

```csharp
public bool RepeatUntilNextClick { get; set; }
```

### Ví dụ

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Lấy chuỗi hiệu ứng cho slide đầu tiên
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Lấy hiệu ứng đầu tiên của chuỗi chính.
    IEffect effect = effectsSequence[0];

    // Thay đổi Timing/Lặp lại của hiệu ứng thành "Until Next Click"
    effect.Timing.RepeatUntilNextClick = true;
}
```

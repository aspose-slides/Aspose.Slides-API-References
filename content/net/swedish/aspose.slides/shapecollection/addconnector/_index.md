## AddConnector(ShapeType, float, float, float, float) {#addconnector}

Skapar en ny anslutningsform med standardmallstil och lägger till den i slutet av formens samling.

```csharp
public IConnector AddConnector(ShapeType shapeType, float x, float y, float width, float height)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| shapeType | ShapeType | Den [`ShapeType`](../../shapetype) för anslutningsformen som ska läggas till. |
| x | Single | x-koordinaten för anslutningens ram, i punkter. |
| y | Single | y-koordinaten för anslutningens ram, i punkter. |
| width | Single | Bredden på anslutningens ram, i punkter. |
| height | Single | Höjden på anslutningens ram, i punkter. |

### Returvärde

Det nyss skapade [`IConnector`](../../iconnector).

### Exempel

Följande exempel visar hur man lägger till en anslutning (en böjd anslutning) mellan två former (en ellips och en rektangel) i en PowerPoint-presentation.

```csharp
[C#]
// Skapar en presentationsklass som representerar en PPTX-fil
using (Presentation input = new Presentation())
{
    // Kommer åt formsamlingen för en specifik bild
    IShapeCollection shapes = input.Slides[0].Shapes;
    // Lägger till en Ellipse-autoshape
    IAutoShape ellipse = shapes.AddAutoShape(ShapeType.Ellipse, 0, 100, 100, 100);
    // Lägger till en Rectangle-autoshape
    IAutoShape rectangle = shapes.AddAutoShape(ShapeType.Rectangle, 100, 300, 100, 100);
    // Lägger till en connector-form i bildens formsamling
    IConnector connector = shapes.AddConnector(ShapeType.BentConnector2, 0, 0, 10, 10);
    // Kopplar formerna med hjälp av connector
    connector.StartShapeConnectedTo = ellipse;
    connector.EndShapeConnectedTo = rectangle;
    // Anropar Reroute som sätter den automatiska kortaste vägen mellan former
    connector.Reroute();
    // Sparar presentationen
    input.Save("Shapes-connector.pptx", SaveFormat.Pptx);
}
```

## InsertSectionZoomFrame(int, float, float, float, float, ISection) {#insertsectionzoomframe}

Skapar en ny Section Zoom frame och infogar den i formsamlingen på det angivna indexet.

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| index | Int32 | Det nollbaserade index där Section Zoom frame ska infogas. |
| x | Single | x-koordinaten för den nya Section Zoom frame, i punkter. |
| y | Single | y-koordinaten för den nya Section Zoom frame, i punkter. |
| width | Single | Bredden på den nya Section Zoom frame, i punkter. |
| height | Single | Höjden på den nya Section Zoom frame, i punkter. |
| section | ISection | [`ISection`](../../isection) som refereras av Section Zoom frame; måste tillhöra denna presentation och innehålla minst ett bildspel. |

### Return Value

Den nyss skapade [`ISectionZoomFrame`](../../isectionzoomframe).

### Exceptions

| Undantag | Villkor |
| --- | --- |
| ArgumentException | Kastas om den refererade sektionen inte tillhör den aktuella presentationen eller saknar bildspel. |

### Examples

Detta exempel visar hur man skapar och infogar ett Section Zoom-objekt på det angivna indexet i en samling (förutsätt att presentationen “Presentation.pptx” innehåller minst två sektioner):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSectionZoomFrame(2, 150, 20, 50, 50, pres.Sections[1]);
}
```

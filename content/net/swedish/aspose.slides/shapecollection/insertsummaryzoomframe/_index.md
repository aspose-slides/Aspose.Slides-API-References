## ShapeCollection.InsertSummaryZoomFrame metod

Skapar en ny Summary Zoom-ram och infogar den i shape-samlingen på det angivna indexet.

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| index | Int32 | Det nollbaserade indexet där Summary Zoom-ramen ska infogas. |
| x | Single | x-koordinaten för den nya Summary Zoom-ramen, i punkter. |
| y | Single | y-koordinaten för den nya Summary Zoom-ramen, i punkter. |
| width | Single | Bredden på den nya Summary Zoom-ramen, i punkter. |
| height | Single | Höjden på den nya Summary Zoom-ramen, i punkter. |

### Returvärde

Det nyss skapade [`ISummaryZoomFrame`](../../isummaryzoomframe).

### Undantag

| Undantag | Villkor |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | Kastas om presentationen saknar avsnitt, eller om målsliden inte tillhör något avsnitt. |

### Anmärkningar

Denna metod skapar en Summary Zoom-ram som samlar sammanfattningslänkar för alla avsnitt i presentationen.

### Exempel

Detta exempel demonstrerar hur man skapar och infogar ett Summary Zoom-objekt på det angivna indexet i en samling (förutsätt att presentationen *Presentation.pptx* innehåller minst två avsnitt):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSummaryZoomFrame(2, 150, 20, 50, 50);
}
```

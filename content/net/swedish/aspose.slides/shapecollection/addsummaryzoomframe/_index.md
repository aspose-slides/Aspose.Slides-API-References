## ShapeCollection.AddSummaryZoomFrame metod

Skapar en ny Summary Zoom-ram och lägger till den i slutet av shape-samlingen.

```csharp
public ISummaryZoomFrame AddSummaryZoomFrame(float x, float y, float width, float height)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| x | Single | x-koordinaten för den nya Summary Zoom-ramen, i punkter. |
| y | Single | y-koordinaten för den nya Summary Zoom-ramen, i punkter. |
| width | Single | Bredden på den nya Summary Zoom-ramen, i punkter. |
| height | Single | Höjden på den nya Summary Zoom-ramen, i punkter. |

### Returvärde

Den nyss skapade [`ISummaryZoomFrame`](../../isummaryzoomframe).

### Undantag

| undantag | villkor |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | Kastas om det inte finns några sektioner i presentationen, eller om mål-sliden inte tillhör någon sektion. |

### Anmärkningar

Denna metod skapar en ny Summary Zoom och placerar en samling av objekt i den för alla sektioner i denna presentation.

### Exempel

Detta exempel visar hur man lägger till ett Summary Zoom-objekt i slutet av en samling (anta att det finns minst två sektioner i presentationen "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSummaryZoomFrame(150, 20, 500, 250);
}
```

```csharp
public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | Int32 | Det nollbaserade index där autoskissen ska infogas. |
| shapeType | ShapeType | [`ShapeType`](../../shapetype) för den autoskiss som ska infogas. |
| x | Single | X-koordinaten för skissens ram, i punkter. |
| y | Single | Y-koordinaten för skissens ram, i punkter. |
| width | Single | Bredden på skissens ram, i punkter. |
| height | Single | Höjden på skissens ram, i punkter. |
| createFromTemplate | Boolean | true för att använda standardmallens stil (inklusive ett icke-tomt namn, enkel stil och centrerad text); false för att skapa skissen med alla egenskaper satta till sina standardvärden. |

## ShapeCollection.AddSmartArt metod

Skapar ett SmartArt-diagram och lägger till det i slutet av shape-samlingen.

```csharp
public ISmartArt AddSmartArt(float x, float y, float width, float height, 
    SmartArtLayoutType layoutType)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| x | Single | x-koordinaten för diagrammets ram, i punkter. |
| y | Single | y-koordinaten för diagrammets ram, i punkter. |
| width | Single | Diagrammets rambredd, i punkter. |
| height | Single | Diagrammets ramhöjd, i punkter. |
| layoutType | SmartArtLayoutType | SmartArt-layouttypen. |

### Returvärde

Den nyss skapade [`ISmartArt`](../../../aspose.slides.smartart/ismartart).

### Exempel

Följande exempel visar hur man lägger till en smart form i en PowerPoint-presentation.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	Slide slide = pres.Slides[0];
	SmartArt smart = slide.Shapes.AddSmartArt(0, 0, 400, 400, SmartArtLayoutType.BasicBlockList);
}
```

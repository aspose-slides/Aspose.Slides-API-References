## InsertZoomFrame(int, float, float, float, float, ISlide) {#insertzoomframe}

Skapar en ny Zoom-ram och infogar den i shape-samlingen på det angivna indexet.

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| index | Int32 | Det nollbaserade indexet där Zoom-ramen ska infogas. |
| x | Single | x-koordinaten för den nya Zoom-ramen, i punkter. |
| y | Single | y-koordinaten för den nya Zoom-ramen, i punkter. |
| width | Single | Bredden på den nya Zoom-ramen, i punkter. |
| height | Single | Höjden på den nya Zoom-ramen, i punkter. |
| slide | ISlide | Den [`ISlide`](../../islide) som refereras av Zoom-ramen. |

### Returvärde

Den nyss skapade [`IZoomFrame`](../../izoomframe).

### Undantag

| Undantag | Villkor |
| --- | --- |
| ArgumentException | Kastas om den refererade bilden inte tillhör den aktuella presentationen. |

### Exempel

Detta exempel demonstrerar skapande och infogning av ett Zoom-objekt på det angivna indexet i en samling (anta att det finns minst två bilder i presentationen "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertZoomFrame(2, 150, 20, 50, 50, pres.Slides[1]);
}
```

## InsertZoomFrame(int, float, float, float, float, ISlide, IPPImage) {#insertzoomframe_1}

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide, IPPImage image)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| index | Int32 | Det nollbaserade indexet där Zoom-ramen ska infogas. |
| x | Single | x-koordinaten för den nya Zoom-ramen, i punkter. |
| y | Single | y-koordinaten för den nya Zoom-ramen, i punkter. |
| width | Single | Bredden på den nya Zoom-ramen, i punkter. |
| height | Single | Höjden på den nya Zoom-ramen, i punkter. |
| slide | ISlide | Den [`ISlide`](../../islide) som refereras av Zoom-ramen. |
| image | IPPImage | Bilden för den refererade sliden [`IPPImage`](../../ippimage). |

### Returvärde

Den nyss skapade [`IZoomFrame`](../../izoomframe).

### Undantag

| Undantag | Villkor |
| --- | --- |
| ArgumentException | Kastas om den refererade bilden inte tillhör den aktuella presentationen. |

### Exempel

Detta exempel demonstrerar skapande och infogning av ett Zoom-objekt på det angivna indexet i en samling (anta att det finns minst två bilder i presentationen "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IPPImage image = pres.Images.AddImage(Image.FromFile("image.png"));
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertZoomFrame(2, 150, 20, 50, 50, pres.Slides[1], image);
}
```

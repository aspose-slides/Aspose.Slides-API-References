## AddAudioFrameEmbedded(float, float, float, float, Stream) {#addaudioframeembedded_1}

Skapar en ny ljudram med en inbäddad WAV-fil och lägger till den i slutet av formernas samling. Den inbäddade ljudfilen läggs till i samlingen Presentation.Audios.

```csharp
public IAudioFrame AddAudioFrameEmbedded(float x, float y, float width, float height, 
    Stream audio_stream)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| x | Single | X-koordinaten för den nya ljudramen, i punkter. |
| y | Single | Y-koordinaten för den nya ljudramen, i punkter. |
| width | Single | Bredden på den nya ljudramen, i punkter. |
| height | Single | Höjden på den nya ljudramen, i punkter. |
| audio_stream | Stream | En inmatningsström som innehåller WAV-ljuddata för inbäddning. |

### Returvärde

Den nyss skapade [`IAudioFrame`](../../iaudioframe).

### Exempel

Följande exempel visar hur man skapar en ljudram.

```csharp
[C#]
// Instansierar en presentationsklass som representerar en presentationsfil
using (Presentation pres = new Presentation())
{
    // Hämtar den första bilden
    ISlide sld = pres.Slides[0];
    // Läser in wav-ljudfilen till en ström
    FileStream fstr = new FileStream("sampleaudio.wav", FileMode.Open, FileAccess.Read);
    // Lägger till ljudramen
    IAudioFrame audioFrame = sld.Shapes.AddAudioFrameEmbedded(50, 150, 100, 100, fstr);
    // Ställer in uppspelningsläge och volym för ljudet
    audioFrame.PlayMode = AudioPlayModePreset.Auto;
    audioFrame.Volume = AudioVolumeMode.Loud;
    // Skriver PowerPoint-filen till disk
    pres.Save("AudioFrameEmbed_out.pptx", SaveFormat.Pptx);
}
```

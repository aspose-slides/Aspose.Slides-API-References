## InsertOleObjectFrame(int, float, float, float, float, IOleEmbeddedDataInfo) {#insertoleobjectframe}

Skapar en ny OLE-objektram och infogar den i formsamlingen på det angivna indexet.

```csharp
public IOleObjectFrame InsertOleObjectFrame(int index, float x, float y, float width, float height, 
    IOleEmbeddedDataInfo dataInfo)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | Int32 | Det nollbaserade indexet där OLE-objektramen ska infogas. |
| x | Single | X-koordinaten för den nya OLE-ramen, i punkter. |
| y | Single | Y-koordinaten för den nya OLE-ramen, i punkter. |
| width | Single | Bredden på den nya OLE-ramen, i punkter. |
| height | Single | Höjden på den nya OLE-ramen, i punkter. |
| dataInfo | IOleEmbeddedDataInfo | Den inbäddade OLE-datinformationsinformationen ([`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo)). |

### Returvärde

Den nyss skapade [`IOleObjectFrame`](../../ioleobjectframe).

Detta exempel visar hur man infogar ett OLE-objekt på det andra indexet:

```csharp
[C#]
```

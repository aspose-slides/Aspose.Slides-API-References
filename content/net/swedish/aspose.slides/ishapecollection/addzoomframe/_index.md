## AddZoomFrame(float, float, float, float, ISlide) {#addzoomframe}

Skapar en ny Zoom-ram och lägger till den i slutet av formsamlingen.

```csharp
public IZoomFrame AddZoomFrame(float x, float y, float width, float height, ISlide slide)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| x | Single | x-koordinaten för den nya Zoom-ramen, i punkter. |
| y | Single | y-koordinaten för den nya Zoom-ramen, i punkter. |
| width | Single | bredden på den nya Zoom-ramen, i punkter. |
| height | Single | höjden på den nya Zoom-ramen, i punkter. |
| slide | ISlide | [`ISlide`](../../islide) som refereras av Zoom-ramen; måste tillhöra den här presentationen. |

### Returvärde

Den nyss skapade [`IZoomFrame`](../../izoomframe).

### Undantag

| undantag | villkor |
| --- | --- |
| ArgumentException | Kastas om den refererade bilden inte tillhör den aktuella presentationen. |

### Exempel

Detta exempel demonstrerar hur man lägger till ett Zoom-objekt i slutet av en samling (förutsätt att det finns minst två bilder i presentationen "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
}
```

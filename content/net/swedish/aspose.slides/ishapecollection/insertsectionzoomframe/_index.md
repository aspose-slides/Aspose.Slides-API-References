## InsertSectionZoomFrame(int, float, float, float, float, ISection) {#insertsectionzoomframe}

Skapar en ny Section Zoom-ram och infogar den i formsamlingen på det angivna indexet.

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| index | Int32 | Det nollbaserade indexet där Section Zoom-ramen ska infogas. |
| x | Single | X-koordinaten för den nya Section Zoom-ramen, i punkter. |
| y | Single | Y-koordinaten för den nya Section Zoom-ramen, i punkter. |
| width | Single | Bredden på den nya Section Zoom-ramen, i punkter. |
| height | Single | Höjden på den nya Section Zoom-ramen, i punkter. |
| section | ISection | Det [`ISection`](../../isection) som refereras av Section Zoom-ramen; måste tillhöra denna presentation och innehålla minst en bild. |

### Returvärde

Den nyss skapade [`ISectionZoomFrame`](../../isectionzoomframe).

### Undantag

| undantag | villkor |
| --- | --- |
| ArgumentException | Kastas om den refererade sektionen inte tillhör den aktuella presentationen eller inte innehåller några bilder. |

### Exempel

Detta exempel demonstrerar skapandet och infogandet av ett Section Zoom-objekt på det angivna indexet i en samling (antag att det finns minst två sektioner i presentationen “Presentation.pptx”):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSectionZoomFrame(2, 150, 20, 50, 50, pres.Sections[1]);
}
```

## InsertSectionZoomFrame(int, float, float, float, float, ISection, IPPImage) {#insertsectionzoomframe_1}

Skapar en ny Section Zoom-ram med en fördefinierad bild och infogar den i formsamlingen på det angivna indexet.

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section, IPPImage image)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| index | Int32 | Det nollbaserade indexet där Section Zoom-ramen ska infogas. |
| x | Single | X-koordinaten för den nya Section Zoom-ramen, i punkter. |
| y | Single | Y-koordinaten för den nya Section Zoom-ramen, i punkter. |
| width | Single | Bredden på den nya Section Zoom-ramen, i punkter. |
| height | Single | Höjden på den nya Section Zoom-ramen, i punkter. |
| section | ISection | Den [`ISection`](../../isection) som refereras av Section Zoom-ramen; måste tillhöra denna presentation och innehålla minst en bild. |
| image | IPPImage | Bilden som ska visas i Section Zoom-ramen. |

### Returvärde

Den nyss skapade [`ISectionZoomFrame`](../../isectionzoomframe).

### Undantag

| undantag | villkor |
| --- | --- |
| ArgumentException | Kastas om den refererade sektionen inte tillhör den aktuella presentationen eller inte innehåller några bilder. |

### Exempel

Detta exempel demonstrerar skapandet och infogandet av ett Section Zoom-objekt på det angivna indexet i en samling (antag att det finns minst två sektioner i presentationen “Presentation.pptx”):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IPPImage image = pres.Images.AddImage(Image.FromFile("image.png"));
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSectionZoomFrame(2, 150, 20, 50, 50, pres.Sections[1], image);
}
```

## AddSectionZoomFrame(float, float, float, float, ISection) {#addsectionzoomframe}

Skapar en ny Section Zoom-ram och lägger till den i slutet av formsamlingen.

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| x | Single | X-koordinaten för den nya Section Zoom-ramen, i punkt. |
| y | Single | Y-koordinaten för den nya Section Zoom-ramen, i punkt. |
| width | Single | Bredden på den nya Section Zoom-ramen, i punkt. |
| height | Single | Höjden på den nya Section Zoom-ramen, i punkt. |
| section | ISection | Den [`ISection`](../../isection) som refereras av Section Zoom-ramen; den måste tillhöra den här presentationen och innehålla minst en bild. |

### Returvärde

Den nyss skapade [`ISectionZoomFrame`](../../isectionzoomframe).

### Undantag

| Undantag | villkor |
| --- | --- |
| ArgumentException | Kastas om den refererade sektionen inte tillhör den aktuella presentationen eller inte innehåller några bilder. |

### Exempel

Detta exempel visar hur man lägger till ett Section Zoom-objekt i slutet av en samling (anta att det finns minst två sektioner i presentationen "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1]);
}
```

## AddSectionZoomFrame(float, float, float, float, ISection, IPPImage) {#addsectionzoomframe_1}

Skapar en ny Section Zoom-ram med en fördefinierad bild och lägger till den i slutet av formsamlingen.

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section, IPPImage image)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| x | Single | X-koordinaten för den nya Section Zoom-ramen, i punkt. |
| y | Single | Y-koordinaten för den nya Section Zoom-ramen, i punkt. |
| width | Single | Bredden på den nya Section Zoom-ramen, i punkt. |
| height | Single | Höjden på den nya Section Zoom-ramen, i punkt. |
| section | ISection | Den [`ISection`](../../isection) som refereras av Section Zoom-ramen; den måste tillhöra den här presentationen och innehålla minst en bild. |
| image | IPPImage | Den [`IPPImage`](../../ippimage) som ska visas i Section Zoom-ramen. |

### Returvärde

Den nyss skapade [`ISectionZoomFrame`](../../isectionzoomframe).

### Undantag

| Undantag | villkor |
| --- | --- |
| ArgumentException | Kastas om den refererade sektionen inte tillhör den aktuella presentationen eller inte innehåller några bilder. |

### Exempel

Detta exempel visar hur man lägger till ett Section Zoom-objekt i slutet av en samling (anta att det finns minst två sektioner i presentationen "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IPPImage image = pres.Images.AddImage(Image.FromFile("image.png"));
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1], image);
}
```

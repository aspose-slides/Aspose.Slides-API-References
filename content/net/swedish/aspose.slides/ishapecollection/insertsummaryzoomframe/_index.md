## IShapeCollection.InsertSummaryZoomFrame metod

Skapar en ny Summary Zoom-ram och infogar den i shape collection på det angivna indexet.

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| index | Int32 | Det nollbaserade indexet där Summary Zoom-ramen ska infogas. |
| x | Single | x-koordinaten för den nya Summary Zoom-ramen, i punkter. |
| y | Single | y-koordinaten för den nya Summary Zoom-ramen, i punkter. |
| width | Single | Bredden på den nya Summary Zoom-ramen, i punkter. |
| height | Single | Höjden på den nya Summary Zoom-ramen, i punkter. |

### Returvärde

Den nyss skapade [`ISummaryZoomFrame`](../../isummaryzoomframe).

### Undantag

| undantag | villkor |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | Kastas om presentationen inte innehåller några sektioner, eller om målsidan inte tillhör någon sektion. |

### Anmärkningar

Denna metod skapar en Summary Zoom-ram som samlar samman länkar för alla sektioner i presentationen.

### Exempel

Detta exempel visar hur man skapar och infogar ett Summary Zoom-objekt på det angivna indexet i en samling (anta att det finns minst två sektioner i presentationen "Presentation.pptx"):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSummaryZoomFrame(2, 150, 20, 50, 50);
}
```

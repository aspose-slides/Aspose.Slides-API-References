## ParagraphCollection.CopyTo method

Kopierar elementen i ICollection till en Array, med början vid ett visst Array-index.

```csharp
public void CopyTo(IParagraph[] array, int arrayIndex)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| array | IParagraph[] | Den endimensionella Array som är destinationen för elementen kopierade från ICollection. Array måste ha nollbaserad indexering. |
| arrayIndex | Int32 | Det nollbaserade indexet i *array* där kopieringen börjar. |

### Undantag

| undantag | villkor |
| --- | --- |
| ArgumentNullException | *array* är null. |
| ArgumentOutOfRangeException | *arrayIndex* är mindre än 0. |

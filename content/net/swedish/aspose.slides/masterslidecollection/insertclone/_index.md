## MasterSlideCollection.InsertClone metod

Infogar en kopia av en specificerad master-slide till den angivna positionen i samlingen. Länkade layout-slides kopieras också.

```csharp
public IMasterSlide InsertClone(int index, IMasterSlide sourceMaster)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| index | Int32 | Index för den nya bilden. |
| sourceMaster | IMasterSlide | Bild att klona. |

### Returvärde

Infogad master-slide.

### Exempel

Följande exempel visar hur man klonar en master-slide i en annan PowerPoint-presentation.

```csharp
[C#]
// Instansiera Presentation-klassen för att läsa in källpresentationfilen
using (Presentation srcPres = new Presentation("CloneToAnotherPresentationWithMaster.pptx"))
{
    // Instansiera Presentation-klassen för målpresentationen (där bilden ska klonas)
    using (Presentation destPres = new Presentation())
    {
        // Instansiera ISlide från samlingen av bilder i källpresentationen tillsammans med
        // Master-slide
        ISlide SourceSlide = srcPres.Slides[0];
        IMasterSlide SourceMaster = SourceSlide.LayoutSlide.MasterSlide;
		// Hämta master-slides för målpresentationen
        IMasterSlideCollection masters = destPres.Masters;
        // Klona den önskade master-sliden från källpresentationen till samlingen av masters i
        // Målpresentationen
        IMasterSlide iSlide = masters.AddClone(SourceMaster);
        // Samling av bilder i målpresentationen
        ISlideCollection slds = destPres.Slides;
		// Klona källbilden till målbildssamlingen.
        slds.AddClone(SourceSlide, iSlide, true);
        // Spara målpresentationen till disk
        destPres.Save("CloneToAnotherPresentationWithMaster_out.pptx", SaveFormat.Pptx);
    }
}
```

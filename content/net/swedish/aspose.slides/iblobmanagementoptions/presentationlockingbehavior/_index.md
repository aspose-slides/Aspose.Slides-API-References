## IBlobManagementOptions.PresentationLockingBehavior egenskap

Denna egenskap definierar om en instans av Presentation-klassen kan vara ägare till källan - fil eller ström under instansens livstid. Om instansen är ägare låser den källan. Detta hjälper till att förbättra minnesanvändning och prestanda när man arbetar med BLOBs, men källan (ström eller fil) kan inte ändras under Presentation-instansens livstid. Detta är ett exempel:

```csharp
public PresentationLockingBehavior PresentationLockingBehavior { get; set; }
```

### Exempel

```csharp
LoadOptions loadOptions = new LoadOptions { 
  BlobManagementOptions = {PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked} };
using (Presentation pres = new Presentation("pres.pptx", loadOptions)) {
  // IOException kommer att kastas eftersom pres.pptx är låst för en Presentation livstid
  // File.Delete("pres.pptx");
}
 // efter att Presentation-objektet har släppts, är filen olåst och kan tas bort
File.Delete("pres.pptx");
```

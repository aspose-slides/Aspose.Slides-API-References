## SlideShowSettings.SlideShowType egenskap

Hämtar eller anger bildspels-typen. Representeras av följande `SlideShowType` förfäder: [`BrowsedAtKiosk`](../../browsedatkiosk), [`PresentedBySpeaker`](../../presentedbyspeaker) och [`BrowsedByIndividual`](../../browsedbyindividual)

```csharp
public SlideShowType SlideShowType { get; set; }
```

### Exempel

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // för att sätta "Browsed at a kiosk (full screen)" typ 
    pres.SlideShowSettings.SlideShowType = new BrowsedAtKiosk();
    
    // för att sätta "Browsed by individual (window)" typ
    pres.SlideShowSettings.SlideShowType = new BrowsedByIndividual();
    
    // för att sätta "Presented by a speaker (full screen)" typ
    pres.SlideShowSettings.SlideShowType = new PresentedBySpeaker();
}
```

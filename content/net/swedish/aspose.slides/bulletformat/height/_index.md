## BulletFormat.Height egenskap

Returnerar eller anger bullet-höjden för ett stycke utan arv. Värdet float.NaN bestämmer att bullet ärver höjden från den första delen i stycket. Läs/skriv Single.

```csharp
public float Height { get; set; }
```

### Anmärkningar

Ett negativt höjdvärde betyder att höjden anges i punkter och ett positivt värde betyder att höjden är en procentandel av den omgivande texten.

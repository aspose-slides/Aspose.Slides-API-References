### Exempel

Följande exempel visar hur du lägger till anteckningar till en specifik PowerPoint-presentationbild.

```csharp
[C#]
	// Skapa ett Presentation-objekt som representerar en presentationsfil
	using(Presentation presentation = new Presentation(dataDir + "AccessSlides.pptx")) {
	  // Lägg till anteckningar till första bilden
	  INotesSlideManager mgr = presentation.Slides[0].NotesSlideManager;
	  INotesSlide noteSlide = mgr.AddNotesSlide();
	  noteSlide.NotesTextFrame.Text = "Your Notes";
	  // Spara presentationen till disk
	  presentation.Save(dataDir + "RemoveNotesAtSpecificSlide_out.pptx", SaveFormat.Pptx);
	}
```

Följande exempel visar hur du tar bort anteckningar från en specifik bild i en PowerPoint-presentation.

```csharp
[C#]
	// Skapa ett Presentation-objekt som representerar en presentationsfil
	using(Presentation presentation = new Presentation(dataDir + "AccessSlides.pptx")) {
	  // Tar bort anteckningar från första bilden
	  INotesSlideManager mgr = presentation.Slides[0].NotesSlideManager;
	  mgr.RemoveNotesSlide();

	  presentation.Save(dataDir + "RemoveNotesAtSpecificSlide_out.pptx", SaveFormat.Pptx);
	}
```

## Portion.PortionFormat egenskap

Returnerar ett formateringsobjekt som innehåller de explicit inställda formateringsegenskaperna för textdelen utan någon ärvd formatering tillämpad. Skrivskyddad [`IPortionFormat`](../../iportionformat).

```csharp
public IPortionFormat PortionFormat { get; }
```

### Anmärkningar

Formateringsobjektet innehåller de formateringsparametrar som definieras endast för den aktuella delen, ärvd data tillämpas inte.

För att få de faktiska värdena inklusive ärvda, använd metoden [`GetEffective`](../../portionformat/geteffective).

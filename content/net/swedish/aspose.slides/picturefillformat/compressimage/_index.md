## CompressImage(bool, PicturesCompression) {#compressimage}

Komprimerar bilden genom att minska dess storlek baserat på formens storlek och angiven upplösning. Eventuellt raderas även beskurna områden.

```csharp
public bool CompressImage(bool deleteCroppedAreasOfImage, PicturesCompression resolution)
```

| Parameter | Type | Description |
| --- | --- | --- |
| deleteCroppedAreasOfImage | Boolean | Om true, kommer metoden att ta bort de beskurna områdena i bilden, vilket kan minska dess storlek ytterligare. |
| resolution | PicturesCompression | Målupplösningen för komprimeringen, angiven som ett värde av [`PicturesCompression`](../../../aspose.slides.export/picturescompression) enum. |

### Returvärde

Ett Boolean-värde som visar om bilden komprimerades framgångsrikt. Returnerar `true` om bilden har skalats om eller beskärts, annars `false`.

### Undantag

| exception | condition |
| --- | --- |
| ArgumentException | Kastas när upplösningen inte är ett giltigt värde. |

### Anmärkningar

Denna metod ändrar bildens storlek och upplösning på liknande sätt som PowerPoints funktion "Picture Format -> Compress Pictures".

### Exempel

Följande exempel visar hur man använder `CompressImage`-metoden för att minska storleken på en bild i en presentation genom att ange en målupplösning och ta bort beskurna områden:

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;
    // Komprimera bilden med en målupplösning på 150 DPI (webbupplösning) och ta bort beskurna områden
    bool result = picFrame.PictureFormat.CompressImage(true, PicturesCompression.Dpi150);
}
```

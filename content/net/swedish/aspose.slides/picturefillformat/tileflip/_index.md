## PictureFillFormat.TileFlip egenskap

Flips the texture tile around its horizontal, vertical or both axis. Läs/skriv [`TileFlip`](../../tileflip).

```csharp
public TileFlip TileFlip { get; set; }
```

### Anmärkningar

Standard är NoFlip.

### Exempel

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Hämtar bildfyllningsformatet för formen
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Ställer in bildfyllningsläget till Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Vänder texturplattan runt dess vertikala axel.
    pictureFillFormat.TileFlip = TileFlip.FlipY;
}
```

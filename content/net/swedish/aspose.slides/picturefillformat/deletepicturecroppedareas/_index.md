## PictureFillFormat.DeletePictureCroppedAreas metod

Ta bort beskurna områden i fyllnings Picture.

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### Returvärde

Beskuren bild eller ursprungsbild om beskärning inte är nödvändig.

### Anmärkningar

Denna metod konverterar WMF/EMF-metafiler till raster-PNG-bild medan den beskär.

### Exempel

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Hämtar PictureFrame
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // Tar bort beskurna områden i PictureFrame-bilden
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

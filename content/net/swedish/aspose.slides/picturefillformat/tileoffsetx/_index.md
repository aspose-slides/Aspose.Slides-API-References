## PictureFillFormat.TileOffsetX egenskap

Returnerar eller anger den horisontella förskjutningen av texturen från figurens ursprung i punkter. Ett positivt värde flyttar texturen åt höger, medan ett negativt värde flyttar den åt vänster. Läs/skriv Single.

```csharp
public float TileOffsetX { get; set; }
```

### Exempel

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Hämtar bildfyllningsformatet för formen
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Ställer in bildfyllningsläget till Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Ställer in den horisontella förskjutningen av texturen till 20 punkter
    pictureFillFormat.TileOffsetX = 20f;
}
```

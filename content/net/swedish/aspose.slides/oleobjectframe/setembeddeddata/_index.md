## OleObjectFrame.SetEmbeddedData metod

Ställer in information om OLE inbäddade data. Denna metod ändrar objektets egenskaper för att återspegla den nya datan och sätter IsObjectLink-flaggan till false, vilket indikerar att OLE-objektet är inbäddat.

```csharp
public void SetEmbeddedData(IOleEmbeddedDataInfo embeddedData)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| embeddedData | IOleEmbeddedDataInfo | Inbäddad data [`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo) |

### Undantag

| undantag | villkor |
| --- | --- |
| ArgumentNullException | När embeddedData parameter är null. |

### Exempel

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    OleObjectFrame oof = pres.Slides[0].Shapes[0] as OleObjectFrame;
    if (oof != null)
    {
        IOleEmbeddedDataInfo newData = new OleEmbeddedDataInfo(File.ReadAllBytes("Picture.png"), "png");
        oof.SetEmbeddedData(newData);
    }
}
```

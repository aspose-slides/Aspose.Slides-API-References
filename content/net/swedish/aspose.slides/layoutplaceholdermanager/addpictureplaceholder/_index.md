## LayoutPlaceholderManager.AddPicturePlaceholder metod

Lägger till en ny platshållarform på layoutbilden för att innehålla en bild.

```csharp
public IAutoShape AddPicturePlaceholder(float x, float y, float width, float height)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| x | Single | X-koordinaten för den nya platshållarformen. |
| y | Single | Y-koordinaten för den nya platshållarformen. |
| width | Single | Bredden på den nya platshållarformen. |
| height | Single | Höjden på den nya platshållarformen. |

### Returvärde

Skapade [`IAutoShape`](../../iautoshape) med en Picture-platshållare.

### Exempel

Följande exempel visar hur man lägger till Picture-platshållarformen på layoutbilden.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddPicturePlaceholder(20, 20, 200, 200);
}
```

## LayoutPlaceholderManager.AddMediaPlaceholder metod

Lägger till en ny platshållarform på layout-bilden för att hålla ett mediaobjekt.

```csharp
public IAutoShape AddMediaPlaceholder(float x, float y, float width, float height)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| x | Single | X-koordinaten för den nya platshållarformen. |
| y | Single | Y-koordinaten för den nya platshållarformen. |
| width | Single | Bredden på den nya platshållarformen. |
| height | Single | Höjden på den nya platshållarformen. |

### Returvärde

Skapade [`IAutoShape`](../../iautoshape) med en media platshållare.

### Exempel

Följande exempel visar hur man lägger till media platshållarformen till layout-bilden.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddMediaPlaceholder(20, 20, 200, 200);
}
```

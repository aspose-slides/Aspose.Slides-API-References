## LayoutPlaceholderManager.AddTextPlaceholder method

Lägger till en ny platshållarform på layoutbilden för att hålla textinnehåll.

```csharp
public IAutoShape AddTextPlaceholder(float x, float y, float width, float height)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| x | Single | X-koordinaten för den nya platshållarformen. |
| y | Single | Y-koordinaten för den nya platshållarformen. |
| width | Single | Bredden på den nya platshållarformen. |
| height | Single | Höjden på den nya platshållarformen. |

### Returvärde

Skapade [`IAutoShape`](../../iautoshape) med en Text placeholder.

### Exempel

Följande exempel visar hur man lägger till Text placeholder-formen på layoutbilden.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTextPlaceholder(20, 20, 500, 300);
}
```

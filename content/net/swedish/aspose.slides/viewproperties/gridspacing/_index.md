## ViewProperties.GridSpacing egenskap

Returnerar eller anger rasteravståndet som ska användas för rastern underliggande presentationsdokumentet, i punkter. Läs/skriv Single.

```csharp
public float GridSpacing { get; set; }
```

### Anmärkningar

Rasteravståndsvärdet måste vara ett positivt tal. Det vanliga värdeintervallet är från 1 mm (2.8349607 punkter) till 2 tum (144 punkter).

### Exempel

Följande exempelprogram visar hur du ändrar rasteravståndet i en PowerPoint-presentation.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

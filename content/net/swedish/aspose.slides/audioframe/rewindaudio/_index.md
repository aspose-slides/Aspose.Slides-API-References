## AudioFrame.RewindAudio egenskap

Bestämmer om ljudet automatiskt spolas tillbaka till början efter uppspelning. Läs/skriv Boolean.

```csharp
public bool RewindAudio { get; set; }
```

### Exempel

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ISlide slide = pres.Slides[0];

    // Lägg till ljudram
    IAudioFrame audioFrame = slide.Shapes.AddAudioFrameLinked(50, 50, 100, 100, "sampleaudio.wav");

    // Ställ in ljudet så att det spelas över alla bilder
    audioFrame.PlayAcrossSlides = true;

    // Ställ in ljudet att automatiskt spolas tillbaka till början efter uppspelning
    audioFrame.RewindAudio = true;

    pres.Save("AudioFrame_out.pptx", SaveFormat.Pptx);
}
```

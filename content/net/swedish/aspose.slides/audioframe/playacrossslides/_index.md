## AudioFrame.PlayAcrossSlides egenskap

Bestämmer om ljud spelas över bildspel. Läs/skriv Boolean.

```csharp
public bool PlayAcrossSlides { get; set; }
```

### Exempel

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ISlide slide = pres.Slides[0];

    // Lägg till ljudram
    IAudioFrame audioFrame = slide.Shapes.AddAudioFrameLinked(50, 50, 100, 100, "sampleaudio.wav");

    // Ställ in ljudet att spelas över bildspelen
    audioFrame.PlayAcrossSlides = true;

    // Ställ in ljudet att automatiskt spola tillbaka till början efter uppspelning
    audioFrame.RewindAudio = true;

    pres.Save("AudioFrame_out.pptx", SaveFormat.Pptx);
}
```

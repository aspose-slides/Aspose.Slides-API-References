## Fonts.SetScriptFont-metod

Tilldelar ett teckensnittsnamn till en specifik skript-tagg, som definierar hur text med det skriptet ska renderas i presentationen.

```csharp
public void SetScriptFont(string script, string fontName)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| script | String | BCP-47-skriptkoden (t.ex. "Arab", "Hebr", "Hans") som identifierar skriftsystemet. |
| fontName | String | Namnet på teckensnittet som ska tilldelas det angivna skriptet. |

### Exempel

Detta exempel visar hur du ställer in teckensnittet för det arabiska skriptet till "Segoe UI":

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segoe UI");
```

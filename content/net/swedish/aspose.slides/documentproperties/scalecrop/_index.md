## DocumentProperties.ScaleCrop egenskap

Anger visningsläget för dokumentets miniatyrbild. Ställ in detta element till **true** för att möjliggöra skalning av dokumentets miniatyrbild till displayen. Ställ in detta element till **false** för att möjliggöra beskärning av dokumentets miniatyrbild så att endast sektioner som passar displayen visas. Läs/skriv Boolean.

public bool ScaleCrop { get; set; }

## PresentationLockingBehavior enumeration

Representerar beteendet när [`IPresentation`](../ipresentation) källan (fil eller Stream) behandlas under inläsning och arbete med en instans av [`IPresentation`](../ipresentation).

```csharp
public enum PresentationLockingBehavior
```

### Värden

| Namn | Värde | Beskrivning |
| --- | --- | --- |
| LoadAndRelease | `0` | Källan kommer endast att låsas under en kort stund under [`IPresentation`](../ipresentation) konstruktörsutförande. Om [`IsTemporaryFilesAllowed`](../iblobmanagementoptions/istemporaryfilesallowed) är satt till false kommer alla BLOB:ar att läsas in i minnet. Annars kan andra metoder såsom tillfälliga filer användas. Detta beteende är långsammare än KeepLocked, och om det är möjligt att överföra ägandet av källan till [`IPresentation`](../ipresentation) rekommenderas att använda KeepLocked. |
| KeepLocked | `1` | Källan kommer att låsas under hela livstiden för en [`IPresentation`](../ipresentation)-instans, tills den avinstalleras. [`IsTemporaryFilesAllowed`](../iblobmanagementoptions/istemporaryfilesallowed) måste vara satt till true för att använda detta beteende, annars kastas ett undantag. Detta beteende rekommenderas, det är snabbare och förbrukar mindre minne än LoadAndRelease. |

### Anmärkningar

Källan är parametern som skickas till [`IPresentation`](../ipresentation)-konstruktören. I exemplet nedan är källan filen "pres.pptx":

```csharp
LoadOptions loadOptions = new LoadOptions { 
  BlobManagementOptions = { PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked } };
using (IPresentation pres = new Presentation("pres.pptx", loadOptions)) { }
```

För detta exempel kommer källan ("pres.pptx"-filen) att låsas under en [`IPresentation`](../ipresentation)-instans livstid, dvs. den kan inte ändras eller raderas av en annan process.

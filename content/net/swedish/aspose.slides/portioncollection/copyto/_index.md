## PortionCollection.CopyTo metod

Kopierar elementen i ICollection till en Array, med början på ett specifikt Array-index.

```csharp
public void CopyTo(IPortion[] array, int arrayIndex)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| array | IPortion[] | Den endimensionella Array som är destinationen för elementen som kopieras från ICollection. Array måste ha nollbaserad indexering. |
| arrayIndex | Int32 | Det nollbaserade indexet i *array* där kopieringen påbörjas. |

### Undantag

| Undantag | villkor |
| --- | --- |
| ArgumentNullException | *array* är null. |
| ArgumentOutOfRangeException | *arrayIndex* är mindre än 0. |
| ArgumentException | Antalet element i källICollection är större än det tillgängliga utrymmet från *arrayIndex* till slutet av destinationen *array*. |

## Presentation.AllCustomXmlParts egenskap

Returnerar alla anpassade datapaket i presentationen. Skrivskyddad [`ICustomXmlPart`](../../icustomxmlpart)[].

```csharp
public ICustomXmlPart[] AllCustomXmlParts { get; }
```

### Exempel

Följande exempel visar hur man rensar alla anpassade xml-delar från PowerPoint-presentationen.

```csharp
[C#]
using (Presentation pres = new Presentation("PresentationWithCustomXml.pptx"))
{
	// Iterera alla anpassade XML-delar
    foreach (ICustomXmlPart item in pres.AllCustomXmlParts)
    {
        item.Remove();
    }
    pres.Save("out.pptx", SaveFormat.Pptx);
}
```

## ISvgImage.WriteAsEmf metod

Sparar SVG-bilden som en EMF-fil.

```csharp
public void WriteAsEmf(Stream stream)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| stream | Stream | Måldataström |

### Undantag

| undantag | villkor |
| --- | --- |
| ArgumentNullException | Måldataströmmen är **null** |

### Exempel

Följande exempel visar hur man sparar SVG-bilden i en metafil.

```csharp
[C#]
// Skapar den nya SVG-bilden
ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));

// Sparar SVG-bilden som en metafil
using (var fileStream = System.IO.File.OpenWrite("SvgAsEmf.emf"))
    svgImage.WriteAsEmf(fileStream);
```

Detta exempel visar hur man lägger till SVG-bilden som en metafil i presentationens bildsamling.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // Skapar den nya SVG-bilden
    ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));
    using (var memStream = new MemoryStream())
    {
        // Sparar SVG-bilden som en metafil
        svgImage.WriteAsEmf(memStream);
        // Lägger till metafil i bildsamlingen
        pres.Images.AddImage(memStream.ToArray());
    }
}
```

## IFonts.SetScriptFont metod

Tilldelar ett typsnittsnamn till en specifik skripttagg, som definierar hur text för det skriptet kommer att renderas i presentationen.

```csharp
public void SetScriptFont(string script, string fontName)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| script | String | BCP-47-skriptkoden (t.ex. "Arab", "Hebr", "Hans") som identifierar skriftsystemet. |
| fontName | String | Namnet på typsnittet som ska tilldelas det angivna skriptet. |

### Exempel

Det här exemplet visar hur man ställer in typsnittet för det arabiska skriptet till "Segoe UI":

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segoe UI");
```

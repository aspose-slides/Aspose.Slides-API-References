## IDocumentProperties.GetSensitivityLabels metod

Hämtar en array med känslighetsetiketter från de anpassade dokumentegenskaperna (Microsoft Information Protection SDK Metadata).

```csharp
public ISensitivityLabel[] GetSensitivityLabels()
```

### Exempel

Följande kod visar hur man flyttar informationen om känslighetsetiketter från de anpassade dokumentegenskaperna till den moderna SensitivityLabels-samlingen:

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    // Hämta känslighetsetiketter från de anpassade dokumentegenskaperna
    ISensitivityLabel[] mipSensitivityLabels = pres.DocumentProperties.GetSensitivityLabels();

    ISensitivityLabelCollection sensitivityLabels = pres.SensitivityLabels;
    foreach (var sensitivityLabel in mipSensitivityLabels)
    {
        // Lägg till etikett i samlingen
        // Här kan du lägga till en kontroll för etikettinformationens giltighet (etiketten är tillgänglig, etc)
        sensitivityLabels.Add(sensitivityLabel);
    }

    pres.Save("SensitivityLabel.pptx", SaveFormat.Pptx);
}
```

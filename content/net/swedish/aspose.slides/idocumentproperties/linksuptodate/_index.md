## IDocumentProperties.LinksUpToDate egenskap

Indikerar om hyperlänkar i ett dokument är aktuella. Ställ in detta element till **true** för att indikera att hyperlänkarna är uppdaterade. Ställ in detta element till **false** för att indikera att hyperlänkarna är föråldrade. Läs/skriv Boolean.

public bool LinksUpToDate { get; set; }

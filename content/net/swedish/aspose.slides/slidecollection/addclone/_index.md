## AddClone(ISlide) {#addclone}

Lägger till en kopia av en angiven slide i slutet av samlingen.

```csharp
public ISlide AddClone(ISlide sourceSlide)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| sourceSlide | ISlide | Slide to clone. |

### Returvärde

Ny slide.

### Anmärkningar

När en slide klonas mellan olika presentationer kan slide’s master kopieras också. Ett internt register används för att spåra automatiskt klonade master-slides för att förhindra skapandet av flera kopior av samma master-slide. Manuell kloning av master-slides kommer varken att förhindras eller registreras. Om du behöver mer kontroll över kloningsprocessen, använd [`AddClone`](../addclone) eller [`AddClone`](../addclone) för att klona slides, [`AddClone`](../../igloballayoutslidecollection/addclone) eller [`AddClone`](../../igloballayoutslidecollection/addclone) för att klona layouter och [`AddClone`](../../imasterslidecollection/addclone) för att klona master-slides.

### Exempel

```csharp
[C#]
using (IPresentation presentation = new Presentation())
{
    presentation.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 200, 50, 300, 100);
    presentation.Sections.AddSection("Section 1", presentation.Slides[0]);
    
    ISection section2 = presentation.Sections.AppendEmptySection("Section 2");
    presentation.Slides.AddClone(presentation.Slides[0], section2);
    
    // Nu innehåller den andra sektionen en kopia av den första sliden.
}
```

## AddClone(ISlide, IMasterSlide, bool) {#addclone_2}

Lägger till en kopia av en angiven källslide i slutet av samlingen. Lämplig layout väljs automatiskt från den angivna master-slide (lämplig layout är den layout som har samma Typ eller Namn som layouten för källsliden). Om det inte finns någon lämplig layout klonas layouten för källsliden (om allowCloneMissingLayout är true) eller så kastas PptxEditException (om allowCloneMissingLayout är false).

```csharp
public ISlide AddClone(ISlide sourceSlide, IMasterSlide destMaster, bool allowCloneMissingLayout)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| sourceSlide | ISlide | Slide to clone. |
| destMaster | IMasterSlide | Master slide for a new slide. |
| allowCloneMissingLayout | Boolean | If there is no appropriate layout in specified master then layout of the source slide will be cloned (if allowCloneMissingLayout is true) or PptxEditException will be thrown (if allowCloneMissingLayout is false). |

### Returvärde

Ny slide.

### Undantag

| undantag | villkor |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | Thrown if there is no appropriate layout in specified master and allowCloneMissingLayout is false. |

## INormalViewProperties.PreferSingleView egenskap

Anger om användaren föredrar att se ett fullfönster med ett enda innehållsområde istället för den vanliga normala vyn med tre innehållsområden. Om den är aktiverad kan programmet välja att visa ett av innehållsområdena i hela fönstret. Läs/skriv Boolean.

public bool PreferSingleView { get; set; }

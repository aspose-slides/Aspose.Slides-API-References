## Shape.Rotation egenskap

Returnerar eller anger antalet grader som den angivna formen roteras runt z-axeln. Ett positivt värde indikerar medurs rotation; ett negativt värde indikerar moturs rotation. Läs/skriv Single.

```csharp
public float Rotation { get; set; }
```

### Anmärkningar

Det returnerade värdet är alltid definierat (är inte float.NaN). Tilldelat värde måste vara definierat (inte float.NaN). Du kan sätta odefinierade värden för RawFrame-instansens egenskaper.

## ZoomObject.ShowBackground egenskap

Hämtar eller anger värde som specificerar om Zoom ska använda bakgrunden på destinationssliden. Läs/skriv Boolean. Standardvärde: true

```csharp
public bool ShowBackground { get; set; }
```

### Exempel

Exemplet demonstrerar hur man tar bort bakgrunden på en bild av ett Zoom-objekt:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
    zoomFrame.ShowBackground = false;
}
```

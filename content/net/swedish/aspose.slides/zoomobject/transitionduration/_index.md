## ZoomObject.TransitionDuration egenskap

Hämtar eller anger varaktigheten för övergången mellan Zoom och bild. Läs/skriv Single. Standardvärde: 1.0f

```csharp
public float TransitionDuration { get; set; }
```

### Anmärkningar

Om den inte anges (TransitionDur = 0) används bildens destinationsövergång och tidsinställningarna som är kopplade till den övergången.

### Exempel

exemplet visar hur man ändrar varaktigheten för övergången mellan Zoom och bild:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
    zoomFrame.TransitionDuration = 2.5f;
}
```

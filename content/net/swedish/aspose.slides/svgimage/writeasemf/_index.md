## SvgImage.WriteAsEmf metod

Sparar SVG-bilden som en EMF-fil.


| Parameter | Typ | Beskrivning |

### Undantag

| undantag | villkor |

### Exempel

Följande exempel visar hur man sparar SVG-bilden till metafilen.

```csharp
[C#]
// Skapar den nya SVG-bilden
ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));

// Sparar SVG-bilden som en metafil
using (var fileStream = System.IO.File.OpenWrite("SvgAsEmf.emf"))
    svgImage.WriteAsEmf(fileStream);
```

Detta exempel visar hur man lägger till SVG-bilden som metafil i presentationsbildsamlingen.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // Skapar den nya SVG-bilden
    ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));
    using (var memStream = new MemoryStream())
    {
        // Sparar SVG-bilden som en metafil
        svgImage.WriteAsEmf(memStream);
        // Lägger till metafilen i bildsamlingen
        pres.Images.AddImage(memStream.ToArray());
    }
}
```

## AddClone(ISlide) {#addclone}

Lägger till en kopia av en specificerad bild i slutet av samlingen.

```csharp
public ISlide AddClone(ISlide sourceSlide)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| sourceSlide | ISlide | Bild att klona. |

### Return Value

Ny bild.

### Remarks

Vid kloning av en bild mellan olika presentationer kan bildens master också klonas. Ett internt register används för att spåra automatiskt klonade master för att förhindra skapandet av flera kloner av samma master-bild. Manuell kloning av master-bilder kommer varken att hindras eller registreras. Om du behöver mer kontroll över kloningsprocessen, använd [`AddClone`](../addclone) eller [`AddClone`](../addclone) för att klona bilder, [`AddClone`](../../igloballayoutslidecollection/addclone) eller [`AddClone`](../../igloballayoutslidecollection/addclone) för att klona layouter och [`AddClone`](../../imasterslidecollection/addclone) för att klona master-bilder.

## AddClone(ISlide, IMasterSlide, bool) {#addclone_2}

Lägger till en kopia av en specificerad källbild i slutet av samlingen. Ett lämpligt layout väljs automatiskt från den angivna master-bilden (ett lämpligt layout är det layout som har samma Typ eller Namn som layouten för källbilden). Om det inte finns ett lämpligt layout klonas layouten för källbilden (om allowCloneMissingLayout är true) eller så kastas PptxEditException (om allowCloneMissingLayout är false).

```csharp
public ISlide AddClone(ISlide sourceSlide, IMasterSlide destMaster, bool allowCloneMissingLayout)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| sourceSlide | ISlide | Bild att klona. |
| destMaster | IMasterSlide | Master-bild för en ny bild. |
| allowCloneMissingLayout | Boolean | Om det inte finns ett lämpligt layout i den angivna master-bilden klonas layouten för källbilden (om allowCloneMissingLayout är true) eller så kastas PptxEditException (om allowCloneMissingLayout är false). |

### Return Value

Ny bild.

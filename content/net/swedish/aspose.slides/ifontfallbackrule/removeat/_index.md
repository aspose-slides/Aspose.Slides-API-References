## IFontFallBackRule.RemoveAt metod

Tar bort FallBack-teckensnittet på det angivna indexet i listan.

```csharp
public void RemoveAt(int index)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| index | Int32 | Det nollbaserade indexet för teckensnittet som ska tas bort. |

### Exempel

```csharp
[C#]
// Skapa en regel som innehåller en lista med typsnitt.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// Tar bort Tahoma från listan
```

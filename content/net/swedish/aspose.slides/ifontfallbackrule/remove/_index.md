## IFontFallBackRule.Remove metod

Tar bort den första förekomsten av ett specifikt FallBack-font från listan.

```csharp
public void Remove(string fontName)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| fontName | String | Typsnittets namn att ta bort från listan. |

### Exempel

```csharp
[C#]
// Skapa en regel som innehåller en lista med typsnitt.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// Tar bort Tahoma från listan
newRule.Remove ("Tahoma");
```

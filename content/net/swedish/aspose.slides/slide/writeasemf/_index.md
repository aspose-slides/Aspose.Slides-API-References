### Exempel

Följande kodexempel visar hur man konverterar den första bilden i en PowerPoint-presentation till en metafil.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.emf"))
    {
        // Sparar den första bilden som en metafil
        pres.Slides[0].WriteAsEmf(fileStream);
    }
}
```

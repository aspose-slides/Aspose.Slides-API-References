## IPresentationInfo.UpdateDocumentProperties metod

Uppdaterar egenskaper för den bindade presentationen.

```csharp
public void UpdateDocumentProperties(IDocumentProperties documentProperties)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| documentProperties | IDocumentProperties | Dokumentegenskaper [`IDocumentProperties`](../../idocumentproperties) |

### Exempel

Detta exempel visar hur man anropar `UpdateDocumentProperties` metod för att uppdatera dokumentegenskaperna som returneras av anropet av [`ReadDocumentProperties`](../readdocumentproperties) metod.

```csharp
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo("pres.pptx");
IDocumentProperties props = info.ReadDocumentProperties();
props.Subject = "New subject";
props.LastSavedTime = DateTime.UtcNow;
info.UpdateDocumentProperties(props);
info.WriteBindedPresentation("new_pres.pptx");
```

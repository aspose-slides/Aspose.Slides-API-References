## FontsLoader.LoadExternalFonts metod

Lägger till extra mappar för att söka typsnitt.

```csharp
public static void LoadExternalFonts(string[] directories)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| directories | String[] | Kataloger för att läsa ytterligare typsnitt. |

### Exempel

Följande exempel visar hur man laddar anpassade typsnitt från .TTF

```csharp
[C#]
// Sökvägen till dokumentkatalogen.
string dataDir = "C:\\";
// mappar att söka efter typsnitt
String[] folders = new String[] { dataDir };
// Ladda anpassade typsnitt från katalogen
FontsLoader.LoadExternalFonts(folders);
// Utför lite arbete och rendera presentation/slide
using (Presentation presentation = new Presentation(dataDir + "DefaultFonts.pptx"))
{
    presentation.Save(dataDir + "NewFonts_out.pptx", SaveFormat.Pptx);
	// Rensa font-cache
	FontsLoader.ClearCache();
}
```

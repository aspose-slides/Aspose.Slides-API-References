| [IsTemporaryFilesAllowed](../../aspose.slides/blobmanagementoptions/istemporaryfilesallowed) { get; set; } | Denna egenskap anger om tillfälliga filer kan skapas medan du arbetar med BLOB:ar, vilket kraftigt minskar minnesförbrukningen men kräver behörighet att skapa filer. | [MaxBlobsBytesInMemory](../../aspose.slides/blobmanagementoptions/maxblobsbytesinmemory) { get; set; } | Definierar den maximala totala storleken (i byte) som alla BLOB:ar får uppta i minnet. Som standard laddas alla BLOB:ar in i minnet; först när denna gräns nås används alternativa mekanismer (t.ex. tillfälliga filer). Att hålla BLOB:ar i minnet maximerar prestanda men kan leda till hög minnesanvändning. Använd denna egenskap för att anpassa beteendet till din miljö eller dina krav. |
| [PresentationLockingBehavior](../../aspose.slides/blobmanagementoptions/presentationlockingbehavior) { get; set; } | Denna egenskap anger om en instans av Presentation klass kan vara ägare till källan – fil eller ström – under instansens livstid. Om instansen är ägare låser den källan. Detta hjälper till att förbättra minnesförbrukning och prestanda när du arbetar med BLOB:ar, men källan (ström eller fil) kan inte ändras under Presentation-instansens livstid. |

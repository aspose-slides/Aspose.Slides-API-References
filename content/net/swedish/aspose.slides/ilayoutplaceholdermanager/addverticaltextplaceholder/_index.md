## ILayoutPlaceholderManager.AddVerticalTextPlaceholder method

Lägger till en ny platshållarform på layoutbilden för att hålla textinnehåll i vertikal riktning.

```csharp
public IAutoShape AddVerticalTextPlaceholder(float x, float y, float width, float height)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| x | Single | X-koordinaten för den nya platshållarformen. |
| y | Single | Y-koordinaten för den nya platshållarformen. |
| width | Single | Bredden på den nya platshållarformen. |
| height | Single | Höjden på den nya platshållarformen. |

### Returvärde

Skapade [`IAutoShape`](../../iautoshape) med en Text (Vertikal) platshållare.

### Exempel

Följande exempel visar hur man lägger till Text (Vertikal) platshållarformen på layoutbilden.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalTextPlaceholder(20, 20, 300, 500);
}
```

## ILayoutPlaceholderManager.AddVerticalContentPlaceholder method

Lägger till en ny platshållarform till layout slide för att hålla innehåll, såsom en bild, tabell, media eller text i vertikal riktning.

```csharp
public IAutoShape AddVerticalContentPlaceholder(float x, float y, float width, float height)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| x | Single | X-koordinaten för den nya platshållarformen. |
| y | Single | Y-koordinaten för den nya platshållarformen. |
| width | Single | Bredden på den nya platshållarformen. |
| height | Single | Höjden på den nya platshållarformen. |

### Returvärde

Skapad [`IAutoShape`](../../iautoshape) med en Content (Vertical) placeholder.

### Exempel

Följande exempel visar hur man lägger till Content (Vertical) placeholder-formen till layout slide.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalContentPlaceholder(20, 20, 300, 500);
}
```

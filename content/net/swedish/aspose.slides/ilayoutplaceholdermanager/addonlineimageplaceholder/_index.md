## ILayoutPlaceholderManager.AddOnlineImagePlaceholder metod

Lägger till en ny platshållarform på layoutbilden för att hålla en online-bild.

```csharp
public IAutoShape AddOnlineImagePlaceholder(float x, float y, float width, float height)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| x | Single | Den X-koordinaten för den nya platshållarformen. |
| y | Single | Den Y-koordinaten för den nya platshållarformen. |
| width | Single | Bredden på den nya platshållarformen. |
| height | Single | Höjden på den nya platshållarformen. |

### Returvärde

Skapade [`IAutoShape`](../../iautoshape) med en online-bild-platshållare.

### Exempel

Följande exempel visar hur man lägger till en online-bild-platshållarform på layoutbilden.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddOnlineImagePlaceholder(20, 20, 200, 200);
}
```

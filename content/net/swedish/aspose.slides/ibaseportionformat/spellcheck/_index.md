## IBasePortionFormat.SpellCheck egenskap

Hämtar eller anger ett värde som indikerar om stavningskontroll är aktiverad för textdelen. När denna egenskap är satt till false, undertrycks stavningskontroller för textelement. När den är satt till true, tillåts stavningskontroll. Standardvärdet är `false`.

```csharp
public bool SpellCheck { get; set; }
```

### Exempel

Nästa exempel visar hur SpellCheck-flaggan aktiveras innan presentationen sparas:

```csharp
[C#]
using (var pres = new Presentation("input.pptx"))
{
    // Åtkomst till den första textdelen i den första formen på den första bilden
    var portion = ((AutoShape)pres.Slides[0].Shapes[0]).TextFrame.Paragraphs[0].Portions[0];
    // Aktivera stavningskontroll för denna textdel
    portion.PortionFormat.SpellCheck = true;
    // Spara den ändrade presentationen
    pres.Save("output-with-spellcheck.pptx", SaveFormat.Pptx);
}
```

## IOleObjectFrame.SetEmbeddedData method

Ställer in information om OLE-inbäddade data.

```csharp
public void SetEmbeddedData(IOleEmbeddedDataInfo embeddedData)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| embeddedData | IOleEmbeddedDataInfo | Inbäddade data [`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo) |

### Undantag

| undantag | villkor |
| --- | --- |
| ArgumentNullException | När parametern embeddedData är null. |

### Anmärkningar

Denna metod ändrar objektets egenskaper för att återspegla den nya datan och sätter flaggan IsObjectLink till false, vilket indikerar att OLE-objektet är inbäddat.

### Exempel

Följande exempel visar hur man ändrar OLE-inbäddade data och dess typ för ett befintligt [`IOleObjectFrame`](../../ioleobjectframe)-objekt

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    OleObjectFrame oof = pres.Slides[0].Shapes[0] as OleObjectFrame;
    if (oof != null)
    {
        IOleEmbeddedDataInfo newData = new OleEmbeddedDataInfo(File.ReadAllBytes("Picture.png"), "png");
        oof.SetEmbeddedData(newData);
    }
}
```

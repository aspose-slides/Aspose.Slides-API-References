## IPictureFillFormat.TileAlignment egenskap

Returnerar eller anger hur texturen är justerad inom formen. Denna inställning styr startpunkten för texturmönstret och hur det upprepas över formen. Läs/skriv [`RectangleAlignment`](../../rectanglealignment).

```csharp
public RectangleAlignment TileAlignment { get; set; }
```

### Anmärkningar

Standard är TopLeft.

### Exempel

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Hämtar bildfyllningsformatet för formen
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Ställer in bildfyllningsläget till Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Ställer in justeringen för mönstret till nedre högra
    pictureFillFormat.TileAlignment = RectangleAlignment.BottomRight;
}
```

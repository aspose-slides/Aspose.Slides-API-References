## IPictureFillFormat.TileScaleY egenskap

Returnerar eller anger den vertikala skalan för teksturfyllning som en procentandel. Läs/skriv Single.

```csharp
public float TileScaleY { get; set; }
```

### Exempel

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Hämtar bildfyllnadsformatet för formen
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Ställer in bildfyllnadsläget till Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Ställer in den vertikala skalan för texturen till 120 procent
    pictureFillFormat.TileScaleY = 120;
}
```

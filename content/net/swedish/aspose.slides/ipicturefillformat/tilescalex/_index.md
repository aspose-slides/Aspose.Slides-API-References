## IPictureFillFormat.TileScaleX egenskap

Returnerar eller anger den horisontella skalan för texturfyllning som en procentandel. Läs/skriv Single.

```csharp
public float TileScaleX { get; set; }
```

### Exempel

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Hämtar bildfyllningsformatet för formen
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Ställer in bildfyllningsläget till Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Ställer in den horisontella skalan för texturen till 120 procent
    pictureFillFormat.TileScaleX = 120;
}
```

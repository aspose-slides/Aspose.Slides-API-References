## IPictureFillFormat.TileOffsetX egenskap

Returnerar eller anger det horisontella avståndet för texturen från figurens ursprung i punkter. Ett positivt värde flyttar texturen åt höger, medan ett negativt värde flyttar den åt vänster. Läs/skriv Single.

```csharp
public float TileOffsetX { get; set; }
```

### Exempel

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Hämtar bildfyllnadsformatet för formen
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Ställer in bildfyllnadsläget till Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Ställer in det horisontella avståndet för texturen till 20 punkter
    pictureFillFormat.TileOffsetX = 20f;
}
```

## LoadOptions.DocumentLevelFontSources egenskap

Anger källor för externa teckensnitt som ska användas av presentationen. Dessa teckensnitt är tillgängliga för presentationen under hela dess livstid och delas inte med andra presentationer

```csharp
public IFontSources DocumentLevelFontSources { get; set; }
```

### Exempel

Följande exempel visar hur man specificerar anpassade teckensnitt som används med PowerPoint-presentation.

```csharp
[C#]
byte[] memoryFont1 = File.ReadAllBytes("customfonts\\CustomFont1.ttf");
byte[] memoryFont2 = File.ReadAllBytes("customfonts\\CustomFont2.ttf");
LoadOptions loadOptions = new LoadOptions();
loadOptions.DocumentLevelFontSources.FontFolders = new string[] { "assets\\fonts", "global\\fonts" };
loadOptions.DocumentLevelFontSources.MemoryFonts = new byte[][] { memoryFont1, memoryFont2 };
using (IPresentation presentation = new Presentation("MyPresentation.pptx", loadOptions))
{
    //arbeta med presentationen
    //CustomFont1, CustomFont2 samt teckensnitt från mapparna assets\fonts & global\fonts och deras undermappar är tillgängliga för presentationen
}
```

## LoadOptions.DefaultRegularFont egenskap

Returnerar eller anger Regular-teckensnitt som används om källteckensnittet inte hittas. Läs/skriv String.

```csharp
public string DefaultRegularFont { get; set; }
```

### Exempel

Följande exempel visar hur man ställer in standardteckensnitt för rendering av PowerPoint-presentation.

```csharp
[C#]
// Använd laddningsalternativ för att definiera standardregulära och asiatiska teckensnitt
LoadOptions loadOptions = new LoadOptions(LoadFormat.Auto);
loadOptions.DefaultRegularFont = "Wingdings";
loadOptions.DefaultAsianFont = "Wingdings";
// Läs in presentationen
using (Presentation pptx = new Presentation("DefaultFonts.pptx", loadOptions))
{
    // Generera bildminiatyr
    pptx.Slides[0].GetThumbnail(1, 1).Save("output_out.png", ImageFormat.Png);
    // Generera PDF
    pptx.Save("output_out.pdf", SaveFormat.Pdf);
    // Generera XPS
    pptx.Save("output_out.xps", SaveFormat.Xps);
}
```

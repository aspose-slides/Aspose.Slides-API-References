## LoadOptions.DeleteEmbeddedBinaryObjects egenskap

Bestämmer om Aspose.Slides kommer att ta bort alla inbäddade binära objekt vid presentationens inläsning.

Typerna av de inbäddade binära objekten:

* VBA-projekt [`VbaProject`](../../ipresentation/vbaproject)
* Inbäddad data för OLE-objekt [`EmbeddedFileData`](../../ioleembeddeddatainfo/embeddedfiledata)
* Binär data för ActiveX-kontroll [`ActiveXControlBinary`](../../icontrol/activexcontrolbinary)

Läs/skriv Boolean.

```csharp
public bool DeleteEmbeddedBinaryObjects { get; set; }
```

### Anmärkningar

Standardvärdet är **false**.

### Exempel

Följande exempel visar hur du laddar presentationen utan några inbäddade binära objekt.

```csharp
[C#]
LoadOptions loadOptions = new LoadOptions();
loadOptions.DeleteEmbeddedBinaryObjects = true;
using (Presentation pres = new Presentation("pres.ppt", loadOptions))
{
    pres.Save("output_WithoutBinaryObjects.ppt", SaveFormat.Ppt);
}
```

## ProtectionManager.CheckWriteProtection metod

Bestämmer om en presentation är lösenordsskyddad för att modifieras.

```csharp
public bool CheckWriteProtection(string password)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| password | String | Lösenordet för kontroll. |

### Returvärde

True if the password is valid; otherwise, false.

### Anmärkningar

1. Du bör kontrollera egenskapen [`IsWriteProtected`](../iswriteprotected) innan du anropar den här metoden. 2. När lösenordet är null eller tomt, returnerar den här metoden false.

### Exempel

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

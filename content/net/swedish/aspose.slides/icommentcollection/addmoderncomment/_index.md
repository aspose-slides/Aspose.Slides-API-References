## ICommentCollection.AddModernComment metod

Lägg till en ny modern kommentar i slutet av en samling.

```csharp
public IModernComment AddModernComment(string text, ISlide slide, IShape shape, PointF position, 
    DateTime creationTime)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| text | String | Oren text för en ny modern kommentar. |
| slide | ISlide | Bild i en presentation där en ny modern kommentar ska läggas till. |
| shape | IShape | Form på en bild som en ny modern kommentar är associerad med. |
| position | PointF | Position på en bild där en ny modern kommentar ska läggas till. |
| creationTime | DateTime | Tidpunkt för en modern kommentars skapande. |

### Returvärde

Tillagd modern kommentar.

### Exempel

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ICommentAuthor newAuthor = pres.CommentAuthors.AddAuthor("Some Author", "SA");
    newAuthor.Comments.AddModernComment("This is modern comment", pres.Slides[0], null, new PointF(100, 100), DateTime.Now);

    pres.Save(outPptxFileName, SaveFormat.Pptx);
}
```

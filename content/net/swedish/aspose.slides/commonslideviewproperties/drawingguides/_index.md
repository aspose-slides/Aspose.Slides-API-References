## CommonSlideViewProperties.DrawingGuides egenskap

Returnerar samlingen av ritningsguiderna. Endast läsning [`IDrawingGuidesCollection`](../../idrawingguidescollection)

```csharp
public IDrawingGuidesCollection DrawingGuides { get; }
```

### Exempel

Följande exempel kod visar hur man lägger till de nya ritningsguiderna i en PowerPoint-presentation.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    var slideSize = pres.SlideSize.Size;
    
    IDrawingGuidesCollection guides = pres.ViewProperties.SlideViewProperties.DrawingGuides;
    // Lägger till den nya vertikala ritningsguiden till höger om bildens centrum
    guides.Add(Orientation.Vertical, slideSize.Width / 2 + 12.5f);
    // Lägger till den nya horisontella ritningsguiden under bildens centrum
    guides.Add(Orientation.Horizontal, slideSize.Height / 2 + 12.5f);
    
    pres.Save("DrawingGuides_out.pptx", SaveFormat.Pptx);
}
```

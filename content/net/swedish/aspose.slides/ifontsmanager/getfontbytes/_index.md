## IFontsManager.GetFontBytes metod

Hämtar bytearrayen som representerar teckensnittsdata för en angiven typsnittsstil och teckensnittsdata.

```csharp
public byte[] GetFontBytes(IFontData fontData, FontStyleType fontStyle)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| fontData | IFontData | Teckensnittsdatatobjektet som innehåller information om teckensnittet [`IFontData`](../../ifontdata). |
| fontStyle | FontStyleType | Stilen på teckensnittet för vilken data ska hämtas [`FontStyleType`](../../fontstyletype). |

### Returvärde

En bytearray som innehåller teckensnittsdata för den angivna typsnittsstilen. Om teckensnittsdata eller stil inte hittas returneras null.

### Exempel

```csharp
[C#]
using (Presentation pres = new Presentation ("Presentation.pptx"))
{
    // Hämta alla teckensnitt som används i presentationen
    IFontData[] fonts = pres.FontsManager.GetFonts();

    // Hämta bytearrayen som representerar den vanliga stilen för det första teckensnittet i presentationen
    byte[] bytes = pres.FontsManager.GetFontBytes(fonts[0], FontStyleType.Regular);
}
```

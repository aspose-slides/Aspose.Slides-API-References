{
    foreach (var fontSubstitution in pres.FontsManager.GetSubstitutions())

## GetSubstitutions(int[]) {#getsubstitutions_1}

Hämtar information om teckensnitt som kommer att ersättas under rendering av de angivna bilderna.

```csharp
public IEnumerable<FontSubstitutionInfo> GetSubstitutions(int[] slides)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| slides | Int32[] | En array av bildindex för vilka teckensnittsersättningsinformation ska hämtas, med början från 1. |

### Returvärde

En samling av alla teckensnittsersättningar ([`FontSubstitutionInfo`](../../fontsubstitutioninfo)) för de angivna bilderna.

### Exempel

```csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    int[] targetSlides = { 1, 2, 5 };
    foreach (var fontSubstitution in pres.FontsManager.GetSubstitutions(targetSlides))
    {
        Console.WriteLine("{0} -> {1}", fontSubstitution.OriginalFontName, fontSubstitution.SubstitutedFontName);
    }
}
```

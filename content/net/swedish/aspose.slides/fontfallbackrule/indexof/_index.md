## FontFallBackRule.IndexOf metod

Returnerar ett index för den angivna regeln i samlingen.

```csharp
public int IndexOf(string fontName)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| fontName | String | Fontens namn att hitta. |

### Returvärde

Index för ett typsnitt eller -1 om typsnittet inte hittas i listan.

### Exempel

```csharp
[C#]
// Skapa en regel som innehåller en lista med typsnitt.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// Hämta index för Tahoma.
int tahomaIndex = newRule.IndexOf("Tahoma");
```

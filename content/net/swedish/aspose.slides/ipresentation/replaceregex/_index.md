## IPresentation.ReplaceRegex-metod

Ersätter alla matchningar av det reguljära uttrycket med den angivna strängen.

```csharp
public void ReplaceRegex(Regex regex, string newText, IFindResultCallback callback)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| regex | Regex | Det reguljära uttrycket Regex för att hämta strängar att ersätta. |
| newText | String | Strängen för att ersätta alla förekomster av de strängar som ska ersättas. |
| callback | IFindResultCallback | Callback-objektet för att ta emot sökresultat [`IFindResultCallback`](../../ifindresultcallback). |

### Exempel

Följande kodexempel visar hur man ersätter text med ett reguljärt uttryck och den angivna strängen.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// Ersätt alla ord med 10 eller fler tecken med '***'
	presentation.ReplaceRegex(regex, "***", null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

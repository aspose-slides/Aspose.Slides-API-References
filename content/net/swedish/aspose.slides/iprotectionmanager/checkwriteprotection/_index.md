## IProtectionManager.CheckWriteProtection metod

Bestämmer om en presentation är lösenordsskyddad för att ändras.

```csharp
public bool CheckWriteProtection(string password)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| password | String | Lösenordet för kontroll. |

### Returvärde

Sant om lösenordet är giltigt; annars falskt.

### Anmärkningar

1. Du bör kontrollera egenskapen [`IsWriteProtected`](../iswriteprotected) innan du anropar den här metoden. 2. När lösenordet är null eller tomt returnerar den här metoden falskt.

### Exempel

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

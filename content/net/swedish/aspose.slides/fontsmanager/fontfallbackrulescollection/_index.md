## FontsManager.FontFallBackRulesCollection egenskap

Representerar en användares samling av FontFallBack-regler för hantering av samlingar av teckensnitt för korrekta ersättningar via fallback-funktionalitet Läs/skriv [`IFontFallBackRulesCollection`](../../ifontfallbackrulescollection).

```csharp
public IFontFallBackRulesCollection FontFallBackRulesCollection { get; set; }
```

### Exempel

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    // Hämtar en tom eller förinitialiserad regelsamling från FontsManager
    IFontFallBackRulesCollection rulesList = pres.FontsManager.FontFallBackRulesCollection;

    // lägger till regler i samlingen
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));

    // eller 
    // initiering av en ny instans av regelsamlingen
    IFontFallBackRulesCollection rulesList = new FontFallBackRulesCollection();

    // lägger till regler i samlingen
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));

    // och ersätter befintlig samling med den nya i FontsManager 
    pres.FontsManager.FontFallBackRulesCollection = rulesList;
}
```

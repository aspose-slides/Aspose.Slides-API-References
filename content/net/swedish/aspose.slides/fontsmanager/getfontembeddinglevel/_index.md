## FontsManager.GetFontEmbeddingLevel metod

Bestämmer inbäddningsnivån för ett typsnitt från den givna byte-arrayen och typsnittsnamnet.

```csharp
public EmbeddingLevel GetFontEmbeddingLevel(byte[] fontBytes, string fontName)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| fontBytes | Byte[] | Byte-arrayen som innehåller typsnittets data. |
| fontName | String | Namnet på typsnittet. |

### Returvärde

Inbäddningsnivån för det angivna typsnittet.

### Undantag

| undantag | villkor |
| --- | --- |
| ArgumentNullException | Kastas när *fontBytes* är null. |

### Exempel

```csharp
[C#]
using (Presentation pres = new Presentation(pptxFileName))
{
    // Hämta alla typsnitt som används i presentationen
    IFontData[] fontDatas = pres.FontsManager.GetFonts();

    // Hämta byte-arrayen som representerar den vanliga stilen för det första typsnittet i presentationen
    byte[] bytes = pres.FontsManager.GetFontBytes(fontDatas[0], FontStyle.Regular);

    // Bestäm inbäddningsnivån för typsnittet
    EmbeddingLevel embeddingLevel = pres.FontsManager.GetFontEmbeddingLevel(bytes, fontDatas[0].FontName);
}
```

## MathParagraph.Remove metod

Tar bort den första förekomsten av ett specifikt objekt från samlingen/&gt;.

```csharp
public bool Remove(IMathBlock mathBlock)
```

| Parameter | Type | Description |
| --- | --- | --- |
| mathBlock | IMathBlock | Objektet som ska tas bort från samlingen. |

### Returvärde

true om *mathBlock* framgångsrikt togs bort från samlingen; annars false. Denna metod returnerar också false om *mathBlock* inte hittas i den ursprungliga samlingen/&gt;.

### Exempel

Example:

```csharp
[C#]
IAutoShape shape = slide.Shapes.AddMathShape(x, y, width, height);
IMathParagraph mathParagraph = (shape.TextFrame.Paragraphs[0].Portions[0] as MathPortion).MathParagraph;
mathParagraph.Add(new MathBlock(new MathematicalText("x")));
IMathBlock block = new MathBlock(new MathematicalText("y"));
mathParagraph.Add(block);
mathParagraph.Remove(block);
```

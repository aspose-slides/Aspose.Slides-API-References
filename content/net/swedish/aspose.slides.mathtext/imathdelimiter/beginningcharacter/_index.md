## IMathDelimiter.BeginningCharacter egenskap

Delimiter Beginning Character anger det inledande, eller öppnande, avgränsartecknet. Matematiska avgränsare är inneslutande tecken såsom parenteser, hakparenteser och klammerparenteser. Standardvärdet: '('.

```csharp
public char BeginningCharacter { get; set; }
```

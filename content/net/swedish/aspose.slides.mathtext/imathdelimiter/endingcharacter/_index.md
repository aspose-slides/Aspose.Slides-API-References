## IMathDelimiter.EndingCharacter egenskap

Delimiter Ending Character specificerar det avslutande, eller slutande, avgränsartecken. Matematiska avgränsare är omslutande tecken såsom parenteser, hakparenteser och klamrar. Standardvärdet: ')'.

```csharp
public char EndingCharacter { get; set; }
```

## IMathDelimiter.DelimiterShape egenskap

Anger formen på avgränsare i avgränsareobjektet. När det är MathDelimiterShape.Centered centreras avgränsare kring den matematiska axeln i den matematiska texten och görs så att de passar hela höjden på deras innehåll. När det är MathDelimiterShape.Match ändras deras höjd och form så att de exakt matchar deras innehåll.

```csharp
public MathDelimiterShape DelimiterShape { get; set; }
```

### Exempel

Exempel:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.DelimiterShape = MathDelimiterShape.Match;
```

## IMathArray.RowSpacing egenskap

Utrymme mellan rader i en array. Det används endast när RowSpacingRule är satt till 3 Exactly, i vilket fall måttenheten är punkter eller Multiple, i vilket fall måttenheten är halvlinjer. Default: 0

```csharp
public uint RowSpacing { get; set; }
```

### Exempel

Exempel:

```csharp
[C#]
IMathArray mathArray = new MathArray(new MathematicalText("item1"));
mathArray.RowSpacingRule = MathRowSpacingRule.Exactly;
mathArray.RowSpacing = 10;
```

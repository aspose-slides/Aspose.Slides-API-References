## IMathMatrix.ColumnGap egenskap

Värdet för horisontellt avstånd mellan kolumner i en matris; Om ColumnGapRule är inställd på 3 ("Exactly"), tolkas enheten som twips (1/20 av en punkt) Om ColumnGapRule är inställd på 4 ("Multiple"), tolkas enheten som antal 0.5 em-steg. I andra fall ignoreras den. Standard: 0

```csharp
public uint ColumnGap { get; set; }
```

### Exempel

Exempel:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.ColumnGapRule = MathSpacingRules.Exactly;
matrix.ColumnGap = 20;
```

## IMathPhantom.Transp egenskap

Hämtar eller anger ett värde som indikerar om fantomen är transparent för klassbaserade avståndsregler.

```csharp
public bool Transp { get; set; }
```

### Anmärkningar

När `true` påverkar operatorer och symboler i fantomen fortfarande matematisk avstånd runt fantomen (som om den var synlig). När `false` ignoreras klassbaserad avstånd. Motsvarar OMML-attributet `m:transp`.

## MathDelimiter.BeginningCharacter egenskap

Delimiter Beginning Character specificerar början, eller öppning, av avgränsartecknet. Matematiska avgränsare är omslutande tecken såsom parenteser, hakparenteser och klammerparenteser. Standardvärdet: '('.

```csharp
public char BeginningCharacter { get; set; }
```

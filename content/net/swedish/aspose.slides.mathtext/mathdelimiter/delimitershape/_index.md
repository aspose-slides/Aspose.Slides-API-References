## MathDelimiter.DelimiterShape egenskap

Specificerar formen på avgränsare i avgränsare-objektet. När den är MathDelimiterShape.Centered, är avgränsarna centrerade kring den matematiska axeln i den matematiska texten och kan fortfarande anpassas för att passa hela höjden av deras innehåll. När den är MathDelimiterShape.Match, ändras deras höjd och form för att exakt matcha deras innehåll.

```csharp
public MathDelimiterShape DelimiterShape { get; set; }
```

### Exempel

Exempel:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.DelimiterShape = MathDelimiterShape.Match;
```

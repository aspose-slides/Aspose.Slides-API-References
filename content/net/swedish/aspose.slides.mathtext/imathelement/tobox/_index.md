## IMathElement.ToBox metod

Placera detta element i en icke-visuell ruta (logisk gruppering) som används för att gruppera komponenter i en ekvation eller annan instans av matematisk text. Ett inramat objekt kan (till exempel) fungera som en operator-emulator med eller utan en justeringspunkt, fungera som en radbrytningspunkt, eller grupperas så att radbrytningar inte tillåts inom det.

```csharp
public IMathBox ToBox()
```

### Returvärde

Logisk ruta med detta element placerat inuti

### Exempel

Exempel:

```csharp
[C#]
IMathBox box = new MathematicalText("x:=y").ToBox();
```

## MathBorderBox.StrikethroughTopLeftToBottomRight egenskap

Genomstruken uppifrån vänster till nedifrån höger (standard är false). Anger det dolda eller visade tillståndet för en genomstruken diagonal linje från det övre vänstra hörnet till det nedre högra hörnet av ramboxen.

```csharp
public bool StrikethroughTopLeftToBottomRight { get; set; }
```

### Exempel

Exempel:

```csharp
[C#]
MathBorderBox borderBox = new MathBorderBox(new MathematicalText("x"));
borderBox.StrikethroughTopLeftToBottomRight = true;
```

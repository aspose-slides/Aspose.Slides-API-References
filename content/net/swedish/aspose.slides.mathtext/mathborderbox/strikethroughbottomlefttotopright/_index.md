## MathBorderBox.StrikethroughBottomLeftToTopRight egenskap

Genomstrykning från nedre vänstra till övre högra (standard är falskt). Anger om en genomstruken diagonal linje från det nedre vänstra hörnet till det övre högra hörnet av ramrutan är dold eller visas.

```csharp
public bool StrikethroughBottomLeftToTopRight { get; set; }
```

### Exempel

Example:

```csharp
[C#]
MathBorderBox borderBox = new MathBorderBox(new MathematicalText("x"));
borderBox.StrikethroughBottomLeftToTopRight = true;
```

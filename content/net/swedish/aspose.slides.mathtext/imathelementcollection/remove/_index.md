## IMathElementCollection.Remove metod

Tar bort den första förekomsten av ett specifikt objekt från samlingen.

```csharp
public bool Remove(IMathElement item)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| item | IMathElement | Objektet som ska tas bort från samlingen. |

### Returvärde

true om *item* togs bort från samlingen; annars false. Denna metod returnerar också false om *item* inte hittas i den ursprungliga samlingen.

### Exempel

Exempel:

```csharp
[C#]
IMathElementCollection collection = new MathBlock(new MathematicalText("x"));
MathematicalText plusElement = new MathematicalText("+");
collection.Add(plusElement);
collection.Add(new MathRadical(new MathematicalText("x"), new MathematicalText("3")));
collection.Remove(plusElement);
```

## MathBox.ExplicitBreak egenskap

Explicit break anger om det finns ett radbrytning i början av Box-objektet, så att raden bryts i början av box-objektet. Anger numret på operatorn på föregående rad med matematisk text som ska användas som justeringspunkt för den aktuella raden med matematisk text möjliga värden: 1..255 Standard: 0 (ingen explicit break)

```csharp
public byte ExplicitBreak { get; set; }
```

### Exempel

Exempel:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.ExplicitBreak = 1;
```

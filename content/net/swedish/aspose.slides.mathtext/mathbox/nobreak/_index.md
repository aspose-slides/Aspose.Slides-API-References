## MathBox.NoBreak egenskap

Ingen brytning Den här egenskapen specificerar egenskapen "unbreakable" på objektboxen. När true kan inga radbrytningar ske inom boxen. Detta kan vara viktigt för operatoremulatorer som består av mer än en binär operator. När detta element inte anges kan radbrytningar ske i boxen. Standard: true

```csharp
public bool NoBreak { get; set; }
```

### Exempel

Exempel:

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("*****"));
box.NoBreak = false;
```

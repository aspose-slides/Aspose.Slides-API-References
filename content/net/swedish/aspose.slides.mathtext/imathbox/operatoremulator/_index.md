## IMathBox.OperatorEmulator egenskap

Operator Emulator. När true, beter lådan och dess innehåll sig som en enda operator och ärver egenskaperna hos en operator. Det betyder till exempel att tecknet kan fungera som en punkt för en radbrytning och kan justeras till andra operatorer. Operator Emulators används ofta när en eller flera glyfer kombineras för att bilda en operator, såsom '=='. Standardvärde: false

```csharp
public bool OperatorEmulator { get; set; }
```

### Exempel

Exempel:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.OperatorEmulator = true;
```

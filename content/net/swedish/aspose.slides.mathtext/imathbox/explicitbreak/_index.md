## IMathBox.ExplicitBreak egenskap

Explicit break anger om det finns ett radbryt vid början av Box-objektet, så att raden bryts vid början av box-objektet. Anger numret på operatorn på föregående rad av matematisk text som ska användas som justeringspunkt för den aktuella raden av matematisk text. Möjliga värden: 1..255 Standard: 0 (ingen explicit radbryt)

```csharp
public byte ExplicitBreak { get; set; }
```

### Exempel

Exempel:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.ExplicitBreak = 1;
```

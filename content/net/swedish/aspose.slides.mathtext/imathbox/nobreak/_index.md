## IMathBox.NoBreak egenskap

Ingen brytning. Denna egenskap anger den "unbreakable"-egenskapen på objektboxen. När den är true kan inga radbrytningar inträffa inom boxen. Detta kan vara viktigt för operator-emulatorer som består av mer än en binär operator. När detta element inte anges kan brytningar inträffa i boxen. Default: true

```csharp
public bool NoBreak { get; set; }
```

### Exempel

Exempel:

```csharp
[C#]
IMathBox box = new MathematicalText("**********").ToBox();
box.NoBreak = false;
```

## IMathBorderBox.StrikethroughBottomLeftToTopRight egenskap

Strikethrough Bottom-Left to Top-Right (standard är falskt). Anger det dolda eller visade tillståndet för en genomstruken diagonal linje från den nedre vänstra hörnet till det övre högra hörnet av kantlådan.

```csharp
public bool StrikethroughBottomLeftToTopRight { get; set; }
```

### Exempel

Exempel:

```csharp
[C#]
IMathBorderBox borderBox = new MathematicalText("x+y+z").ToBorderBox();
borderBox.StrikethroughBottomLeftToTopRight = true;
```

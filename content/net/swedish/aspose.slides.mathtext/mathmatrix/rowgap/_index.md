## MathMatrix.RowGap egenskap

Värdet för vertikalt avstånd mellan rader i en matris; Om RowGapRule är inställd på 3 ("Exactly"), tolkas enheten som twips (1/20 av en punkt) Om RowGapRule är inställd på 4 ("Multiple"), tolkas enheten som halva rader. Standard: 0

```csharp
public uint RowGap { get; set; }
```

### Exempel

Exempel:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

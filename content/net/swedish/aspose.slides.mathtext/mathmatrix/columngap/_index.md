## MathMatrix.ColumnGap egenskap

Värdet på horisontellt avstånd mellan kolumner i en matris; om ColumnGapRule är satt till 3 ("Exactly") tolkas enheten som twips (1/20 av en punkt) om ColumnGapRule är satt till 4 ("Multiple") tolkas enheten som antal 0,5 em-increment. I andra fall ignoreras. Standard: 0

```csharp
public uint ColumnGap { get; set; }
```

### Exempel

Exempel:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.ColumnGapRule = MathSpacingRules.Exactly;
matrix.ColumnGap = 20;
```

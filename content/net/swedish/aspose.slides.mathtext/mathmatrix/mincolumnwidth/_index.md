## MathMatrix.MinColumnWidth egenskap

Minsta kolumnbredd i twips (1/20 av en punkt). Avståndet mellan kolumner (även kallat “Column Gap” eller “Gap Width”) läggs till MinColumnWidth för att bestämma den totala Matrix Column Spacing (avståndet mellan samma kanter på olika kolumner). Standard: 0.

```csharp
public uint MinColumnWidth { get; set; }
```

### Exempel

Exempel:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.MinColumnWidth = 20;
```

## ToTiff(Presentation, string) {#totiff}

Konverterar den angivna presentationen till en uppsättning TIFF-formatbilder. Om utdatafilnamnet anges som "myPath/myFilename.tiff" sparas resultatet som en uppsättning "myPath/myFilename_N.tiff"-filer, där N är ett bildnummer.

public static void ToTiff(Presentation pres, string outputFileName)

## ToTiff(Presentation, string, ITiffOptions, bool) {#totiff_1}

Konverterar den angivna presentationen till TIFF-format med anpassade alternativ. Om utdatafilnamnet anges som "myPath/myFilename.tiff" och *multipage* är `false` sparas resultatet som en uppsättning "myPath/myFilename_N.tiff"-filer, där N är ett bildnummer. Om *multipage* däremot är `true` blir resultatet ett flersidigt "myPath/myFilename.tiff"-dokument.

```csharp
public static void ToTiff(Presentation pres, string outputFileName, ITiffOptions options, 
    bool multipage)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| pres | Presentation | Den inmatade presentationen. |
| outputFileName | String | Utdatafilnamnet. |
| options | ITiffOptions | Inställningarna för att spara TIFF. |
| multipage | Boolean | Anger om det genererade TIFF-dokumentet ska vara flersidigt. |

### Undantag

| Undantag | villkor |
| --- | --- |
| ArgumentException |  |

### Exempel

```csharp
ITiffOptions options = new TiffOptions()
{
    CompressionType = TiffCompressionTypes.CCITT3,
    SlidesLayoutOptions = new NotesCommentsLayoutingOptions()
    {
        NotesPosition = NotesPositions.BottomTruncated
    }
};

using (var pres = new Presentation("pres.pptx"))
   Convert.ToTiff(pres, "pres.tiff", options, false);
```

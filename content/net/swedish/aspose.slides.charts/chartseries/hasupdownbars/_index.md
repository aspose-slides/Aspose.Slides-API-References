## ChartSeries.HasUpDownBars egenskap

Bestämmer om linje- eller aktiediagram har upp/ner staplar. Denna egenskap gäller inte bara för denna serie utan för alla serier i den överordnade serieggruppen - detta är en projicering av den motsvarande gruppergenskapen. Därför är denna egenskap skrivskyddad. Använd ParentSeriesGroup egenskap för åtkomst till den överordnade serieggruppen. Använd ParentSeriesGroup.UpDownBars.HasUpDownBars läs/skriv egenskap för att ändra värdet. Använd ParentSeriesGroup.UpDownBars egenskap för att formatera upp/ner staplar. Skrivskyddad Boolean.

```csharp
public bool HasUpDownBars { get; }
```

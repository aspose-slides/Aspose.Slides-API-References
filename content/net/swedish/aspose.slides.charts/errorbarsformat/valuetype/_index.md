## ErrorBarsFormat.ValueType egenskap

Representerar möjliga sätt att bestämma längden på felstaplarna. Om ett anpassat värdetyp används för att ange värde, använd [`ErrorBarsCustomValues`](../../ichartdatapoint/errorbarscustomvalues) egenskap för en specifik datapunkt i DataPoints-samlingen för serien. Om värdetypen är Fixed, Percentage eller StandardDeviation, använd Value egenskap för att ange värde. Läs/skriv [`ErrorBarValueType`](../../errorbarvaluetype).

## DataLabelFormat.ShowValue egenskap

Representerar en angiven diagrammets datalabels procentsatsvärdes visningsbeteende. True visar procentsatsvärdet. False för att dölja. Läs/skriv Boolean.

```csharp
public bool ShowValue { get; set; }
```

### Anmärkningar

Om föräldern till detta DataLabelFormat-objekt är en DataLabelCollection-samling av datalabels så får eller sätter denna egenskap standardvärdet för ShowValue-egenskapen för de nya datalabelsen i DataLabelCollection-samlingen. Att sätta denna egenskap med ett värde sätter också detta värde till ShowValue-egenskapen för alla datalabels i DataLabelCollection-samlingen (dvs. "DataLabels.DefaultDataLabelFormat.ShowValue = val;" får alla DataLabels[i].ShowValue lika med val).

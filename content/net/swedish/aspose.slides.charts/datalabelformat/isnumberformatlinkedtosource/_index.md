## DataLabelFormat.IsNumberFormatLinkedToSource property

Läs/skriv Boolean.

```csharp
public bool IsNumberFormatLinkedToSource { get; set; }
```

### Anmärkningar

Om föräldern till detta DataLabelFormat-objekt är en DataLabelCollection-samling av dataetiketter, så hämtar eller anger denna egenskap standardvärdet för egenskapen IsNumberFormatLinkedToSource för de nya dataetiketterna i DataLabelCollection-samlingen. Att sätta denna egenskap med ett värde sätter också detta värde på egenskapen IsNumberFormatLinkedToSource för alla dataetiketter i DataLabelCollection-samlingen (t.ex. "DataLabels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = val;" vilket gör att alla DataLabels[i].IsNumberFormatLinkedToSource blir lika med val).

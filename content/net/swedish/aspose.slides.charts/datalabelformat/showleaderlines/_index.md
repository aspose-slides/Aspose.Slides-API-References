## DataLabelFormat.ShowLeaderLines egenskap

Representerar ett specificerat diagrammets dataetikettledarlinjers visningsbeteende. True visar ledarlinjerna. False för att dölja. Läs/skriv Boolean.

```csharp
public bool ShowLeaderLines { get; set; }
```

### Anmärkningar

Om föräldern till detta DataLabelFormat-objekt är en DataLabelCollection-samling av dataetiketter hämtar eller anger denna egenskap standardvärdet för ShowLeaderLines-egenskapen för de nya dataetiketterna i DataLabelCollection-samlingen. Att ange denna egenskap med ett värde sätter också detta värde på ShowLeaderLines-egenskapen för alla dataetiketter i DataLabelCollection-samlingen (dvs. "DataLabels.DefaultDataLabelFormat.ShowLeaderLines = val;" gör att alla DataLabels[i].ShowLeaderLines får värdet val).

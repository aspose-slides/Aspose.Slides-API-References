## DataLabelFormat.ShowCategoryName egenskap

Representerar ett specificerat diagramdataetiketts kategorinamns visningsbeteende. True för att visa kategorinamnet för dataetiketterna på ett diagram. False för att dölja. Läs/skriv Boolean.

```csharp
public bool ShowCategoryName { get; set; }
```

### Anmärkningar

Om föräldern till detta DataLabelFormat-objekt är en DataLabelCollection-samling av dataetiketter så får eller sätter denna egenskap standardvärdet för ShowCategoryName-egenskapen för de nya dataetiketterna i DataLabelCollection-samlingen. Att ange denna egenskap med ett värde sätter också detta värde till ShowCategoryName-egenskapen för alla dataetiketter i DataLabelCollection-samlingen (dvs. "DataLabels.DefaultDataLabelFormat.ShowCategoryName = val;" medför att alla DataLabels[i].ShowCategoryName är lika med val).

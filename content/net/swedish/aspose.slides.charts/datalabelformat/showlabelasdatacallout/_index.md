## DataLabelFormat.ShowLabelAsDataCallout egenskap

Bestämmer om den angivna diagrammets dataetikett ska visas som dataanrop eller som dataetikett. Om föräldern till detta DataLabelFormat objekt är en DataLabelCollection samling av dataetiketter, så hämtar eller anger denna egenskap standardvärdet för ShowLabelAsDataCallout egenskapen för de nya dataetiketterna i DataLabelCollection samlingen. Att ange denna egenskap med ett värde sätter också detta värde på ShowLabelAsDataCallout egenskapen för alla dataetiketter i DataLabelCollection samlingen (t.ex. "DataLabels.DefaultDataLabelFormat.ShowLabelAsDataCallout = val;" får alla DataLabels[i].ShowLabelAsDataCallout lika med val).

public bool ShowLabelAsDataCallout { get; set; }

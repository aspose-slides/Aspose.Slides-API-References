## DataLabelFormat.ShowBubbleSize egenskap

Representerar ett angivet diagrammets datamärkes bubbla storlek värdevisningsbeteende. True visar bubbla storleksvärdet. False för att dölja. Läs/skriv Boolean.

```csharp
public bool ShowBubbleSize { get; set; }
```

### Anmärkningar

Om föräldern till detta DataLabelFormat-objekt är en DataLabelCollection-samling av datamärkningar, så får eller sätter denna egenskap standardvärdet för ShowBubbleSize-egenskapen för de nya datamärkningarna i DataLabelCollection-samlingen. Att sätta denna egenskap med ett värde sätter också detta värde för ShowBubbleSize-egenskapen för alla datamärkningar i DataLabelCollection-samlingen (dvs. "DataLabels.DefaultDataLabelFormat.ShowBubbleSize = val;" orsakar att alla DataLabels[i].ShowBubbleSize är lika med val).

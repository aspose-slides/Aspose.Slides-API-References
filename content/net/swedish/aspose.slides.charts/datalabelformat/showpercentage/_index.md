## DataLabelFormat.ShowPercentage egenskap

Representerar beteendet för hur procentvärdet för ett specificerat diagramdataetikett visas. True visar procentvärdet. False döljer det. Läs/skriv Boolesk.

```csharp
public bool ShowPercentage { get; set; }
```

### Anmärkningar

Om föräldern till detta DataLabelFormat-objekt är en DataLabelCollection-samling av dataetiketter så hämtar eller sätter den här egenskapen standardvärdet för ShowPercentage-egenskapen för de nya dataetiketterna i DataLabelCollection-samlingen. Att sätta denna egenskap med ett värde sätter också detta värde till ShowPercentage-egenskapen för alla dataetiketter i DataLabelCollection-samlingen (dvs. "DataLabels.DefaultDataLabelFormat.ShowPercentage = val;" orsakar att alla DataLabels[i].ShowPercentage är lika med val).

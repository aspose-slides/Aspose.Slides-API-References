## DataLabelFormat.NumberFormat egenskap

Representerar formatsträngen för DataLabels-objektet. Läs/skriv String.

```csharp
public string NumberFormat { get; set; }
```

### Anmärkningar

Om föräldern till detta DataLabelFormat-objekt är en DataLabelCollection-samling av dataetiketter, då får eller sätter denna egenskap standardvärdet för NumberFormat-egenskapen för de nya dataetiketterna i DataLabelCollection-samlingen. När denna egenskap sätts till ett värde, sätts samma värde också för NumberFormat-egenskapen för alla dataetiketter i DataLabelCollection-samlingen (i.e. "DataLabels.DefaultDataLabelFormat.NumberFormat = val;" gör att alla DataLabels[i].NumberFormat får värdet val).

### Exempel

```csharp
[C#]
series.Labels.DefaultDataLabelFormat.ShowValue = true;
series.Labels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = false;
series.Labels.DefaultDataLabelFormat.NumberFormat = "0.0%";
```

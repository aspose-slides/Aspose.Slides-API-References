## DataLabelCollection.Hide metod

Gör datamärkning dold som standard genom att sätta alla Show*-flags (ShowValue, ...) för egenskapen DefaultDataLabelFormat till false. IsVisible blir false efter detta.

```csharp
public void Hide()
```

### Anmärkningar

Om datamärkning inte är synlig som standard (IsVisible är false) kan du göra datamärkning "synlig som standard" genom att sätta Show*-flags (ShowValue, ...) för egenskapen DefaultDataLabelFormat till true.

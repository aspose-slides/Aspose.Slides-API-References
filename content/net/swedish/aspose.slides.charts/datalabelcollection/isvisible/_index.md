## DataLabelCollection.IsVisible egenskap

False betyder att dataetiketten inte är synlig som standard (och att alla Show*-flags (ShowValue, ...) för egenskapen DefaultDataLabelFormat är falska). Skrivskyddad Boolean.

```csharp
public bool IsVisible { get; }
```

### Anmärkningar

Om dataetiketten är synlig som standard kan du dölja den som standard med Hide()-metoden. Men om dataetiketten inte är synlig som standard (IsVisible är false) kan du göra dataetiketten "synlig som standard" genom att sätta Show*-flags (ShowValue, ...) för egenskapen DefaultDataLabelFormat till true-tillstånd.

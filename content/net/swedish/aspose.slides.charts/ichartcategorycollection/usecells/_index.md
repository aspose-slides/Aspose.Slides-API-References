## IChartCategoryCollection.UseCells egenskap

Om true så används worksheet för att lagra kategorier (detta fall stödjer en flernivåkategori). Om false så används worksheet INTE för att lagra värden (och detta fall stödjer inte en flernivåkategori). Läs/skriv Boolean.

public bool UseCells { get; set; }

## ChartData.UseSecondaryCategories egenskap

Om falskt så [`SecondaryCategories`](../secondarycategories) egenskap returnerar null och data i [`Categories`](../categories) egenskap används både för primära och sekundära serier. Om sant så data i [`SecondaryCategories`](../secondarycategories) egenskap används för sekundära serier och data i [`Categories`](../categories) egenskap används för primära serier. Läs/skriv Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

### Exempel

Exempel. Vilka kategorier är relaterade till serier - ChartData.Categories eller ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // relaterade kategorier är series.Chart.ChartData.SecondaryCategories
}
else
{
    // relaterade kategorier är series.Chart.ChartData.Categories
}
```

## ChartData.Categories egenskap

Hämtar de primära kategorierna (eller både primära och sekundära kategorier om [`UseSecondaryCategories`](../usesecondarycategories) property är false). Skrivskyddad [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection Categories { get; }
```

### Anmärkningar

Om [`UseSecondaryCategories`](../usesecondarycategories) property är false så returnerar [`SecondaryCategories`](../secondarycategories) property null och data i denna `Categories` property används både för primära och sekundära serier. Om [`UseSecondaryCategories`](../usesecondarycategories) property är true så används data i [`SecondaryCategories`](../secondarycategories) property för sekundära serier och data i denna `Categories` property används för primära serier.

### Exempel

Exempel. Vilka kategorier är relaterade till serier - ChartData.Categories eller ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // relaterade kategorier är series.Chart.ChartData.SecondaryCategories
}
else
{
    // relaterade kategorier är series.Chart.ChartData.Categories
}
```

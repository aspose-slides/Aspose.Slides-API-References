## ChartCategoryCollection.UseCells egenskap

Om true så används arbetsbladet för att lagra kategorier (detta fall stöder flernivåkategorier). Om false så används arbetsbladet INTE för att lagra värden (och detta fall stöder inte flernivåkategorier). Läs/skriv Boolean.

public bool UseCells { get; set; }

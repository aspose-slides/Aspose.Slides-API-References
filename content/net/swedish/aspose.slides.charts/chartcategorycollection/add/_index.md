## Add(object) {#add_1}

Skapar en ny [`ChartCategory`](../../chartcategory) från värdet och lägger till den i samlingen.

```csharp
public IChartCategory Add(object value)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| value | Object | Värdet. |

### Returvärde

Tillagd [`IChartCategory`](../../ichartcategory).

### Undantag

| undantag | villkor |
| --- | --- |
| InvalidOperationException | om gränsen överskrids |

### Anmärkningar

Denna metod lägger till ett kalkylblad med namnet AUTO_DATA och lägger till alla värden där. Om du använder [`ChartDataWorkbook`](../../chartdataworkbook) för att lägga till eller redigera cellvärden, se till att du inte använder detta kalkylblad. Maximalt antal värden som kan läggas till med denna metod får inte överstiga 16711680.

## DataLabel.IsVisible egenskap

False betyder att datamärket inte är synligt (och därmed är alla Show*-flaggor (ShowValue, ...) falska). Skrivskyddad Boolean.

```csharp
public bool IsVisible { get; }
```

### Anmärkningar

Om datamärket är synligt kan du dölja det med Hide()-metoden. Men om datamärket inte är synligt (IsVisible är false) kan du göra datamärket synligt genom att sätta Show*-flaggor (ShowValue, ...) till true-tillstånd.

## IChartCategory.UseCell egenskap

Om true är AsCell egenskap aktuell. Med andra ord används arbetsblad för att lagra kategori (detta fall stödjer en flernivåkategori). Om false är AsLiteral egenskap aktuell. Med andra ord används arbetsblad INTE för att lagra kategori (och detta fall stödjer inte flernivåkategorier). Skrivskyddad Boolean.

```csharp
public bool UseCell { get; }
```

### Anmärkningar

För att ändra värdet på denna egenskap (för alla kategorier i samlingen) sätt det nya värdet till ChartCategoryCollection.UseCells egenskap.

## IChartData.Categories egenskap

Hämtar de primära kategorierna (eller både primära och sekundära kategorier om [`UseSecondaryCategories`](../usesecondarycategories) egenskap är false). Skrivskyddad [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection Categories { get; }
```

### Anmärkningar

Om [`UseSecondaryCategories`](../usesecondarycategories) egenskap är false så [`SecondaryCategories`](../secondarycategories) egenskap returnerar null och data i denna `Categories` egenskap används både för primära och sekundära serier. Om [`UseSecondaryCategories`](../usesecondarycategories) egenskap är true så används data i [`SecondaryCategories`](../secondarycategories) egenskap för sekundära serier och data i denna `Categories` egenskap för primära serier.

### Exempel

Exempel. Vilka kategorier är relaterade till serier - ChartData.Categories eller ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // relaterade kategorier är series.Chart.ChartData.SecondaryCategories
}
else
{
    // relaterade kategorier är series.Chart.ChartData.Categories
}
```

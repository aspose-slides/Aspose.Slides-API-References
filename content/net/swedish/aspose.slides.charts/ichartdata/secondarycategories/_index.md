## IChartData.SecondaryCategories egenskap

Hämtar de sekundära kategorierna om [`UseSecondaryCategories`](../usesecondarycategories) egenskap är sann. Skrivskyddad [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### Anmärkningar

Om [`UseSecondaryCategories`](../usesecondarycategories) egenskap är falsk returnerar denna `SecondaryCategories` egenskap null och data i [`Categories`](../categories) egenskap används både för primära och sekundära serier. Om [`UseSecondaryCategories`](../usesecondarycategories) egenskap är sann används data i denna `SecondaryCategories` egenskap för sekundära serier och data i [`Categories`](../categories) egenskap används för primära serier.

### Exempel

Exempel. Vilka kategorier är relaterade till serier - ChartData.Categories eller ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // relaterade kategorier är series.Chart.ChartData.SecondaryCategories
}
else
{
    // relaterade kategorier är series.Chart.ChartData.Categories
}
```

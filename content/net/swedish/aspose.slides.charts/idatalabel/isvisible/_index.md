## IDataLabel.IsVisible egenskap

False betyder att data label inte är synlig (och så är alla Show*-flags (ShowValue, ...) falska). Skrivskyddad Boolean.

```csharp
public bool IsVisible { get; }
```

### Anmärkningar

Om data label är synlig kan du dölja den med Hide()-metoden. Men om data label inte är synlig (IsVisible är false) kan du göra data label synlig genom att sätta Show*-flags (ShowValue, ...) till true.

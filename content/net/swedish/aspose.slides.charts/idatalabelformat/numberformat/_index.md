## IDataLabelFormat.NumberFormat egenskap

Representerar formatsträngen för DataLabels-objektet. Läs/skriv String.

```csharp
public string NumberFormat { get; set; }
```

### Anmärkningar

Om föräldern till detta DataLabelFormat-objekt är en DataLabelCollection-samling av dataetiketter, så hämtar eller sätter den här egenskapen standardvärdet för NumberFormat-egenskapen för de nya dataetiketterna i DataLabelCollection-samlingen. När denna egenskap sätts till ett värde, sätts även det värdet för NumberFormat-egenskapen för alla dataetiketter i DataLabelCollection-samlingen (dvs. "DataLabels.DefaultDataLabelFormat.NumberFormat = val;" får alla DataLabels[i].NumberFormat att vara lika med val).

### Exempel

```csharp
[C#]
series.Labels.DefaultDataLabelFormat.ShowValue = true;
series.Labels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = false;
series.Labels.DefaultDataLabelFormat.NumberFormat = "0.0%";
```

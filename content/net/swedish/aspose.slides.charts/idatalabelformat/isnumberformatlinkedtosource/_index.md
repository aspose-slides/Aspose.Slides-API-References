## IDataLabelFormat.IsNumberFormatLinkedToSource egenskap

Läs/skriv Boolean.

```csharp
public bool IsNumberFormatLinkedToSource { get; set; }
```

### Anmärkningar

Om föräldern till detta DataLabelFormat-objekt är en DataLabelCollection-samling av datamärken så hämtar eller anger denna egenskap standardvärdet för egenskapen IsNumberFormatLinkedToSource för de nya datamärkena i DataLabelCollection-samlingen. Att ange denna egenskap med ett värde sätter också detta värde på egenskapen IsNumberFormatLinkedToSource för alla datamärken i DataLabelCollection-samlingen (t.ex. "DataLabels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = val;" vilket gör att alla DataLabels[i].IsNumberFormatLinkedToSource blir lika med val).

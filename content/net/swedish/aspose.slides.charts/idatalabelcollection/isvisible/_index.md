## IDataLabelCollection.IsVisible egenskap

False betyder att datamärket inte är synligt som standard (och så är alla Show*-flaggor (ShowValue, ...) för egenskapen DefaultDataLabelFormat falska). Skrivskyddad Boolean.

```csharp
public bool IsVisible { get; }
```

### Anmärkningar

Om datamärket är synligt som standard kan du göra det dolt som standard med Hide()-metoden. Men om datamärket inte är synligt som standard (IsVisible är false) kan du göra datamärket "synligt som standard" genom att ställa in Show*-flaggor (ShowValue, ...) för egenskapen DefaultDataLabelFormat till true-tillstånd.

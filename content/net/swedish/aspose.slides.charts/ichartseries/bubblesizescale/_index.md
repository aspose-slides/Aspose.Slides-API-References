## IChartSeries.BubbleSizeScale egenskap

Anger skalfaktorn för bubbeldiagrammet (kan vara mellan 0 och 300 procent av standardstorleken). Detta är egenskapen inte bara för denna serie utan för alla serier i den överordnade seriegruppen – detta är en projektion av den motsvarande gruppens egenskap. Därför är denna egenskap skrivskyddad. Använd ParentSeriesGroup egenskap för att komma åt den överordnade seriegruppen. Använd ParentSeriesGroup.BubbleSizeScale läs/skriv egenskap för att ändra värdet.

```csharp
public int BubbleSizeScale { get; }
```

Kopierar elementen i ICollection till en Array, med början på ett specifikt Array-index.

```csharp
public void CopyTo(IChartDataPoint[] array, int arrayIndex)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| array | IChartDataPoint[] | Den endimensionella Array som är destinationen för de element som kopierats från ICollection. Arrayen måste ha nollbaserad indexering. |

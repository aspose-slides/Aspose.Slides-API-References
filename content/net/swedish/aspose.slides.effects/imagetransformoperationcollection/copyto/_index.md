## ImageTransformOperationCollection.CopyTo metod

Kopierar elementen i ICollection till en Array, med start vid ett visst Array-index.

```csharp
public void CopyTo(IImageTransformOperation[] array, int arrayIndex)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| array | IImageTransformOperation[] | Den endimensionella Array som är målet för de element som kopieras från ICollection. Arrayen måste ha nollbaserad indexering. |
| arrayIndex | Int32 | Det nollbaserade indexet i *array* där kopieringen påbörjas. |

### Undantag

| undantag | villkor |
| --- | --- |

| ArgumentOutOfRangeException | *arrayIndex* är mindre än 0. |

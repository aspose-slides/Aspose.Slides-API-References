## SlideUtil.FindAndReplaceText-metod

Hittar och ersätter text i presentationen med givet format

```csharp
public static void FindAndReplaceText(IPresentation presentation, bool withMasters, string find, 
    string replace, PortionFormat format = null)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| presentation | IPresentation | Skannad presentation. |
| withMasters | Boolean | Bestämmer om masterbilder ska skannas. |
| find | String | Strängvärde att hitta. |
| replace | String | Strängvärde att ersätta. |
| format | PortionFormat | Format för att ersätta textdel. Om null används formatet för det första tecknet i den hittade strängen. |

### Exempel

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PortionFormat format = new PortionFormat
    {
        FontHeight = 24f,
        FontItalic = NullableBool.True,
        FillFormat =
        {
            FillType = FillType.Solid,
            SolidFillColor =
            {
                Color = Color.Red
            }
        }
    };
    Aspose.Slides.Util.SlideUtil.FindAndReplaceText(pres, true, "[this block] ", "my text ", format);
    pres.Save("replaced", SaveFormat.Pptx);
}
```

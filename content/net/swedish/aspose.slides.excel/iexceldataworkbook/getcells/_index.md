## IExcelDataWorkbook.GetCells metod

Hämtar en samling celler från arbetsboken som matchar den angivna formeln.

```csharp
public ReadOnlyCollection<IExcelDataCell> GetCells(string formula, bool skipHiddenCells)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| formula | String | En formel eller intervalsexpression (t.ex. "Sheet1!A1:B3") som används för att identifiera målcellern. |
| skipHiddenCells | Boolean | Om `true` kommer dolda celler (t.ex. i dolda rader eller kolumner) att uteslutas från resultatet. |

### Returvärde

En skrivskyddad lista med celler som matchar den angivna formeln.

### Exempel

Exempel:

```csharp
[C#]
ExcelDataWorkbook wb = new ExcelDataWorkbook(testFile);
IReadOnlyList<IExcelDataCell> cells = wb.GetCells("Sheet1!A2:A6", false);
Console.WriteLine(cells.Count); //Utdata: 5
```

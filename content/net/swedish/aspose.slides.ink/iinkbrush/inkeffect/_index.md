## IInkBrush.InkEffect egenskap

Hämtar bläckeffekttypen (t.ex. Galaxy, Gold, Silver) som definierar den visuella stilen för bläckstrecket. Värdet analyseras från penselns egendom "inkEffects". Om ingen igenkänd effekt anges returneras NotDefined.

```csharp
public InkEffectType InkEffect { get; }
```

### Exempel

Exempel:

```csharp
[C#]
using (var pres = new Presentation("Presentation.pptx"))
{
    Ink ink = pres.Slides[0].Shapes[0] as Ink;
    IInkBrush brush = ink.Traces[0].Brush;
    Console.WriteLine("InkEffects = {0}", brush.InkEffect);
}
```

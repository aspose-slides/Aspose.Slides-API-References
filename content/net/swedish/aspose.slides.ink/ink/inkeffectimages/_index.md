## Ink.InkEffectImages egenskap

Hämtar samlingen av anpassade bilder som används för att simulera visuella effekter för bläckborstar. Dessa bilder används vid rendering av bläck med specifika [`InkEffectType`](../../inkeffecttype)-värden, såsom Galaxy, Rainbow osv. Genom att tillhandahålla egna bilder kan du kontrollera hur varje bläck-effekt visas.

```csharp
public static IDictionary<InkEffectType, IImage> InkEffectImages { get; }
```

### Anmärkningar

Denna egenskap möjliggör att ersätta standard-bläck-effekttexturer med användardefinierade, vilket är särskilt användbart när standardtillgångar är licensbegränsade eller otillgängliga vid körning. Varje post i ordboken måste associera ett [`InkEffectType`](../../inkeffecttype)-värde med ett motsvarande [`IImage`](../../../aspose.slides/iimage)-objekt (t.ex. Bitmap eller ett Aspose-bildgränssnitt).

### Exempel

```csharp
[IImage image = Images.FromFile("image.png");
Ink.InkEffectImages.Add(InkEffectType.Galaxy, image);
```

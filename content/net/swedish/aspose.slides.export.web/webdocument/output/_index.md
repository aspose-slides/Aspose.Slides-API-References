Returnerar en samling av output-element i dokumentet. Skrivskyddad `Output`.

### Exempel

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    var options = new WebDocumentOptions
    {
        TemplateEngine = new RazorTemplateEngine(),
        OutputSaver = new FileOutputSaver(),
        EmbedImages = false
    };
    
    WebDocument document = new WebDocument(options);

    // lägg till "index.html" i output-filer, använd "index"-mallen för att generera den och pres-variabeln som modell
    document.Output.Add("index.html", "index", pres);

    // ... ställ in andra alternativ för dokumentet och spara sedan dokumentet
    document.Save();
}
```

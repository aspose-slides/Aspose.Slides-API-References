Returnerar global lagring av dokumentet. Skrivskyddad [`Storage`](../../storage).

### Exempel

Med hjälp av denna `Global` egenskap (implementation av [`Storage`](../../storage) gränssnitt) kan en egenskap placeras för att använda den senare i mallen:

```csharp
[C#]
var options = new WebDocumentOptions
{
    TemplateEngine = new RazorTemplateEngine(),
    OutputSaver = new FileOutputSaver(),
    EmbedImages = false
};

WebDocument document = new WebDocument(options);

// lägg "slideMargin"-egenskapen för att använda den från mallar
document.Global.Put("slideMargin", 10);

// ... ställ in andra alternativ för dokumentet och sedan spara dokumentet
document.Save();
```

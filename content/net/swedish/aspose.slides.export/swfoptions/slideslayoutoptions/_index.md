## SwfOptions.SlidesLayoutOptions egenskap

Hämtar eller anger läget i vilket bilder placeras på sidan vid export av en presentation [`ISlidesLayoutOptions`](../../islideslayoutoptions). Denna egenskap stöder inte tilldelning av objekt av typen [`HandoutLayoutingOptions`](../../handoutlayoutingoptions)

```csharp
public ISlidesLayoutOptions SlidesLayoutOptions { get; set; }
```

### Exempel

Exempel:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    SwfOptions options = new SwfOptions
    {
        SlidesLayoutOptions = new NotesCommentsLayoutingOptions
        {
            CommentsPosition = CommentsPositions.Right
        }
    };
    
    pres.Save("pres.swf", SaveFormat.Swf, options);
}
```

## IGifOptions.DefaultDelay egenskap

Hämtar eller anger standardfördröjningstid [ms]. Detta värde används om [`AdvanceAfterTime`](../../../aspose.slides/islideshowtransition/advanceaftertime) inte är angivet. Standardvärdet är 1000.

```csharp
public int DefaultDelay { get; set; }
```

### Exempel

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    pres.Save("pres.gif", SaveFormat.Gif, new GifOptions { DefaultDelay = 2000 });
}
```

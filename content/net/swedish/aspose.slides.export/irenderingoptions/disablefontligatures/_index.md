## IRenderingOptions.DisableFontLigatures egenskap

Hämtar eller anger ett värde som visar om text renderas utan att använda ligaturer. När den är satt till `true` kommer ligaturer att inaktiveras i den renderade utdata. Som standard är denna egenskap satt till `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Exempel

Exempel:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    RenderingOptions options = new RenderingOptions
    {
        DisableFontLigatures = true // Inaktivera ligaturer i textrendering
    };
    
    Bitmap[] renderedSlides = pres.GetImage(options);
    for (var index = 0; index < renderedSlides.Length; index++)
    {
        var slideImage = renderedSlides[index];
        slideImage.Save($"slide-{index}.png");
    }
}
```

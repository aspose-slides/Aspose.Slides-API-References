## InkOptions.HideInk egenskap

Visar eller döljer Ink element i exporterat dokument.

```csharp
public bool HideInk { get; set; }
```

### Anmärkningar

Standardvärdet är false.

### Exempel

Nästa exempel visar hur man döljer Ink element i exporterat PDF-dokument:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.HideInk = true;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

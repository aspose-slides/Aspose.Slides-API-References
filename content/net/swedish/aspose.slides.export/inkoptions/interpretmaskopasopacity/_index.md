## InkOptions.InterpretMaskOpAsOpacity egenskap

Använder ROP operation eller Opacity för att rendera penseln.

```csharp
public bool InterpretMaskOpAsOpacity { get; set; }
```

### Anmärkningar

Standardvärdet är true.

### Exempel

Nästa exempel visar hur man ställer in med ROP för att exportera Ink-element:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.InterpretMaskOpAsOpacity = false;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

## IHtmlOptions.JpegQuality egenskap

Returnerar eller sätter ett värde som bestämmer kvaliteten på JPEG-bilder i PDF-dokumentet. Läs/skriv Byte.

```csharp
public byte JpegQuality { get; set; }
```

### Anmärkningar

Har endast effekt när ett dokument innehåller JPEG-bilder.

Använd denna egenskap för att hämta eller ange kvaliteten på bilderna i ett dokument när du sparar i PDF-format. Värdet kan variera från 0 till 100 där 0 betyder sämst kvalitet men maximal kompression och 100 betyder bästa kvalitet men minimal kompression.

Standardvärdet är **95**.

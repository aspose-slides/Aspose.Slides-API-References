## SVGOptions.DisableFontLigatures egenskap

Hämtar eller anger ett värde som visar om text renderas utan att använda ligaturer. När den är satt till `true` kommer ligaturer att inaktiveras i den renderade utskriften. Som standard är denna egenskap satt till `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Exempel

Exempel:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    SVGOptions options = new SVFOptions
    {
        DisableFontLigatures = true // Inaktivera ligaturer i textrendering
    };
    
    using (FileStream fileStream = new FileStream("slide-0.svg", FileMode.Create, FileAccess.Write))
    {
        pres.Slides[0].WriteAsSvg(fileStream);   
    }
}
```

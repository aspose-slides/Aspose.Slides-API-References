## IPdfOptions.RasterizeUnsupportedFontStyles egenskap

Indikerar om text ska rasteriseras som en bitmap och sparas till PDF när typsnittet inte stöder fet stil. Detta tillvägagångssätt kan förbättra kvaliteten på text i den resulterande PDF:en för vissa typsnitt. Läs/skriv Boolean.

```csharp
public bool RasterizeUnsupportedFontStyles { get; set; }
```

### Anmärkningar

Standard är **false**.

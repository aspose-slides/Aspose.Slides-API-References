## Html5Options.DisableFontLigatures egenskap

Hämtar eller anger ett värde som indikerar om text renderas utan att använda ligaturer. När den är satt till `true` kommer ligaturer att inaktiveras i den renderade utmatningen. Som standard är denna egenskap satt till `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Exempel

Exempel:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    Html5Options options = new Html5Options
    {
        DisableFontLigatures = true // Inaktivera ligaturer i textrendering
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html5, options);
}
```

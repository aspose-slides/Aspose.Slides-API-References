## PptxOptions.CompressionLevel egenskap

Anger komprimeringsnivån som används när presentationsdokumentet sparas. Standardvärdet är Level6.

```csharp
public CompressionLevel CompressionLevel { get; set; }
```

### Anmärkningar

Högre komprimeringsnivåer ger mindre filer men kräver mer bearbetningstid. Den faktiska komprimeringsgraden beror på innehållet i presentationen.

### Exempel

Exempel:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("demo-level8.pptx", SaveFormat.Pptx, new PptxOptions()
    {
        CompressionLevel = CompressionLevel.Level8
    });
}
```

## CompressionLevel uppräkning

Anger ZIP-komprimeringsnivåer för OpenXML-filen. Högre nivåer ger bättre kompression på bekostnad av långsammare bearbetning.

```csharp
public enum CompressionLevel
```

### Värden

| Namn | Värde | Beskrivning |
| --- | --- | --- |
| None | `0` | Ingen komprimering tillämpas. Filer lagras i befintligt format. |
| Level1 | `1` | Snabbast möjliga komprimering med den lägsta komprimeringsgraden. |
| Level2 | `2` | Snabbare komprimering med något bättre komprimeringsgrad än Level1. |
| Level3 | `3` | Ger bättre komprimering än Level2 med måttlig påverkan på prestanda. |
| Level4 | `4` | Ger bättre komprimering än Level3. |
| Level5 | `5` | Ger förbättrad komprimering jämfört med Level4 med extra bearbetningstid. |
| Level6 | `6` | Standardkomprimering som ger en bra balans mellan komprimeringshastighet och filstorlek. Standardkomprimeringsnivån. |
| Level7 | `7` | Ger högre komprimering än Level6 med långsammare bearbetning. |
| Level8 | `8` | Ger högre komprimering än Level7. |
| Level9 | `9` | Maximal komprimering. Ger den minsta filstorleken med den långsammaste bearbetningshastigheten. |

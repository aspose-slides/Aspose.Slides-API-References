## PdfOptions.EmbedTrueTypeFontsForASCII egenskap

Bestämmer om Aspose.Slides ska bädda in vanliga teckensnitt för ASCII-text (kodintervall 33..127). Teckensnitt för teckenkoder större än 127 bäddas alltid in. Listan med vanliga teckensnitt innehåller PDF:s grundläggande 14-teckensnitt samt ytterligare användarspecificerade teckensnitt. Läs/skriv Boolean.

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### Anmärkningar

Standard är **true**.

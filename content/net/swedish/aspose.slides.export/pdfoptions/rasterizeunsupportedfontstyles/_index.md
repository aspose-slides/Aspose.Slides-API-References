## PdfOptions.RasterizeUnsupportedFontStyles egenskap

Indikerar om text ska rasteriseras som en bitmap och sparas till PDF när teckensnittet inte stödjer fet stil. Detta tillvägagångssätt kan förbättra kvaliteten på text i den resulterande PDF-filen för vissa teckensnitt. Läs/skriv Boolean.

```csharp
public bool RasterizeUnsupportedFontStyles { get; set; }
```

### Anmärkningar

Standard är **false**.

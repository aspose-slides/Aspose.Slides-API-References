## PdfOptions.IncludeOleData egenskap

True för att konvertera all OLE-data från presentationen till inbäddade filer i den resulterande PDF:en. Läs/skriv Boolean.

```csharp
public bool IncludeOleData { get; set; }
```

### Anmärkningar

Standardvärdet är **false**.

### Exempel

Exempel:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions options = new PdfOptions { IncludeOleData = true };
    pres.Save("pres.pdf", SaveFormat.Pdf, options);
}
```

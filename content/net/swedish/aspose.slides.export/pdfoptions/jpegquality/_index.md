## PdfOptions.JpegQuality egenskap

Returnerar eller anger ett värde som bestämmer kvaliteten på JPEG-bilderna i PDF-dokumentet. Läs/skriv Byte.

```csharp
public byte JpegQuality { get; set; }
```

### Anmärkningar

Har bara effekt när ett dokument innehåller JPEG-bilder.

Använd den här egenskapen för att hämta eller ange kvaliteten på bilderna i ett dokument när du sparar i PDF-format. Värdet kan variera från 0 till 100 där 0 betyder sämst kvalitet men maximal komprimering och 100 betyder bästa kvalitet men minimal komprimering.

Standardvärdet är **100**.

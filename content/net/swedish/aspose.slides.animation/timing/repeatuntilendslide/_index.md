## Timing.RepeatUntilEndSlide egenskap

Detta attribut specificerar om effekten kommer att upprepas tills slutet av bilden. Läs/skriv Boolean.

```csharp
public bool RepeatUntilEndSlide { get; set; }
```

### Exempel

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Hämta effektsekvensen för den första bilden
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Hämta den första effekten i huvudsekvensen.
    IEffect effect = effectsSequence[0];

    // Ändra effektens Timing/Repeat till "Till slutet av bilden"
    effect.Timing.RepeatUntilEndSlide = true;
}
```

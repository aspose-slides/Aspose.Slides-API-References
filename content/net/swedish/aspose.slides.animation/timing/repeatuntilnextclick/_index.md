## Timing.RepeatUntilNextClick egenskap

Det här attributet anger om effekten ska upprepas tills nästa klick. Läs/skriv Boolean.

```csharp
public bool RepeatUntilNextClick { get; set; }
```

### Exempel

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Hämta effektsekvensen för den första bilden
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Hämta den första effekten i huvudsekvensen.
    IEffect effect = effectsSequence[0];

    // Ändra effektens Timing/Repeat till "Until Next Click"
    effect.Timing.RepeatUntilNextClick = true;
}
```

## IEffect.AnimateTextType egenskap

Definierar en animerad texttyp för effekt. Formens text kan animeras bokstav för bokstav, ord för ord eller hela på en gång. Läs/skriv `AnimateTextType`.

```csharp
public AnimateTextType AnimateTextType { get; set; }
```

### Exempel

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Hämta den första effekten på den första bilden.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Ändra effektens AnimateTextType till "By letter"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByLetter;
}
```

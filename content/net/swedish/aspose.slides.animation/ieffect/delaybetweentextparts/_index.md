## IEffect.DelayBetweenTextParts egenskap

Definierar en fördröjning mellan animerade textdelar (ord eller bokstäver). Ett positivt värde anger procentsatsen av effektens varaktighet. Ett negativt värde anger fördröjningen i sekunder. Läs/skriv Single.

```csharp
public float DelayBetweenTextParts { get; set; }
```

### Exempel

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Hämta den första effekten på den första bilden.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Ändra effekttypen AnimateTextType till "By word"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByWord;
    
    // Ställ in fördröjningen mellan animerade textdelar till 20% av effektens varaktighet.
    firstSlideEffect.DelayBetweenTextParts = 20f;
}
```

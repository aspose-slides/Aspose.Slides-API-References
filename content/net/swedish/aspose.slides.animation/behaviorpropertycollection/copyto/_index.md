## BehaviorPropertyCollection.CopyTo metod

Kopierar elementen i ICollection till en Array, med start vid ett specifikt Array-index.

```csharp
public void CopyTo(IBehaviorProperty[] array, int arrayIndex)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| array | IBehaviorProperty[] | Den enkeldimensionella Array som är destinationen för elementen som kopierats från ICollection. Array måste ha nollbaserad indexering. |
| arrayIndex | Int32 | Det nollbaserade indexet i *array* där kopieringen börjar. |

### Undantag

| undantag | villkor |
| --- | --- |
| ArgumentNullException | *array* är null. |
| ArgumentOutOfRangeException | *arrayIndex* är mindre än 0. |
| ArgumentException | Antalet element i källan ICollection är större än det tillgängliga utrymmet från *arrayIndex* till slutet av destinationen *array*. |

## Effect.StopPreviousSound egenskap

Detta attribut specificerar om animationseffekten stoppar det föregående ljudet. Läs/skriv Boolean.

```csharp
public bool StopPreviousSound { get; set; }
```

### Exempel

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Hämta den första effekten på den första bilden.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Hämta den första effekten på den andra bilden.
    IEffect secondSlideEffect = presentation.Slides[1].Timeline.MainSequence[0];
       
    if (firstSlideEffect.Sound != null)
    {
        // Ändra den andra effektens Enhancements/Sound till "Stop Previous Sound"
        secondSlideEffect.StopPreviousSound = true;
    }
}
```

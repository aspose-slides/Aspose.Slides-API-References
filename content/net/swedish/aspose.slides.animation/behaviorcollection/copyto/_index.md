## BehaviorCollection.CopyTo metod

Kopierar elementen i ICollection till en Array, med början vid ett bestämt Array-index.

```csharp
public void CopyTo(IBehavior[] array, int arrayIndex)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| array | IBehavior[] | Den endimensionella Array som är mål för elementen som kopieras från ICollection. Array måste ha nollbaserad indexering. |
| arrayIndex | Int32 | Det nollbaserade indexet i *array* där kopieringen börjar. |

### Undantag

| undantag | villkor |
| --- | --- |
| ArgumentNullException | *array* är null. |
| ArgumentOutOfRangeException | *arrayIndex* är mindre än 0. |
| ArgumentException | Antalet element i källan ICollection är större än det tillgängliga utrymmet från *arrayIndex* till slutet av destinations-Arrayen. |

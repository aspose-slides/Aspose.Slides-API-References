## IOverrideThemeManager.IsOverrideThemeEnabled ویژگی

تعیین می‌کند که آیا OverrideTheme تم مؤثر به ارث‌برده را بازنویسی می‌کند یا خیر. برای فعال‌سازی OverrideTheme جهت بازنویسی از متدهای OverrideTheme.Init*() استفاده کنید. برای غیرفعال‌سازی OverrideTheme از بازنویسی از متد OverrideTheme.Clear() استفاده کنید. Boolean فقط‌خواندنی.

public bool IsOverrideThemeEnabled { get; }

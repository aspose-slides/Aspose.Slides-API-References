## RenderingOptions.DisableFontLigatures خصوصیت

مقداری را که نشان می‌دهد آیا متن بدون استفاده از لیگچرها رندر می‌شود دریافت یا تنظیم می‌کند. وقتی به `true` تنظیم شود، لیگچرها در خروجی رندر غیرفعال می‌شوند. به طور پیش‌فرض، این خصوصیت روی `false` تنظیم شده است.

```csharp
public bool DisableFontLigatures { get; set; }
```

### مثال‌ها

مثال:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    RenderingOptions options = new RenderingOptions
    {
        DisableFontLigatures = true // غیرفعال‌سازی لیگچرها در رندر متن
    };
    
    Bitmap[] renderedSlides = pres.GetImage(options);
    for (var index = 0; index < renderedSlides.Length; index++)
    {
        var slideImage = renderedSlides[index];
        slideImage.Save($"slide-{index}.png");
    }
}
```

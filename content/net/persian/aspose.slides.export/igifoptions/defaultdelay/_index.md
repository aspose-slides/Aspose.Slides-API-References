## IGifOptions.DefaultDelay ویژگی

دریافت یا تنظیم زمان تأخیر پیش‌فرض [ms]. این مقدار زمانی استفاده می‌شود که [`AdvanceAfterTime`](../../../aspose.slides/islideshowtransition/advanceaftertime) تنظیم نشده باشد. مقدار پیش‌فرض 1000 است.

```csharp
public int DefaultDelay { get; set; }
```

### مثال‌ها

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
	pres.Save("pres.gif", SaveFormat.Gif, new GifOptions { DefaultDelay = 2000 });
}
```

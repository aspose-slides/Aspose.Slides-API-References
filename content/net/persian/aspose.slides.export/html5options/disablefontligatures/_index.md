## Html5Options.DisableFontLigatures ویژگی

دریافت یا تنظیم مقداری که نشان می‌دهد آیا متن بدون استفاده از لیگچرها رندر می‌شود. وقتی به `true` تنظیم شود، لیگچرها در خروجی رندر شده غیرفعال می‌شوند. به طور پیش‌فرض، این ویژگی به `false` تنظیم شده است.

```csharp
public bool DisableFontLigatures { get; set; }
```

### مثال‌ها

مثال:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    Html5Options options = new Html5Options
    {
        DisableFontLigatures = true // غیرفعال‌سازی لیگچرها در رندر متن
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html5, options);
}
```

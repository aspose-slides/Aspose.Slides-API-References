## SwfOptions.SlidesLayoutOptions خصوصیت

حالت قرارگیری اسلایدها بر روی صفحه هنگام خروجی گرفتن از ارائه را دریافت یا تنظیم می‌کند [`ISlidesLayoutOptions`](../../islideslayoutoptions). این ویژگی از اختصاص اشیاء از نوع [`HandoutLayoutingOptions`](../../handoutlayoutingoptions) پشتیبانی نمی‌کند

```csharp
public ISlidesLayoutOptions SlidesLayoutOptions { get; set; }
```

### مثال‌ها

مثال:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    SwfOptions options = new SwfOptions
    {
        SlidesLayoutOptions = new NotesCommentsLayoutingOptions
        {
            CommentsPosition = CommentsPositions.Right
        }
    };
    
    pres.Save("pres.swf", SaveFormat.Swf, options);
}
```

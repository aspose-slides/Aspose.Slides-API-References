## IInkOptions.InterpretMaskOpAsOpacity ویژگی

از عملگر ROP یا Opacity برای رندر کردن قلم‌مو استفاده می‌کند.

```csharp
public bool InterpretMaskOpAsOpacity { get; set; }
```

### توضیحات

مقدار پیش‌فرض true است.

### مثال‌ها

مثال بعدی نشان می‌دهد چگونه با استفاده از ROP برای خروجی گرفتن عناصر Ink تنظیم می‌شود:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.InterpretMaskOpAsOpacity = false;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

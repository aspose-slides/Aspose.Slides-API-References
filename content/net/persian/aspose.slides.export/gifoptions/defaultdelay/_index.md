## GifOptions.DefaultDelay ویژگی

مقدار زمان تأخیر پیش‌فرض [ms] را دریافت یا تنظیم می‌کند. اگر [`AdvanceAfterTime`](../../../aspose.slides/islideshowtransition/advanceaftertime) تنظیم نشده باشد، از این مقدار استفاده خواهد شد. مقدار پیش‌فرض 1000 است.

```csharp
public int DefaultDelay { get; set; }
```

### مثال‌ها

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
	pres.Save("pres.gif", SaveFormat.Gif, new GifOptions { DefaultDelay = 2000 });
}
```

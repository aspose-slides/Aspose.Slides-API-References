## IHtmlOptions.JpegQuality ویژگی

مقداری را برمی‌گرداند یا تنظیم می‌کند که کیفیت تصاویر JPEG داخل سند PDF را تعیین می‌کند. Read/write Byte.

```csharp
public byte JpegQuality { get; set; }
```

### توضیحات

فقط زمانی اثر دارد که سند شامل تصاویر JPEG باشد.

از این ویژگی برای دریافت یا تنظیم کیفیت تصویرهای داخل یک سند هنگام ذخیره‌سازی در قالب PDF استفاده کنید. مقدار می‌تواند از 0 تا 100 متغیر باشد که 0 به معنی پایین‌ترین کیفیت ولی بیشترین فشار و 100 به معنی بالاترین کیفیت ولی کمترین فشار است.

مقدار پیش‌فرض **95** است.

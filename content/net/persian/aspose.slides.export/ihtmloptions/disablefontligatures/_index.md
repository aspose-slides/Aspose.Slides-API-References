## IHtmlOptions.DisableFontLigatures ویژگی

مقداری را دریافت یا تنظیم می‌کند که نشان می‌دهد آیا متن بدون استفاده از لیگاتورها رندر می‌شود یا نه. هنگامی که به `true` تنظیم شود، لیگاتورها در خروجی رندر شده غیرفعال خواهند شد. به‌طور پیش‌فرض، این ویژگی به `false` تنظیم شده است.

```csharp
public bool DisableFontLigatures { get; set; }
```

### مثال‌ها

مثال:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    HtmlOptions options = new HtmlOptions
    {
        DisableFontLigatures = true // غیرفعال کردن لیگاتورها در رندر متن
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html, options);
}
```

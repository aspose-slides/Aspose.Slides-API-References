## CompressionLevel enumeration

سطوح فشرده‌سازی ZIP را برای فایل OpenXML مشخص می‌کند. سطوح بالاتر فشرده‌سازی بهتری را با هزینه پردازش کندتر فراهم می‌کنند.

```csharp
public enum CompressionLevel
```

### مقادیر

| Name | Value | Description |
| --- | --- | --- |
| None | `0` | هیچ فشرده‌سازی‌ای اعمال نمی‌شود. فایل‌ها همان‌گونه ذخیره می‌شوند. |
| Level1 | `1` | سریع‌ترین فشرده‌سازی با پایین‌ترین نسبت فشرده‌سازی. |
| Level2 | `2` | فشرده‌سازی سریع‌تر با نسبت فشرده‌سازی کمی بهتر نسبت به Level1. |
| Level3 | `3` | فشرده‌سازی بهتری نسبت به Level2 ارائه می‌دهد با تأثیر متوسط بر عملکرد. |
| Level4 | `4` | فشرده‌سازی بهتری نسبت به Level3 ارائه می‌دهد. |
| Level5 | `5` | فشرده‌سازی بهبود یافته نسبت به Level4 با زمان پردازش اضافی ارائه می‌کند. |
| Level6 | `6` | فشرده‌سازی استاندارد که تعادل خوبی بین سرعت فشرده‌سازی و حجم فایل فراهم می‌کند. سطح فشرده‌سازی پیش‌فرض. |
| Level7 | `7` | فشرده‌سازی بالاتری نسبت به Level6 با پردازش کندتر ارائه می‌دهد. |
| Level8 | `8` | فشرده‌سازی بالاتری نسبت به Level7 ارائه می‌دهد. |
| Level9 | `9` | حداکثر فشرده‌سازی. کوچک‌ترین حجم فایل را با کندترین سرعت پردازش تولید می‌کند. |

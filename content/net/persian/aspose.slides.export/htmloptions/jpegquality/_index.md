## HtmlOptions.JpegQuality ویژگی

مقداری را که کیفیت تصاویر JPEG داخل سند PDF را تعیین می‌کند، برمی‌گرداند یا تنظیم می‌کند. خواندنی/نوشتنی Byte.

```csharp
public byte JpegQuality { get; set; }
```

### ملاحظات

فقط زمانی که سند شامل تصاویر JPEG باشد، اثر می‌گذارد.

از این ویژگی برای دریافت یا تنظیم کیفیت تصاویر داخل سند هنگام ذخیره‌سازی در قالب PDF استفاده کنید. مقدار می‌تواند از 0 تا 100 متغیر باشد که در آن 0 به معنی بدترین کیفیت اما حداکثر فشرده‌سازی و 100 به معنی بهترین کیفیت اما حداقل فشرده‌سازی است.

مقدار پیش‌فرض **95** است.

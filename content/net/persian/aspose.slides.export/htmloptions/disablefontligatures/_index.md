## HtmlOptions.DisableFontLigatures ویژگی

دریافت یا تنظیم مقدار نشان‌دهنده این که متن بدون استفاده از لیگچرها رندر شود. هنگامی که به `true` تنظیم شود، لیگچرها در خروجی رندر غیرفعال خواهند شد. به طور پیش‌فرض، این ویژگی به `false` تنظیم شده است.

```csharp
public bool DisableFontLigatures { get; set; }
```

### مثال‌ها

مثال:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    HtmlOptions options = new HtmlOptions
    {
        DisableFontLigatures = true // غیرفعال کردن لیگچرها در رندر متن
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html, options);
}
```

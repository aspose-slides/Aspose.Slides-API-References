## شمارش Zip64Mode

Specifies when to use ZIP64 format extensions for OpenXML file.

```csharp
public enum Zip64Mode
```

### مقادیر

| نام | مقدار | توضیحات |
| --- | --- | --- |
| Never | `0` | از افزونه‌های فرمت ZIP64 استفاده نمی‌شود. |
| IfNecessary | `1` | در صورت لزوم از افزونه‌های فرمت ZIP64 استفاده کنید. |
| Always | `2` | همیشه از افزونه‌های فرمت ZIP64 استفاده کنید. |

### توضیحات

فایل OpenXML یک آرشیو ZIP است که محدودیت ۴ گیگابایت (۲^۳۲ بایت) برای اندازهٔ فشرده‌نشدهٔ یک فایل، اندازهٔ فشردهٔ یک فایل و اندازهٔ کل آرشیو دارد، همچنین محدودیت ۶۵٬۵۳۵ (۲^۱۶-۱) فایل در آرشیو. افزونه‌های فرمت ZIP64 محدودیت‌ها را به ۲^۶۴ افزایش می‌دهند.

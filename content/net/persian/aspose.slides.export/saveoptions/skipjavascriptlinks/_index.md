## SaveOptions.SkipJavaScriptLinks ویژگی

مشخص می‌کند که آیا هنگام ذخیره‌سازی ارائه، پیوندهای فراخوانی JavaScript نادیده گرفته شوند یا نه. خواندنی/نوشتنی Boolean. مقدار پیش‌فرض **false** است.

```csharp
public bool SkipJavaScriptLinks { get; set; }
```

### Remarks
هنگامی که این ویژگی روی **true** تنظیم شود، پیوندهای فراخوانی JavaScript هنگام ذخیره‌سازی نادیده گرفته می‌شوند.

هنگامی که این ویژگی روی **false** تنظیم شود، تمام پیوندها ذخیره خواهند شد.

### مثال‌ها
مثال:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("result_without_JavaScript_links.html", SaveFormat.Html, new HtmlOptions()
    {
        SkipJavaScriptLinks = true
    });
}
```

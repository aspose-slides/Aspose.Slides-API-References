## PdfOptions.BestImagesCompressionRatio ویژگی

نشان می‌دهد آیا فشرده‌سازی مؤثرترین (به جای فشرده‌سازی پیش‌فرض) برای هر تصویر باید به‌صورت خودکار انتخاب شود یا نه. اگر به Boolean.true تنظیم شود، برای هر تصویر در ارائه مناسب‌ترین الگوریتم فشرده‌سازی انتخاب می‌شود که منجر به کوچک‌تر شدن حجم سند PDF خروجی می‌گردد. انتخاب نسبت فشرده‌سازی بهینه برای تصاویر هزینه محاسباتی بالایی دارد و مقدار بیشتری RAM مصرف می‌کند، و این گزینه به طور پیش‌فرض Boolean.false است.

```csharp
public bool BestImagesCompressionRatio { get; set; }
```

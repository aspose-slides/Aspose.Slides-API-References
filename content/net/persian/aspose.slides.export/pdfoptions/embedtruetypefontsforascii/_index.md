## PdfOptions.EmbedTrueTypeFontsForASCII ویژگی

تعیین می‌کند که آیا Aspose.Slides فونت‌های رایج را برای متن ASCII (بازه کد 33..127) جاسازی می‌کند یا نه. فونت‌ها برای کدهای کاراکتری بزرگتر از 127 همیشه جاسازی می‌شوند. فهرست فونت‌های رایج شامل 14 فونت پایه PDF و فونت‌های اضافی مشخص‌شده توسط کاربر است. Boolean قابل خواندن/نوشتن.

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### توضیحات

پیش‌فرض **true** است.

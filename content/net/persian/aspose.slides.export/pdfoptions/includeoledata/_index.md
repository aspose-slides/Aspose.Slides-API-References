## PdfOptions.IncludeOleData ویژگی

True برای تبدیل تمام داده‌های OLE از ارائه به فایل‌های تعبیه‌شده در PDF خروجی. خواندنی/قابل نوشتن Boolean.

```csharp
public bool IncludeOleData { get; set; }
```

### توضیحات

پیش‌فرض **false** است.

### مثال‌ها

مثال:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions options = new PdfOptions { IncludeOleData = true };
    pres.Save("pres.pdf", SaveFormat.Pdf, options);
}
```

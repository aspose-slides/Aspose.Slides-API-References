## IPptOptions.RootDirectoryClsid ویژگی

نمایانگر GUID کلاس شیء (CLSID) که در ورودی ریشه دایرکتوری ذخیره می‌شود. می‌تواند برای فعال‌سازی COM برنامه سند استفاده شود. مقدار پیش‌فرض '64818D11-4F9B-11CF-86EA-00AA00B929E8' است که با 'Microsoft Powerpoint.Slide.8' مرتبط است.

```csharp
public Guid RootDirectoryClsid { get; set; }
```

### نمونه‌ها

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    PptOptions pptOptions = new PptOptions();
    
    /// CLSID را به 'Microsoft Powerpoint.Show.8' تنظیم کنید
    pptOptions.RootDirectoryClsid = new Guid("64818D10-4F9B-11CF-86EA-00AA00B929E8");
    
    pres.Save("pres.ppt", SaveFormat.Ppt, pptOptions);
}
```

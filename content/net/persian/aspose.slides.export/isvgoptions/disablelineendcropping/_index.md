## ISVGOptions.DisableLineEndCropping خصوصیت

SVG 1.1 توانایی تعریف تو رفتگی برای نشانگرها را ندارد. Aspose.Slides SVG writing engine یک راه‌حل برای این مشکل دارد: انتهای خط را با پیکان برش می‌دهد، بنابراین خط با نشانگرها همپوشانی نمی‌کند. این گزینه این رفتار را غیرفعال می‌کند. خواندنی/نوشتنی Boolean.

public bool DisableLineEndCropping { get; set; }

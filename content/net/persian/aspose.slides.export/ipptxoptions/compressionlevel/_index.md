## IPptxOptions.CompressionLevel ویژگی

سطح فشرده‌سازی مورد استفاده هنگام ذخیره‌سازی سند ارائه را مشخص می‌کند. مقدار پیش‌فرض Level6 است.

```csharp
public CompressionLevel CompressionLevel { get; set; }
```

### توضیحات

سطوح فشرده‌سازی بالاتر فایل‌های کوچک‌تری تولید می‌کنند اما زمان پردازش بیشتری می‌طلبند. نسبت فشرده‌سازی واقعی به محتوای ارائه بستگی دارد.

### مثال‌ها

مثال:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("demo-level8.pptx", SaveFormat.Pptx, new PptxOptions()
    {
        CompressionLevel = CompressionLevel.Level8
    });
}
```

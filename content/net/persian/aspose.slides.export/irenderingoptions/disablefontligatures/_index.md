## IRenderingOptions.DisableFontLigatures ویژگی

دریافت یا تنظیم مقدار نشان‌دهنده این‌که آیا متن بدون استفاده از لیگچرها رندر می‌شود. وقتی به `true` تنظیم شود، لیگچرها در خروجی رندر شده غیرفعال می‌شوند. به طور پیش‌فرض، این ویژگی به `false` تنظیم شده است.

```csharp
public bool DisableFontLigatures { get; set; }
```

### مثال‌ها

مثال:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    RenderingOptions options = new RenderingOptions
    {
        DisableFontLigatures = true // غیرفعال کردن لیگچرها در رندر متن
    };
    
    Bitmap[] renderedSlides = pres.GetImage(options);
    for (var index = 0; index < renderedSlides.Length; index++)
    {
        var slideImage = renderedSlides[index];
        slideImage.Save($"slide-{index}.png");
    }
}
```

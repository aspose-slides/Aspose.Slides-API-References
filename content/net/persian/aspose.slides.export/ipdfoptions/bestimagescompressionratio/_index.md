## IPdfOptions.BestImagesCompressionRatio ویژگی

نشان می‌دهد که آیا فشرده‌سازی مؤثرتر (به جای پیش‌فرض) برای هر تصویر باید به‌صورت خودکار انتخاب شود. اگر به Boolean.true تنظیم شود، برای هر تصویر در ارائه مناسب‌ترین الگوریتم فشرده‌سازی انتخاب می‌شود که منجر به اندازهٔ کوچک‌تر سند PDF تولید شده می‌شود. انتخاب بهترین نسبت فشرده‌سازی تصویر هزینهٔ محاسباتی بالایی دارد و مقدار RAM بیشتری مصرف می‌کند، و به طور پیش‌فرض این گزینه Boolean.false است.

```csharp
public bool BestImagesCompressionRatio { get; set; }
```

## IPdfOptions.SaveMetafilesAsPng ویژگی

True برای تبدیل تمام متافایل‌های مورد استفاده در یک ارائه به تصاویر PNG. خواندن/نوشتن Boolean.

```csharp
public bool SaveMetafilesAsPng { get; set; }
```

### توضیحات

پیش‌فرض **true** است. سند Pdf می‌تواند شامل گرافیک‌های برداری و تصاویر پیکسلی باشد. اگر SaveMetafilesAsPng بر روی true تنظیم شود، تصویر Metafile منبع به قالب Png تبدیل شده و به صورت تصویر پیکسلی در Pdf ذخیره می‌شود. اگر SaveMetafilesAsPng بر روی false تنظیم شود، Metafile منبع به گرافیک‌های برداری Pdf تبدیل می‌شود. هر یک از این روش‌ها مزایا و معایبی دارند. به عنوان مثال، اگر Metafile به PNG تبدیل شود، ممکن است در طول مقیاس‌گذاری سند نهایی، برخی از کیفیت از دست برود. اگر Metafile به گرافیک‌های برداری Pdf تبدیل شود، ممکن است مشکلات عملکردی در ابزار مشاهده Pdf رخ دهد.

## ISaveOptions.SkipJavaScriptLinks property

مشخص می‌کند آیا هنگام ذخیره‌سازی ارائه، پیوندهای هیپرلینک با فراخوانی JavaScript نادیده گرفته شوند یا نه. Boolean قابل خواندن/نوشتن. مقدار پیش‌فرض **false**.

```csharp
public bool SkipJavaScriptLinks { get; set; }
```

### توضیحات

وقتی این ویژگی روی **true** تنظیم شود، پیوندهای هیپرلینک با فراخوانی JavaScript هنگام ذخیره‌سازی نادیده گرفته می‌شوند.

وقتی این ویژگی روی **false** تنظیم شود، تمام پیوندهای هیپرلینک ذخیره می‌شوند.

### مثال‌ها

مثال:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("result_without_JavaScript_links.html", SaveFormat.Html, new HtmlOptions()
    {
        SkipJavaScriptLinks = true
    });
}
```

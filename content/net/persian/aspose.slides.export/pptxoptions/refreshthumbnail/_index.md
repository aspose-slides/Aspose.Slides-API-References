## PptxOptions.RefreshThumbnail ویژگی

مشخص می‌کند که آیا تصویر بندانگشتی ارائه تازه شود یا نه. خواندن/نوشتن Boolean. مقدار پیش‌فرض **true**.

```csharp
public bool RefreshThumbnail { get; set; }
```

### توضیحات

هنگامی که مقدار گزینه **true** باشد، تصویر بندانگشتی جدید تولید می‌شود.

هنگامی که مقدار گزینه **false** باشد، تصویر بندانگشتی فعلی همان‌طور ذخیره می‌شود.

### مثال‌ها

مثال:

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    pres.Save("result_with_old_thumbnail.pptx", SaveFormat.Pptx, new PptxOptions()
    {
        RefreshThumbnail = false
    });
}
```

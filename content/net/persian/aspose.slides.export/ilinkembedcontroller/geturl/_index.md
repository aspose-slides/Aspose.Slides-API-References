## ILinkEmbedController.GetUrl متد

یک URL برای شیء خارجی برمی‌گرداند. این متد همیشه زمانی فراخوانی می‌شود که [`GetObjectStoringLocation`](../getobjectstoringlocation) مقدار Link را بازگرداند و ممکن است زمانی فراخوانی شود که [`GetObjectStoringLocation`](../getobjectstoringlocation) مقدار Embed را بازگرداند اما جاسازی امکان‌پذیر نیست. می‌توان چندین بار برای همان شناسهٔ شیء فراخوانی کرد.

```csharp
public string GetUrl(int id, int referrer)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| id | Int32 | شناسهٔ شیء. این شناسه برای کل عملیات ذخیره‌سازی یکتا است. |
| referrer | Int32 | شناسهٔ شیء مرجع یا 0، اگر شیء توسط سند ریشه ارجاع داده شود. ممکن است برای تولید لینک نسبی استفاده شود. |

### مقدار بازگشت

آدرس URL شیء خارجی یا null اگر این شیء باید نادیده گرفته شود.

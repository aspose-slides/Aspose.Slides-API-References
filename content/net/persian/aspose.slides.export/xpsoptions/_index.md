### مثال‌ها

مثال زیر نشان می‌دهد چگونه ارائه‌ها را با استفاده از تنظیمات پیش‌فرض به XPS تبدیل می‌کنید.

```csharp
[C#]
// یک شیء Presentation که نمایانگر یک فایل ارائه است را ایجاد می‌کند
using (Presentation pres = new Presentation("Convert_XPS.pptx"))
{
    // ذخیرهٔ ارائه به سند XPS
    pres.Save("XPS_Output_Without_XPSOption_out.xps", SaveFormat.Xps);
}
```

مثال زیر نشان می‌دهد چگونه ارائه‌ها را با استفاده از تنظیمات سفارشی به XPS تبدیل می‌کنید.

```csharp
[C#]
// یک شیء Presentation که نمایانگر یک فایل ارائه است را ایجاد می‌کند
using (Presentation pres = new Presentation("Convert_XPS_Options.pptx"))
{
    // یک شیء از کلاس TiffOptions ایجاد می‌کند
    XpsOptions options = new XpsOptions();
    // متافایل‌ها را به PNG ذخیره می‌کند
    options.SaveMetafilesAsPng = true;
    // ارائه را به سند XPS ذخیره می‌کند
    pres.Save("XPS_With_Options_out.xps", SaveFormat.Xps, options);
}
```

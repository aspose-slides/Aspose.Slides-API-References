## ISwfOptions.SlidesLayoutOptions ویژگی

دریافت یا تنظیم حالت قرار گرفتن اسلایدها بر روی صفحه هنگام خروجی‌گیری از یک ارائه [`ISlidesLayoutOptions`](../../islideslayoutoptions). این ویژگی از اختصاص اشیائی از نوع `Aspose.Slides.Export.HandoutLayoutingOptions` پشتیبانی نمی‌کند

```csharp
public ISlidesLayoutOptions SlidesLayoutOptions { get; set; }
```

### مثال‌ها

مثال:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    SwfOptions options = new SwfOptions
    {
        SlidesLayoutOptions = new NotesCommentsLayoutingOptions
        {
            CommentsPosition = CommentsPositions.Right
        }
    };
    
    pres.Save("pres.swf", SaveFormat.Swf, options);
}
```

## IHtml5Options.DisableFontLigatures ویژگی

مقداری را دریافت یا تنظیم می‌کند که نشان می‌دهد آیا متن بدون استفاده از لیگچرها رندر می‌شود یا خیر. وقتی به `true` تنظیم شود، لیگچرها در خروجی رندر غیرفعال می‌شوند. به طور پیش‌فرض، این ویژگی بر روی `false` تنظیم شده است.

```csharp
public bool DisableFontLigatures { get; set; }
```

### مثال‌ها

Example:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    Html5Options options = new Html5Options
    {
        DisableFontLigatures = true // غیرفعال کردن لیگچرها در رندر متن
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html5, options);
}
```

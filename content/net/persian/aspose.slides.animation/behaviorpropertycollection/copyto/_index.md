## BehaviorPropertyCollection.CopyTo متد

عناصر ICollection را به یک Array کپی می‌کند، که از یک ایندکس خاص در Array شروع می‌شود.

```csharp
public void CopyTo(IBehaviorProperty[] array, int arrayIndex)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| array | IBehaviorProperty[] | Array تک‌بعدی که مقصد عناصری است که از ICollection کپی می‌شوند. Array باید ایندکس صفر-پایه داشته باشد. |
| arrayIndex | Int32 | ایندکس صفر-پایه در *array* که کپی‌سازی از آن شروع می‌شود. |

## ویژگی Effect.DelayBetweenTextParts

یک تاخیر بین قسمت‌های متنی انیمیشن‌شده (کلمات یا حروف) را تعریف می‌کند. یک مقدار مثبت درصد مدت اثر را مشخص می‌کند. یک مقدار منفی تاخیر را بر حسب ثانیه مشخص می‌کند. خواندنی/نوشتنی Single.

```csharp
public float DelayBetweenTextParts { get; set; }
```

### مثال‌ها

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // دریافت اولین افکت اولین اسلاید.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // تغییر نوع متن انیمیشن افکت به "By word"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByWord;
    
    // تنظیم تاخیر بین قسمت‌های متنی انیمیشن‌شده به 20٪ از مدت افکت.
    firstSlideEffect.DelayBetweenTextParts = 20f;
}
```

## Effect.AnimateTextType property

یک نوع متن متحرک برای افکت تعریف می‌کند. متن شکل می‌تواند به صورت حرف به حرف، کلمه به کلمه یا به طور کلی همزمان انیمیت شود. خواندنی/قابل نوشتن `AnimateTextType`.

```csharp
public AnimateTextType AnimateTextType { get; set; }
```

### Examples

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // دریافت اولین افکت اسلاید اول.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // تغییر نوع متن متحرک افکت به "By letter"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByLetter;
}
```

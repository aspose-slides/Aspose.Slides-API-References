## ITiming.RepeatUntilNextClick ویژگی

این ویژگی مشخص می‌کند که آیا اثر تا کلیک بعدی تکرار می‌شود یا خیر. خواندن/نوشتن Boolean.

```csharp
public bool RepeatUntilNextClick { get; set; }
```

### مثال‌ها

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // دریافت توالی افکت‌ها برای اسلاید اول
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // دریافت اولین افکت توالی اصلی.
    IEffect effect = effectsSequence[0];

    // تغییر Timing/Repeat افکت به "تا کلیک بعدی"
    effect.Timing.RepeatUntilNextClick = true;
}
```

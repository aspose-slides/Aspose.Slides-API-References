## ITiming.RepeatUntilEndSlide ویژگی

این ویژگی مشخص می‌کند که آیا اثر تا انتهای اسلاید تکرار خواهد شد. خواندنی/نویسنی Boolean.

```csharp
public bool RepeatUntilEndSlide { get; set; }
```

### مثال‌ها

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // دریافت دنبالهٔ افکت‌ها برای اسلاید اول
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // دریافت اولین افکت از دنبالهٔ اصلی.
    IEffect effect = effectsSequence[0];

    // تغییر Timing/Repeat افکت به "تا انتهای اسلاید"
    effect.Timing.RepeatUntilEndSlide = true;
}
```

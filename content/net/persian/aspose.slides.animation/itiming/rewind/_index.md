## ITiming.Rewind ویژگی

این ویژگی مشخص می‌کند که آیا اثر پس از اتمام پخش، به حالت اولیه باز می‌گردد یا خیر. قابل خواندن/نوشتن Boolean.

```csharp
public bool Rewind { get; set; }
```

### مثال‌ها

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // دریافت توالی افکت‌ها برای اولین اسلاید
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // دریافت اولین افکت از توالی اصلی.
    IEffect effect = effectsSequence[0];

    // فعال‌سازی Timing/Rewind افکت.
    effect.Timing.Rewind = true;
}
```

## IEffect.DelayBetweenTextParts خاصیت

تاخیر بین بخش‌های متنی انیمیشن‌شده (کلمات یا حروف) را تعریف می‌کند. مقدار مثبت درصد مدت اثر را مشخص می‌کند. مقدار منفی تاخیر را بر حسب ثانیه تعیین می‌کند. خواندن/نوشتن Single.

```csharp
public float DelayBetweenTextParts { get; set; }
```

### مثال‌ها

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // دریافت اولین افکت اسلاید اول.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // تغییر نوع متن انیمیشن افکت به "By word"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByWord;
    
    // تنظیم تاخیر بین بخش‌های متنی انیمیشن‌شده به 20٪ از مدت اثر.
    firstSlideEffect.DelayBetweenTextParts = 20f;
}
```

## BehaviorCollection.CopyTo متد

عناصر ICollection را به یک Array کپی می‌کند، که از یک ایندکس خاص در Array شروع می‌شود.

```csharp
public void CopyTo(IBehavior[] array, int arrayIndex)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| array | IBehavior[] | آریه تک‌بعدی که مقصد عناصری است که از ICollection کپی شده‌اند. Array باید ایندکس صفر پایه داشته باشد. |
| arrayIndex | Int32 | ایندکس صفرپایه در *array* که کپی از آنجا شروع می‌شود. |

### استثناها

| استثنا | شرط |
| --- | --- |
| ArgumentNullException | *array* برابر null است. |
| ArgumentOutOfRangeException | *arrayIndex* کمتر از 0 است. |
| ArgumentException | تعداد عناصر در ICollection منبع بیشتر از فضای موجود از *arrayIndex* تا انتهای *array* مقصد است. |

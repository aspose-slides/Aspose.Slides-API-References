## Timing.RepeatUntilNextClick ویژگی

این ویژگی مشخص می‌کند که آیا اثر تا کلیک بعدی تکرار می‌شود یا نه. قابل خواندن/نوشتن Boolean.

```csharp
public bool RepeatUntilNextClick { get; set; }
```

### مثال‌ها

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // دنباله افکت‌ها را برای اولین اسلاید دریافت کنید
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // اولین افکت از دنباله اصلی را دریافت کنید.
    IEffect effect = effectsSequence[0];

    // تغییر زمان‌بندی/تکرار افکت به "Until Next Click"
    effect.Timing.RepeatUntilNextClick = true;
}
```

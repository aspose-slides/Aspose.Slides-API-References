## Timing.RepeatUntilEndSlide ویژگی

این ویژگی مشخص می‌کند که آیا اثر تا انتهای اسلاید تکرار شود یا نه. خواندنی/نوشتنی Boolean.

```csharp
public bool RepeatUntilEndSlide { get; set; }
```

### مثال‌ها

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // دریافت توالی اثرها برای اولین اسلاید
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // دریافت اولین اثر از توالی اصلی.
    IEffect effect = effectsSequence[0];

    // تغییر Timing/Repeat اثر به "تا انتهای اسلاید"
    effect.Timing.RepeatUntilEndSlide = true;
}
```

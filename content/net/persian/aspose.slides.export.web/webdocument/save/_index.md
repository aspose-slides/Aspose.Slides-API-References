## WebDocument.Save متد

خروجی سند را ذخیره می‌کند.

```csharp
public void Save()
```

### مثال‌ها

```csharp
[C#]        
using (Presentation pres = new Presentation("pres.pptx"))
{
    var options = new WebDocumentOptions
    {
        TemplateEngine = new RazorTemplateEngine(),
        OutputSaver = new FileOutputSaver(),
        EmbedImages = false
    };
    
    WebDocument document = new WebDocument(options);

    // الگوی "index-template.html" را با کلید قالب "index" اضافه می‌کند تا بعدها (برای خروجی) استفاده شود
    document.Input.AddTemplate<Aspose.Slides.Presentation>("index", "index-template.html");

    // "index.html" را به فایل‌های خروجی اضافه می‌کند، با استفاده از الگوی "index" برای تولید آن و متغیر pres به عنوان مدل
    document.Output.Add("index.html", "index", pres);
    
    // ... سایر گزینه‌های سند را تنظیم می‌کند و سپس سند را ذخیره می‌کند
    document.Global.Put("slideMargin", 10);
    document.Global.Put("imagesPath", "root/site/images");
    // ...
 
    document.Save();
}
```

## WebDocument.Global ویژگی

public Storage Global { get; }

### مثال‌ها

با استفاده از این ویژگی `Global` (پیاده‌سازی رابط [`Storage`](../../storage)) می‌توان این ویژگی را بعداً در قالب به کار برد:

```csharp
[C#]
var options = new WebDocumentOptions
{
    TemplateEngine = new RazorTemplateEngine(),
    OutputSaver = new FileOutputSaver(),
    EmbedImages = false
};

WebDocument document = new WebDocument(options);

// ویژگی "slideMargin" را برای استفاده در قالب‌ها قرار می‌دهد
document.Global.Put("slideMargin", 10);

// ... سایر گزینه‌های سند را تنظیم کنید و سپس سند را ذخیره کنید
document.Save();
```

## ExcelDataWorkbook.GetCells متد

یک مجموعه از سلول‌ها را از workbook که با فرمول مشخص‌شده مطابقت دارند، بازیابی می‌کند.

```csharp
public ReadOnlyCollection<IExcelDataCell> GetCells(string formula, bool skipHiddenCells)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| formula | String | یک فرمول یا عبارت بازه (مثلاً "Sheet1!A1:B3") که برای شناسایی سلول‌های هدف استفاده می‌شود. |
| skipHiddenCells | Boolean | اگر `true` باشد، سلول‌های پنهان (مثلاً در ردیف‌ها یا ستون‌های پنهان) از نتیجه حذف خواهند شد. |

### مقدار بازگشتی

یک لیست فقط-قابل‌خواندن از سلول‌هایی که با فرمول تعیین‌شده مطابقت دارند.

### مثال‌ها

مثال:

```csharp
[C#]
ExcelDataWorkbook wb = new ExcelDataWorkbook(testFile);
IReadOnlyList<IExcelDataCell> cells = wb.GetCells("Sheet1!A2:A6", false);
Console.WriteLine(cells.Count); //خروجی: 5
```

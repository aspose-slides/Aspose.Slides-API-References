## ExcelDataWorkbook.GetChartsFromWorksheet متد

یک دیکشنری شامل شاخص‌ها و نام‌های تمام نمودارها در برگه‌کار مشخص‌شده یک کارنامه Excel برمی‌گرداند.

```csharp
public IDictionary<int, string> GetChartsFromWorksheet(string worksheetName)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| worksheetName | String | نام برگه کاری که برای جستجوی نمودارها می‌گردد. |

### مقدار بازگشت

یک دیکشنری که کلید آن شاخص نمودار و مقدار آن نام نمودار است.

### مثال‌ها

مثال:

```csharp
[C#]
ExcelDataWorkbook wb = new ExcelDataWorkbook(testFile);
var sheetCharts = wb.GetChartsFromWorksheet("worksheetName");
foreach (var chart in sheetCharts)
    Console.WriteLine(chart.Key + " : " + chart.Value);
```

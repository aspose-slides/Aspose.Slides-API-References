حاوی کلاس‌هایی است که ویژگی‌های مبتنی بر هوش مصنوعی برای تجزیه و تحلیل و پردازش ارائه‌های PowerPoint را فراهم می‌کنند.

## کلاس‌ها

| کلاس | توضیح |
| --- | --- |
| [AsposeAIWebClient](./asposeaiwebclient) | یک پیاده‌سازی [`IAIWebClient`](../aspose.slides.ai/iaiwebclient) داخلی که به LLM اختصاصی Aspose متصل می‌شود. این مشتری پیش‌فرض است که توسط سازنده بدون پارامتر [`SlidesAIAgent`](../aspose.slides.ai/slidesaiagent/slidesaiagent) استفاده می‌شود. |
| [OpenAICompatibleWebClient](./openaicompatiblewebclient) | یک پیاده‌سازی [`IAIWebClient`](../aspose.slides.ai/iaiwebclient) داخلی که به یک ارائه‌کننده LLM سازگار با OpenAI در URL پایه مشخص‌شده متصل می‌شود. |
| [OpenAIWebClient](./openaiwebclient) | یک پیاده‌سازی [`IAIWebClient`](../aspose.slides.ai/iaiwebclient) داخلی که به API OpenAI متصل می‌شود. |
| [SlidesAIAgent](./slidesaiagent) | ویژگی‌های مبتنی بر هوش مصنوعی برای پردازش ارائه‌ها را فراهم می‌کند. |
| [SlidesAIAgentException](./slidesaiagentexception) | نمایانگر استثناهای مرتبط با Slides AI Agent است. |

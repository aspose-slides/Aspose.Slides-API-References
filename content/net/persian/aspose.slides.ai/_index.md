شامل کلاس‌هایی است که ویژگی‌های مبتنی بر هوش مصنوعی برای تجزیه و تحلیل و پردازش ارائه‌های PowerPoint را فراهم می‌کنند.

## کلاس‌ها

| کلاس | توضیح |
| --- | --- |
| [AsposeAIWebClient](./asposeaiwebclient) | یک پیاده‌سازی داخلی [`IAIWebClient`](../aspose.slides.ai/iaiwebclient) که به LLM اختصاصی Aspose وصل می‌شود. این مشتری پیش‌فرض است که توسط سازنده بدون پارامتر [`SlidesAIAgent`](../aspose.slides.ai/slidesaiagent/slidesaiagent) استفاده می‌شود. |
| [OpenAICompatibleWebClient](./openaicompatiblewebclient) | یک پیاده‌سازی داخلی [`IAIWebClient`](../aspose.slides.ai/iaiwebclient) که به یک ارائه‌دهنده LLM سازگار با OpenAI در URL پایه مشخص شده متصل می‌شود. |
| [OpenAIWebClient](./openaiwebclient) | یک پیاده‌سازی داخلی [`IAIWebClient`](../aspose.slides.ai/iaiwebclient) که به API OpenAI متصل می‌شود. |
| [SlidesAIAgent](./slidesaiagent) | امکانات مبتنی بر هوش مصنوعی برای پردازش ارائه‌ها را فراهم می‌کند. |
| [SlidesAIAgentException](./slidesaiagentexception) | استثناهای مربوط به Slides AI Agent را نمایش می‌دهد. |

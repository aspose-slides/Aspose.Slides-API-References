## IMathBlockCollection.IndexOf متد

شاخص یک IMathBlock خاص را در مجموعه تعیین می‌کند.

```csharp
public int IndexOf(IMathBlock item)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| item | IMathBlock | آیتمی که باید در مجموعه یافت شود. |

### مقدار بازگشتی

اگر *item* در مجموعه یافت شود، شاخص آن؛ در غیر این صورت، -1.

### مثال‌ها

مثال:

```csharp
[C#]
IMathBlockCollection blockCollection = new MathParagraph();
IMathBlock block = new MathBlock(new MathematicalText("y"));
blockCollection.Add(block);
int index = blockCollection.IndexOf(block);
```

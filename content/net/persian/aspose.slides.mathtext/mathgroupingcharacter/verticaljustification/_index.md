## MathGroupingCharacter.VerticalJustification ویژگی

جهت‌گیری عمودی کاراکتر گروه. تراز شی را نسبت به خط پایه مشخص می‌کند. برای مثال، وقتی کاراکتر گروه بالای شی قرار دارد، VerticalJustification مقدار Top نشان می‌دهد که بالای شی بر روی خط پایه قرار می‌گیرد؛ وقتی VerticalJustification بر روی Bottom تنظیم شود، پایین شی روی خط پایه خواهد بود. پیش‌فرض: Bottom برای Position=Top و Top برای Position=Bottom

```csharp
public MathTopBotPositions VerticalJustification { get; set; }
```

### مثال‌ها

Example:

```csharp
[C#]
MathGroupingCharacter groupingCharacter = new MathGroupingCharacter(new MathematicalText("abc"));
groupingCharacter.VerticalJustification = MathTopBotPositions.Top;
```

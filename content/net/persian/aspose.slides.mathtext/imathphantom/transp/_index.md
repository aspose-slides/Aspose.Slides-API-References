## IMathPhantom.Transp ویژگی

دریافت یا تنظیم مقداری که نشان می‌دهد آیا فانتوم برای قواعد فاصله‌گذاری مبتنی بر کلاس شفاف است یا خیر.

```csharp
public bool Transp { get; set; }
```

### توضیحات

هنگامی که `true` باشد، عملگرها و نمادهای داخل فانتوم همچنان بر فاصله‌گذاری ریاضی اطراف فانتوم تأثیر می‌گذارند (گویی قابل مشاهده هستند). هنگامی که `false` باشد، فاصله‌گذاری مبتنی بر کلاس نادیده گرفته می‌شود. مطابق با ویژگی OMML `m:transp`.

## IMathBox.NoBreak ویژگی

بدون شکستن. این ویژگی خاصیت «unbreakable» را بر روی جعبه شیء مشخص می‌کند. وقتی مقدار آن true باشد، هیچ شکستگی خطی نمی‌تواند در داخل جعبه رخ دهد. این می‌تواند برای شبیه‌سازهای عملگر که شامل بیش از یک عملگر باینری هستند مهم باشد. وقتی این عنصر مشخص نشود، شکستگی‌ها می‌توانند داخل جعبه رخ دهند. پیش‌فرض: true

public bool NoBreak { get; set; }

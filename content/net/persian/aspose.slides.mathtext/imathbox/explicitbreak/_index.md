## IMathBox.ExplicitBreak ویژگی

Explicit break مشخص می‌کند آیا در ابتدای شیء Box خط‌شیوه‌ای وجود دارد که موجب بسته شدن خط در ابتدای شیء Box شود. تعداد عملگر در خط قبلی متن ریاضی را که به عنوان نقطه تراز برای خط جاری متن ریاضی استفاده می‌شود، تعیین می‌کند. مقادیر ممکن: 1..255 پیش‌فرض: 0 (بدون شکست صریح)

```csharp
public byte ExplicitBreak { get; set; }
```

مثال:

[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.ExplicitBreak = 1;

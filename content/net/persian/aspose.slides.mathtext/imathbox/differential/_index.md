## IMathBox.Differential ویژگی

Differential. وقتی مقدار true باشد، جعبه به عنوان یک دیفرانسیل عمل می‌کند (مثلاً 𝑑𝑥 در یک انتگرال) و فاصله افقی مناسب برای دیفرانسیل ریاضی دریافت می‌کند. پیش‌فرض: false

public bool Differential { get; set; }

## IMathBox.OperatorEmulator خاصیت

Operator Emulator. When true, جعبه و محتویات آن مانند یک عملگر واحد رفتار می‌کنند و ویژگی‌های یک عملگر را به ارث می‌برند. به عنوان مثال، این بدان معنی است که این کاراکتر می‌تواند به عنوان نقطه‌ای برای شکست خط استفاده شود و می‌تواند به سایر عملگرها تراز شود. Operator Emulators معمولاً زمانی استفاده می‌شوند که یک یا چند گلیف برای تشکیل یک عملگر ترکیب شوند، مانند '==' . Default value: false

```csharp
public bool OperatorEmulator { get; set; }
```

Example:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.OperatorEmulator = true;
```

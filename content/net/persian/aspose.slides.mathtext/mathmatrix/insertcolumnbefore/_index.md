## MathMatrix.InsertColumnBefore متد

یک ستون جدید را پیش از ستون مشخص‌شده درج می‌کند. در ابتدا تمام عناصر ستون جدید مقدار null دارند.

```csharp
public void InsertColumnBefore(int columnIndex)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| columnIndex | Int32 | اندیس ستونی که قبل از آن یک ستون جدید قرار می‌گیرد |

### مثال‌ها

مثال:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.InsertColumnBefore(0);
```

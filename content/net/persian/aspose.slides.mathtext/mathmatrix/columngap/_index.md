## MathMatrix.ColumnGap ویژگی

مقدار فاصله افقی بین ستون‌های ماتریس؛ اگر ColumnGapRule برابر 3 ("Exactly") باشد، واحد به عنوان تویپ (1/20 ام نقطه) تفسیر می‌شود. اگر ColumnGapRule برابر 4 ("Multiple") باشد، واحد به عنوان تعداد افزایشی 0.5 ام تفسیر می‌شود. در سایر موارد نادیده گرفته می‌شود. پیش‌فرض: 0

```csharp
public uint ColumnGap { get; set; }
```

### مثال‌ها

مثال:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.ColumnGapRule = MathSpacingRules.Exactly;
matrix.ColumnGap = 20;
```

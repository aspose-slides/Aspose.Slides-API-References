## MathMatrix.RowGap خصوصیت

مقدار فاصله عمودی بین سطرهای یک ماتریس؛ اگر RowGapRule برابر 3 ("Exactly") باشد، واحد به عنوان twips (1/20 نقطه) تفسیر می‌شود. اگر RowGapRule برابر 4 ("Multiple") باشد، واحد به عنوان نیم‌خط‌ها تفسیر می‌شود. پیش‌فرض: 0

```csharp
public uint RowGap { get; set; }
```

### مثال‌ها

مثال:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

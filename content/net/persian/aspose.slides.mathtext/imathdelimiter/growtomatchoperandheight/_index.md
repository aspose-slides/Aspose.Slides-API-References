## IMMathDelimiter.GrowToMatchOperandHeight ویژگی

رشد BeginningCharacter ، SeparatorCharacter ، EndingCharacter را مشخص می‌کند. وقتی مقدار true باشد، جداکننده‌ها به صورت عمودی رشد می‌کنند تا با ارتفاع operand خود مطابقت داشته باشند. مقدار پیش‌فرض true است

```csharp
public bool GrowToMatchOperandHeight { get; set; }
```

## IMMathDelimiter.SeparatorCharacter ویژگی

Delimiter Separator Character مشخص می‌کند که چه کاراکتری آرگومان‌ها را در شیء delimiter جدا می‌کند. مقدار پیش‌فرض: '&#x7C;'.

```csharp
public char SeparatorCharacter { get; set; }
```

### مثال‌ها

مثال:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Join("y").Enclose();
delimiter.SeparatorCharacter = '$';
```

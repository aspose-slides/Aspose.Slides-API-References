## IMMathDelimiter.DelimiterShape ویژگی

مشخص می‌کند شکل جداکننده‌ها در شیء delimiter چیست. وقتی مقدار MathDelimiterShape.Centered باشد، جداکننده‌ها حول محور ریاضی متن ریاضی متمرکز می‌شوند و همچنان به‌گونه‌ای تنظیم می‌شوند که کل ارتفاع محتوای آن‌ها را بپوشانند. وقتی مقدار MathDelimiterShape.Match باشد، ارتفاع و شکل آن‌ها طوری تغییر می‌کند که دقیقاً با محتوایشان مطابقت داشته باشد.

```csharp
public MathDelimiterShape DelimiterShape { get; set; }
```

### مثال‌ها

Example:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.DelimiterShape = MathDelimiterShape.Match;
```

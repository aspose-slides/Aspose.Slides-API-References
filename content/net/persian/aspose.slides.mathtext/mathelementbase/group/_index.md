## Group(char, MathTopBotPositions, MathTopBotPositions) {#group_1}

این عنصر را با استفاده از یک کاراکتر گروه‌بندی مانند براکت کروی پایین یا کاراکتر دیگری در یک گروه قرار می‌دهد

```csharp
public IMathGroupingCharacter Group(char character, MathTopBotPositions position, 
    MathTopBotPositions verticalJustification)
```

| Parameter | Type | Description |
| --- | --- | --- |
| character | Char | کاراکتر گروه‌بندی مانند BOTTOM CURLY BRACKET (U+23DF) یا هر کاراکتر دیگری |
| position | MathTopBotPositions | موقعیت کاراکتر گروه‌بندی |
| verticalJustification | MathTopBotPositions | تخصیص عمودی کاراکتر گروه‌بندی. موقعیت‌سنجی شی نسبت به خط پایه را مشخص می‌کند. برای مثال، وقتی کاراکتر گروه‌بندی بالای شی قرار دارد، VerticalJustification مقدار Top نشان می‌دهد که بالای شی بر روی خط پایه قرار می‌گیرد؛ وقتی VerticalJustification برابر Bottom تنظیم شده باشد، پایین شی بر روی خط پایه قرار می‌گیرد |

### Return Value

نمونه جدیدی از نوع [`IMathGroupingCharacter`](../../imathgroupingcharacter)

### Examples

مثال:

```csharp
[C#]
IMathGroupingCharacter groupingElement = new MathematicalText("x;y;z").Group('\u23E1', MathTopBotPositions.Bottom, MathTopBotPositions.Top);
```

## IMathMatrix.RowGap ویژگی

مقدار فاصله عمودی بین ردیف‌های یک ماتریس؛ اگر RowGapRule برابر 3 ("Exactly") باشد، واحد به صورت twips (1/20 یک نقطه) تفسیر می‌شود. اگر RowGapRule برابر 4 ("Multiple") باشد، واحد به صورت نصف-خط‌ها تفسیر می‌شود. پیش‌فرض: 0

```csharp
public uint RowGap { get; set; }
```

### نمونه‌ها

مثال:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

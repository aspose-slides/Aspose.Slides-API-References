## IMathMatrix.ColumnGap ویژگی

مقدار فاصله افقی بین ستون‌های یک ماتریس؛ اگر ColumnGapRule بر روی 3 ("Exactly") تنظیم شود، واحد به صورت twips (1/20 نقطه) تفسیر می‌شود اگر ColumnGapRule بر روی 4 ("Multiple") تنظیم شود، واحد به تعداد افزایش‌های 0.5 em تعبیر می‌شود. در موارد دیگر نادیده گرفته می‌شود. پیش‌فرض: 0

```csharp
public uint ColumnGap { get; set; }
```

### مثال‌ها

مثال:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.ColumnGapRule = MathSpacingRules.Exactly;
matrix.ColumnGap = 20;
```

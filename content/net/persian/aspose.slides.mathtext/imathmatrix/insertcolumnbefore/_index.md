## IMathMatrix.InsertColumnBefore متد

```csharp
public void InsertColumnBefore(int columnIndex)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| columnIndex | Int32 | اندیس ستونی که قبل از آن باید یک ستون جدید درج شود |

### مثال‌ها

مثال:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.InsertColumnBefore(0);
```

## MathBox.OperatorEmulator ویژگی

Operator Emulator. وقتی مقدار true باشد، جعبه و محتوای آن مانند یک عملگر واحد رفتار می‌کنند و ویژگی‌های یک عملگر را به ارث می‌برند. این به این معنی است که، به عنوان مثال، کاراکتر می‌تواند به عنوان نقطه‌ای برای شکست خط استفاده شود و می‌تواند به سایر عملگرها هم‌سطح شود. معمولاً از Operator Emulatorها زمانی استفاده می‌شود که یک یا چند گلیف ترکیب شوند تا یک عملگر تشکیل دهند، مانند '=='. مقدار پیش‌فرض: false

```csharp
public bool OperatorEmulator { get; set; }
```

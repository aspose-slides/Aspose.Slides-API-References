## MathBox.ExplicitBreak ویژگی

شکست صریح تعیین می‌کند آیا در ابتدای شیء Box یک شکست خط وجود دارد یا خیر، به طوری که خط در ابتدای شیء Box به‌صورت خودکار بسته شود. تعداد عملگر در خط قبلی متن ریاضی را که به عنوان نقطه‌ی تراز برای خط فعلی متن ریاضی استفاده می‌شود، مشخص می‌کند. مقادیر ممکن: 1..255 پیش‌فرض: 0 (بدون شکست صریح)

```csharp
public byte ExplicitBreak { get; set; }
```

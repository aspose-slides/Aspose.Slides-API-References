## MathBox.NoBreak ویژگی

بدون شکست این ویژگی مشخص می‌کند که ویژگی «غیرقابل شکست» بر روی جعبه شیء تنظیم شده است. وقتی مقدار آن true باشد، هیچ شکستی در خطوط داخل جعبه امکان‌پذیر نیست. این می‌تواند برای شبیه‌سازهای عملگر که شامل بیش از یک عملگر دودویی هستند مهم باشد. هنگامی که این عنصر مشخص نشده باشد، امکان شکستن در داخل جعبه وجود دارد. پیش‌فرض: true

```csharp
public bool NoBreak { get; set; }
```

مثال:

[C#]
MathBox box = new MathBox(new MathematicalText("*****"));
box.NoBreak = false;

## MathDelimiter.SeparatorCharacter خاصیت

Delimiter Separator Character مشخص می‌کند که کدام کاراکتر آرگومان‌ها را در شیء جداکننده جدا می‌کند. مقدار پیش‌فرض: '&#x7C;'.

```csharp
public char SeparatorCharacter { get; set; }
```

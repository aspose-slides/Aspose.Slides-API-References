## MathDelimiter.DelimiterShape خاصیت

شکل delimiters را در شیء delimiter مشخص می‌کند. زمانی که مقدار MathDelimiterShape.Centered باشد، delimiters حول محور ریاضی متن ریاضی متمرکز می‌شوند و همچنان به‌گونه‌ای تنظیم می‌شوند که تمام ارتفاع محتویاتشان را پوشش دهند. زمانی که مقدار MathDelimiterShape.Match باشد، ارتفاع و شکل آنها دقیقاً مطابق محتویاتشان تغییر می‌کند.

```csharp
public MathDelimiterShape DelimiterShape { get; set; }
```

### مثال‌ها

مثال:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.DelimiterShape = MathDelimiterShape.Match;
```

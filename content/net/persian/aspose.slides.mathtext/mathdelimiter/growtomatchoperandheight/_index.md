## MathDelimiter.GrowToMatchOperandHeight ویژگی

رشد BeginningCharacter، SeparatorCharacter، EndingCharacter را مشخص می‌کند. هنگامی که مقدار true باشد، جداکننده‌ها به صورت عمودی رشد می‌کنند تا به ارتفاع عملوند خود منطبق شوند. مقدار پیش‌فرض true است.

```csharp
public bool GrowToMatchOperandHeight { get; set; }
```

### مثال‌ها

مثال:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.GrowToMatchOperandHeight = false;
```

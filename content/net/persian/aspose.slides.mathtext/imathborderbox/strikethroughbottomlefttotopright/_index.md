## IMathBorderBox.StrikethroughBottomLeftToTopRight ویژگی

خط مورب خط‌خورده از پایین-چپ به بالا-راست (به‌صورت پیش‌فرض false است). وضعیت مخفی یا نمایان خط مورب خط‌خورده را از گوشه پایین-چپ تا گوشه بالا-راست جعبه مرزی مشخص می‌کند.

```csharp
public bool StrikethroughBottomLeftToTopRight { get; set; }
```

### مثال‌ها

```csharp

borderBox.StrikethroughBottomLeftToTopRight = true;
```

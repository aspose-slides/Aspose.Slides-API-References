## MathPhantom سازنده

یک نمونه جدید از کلاس [`MathPhantom`](../../mathphantom) را با استفاده از عنصر پایه ریاضی مشخص شده مقداردهی اولیه می‌کند.

```csharp
public MathPhantom(IMathElement element)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| element | IMathElement | پایه [`IMathElement`](../../imathelement) که قابلیت مشاهده و چیدمان آن توسط فانتوم کنترل می‌شود. این عنصر محتوایی را تعریف می‌کند که می‌تواند مخفی یا نمایش داده شود، در حالی که هنوز بر هم‌ترازی هندسی ریاضیات اطراف تأثیر می‌گذارد. |

### یادداشت‌ها

عنصر فانتوم برای حفظ یا سرکوب فضای بصری عبارت پایه استفاده می‌شود بدون این که لزوماً نمایش داده شود. این عنصر معادل عنصر OMML `<m:phant>` است.

### مثال‌ها

```csharp
[C#]
IMathElement fraction = new MathFraction(
    new MathematicalText("1"),
    new MathematicalText("2"));
```

## MathPhantom.Transp ویژگی

مقداری را که نشان می‌دهد آیا فانتوم برای قوانین فاصله‌گذاری مبتنی بر کلاس شفاف است یا نه، دریافت یا تنظیم می‌کند.

```csharp
public bool Transp { get; set; }
```

### یادداشت‌ها

زمانی که `true` باشد، عملگرها و نمادهای داخل فانتوم همچنان بر فاصله‌گذاری ریاضی اطراف فانتوم تأثیر می‌گذارند (همانند قابل رؤیت بودن). هنگامی که `false` باشد، فاصله‌گذاری مبتنی بر کلاس نادیده گرفته می‌شود. معادل ویژگی OMML `m:transp`.

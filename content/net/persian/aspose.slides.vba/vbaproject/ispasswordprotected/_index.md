## VbaProject.IsPasswordProtected ویژگی

نشان می‌دهد آیا VBAProject توسط یک رمز عبور برای مشاهده ویژگی‌های پروژه محافظت می‌شود یا خیر. فقط‌خواندنی Boolean.

```csharp
public bool IsPasswordProtected { get; }
```

### مثال‌ها

```csharp
[C#]
using (var presentation = new Presentation(path + "demo.pptm"))
{
    if (presentation.VbaProject.IsPasswordProtected)
        Console.WriteLine("The VBAProject '" + presentation.VbaProject.Name + 
            "' is protected by password to view project properties.");
}
```

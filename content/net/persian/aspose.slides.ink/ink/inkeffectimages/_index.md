## Ink.InkEffectImages ویژگی

دریافت مجموعه‌ای از تصاویر سفارشی که برای شبیه‌سازی افکت‌های بصری برای قلم‌های جوهر استفاده می‌شوند. این تصاویر هنگام رندر کردن جوهر با مقادیر خاص [`InkEffectType`](../../inkeffecttype)، مانند Galaxy، Rainbow و غیره استفاده می‌شوند. با ارائه تصاویر خود می‌توانید کنترل کنید که هر افکت جوهر چگونه ظاهر شود.

```csharp
public static IDictionary<InkEffectType, IImage> InkEffectImages { get; }
```

### توضیحات

این ویژگی امکان جایگزینی بافت‌های پیش‌فرض افکت جوهر را با موارد تعریف‌شده توسط کاربر فراهم می‌کند، که به‌ویژه زمانی مفید است که دارایی‌های پیش‌فرض توسط مجوز محدود شده یا در زمان اجرا در دسترس نباشند. هر ورودی در دیکشنری باید یک مقدار [`InkEffectType`](../../inkeffecttype) را با یک شیء [`IImage`](../../../aspose.slides/iimage) متناظر (مثلاً Bitmap یا یک رابط تصویر Aspose) مرتبط سازد.

### مثال‌ها

```csharp
[C#]
IImage image = Images.FromFile("image.png");
Ink.InkEffectImages.Add(InkEffectType.Galaxy, image);
```

## IInkBrush.InkEffect ویژگی

نوع اثر جوهر را دریافت می‌کند (e.g., Galaxy, Gold, Silver) که سبک بصری ضربهٔ جوهر را تعریف می‌کند. مقدار از ویژگی brush به نام "inkEffects" تجزیه می‌شود. اگر هیچ اثر شناسایی‌شده‌ای مشخص نشده باشد، NotDefined بازگردانده می‌شود.

```csharp
public InkEffectType InkEffect { get; }
```

### مثال‌ها

مثال:

```csharp
[C#]
using (var pres = new Presentation("Presentation.pptx"))
{
    Ink ink = pres.Slides[0].Shapes[0] as Ink;
    IInkBrush brush = ink.Traces[0].Brush;
    Console.WriteLine("InkEffects = {0}", brush.InkEffect);
}
```

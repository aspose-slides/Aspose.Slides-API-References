## ChartSeriesGroup.Overlap ویژگی

مشخص می‌کند که نوارها و ستون‌ها در نمودارهای دو-بعدی تا چه حد هم‌پوشانی داشته باشند، به‌صورت درصد (از -100٪ تا 100٪). - -100٪: بیشترین فاصله (نوارها کاملاً جدا هستند). - 0٪: نوارها به‌صورت کنار هم بدون هم‌پوشانی یا فاصله قرار می‌گیرند. - 100٪: بیشترین هم‌پوشانی (نوارها کاملاً روی یکدیگر قرار می‌گیرند). این ویژگی خواندنی/نوشتنی از نوع SByte است.

```csharp
public sbyte Overlap { get; set; }
```

### استثناها

| استثنا | شرط |
| --- | --- |
| ArgumentOutOfRangeException | زمانی که مقدار خارج از بازه معتبر ‎-100 تا 100 تنظیم شود، پرتاب می‌شود. |

### مثال‌ها

مثال زیر نشان می‌دهد که چگونه هم‌پوشانی یک گروه سری نمودار را تنظیم کرده و نمودار حاصل را روی یک فرم رندر کنید:

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // همپوشانی را به 55٪ تنظیم کنید

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

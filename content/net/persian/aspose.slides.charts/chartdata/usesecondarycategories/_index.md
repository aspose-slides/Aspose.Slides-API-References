## ChartData.UseSecondaryCategories ویژگی

اگر false باشد، خصوصیت [`SecondaryCategories`](../secondarycategories) null برمی‌گرداند و داده‌های موجود در خصوصیت [`Categories`](../categories) هم برای سری‌های اصلی و هم برای سری‌های ثانویه استفاده می‌شود. اگر true باشد، داده‌های موجود در خصوصیت [`SecondaryCategories`](../secondarycategories) برای سری‌های ثانویه استفاده می‌شود و داده‌های موجود در خصوصیت [`Categories`](../categories) برای سری‌های اصلی استفاده می‌شود. خواندن/نوشتن Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

### مثال‌ها

مثال. دسته‌بندی‌های مربوط به سری‌ها کدامند - ChartData.Categories یا ChartData.SecondaryCategories؟

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // دسته‌بندی‌های مرتبط series.Chart.ChartData.SecondaryCategories هستند
}
else
{
    // دسته‌بندی‌های مرتبط series.Chart.ChartData.Categories هستند
}
```

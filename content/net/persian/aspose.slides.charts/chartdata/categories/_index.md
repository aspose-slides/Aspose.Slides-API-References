## ChartData.Categories ویژگی

دسته‌های اصلی را دریافت می‌کند (یا هر دو دسته اصلی و فرعی اگر ویژگی [`UseSecondaryCategories`](../usesecondarycategories) نادرست باشد). فقط‌خواندنی [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection Categories { get; }
```

### نکات

اگر ویژگی [`UseSecondaryCategories`](../usesecondarycategories) نادرست باشد، ویژگی [`SecondaryCategories`](../secondarycategories) مقدار null بر می‌گرداند و داده‌های این ویژگی `Categories` هم برای سری‌های اصلی و هم برای سری‌های فرعی استفاده می‌شود. اگر ویژگی [`UseSecondaryCategories`](../usesecondarycategories) صحیح باشد، داده‌های ویژگی [`SecondaryCategories`](../secondarycategories) برای سری‌های فرعی استفاده می‌شود و داده‌های این ویژگی `Categories` برای سری‌های اصلی استفاده می‌شود.

### مثال‌ها

مثال. کدام دسته‌ها به سری‌ها مرتبط هستند - ChartData.Categories یا ChartData.SecondaryCategories؟

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // دسته‌های مرتبط series.Chart.ChartData.SecondaryCategories هستند
}
else
{
    // دسته‌های مرتبط series.Chart.ChartData.Categories هستند
}
```

## ChartData.SecondaryCategories ویژگی

دسته‌بندی‌های ثانویه را دریافت می‌کند اگر ویژگی [`UseSecondaryCategories`](../usesecondarycategories) مقدار true باشد. فقط‌خواندنی [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### توضیحات

اگر ویژگی [`UseSecondaryCategories`](../usesecondarycategories) مقدار false باشد، این ویژگی `SecondaryCategories` مقدار null برمی‌گرداند و داده‌های موجود در ویژگی [`Categories`](../categories) هم برای سری‌های اصلی و هم برای سری‌های ثانویه استفاده می‌شود. اگر ویژگی [`UseSecondaryCategories`](../usesecondarycategories) مقدار true باشد، داده‌های موجود در این ویژگی `SecondaryCategories` برای سری‌های ثانویه استفاده می‌شود و داده‌های موجود در ویژگی [`Categories`](../categories) برای سری‌های اصلی استفاده می‌شود.

### مثال‌ها

مثال. کدام دسته‌ها با سری‌ها مرتبط هستند - ChartData.Categories یا ChartData.SecondaryCategories؟

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // دسته‌بندی‌های مرتبط series.Chart.ChartData.SecondaryCategories هستند
}
else
{
    // دسته‌بندی‌های مرتبط series.Chart.ChartData.Categories هستند
}
```

## SetExternalWorkbook(string) {#setexternalworkbook}

یک کتاب‌کار خارجی را به عنوان منبع داده برای نمودار تنظیم می‌کند. داده‌های نمودار از کتاب‌کار هدف به‌روزرسانی خواهند شد.

```csharp
public void SetExternalWorkbook(string workbookPath)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| workbookPath | String | مسیر به کتاب‌کار هدف |

### استثناها

| استثنا | شرط |
| --- | --- |
| InvalidOperationException | کتاب‌کار خارجی در دسترس نیست یا نمی‌تواند بارگذاری شود. |

### مثال‌ها

```csharp
[C#]
using (Presentation pres = new Presentation())

   IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Pie, 50, 50, 400, 600, true);
   IChartData chartData = chart.ChartData;
   (chartData as ChartData).SetExternalWorkbook("../../workbook.xlsx");

```

## SetExternalWorkbook(string, bool) {#setexternalworkbook_1}


| پارامتر | نوع | توضیح |
| --- | --- | --- |
| workbookPath | String | مسیر به کتاب‌کار هدف |
| updateChartData | Boolean | اگر مقدار false باشد، فقط مسیر کتاب‌کار به‌روزرسانی می‌شود. داده‌های نمودار بارگذاری و به‌روزرسانی نمی‌شوند. می‌تواند زمانی استفاده شود که کتاب‌کار هدف وجود نداشته باشد یا در دسترس نباشد. اگر مقدار true باشد، داده‌های نمودار از کتاب‌کار هدف به‌روزرسانی می‌شوند. |

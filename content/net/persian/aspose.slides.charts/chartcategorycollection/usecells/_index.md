## ChartCategoryCollection.UseCells ویژگی

اگر مقدار true باشد، کاربرگ برای ذخیره‌سازی دسته‌ها استفاده می‌شود (در این حالت از دسته‌های چند‌سطحی پشتیبانی می‌شود). اگر مقدار false باشد، کاربرگ برای ذخیره‌سازی مقادیر استفاده نمی‌شود (و در این حالت از دسته‌های چند‌سطحی پشتیبانی نمی‌شود). قابل خواندن/قابل نوشتن Boolean.

```csharp
public bool UseCells { get; set; }
```

## IChartSeriesGroup.Overlap ویژگی

مشخص می‌کند که میله‌ها و ستون‌ها در نمودارهای دو بعدی تا چه اندازه باید هم‌پوشانی داشته باشند، به صورت درصد (از -100% تا 100%). - -100%: حداکثر فاصله (میله‌ها کاملاً جدا شده‌اند). - 0%: میله‌ها بدون هم‌پوشانی یا فاصله کنار هم قرار می‌گیرند. - 100%: حداکثر هم‌پوشانی (میله‌ها کاملاً روی یکدیگر قرار می‌گیرند). این ویژگی قابل خواندن/نوشتن است SByte.

```csharp
public sbyte Overlap { get; set; }
```

### نمونه‌ها

مثال زیر نشان می‌دهد که چگونه هم‌پوشانی یک گروه سری نمودار را تنظیم کرده و نمودار حاصل را بر روی فرم رندر کنید:

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // همپوشانی را به 55% تنظیم کنید

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

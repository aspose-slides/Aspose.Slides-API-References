## ChartSeries.GapDepth خاصیت

فاصله‌ای را که به عنوان درصدی از عرض marker است، بین داده‌های سری در یک نمودار 3D برمی‌گرداند یا تنظیم می‌کند. این خصوصیت نه تنها برای این سری بلکه برای تمام سری‌های گروه سری‌والد است – این یک تصویر از خصوصیت مناسب گروه است. به همین دلیل این خصوصیت فقط-خواندنی است. از خصوصیت ParentSeriesGroup برای دسترسی به گروه سری‌والد استفاده کنید. برای تغییر مقدار، از خصوصیت خواندنی/نوشتنی ParentSeriesGroup.GapDepth استفاده کنید. فقط-خواندنی Int32.

```csharp
public int GapDepth { get; }
```

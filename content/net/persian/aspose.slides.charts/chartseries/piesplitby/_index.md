## ویژگی ChartSeries.PieSplitBy

مشخص می‌کند که کدام نقاط داده در دایره یا نوار دوم یک نمودار pie-of-pie یا bar-of-pie قرار می‌گیرند. این ویژگی نه تنها مربوط به این سری، بلکه به تمام سری‌های گروه سری‌های والد تعلق دارد – این یک پروجکشن از ویژگی مربوط به گروه است. بنابراین این ویژگی فقط قابل خواندن است. برای دسترسی به گروه سری‌های والد از ویژگی ParentSeriesGroup استفاده کنید. برای تغییر مقدار، از ویژگی ParentSeriesGroup.PieSplitBy که قابل خواندن/نوشتن است استفاده کنید. فقط قابل خواندن [`PieSplitType`](../../piesplittype).

```csharp
public PieSplitType PieSplitBy { get; }
```

### توضیحات

1) این یک پروجکشن از ویژگی ParentSeriesGroup.PieSplitBy است.  
2) اگر مقدار ویژگی PieSplitType.Custom باشد می‌توانید اطلاعات تقسیم سفارشی را با ویژگی ParentSeriesGroup.PieSplitCustomPoints تعریف کنید.

## DataLabelFormat.ShowValue ویژگی

نمایانگر رفتار نمایش مقدار درصد برچسب داده در یک نمودار مشخص است. مقدار true مقدار درصد را نمایش می‌دهد. مقدار false برای مخفی کردن استفاده می‌شود. خواندنی/نوشتنی Boolean.

```csharp
public bool ShowValue { get; set; }
```

### توضیحات

اگر والد این شیء DataLabelFormat یک مجموعه DataLabelCollection از برچسب‌های داده باشد، این ویژگی مقدار پیش‌فرض ویژگی ShowValue را برای برچسب‌های داده جدید در مجموعه DataLabelCollection دریافت یا تنظیم می‌کند. تنظیم این ویژگی با مقدار، همچنین این مقدار را برای ویژگی ShowValue تمام برچسب‌های داده در مجموعه DataLabelCollection تنظیم می‌کند (به عنوان مثال "DataLabels.DefaultDataLabelFormat.ShowValue = val;" باعث می‌شود همه DataLabels[i].ShowValue برابر با val باشد).

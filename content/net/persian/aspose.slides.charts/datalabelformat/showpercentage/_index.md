## DataLabelFormat.ShowPercentage ویژگی

نمایانگر رفتار نمایش مقدار درصد برچسب داده نمودار مشخص شده است. True مقدار درصد را نمایش می‌دهد. False برای پنهان کردن. خواندنی/نوشتنی Boolean.

```csharp
public bool ShowPercentage { get; set; }
```

### توضیحات

اگر والد این شیء DataLabelFormat یک مجموعه DataLabelCollection از برچسب‌های داده باشد، این ویژگی مقدار پیش‌فرض ویژگی ShowPercentage را برای برچسب‌های داده جدید در مجموعه DataLabelCollection دریافت یا تنظیم می‌کند. تنظیم این ویژگی با مقدار، همچنین این مقدار را برای ویژگی ShowPercentage برای تمام برچسب‌های داده در مجموعه DataLabelCollection تنظیم می‌کند (به عنوان مثال "DataLabels.DefaultDataLabelFormat.ShowPercentage = val;" باعث می‌شود همه DataLabels[i].ShowPercentage برابر val باشد).

## DataLabelFormat.ShowBubbleSize خصوصیت

نمایانگر رفتار نمایش مقدار اندازه حباب برچسب داده برای نمودار مشخص شده است. True مقدار اندازه حباب را نمایش می‌دهد. False برای پنهان کردن. قابل خواندن/قابل نوشتن Boolean.

```csharp
public bool ShowBubbleSize { get; set; }
```

### توضیحات

اگر والد این شیء DataLabelFormat یک مجموعه DataLabelCollection از برچسب‌های داده باشد، این خصوصیت مقدار پیش‌فرض خصوصیت ShowBubbleSize را برای برچسب‌های داده جدید در مجموعه DataLabelCollection دریافت یا تنظیم می‌کند. تنظیم این خصوصیت با مقدار، همان مقدار را برای خصوصیت ShowBubbleSize تمام برچسب‌های داده در مجموعه DataLabelCollection نیز تنظیم می‌کند (به عنوان مثال "DataLabels.DefaultDataLabelFormat.ShowBubbleSize = val;" منجر می‌شود که تمام DataLabels[i].ShowBubbleSize برابر با val باشد).

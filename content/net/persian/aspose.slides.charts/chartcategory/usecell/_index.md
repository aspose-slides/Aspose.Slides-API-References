## ChartCategory.UseCell ویژگی

اگر true باشد، ویژگی AsCell فعال است. به عبارت دیگر، worksheet برای ذخیره‌سازی دسته استفاده می‌شود (این حالت از دسته‌بندی چند سطحی پشتیبانی می‌کند). اگر false باشد، ویژگی AsLiteral فعال است. به عبارت دیگر، worksheet برای ذخیره‌سازی دسته استفاده نمی‌شود (و این حالت از دسته‌های چند سطحی پشتیبانی نمی‌کند). ویژگی فقط-خواندنی Boolean.

```csharp
public bool UseCell { get; }
```

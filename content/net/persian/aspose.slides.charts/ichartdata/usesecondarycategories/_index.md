## IChartData.UseSecondaryCategories ویژگی

اگر مقدار false باشد، ویژگی [`SecondaryCategories`](../secondarycategories) null برمی‌گرداند و داده‌های موجود در ویژگی [`Categories`](../categories) برای هر دو سری اصلی و ثانویه استفاده می‌شود. اگر مقدار true باشد، داده‌های موجود در ویژگی [`SecondaryCategories`](../secondarycategories) برای سری ثانویه استفاده می‌شود و داده‌های موجود در ویژگی [`Categories`](../categories) برای سری اصلی استفاده می‌شود. قابل خواندن/نوشتن Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

### مثال‌ها

مثال. دسته‌های مرتبط با سری کدامند - ChartData.Categories یا ChartData.SecondaryCategories؟

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // دسته‌های مرتبط series.Chart.ChartData.SecondaryCategories است
}
else
{
    // دسته‌های مرتبط series.Chart.ChartData.Categories است
}
```

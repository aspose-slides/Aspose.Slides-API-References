## SetExternalWorkbook(string) {#setexternalworkbook}

کاربرگ خارجی را به عنوان منبع داده برای نمودار تنظیم می‌کند. داده‌های نمودار از کاربرگ هدف به‌روزرسانی می‌شوند.

```csharp
public void SetExternalWorkbook(string workbookPath)
```

| Parameter | Type | Description |
| --- | --- | --- |
| workbookPath | String | مسیر به کاربرگ هدف |

### استثناها

| exception | condition |
| --- | --- |
| InvalidOperationException | کاربرگ خارجی در دسترس نیست یا نمی‌تواند بارگذاری شود. |

### مثال‌ها

```csharp
[C#]
using (Presentation pres = new Presentation())

   IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Pie, 50, 50, 400, 600, true);
   IChartData chartData = chart.ChartData;
   (chartData as ChartData).SetExternalWorkbook("../../workbook.xlsx");
```

## SetExternalWorkbook(string, bool) {#setexternalworkbook_1}

کاربرگ خارجی را به عنوان منبع داده برای نمودار تنظیم می‌کند.

```csharp
public void SetExternalWorkbook(string workbookPath, bool updateChartData)
```

| Parameter | Type | Description |
| --- | --- | --- |
| workbookPath | String | مسیر به کاربرگ هدف |
| updateChartData | Boolean | اگر مقدار false باشد فقط مسیر کاربرگ به‌روزرسانی می‌شود. داده‌های نمودار از کاربرگ هدف بارگذاری یا به‌روزرسانی نمی‌شود. می‌توان از آن هنگامی که کاربرگ هدف وجود ندارد یا در دسترس نیست استفاده کرد. اگر مقدار true باشد داده‌های نمودار از کاربرگ هدف به‌روزرسانی می‌شوند. |

### استثناها

using (Presentation pres = new Presentation())

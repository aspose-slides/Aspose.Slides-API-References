## IChartData.SecondaryCategories ویژگی

دسته‌های ثانویه را دریافت می‌کند اگر ویژگی [`UseSecondaryCategories`](../usesecondarycategories) مقدار true باشد. فقط خواندنی [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### توضیحات

اگر ویژگی [`UseSecondaryCategories`](../usesecondarycategories) مقدار false باشد، این ویژگی `SecondaryCategories` مقدار null برمی‌گرداند و داده‌های موجود در ویژگی [`Categories`](../categories) هم برای سری اولیه و هم برای سری ثانویه استفاده می‌شوند. اگر ویژگی [`UseSecondaryCategories`](../usesecondarycategories) مقدار true باشد، داده‌های موجود در این ویژگی `SecondaryCategories` برای سری ثانویه استفاده می‌شوند و داده‌های موجود در ویژگی [`Categories`](../categories) برای سری اولیه استفاده می‌شوند.

### مثال‌ها

مثال. کدام دسته‌ها به سری‌ها مرتبط هستند - ChartData.Categories یا ChartData.SecondaryCategories؟

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // دسته‌های مرتبط هستند series.Chart.ChartData.SecondaryCategories
}
else
{
    // دسته‌های مرتبط هستند series.Chart.ChartData.Categories
}
```

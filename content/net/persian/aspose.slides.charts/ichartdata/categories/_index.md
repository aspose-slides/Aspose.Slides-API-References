## IChartData.Categories ویژگی

دسته‌های اصلی را برمی‌گرداند (یا هر دو دسته اصلی و فرعی را اگر [`UseSecondaryCategories`](../usesecondarycategories) ویژگی نادرست باشد). فقط-خواندنی [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection Categories { get; }
```

### توضیحات

اگر [`UseSecondaryCategories`](../usesecondarycategories) ویژگی نادرست باشد، [`SecondaryCategories`](../secondarycategories) ویژگی مقدار null برمی‌گرداند و داده‌های موجود در این ویژگی `Categories` برای هر دو سری اصلی و فرعی استفاده می‌شود. اگر [`UseSecondaryCategories`](../usesecondarycategories) ویژگی درست باشد، داده‌های موجود در [`SecondaryCategories`](../secondarycategories) ویژگی برای سری‌های فرعی استفاده می‌شوند و داده‌های موجود در این ویژگی `Categories` برای سری‌های اصلی استفاده می‌شود.

### مثال‌ها

مثال. کدام دسته‌ها به سری‌ها مربوط می‌شوند - ChartData.Categories یا ChartData.SecondaryCategories؟

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // دسته‌های مرتبط series.Chart.ChartData.SecondaryCategories هستند
}
else
{
    // دسته‌های مرتبط series.Chart.ChartData.Categories هستند
}
```

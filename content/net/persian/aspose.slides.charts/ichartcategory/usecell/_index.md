## IChartCategory.UseCell ویژگی

اگر مقدار true باشد، ویژگی AsCell فعال است. به عبارت دیگر، برگه کار برای ذخیره‌سازی دسته استفاده می‌شود (در این حالت از دسته چندسطحی پشتیبانی می‌کند). اگر مقدار false باشد، ویژگی AsLiteral فعال است. به عبارت دیگر، برگه کار برای ذخیره‌سازی دسته استفاده نمی‌شود (و این حالت از دسته‌های چندسطحی پشتیبانی نمی‌کند). Boolean فقط‌خواندنی.

```csharp
public bool UseCell { get; }
```

### توضیحات

برای تغییر مقدار این ویژگی (برای تمام دسته‌ها در مجموعه) مقدار جدید را به ویژگی ChartCategoryCollection.UseCells اختصاص دهید.

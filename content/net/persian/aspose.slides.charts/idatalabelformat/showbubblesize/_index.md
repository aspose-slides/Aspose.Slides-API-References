## IDataLabelFormat.ShowBubbleSize ویژگی

نمایانگر رفتار نمایش مقدار اندازه حباب برچسب داده‌های یک نمودار خاص است. True مقدار اندازه حباب را نمایش می‌دهد. False برای مخفی کردن. خواندنی/نوشتنی Boolean.

```csharp
public bool ShowBubbleSize { get; set; }
```

### توضیحات

اگر والد این شیء DataLabelFormat یک مجموعه DataLabelCollection از برچسب‌های داده باشد، این ویژگی مقدار پیش‌فرض ویژگی ShowBubbleSize را برای برچسب‌های داده جدید در مجموعه DataLabelCollection دریافت یا تنظیم می‌کند. تنظیم این ویژگی با مقداری، این مقدار را نیز برای ویژگی ShowBubbleSize تمام برچسب‌های داده در مجموعه DataLabelCollection تنظیم می‌گردد (به عبارت دیگر "DataLabels.DefaultDataLabelFormat.ShowBubbleSize = val;" باعث می‌شود که همه DataLabels[i].ShowBubbleSize برابر با val باشد).

## IDataLabelFormat.NumberFormat ویژگی

نمایش می‌دهد رشته قالب‌بندی برای شیء DataLabels. خواندنی/نوشتنی String.

```csharp
public string NumberFormat { get; set; }
```

### توضیحات

اگر والد این شیء DataLabelFormat مجموعه‌ای از DataLabelCollection باشد، این ویژگی مقدار پیش‌فرض ویژگی NumberFormat را برای برچسب‌های داده جدید در مجموعه DataLabelCollection دریافت یا تنظیم می‌کند. هنگامی که این ویژگی با یک مقدار تنظیم می‌شود، همان مقدار برای ویژگی NumberFormat تمام برچسب‌های داده در مجموعه DataLabelCollection نیز تنظیم می‌شود (به عنوان مثال "DataLabels.DefaultDataLabelFormat.NumberFormat = val;" باعث می‌شود همه DataLabels[i].NumberFormat برابر با val باشد).

### مثال‌ها

```csharp
[C#]
series.Labels.DefaultDataLabelFormat.ShowValue = true;
series.Labels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = false;
series.Labels.DefaultDataLabelFormat.NumberFormat = "0.0%";
```

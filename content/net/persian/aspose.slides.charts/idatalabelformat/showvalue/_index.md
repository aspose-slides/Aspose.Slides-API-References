## IDataLabelFormat.ShowValue خاصیت

نمایانگر رفتار نمایش مقدار درصد برچسب داده‌های یک نمودار مشخص است. True درصد را نمایش می‌دهد. False برای پنهان کردن استفاده می‌شود. قابل‌خواندن/قابل‌نوشتن Boolean.

```csharp
public bool ShowValue { get; set; }
```

### توضیحات

اگر والد این شیء DataLabelFormat یک مجموعه DataLabelCollection از برچسب‌های داده باشد، این خاصیت مقدار پیش‌فرض خاصیت ShowValue را برای برچسب‌های داده جدید در مجموعه DataLabelCollection دریافت یا تنظیم می‌کند. تنظیم این خاصیت با مقدار، همچنین این مقدار را برای خاصیت ShowValue تمام برچسب‌های داده در مجموعه DataLabelCollection تنظیم می‌نماید (i.e. "DataLabels.DefaultDataLabelFormat.ShowValue = val;" که باعث می‌شود تمام DataLabels[i].ShowValue برابر با val باشد).

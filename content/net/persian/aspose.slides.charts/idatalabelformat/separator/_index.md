## IDataLabelFormat.Separator ویژگی

یک Variant را تنظیم یا برمی‌گرداند که نشان‌دهنده جداساز استفاده‌شده برای برچسب‌های داده در یک نمودار است. خواندن/نوشتن String.

```csharp
public string Separator { get; set; }
```

### توضیحات

اگر والد این شیء DataLabelFormat یک مجموعه DataLabelCollection از برچسب‌های داده باشد، این ویژگی مقدار پیش‌فرض ویژگی Separator را برای برچسب‌های داده جدید در مجموعه DataLabelCollection دریافت یا تنظیم می‌کند. تنظیم این ویژگی با مقدار، همچنین این مقدار را برای ویژگی Separator تمام برچسب‌های داده در مجموعه DataLabelCollection تنظیم می‌کند (به عنوان مثال "DataLabels.DefaultDataLabelFormat.Separator = val;" باعث می‌شود همه DataLabels[i].Separator برابر val شود).

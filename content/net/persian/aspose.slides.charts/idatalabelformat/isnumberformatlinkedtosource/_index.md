## IDataLabelFormat.IsNumberFormatLinkedToSource ویژگی

خواندنی/نوشتنی Boolean.

```csharp
public bool IsNumberFormatLinkedToSource { get; set; }
```

### ملاحظات

اگر والد این شیء DataLabelFormat یک مجموعه DataLabelCollection از برچسب‌های داده باشد، این ویژگی مقدار پیش‌فرض ویژگی IsNumberFormatLinkedToSource را برای برچسب‌های دادهٔ جدید در مجموعه DataLabelCollection دریافت یا تنظیم می‌کند. تنظیم این ویژگی با مقدار، این مقدار را برای ویژگی IsNumberFormatLinkedToSource همهٔ برچسب‌های داده در مجموعه DataLabelCollection نیز تنظیم می‌کند (به عنوان مثال «DataLabels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = val;» که باعث می‌شود همهٔ DataLabels[i].IsNumberFormatLinkedToSource برابر با val باشد).

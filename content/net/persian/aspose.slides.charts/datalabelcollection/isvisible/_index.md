## DataLabelCollection.IsVisible ویژگی

False به این معنی است که برچسب داده به طور پیش‌فرض قابل مشاهده نیست (و بنابراین تمام پرچم‌های Show*-flags (ShowValue, ...) ویژگی DefaultDataLabelFormat نیز false هستند). فقط-خواندنی Boolean.

```csharp
public bool IsVisible { get; }
```

### توضیحات

اگر برچسب داده به طور پیش‌فرض قابل مشاهده باشد می‌توانید با متد Hide() آن را به‌صورت پیش‌فرض مخفی کنید. اما اگر برچسب داده به طور پیش‌فرض قابل مشاهده نباشد (IsVisible برابر false است) می‌توانید با تنظیم پرچم‌های Show*-flags (ShowValue, ...) ویژگی DefaultDataLabelFormat به حالت true، برچسب داده را «به طور پیش‌فرض قابل مشاهده» کنید.

## IChartCategoryCollection.UseCells ویژگی

اگر مقدار true باشد worksheet برای ذخیره‌سازی دسته‌ها استفاده می‌شود (در این حالت پشتیبان‌‍ی دسته‌های چندسطحی است). اگر مقدار false باشد worksheet برای ذخیره‌سازی مقادیر استفاده نمی‌شود (و در این حالت پشتیبان‌‍ی دسته‌های چندسطحی نیست). خواندنی/نوشتنی Boolean.

public bool UseCells { get; set; }

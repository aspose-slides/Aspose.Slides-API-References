## IDataLabelCollection.IsVisible ویژگی

False به این معنی است که برچسب داده به طور پیش‌فرض قابل مشاهده نیست (و بنابراین تمام پرچم‌های Show*- (ShowValue, ...) از ویژگی DefaultDataLabelFormat برابر false هستند). فقط خواندنی Boolean.

```csharp
public bool IsVisible { get; }
```

### توضیحات

اگر برچسب داده به طور پیش‌فرض قابل مشاهده باشد می‌توانید آن را به طور پیش‌فرض مخفی کنید با متد Hide(). اما اگر برچسب داده به طور پیش‌فرض قابل مشاهده نباشد (IsVisible برابر false است) می‌توانید برچسب داده را «قابل مشاهده به‌صورت پیش‌فرض» کنید با تنظیم پرچم‌های Show*- (ShowValue, ...) از ویژگی DefaultDataLabelFormat به حالت true.

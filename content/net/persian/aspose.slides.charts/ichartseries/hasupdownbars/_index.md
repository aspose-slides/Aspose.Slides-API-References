## IChartSeries.HasUpDownBars ویژگی

مشخص می‌کند که آیا Line- یا Stock-chart دارای نوارهای بالا/پایین است. این ویژگی نه تنها برای این سری بلکه برای تمام سری‌های گروه سری والد است - این بازتابی از ویژگی گروه مناسب است. بنابراین این ویژگی فقط‌خواندنی است. از ویژگی ParentSeriesGroup برای دسترسی به گروه سری والد استفاده کنید. برای تغییر مقدار، از ویژگی ParentSeriesGroup.UpDownBars.HasUpDownBars که قابل خواندن/نوشته است، استفاده کنید. برای قالب‌بندی نوارهای بالا/پایین، از ویژگی ParentSeriesGroup.UpDownBars استفاده کنید. Boolean فقط‌خواندنی.

```csharp
public bool HasUpDownBars { get; }
```

## IChartSeries.FirstSliceAngle ویژگی

زاویهٔ اولین برش نمودار دایره یا دونات را بر حسب درجه مشخص می‌کند (ساعت‌گرد از بالا، از 0 تا 360 درجه). این ویژگی نه تنها برای این سری، بلکه برای تمام سری‌های گروه سری والد است—این بازتابی از ویژگی مناسب گروه می‌باشد. بنابراین این ویژگی فقط‌خواندنی است. برای دسترسی به گروه سری والد از ویژگی ParentSeriesGroup استفاده کنید. برای تغییر مقدار، از ویژگی ParentSeriesGroup.FirstSliceAngle که قابل‌خواندن/قابل‌نوشتن است، استفاده کنید. UInt16 فقط‌خواندنی.

public ushort FirstSliceAngle { get; }

## IChartSeries.Overlap ویژگی

مشخص می‌کند که میله‌ها و ستون‌ها در نمودارهای دو-بعدی تا چه اندازه (به درصد) هم‌پوشانی دارند (از -100٪ تا 100٪). این ویژگی نه فقط برای این سری بلکه برای تمام سری‌های گروه سری والد است. این یک بازتاب از ویژگی مناسب در گروه سری والد است و بنابراین این ویژگی فقط-خواندنی است. برای تغییر مقدار، از ویژگی ParentSeriesGroup.Overlap که خواندنی/قابل نوشتن است استفاده کنید. SByte فقط-خواندنی.

```csharp
public sbyte Overlap { get; }
```

### توضیحات

Overlap درجهٔ هم‌پوشانی یا فاصله بین میله‌ها و ستون‌ها را به‌صورت درصدی از عرض آن‌ها مشخص می‌کند: - -100%: بیشترین فاصله (میله‌ها کاملاً جدا هستند). - 0%: میله‌ها کنار یکدیگر بدون هم‌پوشانی یا فاصله قرار می‌گیرند. - 100%: بیشترین هم‌پوشانی (میله‌ها کاملاً بر یکدیگر هم‌پوشانی می‌کنند). این یک بازتاب از ویژگی ParentSeriesGroup.Overlap است.

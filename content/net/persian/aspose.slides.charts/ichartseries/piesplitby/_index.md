## IChartSeries.PieSplitBy ویژگی

مشخص می‌کند که چگونه تعیین شود کدام نقاط داده در بخش دوم دایره یا نوار در نمودار دایره-در-دایره یا نوار-در-دایره قرار دارند. این ویژگی نه تنها برای این سری بلکه برای تمام سری‌های گروه سری والد است - این یک بازتاب از ویژگی مناسب گروه است. به همین دلیل این ویژگی فقط-خواندنی است. برای دسترسی به گروه سری والد از ویژگی ParentSeriesGroup استفاده کنید. برای تغییر مقدار، از ویژگی خواندن/نوشتن ParentSeriesGroup.PieSplitBy استفاده کنید. فقط-خواندنی [`PieSplitType`](../../piesplittype).

```csharp
public PieSplitType PieSplitBy { get; }
```

### یادداشت‌ها

1) این یک بازتاب از ویژگی ParentSeriesGroup.PieSplitBy است. 2) اگر مقدار ویژگی PieSplitType.Custom باشد، می‌توانید اطلاعات تقسیم سفارشی را با ویژگی ParentSeriesGroup.PieSplitCustomPoints تعریف کنید.

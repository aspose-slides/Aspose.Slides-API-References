شامل کلاس‌هایی است که داده‌ها را به ارائه‌های Microsoft PowerPoint وارد می‌کند.

## کلاس‌ها

| کلاس | توضیح |
| --- | --- |
| [ExcelWorkbookImporter](./excelworkbookimporter) | امکان وارد کردن محتوا از یک کتاب کار Excel به یک ارائه را فراهم می‌کند. |
| [ExternalResourceResolver](./externalresourceresolver) | کلاس Callback که برای حل منابع خارجی هنگام وارد کردن اسناد Html و Svg استفاده می‌شود. استفاده از این resolver می‌تواند باعث ایجاد آسیب‌پذیری شود هنگامی که فایل HTML یا SVG ارائه‌شده توسط کلاینت باعث می‌شود نرم‌افزار سرور به فایل‌های محلی یا شبکه‌ای دسترسی پیدا کند. با احتیاط استفاده کنید. توصیه می‌شود که ExternalResourceResolver را اصلاً مشخص نکنید (فقط اشیای تعبیه‌شده خوانده می‌شوند) یا یک زیرکلاس ایجاد کنید که بررسی کند آیا uri مشخص شده معتبر است یا نه. |
| [PdfImportOptions](./pdfimportoptions) | نمایانگر گزینه‌های وارد کردن PDF |

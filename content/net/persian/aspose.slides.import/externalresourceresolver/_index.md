## ExternalResourceResolver کلاس

کلاس Callback که برای حل منابع خارجی هنگام وارد کردن اسناد Html و Svg استفاده می‌شود. استفاده از این حل‌کننده می‌تواند باعث ایجاد آسیب‌پذیری شود هنگامی که فایل HTML یا SVG ارائه‌شده توسط مشتری باعث می‌شود نرم‌افزار سرور به فایل‌های محلی یا شبکه‌ای دسترسی پیدا کند. با احتیاط استفاده کنید. توصیه می‌شود که ExternalResourceResolver را به‌صورت کلی مشخص نکنید (فقط اشیاء توکار خوانده خواهند شد) یا زیرکلاسی ایجاد کنید که بررسی کند آیا uri مشخص شده معتبر است.

```csharp
public class ExternalResourceResolver : IExternalResourceResolver
```

## کلاس ExternalResourceResolver

کلاس Callback برای حل منابع خارجی هنگام وارد کردن اسناد Html و Svg استفاده می‌شود. استفاده از این حل‌کننده می‌تواند منجر به ایجاد یک آسیب‌پذیری شود که در آن فایل HTML یا SVG ارائه‌شده توسط مشتری باعث دسترسی نرم‌افزار سرور به فایل‌های محلی یا شبکه‌ای می‌شود. با احتیاط استفاده کنید. توصیه می‌شود که ExternalResourceResolver را به‌طور کامل مشخص نکنید (فقط اشیای تعبیه‌شده خوانده می‌شوند) یا یک زیرکلاس ایجاد کنید که بررسی کند آیا URI مشخص‌شده معتبر است یا خیر.

```csharp
public class ExternalResourceResolver : IExternalResourceResolver
```

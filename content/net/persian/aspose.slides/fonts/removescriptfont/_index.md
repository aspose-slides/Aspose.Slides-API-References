## Fonts.RemoveScriptFont متد

قالب نوشتاری مرتبط با یک برچسب اسکریپت خاص را از مجموعهٔ فونت‌های تم حذف می‌کند.

```csharp
public void RemoveScriptFont(string script)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| script | String | کد اسکریپت BCP-47 که تنظیم فونت آن باید حذف شود. |

### مثال‌ها

این مثال نشان می‌دهد چگونه نگاشت فونت برای اسکریپت عبری را حذف کنید:

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.RemoveScriptFont("Hebr");
```

## Fonts.SetScriptFont method

یک نام قلم را به یک برچسب اسکریپت خاص اختصاص می‌دهد که تعیین می‌کند متن آن اسکریپت چگونه در ارائه رندر شود.

```csharp
public void SetScriptFont(string script, string fontName)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| script | String | کد اسکریپت BCP-47 (مانند "Arab", "Hebr", "Hans") که سیستم نوشتاری را شناسایی می‌کند. |
| fontName | String | نام قلمی که به اسکریپت مشخص شده اختصاص داده می‌شود. |

### مثال‌ها

این مثال نشان می‌دهد چگونه قلم اسکریپت عربی به "Segoe UI" تنظیم شود:

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segoe UI");
```

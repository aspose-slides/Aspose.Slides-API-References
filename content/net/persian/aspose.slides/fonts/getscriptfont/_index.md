## Fonts.GetScriptFont متد

نام فونتی که به یک برچسب اسکریپت خاص در تم ارائه مرتبط است را دریافت می‌کند.

```csharp
public string GetScriptFont(string script)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| script | String | کد اسکریپت BCP-47 (مانند "Latn"، "Cyrl"، "Jpan") که برای شناسایی یک سیستم نوشتاری استفاده می‌شود. |

### مقدار بازگشت

نام فونت مورد استفاده برای اسکریپت مشخص‌شده، یا `null` اگر اسکریپت تعریف نشده باشد.

### مثال‌ها

این مثال نشان می‌دهد که چگونه فونت اختصاص داده‌شده به اسکریپت سیریلیک در تم ارائه دریافت شود.

```csharp
[C#]
string font = presentation.MasterTheme.FontScheme.Major.GetScriptFont("Cyrl");
Console.WriteLine("Font for Cyrillic script: " + font);
```

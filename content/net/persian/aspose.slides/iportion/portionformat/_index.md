## IPortion.PortionFormat ویژگی

شیء قالب‌بندی را برمی‌گرداند که شامل ویژگی‌های قالب‌بندی صریحاً تنظیم‌شده برای قسمت متن است و هیچ وراثت‌گیری‌ای اعمال نمی‌شود. فقط‌خواندنی [`IPortionFormat`](../../iportionformat).

```csharp
public IPortionFormat PortionFormat { get; }
```

### یادداشت

شیء قالب‌بندی فقط پارامترهای قالب‌بندی تعریف‌شده برای قسمت فعلی را شامل می‌شود و داده‌های وراثت‌گیری‌شده اعمال نمی‌گردند.

برای به دست آوردن مقادیر مؤثر شامل مقادیر وراثت‌گیری‌شده از روش [`GetEffective`](../../iportionformat/geteffective) استفاده کنید.

## IShape.RawFrame ویژگی

مشخصات چارچوب خام شکل را بر می‌گرداند یا تنظیم می‌کند. خواندنی/نوشتنی [`IShapeFrame`](../../ishapeframe).

```csharp
public IShapeFrame RawFrame { get; set; }
```

### مثال‌ها

کدی که سعی می‌کند چارچوب تعریف‌نشده‌ای را به IShape.Frame اختصاص دهد، در حالت کلی منطقی نیست (به‌ویژه در حالتی که GroupShape والد به‌صورت چندین‌گانه در داخل GroupShape‌های دیگر تو در تو باشد). برای مثال:

```csharp
IShape shape = ...;
shape.Frame = new ShapeFrame(float.NaN, float.NaN, float.NaN, float.NaN, NullableBool.NotDefined, NullableBool.NotDefined, float.NaN);
```

یا

```csharp
slide.Shapes.AddAutoShape(ShapeType.RoundCornerRectangle, float.NaN, float.NaN, float.NaN, float.NaN);
```

چنین کدی می‌تواند منجر به وضعیت‌های نامشخص شود. بنابراین محدودیت‌هایی برای استفاده از مقادیر تعریف‌نشده در IShape.Frame اضافه شده است. مقادیر x، y، width، height، flipH، flipV و rotationAngle باید تعریف شوند (نه float.NaN و نه NullableBool.NotDefined). کد مثال فوق اکنون استثنای ArgumentException را پرتاب می‌کند. این محدودیت‌ها به موارد استفاده زیر اعمال می‌شود:

```csharp
IShape shape = ...;
shape.Frame = ...; // نمی‌تواند نامشخص باشد

IShapeCollection shapes = ...;
// پارامترهای x، y، width، height نمی‌توانند float.NaN باشند:
{
    shapes.AddAudioFrameCD(...);
    shapes.AddAudioFrameEmbedded(...);
    shapes.AddAudioFrameLinked(...);
    shapes.AddAutoShape(...);
    shapes.AddChart(...);
    shapes.AddConnector(...);
    shapes.AddOleObjectFrame(...);
    shapes.AddPictureFrame(...);
    shapes.AddSmartArt(...);
    shapes.AddTable(...);
    shapes.AddVideoFrame(...);
    shapes.InsertAudioFrameEmbedded(...);
    shapes.InsertAudioFrameLinked(...);
    shapes.InsertAutoShape(...);
    shapes.InsertChart(...);
    shapes.InsertConnector(...);
    shapes.InsertOleObjectFrame(...);
    shapes.InsertPictureFrame(...);
    shapes.InsertTable(...);
    shapes.InsertVideoFrame(...);
}
```

اما ویژگی‌های چارچوب IShape.RawFrame می‌توانند تعریف‌نشده باشند. این منطقی است وقتی شکل به یک placeholder پیوند خورده باشد. در این صورت مقادیر چارچوب شکل تعریف‌نشده از شکل placeholder والد جایگزین می‌شوند. اگر برای آن شکل placeholder والد وجود نداشته باشد، شکل از مقادیر پیش‌فرض هنگام ارزیابی چارچوب مؤثر بر پایه IShape.RawFrame استفاده می‌کند. مقادیر پیش‌فرض برای x، y، width، height، flipH، flipV و rotationAngle برابر 0 و NullableBool.False هستند. برای مثال:

```csharp
IShape shape = ...; // شکل به placeholder پیوند خورده است
shape.RawFrame = new ShapeFrame(float.NaN, float.NaN, 100, float.NaN, NullableBool.NotDefined, NullableBool.NotDefined, 0); // اکنون shape مقادیر x، y، height، flipH، flipV را از placeholder به ارث می‌برد و width=100 و rotationAngle=0 را بازنویسی می‌کند.
```

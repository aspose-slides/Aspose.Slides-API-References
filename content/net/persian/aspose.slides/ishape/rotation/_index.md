## IShape.Rotation خصوصیت

مقدار یا تنظیم می‌کند تعداد درجه‌هایی که شکل مشخص‌شده حول محور z چرخانده می‌شود. مقدار مثبت نشان‌دهنده چرخش ساعت‌گرد است؛ مقدار منفی نشان‌دهنده چرخش پادساعت‌گرد است. قابل‌خواندن/قابل‌نوشتن Single.

```csharp
public float Rotation { get; set; }
```

### توضیحات

مقدار بازگردانده‌شده همیشه تعریف‌شده است (float.NaN نیست). مقدار تخصیص‌داده‌شده باید تعریف‌شده باشد (float.NaN نیست). می‌توانید مقادیر تعریف‌نشده را برای ویژگی‌های نمونه RawFrame تنظیم کنید.

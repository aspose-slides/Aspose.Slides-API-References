## TextFrame.SplitTextByColumns متد

محتوای متنی [`ITextFrame`](../../itextframe) را به یک آرایه‌ای از رشته‌ها تقسیم می‌کند که هر عنصر آن به یک ستون متن جداگانه درون فریم مربوط می‌شود.

```csharp
public string[] SplitTextByColumns()
```

### مقدار بازگشت

آرایه‌ای از رشته‌ها که هر رشته محتوای متنی یک ستون خاص در [`ITextFrame`](../../itextframe) را نشان می‌دهد.

### توضیحات

اگر فریم متن شامل چندین ستون نباشد، آرایهٔ برگردانده‌شده یک عنصر تک شامل متن کامل خواهد داشت. ستون‌های خالی به عنوان رشته‌های خالی در آرایه نشان داده می‌شوند.

### مثال‌ها

مثال زیر نشان می‌دهد چگونه می‌توان از `SplitTextByColumns` استفاده کرد:

```csharp
using (Presentation pres = new Presentation("example.pptx"))
{
    // دریافت اولین شکل روی اسلاید و تبدیل آن به ITextFrame
    ITextFrame textFrame = pres.Slides[0].Shapes[0] as ITextFrame;
    // تقسیم محتوای فریم متن به ستون‌ها
    string[] columnsText = textFrame.SplitTextByColumns();
    // چاپ متن هر ستون در کنسول
    foreach (string column in columnsText)
        Console.WriteLine(column);
}
```

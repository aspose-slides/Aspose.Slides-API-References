## TextFrame.ReplaceRegex متد

تمام موارد منطبق با عبارت منظم را با رشتهٔ مشخص شده جایگزین می‌کند.

```csharp
public void ReplaceRegex(Regex regex, string newText, IFindResultCallback callback)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| regex | Regex | عبارت منظم Regex برای دریافت رشته‌های قابل جایگزینی. |
| newText | String | رشته‌ای که تمام موارد رشته‌های قابل جایگزینی را جایگزین می‌کند. |
| callback | IFindResultCallback | شیٔ Callback برای ذخیره نتایج عملیات جایگزینی [`IFindResultCallback`](../../ifindresultcallback). |

### مثال‌ها

کد نمونهٔ زیر نشان می‌دهد چگونه متن را با استفاده از عبارت منظم و رشتهٔ مشخص شده جایگزین کنیم.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx")){
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// تمام کلمات دارای 10 یا بیشتر کاراکتر را با '***' جایگزین کنید
	((AutoShape)presentation.Slides[0].Shapes[0]).TextFrame.ReplaceRegex(regex, "***", null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

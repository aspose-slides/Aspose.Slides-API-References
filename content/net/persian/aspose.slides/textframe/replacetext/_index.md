## TextFrame.ReplaceText متد

تمام رخدادهای متن مشخص‌شده را با متن دیگری جایگزین می‌کند.

```csharp
public void ReplaceText(string oldText, string newText, ITextSearchOptions options, 
    IFindResultCallback callback)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| oldText | String | رشته‌ای که باید جایگزین شود. |
| newText | String | رشته‌ای که تمام رخدادهای oldText را جایگزین می‌کند. |
| options | ITextSearchOptions | گزینه‌های جستجوی متن [`ITextSearchOptions`](../../itextsearchoptions). |
| callback | IFindResultCallback | شیء بازگردانی برای ذخیره نتایج عملیات جایگزینی [`IFindResultCallback`](../../ifindresultcallback). |

### مثال‌ها

کد نمونه زیر نشان می‌دهد که چگونه یک رشته مشخص را با رشتهٔ دیگری جایگزین کنید.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx")){
	// تمام موارد جداگانه‌ی 'the' را با '***' جایگزین کنید
	((AutoShape)presentation.Slides[0].Shapes[0]).TextFrame.ReplaceText("the", "***", new TextSearchOptions()
	{ WholeWordsOnly = true }, null);
	presentation.Save("SomePresentation-out2.pptx", SaveFormat.Pptx);
}
```

## WriteAsSvg(Stream, ISVGOptions) {#writeassvg_1}

محتوای اسلاید را به صورت یک فایل SVG ذخیره می‌کند.

```csharp
public void WriteAsSvg(Stream stream, ISVGOptions svgOptions)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| stream | Stream | جریان هدف |
| svgOptions | ISVGOptions | گزینه‌های تولید SVG |

### مثال‌ها

مثال کد زیر نشان می‌دهد که چگونه اسلاید اول یک ارائه PowerPoint را با گزینه‌ها به یک فایل SVG تبدیل کنیم.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.svg"))
    {
        var options = new SVGOptions() { VectorizeText = true };
        // اسلاید اول را به عنوان یک فایل SVG ذخیره می‌کند
        pres.Slides[0].WriteAsSvg(fileStream, options);
    }
}
```

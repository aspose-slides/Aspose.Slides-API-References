### مثال‌ها

کد نمونه زیر نشان می‌دهد که چگونه اولین اسلاید یک ارائه PowerPoint را به یک متافایل تبدیل کنید.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.emf"))
    {
        // اسلاید اول را به عنوان یک متافایل ذخیره می‌کند
        pres.Slides[0].WriteAsEmf(fileStream);
    }
}
```

## ویژگی ViewProperties.GridSpacing

مقدار یا تنظیم فاصلهٔ شبکه‌ای که باید برای شبکهٔ زیرین سند ارائه استفاده شود، بر حسب نقطه. خواندنی/نوشتنی Single.

```csharp
public float GridSpacing { get; set; }
```

### ملاحظات

مقدار فاصلهٔ شبکه باید یک عدد مثبت باشد. محدودهٔ معمولی مقادیر از 1 میلی‌متر (2.8349607 نقطه) تا 2 اینچ (144 نقطه) است.

### مثال‌ها

کد نمونه زیر نشان می‌دهد چگونه فاصلهٔ شبکه را در یک ارائهٔ PowerPoint تغییر دهیم.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

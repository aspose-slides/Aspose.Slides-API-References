## IBlobManagementOptions.PresentationLockingBehavior ویژگی

این ویژگی تعیین می‌کند که آیا یک نمونه از کلاس Presentation می‌تواند صاحب منبع – فایل یا جریان – در طول زمان حیات نمونه باشد. اگر نمونه صاحب باشد، منبع را قفل می‌کند. این کار به بهبود مصرف حافظه و عملکرد هنگام کار با BLOBها کمک می‌کند، اما منبع (جریان یا فایل) در طول زمان حیات نمونه Presentation قابل تغییر نیست. مثال زیر:

```csharp
public PresentationLockingBehavior PresentationLockingBehavior { get; set; }
```

### مثال‌ها

```csharp
LoadOptions loadOptions = new LoadOptions { 
  BlobManagementOptions = {PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked} };
using (Presentation pres = new Presentation("pres.pptx", loadOptions)) {
  // یک IOException پرتاب می‌شود زیرا pres.pptx برای طول عمر یک Presentation قفل شده است
  // File.Delete("pres.pptx");
}
// پس از این که شیء Presentation آزاد شد، فایل قفل‌برداری می‌شود و می‌تواند حذف شود
File.Delete("pres.pptx");
```

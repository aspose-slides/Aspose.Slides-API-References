یک قانون FallBack جدید را به انتهای مجموعه اضافه می‌کند.

| پارامتر | نوع | توضیح |

### مثال‌ها

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    // دریافت مجموعهٔ قوانین خالی یا پیش‌راه‌اندازی شده از FontsManager
```

## ITextFrame.HighlightRegex متد

تمام مطابقت‌های عبارت منظم را با رنگ مشخص شده برجسته می‌کند.

```csharp
public void HighlightRegex(Regex regex, Color highlightColor, IFindResultCallback callback)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| regex | Regex | عبارت منظم Regex برای دریافت رشته‌های مورد برجسته. |
| highlightColor | Color | رنگ برای برجسته‌کردن متن. |
| callback | IFindResultCallback | شیء callback برای دریافت نتایج جستجو [`IFindResultCallback`](../../ifindresultcallback). |

### مثال‌ها

نمونه کد زیر نشان می‌دهد چگونه می‌توان متن را در یک TextFrame با استفاده از یک عبارت منظم برجسته کرد.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// برجسته‌کردن تمام کلمات با 10 یا بیشتر کاراکتر
	((AutoShape)presentation.Slides[0].Shapes[0]).TextFrame.HighlightRegex(regex, Color.Blue, null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

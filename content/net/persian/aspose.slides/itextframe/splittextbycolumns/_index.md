## ITextFrame.SplitTextByColumns متد

متن محتویات [`ITextFrame`](../../itextframe) را به یک آرایه از رشته‌ها تقسیم می‌کند که هر عنصر متن یک ستون متن جداگانه در داخل قاب را نشان می‌دهد.

```csharp
public string[] SplitTextByColumns()
```

### Return Value

یک آرایه از رشته‌ها که هر رشته محتویات متنی یک ستون خاص در [`ITextFrame`](../../itextframe) را نشان می‌دهد.

### Remarks

اگر قاب متن شامل چندین ستون نباشد، آرایهٔ بازگشتی یک عنصر واحد خواهد داشت که شامل متن کامل است. ستون‌های خالی به عنوان رشته‌های خالی در آرایه نمایش داده می‌شوند.

### Examples

مثال زیر نحوهٔ استفاده از `SplitTextByColumns` را نشان می‌دهد:

```csharp
using (Presentation pres = new Presentation("example.pptx"))
{
    // دریافت اولین شکل در اسلاید و تبدیل آن به ITextFrame
    ITextFrame textFrame = pres.Slides[0].Shapes[0] as ITextFrame;
    // تقسیم محتویات قاب متن به ستون‌ها
    string[] columnsText = textFrame.SplitTextByColumns();
    // چاپ متن هر ستون در کنسول
    foreach (string column in columnsText)
        Console.WriteLine(column);
}
```

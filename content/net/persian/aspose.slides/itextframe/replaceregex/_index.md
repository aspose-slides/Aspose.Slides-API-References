## ITextFrame.ReplaceRegex متد

تمام مطابقت‌های عبارات منظم را با رشته مشخص شده جایگزین می‌کند.

```csharp
public void ReplaceRegex(Regex regex, string newText, IFindResultCallback callback)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| regex | Regex | عبارت منظم Regex برای دریافت رشته‌هایی که باید جایگزین شوند. |
| newText | String | رشته‌ای برای جایگزینی تمام موارد رشته‌هایی که باید جایگزین شوند. |
| callback | IFindResultCallback | شیء callback برای دریافت نتایج جستجو [`IFindResultCallback`](../../ifindresultcallback). |

### مثال‌ها

نمونه کد زیر نشان می‌دهد چگونه متن را با استفاده از عبارت منظم و رشته مشخص شده جایگزین کنید.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// تمام کلماتی که 10 یا بیشتر کاراکتر دارند را با '***' جایگزین کنید
	((AutoShape)presentation.Slides[0].Shapes[0]).TextFrame.ReplaceRegex(regex, "***", null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

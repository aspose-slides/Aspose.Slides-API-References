## خاصیت ProtectionManager.IsOnlyDocumentPropertiesLoaded

این خاصیت زمانی معنا دارد که فایل ارائه با رمز عبور محافظت شده باشد و خواص سند این فایل عمومی باشند. مقدار true یعنی فقط خواص سند از یک فایل ارائهٔ رمزگذاری شده بدون استفاده از رمز عبور بارگذاری می‌شود. مقدار false یعنی کل ارائهٔ رمزگذاری شده با استفاده از رمز صحیح بارگذاری می‌شود و نه فقط خواص سند. اگر ارائه رمزگذاری نشده باشد مقدار خاصیت همیشه false است. اگر خواص سند یک فایل رمزگذاری شده عمومی نباشند مقدار خاصیت همیشه false است. اگر Presentation.EncryptDocumentProperties برابر true باشد، مقدار خاصیت IsOnlyDocumentPropertiesLoaded همیشه false است. Boolean فقط‌خواندنی.

```csharp
public bool IsOnlyDocumentPropertiesLoaded { get; }
```

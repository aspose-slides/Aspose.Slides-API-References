## ProtectionManager.SetWriteProtection متد

محافظت نوشتاری را برای این ارائه با گذرواژه‌ی مشخص شده تنظیم می‌کند.

```csharp
public void SetWriteProtection(string password)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| password | String | گذرواژه. |

### مثال‌ها

کد نمونه زیر نشان می‌دهد که چگونه محافظت نوشتاری را برای یک ارائه تنظیم کنید.

```csharp
[C#]
using (Presentation presentation = new Presentation("pres.pptx"))
{
    presentation.ProtectionManager.SetWriteProtection("123123");
    presentation.Save("write-protected-pres.pptx", SaveFormat.Pptx);
}
```

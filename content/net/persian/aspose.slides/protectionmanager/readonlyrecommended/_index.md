## ProtectionManager.ReadOnlyRecommended ویژگی

دسترس فقط-خواندنی یا نوشتن را دریافت یا تنظیم می‌کند. Boolean قابل خواندن/نوشتن.

```csharp
public bool ReadOnlyRecommended { get; set; }
```

### مثال‌ها

کد نمونه زیر نشان می‌دهد چگونه یک ارائه PowerPoint را به حالت فقط-خواندنی در C# با استفاده از Aspose.Slides تنظیم کنید.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	pres.ProtectionManager.ReadOnlyRecommended = true;
	pres.Save("ReadOnlyPresentation.pptx", SaveFormat.Pptx);
}
```

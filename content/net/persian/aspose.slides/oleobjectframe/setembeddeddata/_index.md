## OleObjectFrame.SetEmbeddedData متد

اطلاعات مربوط به داده‌های توکار OLE را تنظیم می‌کند. این متد ویژگی‌های شیء را طوری تغییر می‌دهد که داده‌های جدید را منعکس کند و پرچم IsObjectLink را به مقدار false تنظیم می‌کند، که نشان می‌دهد شیء OLE توکار است.

```csharp
public void SetEmbeddedData(IOleEmbeddedDataInfo embeddedData)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| embeddedData | IOleEmbeddedDataInfo | دادهٔ توکار [`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo) |

### استثناها

| استثنا | شرط |
| --- | --- |
| ArgumentNullException | وقتی پارامتر embeddedData برابر null باشد. |

### مثال‌ها

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    OleObjectFrame oof = pres.Slides[0].Shapes[0] as OleObjectFrame;
    if (oof != null)
    {
        IOleEmbeddedDataInfo newData = new OleEmbeddedDataInfo(File.ReadAllBytes("Picture.png"), "png");
        oof.SetEmbeddedData(newData);
    }
}
```

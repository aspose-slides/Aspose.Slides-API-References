## OleObjectFrame.LinkPathRelative ویژگی

در صورت وجود، مسیر نسبی به فایل پیوند داده شده را برمی‌گرداند، در غیر این صورت یک رشته خالی را برمی‌گرداند. فقط-خواندنی String.

```csharp
public string LinkPathRelative { get; }
```

### توضیحات

در ارائه‌های Ppt، برخی پیوندهای شیء Ole ممکن است نمایش نسبی داشته باشند.

### مثال‌ها

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.ppt"))
{
    IOleObjectFrame oleFrame = presentation.Slides[0].Shapes[0] as IOleObjectFrame;

    if (oleFrame != null)
    {
        Console.WriteLine("The relative path: " + oleFrame.LinkPathRelative);
    } 
}
```

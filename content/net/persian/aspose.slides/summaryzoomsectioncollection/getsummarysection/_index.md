## SummaryZoomSectionCollection.GetSummarySection متد

برمی‌گرداند عنصر Summary Zoom Section برای بخش داده شده.

```csharp
public ISummaryZoomSection GetSummarySection(ISection section)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| section | ISection | بخشی که باید یافت شود [`ISection`](../../isection) |

### مقدار بازگشتی

[`ISummaryZoomSection`](../../isummaryzoomsection) یا null اگر مجموعه عنصری برای بخش وجود نداشته باشد.

### مثال‌ها

این مثال دریافت عنصر Summary Zoom Section را بر اساس شاخص نشان می‌دهد:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    ISummaryZoomFrame zoomFrame = (ISummaryZoomFrame) pres.Slides[1].Shapes[0];
    ISummaryZoomSectionCollection collection = zoomFrame.SummaryZoomCollection;
    ISummaryZoomSection selectedObject = collection.GetSummarySection(pres.Sections[2]);
}
```

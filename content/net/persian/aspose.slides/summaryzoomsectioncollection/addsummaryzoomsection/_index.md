## SummaryZoomSectionCollection.AddSummaryZoomSection متد

یک شیء جدید Summary Zoom Section ایجاد می‌کند و آن را به مجموعه اضافه می‌نماید

```csharp
public ISummaryZoomSection AddSummaryZoomSection(ISection section)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| section | ISection | بخش برای یک عنصر جدید Summary Zoom Section [`ISection`](../../isection) |

### مقدار بازگشتی

عنصر اضافه‌شده [`ISummaryZoomFrame`](../../isummaryzoomframe)

### توضیحات

اگر عنصری برای این بخش قبلاً در مجموعه موجود باشد، عنصر موجود بازگردانده می‌شود.

### مثال‌ها

این مثال دریافت عنصر Summary Zoom Section بر اساس شاخص را نشان می‌دهد:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    ISummaryZoomFrame zoomFrame = (ISummaryZoomFrame) pres.Slides[1].Shapes[0];
    ISummaryZoomSectionCollection collection = zoomFrame.SummaryZoomCollection;
    ISummaryZoomSection newZoomSection = collection.AddSummaryZoomSection(pres.Sections[3]);
}
```

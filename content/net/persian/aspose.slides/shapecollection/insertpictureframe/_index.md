## ShapeCollection.InsertPictureFrame متد

یک قاب تصویر جدید که حاوی تصویر مشخص شده است ایجاد می‌کند و آن را در مجموعه شکل‌ها در ایندکس مشخص شده وارد می‌سازد.

```csharp
public IPictureFrame InsertPictureFrame(int index, ShapeType shapeType, float x, float y, 
    float width, float height, IPPImage image)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| index | Int32 | اندیس صفر مبنا که برای وارد کردن قاب تصویر استفاده می‌شود. |
| shapeType | ShapeType | نوع شکل موجود در [`ShapeType`](../../shapetype) را مشخص می‌کند، به‌جز تمام انواع خطوط: |
| x | Single | مختصات x قاب تصویر، به واحد نقطه. |
| y | Single | مختصات y قاب تصویر، به واحد نقطه. |
| width | Single | عرض قاب تصویر، به واحد نقطه. |
| height | Single | ارتفاع قاب تصویر، به واحد نقطه. |
| image | IPPImage | [`IPPImage`](../../ippimage) برای نمایش در قاب تصویر. |

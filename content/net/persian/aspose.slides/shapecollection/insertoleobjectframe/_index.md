## InsertOleObjectFrame(int, float, float, float, float, IOleEmbeddedDataInfo) {#insertoleobjectframe}

یک فریم شیء OLE جدید ایجاد می‌کند و آن را در مجموعه شکل‌ها در شاخص مشخص شده درج می‌نماید.

```csharp
public IOleObjectFrame InsertOleObjectFrame(int index, float x, float y, float width, float height, 
    IOleEmbeddedDataInfo dataInfo)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| index | Int32 | اندکس صفر مبنا که فریم شیء OLE در آن درج می‌شود. |
| x | Single | مختصات x فریم OLE جدید، بر حسب نقاط. |
| y | Single | مختصات y فریم OLE جدید، بر حسب نقاط. |
| width | Single | عرض فریم OLE جدید، بر حسب نقاط. |
| height | Single | ارتفاع فریم OLE جدید، بر حسب نقاط. |
| dataInfo | IOleEmbeddedDataInfo | اطلاعات دادهٔ توکار OLE ([`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo)). |

### مقدار بازگشتی

[`IOleObjectFrame`](../../ioleobjectframe) جدید ایجاد شده.

### مثال‌ها

این مثال نحوه درج یک شیء OLE در شاخص دوم را نشان می‌دهد:

```csharp
[C#]
byte[] fileData = File.ReadAllBytes("test.zip");
IOleDataInfo dataInfo = new OleDataInfo(fileData, "zip");
IOleObjectFrame oleObjectFrame = slidees.Shapes.AddOleObjectFrame(2, 150, 20, 50, 50, dataInfo);
```

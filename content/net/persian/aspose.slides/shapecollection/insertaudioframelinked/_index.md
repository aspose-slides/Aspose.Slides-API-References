## ShapeCollection.InsertAudioFrameLinked متد

```csharp
public IAudioFrame InsertAudioFrameLinked(int index, float x, float y, float width, float height, 
    string fname)
```

| index | Int32 | اندیس صفر مبنا که فریم صوتی در آن قرار می‌گیرد. |

| height | Single | ارتفاع فریم صوتی جدید، بر حسب نقاط. |

## AddAudioFrameEmbedded(float, float, float, float, Stream) {#addaudioframeembedded_1}

یک فریم صوتی جدید با فایل WAV جاسازی‌شده ایجاد می‌کند و آن را به انتهای مجموعهٔ اشکال اضافه می‌کند. صوت جاسازی‌شده به مجموعهٔ Presentation.Audios اضافه می‌شود.

```csharp
public IAudioFrame AddAudioFrameEmbedded(float x, float y, float width, float height, 
    Stream audio_stream)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| x | Single | مقدار x فریم صوتی جدید، بر حسب نقطه. |
| y | Single | مقدار y فریم صوتی جدید، بر حسب نقطه. |
| width | Single | عرض فریم صوتی جدید، بر حسب نقطه. |
| height | Single | ارتفاع فریم صوتی جدید، بر حسب نقطه. |
| audio_stream | Stream | یک جریان ورودی که شامل داده‌های صوتی WAV برای جاسازی است. |

### مقدار بازگشت

[`IAudioFrame`](../../iaudioframe) تازه ایجاد شده.

### مثال‌ها

مثال‌های زیر نشان می‌دهند چگونه فریم صوتی ایجاد کنید.

```csharp
[C#]
// یک شی از کلاس ارائه ایجاد می‌کند که نمایانگر یک فایل ارائه است
using (Presentation pres = new Presentation())
{
    // اسلاید اول را دریافت می‌کند
    ISlide sld = pres.Slides[0];
    // فایل صوتی wav را به جریان (stream) بارگذاری می‌کند
    FileStream fstr = new FileStream("sampleaudio.wav", FileMode.Open, FileAccess.Read);
    // فریم صوتی را اضافه می‌کند
    IAudioFrame audioFrame = sld.Shapes.AddAudioFrameEmbedded(50, 150, 100, 100, fstr);
    // حالت پخش و حجم صدا را تنظیم می‌کند
    audioFrame.PlayMode = AudioPlayModePreset.Auto;
    audioFrame.Volume = AudioVolumeMode.Loud;
    // فایل PowerPoint را بر روی دیسک می‌نویسد
    pres.Save("AudioFrameEmbed_out.pptx", SaveFormat.Pptx);
}
```

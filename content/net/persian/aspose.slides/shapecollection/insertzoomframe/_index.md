## InsertZoomFrame(int, float, float, float, float, ISlide) {#insertzoomframe}

یک قاب Zoom جدید ایجاد می‌کند و آن را در مجموعهٔ اشکال در ایندکس مشخص‌شده وارد می‌کند.

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| index | Int32 | ایندکس مبتنی بر صفر که قاب Zoom باید در آن وارد شود. |
| x | Single | مختصات x قاب Zoom جدید، بر حسب نقطه. |
| y | Single | مختصات y قاب Zoom جدید، بر حسب نقطه. |
| width | Single | عرض قاب Zoom جدید، بر حسب نقطه. |
| height | Single | ارتفاع قاب Zoom جدید، بر حسب نقطه. |
| slide | ISlide | [`ISlide`](../../islide) که توسط قاب Zoom ارجاع شده است. |

### مقدار بازگشت

این مثال نشان‌دهندهٔ ایجاد و وارد کردن یک شی Zoom در ایندکس مشخص‌شدهٔ یک مجموعه است (فرض کنید حداقل دو اسلاید در ارائهٔ "Presentation.pptx" وجود دارد):

}

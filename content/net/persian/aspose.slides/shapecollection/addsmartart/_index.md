## ShapeCollection.AddSmartArt متد

یک نمودار SmartArt ایجاد می‌کند و آن را به انتهای مجموعهٔ شکل‌ها اضافه می‌سازد.

```csharp
public ISmartArt AddSmartArt(float x, float y, float width, float height, 
    SmartArtLayoutType layoutType)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| x | Single | مختصات x فریم نمودار، به پوینت. |
| y | Single | مختصات y فریم نمودار، به پوینت. |
| width | Single | عرض فریم نمودار، به پوینت. |
| height | Single | ارتفاع فریم نمودار، به پوینت. |
| layoutType | SmartArtLayoutType | نوع چیدمان SmartArt. |

### مقدار بازگشتی

آبجکت جدید ایجاد شده [`ISmartArt`](../../../aspose.slides.smartart/ismartart).

### مثال‌ها

مثال زیر نشان می‌دهد چگونه شکل هوشمند را در ارائهٔ پاورپوینت اضافه کنیم.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	Slide slide = pres.Slides[0];
	SmartArt smart = slide.Shapes.AddSmartArt(0, 0, 400, 400, SmartArtLayoutType.BasicBlockList);
}
```

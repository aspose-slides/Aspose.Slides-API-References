## InsertConnector(int, ShapeType, float, float, float, float, bool) {#insertconnector_1}

یک شکل اتصال‌گر جدید ایجاد می‌کند و آن را در مجموعهٔ اشکال در ایندکس مشخص‌شده قرار می‌دهد؛ در صورت تمایل سبک قالب پیش‌فرض را اعمال می‌کند.

```csharp
public IConnector InsertConnector(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| index | Int32 | اندیس صفر-مبنایی که شکل اتصال‌گر در آن درج می‌شود. |
| shapeType | ShapeType | [`ShapeType`](../../shapetype) شکل اتصال‌گر برای درج. |
| x | Single | مختصات x چارچوب اتصال‌گر، به نقطه. |
| y | Single | مختصات y چارچوب اتصال‌گر، به نقطه. |
| width | Single | عرض چارچوب اتصال‌گر، به نقطه. |
| height | Single | ارتفاع چارچوب اتصال‌گر، به نقطه. |
| createFromTemplate | Boolean | True برای اعمال قالب پیش‌فرض (نام غیر خالی، سبک ساده)؛ false برای ایجاد اتصال‌گر با مقادیر پیش‌فرض ویژگی‌ها. |

## AddAutoShape(ShapeType, float, float, float, float, bool) {#addautoshape_1}

یک شکل خودکار جدید ایجاد می‌کند و آن را به انتهای مجموعهٔ اشکال اضافه می‌نماید، به‌طور اختیاری با قالب‌بندی پیش‌فرض الگو مقداردهی اولیه می‌کند.

```csharp
public IAutoShape AddAutoShape(ShapeType shapeType, float x, float y, float width, float height, 
    bool createFromTemplate)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| shapeType | ShapeType | [`ShapeType`](../../shapetype) شکل خودکار برای افزودن. |
| x | Single | مختصات x چارچوب شکل، به واحد نقطه. |
| y | Single | مختصات y چارچوب شکل، به واحد نقطه. |
| width | Single | عرض چارچوب شکل، به واحد نقطه. |
| height | Single | ارتفاع چارچوب شکل، به واحد نقطه. |
| createFromTemplate | Boolean | True برای اعمال استایل پیش‌فرض الگو (استایل ساده، متن مرکز‌گذاری شده، و نام غیر خالی) به شکل جدید؛ false برای ایجاد شکل با تمام ویژگی‌ها تنظیم‌شده به مقادیر پیش‌فرض. |

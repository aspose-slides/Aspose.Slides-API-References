## AddZoomFrame(float, float, float, float, ISlide, IPPImage) {#addzoomframe_1}

public IZoomFrame AddZoomFrame(float x, float y, float width, float height, ISlide slide,

| Parameter | Type | Description |

| x | Single | مختصات x فریم زوم جدید، بر حسب نقطه. |
| y | Single | مختصات y فریم زوم جدید، بر حسب نقطه. |
| width | Single | عرض فریم زوم جدید، بر حسب نقطه. |

### Examples

این مثال نحوه افزودن یک شیء Zoom به انتهای یک مجموعه را نشان می‌دهد (فرض کنید حداقل دو اسلاید در ارائهٔ «Presentation.pptx» وجود داشته باشد):

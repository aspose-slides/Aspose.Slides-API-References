## InsertSectionZoomFrame(int, float, float, float, float, ISection) {#insertsectionzoomframe}

یک فریم Section Zoom جدید ایجاد می‌کند و آن را در مجموعه شکل‌ها در شاخص مشخص شده درج می‌نماید.

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| index | Int32 | شاخص صفرپایه‌ای که فریم Section Zoom در آن درج می‌شود. |
| x | Single | مختصات x فریم Section Zoom جدید، بر حسب پوینت. |
| y | Single | مختصات y فریم Section Zoom جدید، بر حسب پوینت. |
| width | Single | عرض فریم Section Zoom جدید، بر حسب پوینت. |
| height | Single | ارتفاع فریم Section Zoom جدید، بر حسب پوینت. |
| section | ISection | [`ISection`](../../isection) ارجاع داده شده توسط فریم Section Zoom؛ باید متعلق به این ارائه باشد و حداقل یک اسلاید داشته باشد. |

### مقدار بازگشت

شیء [`ISectionZoomFrame`](../../isectionzoomframe) جدید ایجاد شده.

### مثال‌ها

این مثال نحوه ایجاد و درج یک شیء Section Zoom را در شاخص مشخص شده یک مجموعه نشان می‌دهد (فرض کنید در ارائه "Presentation.pptx" حداقل دو بخش موجود باشد):

```csharp

}
```

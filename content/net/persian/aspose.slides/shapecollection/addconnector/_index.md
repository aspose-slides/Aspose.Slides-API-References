## AddConnector(ShapeType, float, float, float, float) {#addconnector}

یک شکل اتصال جدید با استایل پیش‌فرض قالب ایجاد می‌کند و آن را به انتهای مجموعهٔ اشکال اضافه می‌دارد.

### مثال‌ها

مثال زیر نشان می‌دهد چگونه یک اتصال (یک اتصال خمیده) بین دو شکل (یک بیضی و یک مستطیل) در ارائهٔ PowerPoint اضافه شود.

```csharp
[C#]
// یک شیء از کلاس Presentation که نمایانگر یک فایل PPTX است را نمونه‌سازی می‌کند
using (Presentation input = new Presentation())
{
    // به مجموعهٔ اشکال اسلاید خاص دسترسی پیدا می‌کند
    IShapeCollection shapes = input.Slides[0].Shapes;
    // یک شکل خودکار بیضی اضافه می‌کند
    IAutoShape ellipse = shapes.AddAutoShape(ShapeType.Ellipse, 0, 100, 100, 100);
    // یک شکل خودکار مستطیل اضافه می‌کند
    IAutoShape rectangle = shapes.AddAutoShape(ShapeType.Rectangle, 100, 300, 100, 100);
    // یک شکل اتصال را به مجموعهٔ اشکال اسلاید اضافه می‌کند
    IConnector connector = shapes.AddConnector(ShapeType.BentConnector2, 0, 0, 10, 10);
    // با استفاده از اتصال، اشکال را به هم متصل می‌کند
    connector.StartShapeConnectedTo = ellipse;
    connector.EndShapeConnectedTo = rectangle;
    // متد Reroute را فراخوانی می‌کند که مسیر کوتاه‌ترین خودکار بین اشکال را تنظیم می‌کند
    connector.Reroute();
    // ارائه را ذخیره می‌کند
    input.Save("Shapes-connector.pptx", SaveFormat.Pptx);
}
```

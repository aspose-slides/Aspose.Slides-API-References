## AddSectionZoomFrame(float, float, float, float, ISection) {#addsectionzoomframe}

یک فریم Section Zoom جدید ایجاد می‌کند و آن را به انتهای مجموعهٔ شکل‌ها اضافه می‌نماید.

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section)
```

| Parameter | Type | Description |
| --- | --- | --- |
| x | Single | مختصات x فریم Section Zoom جدید، به نقطه. |
| y | Single | مختصات y فریم Section Zoom جدید، به نقطه. |
| width | Single | عرض فریم Section Zoom جدید، به نقطه. |
| height | Single | ارتفاع فریم Section Zoom جدید، به نقطه. |
| section | ISection | [`ISection`](../../isection) ارجاع‌شده توسط فریم Section Zoom؛ باید متعلق به این ارائه باشد و حداقل یک اسلاید داشته باشد. |

### Return Value

فریم جدید ایجاد شده [`ISectionZoomFrame`](../../isectionzoomframe).

### Exceptions

| exception | condition |
| --- | --- |
| ArgumentException | اگر بخش ارجاع‌شده به ارائهٔ جاری تعلق نداشته باشد یا اسلایدی نداشته باشد، پرتاب می‌شود. |

### Examples

این مثال افزودن یک شیء Section Zoom به انتهای یک مجموعه را نشان می‌دهد (فرض کنید که در ارائهٔ "Presentation.pptx" حداقل دو بخش وجود دارد):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1]);
}
```

## AddSectionZoomFrame(float, float, float, float, ISection, IPPImage) {#addsectionzoomframe_1}

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section, IPPImage image)
```

| Parameter | Type | Description |


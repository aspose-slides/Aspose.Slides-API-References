## ShapeCollection.InsertSummaryZoomFrame متد

یک قاب Summary Zoom جدید ایجاد می‌کند و آن را در مجموعهٔ اشکال در ایندکس مشخص درج می‌کند.

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| index | Int32 | اندیس صفر-پایه‌ای که قاب Summary Zoom جدید در آن درج می‌شود. |
| x | Single | مختصات x قاب Summary Zoom جدید، بر حسب نقطه. |
| y | Single | مختصات y قاب Summary Zoom جدید، بر حسب نقطه. |
| width | Single | عرض قاب Summary Zoom جدید، بر حسب نقطه. |
| height | Single | ارتفاع قاب Summary Zoom جدید، بر حسب نقطه. |

### مثال‌ها

این مثال نحوهٔ ایجاد و درج یک شیء Summary Zoom را در ایندکس مشخصی از یک مجموعه نشان می‌دهد (فرض کنید در ارائهٔ "Presentation.pptx" حداقل دو بخش وجود دارد):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSummaryZoomFrame(2, 150, 20, 50, 50);
}
```

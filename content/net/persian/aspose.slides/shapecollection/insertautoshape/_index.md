## InsertAutoShape(int, ShapeType, float, float, float, float) {#insertautoshape}

یک شکل خودکار جدید ایجاد می‌کند و آن را در مجموعه شکل‌ها در اندیس مشخص شده وارد می‌کند، با استفاده از قالب پیش‌فرض.

```csharp
public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width, 
    float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | Int32 | اندیس صفر مبنا که شکل خودکار جدید در آن وارد می‌شود. |
| shapeType | ShapeType | نوع [`ShapeType`](../../shapetype) برای شکل خودکاری که قرار است وارد شود. |
| x | Single | مختصات x قاب شکل، بر حسب نقطه. |
| y | Single | مختصات y قاب شکل، بر حسب نقطه. |
| width | Single | عرض قاب شکل، بر حسب نقطه. |
| height | Single | ارتفاع قاب شکل، بر حسب نقطه. |

## InsertAutoShape(int, ShapeType, float, float, float, float, bool) {#insertautoshape_1}

یک شکل خودکار جدید ایجاد می‌کند و آن را در مجموعه شکل‌ها در اندیس مشخص شده وارد می‌کند، به‌صورت اختیاری با سبک قالب پیش‌فرض مقداردهی اولیه می‌شود.

```csharp
public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | Int32 | اندیس صفر مبنا که شکل خودکار در آن وارد می‌شود. |
| shapeType | ShapeType | نوع [`ShapeType`](../../shapetype) برای شکل خودکاری که قرار است وارد شود. |
| x | Single | مختصات x قاب شکل، بر حسب نقطه. |
| y | Single | مختصات y قاب شکل، بر حسب نقطه. |
| width | Single | عرض قاب شکل، بر حسب نقطه. |
| height | Single | ارتفاع قاب شکل، بر حسب نقطه. |
| createFromTemplate | Boolean | True برای اعمال سبک قالب پیش‌فرض (شامل نام غیر خالی، سبک ساده و متن مرکزی)؛ false برای ایجاد شکل با همه خصوصیات تنظیم شده به مقادیر پیش‌فرض. |

## AddOleObjectFrame(float, float, float, float, IOleEmbeddedDataInfo) {#addoleobjectframe}

یک فریم شیء OLE جدید ایجاد می‌کند و آن را به انتهای مجموعهٔ اشکال اضافه می‌نماید.

```csharp
public IOleObjectFrame AddOleObjectFrame(float x, float y, float width, float height, 
    IOleEmbeddedDataInfo dataInfo)
```

| Parameter | Type | Description |
| --- | --- | --- |
| x | Single | مختصات x فریم OLE جدید، بر حسب نقطه. |
| y | Single | مختصات y فریم OLE جدید، بر حسب نقطه. |
| width | Single | عرض فریم OLE جدید، بر حسب نقطه. |
| height | Single | ارتفاع فریم OLE جدید، بر حسب نقطه. |
| dataInfo | IOleEmbeddedDataInfo | اطلاعات دربارهٔ دادهٔ جاسازی‌شدهٔ OLE ([`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo)). |

### Return Value

شیء جدید ساخته شده [`IOleObjectFrame`](../../ioleobjectframe).

### Examples

مثال‌های زیر نحوه افزودن فریم‌های شیء OLE به اسلایدهای ارائه PowerPoint را نشان می‌دهند.

```csharp
[C#]
// یک شی از کلاس Presentation که فایل PPTX را نمایندگی می‌کند، ایجاد می‌کند
using (Presentation pres = new Presentation())
{
    // به اسلاید اول دسترسی پیدا می‌کند
    ISlide sld = pres.Slides[0];
    // فایل اکسل را به یک جریان (stream) بارگذاری می‌کند
    MemoryStream mstream = new MemoryStream();
    using (FileStream fs = new FileStream("book1.xlsx", FileMode.Open, FileAccess.Read))
    {
        byte[] buf = new byte[4096];
        while (true)
        {
            int bytesRead = fs.Read(buf, 0, buf.Length);
            if (bytesRead <= 0)
                break;
            mstream.Write(buf, 0, bytesRead);
        }
    }
    // یک شی داده برای جاسازی ایجاد می‌کند
    IOleEmbeddedDataInfo dataInfo = new OleEmbeddedDataInfo(mstream.ToArray(), "xlsx");
    // یک شکل فریم شیء OLE اضافه می‌کند
    IOleObjectFrame oleObjectFrame = sld.Shapes.AddOleObjectFrame(0, 0, pres.SlideSize.Size.Width,
        pres.SlideSize.Size.Height, dataInfo);
    //فایل PPTX را بر روی دیسک می‌نویسد
    pres.Save("OleEmbed_out.pptx", SaveFormat.Pptx);
}
```

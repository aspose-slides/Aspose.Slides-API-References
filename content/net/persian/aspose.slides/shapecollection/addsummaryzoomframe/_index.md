## متد ShapeCollection.AddSummaryZoomFrame

```csharp
public ISummaryZoomFrame AddSummaryZoomFrame(float x, float y, float width, float height)
```

| x | Single | مختصات x قاب Summary Zoom جدید، به نقطه. |
| y | Single | مختصات y قاب Summary Zoom جدید، به نقطه. |
| width | Single | عرض قاب Summary Zoom جدید، به نقطه. |
| height | Single | ارتفاع قاب Summary Zoom جدید، به نقطه. |

### مثال‌ها

این مثال نشان می‌دهد چگونه یک شیء Summary Zoom را به انتهای یک مجموعه اضافه کنیم (فرض کنید که در ارائهٔ "Presentation.pptx" حداقل دو بخش وجود دارد):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSummaryZoomFrame(150, 20, 500, 250);
}
```

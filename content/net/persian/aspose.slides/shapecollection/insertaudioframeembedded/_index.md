## InsertAudioFrameEmbedded(int, float, float, float, float, Stream) {#insertaudioframeembedded_1}

ساخت یک فریم صوتی جدید با فایل WAV توکار و درج آن در مجموعه اشکال در ایندکس مشخص‌شده. صوت توکار به مجموعه Presentation.Audios اضافه می‌شود.

```csharp
public IAudioFrame InsertAudioFrameEmbedded(int index, float x, float y, float width, float height, 
    Stream audio_stream)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| index | Int32 | ایندکس مبتنی بر صفر که فریم صوتی در آن درج می‌شود. |
| x | Single | مختصات x فریم صوتی جدید، به پوینت. |
| y | Single | مختصات y فریم صوتی جدید، به پوینت. |
| width | Single | عرض فریم صوتی جدید، به پوینت. |
| height | Single | ارتفاع فریم صوتی جدید، به پوینت. |
| audio_stream | Stream | یک جریان ورودی حاوی داده‌های صوتی WAV برای توکار کردن. |

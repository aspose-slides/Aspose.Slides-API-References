## FontsLoader.LoadExternalFonts متد

پوشه‌های اضافی را برای جست‌وجوی قلم‌ها اضافه می‌کند.

```csharp
public static void LoadExternalFonts(string[] directories)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| directories | String[] | پوشه‌هایی برای خواندن قلم‌های اضافی. |

### مثال‌ها

مثال‌های زیر نشان می‌دهند چگونه قلم‌های سفارشی را از .TTF بارگذاری کنید.

```csharp
[C#]
// مسیر به پوشه اسناد.
string dataDir = "C:\\";
// پوشه‌ها برای جست‌وجوی قلم‌ها
String[] folders = new String[] { dataDir };
// بارگذاری قلم‌های پوشه سفارشی
FontsLoader.LoadExternalFonts(folders);
// انجام کاری و رندر ارائه/اسلایدها
using (Presentation presentation = new Presentation(dataDir + "DefaultFonts.pptx"))
{
    presentation.Save(dataDir + "NewFonts_out.pptx", SaveFormat.Pptx);
	// پاک کردن کش قلم
	FontsLoader.ClearCache();
}
```

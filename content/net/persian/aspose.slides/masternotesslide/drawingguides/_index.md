## MasterNotesSlide.DrawingGuides ویژگی

یک مجموعه از راهنمایی‌های رسم برای اسلاید یادداشت‌های اصلی برمی‌گرداند. فقط خواندنی [`IDrawingGuidesCollection`](../../idrawingguidescollection)

```csharp
public IDrawingGuidesCollection DrawingGuides { get; }
```

### مثال‌ها

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    var notesSize = pres.NotesSize.Size;
    
    IDDrawingGuidesCollection guides = pres.MasterNotesSlideManager.SetDefaultMasterNotesSlide().DrawingGuides;
    // افزودن راهنمای رسم افقی جدید در زیر مرکز اسلاید
    guides.Add(Orientation.Horizontal, notesSize.Height / 2 + 50f);
    
    pres.Save("MasterNotesDrawingGuides_out.pptx", SaveFormat.Pptx);
}
```

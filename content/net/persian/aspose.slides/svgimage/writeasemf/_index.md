### مثال‌ها

مثال زیر نشان می‌دهد چگونه تصویر SVG را به متافایل ذخیره کنید.

```csharp
[C#]
// ایجاد تصویر SVG جدید
ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));

// تصویر SVG را به‌عنوان یک متافایل ذخیره می‌کند
using (var fileStream = System.IO.File.OpenWrite("SvgAsEmf.emf"))
    svgImage.WriteAsEmf(fileStream);
```

این نمونه نشان می‌دهد چگونه تصویر SVG را به‌عنوان متافایل به مجموعهٔ تصاویر ارائه اضافه کنید.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // ایجاد تصویر SVG جدید
    ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));
    using (var memStream = new MemoryStream())
    {
        // تصویر SVG را به‌عنوان یک متافایل ذخیره می‌کند
        svgImage.WriteAsEmf(memStream);
        // متافایل را به مجموعه تصاویر اضافه می‌کند
        pres.Images.AddImage(memStream.ToArray());
    }
}
```

## متد FontFallBackRule.IndexOf

یک اندیس از قاعدهٔ مشخص‌شده در مجموعه برمی‌گرداند.

```csharp
public int IndexOf(string fontName)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| fontName | String | نام قلم برای جستجو. |

### مقدار بازگشت

اندیس یک قلم یا -1 اگر قلم در فهرست یافت نشد.

### مثال‌ها

```csharp
[C#]
// یک قاعده که شامل فهرستی از قلم‌ها است ایجاد می‌کند.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// اندیس Tahoma را دریافت کنید.
int tahomaIndex = newRule.IndexOf("Tahoma");
```

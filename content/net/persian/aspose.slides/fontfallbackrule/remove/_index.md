## FontFallBackRule.Remove متد

اولین رخداد یک فونت FallBack خاص را از لیست حذف می‌کند.

```csharp
public void Remove(string fontName)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| fontName | String | نام فونت برای حذف از لیست. |

### مثال‌ها

```csharp
[C#]
// قاعده‌ای ایجاد می‌کند که شامل فهرستی از قلم‌ها است.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// Tahoma را از فهرست حذف می‌کند.
newRule.Remove ("Tahoma");
```

## FontFallBackRule.RemoveAt متد

قلم FallBack را در ایندکس مشخص شدهٔ لیست حذف می‌کند.

```csharp
public void RemoveAt(int index)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| index | Int32 | اندیس صفر-پایهٔ قلم برای حذف. |

### مثال‌ها

```csharp
[C#]
// ایجاد یک قاعده که شامل لیستی از فونت‌ها است.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// حذف Tahoma از لیست.
newRule.Remove (2);
```

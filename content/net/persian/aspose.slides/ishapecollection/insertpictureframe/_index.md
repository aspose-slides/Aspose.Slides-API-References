## IShapeCollection.InsertPictureFrame متد

یک قاب تصویر جدید که شامل تصویر مشخص‌شده است ایجاد می‌کند و آن را در مجموعه اشکال در اندیس مشخص‌شده درج می‌نماید.

```csharp
public IPictureFrame InsertPictureFrame(int index, ShapeType shapeType, float x, float y, 
    float width, float height, IPPImage image)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| index | Int32 | اندیس صفر-پایه‌ای که قاب تصویر در آن درج می‌شود. |
| shapeType | ShapeType | نوع شکل موجود در [`ShapeType`](../../shapetype) را مشخص می‌کند، به‌جز تمام انواع خطوط: |
| x | Single | مختصات x قاب تصویر، بر حسب پوینت. |
| y | Single | مختصات y قاب تصویر، بر حسب پوینت. |
| width | Single | عرض قاب تصویر، بر حسب پوینت. |
| height | Single | ارتفاع قاب تصویر، بر حسب پوینت. |
| image | IPPImage | [`IPPImage`](../../ippimage) که در قاب تصویر نمایش داده می‌شود. |

## InsertConnector(int, ShapeType, float, float, float, float, bool) {#insertconnector_1}

یک شکل اتصال جدید ایجاد می‌کند و آن را در مجموعهٔ اشکال در شاخص مشخص شده وارد می‌سازد، به‌صورت اختیاری استایل قالب پیش‌فرض را اعمال می‌کند.

```csharp
public IConnector InsertConnector(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| index | Int32 | شاخص صفر-محور که در آن شکل اتصال وارد می‌شود. |
| shapeType | ShapeType | نوع [`ShapeType`](../../shapetype) شکل اتصال که باید وارد شود. |
| x | Single | مختصات x چارچوب اتصال، به نقاط. |
| y | Single | مختصات y چارچوب اتصال، به نقاط. |
| width | Single | عرض چارچوب اتصال، به نقاط. |
| height | Single | ارتفاع چارچوب اتصال، به نقاط. |
| createFromTemplate | Boolean | در صورت True استایل قالب پیش‌فرض (نام غیر خالی، سبک ساده) را اعمال می‌کند؛ در صورت False شکل اتصال را با مقادیر پیش‌فرض ویژگی‌ها ایجاد می‌کند. |

## InsertSectionZoomFrame(int, float, float, float, float, ISection) {#insertsectionzoomframe}

یک فریم Section Zoom جدید ایجاد می‌کند و آن را در مجموعهٔ اشکال در شاخص مشخص شده درج می‌نماید.

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | Int32 | شاخص صفر مبنا که فریم Section Zoom باید در آن درج شود. |
| x | Single | مختصات x فریم Section Zoom جدید، به واحد نقاط. |
| y | Single | مختصات y فریم Section Zoom جدید، به واحد نقاط. |
| width | Single | عرض فریم Section Zoom جدید، به واحد نقاط. |
| height | Single | ارتفاع فریم Section Zoom جدید، به واحد نقاط. |
| section | ISection | [`ISection`](../../isection) ارجاع شده توسط فریم Section Zoom؛ باید به این ارائه تعلق داشته باشد و حداقل شامل یک اسلاید باشد. |

### مثال‌ها

این مثال نحوهٔ ایجاد و درج یک شیء Section Zoom را در شاخص مشخصی از یک مجموعه نشان می‌دهد (فرض کنید در ارائهٔ "Presentation.pptx" حداقل دو بخش وجود دارد):

```csharp

}
```

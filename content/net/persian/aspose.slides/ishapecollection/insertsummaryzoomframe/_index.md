## IShapeCollection.InsertSummaryZoomFrame متد

یک فریم Summary Zoom جدید ایجاد می‌کند و آن را در مجموعه اشکال در شاخص مشخص شده وارد می‌سازد.

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| index | Int32 | شاخص صفر مبنایی که فریم Summary Zoom در آن وارد می‌شود. |
| x | Single | مختصات x فریم Summary Zoom جدید، به نقطه. |
| y | Single | مختصات y فریم Summary Zoom جدید، به نقطه. |
| width | Single | عرض فریم Summary Zoom جدید، به نقطه. |
| height | Single | ارتفاع فریم Summary Zoom جدید، به نقطه. |

### مقدار بازگشت

### یادداشت‌ها

این متد یک فریم Summary Zoom ایجاد می‌کند که پیوندهای خلاصه‌سازی را برای تمام بخش‌های ارائه تجمیع می‌کند.

### مثال‌ها

این مثال نحوه ایجاد و درج یک شیء Summary Zoom در شاخص مشخص شده یک مجموعه را نشان می‌دهد (فرض کنید که در ارائه "Presentation.pptx" حداقل دو بخش وجود دارد):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSummaryZoomFrame(2, 150, 20, 50, 50);
}
```

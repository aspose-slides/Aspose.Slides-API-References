## InsertAudioFrameEmbedded(int, float, float, float, float, Stream) {#insertaudioframeembedded_1}

```csharp
    Stream audio_stream)
```

| x | Single | مختصات x قاب صوتی جدید، به نقطه. |
| y | Single | مختصات y قاب صوتی جدید، به نقطه. |
| width | Single | عرض قاب صوتی جدید، به نقطه. |
| height | Single | ارتفاع قاب صوتی جدید، به نقطه. |
| audio_stream | Stream | یک جریان ورودی حاوی داده‌های صوتی WAV برای تعبیه. |

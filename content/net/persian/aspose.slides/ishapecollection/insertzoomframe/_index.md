## InsertZoomFrame(int, float, float, float, float, ISlide) {#insertzoomframe}

یک قاب Zoom جدید ایجاد می‌کند و آن را در مجموعهٔ اشکال در مکان مشخص درج می‌نماید.

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| index | Int32 | اندیس صفر پایه‌ای که قاب Zoom در آن درج می‌شود. |
| x | Single | مختصات x قاب Zoom جدید، به واحد نقاط. |
| y | Single | مختصات y قاب Zoom جدید، به واحد نقاط. |
| width | Single | عرض قاب Zoom جدید، به واحد نقاط. |
| height | Single | ارتفاع قاب Zoom جدید، به واحد نقاط. |
| slide | ISlide | ‎[`ISlide`](../../islide) که توسط قاب Zoom ارجاع داده شده است. |

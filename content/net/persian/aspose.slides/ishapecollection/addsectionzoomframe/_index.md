## AddSectionZoomFrame(float, float, float, float, ISection) {#addsectionzoomframe}

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section)
```

| Parameter | Type | Description |

| x | Single | مختصات x فریم جدید Section Zoom، بر حسب نقاط. |
| y | Single | مختصات y فریم جدید Section Zoom، بر حسب نقاط. |
| width | Single | عرض فریم جدید Section Zoom، بر حسب نقاط. |
| height | Single | ارتفاع فریم جدید Section Zoom، بر حسب نقاط. |
| section | ISection | [`ISection`](../../isection) ارجاع شده توسط فریم Section Zoom؛ باید متعلق به این ارائه باشد و حداقل شامل یک اسلاید باشد. |

این مثال نشان می‌دهد که چگونه یک شیء Section Zoom را به انتهای یک مجموعه اضافه کنیم (فرض کنید در ارائه "Presentation.pptx" حداقل دو بخش وجود دارد):

}

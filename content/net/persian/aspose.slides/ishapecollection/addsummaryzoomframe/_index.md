## IShapeCollection.AddSummaryZoomFrame متد

```csharp
public ISummaryZoomFrame AddSummaryZoomFrame(float x, float y, float width, float height)
```

| y | Single | مختصات y فریم Summary Zoom جدید، بر حسب نقطه. |
| width | Single | عرض فریم Summary Zoom جدید، بر حسب نقطه. |
| height | Single | ارتفاع فریم Summary Zoom جدید، بر حسب نقطه. |

### مثال‌ها

این مثال نحوه اضافه کردن یک شیء Summary Zoom به انتهای یک مجموعه را نشان می‌دهد (فرض کنید حداقل دو بخش در ارائه «Presentation.pptx» وجود دارد):

}

## InsertChart(ChartType, float, float, float, float, int) {#insertchart}

| type | ChartType | نوع نموداری که می‌خواهید ایجاد کنید. |
| x | Single | مختصات x نمودار جدید، به واحد نقطه. |
| y | Single | مختصات y نمودار جدید، به واحد نقطه. |
| width | Single | عرض نمودار جدید، به واحد نقطه. |
| height | Single | ارتفاع نمودار جدید، به واحد نقطه. |
| index | Int32 | شاخص صفر-مبنا که نمودار جدید باید در آن در مجموعهٔ شکل‌ها درج شود. |

## InsertChart(ChartType, float, float, float, float, int, bool) {#insertchart_1}

یک نمودار جدید ایجاد می‌کند، آن را با داده‌ها و تنظیمات نمونه سری‌ها مقداردهی اولیه می‌کند و در مجموعهٔ شکل‌ها در نمایهٔ مشخص‌شده درج می‌نماید.

```csharp
public IChart InsertChart(ChartType type, float x, float y, float width, float height, int index, 
    bool initWithSample)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| type | ChartType | نوع نموداری که می‌خواهید ایجاد کنید. |
| x | Single | مختصات x نمودار جدید، به واحد نقطه. |
| y | Single | مختصات y نمودار جدید، به واحد نقطه. |
| width | Single | عرض نمودار جدید، به واحد نقطه. |
| height | Single | ارتفاع نمودار جدید، به واحد نقطه. |
| index | Int32 | شاخص صفر-مبنا که نمودار جدید باید در آن در مجموعهٔ شکل‌ها درج شود. |
| initWithSample | Boolean | اگر مقدار True باشد، نمودار جدید با داده‌ها و تنظیمات نمونه سری‌ها مقداردهی می‌شود؛ اگر False باشد، نمودار بدون سری و با حداقل تنظیمات ایجاد می‌شود که سرعت ساخت را افزایش می‌دهد. |

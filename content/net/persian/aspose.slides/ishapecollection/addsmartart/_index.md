## IShapeCollection.AddSmartArt متد

```csharp
public ISmartArt AddSmartArt(float x, float y, float width, float height, 
    SmartArtLayoutType layoutType)
```

| پارامتر | نوع | توضیح |

| x | Single | مختصات x چارچوب نمودار، به نقطه. |
| y | Single | مختصات y چارچوب نمودار، به نقطه. |
| width | Single | عرض چارچوب نمودار، به نقطه. |
| height | Single | ارتفاع چارچوب نمودار، به نقطه. |
| layoutType | SmartArtLayoutType | نوع چیدمان SmartArt. |

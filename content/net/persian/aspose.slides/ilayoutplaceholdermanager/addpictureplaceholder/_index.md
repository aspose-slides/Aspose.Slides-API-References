## ILayoutPlaceholderManager.AddPicturePlaceholder متد

یک شکل جای‌گیر جدید به اسلاید طرح اضافه می‌کند تا یک تصویر را نگه دارد.

```csharp
public IAutoShape AddPicturePlaceholder(float x, float y, float width, float height)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| x | Single | مختصات X شکل جای‌گیر جدید. |
| y | Single | مختصات Y شکل جای‌گیر جدید. |
| width | Single | عرض شکل جای‌گیر جدید. |
| height | Single | ارتفاع شکل جای‌گیر جدید. |

### مقدار بازگشت

[`IAutoShape`](../../iautoshape) با یک جای‌گیر تصویر ایجاد شد.

### نمونه‌ها

مثال زیر نشان می‌دهد چگونه شکل جای‌گیر تصویر را به اسلاید طرح اضافه کنیم.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddPicturePlaceholder(20, 20, 200, 200);
}
```

## ILayoutPlaceholderManager.AddContentPlaceholder متد

یک شکل placeholder جدید را به اسلاید طرح‌بندی اضافه می‌کند تا محتوا مانند تصویر، جدول، رسانه یا متن را در خود نگه دارد.

```csharp
public IAutoShape AddContentPlaceholder(float x, float y, float width, float height)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| x | Single | مختصات X شکل جدید placeholder. |
| y | Single | مختصات Y شکل جدید placeholder. |
| width | Single | عرض شکل جدید placeholder. |
| height | Single | ارتفاع شکل جدید placeholder. |

### مقدار برگشت

[`IAutoShape`](../../iautoshape) با یک placeholder از نوع Content ایجاد شد.

### مثال‌ها

مثال زیر نشان می‌دهد چگونه شکل placeholder از نوع Content را به اسلاید طرح‌بندی اضافه کنیم.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddContentPlaceholder(20, 20, 500, 300);
}
```

## ILayoutPlaceholderManager.AddSmartArtPlaceholder method

یک شکل جای‌دار جدید به اسلاید چیدمان اضافه می‌کند تا یک نمودار SmartArt را در خود داشته باشد.

```csharp
public IAutoShape AddSmartArtPlaceholder(float x, float y, float width, float height)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| x | Single | مختصات X شکل جای‌دار جدید. |
| y | Single | مختصات Y شکل جای‌دار جدید. |
| width | Single | عرض شکل جای‌دار جدید. |
| height | Single | ارتفاع شکل جای‌دار جدید. |

### مقدار بازگشتی

یک [`IAutoShape`](../../iautoshape) با یک جای‌دار SmartArt ایجاد شد.

### مثال‌ها

مثال زیر نشان می‌دهد چگونه شکل جای‌دار SmartArt را به اسلاید چیدمان اضافه کنیم.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddSmartArtPlaceholder(20, 20, 200, 200);
}
```

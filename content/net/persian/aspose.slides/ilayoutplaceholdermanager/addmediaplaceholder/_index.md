## ILayoutPlaceholderManager.AddMediaPlaceholder متد

یک شکل نمونه‌گیر جدید به اسلاید قالب اضافه می‌کند تا یک شیء رسانه‌ای را در خود نگه دارد.

```csharp
public IAutoShape AddMediaPlaceholder(float x, float y, float width, float height)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| x | Single | مختصات X شکل نمونه‌گیر جدید. |
| y | Single | مختصات Y شکل نمونه‌گیر جدید. |
| width | Single | عرض شکل نمونه‌گیر جدید. |
| height | Single | ارتفاع شکل نمونه‌گیر جدید. |

### مقدار برگشتی

یک [`IAutoShape`](../../iautoshape) با یک Media placeholder ایجاد شد.

### مثال‌ها

مثال زیر نشان می‌دهد چگونه شکل Media placeholder را به اسلاید قالب اضافه کنیم.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddMediaPlaceholder(20, 20, 200, 200);
}
```

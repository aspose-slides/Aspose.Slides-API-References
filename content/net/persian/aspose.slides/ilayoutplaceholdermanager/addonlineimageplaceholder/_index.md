## ILayoutPlaceholderManager.AddOnlineImagePlaceholder متد

یک شکل جای‌گیر جدید به اسلاید طرح‌بندی اضافه می‌کند تا یک تصویر آنلاین را در خود نگه دارد.

```csharp
public IAutoShape AddOnlineImagePlaceholder(float x, float y, float width, float height)
```

| پارامتر | نوع | توضیحات |
| --- | --- | --- |
| x | Single | مختصات X شکل جای‌گیر جدید. |
| y | Single | مختصات Y شکل جای‌گیر جدید. |
| width | Single | عرض شکل جای‌گیر جدید. |
| height | Single | ارتفاع شکل جای‌گیر جدید. |

### مقدار بازگشت

ایجاد شد [`IAutoShape`](../../iautoshape) با یک جای‌گیر تصویر آنلاین.

### مثال‌ها

مثال زیر نشان می‌دهد چگونه شکل جای‌گیر تصویر آنلاین را به اسلاید طرح‌بندی اضافه کنیم.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddOnlineImagePlaceholder(20, 20, 200, 200);
}
```

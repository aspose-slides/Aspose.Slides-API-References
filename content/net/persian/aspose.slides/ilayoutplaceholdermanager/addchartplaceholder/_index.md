## ILayoutPlaceholderManager.AddChartPlaceholder متد

یک شکل نگهدارنده جدید به اسلاید چیدمان اضافه می‌کند تا نمودار را در خود داشته باشد.

```csharp
public IAutoShape AddChartPlaceholder(float x, float y, float width, float height)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| x | Single | مختصات X شکل نگهدارنده جدید. |
| y | Single | مختصات Y شکل نگهدارنده جدید. |
| width | Single | عرض شکل نگهدارنده جدید. |
| height | Single | ارتفاع شکل نگهدارنده جدید. |

### مقدار بازگشت

[`IAutoShape`](../../iautoshape) ایجاد شد با یک نگهدارندهٔ Chart.

### مثال‌ها

مثال زیر نشان می‌دهد چگونه شکل نگهدارندهٔ Chart را به اسلاید چیدمان اضافه کنیم.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddChartPlaceholder(20, 20, 200, 200);
}
```

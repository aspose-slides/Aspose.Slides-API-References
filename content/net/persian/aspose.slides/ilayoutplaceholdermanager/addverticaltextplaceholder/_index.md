## ILayoutPlaceholderManager.AddVerticalTextPlaceholder متد

یک شکل جای‌نگهدار جدید به اسلاید طرح اضافه می‌کند تا محتوای متن را به جهت عمودی نگه دارد.

```csharp
public IAutoShape AddVerticalTextPlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| x | Single | مختصات X شکل جای‌نگهدار جدید. |
| y | Single | مختصات Y شکل جای‌نگهدار جدید. |
| width | Single | عرض شکل جای‌نگهدار جدید. |
| height | Single | ارتفاع شکل جای‌نگهدار جدید. |

### مقدار بازگشت

یک [`IAutoShape`](../../iautoshape) با یک Text (Vertical) placeholder ایجاد شد.

### مثال‌ها

مثال زیر نشان می‌دهد چگونه شکل Text (Vertical) placeholder را به اسلاید طرح اضافه کنیم.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalTextPlaceholder(20, 20, 300, 500);
}
```

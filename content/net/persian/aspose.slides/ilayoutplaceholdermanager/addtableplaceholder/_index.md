## ILayoutPlaceholderManager.AddTablePlaceholder method

یک شکل placeholder جدید به اسلاید طرح‌بندی اضافه می‌کند تا یک جدول را در خود نگه دارد.

```csharp
public IAutoShape AddTablePlaceholder(float x, float y, float width, float height)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| x | Single | مختصات X شکل placeholder جدید. |
| y | Single | مختصات Y شکل placeholder جدید. |
| width | Single | عرض شکل placeholder جدید. |
| height | Single | ارتفاع شکل placeholder جدید. |

### مقدار برگشتی

[`IAutoShape`](../../iautoshape) ایجاد شد با یک placeholder Table.

### نمونه‌ها

مثال زیر نشان می‌دهد چگونه شکل placeholder Table را به اسلاید طرح‌بندی اضافه کنیم.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTablePlaceholder(20, 20, 500, 200);
}
```

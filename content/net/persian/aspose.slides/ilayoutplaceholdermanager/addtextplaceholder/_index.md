## ILayoutPlaceholderManager.AddTextPlaceholder متد

یک شکل نگهدارندهٔ جدید به اسلاید الگو اضافه می‌کند تا محتوای متنی را دربرداشته باشد.

```csharp
public IAutoShape AddTextPlaceholder(float x, float y, float width, float height)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| x | Single | مختصات X شکل نگهدارندهٔ جدید. |
| y | Single | مختصات Y شکل نگهدارندهٔ جدید. |
| width | Single | عرض شکل نگهدارندهٔ جدید. |
| height | Single | ارتفاع شکل نگهدارندهٔ جدید. |

### مقدار بازگشت

یک [`IAutoShape`](../../iautoshape) با یک نگهدارندهٔ متن ایجاد شد.

### مثال‌ها

مثال زیر نشان می‌دهد که چگونه شکل نگهدارندهٔ متن را به اسلاید الگو اضافه کنید.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTextPlaceholder(20, 20, 500, 300);
}
```

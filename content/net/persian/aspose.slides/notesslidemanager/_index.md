### مثال‌ها

مثال زیر نشان می‌دهد چگونه یادداشت‌ها را به اسلاید خاصی از ارائه PowerPoint اضافه کنید.

```csharp
[C#]
	// یک شیء Presentation ایجاد کنید که نمایانگر یک فایل ارائه باشد
	using(Presentation presentation = new Presentation(dataDir + "AccessSlides.pptx")) {
	  // یادداشت‌ها را به اسلاید اول اضافه کنید
	  INotesSlideManager mgr = presentation.Slides[0].NotesSlideManager;
	  INotesSlide noteSlide = mgr.AddNotesSlide();
	  noteSlide.NotesTextFrame.Text = "Your Notes";
	  // ارائه را در دیسک ذخیره کنید
	  presentation.Save(dataDir + "RemoveNotesAtSpecificSlide_out.pptx", SaveFormat.Pptx);
	}
```

مثال زیر نشان می‌دهد چگونه یادداشت‌ها را از اسلاید خاصی از ارائه PowerPoint حذف کنید.

```csharp
[C#]
	// یک شیء Presentation ایجاد کنید که نمایانگر یک فایل ارائه باشد
	using(Presentation presentation = new Presentation(dataDir + "AccessSlides.pptx")) {
	  // حذف یادداشت‌های اسلاید اول
	  INotesSlideManager mgr = presentation.Slides[0].NotesSlideManager;
	  mgr.RemoveNotesSlide();
	  // ارائه را در دیسک ذخیره کنید
	  presentation.Save(dataDir + "RemoveNotesAtSpecificSlide_out.pptx", SaveFormat.Pptx);
	}
```

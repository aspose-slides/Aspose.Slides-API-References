## IFonts.SetScriptFont متد

نام یک قلم را به یک برچسب اسکریپت خاص اختصاص می‌دهد که نحوه نمایش متن آن اسکریپت را در ارائه تعیین می‌کند.

```csharp
public void SetScriptFont(string script, string fontName)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| script | String | کد اسکریپت BCP-47 (مثلاً "Arab"، "Hebr"، "Hans") که سیستم نوشتاری را شناسایی می‌کند. |
| fontName | String | نام قلمی که برای اسکریپت مشخص شده اختصاص می‌یابد. |

### مثال‌ها

این مثال نشان می‌دهد چگونه قلم برای اسکریپت عربی به «Segoe UI» تنظیم می‌شود:

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segoe UI");
```

## IFonts.GetScriptFont متد

نام فونتی که با یک برچسب اسکریپت خاص در قالب ارائه مرتبط است را دریافت می‌کند.

```csharp
public string GetScriptFont(string script)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| script | String | کد اسکریپت BCP-47 (مثلاً "Latn", "Cyrl", "Jpan") که برای شناسایی یک سیستم نوشتاری استفاده می‌شود. |

### مقدار بازگشت

نام فونتی که برای اسکریپت مشخص شده استفاده می‌شود، یا `null` اگر اسکریپت تعریف نشده باشد.

### مثال‌ها

این مثال نشان می‌دهد چگونه می‌توان فونت اختصاص یافته به اسکریپت سیریلیک را در قالب ارائه بازیابی کرد.

```csharp
[C#]
string font = presentation.MasterTheme.FontScheme.Major.GetScriptFont("Cyrl");
Console.WriteLine("Font for Cyrillic script: " + font);
```

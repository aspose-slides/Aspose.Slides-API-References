## IFonts.RemoveScriptFont متد

تنظیمات قلم مرتبط با یک برچسب اسکریپت خاص را از مجموعه قلم‌های تم حذف می‌کند.

```csharp
public void RemoveScriptFont(string script)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| script | String | کد اسکریپت BCP-47 که تنظیم قلم آن باید حذف شود. |

### مثال‌ها

این مثال نحوه حذف نگاشت قلم برای اسکریپت عبری را نشان می‌دهد:

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.RemoveScriptFont("Hebr");
```

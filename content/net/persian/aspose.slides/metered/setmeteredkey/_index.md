## Metered.SetMeteredKey متد

کلیدهای عمومی و خصوصی متری را تنظیم می‌کند. اگر مجوز متری خریداری کرده‌اید، هنگام شروع برنامه این API باید فراخوانی شود؛ معمولاً این کافی است. اما اگر همیشه در بارگذاری داده‌های مصرفی شکست بخورد و بیش از ۲۴ ساعت بگذرد، مجوز به وضعیت ارزیابی تغییر می‌یابد. برای جلوگیری از این وضعیت، باید به‌طور منظم وضعیت مجوز را بررسی کنید؛ اگر وضعیت ارزیابی باشد، این API را دوباره فراخوانی کنید.

```csharp
public void SetMeteredKey(string publicKey, string privateKey)
```

| privateKey | String | کلید خصوصی |

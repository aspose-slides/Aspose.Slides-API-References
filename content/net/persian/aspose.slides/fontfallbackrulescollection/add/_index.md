## FontFallBackRulesCollection.Add متد

قواعد FallBack مشخص‌شده را به انتهای مجموعه اضافه می‌کند.

```csharp
public void Add(IFontFallBackRule sourceRule)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| sourceRule | IFontFallBackRule | قاعده مشخص‌شده برای اضافه کردن |

### مثال‌ها

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    //دریافت مجموعه قواعد خالی یا پیش‌راه‌اندازی‌شده از FontsManager
    IFontFallBackRulesCollection rulesList = pres.FontsManager.FontFallBackRulesCollection;

    //اضافه کردن قاعده جدید به مجموعه
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));
}
```

## MasterSlideCollection.InsertClone متد

یک نسخه از اسلاید مستر مشخص‌شده را در موقعیت مشخص‌شدهٔ مجموعه اضافه می‌کند. اسلایدهای طرح‌بندی پیوند شده نیز کپی می‌شوند.

```csharp
public IMasterSlide InsertClone(int index, IMasterSlide sourceMaster)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| index | Int32 | شاخص اسلاید جدید. |
| sourceMaster | IMasterSlide | اسلاید برای کلون کردن. |

### مقدار بازگشت

اسلاید مستر افزوده شده.

### مثال‌ها

مثال زیر نشان می‌دهد چگونه می‌توان اسلاید مستر را در یک ارائهٔ PowerPoint دیگر کلون کرد.

```csharp
[C#]
// برای بارگذاری فایل ارائه منبع، کلاس Presentation را نمونه‌سازی کنید
using (Presentation srcPres = new Presentation("CloneToAnotherPresentationWithMaster.pptx"))
{
    // برای ارائه مقصد (جایی که اسلاید باید کلون شود) کلاس Presentation را نمونه‌سازی کنید
    using (Presentation destPres = new Presentation())
    {
        // نمونه‌سازی ISlide از مجموعه اسلایدها در ارائه منبع به همراه
        // اسلاید مستر
        ISlide SourceSlide = srcPres.Slides[0];
        IMasterSlide SourceMaster = SourceSlide.LayoutSlide.MasterSlide;
		// دریافت اسلایدهای مستر از ارائه مقصد
        IMasterSlideCollection masters = destPres.Masters;
        // کلون کردن اسلاید مستر موردنظر از ارائه منبع به مجموعه مسترهای
        // ارائه مقصد
        IMasterSlide iSlide = masters.AddClone(SourceMaster);
        // مجموعه اسلایدها در ارائه مقصد
        ISlideCollection slds = destPres.Slides;
		// کلون کردن اسلاید منبع به مجموعه اسلایدهای مقصد.
        slds.AddClone(SourceSlide, iSlide, true);
        // ذخیرهٔ ارائه مقصد در دیسک
        destPres.Save("CloneToAnotherPresentationWithMaster_out.pptx", SaveFormat.Pptx);
    }
}
```

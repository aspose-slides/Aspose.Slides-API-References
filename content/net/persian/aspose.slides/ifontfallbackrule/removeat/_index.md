## IFontFallBackRule.RemoveAt متد

فونت FallBack را در ایندکس مشخص‌شده از فهرست حذف می‌کند.

```csharp
public void RemoveAt(int index)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| index | Int32 | ایندکس صفر-پایهٔ فونت برای حذف. |

### مثال‌ها

```csharp
[C#]
// ایجاد یک قاعده که شامل فهرستی از قلم‌ها است.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// حذف Tahoma از فهرست
newRule.Remove (2);
```

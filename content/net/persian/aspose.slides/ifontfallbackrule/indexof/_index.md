## IFontFallBackRule.IndexOf متد

یک اندیس از قانون مشخص‌شده در مجموعه برمی‌گرداند.

```csharp
public int IndexOf(string fontName)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| fontName | String | نام قلم برای جستجو. |

### مقدار بازگشتی

اندیس یک قلم یا -1 اگر قلم در فهرست یافت نشود.

### مثال‌ها

```csharp
[C#]
// یک قانون حاوی لیستی از قلم‌ها ایجاد می‌کند.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

//Get index of Tahoma
int tahomaIndex = newRule.IndexOf("Tahoma");
```

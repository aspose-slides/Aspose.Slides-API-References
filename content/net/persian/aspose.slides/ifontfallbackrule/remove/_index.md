## IFontFallBackRule.Remove متد

اولین رخداد یک فونت FallBack خاص را از فهرست حذف می‌کند.

```csharp
public void Remove(string fontName)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| fontName | String | نام فونت برای حذف از فهرست. |

### مثال‌ها

```csharp
[C#]
// ایجاد یک قانون شامل فهرستی از قلم‌ها.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

//حذف Tahoma از فهرست
newRule.Remove ("Tahoma");
```

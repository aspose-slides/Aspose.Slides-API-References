## ToArray() {#toarray}

یک آرایه ایجاد می‌کند و آن را شامل تمام قلم‌های FallBack برای این قاعده باز می‌گرداند.

```csharp
public string[] ToArray()
```

### مقدار بازگشتی

Array of String

### مثال‌ها

```csharp
[C#]
// یک قاعده شامل لیستی از قلم‌ها ایجاد می‌کند.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

//تمام نام‌های قلم‌ها به صورت آرایه

```

| startIndex | Int32 | یک ایندکس برای اولین قلمی که باید اضافه شود. |

## متد FontsManager.GetFontEmbeddingLevel

سطح جاسازی یک قلم را از آرایه بایتی داده شده و نام قلم تعیین می‌کند.

```csharp
public EmbeddingLevel GetFontEmbeddingLevel(byte[] fontBytes, string fontName)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| fontBytes | Byte[] | آرایه بایتی که شامل داده‌های قلم است. |
| fontName | String | نام قلم. |

### مقدار بازگشتی

سطح جاسازی قلم مشخص شده.

### استثناها

| استثنا | شرط |
| --- | --- |
| ArgumentNullException | زمانی که *fontBytes* برابر null باشد، پرتاب می‌شود. |

### نمونه‌ها

```csharp
[C#]
using (Presentation pres = new Presentation(pptxFileName))
{
    // دریافت تمام فونت‌های استفاده شده در ارائه
    IFontData[] fontDatas = pres.FontsManager.GetFonts();

    // دریافت آرایه بایتی که سبک معمولی اولین فونت در ارائه را نشان می‌دهد
    byte[] bytes = pres.FontsManager.GetFontBytes(fontDatas[0], FontStyle.Regular);

    // تعیین سطح جاسازی فونت
    EmbeddingLevel embeddingLevel = pres.FontsManager.GetFontEmbeddingLevel(bytes, fontDatas[0].FontName);
}
```

## GetSubstitutions() {#getsubstitutions}

اطلاعات مربوط به قلم‌هایی که در هنگام رندر ارائه جایگزین می‌شوند را دریافت می‌کند.

```csharp
public IEnumerable<FontSubstitutionInfo> GetSubstitutions()
```

### مقدار بازگشت

مجموعه‌ای از تمام جایگزینی قلم‌ها [`FontSubstitutionInfo`](../../fontsubstitutioninfo).

### مثال‌ها

```csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    foreach (FontSubstitutionInfo fontSubstitution in pres.FontsManager.GetSubstitutions())
    {
        Console.WriteLine("{0} -> {1}", fontSubstitution.OriginalFontName, fontSubstitution.SubstitutedFontName);
    }
}        
```

## GetSubstitutions(int[]) {#getsubstitutions_1}

اطلاعات مربوط به قلم‌هایی که در هنگام رندر اسلایدهای مشخص شده جایگزین می‌شوند را دریافت می‌کند.

```csharp
public IEnumerable<FontSubstitutionInfo> GetSubstitutions(int[] slides)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| slides | Int32[] | آرایه‌ای از ایندکس‌های اسلایدها که اطلاعات جایگزینی قلم برای آن‌ها دریافت می‌شود، شروع از 1. |

### مقدار بازگشت

مجموعه‌ای از تمام جایگزینی قلم‌ها ([`FontSubstitutionInfo`](../../fontsubstitutioninfo)) برای اسلایدهای مشخص شده.

### مثال‌ها

```csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    int[] targetSlides = { 1, 2, 5 };
    foreach (var fontSubstitution in pres.FontsManager.GetSubstitutions(targetSlides))
    {
        Console.WriteLine("{0} -> {1}", fontSubstitution.OriginalFontName, fontSubstitution.SubstitutedFontName);
    }
}
```

## متد FontsManager.GetFontBytes

یک آرایه بایت که نمایانگر داده‌های فونت برای یک سبک فونت مشخص و دادهٔ فونت است را برمی‌گرداند.

```csharp
public byte[] GetFontBytes(IFontData fontData, FontStyleType fontStyle)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| fontData | IFontData | شیء دادهٔ فونت که شامل اطلاعات مربوط به فونت [`IFontData`](../../ifontdata) است. |
| fontStyle | FontStyleType | سبک فونت که داده‌های آن باید دریافت شود [`FontStyleType`](../../fontstyletype). |

### مقدار بازگشت

یک آرایه بایت شامل داده‌های فونت برای سبک فونت مشخص شده. اگر دادهٔ فونت یا سبک یافت نشود، مقدار null برگردانده می‌شود.

### نمونه‌ها

```csharp
[C#]
using (Presentation pres = new Presentation ("Presentation.pptx"))
{
    // دریافت تمام قلم‌های استفاده شده در ارائه
    IFontData[] fonts = pres.FontsManager.GetFonts();

    // دریافت آرایه بایتی که نمایانگر سبک عادی اولین قلم در ارائه است
    byte[] bytes = pres.FontsManager.GetFontBytes(fonts[0], FontStyleType.Regular);
}
```

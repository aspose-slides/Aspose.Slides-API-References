## SetSize(SlideSizeType, SlideSizeScaleType) {#setsize}

اندازه اسلاید را بر اساس نوع تنظیم می‌کند و محتوای موجود را مقیاس‌بندی می‌نماید.

```csharp
public void SetSize(SlideSizeType type, SlideSizeScaleType scaleType)
```

| Parameter | Type | Description |
| --- | --- | --- |
| type | SlideSizeType | اندازه پیش‌تعریف‌شدهٔ اسلاید برای اعمال. |
| scaleType | SlideSizeScaleType | حالت مقیاس‌بندی محتوای مورد استفاده. |

### Remarks

اختصاص هر مقدار غیر از Custom، [`Size`](../size) را بر اساس نوع انتخاب‌شده تنظیم می‌کند، در حالی که [`Orientation`](../orientation) حفظ می‌شود.

## SetSize(float, float, SlideSizeScaleType) {#setsize_1}

ابعاد اسلاید را به‌صورت صریح تنظیم می‌کند و محتوای موجود را مقیاس‌بندی می‌نماید.

```csharp
public void SetSize(float width, float height, SlideSizeScaleType scaleType)
```

| Parameter | Type | Description |
| --- | --- | --- |
| width | Single | عرض جدید اسلاید، به پوینت. |
| height | Single | ارتفاع جدید اسلاید، به پوینت. |
| scaleType | SlideSizeScaleType | حالت مقیاس‌بندی محتوای مورد استفاده. |

### Remarks

این عملیات ویژگی [`Type`](../type) را به Custom بازنشانی می‌کند و [`Orientation`](../orientation) را تنظیم می‌نماید.

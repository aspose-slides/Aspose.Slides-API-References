## IViewProperties.GridSpacing property

مقدار فاصله‌بندی شبکه را که باید برای شبکه زیرساخت سند ارائه استفاده شود، بر حسب نقاط بر می‌گرداند یا تنظیم می‌کند. خواندنی/نوشتنی Single.

```csharp
public float GridSpacing { get; set; }
```

### توضیحات

مقدار فاصله‌بندی شبکه باید عددی مثبت باشد. بازهٔ معمولی مقدار از ۱ میلی‌متر (۲٫۸۳۴۹۶۰۷ نقاط) تا ۲ اینچ (۱۴۴ نقطه) است.

### نمونه‌ها

کد نمونه زیر نشان می‌دهد چگونه فاصله‌بندی شبکه را در یک ارائه PowerPoint تغییر دهید.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

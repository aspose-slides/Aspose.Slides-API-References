## IOleObjectFrame.LinkPathRelative ویژگی

مسیر نسبی به فایلی که لینک شده است را در صورت وجود باز می‌گرداند، در غیر این صورت رشتهٔ خالی را باز می‌گرداند. فقط‌خواندنی String.

```csharp
public string LinkPathRelative { get; }
```

### توضیحات

در ارائه‌های Ppt، برخی از پیوندهای شیء Ole ممکن است دارای نمایش نسبی باشند.

### مثال‌ها

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.ppt"))
{
    IOleObjectFrame oleFrame = presentation.Slides[0].Shapes[0] as IOleObjectFrame;

    if (oleFrame != null)
    {
        Console.WriteLine("The relative path: " + oleFrame.LinkPathRelative);
    } 
}
```

## IOleObjectFrame.SetEmbeddedData متد

اطلاعاتی دربارهٔ دادهٔ جاسازی‌شده OLE تنظیم می‌کند.

```csharp
public void SetEmbeddedData(IOleEmbeddedDataInfo embeddedData)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| embeddedData | IOleEmbeddedDataInfo | دادهٔ جاسازی‌شده [`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo) |

### استثناها

| استثنا | شرط |
| --- | --- |
| ArgumentNullException | هنگامی که پارامتر embeddedData مقدار null باشد. |

### توضیحات

این متد خصوصیات شیء را برای بازتاب داده‌های جدید تغییر می‌دهد و پرچم IsObjectLink را به false تنظیم می‌کند، که نشان‌دهنده این است که شیء OLE جاسازی شده است.

### مثال‌ها

مثال زیر نشان می‌دهد چطور دادهٔ جاسازی‌شده OLE و نوع آن را برای شیء [`IOleObjectFrame`](../../ioleobjectframe) موجود تغییر داد.

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    OleObjectFrame oof = pres.Slides[0].Shapes[0] as OleObjectFrame;
    if (oof != null)
    {
        IOleEmbeddedDataInfo newData = new OleEmbeddedDataInfo(File.ReadAllBytes("Picture.png"), "png");
        oof.SetEmbeddedData(newData);
    }
}
```

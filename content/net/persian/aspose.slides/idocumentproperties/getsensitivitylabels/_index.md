## IDocumentProperties.GetSensitivityLabels متد

آرایه‌ای از برچسب‌های حساسیت را از ویژگی‌های سند سفارشی دریافت می‌کند (Microsoft Information Protection SDK Metadata).

```csharp
public ISensitivityLabel[] GetSensitivityLabels()
```

### مثال‌ها

کد زیر نشان می‌دهد چگونه می‌توانید اطلاعات برچسب‌های حساسیت را از ویژگی‌های سند سفارشی به مجموعهٔ SensitivityLabels مدرن منتقل کنید:

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    // دریافت برچسب‌های حساسیت از ویژگی‌های سفارشی سند
    ISensitivityLabel[] mipSensitivityLabels = pres.DocumentProperties.GetSensitivityLabels();

    ISensitivityLabelCollection sensitivityLabels = pres.SensitivityLabels;
    foreach (var sensitivityLabel in mipSensitivityLabels)
    {
        // اضافه کردن برچسب به مجموعه
        // در اینجا می‌توانید بررسی درستی اطلاعات برچسب (برچسب در دسترس است، و غیره) را اضافه کنید
        sensitivityLabels.Add(sensitivityLabel);
    }

    pres.Save("SensitivityLabel.pptx", SaveFormat.Pptx);
}
```

## IPictureFillFormat.TileAlignment خصوصیت

باز می‌گرداند یا تنظیم می‌کند که بافت چگونه در داخل شکل هم‌راست می‌شود. این تنظیم نقطه شروع الگوی بافت و نحوه تکرار آن در سراسر شکل را کنترل می‌کند. قابل خواندن/قابل نوشتن [`RectangleAlignment`](../../rectanglealignment).

```csharp
public RectangleAlignment TileAlignment { get; set; }
```

### توضیحات

پیش‌فرض TopLeft است.

### مثال‌ها

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // فرمت پر کردن تصویر شکل را دریافت می‌کند
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // حالت پر کردن تصویر را به کاشی تنظیم می‌کند
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // هم‌راستگی کاشی را به پایین-راست تنظیم می‌کند
    pictureFillFormat.TileAlignment = RectangleAlignment.BottomRight;
}
```

## CompressImage(bool, PicturesCompression) {#compressimage}

Compresses the image by reducing its size based on the shape size and specified resolution. Optionally, it also deletes cropped areas.

```csharp
public bool CompressImage(bool deleteCroppedAreasOfImage, PicturesCompression resolution)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| deleteCroppedAreasOfImage | Boolean | اگر مقدار true باشد، این متد نواحی برش‌خورده تصویر را حذف می‌کند و ممکن است اندازه آن را بیشتر کاهش دهد. |
| resolution | PicturesCompression | رزولوشن هدف برای فشرده‌سازی که به عنوان مقدار enum [`PicturesCompression`](../../../aspose.slides.export/picturescompression) مشخص می‌شود. |

### مقدار بازگشت

یک Boolean که نشان می‌دهد آیا تصویر با موفقیت فشرده شده است یا نه. اگر تصویر تغییر اندازه یا برش خورده باشد `true` برگردانده می‌شود، در غیر اینصورت `false`.

### استثناها

| استثنا | شرط |
| --- | --- |
| ArgumentException | در زمانی که رزولوشن مقدار معتبری نباشد، رخ می‌دهد. |

### نکات

این متد اندازه و رزولوشن تصویر را مشابه ویژگی "Picture Format -&gt; Compress Pictures" در PowerPoint تغییر می‌دهد.

### مثال‌ها

مثال زیر نشان می‌دهد چگونه می‌توانید از متد `CompressImage` برای کاهش اندازه یک تصویر در یک ارائه با تنظیم رزولوشن هدف و حذف نواحی برش‌خورده استفاده کنید:

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;
    // تصویر را با رزولوشن هدف 150 DPI (رزولوشن وب) فشرده کرده و نواحی برش‌خورده را حذف می‌کند
    bool result = picFrame.PictureFormat.CompressImage(true, PicturesCompression.Dpi150);
}
```

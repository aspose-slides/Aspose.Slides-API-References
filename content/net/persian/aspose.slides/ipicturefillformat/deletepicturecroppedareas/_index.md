## IPictureFillFormat.DeletePictureCroppedAreas متد

مناطق برش‌خورده تصویر پرکننده را حذف می‌کند.

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### مقدار بازگشت

تصویر برش‌خورده یا تصویر اصلی اگر برش لازم نباشد.

### ملاحظات

این متد فایل‌های متا WMF/EMF را در حین برش به تصویر PNG رستری تبدیل می‌کند.

### مثال‌ها

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // دریافت PictureFrame
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // حذف مناطق برش‌خورده تصویر PictureFrame
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

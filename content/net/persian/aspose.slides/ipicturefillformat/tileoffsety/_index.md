## IPictureFillFormat.TileOffsetY ویژگی

مقدار یا تنظیم می‌کند جابجایی عمودی بافت نسبت به مبدأ شکل بر حسب نقطه. مقدار مثبت بافت را به پایین می‌برد، در حالی که مقدار منفی آن را به بالا می‌برد. خواندنی/نوشتنی Single.

```csharp
public float TileOffsetY { get; set; }
```

### مثال‌ها

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // فرمت پر کردن تصویر شکل را دریافت می‌کند
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // حالت پر کردن تصویر را به Tile تنظیم می‌کند
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // جابجایی عمودی بافت را به -50 نقطه تنظیم می‌کند
    pictureFillFormat.TileOffsetY = -50;
}
```

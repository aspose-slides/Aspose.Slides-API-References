## IProtectionManager.EncryptDocumentProperties ویژگی

این ویژگی منطقی است، اگر ارائه‌نامه با گذرواژه محافظت شده باشد. اگر true باشد، خصوصیات سند در فایل ارائه‌نامه رمزنگاری می‌شود. اگر false باشد، خصوصیات سند عمومی است در حالی که ارائه‌نامه رمزنگاری شده است. قابل خواندن/قابل نوشتن Boolean.

public bool EncryptDocumentProperties { get; set; }

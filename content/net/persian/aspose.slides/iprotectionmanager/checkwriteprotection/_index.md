## IProtectionManager.CheckWriteProtection متد

تعیین می‌کند که آیا یک ارائه به‌صورت رمز‌گذاری شده برای تغییر محافظت شده است یا خیر.

```csharp
public bool CheckWriteProtection(string password)
```

| Parameter | Type | Description |
| --- | --- | --- |
| password | String | رمز عبور برای بررسی. |

### مقدار بازگشت

True اگر رمز عبور معتبر باشد؛ در غیر این صورت false.

### توضیحات

1. شما باید ویژگی [`IsWriteProtected`](../iswriteprotected) را قبل از فراخوانی این متد بررسی کنید. 2. وقتی رمز عبور null یا خالی باشد، این متد false برمی‌گرداند.

### مثال‌ها

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

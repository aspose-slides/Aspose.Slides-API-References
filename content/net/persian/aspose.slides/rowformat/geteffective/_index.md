## RowFormat.GetEffective متد

خواص قالب‌بندی ردیف جدول موثر را با به‌کارگیری ارث‌بری و سبک‌های جدول دریافت می‌کند.

```csharp
public IRowFormatEffectiveData GetEffective()
```

### مقدار بازگشتی

A [`IRowFormatEffectiveData`](../../irowformateffectivedata).

### مثال‌ها

این مثال نحوه دریافت قالب پر کردن موثر برای بخش‌های مختلف منطق جدول را نشان می‌دهد. لطفاً توجه داشته باشید که قالب‌بندی سلول همیشه اولویت بالاتری نسبت به قالب‌بندی ردیف دارد، ردیف نسبت به ستون، ستون نسبت به کل جدول. بنابراین در نهایت خصوصیات CellFormatEffectiveData برای رسم جدول استفاده می‌شوند. کد زیر فقط یک مثال از API است.

```csharp
[C#]
using (Presentation pres = new Presentation(@"MyPresentation.pptx"))
{
    ITable tbl = pres.Slides[0].Shapes[0] as ITable;
    IFillFormatEffectiveData tableFillFormatEffective = tbl.TableFormat.GetEffective().FillFormat;
    IFillFormatEffectiveData rowFillFormatEffective = tbl.Rows[0].RowFormat.GetEffective().FillFormat;
    IFillFormatEffectiveData columnFillFormatEffective = tbl.Columns[0].ColumnFormat.GetEffective().FillFormat;
    IFillFormatEffectiveData cellFillFormatEffective = tbl[0, 0].CellFormat.GetEffective().FillFormat;
    /* خروجی و مقایسه */
}
```

## IBasePortionFormat.SpellCheck ویژگی

یک مقدار را می‌گیرد یا تنظیم می‌کند که نشان می‌دهد آیا بررسی املا برای بخش متن فعال است یا نه. وقتی این ویژگی روی `false` تنظیم شده باشد، بررسی املا برای عناصر متن سرکوب می‌شود. وقتی روی `true` تنظیم شود، بررسی املا مجاز است. مقدار پیش‌فرض `false` است.

```csharp
public bool SpellCheck { get; set; }
```

### مثال‌ها

مثال بعدی نشان می‌دهد که چگونه پرچم SpellCheck را قبل از ذخیره‌سازی ارائه فعال می‌کنیم:

```csharp
[C#]
using (var pres = new Presentation("input.pptx"))
{
    // دسترسی به اولین بخش متن داخل اولین شکل در اولین اسلاید
    var portion = ((AutoShape)pres.Slides[0].Shapes[0]).TextFrame.Paragraphs[0].Portions[0];
    // فعال‌سازی بررسی املای این بخش متن
    portion.PortionFormat.SpellCheck = true;
    // ذخیرهٔ ارائهٔ تغییر یافته
    pres.Save("output-with-spellcheck.pptx", SaveFormat.Pptx);
}
```

## ICommentCollection.AddModernComment متد

یک نظر مدرن جدید به انتهای یک مجموعه اضافه می‌کند.

```csharp
public IModernComment AddModernComment(string text, ISlide slide, IShape shape, PointF position, 
    DateTime creationTime)
```

| پارامتر | نوع | شرح |
| --- | --- | --- |
| text | String | متن ساده یک نظر مدرن جدید. |
| slide | ISlide | اسلاید در یک ارائه که نظر مدرن جدید باید به آن اضافه شود. |
| shape | IShape | شکل روی اسلاید که نظر مدرن جدید به آن مرتبط است. |
| position | PointF | موقعیت روی اسلاید که نظر مدرن جدید به آن اضافه می‌شود. |
| creationTime | DateTime | زمان ایجاد یک نظر مدرن. |

### مقدار بازگشتی

نظر مدرن اضافه شده.

### مثال‌ها

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ICommentAuthor newAuthor = pres.CommentAuthors.AddAuthor("Some Author", "SA");
    newAuthor.Comments.AddModernComment("This is modern comment", pres.Slides[0], null, new PointF(100, 100), DateTime.Now);

    pres.Save(outPptxFileName, SaveFormat.Pptx);
}
```

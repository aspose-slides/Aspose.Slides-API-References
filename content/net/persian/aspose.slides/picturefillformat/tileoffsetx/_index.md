## PictureFillFormat.TileOffsetX ویژگی

افست افقی بافت نسبت به مبدأ شکل را بر حسب نقطه برمی‌گرداند یا تنظیم می‌کند. مقدار مثبت بافت را به سمت راست حرکت می‌دهد، در حالی که مقدار منفی آن را به سمت چپ می‌برد. خواندنی/نوشتنی Single.

```csharp
public float TileOffsetX { get; set; }
```

### مثال‌ها

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // قالب پر کردن تصویر شکل را دریافت می‌کند
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // حالت پر کردن تصویر را به Tile تنظیم می‌کند
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // افست افقی بافت را به 20 نقطه تنظیم می‌کند
    pictureFillFormat.TileOffsetX = 20f;
}
```

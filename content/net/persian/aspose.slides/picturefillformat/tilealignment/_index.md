## PictureFillFormat.TileAlignment ویژگی

مقدار بازگشت یا تنظیم می‌کند که بافت چگونه درون شکل هم‌تراز شود. این تنظیم نقطه شروع الگوی بافت و نحوه تکرار آن در سراسر شکل را کنترل می‌کند. خواندنی/نوشتنی [`RectangleAlignment`](../../rectanglealignment).

```csharp
public RectangleAlignment TileAlignment { get; set; }
```

### توضیحات

پیش‌فرض TopLeft است.

### مثال‌ها

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // دریافت فرمت پر کردن تصویر شکل
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // تنظیم حالت پر کردن تصویر به حالت کاشی
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // تنظیم هم‌ترازی کاشی‌ها به راست-پایین
    pictureFillFormat.TileAlignment = RectangleAlignment.BottomRight;
}
```

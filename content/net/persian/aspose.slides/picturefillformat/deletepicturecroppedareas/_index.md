## PictureFillFormat.DeletePictureCroppedAreas متد

ناحیه‌های برش‌خوردهٔ تصویر پرکننده را حذف می‌کند.

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### مقدار بازگشت

تصویر برش‌خورده یا تصویر اصلی اگر برش ضروری نباشد.

### توضیحات

این متد فایل‌های متا WMF/EMF را هنگام برش به تصویر PNG رستر تبدیل می‌کند.

### مثال‌ها

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // دریافت PictureFrame
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // ناحیه‌های برش‌خوردهٔ تصویر PictureFrame را حذف می‌کند
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

## CompressImage(bool, PicturesCompression) {#compressimage}

تصویر را با کاهش اندازه بر اساس اندازهٔ شکل و وضوح مشخص‌شده فشرده می‌کند. به‌طور اختیاری، نواحی برش‌خورده را نیز حذف می‌نماید.

```csharp
public bool CompressImage(bool deleteCroppedAreasOfImage, PicturesCompression resolution)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| deleteCroppedAreasOfImage | Boolean | اگر مقدار true باشد، متد نواحی برش‌خوردهٔ تصویر را حذف می‌کند و ممکن است اندازهٔ تصویر بیشتر کاهش یابد. |
| resolution | PicturesCompression | وضوح هدف برای فشرده‌سازی که به‌عنوان مقدار enum [`PicturesCompression`](../../../aspose.slides.export/picturescompression) مشخص می‌شود. |

### مقدار بازگشت

یک Boolean که نشان می‌دهد آیا فشرده‌سازی تصویر موفقیت‌آمیز بوده است یا خیر. اگر تصویر تغییر اندازه یا برش داده شده باشد `true` برمی‌گردد، در غیر این‌صورت `false`.

### استثنائات

| استثنا | شرط |
| --- | --- |
| ArgumentException | زمانی که وضوح مقدار معتبری نباشد. |

### توضیحات

این متد اندازه و وضوح تصویر را مشابه ویژگی «قالب تصویر -&gt; فشرده‌سازی تصاویر» در PowerPoint تغییر می‌دهد.

### مثال‌ها

مثال زیر نشان می‌دهد چگونه می‌توان از متد `CompressImage` برای کاهش اندازهٔ تصویر در یک ارائه با تنظیم وضوح هدف و حذف نواحی برش‌خورده استفاده کرد:

```csharp
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;
    // تصویر را با وضوح هدف 150 DPI (وضوح وب) فشرده می‌کند و نواحی برش‌خورده را حذف می‌نماید
    bool result = picFrame.PictureFormat.CompressImage(true, PicturesCompression.Dpi150);
}
```

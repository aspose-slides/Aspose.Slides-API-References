## Control.Persistence ویژگی

دریافت روش مورد استفاده برای ذخیره‌سازی ویژگی‌های کنترل ActiveX. فقط خواندنی [`PersistenceType`](../../persistencetype).

```csharp
public PersistenceType Persistence { get; }
```

### مثال‌ها

مثال بعدی استفاده از ویژگی Persistence را برای بررسی این که آیا می‌توان ویژگی‌های شیء ActiveX را به صورت ویژگی‌های XML-Based تغییر داد، نشان می‌دهد:

```csharp
[C#]
if (control.Persistence == PersistenceType.PersistPropertyBag)
{
    control.Properties["Value"] = value;
}
else
{
    YourMethodHere(control.ActiveXControlBinary); //از روش خودتان برای مدیریت ویژگی‌های ActiveX که در فایل باینری آن ذخیره شده‌اند استفاده کنید
}
```

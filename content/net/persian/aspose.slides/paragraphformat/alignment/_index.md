## ویژگی ParagraphFormat.Alignment

متن هم‌ترازی را در یک پاراگراف بدون وراثت برمی‌گرداند یا تنظیم می‌کند. قابل خواندن/نوشتن [`TextAlignment`](../../textalignment).

```csharp
public TextAlignment Alignment { get; set; }
```

### مثال‌ها

کد نمونه زیر نشان می‌دهد چگونه پاراگراف‌های متن را در PowerPoint Presentation هم‌ترازی کنید.

```csharp
[C#]
// یک شیء Presentation که نمایانگر یک فایل PPTX است را ایجاد می‌کند
using (Presentation pres = new Presentation("ParagraphsAlignment.pptx"))
{
    // دسترسی به اولین اسلاید
    ISlide slide = pres.Slides[0];
    // دسترسی به اولین و دومین placeholder در اسلاید و تبدیل نوع آن به AutoShape
    ITextFrame tf1 = ((IAutoShape)slide.Shapes[0]).TextFrame;
    ITextFrame tf2 = ((IAutoShape)slide.Shapes[1]).TextFrame;
    // متن هر دو placeholder را تغییر می‌دهد
    tf1.Text = "Center Align by Aspose";
    tf2.Text = "Center Align by Aspose";
    // دریافت اولین پاراگراف از placeholderها
    IParagraph para1 = tf1.Paragraphs[0];
    IParagraph para2 = tf2.Paragraphs[0];
    // هم‌ترازی پاراگراف متن به مرکز
    para1.ParagraphFormat.Alignment = TextAlignment.Center;
    para2.ParagraphFormat.Alignment = TextAlignment.Center;
    //نوشتن ارائه به عنوان یک فایل PPTX
    pres.Save("Centeralign_out.pptx", SaveFormat.Pptx);
}
```

## ویژگی BulletFormat.Height

برگرداندن یا تنظیم ارتفاع گلوله یک پاراگراف که ارث‌برداری ندارد. مقدار float.NaN تعیین می‌کند که گلوله ارتفاع را از اولین بخش پاراگراف به ارث می‌برد. قابل خواندن/نوشتن Single.

```csharp
public float Height { get; set; }
```

### توضیحات

## SlideShowSettings.SlideShowType ویژگی

دریافت یا تنظیم نوع نمایش اسلاید. نمایانده شده توسط اجداد `SlideShowType` زیر: [`BrowsedAtKiosk`](../../browsedatkiosk), [`PresentedBySpeaker`](../../presentedbyspeaker) and [`BrowsedByIndividual`](../../browsedbyindividual)

```csharp
public SlideShowType SlideShowType { get; set; }
```

### مثال‌ها

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // برای تنظیم نوع "Browsed at a kiosk (full screen)" 
    pres.SlideShowSettings.SlideShowType = new BrowsedAtKiosk();
    
    // برای تنظیم نوع "Browsed by individual (window)" 
    pres.SlideShowSettings.SlideShowType = new BrowsedByIndividual();
    
    // برای تنظیم نوع "Presented by a speaker (full screen)" 
    pres.SlideShowSettings.SlideShowType = new PresentedBySpeaker();
}
```

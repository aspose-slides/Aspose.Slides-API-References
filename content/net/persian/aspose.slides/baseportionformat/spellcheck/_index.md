## BasePortionFormat.SpellCheck خصوصیت

مقداری را که نشان می‌دهد آیا بررسی املا برای بخش متن فعال است دریافت یا تنظیم می‌کند. زمانی که این خصوصیت به false تنظیم شود، بررسی املا برای عناصر متن سرکوب می‌شود. زمانی که به true تنظیم شود، اجازه بررسی املا داده می‌شود. مقدار پیش‌فرض `false` است.

```csharp
public bool SpellCheck { get; set; }
```

### مثال‌ها

مثال بعدی نشان می‌دهد که چگونه پرچم SpellCheck را قبل از ذخیرهٔ ارائه فعال می‌کنید:

```csharp
[C#]
using (var pres = new Presentation("input.pptx"))
{
    // دسترسی به اولین بخش متن داخل اولین شکل در اولین اسلاید
    var portion = ((AutoShape)pres.Slides[0].Shapes[0]).TextFrame.Paragraphs[0].Portions[0];
    // فعال‌سازی بررسی املا برای این بخش متن
    portion.PortionFormat.SpellCheck = true;
    // ذخیرهٔ ارائهٔ تغییر یافته
    pres.Save("output-with-spellcheck.pptx", SaveFormat.Pptx);
}
```

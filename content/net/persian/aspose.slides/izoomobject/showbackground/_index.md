## IZoomObject.ShowBackground ویژگی

مقداری را دریافت یا تنظیم می‌کند که تعیین می‌کند آیا Zoom از پس‌زمینهٔ اسلاید مقصد استفاده می‌کند. خواندن/نوشتن Boolean. مقدار پیش‌فرض: true

```csharp
public bool ShowBackground { get; set; }
```

### مثال‌ها

این مثال حذف پس‌زمینهٔ یک تصویر از یک شیء Zoom را نشان می‌دهد:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
    zoomFrame.ShowBackground = false;
}
```

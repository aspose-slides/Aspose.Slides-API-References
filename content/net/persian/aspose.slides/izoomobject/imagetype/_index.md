## IZoomObject.ImageType ویژگی

### توضیحات

مشخص می‌کند که آیا شی Zoom از پیش‌نمایش اسلاید یا تصویر جلد استفاده می‌کند.

### مثال‌ها

این مثال نشان می‌دهد که چگونه Image Type به مقدار Preview تغییر می‌یابد. در این حالت تصویر فعلی یک شی Zoom به تصویر اسلاید تغییر می‌کند:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    IPPImage image = pres.Images.AddImage(Image.FromFile("image.png"));
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1], image);
    zoomFrame.ImageType = ZoomImageType.Preview;
}
```

## IAudioFrame.FadeOutDuration ویژگی

مدت زمان پایان محو شدن (fade-out) رسانه را بر حسب میلی‌ثانیه مشخص می‌کند. خواندنی/نوشتنی Single.

```csharp
public float FadeOutDuration { get; set; }
```

### مثال‌ها

مثال:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // افزودن فریم صوتی
    IAudio audio = pres.Audios.AddAudio(File.ReadAllBytes("sampleaudio.mp3"));
    IAudioFrame audioFrame = pres.Slides[0].Shapes.AddAudioFrameEmbedded(50, 50, 100, 100, audio);

    // تنظیم مدت زمان پایان محو شدن به 500 میلی‌ثانیه
    audioFrame.FadeOutDuration = 500f;

    pres.Save("AudioFrameFade_out.pptx", SaveFormat.Pptx);
}
```

## PortionCollection.CopyTo متد

عناصر ICollection را به یک Array کپی می‌کند که از یک ایندکس مشخص در Array آغاز می‌شود.

```csharp
public void CopyTo(IPortion[] array, int arrayIndex)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| array | IPortion[] | Array تک‌بعدی که مقصد عناصری است که از ICollection کپی می‌شوند. Array باید ایندکس‌گذاری صفر مبنا داشته باشد. |
| arrayIndex | Int32 | ایندکس صفر مبنا در *array* که کپی از آن آغاز می‌شود. |

### استثناها

| استثنا | شرط |
| --- | --- |
| ArgumentNullException | *array* برابر null است. |
| ArgumentOutOfRangeException | *arrayIndex* کمتر از 0 است. |
| ArgumentException | تعداد عناصری که در ICollection منبع وجود دارد بیش از فضای موجود از *arrayIndex* تا انتهای *array* مقصد است. |

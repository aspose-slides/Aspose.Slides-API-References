## PresentationInfo.CheckWriteProtection method

بررسی می‌کند آیا رمز عبور برای ویرایش در یک ارائهٔ محافظت‌شده از نوشتن صحیح است یا خیر.

```csharp
public bool CheckWriteProtection(string password)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| password | String | رمز عبوری که باید بررسی شود. |

### مقدار بازگشت

True اگر ارائهٔ محافظت‌شده از نوشتن باشد و رمز عبور درست باشد. در غیر این صورت False.

### استثناها

| استثنا | شرط |
| --- | --- |
| InvalidOperationException |  |

### توضیحات

1. قبل از فراخوانی این متد باید ویژگی [`IsWriteProtected`](../iswriteprotected) را بررسی کنید.  
2. هنگامی که password مقدار null یا خالی داشته باشد، این متد False برمی‌گرداند.

### مثال‌ها

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
if (info.IsWriteProtected == NullableBool.True)
{
    bool isWriteProtectedByPassword = info.CheckWriteProtection("my_password");
}
```

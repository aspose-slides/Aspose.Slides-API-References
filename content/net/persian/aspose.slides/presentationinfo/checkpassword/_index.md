## PresentationInfo.CheckPassword متد

بررسی می‌کند که آیا رمز عبور برای ارائه‌ای که با رمز باز محافظت شده است صحیح است یا خیر.

```csharp
public bool CheckPassword(string password)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| password | String | رمز عبوری که باید بررسی شود. |

### مقدار بازگشت

True اگر ارائه با رمز باز محافظت شده باشد و رمز عبور صحیح باشد و در غیر این صورت false.

### توضیحات

زمانی که رمز عبور null یا خالی باشد، این متد false برمی‌گرداند.

### مثال‌ها

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
bool isPasswordCorrect = info.CheckPassword("my_password");
```

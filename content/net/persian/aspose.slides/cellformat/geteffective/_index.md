## CellFormat.GetEffective متد

ویژگی‌های قالب‌بندی سلول جدول را با در نظر گرفتن ارث‌بری و سبک‌های جدول دریافت می‌کند.

```csharp
public ICellFormatEffectiveData GetEffective()
```

### مقدار برگشتی

یک [`ICellFormatEffectiveData`](../../icellformateffectivedata).

### مثال‌ها

این مثال نحوه دریافت قالب پر کردن مؤثر برای بخش‌های مختلف منطق جدول را نشان می‌دهد. لطفاً توجه داشته باشید که قالب‌بندی سلول همواره اولویت بالاتری نسبت به قالب‌بندی سطر دارد، سطر نسبت به ستون اولویت بالاتری دارد، ستون نسبت به کل جدول اولویت بالاتری دارد. بنابراین در نهایت ویژگی‌های CellFormatEffectiveData برای رسم جدول استفاده می‌شوند. کد زیر فقط یک مثال از API است.

```csharp
[C#]
using (Presentation pres = new Presentation(@"MyPresentation.pptx"))
{
    ITable tbl = pres.Slides[0].Shapes[0] as ITable;
    IFillFormatEffectiveData tableFillFormatEffective = tbl.TableFormat.GetEffective().FillFormat;
    IFillFormatEffectiveData rowFillFormatEffective = tbl.Rows[0].RowFormat.GetEffective().FillFormat;
    IFillFormatEffectiveData columnFillFormatEffective = tbl.Columns[0].ColumnFormat.GetEffective().FillFormat;
    IFillFormatEffectiveData cellFillFormatEffective = tbl[0, 0].CellFormat.GetEffective().FillFormat;
    /* خروجی و مقایسه */
}
```

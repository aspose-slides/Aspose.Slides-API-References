### مثال‌ها

مثال‌های زیر نشان می‌دهند چگونه گزینه‌های پخش صدا را تغییر دهید.

```csharp
using (Presentation pres = new Presentation("AudioFrameEmbed_out.pptx"))
{
    // دریافت شکل AudioFrame
    AudioFrame audioFrame = (AudioFrame)pres.Slides[0].Shapes[0];
    // تنظیم حالت پخش برای پخش با کلیک
    audioFrame.PlayMode = AudioPlayModePreset.OnClick;
    // تنظیم حجم به کم
    audioFrame.Volume = AudioVolumeMode.Low;
    // تنظیم صدا برای پخش در تمام اسلایدها
    audioFrame.PlayAcrossSlides = true;
    // غیرفعال کردن حلقه برای صدا
    audioFrame.PlayLoopMode = false;
    // مخفی کردن AudioFrame در حین نمایش اسلاید
    audioFrame.HideAtShowing = true;
    // عقب‌گرد صدا به ابتدا پس از پخش
    audioFrame.RewindAudio = true;
    // ذخیره فایل PowerPoint بر روی دیسک
    pres.Save("AudioFrameEmbed_changed.pptx", SaveFormat.Pptx);
}
```

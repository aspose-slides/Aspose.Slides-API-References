## PersistenceType شمارش

روشی را که برای ذخیره ویژگی‌های ActiveX control استفاده می‌شود، مشخص می‌کند.

### مقادیر

| نام | مقدار | شرح |
| --- | --- | --- |
| NotDefined | `-1` | شناسهٔ Persistance مشخص نشده است. |
| PersistPropertyBag | `0` | مشخص می‌کند که ActiveX control با استفاده از حفظ‌سازی مبتنی بر property-bag ذخیره می‌شود. حفظ‌سازی مبتنی بر property-bag یک ActiveX control را از طریق مجموعه‌ای از جفت‌های نام و مقدار که داده‌های ذخیره‌شده توسط ActiveX control را مشخص می‌کنند، ذخیره می‌کند. |
| PersistStream | `1` | مشخص می‌کند که ActiveX control با استفاده از حفظ‌سازی مبتنی بر جریان ذخیره می‌شود که از مقداردهی اولیهٔ ActiveX control به حالت پیش‌فرض پشتیبانی نمی‌کند. |
| PersistStreamInit | `2` | مشخص می‌کند که ActiveX control با استفاده از حفظ‌سازی مبتنی بر جریان ذخیره می‌شود که از مقداردهی اولیهٔ ActiveX control به حالت پیش‌فرض پشتیبانی می‌کند. |
| PersistStorage | `3` | مشخص می‌کند که ActiveX control با استفاده از حفظ‌سازی مبتنی بر ذخیره‌سازی ذخیره می‌شود. |

## TableFormat.GetEffective متد

ویژگی‌های قالب‌بندی جدول مؤثر را که شامل ارث‌بری و اعمال سبک‌های جدول هستند، دریافت می‌کند.

```csharp
public ITableFormatEffectiveData GetEffective()
```

### مقدار بازگشت

A [`ITableFormatEffectiveData`](../../itableformateffectivedata).

### مثال‌ها

این مثال نشان می‌دهد که چگونه قالب پر کردن مؤثر برای قسمت‌های مختلف منطق جدول به دست می‌آید. لطفاً توجه داشته باشید که قالب‌بندی سلول همیشه دارای اولویت بالاتری نسبت به قالب‌بندی ردیف است، ردیف نسبت به ستون، ستون نسبت به کل جدول. بنابراین در نهایت ویژگی‌های CellFormatEffectiveData همیشه برای رسم جدول استفاده می‌شوند. کد زیر تنها یک مثال از API است.

```csharp
[C#]
using (Presentation pres = new Presentation(@"MyPresentation.pptx"))
{
    ITable tbl = pres.Slides[0].Shapes[0] as Table;
    IFillFormatEffectiveData tableFillFormatEffective = tbl.TableFormat.GetEffective().FillFormat;
    IFillFormatEffectiveData rowFillFormatEffective = tbl.Rows[0].RowFormat.GetEffective().FillFormat;
    IFillFormatEffectiveData columnFillFormatEffective = tbl.Columns[0].ColumnFormat.GetEffective().FillFormat;
    IFillFormatEffectiveData cellFillFormatEffective = tbl[0, 0].CellFormat.GetEffective().FillFormat;
    /* خروجی و مقایسه */
}
```

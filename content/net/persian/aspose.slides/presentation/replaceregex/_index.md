## Presentation.ReplaceRegex متد

تمام تطابق‌های عبارت منظم را با رشته مشخص شده جایگزین می‌کند.

```csharp
public void ReplaceRegex(Regex regex, string newText, IFindResultCallback callback)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| regex | Regex | عبارت منظم Regex برای به‌دست آوردن رشته‌هایی که باید جایگزین شوند. |
| newText | String | رشته برای جایگزینی تمام رخدادهای رشته‌های جایگزین‌شده. |
| callback | IFindResultCallback | شیء callback برای دریافت نتایج جستجو [`IFindResultCallback`](../../ifindresultcallback). |

### مثال‌ها

نمونه کد زیر نشان می‌دهد چگونه متن را با استفاده از عبارت منظم و رشته مشخص شده جایگزین کنیم.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// تمام کلماتی که 10 یا بیشتر کاراکتر دارند را با '***' جایگزین کنید
	presentation.ReplaceRegex(regex, "***", null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

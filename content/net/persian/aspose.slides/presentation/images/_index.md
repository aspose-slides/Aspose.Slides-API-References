## Presentation.Images ویژگی

مجموعهٔ تمام تصاویر در ارائه را برمی‌گرداند. فقط-خواندنی [`IImageCollection`](../../iimagecollection).

### مثال‌ها

مثال‌های زیر نشان می‌دهند چگونه تصویر را به صورت BLOB به ارائهٔ PowerPoint اضافه کنیم.

```csharp
[C#]
string pathToLargeImage = "large_image.jpg";
// یک ارائهٔ جدید ایجاد می‌کند که تصویر به آن اضافه خواهد شد.
using (Presentation pres = new Presentation())
{
	using (FileStream fileStream = new FileStream(pathToLargeImage, FileMode.Open))
	{
		// تصویر را به ارائه اضافه می‌کنیم - رفتار KeepLocked را انتخاب می‌کنیم زیرا ما
		// قصد دسترسی به فایل "largeImage.png" را نداریم.
		IPPImage img = pres.Images.AddImage(fileStream, LoadingStreamBehavior.KeepLocked);
		pres.Slides[0].Shapes.AddPictureFrame(ShapeType.Rectangle, 0, 0, 300, 200, img);
		// ارائه را ذخیره می‌کند. در حالی که یک ارائهٔ بزرگ خروجی می‌شود، مصرف حافظه
		// در طول دورهٔ حیات شی pres کم می‌ماند
		pres.Save("presentationWithLargeImage.pptx", SaveFormat.Pptx);
	}
}
```

مثال‌های زیر یک پیوند به تصویر در یک ارائهٔ PowerPoint اضافه می‌کنند.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // تصویر را به ارائه اضافه می‌کند
    IPPImage image = pres.Images.AddImage(File.ReadAllBytes("image.png"));
    // قاب تصویر را در اسلاید 1 بر اساس تصویری که قبلاً اضافه شده ایجاد می‌کند
    IPictureFrame pictureFrame = pres.Slides[0].Shapes.AddPictureFrame(ShapeType.Rectangle, 10, 10, 100, 100, image);
    pictureFrame.HyperlinkClick = new Hyperlink("https://www.aspose.com/");
    pictureFrame.HyperlinkClick.Tooltip = "More than 70% Fortune 100 companies trust Aspose APIs";
    pres.Save("pres-out.pptx", SaveFormat.Pptx);
}
```

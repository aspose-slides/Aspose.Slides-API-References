## Presentation.HighlightRegex متد

تمام مطابقت‌های عبارت منظم را با رنگ تعیین‌شده برجسته می‌کند.

```csharp
public void HighlightRegex(Regex regex, Color highlightColor, IFindResultCallback callback)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| regex | Regex | عبارت منظم Regex برای دریافت رشته‌هایی که باید برجسته شوند. |
| highlightColor | Color | رنگ برای برجسته‌سازی متن. |
| callback | IFindResultCallback | شیء callback برای دریافت نتایج جستجو [`IFindResultCallback`](../../ifindresultcallback). |

### نمونه‌ها

نمونه کد زیر نشان می‌دهد چگونه می‌توان متن را در یک ارائه PowerPoint با استفاده از یک عبارت منظم برجسته کرد.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// برجسته‌سازی تمام کلمات با ۱۰ یا بیشتر کاراکتر
	presentation.HighlightRegex(regex, Color.Blue, null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

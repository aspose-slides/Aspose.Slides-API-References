## متد ParagraphCollection.CopyTo

عناصر ICollection را به یک Array کپی می‌کند، شروع از یک ایندکس خاص در Array.

```csharp
public void CopyTo(IParagraph[] array, int arrayIndex)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| array | IParagraph[] | Array تک‌بعدی که مقصد عناصری است که از ICollection کپی می‌شوند. Array باید دارای اندیس‌گذاری صفر مبنا باشد. |
| arrayIndex | Int32 | ایندکس صفر مبنا در *array* که کپی از آن آغاز می‌شود. |

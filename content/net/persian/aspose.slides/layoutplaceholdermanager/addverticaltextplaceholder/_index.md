## LayoutPlaceholderManager.AddVerticalTextPlaceholder متد

یک shape placeholder جدید را به اسلاید طرح اضافه می‌کند تا محتوای متن را در جهت عمودی نگه دارد.

```csharp
public IAutoShape AddVerticalTextPlaceholder(float x, float y, float width, float height)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| x | Single | مختصات X شکل placeholder جدید. |
| y | Single | مختصات Y شکل placeholder جدید. |
| width | Single | عرض شکل placeholder جدید. |
| height | Single | ارتفاع شکل placeholder جدید. |

### مقدار برگشتی

ایجاد شد [`IAutoShape`](../../iautoshape) با یک placeholder متن (عمودی).

### مثال‌ها

مثال زیر نشان می‌دهد چگونه shape placeholder متن (عمودی) را به اسلاید طرح اضافه کنید.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalTextPlaceholder(20, 20, 300, 500);
}
```

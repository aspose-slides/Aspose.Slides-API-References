## متد LayoutPlaceholderManager.AddOnlineImagePlaceholder

یک شکل نگهدارنده جدید به اسلاید طرح‌بندی اضافه می‌کند تا تصویر آنلاین را در خود نگه دارد.

```csharp
public IAutoShape AddOnlineImagePlaceholder(float x, float y, float width, float height)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| x | Single | مختصات X شکل نگهدارنده جدید. |
| y | Single | مختصات Y شکل نگهدارنده جدید. |
| width | Single | عرض شکل نگهدارنده جدید. |
| height | Single | ارتفاع شکل نگهدارنده جدید. |

### مقدار بازگشت

یک [`IAutoShape`](../../iautoshape) با نگهدارنده تصویر آنلاین ایجاد شد.

### مثال‌ها

مثال زیر نشان می‌دهد چگونه شکل نگهدارنده تصویر آنلاین را به اسلاید طرح‌بندی اضافه کنید.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddOnlineImagePlaceholder(20, 20, 200, 200);
}
```

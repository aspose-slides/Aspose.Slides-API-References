## LayoutPlaceholderManager.AddPicturePlaceholder متد

یک شکل جای‌دار جدید به اسلاید طرح اضافه می‌کند تا یک تصویر را نگه دارد.

```csharp
public IAutoShape AddPicturePlaceholder(float x, float y, float width, float height)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| x | Single | مختصات X شکل جای‌دار جدید. |
| y | Single | مختصات Y شکل جای‌دار جدید. |
| width | Single | عرض شکل جای‌دار جدید. |
| height | Single | ارتفاع شکل جای‌دار جدید. |

### مقدار بازگشتی

یک [`IAutoShape`](../../iautoshape) با یک جای‌دار Picture ایجاد شد.

### مثال‌ها

مثال زیر نشان می‌دهد چگونه شکل جای‌دار Picture را به اسلاید طرح اضافه کنید.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddPicturePlaceholder(20, 20, 200, 200);
}
```

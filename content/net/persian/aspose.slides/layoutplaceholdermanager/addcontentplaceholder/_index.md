## LayoutPlaceholderManager.AddContentPlaceholder متد

یک shape placeholder جدید به اسلاید طرح اضافه می‌کند تا محتوا مانند تصویر، جدول، رسانه یا متن را نگه دارد.

```csharp
public IAutoShape AddContentPlaceholder(float x, float y, float width, float height)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| x | Single | مختصات X shape placeholder جدید. |
| y | Single | مختصات Y shape placeholder جدید. |
| width | Single | عرض shape placeholder جدید. |
| height | Single | ارتفاع shape placeholder جدید. |

### مقدار بازگشت

ایجاد شد [`IAutoShape`](../../iautoshape) با یک placeholder از نوع Content.

### مثال‌ها

مثال زیر نشان می‌دهد چگونه shape placeholder Content را به اسلاید طرح اضافه کنیم.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddContentPlaceholder(20, 20, 500, 300);
}
```

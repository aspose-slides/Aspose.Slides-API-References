## LayoutPlaceholderManager.AddTextPlaceholder متد

یک شکل متغیر جایگزین جدید را به اسلاید قالب اضافه می‌کند تا محتوای متنی را نگه دارد.

```csharp
public IAutoShape AddTextPlaceholder(float x, float y, float width, float height)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| x | Single | مختصات X شکل متغیر جایگزین جدید. |
| y | Single | مختصات Y شکل متغیر جایگزین جدید. |
| width | Single | عرض شکل متغیر جایگزین جدید. |
| height | Single | ارتفاع شکل متغیر جایگزین جدید. |

### مقدار بازگشتی

ایجاد شد [`IAutoShape`](../../iautoshape) با یک متغیر جایگزین متن.

### مثال‌ها

مثال زیر نشان می‌دهد چگونه شکل متغیر جایگزین متن را به اسلاید قالب اضافه کنیم.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTextPlaceholder(20, 20, 500, 300);
}
```

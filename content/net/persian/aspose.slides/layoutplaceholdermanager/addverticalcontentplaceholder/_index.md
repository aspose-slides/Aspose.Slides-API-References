## LayoutPlaceholderManager.AddVerticalContentPlaceholder متد

یک شکل نگهدارنده جدید به اسلایدچیدمان اضافه می‌کند تا محتوا را در جهت عمودی نگه دارد، مانند تصویر، جدول، رسانه یا متن.

```csharp
public IAutoShape AddVerticalContentPlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| x | Single | مختصات X شکل نگهدارنده جدید. |
| y | Single | مختصات Y شکل نگهدارنده جدید. |
| width | Single | عرض شکل نگهدارنده جدید. |
| height | Single | ارتفاع شکل نگهدارنده جدید. |

### مقدار برگشتی

یک [`IAutoShape`](../../iautoshape) با یک Content (Vertical) placeholder ایجاد شد.

### مثال‌ها

مثال زیر نشان می‌دهد چگونه شکل نگهدارنده Content (Vertical) را به اسلایدچیدمان اضافه کنید.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalContentPlaceholder(20, 20, 300, 500);
}
```

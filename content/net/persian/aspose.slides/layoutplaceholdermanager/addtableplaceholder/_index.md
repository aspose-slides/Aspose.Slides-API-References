## LayoutPlaceholderManager.AddTablePlaceholder متد

یک شکل پیش‌نمایش جدید به اسلاید طرح‌بندی اضافه می‌کند تا یک جدول را در خود داشته باشد.

```csharp
public IAutoShape AddTablePlaceholder(float x, float y, float width, float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| x | Single | مختصات X شکل پیش‌نمایش جدید. |
| y | Single | مختصات Y شکل پیش‌نمایش جدید. |
| width | Single | عرض شکل پیش‌نمایش جدید. |
| height | Single | ارتفاع شکل پیش‌نمایش جدید. |

### مقدار بازگشت

[`IAutoShape`](../../iautoshape) با یک پیش‌نمایش Table ایجاد شد.

### نمونه‌ها

مثال زیر نشان می‌دهد چگونه شکل پیش‌نمایش Table را به اسلاید طرح‌بندی اضافه کنیم.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTablePlaceholder(20, 20, 500, 200);
}
```

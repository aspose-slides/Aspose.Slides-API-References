## LayoutPlaceholderManager.AddMediaPlaceholder متد

یک شکل‌گیر جای‌گیر جدید به اسلاید قالب اضافه می‌کند تا یک شی رسانه‌ای را در خود نگه دارد.

```csharp
public IAutoShape AddMediaPlaceholder(float x, float y, float width, float height)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| x | Single | مختصات X شکل جای‌گیر جدید. |
| y | Single | مختصات Y شکل جای‌گیر جدید. |
| width | Single | عرض شکل جای‌گیر جدید. |
| height | Single | ارتفاع شکل جای‌گیر جدید. |

### مقدار بازگردانده شده

یک [`IAutoShape`](../../iautoshape) با Media placeholder ایجاد شد.

### مثال‌ها

مثال زیر نشان می‌دهد چگونه شکل Media placeholder را به اسلاید قالب اضافه کنیم.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddMediaPlaceholder(20, 20, 200, 200);
}
```

## LayoutPlaceholderManager.AddChartPlaceholder متد

یک شکل پیش‌گیر جدید به اسلاید چیدمان اضافه می‌کند تا یک نمودار را در خود نگه دارد.

```csharp
public IAutoShape AddChartPlaceholder(float x, float y, float width, float height)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| x | Single | مختصات X شکل پیش‌گیر جدید. |
| y | Single | مختصات Y شکل پیش‌گیر جدید. |
| width | Single | عرض شکل پیش‌گیر جدید. |
| height | Single | ارتفاع شکل پیش‌گیر جدید. |

### مقدار بازگشتی

ایجاد شد [`IAutoShape`](../../iautoshape) با یک پیش‌گیر نمودار.

### مثال‌ها

مثال زیر نشان می‌دهد چگونه شکل پیش‌گیر نمودار را به اسلاید چیدمان اضافه کنیم.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddChartPlaceholder(20, 20, 200, 200);
}
```

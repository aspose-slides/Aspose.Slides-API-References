## LayoutPlaceholderManager.AddSmartArtPlaceholder متد

یک شکل محل‌نگهدار جدید به اسلاید طرح‌بندی اضافه می‌کند تا یک نمودار SmartArt را در خود نگه دارد.

```csharp
public IAutoShape AddSmartArtPlaceholder(float x, float y, float width, float height)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| x | Single | مختصات X شکل محل‌نگهدار جدید. |
| y | Single | مختصات Y شکل محل‌نگهدار جدید. |
| width | Single | عرض شکل محل‌نگهدار جدید. |
| height | Single | ارتفاع شکل محل‌نگهدار جدید. |

### مقدار بازگشتی

یک [`IAutoShape`](../../iautoshape) با یک محل‌نگهدار SmartArt ایجاد شد.

### مثال‌ها

مثال زیر نشان می‌دهد چگونه شکل محل‌نگهدار SmartArt را به اسلاید طرح‌بندی اضافه کنیم.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddSmartArtPlaceholder(20, 20, 200, 200);
}
```

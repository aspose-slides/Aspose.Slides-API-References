## SetLicense(string) {#setlicense_1}

مجوز مؤلفه را فعال می‌کند.

```csharp
public void SetLicense(string licenseName)
```

| Parameter | Type | Description |
| --- | --- | --- |
| licenseName | String | می‌تواند نام کامل یا کوتاه فایل یا نام یک منبع جاسازی‌شده باشد. برای تغییر به حالت ارزیابی یک رشتهٔ خالی استفاده کنید. |

### نکات

سعی می‌کند مجوز را در مکان‌های زیر پیدا کند:

1. مسیر صریح.
2. پوشهٔ مجموعه مؤلفه.
3. پوشهٔ مجموعه فراخوانی‌گر مشتری.
4. پوشهٔ مجموعه ورودی.
5. منبعی جاسازی‌شده در مجموعه فراخوانی‌گر مشتری.

**توجه:**در .NET Compact Framework، سعی می‌کند مجوز را فقط در این مکان‌ها پیدا کند:

1. مسیر صریح.
2. منبعی جاسازی‌شده در مجموعه فراخوانی‌گر مشتری.

### مثال‌ها

در این مثال، سعی می‌شود فایلی با نام MyLicense.lic در پوشهٔ حاوی مؤلفه، در پوشهٔ حاوی مجموعه فراخوانی، در پوشهٔ مجموعه ورودی و سپس در منابع جاسازی‌شدهٔ مجموعه فراخوانی پیدا شود.

```csharp
[C#]

License license = new License();
license.SetLicense("MyLicense.lic");


[Visual Basic]

Dim license As License = New License
license.SetLicense("MyLicense.lic")
```

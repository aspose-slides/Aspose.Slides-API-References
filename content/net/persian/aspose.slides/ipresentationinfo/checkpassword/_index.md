## IPresentationInfo.CheckPassword method

بررسی می‌کند که آیا رمز عبور برای یک ارائه که با رمز باز محافظت شده است صحیح است یا خیر.

```csharp
public bool CheckPassword(string password)
```

| پارامتر | نوع | توضیحات |
| --- | --- | --- |
| password | String | password را برای بررسی. |

### مقدار بازگشت

True اگر ارائه با رمز باز محافظت شده باشد و password صحیح باشد و در غیر این صورت false.

### توضیحات

هنگامی که password مقدار null یا خالی باشد، این متد false بر می‌گرداند.

### مثال‌ها

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo("pres.pptx");
bool isPasswordCorrect = info.CheckPassword("my_password");
```

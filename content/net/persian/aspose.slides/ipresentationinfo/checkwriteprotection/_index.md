## IPresentationInfo.CheckWriteProtection متد

بررسی می‌کند که آیا رمز عبور برای اصلاح، برای ارائه‌ای که محافظت نوشتاری دارد، صحیح است یا خیر.

```csharp
public bool CheckWriteProtection(string password)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| password | String | رمز عبوری که باید بررسی شود. |

### مقدار برگشتی

True اگر ارائه محافظت نوشتاری باشد و رمز عبور صحیح باشد. False در غیر این صورت.

### استثناها

| استثنا | شرط |
| --- | --- |
| InvalidOperationException |  |

### توضیحات

1. باید ویژگی [`IsWriteProtected`](../iswriteprotected) را قبل از فراخوانی این متد بررسی کنید. 2. وقتی مقدار password برابر null یا خالی باشد، این متد false برمی‌گرداند.

### مثال‌ها

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
if (info.IsWriteProtected == NullableBool.True)
{
    bool isWriteProtectedByPassword = info.CheckWriteProtection("my_password");
}
```

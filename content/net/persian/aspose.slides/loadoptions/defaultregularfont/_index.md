## ویژگی LoadOptions.DefaultRegularFont

قلم Regular را که در صورت عدم یافتن قلم منبع استفاده می‌شود، برمی‌گرداند یا تنظیم می‌کند. خواندن/نوشتن String.

```csharp
public string DefaultRegularFont { get; set; }
```

### مثال‌ها

مثال زیر نشان می‌دهد چگونه می‌توان قلم‌های پیش‌فرض را برای رندر کردن ارائه PowerPoint تنظیم کرد.

```csharp
[C#]
// از گزینه‌های بارگذاری برای تعریف قلم‌های پیش‌فرض عادی و آسیایی استفاده کنید
LoadOptions loadOptions = new LoadOptions(LoadFormat.Auto);
loadOptions.DefaultRegularFont = "Wingdings";
loadOptions.DefaultAsianFont = "Wingdings";
// بارگذاری ارائه
using (Presentation pptx = new Presentation("DefaultFonts.pptx", loadOptions))
{
    // ایجاد تصویر بندانگشتی اسلاید
    pptx.Slides[0].GetThumbnail(1, 1).Save("output_out.png", ImageFormat.Png);
    // ایجاد PDF
    pptx.Save("output_out.pdf", SaveFormat.Pdf);
    // ایجاد XPS
    pptx.Save("output_out.xps", SaveFormat.Xps);
}
```

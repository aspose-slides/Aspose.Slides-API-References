## IFontsManager.FontFallBackRulesCollection ویژگی

نمایش‌دهندهٔ مجموعه‌ای از قوانین FontFallBack کاربر برای مدیریت مجموعه‌های فونت‌ها به منظور تعویض‌های صحیح توسط عملکرد fallback خواندنی/نوشتنی [`IFontFallBackRulesCollection`](../../ifontfallbackrulescollection).

```csharp
public IFontFallBackRulesCollection FontFallBackRulesCollection { get; set; }
```

### مثال‌ها

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    // دریافت مجموعه قوانین خالی یا از پیش مقداردهی شده از FontsManager
    IFontFallBackRulesCollection rulesList = pres.FontsManager.FontFallBackRulesCollection;

    // افزودن قوانین به مجموعه
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));

    // یا 
    // مقداردهی یک نمونه جدید از مجموعه قوانین
    IFontFallBackRulesCollection rulesList = new FontFallBackRulesCollection();

    // افزودن قوانین به مجموعه
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));

    // و جایگزینی مجموعه موجود با مجموعه جدید در FontsManager 
    pres.FontsManager.FontFallBackRulesCollection = rulesList;
}
```

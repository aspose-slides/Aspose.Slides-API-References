## IPresentation.ReplaceRegex متد

تمام موارد تطابق با عبارت منظم را با رشتهٔ مشخص‌شده جایگزین می‌کند.

```csharp
public void ReplaceRegex(Regex regex, string newText, IFindResultCallback callback)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| regex | Regex | عبارت منظم Regex برای دریافت رشته‌هایی که باید جایگزین شوند. |
| newText | String | رشته‌ای برای جایگزینی تمام رخدادهای رشته‌های قابل جایگزینی. |
| callback | IFindResultCallback | شیء callback برای دریافت نتایج جستجو [`IFindResultCallback`](../../ifindresultcallback). |

### مثال‌ها

نمونه کد زیر نشان می‌دهد چگونه متن را با استفاده از عبارت منظم و رشتهٔ مشخص‌شده جایگزین کنیم.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// تمام کلمات با ۱۰ یا بیشتر کاراکتر را با '***' جایگزین می‌کند
	presentation.ReplaceRegex(regex, "***", null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

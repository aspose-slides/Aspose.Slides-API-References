## IPresentation.HighlightRegex متد

عبارت‌های مطابق با عبارت منظم را با رنگ مشخص شده برجسته می‌کند.

```csharp
public void HighlightRegex(Regex regex, Color highlightColor, IFindResultCallback callback)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| regex | Regex | عبارت منظم Regex برای دریافت رشته‌های مورد برجسته‌سازی. |
| highlightColor | Color | رنگی که متن را برجسته می‌کند. |
| callback | IFindResultCallback | شیء callback برای دریافت نتایج جستجو [`IFindResultCallback`](../../ifindresultcallback). |

### مثال‌ها

کد نمونه زیر نشان می‌دهد چگونه می‌توان متن را در یک ارائه PowerPoint با استفاده از یک عبارت منظم برجسته کرد.

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx"))
{
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// برجسته‌کردن تمام کلماتی که 10 یا بیشتر حرف دارند
	presentation.HighlightRegex(regex, Color.Blue, null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

## ZoomObject.ShowBackground ویژگی

مقداری را که مشخص می‌کند آیا Zoom از پس‌زمینه اسلاید مقصد استفاده می‌کند یا خیر، دریافت یا تنظیم می‌کند. خواندنی/نوشتنی Boolean. مقدار پیش‌فرض: true

```csharp
public bool ShowBackground { get; set; }
```

### مثال‌ها

این مثال حذف پس‌زمینه یک تصویر از یک شی Zoom را نشان می‌دهد:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
    zoomFrame.ShowBackground = false;
}
```

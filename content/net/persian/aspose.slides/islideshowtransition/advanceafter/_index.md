## ISlideShowTransition.AdvanceAfter ویژگی

این ویژگی مشخص می‌کند که آیا اسلایدشو پس از زمان معینی به اسلاید بعدی می‌رود. خواندنی/نوشتنی Boolean.

```csharp
public bool AdvanceAfter { get; set; }
```

### مثال‌ها

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    // دریافت اولین انتقال اسلاید
    ISlideShowTransition slideTransition = pres.Slides[0].SlideShowTransition;
    
    // بررسی اینکه آیا پرچم Advance Slide After فعال است
    if (slideTransition.AdvanceAfter)
    {
        // دریافت مقدار زمان Advance Slide After
        uint advanceAfterTime = slideTransition.AdvanceAfterTime;
    }
}
```

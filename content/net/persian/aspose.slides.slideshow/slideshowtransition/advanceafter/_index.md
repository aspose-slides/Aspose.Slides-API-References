## SlideShowTransition.AdvanceAfter ویژگی

این ویژگی مشخص می‌کند که آیا نمایش اسلاید پس از زمان معینی به اسلاید بعدی منتقل می‌شود یا خیر. قابل خواندن/قابل نوشتن Boolean.

```csharp
public bool AdvanceAfter { get; set; }
```

### نمونه‌ها

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    // دریافت اولین انتقال اسلاید
    ISlideShowTransition slideTransition = pres.Slides[0].SlideShowTransition;
    
    // بررسی می‌شود آیا پرچم Advance Slide After فعال است
    if (slideTransition.AdvanceAfter)
    {
        // دریافت مقدار زمان Advance Slide After
        uint advanceAfterTime = slideTransition.AdvanceAfterTime;
    }
}
```

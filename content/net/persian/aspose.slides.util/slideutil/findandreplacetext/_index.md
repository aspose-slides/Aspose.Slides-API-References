## SlideUtil.FindAndReplaceText متد

متن را در ارائه جستجو و جایگزین می‌کند با قالب داده‌شده

```csharp
public static void FindAndReplaceText(IPresentation presentation, bool withMasters, string find, 
    string replace, PortionFormat format = null)
```

| پارامتر | نوع | توضیح |
| --- | --- | --- |
| presentation | IPresentation | ارائه اسکن‌شده. |
| withMasters | Boolean | مشخص می‌کند آیا اسلایدهای اصلی باید اسکن شوند. |
| find | String | مقدار رشته برای جستجو. |
| replace | String | مقدار رشته برای جایگزینی. |
| format | PortionFormat | قالب برای جایگزین کردن بخش متن. اگر null باشد، قالب اولین کاراکتر رشته پیدا‌شده استفاده خواهد شد. |

### مثال‌ها

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PortionFormat format = new PortionFormat
    {
        FontHeight = 24f,
        FontItalic = NullableBool.True,
        FillFormat =
        {
            FillType = FillType.Solid,
            SolidFillColor =
            {
                Color = Color.Red
            }
        }
    };
    Aspose.Slides.Util.SlideUtil.FindAndReplaceText(pres, true, "[this block] ", "my text ", format);
    pres.Save("replaced", SaveFormat.Pptx);
}
```

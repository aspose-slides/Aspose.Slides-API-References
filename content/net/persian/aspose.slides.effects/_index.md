کلاسی‌ها برای کار با افکت‌های مختلف در ارائه‌های Microsoft PowerPoint را شامل می‌شود.

## کلاس‌ها

| کلاس | توضیح |
| --- | --- |
| [AlphaBiLevel](./alphabilevel) | نمایانگر یک افکت Alpha Bi-Level است. مقادیر Alpha (Opacity) کمتر از آستانه به 0 (کاملاً شفاف) تغییر می‌کند و مقادیر Alpha برابر یا بزرگتر از آستانه به 100٪ (کاملاً مات) تغییر می‌یابد. |
| [AlphaCeiling](./alphaceiling) | نمایانگر یک افکت Alpha Ceiling است. مقادیر Alpha (opacity) بزرگ‌تر از صفر به 100٪ تغییر می‌یابد. به عبارت دیگر، هر چیزی که جزئی شفاف باشد، به طور کامل شفاف می‌شود. |
| [AlphaFloor](./alphafloor) | نمایانگر یک افکت Alpha Floor است. مقادیر Alpha (opacity) کمتر از 100٪ به صفر تغییر می‌یابد. به عبارت دیگر، هر چیزی که جزئی مات باشد، به طور کامل مات می‌شود. |
| [AlphaInverse](./alphainverse) | نمایانگر یک افکت Alpha Inverse است. مقادیر Alpha (opacity) با کسر از 100٪ معکوس می‌شوند. |
| [AlphaModulate](./alphamodulate) | نمایانگر یک افکت Alpha Modulate است. مقادیر Alpha (opacity) افکت در درصد ثابت ضرب می‌شوند. محفظهٔ افکت، افکتی را که شامل مقادیر Alpha برای تنظیم است، مشخص می‌کند. |
| [AlphaModulateFixed](./alphamodulatefixed) | نمایانگر یک افکت Alpha Modulate Fixed است. مقادیر Alpha (opacity) افکت در درصد ثابت ضرب می‌شوند. |
| [AlphaReplace](./alphareplace) | نمایانگر یک افکت Alpha Replace است. مقادیر Alpha (opacity) افکت با یک مقدار Alpha ثابت جایگزین می‌شوند. |
| [BiLevel](./bilevel) | نمایانگر یک افکت Bi-Level (سیاه/سفید) است. رنگ‌های ورودی که روشنایی آن‌ها کمتر از مقدار آستانهٔ مشخص شده است به سیاه تغییر می‌یابد. رنگ‌های ورودی که روشنایی آن‌ها برابر یا بیشتر از مقدار آستانه است به سفید تنظیم می‌شود. مقادیر Alpha تحت این افکت تغییری نمی‌کنند. |
| [Blur](./blur) | نمایانگر یک افکت Blur است که بر کل شکل، از جمله پرکننده‌اش اعمال می‌شود. تمام کانال‌های رنگ، شامل Alpha، تحت تأثیر قرار می‌گیرند. |
| [BrightnessContrast](./brightnesscontrast) | نمایانگر یک افکت BrightnessContrast است. روشنایی و کنتراست را تنظیم می‌کند. |
| [ColorChange](./colorchange) | نمایانگر یک افکت Color Change است. نمونه‌های FromColor با نمونه‌های ToColor جایگزین می‌شوند. |
| [ColorReplace](./colorreplace) | نمایانگر یک افکت Color Replacement است. تمام رنگ‌های افکت به یک رنگ ثابت تغییر می‌یابد. مقادیر Alpha تحت تأثیر قرار نمی‌گیرند. |
| [Duotone](./duotone) | نمایانگر یک افکت Duotone است. برای هر پیکسل، Color1 و Color2 از طریق یک درون‌یابی خطی ترکیب می‌شوند تا رنگ جدید پیکسل تعیین شود. |
| [EffectFactory](./effectfactory) | امکان ایجاد افکت‌ها را فراهم می‌کند. |
| [FillOverlay](./filloverlay) | نمایانگر یک افکت Fill Overlay است. یک Fill Overlay می‌تواند برای مشخص کردن یک پرکنندهٔ اضافی برای یک شیء و ترکیب دو پرکننده استفاده شود. |
| [Glow](./glow) | نمایانگر یک افکت Glow است که در آن یک حاشیهٔ رنگی محو در خارج از لبه‌های شیء اضافه می‌شود. |
| [GrayScale](./grayscale) | نمایانگر یک افکت Gray Scale است. تمام مقادیر رنگ افکت به سایه‌ای از خاکستری که متناظر با روشنایی آن‌هاست تبدیل می‌شود. مقادیر Alpha تحت تأثیر قرار نمی‌گیرند. |
| [HSL](./hsl) | نمایانگر یک افکت Hue/Saturation/Luminance است. hue، saturation و luminance می‌توانند به صورت جداگانه نسبت به مقدار فعلی خود تنظیم شوند. |
| [ImageTransformOCollectionEffectiveData](./imagetransformocollectioneffectivedata) | شیء غیرقابل تغییر که یک مجموعهٔ فقط‌خواندنی از افکت‌های تبدیل تصویر مؤثر را نشان می‌دهد. |
| [ImageTransformOperation](./imagetransformoperation) | نمایانگر یک افکت انتزاعی تبدیل تصویر است. |
| [ImageTransformOperationCollection](./imagetransformoperationcollection) | نمایانگر یک مجموعه از افکت‌های اعمال‌شده بر تصویر است. |
| [ImageTransformOperationFactory](./imagetransformoperationfactory) | امکان ایجاد عملیات‌های تبدیل تصویر را فراهم می‌کند. |
| [InnerShadow](./innershadow) | نمایانگر یک افکت Inner Shadow است. |
| [Luminance](./luminance) | نمایانگر یک افکت Luminance است. روشنایی به صورت خطی تمام رنگ‌ها را به سمت سفید یا سیاه حرکت می‌دهد. کنتراست تمام رنگ‌ها را تا نزدیک‌تر یا دورتر شدن از یکدیگر تنظیم می‌کند. |
| [OuterShadow](./outershadow) | نمایانگر یک افکت Outer Shadow است. |
| [PresetShadow](./presetshadow) | نمایانگر یک افکت Preset Shadow است. |
| [Reflection](./reflection) | نمایانگر یک افکت Reflection است. |
| [SoftEdge](./softedge) | نمایانگر یک افکت soft edge است. لبه‌های شکل محو می‌شوند، در حالی که پرکننده تحت تأثیر قرار نمی‌گیرد. |
| [Tint](./tint) | نمایانگر یک افکت Tint است. مقادیر رنگ افکت را به سمت/از سمت hue به مقدار مشخص‌شده جابه‌جا می‌کند. |

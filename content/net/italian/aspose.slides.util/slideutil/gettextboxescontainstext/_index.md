## SlideUtil.GetTextBoxesContainsText metodo

Restituisce tutti i riquadri di testo nella diapositiva specificata che contengono il testo fornito.

```csharp
public static ITextFrame[] GetTextBoxesContainsText(IBaseSlide slide, string text, 
    bool checkPlaceholderText)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| slide | IBaseSlide | La diapositiva da cercare. |
| text | String | Il testo da cercare nei riquadri di testo. |
| checkPlaceholderText | Boolean | Indica se includere i riquadri di testo vuoti, ma il cui testo segnaposto contiene il testo di ricerca. |

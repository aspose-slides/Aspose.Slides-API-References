## SlideUtil.FindAndReplaceText metodo

Trova e sostituisce il testo nella presentazione con il formato fornito

```csharp
public static void FindAndReplaceText(IPresentation presentation, bool withMasters, string find, 
    string replace, PortionFormat format = null)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| presentation | IPresentation | Presentazione scansionata. |
| withMasters | Boolean | Determina se le diapositive master devono essere scansionate. |
| find | String | Valore stringa da trovare. |
| replace | String | Valore stringa da sostituire. |
| format | PortionFormat | Formato per la porzione di testo da sostituire. Se null verrà utilizzato il formato del primo carattere della stringa trovata |

### Esempi

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PortionFormat format = new PortionFormat
    {
        FontHeight = 24f,
        FontItalic = NullableBool.True,
        FillFormat =
        {
            FillType = FillType.Solid,
            SolidFillColor =
            {
                Color = Color.Red
            }
        }
    };
    Aspose.Slides.Util.SlideUtil.FindAndReplaceText(pres, true, "[this block] ", "my text ", format);
    pres.Save("replaced", SaveFormat.Pptx);
}
```

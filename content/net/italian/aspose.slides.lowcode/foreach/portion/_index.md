## Portion(Presentation, ForEachPortionCallback) {#portion_1}

Itera ogni `Portion` nella [`Presentation`](../../../aspose.slides/presentation). Le `Portion` verranno iterate in tutti i tipi di diapositive - [`Slide`](../slide), [`MasterSlide`](../masterslide) e [`LayoutSlide`](../layoutslide)

```csharp
public static void Portion(Presentation pres, ForEachPortionCallback forEachPortion)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| pres | Presentation | Presentazione da iterare le porzioni |
| forEachPortion | ForEachPortionCallback | Callback che verrà invocato per ogni porzione |

### Esempi

```csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    ForEach.Portion(pres, (portion, para, slide, index) =>
    {
        System.Console.WriteLine($"{portion.Text}, index: {index}");
    });
} 
```

## Portion(Presentation, bool, ForEachPortionCallback) {#portion}

Itera ogni [`Portion`](../portion) nella [`Presentation`](../../../aspose.slides/presentation). Le `Portion` verranno iterate in tutti i tipi di diapositive - [`Slide`](../slide), [`MasterSlide`](../masterslide), [`LayoutSlide`](../layoutslide) e [`NotesSlide`](../../../aspose.slides/notesslide)

```csharp
public static void Portion(Presentation pres, bool includeNotes, 
    ForEachPortionCallback forEachPortion)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| pres | Presentation | Presentazione da iterare le porzioni |
| includeNotes | Boolean | Flag che indica se i NotesSlides devono essere inclusi nell'elaborazione. |
| forEachPortion | ForEachPortionCallback | Callback che verrà invocato per ogni porzione |

### Esempi

```csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    ForEach.Portion(pres, true, (portion, para, slide, index) =>
    {
        System.Console.WriteLine($"{portion.Text}, index: {index}");
    });
} 
```

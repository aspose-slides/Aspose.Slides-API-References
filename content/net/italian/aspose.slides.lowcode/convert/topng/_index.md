## ToPng(Presentation, string) {#topng}

Converte la presentazione di input in un insieme di immagini in formato PNG. Se il nome del file di output è fornito come "myPath/myFilename.png", il risultato verrà salvato come un insieme di file "myPath/myFilename_N.png", dove N è il numero di diapositiva.

```csharp
public static void ToPng(Presentation pres, string outputFileName)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| pres | Presentation | La presentazione di input. |
| outputFileName | String | Il nome del file di output. |

## ExternalResourceResolver classe

Classe di callback usata per risolvere risorse esterne durante l'importazione di documenti Html, Svg. L'uso di questo risolutore potrebbe creare una vulnerabilità quando un file HTML o SVG fornito dal client fa ottenere al software server un file locale o di rete. Usare con cautela. Si raccomanda di non specificare ExternalResourceResolver affatto (verranno letti solo gli oggetti incorporati) oppure di creare qualche sottoclasse che verifichi se l'uri specificato è valido.

```csharp
public class ExternalResourceResolver : IExternalResourceResolver
```

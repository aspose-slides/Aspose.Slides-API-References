## DataLabel.IsVisible proprietà

False indica che l'etichetta dati non è visibile (e quindi tutti i flag Show* (ShowValue, ...) sono falsi). Boolean di sola lettura.

```csharp
public bool IsVisible { get; }
```

### Osservazioni

Se l'etichetta dati è visibile è possibile nasconderla con il metodo Hide(). Tuttavia, se l'etichetta dati non è visibile (IsVisible è false) è possibile renderla visibile impostando i flag Show* (ShowValue, ...) allo stato true.

## IChartData.Categories proprietà

Restituisce le categorie primarie (o sia le categorie primarie che secondarie se la proprietà [`UseSecondaryCategories`](../usesecondarycategories) è falsa). Solo lettura [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection Categories { get; }
```

### Osservazioni

Se la proprietà [`UseSecondaryCategories`](../usesecondarycategories) è falsa, allora la proprietà [`SecondaryCategories`](../secondarycategories) restituisce null e i dati in questa proprietà `Categories` sono usati sia per le serie primarie che per le secondarie. Se la proprietà [`UseSecondaryCategories`](../usesecondarycategories) è vera, allora i dati nella proprietà [`SecondaryCategories`](../secondarycategories) sono usati per le serie secondarie e i dati in questa proprietà `Categories` sono usati per le serie primarie.

### Esempi

Esempio. Quali categorie sono associate alle serie – ChartData.Categories o ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // le categorie correlate sono series.Chart.ChartData.SecondaryCategories
}
else
{
    // le categorie correlate sono series.Chart.ChartData.Categories
}
```

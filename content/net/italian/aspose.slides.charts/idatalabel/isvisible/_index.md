## IDataLabel.IsVisible property

False significa che l’etichetta dei dati non è visibile (e quindi tutti i flag Show* (ShowValue, ...) sono falsi). Boolean di sola lettura.

```csharp
public bool IsVisible { get; }
```

### Osservazioni

Se l’etichetta dei dati è visibile, è possibile nasconderla con il metodo Hide(). Ma se l’etichetta dei dati non è visibile (IsVisible è false) è possibile renderla visibile impostando i flag Show* (ShowValue, ...) nello stato true.

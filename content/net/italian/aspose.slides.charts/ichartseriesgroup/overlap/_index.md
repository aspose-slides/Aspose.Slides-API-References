## IChartSeriesGroup.Overlap proprietà

Specifica quanto le barre e le colonne devono sovrapporsi nei grafici 2-D, come percentuale (da -100% a 100%).  
- -100%: spaziatura massima (le barre sono completamente separate).  
- 0%: le barre sono posizionate una accanto all'altra senza sovrapposizione né spaziatura.  
- 100%: sovrapposizione massima (le barre si sovrappongono completamente).  
Questa proprietà è di tipo SByte di lettura/scrittura.

```csharp
public sbyte Overlap { get; set; }
```

### Eccezioni

| eccezione | condizione |
| --- | --- |
| ArgumentOutOfRangeException | Generata quando il valore è impostato al di fuori dell'intervallo valido da -100 a 100. |

### Esempi

Il seguente esempio dimostra come impostare la sovrapposizione per un gruppo di serie di diagramma e visualizzare il diagramma risultante su un modulo:

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // Imposta la sovrapposizione al 55%

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

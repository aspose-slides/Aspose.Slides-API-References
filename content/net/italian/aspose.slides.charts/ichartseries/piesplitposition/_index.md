## IChartSeries.PieSplitPosition proprietà

Specifica un valore da utilizzare per determinare quali punti dati si trovano nella seconda torta o barra in un grafico a torta-di-torta o barra-di-torta. Viene usato insieme alla proprietà PieSplitBy. Questa proprietà non è solo di questa serie, ma di tutte le serie del gruppo di serie genitore – è la proiezione della proprietà del gruppo appropriato. Pertanto questa proprietà è sola lettura. Usa la proprietà ParentSeriesGroup per accedere al gruppo di serie genitore. Usa la proprietà di lettura/scrittura ParentSeriesGroup.PieSplitPosition per modificare il valore. Solo lettura Double.

```csharp
public double PieSplitPosition { get; }
```

### Osservazioni

Questa è la proiezione della proprietà ParentSeriesGroup.PieSplitPosition.

## IChartSeries.SecondPieSize proprietà

Specifica la dimensione della seconda fetta o barra di un grafico a torta dentro torta o di un grafico a barra dentro torta, come percentuale della dimensione della prima torta (può variare tra il 5% e il 200%). Questa è la proprietà non solo di questa serie ma di tutte le serie del gruppo di serie padre – è la proiezione della proprietà di gruppo corrispondente. Perciò questa proprietà è solo lettura. Usa la proprietà ParentSeriesGroup per accedere al gruppo di serie padre. Usa la proprietà ParentSeriesGroup.SecondPieSize lettura/scrittura per modificare il valore. Solo lettura UInt16.

```csharp
public ushort SecondPieSize { get; }
```

### Osservazioni

Questa è la proiezione della proprietà ParentSeriesGroup.SecondPieSize.

## IChartSeries.GapWidth proprietà

Specifica lo spazio tra i gruppi di barre o colonne, come percentuale della larghezza della barra o colonna. Questa è la proprietà non solo di questa serie ma di tutte le serie del gruppo di serie principale – è la proiezione della proprietà corrispondente del gruppo. Pertanto questa proprietà è di sola lettura. Utilizzare la proprietà ParentSeriesGroup per accedere al gruppo di serie principale. Utilizzare la proprietà ParentSeriesGroup.GapWidth di lettura/scrittura per modificare il valore. Solo lettura Int32.

```csharp
public int GapWidth { get; }
```

### Osservazioni

Questa è la proiezione della proprietà ParentSeriesGroup.GapWidth.

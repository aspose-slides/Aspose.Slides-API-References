## IChartSeries.HasUpDownBars proprietà

Determina se un grafico a linee o a barre ha barre su/giù. Questa è la proprietà non solo di questa serie ma di tutte le serie del gruppo di serie padre – è la proiezione della proprietà corrispondente del gruppo. Perciò questa proprietà è di sola lettura. Utilizzare la proprietà ParentSeriesGroup per accedere al gruppo di serie padre. Utilizzare la proprietà ParentSeriesGroup.UpDownBars.HasUpDownBars di lettura/scrittura per modificare il valore. Utilizzare la proprietà ParentSeriesGroup.UpDownBars per formattare le barre su/giù. Boolean di sola lettura.

public bool HasUpDownBars { get; }

## IChartSeries.BubbleSizeScale proprietà

Specifica il fattore di scala per il grafico a bolle (può essere compreso tra lo 0 e il 300 percento della dimensione predefinita). Questa è la proprietà non solo di questa serie ma di tutte le serie del gruppo di serie padre - questa è la proiezione della proprietà di gruppo appropriata. Pertanto questa proprietà è sola lettura. Utilizzare la proprietà ParentSeriesGroup per accedere al gruppo di serie padre. Utilizzare la proprietà ParentSeriesGroup.BubbleSizeScale di lettura/scrittura per modificare il valore.

```csharp
public int BubbleSizeScale { get; }
```

### Note

Questa è la proiezione della proprietà ParentSeriesGroup.BubbleSizeScale.

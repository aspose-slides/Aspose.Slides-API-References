## IChartSeries.PieSplitBy proprietà

Specifica come determinare quali punti dati sono nel secondo spicchio o barra in un grafico a torta-su-torta o a barra-su-torta. Questa è la proprietà non solo di questa serie ma di tutte le serie del gruppo di serie padre – è la proiezione della proprietà del gruppo appropriato. Pertanto questa proprietà è di sola lettura. Utilizzare la proprietà ParentSeriesGroup per accedere al gruppo di serie padre. Utilizzare la proprietà ParentSeriesGroup.PieSplitBy di lettura/scrittura per modificare il valore. Sola lettura [`PieSplitType`](../../piesplittype).

```csharp
public PieSplitType PieSplitBy { get; }
```

### Osservazioni

1) Questa è la proiezione della proprietà ParentSeriesGroup.PieSplitBy. 2) Se il valore della proprietà è PieSplitType.Custom, è possibile definire informazioni di divisione personalizzate con la proprietà ParentSeriesGroup.PieSplitCustomPoints.

## Metodo ChartDataWorkbook.CalculateFormulas

Calcola tutte le formule nella cartella di lavoro e aggiorna i valori delle celle corrispondenti.

```csharp
public void CalculateFormulas()
```

### Eccezioni

| eccezione | condizione |
| --- | --- |
| [CellCircularReferenceException](../../../aspose.slides.spreadsheet/cellcircularreferenceexception) | La cartella di lavoro contiene formule con un riferimento circolare. |
| [CellUnsupportedDataException](../../../aspose.slides.spreadsheet/cellunsupporteddataexception) | I dati della cella non sono supportati. |

### Esempi

L'esempio mostra come assegnare una formula alla cella e calcolare un valore. Il valore della cella "B4" viene impostato a 5.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Pie, 100, 100, 300, 400);
    IChartDataWorkbook wb = chart.ChartData.ChartDataWorkbook;
    wb.GetCell(0, "B2", 2);	
	wb.GetCell(0, "B3", 3);	
    wb.GetCell(0, "B4").Formula = "B2+B3";
    wb.CalculateFormulas();
    ...
}
```

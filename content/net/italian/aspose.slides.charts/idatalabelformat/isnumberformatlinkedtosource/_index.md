## IDataLabelFormat.IsNumberFormatLinkedToSource proprietà

Lettura/scrittura Boolean.

```csharp
public bool IsNumberFormatLinkedToSource { get; set; }
```

### Osservazioni

Se il genitore di questo oggetto DataLabelFormat è una raccolta DataLabelCollection di etichette dati, questa proprietà ottiene o imposta il valore predefinito della proprietà IsNumberFormatLinkedToSource per le nuove etichette dati nella raccolta DataLabelCollection. Impostare questa proprietà con un valore imposta anche questo valore nella proprietà IsNumberFormatLinkedToSource per tutte le etichette dati nella raccolta DataLabelCollection (ad esempio "DataLabels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = val;" causa che tutti i DataLabels[i].IsNumberFormatLinkedToSource siano uguali a val).

## DataLabelFormat.ShowBubbleSize proprietà

Rappresenta il comportamento di visualizzazione del valore della dimensione del bubble dell'etichetta dati di un grafico specificato. True visualizza il valore della dimensione del bubble. False per nasconderlo. Lettura/Scrittura Boolean.

```csharp
public bool ShowBubbleSize { get; set; }
```

### Osservazioni

Se il genitore di questo oggetto DataLabelFormat è una raccolta DataLabelCollection di etichette dati, allora questa proprietà ottiene o imposta il valore predefinito della proprietà ShowBubbleSize per le nuove etichette dati nella raccolta DataLabelCollection. Impostare questa proprietà con un valore imposta anche tale valore alla proprietà ShowBubbleSize per tutte le etichette dati nella raccolta DataLabelCollection (ad esempio "DataLabels.DefaultDataLabelFormat.ShowBubbleSize = val;" fa sì che tutti i DataLabels[i].ShowBubbleSize siano uguali a val).

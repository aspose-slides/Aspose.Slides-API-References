## DataLabelFormat.ShowValue proprietà

Rappresenta il comportamento di visualizzazione del valore percentuale dell’etichetta dati di un grafico specificato. True visualizza il valore percentuale. False lo nasconde. Boolean a lettura/scrittura.

```csharp
public bool ShowValue { get; set; }
```

### Osservazioni

Se il genitore di questo oggetto DataLabelFormat è una raccolta DataLabelCollection di etichette dati, questa proprietà ottiene o imposta il valore predefinito della proprietà ShowValue per le nuove etichette dati nella raccolta DataLabelCollection. Impostare questa proprietà con un valore imposta lo stesso valore sulla proprietà ShowValue per tutte le etichette dati nella raccolta DataLabelCollection (ad esempio “DataLabels.DefaultDataLabelFormat.ShowValue = val;” causa che tutti i DataLabels[i].ShowValue siano uguali a val).

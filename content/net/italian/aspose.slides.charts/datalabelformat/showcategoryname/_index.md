## DataLabelFormat.ShowCategoryName proprietà

Rappresenta il comportamento di visualizzazione del nome della categoria dell'etichetta dati di un grafico specificato. True per visualizzare il nome della categoria per le etichette dati su un grafico. False per nascondere. Lettura/scrittura Boolean.

```csharp
public bool ShowCategoryName { get; set; }
```

### Osservazioni

Se il genitore di questo oggetto DataLabelFormat è una raccolta DataLabelCollection di etichette dati, questa proprietà ottiene o imposta il valore predefinito della proprietà ShowCategoryName per le nuove etichette dati nella raccolta DataLabelCollection. Impostare questa proprietà con un valore imposta anche questo valore alla proprietà ShowCategoryName per tutte le etichette dati nella raccolta DataLabelCollection (ad esempio "DataLabels.DefaultDataLabelFormat.ShowCategoryName = val;" fa sì che tutti DataLabels[i].ShowCategoryName sia uguale a val).

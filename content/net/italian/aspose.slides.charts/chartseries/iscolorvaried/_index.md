## ChartSeries.IsColorVaried proprietà

Specifica che ogni marcatore dei dati nella serie ha un colore diverso. Questa è la proprietà non solo di questa serie ma di tutte le serie del gruppo di serie padre – è la proiezione della proprietà del gruppo appropriato. Pertanto questa proprietà è di sola lettura. Usa la proprietà ParentSeriesGroup per accedere al gruppo di serie padre. Usa la proprietà ParentSeriesGroup.IsColorVaried di lettura/scrittura per modificare il valore. Boolean di sola lettura.

```csharp
public bool IsColorVaried { get; }
```

Questa è la proiezione della proprietà ParentSeriesGroup.IsColorVaried.

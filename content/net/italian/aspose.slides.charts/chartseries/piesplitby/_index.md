## ChartSeries.PieSplitBy proprietà

Specifica come determinare quali punti dati si trovano nella seconda fetta o barra in un grafico a torta-sopra-torta o a barra-sopra-torta. Questa è la proprietà non solo di questa serie ma di tutte le serie del gruppo di serie padre - è la proiezione della proprietà del gruppo appropriato. Pertanto questa proprietà è sola lettura. Utilizza la proprietà ParentSeriesGroup per accedere al gruppo di serie padre. Utilizza la proprietà ParentSeriesGroup.PieSplitBy lettura/scrittura per modificare il valore. Sola lettura [`PieSplitType`](../../piesplittype).

```csharp
public PieSplitType PieSplitBy { get; }
```

### Osservazioni

1) Questa è la proiezione della proprietà ParentSeriesGroup.PieSplitBy. 2) Se il valore della proprietà è PieSplitType.Custom, è possibile definire informazioni di divisione personalizzate con la proprietà ParentSeriesGroup.PieSplitCustomPoints.

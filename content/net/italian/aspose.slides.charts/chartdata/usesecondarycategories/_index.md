## ChartData.UseSecondaryCategories proprietà

Se false allora [`SecondaryCategories`](../secondarycategories) proprietà restituisce null e i dati nella proprietà [`Categories`](../categories) sono usati sia per le serie primarie che per le secondarie. Se true allora i dati nella proprietà [`SecondaryCategories`](../secondarycategories) sono usati per le serie secondarie e i dati nella proprietà [`Categories`](../categories) sono usati per le serie primarie. Lettura/scrittura Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

### Esempi

Esempio. Quali categorie sono associate alle serie – ChartData.Categories o ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // le categorie correlate sono series.Chart.ChartData.SecondaryCategories
}
else
{
    // le categorie correlate sono series.Chart.ChartData.Categories
}
```

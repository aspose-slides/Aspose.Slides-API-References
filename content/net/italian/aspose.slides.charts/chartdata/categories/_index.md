## ChartData.Categories proprietà

Ottiene le categorie primarie (oppure sia le categorie primarie che secondarie se la proprietà [`UseSecondaryCategories`](../usesecondarycategories) è false). Sola lettura [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection Categories { get; }
```

### Osservazioni

Se la proprietà [`UseSecondaryCategories`](../usesecondarycategories) è false, allora la proprietà [`SecondaryCategories`](../secondarycategories) restituisce null e i dati in questa `Categories` proprietà sono usati sia per le serie primarie che secondarie. Se la proprietà [`UseSecondaryCategories`](../usesecondarycategories) è true, allora i dati nella proprietà [`SecondaryCategories`](../secondarycategories) sono usati per le serie secondarie e i dati in questa `Categories` proprietà sono usati per le serie primarie.

### Esempi

Esempio. Quali categorie sono associate alle serie – ChartData.Categories o ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // le categorie correlate sono series.Chart.ChartData.SecondaryCategories
}
else
{
    // le categorie correlate sono series.Chart.ChartData.Categories
}
```

## IDataLabelCollection.Hide metodo

Rendi l'etichetta dati nascosta per impostazione predefinita impostando tutti i flag Show* (ShowValue, …) della proprietà DefaultDataLabelFormat su false. IsVisible sarà false dopo questa operazione.

```csharp
public void Hide()
```

### Osservazioni

Se l'etichetta dati non è visibile per impostazione predefinita (IsVisible è false) è possibile renderla "visibile per impostazione predefinita" impostando i flag Show* (ShowValue, …) della proprietà DefaultDataLabelFormat su true.

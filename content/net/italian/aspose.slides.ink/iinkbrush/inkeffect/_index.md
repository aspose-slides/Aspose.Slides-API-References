## Proprietà IInkBrush.InkEffect

Restituisce il tipo di effetto inchiostro (ad es., Galaxy, Gold, Silver) che definisce lo stile visivo del tratto di inchiostro. Il valore è analizzato dalla proprietà brush "inkEffects". Se non viene specificato alcun effetto riconosciuto, viene restituito NotDefined.

```csharp
public InkEffectType InkEffect { get; }
```

### Esempi

Esempio:

```csharp
[C#]
using (var pres = new Presentation("Presentation.pptx"))
{
    Ink ink = pres.Slides[0].Shapes[0] as Ink;
    IInkBrush brush = ink.Traces[0].Brush;
    Console.WriteLine("InkEffects = {0}", brush.InkEffect);
}
```

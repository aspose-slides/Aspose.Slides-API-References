## PdfAccessPermissions enumerazione

Contiene un insieme di flag che specificano quali autorizzazioni di accesso devono essere concesse quando il documento viene aperto con accesso utente.

```csharp
[Flags]
public enum PdfAccessPermissions
```

### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| None | `0` | Specifica che un utente non ha permessi di accesso. |
| PrintDocument | `4` | Specifica se un utente può stampare il documento (possibilmente non al livello di massima qualità, a seconda che il bit HighQualityPrint sia anche impostato). |
| ModifyContent | `8` | Specifica se un utente può modificare i contenuti del documento mediante operazioni diverse da quelle controllate dai bit AddOrModifyFields, FillExistingFields, AssembleDocument. |
| CopyTextAndGraphics | `10` | Specifica se un utente può copiare o altrimenti estrarre testo e grafica dal documento mediante operazioni diverse da quelle controllate dal bit ExtractTextAndGraphics. |
| AddOrModifyFields | `20` | Specifica se un utente può aggiungere o modificare annotazioni di testo, compilare campi di modulo interattivi e, se il bit ModifyContent è anche impostato, creare o modificare campi di modulo interattivi (inclusi i campi firma). |
| FillExistingFields | `100` | Specifica se un utente può compilare i campi di modulo interattivi esistenti (inclusi i campi firma), anche se il bit AddOrModifyFields è disattivato. |
| ExtractTextAndGraphics | `200` | Specifica se un utente può estrarre testo e grafica a supporto dell'accessibilità per utenti con disabilità o per altri scopi. |
| AssembleDocument | `400` | Specifica se un utente può assemblare il documento (inserire, ruotare o eliminare pagine e creare segnalibri o immagini in miniatura), anche se il bit ModifyContent è disattivato. |
| HighQualityPrint | `800` | Specifica se un utente può stampare il documento in una rappresentazione da cui può essere generata una fedele copia digitale del contenuto PDF. Quando questo bit è disattivato (e il bit PrintDocument è impostato), la stampa è limitata a una rappresentazione di basso livello dell'aspetto, possibilmente di qualità degradata. |

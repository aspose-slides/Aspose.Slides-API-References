## IInkOptions.InterpretMaskOpAsOpacity proprietà

Utilizza l'operazione ROP o Opacity per il rendering del pennello.

```csharp
public bool InterpretMaskOpAsOpacity { get; set; }
```

### Osservazioni

Il valore predefinito è true.

### Esempi

Il prossimo esempio dimostra come impostare usando ROP per l'esportazione degli elementi Ink:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.InterpretMaskOpAsOpacity = false;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

## IInkOptions.HideInk proprietà

Mostra o nasconde gli elementi Ink nel documento esportato.

```csharp
public bool HideInk { get; set; }
```

### Osservazioni

Il valore predefinito è false.

### Esempi

Il prossimo esempio dimostra come nascondere gli elementi Ink in un documento PDF esportato:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();
    pdfOptions.InkOptions.HideInk = true;
    pres.Save("output.pptx", SaveFormat.Pdf, pdfOptions);
}
```

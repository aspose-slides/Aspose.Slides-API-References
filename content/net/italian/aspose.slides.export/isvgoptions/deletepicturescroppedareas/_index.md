## ISVGOptions.DeletePicturesCroppedAreas proprietà

Una bandiera booleana indica se le parti ritagliate rimangono come parte del documento. Se true le parti ritagliate verranno rimosse, se false verranno serializzate nel documento (il che può potenzialmente portare a un file più grande) Lettura/Scrittura Boolean.

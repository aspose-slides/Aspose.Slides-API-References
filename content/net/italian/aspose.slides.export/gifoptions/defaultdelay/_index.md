## GifOptions.DefaultDelay proprietà

Ottiene o imposta il tempo di ritardo predefinito [ms]. Questo valore verrà usato se [`AdvanceAfterTime`](../../../aspose.slides/islideshowtransition/advanceaftertime) non è impostato. Il valore predefinito è 1000.

```csharp
public int DefaultDelay { get; set; }
```

### Esempi

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
	pres.Save("pres.gif", SaveFormat.Gif, new GifOptions { DefaultDelay = 2000 });
}
```

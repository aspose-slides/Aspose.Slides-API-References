## MarkdownSaveOptions.SlideNumberFormat proprietà

Ottiene o imposta la stringa di formato utilizzata per le intestazioni del numero di diapositiva nell'output Markdown. Il formato deve includere il segnaposto "{0}", che verrà sostituito con l'indice della diapositiva durante l'esportazione. Esempio: "# Slide {0}" produrrà "# Slide 1", "# Slide 2", ecc.

```csharp
public string SlideNumberFormat { get; set; }
```

### Eccezioni

| eccezione | condizione |
| --- | --- |
| ArgumentNullException | Generata se il valore fornito è nullo o vuoto. |
| ArgumentException | Generata se la stringa di formato non contiene il segnaposto "{0}". |

## TiffOptions.SlidesLayoutOptions proprietà

Ottiene o imposta la modalità con cui le diapositive vengono posizionate nella pagina durante l'esportazione di una presentazione [`ISlidesLayoutOptions`](../../islideslayoutoptions).

```csharp

```

### Esempi

Esempio:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    TiffOptions options = new TiffOptions
    {
        SlidesLayoutOptions = new HandoutLayoutingOptions
        {
            Handout = HandoutType.Handouts4Horizontal
        }
    };
    
    pres.Save("pres.tiff", SaveFormat.Tiff, options);
}
```

## PdfOptions.JpegQuality proprietà

Restituisce o imposta un valore che determina la qualità delle immagini JPEG all'interno del documento PDF. Lettura/Scrittura Byte.

```csharp
public byte JpegQuality { get; set; }
```

### Note

Ha effetto solo quando un documento contiene immagini JPEG.

Utilizzare questa proprietà per ottenere o impostare la qualità delle immagini all'interno di un documento quando si salva in formato PDF. Il valore può variare da 0 a 100 dove 0 indica la qualità peggiore ma la massima compressione e 100 indica la migliore qualità ma la compressione minima.

Il valore predefinito è **100**.

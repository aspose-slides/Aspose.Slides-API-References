## PdfOptions.EmbedTrueTypeFontsForASCII proprietà

Determina se Aspose.Slides incorporerà i font comuni per il testo ASCII (intervallo di codici 33..127). I font per i codici di carattere superiori a 127 sono sempre incorporati. L'elenco dei font comuni include i 14 font di base del PDF e altri font specificati dall'utente. Lettura/scrittura Boolean.

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### Osservazioni

Il valore predefinito è **true**.

## IRenderingOptions.DisableFontLigatures proprietà

Ottiene o imposta un valore che indica se il testo viene renderizzato senza utilizzare le legature. Quando impostato su `true`, le legature verranno disattivate nell'output renderizzato. Per impostazione predefinita, questa proprietà è impostata su `false`.

```csharp
public bool DisableFontLigatures { get; set; }
```

### Esempi

Esempio:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    RenderingOptions options = new RenderingOptions
    {
        DisableFontLigatures = true // Disabilita le legature nella resa del testo
    };
    
    Bitmap[] renderedSlides = pres.GetImage(options);
    for (var index = 0; index < renderedSlides.Length; index++)
    {
        var slideImage = renderedSlides[index];
        slideImage.Save($"slide-{index}.png");
    }
}
```

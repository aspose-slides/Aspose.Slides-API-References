## IPdfOptions.IncludeOleData proprietà

True per convertire tutti i dati OLE dalla presentazione in file incorporati nel PDF risultante. Boolean di lettura/scrittura.

```csharp
public bool IncludeOleData { get; set; }
```

### Note

Il valore predefinito è **false**.

### Esempi

Esempio:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    PdfOptions options = new PdfOptions { IncludeOleData = true };
    pres.Save("pres.pdf", SaveFormat.Pdf, options);
}
```

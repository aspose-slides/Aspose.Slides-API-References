## IPdfOptions.EmbedTrueTypeFontsForASCII proprietà

Impostare su True per incorporare i font TrueType per i caratteri ASCII da 32 a 127. I font per i codici di carattere superiori a 127 vengono sempre incorporati. Lettura/scrittura Boolean.

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### Osservazioni

Il valore predefinito è **true**.

## IPdfOptions.RasterizeUnsupportedFontStyles proprietà

Indica se il testo deve essere rasterizzato come bitmap e salvato in PDF quando il carattere non supporta lo stile grassetto. Questo approccio può migliorare la qualità del testo nel PDF risultante per alcuni caratteri. Lettura/scrittura Boolean.

```csharp
public bool RasterizeUnsupportedFontStyles { get; set; }
```

### Osservazioni

Il valore predefinito è **false**.

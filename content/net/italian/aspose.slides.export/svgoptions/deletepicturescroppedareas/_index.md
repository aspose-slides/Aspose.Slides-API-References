## SVGOptions.DeletePicturesCroppedAreas proprietà

Un flag booleano indica se le parti ritagliate rimangono parte del documento. Se true le parti ritagliate verranno rimosse, se false saranno serializzate nel documento (il che può portare a un file più grande)

## SVGOptions.DisableLineEndCropping proprietà

SVG 1.1 non consente di definire gli interni per i marcatori. Il motore di scrittura SVG di Aspose.Slides ha una soluzione per questo problema: ritaglia l'estremità della linea con la freccia, così la linea non si sovrappone ai marcatori. Questa opzione disattiva tale comportamento. Lettura/scrittura Boolean.

```csharp
public bool DisableLineEndCropping { get; set; }
```

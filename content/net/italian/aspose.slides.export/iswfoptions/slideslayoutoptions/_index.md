## ISwfOptions.SlidesLayoutOptions proprietà

Restituisce o imposta la modalità con cui le diapositive vengono posizionate sulla pagina durante l'esportazione di una presentazione [`ISlidesLayoutOptions`](../../islideslayoutoptions). Questa proprietà non supporta l'assegnazione di oggetti di tipo `Aspose.Slides.Export.HandoutLayoutingOptions`

```csharp
public ISlidesLayoutOptions SlidesLayoutOptions { get; set; }
```

### Esempi

Esempio:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    SwfOptions options = new SwfOptions
    {
        SlidesLayoutOptions = new NotesCommentsLayoutingOptions
        {
            CommentsPosition = CommentsPositions.Right
        }
    };
    
    pres.Save("pres.swf", SaveFormat.Swf, options);
}
```

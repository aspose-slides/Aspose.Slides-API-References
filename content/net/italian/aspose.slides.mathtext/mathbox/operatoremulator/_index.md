## MathBox.OperatorEmulator proprietà

Emulatore di operatore. Quando è true, la casella e il suo contenuto si comportano come un unico operatore e ne ereditano le proprietà. Ciò significa, per esempio, che il carattere può servire come punto per un'interruzione di riga e può essere allineato ad altri operatori. Gli emulatori di operatore sono spesso usati quando uno o più glifi si combinano per formare un operatore, come '=='. Valore predefinito: false

```csharp
public bool OperatorEmulator { get; set; }
```

### Esempi

Esempio:

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("=="));
box.OperatorEmulator = true;
```

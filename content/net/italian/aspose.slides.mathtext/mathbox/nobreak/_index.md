## MathBox.NoBreak proprietà

No break Questa proprietà specifica la proprietà "unbreakable" sull'oggetto box. Quando è true, non possono verificarsi interruzioni di riga all'interno del box. Questo può essere importante per gli emulatori di operatori che consistono in più di un operatore binario. Quando questo elemento non è specificato, le interruzioni possono verificarsi all'interno del box. Default: true

```csharp
public bool NoBreak { get; set; }
```

Example:

box.NoBreak = false;

## MathBorderBox.StrikethroughTopLeftToBottomRight proprietà

Strikethrough dall'angolo superiore sinistro all'angolo inferiore destro (default is false). Specifica lo stato nascosto o mostrato di una linea diagonale di barrato dall'angolo superiore sinistro all'angolo inferiore destro della casella del bordo.

```csharp
public bool StrikethroughTopLeftToBottomRight { get; set; }
```

### Esempi

Esempio:

```csharp
[C#]
MathBorderBox borderBox = new MathBorderBox(new MathematicalText("x"));
borderBox.StrikethroughTopLeftToBottomRight = true;
```

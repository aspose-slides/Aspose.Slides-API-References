## MathMatrix.RowGap proprietà

Il valore della spaziatura verticale tra le righe di una matrice; Se il RowGapRule è impostato a 3 ("Exactly"), l'unità è interpretata come twips (1/20 di punto) Se il RowGapRule è impostato a 4 ("Multiple"), l'unità è interpretata come mezze linee. Predefinito: 0

```csharp
public uint RowGap { get; set; }
```

### Esempi

Esempio:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

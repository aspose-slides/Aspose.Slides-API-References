## MathMatrix.ColumnGapRule proprietà

Il tipo di spaziatura orizzontale tra le colonne di una matrice; le unità di spaziatura orizzontale possono essere ems o punti (memorizzati come twips). Predefinito: SingleSpacingGap (0)

```csharp
public MathSpacingRules ColumnGapRule { get; set; }
```

### Esempi

Esempio:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.ColumnGapRule = MathSpacingRules.OneAndHalfSpacingGap;
```

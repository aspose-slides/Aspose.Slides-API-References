## MathArray.RowSpacing proprietà

Spaziatura tra le righe di un array. Viene usata solo quando RowSpacingRule è impostato su 3 Exactly, nel qual caso l'unità di misura è punti o Multiple, nel qual caso l'unità di misura è mezze linee. Predefinito: 0

```csharp
public uint RowSpacing { get; set; }
```

### Esempi

Esempio:

```csharp
[C#]
IMathArray mathArray = new MathArray(new MathematicalText("item1"));
mathArray.RowSpacingRule = MathRowSpacingRule.Exactly;
mathArray.RowSpacing = 10;
```

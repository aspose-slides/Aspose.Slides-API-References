## IMathArray.RowSpacing proprietà

Spaziatura tra le righe di un array. Viene utilizzata solo quando RowSpacingRule è impostato su 3 Exact, nel qual caso l'unità di misura è i punti, o su Multiple, nel qual caso l'unità di misura è le mezze linee. Predefinito: 0

```csharp
public uint RowSpacing { get; set; }
```

### Esempi

Example:

```csharp
[C#]
IMathArray mathArray = new MathArray(new MathematicalText("item1"));
mathArray.RowSpacingRule = MathRowSpacingRule.Exactly;
mathArray.RowSpacing = 10;
```

## IMathMatrix.ColumnGap proprietà

Il valore della spaziatura orizzontale tra le colonne di una matrice; se ColumnGapRule è impostato a 3 ("Exactly"), l'unità viene interpretata come twips (1/20 di punto) se ColumnGapRule è impostato a 4 ("Multiple"), l'unità viene interpretata come numero di incrementi di 0,5 em. Negli altri casi viene ignorato. Predefinito: 0

```csharp
public uint ColumnGap { get; set; }
```

### Esempi

Esempio:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.ColumnGapRule = MathSpacingRules.Exactly;
matrix.ColumnGap = 20;
```

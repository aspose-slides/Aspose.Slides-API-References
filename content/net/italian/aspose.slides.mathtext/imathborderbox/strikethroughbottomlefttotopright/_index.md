## IMathBorderBox.StrikethroughBottomLeftToTopRight proprietà

Barra diagonale dal basso a sinistra all'alto a destra (il valore predefinito è false). Specifica lo stato nascosto o mostrato di una linea diagonale barrata dall'angolo inferiore sinistro all'angolo superiore destro della casella del bordo.

```csharp
public bool StrikethroughBottomLeftToTopRight { get; set; }
```

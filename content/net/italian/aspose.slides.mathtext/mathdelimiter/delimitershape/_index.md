## MathDelimiter.DelimiterShape proprietà

Specifica la forma dei delimitatori nell'oggetto delimitatore. Quando è MathDelimiterShape.Centered, i delimitatori sono centrati attorno all'asse matematico del testo matematico e vengono adattati per occupare l'intera altezza del loro contenuto. Quando è MathDelimiterShape.Match, la loro altezza e forma vengono modificate per corrispondere esattamente al loro contenuto.

```csharp
public MathDelimiterShape DelimiterShape { get; set; }
```

### Esempi

Example:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.DelimiterShape = MathDelimiterShape.Match;
```

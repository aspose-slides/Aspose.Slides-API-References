## IMathPhantom.ZeroDesc proprietà

Ottiene o imposta un valore che indica se la discesa (profondità sotto la linea di base) dell'elemento base deve essere trattata come zero.

```csharp
public bool ZeroDesc { get; set; }
```

### Osservazioni

Quando `true`, il fantasma non abbassa la linea di base della riga matematica circostante. Corrisponde all'attributo OMML `m:zeroDesc`.

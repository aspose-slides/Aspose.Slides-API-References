## IMathPhantom.ZeroAsc proprietà

Ottiene o imposta un valore che indica se l'ascesa (altezza al di sopra della linea di base) dell'elemento base deve essere trattata come zero.

```csharp
public bool ZeroAsc { get; set; }
```

### Osservazioni

Quando `true`, il fantasma non alza la linea di base della riga matematica circostante. Corrisponde all'attributo OMML `m:zeroAsc`.

## IMathBox.NoBreak proprietà

Nessuna interruzione. Questa proprietà specifica la proprietà "ininterruttibile" sulla casella dell'oggetto. Quando è true, non possono verificarsi interruzioni di riga all'interno della casella. Ciò può essere importante per gli emulatori di operatori che consistono in più di un operatore binario. Quando questo elemento non è specificato, possono verificarsi interruzioni all'interno della casella. Default: true

```csharp
public bool NoBreak { get; set; }
```

## IMathBox.ExplicitBreak proprietà

L'interruzione esplicita indica se c'è un ritorno a capo all'inizio dell'oggetto Box, in modo che la linea si interrompa all'inizio dell'oggetto box. Specifica il numero dell'operatore nella riga precedente del testo matematico che deve essere usato come punto di allineamento per la riga corrente del testo matematico. Valori possibili: 1..255 Predefinito: 0 (nessuna interruzione esplicita)

```csharp
public byte ExplicitBreak { get; set; }
```

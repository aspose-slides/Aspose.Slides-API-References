## IMathBox.OperatorEmulator proprietà

Emulatore di operatore. Quando è true, la casella e i suoi contenuti si comportano come un unico operatore e ereditano le proprietà di un operatore. Ciò significa, ad esempio, che il carattere può fungere da punto per un'interruzione di riga e può essere allineato ad altri operatori. Gli emulatori di operatore sono spesso usati quando uno o più glifi si combinano per formare un operatore, ad esempio '=='. Valore predefinito: false

```csharp
public bool OperatorEmulator { get; set; }
```

### Esempi

Esempio:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.OperatorEmulator = true;
```

## MathParagraph.Remove metodo

Rimuove la prima occorrenza di un oggetto specifico dalla collezione/>.

```csharp
public bool Remove(IMathBlock mathBlock)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| mathBlock | IMathBlock | L'oggetto da rimuovere dalla collezione. |

### Valore restituito

true se *mathBlock* è stato rimosso con successo dalla collezione; altrimenti, false. Questo metodo restituisce anche false se *mathBlock* non è stato trovato nella collezione originale/>.

### Esempi

Esempio:

```csharp
[C#]
IAutoShape shape = slide.Shapes.AddMathShape(x, y, width, height);
IMathParagraph mathParagraph = (shape.TextFrame.Paragraphs[0].Portions[0] as MathPortion).MathParagraph;
mathParagraph.Add(new MathBlock(new MathematicalText("x")));
IMathBlock block = new MathBlock(new MathematicalText("y"));
mathParagraph.Add(block);
mathParagraph.Remove(block);
```

## MathParagraph indexer

Restituisce l'elemento all'indice specificato. Solo lettura [`IMathBlock`](../../imathblock).

```csharp
public IMathBlock this[int index] { get; set; }
```

| Parametro | Descrizione |
| --- | --- |
| index | L'indice basato su zero dell'elemento da ottenere |

### Valore restituito

Il blocco di un testo matematico.

### Esempi

Esempio:

```csharp
[C#]
IAutoShape shape = slide.Shapes.AddMathShape(x, y, width, height);
IMathParagraph mathParagraph = (shape.TextFrame.Paragraphs[0].Portions[0] as MathPortion).MathParagraph;
mathParagraph.Add(new MathBlock(new MathematicalText("block1")));
mathParagraph.Add(new MathBlock(new MathematicalText("block2")));
```

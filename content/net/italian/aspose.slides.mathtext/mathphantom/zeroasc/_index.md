## MathPhantom.ZeroAsc proprietà

Ottiene o imposta un valore che indica se l’altezza di ascensione (altezza sopra la linea di base) dell’elemento di base deve essere considerata zero.

```csharp
public bool ZeroAsc { get; set; }
```

### Osservazioni

Quando `true`, il phantom non solleva la linea di base della riga matematica circostante. Corrisponde all’attributo OMML `m:zeroAsc`.

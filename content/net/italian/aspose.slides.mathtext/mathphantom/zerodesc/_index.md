## MathPhantom.ZeroDesc proprietà

Ottiene o imposta un valore che indica se la discesa (profondità sotto la linea di base) dell'elemento base debba essere trattata come zero.

```csharp
public bool ZeroDesc { get; set; }
```

### Osservazioni

Quando `true`, il phantom non abbassa la linea di base della linea matematica circostante. Corrisponde all'attributo OMML `m:zeroDesc`.

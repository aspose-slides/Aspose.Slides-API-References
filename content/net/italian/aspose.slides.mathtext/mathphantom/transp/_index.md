## MathPhantom.Transp proprietà

Ottiene o imposta un valore che indica se il fantasma è trasparente per le regole di spaziatura basate su classi.

```csharp
public bool Transp { get; set; }
```

### Note

Quando `true`, gli operatori e i simboli all'interno del fantasma influiscono ancora sulla spaziatura matematica attorno al fantasma (come se fosse visibile). Quando `false`, la spaziatura basata su classi è ignorata. Corrisponde all'attributo OMML `m:transp`.

## Effect.AnimateTextType proprietà

Definisce un tipo di animazione del testo per l'effetto. Il testo della forma può essere animato per lettera, per parola o tutto in una volta. Lettura/scrittura `AnimateTextType`.

```csharp
public AnimateTextType AnimateTextType { get; set; }
```

### Esempi

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Ottieni il primo effetto della prima diapositiva.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Modifica il tipo di animazione del testo dell'effetto a "Per lettera"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByLetter;
}
```

## Effect.StopPreviousSound proprietà

Questo attributo specifica se l'effetto di animazione interrompe il suono precedente. Read/write Boolean.

```csharp
public bool StopPreviousSound { get; set; }
```

### Esempi

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Ottieni il primo effetto della prima diapositiva.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Ottieni il primo effetto della seconda diapositiva.
    IEffect secondSlideEffect = presentation.Slides[1].Timeline.MainSequence[0];
       
    if (firstSlideEffect.Sound != null)
    {
        // Cambia le migliorie/Audio del secondo effetto in "Stop Previous Sound"
        secondSlideEffect.StopPreviousSound = true;
    }
}
```

## Effect.DelayBetweenTextParts proprietà

Definisce un ritardo tra le parti di testo animate (parole o lettere). Un valore positivo specifica la percentuale della durata dell'effetto. Un valore negativo specifica il ritardo in secondi. Lettura/scrittura Single.

```csharp
public float DelayBetweenTextParts { get; set; }
```

### Esempi

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Ottieni il primo effetto della prima diapositiva.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Cambia il tipo di animazione del testo dell'effetto in "By word"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByWord;
    
    // Imposta il ritardo tra le parti di testo animate al 20% della durata dell'effetto.
    firstSlideEffect.DelayBetweenTextParts = 20f;
}
```

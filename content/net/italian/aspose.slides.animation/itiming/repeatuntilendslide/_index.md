## ITiming.RepeatUntilEndSlide proprietà

Questo attributo specifica se l'effetto verrà ripetuto fino alla fine della diapositiva. Lettura/scrittura Boolean.

```csharp
public bool RepeatUntilEndSlide { get; set; }
```

### Esempi

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Ottieni la sequenza degli effetti per la prima diapositiva
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Ottieni il primo effetto della sequenza principale.
    IEffect effect = effectsSequence[0];

    // Modifica il Timing/Ripetizione dell'effetto su "Fino alla fine della diapositiva"
    effect.Timing.RepeatUntilEndSlide = true;
}
```

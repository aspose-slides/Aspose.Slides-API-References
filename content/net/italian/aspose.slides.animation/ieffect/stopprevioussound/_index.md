## Proprietà IEffect.StopPreviousSound

Questo attributo specifica se l'effetto di animazione interrompe il suono precedente. Lettura/scrittura Boolean.

```csharp
public bool StopPreviousSound { get; set; }
```

### Esempi

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Recupera il primo effetto della prima diapositiva.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Recupera il primo effetto della seconda diapositiva.
    IEffect secondSlideEffect = presentation.Slides[1].Timeline.MainSequence[0];
       
    if (firstSlideEffect.Sound != null)
    {
        // Modifica l'effetto della seconda diapositiva Enhancements/Sound in "Stop Previous Sound"
        secondSlideEffect.StopPreviousSound = true;
    }
}
```

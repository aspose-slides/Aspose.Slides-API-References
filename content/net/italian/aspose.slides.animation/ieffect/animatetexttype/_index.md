## IEffect.AnimateTextType proprietà

Definisce un tipo di animazione del testo per l'effetto. Il testo della forma può essere animato per lettera, per parola o tutti insieme. Lettura/scrittura `AnimateTextType`.

```csharp
public AnimateTextType AnimateTextType { get; set; }
```

### Esempi

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Ottieni il primo effetto della prima diapositiva.
    IEffect firstSlideEffect = presentation.Slides[0].Timeline.MainSequence[0];
    
    // Cambia il tipo di animazione del testo dell'effetto in "By letter"
    firstSlideEffect.AnimateTextType = AnimateTextType.ByLetter;
}
```

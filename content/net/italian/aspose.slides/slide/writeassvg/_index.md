## WriteAsSvg(Stream) {#writeassvg}

Salva il contenuto della diapositiva come file SVG.

```csharp
public void WriteAsSvg(Stream stream)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| stream | Stream | Stream di destinazione |

### Esempi

Il seguente esempio di codice dimostra come convertire la prima diapositiva di una presentazione PowerPoint in un file SVG.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.svg"))
    {
        // Salva la prima diapositiva come file SVG
        pres.Slides[0].WriteAsSvg(fileStream);
    }
}
```

```csharp
public void WriteAsSvg(Stream stream, ISVGOptions svgOptions)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| stream | Stream | Stream di destinazione |
| svgOptions | ISVGOptions | opzioni di generazione SVG |

### Esempi

Il seguente esempio di codice dimostra come convertire la prima diapositiva di una presentazione PowerPoint in un file SVG con opzioni.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.svg"))
    {
        var options = new SVGOptions() { VectorizeText = true };
        // Salva la prima diapositiva come file SVG
        pres.Slides[0].WriteAsSvg(fileStream, options);
    }
}
```

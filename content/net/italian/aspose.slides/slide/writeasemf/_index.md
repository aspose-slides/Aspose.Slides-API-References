## Slide.WriteAsEmf metodo

Salva il contenuto della diapositiva come file EMF.

```csharp
public void WriteAsEmf(Stream stream)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| stream | Stream | Stream di destinazione |

### Eccezioni

| eccezione | condizione |
| --- | --- |
| ArgumentNullException | Lo stream di destinazione è `null` |

### Esempi

Il seguente esempio di codice dimostra come convertire la prima diapositiva di una presentazione PowerPoint in un metafile.

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.emf"))
    {
        // Salva la prima diapositiva come metafile
        pres.Slides[0].WriteAsEmf(fileStream);
    }
}
```

## PictureFillFormat.TileOffsetX proprietà

Restituisce o imposta lo spostamento orizzontale della trama rispetto all'origine della forma in punti. Un valore positivo sposta la trama verso destra, mentre un valore negativo la sposta verso sinistra. Lettura/Scrittura Single.

```csharp
public float TileOffsetX { get; set; }
```

### Esempi

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Ottiene il formato di riempimento immagine della forma
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Imposta la modalità di riempimento immagine su Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Imposta lo spostamento orizzontale della trama a 20 punti
    pictureFillFormat.TileOffsetX = 20f;
}
```

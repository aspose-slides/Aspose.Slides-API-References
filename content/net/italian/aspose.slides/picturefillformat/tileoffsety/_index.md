## PictureFillFormat.TileOffsetY proprietà

Restituisce o imposta lo spostamento verticale della texture dall'origine della forma in punti. Un valore positivo sposta la texture verso il basso, mentre un valore negativo la sposta verso l'alto. Lettura/Scrittura Single.

```csharp
public float TileOffsetY { get; set; }
```

### Esempi

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Ottiene il formato di riempimento immagine della forma
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Imposta la modalità di riempimento immagine su Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Imposta lo spostamento verticale della texture a -50 punti
    pictureFillFormat.TileOffsetY = -50;
}
```

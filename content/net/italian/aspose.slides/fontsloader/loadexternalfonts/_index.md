## FontsLoader.LoadExternalFonts metodo

Aggiunge cartelle aggiuntive per cercare i caratteri.

```csharp
public static void LoadExternalFonts(string[] directories)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| directories | String[] | Directory da cui leggere caratteri aggiuntivi. |

### Esempi

Gli esempi seguenti mostrano come caricare caratteri personalizzati da .TTF

```csharp
[C#]
// Il percorso della cartella dei documenti.
string dataDir = "C:\\";
// cartelle per cercare i font
String[] folders = new String[] { dataDir };
// Carica i font della cartella dei font personalizzati
FontsLoader.LoadExternalFonts(folders);
// Esegui qualche operazione e renderizza la presentazione/diapositive
using (Presentation presentation = new Presentation(dataDir + "DefaultFonts.pptx"))
{
    presentation.Save(dataDir + "NewFonts_out.pptx", SaveFormat.Pptx);
	// Svuota la cache dei font
	FontsLoader.ClearCache();
}
```

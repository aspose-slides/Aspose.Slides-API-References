## SlideShowSettings.SlideShowType proprietà

Ottiene o imposta il tipo di presentazione. Rappresentato dai seguenti antenati `SlideShowType`: [`BrowsedAtKiosk`](../../browsedatkiosk), [`PresentedBySpeaker`](../../presentedbyspeaker) e [`BrowsedByIndividual`](../../browsedbyindividual)

```csharp
public SlideShowType SlideShowType { get; set; }
```

### Esempi

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // per impostare il tipo "Browsed at a kiosk (full screen)"
    pres.SlideShowSettings.SlideShowType = new BrowsedAtKiosk();
    
    // per impostare il tipo "Browsed by individual (window)"
    pres.SlideShowSettings.SlideShowType = new BrowsedByIndividual();
    
    // per impostare il tipo "Presented by a speaker (full screen)"
    pres.SlideShowSettings.SlideShowType = new PresentedBySpeaker();
}
```

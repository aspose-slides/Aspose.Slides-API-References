## BasePortionFormat.SpellCheck proprietà

Ottiene o imposta un valore che indica se il controllo ortografico è abilitato per la porzione di testo. Quando questa proprietà è impostata su false, le verifiche ortografiche per gli elementi di testo sono soppresse. Quando è impostata su true, il controllo ortografico è consentito. Il valore predefinito è `false`.

```csharp
public bool SpellCheck { get; set; }
```

### Esempi

Il prossimo esempio dimostra come abilitare il flag SpellCheck prima di salvare la presentazione:

```csharp
[C#]
using (var pres = new Presentation("input.pptx"))
{
    // Accedi alla prima porzione di testo all'interno della prima forma nella prima diapositiva
    var portion = ((AutoShape)pres.Slides[0].Shapes[0]).TextFrame.Paragraphs[0].Portions[0];
    // Abilita il controllo ortografico per questa porzione di testo
    portion.PortionFormat.SpellCheck = true;
    // Salva la presentazione modificata
    pres.Save("output-with-spellcheck.pptx", SaveFormat.Pptx);
}
```

## ILoadOptions.DeleteEmbeddedBinaryObjects proprietà

Determina se Aspose.Slides eliminerà tutti gli oggetti binari incorporati durante il caricamento della presentazione.

I tipi di oggetti binari incorporati:

* VBA Project [`VbaProject`](../../ipresentation/vbaproject)
* OLE Object dati incorporati [`EmbeddedFileData`](../../ioleembeddeddatainfo/embeddedfiledata)
* ActiveX Control dati binari [`ActiveXControlBinary`](../../icontrol/activexcontrolbinary)

Lettura/scrittura Boolean.

```csharp
public bool DeleteEmbeddedBinaryObjects { get; set; }
```

### Osservazioni

Il valore predefinito è **false**.

### Esempi

Il seguente esempio mostra come caricare la presentazione senza alcun oggetto binario incorporato.

```csharp
[C#]
LoadOptions loadOptions = new LoadOptions();
loadOptions.DeleteEmbeddedBinaryObjects = true;
using (Presentation pres = new Presentation("pres.ppt", loadOptions))
{
    pres.Save("output_WithoutBinaryObjects.ppt", SaveFormat.Ppt);
}
```

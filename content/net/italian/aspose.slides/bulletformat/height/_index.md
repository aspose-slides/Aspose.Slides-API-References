## BulletFormat.Height proprietà

Restituisce o imposta l'altezza del bullet di un paragrafo senza ereditarietà. Il valore float.NaN determina che il bullet eredita l'altezza dalla prima porzione nel paragrafo. Lettura/scrittura Single.

```csharp
public float Height { get; set; }
```

### Osservazioni

Un valore di altezza negativo indica che l'altezza è espressa in punti e un valore positivo indica che l'altezza è una percentuale del testo circostante.

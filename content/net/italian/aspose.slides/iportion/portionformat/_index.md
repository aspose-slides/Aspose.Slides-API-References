## IPortion.PortionFormat proprietà

Restituisce l'oggetto di formattazione che contiene le proprietà di formattazione impostate esplicitamente della porzione di testo senza alcuna eredità applicata. Sola lettura [`IPortionFormat`](../../iportionformat).

```csharp
public IPortionFormat PortionFormat { get; }
```

### Osservazioni

L'oggetto di formattazione contiene i parametri di formattazione definiti solo per la porzione corrente, i dati ereditati non vengono applicati.

Per ottenere i valori effettivi, inclusi quelli ereditati, utilizzare il metodo [`GetEffective`](../../iportionformat/geteffective).

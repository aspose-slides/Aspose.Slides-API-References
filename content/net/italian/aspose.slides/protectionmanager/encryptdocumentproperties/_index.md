## ProtectionManager.EncryptDocumentProperties proprietà

Questa proprietà ha senso se la presentazione è protetta da password. Se true allora le proprietà del documento sono crittografate nel file della presentazione. Se false le proprietà del documento sono pubbliche mentre la presentazione è crittografata. Lettura/scrittura Boolean.

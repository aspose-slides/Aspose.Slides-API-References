## AudioFrame.RewindAudio proprietà

Determina se l'audio viene automaticamente riavvolto all'inizio dopo la riproduzione. Booleano Lettura/Scrittura.

```csharp
public bool RewindAudio { get; set; }
```

### Esempi

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ISlide slide = pres.Slides[0];

    // Aggiungi frame audio
    IAudioFrame audioFrame = slide.Shapes.AddAudioFrameLinked(50, 50, 100, 100, "sampleaudio.wav");

    // Imposta l'audio per riprodursi su tutte le diapositive
    audioFrame.PlayAcrossSlides = true;

    // Imposta l'audio per riavvolgere automaticamente all'inizio dopo la riproduzione
    audioFrame.RewindAudio = true;

    pres.Save("AudioFrame_out.pptx", SaveFormat.Pptx);
}
```

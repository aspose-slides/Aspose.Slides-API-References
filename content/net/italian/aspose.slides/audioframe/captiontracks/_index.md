## AudioFrame.CaptionTracks proprietà

Ottiene la raccolta di sottotitoli chiusi associati al frame audio. Questa proprietà è di sola lettura e restituisce un [`ICaptionsCollection`](../../icaptionscollection) contenente tutte le tracce dei sottotitoli.

```csharp
public ICaptionsCollection CaptionTracks { get; }
```

### Esempi

Esempio:

```csharp
[C#]
using (Presentation pres = new Presentation("audio with captions.pptx"))
{
    foreach (IShape shape in pres.Slides[0].Shapes)
    {
        if (shape is IAudioFrame audioFrame)
        {
            // Salva i dati binari della traccia dei sottotitoli come file .vtt
            foreach (var captionTrack in audioFrame.CaptionTracks)
                File.WriteAllBytes(captionTrack.CaptionId + ".vtt", captionTrack.BinaryData);
        }
    }
}
```

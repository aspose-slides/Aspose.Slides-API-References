## BlobManagementOptions.PresentationLockingBehavior proprietà

Questa proprietà definisce se un'istanza della classe Presentation può essere proprietaria della sorgente - file o stream durante la durata dell'istanza. Se l'istanza è proprietaria, blocca la sorgente. Questo aiuta a migliorare il consumo di memoria e le prestazioni durante il lavoro con i BLOBs, ma la sorgente (stream o file) non può essere modificata durante la durata dell'istanza di Presentation.

public PresentationLockingBehavior PresentationLockingBehavior { get; set; }

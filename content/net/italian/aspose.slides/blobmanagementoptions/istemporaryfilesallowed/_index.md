## BlobManagementOptions.IsTemporaryFilesAllowed proprietà

Questa proprietà definisce se è possibile creare file temporanei durante la lavorazione con i BLOB, il che riduce notevolmente il consumo di memoria ma richiede i permessi per creare file. Tutti i file verranno eliminati al termine del lavoro con la presentazione.

public bool IsTemporaryFilesAllowed { get; set; }

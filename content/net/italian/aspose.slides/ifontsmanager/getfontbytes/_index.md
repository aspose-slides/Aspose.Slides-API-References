## IFontsManager.GetFontBytes metodo

Recupera l'array di byte che rappresenta i dati del carattere per uno stile di carattere e dati del carattere specificati.

```csharp
public byte[] GetFontBytes(IFontData fontData, FontStyleType fontStyle)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| fontData | IFontData | L'oggetto font data contenente le informazioni sul carattere [`IFontData`](../../ifontdata). |
| fontStyle | FontStyleType | Lo stile del carattere per il quale devono essere recuperati i dati [`FontStyleType`](../../fontstyletype). |

### Valore di ritorno

Un array di byte contenente i dati del carattere per lo stile del carattere specificato. Se i dati del carattere o lo stile non sono trovati, restituisce null.

### Esempi

```csharp
[C#]
using (Presentation pres = new Presentation ("Presentation.pptx"))
{
    // Recupera tutti i caratteri usati nella presentazione
    IFontData[] fonts = pres.FontsManager.GetFonts();

    // Ottieni l'array di byte che rappresenta lo stile normale del primo carattere nella presentazione
    byte[] bytes = pres.FontsManager.GetFontBytes(fonts[0], FontStyleType.Regular);
}
```

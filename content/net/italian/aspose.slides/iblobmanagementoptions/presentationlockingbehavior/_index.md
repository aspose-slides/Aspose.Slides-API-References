## IBlobManagementOptions.PresentationLockingBehavior proprietà

Questa proprietà definisce se un'istanza della classe Presentation può essere proprietaria della sorgente – file o stream – durante la durata dell'istanza. Se l'istanza è proprietaria, blocca la sorgente. Questo aiuta a migliorare il consumo di memoria e le prestazioni durante il lavoro con i BLOB, ma la sorgente (stream o file) non può essere modificata durante la durata dell'istanza di Presentation. Questo è un esempio:

```csharp
public PresentationLockingBehavior PresentationLockingBehavior { get; set; }
```

### Esempi

```csharp
LoadOptions loadOptions = new LoadOptions { 
  BlobManagementOptions = {PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked} };
using (Presentation pres = new Presentation("pres.pptx", loadOptions)) {
  // IOException verrà lanciata perché pres.pptx è bloccato per la durata di una Presentation
  // File.Delete("pres.pptx");
}
// dopo che l'oggetto Presentation è stato smaltito, il file è sbloccato e può essere eliminato
File.Delete("pres.pptx");
```

## IBlobManagementOptions.IsTemporaryFilesAllowed proprietà

Questa proprietà definisce se è consentito creare file temporanei durante il lavoro con BLOB, il che riduce notevolmente il consumo di memoria ma richiede i permessi per creare file. Tutti i file saranno eliminati al termine del lavoro con la presentazione.

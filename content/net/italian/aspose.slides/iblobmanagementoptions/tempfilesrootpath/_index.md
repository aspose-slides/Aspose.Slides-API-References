## IBlobManagementOptions.TempFilesRootPath proprietà

Il percorso radice in cui verranno creati i file temporanei. Per impostazione predefinita verrà utilizzata la directory temporanea di sistema. Il processo di hosting deve disporre delle autorizzazioni per creare file e cartelle in quel percorso.

## FontsManager.GetFontBytes metodo

Recupera l'array di byte che rappresenta i dati del font per uno stile di font specificato e per i dati del font.

```csharp
public byte[] GetFontBytes(IFontData fontData, FontStyleType fontStyle)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| fontData | IFontData | L'oggetto font data contenente le informazioni sul font [`IFontData`](../../ifontdata). |
| fontStyle | FontStyleType | Lo stile del font per cui i dati devono essere recuperati [`FontStyleType`](../../fontstyletype). |

### Valore di ritorno

Un array di byte contenente i dati del font per lo stile di font specificato. Se i dati del font o lo stile non vengono trovati, restituisce null.

### Esempi

```csharp
[C#]
using (Presentation pres = new Presentation ("Presentation.pptx"))
{
    // Recupera tutti i font utilizzati nella presentazione
    IFontData[] fonts = pres.FontsManager.GetFonts();

    // Ottieni l'array di byte che rappresenta lo stile regolare del primo font nella presentazione
    byte[] bytes = pres.FontsManager.GetFontBytes(fonts[0], FontStyleType.Regular);
}
```

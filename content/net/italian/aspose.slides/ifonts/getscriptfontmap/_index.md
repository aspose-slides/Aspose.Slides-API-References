## IFonts.GetScriptFontMap metodo

Restituisce un dizionario di tutte le definizioni di caratteri script nella presentazione.

```csharp
public IDictionary<string, string> GetScriptFontMap()
```

### Valore di ritorno

Un dizionario che associa i codici script ai nomi dei caratteri.

### Esempi

```csharp
[C#]
IDictionary<string, string> map = presentation.MasterTheme.FontScheme.Major.GetScriptFontMap();
foreach (var kvp in map)
{
    Console.WriteLine(kvp.Key + " ? " + kvp.Value);
}
```

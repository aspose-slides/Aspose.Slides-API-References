## IFonts.RemoveScriptFont metodo

Rimuove l'impostazione del carattere associata a un tag di script specifico dalla raccolta di caratteri del tema.

```csharp
public void RemoveScriptFont(string script)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| script | String | Il codice di script BCP-47 la cui impostazione del carattere deve essere rimossa. |

### Esempi

Questo esempio dimostra come rimuovere la mappatura del carattere per lo script ebraico:

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.RemoveScriptFont("Hebr");
```

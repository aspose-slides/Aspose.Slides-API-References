## IViewProperties.GridSpacing proprietà

Restituisce o imposta la spaziatura della griglia da utilizzare per la griglia sottostante al documento di presentazione, in punti. Lettura/scrittura Single.

```csharp
public float GridSpacing { get; set; }
```

### Osservazioni

Il valore della spaziatura della griglia deve essere un numero positivo. L'intervallo tipico va da 1 mm (2.8349607 punti) a 2 pollici (144 punti).

### Esempi

Il seguente esempio di codice mostra come modificare la spaziatura della griglia in una presentazione PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

## IPresentationInfo.IsWriteProtected proprietà

Restituisce un valore che indica se una presentazione collegata è protetta dalla scrittura.

```csharp
public NullableBool IsWriteProtected { get; }
```

### Osservazioni

Se la presentazione è protetta da una password per l'apertura, il valore della proprietà è uguale a NotDefined. Vedi [`NullableBool`](../../nullablebool) enumeration.

### Esempi

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
if (info.IsWriteProtected == NullableBool.True)
{
    Console.WriteLine("The presentation '" + presentationFilePath + "' is write protected by a password.");
}
```

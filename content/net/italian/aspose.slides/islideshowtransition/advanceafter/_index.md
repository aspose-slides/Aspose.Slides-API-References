## ISlideShowTransition.AdvanceAfter proprietà

Questo attributo specifica se la presentazione si sposterà alla diapositiva successiva dopo un determinato intervallo di tempo. Lettura/scrittura Boolean.

```csharp
public bool AdvanceAfter { get; set; }
```

### Esempi

```csharp
[C#]
using (Presentation pres = new Presentation("demo.pptx"))
{
    // Ottieni la prima transizione della diapositiva
    ISlideShowTransition slideTransition = pres.Slides[0].SlideShowTransition;
    
    // Verifica se il flag Advance Slide After è attivo
    if (slideTransition.AdvanceAfter)
    {
        // Ottieni il valore del tempo di Advance Slide After
        uint advanceAfterTime = slideTransition.AdvanceAfterTime;
    }
}
```

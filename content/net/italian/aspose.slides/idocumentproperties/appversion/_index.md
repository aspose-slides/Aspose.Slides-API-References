## IDocumentProperties.AppVersion proprietà

Restituisce la versione dell'app. Solo lettura String.

```csharp
public string AppVersion { get; }
```

### Osservazioni

Il contenuto di questo elemento deve essere nel formato XX.YYYY, dove X e Y rappresentano valori numerici; altrimenti, il documento sarà considerato non conforme. Aspose.Slides rappresenta la sua versione nel formato XX.YYZZ, dove: XX - versione principale YY - versione minore ZZ - versione di patch Per esempio, il valore 23.0105 indica la versione 23.1.5 di Aspose.Slides.

## IDocumentProperties.GetSensitivityLabels metodo

Ottiene un array di etichette di sensibilità dalle proprietà personalizzate del documento (Microsoft Information Protection SDK Metadata).

```csharp
public ISensitivityLabel[] GetSensitivityLabels()
```

### Esempi

Il codice seguente mostra come spostare le informazioni sulle etichette di sensibilità dalle proprietà personalizzate del documento nella moderna collezione SensitivityLabels:

```csharp
[C#]
using (Presentation pres = new Presentation("SomePresentation.pptx"))
{
    // Ottieni le etichette di sensibilità dalle proprietà personalizzate del documento
    ISensitivityLabel[] mipSensitivityLabels = pres.DocumentProperties.GetSensitivityLabels();

    ISensitivityLabelCollection sensitivityLabels = pres.SensitivityLabels;
    foreach (var sensitivityLabel in mipSensitivityLabels)
    {
        // Aggiungi l'etichetta alla collezione
        // Qui puoi aggiungere un controllo sulla validità delle informazioni dell'etichetta (l'etichetta è disponibile, ecc)
        sensitivityLabels.Add(sensitivityLabel);
    }

    pres.Save("SensitivityLabel.pptx", SaveFormat.Pptx);
}
```

## FontFallBackRule.IndexOf metodo

Restituisce un indice della regola specificata nella raccolta.

```csharp
public int IndexOf(string fontName)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| fontName | String | Nome del font da trovare. |

### Valore restituito

Indice di un font o -1 se il font non è trovato nell'elenco.

### Esempi

```csharp
[C#]
// Crea una regola che contiene un elenco di font.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// Ottieni l'indice di Tahoma.
int tahomaIndex = newRule.IndexOf("Tahoma");
```

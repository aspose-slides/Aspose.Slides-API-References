## IProtectionManager.CheckWriteProtection metodo

Determina se una presentazione è protetta da password per la modifica.

```csharp
public bool CheckWriteProtection(string password)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| password | String | La password da verificare. |

### Valore restituito

True se la password è valida; altrimenti, false.

### Osservazioni

1. Dovresti controllare la proprietà [`IsWriteProtected`](../iswriteprotected) prima di chiamare questo metodo. 2. Quando la password è null o vuota, questo metodo restituisce false.

### Esempi

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

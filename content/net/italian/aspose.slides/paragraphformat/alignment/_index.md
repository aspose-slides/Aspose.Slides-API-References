## ParagraphFormat.Alignment proprietà

Restituisce o imposta l'allineamento del testo in un paragrafo senza ereditarietà. Lettura/scrittura [`TextAlignment`](../../textalignment).

```csharp
public TextAlignment Alignment { get; set; }
```

### Esempi

Il seguente codice di esempio mostra come allineare i paragrafi di testo in una presentazione PowerPoint.

```csharp
[C#]
// Instanzia un oggetto Presentation che rappresenta un file PPTX
using (Presentation pres = new Presentation("ParagraphsAlignment.pptx"))
{
    // Accesso alla prima slide
    ISlide slide = pres.Slides[0];
    // Accesso al primo e al secondo segnaposto nella slide e casting a AutoShape
    ITextFrame tf1 = ((IAutoShape)slide.Shapes[0]).TextFrame;
    ITextFrame tf2 = ((IAutoShape)slide.Shapes[1]).TextFrame;
    // Modifica il testo in entrambi i segnaposto
    tf1.Text = "Center Align by Aspose";
    tf2.Text = "Center Align by Aspose";
    // Ottenimento del primo paragrafo dei segnaposto
    IParagraph para1 = tf1.Paragraphs[0];
    IParagraph para2 = tf2.Paragraphs[0];
    // Allineamento del paragrafo di testo al centro
    para1.ParagraphFormat.Alignment = TextAlignment.Center;
    para2.ParagraphFormat.Alignment = TextAlignment.Center;
    //Scrivi la presentazione come file PPTX
    pres.Save("Centeralign_out.pptx", SaveFormat.Pptx);
}
```

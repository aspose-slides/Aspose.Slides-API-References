## AddAudioFrameEmbedded(float, float, float, float, Stream) {#addaudioframeembedded_1}

Crea un nuovo fotogramma audio con un file WAV incorporato e lo aggiunge alla fine della raccolta di forme. L'audio incorporato viene aggiunto alla raccolta Presentation.Audios.

```csharp
public IAudioFrame AddAudioFrameEmbedded(float x, float y, float width, float height, 
    Stream audio_stream)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| x | Single | La coordinata x del nuovo fotogramma audio, in punti. |
| y | Single | La coordinata y del nuovo fotogramma audio, in punti. |
| width | Single | La larghezza del nuovo fotogramma audio, in punti. |
| height | Single | L'altezza del nuovo fotogramma audio, in punti. |
| audio_stream | Stream | Un flusso di input contenente dati audio WAV da incorporare. |

### Valore di ritorno

Il nuovo [`IAudioFrame`](../../iaudioframe) creato.

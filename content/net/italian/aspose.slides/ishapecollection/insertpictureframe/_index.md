## IShapeCollection.InsertPictureFrame metodo

Crea un nuovo frame immagine contenente l'immagine specificata e lo inserisce nella raccolta di forme all'indice specificato.

```csharp
public IPictureFrame InsertPictureFrame(int index, ShapeType shapeType, float x, float y, 
    float width, float height, IPPImage image)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| index | Int32 | L'indice basato su zero al quale inserire il frame immagine. |
| shapeType | ShapeType | Specifica il tipo di forma contenuto in [`ShapeType`](../../shapetype), eccetto tutti i tipi di linee: |
| x | Single | La coordinata x del frame immagine, in punti. |
| y | Single | La coordinata y del frame immagine, in punti. |
| width | Single | La larghezza del frame immagine, in punti. |
| height | Single | L'altezza del frame immagine, in punti. |
| image | IPPImage | Il [`IPPImage`](../../ippimage) da visualizzare nel frame immagine. |

### Valore di ritorno

Il nuovo [`IPictureFrame`](../../ipictureframe).

## InsertAutoShape(int, ShapeType, float, float, float, float) {#insertautoshape}

```csharp
public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width, 
    float height)
```

| Parametro | Tipo | Descrizione |

| shapeType | ShapeType | Il [`ShapeType`](../../shapetype) della forma automatica da inserire. |
| x | Single | La coordinata x del riquadro della forma, in punti. |

## InsertAutoShape(int, ShapeType, float, float, float, float, bool) {#insertautoshape_1}

Crea una nuova forma automatica e la inserisce nella raccolta di forme all’indice specificato, opzionalmente inizializzandola con lo stile predefinito del modello.

```csharp
public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| index | Int32 | L’indice basato su zero al quale inserire la forma automatica. |
| shapeType | ShapeType | Il [`ShapeType`](../../shapetype) della forma automatica da inserire. |
| x | Single | La coordinata x del riquadro della forma, in punti. |
| y | Single | La coordinata y del riquadro della forma, in punti. |
| width | Single | La larghezza del riquadro della forma, in punti. |
| height | Single | L’altezza del riquadro della forma, in punti. |
| createFromTemplate | Boolean | True per applicare lo stile predefinito del modello (incluso un nome non vuoto, stile semplice e testo centrato); false per creare la forma con tutte le proprietà impostate ai valori predefiniti. |

### Valore restituito

L’oggetto appena creato [`IAutoShape`](../../iautoshape).

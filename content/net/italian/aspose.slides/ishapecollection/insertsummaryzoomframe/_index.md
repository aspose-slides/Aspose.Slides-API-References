## IShapeCollection.InsertSummaryZoomFrame method

Crea un nuovo Summary Zoom frame e lo inserisce nella shape collection all'indice specificato.

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| index | Int32 | L'indice basato su zero al quale inserire il Summary Zoom frame. |
| x | Single | La coordinata x del nuovo Summary Zoom frame, in punti. |
| y | Single | La coordinata y del nuovo Summary Zoom frame, in punti. |
| width | Single | La larghezza del nuovo Summary Zoom frame, in punti. |
| height | Single | L'altezza del nuovo Summary Zoom frame, in punti. |

### Valore di ritorno

L'[`ISummaryZoomFrame`](../../isummaryzoomframe) appena creato.

### Eccezioni

| eccezione | condizione |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | Generata se la presentazione non contiene sezioni, o se la diapositiva di destinazione non appartiene a nessuna sezione. |

### Osservazioni

Questo metodo crea un Summary Zoom frame che aggrega i collegamenti di riepilogo per tutte le sezioni nella presentazione.

### Esempi

Questo esempio dimostra la creazione e l'inserimento di un oggetto Summary Zoom all'indice specificato di una collezione (supponendo che nella presentazione "Presentation.pptx" ci siano almeno due sezioni):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSummaryZoomFrame(2, 150, 20, 50, 50);
}
```

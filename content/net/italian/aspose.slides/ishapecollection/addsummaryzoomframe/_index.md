## Metodo IShapeCollection.AddSummaryZoomFrame

Crea un nuovo frame Summary Zoom e lo aggiunge alla fine della raccolta di forme.

```csharp
public ISummaryZoomFrame AddSummaryZoomFrame(float x, float y, float width, float height)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| x | Single | La coordinata x del nuovo frame Summary Zoom, in punti. |
| y | Single | La coordinata y del nuovo frame Summary Zoom, in punti. |
| width | Single | La larghezza del nuovo frame Summary Zoom, in punti. |
| height | Single | L'altezza del nuovo frame Summary Zoom, in punti. |

### Valore di ritorno

Il [`ISummaryZoomFrame`](../../isummaryzoomframe) appena creato.

### Eccezioni

| eccezione | condizione |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | Generata se non ci sono sezioni nella presentazione, o se la diapositiva di destinazione non appartiene a nessuna sezione. |

### Osservazioni

Questo metodo crea un frame Summary Zoom che aggrega i collegamenti di riepilogo per tutte le sezioni nella presentazione.

### Esempi

Questo esempio dimostra come aggiungere un oggetto Summary Zoom alla fine di una raccolta (si assume che nella presentazione "Presentation.pptx" ci siano almeno due sezioni):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSummaryZoomFrame(150, 20, 500, 250);
}
```

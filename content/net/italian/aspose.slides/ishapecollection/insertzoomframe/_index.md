## InsertZoomFrame(int, float, float, float, float, ISlide) {#insertzoomframe}

Crea un nuovo frame Zoom e lo inserisce nella collezione di forme all'indice specificato.

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| index | Int32 | L'indice a base zero al quale inserire il frame Zoom. |
| x | Single | La coordinata x del nuovo frame Zoom, in punti. |
| y | Single | La coordinata y del nuovo frame Zoom, in punti. |
| width | Single | La larghezza del nuovo frame Zoom, in punti. |
| height | Single | L'altezza del nuovo frame Zoom, in punti. |
| slide | ISlide | Il [`ISlide`](../../islide) referenziato dal frame Zoom. |

### Valore restituito

Il [`IZoomFrame`](../../izoomframe) appena creato.

### Eccezioni

| eccezione | condizione |
| --- | --- |
| ArgumentException | Generata se la diapositiva di riferimento non appartiene alla presentazione corrente. |

### Esempi

Questo esempio dimostra la creazione e l'inserimento di un oggetto Zoom all'indice specificato di una collezione (si assume che nella presentazione "Presentation.pptx" vi siano almeno due diapositive):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertZoomFrame(2, 150, 20, 50, 50, pres.Slides[1]);
}
```

public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide, IPPImage image)

| Parametro | Tipo | Descrizione |

| index | Int32 | L'indice a base zero al quale inserire il frame Zoom. |
| x | Single | La coordinata x del nuovo frame Zoom, in punti. |
| y | Single | La coordinata y del nuovo frame Zoom, in punti. |
| width | Single | La larghezza del nuovo frame Zoom, in punti. |
| height | Single | L'altezza del nuovo frame Zoom, in punti. |

| eccezione | condizione |

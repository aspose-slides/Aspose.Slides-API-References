## IShapeCollection.AddSmartArt metodo

Crea un diagramma SmartArt e lo aggiunge alla fine della raccolta di forme.

```csharp
public ISmartArt AddSmartArt(float x, float y, float width, float height, 
    SmartArtLayoutType layoutType)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| x | Single | La coordinata x del frame del diagramma, in punti. |
| y | Single | La coordinata y del frame del diagramma, in punti. |
| width | Single | La larghezza del frame del diagramma, in punti. |
| height | Single | L'altezza del frame del diagramma, in punti. |
| layoutType | SmartArtLayoutType | Il tipo di layout SmartArt. |

### Valore di ritorno

Il [`ISmartArt`](../../../aspose.slides.smartart/ismartart) appena creato.

### Esempi

Esempio:

```csharp
[C#]
Presentation pres = new Presentation();
Slide slide = pres.Slides[0];
SmartArt smart = slide.Shapes.AddSmartArt(0, 0, 400, 400, SmartArtLayoutType.BasicBlockList);
```

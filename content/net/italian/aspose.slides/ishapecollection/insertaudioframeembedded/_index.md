## InsertAudioFrameEmbedded(int, float, float, float, float, Stream) {#insertaudioframeembedded_1}

Crea un nuovo fotogramma audio con un file WAV incorporato e lo inserisce nella raccolta di forme all'indice specificato. L'audio incorporato viene aggiunto alla collezione Presentation.Audios.

```csharp
public IAudioFrame InsertAudioFrameEmbedded(int index, float x, float y, float width, float height, 
    Stream audio_stream)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| index | Int32 | L'indice basato su zero al quale inserire il fotogramma audio. |
| x | Single | La coordinata x del nuovo fotogramma audio, in punti. |
| y | Single | La coordinata y del nuovo fotogramma audio, in punti. |
| width | Single | La larghezza del nuovo fotogramma audio, in punti. |
| height | Single | L'altezza del nuovo fotogramma audio, in punti. |
| audio_stream | Stream | Un flusso di input contenente dati audio WAV da incorporare. |

### Valore restituito

L'oggetto [`IAudioFrame`](../../iaudioframe) appena creato.

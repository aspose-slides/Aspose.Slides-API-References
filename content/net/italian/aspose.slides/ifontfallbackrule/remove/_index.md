## IFontFallBackRule.Remove metodo

Rimuove la prima occorrenza di un font FallBack specifico dall'elenco.

```csharp
public void Remove(string fontName)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| fontName | String | Il nome del font da rimuovere dall'elenco. |

### Esempi

```csharp
[C#]

IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

//Rimozione di Tahoma dall'elenco
newRule.Remove ("Tahoma");
```

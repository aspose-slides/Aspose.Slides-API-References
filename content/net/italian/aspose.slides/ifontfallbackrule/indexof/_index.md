## IFontFallBackRule.IndexOf metodo

Restituisce l'indice della regola specificata nella raccolta.

```csharp
public int IndexOf(string fontName)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| fontName | String | Nome del carattere da trovare. |

### Valore restituito

Indice di un carattere o -1 se il carattere non è trovato nell'elenco.

### Esempi

```csharp
[C#]
// Crea una regola contenente un elenco di caratteri.
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

//Ottieni l'indice di Tahoma
int tahomaIndex = newRule.IndexOf("Tahoma");
```

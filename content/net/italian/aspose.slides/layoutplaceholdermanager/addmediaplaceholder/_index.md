## LayoutPlaceholderManager.AddMediaPlaceholder metodo

Aggiunge una nuova forma segnaposto alla diapositiva di layout per contenere un oggetto multimediale.

```csharp
public IAutoShape AddMediaPlaceholder(float x, float y, float width, float height)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| x | Single | La coordinata X della nuova forma segnaposto. |
| y | Single | La coordinata Y della nuova forma segnaposto. |
| width | Single | La larghezza della nuova forma segnaposto. |
| height | Single | L'altezza della nuova forma segnaposto. |

### Valore restituito

Creato [`IAutoShape`](../../iautoshape) con un segnaposto Media.

### Esempi

Il seguente esempio mostra come aggiungere la forma segnaposto Media alla diapositiva di layout.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddMediaPlaceholder(20, 20, 200, 200);
}
```

## LayoutPlaceholderManager.AddTextPlaceholder metodo

Aggiunge una nuova forma placeholder alla diapositiva di layout per contenere il contenuto del testo.

```csharp
public IAutoShape AddTextPlaceholder(float x, float y, float width, float height)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| x | Single | La coordinata X della nuova forma placeholder. |
| y | Single | La coordinata Y della nuova forma placeholder. |
| width | Single | La larghezza della nuova forma placeholder. |
| height | Single | L'altezza della nuova forma placeholder. |

### Valore di ritorno

Creato [`IAutoShape`](../../iautoshape) con un placeholder Text.

### Esempi

Il seguente esempio mostra come aggiungere la forma placeholder Text alla diapositiva di layout.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTextPlaceholder(20, 20, 500, 300);
}
```

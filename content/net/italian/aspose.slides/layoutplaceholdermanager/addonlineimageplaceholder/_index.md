## LayoutPlaceholderManager.AddOnlineImagePlaceholder metodo

Aggiunge una nuova forma segnaposto alla slide di layout per contenere un'immagine online.

```csharp
public IAutoShape AddOnlineImagePlaceholder(float x, float y, float width, float height)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| x | Single | La coordinata X della nuova forma segnaposto. |
| y | Single | La coordinata Y della nuova forma segnaposto. |
| width | Single | La larghezza della nuova forma segnaposto. |
| height | Single | L'altezza della nuova forma segnaposto. |

### Valore di ritorno

Creato [`IAutoShape`](../../iautoshape) con un segnaposto Immagine online.

### Esempi

Il seguente esempio mostra come aggiungere la forma segnaposto Immagine online alla slide di layout.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddOnlineImagePlaceholder(20, 20, 200, 200);
}
```

## LayoutPlaceholderManager.AddTablePlaceholder metodo

Aggiunge una nuova forma segnaposto alla diapositiva layout per contenere una Table.

```csharp
public IAutoShape AddTablePlaceholder(float x, float y, float width, float height)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| x | Single | La coordinata X della nuova forma segnaposto. |
| y | Single | La coordinata Y della nuova forma segnaposto. |
| width | Single | La larghezza della nuova forma segnaposto. |
| height | Single | L'altezza della nuova forma segnaposto. |

### Valore di ritorno

Creato [`IAutoShape`](../../iautoshape) con un segnaposto Table.

### Esempi

Il seguente esempio mostra come aggiungere la forma segnaposto Table alla diapositiva layout.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTablePlaceholder(20, 20, 500, 200);
}
```

## metodo SvgImage.WriteAsEmf

Salva l'immagine SVG come file EMF.

```csharp
public void WriteAsEmf(Stream stream)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| stream | Stream | Flusso di destinazione |

### Eccezioni

| Eccezione | Condizione |
| --- | --- |
| ArgumentNullException | Il flusso di destinazione è **null** |

### Esempi

Il seguente esempio mostra come salvare l'immagine SVG nel metafile.

```csharp
[C#]
// Crea la nuova immagine SVG
ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));

// Salva l'immagine SVG come metafile
using (var fileStream = System.IO.File.OpenWrite("SvgAsEmf.emf"))
    svgImage.WriteAsEmf(fileStream);
```

Questo esempio dimostra come aggiungere l'immagine SVG come metafile alla collezione di immagini della presentazione.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // Crea la nuova immagine SVG
    ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));
    using (var memStream = new MemoryStream())
    {
        // Salva l'immagine SVG come metafile
        svgImage.WriteAsEmf(memStream);
        // Aggiunge il metafile alla collezione di immagini
        pres.Images.AddImage(memStream.ToArray());
    }
}
```

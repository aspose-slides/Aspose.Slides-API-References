## ILayoutPlaceholderManager.AddSmartArtPlaceholder method

Aggiunge una nuova forma segnaposto alla diapositiva layout per contenere un diagramma SmartArt.

```csharp
public IAutoShape AddSmartArtPlaceholder(float x, float y, float width, float height)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| x | Single | La coordinata X della nuova forma segnaposto. |
| y | Single | La coordinata Y della nuova forma segnaposto. |
| width | Single | La larghezza della nuova forma segnaposto. |
| height | Single | L'altezza della nuova forma segnaposto. |

### Valore restituito

Creato [`IAutoShape`](../../iautoshape) con un segnaposto SmartArt.

### Esempi

Il seguente esempio mostra come aggiungere la forma segnaposto SmartArt alla diapositiva layout.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddSmartArtPlaceholder(20, 20, 200, 200);
}
```

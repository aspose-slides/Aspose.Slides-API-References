## ILayoutPlaceholderManager.AddPicturePlaceholder metodo

Aggiunge una nuova forma segnaposto alla diapositiva layout per contenere un'immagine.

```csharp
public IAutoShape AddPicturePlaceholder(float x, float y, float width, float height)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| x | Single | La coordinata X della nuova forma segnaposto. |
| y | Single | La coordinata Y della nuova forma segnaposto. |
| width | Single | La larghezza della nuova forma segnaposto. |
| height | Single | L'altezza della nuova forma segnaposto. |

### Valore restituito

Creato [`IAutoShape`](../../iautoshape) con un segnaposto Picture.

### Esempi

Il seguente esempio mostra come aggiungere la forma segnaposto Picture alla diapositiva layout.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddPicturePlaceholder(20, 20, 200, 200);
}
```

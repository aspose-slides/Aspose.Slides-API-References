## ILayoutPlaceholderManager.AddVerticalTextPlaceholder metodo

Aggiunge una nuova forma segnaposto alla diapositiva layout per contenere il contenuto di testo in direzione verticale.

```csharp
public IAutoShape AddVerticalTextPlaceholder(float x, float y, float width, float height)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| x | Single | La coordinata X della nuova forma segnaposto. |
| y | Single | La coordinata Y della nuova forma segnaposto. |
| width | Single | La larghezza della nuova forma segnaposto. |
| height | Single | L’altezza della nuova forma segnaposto. |

### Valore restituito

Creato [`IAutoShape`](../../iautoshape) con un segnaposto Testo (Verticale).

### Esempi

Il seguente esempio mostra come aggiungere la forma segnaposto Testo (Verticale) alla diapositiva layout.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalTextPlaceholder(20, 20, 300, 500);
}
```

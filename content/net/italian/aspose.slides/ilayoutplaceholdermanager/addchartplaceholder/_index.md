## ILayoutPlaceholderManager.AddChartPlaceholder method

Aggiunge una nuova forma di segnaposto alla diapositiva del layout per contenere un Chart.

```csharp
public IAutoShape AddChartPlaceholder(float x, float y, float width, float height)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| x | Single | La coordinata X della nuova forma di segnaposto. |
| y | Single | La coordinata Y della nuova forma di segnaposto. |
| width | Single | La larghezza della nuova forma di segnaposto. |
| height | Single | L'altezza della nuova forma di segnaposto. |

### Valore di ritorno

Creato [`IAutoShape`](../../iautoshape) con un segnaposto Chart.

### Esempi

Il seguente esempio mostra come aggiungere la forma di segnaposto Chart alla diapositiva del layout.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddChartPlaceholder(20, 20, 200, 200);
}
```

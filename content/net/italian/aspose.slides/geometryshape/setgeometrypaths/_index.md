## GeometryShape.SetGeometryPaths metodo

Aggiorna la geometria della forma da un array di [`IGeometryPath`](../../igeometrypath). Le coordinate devono essere relative all'angolo superiore sinistro della forma. Cambia il tipo della forma ([`ShapeType`](../shapetype)) in Custom.

```csharp
public void SetGeometryPaths(IGeometryPath[] geometryPaths)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| geometryPaths | IGeometryPath[] | Array di percorsi di geometria |

### Eccezioni

| eccezione | condizione |
| --- | --- |
| ArgumentException | Nessun percorso trovato |
| ArgumentException | Percorso vuoto |

### Esempi

Esempio:

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    GeometryShape shape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 200, 100) as GeometryShape;

    GeometryPath geometryPath = shape.GetGeometryPaths()[0];

    geometryPath.LineTo(100, 50, 1);
    geometryPath.LineTo(100, 50, 4);

    shape.SetGeometryPath(geometryPath);

    pres.Save("output.pptx", SaveFormat.Pptx);
}
```

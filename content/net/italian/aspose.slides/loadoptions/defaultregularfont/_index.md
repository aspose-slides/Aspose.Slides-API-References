## LoadOptions.DefaultRegularFont proprietà

Restituisce o imposta il carattere Regolare usato nel caso in cui il carattere di origine non venga trovato. Lettura/Scrittura String.

```csharp
public string DefaultRegularFont { get; set; }
```

### Esempi

L'esempio seguente mostra come impostare i caratteri predefiniti per il rendering di una presentazione PowerPoint.

```csharp
[C#]
// Utilizza le opzioni di caricamento per definire i caratteri predefiniti regolari e asiatici
LoadOptions loadOptions = new LoadOptions(LoadFormat.Auto);
loadOptions.DefaultRegularFont = "Wingdings";
loadOptions.DefaultAsianFont = "Wingdings";
// Carica la presentazione
using (Presentation pptx = new Presentation("DefaultFonts.pptx", loadOptions))
{
    // Genera miniatura della diapositiva
    pptx.Slides[0].GetThumbnail(1, 1).Save("output_out.png", ImageFormat.Png);
    // Genera PDF
    pptx.Save("output_out.pdf", SaveFormat.Pdf);
    // Genera XPS
    pptx.Save("output_out.xps", SaveFormat.Xps);
}
```

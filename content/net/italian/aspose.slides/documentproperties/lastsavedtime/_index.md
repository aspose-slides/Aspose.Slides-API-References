## DocumentProperties.LastSavedTime proprietà

Restituisce la data in cui una presentazione è stata modificata per l'ultima volta. I valori sono in UTC. Di sola lettura nel caso di Presentation.DocumentProperties (perché verrà aggiornato internamente durante il processo di salvataggio dell'oggetto IPresentation). Può essere modificato tramite l'istanza di DocumentProperties restituita dal metodo [`ReadDocumentProperties`](../../ipresentationinfo/readdocumentproperties). Si prega di vedere l'esempio nel riepilogo del metodo [`UpdateDocumentProperties`](../../ipresentationinfo/updatedocumentproperties).

public DateTime LastSavedTime { get; set; }

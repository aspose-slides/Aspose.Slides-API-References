## ICameraEffectiveData.GetRotation metodo

Una rotazione è definita mediante l'uso di una coordinata di latitudine, una coordinata di longitudine e una rivoluzione attorno all'asse in base alle coordinate di latitudine e longitudine. Il primo elemento nell'array restituito - latitudine, il secondo - longitudine, il terzo - rivoluzione. Restituisce null se non è definita alcuna rotazione.

```csharp
public float[] GetRotation()
```

Array di valori di rotazione come Single[].

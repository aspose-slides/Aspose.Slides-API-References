## FontFallBackRulesCollection.Add metodo

Aggiunge una regola FallBack specificata alla fine della raccolta.

| sourceRule | IFontFallBackRule | Regola specificata per l'aggiunta |

### Esempi

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    //Ottenimento della collezione di regole vuota o preinizializzata da FontsManager
```

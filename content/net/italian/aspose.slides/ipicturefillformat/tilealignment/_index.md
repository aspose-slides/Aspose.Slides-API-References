## IPictureFillFormat.TileAlignment proprietà

Restituisce o imposta come la texture è allineata all'interno della forma. Questa impostazione controlla il punto di partenza del modello di texture e come si ripete sulla forma. Lettura/scrittura [`RectangleAlignment`](../../rectanglealignment).

```csharp
public RectangleAlignment TileAlignment { get; set; }
```

### Osservazioni

Il valore predefinito è TopLeft.

### Esempi

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Ottiene il formato di riempimento immagine della forma
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Imposta la modalità di riempimento immagine su Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Imposta l'allineamento del mosaico in basso a destra
    pictureFillFormat.TileAlignment = RectangleAlignment.BottomRight;
}
```

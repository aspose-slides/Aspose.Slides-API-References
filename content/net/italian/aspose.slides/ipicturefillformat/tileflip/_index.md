## IPictureFillFormat.TileFlip proprietà

Capovolge la tessera di texture lungo l'asse orizzontale, verticale o su entrambi gli assi. Lettura/Scrittura [`TileFlip`](../../tileflip).

```csharp
public TileFlip TileFlip { get; set; }
```

### Osservazioni

Il valore predefinito è NoFlip.

### Esempi

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Ottiene il formato di riempimento immagine della forma
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Imposta la modalità di riempimento immagine su Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Capovolge la tessera di texture lungo l'asse verticale.
    pictureFillFormat.TileFlip = TileFlip.FlipY;
}
```

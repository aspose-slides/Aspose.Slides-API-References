## IPictureFillFormat.TileScaleX proprietà

Restituisce o imposta la scala orizzontale per il riempimento a trama come percentuale. Lettura/scrittura Single.

```csharp
public float TileScaleX { get; set; }
```

### Esempi

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Ottiene il formato di riempimento immagine della forma
    IPictureFillFormat pictureFillFormat = slide.Shapes[0].FillFormat.PictureFillFormat;

    // Imposta la modalità di riempimento immagine su Tile
    pictureFillFormat.PictureFillMode = PictureFillMode.Tile;

    // Imposta la scala orizzontale della trama al 120 per cento
    pictureFillFormat.TileScaleX = 120;
}
```

## IPictureFillFormat.DeletePictureCroppedAreas metodo

Elimina le aree ritagliate del Picture di riempimento.

```csharp
public IPPImage DeletePictureCroppedAreas()
```

### Valore restituito

Immagine ritagliata o immagine originale se il ritaglio non è necessario.

### Note

Questo metodo converte i metafili WMF/EMF in immagini raster PNG durante il ritaglio.

### Esempi

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    ISlide slide = presentation.Slides[0];

    // Ottiene il PictureFrame
    IPictureFrame picFrame = slide.Shapes[0] as IPictureFrame;

    // Elimina le aree ritagliate dell'immagine PictureFrame
    IPPImage croppedImage = picFrame.PictureFormat.DeletePictureCroppedAreas();
}
```

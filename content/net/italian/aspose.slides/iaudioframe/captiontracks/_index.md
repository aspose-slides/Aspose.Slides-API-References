## IAudioFrame.CaptionTracks proprietà

Ottiene la raccolta di didascalie chiuse associate al fotogramma audio. Questa proprietà è di sola lettura e restituisce un [`ICaptionsCollection`](../../icaptionscollection) contenente tutte le tracce di didascalie.

```csharp
public ICaptionsCollection CaptionTracks { get; }
```

### Esempi

Esempio:

```csharp
[C#]
using (Presentation pres = new Presentation("audio with captions.pptx"))
{
    foreach (IShape shape in pres.Slides[0].Shapes)
    {
        if (shape is IAudioFrame audioFrame)
        {
            // Salva i dati binari della traccia di didascalia come file .vtt
            foreach (var captionTrack in audioFrame.CaptionTracks)
                File.WriteAllBytes(captionTrack.CaptionId + ".vtt", captionTrack.BinaryData);
        }
    }
}
```

## SetLicense(string) {#setlicense_1}

Registra la licenza del componente.

```csharp
public void SetLicense(string licenseName)
```

| Parameter | Type | Description |
| --- | --- | --- |
| licenseName | String | Può essere un nome file completo o breve oppure il nome di una risorsa incorporata. Utilizzare una stringa vuota per passare alla modalità di valutazione. |

### Osservazioni

Tenta di trovare la licenza nelle seguenti posizioni:

1. Percorso esplicito.
2. La cartella dell'assembly del componente.
3. La cartella dell'assembly chiamante del client.
4. La cartella dell'assembly di avvio.
5. Una risorsa incorporata nell'assembly chiamante del client.

**Nota:**Su il .NET Compact Framework, tenta di trovare la licenza solo in queste posizioni:

1. Percorso esplicito.
2. Una risorsa incorporata nell'assembly chiamante del client.

### Esempi

In questo esempio, si tenterà di trovare un file di licenza denominato MyLicense.lic nella cartella che contiene il componente, nella cartella che contiene l'assembly chiamante, nella cartella dell'assembly di avvio e infine nelle risorse incorporate dell'assembly chiamante.

```csharp
[C#]

License license = new License();
license.SetLicense("MyLicense.lic");


[Visual Basic]

Dim license As License = New License
license.SetLicense("MyLicense.lic")
```

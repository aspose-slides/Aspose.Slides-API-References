## Fonts.SetScriptFont method

Assegna un nome di carattere a un tag script specifico, che definisce come il testo di quello script verrà visualizzato nella presentazione.

```csharp
public void SetScriptFont(string script, string fontName)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| script | String | Il codice script BCP-47 (ad es., "Arab", "Hebr", "Hans") che identifica il sistema di scrittura. |
| fontName | String | Il nome del carattere da assegnare allo script specificato. |

### Esempi

Questo esempio mostra come impostare il carattere per lo script arabo su "Segoe UI":

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segue UI");
```

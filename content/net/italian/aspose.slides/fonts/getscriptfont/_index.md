## Fonts.GetScriptFont metodo

Restituisce il nome del carattere associato a un tag di script specifico dal tema della presentazione.

```csharp
public string GetScriptFont(string script)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| script | String | Il codice script BCP-47 (ad esempio "Latn", "Cyrl", "Jpan") usato per identificare un sistema di scrittura. |

### Valore restituito

Il nome del carattere usato per lo script specificato, o `null` se lo script non è definito.

### Esempi

Questo esempio mostra come recuperare il carattere assegnato allo script cirillico nel tema della presentazione.

```csharp
[C#]
string font = presentation.MasterTheme.FontScheme.Major.GetScriptFont("Cyrl");
Console.WriteLine("Font for Cyrillic script: " + font);
```

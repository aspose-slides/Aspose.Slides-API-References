## Tab.CompareTo metodo

Confronta l'istanza corrente con un altro oggetto dello stesso tipo.

```csharp
public int CompareTo(object obj)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| obj | Object | Un oggetto da confrontare con questa istanza. |

### Valore di ritorno

Un intero a 32 bit che indica l'ordine relativo dei comparandi. Il valore restituito ha i seguenti significati:  &lt; 0 - Questa istanza è minore di obj. = 0 - Questa istanza è uguale a obj. &gt; 0 - Questa istanza è maggiore di obj.

## IFontFallBackRulesCollection.Remove metodo

Rimuove la prima occorrenza di una regola FallBack specifica dalla raccolta.

```csharp
public void Remove(IFontFallBackRule targetRule)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| targetRule | IFontFallBackRule | La regola da rimuovere dalla raccolta. |

### Esempi

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    //Ottenimento della raccolta di regole vuota o preinizializzata da FontsManager
    IFontFallBackRulesCollection rulesList = pres.FontsManager.FontFallBackRulesCollection;

    //Aggiunta di diverse regole alla raccolta
    rulesList.Add(new FontFallBackRule(0x400,0x4FF, "Times New Roman"));
    rulesList.Add(new FontFallBackRule(0x3040, 0x309F, "MS Mincho"));

    //Recupero dell'oggetto della prima regola nella raccolta
    IFontFallBackRule firstRule = rulesList[0];
    //Rimozione 
    rulesList.Remove (firstRule);
}
```

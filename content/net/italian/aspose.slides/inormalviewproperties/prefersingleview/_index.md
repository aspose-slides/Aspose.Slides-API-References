## INormalViewProperties.PreferSingleView proprietà

Specifica se l'utente preferisce vedere una regione di contenuto singola a finestra intera rispetto alla visualizzazione normale standard con tre regioni di contenuto. Se abilitata, l'applicazione può scegliere di visualizzare una delle regioni di contenuto nell'intera finestra. Lettura/scrittura Boolean.

public bool PreferSingleView { get; set; }

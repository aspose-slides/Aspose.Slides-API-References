## MasterLayoutSlideCollection.Add metodo

Aggiunge una nuova diapositiva di layout alla fine della raccolta.

```csharp
public ILayoutSlide Add(SlideLayoutType layoutType, string layoutName)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| layoutType | SlideLayoutType | Tipo di layout per un nuovo layout. Tipi di layout supportati: Title, TitleOnly, Blank, TitleAndObject, VerticalText, VerticalTitleAndText, TwoObjects, SectionHeader, TwoTextAndTwoObjects, TitleObjectAndCaption, PictureAndCaption, Custom. Altri tipi di layout non sono attualmente supportati: Text, TwoColumnText, Table, TextAndChart, ChartAndText, Diagram, Chart, TextAndClipArt, ClipArtAndText, TextAndObject, ObjectAndText, Object, TextAndMedia, MediaAndText, ObjectOverText, TextOverObject, TextAndTwoObjects, TwoObjectsAndText, TwoObjectsOverText, FourObjects, ClipArtAndVerticalText, VerticalTitleAndTextOverChart, ObjectAndTwoObject, TwoObjectsAndObject. |
| layoutName | String | Nome per un nuovo layout. Se il nome fornito è già in uso verrà generata un'ArgumentException. Se viene passato un parametro null il nome sarà generato automaticamente in base al tipo di layout fornito (ad esempio "Title Slide" o "1_Title Slide", "2_..", ecc.). |

### Valore restituito

Diapositiva aggiunta.

### Eccezioni

| eccezione | condizione |
| --- | --- |
| NotImplementedException | Lanciata se viene passato un valore non supportato per il parametro *layoutType*. Tipi di layout non supportati al momento: Text, TwoColumnText, Table, TextAndChart, ChartAndText, Diagram, Chart, TextAndClipArt, ClipArtAndText, TextAndObject, ObjectAndText, Object, TextAndMedia, MediaAndText, ObjectOverText, TextOverObject, TextAndTwoObjects, TwoObjectsAndText, TwoObjectsOverText, FourObjects, ClipArtAndVerticalText, VerticalTitleAndTextOverChart, ObjectAndTwoObject, TwoObjectsAndObject. |
| ArgumentException | Lanciata se il valore del nome del layout *layoutName* è già in uso in questa raccolta di layout. |

### Osservazioni

1) Il layout aggiunto per il valore SlideLayoutType.Custom di *layoutType* non contiene segnaposti né forme. 2) L'analogo di questo metodo è il metodo [`Add`](../../igloballayoutslidecollection/add) accessibile tramite la proprietà [`LayoutSlides`](../../ipresentation/layoutslides).

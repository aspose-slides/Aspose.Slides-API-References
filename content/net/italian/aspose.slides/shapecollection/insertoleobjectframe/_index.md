## InsertOleObjectFrame(int, float, float, float, float, IOleEmbeddedDataInfo) {#insertoleobjectframe}

Crea un nuovo frame di oggetto OLE e lo inserisce nella raccolta di forme all’indice specificato.

```csharp
public IOleObjectFrame InsertOleObjectFrame(int index, float x, float y, float width, float height, 
    IOleEmbeddedDataInfo dataInfo)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| index | Int32 | L’indice basato su zero al quale inserire il frame dell’oggetto OLE. |
| x | Single | La coordinata x del nuovo frame OLE, in punti. |
| y | Single | La coordinata y del nuovo frame OLE, in punti. |
| width | Single | La larghezza del nuovo frame OLE, in punti. |
| height | Single | L’altezza del nuovo frame OLE, in punti. |
| dataInfo | IOleEmbeddedDataInfo | Le informazioni sui dati OLE incorporati ([`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo)). |

### Valore di ritorno

Il nuovo [`IOleObjectFrame`](../../ioleobjectframe).

### Esempi

Questo esempio mostra come inserire un oggetto OLE al secondo indice:

```csharp
[C#]
byte[] fileData = File.ReadAllBytes("test.zip");
IOleDataInfo dataInfo = new OleDataInfo(fileData, "zip");
IOleObjectFrame oleObjectFrame = slidees.Shapes.AddOleObjectFrame(2, 150, 20, 50, 50, dataInfo);
```

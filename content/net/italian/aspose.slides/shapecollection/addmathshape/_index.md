## ShapeCollection.AddMathShape metodo

Crea una nuova forma automatica rettangolare per ospitare contenuti matematici e la aggiunge alla fine della collezione di forme.

```csharp
public IAutoShape AddMathShape(float x, float y, float width, float height)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| x | Single | La coordinata x del riquadro della forma, in punti. |
| y | Single | La coordinata y del riquadro della forma, in punti. |
| width | Single | La larghezza del riquadro della forma, in punti. |
| height | Single | L'altezza del riquadro della forma, in punti. |

### Valore di ritorno

Il [`IAutoShape`](../../iautoshape) appena creato.

### Esempi

Il seguente esempio mostra come aggiungere un'equazione matematica in una presentazione PowerPoint.

```csharp
[C#]
using (Presentation pres = new Presentation())
{
   IAutoShape mathShape = pres.Slides[0].Shapes.AddMathShape(0, 0, 720, 150);
   var mathParagraph = (mathShape.TextFrame.Paragraphs[0].Portions[0] as MathPortion).MathParagraph;
   var fraction = new MathematicalText("x").Divide("y");
   mathParagraph.Add(new MathBlock(fraction));
   var mathBlock = new MathematicalText("c")
        .SetSuperscript("2")
        .Join("=")
        .Join(new MathematicalText("a").SetSuperscript("2"))
        .Join("+")
        .Join(new MathematicalText("b").SetSuperscript("2"));
    mathParagraph.Add(mathBlock);
    pres.Save("math.pptx", SaveFormat.Pptx);
}
```

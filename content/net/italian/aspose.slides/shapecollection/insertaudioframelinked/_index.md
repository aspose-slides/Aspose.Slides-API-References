```csharp
public IAudioFrame InsertAudioFrameLinked(int index, float x, float y, float width, float height, 
    string fname)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| index | Int32 | L'indice basato su zero in cui inserire il fotogramma audio. |
| x | Single | La coordinata x del nuovo fotogramma audio, in punti. |
| y | Single | La coordinata y del nuovo fotogramma audio, in punti. |
| width | Single | La larghezza del nuovo fotogramma audio, in punti. |
| height | Single | L'altezza del nuovo fotogramma audio, in punti. |

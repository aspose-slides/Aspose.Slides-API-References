## AddZoomFrame(float, float, float, float, ISlide) {#addzoomframe}

Crea un nuovo frame Zoom e lo aggiunge alla fine della collezione di forme.

```csharp
public IZoomFrame AddZoomFrame(float x, float y, float width, float height, ISlide slide)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| x | Single | La coordinata x del nuovo frame Zoom, in punti. |
| y | Single | La coordinata y del nuovo frame Zoom, in punti. |
| width | Single | La larghezza del nuovo frame Zoom, in punti. |
| height | Single | L'altezza del nuovo frame Zoom, in punti. |
| slide | ISlide | Il [`ISlide`](../../islide) a cui fa riferimento il frame Zoom; deve appartenere a questa presentazione. |

### Valore restituito

Il nuovo [`IZoomFrame`](../../izoomframe).

### Eccezioni

| Eccezione | Condizione |
| --- | --- |
| ArgumentException | Lanciata se la slide a cui si fa riferimento non appartiene alla presentazione corrente. |

### Esempi

Questo esempio dimostra come aggiungere un oggetto Zoom alla fine di una collezione (supponendo che nella presentazione "Presentation.pptx" ci siano almeno due slide):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
}
```

## AddZoomFrame(float, float, float, float, ISlide, IPPImage) {#addzoomframe_1}

Crea un nuovo frame Zoom e lo aggiunge alla fine della collezione di forme.

```csharp
public IZoomFrame AddZoomFrame(float x, float y, float width, float height, ISlide slide, 
    IPPImage image)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| x | Single | La coordinata x del nuovo frame Zoom, in punti. |
| y | Single | La coordinata y del nuovo frame Zoom, in punti. |
| width | Single | La larghezza del nuovo frame Zoom, in punti. |
| height | Single | L'altezza del nuovo frame Zoom, in punti. |
| slide | ISlide | Il [`ISlide`](../../islide) a cui fa riferimento il frame Zoom; deve appartenere a questa presentazione. |
| image | IPPImage | L'immagine per la slide [`IPPImage`](../../ippimage) a cui si fa riferimento. |

### Valore restituito

Il nuovo [`IZoomFrame`](../../izoomframe).

### Eccezioni

| Eccezione | Condizione |
| --- | --- |
| ArgumentException | Lanciata se la slide a cui si fa riferimento non appartiene alla presentazione corrente. |

### Esempi

Questo esempio dimostra come aggiungere un oggetto Zoom alla fine di una collezione (supponendo che nella presentazione "Presentation.pptx" ci siano almeno due slide):

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IPPImage image = pres.Images.AddImage(Image.FromFile("image.png"));
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1], image);
}
```

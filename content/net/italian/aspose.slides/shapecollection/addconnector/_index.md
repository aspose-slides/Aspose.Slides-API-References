## AddConnector(ShapeType, float, float, float, float) {#addconnector}

Crea una nuova forma connettore con lo stile predefinito del modello e la aggiunge alla fine della raccolta di forme.

```csharp
public IConnector AddConnector(ShapeType shapeType, float x, float y, float width, float height)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| shapeType | ShapeType | Il [`ShapeType`](../../shapetype) della forma connettore da aggiungere. |
| x | Single | La coordinata x del riquadro del connettore, in punti. |
| y | Single | La coordinata y del riquadro del connettore, in punti. |
| width | Single | La larghezza del riquadro del connettore, in punti. |
| height | Single | L'altezza del riquadro del connettore, in punti. |

### Valore restituito

Il [`IConnector`](../../iconnector) appena creato.

### Esempi

Il seguente esempio mostra come aggiungere un connettore (un connettore curvo) tra due forme (un'ellisse e un rettangolo) in una presentazione PowerPoint.

```csharp
[C#]
// Istanzia una classe Presentation che rappresenta un file PPTX
using (Presentation input = new Presentation())
{
    // Accede alla raccolta di forme per una diapositiva specifica
    IShapeCollection shapes = input.Slides[0].Shapes;
    // Aggiunge una forma automatica Ellipse
    IAutoShape ellipse = shapes.AddAutoShape(ShapeType.Ellipse, 0, 100, 100, 100);
    // Aggiunge una forma automatica Rectangle
    IAutoShape rectangle = shapes.AddAutoShape(ShapeType.Rectangle, 100, 300, 100, 100);
    // Aggiunge una forma connettore alla raccolta di forme della diapositiva
    IConnector connector = shapes.AddConnector(ShapeType.BentConnector2, 0, 0, 10, 10);
    // Collega le forme usando il connettore
    connector.StartShapeConnectedTo = ellipse;
    connector.EndShapeConnectedTo = rectangle;
    // Chiama Reroute che imposta il percorso più corto automatico tra le forme
    connector.Reroute();
    // Salva la presentazione
    input.Save("Shapes-connector.pptx", SaveFormat.Pptx);
}
```

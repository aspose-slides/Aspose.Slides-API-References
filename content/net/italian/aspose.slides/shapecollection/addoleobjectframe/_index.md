## AddOleObjectFrame(float, float, float, float, IOleEmbeddedDataInfo) {#addoleobjectframe}

Crea un nuovo frame oggetto OLE e lo aggiunge alla fine della raccolta di forme.

```csharp
public IOleObjectFrame AddOleObjectFrame(float x, float y, float width, float height, 
    IOleEmbeddedDataInfo dataInfo)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| x | Single | La coordinata x del nuovo frame OLE, in punti. |
| y | Single | La coordinata y del nuovo frame OLE, in punti. |
| width | Single | La larghezza del nuovo frame OLE, in punti. |
| height | Single | L'altezza del nuovo frame OLE, in punti. |
| dataInfo | IOleEmbeddedDataInfo | Le informazioni sui dati OLE incorporati ([`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo)). |

### Valore restituito

Il nuovo [`IOleObjectFrame`](../../ioleobjectframe) creato.

### Esempi

I seguenti esempi mostrano come aggiungere frame oggetto OLE alle diapositive di una presentazione PowerPoint.

```csharp
[C#]
// Istanzia la classe Presentation che rappresenta il file PPTX
using (Presentation pres = new Presentation())
{
    // Accede alla prima diapositiva
    ISlide sld = pres.Slides[0];
    // Carica un file Excel in uno stream
    MemoryStream mstream = new MemoryStream();
    using (FileStream fs = new FileStream("book1.xlsx", FileMode.Open, FileAccess.Read))
    {
        byte[] buf = new byte[4096];
        while (true)
        {
            int bytesRead = fs.Read(buf, 0, buf.Length);
            if (bytesRead <= 0)
                break;
            mstream.Write(buf, 0, bytesRead);
        }
    }
    // Crea un oggetto dati per l'incorporamento
    IOleEmbeddedDataInfo dataInfo = new OleEmbeddedDataInfo(mstream.ToArray(), "xlsx");
    // Aggiunge una forma Ole Object Frame
    IOleObjectFrame oleObjectFrame = sld.Shapes.AddOleObjectFrame(0, 0, pres.SlideSize.Size.Width,
        pres.SlideSize.Size.Height, dataInfo);
    // Scrive il file PPTX su disco
    pres.Save("OleEmbed_out.pptx", SaveFormat.Pptx);
}
```

## Portion.PortionFormat proprietà

Restituisce un oggetto di formattazione che contiene le proprietà di formattazione impostate esplicitamente della porzione di testo senza ereditarietà applicata. Solo lettura [`IPortionFormat`](../../iportionformat).

```csharp
public IPortionFormat PortionFormat { get; }
```

### Osservazioni

L'oggetto di formattazione contiene i parametri di formattazione definiti solo per la porzione corrente; i dati ereditati non sono applicati.

Per ottenere i valori effettivi includendo quelli ereditati, utilizzare il metodo [`GetEffective`](../../portionformat/geteffective).

## Shape.Rotation property

Restituisce o imposta il numero di gradi di rotazione della forma specificata attorno all'asse z. Un valore positivo indica una rotazione in senso orario; un valore negativo indica una rotazione in senso antiorario. Lettura/scrittura Single.

```csharp
public float Rotation { get; set; }
```

### Osservazioni

Il valore restituito è sempre definito (non è float.NaN). Il valore assegnato deve essere definito (non è float.NaN). È possibile impostare valori non definiti per le proprietà dell'istanza RawFrame.

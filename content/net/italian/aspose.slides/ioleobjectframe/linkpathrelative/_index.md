## IOleObjectFrame.LinkPathRelative proprietà

Restituisce il percorso relativo a un file collegato, se presente, altrimenti restituisce una stringa vuota. Solo lettura String.

```csharp
public string LinkPathRelative { get; }
```

### Osservazioni

Nelle presentazioni Ppt, alcuni collegamenti a oggetti Ole possono avere una rappresentazione relativa.

### Esempi

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.ppt"))
{
    IOleObjectFrame oleFrame = presentation.Slides[0].Shapes[0] as IOleObjectFrame;

    if (oleFrame != null)
    {
        Console.WriteLine("The relative path: " + oleFrame.LinkPathRelative);
    } 
}
```

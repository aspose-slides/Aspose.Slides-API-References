## BehaviorPropertyCollection.CopyTo メソッド

ICollection の要素を Array にコピーします。コピーは特定の Array インデックスから開始します。

```csharp
public void CopyTo(IBehaviorProperty[] array, int arrayIndex)
```

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| array | IBehaviorProperty[] | ICollection からコピーされた要素の宛先となる一次元の Array です。Array はゼロベースのインデックスを持つ必要があります。 |
| arrayIndex | Int32 | コピーが開始される *array* のゼロベースインデックスです。 |

### 例外

| 例外 | 条件 |
| --- | --- |
| ArgumentNullException | *array* が null です。 |
| ArgumentOutOfRangeException | *arrayIndex* が 0 未満です。 |
| ArgumentException | ソース ICollection の要素数が、*arrayIndex* から宛先 *array* の末尾までの利用可能な空き領域より大きいです。 |

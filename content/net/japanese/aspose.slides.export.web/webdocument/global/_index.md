## WebDocument.Global プロパティ

ドキュメントのグローバル ストレージを返します。読み取り専用 [`Storage`](../../storage).

```csharp
public Storage Global { get; }
```

### 例

この `Global` プロパティ（[`Storage`](../../storage) インターフェイスの実装）を使用すると、プロパティをテンプレートの後で使用できるように設定できます。

```csharp
[C#]
var options = new WebDocumentOptions
{
    TemplateEngine = new RazorTemplateEngine(),
    OutputSaver = new FileOutputSaver(),
    EmbedImages = false
};

WebDocument document = new WebDocument(options);

// テンプレートから使用するために "slideMargin" プロパティを設定する
document.Global.Put("slideMargin", 10);

// ... ドキュメントの他のオプションを設定し、次にドキュメントを保存します
document.Save();
```

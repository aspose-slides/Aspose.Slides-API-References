## SvgImage.WriteAsEmf メソッド

SVG イメージを EMF ファイルとして保存します。

```csharp
public void WriteAsEmf(Stream stream)
```

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| stream | Stream | 対象ストリーム |

### 例外

| 例外 | 条件 |
| --- | --- |
| ArgumentNullException | 対象ストリームが **null** です |

### 例

以下の例は、SVG イメージをメタファイルに保存する方法を示します。

```csharp
[C#]
// 新しい SVG イメージを作成します
ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));

// SVG イメージをメタファイルとして保存します
using (var fileStream = System.IO.File.OpenWrite("SvgAsEmf.emf"))
    svgImage.WriteAsEmf(fileStream);
```

このサンプルは、SVG イメージをメタファイルとしてプレゼンテーションの画像コレクションに追加する方法を示しています。

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // 新しい SVG イメージを作成します
    ISvgImage svgImage = new SvgImage(System.IO.File.ReadAllText("content.svg"));
    using (var memStream = new MemoryStream())
    {
        // SVG イメージをメタファイルとして保存します
        svgImage.WriteAsEmf(memStream);
        // メタファイルを画像コレクションに追加します
        pres.Images.AddImage(memStream.ToArray());
    }
}
```

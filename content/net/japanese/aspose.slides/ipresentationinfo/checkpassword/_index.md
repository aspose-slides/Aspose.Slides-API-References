## IPresentationInfo.CheckPassword メソッド

オープンパスワードで保護されたプレゼンテーションに対して、パスワードが正しいかどうかを確認します。

```csharp
public bool CheckPassword(string password)
```

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| password | String | チェックするパスワード。 |

### 戻り値

プレゼンテーションがオープンパスワードで保護され、かつパスワードが正しい場合は True、そうでない場合は false を返します。

### 備考

パスワードが null または空文字列の場合、このメソッドは false を返します。

### 例

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo("pres.pptx");
bool isPasswordCorrect = info.CheckPassword("my_password");
```

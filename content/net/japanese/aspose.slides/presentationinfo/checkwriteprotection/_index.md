## PresentationInfo.CheckWriteProtection メソッド

プレゼンテーションが書き込み保護されており、変更用パスワードが正しいかどうかを確認します。

```csharp
public bool CheckWriteProtection(string password)
```

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| password | String | 確認するパスワード。 |

### 戻り値

プレゼンテーションが書き込み保護されており、パスワードが正しい場合は True。そうでない場合は False。

### 例外

| 例外 | 条件 |
| --- | --- |
| InvalidOperationException |  |

### 備考

1. このメソッドを呼び出す前に [`IsWriteProtected`](../iswriteprotected) プロパティを確認してください。  
2. パスワードが null または空の場合、このメソッドは false を返します。

### 例

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
if (info.IsWriteProtected == NullableBool.True)
{
    bool isWriteProtectedByPassword = info.CheckWriteProtection("my_password");
}
```

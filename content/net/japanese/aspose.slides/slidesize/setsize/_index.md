## SetSize(SlideSizeType, SlideSizeScaleType) {#setsize}

スライドのサイズを種類で設定し、既存のコンテンツをスケーリングします。

```csharp
public void SetSize(SlideSizeType type, SlideSizeScaleType scaleType)
```

| Parameter | Type | Description |
| --- | --- | --- |
| type | SlideSizeType | 適用する事前定義済みスライドサイズです。 |
| scaleType | SlideSizeScaleType | 使用するコンテンツのスケーリングモードです。 |

### 備考

Custom 以外の値を割り当てると、[`Size`](../size) が選択したタイプに基づいて調整され、[`Orientation`](../orientation) は保持されます。

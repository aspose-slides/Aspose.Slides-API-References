## Fonts.SetScriptFont メソッド

特定のスクリプトタグにフォント名を割り当てます。これにより、そのスクリプトのテキストがプレゼンテーションでどのようにレンダリングされるかが定義されます。

```csharp
public void SetScriptFont(string script, string fontName)
```

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| script | String | 書記体系を識別する BCP-47 スクリプトコード（例: "Arab", "Hebr", "Hans"）。 |
| fontName | String | 指定されたスクリプトに割り当てるフォント名。 |

### 例

この例では、アラビア文字スクリプト用のフォントを "Segoe UI" に設定する方法を示します：

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segoe UI");
```

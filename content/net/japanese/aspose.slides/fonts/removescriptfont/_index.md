## Fonts.RemoveScriptFont メソッド

テーマのフォントコレクションから、特定のスクリプトタグに関連付けられたフォント設定を削除します。

```csharp
public void RemoveScriptFont(string script)
```

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| script | String | 削除すべきフォント設定が設定されている BCP-47 スクリプトコード。 |

### 例

この例では、ヘブライ語スクリプトのフォントマッピングを削除する方法を示します：

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.RemoveScriptFont("Hebr");
```

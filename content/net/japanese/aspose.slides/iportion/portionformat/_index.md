## IPortion.PortionFormat プロパティ

継承が適用されていない、テキスト部分に対して明示的に設定された書式プロパティを含む書式オブジェクトを返します。読み取り専用 [`IPortionFormat`](../../iportionformat)。

```csharp
public IPortionFormat PortionFormat { get; }
```

### 備考

書式オブジェクトは現在の部分に対して定義された書式パラメータのみを含み、継承されたデータは適用されません。

継承された値を含む実効値を取得するには、[`GetEffective`](../../iportionformat/geteffective) メソッドを使用します。

## IProtectionManager.CheckWriteProtection メソッド

プレゼンテーションが変更のためにパスワードで保護されているかどうかを判断します。

```csharp
public bool CheckWriteProtection(string password)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| password | String | 確認用のパスワードです。 |

### 戻り値

パスワードが有効な場合は true、そうでない場合は false。

### 備考

1. このメソッドを呼び出す前に [`IsWriteProtected`](../iswriteprotected) プロパティを確認する必要があります。  
2. パスワードが null または空文字列の場合、このメソッドは false を返します。

### 例

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

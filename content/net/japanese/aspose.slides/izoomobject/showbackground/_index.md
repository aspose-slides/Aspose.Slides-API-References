## IZoomObject.ShowBackground プロパティ

ズームが宛先スライドの背景を使用するかどうかを指定する値を取得または設定します。読み取り/書き込み Boolean。既定値: true

```csharp
public bool ShowBackground { get; set; }
```

### 例

この例は、Zoom オブジェクトの画像の背景を削除する方法を示しています：

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.AddZoomFrame(150, 20, 50, 50, pres.Slides[1]);
    zoomFrame.ShowBackground = false;
}
```

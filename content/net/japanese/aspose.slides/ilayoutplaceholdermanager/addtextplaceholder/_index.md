## ILayoutPlaceholderManager.AddTextPlaceholder メソッド

レイアウト スライドにテキスト コンテンツを保持する新しいプレースホルダー シェイプを追加します。

```csharp
public IAutoShape AddTextPlaceholder(float x, float y, float width, float height)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| x | Single | 新しいプレースホルダー シェイプの X 座標です。 |
| y | Single | 新しいプレースホルダー シェイプの Y 座標です。 |
| width | Single | 新しいプレースホルダー シェイプの幅です。 |
| height | Single | 新しいプレースホルダー シェイプの高さです。 |

### 戻り値

[`IAutoShape`](../../iautoshape) が Text プレースホルダーとして作成されました。

### 例

次の例は、レイアウト スライドに Text プレースホルダー シェイプを追加する方法を示しています。

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTextPlaceholder(20, 20, 500, 300);
}
```

## ILayoutPlaceholderManager.AddSmartArtPlaceholder メソッド

レイアウト スライドに新しいプレースホルダー シェイプを追加し、SmartArt 図を保持します。

```csharp
public IAutoShape AddSmartArtPlaceholder(float x, float y, float width, float height)
```

| パラメーター | タイプ | 説明 |
| --- | --- | --- |
| x | Single | 新しいプレースホルダー シェイプの X 座標。 |
| y | Single | 新しいプレースホルダー シェイプの Y 座標。 |
| width | Single | 新しいプレースホルダー シェイプの幅。 |
| height | Single | 新しいプレースホルダー シェイプの高さ。 |

### 戻り値

[`IAutoShape`](../../iautoshape) を SmartArt プレースホルダーで作成しました。

### 使用例

以下の例は、SmartArt プレースホルダー シェイプをレイアウト スライドに追加する方法を示しています。

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddSmartArtPlaceholder(20, 20, 200, 200);
}
```

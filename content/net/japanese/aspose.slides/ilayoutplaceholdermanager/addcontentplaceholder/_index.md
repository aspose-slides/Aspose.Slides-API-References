## ILayoutPlaceholderManager.AddContentPlaceholder メソッド

レイアウトスライドに新しいプレースホルダーシェイプを追加し、画像、表、メディア、テキストなどのコンテンツを保持します。

```csharp
public IAutoShape AddContentPlaceholder(float x, float y, float width, float height)
```

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| x | Single | 新しいプレースホルダーシェイプの X 座標。 |
| y | Single | 新しいプレースホルダーシェイプの Y 座標。 |
| width | Single | 新しいプレースホルダーシェイプの幅。 |
| height | Single | 新しいプレースホルダーシェイプの高さ。 |

### 戻り値

[`IAutoShape`](../../iautoshape) を Content プレースホルダーとして作成しました。

### 例

次の例は、レイアウトスライドに Content プレースホルダーシェイプを追加する方法を示しています。

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddContentPlaceholder(20, 20, 500, 300);
}
```

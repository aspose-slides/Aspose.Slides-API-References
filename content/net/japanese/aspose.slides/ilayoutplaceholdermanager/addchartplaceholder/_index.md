## ILayoutPlaceholderManager.AddChartPlaceholder メソッド

レイアウト スライドに新しいプレースホルダー シェイプを追加し、チャートを保持できるようにします。

```csharp
public IAutoShape AddChartPlaceholder(float x, float y, float width, float height)
```

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| x | Single | 新しいプレースホルダー シェイプの X 座標です。 |
| y | Single | 新しいプレースホルダー シェイプの Y 座標です。 |
| width | Single | 新しいプレースホルダー シェイプの幅です。 |
| height | Single | 新しいプレースホルダー シェイプの高さです。 |

### 戻り値

[`IAutoShape`](../../iautoshape) を Chart プレースホルダーとして作成しました。

### 例

以下の例は、レイアウト スライドに Chart プレースホルダー シェイプを追加する方法を示しています。

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddChartPlaceholder(20, 20, 200, 200);
}
```

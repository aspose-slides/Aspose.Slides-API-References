## ILayoutPlaceholderManager.AddMediaPlaceholder メソッド

レイアウト スライドにメディア オブジェクトを格納するための新しいプレースホルダーシェイプを追加します。

```csharp
public IAutoShape AddMediaPlaceholder(float x, float y, float width, float height)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| x | Single | 新しいプレースホルダーシェイプの X 座標。 |
| y | Single | 新しいプレースホルダーシェイプの Y 座標。 |
| width | Single | 新しいプレースホルダーシェイプの幅。 |
| height | Single | 新しいプレースホルダーシェイプの高さ。 |

### 戻り値

[`IAutoShape`](../../iautoshape) を Media プレースホルダーで作成しました。

### 例

以下の例は、レイアウト スライドに Media プレースホルダーシェイプを追加する方法を示します。

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddMediaPlaceholder(20, 20, 200, 200);
}
```

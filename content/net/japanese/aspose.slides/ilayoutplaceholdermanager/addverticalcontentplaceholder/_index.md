## ILayoutPlaceholderManager.AddVerticalContentPlaceholder メソッド

レイアウトスライドに新しいプレースホルダー シェイプを追加し、画像、表、メディア、またはテキストなどのコンテンツを垂直方向に保持します。

```csharp
public IAutoShape AddVerticalContentPlaceholder(float x, float y, float width, float height)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| x | Single | 新しいプレースホルダー シェイプの X 座標。 |
| y | Single | 新しいプレースホルダー シェイプの Y 座標。 |
| width | Single | 新しいプレースホルダー シェイプの幅。 |
| height | Single | 新しいプレースホルダー シェイプの高さ。 |

### 戻り値

Content (Vertical) プレースホルダーを使用して [`IAutoShape`](../../iautoshape) を作成しました。

### 例

次の例は、Content (Vertical) プレースホルダー シェイプをレイアウトスライドに追加する方法を示しています。

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
    IAutoShape placeholder = layout.PlaceholderManager.AddVerticalContentPlaceholder(20, 20, 300, 500);
}
```

## BulletFormat.Height プロパティ

段落のバレット高さを取得または設定します（継承なし）。値 float.NaN は、バレットが段落の最初の部分から高さを継承することを示します。読み書き Single.

```csharp
public float Height { get; set; }
```

### 備考

負の高さの値は高さがポイントで指定されていることを意味し、正の値は高さが周囲のテキストのパーセンテージであることを意味します。

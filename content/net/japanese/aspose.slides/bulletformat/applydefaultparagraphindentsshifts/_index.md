## BulletFormat.ApplyDefaultParagraphIndentsShifts メソッド

 bullets が有効な場合（PowerPoint で段落の箇条書き/番号付けを有効にしたときと同様）に、効果的な段落の Indent と MarginLeft に対してデフォルトの非ゼロシフトを設定します。 bullets が無効な場合は、段落の Indent と MarginLeft をリセットします（PowerPoint で段落の箇条書き/番号付けを無効にしたときと同様）。 シフトは現在の bullet コンテキスト（IBulletFormat.Type、.NumberedBulletStyle、最初の部分の FontHeight）に基づいて適用されます。 非ゼロのインデントシフトは、現在の段落の効果的な Indent と MarginLeft に適用され、結果の値はローカル値になります。

```csharp
public void ApplyDefaultParagraphIndentsShifts()
```

## ParagraphCollection.CopyTo メソッド

ICollection の要素を Array にコピーし、特定の Array インデックスから開始します。

```csharp
public void CopyTo(IParagraph[] array, int arrayIndex)
```

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| array | IParagraph[] | ICollection からコピーされた要素の宛先となる一次元の Array です。Array はゼロベースのインデックスを持つ必要があります。 |
| arrayIndex | Int32 | *array* の中でコピーを開始するゼロベースのインデックスです。 |

### 例外

| 例外 | 条件 |
| --- | --- |
| ArgumentNullException | *array* が null です。 |
| ArgumentOutOfRangeException | *arrayIndex* が 0 未満です。 |
| ArgumentException | ソース ICollection の要素数が、*arrayIndex* から宛先 *array* の末尾までの空き領域を超えています。 |

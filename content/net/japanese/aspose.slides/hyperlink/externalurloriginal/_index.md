## Hyperlink.ExternalUrlOriginal プロパティ

この部分の実際のコンテンツに関係なく、この部分に設定されたハイパーリンクを表します。  
PowerPoint は、リンクとその対応するテキストに対して特別な動作を行います。  
有効な URL の形式でハイパーリンクのテキストを作成でき、リンクの実際のアドレスとは異なります。  
この場合、編集ウィンドウでリンクを表示すると、テキスト部分に合わせて変更されます。  
このプロパティはハイパーリンクの元の値を表します。

```csharp
public string ExternalUrlOriginal { get; }
```

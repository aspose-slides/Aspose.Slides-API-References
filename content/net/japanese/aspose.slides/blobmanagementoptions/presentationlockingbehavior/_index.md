## BlobManagementOptions.PresentationLockingBehavior プロパティ

このプロパティは、Presentation クラスのインスタンスがインスタンスの存続期間中にソース（ファイルまたはストリーム）の所有者になれるかどうかを定義します。インスタンスが所有者である場合、ソースはロックされます。これにより BLOB を扱う際のメモリ使用量とパフォーマンスが向上しますが、Presentation のインスタンス存続期間中はソース（ストリームまたはファイル）を変更できません。

```csharp
public PresentationLockingBehavior PresentationLockingBehavior { get; set; }
```

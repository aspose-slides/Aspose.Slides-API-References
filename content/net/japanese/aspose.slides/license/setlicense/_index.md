## SetLicense(string) {#setlicense_1}

コンポーネントにライセンスを設定します。

```csharp
public void SetLicense(string licenseName)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| licenseName | String | フルまたは短いファイル名、または埋め込みリソースの名前を指定できます。空文字列を使用すると評価モードに切り替わります。 |

### 備考

ライセンスを次の場所で検索します。

1. 明示的なパス。
2. コンポーネント アセンブリのフォルダー。
3. クライアントの呼び出し元アセンブリのフォルダー。
4. エントリ アセンブリのフォルダー。
5. クライアントの呼び出し元アセンブリに埋め込まれたリソース。

**注:** .NET Compact Framework では、ライセンスは次の場所でのみ検索されます。

1. 明示的なパス。
2. クライアントの呼び出し元アセンブリに埋め込まれたリソース。

### 例

この例では、コンポーネントが格納されているフォルダー、呼び出し元アセンブリが格納されているフォルダー、エントリ アセンブリのフォルダー、そして呼び出し元アセンブリの埋め込みリソースの順に、MyLicense.lic という名前のライセンス ファイルを探します。

```csharp
[C#]

License license = new License();
license.SetLicense("MyLicense.lic");


[Visual Basic]

Dim license As License = New License
license.SetLicense("MyLicense.lic")
```

## DocumentProperties.ScaleCrop プロパティ

ドキュメントサムネイルの表示モードを示します。  
この要素を **true** に設定すると、ドキュメントサムネイルを表示領域に合わせて拡大縮小できるようになります。  
この要素を **false** に設定すると、ドキュメントサムネイルの切り抜きが有効になり、表示領域に合わせた部分だけが表示されます。  
読み書き Boolean。

```csharp
public bool ScaleCrop { get; set; }
```

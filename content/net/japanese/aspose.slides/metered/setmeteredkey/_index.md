## Metered.SetMeteredKey メソッド

メーター制御の公開キーと非公開キーを設定します。メーターライセンスを購入した場合、アプリケーション起動時にこの API を呼び出すだけで通常は十分です。但し、使用状況データのアップロードが常に失敗し 24 時間を超えるとライセンスが評価版ステータスに設定されます。このような状況を回避するため、定期的にライセンスステータスを確認し、評価版ステータスである場合は再度この API を呼び出してください。

```csharp
public void SetMeteredKey(string publicKey, string privateKey)
```

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| publicKey | String | 公開キー |
| privateKey | String | 非公開キー |

## IFontFallBackRule.RemoveAt メソッド

リスト内の指定したインデックスにある FallBack フォントを削除します。

```csharp
public void RemoveAt(int index)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| index | Int32 | フォントを削除するゼロベースのインデックス。 |

### 例

```csharp
[C#]
// フォントのリストを含むルールを作成します。
IFontFallBackRule newRule = new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic, Tahoma, Times New Roman");

// リストから Tahoma を削除
newRule.Remove (2);
```

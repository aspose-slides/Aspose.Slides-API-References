## IHyperlink.ExternalUrlOriginal プロパティ

この部分に設定されたハイパーリンクを、実際のコンテンツに関係なく表します。PowerPoint は、部分内のリンクとそれに対応するテキストに対して特別な動作を行います。リンクの実際のアドレスとは異なる有効な URL の形でハイパーリンク用のテキストを作成できます。この場合、編集ウィンドウでリンクを表示すると、テキスト部分に合わせて変更されます。このプロパティはハイパーリンクの元の値を表します。

```csharp
public string ExternalUrlOriginal { get; }
```

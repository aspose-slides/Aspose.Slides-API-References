## TextFrame.SplitTextByColumns メソッド

[`ITextFrame`](../../itextframe) のテキスト コンテンツを文字列の配列に分割し、各要素はフレーム内の個別のテキスト列に対応します。

```csharp
public string[] SplitTextByColumns()
```

### 戻り値

文字列の配列で、各文字列は [`ITextFrame`](../../itextframe) の特定の列のテキスト コンテンツを表します。

### 備考

テキスト フレームに複数列が存在しない場合、返される配列は全文テキストを含む単一要素になります。空の列は配列内で空文字列として表されます。

### 例

次の例は `SplitTextByColumns` の使用方法を示しています。

```csharp
using (Presentation pres = new Presentation("example.pptx"))
{
    // スライド上の最初のシェイプを取得し、ITextFrame にキャストします
    ITextFrame textFrame = pres.Slides[0].Shapes[0] as ITextFrame;
    // テキストフレームの内容を列に分割します
    string[] columnsText = textFrame.SplitTextByColumns();
    // 各列のテキストをコンソールに出力します
    foreach (string column in columnsText)
        Console.WriteLine(column);
}
```

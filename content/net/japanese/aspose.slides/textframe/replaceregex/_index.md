## TextFrame.ReplaceRegex メソッド

正規表現のすべての一致を指定された文字列に置き換えます。

```csharp
public void ReplaceRegex(Regex regex, string newText, IFindResultCallback callback)
```

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| regex | Regex | 置換対象の文字列を取得する正規表現 Regex。 |
| newText | String | 置換対象の文字列のすべての出現箇所を置き換える文字列。 |
| callback | IFindResultCallback | 置換操作の結果を保存するためのコールバック オブジェクト [`IFindResultCallback`](../../ifindresultcallback)。 |

### 例

以下のサンプルコードは、正規表現と指定された文字列を使用してテキストを置換する方法を示しています。

```csharp
[C#]
using (Presentation presentation = new Presentation("SomePresentation.pptx")){
	Regex regex = new Regex(@"\b[^\s]{10,}\b");
	// 10文字以上の単語をすべて'***'に置き換える
	((AutoShape)presentation.Slides[0].Shapes[0]).TextFrame.ReplaceRegex(regex, "***", null);
	presentation.Save("SomePresentation-out.pptx", SaveFormat.Pptx);
}
```

## LayoutPlaceholderManager.AddPicturePlaceholder メソッド

レイアウト スライドに画像を保持するための新しいプレースホルダー シェイプを追加します。

```csharp
public IAutoShape AddPicturePlaceholder(float x, float y, float width, float height)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| x | Single | 新しいプレースホルダー シェイプの X 座標です。 |
| y | Single | 新しいプレースホルダー シェイプの Y 座標です。 |
| width | Single | 新しいプレースホルダー シェイプの幅です。 |
| height | Single | 新しいプレースホルダー シェイプの高さです。 |

### 戻り値

[`IAutoShape`](../../iautoshape) を Picture プレースホルダーで作成しました。

### 例

次の例は、レイアウト スライドに Picture プレースホルダー シェイプを追加する方法を示しています。

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddPicturePlaceholder(20, 20, 200, 200);
}
```

## LayoutPlaceholderManager.AddSmartArtPlaceholder メソッド

新しいプレースホルダー シェイプをレイアウト スライドに追加し、SmartArt 図を保持します。

```csharp
public IAutoShape AddSmartArtPlaceholder(float x, float y, float width, float height)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| x | Single | 新しいプレースホルダー シェイプの X 座標です。 |
| y | Single | 新しいプレースホルダー シェイプの Y 座標です。 |
| width | Single | 新しいプレースホルダー シェイプの幅です。 |
| height | Single | 新しいプレースホルダー シェイプの高さです。 |

### 戻り値

[`IAutoShape`](../../iautoshape) を SmartArt プレースホルダーで作成しました。

### 例

次の例は、レイアウト スライドに SmartArt プレースホルダー シェイプを追加する方法を示します。

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddSmartArtPlaceholder(20, 20, 200, 200);
}
```

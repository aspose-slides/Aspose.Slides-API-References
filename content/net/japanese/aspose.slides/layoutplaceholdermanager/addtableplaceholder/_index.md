## LayoutPlaceholderManager.AddTablePlaceholder メソッド

レイアウト スライドにテーブルを保持する新しいプレースホルダー シェイプを追加します。

```csharp
public IAutoShape AddTablePlaceholder(float x, float y, float width, float height)
```

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| x | Single | 新しいプレースホルダー シェイプの X 座標です。 |
| y | Single | 新しいプレースホルダー シェイプの Y 座標です。 |
| width | Single | 新しいプレースホルダー シェイプの幅です。 |
| height | Single | 新しいプレースホルダー シェイプの高さです。 |

### 戻り値

[`IAutoShape`](../../iautoshape) が Table プレースホルダーとして作成されます。

### 例

次の例は、レイアウト スライドに Table プレースホルダー シェイプを追加する方法を示しています。

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddTablePlaceholder(20, 20, 500, 200);
}
```

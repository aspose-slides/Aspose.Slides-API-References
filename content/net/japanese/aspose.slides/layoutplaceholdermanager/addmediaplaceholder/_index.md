## LayoutPlaceholderManager.AddMediaPlaceholder メソッド

レイアウト スライドにメディア オブジェクトを保持する新しいプレースホルダー シェイプを追加します。

```csharp
public IAutoShape AddMediaPlaceholder(float x, float y, float width, float height)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| x | Single | 新しいプレースホルダー シェイプの X 座標。 |
| y | Single | 新しいプレースホルダー シェイプの Y 座標。 |
| width | Single | 新しいプレースホルダー シェイプの幅。 |
| height | Single | 新しいプレースホルダー シェイプの高さ。 |

### 戻り値

メディア プレースホルダーを使用して [`IAutoShape`](../../iautoshape) を作成しました。

### 例

以下の例は、レイアウト スライドにメディア プレースホルダー シェイプを追加する方法を示しています。

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddMediaPlaceholder(20, 20, 200, 200);
}
```

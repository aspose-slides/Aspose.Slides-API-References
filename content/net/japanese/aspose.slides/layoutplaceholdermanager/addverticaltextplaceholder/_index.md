## LayoutPlaceholderManager.AddVerticalTextPlaceholder メソッド

新しいプレースホルダー シェイプをレイアウト スライドに追加し、テキスト コンテンツを垂直方向で保持します。

```csharp
public IAutoShape AddVerticalTextPlaceholder(float x, float y, float width, float height)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| x | Single | 新しいプレースホルダー シェイプの X 座標です。 |
| y | Single | 新しいプレースホルダー シェイプの Y 座標です。 |
| width | Single | 新しいプレースホルダー シェイプの幅です。 |
| height | Single | 新しいプレースホルダー シェイプの高さです。 |

### 戻り値

[`IAutoShape`](../../iautoshape) が Text (Vertical) プレースホルダーとして作成されました。

### 例

次の例は、レイアウト スライドに Text (Vertical) プレースホルダー シェイプを追加する方法を示しています。

```csharp
[C#]
using (Presentation pres = new Presentation())
{
	ILayoutSlide layout = pres.LayoutSlides.GetByType(SlideLayoutType.Blank);
	IAutoShape placeholder = layout.PlaceholderManager.AddVerticalTextPlaceholder(20, 20, 300, 500);
}
```

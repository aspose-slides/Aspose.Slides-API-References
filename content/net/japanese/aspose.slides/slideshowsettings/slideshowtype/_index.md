## SlideShowSettings.SlideShowType プロパティ

スライドショーの種類を取得または設定します。次の `SlideShowType` の先祖で表されます: [`BrowsedAtKiosk`](../../browsedatkiosk), [`PresentedBySpeaker`](../../presentedbyspeaker) と [`BrowsedByIndividual`](../../browsedbyindividual)

```csharp
public SlideShowType SlideShowType { get; set; }
```

### 例

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    // 「Browsed at a kiosk (full screen)」タイプを設定する
    pres.SlideShowSettings.SlideShowType = new BrowsedAtKiosk();
    
    // 「Browsed by individual (window)」タイプを設定する
    pres.SlideShowSettings.SlideShowType = new BrowsedByIndividual();
    
    // 「Presented by a speaker (full screen)」タイプを設定する
    pres.SlideShowSettings.SlideShowType = new PresentedBySpeaker();
}
```

## LoadOptions.DeleteEmbeddedBinaryObjects プロパティ

プレゼンテーションの読み込み時に Aspose.Slides が埋め込みバイナリ オブジェクトをすべて削除するかどうかを決定します。

埋め込みバイナリ オブジェクトの種類:

* VBA プロジェクト [`VbaProject`](../../ipresentation/vbaproject)
* OLE オブジェクト 埋め込みデータ [`EmbeddedFileData`](../../ioleembeddeddatainfo/embeddedfiledata)
* ActiveX コントロール バイナリ データ [`ActiveXControlBinary`](../../icontrol/activexcontrolbinary)

読み取り/書き込み Boolean。

```csharp
public bool DeleteEmbeddedBinaryObjects { get; set; }
```

### 備考

既定は **false** です。

### 例

次の例は、埋め込みバイナリ オブジェクトを含まない状態でプレゼンテーションを読み込む方法を示しています。

```csharp
[C#]
LoadOptions loadOptions = new LoadOptions();
loadOptions.DeleteEmbeddedBinaryObjects = true;
using (Presentation pres = new Presentation("pres.ppt", loadOptions))
{
    pres.Save("output_WithoutBinaryObjects.ppt", SaveFormat.Ppt);
}
```

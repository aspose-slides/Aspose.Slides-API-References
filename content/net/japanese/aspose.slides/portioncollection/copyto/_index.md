## PortionCollection.CopyTo メソッド

ICollection の要素を Array にコピーし、特定の Array インデックスから開始します。

```csharp
public void CopyTo(IPortion[] array, int arrayIndex)
```

| Parameter | Type | Description |
| --- | --- | --- |
| array | IPortion[] | ICollection からコピーされた要素の宛先となる一次元の Array。Array はゼロベースインデックスである必要があります。 |
| arrayIndex | Int32 | *array* でコピーが開始されるゼロベースのインデックスです。 |

### 例外

| exception | condition |
| --- | --- |
| ArgumentNullException | *array* が null です。 |
| ArgumentOutOfRangeException | *arrayIndex* が 0 未満です。 |
| ArgumentException | ソース ICollection の要素数が、宛先 *array* の *arrayIndex* から末尾までの空き領域より大きいです。 |

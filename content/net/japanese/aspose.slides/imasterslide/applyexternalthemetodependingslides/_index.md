## IMasterSlide.ApplyExternalThemeToDependingSlides メソッド

現在のスライドを基に新しいマスタースライドを作成し、外部テーマを適用して、作成したマスタースライドをすべての依存スライドに適用します。

```csharp
public IMasterSlide ApplyExternalThemeToDependingSlides(string fname)
```

| Parameter | Type | Description |
| --- | --- | --- |
| fname | String | 外部テーマファイル (.thmx) へのパス。 |

### 戻り値

新しいテーマが適用された MasterSlide。

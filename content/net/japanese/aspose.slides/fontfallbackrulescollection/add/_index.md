## FontFallBackRulesCollection.Add メソッド

指定された FallBack ルールをコレクションの末尾に追加します。


| sourceRule | IFontFallBackRule | 追加するための指定されたルール |

### 例

```csharp
[C#]
using (Presentation pres = new Presentation ())
{
    //FontsManager から空または事前初期化されたルールコレクションを取得
```

## IViewProperties.GridSpacing プロパティ

プレゼンテーション ドキュメントの基になるグリッドに使用するグリッド間隔をポイント単位で取得または設定します。 読み取り/書き込み Single.

```csharp
public float GridSpacing { get; set; }
```

### Remarks

グリッド間隔の値は正の数でなければなりません。一般的な値の範囲は 1 mm (2.8349607 ポイント) から 2 インチ (144 ポイント) です。

### Examples

以下のサンプルコードは、PowerPoint プレゼンテーションでグリッド間隔を変更する方法を示しています。

```csharp
[C#]
using (Presentation pres = new Presentation())
{
    pres.ViewProperties.GridSpacing = 72f;
    pres.Save("GridSpacing_out.pptx", SaveFormat.Pptx);
}
```

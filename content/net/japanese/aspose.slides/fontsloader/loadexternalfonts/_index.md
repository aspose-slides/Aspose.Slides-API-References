## FontsLoader.LoadExternalFonts メソッド

フォントを検索するために追加のフォルダーを追加します。

```csharp
public static void LoadExternalFonts(string[] directories)
```

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| directories | String[] | 追加のフォントを読み込むディレクトリ。 |

### 例

以下の例は、.TTF からカスタム フォントを読み込む方法を示しています。

```csharp
[C#]
// ドキュメント ディレクトリへのパスです。
string dataDir = "C:\\";
// フォントを検索するフォルダー
String[] folders = new String[] { dataDir };
// カスタム フォント ディレクトリのフォントをロードします
FontsLoader.LoadExternalFonts(folders);
// 作業を実行し、プレゼンテーション/スライドのレンダリングを行います
using (Presentation presentation = new Presentation(dataDir + "DefaultFonts.pptx"))
{
    presentation.Save(dataDir + "NewFonts_out.pptx", SaveFormat.Pptx);
	// フォントキャッシュをクリアします
	FontsLoader.ClearCache();
}
```

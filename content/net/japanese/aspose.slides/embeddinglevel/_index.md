## EmbeddingLevel 列挙型

フォントの埋め込みに関するライセンス権限を表します。

```csharp
[Flags]
public enum EmbeddingLevel : ushort
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| Installable | `0` | この設定が付いたフォントは、アプリケーションによってリモートシステムに埋め込まれ、永続的にインストールできることを示します。リモートシステムのユーザーは、そのフォントの元の購入者と同じ権利、義務、ライセンスを取得し、元の購入者と同様のエンドユーザー利用許諾契約、著作権、意匠特許、または商標の対象となります。 |
| Restricted | `2` | このビットのみが設定されたフォントは、法的所有者の許可を事前に取得せずに、いかなる方法でも変更、埋め込み、または交換してはなりません。 |
| PreviewPrint | `4` | このビットが設定されている場合、フォントは埋め込むことができ、リモートシステムに一時的にロードされます。Preview &amp; Print フォントを含む文書は「読み取り専用」で開く必要があり、文書に対して編集は行えません。 |
| Editable | `8` | このビットが設定されている場合、フォントは埋め込むことができますが、他のシステムに一時的にのみインストールすることが許可されます。Preview &amp; Print フォントとは異なり、Editable フォントを含む文書は読み取り用に開くことができ、編集が許可され、変更を保存できます。 |
| NoSubsetting | `100` | このビットが設定されている場合、フォントは埋め込む前にサブセット化できません。ビット0〜3および9で指定されたその他の埋め込み制限も適用されます。 |
| BitmapOnly | `200` | このビットが設定されている場合、フォントに含まれるビットマップのみが埋め込めます。アウトラインデータは埋め込めません。フォントにビットマップが存在しない場合、そのフォントは埋め込み不可とみなされ、埋め込みサービスは失敗します。 |

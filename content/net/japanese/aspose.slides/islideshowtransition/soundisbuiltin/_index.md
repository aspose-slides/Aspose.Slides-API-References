## ISlideShowTransition.SoundIsBuiltIn プロパティ

このサウンドが組み込みサウンドかどうかを指定します。この属性が true に設定されている場合、生成アプリケーションはこのサウンドの name 属性が組み込みサウンドのリストにあるかを確認するよう通知され、必要に応じてカスタム名や UI を表示できます。読み書き可能な Boolean。

```csharp
public bool SoundIsBuiltIn { get; set; }
```

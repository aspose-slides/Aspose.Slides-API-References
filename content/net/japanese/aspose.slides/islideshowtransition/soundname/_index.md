## ISlideShowTransition.SoundName プロパティ

遷移効果のサウンドに対する人間が読みやすい名前を指定します。[`Sound`](../sound) プロパティに割り当てることで、サウンド名を取得または設定できます。読み取り/書き込み String.

```csharp
public string SoundName { get; set; }
```

### 備考

この名前は、PowerPoint のユーザーインターフェイスで遷移サウンドを手動で設定するときに表示されます。

## Presentation.AllCustomXmlParts プロパティ

プレゼンテーション内のすべてのカスタム データ パーツを返します。読み取り専用 [`ICustomXmlPart`](../../icustomxmlpart)[].

```csharp
public ICustomXmlPart[] AllCustomXmlParts { get; }
```

### 例

以下の例は、PowerPoint プレゼンテーションからすべてのカスタム XML パーツをクリアする方法を示しています。

```csharp
[C#]
using (Presentation pres = new Presentation("PresentationWithCustomXml.pptx"))
{
	// すべてのカスタム XML パーツを列挙
    foreach (ICustomXmlPart item in pres.AllCustomXmlParts)
    {
        item.Remove();
    }
    pres.Save("out.pptx", SaveFormat.Pptx);
}
```

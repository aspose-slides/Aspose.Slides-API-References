## IFonts.SetScriptFont メソッド

特定のスクリプトタグにフォント名を割り当てます。このタグは、プレゼンテーション内でそのスクリプトのテキストがどのように表示されるかを定義します。

```csharp
public void SetScriptFont(string script, string fontName)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| script | String | BCP-47 スクリプトコード (例: "Arab", "Hebr", "Hans") で、書記体系を識別します。 |
| fontName | String | 指定されたスクリプトに割り当てるフォントの名前。 |

### 例

この例は、アラビア文字スクリプトのフォントを "Segoe UI" に設定する方法を示しています：

```csharp
[C#]
presentation.MasterTheme.FontScheme.Major.SetScriptFont("Arab", "Segoe UI");
```

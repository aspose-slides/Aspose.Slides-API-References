## IFonts.GetScriptFont メソッド

プレゼンテーションのテーマから特定のスクリプトタグに関連付けられたフォント名を取得します。

```csharp
public string GetScriptFont(string script)
```

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| script | String | 書記体系を識別するために使用される BCP-47 スクリプトコード (例: "Latn", "Cyrl", "Jpan")。 |

### 戻り値

指定されたスクリプトに使用されるフォントの名前。スクリプトが定義されていない場合は `null`。

### 例

この例は、プレゼンテーションのテーマでキリル文字スクリプトに割り当てられたフォントを取得する方法を示しています。

```csharp
[C#]
string font = presentation.MasterTheme.FontScheme.Major.GetScriptFont("Cyrl");
Console.WriteLine("Font for Cyrillic script: " + font);
```

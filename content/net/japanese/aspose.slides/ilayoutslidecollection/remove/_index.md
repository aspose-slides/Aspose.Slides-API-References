## ILayoutSlideCollection.Remove メソッド

レイアウトをコレクションから削除します。

```csharp
public void Remove(ILayoutSlide value)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| value | ILayoutSlide | コレクションから削除するレイアウトスライド。 |

### 例外

| 例外 | 条件 |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | プレゼンテーションでレイアウトが使用されている場合にスローされます（HasDependingSlides プロパティが true の場合）。 |

### 備考

1) PptxEditException のスローを防ぐために、事前にレイアウトの HasDependingSlides プロパティを確認してください。  
2) コードを簡略化するために、[`Remove`](../../ilayoutslide/remove) メソッドも使用できます。

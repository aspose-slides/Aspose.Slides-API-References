## INormalViewProperties.PreferSingleView プロパティ

ユーザーが標準の 3 つのコンテンツ領域を持つ通常ビューではなく、ウィンドウ全体に単一コンテンツ領域を表示することを好むかどうかを指定します。 有効にすると、アプリケーションはコンテンツ領域の 1 つをウィンドウ全体に表示することを選択できる場合があります。 読み取り/書き込み Boolean。

```csharp
public bool PreferSingleView { get; set; }
```

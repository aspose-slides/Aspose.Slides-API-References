## InsertClone(int, ISlide) {#insertclone}

指定したスライドのコピーをコレクションの指定位置に挿入します。

```csharp
public ISlide InsertClone(int index, ISlide sourceSlide)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| index | Int32 | 新しいスライドのインデックス。 |
| sourceSlide | ISlide | クローン対象のスライド。 |

### 戻り値

挿入されたスライド。

### 備考

異なるプレゼンテーション間でスライドをクローンする場合、スライドのマスターもクローンされる可能性があります。内部レジストリを使用して自動的にクローンされたマスターを追跡し、同一マスター スライドの複数のクローン作成を防止します。マスター スライドの手動クローンは防止も登録もされません。クローン処理をより細かく制御する必要がある場合は、[`InsertClone`](../insertclone) または [`InsertClone`](../insertclone) を使用してスライドをクローンし、[`AddClone`](../../imasterslidecollection/addclone) を使用してマスターをクローンしてください。

## InsertClone(int, ISlide, IMasterSlide, bool) {#insertclone_2}

指定したソース スライドのコピーをコレクションの指定位置に挿入します。適切なレイアウトは、指定されたマスターから自動的に選択されます（適切なレイアウトとは、ソース スライドのレイアウトと同じ Type または Name を持つレイアウト）。適切なレイアウトが存在しない場合、ソース スライドのレイアウトはクローンされます（allowCloneMissingLayout が true のとき）。allowCloneMissingLayout が false のときは PptxEditException がスローされます。

```csharp
public ISlide InsertClone(int index, ISlide sourceSlide, IMasterSlide destMaster, 
    bool allowCloneMissingLayout)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| index | Int32 | 新しいスライドのインデックス。 |
| sourceSlide | ISlide | クローン対象のスライド。 |
| destMaster | IMasterSlide | 新しいスライドのマスター スライド。 |
| allowCloneMissingLayout | Boolean | 指定されたマスターに適切なレイアウトがない場合、ソース スライドのレイアウトをクローンするか（allowCloneMissingLayout が true のとき）、PptxEditException をスローするか（allowCloneMissingLayout が false のとき）を指定します。 |

### 戻り値

挿入されたスライド。

### 例外

| 例外 | 条件 |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | 指定されたマスターに適切なレイアウトがなく、allowCloneMissingLayout が false のときスローされます。 |

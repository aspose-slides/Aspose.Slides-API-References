## ShapeCollection.AddSummaryZoomFrame メソッド

新しい Summary Zoom フレームを作成し、シェイプ コレクションの末尾に追加します。

```csharp
public ISummaryZoomFrame AddSummaryZoomFrame(float x, float y, float width, float height)
```

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| x | Single | 新しい Summary Zoom フレームの x 座標 (ポイント単位)。 |
| y | Single | 新しい Summary Zoom フレームの y 座標 (ポイント単位)。 |
| width | Single | 新しい Summary Zoom フレームの幅 (ポイント単位)。 |
| height | Single | 新しい Summary Zoom フレームの高さ (ポイント単位)。 |

### 戻り値

新しく作成された [`ISummaryZoomFrame`](../../isummaryzoomframe)。

### 例外

| 例外 | 条件 |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | プレゼンテーションにセクションが存在しない場合、または対象スライドがいずれのセクションにも属さない場合にスローされます。 |

### 備考

このメソッドは新しい Summary Zoom を作成し、プレゼンテーション内のすべてのセクションに対してオブジェクトのコレクションを配置します。

### 例

この例は、コレクションの末尾に Summary Zoom オブジェクトを追加する方法を示します (「Presentation.pptx」プレゼンテーションに少なくとも 2 つのセクションがあると仮定します)：

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSummaryZoomFrame(150, 20, 500, 250);
}
```

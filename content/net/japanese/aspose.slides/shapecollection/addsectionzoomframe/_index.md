## AddSectionZoomFrame(float, float, float, float, ISection) {#addsectionzoomframe}

新しい Section Zoom フレームを作成し、シェイプ コレクションの末尾に追加します。

```csharp
public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height, 
    ISection section)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| x | Single | 新しい Section Zoom フレームの x 座標（ポイント）。 |
| y | Single | 新しい Section Zoom フレームの y 座標（ポイント）。 |
| width | Single | 新しい Section Zoom フレームの幅（ポイント）。 |
| height | Single | 新しい Section Zoom フレームの高さ（ポイント）。 |
| section | ISection | Section Zoom フレームが参照する [`ISection`](../../isection)。このプレゼンテーションに属し、少なくとも 1 枚のスライドを含んでいる必要があります。 |

### 戻り値

新しく作成された [`ISectionZoomFrame`](../../isectionzoomframe)。

### 例外

| 例外 | 条件 |
| --- | --- |
| ArgumentException | 参照されたセクションが現在のプレゼンテーションに属していない、またはスライドが含まれていない場合にスローされます。 |

### 使用例

この例では、コレクションの末尾に Section Zoom オブジェクトを追加する方法を示します（「Presentation.pptx」プレゼンテーションに少なくとも 2 つのセクションがあると仮定します）：

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(150, 20, 50, 50, pres.Sections[1]);
}
```

public ISectionZoomFrame AddSectionZoomFrame(float x, float y, float width, float height,

### 使用例

この例では、コレクションの末尾に Section Zoom オブジェクトを追加する方法を示します（「Presentation.pptx」プレゼンテーションに少なくとも 2 つのセクションがあると仮定します）：

```csharp

}
```

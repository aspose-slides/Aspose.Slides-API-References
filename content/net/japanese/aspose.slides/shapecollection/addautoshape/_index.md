## AddAutoShape(ShapeType, float, float, float, float, bool) {#addautoshape_1}

新しい自動図形を作成し、シェイプコレクションの末尾に追加します。オプションで既定のテンプレート書式を適用できます。

```csharp
public IAutoShape AddAutoShape(ShapeType shapeType, float x, float y, float width, float height, 
    bool createFromTemplate)
```

| Parameter | Type | Description |
| --- | --- | --- |
| shapeType | ShapeType | 追加する自動図形の[`ShapeType`](../../shapetype)。 |
| x | Single | シェイプのフレームの x 座標（ポイント）。 |
| y | Single | シェイプのフレームの y 座標（ポイント）。 |
| width | Single | シェイプのフレームの幅（ポイント）。 |
| height | Single | シェイプのフレームの高さ（ポイント）。 |
| createFromTemplate | Boolean | true の場合、デフォルトテンプレートのスタイル（シンプルなスタイル、中央揃えテキスト、空でない名前）を新しいシェイプに適用します。false の場合、すべてのプロパティを既定値に設定した状態でシェイプを作成します。 |

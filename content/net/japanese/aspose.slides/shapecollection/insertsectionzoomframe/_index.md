## InsertSectionZoomFrame(int, float, float, float, float, ISection) {#insertsectionzoomframe}

指定されたインデックスに新しい Section Zoom フレームを作成し、シェイプ コレクションに挿入します。

```csharp
public ISectionZoomFrame InsertSectionZoomFrame(int index, float x, float y, float width, 
    float height, ISection section)
```

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| index | Int32 | Section Zoom フレームを挿入するゼロベースのインデックス。 |
| x | Single | 新しい Section Zoom フレームの x 座標（ポイント単位）。 |
| y | Single | 新しい Section Zoom フレームの y 座標（ポイント単位）。 |
| width | Single | 新しい Section Zoom フレームの幅（ポイント単位）。 |
| height | Single | 新しい Section Zoom フレームの高さ（ポイント単位）。 |
| section | ISection | Section Zoom フレームが参照する [`ISection`](../../isection)。このプレゼンテーションに属し、少なくとも 1 スライドを含んでいる必要があります。 |

### 戻り値

新しく作成された [`ISectionZoomFrame`](../../isectionzoomframe)。

### 例外

| 例外 | 条件 |
| --- | --- |
| ArgumentException | 参照されたセクションが現在のプレゼンテーションに属していない、またはスライドが存在しない場合にスローされます。 |

### 例

この例では、コレクションの指定されたインデックスに Section Zoom オブジェクトを作成して挿入する方法を示します（"Presentation.pptx" プレゼンテーションに少なくとも 2 つのセクションがあると仮定します）：

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISectionZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSectionZoomFrame(2, 150, 20, 50, 50, pres.Sections[1]);
}
```

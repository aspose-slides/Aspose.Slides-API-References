## InsertOleObjectFrame(int, float, float, float, float, IOleEmbeddedDataInfo) {#insertoleobjectframe}

新しい OLE オブジェクト フレームを作成し、指定されたインデックスでシェイプ コレクションに挿入します。

```csharp
public IOleObjectFrame InsertOleObjectFrame(int index, float x, float y, float width, float height, 
    IOleEmbeddedDataInfo dataInfo)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| index | Int32 | OLE オブジェクト フレームを挿入するゼロベースのインデックス。 |
| x | Single | 新しい OLE フレームの x 座標（ポイント単位）。 |
| y | Single | 新しい OLE フレームの y 座標（ポイント単位）。 |
| width | Single | 新しい OLE フレームの幅（ポイント単位）。 |
| height | Single | 新しい OLE フレームの高さ（ポイント単位）。 |
| dataInfo | IOleEmbeddedDataInfo | 埋め込み OLE データ情報（[`IOleEmbeddedDataInfo`](../../ioleembeddeddatainfo)）。 |

### 戻り値

### 例

この例は、2 番目のインデックスに OLE オブジェクトを挿入する方法を示しています：

```csharp
[C#]

IOleDataInfo dataInfo = new OleDataInfo(fileData, "zip");
IOleObjectFrame oleObjectFrame = slidees.Shapes.AddOleObjectFrame(2, 150, 20, 50, 50, dataInfo);
```

## ShapeCollection.InsertPictureFrame メソッド

指定された画像を含む新しいピクチャーフレームを作成し、指定されたインデックスでシェイプコレクションに挿入します。

```csharp
public IPictureFrame InsertPictureFrame(int index, ShapeType shapeType, float x, float y, 
    float width, float height, IPPImage image)
```

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| index | Int32 | 挿入先のピクチャーフレームのゼロベースインデックス。 |
| shapeType | ShapeType | [`ShapeType`](../../shapetype) に含まれる形状タイプを指定します。ただし、すべての種類の線は除きます。 |
| x | Single | ピクチャーフレームの x 座標（ポイント単位）。 |
| y | Single | ピクチャーフレームの y 座標（ポイント単位）。 |
| width | Single | ピクチャーフレームの幅（ポイント単位）。 |
| height | Single | ピクチャーフレームの高さ（ポイント単位）。 |
| image | IPPImage | ピクチャーフレームに表示する [`IPPImage`](../../ippimage)。 |

## Slide.WriteAsEmf メソッド

スライドの内容を EMF ファイルとして保存します。

```csharp
public void WriteAsEmf(Stream stream)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| stream | Stream | 対象ストリーム |

### 例外

| 例外 | 条件 |
| --- | --- |
| ArgumentNullException | 対象ストリームが `null` の場合 |

### 例

次のコード例は、PowerPoint プレゼンテーションの最初のスライドをメタファイルに変換する方法を示しています。

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.emf"))
    {
        // 最初のスライドをメタファイルとして保存します
        pres.Slides[0].WriteAsEmf(fileStream);
    }
}
```

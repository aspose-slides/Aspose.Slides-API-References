## WriteAsSvg(Stream) {#writeassvg}

スライドの内容をSVGファイルとして保存します。

```csharp
public void WriteAsSvg(Stream stream)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| stream | Stream | 対象ストリーム |

### 例

次のコード例は、PowerPoint プレゼンテーションの最初のスライドを SVG ファイルに変換する方法を示しています。

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.svg"))
    {
        // 最初のスライドを SVG ファイルとして保存します
        pres.Slides[0].WriteAsSvg(fileStream);
    }
}
```

| svgOptions | ISVGOptions | SVG 生成オプション |

### 例

次のコード例は、PowerPoint プレゼンテーションの最初のスライドをオプション付きで SVG ファイルに変換する方法を示しています。

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    using (Stream fileStream = System.IO.File.Create("slide_1.svg"))
    {
        var options = new SVGOptions() { VectorizeText = true };
        // 最初のスライドを SVG ファイルとして保存します
        pres.Slides[0].WriteAsSvg(fileStream, options);
    }
}
```

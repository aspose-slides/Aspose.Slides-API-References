## Control.Persistence プロパティ

ActiveX コントロールのプロパティを保存するために使用されるメソッドを取得します。読み取り専用 [`PersistenceType`](../../persistencetype)。

```csharp
public PersistenceType Persistence { get; }
```

### 例

次の例は、Persistence プロパティを使用して、ActiveX オブジェクトのプロパティが XML ベースの ActiveX プロパティとして変更可能かどうかを確認する方法を示しています：

```csharp
[C#]
if (control.Persistence == PersistenceType.PersistPropertyBag)
{
    control.Properties["Value"] = value;
}
else
{
    YourMethodHere(control.ActiveXControlBinary); //バイナリ ファイルに保存された ActiveX プロパティを管理するために独自のメソッドを使用してください
}
```

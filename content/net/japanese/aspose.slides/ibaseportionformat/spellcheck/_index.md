## IBasePortionFormat.SpellCheck プロパティ

テキスト部分のスペルチェックが有効かどうかを示す値を取得または設定します。このプロパティが false に設定されている場合、テキスト要素のスペルチェックは抑制されます。true に設定されている場合、スペルチェックが許可されます。デフォルト値は `false` です。

```csharp
public bool SpellCheck { get; set; }
```

### 例

次の例は、プレゼンテーションを保存する前に SpellCheck フラグを有効にする方法を示しています：

```csharp
[C#]
using (var pres = new Presentation("input.pptx"))
{
    // 最初のスライドの最初のシェイプ内のテキストの最初の部分にアクセスします
    var portion = ((AutoShape)pres.Slides[0].Shapes[0]).TextFrame.Paragraphs[0].Portions[0];
    // このテキスト部分のスペルチェックを有効にします
    portion.PortionFormat.SpellCheck = true;
    // 変更されたプレゼンテーションを保存します
    pres.Save("output-with-spellcheck.pptx", SaveFormat.Pptx);
}
```

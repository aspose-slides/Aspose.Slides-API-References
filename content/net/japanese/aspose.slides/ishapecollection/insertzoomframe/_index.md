## InsertZoomFrame(int, float, float, float, float, ISlide) {#insertzoomframe}

新しいズーム フレームを作成し、指定されたインデックスでシェイプ コレクションに挿入します。

```csharp
public IZoomFrame InsertZoomFrame(int index, float x, float y, float width, float height, 
    ISlide slide)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| index | Int32 | ズーム フレームを挿入する 0 ベースのインデックスです。 |
| x | Single | 新しいズーム フレームの X 座標（ポイント単位）です。 |
| y | Single | 新しいズーム フレームの Y 座標（ポイント単位）です。 |
| width | Single | 新しいズーム フレームの幅（ポイント単位）です。 |
| height | Single | 新しいズーム フレームの高さ（ポイント単位）です。 |
| slide | ISlide | ズーム フレームが参照する [`ISlide`](../../islide)です。 |

### 戻り値

新しく作成された [`IZoomFrame`](../../izoomframe)です。

### 例外

| 例外 | 条件 |
| --- | --- |
| ArgumentException | 参照されたスライドが現在のプレゼンテーションに属していない場合にスローされます。 |

### 例

この例は、コレクションの指定されたインデックスにズーム オブジェクトを作成して挿入する方法を示します（"Presentation.pptx" プレゼンテーションに少なくとも 2 つのスライドがあると仮定します）：

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    IZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertZoomFrame(2, 150, 20, 50, 50, pres.Slides[1]);
}
```

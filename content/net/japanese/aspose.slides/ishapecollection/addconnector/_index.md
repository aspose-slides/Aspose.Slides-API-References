## AddConnector(ShapeType, float, float, float, float, bool) {#addconnector_1}

新しいコネクタ形状を作成し、シェイプコレクションの末尾に追加します。オプションでデフォルトのテンプレートスタイルを適用できます。

```csharp
public IConnector AddConnector(ShapeType shapeType, float x, float y, float width, float height, 
    bool createFromTemplate)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| shapeType | ShapeType | 作成するコネクタ形状の [`ShapeType`](../../shapetype)。 |
| x | Single | コネクタのフレームの x 座標（ポイント単位）。 |
| y | Single | コネクタのフレームの y 座標（ポイント単位）。 |
| width | Single | コネクタのフレームの幅（ポイント単位）。 |
| height | Single | コネクタのフレームの高さ（ポイント単位）。 |
| createFromTemplate | Boolean | True の場合、デフォルトのテンプレートスタイル（名前が空でなく、シンプルなスタイル）を適用します。false の場合、デフォルトのプロパティ値でコネクタを作成します。 |

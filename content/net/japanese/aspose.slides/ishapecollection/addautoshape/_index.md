## AddAutoShape(ShapeType, float, float, float, float, bool) {#addautoshape_1}

新しい自動図形を作成し、シェイプコレクションの末尾に追加します。オプションで、デフォルトのテンプレート書式設定で初期化することもできます。

```csharp
public IAutoShape AddAutoShape(ShapeType shapeType, float x, float y, float width, float height, 
    bool createFromTemplate)
```

| Parameter | Type | Description |
| --- | --- | --- |
| shapeType | ShapeType | 追加する自動図形の[`ShapeType`](../../shapetype)。 |
| x | Single | シェイプのフレームの x 座標（ポイント単位）。 |
| y | Single | シェイプのフレームの y 座標（ポイント単位）。 |
| width | Single | シェイプのフレームの幅（ポイント単位）。 |
| height | Single | シェイプのフレームの高さ（ポイント単位）。 |
| createFromTemplate | Boolean | true の場合、新しいシェイプにデフォルトのテンプレート スタイル（シンプル スタイル、中央揃えテキスト、空でない名前）を適用します。false の場合、すべてのプロパティをデフォルト値に設定した状態でシェイプを作成します。 |

## InsertAutoShape(int, ShapeType, float, float, float, float) {#insertautoshape}

新しいオートシェイプを作成し、指定されたインデックスでシェイプ コレクションに挿入し、デフォルトのテンプレート書式を適用します。

```csharp
public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width, 
    float height)
```

| パラメーター | 型 | 説明 |

| index | Int32 | 新しいオートシェイプを挿入するゼロベースのインデックス。 |
| shapeType | ShapeType | 挿入するオートシェイプの [`ShapeType`](../../shapetype)。 |
| x | Single | シェイプのフレームの x 座標（ポイント単位）。 |
| y | Single | シェイプのフレームの y 座標（ポイント単位）。 |
| width | Single | シェイプのフレームの幅（ポイント単位）。 |
| height | Single | シェイプのフレームの高さ（ポイント単位）。 |

## InsertAutoShape(int, ShapeType, float, float, float, float, bool) {#insertautoshape_1}

新しいオートシェイプを作成し、指定されたインデックスでシェイプ コレクションに挿入します。オプションでデフォルトのテンプレート スタイルを初期化できます。

```csharp
public IAutoShape InsertAutoShape(int index, ShapeType shapeType, float x, float y, float width, 
    float height, bool createFromTemplate)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| index | Int32 | オートシェイプを挿入するゼロベースのインデックス。 |
| shapeType | ShapeType | 挿入するオートシェイプの [`ShapeType`](../../shapetype)。 |
| x | Single | シェイプのフレームの x 座標（ポイント単位）。 |
| y | Single | シェイプのフレームの y 座標（ポイント単位）。 |
| width | Single | シェイプのフレームの幅（ポイント単位）。 |
| height | Single | シェイプのフレームの高さ（ポイント単位）。 |
| createFromTemplate | Boolean | True はデフォルトテンプレート スタイル（空でない名前、シンプルなスタイル、中央揃えテキストを含む）を適用することを意味します。false はすべてのプロパティを既定値に設定した状態でシェイプを作成することを意味します。 |

## IShapeCollection.InsertPictureFrame メソッド

指定された画像を含む新しい画像フレームを作成し、指定されたインデックスにシェイプコレクションへ挿入します。

```csharp
public IPictureFrame InsertPictureFrame(int index, ShapeType shapeType, float x, float y, 
    float width, float height, IPPImage image)
```

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| index | Int32 | 画像フレームを挿入するゼロベースのインデックスです。 |
| shapeType | ShapeType | [`ShapeType`](../../shapetype) に含まれるシェイプの種類を指定します。線のすべての種類は除外されます。 |
| x | Single | 画像フレームの X 座標（ポイント単位）です。 |
| y | Single | 画像フレームの Y 座標（ポイント単位）です。 |
| width | Single | 画像フレームの幅（ポイント単位）です。 |
| height | Single | 画像フレームの高さ（ポイント単位）です。 |
| image | IPPImage | 画像フレームに表示する [`IPPImage`](../../ippimage) です。 |

## SetSize(SlideSizeType, SlideSizeScaleType) {#setsize}

スライドサイズをタイプで設定し、既存のコンテンツをスケールします。

```csharp
public void SetSize(SlideSizeType type, SlideSizeScaleType scaleType)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| type | SlideSizeType | 適用される事前定義済みのスライドサイズです。 |
| scaleType | SlideSizeScaleType | 使用するコンテンツのスケーリングモードです。 |

### 備考

Custom 以外のいずれかの値を設定すると、選択されたタイプに基づいて [`Size`](../size) が調整され、[`Orientation`](../orientation) は保持されます。

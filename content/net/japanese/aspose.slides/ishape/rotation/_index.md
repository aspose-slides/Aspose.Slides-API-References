## IShape.Rotation プロパティ

指定されたシェイプが z 軸の周りで回転する角度（度数）を取得または設定します。正の値は時計回りの回転を示し、負の値は反時計回りの回転を示します。読み書き Single。

```csharp
public float Rotation { get; set; }
```

### 備考

返される値は常に定義されています（float.NaN ではありません）。割り当てられる値も定義されたものでなければなりません（float.NaN ではありません）。RawFrame インスタンスのプロパティには未定義の値を設定できます。

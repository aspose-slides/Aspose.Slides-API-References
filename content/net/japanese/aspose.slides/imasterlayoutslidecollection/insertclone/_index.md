## IMasterLayoutSlideCollection.InsertClone メソッド

指定されたレイアウト スライドのコピーをコレクションの指定位置に挿入します。

```csharp
public ILayoutSlide InsertClone(int index, ILayoutSlide sourceLayout)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| index | Int32 | 新しいスライドのインデックス。 |
| sourceLayout | ILayoutSlide | クローンするスライド。 |

### Return Value

挿入されたスライド。

### Remarks

新しいレイアウトは、このレイアウト スライド コレクションの親マスタ スライドにリンクされます。そのため、PowerPoint の「宛先のテーマを使用」オプションを使用したコピー/貼り付けと同等です。

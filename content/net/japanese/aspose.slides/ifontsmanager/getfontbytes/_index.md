## IFontsManager.GetFontBytes method

指定されたフォントスタイルとフォントデータのフォントデータを表すバイト配列を取得します。

```csharp
public byte[] GetFontBytes(IFontData fontData, FontStyleType fontStyle)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| fontData | IFontData | フォントに関する情報を含むフォントデータオブジェクト [`IFontData`](../../ifontdata)。 |
| fontStyle | FontStyleType | データを取得する対象となるフォントのスタイル [`FontStyleType`](../../fontstyletype)。 |

### 戻り値

指定されたフォントスタイルのフォントデータを含むバイト配列。フォントデータまたはスタイルが見つからない場合は null を返します。

### 例

```csharp
[C#]
using (Presentation pres = new Presentation ("Presentation.pptx"))
{
    // プレゼンテーションで使用されているすべてのフォントを取得
    IFontData[] fonts = pres.FontsManager.GetFonts();

    // プレゼンテーション内の最初のフォントの通常スタイルを表すバイト配列を取得
    byte[] bytes = pres.FontsManager.GetFontBytes(fonts[0], FontStyleType.Regular);
}
```

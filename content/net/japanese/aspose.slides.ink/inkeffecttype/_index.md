## InkEffectType 列挙体

インクの描画に使用される事前定義された視覚効果のセットを指定します。各効果は、スタイライズされたインク外観をシミュレートするテクスチャまたは画像に対応しています。これらの値は、レンダリング中にデジタルインクストロークの視覚スタイルをカスタマイズするために使用できます。

```csharp
public enum InkEffectType
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| NotDefined | `-1` | 効果が定義されていないか不明です。この値はデフォルトまたはフォールバックとして使用できます。 |
| Bronze | `0` | ブロンズインクに似た、暖かみのある茶色系の金属テクスチャです。 |
| Galaxy | `1` | 銀河に似た、カラフルできらめくテクスチャで、宇宙的な色調を持っています。 |
| Gold | `2` | インクストロークに豪華な外観を与える、明るい金属の金テクスチャです。 |
| Lava | `3` | 溶岩のように燃えるテクスチャで、赤とオレンジの色合いがあります。 |
| Ocean | `4` | 深い青色の、海の波や水ベースのインクを模した流体のようなテクスチャです。 |
| RainbowGlitter | `5` | 祭りや鮮やかなインクストロークに使用される、カラフルできらめくレインボーグリッター効果です。 |
| RoseGold | `6` | エレガントなインクストロークのための、ローズゴールドに似た柔らかなピンクゴールドのブレンドです。 |
| Silver | `7` | クラシックなシルバーインクをシミュレートする、クールな金属シルバーテクスチャです。 |

## InkEffectType 列挙体

インク描画用に事前定義された視覚効果のセットを指定します。各効果は、スタイライズされたインクの外観をシミュレートするテクスチャまたは画像に対応しています。これらの値は、レンダリング中のデジタルインクストロークの視覚スタイルをカスタマイズするために使用できます。

```csharp
public enum InkEffectType
```

### Values

| 名前 | 値 | 説明 |
| --- | --- | --- |
| NotDefined | `-1` | 効果が定義されていないか不明です。この値はデフォルトまたはフォールバックとして使用できます。 |
| Bronze | `0` | ブロンズインクに似た、暖かみのある茶色がかった金属テクスチャです。 |
| Galaxy | `1` | 銀河に似たカラフルできらめくテクスチャで、宇宙的な色調です。 |
| Gold | `2` | インクストロークに豪華な外観を与える、明るい金属の金テクスチャです。 |
| Lava | `3` | 溶岩のような赤とオレンジの色合いを持つ炎のテクスチャです。 |
| Ocean | `4` | 海の波や水性インクを模した、深い青色の流体のようなテクスチャです。 |
| RainbowGlitter | `5` | 祝祭的または鮮やかなインクストロークに使用される、カラフルできらきら光るレインボーグリッター効果です。 |
| RoseGold | `6` | エレガントなインクストロークのための、ローズゴールドに似た柔らかなピンクとゴールドのブレンドです。 |
| Silver | `7` | クラシックなシルバーインクをシミュレートする、クールな金属シルバーテクスチャです。 |

## GifOptions.DefaultDelay プロパティ

デフォルトの遅延時間 [ms] を取得または設定します。この値は [`AdvanceAfterTime`](../../../aspose.slides/islideshowtransition/advanceaftertime) が設定されていない場合に使用されます。デフォルト値は 1000 です。

```csharp
public int DefaultDelay { get; set; }
```

### 例

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
	pres.Save("pres.gif", SaveFormat.Gif, new GifOptions { DefaultDelay = 2000 });
}
```

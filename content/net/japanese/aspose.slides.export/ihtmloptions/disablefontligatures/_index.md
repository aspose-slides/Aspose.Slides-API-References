## IHtmlOptions.DisableFontLigatures プロパティ

テキストがリガチャを使用せずにレンダリングされるかどうかを示す値を取得または設定します。`true` に設定すると、レンダリングされた出力でリガチャが無効になります。デフォルトでは、このプロパティは `false` に設定されています。

```csharp
public bool DisableFontLigatures { get; set; }
```

### 例

例:

```csharp
[C#]
using (Presentation pres = new Presentation("pres.pptx"))
{
    HtmlOptions options = new HtmlOptions
    {
        DisableFontLigatures = true // テキストレンダリング時にリガチャを無効にする
    };
    
    pres.Save(outputSlidePath, SaveFormat.Html, options);
}
```

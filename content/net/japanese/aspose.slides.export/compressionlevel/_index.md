## CompressionLevel 列挙型

OpenXML ファイルの ZIP 圧縮レベルを指定します。レベルが高くなるほど、圧縮率は向上しますが、処理速度は遅くなります。

```csharp
public enum CompressionLevel
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| None | `0` | 圧縮は行われません。ファイルはそのまま保存されます。 |
| Level1 | `1` | 最も速い圧縮で、圧縮率は最低です。 |
| Level2 | `2` | Level1 よりわずかに圧縮率が向上した、より速い圧縮です。 |
| Level3 | `3` | Level2 より圧縮率が向上し、性能への影響は中程度です。 |
| Level4 | `4` | Level3 より圧縮率が向上します。 |
| Level5 | `5` | Level4 より圧縮率が向上し、処理時間が増加します。 |
| Level6 | `6` | 標準的な圧縮で、圧縮速度とファイルサイズのバランスが取れています。デフォルトの圧縮レベルです。 |
| Level7 | `7` | Level6 より高い圧縮率ですが、処理は遅くなります。 |
| Level8 | `8` | Level7 より圧縮率が高くなります。 |
| Level9 | `9` | 最大の圧縮率です。最も小さなファイルサイズを生成しますが、処理速度は最も遅くなります。 |

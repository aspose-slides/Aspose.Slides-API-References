## IPdfOptions.RasterizeUnsupportedFontStyles プロパティ

フォントが太字スタイルをサポートしていない場合に、テキストをビットマップとしてラスター化し、PDF に保存すべきかどうかを示します。このアプローチは、特定のフォントにおいて生成された PDF のテキスト品質を向上させることができます。読み書き可能な Boolean。

```csharp
public bool RasterizeUnsupportedFontStyles { get; set; }
```

### 備考

デフォルトは **false** です。

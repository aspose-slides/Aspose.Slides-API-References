## IPdfOptions.EmbedTrueTypeFontsForASCII プロパティ

ASCII 文字 32-127 に対して TrueType フォントを埋め込む場合は True を指定します。文字コード 127 より大きいフォントは常に埋め込まれます。読み取り/書き込み可能な Boolean。

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### 備考

デフォルトは **true** です。

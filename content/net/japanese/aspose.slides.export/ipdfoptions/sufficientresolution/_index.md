## IPdfOptions.SufficientResolution プロパティ

PDF ドキュメント内の画像の解像度を決定する値を取得または設定します。

このプロパティはファイルサイズ、エクスポート時間、および画像品質に影響します。

デフォルト値は **96** です。

このパラメーターの効果は複数の要因に依存します。アルゴリズムは、プロパティの値、元画像のサイズ、および画像フレームのサイズに基づいて最適な出力画像サイズを取得しようとします。同様のプロパティ値を使用すると同じ結果になることがあります。目に見える効果を得るには、ステップ 16 または 32 を使用することを推奨します。読み取り/書き込み Single.

public float SufficientResolution { get; set; }

## ILinkEmbedController.GetObjectStoringLocation メソッド

オブジェクトの格納場所を決定します。このメソッドはオブジェクト id ごとに一度呼び出されます。同じデータ、semanticName、contentType を持ち、id が異なるオブジェクトが 2 つ存在する可能性があることは保証されません。

```csharp
public LinkEmbedDecision GetObjectStoringLocation(int id, byte[] entityData, string semanticName, 
    string contentType, string recomendedExtension)
```

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| id | Int32 | オブジェクト ID。本 ID は保存操作全体で一意です。 |
| entityData | Byte[] | オブジェクトのバイナリデータ。このパラメータは、オブジェクトのバイナリデータがまだ生成されていない場合は null にできます。 |
| semanticName | String | オブジェクトの意味を説明する短いテキスト。Controller はこれを外部オブジェクト名の一部として使用できるが、名前が一意で許可された文字のみを含むことを保証するのは dispatcher の役割です。 |
| contentType | String | オブジェクトの MIME タイプ。 |
| recomendedExtension | String | この MIME タイプに推奨されるファイル名拡張子。 |

## NotesCommentsLayoutingOptions.ShowCommentsByNoAuthor プロパティ

作者が設定されていないコメントの表示可否を取得または設定します。true の場合、コメントが表示されます。（コメントが表示されている場合にのみ適用されます。）

```csharp
public bool ShowCommentsByNoAuthor { get; set; }
```

### 備考

デフォルト値は **false** です。

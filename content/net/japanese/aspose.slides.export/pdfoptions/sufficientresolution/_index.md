## PdfOptions.SufficientResolution プロパティ

PDF ドキュメント内の画像の解像度を決定する値を取得または設定します。

このプロパティはファイルサイズ、エクスポート時間、画像品質に影響します。

既定値は **96** です。

このパラメーターの効果は複数の要因に依存します。アルゴリズムはプロパティの値、元画像のサイズ、画像フレームのサイズに基づいて最適な出力画像サイズを算出しようとします。同様のプロパティ値を使用すると同じ結果になることがあります。目に見える効果を得るには 16 または 32 のステップを使用することを推奨します。読み書き Single.

```csharp
public float SufficientResolution { get; set; }
```

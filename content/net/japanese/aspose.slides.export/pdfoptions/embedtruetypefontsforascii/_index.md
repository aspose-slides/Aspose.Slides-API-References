## PdfOptions.EmbedTrueTypeFontsForASCII プロパティ

ASCII (33..127 のコード範囲) テキスト用に Aspose.Slides が一般的なフォントを埋め込むかどうかを決定します。文字コードが 127 を超えるフォントは常に埋め込まれます。一般的なフォントリストには PDF の基本 14 フォントと、ユーザーが指定した追加フォントが含まれます。読み書き可能な Boolean。

```csharp
public bool EmbedTrueTypeFontsForASCII { get; set; }
```

### 備考

デフォルトは **true** です。

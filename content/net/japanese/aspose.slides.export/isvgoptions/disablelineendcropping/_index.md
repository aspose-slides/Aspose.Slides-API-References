## ISVGOptions.DisableLineEndCropping プロパティ

SVG 1.1 ではマーカーのインセットを定義する機能がありません。Aspose.Slides の SVG 書き込みエンジンはこの問題に対する回避策として、矢印付きの線の端を切り取ります。その結果、線がマーカーと重なりません。このオプションはその動作を無効にします。読み取り/書き込み Boolean。

```csharp
public bool DisableLineEndCropping { get; set; }
```

## MarkdownSaveOptions.SlideNumberFormat プロパティ

Markdown 出力のスライド番号ヘッダーに使用される書式文字列を取得または設定します。書式には必ず「{0}」プレースホルダーを含める必要があり、エクスポート時にスライドインデックスに置き換えられます。例: "# Slide {0}" は "# Slide 1", "# Slide 2" などを生成します。

```csharp
public string SlideNumberFormat { get; set; }
```

### 例外

| 例外 | 条件 |
| --- | --- |
| ArgumentNullException | 提供された値が null または空の場合にスローされます。 |
| ArgumentException | 書式文字列に「{0}」プレースホルダーが含まれていない場合にスローされます。 |

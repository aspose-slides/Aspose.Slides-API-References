```csharp

```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| model | String | LLM プロバイダーがサポートするモデル名。 |
| apiKey | String | API キー（トークン）。 |
| baseUrl | String | OpenAI 互換 LLM のベース URL。 |
| httpClient | HttpClient | 外部で管理された HttpClient インスタンス。 |

| 例外 | 条件 |

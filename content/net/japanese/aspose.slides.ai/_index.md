PowerPoint プレゼンテーションの分析と処理のための AI 機能を提供するクラスが含まれています。

## クラス

| クラス | 説明 |
| --- | --- |
| [AsposeAIWebClient](./asposeaiwebclient) | 組み込み [`IAIWebClient`](../aspose.slides.ai/iaiwebclient) 実装で、Aspose の独自 LLM に接続します。これはパラメータなしの [`SlidesAIAgent`](../aspose.slides.ai/slidesaiagent/slidesaiagent) コンストラクタで使用されるデフォルト クライアントです。 |
| [OpenAICompatibleWebClient](./openaicompatiblewebclient) | 組み込み [`IAIWebClient`](../aspose.slides.ai/iaiwebclient) 実装で、指定されたベース URL の OpenAI 互換 LLM プロバイダーに接続します。 |
| [OpenAIWebClient](./openaiwebclient) | 組み込み [`IAIWebClient`](../aspose.slides.ai/iaiwebclient) 実装で、OpenAI API に接続します。 |
| [SlidesAIAgent](./slidesaiagent) | プレゼンテーションを処理するための AI 駆動機能を提供します。 |
| [SlidesAIAgentException](./slidesaiagentexception) | Slides AI Agent に関連する例外を表します。 |

## インターフェイス

| インターフェイス | 説明 |
| --- | --- |
| [IAIConversation](./iaiconversation) | 会話インスタンスを表します。通常の AI 呼び出しとは異なり、会話は全体のコンテキストを保持します。 |
| [IAIWebClient](./iaiwebclient) | AI Web クライアント インターフェイス。このインターフェイスにより、異なる AI 言語モデルを差し替えることができます。このインターフェイスを実装するクラスは `SlidesAIAgent` と共に使用することが想定されています。 |

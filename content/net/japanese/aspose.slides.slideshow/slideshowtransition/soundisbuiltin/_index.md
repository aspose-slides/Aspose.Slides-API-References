## SlideShowTransition.SoundIsBuiltIn プロパティ

このサウンドが組み込みサウンドかどうかを指定します。この属性が true に設定されている場合、生成アプリケーションはこのサウンドに指定された name 属性を組み込みサウンドの一覧で確認し、必要に応じてカスタム名や UI を表示できるように通知されます。読み書き可能 Boolean.

public bool SoundIsBuiltIn { get; set; }

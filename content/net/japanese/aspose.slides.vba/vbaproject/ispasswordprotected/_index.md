## VbaProject.IsPasswordProtected プロパティ

VBAProject がプロジェクトのプロパティの表示にパスワードで保護されているかどうかを示します。読み取り専用 Boolean。


### 例

```csharp

{
    if (presentation.VbaProject.IsPasswordProtected)
        Console.WriteLine("The VBAProject '" + presentation.VbaProject.Name +

}
```

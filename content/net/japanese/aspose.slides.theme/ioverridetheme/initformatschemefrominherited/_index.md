## IOverrideTheme.InitFormatSchemeFromInherited メソッド

新しいオブジェクトで FormatScheme を初期化し、InheritedTheme の FormatScheme を上書きします。また、この新しいオブジェクトのデータを InheritedTheme の FormatScheme のデータで初期化します。

```csharp
public void InitFormatSchemeFromInherited()
```

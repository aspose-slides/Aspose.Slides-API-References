## BaseOverrideThemeManager.IsOverrideThemeEnabled プロパティ

OverrideTheme が継承された有効テーマを上書きするかどうかを決定します。OverrideTheme を有効にして上書きするには OverrideTheme.Init*() メソッドを使用します。OverrideTheme の上書きを無効にするには OverrideTheme.Clear() メソッドを使用します。読み取り専用 Boolean。

public bool IsOverrideThemeEnabled { get; }

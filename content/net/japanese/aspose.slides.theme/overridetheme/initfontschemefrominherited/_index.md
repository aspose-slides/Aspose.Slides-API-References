## OverrideTheme.InitFontSchemeFromInherited メソッド

InheritedTheme の FontScheme を上書きするために、新しいオブジェクトで FontScheme を初期化します。また、この新しいオブジェクトのデータを InheritedTheme の FontScheme のデータで初期化します。

```csharp
public void InitFontSchemeFromInherited()
```

### 例外

| 例外 | 条件 |
| --- | --- |
| InvalidOperationException | FontScheme がすでに初期化されている (null でない) 場合にスローされます。 |

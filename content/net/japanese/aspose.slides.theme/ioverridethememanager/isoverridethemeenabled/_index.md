## IOverrideThemeManager.IsOverrideThemeEnabled プロパティ

OverrideTheme が継承された有効なテーマをオーバーライドするかどうかを決定します。OverrideTheme を有効にしてオーバーライドするには OverrideTheme.Init*() メソッドを使用します。OverrideTheme のオーバーライドを無効にするには OverrideTheme.Clear() メソッドを使用します。読み取り専用 Boolean。

```csharp
public bool IsOverrideThemeEnabled { get; }
```

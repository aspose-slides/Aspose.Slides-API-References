## DataLabel.IsVisible プロパティ

False はデータラベルが表示されないことを意味します (そのためすべての Show*-フラグ (ShowValue, ...) は false になります)。 読み取り専用 Booleanです。

```csharp
public bool IsVisible { get; }
```

### 備考

データラベルが表示されている場合は Hide() メソッドで非表示にできます。 ただしデータラベルが表示されていない場合 (IsVisible が false) は、Show*-フラグ (ShowValue, ...) を true に設定することでデータラベルを表示できます。

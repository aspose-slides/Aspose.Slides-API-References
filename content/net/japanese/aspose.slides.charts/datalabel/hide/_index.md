## DataLabel.Hide メソッド

すべての Show*-フラグ (ShowValue, ...) を false に設定することでデータラベルを非表示にします。これにより IsVisible は false になります。

```csharp
public void Hide()
```

### 備考

データラベルが表示されていない場合 (IsVisible が false のとき)、Show*-フラグ (ShowValue, ...) を true に設定することでデータラベルを表示できます。

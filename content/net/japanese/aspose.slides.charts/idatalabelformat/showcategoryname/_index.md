## IDataLabelFormat.ShowCategoryName プロパティ

指定された chart のデータ ラベル カテゴリ名の表示動作を表します。chart 上のデータ ラベルにカテゴリ名を表示する場合は True、非表示にする場合は False を設定します。読み取り/書き込み Boolean。

```csharp
public bool ShowCategoryName { get; set; }
```

### 備考

この DataLabelFormat オブジェクトの親がデータ ラベルの DataLabelCollection コレクションである場合、このプロパティは DataLabelCollection 内の新しいデータ ラベルに対する ShowCategoryName プロパティの既定値を取得または設定します。値を設定すると、このプロパティは DataLabelCollection 内のすべてのデータ ラベルの ShowCategoryName プロパティにも同じ値が設定されます（例: "DataLabels.DefaultDataLabelFormat.ShowCategoryName = val;" により、すべての DataLabels[i].ShowCategoryName が val と等しくなります）。

## IDataLabelFormat.IsNumberFormatLinkedToSource property

読み書き Boolean.

```csharp
public bool IsNumberFormatLinkedToSource { get; set; }
```

### 備考

この DataLabelFormat オブジェクトの親がデータ ラベルの DataLabelCollection コレクションである場合、このプロパティは DataLabelCollection 内の新しいデータ ラベルに対する IsNumberFormatLinkedToSource プロパティの既定値を取得または設定します。このプロパティに値を設定すると、DataLabelCollection 内のすべてのデータ ラベルの IsNumberFormatLinkedToSource プロパティにも同じ値が設定されます（例: "DataLabels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = val;" により、すべての DataLabels[i].IsNumberFormatLinkedToSource が val と等しくなります）。

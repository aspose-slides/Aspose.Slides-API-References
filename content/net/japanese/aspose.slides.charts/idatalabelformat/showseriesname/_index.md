## IDataLabelFormat.ShowSeriesName プロパティ

チャート上のデータラベルの系列名表示動作を示す Boolean を取得または設定します。True は系列名を表示し、False は非表示にします。読み書き可能 Boolean。

```csharp
public bool ShowSeriesName { get; set; }
```

### 備考

この DataLabelFormat オブジェクトの親がデータラベルの DataLabelCollection コレクションである場合、このプロパティは DataLabelCollection コレクション内の新しいデータラベルの ShowSeriesName プロパティの既定値を取得または設定します。値を設定すると、この値は DataLabelCollection コレクション内のすべてのデータラベルの ShowSeriesName プロパティにも設定されます（例: `DataLabels.DefaultDataLabelFormat.ShowSeriesName = val;` により、すべての DataLabels[i].ShowSeriesName が val と等しくなります）。

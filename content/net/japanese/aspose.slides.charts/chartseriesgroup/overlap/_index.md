## ChartSeriesGroup.Overlap プロパティ

2-D チャートにおける棒や列のオーバーラップ率をパーセンテージで指定します（-100%〜100%）。  
- -100%: 最大間隔（棒が完全に分離）。  
- 0%: 棒が隣接して配置され、オーバーラップも間隔もなし。  
- 100%: 最大オーバーラップ（棒が互いに完全に重なる）。  
このプロパティは読み取り/書き込み可能で、型は SByte です。

```csharp
public sbyte Overlap { get; set; }
```

### 例外

| 例外 | 条件 |
| --- | --- |
| ArgumentOutOfRangeException | 値が -100 から 100 の有効範囲外に設定されたときにスローされます。 |

### 例

以下の例は、チャートシリーズグループのオーバーラップを設定し、フォーム上に結果のチャートを描画する方法を示しています：

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // オーバーラップを55%に設定

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

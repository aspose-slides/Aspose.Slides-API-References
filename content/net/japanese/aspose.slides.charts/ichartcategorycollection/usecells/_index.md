## IChartCategoryCollection.UseCells プロパティ

true の場合、ワークシートはカテゴリの保存に使用されます（この場合は多階層カテゴリをサポートします）。false の場合、ワークシートは値の保存に使用されず（この場合は多階層カテゴリをサポートしません）。読み取り/書き込み Boolean.

public bool UseCells { get; set; }

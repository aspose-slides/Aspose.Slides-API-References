## Add(IChartDataCell) {#add}

コレクションにカテゴリが存在する場合はそれを返します。存在しない場合は [`IChartDataCell`](../../ichartdatacell) から新しいチャートカテゴリを作成し、コレクションに追加します。

```csharp
public IChartCategory Add(IChartDataCell chartDataCell)
```

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| chartDataCell | IChartDataCell | チャートカテゴリを作成するために使用されるセル。 |

### 戻り値

追加されたカテゴリまたは既存のカテゴリ。

## Add(object) {#add_1}

値から新しい [`IChartCategory`](../../ichartcategory) を作成し、コレクションに追加します。

```csharp
public IChartCategory Add(object value)
```

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| value | Object | 値。 |

### 戻り値

追加された [`IChartCategory`](../../ichartcategory)。

### 例外

| 例外 | 条件 |
| --- | --- |
| InvalidOperationException | 制限を超えた場合 |

### 備考

このメソッドは名前が AUTO_DATA のワークシートを追加し、すべての値をそこに追加します。[`IChartDataWorkbook`](../../ichartdataworkbook) を使用してセルの値を追加または編集する場合は、このワークシートを使用しないでください。このメソッドで追加できる値の最大数は 16711680 を超えてはいけません。

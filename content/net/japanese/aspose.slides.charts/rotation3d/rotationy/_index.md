## Rotation3D.RotationY プロパティ

Y軸周りの回転角度（3DチャートではX方向）を取得または設定します（0 から 360 度 の範囲）。このプロパティは ECMA-376 の 21.2.2.158 rotY（Y Rotation）項目および PowerPoint 2007+ の「X Rotation」オプションに対応しています。読み取り/書き込み UInt16.

```csharp
public ushort RotationY { get; set; }
```

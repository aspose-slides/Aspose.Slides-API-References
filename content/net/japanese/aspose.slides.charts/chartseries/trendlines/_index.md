## ChartSeries.TrendLines プロパティ

シリーズのトレンドラインのコレクション。読み取り専用 [`ITrendlineCollection`](../../itrendlinecollection)。

```csharp
public ITrendlineCollection TrendLines { get; }
```

### 備考

TrendLines は、スタックされていない 2-D エリア、棒、縦棒、折れ線、株価、XY（散布図）、およびバブル チャートのデータ系列で利用可能（null ではありません）。スタックされたチャートや 3-D チャートのデータ系列ではトレンドラインは利用できません。また、レーダー、円グラフ、サーフェス、ドーナツ チャートでも利用できません。

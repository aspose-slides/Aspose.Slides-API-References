## ChartSeries.DoughnutHoleSize プロパティ

ドーナツ チャートの穴のサイズを指定します（プロット領域のサイズの 10%〜90% の範囲で設定できます）。このプロパティはこのシリーズだけでなく、親シリーズ グループのすべてのシリーズに適用されるプロパティの投影です。そのためこのプロパティは読み取り専用です。親シリーズ グループへは ParentSeriesGroup プロパティでアクセスできます。値を変更するには ParentSeriesGroup.DoughnutHoleSize の読み書き可能プロパティを使用してください。読み取り専用 Byte。

```csharp
public byte DoughnutHoleSize { get; }
```

## ChartSeries.IsColorVaried プロパティ

シリーズ内の各データマーカーが異なる色を持つことを指定します。このプロパティはこのシリーズだけでなく、親シリーズグループのすべてのシリーズに適用されます。これは該当するグループプロパティの投影です。したがってこのプロパティは読み取り専用です。親シリーズグループにアクセスするには ParentSeriesGroup プロパティを使用します。値を変更するには ParentSeriesGroup.IsColorVaried 読み取り/書き込みプロパティを使用します。読み取り専用 Boolean。

```csharp
public bool IsColorVaried { get; }
```

### 備考

これはプロパティ ParentSeriesGroup.IsColorVaried の投影です。

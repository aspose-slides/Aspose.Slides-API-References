## ChartSeries.HasSeriesLines property

この系列および関連系列にシリーズラインがあるかどうかを判定します。このプロパティはこの系列だけでなく、親系列グループのすべての系列に適用されます——適切なグループプロパティの投影です。そのためこのプロパティは読み取り専用です。親系列グループへのアクセスには ParentSeriesGroup プロパティを使用します。値を変更するには ParentSeriesGroup.HasSeriesLines 読み書きプロパティを使用します。シリーズラインの書式設定には ParentSeriesGroup.SeriesLinesFormat プロパティを使用します。読み取り専用の Boolean。

```csharp
public bool HasSeriesLines { get; }
```

### Remarks

このプロパティは ParentSeriesGroup.HasSeriesLines プロパティの投影です。

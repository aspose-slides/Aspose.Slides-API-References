## ChartSeries.BubbleSizeScale プロパティ

バブルチャートのスケール係数を指定します（既定サイズの 0% から 300% の範囲です）。このプロパティはこのシリーズだけでなく、親シリーズグループのすべてのシリーズに適用されるプロパティの投影です。したがってこのプロパティは読み取り専用です。親シリーズグループにアクセスするには ParentSeriesGroup プロパティを使用します。値を変更するには ParentSeriesGroup.BubbleSizeScale 読み書き可能なプロパティを使用します。

```csharp
public int BubbleSizeScale { get; }
```

### 備考

このプロパティは ParentSeriesGroup.BubbleSizeScale プロパティの投影です。

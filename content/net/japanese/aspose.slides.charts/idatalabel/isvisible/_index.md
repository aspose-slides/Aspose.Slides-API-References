## IDataLabel.IsVisible property

False はデータ ラベルが表示されないことを意味します (したがってすべての Show*-flags (ShowValue, ...) は false になります)。読み取り専用の Boolean。

```csharp
public bool IsVisible { get; }
```

### 備考

データ ラベルが表示されている場合は、Hide() メソッドで非表示にできます。データ ラベルが表示されていない場合 (IsVisible が false) は、Show*-flags (ShowValue, ...) を true の状態に設定することでデータ ラベルを表示できます。

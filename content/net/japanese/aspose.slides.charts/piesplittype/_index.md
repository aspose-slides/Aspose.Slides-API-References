## PieSplitType 列挙体

パイ・オブ・パイ または バー・オブ・パイ チャートの、2 番目のパイまたはバー上の分割ポイントのタイプを表します。

```csharp
public enum PieSplitType
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| Default | `0` | このチャート タイプのデフォルト メカニズムを使用してデータ ポイントを分割することを指定します。 |
| Custom | `1` | データ ポイントがパイと 2 番目のチャート間で、Custom Split 値に従って分割されることを指定します。 |
| ByPercentage | `2` | データ ポイントがパイと 2 番目のチャート間で、Split Position パーセンテージ 未満のパーセンテージを持つポイントを 2 番目のチャートに配置することにより分割されることを指定します。 |
| ByPos | `3` | データ ポイントがパイと 2 番目のチャート間で、データ ポイントの最後の Split Position を 2 番目のチャートに配置することにより分割されることを指定します。 |
| ByValue | `4` | データ ポイントがパイと 2 番目のチャート間で、Split Position 未満の値を持つデータ ポイントを 2 番目のチャートに配置することにより分割されることを指定します。 |

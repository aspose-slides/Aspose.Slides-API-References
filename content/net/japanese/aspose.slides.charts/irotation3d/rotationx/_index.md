## IRotation3D.RotationX プロパティ

X 軸の回転角度（3D チャートの場合は Y 方向）を取得または設定します（-90 度から 90 度の範囲）。このプロパティは ECMA-376 の 21.2.2.157 rotX (X Rotation) 項目および PowerPoint 2007 以降の「Y Rotation」オプションに対応しています。読み取り/書き込み SByte。

```csharp
public sbyte RotationX { get; set; }
```

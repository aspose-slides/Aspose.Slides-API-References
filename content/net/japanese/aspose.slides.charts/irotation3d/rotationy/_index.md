## IRotation3D.RotationY プロパティ

Y 軸周りの回転角度を取得または設定します。3D チャートでは X 方向に相当し、0 から 360 度の間で指定できます。このプロパティは ECMA-376 の 21.2.2.158 rotY（Y Rotation）項目および PowerPoint 2007+ の「X Rotation」オプションに対応しています。読み書き UInt16。

```csharp
public ushort RotationY { get; set; }
```

## ChartData.SecondaryCategories プロパティ

[`UseSecondaryCategories`](../usesecondarycategories) プロパティが true の場合に二次カテゴリを取得します。読み取り専用 [`IChartCategoryCollection`](../../ichartcategorycollection)。

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### 備考

[`UseSecondaryCategories`](../usesecondarycategories) プロパティが false の場合、この `SecondaryCategories` プロパティは null を返し、[`Categories`](../categories) プロパティのデータは一次シリーズと二次シリーズの両方に使用されます。[`UseSecondaryCategories`](../usesecondarycategories) プロパティが true の場合、この `SecondaryCategories` プロパティのデータは二次シリーズに使用され、[`Categories`](../categories) プロパティのデータは一次シリーズに使用されます。

### 例

例。シリーズに関連付けられるカテゴリはどれか - ChartData.Categories または ChartData.SecondaryCategories？

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // 関連するカテゴリは series.Chart.ChartData.SecondaryCategories
}
else
{
    // 関連するカテゴリは series.Chart.ChartData.Categories
}
```

## IChartSeriesGroup.Overlap プロパティ

2-D チャートにおける棒や列がどれだけ重なるかをパーセンテージ (−100% から 100% まで) で指定します。  
- −100%: 最大の間隔 (棒は完全に分離)。  
- 0%: 棒が重なりも間隔もなく横に並ぶ。  
- 100%: 最大の重なり (棒が完全に互いに重なる)。  
このプロパティは read/write で、型は SByte です。

```csharp
public sbyte Overlap { get; set; }
```

### 例外

| 例外 | 条件 |
| --- | --- |
| ArgumentOutOfRangeException | 値が有効範囲 −100 から 100 の外に設定されたときにスローされます。 |

### 例

以下の例は、チャート系列グループのオーバーラップを設定し、結果のチャートをフォームに描画する方法を示しています。

```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 10, 10, 600, 300);
    IChartSeriesCollection series = chart.ChartData.Series;
    series[0].ParentSeriesGroup.Overlap = 55; // オーバーラップを 55% に設定

    var image=pres.Slides[0].GetImage(1, 1);
    image.Save("image.png",ImageFormat.Png);
}
```

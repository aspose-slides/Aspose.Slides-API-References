## IChartData.Categories プロパティ

主なカテゴリを取得します（[`UseSecondaryCategories`](../usesecondarycategories) プロパティが false の場合は、主カテゴリと副カテゴリの両方を取得します）。読み取り専用 [`IChartCategoryCollection`](../../ichartcategorycollection)。

```csharp
public IChartCategoryCollection Categories { get; }
```

### 備考

[`UseSecondaryCategories`](../usesecondarycategories) プロパティが false の場合、[`SecondaryCategories`](../secondarycategories) プロパティは null を返し、この `Categories` プロパティのデータは主系列と副系列の両方に使用されます。[`UseSecondaryCategories`](../usesecondarycategories) プロパティが true の場合、[`SecondaryCategories`](../secondarycategories) プロパティのデータは副系列に使用され、この `Categories` プロパティのデータは主系列に使用されます。

### 例

例。どのカテゴリが系列に関連付けられるか - ChartData.Categories または ChartData.SecondaryCategories？

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // 関連するカテゴリは series.Chart.ChartData.SecondaryCategories です
}
else
{
    // 関連するカテゴリは series.Chart.ChartData.Categories です
}
```

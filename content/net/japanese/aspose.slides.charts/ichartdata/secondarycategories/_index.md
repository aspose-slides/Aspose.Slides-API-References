## IChartData.SecondaryCategories プロパティ

[`UseSecondaryCategories`](../usesecondarycategories) プロパティが true の場合、セカンダリ カテゴリを取得します。読み取り専用 [`IChartCategoryCollection`](../../ichartcategorycollection).

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### 備考

[`UseSecondaryCategories`](../usesecondarycategories) プロパティが false の場合、この `SecondaryCategories` プロパティは null を返し、[`Categories`](../categories) プロパティのデータは一次系列と二次系列の両方に使用されます。[`UseSecondaryCategories`](../usesecondarycategories) プロパティが true の場合、この `SecondaryCategories` プロパティのデータは二次系列に使用され、[`Categories`](../categories) プロパティのデータは一次系列に使用されます。

### 例

例。シリーズに関連付けられているカテゴリは ChartData.Categories ですか、それとも ChartData.SecondaryCategories ですか？

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // 関連するカテゴリは series.Chart.ChartData.SecondaryCategories です
}
else
{
    // 関連するカテゴリは series.Chart.ChartData.Categories です
}
```

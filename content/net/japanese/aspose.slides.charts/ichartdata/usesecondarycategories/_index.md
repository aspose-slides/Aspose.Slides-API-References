## IChartData.UseSecondaryCategories プロパティ

false の場合は [`SecondaryCategories`](../secondarycategories) プロパティが null を返し、[`Categories`](../categories) プロパティのデータは一次系列および二次系列の両方に使用されます。true の場合は、[`SecondaryCategories`](../secondarycategories) プロパティのデータが二次系列に使用され、[`Categories`](../categories) プロパティのデータが一次系列に使用されます。読み取り/書き込み Boolean.

```csharp
public bool UseSecondaryCategories { get; set; }
```

### 例

例。系列に関連付けられるカテゴリは ChartData.Categories か ChartData.SecondaryCategories か？

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // 関連するカテゴリは series.Chart.ChartData.SecondaryCategories です
}
else
{
    // 関連するカテゴリは series.Chart.ChartData.Categories です
}
```

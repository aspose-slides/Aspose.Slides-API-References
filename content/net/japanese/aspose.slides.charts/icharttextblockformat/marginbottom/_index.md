## IChartTextBlockFormat.MarginBottom プロパティ

TextFrame の下余白 (ポイント) を取得または設定します。このプロパティを変更すると、次のチャート パーツにのみ影響を与える可能性があります: DataLabel と DataLabelFormat (PowerPoint 2013 では完全にサポート; PowerPoint 2007 ではレンダリングに効果がありません)。読み取り/書き込み Double.

public double MarginBottom { get; set; }

## IChartSeries.HasSeriesLines プロパティ

この系列および関連系列に系列線があるかどうかを決定します。このプロパティはこの系列だけでなく、親系列グループのすべての系列に対するものであり、対応するグループプロパティの投影です。したがって、このプロパティは読み取り専用です。親系列グループへアクセスするには ParentSeriesGroup プロパティを使用します。値を変更するには ParentSeriesGroup.HasSeriesLines の読み書き可能なプロパティを使用します。系列線の書式設定には ParentSeriesGroup.SeriesLinesFormat プロパティを使用します。読み取り専用 Boolean。

```csharp
public bool HasSeriesLines { get; }
```

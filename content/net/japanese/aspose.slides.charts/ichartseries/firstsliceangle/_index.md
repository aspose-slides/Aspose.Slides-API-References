## IChartSeries.FirstSliceAngle プロパティ

最初の円形またはドーナツ形チャートのスライスの角度を、度数（上から時計回りに、0 から 360 度）で指定します。このプロパティはこのシリーズだけでなく、親シリーズグループのすべてのシリーズに対しても適用される、対応するグループプロパティの投影です。したがって、このプロパティは読み取り専用です。親シリーズグループにアクセスするには ParentSeriesGroup プロパティを使用します。値を変更するには ParentSeriesGroup.FirstSliceAngle の読み書き可能なプロパティを使用します。読み取り専用 UInt16。

```csharp
public ushort FirstSliceAngle { get; }
```

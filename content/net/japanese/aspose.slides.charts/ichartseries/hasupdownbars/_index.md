## IChartSeries.HasUpDownBars プロパティ

Line-または Stock-chart に上下バーがあるかどうかを決定します。このプロパティはこの系列だけでなく、親系列グループのすべての系列に適用されるもので、該当グループプロパティの投影です。そのため、このプロパティは読み取り専用です。親系列グループにアクセスするには ParentSeriesGroup プロパティを使用します。値を変更するには ParentSeriesGroup.UpDownBars.HasUpDownBars の読み書き可能なプロパティを使用してください。上下バーの書式設定には ParentSeriesGroup.UpDownBars プロパティを使用します。読み取り専用の Boolean。

```csharp
public bool HasUpDownBars { get; }
```

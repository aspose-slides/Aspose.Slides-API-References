## IChartSeries.PieSplitBy プロパティ

パイ・オブ・パイ または バー・オブ・パイ チャートで、どのデータ ポイントが第2のパイまたはバーに含まれるかを決定する方法を指定します。このプロパティはこのシリーズだけでなく、親シリーズ グループのすべてのシリーズに対しても適用されるプロパティであり、該当グループ プロパティの投影です。したがって、このプロパティは読み取り専用です。親シリーズ グループにアクセスするには ParentSeriesGroup プロパティを使用します。ParentSeriesGroup.PieSplitBy 読み取り/書き込み プロパティを使用して値を変更します。読み取り専用 [`PieSplitType`](../../piesplittype)。

```csharp
public PieSplitType PieSplitBy { get; }
```

### 備考

1) これはプロパティ ParentSeriesGroup.PieSplitBy の投影です。 2) プロパティの値が PieSplitType.Custom の場合、ParentSeriesGroup.PieSplitCustomPoints プロパティでカスタムの分割情報を定義できます。

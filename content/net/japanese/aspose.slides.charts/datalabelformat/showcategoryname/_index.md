## DataLabelFormat.ShowCategoryName property

指定されたチャートのデータラベルのカテゴリ名表示動作を表します。True はチャート上のデータラベルにカテゴリ名を表示します。False は非表示にします。読み取り/書き込み Boolean.

```csharp
public bool ShowCategoryName { get; set; }
```

### 備考

この DataLabelFormat オブジェクトの親が DataLabelCollection（データラベルのコレクション）である場合、このプロパティは DataLabelCollection 内の新しいデータラベルに対する ShowCategoryName プロパティのデフォルト値を取得または設定します。値を設定すると、DataLabelCollection 内のすべてのデータラベルの ShowCategoryName プロパティにも同じ値が設定されます（例: "DataLabels.DefaultDataLabelFormat.ShowCategoryName = val;" はすべての DataLabels[i].ShowCategoryName が val と等しくなることを意味します）。

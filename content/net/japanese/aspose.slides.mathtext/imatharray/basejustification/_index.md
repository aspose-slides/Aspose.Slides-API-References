## IMathArray.BaseJustification プロパティ

配列の周囲のテキストに対する配置を指定します。配列の外側のテキストは、配列オブジェクトの底部、上部、または中央に合わせて配置できます。デフォルト値: Center

```csharp
public MathVerticalAlignment BaseJustification { get; set; }
```

### 例

Example:

```csharp
[C#]
IMathArray mathArray = new MathArray(new MathematicalText("item1"));
mathArray.BaseJustification = MathVerticalAlignment.Top;
```

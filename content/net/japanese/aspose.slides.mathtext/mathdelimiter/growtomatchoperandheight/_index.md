## MathDelimiter.GrowToMatchOperandHeight プロパティ

BeginningCharacter、SeparatorCharacter、EndingCharacter の成長を指定します。true の場合、区切り記号はオペランドの高さに合わせて垂直方向に伸びます。デフォルト値は true です。

```csharp
public bool GrowToMatchOperandHeight { get; set; }
```

### 例

例:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.GrowToMatchOperandHeight = false;
```

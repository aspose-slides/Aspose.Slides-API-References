## MathDelimiter.EndingCharacter プロパティ

Delimiter Ending Character は、終了または閉じるデリミタ文字を指定します。数式デリミタは、丸括弧、角括弧、波括弧などの囲む文字です。デフォルト: ')'.

```csharp
public char EndingCharacter { get; set; }
```

### 例

Example:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Join("y").Enclose();
delimiter.EndingCharacter = ']';
```

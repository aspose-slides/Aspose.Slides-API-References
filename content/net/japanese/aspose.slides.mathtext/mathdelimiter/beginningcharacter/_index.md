## MathDelimiter.BeginningCharacter プロパティ

区切り文字の開始文字は、開始（または開く）区切り文字を指定します。数学的区切り文字は、かっこ、角括弧、波括弧などの囲み文字です。既定値は '(' です。

```csharp
public char BeginningCharacter { get; set; }
```

例:


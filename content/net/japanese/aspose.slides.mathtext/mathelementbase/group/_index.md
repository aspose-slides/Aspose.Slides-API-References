## Group(char, MathTopBotPositions, MathTopBotPositions) {#group_1}

この要素を下部中括弧などのグルーピング文字を使用してグループに配置します

```csharp
public IMathGroupingCharacter Group(char character, MathTopBotPositions position, 
    MathTopBotPositions verticalJustification)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| character | Char | 下部中括弧 (U+23DF) などのグルーピング文字 |
| position | MathTopBotPositions | グルーピング文字の位置 |
| verticalJustification | MathTopBotPositions | グループ文字の垂直位置揃え。オブジェクトのベースラインに対する配置を指定します。例えば、グループ文字がオブジェクトの上にある場合、Top の VerticalJustification はオブジェクトの上部がベースライン上にあることを示します。Bottom に設定すると、オブジェクトの下部がベースライン上にあります。 |

### 戻り値

型の新しいインスタンス [`IMathGroupingCharacter`](../../imathgroupingcharacter)

### 例

例:

```csharp
[C#]
IMathGroupingCharacter groupingElement = new MathematicalText("x;y;z").Group('\u23E1', MathTopBotPositions.Bottom, MathTopBotPositions.Top);
```

## IMathDelimiter.GrowToMatchOperandHeight プロパティ

BeginningCharacter、SeparatorCharacter、EndingCharacter の成長を指定します。true の場合、デリミタはオペランドの高さに合わせて縦方向に成長します。デフォルト値は true です

```csharp
public bool GrowToMatchOperandHeight { get; set; }
```

### 例

例:

```csharp
[C#]
IMathDelimiter delimiter = new MathematicalText("x").Divide("y").Enclose();
delimiter.GrowToMatchOperandHeight = false;
```

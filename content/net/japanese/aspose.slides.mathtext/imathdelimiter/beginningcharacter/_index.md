## IMathDelimiter.BeginningCharacter プロパティ

区切り文字の開始文字は、開始（または開く）区切り文字を指定します。数学的区切り文字は、丸括弧、角括弧、波括弧などの囲む文字です。デフォルト値: '('。

```csharp
public char BeginningCharacter { get; set; }
```

例:

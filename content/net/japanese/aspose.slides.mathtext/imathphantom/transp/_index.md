## IMathPhantom.Transp プロパティ

ファントムがクラスベースの間隔規則に対して透明であるかどうかを示す値を取得または設定します。

```csharp
public bool Transp { get; set; }
```

### 備考

`true` の場合、ファントム内の演算子や記号はファントムの周囲の数式間隔に依然として影響します（可視であるかのように）。`false` の場合、クラスベースの間隔は無視されます。OMML 属性 `m:transp` に対応します。

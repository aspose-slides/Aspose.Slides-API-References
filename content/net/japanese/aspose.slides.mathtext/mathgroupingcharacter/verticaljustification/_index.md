## MathGroupingCharacter.VerticalJustification プロパティ

グループ文字の垂直配置。オブジェクトのベースラインに対する配置を指定します。例として、グループ文字がオブジェクトの上にある場合、Top の VerticalJustification はオブジェクトの上部がベースライン上になることを示します。VerticalJustification が Bottom に設定されている場合、オブジェクトの下部がベースライン上になります。デフォルト: Position=Top の場合は Bottom、Position=Bottom の場合は Top

```csharp
public MathTopBotPositions VerticalJustification { get; set; }
```

### 例

例:

```csharp
[C#]
MathGroupingCharacter groupingCharacter = new MathGroupingCharacter(new MathematicalText("abc"));
groupingCharacter.VerticalJustification = MathTopBotPositions.Top;
```

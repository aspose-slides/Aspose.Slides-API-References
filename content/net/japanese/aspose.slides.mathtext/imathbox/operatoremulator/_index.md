## IMathBox.OperatorEmulator プロパティ

Operator Emulator。true の場合、ボックスとその内容は単一のオペレーターとして動作し、オペレーターのプロパティを継承します。これは、たとえばその文字が改行位置として機能し、他のオペレーターに揃えられることを意味します。Operator Emulator は、'==' のように 1 つ以上の字形が結合してオペレーターを構成する場合に頻繁に使用されます。デフォルト値: false

```csharp
public bool OperatorEmulator { get; set; }
```

例:

[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.OperatorEmulator = true;

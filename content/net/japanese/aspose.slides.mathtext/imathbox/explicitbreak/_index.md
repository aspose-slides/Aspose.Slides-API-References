## IMathBox.ExplicitBreak プロパティ

ExplicitBreak は、Box オブジェクトの先頭に改行があるかどうかを指定し、行が Box オブジェクトの先頭で折り返されるようにします。前の行の数式テキストの演算子の番号を指定し、現在の行の数式テキストの整列ポイントとして使用します。可能な値: 1..255 デフォルト: 0 (明示的な改行なし)

```csharp
public byte ExplicitBreak { get; set; }
```

### 例

例:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.ExplicitBreak = 1;
```

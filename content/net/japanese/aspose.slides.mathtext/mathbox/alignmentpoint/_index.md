## MathBox.AlignmentPoint プロパティ

true の場合、この演算子エミュレーターはアラインメントポイントとして機能します。つまり、他の式で指定されたアラインメントポイントとこのポイントを合わせることができます。デフォルト: false

```csharp
public bool AlignmentPoint { get; set; }
```

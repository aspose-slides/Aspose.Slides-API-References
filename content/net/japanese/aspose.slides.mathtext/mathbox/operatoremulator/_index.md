## MathBox.OperatorEmulator プロパティ

Operator Emulator。true の場合、ボックスとその内容は単一の演算子として動作し、演算子のプロパティを継承します。たとえば、この文字は改行位置として機能でき、他の演算子に合わせて配置することができます。Operator Emulator は、'==' のように 1 つ以上の字形が組み合わさって演算子を構成する場合に頻繁に使用されます。既定値: false

```csharp
public bool OperatorEmulator { get; set; }
```

### 例

Example:

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("=="));
box.OperatorEmulator = true;
```

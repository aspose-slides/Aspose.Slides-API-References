## MathBox.ExplicitBreak プロパティ

Explicit break は、Box オブジェクトの開始位置に改行があるかどうかを指定します。これにより、行は Box オブジェクトの開始位置で折り返されます。 前の行の数式テキスト内の演算子の番号を指定します。この番号は、現在の行の数式テキストの配置基準点として使用されます。可能な値: 1..255 デフォルト: 0 (明示的な改行なし)

```csharp
public byte ExplicitBreak { get; set; }
```

### 例

例:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.ExplicitBreak = 1;
```

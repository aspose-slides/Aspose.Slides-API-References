## MathMatrix.RowGap プロパティ

行列の行間の垂直間隔の値です。RowGapRule が 3 ("Exactly") に設定されている場合、単位はツイプ (1/20 ポイント) と解釈されます。RowGapRule が 4 ("Multiple") に設定されている場合、単位は半行として解釈されます。デフォルト: 0

```csharp
public uint RowGap { get; set; }
```

### 例

例:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

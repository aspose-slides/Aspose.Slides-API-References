## MathMatrix.MinColumnWidth プロパティ

ツイプ（1ポイントの1/20）での最小列幅です。ギャップ間隔（「Column Gap」または「Gap Width」とも呼ばれます）は MinColumnWidth に加算され、合計の Matrix 列間隔（異なる列の同じエッジ間の距離）を決定します。デフォルト: 0.

```csharp
public uint MinColumnWidth { get; set; }
```

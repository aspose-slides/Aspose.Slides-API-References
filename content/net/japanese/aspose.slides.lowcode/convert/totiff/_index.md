## ToTiff(Presentation, string) {#totiff}

入力プレゼンテーションを TIFF 形式の画像セットに変換します。出力ファイル名が "myPath/myFilename.tiff" の場合、結果は "myPath/myFilename_N.tiff" という形式のファイルセットとして保存されます。ここで N はスライド番号です。

public static void ToTiff(Presentation pres, string outputFileName)

## ToTiff(Presentation, string, ITiffOptions, bool) {#totiff_1}

入力プレゼンテーションをカスタムオプションで TIFF 形式に変換します。出力ファイル名が "myPath/myFilename.tiff" で、*multipage* が `false` の場合、結果は "myPath/myFilename_N.tiff" という形式のファイルセットとして保存されます（N はスライド番号です）。*multipage* が `true` の場合は、マルチページの "myPath/myFilename.tiff" ドキュメントが生成されます。

```csharp
public static void ToTiff(Presentation pres, string outputFileName, ITiffOptions options, 
    bool multipage)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| pres | Presentation | 入力プレゼンテーション。 |
| outputFileName | String | 出力ファイル名。 |
| options | ITiffOptions | TIFF 保存オプション。 |
| multipage | Boolean | 生成された TIFF ドキュメントがマルチページであるかどうかを指定します。 |

### 例外

| 例外 | 条件 |
| --- | --- |
| ArgumentException |  |

### 例

```csharp
ITiffOptions options = new TiffOptions()
{
    CompressionType = TiffCompressionTypes.CCITT3,
    SlidesLayoutOptions = new NotesCommentsLayoutingOptions()
    {
        NotesPosition = NotesPositions.BottomTruncated
    }
};

using (var pres = new Presentation("pres.pptx"))
   Convert.ToTiff(pres, "pres.tiff", options, false);
```

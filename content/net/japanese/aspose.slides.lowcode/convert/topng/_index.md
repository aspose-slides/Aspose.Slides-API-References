## ToPng(Presentation, string) {#topng}

入力プレゼンテーションを PNG 形式の画像セットに変換します。出力ファイル名が "myPath/myFilename.png" の場合、結果は "myPath/myFilename_N.png" という名前のファイルセットとして保存されます。ここで N はスライド番号です。

```csharp
public static void ToPng(Presentation pres, string outputFileName)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| pres | Presentation | 入力プレゼンテーション。 |
| outputFileName | String | 出力ファイル名。 |

### 例外

| 例外 | 条件 |
| --- | --- |
| ArgumentException |  |

### 例

```csharp

   Convert.ToPng(pres, "presImage.png");
```

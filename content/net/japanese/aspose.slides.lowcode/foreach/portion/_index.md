## Portion(Presentation, ForEachPortionCallback) {#portion_1}

[`Presentation`](../../../aspose.slides/presentation) の中の各 `Portion` を反復処理します。`Portion` はすべてのスライドタイプ - [`Slide`](../slide), [`MasterSlide`](../masterslide) および [`LayoutSlide`](../layoutslide) で反復処理されます。

```csharp
public static void Portion(Presentation pres, ForEachPortionCallback forEachPortion)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| pres | Presentation | `Portion` を反復処理する対象の Presentation |
| forEachPortion | ForEachPortionCallback | 各 `Portion` に対して呼び出されるコールバック |

### 例

```csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    ForEach.Portion(pres, (portion, para, slide, index) =>
    {
        System.Console.WriteLine($"{portion.Text}, index: {index}");
```

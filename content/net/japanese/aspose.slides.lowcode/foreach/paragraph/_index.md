## Paragraph(Presentation, ForEachParagraphCallback) {#paragraph_1}

[`Presentation`](../../../aspose.slides/presentation)内の各`Paragraph`を反復します。シェイプはすべてのタイプのスライド（[`Slide`](../slide)、[`MasterSlide`](../masterslide)、[`LayoutSlide`](../layoutslide)）で反復されます。

```csharp
public static void Paragraph(Presentation pres, ForEachParagraphCallback forEachParagraph)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| pres | Presentation | 段落を反復するための Presentation |
| forEachParagraph | ForEachParagraphCallback | 各段落に対して呼び出されるコールバック |

### 例

```csharp
using (Presentation pres = new Presentation("pres.pptx"))
{
    ForEach.Paragraph(pres, (para, slide, index) =>
    {
        System.Console.WriteLine($"{para.Text}, index: {index}");
    });
}        
```

## Paragraph(Presentation, bool, ForEachParagraphCallback) {#paragraph}

[`Presentation`](../../../aspose.slides/presentation)内の各[`Paragraph`](../paragraph)を反復します。シェイプはすべてのタイプのスライド（[`Slide`](../slide)、[`MasterSlide`](../masterslide)、[`LayoutSlide`](../layoutslide)、[`NotesSlide`](../../../aspose.slides/notesslide)）で反復されます。

```csharp
public static void Paragraph(Presentation pres, bool includeNotes, 
    ForEachParagraphCallback forEachParagraph)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| pres | Presentation | 段落を反復するための Presentation |
| includeNotes | Boolean | NotesSlides を処理に含めるかどうかを示すフラグ |
| forEachParagraph | ForEachParagraphCallback | 各段落に対して呼び出されるコールバック |

### 例

```csharp

{
    ForEach.Paragraph(pres, true, (para, slide, index) =>
    {
        System.Console.WriteLine($"{para.Text}, index: {index}");
    });
}        
```

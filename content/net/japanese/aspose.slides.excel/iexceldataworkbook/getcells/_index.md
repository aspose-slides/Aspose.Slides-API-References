## IExcelDataWorkbook.GetCells メソッド

指定された数式に一致するブックからセルのコレクションを取得します。

```csharp
public ReadOnlyCollection<IExcelDataCell> GetCells(string formula, bool skipHiddenCells)
```

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| formula | String | 対象のセルを識別するために使用される数式または範囲式（例: "Sheet1!A1:B3"）です。 |
| skipHiddenCells | Boolean | `true` の場合、非表示のセル（非表示の行や列にあるセルなど）は結果から除外されます。 |

### 戻り値

指定された数式に一致するセルの読み取り専用リストです。

### 例

例:

```csharp
[C#]
ExcelDataWorkbook wb = new ExcelDataWorkbook(testFile);
IReadOnlyList<IExcelDataCell> cells = wb.GetCells("Sheet1!A2:A6", false);
Console.WriteLine(cells.Count); //出力: 5
```

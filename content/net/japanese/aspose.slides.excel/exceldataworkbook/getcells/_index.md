## ExcelDataWorkbook.GetCells メソッド

指定された数式に一致するブック内のセルのコレクションを取得します。

```csharp
public ReadOnlyCollection<IExcelDataCell> GetCells(string formula, bool skipHiddenCells)
```

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| formula | String | 対象セルを識別するために使用される数式または範囲式 (例: "Sheet1!A1:B3")。 |
| skipHiddenCells | Boolean | `true` の場合、非表示の行や列にある隠しセルは結果から除外されます。 |

### 戻り値

指定された数式に一致するセルの読み取り専用リスト。

### 例

例:

```csharp
[C#]
ExcelDataWorkbook wb = new ExcelDataWorkbook(testFile);
IReadOnlyList<IExcelDataCell> cells = wb.GetCells("Sheet1!A2:A6", false);
Console.WriteLine(cells.Count); //出力: 5
```

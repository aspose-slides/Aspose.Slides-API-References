## ImageFormat class

 Represents the file format of the image.
 

## Constants

| Name | Value | Description |
| --- | --- | --- |
[MemoryBmp](#MemoryBmp) | 0 | Represents the format of a bitmap in memory. |
[Bmp](#Bmp) | 1 | Represents the bitmap (BMP) image format. |
[Emf](#Emf) | 2 | Represents the enhanced metafile (EMF) image format. |
[Wmf](#Wmf) | 3 | Represents the Windows metafile (WMF) image format. |
[Gif](#Gif) | 4 | Represents the Graphics Interchange Format (GIF) image format. |
[Jpeg](#Jpeg) | 5 | Represents the Joint Photographic Experts Group (JPEG) image format. |
[Png](#Png) | 6 | Represents the W3C Portable Network Graphics (PNG) image format. |
[Tiff](#Tiff) | 7 | Represents the Tagged Image File Format (TIFF) image format. |
[Exif](#Exif) | 8 | Represents the Exchangeable Image File (Exif) format. |
[Icon](#Icon) | 9 | Represents the Windows icon image format. |

## SaveFormat class

 Constants which define the format of a saved presentation.

[Odp](#Odp) | 6 | Save presentation in ODP format. |

[Swf](#Swf) | 15 | Save presentation in SWF format. |
[Otp](#Otp) | 17 | Save presentation in OTP (presentation template) format. |

## MarkdownSaveOptions class

 Represents options that control how presentation should be saved to markdown.
 

## Constructors

| Name | Description |
| --- | --- |
| [MarkdownSaveOptions](markdownsaveoptions)() | Ctor. |

## Methods

| Name | Description |
| --- | --- |
| [getBasePath](getbasepath)() | Specifies the base path where document with resources will be saved. Default is the current directory of the application. |
| [getExportType](getexporttype)() | Specifies markdown specification to convert presentation. Default is TextOnly. |
| [getFlavor](getflavor)() | Specifies markdown specification to convert presentation. Default is Multi-markdown. |
| [getImagesSaveFolderName](getimagessavefoldername)() | Specifies folder name to save images. Default is Images. |
| [getNewLineType](getnewlinetype)() | Specifies whether the generated document should have new lines \r(Macintosh) of \n(Unix) or \r\n(Windows). Default is Unix. |
| [getShowComments](getshowcomments)() | Specifies whether the generated document should show comments or not. Default is false. |
| [getShowHiddenSlides](getshowhiddenslides)() | Specifies whether the generated document should include hidden slides or not. Default is false. |
| [getShowSlideNumber](getshowslidenumber)() | Specifies whether the generated document should show number of each slide or not. Default is false. |
| [setBasePath](setbasepath)(String) | Specifies the base path where document with resources will be saved. Default is the current directory of the application. |

| [setShowSlideNumber](setshowslidenumber)(boolean) | Specifies whether the generated document should show number of each slide or not. Default is false. |

## SmartArtNode class

 Represents node of a SmartArt object
 

## Methods

| Name | Description |
| --- | --- |
| [getBulletFillFormat](getbulletfillformat)() | Returns the FillFormat object that contains fill formatting properties for a node bullet. Note: can return null for certain types of SmartArt layout which does not provide bullets for nodes. Read-only IFillFormat. |
| [getChildNodes](getchildnodes)() | Returns collections of all child nodes of the current node. Read-only ISmartArtNodeCollection. |
| [getLevel](getlevel)() | Returns nesting level of the node. Read-only int. |
| [getOrganizationChartLayout](getorganizationchartlayout)() | Returns or sets organization chart layout type associated with current node. Read/write OrganizationChartLayoutType. |
| [getPosition](getposition)() | Returns or sets zero-based position of node among sibling nodes. Read/write int. |
| [getShapes](getshapes)() | Returns collections of all shapes associated with the node. Read-only ISmartArtShapeCollection. |
| [getTextFrame](gettextframe)() | Returns or sets text of the node. Read-only ITextFrame. |
| [isAssistant](isassistant)() | Returns or sets the node as assistant. Read/write boolean. |
| [isHidden](ishidden)() | Returns true if this node is a hidden node in the data model. Read-only boolean. |
| [remove](remove)() | Remove current node. |
| [setAssistant](setassistant)(boolean) | Returns or sets the node as assistant. Read/write boolean. |
| [setOrganizationChartLayout](setorganizationchartlayout)(int) | Returns or sets organization chart layout type associated with current node. Read/write OrganizationChartLayoutType. |
| [setPosition](setposition)(int) | Returns or sets zero-based position of node among sibling nodes. Read/write int. |

## Aspose.Slides for .NET

{{% alert color="primary" %}} 

![Aspose.Slides for .NET Product Logo](home_1.png)

Aspose.Slides for .NET は、Microsoft PowerPoint® を使用せずに .NET アプリケーションで PowerPoint® ドキュメントの読み取りと書き込みを可能にする Microsoft PowerPoint® 管理 API です。Aspose.Slides for .NET は、PowerPoint® ドキュメントを管理する機能を提供する唯一のコンポーネントです。テキスト、図形、テーブルおよびアニメーションの管理、スライドへのオーディオおよびビデオの追加、スライドのプレビュー、スライドを SVG や PDF 形式にエクスポートする機能など、多くの主要機能を提供します。

{{% /alert %}}

- [Aspose.Slides for .NET API Tutorials](/tutorials/slides/net/ja/)

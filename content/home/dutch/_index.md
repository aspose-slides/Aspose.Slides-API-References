## Aspose.Slides for .NET

{{% alert color="primary" %}} 

![Aspose.Slides for .NET productlogo](home_1.png)

Aspose.Slides for .NET is een Microsoft PowerPoint® beheer-API die .NET-toepassingen in staat stelt PowerPoint®-documenten te lezen en te schrijven zonder Microsoft PowerPoint® te gebruiken. Aspose.Slides for .NET is de eerste en enige component die functionaliteit biedt om PowerPoint®-documenten te beheren. Aspose.Slides for .NET biedt veel belangrijke functies, zoals tekst, vormen, tabellen en animaties beheren, audio en video aan dia’s toevoegen, dia’s vooraf bekijken, dia’s exporteren naar SVG, PDF-formaat en meer.

{{% /alert %}}

Dit zijn links naar enkele nuttige bronnen:
- [Aspose.Slides for .NET API Reference](/slides/net/nl/)
- [Aspose.Slides for .NET API Tutorials](/tutorials/slides/net/nl/)

## Aspose.Slides for .NET

{{% alert color="primary" %}} 

![Aspose.Slides for .NET Logo produktu](home_1.png)

Aspose.Slides for .NET je Microsoft PowerPoint® management API, které umožňuje aplikacím .NET číst a zapisovat dokumenty PowerPoint® bez použití Microsoft PowerPoint®. Aspose.Slides for .NET je první a jediná součást, která poskytuje funkčnost pro správu dokumentů PowerPoint®. Aspose.Slides for .NET nabízí řadu klíčových funkcí, jako je správa textu, tvarů, tabulek a animací, přidávání zvuku a videa do snímků, náhled snímků, export snímků do formátů SVG, PDF a další test.

{{% /alert %}}

## Aspose.Slides for .NET

{{% alert color="primary" %}} 

![Aspose.Slides for .NET Product Logo](home_1.png)

Aspose.Slides for .NET คือ API การจัดการ Microsoft PowerPoint® ที่ช่วยให้แอปพลิเคชัน .NET อ่านและเขียนเอกสาร PowerPoint® ได้โดยไม่ต้องใช้ Microsoft PowerPoint® . Aspose.Slides for .NET เป็นส่วนแรกและเพียงส่วนเดียวที่ให้ความสามารถในการจัดการเอกสาร PowerPoint® . Aspose.Slides for .NET มีคุณลักษณะสำคัญหลายอย่าง เช่น การจัดการข้อความ, รูปร่าง, ตารางและแอนิเมชัน, การเพิ่มเสียงและวิดีโอในสไลด์, การแสดงตัวอย่างสไลด์, การส่งออกสไลด์เป็น SVG, PDF format และอื่น ๆ

{{% /alert %}}

These are links to some useful resources:
- [Aspose.Slides for .NET API Reference](/slides/net/th/)
- [Aspose.Slides for .NET API Tutorials](/tutorials/slides/net/th/)

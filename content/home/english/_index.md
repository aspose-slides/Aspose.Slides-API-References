## Aspose.Slides for .NET

{{% alert color="primary" %}} 

![Aspose.Slides for .NET Product Logo](home_1.png)

Aspose.Slides for .NET is a Microsoft PowerPoint® management API that enables .NET applications to read and write PowerPoint® documents without using Microsoft PowerPoint®. Aspose.Slides for .NET is the first and only component that provides the functionality to manage PowerPoint® documents. Aspose.Slides for .NET provides a lot of key features such as managing text, shapes, tables & animations, adding audio and video to slides, previewing slides, exporting slides to SVG, PDF format and more test.

{{% /alert %}}

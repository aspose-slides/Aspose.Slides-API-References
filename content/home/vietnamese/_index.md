## Aspose.Slides for .NET

{{% alert color="primary" %}} 

![Aspose.Slides for .NET Product Logo](home_1.png)

Aspose.Slides for .NET là một API quản lý Microsoft PowerPoint® cho phép các ứng dụng .NET đọc và ghi tài liệu PowerPoint® mà không cần sử dụng Microsoft PowerPoint®. Aspose.Slides for .NET là thành phần đầu tiên và duy nhất cung cấp khả năng quản lý tài liệu PowerPoint®. Aspose.Slides for .NET cung cấp rất nhiều tính năng chính như quản lý văn bản, hình dạng, bảng & hoạt hình, thêm âm thanh và video vào các slide, xem trước slide, xuất slide ra định dạng SVG, PDF và nhiều tính năng khác.

{{% /alert %}}

Đây là các liên kết tới một số tài nguyên hữu ích:
- [Aspose.Slides for .NET API Reference](/slides/net/vi/)
- [Aspose.Slides for .NET API Tutorials](/tutorials/slides/net/vi/)

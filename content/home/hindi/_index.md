## Aspose.Slides for .NET

{{% alert color="primary" %}} 

![Aspose.Slides for .NET Product Logo](home_1.png)

Aspose.Slides for .NET एक Microsoft PowerPoint® प्रबंधन API है जो .NET अनुप्रयोगों को Microsoft PowerPoint® का उपयोग किए बिना PowerPoint® दस्तावेज़ पढ़ने और लिखने की अनुमति देती है। Aspose.Slides for .NET वह पहला और唯一 घटक है जो PowerPoint® दस्तावेज़ों को प्रबंधित करने की कार्यक्षमता प्रदान करता है। Aspose.Slides for .NET कई मुख्य सुविधाएँ प्रदान करता है जैसे पाठ, आकार, तालिकाएँ एवं एनीमेशन का प्रबंधन, स्लाइड्स में ऑडियो और वीडियो जोड़ना, स्लाइड्स का पूर्वावलोकन, स्लाइड्स को SVG, PDF प्रारूप में निर्यात करना और अधिक परीक्षण।

{{% /alert %}}

ये कुछ उपयोगी संसाधनों के लिंक हैं:
- [Aspose.Slides for .NET API Reference](/slides/net/hi/)
- [Aspose.Slides for .NET API Tutorials](/tutorials/slides/net/hi/)

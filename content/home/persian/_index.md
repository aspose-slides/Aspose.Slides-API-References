## Aspose.Slides برای .NET

{{% alert color="primary" %}} 

![Aspose.Slides برای .NET لوگوی محصول](home_1.png)

Aspose.Slides برای .NET یک Microsoft PowerPoint® management API است که به برنامه‌های .NET امکان می‌دهد اسناد PowerPoint® را بدون استفاده از Microsoft PowerPoint® بخوانند و بنویسند. Aspose.Slides برای .NET اولین و تنها مؤلفه‌ای است که قابلیت مدیریت اسناد PowerPoint® را فراهم می‌کند. Aspose.Slides برای .NET ویژگی‌های کلیدی زیادی ارائه می‌دهد مانند مدیریت متن، اشکال، جدول‌ها & انیمیشن‌ها، افزودن صدا و ویدئو به اسلایدها، پیش‌نمایش اسلایدها، خروجی اسلایدها به SVG، فرمت PDF و موارد دیگر.

{{% /alert %}}

اینها لینک‌هایی به برخی منابع مفید هستند:
- [Aspose.Slides for .NET API Reference](/slides/net/fa/)
- [Aspose.Slides for .NET API Tutorials](/tutorials/slides/net/fa/)

## Aspose.Slides for .NET

{{% alert color="primary" %}} 

![Aspose.Slides for .NET Product Logo](home_1.png)

Aspose.Slides for .NET هو API لإدارة Microsoft PowerPoint® يتيح لتطبيقات .NET قراءة وكتابة مستندات PowerPoint® دون الحاجة إلى استخدام Microsoft PowerPoint®. Aspose.Slides for .NET هو المكوّن الأول والوحيد الذي يوفر إمكانية إدارة مستندات PowerPoint®. يقدم Aspose.Slides for .NET الكثير من المميزات الرئيسية مثل إدارة النصوص، الأشكال، الجداول والرسوم المتحركة، إضافة الصوت والفيديو إلى الشرائح، معاينة الشرائح، تصدير الشرائح إلى SVG، صيغة PDF والمزيد من الاختبارات.

{{% /alert %}}

- [Aspose.Slides for .NET API Reference](/slides/net/ar/)
- [Aspose.Slides for .NET API Tutorials](/tutorials/slides/net/ar/)

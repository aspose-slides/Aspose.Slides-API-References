## SlideShowTransition::set_SoundName(System::String) μέθοδος

Καθορίζει ένα ανθρώπινα αναγνώσιμο όνομα για τον ήχο της μετάβασης. Το [ISlideShowTransition::set_Sound](../../../aspose.slides/islideshowtransition/set_sound/) πρέπει να εκχωρηθεί για να ληφθεί ή να οριστεί το όνομα ήχου. Γράφει [System::String](../../../system/string/).

```cpp
void Aspose::Slides::SlideShow::SlideShowTransition::set_SoundName(System::String value) override
```

## Παρατηρήσεις

Αυτό το όνομα εμφανίζεται στη διεπαφή χρήστη του PowerPoint όταν ρυθμίζεται ο ήχος της μετάβασης μέσω χειροκίνητης επιλογής.

## SlideShowTransition::set_SoundIsBuiltIn(bool) μέθοδος

Καθορίζει εάν ή όχι αυτός ο ήχος είναι ενσωματωμένος ήχος. Εάν αυτό το χαρακτηριστικό οριστεί σε true, τότε η δημιουργική εφαρμογή ειδοποιείται να ελέγξει το χαρακτηριστικό name που έχει καθοριστεί για αυτόν τον ήχο στη λίστα των ενσωματωμένων ήχων και μπορεί στη συνέχεια να εμφανίσει ένα προσαρμοσμένο όνομα ή UI, όπως απαιτείται. Γράφει **bool**.

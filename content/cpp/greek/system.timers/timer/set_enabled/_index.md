## Timer::set_Enabled(bool) μέθοδος

Ξεκινά ή σταματά το χρονόμετρο. Η εκκίνηση του χρονόμετρου δεν επανεκκινεί την μέτρηση του χρόνου εάν το χρονόμετρο εκτελείται ήδη.

```cpp
void System::Timers::Timer::set_Enabled(bool value)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| value | **bool** | true για εκκίνηση χρονόμετρου, false για διακοπή. |

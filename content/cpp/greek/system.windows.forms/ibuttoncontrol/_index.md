## IButtonControl κλάση

Ψεύτικη κλάση για να γίνει ο μεταγλωττιστέος κώδικας που χρησιμοποιεί την [IButtonControl](./) διασύνδεση. Τα αντικείμενα αυτής της κλάσης πρέπει να διατίθενται μόνο μέσω της συνάρτησης [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργήσετε αντίγραφο αυτού του τύπου στη στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς θα προκαλέσει σφάλματα χρόνου εκτέλεσης και/ή σφάλματα δήλωσης. Πάντα τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να το περάσετε στις συναρτήσεις ως παράμετρο.

```cpp
class IButtonControl : public virtual System::Object
```

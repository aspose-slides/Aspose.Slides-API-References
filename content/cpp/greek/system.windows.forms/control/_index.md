## Control κλάση

Dummy κλάση to make translated code that uses [Control](./) κλάση compilable. Οι μέθοδοι are not implemented. Τα αντικείμενα αυτής της κλάσης should only be allocated using [System::MakeObject()](../../system/makeobject/) function. Ποτέ μην δημιουργείτε instance of this type on stack or using operator new, as it will result in runtime errors and/or assertion faults. Πάντα wrap this κλάση into [System::SmartPtr](../../system/smartptr/) pointer and use this pointer to pass it to functions as argument.

```cpp
class Control : public System::Object
```

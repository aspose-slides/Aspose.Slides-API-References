## AutoSizeMode enum


Ορίζει πώς λειτουργεί η αυτόματη προσαρμογή.

```cpp
enum class AutoSizeMode
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| GrowAndShrink | 0 | [Control](../control/) μεγαλώνει και συρρικνώνεται όπως χρειάζεται. Δεν μπορεί να αλλάξει μέγεθος χειροκίνητα. |
| GrowOnly | 1 | [Control](../control/) μεγαλώνει όπως χρειάζεται αλλά δεν μπορεί να συρρικνωθεί κάτω από το ελάχιστο μέγεθος. Μπορεί να αλλάξει μέγεθος χειροκίνητα. |

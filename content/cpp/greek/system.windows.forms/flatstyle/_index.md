## FlatStyle enum

Καθορίζει εάν ο ελέγχος σχεδιάζεται επίπεδο ή τρισδιάστατο.

```cpp
enum class FlatStyle
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Flat | 0 | [Control](../control/) σχεδιάζεται επίπεδο. |
| Popup | 1 | [Control](../control/) σχεδιάζεται επίπεδο εκτός αν το ποντίκι περάσει από αυτό όταν σχεδιάζεται 3D. |
| Standard | 2 | [Control](../control/) σχεδιάζεται 3D. |
| System | 3 | Το στυλ ελέγχεται από το λειτουργικό σύστημα. |

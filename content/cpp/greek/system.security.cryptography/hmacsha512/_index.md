## HMACSHA512 κλάση

Κώδικας [Authentication](../../system.security.authentication/) βασισμένος σε hash μηνύματος που χρησιμοποιεί τη συνάρτηση hash [SHA512](../sha512/). Μερική υλοποίηση. Αντικείμενα αυτής της κλάσης πρέπει να εκχωρούνται μόνο χρησιμοποιώντας τη συνάρτηση [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε στιγμιότυπο αυτού του τύπου στη στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς αυτό θα προκαλέσει σφάλματα χρόνου εκτέλεσης και/ή σφάλματα επιβεβαίωσης. Πάντα τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να τον περάσετε σε συναρτήσεις ως όρισμα.

```cpp
class HMACSHA512 : public System::Security::Cryptography::HashAlgorithm
```

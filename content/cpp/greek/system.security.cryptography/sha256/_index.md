## SHA256 κλάση

Υπολογίζει το hash [SHA256](./) για τα εισαγόμενα δεδομένα. Στη C++ το [SHA256](./) δεν είναι αφηρημένη κλάση. Τα αντικείμενα αυτής της κλάσης πρέπει να εκχωρούνται μόνο χρησιμοποιώντας τη συνάρτηση [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε ένα στιγμιότυπο αυτού του τύπου στη στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς αυτό θα προκαλέσει σφάλματα χρόνου εκτέλεσης και/ή σφαλματικές διαβεβαιώσεις. Πάντα τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να το περάσετε ως όρισμα σε συναρτήσεις.

```cpp
class SHA256 : public System::Security::Cryptography::HashAlgorithm
```

## FromBase64TransformMode enum

Καθορίζει αν πρέπει να αγνοηθεί το λευκό διάστημα στη μετατροπή base 64.

```cpp
enum class FromBase64TransformMode
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| IgnoreWhiteSpaces | 0 | Το λευκό διάστημα πρέπει να αγνοείται. |
| DoNotIgnoreWhiteSpaces | 1 | Το λευκό διάστημα δεν πρέπει να αγνοείται. |

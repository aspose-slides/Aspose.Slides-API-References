## Rfc2898DeriveBytes κλάση

Υλοποιεί παραγωγή κλειδιού βάσει κωδικού πρόσβασης, PBKDF2. Τα αντικείμενα αυτής της κλάσης πρέπει να δεσμεύονται μόνο χρησιμοποιώντας τη λειτουργία [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε αντικείμενο αυτού του τύπου στη στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς θα προκληθούν σφάλματα χρόνου εκτέλεσης και/ή σφάλματα δήλωσης. Πάντα τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να τη περάσετε ως όρισμα σε συναρτήσεις.

```cpp
class Rfc2898DeriveBytes : public System::Security::Cryptography::DeriveBytes
```

## SHA1CryptoServiceProvider κλάση

[SHA1](../sha1/) Υλοποίηση CSP. Τα αντικείμενα αυτής της κλάσης πρέπει να εκχωρούνται μόνο μέσω της συνάρτησης [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε παρουσία αυτού του τύπου στη στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης και/ή αποτυχίες ελέγχου. Πάντα να τυλίγετε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και να χρησιμοποιείτε αυτόν τον δείκτη για να τη μεταβιβάζετε στις συναρτήσεις ως όρισμα.

```cpp
class SHA1CryptoServiceProvider : public System::Security::Cryptography::SHA1
```

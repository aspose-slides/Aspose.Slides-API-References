## DeriveBytes κλάση


Αφηρημένη κλάση από την οποία κληρονομούν όλες οι κλάσεις που παράγουν ακολουθίες byte συγκεκριμένου μήκους. Τα αντικείμενα αυτής της κλάσης πρέπει να καταλαμβάνονται μόνο χρησιμοποιώντας τη λειτουργία [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε αντικείμενο αυτού του τύπου στη στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς θα προκαλέσει σφάλματα χρόνου εκτέλεσης και/ή σφάλματα επιβεβαίωσης. Πάντα τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να το περάσετε σε λειτουργίες ως όρισμα.

```cpp
class DeriveBytes : public System::Object
```

## RSAParameters δομή


[Data](../../system.data/) δομή των [RSA](../rsa/) παραμέτρων αλγορίθμου. Αυτός ο τύπος πρέπει να εκχωρηθεί στη στοίβα και να περάσει σε συναρτήσεις κατά τιμή ή κατά αναφορά. Ποτέ μην χρησιμοποιείτε την κλάση [System::SmartPtr](../../system/smartptr/) για τη διαχείριση αντικειμένων αυτού του τύπου.

```cpp
class RSAParameters : public System::Object
```

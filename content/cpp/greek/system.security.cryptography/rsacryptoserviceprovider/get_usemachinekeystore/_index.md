## RSACryptoServiceProvider::get_UseMachineKeyStore() μέθοδος

Ελέγχει αν το κλειδί παραμένει στον αποθηκευτικό χώρο του μηχανήματος αντί στον αποθηκευτικό χώρο του χρήστη.

```cpp
static bool System::Security::Cryptography::RSACryptoServiceProvider::get_UseMachineKeyStore()
```

### Τιμή Επιστροφής

True εάν το κλειδί πρέπει να παραμείνει στον αποθηκευτικό χώρο του μηχανήματος, false εάν πρέπει να παραμείνει στον αποθηκευτικό χώρο του χρήστη.

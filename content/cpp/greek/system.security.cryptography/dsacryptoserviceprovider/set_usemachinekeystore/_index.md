## DSACryptoServiceProvider::set_UseMachineKeyStore(bool) μέθοδος

```cpp
static void System::Security::Cryptography::DSACryptoServiceProvider::set_UseMachineKeyStore(bool value)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| value | **bool** | True εάν το κλειδί πρέπει να παραμείνει αποθηκευμένο στο αποθηκευτικό χώρο του μηχανήματος, false εάν πρέπει να παραμείνει αποθηκευμένο στο αποθηκευτικό χώρο του χρήστη. |

## IContainer κλάση

Ψεύτικη διεπαφή για κώδικα που χρησιμοποιεί IContainer ώστε να μεταγλωττιστεί. Αντικείμενα αυτής της κλάσης πρέπει να δημιουργούνται μόνο μέσω της συνάρτησης [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε στιγμιότυπο αυτού του τύπου στη στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς θα προκαλέσει σφάλματα χρόνου εκτέλεσης και/ή σφάλματα ανάκλησης. Πάντα να τυλίγετε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και να χρησιμοποιείτε αυτόν τον δείκτη για να το περάσετε στις συναρτήσεις ως όρισμα.

```cpp
class IContainer : public virtual System::Object
```

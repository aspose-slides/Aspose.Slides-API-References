## Details_Win32Exception κλάση

Κάνει raise μιας εξαίρεσης για κωδικό σφάλματος Win32. Ποτέ μη δημιουργείτε στιγμιότυπα αυτής της κλάσης χειροκίνητα. Χρησιμοποιήστε την κλάση Win32Exception αντί αυτού. Ποτέ μη περιτυλίγετε τις στιγμιότυπες της κλάσης Win32Exception σε [System::SmartPtr](../../system/smartptr/).

```cpp
class Details_Win32Exception : public System::Details_ExceptionWithErrorCode<Details_SystemException>
```

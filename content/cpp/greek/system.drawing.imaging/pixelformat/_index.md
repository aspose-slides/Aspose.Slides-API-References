## PixelFormat enum

Καθορίζει τη μορφή των χρωματικών δεδομένων ενός εικονοστοιχείου.

```cpp
enum class PixelFormat
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Indexed | 65536 | Καθορίζει ότι τα δεδομένα του εικονοστοιχείου περιέχουν χρωματικές τιμές με ευρετήριο, πράγμα που σημαίνει ότι είναι δείκτης σε χρώματα στον πίνακα χρωμάτων του συστήματος. |
| Gdi | 131072 | Καθορίζει ότι τα δεδομένα εικονοστοιχείου περιέχουν χρώματα GDI. |
| Alpha | 262144 | Καθορίζει ότι τα δεδομένα εικονοστοιχείου περιέχουν τιμές άλφα που δεν έχουν προ-πολλαπλασιαστεί. |
| PAlpha | 524288 | Καθορίζει ότι τα δεδομένα εικονοστοιχείου περιέχουν προ-πολλαπλασιασμένες τιμές άλφα. |
| Extended | 1048576 | Κρατημένο. |
| Canonical | 2097152 | Καθορίζει τη μορφή εικονοστοιχείου 32 bit ανά εικονοστοιχείο με βάθος χρώματος 24 bit και κανάλι άλφα 8 bit. |
| Undefined | 0 | Καθορίζει ότι η μορφή εικονοστοιχείου δεν είναι καθορισμένη. |
| DontCare | 0 | Η μορφή εικονοστοιχείου δεν είναι καθορισμένη. |
| Format1bppIndexed | n/a | Καθορίζει ότι η μορφή εικονοστοιχείου είναι χρωματική με ευρετήριο 1 bit ανά εικονοστοιχείο. |
| Format4bppIndexed | n/a | Καθορίζει ότι η μορφή εικονοστοιχείου είναι χρωματική με ευρετήριο 4 bit ανά εικονοστοιχείο. |
| Format8bppIndexed | n/a | Καθορίζει ότι η μορφή εικονοστοιχείου είναι χρωματική με ευρετήριο 8 bit ανά εικονοστοιχείο. |
| Format16bppGrayScale | n/a | Καθορίζει ότι η μορφή εικονοστοιχείου είναι 16 bit ανά εικονοστοιχείο. Οι πληροφορίες χρώματος καθορίζουν 65536 αποχρώσεις του γκρι. |
| Format16bppRgb555 | n/a | Καθορίζει ότι η μορφή εικονοστοιχείου είναι 16 bit ανά εικονοστοιχείο με 5 bit για κάθε μία από τις συνιστώσες κόκκινο, πράσινο και μπλε, και το υπόλοιπο bit δεν χρησιμοποιείται. |
| Format16bppRgb565 | n/a | Καθορίζει ότι η μορφή εικονοστοιχείου είναι 16 bit ανά εικονοστοιχείο με 5 bit για το κόκκινο, 6 bit για το πράσινο και 5 bit για το μπλε. |
| Format16bppArgb1555 | n/a | Καθορίζει ότι η μορφή εικονοστοιχείου είναι 16 bit ανά εικονοστοιχείο με 5 bit για κάθε μία από τις συνιστώσες κόκκινο, πράσινο και μπλε και 1 bit για άλφα. |
| Format24bppRgb | n/a | Καθορίζει ότι η μορφή εικονοστοιχείου είναι 24 bit ανά εικονοστοιχείο με 8 bit για κάθε μία από τις συνιστώσες κόκκινο, πράσινο και μπλε. |
| Format32bppRgb | n/a | Καθορίζει ότι η μορφή εικονοστοιχείου είναι 32 bit ανά εικονοστοιχείο με 8 bit για κάθε μία από τις συνιστώσες κόκκινο, πράσινο και μπλε και τα υπόλοιπα 8 bit δεν χρησιμοποιούνται. |
| Format32bppArgb | n/a | Καθορίζει ότι η μορφή εικονοστοιχείου είναι 32 bit ανά εικονοστοιχείο με 8 bit για κάθε μία από τις συνιστώσες κόκκινο, πράσινο και μπλε και 8 bit για άλφα. |
| Format32bppPArgb | n/a | Καθορίζει ότι η μορφή εικονοστοιχείου είναι 32 bit ανά εικονοστοιχείο με 8 bit για κάθε μία από τις συνιστώσες κόκκινο, πράσινο και μπλε και 8 bit για άλφα. Οι συνιστώσες κόκκινο, πράσινο και μπλε προ-πολλαπλασιάζονται σύμφωνα με την τιμή της συνιστώσας άλφα. |
| Format48bppRgb | n/a | Καθορίζει ότι η μορφή εικονοστοιχείου είναι 48 bit ανά εικονοστοιχείο με 16 bit για κάθε μία από τις συνιστώσες κόκκινο, πράσινο και μπλε. |
| Format64bppArgb | n/a | Καθορίζει ότι η μορφή εικονοστοιχείου είναι 64 bit ανά εικονοστοιχείο με 16 bit για κάθε μία από τις συνιστώσες κόκκινο, πράσινο και μπλε και 16 bit για άλφα. |
| Format64bppPArgb | n/a | Καθορίζει ότι η μορφή εικονοστιχείου είναι 64 bit ανά εικονοστοιχείο με 16 bit για κάθε μία από τις συνιστώσες κόκκινο, πράσινο και μπλε και 16 bit για άλφα. Οι συνιστώσες κόκκινο, πράσινο και μπλε προ-πολλαπλασιάζονται σύμφωνα με την τιμή της συνιστώσας άλφα. |
| Format32bppCMYK | n/a | Καθορίζει ότι η μορφή εικονοστοιχείου είναι 32 bit ανά εικονοστοιχείο με 8 bit για κάθε μία από τις συνιστώσες κυανό, ματζέντα, κίτρινο και κλειδί. |
| Max | 16 | Η μέγιστη τιμή αυτού του enum. |

## ImageCodecInfo κλάση

Παρέχει πληροφορίες σχετικά με έναν κωδικοποιητή εικόνας. Τα αντικείμενα αυτής της κλάσης πρέπει να allocated μόνο χρησιμοποιώντας τη συνάρτηση [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε ένα στιγμιότυπο αυτού του τύπου στη στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς αυτό θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης και/ή σφαλμάτων επισήμανσης. Πάντα τυλίγετε αυτή την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να το περάσετε σε συναρτήσεις ως όρισμα.

```cpp
class ImageCodecInfo : public System::Object
```

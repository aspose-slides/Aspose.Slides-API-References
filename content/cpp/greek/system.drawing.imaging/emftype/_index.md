## EmfType enum

Ορίζει τους τύπους των εγγραφών που τοποθετούνται σε αρχείο EMF.

```cpp
enum class EmfType
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| EmfOnly | n/a | [Windows](../../system.windows/) βελτιωμένο μετααρχείο. Περιέχει εντολές GDI. Τα μετααρχεία αυτού του τύπου αναφέρονται ως αρχείο EMF. |
| EmfPlusOnly | n/a | [Windows](../../system.windows/) βελτιωμένο μετααρχείο plus. Περιέχει εντολές GDI+. Τα μετααρχεία αυτού του τύπου αναφέρονται ως αρχείο EMF+. |
| EmfPlusDual | n/a | Διπλό [Windows](../../system.windows/) βελτιωμένο μετααρχείο. Περιέχει ισοδύναμες εντολές GDI και GDI+. Τα μετααρχεία αυτού του τύπου αναφέρονται ως αρχείο EMF+. |

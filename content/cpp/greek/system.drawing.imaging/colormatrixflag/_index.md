## ColorMatrixFlag enum

Καθορίζει τους τύπους εικόνων και χρωμάτων που θα επηρεαστούν από τις ρυθμίσεις χρώματος και γκρι κλίμακας ενός αντικειμένου [ImageAttributes](../imageattributes/).

```cpp
enum class ColorMatrixFlag
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Default | 0 | Τόσο τα χρώματα όσο και οι τιμές γκρι κλίμακας προσαρμόζονται χρωματικά. |
| SkipGrays | 1 | Οι τιμές γκρι κλίμακας δεν προσαρμόζονται χρωματικά. |
| AltGrays | 2 | Μόνο οι τιμές γκρι κλίμακας προσαρμόζονται χρωματικά. |

## MetafileType enum

Αντιπροσωπεύει έναν τύπο γραφικού μετααρχείου.

```cpp
enum class MetafileType
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Invalid | 0 | Μη έγκυρο μετααρχείο. |
| Wmf | 1 | Τυπικό WMF. |
| WmfPlaceable | 2 | Με δυνατότητα τοποθέτησης [Metafile](../metafile/) μορφή. |
| Emf | 3 | EMF (όχι EMF+). |
| EmfPlusOnly | 4 | EMF+ χωρίς διπλό, καταγραφές χαμηλότερου επιπέδου. |
| EmfPlusDual | 5 | EMF+ με διπλό, καταγραφές χαμηλότερου επιπέδου. |

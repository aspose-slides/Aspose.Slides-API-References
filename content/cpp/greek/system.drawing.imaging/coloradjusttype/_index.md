## ColorAdjustType απαρίθμηση

Καθορίζει ποια αντικείμενα χρησιμοποιούν πληροφορίες προσαρμογής χρώματος.

```cpp
enum class ColorAdjustType
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Default | 0 | Ορίζει πληροφορίες προσαρμογής χρώματος που χρησιμοποιούνται από όλα τα αντικείμενα που δεν διαθέτουν τις δικές τους πληροφορίες προσαρμογής χρώματος. |
| Bitmap | 1 | Ορίζει πληροφορίες προσαρμογής χρώματος για αντικείμενα [Bitmap](../../system.drawing/bitmap/). |
| Brush | 2 | Ορίζει πληροφορίες προσαρμογής χρώματος για αντικείμενα [Brush](../../system.drawing/brush/). |
| Pen | 3 | Ορίζει πληροφορίες προσαρμογής χρώματος για αντικείμενα [Pen](../../system.drawing/pen/). |
| Text | 4 | Ορίζει πληροφορίες προσαρμογής χρώματος για κείμενο. |
| Count | 5 | Καθορίζει τον αριθμό των τύπων που έχουν καθοριστεί. |
| Any | 6 | Καθορίζει τον αριθμό των τύπων που έχουν καθοριστεί. |

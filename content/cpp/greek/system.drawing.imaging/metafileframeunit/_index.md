## MetafileFrameUnit enum

Καθορίζει τη μονάδα μέτρησης για το ορθογώνιο που χρησιμοποιείται για το μέγεθος και τη θέση ενός μετααρχείου.

```cpp
enum class MetafileFrameUnit
```

### Values

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Pixel | n/a | Καθορίζει ένα pixel ως μονάδα μέτρησης. |
| Point | n/a | Καθορίζει ένα σημείο του εκτυπωτή ως μονάδα μέτρησης. |
| Inch | n/a | Καθορίζει μια ίντσα ως μονάδα μέτρησης. |
| Document | n/a | Καθορίζει 1/300 της ίντσας ως μονάδα μέτρησης. |
| Millimeter | n/a | Καθορίζει ένα χιλιοστόμετρο ως μονάδα μέτρησης. |
| GdiCompatible | n/a | Καθορίζει .01 χιλιοστόμετρο ως μονάδα μέτρησης. Παρέχεται για συμβατότητα με το GDI. |

## EncoderParameterValueType enum

Καθορίζει τον τύπο μιας τιμής που εκπροσωπείται από την κλάση [EncoderParameter](../encoderparameter/).


| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| ValueTypeByte | 1 | Ένας 8-bit ακέραιος χωρίς πρόσημο. |
| ValueTypeAscii | 2 | Ένας 8-bit ακέραιος χωρίς πρόσημο που περιέχει ένα 7-bit ASCII. |
| ValueTypeShort | 3 | Ένας 16-bit ακέραιος χωρίς πρόσημο. |
| ValueTypeLong | 4 | Ένας 32-bit ακέραιος χωρίς πρόσημο. |
| ValueTypeRational | 5 | Ένα κλάσμα που καθορίζεται από δύο longs - αριθμητή και παρονομαστή. |
| ValueTypeLongRange | 6 | Ένα εύρος ακεραίων. |
| ValueTypeUndefined | 7 |  |
| ValueTypeRationalRange | 8 | Ένα εύρος κλασμάτων. |

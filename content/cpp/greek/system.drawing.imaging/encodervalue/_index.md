## EncoderValue enum

Καθορίζει την τιμή παραμέτρου που περνιέται σε κωδικοποιητή εικόνας JPEG ή TIFF.

```cpp
enum class EncoderValue
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| ColorTypeCMYK | 0 | Ο χρωματικός χώρος CMYK. |
| ColorTypeYCCK | 1 | Ο χρωματικός χώρος YCCK. |
| CompressionLZW | 2 | Η μέθοδος συμπίεσης LZW. |
| CompressionCCITT3 | 3 | Καθορίζει τη μέθοδο συμπίεσης CCITT3 για εικόνα TIFF. |
| CompressionCCITT4 | 4 | Καθορίζει τη μέθοδο συμπίεσης CCITT4 για εικόνα TIFF. |
| CompressionRle | 5 | Καθορίζει τη μέθοδο συμπίεσης RLE για εικόνα TIFF. |
| CompressionNone | 6 | Καθορίζει χωρίς συμπίεση για εικόνα TIFF. |
| ScanMethodInterlaced | 7 | Λειτουργία διαπλέγματος. |
| ScanMethodNonInterlaced | 8 | Λειτουργία χωρίς διαπλέγματο. |
| VersionGif87 | 9 | Καθορίζει την έκδοση 87 για εικόνα TIFF. |
| VersionGif89 | 10 | Καθορίζει την έκδοση 89a για εικόνα GIF. |
| RenderProgressive | 11 | Σταδιακή λειτουργία. |
| RenderNonProgressive | 12 | Μη-σταδιακή λειτουργία. |
| TransformRotate90 | 13 | Καθορίζει περιστροφή 90 μοιρών δεξιόστροφα χωρίς απώλειες για εικόνα JPEG. |
| TransformRotate180 | 14 | Καθορίζει περιστροφή 180 μοιρών χωρίς απώλειες για εικόνα JPEG. |
| TransformRotate270 | 15 | Καθορίζει περιστροφή 270 μοιρών δεξιόστροφα χωρίς απώλειες για εικόνα JPEG. |
| TransformFlipHorizontal | 16 | Καθορίζει οριζόντια αναστροφή χωρίς απώλειες για εικόνα JPEG. |
| TransformFlipVertical | 17 | Καθορίζει κατακόρυφη αναστροφή χωρίς απώλειες για εικόνα JPEG. |
| MultiFrame | 18 | Κωδικοποίηση πολλαπλών πλαισίων. |
| LastFrame | 19 | Το τελευταίο πλαίσιο μιας εικόνας πολλαπλών πλαισίων. |
| Flush | 20 | Το αντικείμενο κωδικοποιητή πρέπει να κλείσει. |
| FrameDimensionTime | 21 | Καθορίζει τη διάσταση χρονικού πλαισίου για εικόνα GIF. |
| FrameDimensionResolution | 22 | Διάσταση πλαισίου ανάλυσης. |
| FrameDimensionPage | 23 | Καθορίζει τη διάσταση πλαισίου σελίδας για εικόνα TIFF. |

## IEnumeratorImplRefType::MoveNext() μέθοδος


Μετακινεί τον επαναλήπτη στο επόμενο στοιχείο. Εάν δεν υπήρχε κάποιο στοιχείο που να έχει αναφερθεί προηγουμένως, ορίζει την αναφορά στο πρώτο διαθέσιμο στοιχείο. Εάν φτάσει στο τέλος του container, δεν κάνει τίποτα.

```cpp
virtual bool System::Collections::IEnumeratorImplRefType<T>::MoveNext() override
```


### Τιμή Επιστροφής

True αν κάποιο στοιχείο είναι διαθέσιμο μετά την κλήση, false διαφορετικά.

## HttpContext κλάση

Πρόχειρη κλάση που καθιστά δυνατό το σύνδεσμο μεταφρασμένου κώδικα με αναφορές [HttpContext](./), αλλά χωρίς εκτέλεση. Δεν περιέχει σωστά υλοποιημένα μέλη. Τα αντικείμενα αυτής της κλάσης θα πρέπει να εκχωρούνται μόνο χρησιμοποιώντας τη λειτουργία [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε παράδειγμα αυτού του τύπου στην στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης και/ή σφαλματα επιβεβαίωσης. Πάντα τυλίξτε αυτήν την κλάση με δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να τη περάσετε σε συναρτήσεις ως όρισμα.

```cpp
class HttpContext : public System::Object
```

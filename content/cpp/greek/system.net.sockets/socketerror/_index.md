## SocketError enum


Enumerates the socket error types.

```cpp
enum class SocketError
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Success | 0 | Μια λειτουργία socket ολοκληρώθηκε με επιτυχία. |
| SocketError | -1 | Συνεβή ένα μη καθορισμένο σφάλμα socket. |
| Interrupted | 10004 | Μια κλήση μπλοκάρισματος socket ακυρώθηκε. |
| AccessDenied | 10013 | Η πρόσβαση σε socket απορρίπτεται. |
| Fault | 10014 | Εντοπίστηκε μη έγκυρη διεύθυνση δείκτη. |
| InvalidArgument | 10022 | Παρέχεται μη έγκυρο όρισμα. |
| TooManyOpenSockets | 10024 | Υπάρχουν πάρα πολλά ανοιχτά sockets στον υποκείμενο πάροχο socket. |
| WouldBlock | 10035 | Μια λειτουργία δεν μπορεί να ολοκληρωθεί αμέσως σε μη μπλοκαριστό socket. |
| InProgress | 10036 | Μία λειτουργία μπλοκαρίσματος βρίσκεται σε εξέλιξη. |
| AlreadyInProgress | 10037 | Ένα μη μπλοκαριστό socket έχει ήδη μια εκτελούμενη λειτουργία. |
| NotSocket | 10038 | Προσπάθεια κλήσης λειτουργίας socket σε μη socket. |
| DestinationAddressRequired | 10039 | Απαιτείται διεύθυνση που παραλείπεται από μια λειτουργία socket. |
| MessageSize | 10040 | Ένα datagram είναι πολύ μεγάλο. |
| ProtocolType | 10041 | Τύπος πρωτοκόλλου δεν υποστηρίζεται από αυτό το socket. |
| ProtocolOption | 10042 | Χρησιμοποιείται άγνωστη, μη έγκυρη ή μη υποστηριζόμενη επιλογή ή επίπεδο. |
| ProtocolNotSupported | 10043 | Ένα πρωτόκολλο δεν έχει υλοποιηθεί ή δεν είναι διαμορφωμένο. |
| SocketNotSupported | 10044 | Μια οικογένεια διευθύνσεων δεν υποστηρίζει το καθορισμένο socket. |
| OperationNotSupported | 10045 | Μια οικογένεια πρωτοκόλλου δεν υποστηρίζει μια οικογένεια διευθύνσεων. |
| ProtocolFamilyNotSupported | 10046 | Μια οικογένεια πρωτοκόλλου δεν έχει υλοποιηθεί ή δεν είναι διαμορφωμένη. |
| AddressFamilyNotSupported | 10047 | Η καθορισμένη οικογένεια διευθύνσεων δεν υποστηρίζεται. |
| AddressAlreadyInUse | 10048 | Μια διεύθυνση μπορεί να χρησιμοποιηθεί μόνο μία φορά. |
| AddressNotAvailable | 10049 | Η επιλεγμένη διεύθυνση IP δεν είναι έγκυρη σε αυτό το πλαίσιο. |
| NetworkDown | 10050 | Το δίκτυο δεν είναι διαθέσιμο. |
| NetworkUnreachable | 10051 | Δεν υπάρχει διαδρομή προς το απομακρυσμένο σύστημα. |
| NetworkReset | 10052 | Μια εφαρμογή προσπάθησε να ορίσει 'Keep-Alive' σε σύνδεση που έχει ήδη λήξει. |
| ConnectionAborted | 10053 | Μια σύνδεση τερματίστηκε. |
| ConnectionReset | 10054 | Μια σύνδεση επαναρυθμίστηκε από απομακρυσμένο κόμβο. |
| NoBufferSpaceAvailable | 10055 | Δεν υπάρχει ελεύθερος χώρος buffer διαθέσιμος για λειτουργία socket. |
| IsConnected | 10056 | Ένα socket είναι ήδη συνδεδεμένο. |
| NotConnected | 10057 | Μια εφαρμογή προσπάθησε να στείλει ή να λάβει δεδομένα, και ένα socket δεν είναι συνδεδεμένο. |
| Shutdown | 10058 | Αίτηση αποστολής ή λήψης δεδομένων απαγορεύεται επειδή το socket είναι ήδη κλειστό. |
| TimedOut | 10060 | Μια προσπάθεια σύνδεσης έληξε, ή ένα συνδεδεμένο σύστημα δεν ανταποκρίθηκε. |
| ConnectionRefused | 10061 | Ένα απομακρυσμένο σύστημα απορρίπτει ενεργά τη σύνδεση. |
| HostDown | 10064 | Μια λειτουργία απέτυχε επειδή ένα απομακρυσμένο σύστημα είναι εκτός λειτουργίας. |
| HostUnreachable | 10065 | Δεν υπάρχει διαδρομή δικτύου προς το καθορισμένο σύστημα. |

| SystemNotReady | 10091 | Ένα υποσύστημα δικτύου δεν είναι διαθέσιμο. |

| NotInitialized | 10093 | Ο υποκείμενος πάροχος socket δεν έχει αρχικοποιηθεί. |
| Disconnecting | 10101 | Πραγματοποιείται μια καλή διακοπή. |
| TypeNotFound | 10109 | Η καθορισμένη κλάση δεν βρέθηκε. |
| HostNotFound | 11001 | Το καθορισμένο σύστημα είναι άγνωστο. |
| TryAgain | 11002 | Το όνομα ενός συστήματος δεν μπορεί να επιλυθεί. |
| NoRecovery | 11003 | Ένα σφάλμα δεν είναι ανακτήσιμο ή η ζητούμενη βάση δεδομένων δεν μπορεί να βρεθεί. |
| NoData | 11004 | Το ζητούμενο όνομα ή η διεύθυνση IP δεν βρέθηκε στον διακομιστή ονομάτων. |

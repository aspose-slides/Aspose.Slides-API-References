## PrintRange enum

Καθορίζει ποιες σελίδες θα εκτυπωθούν.

```cpp
enum class PrintRange
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| AllPages | 0 | Εκτύπωση όλων των σελίδων. |
| Selection | 1 | Εκτύπωση επιλεγμένων σελίδων. |
| SomePages | 2 | Εκτύπωση σελίδων από το καθορισμένο εύρος. |
| CurrentPage | 4194304 | Εκτύπωση της τρέχουσας εμφανιζόμενης σελίδας. |

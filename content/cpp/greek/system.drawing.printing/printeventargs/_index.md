## PrintEventArgs κλάση

Παρέχει δεδομένα για τα γεγονότα BeginPrint και EndPrint. Αντικείμενα αυτής της κλάσης θα πρέπει να εκχωρούνται μόνο χρησιμοποιώντας τη συνάρτηση [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε ένα αντικείμενο αυτού του τύπου στο stack ή χρησιμοποιώντας τον τελεστή new, καθώς θα προκαλέσει σφάλματα χρόνου εκτέλεσης και/ή αποτυχίες ελέγχου. Πάντα να τυλίγετε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και να χρησιμοποιείτε αυτόν το δείκτη για να το μεταβιβάζετε στις συναρτήσεις ως όρισμα.

```cpp
class PrintEventArgs : public System::ComponentModel::CancelEventArgs
```

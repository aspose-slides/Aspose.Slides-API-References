## MemoryStream::TryGetBuffer(ArraySegment\<uint8_t\>\&) μέθοδος

Επιστρέφει τον πίνακα των unsigned bytes από τα οποία δημιουργήθηκε αυτή η ροή.

```cpp
bool System::IO::MemoryStream::TryGetBuffer(ArraySegment<uint8_t> &buffer)
```

### Παράμετροι

| Parameter | Type | Description |
| --- | --- | --- |
| buffer | [ArraySegment](../../../system/arraysegment/)\<**uint8_t**\>\& | πίνακας byte - παράμετρος εξόδου. Όταν αυτή η μέθοδος επιστρέφει true, το τμήμα του πίνακα byte από το οποίο δημιουργήθηκε αυτή η ροή· όταν επιστρέφει false, αυτή η παράμετρος ορίζεται στην προεπιλογή. |

### Τιμή επιστροφής

True αν η μετατροπή ήταν επιτυχής.

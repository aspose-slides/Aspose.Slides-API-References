## Directory::Delete(const String\&, bool) μέθοδος


Αφαιρεί το καθορισμένο αρχείο ή φάκελο. Δεν προκαλεί εξαίρεση.

```cpp
static void System::IO::Directory::Delete(const String &path, bool recursive=false)
```


### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| path | const [String](../../../system/string/)\& | Η διαδρομή προς το φάκελο ή το αρχείο που θα αφαιρεθεί |
| recursive | **bool** | Αν το **path** καθορίζει έναν μη-κενό φάκελο, τότε το **recursive** καθορίζει εάν όλο το περιεχόμενο του φακέλου πρέπει να αφαιρεθεί αναδρομικά· αν ο φάκελος που καθορίζεται από το **path** δεν είναι κενός και το **recursive** είναι 'false', τότε η λειτουργία αποτυγχάνει |

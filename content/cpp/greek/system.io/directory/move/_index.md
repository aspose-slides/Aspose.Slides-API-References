## Directory::Move(const String&, const String&) μέθοδος

Μετακινεί την καθορισμένη οντότητα στη νέα θέση. Εάν η οντότητα προς μετακίνηση είναι κατάλογος, μετακινείται μαζί με όλο το περιεχόμενό της.

```cpp
static void System::IO::Directory::Move(const String &sourceDirName, const String &destDirName)
```

### Arguments

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| sourceDirName | const [String](../../../system/string/)\& | Ένας κατάλογος ή αρχείο προς μετακίνηση |
| destDirName | const [String](../../../system/string/)\& | Η νέα θέση του **sourceDirName** |

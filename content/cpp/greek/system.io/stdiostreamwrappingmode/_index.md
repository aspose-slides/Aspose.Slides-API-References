## STDIOStreamWrappingMode enum

Καθορίζει τη λειτουργία των λειτουργιών I/O που οι περιτυλίξεις θα εκτελέσουν σε ροές τύπου std::iostreams-like.

```cpp
enum class STDIOStreamWrappingMode
```

### Values

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Binary | 0 | Μια λειτουργία που επιτρέπει στις εντολές εισόδου να αποκωδικοποιούν δεδομένα ροής τύπου char_type σε byte και να κωδικοποιούν byte σε δεδομένα τύπου char_type για εντολές εξόδου. |
| Conversion | 1 | Μια λειτουργία που επιτρέπει στις εντολές εισόδου να μετατρέπουν δεδομένα ροής από τύπο char_type σε τύπο **uint8_t** και αντίστροφα για εντολές εξόδου. |

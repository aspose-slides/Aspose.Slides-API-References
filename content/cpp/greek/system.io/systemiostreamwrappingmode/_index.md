## SystemIOStreamWrappingMode enum

Καθορίζει τη λειτουργία των λειτουργιών I/O που οι περιτυλίξεις θα εκτελέσουν σε ροές τύπου [System::IO::Stream](../stream/)-like streams.

```cpp
enum class SystemIOStreamWrappingMode
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Binary | 0 | Μια λειτουργία που επιτρέπει στις λειτουργίες εισόδου να κωδικοποιούν τα byte της ροής σε δεδομένα τύπου char_type και να αποκωδικοποιούν τα δεδομένα τύπου char_type σε byte της ροής για λειτουργίες εξόδου. |
| Conversion | 1 | Μια λειτουργία που επιτρέπει στις λειτουργίες εισόδου να μετατρέπουν τα byte της ροής από τύπο **uint8_t** σε τύπο char_type και αντίστροφα για λειτουργίες εξόδου. |

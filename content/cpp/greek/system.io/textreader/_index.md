## TextReader κλάση

Μια βασική κλάση για κλάσεις που αντιπροσωπεύουν αναγνώστες που διαβάζουν ακολουθίες χαρακτήρων από διαφορετικές πηγές. Τα αντικείμενα αυτής της κλάσης θα πρέπει να εκχωρούνται μόνο χρησιμοποιώντας τη [System::MakeObject()](../../system/makeobject/) συνάρτηση. Ποτέ μην δημιουργείτε εκτελέσιμο αυτού του τύπου στη στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς θα προκαλέσει σφάλματα χρόνου εκτέλεσης και/ή σφάλματα ισχυρισμού. Πάντα τυλίξτε αυτήν τη κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να το περάσετε σε συναρτήσεις ως όρισμα.

```cpp
class TextReader : public System::IDisposable
```

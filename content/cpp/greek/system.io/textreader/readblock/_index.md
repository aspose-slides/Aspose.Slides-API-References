## TextReader::ReadBlock(ArrayPtr\<char_t\>, int, int) μέθοδος

Αναγνάει τον καθορισμένο μέγιστο αριθμό χαρακτήρων από τον τρέχοντα αναγνώστη κειμένου και γράφει τα δεδομένα σε μια ενδιάμεση μνήμη, αρχίζοντας από το καθορισμένο δείκτη.

```cpp
virtual int System::IO::TextReader::ReadBlock(ArrayPtr<char_t> buffer, int index, int count)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| buffer | [ArrayPtr](../../../system/arrayptr/)\<char_t\> | Μία ενδιάμεση μνήμη χαρακτήρων για την εγγραφή των αναγνωσμένων δεδομένων |
| index | int | Ένας δείκτης μηδενικής βάσης στο **buffer** από όπου ξεκινά η εγγραφή |
| count | int | Ο μέγιστος αριθμός χαρακτήρων προς ανάγνωση |

### Τιμή Επιστροφής

Ο πραγματικός αριθμός των χαρακτήρων που αναγνώστηκαν

## File::Copy(const String&, const String&, bool) μέθοδος

Αντιγράφει το καθορισμένο αρχείο στην καθορισμένη τοποθεσία. Εάν το αρχείο προορισμού υπάρχει ήδη, μια παράμετρος καθορίζει εάν πρέπει να αντικατασταθεί.

```cpp
static void System::IO::File::Copy(const String &sourceFileName, const String &destFileName, bool overwrite=false)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| sourceFileName | const [String](../../../system/string/)\& | Η διαδρομή του αρχείου προς αντιγραφή |
| destFileName | const [String](../../../system/string/)\& | Η διαδρομή της νέας τοποθεσίας του αρχείου προς αντιγραφή |
| overwrite | **bool** | Αληθής εάν το υπάρχον αρχείο προορισμού πρέπει να αντικατασταθεί, ψευδής εάν η αντιγραφή πρέπει να αποτύχει εάν το αρχείο προορισμού υπάρχει ήδη |

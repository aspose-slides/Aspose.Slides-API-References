## File::AppendText(const String\&) μέθοδος


Δημιουργεί ένα [StreamWriter](../../streamwriter/) αντικείμενο που προσθέτει κείμενο στο καθορισμένο αρχείο χρησιμοποιώντας κωδικοποίηση UTF-8. Εάν το καθορισμένο αρχείο δεν υπάρχει, δημιουργείται.

```cpp
static StreamWriterPtr System::IO::File::AppendText(const String &path)
```


### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| path | const [String](../../../system/string/)\& | Η διαδρομή του αρχείου για άνοιγμα ή δημιουργία |

### Τιμή Επιστροφής

Ένας shared pointer στο δημιουργημένο [StreamWriter](../../streamwriter/) αντικείμενο που σχετίζεται με το καθορισμένο αρχείο

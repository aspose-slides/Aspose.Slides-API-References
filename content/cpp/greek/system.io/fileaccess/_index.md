## FileAccess enum

Καθορίζει τον τύπο πρόσβασης κατά το άνοιγμα του αρχείου.

```cpp
enum class FileAccess
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Read | 1 | Πρόσβαση μόνο για ανάγνωση. |
| Write | 2 | Πρόσβαση μόνο για εγγραφή. |
| ReadWrite | n/a | Πρόσβαση για ανάγνωση και εγγραφή. |

## SearchOption enum

Καθορίζει ότι η αναζήτηση πρέπει να εκτελείται μόνο στον τρέχοντα φάκελο, ή στον τρέχοντα φάκελο και σε όλους τους υποφακέλους του.

```cpp
enum class SearchOption
```

### Τιμές

| Name | Value | Description |
| --- | --- | --- |
| AllDirectories | 0 | Αναζήτηση στον τρέχοντα φάκελο και σε όλους τους υποφακέλους του. |
| TopDirectoryOnly | 1 | Αναζήτηση μόνο στον τρέχοντα φάκελο. |

## XmlAttributeOverrides κλάση


Επιτρέπει την παράκαμψη χαρακτηριστικών όταν [XmlSerializer](../xmlserializer/) χρησιμοποιείται για την σειριοποίηση ή αποσειριοποίηση ενός αντικειμένου. Αντικείμενα αυτής της κλάσης πρέπει να δημιουργούνται μόνο μέσω της συνάρτησης [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε στιγμιότυπο αυτού του τύπου στη στοίβα ή χρησιμοποιώντας τον τελεστή new, επειδή θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης και/ή σφάλματα εικαστηρίων. Πάντα τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να τον περάσετε σε συναρτήσεις ως όρισμα.

```cpp
class XmlAttributeOverrides : public System::Object
```

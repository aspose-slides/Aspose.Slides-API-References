## HandoutType αρίθμηση

Καθορίζει πόσες διαφάνειες και σε ποια ακολουθία θα τοποθετηθούν στη σελίδα.

```cpp
enum class HandoutType
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Handouts1 | 0 | Μία διαφάνεια ανά σελίδα. |
| Handouts2 | 1 | Δύο διαφάνειες ανά σελίδα. |
| Handouts3 | 2 | Τρεις διαφάνειες ανά σελίδα. |
| Handouts4Horizontal | 3 | Τέσσερις διαφάνειες ανά σελίδα σε οριζόντια ακολουθία. |
| Handouts4Vertical | 4 | Τέσσερις διαφάνειες ανά σελίδα σε κάθετη ακολουθία. |
| Handouts6Horizontal | 5 | Έξι διαφάνειες ανά σελίδα σε οριζόντια ακολουθία. |
| Handouts6Vertical | 6 | Έξι διαφάνειες ανά σελίδα σε κάθετη ακολουθία. |
| Handouts9Horizontal | 7 | Εννέα διαφάνειες ανά σελίδα σε οριζόντια ακολουθία. |
| Handouts9Vertical | 8 | Εννέα διαφάνειες ανά σελίδα σε κάθετη ακολουθία. |

## PdfOptions::set_BestImagesCompressionRatio(bool) μέθοδος

Δηλώνει εάν η πιο αποτελεσματική συμπίεση (αντί της προεπιλεγμένης) για κάθε εικόνα πρέπει να επιλέγεται αυτόματα. Εάν οριστεί σε **bool**.true, για κάθε εικόνα στην παρουσίαση θα επιλεγεί ο πιο κατάλληλος αλγόριθμος συμπίεσης, κάτι που θα οδηγήσει σε μικρότερο μέγεθος του παραγόμενου εγγράφου PDF.

```cpp
void Aspose::Slides::Export::PdfOptions::set_BestImagesCompressionRatio(bool value) override
```
## Παρατηρήσεις

Η επιλογή του βέλτιστου λόγου συμπίεσης εικόνας είναι υπολογιστικά δαπανηρή και απαιτεί επιπλέον μνήμη RAM, και αυτή η επιλογή είναι **bool**.false εξ ορισμού.

Η προεπιλογή είναι **bool**.false.

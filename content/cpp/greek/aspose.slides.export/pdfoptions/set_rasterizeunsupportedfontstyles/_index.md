## PdfOptions::set_RasterizeUnsupportedFontStyles(bool) μέθοδος

Καθορίζει εάν το κείμενο πρέπει να μετατραπεί σε bitmap και να αποθηκευτεί σε PDF όταν η γραμματοσειρά δεν υποστηρίζει έντονη μορφοποίηση. Αυτή η προσέγγιση μπορεί να βελτιώσει την ποιότητα του κειμένου στο παραγόμενο PDF για ορισμένες γραμματοσειρές. Γράψτε **bool**.

```cpp
void Aspose::Slides::Export::PdfOptions::set_RasterizeUnsupportedFontStyles(bool value) override
```

## Παρατηρήσεις

Η προεπιλογή είναι **false**.

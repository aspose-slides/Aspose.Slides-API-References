## PdfOptions::set_EmbedTrueTypeFontsForASCII(bool) μέθοδος


Καθορίζει αν [Aspose.Slides](../../../aspose.slides/) θα ενσωματώνει κοινές γραμματοσειρές για κείμενο ASCII (εύρος κώδικα 33..127). [Fonts](../../../aspose.slides/fonts/) για κωδικούς χαρακτήρων μεγαλύτερους από 127 ενσωματώνονται πάντα. Η λίστα κοινών γραμματοσειρών περιλαμβάνει τις 14 βασικές γραμματοσειρές του PDF και επιπλέον γραμματοσειρές που καθορίζονται από τον χρήστη. Γράψτε **bool**.

```cpp
void Aspose::Slides::Export::PdfOptions::set_EmbedTrueTypeFontsForASCII(bool value) override
```

## Παρατηρήσεις


Η προεπιλογή είναι **true**.

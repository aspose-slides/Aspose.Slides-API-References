## PdfOptions::get_BestImagesCompressionRatio() μέθοδος


Δείχνει εάν πρέπει να επιλέγεται αυτόματα η πιο αποτελεσματική συμπίεση (αντί για την προεπιλεγμένη) για κάθε εικόνα. Εάν οριστεί σε **bool**.true, για κάθε εικόνα στην παρουσίαση θα επιλεγεί ο πιο κατάλληλος αλγόριθμος συμπίεσης, κάτι που θα οδηγήσει στο μικρότερο μέγεθος του παραγόμενου εγγράφου PDF.

```cpp
bool Aspose::Slides::Export::PdfOptions::get_BestImagesCompressionRatio() override
```

## Παρατηρήσεις


Η επιλογή του βέλτιστου λόγου συμπίεσης εικόνας είναι υπολογιστικά δαπανηρή και απαιτεί πρόσθετη ποσότητα RAM, και αυτή η επιλογή είναι προεπιλεγμένη **bool**.false.

Η προεπιλογή είναι **bool**.false.

## PdfOptions::set_JpegQuality(uint8_t) μέθοδος


Ορίζει μια τιμή που καθορίζει την ποιότητα των εικόνων JPEG μέσα στο έγγραφο PDF. Γράψτε **uint8_t**.

```cpp
void Aspose::Slides::Export::PdfOptions::set_JpegQuality(uint8_t value) override
```

## Παρατηρήσεις


Έχει αποτέλεσμα μόνο όταν ένα έγγραφο περιέχει εικόνες JPEG.

Χρησιμοποιήστε αυτήν την ιδιότητα για να λάβετε ή να ορίσετε την ποιότητα των εικόνων μέσα σε ένα έγγραφο κατά την αποθήκευση σε μορφή PDF. Η τιμή μπορεί να κυμαίνεται από 0 έως 100, όπου το 0 σημαίνει την χαμηλότερη ποιότητα αλλά τη μέγιστη συμπίεση και το 100 σημαίνει την καλύτερη ποιότητα αλλά την ελάχιστη συμπίεση.

Η προεπιλεγμένη τιμή είναι **100**.

## CommentsPositions enum

Αναπαριστά τον κανόνα για την απόδοση των σχολίων στο εξαγόμενο έγγραφο

```cpp
enum class CommentsPositions
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| None | 0 | Καθορίζει ότι τα σχόλια δεν πρέπει να εμφανιστούν καθόλου. |
| Bottom | 1 | Καθορίζει ότι τα σχόλια πρέπει να εμφανιστούν στο κάτω μέρος της σελίδας. |
| Right | 2 | Καθορίζει ότι τα σχόλια πρέπει να εμφανιστούν δεξιά της σελίδας. |

## MarkdownSaveOptions::get_HandleRepeatedSpaces() const μέθοδος

Καθορίζει πώς πρέπει να αντιμετωπίζονται οι διαδοχικοί κανονικοί χαρακτήρες κενού κατά την εξαγωγή σε Markdown.

```cpp
Aspose::Slides::Export::HandleRepeatedSpaces Aspose::Slides::Export::MarkdownSaveOptions::get_HandleRepeatedSpaces() const
```

## Παρατηρήσεις

Αυτή η ιδιότητα καθορίζει εάν οι διαδοχικοί κενά είναι:
* διατηρούνται ως κανονικοί χαρακτήρες κενού,
* εναλλάσονται μεταξύ κανονικών κενών και οντοτήτων ακατάσπαστου κενού (**&nbsp;**),
* ή αντικαθίστανται πλήρως (μετά το πρώτο) με **&nbsp;** για να διατηρηθεί η οπτική στοίχηση στην έξοδο Markdown.

Η προεπιλεγμένη τιμή είναι [HandleRepeatedSpaces::AlternateSpacesToNbsp](../../handlerepeatedspaces/).

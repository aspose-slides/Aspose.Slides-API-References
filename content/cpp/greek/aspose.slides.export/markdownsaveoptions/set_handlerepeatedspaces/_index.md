## MarkdownSaveOptions::set_HandleRepeatedSpaces(Aspose::Slides::Export::HandleRepeatedSpaces) μέθοδος

Καθορίζει πώς θα πρέπει να αντιμετωπίζονται οι επαναλαμβανόμενοι κανονικοί χαρακτήρες διαστήματος κατά την εξαγωγή σε Markdown.

```cpp
void Aspose::Slides::Export::MarkdownSaveOptions::set_HandleRepeatedSpaces(Aspose::Slides::Export::HandleRepeatedSpaces value)
```

## Παρατηρήσεις

Αυτή η ιδιότητα καθορίζει αν τα διαδοχικά διαστήματα είναι:
* διατηρούνται ως κανονικοί χαρακτήρες διαστήματος,
* εναλλάσσονται μεταξύ κανονικών διαστημάτων και οντοτήτων αχώριστων διαστημάτων (**&nbsp;**),
* ή πλήρως αντικαθίστανται (μετά το πρώτο) με **&nbsp;** για τη διατήρηση της οπτικής ευθυγράμμισης στην έξοδο Markdown.

Η προεπιλεγμένη τιμή είναι [HandleRepeatedSpaces::AlternateSpacesToNbsp](../../handlerepeatedspaces/).

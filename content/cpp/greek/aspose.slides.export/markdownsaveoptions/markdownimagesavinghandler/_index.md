## MarkdownImageSavingHandler typedef

Καλείται για κάθε εικόνα που δεν είναι SVG (bitmap ή metafile) κατά τη διαδικασία εξαγωγής Markdown. 

Επιστρέφει **true** για να χρησιμοποιήσετε το καθορισμένο *σύνδεσμο* , 

ή **false** για να εφαρμόσετε την προεπιλεγμένη λογική αποθήκευσης.

```cpp
using Aspose::Slides::Export::MarkdownSaveOptions::MarkdownImageSavingHandler =  System::MulticastDelegate<bool(System::SharedPtr<IImage>, ImageFormat, System::String&)>
```

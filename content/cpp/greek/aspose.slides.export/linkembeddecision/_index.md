## LinkEmbedDecision enum


Καθορίζει πώς θα επεξεργαστεί το αντικείμενο κατά την αποθήκευση.

```cpp
enum class LinkEmbedDecision
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Link | 0 | Το αντικείμενο θα αποθηκευτεί εξωτερικά, αναφερόμενο μέσω URL |
| Embed | 1 | Το αντικείμενο θα πρέπει να ενσωματωθεί σε ένα παραγόμενο αρχείο εάν είναι δυνατόν. Εάν η ενσωμάτωση είναι αδύνατη, θα κληθεί η GetUrl και, ανάλογα με το αποτέλεσμα, το αντικείμενο θα αναφερθεί μέσω URL ή θα αγνοηθεί. |
| Ignore | 2 | Το αντικείμενο θα αγνοηθεί. |

## PicturesCompression enum

Αναπαριστά το επίπεδο συμπίεσης των εικόνων

```cpp
enum class PicturesCompression
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Dpi330 | 0 | Καλή ποιότητα για οθόνες υψηλής ανάλυσης (HD) |
| Dpi220 | 1 | Εξαιρετική ποιότητα στα περισσότερα εκτυπωτές και οθόνες |
| Dpi150 | 2 | Καλή για ιστοσελίδες και προβολείς |
| Dpi96 | 3 | Ελαχιστοποιήστε το μέγεθος του εγγράφου για κοινή χρήση |
| Dpi72 | 4 | Προεπιλεγμένο επίπεδο συμπίεσης |
| DocumentResolution | 5 | Χρησιμοποιήστε την ανάλυση του εγγράφου - η εικόνα δεν θα συμπιεστεί και θα χρησιμοποιηθεί στο έγγραφο όπως είναι |

## SvgEvent enum

Αναπαριστά επιλογές για σχήμα SVG.

```cpp
enum class SvgEvent
```

### Τιμές

| Όνομα | Αξία | Περιγραφή |
| --- | --- | --- |
| OnFocusIn | 0 | Συμβαίνει όταν ένα στοιχείο λαμβάνει εστίαση, όπως όταν ένα κείμενο επιλέγεται. |
| OnFocusOut | 1 | Συμβαίνει όταν ένα στοιχείο χάνει εστίαση, όπως όταν ένα κείμενο αποεπιλέγεται. |
| OnActivate | 2 | Συμβαίνει όταν ένα στοιχείο ενεργοποιείται, για παράδειγμα μέσω κλικ ποντικιού ή πίεσης πλήκτρου. |
| OnClick | 3 | Συμβαίνει όταν το κουμπί της συσκευής κατάδειξης πατιέται πάνω σε ένα στοιχείο. |
| OnMouseDown | 4 | Συμβαίνει όταν το κουμπί της συσκευής κατάδειξης πιέζεται πάνω σε ένα στοιχείο. |
| OnMouseUp | 5 | Συμβαίνει όταν το κουμπί της συσκευής κατάδειξης απελευθερώνεται πάνω σε ένα στοιχείο. |
| OnMouseOver | 6 | Συμβαίνει όταν η συσκευή κατάδειξης μετακινείται πάνω σε ένα στοιχείο. |
| OnMouseMove | 7 | Συμβαίνει όταν η συσκευή κατάδειξης μετακινείται ενώ βρίσκεται πάνω σε ένα στοιχείο. |
| OnMouseOut | 8 | Συμβαίνει όταν η συσκευή κατάδειξης απομακρύνεται από ένα στοιχείο. |
| OnLoad | 9 | Συμβαίνει όταν ο πράκτορας χρήστη έχει αναλύσει πλήρως το στοιχείο και τα απογόνους του καθώς και όλους τους αναφερόμενους πόρους που απαιτούνται για την απόδοσή του. |
| OnUnload | 10 | Συμβαίνει όταν η υλοποίηση [DOM](../../aspose.slides.dom/) αφαιρεί ένα έγγραφο από ένα παράθυρο ή πλαίσιο. Ισχύει μόνο για τα εξωτερικότερα στοιχεία svg. |
| OnAbort | 11 | Συμβαίνει όταν η φόρτωση της σελίδας σταματά πριν ένα στοιχείο επιτραπεί να φορτωθεί πλήρως. |
| OnError | 12 | Συμβαίνει όταν ένα στοιχείο δεν φορτώνεται σωστά ή όταν εμφανίζεται σφάλμα κατά την εκτέλεση του script. |
| OnResize | 13 | Συμβαίνει όταν η προβολή εγγράφου αλλάζει μέγεθος. Ισχύει μόνο για τα εξωτερικότερα στοιχεία svg. |
| OnScroll | 14 | Συμβαίνει όταν η προβολή εγγράφου μετατοπίζεται κατά τον άξονα X ή Y ή και τους δύο. Ισχύει μόνο για τα εξωτερικότερα στοιχεία svg. |
| OnZoom | 15 | Συμβαίνει όταν το επίπεδο ζουμ της προβολής εγγράφου αλλάζει. Ισχύει μόνο για τα εξωτερικότερα στοιχεία svg. |
| OnBegin | 16 | Συμβαίνει όταν ένα στοιχείο animation αρχίζει. |
| OnEnd | 17 | Συμβαίνει όταν ένα στοιχείο animation τελειώνει. |
| OnRepeat | 18 | Συμβαίνει όταν ένα στοιχείο animation επαναλαμβάνεται. |

## SVGOptions::set_DisableLineEndCropping(bool) μέθοδος

Το SVG 1.1 δεν διαθέτει δυνατότητα να ορίζει εσοχές για τους δείκτες. [Aspose.Slides](../../../aspose.slides/) Η μηχανή γραφής SVG έχει λύση για το πρόβλημα αυτό: κόβει το άκρο της γραμμής με βέλος, έτσι η γραμμή δεν επικαλύπτει τους δείκτες. Αυτή η επιλογή απενεργοποιεί τέτοια συμπεριφορά. Γράψτε **bool**.

```cpp
void Aspose::Slides::Export::SVGOptions::set_DisableLineEndCropping(bool value) override
```

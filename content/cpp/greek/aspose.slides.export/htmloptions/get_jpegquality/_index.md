## HtmlOptions::get_JpegQuality() μέθοδος

Επιστρέφει μια τιμή που καθορίζει την ποιότητα των εικόνων JPEG μέσα σε έγγραφο PDF. Ανάγνωση **uint8_t**.

```cpp
uint8_t Aspose::Slides::Export::HtmlOptions::get_JpegQuality() override
```

## Παρατηρήσεις

Έχει αποτέλεσμα μόνο όταν ένα έγγραφο περιέχει εικόνες JPEG.

Χρησιμοποιήστε αυτήν την ιδιότητα για να λάβετε ή να ορίσετε την ποιότητα των εικόνων μέσα σε ένα έγγραφο όταν αποθηκεύεται σε μορφή PDF. Η τιμή μπορεί να κυμαίνεται από 0 έως 100, όπου το 0 σημαίνει χειρότερη ποιότητα αλλά μέγιστη συμπίεση και το 100 σημαίνει καλύτερη ποιότητα αλλά ελάχιστη συμπίεση.

Η προεπιλεγμένη τιμή είναι **95**.

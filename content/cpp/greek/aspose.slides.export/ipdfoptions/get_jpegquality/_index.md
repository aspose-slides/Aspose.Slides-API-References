## IPdfOptions::get_JpegQuality() μέθοδος


Επιστρέφει μια τιμή που καθορίζει την ποιότητα των εικόνων JPEG μέσα σε έγγραφο PDF. Διαβάζει **uint8_t**.

```cpp
virtual uint8_t Aspose::Slides::Export::IPdfOptions::get_JpegQuality()=0
```

## Παρατηρήσεις


Έχει αποτέλεσμα μόνο όταν ένα έγγραφο περιέχει εικόνες JPEG.

Χρησιμοποιήστε αυτήν την ιδιότητα για να λάβετε ή να ορίσετε την ποιότητα των εικόνων μέσα σε ένα έγγραφο κατά την αποθήκευση σε μορφή PDF. Η τιμή μπορεί να κυμαίνεται από 0 έως 100, όπου το 0 σημαίνει τη χαμηλότερη ποιότητα αλλά μέγιστη συμπίεση και το 100 σημαίνει τη υψηλότερη ποιότητα αλλά ελάχιστη συμπίεση.

Η προεπιλεγμένη τιμή είναι **100**.

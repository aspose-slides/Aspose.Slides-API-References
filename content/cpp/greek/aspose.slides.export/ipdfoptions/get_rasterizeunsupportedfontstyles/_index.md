## IPdfOptions::get_RasterizeUnsupportedFontStyles() μέθοδος

Δείχνει εάν το κείμενο πρέπει να rasterize ως bitmap και να αποθηκευτεί σε PDF όταν η γραμματοσειρά δεν υποστηρίζει έντονη μορφοποίηση. Αυτή η προσέγγιση μπορεί να βελτιώσει την ποιότητα του κειμένου στο παραγόμενο PDF για ορισμένες γραμματοσειρές. Διαβάστε **bool**.

```cpp
virtual bool Aspose::Slides::Export::IPdfOptions::get_RasterizeUnsupportedFontStyles()=0
```

## Παρατηρήσεις

Η προεπιλογή είναι **false**.

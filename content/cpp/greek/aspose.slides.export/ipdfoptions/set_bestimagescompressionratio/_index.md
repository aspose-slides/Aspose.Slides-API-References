## IPdfOptions::set_BestImagesCompressionRatio(bool) μέθοδος

Δείχνει εάν η πιο αποδοτική συμπίεση (αντί της προεπιλεγμένης) για κάθε εικόνα πρέπει να επιλέγεται αυτόματα. Εάν οριστεί σε **bool**.true, για κάθε εικόνα στην παρουσίαση θα επιλεγεί ο πιο κατάλληλος αλγόριθμος συμπίεσης, κάτι που θα οδηγήσει σε μικρότερο μέγεθος του τελικού εγγράφου PDF.

```cpp
virtual void Aspose::Slides::Export::IPdfOptions::set_BestImagesCompressionRatio(bool value)=0
```

## Παρατηρήσεις

Η επιλογή του βέλτιστου λόγου συμπίεσης εικόνας είναι υπολογιστικά δαπανηρή και απαιτεί επιπλέον ποσό RAM, και αυτή η επιλογή είναι **bool**.false από προεπιλογή.

Η προεπιλογή είναι **bool**.false.

## IPdfOptions::set_RasterizeUnsupportedFontStyles(bool) μέθοδος

Δείχνει εάν το κείμενο πρέπει να ραστεροποιηθεί ως bitmap και να αποθηκευτεί σε PDF όταν η γραμματοσειρά δεν υποστηρίζει έντονη μορφοποίηση. Αυτή η προσέγγιση μπορεί να βελτιώσει την ποιότητα του κειμένου στο παραγόμενο PDF για ορισμένες γραμματοσειρές. Γράψτε **bool**.

```cpp
virtual void Aspose::Slides::Export::IPdfOptions::set_RasterizeUnsupportedFontStyles(bool value)=0
```

## Παρατηρήσεις

Η προεπιλογή είναι **false**.

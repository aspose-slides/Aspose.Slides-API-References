## IPdfOptions::set_EmbedTrueTypeFontsForASCII(bool) μέθοδος


True για ενσωμάτωση γραμματοσειρών TrueType για χαρακτήρες ASCII 32-127. [Fonts](../../../aspose.slides/fonts/) για κώδικες χαρακτήρων μεγαλύτερους από 127 ενσωματώνονται πάντα. Γράψτε **bool**.

```cpp
virtual void Aspose::Slides::Export::IPdfOptions::set_EmbedTrueTypeFontsForASCII(bool value)=0
```

## Παρατηρήσεις


Η προεπιλογή είναι **true**.

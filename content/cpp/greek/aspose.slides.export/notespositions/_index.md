## NotesPositions enum

Αναπαριστά τον κανόνα για την απόδοση των σημειώσεων στο εξαγόμενο έγγραφο

```cpp
enum class NotesPositions
```

### Values

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| None | 0 | Καθορίζει ότι οι σημειώσεις δεν πρέπει να εμφανίζονται καθόλου. |
| BottomFull | 1 | Καθορίζει ότι οι σημειώσεις πρέπει να εμφανίζονται πλήρως, χρησιμοποιώντας πρόσθετες σελίδες ανάλογα με τις ανάγκες. |
| BottomTruncated | 2 | Καθορίζει ότι οι σημειώσεις πρέπει να εμφανίζονται σε μία μόνο σελίδα. |

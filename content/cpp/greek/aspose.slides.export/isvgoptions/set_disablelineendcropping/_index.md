## ISVGOptions::set_DisableLineEndCropping(bool) μέθοδος


Το SVG 1.1 δεν διαθέτει δυνατότητα ορισμού εσοχών για τα σημεία σήμανσης. [Aspose.Slides](../../../aspose.slides/) Η μηχανή εγγραφής SVG έχει λύση για αυτό το πρόβλημα: περικόπτει το άκρο της γραμμής με το βέλος, ώστε η γραμμή να μην επικαλύπτει τα σημεία σήμανσης. Αυτή η επιλογή απενεργοποιεί τέτοια συμπεριφορά. Γράψτε **bool**.

```cpp
virtual void Aspose::Slides::Export::ISVGOptions::set_DisableLineEndCropping(bool value)=0
```

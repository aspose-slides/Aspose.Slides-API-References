## IHtmlOptions::set_DeletePicturesCroppedAreas(bool) μέθοδος


Μια λογική σημαία υποδεικνύει εάν τα περικομμένα τμήματα παραμένουν ως μέρος του εγγράφου. Αν είναι true, τα περικομμένα τμήματα θα αφαιρεθούν, αν είναι false θα σειριοποιηθούν στο έγγραφο (που μπορεί πιθανώς να οδηγήσει σε μεγαλύτερο αρχείο). Γράψτε **bool**.

```cpp
virtual void Aspose::Slides::Export::IHtmlOptions::set_DeletePicturesCroppedAreas(bool value)=0
```

## has_data_method struct

Ελέγχει αν ένας τύπος έχει τη μέθοδο data(). Αν το κάνει, κληρονομεί το std::true_type, διαφορετικά κληρονομεί το std::false_type.

```cpp
template<typename T,typename Enable>class has_data_method : public std::false_type
```

### Template parameters

| Παράμετρος | Περιγραφή |
| --- | --- |
| T | Τύπος για έλεγχο. |
| Enable | Ορίσμα για τη λειτουργία του SFINAE. |

## HttpContent κλάση

Αντιπροσωπεύει το περιεχόμενο μιας οντότητας HTTP. [Object](../../system/object/) αυτής της κλάσης θα πρέπει να εκχωρείται μόνο χρησιμοποιώντας τη συνάρτηση [System::MakeObject()](../../system/makeobject/). Ποτέ μη δημιουργείτε ένα αντικείμενο αυτού του τύπου στο stack ή χρησιμοποιώντας τον τελεστή new, καθώς αυτό θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης και/ή αποτυχίες ελέγχου. Πάντοτε τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να τον περάσετε σε συναρτήσεις ως όρισμα.

```cpp
class HttpContent : public System::IDisposable
```

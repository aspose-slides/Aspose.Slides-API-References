## InstalledFontCollection κλάση

Αντιπροσωπεύει τη συλλογή των γραμματοσειρών που είναι εγκατεστημένες στο σύστημα. NOT IMPLEMENTED. Τα αντικείμενα αυτής της κλάσης πρέπει να εκχωρούνται μόνο με τη λειτουργία [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε στιγμιότυπο αυτού του τύπου στη στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς αυτό θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης και/ή σφάλματα ελέγχου. Πάντα τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να τη περάσετε σε συναρτήσεις ως όρισμα.

```cpp
class InstalledFontCollection : public System::Drawing::Text::FontCollection
```

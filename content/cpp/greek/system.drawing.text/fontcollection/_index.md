## FontCollection κλάση

Μία βασική κλάση για εγκατεστημένες και ιδιωτικές συλλογές γραμματοσειρών. Αντικείμενα αυτής της κλάσης πρέπει να εκχωρούνται μόνο με τη συνάρτηση [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε στιγμιότυπο αυτού του τύπου στη στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς αυτό θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης και/ή σφάλματα δήλωσης. Πάντα να ενσαρκώνετε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και να χρησιμοποιείτε αυτόν τον δείκτη για να τον περάσετε στις συναρτήσεις ως όρισμα.

```cpp
class FontCollection : public System::IDisposable
```

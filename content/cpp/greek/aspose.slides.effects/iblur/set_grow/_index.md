## IBlur::set_Grow(bool) μέθοδος

Καθορίζει εάν τα όρια του αντικειμένου πρέπει να αυξηθούν ως αποτέλεσμα του θόλωσης. Η τιμή True υποδεικνύει ότι τα όρια αυξάνονται ενώ η τιμή false υποδεικνύει ότι δεν αυξάνονται. Γράψτε **bool**.

virtual void Aspose::Slides::Effects::IBlur::set_Grow(bool value)=0

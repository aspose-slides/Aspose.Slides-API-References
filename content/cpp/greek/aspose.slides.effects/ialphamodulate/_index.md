## IAlphaModulate κλάση

Αναπαριστά ένα εφέ Alpha Modulate. Οι τιμές του alpha (διαφάνειας) του εφέ πολλαπλασιάζονται με ένα σταθερό ποσοστό. Το δοχείο εφέ καθορίζει ένα εφέ που περιέχει τιμές alpha προς διαμόρφωση.

```cpp
class IAlphaModulate : public virtual Aspose::Slides::Effects::IImageTransformOperation,
                       public Aspose::Slides::IAccessiblePVIObject<System::SharedPtr<Aspose::Slides::Effects::IAlphaModulateEffectiveData>>
```

## FillOverlay κλάση

Αντιπροσωπεύει ένα εφέ Fill Overlay. Ένα fill overlay μπορεί να χρησιμοποιηθεί για να καθορίσει μια πρόσθετη γεμιστική για ένα αντικείμενο και να συνδυάσει τα δύο γεμίσματα μεταξύ τους.

```cpp
class FillOverlay : public Aspose::Slides::Effects::ImageTransformOperation,
                    public Aspose::Slides::Effects::IFillOverlay,
                    public Aspose::Slides::Effects::IVisualEffect
```

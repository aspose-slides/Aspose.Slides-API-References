## IAlphaCeiling κλάση

Αναπαριστά ένα εφέ Alpha Ceiling. Οι τιμές Alpha (αδιαφάνειας) μεγαλύτερες του μηδενός αλλάζουν στο 100%. Με άλλα λόγια, οτιδήποτε είναι μερικώς αδιαφανές γίνεται πλήρως αδιαφανές.

```cpp
class IAlphaCeiling : public virtual Aspose::Slides::Effects::IImageTransformOperation,
                      public Aspose::Slides::IAccessiblePVIObject<System::SharedPtr<Aspose::Slides::Effects::IAlphaCeilingEffectiveData>>
```

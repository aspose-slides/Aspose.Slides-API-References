## AlphaCeiling κλάση

Αναπαριστά ένα εφέ Alpha Ceiling. Τιμές Alpha (διαφάνεια) μεγαλύτερες του μηδενός μετατρέπονται στο 100%. Με άλλα λόγια, οτιδήποτε είναι μερικώς αδιαφανές γίνεται πλήρως αδιαφανές.

```cpp
class AlphaCeiling : public Aspose::Slides::Effects::ImageTransformOperation,
                     public Aspose::Slides::Effects::IAlphaCeiling,
                     public Aspose::Slides::Effects::IVisualEffect
```

## IAlphaCeilingEffectiveData κλάση

Αμετάβλητο αντικείμενο που αντιπροσωπεύει το αποτέλεσμα Alpha Ceiling. Οι τιμές Alpha (αδιαφάνειας) μεγαλύτερες από το μηδέν αλλάζουν σε 100%. Με άλλα λόγια, ό,τι είναι μερικώς αδιαφανές γίνεται πλήρως αδιαφανές.

```cpp
class IAlphaCeilingEffectiveData : public virtual Aspose::Slides::Effects::IEffectEffectiveData
```

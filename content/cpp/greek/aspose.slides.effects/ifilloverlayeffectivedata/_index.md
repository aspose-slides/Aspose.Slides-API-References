## IFillOverlayEffectiveData κλάση


Αμετάβλητο αντικείμενο που αντιπροσωπεύει ένα εφέ Fill Overlay. Ένα fill overlay μπορεί να χρησιμοποιηθεί για τον καθορισμό πρόσθετης γεμίσματος για ένα αντικείμενο και την ανάμειξη των δύο γεμισμάτων μαζί.

```cpp
class IFillOverlayEffectiveData : public virtual Aspose::Slides::Effects::IEffectEffectiveData
```

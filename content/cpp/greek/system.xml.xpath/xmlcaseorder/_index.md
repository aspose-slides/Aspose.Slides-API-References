## XmlCaseOrder enum


Καθορίζει τη σειρά ταξινόμησης για κεφαλαία και μικρά γράμματα.

```cpp
enum class XmlCaseOrder
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| None | 0 | Αγνοεί τη διάκριση πεζών/κεφαλαίων. |
| UpperFirst | 1 | Τα κεφαλαία γράμματα ταξινομούνται πριν από τα μικρά γράμματα. |
| LowerFirst | 2 | Τα μικρά γράμματα ταξινομούνται πριν από τα κεφαλαία γράμματα. |

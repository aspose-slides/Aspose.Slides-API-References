## XPathNavigator::LookupPrefix(const String\&) μέθοδος

Επιστρέφει το πρόθεμα που δηλώθηκε για το καθορισμένο URI ονόματος χώρου.

```cpp
String System::Xml::XPath::XPathNavigator::LookupPrefix(const String &namespaceURI) override
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| namespaceURI | const [String](../../../system/string/)\& | Το URI του ονόματος χώρου που πρέπει να επιλυθεί για το πρόθεμα. |

### Τιμή Επιστροφής

Ένα [String](../../../system/string/) που περιέχει το πρόθεμα του ονόματος χώρου που έχει εκχωρηθεί στο καθορισμένο URI ονόματος χώρου· διαφορετικά, [String::Empty](../../../system/string/empty/) εάν δεν έχει εκχωρηθεί πρόθεμα στο καθορισμένο URI ονόματος χώρου. Το [String](../../../system/string/) που επιστρέφεται είναι ατομικό.

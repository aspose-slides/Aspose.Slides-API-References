## XPathNavigator::InsertElementAfter(String, String, String, String) μέθοδος


Δημιουργεί ένα νέο στοιχείο αδέλφου μετά τον τρέχοντα κόμβο χρησιμοποιώντας το πρόθεμα ονοματοχώρου, το τοπικό όνομα και το URI του ονοματοχώρου που έχουν καθοριστεί, με την καθορισμένη τιμή.

```cpp
virtual void System::Xml::XPath::XPathNavigator::InsertElementAfter(String prefix, String localName, String namespaceURI, String value)
```


### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| prefix | [String](../../../system/string/) | Το πρόθεμα ονοματοχώρου του νέου στοιχείου-παιδιού (εάν υπάρχει). |
| localName | [String](../../../system/string/) | Το τοπικό όνομα του νέου στοιχείου-παιδιού (εάν υπάρχει). |
| namespaceURI | [String](../../../system/string/) | Το URI ονοματοχώρου του νέου στοιχείου-παιδιού (εάν υπάρχει). [String::Empty](../../../system/string/empty/) και **nullptr** είναι ισοδύναμα. |
| value | [String](../../../system/string/) | Η τιμή του νέου στοιχείου-παιδιού. Εάν [String::Empty](../../../system/string/empty/) ή **nullptr** περαστούν, δημιουργείται ένα κενό στοιχείο. |

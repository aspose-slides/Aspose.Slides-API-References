## XPathNavigator::InsertElementBefore(String, String, String, String) μέθοδος

Δημιουργεί ένα νέο στοιχείο αδερφό πριν από τον τρέχοντα κόμβο χρησιμοποιώντας το πρόθεμα χώρου ονομάτων, το τοπικό όνομα και το URI του χώρου ονομάτων που έχουν οριστεί, με την καθορισμένη τιμή.

```cpp
virtual void System::Xml::XPath::XPathNavigator::InsertElementBefore(String prefix, String localName, String namespaceURI, String value)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| prefix | [String](../../../system/string/) | Το πρόθεμα χώρου ονομάτων του νέου στοιχείου τέκνου (εάν υπάρχει). |
| localName | [String](../../../system/string/) | Το τοπικό όνομα του νέου στοιχείου τέκνου (εάν υπάρχει). |
| namespaceURI | [String](../../../system/string/) | Το URI του χώρου ονομάτων του νέου στοιχείου τέκνου (εάν υπάρχει). [String::Empty](../../../system/string/empty/) και **nullptr** είναι ισοδύναμα. |
| value | [String](../../../system/string/) | Η τιμή του νέου στοιχείου τέκνου. Εάν περαστεί [String::Empty](../../../system/string/empty/) ή **nullptr**, δημιουργείται ένα κενό στοιχείο. |

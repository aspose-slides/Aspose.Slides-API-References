## XPathNavigator::MoveToNamespace(String) μέθοδος

Moves the [XPathNavigator](../) to the namespace node with the specified namespace prefix.

```cpp
virtual bool System::Xml::XPath::XPathNavigator::MoveToNamespace(String name)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| name | [String](../../../system/string/) | Το πρόθεμα χώρου ονομάτων του κόμβου χώρου ονομάτων. |

### Τιμή Επιστροφής

**true** αν το [XPathNavigator](../) είναι επιτυχής στη μετακίνηση στο καθορισμένο χώρο ονομάτων· **false** εάν δεν βρέθηκε αντίστοιχο κόμβο χώρου ονομάτων, ή εάν το [XPathNavigator](../) δεν είναι τοποθετημένο σε κόμβο στοιχείου. Εάν **false**, η θέση του [XPathNavigator](../) παραμένει αμετάβλητη.

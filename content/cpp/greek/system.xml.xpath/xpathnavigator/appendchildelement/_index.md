## XPathNavigator::AppendChildElement(String, String, String, String) μέθοδος

Δημιουργεί έναν νέο κόμβο στοιχείου παιδιού στο τέλος της λίστας των θυγών κόμβων του τρέχοντος κόμβου, χρησιμοποιώντας το πρόθεμα ονομαχώρου, το τοπικό όνομα και το URI ονομαχώρου που καθορίζονται με την τιμή που δίνεται.

```cpp
virtual void System::Xml::XPath::XPathNavigator::AppendChildElement(String prefix, String localName, String namespaceURI, String value)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| prefix | [String](../../../system/string/) | Το πρόθεμα ονομαχώρου του νέου κόμβου στοιχείου παιδιού (αν υπάρχει). |
| localName | [String](../../../system/string/) | Το τοπικό όνομα του νέου κόμβου στοιχείου παιδιού (αν υπάρχει). |
| namespaceURI | [String](../../../system/string/) | Το URI ονομαχώρου του νέου κόμβου στοιχείου παιδιού (αν υπάρχει). [String::Empty](../../../system/string/empty/) και **nullptr** είναι ισοδύναμα. |
| value | [String](../../../system/string/) | Η τιμή του νέου κόμβου στοιχείου παιδιού. Αν [String::Empty](../../../system/string/empty/) ή **nullptr** περαστούν, δημιουργείται ένα κενό στοιχείο. |

## XPathNavigator::get_NamespaceURI() μέθοδος

Όταν παρακαμφθεί σε μια παράγωγη κλάση, λαμβάνει το URI του ονοματοχώρου του τρέχοντος κόμβου.

```cpp
virtual String System::Xml::XPath::XPathNavigator::get_NamespaceURI()=0
```

### Τιμή επιστροφής

Ένα [String](../../../system/string/) που περιέχει το URI του ονοματοχώρου του τρέχοντος κόμβου, ή [String::Empty](../../../system/string/empty/) εάν ο τρέχων κόμβος δεν έχει URI ονοματοχώρου.

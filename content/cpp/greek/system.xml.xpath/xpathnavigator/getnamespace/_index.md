## XPathNavigator::GetNamespace(String) μέθοδος

Επιστρέφει την τιμή του κόμβου namespace που αντιστοιχεί στο καθορισμένο τοπικό όνομα.

```cpp
virtual String System::Xml::XPath::XPathNavigator::GetNamespace(String name)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| name | [String](../../../system/string/) | Το τοπικό όνομα του κόμβου namespace. |

### Τιμή Επιστροφής

Ένα [String](../../../system/string/) που περιέχει την τιμή του κόμβου namespace· [String::Empty](../../../system/string/empty/) εάν δεν βρεθεί αντίστοιχος κόμβος namespace ή εάν το [XPathNavigator](../) δεν είναι τοποθετημένο σε κόμβο στοιχείου.

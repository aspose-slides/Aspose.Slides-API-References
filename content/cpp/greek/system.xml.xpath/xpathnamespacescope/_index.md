## XPathNamespaceScope enum


Ορίζει την εμβέλεια του ονομάτων χώρου.

```cpp
enum class XPathNamespaceScope
```

### Values

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| All | 0 | Επιστρέφει όλα τα ονόματα χώρων που ορίζονται στην εμβέλεια του τρέχοντος κόμβου. Αυτό περιλαμβάνει το όνομα χώρου **xmlns:xml** το οποίο δηλώνεται πάντα έμματα. Η σειρά των επιστρεφόμενων ονομάτων χώρων δεν ορίζεται. |
| ExcludeXml | 1 | Επιστρέφει όλα τα ονόματα χώρων που ορίζονται στην εμβέλεια του τρέχοντος κόμβου, εξαιρουμένου του ονόματος χώρου **xmlns:xml**. Το όνομα χώρου **xmlns:xml** δηλώνεται πάντα έμματα. Η σειρά των επιστρεφόμενων ονομάτων χώρων δεν ορίζεται. |
| Local | 2 | Επιστρέφει όλα τα ονόματα χώρων που ορίζονται τοπικά στο τρέχον κόμβο. |

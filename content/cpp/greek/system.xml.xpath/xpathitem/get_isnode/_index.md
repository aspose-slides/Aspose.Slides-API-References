## XPathItem::get_IsNode() μέθοδος


Όταν παρακαμφθεί σε μια παράγωγη κλάση, επιστρέφει μια τιμή που υποδεικνύει εάν το στοιχείο αντιπροσωπεύει έναν [XPath](../../) κόμβο ή μια ατομική τιμή.

```cpp
virtual bool System::Xml::XPath::XPathItem::get_IsNode()=0
```


### Τιμή Επιστροφής

**true** εάν το στοιχείο αντιπροσωπεύει έναν [XPath](../../) κόμβο· **false** εάν το στοιχείο αντιπροσωπεύει μια ατομική τιμή.

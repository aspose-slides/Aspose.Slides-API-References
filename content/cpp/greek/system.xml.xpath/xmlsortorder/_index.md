## XmlSortOrder enum

Καθορίζει τη σειρά ταξινόμησης.

```cpp
enum class XmlSortOrder
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Ascending | 1 | Οι κόμβοι ταξινομούνται σε αυξανόμενη σειρά. Για παράδειγμα, εάν οι αριθμοί 1,2,3 και 4 ταξινομηθούν σε αυξανόμενη σειρά, εμφανίζονται ως 1,2,3,4. |
| Descending | 2 | Οι κόμβοι ταξινομούνται σε φθίνουσα σειρά. Για παράδειγμα, εάν οι αριθμοί 1,2,3 και 4 ταξινομηθούν σε φθίνουσα σειρά, εμφανίζονται ως 4,3,2,1. |

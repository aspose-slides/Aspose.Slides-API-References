## BaseOverrideThemeManager::get_IsOverrideThemeEnabled() μέθοδος

Καθορίζει αν [OverrideTheme](../../overridetheme/) παρακάμπτει το κληρονομημένο αποτελεσματικό θέμα ή όχι. Για να ενεργοποιήσετε [OverrideTheme](../../overridetheme/) για παράκαμψη χρησιμοποιήστε τις μεθόδους OverrideTheme.Init*(). Για να απενεργοποιήσετε [OverrideTheme](../../overridetheme/) από την παράκαμψη χρησιμοποιήστε τη μέθοδο [OverrideTheme::Clear](../../overridetheme/clear/). Μόνο για ανάγνωση **bool**.

```cpp
bool Aspose::Slides::Theme::BaseOverrideThemeManager::get_IsOverrideThemeEnabled() override
```

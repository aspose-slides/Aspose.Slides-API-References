## IMathBox::set_OperatorEmulator(bool) μέθοδος

Προσομοιωτής τελεστή. Όταν είναι true, το κουτί και τα περιεχόμενά του συμπεριφέρονται ως ένας μόνο τελεστής και κληρονομούν τις ιδιότητες ενός τελεστή. Αυτό σημαίνει, για παράδειγμα, ότι ο χαρακτήρας μπορεί να λειτουργήσει ως σημείο για αλλαγή γραμμής και μπορεί να ευθυγραμμιστεί με άλλους τελεστές. Οι προσομοιωτές τελεστή χρησιμοποιούνται συχνά όταν ένας ή περισσότεροι γλύφοι συνδυάζονται για να σχηματίσουν έναν τελεστή, όπως το '=='. Προεπιλεγμένη τιμή: false

```cpp
virtual void Aspose::Slides::MathText::IMathBox::set_OperatorEmulator(bool value)=0
```

## Σχόλια

Παράδειγμα:
```cpp
auto box = System::MakeObject<MathematicalText>(u"==")->ToBox();
box->set_OperatorEmulator(true);
```

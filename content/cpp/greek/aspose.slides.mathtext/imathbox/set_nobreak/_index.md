## IMathBox::set_NoBreak(bool) μέθοδος

Χωρίς διακοπή. Αυτή η ιδιότητα καθορίζει την \"αδιάσπαστη\" ιδιότητα στο κουτί αντικειμένου. Όταν είναι true, δεν μπορούν να συμβούν διακοπές γραμμής μέσα στο κουτί. Αυτό μπορεί να είναι σημαντικό για εξομοιωτές τελεστών που αποτελούνται από περισσότερους από έναν δυαδικό τελεστή. Όταν αυτό το στοιχείο δεν καθορίζεται, μπορούν να συμβούν διακοπές μέσα στο κουτί. Προεπιλογή: true

```cpp
virtual void Aspose::Slides::MathText::IMathBox::set_NoBreak(bool value)=0
```

## Παρατηρήσεις

Παράδειγμα:
```cpp
auto box = System::MakeObject<MathematicalText>(u"**********")->ToBox();
box->set_NoBreak(false);
```

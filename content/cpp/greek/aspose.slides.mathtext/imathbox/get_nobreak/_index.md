## IMathBox::get_NoBreak() μέθοδος

Καμία διάσπαση. Αυτή η ιδιότητα καθορίζει την ιδιότητα \"unbreakable\" στο πλαίσιο αντικειμένου. Όταν είναι αληθής, δεν μπορούν να εμφανιστούν αλλαγές γραμμής μέσα στο πλαίσιο. Αυτό μπορεί να είναι σημαντικό για προσομοιωτές τελεστών που αποτελούνται από περισσότερο από έναν δυαδικό τελεστή. Όταν αυτό το στοιχείο δεν καθορίζεται, μπορούν να εμφανιστούν αλλαγές μέσα στο πλαίσιο. Προεπιλογή: true

```cpp
virtual bool Aspose::Slides::MathText::IMathBox::get_NoBreak()=0
```

## Παρατηρήσεις

Παράδειγμα:
```cpp
auto box = System::MakeObject<MathematicalText>(u"**********")->ToBox();
box->set_NoBreak(false);
```

## MathBox::set_OperatorEmulator(bool) μέθοδος

Προσομοιωτής τελεστή. Όταν είναι true, το κουτί και τα περιεχόμενά του λειτουργούν ως ένας ενιαίος τελεστής και κληρονομούν τις ιδιότητες ενός τελεστή. Αυτό σημαίνει, για παράδειγμα, ότι ο χαρακτήρας μπορεί να χρησιμεύσει ως σημείο για μια διάσπαση γραμμής και μπορεί να ευθυγραμμιστεί με άλλους τελεστές. Οι Προσομοιωτές τελεστή χρησιμοποιούνται συχνά όταν ένας ή περισσότεροι γλύφοι συνδυάζονται για να σχηματίσουν έναν τελεστή, όπως το '=='. Προεπιλεγμένη τιμή: false

```cpp
void Aspose::Slides::MathText::MathBox::set_OperatorEmulator(bool value) override
```

## Παρατηρήσεις

Παράδειγμα: 
```cpp
auto box = System::MakeObject<MathBox>(System::MakeObject<MathematicalText>(u"=="));
box->set_OperatorEmulator(true);
```

## MathBox::set_NoBreak(bool) μέθοδος

Χωρίς διακοπή Αυτή η ιδιότητα προσδιορίζει την ιδιότητα "unbreakable" στο πλαίσιο του αντικειμένου. Όταν είναι true, δεν μπορούν να συμβούν αλλαγές γραμμής μέσα στο πλαίσιο. Αυτό μπορεί να είναι σημαντικό για εξομοιωτές τελεστών που αποτελούνται από περισσότερους από έναν δυαδικό τελεστή. Όταν αυτό το στοιχείο δεν καθορίζεται, μπορεί να συμβούν αλλαγές μέσα στο πλαίσιο. Προεπιλογή: true

```cpp
void Aspose::Slides::MathText::MathBox::set_NoBreak(bool value) override
```

## Παρατηρήσεις

Παράδειγμα:
```cpp
auto box = System::MakeObject<MathBox>(System::MakeObject<MathematicalText>(u"*****"));
box->set_NoBreak(false);
```

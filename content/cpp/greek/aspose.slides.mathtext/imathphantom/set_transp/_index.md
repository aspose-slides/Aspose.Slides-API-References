## IMathPhantom::set_Transp(bool) μέθοδος


Ορίζει μια τιμή που υποδεικνύει εάν το φανταστικό είναι διαφανές για κανόνες διάστιχου βάσει κλάσης.

```cpp
virtual void Aspose::Slides::MathText::IMathPhantom::set_Transp(bool value)=0
```

## Παρατηρήσεις


Όταν **true**, οι τελεστές και τα σύμβολα μέσα στο φανταστικό εξακολουθούν να επηρεάζουν το μαθηματικό διάστιχο γύρω από το φανταστικό (σαν να είναι ορατό). Όταν **false**, η διάταξη βάσει κλάσης αγνοείται. Αντιστοιχεί στο χαρακτηριστικό OMML **m:transp**.

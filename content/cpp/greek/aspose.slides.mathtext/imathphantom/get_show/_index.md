## IMathPhantom::get_Show() μέθοδος

Επιστρέφει μια τιμή που υποδεικνύει εάν το βασικό στοιχείο εμφανίζεται.

```cpp
virtual bool Aspose::Slides::MathText::IMathPhantom::get_Show()=0
```

## Σχόλια

Όταν **false**, το βασικό στοιχείο είναι κρυμμένο αλλά μπορεί ακόμα να καταλαμβάνει χώρο ανάλογα με άλλες ρυθμίσεις phantom. Αντιστοιχεί στο χαρακτηριστικό OMML **m:show**.

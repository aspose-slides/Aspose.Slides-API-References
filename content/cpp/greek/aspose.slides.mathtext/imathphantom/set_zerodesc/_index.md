## IMathPhantom::set_ZeroDesc(bool) μέθοδος


Ορίζει μια τιμή που υποδεικνύει εάν η καταβάθμιση (βάθος κάτω από τη γραμμή βάσης) του στοιχείου βάσης πρέπει να θεωρείται μηδέν.

```cpp
virtual void Aspose::Slides::MathText::IMathPhantom::set_ZeroDesc(bool value)=0
```

## Παρατηρήσεις


Όταν **true**, το φάντασμα δεν χαμηλώνει τη γραμμή βάσης της περιβάλλουσας μαθηματικής γραμμής. Ανταποκρίνεται στο χαρακτηριστικό OMML **m:zeroDesc**.

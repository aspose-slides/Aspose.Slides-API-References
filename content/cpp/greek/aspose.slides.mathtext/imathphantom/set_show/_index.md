## IMathPhantom::set_Show(bool) μέθοδος

Ορίζει μια τιμή που υποδεικνύει εάν το βασικό στοιχείο εμφανίζεται.

```cpp
virtual void Aspose::Slides::MathText::IMathPhantom::set_Show(bool value)=0
```

## Παρατηρήσεις

Όταν **false**, το βασικό στοιχείο είναι κρυφό αλλά μπορεί ακόμα να καταλαμβάνει χώρο ανάλογα με άλλες ρυθμίσεις phantom. Συμφωνεί με το χαρακτηριστικό OMML **m:show**.

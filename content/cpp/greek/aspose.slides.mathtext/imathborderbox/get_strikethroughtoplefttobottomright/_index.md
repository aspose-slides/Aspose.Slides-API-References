## IMathBorderBox::get_StrikethroughTopLeftToBottomRight() μέθοδος


Strikethrough Top-Left to Bottom-Right (default is false). Καθορίζει την κρυμμένη ή εμφανιζόμενη κατάσταση μιας διαγώνιας γραμμής διαγράμμισης από την επάνω αριστερή γωνία έως την κάτω δεξιά γωνία του πλαισίου περιγράμματος.

```cpp
virtual bool Aspose::Slides::MathText::IMathBorderBox::get_StrikethroughTopLeftToBottomRight()=0
```

## Σχόλια


Παράδειγμα: 
```cpp
auto borderBox = System::MakeObject<MathematicalText>(u"x+y+z")->ToBorderBox();
borderBox->set_StrikethroughTopLeftToBottomRight(true);
```

## MathMatrix::set_MinColumnWidth(uint32_t) μέθοδος

Ελάχιστο πλάτος στήλης σε twips (1/20 του σημείου) Η απόσταση κεντού (επίσης αναφέρεται ως “Column Gap” ή “Gap Width”) προστίθεται στο MinColumnWidth για να καθοριστεί η συνολική Matrix [Column](../../../aspose.slides/column/) Απόσταση (απόσταση μεταξύ των ίδιων άκρων διαφορετικών στηλών). Προεπιλογή: 0.

```cpp
void Aspose::Slides::MathText::MathMatrix::set_MinColumnWidth(uint32_t value) override
```

## Παρατηρήσεις

Παράδειγμα: 
```cpp
auto matrix = System::MakeObject<MathMatrix>(2, 3);
matrix->set_MinColumnWidth(20);
```

## MathMatrix::get_MinColumnWidth() μέθοδος

Ελάχιστο πλάτος στήλης σε twips (1/20th of a point) Το διάστημα κενού (επίσης αναφέρεται ως \\u201CColumn Gap\\u201D ή \\u201CGap Width\\u201D) προστίθεται στο MinColumnWidth για να καθορίσει το συνολικό Matrix [Column](../../../aspose.slides/column/) Spacing (απόσταση μεταξύ των ίδιων άκρων διαφορετικών στηλών). Προεπιλογή: 0.

```cpp
uint32_t Aspose::Slides::MathText::MathMatrix::get_MinColumnWidth() override
```

## Παρατηρήσεις

Παράδειγμα:
```cpp
auto matrix = System::MakeObject<MathMatrix>(2, 3);
matrix->set_MinColumnWidth(20);
```

## IMathMatrix::get_MinColumnWidth() method


Ελάχιστο πλάτος στήλης σε twips (1/20 του σημείου) Η απόσταση του κενόμενου (επίσης αναφέρεται ως \\u201CColumn Gap\\u201D ή \\u201CGap Width\\u201D) προστίθεται στο MinColumnWidth για να καθορίσει το συνολικό Matrix [Column](../../../aspose.slides/column/) Spacing (απόσταση μεταξύ των ίδιων άκρων διαφορετικών στηλών). Προεπιλογή: 0.

```cpp
virtual uint32_t Aspose::Slides::MathText::IMathMatrix::get_MinColumnWidth()=0
```

## Σχόλια


Παράδειγμα: 
```cpp
auto matrix = System::MakeObject<MathMatrix>(2, 3);
matrix->set_MinColumnWidth(20);
```

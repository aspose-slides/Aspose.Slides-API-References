## IMathMatrix::set_MinColumnWidth(uint32_t) μέθοδος


Ελάχιστο πλάτος στήλης σε twips (1/20 του σημείου) Η απόσταση του κενού (επίσης αναφέρεται ως \\u201CColumn Gap\\u201D ή \\u201CGap Width\\u201D) προστίθεται στο MinColumnWidth για να προσδιοριστεί το συνολικό Matrix [Column](../../../aspose.slides/column/) Spacing (απόσταση μεταξύ των ίδιων άκρων διαφορετικών στηλών). Προεπιλογή: 0.

```cpp
virtual void Aspose::Slides::MathText::IMathMatrix::set_MinColumnWidth(uint32_t value)=0
```

## Παρατηρήσεις


Παράδειγμα: 
```cpp
auto matrix = System::MakeObject<MathMatrix>(2, 3);
matrix->set_MinColumnWidth(20);
```

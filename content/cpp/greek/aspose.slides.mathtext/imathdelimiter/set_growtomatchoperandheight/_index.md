## IMathDelimiter::set_GrowToMatchOperandHeight(bool) μέθοδος

Καθορίζει την ανάπτυξη των BeginningCharacter, SeparatorCharacter, EndingCharacter. Όταν είναι true, τα διαχωριστικά μεγαλώνουν κάθετα ώστε να ταιριάζουν με το ύψος του τελεστή. Η προεπιλεγμένη τιμή είναι true

```cpp
virtual void Aspose::Slides::MathText::IMathDelimiter::set_GrowToMatchOperandHeight(bool value)=0
```

## Παρατηρήσεις

```cpp
auto delimiter = System::MakeObject<MathematicalText>(u"x")->Divide(u"y")->Enclose();
delimiter->set_GrowToMatchOperandHeight(false);
```

## MathBorderBox::set_StrikethroughTopLeftToBottomRight(bool) μέθοδος

Strikethrough Top-Left to Bottom-Right (προεπιλογή είναι false). Καθορίζει την κρυφή ή εμφανή κατάσταση μιας διαγώνιας γραμμής διαγράμμισης από την επάνω αριστερή γωνία στην κάτω δεξιά γωνία του πλαίσιου περιθωρίου.

```cpp
void Aspose::Slides::MathText::MathBorderBox::set_StrikethroughTopLeftToBottomRight(bool value) override
```

## Παρατηρήσεις

Παράδειγμα: 
```cpp
auto borderBox = System::MakeObject<MathBorderBox>(System::MakeObject<MathematicalText>(u"x"));
borderBox->set_StrikethroughTopLeftToBottomRight(true);
```

## MathPhantom::set_ZeroDesc(bool) μέθοδος

Ορίζει μια τιμή που υποδεικνύει εάν η κατάβαση (βάθος κάτω από τη γραμμή βάσης) του βασικού στοιχείου πρέπει να θεωρηθεί μηδενική.

```cpp
void Aspose::Slides::MathText::MathPhantom::set_ZeroDesc(bool value) override
```

## Σχόλια

Όταν **true**, το φάντασμα δεν χαμηλώνει τη γραμμή βάσης του περιβάλλοντος μαθηματικού κειμένου. Αντιστοιχεί στο χαρακτηριστικό OMML **m:zeroDesc**.

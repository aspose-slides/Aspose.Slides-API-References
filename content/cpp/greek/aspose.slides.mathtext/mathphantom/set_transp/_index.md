## MathPhantom::set_Transp(bool) μέθοδος

Ορίζει μια τιμή που υποδεικνύει εάν το φάντασμα είναι διαφανές για κανόνες διάστιξης βάσει κλάσης.

```cpp
void Aspose::Slides::MathText::MathPhantom::set_Transp(bool value) override
```

## Παρατηρήσεις

Όταν **true**, οι τελεστές και τα σύμβολα μέσα στο φάντασμα εξακολουθούν να επηρεάζουν τη μαθηματική διάστιξη γύρω από το φάντασμα (σαν να είναι ορατό). Όταν **false**, η διάστιξη βάσει κλάσης αγνοείται. Αντιστοιχεί στο χαρακτηριστικό OMML **m:transp**.

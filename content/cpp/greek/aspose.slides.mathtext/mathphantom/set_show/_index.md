## MathPhantom::set_Show(bool) μέθοδος

Ορίζει μια τιμή που υποδεικνύει αν το βασικό στοιχείο εμφανίζεται.

```cpp
void Aspose::Slides::MathText::MathPhantom::set_Show(bool value) override
```

## Παρατηρήσεις

Όταν **false**, το βασικό στοιχείο είναι κρυμμένο αλλά μπορεί ακόμη να καταλαμβάνει χώρο ανάλογα με άλλες ρυθμίσεις phantom. Αντιστοιχεί στο χαρακτηριστικό OMML **m:show**.

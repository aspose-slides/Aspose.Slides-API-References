## MathPhantom::set_ZeroAsc(bool) μέθοδος

Ορίζει μια τιμή που υποδεικνύει αν η άνοδος (ύψος πάνω από τη γραμμή βάσης) του βασικού στοιχείου θα πρέπει να θεωρείται μηδενική.

```cpp
void Aspose::Slides::MathText::MathPhantom::set_ZeroAsc(bool value) override
```

## Παρατηρήσεις

Όταν **true**, το φάντασμα δεν αυξάνει τη γραμμή βάσης της περιβάλλουσας μαθηματικής γραμμής. Αντιστοιχεί στο χαρακτηριστικό OMML **m:zeroAsc**.

## MathSpacingRules enum

Τύποι κενών (οριζόντια απόσταση) μεταξύ των στηλών ενός πίνακα

```cpp
enum class MathSpacingRules
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| SingleSpacingGap | 0 | Μονή (1 em για στήλες ή 1 γραμμή για γραμμές) |
| OneAndHalfSpacingGap | 1 | Ένα και μισό (1.5 ems για στήλες ή 1.5 γραμμές για γραμμές) |
| DoubleSpacingGap | 2 | Διπλή απόσταση (2 ems για στήλες ή 2 γραμμές για γραμμές) |
| Exactly | 3 | Ακριβής τιμή Για στήλες, βασίζεται στην τιμή του ColumnGap, μετρημένη σε twips Για γραμμές, βασίζεται στην τιμή του RowGap, μετρημένη σε twips |
| Multiple | 4 | Πολλαπλή Για στήλες, βασίζεται στην τιμή του ColumnGap, μετρημένη σε βήματα 0.5 em Για γραμμές, βασίζεται στην τιμή του RowGap, μετρημένη σε γραμμές |

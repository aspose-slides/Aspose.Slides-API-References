## EnumeratorSelectManyAdapter::MoveNext() μέθοδος

Μετακινεί τον enumerator στο επόμενο στοιχείο. Εάν δεν υπήρχε προηγούμενη αναφορά σε στοιχείο, ορίζει την αναφορά στο πρώτο διαθέσιμο στοιχείο. Αν έχει φτάσει στο τέλος του container, δεν κάνει τίποτα.

```cpp
bool System::Collections::Generic::Details::EnumeratorSelectManyAdapter<Source, Result>::MoveNext() override
```

### Τιμή Επιστροφής

True εάν κάποιο στοιχείο είναι διαθέσιμο μετά την κλήση, false διαφορετικά.

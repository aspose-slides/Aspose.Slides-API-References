## EnumeratorCastAdapter::MoveNext() μέθοδος

Μετακινεί τον απαριθμητή στο επόμενο στοιχείο. Εάν δεν είχε γίνει αναφορά σε κανένα στοιχείο προηγουμένως, θέτει την αναφορά στο πρώτο διαθέσιμο στοιχείο. Εάν έχει φθάσει το τέλος του container, δεν κάνει τίποτα.

```cpp
bool System::Collections::Generic::Details::EnumeratorCastAdapter<Source, Result>::MoveNext() override
```

### Τιμή Επιστροφής

True αν κάποιο στοιχείο είναι διαθέσιμο μετά την κλήση, false διαφορετικά.

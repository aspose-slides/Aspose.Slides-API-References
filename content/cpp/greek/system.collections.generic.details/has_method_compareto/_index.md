## has_method_compareto struct

Ελέγχει εάν υπάρχει η μέθοδος CompareTo στον καθορισμένο τύπο. Εάν ναι, κληρονομεί std::true_type, διαφορετικά κληρονομεί std::false_type. Μπορεί να χρησιμοποιηθεί σε std::enable_if.

```cpp
template<typename T,typename Sfinae>class has_method_compareto : public std::false_type
```

### Παράμετροι προτύπου

| Παράμετρος | Περιγραφή |
| --- | --- |
| T | Τύπος για έλεγχο της ύπαρξης της μεθόδου Equals. |
| Sfinae | Επίσημο όρισμα προτύπου για την λειτουργία του SFINAE. |

## has_method_compareto_shared_ptr δομή


Ελέγχει εάν η μέθοδος CompareTo(SharedPtr<T>) υπάρχει στον καθορισμένο τύπο. Αν ναι, κληρονομεί std::true_type, αλλιώς κληρονομεί std::false_type. Μπορεί να χρησιμοποιηθεί σε std::enable_if.

```cpp
template<typename T,typename Sfinae>class has_method_compareto_shared_ptr : public std::false_type
```


### Παράμετροι προτύπου

| Parameter | Description |
| --- | --- |
| T | Τύπος για να ελεγχθεί η ύπαρξη της μεθόδου Equals. |
| Sfinae | Τυπικό όρισμα προτύπου ώστε το SFINAE να λειτουργήσει. |

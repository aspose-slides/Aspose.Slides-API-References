## EnumeratorSelectAdapter::MoveNext() μέθοδος

Moves enumerator to the next element. If no element was referenced before, sets reference to the first element available. If container end was hit, does nothing.

```cpp
bool System::Collections::Generic::Details::EnumeratorSelectAdapter<Source, Result>::MoveNext() override
```

### Τιμή Επιστροφής

True αν κάποιο στοιχείο είναι διαθέσιμο μετά την κλήση, false διαφορετικά.

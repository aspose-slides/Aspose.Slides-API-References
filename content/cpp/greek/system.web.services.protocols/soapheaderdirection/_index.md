## SoapHeaderDirection enum

Καταγράφει τις κατευθύνσεις των κεφαλίδων SOAP.

```cpp
enum class SoapHeaderDirection
```

### Values

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| In | 1 | Ο παραλήπτης της κεφαλίδας SOAP είναι η υπηρεσία XML [Web](../../system.web/). |
| InOut | 3 | Οι παραλήπτες της κεφαλίδας SOAP είναι και η υπηρεσία XML [Web](../../system.web/) και ο πελάτης της υπηρεσίας XML [Web](../../system.web/). |
| Out | 2 | Ο παραλήπτης της κεφαλίδας SOAP είναι ο πελάτης της υπηρεσίας XML [Web](../../system.web/). |
| Fault | 4 | Η κεφαλίδα SOAP αποστέλλεται στον πελάτη της υπηρεσίας XML [Web](../../system.web/) όταν μια εξαίρεση ρίχνεται από τη μέθοδο της υπηρεσίας XML [Web](../../system.web/). |

## SoapServiceRoutingStyle enum

Καθορίζει τις επιλογές για το πώς ένα μήνυμα SOAP δρομολογείται στην υπηρεσία XML [Web](../../system.web/).

```cpp
enum class SoapServiceRoutingStyle
```

### Τιμές

| Name | Value | Description |
| --- | --- | --- |
| SoapAction | 0 | Το μήνυμα SOAP δρομολογείται με βάση το χαρακτηριστικό 'SOAPAction'. |
| RequestElement | 1 | Το μήνυμα SOAP δρομολογείται με βάση το πρώτο στοιχείο-παιδί μετά το στοιχείο 'Body'. |

## SoapParameterStyle enum

Καθορίζει τις μορφές παραμέτρων σε ένα μήνυμα SOAP.

```cpp
enum class SoapParameterStyle
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Default | 0 | Αν '[SoapDocumentServiceAttribute](../soapdocumentserviceattribute/)' δεν εφαρμόζεται στην κλάση, τότε η προεπιλεγμένη τιμή είναι 'Wrapped'. |
| Bare | 1 | Οι παράμετροι τοποθετούνται σε στοιχεία XML που ακολουθούν το στοιχείο 'Body'. |
| Wrapped | 2 | Οι παράμετροι ενσωματώνονται μέσα σε ένα μόνο στοιχείο XML που ακολουθεί το στοιχείο 'Body'. |

## SoapHeaderAttribute κλάση


Καθορίζει την κεφαλίδα SOAP που η μέθοδος υπηρεσίας XML [Web](../../system.web/) ή ο πελάτης υπηρεσίας XML [Web](../../system.web/) μπορούν να επεξεργαστούν. Τα αντικείμενα αυτής της κλάσης θα πρέπει να εκχωρούνται μόνο χρησιμοποιώντας τη συνάρτηση [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε στιγμιότυπο αυτού του τύπου στη στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης και/ή σφάλματα ελέγχου. Πάντα να ενσύλλετε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και να χρησιμοποιείτε αυτόν τον δείκτη για να τη μεταβιβάζετε στις συναρτήσεις ως όρισμα.

```cpp
class SoapHeaderAttribute : public System::Attribute
```

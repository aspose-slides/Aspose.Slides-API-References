## InkEffectType enum

Καθορίζει ένα σύνολο προεπιλεγμένων οπτικών εφέ για την απόδοση μελάνης. Κάθε εφέ αντιστοιχεί σε μια υφή ή εικόνα που προσομοιώνει μια στιλιζαρισμένη εμφάνιση μαρκαδίας. Αυτές οι τιμές μπορούν να χρησιμοποιηθούν για την προσαρμογή του οπτικού στυλ των ψηφιακών γραμμών μελάνης κατά την απόδοση.

```cpp
enum class InkEffectType
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| NotDefined | -1 | Το εφέ δεν έχει οριστεί ή είναι άγνωστο. Αυτή η τιμή μπορεί να χρησιμοποιηθεί ως προεπιλογή ή εναλλακτική. |
| Bronze | 0 | Μια ζεστή, καστανή μεταλλική υφή που θυμίζει μελάνη μπρούντζου. |
| Galaxy | 1 | Μια πολύχρωμη, λαμπερή υφή που θυμίζει γαλαξία, με κοσμικούς τόνους. |
| Gold | 2 | Μια φωτεινή, μεταλλική χρυσή υφή που προσδίδει στις γραμμές μελάνης πολυτελή εμφάνιση. |
| Lava | 3 | Μια φλογερή υφή που μοιάζει με λιωμένη λάβα, με κόκκινες και πορτοκαλί αποχρώσεις. |
| Ocean | 4 | Μια βαθιά μπλε, υγρή-όπως υφή που μιμείται τα κύματα του ωκεανού ή μελάνη βασισμένη σε νερό. |
| RainbowGlitter | 5 | Ένα πολύχρωμο, λαμπερό εφέ ουράνιου τόξου που χρησιμοποιείται για εορταστικές ή έντονες γραμμές μελάνης. |
| RoseGold | 6 | Ένα απαλό μίγμα ροζ-χρυσού, παρόμοιο με το ροζ χρυσό, για κομψές γραμμές μελάνης. |
| Silver | 7 | Μια δροσερή, μεταλλική αργυρή υφή που προσομοιώνει κλασική αργυρή μελάνη. |

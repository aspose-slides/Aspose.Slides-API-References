## SignedCms κλάση

Υπογράφει περιεχόμενο σύμφωνα με το πρότυπο CMS/PKCS #7. Δεν έχει υλοποιηθεί. Αντικείμενα αυτής της κλάσης πρέπει να καταλαμβάνονται μόνο μέσω της συνάρτησης [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε αντίτυπο αυτού του τύπου στην στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς αυτό θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης και/ή σφάλματα ισχυρισμών. Πάντα τυλίξτε αυτή την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να τη μεταβιβάσετε σε συναρτήσεις ως όρισμα.

```cpp
class SignedCms : public System::Object
```

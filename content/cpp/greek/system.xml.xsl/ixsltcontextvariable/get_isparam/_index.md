## IXsltContextVariable::get_IsParam() μέθοδος

Επιστρέφει μια τιμή που υποδεικνύει εάν η μεταβλητή είναι μια παράμετρος Extensible Stylesheet Language Transformations (XSLT). Αυτό μπορεί να είναι μια παράμετρος για ένα φύλλο στυλ ή για ένα πρότυπο.

```cpp
virtual bool System::Xml::Xsl::IXsltContextVariable::get_IsParam()=0
```

### Τιμή Επιστροφής

**true** if the variable is an XSLT parameter; otherwise, **false**.

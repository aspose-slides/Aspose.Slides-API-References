```cpp
SharedPtr<Object> System::Xml::Xsl::XsltArgumentList::GetParam(const String &name, const String &namespaceUri)
```


### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| name | const [String](../../../system/string/)\& | Το όνομα της παραμέτρου. [XsltArgumentList](../) δεν ελέγχει αν το παρελθόν όνομα είναι έγκυρο τοπικό όνομα· ωστόσο, το όνομα δεν μπορεί να είναι **nullptr**. |
| namespaceUri | const [String](../../../system/string/)\& | Το URI του χώρου ονομάτων που συσχετίζεται με την παράμετρο. |

### Τιμή Επιστροφής

Το αντικείμενο παραμέτρου ή **nullptr** εάν δεν βρέθηκε.

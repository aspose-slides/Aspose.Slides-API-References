## XsltContext::get_Whitespace() μέθοδος

Όταν παρακαμφθεί σε παράγωγη κλάση, παίρνει μια τιμή που υποδεικνύει εάν θα συμπεριληφθούν κόμβοι λευκού διαστήματος στην έξοδο.

```cpp
virtual bool System::Xml::Xsl::XsltContext::get_Whitespace()=0
```

### Τιμή Επιστροφής

**true** για να ελέγξετε τους κόμβους λευκού διαστήματος στο πηγαίο έγγραφο για συμπερίληψη στην έξοδο· **false** για να μην αξιολογηθούν οι κόμβοι λευκού διαστήματος. Η προεπιλογή είναι **true**.

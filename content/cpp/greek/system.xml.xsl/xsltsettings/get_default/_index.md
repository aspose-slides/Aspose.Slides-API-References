## XsltSettings::get_Default() μέθοδος


Επιστρέφει ένα αντικείμενο [XsltSettings](../) με προεπιλεγμένες ρυθμίσεις. Η υποστήριξη για τη **document()** συνάρτηση του XSLT και τα ενσωματωμένα μπλοκ σεναρίων είναι απενεργοποιημένη.

```cpp
static SharedPtr<XsltSettings> System::Xml::Xsl::XsltSettings::get_Default()
```


### Τιμή επιστροφής

Ένα αντικείμενο [XsltSettings](../) με τις επιλογές [XsltSettings::set_EnableDocumentFunction](../set_enabledocumentfunction/) και [XsltSettings::set_EnableScript](../set_enablescript/) ορισμένες σε **false**.

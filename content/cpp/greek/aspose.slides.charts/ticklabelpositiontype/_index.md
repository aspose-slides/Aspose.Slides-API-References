## TickLabelPositionType enum

Αντιπροσωπεύει τον τύπο θέσης των ετικετών σημείων σήμανσης στον καθορισμένο άξονα.

```cpp
enum class TickLabelPositionType
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| High | 0 | Καθορίζει ότι οι ετικέτες του άξονα θα είναι στο υψηλό άκρο του κάθετου άξονα. |
| Low | 1 | Καθορίζει ότι οι ετικέτες του άξονα θα είναι στο χαμηλό άκρο του κάθετου άξονα. |
| NextTo | 2 | Καθορίζει ότι οι ετικέτες του άξονα θα είναι δίπλα στον άξονα. |
| None | 3 | Καθορίζει ότι οι ετικέτες του άξονα δεν σχεδιάζονται. |

## DataLabel::get_IsVisible() μέθοδος

False σημαίνει ότι η ετικέτα δεδομένων δεν είναι ορατή (και έτσι όλες οι σημαίες Show*-flags (ShowValue, ...) είναι ψευδείς). Μόνο για ανάγνωση **bool**.

```cpp
bool Aspose::Slides::Charts::DataLabel::get_IsVisible() override
```

## Παρατηρήσεις

Αν η ετικέτα δεδομένων είναι ορατή, μπορείτε να την κρύψετε με τη μέθοδο [Hide()](../hide/). Αλλά αν η ετικέτα δεδομένων δεν είναι ορατή (IsVisible είναι false) μπορείτε να την κάνετε ορατή ορίζοντας τις σημαίες Show*-flags (ShowValue, ...) σε κατάσταση true.

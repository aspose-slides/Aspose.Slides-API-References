## DataLabel::Hide() μέθοδος

Κάντε την ετικέτα δεδομένων κρυφή ορίζοντας όλα τα Show*-flags (ShowValue, ...) σε κατάσταση false. Το IsVisible θα είναι false μετά από αυτό.

```cpp
void Aspose::Slides::Charts::DataLabel::Hide() override
```

## Σχόλια

Αν η ετικέτα δεδομένων δεν είναι ορατή (IsVisible είναι false) μπορείτε να κάνετε την ετικέτα δεδομένων ορατή ορίζοντας τα Show*-flags (ShowValue, ...) σε κατάσταση true.

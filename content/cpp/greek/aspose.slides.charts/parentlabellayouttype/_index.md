## ParentLabelLayoutType enum

Αναπαριστά τη διάταξη των ετικετών δεδομένων κατηγορίας. Χρησιμοποιείται μόνο με τις σειρές Treemap.

```cpp
enum class ParentLabelLayoutType
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Banner | 0 | Καθορίζει ότι οι ετικέτες γονικής κατηγορίας εμφανίζονται ως banner |
| None | 1 | Καθορίζει ότι οι ετικέτες γονικής κατηγορίας δεν σχεδιάζονται |
| Overlapping | 2 | Καθορίζει ότι οι ετικέτες γονικής κατηγορίας εμφανίζονται σε αλληλοεπικάλυψη |

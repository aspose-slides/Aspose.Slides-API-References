## TickMarkType enum

Αντιπροσωπεύει τον τύπο του σημείου σήμανσης για τον καθορισμένο άξονα.

```cpp
enum class TickMarkType
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Cross | 0 | Καθορίζει ότι τα σημεία σήμανσης θα διασχίζουν τον άξονα. |
| Inside | 1 | Καθορίζει ότι τα σημεία σήμανσης θα είναι μέσα στην περιοχή σχεδίασης. |
| None | 2 | Καθορίζει ότι δεν θα υπάρχουν σημεία σήμανσης. |
| Outside | 3 | Καθορίζει ότι τα σημεία σήμανσης θα είναι έξω από την περιοχή σχεδίασης. |

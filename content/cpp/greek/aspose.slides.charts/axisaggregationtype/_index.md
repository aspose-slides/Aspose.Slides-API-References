## AxisAggregationType enum

Αναπαριστά τον τύπο συγκέντρωσης του άξονα κατηγορίας. Χρησιμοποιείται μόνο με σειρές Histogram ή HistogramPareto.

```cpp
enum class AxisAggregationType
```

### Values

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| ByCategory | 0 | Συγκεντρώστε ανά κατηγορίες |
| Automatic | 1 | Συγκεντρώνει τις κατηγορίες αυτόματα. |
| ByBinWidth | 2 | Συγκεντρώνει τις κατηγορίες κατά το πλάτος δοκιμίου. Η ιδιότητα IAxis::get(set)_BinWidth πρέπει να οριστεί. |
| ByNumberOfBins | 3 | Συγκεντρώνει τις κατηγορίες κατά αριθμό δοκιμίων. Η ιδιότητα IAxis::get(set)_NumberOfBins πρέπει να οριστεί. |

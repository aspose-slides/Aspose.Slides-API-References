## DisplayBlanksAsType enum


Καθορίζει πώς θα εμφανίζονται τα ελλιπή δεδομένα.

```cpp
enum class DisplayBlanksAsType
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Gap | 0 | Καθορίζει ότι οι κενές τιμές θα παραμείνουν ως κενό. |
| Span | 1 | Καθορίζει ότι οι κενές τιμές θα καλυφθούν με μια γραμμή. |
| Zero | 2 | Καθορίζει ότι οι κενές τιμές θα θεωρηθούν μηδέν. |

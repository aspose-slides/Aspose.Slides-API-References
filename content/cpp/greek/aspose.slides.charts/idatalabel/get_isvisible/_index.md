## IDataLabel::get_IsVisible() μέθοδος

False σημαίνει ότι η ετικέτα δεδομένων δεν είναι ορατή (και έτσι όλες οι σημαίες Show*- (ShowValue, ...) είναι False). Μόνο για ανάγνωση **bool**.

```cpp
virtual bool Aspose::Slides::Charts::IDataLabel::get_IsVisible()=0
```

## Παρατηρήσεις

Εάν η ετικέτα δεδομένων είναι ορατή, μπορείτε να την κρύψετε με τη μέθοδο [Hide()](../hide/). Αλλά εάν η ετικέτα δεδομένων δεν είναι ορατή (IsVisible είναι false) μπορείτε να κάνετε την ετικέτα δεδομένων ορατή ορίζοντας τις σημαίες Show*- (ShowValue, ...) σε κατάσταση true.

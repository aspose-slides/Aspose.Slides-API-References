## ITrendline::set_Backward(double) μέθοδος


Καθορίζει τον αριθμό των κατηγοριών (ή μονάδων σε ένα διάγραμμα διασκορπισμού) που η γραμμή τάσης επεκτείνεται πριν τα δεδομένα για τη σειρά που εξελίσσεται. Σε διαγράμματα διασκορπισμού και μη-διασκορπισμού, η τιμή πρέπει να είναι οποιαδήποτε μη αρνητική τιμή. Γράψτε **double**.

```cpp
virtual void Aspose::Slides::Charts::ITrendline::set_Backward(double value)=0
```

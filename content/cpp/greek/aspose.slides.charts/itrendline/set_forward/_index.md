## ITrendline::set_Forward(double) μέθοδος

Καθορίζει τον αριθμό των κατηγοριών (ή μονάδων σε διάγραμμα διασποράς) που η γραμμή τάσης επεκτείνεται μετά τα δεδομένα για τη σειρά που υποβάλλεται σε ανάλυση τάσης. Σε διαγράμματα διασποράς και μη διασποράς, η τιμή πρέπει να είναι οποιαδήποτε μη αρνητική τιμή. Γράψτε **double**.

```cpp
virtual void Aspose::Slides::Charts::ITrendline::set_Forward(double value)=0
```

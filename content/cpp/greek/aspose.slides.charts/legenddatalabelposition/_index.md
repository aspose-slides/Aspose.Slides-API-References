## LegendDataLabelPosition enum

Καθορίζει τη θέση των ετικετών δεδομένων.

```cpp
enum class LegendDataLabelPosition
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| NotDefined | -1 | Η θέση δεν έχει οριστεί. |
| Bottom | 0 | Καθορίζει ότι οι ετικέτες δεδομένων θα εμφανίζονται κάτω από το δείκτη δεδομένων. |
| BestFit | 1 | Καθορίζει ότι οι ετικέτες δεδομένων θα εμφανίζονται στην καλύτερη θέση. |
| Center | 2 | Καθορίζει ότι οι ετικέτες δεδομένων θα εμφανίζονται κεντραρισμένες πάνω στο δείκτη δεδομένων. |
| InsideBase | 3 | Καθορίζει ότι οι ετικέτες δεδομένων θα εμφανίζονται μέσα στη βάση του δείκτη δεδομένων. |
| InsideEnd | 4 | Καθορίζει ότι οι ετικέτες δεδομένων θα εμφανίζονται μέσα στο άκρο του δείκτη δεδομένων. |
| Left | 5 | Καθορίζει ότι οι ετικέτες δεδομένων θα εμφανίζονται στα αριστερά του δείκτη δεδομένων. |
| OutsideEnd | 6 | Καθορίζει ότι οι ετικέτες δεδομένων θα εμφανίζονται εκτός του άκρου του δείκτη δεδομένων. |
| Right | 7 | Καθορίζει ότι οι ετικέτες δεδομένων θα εμφανίζονται στα δεξιά του δείκτη δεδομένων. |
| Top | 8 | Καθορίζει ότι οι ετικέτες δεδομένων θα εμφανίζονται πάνω από το δείκτη δεδομένων. |

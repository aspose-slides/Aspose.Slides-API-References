## ErrorBarsCustomValues κλάση

Καθορίζει τις τιμές των ράβδων σφάλματος. Θα πρέπει να χρησιμοποιείται μόνο όταν ο τύπος τιμής των ράβδων σφάλματος είναι Προσαρμοσμένος.

```cpp
class ErrorBarsCustomValues : public Aspose::Slides::DomObject<System::SharedPtr<Aspose::Slides::Charts::ChartDataPoint>>,
                              public Aspose::Slides::Charts::IErrorBarsCustomValues
```

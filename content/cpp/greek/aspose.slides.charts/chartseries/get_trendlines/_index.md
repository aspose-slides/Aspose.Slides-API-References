## ChartSeries::get_TrendLines() μέθοδος

Συλλογή των γραμμών τάσης της σειράς. Μόνο για ανάγνωση [ITrendlineCollection](../../itrendlinecollection/).

```cpp
System::SharedPtr<ITrendlineCollection> Aspose::Slides::Charts::ChartSeries::get_TrendLines() override
```

## Σχόλια

Οι TrendLines είναι διαθέσιμες (μη μηδενικές) για σειρές δεδομένων σε μη στοιβαγμένα 2-D περιοχή, ράβδους, στήλες, γραμμές, μετοχές, xy (διάσπαρσης) και φυσαλίδες διαγράμματα. Μια γραμμή τάσης δεν είναι διαθέσιμη για σειρές δεδομένων σε οποιοδήποτε τύπο διαγράμματος που είναι στοιβαγμένο ή 3-D. Οι TrendLines επίσης δεν είναι διαθέσιμες για ραδάρ, πίτα, επιφάνεια ή δαχτυλίδιο διαγράμματα.

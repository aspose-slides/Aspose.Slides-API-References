## IDataLabelFormat::set_ShowValue(bool) μέθοδος


Αντιπροσωπεύει τη συμπεριφορά εμφάνισης της τιμής ποσοστού ετικέτας δεδομένων σε ένα καθορισμένο διάγραμμα. True εμφανίζει την τιμή ποσοστού. False την κρύβει. Γράψτε **bool**.

```cpp
virtual void Aspose::Slides::Charts::IDataLabelFormat::set_ShowValue(bool value)=0
```

## Σχόλια


Εάν ο γονέας αυτού του αντικειμένου [DataLabelFormat](../../datalabelformat/) είναι μια συλλογή [DataLabelCollection](../../datalabelcollection/) ετικετών δεδομένων, τότε αυτή η ιδιότητα λαμβάνει ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας ShowValue για τις νέες ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/). Η ρύθμιση αυτής της ιδιότητας με τιμή θέτει επίσης αυτήν την τιμή στην ιδιότητα ShowValue για όλες τις ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/) (i.e. \"DataLabels.DefaultDataLabelFormat.ShowValue = val;\" cause to all DataLabels[i].ShowValue is equal to val).

## IDataLabelFormat::get_ShowPercentage() μέθοδος

Αντιπροσωπεί τη συμπεριφορά εμφάνισης του ποσοστού ετικέτας δεδομένων ενός καθορισμένου διαγράμματος. True εμφανίζει την τιμή του ποσοστού. False για απόκρυψη. Ανάγνωση **bool**.

```cpp
virtual bool Aspose::Slides::Charts::IDataLabelFormat::get_ShowPercentage()=0
```

## Σχόλια

Εάν ο γονέας αυτού του αντικειμένου [DataLabelFormat](../../datalabelformat/) είναι μια συλλογή [DataLabelCollection](../../datalabelcollection/) ετικετών δεδομένων, τότε αυτή η ιδιότητα αποκτά ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας ShowPercentage για τις νέες ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/). Ο ορισμός αυτής της ιδιότητας με τιμή επίσης ορίζει αυτήν την τιμή στην ιδιότητα ShowPercentage για όλες τις ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/) (i.e. "DataLabels.DefaultDataLabelFormat.ShowPercentage = val;" προκαλεί ώστε όλα τα DataLabels[i].ShowPercentage να είναι ίσα με val).

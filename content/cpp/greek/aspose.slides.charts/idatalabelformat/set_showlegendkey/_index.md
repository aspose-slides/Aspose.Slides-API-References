## IDataLabelFormat::set_ShowLegendKey(bool) μέθοδος

Αντιπροσωπεί τη συμπεριφορά εμφάνισης του κλειδιού υπομνήματος ετικέτας δεδομένων ενός συγκεκριμένου διαγράμματος. Αληθές εάν το κλειδί υπομνήματος ετικέτας δεδομένων είναι ορατό. Γράψτε **bool**.

```cpp
virtual void Aspose::Slides::Charts::IDataLabelFormat::set_ShowLegendKey(bool value)=0
```

## Παρατηρήσεις

Εάν ο γονέας αυτού του [DataLabelFormat](../../datalabelformat/) αντικειμένου είναι μια συλλογή [DataLabelCollection](../../datalabelcollection/) ετικετών δεδομένων, τότε αυτή η ιδιότητα λαμβάνει ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας ShowLegendKey για τις νέες ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/). Ο καθορισμός αυτής της ιδιότητας με τιμή ορίζει επίσης αυτή τη τιμή στην ιδιότητα ShowLegendKey για όλες τις ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/) (i.e. "DataLabels.DefaultDataLabelFormat.ShowLegendKey = val;" cause to all DataLabels[i].ShowLegendKey is equal to val).

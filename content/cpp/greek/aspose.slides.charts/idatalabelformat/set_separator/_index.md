## IDataLabelFormat::set_Separator(System::String) μέθοδος

Ορίζει ή επιστρέφει ένα Variant που αντιπροσωπεύει το διαχωριστικό που χρησιμοποιείται για τις ετικέτες δεδομένων σε ένα γράφημα. Γράψτε [System::String](../../../system/string/).

```cpp
virtual void Aspose::Slides::Charts::IDataLabelFormat::set_Separator(System::String value)=0
```

## Σχόλια

Αν ο γονέας αυτού του αντικειμένου [DataLabelFormat](../../datalabelformat/) είναι μια συλλογή [DataLabelCollection](../../datalabelcollection/) ετικετών δεδομένων, τότε αυτή η ιδιότητα λαμβάνει ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας Separator για τις νέες ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/). Ορισμός αυτής της ιδιότητας με τιμή ορίζει επίσης αυτή την τιμή στην ιδιότητα Separator για όλες τις ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/) (π.χ. "DataLabels.DefaultDataLabelFormat.Separator = val;" με αποτέλεσμα όλα τα DataLabels[i].Separator να είναι ίσα με val).

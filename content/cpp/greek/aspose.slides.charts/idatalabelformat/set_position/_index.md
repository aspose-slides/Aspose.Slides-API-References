## IDataLabelFormat::set_Position(LegendDataLabelPosition) μέθοδος

Αντιπροσωπεύει τη θέση της ετικέτας δεδομένων. Γράψτε [LegendDataLabelPosition](../../legenddatalabelposition/).

```cpp
virtual void Aspose::Slides::Charts::IDataLabelFormat::set_Position(LegendDataLabelPosition value)=0
```

## Παρατηρήσεις

Αν το γονικό αντικείμενο του [DataLabelFormat](../../datalabelformat/) είναι μια συλλογή [DataLabelCollection](../../datalabelcollection/) ετικετών δεδομένων, τότε αυτή η ιδιότητα λαμβάνει ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας Position για τις νέες ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/). Αντιπροσωπεύει τη θέση για τα αντικείμενα [DataLabel](../../datalabel/). Ορισμός αυτής της ιδιότητας με τιμή ορίζει επίσης αυτήν την τιμή στην ιδιότητα Position για όλες τις ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/) (π.χ. \"DataLabels.DefaultDataLabelFormat.Position = val;\" προκαλεί όλες τις DataLabels[i].Position να είναι ίση με val).

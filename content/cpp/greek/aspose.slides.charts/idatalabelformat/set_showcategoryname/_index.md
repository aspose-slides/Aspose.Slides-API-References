## IDataLabelFormat::set_ShowCategoryName(bool) μέθοδος

Αντιπροσωπεύει τη συμπεριφορά εμφάνισης του ονόματος κατηγορίας ετικέτας δεδομένων σε συγκεκριμένο γράφημα. True για να εμφανίσετε το όνομα κατηγορίας για τις ετικέτες δεδομένων σε ένα γράφημα. False για να το κρύψετε. Write **bool**.

```cpp
virtual void Aspose::Slides::Charts::IDataLabelFormat::set_ShowCategoryName(bool value)=0
```

## Παρατηρήσεις

Εάν το γονικό αντικείμενο του [DataLabelFormat](../../datalabelformat/) είναι μια συλλογή [DataLabelCollection](../../datalabelcollection/) ετικετών δεδομένων, τότε αυτή η ιδιότητα λαμβάνει ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας ShowCategoryName για τις νέες ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/). Ο καθορισμός αυτής της ιδιότητας με τιμή επίσης ορίζει αυτήν την τιμή στην ιδιότητα ShowCategoryName για όλες τις ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/) (π.χ. \"DataLabels.DefaultDataLabelFormat.ShowCategoryName = val;\" προκαλεί όλες οι DataLabels[i].ShowCategoryName να είναι ίσες με val).

## IDataLabelFormat::get_ShowLabelAsDataCallout() μέθοδος

Καθορίζει εάν η ετικέτα δεδομένων του καθορισμένου διαγράμματος θα εμφανίζεται ως απόσπασμα δεδομένων ή ως ετικέτα δεδομένων.

```cpp
virtual bool Aspose::Slides::Charts::IDataLabelFormat::get_ShowLabelAsDataCallout()=0
```

## Σχόλια

Αν ο γονέας αυτού του αντικειμένου [DataLabelFormat](../../datalabelformat/) είναι μια συλλογή [DataLabelCollection](../../datalabelcollection/) ετικετών δεδομένων, τότε αυτή η ιδιότητα λαμβάνει ή θέτει την προεπιλεγμένη τιμή της ιδιότητας ShowLabelAsDataCallout για τις νέες ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/). Ο ορισμός αυτής της ιδιότητας με τιμή θέτει επίσης αυτήν την τιμή στην ιδιότητα ShowLabelAsDataCallout για όλες τις ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/) (π.χ. "DataLabels.DefaultDataLabelFormat.ShowLabelAsDataCallout = val;" προκαλεί όλα τα DataLabels[i].ShowLabelAsDataCallout να είναι ίσα με val).

## IDataLabelFormat::get_Position() μέθοδος

Αναπαριστά τη θέση της ετικέτας δεδομένων. Διαβάστε [LegendDataLabelPosition](../../legenddatalabelposition/).

```cpp
virtual LegendDataLabelPosition Aspose::Slides::Charts::IDataLabelFormat::get_Position()=0
```

## Σχόλια

Εάν ο γονέας αυτού του [DataLabelFormat](../../datalabelformat/) αντικειμένου είναι μια συλλογή [DataLabelCollection](../../datalabelcollection/) ετικετών δεδομένων, τότε αυτή η ιδιότητα λαμβάνει ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας Position για τις νέες ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/). Αναπαριστά τη θέση για τα αντικείμενα [DataLabel](../../datalabel/). Ορίστε αυτή την ιδιότητα με τιμή ώστε επίσης να ορίσει αυτή την τιμή στην ιδιότητα Position για όλες τις ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/) (π.χ. \"DataLabels.DefaultDataLabelFormat.Position = val;\" προκαλεί όλες οι DataLabels[i].Position να είναι ίσες με val).

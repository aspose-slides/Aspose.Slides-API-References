## IDataLabelFormat::set_ShowPercentage(bool) μέθοδος

Αναπαριστά τη συμπεριφορά εμφάνισης της τιμής ποσοστού ετικέτας δεδομένων ενός συγκεκριμένου διαγράμματος. Η τιμή True εμφανίζει την τιμή ποσοστού. Η τιμή False την κρύβει. Γράψτε **bool**.

```cpp
virtual void Aspose::Slides::Charts::IDataLabelFormat::set_ShowPercentage(bool value)=0
```

## Παρατηρήσεις

Εάν ο γονέας αυτού του αντικειμένου [DataLabelFormat](../../datalabelformat/) είναι μια συλλογή [DataLabelCollection](../../datalabelcollection/) ετικετών δεδομένων, τότε αυτή η ιδιότητα λαμβάνει ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας ShowPercentage για τις νέες ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/). Ο ορισμός αυτής της ιδιότητας με τιμή ορίζει επίσης αυτή την τιμή στην ιδιότητα ShowPercentage για όλες τις ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/) (π.χ. "DataLabels.DefaultDataLabelFormat.ShowPercentage = val;" προκαλεί όλες οι DataLabels[i].ShowPercentage να είναι ίσες με val).

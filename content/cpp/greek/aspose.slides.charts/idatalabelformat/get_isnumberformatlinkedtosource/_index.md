## IDataLabelFormat::get_IsNumberFormatLinkedToSource() μέθοδος

Διάβασε **bool**.

```cpp
virtual bool Aspose::Slides::Charts::IDataLabelFormat::get_IsNumberFormatLinkedToSource()=0
```

## Παρατηρήσεις

Αν ο γονέας αυτού του αντικειμένου [DataLabelFormat](../../datalabelformat/) είναι μια συλλογή [DataLabelCollection](../../datalabelcollection/) ετικετών δεδομένων, τότε αυτή η ιδιότητα παίρνει ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας IsNumberFormatLinkedToSource για τις νέες ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/). Ορισμός αυτής της ιδιότητας με τιμή επίσης ορίζει αυτήν την τιμή στην ιδιότητα IsNumberFormatLinkedToSource για όλες τις ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/) (π.χ. "DataLabels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = val;" που προκαλεί όλα τα DataLabels[i].IsNumberFormatLinkedToSource να είναι ίσα με val).

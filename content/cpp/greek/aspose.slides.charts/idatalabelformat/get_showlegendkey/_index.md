## IDataLabelFormat::get_ShowLegendKey() μέθοδος


Αντιπροσωπεύει τη συμπεριφορά εμφάνισης του κλειδιού υπομνήματος ετικέτας δεδομένων ενός καθορισμένου διαγράμματος. Αληθές εάν το κλειδί υπομνήματος ετικέτας δεδομένων είναι ορατό. Ανάγνωση **bool**.

```cpp
virtual bool Aspose::Slides::Charts::IDataLabelFormat::get_ShowLegendKey()=0
```

## Παρατηρήσεις


Εάν ο γονέας αυτού του αντικειμένου [DataLabelFormat](../../datalabelformat/) είναι μια συλλογή [DataLabelCollection](../../datalabelcollection/) ετικετών δεδομένων, τότε αυτή η ιδιότητα παίρνει ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας ShowLegendKey για τις νέες ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/). Ορισμός αυτής της ιδιότητας με τιμή ορίζει επίσης αυτήν την τιμή στην ιδιότητα ShowLegendKey για όλες τις ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/) (i.e. "DataLabels.DefaultDataLabelFormat.ShowLegendKey = val;" προκαλεί όλες οι DataLabels[i].ShowLegendKey να είναι ίσες με val).

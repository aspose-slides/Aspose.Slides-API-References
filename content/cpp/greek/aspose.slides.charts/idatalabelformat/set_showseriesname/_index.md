## IDataLabelFormat::set_ShowSeriesName(bool) μέθοδος

Ορίζει μια Boolean για να υποδείξει τη συμπεριφορά εμφάνισης του ονόματος σειράς για τις ετικέτες δεδομένων σε ένα γράφημα. True για να εμφανίσετε το όνομα σειράς. False για να το κρύψετε. Γράψτε **bool**.

```cpp
virtual void Aspose::Slides::Charts::IDataLabelFormat::set_ShowSeriesName(bool value)=0
```

## Παρατηρήσεις

Εάν ο γονέας αυτού του [DataLabelFormat](../../datalabelformat/) αντικειμένου είναι μια συλλογή [DataLabelCollection](../../datalabelcollection/) ετικετών δεδομένων, τότε αυτή η ιδιότητα λαμβάνει ή θέτει την προεπιλεγμένη τιμή της ιδιότητας ShowSeriesName για τις νέες ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/). Ορίζοντας αυτήν την ιδιότητα με τιμή, ορίζεται επίσης αυτή η τιμή στην ιδιότητα ShowSeriesName για όλες τις ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/) (π.χ. "DataLabels.DefaultDataLabelFormat.ShowSeriesName = val;" προκαλεί όλες οι DataLabels[i].ShowSeriesName να είναι ίσες με val).

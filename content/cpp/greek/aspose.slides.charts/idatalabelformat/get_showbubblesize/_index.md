## IDataLabelFormat::get_ShowBubbleSize() μέθοδος

Αντιπροσωπεύει τη συμπεριφορά εμφάνισης της τιμής μεγέθους φυσαλιού ετικέτας δεδομένων ενός συγκεκριμένου διαγράμματος. True εμφανίζει την τιμή του μεγέθους φυσαλιού. False για απόκρυψη. Ανάγνωση **bool**.

```cpp
virtual bool Aspose::Slides::Charts::IDataLabelFormat::get_ShowBubbleSize()=0
```

## Παρατηρήσεις

Εάν ο γονέας αυτού του [DataLabelFormat](../../datalabelformat/) αντικειμένου είναι μια συλλογή [DataLabelCollection](../../datalabelcollection/) ετικετών δεδομένων, τότε αυτή η ιδιότητα λαμβάνει ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας ShowBubbleSize για τις νέες ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/). Ορισμός αυτής της ιδιότητας με τιμή ορίζει επίσης αυτήν την τιμή στην ιδιότητα ShowBubbleSize για όλες τις ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/) (π.χ. "DataLabels.DefaultDataLabelFormat.ShowBubbleSize = val;" προκαλεί να είναι όλες οι DataLabels[i].ShowBubbleSize ίσες με val).

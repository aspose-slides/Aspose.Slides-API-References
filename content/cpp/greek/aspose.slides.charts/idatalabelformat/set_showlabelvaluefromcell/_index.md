## IDataLabelFormat::set_ShowLabelValueFromCell(bool) μέθοδος

Αναπαριστά τη συμπεριφορά εμφάνισης της τιμής κελιού ετικέτας δεδομένων ενός συγκεκριμένου διαγράμματος. True εμφανίζει την τιμή του κελιού. False για απόκρυψη. Γράψτε **bool**.

```cpp
virtual void Aspose::Slides::Charts::IDataLabelFormat::set_ShowLabelValueFromCell(bool value)=0
```

## Σχόλια

Εάν ο γονέας αυτού του αντικειμένου [DataLabelFormat](../../datalabelformat/) είναι μια συλλογή [DataLabelCollection](../../datalabelcollection/) ετικετών δεδομένων, τότε αυτή η ιδιότητα λαμβάνει ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας ShowLabelValueFromCell για τις νέες ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/). Ορίζοντας αυτήν την ιδιότητα με μια τιμή, η ίδια τιμή ορίζεται επίσης στην ιδιότητα ShowLabelValueFromCell για όλες τις ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/) (π.χ. "DataLabels.DefaultDataLabelFormat.ShowLabelValueFromCell = val;" προκαλεί ώστε όλα τα DataLabels[i].ShowLabelValueFromCell να είναι ίσα με val).

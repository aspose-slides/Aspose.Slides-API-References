## DataLabelFormat::set_ShowLegendKey(bool) μέθοδος

Κατανοεί τη συμπεριφορά εμφάνισης του κλειδιού υπομνήματος ετικέτας δεδομένων ενός συγκεκριμένου διαγράμματος. Αληθές εάν το κλειδί υπομνήματος ετικέτας δεδομένων είναι ορατό. Write **bool**.

```cpp
void Aspose::Slides::Charts::DataLabelFormat::set_ShowLegendKey(bool value) override
```

## Παρατηρήσεις

Εάν ο γονέας αυτού του αντικειμένου [DataLabelFormat](../) είναι μια συλλογή [DataLabelCollection](../../datalabelcollection/) ετικετών δεδομένων, τότε αυτή η ιδιότητα λαμβάνει ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας ShowLegendKey για τις νέες ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/). Ορισμός αυτής της ιδιότητας με τιμή ορίζει επίσης αυτήν την τιμή στην ιδιότητα ShowLegendKey για όλες τις ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/) (δηλ. "DataLabels.DefaultDataLabelFormat.ShowLegendKey = val;" προκαλεί ώστε όλα τα DataLabels[i].ShowLegendKey να είναι ίσα με val).

## DataLabelFormat::get_Position() μέθοδος

Αναπαριστά τη θέση της ετικέτας δεδομένων. Διαβάστε [LegendDataLabelPosition](../../legenddatalabelposition/).

```cpp
LegendDataLabelPosition Aspose::Slides::Charts::DataLabelFormat::get_Position() override
```

## Παρατηρήσεις

Αν ο γονέας αυτού του αντικειμένου [DataLabelFormat](../) είναι μια συλλογή [DataLabelCollection](../../datalabelcollection/) ετικετών δεδομένων, τότε αυτή η ιδιότητα λαμβάνει ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας Position για τις νέες ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/). Αναπαριστά τη θέση για τα αντικείμενα [DataLabel](../../datalabel/). Ορισμός αυτής της ιδιότητας με τιμή ορίζει επίσης αυτή την τιμή στην ιδιότητα Position για όλες τις ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/) (π.χ. \"DataLabels.DefaultDataLabelFormat.Position = val;\" προκαλεί όλα τα DataLabels[i].Position να είναι ίσα με val).

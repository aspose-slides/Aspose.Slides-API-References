## DataLabelFormat::set_ShowCategoryName(bool) μέθοδος

Αντιπροσωπεί τη συμπεριφορά εμφάνισης του ονόματος κατηγορίας των ετικετών δεδομένων ενός συγκεκριμένου διαγράμματος. True για να εμφανίζεται το όνομα κατηγορίας για τις ετικέτες δεδομένων σε ένα διάγραμμα. False για να κρύβεται. Write **bool**.

```cpp
void Aspose::Slides::Charts::DataLabelFormat::set_ShowCategoryName(bool value) override
```

## Σχόλια

Εάν ο γονέας αυτού του [DataLabelFormat](../) αντικειμένου είναι μια συλλογή [DataLabelCollection](../../datalabelcollection/) ετικετών δεδομένων, τότε αυτή η ιδιότητα παίρνει ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας ShowCategoryName για τις νέες ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/). Ορισμός αυτής της ιδιότητας με τιμή ορίζει επίσης αυτή την τιμή στην ιδιότητα ShowCategoryName για όλες τις ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/) (i.e. "DataLabels.DefaultDataLabelFormat.ShowCategoryName = val;" cause to all DataLabels[i].ShowCategoryName is equal to val).

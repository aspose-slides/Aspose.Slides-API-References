## DataLabelFormat::get_ShowLabelAsDataCallout() μέθοδος


Καθορίζει εάν η ετικέτα δεδομένων του καθορισμένου διαγράμματος θα εμφανίζεται ως κλήση δεδομένων ή ως ετικέτα δεδομένων.

```cpp
bool Aspose::Slides::Charts::DataLabelFormat::get_ShowLabelAsDataCallout() override
```

## Παρατηρήσεις


Εάν ο γονέας αυτού του αντικειμένου [DataLabelFormat](../) είναι μια συλλογή [DataLabelCollection](../../datalabelcollection/) ετικετών δεδομένων, τότε αυτή η ιδιότητα παίρνει ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας ShowLabelAsDataCallout για τις νέες ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/). Η ρύθμιση αυτής της ιδιότητας με τιμή επίσης ορίζει αυτή την τιμή στην ιδιότητα ShowLabelAsDataCallout για όλες τις ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/) (π.χ. \"DataLabels.DefaultDataLabelFormat.ShowLabelAsDataCallout = val;\" οδηγεί ώστε όλα τα DataLabels[i].ShowLabelAsDataCallout να είναι ίσα με val).

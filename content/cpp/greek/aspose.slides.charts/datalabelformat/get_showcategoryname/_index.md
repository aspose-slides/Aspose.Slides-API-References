## DataLabelFormat::get_ShowCategoryName() μέθοδος

Αντιπροσωπεύει τη συμπεριφορά εμφάνισης του ονόματος κατηγορίας των ετικετών δεδομένων σε ένα συγκεκριμένο γράφημα. True για εμφάνιση του ονόματος κατηγορίας στις ετικέτες δεδομένων ενός γραφήματος. False για απόκρυψη. Ανάγνωση **bool**.

```cpp
bool Aspose::Slides::Charts::DataLabelFormat::get_ShowCategoryName() override
```

## Σχόλια

Εάν ο γονέας αυτού του [DataLabelFormat](../) αντικειμένου είναι μια [DataLabelCollection](../../datalabelcollection/) συλλογή ετικετών δεδομένων, τότε αυτή η ιδιότητα παίρνει ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας ShowCategoryName για τις νέες ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/). Ο ορισμός αυτής της ιδιότητας με τιμή καθορίζει επίσης αυτή την τιμή στην ιδιότητα ShowCategoryName για όλες τις ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/) (π.χ. "DataLabels.DefaultDataLabelFormat.ShowCategoryName = val;" οδηγεί ώστε όλα τα DataLabels[i].ShowCategoryName να είναι ίσα με val).

## DataLabelFormat::set_ShowBubbleSize(bool) μέθοδος

Αναπαριστά τη συμπεριφορά εμφάνισης της τιμής του μεγέθους φυσαλίδας ετικέτας δεδομένων ενός συγκεκριμένου διαγράμματος. Η τιμή True εμφανίζει την τιμή του μεγέθους φυσαλίδας. Η τιμή False την κρύβει. Γράψτε **bool**.

```cpp
void Aspose::Slides::Charts::DataLabelFormat::set_ShowBubbleSize(bool value) override
```

## Παρατηρήσεις

Εάν ο γονέας του αντικειμένου [DataLabelFormat](../) είναι μια [DataLabelCollection](../../datalabelcollection/) συλλογή ετικετών δεδομένων, τότε αυτή η ιδιότητα ανακτά ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας ShowBubbleSize για τις νέες ετικέτες δεδομένων στη [DataLabelCollection](../../datalabelcollection/) συλλογή. Ο ορισμός αυτής της ιδιότητας με τιμή ορίζει επίσης αυτήν την τιμή στην ιδιότητα ShowBubbleSize για όλες τις ετικέτες δεδομένων στη [DataLabelCollection](../../datalabelcollection/) συλλογή (π.χ. "DataLabels.DefaultDataLabelFormat.ShowBubbleSize = val;" προκαλεί όλα τα DataLabels[i].ShowBubbleSize να είναι ίσα με val).

## DataLabelFormat::get_ShowLeaderLines() μέθοδος

Αντιπροσωπεύει τη συμπεριφορά εμφάνισης γραμμών οδηγού των ετικετών δεδομένων ενός συγκεκριμένου διαγράμματος. True εμφανίζει τις γραμμές οδηγού. False κρύβει. Read **bool**.

```cpp
bool Aspose::Slides::Charts::DataLabelFormat::get_ShowLeaderLines() override
```

## Παρατηρήσεις

Εάν ο γονέας αυτού του αντικειμένου [DataLabelFormat](../) είναι μια συλλογή [DataLabelCollection](../../datalabelcollection/) ετικετών δεδομένων, τότε αυτή η ιδιότητα παίρνει ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας ShowLeaderLines για τις νέες ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/). Ορισμός αυτής της ιδιότητας με τιμή επίσης ορίζει αυτήν την τιμή στην ιδιότητα ShowLeaderLines για όλες τις ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/) (π.χ. "DataLabels.DefaultDataLabelFormat.ShowLeaderLines = val;" προκαλεί να είναι όλα DataLabels[i].ShowLeaderLines ίσα με val).

## DataLabelFormat::set_ShowLabelAsDataCallout(bool) method


Καθορίζει αν η ετικέτα δεδομένων του συγκεκριμένου διαγράμματος θα εμφανιστεί ως επισήμανση δεδομένων ή ως ετικέτα δεδομένων.

```cpp
void Aspose::Slides::Charts::DataLabelFormat::set_ShowLabelAsDataCallout(bool value) override
```

## Παρατηρήσεις


Εάν ο γονέας αυτού του αντικειμένου [DataLabelFormat](../) είναι μια συλλογή [DataLabelCollection](../../datalabelcollection/) ετικετών δεδομένων, τότε αυτή η ιδιότητα λαμβάνει ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας ShowLabelAsDataCallout για τις νέες ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/). Ορισμός αυτής της ιδιότητας με τιμή επίσης ορίζει αυτή την τιμή στην ιδιότητα ShowLabelAsDataCallout για όλες τις ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/) (π.χ. \"DataLabels.DefaultDataLabelFormat.ShowLabelAsDataCallout = val;\" οδηγεί ώστε όλες οι DataLabels[i].ShowLabelAsDataCallout να είναι ίσες με val).

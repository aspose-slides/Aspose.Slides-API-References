## DataLabelFormat::set_ShowLeaderLines(bool) μέθοδος

Αντιπροσωπεύει τη συμπεριφορά εμφάνισης των γραμμών οδηγού των ετικετών δεδομένων ενός συγκεκριμένου διαγράμματος. True εμφανίζει τις γραμμές οδηγού. False για απόκρυψη. Γράψτε **bool**.

```cpp
void Aspose::Slides::Charts::DataLabelFormat::set_ShowLeaderLines(bool value) override
```

## Σχόλια

Εάν ο γονέας αυτού του αντικειμένου [DataLabelFormat](../) είναι μια συλλογή [DataLabelCollection](../../datalabelcollection/) ετικετών δεδομένων, τότε αυτή η ιδιότητα λαμβάνει ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας ShowLeaderLines για τις νέες ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/). Ο καθορισμός αυτής της ιδιότητας με τιμή ορίζει επίσης αυτήν την τιμή στην ιδιότητα ShowLeaderLines για όλες τις ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/) (π.χ. \"DataLabels.DefaultDataLabelFormat.ShowLeaderLines = val;\" προκαλεί όλες τις DataLabels[i].ShowLeaderLines να είναι ίσες με val).

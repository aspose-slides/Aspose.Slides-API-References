## DataLabelFormat::set_Position(LegendDataLabelPosition) μέθοδος


Αντιπροσωπεύει τη θέση της ετικέτας δεδομένων. Γράψτε [LegendDataLabelPosition](../../legenddatalabelposition/).

```cpp
void Aspose::Slides::Charts::DataLabelFormat::set_Position(LegendDataLabelPosition value) override
```

## Σχόλια


Αν το γονικό αντικείμενο αυτού του [DataLabelFormat](../) είναι μια συλλογή [DataLabelCollection](../../datalabelcollection/) ετικετών δεδομένων, τότε αυτή η ιδιότητα αποκτά ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας Position για τις νέες ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/). Αντιπροσωπεύει τη θέση για τα αντικείμενα [DataLabel](../../datalabel/). Ορισμός αυτής της ιδιότητας με τιμή επίσης ορίζει αυτή την τιμή στην ιδιότητα Position για όλες τις ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/) (π.χ. "DataLabels.DefaultDataLabelFormat.Position = val;" προκαλεί όλες οι DataLabels[i].Position να είναι ίσες με val).

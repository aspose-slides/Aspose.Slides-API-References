## DataLabelFormat::set_ShowLabelValueFromCell(bool) μέθοδος

Αναπαριστά τη συμπεριφορά εμφάνισης της τιμής του κελιού ετικέτας δεδομένων σε ένα συγκεκριμένο διάγραμμα. True εμφανίζει την τιμή του κελιού. False κρύβει. Γράψτε **bool**.

```cpp
void Aspose::Slides::Charts::DataLabelFormat::set_ShowLabelValueFromCell(bool value) override
```

## Παρατηρήσεις

Εάν ο γονέας αυτού του αντικειμένου [DataLabelFormat](../) είναι μια συλλογή [DataLabelCollection](../../datalabelcollection/) ετικετών δεδομένων, τότε αυτή η ιδιότητα λαμβάνει ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας ShowLabelValueFromCell για τις νέες ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/). Η ρύθμιση αυτής της ιδιότητας με τιμή ορίζει επίσης αυτή την τιμή στην ιδιότητα ShowLabelValueFromCell για όλες τις ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/) (π.χ. "DataLabels.DefaultDataLabelFormat.ShowLabelValueFromCell = val;" οδηγεί ώστε όλα τα DataLabels[i].ShowLabelValueFromCell να είναι ίσα με val).

## DataLabelFormat::set_ShowPercentage(bool) μέθοδος

Αντιπροσωπεύει τη συμπεριφορά εμφάνισης της τιμής ποσοστού ετικέτας δεδομένων ενός συγκεκριμένου διαγράμματος. Η τιμή True εμφανίζει την τιμή ποσοστού. Η τιμή False κρύβει. Γράψτε **bool**.

```cpp
void Aspose::Slides::Charts::DataLabelFormat::set_ShowPercentage(bool value) override
```

## Σχόλια

Εάν ο γονέας αυτού του [DataLabelFormat](../) αντικειμένου είναι μια συλλογή [DataLabelCollection](../../datalabelcollection/) ετικετών δεδομένων, τότε αυτή η ιδιότητα παίρνει ή ορίζει την προεπιλεγμένη τιμή της ιδιότητας ShowPercentage για τις νέες ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/). Ορισμός αυτής της ιδιότητας με τιμή θέτει επίσης αυτήν την τιμή στην ιδιότητα ShowPercentage για όλες τις ετικέτες δεδομένων στη συλλογή [DataLabelCollection](../../datalabelcollection/) (π.χ. \"DataLabels.DefaultDataLabelFormat.ShowPercentage = val;\" προκαλεί όλα τα DataLabels[i].ShowPercentage να είναι ίσα με val).

## DataLabelCollection::get_IsVisible() μέθοδος

False σημαίνει ότι η ετικέτα δεδομένων δεν είναι ορατή από προεπιλογή (και έτσι όλες οι σημαίες Show*-flags (ShowValue, ...) της ιδιότητας DefaultDataLabelFormat είναι false). Μόνο για ανάγνωση **bool**.

```cpp
bool Aspose::Slides::Charts::DataLabelCollection::get_IsVisible() override
```

## Παρατηρήσεις

Εάν η ετικέτα δεδομένων είναι ορατή από προεπιλογή, μπορείτε να την κρύψετε από προεπιλογή με τη μέθοδο [Hide()](../hide/). Αλλά εάν η ετικέτα δεδομένων δεν είναι ορατή από προεπιλογή (IsVisible είναι false) μπορείτε να κάνετε την ετικέτα δεδομένων \"visible 
by default\" ορίζοντας τις σημαίες Show*-flags (ShowValue, ...) της ιδιότητας DefaultDataLabelFormat σε κατάσταση true.

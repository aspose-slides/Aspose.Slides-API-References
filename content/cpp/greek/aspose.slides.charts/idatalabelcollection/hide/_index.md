## IDataLabelCollection::Hide() μέθοδος

Κάντε την ετικέτα δεδομένων κρυφή από προεπιλογή ορίζοντας όλες τις Show*-flags (ShowValue, ...) της ιδιότητας DefaultDataLabelFormat σε κατάσταση false. Το IsVisible θα είναι false μετά από αυτό.

```cpp
virtual void Aspose::Slides::Charts::IDataLabelCollection::Hide()=0
```

## Παρατηρήσεις

Εάν η ετικέτα δεδομένων δεν είναι ορατή από προεπιλογή (IsVisible είναι false) μπορείτε να κάνετε την ετικέτα δεδομένων \"visible \nby default\" ορίζοντας τις Show*-flags (ShowValue, ...) της ιδιότητας DefaultDataLabelFormat σε κατάσταση true.

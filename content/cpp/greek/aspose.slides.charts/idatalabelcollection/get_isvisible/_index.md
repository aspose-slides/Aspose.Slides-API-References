## IDataLabelCollection::get_IsVisible() μέθοδος

False σημαίνει ότι η ετικέτα δεδομένων δεν είναι ορατή από προεπιλογή (και έτσι όλες οι σημαίες Show*-flags (ShowValue, ...) της ιδιότητας DefaultDataLabelFormat είναι false). Μόνο ανάγνωση **bool**.

```cpp
virtual bool Aspose::Slides::Charts::IDataLabelCollection::get_IsVisible()=0
```

## Παρατηρήσεις

Αν η ετικέτα δεδομένων είναι ορατή από προεπιλογή, μπορείτε να τη κάνετε κρυφή από προεπιλογή με τη μέθοδο [Hide()](../hide/). Αλλά αν η ετικέτα δεδομένων δεν είναι ορατή από προεπιλογή (IsVisible είναι false) μπορείτε να κάνετε την ετικέτα δεδομένων "visible 
by default" ρυθμίζοντας τις σημαίες Show*-flags (ShowValue, ...) της ιδιότητας DefaultDataLabelFormat σε κατάσταση true.

## BaseChartValue::set_DataSourceType(Aspose::Slides::Charts::DataSourceType) μέθοδος

Καθορίζει αν η ιδιότητα AsCell, AsCells, AsLiteralString ή AsLiteralDouble είναι ενεργή στους απογόνους. Με άλλα λόγια, καθορίζει τον τύπο της τιμής της ιδιότητας Data. Γράψτε [Charts::DataSourceType](../../datasourcetype/).

```cpp
void Aspose::Slides::Charts::BaseChartValue::set_DataSourceType(Aspose::Slides::Charts::DataSourceType value) override
```

## Παρατηρήσεις


Για σημεία στο [ChartDataPointCollection](../../chartdatapointcollection/) αυτή η ιδιότητα είναι μόνο για ανάγνωση. Σε αυτή την περίπτωση, για να αλλάξετε την τιμή αυτής της ιδιότητας, μπορείτε να χρησιμοποιήσετε μία από τις ιδιότητες ChartDataPointCollection.DataSourceTypeFor<...>.

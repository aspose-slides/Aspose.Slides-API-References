## TimeUnitType απαρίθμηση

Αναπαριστά τη βασική μονάδα για τον άξονα κατηγορίας

```cpp
enum class TimeUnitType
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| None | 0 | Οι τιμές θα εμφανιστούν όπως είναι. |
| Days | 1 | Καθορίζει ότι τα δεδομένα του διαγράμματος θα εμφανίζονται σε ημέρες. |
| Months | 2 | Καθορίζει ότι τα δεδομένα του διαγράμματος θα εμφανίζονται σε μήνες. |
| Years | 3 | Καθορίζει ότι τα δεδομένα του διαγράμματος θα εμφανίζονται σε χρόνια. |

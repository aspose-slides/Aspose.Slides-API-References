## TrendlineType enum

Αντιπροσωπεύει τον τύπο της γραμμής τάσης

```cpp
enum class TrendlineType
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Exponential | 0 | Καθορίζει ότι η γραμμή τάσης πρέπει να είναι μια καμπύλη exponential. |
| Linear | 1 | Καθορίζει ότι η γραμμή τάσης πρέπει να είναι μια γραμμή στην μορφή. |
| Logarithmic | 2 | Καθορίζει ότι η γραμμή τάσης πρέπει να είναι μια καμπύλη logarithmic στην μορφή, όπου log είναι ο φυσικός λογάριθμος. |
| MovingAverage | 3 | Καθορίζει ότι η γραμμή τάσης πρέπει να είναι ένας κινητός μέσος με περίοδο Period. |
| Polynomial | 4 | Καθορίζει ότι η γραμμή τάσης πρέπει να είναι μια καμπύλη polynomial τάξης Order στην μορφή. |
| Power | 5 | Καθορίζει ότι η γραμμή τάσης πρέπει να είναι μια καμπύλη power στην μορφή. |

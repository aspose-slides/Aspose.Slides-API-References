## MarkerStyleType enum

Καθορίζει τη μορφή του δείκτη σε σημείο δεδομένων του διαγράμματος.

```cpp
enum class MarkerStyleType
```

| Circle | 0 | Καθορίζει ότι θα σχεδιαστεί ένας κύκλος σε κάθε σημείο δεδομένων. |

| Diamond | 2 | Καθορίζει ότι θα σχεδιαστεί ένας ρόμβος σε κάθε σημείο δεδομένων. |

| None | 4 | Καθορίζει ότι δεν θα σχεδιαστεί τίποτα σε κάθε σημείο δεδομένων. |
| Picture | 5 | Καθορίζει ότι θα σχεδιαστεί μια εικόνα σε κάθε σημείο δεδομένων. |
| Plus | 6 | Καθορίζει ότι θα σχεδιαστεί ένα σύμβολο συν σε κάθε σημείο δεδομένων. |
| Square | 7 | Καθορίζει ότι θα σχεδιαστεί ένα τετράγωνο σε κάθε σημείο δεδομένων. |

## PieSplitType enum

Αντιπροσωπεύει έναν τύπο σημείων διαίρεσης στο δεύτερο πίτα ή μπάρα σε γράφημα τύπου πίτα-εντός-πίτας ή μπάρα-εντός-μπάρας.

```cpp
enum class PieSplitType
```

### Values

| Name | Value | Description |
| --- | --- | --- |
| Default | 0 | Καθορίζει ότι τα σημεία δεδομένων θα χωριστούν χρησιμοποιώντας τον προεπιλεγμένο μηχανισμό για αυτόν τον τύπο γραφήματος. |
| Custom | 1 | Καθορίζει ότι τα σημεία δεδομένων θα χωριστούν μεταξύ της πίτας και του δεύτερου γραφήματος σύμφωνα με τις προσαρμοσμένες τιμές διαίρεσης. |
| ByPercentage | 2 | Καθορίζει ότι τα σημεία δεδομένων θα χωριστούν μεταξύ της πίτας και του δεύτερου γραφήματος τοποθετώντας τα σημεία με ποσοστό μικρότερο από Split Position στο δεύτερο γράφημα. |
| ByPos | 3 | Καθορίζει ότι τα σημεία δεδομένων θα χωριστούν μεταξύ της πίτας και του δεύτερου γραφήματος τοποθετώντας την τελευταία Split Position των σημείων δεδομένων στο δεύτερο γράφημα. |
| ByValue | 4 | Καθορίζει ότι τα σημεία δεδομένων θα χωριστούν μεταξύ της πίτας και του δεύτερου γραφήματος τοποθετώντας τα σημεία δεδομένων με τιμή μικρότερη από Split Position στο δεύτερο γράφημα. |

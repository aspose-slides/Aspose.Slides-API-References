## DataSourceType enum

Τύποι πηγής δεδομένων.

```cpp
enum class DataSourceType
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Worksheet | 0 | Χρησιμοποιήστε αντικείμενα CellXls για την αποθήκευση των XValue, YValue, Value ή BubbleSize του σημείου δεδομένων στο φύλλο εργασίας. |
| StringLiterals | 1 | Χρησιμοποιήστε μεταβλητή συμβολοσειράς γρήγορης πρόσβασης για την αποθήκευση των XValue, YValue, Value ή BubbleSize του σημείου δεδομένων. Το Worksheet δεν χρησιμοποιείται για την αποθήκευση τιμών. |
| DoubleLiterals | 2 | Χρησιμοποιήστε μεταβλητή τύπου double γρήγορης πρόσβασης για την αποθήκευση των XValue, YValue, Value ή BubbleSize του σημείου δεδομένων. Το Worksheet δεν χρησιμοποιείται για την αποθήκευση τιμών. |

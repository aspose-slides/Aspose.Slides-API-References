## Trendline::set_Forward(double) μέθοδος

Καθορίζει τον αριθμό των κατηγοριών (ή μονάδες σε scatter chart) που η trendline επεκτείνεται μετά τα δεδομένα της series που εξετάζεται. Σε scatter και non-scatter charts, η τιμή πρέπει να είναι οποιαδήποτε μη αρνητική τιμή. Γράψτε **double**.

void Aspose::Slides::Charts::Trendline::set_Forward(double value) override

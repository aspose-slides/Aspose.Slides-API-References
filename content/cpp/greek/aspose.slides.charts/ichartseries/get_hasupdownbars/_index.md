## IChartSeries::get_HasUpDownBars() μέθοδος

Καθορίζει εάν το Διάγραμμα Γραμμής ή Stock-chart έχει μπάρες ανοδ/καθοδ. Αυτή είναι η ιδιότητα όχι μόνο αυτής της σειράς αλλά και όλων των σειρών της γονικής ομάδας σειρών - αυτή είναι η προβολή της αντίστοιχης ιδιότητας ομάδας. Άρα αυτή η ιδιότητα είναι μόνο για ανάγνωση. Χρησιμοποιήστε την ιδιότητα ParentSeriesGroup για πρόσβαση στη γονική ομάδα σειρών. Χρησιμοποιήστε [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get_UpDownBars()->get(set)_HasUpDownBars() ιδιότητα ανάγνωση/εγγραφή για αλλαγή τιμής. Χρησιμοποιήστε την ιδιότητα [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get_UpDownBars() για μορφοποίηση των μπαρών ανοδ/καθοδ. Μόνο για ανάγνωση **bool**.

```cpp
virtual bool Aspose::Slides::Charts::IChartSeries::get_HasUpDownBars()=0
```

## Παρατηρήσεις

Αυτή είναι η προβολή της ιδιότητας [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get_UpDownBars()->get(set)_HasUpDownBars().

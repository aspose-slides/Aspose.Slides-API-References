## ErrorBarValueType enum

Αναπαριστά τον τύπο της τιμής της γραμμής σφάλματος

```cpp
enum class ErrorBarValueType
```

### Τιμές

| Name | Value | Description |
| --- | --- | --- |
| Custom | 0 | Καθορίζει ότι το μήκος των γραμμών σφάλματος θα καθορίζεται από τα στοιχεία Plus και Minus. |
| Fixed | 1 | Καθορίζει ότι το μήκος των γραμμών σφάλματος θα είναι η σταθερή τιμή που καθορίζεται από Error Bar Value. |
| Percentage | 2 | Καθορίζει ότι το μήκος των γραμμών σφάλματος θα είναι το ποσοστό Error Bar Value των δεδομένων. |
| StandardDeviation | 3 | Καθορίζει ότι το μήκος των γραμμών σφάλματος θα είναι οι τυπικές αποκλίσεις Error Bar Value των δεδομένων. |
| StandardError | 4 | Καθορίζει ότι το μήκος των γραμμών σφάλματος θα είναι τα τυπικά σφάλματα Error Bar Value των δεδομένων. |

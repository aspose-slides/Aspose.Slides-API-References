## IChartTextBlockFormat::set_MarginLeft(double) μέθοδος

Ορίζει το αριστερό περιθώριο (σημεία) σε ένα [TextFrame](../../../aspose.slides/textframe/). Η αλλαγή αυτής της ιδιότητας μπορεί να έχει ορισμένη επίδραση μόνο για αυτά τα τμήματα γραφήματος: [DataLabel](../../datalabel/) και [DataLabelFormat](../../datalabelformat/) (πλήρης υποστήριξη στο PowerPoint 2013; στο PowerPoint 2007 δεν υπάρχει αποτέλεσμα στην απόδοση). Γράψτε **double**.

virtual void Aspose::Slides::Charts::IChartTextBlockFormat::set_MarginLeft(double value)=0

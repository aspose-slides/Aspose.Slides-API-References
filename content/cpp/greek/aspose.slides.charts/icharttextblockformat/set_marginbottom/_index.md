## IChartTextBlockFormat::set_MarginBottom(double) μέθοδος

Ορίζει το κάτω περιθώριο (σημεία) σε ένα [TextFrame](../../../aspose.slides/textframe/). Η αλλαγή αυτής της ιδιότητας μπορεί να έχει ορισμένη επίδραση μόνο για τα ακόλουθα τμήματα του διαγράμματος: [DataLabel](../../datalabel/) και [DataLabelFormat](../../datalabelformat/) (πλήρης υποστήριξη στο PowerPoint 2013· στο PowerPoint 2007 δεν υπάρχει αποτέλεσμα στην απόδοση). Γράψτε **double**.

```cpp
virtual void Aspose::Slides::Charts::IChartTextBlockFormat::set_MarginBottom(double value)=0
```

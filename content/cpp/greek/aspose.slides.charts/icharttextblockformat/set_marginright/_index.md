## IChartTextBlockFormat::set_MarginRight(double) μέθοδος


Ορίζει το δεξί περιθώριο (σημεία) σε μια [TextFrame](../../../aspose.slides/textframe/). Η αλλαγή αυτής της ιδιότητας μπορεί να έχει συγκεκριμένη επίδραση μόνο για αυτά τα μέρη του διαγράμματος: [DataLabel](../../datalabel/) και [DataLabelFormat](../../datalabelformat/) (πλήρης υποστήριξη στο PowerPoint 2013· στο PowerPoint 2007 δεν υπάρχει αποτέλεσμα στην απόδοση). Γράψτε **double**.

```cpp
virtual void Aspose::Slides::Charts::IChartTextBlockFormat::set_MarginRight(double value)=0
```

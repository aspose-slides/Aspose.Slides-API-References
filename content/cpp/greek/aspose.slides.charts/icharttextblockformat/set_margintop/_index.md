## IChartTextBlockFormat::set_MarginTop(double) μέθοδος

Ορίζει το άνω περιθώριο (σημείων) σε ένα [TextFrame](../../../aspose.slides/textframe/). Η αλλαγή αυτής της ιδιότητας μπορεί να έχει κάποια επίδραση μόνο για αυτά τα μέρη του διαγράμματος: [DataLabel](../../datalabel/) και [DataLabelFormat](../../datalabelformat/) (πλήρης υποστήριξη στο PowerPoint 2013· στο PowerPoint 2007 δεν υπάρχει αποτέλεσμα στην απόδοση). Γράψτε **double**.

virtual void Aspose::Slides::Charts::IChartTextBlockFormat::set_MarginTop(double value)=0

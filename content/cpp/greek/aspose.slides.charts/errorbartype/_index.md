## ErrorBarType enum

Αναπαριστά τον τύπο της γραμμής σφάλματος

```cpp
enum class ErrorBarType
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Both | 0 | Καθορίζει ότι οι γραμμές σφάλματος θα εμφανίζονται στις θετικές και αρνητικές κατευθύνσεις. |
| Minus | 1 | Καθορίζει ότι οι γραμμές σφάλματος θα εμφανίζονται μόνο στην αρνητική κατεύθυνση. |
| Plus | 2 | Καθορίζει ότι οι γραμμές σφάλματος θα εμφανίζονται μόνο στην θετική κατεύθυνση. |

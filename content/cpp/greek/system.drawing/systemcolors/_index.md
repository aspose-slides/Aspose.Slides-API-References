## SystemColors κλάση

Κλάση που παρέχει ένα σύνολο προδημιουργημένων [Color](../color/) αντικειμένων που αντιπροσωπεύουν τα χρώματα των στοιχείων εμφάνισης [Windows](../../system.windows/). Πρόκειται για έναν στατικό τύπο χωρίς υπηρεσίες δημιουργίας στιγμιοτύπων. Δεν πρέπει ποτέ να δημιουργείτε στιγμιότυπα του με κανέναν τρόπο.

```cpp
class SystemColors
```

## RectangleF::RectangleF() Κατασκευαστής

Δημιουργεί ένα νέο στιγμιότυπο του αντικειμένου [RectangleF](../) που αντιπροσωπεύει ένα ορθογώνιο με συντεταγμένες X και Y και τιμές πλάτους και hegiht ορισμένες στο 0.

```cpp
System::Drawing::RectangleF::RectangleF()
```

## RectangleF::RectangleF(float, float, float, float) Κατασκευαστής

Δημιουργεί ένα νέο στιγμιότυπο του αντικειμένου [RectangleF](../) που αντιπροσωπεύει ένα ορθογώνιο με τις καθορισμένες συντεταγμένες του άνω αριστερού γωνίου του και πλάτος και ύψος.

```cpp
System::Drawing::RectangleF::RectangleF(float x, float y, float width, float height)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| x | **float** | Μια τιμή της συντεταγμένης X του άνω αριστερού γωνία του ορθογωνίου |
| y | **float** | Μια τιμή της συντεταγμένης Y του άνω αριστερού γωνία του ορθογωνίου |
| width | **float** | Το πλάτος του ορθογωνίου |
| height | **float** | Το ύψος του ορθογωνίου |

## RectangleF::RectangleF(const PointF&, const SizeF&) Κατασκευαστής

Δημιουργεί ένα νέο στιγμιότυπο του αντικειμένου [RectangleF](../) που αντιπροσωπεύει ένα ορθογώνιο με τις συντεταγμένες του άνω αριστερού γωνίου που καθορίζονται ως μια παρουσία της κλάσης [PointF](../../pointf/) και το πλάτος και το ύψος του ως μια παρουσία της κλάσης [SizeF](../../sizef/).

```cpp
System::Drawing::RectangleF::RectangleF(const PointF &location, const SizeF &size)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| location | const [PointF](../../pointf/)\& | Καθορίζει τη θέση του άνω αριστερού γωνία του ορθογωνίου |
| size | const [SizeF](../../sizef/)\& | Καθορίζει το πλάτος και το hegiht του ορθογωνίου |

## RectangleF::RectangleF(const Rectangle&) Κατασκευαστής

Δημιουργεί ένα νέο στιγμιότυπο του αντικειμένου [RectangleF](../) που αντιπροσωπεύει το ορθογώνιο ισοδύναμο με το καθορισμένο.

```cpp
System::Drawing::RectangleF::RectangleF(const Rectangle &rect)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| rect | const [Rectangle](../../rectangle/)\& | Μια παρουσία της κλάσης [Rectangle](../../rectangle/) που καθορίζει τη θέση και το μέγεθος του ορθογωνίου που θα αντιπροσωπευθεί από το αντικείμενο που κατασκευάζεται |

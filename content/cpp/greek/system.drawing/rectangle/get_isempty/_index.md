## Rectangle::get_IsEmpty() const μέθοδος

Determines if X and Y coordinates of the upper left corner of the παραλληλόγραμμου represented by the current object as well as its width and height have values of 0.

```cpp
bool System::Drawing::Rectangle::get_IsEmpty() const
```

### Τιμή Επιστροφής

True if X and Y coordinates of the upper left corner of the παραλληλόγραμμου represented by the current object as well as its width and height have values of 0, otherwise - false

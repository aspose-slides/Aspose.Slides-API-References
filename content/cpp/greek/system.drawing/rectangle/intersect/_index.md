## Rectangle::Intersect(const Rectangle\&) μέθοδος

Αντικαθιστά το ορθογώνιο που αντιπροσωπεύεται από το τρέχον αντικείμενο με το ορθογώνιο που προκύπτει από τη διατομή του με το ορθογώνιο που αντιπροσωπεύεται από το συγκεκριμένο αντικείμενο.

```cpp
void System::Drawing::Rectangle::Intersect(const Rectangle &rect)
```

### Παράμετροι

| Parameter | Type | Description |
| --- | --- | --- |
| rect | const [Rectangle](../)\& | Το αντικείμενο [Rectangle](../) που αντιπροσωπεύει το ορθογώνιο για τη διατομή του ορθογωνίου που αντιπροσωπεύεται από το τρέχον αντικείμενο |

## Rectangle::Intersect(const Rectangle\&, const Rectangle\&) μέθοδος

Επιστρέφει ένα ορθογώνιο που είναι αποτέλεσμα της διατομής των καθορισμένων ορθογώνιων.

```cpp
static Rectangle System::Drawing::Rectangle::Intersect(const Rectangle &a, const Rectangle &b)
```

### Παράμετροι

| Parameter | Type | Description |
| --- | --- | --- |
| a | const [Rectangle](../)\& | Το πρώτο ορθογώνιο προς διατομή |
| b | const [Rectangle](../)\& | Το δεύτερο ορθογώνιο προς διατομή |

### Τιμή Επιστροφής

Το αποτέλεσμα της διατομής του **a** με **b**

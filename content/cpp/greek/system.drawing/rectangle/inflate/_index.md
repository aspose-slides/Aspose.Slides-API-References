## Rectangle::Inflate(int, int) μέθοδος


Αυξάνει το πλάτος και το ύψος του ορθογωνίου που αντιπροσωπεύεται από το τρέχον αντικείμενο, διατηρώντας τη θέση του γεωμετρικού κέντρου του ορθογωνίου. Το πλάτος και το ύψος αυξάνονται και στις δύο κατευθύνσεις κατά τα καθορισμένα ποσά.

```cpp
void System::Drawing::Rectangle::Inflate(int width, int height)
```


### Παράμετροι

| Parameter | Type | Description |
| --- | --- | --- |
| width | int | Το ποσό κατά το οποίο το πλάτος του ορθογωνίου πρέπει να αυξηθεί και στις δύο κατευθύνσεις |
| height | int | Το ποσό κατά το οποίο το ύψος του ορθογωνίου πρέπει να αυξηθεί και στις δύο κατευθύνσεις |

## StringFormatFlags enum

Specifies the display and layout information for the strings of text.

```cpp
enum class StringFormatFlags
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| DirectionRightToLeft | 1 | Κατεύθυνση κειμένου από δεξιά προς αριστερά. |
| DirectionVertical | 2 | [Text](../../system.drawing.text/) ευθυγραμμίζεται κάθετα. |
| FitBlackBox | 4 | Μέρη χαρακτήρων μπορούν να υπερχειλίζουν το πλαίσιο διάταξης. |
| DisplayFormatControl | 32 | Οι χαρακτήρες ελέγχου εμφανίζονται. |
| NoFontFallback | 1024 | [Font](../font/) η εφεδρεία γραμματοσειράς είναι απενεργοποιημένη. |
| MeasureTrailingSpaces | 2048 | Εισάγει το διαστημικό χαρακτήρα στο τέλος κάθε γραμμής. |
| NoWrap | 4096 | [Text](../../system.drawing.text/) η περιτύλιξη είναι απενεργοποιημένη. |
| LineLimit | 8192 | Ολόκληρες γραμμές διατάσσονται στο ορθογώνιο μορφοποίησης. |
| NoClip | 16384 | Επιτρέπεται η εμφάνιση των υπερέχοντων μερών των γλύφων. |

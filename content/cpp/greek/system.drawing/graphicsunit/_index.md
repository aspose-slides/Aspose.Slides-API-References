## GraphicsUnit enum

Αντιπροσωπεύει μια μονάδα μέτρησης.

```cpp
enum class GraphicsUnit
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| World | 0 | Συντεταγμένη κόσμου (μη φυσική μονάδα). |
| Display | 1 | Οι μονάδες μέτρησης της συσκευής εμφάνισης. |
| Pixel | 2 | Ένα pixel συσκευής. |
| Point | 3 | Σημείο εκτυπωτή (1/72 ιντσών). |
| Inch | 4 | Ένα ίντσα. |
| Document | 5 | 1/300 ιντσών. |
| Millimeter | 6 | Ένα χιλιοστόμετρο. |

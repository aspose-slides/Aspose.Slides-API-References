## PointF::Add(const PointF\&, const SizeF\&) μέθοδος


Προσθέτει τις τιμές πλάτους και ύψους του καθορισμένου [SizeF](../../sizef/) αντικειμένου στις τιμές των συντεταγμένων X και Y του καθορισμένου [PointF](../) αντικειμένου αντίστοιχα.

```cpp
static PointF System::Drawing::PointF::Add(const PointF &point, const SizeF &size)
```


### Παραμέτρους

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| point | const [PointF](../)\& | Το σημείο για μετάφραση |
| size | const [SizeF](../../sizef/)\& | Το [SizeF](../../sizef/) αντικείμενο που καθορίζει τις τιμές προς προσθήκη στις τιμές των συντεταγμένων του **point** |

### Τιμή Επιστροφής

Ένα νέο [PointF](../) αντικείμενο του οποίου η τιμή της συντεταγμένης X είναι ίση με το άθροισμα της τιμής της συντεταγμένης X του **point** και της τιμής πλάτους του **size** και η τιμή της συντεταγμένης Y είναι ίση με το άθροισμα της τιμής της συντεταγμένης Y του **point** και της τιμής ύψους του **size**.

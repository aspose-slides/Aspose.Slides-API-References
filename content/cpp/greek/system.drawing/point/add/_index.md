## Point::Add(const Point\&, const Size\&) μέθοδος

Προσθέτει τις τιμές πλάτους και ύψους του συγκεκριμένου αντικείμενου [Size](../../size/) στις τιμές συντεταγμένων X και Y του συγκεκριμένου αντικείμενου [Point](../) αντίστοιχα.

```cpp
static Point System::Drawing::Point::Add(const Point &point, const Size &size)
```

### Arguments

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| point | const [Point](../)\& | Το **point** για μεταφορά |
| size | const [Size](../../size/)\& | Το αντικείμενο [Size](../../size/) που καθορίζει τις τιμές που θα προστεθούν στις τιμές των συντεταγμένων του **point** |

### Return Value

Ένα νέο αντικείμενο [Point](../) του οποίου η τιμή συντεταγμένης X είναι ίση με το άθροισμα της τιμής συντεταγμένης X του **point** και της τιμής πλάτους του **size** και η τιμή συντεταγμένης Y είναι ίση με το άθροισμα της τιμής συντεταγμένης Y του **point** και της τιμής ύψους του **size**.

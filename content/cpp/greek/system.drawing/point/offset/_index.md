## Point::Offset(int, int) μέθοδος

Μετατοπίζει την τιμή των συντεταγμένων X και Y που αντιπροσωπεύεται από το τρέχον αντικείμενο κατά τις καθορισμένες τιμές.

```cpp
void System::Drawing::Point::Offset(int dx, int dy)
```

### Ορίσματα

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| dx | int | Η τιμή που θα χρησιμοποιηθεί για τη μετατόπιση της συντεταγμένης X |
| dy | int | Η τιμή που θα χρησιμοποιηθεί για τη μετατόπιση της συντεταγμένης Y |

## Point::Offset(Point) μέθοδος

Μετατοπίζει τις συντεταγμένες X και Y που αντιπροσωπεύονται από το τρέχον αντικείμενο κατά τις τιμές των συντεταγμένων X και Y που αντιπροσωπεύονται από το καθορισμένο αντικείμενο [Point](../) αντίστοιχα.

```cpp
void System::Drawing::Point::Offset(Point point)
```

### Ορίσματα

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| point | [Point](../) | Το αντικείμενο [Point](../) που καθορίζει τις τιμές για τη μετατόπιση των συντεταγμένων X και Y |

## Point::Subtract(const Point\&, const Size\&) μέθοδος

Αφαιρεί τις τιμές πλάτους και ύψους του συγκεκριμένου [Size](../../size/) αντικειμένου από τις τιμές των συντεταγμένων X και Y του συγκεκριμένου [Point](../) αντικειμένου αντίστοιχα.

```cpp
static Point System::Drawing::Point::Subtract(const Point &point, const Size &size)
```

### Παράμετροι

| Parameter | Type | Description |
| --- | --- | --- |
| point | const [Point](../)\& | Το σημείο προς μετάφραση |
| size | const [Size](../../size/)\& | Το [Size](../../size/) αντικείμενο που καθορίζει τις τιμές που θα αφαιρεθούν από τις τιμές των συντεταγμένων του **point** |

### Τιμή Επιστροφής

Ένα νέο [Point](../) αντικείμενο του οποίου η τιμή της συντεταγμένης X είναι ίση με το αποτέλεσμα της αφαίρεσης της τιμής του πλάτους του **size** από την τιμή της συντεταγμένης X του **point** και η τιμή της συντεταγμένης Y είναι ίση με το αποτέλεσμα της αφαίρεσης της τιμής του ύψους του **size** από την τιμή της συντεταγμένης Y του **point**

## Point::Point() κατασκευαστής

Δημιουργεί ένα νέο αντικείμενο [Point](../) και αρχικοποιεί τις τιμές των συντεταγμένων X και Y του με 0.

```cpp
System::Drawing::Point::Point()
```

## Point::Point(int, int) κατασκευαστής

Δημιουργεί ένα νέο αντικείμενο [Point](../) και το αρχικοποιεί με τις καθορισμένες τιμές.

```cpp
System::Drawing::Point::Point(int x, int y)
```

### Παράμετροι

| Parameter | Type | Description |
| --- | --- | --- |
| x | int | Η τιμή της συντεταγμένης X |
| y | int | Η τιμή της συντεταγμένης Y |

## Point::Point(const Size\&) κατασκευαστής

Δημιουργεί ένα νέο αντικείμενο [Point](../) και αρχικοποιεί τις τιμές των συντεταγμένων X και Y του με τις τιμές του πλάτους και του ύψους του καθορισμένου αντικειμένου [SizeF](../../sizef/) αντίστοιχα.

```cpp
System::Drawing::Point::Point(const Size &size)
```

### Παράμετροι

| Parameter | Type | Description |
| --- | --- | --- |
| size | const [Size](../../size/)\& | Ένα αντικείμενο [SizeF](../../sizef/) του οποίου οι τιμές πλάτους και ύψους χρησιμοποιούνται για την αρχικοποίηση των τιμών των συντεταγμένων X και Y του αντικειμένου [Point](../) που δημιουργείται |

## Point::Point(int) κατασκευαστής

Δημιουργεί ένα νέο αντικείμενο [Point](../) και αρχικοποιεί την τιμή της συντεταγμένης X με μια τιμή που προέρχεται από τα υψηλά 16 bits του καθορισμένου 32-bit ακεραίου και την τιμή της συντεταγμένης Y με μια τιμή που προέρχεται από τα χαμηλά 16 bits του καθορισμένου 32-bit ακεραίου.

```cpp
System::Drawing::Point::Point(int dw)
```

### Παράμετροι

| Parameter | Type | Description |
| --- | --- | --- |
| dw | int | Η 32-bit ακέραια τιμή της οποίας τα υψηλά 16 bits καθορίζουν την τιμή της συντεταγμένης X και τα χαμηλά 16 bits καθορίζουν την τιμή της συντεταγμένης Y του αντικειμένου που δημιουργείται |

## Κλάση Point

Αναπαριστά ένα ζεύγος ακεραίων συντεταγμένων X και Y ενός σημείου σε επίπεδο 2 διαστάσεων. Αυτός ο τύπος θα πρέπει να καταλαμβάνεται στη στοίβα και να μεταβιβάζεται σε συναρτήσεις με τιμή ή με αναφορά. Ποτέ μην χρησιμοποιείτε την κλάση [System::SmartPtr](../../system/smartptr/) για τη διαχείριση αντικειμένων αυτού του τύπου.

```cpp
class Point
```

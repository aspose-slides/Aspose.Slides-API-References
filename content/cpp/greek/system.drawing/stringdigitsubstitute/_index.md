## StringDigitSubstitute enum

Καθορίζει πώς υποκαθίστανται τα ψηφία σε μια συμβολοσειρά σύμφωνα με την τοπική ρύθμιση ή τη γλώσσα.

```cpp
enum class StringDigitSubstitute
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| User | 0 | Υποκατάσταση που ορίζεται από τον χρήστη. |
| None | 1 | Η υποκατάσταση είναι απενεργοποιημένη. |
| National | 2 | Αντικαταστήστε τα ψηφία που αντιστοιχούν στην επίσημη εθνική γλώσσα της τοπικής ρύθμισης του χρήστη. |
| Traditional | 3 | Αντικαταστήστε τα ψηφία που αντιστοιχούν στη γλώσσα ή το γράψιμο του χρήστη, το οποίο μπορεί να μην είναι το ίδιο με την επίσημη εθνική γλώσσα της τοπικής ρυθμίσης του χρήστη. |

## Size::Size(const Point\&) Κατασκευαστής

Δημιουργεί ένα νέο αντικείμενο [Size](../) και αρχικοποιεί τις τιμές πλάτους και ύψους του με τις τιμές των συντεταγμένων X και Y του συγκεκριμένου σημείου αντίστοιχα.

```cpp
System::Drawing::Size::Size(const Point &point)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| point | const [Point](../../point/)\& | Ένα αντικείμενο [Point](../../point/) του οποίου οι συντεταγμένες X και Y χρησιμοποιούνται για την αρχικοποίηση της τιμής πλάτους και ύψους του αντικειμένου [Size](../) που δημιουργείται |

## Size::Size(int, int) Κατασκευαστής

Δημιουργεί ένα νέο αντικείμενο [Size](../) και το αρχικοποιεί με την καθορισμένη τιμή.

```cpp
System::Drawing::Size::Size(int width, int height)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| width | int | Η τιμή για το πλάτος |
| height | int | Η τιμή του ύψους |

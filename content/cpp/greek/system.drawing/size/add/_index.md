## Size::Add(const Size\&, const Size\&) μέθοδος

Επιστρέφει ένα νέο [Size](../) αντικείμενο που είναι το άθροισμα του καθορισμένου [Size](../) αντικειμένου, δηλαδή το οποίο η τιμή του πλάτους είναι ίση με το άθροισμα των τιμών του πλάτους των καθορισμένων αντικειμένων και η τιμή του ύψους είναι ίση με το άθροισμα των τιμών του ύψους των καθορισμένων αντικειμένων.

```cpp
static Size System::Drawing::Size::Add(const Size &size1, const Size &size2)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| size1 | const [Size](../)\& | Το πρώτο τελεστέο |
| size2 | const [Size](../)\& | Το δεύτερο τελεστέο |

### Τιμή Επιστροφής

Το [Size](../) αντικείμενο το οποίο είναι το άθροισμα του **size1** και του **size2**

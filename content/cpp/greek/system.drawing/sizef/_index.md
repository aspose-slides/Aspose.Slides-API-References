## SizeF κλάση


Represents a pair of single-precision floating point values that represent width and height of an image. This type should be allocated on stack and passed to functions by value or by reference. Never use [System::SmartPtr](../../system/smartptr/) κλάση to manage objects of this type.

```cpp
class SizeF
```

## SizeF::SizeF(const PointF\&) κατασκευαστής

Δημιουργεί ένα νέο [SizeF](../) αντικείμενο και αρχικοποιεί τις τιμές του width και του height με τις τιμές των συντεταγμένων X και Y του καθορισμένου σημείου αντίστοιχα.

```cpp
System::Drawing::SizeF::SizeF(const PointF &point)
```

### Παράμετροι

| Parameter | Type | Description |
| --- | --- | --- |
| point | const [PointF](../../pointf/)\& | Ένα [Point](../../point/) αντικείμενο του οποίου οι συντεταγμένες X και Y χρησιμοποιούνται για την αρχικοποίηση της τιμής width και height του [SizeF](../) αντικειμένου που δημιουργείται |

## SizeF::SizeF(float, float) κατασκευαστής

Δημιουργεί ένα νέο [SizeF](../) αντικείμενο και το αρχικοποιεί με την καθορισμένη τιμή.

```cpp
System::Drawing::SizeF::SizeF(float width, float height)
```

### Παράμετροι

| Parameter | Type | Description |
| --- | --- | --- |
| width | **float** | Η τιμή για το width |
| height | **float** | Η τιμή του height |

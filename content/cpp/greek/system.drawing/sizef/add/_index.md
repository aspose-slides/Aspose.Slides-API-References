## SizeF::Add(const SizeF\&, const SizeF\&) μέθοδος

Επιστρέφει ένα νέο [SizeF](../) αντικείμενο που είναι άθροισμα των καθορισμένων [SizeF](../) αντικειμένων, δηλαδή η τιμή του πλάτους είναι ίση με το άθροισμα των τιμών πλάτους των καθορισμένων αντικειμένων και η τιμή του ύψους είναι ίση με το άθροισμα των τιμών ύψους των καθορισμένων αντικειμένων.

```cpp
static SizeF System::Drawing::SizeF::Add(const SizeF &size1, const SizeF &size2)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| size1 | const [SizeF](../)\& | Το πρώτο τελεστέο |
| size2 | const [SizeF](../)\& | Το δεύτερο τελεστέο |

### Τιμή Επιστροφής

Το [SizeF](../) αντικείμενο που είναι το άθροισμα του **size1** και του **size2**

## StringTrimming enum

Καθορίζει πώς θα πρέπει να περικόπτονται οι χαρακτήρες από μια συμβολοσειρά που δεν χωράει στο σχήμα διάταξης.

```cpp
enum class StringTrimming
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| None | 0 | Καμία περικοπή. |
| Character | 1 | Περικοπή μέχρι τον πλησιέστερο χαρακτήρα. |
| Word | 2 | Περικοπή μέχρι τη πλησιέστερη λέξη. |
| EllipsisCharacter | 3 | Περικοπή μέχρι τον πλησιέστερο χαρακτήρα και εισαγωγή ελλειπτικού στο τέλος της συμβολοσειράς. |
| EllipsisWord | 4 | Περικοπή μέχρι τη πλησιέστερη λέξη και εισαγωγή ελλειπτικού στο τέλος της συμβολοσειράς. |
| EllipsisPath | 5 | Το κέντρο αφαιρείται από τις περικομμένες γραμμές και αντικαθίσταται με ελλειπτικό. Διατηρήστε όσο το δυνατόν περισσότερο το τελευταίο τμήμα της γραμμής που διαχωρίζεται με καθέτους. |

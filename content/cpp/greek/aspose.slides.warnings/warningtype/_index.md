## WarningType enum

Αντιπροσωπεύει έναν τύπο προειδοποίησης.

```cpp
enum class WarningType
```

### Values

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| SourceFileCorruption | 0 | Ένα πρόβλημα εντοπίστηκε στο αρχικό έγγραφο, το οποίο καθιστά πολύ πιθανό ότι το έγγραφο δεν θα μπορέσει να ανοίξει εάν αποθηκευτεί στην αρχική του μορφή. |
| DataLoss | 1 | Το κείμενο/διάγραμμα/εικόνα ή άλλα δεδομένα θα λείπουν εντελώς είτε από το δέντρο του εγγράφου μετά τη φόρτωση, είτε από το δημιουργημένο έγγραφο μετά την αποθήκευση. |
| MajorFormattingLoss | 2 | Σημαντική απώλεια μορφοποίησης. |
| MinorFormattingLoss | 3 | Μικρή απώλεια μορφοποίησης. |
| CompatibilityIssue | 4 | Αυτό είναι ένα γνωστό ζήτημα που θα εμποδίσει το άνοιγμα του εγγράφου από ορισμένους user agents ή προηγούμενες εκδόσεις των user agents. |
| UnexpectedContent | 99 | Κάποιο περιεχόμενο στο αρχικό έγγραφο δεν μπόρεσε να αναγνωριστεί (π.χ. δεν υποστηρίζεται), αυτό ενδέχεται ή όχι να προκαλέσει προβλήματα ή να οδηγήσει σε απώλεια δεδομένων/μορφοποίησης. |

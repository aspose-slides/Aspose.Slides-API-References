## Κλάσεις

| Κλάση | Περιγραφή |
| --- | --- |
| [HttpRequestCachePolicy](./httprequestcachepolicy/) | Πολιτική προσωρινής μνήμης HTTP που εκφράζει τη σημασιολογία της προσωρινής μνήμης HTTP σύμφωνα με το RFC2616. Τα αντικείμενα αυτού του κλάσης πρέπει να εκχωρούνται μόνο χρησιμοποιώντας τη συνάρτηση [System::MakeObject()](../system/makeobject/). Ποτέ μην δημιουργείτε παρουσία αυτού του τύπου στη στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης και/ή αποτυχίες ελέγχου. Πάντα τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να τον περάσετε σε συναρτήσεις ως όρισμα. |
| [RequestCachePolicy](./requestcachepolicy/) | Κοινή πολιτική προσωρινής μνήμης αιτήματος που χρησιμοποιείται για προσωρινή μνήμη του [Http](../system.net.http/), FTP κ.λπ. Τα αντικείμενα αυτού του κλάσης πρέπει να εκχωρούνται μόνο χρησιμοποιώντας τη συνάρτηση [System::MakeObject()](../system/makeobject/). Ποτέ μην δημιουργείτε αντίγραφο αυτού του τύπου στη στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης και/ή αποτυχίες ελέγχου. Πάντα τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να την περάσετε σε συναρτήσεις ως όρισμα. |

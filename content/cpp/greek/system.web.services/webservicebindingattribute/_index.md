## WebServiceBindingAttribute κλάση

Χρησιμοποιείται για τη δήλωση ενός binding που ορίζει μία ή περισσότερες μεθόδους της υπηρεσίας XML [Web](../../system.web/). Αντικείμενα αυτής της κλάσης πρέπει να δημιουργούνται μόνο μέσω της συνάρτησης [System::MakeObject()](../../system/makeobject/). Ποτέ δεν δημιουργείτε παρουσία αυτού του τύπου στη στοίβα ή με τη χρήση του τελεστή new, καθώς θα προκύψουν σφάλματα χρόνου εκτέλεσης και/ή σφάλματα αδυνατοποίησης. Πάντα τυλίξτε αυτήν την κλάση σε pointer [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον pointer για να τον περάσετε στις συναρτήσεις ως όρισμα.

```cpp
class WebServiceBindingAttribute : public System::Attribute
```

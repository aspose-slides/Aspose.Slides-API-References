## WebServiceAttribute κλάση

Προσθέτει πρόσθετες πληροφορίες στην υπηρεσία XML [Web](../../system.web/). Τα αντικείμενα αυτής της κλάσης πρέπει να εκχωρούνται μόνο χρησιμοποιώντας τη συνάρτηση [System::MakeObject()](../../system/makeobject/). Ποτέ δεν πρέπει να δημιουργείτε παρουσία αυτού του τύπου στη στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης και/ή σφάλματα αστοχίας. Πάντα να τυλίγετε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και να χρησιμοποιείτε αυτόν τον δείκτη για να τη μεταβιβάζετε στις συναρτήσεις ως όρισμα.

```cpp
class WebServiceAttribute : public System::Attribute
```

## XmlNode::get_Prefix() μέθοδος

Επιστρέφει το πρόθεμα χώρου ονομάτων αυτού του κόμβου.

```cpp
virtual String System::Xml::XmlNode::get_Prefix()
```

### Τιμή Επιστροφής

Το πρόθεμα χώρου ονομάτων αυτού του κόμβου. Για παράδειγμα, **Prefix** είναι **bk** για το στοιχείο **<bk:book>**. Εάν δεν υπάρχει πρόθεμα, αυτή η μέθοδος επιστρέφει [String::Empty](../../../system/string/empty/).

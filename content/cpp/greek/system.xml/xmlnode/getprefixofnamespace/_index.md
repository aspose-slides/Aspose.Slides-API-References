## XmlNode::GetPrefixOfNamespace(String) μέθοδος


Αναζητά τη πλησιέστερη δήλωση **xmlns** για το συγκεκριμένο URI ονοματοχώρου που είναι εντός εμβέλειας του τρέχοντος κόμβου και επιστρέφει το πρόθεμα που ορίζεται σε αυτή τη δήλωση.

```cpp
virtual String System::Xml::XmlNode::GetPrefixOfNamespace(String namespaceURI)
```


### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| namespaceURI | [String](../../../system/string/) | Το URI του ονοματικού χώρου του οποίου το πρόθεμα θέλετε να βρείτε. |

### Τιμή Επιστροφής

Το πρόθεμα για το καθορισμένο URI ονοματοχώρου.

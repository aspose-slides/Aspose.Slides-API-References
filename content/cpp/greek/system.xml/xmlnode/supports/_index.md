## XmlNode::Supports(String, String) method


Δοκιμάζει αν η υλοποίηση του DOM υλοποιεί μια συγκεκριμένη δυνατότητα.

```cpp
virtual bool System::Xml::XmlNode::Supports(String feature, String version)
```


### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| feature | [String](../../../system/string/) | Το όνομα του πακέτου της δυνατότητας που θα δοκιμαστεί. Αυτό το όνομα δεν διακρίνει πεζά-κεφαλαία. |
| version | [String](../../../system/string/) | Ο αριθμός έκδοσης του ονόματος του πακέτου που θα δοκιμαστεί. Εάν η έκδοση δεν έχει καθοριστεί (null), η υποστήριξη οποιασδήποτε έκδοσης της δυνατότητας προκαλεί την επιστροφή true από τη μέθοδο. |

### Τιμή Επιστροφής

**true** εάν η δυνατότητα είναι υλοποιημένη στην καθορισμένη έκδοση· διαφορετικά, **false**.

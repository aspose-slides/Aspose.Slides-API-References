## XmlNode::GetNamespaceOfPrefix(String) μέθοδος

```cpp
virtual String System::Xml::XmlNode::GetNamespaceOfPrefix(String prefix)
```

### Παράμετροι

| Parameter | Type | Description |
| --- | --- | --- |
| prefix | [String](../../../system/string/) | Το πρόθεμα του οποίου το URI του ονοματοχώρου θέλετε να βρείτε. |

### Τιμή Επιστροφής

Το URI του ονοματοχώρου του συγκεκριμένου προθέματος.

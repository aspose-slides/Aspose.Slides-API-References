## XmlValidatingReader::get_IsDefault() μέθοδος

Επιστρέφει μια τιμή που υποδεικνύει εάν ο τρέχων κόμβος είναι ένα γνώρισμα που δημιουργήθηκε από την προεπιλεγμένη τιμή που ορίζεται στον ορισμό τύπου εγγράφου (DTD) ή στο σχήμα.

```cpp
bool System::Xml::XmlValidatingReader::get_IsDefault() override
```

### Τιμή Επιστροφής

**true** εάν ο τρέχων κόμβος είναι ένα γνώρισμα του οποίου η τιμή δημιουργήθηκε από την προεπιλεγμένη τιμή που ορίζεται στο DTD ή στο σχήμα; **false** εάν η τιμή του γνωρίσματος ορίστηκε ρητά.

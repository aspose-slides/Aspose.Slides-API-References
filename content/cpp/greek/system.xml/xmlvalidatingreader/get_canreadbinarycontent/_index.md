bool System::Xml::XmlValidatingReader::get_CanReadBinaryContent() override

**true** εάν οι μέθοδοι ανάγνωσης δυαδικού περιεχομένου έχουν υλοποιηθεί· διαφορετικά **false**. Η κλάση [XmlValidatingReader](../) επιστρέφει **true**.

## XmlValidatingReader::LookupNamespace(const String\&) μέθοδος

Ανιχνεύει ένα πρόθεμα χώρου ονομάτων στην εμβέλεια του τρέχοντος στοιχείου.

```cpp
String System::Xml::XmlValidatingReader::LookupNamespace(const String &prefix) override
```


### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| prefix | const [String](../../../system/string/)\& | Το πρόθεμα του οποίου το Uniform Resource Identifier (URI) του χώρου ονομάτων θέλετε να επιλύσετε. Για να ταιριάξετε τον προεπιλεγμένο χώρο ονομάτων, περάστε μια κενή συμβολοσειρά. |

### Τιμή Επιστροφής

Το URI του χώρου ονομάτων στο οποίο αντιστοιχίζεται το πρόθεμα ή **nullptr** εάν δεν βρεθεί αντιστοιχούμενο πρόθεμα.

## XmlValidatingReader::ReadString() μέθοδος

Διαβάζει το περιεχόμενο ενός στοιχείου ή κόμβου κειμένου ως συμβολοσειρά.

```cpp
String System::Xml::XmlValidatingReader::ReadString() override
```

### Τιμή Επιστροφής

Το περιεχόμενο του στοιχείου ή του κόμβου κειμένου. Μπορεί να είναι μια κενή συμβολοσειρά εάν ο αναγνώστης είναι τοποθετημένος σε κάτι διαφορετικό από ένα στοιχείο ή κόμβο κειμένου, ή εάν δεν υπάρχει περισσότερο κειμενικό περιεχόμενο να επιστραφεί στο τρέχον πλαίσιο.

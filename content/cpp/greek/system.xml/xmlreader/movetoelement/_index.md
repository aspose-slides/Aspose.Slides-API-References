## XmlReader::MoveToElement() μέθοδος

Όταν παρακαμφθεί σε μια παράγωγη κλάση, μετακινείται στο στοιχείο που περιέχει τον τρέχοντα κόμβο ιδιότητας.

```cpp
virtual bool System::Xml::XmlReader::MoveToElement()=0
```

### Τιμή Επιστροφής

**true** εάν ο αναγνώστης βρίσκεται σε μια ιδιότητα (ο αναγνώστης μετακινείται στο στοιχείο που ανήκει στην ιδιότητα); **false** εάν ο αναγνώστης δεν βρίσκεται σε μια ιδιότητα (η θέση του αναγνώστη δεν αλλάζει).

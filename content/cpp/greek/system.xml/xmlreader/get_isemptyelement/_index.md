## XmlReader::get_IsEmptyElement() μέθοδος

When overridden in a derived class, gets a value indicating whether the current node is an empty element (for example, **<MyElement/>**).

```cpp
virtual bool System::Xml::XmlReader::get_IsEmptyElement()=0
```

### Τιμή Επιστροφής

**true** εάν ο τρέχων κόμβος είναι ένα στοιχείο ([XmlReader::get_NodeType](../get_nodetype/) ισούται με [XmlNodeType::Element](../../xmlnodetype/)) που τελειώνει με **/>**· αλλιώς, **false**.

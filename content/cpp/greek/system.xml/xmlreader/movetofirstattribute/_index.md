## XmlReader::MoveToFirstAttribute() μέθοδος

Όταν παρακάμπτεται σε μια κληρονομική κλάση, μεταβαίνει στο πρώτο χαρακτηριστικό.

```cpp
virtual bool System::Xml::XmlReader::MoveToFirstAttribute()=0
```

### Τιμή Επιστροφής

**true** εάν υπάρχει ένα χαρακτηριστικό (ο αναγνώστης μετακινείται στο πρώτο χαρακτηριστικό); διαφορετικά, **false** (η θέση του αναγνώστη δεν αλλάζει).

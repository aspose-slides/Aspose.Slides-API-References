## XmlReader::get_EOF() μέθοδος

Όταν αντικαθίσταται σε μια παράγωγη κλάση, επιστρέφει μια τιμή που υποδεικνύει εάν ο αναγνώστης βρίσκεται στο τέλος της ροής.

```cpp
virtual bool System::Xml::XmlReader::get_EOF()=0
```

### Τιμή Επιστροφής

**true** εάν ο αναγνώστης βρίσκεται στο τέλος της ροής; διαφορετικά, **false**.

## XmlReader::get_NamespaceURI() μέθοδος

Όταν παρακαμφθεί σε μια παράγωγη κλάση, επιστρέφει το namespace URI (όπως ορίζεται στην προδιαγραφή Namespace του W3C) του κόμβου στον οποίο βρίσκεται ο αναγνώστης.

```cpp
virtual String System::Xml::XmlReader::get_NamespaceURI()=0
```

### Τιμή Επιστροφής

Το namespace URI του τρέχοντος κόμβου· διαφορετικά μια κενή συμβολοσειρά.

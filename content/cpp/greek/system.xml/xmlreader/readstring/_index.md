## XmlReader::ReadString() μέθοδος

Όταν παρακαμφθεί σε κληρονομική κλάση, διαβάζει τα περιεχόμενα ενός στοιχείου ή ενός κόμβου κειμένου ως συμβολοσειρά. Ωστόσο, συνιστάται η χρήση της μεθόδου [XmlReader::ReadElementContentAsString](../readelementcontentasstring/) αντί αυτού, επειδή παρέχει έναν πιο απλό τρόπο χειρισμού αυτής της λειτουργίας.

```cpp
virtual String System::Xml::XmlReader::ReadString()
```

### Τιμή Επιστροφής

Τα περιεχόμενα του στοιχείου ή μια κενή συμβολοσειρά.

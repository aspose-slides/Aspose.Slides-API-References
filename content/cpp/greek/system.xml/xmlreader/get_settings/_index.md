## XmlReader::get_Settings() μέθοδος

Επιστρέφει το αντικείμενο [XmlReaderSettings](../../xmlreadersettings/) που χρησιμοποιείται για τη δημιουργία αυτού του αντικειμένου [XmlReader](../).

```cpp
virtual SharedPtr<XmlReaderSettings> System::Xml::XmlReader::get_Settings()
```

### Τιμή Επιστροφής

Το αντικείμενο [XmlReaderSettings](../../xmlreadersettings/) που χρησιμοποιείται για τη δημιουργία αυτού του αναγνώστη. Εάν αυτός ο αναγνώστης δεν δημιουργήθηκε χρησιμοποιώντας τη μέθοδο [XmlReader::Create](../create/), αυτή η μέθοδος επιστρέφει **nullptr**.

## XmlWriterSettings::get_CloseOutput() μέθοδος

Επιστρέφει μια τιμή που υποδεικνύει εάν το [XmlWriter](../../xmlwriter/) πρέπει επίσης να κλείσει τη βασική ροή ή το TextWriter όταν καλείται η [XmlWriter::Close](../../xmlwriter/close/) μέθοδος.

```cpp
bool System::Xml::XmlWriterSettings::get_CloseOutput()
```

### Τιμή Επιστροφής

**true** ώστε επίσης να κλείσει τη βασική ροή ή το TextWriter· διαφορετικά, **false**. Η προεπιλογή είναι **false**.

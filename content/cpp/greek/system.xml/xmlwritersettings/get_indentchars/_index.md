## XmlWriterSettings::get_IndentChars() μέθοδος

Επιστρέφει τη συμβολοσειρά χαρακτήρων που χρησιμοποιείται για την εσοχή. Αυτή η ρύθμιση χρησιμοποιείται όταν η τιμή [XmlWriterSettings::set_Indent](../set_indent/) ορίζεται σε **true**.

```cpp
String System::Xml::XmlWriterSettings::get_IndentChars()
```

### Τιμή Επιστροφής

Η συμβολοσειρά χαρακτήρων που χρησιμοποιείται για την εσοχή. Μπορεί να οριστεί σε οποιαδήποτε τιμή συμβολοσειράς. Ωστόσο, για να εξασφαλιστεί έγκυρο XML, θα πρέπει να καθορίζετε μόνο έγκυρους χαρακτήρες λευκού διαστήματος, όπως χαρακτήρες διαστήματος, στηλοθέτες, επιστροφές carriage ή νέες γραμμές. Η προεπιλογή είναι δύο κενά.

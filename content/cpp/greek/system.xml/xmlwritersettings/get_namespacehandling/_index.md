## XmlWriterSettings::get_NamespaceHandling() μέθοδος

Επιστρέφει μια τιμή που υποδεικνύει εάν το [XmlWriter](../../xmlwriter/) πρέπει να αφαιρέσει τις διπλότυπες δηλώσεις χώρου ονομάτων κατά τη σύνταξη περιεχομένου XML. Η προεπιλεγμένη συμπεριφορά είναι ο συγγραφέας να εξάγει όλες τις δηλώσεις χώρου ονομάτων που είναι παρούσες στον ανιχνευτή χώρου ονομάτων του συγγραφέα.

```cpp
System::Xml::NamespaceHandling System::Xml::XmlWriterSettings::get_NamespaceHandling()
```

### Τιμή Επιστροφής

Η απαρίθμηση NamespaceHandling που χρησιμοποιείται για να καθορίσει εάν θα αφαιρεθούν οι διπλότυπες δηλώσεις χώρου ονομάτων στο [XmlWriter](../../xmlwriter/).

## XmlWriterSettings::get_CheckCharacters() μέθοδος


Επιστρέφει μια τιμή που υποδεικνύει εάν ο δημιουργός XML πρέπει να ελέγχει για να διασφαλίσει ότι όλοι οι χαρακτήρες στο έγγραφο συμμορφώνονται με την ενότητα \"2.2 Characters\" του W3C [XML 1.0 Recommendation](https://www.w3.org/TR/REC-xml/#charsets).

```cpp
bool System::Xml::XmlWriterSettings::get_CheckCharacters()
```


### Τιμή Επιστροφής

**true** για έλεγχο χαρακτήρων· διαφορετικά, **false**. Η προεπιλογή είναι **true**.

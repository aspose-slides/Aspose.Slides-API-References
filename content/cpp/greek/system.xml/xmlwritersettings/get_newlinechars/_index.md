## XmlWriterSettings::get_NewLineChars() μέθοδος

Επιστρέφει τη συμβολοσειρά χαρακτήρων που χρησιμοποιείται για αλλαγές γραμμής.

```cpp
String System::Xml::XmlWriterSettings::get_NewLineChars()
```

### Τιμή Επιστροφής

Η συμβολοσειρά χαρακτήρων που χρησιμοποιείται για αλλαγές γραμμής. Μπορεί να οριστεί σε οποιαδήποτε τιμή συμβολοσειράς. Ωστόσο, για να εξασφαλιστεί έγκυρο XML, θα πρέπει να καθορίζετε μόνο έγκυρους χαρακτήρες λευκού διαστήματος, όπως χαρακτήρες κενών, καρτέλες, επιστροφές carriage ή νέες γραμμές. Η προεπιλογή είναι **\r\n** (επιστροφή carriage, νέα γραμμή).

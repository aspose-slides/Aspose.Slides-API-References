## XmlWriterSettings::get_WriteEndDocumentOnClose() μέθοδος


Επιστρέφει μια τιμή που υποδεικνύει εάν το [XmlWriter](../../xmlwriter/) θα προσθέσει ετικέτες κλεισίματος σε όλες τις ανοιχτές ετικέτες στοιχείων όταν η [XmlWriter::Close](../../xmlwriter/close/) μέθοδος κληθεί.

```cpp
bool System::Xml::XmlWriterSettings::get_WriteEndDocumentOnClose()
```


### Τιμή Επιστροφής

**true** εάν όλες οι ανοιχτές ετικέτες στοιχείων θα κλείσουν· διαφορετικά, **false**. Η προεπιλεγμένη τιμή είναι **true**.

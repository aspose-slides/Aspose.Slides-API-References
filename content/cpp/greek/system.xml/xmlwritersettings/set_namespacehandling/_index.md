## XmlWriterSettings::set_NamespaceHandling(System::Xml::NamespaceHandling) μέθοδος

Ορίζει μια τιμή που υποδεικνύει εάν το [XmlWriter](../../xmlwriter/) θα πρέπει να αφαιρέσει διπλότυπες δηλώσεις χώρου ονομάτων κατά τη γραφή περιεχομένου XML. Η προεπιλεγμένη συμπεριφορά είναι ο γραφέας να εξάγει όλες τις δηλώσεις χώρου ονομάτων που είναι παρούσες στον επιλυτή χώρου ονομάτων του γραφέα.

```cpp
void System::Xml::XmlWriterSettings::set_NamespaceHandling(System::Xml::NamespaceHandling value)
```

| --- | --- | --- |

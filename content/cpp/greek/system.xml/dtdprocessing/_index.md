## DtdProcessing enum


Καθορίζει τις επιλογές για την επεξεργασία DTD. Η απαρίθμηση DtdProcessing χρησιμοποιείται από την κλάση [XmlReaderSettings](../xmlreadersettings/).

```cpp
enum class DtdProcessing
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Prohibit | 0 | Καθορίζει ότι όταν εντοπίζεται ένα DTD, ρίχνεται μια XmlException με μήνυμα που δηλώνει ότι τα DTD απαγορεύονται. Αυτή είναι η προεπιλεγμένη συμπεριφορά. |
| Ignore | 1 | Κάνει το στοιχείο DOCTYPE να αγνοείται. Δεν γίνεται καμία επεξεργασία DTD και το DTD/DOCTYPE χάθηκε στην έξοδο. |
| Parse | 2 | Χρησιμοποιείται για την ανάλυση DTD. |

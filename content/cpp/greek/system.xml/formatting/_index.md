## Enum μορφοποίησης


Καθορίζει επιλογές μορφοποίησης για το [XmlTextWriter](../xmltextwriter/).

```cpp
enum class Formatting
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| None | 0 | Δεν εφαρμόζεται καμία ειδική μορφοποίηση. Αυτό είναι το προεπιλεγμένο. |
| Indented | 1 | Προκαλεί την εσοχή των στοιχείων-παιδιών σύμφωνα με τις ρυθμίσεις [XmlTextWriter::set_Indentation](../xmltextwriter/set_indentation/) και [XmlTextWriter::set_IndentChar](../xmltextwriter/set_indentchar/). |

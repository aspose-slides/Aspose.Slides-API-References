## WhitespaceHandling enum

Καθορίζει πώς διαχειρίζεται το κενό διάστημα.

```cpp
enum class WhitespaceHandling
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| All | 0 | Επιστρέφει κόμβους **Whitespace** και **SignificantWhitespace**. Αυτό είναι το προεπιλεγμένο. |
| Significant | 1 | Επιστρέφει μόνο κόμβους **SignificantWhitespace**. |
| None | 2 | Δεν επιστρέφει κόμβους **Whitespace** ούτε **SignificantWhitespace**. |

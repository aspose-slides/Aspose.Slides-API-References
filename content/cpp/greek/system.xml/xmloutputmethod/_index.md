## XmlOutputMethod enum

Καθορίζει τη μέθοδο που χρησιμοποιείται για τη σειριοποίηση της [XmlWriter](../xmlwriter/) εξόδου.

```cpp
enum class XmlOutputMethod
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Xml | 0 | Σειριοποιεί σύμφωνα με τους κανόνες XML 1.0. |
| Html | 1 | Σειριοποιεί σύμφωνα με τους κανόνες HTML που καθορίζονται από το XSLT. |
| Text | 2 | Σειριοποιεί μόνο μπλοκ κειμένου. |
| AutoDetect | 3 | Χρησιμοποιεί τους κανόνες XSLT για να επιλέξει μεταξύ των [XmlOutputMethod::Xml](./) και [XmlOutputMethod::Html](./) μεθόδων εξόδου κατά την εκτέλεση. |

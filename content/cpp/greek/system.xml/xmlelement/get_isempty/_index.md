## XmlElement::get_IsEmpty() μέθοδος


Επιστρέφει τη μορφή ετικέτας του στοιχείου.

```cpp
bool System::Xml::XmlElement::get_IsEmpty()
```


### Τιμή Επιστροφής

**true** αν το στοιχείο πρέπει να σειριοποιηθεί σε μορφή σύντομης ετικέτας \"<tt><item/></tt>\"· **false** για τη μακρά μορφή \"<tt><item></item></tt>\".

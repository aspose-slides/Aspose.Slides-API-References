## XmlElement::RemoveAttribute(String) method

Αφαιρεί ένα χαρακτηριστικό με όνομα.

```cpp
virtual void System::Xml::XmlElement::RemoveAttribute(String name)
```

### Ορίσματα

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| name | [String](../../../system/string/) | Το όνομα του χαρακτηριστικού που θα αφαιρεθεί. Αυτό είναι ένα πλήρες όνομα. Συμφωνεί με την τιμή **get_Name** του αντίστοιχου κόμβου. |

| namespaceURI | [String](../../../system/string/) | Το URI του χώρου ονομάτων του χαρακτηριστικού που θα αφαιρεθεί. |

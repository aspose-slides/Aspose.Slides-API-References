## XmlNodeReader::LookupNamespace(const String\&) μέθοδος

Επιλύει ένα πρόθεμα χώρων ονομάτων στο πεδίο ισχύος του τρέχοντος στοιχείου.

```cpp
String System::Xml::XmlNodeReader::LookupNamespace(const String &prefix) override
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| prefix | const [String](../../../system/string/)\& | Το πρόθεμα του οποίου το URI του χώρου ονομάτων θέλετε να επιλύσετε. Για να ταιριάζει με το προεπιλεγμένο namespace, περάστε ένα κενό string. Αυτό το string δεν χρειάζεται να ατομοποιηθεί. |

### Τιμή Επιστροφής

Το URI του χώρου ονομάτων στο οποίο αντιστοιχεί το πρόθεμα ή **nullptr** εάν δεν βρεθεί αντίστοιχο πρόθεμα.

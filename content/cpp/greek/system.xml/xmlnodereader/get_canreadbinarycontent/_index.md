## XmlNodeReader::get_CanReadBinaryContent() μέθοδος


Επιστρέφει μια τιμή που υποδεικνύει εάν το [XmlNodeReader](../) υλοποιεί τις μεθόδους ανάγνωσης δυαδικού περιεχομένου.

```cpp
bool System::Xml::XmlNodeReader::get_CanReadBinaryContent() override
```


### Τιμή Επιστροφής

**true** εάν οι μέθοδοι ανάγνωσης δυαδικού περιεχομένου υλοποιούνται· διαφορετικά **false**. Η κλάση [XmlNodeReader](../) πάντα επιστρέφει **true**.

## XmlNodeReader::ReadString() μέθοδος


Διαβάζει το περιεχόμενο ενός στοιχείου ή κόμβου κειμένου ως συμβολοσειρά.

```cpp
String System::Xml::XmlNodeReader::ReadString() override
```


### Return Value

Το περιεχόμενο του στοιχείου ή του κόμβου που μοιάζει με κείμενο (Αυτό μπορεί να περιλαμβάνει CDATA, [Text](../../../system.text/) κόμβους, κ.λπ.). Μπορεί να είναι κενή συμβολοσειρά εάν ο αναγνώστης βρίσκεται σε κάτι διαφορετικό από στοιχείο ή κόμβο κειμένου, ή εάν δεν υπάρχει άλλο κείμενο για επιστροφή στο τρέχον πλαίσιο. Σημείωση: Ο κόμβος κειμένου μπορεί να είναι είτε ένα στοιχείο είτε έναν κόμβο κειμένου ιδιότητας.

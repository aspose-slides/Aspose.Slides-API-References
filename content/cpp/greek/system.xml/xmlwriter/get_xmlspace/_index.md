## XmlWriter::get_XmlSpace() μέθοδος


Όταν αντικαθίσταται σε μια παράγωγη κλάση, επιστρέφει ένα XmlSpace που αντιπροσωπεύει το τρέχον **xml:space** πεδίο.

```cpp
virtual System::Xml::XmlSpace System::Xml::XmlWriter::get_XmlSpace()
```


### Τιμή Επιστροφής

Ένα XmlSpace που αντιπροσωπεύει το τρέχον **xml:space** πεδίο.



| Τιμή | Νόημα |
| --- | --- |
| `None`| Αυτή είναι η προεπιλογή εάν δεν υπάρχει πεδίο `xml:space`. |
| `Default`| Το τρέχον πεδίο είναι `xml:space="default"`. |
| `Preserve`| Το τρέχον πεδίο είναι `xml:space="preserve"`. |

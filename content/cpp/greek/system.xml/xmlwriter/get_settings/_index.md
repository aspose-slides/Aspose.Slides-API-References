## XmlWriter::get_Settings() μέθοδος


Επιστρέφει το [XmlWriterSettings](../../xmlwritersettings/) αντικείμενο που χρησιμοποιείται για τη δημιουργία αυτού του [XmlWriter](../) αντίτυπου.

```cpp
virtual SharedPtr<XmlWriterSettings> System::Xml::XmlWriter::get_Settings()
```


### Τιμή Επιστροφής

Το [XmlWriterSettings](../../xmlwritersettings/) αντικείμενο που χρησιμοποιείται για τη δημιουργία αυτού του writer αντίτυπου. Αν αυτός ο writer δεν δημιουργήθηκε χρησιμοποιώντας τη μέθοδο [XmlWriter::Create](../create/), αυτή η μέθοδος επιστρέφει **nullptr**.

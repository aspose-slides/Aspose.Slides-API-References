## XmlWriter::WriteSurrogateCharEntity(char16_t, char16_t) μέθοδος

Όταν παρακαμφθεί σε μια κλημένη κλάση, δημιουργεί και γράφει την οντότητα χαρακτήρα αντιπροσωπευτικού για το ζεύγος χαρακτήρων αντιπροσωπευτικού.

```cpp
virtual void System::Xml::XmlWriter::WriteSurrogateCharEntity(char16_t lowChar, char16_t highChar)=0
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| lowChar | char16_t | Το χαμηλό αντιπροσωπευτικό. Πρέπει να είναι μια τιμή μεταξύ 0xDC00 και 0xDFFF. |
| highChar | char16_t | Το υψηλό αντιπροσωπευτικό. Πρέπει να είναι μια τιμή μεταξύ 0xD800 και 0xDBFF. |

## XmlNodeOrder enum

Περιγράφει τη σειρά του εγγράφου ενός κόμβου σε σύγκριση με έναν δεύτερο κόμβο.

```cpp
enum class XmlNodeOrder
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Before | 0 | Ο τρέχων κόμβος αυτού του πλοηγού βρίσκεται πριν τον τρέχων κόμβο του παρεχόμενου πλοηγού. |
| After | 1 | Ο τρέχων κόμβος αυτού του πλοηγού βρίσκεται μετά τον τρέχων κόμβο του παρεχόμενου πλοηγού. |
| Same | 2 | Οι δύο πλοηγοί βρίσκονται στον ίδιο κόμβο. |
| Unknown | 3 | Οι θέσεις των κόμβων δεν μπορούν να προσδιοριστούν με βάση τη σειρά του εγγράφου, σε σχέση μεταξύ τους. Αυτό μπορεί να συμβεί αν οι δύο κόμβοι βρίσκονται σε διαφορετικά δέντρα. |

## XmlTextWriter::get_QuoteChar() μέθοδος

Επιστρέφει ποιος χαρακτήρας πρέπει να χρησιμοποιηθεί για την παράθεση τιμών χαρακτηριστικών.

```cpp
char16_t System::Xml::XmlTextWriter::get_QuoteChar()
```

### Τιμή Επιστροφής

Ο χαρακτήρας που θα χρησιμοποιηθεί για την παράθεση τιμών χαρακτηριστικών. Πρέπει να είναι μονό εισαγωγικό **'** ή διπλό εισαγωγικό **\"**. Η προεπιλογή είναι το διπλό εισαγωγικό.

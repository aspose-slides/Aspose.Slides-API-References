## XmlTextWriter::WriteProcessingInstruction(String, String) μέθοδος

Γράφει μια οδηγία επεξεργασίας με ένα κενό μεταξύ του ονόματος και του κειμένου ως εξής: **<?name text?>**.

```cpp
void System::Xml::XmlTextWriter::WriteProcessingInstruction(String name, String text) override
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| name | [String](../../../system/string/) | Το όνομα της οδηγίας επεξεργασίας. |
| text | [String](../../../system/string/) | [Text](../../../system.text/) για να συμπεριληφθεί στην οδηγία επεξεργασίας. |

## Παρατηρήσεις

Αυτή η μέθοδος χρησιμοποιείται για τη δημιουργία δήλωσης XML μετά την κλήση του [XmlTextWriter::WriteStartDocument](../writestartdocument/).

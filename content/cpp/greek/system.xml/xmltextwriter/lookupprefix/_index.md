## XmlTextWriter::LookupPrefix(String) μέθοδος


Επιστρέφει το πιο κοντινό πρόθεμα που ορίζεται στην τρέχουσα περιοχή ονοματοχώρου για το URI του ονοματοχώρου.

```cpp
String System::Xml::XmlTextWriter::LookupPrefix(String ns) override
```


### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| ns | [String](../../../system/string/) | URI ονοματοχώρου του οποίου το πρόθεμα θέλετε να βρείτε. |

### Τιμή Επιστροφής

Το αντίστοιχο πρόθεμα. Ή **nullptr** εάν δεν βρεθεί αντίστοιχο URI ονοματοχώρου στην τρέχουσα περιοχή.

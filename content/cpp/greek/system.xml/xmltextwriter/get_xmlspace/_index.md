## XmlTextWriter::get_XmlSpace() μέθοδος

Επιστρέφει ένα XmlSpace που αντιπροσωπεύει την τρέχουσα περιοχή **xml:space**.

```cpp
System::Xml::XmlSpace System::Xml::XmlTextWriter::get_XmlSpace() override
```

### Τιμή Επιστροφής

Ένα XmlSpace που αντιπροσωπεύει την τρέχουσα περιοχή **xml:space**.

| Value | Meaning |
| --- | --- |
| None | Αυτή είναι η προεπιλογή εάν δεν υπάρχει πεδίο `xml:space`. |
| Default | Το τρέχον πεδίο είναι `xml:space="default"`. |
| Preserve | Το τρέχον πεδίο είναι `xml:space="preserve"`. |

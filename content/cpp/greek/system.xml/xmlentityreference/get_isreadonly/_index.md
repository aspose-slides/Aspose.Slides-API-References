## XmlEntityReference::get_IsReadOnly() μέθοδος


Επιστρέφει μια τιμή που υποδεικνύει αν ο κόμβος είναι μόνο για ανάγνωση.

```cpp
bool System::Xml::XmlEntityReference::get_IsReadOnly() override
```


### Τιμή Επιστροφής

**true** αν ο κόμβος είναι μόνο για ανάγνωση; διαφορετικά **false**. Επειδή οι [XmlEntityReference](../) κόμβοι είναι μόνο για ανάγνωση, αυτή η μέθοδος πάντα επιστρέφει **true**.

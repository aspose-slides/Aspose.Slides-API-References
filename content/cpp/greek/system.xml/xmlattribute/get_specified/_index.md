## XmlAttribute::get_Specified() μέθοδος

Επιστρέφει μια τιμή που υποδεικνύει εάν η τιμή του χαρακτηριστικού ορίστηκε ρητά.

```cpp
virtual bool System::Xml::XmlAttribute::get_Specified()
```

### Τιμή Επιστροφής

**true** αν αυτό το χαρακτηριστικό δόθηκε ρητή τιμή στο αρχικό έγγραφο της παρουσίασης· διαφορετικά, **false**. Μια τιμή **false** υποδεικνύει ότι η τιμή του χαρακτηριστικού προέρχεται από το DTD.

## XmlDeclaration::get_Standalone() μέθοδος


Επιστρέφει την τιμή του χαρακτηριστικού standalone.

```cpp
String System::Xml::XmlDeclaration::get_Standalone()
```


### Τιμή Επιστροφής

Οι έγκυρες τιμές είναι **yes** εάν όλες οι δηλώσεις οντοτήτων που απαιτούνται από το έγγραφο XML περιλαμβάνονται στο έγγραφο ή **no** εάν απαιτείται εξωτερικός ορισμός τύπου εγγράφου (DTD). Εάν ένα χαρακτηριστικό standalone δεν υπάρχει στη δήλωση XML, αυτή η μέθοδος επιστρέφει [String::Empty](../../../system/string/empty/).

## XmlDocumentType::get_IsReadOnly() μέθοδος

Επιστρέφει μια τιμή που υποδεικνύει εάν ο κόμβος είναι μόνο για ανάγνωση.

```cpp
bool System::Xml::XmlDocumentType::get_IsReadOnly() override
```

### Τιμή επιστροφής

**true** εάν ο κόμβος είναι μόνο για ανάγνωση· διαφορετικά, **false**. Επειδή οι κόμβοι DocumentType είναι μόνο για ανάγνωση, αυτή η μέθοδος πάντα επιστρέφει **true**.

## XmlNotation::get_IsReadOnly() μέθοδος

Επιστρέφει μία τιμή που υποδηλώνει εάν ο κόμβος είναι μόνο για ανάγνωση.

```cpp
bool System::Xml::XmlNotation::get_IsReadOnly() override
```

### Τιμή επιστροφής

**true** εάν ο κόμβος είναι μόνο για ανάγνωση· διαφορετικά, **false**. Επειδή οι κόμβοι [XmlNotation](../) είναι μόνο για ανάγνωση, αυτή η μέθοδος πάντα επιστρέφει **true**.

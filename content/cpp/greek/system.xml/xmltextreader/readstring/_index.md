## XmlTextReader::ReadString() μέθοδος

Αναγνώζει τα περιεχόμενα ενός στοιχείου ή ενός κόμβου κειμένου ως συμβολοσειρά.

```cpp
String System::Xml::XmlTextReader::ReadString() override
```

### Τιμή επιστροφής

Τα περιεχόμενα του στοιχείου ή του κόμβου κειμένου. Μπορεί να είναι μια κενή συμβολοσειρά εάν ο αναγνώστης είναι τοποθετημένος σε κάτι διαφορετικό από στοιχείο ή κόμβο κειμένου, ή εάν δεν υπάρχει άλλο κειμενικό περιεχόμενο προς επιστροφή στο τρέχον πλαίσιο. **Σημείωση:** Ο κόμβος κειμένου μπορεί να είναι είτε στοιχείο είτε κόμβος κειμένου ιδιότητας.

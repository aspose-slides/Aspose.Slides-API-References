Αναλύει Base64 και επιστρέφει τα αποκωδικοποιημένα δυαδικά bytes.

```cpp
int32_t System::Xml::XmlTextReader::ReadBase64(const ArrayPtr<uint8_t> &array, int32_t offset, int32_t len)
```

### Ορίσματα

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| array | const [ArrayPtr](../../../system/arrayptr/)\<**uint8_t**\>\& | Ο πίνακας χαρακτήρων που χρησιμεύει ως buffer στον οποίο γράφονται τα περιεχόμενα κειμένου. |
| offset | **int32_t** | Ο δείκτης μηδενικής βάσης στον πίνακα που υποδεικνύει πού μπορεί η μέθοδος να αρχίσει να γράφει στο buffer. |
| len | **int32_t** | Ο αριθμός των bytes που θα γραφτούν στο buffer. |

### Τιμή Επιστροφής

Ο αριθμός των bytes που γράφτηκαν στο buffer.

## XmlTextReader::ReadChars(const ArrayPtr\<char16_t\>\&, int32_t, int32_t) μέθοδος

Διαβάζει το περιεχόμενο κειμένου ενός στοιχείου σε ένα buffer χαρακτήρων. Αυτή η μέθοδος έχει σχεδιαστεί για να διαβάζει μεγάλες ροές ενσωματωμένου κειμένου καλώντας την διαδοχικά.

```cpp
int32_t System::Xml::XmlTextReader::ReadChars(const ArrayPtr<char16_t> &buffer, int32_t index, int32_t count)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| buffer | const [ArrayPtr](../../../system/arrayptr/)\<char16_t\>\& | Ο πίνακας χαρακτήρων που χρησιμεύει ως buffer στον οποίο γράφεται το κείμενο. |
| index | **int32_t** | Η θέση μέσα στο **buffer** όπου η μέθοδος μπορεί να αρχίσει να γράφει το κείμενο. |
| count | **int32_t** | Ο αριθμός των χαρακτήρων που θα γραφούν στο **buffer**. |

### Τιμή Επιστροφής

Ο αριθμός των χαρακτήρων που διαβάστηκαν. Μπορεί να είναι 0 εάν ο αναγνώστης δεν είναι τοποθετημένος σε στοιχείο ή εάν δεν υπάρχει άλλο κείμενο προς επιστροφή στο τρέχον πλαίσιο.

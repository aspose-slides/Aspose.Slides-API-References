## XmlTextReader::get_CanReadBinaryContent() μέθοδος

Επιστρέφει μια τιμή που υποδεικνύει εάν το [XmlTextReader](../) υλοποιεί τις μεθόδους ανάγνωσης δυαδικού περιεχομένου.

```cpp
bool System::Xml::XmlTextReader::get_CanReadBinaryContent() override
```

### Τιμή Επιστροφής

**true** εάν οι μέθοδοι ανάγνωσης δυαδικού περιεχομένου υλοποιούνται· διαφορετικά **false**. Η κλάση [XmlTextReader](../) πάντα επιστρέφει **true**.

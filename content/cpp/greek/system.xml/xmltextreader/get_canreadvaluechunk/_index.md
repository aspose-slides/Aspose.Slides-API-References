## XmlTextReader::get_CanReadValueChunk() μέθοδος

```cpp
bool System::Xml::XmlTextReader::get_CanReadValueChunk() override
```

### Τιμή Επιστροφής

**true** εάν το [XmlTextReader](../) υλοποιεί τη μέθοδο [XmlReader::ReadValueChunk](../../xmlreader/readvaluechunk/)· διαφορετικά **false**. Η κλάση [XmlTextReader](../) πάντα επιστρέφει **false**.

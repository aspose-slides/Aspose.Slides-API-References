## IXmlLineInfo::HasLineInfo() μέθοδος


Επιστρέφει μια τιμή που υποδεικνύει εάν η κλάση μπορεί να επιστρέψει πληροφορίες γραμμής.

```cpp
virtual bool System::Xml::IXmlLineInfo::HasLineInfo()=0
```


### Τιμή Επιστροφής

**true** εάν μπορούν να δοθούν [IXmlLineInfo::get_LineNumber](../get_linenumber/) και [IXmlLineInfo::get_LinePosition](../get_lineposition/)· διαφορετικά, **false**.

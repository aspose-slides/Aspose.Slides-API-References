## XmlConvert::VerifyXmlChars(const String\&) μέθοδος

Επιστρέφει τη δοθείσα συμβολοσειρά εάν όλοι οι χαρακτήρες και οι χαρακτήρες ζεύγους αντιπροσωπευτικών στην παράμετρο συμβολοσειρά είναι έγκυροι χαρακτήρες XML, διαφορετικά ρίχνεται μια XmlException με πληροφορίες για τον πρώτο μη έγκυρο χαρακτήρα που εντοπίστηκε.

```cpp
static String System::Xml::XmlConvert::VerifyXmlChars(const String &content)
```

### Παράμετροι

| Parameter | Type | Description |
| --- | --- | --- |
| content | const [String](../../../system/string/)\& | [String](../../../system/string/) που περιέχει χαρακτήρες προς επαλήθευση. |

### Τιμή επιστροφής

Η δοθείσα συμβολοσειρά εάν όλοι οι χαρακτήρες και οι χαρακτήρες ζεύγους αντιπροσωπευτικών στην παράμετρο συμβολοσειρά είναι έγκυροι χαρακτήρες XML, διαφορετικά ρίχνεται μια XmlException με πληροφορίες για τον πρώτο μη έγκυρο χαρακτήρα που εντοπίστηκε.

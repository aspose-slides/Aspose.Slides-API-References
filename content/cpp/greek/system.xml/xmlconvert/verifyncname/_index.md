## XmlConvert::VerifyNCName(const String\&) μέθοδος


Επαληθεύει ότι το όνομα είναι ένα έγκυρο **NCName** σύμφωνα με την πρόταση W3C Extended Markup Language. Ένα **NCName** είναι ένα όνομα που δεν μπορεί να περιέχει άνω-κάθετο.

```cpp
static String System::Xml::XmlConvert::VerifyNCName(const String &name)
```


### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| name | const [String](../../../system/string/)\& | Το όνομα για επαλήθευση. |

### Τιμή Επιστροφής

Το όνομα, εφόσον είναι έγκυρο NCName.

## XmlConvert::IsNCNameChar(char16_t) μέθοδος

Ελέγχει εάν ο δοσμένος χαρακτήρας είναι έγκυρος τύπος χαρακτήρα χωρίς άνω-κάθετο.

```cpp
static bool System::Xml::XmlConvert::IsNCNameChar(char16_t ch)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| ch | char16_t | Ο χαρακτήρας που θα ελεγθεί ως χαρακτήρας χωρίς άνω-κάθετο. |

### Τιμή επιστροφής

**true** if the character is a valid non-colon character type; otherwise, **false**.

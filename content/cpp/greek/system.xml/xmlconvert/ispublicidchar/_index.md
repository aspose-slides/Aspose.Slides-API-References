## XmlConvert::IsPublicIdChar(char16_t) μέθοδος

Επιστρέφει το αντικείμενο χαρακτήρα που δόθηκε εάν ο χαρακτήρας στο όρισμα είναι έγκυρος δημόσιος ταυτοποιητικός χαρακτήρας, διαφορετικά **nullptr**.

```cpp
static bool System::Xml::XmlConvert::IsPublicIdChar(char16_t ch)
```

### Παράμετροι

| Parameter | Type | Description |
| --- | --- | --- |
| ch | char16_t | [Char](../../../system/char/) αντικείμενο για επαλήθευση. |

### Τιμή Επιστροφής

Ο δοσμένος χαρακτήρας εάν ο χαρακτήρας είναι έγκυρος δημόσιος ταυτοποιητικός χαρακτήρας, διαφορετικά **nullptr**.

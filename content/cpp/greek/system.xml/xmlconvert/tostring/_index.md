## XmlConvert::ToString(DateTimeOffset, const String\&) μέθοδος

Μετατρέπει το παρεχόμενο [DateTimeOffset](../../../system/datetimeoffset/) σε ένα [String](../../../system/string/) στην καθορισμένη μορφή.

```cpp
static String System::Xml::XmlConvert::ToString(DateTimeOffset value, const String &format)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| value | [DateTimeOffset](../../../system/datetimeoffset/) | Το [DateTimeOffset](../../../system/datetimeoffset/) που πρέπει να μετατραπεί. |
| format | const [String](../../../system/string/)\& | Η μορφή στην οποία **s** μετατρέπεται. Η παράμετρος μορφής μπορεί να είναι οποιοδήποτε υποσύνολο της σύστασης W3C για τον τύπο XML dateTime. Για περισσότερες πληροφορίες, δείτε την ενότητα [dateTime](https://www.w3.org/TR/xmlschema-2/#dateTime) της προδιαγραφής XML [Schema](../../../system.xml.schema/). |

### Τιμή Επιστροφής

Μια [String](../../../system/string/) αναπαράσταση στην καθορισμένη μορφή του παρεχόμενου [DateTimeOffset](../../../system/datetimeoffset/).

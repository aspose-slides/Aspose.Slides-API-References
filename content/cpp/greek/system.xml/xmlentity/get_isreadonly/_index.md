## XmlEntity::get_IsReadOnly() μέθοδος


Επιστρέφει μια τιμή που υποδεικνύει εάν ο κόμβος είναι μόνο για ανάγνωση.

```cpp
bool System::Xml::XmlEntity::get_IsReadOnly() override
```


### Τιμή Επιστροφής

**true** αν ο κόμβος είναι μόνο για ανάγνωση· διαφορετικά, **false**. Επειδή οι κόμβοι [XmlEntity](../) είναι μόνο για ανάγνωση, αυτή η μέθοδος επιστρέφει πάντα **true**.

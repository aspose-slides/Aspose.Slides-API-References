## XmlReaderSettings::get_ProhibitDtd() μέθοδος


Επιστρέφει μια τιμή που υποδεικνύει εάν θα απαγορευτεί η επεξεργασία του ορισμού τύπου εγγράφου (DTD).

```cpp
bool System::Xml::XmlReaderSettings::get_ProhibitDtd()
```


### Τιμή επιστροφής

**true** για να απαγορεύσετε την επεξεργασία DTD· διαφορετικά **false**. Η προεπιλογή είναι **true**.

## XmlReaderSettings::get_ValidationType() μέθοδος

Επιστρέφει μια τιμή που υποδεικνύει εάν το [XmlReader](../../xmlreader/) θα εκτελέσει επικύρωση ή εκχώρηση τύπου κατά την ανάγνωση.

```cpp
System::Xml::ValidationType System::Xml::XmlReaderSettings::get_ValidationType()
```

### Τιμή επιστροφής

Μία από τις τιμές ValidationType που υποδεικνύει εάν το [XmlReader](../../xmlreader/) θα εκτελέσει επικύρωση ή εκχώρηση τύπου κατά την ανάγνωση. Η προεπιλογή είναι [ValidationType::None](../../validationtype/).

## XmlNamespaceManager::GetNamespacesInScope(XmlNamespaceScope) μέθοδος

Επιστρέφει μια συλλογή από ονόματα χώρων ονομάτων που είναι κλειδωμένα με πρόθεμα και μπορούν να χρησιμοποιηθούν για την απαρίθμηση των χώρων ονομάτων που βρίσκονται αυτή τη στιγμή σε εμβέλεια.

```cpp
SharedPtr<Collections::Generic::IDictionary<String, String>> System::Xml::XmlNamespaceManager::GetNamespacesInScope(XmlNamespaceScope scope) override
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| scope | [XmlNamespaceScope](../../xmlnamespacescope/) | Μια τιμή απαρίθμησης που καθορίζει τον τύπο των κόμβων χώρων ονομάτων που θα επιστραφούν. |

### Τιμή Επιστροφής

Μια συλλογή από ζεύγη χώρων ονομάτων και προθέματος που βρίσκονται αυτή τη στιγμή σε εμβέλεια.

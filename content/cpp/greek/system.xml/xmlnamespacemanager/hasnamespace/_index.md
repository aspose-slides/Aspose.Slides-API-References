## XmlNamespaceManager::HasNamespace(String) μέθοδος

Επιστρέφει μια τιμή που υποδεικνύει εάν το δοσμένο πρόθεμα έχει ορισμένο namespace για το τρέχον pushed scope.

```cpp
virtual bool System::Xml::XmlNamespaceManager::HasNamespace(String prefix)
```

### Παράμετροι

| Parameter | Type | Description |
| --- | --- | --- |
| prefix | [String](../../../system/string/) | Το πρόθεμα του namespace που θέλετε να βρείτε. |

### Τιμή Επιστροφής

**true** εάν υπάρχει ορισμένο namespace· διαφορετικά, **false**.

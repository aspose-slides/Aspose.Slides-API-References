## Σχόλια

Τα αντικείμενα αυτής της κλάσης πρέπει να καταλαμβάνονται μόνο με τη λειτουργία [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε παραδείγματα αυτού του τύπου στη στοίβα ή με τον τελεστή new, καθώς αυτό θα οδηγήσει σε σφάλματα εκτέλεσης και/ή αστοχίες ελέγχου. Πάντα να τυλίγετε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και να χρησιμοποιείτε αυτόν τον δείκτη για να τον περάσετε σε συναρτήσεις ως όρισμα.

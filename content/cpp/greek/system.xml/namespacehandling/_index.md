## NamespaceHandling enum

Καθορίζει εάν θα αφαιρεθούν οι διπλές δηλώσεις ονομάτων χώρου στο [XmlWriter](../xmlwriter/).

```cpp
enum class NamespaceHandling
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Default | 0 | Καθορίζει ότι οι διπλές δηλώσεις ονομάτων χώρου δεν θα αφαιρεθούν. |
| OmitDuplicates | 1 | Καθορίζει ότι οι διπλές δηλώσεις ονομάτων χώρου θα αφαιρεθούν. Για να αφαιρεθεί η διπλή δήλωση ονόματος χώρου, το πρόθεμα και ο χώρος ονομάτων πρέπει να ταιριάζουν. |

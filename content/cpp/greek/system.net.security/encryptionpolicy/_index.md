## EncryptionPolicy enum

Καταγράφει τις πολιτικές κρυπτογράφησης.

```cpp
enum class EncryptionPolicy
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| RequireEncryption | 0 | Απαιτεί κρυπτογράφηση και δεν επιτρέπει ποτέ ένα 'Null' cipher. |
| AllowNoEncryption | 1 | Προτιμά τη χρήση πλήρους κρυπτογράφησης, αλλά μπορεί να χρησιμοποιηθεί ένα 'Null' cipher εάν ο διακομιστής συμφωνήσει. |
| NoEncryption | 2 | Επιτρέπει την απουσία κρυπτογράφησης και ζητά να χρησιμοποιηθεί ένα 'Null' cipher εάν το άλλο άκρο μπορεί να διαχειριστεί ένα 'Null' cipher. |

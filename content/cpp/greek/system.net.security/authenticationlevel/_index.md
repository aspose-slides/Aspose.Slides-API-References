## AuthenticationLevel enum

Σημαίες ελέγχου ταυτότητας ειδικές για WebRequest.

```cpp
enum class AuthenticationLevel
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| None | 0 | Δεν απαιτείται έλεγχος ταυτότητας για τον πελάτη και τον διακομιστή. |
| MutualAuthRequested | 1 | Η αίτηση δεν αποτυγχάνει αν ο διακομιστής δεν είναι αυθεντικοποιημένος. |
| MutualAuthRequired | 2 | Η τρέχουσα εφαρμογή θα λάβει 'IOException' όταν ο διακομιστής δεν είναι αυθεντικοποιημένος. |

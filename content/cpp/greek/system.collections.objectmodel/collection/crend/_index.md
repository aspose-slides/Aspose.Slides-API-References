## Collection::crend() const μέθοδος

Λαμβάνει έναν αντίστροφο επαναλήπτη για ένα μη υπάρχον στοιχείο που έχει τροποποιητή const πριν από την αρχή της συλλογής.

```cpp
const_reverse_iterator System::Collections::ObjectModel::Collection<T>::crend() const noexcept
```

### Return Value

Ένας αντίστροφος επαναλήπτης που δείχνει το θεωρητικό const-ποιοτικό στοιχείο που προηγείται του πρώτου στοιχείου στη λίστα (τελευταίο στην αντίστροφη σειρά).

## XmlSchemaChoice κλάση

Αντιπροσωπεύει το στοιχείο **choice** (σύνθεση) από το XML [Schema](../) όπως ορίζεται από το Παγκόσμιο [Web](../../system.web/) Consortium (W3C). Το **choice** επιτρέπει μόνο ένα από τα παιδιά του να εμφανίζεται σε μια παρουσία.

```cpp
class XmlSchemaChoice : public System::Xml::Schema::XmlSchemaGroupBase
```

## Παρατηρήσεις

Τα αντικείμενα αυτής της κλάσης πρέπει να καταλαμβάνονται μόνο με τη λειτουργία [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε εμφανίσεις αυτού του τύπου στην στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς αυτό θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης και/ή προβλήματα assert. Πάντα τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να τον περάσετε σε συναρτήσεις ως όρισμα.

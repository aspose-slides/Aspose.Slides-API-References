## XmlSchemaMinLengthFacet κλάση

Αντιπροσωπεύει το στοιχείο **minLength** από XML [Schema](../) όπως καθορίζεται από το Παγκόσμιο [Web](../../system.web/) Consortium (W3C). Αυτή η κλάση μπορεί να χρησιμοποιηθεί για να ορίσει περιορισμό στο ελάχιστο μήκος της τιμής δεδομένων ενός στοιχείου **simpleType**. Το μήκος πρέπει να είναι μεγαλύτερο από την τιμή του στοιχείου **minLength**.

```cpp
class XmlSchemaMinLengthFacet : public System::Xml::Schema::XmlSchemaNumericFacet
```

## Παρατηρήσεις

Τα αντικείμενα αυτής της κλάσης θα πρέπει να δημιουργούνται μόνο χρησιμοποιώντας τη συνάρτηση [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε παραδείγματα αυτού του τύπου στην στοίβα ή με τον τελεστή new, καθώς θα προκαλέσει σφάλματα εκτέλεσης και/ή σφάλματα ελέγχου. Πάντα να τυλίγετε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και να χρησιμοποιείτε αυτόν τον δείκτη για να τον περάσετε σε συναρτήσεις ως όρισμα.

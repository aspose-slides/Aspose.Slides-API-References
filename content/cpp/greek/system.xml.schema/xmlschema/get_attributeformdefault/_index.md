## XmlSchema::get_AttributeFormDefault() μέθοδος


Επιστρέφει τη μορφή για τα χαρακτηριστικά που δηλώνονται στον προορισμένο χώρο ονομάτων του σχήματος.

```cpp
XmlSchemaForm System::Xml::Schema::XmlSchema::get_AttributeFormDefault()
```


### Τιμή Επιστροφής

Η τιμή XmlSchemaForm που υποδεικνύει εάν τα χαρακτηριστικά από τον προορισμένο χώρο ονομάτων απαιτείται να είναι προσδιορισμένα με το πρόθεμα του χώρου ονομάτων. Η προεπιλογή είναι [XmlSchemaForm::None](../../xmlschemaform/).

## XmlSchema::get_ElementFormDefault() μέθοδος


Επιστρέφει τη μορφή για στοιχεία που δηλώνονται στον χώρο ονομάτων-στόχο του σχήματος.

```cpp
XmlSchemaForm System::Xml::Schema::XmlSchema::get_ElementFormDefault()
```


### Τιμή επιστροφής

Η τιμή XmlSchemaForm που υποδεικνύει εάν τα στοιχεία από το χώρο ονομάτων-στόχο πρέπει να είναι πιστοποιημένα με το πρόθεμα του χώρου ονομάτων. Η προεπιλογή είναι [XmlSchemaForm::None](../../xmlschemaform/).

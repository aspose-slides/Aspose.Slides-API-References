## XmlSchemaCompilationSettings::get_EnableUpaCheck() method


Επιστρέφει μια τιμή που υποδεικνύει εάν το [XmlSchemaSet](../../xmlschemaset/) πρέπει να ελέγχει για παραβιάσεις Unique Particle Attribution (UPA).

```cpp
bool System::Xml::Schema::XmlSchemaCompilationSettings::get_EnableUpaCheck()
```


### Τιμή Επιστροφής

**true** εάν το [XmlSchemaSet](../../xmlschemaset/) πρέπει να ελέγχει για παραβιάσεις Unique Particle Attribution (UPA); διαφορετικά, **false**. Η προεπιλογή είναι **true**.

## XmlSchemaType::get_IsMixed() μέθοδος

Επιστρέφει μια τιμή που υποδεικνύει εάν αυτός ο τύπος έχει ένα μεικτό μοντέλο περιεχομένου. Αυτή η κλήση είναι έγκυρη μόνο σε σύνθετο τύπο.

```cpp
virtual bool System::Xml::Schema::XmlSchemaType::get_IsMixed()
```

### Τιμή Επιστροφής

**true** αν ο τύπος έχει ένα μεικτό μοντέλο περιεχομένου· διαφορετικά, **false**. Η προεπιλογή είναι **false**.

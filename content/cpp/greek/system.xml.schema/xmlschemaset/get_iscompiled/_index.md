## XmlSchemaSet::get_IsCompiled() μέθοδος

Επιστρέφει μια τιμή που υποδεικνύει εάν τα σχήματα XML [Schema](../../) γλώσσας ορισμού (XSD) στο [XmlSchemaSet](../) έχουν μεταγλωττιστεί.

```cpp
bool System::Xml::Schema::XmlSchemaSet::get_IsCompiled()
```

### Τιμή Επιστροφής

**true** εάν τα σχήματα στο [XmlSchemaSet](../) έχουν μεταγλωττιστεί από την τελευταία φορά που ένα σχήμα προστέθηκε ή αφαιρέθηκε από το [XmlSchemaSet](../)· διαφορετικά, **false**.

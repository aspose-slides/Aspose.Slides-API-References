## XmlSchemaMinExclusiveFacet κλάση


Αναπαριστά το στοιχείο **minExclusive** από το XML [Schema](../) όπως καθορίζεται από το Παγκόσμιο [Web](../../system.web/) Consortium (W3C). Αυτή η κλάση μπορεί να χρησιμοποιηθεί για να καθορίσει περιορισμό στην ελάχιστη τιμή ενός στοιχείου **simpleType**. Η τιμή του στοιχείου πρέπει να είναι μεγαλύτερη από την τιμή του στοιχείου **minExclusive**.

```cpp
class XmlSchemaMinExclusiveFacet : public System::Xml::Schema::XmlSchemaFacet
```

## Παρατηρήσεις



Τα αντικείμενα αυτής της κλάσης πρέπει να δεσμεύονται μόνο χρησιμοποιώντας τη λειτουργία [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε παρουσίες αυτού του τύπου στο stack ή χρησιμοποιώντας τον τελεστή new, καθώς θα προκαλέσει σφάλματα χρόνου εκτέλεσης και/ή σφάλματα ελέγχου. Πάντα να περιβάλλετε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και να χρησιμοποιείτε αυτόν τον δείκτη για να τον περάσετε σε συναρτήσεις ως όρισμα.

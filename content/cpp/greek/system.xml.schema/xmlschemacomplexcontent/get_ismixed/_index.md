## XmlSchemaComplexContent::get_IsMixed() μέθοδος

Επιστρέφει πληροφορίες που καθορίζουν αν ο τύπος έχει μικτό μοντέλο περιεχομένου.

```cpp
bool System::Xml::Schema::XmlSchemaComplexContent::get_IsMixed()
```

### Τιμή Επιστροφής

Αν αυτή η τιμή είναι **true**, τα δεδομένα χαρακτήρων επιτρέπεται να εμφανίζονται μεταξύ των θυγατρικών στοιχείων του σύνθετου τύπου (μικτό μοντέλο περιεχομένου). Η προεπιλογή είναι **false**. Προαιρετικό.

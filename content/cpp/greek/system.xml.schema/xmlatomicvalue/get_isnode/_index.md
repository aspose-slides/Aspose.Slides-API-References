## XmlAtomicValue::get_IsNode() μέθοδος

Επιστρέφει μια τιμή που υποδεικνύει εάν το επικυρωμένο στοιχείο XML ή το γνώρισμα είναι ένας [XPath](../../../system.xml.xpath/) κόμβος ή μια ατομική τιμή.

```cpp
bool System::Xml::Schema::XmlAtomicValue::get_IsNode() override
```

### Τιμή Επιστροφής

**true** εάν το επικυρωμένο στοιχείο XML ή το γνώρισμα είναι ένας [XPath](../../../system.xml.xpath/) κόμβος; **false** εάν το επικυρωμένο στοιχείο XML ή το γνώρισμα είναι μια ατομική τιμή.

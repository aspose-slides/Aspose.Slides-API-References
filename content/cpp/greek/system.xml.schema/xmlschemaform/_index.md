## XmlSchemaForm απαρίθμηση

Δηλώνει εάν τα χαρακτηριστικά ή τα στοιχεία πρέπει να είναι προσδιορισμένα με πρόθεμα χώρου ονομάτων.

```cpp
enum class XmlSchemaForm
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| None | 0 | Η μορφή στοιχείου και χαρακτηριστικού δεν είναι καθορισμένη στο σχήμα. |
| Qualified | 1 | Τα στοιχεία και τα χαρακτηριστικά πρέπει να είναι προσδιορισμένα με πρόθεμα χώρου ονομάτων. |
| Unqualified | 2 | Τα στοιχεία και τα χαρακτηριστικά δεν απαιτείται να είναι προσδιορισμένα με πρόθεμα χώρου ονομάτων. |

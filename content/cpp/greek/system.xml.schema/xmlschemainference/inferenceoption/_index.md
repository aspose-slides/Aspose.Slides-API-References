## InferenceOption enum

Επηρεάζει την εμφάνιση και τις πληροφορίες τύπου που προέρχονται από την κλάση [XmlSchemaInference](../) για στοιχεία και χαρακτηριστικά σε ένα έγγραφο XML.

```cpp
enum class InferenceOption
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Restricted | 0 | Δείχνει ότι θα πρέπει να προέλθει μια πιο περιοριστική δήλωση σχήματος για ένα συγκεκριμένο στοιχείο ή χαρακτηριστικό. |
| Relaxed | 1 | Δείχνει ότι θα πρέπει να προέλθει μια λιγότερο περιοριστική δήλωση σχήματος για ένα συγκεκριμένο στοιχείο ή χαρακτηριστικό. |

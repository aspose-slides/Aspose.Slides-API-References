## XmlSchemaAny::get_ProcessContents() μέθοδος

Επιστρέφει πληροφορίες σχετικά με το πώς μια εφαρμογή ή επεξεργαστής XML πρέπει να χειρίζεται την επαλήθευση των εγγράφων XML για τα στοιχεία που ορίζονται από το στοιχείο **any**.

```cpp
XmlSchemaContentProcessing System::Xml::Schema::XmlSchemaAny::get_ProcessContents()
```

### Τιμή Επιστροφής

Μία από τις τιμές του XmlSchemaContentProcessing. Εάν δεν έχει καθοριστεί το χαρακτηριστικό **processContents**, η προεπιλογή είναι **Strict**.


**true** εάν αυτός ο επικυρωμένος κόμβος XML ορίστηκε ως αποτέλεσμα μιας προεπιλογής που εφαρμόστηκε κατά τη διάρκεια της επικύρωσης του σχήματος· διαφορετικά, **false**.

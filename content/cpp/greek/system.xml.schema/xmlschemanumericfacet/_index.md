## XmlSchemaNumericFacet κλάση

A vase class for defining **numeric** facets. This class is the base class for numeric facet classes such as [XmlSchemaMinLengthFacet](../xmlschemaminlengthfacet/).

## Παρατηρήσεις

Τα αντικείμενα αυτής της κλάσης πρέπει να εκχωρούνται μόνο μέσω της συνάρτησης [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε παρουσίες αυτού του τύπου στη στοίβα ή χρησιμοποιώντας το τελεστή new, καθώς αυτό οδηγεί σε σφάλματα χρόνου εκτέλεσης και/ή σφάλματα δήλωσης. Πάντα να τυλίγετε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και να χρησιμοποιείτε αυτόν τον δείκτη για να την περάσετε ως όρισμα σε συναρτήσεις.

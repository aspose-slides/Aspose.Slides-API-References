## XmlSchemaValidator::ValidateWhitespace(const String\&) μέθοδος

Επικυρώνει εάν το κενό διάστημα στη **συμβολοσειρά** που καθορίζεται επιτρέπεται στο τρέχον πλαίσιο στοιχείου και καταχωρεί το κενό διάστημα για επικύρωση εάν το τρέχον στοιχείο έχει απλό περιεχόμενο.

```cpp
void System::Xml::Schema::XmlSchemaValidator::ValidateWhitespace(const String &elementValue)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| elementValue | const [String](../../../system/string/)\& | Ένα κενό διάστημα **συμβολοσειρά** για επικύρωση στο τρέχον πλαίσιο στοιχείου. |

```cpp
void System::Xml::Schema::XmlSchemaValidator::ValidateWhitespace(XmlValueGetter elementValue)
```

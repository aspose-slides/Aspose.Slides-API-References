## XmlSchemaComplexType::get_IsMixed() μέθοδος

Returns information that determines if the complex type has a mixed content model (markup within the content).

```cpp
bool System::Xml::Schema::XmlSchemaComplexType::get_IsMixed() override
```

### Τιμή επιστροφής

**true**, if character data can appear between child elements of this complex type; otherwise, **false**. The default is **false**. Προαιρετικό.

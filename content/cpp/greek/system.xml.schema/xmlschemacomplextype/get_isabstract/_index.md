## XmlSchemaComplexType::get_IsAbstract() μέθοδος


Επιστρέφει τις πληροφορίες που καθορίζουν αν το στοιχείο **complexType** μπορεί να χρησιμοποιηθεί στο έγγραφο στιγμιότυπου.

```cpp
bool System::Xml::Schema::XmlSchemaComplexType::get_IsAbstract()
```


### Τιμή Επιστροφής

Αν **true**, ένα στοιχείο δεν μπορεί να χρησιμοποιήσει αυτό το στοιχείο **complexType** απευθείας και πρέπει να χρησιμοποιήσει έναν σύνθετο τύπο που προέρχεται από αυτό το στοιχείο **complexType**. Η προεπιλογή είναι **false**. Προαιρετικό.

## XmlSchemaDerivationMethod enum

Παρέχει διαφορετικές μεθόδους για την αποτροπή της παράγωγης.

```cpp
enum class XmlSchemaDerivationMethod
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Empty | 0 | Παρακάμπτει την προεπιλεγμένη μέθοδο παράγωγης για να επιτρέψει οποιαδήποτε παράγωγή. |
| Substitution | 1 | Αναφέρεται σε παραγώγους μέσω **Substitution**. |
| Extension | 2 | Αναφέρεται σε παραγώγους μέσω **Extension**. |
| Restriction | 4 | Αναφέρεται σε παραγώγους μέσω **Restriction**. |
| List | 8 | Αναφέρεται σε παραγώγους μέσω **List**. |
| Union | 16 | Αναφέρεται σε παραγώγους μέσω **Union**. |
| All | 255 | **#all**. Αναφέρεται σε όλες τις μεθόδους παράγωγης. |
| None | 256 | Δέχεται την προεπιλεγμένη μέθοδο παράγωγης. |

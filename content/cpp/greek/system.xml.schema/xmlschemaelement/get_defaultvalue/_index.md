## XmlSchemaElement::get_DefaultValue() μέθοδος


Επιστρέφει την προεπιλεγμένη τιμή του στοιχείου εάν το περιεχόμενό του είναι απλός τύπος ή το περιεχόμενο του στοιχείου είναι **textOnly**.

```cpp
String System::Xml::Schema::XmlSchemaElement::get_DefaultValue()
```


### Τιμή Επιστροφής

Η προεπιλεγμένη τιμή για το στοιχείο. Η προεπιλογή είναι **nullptr**. Προαιρετικό.

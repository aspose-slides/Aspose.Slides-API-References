## XmlSchemaElement::get_IsNillable() method

Επιστρέφει πληροφορίες που υποδεικνύουν αν το **xsi:nil** μπορεί να εμφανιστεί στα δεδομένα της παρουσίασης. Υποδεικνύει αν μια ρητή τιμή nil μπορεί να αντιστοιχιστεί στο στοιχείο.

```cpp
bool System::Xml::Schema::XmlSchemaElement::get_IsNillable()
```

### Τιμή Επιστροφής

Αν το nillable είναι **true**, αυτό επιτρέπει σε μια παρουσία του στοιχείου να έχει το χαρακτηριστικό **nil** ορισμένο σε **true**. Το χαρακτηριστικό **nil** ορίζεται ως μέρος του XML [Schema](../../) χώρου ονομάτων για παρουσίες. Η προεπιλογή είναι **false**. Προαιρετικό.

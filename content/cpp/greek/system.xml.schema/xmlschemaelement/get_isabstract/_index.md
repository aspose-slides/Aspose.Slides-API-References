## XmlSchemaElement::get_IsAbstract() μέθοδος


Επιστρέφει πληροφορίες που δείχνουν αν το στοιχείο μπορεί να χρησιμοποιηθεί σε ένα έγγραφο στιγμιοτύπου.

```cpp
bool System::Xml::Schema::XmlSchemaElement::get_IsAbstract()
```


### Τιμή Επιστροφής

Αν **true**, το στοιχείο δεν μπορεί να εμφανιστεί στο έγγραφο στιγμιοτύπου. Η προεπιλογή είναι **false**. Προαιρετικό.

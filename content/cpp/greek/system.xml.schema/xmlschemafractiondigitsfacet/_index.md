## XmlSchemaFractionDigitsFacet κλάση


Καθορίζει περιορισμό στον αριθμό των ψηφίων που μπορούν να εισαχθούν για την κλασματική τιμή ενός στοιχείου simpleType. Η τιμή του fractionDigits πρέπει να είναι θετικός ακέραιος. Αντιπροσωπεύει το World Wide [Web](../../system.web/) Consortium (W3C) **fractionDigits** facet.

```cpp
class XmlSchemaFractionDigitsFacet : public System::Xml::Schema::XmlSchemaNumericFacet
```

## Σχόλια



Τα αντικείμενα αυτής της κλάσης πρέπει να δημιουργούνται μόνο μέσω της συνάρτησης [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε εμφανίσεις αυτού του τύπου στη στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης και/ή σφάλματα assert. Πάντα να περιβάλλετε αυτήν την κλάση με δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιείτε αυτόν τον δείκτη για μεταβίβαση σε συναρτήσεις ως όρισμα.

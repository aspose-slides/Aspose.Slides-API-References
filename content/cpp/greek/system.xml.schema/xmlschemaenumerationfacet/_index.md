## XmlSchemaEnumerationFacet κλάση

Αναπαριστά το χαρακτηριστικό **enumeration** από το XML [Schema](../) όπως καθορίζεται από το Παγκόσμιο [Web](../../system.web/) Consortium (W3C). Αυτή η κλάση καθορίζει μια λίστα έγκυρων τιμών για ένα στοιχείο simpleType. Η δήλωση περιέχεται εντός μιας δήλωσης **restriction**.

```cpp
class XmlSchemaEnumerationFacet : public System::Xml::Schema::XmlSchemaFacet
```

## Παρατηρήσεις

Τα αντικείμενα αυτής της κλάσης πρέπει να εκχωρούνται μόνο χρησιμοποιώντας τη συνάρτηση [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε παρουσίες αυτού του τύπου στη στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης και/ή σφάλματα assertion. Πάντα τυλίξτε αυτή την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτό το δείκτη για να το περάσετε σε συναρτήσεις ως όρισμα.

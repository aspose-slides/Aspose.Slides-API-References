## XmlSchemaAnyAttribute::get_ProcessContents() μέθοδος

Επιστρέφει πληροφορίες σχετικά με το πώς μια εφαρμογή ή επεξεργαστής XML πρέπει να διαχειρίζεται την επικύρωση εγγράφων XML για τα χαρακτηριστικά που καθορίζονται από το στοιχείο **anyAttribute**.

```cpp
XmlSchemaContentProcessing System::Xml::Schema::XmlSchemaAnyAttribute::get_ProcessContents()
```

### Τιμή Επιστροφής

Μία από τις τιμές του XmlSchemaContentProcessing. Εάν δεν έχει καθοριστεί το χαρακτηριστικό **processContents**, η προεπιλογή είναι **Strict**.

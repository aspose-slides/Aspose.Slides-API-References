## XmlSeverityType enum

Αναπαριστά τη σοβαρότητα του γεγονότος επικύρωσης.

```cpp
enum class XmlSeverityType
```

### Values

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Error | 0 | Δείχνει ότι συνέβη σφάλμα επικύρωσης κατά την επικύρωση του εγγράφου του στιγμιότυπου. Αυτό ισχύει για ορισμούς τύπων εγγράφου (DTDs) και XML [Schema](../) definition language (XSD) σχήματα. Οι περιορισμοί εγκυρότητας του World Wide [Web](../../system.web/) Consortium (W3C) θεωρούνται σφάλματα. Εάν δεν έχει δημιουργηθεί χειριστής γεγονότος επικύρωσης, τα σφάλματα προκαλούν εξαίρεση. |
| Warning | 1 | Δείχνει ότι συνέβη γεγονός επικύρωσης που δεν είναι σφάλμα. Μια προειδοποίηση εκδίδεται συνήθως όταν δεν υπάρχει DTD ή XML [Schema](../) για να επικυρωθεί ένα συγκεκριμένο στοιχείο ή χαρακτηριστικό. Σε αντίθεση με τα σφάλματα, οι προειδοποιήσεις δεν προκαλούν εξαίρεση εάν δεν υπάρχει χειριστής γεγονότος επικύρωσης. |

## EventResetMode απαρίθμηση

Δείχνει πώς επαναρυθμίζεται η κατάσταση του συμβάντος.

```cpp
enum class EventResetMode
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| AutoReset | 0 | Η κατάσταση του συμβάντος επαναστέλνεται αυτόματα κατά την ανάγνωση. |
| ManualReset | 1 | Η κατάσταση του συμβάντος πρέπει να επαναρυθμιστεί χειροκίνητα. |

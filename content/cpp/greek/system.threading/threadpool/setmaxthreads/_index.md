Ορίζει τον αριθμό των νήρων που ανήκουν στο pool.


### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| wt | int | Μέγιστος αριθμός νήρων εργαζομένων. |
| cpt | int | Μέγιστος αριθμός νήρων [IO](../../../system.io/). |

### Τιμή Επιστροφής

True αν τα παραμέτρους έγιναν αποδεκτά, false διαφορετικά.

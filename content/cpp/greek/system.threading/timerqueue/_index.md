## TimerQueue κλάση

Ουρά που διαχειρίζεται αντικείμενα [Timer](../timer/). Αυτή είναι απλώς μια υλοποίηση. Τα αντικείμενα [Timer](../timer/) καταχωρούνται εκεί από μόνα τους, δεν χρειάζεται να το κάνετε για να τα χρησιμοποιήσετε - χρησιμοποιήστε το API της κλάσης [Timer](../timer/) αντ' αυτού. Αυτή είναι ένας τύπος singleton με διαχείριση μνήμης που γίνεται μέσω συναρτήσεων πρόσβασης. Δεν πρέπει ποτέ να δημιουργείτε στιγμιότυπα του απευθείας.

```cpp
class TimerQueue
```

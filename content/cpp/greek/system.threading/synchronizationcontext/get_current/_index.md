## SynchronizationContext::get_Current() μέθοδος


Αντλεί το πλαίσιο συγχρονισμού για το τρέχον νήμα.

```cpp
static const SharedPtr<SynchronizationContext> & System::Threading::SynchronizationContext::get_Current()
```


### Τιμή Επιστροφής

SharedPtr<SynchronizationContext> Ένας κοινός δείκτης στο πλαίσιο συγχρονισμού του τρέχοντος νήματος.
## Σχόλια



Επιστρέφει null αν δεν έχει οριστεί πλαίσιο συγχρονισμού για το τρέχον νήμα.

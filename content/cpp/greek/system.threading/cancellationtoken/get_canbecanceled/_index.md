## CancellationToken::get_CanBeCanceled() const μέθοδος

Λαμβάνει εάν αυτό το διακριτικό μπορεί να βρίσκεται στην κατάσταση ακύρωσης.

```cpp
bool System::Threading::CancellationToken::get_CanBeCanceled() const
```

### Τιμή Επιστροφής

true εάν αυτό το διακριτικό μπορεί να βρίσκεται στην κατάσταση ακύρωσης· διαφορετικά, false.

## Σχόλια

Διακριτικά που δημιουργήθηκαν από [CancellationTokenSource](../../cancellationtokensource/) θα επιστρέφουν true, ενώ το token None θα επιστρέφει πάντα false.

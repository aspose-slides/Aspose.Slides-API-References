## IComparer κλάση

Διεπαφή που συγκρίνει δύο αντικείμενα σε αίσθηση μεγαλύτερο-ίσον-μικρότερο. Αντικείμενα αυτής της κλάσης πρέπει να καταλαμβάνονται μόνο μέσω της συνάρτησης [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε实例 αυτού του τύπου στην στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς θα προκαλέσει σφάλματα χρόνου εκτέλεσης και/ή αστοχίες ελέγχου. Πάντα τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να τη περάσετε σε συναρτήσεις ως όρισμα.

```cpp
template<typename T>class IComparer : public virtual System::Object
```

## IList κλάση

Διεπαφή ενός ευρετήριου περιέκτη στοιχείων. Τα αντικείμενα αυτής της κλάσης πρέπει να εκχωρούνται μόνο με τη συνάρτηση [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε στιγμιότυπο αυτού του τύπου στη στοίβα ή με τον τελεστή new, διότι θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης ή/και σφάλματα assert. Πάντα να περιβάλλετε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και να χρησιμοποιείτε αυτόν τον δείκτη για να τη περνάτε σε συναρτήσεις ως όρισμα.

```cpp
template<typename T>class IList : public System::Collections::Generic::ICollection<T>
```

## IEnumerable::LINQ_All(std::function\<bool(T)>) μέθοδος

Καθορίζει εάν όλα τα στοιχεία μιας ακολουθίας ικανοποιούν μια συνθήκη.

```cpp
bool System::Collections::Generic::IEnumerable<T>::LINQ_All(std::function<bool(T)> predicate)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| predicate | std::function\<**bool**(T)> | Μία συνάρτηση για τη δοκιμή κάθε στοιχείου κατά μια συνθήκη. |

### Τιμή επιστροφής

true εάν κάθε στοιχείο της πηγής ακολουθίας περνά τη δοκιμή στον καθορισμένο predicate, ή εάν η ακολουθία είναι κενή· διαφορετικά, false.

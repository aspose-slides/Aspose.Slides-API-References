## IEnumerable::LINQ_Any() μέθοδος


Καθορίζει αν μια ακολουθία περιέχει κάποιο στοιχείο.

```cpp
bool System::Collections::Generic::IEnumerable<T>::LINQ_Any()
```


### Τιμή Επιστροφής

true if the source sequence contains any elements; otherwise, false.

## IEnumerable::LINQ_Any(std::function\<bool(T)>) μέθοδος

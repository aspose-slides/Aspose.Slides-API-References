## ISet::IsProperSubsetOf(IEnumerablePtr) method

Ελέγχει εάν το τρέχον σύνολο είναι αυστηρό υποσύνολο του άλλου container.

```cpp
virtual bool System::Collections::Generic::ISet<T>::IsProperSubsetOf(IEnumerablePtr other)=0
```

### Arguments

| Parameter | Type | Description |
| --- | --- | --- |
| other | [IEnumerablePtr](../ienumerableptr/) | Υπερσύνολο με το οποίο γίνεται ο έλεγχος. |

### Return Value

True εάν όλα τα στοιχεία στο τρέχον σύνολο υπάρχουν στο **other** και το **other** έχει περισσότερα στοιχεία από το τρέχον σύνολο, false διαφορετικά.

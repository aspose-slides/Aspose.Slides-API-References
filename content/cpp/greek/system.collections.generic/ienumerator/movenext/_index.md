## IEnumerator::MoveNext() μέθοδος

Μετακινεί τον απαριθμητή στο επόμενο στοιχείο. Εάν κανένα στοιχείο δεν είχε αναφερθεί προηγουμένως, ορίζει την αναφορά στο πρώτο διαθέσιμο στοιχείο. Εάν φτάσει το τέλος του container, δεν κάνει τίποτα.

```cpp
virtual bool System::Collections::Generic::IEnumerator<T>::MoveNext()=0
```

### Τιμή επιστροφής

True εάν κάποιο στοιχείο είναι διαθέσιμο μετά την κλήση, false αλλιώς.

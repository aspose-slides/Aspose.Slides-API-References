## LinkedList::crend() const μέθοδος

Παίρνει έναν ανάστροφο επαναλήπτη για ένα μη υπάρχων const-προσδιορισμένο στοιχείο πριν από την αρχή της συλλογής.

```cpp
const_reverse_iterator System::Collections::Generic::LinkedList<T>::crend() const noexcept
```

### Τιμή Επιστροφής

Ένας ανάστροφος επαναλήπτης που δείχνει στο θεωρητικό const-προσδιορισμένο στοιχείο που προηγείται του πρώτου στοιχείου στη συνδεδεμένη λίστα (τελευταίο στην ανάστροφη σειρά).

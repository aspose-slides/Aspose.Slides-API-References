## BaseDictionary::begin() const μέθοδος

Επιστρέφει έναν iterator στο KVPair-wrapper για το στοιχείο κλειδί-τιμή του container. Υλοποιήθηκε σε στυλ C# - ο iterator πρέπει να επιστρέφει το KVPair-object με την διεπαφή get_Key() και get_Value(). Εάν το container είναι κενό, ο επιστρεφόμενος iterator θα είναι ίσος με [end()](../../ienumerable/end/).

```cpp
const_iterator System::Collections::Generic::BaseDictionary<Map>::begin() const noexcept
```

### Τιμή Επιστροφής

Ένας iterator που δείχνει στο πρώτο στοιχείο της συλλογής.

## BaseDictionary::cbegin() const μέθοδος

Επιστρέφει έναν επαναλήπτη στο πρώτο στοιχείο του container. Υλοποιείται σε στυλ STL. Εάν το container είναι κενό, ο επιστραφέν επαναλήπτης θα είναι ίσος με [end()](../../ienumerable/end/).

```cpp
stl_const_iterator System::Collections::Generic::BaseDictionary<Map>::cbegin() const noexcept
```

### Τιμή Επιστροφής

Ένας επαναλήπτης που δείχνει στο πρώτο στοιχείο της συλλογής που είναι const-qualified.

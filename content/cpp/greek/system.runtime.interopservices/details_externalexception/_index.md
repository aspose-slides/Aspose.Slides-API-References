## Details_ExternalException κλάση


Ο βασικός τύπος εξαίρεσης για όλες τις εξαιρέσεις διαλειτουργικότητας COM και τις εξαιρέσεις διαχείρισης δομημένων εξαιρέσεων (SEH). Ποτέ μην δημιουργείτε στιγμιότυπα αυτής της κλάσης χειροκίνητα. Χρησιμοποιήστε την κλάση ExternalException αντ' αυτού. Ποτέ μην τυλίγετε τα στιγμιότυπα της κλάσης ExternalException σε [System::SmartPtr](../../system/smartptr/).

```cpp
class Details_ExternalException : public System::Details_ExceptionWithErrorCode<Details_SystemException>
```

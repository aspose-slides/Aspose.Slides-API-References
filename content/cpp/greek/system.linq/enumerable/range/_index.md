## Enumerable::Range(int32_t, int32_t) μέθοδος


Δημιουργεί μια ακολουθία ακεραίων αριθμών εντός ενός καθορισμένου εύρους.

```cpp
static SharedPtr<Collections::Generic::IEnumerable<int32_t>> System::Linq::Enumerable::Range(int32_t start, int32_t count)
```


### Παράμετροι

| Parameter | Type | Description |
| --- | --- | --- |
| start | **int32_t** | Η τιμή του πρώτου ακέραιου στην ακολουθία. |
| count | **int32_t** | Ο αριθμός των διαδοχικών ακεραίων που θα δημιουργηθούν. |

### Τιμή επιστροφής

Ένα IEnumerable από **int32_t**'s που περιέχει ένα εύρος διαδοχικών ακεραίων αριθμών.

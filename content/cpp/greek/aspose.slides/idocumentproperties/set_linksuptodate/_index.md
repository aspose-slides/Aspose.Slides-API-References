## IDocumentProperties::set_LinksUpToDate(bool) μέθοδος


Δείχνει εάν οι υπερσυνδέσεις σε ένα έγγραφο είναι ενημερωμένες. Ορίστε αυτό το στοιχείο σε **true** για να υποδείξετε ότι οι υπερσυνδέσεις έχουν ενημερωθεί. Ορίστε αυτό το στοιχείο σε **false** για να υποδείξετε ότι οι υπερσυνδέσεις είναι παλαιές. Γράψτε **bool**.

```cpp
virtual void Aspose::Slides::IDocumentProperties::set_LinksUpToDate(bool value)=0
```

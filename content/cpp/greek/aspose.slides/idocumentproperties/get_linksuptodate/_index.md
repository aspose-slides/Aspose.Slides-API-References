## IDocumentProperties::get_LinksUpToDate() μέθοδος

Δείχνει αν οι υπερσυνδέσεις σε ένα έγγραφο είναι ενημερωμένες. Ορίστε αυτό το στοιχείο σε **true** για να υποδείξετε ότι οι υπερσυνδέσεις έχουν ενημερωθεί. Ορίστε αυτό το στοιχείο σε **false** για να υποδείξετε ότι οι υπερσυνδέσεις είναι παρωχημένες. Ανάγνωση **bool**.

```cpp
virtual bool Aspose::Slides::IDocumentProperties::get_LinksUpToDate()=0
```

## IDocumentProperties::set_ScaleCrop(bool) μέθοδος


Δηλώνει τη λειτουργία εμφάνισης της μικρογραφίας του εγγράφου. Ορίστε αυτό το στοιχείο σε **true** για να ενεργοποιήσετε την κλιμάκωση της μικρογραφίας του εγγράφου στην οθόνη. Ορίστε αυτό το στοιχείο σε **false** για να ενεργοποιήσετε την περικοπή της μικρογραφίας του εγγράφου ώστε να εμφανίζονται μόνο τμήματα που ταιριάζουν στην οθόνη. Γράψτε **bool**.

```cpp
virtual void Aspose::Slides::IDocumentProperties::set_ScaleCrop(bool value)=0
```

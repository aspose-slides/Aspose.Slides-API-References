## IDocumentProperties::get_ScaleCrop() μέθοδος


Καθορίζει τη λειτουργία εμφάνισης της μικρογραφίας του εγγράφου. Ορίστε αυτό το στοιχείο σε **true** για να ενεργοποιήσετε την κλιμάκωση της μικρογραφίας του εγγράφου στην οθόνη. Ορίστε αυτό το στοιχείο σε **false** για να ενεργοποιήσετε την περικοπή της μικρογραφίας του εγγράφου ώστε να εμφανίζει μόνο τμήματα που ταιριάζουν στην οθόνη. Ανάγνωση **bool**.

```cpp
virtual bool Aspose::Slides::IDocumentProperties::get_ScaleCrop()=0
```

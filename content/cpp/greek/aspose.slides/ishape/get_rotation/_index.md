## IShape::get_Rotation() μέθοδος


Επιστρέφει τον αριθμό των μοιρών που το καθορισμένο σχήμα περιστρέφεται γύρω από τον άξονα z. Μια θετική τιμή υποδεικνύει δεξιόστροφη περιστροφή· μια αρνητική τιμή υποδεικνύει αριστερόστροφη περιστροφή. Διαβάζει **float**.

```cpp
virtual float Aspose::Slides::IShape::get_Rotation()=0
```

## Παρατηρήσεις


Η επιστρεφόμενη τιμή είναι πάντα ορισμένη (δεν είναι std::numeric_limits<float>::quiet_NaN()). Η ανατεθειμένη τιμή πρέπει να είναι ορισμένη (όχι std::numeric_limits<float>::quiet_NaN()). Μπορείτε να ορίσετε μη ορισμένες τιμές για τις ιδιότητες του αντικειμένου RawFrame.

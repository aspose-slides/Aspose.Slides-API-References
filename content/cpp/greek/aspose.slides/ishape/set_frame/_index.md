## IShape::set_Frame(System::SharedPtr\<IShapeFrame\>) μέθοδος

Ορίζει τις ιδιότητες του πλαισίου του σχήματος. Γράψτε [IShapeFrame](../../ishapeframe/).

```cpp
virtual void Aspose::Slides::IShape::set_Frame(System::SharedPtr<IShapeFrame> value)=0
```

## Παρατηρήσεις

Η τιμή κάθε ιδιότητας της επιστρεφόμενης [IShapeFrame](../../ishapeframe/) εμφάνισης δεν είναι ακαθόριστη (δεν είναι NaN ή NotDefined). Η τιμή κάθε ιδιότητας της ανατιθέμενης [IShapeFrame](../../ishapeframe/) εμφάνισης πρέπει να μην είναι ακαθόριστη (πρέπει να μην είναι NaN ή NotDefined). Μπορείτε να ορίσετε ακαθόριστες τιμές για τις ιδιότητες της παρουσίασης RawFrame.

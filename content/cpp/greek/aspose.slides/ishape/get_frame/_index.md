## IShape::get_Frame() μέθοδος


Επιστρέφει τις ιδιότητες του πλαισίου του σχήματος. Διαβάστε [IShapeFrame](../../ishapeframe/).

```cpp
virtual System::SharedPtr<IShapeFrame> Aspose::Slides::IShape::get_Frame()=0
```

## Παρατηρήσεις


Η τιμή κάθε ιδιότητας του επιστρεφόμενου [IShapeFrame](../../ishapeframe/) αντικειμένου δεν είναι ακαθόριστη (δεν είναι NaN ή NotDefined). Η τιμή κάθε ιδιότητας του εκχωρημένου [IShapeFrame](../../ishapeframe/) αντικειμένου πρέπει να μην είναι ακαθόριστη (πρέπει να μην είναι NaN ή NotDefined). Μπορείτε να ορίσετε ακαθόριστες τιμές για τις ιδιότητες του αντικειμένου RawFrame.

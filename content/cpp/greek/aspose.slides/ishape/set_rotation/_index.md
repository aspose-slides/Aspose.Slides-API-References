## IShape::set_Rotation(float) μέθοδος

Ορίζει τον αριθμό των μοιρών που το συγκεκριμένο σχήμα περιστρέφεται γύρω από τον άξονα z. Μια θετική τιμή υποδεικνύει δεξιόστροφη περιστροφή· μια αρνητική τιμή υποδεικνύει αριστερόστροφη περιστροφή. Γράψτε **float**.

```cpp
virtual void Aspose::Slides::IShape::set_Rotation(float value)=0
```

## Παρατηρήσεις

Η επιστρεφόμενη τιμή είναι πάντα ορισμένη (δεν είναι std::numeric_limits<float>::quiet_NaN()). Η εκχωρημένη τιμή πρέπει να είναι ορισμένη (δεν είναι std::numeric_limits<float>::quiet_NaN()). Μπορείτε να ορίσετε μη ορισμένες τιμές για τις ιδιότητες της περίπτωσης RawFrame.

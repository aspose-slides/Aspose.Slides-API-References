## SensitivityLabelContentType enum


Αντιπροσωπεύει τους τύπους σήμανσης περιεχομένου που πρέπει να εφαρμοστούν σε ένα έγγραφο.

```cpp
enum class SensitivityLabelContentType
```

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| None | 0 | Υποδηλώνει ότι η ετικέτα εφαρμόζεται προεπιλεγμένα ή αυτόματα. |

| Footer | 2 | Υποδηλώνει ότι η ετικέτα εφαρμόστηκε χειροκίνητα. Χρησιμοποιείται για οποιαδήποτε ετικέτα ευαισθησίας που εφαρμόστηκε άμεσα από το χρήστη. |
| Watermark | 4 | Υποδηλώνει ότι η ετικέτα εφαρμόστηκε χειροκίνητα. Χρησιμοποιείται για οποιαδήποτε ετικέτα ευαισθησίας που εφαρμόστηκε άμεσα από το χρήστη. |

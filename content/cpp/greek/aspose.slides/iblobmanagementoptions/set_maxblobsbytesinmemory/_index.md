## IBlobManagementOptions::set_MaxBlobsBytesInMemory(uint64_t) μέθοδος

Ορίζει το μέγιστο συνολικό μέγεθος (σε byte) που μπορούν να καταλαμβάνουν όλα τα BLOB στη μνήμη. Από προεπιλογή, όλα τα BLOB φορτώνονται στη μνήμη· μόνο όταν φτάσει αυτό το όριο, χρησιμοποιούνται εναλλακτικοί μηχανισμοί (όπως προσωρινά αρχεία). Η διατήρηση των BLOB στη μνήμη μεγιστοποιεί την απόδοση αλλά μπορεί να οδηγήσει σε υψηλή χρήση μνήμης. Χρησιμοποιήστε αυτή την ιδιότητα για να προσαρμόσετε τη συμπεριφορά στο περιβάλλον ή τις απαιτήσεις σας.

```cpp
virtual void Aspose::Slides::IBlobManagementOptions::set_MaxBlobsBytesInMemory(uint64_t value)=0
```

## Παρατηρήσεις

Αυτή η τιμή αγνοείται εάν το [IBlobManagementOptions::set_IsTemporaryFilesAllowed](../set_istemporaryfilesallowed/) είναι ορισμένο σε false, επειδή η μνήμη είναι τότε η μόνη διαθέσιμη τοποθεσία αποθήκευσης και ο περιορισμός της χρήσης BLOB στη μνήμη δεν έχει αποτέλεσμα.  

Η προεπιλεγμένη τιμή είναι 629 145 600 byte (600 MB).  

Μπορείτε να ορίσετε αυτή την ιδιότητα στο μηδέν, αλλά ένα μικρό ελάχιστο ποσό μνήμης θα εξακολουθεί να δεσμεύεται.

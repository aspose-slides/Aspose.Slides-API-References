## Portion::get_PortionFormat() μέθοδος

Επιστρέφει αντικείμενο μορφοποίησης που περιέχει ρητά ορισμένες ιδιότητες μορφοποίησης του κειμένου της μερίδας χωρίς κληρονομική εφαρμογή. Μόνο για ανάγνωση [IPortionFormat](../../iportionformat/).

```cpp
System::SharedPtr<IPortionFormat> Aspose::Slides::Portion::get_PortionFormat() override
```

## Παρατηρήσεις

Το αντικείμενο μορφοποίησης περιέχει τις παραμέτρους μορφοποίησης που ορίζονται μόνο για την τρέχουσα μερίδα, δεν εφαρμόζονται τα κληρονομημένα δεδομένα.

Για να λάβετε τις αποτελεσματικές τιμές, συμπεριλαμβανομένων των κληρονομημένων, χρησιμοποιήστε τη [PortionFormat::GetEffective](../../portionformat/geteffective/) μέθοδο.

## IShapeCollection::AddMathShape(float, float, float, float) μέθοδος

Δημιουργεί ένα νέο αυτόματο σχήμα ορθογωνίου για τη φιλοξενία μαθηματικού περιεχομένου και το προσθέτει στο τέλος της συλλογής σχημάτων.

```cpp
virtual System::SharedPtr<IAutoShape> Aspose::Slides::IShapeCollection::AddMathShape(float x, float y, float width, float height)=0
```

### Παράμετροι

| Parameter | Type | Description |
| --- | --- | --- |
| x | **float** | Η συντεταγμένη x του πλαισίου του σχήματος, σε μονάδες σημείου. |
| y | **float** | Η συντεταγμένη y του πλαισίου του σχήματος, σε μονάδες σημείου. |
| width | **float** | Το πλάτος του πλαισίου του σχήματος, σε μονάδες σημείου. |
| height | **float** | Το ύψος του πλαισίου του σχήματος, σε μονάδες σημείου. |

## IProtectionManager::set_EncryptDocumentProperties(bool) μέθοδος

Αυτή η ιδιότητα έχει νόημα, εάν η παρουσίαση είναι προστατευμένη με κωδικό. Εάν είναι true, τότε οι ιδιότητες του εγγράφου κρυπτογραφούνται στο αρχείο παρουσίασης. Εάν είναι false, τότε οι ιδιότητες του εγγράφου είναι δημόσιες ενώ η παρουσίαση είναι κρυπτογραφημένη. Γράψτε **bool**.

```cpp
virtual void Aspose::Slides::IProtectionManager::set_EncryptDocumentProperties(bool value)=0
```

## IProtectionManager::get_IsEncrypted() μέθοδος


Λαμβάνει μια τιμή που υποδεικνύει εάν αυτή η παρουσία είναι κρυπτογραφημένη. Μόνο ανάγνωση **bool**.

```cpp
virtual bool Aspose::Slides::IProtectionManager::get_IsEncrypted()=0
```

## Παρατηρήσεις


**true** εάν η παρουσία φορτώθηκε από κρυπτογραφημένο αρχείο ή κλήθηκε η μέθοδος [Encrypt(String)](../encrypt/)· διαφορετικά, **false**.

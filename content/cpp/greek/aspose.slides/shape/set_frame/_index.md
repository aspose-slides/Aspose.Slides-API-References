## Shape::set_Frame(System::SharedPtr\<IShapeFrame\>) μέθοδος

Ορίζει τις ιδιότητες του πλαισίου του σχήματος. Γράψτε [IShapeFrame](../../ishapeframe/).

```cpp
void Aspose::Slides::Shape::set_Frame(System::SharedPtr<IShapeFrame> value) override
```

## Σχόλια

Η τιμή κάθε ιδιότητας του επιστρεφόμενου [IShapeFrame](../../ishapeframe/) αντικειμένου δεν είναι undefined (δεν είναι NaN ή NotDefined). Η τιμή κάθε ιδιότητας του εκχωρημένου [IShapeFrame](../../ishapeframe/) αντικειμένου πρέπει να μην είναι undefined (πρέπει να μην είναι NaN ή NotDefined). Μπορείτε να ορίσετε τιμές undefined για τις ιδιότητες του αντικειμένου RawFrame.

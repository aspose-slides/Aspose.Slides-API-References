## Shape::get_Rotation() μέθοδος

Επιστρέφει τον αριθμό των μοιρών κατά τις οποίες το συγκεκριμένο σχήμα περιστρέφεται γύρω από τον άξονα z. Μια θετική τιμή υποδηλώνει δεξιόστροφη περιστροφή· μια αρνητική τιμή υποδηλώνει αριστερόστροφη περιστροφή. Ανάγνωση **float**.

```cpp
float Aspose::Slides::Shape::get_Rotation() override
```

## Παρατηρήσεις

Η επιστρεφόμενη τιμή είναι πάντα ορισμένη (δεν είναι std::numeric_limits<float>::quiet_NaN()). Η τιμή που αποδίδεται πρέπει να είναι ορισμένη (όχι std::numeric_limits<float>::quiet_NaN()). Μπορείτε να ορίσετε μη ορισμένες τιμές για τις ιδιότητες του αντικειμένου RawFrame.

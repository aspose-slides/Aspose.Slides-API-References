## Shape::get_Frame() μέθοδος


Επιστρέφει τις ιδιότητες του πλαισίου του σχήματος. Διαβάστε [IShapeFrame](../../ishapeframe/).

```cpp
System::SharedPtr<IShapeFrame> Aspose::Slides::Shape::get_Frame() override
```

## Παρατηρήσεις


Η τιμή κάθε ιδιότητας του επιστρεφόμενου [IShapeFrame](../../ishapeframe/) αντικειμένου δεν είναι αόριστη (δεν είναι NaN ή NotDefined). Η τιμή κάθε ιδιότητας του εκχωρημένου [IShapeFrame](../../ishapeframe/) αντικειμένου πρέπει να μην είναι αόριστη (πρέπει να μην είναι NaN ή NotDefined). Μπορείτε να ορίσετε αόριστες τιμές για τις ιδιότητες του αντικειμένου RawFrame.

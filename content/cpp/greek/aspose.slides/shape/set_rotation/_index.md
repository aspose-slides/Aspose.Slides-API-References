## Shape::set_Rotation(float) μέθοδος

Ορίζει τον αριθμό των μοιρών κατά τον οποίο το συγκεκριμένο σχήμα περιστρέφεται γύρω από τον άξονα z. Μία θετική τιμή δείχνει δεξιόστροφη περιστροφή· μια αρνητική τιμή δείχνει αριστερόστροφη περιστροφή. Γράψτε **float**.

```cpp
void Aspose::Slides::Shape::set_Rotation(float value) override
```

## Σχόλια

Η επιστρεφόμενη τιμή είναι πάντα ορισμένη (δεν είναι std::numeric_limits<float>::quiet_NaN()). Η εκχωρημένη τιμή πρέπει να είναι ορισμένη (μη std::numeric_limits<float>::quiet_NaN()). Μπορείτε να ορίσετε αόριστες τιμές για τις ιδιότητες του αντικειμένου RawFrame.

## EmbeddingLevel enum

Αντιπροσωπεύει τα δικαιώματα άδειας χρήσης για την ενσωμάτωση της γραμματοσειράς.

```cpp
enum class EmbeddingLevel : uint16_t
```

### Values

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Installable | 0 | [Fonts](../fonts/) με αυτή τη ρύθμιση υποδεικνύει ότι μπορούν να ενσωματωθούν και να εγκατασταθούν μόνιμα στο απομακρυσμένο σύστημα από μια εφαρμογή. Ο χρήστης του απομακρυσμένου συστήματος αποκτά τα ίδια δικαιώματα, υποχρεώσεις και άδειες για αυτή τη γραμματοσειρά όπως ο αρχικός αγοραστής της γραμματοσειράς, και υπόκειται στην ίδια συμφωνία άδειας τελικού χρήστη, πνευματικά δικαιώματα, πατέντα σχεδίου και/ή εμπορικό σήμα όπως ο αρχικός αγοραστής. |
| Restricted | 2 | [Fonts](../fonts/) που έχουν μόνο αυτό το δυαδικό σημείο ενεργοποιημένο δεν πρέπει να τροποποιούνται, ενσωματώνονται ή ανταλλαγούν με κανέναν τρόπο χωρίς πρώτα την άδεια του νόμιμου κατόχου. |
| PreviewPrint | 4 | Όταν αυτό το δυαδικό σημείο είναι ενεργοποιημένο, η γραμματοσειρά μπορεί να ενσωματωθεί και να φορτωθεί προσωρινά στο απομακρυσμένο σύστημα. Τα έγγραφα που περιέχουν γραμματοσειρές Preview & Print πρέπει να ανοίγονται \"read-only;\"· δεν επιτρέπεται καμία επεξεργασία του εγγράφου. |
| Editable | 8 | Όταν αυτό το δυαδικό σημείο είναι ενεργοποιημένο, η γραμματοσειρά μπορεί να ενσωματωθεί αλλά πρέπει να εγκατασταθεί μόνο προσωρινά σε άλλα συστήματα. Σε αντίθεση με τις γραμματοσειρές Preview & Print, τα έγγραφα που περιέχουν γραμματοσειρές Editable μπορούν να ανοίγονται για ανάγνωση, επιτρέπεται η επεξεργασία και οι αλλαγές μπορούν να αποθηκευτούν. |
| NoSubsetting | 256 | Όταν αυτό το δυαδικό σημείο είναι ενεργοποιημένο, η γραμματοσειρά δεν μπορεί να υποσύνολο (subset) πριν την ενσωμάτωση. Εφαρμόζονται επίσης άλλοι περιορισμοί ενσωμάτωσης που καθορίζονται στα bits 0-3 και 9. |
| BitmapOnly | 512 | Όταν αυτό το δυαδικό σημείο είναι ενεργοποιημένο, μπορούν να ενσωματωθούν μόνο τα bitmap που περιέχονται στη γραμματοσειρά. Δεν μπορούν να ενσωματωθούν δεδομένα γραμμής. Εάν δεν υπάρχουν διαθέσιμα bitmap στη γραμματοσειρά, τότε η γραμματοσειρά θεωρείται μη ενσωματώσιμη και οι υπηρεσίες ενσωμάτωσης θα αποτύχουν. |

## Tab::CompareTo(System::SharedPtr\<ITab\>) μέθοδος


Compares the current instance with another object of the same type.

```cpp
int32_t Aspose::Slides::Tab::CompareTo(System::SharedPtr<ITab> obj) override
```


### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| obj | [System::SharedPtr](../../../system/sharedptr/)\<[ITab](../../itab/)\> | Ένα αντικείμενο προς σύγκριση με αυτήν την παρουσία. |

### Τιμή Επιστροφής

Ένας ακέραιος 32-bit που υποδεικνύει τη σχετική σειρά των συγκρίσιμων. Η τιμή επιστροφής έχει τις εξής σημασίες:
* < 0 - Αυτή η παρουσία είναι μικρότερη από το obj.
* = 0 - Αυτή η παρουσία είναι ίση με το obj.
* > 0 - Αυτή η παρουσία είναι μεγαλύτερη από το obj.

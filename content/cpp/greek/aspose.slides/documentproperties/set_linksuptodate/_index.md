## DocumentProperties::set_LinksUpToDate(bool) μέθοδος

Καθορίζει εάν οι υπερσύνδεσμοι σε ένα έγγραφο είναι ενημερωμένοι. Ορίστε αυτό το στοιχείο σε **true** για να υποδείξετε ότι οι υπερσύνδεσμοι έχουν ενημερωθεί. Ορίστε αυτό το στοιχείο σε **false** για να υποδείξετε ότι οι υπερσύνδεσμοι είναι παρωχημένοι. Γράψτε **bool**.

```cpp
void Aspose::Slides::DocumentProperties::set_LinksUpToDate(bool value) override
```

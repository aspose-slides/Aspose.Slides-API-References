## DocumentProperties::set_ScaleCrop(bool) μέθοδος

Δείχνει τη λειτουργία εμφάνισης της μικρογραφίας του εγγράφου. Ορίστε αυτό το στοιχείο σε **true** για να ενεργοποιήσετε την κλιμάκωση της μικρογραφίας του εγγράφου στην οθόνη. Ορίστε αυτό το στοιχείο σε **false** για να ενεργοποιήσετε την περικοπή της μικρογραφίας του εγγράφου ώστε να εμφανίζονται μόνο τμήματα που ταιριάζουν στην οθόνη. Γράψτε **bool**.

```cpp
void Aspose::Slides::DocumentProperties::set_ScaleCrop(bool value) override
```

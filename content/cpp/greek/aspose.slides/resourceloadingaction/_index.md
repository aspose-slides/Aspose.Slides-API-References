## ResourceLoadingAction enum

Καθορίζει τη λειτουργία φόρτωσης εξωτερικών πόρων.

```cpp
enum class ResourceLoadingAction
```

### Values

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Default | 0 | [Aspose.Slides](../) θα φορτώσει τον εξωτερικό πόρο όπως συνήθως. |
| Skip | 1 | [Aspose.Slides](../) θα παραλείψει τη φόρτωση του εξωτερικού πόρου. Μόνο ο σύνδεσμος χωρίς δεδομένα θα αποθηκευτεί για μια εικόνα. |
| UserProvided | 2 | [Aspose.Slides](../) θα χρησιμοποιήσει τον πίνακα byte που παρέχεται από τον χρήστη στο [IResourceLoadingArgs::SetData](../iresourceloadingargs/setdata/) ως δεδομένα εικόνας. |

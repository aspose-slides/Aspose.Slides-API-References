## ILoadOptions::get_BlobManagementOptions() μέθοδος

Αντιπροσωπεύει τις επιλογές που μπορούν να χρησιμοποιηθούν για τη διαχείριση της συμπεριφοράς χειρισμού των Binary Large Objects (BLOBs), όπως η χρήση προσωρινών αρχείων ή το μέγιστο αριθμό bytes BLOB στη μνήμη. Αυτές οι επιλογές προορίζονται να ρυθμίσουν τον καλύτερο λόγο απόδοσης/κατανάλωσης μνήμης για ένα συγκεκριμένο περιβάλλον ή απαιτήσεις.

```cpp
virtual System::SharedPtr<IBlobManagementOptions> Aspose::Slides::ILoadOptions::get_BlobManagementOptions()=0
```

## Παρατηρήσεις

Ένα Binary Large Object (BLOB) είναι δυαδικά δεδομένα αποθηκευμένα ως μία ενιαία οντότητα - δηλαδή το BLOB μπορεί να είναι ήχος, βίντεο ή η παρουσίαση ίδια.

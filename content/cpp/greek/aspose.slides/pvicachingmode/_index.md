## PVICachingMode enum

Αντιπροσωπεύει διαφορετικά επίπεδα προσωρινής αποθήκευσης για τα αποτελεσματικά αντικείμενα PVI.

```cpp
enum class PVICachingMode
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| NoCaching | 0 | Κλασική λειτουργία με δημιουργία κάθε στιγμιότυπου. |
| NotDefinedOnly | 1 | Χρήση προσωρινής αποθήκευσης αποτελεσματικών τιμών για μορφές NotDefined, κλασική λειτουργία για τις υπόλοιπες. |
| Full | 2 | Χρήση προσωρινής αποθήκευσης αποτελεσματικών τιμών για όλες τις μορφές. |

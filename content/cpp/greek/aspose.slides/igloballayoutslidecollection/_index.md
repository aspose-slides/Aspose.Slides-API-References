## IGlobalLayoutSlideCollection κλάση

Represents a collection of all layout slides in presentation. Extends [ILayoutSlideCollection](../ilayoutslidecollection/) interface with methods for adding/cloning layout slides in context of uniting of the individual collections of master's layout slides.

```cpp
class IGlobalLayoutSlideCollection : public virtual Aspose::Slides::ILayoutSlideCollection
```

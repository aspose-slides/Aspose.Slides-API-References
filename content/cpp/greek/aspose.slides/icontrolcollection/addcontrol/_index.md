## IControlCollection::AddControl(ControlType, float, float, float, float) method

Δημιουργεί και προσθέτει έναν νέο έλεγχο στη συλλογή.

```cpp
virtual System::SharedPtr<IControl> Aspose::Slides::IControlCollection::AddControl(ControlType controlType, float x, float y, float width, float height)=0
```

### Arguments

| Parameter | Type | Description |
| --- | --- | --- |
| controlType | [ControlType](../../controltype/) | Τύπος ελέγχου προς προσθήκη. |
| x | **float** | Η συντεταγμένη X για το αριστερό μέρος του πλαισίου του σχήματος. |
| y | **float** | Η συντεταγμένη Y για το επάνω μέρος του πλαισίου του σχήματος. |
| width | **float** | Το πλάτος του πλαισίου του σχήματος. |
| height | **float** | Το ύψος του πλαισίου του σχήματος. |

### Return Value

Δημιουργημένος έλεγχος [IControl](../../icontrol/).

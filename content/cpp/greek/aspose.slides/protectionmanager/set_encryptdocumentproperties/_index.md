## ProtectionManager::set_EncryptDocumentProperties(bool) μέθοδος

Αυτή η ιδιότητα έχει νόημα, εάν η παρουσίαση είναι προστατευμένη με κωδικό. Εάν είναι true, τότε οι ιδιότητες εγγράφου κρυπτογραφούνται στο αρχείο παρουσίασης. Εάν είναι false, τότε οι ιδιότητες εγγράφου είναι δημόσιες ενώ η παρουσίαση είναι κρυπτογραφημένη. Γράψτε **bool**.

```cpp
void Aspose::Slides::ProtectionManager::set_EncryptDocumentProperties(bool value) override
```

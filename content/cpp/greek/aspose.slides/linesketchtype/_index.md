## LineSketchType enum

Αναπαριστά τον τύπο σκίτσου ή το εφέ που έχει εκχωρηθεί σε ένα σχήμα.

```cpp
enum class LineSketchType
```

### Values

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| NotDefined | -1 | Καθορίζει ότι το εφέ Sketch ενός σχήματος είναι ακαθόριστο. |
| None | 0 | Καθορίζει ότι ένα σχήμα δεν έχει εφέ Sketch. Αυτό ισοδυναμεί με το ότι αυτή η ιδιότητα είναι κενή. |
| Curved | 1 | Καθορίζει ότι ένα σχήμα έχει το εφέ Curved, που μετατρέπει κάθε πλευρά του σχήματος σε μια μεγάλη ήπια καμπύλη. |
| Freehand | 2 | Καθορίζει ότι ένα σχήμα έχει το εφέ Freehand, που πιο πολύ μοιάζει με μια ατελώς σχεδιασμένη γραμμή. |
| Scribble | 3 | Καθορίζει ότι ένα σχήμα έχει το εφέ Scribble, που έχει υπερβολική ταλάντωση σαν να σχεδιάστηκε σκόπιμα ακατάστατα. |

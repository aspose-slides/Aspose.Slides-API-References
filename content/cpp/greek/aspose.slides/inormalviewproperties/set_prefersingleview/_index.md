## INormalViewProperties::set_PreferSingleView(bool) μέθοδος


Καθορίζει αν ο χρήστης προτιμά να δει μια περιοχή περιεχομένου μονού στοιχείου σε πλήρες παράθυρο αντί για την τυπική κανονική προβολή με τρεις περιοχές περιεχομένου. Εάν είναι ενεργοποιημένο, η εφαρμογή μπορεί να επιλέξει να εμφανίσει μία από τις περιοχές περιεχομένου σε ολόκληρο το παράθυρο. Γράψτε **bool**.

```cpp
virtual void Aspose::Slides::INormalViewProperties::set_PreferSingleView(bool value)=0
```

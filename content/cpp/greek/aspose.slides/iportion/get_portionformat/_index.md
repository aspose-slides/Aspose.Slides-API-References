## IPortion::get_PortionFormat() μέθοδος


Επιστρέφει αντικείμενο μορφοποίησης που περιέχει ρητά ορισμένες ιδιότητες μορφοποίησης του τμήματος κειμένου χωρίς να εφαρμοστεί κληρονομικότητα. Μόνο για ανάγνωση [IPortionFormat](../../iportionformat/).

```cpp
virtual System::SharedPtr<IPortionFormat> Aspose::Slides::IPortion::get_PortionFormat()=0
```

## Παρατηρήσεις


Το αντικείμενο μορφοποίησης περιέχει τις παραμέτρους μορφοποίησης που ορίζονται μόνο για το τρέχον τμήμα, χωρίς να εφαρμοστούν κληρονομημένα δεδομένα.

Για να λάβετε τις αποτελεσματικές τιμές, συμπεριλαμβανομένων των κληρονομημένων, χρησιμοποιήστε τη μέθοδο [IPortionFormat::GetEffective](../../iportionformat/geteffective/).

## SplitterBarStateType enum

Καθορίζει την κατάσταση στην οποία πρέπει να εμφανίζεται η γραμμή διαχωριστή. Η γραμμή διαχωριστή χωρίζει μια πρωτεύουσα και μια δευτερεύουσα περιοχή μέσα σε μια περιοχή προβολής.

```cpp
enum class SplitterBarStateType
```

### Τιμές

| Name | Value | Description |
| --- | --- | --- |
| Minimized | 0 | Η πρωτεύουσα περιοχή καταλαμβάνει το ελάχιστο μέρος της περιοχής προβολής που επιτρέπεται από την εφαρμογή. |
| Restored | 1 | Η πρωτεύουσα περιοχή έχει ένα συγκεκριμένο ενδιάμεσο μέγεθος. |
| Maximized | 2 | Η πρωτεύουσα περιοχή καταλαμβάνει το μέγιστο μέρος της περιοχής προβολής που επιτρέπεται από την εφαρμογή. |

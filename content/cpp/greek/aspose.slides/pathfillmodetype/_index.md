## PathFillModeType enum

Καθορίζει τον τρόπο με τον οποίο πρέπει να γεμιστεί μια διαδρομή

```cpp
enum class PathFillModeType : uint8_t
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| None | 0 | Χωρίς γέμισμα διαδρομής. Αυτό καθορίζει ότι η αντίστοιχη διαδρομή δεν πρέπει να έχει γέμισμα. |
| Normal | 1 | Κανονικό γέμισμα διαδρομής. Αυτό καθορίζει ότι η αντίστοιχη διαδρομή πρέπει να έχει ένα κανονικά σκιασμένο χρώμα εφαρμοσμένο στο it\\u2019s fill. |
| Lighten | 2 | Ανοιχτό γέμισμα διαδρομής. Αυτό καθορίζει ότι η αντίστοιχη διαδρομή πρέπει να έχει ένα ελαφρώς σκιασμένο χρώμα εφαρμοσμένο στο it\\u2019s fill. |
| LightenLess | 3 | Λιγότερο ανοιχτό γέμισμα διαδρομής. Αυτό καθορίζει ότι η αντίστοιχη διαδρομή πρέπει να έχει ένα ελαφρώς πιο ανοιχτό σκιασμένο χρώμα εφαρμοσμένο στο it\\u2019s fill. |
| Darken | 4 | Σκοτεινό γέμισμα διαδρομής. Αυτό καθορίζει ότι η αντίστοιχη διαδρομή πρέπει να έχει ένα πιο σκούρο σκιασμένο χρώμα εφαρμοσμένο στο it\\u2019s fill. |
| DarkenLess | 5 | Λιγότερο σκοτεινό γέμισμα διαδρομής. Αυτό καθορίζει ότι η αντίστοιχη διαδρομή πρέπει να έχει ένα ελαφρώς πιο σκούρο σκιασμένο χρώμα εφαρμοσμένο στο it\\u2019s fill. |

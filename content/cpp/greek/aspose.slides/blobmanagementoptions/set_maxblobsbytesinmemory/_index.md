## BlobManagementOptions::set_MaxBlobsBytesInMemory(uint64_t) μέθοδος

Ορίζει το μέγιστο συνολικό μέγεθος (σε byte) που μπορούν να καταλαμβάνουν όλα τα BLOBs στη μνήμη. Από προεπιλογή, όλα τα BLOB φορτώνονται στη μνήμη· μόνο όταν αυτό το όριο φτάσει θα χρησιμοποιηθούν εναλλακτικοί μηχανισμοί (όπως προσωρινά αρχεία). Η διατήρηση των BLOB στη μνήμη βελτιστοποιεί την απόδοση, αλλά μπορεί να οδηγήσει σε υψηλή κατανάλωση μνήμης. Χρησιμοποιήστε αυτή την ιδιότητα για να προσαρμόσετε τη συμπεριφορά στο περιβάλλον ή τις απαιτήσεις σας.

```cpp
void Aspose::Slides::BlobManagementOptions::set_MaxBlobsBytesInMemory(uint64_t value) override
```

## Παρατηρήσεις

Αυτή η τιμή αγνοείται εάν [BlobManagementOptions::set_IsTemporaryFilesAllowed](../set_istemporaryfilesallowed/) ορίζεται σε false, καθώς η μνήμη είναι τότε η μοναδική διαθέσιμη τοποθεσία αποθήκευσης και ο περιορισμός της χρήσης BLOB στη μνήμη δεν έχει αποτέλεσμα.

Η προεπιλεγμένη τιμή είναι 629,145,600 bytes (600 MB).

Μπορείτε να ορίσετε αυτή την ιδιότητα στο μηδέν, αλλά ένα μικρό ελάχιστο ποσό μνήμης θα παραμείνει δεσμευμένο.

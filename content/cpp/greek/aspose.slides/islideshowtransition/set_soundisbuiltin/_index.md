## ISlideShowTransition::set_SoundIsBuiltIn(bool) μέθοδος

Καθορίζει αν αυτός ο ήχος είναι ενσωματωμένος. Εάν αυτό το χαρακτηριστικό οριστεί σε true, τότε η εφαρμογή που δημιουργεί ειδοποιείται να ελέγξει το χαρακτηριστικό name που καθορίζεται για αυτόν τον ήχο στη λίστα των ενσωματωμένων ήχων και μπορεί στη συνέχεια να εμφανίσει ένα προσαρμοσμένο όνομα ή UI ανάλογα με τις ανάγκες. Γράφει **bool**.

virtual void Aspose::Slides::ISlideShowTransition::set_SoundIsBuiltIn(bool value)=0

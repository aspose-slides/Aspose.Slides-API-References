## License::License() Κατασκευαστής

Αρχικοποιεί μια νέα παρουσία αυτής της κλάσης.

```cpp
Aspose::Slides::License::License()
```

## Παρατηρήσεις

Σε αυτό το παράδειγμα, θα γίνει προσπάθεια να βρεθεί ένα αρχείο άδειας με όνομα MyLicense.lic στο φάκελο που περιέχει το στοιχείο, στο φάκελο που περιέχει τη καλούμενη συναρμολόγηση, στο φάκελο της εισαγόμενης συναρμολόγησης και, στη συνέχεια, στους ενσωματωμένους πόρους της καλούμενης συναρμολόγησης.
```cpp
auto license = MakeObject<License>();
license->SetLicense(u"MyLicense.lic");
```

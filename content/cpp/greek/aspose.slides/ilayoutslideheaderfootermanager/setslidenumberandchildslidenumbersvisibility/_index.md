## ILayoutSlideHeaderFooterManager::SetSlideNumberAndChildSlideNumbersVisibility(bool) μέθοδος


Αλλάζει την ορατότητα του σημείου κράτησης αριθμού σελίδας της διαφάνειας διάταξης και όλων των σημείων κράτησης αριθμών των θυγατρικών διαφανειών. Τα θυγατρικά σημεία κράτησης σημαίνουν ότι τα σημεία κράτησης περιέχονται σε εξαρτημένες διαφάνειες. Οι εξαρτημένες διαφάνειες χρησιμοποιούν και εξαρτώνται από τη διαφάνεια διάταξης.

```cpp
virtual void Aspose::Slides::ILayoutSlideHeaderFooterManager::SetSlideNumberAndChildSlideNumbersVisibility(bool isVisible)=0
```


### Παράμετροι

| Parameter | Type | Description |
| --- | --- | --- |
| isVisible | **bool** | true - κάνει τις ετικέτες αριθμού σελίδας ορατές, διαφορετικά - τις κρύβει. |

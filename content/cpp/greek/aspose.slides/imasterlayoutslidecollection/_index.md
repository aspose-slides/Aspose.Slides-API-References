## IMasterLayoutSlideCollection κλάση

Αντιπροσωπεύει μια συλλογή όλων των διαφανειών διάταξης του καθορισμένου κύριου σλάϊδα. Επεκτείνει τη διεπαφή [ILayoutSlideCollection](../ilayoutslidecollection/) με μεθόδους για προσθήκη/εισαγωγή/αφαίρεση/κλωνοποίηση διαφανειών διάταξης στο πλαίσιο των μεμονωμένων συλλογών των διαφανειών διάταξης του κύριου σλάϊδα.

```cpp
class IMasterLayoutSlideCollection : public virtual Aspose::Slides::ILayoutSlideCollection
```

## FillType enum

Καθορίζει τον εσωτερικό τύπο γεμίσματος διαφόρων οπτικών αντικειμένων.

```cpp
enum class FillType : int8_t
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| NotDefined | -1 | Ο τύπος γεμίσματος δεν έχει οριστεί. |
| NoFill | 0 | Δεν έχει εφαρμοστεί γέμισμα. |
| Solid | 1 | Γεμισμένο με ένα ενιαίο χρώμα. |
| Gradient | 2 | Το γέμισμα είναι διαβάθμιση. |
| Pattern | 3 | Χρησιμοποιείται επαναλαμβανόμενο μοτίβο για το γέμισμα του αντικειμένου. |
| Picture | 4 | Μία ενιαία εικόνα χρησιμοποιείται για το γέμισμα του αντικειμένου. |
| Group | 5 | Το οπτικό αντικείμενο κληρονομεί τις ιδιότητες γεμίσματος από την ομάδα. |

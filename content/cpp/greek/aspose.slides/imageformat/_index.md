## ImageFormat enum

Αναπαριστά τη μορφή αρχείου της εικόνας.

```cpp
enum class ImageFormat
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| MemoryBmp | 0 | Αναπαριστά τη μορφή ενός bitmap στη μνήμη. |
| Bmp | 1 | Αναπαριστά τη μορφή εικόνας bitmap (BMP). |
| Emf | 2 | Αναπαριστά τη μορφή εικόνας βελτιωμένου μετααρχείου (EMF). |
| Wmf | 3 | Αναπαριστά τη μορφή εικόνας Windows metafile (WMF). |
| Gif | 4 | Αναπαριστά τη μορφή εικόνας Graphics Interchange Format (GIF). |
| Jpeg | 5 | Αναπαριστά τη μορφή εικόνας Joint Photographic Experts Group (JPEG). |
| Png | 6 | Αναπαριστά τη μορφή εικόνας W3C Portable Network Graphics (PNG). |
| Tiff | 7 | Αναπαριστά τη μορφή εικόνας Tagged Image File Format (TIFF). |
| Exif | 8 | Αναπαριστά τη μορφή Exchangeable Image File (Exif). |
| Icon | 9 | Αναπαριστά τη μορφή εικόνας εικονιδίου Windows. |

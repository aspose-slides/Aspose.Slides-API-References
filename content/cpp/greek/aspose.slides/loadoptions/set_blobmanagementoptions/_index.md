## LoadOptions::set_BlobManagementOptions(System::SharedPtr\<IBlobManagementOptions\>) μέθοδος

Αντιπροσωπεύει τις επιλογές που μπορούν να χρησιμοποιηθούν για τη διαχείριση της συμπεριφοράς χειρισμού Binary Large Objects (BLOBs), όπως η χρήση προσωρινών αρχείων ή το μέγιστο αριθμό byte BLOBs στη μνήμη. Αυτές οι επιλογές προορίζονται να θέσουν το καλύτερο λόγο απόδοσης/κατανάλωσης μνήμης για ένα συγκεκριμένο περιβάλλον ή απαιτήσεις.

```cpp
void Aspose::Slides::LoadOptions::set_BlobManagementOptions(System::SharedPtr<IBlobManagementOptions> value) override
```

## Παρατηρήσεις

Ένα Binary Large Object (BLOB) είναι ένα δυαδικό δεδομένο που αποθηκεύεται ως μία ενιαία οντότητα - δηλαδή το BLOB μπορεί να είναι ίδιο το ηχητικό, βίντεο ή παρουσίαση.

## Details_ReflectionTypeLoadException κλάση


ReflectionTypeLoadException ῥίχνεται από τη μέθοδο Module.GetTypes εάν οποιαδήποτε από τις κλάσεις σε μια μονάδα αποτύχει να φορτωθεί. Ποτέ μην δημιουργείτε στιγμιότυπα αυτής της κλάσης μη αυτόματα. Χρησιμοποιήστε την κλάση ReflectionTypeLoadException αντί αυτού. Ποτέ μην ενσωματώνετε στιγμιότυπα της κλάσης ReflectionTypeLoadException στο [System::SmartPtr](../../system/smartptr/).

```cpp
class Details_ReflectionTypeLoadException : public System::Details_Exception
```

## BindingFlags enum

Ορίζει μέλη και τρόπους αναζήτησης τύπων και δεσμεύσεων.

```cpp
enum class BindingFlags
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Default | 0 | Καμία ειδική επιλογή. |
| IgnoreCase | 1 | Αγνοεί τη διαφορά πεζών-κεφαλαίων του ονόματος κατά την αναζήτηση του στοιχείου. |
| DeclaredOnly | 2 | Αναζητά μόνο τα μέλη που δηλώνονται στον τύπο και όχι στους βασικούς τύπους. |
| Instance | 4 | Αναζητά στα μέλη της εμφάνισης. |
| Static | 8 | Αναζητά στα στατικά μέλη. |
| Public | 16 | Αναζητά στα δημόσια μέλη. |
| NonPublic | 32 | Αναζητά στα μη δημόσια μέλη. |
| FlattenHierarchy | 64 | Αναζητά στα δημόσια και προστατευμένα στατικά μέλη του βασικού τύπου. |
| InvokeMethod | 256 | Καλεί τη μέθοδο. |
| CreateInstance | 512 | Δημιουργεί μια παρουσία του αντανακλώμενου τύπου. |
| GetField | 1024 | Λαμβάνει την τιμή του πεδίου. |
| SetField | 2048 | Ορίζει την τιμή του πεδίου. |
| GetProperty | 4096 | Λαμβάνει την τιμή της ιδιότητας. |
| SetProperty | 8192 | Ορίζει την τιμή της ιδιότητας. |
| PutDispProperty | 16384 | Ορίζει ιδιότητα COM. |
| PutRefDispProperty | 32768 | Ορίζει ιδιότητα αναφοράς COM. |
| ExactBinding | 65536 | Η δέσμευση τύπου πρέπει να είναι ακριβής, χωρίς αλλαγές τύπου. |
| SuppressChangeType | 131072 | Δεν υποστηρίζεται. |
| OptionalParamBinding | 262144 | Επιλέγει την υπερφόρτωση με βάση τον αριθμό των ορισμάτων. |
| IgnoreReturn | 16777216 | Αγνοεί την τιμή επιστροφής του COM interop. |

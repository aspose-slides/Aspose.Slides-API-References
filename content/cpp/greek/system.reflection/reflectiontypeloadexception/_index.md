## ReflectionTypeLoadException typedef

ReflectionTypeLoadException εγείρεται από τη μέθοδο Module.GetTypes εάν κάποια από τις κλάσεις σε ένα module αποτύχει να φορτωθεί. Ποτέ μην ενσωματώνετε τα αντικείμενα κλάσης ReflectionTypeLoadException μέσα σε [System::SmartPtr](../../system/smartptr/).

```cpp
using System::Reflection::ReflectionTypeLoadException = typedef System::ExceptionWrapper<Details_ReflectionTypeLoadException>
```

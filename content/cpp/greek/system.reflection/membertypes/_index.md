## MemberTypes enum


Καθορίζει κάθε τύπο μέλους.

```cpp
enum class MemberTypes
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Constructor | 1 | Καθορίζει ότι το μέλος είναι ένας κατασκευαστής. |
| Event | 2 | Καθορίζει ότι το μέλος είναι ένα συμβάν. |
| Field | 4 | Καθορίζει ότι το μέλος είναι ένα πεδίο. |
| Method | 8 | Καθορίζει ότι το μέλος είναι μια μέθοδος. |
| Property | 16 | Καθορίζει ότι το μέλος είναι μια ιδιότητα. |
| TypeInfo | 32 | Καθορίζει ότι το μέλος είναι ένας τύπος. |
| Custom | 64 | Καθορίζει ότι το μέλος είναι ένας προσαρμοσμένος τύπος μέλους. |
| NestedType | 128 | Καθορίζει ότι το μέλος είναι ένας ένθετος τύπος. |
| All | 191 | Καθορίζει όλους τους τύπους μελών. |

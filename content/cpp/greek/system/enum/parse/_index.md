## Enum::Parse(const String\&, bool) μέθοδος


Μετατρέπει τη δοσμένη συμβολοσειρά σε ισοδύναμη σταθερά enum.

```cpp
static E System::Enum<E, Guard>::Parse(const String &str, bool ignoreCase=false)
```


### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| str | const [String](../../string/)\& | [String](../../string/) που ερμηνεύεται ως περιέχουσα το όνομα της σταθεράς enum |
| ignoreCase | **bool** | Καθορίζει εάν πρέπει να αγνοηθεί η διάκριση πεζών-κεφαλαίων κατά την ερμηνεία της συμβολοσειράς |

### Τιμή επιστροφής

Η σταθερά της απαρίθμησης του οποίου το όνομα καθορίζεται στο **str**

## Enum::HasFlag(E, E) μέθοδος

Καθορίζει εάν τα καθορισμένα δυαδικά ψηφία είναι ορισμένα σε μια δυαδική αναπαράσταση της καθορισμένης τιμής enum.

```cpp
static bool System::Enum<E, Guard>::HasFlag(E value, E mask)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| value | E | Η τιμή enum προς δοκιμή |
| mask | E | Η μάσκα για έλεγχο των bits της value |

### Τιμή επιστροφής

Αληθές εάν τα bits που είναι ορισμένα στο **mask** είναι επίσης ορισμένα στο **value**, διαφορετικά - ψευδές

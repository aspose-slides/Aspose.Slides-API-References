## Details_UriFormatException κλάση

Το UriFormatException προκαλείται όταν η μορφή του URI δεν είναι έγκυρη. Ποτέ μην δημιουργείτε αντικείμενα αυτής της κλάσης χειροκίνητα. Χρησιμοποιήστε την κλάση UriFormatException. Ποτέ μην ενθυλακώνετε τις στιγμές της κλάσης UriFormatException στο [System::SmartPtr](../smartptr/).

```cpp
class Details_UriFormatException : public System::Details_FormatException
```

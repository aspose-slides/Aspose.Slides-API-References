## Version κλάση

Αντιπροσωπεύει έναν αριθμό έκδοσης. Αυτός ο τύπος πρέπει να δεσμεύεται στη στοίβα και να περνάται στις συναρτήσεις κατά τιμή ή κατά αναφορά. Ποτέ μην χρησιμοποιείτε την κλάση [System::SmartPtr](../smartptr/) για τη διαχείριση αντικειμένων αυτού του τύπου.

```cpp
class Version
```

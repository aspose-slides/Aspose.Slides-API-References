## Version::Version(int, int, int, int) κατασκευαστής

Δημιουργεί μια παρουσία που αντιπροσωπεύει τις καθορισμένες τιμές κύριας, δευτερεύουσας, κατασκευής και αναθεώρησης.

```cpp
System::Version::Version(int major, int minor, int build, int revision)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| major | int | Ο αριθμός κύριας έκδοσης |
| minor | int | Ο αριθμός δευτερεύουσας έκδοσης |
| build | int | Ο αριθμός κατασκευής |
| revision | int | Ο αριθμός αναθεώρησης |

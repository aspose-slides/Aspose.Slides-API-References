```cpp
static Version System::Version::Parse(const String &input)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |

| input | const [String](../../string/)\& | Μια συμβολοσειρά που περιέχει την αναπαράσταση συμβολοσειράς ενός αριθμού έκδοσης |

### Τιμή Επιστροφής

Ένα αντικείμενο της κλάσης [Version](../) που αντιπροσωπεύει τον αριθμό έκδοσης ισοδύναμο με αυτόν που περιγράφεται από το **input**

## Δομή IsWeakPtr

Κλάση χαρακτηριστικών για να ελέγχει αν μια συγκεκριμένη κλάση είναι ειδική έκδοση του [System::WeakPtr](../weakptr/). Δεν ελέγχει εάν το στιγμιότυπο βρίσκεται στην αδύναμη κατάσταση.

```cpp
template<class T>class IsWeakPtr : public System::detail::is_a<T, System::WeakPtr>
```

### Παράμετροι προτύπου

| Parameter | Description |
| --- | --- |
| T | Τύπος που ελέγχεται. |

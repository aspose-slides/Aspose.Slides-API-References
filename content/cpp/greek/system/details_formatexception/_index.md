## Details_FormatException κλάση


Η FormatException εκτοξεύεται όταν η μορφή του ορίσματος της μεθόδου δεν είναι έγκυρη. Ποτέ μην δημιουργείτε στιγμές αυτού του κλάσης χειροκίνητα. Χρησιμοποιήστε την κλάση FormatException αντί αυτού. Ποτέ μην τυλίγετε τις στιγμές της κλάσης FormatException σε [System::SmartPtr](../smartptr/).

```cpp
class Details_FormatException : public System::Details_SystemException
```

## System::Set(ExpressionT\&, const ExpressionT\&) συνάρτηση

Εφαρμόζει τη μετάφραση του προτύπου 'var'.

```cpp
template<class ExpressionT> bool System::Set(ExpressionT &var, const ExpressionT &value)
```

### Παράμετροι προτύπου

| Parameter | Description |
| --- | --- |
| ExpressionT | τύπος μεταβλητής. |

### Παράμετροι

| Parameter | Type | Description |
| --- | --- | --- |
| var | ExpressionT\& | αναφορά στη μεταβλητή που θα αρχικοποιηθεί. |
| value | const ExpressionT\& | τιμή που θα εκχωρηθεί στη μεταβλητή. |

### Τιμή επιστροφής

πάντα true (αυτό το πρότυπο πάντα ενεργοποιείται).

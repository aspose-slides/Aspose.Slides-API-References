## IConvertible::ToSingle(System::SharedPtr\<System::IFormatProvider\>) μέθοδος

Μετατρέπει την τιμή αυτού του αντικειμένου σε έναν ισοδύναμο αριθμό κινητής υποδιαστολής μονής ακρίβειας χρησιμοποιώντας τις καθορισμένες πληροφορίες μορφοποίησης ειδικές για πολιτισμό.

```cpp
virtual float System::IConvertible::ToSingle(System::SharedPtr<System::IFormatProvider> provider)=0
```

### Παράμετροι

| Parameter | Type | Description |
| --- | --- | --- |
| provider | [System::SharedPtr](../../sharedptr/)\<[System::IFormatProvider](../../iformatprovider/)\> | Μια υλοποίηση διεπαφής [System::IFormatProvider](../../iformatprovider/) που παρέχει πληροφορίες μορφοποίησης ειδικές για πολιτισμό. |

### Τιμή επιστροφής

Ένας αριθμός κινητής υποδιαστολής μονής ακρίβειας ισοδύναμος με την τιμή αυτού του αντικειμένου.

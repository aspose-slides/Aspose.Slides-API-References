## String::IsEmpty() const μέθοδος

Ελέγχει εάν η συμβολοσειρά είναι τόσο μη μηδενική όσο και κενή.

```cpp
bool System::String::IsEmpty() const
```

### Τιμή Επιστροφής

true εάν η συμβολοσειρά είναι κενή και όχι μηδενική, false διαφορετικά.

## String::begin() const μέθοδος

Επιστρέφει δείκτη στην αρχή του πραγματικού buffer συμβολοσειράς. Ποτέ δεν κάνει επανεκχώρηση σε κάτι. Δεν εγγυάται ότι το buffer είναι τερματισμένο με μηδενικό χαρακτήρα.

```cpp
const UChar * System::String::begin() const
```

### Τιμή Επιστροφής

Δείκτης στο πραγματικό buffer συμβολοσειράς, τερματισμένο με μηδενικό ή όχι.

## String::Is(const System::TypeInfo\&) const μέθοδος

Ελέγχει αν το αντικείμενο string είναι του τύπου που καθορίζεται από [TypeInfo](../../typeinfo/) που περνάται.

```cpp
bool System::String::Is(const System::TypeInfo &target) const
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) για έλεγχο. |

### Τιμή Επιστροφής

true αν target είναι [TypeInfo](../../typeinfo/) της κλάσης [String](../), false διαφορετικά.

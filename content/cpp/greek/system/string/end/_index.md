## String::end() const μέθοδος


Επιστρέφει δείκτη προς το τέλος του τρέχοντος buffer της συμβολοσειράς. Ποτέ δεν επανεκχωρεί τίποτα. Δεν εγγυάται ότι το buffer είναι null-terminated.

```cpp
const UChar * System::String::end() const
```


### Τιμή Επιστροφής

Δείκτης στο πρώτο στοιχείο μετά το τρέχον buffer της συμβολοσειράς, null-terminated ή όχι.

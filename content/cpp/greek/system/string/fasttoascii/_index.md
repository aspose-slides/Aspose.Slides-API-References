## String::FastToAscii(char, int) const μέθοδος

Προσπαθεί να μετατρέψει ένα [String](../) σε συμβολοσειρά ASCII.

```cpp
int System::String::FastToAscii(char buffer[], int buffer_size) const
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| buffer | char | Ένα buffer για αποθήκευση της προκύπτουσας συμβολοσειράς. |
| buffer_size | int | [Buffer](../../buffer/)'s μέγεθος. |

### Τιμή επιστροφής

Το μήκος της προκύπτουσας συμβολοσειράς ή -1 αν ένα [String](../) δεν είναι συμβολοσειρά ASCII.

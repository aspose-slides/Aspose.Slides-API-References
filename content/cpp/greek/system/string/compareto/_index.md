## String::CompareTo(const String\&) const μέθοδος

Συγκρίνει δύο συμβολοσειρές σε στυλ 'μικρότερο-ίσο-μεγαλύτερο'. Χρησιμοποιεί την τρέχουσα πολιτισμική ρύθμιση.

```cpp
int System::String::CompareTo(const String &str) const
```

### Ορίσματα

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| str | const [String](../)\& | [String](../) για σύγκριση με αυτό. |

### Τιμή Επιστροφής

Αρνητική τιμή εάν αυτό θεωρείται μικρότερο από το str, μηδέν εάν ταιριάζουν, θετική τιμή εάν αυτό είναι μεγαλύτερο από το str.

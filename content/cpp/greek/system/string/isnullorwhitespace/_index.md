## String::IsNullOrWhiteSpace(const String\&) μέθοδος

Δείχνει αν μια συγκεκριμένη συμβολοσειρά είναι null, κενή ή αποτελείται μόνο από λευκούς χαρακτήρες.

```cpp
static bool System::String::IsNullOrWhiteSpace(const String &str)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| str | const [String](../)\& | [String](../) για έλεγχο. |

### Τιμή Επιστροφής

true αν η συμβολοσειρά είναι null ή κενή ή αποτελείται αποκλειστικά από λευκούς χαρακτήρες, false διαφορετικά.

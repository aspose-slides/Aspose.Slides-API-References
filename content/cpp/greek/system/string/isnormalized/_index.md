## String::IsNormalized(System::Text::NormalizationForm) const μέθοδος

Ελέγχει εάν η συμβολοσειρά Unicode είναι κανονικοποιημένη χρησιμοποιώντας τη μορφή κανονικοποίησης που έχει καθοριστεί.

```cpp
bool System::String::IsNormalized(System::Text::NormalizationForm form) const
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| form | [System::Text::NormalizationForm](../../../system.text/normalizationform/) | Μορφή κανονικοποίησης, δείτε [System::Text::NormalizationForm](../../../system.text/normalizationform/) για λεπτομέρειες. |

### Τιμή επιστροφής

true εάν η συμβολοσειρά είναι ήδη κανονικοποιημένη, false αλλιώς.

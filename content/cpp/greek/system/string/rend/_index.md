## String::rend() const μέθοδος


Επιστρέφει αντίστροφο iterator στο πριν από τον πρώτο χαρακτήρα (εάν υπάρχει) του τρέχοντος buffer συμβολοσειράς.

```cpp
reverse_iterator System::String::rend() const
```


### Τιμή Επιστροφής

Αντίστροφο iterator στο στοιχείο που προηγείται του πρώτου χαρακτήρα του τρέχοντος buffer συμβολοσειράς (εάν υπάρχει) ή στο στοιχείο που βρίσκεται ακριβώς πριν από το κατανεμημένο buffer.

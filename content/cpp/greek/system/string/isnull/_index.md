## String::IsNull() const μέθοδος


Ελέγχει εάν η συμβολοσειρά θεωρείται null. [String](../) είναι null και μόνο εάν έχει δημιουργηθεί μέσω του κατασκευαστή [String()](../string/), μετακινήθηκε, αντιγράφηκε ή εκχωρήθηκε από null συμβολοσειρά ή κλήθηκε η μέθοδος [reset()](../reset/).

```cpp
bool System::String::IsNull() const
```


### Τιμή Επιστροφής

true αν η συμβολοσειρά θεωρείται null, false διαφορετικά.

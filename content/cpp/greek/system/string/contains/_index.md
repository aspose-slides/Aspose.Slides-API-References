## String::Contains(const String\&) const μέθοδος

Ελέγχει αν η str είναι υποσυμβολοσειρά της τρέχουσας συμβολοσειράς.

```cpp
bool System::String::Contains(const String &str) const
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| str | const [String](../)\& | Συμβολοσειρά αναζήτησης. |

### Τιμή Επιστροφής

true εάν υπάρχει η υποσυμβολοσειρά, false διαφορετικά.

## String::IndexOfAny(char_t, int) const μέθοδος

Αναζήτηση χαρακτήρα προς τα εμπρός.

```cpp
int System::String::IndexOfAny(char_t c, int startIndex=0) const
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| c | char_t | Χαρακτήρας προς αναζήτηση. |
| startIndex | int | Δείκτης από τον οποίο ξεκινά η αναζήτηση. |

### Τιμή Επιστροφής

Δείκτης της πρώτης θέσης χαρακτήρα από το startIndex ή -1 αν δεν βρεθεί.

## String::IndexOfAny(const String\&, int) const μέθοδος

Κατ' αυτόν τον τρόπο αναζητά όλους τους χαρακτήρες του str σε αυτό το αντικείμενο. Εάν βρεθεί ο πρώτος χαρακτήρας, επιστρέφεται η θέση του· διαφορετικά αναζητείται ο δεύτερος και ούτω καθεξής.

```cpp
int System::String::IndexOfAny(const String &str, int startIndex=0) const
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| str | const [String](../)\& | [String](../) των χαρακτήρων προς αναζήτηση. Η σειρά των χαρακτήρων έχει σημασία. |
| startIndex | int | Θέση από την οποία ξεκινά η αναζήτηση. |

### Τιμή Επιστροφής

Δείκτης του πρώτου βρέθηκε χαρακτήρα ή -1 αν δεν βρεθεί κανένας.

```cpp

```

| anyOf | const [ArrayPtr](../../arrayptr/)\<char_t\>\& | [Array](../../array/) των χαρακτήρων προς αναζήτηση. Η σειρά δεν έχει σημασία. |

### Τιμή Επιστροφής

Δείκτης του πρώτου ταιριαστού χαρακτήρα ή -1 αν δεν βρεθεί.

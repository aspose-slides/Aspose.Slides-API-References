## String::rbegin() const μέθοδος


Επιστρέφει αντίστροφο iterator προς τον τελευταίο χαρακτήρα (αν υπάρχει) του τρέχοντος buffer συμβολοσειράς.

```cpp
reverse_iterator System::String::rbegin() const
```


### Τιμή Επιστροφής

Αντίστροφο iterator προς τον τελευταίο χαρακτήρα του τρέχοντος buffer συμβολοσειράς (αν υπάρχει) ή ίδιο με [rend()](../rend/).

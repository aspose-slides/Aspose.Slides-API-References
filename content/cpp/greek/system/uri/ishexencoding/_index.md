```cpp
static bool System::Uri::IsHexEncoding(const String &pattern, int32_t index)
```

| Parameter | Type | Description |

| index | **int32_t** | Θέση με βάση το μηδέν του χαρακτήρα προς δοκιμή |

Αληθές εάν ο καθορισμένος χαρακτήρας είναι κωδικοποιημένος σε δεκαεξαδική μορφή, διαφορετικά - ψευδές

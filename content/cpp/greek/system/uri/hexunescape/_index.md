## Uri::HexUnescape(const String\&, int32_t\&) μέθοδος

Μετατρέπει τη συγκεκριμένη δεκαεξαδική αναπαράσταση ενός χαρακτήρα σε χαρακτήρα.

```cpp
static char16_t System::Uri::HexUnescape(const String &pattern, int32_t &index)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| pattern | const [String](../../string/)\& | Μια συμβολοσειρά που περιέχει τη δεκαεξαδική αναπαράσταση ενός χαρακτήρα |
| index | **int32_t**\& | Η θέση στο **pattern** όπου αρχίζει η δεκαεξαδική αναπαράσταση ενός χαρακτήρα |

### Τιμή Επιστροφής

Ο χαρακτήρας που αντιπροσωπεύεται από την δεκαεξαδική κωδικοποίηση στη θέση **index**. Εάν ο χαρακτήρας στη **index** δεν είναι κωδικοποιημένος δεκαεξαδικά, επιστρέφεται ο χαρακτήρας στη **index**. Η τιμή της **index** αυξάνεται ώστε να δείχνει τον χαρακτήρα που ακολουθεί αυτόν που επιστράφηκε.

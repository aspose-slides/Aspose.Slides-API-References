## Span κλάση

Αντιπροσωπεύει μια συνεχόμενη περιοχή αυθαίρετης μνήμης παρόμοια με το std::span του C++20.

```cpp
template<typename T>class Span : public System::Details::SpanCore<T, Span<T>, Span<T>>
```

### Παράμετροι προτύπου

| Παράμετρος | Περιγραφή |
| --- | --- |
| T | Ο τύπος των στοιχείων στο span. Αυτή η κλάση παρέχει έναν τύπο-ασφαλή τρόπο εργασίας με συνεχόμενες ακολουθίες αντικειμένων. Μπορεί να χρησιμοποιηθεί για περικάλυψη πινάκων, πινάκων στο stack ή ακατέργαστων δεικτών ενώ διατηρεί έλεγχο ορίων. Το [Span](./) δεν κατέχει τη μνήμη στην οποία δείχνει - είναι απλώς μια προβολή στην υπάρχουσα μνήμη. |

## Μέθοδοι

| Μέθοδος | Περιγραφή |
| --- | --- |
| void [Clear](./clear/)() const | Καθαρίζει το περιεχόμενο του span ορίζοντας όλα τα στοιχεία στην προεπιλεγμένη τιμή. |
| void [Fill](./fill/)(const T\&) const | Γεμίζει το span με την καθορισμένη τιμή. |
| static [ThisType](./) [to_Span](./to_span/)(const typename BaseType::ArrayPtrT\&) | Μετατρέπει έναν πίνακα σε ένα [Span](./). |

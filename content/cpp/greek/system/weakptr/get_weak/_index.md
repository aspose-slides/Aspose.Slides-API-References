## WeakPtr::get_weak() const μέθοδος

Αποκτά το αντικείμενο που αναφέρεται. Επιβεβαιώνει ότι ο δείκτης είναι σε αδυνατική λειτουργία.

```cpp
Object * System::WeakPtr<T>::get_weak() const
```

### Τιμή Επιστροφής

Ακατέργαστος δείκτης στο αντικείμενο που αναφέρεται (εάν έχει οριστεί και δεν έχει λήξει) ή nullptr.

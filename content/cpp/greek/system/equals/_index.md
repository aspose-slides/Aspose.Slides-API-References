## System::Equals(const TA\&, const TB\&) συνάρτηση

Καθορίζει την ισότητα δύο τιμών εφαρμόζοντας [operator==()](../operator_equal_equal/) σε αυτές.

```cpp
template<typename TA,typename TB> bool System::Equals(const TA &a, const TB &b)
```

### Παράμετροι προτύπου

| Παράμετρος | Περιγραφή |
| --- | --- |
| TA | Ο τύπος του πρώτου συγκριζόμενου |
| TB | Ο τύπος του δεύτερου συγκριζόμενου |

### Ορίσματα

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| a | const TA\& | Το πρώτο συγκριζόμενο |
| b | const TB\& | Το δεύτερο συγκριζόμενο |

### Τιμή επιστροφής

Η λογική τιμή που επιστρέφεται από το [operator==()](../operator_equal_equal/) που εφαρμόζεται στο **a** και **b**

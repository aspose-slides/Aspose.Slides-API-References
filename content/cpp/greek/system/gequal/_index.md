## System::GEqual(const ExpressionT\&, const ConstantT\&) συνάρτηση

Εφαρμόζει τη μετάφραση του σχετικού προτύπου '>='.

```cpp
template<class ExpressionT,class ConstantT> bool System::GEqual(const ExpressionT &left, const ConstantT &constant)
```

### Παράμετροι προτύπου

| Παράμετρος | Περιγραφή |
| --- | --- |
| ExpressionT | Τύπος αριστερής έκφρασης. |
| ConstantT | Τύπος σταθερής έκφρασης. |

### Ορίσματα

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| left | const ExpressionT\& | έκφραση που θα ελεγχθεί. |
| constant | const ConstantT\& | έκφραση που θα συγκριθεί με την αριστερή. |

### Τιμή Επιστροφής

true εάν ο έλεγχος τύπου είναι επιτυχής, false διαφορετικά.

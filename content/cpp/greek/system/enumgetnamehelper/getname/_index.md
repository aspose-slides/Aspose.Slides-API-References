## EnumGetNameHelper::GetName(T) μέθοδος

Επιστρέφει το όνομα σε μορφή συμβολοσειράς της συγκεκριμένης τιμής σταθεράς enum.

```cpp
template<class T> static String System::EnumGetNameHelper<E, G, Guard>::GetName(T value)
```

### Παράμετροι προτύπου

| Parameter | Description |
| --- | --- |
| T | Ο τύπος των τιμών enum που γίνονται αποδεκτές από τη μέθοδο· **T** πρέπει να είναι μετατρεπτέος σε **E** |

### Ορίσματα

| Parameter | Type | Description |
| --- | --- | --- |
| value | T | Η τιμή σταθεράς enum της οποίας το όνομα σε συμβολοσειρά πρέπει να ληφθεί |

### Τιμή επιστροφής

Το όνομα σε μορφή συμβολοσειράς της συγκεκριμένης τιμής σταθεράς enum

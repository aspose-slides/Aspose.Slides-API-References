## UriParser κλάση

Χρησιμοποιείται για την ανάλυση ενός νέου σχήματος URI. Τα αντικείμενα αυτής της κλάσης θα πρέπει να αντιστοιχίζονται μόνο χρησιμοποιώντας τη λειτουργία [System::MakeObject()](../makeobject/). Ποτέ μην δημιουργείτε παρουσία αυτού του τύπου στη στοίβα ή χρησιμοποιώντας operator new, καθώς αυτό θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης και/ή σφάλματα δήλωσης. Πάντα τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να τη περάσετε σε συναρτήσεις ως επιχείρημα.

```cpp
class UriParser
```

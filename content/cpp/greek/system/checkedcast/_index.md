## System::CheckedCast(TFrom) συνάρτηση

Καθορίζει εάν η καθορισμένη τιμή βρίσκεται εντός του εύρους τιμών του τύπου **TTo** και εάν ναι την μετατρέπει στον τύπο **TTo**.

```cpp
template<typename TTo,typename TFrom> TTo System::CheckedCast(TFrom value)
```

### Παράμετροι προτύπου

| Parameter | Description |
| --- | --- |
| TTo | Ο τύπος στον οποίο θα μετατραπεί η καθορισμένη τιμή |
| TFrom | Ο τύπος της καθορισμένης τιμής |

### Ορίσματα

| Parameter | Type | Description |
| --- | --- | --- |
| value | TFrom | Η τιμή προς μετατροπή |

### Τιμή επιστροφής

Η τιμή του τύπου **TTo** ισοδύναμη με **value**

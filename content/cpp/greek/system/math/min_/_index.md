## Math::Min_(float, float) μέθοδος


Επιστρέφει τη μικρότερη τιμή μονής ακρίβειας κινητής υποδιαστολής από τις δύο που ορίστηκαν.

```cpp
float System::Math::Min_(float val1, float val2)
```


### Παράμετροι

| Parameter | Type | Description |
| --- | --- | --- |
| val1 | **float** | Η πρώτη συγκριόμενη τιμή |
| val2 | **float** | Η δεύτερη συγκριόμενη τιμή |

### Τιμή επιστροφής

Η μικρότερη τιμή από το **val1** και το **val2**; ή NaN εάν ένα από τα τελεστές είναι NaN

## Παρατηρήσεις



```cpp
#include "system/math.h"
#include <iostream>

int main()
{
  using namespace System;

  // Εκτυπώνει τις απόλυτες τιμές.
  for (int i = -1; i < 2; ++i)
  {
    std::cout << Math::Abs(i) << " ";
  }
  std::cout << std::endl;

  // Εκτυπώνει το ημίτονο του PI/2 και το συνημίτονο του PI.
  std::cout << "sin(PI/2)=" << Math::Sin(Math::PI/2) << "; cos(PI)=" << Math::Cos(Math::PI) << std::endl;

  return 0;
}
/*
Αυτό το παράδειγμα κώδικα παράγει την παρακάτω έξοδο:
1 0 1
sin(PI/2)=1; cos(PI)=-1
*/
```

## DBNull κλάση

Represents a non-existing value. Objects of this κλάση should only be allocated using [System::MakeObject()](../makeobject/) συνάρτηση. Never create instance of this type on stack or using operator new, as it will result in runtime errors and/or assertion faults. Always wrap this κλάση into [System::SmartPtr](../smartptr/) pointer and use this pointer to pass it to functions as argument.

```cpp
class DBNull : public System::Object
```

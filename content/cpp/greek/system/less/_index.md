## System::Less(const ExpressionT\&, const ConstantT\&) συνάρτηση

Υλοποιεί τη σχετική μετάφραση του προτύπου '<'.

```cpp
template<class ExpressionT,class ConstantT> bool System::Less(const ExpressionT &left, const ConstantT &constant)
```

### Παράμετροι προτύπου

| Parameter | Description |
| --- | --- |
| ExpressionT | τύπος αριστερού έκφρασης. |
| ConstantT | τύπος σταθερής έκφρασης. |

### Arguments

| Parameter | Type | Description |
| --- | --- | --- |
| left | const ExpressionT\& | έκφραση που θα ελεγχθεί. |
| constant | const ConstantT\& | έκφραση που θα συγκριθεί με την αριστερή. |

### Τιμή Επιστροφής

true αν ο έλεγχος τύπου είναι επιτυχής, false διαφορετικά.

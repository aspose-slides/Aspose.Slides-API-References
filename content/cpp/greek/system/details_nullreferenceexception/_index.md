## Details_NullReferenceException κλάση

NullReferenceException ρίχνεται όταν γίνεται προσπάθεια αποαναφοράς ενός μηδενικού αναφοράς. Ποτέ μην δημιουργείτε στιγμιότυπα αυτής της κλάσης μη αυτόματα. Χρησιμοποιήστε την κλάση NullReferenceException αντ' αυτού. Ποτέ μην ενσωματώνετε τις στιγμιότυπα της κλάσης NullReferenceException σε [System::SmartPtr](../smartptr/).

```cpp
class Details_NullReferenceException : public System::Details_SystemException
```

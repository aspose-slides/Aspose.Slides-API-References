### Arguments

| Parameter | Type | Description |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) δομή που περιγράφει τον τύπο ενάντια στον οποίο θα ελεγχθεί το τρέχον αντικείμενο. |

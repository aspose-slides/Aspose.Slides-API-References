## System::Greater(const ExpressionT\&, const ConstantT\&) συνάρτηση

Εφαρμόζει τη μετάφραση σχετικού προτύπου '>'.

```cpp
template<class ExpressionT,class ConstantT> bool System::Greater(const ExpressionT &left, const ConstantT &constant)
```

### Παράμετροι προτύπου

| Παράμετρος | Περιγραφή |
| --- | --- |
| ExpressionT | τύπος της αριστερής έκφρασης. |
| ConstantT | τύπος της σταθερής έκφρασης. |

### Ορίσματα

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| left | const ExpressionT\& | έκφραση που θα ελεγχθεί. |
| constant | const ConstantT\& | έκφραση που θα συγκριθεί με την αριστερή. |

### Τιμή επιστροφής

true αν ο έλεγχος τύπου είναι επιτυχής, false διαφορετικά.

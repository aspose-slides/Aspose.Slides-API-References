## TimeSpan::CompareTo(TimeSpan) const μέθοδος

Συγκρίνει το τρέχον και τα καθορισμένα αντικείμενα.

```cpp
constexpr int System::TimeSpan::CompareTo(TimeSpan value) const
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| value | [TimeSpan](../) | The [TimeSpan](../) object to compare the current object with |

### Τιμή Επιστροφής

- 1 αν το τρέχον αντικείμενο αντιπροσωπεύει το διάστημα που είναι πιο σύντομο από **value**; 0 αν το τρέχον αντικείμενο αντιπροσωπεύει το διάστημα που είναι ίσο με **value**; 1 αν το τρέχον αντικείμενο αντιπροσωπεύει το διάστημα που είναι μεγαλύτερο από **value**

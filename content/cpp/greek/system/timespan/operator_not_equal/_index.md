## TimeSpan::operator!=(TimeSpan) const μέθοδος

Καθορίζει αν η χρονική περίοδος που αντιπροσωπεύεται από το τρέχων αντικείμενο δεν είναι ίση με τη χρονική περίοδο που αντιπροσωπεύεται από το καθορισμένο αντικείμενο.

```cpp
constexpr bool System::TimeSpan::operator!=(TimeSpan value) const
```

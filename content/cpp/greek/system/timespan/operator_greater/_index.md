## TimeSpan::operator>(TimeSpan) const μέθοδος

Καθορίζει εάν το χρονοδιάστημα που αντιπροσωπεύεται από το τρέχον αντικείμενο είναι μεγαλύτερο από το χρονοδιάστημα που αντιπροσωπεύεται από το συγκεκριμένο αντικείμενο.

```cpp
constexpr bool System::TimeSpan::operator>(TimeSpan value) const
```

### Παράμετροι

| Parameter | Type | Description |
| --- | --- | --- |
| value | [TimeSpan](../) | Το αντικείμενο [TimeSpan](../) για σύγκριση με το τρέχον αντικείμενο |

### Τιμή επιστροφής

True εάν το χρονοδιάστημα που αντιπροσωπεύεται από το τρέχον αντικείμενο είναι μεγαλύτερο από το χρονοδιάστημα που αντιπροσωπεύεται από το **value**, αλλιώς - false

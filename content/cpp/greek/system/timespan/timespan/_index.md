## TimeSpan::TimeSpan() κατασκευαστής

Δημιουργεί ένα αντικείμενο [TimeSpan](../) που αντιπροσωπεύει ένα μηδενικό διάστημα χρόνου.

```cpp
constexpr System::TimeSpan::TimeSpan()
```

## TimeSpan::TimeSpan(int64_t) κατασκευαστής

Δημιουργεί μια παρουσία της κλάσης [TimeSpan](../) που αντιπροσωπεύει το συγκεκριμένο διάστημα χρόνου.

```cpp
constexpr System::TimeSpan::TimeSpan(int64_t ticks)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| ticks | **int64_t** | Το διάστημα χρόνου που θα αντιπροσωπευτεί από το παράδειγμα που κατασκευάζεται, εκφρασμένο ως ο αριθμός των 100-νανοδευτερόλεπτων διαστημάτων. |

## TimeSpan::TimeSpan(int, int, int) κατασκευαστής

Δημιουργεί μια παρουσία της κλάσης [TimeSpan](../) που αντιπροσωπεύει το διάστημα χρόνου που ισούται με το άθροισμα του συγκεκριμένου αριθμού ωρών, λεπτών και δευτερολέπτων.

```cpp
System::TimeSpan::TimeSpan(int hours, int minutes, int seconds)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| hours | int | Ο αριθμός των ωρών στο στοιχείο ωρών του διαστήματος χρόνου που θα αντιπροσωπευτεί από το παράδειγμα που κατασκευάζεται |
| minutes | int | Ο αριθμός των λεπτών στο στοιχείο λεπτών του διαστήματος χρόνου που θα αντιπροσωπευτεί από το παράδειγμα που κατασκευάζεται |
| seconds | int | Ο αριθμός των δευτερολέπτων στο στοιχείο δευτερολέπτων του διαστήματος χρόνου που θα αντιπροσωπευτεί από το παράδειγμα που κατασκευάζεται |

## TimeSpan::TimeSpan(int, int, int, int, int) κατασκευαστής

Δημιουργεί μια παρουσία της κλάσης [TimeSpan](../) που αντιπροσωπεύει το διάστημα χρόνου που ισούται με το άθροισμα του συγκεκριμένου αριθμού ωρών, λεπτών, δευτερολέπτων και χιλιοστών του δευτερολέπτου.

```cpp
System::TimeSpan::TimeSpan(int days, int hours, int minutes, int seconds, int milliseconds=0)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| days | int | Ο αριθμός των ημερών στο στοιχείο ημερών του διαστήματος χρόνου που θα αντιπροσωπευτεί από το παράδειγμα που κατασκευάζεται |
| hours | int | Ο αριθμός των ωρών στο στοιχείο ωρών του διαστήματος χρόνου που θα αντιπροσωπευτεί από το παράδειγμα που κατασκευάζεται |
| minutes | int | Ο αριθμός των λεπτών στο στοιχείο λεπτών του διαστήματος χρόνου που θα αντιπροσωπευτεί από το παράδειγμα που κατασκευάζεται |
| seconds | int | Ο αριθμός των δευτερολέπτων στο στοιχείο δευτερολέπτων του διαστήματος χρόνου που θα αντιπροσωπευτεί από το παράδειγμα που κατασκευάζεται |
| milliseconds | int | Ο αριθμός των χιλιοστών του δευτερολέπτου στο στοιχείο χιλιοστών του δευτερολέπτου του διαστήματος χρόνου που θα αντιπροσωπευτεί από το παράδειγμα που κατασκευάζεται |

## TimeSpan::TimeSpan(const TimeSpan\&) κατασκευαστής

Δημιουργεί ένα αντικείμενο [TimeSpan](../) που αντιπροσωπεύει το διάστημα χρόνου ίσο με το διάστημα χρόνου που αντιπροσωπεύεται από το συγκεκριμένο αντικείμενο [TimeSpan](../).

```cpp
constexpr System::TimeSpan::TimeSpan(const TimeSpan &)=default
```

## Details_MethodAccessException κλάση


MemberAccessException εκτοξεύεται όταν προσπαθείται πρόσβαση σε μη-υπάρχουσα μέθοδο ή όταν η πρόσβαση στην μέθοδο δεν επιτρέπεται. Ποτέ μην δημιουργείτε στιγμιότυπα αυτής της κλάσης χειροκίνητα. Χρησιμοποιήστε την κλάση MethodAccessException αντʼ αυτού. Ποτέ μην τυλίγετε τις στιγμές της κλάσης MethodAccessException σε [System::SmartPtr](../smartptr/).

```cpp
class Details_MethodAccessException : public System::Details_MemberAccessException
```

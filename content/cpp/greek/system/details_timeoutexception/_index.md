## Details_TimeoutException κλάση

TimeoutException υποδεικνύει εάν ο χρόνος που έχει καθοριστεί για μια διαδικασία ή λειτουργία έχει λήξει. Ποτέ μην δημιουργείτε εκδόσεις αυτής της κλάσης χειροκίνητα. Χρησιμοποιήστε την κλάση TimeoutException αντί αυτού. Ποτέ μην ενθυλακώνετε τις εκδόσεις της κλάσης TimeoutException στο [System::SmartPtr](../smartptr/).

```cpp
class Details_TimeoutException : public System::Details_SystemException
```

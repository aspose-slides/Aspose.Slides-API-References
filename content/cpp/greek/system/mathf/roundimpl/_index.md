## MathF::RoundImpl(float, int, MidpointRounding) method

Στρογγυλοποιεί την καθορισμένη τιμή στον πλησιέστερο αριθμό με τον καθορισμένο αριθμό δεκαδικών ψηφίων. Μια παράμετρος καθορίζει τη συμπεριφορά της συνάρτησης εάν η καθορισμένη τιμή είναι εξίσου κοντά σε δύο πλησιέστερους αριθμούς.

```cpp
static float System::MathF::RoundImpl(float value, int digits, MidpointRounding mode)
```

### Ορίσματα

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| value | **float** | Η τιμή που θα στρογγυλοποιηθεί |
| digits | int | Ο αριθμός των δεκαδικών ψηφίων στην στρογγυλοποιημένη τιμή |
| mode | [MidpointRounding](../../midpointrounding/) | Καθορίζει πώς θα γίνει η στρογγυλοποίηση εάν η **value** είναι εξίσου κοντά σε δύο πλησιέστερους αριθμούς. |

### Τιμή Επιστροφής

Ο αριθμός με τον καθορισμένο αριθμό ψηφίων που είναι πιο κοντά στην **value**

## EventHandler typedef


Αναπαριστά μια μέθοδο που αντιδρά και επεξεργάζεται ένα γεγονός. Αυτός ο τύπος πρέπει να εκχωρείται στη στοίβα και να περνιέται στις συναρτήσεις με τιμή ή με αναφορά. Ποτέ μην χρησιμοποιείτε τη κλάση [System::SmartPtr](../smartptr/) για τη διαχείριση των αντικειμένων αυτού του τύπου.

```cpp
using System::EventHandler = typedef MulticastDelegate<void(System::SharedPtr<Object>, TEventArgs)>
```

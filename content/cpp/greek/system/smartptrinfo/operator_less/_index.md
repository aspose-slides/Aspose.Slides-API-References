## SmartPtrInfo::operator<(const SmartPtrInfo\&) const μέθοδος

Με την less-συγκριση συγκρίνονται οι τιμές των δεικτών που αναφέρονται από δύο αντικείμενα πληροφορίας.

```cpp
bool System::SmartPtrInfo::operator<(const SmartPtrInfo &other) const
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| other | const [SmartPtrInfo](../)\& | Πληροφορίες για τον δείκτη προς σύγκριση. |

### Τιμή Επιστροφής

True εάν η τιμή του δείκτη που αναφέρεται από το τρέχον αντικείμενο info προηγείται της τιμής του άλλου, false διαφορετικά.

## SmartPtrInfo::operator!() const μέθοδος

Ελέγχει αν το αντικείμενο info δεν δείχνει σε δείκτη διαφορετικό του μηδενός.

```cpp
bool System::SmartPtrInfo::operator!() const
```

### Τιμή Επιστροφής

False αν το αντικείμενο info δεν δείχνει σε κανέναν δείκτη ή αν ο αναφερόμενος δείκτης είναι null, true διαφορετικά.

## Boolean::TryParse(const String&, bool&) μέθοδος

Μετατρέπει τη συγκεκριμένη συμβολοσειρά σε μια τιμή τύπου bool.

```cpp
static bool System::Boolean::TryParse(const String &value, bool &result)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| value | const [String](../../string/)\& | Η συμβολοσειρά για μετατροπή. |
| result | **bool**\& | Η αναφορά σε μια μεταβλητή τύπου bool όπου το αποτέλεσμα της μετατροπής τοποθετείται· το αποτέλεσμα είναι true αν η συγκεκριμένη συμβολοσειρά ισούται με \"True\" και false αν η συγκεκριμένη συμβολοσειρά ισούται με \"False\". |

### Τιμή Επιστροφής

True αν η συγκεκριμένη συμβολοσειρά είναι ίση είτε με \"True\" είτε με \"False\", διαφορετικά - false.

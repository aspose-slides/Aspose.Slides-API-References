## Boolean::Parse(const String\&) μέθοδος


Μετατρέπει το συγκεκριμένο κείμενο σε τιμή τύπου bool.

```cpp
static bool System::Boolean::Parse(const String &value)
```


### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| value | const [String](../../string/)\& | Το κείμενο προς μετατροπή. |

### Τιμή Επιστροφής

Η μεταβλητή bool όπου αποθηκεύεται το αποτέλεσμα της μετατροπής· το αποτέλεσμα είναι true εάν το συγκεκριμένο κείμενο ισούται με "True" και false εάν το συγκεκριμένο κείμενο ισούται με "False".

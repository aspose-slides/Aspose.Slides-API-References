## IComparable κλάση

Ορίζει μια μέθοδο που συγκρίνει δύο αντικείμενα. Τα αντικείμενα αυτής της κλάσης θα πρέπει να εκχωρούνται μόνο μέσω της συνάρτησης [System::MakeObject()](../makeobject/). Ποτέ μην δημιουργείτε αντίστοιχο αντικείμενο στο στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης και/ή σφαλματικές δηλώσεις. Πάντα τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να τη περάσετε σε συναρτήσεις ως όρισμα.

```cpp
template<typename T>class IComparable : public virtual System::Object
```

### Template parameters

| Parameter | Description |
| --- | --- |
| T | Ο τύπος των αντικειμένων με τους οποίους συγκρίνεται το τρέχον αντικείμενο |

## TypeInfo::IsDefined(const TypeInfo\&, bool) const method

ΔΕΝ ΕΧΕΙ ΥΛΟΠΟΙΗΘΕΙ. Δείχνει εάν ένα ή περισσότερα χαρακτηριστικά του καθορισμένου τύπου ή των παραγόμενων τύπων του έχουν εφαρμοστεί σε αυτό το μέλος.

```cpp
bool System::TypeInfo::IsDefined(const TypeInfo &attributeType, bool inherit) const
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| attributeType | const [TypeInfo](../)\& | Ο τύπος του προσαρμοσμένου χαρακτηριστικού που θα αναζητηθεί. Η αναζήτηση περιλαμβάνει παραγώμενους τύπους. |
| inherit | **bool** | true για να αναζητηθεί η αλυσίδα κληρονομικότητας αυτού του μέλους ώστε να βρεθούν τα χαρακτηριστικά· διαφορετικά, false. Αυτή η παράμετρος αγνοείται για ιδιότητες και συμβάντα. |

### Τιμή Επιστροφής

true εάν ένα ή περισσότερα στιγμιότυπα του attributeType ή οποιοσδήποτε από τους παραγώμενους τύπους του έχει εφαρμοστεί σε αυτό το μέλος· διαφορετικά, false.

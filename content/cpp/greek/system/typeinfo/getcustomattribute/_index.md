## TypeInfo::GetCustomAttribute(const TypeInfo\&) const μέθοδος


Αναζητά το προσαρμοσμένο χαρακτηριστικό που έχει εφαρμοστεί με τον καθορισμένο τύπο και εφαρμόζεται στον τύπο που αντιπροσωπεύεται από το τρέχον αντικείμενο.

```cpp
ObjectPtr System::TypeInfo::GetCustomAttribute(const TypeInfo &attributeType) const
```


### Παράμετροι

| Parameter | Type | Description |
| --- | --- | --- |
| attributeType | const [TypeInfo](../)\& | Η σταθερή αναφορά στο αντικείμενο [TypeInfo](../) που αντιπροσωπεύει τον τύπο του χαρακτηριστικού προς αναζήτηση |

### Τιμή επιστροφής

Ένας δείκτης σε αντικείμενο που αντιπροσωπεύει το ευρέθη χαρακτηριστικό, ή null-pointer εάν δεν βρέθηκε χαρακτηριστικό που να ταιριάζει με τα κριτήρια αναζήτησης

## Array::Resize(ArrayPtr\<Type\>\&, int) μέθοδος


Αλλάζει το μέγεθος του καθορισμένου πίνακα στην καθορισμένη τιμή ή δημιουργεί νέο πίνακα με το συγκεκριμένο μέγεθος.

```cpp
template<typename Type> static void System::Array<T>::Resize(ArrayPtr<Type> &arr, int new_size)
```


### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| arr | [ArrayPtr](../../arrayptr/)\<[Type](../../object/type/)\>\& | [Array](../) για αλλαγή μεγέθους. Εάν **arr** είναι null-pointer, ο νέος πίνακας θα δημιουργηθεί |
| new_size | int | Το νέο μέγεθος του πίνακα, ή το μέγεθος του νέου πίνακα όταν **arr** είναι null |

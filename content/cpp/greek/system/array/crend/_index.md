## Array::crend() const μέθοδος


Επιστρέφει έναν αντίστροφο επαναλήπτη προς το στοιχείο που ακολουθεί το τελευταίο στοιχείο του αντεστραμμένου κοντέινερ. Αντιστοιχεί στο στοιχείο που προηγείται του πρώτου στοιχείου του μη αντεστραμμένου κοντέινερ. Αυτό το στοιχείο λειτουργεί ως σύμβολο κράτησης, η προσπάθεια πρόσβασης σε αυτό οδηγεί σε ακαθόριστη συμπεριφορά.

```cpp
const_reverse_iterator System::Array<T>::crend() const noexcept
```


### Τιμή επιστροφής

Ένας επαναλήπτης που δείχνει στο θεωρητικό στοιχείο με const-προσδιορισμό που προηγείται του πρώτου στοιχείου του κοντέινερ.

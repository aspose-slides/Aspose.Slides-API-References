## Array::crbegin() const μέθοδος

Επιστρέφει έναν αντίστροφο επαναληπτή προς το πρώτο στοιχείο του αντίστροφου κοντέινερ. Συμφωνεί με το τελευταίο στοιχείο του μη-αντίστροφου κοντέινερ. Εάν το κοντέινερ είναι κενό, ο επιστρεφόμενος επαναληπτής είναι ίσος με [crend()](../crend/).

```cpp
const_reverse_iterator System::Array<T>::crbegin() const noexcept
```

### Τιμή Επιστροφής

Ένας επαναληπτή που δείχνει στο τελευταίο στοιχείο του κοντέινερ που είναι const-qualified.

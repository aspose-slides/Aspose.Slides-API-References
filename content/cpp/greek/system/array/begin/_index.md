## Array::begin() μέθοδος

Επιστρέφει έναν επαναλήπτη προς το πρώτο στοιχείο του δοχείου. Εάν το δοχείο είναι άδειο, ο επιστραφμένος επαναλήπτης θα είναι ίσος με [end()](../end/).

```cpp
iterator System::Array<T>::begin() noexcept
```

### Τιμή Επιστροφής

Ένας επαναλήπτης που δείχνει στο πρώτο στοιχείο του δοχείου.

## Array::begin() const μέθοδος

Επιστρέφει έναν επαναλήπτη προς το πρώτο στοιχείο του const-διασφαλισμένου δοχείου. Εάν το δοχείο είναι άδειο, ο επιστραφμένος επαναλήπτης θα είναι ίσος με [end()](../end/).

```cpp
const_iterator System::Array<T>::begin() const noexcept
```

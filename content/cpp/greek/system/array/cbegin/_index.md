## Array::cbegin() const μέθοδος

Επιστρέφει έναν επαναλήπτη στο πρώτο στοιχείο με const-επιθεώρηση του container. Εάν το container είναι κενό, ο επαναλήπτης που επιστρέφεται θα είναι ίσος με [cend()](../cend/).

```cpp
const_iterator System::Array<T>::cbegin() const noexcept
```

### Τιμή Επιστροφής

Ένας επαναλήπτης που δείχνει στο πρώτο στοιχείο με const-επιθεώρηση του container.

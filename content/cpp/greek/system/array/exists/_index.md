## Array::Exists(ArrayPtr\<T\>, std::function\<bool(T)>) μέθοδος

Καθορίζει εάν το συγκεκριμένο αντικείμενο [Array](../) περιέχει ένα στοιχείο που ικανοποιεί τις απαιτήσεις της συγκεκριμένης συνθήκης.

```cpp
static bool System::Array<T>::Exists(ArrayPtr<T> arr, std::function<bool(T)> match)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| arr | [ArrayPtr](../../arrayptr/)\<T\> | Ο πίνακας στον οποίο αναζητείται το στοιχείο |
| match | std::function\<**bool**(T)> | Αντικείμενο συνάρτησης που ορίζει τις απαιτήσεις και ελέγχει εάν ένα στοιχείο τις ικανοποιεί |

### Τιμή Επιστροφής

Αληθές εάν το **arr** περιέχει ένα στοιχείο που ικανοποιεί τις απαιτήσεις που ορίζονται από το **match**

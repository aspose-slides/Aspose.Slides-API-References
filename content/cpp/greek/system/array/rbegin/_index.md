## Array::rbegin() μέθοδος

Επιστρέφει έναν αντίστροφο επαναλήπτη στο πρώτο στοιχείο του αντιστραμμένου δοχείου. Αντιστοιχεί στο τελευταίο στοιχείο του μη αντιστραμμένου δοχείου. Εάν το δοχείο είναι κενό, ο επιστρεφόμενος επαναλήπτης είναι ίσος με [rend()](../rend/).

```cpp
reverse_iterator System::Array<T>::rbegin() noexcept
```

### Τιμή Επιστροφής

Ένας επαναλήπτης που δείχνει στο τελευταίο στοιχείο του δοχείου.

const_reverse_iterator System::Array<T>::rbegin() const noexcept

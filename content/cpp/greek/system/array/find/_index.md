## Array::Find(System::ArrayPtr\<T\>, System::Predicate\<T\>) μέθοδος

Αναζητεί το πρώτο στοιχείο στον καθορισμένο πίνακα που ικανοποιεί τις συνθήκες του καθορισμένου προδιαγραφέα.

```cpp
static T System::Array<T>::Find(System::ArrayPtr<T> arr, System::Predicate<T> match)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| arr | [System::ArrayPtr](../../arrayptr/)\<T\> | [Array](../) για την αναζήτηση ενός στοιχείου σε |
| match | [System::Predicate](../../predicate/)\<T\> | Ένας προδιαγραφέας που ορίζει τις συνθήκες για την αντιστοίχιση των στοιχείων του πίνακα |

### Τιμή Επιστροφής

Αντίγραφο του πρώτου στοιχείου στον πίνακα που ικανοποιεί τις συνθήκες που ορίζονται από τον προδιαγραφέα, διαφορετικά η προεπιλεγμένη τιμή του τύπου T

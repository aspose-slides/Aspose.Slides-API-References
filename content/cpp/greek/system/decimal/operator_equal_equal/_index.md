## Decimal::operator==(const Decimal\&) const μέθοδος

Καθορίζει αν οι τιμές που αντιπροσωπεύονται από το τρέχον αντικείμενο και το καθορισμένο αντικείμενο είναι ίσες.

```cpp
bool System::Decimal::operator==(const Decimal &d) const
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| d | const [Decimal](../)\& | Το αντικείμενο [Decimal](../) για σύγκριση του τρέχοντος αντικειμένου |

### Τιμή Επιστροφής

True αν η τιμή που αντιπροσωπεύει το τρέχον και το καθορισμένο αντικείμενα είναι ίση, διαφορετικά - false

## Decimal::Round(const Decimal\&, MidpointRounding) μέθοδος


Στρογγυλοποιεί την καθορισμένη τιμή στον πλησιέστερο ακέραιο αριθμό. Μια παράμετρος καθορίζει τη συμπεριφορά της συνάρτησης εάν η καθορισμένη τιμή είναι εξίσου κοντά σε δύο πλησιέστερους αριθμούς.

```cpp
static Decimal System::Decimal::Round(const Decimal &d, MidpointRounding mode=MidpointRounding::ToEven)
```


### Ορίσματα

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| d | const [Decimal](../)\& | Η τιμή προς στρογγυλοποίηση |
| mode | [MidpointRounding](../../midpointrounding/) | Καθορίζει πώς γίνεται η στρογγυλοποίηση εάν **value** είναι εξίσου κοντά σε δύο πλησιέστερους αριθμούς. |

### Τιμή Επιστροφής

**d** στρογγυλοποιημένο στον πλησιέστερο ακέραιο

## Decimal::Round(const Decimal\&, int, MidpointRounding) μέθοδος


Στρογγυλοποιεί την καθορισμένη τιμή στην πλησιέστερη τιμή με τον καθορισμένο αριθμό δεκαδικών ψηφίων. Μια παράμετρος καθορίζει τη συμπεριφορά της συνάρτησης εάν η καθορισμένη τιμή είναι εξίσου κοντά σε δύο πλησιέστερους αριθμούς.

```cpp
static Decimal System::Decimal::Round(const Decimal &d, int digits, MidpointRounding mode=MidpointRounding::ToEven)
```


### Ορίσματα

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| d | const [Decimal](../)\& | Η τιμή προς στρογγυλοποίηση |
| digits | int | Ο αριθμός των δεκαδικών ψηφίων στην στρογγυλοποιημένη τιμή |
| mode | [MidpointRounding](../../midpointrounding/) | Καθορίζει πώς γίνεται η στρογγυλοποίηση εάν **value** είναι εξίσου κοντά σε δύο πλησιέστερους αριθμούς. |

### Τιμή Επιστροφής

Ο αριθμός με τον καθορισμένο αριθμό ψηφίων πλησιέστερος στο **value**

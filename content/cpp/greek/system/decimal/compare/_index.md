## Decimal::Compare(const Decimal\&, const Decimal\&) μέθοδος

Καθορίζει εάν η τιμή που αντιπροσωπεύεται από το πρώτο αντικείμενο [Decimal](../) είναι μικρότερη, ίση ή μεγαλύτερη από την τιμή που αντιπροσωπεύεται από το δεύτερο αντικείμενο [Decimal](../).

```cpp
static int System::Decimal::Compare(const Decimal &d1, const Decimal &d2)
```

### Arguments

| Parameter | Type | Description |
| --- | --- | --- |
| d1 | const [Decimal](../)\& | Το πρώτο συγκριτέο |
| d2 | const [Decimal](../)\& | Το δεύτερο συγκριτέο |

### Return Value

- 1 εάν η τιμή που αντιπροσωπεύεται από το **d1** είναι μικρότερη από την τιμή που αντιπροσωπεύεται από το **d2**; 0 εάν οι τιμές είναι ίσες; 1 εάν η τιμή που αντιπροσωπεύεται από το **d1** είναι μεγαλύτερη από την τιμή που αντιπροσωπεύεται από το **d2**

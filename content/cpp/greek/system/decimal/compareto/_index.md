## Decimal::CompareTo(const Decimal\&) const μέθοδος


Determines if the value represented by the current object is less than, equal to or greater than the value represented by the specified object.

```cpp
int System::Decimal::CompareTo(const Decimal &d) const
```


### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| d | const [Decimal](../)\& | Το συγκρίσιμο |

### Τιμή επιστροφής

- 1 εάν η τιμή που αντιπροσωπεύεται από το τρέχον αντικείμενο είναι μικρότερη από την τιμή που αντιπροσωπεύεται από **d**; 0 εάν οι τιμές είναι ίσες; 1 εάν η τιμή που αντιπροσωπεύεται από το τρέχον αντικείμενο είναι μεγαλύτερη από την τιμή που αντιπροσωπεύεται από **d**

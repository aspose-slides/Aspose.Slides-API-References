## HolderInitializer struct

Αυτή η κλάση χρησιμοποιείται για την απόκτηση μόνιμης αναφοράς στο αντίγραφο του αντικειμένου, ανεξαρτήτως αν είναι lvalue ή rvalue. Για να λάβετε τέτοια αναφορά, χρησιμοποιήστε τη μέθοδο 'HoldIfTemporary', η οποία έχει τρία υπερφορτωμένα. Δύο από αυτά δέχονται rvalue ως παράμετρο και απλώς επιστρέφουν την αναφορά σε αυτό. Το τρίτο, αντίστροφα, δέχεται lvalue ως παράμετρο, κάνει αντίγραφο δείκτη, και στη συνέχεια επιστρέφει την αναφορά σε αυτό το αντίγραφο. Επίσης, η κλάση περιέχει τη μέθοδο 'Hold' για να κρατήσει την περασμένη τιμή χωρίς προϋποθέσεις (χρησιμοποιείται για την αντιγραφή τιμών τοπικών μεταβλητών στο stack ή των αναφορών των παιδιών τους).

```cpp
template<typename T,bool>class HolderInitializer
```

### Template parameters

| Παράμετρος | Περιγραφή |
| --- | --- |
| T | Ο τύπος του αντικειμένου που θα κρατηθεί. |
| R | True, εάν το T είναι τύπος αναφοράς (ειδικός τύπος [SmartPtr](../smartptr/) ή τύπος [System::String](../string/)), και η διατήρηση προσωρινών αναφορών είναι πράγματι απαραίτητη, false - διαφορετικά. |

## ICustomFormatter κλάση

Ορίζει μια μέθοδο που εκτελεί προσαρμοσμένη μορφοποίηση της αναπαράστασης συμβολοσειράς μιας τιμής που αντιπροσωπεύεται από το συγκεκριμένο αντικείμενο. Τα αντικείμενα αυτής της κλάσης πρέπει να εκχωρούνται μόνο χρησιμοποιώντας τη συνάρτηση [System::MakeObject()](../makeobject/). Ποτέ μην δημιουργείτε αντικείμενο αυτού του τύπου στο stack ή χρησιμοποιώντας operator new, επειδή θα προκαλέσει σφάλματα χρόνου εκτέλεσης και/ή σφαλμάτων ελέγχου. Πάντα τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να την περάσετε σε συναρτήσεις ως όρισμα.

```cpp
class ICustomFormatter : public virtual System::Object
```

## Attribute κλάση

Μια βασική κλάση για προσαρμοσμένα χαρακτηριστικά. Αντικείμενα αυτής της κλάσης πρέπει να καλύπτονται μόνο μέσω της συνάρτησης [System::MakeObject()](../makeobject/). Ποτέ μην δημιουργήσετε ένα στιγμιότυπο αυτού του τύπου στη στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς αυτό θα οδηγήσει σε σφάλματα εκτέλεσης και/ή σφάλματα εικασίας. Πάντα τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να τον περάσετε σε συναρτήσεις ως όρισμα.

```cpp
class Attribute : public System::Object
```

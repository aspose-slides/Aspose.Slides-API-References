## Attribute::GetCustomAttribute(const TypeInfo\&, const TypeInfo\&) μέθοδος

Επιστρέφει ένα προσαρμοσμένο χαρακτηριστικό συγκεκριμένου τύπου που εφαρμόζεται σε συγκεκριμένο τύπο.

```cpp
static Object::ptr System::Attribute::GetCustomAttribute(const TypeInfo &type, const TypeInfo &attributeType)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| type | const [TypeInfo](../../typeinfo/)\& | Τύπος χαρακτηριστικού που ανακτάται |
| attributeType | const [TypeInfo](../../typeinfo/)\& | Τύπος του χαρακτηριστικού που πρέπει να ανακτηθεί |

### Τιμή Επιστροφής

Ένα ανακτημένο χαρακτηριστικό ή null εάν ο καθορισμένος τύπος δεν έχει χαρακτηριστικό του καθορισμένου τύπου.

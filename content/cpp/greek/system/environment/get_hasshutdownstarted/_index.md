## Environment::get_HasShutdownStarted() μέθοδος

Ελέγχει εάν η διακοπή λειτουργίας βρίσκεται σε εξέλιξη. Δεν είναι υλοποιημένο.

```cpp
static bool System::Environment::get_HasShutdownStarted()
```

### Τιμή Επιστροφής

true εάν βρίσκεται σε εξέλιξη, false διαφορετικά.

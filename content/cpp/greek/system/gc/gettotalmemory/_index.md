## GC::GetTotalMemory(bool) μέθοδος

Επιστρέφει τον αριθμό των bytes της ιδιωτικής μνήμης που έχει εκχωρηθεί αυτήν τη στιγμή από την τρέχουσα διαδικασία.

```cpp
static int64_t System::GC::GetTotalMemory(bool force_full_collection)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| force_full_collection | **bool** | IGNORED |

### Τιμή επιστροφής

Ο αριθμός των bytes της ιδιωτικής μνήμης που έχει εκχωρηθεί αυτήν τη στιγμή από την τρέχουσα διαδικασία.

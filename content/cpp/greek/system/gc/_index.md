## GC κλάση

Αντιπροσωπεύει μια προσομοιωμένη Garbage Collection που λειτουργεί πιο πολύ σαν ένα σκελετό που ουσιαστικά δεν κάνει τίποτα. Αυτό είναι ένας στατικός τύπος χωρίς υπηρεσίες στιγμιοτύπου. Δεν θα πρέπει ποτέ να δημιουργείτε στιγμιότυπα του με κανένα τρόπο.

```cpp
class GC : public System::Object
```

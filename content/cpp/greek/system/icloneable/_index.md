## ICloneable κλάση

Προσδιορίζει μια μέθοδο που επιτρέπει την κλωνοποίηση αντικειμένων – τη δημιουργία αντιγράφου ενός αντικειμένου. Τα αντικείμενα αυτής της κλάσης θα πρέπει να δημιουργούνται μόνο με τη λειτουργία [System::MakeObject()](../makeobject/). Ποτέ μην δημιουργείτε εμφάνιση αυτού του τύπου στο στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς θα προξενήσει σφάλματα χρόνου εκτέλεσης και/ή σφάλματα ελέγχου. Πάντα τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να τον περάσετε στις συναρτήσεις ως παράμετρο.

```cpp
class ICloneable : public virtual System::Object
```

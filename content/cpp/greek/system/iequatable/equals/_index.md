## IEquatable::Equals(T) μέθοδος

Καθορίζει εάν τα τρέχοντα και τα καθορισμένα αντικείμενα είναι ίσα.

```cpp
virtual bool System::IEquatable<T>::Equals(T other)=0
```

### Παράμετροι

| Parameter | Type | Description |
| --- | --- | --- |
| other | T | Το αντικείμενο με το οποίο συγκρίνεται το τρέχον αντικείμενο |

### Τιμή Επιστροφής

True εάν τα αντικείμενα είναι ίσα, διαφορετικά - false

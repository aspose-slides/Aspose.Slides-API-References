## System::IsTuple(const SharedPtr\<Object\>\&, int32_t) συνάρτηση


Ελέγχει αν το object είναι πλειάδα (εφαρμόζει τη διεπαφή ITuple). Χρησιμοποιείται στην υλοποίηση του positional pattern.

```cpp
bool System::IsTuple(const SharedPtr<Object> &object, int32_t length)
```


### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| object | const [SharedPtr](../sharedptr/)\<[Object](../object/)\>\& | object to check. |
| length | **int32_t** | expected tuple length |

### Τιμή Επιστροφής

είναι το object πλειάδα και το μήκος του ίσο με το αναμενόμενο

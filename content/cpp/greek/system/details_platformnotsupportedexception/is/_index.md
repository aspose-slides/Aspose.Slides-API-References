## Details_PlatformNotSupportedException::Is(const System::TypeInfo\&) const μέθοδος

```cpp
bool System::Details_PlatformNotSupportedException::Is(const System::TypeInfo &target) const override
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) δομή που περιγράφει τον τύπο για τον οποίο ελέγχεται το τρέχον αντικείμενο. |

### Τιμή Επιστροφής

True εάν το αντικείμενο είναι του επισημασμένου τύπου ή της υποκλάσης του, false διαφορετικά.

## Σχόλια

Ελέγχει εάν το αντικείμενο αντιπροσωπεύει μια παρουσία του τύπου που περιγράφεται από targetType. Αναλογικό του τελεστή 'is' της C#.

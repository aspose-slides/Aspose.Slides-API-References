## SmartPtr::Is(const System::TypeInfo\&) const μέθοδος

Ελέγχει εάν το αντικείμενο στο οποίο δείχνει είναι συγκεκριμένου τύπου ή τύπου απογόνου. Ακολουθεί τη σημασιολογία του C# 'is'.

```cpp
bool System::SmartPtr<T>::Is(const System::TypeInfo &target) const
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | Καθορίζει τον τύπο-στόχο για σύγκριση. |

### Τιμή Επιστροφής

True εάν ο έλεγχος τύπου C# 'is' είναι θετικός και false διαφορετικά.

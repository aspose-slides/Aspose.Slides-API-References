## SmartPtr::get_shared_count() const method


Επιστρέφει τον αριθμό των κοινών δεικτών που υπάρχουν στο αναφερόμενο αντικείμενο, συμπεριλαμβανομένου του τρέχοντος. Επιβεβαιώνει ότι ο τρέχων δείκτης βρίσκεται σε κοινή λειτουργία.

```cpp
int System::SmartPtr<T>::get_shared_count() const
```


### Τιμή Επιστροφής

Αριθμός των κοινών δεικτών που υπάρχουν στο αναφερόμενο αντικείμενο, εάν υπάρχει. Εάν ο δείκτης είναι null, επιστρέφει 0.

## SmartPtr::RemoveAliasing() const μέθοδος

Καταργεί την aliasing (που δημιουργείται από έναν aliasing constructor) από τον pointer, εξασφαλίζει ότι διαχειρίζεται (εάν shared) ή παρακολουθεί (εάν weak) το ίδιο αντικείμενο στο οποίο δείχνει.

```cpp
SmartPtr_ System::SmartPtr<T>::RemoveAliasing() const
```


### Τιμή Επιστροφής

Ο pointer στο ίδιο αντικείμενο στο οποίο δείχνει αυτός ο pointer, που φροντίζει για τη διάρκεια ζωής του ίδιου αντικειμένου.

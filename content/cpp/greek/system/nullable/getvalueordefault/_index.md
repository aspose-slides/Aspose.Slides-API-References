## Nullable::GetValueOrDefault(T) μέθοδος


Επιστρέφει την τιμή που αντιπροσωπεύεται από το τρέχον αντικείμενο ή τη συγκεκριμένη τιμή εάν η τιμή που αντιπροσωπεύεται από το τρέχον αντικείμενο είναι null.

```cpp
T System::Nullable<T>::GetValueOrDefault(T default_value)
```


### Arguments

| Parameter | Type | Description |
| --- | --- | --- |
| default_value | T | Η τιμή που επιστρέφεται από τη μέθοδο εάν η τιμή που αντιπροσωπεύεται από το τρέχον αντικείμενο είναι null |

### Return Value

Ένα αντίγραφο της τιμής που αντιπροσωπεύεται από το τρέχον αντικείμενο εάν δεν είναι null, διαφορετικά επιστρέφεται η συγκεκριμένη τιμή

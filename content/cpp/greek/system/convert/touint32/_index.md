```cpp

```

| Parameter | Type | Description |
| --- | --- | --- |
| obj | const [SharedPtr](../../sharedptr/)\<[Object](../../object/)\>\& | Ο shared pointer προς το αντικείμενο που κουτίζει την τιμή προς μετατροπή |

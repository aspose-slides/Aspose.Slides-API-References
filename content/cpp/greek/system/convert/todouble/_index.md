## Convert::ToDouble(const SharedPtr\<Object\>\&, const SharedPtr\<IFormatProvider\>\&) μέθοδος


Μετατρέπει την καθορισμένη τιμή σε κουβώτιο (boxed) σε τιμή κινητής υποδιαστολής διπλής ακρίβειας. Εάν ο τύπος της κουβώτιας τιμής είναι [String](../../string/), χρησιμοποιείται η καθορισμένη μορφοποίηση συμβολοσειράς κατά τη μετατροπή.

```cpp
static double System::Convert::ToDouble(const SharedPtr<Object> &obj, const SharedPtr<IFormatProvider> &provider=nullptr)
```


### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| obj | const [SharedPtr](../../sharedptr/)\<[Object](../../object/)\>\& | Ο shared pointer στο αντικείμενο που κουβώνει την τιμή προς μετατροπή |
| provider | const [SharedPtr](../../sharedptr/)\<[IFormatProvider](../../iformatprovider/)\>\& | Η μορφοποίηση συμβολοσειράς που θα χρησιμοποιηθεί εάν ο τύπος της κουβώτιας τιμής είναι [String](../../string/) |

### Τιμή Επιστροφής

Μια τιμή κινητής υποδιαστολής διπλής ακρίβειας ισοδύναμη με την καθορισμένη κουβώτια τιμή

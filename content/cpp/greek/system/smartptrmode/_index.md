## SmartPtrMode enum


[SmartPtr](../smartptr/) τύπος δείκτη: weak ή shared. Ορίζει εάν ο δείκτης μετράται όταν αποφασίζεται αν θα διαγραφεί το αντικείμενο ή όχι.

```cpp
enum class SmartPtrMode : char
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Shared | 0 | Shared mode: ο δείκτης συμμετέχει στην καταμέτρηση αναφορών. |
| Weak | 1 | Weak mode: ο δείκτης δεν συμμετέχει στην καταμέτρηση αναφορών. |

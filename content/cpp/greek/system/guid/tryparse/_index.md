## Guid::TryParse(const String\&, Guid\&) μέθοδος


Προσπαθεί να μετατρέψει το καθορισμένο κείμενο σε αντικείμενο [Guid](../).

```cpp
static bool System::Guid::TryParse(const String &input, Guid &g)
```


### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| input | const [String](../../string/)\& | Η συμβολοσειρά για μετατροπή |
| g | [Guid](../)\& | Αντικείμενο εξόδου [Guid](../), εάν πετύχει. |

### Τιμή επιστροφής

True, εάν η συμβολοσειρά εισόδου αντιπροσωπεύει έγκυρο [Guid](../), διαφορετικά false.

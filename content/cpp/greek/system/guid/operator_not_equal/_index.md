## Guid::operator!=(const Guid\&) const μέθοδος

Καθορίζει εάν τα GUID που αντιπροσωπεύουν το τρέχον και το καθορισμένο αντικείμενο δεν είναι ίσα.

```cpp
bool System::Guid::operator!=(const Guid &b) const
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| b | const [Guid](../)\& | Η τιμή σύγκρισης |

### Τιμή Επιστροφής

true εάν τα GUID που αντιπροσωπεύονται από το τρέχον και το καθορισμένο αντικείμενο δεν είναι ίσα, διαφορετικά - false

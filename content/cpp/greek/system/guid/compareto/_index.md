## Guid::CompareTo(const Guid\&) const μέθοδος


Διεξάγει αριθμητική σύγκριση των GUID που αντιπροσωπεύονται από το τρέχον και το καθορισμένο αντικείμενο.

```cpp
int System::Guid::CompareTo(const Guid &g) const
```


### Arguments

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| g | const [Guid](../)\& | Το συγκριτικό |

### Return Value

- 1 εάν το GUID που αντιπροσωπεύεται από το τρέχον αντικείμενο είναι μικρότερο από το GUID που αντιπροσωπεύεται από **g**; 0 εάν τα GUID είναι ίσα; 1 εάν το GUID που αντιπροσωπεύεται από το τρέχον αντικείμενο είναι μεγαλύτερο από το GUID που αντιπροσωπεύεται από **g**

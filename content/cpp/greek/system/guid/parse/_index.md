## Guid::Parse(const String\&) μέθοδος

```cpp
static Guid System::Guid::Parse(const String &input)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| input | const [String](../../string/)\& | Η συμβολοσειρά που περιέχει την αναπαράσταση συμβολοσειράς του GUID |

### Τιμή Επιστροφής

Ένα αντικείμενο [Guid](../) που αντιπροσωπεύει ένα GUID ισοδύναμο με αυτό που αναπαριστάται από **input**

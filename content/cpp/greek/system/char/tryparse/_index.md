## Char::TryParse(const System::String\&, char_t\&) μέθοδος

Προσπαθεί να μετατρέψει μια συμβολοσειρά που αποτελείται από ένα μόνο χαρακτήρα σε χαρακτήρα UTF-16. Η συνάρτηση επιτυγχάνει μόνο όταν η συμβολοσειρά εισόδου δεν είναι null και έχει μήκος ακριβώς ενός χαρακτήρα.

```cpp
static bool System::Char::TryParse(const System::String &s, char_t &result)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| s | const [System::String](../../string/)\& | [String](../../string/) για μετατροπή |
| result | char_t\& | Η μεταβλητή εξόδου που θα περιέχει το αποτέλεσμα της μετατροπής εάν η μετατροπή επιτύχει |

### Τιμή Επιστροφής

True εάν η μετατροπή πέτυχε, διαφορετικά - false

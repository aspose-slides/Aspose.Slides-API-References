## Char::IsUpper(const String\&, int) μέθοδος


Καθορίζει εάν το χαρακτήρας στη συγκεκριμένη θέση στο συγκεκριμένο String είναι ταξινομημένο ως γράμμα κεφαλαίου.

```cpp
static bool System::Char::IsUpper(const String &str, int idx)
```


### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| str | const [String](../../string/)\& | Το κείμενο που περιέχει ένα χαρακτήρα. |
| idx | int | Μηδενική θέση στο συγκεκριμένο κείμενο προς έλεγχο |

### Τιμή Επιστροφής

Αληθές εάν το χαρακτήρα στη συγκεκριμένη θέση είναι γράμμα κεφαλαίου, διαφορετικά - ψευδές

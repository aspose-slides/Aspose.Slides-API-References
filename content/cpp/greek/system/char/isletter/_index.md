## Char::IsLetter(const char_t *, int) μέθοδος

Καθορίζει εάν ο χαρακτήρας στην καθορισμένη θέση στον καθορισμένο buffer χαρακτήρων ταξινομείται ως γράμμα Unicode.

```cpp
static bool System::Char::IsLetter(const char_t *str, int idx)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| str | const char_t * | Δείκτης στην αρχή του buffer χαρακτήρων |
| idx | int | Μηδενική θέση στο καθορισμένο buffer του χαρακτήρα που θα ελεγχθεί |

### Τιμή Επιστροφής

True εάν ο χαρακτήρας στην καθορισμένη θέση είναι γράμμα Unicode, διαφορετικά - false

## Char::IsLetter(char_t) μέθοδος

Καθορίζει εάν ο συγκεκριμένος χαρακτήρας ταξινομείται ως γράμμα Unicode.

```cpp
static bool System::Char::IsLetter(char_t c)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| c | char_t | Ο χαρακτήρας που θα ελεγχθεί |

### Τιμή Επιστροφής

True εάν ο συγκεκριμένος χαρακτήρας είναι γράμμα Unicode, διαφορετικά - false

## Char::IsLower(const char_t *, int) μέθοδος

Καθορίζει εάν ο χαρακτήρας στη συγκεκριμένη θέση στο καθορισμένο buffer χαρακτήρων ταξινομείται ως μικρό γράμμα.

```cpp
static bool System::Char::IsLower(const char_t *str, int idx)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| str | const char_t * | Δείκτης στην αρχή του buffer χαρακτήρων |
| idx | int | Μηδενική θέση στο καθορισμένο buffer του χαρακτήρα προς έλεγχο |

### Τιμή Επιστροφής

True εάν ο χαρακτήρας στη συγκεκριμένη θέση είναι μικρό γράμμα, διαφορετικά - false

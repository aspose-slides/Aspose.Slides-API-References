## Char::IsSymbol(const char_t *, int) μέθοδος

Καθορίζει εάν ο χαρακτήρας στη συγκεκριμένη θέση στο καθορισμένο buffer χαρακτήρων ταξινομείται ως χαρακτήρας συμβόλου.

```cpp
static bool System::Char::IsSymbol(const char_t *str, int idx)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| str | const char_t * | Δείκτης στην αρχή του buffer χαρακτήρων |
| idx | int | Δείκτης αρχής μηδενικής βάσης στο καθορισμένο buffer του χαρακτήρα προς έλεγχο |

### Τιμή Επιστροφής

Αληθές εάν ο χαρακτήρας στη συγκεκριμένη θέση είναι χαρακτήρας συμβόλου, διαφορετικά - ψευδές

static bool System::Char::IsSymbol(char_t c)

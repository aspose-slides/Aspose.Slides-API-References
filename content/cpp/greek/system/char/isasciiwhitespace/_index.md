## Char::IsAsciiWhiteSpace(char_t) μέθοδος

Determines whether the specified character is classified as a ASCII white space character.

```cpp
static constexpr bool System::Char::IsAsciiWhiteSpace(char_t c)
```

### Ορίσματα

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| c | char_t | Ο χαρακτήρας προς δοκιμή |

### Τιμή επιστροφής

True αν ο καθορισμένος χαρακτήρας είναι χαρακτήρας ASCII λευκού διαστήματος, διαφορετικά - false

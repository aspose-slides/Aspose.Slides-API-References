## Char::IsSurrogate(char_t) μέθοδος

Καθορίζει αν ο καθορισμένος χαρακτήρας είναι μονάδα κώδικα UTF-16 surrogate.

```cpp
static bool System::Char::IsSurrogate(char_t c)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| c | char_t | Ένας χαρακτήρας |

### Τιμή Επιστροφής

Αληθής εάν ο καθορισμένος χαρακτήρας είναι μονάδα κώδικα UTF-16 surrogate, διαφορετικά - ψευδής

## Char::IsSurrogate(const String\&, int) μέθοδος

Καθορίζει εάν ο χαρακτήρας στη συγκεκριμένη θέση της καθορισμένης συμβολοσειράς είναι μονάδα κώδικα UTF-16 surrogate.

```cpp
static bool System::Char::IsSurrogate(const String &s, int index)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| s | const [String](../../string/)\& | Μία συμβολοσειρά |
| index | int | Ο δείκτης του χαρακτήρα στη καθορισμένη συμβολοσειρά |

### Τιμή Επιστροφής

Αληθής εάν ο χαρακτήρας στη συγκεκριμένη θέση είναι μονάδα κώδικα UTF-16 surrogate, διαφορετικά - ψευδής

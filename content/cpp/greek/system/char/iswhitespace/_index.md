## Char::IsWhiteSpace(const char_t *, int) μέθοδος


Καθορίζει εάν ο χαρακτήρας στη συγκεκριμένη θέση στο καθορισμένο buffer χαρακτήρων ταξινομείται ως χαρακτήρας κενό.

```cpp
static bool System::Char::IsWhiteSpace(const char_t *str, int idx)
```


### Ορίσματα

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| str | const char_t * | Δείκτης στην αρχή του buffer χαρακτήρων |
| idx | int | Μηδενική διεύθυνση στη συγκεκριμένη θέση του buffer του χαρακτήρα που ελέγχεται |

### Τιμή Επιστροφής

True εάν ο χαρακτήρας στη συγκεκριμένη θέση είναι χαρακτήρας κενό, διαφορετικά - false

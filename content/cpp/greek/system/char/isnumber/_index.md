## Char::IsNumber(const char_t *, int) μέθοδος

Καθορίζει εάν ο χαρακτήρας στη συγκεκριμένη θέση του καθορισμένου buffer χαρακτήρων ταξινομείται ως αριθμός.

```cpp
static bool System::Char::IsNumber(const char_t *str, int idx)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| str | const char_t * | Δείκτης στην αρχή του buffer χαρακτήρων |
| idx | int | Δείκτης μηδενικής βάσης στο καθορισμένο buffer του χαρακτήρα προς έλεγχο |

### Τιμή επιστροφής

Αληθές αν ο χαρακτήρας στη συγκεκριμένη θέση είναι αριθμός, διαφορετικά - ψευδές

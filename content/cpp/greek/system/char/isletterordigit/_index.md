## Char::IsLetterOrDigit(const char_t *, int) μέθοδος

Καθορίζει εάν ο χαρακτήρας στην καθορισμένη θέση στον καθορισμένο buffer χαρακτήρων ταξινομείται ως Unicode γράμμα ή δεκαδικό ψηφίο.

```cpp
static bool System::Char::IsLetterOrDigit(const char_t *str, int idx)
```

### Παράμετρα

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| str | const char_t * | Δείκτης στην αρχή του buffer χαρακτήρων |
| idx | int | Μηδενική βάση δείκτης στο καθορισμένο buffer του χαρακτήρα προς δοκιμή |

### Τιμή Επιστροφής

True εάν ο χαρακτήρας στην καθορισμένη θέση είναι Unicode γράμμα ή δεκαδικό ψηφίο, διαφορετικά - false

static bool System::Char::IsLetterOrDigit(char_t c)

### Παράμετρα

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| c | char_t | Ο χαρακτήρας προς δοκιμή |

### Τιμή Επιστροφής

True εάν ο καθορισμένος χαρακτήρας είναι Unicode γράμμα ή δεκαδικό ψηφίο, διαφορετικά - false

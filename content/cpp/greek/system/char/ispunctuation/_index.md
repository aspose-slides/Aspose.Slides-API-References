## Char::IsPunctuation(const char_t *, int) μέθοδος

Καθορίζει εάν ο χαρακτήρας στη συγκεκριμένη θέση στον συγκεκριμένο buffer χαρακτήρων ταξινομείται ως χαρακτήρας στίξης.

```cpp
static bool System::Char::IsPunctuation(const char_t *str, int idx)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| str | const char_t * | Δείκτης στην αρχή του buffer χαρακτήρων |
| idx | int | Μηδενική θέση στο καθορισμένο buffer του χαρακτήρα που θα ελεγχθεί |

### Τιμή Επιστροφής

Αληθές εάν ο χαρακτήρας στην καθορισμένη θέση είναι χαρακτήρας στίξης, διαφορετικά - ψευδές

Καθορίζει εάν ο συγκεκριμένος χαρακτήρας ταξινομείται ως χαρακτήρας στίξης.

```cpp
static bool System::Char::IsPunctuation(char_t c)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| c | char_t | Ο χαρακτήρας που θα ελεγχθεί |

### Τιμή Επιστροφής

Αληθές εάν ο συγκεκριμένος χαρακτήρας είναι χαρακτήρας στίξης, διαφορετικά - ψευδές

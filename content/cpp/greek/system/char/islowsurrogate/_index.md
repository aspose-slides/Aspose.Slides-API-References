## Char::IsLowSurrogate(const char_t *, int) μέθοδος

Καθορίζει εάν ο χαρακτήρας στη συγκεκριμένη θέση στο καθορισμένο buffer χαρακτήρων είναι χαμηλό υποκατάστατο.

```cpp
static bool System::Char::IsLowSurrogate(const char_t *str, int idx)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| str | const char_t * | Δείκτης στην αρχή του buffer χαρακτήρων |
| idx | int | Δείκτης που ξεκινά από το μηδέν στο συγκεκριμένο buffer του χαρακτήρα προς έλεγχο |

### Τιμή επιστροφής

True εάν ο χαρακτήρας στη συγκεκριμένη θέση είναι χαμηλό υποκατάστατο, διαφορετικά - false

Καθορίζει εάν ο συγκεκριμένος χαρακτήρας είναι χαμηλό υποκατάστατο.

static bool System::Char::IsLowSurrogate(char_t c)

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| c | char_t | Ο χαρακτήρας προς έλεγχο |

### Τιμή επιστροφής

True εάν ο συγκεκριμένος χαρακτήρας είναι χαμηλό υποκατάστατο, διαφορετικά - false

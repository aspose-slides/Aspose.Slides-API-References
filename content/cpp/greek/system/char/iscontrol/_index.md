## Char::IsControl(const char_t *, int) μέθοδος


Καθορίζει εάν ο χαρακτήρας στη συγκεκριμένη θέση στο συγκεκριμένο buffer χαρακτήρων ταξινομείται ως χαρακτήρας ελέγχου Unicode.

```cpp
static bool System::Char::IsControl(const char_t *str, int idx)
```


### Παραμέτρους

| Parameter | Type | Description |
| --- | --- | --- |
| str | const char_t * | Δεικτής στην αρχή του buffer χαρακτήρων |
| idx | int | Δείκτης μηδενικής βάσης στο συγκεκριμένο buffer του χαρακτήρα που ελέγχεται |

### Τιμή επιστροφής

Αληθές εάν ο χαρακτήρας στη συγκεκριμένη θέση είναι χαρακτήρας ελέγχου Unicode, διαφορετικά - ψευδές

## Char::ConvertToUtf32(char_t, char_t) μέθοδος


Μετατρέπει το καθορισμένο ζεύγος αντιπροσώπων UTF-16 σε μονάδα κώδικα UTF-32.

```cpp
static int System::Char::ConvertToUtf32(char_t highSurrogate, char_t lowSurrogate)
```


### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| highSurrogate | char_t | Ο υψηλός αντιπρόσωπος του ζεύγους αντιπροσώπων UTF-16 για μετατροπή |
| lowSurrogate | char_t | Ο χαμηλός αντιπρόσωπος του ζεύγους αντιπροσώπων UTF-16 για μετατροπή |

### Τιμή Επιστροφής

Μονάδα κώδικα UTF-32 που προκύπτει από τη μετατροπή

## Char::ConvertToUtf32(const String\&, int) μέθοδος


Μετατρέπει την τιμή ενός χαρακτήρα ή ζεύγους αντιπροσώπων κωδικοποιημένου σε UTF-16 σε συγκεκριμένη θέση σε μια συμβολοσειρά σε μονάδα κώδικα UTF-32.

```cpp
static int System::Char::ConvertToUtf32(const String &s, int index)
```


### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| s | const [String](../../string/)\& | Μια συμβολοσειρά που περιέχει έναν χαρακτήρα ή ζεύγος αντιπροσώπων |
| index | int | Η θέση δείκτη του χαρακτήρα ή ζεύγους αντιπροσώπων στην καθορισμένη συμβολοσειρά |

### Τιμή Επιστροφής

Μονάδα κώδικα UTF-32 που προκύπτει από τη μετατροπή

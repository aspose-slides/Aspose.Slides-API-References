## Char::IsDigit(const char_t *, int) μέθοδος

Καθορίζει αν ο χαρακτήρας στη συγκεκριμένη θέση στο συγκεκριμένο buffer χαρακτήρων ταξινομείται ως δεκαδικό ψηφίο.

```cpp
static bool System::Char::IsDigit(const char_t *str, int idx)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| str | const char_t * | Δείκτης στην αρχή του buffer χαρακτήρων |
| idx | int | Μηδενική θέση στο συγκεκριμένο buffer του χαρακτήρα προς έλεγχο |

### Τιμή Επιστροφής

True εάν ο χαρακτήρας στη συγκεκριμένη θέση είναι δεκαδικό ψηφίο, διαφορετικά - false

## Char::IsSeparator(const char_t *, int) μέθοδος

Καθορίζει αν ο χαρακτήρας στη συγκεκριμένη θέση στο καθορισμένο buffer χαρακτήρων ταξινομείται ως χαρακτήρας διαχωριστή.

```cpp
static bool System::Char::IsSeparator(const char_t *str, int idx)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| str | const char_t * | Δείκτης στην αρχή του buffer χαρακτήρων |
| idx | int | Δείκτης μηδενικής βάσης στον καθορισμένο buffer του χαρακτήρα που ελέγχεται |

### Τιμή Επιστροφής

Αληθές εάν ο χαρακτήρας στη συγκεκριμένη θέση είναι χαρακτήρας διαχωριστή, διαφορετικά - ψευδές

Καθορίζει αν ο συγκεκριμένος χαρακτήρας ταξινομείται ως χαρακτήρας διαχωριστή.

```cpp
static bool System::Char::IsSeparator(char_t c)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| c | char_t | Ο χαρακτήρας που ελέγχεται |

### Τιμή Επιστροφής

Αληθές εάν ο συγκεκριμένος χαρακτήρας είναι χαρακτήρας διαχωριστή, διαφορετικά - ψευδές

## EnumParseHelper struct

Βοηθητική κλάση που παρέχει λειτουργίες μετατροπής μιας αλφαριθμητικής αναπαράστασης ενός enum σε ισοδύναμη τιμή enum.

```cpp
template<class E,class G,class Guard>class EnumParseHelper
```

### Template parameters

| Parameter | Description |
| --- | --- |
| E | Ο τύπος του enum με τον οποίο λειτουργούν οι μέθοδοι της κλάσης |
| G | Δεύτερο επίσημο επιχείρημα του [System::Enum](../enum/) για χρήση. |
| Guard | Επίσημη παράμετρος προτύπου για την επιλογή του διαθέσιμου αλγορίθμου ανάλυσης. |

## Methods

| Method | Description |
| --- | --- |
| static E [Parse](./parse/)(const [String](../string/)\&, **bool**) | Μετατρέπει το καθορισμένο κείμενο σε ισοδύναμη τιμή σταθεράς enum. |

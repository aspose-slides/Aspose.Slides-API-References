## EnumParseHelper::Parse(const String\&, bool) μέθοδος

Μετατρέπει τη δοσμένη συμβολοσειρά σε ισοδύναμη τιμή σταθεράς enum.

```cpp
static E System::EnumParseHelper<E, G, Guard>::Parse(const String &str, bool ignoreCase=false)
```

### Ορίσματα

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| str | const [String](../../string/)\& | [String](../../string/) που ερμηνεύεται ως περιέχοντας το όνομα της σταθεράς enum |
| ignoreCase | **bool** | Καθορίζει αν η πεζοκεφαλαία διαφορά θα αγνοηθεί κατά την ερμηνεία της συμβολοσειράς |

### Τιμή Επιστροφής

Η σταθερά enum του οποίου το όνομα προσδιορίζεται στο **str**

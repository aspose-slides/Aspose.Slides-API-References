## Details_NotSupportedException κλάση

Το NotSupportedException προκαλείται όταν μια μέθοδος που κλήθηκε δεν υποστηρίζεται ή όταν μια ενέργεια που έγινε σε μια ροή δεν υποστηρίζεται. Μην δημιουργείτε αντικείμενα αυτής της κλάσης χειροκίνητα. Χρησιμοποιήστε την κλάση NotSupportedException αντί για αυτήν. Μην ενσωματώνετε τις αντικείμενα της κλάσης NotSupportedException στο [System::SmartPtr](../smartptr/).

```cpp
class Details_NotSupportedException : public System::Details_SystemException
```

## Object::Is(const TypeInfo\&) const μέθοδος

Check if αντικείμενο represents an instance of type described by targetType. Analog of C# 'is' operator.

```cpp
virtual bool System::Object::Is(const TypeInfo &targetType) const
```

### Παραμέτρους

| Parameter | Type | Description |
| --- | --- | --- |
| targetType | const [TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) δομή που περιγράφει τον τύπο για να ελέγξει το τρέχον αντικείμενο απέναντι. |

### Τιμή Επιστροφής

True αν αντικείμενο είναι του επισημασμένου τύπου ή της υποκλάσης του, false διαφορετικά.

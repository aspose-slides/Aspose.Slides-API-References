## Object::Equals(float const\&, float const\&) μέθοδος

Εξομοιώνει σύγκριση κινητής υποδιαστολής σε στυλ C# όπου δύο NaN θεωρούνται ίσα παρόλο που σύμφωνα με IEC 60559:1989 το NaN δεν είναι ίσο με καμία τιμή, συμπεριλαμβανομένου του NaN.

```cpp
bool System::Object::Equals(float const &objA, float const &objB)
```

### Ορίσματα

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| objA | **float** const\& | Τιμή κινητής υποδιαστολής αριστερά. |
| objB | **float** const\& | Τιμή κινητής υποδιαστολής δεξιά. |

### Τιμή Επιστροφής

Αληθές εάν τα **objA** και **objB** είναι και τα δύο NaN ή ίσα, ψευδές διαφορετικά.

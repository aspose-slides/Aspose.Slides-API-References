## DateTime::operator>=(DateTime) const μέθοδος


Καθορίζει αν το τρέχον αντικείμενο αντιπροσωπεύει την τιμή ημερομηνίας και ώρας που είναι μεταγενέστερη ή ίδια με την τιμή που αντιπροσωπεύεται από το συγκεκριμένο [DateTime](../) αντικείμενο.

```cpp
constexpr bool System::DateTime::operator>=(DateTime other) const
```


### Arguments

| Parameter | Type | Description |
| --- | --- | --- |
| other | [DateTime](../) | Το [DateTime](../) αντικείμενο για σύγκριση του τρέχοντος αντικειμένου με |

### Τιμή επιστροφής

Αληθές εάν η τιμή ημερομηνίας και ώρας που αντιπροσωπεύεται από το τρέχον αντικείμενο είναι μεταγενέστερη ή ίδια με την τιμή που αντιπροσωπεύεται από **other**, διαφορετικά - ψευδές

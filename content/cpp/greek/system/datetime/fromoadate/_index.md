```cpp
static DateTime System::DateTime::FromOADate(double date)
```

### Arguments

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| date | **double** | OLE Automation Date για μετατροπή σε αντικείμενο [DateTime](../) |

### Return Value

Ένα νέο αντικείμενο της κλάσης [DateTime](../) που αντιπροσωπεύει την ίδια ημερομηνία και ώρα με το **date**

## DateTime::operator>(DateTime) const μέθοδος


Καθορίζει εάν το τρέχον αντικείμενο αντιπροσωπεύει την τιμή ημερομηνίας και ώρας που είναι μεταγενέστερη από την τιμή που αντιπροσωπεύεται από το συγκεκριμένο [DateTime](../) αντικείμενο.

```cpp
constexpr bool System::DateTime::operator>(DateTime other) const
```


### Ορίσματα

| Parameter | Type | Description |
| --- | --- | --- |
| other | [DateTime](../) | Το [DateTime](../) αντικείμενο για σύγκριση με το τρέχον αντικείμενο |

### Τιμή Επιστροφής

True εάν η τιμή ημερομηνίας και ώρας που αντιπροσωπεύεται από το τρέχον αντικείμενο είναι μεταγενέστερη από την τιμή που αντιπροσωπεύεται από **other**, αλλιώς - false

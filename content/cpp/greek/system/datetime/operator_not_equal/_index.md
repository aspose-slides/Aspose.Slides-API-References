## DateTime::operator!=(DateTime) const μέθοδος


Καθορίζει εάν το τρέχον αντικείμενο και το καθορισμένο [DateTime](../) αντικείμενο αντιπροσωπεύουν διαφορετικές τιμές ημερομηνίας και ώρας.

```cpp
constexpr bool System::DateTime::operator!=(DateTime other) const
```


### Παράμετροι

| Parameter | Type | Description |
| --- | --- | --- |
| other | [DateTime](../) | Το [DateTime](../) αντικείμενο για σύγκριση με το τρέχον αντικείμενο |

### Τιμή επιστροφής

True if the current object and the specified object represent distinct date and time values, otherwise - false

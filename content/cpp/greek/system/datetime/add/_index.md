## DateTime::Add(TimeSpan) const μέθοδος

```cpp
DateTime System::DateTime::Add(TimeSpan value) const
```

### Παράμετροι

| value | [TimeSpan](../../timespan/) | Το χρονικό διάστημα προς προσθήκη |

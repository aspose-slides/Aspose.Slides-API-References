## DateTime::CompareTo(DateTime) const μέθοδος

Συγκρίνει δύο τιμές ημερομηνίας και ώρας που αντιπροσωπεύονται από το τρέχον αντικείμενο και το συγκεκριμένο αντικείμενο της κλάσης [DateTime](../) και επιστρέφει τη τιμή που υποδεικνύει τη σχετική θέση των τιμών στη γραμμή χρόνου.

```cpp
constexpr int System::DateTime::CompareTo(DateTime value) const
```

### Arguments

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| value | [DateTime](../) | Ένα αντικείμενο της κλάσης [DateTime](../) για σύγκριση με το τρέχον αντικείμενο |

### Τιμή Επιστροφής

Μια τιμή μικρότερη από 0 εάν το τρέχον αντικείμενο αντιπροσωπεύει μια τιμή που είναι νωρίτερη από αυτήν που αντιπροσωπεύεται από **value**· 0 εάν οι τιμές που αντιπροσωπεύονται και από τα δύο αντικείμενα είναι ίδιες· τιμή μεγαλύτερη από 0 εάν η τιμή που αντιπροσωπεύεται από το τρέχον αντικείμενο είναι μεταγενέστερη από αυτήν που αντιπροσωπεύεται από **value**.

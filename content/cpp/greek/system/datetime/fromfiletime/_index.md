## DateTime::FromFileTime(int64_t) μέθοδος

Μετατρέπει το καθορισμένο File time σε ένα αντικείμενο της κλάσης [DateTime](../) που αντιπροσωπεύει την ίδια ημερομηνία και ώρα όπως η τοπική ώρα.

```cpp
static DateTime System::DateTime::FromFileTime(int64_t value)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| value | **int64_t** | File time to convert |

### Τιμή Επιστροφής

Μια νέα παρουσία της κλάσης [DateTime](../) που αντιπροσωπεύει την ίδια ημερομηνία και ώρα όπως **file_time**

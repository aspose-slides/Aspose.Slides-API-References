## Details_InvalidProgramException κλάση

Η InvalidProgramException υπάρχει μόνο για λόγους συμβατότητας. Μην δημιουργείτε ποτέ αντικείμενα αυτής της κλάσης χειροκίνητα. Χρησιμοποιήστε την κλάση InvalidProgramException αντί αυτού. Μην τυλίγετε ποτέ τα στιγμιότυπα της κλάσης InvalidProgramException στο [System::SmartPtr](../smartptr/).

```cpp
class Details_InvalidProgramException : public System::Details_Exception
```

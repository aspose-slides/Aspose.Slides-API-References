## Details_ArgumentException κλάση


ArgumentException ρίχνεται όταν ένα επιχείρημα που πέρασε σε μια μέθοδο που εκτελείται είναι άκυρο. Ποτέ μην δημιουργείτε στιγμιότυπα αυτής της κλάσης χειροκίνητα. Χρησιμοποιήστε την κλάση ArgumentException αντί για αυτήν. Ποτέ μην ενσωματώνετε στιγμιότυπα της κλάσης ArgumentException στο [System::SmartPtr](../smartptr/).

```cpp
class Details_ArgumentException : public System::Details_SystemException
```

## Παρατηρήσεις


ArgumentNullException ρίχνεται όταν μια μέθοδος που εκτελείται λαμβάνει ένα null επιχείρημα ενώ η μέθοδος αναμένει μια μη-null τιμή. Ποτέ μην δημιουργείτε στιγμιότυπα αυτής της κλάσης χειροκίνητα. Χρησιμοποιήστε την κλάση ArgumentNullException αντί για αυτήν. Ποτέ μην ενσωματώνετε στιγμιότυπα της κλάσης ArgumentNullException στο [System::SmartPtr](../smartptr/).

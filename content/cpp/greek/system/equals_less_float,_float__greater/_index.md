## System::Equals< float, float >(const float\&, const float\&) συνάρτηση

Ειδίκευση για τιμές κινητής υποδιαστολής μονής ακρίβειας. Παρόλο που δύο NaN κινητής υποδιαστολής ορίζονται από το IEC 60559:1989 να συγκρίνονται πάντα ως διαφορετικά, η σύμβαση για [System.Object.Equals](../object/equals/) απαιτεί οι υπερφορτώσεις να πληρούν τις απαιτήσεις για έναν τελεστή ισοδυναμίας. Ως εκ τούτου, το System.Double.Equals και το System.Single.Equals επιστρέφουν True όταν συγκρίνονται δύο NaN, ενώ ο τελεστής ισότητας επιστρέφει False σε αυτήν την περίπτωση, όπως απαιτεί το πρότυπο.

```cpp
bool System::Equals<float, float>(const float &a, const float &b)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| a | const **float**\& | Ο πρώτος συγκριτέος |
| b | const **float**\& | Ο δεύτερος συγκριτέος |

### Τιμή Επιστροφής

True εάν και οι δύο τιμές είναι NaN ή είναι ίσες, διαφορετικά - false

## Παρατηρήσεις

```cpp
#include "system/console.h"
#include <array>

int main()
{
  using namespace System;

  // Εκτυπώνει το μήνυμα καλωσορίσματος.
  Console::WriteLine(u"Hello, world!");

  // Δημιουργεί μια εμφάνιση της κλάσης 'std::array'.
  std::array<int, 5> arr = {1, 2, 3, 4, 5};

  // Εκτυπώνει τα στοιχεία του πίνακα.
  for (auto el: arr)
  {
    Console::Write(u"{0} ", el);
  }
  Console::WriteLine();

  return 0;
}
/*
Αυτό το παράδειγμα κώδικα παράγει την ακόλουθη έξοδο:
Γειά σου, κόσμε!
1 2 3 4 5
*/
```

## Details_ApplicationException κλάση

Μια βασική κλάση για κλάσεις που αντιπροσωπεύουν εξαιρέσεις εφαρμογής (αντί για συστήματος). Μην δημιουργείτε ποτέ αντικείμενα αυτής της κλάσης χειροκίνητα. Χρησιμοποιήστε την κλάση ApplicationException ως εναλλακτική. Μην περιβάλλετε ποτέ τις εμφανίσεις της κλάσης ApplicationException στο [System::SmartPtr](../smartptr/).

```cpp
class Details_ApplicationException : public System::Details_Exception
```

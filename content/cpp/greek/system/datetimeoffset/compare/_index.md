## DateTimeOffset::Compare(const DateTimeOffset\&, const DateTimeOffset\&) μέθοδος

Συγκρίνει δύο [DateTimeOffset](../) αντικείμενα.

```cpp
static int System::DateTimeOffset::Compare(const DateTimeOffset &first, const DateTimeOffset &second)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| first | const [DateTimeOffset](../)\& | Πρώτο αντικείμενο προς σύγκριση. |
| second | const [DateTimeOffset](../)\& | Δεύτερο αντικείμενο προς σύγκριση. |

### Τιμή Επιστροφής

- 1 - εάν **first** είναι νωρίτερα από **second**, 1 - εάν **first** είναι αργότερα από **second**, 0 - εάν **first** είναι ίσο με **second**.

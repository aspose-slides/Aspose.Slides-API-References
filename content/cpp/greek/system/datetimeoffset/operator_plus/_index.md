## DateTimeOffset::operator+(TimeSpan) const μέθοδος

Επιστρέφει μια νέα παρουσία της κλάσης [DateTimeOffset](../) που αντιπροσωπεύει την τιμή ημερομηνίας και ώρας που είναι το άθροισμα της τιμής που αντιπροσωπεύεται από το τρέχον αντικείμενο και του καθορισμένου χρονικού διαστήματος.

```cpp
DateTimeOffset System::DateTimeOffset::operator+(TimeSpan value) const
```

### Παράμετροι

| Parameter | Type | Description |
| --- | --- | --- |
| value | [TimeSpan](../../timespan/) | Το χρονικό διάστημα που θα προστεθεί στην τιμή που αντιπροσωπεύεται από το τρέχον αντικείμενο |

### Τιμή Επιστροφής

Μια νέα παρουσία της κλάσης [DateTimeOffset](../) που αντιπροσωπεύει την τιμή ημερομηνίας και ώρας που είναι το άθροισμα της τιμής που αντιπροσωπεύεται από το τρέχον αντικείμενο και **value**.

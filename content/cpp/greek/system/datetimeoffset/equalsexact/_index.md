## DateTimeOffset::EqualsExact(const DateTimeOffset\&) const μέθοδος

Ελέγχει εάν δύο [DateTimeOffset](../) αντικείμενα αντιπροσωπεύουν το ίδιο χρονικό σημείο και έχουν την ίδια μετατόπιση.

```cpp
bool System::DateTimeOffset::EqualsExact(const DateTimeOffset &other) const
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| other | const [DateTimeOffset](../)\& | [Object](../../object/) για σύγκριση. |

### Τιμή Επιστροφής

true αν το τρέχον αντικείμενο είναι ακριβώς ίσο με **other**, διαφορετικά - false.

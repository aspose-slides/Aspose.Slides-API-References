## CookieVariant enum

Απαριθμεί τις προδιαγραφές των cookie.

```cpp
enum class CookieVariant
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Unknown | 0 | Η άγνωστη προδιαγραφή cookie. |
| Plain | 1 | Η απλή προδιαγραφή cookie. |
| Rfc2109 | 2 | Η προδιαγραφή cookie 'RFC 2109'. |
| Rfc2965 | 3 | Η προδιαγραφή cookie 'RFC 2965'. |
| Default | 2 | Η προεπιλεγμένη τιμή ίση με 'Rfc2109'. |

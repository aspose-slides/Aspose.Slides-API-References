## HeaderVariantInfo κλάση

Χρησιμοποιείται για την αντιστοίχιση του ονόματος και της προδιαγραφής του cookie. Αντικείμενα αυτής της κλάσης πρέπει να εκχωρούνται μόνο χρησιμοποιώντας τη συνάρτηση [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε εμφάνιση αυτού του τύπου στο στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς αυτό θα οδηγούσες σε σφάλματα χρόνου εκτέλεσης και/ή σφάλματα επαλήθευσης. Πάντα τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για τη μεταβίβαση σε συναρτήσεις ως όρισμα.

```cpp
class HeaderVariantInfo : public System::Object
```

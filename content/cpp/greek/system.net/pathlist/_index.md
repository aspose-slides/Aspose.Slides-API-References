## PathList κλάση

Αναπαριστά τη λίστα των αντικειμένων κλάσης [CookieCollection](../cookiecollection/). Τα αντικείμενα αυτής της κλάσης θα πρέπει να εκχωρούνται μόνο χρησιμοποιώντας τη συνάρτηση [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε αντικείμενο αυτού του τύπου στη στοίβα ή χρησιμοποιώντας τον operator new, καθώς θα προκύψουν σφάλματα χρόνου εκτέλεσης και/ή σφαλμάτων ελέγχου. Πάντα τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να το περάσετε σε συναρτήσεις ως παράμετρο.

```cpp
class PathList : public System::Object
```

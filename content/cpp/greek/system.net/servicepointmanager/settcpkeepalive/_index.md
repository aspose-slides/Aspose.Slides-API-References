## ServicePointManager::SetTcpKeepAlive(bool, int32_t, int32_t) μέθοδος

Ορίζει την τιμή που υποδεικνύει εάν η επιλογή 'Keep-Alive' είναι ενεργοποιημένη.

```cpp
static void System::Net::ServicePointManager::SetTcpKeepAlive(bool enabled, int32_t keepAliveTime, int32_t keepAliveInterval)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| enabled | **bool** | Μια τιμή που υποδεικνύει εάν η επιλογή 'Keep-Alive' είναι ενεργοποιημένη. |
| keepAliveTime | **int32_t** | Ένα χρονικό διάστημα σε χιλιοστά του δευτερολέπτου μετά το οποίο θα αποσταλεί το πρώτο πακέτο 'Keep-Alive'. |
| keepAliveInterval | **int32_t** | Ένα χρονικό διάστημα σε χιλιοστά του δευτερολέπτου μεταξύ της αποστολής των πακέτων 'Keep-Alive'. |

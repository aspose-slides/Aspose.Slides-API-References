## UriScheme κλάση

Αντιπροσωπεύει το σχήμα για URI. Αντικείμενα αυτής της κλάσης πρέπει να δεσμεύονται μόνο μέσω της συνάρτησης [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε μια περίπτωση αυτού του τύπου στο stack ή χρησιμοποιώντας τον τελεστή new, καθώς θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης και/ή σφάλματα assert. Πάντα να τυλίγετε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και να χρησιμοποιείτε αυτόν τον δείκτη για να την περάσετε ως όρισμα σε συναρτήσεις.

```cpp
class UriScheme : public System::Object
```

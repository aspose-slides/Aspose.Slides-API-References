## IPHostEntry κλάση

Αντιπροσωπεύει πληροφορίες για μια διεύθυνση κεντρικού υπολογιστή στο διαδίκτυο. Τα αντικείμενα αυτής της κλάσης πρέπει να κατανεμηθούν μόνο μέσω της συνάρτησης [System::MakeObject()](../../system/makeobject/). Ποτέ μη δημιουργείτε ένα στιγμιότυπο αυτού του τύπου στην στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς θα προκαλέσει σφάλματα εκτέλεσης και/ή σφάλματα ελέγχου. Πάντα τυλίγετε αυτή τη κλάση σε έναν δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να τη περάσετε σε συναρτήσεις ως όρισμα.

```cpp
class IPHostEntry : public System::Object
```

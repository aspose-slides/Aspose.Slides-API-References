## HttpWebResponse κλάση


Αναπαριστά την απάντηση HTTP web. Τα αντικείμενα αυτής της κλάσης πρέπει να εκχωρούνται μόνο χρησιμοποιώντας τη συνάρτηση [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε στιγμιότυπο αυτού του τύπου στη στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς θα οδηγήσει σε σφάλματα εκτέλεσης και/ή σφάλματα ισχυρισμού. Πάντοτε τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να την περάσετε στις συναρτήσεις ως όρισμα.

```cpp
class HttpWebResponse : public System::Net::WebResponse
```

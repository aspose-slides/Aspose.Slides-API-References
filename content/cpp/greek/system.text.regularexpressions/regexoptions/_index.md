## RegexOptions enum


[Regex](../regex/) επιλογές.

```cpp
enum class RegexOptions
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| None | 0 | Προεπιλεγμένη συμπεριφορά. |
| Compiled | 1 | Μεταγλώττιση regex για απόδοση. Πάντα γίνεται από προεπιλογή. |
| CultureInvariant | 2 | Χρήση αντιστοίχισης ανεξάρτητης από πολιτισμό. Αγνοείται. |
| ECMAScript | 4 | Χρήση σύνταξης ECMAScript. Αγνοείται. |
| ExplicitCapture | 8 | Μόνο ρητή καταγραφή. Αγνοείται. |
| IgnoreCase | 16 | Παράβλεψη πεζών-κεφαλαίων κατά την αντιστοίχιση. |
| IgnorePatternWhitespace | 32 | Παράβλεψη κενών στο μοτίβο. Δεν υποστηρίζεται. |
| Multiline | 64 | Θεωρεί το '^' και το '$' ως αρχή και τέλος γραμμής, όχι ολόκληρης συμβολοακολουθίας. |
| RightToLeft | 128 | Αντιστοίχιση από δεξιά προς αριστερά. Δεν υποστηρίζεται. |
| Singleline | 256 | Κάνει το '.' να ταιριάζει με οποιοδήποτε χαρακτήρα χωρίς εξαιρέσεις (κανονικά, οι χαρακτήρες νέας γραμμής δεν ταιριάζουν). |

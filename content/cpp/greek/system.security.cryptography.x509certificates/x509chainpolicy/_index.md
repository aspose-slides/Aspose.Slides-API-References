## X509ChainPolicy κλάση

Η πολιτική αλυσίδας που θα εφαρμοστεί κατά την κατασκευή μιας αλυσίδας πιστοποιητικών X509. Τα αντικείμενα αυτής της κλάσης πρέπει να εκχωρούνται μόνο χρησιμοποιώντας τη συνάρτηση [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε παρουσία αυτού του τύπου στο stack ή χρησιμοποιώντας operator new, καθώς θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης και/ή σε σφάλματα ελέγχου. Πάντα να περιβάλλετε αυτή τη κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να το περάσετε σε συναρτήσεις ως παράμετρο.

```cpp
class X509ChainPolicy : public System::Object
```

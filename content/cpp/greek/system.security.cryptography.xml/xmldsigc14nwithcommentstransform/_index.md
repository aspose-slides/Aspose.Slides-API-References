## XmlDsigC14NWithCommentsTransform κλάση


Αντιπροσωπεύει τη μετασχηματισμό κανονικοποίησης XML C14N για ψηφιακή υπογραφή με σχόλια. Τα αντικείμενα αυτής της κλάσης πρέπει να εκχωρούνται μόνο χρησιμοποιώντας τη συνάρτηση [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε παρουσία αυτού του τύπου στη στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης και/ή σε σφάλματα ασsertion. Πάντα τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να τον περάσετε σε συναρτήσεις ως όρισμα.

```cpp
class XmlDsigC14NWithCommentsTransform : public System::Security::Cryptography::Xml::XmlDsigC14NTransform
```

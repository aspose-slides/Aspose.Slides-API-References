## XmlDsigExcC14NTransform κατηγορία

Αντιπροσωπεύει τον αποκλειστικό μετασχηματισμό κανονικοποίησης C14N XML για ψηφιακή υπογραφή χωρίς σχόλια. Αντικείμενα αυτής της κατηγορίας πρέπει να κατανεμηθούν μόνο χρησιμοποιώντας τη συνάρτηση [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε παρουσία αυτού του τύπου στην στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς θα προκαλέσει σφάλματα εκτέλεσης και/ή σφαλμάτων ελέγχου. Πάντα τυλίξτε αυτήν την κατηγορία σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να τον περάσετε στις συναρτήσεις ως όρισμα.

```cpp
class XmlDsigExcC14NTransform : public System::Security::Cryptography::Xml::Transform
```

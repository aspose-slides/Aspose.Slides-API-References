## SignedXml κλάση


Χρησιμοποιείται για υπογραφή και επαλήθευση XML. Τα αντικείμενα αυτής της κλάσης πρέπει να καταλαμβάνονται μόνο με τη συνάρτηση [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε στιγμιότυπο αυτού του τύπου στη στοίβα ή με τον τελεστή new, καθώς θα προκαλέσει σφάλματα χρόνου εκτέλεσης και/ή σφαλματικές συναρτήσεις. Πάντα περικλείετε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να τη μεταβιβάσετε σε συναρτήσεις ως όρισμα.

```cpp
class SignedXml : public System::Object
```

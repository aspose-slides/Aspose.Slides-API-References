## Κλάσεις

| Κλάση | Περιγραφή |
| --- | --- |
| [Evidence](./evidence/) | Stub κλάση για μεταφρασμένο κώδικα που χρησιμοποιεί την κλάση [Evidence](./evidence/) για μεταγλώττιση. Αντικείμενα αυτής της κλάσης θα πρέπει να εκχωρούνται μόνο χρησιμοποιώντας τη συνάρτηση [System::MakeObject()](../system/makeobject/). Ποτέ μην δημιουργήσετε αντίτυπο αυτού του τύπου στη στοίβα ή χρησιμοποιώντας operator new, καθώς θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης και/ή σφάλματα ελέγχου. Πάντα τυλίξτε αυτή την κλάση σε δείκτη [System::SmartPtr](../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να το περάσετε σε συναρτήσεις ως όρισμα. |

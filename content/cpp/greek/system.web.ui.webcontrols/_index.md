## Κλάσεις

| Κλάση | Περιγραφή |
| --- | --- |
| [DataGrid](./datagrid/) | Ψεύδος κλάση που καθιστά δυνατή τη σύνδεση του μεταγλωτισμένου κώδικα με τις αναφορές [DataGrid](./datagrid/), αλλά δεν την εκτελεί. Δεν περιέχει σωστά υλοποιημένα μέλη. Τα αντικείμενα αυτής της κλάσης πρέπει να εκχωρούνται μόνο με τη λειτουργία [System::MakeObject()](../system/makeobject/). Ποτέ μην δημιουργείτε στιγμιότυπο αυτού του τύπου στη στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης και/ή αποτυχίες ελέγχου. Πάντα να τυλίγετε αυτήν την κλάση σε δείκτη [System::SmartPtr](../system/smartptr/) και να χρησιμοποιείτε αυτόν τον δείκτη για να τη μεταβιβάζετε σε συναρτήσεις ως όρισμα. |

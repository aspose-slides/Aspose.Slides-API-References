## DataGrid κλάση

Dummy κλάση making it possible linking translated code with [DataGrid](./) references, but not executing it. Contains no properly implemented members. Objects of this κλάση should only be allocated using [System::MakeObject()](../../system/makeobject/) function. Never create instance of this type on stack or using operator new, as it will result in runtime errors and/or assertion faults. Always wrap this κλάση into [System::SmartPtr](../../system/smartptr/) pointer and use this pointer to pass it to functions as argument.

```cpp
class DataGrid : public System::Object
```

## DbConnection κλάση

Σύνδεση βάσης δεδομένων. Τα αντικείμενα αυτής της κλάσης πρέπει να εκχωρούνται μόνο χρησιμοποιώντας τη [System::MakeObject()](../../system/makeobject/) συνάρτηση. Ποτέ μην δημιουργείτε αντίγραφο αυτού του τύπου στο stack ή χρησιμοποιώντας operator new, καθώς θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης και/ή σε assertion faults. Πάντα να τυλίγετε αυτήν την κλάση σε [System::SmartPtr](../../system/smartptr/) δείκτη και να χρησιμοποιείτε αυτόν τον δείκτη για να τον περάσετε σε functions ως όρισμα.

```cpp
class DbConnection : public System::Object
```

## LinearGradientMode enum

Αντιπροσωπεύει μια διεύθυνση γραμμικού gradient.

```cpp
enum class LinearGradientMode
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Horizontal | 0 | Αριστερά προς δεξιά. |
| Vertical | 1 | Κάτω προς πάνω. |
| ForwardDiagonal | 2 | Από την επάνω αριστερή γωνία στην κάτω δεξιά γωνία. |
| BackwardDiagonal | 3 | Από την επάνω δεξιά γωνία στην κάτω αριστερή γωνία. |

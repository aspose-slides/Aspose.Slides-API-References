## SmoothingMode enum

Καθορίζει τον τύπο της εξομάλυνσης (antialiasing) που εφαρμόζεται σε γραμμές, καμπύλες και άκρα γεμισμένων περιοχών.

```cpp
enum class SmoothingMode
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Invalid | n/a | Μη έγκυρη τιμή. |
| Default | n/a | Προεπιλεγμένη λειτουργία, χωρίς εξομάλυνση. |
| HighSpeed | n/a | Χωρίς εξομάλυνση, υψηλή ταχύτητα. |
| HighQuality | n/a | Απόδοση με εξομάλυνση, χαμηλή ταχύτητα. |
| None | n/a | Χωρίς εξομάλυνση. |
| AntiAlias | n/a | Απόδοση με εξομάλυνση. |

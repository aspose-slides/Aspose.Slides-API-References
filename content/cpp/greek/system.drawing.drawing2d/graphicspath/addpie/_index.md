## GraphicsPath::AddPie(float, float, float, float, float, float) method


Προσθέτει το καθορισμένο περίγραμμα του σχήματος πίτας στο μονοπάτι που αντιπροσωπεύει το τρέχον αντικείμενο.

```cpp
void System::Drawing::Drawing2D::GraphicsPath::AddPie(float x, float y, float width, float height, float startAngle, float sweepAngle)
```


### Arguments

| Parameter | Type | Description |
| --- | --- | --- |
| x | **float** | Η συντεταγμένη X της άνω αριστερής γωνίας του ορθογωνίου που περιβάλλει την έλλειψη από την οποία σχεδιάζεται η πίτα |
| y | **float** | Η συντεταγμένη Y της άνω αριστερής γωνίας του ορθογωνίου που περιβάλλει την έλλειψη από την οποία σχεδιάζεται η πίτα |
| width | **float** | Το πλάτος της άνω αριστερής γωνίας του ορθογωνίου που περιβάλλει την έλλειψη από την οποία σχεδιάζεται η πίτα |
| height | **float** | Το ύψος της άνω αριστερής γωνίας του ορθογωνίου που περιβάλλει την έλλειψη από την οποία σχεδιάζεται η πίτα |
| startAngle | **float** | Καθορίζει τη γωνία έναρξης της πίτας σε μοίρες, μετρώντας δεξιόστροφα από τον άξονα X |
| sweepAngle | **float** | Καθορίζει τη γωνία μεταξύ της γωνίας έναρξης και του τέλους της πίτας |

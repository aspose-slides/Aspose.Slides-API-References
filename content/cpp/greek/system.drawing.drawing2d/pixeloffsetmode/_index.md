## PixelOffsetMode enum

Specifies how pixels are offset when being rendered.

```cpp
enum class PixelOffsetMode
```

### Values

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Invalid | n/a | Μη έγκυρη τιμή. |
| Default | n/a | Η προεπιλεγμένη λειτουργία. |
| HighSpeed | n/a | Λειτουργία χαμηλής ποιότητας (υψηλή απόδοση). |
| HighQuality | n/a | Λειτουργία υψηλής ποιότητας (χαμηλότερη απόδοση). |
| None | n/a | Χωρίς μετατόπιση εικονοστοιχείου. |
| Half | n/a | Τα εικονοστοιχεία μετατοπίζονται κατά -0.5 μονάδες οριζόντια και κάθετα για υψηλής απόδοσης εξομάλυνση. |

## LineCap enum

Δείχνει τα διαθέσιμα στυλ άκρων με τα οποία ένα αντικείμενο [Pen](../../system.drawing/pen/) μπορεί να τερματίσει μια γραμμή.

```cpp
enum class LineCap
```

### Values

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Flat | 0 | Άκρο γραμμής Flat. |
| Square | 1 | Άκρο γραμμής Square. |
| Round | 2 | Άκρο γραμμής Round. |
| Triangle | 3 | Άκρο γραμμής Triangle. |
| NoAnchor | 16 | Χωρίς άγκυρα. |
| SquareAnchor | 17 | Άκρο γραμμής SquareAnchor. |
| RoundAnchor | 18 | Άκρο γραμμής RoundAnchor. |
| DiamondAnchor | 19 | Άκρο γραμμής DiamondAnchor. |
| ArrowAnchor | 20 | Άκρο γραμμής ArrowAnchor. |
| AnchorMask | 240 | Μάσκα που χρησιμοποιείται για τον έλεγχο εάν ένα άκρο γραμμής είναι άγκυρα. |
| Custom | 255 | Προσαρμοσμένο άκρο. |

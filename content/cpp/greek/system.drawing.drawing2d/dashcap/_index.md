## DashCap enum

Καθορίζει τον τύπο του καπακιού που χρησιμοποιείται και στα δύο άκρα μιας παύλης σε μια διακεκομμένη γραμμή.

```cpp
enum class DashCap
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Flat | 0 | Ένα τετράγωνο καπάκι που τετραγωνίζει και στα δύο άκρα μιας παύλης. |
| Round | 2 | Ένα κυκλικό καπάκι που στρογγυλοποιεί και στα δύο άκρα μιας παύλης. |
| Triangle | 3 | Ένα τριγωνικό καπάκι που δείχνει και στα δύο άκρα μιας παύλης. |

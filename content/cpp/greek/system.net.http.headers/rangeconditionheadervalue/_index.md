## RangeConditionHeaderValue κλάση

Αντιπροσωπεύει μια τιμή της κεφαλίδας 'If-Range'. Τα αντικείμενα αυτής της κλάσης πρέπει να δημιουργούνται μόνο μέσω της λειτουργίας [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε ένα στιγμιότυπο αυτού του τύπου στο stack ή χρησιμοποιώντας τον τελεστή new, καθώς θα προκύψουν σφάλματα χρόνου εκτέλεσης και/ή σφάλματα αναφοράς. Πάντα περικλείετε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιείτε αυτόν τον δείκτη για να το περάσετε στις λειτουργίες ως όρισμα.

```cpp
class RangeConditionHeaderValue : public System::ICloneable
```

## HttpResponseHeaders κλάση

Αντιπροσωπεύει τη συλλογή των επικεφαλίδων «Response». Αντικείμενα αυτής της κλάσης θα πρέπει να δημιουργούνται μόνο χρησιμοποιώντας τη λειτουργία [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε στιγμιότυπο αυτού του τύπου στη στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς θα προκαλέσει σφάλματα χρόνου εκτέλεσης και/ή αποτυχίες ελέγχου. Πάντα τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να την περάσετε στις συναρτήσεις ως όρισμα.

```cpp
class HttpResponseHeaders : public System::Net::Http::Headers::HttpHeaders
```

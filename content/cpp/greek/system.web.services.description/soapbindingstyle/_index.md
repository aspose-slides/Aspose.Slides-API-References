## SoapBindingStyle enum

Καθορίζει τον τύπο ενέργειας στην XML [Web](../../system.web/) υπηρεσία στο επίπεδο ιεραρχίας κλάσεων στην οποία αυτή η αρίθμηση εφαρμόζεται.

```cpp
enum class SoapBindingStyle
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Default | 0 | Ο προεπιλεγμένος τύπος ενέργειας. |
| Document | 1 | Το μήνυμα είναι προσανατολισμένο σε έγγραφα. |
| Rpc | 2 | Το μήνυμα περιέχει τις παραμέτρους για την κλήση μιας διαδικασίας ή τις τιμές επιστροφής εκείνης της διαδικασίας. |

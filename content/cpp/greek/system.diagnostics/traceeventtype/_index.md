## TraceEventType enum

Προσδιορίζει τον τύπο του γεγονότος που προκάλεσε την καταγραφή.

```cpp
enum class TraceEventType
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Critical | 1 | Σοβαρό σφάλμα ή κατάρρευση της εφαρμογής. |
| Error | 2 | Ανακτήσιμο σφάλμα. |
| Warning | 4 | Μη κρίσιμο πρόβλημα. |
| Information | 8 | Πληροφοριακό μήνυμα. |
| Verbose | 16 | Καταγραφή εντοπισμού σφαλμάτων. |
| Start | 256 | Έναρξη λογικής λειτουργίας. |
| Stop | 512 | Παύση λογικής λειτουργίας. |
| Suspend | 1024 | Αναστολή λογικής λειτουργίας. |
| Resume | 2048 | Συνέχιση λογικής λειτουργίας. |
| Transfer | 4096 | Αλλαγή ταυτότητας συσχέτισης. |

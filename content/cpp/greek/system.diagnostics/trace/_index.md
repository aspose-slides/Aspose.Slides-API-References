## Δομή Trace


Παρέχει διεπαφή για πρόσβαση στο debugger trace (αν υπάρχει). Λειτουργεί μόνο σε λειτουργία [Debug](../debug/). Αυτός είναι ένας στατικός τύπος χωρίς υπηρεσίες στιγμιοτύπων. Δεν πρέπει ποτέ να δημιουργήσετε στιγμιότυπα αυτού με κανέναν τρόπο.

```cpp
class Trace
```

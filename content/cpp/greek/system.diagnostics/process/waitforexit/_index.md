## Process::WaitForExit(int) μέθοδος

Περιμένει τη διαδικασία να τερματιστεί. Δεν έχει υλοποιηθεί.

```cpp
bool System::Diagnostics::Process::WaitForExit(int milliseconds)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| milliseconds | int | Μέγιστη καθυστέρηση αναμονής. |

### Τιμή Επιστροφής

True αν η διαδικασία ολοκληρωθεί, false αν το χρονικό όριο ξεπεραστεί.

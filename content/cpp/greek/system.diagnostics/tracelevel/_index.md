## TraceLevel enum

Καθορίζει ποια μηνύματα να εμφανίζονται για τις κλάσεις [System.Diagnostics.Debug](../debug/), [System.Diagnostics.Trace](../trace/) και System.Diagnostics.TraceSwitch classes.

```cpp
enum class TraceLevel
```

### Values

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Off | 0 | Δεν εμφανίζει μηνύματα εντοπισμού και αποσφαλμάτωσης. |
| Error | 1 | Εμφανίζει μηνύματα διαχείρισης σφαλμάτων. |
| Warning | 2 | Εμφανίζει προειδοποιήσεις και μηνύματα διαχείρισης σφαλμάτων. |
| Info | 3 | Εμφανίζει πληροφοριακά μηνύματα, προειδοποιήσεις και μηνύματα διαχείρισης σφαλμάτων. |
| Verbose | 4 | Εμφανίζει όλα τα μηνύματα αποσφαλμάτωσης και εντοπισμού. |

## CompareInfo::IsSortable(char16_t) μέθοδος

Ελέγχει εάν ένας συγκεκριμένος χαρακτήρας είναι ταξινομήσιμος.

```cpp
static bool System::Globalization::CompareInfo::IsSortable(char16_t ch)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| ch | char16_t | Χαρακτήρας Unicode. |

### Τιμή Επιστροφής

True αν **ch** είναι ταξινομήσιμος· διαφορετικά false.

## CompareInfo::IsSortable(const String\&) μέθοδος

Ελέγχει εάν μια συγκεκριμένη συμβολοσειρά είναι ταξινομήσιμη.

```cpp
static bool System::Globalization::CompareInfo::IsSortable(const String &text)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| text | const [String](../../../system/string/)\& | Μια συμβολοσειρά. |

### Τιμή Επιστροφής

True αν **text** δεν είναι κενό και όλοι οι χαρακτήρες στο **text** είναι ταξινομήσιμοι· διαφορετικά false.

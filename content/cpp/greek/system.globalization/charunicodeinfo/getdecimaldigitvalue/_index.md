## CharUnicodeInfo::GetDecimalDigitValue(char16_t) μέθοδος


Λαμβάνει την δεκαδική τιμή ψηφίου του καθορισμένου χαρακτήρα.

```cpp
static int System::Globalization::CharUnicodeInfo::GetDecimalDigitValue(char16_t ch)
```


### Παράμετροι

| Parameter | Type | Description |
| --- | --- | --- |
| ch | char16_t | Unicode χαρακτήρας. |

### Τιμή Επιστροφής

Η δεκαδική τιμή ψηφίου ή -1 εάν ο καθορισμένος χαρακτήρας δεν είναι δεκαδικό ψηφίο.

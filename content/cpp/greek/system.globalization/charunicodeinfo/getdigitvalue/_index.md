## CharUnicodeInfo::GetDigitValue(char16_t) μέθοδος

Λαμβάνει την αριθμητική τιμή του συγκεκριμένου χαρακτήρα.

```cpp
static int System::Globalization::CharUnicodeInfo::GetDigitValue(char16_t ch)
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| ch | char16_t | Unicode χαρακτήρας. |

### Τιμή Επιστροφής

Η αριθμητική τιμή ή -1 αν ο συγκεκριμένος χαρακτήρας δεν είναι ψηφίο.

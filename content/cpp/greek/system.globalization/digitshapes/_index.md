## DigitShapes enum

Καθορίζει την εμφάνιση των ψηφίων.

```cpp
enum class DigitShapes : int32_t
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Context | 0 | Το σχήμα του ψηφίου εξαρτάται από το προηγούμενο κείμενο. |
| None | 1 | Το σχήμα του ψηφίου δεν αλλάζει. |
| NativeNational | 2 | Τα ψηφία ASCII 0..9 αντικαθίστανται από εγχώρια εθνικά ψηφία. |

## NumberFormatInfo::get_PercentGroupSizes() const μέθοδος


Λαμβάνει αριθμούς ψηφίων ανά ομάδα τιμής ποσοστού.

```cpp
ArrayPtr<int> System::Globalization::NumberFormatInfo::get_PercentGroupSizes() const
```


### Τιμή επιστροφής

[Array](../../../system/array/) ψηφίων ανά ομάδα, από αριστερά προς δεξιά; κάθε στοιχείο πρέπει να είναι 1 έως 9, το τελευταίο μπορεί να είναι 0 που σημαίνει "combine all"; το τελευταίο στοιχείο επαναλαμβάνεται.

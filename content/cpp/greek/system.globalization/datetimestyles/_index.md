## DateTimeStyles enum

Καθορίζει επιλογές μορφοποίησης ημερομηνίας και ώρας. Δυαδικές σημαίες.

```cpp
enum class DateTimeStyles : int32_t
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| None | 0 | Προεπιλογή. |
| AllowLeadingWhite | 1 | Παράβλεψη αρχικών κενών διαστημάτων. |
| AllowTrailingWhite | 2 | Παράβλεψη τελικών κενών διαστημάτων. |
| AllowInnerWhite | 4 | Παράβλεψη εσωτερικών κενών διαστημάτων. |
| AllowWhiteSpaces | n/a | Παράβλεψη όλων των κενών διαστημάτων. |
| NoCurrentDateDefault | 8 | Κατά την ανάλυση μιας συμβολοσειράς ημερομηνίας/ώρας, εάν λείπουν όλα τα έτος/μήνας/ημέρα, ορίστε την προεπιλεγμένη ημερομηνία στο 0001/1/1, αντί για το τρέχον έτος/μήνας/ημέρα. |
| AdjustToUniversal | 16 | Κατά την ανάλυση μιας συμβολοσειράς ημερομηνίας/ώρας, εάν υπάρχει προσδιοριστής ζώνης ώρας ("GMT","Z","+xxxx", "-xxxx"), θα προσαρμόσουμε τον αναλυθέντα χρόνο βασισμένο στο GMT. |
| AssumeLocal | 32 | Εάν δεν δοθεί ζώνη ώρας, χρησιμοποιήστε την τοπική ζώνη ώρας. |
| AssumeUniversal | 64 | Εάν δεν δοθεί ζώνη ώρας, χρησιμοποιήστε UTC. |
| RoundtripKind | 128 | Προσπαθήστε να διατηρήσετε αν η είσοδος είναι ακαθόριστη, τοπική ή UTC. |

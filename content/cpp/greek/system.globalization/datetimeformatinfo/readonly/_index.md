## DateTimeFormatInfo::ReadOnly(const DateTimeFormatInfoPtr\&) μέθοδος


Λαμβάνει την έκδοση μόνο για ανάγνωση του μορφοποιητή.

```cpp
static DateTimeFormatInfoPtr System::Globalization::DateTimeFormatInfo::ReadOnly(const DateTimeFormatInfoPtr &info)
```


### Arguments

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| info | const [DateTimeFormatInfoPtr](../../datetimeformatinfoptr/)\& | Formatter to get read-only version of. |

### Return Value

if **info** is read-only, returns it, otherwise creates a copy of it and marks it as read-only.

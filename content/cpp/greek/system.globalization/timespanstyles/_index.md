## TimeSpanStyles enum

Καθορίζει τις επιλογές μορφοποίησης που προσαρμόζουν την ανάλυση συμβολοσειράς για μεθόδους που μετατρέπουν μια αναπαράσταση συμβολοσειράς ενός χρονικού διαστήματος σε αντικείμενο Timespan.

```cpp
enum class TimeSpanStyles : int32_t
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| None | 0 | Το χρονικό διάστημα που αντιπροσωπεύεται από τη δοσμένη συμβολοσειρά ερμηνεύεται ως αρνητικό μόνο εάν υπάρχει αρνητικό πρόσημο. |
| AssumeNegative | 1 | Το χρονικό διάστημα που αντιπροσωπεύεται από τη δοσμένη συμβολοσειρά ερμηνεύεται πάντα ως αρνητικό. |

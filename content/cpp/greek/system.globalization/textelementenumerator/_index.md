## TextElementEnumerator κλάση


Απαριθμητής για την επανάληψη στοιχείων της συμβολοσειράς (χαρακτήρες). Τα αντικείμενα αυτής της κλάσης πρέπει να εκχωρούνται μόνο με τη λειτουργία [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε αντικείμενο αυτού του τύπου στην στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης και/ή σφάλματα ελέγχου. Πάντα τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να τη μεταβιβάσετε στις συναρτήσεις ως όρισμα.

```cpp
class TextElementEnumerator : public virtual System::Object
```

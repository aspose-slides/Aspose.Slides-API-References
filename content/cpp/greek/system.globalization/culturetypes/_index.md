## CultureTypes enum

Καταχωρίσεις bitmask κατηγοριών πολιτισμού.

```cpp
enum class CultureTypes : int32_t
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| NeutralCultures | 1 | Πολιτισμός που είναι συγκεκριμένος για τη γλώσσα αλλά όχι για χώρα ή περιοχή. |
| SpecificCultures | 2 | Συγκεκριμένος για χώρα/περιοχή. |
| InstalledWin32Cultures | 4 | Πολιτισμοί που εγκαθίστανται στο OS. |
| AllCultures | n/a | Όλοι οι διαθέσιμοι πολιτισμοί. |
| UserCustomCulture | 8 | Πολιτισμοί που ορίζονται από τον χρήστη. |
| ReplacementCultures | 16 | Αντικαταστάσεις που ορίζονται από τον χρήστη για υπάρχοντες πολιτισμούς. |
| WindowsOnlyCultures | 32 | Παρωχημένο. |
| FrameworkCultures | 64 | Παρωχημένο. |

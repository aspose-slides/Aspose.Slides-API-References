## EncoderFallbackBuffer κλάση

Παρέχει buffer για υλοποίηση fallback. Τα αντικείμενα αυτής της κλάσης πρέπει να καταλαμβάνονται μόνο μέσω της συνάρτησης [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε στιγμιότυπο αυτού του τύπου στο stack ή χρησιμοποιώντας τον τελεστή new, καθώς θα οδηγήσουν σε σφάλματα χρόνου εκτέλεσης και/ή σε σφάλματα επιβεβαίωσης. Πάντα τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να τη μεταβιβάσετε σε συναρτήσεις ως όρισμα.

```cpp
class EncoderFallbackBuffer : public System::Object
```

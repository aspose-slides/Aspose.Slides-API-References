## UTF8Encoding::GetMaxByteCount(int) μέθοδος

Επιστρέφει τον μέγιστο αριθμό bytes που απαιτούνται για την κωδικοποίηση ενός συγκεκριμένου αριθμού χαρακτήρων.

```cpp
int System::Text::UTF8Encoding::GetMaxByteCount(int char_count) override
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| char_count | int | Αριθμός χαρακτήρων προς κωδικοποίηση. |

### Τιμή Επιστροφής

Αριθμός bytes που είναι επαρκής για την κωδικοποίηση οποιουδήποτε συμβολοσειράς με το δεδομένο μήκος χαρακτήρων.

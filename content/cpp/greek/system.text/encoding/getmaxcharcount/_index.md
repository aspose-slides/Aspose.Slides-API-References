## Encoding::GetMaxCharCount(int) μέθοδος

Λαμβάνει τον μέγιστο αριθμό χαρακτήρων που απαιτούνται για την αποκωδικοποίηση ενός συγκεκριμένου αριθμού byte.

```cpp
virtual int System::Text::Encoding::GetMaxCharCount(int byte_count)=0
```

### Παράμετροι

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| byte_count | int | Αριθμός byte για αποκωδικοποίηση. |

### Τιμή επιστροφής

Αριθμός byte που είναι επαρκής για την αποκωδικοποίηση οποιουδήποτε συμβολοσυνόλου με δεδομένο μήκος χαρακτήρων.

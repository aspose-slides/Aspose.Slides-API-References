## EncodingInfo κλάση

Σύντομες πληροφορίες κωδικοποίησης. Τα αντικείμενα αυτής της κλάσης θα πρέπει να καταλαμβάνονται μόνο με τη συνάρτηση [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε αντίγραφο αυτού του τύπου στην στοίβα ή χρησιμοποιώντας τον τελεστή new, επειδή θα προκαλέσει σφάλματα χρόνου εκτέλεσης και/ή σφάλματα επιβεβαίωσης. Πάντα τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν το δείκτη για να το περάσετε σε συναρτήσεις ως όρισμα.

```cpp
class EncodingInfo : public System::Object
```

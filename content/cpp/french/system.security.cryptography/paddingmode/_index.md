## PaddingMode énum

Définit comment traiter les messages qui sont plus courts que le bloc requis par l'opération cryptographique.

```cpp
enum class PaddingMode
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| None | 1 | Aucun remplissage. |
| PKCS7 | 2 | Utiliser la chaîne de remplissage PKCS #7. |
| Zeros | 3 | Utiliser des octets zéro. |
| ANSIX923 | 4 | Utiliser la chaîne de remplissage ANSIX923. |
| ISO10126 | 5 | Utiliser la chaîne de remplissage ISO10126. |

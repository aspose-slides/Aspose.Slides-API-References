## RSACryptoServiceProvider::get_UseMachineKeyStore() méthode


Vérifie si la clé persiste dans le magasin de la machine plutôt que dans le magasin de l'utilisateur.

```cpp
static bool System::Security::Cryptography::RSACryptoServiceProvider::get_UseMachineKeyStore()
```


### Valeur de retour

True si la clé doit être conservée dans le magasin de la machine, false si elle doit être conservée dans le magasin de l'utilisateur.

## StringCollection::crend() const méthode

Retourne un itérateur inversé vers l’élément qui suit le dernier élément du conteneur inversé. Il correspond à l’élément qui précède le premier élément du conteneur non inversé. Cet élément agit comme un espace réservé ; toute tentative d’accès entraîne un comportement indéfini.

```cpp
const_reverse_iterator System::Collections::Specialized::StringCollection::crend() const noexcept
```

### Valeur de retour

Un itérateur pointant vers l’élément théorique const-qualifié qui précède le premier élément du conteneur.

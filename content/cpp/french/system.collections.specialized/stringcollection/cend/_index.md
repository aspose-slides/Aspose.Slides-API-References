## StringCollection::cend() const method

Retourne un itérateur vers l'élément suivant le dernier élément du conteneur. Cet élément sert de repère; tenter d'y accéder entraîne un comportement indéfini.

```cpp
const_iterator System::Collections::Specialized::StringCollection::cend() const noexcept
```

### Valeur de retour

Un itérateur pointant vers l'élément théorique const-qualifié placé après le dernier élément du conteneur.

## StringCollection::crbegin() const méthode

Retourne un itérateur inverse vers le premier élément du conteneur inversé. Il correspond au dernier élément du conteneur non inversé. Si le conteneur est vide, l'itérateur retourné est égal à [crend()](../crend/).

```cpp
const_reverse_iterator System::Collections::Specialized::StringCollection::crbegin() const noexcept
```

### Valeur de retour

Un itérateur pointant vers le dernier élément const-qualifié du conteneur.

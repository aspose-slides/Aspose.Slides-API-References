## FileAttributes enum

Représente les attributs d'un répertoire ou d'un fichier.

```cpp
enum class FileAttributes
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Archive | 32 | L'entité du système de fichiers est un candidat à la suppression ou à la sauvegarde. |
| Compressed | 2048 | L'entité du système de fichiers est compressée. |
| Device | 64 | Non utilisé. |
| Directory | 16 | L'entité du système de fichiers est un répertoire. |
| Encrypted | 16384 | L'entité du système de fichiers est chiffrée. |
| Hidden | 2 | L'entité du système de fichiers est cachée. |
| IntegrityStream | 32768 | L'entité du système de fichiers contient un support d'intégrité des données. |
| Normal | 128 | L'entité du système de fichiers est un fichier qui ne possède aucun attribut spécial. |
| NotContentIndexed | 8192 | L'entité du système de fichiers ne sera pas indexée par le service d'indexation de contenu. |
| NoScrubData | 131072 | L'entité du système de fichiers n'est pas incluse dans l'analyse d'intégrité des données. |
| Offline | 4096 | L'entité du système de fichiers est hors ligne et ses données ne sont pas immédiatement disponibles. |
| ReadOnly | 1 | L'entité du système de fichiers est en lecture seule. |
| ReparsePoint | 1024 | L'entité du système de fichiers est un fichier qui contient un reparse point. |
| SparseFile | 512 | L'entité du système de fichiers est un fichier sparse. |
| System | 4 | L'entité du système de fichiers est un fichier système. |
| Temporary | 256 | L'entité du système de fichiers est un fichier temporaire. |
| Invalid | 0 | Valeur invalide. |

## Path::GetTempFileNameSafe() méthode


Crée un nouveau fichier avec un nom unique et renvoie un chemin complet vers celui-ci. C’est un synonyme de la méthode [GetTempFileName_()](../gettempfilename_/).

```cpp
static String System::IO::Path::GetTempFileNameSafe()
```


### Valeur de retour

Un chemin complet vers le fichier créé

## DirectoryInfo::Delete() méthode

Supprime le répertoire référencé par le chemin représenté par l'objet actuel si le répertoire est vide.

```cpp
virtual void System::IO::DirectoryInfo::Delete() override
```

## DirectoryInfo::Delete(bool) méthode

Supprime le répertoire référencé par le chemin représenté par l'objet actuel. Un paramètre indique si le contenu du répertoire doit être supprimé de façon récursive si le répertoire n'est pas vide.

```cpp
void System::IO::DirectoryInfo::Delete(bool recursive)
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| recursive | **bool** | Spécifie si le contenu du répertoire doit être supprimé de façon récursive s'il n'est pas vide |

## TextReader::Peek() méthode

Lit un seul caractère du flux sans modifier le curseur de lecture du flux.

```cpp
virtual int System::IO::TextReader::Peek()
```

### Valeur de retour

Caractère lu encodé en UTF-16 ; si le caractère lu est représenté par deux points de code en UTF-16, seul le surragate haut est retourné ; si aucun caractère n'a été lu, -1 est retourné

## Details_IOException::Is(const System::TypeInfo\&) const méthode

```cpp
bool System::IO::Details_IOException::Is(const System::TypeInfo &target) const override
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| target | const [System::TypeInfo](../../../system/typeinfo/)\& | [TypeInfo](../../../system/typeinfo/) structure décrivant le type à tester contre l'objet actuel. |

### Valeur de retour

True si l'objet est du type indiqué ou d'une sous-classe, false sinon.

## Remarques

Vérifie si l'objet représente une instance du type décrit par targetType. Analogue de l'opérateur C# 'is'.

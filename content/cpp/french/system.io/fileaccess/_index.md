## FileAccess enum

Spécifie le type d'accès lors de l'ouverture du fichier.

```cpp
enum class FileAccess
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Read | 1 | Accès en lecture seule. |
| Write | 2 | Accès en écriture seule. |
| ReadWrite | n/a | Accès en lecture et écriture. |

## Directory::Delete(const String\&, bool) méthode


Supprime le fichier ou le répertoire spécifié. Ne lève pas d'exception.

```cpp
static void System::IO::Directory::Delete(const String &path, bool recursive=false)
```


### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| path | const [String](../../../system/string/)\& | Le chemin du répertoire ou du fichier à supprimer |
| recursive | **bool** | Si **path** indique un répertoire non vide, alors **recursive** indique si tout le contenu du répertoire doit être supprimé récursivement ; si le répertoire indiqué par **path** n'est pas vide et que **recursive** vaut 'false', l'opération échoue |

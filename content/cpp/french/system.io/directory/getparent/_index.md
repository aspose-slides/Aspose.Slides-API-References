## Directory::GetParent(const String\&) méthode

Renvoie un pointeur partagé vers l'objet [DirectoryInfo](../../directoryinfo/) représentant le répertoire parent de l'entité spécifiée.


### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| path | const [String](../../../system/string/)\& | Le chemin de l'entité dont le parent doit être obtenu |

### Valeur de retour

Un pointeur partagé vers l'objet [DirectoryInfo](../../directoryinfo/) représentant le répertoire parent de l'entité spécifiée par **path**

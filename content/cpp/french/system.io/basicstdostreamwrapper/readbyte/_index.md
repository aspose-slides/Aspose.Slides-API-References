## BasicSTDOStreamWrapper::ReadByte() méthode


Si le mode d'encapsulation est binaire, lit un octet unique depuis le stockage du dernier caractère décodé, sinon lit un caractère unique depuis le flux et le convertit en type **uint8_t**. Non supporté!

```cpp
virtual int System::IO::BasicSTDOStreamWrapper<T, typename>::ReadByte() override
```


### Valeur de retour

Une valeur entière 32 bits équivalente au résultat de la lecture

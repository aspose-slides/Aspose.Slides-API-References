## Details_PathTooLongException::Is(const System::TypeInfo\&) const method

```cpp
bool System::IO::Details_PathTooLongException::Is(const System::TypeInfo &target) const override
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| target | const [System::TypeInfo](../../../system/typeinfo/)\& | [TypeInfo](../../../system/typeinfo/) structure décrivant le type contre lequel tester l'objet actuel. |

### Valeur de retour

true si l'objet est du type étiqueté ou de sa sous-classe, false sinon.

## Remarques

Vérifier si l'objet représente une instance du type décrit par targetType. Analogue de l'opérateur 'is' de C#.

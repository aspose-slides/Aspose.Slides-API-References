## BinaryWriter::Seek(int, System::IO::SeekOrigin) méthode


Définit la position du flux représenté par l'objet actuel.

```cpp
int64_t System::IO::BinaryWriter::Seek(int offset, System::IO::SeekOrigin origin=System::IO::SeekOrigin::Begin)
```


### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| offset | int | Le décalage en octets par rapport à une position spécifiée par **origin** |
| origin | [System::IO::SeekOrigin](../../seekorigin/) | Spécifie la position à partir de laquelle et la direction vers laquelle le décalage est calculé |

### Valeur de retour

La nouvelle position du flux

## XsltSettings::get_EnableScript() méthode


Renvoie une valeur indiquant s'il faut activer la prise en charge des blocs de script intégrés.

```cpp
bool System::Xml::Xsl::XsltSettings::get_EnableScript()
```


### Valeur de retour

**true** pour prendre en charge les blocs de script dans les feuilles de style XSLT ; sinon, **false**. La valeur par défaut est **false**.

## XsltSettings::get_Default() méthode

Renvoie un objet [XsltSettings](../) avec les paramètres par défaut. La prise en charge de la fonction **document()** XSLT et des blocs de script intégrés est désactivée.

```cpp
static SharedPtr<XsltSettings> System::Xml::Xsl::XsltSettings::get_Default()
```

### Valeur de retour

Un objet [XsltSettings](../) avec les options [XsltSettings::set_EnableDocumentFunction](../set_enabledocumentfunction/) et [XsltSettings::set_EnableScript](../set_enablescript/) définies sur **false**.

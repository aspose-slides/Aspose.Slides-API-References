## XsltContext::get_Whitespace() méthode

Lorsqu'elle est redéfinie dans une classe dérivée, renvoie une valeur indiquant s'il faut inclure les nœuds d'espacement blanc dans la sortie.

```cpp
virtual bool System::Xml::Xsl::XsltContext::get_Whitespace()=0
```

### Valeur de retour

**true** pour vérifier les nœuds d'espacement blanc dans le document source afin de les inclure dans la sortie ; **false** pour ne pas évaluer les nœuds d'espacement blanc. La valeur par défaut est **true**.

## IXsltContextFunction::get_ArgTypes() méthode


Renvoie les types du XML Path Language ([XPath](../../../system.xml.xpath/)) fournis pour la liste des arguments de la fonction. Cette information peut être utilisée pour découvrir la signature de la fonction, ce qui vous permet de différencier les fonctions surchargées.

```cpp
virtual ArrayPtr<System::Xml::XPath::XPathResultType> System::Xml::Xsl::IXsltContextFunction::get_ArgTypes()=0
```


### Valeur de retour

Un tableau de XPathResultType représentant les types de la liste des arguments de la fonction.

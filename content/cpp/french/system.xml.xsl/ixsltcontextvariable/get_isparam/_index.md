## IXsltContextVariable::get_IsParam() méthode

Renvoie une valeur indiquant si la variable est un paramètre Extensible Stylesheet Language Transformations (XSLT). Il peut s'agir d'un paramètre d'une feuille de style ou d'un modèle.

```cpp
virtual bool System::Xml::Xsl::IXsltContextVariable::get_IsParam()=0
```

### Valeur de retour

**true** si la variable est un paramètre XSLT ; sinon, **false**.

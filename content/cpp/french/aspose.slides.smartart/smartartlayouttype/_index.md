## SmartArtLayoutType enum

Représente le type de mise en page d’un diagramme [SmartArt](../smartart/).

```cpp
enum class SmartArtLayoutType
```

### Valeurs

| Name | Value | Description |
| --- | --- | --- |
| AccentProcess | 0 | Utilisé pour montrer une progression, une frise chronologique ou des étapes séquentielles dans une tâche, un processus ou un flux de travail. Fonctionne bien pour illustrer à la fois le texte Niveau 1 et le texte Niveau 2. |
| AccentedPicture | 1 | Utilisé pour présenter une idée photographique centrale avec des idées connexes sur le côté. Le texte Niveau 1 supérieur apparaît au-dessus de l’image centrale.

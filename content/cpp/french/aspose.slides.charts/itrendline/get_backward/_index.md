## ITrendline::get_Backward() méthode

Spécifie le nombre de catégories (ou d'unités sur un diagramme de dispersion) que la droite de tendance s'étend avant les données pour la série qui est mise en tendance. Sur les diagrammes de dispersion et les diagrammes non dispersion, la valeur doit être toute valeur non négative. Lire **double**.

```cpp
virtual double Aspose::Slides::Charts::ITrendline::get_Backward()=0
```

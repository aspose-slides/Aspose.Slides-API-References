## ITrendline::set_Forward(double) méthode


Spécifie le nombre de catégories (ou unités sur un diagramme de dispersion) que la ligne de tendance prolonge après les données de la série en cours d'analyse. Sur les diagrammes de dispersion et les diagrammes non dispersés, la valeur doit être une valeur non négative. Écrire **double**.

virtual void Aspose::Slides::Charts::ITrendline::set_Forward(double value)=0

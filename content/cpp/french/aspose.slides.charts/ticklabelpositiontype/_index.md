## TickLabelPositionType enum


Représente le type de position des étiquettes de repère sur l'axe spécifié.

```cpp
enum class TickLabelPositionType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| High | 0 | Indique que les étiquettes d'axe doivent être à l'extrémité haute de l'axe perpendiculaire. |
| Low | 1 | Indique que les étiquettes d'axe doivent être à l'extrémité basse de l'axe perpendiculaire. |
| NextTo | 2 | Indique que les étiquettes d'axe doivent être à côté de l'axe. |
| None | 3 | Indique que les étiquettes d'axe ne sont pas dessinées. |

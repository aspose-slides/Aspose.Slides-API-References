## AxisAggregationType enum


Représente le type d'agrégation de l'axe des catégories. Utilisé uniquement avec les séries Histogram ou HistogramPareto.

```cpp
enum class AxisAggregationType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| ByCategory | 0 | Agréguer par catégories |
| Automatic | 1 | Agréguer les catégories automatiquement. |
| ByBinWidth | 2 | Agréguer les catégories par largeur de bac. La propriété IAxis::get(set)_BinWidth doit être définie. |
| ByNumberOfBins | 3 | Agréguer les catégories par nombre de bacs. La propriété IAxis::get(set)_NumberOfBins doit être définie. |

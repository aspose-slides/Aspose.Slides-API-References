## TrendlineType enum

Représente le type de ligne de tendance

```cpp
enum class TrendlineType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Exponential | 0 | Spécifie que la ligne de tendance doit être une courbe exponentielle sous la forme. |
| Linear | 1 | Spécifie que la ligne de tendance doit être une ligne sous la forme. |
| Logarithmic | 2 | Spécifie que la ligne de tendance doit être une courbe logarithmique sous la forme , où log est le logarithme naturel. |
| MovingAverage | 3 | Spécifie que la ligne de tendance doit être une moyenne mobile de période Period. |
| Polynomial | 4 | Spécifie que la ligne de tendance doit être une courbe polynomiale d'ordre Order sous la forme. |
| Power | 5 | Spécifie que la ligne de tendance doit être une courbe puissance sous la forme. |

## DataLabelCollection::Hide() méthode

Masquez l'étiquette de données par défaut en réglant tous les indicateurs Show*-flags (ShowValue, ...) de la propriété DefaultDataLabelFormat sur l'état false. IsVisible sera false après cela.

```cpp
void Aspose::Slides::Charts::DataLabelCollection::Hide() override
```

## Remarques

Si l'étiquette de données n'est pas visible par défaut (IsVisible est false), vous pouvez rendre l'étiquette de données « visible par défaut » en réglant les indicateurs Show*-flags (ShowValue, ...) de la propriété DefaultDataLabelFormat sur l'état true.

## DataSourceType enum

Types de source de données.

```cpp
enum class DataSourceType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Worksheet | 0 | Utilisez les objets CellXls pour stocker XValue, YValue, Value ou BubbleSize d'un point de données dans la feuille de calcul. |
| StringLiterals | 1 | Utilisez une variable de chaîne à accès rapide pour stocker XValue, YValue, Value ou BubbleSize d'un point de données. La feuille de calcul n'est PAS utilisée pour stocker les valeurs. |
| DoubleLiterals | 2 | Utilisez une variable double à accès rapide pour stocker XValue, YValue, Value ou BubbleSize d'un point de données. La feuille de calcul n'est PAS utilisée pour stocker les valeurs. |

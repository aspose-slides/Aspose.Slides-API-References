## TickMarkType enum

Représente le type de repère de graduation pour l'axe spécifié.

```cpp
enum class TickMarkType
```

### Values

| Nom | Valeur | Description |
| --- | --- | --- |
| Cross | 0 | Indique que les marques de graduation doivent traverser l'axe. |
| Inside | 1 | Indique que les marques de graduation doivent être à l'intérieur de la zone du graphique. |
| None | 2 | Indique qu'il n'y aura aucune marque de graduation. |
| Outside | 3 | Indique que les marques de graduation doivent être à l'extérieur de la zone du graphique. |

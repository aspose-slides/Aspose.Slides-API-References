## IChartCategory::get_UseCell() méthode


Si vrai, alors la propriété AsCell est effective. En d’autres termes, worksheet est utilisé pour stocker la catégorie (this case supports a multi-level category). Si faux, alors la propriété AsLiteral est effective. En d’autres termes, worksheet est NOT used for storing category (and this case doesn't support a multi-level categories). Lecture seule **bool**.

```cpp
virtual bool Aspose::Slides::Charts::IChartCategory::get_UseCell()=0
```

## Remarques


Pour changer la valeur de cette propriété (pour toutes les catégories de la collection), définissez la nouvelle valeur sur la propriété ChartCategoryCollection.UseCells property.

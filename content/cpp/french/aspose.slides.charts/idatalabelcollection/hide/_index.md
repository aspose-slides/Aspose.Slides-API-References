## IDataLabelCollection::Hide() méthode

Masquez l'étiquette de données par défaut en définissant tous les drapeaux Show*-flags (ShowValue, ...) de la propriété DefaultDataLabelFormat à l'état false. IsVisible sera false après cela.

```cpp
virtual void Aspose::Slides::Charts::IDataLabelCollection::Hide()=0
```

## Remarques

Si l'étiquette de données n'est pas visible par défaut (IsVisible est false) vous pouvez rendre l'étiquette de données « visible par défaut » en réglant les drapeaux Show*-flags (ShowValue, ...) de la propriété DefaultDataLabelFormat à l'état true.

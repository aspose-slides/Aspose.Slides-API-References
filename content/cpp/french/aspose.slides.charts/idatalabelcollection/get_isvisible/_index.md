## IDataLabelCollection::get_IsVisible() méthode

False signifie que l'étiquette de données n'est pas visible par défaut (et donc tous les indicateurs Show*-flags (ShowValue, ...) de la propriété DefaultDataLabelFormat sont false). Lecture seule **bool**.

```cpp
virtual bool Aspose::Slides::Charts::IDataLabelCollection::get_IsVisible()=0
```

## Remarques

Si l'étiquette de données est visible par défaut, vous pouvez la masquer par défaut avec la méthode [Hide()](../hide/). Mais si l'étiquette de données n'est pas visible par défaut (IsVisible est false), vous pouvez rendre l'étiquette de données \"visible \npar défaut\" en définissant les indicateurs Show*-flags (ShowValue, ...) de la propriété DefaultDataLabelFormat à l'état true.

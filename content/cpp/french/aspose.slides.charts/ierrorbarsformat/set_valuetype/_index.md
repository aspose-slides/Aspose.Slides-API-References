## IErrorBarsFormat::set_ValueType(ErrorBarValueType) méthode

Représente les façons possibles de déterminer la longueur des barres d'erreur. Dans le cas d'un type de valeur personnalisé, utilisez la propriété [IChartDataPoint::get_ErrorBarsCustomValues()](../../ichartdatapoint/get_errorbarscustomvalues/) du point de données spécifique dans la collection DataPoints de la série.

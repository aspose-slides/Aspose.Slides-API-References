## IChartSeries::get_PieSplitPosition() méthode


Spécifie une valeur qui doit être utilisée pour déterminer quels points de données se trouvent dans la deuxième part ou barre d'un graphique pie-of-pie ou bar-of-pie. Est utilisée conjointement avec la propriété PieSplitBy. Cette propriété appartient non seulement à cette série mais à toutes les séries du groupe de séries parent - il s'agit d'une projection de la propriété du groupe approprié. Ainsi, cette propriété est en lecture seule. Utilisez la propriété ParentSeriesGroup pour accéder au groupe de séries parent. Utilisez la propriété en lecture/ecriture [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get(set)_PieSplitPosition() pour modifier la valeur. Lecture seule **double**.

```cpp
virtual double Aspose::Slides::Charts::IChartSeries::get_PieSplitPosition()=0
```

## Remarques


C'est la projection de la propriété [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get(set)_PieSplitPosition().

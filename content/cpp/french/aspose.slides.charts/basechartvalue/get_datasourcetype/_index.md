## BaseChartValue::get_DataSourceType() méthode

Spécifie si la propriété AsCell, AsCells, AsLiteralString ou AsLiteralDouble est effective dans les descendants. En d'autres termes, cela spécifie le type de valeur de la propriété Data. Lire [Charts::DataSourceType](../../datasourcetype/).

```cpp
Aspose::Slides::Charts::DataSourceType Aspose::Slides::Charts::BaseChartValue::get_DataSourceType() override
```

## Remarques

Pour les points dans [ChartDataPointCollection](../../chartdatapointcollection/) cette propriété est en lecture seule. Dans ce cas, pour modifier la valeur de cette propriété, vous pouvez utiliser l'une des propriétés ChartDataPointCollection.DataSourceTypeFor<...>.

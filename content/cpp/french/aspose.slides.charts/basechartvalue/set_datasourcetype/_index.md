## BaseChartValue::set_DataSourceType(Aspose::Slides::Charts::DataSourceType) méthode


Spécifie si la propriété AsCell, AsCells, AsLiteralString ou AsLiteralDouble est effective dans les descendants. En d’autres termes, elle spécifie le type de valeur de la propriété Data. Écrire [Charts::DataSourceType](../../datasourcetype/).

```cpp
void Aspose::Slides::Charts::BaseChartValue::set_DataSourceType(Aspose::Slides::Charts::DataSourceType value) override
```

## Remarques


Pour les points dans [ChartDataPointCollection](../../chartdatapointcollection/) cette propriété est en lecture seule. Dans ce cas, pour modifier la valeur de cette propriété, vous pouvez utiliser l’une des propriétés ChartDataPointCollection.DataSourceTypeFor<...>.

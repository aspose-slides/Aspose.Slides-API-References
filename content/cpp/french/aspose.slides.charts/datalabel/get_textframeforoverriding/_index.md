## DataLabel::get_TextFrameForOverriding() méthode

Peut contenir un texte richement formaté. Si cette propriété n’est pas nulle, la valeur de texte formaté remplace le texte généré automatiquement de l’étiquette de données. Le texte généré automatiquement de l’étiquette de données désigne le texte géré par les propriétés ShowSeriesName, ShowValue, … et formaté avec la propriété TextFormatManager.TextFormat.

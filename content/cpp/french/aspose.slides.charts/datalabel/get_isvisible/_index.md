## DataLabel::get_IsVisible() méthode


False signifie que l'étiquette de données n'est pas visible (et donc tous les indicateurs Show*-flags (ShowValue, ...) sont false). Lecture seule **bool**.

```cpp
bool Aspose::Slides::Charts::DataLabel::get_IsVisible() override
```

## Remarques


Si l'étiquette de données est visible, vous pouvez la masquer avec [Hide()](../hide/) méthode. Mais si l'étiquette de données n'est pas visible (IsVisible est false), vous pouvez rendre l'étiquette de données visible en définissant les indicateurs Show*-flags (ShowValue, ...) à l'état true.

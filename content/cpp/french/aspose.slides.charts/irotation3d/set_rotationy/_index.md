## IRotation3D::set_RotationY(uint16_t) méthode

Définit le degré de rotation autour de l'axe Y, c’est-à-dire dans la direction X pour les graphiques 3D (entre 0 et 360 degrés). La propriété correspond à l'élément 21.2.2.158 rotY (Y Rotation) dans ECMA-376 et à l'option \"X Rotation\" dans PowerPoint 2007+. Écrire **uint16_t**.

virtual void Aspose::Slides::Charts::IRotation3D::set_RotationY(uint16_t value)=0

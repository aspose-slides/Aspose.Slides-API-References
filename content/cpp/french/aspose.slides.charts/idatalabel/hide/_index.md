## IDataLabel::Hide() méthode

Masquez l'étiquette de données en définissant tous les drapeaux Show*-flags (ShowValue, ...) sur l'état false. IsVisible sera false après cela.

```cpp
virtual void Aspose::Slides::Charts::IDataLabel::Hide()=0
```

## Remarques

Si l'étiquette de données n'est pas visible (IsVisible est false) vous pouvez rendre l'étiquette de données visible en définissant les drapeaux Show*-flags (ShowValue, ...) à l'état true.

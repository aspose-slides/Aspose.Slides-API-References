## IDataLabel::get_IsVisible() méthode


False signifie que le libellé de données n'est pas visible (et donc tous les drapeaux Show*-flags (ShowValue, ...) sont à false). Lecture seule **bool**.

```cpp
virtual bool Aspose::Slides::Charts::IDataLabel::get_IsVisible()=0
```

## Remarques


Si le libellé de données est visible, vous pouvez le masquer avec la méthode [Hide()](../hide/). Mais si le libellé de données n'est pas visible (IsVisible est false), vous pouvez rendre le libellé de données visible en définissant les drapeaux Show*-flags (ShowValue, ...) à l'état true.

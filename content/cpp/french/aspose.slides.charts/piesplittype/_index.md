## PieSplitType enum

Représente un type de points de fractionnement dans le deuxième secteur ou barre d'un graphique à secteur-dans-secteur ou barre-dans-secteur.

```cpp
enum class PieSplitType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Default | 0 | Spécifie que les points de données seront séparés en utilisant le mécanisme par défaut pour ce type de graphique. |
| Custom | 1 | Spécifie que les points de données seront séparés entre le secteur et le deuxième graphique selon les valeurs de fractionnement personnalisé. |
| ByPercentage | 2 | Spécifie que les points de données seront séparés entre le secteur et le deuxième graphique en plaçant les points dont le pourcentage est inférieur à Split Position pour cent dans le deuxième graphique. |
| ByPos | 3 | Spécifie que les points de données seront séparés entre le secteur et le deuxième graphique en plaçant la dernière Split Position des points de données dans le deuxième graphique. |
| ByValue | 4 | Spécifie que les points de données seront séparés entre le secteur et le deuxième graphique en plaçant les points de données dont la valeur est inférieure à Split Position dans le deuxième graphique. |

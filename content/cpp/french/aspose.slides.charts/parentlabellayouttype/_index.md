## ParentLabelLayoutType enum

Représente la disposition des étiquettes de données de catégorie. Utilisé uniquement avec les séries Treemap.

```cpp
enum class ParentLabelLayoutType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Banner | 0 | Spécifie que les étiquettes de catégorie parentes sont affichées en bannière |
| None | 1 | Spécifie que les étiquettes de catégorie parentes ne sont pas dessinées |
| Overlapping | 2 | Spécifie que les étiquettes de catégorie parentes sont affichées en chevauchement |

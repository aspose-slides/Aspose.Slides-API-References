## IBaseChartValue::set_DataSourceType(Aspose::Slides::Charts::DataSourceType) méthode

Spécifie si la propriété AsCell ou AsLiteralString ou AsLiteralDouble est effective. En d'autres termes, il spécifie le type de valeur de la propriété Data. Cette propriété est lecture seule. Pour modifier la valeur de cette propriété, vous pouvez utiliser l'une des propriétés ChartDataPointCollection.DataSourceTypeFor<...>. Écrivez [DataSourceType](../../datasourcetype/).

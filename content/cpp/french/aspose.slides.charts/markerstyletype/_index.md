## MarkerStyleType énumération

Détermine la forme du marqueur sur le point de données du graphique.

```cpp
enum class MarkerStyleType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| NotDefined | -1 |  |
| Circle | 0 | Spécifie qu'un cercle doit être dessiné à chaque point de données. |
| Dash | 1 | Spécifie qu'un tiret doit être dessiné à chaque point de données. |
| Diamond | 2 | Spécifie qu'un losange doit être dessiné à chaque point de données. |
| Dot | 3 | Spécifie qu'un point doit être dessiné à chaque point de données. |
| None | 4 | Spécifie qu'aucun élément ne doit être dessiné à chaque point de données. |
| Picture | 5 | Spécifie qu'une image doit être dessinée à chaque point de données. |
| Plus | 6 | Spécifie qu'un plus doit être dessiné à chaque point de données. |
| Square | 7 | Spécifie qu'un carré doit être dessiné à chaque point de données. |
| Star | 8 | Spécifie qu'une étoile doit être dessinée à chaque point de données. |
| Triangle | 9 | Spécifie qu'un triangle doit être dessiné à chaque point de données. |
| X | 10 | Spécifie qu'un X doit être dessiné à chaque point de données. |

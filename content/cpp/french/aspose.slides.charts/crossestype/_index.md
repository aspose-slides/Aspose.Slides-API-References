## CrossesType énumération

Détermine où l'axe croise.

```cpp
enum class CrossesType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| AxisCrossesAtZero | 0 | L'axe des catégories croise le point zéro de l'axe des valeurs (si possible), ou la valeur minimale (si le minimum est supérieur à zéro) ou le maximum (si le maximum est inférieur à zéro). |
| Maximum | 1 | L'axe croise la valeur maximale. |
| Custom | 2 | Valeur personnalisée provenant de la propriété CrossAt |

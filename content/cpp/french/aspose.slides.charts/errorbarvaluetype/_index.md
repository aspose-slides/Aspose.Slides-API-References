## ErrorBarValueType enum

Représente le type de valeur de barre d'erreur

```cpp
enum class ErrorBarValueType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Custom | 0 | Spécifie que la longueur des barres d'erreur doit être déterminée par les éléments Plus et Minus. |
| Fixed | 1 | Spécifie que la longueur des barres d'erreur doit être la valeur fixe déterminée par Error Bar Value. |
| Percentage | 2 | Spécifie que la longueur des barres d'erreur doit être Error Bar Value pour cent des données. |
| StandardDeviation | 3 | Spécifie que la longueur des barres d'erreur doit être Error Bar Value écarts-types des données. |
| StandardError | 4 | Spécifie que la longueur des barres d'erreur doit être Error Bar Value erreurs standard des données. |

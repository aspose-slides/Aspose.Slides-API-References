## Trendline::get_TextFrameForOverriding() méthode

Peut contenir un texte richement formaté. Si cette propriété n'est pas nulle, alors cette valeur de texte formaté remplace le texte auto-généré de l'étiquette de données. Le texte auto-généré de l'étiquette de données désigne le texte géré par les propriétés ShowSeriesName, ShowValue, ... et formaté avec la propriété TextFormatManager.TextFormat.

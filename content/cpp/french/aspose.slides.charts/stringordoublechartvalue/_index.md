## StringOrDoubleChartValue classe

Représente une valeur de chaîne ou de double qui peut être stockée dans un document de présentation pptx de deux manières : 1) dans la/les cellule(s) du classeur lié au graphique ; 2) en tant que valeur littérale.

```cpp
class StringOrDoubleChartValue : public Aspose::Slides::Charts::BaseChartValue,
                                 public Aspose::Slides::Charts::IStringOrDoubleChartValue
```

## TimeUnitType enum

Représente l'unité de base pour l'axe des catégories

```cpp
enum class TimeUnitType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| None | 0 | Les valeurs seront affichées telles quelles. |
| Days | 1 | Spécifie que les données du graphique seront affichées en jours. |
| Months | 2 | Spécifie que les données du graphique seront affichées en mois. |
| Years | 3 | Spécifie que les données du graphique seront affichées en années. |

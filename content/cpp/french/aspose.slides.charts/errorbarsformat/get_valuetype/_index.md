## ErrorBarsFormat::get_ValueType() méthode

Représente les manières possibles de déterminer la longueur des barres d’erreur. Dans le cas d’un type de valeur personnalisé, utilisez la propriété [IChartDataPoint::get_ErrorBarsCustomValues()](../../ichartdatapoint/get_errorbarscustomvalues/) du point de données spécifique dans la collection DataPoints de la série. Dans le cas d’un type de valeur Fixed, Percentage ou StandardDeviation, utilisez la propriété Value pour spécifier la valeur.

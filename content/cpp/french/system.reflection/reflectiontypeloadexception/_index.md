## ReflectionTypeLoadException déclaration de type

ReflectionTypeLoadException est levée par la méthode Module.GetTypes si l’une des classes d’un module ne peut être chargée. Ne jamais encapsuler les instances de la classe ReflectionTypeLoadException dans [System::SmartPtr](../../system/smartptr/).

```cpp
using System::Reflection::ReflectionTypeLoadException = typedef System::ExceptionWrapper<Details_ReflectionTypeLoadException>
```

Retourne l'attribut **block**.

```cpp
XmlSchemaDerivationMethod System::Xml::Schema::XmlSchemaComplexType::get_Block()
```

### Valeur de retour

L'attribut **block** empêche un type complexe d'être utilisé dans le type de dérivation spécifié. La valeur par défaut est [XmlSchemaDerivationMethod::None](../../xmlschemaderivationmethod/). Optionnel.

## XmlSchemaComplexType::get_IsMixed() méthode


Renvoie des informations qui déterminent si le type complexe possède un modèle de contenu mixte (balises à l'intérieur du contenu).

```cpp
bool System::Xml::Schema::XmlSchemaComplexType::get_IsMixed() override
```


### Valeur de retour

**true**, si des données de caractères peuvent apparaître entre les éléments enfants de ce type complexe ; sinon, **false**. La valeur par défaut est **false**. Optionnel.

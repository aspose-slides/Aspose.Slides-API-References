## XmlSchemaComplexType::get_IsAbstract() méthode

Renvoie les informations qui déterminent si l'élément **complexType** peut être utilisé dans le document d'instance.

```cpp
bool System::Xml::Schema::XmlSchemaComplexType::get_IsAbstract()
```

### Valeur de retour

Si **true**, un élément ne peut pas utiliser directement cet élément **complexType** et doit utiliser un type complexe dérivé de cet élément **complexType**. La valeur par défaut est **false**. Optionnel.

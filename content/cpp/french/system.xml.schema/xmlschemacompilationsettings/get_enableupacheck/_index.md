## XmlSchemaCompilationSettings::get_EnableUpaCheck() méthode


Renvoie une valeur indiquant si le [XmlSchemaSet](../../xmlschemaset/) doit vérifier les violations de l'attribution unique de particules (UPA).

```cpp
bool System::Xml::Schema::XmlSchemaCompilationSettings::get_EnableUpaCheck()
```


### Valeur de retour

**true** si le [XmlSchemaSet](../../xmlschemaset/) doit vérifier les violations de l'attribution unique de particules (UPA) ; sinon, **false**. La valeur par défaut est **true**.

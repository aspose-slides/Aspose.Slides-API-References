## XmlSchema::get_FinalDefault() méthode

Renvoie l'attribut **finalDefault** qui définit la valeur par défaut de l'attribut **final** sur les éléments et les types complexes dans l'espace de noms cible du schéma.

```cpp
XmlSchemaDerivationMethod System::Xml::Schema::XmlSchema::get_FinalDefault()
```

### Valeur de retour

Une valeur XmlSchemaDerivationMethod représentant les différentes méthodes de prévention de la dérivation. La valeur par défaut est [XmlSchemaDerivationMethod::None](../../xmlschemaderivationmethod/).

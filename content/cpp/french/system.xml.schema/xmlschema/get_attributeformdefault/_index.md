## XmlSchema::get_AttributeFormDefault() méthode

Renvoie le formulaire pour les attributs déclarés dans l’espace de noms cible du schéma.

```cpp
XmlSchemaForm System::Xml::Schema::XmlSchema::get_AttributeFormDefault()
```

### Valeur de retour

La valeur XmlSchemaForm qui indique si les attributs de l’espace de noms cible doivent être qualifiés avec le préfixe d’espace de noms. La valeur par défaut est [XmlSchemaForm::None](../../xmlschemaform/).

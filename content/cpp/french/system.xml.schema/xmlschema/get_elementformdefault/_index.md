## XmlSchema::get_ElementFormDefault() méthode

Renvoie la forme des éléments déclarés dans l'espace de noms cible du schéma.

```cpp
XmlSchemaForm System::Xml::Schema::XmlSchema::get_ElementFormDefault()
```

### Valeur de retour

La valeur XmlSchemaForm qui indique si les éléments de l'espace de noms cible doivent être qualifiés avec le préfixe d'espace de noms. La valeur par défaut est [XmlSchemaForm::None](../../xmlschemaform/).

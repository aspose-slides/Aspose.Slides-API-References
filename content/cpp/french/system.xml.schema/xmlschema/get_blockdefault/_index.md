## XmlSchema::get_BlockDefault() method

Renvoie l'attribut **blockDefault** qui définit la valeur par défaut de l'attribut **block** sur les éléments et les types complexes dans le **targetNamespace** du schéma.

```cpp
XmlSchemaDerivationMethod System::Xml::Schema::XmlSchema::get_BlockDefault()
```

### Valeur de retour

Une valeur XmlSchemaDerivationMethod représentant les différentes méthodes de prévention de la dérivation. La valeur par défaut est [XmlSchemaDerivationMethod::None](../../xmlschemaderivationmethod/).

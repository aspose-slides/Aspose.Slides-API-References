## XmlSchemaObjectCollection::Contains(const SharedPtr\<XmlSchemaObject\>\&) méthode

Indique si le [XmlSchemaObject](../../xmlschemaobject/) spécifié se trouve dans le [XmlSchemaObjectCollection](../).

```cpp
bool System::Xml::Schema::XmlSchemaObjectCollection::Contains(const SharedPtr<XmlSchemaObject> &item)
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| item | const [SharedPtr](../../../system/sharedptr/)\<[XmlSchemaObject](../../xmlschemaobject/)\>\& | Le [XmlSchemaObject](../../xmlschemaobject/). |

### Valeur de retour

**true** si le nom qualifié spécifié se trouve dans la collection ; sinon, renvoie **false**. Si **nullptr** est fourni, **false** est renvoyé car il n'existe aucun nom qualifié avec un nom nul.

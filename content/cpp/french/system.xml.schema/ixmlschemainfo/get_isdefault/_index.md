## IXmlSchemaInfo::get_IsDefault() méthode


Renvoie une valeur indiquant si ce nœud XML validé a été défini comme résultat d’une valeur par défaut appliquée lors de la validation du schéma XML [Schema](../../) Definition Language (XSD).

```cpp
virtual bool System::Xml::Schema::IXmlSchemaInfo::get_IsDefault()=0
```


### Valeur de retour

**true** si ce nœud XML validé a été défini comme résultat d’une valeur par défaut appliquée lors de la validation du schéma ; sinon, **false**.

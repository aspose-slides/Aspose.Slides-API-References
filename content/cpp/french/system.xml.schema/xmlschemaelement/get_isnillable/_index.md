## XmlSchemaElement::get_IsNillable() méthode


Renvoie des informations indiquant si **xsi:nil** peut apparaître dans les données d'instance. Indique si une valeur nil explicite peut être assignée à l'élément.

```cpp
bool System::Xml::Schema::XmlSchemaElement::get_IsNillable()
```


### Valeur de retour

Si nillable est **true**, cela permet à une instance de l'élément d'avoir l'attribut **nil** défini sur **true**. L'attribut **nil** est défini comme faisant partie de l'espace de noms XML [Schema](../../) pour les instances. La valeur par défaut est **false**. Optionnel.

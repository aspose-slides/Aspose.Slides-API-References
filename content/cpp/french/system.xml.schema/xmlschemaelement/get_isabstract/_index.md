## XmlSchemaElement::get_IsAbstract() méthode

Renvoie des informations indiquant si l'élément peut être utilisé dans un document d'instance.

```cpp
bool System::Xml::Schema::XmlSchemaElement::get_IsAbstract()
```

### Valeur de retour

Si **true**, l'élément ne peut pas apparaître dans le document d'instance. La valeur par défaut est **false**. Optionnel.

## XmlSchemaAny::get_ProcessContents() méthode

Renvoie des informations sur la façon dont une application ou un processeur XML doit gérer la validation des documents XML pour les éléments spécifiés par l'élément **any**.

```cpp
XmlSchemaContentProcessing System::Xml::Schema::XmlSchemaAny::get_ProcessContents()
```

### Valeur de retour

Une des valeurs XmlSchemaContentProcessing. Si aucun attribut **processContents** n'est spécifié, la valeur par défaut est **Strict**.

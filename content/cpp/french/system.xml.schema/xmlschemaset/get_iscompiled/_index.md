## XmlSchemaSet::get_IsCompiled() méthode


Renvoie une valeur qui indique si les schémas du langage de définition XML [Schema](../../) (XSD) dans le [XmlSchemaSet](../) ont été compilés.

```cpp
bool System::Xml::Schema::XmlSchemaSet::get_IsCompiled()
```


### Valeur de retour

**true** si les schémas dans le [XmlSchemaSet](../) ont été compilés depuis la dernière fois qu'un schéma a été ajouté ou supprimé du [XmlSchemaSet](../); sinon, **false**.

## DashCap enum

Spécifie le type de terminaison utilisé aux deux extrémités d'un tiret dans une ligne pointillée.

```cpp
enum class DashCap
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Flat | 0 | Une coiffe carrée qui donne une forme carrée aux deux extrémités d'un tiret. |
| Round | 2 | Une coiffe circulaire qui arrondit les deux extrémités d'un tiret. |
| Triangle | 3 | Une coiffe triangulaire qui pointe aux deux extrémités d'un tiret. |

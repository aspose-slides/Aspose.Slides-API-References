## PixelOffsetMode enum


Spécifie comment les pixels sont décalés lors du rendu.

```cpp
enum class PixelOffsetMode
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Invalid | n/a | Valeur invalide. |
| Default | n/a | Le mode par défaut. |
| HighSpeed | n/a | Mode basse qualité (haute performance). |
| HighQuality | n/a | Mode haute qualité (performance moindre). |
| None | n/a | Pas de décalage de pixel. |
| Half | n/a | Les pixels sont décalés de -0.5 unités à la fois horizontalement et verticalement pour un anti-aliasing haute performance. |

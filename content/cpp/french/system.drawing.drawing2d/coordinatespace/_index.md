## CoordinateSpace enum

Spécifie comment évaluer les coordonnées.

```cpp
enum class CoordinateSpace
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| World | 0 | Les coordonnées sont dans le contexte de coordonnées du monde. |
| Page | 1 | Les coordonnées sont dans le contexte de coordonnées de la page. |
| Device | 2 | Indique que les coordonnées sont dans le contexte de coordonnées du dispositif. |

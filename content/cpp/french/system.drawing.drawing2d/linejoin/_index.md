## LineJoin énumération

Spécifie comment les lignes ou courbes consécutives sont jointes.

```cpp
enum class LineJoin
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Miter | 0 | Jonction en onglet qui produit un coin aigu ou tronqué selon que la longueur de l'onglet dépasse la limite de l'onglet. |
| Bevel | 1 | Jonction biseau qui produit un coin diagonal. |
| Round | 2 | Jonction arrondie qui produit un arc circulaire lisse entre les lignes. |
| MiterClipped | 3 | Jonction en onglet qui produit un coin aigu ou biseauté selon que la longueur de l'onglet dépasse la limite de l'onglet. |

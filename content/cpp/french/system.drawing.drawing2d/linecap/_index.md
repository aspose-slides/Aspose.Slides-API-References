## enum LineCap

Indique les styles de bouchon disponibles avec lesquels un objet [Pen](../../system.drawing/pen/) peut terminer une ligne.

```cpp
enum class LineCap
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Flat | 0 | Bouchon de ligne plat. |
| Square | 1 | Bouchon de ligne carré. |
| Round | 2 | Bouchon de ligne arrondi. |
| Triangle | 3 | Bouchon de ligne triangulaire. |
| NoAnchor | 16 | Pas d'ancre. |
| SquareAnchor | 17 | Bouchon de ligne d'ancre carrée. |
| RoundAnchor | 18 | Bouchon de ligne d'ancre arrondie. |
| DiamondAnchor | 19 | Bouchon de ligne d'ancre en diamant. |
| ArrowAnchor | 20 | Bouchon de ligne d'ancre flèche. |
| AnchorMask | 240 | Un masque utilisé pour vérifier si un bouchon de ligne est un bouchon d'ancre. |
| Custom | 255 | Bouchon personnalisé. |

## WrapMode enum

Spécifie comment une texture ou un gradient est répété lorsqu’il est plus petit que la zone à remplir.

```cpp
enum class WrapMode
```

### Values

| Nom | Valeur | Description |
| --- | --- | --- |
| Tile | 0 | Répète le gradient ou la texture. |
| TileFlipX | 1 | Inverse la texture ou le gradient horizontalement, puis répète la texture ou le gradient. |
| TileFlipY | 2 | Inverse la texture ou le gradient verticalement, puis répète la texture ou le gradient. |
| TileFlipXY | 3 | Inverse la texture ou le gradient horizontalement et verticalement, puis répète la texture ou le gradient. |
| Clamp | 4 | Contraint la texture ou le gradient à la limite de l'objet. |

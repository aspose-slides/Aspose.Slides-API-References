## InterpolationMode enum


Spécifie un algorithme à utiliser lorsque les images sont pivotées ou redimensionnées.

```cpp
enum class InterpolationMode
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Invalid | n/a | Equivalent to [QualityMode::Invalid](../qualitymode/). |
| Default | n/a | Mode par défaut. |
| Low | n/a | Qualité basse. |
| High | n/a | Qualité élevée. |
| Bilinear | n/a | Interpolation bilinéaire. |
| Bicubic | n/a | Interpolation bicubique. |
| NearestNeighbor | n/a | Interpolation du voisin le plus proche. |
| HighQualityBilinear | n/a | Interpolation bilinéaire de haute qualité. |
| HighQualityBicubic | n/a | Interpolation bicubique de haute qualité. |

## PathPointType enum


Spécifie le type d'un point de chemin.

```cpp
enum class PathPointType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Start | 0 | Le point de départ. |
| Line | 1 | Un segment de ligne. |
| Bezier | 3 | Courbe de Bézier. |
| PathTypeMask | 7 | Un point de masque de type. |
| DashMode | 16 | Spécifie qu'un segment est en pointillés. |
| PathMarker | 32 | Un marqueur de chemin. |
| CloseSubpath | 128 | Le point de fin d'un sous-chemin. |
| Bezier3 | 3 | Courbe de Bézier cubique. |

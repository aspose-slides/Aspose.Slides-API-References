Spécifie le format des données de couleur d'un pixel.


### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Format32bppRgb | 2498570 | Indique que le format de pixel est de 32 bits par pixel avec 8 bits pour chaque composant rouge, vert et bleu et les 8 bits restants ne sont pas utilisés. |

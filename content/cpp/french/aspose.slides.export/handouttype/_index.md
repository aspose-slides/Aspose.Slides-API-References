## HandoutType énum

Spécifie le nombre de diapositives et la séquence dans laquelle elles seront placées sur la page.

```cpp
enum class HandoutType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Handouts1 | 0 | Une diapositive par page. |
| Handouts2 | 1 | Deux diapositives par page. |
| Handouts3 | 2 | Trois diapositives par page. |
| Handouts4Horizontal | 3 | Quatre diapositives par page dans une séquence horizontale. |
| Handouts4Vertical | 4 | Quatre diapositives par page dans une séquence verticale. |
| Handouts6Horizontal | 5 | Six diapositives par page dans une séquence horizontale. |
| Handouts6Vertical | 6 | Six diapositives par page dans une séquence verticale. |
| Handouts9Horizontal | 7 | Neuf diapositives par page dans une séquence horizontale. |
| Handouts9Vertical | 8 | Neuf diapositives par page dans une séquence verticale. |

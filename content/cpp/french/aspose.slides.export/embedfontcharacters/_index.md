## EmbedFontCharacters enum

Représente la règle à utiliser pour ajouter une nouvelle police d’incrustation dans [IPresentation](../../aspose.slides/ipresentation/)

```cpp
enum class EmbedFontCharacters
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| OnlyUsed | 0 | Incorpore uniquement les caractères utilisés dans la présentation (idéal pour réduire la taille du fichier). |
| All | 1 | Incorpore tous les caractères (idéal pour l'édition par d'autres personnes). |

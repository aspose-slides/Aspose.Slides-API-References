## NotesCommentsLayoutingOptions::set_ShowCommentsByNoAuthor(bool) méthode

Définit la visibilité des commentaires qui n'ont pas d'auteur. Si true alors les commentaires seront affichés. (S'applique uniquement si les commentaires sont affichés).

```cpp
void Aspose::Slides::Export::NotesCommentsLayoutingOptions::set_ShowCommentsByNoAuthor(bool value)
```

## Remarques

La valeur par défaut est **false**.

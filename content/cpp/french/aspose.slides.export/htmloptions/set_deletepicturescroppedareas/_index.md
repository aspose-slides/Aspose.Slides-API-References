## HtmlOptions::set_DeletePicturesCroppedAreas(bool) méthode

Un indicateur booléen indique si les parties recadrées restent dans le document. Si vrai, les parties recadrées seront supprimées, si faux elles seront sérialisées dans le document (ce qui peut éventuellement conduire à un fichier plus volumineux)

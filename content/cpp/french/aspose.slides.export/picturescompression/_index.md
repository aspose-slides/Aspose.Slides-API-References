## PicturesCompression énum

Représente le niveau de compression des images

```cpp
enum class PicturesCompression
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Dpi330 | 0 | Bonne qualité pour les écrans haute définition (HD) |
| Dpi220 | 1 | Excellente qualité sur la plupart des imprimantes et écrans |
| Dpi150 | 2 | Bon pour les pages web et les projecteurs |
| Dpi96 | 3 | Réduire la taille du document pour le partage |
| Dpi72 | 4 | Niveau de compression par défaut |
| DocumentResolution | 5 | Utiliser la résolution du document - l'image ne sera pas compressée et sera utilisée dans le document telle quelle |

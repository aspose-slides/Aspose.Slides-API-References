## ISSVGOptions::get_DeletePicturesCroppedAreas() méthode

Un drapeau booléen indique si les parties recadrées restent dans le document. Si true les parties recadrées seront supprimées, si false elles seront sérialisées dans le document (ce qui peut éventuellement entraîner un fichier plus volumineux) Lire **bool**.

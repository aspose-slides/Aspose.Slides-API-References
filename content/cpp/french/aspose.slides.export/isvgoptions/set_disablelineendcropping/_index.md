## ISVGOptions::set_DisableLineEndCropping(bool) méthode

SVG 1.1 ne permet pas de définir des retraits pour les repères. [Aspose.Slides](../../../aspose.slides/) le moteur d'écriture SVG propose une solution de contournement pour ce problème : il recadre la fin de ligne avec la flèche, ainsi la ligne ne chevauche pas les repères. Cette option désactive ce comportement. Écrivez **bool**.

```cpp
virtual void Aspose::Slides::Export::ISVGOptions::set_DisableLineEndCropping(bool value)=0
```

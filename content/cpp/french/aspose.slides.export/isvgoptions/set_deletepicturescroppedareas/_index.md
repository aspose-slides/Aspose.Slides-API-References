## ISVGOptions::set_DeletePicturesCroppedAreas(bool) méthode

Un indicateur booléen indique si les parties recadrées restent intégrées au document. Si true les parties recadrées seront supprimées, si false elles seront sérialisées dans le document (ce qui peut éventuellement entraîner un fichier plus volumineux) Write **bool**.

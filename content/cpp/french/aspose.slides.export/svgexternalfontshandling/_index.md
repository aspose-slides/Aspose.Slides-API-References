## SvgExternalFontsHandling enum

Représente une façon de gérer les polices externes utilisées pour le dessin du texte.

```cpp
enum class SvgExternalFontsHandling
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| AddLinksToFontFiles | 0 | Ajoute des liens vers des fichiers de polices séparés à la section style du fichier SVG. |
| Embed | 1 | Enregistre les données des polices directement dans le fichier SVG. Veuillez vérifier tous les accords de licence des polices externes avant d’utiliser cette option. |
| Vectorize | 2 | Enregistre tout le texte utilisant des polices externes sous forme de graphiques. |

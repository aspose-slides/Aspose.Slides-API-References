## TextInheritanceLimit enum

Contrôle la profondeur de l'héritage des propriétés de texte.

```cpp
enum class TextInheritanceLimit
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| All | 0 | Hériter de toutes les propriétés de texte. |
| TextBox | 1 | Hériter uniquement du style de [TextFrame](../../aspose.slides/textframe/). |
| ParagraphOnly | 2 | Utiliser uniquement les propriétés définies pour le paragraphe. |

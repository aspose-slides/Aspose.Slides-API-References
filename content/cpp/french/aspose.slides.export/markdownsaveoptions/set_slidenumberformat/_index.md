## MarkdownSaveOptions::set_SlideNumberFormat(System::String) méthode

Définit la chaîne de format utilisée pour les en-têtes de numéros de diapositives dans la sortie Markdown. Le format doit inclure le placeholder \"{0}\", qui sera remplacé par l'index de la diapositive lors de l'exportation. Exemple : \"# Slide {0}\" produira \"# Slide 1\", \"# Slide 2\", etc.

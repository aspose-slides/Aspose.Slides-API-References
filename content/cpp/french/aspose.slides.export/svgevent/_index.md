## SvgEvent enum

Représente les options pour une forme SVG.

```cpp
enum class SvgEvent
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| OnFocusIn | 0 | Se produit lorsqu'un élément reçoit le focus, par exemple lorsqu'un texte devient sélectionné. |
| OnFocusOut | 1 | Se produit lorsqu'un élément perd le focus, par exemple lorsqu'un texte devient désélectionné. |
| OnActivate | 2 | Se produit lorsqu'un élément est activé, par exemple via un clic de souris ou une pression de touche. |
| OnClick | 3 | Se produit lorsque le bouton du dispositif de pointage est cliqué sur un élément. |
| OnMouseDown | 4 | Se produit lorsque le bouton du dispositif de pointage est enfoncé sur un élément. |
| OnMouseUp | 5 | Se produit lorsque le bouton du dispositif de pointage est relâché sur un élément. |
| OnMouseOver | 6 | Se produit lorsque le dispositif de pointage est déplacé sur un élément. |
| OnMouseMove | 7 | Se produit lorsque le dispositif de pointage est déplacé alors qu'il est au-dessus d'un élément. |
| OnMouseOut | 8 | Se produit lorsque le dispositif de pointage est déplacé hors d'un élément. |
| OnLoad | 9 | Se produit lorsque l'agent utilisateur a entièrement analysé l'élément et ses descendants ainsi que toutes les ressources référencées nécessaires à son rendu. |
| OnUnload | 10 | Se produit lorsque l'implémentation [DOM](../../aspose.slides.dom/) supprime un document d'une fenêtre ou d'un cadre. Applicable uniquement aux éléments SVG les plus externes. |
| OnAbort | 11 | Se produit lorsque le chargement de la page est arrêté avant qu'un élément n'ait pu se charger complètement. |
| OnError | 12 | Se produit lorsqu'un élément ne se charge pas correctement ou lorsqu'une erreur survient pendant l'exécution du script. |
| OnResize | 13 | Se produit lorsqu'une vue de document est redimensionnée. Applicable uniquement aux éléments SVG les plus externes. |
| OnScroll | 14 | Se produit lorsqu'une vue de document est déplacée le long de l'axe X, Y ou les deux. Applicable uniquement aux éléments SVG les plus externes. |
| OnZoom | 15 | Se produit lorsque le niveau de zoom d'une vue de document est modifié. Applicable uniquement aux éléments SVG les plus externes. |
| OnBegin | 16 | Se produit lorsqu'un élément d'animation démarre. |
| OnEnd | 17 | Se produit lorsqu'un élément d'animation se termine. |
| OnRepeat | 18 | Se produit lorsqu'un élément d'animation se répète. |

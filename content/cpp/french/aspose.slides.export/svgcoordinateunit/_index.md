## SvgCoordinateUnit enum

```cpp
enum class SvgCoordinateUnit
```

| Nom | Valeur | Description |
| --- | --- | --- |
| Inch | 0 | Inch |
| Centimeter | 1 | Centimeter. |
| Millimeter | 2 | Millimeter. |

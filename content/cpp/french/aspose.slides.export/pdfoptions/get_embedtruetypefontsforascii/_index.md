## PdfOptions::get_EmbedTrueTypeFontsForASCII() méthode


Détermine si [Aspose.Slides](../../../aspose.slides/) intégrera les polices communes pour le texte ASCII (plage de codes 33..127). [Fonts](../../../aspose.slides/fonts/) pour les codes de caractères supérieurs à 127 sont toujours intégrés. La liste des polices communes comprend les 14 polices de base du PDF et les polices supplémentaires spécifiées par l'utilisateur. Lire **bool**.

```cpp
bool Aspose::Slides::Export::PdfOptions::get_EmbedTrueTypeFontsForASCII() override
```

## Remarques


La valeur par défaut est **true**.

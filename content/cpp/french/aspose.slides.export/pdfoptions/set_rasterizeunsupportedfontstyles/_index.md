## PdfOptions::set_RasterizeUnsupportedFontStyles(bool) méthode


Indique si le texte doit être rasterisé sous forme de bitmap et enregistré en PDF lorsque la police ne prend pas en charge le style gras. Cette approche peut améliorer la qualité du texte dans le PDF résultant pour certaines polices. Écrire **bool**.

```cpp
void Aspose::Slides::Export::PdfOptions::set_RasterizeUnsupportedFontStyles(bool value) override
```

## Remarques


La valeur par défaut est **false**.

## PdfOptions::get_RasterizeUnsupportedFontStyles() méthode

Indique si le texte doit être rasterisé en tant que bitmap et enregistré au format PDF lorsque la police ne prend pas en charge le style gras. Cette approche peut améliorer la qualité du texte dans le PDF résultant pour certaines polices. Lire **bool**.

```cpp
bool Aspose::Slides::Export::PdfOptions::get_RasterizeUnsupportedFontStyles() override
```

## Remarques

Par défaut, la valeur est **false**.

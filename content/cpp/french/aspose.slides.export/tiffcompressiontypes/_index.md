## TiffCompressionTypes enum

Fournit des options qui contrôlent la façon dont une présentation est compressée au format TIFF.

```cpp
enum class TiffCompressionTypes
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Default | 0 | Spécifie le schéma de compression par défaut (LZW). |
| None | 1 | Spécifie aucune compression. |
| CCITT3 | 2 | Spécifie le schéma de compression CCITT3. |
| CCITT4 | 3 | Spécifie le schéma de compression CCITT4. |
| LZW | 4 | Spécifie le schéma de compression LZW (Par défaut). |
| RLE | 5 | Spécifie le schéma de compression RLE. |

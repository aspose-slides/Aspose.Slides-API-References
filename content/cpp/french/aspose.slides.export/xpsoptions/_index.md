## Remarques

L’exemple suivant montre comment convertir des présentations en XPS en utilisant les réglages par défaut. 
```cpp
// Instancie un objet Presentation qui représente un fichier de présentation
auto pres = System::MakeObject<Presentation>(u"Convert_XPS.pptx");

// Enregistre la présentation dans un document XPS
pres->Save(u"XPS_Output_Without_XPSOption_out.xps", SaveFormat::Xps);
```
L’exemple suivant montre comment convertir des présentations en XPS en utilisant des réglages personnalisés. 
```cpp
// Instancie un objet Presentation qui représente un fichier de présentation
auto pres = System::MakeObject<Presentation>(u"Convert_XPS_Options.pptx");

// Instancie la classe TiffOptions
System::SharedPtr<XpsOptions> options = System::MakeObject<XpsOptions>();
// Enregistre les MetaFiles en PNG
options->set_SaveMetafilesAsPng(true);
// Enregistre la présentation dans un document XPS
pres->Save(u"XPS_With_Options_out.xps", SaveFormat::Xps, options);
```

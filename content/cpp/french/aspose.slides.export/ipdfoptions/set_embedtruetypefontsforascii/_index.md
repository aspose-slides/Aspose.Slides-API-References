## IPdfOptions::set_EmbedTrueTypeFontsForASCII(bool) méthode

Vrai pour incorporer les polices TrueType pour les caractères ASCII 32-127. [Fonts](../../../aspose.slides/fonts/) pour les codes de caractères supérieurs à 127 sont toujours incorporés. Écrire **bool**.

```cpp
virtual void Aspose::Slides::Export::IPdfOptions::set_EmbedTrueTypeFontsForASCII(bool value)=0
```

## Remarques

La valeur par défaut est **true**.

## IPdfOptions::set_RasterizeUnsupportedFontStyles(bool) méthode


Indique si le texte doit être rasterisé en tant que bitmap et enregistré au format PDF lorsque la police ne prend pas en charge le style gras. Cette approche peut améliorer la qualité du texte dans le PDF généré pour certaines polices. Écrire **bool**.

```cpp
virtual void Aspose::Slides::Export::IPdfOptions::set_RasterizeUnsupportedFontStyles(bool value)=0
```

## Remarques


La valeur par défaut est **false**.

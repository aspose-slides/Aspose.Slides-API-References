## IPdfOptions::set_BestImagesCompressionRatio(bool) méthode


Indique si la compression la plus efficace (au lieu de celle par défaut) pour chaque image doit être sélectionnée automatiquement. Si la valeur est **bool**.true, pour chaque image de la présentation l'algorithme de compression le plus approprié sera choisi, ce qui entraînera une taille plus petite du document PDF résultant.

```cpp
virtual void Aspose::Slides::Export::IPdfOptions::set_BestImagesCompressionRatio(bool value)=0
```

## Remarques


La sélection du meilleur taux de compression d'image est coûteuse en calcul et consomme une quantité supplémentaire de RAM, et cette option est **bool**.false par défaut.

La valeur par défaut est **bool**.false.

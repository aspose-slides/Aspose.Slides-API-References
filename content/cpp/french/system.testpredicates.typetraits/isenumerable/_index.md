## IsEnumerable struct

Vérifie si le type possède une spécialisation [System::Collections::Generic::IEnumerable](../../system.collections.generic/ienumerable/) en tant que type de base. Si c’est le cas, le membre value est défini sur true, sinon il est défini sur false.

```cpp
template<typename T,typename Enable>class IsEnumerable : public std::false_type
```

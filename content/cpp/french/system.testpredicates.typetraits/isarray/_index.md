## IsArray typedef

Vérifie si le type est une spécialisation [System::Array](../../system/array/). Si c’est le cas, le membre value est défini sur true, sinon il est défini sur false.

```cpp
using System::TestPredicates::TypeTraits::IsArray = typedef std::is_same<T, System::Array<typename T::ValueType> >
```

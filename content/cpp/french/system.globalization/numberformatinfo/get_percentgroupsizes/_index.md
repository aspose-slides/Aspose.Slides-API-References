## NumberFormatInfo::get_PercentGroupSizes() const méthode

Obtient le nombre de chiffres par groupe de valeur en pourcentage.

```cpp
ArrayPtr<int> System::Globalization::NumberFormatInfo::get_PercentGroupSizes() const
```

### Valeur de retour

[Array](../../../system/array/) de chiffres par groupe, de gauche à droite; chaque élément doit être compris entre 1 et 9, le dernier peut être 0 ce qui signifie \"combiner tout\"; le dernier élément se répète.

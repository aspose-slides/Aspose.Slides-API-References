## NumberFormatInfo::get_NumberNegativePattern() const méthode

Obtient le motif négatif du nombre.

```cpp
int System::Globalization::NumberFormatInfo::get_NumberNegativePattern() const
```

### Valeur de retour

Indice du motif dans le tableau {"(n)", "-n", "- n", "n-", "n -"}, où "n" est le nombre et "-" est le signe moins.

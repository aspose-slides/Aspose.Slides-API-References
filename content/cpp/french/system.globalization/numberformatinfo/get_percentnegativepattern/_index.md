## NumberFormatInfo::get_PercentNegativePattern() const méthode


Obtient le modèle de pourcentage négatif.

```cpp
int System::Globalization::NumberFormatInfo::get_PercentNegativePattern() const
```


### Valeur de retour

Indice du modèle dans le tableau {"-n %", "-n%", "-%n", "%-n", "%n-", "n-%", "n%-", "-% n", "n %-", "% n-", "% -n", "n- %"}, où "n" est un nombre, "%" est le symbole de pourcentage et "-" est le signe moins.

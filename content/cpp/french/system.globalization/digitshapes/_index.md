## DigitShapes enum


Spécifie l'affichage des chiffres.

```cpp
enum class DigitShapes : int32_t
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Context | 0 | La forme du chiffre dépend du texte précédent. |
| None | 1 | La forme du chiffre n'est pas modifiée. |
| NativeNational | 2 | Les chiffres ASCII 0..9 sont remplacés par les chiffres nationaux natifs. |

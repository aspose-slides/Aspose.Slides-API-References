## NumberStyles énum

Style de nombre autorisé lors de l'analyse.

```cpp
enum class NumberStyles : int32_t
```

### Valeurs

| Name | Value | Description |
| --- | --- | --- |
| None | 0 | Aucun symbole non numérique. |
| AllowLeadingWhite | 1 | Autoriser les espaces blancs en tête. |
| AllowTrailingWhite | 2 | Autoriser les espaces blancs en fin. |
| AllowLeadingSign | 4 | Autoriser le signe en tête. |
| Integer | n/a | Mise en forme d'entier par défaut. |
| AllowTrailingSign | 8 | Autoriser le signe en fin. |
| AllowParentheses | 16 | Autoriser les parenthèses pour les valeurs négatives. |
| AllowDecimalPoint | 32 | Autoriser le point décimal. |
| AllowThousands | 64 | Autoriser les séparateurs de groupe. |
| Number | n/a | Format de nombre composite par défaut. |
| AllowExponent | 128 | Autoriser le signe d'exposant. |
| Float | n/a | Format de nombre à virgule flottante par défaut. |
| AllowCurrencySymbol | 256 | Autoriser le symbole monétaire. |
| Currency | n/a | Format monétaire par défaut. |
| Any | n/a | Autoriser tout spécificateur de format. |
| AllowHexSpecifier | 512 | Autoriser les nombres hexadécimaux. |
| HexNumber | n/a | Format hexadécimal par défaut. |

## XmlConvert::VerifyXmlChars(const String\&) méthode

Retourne la chaîne passée en argument si tous les caractères et les caractères de paires de substitution dans l'argument de chaîne sont des caractères XML valides, sinon une XmlException est levée avec des informations sur le premier caractère invalide rencontré.

```cpp
static String System::Xml::XmlConvert::VerifyXmlChars(const String &content)
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| content | const [String](../../../system/string/)\& | [String](../../../system/string/) qui contient les caractères à vérifier. |

### Valeur de retour

La chaîne passée en argument si tous les caractères et les caractères de paires de substitution dans l'argument de chaîne sont des caractères XML valides, sinon une XmlException est levée avec des informations sur le premier caractère invalide rencontré.

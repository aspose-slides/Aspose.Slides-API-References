## XmlConvert::VerifyNCName(const String\&) méthode


Vérifie que le nom est un **NCName** valide selon la recommandation W3C pour le langage de balisage étendu. Un **NCName** est un nom qui ne peut pas contenir de deux-points.

```cpp
static String System::Xml::XmlConvert::VerifyNCName(const String &name)
```


### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| name | const [String](../../../system/string/)\& | Le nom à vérifier. |

### Valeur de retour

Le nom, s’il s’agit d’un NCName valide.

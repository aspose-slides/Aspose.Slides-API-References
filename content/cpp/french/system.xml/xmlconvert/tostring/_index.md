## XmlConvert::ToString(DateTimeOffset, const String\&) méthode

Convertit le [DateTimeOffset](../../../system/datetimeoffset/) fourni en un [String](../../../system/string/) dans le format spécifié.

```cpp
static String System::Xml::XmlConvert::ToString(DateTimeOffset value, const String &format)
```

### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| value | [DateTimeOffset](../../../system/datetimeoffset/) | The [DateTimeOffset](../../../system/datetimeoffset/) to be converted. |
| format | const [String](../../../system/string/)\& | Le format auquel **s** est converti. Le paramètre de format peut être n’importe quel sous-ensemble de la Recommandation W3C pour le type XML dateTime. Pour plus d’informations, consultez la section [dateTime](https://www.w3.org/TR/xmlschema-2/#dateTime) de la spécification XML [Schema](../../../system.xml.schema/). |

### Valeur de retour

Une représentation [String](../../../system/string/) dans le format spécifié du [DateTimeOffset](../../../system/datetimeoffset/) fourni.

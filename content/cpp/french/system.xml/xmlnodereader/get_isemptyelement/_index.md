## XmlNodeReader::get_IsEmptyElement() méthode

Renvoie une valeur indiquant si le nœud actuel est un élément vide (par exemple, **<MyElement/>**).

```cpp
bool System::Xml::XmlNodeReader::get_IsEmptyElement() override
```

### Valeur de retour

**true** si le nœud actuel est un élément ([XmlNodeReader::get_NodeType](../get_nodetype/) égale [XmlNodeType::Element](../../xmlnodetype/)) et qu’il se termine par **/>** ; sinon, **false**.

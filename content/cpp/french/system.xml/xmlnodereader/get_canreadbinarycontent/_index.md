## XmlNodeReader::get_CanReadBinaryContent() méthode

Renvoie une valeur indiquant si le [XmlNodeReader](../) implémente les méthodes de lecture du contenu binaire.

```cpp
bool System::Xml::XmlNodeReader::get_CanReadBinaryContent() override
```

### Valeur de retour

**true** si les méthodes de lecture du contenu binaire sont implémentées ; sinon **false**. La classe [XmlNodeReader](../) renvoie toujours **true**.

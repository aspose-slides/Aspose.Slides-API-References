## IXmlNamespaceResolver::LookupNamespace(const String\&) méthode

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| prefix | const [String](../../../system/string/)\& | Le préfixe dont vous souhaitez trouver l'URI de l'espace de noms. |

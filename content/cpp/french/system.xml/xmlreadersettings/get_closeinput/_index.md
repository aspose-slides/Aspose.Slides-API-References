## XmlReaderSettings::get_CloseInput() méthode

Renvoie une valeur indiquant si le flux sous-jacent ou le TextReader doit être fermé lorsque le lecteur est fermé.

```cpp
bool System::Xml::XmlReaderSettings::get_CloseInput()
```

### Valeur de retour

**true** pour fermer le flux sous-jacent ou le TextReader lorsque le lecteur est fermé ; sinon **false**. La valeur par défaut est **false**.

## XmlReaderSettings::get_ConformanceLevel() méthode

Renvoie le niveau de conformité que [XmlReader](../../xmlreader/) respectera.

```cpp
System::Xml::ConformanceLevel System::Xml::XmlReaderSettings::get_ConformanceLevel()
```

### Valeur de retour

L'une des valeurs d'énumération qui indique le niveau de conformité que le lecteur XML appliquera. La valeur par défaut est [ConformanceLevel::Document](../../conformancelevel/).

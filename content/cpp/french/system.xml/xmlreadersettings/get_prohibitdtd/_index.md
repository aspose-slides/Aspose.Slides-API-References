## XmlReaderSettings::get_ProhibitDtd() method


Renvoie une valeur indiquant s’il faut interdire le traitement de la définition de type de document (DTD).

```cpp
bool System::Xml::XmlReaderSettings::get_ProhibitDtd()
```


### Valeur de retour

**true** pour interdire le traitement DTD ; sinon **false**. La valeur par défaut est **true**.

Obsolète
:   Cette méthode est obsolète. Utilisez [XmlTextReader::get_DtdProcessing](../../xmltextreader/get_dtdprocessing/) à la place.

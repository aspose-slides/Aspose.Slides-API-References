## XmlReaderSettings::get_MaxCharactersFromEntities() méthode

Renvoie une valeur indiquant le nombre maximal de caractères admissibles dans un document résultant de l'expansion des entités.

```cpp
int64_t System::Xml::XmlReaderSettings::get_MaxCharactersFromEntities()
```

### Valeur de retour

Le nombre maximal de caractères admissibles provenant des entités développées. La valeur par défaut est 0.

## XmlReaderSettings::get_MaxCharactersInDocument() méthode


Renvoie une valeur indiquant le nombre maximal autorisé de caractères dans un document XML. Une valeur zéro (0) signifie qu'il n'y a aucune limite à la taille du document XML. Une valeur non nulle spécifie la taille maximale, en caractères.

```cpp
int64_t System::Xml::XmlReaderSettings::get_MaxCharactersInDocument()
```


### Valeur de retour

Le nombre maximal autorisé de caractères dans un document XML. La valeur par défaut est 0.

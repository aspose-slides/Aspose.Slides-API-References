## XmlReaderSettings::set_MaxCharactersInDocument(int64_t) méthode

Définit une valeur indiquant le nombre maximal de caractères autorisés dans un document XML. Une valeur zéro (0) signifie qu'il n'y a aucune limite à la taille du document XML. Une valeur non nulle spécifie la taille maximale, en caractères.

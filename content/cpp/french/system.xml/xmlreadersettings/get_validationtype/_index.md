## XmlReaderSettings::get_ValidationType() méthode


Renvoie une valeur indiquant si le [XmlReader](../../xmlreader/) effectuera une validation ou une affectation de type lors de la lecture.

```cpp
System::Xml::ValidationType System::Xml::XmlReaderSettings::get_ValidationType()
```


### Valeur de retour

Une des valeurs ValidationType qui indique si le [XmlReader](../../xmlreader/) effectuera une validation ou une affectation de type lors de la lecture. La valeur par défaut est [ValidationType::None](../../validationtype/).

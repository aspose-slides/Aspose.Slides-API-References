## XmlTextReader::get_ProhibitDtd() méthode


Renvoie une valeur indiquant s'il faut autoriser le traitement DTD.

```cpp
bool System::Xml::XmlTextReader::get_ProhibitDtd()
```


### Valeur de retour

**true** pour interdire le traitement DTD ; sinon **false**. La valeur par défaut est **false**.

Obsolète
:   Cette méthode est obsolète. Utilisez [XmlTextReader::get_DtdProcessing](../get_dtdprocessing/) à la place.

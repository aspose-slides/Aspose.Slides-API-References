## XmlTextReader::get_Namespaces() méthode

Retourne une valeur indiquant s’il faut prendre en charge les espaces de noms.

```cpp
bool System::Xml::XmlTextReader::get_Namespaces()
```

### Valeur de retour

**true** pour prendre en charge les espaces de noms ; sinon, **false**. La valeur par défaut est **true**.

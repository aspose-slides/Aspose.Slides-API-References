## XmlTextReader::get_CanReadValueChunk() méthode

Renvoie une valeur indiquant si le [XmlTextReader](../) implémente la méthode [XmlReader::ReadValueChunk](../../xmlreader/readvaluechunk/).

```cpp
bool System::Xml::XmlTextReader::get_CanReadValueChunk() override
```

### Valeur de retour

**true** si le [XmlTextReader](../) implémente la méthode [XmlReader::ReadValueChunk](../../xmlreader/readvaluechunk/) ; sinon **false**. La classe [XmlTextReader](../) renvoie toujours **false**.

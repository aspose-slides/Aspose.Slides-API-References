## XmlTextReader::ReadString() méthode

Lit le contenu d'un élément ou d'un nœud texte sous forme de chaîne.

```cpp
String System::Xml::XmlTextReader::ReadString() override
```

### Valeur de retour

Le contenu de l'élément ou du nœud texte. Il peut s'agir d'une chaîne vide si le lecteur est positionné sur autre chose qu'un élément ou un nœud texte, ou s'il n'y a plus de contenu texte à retourner dans le contexte actuel. **Remarque:** Le nœud texte peut être soit un élément, soit un nœud texte d'attribut.

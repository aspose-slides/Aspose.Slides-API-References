## XmlTextReader::get_CanReadBinaryContent() méthode


Renvoie une valeur indiquant si le [XmlTextReader](../) implémente les méthodes de lecture de contenu binaire.

```cpp
bool System::Xml::XmlTextReader::get_CanReadBinaryContent() override
```


### Valeur de retour

**true** si les méthodes de lecture de contenu binaire sont implémentées ; sinon **false**. La classe [XmlTextReader](../) renvoie toujours **true**.

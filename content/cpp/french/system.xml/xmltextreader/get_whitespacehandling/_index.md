## XmlTextReader::get_WhitespaceHandling() méthode


Renvoie une valeur qui indique comment les espaces blancs sont gérés.

```cpp
System::Xml::WhitespaceHandling System::Xml::XmlTextReader::get_WhitespaceHandling()
```


### Valeur de retour

Une des valeurs WhitespaceHandling. La valeur par défaut est **[WhitespaceHandling.All](../../whitespacehandling/)** (renvoie les nœuds **Whitespace** et **SignificantWhitespace**).

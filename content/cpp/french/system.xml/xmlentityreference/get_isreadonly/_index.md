## XmlEntityReference::get_IsReadOnly() méthode


Renvoie une valeur indiquant si le nœud est en lecture seule.

```cpp
bool System::Xml::XmlEntityReference::get_IsReadOnly() override
```


### Valeur de retour

**true** si le nœud est en lecture seule ; sinon **false**. Comme les nœuds [XmlEntityReference](../) sont en lecture seule, cette méthode renvoie toujours **true**.

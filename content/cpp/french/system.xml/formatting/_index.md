## Formatting enum

Spécifie les options de formatage pour le [XmlTextWriter](../xmltextwriter/).

```cpp
enum class Formatting
```

### Values

| Nom | Valeur | Description |
| --- | --- | --- |
| None | 0 | Aucun formatage spécial n'est appliqué. C’est la valeur par défaut. |
| Indented | 1 | Induit l'indentation des éléments enfants selon les paramètres [XmlTextWriter::set_Indentation](../xmltextwriter/set_indentation/) et [XmlTextWriter::set_IndentChar](../xmltextwriter/set_indentchar/). |

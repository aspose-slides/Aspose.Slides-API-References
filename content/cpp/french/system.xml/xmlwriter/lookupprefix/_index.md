## XmlWriter::LookupPrefix(String) méthode

Lorsqu'elle est remplacée dans une classe dérivée, renvoie le préfixe le plus proche défini dans la portée du namespace actuel pour l'URI du namespace.

```cpp
virtual String System::Xml::XmlWriter::LookupPrefix(String ns)=0
```

### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| ns | [String](../../../system/string/) | L'URI du namespace dont vous souhaitez trouver le préfixe. |

### Valeur de retour

Le préfixe correspondant ou **nullptr** si aucun URI de namespace correspondant n'est trouvé dans la portée actuelle.

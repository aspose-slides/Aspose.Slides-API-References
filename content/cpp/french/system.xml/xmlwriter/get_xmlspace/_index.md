## XmlWriter::get_XmlSpace() méthode

Lorsqu'elle est substituée dans une classe dérivée, récupère un XmlSpace représentant la portée actuelle de **xml:space**.

```cpp
virtual System::Xml::XmlSpace System::Xml::XmlWriter::get_XmlSpace()
```

### Valeur de retour

Un XmlSpace représentant la portée actuelle de **xml:space**.



| Valeur | Signification |
| --- | --- |
| `None`| C’est la valeur par défaut s’il n’existe aucune portée `xml:space` |
| `Default`| La portée actuelle est `xml:space="default"` |
| `Preserve`| La portée actuelle est `xml:space="preserve"` |

## WhitespaceHandling enum

Spécifie comment les espaces blancs sont gérés.

```cpp
enum class WhitespaceHandling
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| All | 0 | Renvoie les nœuds **Whitespace** et **SignificantWhitespace**. C’est la valeur par défaut. |
| Significant | 1 | Renvoie uniquement les nœuds **SignificantWhitespace**. |
| None | 2 | Ne renvoie aucun nœud **Whitespace** ni **SignificantWhitespace**. |

## XmlValidatingReader::get_IsDefault() méthode

Renvoie une valeur indiquant si le nœud actuel est un attribut qui a été généré à partir de la valeur par défaut définie dans la définition de type de document (DTD) ou le schéma.

```cpp
bool System::Xml::XmlValidatingReader::get_IsDefault() override
```

### Valeur de retour

**true** si le nœud actuel est un attribut dont la valeur a été générée à partir de la valeur par défaut définie dans le DTD ou le schéma ; **false** si la valeur de l'attribut a été définie explicitement.

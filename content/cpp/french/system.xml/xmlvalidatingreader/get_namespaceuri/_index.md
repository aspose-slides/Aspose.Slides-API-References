## XmlValidatingReader::get_NamespaceURI() méthode

Renvoie l'identifiant de ressource uniforme (URI) de l'espace de noms (tel que défini dans la spécification d'espace de noms du World Wide [Web](../../../system.web/) Consortium (W3C)) du nœud sur lequel le lecteur est positionné.

```cpp
String System::Xml::XmlValidatingReader::get_NamespaceURI() override
```

### Valeur de retour

L'URI de l'espace de noms du nœud actuel ; sinon une chaîne vide.

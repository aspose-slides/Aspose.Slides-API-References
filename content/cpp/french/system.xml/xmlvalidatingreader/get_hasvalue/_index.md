## XmlValidatingReader::get_HasValue() méthode

Renvoie une valeur indiquant si le nœud actuel peut avoir un [XmlValidatingReader::get_Value](../get_value/) autre que [String::Empty](../../../system/string/empty/).

```cpp
bool System::Xml::XmlValidatingReader::get_HasValue() override
```

### Valeur de retour

**true** si le nœud sur lequel le lecteur est actuellement positionné peut avoir une **Value** ; sinon, **false**.

## XmlValidatingReader::get_CanReadBinaryContent() méthode

Renvoie une valeur indiquant si le [XmlValidatingReader](../) implémente les méthodes de lecture du contenu binaire.

```cpp
bool System::Xml::XmlValidatingReader::get_CanReadBinaryContent() override
```

### Valeur de retour

**true** si les méthodes de lecture du contenu binaire sont implémentées ; sinon **false**. La classe [XmlValidatingReader](../) renvoie **true**.

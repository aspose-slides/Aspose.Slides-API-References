## XmlReader::MoveToFirstAttribute() méthode


Lorsqu'il est remplacé dans une classe dérivée, se déplace vers le premier attribut.

```cpp
virtual bool System::Xml::XmlReader::MoveToFirstAttribute()=0
```


### Valeur de retour

**true** si un attribut existe (le lecteur se déplace vers le premier attribut) ; sinon, **false** (la position du lecteur ne change pas).

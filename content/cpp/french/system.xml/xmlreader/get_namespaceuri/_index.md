## XmlReader::get_NamespaceURI() méthode


Lorsqu'elle est remplacée dans une classe dérivée, récupère l'URI de l'espace de noms (tel que défini dans la spécification d'espace de noms du W3C) du nœud sur lequel le lecteur est positionné.

```cpp
virtual String System::Xml::XmlReader::get_NamespaceURI()=0
```


### Valeur de retour

L'URI de l'espace de noms du nœud actuel ; sinon une chaîne vide.

## XmlReader::get_IsDefault() méthode

When overridden in a derived class, gets a value indicating whether the current node is an attribute that was generated from the default value defined in the DTD or schema.

```cpp
virtual bool System::Xml::XmlReader::get_IsDefault()
```

### Valeur de retour

**true** si le nœud actuel est un attribut dont la valeur a été générée à partir de la valeur par défaut définie dans le DTD ou le schéma ; **false** si la valeur de l’attribut a été définie explicitement.

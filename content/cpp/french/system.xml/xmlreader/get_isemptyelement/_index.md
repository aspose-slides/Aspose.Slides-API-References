## XmlReader::get_IsEmptyElement() méthode


Lorsqu'elle est redéfinie dans une classe dérivée, renvoie une valeur indiquant si le nœud actuel est un élément vide (par exemple, **<MyElement/>**).

```cpp
virtual bool System::Xml::XmlReader::get_IsEmptyElement()=0
```


### Valeur de retour

**true** si le nœud actuel est un élément ([XmlReader::get_NodeType](../get_nodetype/) égal à [XmlNodeType::Element](../../xmlnodetype/)) qui se termine par **/>** ; sinon, **false**.

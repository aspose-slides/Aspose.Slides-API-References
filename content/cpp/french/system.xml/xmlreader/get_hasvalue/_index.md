## XmlReader::get_HasValue() méthode


Lorsqu'il est remplacé dans une classe dérivée, obtient une indication indiquant si le nœud actuel peut avoir une valeur [XmlReader::get_Value](../get_value/).

```cpp
virtual bool System::Xml::XmlReader::get_HasValue()
```


### Valeur de retour

**true** si le nœud sur lequel le lecteur est actuellement positionné peut avoir une **Value** ; sinon, **false**. Si **false**, le nœud a une valeur de [String::Empty](../../../system/string/empty/).

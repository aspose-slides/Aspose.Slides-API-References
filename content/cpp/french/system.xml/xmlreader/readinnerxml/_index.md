## Méthode XmlReader::ReadInnerXml()

Lorsqu'elle est remplacée dans une classe dérivée, lit tout le contenu, balises incluses, sous forme de chaîne.

```cpp
virtual String System::Xml::XmlReader::ReadInnerXml()
```

### Valeur de retour

Tout le contenu XML, balises incluses, du nœud actuel. Si le nœud actuel n'a aucun enfant, une chaîne vide est renvoyée. Si le nœud actuel n'est ni un élément ni un attribut, une chaîne vide est renvoyée.

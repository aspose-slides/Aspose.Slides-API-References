## XmlReader::ReadString() méthode


Lorsqu'elle est surchargée dans une classe dérivée, lit le contenu d'un élément ou d'un nœud texte sous forme de chaîne. Cependant, il est recommandé d'utiliser la méthode [XmlReader::ReadElementContentAsString](../readelementcontentasstring/) à la place, car elle offre une façon plus simple de gérer cette opération.

```cpp
virtual String System::Xml::XmlReader::ReadString()
```


### Valeur de retour

Le contenu de l'élément ou une chaîne vide.

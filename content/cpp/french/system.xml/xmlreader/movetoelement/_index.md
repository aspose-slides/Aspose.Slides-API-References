## XmlReader::MoveToElement() méthode

Lorsqu'elle est remplacée dans une classe dérivée, déplace le lecteur vers l'élément qui contient le nœud d'attribut actuel.

```cpp
virtual bool System::Xml::XmlReader::MoveToElement()=0
```

### Valeur de retour

**true** si le lecteur est positionné sur un attribut (le lecteur se déplace vers l'élément qui possède l'attribut) ; **false** si le lecteur n'est pas positionné sur un attribut (la position du lecteur ne change pas).

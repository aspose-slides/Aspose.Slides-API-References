## XmlNamespaceManager::HasNamespace(String) méthode

Retourne une valeur indiquant si le préfixe fourni possède un espace de noms défini pour la portée poussée actuelle.

```cpp
virtual bool System::Xml::XmlNamespaceManager::HasNamespace(String prefix)
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| prefix | [String](../../../system/string/) | Le préfixe de l'espace de noms que vous souhaitez trouver. |

### Valeur de retour

**true** s'il y a un espace de noms défini ; sinon, **false**.

## XmlDocument::get_PreserveWhitespace() méthode


Renvoie une valeur indiquant s'il faut préserver les espaces blancs dans le contenu de l'élément.

```cpp
bool System::Xml::XmlDocument::get_PreserveWhitespace()
```


### Valeur de retour

**true** pour préserver les espaces blancs ; sinon **false**. La valeur par défaut est **false**.

## XmlDeclaration::get_Standalone() méthode

Renvoie la valeur de l'attribut standalone.

```cpp
String System::Xml::XmlDeclaration::get_Standalone()
```

### Valeur de retour

Les valeurs valides sont **yes** si toutes les déclarations d'entités requises par le document XML sont contenues dans le document ou **no** si une définition de type de document externe (DTD) est requise. Si un attribut standalone n'est pas présent dans la déclaration XML, cette méthode renvoie [String::Empty](../../../system/string/empty/).

## NamespaceHandling enum

Spécifie s’il faut supprimer les déclarations d’espace de noms dupliquées dans le [XmlWriter](../xmlwriter/).

```cpp
enum class NamespaceHandling
```

### Values

| Nom | Valeur | Description |
| --- | --- | --- |
| Default | 0 | Spécifie que les déclarations d’espace de noms dupliquées ne seront pas supprimées. |
| OmitDuplicates | 1 | Spécifie que les déclarations d’espace de noms dupliquées seront supprimées. Pour que l’espace de noms dupliqué soit supprimé, le préfixe et l’espace de noms doivent correspondre. |

## XmlDocumentType::get_InternalSubset() méthode

Renvoie la valeur du sous-ensemble interne de la définition de type de document (DTD) dans la déclaration DOCTYPE.

```cpp
String System::Xml::XmlDocumentType::get_InternalSubset()
```

### Valeur de retour

Le sous-ensemble interne DTD du DOCTYPE. S'il n'y a pas de sous-ensemble interne DTD, [String::Empty](../../../system/string/empty/) est retourné.

## XmlNodeOrder enum

Décrit l'ordre du document d'un nœud par rapport à un deuxième nœud.

```cpp
enum class XmlNodeOrder
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Before | 0 | Le nœud actuel de ce navigateur se trouve avant le nœud actuel du navigateur fourni. |
| After | 1 | Le nœud actuel de ce navigateur se trouve après le nœud actuel du navigateur fourni. |
| Same | 2 | Les deux navigateurs sont positionnés sur le même nœud. |
| Unknown | 3 | Les positions des nœuds ne peuvent pas être déterminées dans l'ordre du document, l'une par rapport à l'autre. Cela peut se produire si les deux nœuds se trouvent dans des arbres différents. |

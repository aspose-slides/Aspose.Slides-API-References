## XmlCharacterData::Substring(int32_t, int32_t) méthode


Récupère une sous-chaîne de la chaîne complète à partir de la plage spécifiée.

```cpp
virtual String System::Xml::XmlCharacterData::Substring(int32_t offset, int32_t count)
```


### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| offset | **int32_t** | La position dans la chaîne où commencer la récupération. Un décalage de zéro indique que le point de départ est au début des données. |
| count | **int32_t** | Le nombre de caractères à récupérer. |

### Valeur de retour

La sous-chaîne correspondant à la plage spécifiée.

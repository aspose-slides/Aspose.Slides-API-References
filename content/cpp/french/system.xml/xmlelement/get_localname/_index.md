## XmlElement::get_LocalName() méthode

Renvoie le nom local du nœud actuel.

```cpp
String System::Xml::XmlElement::get_LocalName() override
```

### Valeur de retour

Le nom du nœud actuel sans le préfixe. Par exemple, la valeur de **get_LocalName** est **book** pour l'élément **<bk:book>**.

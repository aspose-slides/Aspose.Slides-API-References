## XmlElement::get_IsEmpty() méthode

Renvoie le format de balise de l'élément.

```cpp
bool System::Xml::XmlElement::get_IsEmpty()
```

### Valeur de retour

**true** si l'élément doit être sérialisé au format de balise court \"<tt><item/></tt>\" ; **false** pour le format long \"<tt><item></item></tt>\".

## XmlElement::set_IsEmpty(bool) méthode


Définit le format de balise de l'élément.

```cpp
void System::Xml::XmlElement::set_IsEmpty(bool value)
```


### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| value | **bool** | La valeur à définir. Lors de la définition de cette valeur, si elle est **true**, les enfants de l'élément sont supprimés et l'élément est sérialisé au format de balise courte. Si elle est **false**, la valeur est modifiée (indépendamment du fait que l'élément contienne ou non du contenu) ; si l'élément est vide, il est sérialisé au format long. |

## XmlNodeList::idx_get(int32_t) méthode


Renvoie un nœud à l'index donné.

```cpp
virtual SharedPtr<XmlNode> System::Xml::XmlNodeList::idx_get(int32_t i)
```


### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| i | **int32_t** | L'index basé sur zéro dans la liste des nœuds. |

### Valeur de retour

Le [XmlNode](../../xmlnode/) avec l'index spécifié dans la collection. Si l'index est supérieur ou égal au nombre de nœuds dans la liste, cela renvoie **nullptr**.

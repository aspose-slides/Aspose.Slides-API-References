## XmlAttribute::get_Specified() méthode


Renvoie une valeur indiquant si la valeur de l'attribut a été définie explicitement.

```cpp
virtual bool System::Xml::XmlAttribute::get_Specified()
```


### Valeur de retour

**true** si cet attribut a été explicitement donné une valeur dans le document d'instance original ; sinon, **false**. Une valeur **false** indique que la valeur de l'attribut provient de la DTD.

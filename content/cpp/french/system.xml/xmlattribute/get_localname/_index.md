## XmlAttribute::get_LocalName() méthode

Renvoie le nom local du nœud.

```cpp
String System::Xml::XmlAttribute::get_LocalName() override
```

### Valeur de retour

Le nom du nœud d'attribut sans le préfixe. Dans l'exemple suivant **<book bk:genre= 'novel'>**, le **LocalName** de l'attribut est **genre**.

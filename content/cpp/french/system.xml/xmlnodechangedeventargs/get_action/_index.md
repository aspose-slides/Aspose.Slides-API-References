## XmlNodeChangedEventArgs::get_Action() méthode

Renvoie une valeur indiquant le type d'événement de modification de nœud qui se produit.

```cpp
XmlNodeChangedAction System::Xml::XmlNodeChangedEventArgs::get_Action()
```

### Valeur de retour

L'une des valeurs d'énumération qui décrit l'événement de modification de nœud. La valeur **Action** ne différencie pas le moment où l'événement s'est produit (avant ou après). Vous pouvez créer des gestionnaires d'événements distincts pour gérer les deux cas.

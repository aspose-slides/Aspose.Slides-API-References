## XmlTextWriter::get_XmlSpace() méthode


Renvoie un XmlSpace représentant la portée actuelle de **xml:space**.

```cpp
System::Xml::XmlSpace System::Xml::XmlTextWriter::get_XmlSpace() override
```


### Valeur de retour

Un XmlSpace représentant la portée actuelle de **xml:space**.



| Valeur | Signification |
| --- | --- |
| None | Ceci est la valeur par défaut s'il n'existe aucune portée `xml:space`. |
| Default | La portée actuelle est `xml:space="default"`. |
| Preserve | La portée actuelle est `xml:space="preserve"`. |

## XmlTextWriter::get_QuoteChar() méthode


Retourne le caractère à utiliser pour entourer les valeurs d'attribut avec des guillemets.

```cpp
char16_t System::Xml::XmlTextWriter::get_QuoteChar()
```


### Valeur de retour

Le caractère à utiliser pour encadrer les valeurs d'attribut. Il doit s'agir d'une apostrophe simple **'** ou d'un guillemet double **\"**. La valeur par défaut est un guillemet double.

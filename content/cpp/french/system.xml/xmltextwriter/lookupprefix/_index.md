## XmlTextWriter::LookupPrefix(String) méthode

Renvoie le préfixe le plus proche défini dans la portée de l'espace de noms actuel pour l'URI de l'espace de noms.

```cpp
String System::Xml::XmlTextWriter::LookupPrefix(String ns) override
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| ns | [String](../../../system/string/) | Namespace URI whose prefix you want to find. |

### Valeur de retour

Le préfixe correspondant. Ou **nullptr** si aucun URI d'espace de noms correspondant n'est trouvé dans la portée actuelle.

## XmlNode::GetPrefixOfNamespace(String) method


Recherche la déclaration **xmlns** la plus proche pour l'URI de l'espace de noms donné qui est en portée pour le nœud actuel et renvoie le préfixe défini dans cette déclaration.

```cpp
virtual String System::Xml::XmlNode::GetPrefixOfNamespace(String namespaceURI)
```


### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| namespaceURI | [String](../../../system/string/) | L'URI de l'espace de noms dont vous voulez trouver le préfixe. |

### Valeur de retour

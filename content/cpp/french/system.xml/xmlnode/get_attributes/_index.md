## XmlNode::get_Attributes() méthode

Renvoie un [XmlAttributeCollection](../../xmlattributecollection/) contenant les attributs de ce nœud.

```cpp
virtual SharedPtr<XmlAttributeCollection> System::Xml::XmlNode::get_Attributes() final
```

### Valeur de retour

Un [XmlAttributeCollection](../../xmlattributecollection/) contenant les attributs du nœud. Si le nœud est de type [XmlNodeType::Element](../../xmlnodetype/), les attributs du nœud sont retournés. Sinon, cette méthode renvoie **nullptr**.

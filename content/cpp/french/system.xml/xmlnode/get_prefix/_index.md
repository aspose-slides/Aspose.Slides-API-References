## XmlNode::get_Prefix() méthode

Renvoie le préfixe d'espace de noms de ce nœud.

```cpp
virtual String System::Xml::XmlNode::get_Prefix()
```

### Valeur de retour

Le préfixe d'espace de noms de ce nœud. Par exemple, **Prefix** est **bk** pour l'élément **<bk:book>**. S'il n'y a pas de préfixe, cette méthode renvoie [String::Empty](../../../system/string/empty/).

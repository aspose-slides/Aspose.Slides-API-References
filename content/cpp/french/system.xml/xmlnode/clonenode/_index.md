## XmlNode::CloneNode(bool) méthode

Crée un doublon du nœud, lorsqu'il est redéfini dans une classe dérivée.

```cpp
virtual SharedPtr<XmlNode> System::Xml::XmlNode::CloneNode(bool deep)=0
```

### Arguments

| Parameter | Type | Description |
| --- | --- | --- |
| deep | **bool** | **true** pour cloner récursivement le sous-arbre sous le nœud spécifié; **false** pour cloner uniquement le nœud lui-même. |

### Valeur de retour

Le nœud cloné.

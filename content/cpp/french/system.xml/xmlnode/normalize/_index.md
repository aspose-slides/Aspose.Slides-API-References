## XmlNode::Normalize() méthode

Place tous les [XmlText](../../xmltext/) noeuds à toute profondeur du sous arborescence sous ce [XmlNode](../) dans une forme \"normale\" où seul le balisage (c’est à dire les balises, les commentaires, les instructions de traitement, les sections CDATA et les références d’entité) sépare les [XmlText](../../xmltext/) noeuds, c’est à dire qu’il n’y a pas de [XmlText](../../xmltext/) noeuds adjacents.

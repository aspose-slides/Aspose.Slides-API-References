## XmlWriterSettings::get_CheckCharacters() méthode


Renvoie une valeur qui indique si l'écrivain XML doit vérifier que tous les caractères du document sont conformes à la section \"2.2 Characters\" du W3C [XML 1.0 Recommendation](https://www.w3.org/TR/REC-xml/#charsets).

```cpp
bool System::Xml::XmlWriterSettings::get_CheckCharacters()
```


### Valeur de retour

**true** to do character checking; otherwise, **false**. The default is **true**.

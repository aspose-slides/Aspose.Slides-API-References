## XmlWriterSettings::get_WriteEndDocumentOnClose() méthode

Renvoie une valeur qui indique si le [XmlWriter](../../xmlwriter/) ajoutera des balises de fermeture à toutes les balises d'élément non fermées lorsque la méthode [XmlWriter::Close](../../xmlwriter/close/) est appelée.

```cpp
bool System::Xml::XmlWriterSettings::get_WriteEndDocumentOnClose()
```

### Valeur de retour

**true** si toutes les balises d'élément non fermées seront fermées ; sinon, **false**. La valeur par défaut est **true**.

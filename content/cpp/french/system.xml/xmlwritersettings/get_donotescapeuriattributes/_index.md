## XmlWriterSettings::get_DoNotEscapeUriAttributes() méthode

Renvoie une valeur qui indique si le [XmlWriter](../../xmlwriter/) n'échappe pas les attributs URI.

```cpp
bool System::Xml::XmlWriterSettings::get_DoNotEscapeUriAttributes()
```

**true** si le [XmlWriter](../../xmlwriter/) n'échappe pas les attributs URI ; sinon, **false**. La valeur par défaut est **false**.

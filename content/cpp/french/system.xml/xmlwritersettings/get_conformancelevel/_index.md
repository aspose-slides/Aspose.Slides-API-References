## XmlWriterSettings::get_ConformanceLevel() méthode


Renvoie le niveau de conformité que l'écrivain XML vérifie dans la sortie XML.

```cpp
System::Xml::ConformanceLevel System::Xml::XmlWriterSettings::get_ConformanceLevel()
```


### Valeur de retour

L'une des valeurs d'énumération qui spécifie le niveau de conformité (document, fragment ou détection automatique). La valeur par défaut est [ConformanceLevel::Document](../../conformancelevel/).

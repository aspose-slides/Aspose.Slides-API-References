## XmlWriterSettings::set_NamespaceHandling(System::Xml::NamespaceHandling) method


Définit une valeur indiquant si le [XmlWriter](../../xmlwriter/) doit supprimer les déclarations d'espace de noms dupliquées lors de l'écriture du contenu XML. Le comportement par défaut est que l'écrivain émet toutes les déclarations d'espace de noms présentes dans le résolveur d'espaces de noms de l'écrivain.

```cpp
void System::Xml::XmlWriterSettings::set_NamespaceHandling(System::Xml::NamespaceHandling value)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| value | [System::Xml::NamespaceHandling](../../namespacehandling/) | La valeur à définir. |

## XmlWriterSettings::get_Indent() méthode

Renvoie une valeur indiquant s'il faut indenter les éléments.

```cpp
bool System::Xml::XmlWriterSettings::get_Indent()
```

### Valeur de retour

**true** pour écrire chaque élément sur une nouvelle ligne et l’indenter ; sinon, **false**. La valeur par défaut est **false**.

## XmlWriterSettings::get_NamespaceHandling() méthode

Renvoie une valeur qui indique si le [XmlWriter](../../xmlwriter/) doit supprimer les déclarations d'espace de noms en double lors de l'écriture de contenu XML. Le comportement par défaut est que le writer écrive toutes les déclarations d'espace de noms présentes dans le résolveur d'espaces de noms du writer.

```cpp
System::Xml::NamespaceHandling System::Xml::XmlWriterSettings::get_NamespaceHandling()
```

### Valeur de retour

L'énumération NamespaceHandling utilisée pour spécifier s'il faut supprimer les déclarations d'espace de noms en double dans le [XmlWriter](../../xmlwriter/).

## XmlWriterSettings::get_IndentChars() méthode

Renvoie la chaîne de caractères à utiliser lors de l'indentation. Ce paramètre est utilisé lorsque la valeur [XmlWriterSettings::set_Indent](../set_indent/) est définie sur **true**.

```cpp
String System::Xml::XmlWriterSettings::get_IndentChars()
```

### Valeur de retour

La chaîne de caractères à utiliser lors de l'indentation. Elle peut être définie sur n'importe quelle valeur de chaîne. Cependant, pour garantir un XML valide, vous devez spécifier uniquement des caractères d'espace blancs valides, tels que les espaces, les tabulations, les retours chariot ou les sauts de ligne. La valeur par défaut est de deux espaces.

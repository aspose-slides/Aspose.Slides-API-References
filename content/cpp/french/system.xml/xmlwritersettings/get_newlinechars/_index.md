## XmlWriterSettings::get_NewLineChars() méthode


Renvoie la chaîne de caractères à utiliser pour les sauts de ligne.

```cpp
String System::Xml::XmlWriterSettings::get_NewLineChars()
```


### Valeur de retour

La chaîne de caractères à utiliser pour les sauts de ligne. Elle peut être définie sur n'importe quelle valeur de chaîne. Cependant, pour garantir un XML valide, vous devez spécifier uniquement des caractères d'espace blancs valides, tels que les espaces, les tabulations, les retours chariot ou les sauts de ligne. La valeur par défaut est **\r\n** (retour chariot, nouvelle ligne).

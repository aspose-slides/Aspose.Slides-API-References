## XmlWriterSettings::get_OmitXmlDeclaration() méthode

Renvoie une valeur indiquant s'il faut omettre une déclaration XML.

```cpp
bool System::Xml::XmlWriterSettings::get_OmitXmlDeclaration()
```

### Valeur de retour

**true** pour omettre la déclaration XML ; sinon, **false**. La valeur par défaut est **false**, une déclaration XML est écrite.

## XmlWriterSettings::get_NewLineOnAttributes() méthode


Retourne une valeur indiquant s’il faut écrire les attributs sur une nouvelle ligne.

```cpp
bool System::Xml::XmlWriterSettings::get_NewLineOnAttributes()
```


### Valeur de retour

**true** pour écrire les attributs sur des lignes individuelles ; sinon, **false**. La valeur par défaut est **false**.

## BaseOverrideThemeManager::get_IsOverrideThemeEnabled() méthode


Détermine si [OverrideTheme](../../overridetheme/) remplace le thème effectif hérité ou non. Pour activer [OverrideTheme](../../overridetheme/) pour le remplacement, utilisez les méthodes OverrideTheme.Init*(). Pour empêcher [OverrideTheme](../../overridetheme/) de remplacer, utilisez la méthode [OverrideTheme::Clear](../../overridetheme/clear/). Lecture seule **bool**.

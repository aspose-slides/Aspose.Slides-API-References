## PrintRange enum

Spécifie quelles pages sont imprimées.

```cpp
enum class PrintRange
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| AllPages | 0 | Imprime toutes les pages. |
| Selection | 1 | Imprime les pages sélectionnées. |
| SomePages | 2 | Imprime les pages de la plage spécifiée. |
| CurrentPage | 4194304 | Imprime la page affichée actuellement. |

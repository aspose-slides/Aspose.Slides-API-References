## AlphaFloor classe

Représente un effet Alpha Floor. Les valeurs Alpha (opacité) inférieures à 100 % sont changées à zéro. En d’autres termes, tout ce qui est partiellement transparent devient complètement transparent.

```cpp
class AlphaFloor : public Aspose::Slides::Effects::ImageTransformOperation,
                   public Aspose::Slides::Effects::IAlphaFloor,
                   public Aspose::Slides::Effects::IVisualEffect
```

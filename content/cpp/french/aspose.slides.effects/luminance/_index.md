## Luminance classe


Représente un effet [Luminance](./). La luminosité décale linéairement toutes les couleurs vers le blanc ou le noir. Le contraste ajuste toutes les couleurs pour qu'elles soient plus proches ou plus éloignées les unes des autres.

```cpp
class Luminance : public Aspose::Slides::Effects::ImageTransformOperation,
                  public Aspose::Slides::Effects::ILuminance,
                  public Aspose::Slides::Effects::IVisualEffect
```

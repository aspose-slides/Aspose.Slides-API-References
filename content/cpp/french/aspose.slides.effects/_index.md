## Classes

| Class | Description |
| --- | --- |
| [AlphaBiLevel](./alphabilevel/) | Représente un effet Alpha Bi-Level. Les valeurs Alpha (Opacity) inférieures au seuil sont modifiées à 0 (totalement transparent) et les valeurs alpha supérieures ou égales au seuil sont modifiées à 100% (totalement opaque). |
| [AlphaCeiling](./alphaceiling/) | Représente un effet Alpha Ceiling. Les valeurs Alpha (opacity) supérieures à zéro sont modifiées à 100%. En d’autres termes, tout ce qui est partiellement opaque devient totalement opaque. |
| [AlphaFloor](./alphafloor/) | Représente un effet Alpha Floor. Les valeurs Alpha (opacity) inférieures à 100% sont modifiées à zéro. En d’autres termes, tout ce qui est partiellement transparent devient totalement transparent. |
| [AlphaInverse](./alphainverse/) | Représente un effet Alpha Inverse. Les valeurs Alpha (opacity) sont inversées en soustrayant de 100%. |
| [AlphaModulate](./alphamodulate/) | Représente un effet Alpha Modulate. Les valeurs Alpha (opacity) de l’effet sont multipliées par un pourcentage fixe. Le conteneur d’effet indique un effet contenant les valeurs alpha à moduler. |

| [AlphaReplace](./alphareplace/) | Représente un effet Alpha Replace. Les valeurs Alpha (opacity) de l’effet sont remplacées par un alpha fixe. |
| [BiLevel](./bilevel/) | Représente un effet Bi-Level (noir/blanc). Les couleurs d’entrée dont la luminance est inférieure à la valeur seuil spécifiée sont changées en noir.

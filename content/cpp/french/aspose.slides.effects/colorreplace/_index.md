## ColorReplace classe

Représente un effet de remplacement de couleur. Toutes les couleurs d'effet sont remplacées par une couleur fixe. Les valeurs alpha ne sont pas affectées.

```cpp
class ColorReplace : public Aspose::Slides::Effects::ImageTransformOperation,
                     public Aspose::Slides::Effects::IColorReplace,
                     public Aspose::Slides::Effects::IVisualEffect
```

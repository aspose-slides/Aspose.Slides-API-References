## classe EncoderExceptionFallback

Fournit une stratégie de secours générant une exception. Les objets de cette classe ne doivent être alloués qu'en utilisant la fonction [System::MakeObject()](../../system/makeobject/). Ne jamais créer d'instance de ce type sur la pile ou à l'aide de l'opérateur new, car cela entraînera des erreurs d'exécution et/ou des défauts d'assertion. Toujours encapsuler cette classe dans un pointeur [System::SmartPtr](../../system/smartptr/) et utiliser ce pointeur pour le passer aux fonctions en tant qu'argument.

```cpp
class EncoderExceptionFallback : public System::Text::EncoderFallback
```

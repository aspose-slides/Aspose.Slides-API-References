## TransitionMorphType enum

Représente un type de transition morphologique.

```cpp
enum class TransitionMorphType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| ByObject | 0 | La transition morph sera effectuée en considérant les formes comme des objets indivisibles. |
| ByWord | 1 | La transition morph sera effectuée en transférant le texte par mots lorsque cela est possible. |
| ByChar | 2 | La transition morph sera effectuée en transférant le texte par caractères lorsque cela est possible. |

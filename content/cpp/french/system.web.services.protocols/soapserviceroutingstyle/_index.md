## SoapServiceRoutingStyle enum


Énumère les options de la façon dont un message SOAP est acheminé vers le service XML [Web](../../system.web/).

```cpp
enum class SoapServiceRoutingStyle
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| SoapAction | 0 | Le message SOAP est acheminé sur la base de l'attribut 'SOAPAction'. |
| RequestElement | 1 | Le message SOAP est acheminé sur la base du premier élément enfant après l'élément 'Body'. |

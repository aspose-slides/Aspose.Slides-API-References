## EncoderParameterValueType enum


Spécifie le type d'une valeur représentée par la classe [EncoderParameter](../encoderparameter/).

enum class EncoderParameterValueType

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| ValueTypeByte | 1 | Un entier non signé de 8 bits. |
| ValueTypeAscii | 2 | Un entier non signé de 8 bits contenant un caractère ASCII de 7 bits. |
| ValueTypeShort | 3 | Un entier non signé de 16 bits. |
| ValueTypeLong | 4 | Un entier non signé de 32 bits. |
| ValueTypeRational | 5 | Une fraction spécifiée par deux entiers longs - numérateur et dénominateur. |
| ValueTypeLongRange | 6 | Une plage d'entiers. |
| ValueTypeUndefined | 7 |  |

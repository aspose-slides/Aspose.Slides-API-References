## EmfType enum


Spécifie les types d'enregistrements placés dans un fichier EMF.

```cpp
enum class EmfType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| EmfOnly | n/a | [Windows](../../system.windows/) metafile amélioré. Contient des commandes GDI. Les metafiles de ce type sont appelés fichier EMF. |
| EmfPlusOnly | n/a | [Windows](../../system.windows/) metafile amélioré plus. Contient des commandes GDI+. Les metafiles de ce type sont appelés fichier EMF+. |
| EmfPlusDual | n/a | Dual [Windows](../../system.windows/) metafile amélioré. Contient des commandes GDI et GDI+ équivalentes. Les metafiles de ce type sont appelés fichier EMF+. |

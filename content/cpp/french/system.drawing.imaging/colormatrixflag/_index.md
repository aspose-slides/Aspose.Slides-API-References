## ColorMatrixFlag énum

Spécifie les types d'images et de couleurs qui seront affectés par les paramètres d'ajustement des couleurs et des niveaux de gris d'un objet [ImageAttributes](../imageattributes/).

```cpp
enum class ColorMatrixFlag
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Default | 0 | Les deux, les couleurs et les valeurs en niveaux de gris, sont ajustées en couleur. |
| SkipGrays | 1 | Les valeurs en niveaux de gris ne sont pas ajustées en couleur. |
| AltGrays | 2 | Seules les valeurs en niveaux de gris sont ajustées en couleur. |

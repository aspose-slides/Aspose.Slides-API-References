## MetafileFrameUnit enum

Spécifie l'unité de mesure du rectangle utilisé pour dimensionner et positionner un metafile.

```cpp
enum class MetafileFrameUnit
```

### Valeurs

| Name | Value | Description |
| --- | --- | --- |
| Pixel | n/a | Spécifie un pixel comme unité de mesure. |
| Point | n/a | Spécifie le point d'imprimante comme unité de mesure. |
| Inch | n/a | Spécifie un pouce comme unité de mesure. |
| Document | n/a | Spécifie 1/300 de pouce comme unité de mesure. |
| Millimeter | n/a | Spécifie un millimètre comme unité de mesure. |
| GdiCompatible | n/a | Spécifie .01 millimètre comme unité de mesure. Fourni pour la compatibilité avec GDI. |

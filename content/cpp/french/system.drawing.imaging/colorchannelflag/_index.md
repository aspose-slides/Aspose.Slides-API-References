## ColorChannelFlag enum


Spécifie un canal de couleur dans l'espace colorimétrique CMYK.

```cpp
enum class ColorChannelFlag
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| ColorChannelC | 0 | Le canal couleur cyan. |
| ColorChannelM | 1 | Le canal couleur magenta. |
| ColorChannelY | 2 | Le canal couleur jaune. |
| ColorChannelK | 3 | Le canal couleur noir. |
| ColorChannelLast | 4 | Le dernier canal sélectionné doit être utilisé. |

## ImageLockMode enum

Spécifie les propriétés de la région d'une image qui est verrouillée.

```cpp
enum class ImageLockMode
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| ReadOnly | 1 | La région est verrouillée uniquement pour la lecture. |
| WriteOnly | 2 | La région est verrouillée uniquement pour l'écriture. |
| ReadWrite | n/a | La région est verrouillée à la fois pour la lecture et pour l'écriture. |
| UserInputBuffer | 4 | L'image se trouve dans un tampon d'entrée utilisateur, dont l'accès est contrôlé par l'utilisateur. |

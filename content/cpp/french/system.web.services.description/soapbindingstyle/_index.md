## SoapBindingStyle enum

Spécifie le type d'action dans le service XML [Web](../../system.web/) au niveau de la hiérarchie de classe auquel cette énumération est appliquée.

```cpp
enum class SoapBindingStyle
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Default | 0 | Le type d'action par défaut. |
| Document | 1 | Le message est orienté document. |
| Rpc | 2 | Le message contient les paramètres pour appeler une procédure ou les valeurs de retour de cette procédure. |

## Classes

| Classe | Description |
| --- | --- |
| [DataGrid](./datagrid/) | Classe factice rendant possible le liage du code traduit avec les références [DataGrid](./datagrid/), mais sans l'executer. Ne contient aucun membre correctement implémenté. Les objets de cette classe ne doivent être alloués qu'en utilisant la fonction [System::MakeObject()](../system/makeobject/). Ne créez jamais d'instance de ce type sur la pile ou en utilisant l'opérateur new, car cela entraînera des erreurs d'exécution et/ou des échecs d'assertion. Enveloppez toujours cette classe dans un pointeur [System::SmartPtr](../system/smartptr/) et utilisez ce pointeur pour le passer aux fonctions en tant qu'argument. |

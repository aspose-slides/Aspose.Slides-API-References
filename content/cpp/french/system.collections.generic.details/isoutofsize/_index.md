## System::Collections::Generic::Details::IsOutOfSize(int, const Container\&) fonction

Vérifie si l'indice est hors des limites du conteneur, y compris la taille du conteneur.

```cpp

```

### Paramètres de modèle

| Paramètre | Description |

| Paramètre | Type | Description |

### Valeur de retour

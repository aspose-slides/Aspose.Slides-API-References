## has_method_compareto_shared_ptr struct

Vérifie si la méthode CompareTo(SharedPtr<T>) existe dans le type spécifié. Le cas échéant, hérite de std::true_type, sinon il hérite de std::false_type. Peut être utilisé dans std::enable_if.

```cpp
template<typename T,typename Sfinae>class has_method_compareto_shared_ptr : public std::false_type
```

### Paramètres de modèle

| Paramètre | Description |
| --- | --- |
| T | Type à vérifier pour l'existence de la méthode Equals. |
| Sfinae | Argument de modèle formel pour que SFINAE fonctionne. |

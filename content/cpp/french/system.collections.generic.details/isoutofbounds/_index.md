## System::Collections::Generic::Details::IsOutOfBounds(int, const Container\&) fonction

Vérifie si l’indice est hors des limites du conteneur, à l’exclusion de la taille du conteneur.

```cpp
template<typename Container> bool System::Collections::Generic::Details::IsOutOfBounds(int idx, const Container &container)
```

### Paramètres de modèle

| Paramètre | Description |
| --- | --- |
| Container | Type du conteneur. |

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| idx | int | Indice. |
| container | const Container\& | Conteneur dont l’indice doit être vérifié. |

### Valeur de retour

Vrai si l’indice est hors des limites du conteneur, faux sinon.

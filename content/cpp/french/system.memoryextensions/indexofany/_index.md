## System::MemoryExtensions::IndexOfAny(const ReadOnlySpan\<T\>\&, const T\&, const T\&) fonction

Recherche l'indice de la première occurrence de l'une des deux valeurs spécifiées dans un ReadOnlySpan<T>

```cpp
template<typename T> int32_t System::MemoryExtensions::IndexOfAny(const ReadOnlySpan<T> &span, const T &value0, const T &value1)
```

### Paramètres de modèle

| Paramètre | Description |
| --- | --- |
| T | Le type des éléments dans le span |

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| span | const [ReadOnlySpan](../../system/readonlyspan/)\<T\>\& | Le span à rechercher |
| value0 | const T\& | La première valeur à rechercher |
| value1 | const T\& | La deuxième valeur à rechercher |

### Valeur de retour

L'indice basé sur zéro de la première occurrence, ou -1 si non trouvé

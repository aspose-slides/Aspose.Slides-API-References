## X500DistinguishedNameFlags enum

Règles de formatage du nom distinctif du certificat X509.

```cpp
enum class X500DistinguishedNameFlags
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| None | 0 | Aucune caractéristique spéciale. |
| Reversed | 1 | Le nom est réservé. |
| UseSemicolons | 16 | Utiliser des points-virgules. |
| DoNotUsePlusSign | 32 | Le nom n'utilise pas le signe plus. |
| DoNotUseQuotes | 64 | Désactive les guillemets dans le nom. |
| UseCommas | 128 | Permet l'utilisation de virgules. |
| UseNewLines | 256 | Permet l'utilisation de nouvelles lignes. |
| UseUTF8Encoding | 4096 | Passe de l'utilisation d'Unicode à l'utilisation de l'encodage UTF-8. |
| UseT61Encoding | 8192 | Passe à l'encodage T61. |
| ForceUTF8Encoding | 16384 | Force l'utilisation de l'UTF-8 lors de l'encodage de clés X500 spécifiques. |

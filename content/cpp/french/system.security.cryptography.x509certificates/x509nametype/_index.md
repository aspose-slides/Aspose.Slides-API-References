## X509NameType enum

Type de nom contenu dans un certificat X.509 qui concerne soit l'émetteur, soit le sujet du certificat.

```cpp
enum class X509NameType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| SimpleName | 0 | Nom simple du sujet ou de l'émetteur. |
| EmailName | 1 | Adresse e-mail. |
| UpnName | 2 | Nom UPN. |
| DnsName | 3 | Nom DNS. |
| DnsFromAlternativeName | 4 | Nom DNS alternatif équivalent au nom DNS. |
| UrlName | 5 | URL. |

## Char::IsSurrogate(char_t) méthode

Détermine si le caractère spécifié est une unité de code de substitution UTF-16.

```cpp
static bool System::Char::IsSurrogate(char_t c)
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| c | char_t | Un caractère |

### Valeur de retour

Vrai si le caractère spécifié est une unité de code de substitution UTF-16, sinon - faux

## Char::IsUpper(const String\&, int) méthode


Détermine si le caractère à l'index spécifié dans la chaîne spécifiée est classé comme une lettre majuscule.

```cpp
static bool System::Char::IsUpper(const String &str, int idx)
```


### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| str | const [String](../../string/)\& | La chaîne contenant un caractère. |
| idx | int | Un index basé sur zéro dans la chaîne spécifiée à tester |

### Valeur de retour

True si le caractère à l'index spécifié est une lettre majuscule, sinon - false

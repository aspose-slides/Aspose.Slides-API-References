## Char::IsSeparator(const char_t *, int) méthode

Détermine si le caractère à l'index spécifié dans le tampon de caractères indiqué est classifié comme un séparateur.

```cpp
static bool System::Char::IsSeparator(const char_t *str, int idx)
```

### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| str | const char_t * | Pointeur vers le début du tampon de caractères |
| idx | int | Index basé à zéro dans le tampon indiqué du caractère à tester |

### Valeur de retour

True si le caractère à l'index spécifié est un séparateur, sinon - false

static bool System::Char::IsSeparator(char_t c)

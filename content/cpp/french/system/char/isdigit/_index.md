## Char::IsDigit(const char_t *, int) méthode

Détermine si le caractère à l'indice spécifié dans le tampon de caractères spécifié est classé comme chiffre décimal.

```cpp
static bool System::Char::IsDigit(const char_t *str, int idx)
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| str | const char_t * | Pointeur vers le début du tampon de caractères |
| idx | int | Un indice basé sur zéro dans le tampon spécifié du caractère à tester |

### Valeur de retour

True si le caractère à l'indice spécifié est un chiffre décimal, sinon - false

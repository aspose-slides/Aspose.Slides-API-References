## Char::GetUnicodeCategory(char_t) méthode

Renvoie une valeur qui représente une catégorie Unicode du caractère spécifié.

```cpp
static Globalization::UnicodeCategory System::Char::GetUnicodeCategory(char_t ch)
```

### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| ch | char_t | Le caractère à classer |

### Valeur de retour

Une des valeurs de l'énumération UnicodeCategory représentant la catégorie Unicode à laquelle appartient le caractère spécifié.

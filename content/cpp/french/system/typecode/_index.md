## TypeCode enum

Représente le type d'un objet.

```cpp
enum class TypeCode
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Empty | 0 | Une référence nulle. |
| Object | 1 | Un type général représentant toute référence ou tout type valeur non représenté explicitement par un autre TypeCode. |
| DBNull | 2 | Une valeur null (colonne) de base de données. |
| Boolean | 3 | Un type simple représentant les valeurs [Boolean](../boolean/) de true ou false. |
| Char | 4 | Un type intégral représentant des entiers non signés de 16 bits avec des valeurs comprises entre 0 et 65535. |
| SByte | 5 | Un type intégral représentant des entiers signés de 8 bits avec des valeurs comprises entre -128 et 127. |
| Byte | 6 | Un type intégral représentant des entiers non signés de 8 bits avec des valeurs comprises entre 0 et 255. |
| Int16 | 7 | Un type intégral représentant des entiers signés de 16 bits avec des valeurs comprises entre -32768 et 32767. |
| UInt16 | 8 | Un type intégral représentant des entiers non signés de 16 bits avec des valeurs comprises entre 0 et 65535. |
| Int32 | 9 | Un type intégral représentant des entiers signés de 32 bits avec des valeurs comprises entre -2147483648 et 2147483647. |
| UInt32 | 10 | Un type intégral représentant des entiers non signés de 32 bits avec des valeurs comprises entre 0 et 4294967295. |
| Int64 | 11 | Un type intégral représentant des entiers signés de 64 bits avec des valeurs comprises entre -9223372036854775808 et 9223372036854775807. |
| UInt64 | 12 | Un type intégral représentant des entiers non signés de 64 bits avec des valeurs comprises entre 0 et 18446744073709551615. |
| Single | 13 | Un type à virgule flottante représentant des valeurs allant d'environ 1.5 x 10 -45 à 3.4 x 10 38 avec une précision de 7 chiffres. |
| Double | 14 | Un type à virgule flottante représentant des valeurs allant d'environ 5.0 x 10 -324 à 1.7 x 10 308 avec une précision de 15-16 chiffres. |
| Decimal | 15 | Un type simple représentant des valeurs allant de 1.0 x 10 -28 à environ 7.9 x 10 28 avec 28-29 chiffres significatifs. |
| DateTime | 16 | Un type représentant une valeur de date et d'heure. |
| String | 18 | Un type de classe scellée représentant des chaînes de caractères Unicode. |

## Version::CompareTo(const Version\&) const méthode


Compare les versions représentées par l'objet actuel et l'objet spécifié.

```cpp
int System::Version::CompareTo(const Version &ver) const
```


### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| ver | const [Version](../)\& | Un objet [Version](../) à comparer avec l'objet actuel |

### Valeur de retour

- 1 si le numéro de version représenté par l'objet actuel est inférieur à celui représenté par **ver**; 0 si les numéros de version sont égaux; 1 si le numéro de version représenté par l'objet actuel est supérieur à la valeur représentée par **ver**

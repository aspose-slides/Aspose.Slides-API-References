## Version::Parse(const String\&) méthode

```cpp
static Version System::Version::Parse(const String &input)
```


### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| input | const [String](../../string/)\& | Une chaîne contenant la représentation sous forme de chaîne d'un numéro de version |

### Valeur de retour

Une instance de la classe [Version](../) qui représente le numéro de version équivalent à celui représenté par **input**

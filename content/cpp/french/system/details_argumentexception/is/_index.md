## Details_ArgumentException::Is(const System::TypeInfo\&) const method




```cpp
bool System::Details_ArgumentException::Is(const System::TypeInfo &target) const override
```


### Arguments

| Parameter | Type | Description |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) structure décrivant le type contre lequel tester l'objet actuel. |

### Valeur de retour

True si l'objet est du type indiqué ou d'une de ses sous-classes, false sinon.

## Remarques

Vérifiez si l'objet représente une instance du type décrit par targetType. Analogique de l'opérateur C# 'is'.

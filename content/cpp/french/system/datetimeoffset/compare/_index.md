## DateTimeOffset::Compare(const DateTimeOffset\&, const DateTimeOffset\&) méthode

Compare deux objets [DateTimeOffset](../).

```cpp
static int System::DateTimeOffset::Compare(const DateTimeOffset &first, const DateTimeOffset &second)
```

### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| first | const [DateTimeOffset](../)\& | Premier objet à comparer. |
| second | const [DateTimeOffset](../)\& | Deuxième objet à comparer. |

### Valeur de retour

- 1 - si **first** est antérieur à **second**, 1 - si **first** est postérieur à **second**, 0 - si **first** est égal à **second**.

## DateTimeOffset::operator>(const DateTimeOffset\&) const method


Détermine si l'objet actuel représente la valeur de date et d'heure qui est postérieure à la valeur représentée par l'objet [DateTimeOffset](../) spécifié.

```cpp
bool System::DateTimeOffset::operator>(const DateTimeOffset &other) const
```


### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| other | const [DateTimeOffset](../)\& | L'objet [DateTimeOffset](../) à comparer avec l'objet actuel |

### Valeur de retour

Vrai si la valeur de date et d'heure représentée par l'objet actuel est postérieure à la valeur représentée par **other**, sinon - faux

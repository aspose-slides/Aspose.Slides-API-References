## System::MemberwiseClone(T *) fonction

Effectue une copie superficielle en utilisant le constructeur de copie.

```cpp
template<typename T> SmartPtr<Object> System::MemberwiseClone(T *ptr)
```

### Paramètres de modèle

| Paramètre | Description |
| --- | --- |
| T | Classe à copier par constructeur. Les informations de sous-classe sont perdues. |

### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| ptr | T * | Pointeur vers l'objet à cloner. |

### Valeur de retour

Pointeur vers l'objet cloné.

## System::dynamic_pointer_cast(SmartPtr\<X\> const\&) fonction

Convertit les pointeurs intelligents en utilisant dynamic_cast.

```cpp
template<class Y,class X> SmartPtr<Y> System::dynamic_pointer_cast(SmartPtr<X> const &x)
```


### Paramètres du modèle

| Paramètre | Description |
| --- | --- |
| X | Type d’objet pointé par le pointeur source. |
| Y | Type d’objet pointé par le pointeur cible. |

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| x | [SmartPtr](../smartptr/)\<X\> const\& | Pointeur source. |

### Valeur de retour

Pointeur après conversion.

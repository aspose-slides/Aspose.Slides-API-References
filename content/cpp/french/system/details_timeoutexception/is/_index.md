

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) structure décrivant le type contre lequel tester l'objet actuel. |

### Valeur de retour

True si l'objet est du type balisé ou de sa sous-classe, false sinon.
## Remarques


Vérifiez si l'objet représente une instance du type décrit par targetType. Analogue de l'opérateur C# 'is'.

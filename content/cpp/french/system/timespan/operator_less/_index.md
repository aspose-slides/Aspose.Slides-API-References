## TimeSpan::operator<(TimeSpan) const method


Détermine si l'intervalle de temps représenté par l'objet actuel est plus court que l'intervalle de temps représenté par l'objet spécifié.

```cpp
constexpr bool System::TimeSpan::operator<(TimeSpan value) const
```


### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| value | [TimeSpan](../) | L'objet [TimeSpan](../) à comparer avec l'objet actuel |

### Valeur de retour

True si l'intervalle de temps représenté par l'objet actuel est plus court que l'intervalle de temps représenté par **value**, sinon - false

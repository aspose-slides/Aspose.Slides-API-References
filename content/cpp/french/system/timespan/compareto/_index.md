

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| value | [TimeSpan](../) | L'objet [TimeSpan](../) à comparer avec l'objet actuel |

### Valeur de retour

- 1 si l'objet actuel représente l'intervalle qui est plus court que **value** ; 0 si l'objet actuel représente l'intervalle qui est égal à **value** ; 1 si l'objet actuel représente l'intervalle qui est plus long que **value**

## NullableUtils classe

Représente la classe statique C# [System.Nullable](../nullable/) (sans arguments de type). Impossible d'utiliser le nom original en raison de l'impossibilité de surcharger les modèles de classe en C++. Prend en charge un type valeur qui peut recevoir la valeur null. Cette classe ne peut pas être héritée.

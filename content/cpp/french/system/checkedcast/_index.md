## System::CheckedCast(TFrom) fonction

Détermine si la valeur spécifiée se situe dans l'intervalle des valeurs du type **TTo** et, le cas échéant, la convertit en type **TTo**.

```cpp
template<typename TTo,typename TFrom> TTo System::CheckedCast(TFrom value)
```

### Paramètres de modèle

| Paramètre | Description |
| --- | --- |
| TTo | Le type vers lequel la valeur spécifiée doit être convertie |
| TFrom | Le type de la valeur spécifiée |

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| value | TFrom | La valeur à convertir |

### Valeur de retour

La valeur du type **TTo** équivalente à **value**

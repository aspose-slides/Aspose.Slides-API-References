## System::IsNaN(const T\&) fonction


Détermine si la valeur spécifiée est une valeur Not-A-Number.

```cpp
template<typename T> bool System::IsNaN(const T &value)
```


### Paramètres de modèle

| Paramètre | Description |
| --- | --- |
| T | The type of the value checked by the function |

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| value | const T\& | The value to check |

### Valeur de retour

Vrai si **value** est une valeur NaN, sinon - false

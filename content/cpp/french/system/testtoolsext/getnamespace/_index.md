## TestToolsExt::GetNamespace(const char *, const char *, std::string\&) méthode

Récupère l'espace de noms du test spécifié.

```cpp
static bool System::TestToolsExt::GetNamespace(const char *class_name, const char *method_name, std::string &name_space)
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| class_name | const char * | Classe à rechercher. |
| method_name | const char * | Méthode à rechercher. |
| name_space | std::string\& | Variable dans laquelle placer le nom de l'espace de noms, si trouvé. |

### Valeur de retour

Vrai si la méthode de test est trouvée, faux sinon.

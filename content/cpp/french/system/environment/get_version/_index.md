## Environment::get_Version() méthode


Renvoie l'objet [Version](../../version/) qui représente les informations sur la version du Common Language Runtime. Le numéro de version renvoyé par cette méthode est plutôt factice et ne signifie pas que toutes les classes de la bibliothèque se comportent conformément à la version renvoyée.

```cpp
static Version System::Environment::get_Version()
```

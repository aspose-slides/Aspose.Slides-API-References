## SpecialFolder enum

Représente les dossiers spéciaux du système.

```cpp
enum class SpecialFolder
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Desktop | 0 | Le bureau logique. |
| Programs | 2 | Le dossier qui contient les groupes de programmes de l'utilisateur. |
| Personal | 5 | Le dossier qui sert de référentiel commun pour les documents. |
| MyDocuments | 5 | Le répertoire \"My Documents\". |
| Favorites | 6 | Le dossier qui sert de référentiel commun pour les éléments favoris de l'utilisateur. |
| Startup | 7 | Le dossier qui correspond au groupe de programmes de démarrage de l'utilisateur. |
| Recent | 8 | Le dossier qui contient les documents les plus récemment utilisés par l'utilisateur. |
| SendTo | 9 | Le dossier qui sert de conteneur pour les éléments du menu Envoyer vers. |
| StartMenu | 11 | Le dossier qui sert de conteneur pour les éléments du menu Démarrer. |
| MyMusic | 13 | Le répertoire \"My Music\". |
| DesktopDirectory | 16 | Le dossier utilisé pour stocker physiquement les éléments sur le bureau. |
| MyComputer | 17 | Le répertoire \"My Computer\". |
| Templates | 21 | Le dossier qui sert de conteneur commun pour les modèles de documents. |
| ApplicationData | 26 | Le dossier utilisé pour stocker les données spécifiques à l'application pour l'utilisateur itinérant actuel. |
| LocalApplicationData | 28 | Le dossier utilisé pour stocker les données spécifiques à l'application pour l'utilisateur non itinérant actuel. |
| InternetCache | 32 | Le dossier utilisé comme stockage pour les fichiers temporaires d'Internet. |
| Cookies | 33 | Le dossier utilisé comme stockage pour les cookies Internet. |
| History | 34 | Le dossier utilisé comme stockage pour les éléments d'historique d'Internet. |
| CommonApplicationData | 35 | Le dossier utilisé pour stocker les données spécifiques à l'application utilisées par tous les utilisateurs. |
| System | 37 | Le dossier système. |
| ProgramFiles | 38 | Le dossier des fichiers de programme. |
| MyPictures | 39 | Le dossier \"My picutres\". |
| UserProfile | 40 | Le dossier de profil de l'utilisateur. |
| CommonProgramFiles | 43 | Le répertoire utilisé comme stockage pour les composants partagés par les applications. |

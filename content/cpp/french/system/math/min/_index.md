## Math::Min(T0, T1) méthode

Renvoie la plus petite valeur parmi deux valeurs numériques spécifiées.

```cpp
template<class T0,class T1,class> static auto System::Math::Min(T0 val1, T1 val2) -> decltype(val1+val2)
```

### Paramètres de modèle

| Paramètre | Description |
| --- | --- |
| T0 | Type de la première valeur. |
| T1 | Type de la deuxième valeur. |

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| val1 | T0 | Première valeur. |
| val2 | T1 | Deuxième valeur. |

### Valeur de retour

Valeur la plus petite entre **val1** et **val2**.

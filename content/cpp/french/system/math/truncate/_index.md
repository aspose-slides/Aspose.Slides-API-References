## Math::Truncate(const Decimal\&) méthode

Renvoie le [Decimal](../../decimal/) objet représentant une valeur dont la partie entière est égale à celle de la valeur représentée par le [Decimal](../../decimal/) spécifié, tous les chiffres fractionnaires étant supprimés.

```cpp
static Decimal System::Math::Truncate(const Decimal &d)
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| d | const [Decimal](../../decimal/)\& | Un nombre décimal |

### Valeur de retour

Un [Decimal](../../decimal/) objet représentant une valeur décimale dont la partie entière est égale à celle de la valeur spécifiée et dont les chiffres fractionnaires sont tous à 0.

## Math::Truncate(double) méthode

Renvoie une valeur à virgule flottante double précision dont la partie entière est égale à celle de la valeur spécifiée, tous les chiffres fractionnaires étant supprimés.

```cpp
static double System::Math::Truncate(double d)
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| d | **double** | Un nombre décimal |

### Valeur de retour

Une valeur à virgule flottante double précision dont la partie entière est égale à celle de la valeur spécifiée et dont les chiffres fractionnaires sont tous à 0.

Renvoie la plus grande valeur à virgule flottante simple précision parmi les deux spécifiées.



### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| val1 | **float** | Le premier comparand |
| val2 | **float** | Le second comparand |

### Valeur de retour

La plus grande valeur parmi **val1** et **val2** ; ou NaN si l’un des opérandes est NaN



### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| val1 | **float** | Le premier comparand |
| val2 | **float** | Le deuxième comparand |

### Valeur de retour

La plus petite valeur parmi **val1** et **val2** ; ou NaN si l'un des opérandes est NaN

## Math::Abs(T) méthode


Renvoie la valeur absolue de la valeur spécifiée.

```cpp
template<class T> static T System::Math::Abs(T value)
```


### Paramètres du modèle

| Paramètre | Description |
| --- | --- |
| T | Le type de la valeur acceptée par la méthode comme argument |

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| value | T | Une valeur de type arithmétique |

### Valeur de retour

La valeur absolue de **value**

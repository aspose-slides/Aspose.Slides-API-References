## System::GEqual(const ExpressionT\&, const ConstantT\&) fonction

```cpp
template<class ExpressionT,class ConstantT> bool System::GEqual(const ExpressionT &left, const ConstantT &constant)
```


### Paramètres de modèle

| Paramètre | Description |
| --- | --- |
| ExpressionT | type d'expression gauche. |
| ConstantT | type d'expression constante. |

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| left | const ExpressionT\& | expression qui sera vérifiée. |
| constant | const ConstantT\& | expression qui sera comparée avec celle de gauche. |

### Valeur de retour

true si la vérification du type réussit, false sinon.

## System::Equals(const TA\&, const TB\&) fonction

Détermine l'égalité de deux valeurs en appliquant [operator==()](../operator_equal_equal/) à celles-ci.

```cpp
template<typename TA,typename TB> bool System::Equals(const TA &a, const TB &b)
```

### Paramètres de modèle

| Paramètre | Description |
| --- | --- |
| TA | Le type du premier comparand |
| TB | Le type du deuxième comparand |

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| a | const TA\& | Le premier comparand |
| b | const TB\& | Le deuxième comparand |

### Valeur de retour

La valeur booléenne renvoyée par [operator==()](../operator_equal_equal/) appliquée à **a** et **b**

## Boolean::TryParse(const String\&, bool\&) méthode

Convertit la chaîne spécifiée en une valeur de type bool.

```cpp
static bool System::Boolean::TryParse(const String &value, bool &result)
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| value | const [String](../../string/)\& | La chaîne à convertir. |
| result | **bool**\& | La référence à une variable bool où le résultat de la conversion est placé ; le résultat est true si la chaîne spécifiée est égale à \"True\" et false si la chaîne spécifiée est égale à \"False\". |

### Valeur de retour

True si la chaîne spécifiée est égale soit à \"True\" soit à \"False\", sinon - false.

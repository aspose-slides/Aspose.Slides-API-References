## Boolean::Parse(const String\&) méthode

Convertit la chaîne spécifiée en une valeur de type bool.

```cpp
static bool System::Boolean::Parse(const String &value)
```

### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| value | const [String](../../string/)\& | La chaîne à convertir. |

### Valeur de retour

La variable bool où le résultat de la conversion est placé ; le résultat est true si la chaîne spécifiée est égale à \"True\" et false si la chaîne spécifiée est égale à \"False\".

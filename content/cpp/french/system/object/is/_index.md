## Object::Is(const TypeInfo\&) const méthode

Vérifiez si l'objet représente une instance du type décrit par targetType. Analogue de l'opérateur 'is' de C#.

```cpp
virtual bool System::Object::Is(const TypeInfo &targetType) const
```

### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| targetType | const [TypeInfo](../../typeinfo/)\& | structure [TypeInfo](../../typeinfo/) décrivant le type à tester contre l'objet actuel. |

### Valeur de retour

Vrai si l'objet est du type indiqué ou de l'une de ses sous-classes, sinon faux.

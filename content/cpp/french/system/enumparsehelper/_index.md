## struct EnumParseHelper

Classe d’aide qui fournit la fonctionnalité de conversion d’une représentation sous forme de chaîne d’un constant d’énumération en valeur d’énumération équivalente.

```cpp
template<class E,class G,class Guard>class EnumParseHelper
```

### Paramètres de modèle

| Paramètre | Description |
| --- | --- |
| E | The type of the enum with which the class' methods operate |
| G | Second formal argument of [System::Enum](../enum/) to use. |
| Guard | Formal template argument to select parsing algorithm available. |

## Méthodes

| Méthode | Description |
| --- | --- |
| static E [Parse](./parse/)(const [String](../string/)\&, **bool**) | Convertit la chaîne spécifiée en valeur de constant d’énumération équivalente. |

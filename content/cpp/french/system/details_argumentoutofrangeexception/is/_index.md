
### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) structure décrivant le type à tester par rapport à l'objet actuel. |

### Valeur de retour

Vérifie si l'objet représente une instance du type décrit par targetType. Analogue de l'opérateur C# 'is'.

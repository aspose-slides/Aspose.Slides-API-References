```cpp
bool System::Details_RankException::Is(const System::TypeInfo &target) const override
```


### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) structure décrivant le type contre lequel tester l'objet actuel. |

### Valeur de retour

Vrai si l'objet est du type balisé ou de l'une de ses sous-classes, sinon faux.
## Remarques

Vérifie si l'objet représente une instance du type décrit par targetType. Analogue de l'opérateur C# 'is'.

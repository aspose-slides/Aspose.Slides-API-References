## Uri::IsHexEncoding(const String\&, int32_t) méthode

Détermine si un caractère dans la chaîne spécifiée à la position spécifiée est encodé en hexadécimal.

```cpp
static bool System::Uri::IsHexEncoding(const String &pattern, int32_t index)
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| pattern | const [String](../../string/)\& | La chaîne à vérifier |
| index | **int32_t** | Position zéro-base du caractère à tester |

### Valeur de retour

True si le caractère spécifié est encodé en hexadécimal, sinon - false

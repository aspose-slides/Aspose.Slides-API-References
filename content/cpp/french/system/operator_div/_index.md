## System::operator/(const T\&, const Decimal\&) fonction

Renvoie une nouvelle instance de la classe [Decimal](../decimal/) qui représente une valeur résultant de la division de la valeur spécifiée et de la valeur représentée par l’objet [Decimal](../decimal/) spécifié.

```cpp
template<typename T,typename _> Decimal System::operator/(const T &x, const Decimal &d)
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| x | const T\& | La valeur à diviser |
| d | const [Decimal](../decimal/)\& | L’objet [Decimal](../decimal/) représentant le diviseur |

### Valeur de retour

Une nouvelle instance de la classe [Decimal](../decimal/) qui représente une valeur résultant de la division de **x** par la valeur représentée par **d**.

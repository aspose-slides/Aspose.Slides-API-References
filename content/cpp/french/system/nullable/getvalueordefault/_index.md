## Nullable::GetValueOrDefault(T) méthode

Renvoie la valeur représentée par l'objet actuel ou la valeur spécifiée si la valeur représentée par l'objet actuel est nulle.

```cpp
T System::Nullable<T>::GetValueOrDefault(T default_value)
```

### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| default_value | T | La valeur renvoyée par la méthode si la valeur représentée par l'objet actuel est nulle |

### Valeur de retour

Une copie de la valeur représentée par l'objet actuel est retournée si elle n'est pas nulle, sinon la valeur spécifiée est renvoyée

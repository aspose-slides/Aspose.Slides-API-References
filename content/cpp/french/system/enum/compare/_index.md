## Enum::Compare(E, T) méthode


Effectue la comparaison arithmétique des valeurs des constantes d'énumération spécifiées.

```cpp
template<typename T,typename _> static int System::Enum<E, Guard>::Compare(E a, T b)
```


### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| a | E | Le premier comparand |
| b | T | Le second comparand |

### Valeur de retour

- 1 si **a** est inférieur à **b**; 0 si les valeurs sont égales; 1 si **a** est supérieur à **b**

## SmartPtrInfo::operator!() const méthode

Vérifie si l'objet info ne pointe pas vers un pointeur non nul.

```cpp
bool System::SmartPtrInfo::operator!() const
```

### Valeur de retour

Faux si l'objet info ne pointe vers aucun pointeur ou si le pointeur référencé est nul, vrai sinon.

## Détails_TypeInitializationException::Is(const System::TypeInfo\&) const method

```cpp
bool System::Details_TypeInitializationException::Is(const System::TypeInfo &target) const override
```

### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) structure décrivant le type contre lequel tester l'objet actuel. |

### Valeur de retour

True si l'objet est du type indiqué ou de l'une de ses sous-classes, false sinon.

## Remarques

Vérifie si l'objet représente une instance du type décrit par targetType. Analogue de l'opérateur 'is' de C#.

## String::end() const méthode

Retourne un pointeur vers la fin du tampon de chaîne actuel. Ne réalloue jamais rien. Ne garantit pas que le tampon soit terminé par un caractère nul.

```cpp
const UChar * System::String::end() const
```

### Valeur de retour

Pointeur vers le premier élément après le tampon de chaîne actuel, terminé par un caractère nul ou non.

## String::rend() const méthode


Renvoie un itérateur inverse vers le caractère précédant le premier caractère (le cas échéant) du tampon de chaîne réel.

```cpp
reverse_iterator System::String::rend() const
```


### Valeur de retour

Itérateur inverse vers l'élément précédant le premier caractère du tampon de chaîne réel (le cas échéant) ou vers l'élément juste avant le tampon alloué.

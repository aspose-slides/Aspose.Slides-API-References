## String::operator==(const String\&) const méthode


Opérateur de comparaison d'égalité.

```cpp

```


### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| str | const [String](../)\& | [String](../) pour comparer l'actuel à. |

### Valeur de retour

bool System::String::operator==(std::nullptr_t) const

### Valeur de retour

## String::begin() const méthode


Renvoie un pointeur vers le début du tampon de chaîne réel. Ne réalloue jamais rien. Ne garantit pas que le tampon soit terminé par un caractère nul.

```cpp
const UChar * System::String::begin() const
```


### Valeur de retour

Pointeur vers le tampon de chaîne réel, terminé par un caractère nul ou non.

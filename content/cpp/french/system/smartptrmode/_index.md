## SmartPtrMode enum

[SmartPtr](../smartptr/) type de pointeur: weak ou shared. Définit si le pointeur est compté lorsqu'il est décidé de supprimer l'objet ou non.

```cpp
enum class SmartPtrMode : char
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Shared | 0 | Shared mode: le pointeur participe au comptage des références. |
| Weak | 1 | Weak mode: le pointeur ne participe pas au comptage des références. |

## Array::crbegin() const méthode


Renvoie un itérateur inversé vers le premier élément du conteneur inversé. Il correspond au dernier élément du conteneur non inversé. Si le conteneur est vide, l'itérateur retourné est égal à [crend()](../crend/).

```cpp
const_reverse_iterator System::Array<T>::crbegin() const noexcept
```


### Valeur de retour

Un itérateur pointant vers le dernier élément qualifié const du conteneur.

## Array::cend() const méthode

Renvoie un itérateur vers l'élément suivant le dernier élément du conteneur. Cet élément agit comme un espace réservé; tenter d'y accéder entraîne un comportement indéfini.

```cpp
const_iterator System::Array<T>::cend() const noexcept
```

### Valeur de retour

Un itérateur pointant vers l'élément théorique const-qualifié placé après le dernier élément du conteneur.

## Array::cbegin() const méthode

Renvoie un itérateur vers le premier élément const-qualifié du conteneur. Si le conteneur est vide, l’itérateur renvoyé sera égal à [cend()](../cend/).

```cpp
const_iterator System::Array<T>::cbegin() const noexcept
```

### Valeur de retour

Un itérateur pointant vers le premier élément const-qualifié du conteneur.

## Array::Resize(ArrayPtr\<Type\>\&, int) méthode


Modifie la taille du tableau spécifié à la valeur indiquée ou crée un nouveau tableau avec la taille spécifiée.

```cpp
template<typename Type> static void System::Array<T>::Resize(ArrayPtr<Type> &arr, int new_size)
```


### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| arr | [ArrayPtr](../../arrayptr/)\<[Type](../../object/type/)\>\& | [Array](../) à redimensionner. Si **arr** est un pointeur nul, le nouveau tableau sera créé |
| new_size | int | La nouvelle taille du tableau, ou la taille du nouveau tableau si **arr** est nul |

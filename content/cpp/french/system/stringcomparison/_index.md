## Enum StringComparison

Définit le style de comparaison de chaînes.

```cpp
enum class StringComparison
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| CurrentCulture | 0 | Utilise la culture actuelle. |
| CurrentCultureIgnoreCase | 1 | Utilise la culture actuelle, mais ignore la casse. |
| InvariantCulture | 2 | Utilise la culture invariante. |
| InvariantCultureIgnoreCase | 3 | Utilise la culture invariante, mais ignore la casse. |
| Ordinal | 4 | Utilise les règles de tri ordinales. |
| OrdinalIgnoreCase | 5 | Utilise les règles de tri ordinales, mais ignore la casse. |

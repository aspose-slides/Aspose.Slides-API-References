## Decimal::operator<(const Decimal\&) const méthode

Détermine si la valeur représentée par l'objet actuel est inférieure à la valeur représentée par l'objet spécifié.

```cpp
bool System::Decimal::operator<(const Decimal &d) const
```

### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| d | const [Decimal](../)\& | L'objet [Decimal](../) à comparer à l'objet actuel |

### Valeur de retour

True si la valeur représentée par l'objet actuel est inférieure à la valeur représentée par **d**, sinon - false

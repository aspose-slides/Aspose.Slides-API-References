## Decimal::CompareTo(const Decimal\&) const méthode

Détermine si la valeur représentée par l'objet actuel est inférieure, égale ou supérieure à la valeur représentée par l'objet spécifié.

```cpp
int System::Decimal::CompareTo(const Decimal &d) const
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| d | const [Decimal](../)\& | Le comparand |

### Valeur de retour

- 1 si la valeur représentée par l'objet actuel est inférieure à la valeur représentée par **d** ; 0 si les valeurs sont égales ; 1 si la valeur représentée par l'objet actuel est supérieure à la valeur représentée par **d**

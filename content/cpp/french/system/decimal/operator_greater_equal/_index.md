## Decimal::operator>=(const Decimal\&) const méthode


Détermine si la valeur représentée par l'objet actuel est supérieure ou égale à la valeur représentée par l'objet spécifié.

```cpp
bool System::Decimal::operator>=(const Decimal &d) const
```


### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| d | const [Decimal](../)\& | L'objet [Decimal](../) à comparer avec l'objet actuel |

### Valeur de retour

Vrai si la valeur représentée par l'objet actuel est supérieure ou égale à la valeur représentée par **d**, sinon - faux

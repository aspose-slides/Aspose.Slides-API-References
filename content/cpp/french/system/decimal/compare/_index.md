## Decimal::Compare(const Decimal\&, const Decimal\&) méthode

Détermine si la valeur représentée par le premier objet [Decimal](../) est inférieure, égale ou supérieure à la valeur représentée par le deuxième objet [Decimal](../).

```cpp
static int System::Decimal::Compare(const Decimal &d1, const Decimal &d2)
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| d1 | const [Decimal](../)\& | Le premier comparand |
| d2 | const [Decimal](../)\& | Le deuxième comparand |

### Return Value

- 1 si la valeur représentée par **d1** est inférieure à la valeur représentée par **d2** ; 0 si les valeurs sont égales ; 1 si la valeur représentée par **d1** est supérieure à la valeur représentée par **d2**

## classe ValueType

Classe de base pour les types valeur dont l'héritage [Object](../object/) est tronqué pour des raisons de performance. Ce type doit être alloué sur la pile et passé aux fonctions par valeur ou par référence. N'utilisez jamais la classe [System::SmartPtr](../smartptr/) pour gérer des objets de ce type.

```cpp
class ValueType
```

## Guid::CompareTo(const Guid\&) const méthode

Effectue une comparaison arithmétique des GUID représentés par l'objet actuel et l'objet spécifié.

```cpp
int System::Guid::CompareTo(const Guid &g) const
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| g | const [Guid](../)\& | Le comparand |

### Valeur de retour

- 1 si le GUID représenté par l'objet actuel est inférieur au GUID représenté par **g** ; 0 si les GUID sont égaux ; 1 si le GUID représenté par l'objet actuel est supérieur au GUID représenté par **g**

## Details_OperationCanceledException::Is(const System::TypeInfo\&) const method




```cpp
bool System::Details_OperationCanceledException::Is(const System::TypeInfo &target) const override
```


### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) structure décrivant le type contre lequel tester l'objet actuel. |

### Valeur de retour

True si l'objet est de type marqué ou de l'une de ses sous-classes, false sinon.

## Remarques

Vérifier si l'objet représente une instance du type décrit par targetType. Analogue de l'opérateur C# 'is'.

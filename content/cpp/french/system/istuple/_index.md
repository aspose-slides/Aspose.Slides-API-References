## System::IsTuple(const SharedPtr\<Object\>\&, int32_t) fonction

Vérifie si l'objet est un tuple (implémente l'interface ITuple). Utilisé dans l'implémentation des motifs positionnels.

```cpp
bool System::IsTuple(const SharedPtr<Object> &object, int32_t length)
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| object | const [SharedPtr](../sharedptr/)\<[Object](../object/)\>\& | objet à vérifier. |
| length | **int32_t** | longueur de tuple attendue |

### Valeur de retour

l'objet est un tuple et sa longueur est égale à celle attendue

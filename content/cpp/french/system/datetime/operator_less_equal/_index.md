## DateTime::operator<=(DateTime) const méthode

Détermine si l'objet actuel représente la valeur de date et d'heure antérieure ou identique à celle représentée par l'objet [DateTime](../) spécifié.

```cpp
constexpr bool System::DateTime::operator<=(DateTime other) const
```

### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| other | [DateTime](../) | L'objet [DateTime](../) avec lequel comparer l'objet actuel |

### Valeur de retour

Vrai si la valeur de date et d'heure représentée par l'objet actuel est antérieure ou identique à celle représentée par **other**, sinon - faux

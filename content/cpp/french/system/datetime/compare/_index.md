## DateTime::Compare(DateTime, DateTime) méthode

Compare deux valeurs représentées par les instances spécifiées de la classe [DateTime](../) et renvoie la valeur indiquant les positions relatives des valeurs sur la ligne de temps.

```cpp
static constexpr int System::DateTime::Compare(DateTime t1, DateTime t2)
```

### Arguments

| Parameter | Type | Description |
| --- | --- | --- |
| t1 | [DateTime](../) | Le premier comparand |
| t2 | [DateTime](../) | Le deuxième comparand |

### Valeur de retour

Une valeur inférieure à 0 si **t1** est antérieur à **t2** ; 0 si **t1** est identique à **t2** ; une valeur supérieure à 0 si **t1** est postérieur à **t2**

## DateTime::Equals(DateTime) const méthode

Détermine si l'instance spécifiée de la classe [DateTime](../) représente la même valeur de date et d'heure que l'objet actuel.

```cpp
constexpr bool System::DateTime::Equals(DateTime other) const
```

### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| other | [DateTime](../) | L'instance de la classe [DateTime](../) à comparer avec l'objet actuel |

### Valeur de retour

Vrai si **other** et l'objet actuel représentent la même valeur, sinon - faux

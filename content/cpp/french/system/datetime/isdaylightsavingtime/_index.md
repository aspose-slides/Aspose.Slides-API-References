## DateTime::IsDaylightSavingTime() const method


Détermine si la valeur de date et d'heure représentée par l'objet actuel se situe dans la période d'heure d'été pour le fuseau horaire actuel.

```cpp
bool System::DateTime::IsDaylightSavingTime() const
```


### Valeur de retour

True si la valeur de date et d'heure spécifiée par l'objet actuel se situe dans la période d'heure d'été, sinon - false

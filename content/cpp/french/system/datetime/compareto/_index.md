## DateTime::CompareTo(DateTime) const method


Compare deux valeurs de date et d'heure représentées par l'objet actuel et l'instance spécifiée de la classe [DateTime](../) et renvoie la valeur indiquant la position relative des valeurs sur la ligne du temps.

```cpp
constexpr int System::DateTime::CompareTo(DateTime value) const
```


### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| value | [DateTime](../) | Une instance de la classe [DateTime](../) pour comparer l'objet actuel avec |

### Valeur de retour

Une valeur inférieure à 0 si l'objet actuel représente une valeur antérieure à celle représentée par **value** ; 0 si les valeurs représentées par les deux objets sont identiques ; une valeur supérieure à 0 si la valeur représentée par l'objet actuel est postérieure à celle représentée par **value**.

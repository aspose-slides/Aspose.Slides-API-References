## DateTime::FromUnixTime(time_t) méthode

Convertit la valeur de temps Unix spécifiée en une instance de la classe [DateTime](../). POUR USAGE INTERNE.

```cpp
static DateTime System::DateTime::FromUnixTime(time_t value)
```

### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| value | time_t | Valeur de temps Unix à convertir |

### Valeur de retour

Une nouvelle instance de la classe [DateTime](../) représentant la même date et la même heure que **value**

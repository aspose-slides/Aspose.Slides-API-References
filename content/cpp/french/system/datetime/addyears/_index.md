## DateTime::AddYears(int) const méthode

Renvoie une nouvelle instance de la classe [DateTime](../) représentant la valeur de date et d'heure égale à celle représentée par l'objet actuel avec le composant année augmenté du nombre spécifié.

```cpp
DateTime System::DateTime::AddYears(int value) const
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| value | int | Le nombre d'années à ajouter ; la valeur peut être positive ou négative |

### Valeur de retour

Une nouvelle instance de la classe [DateTime](../) représentant la valeur de date et d'heure égale à celle représentée par l'objet actuel avec le composant année augmenté de **value**

## DateTime::operator>=(DateTime) const method


Détermine si l'objet actuel représente la valeur de date et d'heure qui est postérieure ou identique à la valeur représentée par l'objet [DateTime](../) spécifié.

```cpp
constexpr bool System::DateTime::operator>=(DateTime other) const
```


### Arguments

| Parameter | Type | Description |
| --- | --- | --- |
| other | [DateTime](../) | L'objet [DateTime](../) à comparer avec l'objet actuel |

### Valeur de retour

True si la valeur de date et d'heure représentée par l'objet actuel est postérieure ou identique à la valeur représentée par **other**, sinon - false

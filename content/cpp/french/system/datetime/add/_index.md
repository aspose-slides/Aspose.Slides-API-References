## DateTime::Add(TimeSpan) const méthode

Renvoie une nouvelle instance de [DateTime](../) classe qui représente une valeur de date et d'heure résultant de l'ajout de l'intervalle de temps spécifié à la valeur de date et d'heure représentée par l'objet actuel.

```cpp
DateTime System::DateTime::Add(TimeSpan value) const
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| value | [TimeSpan](../../timespan/) | L'intervalle de temps à ajouter |

### Valeur de retour

Une nouvelle instance de [DateTime](../) classe qui représente une valeur de date et d'heure résultant de l'ajout de l'intervalle de temps **tm** à la valeur de date et d'heure représentée par l'objet actuel

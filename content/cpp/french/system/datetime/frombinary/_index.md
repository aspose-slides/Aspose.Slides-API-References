## DateTime::FromBinary(int64_t) méthode

Désérialise la valeur de date et heure à partir de l'entier non signé de 64 bits spécifié et définit la nouvelle instance de la classe [DateTime](../) à cette valeur.

```cpp
static DateTime System::DateTime::FromBinary(int64_t value)
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| value | **int64_t** | L'entier de 64 bits à désérialiser pour la valeur de date et heure |

### Valeur de retour

Une nouvelle instance de la classe [DateTime](../) représentant la valeur de date et heure désérialisée à partir de l'entier non signé de 64 bits spécifié

## SmartPtr::get_shared_count() const méthode


Renvoie le nombre de pointeurs partagés existant vers l'objet référencé, y compris le pointeur actuel. Vérifie que le pointeur actuel est en mode partagé.

```cpp
int System::SmartPtr<T>::get_shared_count() const
```


### Valeur de retour

Nombre de pointeurs partagés existant vers l'objet référencé, le cas échéant. Si le pointeur est nul, renvoie 0.

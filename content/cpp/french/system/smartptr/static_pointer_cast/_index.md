## SmartPtr::static_pointer_cast() const méthode

Convertit le pointeur vers un type différent en utilisant static_cast sur l'objet pointé.

```cpp
template<class Y> SmartPtr<Y> System::SmartPtr<T>::static_pointer_cast() const
```

### Paramètres du modèle

| Paramètre | Description |
| --- | --- |
| Y | Type cible de l'objet pointé. |

### Valeur de retour

Pointeur du type modifié qui est toujours en mode partagé.

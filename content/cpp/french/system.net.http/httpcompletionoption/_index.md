## HttpCompletionOption enum

Indique quand une opération [HttpClient](../httpclient/) doit être terminée.

```cpp
enum class HttpCompletionOption
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| ResponseContentRead | 0 | L'opération doit être terminée après la lecture du contenu de la réponse. |
| ResponseHeadersRead | 1 | L'opération doit être terminée après la lecture des en-têtes d'une réponse. |

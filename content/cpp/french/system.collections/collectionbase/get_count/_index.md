## CollectionBase::get_Count() méthode


Renvoie le nombre d'éléments contenus dans l'instance de la collection. Cette méthode ne peut pas être remplacée.

```cpp
int32_t System::Collections::CollectionBase<T>::get_Count()
```


### Valeur de retour

Le nombre d'éléments contenus dans l'instance de la collection. Récupérer la valeur de cette méthode est une opération O(1).

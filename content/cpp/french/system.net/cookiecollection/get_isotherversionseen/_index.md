## CookieCollection::get_IsOtherVersionSeen() méthode

Renvoie une valeur qui indique si la collection contient un cookie dont la version n’est pas égale à [Cookie::MaxSupportedVersion](../../cookie/maxsupportedversion/).

```cpp
bool System::Net::CookieCollection::get_IsOtherVersionSeen()
```

### Valeur de retour

Vrai lorsque la collection contient un cookie dont la version n’est pas égale à [Cookie::MaxSupportedVersion](../../cookie/maxsupportedversion/), sinon faux.

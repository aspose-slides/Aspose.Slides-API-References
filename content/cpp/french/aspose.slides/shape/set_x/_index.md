## Shape::set_X(float) méthode

Définit la coordonnée x du coin supérieur gauche de la forme, mesurée en points. Écrire **float**.

```cpp
void Aspose::Slides::Shape::set_X(float value) override
```

## Remarques

La valeur renvoyée est toujours définie et n'est jamais [std::numeric_limits<float>::quiet_NaN()](../). La valeur affectée doit également être définie; affectez [std::numeric_limits<float>::quiet_NaN()](../) uniquement aux propriétés d'une instance de **RawFrame**.

## Remarques

La valeur retournée est toujours définie et n'est jamais [std::numeric_limits<float>::quiet_NaN()](../). La valeur assignée doit également être définie ; assignez [std::numeric_limits<float>::quiet_NaN()](../) uniquement aux propriétés d'une instance de **RawFrame**.

## Shape::set_Y(float) méthode

Définit la coordonnée y du coin supérieur gauche de la forme, exprimée en points. Écrire **float**.

```cpp
void Aspose::Slides::Shape::set_Y(float value) override
```

## Remarques

La valeur renvoyée est toujours définie et jamais [std::numeric_limits<float>::quiet_NaN()](../). La valeur assignée doit également être définie ; assigner [std::numeric_limits<float>::quiet_NaN()](../) uniquement aux propriétés d'une instance **RawFrame**.

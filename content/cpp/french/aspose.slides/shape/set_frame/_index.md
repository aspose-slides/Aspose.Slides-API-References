## Shape::set_Frame(System::SharedPtr\<IShapeFrame\>) méthode

Définit les propriétés du cadre de forme. Écrivez [IShapeFrame](../../ishapeframe/).

```cpp
void Aspose::Slides::Shape::set_Frame(System::SharedPtr<IShapeFrame> value) override
```

## Remarques

La valeur de chaque propriété de l'instance [IShapeFrame](../../ishapeframe/) renvoyée n'est pas indéfinie (n'est pas NaN ou NotDefined). La valeur de chaque propriété de l'instance [IShapeFrame](../../ishapeframe/) assignée doit ne pas être indéfinie (doit ne pas être NaN ou NotDefined). Vous pouvez définir des valeurs indéfinies pour les propriétés de l'instance RawFrame.

## IParagraphFormat::set_SpaceWithin(float) méthode

Définit la quantité d'espace entre les lignes de base d'un paragraphe. Une valeur positive représente un pourcentage, une valeur négative - la taille en points. Aucun héritage appliqué. Écrire **float**.

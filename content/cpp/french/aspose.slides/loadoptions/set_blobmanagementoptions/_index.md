## LoadOptions::set_BlobManagementOptions(System::SharedPtr\<IBlobManagementOptions\>) méthode

Représente les options qui peuvent être utilisées pour gérer le comportement de traitement des objets binaires volumineux (BLOB), comme l’utilisation de fichiers temporaires ou le nombre maximal d’octets BLOB en mémoire. Ces options sont destinées à configurer le meilleur rapport performance/consommation de mémoire pour un environnement ou des exigences particuliers.

```cpp
void Aspose::Slides::LoadOptions::set_BlobManagementOptions(System::SharedPtr<IBlobManagementOptions> value) override
```

## Remarques

Un objet binaire volumineux (BLOB) est une donnée binaire stockée comme une entité unique – c’est-à-dire qu’un BLOB peut être un son, une vidéo ou la présentation elle-même.

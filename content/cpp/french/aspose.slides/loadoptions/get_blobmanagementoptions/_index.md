## LoadOptions::get_BlobManagementOptions() méthode

Représente les options qui peuvent être utilisées pour gérer le comportement de traitement des Binary Large Objects (BLOBs), comme l'utilisation de fichiers temporaires ou le nombre maximal d'octets BLOBs en mémoire. Ces options sont destinées à établir le meilleur rapport performance/consommation mémoire pour un environnement ou des exigences particuliers.

```cpp
System::SharedPtr<IBlobManagementOptions> Aspose::Slides::LoadOptions::get_BlobManagementOptions() override
```

## Remarques

Un Binary Large Object (BLOB) est une donnée binaire stockée en tant qu'entité unique - c'est-a-dire qu'un BLOB peut être un audio, une vidéo ou une présentation elle-même.

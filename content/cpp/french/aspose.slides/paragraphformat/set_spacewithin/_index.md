## ParagraphFormat::set_SpaceWithin(float) méthode

Définit la quantité d'espace entre les lignes de base dans un paragraphe. Une valeur positive représente un pourcentage, une valeur négative - une taille en points. Aucun héritage appliqué. Écrire **float**.

## TextAutofitType enum

Représente le mode d'ajustement automatique du texte.

```cpp
enum class TextAutofitType : int8_t
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| NotDefined | -1 | Non défini. |
| None | 0 | Pas d'ajustement automatique. |
| Normal | 1 | Ajustement automatique normal. La taille de police et l'interligne seront réduits pour s'adapter à la forme. |
| Shape | 2 | [Shape](../shape/) autofit. La taille de [Shape](../shape/) sera modifiée pour s'adapter au texte. |

## ILayoutSlideHeaderFooterManager::SetFooterAndChildFootersVisibility(bool) méthode

Modifie la visibilité de l'espace réservé du pied de page de la diapositive de mise en page et de tous les espaces réservés des pieds de page enfants. Les espaces réservés enfants désignent les espaces réservés contenus sur les diapositives dépendantes. Les diapositives dépendantes utilisent et dépendent de la diapositive maîtresse.

```cpp
virtual void Aspose::Slides::ILayoutSlideHeaderFooterManager::SetFooterAndChildFootersVisibility(bool isVisible)=0
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| isVisible | **bool** | true - rend les espaces réservés de pied de page visibles, sinon - les masque. |

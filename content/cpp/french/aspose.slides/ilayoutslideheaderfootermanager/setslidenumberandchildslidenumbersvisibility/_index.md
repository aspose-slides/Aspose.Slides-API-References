## ILayoutSlideHeaderFooterManager::SetSlideNumberAndChildSlideNumbersVisibility(bool) méthode


Modifie la visibilité du marqueur de numéro de page de la diapositive modèle et de tous les marqueurs de numéros de page enfants. Les espaces réservés enfants désignent les espaces réservés contenus dans les diapositives dépendantes. Les diapositives dépendantes utilisent et dépendent de la diapositive modèle.

```cpp
virtual void Aspose::Slides::ILayoutSlideHeaderFooterManager::SetSlideNumberAndChildSlideNumbersVisibility(bool isVisible)=0
```


### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| isVisible | **bool** | true - rend les espaces réservés de numéro de page visibles, sinon - les masque. |

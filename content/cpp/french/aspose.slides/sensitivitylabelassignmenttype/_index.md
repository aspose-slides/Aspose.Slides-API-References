## SensitivityLabelAssignmentType enum

Représente la méthode d'affectation pour l'étiquette de sensibilité.

```cpp
enum class SensitivityLabelAssignmentType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Standard | 0 | Utilisez pour toute étiquette de sensibilité qui n'a pas été appliquée directement par l'utilisateur. Cela inclut toutes les étiquettes par défaut, les étiquettes appliquées automatiquement. |
| Privileged | 1 | Utilisez pour toute étiquette de sensibilité qui a été appliquée directement par l'utilisateur. Cela inclut toutes les étiquettes de sensibilité appliquées manuellement ainsi que l'étiquetage recommandé ou obligatoire ou toute fonctionnalité où l'utilisateur décide quelle étiquette de sensibilité appliquer. |

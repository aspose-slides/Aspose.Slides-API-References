## LayoutSlideHeaderFooterManager::SetDateTimeAndChildDateTimesVisibility(bool) méthode


Modifie la visibilité de l'espace réservé de date-heure de la diapositive de mise en page et de tous les espaces réservés de date-heure enfants. Les espaces réservés enfants désignent les espaces réservés contenus sur les diapositives dépendantes. Les diapositives dépendantes utilisent et dépendent de la diapositive de mise en page.

```cpp
void Aspose::Slides::LayoutSlideHeaderFooterManager::SetDateTimeAndChildDateTimesVisibility(bool isVisible) override
```


### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| isVisible | **bool** | true - rend les espaces réservés de date-heure visibles, sinon - les masque. |

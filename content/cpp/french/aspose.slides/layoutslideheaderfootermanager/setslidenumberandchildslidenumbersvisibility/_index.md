## LayoutSlideHeaderFooterManager::SetSlideNumberAndChildSlideNumbersVisibility(bool) méthode

Modifie la visibilité de l’espace réservé du numéro de page du slide de mise en page et de tous les espaces réservés des numéros de page des slides enfants. Les espaces réservés enfants désignent les espaces réservés contenus dans les diapositives dépendantes. Les diapositives dépendantes utilisent et dépendent du slide de mise en page.

```cpp
void Aspose::Slides::LayoutSlideHeaderFooterManager::SetSlideNumberAndChildSlideNumbersVisibility(bool isVisible) override
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| isVisible | **bool** | true - rend les espaces réservés de numéros de page visibles, sinon - les masque. |

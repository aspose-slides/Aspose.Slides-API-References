## ISlideSize::get_Type() méthode


Obtient le type de taille de diapositive. L'affectation de toute valeur autre que [SlideSizeType::Custom](../../slidesizetype/) ajuste le [ISlideSize::get_Size](../get_size/) selon les dimensions prédéfinies, tout en conservant le [ISlideSize::get_Orientation](../get_orientation/) actuel.

```cpp
virtual SlideSizeType Aspose::Slides::ISlideSize::get_Type()=0
```

## Remarques


L'affectation de toute valeur autre que [SlideSizeType::Custom](../../slidesizetype/) ajuste le [System::Drawing::Size](../../../system.drawing/size/) selon les dimensions prédéfinies, tout en conservant le [Orientation](../../orientation/) actuel.

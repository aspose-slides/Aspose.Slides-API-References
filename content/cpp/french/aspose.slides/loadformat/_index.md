## LoadFormat enum

Indique le format du document à charger.

```cpp
enum class LoadFormat
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Auto | 0 | Indique à [Aspose.Slides](../) de reconnaître automatiquement le format. |
| Ppt | 1 | Microsoft PowerPoint 97-2003 [Presentation](../presentation/). |
| Pps | 2 | Microsoft PowerPoint 97-2003 [SlideShow](../../aspose.slides.slideshow/). |
| Pptx | 3 | Document PresentationML Open XML de Microsoft Office. |
| Ppsx | 4 | PresentationML Open XML de Microsoft Office [SlideShow](../../aspose.slides.slideshow/). |
| Odp | 5 | OpenOffice ODF [Presentation](../presentation/). |
| Potx | 6 | Modèle PresentationML Open XML de Microsoft Office. |
| Pptm | 7 | Document PresentationML Open XML de Microsoft Office avec macros. |
| Ppsm | 8 | PresentationML Open XML de Microsoft Office avec macros [SlideShow](../../aspose.slides.slideshow/). |
| Potm | 9 | Modèle PresentationML Open XML de Microsoft Office avec macros. |
| Otp | 10 | Modèle OTP OpenOffice [Presentation](../presentation/). |
| Ppt95 | 11 | Microsoft PowerPoint 95 [Presentation](../presentation/). |
| Pot | 12 | Modèle Microsoft PowerPoint 97-2003 [Presentation](../presentation/). |
| Fodp | 13 | Flat XML ODF [Presentation](../presentation/). |
| Unknown | 255 | Format de fichier inconnu |

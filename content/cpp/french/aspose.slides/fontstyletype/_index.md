## FontStyleType enum

Définit les styles de mise en forme qui peuvent être appliqués au texte.

```cpp
enum class FontStyleType
```

### Valeurs

| Name | Value | Description |
| --- | --- | --- |
| Regular | 0 | Texte standard. |
| Bold | 1 | Texte en gras. |
| Italic | 2 | Texte en italique. |
| Underline | 4 | Texte souligné. |
| Strikeout | 8 | Texte barré d'une ligne horizontale. |

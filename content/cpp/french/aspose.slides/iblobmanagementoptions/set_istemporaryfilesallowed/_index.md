## IBlobManagementOptions::set_IsTemporaryFilesAllowed(bool) méthode


Cette propriété définit si des fichiers temporaires peuvent être créés lors de la manipulation des BLOBs, ce qui réduit considérablement la consommation de mémoire mais nécessite des autorisations pour créer des fichiers.

```cpp
virtual void Aspose::Slides::IBlobManagementOptions::set_IsTemporaryFilesAllowed(bool value)=0
```

## Remarques


Tous les fichiers seront supprimés une fois le travail avec la présentation terminé.

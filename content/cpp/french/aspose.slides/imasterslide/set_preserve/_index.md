## IMasterSlide::set_Preserve(bool) méthode

Détermine si le maître correspondant est supprimé lorsque toutes les diapositives qui suivent ce maître sont supprimées. Remarque : [Aspose.Slides](../../) ne supprimera jamais aucun maître inutilisé par lui-même, pour supprimer réellement les maîtres inutilisés, appelez [IMasterSlideCollection::RemoveUnused](../../imasterslidecollection/removeunused/) Écrit **bool**.

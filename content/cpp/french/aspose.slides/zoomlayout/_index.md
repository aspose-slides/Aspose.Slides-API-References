## ZoomLayout enum

Spécifie la disposition de zoom du résumé.

```cpp
enum class ZoomLayout
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| GridLayout | 0 | Spécifie la disposition comme une disposition en grille pour une liste d'objets. |
| FixedLayout | 1 | Spécifie la disposition avec un positionnement Zoom défini par l'utilisateur. |

## IDocumentProperties::set_HyperlinksChanged(bool) méthode

Spécifie qu'un ou plusieurs hyperliens dans cette partie ont été mis à jour exclusivement dans cette partie par un producteur. Le producteur suivant qui ouvrira ce document doit mettre à jour les relations de liens hypertexte avec les nouveaux hyperliens spécifiés dans cette partie. Écrire **bool**.

## IDocumentProperties::get_HyperlinksChanged() méthode

Spécifie qu'un ou plusieurs hyperliens dans cette partie ont été mis à jour exclusivement dans cette partie par un producteur. Le prochain producteur qui ouvrira ce document devra mettre à jour les relations d'hyperliens avec les nouveaux hyperliens spécifiés dans cette partie. Lecture **bool**.

virtual bool Aspose::Slides::IDocumentProperties::get_HyperlinksChanged()=0

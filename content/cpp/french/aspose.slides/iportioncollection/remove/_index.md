## IPortionCollection::Remove(System::SharedPtr\<IPortion\>) méthode

Supprime la première occurrence d'un objet spécifique de [ICollection](../../../system.collections.generic/icollection/).

```cpp
virtual bool Aspose::Slides::IPortionCollection::Remove(System::SharedPtr<IPortion> item)=0
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| item | [System::SharedPtr](../../../system/sharedptr/)\<[IPortion](../../iportion/)\> | L'objet à supprimer de [ICollection](../../../system.collections.generic/icollection/). |

### Valeur de retour

true si *item* a été supprimé avec succès du [ICollection](../../../system.collections.generic/icollection/) ; sinon, false. Cette méthode renvoie également false si *item* n'est pas trouvé dans le [ICollection](../../../system.collections.generic/icollection/) original.

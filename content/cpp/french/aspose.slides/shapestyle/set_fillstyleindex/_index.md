## ShapeStyle::set_FillStyleIndex(int16_t) méthode

Définit l'index de la colonne de remplissage de la forme dans les matrices de style. 0 signifie aucun remplissage, une valeur positive - index dans les styles de remplissage du thème, une valeur négative - index dans les styles d'arrière-plan du thème. Écrire **int16_t**.

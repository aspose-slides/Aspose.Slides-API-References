## HyperlinkActionType enum


Représente un type d'action de lien hypertexte.

```cpp
enum class HyperlinkActionType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Unknown | -1 | Type d'action non reconnu. |
| NoAction | 0 | Aucune action. |
| Hyperlink | 1 | Hyperlien habituel. |
| JumpFirstSlide | 2 | Aller à la première diapositive de la présentation. |
| JumpPreviousSlide | 3 | Aller à la diapositive précédente. |
| JumpNextSlide | 4 | Aller à la diapositive suivante. |
| JumpLastSlide | 5 | Aller à la dernière diapositive de la présentation. |
| JumpEndShow | 6 | Aller à la fin du diaporama. |
| JumpLastViewedSlide | 7 | Aller à la dernière diapositive affichée. |
| JumpSpecificSlide | 8 | Aller à la diapositive spécifique, référée par la valeur [IHyperlink::get_TargetSlide](../ihyperlink/get_targetslide/). |
| StartCustomSlideShow | 9 | Démarrer le diaporama personnalisé. |
| OpenFile | 10 | Ouvrir le fichier référencé. |
| OpenPresentation | 11 | Ouvrir la présentation référencée. |
| StartStopMedia | 12 | Démarrer/arrêter la lecture du fichier média. |
| StartMacro | 13 | Démarrer l'exécution du script macro. |
| StartProgram | 14 | Démarrer le programme. |

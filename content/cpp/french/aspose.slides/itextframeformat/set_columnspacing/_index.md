## ITextFrameFormat::set_ColumnSpacing(double) méthode

Définit l'espacement entre les colonnes de texte dans la zone de texte (en points). Cela ne doit s'appliquer que lorsqu'il y a plus d'une colonne. Cette valeur doit être un nombre positif. Sinon, la valeur sera définie à zéro. Écrire **double**.

virtual void Aspose::Slides::ITextFrameFormat::set_ColumnSpacing(double value)=0

## MasterNotesSlideHeaderFooterManager::SetFooterAndChildFootersVisibility(bool) méthode

Modifie la visibilité de l’espace réservé de pied de page de la diapositive maître et de tous les espaces réservés de pied de page enfants. Les espaces réservés enfants désignent les espaces réservés contenus sur les diapositives de notes dépendantes. Les diapositives de notes dépendantes utilisent et dépendent de la diapositive maître de notes.

```cpp
void Aspose::Slides::MasterNotesSlideHeaderFooterManager::SetFooterAndChildFootersVisibility(bool isVisible) override
```

### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| isVisible | **bool** | true - rend les espaces réservés de pied de page visibles, sinon - les masque. |

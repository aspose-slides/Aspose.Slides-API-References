## MasterNotesSlideHeaderFooterManager::SetDateTimeAndChildDateTimesVisibility(bool) method


Modifie la visibilité du espace réservé de date-heure de la diapositive maîtresse et de tous les espaces réservés de date-heure enfants. Les espaces réservés enfants désignent les espaces réservés contenus dans les diapositives de notes dépendantes. Les diapositives de notes dépendantes utilisent et dépendent de la diapositive maîtresse de notes.

```cpp
void Aspose::Slides::MasterNotesSlideHeaderFooterManager::SetDateTimeAndChildDateTimesVisibility(bool isVisible) override
```


### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| isVisible | **bool** | true - rend les espaces réservés de date-heure visibles, sinon - les masque. |

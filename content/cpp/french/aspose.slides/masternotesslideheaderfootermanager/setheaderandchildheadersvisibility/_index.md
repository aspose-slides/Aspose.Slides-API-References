## MasterNotesSlideHeaderFooterManager::SetHeaderAndChildHeadersVisibility(bool) méthode


Modifie la visibilité de l'espace réservé d'en-tête de la diapositive de notes maîtresse et de tous les espaces réservés d'en-tête enfants. Les espaces réservés enfants signifient que les espaces réservés sont contenus sur les diapositives de notes dépendantes. Les diapositives de notes dépendantes utilisent et dépendent de la diapositive de notes maîtresse.

```cpp
void Aspose::Slides::MasterNotesSlideHeaderFooterManager::SetHeaderAndChildHeadersVisibility(bool isVisible) override
```


### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| isVisible | **bool** | true - rend les espaces réservés d'en-tête visibles, sinon - les masque. |

## ParagraphCollection::Remove(System::SharedPtr\<IParagraph\>) méthode

Supprime la première occurrence d'un objet spécifique du [ICollection](../../../system.collections.generic/icollection/).

```cpp
bool Aspose::Slides::ParagraphCollection::Remove(System::SharedPtr<IParagraph> item) override
```

### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| item | [System::SharedPtr](../../../system/sharedptr/)\<[IParagraph](../../iparagraph/)\> | L'objet à supprimer du [ICollection](../../../system.collections.generic/icollection/). |

### Valeur de retour

true si *item* a été supprimé avec succès du [ICollection](../../../system.collections.generic/icollection/) ; sinon, false. Cette méthode renvoie également false si *item* n'est pas trouvé dans le [ICollection](../../../system.collections.generic/icollection/) original.

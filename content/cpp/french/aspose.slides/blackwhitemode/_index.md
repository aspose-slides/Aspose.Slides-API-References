## BlackWhiteMode enum

Détermine comment la forme colorée doit être transformée en noir et blanc.

```cpp
enum class BlackWhiteMode : int8_t
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| NotDefined | -1 | La propriété n'est pas définie |
| Color | 0 | L'objet est rendu avec une coloration normale |
| Automatic | 1 | L'objet est rendu avec une coloration automatique |
| Gray | 2 | L'objet est rendu avec une coloration grise |
| LightGray | 3 | L'objet est rendu avec une coloration gris clair |
| InverseGray | 4 | L'objet est rendu avec une coloration grise inversée |
| GrayWhite | 5 | L'objet est rendu avec une coloration grise et blanche |
| BlackGray | 6 | L'objet est rendu avec une coloration noire et grise |
| BlackWhite | 7 | L'objet est rendu avec une coloration noire et blanche |
| Black | 8 | L'objet est rendu uniquement avec une coloration noire |
| White | 9 | L'objet est rendu avec une coloration blanche |
| Hidden | 10 | L'objet n'est pas rendu |

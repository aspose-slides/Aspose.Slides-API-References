## PVICachingMode enum

Représente différents niveaux de mise en cache pour les objets effectifs PVI.

```cpp
enum class PVICachingMode
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| NoCaching | 0 | Mode classique avec création de chaque instance. |
| NotDefinedOnly | 1 | Utiliser la mise en cache des valeurs effectives pour les formats NotDefined, mode classique pour le reste. |
| Full | 2 | Utiliser la mise en cache des valeurs effectives pour tous les formats. |

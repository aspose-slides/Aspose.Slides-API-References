## Remarques


La valeur renvoyée est toujours définie et n'est jamais [std::numeric_limits<float>::quiet_NaN()](../). La valeur assignée doit également être définie ; affectez [std::numeric_limits<float>::quiet_NaN()](../) uniquement aux propriétés d'une instance **RawFrame**.

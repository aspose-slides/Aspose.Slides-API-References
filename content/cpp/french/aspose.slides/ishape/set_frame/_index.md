## IShape::set_Frame(System::SharedPtr\<IShapeFrame\>) méthode

Définit les propriétés du cadre de la forme. Écrivez [IShapeFrame](../../ishapeframe/).

```cpp
virtual void Aspose::Slides::IShape::set_Frame(System::SharedPtr<IShapeFrame> value)=0
```

## Remarques

La valeur de chaque propriété de l'instance [IShapeFrame](../../ishapeframe/) retournée n'est pas indéfinie (n'est pas NaN ou NotDefined). La valeur de chaque propriété de l'instance [IShapeFrame](../../ishapeframe/) assignée doit être définie (ne doit pas être NaN ou NotDefined). Vous pouvez définir des valeurs indéfinies pour les propriétés de l'instance RawFrame.

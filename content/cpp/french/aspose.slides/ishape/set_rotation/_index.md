## IShape::set_Rotation(float) méthode


Définit le nombre de degrés selon lesquels la forme spécifiée est tournée autour de l'axe z. Une valeur positive indique une rotation dans le sens des aiguilles d'une montre ; une valeur négative indique une rotation dans le sens inverse. Écrire **float**.

```cpp
virtual void Aspose::Slides::IShape::set_Rotation(float value)=0
```

## Remarques


La valeur retournée est toujours définie (n'est pas std::numeric_limits<float>::quiet_NaN()). La valeur assignée doit être définie (pas std::numeric_limits<float>::quiet_NaN()). Vous pouvez définir des valeurs indéfinies pour les propriétés d'instance RawFrame.

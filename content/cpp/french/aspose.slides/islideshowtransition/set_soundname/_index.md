## ISlideShowTransition::set_SoundName(System::String) méthode


Spécifie un nom lisible par l'homme pour le son de la transition. Le [ISlideShowTransition::set_Sound](../set_sound/) doit être affecté pour obtenir ou définir le nom du son. Écrit [System::String](../../../system/string/).

```cpp
virtual void Aspose::Slides::ISlideShowTransition::set_SoundName(System::String value)=0
```

## Remarques


Ce nom apparaît dans l'interface utilisateur de PowerPoint lors de la configuration manuelle du son de la transition.

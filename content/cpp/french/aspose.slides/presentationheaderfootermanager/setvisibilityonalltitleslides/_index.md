## PresentationHeaderFooterManager::SetVisibilityOnAllTitleSlides(bool) méthode


Modifie la visibilité des espaces réservés du pied de page, de la date-heure et du numéro de page pour toutes les diapositives de titre et pour la première diapositive de mise en page. Les diapositives de titre – diapositives basées sur la première diapositive de mise en page (quelle que soit le type de cette première mise en page).

```cpp
void Aspose::Slides::PresentationHeaderFooterManager::SetVisibilityOnAllTitleSlides(bool isVisible) override
```


### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| isVisible | **bool** | true - rend les espaces réservés visibles, sinon - les masque. |

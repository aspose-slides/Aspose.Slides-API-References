## IMasterNotesSlideHeaderFooterManager::SetDateTimeAndChildDateTimesVisibility(bool) méthode

Modifie la visibilité du placeholder de date-heure de la diapositive maître des notes et de tous les placeholders de date-heure enfants. Les placeholders enfants sont contenus sur les diapositives de notes dépendantes. Les diapositives de notes dépendantes utilisent et dépendent de la diapositive maître des notes.

```cpp
virtual void Aspose::Slides::IMasterNotesSlideHeaderFooterManager::SetDateTimeAndChildDateTimesVisibility(bool isVisible)=0
```

### Arguments

| Parameter | Type | Description |
| --- | --- | --- |
| isVisible | **bool** | true - rend les placeholders de date-heure visibles, sinon - les masque. |

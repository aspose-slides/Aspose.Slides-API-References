## IMasterNotesSlideHeaderFooterManager::SetFooterAndChildFootersVisibility(bool) méthode

Modifie la visibilité de l'espace réservé du pied de page de la diapositive maître des notes ainsi que celle de tous les espaces réservés de pied de page enfants. Les espaces réservés enfants désignent les espaces réservés contenus sur les diapositives de notes dépendantes. Les diapositives de notes dépendantes utilisent et dépendent de la diapositive maître des notes.

```cpp
virtual void Aspose::Slides::IMasterNotesSlideHeaderFooterManager::SetFooterAndChildFootersVisibility(bool isVisible)=0
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| isVisible | **bool** | true - rend les espaces réservés du pied de page visibles, sinon - les masque. |

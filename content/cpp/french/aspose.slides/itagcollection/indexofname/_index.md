## ITagCollection::IndexOfName(System::String) méthode

Renvoie l'index basé sur zéro de la clé spécifiée dans la collection.

```cpp
virtual int32_t Aspose::Slides::ITagCollection::IndexOfName(System::String name)=0
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| name | [System::String](../../../system/string/) | Le nom à rechercher dans la collection. |

### Valeur de retour

L'index basé sur zéro de la clé, si la clé est trouvée dans la collection ; sinon, -1.

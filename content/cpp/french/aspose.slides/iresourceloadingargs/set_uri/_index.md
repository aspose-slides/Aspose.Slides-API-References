## IResourceLoadingArgs::set_Uri(System::String) méthode

URI de la ressource qui est utilisé pour le téléchargement si [IResourceLoadingCallback::ResourceLoading()](../../iresourceloadingcallback/resourceloading/) renvoie [ResourceLoadingAction::Default](../../resourceloadingaction/). Initialement, il est défini sur l'URI original de la ressource, mais peut être redéfini à n'importe quelle valeur.

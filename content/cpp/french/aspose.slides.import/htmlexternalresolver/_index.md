## Remarques

Obsolète
:   Obsolète. Utilisez [ExternalResourceResolver](../externalresourceresolver/) à la place. Sera supprimé dans la version 17.12.

L'utilisation de ce résolveur pourrait créer une vulnérabilité lorsqu'un fichier HTML fourni par le client fait récupérer un fichier local ou réseau par le serveur. Utilisez-le avec prudence. Il est recommandé de ne pas spécifier [HtmlExternalResolver](./) du tout (seuls les objets embarqués seront lus) ou de créer une sous-classe qui vérifie si l'uri spécifié est valide.

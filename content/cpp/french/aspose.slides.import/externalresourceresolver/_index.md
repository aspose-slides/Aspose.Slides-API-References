## Remarques


L'utilisation de ce résolveur pourrait créer une vulnérabilité lorsqu'un fichier HTML ou SVG fourni par le client amène le logiciel serveur à accéder à un fichier local ou réseau. Utilisez-le avec prudence. Il est recommandé de ne pas spécifier [ExternalResourceResolver](./) du tout (seuls les objets intégrés seront lus) ou de créer une sous-classe qui vérifie si l'URI spécifiée est valide.

## ISet::IsProperSupersetOf(IEnumerablePtr) méthode

Vérifie si l'ensemble actuel est un sur-ensemble strict d'un autre conteneur.

```cpp
virtual bool System::Collections::Generic::ISet<T>::IsProperSupersetOf(IEnumerablePtr other)=0
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| other | [IEnumerablePtr](../ienumerableptr/) | Sous-ensemble à vérifier. |

### Valeur de retour

Vrai si tous les éléments de **other** sont présents dans l'ensemble et que l'ensemble possède plus d'éléments que **other**, sinon faux.

## ISet::IsProperSubsetOf(IEnumerablePtr) méthode

Vérifie si l'ensemble actuel est un sous-ensemble strict d'un autre conteneur.

```cpp
virtual bool System::Collections::Generic::ISet<T>::IsProperSubsetOf(IEnumerablePtr other)=0
```

### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| other | [IEnumerablePtr](../ienumerableptr/) | Ensemble supérieur à vérifier. |

### Valeur de retour

Vrai si tous les éléments de l'ensemble actuel sont présents dans **other** et que **other** possède plus d'éléments que l'ensemble actuel, sinon faux.

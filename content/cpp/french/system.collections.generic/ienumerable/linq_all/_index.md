## IEnumerable::LINQ_All(std::function\<bool(T)>) méthode


Détermine si tous les éléments d'une séquence satisfont une condition.

```cpp
bool System::Collections::Generic::IEnumerable<T>::LINQ_All(std::function<bool(T)> predicate)
```


### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| predicate | std::function\<**bool**(T)> | Une fonction pour tester chaque élément selon une condition. |

### Valeur de retour

true si chaque élément de la séquence source passe le test du prédicat spécifié, ou si la séquence est vide ; sinon, false.

## IEnumerable::begin() méthode

Obtient un itérateur pointant vers le premier élément (le cas échéant) de la collection. Cet itérateur ne peut pas être utilisé pour modifier un objet référencé car [GetEnumerator()](../getenumerator/) renvoie un objet copie de T.

```cpp
iterator System::Collections::Generic::IEnumerable<T>::begin()
```

### Valeur de retour

Un itérateur pointant vers le premier élément (le cas échéant) de la collection

## IEnumerable::begin() const méthode

Obtient un itérateur pointant vers le premier élément (le cas échéant) de l'instance qualifiée const de la collection.

```cpp
const_iterator System::Collections::Generic::IEnumerable<T>::begin() const
```

### Valeur de retour

Un itérateur pointant vers le premier élément (le cas échéant) de l'instance qualifiée const de la collection

## IEnumerable::end() méthode

Obtient un itérateur pointant juste après le dernier élément (le cas échéant) de la collection. Cet itérateur ne peut pas être utilisé pour modifier l'objet référencé car [GetEnumerator()](../getenumerator/) renvoie une copie-objet de T.

```cpp
iterator System::Collections::Generic::IEnumerable<T>::end()
```

### Valeur de retour

Un itérateur pointant juste après le dernier élément (le cas échéant) de la collection

## IEnumerable::end() const méthode

Obtient un itérateur pointant juste après le dernier élément (le cas échéant) de l'instance de la collection qualifiée const.

```cpp
const_iterator System::Collections::Generic::IEnumerable<T>::end() const
```

### Valeur de retour

Un itérateur pointant juste après le dernier élément (le cas échéant) de l'instance de la collection qualifiée const

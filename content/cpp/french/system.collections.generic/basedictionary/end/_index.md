## BaseDictionary::end() const méthode

Renvoie un itérateur vers l'enveloppe KVPair pour l'élément clé-valeur suivant le dernier élément du conteneur. Implémenté dans le style C# - l'itérateur doit renvoyer l'objet KVPair avec les interfaces get_Key() et get_Value(). Cet élément agit comme un espace réservé; tenter d'y accéder entraîne un comportement indéfini.

```cpp
const_iterator System::Collections::Generic::BaseDictionary<Map>::end() const noexcept
```

### Valeur de retour

Un itérateur pointant vers l'élément théorique placé après l'élément de fin de la collection.

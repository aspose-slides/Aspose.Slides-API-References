```cpp
bool System::Threading::Details_ThreadAbortException::Is(const System::TypeInfo &target) const override
```


### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| target | const [System::TypeInfo](../../../system/typeinfo/)\& | [TypeInfo](../../../system/typeinfo/) structure describing the type to test current object against. |

### Valeur de retour

True si l'objet est d'un type balisé ou d'une de ses sous-classes, false sinon.
## Remarques

Vérifie si l'objet représente une instance du type décrit par targetType. Analogue de l'opérateur C# 'is'.

## Details_SemaphoreFullException::Is(const System::TypeInfo\&) const method

```cpp
bool System::Threading::Details_SemaphoreFullException::Is(const System::TypeInfo &target) const override
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| target | const [System::TypeInfo](../../../system/typeinfo/)\& | [TypeInfo](../../../system/typeinfo/) structure décrivant le type contre lequel tester l'objet actuel. |

## Valeur de retour

True si l'objet est d'un type balisé ou de l'une de ses sous-classes, false sinon.

## Remarques

Vérifie si l'objet représente une instance du type décrit par targetType. Analogue de l'opérateur C# 'is'.

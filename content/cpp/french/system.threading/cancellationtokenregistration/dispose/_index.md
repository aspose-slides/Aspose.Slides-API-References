## CancellationTokenRegistration::Dispose() méthode

Libère l'enregistrement et supprime le rappel de l'[CancellationTokenSource](../../cancellationtokensource/) associé. Après avoir appelé cette méthode, le rappel enregistré ne sera plus invoqué lorsque l'[CancellationTokenSource](../../cancellationtokensource/) associé sera annulé.

```cpp
void System::Threading::CancellationTokenRegistration::Dispose()
```

## Remarques

Il est sûr d’appeler cette méthode plusieurs fois - les appels suivants n’auront aucun effet.

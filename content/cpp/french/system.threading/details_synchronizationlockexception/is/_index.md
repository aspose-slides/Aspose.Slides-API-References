## Détails_SynchronizationLockException::Is(const System::TypeInfo\&) const méthode

```cpp
bool System::Threading::Details_SynchronizationLockException::Is(const System::TypeInfo &target) const override
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| target | const [System::TypeInfo](../../../system/typeinfo/)\& | structure [TypeInfo](../../../system/typeinfo/) décrivant le type contre lequel tester l'objet actuel. |

### Valeur de retour

True si l'objet est du type indiqué ou de sa sous-classe, false sinon.

## Remarques

Vérifie si l'objet représente une instance du type décrit par targetType. Analogue de l'opérateur C# 'is'.

## Thread::get_ManagedThreadId() const méthode

Obtient l'identifiant du thread. Peut être récupéré depuis le système d'exploitation, mais si l'identifiant du thread du système d'exploitation dépasse les limites de int, les identifiants des threads peuvent se chevaucher.

```cpp
int System::Threading::Thread::get_ManagedThreadId() const
```

### Valeur de retour

Identifiant de thread pseudo-unique.

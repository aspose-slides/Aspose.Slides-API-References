## AnimateTextType enum

Représente le type de texte animé d'un effet d'animation.


### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| AllAtOnce | 0 | Anime tout le texte d'un coup. |
| ByWord | 1 | Anime le texte mot par mot. |
| ByLetter | 2 | Anime le texte lettre par lettre. |

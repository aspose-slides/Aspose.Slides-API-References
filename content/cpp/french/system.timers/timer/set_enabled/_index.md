## Timer::set_Enabled(bool) method


Démarre ou arrête le minuteur. Le démarrage du minuteur ne redémarre pas le comptage du temps si le minuteur est déjà en cours d'exécution.

```cpp
void System::Timers::Timer::set_Enabled(bool value)
```


### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| value | **bool** | true pour démarrer le minuteur, false pour l'arrêter. |

## Details_SecurityException::Is(const System::TypeInfo\&) const method

```cpp
bool System::Security::Details_SecurityException::Is(const System::TypeInfo &target) const override
```

### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| target | const [System::TypeInfo](../../../system/typeinfo/)\& | [TypeInfo](../../../system/typeinfo/) structure décrivant le type contre lequel tester l'objet actuel. |

### Valeur de retour

True si l'objet est du type indiqué ou d'une sous-classe, false sinon.
## Remarques

Vérifie si l'objet représente une instance du type décrit par targetType. Analogue de l'opérateur 'is' de C#.

## SystemColors classe

Classe qui fournit un ensemble d'objets [Color](../color/) pré-crés qui représentent les couleurs des éléments d'affichage [Windows](../../system.windows/). Il s'agit d'un type statique sans services d'instance. Vous ne devez jamais créer d'instances de celui-ci par aucun moyen.

```cpp
class SystemColors
```

## StringDigitSubstitute enum

Spécifie comment les chiffres d’une chaîne sont substitués selon la langue ou la localisation.

```cpp
enum class StringDigitSubstitute
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| User | 0 | Substitution définie par l'utilisateur. |
| None | 1 | La substitution est désactivée. |
| National | 2 | Remplace les chiffres qui correspondent à la langue nationale officielle de la localisation de l'utilisateur. |
| Traditional | 3 | Remplace les chiffres qui correspondent à l'écriture ou à la langue native de l'utilisateur, qui peut ne pas être la même que la langue nationale officielle de la localisation de l'utilisateur. |

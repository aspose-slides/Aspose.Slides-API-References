## RotateFlipType énumération

Spécifie le type d'opération de rotation et/ou de retournement.

```cpp
enum class RotateFlipType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| RotateNoneFlipNone | 0 | Aucune rotation et aucun retournement. |
| Rotate90FlipNone | 1 | Rotation de 90 degrés dans le sens horaire et aucun retournement. |
| Rotate180FlipNone | 2 | Rotation de 180 degrés dans le sens horaire et aucun retournement. |
| Rotate270FlipNone | 3 | Rotation de 270 degrés dans le sens horaire et aucun retournement. |
| RotateNoneFlipX | 4 | Retournement horizontal et aucune rotation. |
| Rotate90FlipX | 5 | Rotation de 90 degrés dans le sens horaire et retournement horizontal. |
| Rotate180FlipX | 6 | Rotation de 180 degrés dans le sens horaire et retournement horizontal. |
| Rotate270FlipX | 7 | Rotation de 270 degrés dans le sens horaire et retournement horizontal. |
| RotateNoneFlipY | n/a | Retournement vertical et aucune rotation. |
| Rotate90FlipY | n/a | Rotation de 90 degrés dans le sens horaire et retournement vertical. |
| Rotate180FlipY | n/a | Rotation de 180 degrés dans le sens horaire et retournement vertical. |
| Rotate270FlipY | n/a | Rotation de 270 degrés dans le sens horaire et retournement vertical. |
| RotateNoneFlipXY | n/a | Retournement horizontal et vertical et aucune rotation. |
| Rotate90FlipXY | n/a | Rotation de 90 degrés dans le sens horaire et retournement horizontal et vertical. |
| Rotate180FlipXY | n/a | Rotation de 180 degrés dans le sens horaire et retournement horizontal et vertical. |
| Rotate270FlipXY | n/a | Rotation de 270 degrés dans le sens horaire et retournement horizontal et vertical. |

## GraphicsUnit enum

Représente une unité de mesure.

```cpp
enum class GraphicsUnit
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| World | 0 | Coordonnée du monde (unité non physique). |
| Display | 1 | Les unités de mesure du dispositif d'affichage. |
| Pixel | 2 | Un pixel de l'appareil. |
| Point | 3 | Un point d'imprimante (1/72 de pouce). |
| Inch | 4 | Un pouce. |
| Document | 5 | 1/300 de pouce. |
| Millimeter | 6 | Un millimètre. |

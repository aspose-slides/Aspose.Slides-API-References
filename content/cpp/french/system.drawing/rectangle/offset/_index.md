## Rectangle::Offset(const Point\&) méthode

Décale la position du rectangle représenté par l'objet actuel des quantités spécifiées.

```cpp
void System::Drawing::Rectangle::Offset(const Point &pos)
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| pos | const [Point](../../point/)\& | L'objet [Point](../../point/) qui spécifie la quantité dont le rectangle est décalé ; la valeur de la coordonnée X du point spécifie la quantité à ajouter à la coordonnée X du coin supérieur gauche du rectangle et la coordonnée Y du point spécifie la quantité à ajouter à la coordonnée Y du coin supérieur gauche du rectangle |

## Rectangle::Offset(int, int) méthode

Décale la position du rectangle représenté par l'objet actuel des quantités spécifiées.

```cpp
void System::Drawing::Rectangle::Offset(int x, int y)
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| x | int | La valeur à ajouter à la coordonnée X actuelle du rectangle |
| y | int | La valeur à ajouter à la coordonnée Y actuelle du rectangle |

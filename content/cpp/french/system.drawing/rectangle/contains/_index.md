## Rectangle::Contains(int, int) const méthode


Détermine si le point spécifié est situé à l’intérieur du rectangle représenté par l’objet actuel.

```cpp
bool System::Drawing::Rectangle::Contains(int x, int y) const
```


### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| x | int | La coordonnée X du point à vérifier |
| y | int | La coordonnée Y du point à vérifier |

### Valeur de retour

True si le point spécifié est situé à l’intérieur du rectangle représenté par l’objet actuel, sinon - false

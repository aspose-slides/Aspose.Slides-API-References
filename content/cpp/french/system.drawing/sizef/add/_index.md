## SizeF::Add(const SizeF\&, const SizeF\&) méthode


Renvoie un nouvel objet [SizeF](../) qui est la somme des objets [SizeF](../) spécifiés, c.-à-d. dont la valeur de largeur est égale à la somme des valeurs de largeur des objets spécifiés et dont la valeur de hauteur est égale à la somme des valeurs de hauteur des objets spécifiés.

```cpp
static SizeF System::Drawing::SizeF::Add(const SizeF &size1, const SizeF &size2)
```


### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| size1 | const [SizeF](../)\& | Le premier opérande |
| size2 | const [SizeF](../)\& | Le deuxième opérande |

### Valeur de retour

L'objet [SizeF](../) qui est la somme de **size1** et **size2**

## SizeF::Subtract(const SizeF\&, const SizeF\&) méthode

Renvoie un nouvel objet [SizeF](../) qui est le résultat de la soustraction de **size2** à **size1**, c’est-à-dire dont la valeur de largeur est le résultat de la soustraction de la largeur de **size2** à celle de **size1** et dont la valeur de hauteur est le résultat de la soustraction de la hauteur de **size2** à celle de **size1**.

```cpp
static SizeF System::Drawing::SizeF::Subtract(const SizeF &size1, const SizeF &size2)
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| size1 | const [SizeF](../)\& | L'objet [SizeF](../) dont il faut soustraire |
| size2 | const [SizeF](../)\& | L'objet [SizeF](../) à soustraire |

### Valeur de retour

L'objet [SizeF](../) qui est le résultat de la soustraction de **size2** à **size1**

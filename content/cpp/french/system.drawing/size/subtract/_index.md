## Size::Subtract(const Size\&, const Size\&) méthode

Renvoie un nouvel objet [Size](../) qui est le résultat de la soustraction de **size2** a **size1**, c'est a dire dont la valeur de largeur est le résultat de la soustraction de la largeur de **size2's** a celle de **size1's** et dont la valeur de hauteur est le résultat de la soustraction de la hauteur de **size2's** a celle de **size1's**.

```cpp
static Size System::Drawing::Size::Subtract(const Size &size1, const Size &size2)
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| size1 | const [Size](../)\& | L'objet [Size](../) dont on soustrait |
| size2 | const [Size](../)\& | L'objet [Size](../) a soustraire |

### Valeur de retour

L'objet [Size](../) qui est le résultat de la soustraction de **size2** a **size1**

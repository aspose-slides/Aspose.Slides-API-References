## Size::Add(const Size\&, const Size\&) méthode


Renvoie un nouvel objet [Size](../) qui est la somme de l'objet [Size](../) spécifié, c’est-à-dire dont la valeur de largeur est égale à la somme des valeurs de largeur des objets spécifiés et la valeur de hauteur est égale à la somme des valeurs de hauteur des objets spécifiés.

```cpp
static Size System::Drawing::Size::Add(const Size &size1, const Size &size2)
```


### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| size1 | const [Size](../)\& | Le premier opérande |
| size2 | const [Size](../)\& | Le deuxième opérande |

### Valeur de retour

L’objet [Size](../) qui est la somme de **size1** et **size2**

## Color::get_IsNamedColor() const méthode

Renvoie une valeur qui détermine si la structure [Color](../) représente une couleur nommée ou un membre de l'énumération KnownColor.

```cpp
bool System::Drawing::Color::get_IsNamedColor() const
```

### Valeur de retour

True si le [Color](../) a été créé en utilisant soit la méthode FromName(String), soit la méthode [FromKnownColor(KnownColor)](../fromknowncolor/) ; sinon, il renvoie false.

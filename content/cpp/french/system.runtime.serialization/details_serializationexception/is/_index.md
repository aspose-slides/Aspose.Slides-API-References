## Details_SerializationException::Is(const System::TypeInfo\&) const method




```cpp
bool System::Runtime::Serialization::Details_SerializationException::Is(const System::TypeInfo &target) const override
```


### Arguments

| Parameter | Type | Description |
| --- | --- | --- |
| target | const [System::TypeInfo](../../../system/typeinfo/)\& | [TypeInfo](../../../system/typeinfo/) structure décrivant le type contre lequel tester l'objet actuel. |

### Valeur de retour

True si l'objet est du type indiqué ou d'un de ses sous-classes, false sinon.

## Remarques

Vérifie si l'objet représente une instance du type décrit par targetType. Analogue de l'opérateur 'is' de C#.

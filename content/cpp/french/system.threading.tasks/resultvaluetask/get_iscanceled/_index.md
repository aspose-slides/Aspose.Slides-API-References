## ResultValueTask::get_IsCanceled() const méthode

Obtient une valeur indiquant si la tâche s’est terminée parce qu’elle a été annulée.

```cpp
bool System::Threading::Tasks::ResultValueTask<T>::get_IsCanceled() const
```

### Valeur de retour

bool Vrai si la tâche a été annulée ; sinon, faux.

Renvoie faux si la tâche contient une valeur de résultat directe.

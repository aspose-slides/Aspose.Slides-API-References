## MathArray::set_MaximumDistribution(bool) méthode

Distribution maximale : lorsque la valeur est vraie, le tableau est espacé à la largeur maximale de l'élément contenant (page, colonne, cellule, etc.).

```cpp
void Aspose::Slides::MathText::MathArray::set_MaximumDistribution(bool value) override
```

## Remarques

Exemple :
```cpp
auto mathArray = System::MakeObject<MathArray>(System::MakeObject<MathematicalText>(u"item1"));
mathArray->set_MaximumDistribution(true);
```

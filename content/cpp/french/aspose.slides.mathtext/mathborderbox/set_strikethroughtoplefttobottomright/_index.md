## MathBorderBox::set_StrikethroughTopLeftToBottomRight(bool) méthode

Barrer du coin supérieur gauche au coin inférieur droit (la valeur par défaut est false). Spécifie l'état masqué ou affiché d'une ligne diagonale de barré du coin supérieur gauche au coin inférieur droit de la boîte de bordure.

```cpp
void Aspose::Slides::MathText::MathBorderBox::set_StrikethroughTopLeftToBottomRight(bool value) override
```

## Remarques

Exemple : 
```cpp
auto borderBox = System::MakeObject<MathBorderBox>(System::MakeObject<MathematicalText>(u"x"));
borderBox->set_StrikethroughTopLeftToBottomRight(true);
```

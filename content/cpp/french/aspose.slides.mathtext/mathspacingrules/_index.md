## MathSpacingRules enum

Types d'écart (espacement horizontal) entre les colonnes d'une matrice

```cpp
enum class MathSpacingRules
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| SingleSpacingGap | 0 | Simple (1 em pour les colonnes ou 1 ligne pour les rangées) |
| OneAndHalfSpacingGap | 1 | Un et demi (1,5 em pour les colonnes ou 1,5 lignes pour les rangées) |
| DoubleSpacingGap | 2 | Double espacement (2 em pour les colonnes ou 2 lignes pour les rangées) |
| Exactly | 3 | Valeur exacte Pour les colonnes, se base sur la valeur de ColumnGap, mesurée en twips Pour les rangées, se base sur la valeur de RowGap, mesurée en twips |
| Multiple | 4 | Multiple Pour les colonnes, se base sur la valeur de ColumnGap, mesurée en incréments de 0,5 em Pour les rangées, se base sur la valeur de RowGap, mesurée en lignes |

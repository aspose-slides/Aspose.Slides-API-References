## IMathPhantom::get_ZeroWidth() méthode

Obtient une valeur indiquant si la largeur de l'element de base doit être considérée comme nulle.

```cpp
virtual bool Aspose::Slides::MathText::IMathPhantom::get_ZeroWidth()=0
```

## Remarques

Lorsque **true**, le phantom ne réserve pas d'espace horizontal pour sa base. Correspond à l'attribut OMML **m:zeroWid**.

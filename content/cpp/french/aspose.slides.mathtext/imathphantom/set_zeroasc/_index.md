## IMathPhantom::set_ZeroAsc(bool) méthode


Définit une valeur indiquant si la montée (hauteur au-dessus de la ligne de base) de l’élément de base doit être considérée comme zéro.

```cpp
virtual void Aspose::Slides::MathText::IMathPhantom::set_ZeroAsc(bool value)=0
```

## Remarques


Lorsque **true**, le fantôme ne soulève pas la ligne de base de la ligne mathématique environnante. Correspond à l’attribut OMML **m:zeroAsc**.

## IMathPhantom::set_ZeroDesc(bool) méthode

Définit une valeur indiquant si la descente (profondeur sous la ligne de base) de l'élément de base doit être considérée comme zéro.

```cpp
virtual void Aspose::Slides::MathText::IMathPhantom::set_ZeroDesc(bool value)=0
```

## Remarques

Lorsque **true**, le fantôme n'abaisse pas la ligne de base de la ligne mathématique environnante. Correspond à l'attribut OMML **m:zeroDesc**.

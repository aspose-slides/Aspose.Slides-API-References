## IMathPhantom::set_Transp(bool) méthode

Définit une valeur indiquant si le fantôme est transparent pour les règles d'espacement basées sur les classes.

```cpp
virtual void Aspose::Slides::MathText::IMathPhantom::set_Transp(bool value)=0
```

## Remarques

Lorsque **true**, les opérateurs et symboles à l'intérieur du fantôme affectent toujours l'espacement mathématique autour du fantôme (comme s'il était visible). Lorsque **false**, l'espacement basé sur les classes est ignoré. Correspond à l'attribut OMML **m:transp**.

## IMathPhantom::set_Show(bool) méthode

Définit une valeur indiquant si l'élément de base est affiché.

```cpp
virtual void Aspose::Slides::MathText::IMathPhantom::set_Show(bool value)=0
```

## Remarques

Lorsque **false**, l'élément de base est masqué mais peut encore occuper de l'espace selon les autres paramètres fantômes. Correspond à l'attribut OMML **m:show**.

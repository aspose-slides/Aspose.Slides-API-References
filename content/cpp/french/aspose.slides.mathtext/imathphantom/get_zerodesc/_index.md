## IMathPhantom::get_ZeroDesc() méthode

Renvoie une valeur indiquant si la descente (profondeur sous la ligne de base) de l'élément de base doit être traitée comme zéro.

```cpp
virtual bool Aspose::Slides::MathText::IMathPhantom::get_ZeroDesc()=0
```

## Remarques

Lorsque **true**, le phantom ne descend pas la ligne de base de la ligne de mathématiques environnante. Correspond à l'attribut OMML **m:zeroDesc**.

## IMathPhantom::get_Show() méthode


Obtient une valeur indiquant si l'élément de base est affiché.

```cpp
virtual bool Aspose::Slides::MathText::IMathPhantom::get_Show()=0
```

## Remarques


Lorsque **false**, l'élément de base est masqué mais peut encore occuper de l'espace en fonction des autres paramètres fantômes. Correspond à l'attribut OMML **m:show**.

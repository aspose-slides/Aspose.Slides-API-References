## IMathPhantom::set_ZeroWidth(bool) méthode


Définit une valeur indiquant si la largeur de l'élément de base doit être considérée comme nulle.

## Remarques


Lorsque **true**, le phantom ne réserve pas d'espace horizontal pour sa base. Correspond à l'attribut OMML **m:zeroWid**.

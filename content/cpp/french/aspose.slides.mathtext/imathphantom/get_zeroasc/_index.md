## IMathPhantom::get_ZeroAsc() méthode


Obtient une valeur indiquant si l'ascension (hauteur au-dessus de la ligne de base) de l'élément de base doit être traitée comme zéro.

```cpp
virtual bool Aspose::Slides::MathText::IMathPhantom::get_ZeroAsc()=0
```

## Remarques


Lorsque **true**, le fantôme ne soulève pas la ligne de base de la ligne mathématique environnante. Correspond à l'attribut OMML **m:zeroAsc**.

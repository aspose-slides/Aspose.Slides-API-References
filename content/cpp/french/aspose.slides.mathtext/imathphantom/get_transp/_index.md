## IMathPhantom::get_Transp() méthode

Obtient une valeur indiquant si le phantom est transparent pour les règles d'espacement basées sur les classes.

```cpp
virtual bool Aspose::Slides::MathText::IMathPhantom::get_Transp()=0
```

## Remarques

Lorsque **true**, les opérateurs et symboles à l'intérieur du phantom affectent toujours l'espacement mathématique autour du phantom (comme s'il était visible). Lorsque **false**, l'espacement basé sur les classes est ignoré. Correspond à l'attribut OMML **m:transp**.

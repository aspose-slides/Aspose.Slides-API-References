## Enum MathJustification

Spécifie la justification du paragraphe mathématique (une série d'instances de texte mathématique adjacentes au sein du même paragraphe)

```cpp
enum class MathJustification
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| LeftJustified | 1 | Aligne le texte mathématique à gauche |
| RightJustified | 2 | Aligne le texte mathématique à droite |
| Centered | 3 | Aligne le texte mathématique au centre |
| CenteredAsGroup | 4 | Centre sur la page l'ensemble du groupe de blocs mathématiques en une seule unité. Cela signifie que les instances de texte mathématique peuvent être alignées les unes par rapport aux autres, mais l'ensemble du groupe de texte mathématique est centré dans son ensemble. |

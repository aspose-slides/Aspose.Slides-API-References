## IMathBox classe


Spécifie l'encapsulation logique (emballage) d'un élément mathématique. Par exemple, un objet encadré peut servir d'émulateur d'opérateur avec ou sans point d'alignement, servir de point de rupture de ligne, ou être groupé de manière à ne pas autoriser les sauts de ligne à l'intérieur. Par exemple, l'opérateur \"==\" doit être encadré pour empêcher les sauts de ligne.

```cpp
class IMathBox : public virtual Aspose::Slides::MathText::IMathElement
```

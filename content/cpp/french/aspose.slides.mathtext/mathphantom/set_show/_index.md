## MathPhantom::set_Show(bool) méthode

Définit une valeur indiquant si l'élément de base est affiché.

```cpp
void Aspose::Slides::MathText::MathPhantom::set_Show(bool value) override
```

## Remarques

Lorsque **false**, l'élément de base est masqué mais peut encore occuper de l'espace en fonction des autres paramètres phantom. Correspond à l'attribut OMML **m:show**.

## MathPhantom::set_ZeroDesc(bool) méthode

Définit une valeur indiquant si la descente (profondeur sous la ligne de base) de l'élément de base doit être considérée comme zéro.

```cpp
void Aspose::Slides::MathText::MathPhantom::set_ZeroDesc(bool value) override
```

## Remarques

Lorsque **true**, le phantom ne baisse pas la ligne de base de la ligne mathématique environnante. Correspond à l'attribut OMML **m:zeroDesc**.

## MathPhantom::set_ZeroWidth(bool) méthode


Définit une valeur indiquant si la largeur de l'élément de base doit être considérée comme nulle.

```cpp
void Aspose::Slides::MathText::MathPhantom::set_ZeroWidth(bool value) override
```

## Remarques


Lorsque **true**, le fantôme ne réserve pas d'espace horizontal pour sa base. Correspond à l'attribut OMML **m:zeroWid**.

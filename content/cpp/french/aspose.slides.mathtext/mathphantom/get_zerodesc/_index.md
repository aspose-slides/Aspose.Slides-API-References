## MathPhantom::get_ZeroDesc() méthode


Obtient une valeur indiquant si la descente (profondeur sous la ligne de base) de l'élément de base doit être considérée comme zéro.

```cpp
bool Aspose::Slides::MathText::MathPhantom::get_ZeroDesc() override
```

## Remarques


Lorsque **true**, le phantom ne baisse pas la ligne de base de la ligne mathématique environnante. Correspond à l'attribut OMML **m:zeroDesc**.

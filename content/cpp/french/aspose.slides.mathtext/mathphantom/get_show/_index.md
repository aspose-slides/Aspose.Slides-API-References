## MathPhantom::get_Show() méthode


Renvoie une valeur indiquant si l'élément de base est affiché.

```cpp
bool Aspose::Slides::MathText::MathPhantom::get_Show() override
```

## Remarques


Lorsque **false**, l'élément de base est masqué mais peut toujours occuper de l'espace en fonction d'autres paramètres phantom. Correspond à l'attribut OMML **m:show**.

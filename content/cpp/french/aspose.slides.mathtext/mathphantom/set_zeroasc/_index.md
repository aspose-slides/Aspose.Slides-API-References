## MathPhantom::set_ZeroAsc(bool) méthode

Définit une valeur indiquant si la distance d'ascension (hauteur au-dessus de la ligne de base) de l'élément de base doit être traitée comme zéro.

```cpp
void Aspose::Slides::MathText::MathPhantom::set_ZeroAsc(bool value) override
```

## Remarques

Lorsque **true**, le phantom ne soulève pas la ligne de base de la ligne mathématique environnante. Correspond à l'attribut OMML **m:zeroAsc**.

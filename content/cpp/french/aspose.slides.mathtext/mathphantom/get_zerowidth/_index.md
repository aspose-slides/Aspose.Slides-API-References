## MathPhantom::get_ZeroWidth() méthode


Obtient une valeur indiquant si la largeur de l'élément de base doit être considérée comme zéro.

```cpp
bool Aspose::Slides::MathText::MathPhantom::get_ZeroWidth() override
```

## Remarques


Lorsque **true**, le phantom ne réserve pas d'espace horizontal pour sa base. Correspond à l'attribut OMML **m:zeroWid**.

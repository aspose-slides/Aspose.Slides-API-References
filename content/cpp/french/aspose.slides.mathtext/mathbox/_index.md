## MathBox classe

Spécifie l'encapsulation logique (emballage) d'un élément mathématique. Par exemple, un objet encadré peut servir d'émulateur d'opérateur avec ou sans point d'alignement, servir de point de saut de ligne, ou être groupé de manière à ne pas autoriser de sauts de ligne à l'intérieur. Par exemple, l'opérateur "==" doit être encadré pour éviter les sauts de ligne.

```cpp
class MathBox : public Aspose::Slides::MathText::MathElementBase,
                public Aspose::Slides::MathText::IMathBox,
                public Aspose::Slides::MathText::IHasControlCharacterProperties
```


### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Socket | 65535 | Les options de socket spécifiées s'appliquent à toutes les sockets. |
| IP | n/a | Les options de socket spécifiées s'appliquent aux sockets IP. |
| IPv6 | n/a | Les options de socket spécifiées s'appliquent aux sockets IPv6. |
| Tcp | n/a | Les options de socket spécifiées s'appliquent aux sockets TCP. |
| Udp | n/a | Les options de socket spécifiées s'appliquent aux sockets UDP. |

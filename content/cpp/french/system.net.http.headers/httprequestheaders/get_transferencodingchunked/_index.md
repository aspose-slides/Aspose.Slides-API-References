## HttpRequestHeaders::get_TransferEncodingChunked() méthode


Obtient une valeur qui indique si la valeur de l’en-tête 'Transfer-Encoding' contient 'Chunked'.

```cpp
Nullable<bool> System::Net::Http::Headers::HttpRequestHeaders::get_TransferEncodingChunked()
```


### Valeur de retour

Vrai lorsque la valeur de l’en-tête 'Transfer-Encoding' contient 'Chunked', sinon faux.

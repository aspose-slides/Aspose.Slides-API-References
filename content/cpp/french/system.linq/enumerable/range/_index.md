## Enumerable::Range(int32_t, int32_t) méthode

Génère une séquence de nombres entiers dans une plage spécifiée.

```cpp
static SharedPtr<Collections::Generic::IEnumerable<int32_t>> System::Linq::Enumerable::Range(int32_t start, int32_t count)
```

### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| start | **int32_t** | La valeur du premier entier de la séquence. |
| count | **int32_t** | Le nombre d'entiers séquentiels à générer. |

### Valeur de retour

Un IEnumerable de **int32_t** contenant une plage de nombres entiers séquentiels.

## TraceLevel enum

Spécifie quels messages afficher pour les [System.Diagnostics.Debug](../debug/), [System.Diagnostics.Trace](../trace/) et les classes System.Diagnostics.TraceSwitch.

```cpp
enum class TraceLevel
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Off | 0 | Ne produit aucun message de traçage ni de débogage. |
| Error | 1 | Produit les messages de gestion des erreurs. |
| Warning | 2 | Produit les avertissements et les messages de gestion des erreurs. |
| Info | 3 | Produit les messages d'information, les avertissements et les messages de gestion des erreurs. |
| Verbose | 4 | Produit tous les messages de débogage et de traçage. |

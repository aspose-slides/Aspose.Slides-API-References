## TraceEventType enum

Identifie le type d'événement qui a déclenché la trace.

```cpp
enum class TraceEventType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Critical | 1 | Erreur fatale ou plantage de l'application. |
| Error | 2 | Erreur récupérable. |
| Warning | 4 | Problème non critique. |
| Information | 8 | Message informatif. |
| Verbose | 16 | Trace de débogage. |
| Start | 256 | Démarrage d'une opération logique. |
| Stop | 512 | Arrêt d'une opération logique. |
| Suspend | 1024 | Suspension d'une opération logique. |
| Resume | 2048 | Reprise d'une opération logique. |
| Transfer | 4096 | Modification de l'identité de corrélation. |

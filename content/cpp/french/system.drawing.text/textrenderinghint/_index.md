## TextRenderingHint enum

Spécifie la qualité du rendu du texte.

```cpp
enum class TextRenderingHint
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| SystemDefault | 0 | Glyph avec indice de rendu système par défaut. |
| SingleBitPerPixelGridFit | 1 | Bitmap de glyph avec hinting. |
| SingleBitPerPixel | 2 | Bitmap de glyph sans hinting. |
| AntiAliasGridFit | 3 | Anticrénelage avec hinting. |
| AntiAlias | 4 | Bitmap anti-crénelé de glyph sans hinting. |
| ClearTypeGridFit | 5 | Bitmap CT de glyph avec hinting. |

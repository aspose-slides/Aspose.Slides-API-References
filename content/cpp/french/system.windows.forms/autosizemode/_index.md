## AutoSizeMode enum

Définit le fonctionnement de l'ajustement automatique.

```cpp
enum class AutoSizeMode
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| GrowAndShrink | 0 | [Control](../control/) s'agrandit et se rétrécit si nécessaire. Il ne peut pas être redimensionné manuellement. |
| GrowOnly | 1 | [Control](../control/) s'agrandit si nécessaire mais ne peut pas se rétrécir en dessous de la taille minimale. Il peut être redimensionné manuellement. |

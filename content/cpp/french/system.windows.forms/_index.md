## Classes

| Classe | Description |
| --- | --- |
| [Control](./control/) | Classe factice pour rendre le code traduit qui utilise la classe [Control](./control/) compilable. Les méthodes ne sont pas implémentées. Les objets de cette classe doivent être alloués uniquement à l’aide de la fonction [System::MakeObject()](../system/makeobject/). Ne jamais créer d’instance de ce type sur la pile ou en utilisant l’opérateur new, car cela entraînera des erreurs d’exécution et/ou des violations d’assertion. Toujours encapsuler cette classe dans un pointeur [System::SmartPtr](../system/smartptr/) et utiliser ce pointeur pour le transmettre aux fonctions en tant qu’argument. |

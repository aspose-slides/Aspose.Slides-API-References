## FlatStyle enum

Définit si le contrôle est dessiné plat ou tridimensionnel.

```cpp
enum class FlatStyle
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Flat | 0 | [Control](../control/) est dessiné plat. |
| Popup | 1 | [Control](../control/) est dessiné plat sauf si la souris le survole lorsqu'il est dessiné en 3D. |
| Standard | 2 | [Control](../control/) est dessiné en 3D. |
| System | 3 | Le style est contrôlé par le système d'exploitation. |

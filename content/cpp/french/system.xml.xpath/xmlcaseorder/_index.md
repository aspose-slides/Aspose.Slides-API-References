## XmlCaseOrder enum

Spécifie l'ordre de tri pour les lettres majuscules et minuscules.

```cpp
enum class XmlCaseOrder
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| None | 0 | Ignorer la casse. |
| UpperFirst | 1 | Les lettres majuscules sont triées avant les lettres minuscules. |
| LowerFirst | 2 | Les lettres minuscules sont triées avant les lettres majuscules. |

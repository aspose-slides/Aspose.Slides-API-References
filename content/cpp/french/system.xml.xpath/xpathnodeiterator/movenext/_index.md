## XPathNodeIterator::MoveNext() méthode


Lorsqu'elle est remplacée dans une classe dérivée, déplace l'objet [XPathNavigator](../../xpathnavigator/) renvoyé par la méthode [XPathNodeIterator::get_Current](../get_current/) vers le nœud suivant dans l'ensemble de nœuds sélectionnés.

```cpp
virtual bool System::Xml::XPath::XPathNodeIterator::MoveNext()=0
```


### Valeur de retour

**true** si l'objet [XPathNavigator](../../xpathnavigator/) a été déplacé vers le nœud suivant ; **false** s'il n'y a plus de nœuds sélectionnés.

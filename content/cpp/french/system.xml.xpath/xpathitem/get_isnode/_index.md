## XPathItem::get_IsNode() méthode

Lorsqu'elle est redéfinie dans une classe dérivée, renvoie une valeur indiquant si l'élément représente un nœud [XPath](../../) ou une valeur atomique.

```cpp
virtual bool System::Xml::XPath::XPathItem::get_IsNode()=0
```

### Valeur de retour

**true** si l'élément représente un nœud [XPath](../../) ; **false** si l'élément représente une valeur atomique.

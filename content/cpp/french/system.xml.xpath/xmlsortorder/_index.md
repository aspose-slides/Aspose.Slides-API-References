## XmlSortOrder enum

Spécifie l'ordre de tri.

```cpp
enum class XmlSortOrder
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Ascending | 1 | Les nœuds sont triés par ordre croissant. Par exemple, si les nombres 1,2,3 et 4 sont triés par ordre croissant, ils apparaissent sous la forme 1,2,3,4. |
| Descending | 2 | Les nœuds sont triés par ordre décroissant. Par exemple, si les nombres 1,2,3 et 4 sont triés par ordre décroissant, ils apparaissent sous la forme 4,3,2,1. |

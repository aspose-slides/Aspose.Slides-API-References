## XPathNavigator::get_LocalName() méthode

Lorsqu'elle est remplacée dans une classe dérivée, elle obtient le [XPathNavigator::get_Name](../get_name/) du nœud actuel sans aucun préfixe d'espace de noms.

```cpp
virtual String System::Xml::XPath::XPathNavigator::get_LocalName()=0
```

### Valeur de retour

Un [String](../../../system/string/) qui contient le nom local du nœud actuel, ou [String::Empty](../../../system/string/empty/) si le nœud actuel n'a pas de nom (par exemple, des nœuds texte ou commentaire).

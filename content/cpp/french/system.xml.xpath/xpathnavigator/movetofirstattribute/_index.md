## XPathNavigator::MoveToFirstAttribute() méthode

Lorsqu'elle est substituée dans une classe dérivée, déplace le [XPathNavigator](../) vers le premier attribut du nœud actuel.

```cpp
virtual bool System::Xml::XPath::XPathNavigator::MoveToFirstAttribute()=0
```

### Valeur de retour

**true** si le [XPathNavigator](../) réussit à se déplacer vers le premier attribut du nœud actuel ; sinon, **false**. Si **false**, la position du [XPathNavigator](../) reste inchangée.

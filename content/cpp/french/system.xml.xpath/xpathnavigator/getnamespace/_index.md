## XPathNavigator::GetNamespace(String) méthode

Renvoie la valeur du nœud d'espace de noms correspondant au nom local spécifié.

```cpp
virtual String System::Xml::XPath::XPathNavigator::GetNamespace(String name)
```

### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| name | [String](../../../system/string/) | Le nom local du nœud d'espace de noms. |

### Valeur de retour

Un [String](../../../system/string/) qui contient la valeur du nœud d'espace de noms ; [String::Empty](../../../system/string/empty/) si aucun nœud d'espace de noms correspondant n'est trouvé, ou si le [XPathNavigator](../) n'est pas positionné sur un nœud d'élément.

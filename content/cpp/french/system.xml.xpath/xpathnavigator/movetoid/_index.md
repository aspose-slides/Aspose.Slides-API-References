## XPathNavigator::MoveToId(String) méthode

Lorsqu'elle est remplacée dans une classe dérivée, se déplace vers le nœud qui possède un attribut de type **ID** dont la valeur correspond au [String](../../../system/string/) spécifié.

```cpp
virtual bool System::Xml::XPath::XPathNavigator::MoveToId(String id)=0
```

### Arguments

| Paramètre | Type | Description |
| --- | --- | --- |
| id | [String](../../../system/string/) | Un [String](../../../system/string/) représentant la valeur **ID** du nœud vers lequel vous souhaitez vous déplacer. |

### Valeur de retour

**true** si le [XPathNavigator](../) se déplace avec succès ; sinon, **false**. Si **false**, la position du navigateur reste inchangée.

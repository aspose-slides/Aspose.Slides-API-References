## XPathNavigator::get_HasAttributes() méthode


Renvoie une valeur indiquant si le nœud actuel possède des attributs.

```cpp
virtual bool System::Xml::XPath::XPathNavigator::get_HasAttributes()
```


### Valeur de retour

**true** si le nœud actuel possède des attributs ; renvoie **false** si le nœud actuel n’a aucun attribut, ou si le [XPathNavigator](../) n’est pas positionné sur un nœud d’élément.

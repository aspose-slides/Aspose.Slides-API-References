## XPathNavigator::MoveToFirstChild() méthode

When overridden in a derived class, moves the [XPathNavigator](../) to the first child node of the current node.

```cpp
virtual bool System::Xml::XPath::XPathNavigator::MoveToFirstChild()=0
```

### Valeur de retour

**true** si le [XPathNavigator](../) réussit à se déplacer vers le premier nœud enfant du nœud actuel ; sinon, **false**. Si **false**, la position du [XPathNavigator](../) reste inchangée.

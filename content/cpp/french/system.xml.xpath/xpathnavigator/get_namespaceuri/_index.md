## XPathNavigator::get_NamespaceURI() méthode

Lorsqu'elle est substituée dans une classe dérivée, récupère l'URI d'espace de noms du nœud actuel.

```cpp
virtual String System::Xml::XPath::XPathNavigator::get_NamespaceURI()=0
```

### Valeur de retour

Un [String](../../../system/string/) qui contient l'URI d'espace de noms du nœud actuel, ou [String::Empty](../../../system/string/empty/) si le nœud actuel n'a pas d'URI d'espace de noms.

## XPathNavigator::MoveToNamespace(String) méthode


Déplace le [XPathNavigator](../) vers le nœud d'espace de noms avec le préfixe d'espace de noms spécifié.

```cpp
virtual bool System::Xml::XPath::XPathNavigator::MoveToNamespace(String name)
```


### Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| name | [String](../../../system/string/) | Le préfixe d'espace de noms du nœud d'espace de noms. |

### Valeur de retour

**true** si le [XPathNavigator](../) réussit à se déplacer vers l'espace de noms spécifié ; **false** si aucun nœud d'espace de noms correspondant n'a été trouvé, ou si le [XPathNavigator](../) n'est pas positionné sur un nœud d'élément. Si **false**, la position du [XPathNavigator](../) reste inchangée.

## XPathNavigator::MoveToParent() méthode

Lorsqu'il est remplacé dans une classe dérivée, déplace le [XPathNavigator](../) vers le nœud parent du nœud actuel.

```cpp
virtual bool System::Xml::XPath::XPathNavigator::MoveToParent()=0
```

### Valeur de retour

**true** si le [XPathNavigator](../) réussit à se déplacer vers le nœud parent du nœud actuel ; sinon, **false**. Si **false**, la position du [XPathNavigator](../) reste inchangée.

## XPathNavigator::MoveToNextAttribute() méthode


Lorsqu'elle est remplacée dans une classe dérivée, déplace le [XPathNavigator](../) vers l'attribut suivant.

```cpp
virtual bool System::Xml::XPath::XPathNavigator::MoveToNextAttribute()=0
```


### Valeur de retour

**true** si le [XPathNavigator](../) réussit à se déplacer vers l'attribut suivant ; **false** s'il n'y a plus d'attributs. Si **false**, la position du [XPathNavigator](../) reste inchangée.

## XPathNavigator::MoveToPrevious() méthode

Lorsque remplacée dans une classe dérivée, déplace le [XPathNavigator](../) vers le nœud frère précédent du nœud actuel.

```cpp
virtual bool System::Xml::XPath::XPathNavigator::MoveToPrevious()=0
```

### Valeur de retour

**true** si le [XPathNavigator](../) réussit à se déplacer vers le nœud frère précédent ; sinon, **false** s'il n'y a pas de nœud frère précédent ou si le [XPathNavigator](../) est actuellement positionné sur un nœud d'attribut. Si **false**, la position du [XPathNavigator](../) reste inchangée.

## MathPhantom::set_ZeroDesc(bool) method


Sets a value indicating whether the descent (depth below baseline) of the base element should be treated as zero.

```cpp
void Aspose::Slides::MathText::MathPhantom::set_ZeroDesc(bool value) override
```

## Remarks


When **true**, the phantom does not lower the baseline of the surrounding math line. Corresponds to the OMML attribute **m:zeroDesc**.

## IDocumentProperties::get_ScaleCrop() method


Indicates the display mode of the document thumbnail. Set this element to **true** to enable scaling of the document thumbnail to the display. Set this element to **false** to enable cropping of the document thumbnail to show only sections that fits the display. Read **bool**.

virtual bool Aspose::Slides::IDocumentProperties::get_ScaleCrop()=0

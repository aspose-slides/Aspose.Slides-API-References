## XmlSchemaMaxExclusiveFacet Klasse

Stellt das **maxExclusive**-Element aus XML [Schema](../) dar, wie vom World Wide [Web](../../system.web/) Consortium (W3C) spezifiziert. Diese Klasse kann verwendet werden, um eine Einschränkung für den Maximalwert eines **simpleType**-Elements festzulegen. Der Elementwert muss kleiner sein als der Wert des **maxExclusive**-Elements.

```cpp
class XmlSchemaMaxExclusiveFacet : public System::Xml::Schema::XmlSchemaFacet
```

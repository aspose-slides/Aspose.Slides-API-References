## XmlSchemaElement::get_IsNillable() Methode


Gibt Informationen zurück, die anzeigen, ob **xsi:nil** in den Instanzdaten auftreten kann. Gibt an, ob einem Element ein expliziter Nilwert zugewiesen werden kann.

```cpp
bool System::Xml::Schema::XmlSchemaElement::get_IsNillable()
```


### Rückgabewert

Wenn nillable **true** ist, ermöglicht dies, dass eine Instanz des Elements das **nil**-Attribut auf **true** setzt. Das **nil**-Attribut ist als Teil des XML [Schema](../../) Namensraums für Instanzen definiert. Der Standardwert ist **false**. Optional.

## XmlSchemaElement::get_IsAbstract() Methode

Gibt Informationen zurück, die anzeigen, ob das Element in einem Instanzdokument verwendet werden kann.

```cpp
bool System::Xml::Schema::XmlSchemaElement::get_IsAbstract()
```

### Rückgabewert

Wenn **true**, kann das Element im Instanzdokument nicht erscheinen. Der Standardwert ist **false**. Optional.

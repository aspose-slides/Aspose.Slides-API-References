## XmlSchemaForm Enum

Gibt an, ob Attribute oder Elemente mit einem Namensraum-Präfix qualifiziert werden müssen.

```cpp
enum class XmlSchemaForm
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | 0 | Element- und Attributform ist im Schema nicht angegeben. |
| Qualified | 1 | Elemente und Attribute müssen mit einem Namensraum-Präfix qualifiziert werden. |
| Unqualified | 2 | Elemente und Attribute müssen nicht mit einem Namensraum-Präfix qualifiziert werden. |

## XmlSchemaComplexContent::get_IsMixed() Methode

Gibt Informationen zurück, die bestimmen, ob der Typ ein gemischtes Inhaltsmodell hat.

```cpp
bool System::Xml::Schema::XmlSchemaComplexContent::get_IsMixed()
```

### Rückgabewert

Wenn dieser Wert **true** ist, dürfen Zeichendaten zwischen den Kind-Elementen des komplexen Typs erscheinen (gemischtes Inhaltsmodell). Der Standardwert ist **false**. Optional.

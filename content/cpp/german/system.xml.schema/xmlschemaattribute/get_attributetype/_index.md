## XmlSchemaAttribute::get_AttributeType() Methode

Gibt das Objekt zurück, das auf dem [XmlSchemaAttribute::get_SchemaType](../get_schematype/)- oder [XmlSchemaAttribute::get_SchemaTypeName](../get_schematypename/)-Wert des Attributs basiert, das die Nachkompilierungsinterpretation des **AttributeType**-Werts enthält.

```cpp
SharedPtr<Object> System::Xml::Schema::XmlSchemaAttribute::get_AttributeType()
```

### Rückgabewert

Veraltet
:   Es wird empfohlen, stattdessen die Methode get_AttributeSchemaType zu verwenden.

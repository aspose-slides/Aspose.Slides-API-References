## XmlSchemaRedefine Klasse


Stellt das **redefine**-Element aus XML [Schema](../) dar, wie vom World Wide [Web](../../system.web/) Consortium (W3C) spezifiziert. Diese Klasse kann verwendet werden, um einfache und komplexe Typen, Gruppen und Attributgruppen aus externen Schemadateien im aktuellen Schema neu zu definieren. Diese Klasse kann auch verwendet werden, um Versionierung für die Schemaelemente bereitzustellen.

```cpp
class XmlSchemaRedefine : public System::Xml::Schema::XmlSchemaExternal
```

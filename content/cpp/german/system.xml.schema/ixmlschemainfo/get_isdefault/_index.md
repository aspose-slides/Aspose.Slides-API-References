## IXmlSchemaInfo::get_IsDefault() Methode

Gibt einen Wert zurück, der angibt, ob dieser validierte XML-Knoten als Ergebnis einer während der XML [Schema](../../) Definitionssprache (XSD) Schema-Validierung angewendeten Vorgabe festgelegt wurde.

```cpp
virtual bool System::Xml::Schema::IXmlSchemaInfo::get_IsDefault()=0
```

### Rückgabewert

**true** wenn dieser validierte XML-Knoten als Ergebnis einer während der Schema-Validierung angewendeten Vorgabe festgelegt wurde; andernfalls **false**.

## XmlSchemaMaxLengthFacet Klasse

Stellt das **maxLength**-Element aus XML [Schema](../) dar, wie vom World Wide [Web](../../system.web/) Consortium (W3C) spezifiziert. Diese Klasse kann verwendet werden, um eine Einschränkung der maximalen Länge des Datenwerts eines **simpleType**-Elements anzugeben. Die Länge muss kleiner sein als der Wert des **maxLength**-Elements.

```cpp
class XmlSchemaMaxLengthFacet : public System::Xml::Schema::XmlSchemaNumericFacet
```

## Anmerkungen

Objekte dieser Klasse sollten nur über die Funktion [System::MakeObject()](../../system/makeobject/) alloziert werden. Nie Instanzen dieses Typs auf dem Stack oder mit operator new erzeugen, da dies zu Laufzeitfehlern und/oder Assertionsfehlern führt. Immer diese Klasse in einen [System::SmartPtr](../../system/smartptr/)-Pointer einwickeln und diesen Pointer verwenden, um ihn als Argument an Funktionen zu übergeben.

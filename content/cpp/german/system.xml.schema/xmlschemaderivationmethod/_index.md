## XmlSchemaDerivationMethod enum

Stellt verschiedene Methoden zur Verhinderung der Ableitung bereit.

```cpp
enum class XmlSchemaDerivationMethod
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Empty | 0 | Standardableitungsmethode überschreiben, um beliebige Ableitungen zu erlauben. |
| Substitution | 1 | Bezieht sich auf Ableitungen durch **Substitution**. |
| Extension | 2 | Bezieht sich auf Ableitungen durch **Extension**. |
| Restriction | 4 | Bezieht sich auf Ableitungen durch **Restriction**. |
| List | 8 | Bezieht sich auf Ableitungen durch **List**. |
| Union | 16 | Bezieht sich auf Ableitungen durch **Union**. |
| All | 255 | **#all**. Bezieht sich auf alle Ableitungsmethoden. |
| None | 256 | Akzeptiert die Standardableitungsmethode. |

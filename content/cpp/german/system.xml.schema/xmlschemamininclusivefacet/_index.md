## XmlSchemaMinInclusiveFacet Klasse


Vertreten das **minInclusive**-Element aus XML [Schema](../) wie vom World Wide [Web](../../system.web/) Consortium (W3C) spezifiziert. Diese Klasse kann verwendet werden, um eine Einschränkung des Minimalwerts eines simpleType-Elements anzugeben. Der Elementwert muss größer oder gleich dem Wert des **minInclusive**-Elements sein.

```cpp
class XmlSchemaMinInclusiveFacet : public System::Xml::Schema::XmlSchemaFacet
```

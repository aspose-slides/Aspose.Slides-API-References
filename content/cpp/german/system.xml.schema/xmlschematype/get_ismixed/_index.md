## XmlSchemaType::get_IsMixed() method

Gibt einen Wert zurück, der angibt, ob dieser Typ ein gemischtes Inhaltsmodell hat. Dieser Aufruf ist nur bei einem komplexen Typ gültig.

```cpp
virtual bool System::Xml::Schema::XmlSchemaType::get_IsMixed()
```

### Rückgabewert

**true** wenn der Typ ein gemischtes Inhaltsmodell hat; andernfalls **false**. Der Standardwert ist **false**.

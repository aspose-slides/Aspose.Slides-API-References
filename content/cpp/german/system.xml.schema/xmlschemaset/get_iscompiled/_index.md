## XmlSchemaSet::get_IsCompiled() Methode


Gibt einen Wert zurück, der anzeigt, ob die XML [Schema](../../)-Definitionssprache (XSD)-Schemata im [XmlSchemaSet](../) kompiliert wurden.

```cpp
bool System::Xml::Schema::XmlSchemaSet::get_IsCompiled()
```


### Rückgabewert

**true** wenn die Schemata im [XmlSchemaSet](../) seit dem letzten Hinzufügen oder Entfernen eines Schemas aus dem [XmlSchemaSet](../) kompiliert wurden; andernfalls **false**.

## HttpCompletionOption Enum

Gibt an, wann ein [HttpClient](../httpclient/)-Vorgang abgeschlossen sein soll.

```cpp
enum class HttpCompletionOption
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| ResponseContentRead | 0 | Der Vorgang sollte nach dem Lesen des Antwortinhalts abgeschlossen sein. |
| ResponseHeadersRead | 1 | Der Vorgang sollte nach dem Lesen der Header einer Antwort abgeschlossen sein. |

Gibt ein [XsltSettings](../) Objekt mit Standardeinstellungen zurück. Die Unterstützung für die XSLT **document()** Funktion und eingebettete Skriptblöcke ist deaktiviert.

static SharedPtr<XsltSettings> System::Xml::Xsl::XsltSettings::get_Default()

Ein [XsltSettings](../) Objekt mit den [XsltSettings::set_EnableDocumentFunction](../set_enabledocumentfunction/) und [XsltSettings::set_EnableScript](../set_enablescript/) Optionen auf **false** gesetzt.

## XsltSettings::get_EnableScript() Methode


Gibt einen Wert zurück, der angibt, ob die Unterstützung für eingebettete Skriptblöcke aktiviert werden soll.

```cpp
bool System::Xml::Xsl::XsltSettings::get_EnableScript()
```


### Rückgabewert

**true** um Skriptblöcke in XSLT-Stylesheets zu unterstützen; andernfalls **false**. Der Standardwert ist **false**.

## IXsltContextVariable::get_IsParam() Methode

Gibt einen Wert zurück, der angibt, ob die Variable ein Extensible Stylesheet Language Transformations (XSLT)-Parameter ist. Dies kann ein Parameter für ein Stylesheet oder eine Vorlage sein.

```cpp
virtual bool System::Xml::Xsl::IXsltContextVariable::get_IsParam()=0
```

### Rückgabewert

**true** wenn die Variable ein XSLT-Parameter ist; andernfalls **false**.

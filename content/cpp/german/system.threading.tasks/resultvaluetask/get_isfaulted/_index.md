## ResultValueTask::get_IsFaulted() const Methode


Erhält einen Wert, der angibt, ob die Aufgabe aufgrund einer nicht behandelten Ausnahme abgeschlossen wurde.

```cpp

```


### Rückgabewert

bool True, wenn die Aufgabe fehlerhaft ist; andernfalls false.

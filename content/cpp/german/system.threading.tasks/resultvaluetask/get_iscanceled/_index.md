## ResultValueTask::get_IsCanceled() const Methode

Gibt einen Wert zurück, der angibt, ob die Aufgabe aufgrund eines Abbruchs beendet wurde.

```cpp
bool System::Threading::Tasks::ResultValueTask<T>::get_IsCanceled() const
```

### Rückgabewert

bool True, wenn die Aufgabe abgebrochen wurde; andernfalls false.

## Anmerkungen

Gibt false zurück, wenn die Aufgabe einen direkten Ergebniswert enthält.

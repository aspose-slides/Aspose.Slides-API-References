## DataTableCollection Klasse

Sammlung von Datentabellen. Objekte dieser Klasse sollten nur mithilfe der [System::MakeObject()](../../system/makeobject/) Funktion zugewiesen werden. Erzeugen Sie niemals eine Instanz dieses Typs auf dem Stack oder mit dem Operator new, da dies zu Laufzeitfehlern und/oder Assertion-Fehlern führen kann. Wickeln Sie diese Klasse immer in einen [System::SmartPtr](../../system/smartptr/) Zeiger und verwenden Sie diesen Zeiger, um ihn als Argument an Funktionen zu übergeben.

```cpp
class DataTableCollection : public System::Object
```

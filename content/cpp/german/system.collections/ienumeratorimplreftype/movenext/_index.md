## IEnumeratorImplRefType::MoveNext() Methode


Verschiebt den Enumerator zum nächsten Element. Wenn vorher kein Element referenziert wurde, setzt die Referenz auf das erste verfügbare Element. Wenn das Ende des Containers erreicht ist, geschieht nichts.

```cpp
virtual bool System::Collections::IEnumeratorImplRefType<T>::MoveNext() override
```


### Rückgabewert

True, wenn nach dem Aufruf ein Element verfügbar ist, sonst false.

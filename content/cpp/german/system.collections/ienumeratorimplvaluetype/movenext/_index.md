## IEnumeratorImplValueType::MoveNext() Methode

Verschiebt den Enumerator zum nächsten Element. Wenn vorher kein Element referenziert wurde, wird die Referenz auf das erste verfügbare Element gesetzt. Wenn das Ende des Containers erreicht wurde, passiert nichts.

```cpp
virtual bool System::Collections::IEnumeratorImplValueType<T>::MoveNext() override
```

### Rückgabewert

True if some element is available after call, false otherwise.

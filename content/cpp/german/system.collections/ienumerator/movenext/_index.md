## IEnumerator::MoveNext() Methode

Verschiebt den Enumerator zum nächsten Element. Falls vorher kein Element referenziert wurde, wird die Referenz auf das erste verfügbare Element gesetzt. Wird das Ende des Containers erreicht, passiert nichts.

```cpp
virtual bool System::Collections::IEnumerator::MoveNext()=0
```

### Rückgabewert

True, wenn nach dem Aufruf ein Element verfügbar ist, andernfalls false.

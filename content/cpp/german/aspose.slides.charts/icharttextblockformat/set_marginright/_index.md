## IChartTextBlockFormat::set_MarginRight(double) Methode

Legt den rechten Rand (Punkte) in einem [TextFrame](../../../aspose.slides/textframe/) fest. Das Ändern dieser Eigenschaft kann nur für diese Diagrammteile einen bestimmten Einfluss haben: [DataLabel](../../datalabel/) und [DataLabelFormat](../../datalabelformat/) (vollständige Unterstützung in PowerPoint 2013; in PowerPoint 2007 hat es keine Auswirkung auf das Rendern). Schreiben **double**.

virtual void Aspose::Slides::Charts::IChartTextBlockFormat::set_MarginRight(double value)=0

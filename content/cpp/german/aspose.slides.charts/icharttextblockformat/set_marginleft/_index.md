## IChartTextBlockFormat::set_MarginLeft(double) Methode

Setzt den linken Rand (Punkte) in einem [TextFrame](../../../aspose.slides/textframe/). Die Änderung dieser Eigenschaft kann nur für diese Diagrammteile einen bestimmten Einfluss haben: [DataLabel](../../datalabel/) und [DataLabelFormat](../../datalabelformat/) (vollständige Unterstützung in PowerPoint 2013; in PowerPoint 2007 gibt es keinen Effekt beim Rendern). Verwenden Sie **double**.

virtual void Aspose::Slides::Charts::IChartTextBlockFormat::set_MarginLeft(double value)=0

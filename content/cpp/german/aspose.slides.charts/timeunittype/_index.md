## TimeUnitType enum


Stellt die Basiseinheit für die Kategorie-Achse dar.

```cpp
enum class TimeUnitType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | 0 | Werte werden unverändert angezeigt. |
| Days | 1 | Gibt an, dass die Diagrammdaten in Tagen angezeigt werden. |
| Months | 2 | Gibt an, dass die Diagrammdaten in Monaten angezeigt werden. |
| Years | 3 | Gibt an, dass die Diagrammdaten in Jahren angezeigt werden. |

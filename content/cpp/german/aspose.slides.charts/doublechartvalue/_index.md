## DoubleChartValue Klasse

Stellt einen double-Wert dar, der in einer pptx-Präsentationsdatei auf zwei Arten gespeichert werden kann: 1) in Zelle/Zellen einer dem Diagramm zugehörigen Arbeitsmappe; 2) als Literalwert.

```cpp
class DoubleChartValue : public Aspose::Slides::Charts::BaseChartValue,
                         public Aspose::Slides::Charts::IDoubleChartValue
```

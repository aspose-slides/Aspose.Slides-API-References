## AxisAggregationType enum

Stellt den Aggregationstyp der Kategorienachse dar. Wird nur mit Histogram oder HistogramPareto Serien verwendet.

```cpp
enum class AxisAggregationType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| ByCategory | 0 | Aggregiert Kategorien |
| Automatic | 1 | Aggregiert die Kategorien automatisch. |
| ByBinWidth | 2 | Aggregiert Kategorien nach Bin-Breite. Property IAxis::get(set)_BinWidth muss gesetzt werden. |
| ByNumberOfBins | 3 | Aggregiert Kategorien nach Anzahl der Bins. Property IAxis::get(set)_NumberOfBins muss gesetzt werden. |

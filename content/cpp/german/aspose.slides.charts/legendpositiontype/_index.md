## LegendPositionType enum

Bestimmt die Position der Legende in einem Diagramm.

```cpp
enum class LegendPositionType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Bottom | 0 | Gibt an, dass die Legende unten im Diagramm gezeichnet werden soll. |
| Left | 1 | Gibt an, dass die Legende links im Diagramm gezeichnet werden soll. |
| Right | 2 | Gibt an, dass die Legende rechts im Diagramm gezeichnet werden soll. |
| Top | 3 | Gibt an, dass die Legende oben im Diagramm gezeichnet werden soll. |
| TopRight | 4 | Gibt an, dass die Legende oben rechts im Diagramm gezeichnet werden soll. |

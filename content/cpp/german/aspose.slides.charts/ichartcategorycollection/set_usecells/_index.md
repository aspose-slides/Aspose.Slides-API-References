## IChartCategoryCollection::set_UseCells(bool) Methode

Wenn true, dann wird das Arbeitsblatt zur Speicherung von Kategorien verwendet (dieser Fall unterstützt mehrstufige Kategorien). Wenn false, dann wird das Arbeitsblatt NICHT zur Speicherung von Werten verwendet (und dieser Fall unterstützt keine mehrstufigen Kategorien). Schreiben **bool**.

virtual void Aspose::Slides::Charts::IChartCategoryCollection::set_UseCells(bool value)=0

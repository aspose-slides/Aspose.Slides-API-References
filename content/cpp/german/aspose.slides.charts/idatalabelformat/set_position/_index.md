## IDataLabelFormat::set_Position(LegendDataLabelPosition) Methode


Stellt die Position des Datenlabels dar. Schreiben Sie [LegendDataLabelPosition](../../legenddatalabelposition/).

```cpp
virtual void Aspose::Slides::Charts::IDataLabelFormat::set_Position(LegendDataLabelPosition value)=0
```

## Anmerkungen


Wenn das übergeordnete Element dieses [DataLabelFormat](../../datalabelformat/)-Objekts eine [DataLabelCollection](../../datalabelcollection/)-Sammlung von Datenbeschriftungen ist, dann gibt diese Eigenschaft den Standardwert der Position-Eigenschaft für die neuen Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/)-Sammlung zurück oder legt ihn fest. Stellt die Position für die [DataLabel](../../datalabel/)-Objekte dar. Wird diese Eigenschaft mit einem Wert gesetzt, wird dieser Wert auch für die Position-Eigenschaft aller Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/)-Sammlung gesetzt (z. B. \"DataLabels.DefaultDataLabelFormat.Position = val;\" cause to all DataLabels[i].Position is equal to val).

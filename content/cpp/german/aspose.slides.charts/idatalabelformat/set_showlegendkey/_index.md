## IDataLabelFormat::set_ShowLegendKey(bool) Methode

Stellt das Anzeigeverhalten des Legenden-Schlüssels einer Datenbeschriftung eines angegebenen Diagramms dar. Wahr, wenn der Legenden-Schlüssel der Datenbeschriftung sichtbar ist. Schreiben **bool**.

```cpp
virtual void Aspose::Slides::Charts::IDataLabelFormat::set_ShowLegendKey(bool value)=0
```

## Anmerkungen

Wenn der übergeordnete Knoten dieses [DataLabelFormat](../../datalabelformat/)-Objekts eine [DataLabelCollection](../../datalabelcollection/)-Sammlung von Datenbeschriftungen ist, dann gibt diese Eigenschaft den Standardwert der ShowLegendKey-Eigenschaft für die neuen Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/)-Sammlung zurück oder legt ihn fest. Das Setzen dieser Eigenschaft mit einem Wert setzt diesen Wert auch für die ShowLegendKey-Eigenschaft aller Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/)-Sammlung (d. h. "DataLabels.DefaultDataLabelFormat.ShowLegendKey = val;" führt dazu, dass alle DataLabels[i].ShowLegendKey gleich val ist).

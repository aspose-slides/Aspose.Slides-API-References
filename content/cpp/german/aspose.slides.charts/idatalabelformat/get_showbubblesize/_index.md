## IDataLabelFormat::get_ShowBubbleSize() Methode


Stellt das Anzeigeverhalten des Bubble-Größenwerts eines bestimmten Diagrammsdatenlabels dar. True zeigt den Bubble-Größenwert an. False verbirgt ihn. Lesen **bool**.

```cpp
virtual bool Aspose::Slides::Charts::IDataLabelFormat::get_ShowBubbleSize()=0
```

## Hinweise


Wenn das übergeordnete Element dieses [DataLabelFormat](../../datalabelformat/) Objekts eine [DataLabelCollection](../../datalabelcollection/) Sammlung von Datenbeschriftungen ist, dann ruft diese Eigenschaft den Standardwert der ShowBubbleSize-Eigenschaft für die neuen Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/) Sammlung ab oder setzt ihn. Das Setzen dieser Eigenschaft mit einem Wert setzt diesen ebenfalls für die ShowBubbleSize-Eigenschaft aller Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/) Sammlung (z. B. \"DataLabels.DefaultDataLabelFormat.ShowBubbleSize = val;\" bewirkt, dass alle DataLabels[i].ShowBubbleSize gleich val sind).

## IDataLabelFormat::get_IsNumberFormatLinkedToSource() method


Lesen **bool**.

```cpp
virtual bool Aspose::Slides::Charts::IDataLabelFormat::get_IsNumberFormatLinkedToSource()=0
```

## Bemerkungen


Wenn das übergeordnete Element dieses [DataLabelFormat](../../datalabelformat/)-Objekts eine [DataLabelCollection](../../datalabelcollection/)-Sammlung von Datenbeschriftungen ist, dann gibt diese Eigenschaft den Standardwert der IsNumberFormatLinkedToSource-Eigenschaft für die neuen Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/)-Sammlung zurück oder setzt ihn. Das Setzen dieser Eigenschaft mit einem Wert legt diesen Wert ebenfalls für die IsNumberFormatLinkedToSource-Eigenschaft aller Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/)-Sammlung fest (i.e. \"DataLabels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = val;\" cause to all DataLabels[i].IsNumberFormatLinkedToSource is equal to val).

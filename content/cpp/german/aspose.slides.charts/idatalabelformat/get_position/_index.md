## IDataLabelFormat::get_Position() Methode

Stellt die Position der Datenbeschriftung dar. Lesen Sie [LegendDataLabelPosition](../../legenddatalabelposition/).

```cpp
virtual LegendDataLabelPosition Aspose::Slides::Charts::IDataLabelFormat::get_Position()=0
```

## Anmerkungen

Wenn das übergeordnete Element dieses [DataLabelFormat](../../datalabelformat/)-Objekts eine [DataLabelCollection](../../datalabelcollection/)-Sammlung von Datenbeschriftungen ist, dann ruft diese Eigenschaft den Standardwert der Position-Eigenschaft für die neuen Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/)-Sammlung ab oder legt ihn fest. Stellt die Position für die [DataLabel](../../datalabel/)-Objekte dar. Setzen Sie diese Eigenschaft mit einem Wert, legt diesen Wert auch für die Position-Eigenschaft aller Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/)-Sammlung fest (z. B. "DataLabels.DefaultDataLabelFormat.Position = val;" bewirkt, dass alle DataLabels[i].Position den Wert val haben).

## IDataLabelFormat::get_ShowPercentage() Methode

Stellt das Anzeigeverhalten des Prozentwerts eines angegebenen Diagrammdatenelements dar. True zeigt den Prozentwert an. False versteckt ihn. Lese **bool**.

```cpp
virtual bool Aspose::Slides::Charts::IDataLabelFormat::get_ShowPercentage()=0
```

## Hinweise

Wenn der übergeordnete von diesem [DataLabelFormat](../../datalabelformat/) Objekt eine [DataLabelCollection](../../datalabelcollection/) Sammlung von Datenbeschriftungen ist, dann bekommt oder setzt diese Eigenschaft den Standardwert der ShowPercentage-Eigenschaft für die neuen Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/) Sammlung. Setzt man diese Eigenschaft mit einem Wert, wird dieser Wert auch für die ShowPercentage-Eigenschaft aller Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/) Sammlung gesetzt (i.e. \"DataLabels.DefaultDataLabelFormat.ShowPercentage = val;\" cause to all DataLabels[i].ShowPercentage is equal to val).

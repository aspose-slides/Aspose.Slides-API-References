## IDataLabelFormat::set_ShowPercentage(bool) Methode


Stellt das Anzeigeverhalten des Prozentwerts von Datenbeschriftungen eines angegebenen Diagramms dar. True zeigt den Prozentwert an. False blendet ihn aus. Schreiben **bool**.

```cpp
virtual void Aspose::Slides::Charts::IDataLabelFormat::set_ShowPercentage(bool value)=0
```

## Hinweise


Wenn das übergeordnete Element dieses [DataLabelFormat](../../datalabelformat/)-Objekts eine [DataLabelCollection](../../datalabelcollection/)-Sammlung von Datenbeschriftungen ist, ruft diese Eigenschaft den Standardwert der ShowPercentage-Eigenschaft für die neuen Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/)-Sammlung ab oder setzt ihn. Das Setzen dieser Eigenschaft mit einem Wert setzt diesen Wert ebenfalls für die ShowPercentage-Eigenschaft aller Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/)-Sammlung (z. B. "DataLabels.DefaultDataLabelFormat.ShowPercentage = val;" bewirkt, dass für alle DataLabels[i].ShowPercentage der Wert val gilt).

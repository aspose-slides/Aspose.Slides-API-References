## LegendDataLabelPosition enum

Bestimmt die Position von Datenbeschriftungen.

```cpp
enum class LegendDataLabelPosition
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| NotDefined | -1 | Position ist nicht definiert. |
| Bottom | 0 | Gibt an, dass Datenbeschriftungen unter dem Datenmarker angezeigt werden sollen. |
| BestFit | 1 | Gibt an, dass Datenbeschriftungen in der besten Position angezeigt werden sollen. |
| Center | 2 | Gibt an, dass Datenbeschriftungen zentriert auf dem Datenmarker angezeigt werden sollen. |
| InsideBase | 3 | Gibt an, dass Datenbeschriftungen innerhalb der Basis des Datenmarkers angezeigt werden sollen. |
| InsideEnd | 4 | Gibt an, dass Datenbeschriftungen innerhalb des Endes des Datenmarkers angezeigt werden sollen. |
| Left | 5 | Gibt an, dass Datenbeschriftungen links vom Datenmarker angezeigt werden sollen. |
| OutsideEnd | 6 | Gibt an, dass Datenbeschriftungen außerhalb des Endes des Datenmarkers angezeigt werden sollen. |
| Right | 7 | Gibt an, dass Datenbeschriftungen rechts vom Datenmarker angezeigt werden sollen. |
| Top | 8 | Gibt an, dass Datenbeschriftungen über dem Datenmarker angezeigt werden sollen. |

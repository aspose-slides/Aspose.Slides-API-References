## ChartSeries::get_BubbleSizeRepresentation() Methode

Specifiziert, wie die bubble size values auf dem bubble chart dargestellt werden. Dies ist die property nicht nur dieser series, sondern aller series der parent series group – dies ist die projection der entsprechenden group property. Und daher ist diese property schreibgeschützt. Verwenden Sie die property ParentSeriesGroup für den Zugriff auf die parent series group. Verwenden Sie [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get(set)_BubbleSizeRepresentation() Lese-/Schreib-Eigenschaft, um den Wert zu ändern.

```cpp
BubbleSizeRepresentationType Aspose::Slides::Charts::ChartSeries::get_BubbleSizeRepresentation() override
```

## Anmerkungen

Dies ist die projection der property [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get(set)_BubbleSizeRepresentation().

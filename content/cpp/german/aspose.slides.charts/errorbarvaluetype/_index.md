## ErrorBarValueType Enum

Stellt den Typ des Error Bar Value dar

```cpp
enum class ErrorBarValueType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Custom | 0 | Gibt an, dass die Länge der Fehlerbalken durch die Plus- und Minus-Elemente bestimmt wird. |
| Fixed | 1 | Gibt an, dass die Länge der Fehlerbalken der feste Wert ist, der durch Error Bar Value bestimmt wird. |
| Percentage | 2 | Gibt an, dass die Länge der Fehlerbalken Error Bar Value Prozent der Daten beträgt. |
| StandardDeviation | 3 | Gibt an, dass die Länge der Fehlerbalken Error Bar Value Standardabweichungen der Daten beträgt. |
| StandardError | 4 | Gibt an, dass die Länge der Fehlerbalken Error Bar Value Standardfehler der Daten beträgt. |

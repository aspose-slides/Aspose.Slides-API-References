## DataLabelCollection::get_IsVisible() Methode

False bedeutet, dass das Datenbeschriftung standardmäßig nicht sichtbar ist (und somit alle Show*-Flags (ShowValue, ...) der Eigenschaft DefaultDataLabelFormat false sind). Nur lesbar **bool**.

```cpp
bool Aspose::Slides::Charts::DataLabelCollection::get_IsVisible() override
```

## Hinweise

Wenn die Datenbeschriftung standardmäßig sichtbar ist, können Sie sie standardmäßig mit der [Hide()](../hide/) Methode ausblenden. Wenn die Datenbeschriftung jedoch standardmäßig nicht sichtbar ist (IsVisible ist false), können Sie die Datenbeschriftung "standardmäßig sichtbar" machen, indem Sie die Show*-Flags (ShowValue, ...) der Eigenschaft DefaultDataLabelFormat auf true setzen.

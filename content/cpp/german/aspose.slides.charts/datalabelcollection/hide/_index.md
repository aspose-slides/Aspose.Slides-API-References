## DataLabelCollection::Hide() Methode

Machen Sie das Datenbeschriftung standardmäßig ausgeblendet, indem Sie alle Show*-Flags (ShowValue, ...) der Eigenschaft DefaultDataLabelFormat auf den falschen Zustand setzen. IsVisible wird danach false sein.

```cpp
void Aspose::Slides::Charts::DataLabelCollection::Hide() override
```

## Anmerkungen

Wenn das Datenbeschriftung standardmäßig nicht sichtbar ist (IsVisible ist false), können Sie das Datenbeschriftung "standardmäßig sichtbar" machen, indem Sie die Show*-Flags (ShowValue, ...) der Eigenschaft DefaultDataLabelFormat auf den wahren Zustand setzen.

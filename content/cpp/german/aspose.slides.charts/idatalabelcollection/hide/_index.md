## IDataLabelCollection::Hide() Methode

Machen Sie die Datenbeschriftung standardmäßig ausgeblendet, indem Sie alle Show*-Flags (ShowValue, ...) der Eigenschaft DefaultDataLabelFormat auf den Zustand false setzen. IsVisible wird danach false sein.

```cpp
virtual void Aspose::Slides::Charts::IDataLabelCollection::Hide()=0
```

## Anmerkungen

Wenn die Datenbeschriftung standardmäßig nicht sichtbar ist (IsVisible ist false), können Sie die Datenbeschriftung "standardmäßig sichtbar" machen, indem Sie die Show*-Flags (ShowValue, ...) der Eigenschaft DefaultDataLabelFormat auf den Zustand true setzen.

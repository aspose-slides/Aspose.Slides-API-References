## BubbleSizeRepresentationType enum

Gibt die möglichen Arten an, Daten als Größen von Blasendiagrammen darzustellen.

```cpp
enum class BubbleSizeRepresentationType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Area | 0 | Gibt an, dass die Fläche der Blasen proportional zum Wert der Blasengröße sein soll. |
| Width | 1 | Gibt an, dass der Radius der Blasen proportional zum Wert der Blasengröße sein soll. |

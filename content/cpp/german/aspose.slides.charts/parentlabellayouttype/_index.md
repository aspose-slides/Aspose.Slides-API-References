## ParentLabelLayoutType Enum

Stellt das Layout von Kategoriedatenbeschriftungen dar. Wird nur mit Treemap-Serien verwendet.

```cpp
enum class ParentLabelLayoutType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Banner | 0 | Gibt an, dass übergeordnete Kategoriebeschriftungen als Banner angezeigt werden |
| None | 1 | Gibt an, dass übergeordnete Kategoriebeschriftungen nicht gezeichnet werden |
| Overlapping | 2 | Gibt an, dass übergeordnete Kategoriebeschriftungen überlappend angezeigt werden |

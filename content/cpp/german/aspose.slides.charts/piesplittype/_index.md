## PieSplitType enum


Stellt einen Typ von Aufteilungspunkten im zweiten Kuchen oder Balken eines Kuchen-aus-Kuchen- oder Balken-aus-Kuchen-Diagramms dar.

```cpp
enum class PieSplitType
```

### Values

| Name | Value | Description |
| --- | --- | --- |
| Default | 0 | Gibt an, dass die Datenpunkte mit dem Standardmechanismus für diesen Diagrammtyp aufgeteilt werden. |
| Custom | 1 | Gibt an, dass die Datenpunkte gemäß den Custom Split Werten zwischen dem Kuchen und dem zweiten Diagramm aufgeteilt werden. |
| ByPercentage | 2 | Gibt an, dass die Datenpunkte zwischen dem Kuchen und dem zweiten Diagramm aufgeteilt werden, indem Punkte mit einem Prozentsatz kleiner als Split Position Prozent im zweiten Diagramm platziert werden. |
| ByPos | 3 | Gibt an, dass die Datenpunkte zwischen dem Kuchen und dem zweiten Diagramm aufgeteilt werden, indem die letzte Split Position der Datenpunkte im zweiten Diagramm platziert wird. |
| ByValue | 4 | Gibt an, dass die Datenpunkte zwischen dem Kuchen und dem zweiten Diagramm aufgeteilt werden, indem Datenpunkte mit einem Wert kleiner als Split Position im zweiten Diagramm platziert werden. |

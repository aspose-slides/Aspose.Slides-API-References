## IDataLabel::get_IsVisible() Methode


False means that data label is not visible (and so all Show*-flags (ShowValue, ...) are false). Nur lesbar **bool**.

```cpp
virtual bool Aspose::Slides::Charts::IDataLabel::get_IsVisible()=0
```

## Anmerkungen


Wenn das Datenbeschriftungselement sichtbar ist, können Sie es mit der [Hide()](../hide/) Methode ausblenden. Wenn das Datenbeschriftungselement jedoch nicht sichtbar ist (IsVisible ist false), können Sie das Datenbeschriftungselement sichtbar machen, indem Sie die Show*-flags (ShowValue, ...) auf true setzen.

## IDataLabel::Hide() Methode


Versteckt das Datenbeschriftungselement, indem alle Show*-Flags (ShowValue, ...) auf den falschen Zustand gesetzt werden. IsVisible wird danach false sein.

```cpp
virtual void Aspose::Slides::Charts::IDataLabel::Hide()=0
```

## Anmerkungen


Wenn die Datenbeschriftung nicht sichtbar ist (IsVisible ist false), können Sie die Datenbeschriftung sichtbar machen, indem Sie die Show*-Flags (ShowValue, ...) auf den wahren Zustand setzen.

## TickMarkType enum

Stellt den Tick-Mark-Typ für die angegebene Achse dar.

```cpp
enum class TickMarkType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Cross | 0 | Gibt an, dass die Tick-Marken die Achse kreuzen. |
| Inside | 1 | Gibt an, dass die Tick-Marken im Plot-Bereich liegen. |
| None | 2 | Gibt an, dass keine Tick-Marken vorhanden sind. |
| Outside | 3 | Gibt an, dass die Tick-Marken außerhalb des Plot-Bereichs liegen. |

## DataSourceType Enum

Datentypen für Datenquellen.

```cpp
enum class DataSourceType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Worksheet | 0 | Verwenden Sie Worksheet-Objekte zum Speichern von XValue, YValue, Value oder BubbleSize des Datenpunkts im Arbeitsblatt. |
| StringLiterals | 1 | Verwenden Sie eine schnellzugreifbare String-Variable zum Speichern von XValue, YValue, Value oder BubbleSize des Datenpunkts. Worksheet wird NICHT zum Speichern von Werten verwendet. |
| DoubleLiterals | 2 | Verwenden Sie eine schnellzugreifbare Double-Variable zum Speichern von XValue, YValue, Value oder BubbleSize des Datenpunkts. Worksheet wird NICHT zum Speichern von Werten verwendet. |

## DataLabelFormat::set_Position(LegendDataLabelPosition) Methode


Stellt die Position des Datenlabels dar. Schreiben Sie [LegendDataLabelPosition](../../legenddatalabelposition/).

```cpp
void Aspose::Slides::Charts::DataLabelFormat::set_Position(LegendDataLabelPosition value) override
```

## Hinweise


Wenn das übergeordnete Objekt dieses [DataLabelFormat](../) ein [DataLabelCollection](../../datalabelcollection/)-Sammlung von Datenbeschriftungen ist, dann liefert diese Eigenschaft den Standardwert der Position-Eigenschaft für die neuen Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/)-Sammlung oder legt ihn fest. Stellt die Position für die [DataLabel](../../datalabel/)-Objekte dar. Wenn Sie diese Eigenschaft mit einem Wert setzen, wird dieser Wert auch der Position-Eigenschaft aller Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/)-Sammlung zugewiesen (z. B. "DataLabels.DefaultDataLabelFormat.Position = val;" bewirkt, dass alle DataLabels[i].Position den Wert val haben).

## DataLabelFormat::get_IsNumberFormatLinkedToSource() Methode

```cpp
bool Aspose::Slides::Charts::DataLabelFormat::get_IsNumberFormatLinkedToSource() override
```

## Hinweise

Wenn das übergeordnete Element dieses [DataLabelFormat](../) Objekts eine [DataLabelCollection](../../datalabelcollection/) Sammlung von Datenbeschriftungen ist, dann ruft diese Eigenschaft den Standardwert der IsNumberFormatLinkedToSource-Eigenschaft für die neuen Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/) Sammlung ab bzw. setzt ihn. Das Setzen dieser Eigenschaft mit einem Wert legt diesen Wert auch für die IsNumberFormatLinkedToSource-Eigenschaft aller Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/) Sammlung fest (d.h. "DataLabels.DefaultDataLabelFormat.IsNumberFormatLinkedToSource = val;" führt dazu, dass alle DataLabels[i].IsNumberFormatLinkedToSource den Wert val haben).

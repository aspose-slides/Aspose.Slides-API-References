## DataLabelFormat::get_ShowSeriesName() Methode

Rückgibt einen Boolean, um das Anzeigeverhalten des Seriennamens für die Datenbeschriftungen in einem Diagramm anzugeben. True, um den Seriennamen anzuzeigen. False, um ihn zu verbergen. Lesen **bool**.

```cpp
bool Aspose::Slides::Charts::DataLabelFormat::get_ShowSeriesName() override
```

## Anmerkungen

Wenn das übergeordnete Element dieses [DataLabelFormat](../)-Objekts eine [DataLabelCollection](../../datalabelcollection/)-Sammlung von Datenbeschriftungen ist, dann ruft diese Eigenschaft den Standardwert der ShowSeriesName-Eigenschaft für die neuen Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/)-Sammlung ab oder setzt ihn. Das Setzen dieser Eigenschaft mit einem Wert legt diesen Wert auch für die ShowSeriesName-Eigenschaft aller Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/)-Sammlung fest (z. B. "DataLabels.DefaultDataLabelFormat.ShowSeriesName = val;" bewirkt, dass für alle DataLabels[i].ShowSeriesName der Wert val gilt).

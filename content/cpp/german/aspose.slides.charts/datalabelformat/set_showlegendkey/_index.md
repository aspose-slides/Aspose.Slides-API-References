## DataLabelFormat::set_ShowLegendKey(bool) Methode

Stellt das Anzeigeverhalten des Legenden-Schlüssels der Datenbeschriftung eines bestimmten Diagramms dar. Wahr, wenn der Legenden-Schlüssel der Datenbeschriftung sichtbar ist. Schreiben **bool**.

```cpp
void Aspose::Slides::Charts::DataLabelFormat::set_ShowLegendKey(bool value) override
```

## Hinweise

Wenn das übergeordnete Element dieses [DataLabelFormat](../)-Objekts eine [DataLabelCollection](../../datalabelcollection/)-Sammlung von Datenbeschriftungen ist, dann ruft diese Eigenschaft den Standardwert der ShowLegendKey-Eigenschaft für die neuen Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/)-Sammlung ab oder legt ihn fest. Das Setzen dieser Eigenschaft mit einem Wert setzt diesen Wert ebenfalls für die ShowLegendKey-Eigenschaft aller Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/)-Sammlung (z.B. "DataLabels.DefaultDataLabelFormat.ShowLegendKey = val;" bewirkt, dass für alle DataLabels[i].ShowLegendKey der Wert val gilt).

## DataLabelFormat::get_ShowBubbleSize() Methode

Stellt das Anzeigeverhalten des Bubble-Size-Werts einer angegebenen Diagrammdatenbeschriftung dar. True zeigt den Bubble-Size-Wert an. False zum Ausblenden. Lesen **bool**.

```cpp
bool Aspose::Slides::Charts::DataLabelFormat::get_ShowBubbleSize() override
```

## Hinweise

Wenn das übergeordnete Element dieses [DataLabelFormat](../)-Objekts eine [DataLabelCollection](../../datalabelcollection/)-Sammlung von Datenbeschriftungen ist, dann ruft diese Eigenschaft den Standardwert der ShowBubbleSize-Eigenschaft für die neuen Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/)-Sammlung ab oder legt ihn fest. Das Setzen dieser Eigenschaft mit einem Wert setzt diesen Wert ebenfalls für die ShowBubbleSize-Eigenschaft aller Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/)-Sammlung (z. B. "DataLabels.DefaultDataLabelFormat.ShowBubbleSize = val;" bewirkt, dass alle DataLabels[i].ShowBubbleSize den Wert val haben).

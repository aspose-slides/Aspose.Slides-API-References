## DataLabelFormat::get_ShowLeaderLines() Methode


Stellt das Anzeigeverhalten der LeaderLines von Datenbeschriftungen eines angegebenen Diagramms dar. True zeigt die LeaderLines an. False blendet sie aus. Lesen **bool**.

```cpp
bool Aspose::Slides::Charts::DataLabelFormat::get_ShowLeaderLines() override
```

## Anmerkungen


Wenn der übergeordnete Knoten dieses [DataLabelFormat](../)-Objekts eine [DataLabelCollection](../../datalabelcollection/)-Sammlung von Datenbeschriftungen ist, dann ruft diese Eigenschaft den Standardwert der ShowLeaderLines-Eigenschaft für die neuen Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/)-Sammlung ab oder setzt ihn. Das Setzen dieser Eigenschaft mit einem Wert legt diesen Wert auch für die ShowLeaderLines-Eigenschaft aller Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/)-Sammlung fest (z. B. "DataLabels.DefaultDataLabelFormat.ShowLeaderLines = val;" bewirkt, dass für alle DataLabels[i].ShowLeaderLines der Wert val gilt).

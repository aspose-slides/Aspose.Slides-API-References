## DataLabelFormat::set_ShowBubbleSize(bool) Methode

Stellt das Anzeigeverhalten des Datenbeschriftungs-Bubble-Größenwerts eines angegebenen Diagramms dar. True zeigt den Bubble-Größenwert an. False verbirgt ihn. Schreiben **bool**.

```cpp
void Aspose::Slides::Charts::DataLabelFormat::set_ShowBubbleSize(bool value) override
```

## Anmerkungen

Wenn das übergeordnete Element dieses [DataLabelFormat](../)-Objekts eine [DataLabelCollection](../../datalabelcollection/)-Sammlung von Datenbeschriftungen ist, dann ruft diese Eigenschaft den Standardwert der ShowBubbleSize-Eigenschaft ab oder setzt ihn für die neuen Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/)-Sammlung. Das Setzen dieser Eigenschaft mit einem Wert setzt diesen Wert ebenfalls für die ShowBubbleSize-Eigenschaft aller Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/)-Sammlung (z. B. „DataLabels.DefaultDataLabelFormat.ShowBubbleSize = val;“, wodurch alle DataLabels[i].ShowBubbleSize gleich val sind).

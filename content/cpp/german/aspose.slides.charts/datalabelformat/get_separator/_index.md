## DataLabelFormat::get_Separator() Methode

Setzt oder gibt eine Variant zurück, die das Trennzeichen für die Datenbeschriftungen in einem Diagramm darstellt. Lesen [System::String](../../../system/string/).

```cpp
System::String Aspose::Slides::Charts::DataLabelFormat::get_Separator() override
```

## Hinweise

Wenn das übergeordnete Element dieses [DataLabelFormat](../)-Objekts eine [DataLabelCollection](../../datalabelcollection/)-Sammlung von Datenbeschriftungen ist, dann liest bzw. schreibt diese Eigenschaft den Standardwert der Eigenschaft Separator für die neuen Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/)-Sammlung. Das Setzen dieser Eigenschaft mit einem Wert setzt diesen Wert außerdem in der Eigenschaft Separator für alle Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/)-Sammlung (z. B. "DataLabels.DefaultDataLabelFormat.Separator = val;" bewirkt, dass alle DataLabels[i].Separator den Wert val haben).

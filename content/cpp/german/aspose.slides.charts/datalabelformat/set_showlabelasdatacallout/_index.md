## DataLabelFormat::set_ShowLabelAsDataCallout(bool) Methode

Bestimmt, ob die Datenbeschriftung des angegebenen Diagramms als Daten-Callout oder als Datenbeschriftung angezeigt wird.

```cpp
void Aspose::Slides::Charts::DataLabelFormat::set_ShowLabelAsDataCallout(bool value) override
```

## Bemerkungen

Wenn das übergeordnete [DataLabelFormat](../)-Objekt eine [DataLabelCollection](../../datalabelcollection/)-Sammlung von Datenbeschriftungen ist, dann ruft diese Eigenschaft den Standardwert der ShowLabelAsDataCallout-Eigenschaft für die neuen Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/)-Sammlung ab oder legt ihn fest. Das Setzen dieser Eigenschaft mit einem Wert legt diesen Wert ebenfalls für die ShowLabelAsDataCallout-Eigenschaft aller Datenbeschriftungen in der [DataLabelCollection](../../datalabelcollection/)-Sammlung fest (z. B. "DataLabels.DefaultDataLabelFormat.ShowLabelAsDataCallout = val;" bewirkt, dass alle DataLabels[i].ShowLabelAsDataCallout den Wert val haben).

## DataLabel::Hide() Methode


Macht das Daten-Label unsichtbar, indem alle Show*-Flags (ShowValue, ...) auf den falschen Zustand gesetzt werden. IsVisible wird danach false sein.

```cpp
void Aspose::Slides::Charts::DataLabel::Hide() override
```

## Bemerkungen


Falls das Daten-Label nicht sichtbar ist (IsVisible ist false), kann das Daten-Label wieder sichtbar gemacht werden, indem die Show*-Flags (ShowValue, ...) auf den wahren Zustand gesetzt werden.

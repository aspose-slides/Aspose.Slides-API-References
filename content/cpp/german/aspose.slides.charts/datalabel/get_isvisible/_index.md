## DataLabel::get_IsVisible() Methode


False bedeutet, dass das Datenbeschriftung nicht sichtbar ist (und so alle Show*-Flags (ShowValue, ...) false sind). Schreibgeschützt **bool**.

```cpp
bool Aspose::Slides::Charts::DataLabel::get_IsVisible() override
```

## Hinweise


Wenn das Datenbeschriftung sichtbar ist, können Sie es mit der [Hide()](../hide/) Methode ausblenden. Wenn das Datenbeschriftung jedoch nicht sichtbar ist (IsVisible ist false), können Sie das Datenbeschriftung wieder sichtbar machen, indem Sie die Show*-Flags (ShowValue, ...) auf true setzen.

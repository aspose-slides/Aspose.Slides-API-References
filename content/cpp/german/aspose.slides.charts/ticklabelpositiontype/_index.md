## TickLabelPositionType enum

Stellt den Positionstyp von Tick-Mark-Beschriftungen auf der angegebenen Achse dar.

```cpp
enum class TickLabelPositionType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| High | 0 | Gibt an, dass die Achsenbeschriftungen am oberen Ende der senkrechten Achse liegen. |
| Low | 1 | Gibt an, dass die Achsenbeschriftungen am unteren Ende der senkrechten Achse liegen. |
| NextTo | 2 | Gibt an, dass die Achsenbeschriftungen neben der Achse liegen. |
| None | 3 | Gibt an, dass die Achsenbeschriftungen nicht gezeichnet werden. |

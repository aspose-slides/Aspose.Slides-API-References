## IAxis::set_NumberOfBins(uint32_t) Methode


Gibt die Anzahl der Bins an, wenn der Wert der AggregationType-Eigenschaft auf [AxisAggregationType::ByNumberOfBins](../../axisaggregationtype/) gesetzt ist. Wird auf kategorialen Achsen angewendet. Wird ausschließlich mit Histogram- oder HistogramPareto-Serien verwendet.

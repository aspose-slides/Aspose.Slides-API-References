## BaseChartValue::set_DataSourceType(Aspose::Slides::Charts::DataSourceType) Methode

Gibt an, ob die Eigenschaften AsCell, AsCells, AsLiteralString oder AsLiteralDouble in abgeleiteten Klassen tatsächlich vorhanden sind. Mit anderen Worten gibt sie den Typ des Werts der Data-Eigenschaft an. Schreiben Sie [Charts::DataSourceType](../../datasourcetype/).

```cpp
void Aspose::Slides::Charts::BaseChartValue::set_DataSourceType(Aspose::Slides::Charts::DataSourceType value) override
```

## Anmerkungen

Für Punkte in [ChartDataPointCollection](../../chartdatapointcollection/) ist diese Eigenschaft schreibgeschützt. In diesem Fall können Sie zum Ändern des Werts dieser Eigenschaft eine der ChartDataPointCollection.DataSourceTypeFor<...>-Eigenschaften verwenden.

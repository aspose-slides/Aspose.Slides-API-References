## IStringChartValue Klasse

Stellt einen Zeichenkettenwert dar, der in einer pptx-Präsentationsdatei auf zwei Arten gespeichert werden kann: 1) in Zelle(n) einer zum Diagramm zugehörigen Arbeitsmappe; 2) als Literalwert.

```cpp
class IStringChartValue : public Aspose::Slides::Charts::IMultipleCellChartValue
```

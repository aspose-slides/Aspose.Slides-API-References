## PictureType Enum

Bestimmt den Modus der Bildfüllung für Balken.

```cpp
enum class PictureType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| NotDefined | -1 | Nicht definiert. |
| Stack | 0 | Das Bild wird so skaliert, dass es maximal 15 Mal im längsten gestapelten Balken wiederholt wird. |
| StackScale | 1 | Das Bild wird auf eine angegebene Anzahl von Einheiten skaliert und über die Länge des Balkens wiederholt. |
| Stretch | 2 | Das Bild wird über die gesamte Länge des gestapelten Balkens gestreckt. |

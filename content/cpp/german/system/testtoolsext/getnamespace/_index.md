## TestToolsExt::GetNamespace(const char *, const char *, std::string\&) Methode

Ruft den Namensraum des angegebenen Tests ab.

```cpp
static bool System::TestToolsExt::GetNamespace(const char *class_name, const char *method_name, std::string &name_space)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| class_name | const char * | Klasse, nach der gesucht wird. |
| method_name | const char * | Methode, nach der gesucht wird. |
| name_space | std::string\& | Variable, in die der Namensraumname geschrieben wird, falls gefunden. |

### Rückgabewert

true, wenn die Testmethode gefunden wird, sonst false.

## Convert::ToDateTime(const SharedPtr\<Object\>\&, const SharedPtr\<IFormatProvider\>\&) Methode


Konvertiert den angegebenen verpackten Wert in den entsprechenden [DateTime](../../datetime/)-Wert.

```cpp
static DateTime System::Convert::ToDateTime(const SharedPtr<Object> &obj, const SharedPtr<IFormatProvider> &provider=nullptr)
```


### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| obj | const [SharedPtr](../../sharedptr/)\<[Object](../../object/)\>\& | Der Shared-Pointer auf das Objekt, das den zu konvertierenden Wert verpackt |
| provider | const [SharedPtr](../../sharedptr/)\<[IFormatProvider](../../iformatprovider/)\>\& | Das Zeichenfolgenformat, das verwendet werden soll, wenn der Typ des verpackten Werts [String](../../string/) ist |

### Rückgabewert

Ein [DateTime](../../datetime/)-Wert, der dem angegebenen verpackten Wert entspricht

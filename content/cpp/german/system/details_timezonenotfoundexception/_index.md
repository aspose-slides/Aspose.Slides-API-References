## Details_TimeZoneNotFoundException Klasse

TimeZoneNotFoundException wird ausgelöst, wenn Zeitzoneninformationen nicht gefunden werden. Erstellen Sie niemals manuell Instanzen dieser Klasse. Verwenden Sie stattdessen die TimeZoneNotFoundException-Klasse. Wickeln Sie die TimeZoneNotFoundException-Klasseninstanzen niemals in [System::SmartPtr](../smartptr/) ein.

```cpp
class Details_TimeZoneNotFoundException : public System::Details_Exception
```

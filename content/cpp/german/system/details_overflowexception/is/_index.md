## Details_OverflowException::Is(const System::TypeInfo\&) const method




```cpp
bool System::Details_OverflowException::Is(const System::TypeInfo &target) const override
```


### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) Struktur, die den Typ beschreibt, gegen den das aktuelle Objekt getestet wird. |

### Rückgabewert

True, wenn das Objekt vom angegebenen Typ oder dessen Unterklasse ist, sonst false.

## Anmerkungen

Prüft, ob das Objekt eine Instanz des von targetType beschriebenen Typs darstellt. Analogie zum C#-Operator 'is'.

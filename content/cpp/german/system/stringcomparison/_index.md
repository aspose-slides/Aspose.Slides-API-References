## StringComparison enum

Definiert den Stil des Zeichenfolgenvergleichs.

```cpp
enum class StringComparison
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| CurrentCulture | 0 | Verwendet die aktuelle Kultur. |
| CurrentCultureIgnoreCase | 1 | Verwendet die aktuelle Kultur, ignoriert jedoch die Groß-/Kleinschreibung. |
| InvariantCulture | 2 | Verwendet die invariant Kultur. |
| InvariantCultureIgnoreCase | 3 | Verwendet die invariant Kultur, ignoriert jedoch die Groß-/Kleinschreibung. |
| Ordinal | 4 | Verwendet ordinale Sortierregeln. |
| OrdinalIgnoreCase | 5 | Verwendet ordinale Sortierregeln, ignoriert jedoch die Groß-/Kleinschreibung. |

## Details_RankException::Is(const System::TypeInfo\&) const method

```cpp
bool System::Details_RankException::Is(const System::TypeInfo &target) const override
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) Struktur, die den zu testenden Typ des aktuellen Objekts beschreibt. |

### Rückgabewert

True, wenn das Objekt vom markierten Typ oder dessen Unterklasse ist, sonst false.

## Bemerkungen

Überprüft, ob das Objekt eine Instanz des durch targetType beschriebenen Typs darstellt. Entspricht dem C#-Operator 'is'.

## ValueType Klasse

Basisklasse für Werttypen mit [Object](../object/) Vererbung, die aus Leistungsgründen abgeschnitten wird. Dieser Typ sollte im Stack zugewiesen und an Funktionen per Wert oder per Referenz übergeben werden. Verwenden Sie niemals die [System::SmartPtr](../smartptr/) Klasse, um Objekte dieses Typs zu verwalten.

```cpp
class ValueType
```

## DateTime::Compare(DateTime, DateTime) Methode

Vergleicht zwei von den angegebenen Instanzen der [DateTime](../)-Klasse vertretene Werte und gibt einen Wert zurück, der die relative Position der Werte auf der Zeitlinie angibt.

```cpp
static constexpr int System::DateTime::Compare(DateTime t1, DateTime t2)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| t1 | [DateTime](../) | Der erste Vergleichswert |
| t2 | [DateTime](../) | Der zweite Vergleichswert |

### Rückgabewert

Ein Wert, der kleiner als 0 ist, wenn **t1** früher als **t2** ist; 0, wenn **t1** gleich **t2** ist; ein Wert, der größer als 0 ist, wenn **t1** später als **t2** ist.

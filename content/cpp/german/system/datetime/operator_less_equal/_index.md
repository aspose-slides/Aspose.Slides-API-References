## DateTime::operator<=(DateTime) const method

Bestimmt, ob das aktuelle Objekt den Datum- und Uhrzeitwert darstellt, der früher liegt als oder dem von dem angegebenen [DateTime](../)-Objekt entspricht.

```cpp
constexpr bool System::DateTime::operator<=(DateTime other) const
```

### Arguments

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| other | [DateTime](../) | Das [DateTime](../) Objekt, mit dem das aktuelle Objekt verglichen wird |

### Rückgabewert

True wenn der vom aktuellen Objekt dargestellte Datum- und Zeitwert früher liegt als oder dem von **other** dargestellten Wert entspricht, sonst - false

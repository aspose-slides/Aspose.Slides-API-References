## DateTime::operator>=(DateTime) const Methode


Bestimmt, ob das aktuelle Objekt den Datums- und Uhrzeitwert darstellt, der später oder derselbe ist wie der Wert, der vom angegebenen [DateTime](../) Objekt repräsentiert wird.

```cpp
constexpr bool System::DateTime::operator>=(DateTime other) const
```


### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| other | [DateTime](../) | Das [DateTime](../) Objekt, mit dem das aktuelle Objekt verglichen wird |

### Rückgabewert

True, wenn der vom aktuellen Objekt dargestellte Datum- und Uhrzeitwert später oder derselbe ist wie der von **other** dargestellte Wert, andernfalls - false

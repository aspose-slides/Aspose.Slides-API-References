## DateTime::operator<(DateTime) const Methode


Bestimmt, ob das aktuelle Objekt den Datum-Uhrzeit-Wert darstellt, der früher ist als der von dem angegebenen [DateTime](../)-Objekt dargestellte Wert.

```cpp
constexpr bool System::DateTime::operator<(DateTime other) const
```


### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| other | [DateTime](../) | Das [DateTime](../)-Objekt, mit dem das aktuelle Objekt verglichen wird |

### Rückgabewert

Wahr, wenn der vom aktuellen Objekt dargestellte Datum-Uhrzeit-Wert früher ist als der von **other** dargestellte Wert, sonst – falsch

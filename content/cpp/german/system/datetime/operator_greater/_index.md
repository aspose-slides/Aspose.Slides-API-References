## DateTime::operator>(DateTime) const Methode


Ermittelt, ob das aktuelle Objekt den Datum- und Uhrzeitwert darstellt, der später ist als der von dem angegebenen [DateTime](../) Objekt.

```cpp
constexpr bool System::DateTime::operator>(DateTime other) const
```


### Parameter

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| other | [DateTime](../) | Das [DateTime](../) Objekt, mit dem das aktuelle Objekt verglichen wird |

### Rückgabewert

True wenn der von dem aktuellen Objekt dargestellte Datum- und Uhrzeitwert später ist als der von **other** dargestellte Wert, sonst - false

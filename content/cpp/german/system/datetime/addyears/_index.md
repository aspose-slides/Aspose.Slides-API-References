## DateTime::AddYears(int) const Methode

Gibt eine neue Instanz der [DateTime](../)-Klasse zurück, die den Datums- und Zeitwert darstellt, der dem vom aktuellen Objekt repräsentierten Wert entspricht, wobei die Jahreskomponente um die angegebene Zahl erhöht wird.

```cpp
DateTime System::DateTime::AddYears(int value) const
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| value | int | Die Anzahl der hinzuzufügenden Jahre; der Wert kann positiv oder negativ sein |

### Rückgabewert

Eine neue Instanz der [DateTime](../)-Klasse, die den Datums- und Zeitwert darstellt, der dem vom aktuellen Objekt repräsentierten Wert entspricht, wobei die Jahreskomponente um **value** erhöht wird.

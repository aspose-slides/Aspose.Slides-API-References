## DateTime::Equals(DateTime) const Methode

Bestimmt, ob die angegebene Instanz der Klasse [DateTime](../) denselben Datums- und Uhrzeitwert wie das aktuelle Objekt darstellt.

```cpp
constexpr bool System::DateTime::Equals(DateTime other) const
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| other | [DateTime](../) | Die Instanz der Klasse [DateTime](../) zum Vergleich mit dem aktuellen Objekt |

### Rückgabewert

True wenn sowohl **other** als auch das aktuelle Objekt denselben Wert darstellen, sonst - false

## DateTime::Equals(DateTime, DateTime) Methode

### Argumente

| Parameter | Typ | Beschreibung |

| t2 | [DateTime](../) | Der zweite Vergleichswert |

### Rückgabewert

True wenn sowohl **t1** als auch **t2** denselben Wert darstellen, sonst - false

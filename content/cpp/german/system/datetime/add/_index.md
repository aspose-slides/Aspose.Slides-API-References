## DateTime::Add(TimeSpan) const Methode

```cpp
DateTime System::DateTime::Add(TimeSpan value) const
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| value | [TimeSpan](../../timespan/) | Das hinzuzufügende Zeitintervall |

### Rückgabewert

Eine neue Instanz der [DateTime](../) Klasse, die einen Datum- und Uhrzeitwert darstellt, der aus der Addition des **tm** Zeitintervalls zu dem von dem aktuellen Objekt dargestellten Datum- und Uhrzeitwert resultiert.

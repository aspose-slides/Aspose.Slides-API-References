## DateTime::FromFileTime(int64_t) Methode

Konvertiert die angegebene Dateizeit in eine Instanz der Klasse [DateTime](../), die denselben Datums- und Zeitwert wie die lokale Zeit darstellt.

```cpp
static DateTime System::DateTime::FromFileTime(int64_t value)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| value | **int64_t** | Dateizeit zum Konvertieren |

### Rückgabewert

Eine neue Instanz der Klasse [DateTime](../), die dasselbe Datum und dieselbe Zeit wie **file_time** darstellt.

## DateTime::IsDaylightSavingTime() const Methode


Bestimmt, ob der von dem aktuellen Objekt dargestellte Datums- und Uhrzeitwert im Zeitraum der Sommerzeit für die aktuelle Zeitzone liegt.

```cpp
bool System::DateTime::IsDaylightSavingTime() const
```


### Rückgabewert

Wahr, wenn der vom aktuellen Objekt angegebene Datums- und Uhrzeitwert im Zeitraum der Sommerzeit liegt, andernfalls - falsch

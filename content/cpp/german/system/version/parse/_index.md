## Version::Parse(const String\&) Methode

Konvertiert die Zeichenkettenrepräsentation einer Versionsnummer in eine äquivalente Instanz der Klasse [Version](../).

```cpp
static Version System::Version::Parse(const String &input)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| input | const [String](../../string/)\& | Eine Zeichenkette, die die Zeichenkettenrepräsentation einer Versionsnummer enthält |

### Rückgabewert

Eine Instanz der Klasse [Version](../), die die Versionsnummer darstellt, die der durch **input** dargestellten entspricht.

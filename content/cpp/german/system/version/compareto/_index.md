## Version::CompareTo(const Version\&) const Methode

Vergleicht die vom aktuellen Objekt und dem angegebenen Objekt dargestellten Versionen.

```cpp
int System::Version::CompareTo(const Version &ver) const
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| ver | const [Version](../)\& | Ein [Version](../) Objekt zum Vergleich mit dem aktuellen Objekt |

### Rückgabewert

- 1, wenn die vom aktuellen Objekt dargestellte Versionsnummer kleiner ist als die von **ver** dargestellte; 0, wenn die Versionsnummern gleich sind; 1, wenn die vom aktuellen Objekt dargestellte Versionsnummer größer ist als die von **ver** dargestellte

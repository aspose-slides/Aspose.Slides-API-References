## UriFormat Enum


Gibt an, wie die URI escaped wird.

```cpp

```

### Werte

| Name | Wert | Beschreibung |

| UriEscaped | 1 | Escaping gemäß RFC-2396. |
| Unescaped | 2 | Kein Escaping. |

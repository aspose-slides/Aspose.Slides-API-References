## GC Klasse

Stellt eine emulierte Garbage Collection dar, die eher wie ein Stub funktioniert und effektiv nichts tut. Es handelt sich um einen statischen Typ ohne Instanzdienste. Sie sollten niemals Instanzen davon auf irgendeine Weise erstellen.

```cpp
class GC : public System::Object
```

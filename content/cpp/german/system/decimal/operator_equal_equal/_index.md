## Decimal::operator==(const Decimal\&) const Methode

```cpp

```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| d | const [Decimal](../)\& | Das [Decimal](../)-Objekt, mit dem das aktuelle Objekt verglichen wird |

### Rückgabewert

True, wenn der von den aktuellen und den angegebenen Objekten dargestellte Wert gleich ist, sonst - false

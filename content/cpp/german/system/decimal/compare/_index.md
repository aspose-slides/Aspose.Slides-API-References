## Decimal::Compare(const Decimal\&, const Decimal\&) Methode

Bestimmt, ob der durch das erste [Decimal](../)-Objekt dargestellte Wert kleiner, gleich oder größer ist als der durch das zweite [Decimal](../)-Objekt dargestellte Wert.

```cpp
static int System::Decimal::Compare(const Decimal &d1, const Decimal &d2)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| d1 | const [Decimal](../)\& | Der erste Vergleichswert |
| d2 | const [Decimal](../)\& | Der zweite Vergleichswert |

### Rückgabewert

- 1, wenn der durch **d1** dargestellte Wert kleiner ist als der durch **d2** dargestellte Wert; 0, wenn die Werte gleich sind; 1, wenn der durch **d1** dargestellte Wert größer ist als der durch **d2** dargestellte Wert

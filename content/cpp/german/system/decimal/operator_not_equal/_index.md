## Decimal::operator!=(const Decimal\&) const method

Bestimmt, ob die von dem aktuellen Objekt und dem angegebenen Objekt dargestellten Werte nicht gleich sind.

```cpp
bool System::Decimal::operator!=(const Decimal &d) const
```

### Arguments

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| d | const [Decimal](../)\& | Das [Decimal](../)-Objekt, mit dem das aktuelle Objekt verglichen wird |

### Return Value

True, wenn der von dem aktuellen und dem angegebenen Objekt dargestellte Wert nicht gleich ist, sonst - false

## Decimal::operator<=(const Decimal\&) const Methode

Bestimmt, ob der von dem aktuellen Objekt dargestellte Wert kleiner oder gleich dem von dem angegebenen Objekt dargestellten Wert ist.

```cpp
bool System::Decimal::operator<=(const Decimal &d) const
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| d | const [Decimal](../)\& | Das [Decimal](../)-Objekt, mit dem das aktuelle Objekt verglichen wird |

### Rückgabewert

True, wenn der von dem aktuellen Objekt dargestellte Wert kleiner oder gleich dem von **d** dargestellten Wert ist, sonst - false

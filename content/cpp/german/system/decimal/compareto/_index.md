## Decimal::CompareTo(const Decimal\&) const Methode


Bestimmt, ob der von dem aktuellen Objekt dargestellte Wert kleiner, gleich oder größer als der von dem angegebenen Objekt dargestellte Wert ist.

```cpp
int System::Decimal::CompareTo(const Decimal &d) const
```


### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| d | const [Decimal](../)\& | Der Vergleichswert |

### Rückgabewert

- 1, wenn der von dem aktuellen Objekt dargestellte Wert kleiner ist als der von **d** dargestellte Wert; 0, wenn die Werte gleich sind; 1, wenn der von dem aktuellen Objekt dargestellte Wert größer ist als der von **d** dargestellte Wert

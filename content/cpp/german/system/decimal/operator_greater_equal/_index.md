## Decimal::operator>=(const Decimal\&) const Methode


Bestimmt, ob der vom aktuellen Objekt dargestellte Wert größer oder gleich dem vom angegebenen Objekt dargestellten Wert ist.

```cpp
bool System::Decimal::operator>=(const Decimal &d) const
```


### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| d | const [Decimal](../)\& | Das [Decimal](../) Objekt, um das aktuelle Objekt zu vergleichen |

### Rückgabewert

True wenn der vom aktuellen Objekt dargestellte Wert größer oder gleich dem von **d** dargestellten Wert ist, sonst - false

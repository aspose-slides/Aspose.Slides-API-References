## DateTimeOffset::Compare(const DateTimeOffset\&, const DateTimeOffset\&) Methode

Vergleicht zwei [DateTimeOffset](../) Objekte.

```cpp
static int System::DateTimeOffset::Compare(const DateTimeOffset &first, const DateTimeOffset &second)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| first | const [DateTimeOffset](../)\& | Erstes Objekt zum Vergleichen. |
| second | const [DateTimeOffset](../)\& | Zweites Objekt zum Vergleichen. |

### Rückgabewert

- 1 - wenn **first** früher als **second** ist, 1 - wenn **first** später als **second** ist, 0 - wenn **first** gleich **second** ist.

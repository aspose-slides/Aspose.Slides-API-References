## DateTimeOffset::operator+(TimeSpan) const Methode

Gibt eine neue Instanz der [DateTimeOffset](../) Klasse zurück, die den Datums- und Uhrzeitwert darstellt, der die Summe des von dem aktuellen Objekt dargestellten Werts und des angegebenen Zeitintervalls ist.

```cpp
DateTimeOffset System::DateTimeOffset::operator+(TimeSpan value) const
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| value | [TimeSpan](../../timespan/) | Das Zeitintervall, das zum von dem aktuellen Objekt dargestellten value hinzugefügt werden soll |

### Rückgabewert

Eine neue Instanz der [DateTimeOffset](../) Klasse, die den Datums- und Uhrzeitwert darstellt, der die Summe des von dem aktuellen Objekt dargestellten Werts und **value** ist.

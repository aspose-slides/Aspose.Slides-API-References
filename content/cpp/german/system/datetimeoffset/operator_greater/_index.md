## DateTimeOffset::operator>(const DateTimeOffset\&) const Methode

Bestimmt, ob das aktuelle Objekt den Datum- und Uhrzeitwert darstellt, der später ist als der von dem angegebenen [DateTimeOffset](../)-Objekt dargestellte Wert.

```cpp
bool System::DateTimeOffset::operator>(const DateTimeOffset &other) const
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| other | const [DateTimeOffset](../)\& | Das [DateTimeOffset](../)-Objekt, mit dem das aktuelle Objekt verglichen wird |

### Rückgabewert

True, wenn der vom aktuellen Objekt dargestellte Datum- und Uhrzeitwert später ist als der von **other** dargestellte Wert, andernfalls - false

## DateTimeOffset::operator<=(const DateTimeOffset\&) const Methode


Bestimmt, ob das aktuelle Objekt den Datum- und Uhrzeitwert darstellt, der früher ist als oder derselbe wie der von dem angegebenen [DateTimeOffset](../) Objekt.

```cpp
bool System::DateTimeOffset::operator<=(const DateTimeOffset &other) const
```


### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| other | const [DateTimeOffset](../)\& | Das [DateTimeOffset](../) Objekt, mit dem das aktuelle Objekt verglichen wird |

### Rückgabewert

True, wenn der von dem aktuellen Objekt dargestellte Datum- und Uhrzeitwert früher ist als oder derselbe wie der von **other** dargestellte Wert, andernfalls - false

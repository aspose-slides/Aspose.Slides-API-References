## TimeSpan::operator-(TimeSpan) const Methode

Gibt eine neue Instanz der Klasse [TimeSpan](../) zurück, die ein Zeitintervall darstellt, das das Ergebnis der Subtraktion des von dem angegebenen Objekt dargestellten Zeitintervalls vom von dem aktuellen Objekt dargestellten Zeitintervall ist.

```cpp
TimeSpan System::TimeSpan::operator-(TimeSpan value) const
```

### Argumente

| Parameter | Type | Description |
| --- | --- | --- |
| value | [TimeSpan](../) | Das [TimeSpan](../)-Objekt, das das zu subtrahierende Zeitintervall darstellt |

### Rückgabewert

Eine neue Instanz der Klasse [TimeSpan](../) wird zurückgegeben, die ein Zeitintervall darstellt, das das Ergebnis der Subtraktion des von dem angegebenen Objekt dargestellten Zeitintervalls vom von dem aktuellen Objekt dargestellten Zeitintervall ist.

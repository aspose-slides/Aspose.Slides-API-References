## TimeSpan::Equals(TimeSpan) const Methode

Bestimmt, ob das vom aktuellen Objekt dargestellte Zeitintervall dem vom angegebenen Objekt dargestellten Zeitintervall entspricht.

```cpp
constexpr bool System::TimeSpan::Equals(TimeSpan value) const
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| value | [TimeSpan](../) | Das [TimeSpan](../)-Objekt, mit dem das aktuelle Objekt verglichen wird |

### Rückgabewert

Wahr, wenn das aktuelle Objekt und das angegebene Objekt dasselbe Zeitintervall darstellen, andernfalls - falsch

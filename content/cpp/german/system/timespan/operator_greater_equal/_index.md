## TimeSpan::operator>=(TimeSpan) const Methode


Bestimmt, ob das vom aktuellen Objekt dargestellte Zeitintervall länger ist als oder gleich dem vom angegebenen Objekt dargestellten Zeitintervall.

```cpp
constexpr bool System::TimeSpan::operator>=(TimeSpan value) const
```


### Arguments

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| value | [TimeSpan](../) | Das [TimeSpan](../)-Objekt, mit dem das aktuelle Objekt verglichen wird |

### Rückgabewert

True, wenn das vom aktuellen Objekt dargestellte Zeitintervall länger ist als oder gleich dem vom **value** dargestellten Zeitintervall, sonst - false

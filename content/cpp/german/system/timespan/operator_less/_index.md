## TimeSpan::operator<(TimeSpan) const Methode

Bestimmt, ob das vom aktuellen Objekt repräsentierte Zeitintervall kürzer ist als das vom angegebenen Objekt repräsentierte Zeitintervall.

```cpp
constexpr bool System::TimeSpan::operator<(TimeSpan value) const
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| value | [TimeSpan](../) | Das [TimeSpan](../)-Objekt, mit dem das aktuelle Objekt verglichen wird |

### Rückgabewert

Wahr, wenn das vom aktuellen Objekt repräsentierte Zeitintervall kürzer ist als das vom **value** repräsentierte Zeitintervall, andernfalls - false

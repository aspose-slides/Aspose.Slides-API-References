```cpp
constexpr bool System::TimeSpan::operator<=(TimeSpan value) const
```


### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| value | [TimeSpan](../) | Das [TimeSpan](../) Objekt, mit dem das aktuelle Objekt verglichen wird |

### Rückgabewert

Wahr, wenn das vom aktuellen Objekt dargestellte Zeitintervall kürzer als oder gleich dem vom **value** dargestellten Zeitintervall ist, andernfalls - falsch

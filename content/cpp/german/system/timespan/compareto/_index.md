## TimeSpan::CompareTo(TimeSpan) const Methode


Vergleicht das aktuelle Objekt mit dem angegebenen Objekt.

```cpp
constexpr int System::TimeSpan::CompareTo(TimeSpan value) const
```


### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| value | [TimeSpan](../) | Das [TimeSpan](../) Objekt, mit dem das aktuelle Objekt verglichen wird |

### Rückgabewert

- 1, wenn das aktuelle Objekt das Intervall repräsentiert, das kürzer ist als **value**; 0, wenn das aktuelle Objekt das Intervall repräsentiert, das gleich **value** ist; 1, wenn das aktuelle Objekt das Intervall repräsentiert, das länger ist als **value**

## Guid::Parse(const String\&) Methode

Konvertiert die angegebene Zeichenkettenrepräsentation einer GUID in ein gleichwertiges [Guid](../)-Objekt.

```cpp
static Guid System::Guid::Parse(const String &input)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| input | const [String](../../string/)\& | Die Zeichenkette, die die Zeichenkettenrepräsentation der GUID enthält |

### Rückgabewert

Ein [Guid](../)-Objekt, das eine GUID darstellt, die der von **input** dargestellten entspricht.

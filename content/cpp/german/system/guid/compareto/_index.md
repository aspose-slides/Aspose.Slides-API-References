## Guid::CompareTo(const Guid\&) const Methode

Führt einen arithmetischen Vergleich der GUIDs durch, die vom aktuellen und angegebenen Objekt repräsentiert werden.

```cpp
int System::Guid::CompareTo(const Guid &g) const
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| g | const [Guid](../)\& | Der Vergleichswert |

### Rückgabewert

- 1 wenn die vom aktuellen Objekt repräsentierte GUID kleiner ist als die vom **g** repräsentierte GUID; 0 wenn die GUIDs gleich sind; 1 wenn die vom aktuellen Objekt repräsentierte GUID größer ist als die vom **g** repräsentierte GUID

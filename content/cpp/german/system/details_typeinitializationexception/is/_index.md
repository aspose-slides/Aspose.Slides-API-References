```cpp
bool System::Details_TypeInitializationException::Is(const System::TypeInfo &target) const override
```


### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) Struktur, die den zu testenden Typ des aktuellen Objekts beschreibt. |

### Rückgabewert

Wahr, wenn das Objekt vom angegebenen Typ oder dessen Unterklasse ist, andernfalls falsch.

## Bemerkungen

Prüft, ob das Objekt eine Instanz des durch targetType beschriebenen Typs darstellt. Entspricht dem C#-Operator 'is'.

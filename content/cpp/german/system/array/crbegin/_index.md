## Array::crbegin() const Methode

Gibt einen Reverse-Iterator auf das erste Element des umgekehrten Containers zurück. Er entspricht dem letzten Element des nicht umgekehrten Containers. Ist der Container leer, ist der zurückgegebene Iterator gleich [crend()](../crend/).

```cpp
const_reverse_iterator System::Array<T>::crbegin() const noexcept
```

### Rückgabewert

Ein Iterator, der auf das letzte const-qualifizierte Element des Containers zeigt.

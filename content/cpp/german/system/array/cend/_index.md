## Array::cend() const Methode


Gibt einen Iterator auf das Element zurück, das dem letzten Element des Containers folgt. Dieses Element dient als Platzhalter; der Versuch, darauf zuzugreifen, führt zu undefiniertem Verhalten.

```cpp
const_iterator System::Array<T>::cend() const noexcept
```


### Rückgabewert

Einen Iterator, der auf das theoretisch const-qualifizierte Element zeigt, das nach dem letzten Element des Containers liegt.

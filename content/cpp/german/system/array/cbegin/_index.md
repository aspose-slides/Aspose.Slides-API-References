## Array::cbegin() const Methode

Gibt einen Iterator auf das erste const-qualifizierte Element des Containers zurück. Wenn der Container leer ist, ist der zurückgegebene Iterator gleich [cend()](../cend/).

```cpp
const_iterator System::Array<T>::cbegin() const noexcept
```

### Rückgabewert

Ein Iterator, der auf das erste const-qualifizierte Element des Containers zeigt.

## Environment::get_Version() Methode

Gibt das [Version](../../version/)-Objekt zurück, das die Information über die Version der Common Language Runtime darstellt. Die von dieser Methode zurückgegebene Versionsnummer ist eher ein Platzhalter und bedeutet nicht, dass sich alle Bibliotheksklassen gemäß der zurückgegebenen Version verhalten.

```cpp
static Version System::Environment::get_Version()
```

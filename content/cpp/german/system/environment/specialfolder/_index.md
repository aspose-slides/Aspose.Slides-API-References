## SpecialFolder Enum

Stellt System-Sonderordner dar.

```cpp
enum class SpecialFolder
```

### Werte

| Name | Value | Description |
| --- | --- | --- |
| Desktop | 0 | Der logische Desktop. |
| Programs | 2 | Der Ordner, der die Programmgruppen des Benutzers enthält. |
| Personal | 5 | Der Ordner, der als gemeinsames Verzeichnis für Dokumente dient. |
| MyDocuments | 5 | Das Verzeichnis \"My Documents\". |
| Favorites | 6 | Der Ordner, der als gemeinsames Verzeichnis für die Lieblingsobjekte des Benutzers dient. |
| Startup | 7 | Der Ordner, der der Startprogrammgruppe des Benutzers entspricht. |
| Recent | 8 | Der Ordner, der die zuletzt verwendeten Dokumente des Benutzers enthält. |
| SendTo | 9 | Der Ordner, der als Container für die Elemente des Menüs \"Senden an\" dient. |
| StartMenu | 11 | Der Ordner, der als Container für die Elemente des Startmenüs dient. |
| MyMusic | 13 | Das Verzeichnis \"My Music\". |
| DesktopDirectory | 16 | Der Ordner, der zum physischen Speichern von Elementen auf dem Desktop verwendet wird. |
| MyComputer | 17 | Das Verzeichnis \"My Computer\". |
| Templates | 21 | Der Ordner, der als gemeinsamer Container für Dokumentvorlagen dient. |
| ApplicationData | 26 | Der Ordner, der verwendet wird, um anwendungsbezogene Daten für den aktuellen Roaming-Benutzer zu speichern. |
| LocalApplicationData | 28 | Der Ordner, der verwendet wird, um anwendungsbezogene Daten für den aktuellen nicht-Roaming-Benutzer zu speichern. |
| InternetCache | 32 | Der Ordner, der als Speicher für temporäre Internetdateien dient. |
| Cookies | 33 | Der Ordner, der als Speicher für Internet-Cookies dient. |
| History | 34 | Der Ordner, der als Speicher für Internet-Verlaufselemente dient. |
| CommonApplicationData | 35 | Der Ordner, der zum Speichern anwendungsbezogener Daten verwendet wird, die allen Benutzern zur Verfügung stehen. |
| System | 37 | Der Systemordner. |
| ProgramFiles | 38 | Der Programmdateien-Ordner. |
| MyPictures | 39 | Der Ordner \"My picutres\". |
| UserProfile | 40 | Der Profilordner des Benutzers. |
| CommonProgramFiles | 43 | Das Verzeichnis, das als Speicher für von Anwendungen gemeinsam genutzte Komponenten dient. |

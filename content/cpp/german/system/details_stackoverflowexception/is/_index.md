## Details_StackOverflowException::Is(const System::TypeInfo\&) const method

```cpp
bool System::Details_StackOverflowException::Is(const System::TypeInfo &target) const override
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) Struktur, die den zu testenden Typ für das aktuelle Objekt beschreibt. |

### Rückgabewert

True, wenn das Objekt vom angegebenen Typ oder einer Unterklasse ist, sonst false.

## Hinweise

Überprüft, ob das Objekt eine Instanz des durch targetType beschriebenen Typs darstellt. Analogie zum C#-Operator 'is'.

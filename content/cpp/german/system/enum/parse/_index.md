## Enum::Parse(const String\&, bool) Methode


Konvertiert die angegebene Zeichenkette in das entsprechende Enum-Element.

```cpp
static E System::Enum<E, Guard>::Parse(const String &str, bool ignoreCase=false)
```


### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| str | const [String](../../string/)\& | [String](../../string/) that is interpreted as containing the name of enum constant |
| ignoreCase | **bool** | Gibt an, ob die Groß-/Kleinschreibung beim Interpretieren der Zeichenkette ignoriert werden soll |

### Rückgabewert

The enumeration constant whose name is specified in **str**

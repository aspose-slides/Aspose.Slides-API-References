## System::operator*(const T\&, const Decimal\&) Funktion

Gibt eine neue Instanz der Klasse [Decimal](../decimal/) zurück, die einen Wert darstellt, der das Ergebnis der Multiplikation des angegebenen Werts und des durch das angegebene [Decimal](../decimal/)-Objekt dargestellten Werts ist.

```cpp
template<typename T,typename _> Decimal System::operator*(const T &x, const Decimal &d)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| x | const T\& | Der erste Multiplikand |
| d | const [Decimal](../decimal/)\& | Das [Decimal](../decimal/)-Objekt, das den zweiten Multiplikanden darstellt |

### Rückgabewert

Eine neue Instanz der Klasse [Decimal](../decimal/), die einen Wert darstellt, der das Ergebnis der Multiplikation von **x** und dem durch **d** dargestellten Wert ist.

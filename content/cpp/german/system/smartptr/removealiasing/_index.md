## SmartPtr::RemoveAliasing() const Methode

Entfernt Aliasing (erstellt durch einen Aliasing-Konstruktor) vom Zeiger und stellt sicher, dass er (falls gemeinsam genutzt) verwaltet oder (falls schwach) verfolgt dasselbe Objekt, auf das er zeigt.

```cpp
SmartPtr_ System::SmartPtr<T>::RemoveAliasing() const
```

### Rückgabewert

Der Zeiger auf dasselbe Objekt, auf das dieser Zeiger zeigt und das die Lebensdauer dieses Objekts überwacht.

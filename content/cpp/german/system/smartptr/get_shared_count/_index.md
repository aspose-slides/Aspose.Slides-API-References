## SmartPtr::get_shared_count() const Methode


Ermittelt die Anzahl der gemeinsam genutzten Zeiger, die auf das referenzierte Objekt verweisen, einschließlich des aktuellen Zeigers. Behauptet, dass der aktuelle Zeiger im gemeinsam genutzten Modus ist.

```cpp
int System::SmartPtr<T>::get_shared_count() const
```


### Rückgabewert

Anzahl der gemeinsam genutzten Zeiger, die auf das referenzierte Objekt verweisen, falls vorhanden. Ist der Zeiger null, wird 0 zurückgegeben.

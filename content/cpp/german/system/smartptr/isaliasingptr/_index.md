## SmartPtr::IsAliasingPtr() const Methode

Überprüft, ob der Zeiger auf ein anderes Objekt als das Besessene zeigt (erstellt durch einen aliasing-Konstruktor).

```cpp
bool System::SmartPtr<T>::IsAliasingPtr() const
```

### Rückgabewert

Wahr, wenn der Zeiger auf unterschiedliche Objekte (gezeigtes und besessenes) verweist, andernfalls falsch.

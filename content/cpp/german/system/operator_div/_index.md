## System::operator/(const T\&, const Decimal\&) Funktion


Gibt eine neue Instanz der [Decimal](../decimal/) Klasse zurück, die einen Wert darstellt, der das Ergebnis der Division des angegebenen Werts und des durch das angegebene [Decimal](../decimal/) Objekt dargestellten Werts ist.

```cpp
template<typename T,typename _> Decimal System::operator/(const T &x, const Decimal &d)
```


### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| x | const T\& | Der zu teilende Wert |
| d | const [Decimal](../decimal/)\& | Das [Decimal](../decimal/) Objekt, das den Divisor darstellt |

### Rückgabewert

Eine neue Instanz der [Decimal](../decimal/) Klasse, die einen Wert darstellt, der das Ergebnis der Division von **x** durch den durch **d** dargestellten Wert ist.

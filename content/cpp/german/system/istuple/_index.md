## System::IsTuple(const SharedPtr\<Object\>\&, int32_t) Funktion

Überprüft, ob das Objekt ein Tupel ist (implementiert das ITuple-Interface). Wird in der Implementierung von Positionsmustern verwendet.

```cpp
bool System::IsTuple(const SharedPtr<Object> &object, int32_t length)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| object | const [SharedPtr](../sharedptr/)\<[Object](../object/)\>\& | Objekt zum Überprüfen. |
| length | **int32_t** | erwartete Tupellänge |

### Rückgabewert

Gibt zurück, ob das Objekt ein Tupel ist und seine Länge der erwarteten entspricht.

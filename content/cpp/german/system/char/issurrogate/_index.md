## Char::IsSurrogate(char_t) Methode

Bestimmt, ob das angegebene Zeichen eine UTF-16 Surrogat-Codeeinheit ist.

```cpp
static bool System::Char::IsSurrogate(char_t c)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| c | char_t | Ein Zeichen |

### Rückgabewert

True, wenn das angegebene Zeichen eine UTF-16 Surrogat-Codeeinheit ist, sonst - false

## Char::IsSurrogate(const String\&, int) Methode

Bestimmt, ob das Zeichen am angegebenen Index im angegebenen String eine UTF-16 Surrogat-Codeeinheit ist.

static bool System::Char::IsSurrogate(const String &s, int index)

| Parameter | Typ | Beschreibung |

| index | int | Der Index des Zeichens im angegebenen String |

### Rückgabewert

## Char::IsPunctuation(const char_t *, int) Methode

Ermittelt, ob das Zeichen am angegebenen Index im angegebenen Zeichenpuffer als Satzzeichen klassifiziert ist.

```cpp
static bool System::Char::IsPunctuation(const char_t *str, int idx)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| str | const char_t * | Zeiger auf den Anfang des Zeichenpuffers |
| idx | int | Ein nullbasierter Index im angegebenen Puffer des zu testenden Zeichens |

### Rückgabewert

True, wenn das Zeichen am angegebenen Index ein Satzzeichen ist, sonst - false

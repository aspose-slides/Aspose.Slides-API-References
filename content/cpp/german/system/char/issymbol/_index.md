## Char::IsSymbol(const char_t *, int) Methode

Bestimmt, ob das Zeichen am angegebenen Index im angegebenen Zeichenpuffer als Symbolzeichen klassifiziert wird.

```cpp
static bool System::Char::IsSymbol(const char_t *str, int idx)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| str | const char_t * | Zeiger auf den Beginn des Zeichenpuffers |
| idx | int | Nullbasierter Index im angegebenen Puffer des zu testenden Zeichens |

### Rückgabewert

True, wenn das Zeichen am angegebenen Index ein Symbolzeichen ist, sonst - false

static bool System::Char::IsSymbol(char_t c)

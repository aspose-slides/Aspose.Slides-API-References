## Char::IsUpper(const String\&, int) Methode

Bestimmt, ob das Zeichen am angegebenen Index in der angegebenen Zeichenkette als Großbuchstabe klassifiziert wird.

```cpp
static bool System::Char::IsUpper(const String &str, int idx)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| str | const [String](../../string/)\& | Die Zeichenkette, die ein Zeichen enthält. |
| idx | int | Ein nullbasierter Index in der angegebenen Zeichenkette zum Testen |

### Rückgabewert

True, wenn das Zeichen am angegebenen Index ein Großbuchstabe ist, andernfalls - false

## Char::IsLetter(const char_t *, int) Methode

Bestimmt, ob das Zeichen am angegebenen Index im angegebenen Zeichenpuffer als Unicode-Buchstabe klassifiziert wird.

```cpp
static bool System::Char::IsLetter(const char_t *str, int idx)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| str | const char_t * | Zeiger auf den Anfang des Zeichenpuffers |
| idx | int | Nullbasierter Index im angegebenen Puffer des zu prüfenden Zeichens |

### Rückgabewert

True wenn das Zeichen am angegebenen Index ein Unicode-Buchstabe ist, sonst - false

## Char::IsLower(const char_t *, int) Methode

Ermittelt, ob das Zeichen am angegebenen Index im angegebenen Zeichenpuffer als Kleinbuchstabe klassifiziert wird.

```cpp
static bool System::Char::IsLower(const char_t *str, int idx)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| str | const char_t * | Zeiger auf den Anfang des Zeichenpuffers |
| idx | int | Ein nullbasierter Index im angegebenen Puffer für das zu testende Zeichen |

### Rückgabewert

Wahr, wenn das Zeichen am angegebenen Index ein Kleinbuchstabe ist, andernfalls - falsch

## Char::IsSeparator(const char_t *, int) Methode


Ermittelt, ob das Zeichen am angegebenen Index im angegebenen Zeichenpuffer als Trennzeichen klassifiziert ist.

```cpp
static bool System::Char::IsSeparator(const char_t *str, int idx)
```


### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| str | const char_t * | Zeiger auf den Beginn des Zeichenpuffers |
| idx | int | Nullbasierter Index im angegebenen Puffer des zu prüfenden Zeichens |

### Rückgabewert

True, wenn das Zeichen am angegebenen Index ein Trennzeichen ist, andernfalls - false

static bool System::Char::IsSeparator(char_t c)

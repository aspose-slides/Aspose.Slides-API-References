## Char::IsWhiteSpace(const char_t *, int) Methode

Bestimmt, ob das Zeichen am angegebenen Index im angegebenen Zeichenpuffer als Leerzeichen klassifiziert wird.

```cpp
static bool System::Char::IsWhiteSpace(const char_t *str, int idx)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| str | const char_t * | Zeiger auf den Beginn des Zeichenpuffers |
| idx | int | Ein nullbasierter Index im angegebenen Puffer des zu testenden Zeichens |

### Rückgabewert

True, wenn das Zeichen am angegebenen Index ein Leerzeichen ist, sonst - false

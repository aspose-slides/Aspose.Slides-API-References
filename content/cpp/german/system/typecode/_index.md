## TypeCode enum

Stellt den Typ eines Objekts dar.

```cpp
enum class TypeCode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Empty | 0 | Eine Null-Referenz. |
| Object | 1 | Ein allgemeiner Typ, der jede Referenz- oder Werttyp darstellt, der nicht explizit durch einen anderen TypeCode repräsentiert wird. |
| DBNull | 2 | Ein Datenbank-Nullwert (Spalte). |
| Boolean | 3 | Ein einfacher Typ, der [Boolean](../boolean/) Werte von true oder false darstellt. |
| Char | 4 | Ein ganzzahliger Typ, der unsigned 16-Bit-Ganzzahlen mit Werten zwischen 0 und 65535 darstellt. |
| SByte | 5 | Ein ganzzahliger Typ, der signed 8-Bit-Ganzzahlen mit Werten zwischen -128 und 127 darstellt. |
| Byte | 6 | Ein ganzzahliger Typ, der unsigned 8-Bit-Ganzzahlen mit Werten zwischen 0 und 255 darstellt. |
| Int16 | 7 | Ein ganzzahliger Typ, der signed 16-Bit-Ganzzahlen mit Werten zwischen -32768 und 32767 darstellt. |
| UInt16 | 8 | Ein ganzzahliger Typ, der unsigned 16-Bit-Ganzzahlen mit Werten zwischen 0 und 65535 darstellt. |
| Int32 | 9 | Ein ganzzahliger Typ, der signed 32-Bit-Ganzzahlen mit Werten zwischen -2147483648 und 2147483647 darstellt. |
| UInt32 | 10 | Ein ganzzahliger Typ, der unsigned 32-Bit-Ganzzahlen mit Werten zwischen 0 und 4294967295 darstellt. |
| Int64 | 11 | Ein ganzzahliger Typ, der signed 64-Bit-Ganzzahlen mit Werten zwischen -9223372036854775808 und 9223372036854775807 darstellt. |
| UInt64 | 12 | Ein ganzzahliger Typ, der unsigned 64-Bit-Ganzzahlen mit Werten zwischen 0 und 18446744073709551615 darstellt. |
| Single | 13 | Ein Gleitkommatyp, der Werte von etwa 1.5 x 10 -45 bis 3.4 x 10 38 mit einer Genauigkeit von 7 Ziffern darstellt. |
| Double | 14 | Ein Gleitkommatyp, der Werte von etwa 5.0 x 10 -324 bis 1.7 x 10 308 mit einer Genauigkeit von 15-16 Ziffern darstellt. |
| Decimal | 15 | Ein einfacher Typ, der Werte von 1.0 x 10 -28 bis etwa 7.9 x 10 28 mit 28-29 signifikanten Ziffern darstellt. |
| DateTime | 16 | Ein Typ, der einen Datum- und Uhrzeitwert darstellt. |
| String | 18 | Ein versiegelter Klassentyp, der Unicode-Zeichenketten darstellt. |

## Details_MemberAccessException::Is(const System::TypeInfo\&) const method

```cpp
bool System::Details_MemberAccessException::Is(const System::TypeInfo &target) const override
```

### Arguments

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) Struktur, die den zu prüfenden Typ des aktuellen Objekts beschreibt. |

### Rückgabewert

True, wenn das Objekt vom markierten Typ oder einem Untertyp ist, sonst false.

## Bemerkungen

Überprüft, ob das Objekt eine Instanz des durch targetType beschriebenen Typs darstellt. Analogie zum C#-Operator 'is'.

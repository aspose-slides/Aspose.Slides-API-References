## SmartPtrMode Enum

[SmartPtr](../smartptr/) Zeigertyp: weak oder shared. Definiert, ob der Zeiger gezählt wird, wenn entschieden wird, ob das Objekt gelöscht werden soll oder nicht.

```cpp
enum class SmartPtrMode : char
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Shared | 0 | Shared-Modus: Zeiger nimmt am Referenzzählen teil. |
| Weak | 1 | Weak-Modus: Zeiger nimmt nicht am Referenzzählen teil. |

## Boolean::Parse(const String\&) Methode


Konvertiert die angegebene Zeichenfolge in einen bool-Wert.

```cpp
static bool System::Boolean::Parse(const String &value)
```


### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| value | const [String](../../string/)\& | Die zu konvertierende Zeichenfolge. |

### Rückgabewert

Die bool-Variable, in die das Ergebnis der Konvertierung geschrieben wird; das Ergebnis ist true, wenn die angegebene Zeichenfolge gleich \"True\" ist, und false, wenn die angegebene Zeichenfolge gleich \"False\" ist.

## Boolean::TryParse(const String\&, bool\&) Methode

Konvertiert die angegebene Zeichenkette in einen Wert des Typs bool.

```cpp
static bool System::Boolean::TryParse(const String &value, bool &result)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| value | const [String](../../string/)\& | Die zu konvertierende Zeichenkette. |
| result | **bool**\& | Die Referenz auf eine bool-Variable, in die das Ergebnis der Konvertierung geschrieben wird; das Ergebnis ist true, wenn die angegebene Zeichenkette gleich \"True\" ist, und false, wenn sie gleich \"False\" ist. |

### Rückgabewert

True, wenn die angegebene Zeichenkette entweder \"True\" oder \"False\" entspricht, sonst false.

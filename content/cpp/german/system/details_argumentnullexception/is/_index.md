## Details_ArgumentNullException::Is(const System::TypeInfo\&) const method

```cpp
bool System::Details_ArgumentNullException::Is(const System::TypeInfo &target) const override
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) Struktur, die den Typ beschreibt, gegen den das aktuelle Objekt getestet wird. |

### Rückgabewert

True, wenn das Objekt vom markierten Typ oder dessen Unterklasse ist, sonst false.

## Bemerkungen

Überprüft, ob das Objekt eine Instanz des durch targetType beschriebenen Typs darstellt. Analog zum C#-Operator 'is'.

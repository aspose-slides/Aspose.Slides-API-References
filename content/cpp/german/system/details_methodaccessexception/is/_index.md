## Details_MethodAccessException::Is(const System::TypeInfo\&) const method

```cpp
bool System::Details_MethodAccessException::Is(const System::TypeInfo &target) const override
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) Struktur, die den Typ beschreibt, gegen den das aktuelle Objekt getestet wird. |

### Rückgabewert

True, wenn das Objekt vom angegebenen Typ oder dessen Untertyp ist, sonst false.

## Hinweise

Überprüfen, ob das Objekt eine Instanz des von targetType beschriebenen Typs darstellt. Analog zum C#-Operator 'is'.

## Object::Is(const TypeInfo\&) const Methode

Prüft, ob das Objekt eine Instanz des durch targetType beschriebenen Typs darstellt. Analogie zum C#-Operator 'is'.

```cpp
virtual bool System::Object::Is(const TypeInfo &targetType) const
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| targetType | const [TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) Struktur, die den Typ beschreibt, gegen den das aktuelle Objekt getestet wird. |

### Rückgabewert

True, wenn das Objekt vom markierten Typ oder einem Untertyp ist, false sonst.

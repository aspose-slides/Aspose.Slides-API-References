## System::Equals(const TA\&, const TB\&) Funktion

Bestimmt die Gleichheit zweier Werte, indem [operator==()](../operator_equal_equal/) darauf angewendet wird.

```cpp
template<typename TA,typename TB> bool System::Equals(const TA &a, const TB &b)
```

### Template-Parameter

| Parameter | Beschreibung |
| --- | --- |
| TA | Der Typ des ersten Vergleichswerts |
| TB | Der Typ des zweiten Vergleichswerts |

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| a | const TA\& | Der erste Vergleichswert |
| b | const TB\& | Der zweite Vergleichswert |

### Rückgabewert

Der boolesche Wert, der von [operator==()](../operator_equal_equal/) auf **a** und **b** angewendet zurückgegeben wird.

## Math::Truncate(const Decimal\&) Methode


Gibt das [Decimal](../../decimal/) Objekt zurück, das einen Wert darstellt, dessen ganzzahliger Teil dem des von dem angegebenen [Decimal](../../decimal/) Objekt dargestellten Wertes entspricht, wobei alle Nachkommastellen verworfen werden.

```cpp
static Decimal System::Math::Truncate(const Decimal &d)
```


### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| d | const [Decimal](../../decimal/)\& | Eine Dezimalzahl |

### Rückgabewert

Ein [Decimal](../../decimal/) Objekt, das einen Dezimalwert darstellt, dessen ganzzahliger Teil dem des angegebenen Wertes entspricht und dessen Nachkommastellen alle 0 sind.

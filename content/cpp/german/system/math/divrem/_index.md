## Math::DivRem(int, int, int\&) Methode


Berechnet den Quotienten zweier 32-Bit-Integer und den Rest.

```cpp
static int System::Math::DivRem(int a, int b, int &result)
```


### Arguments

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| a | int | Der Dividend |
| b | int | Der Divisor |
| result | int\& | Der Ausgabewert, der den Rest enthält, welcher sich aus der Division von **a** durch **b** ergibt |

### Return Value

Der Quotient von **a** und **b**

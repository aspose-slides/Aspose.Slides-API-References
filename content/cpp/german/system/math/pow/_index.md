## Math::Pow(double, double) Methode

Gibt den angegebenen Wert zurück, der auf die angegebene Potenz erhöht wurde.

```cpp
static double System::Math::Pow(double x, double y)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| x | **double** | Der Wert, der auf die angegebene Potenz erhöht werden soll |
| y | **double** | Die Potenz, auf die der angegebene Wert erhöht werden soll |

### Rückgabewert

**x** erhöht auf die Potenz **y**

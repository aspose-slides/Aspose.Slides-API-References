## NullableUtils Klasse

Stellt dar C# [System.Nullable](../nullable/) (ohne Typargumente) statische Klasse. Aufgrund der Unfähigkeit, Klassenvorlagen in C++ zu überladen, kann der ursprüngliche Name nicht verwendet werden. Unterstützt einen Werttyp, dem null zugewiesen werden kann. Diese Klasse kann nicht abgeleitet werden.

```cpp
class NullableUtils
```

## String::IsNull() const Methode

Überprüft, ob die Zeichenkette als null betrachtet wird. [String](../) ist null und nur, wenn sie über [String()](../string/) Konstruktor erstellt, verschoben, kopiert oder aus einer null Zeichenkette zugewiesen wird oder die [reset()](../reset/) Methode aufgerufen wurde.

```cpp
bool System::String::IsNull() const
```

### Rückgabewert

true, wenn die Zeichenkette als null betrachtet wird, false sonst.

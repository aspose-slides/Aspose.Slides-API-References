## String::rend() const Methode


Gibt den Rückwärts-Iterator auf das Zeichen vor dem ersten Zeichen (falls vorhanden) des tatsächlichen Zeichenkettenpuffers zurück.

```cpp
reverse_iterator System::String::rend() const
```


### Rückgabewert

Rückwärts-Iterator auf das Element, das dem ersten Zeichen des tatsächlichen Zeichenkettenpuffers (falls vorhanden) vorausgeht oder auf das Element unmittelbar vor dem zugewiesenen Puffer.

## String::begin() const Methode

Gibt einen Zeiger auf den Beginn des tatsächlichen Zeichenkettenpuffers zurück. Allokiert niemals etwas neu. Garantiert nicht, dass der Puffer null-terminiert ist.

```cpp
const UChar * System::String::begin() const
```

### Rückgabewert

Zeiger auf den tatsächlichen Zeichenkettenpuffer, null-terminiert oder nicht.

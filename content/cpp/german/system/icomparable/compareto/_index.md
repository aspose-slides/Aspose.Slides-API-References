## IComparable::CompareTo(T) Methode

Vergleicht das aktuelle Objekt mit dem angegebenen Objekt.

```cpp
virtual int System::IComparable<T>::CompareTo(T other)=0
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| other | T | Das Objekt, mit dem das aktuelle Objekt verglichen wird |

### Rückgabewert

Eine negative Zahl, wenn das aktuelle Objekt dem angegebenen Objekt in der Sortierreihenfolge vorausgeht; 0, wenn die verglichenen Objekte gleich sind; eine positive Zahl, wenn das aktuelle Objekt dem angegebenen Objekt in der Sortierreihenfolge nachfolgt

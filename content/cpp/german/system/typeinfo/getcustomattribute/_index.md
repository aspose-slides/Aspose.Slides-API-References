## TypeInfo::GetCustomAttribute(const TypeInfo\&) const method

Durchsucht nach dem benutzerdefinierten Attribut, das den angegebenen Typ hat und auf den vom aktuellen Objekt repräsentierten Typ angewendet wurde.

```cpp
ObjectPtr System::TypeInfo::GetCustomAttribute(const TypeInfo &attributeType) const
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| attributeType | const [TypeInfo](../)& | Die konstante Referenz auf das [TypeInfo](../)-Objekt, das den Typ des zu suchenden Attributs darstellt |

### Rückgabewert

Ein Zeiger auf ein Objekt, das das gefundene Attribut darstellt, oder ein Nullzeiger, wenn kein Attribut gefunden wurde, das den Suchkriterien entspricht.

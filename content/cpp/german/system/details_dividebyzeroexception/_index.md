## Details_DivideByZeroException Klasse


DivideByZeroException wird ausgelöst, wenn bei einer arithmetischen Operation eine Division durch 0 versucht wird. Erstellen Sie niemals Instanzen dieser Klasse manuell. Verwenden Sie stattdessen die DivideByZeroException-Klasse. Wickeln Sie niemals die DivideByZeroException-Klasseninstanzen in [System::SmartPtr](../smartptr/) ein.

```cpp
class Details_DivideByZeroException : public System::Details_ArithmeticException
```

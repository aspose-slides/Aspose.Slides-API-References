## EnumParseHelper struct

Hilfsklasse, die die Funktion bereitstellt, eine Zeichenkettenrepräsentation eines Aufzählungs-Konstanten in den entsprechenden Aufzählungswert zu konvertieren.

```cpp
template<class E,class G,class Guard>class EnumParseHelper
```

### Template-Parameter

| Parameter | Beschreibung |
| --- | --- |
| E | Der Typ des enum, mit dem die Methoden der Klasse arbeiten |
| G | Zweites formales Argument von [System::Enum](../enum/) zur Verwendung |
| Guard | Formaler Vorlagenparameter, um den verfügbaren Parsing-Algorithmus auszuwählen |

| --- | --- |
| static E [Parse](./parse/)(const [String](../string/)\&, **bool**) | Konvertiert die angegebene Zeichenkette in den entsprechenden Aufzählungs-Konstantenwert |

## PaddingMode Enum

Definiert, wie Nachrichten behandelt werden, die kürzer sind als der durch die Krypto-Operation benötigte Block.

```cpp
enum class PaddingMode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | 1 | No padding. |
| PKCS7 | 2 | Use PKCS #7 padding string. |
| Zeros | 3 | Use zero bytes. |
| ANSIX923 | 4 | Use ANSIX923 padding string. |
| ISO10126 | 5 | Use ISO10126 padding string. |

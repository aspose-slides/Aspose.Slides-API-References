## XPathNavigator::MoveToNextAttribute() Methode

Wird in einer abgeleiteten Klasse überschrieben, verschiebt [XPathNavigator](../) zum nächsten Attribut.

```cpp
virtual bool System::Xml::XPath::XPathNavigator::MoveToNextAttribute()=0
```

### Rückgabewert

**true** wenn [XPathNavigator](../) erfolgreich zum nächsten Attribut wechselt; **false** wenn keine weiteren Attribute vorhanden sind. Wenn **false**, bleibt die Position von [XPathNavigator](../) unverändert.

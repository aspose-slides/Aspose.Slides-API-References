## XPathNavigator::InsertElementBefore(String, String, String, String) Methode


Erstellt ein neues Geschwisterelement vor dem aktuellen Knoten unter Verwendung des angegebenen Namespace-Präfixes, des lokalen Namens und des angegebenen Namespace-URI mit dem angegebenen Wert.

```cpp
virtual void System::Xml::XPath::XPathNavigator::InsertElementBefore(String prefix, String localName, String namespaceURI, String value)
```


### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| prefix | [String](../../../system/string/) | Das Namespace-Präfix des neuen Kindelements (falls vorhanden). |
| localName | [String](../../../system/string/) | Der lokale Name des neuen Kindelements (falls vorhanden). |
| namespaceURI | [String](../../../system/string/) | Der Namespace-URI des neuen Kindelements (falls vorhanden). [String::Empty](../../../system/string/empty/) und **nullptr** sind äquivalent. |
| value | [String](../../../system/string/) | Der Wert des neuen Kindelements. Wenn [String::Empty](../../../system/string/empty/) oder **nullptr** übergeben werden, wird ein leeres Element erstellt. |

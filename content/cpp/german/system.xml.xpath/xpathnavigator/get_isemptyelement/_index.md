## XPathNavigator::get_IsEmptyElement() Methode

Wenn in einer abgeleiteten Klasse überschrieben, gibt einen Wert zurück, der anzeigt, ob der aktuelle Knoten ein leeres Element ohne End-Tag ist.

```cpp
virtual bool System::Xml::XPath::XPathNavigator::get_IsEmptyElement()=0
```

### Rückgabewert

**true** wenn der aktuelle Knoten ein leeres Element ist; andernfalls **false**.

## XPathNavigator::get_HasAttributes() Methode


Gibt einen Wert zurück, der anzeigt, ob der aktuelle Knoten Attribute besitzt.

```cpp
virtual bool System::Xml::XPath::XPathNavigator::get_HasAttributes()
```


### Rückgabewert

**true** wenn der aktuelle Knoten Attribute hat; gibt **false** zurück, wenn der aktuelle Knoten keine Attribute hat oder wenn der [XPathNavigator](../) nicht auf einem Elementknoten positioniert ist.

## XPathNavigator::get_NamespaceURI() Methode


Wenn in einer abgeleiteten Klasse überschrieben, wird die Namespace-URI des aktuellen Knotens zurückgegeben.

```cpp
virtual String System::Xml::XPath::XPathNavigator::get_NamespaceURI()=0
```


### Rückgabewert

Ein [String](../../../system/string/) der die Namespace-URI des aktuellen Knotens enthält, oder [String::Empty](../../../system/string/empty/) wenn der aktuelle Knoten keine Namespace-URI hat.

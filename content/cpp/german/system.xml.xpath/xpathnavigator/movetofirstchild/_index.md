## XPathNavigator::MoveToFirstChild() Methode

Wenn sie in einer abgeleiteten Klasse überschrieben wird, verschiebt sie das [XPathNavigator](../) zum ersten Kindknoten des aktuellen Knotens.

```cpp
virtual bool System::Xml::XPath::XPathNavigator::MoveToFirstChild()=0
```

### Rückgabewert

**true**, wenn das [XPathNavigator](../) erfolgreich zum ersten Kindknoten des aktuellen Knotens verschoben wird; andernfalls **false**. Wenn **false**, ist die Position des [XPathNavigator](../) unverändert.

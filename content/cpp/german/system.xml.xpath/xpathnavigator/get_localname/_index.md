## XPathNavigator::get_LocalName() Methode

Wenn in einer abgeleiteten Klasse überschrieben, gibt den [XPathNavigator::get_Name](../get_name/) des aktuellen Knotens ohne Namenspräfix zurück.

```cpp
virtual String System::Xml::XPath::XPathNavigator::get_LocalName()=0
```

### Rückgabewert

Ein [String](../../../system/string/), der den lokalen Namen des aktuellen Knotens enthält, oder [String::Empty](../../../system/string/empty/) wenn der aktuelle Knoten keinen Namen hat (zum Beispiel Text- oder Kommentarknoten).

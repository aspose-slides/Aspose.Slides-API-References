## XmlCaseOrder Enum

Gibt die Sortierreihenfolge für Groß- und Kleinbuchstaben an.

```cpp
enum class XmlCaseOrder
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | 0 | Ignoriert die Groß- und Kleinschreibung. |
| UpperFirst | 1 | Großbuchstaben werden vor Kleinbuchstaben sortiert. |
| LowerFirst | 2 | Kleinbuchstaben werden vor Großbuchstaben sortiert. |

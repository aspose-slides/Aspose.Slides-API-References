## HttpContext Klasse

Dummy-Klasse, die es ermöglicht, übersetzten Code mit [HttpContext](./)-Referenzen zu verknüpfen, aber ihn nicht auszuführen. Enthält keine korrekt implementierten Mitglieder. Objekte dieser Klasse sollten nur mit der [System::MakeObject()](../../system/makeobject/)-Funktion alloziert werden. Erstellen Sie niemals eine Instanz dieses Typs auf dem Stack oder mit dem operator new, da dies zu Laufzeitfehlern und/oder Assertion-Fehlern führt. Schließen Sie diese Klasse immer in einen [System::SmartPtr](../../system/smartptr/)-Zeiger ein und verwenden Sie diesen Zeiger, um ihn Funktionen als Argument zu übergeben.

```cpp
class HttpContext : public System::Object
```

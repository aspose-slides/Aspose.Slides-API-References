## HttpResponse Klasse

Dummy-Klasse, die es ermöglicht, übersetzten Code mit [HttpResponse](./)-Referenzen zu verknüpfen, aber ihn nicht auszuführen. Enthält keine korrekt implementierten Member. Objekte dieser Klasse sollten nur mit der Funktion [System::MakeObject()](../../system/makeobject/) alloziert werden. Erstellen Sie niemals eine Instanz dieses Typs auf dem Stack oder mit dem Operator new, da dies zu Laufzeitfehlern und/oder Assertion-Fehlern führen kann. Umwickeln Sie diese Klasse immer in einen [System::SmartPtr](../../system/smartptr/)-Zeiger und verwenden Sie diesen Zeiger, um ihn als Argument an Funktionen zu übergeben.

```cpp
class HttpResponse : public System::Object
```

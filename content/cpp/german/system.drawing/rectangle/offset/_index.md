## Rectangle::Offset(const Point\&) Methode


Verschiebt die Position des Rechtecks, das durch das aktuelle Objekt dargestellt wird, um die angegebenen Beträge.

```cpp
void System::Drawing::Rectangle::Offset(const Point &pos)
```


### Arguments

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| pos | const [Point](../../point/)\& | Das [Point](../../point/)-Objekt, das den Betrag angibt, um den das Rechteck verschoben wird; der X-Koordinatenwert des Punktes gibt den Betrag an, der zur X-Koordinate der oberen linken Ecke des Rechtecks hinzugefügt werden soll, und der Y-Koordinatenwert des Punktes gibt den Betrag an, der zur Y-Koordinate der oberen linken Ecke des Rechtecks hinzugefügt werden soll |

## Rectangle::Offset(int, int) Methode


Verschiebt die Position des Rechtecks, das durch das aktuelle Objekt dargestellt wird, um die angegebenen Beträge.

```cpp
void System::Drawing::Rectangle::Offset(int x, int y)
```


### Arguments

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| x | int | Der Wert, der zur aktuellen X-Koordinate des Rechtecks hinzugefügt wird |
| y | int | Der Wert, der zur aktuellen Y-Koordinate des Rechtecks hinzugefügt wird |

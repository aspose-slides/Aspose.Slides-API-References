## Brush Klasse

Eine Basisklasse für Klassen, die Füllungen darstellen, die zum Füllen der Innenbereiche grafischer Formen verwendet werden. Objekte dieser Klasse sollten nur über die [System::MakeObject()](../../system/makeobject/)-Funktion allokiert werden. Erzeugen Sie niemals eine Instanz dieses Typs auf dem Stack oder mit dem operator new, da dies zu Laufzeitfehlern und/oder Assertionsfehlern führt. Wickeln Sie diese Klasse immer in einen [System::SmartPtr](../../system/smartptr/)-Zeiger ein und verwenden Sie diesen Zeiger, um ihn Funktionen als Argument zu übergeben.

```cpp
class Brush : public System::ICloneable,
              public System::IDisposable
```

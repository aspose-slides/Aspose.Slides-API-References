## RotateFlipType Enum

Specifies the type of rotation and/or flipping operation.

```cpp
enum class RotateFlipType
```

### Werte

| Name | Value | Description |
| --- | --- | --- |
| RotateNoneFlipNone | 0 | Keine Drehung und kein Spiegeln. |
| Rotate90FlipNone | 1 | Um 90 Grad im Uhrzeigersinn drehen und nicht spiegeln. |
| Rotate180FlipNone | 2 | Um 180 Grad im Uhrzeigersinn drehen und nicht spiegeln. |
| Rotate270FlipNone | 3 | Um 270 Grad im Uhrzeigersinn drehen und nicht spiegeln. |
| RotateNoneFlipX | 4 | Horizontales Spiegeln und keine Drehung. |
| Rotate90FlipX | 5 | Um 90 Grad im Uhrzeigersinn drehen und horizontal spiegeln. |
| Rotate180FlipX | 6 | Um 180 Grad im Uhrzeigersinn drehen und horizontal spiegeln. |
| Rotate270FlipX | 7 | Um 270 Grad im Uhrzeigersinn drehen und horizontal spiegeln. |
| RotateNoneFlipY | n/a | Vertikales Spiegeln und keine Drehung. |
| Rotate90FlipY | n/a | Um 90 Grad im Uhrzeigersinn drehen und vertikal spiegeln. |
| Rotate180FlipY | n/a | Um 180 Grad im Uhrzeigersinn drehen und vertikal spiegeln. |
| Rotate270FlipY | n/a | Um 270 Grad im Uhrzeigersinn drehen und vertikal spiegeln. |
| RotateNoneFlipXY | n/a | Horizontales und vertikales Spiegeln und keine Drehung. |
| Rotate90FlipXY | n/a | Um 90 Grad im Uhrzeigersinn drehen und horizontal sowie vertikal spiegeln. |
| Rotate180FlipXY | n/a | Um 180 Grad im Uhrzeigersinn drehen und horizontal sowie vertikal spiegeln. |
| Rotate270FlipXY | n/a | Um 270 Grad im Uhrzeigersinn drehen und horizontal sowie vertikal spiegeln. |

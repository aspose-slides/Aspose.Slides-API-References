## RectangleF::Intersect(const RectangleF\&) Methode

Ersetzt das vom aktuellen Objekt dargestellte Rechteck durch das Rechteck, das sich aus dessen Schnittmenge mit dem vom angegebenen Objekt dargestellten Rechteck ergibt.

```cpp
void System::Drawing::RectangleF::Intersect(const RectangleF &rect)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| rect | const [RectangleF](../)\& | Das [RectangleF](../) Objekt, das das Rechteck darstellt, das mit dem vom aktuellen Objekt dargestellten Rechteck zu schneiden ist |

## RectangleF::Intersect(const RectangleF\&, const RectangleF\&) Methode

Gibt ein Rechteck zurück, das das Ergebnis der Schnittmenge der angegebenen Rechtecke ist.

```cpp
static RectangleF System::Drawing::RectangleF::Intersect(const RectangleF &a, const RectangleF &b)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| a | const [RectangleF](../)\& | Das erste Rechteck zum Schneiden |
| b | const [RectangleF](../)\& | Das zweite Rechteck zum Schneiden |

### Rückgabewert

Das Ergebnis der Schnittmenge von **a** mit **b**

## Point::Offset(int, int) Methode


Verschiebt den von dem aktuellen Objekt dargestellten X- und Y-Koordinatenwert um die angegebenen Werte.

```cpp
void System::Drawing::Point::Offset(int dx, int dy)
```


### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| dx | int | Der Wert, um den der X-Koordinatenwert verschoben wird |
| dy | int | Der Wert, um den der Y-Koordinatenwert verschoben wird |

## Point::Offset(Point) Methode


Verschiebt die vom aktuellen Objekt dargestellten X- und Y-Koordinaten um die Werte der X- und Y-Koordinaten, die vom angegebenen [Point](../)-Objekt entsprechend dargestellt werden.

```cpp
void System::Drawing::Point::Offset(Point point)
```


### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| point | [Point](../) | Das [Point](../)-Objekt, das die Werte zum Verschieben der X- und Y-Koordinaten angibt |

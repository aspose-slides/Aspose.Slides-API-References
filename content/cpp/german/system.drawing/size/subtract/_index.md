## Size::Subtract(const Size\&, const Size\&) Methode

Gibt ein neues [Size](../) Objekt zurück, das das Ergebnis der Subtraktion von **size2** von **size1** ist, d. h. dessen Breitenwert das Ergebnis der Subtraktion des Breitenwerts von **size2** vom Breitenwert von **size1** und dessen Höhenwert das Ergebnis der Subtraktion des Höhenwerts von **size2** vom Höhenwert von **size1** ist.

```cpp
static Size System::Drawing::Size::Subtract(const Size &size1, const Size &size2)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| size1 | const [Size](../)\& | Das [Size](../) Objekt, von dem subtrahiert wird |
| size2 | const [Size](../)\& | Das [Size](../) Objekt, das subtrahiert wird |

### Rückgabewert

Das [Size](../) Objekt, das das Ergebnis der Subtraktion von **size2** von **size1** ist.

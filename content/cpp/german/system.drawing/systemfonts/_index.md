## SystemFonts Klasse

Bietet eine Menge vordefinierter [Font](../font/)-Objekte, die Schriftarten repräsentieren, die zum Darstellen von Text in [Windows](../../system.windows/) Anzeigeelementen verwendet werden. Dies ist ein statischer Typ ohne Instanzdienste. Sie sollten niemals Instanzen davon auf irgendeine Weise erstellen.

```cpp
class SystemFonts : public System::Object
```

## StringFormatFlags Aufzählung

Gibt die Anzeige- und Layoutinformationen für die Zeichenketten an.

```cpp
enum class StringFormatFlags
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| DirectionRightToLeft | 1 | Textrichtung von rechts nach links. |
| DirectionVertical | 2 | [Text](../../system.drawing.text/) ist vertikal ausgerichtet. |
| FitBlackBox | 4 | Teile von Zeichen können über den Layout-Box hinausragen. |
| DisplayFormatControl | 32 | Steuerzeichen werden angezeigt. |
| NoFontFallback | 1024 | [Font](../font/) Fallback ist deaktiviert. |
| MeasureTrailingSpaces | 2048 | Fügt das nachfolgende Leerzeichen am Ende jeder Zeile ein. |
| NoWrap | 4096 | [Text](../../system.drawing.text/) Zeilenumbruch ist deaktiviert. |
| LineLimit | 8192 | Ganze Zeilen werden im Formatierungsrechteck angeordnet. |
| NoClip | 16384 | Überhängende Teile von Glyphen dürfen angezeigt werden. |

## SystemColors Klasse


Klasse, die einen Satz vorgefertigter [Color](../color/)-Objekte bereitstellt, die Farben von [Windows](../../system.windows/) Anzeigeelementen darstellen. Dies ist ein statischer Typ ohne Instanzdienste. Sie sollten zu keinem Zeitpunkt Instanzen davon erstellen.

```cpp
class SystemColors
```

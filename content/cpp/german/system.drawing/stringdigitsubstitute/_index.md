## StringDigitSubstitute enum

Gibt an, wie Ziffern in einem String gemäß dem Gebietsschema oder der Sprache ersetzt werden.

```cpp
enum class StringDigitSubstitute
```

### Values

| Name | Wert | Beschreibung |
| --- | --- | --- |
| User | 0 | Benutzerdefinierte Ersetzung. |
| None | 1 | Ersetzung ist deaktiviert. |
| National | 2 | Ersetzt Ziffern, die der offiziellen Landessprache des Gebietsschemas des Benutzers entsprechen. |
| Traditional | 3 | Ersetzt Ziffern, die dem nativen Schriftsystem oder der Sprache des Benutzers entsprechen, die möglicherweise nicht mit der offiziellen Landessprache des Gebietsschemas des Benutzers übereinstimmen. |

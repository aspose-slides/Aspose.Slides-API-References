## StringTrimming Enum

Gibt an, wie Zeichen aus einem String getrimmt werden sollen, der nicht in die Layoutform passt.

```cpp
enum class StringTrimming
```

### Werte

| Name | Value | Description |
| --- | --- | --- |
| None | 0 | Kein Trimmen. |
| Character | 1 | Zum nächsten Zeichen trimmen. |
| Word | 2 | Zum nächsten Wort trimmen. |
| EllipsisCharacter | 3 | Zum nächsten Zeichen trimmen und am Ende des Strings ein Auslassungszeichen einfügen. |
| EllipsisWord | 4 | Zum nächsten Wort trimmen und am Ende des Strings ein Auslassungszeichen einfügen. |
| EllipsisPath | 5 | Der Mittelteil wird aus den getrimmten Zeilen entfernt und durch ein Auslassungszeichen ersetzt. So viel wie möglich vom letzten schrägstrichgetrennten Segment der Zeile beibehalten. |

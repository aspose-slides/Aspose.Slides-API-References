## AlphaFloor Klasse

Stellt einen Alpha-Floor-Effekt dar. Alpha-(Opazitäts-)Werte unter 100 % werden auf Null gesetzt. Mit anderen Worten wird alles, das teilweise transparent ist, vollständig transparent.

```cpp
class AlphaFloor : public Aspose::Slides::Effects::ImageTransformOperation,
                   public Aspose::Slides::Effects::IAlphaFloor,
                   public Aspose::Slides::Effects::IVisualEffect
```

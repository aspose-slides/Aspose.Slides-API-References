## IAlphaCeilingEffectiveData Klasse

Unveränderliches Objekt, das einen Alpha-Ceiling-Effekt darstellt. Alpha-(Opazitäts-)Werte größer als null werden auf 100 % geändert. Mit anderen Worten wird alles, das teilweise undurchsichtig ist, vollständig undurchsichtig.

```cpp
class IAlphaCeilingEffectiveData : public virtual Aspose::Slides::Effects::IEffectEffectiveData
```

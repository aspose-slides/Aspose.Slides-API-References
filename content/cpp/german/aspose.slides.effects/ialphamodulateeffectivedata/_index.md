## IAlphaModulateEffectiveData Klasse


Unveränderliches Objekt, das einen Alpha-Modulate-Effekt darstellt. Effekt-Alpha-(Deckkraft-)Werte werden mit einem festen Prozentsatz multipliziert. Der Effekt-Container gibt einen Effekt an, der Alpha-Werte zur Modulation enthält.

```cpp
class IAlphaModulateEffectiveData : public virtual Aspose::Slides::Effects::IEffectEffectiveData
```

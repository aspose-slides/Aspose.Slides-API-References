## IAlphaModulate Klasse


Stellt einen Alpha Modulate-Effekt dar. Effekt-Alpha-(Opazitäts-)Werte werden mit einem festen Prozentsatz multipliziert. Der Effekt-Container gibt einen Effekt an, der Alpha-Werte zum Modulieren enthält.

```cpp
class IAlphaModulate : public virtual Aspose::Slides::Effects::IImageTransformOperation,
                       public Aspose::Slides::IAccessiblePVIObject<System::SharedPtr<Aspose::Slides::Effects::IAlphaModulateEffectiveData>>
```

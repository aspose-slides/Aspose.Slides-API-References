## Directory::Move(const String&, const String&) Methode

Verschiebt das angegebene Element an den neuen Speicherort. Wenn das zu verschiebende Element ein Verzeichnis ist, wird es zusammen mit seinem gesamten Inhalt verschoben.

```cpp
static void System::IO::Directory::Move(const String &sourceDirName, const String &destDirName)
```

| destDirName | const [String](../../../system/string/)\& | Der neue Speicherort von **sourceDirName** |

## FileAttributes enum

Stellt Attribute eines Verzeichnisses oder einer Datei dar.

```cpp
enum class FileAttributes
```

### Values

| Name | Value | Description |
| --- | --- | --- |
| Archive | 32 | Das Dateisystemobjekt ist ein Kandidat für das Entfernen oder die Sicherung. |
| Compressed | 2048 | Das Dateisystemobjekt ist komprimiert. |
| Device | 64 | Nicht verwendet. |
| Directory | 16 | Das Dateisystemobjekt ist ein Verzeichnis. |
| Encrypted | 16384 | Das Dateisystemobjekt ist verschlüsselt. |
| Hidden | 2 | Das Dateisystemobjekt ist verborgen. |
| IntegrityStream | 32768 | Das Dateisystemobjekt enthält Unterstützung für Datenintegrität. |
| Normal | 128 | Das Dateisystemobjekt ist eine Datei, die keine besonderen Attribute hat. |
| NotContentIndexed | 8192 | Das Dateisystemobjekt wird vom Inhaltsindizierungsdienst nicht indiziert. |
| NoScrubData | 131072 | Das Dateisystemobjekt ist nicht im Datenintegritätsscan enthalten. |
| Offline | 4096 | Das Dateisystemobjekt ist offline und seine Daten sind nicht sofort verfügbar. |
| ReadOnly | 1 | Das Dateisystemobjekt ist schreibgeschützt. |
| ReparsePoint | 1024 | Das Dateisystemobjekt ist eine Datei, die einen reparse point enthält. |
| SparseFile | 512 | Das Dateisystemobjekt ist eine Sparse-Datei. |
| System | 4 | Das Dateisystemobjekt ist eine Systemdatei. |
| Temporary | 256 | Das Dateisystemobjekt ist eine temporäre Datei. |
| Invalid | 0 | Ungültiger Wert. |

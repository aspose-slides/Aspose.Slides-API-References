## FileInfo::OpenText() Methode

Öffnet die bestehende Datei an dem Ort, der durch den Pfad angegeben ist, der vom aktuellen Objekt repräsentiert wird, zum Lesen von Text unter Verwendung der UTF-8-Codierung ohne Freigabe.

```cpp
StreamReaderPtr System::IO::FileInfo::OpenText()
```

### Rückgabewert

Ein shared pointer auf ein [StreamWriter](../../streamwriter/) Objekt, das mit der geöffneten Datei verknüpft ist.

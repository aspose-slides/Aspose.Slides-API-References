## FileInfo::CreateText() Methode

Erstellt eine Datei an dem durch den Pfad des aktuellen Objekts angegebenen Speicherort und öffnet sie zum Schreiben von Text im UTF-8-Encoding ohne Freigabe.

```cpp
StreamWriterPtr System::IO::FileInfo::CreateText()
```

### Rückgabewert

Ein Shared Pointer zu einem [StreamWriter](../../streamwriter/)-Objekt, das mit der erstellten Datei verknüpft ist

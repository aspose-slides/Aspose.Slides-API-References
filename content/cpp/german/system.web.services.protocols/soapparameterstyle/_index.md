## SoapParameterStyle enum

Enumeriert die Parameterformate in einer SOAP-Nachricht.

```cpp
enum class SoapParameterStyle
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Default | 0 | Wenn '[SoapDocumentServiceAttribute](../soapdocumentserviceattribute/)' nicht auf die Klasse angewendet wird, ist der Standardwert 'Wrapped'. |
| Bare | 1 | Die Parameter werden in XML-Elementen platziert, die dem 'Body'-Element folgen. |
| Wrapped | 2 | Die Parameter werden in einem einzelnen XML-Element gekapselt, das dem 'Body'-Element folgt. |

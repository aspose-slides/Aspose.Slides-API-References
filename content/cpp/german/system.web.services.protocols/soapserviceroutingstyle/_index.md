## SoapServiceRoutingStyle enum

Enumeriert Optionen, wie eine SOAP-Nachricht zum XML [Web](../../system.web/)-Dienst geroutet wird.

```cpp
enum class SoapServiceRoutingStyle
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| SoapAction | 0 | Die SOAP-Nachricht wird basierend auf dem Attribut 'SOAPAction' geroutet. |
| RequestElement | 1 | Die SOAP-Nachricht wird basierend auf dem ersten Kind-Element nach dem 'Body'-Element geroutet. |

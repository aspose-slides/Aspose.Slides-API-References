Enumeriert die SOAP-Header-Richtungen.


### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| In | 1 | Der SOAP-Headerempfänger ist der XML [Web](../../system.web/) Dienst. |
| InOut | 3 | Die SOAP-Headerempfänger sind sowohl der XML [Web](../../system.web/) Dienst als auch der XML [Web](../../system.web/) Dienstclient. |

| Fault | 4 | Der SOAP-Header wird an den XML [Web](../../system.web/) Dienstclient gesendet, wenn eine Ausnahme durch die XML [Web](../../system.web/) Servicemethode ausgelöst wird. |

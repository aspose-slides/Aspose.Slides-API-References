## PicturesCompression enum


Stellt die Bildkomprimierungsstufe dar

```cpp
enum class PicturesCompression
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Dpi330 | 0 | Gute Qualität für hochauflösende (HD) Anzeigen |
| Dpi220 | 1 | Ausgezeichnete Qualität auf den meisten Druckern und Bildschirmen |
| Dpi150 | 2 | Gut für Webseiten und Projektoren |
| Dpi96 | 3 | Minimiert die Dokumentgröße zum Teilen |
| Dpi72 | 4 | Standard-Komprimierungsstufe |
| DocumentResolution | 5 | Verwendet die Dokumentauflösung – das Bild wird nicht komprimiert und im Dokument unverändert verwendet |

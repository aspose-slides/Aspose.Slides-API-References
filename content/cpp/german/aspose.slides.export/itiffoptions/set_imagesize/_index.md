## ITiffOptions::set_ImageSize(System::Drawing::Size) Methode


Gibt die Größe eines erzeugten TIFF-Bildes an. Der Standardwert ist 0x0, was bedeutet, dass die Größen der erzeugten Bilder basierend auf dem Wert der Foliengröße der Präsentation berechnet werden. Schreiben Sie [System::Drawing::Size](../../../system.drawing/size/).

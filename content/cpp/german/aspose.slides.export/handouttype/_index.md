## HandoutType Enum

Gibt an, wie viele Folien und in welcher Reihenfolge auf der Seite platziert werden.

```cpp
enum class HandoutType
```

### Werte

| Name | Wert | Beschreibung |

| Handouts1 | 0 | Eine Folie pro Seite. |
| Handouts2 | 1 | Zwei Folien pro Seite. |
| Handouts3 | 2 | Drei Folien pro Seite. |
| Handouts4Horizontal | 3 | Vier Folien pro Seite in einer horizontalen Reihenfolge. |
| Handouts4Vertical | 4 | Vier Folien pro Seite in einer vertikalen Reihenfolge. |
| Handouts6Horizontal | 5 | Sechs Folien pro Seite in einer horizontalen Reihenfolge. |
| Handouts6Vertical | 6 | Sechs Folien pro Seite in einer vertikalen Reihenfolge. |
| Handouts9Horizontal | 7 | Neun Folien pro Seite in einer horizontalen Reihenfolge. |
| Handouts9Vertical | 8 | Neun Folien pro Seite in einer vertikalen Reihenfolge. |

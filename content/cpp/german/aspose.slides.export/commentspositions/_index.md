## CommentsPositions Enum

Gibt die Regel an, Kommentare in das exportierte Dokument zu rendern.

```cpp
enum class CommentsPositions
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | 0 | Gibt an, dass Kommentare überhaupt nicht angezeigt werden sollen. |
| Bottom | 1 | Gibt an, dass Kommentare am unteren Rand der Seite angezeigt werden sollen. |
| Right | 2 | Gibt an, dass Kommentare rechts von der Seite angezeigt werden sollen. |

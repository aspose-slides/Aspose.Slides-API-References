## IHtmlOptions::set_JpegQuality(uint8_t) Methode


Setzt einen Wert, der die Qualität der JPEG-Bilder im PDF-Dokument bestimmt. Schreiben **uint8_t**.

```cpp
virtual void Aspose::Slides::Export::IHtmlOptions::set_JpegQuality(uint8_t value)=0
```

## Hinweise


Wirkt nur, wenn ein Dokument JPEG-Bilder enthält.

Verwenden Sie diese Eigenschaft, um die Qualität der Bilder in einem Dokument beim Speichern im PDF-Format zu erhalten oder festzulegen. Der Wert kann von 0 bis 100 variieren, wobei 0 die schlechteste Qualität aber maximale Kompression bedeutet und 100 die beste Qualität aber minimale Kompression.

Der Standardwert ist **95**.

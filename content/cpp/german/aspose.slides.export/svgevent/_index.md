## SvgEvent Enum

Stellt Optionen für SVG-Formen dar.

```cpp
enum class SvgEvent
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| OnFocusIn | 0 | Tritt auf, wenn ein Element den Fokus erhält, etwa wenn ein Text ausgewählt wird. |
| OnFocusOut | 1 | Tritt auf, wenn ein Element den Fokus verliert, etwa wenn ein Text abgewählt wird. |
| OnActivate | 2 | Tritt auf, wenn ein Element aktiviert wird, zum Beispiel durch einen Mausklick oder einen Tastendruck. |
| OnClick | 3 | Tritt auf, wenn die Schaltfläche des Zeigegeräts über einem Element geklickt wird. |
| OnMouseDown | 4 | Tritt auf, wenn die Schaltfläche des Zeigegeräts über einem Element gedrückt wird. |
| OnMouseUp | 5 | Tritt auf, wenn die Schaltfläche des Zeigegeräts über einem Element losgelassen wird. |
| OnMouseOver | 6 | Tritt auf, wenn das Zeigegerät über ein Element bewegt wird. |
| OnMouseMove | 7 | Tritt auf, wenn das Zeigegerät bewegt wird, während es über einem Element ist. |
| OnMouseOut | 8 | Tritt auf, wenn das Zeigegerät von einem Element wegbewegt wird. |
| OnLoad | 9 | Tritt auf, wenn der User-Agent das Element und seine Nachkommen sowie alle referenzierten Ressourcen, die für die Darstellung erforderlich sind, vollständig geparst hat. |
| OnUnload | 10 | Tritt auf, wenn die [DOM](../../aspose.slides.dom/)-Implementierung ein Dokument aus einem Fenster oder Frame entfernt. Gilt nur für äußerste svg-Elemente. |
| OnAbort | 11 | Tritt auf, wenn das Laden der Seite gestoppt wird, bevor ein Element vollständig laden durfte. |
| OnError | 12 | Tritt auf, wenn ein Element nicht korrekt lädt oder ein Fehler während der Skriptausführung auftritt. |
| OnResize | 13 | Tritt auf, wenn die Ansicht eines Dokuments skaliert wird. Gilt nur für äußerste svg-Elemente. |
| OnScroll | 14 | Tritt auf, wenn die Ansicht eines Dokuments entlang der X-, Y- oder beider Achsen verschoben wird. Gilt nur für äußerste svg-Elemente. |
| OnZoom | 15 | Tritt auf, wenn die Zoomstufe einer Dokumentenansicht geändert wird. Gilt nur für äußerste svg-Elemente. |
| OnBegin | 16 | Tritt auf, wenn ein Animationselement beginnt. |
| OnEnd | 17 | Tritt auf, wenn ein Animationselement endet. |
| OnRepeat | 18 | Tritt auf, wenn ein Animationselement wiederholt wird. |

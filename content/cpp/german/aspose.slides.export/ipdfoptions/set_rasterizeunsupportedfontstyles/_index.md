## IPdfOptions::set_RasterizeUnsupportedFontStyles(bool) Methode

Gibt an, ob Text als Bitmap gerastert und als PDF gespeichert werden soll, wenn die Schriftart keine fette Formatierung unterstützt. Dieser Ansatz kann die Qualität des Textes im resultierenden PDF für bestimmte Schriftarten verbessern. Schreiben **bool**.

```cpp
virtual void Aspose::Slides::Export::IPdfOptions::set_RasterizeUnsupportedFontStyles(bool value)=0
```

## Hinweise

Standard ist **false**.

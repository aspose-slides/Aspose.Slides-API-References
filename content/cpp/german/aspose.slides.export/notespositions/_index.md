## NotesPositions Enum

Stellt die Regel dar, nach der Notizen in das exportierte Dokument gerendert werden.

```cpp
enum class NotesPositions
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | 0 | Gibt an, dass Notizen überhaupt nicht angezeigt werden sollen. |
| BottomFull | 1 | Gibt an, dass Notizen vollständig angezeigt werden, indem bei Bedarf zusätzliche Seiten verwendet werden. |
| BottomTruncated | 2 | Gibt an, dass Notizen nur auf einer Seite angezeigt werden sollen. |

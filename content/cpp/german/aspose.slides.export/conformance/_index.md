## Konformitäts-Enum

Gibt an, zu welcher Konformitätsklasse das PresentationML-Dokument konform ist.

```cpp
enum class Conformance
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Ecma376_2006 | 0 | Gibt an, dass das Dokument dem ECMA376:2006 entspricht. |
| Iso29500_2008_Transitional | 1 | Gibt an, dass das Dokument der ISO/IEC 29500:2008 Transitional-Konformitätsklasse entspricht. |
| Iso29500_2008_Strict | 2 | Gibt an, dass das Dokument der ISO/IEC 29500:2008 Strict-Konformitätsklasse entspricht. |

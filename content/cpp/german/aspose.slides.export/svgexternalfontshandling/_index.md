## SvgExternalFontsHandling enum

Stellt eine Methode dar, um externe Schriftarten, die für die Textdarstellung verwendet werden, zu verarbeiten.

```cpp
enum class SvgExternalFontsHandling
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| AddLinksToFontFiles | 0 | Fügt Links zu separaten Schriftdateien zum Stilabschnitt der SVG-Datei hinzu. |
| Embed | 1 | Speichert Schriftartdaten direkt in der SVG-Datei. Bitte prüfen Sie alle Lizenzvereinbarungen für externe Schriftarten, bevor Sie diese Option verwenden. |
| Vectorize | 2 | Speichert allen Text, der externe Schriftarten verwendet, als Grafiken. |

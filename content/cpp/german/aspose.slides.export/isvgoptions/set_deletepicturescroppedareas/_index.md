## ISVGOptions::set_DeletePicturesCroppedAreas(bool) Methode

Ein boolesches Flag gibt an, ob die beschnittenen Teile Teil des Dokuments bleiben. Wenn true, werden die beschnittenen Teile entfernt, wenn false werden sie im Dokument serialisiert (was zu einer größeren Datei führen kann). Schreiben **bool**.

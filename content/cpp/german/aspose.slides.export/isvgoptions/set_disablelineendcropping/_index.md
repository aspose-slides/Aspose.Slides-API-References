## ISVGOptions::set_DisableLineEndCropping(bool) Methode


SVG 1.1 bietet keine Möglichkeit, Einzüge für Markierungen zu definieren. [Aspose.Slides](../../../aspose.slides/) SVG-Schreibengine hat eine Lösung für dieses Problem: Sie schneidet das Linienende mit dem Pfeil zu, sodass die Linie die Markierungen nicht überlappt. Diese Option schaltet dieses Verhalten ab. Schreiben **bool**.

```cpp
virtual void Aspose::Slides::Export::ISVGOptions::set_DisableLineEndCropping(bool value)=0
```

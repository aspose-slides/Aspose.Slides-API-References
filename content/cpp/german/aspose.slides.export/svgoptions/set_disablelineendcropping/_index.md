## SVGOptions::set_DisableLineEndCropping(bool) Methode

SVG 1.1 fehlt die Möglichkeit, Einzüge für Marker zu definieren. [Aspose.Slides](../../../aspose.slides/) Die SVG-Schreibengine hat dafür eine Lösung: Sie schneidet das Ende der Linie mit Pfeil ab, sodass die Linie die Marker nicht überlappt. Diese Option deaktiviert dieses Verhalten. Schreiben **bool**.

## EmbedFontCharacters enum


Stellt die Regel dar, die für das Hinzufügen neuer eingebetteter Schriften in [IPresentation](../../aspose.slides/ipresentation/) verwendet wird.

```cpp
enum class EmbedFontCharacters
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| OnlyUsed | 0 | Betten Sie nur die in der Präsentation verwendeten Zeichen ein (am besten zur Reduzierung der Dateigröße). |
| All | 1 | Betten Sie alle Zeichen ein (am besten für die Bearbeitung durch andere Personen). |

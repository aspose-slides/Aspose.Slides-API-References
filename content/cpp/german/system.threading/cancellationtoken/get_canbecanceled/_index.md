## CancellationToken::get_CanBeCanceled() const Methode

Ermittelt, ob dieses Token in den abgebrochenen Zustand versetzt werden kann.

```cpp
bool System::Threading::CancellationToken::get_CanBeCanceled() const
```

### Rückgabewert

true, wenn dieses Token fähig ist, sich im abgebrochenen Zustand zu befinden; andernfalls false.
## Hinweise

Tokens, die aus [CancellationTokenSource](../../cancellationtokensource/) erstellt wurden, geben true zurück, während das None-Token immer false zurückgibt.

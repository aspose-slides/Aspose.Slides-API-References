

### Argumente

| Parameter | Type | Description |
| --- | --- | --- |
| target | const [System::TypeInfo](../../../system/typeinfo/)\& | [TypeInfo](../../../system/typeinfo/) Struktur, die den Typ beschreibt, gegen den das aktuelle Objekt getestet wird. |

### Rückgabewert

True, wenn das Objekt vom angegebenen Typ oder dessen Unterklasse ist, sonst false.

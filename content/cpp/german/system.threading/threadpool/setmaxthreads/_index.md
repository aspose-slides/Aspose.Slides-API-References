## ThreadPool::SetMaxThreads(int, int) Methode

Setzt die Anzahl von Threads, die dem Pool gehören.

```cpp

```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| wt | int | Maximale Anzahl von Worker-Threads. |
| cpt | int | Maximale Anzahl von [IO](../../../system.io/) Threads. |

### Rückgabewert

True wenn die Parameter akzeptiert wurden, false sonst.

## Thread::get_ManagedThreadId() const Methode


Liefert den Bezeichner des Threads. Kann vom Betriebssystem erhalten werden, aber wenn die Thread-Bezeichner des Betriebssystems die Grenzen von int überschreiten, können sich Thread-IDs überschneiden.

```cpp
int System::Threading::Thread::get_ManagedThreadId() const
```


### Rückgabewert

Pseudo-einzigartige Thread-Kennung.

## Details_ExternalException Klasse

Die Basisausnahmetyp für alle COM-Interop-Ausnahmen und Structured Exception Handling (SEH)-Ausnahmen. Erstellen Sie niemals Instanzen dieser Klasse manuell. Verwenden Sie stattdessen die ExternalException Klasse. Wickeln Sie die ExternalException Klasseninstanzen niemals in [System::SmartPtr](../../system/smartptr/).

```cpp
class Details_ExternalException : public System::Details_ExceptionWithErrorCode<Details_SystemException>
```

## FILETIME Struktur

Enthält Dateizeitkomponenten. Dieser Typ sollte auf dem Stack alloziiert und an Funktionen per Wert oder per Referenz übergeben werden. Verwenden Sie niemals die [System::SmartPtr](../../system/smartptr/) Klasse, um Objekte dieses Typs zu verwalten.

```cpp
class FILETIME
```

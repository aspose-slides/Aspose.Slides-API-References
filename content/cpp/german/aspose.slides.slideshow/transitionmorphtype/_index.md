## TransitionMorphType Enum

Stellt einen Typ von Morph-Übergang dar.

```cpp
enum class TransitionMorphType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| ByObject | 0 | Der Morph-Übergang wird ausgeführt, wobei Formen als unteilbare Objekte betrachtet werden. |
| ByWord | 1 | Der Morph-Übergang wird nach Möglichkeit mit der Übertragung von Text Wort für Wort durchgeführt. |
| ByChar | 2 | Der Morph-Übergang wird nach Möglichkeit mit der Übertragung von Text Zeichen für Zeichen durchgeführt. |

## PixelOffsetMode-Enum

Gibt an, wie Pixel beim Rendern versetzt werden.

```cpp
enum class PixelOffsetMode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Invalid | n/a | Ungültiger Wert. |
| Default | n/a | Der Standardmodus. |
| HighSpeed | n/a | Modus mit niedriger Qualität (hohe Leistung). |
| HighQuality | n/a | Modus mit hoher Qualität (geringere Leistung). |
| None | n/a | Keine Pixelverschiebung. |
| Half | n/a | Pixel werden um -0.5 Einheiten sowohl horizontal als auch vertikal versetzt, um eine leistungsstarke Kantenglättung zu ermöglichen. |

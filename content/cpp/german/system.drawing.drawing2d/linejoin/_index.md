## LineJoin enum

Gibt an, wie aufeinanderfolgende Linien oder Kurven verbunden werden.

```cpp
enum class LineJoin
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Miter | 0 | Mitered-Join, das eine scharfe oder abgeschnittene Ecke erzeugt, abhängig davon, ob die Länge des Miter das Miter-Limit überschreitet. |
| Bevel | 1 | Bevel-Join, das eine diagonale Ecke erzeugt. |
| Round | 2 | Round-Join, das einen glatten kreisförmigen Bogen zwischen den Linien erzeugt. |
| MiterClipped | 3 | Mitered-Join, das eine scharfe oder abgeschrägte Ecke erzeugt, abhängig davon, ob die Länge des Miter das Miter-Limit überschreitet. |

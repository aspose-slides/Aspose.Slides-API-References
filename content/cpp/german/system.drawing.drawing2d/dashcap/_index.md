## DashCap enum

Gibt den Typ einer Kappe an, die an beiden Enden eines Strichs in einer gestrichelten Linie verwendet wird.

```cpp
enum class DashCap
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Flat | 0 | Eine quadratische Kappe, die beide Enden eines Strichs quadratisch abschließt. |
| Round | 2 | Eine kreisförmige Kappe, die beide Enden eines Strichs abrundet. |
| Triangle | 3 | Eine dreieckige Kappe, die beide Enden eines Strichs spitz zuläuft. |

## SmoothingMode enum

Gibt an, welche Art von Glättung (antialiasing) auf Linien, Kurven und Kanten von gefüllten Bereichen angewendet wird.

```cpp
enum class SmoothingMode
```

### Werte

| Name | Value | Description |
| --- | --- | --- |
| Invalid | n/a | Ungültiger Wert. |
| Default | n/a | Standardmodus, kein antialiasing. |
| HighSpeed | n/a | Kein antialiasing, hohe Geschwindigkeit. |
| HighQuality | n/a | Antialiasing-basiertes Rendering, niedrige Geschwindigkeit. |
| None | n/a | Kein antialiasing. |
| AntiAlias | n/a | Antialiasing-basiertes Rendering. |

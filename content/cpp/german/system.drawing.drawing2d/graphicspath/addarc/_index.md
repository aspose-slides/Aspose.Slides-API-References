## GraphicsPath::AddArc(float, float, float, float, float, float) Methode

Fügt dem Pfad, der vom aktuellen Objekt dargestellt wird, den angegebenen elliptischen Bogen hinzu.

```cpp
void System::Drawing::Drawing2D::GraphicsPath::AddArc(float x, float y, float width, float height, float startAngle, float sweepAngle)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| x | **float** | Die X-Koordinate der oberen linken Ecke des Rechtecks, das die Ellipse umschließt, aus der der Bogen gezeichnet wird |
| y | **float** | Die Y-Koordinate der oberen linken Ecke des Rechtecks, das die Ellipse umschließt, aus der der Bogen gezeichnet wird |
| width | **float** | Die Breite der oberen linken Ecke des Rechtecks, das die Ellipse umschließt, aus der der Bogen gezeichnet wird |
| height | **float** | Die Höhe der oberen linken Ecke des Rechtecks, das die Ellipse umschließt, aus der der Bogen gezeichnet wird |
| startAngle | **float** | Gibt den Startwinkel des Bogens in Grad an, gemessen im Uhrzeigersinn von der X-Achse |
| sweepAngle | **float** | Gibt den Winkel zwischen dem Startwinkel und dem Ende des Bogens an |

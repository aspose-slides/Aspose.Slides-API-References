## GraphicsPath::AddPie(float, float, float, float, float, float) Methode

Fügt die angegebene Kontur der Kuchenform dem Pfad hinzu, der vom aktuellen Objekt repräsentiert wird.

```cpp
void System::Drawing::Drawing2D::GraphicsPath::AddPie(float x, float y, float width, float height, float startAngle, float sweepAngle)
```

### Arguments

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| x | **float** | Die X-Koordinate der oberen linken Ecke des Rechtecks, das die Ellipse begrenzt, aus der der Kuchen gezeichnet wird |
| y | **float** | Die Y-Koordinate der oberen linken Ecke des Rechtecks, das die Ellipse begrenzt, aus der der Kuchen gezeichnet wird |
| width | **float** | Die Breite des Rechtecks, das die Ellipse begrenzt, aus der der Kuchen gezeichnet wird |
| height | **float** | Die Höhe des Rechtecks, das die Ellipse begrenzt, aus der der Kuchen gezeichnet wird |
| startAngle | **float** | Gibt den Startwinkel des Kuchens in Grad an, gemessen im Uhrzeigersinn von der X-Achse |
| sweepAngle | **float** | Gibt den Winkel zwischen dem Startwinkel und dem Ende des Kuchens an |

## Timer::set_Enabled(bool) Methode


Startet oder stoppt den Timer. Das Starten des Timers setzt die Zeitmessung nicht zurück, wenn der Timer bereits ausgeführt wird.

```cpp
void System::Timers::Timer::set_Enabled(bool value)
```


### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| value | **bool** | true, um den Timer zu starten, false, um ihn zu stoppen. |

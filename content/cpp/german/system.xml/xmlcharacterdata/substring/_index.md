## XmlCharacterData::Substring(int32_t, int32_t) Methode


Gibt einen Teilstring der gesamten Zeichenkette aus dem angegebenen Bereich zurück.

```cpp
virtual String System::Xml::XmlCharacterData::Substring(int32_t offset, int32_t count)
```

### Arguments

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| offset | **int32_t** | Die Position innerhalb der Zeichenkette, an der das Abrufen beginnen soll. Ein Offset von null bedeutet, dass der Startpunkt am Anfang der Daten liegt. |
| count | **int32_t** | Die Anzahl der abzurufenden Zeichen. |

### Rückgabewert

Der Teilstring, der dem angegebenen Bereich entspricht.

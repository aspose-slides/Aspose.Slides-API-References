## XmlTextReader::ReadString() Methode

Liest den Inhalt eines Elements oder eines Textknotens als Zeichenkette.

```cpp
String System::Xml::XmlTextReader::ReadString() override
```

### Rückgabewert

Der Inhalt des Elements oder Textknotens. Dies kann eine leere Zeichenkette sein, wenn sich der Leser auf etwas anderem als einem Element oder Textknoten befindet oder wenn im aktuellen Kontext kein weiterer Textinhalt mehr zurückgegeben werden kann. **Hinweis:** Der Textknoten kann entweder ein Element- oder ein Attribut-Textknoten sein.

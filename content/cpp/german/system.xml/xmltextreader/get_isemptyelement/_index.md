## XmlTextReader::get_IsEmptyElement() Methode

Gibt einen Wert zurück, der angibt, ob das aktuelle Knoten ein leeres Element ist (zum Beispiel **<MyElement/>**).

```cpp
bool System::Xml::XmlTextReader::get_IsEmptyElement() override
```

### Rückgabewert

**true** wenn das aktuelle Knoten ein Element ist ([XmlTextReader::get_NodeType](../get_nodetype/) Wert gleich [XmlNodeType::Element](../../xmlnodetype/)) das mit **/>** endet; andernfalls **false**.

## XmlTextReader::LookupNamespace(const String\&) Methode


Löst ein Namensraum-Präfix im Gültigkeitsbereich des aktuellen Elements auf.

```cpp
String System::Xml::XmlTextReader::LookupNamespace(const String &prefix) override
```


### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| prefix | const [String](../../../system/string/)\& | Das Präfix, dessen Namensraum-URI aufgelöst werden soll. Um den Standard-Namensraum zu verwenden, übergeben Sie eine leere Zeichenkette. Diese Zeichenkette muss nicht atomisiert werden. |

### Rückgabewert

Der Namensraum-URI, dem das Präfix zugeordnet ist, oder **nullptr**, wenn kein passendes Präfix gefunden wurde.

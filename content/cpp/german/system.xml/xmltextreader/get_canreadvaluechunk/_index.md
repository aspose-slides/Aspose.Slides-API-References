## XmlTextReader::get_CanReadValueChunk() Methode

Gibt einen Wert zurück, der angibt, ob die [XmlTextReader](../) die [XmlReader::ReadValueChunk](../../xmlreader/readvaluechunk/)-Methode implementiert.

```cpp
bool System::Xml::XmlTextReader::get_CanReadValueChunk() override
```

### Rückgabewert

**true** wenn die [XmlTextReader](../) die [XmlReader::ReadValueChunk](../../xmlreader/readvaluechunk/)-Methode implementiert; sonst **false**. Die [XmlTextReader](../)-Klasse gibt immer **false** zurück.

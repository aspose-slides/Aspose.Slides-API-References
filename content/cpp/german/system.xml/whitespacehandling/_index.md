## WhitespaceHandling enum

Gibt an, wie Leerzeichen verarbeitet werden.


### Values

| Name | Wert | Beschreibung |
| --- | --- | --- |

| Significant | 1 | Gibt nur **SignificantWhitespace**-Knoten zurück. |

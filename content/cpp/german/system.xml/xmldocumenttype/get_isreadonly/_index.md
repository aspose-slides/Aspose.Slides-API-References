## XmlDocumentType::get_IsReadOnly() Methode

Gibt einen Wert zurück, der angibt, ob der Knoten schreibgeschützt ist.

```cpp
bool System::Xml::XmlDocumentType::get_IsReadOnly() override
```

### Rückgabewert

**true** wenn der Knoten schreibgeschützt ist; andernfalls **false**. Da DocumentType-Knoten schreibgeschützt sind, gibt diese Methode immer **true** zurück.

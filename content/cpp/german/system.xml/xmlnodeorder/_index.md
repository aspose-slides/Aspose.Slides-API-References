## XmlNodeOrder enum


Beschreibt die Dokumentreihenfolge eines Knotens im Vergleich zu einem zweiten Knoten.

```cpp
enum class XmlNodeOrder
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Before | 0 | Der aktuelle Knoten dieses Navigators befindet sich vor dem aktuellen Knoten des übergebenen Navigators. |
| After | 1 | Der aktuelle Knoten dieses Navigators befindet sich nach dem aktuellen Knoten des übergebenen Navigators. |
| Same | 2 | Beide Navigatoren befinden sich auf demselben Knoten. |
| Unknown | 3 | Die Knotepositionen können in der Dokumentreihenfolge nicht relativ zueinander bestimmt werden. Dies kann auftreten, wenn die beiden Knoten in unterschiedlichen Bäumen liegen. |

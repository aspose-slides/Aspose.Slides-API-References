## NamespaceHandling Enum

Gibt an, ob duplizierte Namespace-Deklarationen im [XmlWriter](../xmlwriter/) entfernt werden sollen.

```cpp
enum class NamespaceHandling
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Default | 0 | Gibt an, dass duplizierte Namespace-Deklarationen nicht entfernt werden. |
| OmitDuplicates | 1 | Gibt an, dass duplizierte Namespace-Deklarationen entfernt werden. Damit ein duplizierter Namespace entfernt wird, müssen Präfix und Namespace übereinstimmen. |

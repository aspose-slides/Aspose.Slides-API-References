## XmlNodeReader::LookupNamespace(const String\&) Methode


Löst einen Namensraum-Präfix im Geltungsbereich des aktuellen Elements auf.

```cpp
String System::Xml::XmlNodeReader::LookupNamespace(const String &prefix) override
```


### Arguments

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| prefix | const [String](../../../system/string/)\& | Der Präfix, dessen Namensraum-URI Sie auflösen möchten. Um den Standardsnamensraum zu entsprechen, übergeben Sie eine leere Zeichenkette. Diese Zeichenkette muss nicht atomisiert werden. |

### Return Value

Der Namensraum-URI, auf den der Präfix verweist, oder **nullptr**, wenn kein passender Präfix gefunden wird.

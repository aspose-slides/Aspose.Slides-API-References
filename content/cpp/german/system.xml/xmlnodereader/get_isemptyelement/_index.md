## XmlNodeReader::get_IsEmptyElement() Methode

Gibt einen Wert zurück, der angibt, ob der aktuelle Knoten ein leeres Element ist (zum Beispiel **<MyElement/>**).

```cpp
bool System::Xml::XmlNodeReader::get_IsEmptyElement() override
```

### Rückgabewert

**true** wenn der aktuelle Knoten ein Element ist ([XmlNodeReader::get_NodeType](../get_nodetype/) gleich [XmlNodeType::Element](../../xmlnodetype/)) und es mit **/>** endet; andernfalls **false**.

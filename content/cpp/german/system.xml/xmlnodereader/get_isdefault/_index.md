## XmlNodeReader::get_IsDefault() Methode

Gibt einen Wert zurück, der angibt, ob das aktuelle Knotenobjekt ein Attribut ist, das aus dem in der Document Type Definition (DTD) oder dem Schema definierten Standardwert generiert wurde.

```cpp
bool System::Xml::XmlNodeReader::get_IsDefault() override
```

### Rückgabewert

**true** wenn das aktuelle Knotenobjekt ein Attribut ist, dessen Wert aus dem in der DTD oder dem Schema definierten Standardwert generiert wurde; **false** wenn der Attributwert explizit festgelegt wurde.

## Formatierungs-Enum

Spezifiziert die Formatierungsoptionen für die [XmlTextWriter](../xmltextwriter/).

```cpp
enum class Formatting
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | 0 | Es wird keine spezielle Formatierung angewendet. Dies ist die Vorgabe. |
| Indented | 1 | Veranlasst, dass untergeordnete Elemente gemäß den Einstellungen [XmlTextWriter::set_Indentation](../xmltextwriter/set_indentation/) und [XmlTextWriter::set_IndentChar](../xmltextwriter/set_indentchar/) eingerückt werden. |

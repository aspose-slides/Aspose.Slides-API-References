## XmlReaderSettings::get_MaxCharactersInDocument() Methode

Gibt einen Wert zurück, der die maximal zulässige Anzahl von Zeichen in einem XML-Dokument angibt. Ein Wert von Null (0) bedeutet, dass es keine Beschränkungen für die Größe des XML-Dokuments gibt. Ein von Null verschiedener Wert gibt die maximale Größe in Zeichen an.

```cpp
int64_t System::Xml::XmlReaderSettings::get_MaxCharactersInDocument()
```

### Rückgabewert

Die maximal zulässige Anzahl von Zeichen in einem XML-Dokument. Der Standardwert ist 0.

## XmlReader::ReadInnerXml() Methode

When overridden in a derived class, reads all the content, including markup, as a string.

```cpp
virtual String System::Xml::XmlReader::ReadInnerXml()
```

### Rückgabewert

Der gesamte XML-Inhalt, einschließlich Markup, im aktuellen Knoten. Hat der aktuelle Knoten keine Kindknoten, wird eine leere Zeichenkette zurückgegeben. Ist der aktuelle Knoten weder ein Element noch ein Attribut, wird ebenfalls eine leere Zeichenkette zurückgegeben.

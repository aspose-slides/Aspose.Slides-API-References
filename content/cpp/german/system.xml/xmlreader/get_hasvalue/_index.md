## XmlReader::get_HasValue() Methode


Wenn in einer abgeleiteten Klasse überschrieben, gibt einen Wert zurück, der angibt, ob der aktuelle Knoten einen [XmlReader::get_Value](../get_value/)-Wert haben kann.

```cpp
virtual bool System::Xml::XmlReader::get_HasValue()
```


### Return Value

**true** wenn der Knoten, auf dem sich der Reader gerade befindet, einen **Value** haben kann; sonst **false**. Wenn **false**, hat der Knoten einen Wert von [String::Empty](../../../system/string/empty/).

## XmlReader::MoveToFirstAttribute() method


Wenn sie in einer abgeleiteten Klasse überschrieben wird, bewegt sie sich zum ersten Attribut.

```cpp
virtual bool System::Xml::XmlReader::MoveToFirstAttribute()=0
```


### Rückgabewert

**true** wenn ein Attribut existiert (der Reader bewegt sich zum ersten Attribut); andernfalls **false** (die Position des Readers ändert sich nicht).

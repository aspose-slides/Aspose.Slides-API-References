## XmlReader::get_NamespaceURI() Methode


Wenn sie in einer abgeleiteten Klasse überschrieben wird, gibt sie den namespace URI (wie in der W3C Namespace specification definiert) des Knotens zurück, auf dem sich der Reader befindet.

```cpp
virtual String System::Xml::XmlReader::get_NamespaceURI()=0
```


### Rückgabewert

Der namespace URI des aktuellen Knotens; andernfalls eine leere Zeichenkette.

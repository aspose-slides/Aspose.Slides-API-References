## XmlReader::get_IsDefault() Methode


Wenn sie in einer abgeleiteten Klasse überschrieben wird, liefert sie einen Wert, der angibt, ob das aktuelle Knoten ein Attribut ist, das aus dem im DTD oder Schema definierten Standardwert generiert wurde.

```cpp
virtual bool System::Xml::XmlReader::get_IsDefault()
```


### Rückgabewert

**true** wenn das aktuelle Knoten ein Attribut ist, dessen Wert aus dem im DTD oder Schema definierten Standardwert generiert wurde; **false** wenn der Attributwert explizit gesetzt wurde.

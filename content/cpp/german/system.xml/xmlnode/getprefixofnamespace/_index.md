## XmlNode::GetPrefixOfNamespace(String) Methode

Sucht die nächstgelegene **xmlns**-Deklaration für die angegebene Namespace-URI, die für den aktuellen Knoten im Gültigkeitsbereich ist, und gibt das in dieser Deklaration definierte Präfix zurück.

```cpp
virtual String System::Xml::XmlNode::GetPrefixOfNamespace(String namespaceURI)
```


### Argumente

| Parameter | Typ | Beschreibung |

| namespaceURI | [String](../../../system/string/) | Die Namespace-URI, deren Präfix Sie finden möchten. |

### Rückgabewert

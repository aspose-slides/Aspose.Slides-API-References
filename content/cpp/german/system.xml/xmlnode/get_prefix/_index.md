Gibt das Namensraum-Präfix dieses Knotens zurück.

### Rückgabewert

Das Namensraum-Präfix dieses Knotens. Zum Beispiel ist **Prefix** **bk** für das Element **<bk:book>**. Wenn kein Präfix vorhanden ist, gibt diese Methode [String::Empty](../../../system/string/empty/) zurück.

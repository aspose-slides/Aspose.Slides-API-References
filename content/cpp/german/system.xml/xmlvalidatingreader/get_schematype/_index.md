## XmlValidatingReader::get_SchemaType() Methode

Gibt ein Schema-Typ-Objekt zurück.

```cpp
SharedPtr<Object> System::Xml::XmlValidatingReader::get_SchemaType()
```

### Rückgabewert

XmlSchemaDatatype, XmlSchemaSimpleType oder XmlSchemaComplexType abhängig davon, ob der Knotenwert ein integrierter XML [Schema](../../../system.xml.schema/) Definitionssprache (XSD)-Typ oder ein benutzerdefinierter simpleType oder complexType ist; **nullptr** falls der aktuelle Knoten keinen Schema-Typ hat.

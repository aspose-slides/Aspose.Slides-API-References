## XmlValidatingReader::get_HasValue() Methode

Gibt einen Wert zurück, der angibt, ob der aktuelle Knoten ein [XmlValidatingReader::get_Value](../get_value/) haben kann, das nicht [String::Empty](../../../system/string/empty/) ist.

```cpp
bool System::Xml::XmlValidatingReader::get_HasValue() override
```


### Rückgabewert

**true** if the node on which the reader is currently positioned can have a **Value**; otherwise, **false**.

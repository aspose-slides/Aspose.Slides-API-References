## XmlValidatingReader::LookupNamespace(const String\&) Methode


Löst ein Namensraum-Präfix im Geltungsbereich des aktuellen Elements auf.

```cpp
String System::Xml::XmlValidatingReader::LookupNamespace(const String &prefix) override
```


### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| prefix | const [String](../../../system/string/)\& | Das Präfix, dessen Namensraum-Uniform-Resource-Identifier (URI) Sie auflösen möchten. Um den Standard-Namensraum zuzuordnen, übergeben Sie eine leere Zeichenkette. |

### Rückgabewert

Der Namensraum-URI, dem das Präfix zugeordnet ist, oder **nullptr** wenn kein passendes Präfix gefunden wird.

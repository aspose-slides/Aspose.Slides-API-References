## XmlValidatingReader::ReadString() Methode

Liest den Inhalt eines Elements oder Textknotens als Zeichenkette.

```cpp
String System::Xml::XmlValidatingReader::ReadString() override
```

### Rückgabewert

Der Inhalt des Elements oder Textknotens. Dies kann eine leere Zeichenkette sein, wenn der Reader auf etwas anderes als ein Element oder Textknoten positioniert ist oder wenn im aktuellen Kontext kein weiterer Textinhalt zurückzugeben ist.

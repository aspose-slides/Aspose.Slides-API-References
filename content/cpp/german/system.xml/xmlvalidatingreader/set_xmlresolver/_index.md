## XmlValidatingReader::set_XmlResolver(const SharedPtr\<System::Xml::XmlResolver\>\&) method

Legt den [XmlResolver](../../xmlresolver/) fest, der zur Auflösung externer Document Type Definition (DTD)- und Schema-Standortreferenzen verwendet wird. Der [XmlResolver](../../xmlresolver/) wird außerdem verwendet, um Import- oder Include-Elemente zu verarbeiten, die in XML [Schema](../../../system.xml.schema/)-Definitionssprache (XSD)-Schemata gefunden werden.

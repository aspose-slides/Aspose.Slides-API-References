## XmlNodeChangedEventArgs::get_Action() Methode

Gibt einen Wert zurück, der angibt, welcher Typ von Knotenänderungsereignis auftritt.

```cpp
XmlNodeChangedAction System::Xml::XmlNodeChangedEventArgs::get_Action()
```

### Rückgabewert

Ein Wert aus der Aufzählung, der das Knotenwechselereignis beschreibt. Der **Action**-Wert unterscheidet nicht, wann das Ereignis aufgetreten ist (vorher oder danach). Sie können separate Ereignishandler erstellen, um beide Fälle zu behandeln.

## XmlConvert::VerifyNCName(const String\&) Methode


Verifiziert, dass der Name ein gültiger **NCName** gemäß der W3C-Empfehlung für erweiterte Auszeichnungssprache ist. Ein **NCName** ist ein Name, der keinen Doppelpunkt enthalten darf.

```cpp
static String System::Xml::XmlConvert::VerifyNCName(const String &name)
```


### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| name | const [String](../../../system/string/)\& | Der zu überprüfende Name. |

### Rückgabewert

Der Name, wenn er ein gültiger NCName ist.

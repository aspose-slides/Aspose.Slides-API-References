## XmlConvert::VerifyXmlChars(const String\&) Methode

Gibt die übergebene Zeichenkette zurück, wenn alle Zeichen und Surrogat-Paar-Zeichen im String-Argument gültige XML-Zeichen sind; andernfalls wird eine XmlException mit Informationen über das erste ungültige Zeichen ausgelöst.

```cpp
static String System::Xml::XmlConvert::VerifyXmlChars(const String &content)
```

### Rückgabewert

Gibt die übergebene Zeichenkette zurück, wenn alle Zeichen und Surrogat-Paar-Zeichen im String-Argument gültige XML-Zeichen sind; andernfalls wird eine XmlException mit Informationen über das erste ungültige Zeichen ausgelöst.

## XmlElement::set_IsEmpty(bool) Methode

Setzt das Tag-Format des Elements.

```cpp
void System::Xml::XmlElement::set_IsEmpty(bool value)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| value | **bool** | Der zu setzende Wert. Beim Setzen dieses Wertes, wenn er auf **true** gesetzt ist, werden die Kind-Elemente entfernt und das Element wird im Kurz-Tag-Format serialisiert. Wird er auf **false** gesetzt, wird der Wert geändert (unabhängig davon, ob das Element Inhalt hat oder nicht); ist das Element leer, wird es im Lang-Tag-Format serialisiert. |

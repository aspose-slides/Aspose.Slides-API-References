## XmlWriterSettings::set_NamespaceHandling(System::Xml::NamespaceHandling) Methode

Legt einen Wert fest, der angibt, ob [XmlWriter](../../xmlwriter/) beim Schreiben von XML-Inhalten doppelte Namespace-Deklarationen entfernen soll. Das Standardverhalten besteht darin, dass der Writer alle Namespace-Deklarationen ausgibt, die im Namespace-Resolver des Writers vorhanden sind.

```cpp
void System::Xml::XmlWriterSettings::set_NamespaceHandling(System::Xml::NamespaceHandling value)
```

| --- | --- | --- |
| value | [System::Xml::NamespaceHandling](../../namespacehandling/) | Der zu setzende Wert. |

## XmlWriterSettings::get_NamespaceHandling() Methode


Gibt einen Wert zurück, der angibt, ob die [XmlWriter](../../xmlwriter/) doppelte Namespace-Deklarationen beim Schreiben von XML-Inhalten entfernen soll. Das Standardverhalten besteht darin, dass der Writer alle Namespace-Deklarationen ausgibt, die im Namespace-Resolver des Writers vorhanden sind.

```cpp
System::Xml::NamespaceHandling System::Xml::XmlWriterSettings::get_NamespaceHandling()
```


### Rückgabewert

Der NamespaceHandling-Aufzählungstyp, der angibt, ob doppelte Namespace-Deklarationen im [XmlWriter](../../xmlwriter/) entfernt werden sollen.

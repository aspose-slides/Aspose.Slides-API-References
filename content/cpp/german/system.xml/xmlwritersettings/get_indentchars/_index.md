## XmlWriterSettings::get_IndentChars() Methode

Gibt die Zeichenfolge zurück, die beim Einrücken verwendet wird. Diese Einstellung wird verwendet, wenn der [XmlWriterSettings::set_Indent](../set_indent/) Wert auf **true** gesetzt ist.

```cpp
String System::Xml::XmlWriterSettings::get_IndentChars()
```

### Rückgabewert

Die Zeichenfolge, die beim Einrücken verwendet wird. Sie kann auf jeden Zeichenkettenwert gesetzt werden. Um jedoch gültiges XML sicherzustellen, sollten Sie nur gültige Leerzeichen-Zeichen angeben, wie z. B. Leerzeichen, Tabulatoren, Wagenrückläufe oder Zeilenumbrüche. Die Vorgabe ist zwei Leerzeichen.

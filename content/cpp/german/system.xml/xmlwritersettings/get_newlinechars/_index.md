## XmlWriterSettings::get_NewLineChars() Methode


Gibt die Zeichenkette zurück, die für Zeilenumbrüche verwendet wird.

```cpp
String System::Xml::XmlWriterSettings::get_NewLineChars()
```


### Rückgabewert

Die Zeichenkette, die für Zeilenumbrüche verwendet wird. Sie kann auf jeden Zeichenwert gesetzt werden. Um jedoch gültiges XML zu gewährleisten, sollten nur gültige Leerzeichenzeichen angegeben werden, wie zum Beispiel Leerzeichen, Tabulatoren, Wagenrückläufe oder Zeilenumbrüche. Der Standardwert ist **\r\n** (Wagenrücklauf, neue Zeile).

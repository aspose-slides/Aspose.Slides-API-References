## XmlNamespaceManager::RemoveNamespace(String, String) Methode

Entfernt den angegebenen Namespace für das angegebene Präfix.

```cpp
virtual void System::Xml::XmlNamespaceManager::RemoveNamespace(String prefix, String uri)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| prefix | [String](../../../system/string/) | Das Präfix für den Namespace. |
| uri | [String](../../../system/string/) | Der Namespace, der für das angegebene Präfix entfernt werden soll. Der entfernte Namespace stammt aus dem aktuellen Namespace-Bereich. Namespaces außerhalb des aktuellen Bereichs werden ignoriert. |

## XmlNamespaceManager::HasNamespace(String) Methode

Gibt einen Wert zurück, der angibt, ob für das angegebene Präfix im aktuellen eingeschobenen Gültigkeitsbereich ein Namensraum definiert ist.

```cpp
virtual bool System::Xml::XmlNamespaceManager::HasNamespace(String prefix)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| prefix | [String](../../../system/string/) | Das Präfix des Namensraums, den Sie finden möchten. |

### Rückgabewert

**true**, wenn ein Namensraum definiert ist; andernfalls **false**.

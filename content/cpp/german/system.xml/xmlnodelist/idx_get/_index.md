## XmlNodeList::idx_get(int32_t) Methode

Gibt einen Knoten am angegebenen Index zurück.

```cpp
virtual SharedPtr<XmlNode> System::Xml::XmlNodeList::idx_get(int32_t i)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| i | **int32_t** | Der nullbasierte Index in der Liste der Knoten. |

### Rückgabewert

Das [XmlNode](../../xmlnode/) mit dem angegebenen Index in der Sammlung. Wenn der Index größer als oder gleich der Anzahl der Knoten in der Liste ist, wird **nullptr** zurückgegeben.

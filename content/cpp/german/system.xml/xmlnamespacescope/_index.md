## XmlNamespaceScope enum

Definiert den Namensbereich.

```cpp
enum class XmlNamespaceScope
```

### Werte

| Name | Value | Description |
| --- | --- | --- |
| All | 0 | Alle im Geltungsbereich des aktuellen Knotens definierten Namensräume. Dies schließt den xmlns:xml Namensraum ein, der immer implizit deklariert wird. Die Reihenfolge der zurückgegebenen Namensräume ist nicht definiert. |
| ExcludeXml | 1 | Alle im Geltungsbereich des aktuellen Knotens definierten Namensräume, ausgenommen den xmlns:xml Namensraum, der immer implizit deklariert wird. Die Reihenfolge der zurückgegebenen Namensräume ist nicht definiert. |
| Local | 2 | Alle Namensräume, die lokal am aktuellen Knoten definiert sind. |

## NewLineHandling enum

Gibt an, wie Zeilenumbrüche behandelt werden.

```cpp
enum class NewLineHandling
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Replace | 0 | Zeilenzeichen werden ersetzt, um dem in [XmlWriterSettings::set_NewLineChars](../xmlwritersettings/set_newlinechars/) angegebenen Zeichen zu entsprechen. |
| Entitize | 1 | Zeilenzeichen werden entitisiert. Diese Einstellung bewahrt alle Zeichen, wenn die Ausgabe von einem normalisierenden [XmlReader](../xmlreader/) gelesen wird. |
| None | 2 | Die Zeilenzeichen bleiben unverändert. Die Ausgabe ist identisch mit der Eingabe. |

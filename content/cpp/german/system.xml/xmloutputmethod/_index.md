## XmlOutputMethod enum

Gibt die Methode an, die zum Serialisieren der [XmlWriter](../xmlwriter/) Ausgabe verwendet wird.

```cpp
enum class XmlOutputMethod
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Xml | 0 | Serialisiert nach den XML 1.0 Regeln. |
| Html | 1 | Serialisiert nach den HTML Regeln, die von XSLT festgelegt werden. |
| Text | 2 | Serialisiert nur Textblöcke. |
| AutoDetect | 3 | Verwendet die XSLT Regeln, um zur Laufzeit zwischen den [XmlOutputMethod::Xml](./) und [XmlOutputMethod::Html](./) Ausgabe-Methoden zu wählen. |

## XmlWriter::WriteQualifiedName(const String\&, const String\&) Methode

Wenn sie in einer abgeleiteten Klasse überschrieben wird, schreibt sie den namespace-qualifizierten Namen. Diese Methode sucht das Präfix, das für den angegebenen Namespace im Gültigkeitsbereich ist.

```cpp
virtual void System::Xml::XmlWriter::WriteQualifiedName(const String &localName, const String &ns)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| localName | const [String](../../../system/string/)\& | Der lokale Name, der geschrieben werden soll. |
| ns | const [String](../../../system/string/)\& | Der Namespace-URI für den Namen. |

## XmlTextWriter::get_XmlSpace() Methode

Gibt einen XmlSpace zurück, der den aktuellen **xml:space** Bereich darstellt.

```cpp
System::Xml::XmlSpace System::Xml::XmlTextWriter::get_XmlSpace() override
```

### Rückgabewert

Ein XmlSpace, der den aktuellen **xml:space** Bereich darstellt.

| Wert | Bedeutung |
| --- | --- |
| None | Dies ist die Vorgabe, wenn kein `xml:space` Bereich existiert. |
| Default | Der aktuelle Bereich ist `xml:space="default"`. |
| Preserve | Der aktuelle Bereich ist `xml:space="preserve"`. |

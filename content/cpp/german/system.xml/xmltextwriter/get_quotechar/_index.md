## XmlTextWriter::get_QuoteChar() Methode


Gibt zurück, welches Zeichen zum Anführen von Attributwerten verwendet werden soll.

```cpp
char16_t System::Xml::XmlTextWriter::get_QuoteChar()
```


### Rückgabewert

Das Zeichen, das zum Anführen von Attributwerten verwendet wird. Es muss ein einfaches Anführungszeichen **'** oder ein doppeltes Anführungszeichen **\"** sein. Der Standard ist ein doppeltes Anführungszeichen.

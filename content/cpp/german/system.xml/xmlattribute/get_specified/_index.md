## XmlAttribute::get_Specified() Methode

Gibt einen Wert zurück, der angibt, ob der Attributwert explizit gesetzt wurde.

```cpp
virtual bool System::Xml::XmlAttribute::get_Specified()
```

### Rückgabewert

**true** wenn dieses Attribut im ursprünglichen Instanzdokument explizit einen Wert erhalten hat; andernfalls **false**. Ein Wert von **false** zeigt an, dass der Wert des Attributs aus dem DTD stammt.

## XmlDeclaration::get_Standalone() Methode


Gibt den Wert des Attributs standalone zurück.

```cpp
String System::Xml::XmlDeclaration::get_Standalone()
```


### Rückgabewert

Gültige Werte sind **yes** wenn alle von dem XML-Dokument erforderlichen Entity-Deklarationen im Dokument enthalten sind oder **no** wenn eine externe Document Type Definition (DTD) erforderlich ist. Wenn das Attribut standalone nicht in der XML-Deklaration vorhanden ist, gibt diese Methode [String::Empty](../../../system/string/empty/) zurück.

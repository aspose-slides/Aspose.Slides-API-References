## XmlKnownDtds enum

Die [Resolvers::XmlKnownDtds](./)-Aufzählung wird von [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) verwendet und definiert, welche bekannten DTDs [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) erkennt.

```cpp
enum class XmlKnownDtds
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | 0 | Gibt an, dass [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) keine der vordefinierten DTDs erkennt. |
| Xhtml10 | 1 | Gibt an, dass [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) DTDs und Entitäten erkennt, die in XHTML 1.0 definiert sind. |
| Rss091 | 2 | Gibt an, dass [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) DTDs und Entitäten erkennt, die in RSS 0.91 definiert sind. |
| All | 65535 | Gibt an, dass [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) alle derzeit unterstützten DTDs erkennt. Dies ist das Standardverhalten. |

## X509KeyUsageExtension Klasse

Extension-Objekt, um zusätzliche Informationen über die Verwendung eines Schlüssels zu speichern. Objekte dieser Klasse sollten nur mit der [System::MakeObject()](../../system/makeobject/) Funktion alloziert werden. Erzeugen Sie niemals eine Instanz dieses Typs auf dem Stack oder mithilfe des Operators new, da dies zu Laufzeitfehlern und/oder Assertion-Fehlern führen kann. Umwickeln Sie diese Klasse stets in einen [System::SmartPtr](../../system/smartptr/) Zeiger und verwenden Sie diesen Zeiger, um ihn als Argument an Funktionen zu übergeben.

```cpp
class X509KeyUsageExtension : public System::Security::Cryptography::X509Certificates::X509Extension
```

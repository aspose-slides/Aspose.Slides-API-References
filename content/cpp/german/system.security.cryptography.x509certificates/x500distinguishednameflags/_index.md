## X500DistinguishedNameFlags enum

Formatierungsregeln für den distinguished name eines X509-Zertifikats.

```cpp
enum class X500DistinguishedNameFlags
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | 0 | Keine besonderen Merkmale. |
| Reversed | 1 | Name ist reserviert. |
| UseSemicolons | 16 | Semikolons verwenden. |
| DoNotUsePlusSign | 32 | Name verwendet kein Pluszeichen. |
| DoNotUseQuotes | 64 | Anführungszeichen im Namen deaktivieren. |
| UseCommas | 128 | Kommas verwenden aktivieren. |
| UseNewLines | 256 | Zeilenumbrüche verwenden aktivieren. |
| UseUTF8Encoding | 4096 | Wechselt von Unicode zu UTF-8-Kodierung. |
| UseT61Encoding | 8192 | Wechselt zur T61-Kodierung. |
| ForceUTF8Encoding | 16384 | Erzwingt die Verwendung von UTF-8 beim Kodieren spezifischer X500-Schlüssel. |

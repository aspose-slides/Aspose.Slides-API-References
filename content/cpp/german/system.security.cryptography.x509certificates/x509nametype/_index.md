## X509NameType Enum


Typ eines im X.509-Zertifikat enthaltenen Namens, der sich entweder auf den Aussteller oder das Subjekt des Zertifikats bezieht.

```cpp
enum class X509NameType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| SimpleName | 0 | Einfacher Name des Subjekts oder des Ausstellers. |
| EmailName | 1 | E-Mail-Adresse. |
| UpnName | 2 | UPN-Name. |
| DnsName | 3 | DNS-Name. |
| DnsFromAlternativeName | 4 | Alternativer DNS-Name, der dem DNS-Namen entspricht. |
| UrlName | 5 | URL. |

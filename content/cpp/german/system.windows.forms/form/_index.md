## Form-Klasse

Dummy-Klasse, um übersetzten Code, der die Klasse [Form](./) verwendet, kompilierbar zu machen. Methoden sind nicht implementiert. Objekte dieser Klasse dürfen nur über die Funktion [System::MakeObject()](../../system/makeobject/) allokiert werden. Erzeugen Sie niemals eine Instanz dieses Typs auf dem Stack oder mit dem Operator new, da dies zu Laufzeitfehlern und/oder Assertion-Fehlern führt. Wickeln Sie diese Klasse immer in einen [System::SmartPtr](../../system/smartptr/)-Zeiger ein und verwenden Sie diesen Zeiger, um ihn als Argument an Funktionen zu übergeben.

```cpp
class Form : public System::Windows::Forms::Control
```

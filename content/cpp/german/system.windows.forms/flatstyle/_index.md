## FlatStyle enum

Definiert, ob die Steuerung flach oder dreidimensional gezeichnet wird.

```cpp
enum class FlatStyle
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Flat | 0 | [Control](../control/) wird flach gezeichnet. |
| Popup | 1 | [Control](../control/) wird flach gezeichnet, es sei denn, die Maus bewegt sich darüber, wenn es 3-D-artig gezeichnet wird. |
| Standard | 2 | [Control](../control/) wird 3-D-artig gezeichnet. |
| System | 3 | Style wird vom Betriebssystem gesteuert. |

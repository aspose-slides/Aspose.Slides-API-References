## LoadOptions::get_BlobManagementOptions() Methode

Stellt die Optionen dar, die zur Verwaltung des Verhaltens von Binary Large Objects (BLOBs) verwendet werden können, z. B. die Verwendung von temporären Dateien oder die maximale BLOB-Bytes im Speicher. Diese Optionen sollen das beste Verhältnis von Leistung zu Speicherverbrauch für eine bestimmte Umgebung oder Anforderung festlegen.

```cpp
System::SharedPtr<IBlobManagementOptions> Aspose::Slides::LoadOptions::get_BlobManagementOptions() override
```

## Hinweise

Ein Binary Large Object (BLOB) ist ein Binärdatensatz, der als einzelne Entität gespeichert wird - das heißt, ein BLOB kann ein Audio, Video oder eine Präsentation selbst sein.

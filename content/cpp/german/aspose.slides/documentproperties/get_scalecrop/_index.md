## DocumentProperties::get_ScaleCrop() Methode


Gibt den Anzeigemodus der Dokumentvorschau an. Setzen Sie dieses Element auf **true**, um die Skalierung der Dokumentvorschau auf die Anzeige zu aktivieren. Setzen Sie dieses Element auf **false**, um das Zuschneiden der Dokumentvorschau zu aktivieren, damit nur Abschnitte angezeigt werden, die zur Anzeige passen. Lese **bool**.

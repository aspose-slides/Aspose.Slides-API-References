## DocumentProperties::set_ScaleCrop(bool) Methode


Gibt den Anzeigemodus des Dokument-Vorschaubildes an. Setzen Sie dieses Element auf **true**, um das Skalieren des Dokument-Vorschaubildes an die Anzeige zu aktivieren. Setzen Sie dieses Element auf **false**, um das Zuschneiden des Dokument-Vorschaubildes zu aktivieren, sodass nur Abschnitte angezeigt werden, die in die Anzeige passen. Schreiben **bool**.

void Aspose::Slides::DocumentProperties::set_ScaleCrop(bool value) override

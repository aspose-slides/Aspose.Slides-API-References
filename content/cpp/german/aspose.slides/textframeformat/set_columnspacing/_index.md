## TextFrameFormat::set_ColumnSpacing(double) Methode


Legt den Abstand zwischen Textspalten im Textbereich (in Punkten) fest. Dies sollte nur gelten, wenn mehr als 1 Spalte vorhanden ist. Dieser Wert muss eine positive Zahl sein. Andernfalls wird der Wert auf Null gesetzt. Schreiben **double**.

void Aspose::Slides::TextFrameFormat::set_ColumnSpacing(double value) override

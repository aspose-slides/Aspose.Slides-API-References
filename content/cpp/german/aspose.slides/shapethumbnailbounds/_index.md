## ShapeThumbnailBounds Enum

Aufzählung der Typen von Form-Miniaturansichtsgrenzen.

```cpp
enum class ShapeThumbnailBounds
```

### Values

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Slide | 0 | [Shape](../shape/) Miniaturansicht wird die Größe erhalten, die der Foliengröße entspricht. [Shape](../shape/) Position wird gespeichert. |
| Shape | 1 | [Shape](../shape/) Miniaturansicht hat die Größe des Rechtecks, das die Formbegrenzung darstellt, wobei die Einstellungen der Formkontur berücksichtigt werden. |
| Appearance | 2 | [Shape](../shape/) Miniaturansicht hat die Größe des Form-Erscheinungsbildes (innerhalb der Folienbegrenzungen). Es kann Fälle geben, in denen das Form-Erscheinungsbild nicht in die Formbegrenzungen passt. Z. B. Drehung, Gehrungsschnitt eines spitzen Winkels, 3-D-Effekte usw. |

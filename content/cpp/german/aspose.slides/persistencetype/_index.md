## PersistenceType enum

Gibt die Methode an, die zum Speichern von Eigenschaften des ActiveX-Steuerelements verwendet wird.

```cpp
enum class PersistenceType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| NotDefined | -1 | Persistenz-ID nicht angegeben. |
| PersistPropertyBag | 0 | Gibt an, dass das ActiveX-Steuerelement mithilfe einer property-bag-basierten Persistenz gespeichert wird. Property-bag-basierte Persistenz speichert ein ActiveX-Steuerelement mittels einer Sammlung von Namens-Wert-Paaren, die die vom ActiveX-Steuerelement gespeicherten Daten festlegen. |
| PersistStream | 1 | Gibt an, dass das ActiveX-Steuerelement mithilfe einer stream-basierten Persistenz gespeichert wird, die keine Initialisierung des ActiveX-Steuerelements in einen Standardzustand unterstützt. |
| PersistStreamInit | 2 | Gibt an, dass das ActiveX-Steuerelement mithilfe einer stream-basierten Persistenz gespeichert wird, die die Initialisierung des ActiveX-Steuerelements in einen Standardzustand unterstützt. |
| PersistStorage | 3 | Gibt an, dass das ActiveX-Steuerelement mithilfe einer speicher-basierten Persistenz gespeichert wird. |

## PathCommandType Enum

Stellt Grafik-Pfadbefehle dar

```cpp
enum class PathCommandType : uint8_t
```

### Values

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Close | 0 | Schließt den Geometrie-Pfad |
| MoveTo | 1 | Setzt die Position des nächsten Punkts des Geometrie-Pfads |
| LineTo | 2 | Fügt eine Linie zum Geometrie-Pfad hinzu |
| ArcTo | 3 | Fügt den angegebenen Bogen zum Geometrie-Pfad hinzu. |
| QuadBezierTo | 4 | Fügt eine quadratische Bezierkurve zum Geometrie-Pfad hinzu |
| CubicBezierTo | 5 | Fügt eine kubische Bezierkurve zum Geometrie-Pfad hinzu |

## IBackdrop3DScene::set_NormalVector(System::ArrayPtr\<float\>) Methode

Setzt einen Normalenvektor. Genauer gesagt definiert dieses Attribut einen Vektor, der senkrecht zur Fläche der Hintergrundebene steht. Der Vektor wird durch ein Array von 3 float-Werten dargestellt, die die X-, Y- und Z-Koordinaten definieren. Schreiben **float**[].

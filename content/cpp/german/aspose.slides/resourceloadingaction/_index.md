## ResourceLoadingAction Enum

Gibt den Modus des Ladens externer Ressourcen an.

```cpp
enum class ResourceLoadingAction
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Default | 0 | [Aspose.Slides](../) lädt externe Ressourcen wie üblich. |
| Skip | 1 | [Aspose.Slides](../) überspringt das Laden externer Ressourcen. Nur ein Link ohne Daten wird für ein Bild gespeichert. |
| UserProvided | 2 | [Aspose.Slides](../) verwendet das vom Benutzer in [IResourceLoadingArgs::SetData](../iresourceloadingargs/setdata/) bereitgestellte Byte-Array als Bilddaten. |

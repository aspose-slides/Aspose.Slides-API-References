## ISlideShowTransition::get_SoundIsBuiltIn() Methode

Gibt an, ob dieser Sound ein integrierter Sound ist. Wenn dieses Attribut auf true gesetzt ist, wird die erzeugende Anwendung darauf hingewiesen, das Namensattribut zu prüfen, das für diesen Sound in ihrer Liste integrierter Sounds angegeben ist, und kann dann bei Bedarf einen benutzerdefinierten Namen oder eine UI bereitstellen. Liest **bool**.

```cpp
virtual bool Aspose::Slides::ISlideShowTransition::get_SoundIsBuiltIn()=0
```

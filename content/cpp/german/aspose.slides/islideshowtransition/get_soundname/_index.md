## ISlideShowTransition::get_SoundName() Methode

Gibt einen menschenlesbaren Namen für das Geräusch des Übergangs an. Das [ISlideShowTransition::set_Sound](../set_sound/) muss zugewiesen werden, um den Soundnamen zu erhalten oder festzulegen. Liest [System::String](../../../system/string/).

```cpp
virtual System::String Aspose::Slides::ISlideShowTransition::get_SoundName()=0
```

## Hinweise

Dieser Name erscheint in der PowerPoint-Benutzeroberfläche, wenn der Übergangs-Sound manuell konfiguriert wird.

## ISlideShowTransition::set_SoundName(System::String) Methode


Gibt einen menschenlesbaren Namen für den Sound des Übergangs an. Die [ISlideShowTransition::set_Sound](../set_sound/) muss zugewiesen werden, um den Soundnamen zu erhalten oder festzulegen. Schreibt [System::String](../../../system/string/).

```cpp
virtual void Aspose::Slides::ISlideShowTransition::set_SoundName(System::String value)=0
```

## Anmerkungen


Dieser Name wird in der PowerPoint-Benutzeroberfläche angezeigt, wenn der Übergangssound manuell konfiguriert wird.

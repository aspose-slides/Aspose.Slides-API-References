## ISlideShowTransition::set_SoundIsBuiltIn(bool) Methode

Legt fest, ob dieser Sound ein eingebauter Sound ist oder nicht. Wenn dieses Attribut auf true gesetzt ist, wird die erzeugende Anwendung darauf hingewiesen, das Namensattribut zu überprüfen, das für diesen Sound in ihrer Liste integrierter Sounds angegeben ist, und kann dann bei Bedarf einen benutzerdefinierten Namen oder eine Benutzeroberfläche bereitstellen. Schreibt **bool**.

virtual void Aspose::Slides::ISlideShowTransition::set_SoundIsBuiltIn(bool value)=0

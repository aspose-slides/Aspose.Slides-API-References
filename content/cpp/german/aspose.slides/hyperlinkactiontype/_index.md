## HyperlinkActionType-Enum

Stellt einen Typ einer Hyperlink-Aktion dar.

```cpp
enum class HyperlinkActionType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Unknown | -1 | Unerkannter Aktionstyp. |
| NoAction | 0 | Keine Aktion. |
| Hyperlink | 1 | Gewöhnlicher Hyperlink. |
| JumpFirstSlide | 2 | Springt zur ersten Folie der Präsentation. |
| JumpPreviousSlide | 3 | Springt zur vorherigen Folie. |
| JumpNextSlide | 4 | Springt zur nächsten Folie. |
| JumpLastSlide | 5 | Springt zur letzten Folie der Präsentation. |
| JumpEndShow | 6 | Springt zum Ende der Diashow. |
| JumpLastViewedSlide | 7 | Springt zur zuletzt angezeigten Folie. |
| JumpSpecificSlide | 8 | Springt zur spezifischen Folie, die durch den [IHyperlink::get_TargetSlide](../ihyperlink/get_targetslide/)-Wert referenziert wird. |
| StartCustomSlideShow | 9 | Startet benutzerdefinierte Diashow. |
| OpenFile | 10 | Öffnet die referenzierte Datei. |
| OpenPresentation | 11 | Öffnet die referenzierte Präsentation. |
| StartStopMedia | 12 | Startet/stoppt die Wiedergabe der Mediendatei. |
| StartMacro | 13 | Startet die Ausführung des Makro-Skripts. |
| StartProgram | 14 | Startet das Programm. |

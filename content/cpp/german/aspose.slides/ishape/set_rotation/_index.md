## IShape::set_Rotation(float) Methode


Setzt die Anzahl der Grad, um die die angegebene Form um die Z-Achse gedreht wird. Ein positiver Wert bedeutet eine Drehung im Uhrzeigersinn; ein negativer Wert bedeutet eine Drehung gegen den Uhrzeigersinn. Schreiben **float**.

```cpp
virtual void Aspose::Slides::IShape::set_Rotation(float value)=0
```

## Hinweise


Der zurückgegebene Wert ist immer definiert (ist nicht std::numeric_limits<float>::quiet_NaN()). Der zugewiesene Wert muss definiert sein (nicht std::numeric_limits<float>::quiet_NaN()). Sie können undefinierte Werte für RawFrame Instanzeigenschaften festlegen.

## IShape::set_X(float) Methode

Legt die X-Koordinate der oberen linken Ecke der Form fest, gemessen in Punkten. Schreiben Sie **float**.

```cpp
virtual void Aspose::Slides::IShape::set_X(float value)=0
```

## Hinweise

Der zurückgegebene Wert ist stets definiert und niemals [std::numeric_limits<float>::quiet_NaN()](../). Der zugewiesene Wert muss ebenfalls definiert sein; weisen Sie [std::numeric_limits<float>::quiet_NaN()](../) nur den Eigenschaften einer **RawFrame** Instanz zu.

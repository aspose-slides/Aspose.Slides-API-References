## Camera::GetRotation() Methode

Eine Rotation wird über die Verwendung einer Breitengrad-Koordinate, einer Längengrad-Koordinate und einer Umdrehung um die Achse definiert, wobei die Breitengrad- und Längengrad-Koordinaten verwendet werden. Erstes Element im Rückgabe-Array - Breitengrad, zweites - Längengrad, drittes - Umdrehung. Gibt null zurück, wenn keine Rotation definiert ist.

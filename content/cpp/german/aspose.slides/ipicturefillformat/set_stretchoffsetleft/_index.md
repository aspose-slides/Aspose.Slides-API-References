## IPictureFillFormat::set_StretchOffsetLeft(float) Methode

Legt die linke Kante des Füllrechtecks fest, das durch einen prozentualen Versatz von der linken Kante des Begrenzungsrahmens der Form definiert wird. Ein positiver Prozentsatz gibt eine Einziehung an, während ein negativer Prozentsatz eine Ausdehnung angibt. Schreiben **float**.

virtual void Aspose::Slides::IPictureFillFormat::set_StretchOffsetLeft(float value)=0

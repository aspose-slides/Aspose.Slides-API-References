## IPictureFillFormat::set_StretchOffsetTop(float) Methode

Setzt die obere Kante des Füllrechtecks, das durch einen prozentualen Versatz von der oberen Kante des Begrenzungsrahmens der Form definiert ist. Ein positiver Prozentsatz gibt eine Einziehung an, während ein negativer Prozentsatz eine Ausdehnung angibt. Schreiben **float**.

virtual void Aspose::Slides::IPictureFillFormat::set_StretchOffsetTop(float value)=0

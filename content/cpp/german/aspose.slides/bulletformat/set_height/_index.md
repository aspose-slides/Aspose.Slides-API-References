## BulletFormat::set_Height(float) Methode

Setzt die Aufzählungshöhe eines Absatzes ohne Vererbung. Der Wert std::numeric_limits<float>::quiet_NaN() bestimmt, dass die Aufzählung die Höhe vom ersten Teilabschnitt im Absatz erbt. Schreiben **float**.

```cpp
void Aspose::Slides::BulletFormat::set_Height(float value) override
```

## Hinweise

Ein negativer Höhenwert bedeutet, dass die Höhe in Punkten angegeben wird, und ein positiver Wert bedeutet, dass die Höhe ein Prozentsatz des umgebenden Textes ist.

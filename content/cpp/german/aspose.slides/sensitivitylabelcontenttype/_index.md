## SensitivityLabelContentType Enum


Stellt die Arten von Inhaltskennzeichnung dar, die auf ein Dokument angewendet werden sollen.

```cpp
enum class SensitivityLabelContentType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | 0 | Bedeutet, dass das Label standardmäßig oder automatisch angewendet wird. |
| Header | 1 | Bedeutet, dass das Label manuell angewendet wurde. Verwenden Sie jedes Sensitivitäts-Label, das vom Benutzer direkt angewendet wurde. |
| Footer | 2 | Bedeutet, dass das Label manuell angewendet wurde. Verwenden Sie jedes Sensitivitäts-Label, das vom Benutzer direkt angewendet wurde. |
| Watermark | 4 | Bedeutet, dass das Label manuell angewendet wurde. Verwenden Sie jedes Sensitivitäts-Label, das vom Benutzer direkt angewendet wurde. |
| Encryption | 8 | Bedeutet, dass das Label manuell angewendet wurde. Verwenden Sie jedes Sensitivitäts-Label, das vom Benutzer direkt angewendet wurde. |

## IBlobManagementOptions::get_IsTemporaryFilesAllowed() Methode


Diese Eigenschaft legt fest, ob temporäre Dateien beim Arbeiten mit BLOBs erstellt werden können, was den Speicherverbrauch erheblich verringert, aber Berechtigungen zum Erstellen von Dateien erfordert.

```cpp
virtual bool Aspose::Slides::IBlobManagementOptions::get_IsTemporaryFilesAllowed()=0
```

## Anmerkungen


Alle Dateien werden gelöscht, nachdem die Arbeit mit der Präsentation abgeschlossen ist.

## IBlobManagementOptions::set_IsTemporaryFilesAllowed(bool) Methode


Diese Eigenschaft definiert, ob temporäre Dateien während der Arbeit mit BLOBs erstellt werden können, was den Speicherverbrauch erheblich reduziert, jedoch Berechtigungen zum Erstellen von Dateien erfordert.

```cpp
virtual void Aspose::Slides::IBlobManagementOptions::set_IsTemporaryFilesAllowed(bool value)=0
```

## Anmerkungen


Alle Dateien werden gelöscht, nachdem die Arbeit mit der Präsentation abgeschlossen ist.

## ShapeAdjustmentType enum


Gibt verschiedene Arten von Formanpassungswerten an.

```cpp
enum class ShapeAdjustmentType
```

### Werte

| Name | Value | Description |
| --- | --- | --- |
| Custom | 0 | Benutzerdefinierter Typ (unbekannte Anpassung) |
| CornerSize | 1 | Steuert die Größe der Ecken. |
| LeftUpCorner | 2 | Steuert die obere linke Ecke. |
| RightUpCorner | 3 | Steuert die obere rechte Ecke. |
| UpCorners | 4 | Steuert die oberen Ecken. |
| DownCorners | 5 | Steuert die unteren Ecken. |
| Thickness | 6 | Steuert die Dicke der Figur. |
| ArrowTailThickness | 7 | Steuert die Dicke des Pfeilendes. |
| ArrowheadLength | 8 | Steuert die Länge der Pfeilspitze. |
| ArrowheadWidth | 9 | Steuert die Breite der Pfeilspitze. |
| Radius | 10 | Steuert die Größe des Radius der Form oder ihres Teils. |
| Depth | 11 | Steuert die Tiefe der Zahnradzähne oder den Abschrägungseffekt. |
| ShapePartOffset | 12 | Steuert den Versatz eines Teils der Figur relativ zu einem anderen. |
| Angle | 13 | Steuert den Winkel für die Figur oder ihren Teil. |
| StartAngle | 14 | Startwinkel für Kuchen- und Bogenformen. |
| EndAngle | 15 | Endwinkel für Kuchen- und Bogenformen. |
| StartPointX | 16 | Steuert die X-Position des Callout-Startpunkts. |
| StartPointY | 17 | Steuert die Y-Position des Callout-Startpunkts. |
| ConnectorPoint1X | 18 | Steuert die horizontale Position des Callout-Winkel-Anpassungspunkts. |

| ConnectorPoint2Y | 21 | Steuert die vertikale Position des Callout-Winkel-Anpassungspunkts. |
| ConnectorBendPositionX | 22 | Steuert die horizontale Position der Biegung im Verbinder. |
| ConnectorBendPositionY | 23 | Steuert die vertikale Position der Biegung im Verbinder. |
| AttachY | 24 | Steuert den vertikalen Ansetzpunkt der Figur. |
| AttachX | 25 | Steuert den horizontalen Ansetzpunkt der Figur. |
| CalloutHeight | 26 | Steuert die vertikale Größe des Callouts. |
| CalloutWidth | 27 | Steuert die horizontale Größe des Callouts. |

| VerticalSize | 32 | Steuert die vertikale Größe der Form. |

| Top | 34 | Steuert die Oberseite einer Form. |

## IDrawingGuide::get_Position() Methode

Gibt die Position des Zeichenleitfadens in Punkten vom oberen linken Eck der Folie zurück. Lese **float**.

```cpp
virtual float Aspose::Slides::IDrawingGuide::get_Position()=0
```

## Hinweise

Der typische Wertebereich reicht von null bis zur Folienhöhe für einen horizontalen Führer und von null bis zur Folienbreite für einen vertikalen Führer.

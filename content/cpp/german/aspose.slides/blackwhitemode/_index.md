## BlackWhiteMode enum

Bestimmt, wie eine farbige Form in Schwarz-weiß umgewandelt werden soll.

```cpp
enum class BlackWhiteMode : int8_t
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| NotDefined | -1 | Die Eigenschaft ist nicht gesetzt |
| Color | 0 | Das Objekt wird mit normaler Farbgebung dargestellt |
| Automatic | 1 | Das Objekt wird mit automatischer Farbgebung dargestellt |
| Gray | 2 | Das Objekt wird mit grauer Farbgebung dargestellt |
| LightGray | 3 | Das Objekt wird mit hellgrauer Farbgebung dargestellt |
| InverseGray | 4 | Das Objekt wird mit invertierter grauer Farbgebung dargestellt |
| GrayWhite | 5 | Das Objekt wird mit grauer und weißer Farbgebung dargestellt |
| BlackGray | 6 | Das Objekt wird mit schwarzer und grauer Farbgebung dargestellt |
| BlackWhite | 7 | Das Objekt wird mit schwarzer und weißer Farbgebung dargestellt |
| Black | 8 | Das Objekt wird nur mit schwarzer Farbgebung dargestellt |
| White | 9 | Das Objekt wird mit weißer Farbgebung dargestellt |
| Hidden | 10 | Das Objekt wird nicht dargestellt |

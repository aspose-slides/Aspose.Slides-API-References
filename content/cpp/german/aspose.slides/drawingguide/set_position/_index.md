## DrawingGuide::set_Position(float) Methode

Legt die Position der Zeichnungshilfe in Punkten vom oberen linken Eck der Folie fest. Schreiben Sie **float**.

```cpp
void Aspose::Slides::DrawingGuide::set_Position(float value) override
```

## Hinweise

Der typische Wertebereich reicht von Null bis zur Folienhöhe für eine horizontale Führungslinie und von Null bis zur Folienbreite für eine vertikale Führungslinie.

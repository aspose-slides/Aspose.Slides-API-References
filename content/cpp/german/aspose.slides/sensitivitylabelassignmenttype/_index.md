## SensitivityLabelAssignmentType Enum

Stellt die Zuweisungsmethode für die Sensitivitätskennzeichnung dar.

```cpp
enum class SensitivityLabelAssignmentType
```

### Values

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Standard | 0 | Verwenden für jede Sensitivitätskennzeichnung, die nicht direkt vom Benutzer angewendet wurde. Dies schließt alle Standardkennzeichnungen und automatisch angewendeten Kennzeichnungen ein. |
| Privileged | 1 | Verwenden für jede Sensitivitätskennzeichnung, die direkt vom Benutzer angewendet wurde. Dies schließt alle manuell angewendeten Sensitivitätskennzeichnungen ebenso ein wie empfohlene oder obligatorische Kennzeichnungen oder jede Funktion, bei der der Benutzer entscheidet, welche Sensitivitätskennzeichnung er anwenden soll. |

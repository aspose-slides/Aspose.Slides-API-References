## AudioPlayModePreset Enum

Konstanten, die festlegen, wie ein Ton abgespielt wird.

```cpp
enum class AudioPlayModePreset
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Mixed | -1 |  |
| Auto | 0 | Wird automatisch abgespielt. |
| OnClick | 1 | Wird nur abgespielt, wenn das Symbol angeklickt wird. |
| AllSlides | 2 | Wird auf allen Folien abgespielt. Bitte verwenden Sie stattdessen die [IAudioFrame::set_PlayAcrossSlides()](../iaudioframe/set_playacrossslides/)-Methode. |
| InClickSequence | 3 | Wird automatisch mit einem Klick abgespielt. |

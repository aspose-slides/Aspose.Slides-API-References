## ILoadOptions::set_BlobManagementOptions(System::SharedPtr\<IBlobManagementOptions\>) Methode


Stellt die Optionen dar, die zur Verwaltung des Verhaltens von Binary Large Objects (BLOBs) verwendet werden können, z. B. die Verwendung temporärer Dateien oder maximaler BLOB-Bytes im Speicher. Diese Optionen dienen dazu, das optimale Leistungs-/Speicherverbrauchs-Verhältnis für eine bestimmte Umgebung oder Anforderung einzustellen.

```cpp
virtual void Aspose::Slides::ILoadOptions::set_BlobManagementOptions(System::SharedPtr<IBlobManagementOptions> value)=0
```

## Hinweise


Ein Binary Large Object (BLOB) ist ein binärer Datensatz, der als einzelne Einheit gespeichert wird – d. h. BLOB kann selbst ein Audio, Video oder eine Präsentation sein.

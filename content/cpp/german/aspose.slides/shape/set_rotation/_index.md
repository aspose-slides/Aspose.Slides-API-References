## Shape::set_Rotation(float) Methode


Legt die Anzahl der Grad fest, um die die angegebene Form um die Z-Achse gedreht wird. Ein positiver Wert bedeutet eine Drehung im Uhrzeigersinn; ein negativer Wert bedeutet eine Drehung gegen den Uhrzeigersinn. Schreiben **float**.

```cpp
void Aspose::Slides::Shape::set_Rotation(float value) override
```

## Anmerkungen


Der zurückgegebene Wert ist stets definiert (ist nicht std::numeric_limits<float>::quiet_NaN()). Der zugewiesene Wert muss definiert sein (nicht std::numeric_limits<float>::quiet_NaN()). Sie können undefinierte Werte für Eigenschaften von RawFrame-Instanzen festlegen.

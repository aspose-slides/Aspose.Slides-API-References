## TextAutofitType enum

Stellt den Text-Autofit-Modus dar.

```cpp
enum class TextAutofitType : int8_t
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| NotDefined | -1 | Nicht definiert. |
| None | 0 | Keine automatische Anpassung. |
| Normal | 1 | Normale automatische Anpassung. Schriftgröße und Zeilenabstand werden reduziert, um in die Form zu passen. |

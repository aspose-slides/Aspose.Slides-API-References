## IDocumentProperties::get_ScaleCrop() Methode


Gibt den Anzeigemodus der Dokumentenminiatur an. Setzen Sie dieses Element auf **true**, um das Skalieren der Dokumentenminiatur an die Anzeige zu aktivieren. Setzen Sie dieses Element auf **false**, um das Beschneiden der Dokumentenminiatur zu aktivieren, sodass nur Abschnitte angezeigt werden, die in die Anzeige passen. Lese **bool**.

```cpp
virtual bool Aspose::Slides::IDocumentProperties::get_ScaleCrop()=0
```

## IDocumentProperties::get_LinksUpToDate() Methode


Gibt an, ob Hyperlinks in einem Dokument auf dem neuesten Stand sind. Setzen Sie dieses Element auf **true**, um anzugeben, dass Hyperlinks aktualisiert sind. Setzen Sie dieses Element auf **false**, um anzugeben, dass Hyperlinks veraltet sind. Lese **bool**.

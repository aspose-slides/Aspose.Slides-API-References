## IDocumentProperties::set_LinksUpToDate(bool) Methode

Gibt an, ob Hyperlinks in einem Dokument aktuell sind. Setzen Sie dieses Element auf **true**, um anzugeben, dass Hyperlinks aktualisiert sind. Setzen Sie dieses Element auf **false**, um anzugeben, dass Hyperlinks nicht mehr aktuell sind. Schreiben Sie **bool**.

virtual void Aspose::Slides::IDocumentProperties::set_LinksUpToDate(bool value)=0

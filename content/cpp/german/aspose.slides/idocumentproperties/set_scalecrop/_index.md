## IDocumentProperties::set_ScaleCrop(bool) Methode

Gibt den Anzeigemodus der Dokumentvorschau an. Setzen Sie dieses Element auf **true**, um die Skalierung der Dokumentvorschau an die Anzeige zu aktivieren. Setzen Sie dieses Element auf **false**, um das Zuschneiden der Dokumentvorschau zu aktivieren, sodass nur Abschnitte angezeigt werden, die in die Anzeige passen. Schreiben Sie **bool**.

virtual void Aspose::Slides::IDocumentProperties::set_ScaleCrop(bool value)=0

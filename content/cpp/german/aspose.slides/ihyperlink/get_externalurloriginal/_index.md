## IHyperlink::get_ExternalUrlOriginal() Methode

Stellt einen Hyperlink dar, der für diesen Abschnitt festgelegt ist, ohne Rücksicht auf den tatsächlichen Inhalt des Abschnitts.

```cpp
virtual System::String Aspose::Slides::IHyperlink::get_ExternalUrlOriginal()=0
```

## Anmerkungen

PowerPoint verhält sich bei Links und dem zugehörigen Text in einem Abschnitt speziell. Es ermöglicht, Text für den Hyperlink in Form einer gültigen URL zu erstellen, die von der tatsächlichen Adresse des Links abweicht. In diesem Fall wird, wenn Sie den Link im Bearbeitungsfenster anzeigen, dieser so geändert, dass er dem Textabschnitt entspricht. Diese Eigenschaft stellt den ursprünglichen Wert des Hyperlinks dar.

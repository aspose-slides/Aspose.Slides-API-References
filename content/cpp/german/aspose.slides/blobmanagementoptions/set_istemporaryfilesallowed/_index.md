## BlobManagementOptions::set_IsTemporaryFilesAllowed(bool) Methode

Diese Eigenschaft legt fest, ob temporäre Dateien beim Arbeiten mit BLOBs erstellt werden dürfen, was den Speicherverbrauch erheblich reduziert, aber Berechtigungen zum Erstellen von Dateien erfordert.

```cpp
void Aspose::Slides::BlobManagementOptions::set_IsTemporaryFilesAllowed(bool value) override
```

## Hinweise

Alle Dateien werden gelöscht, sobald die Arbeit mit der Präsentation abgeschlossen ist.

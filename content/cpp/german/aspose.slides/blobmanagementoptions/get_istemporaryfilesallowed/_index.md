## BlobManagementOptions::get_IsTemporaryFilesAllowed() Methode

Diese Eigenschaft definiert, ob temporäre Dateien erstellt werden können, während mit BLOBs gearbeitet wird, was den Speicherverbrauch erheblich verringert, aber Berechtigungen zum Erstellen von Dateien erfordert.

```cpp
bool Aspose::Slides::BlobManagementOptions::get_IsTemporaryFilesAllowed() override
```

## Hinweise

Alle Dateien werden gelöscht, nachdem die Arbeit mit der Präsentation abgeschlossen ist.

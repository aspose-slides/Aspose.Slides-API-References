## PathFillModeType enum

Gibt an, wie ein Pfad gefüllt werden soll

```cpp
enum class PathFillModeType : uint8_t
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | 0 | Keine Pfadfüllung Dies gibt an, dass der entsprechende Pfad keine Füllung hat. |
| Normal | 1 | Normale Pfadfüllung Dies gibt an, dass der entsprechende Pfad eine normal schattierte Farbe auf seine\\u2019s Füllung angewendet wird. |
| Lighten | 2 | Aufhellende Pfadfüllung Dies gibt an, dass der entsprechende Pfad eine leicht schattierte Farbe auf seine\\u2019s Füllung angewendet wird. |
| LightenLess | 3 | Weniger Aufhellende Pfadfüllung Dies gibt an, dass der entsprechende Pfad eine leicht hellere schattierte Farbe auf seine\\u2019s Füllung angewendet wird. |
| Darken | 4 | Abgedunkelte Pfadfüllung Dies gibt an, dass der entsprechende Pfad eine dunkler schattierte Farbe auf seine\\u2019s Füllung angewendet wird. |
| DarkenLess | 5 | Weniger Abgedunkelte Pfadfüllung Dies gibt an, dass der entsprechende Pfad eine leicht dunklere schattierte Farbe auf seine\\u2019s Füllung angewendet wird. |

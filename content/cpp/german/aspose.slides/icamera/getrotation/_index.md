## ICamera::GetRotation() Methode

Eine Rotation wird durch die Verwendung einer Breitengradkoordinate, einer Längengradkoordinate und einer Umdrehung um die Achse als Breitengrad- und Längengradkoordinaten definiert. Das erste Element im Rückgabearray ist die Breite, das zweite die Länge, das dritte die Umdrehung. Gibt null zurück, wenn keine Rotation definiert ist.

```cpp
virtual System::ArrayPtr<float> Aspose::Slides::ICamera::GetRotation()=0
```

### Rückgabewert

Array von Rotationswerten als **float**[].

## LineSketchType Enum

Gibt an, welchen Skizzentyp oder Effekt einer Form zugeordnet wurde.

```cpp
enum class LineSketchType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| NotDefined | -1 | Gibt an, dass ein Sketch-Effekt einer Form undefiniert ist. |
| None | 0 | Gibt an, dass eine Form keinen Sketch-Effekt hat. Dies ist äquivalent dazu, dass diese Eigenschaft leer ist. |
| Curved | 1 | Gibt an, dass eine Form den Curved-Effekt hat, der jede Kante der Form in eine große sanfte Kurve verwandelt. |
| Freehand | 2 | Gibt an, dass eine Form den Freehand-Effekt hat, der am ehesten einer unvollkommen gezeichneten Linie ähnelt. |
| Scribble | 3 | Gibt an, dass eine Form den Scribble-Effekt hat, der übertriebene Oszillationen aufweist, als wäre er absichtlich unordentlich gezeichnet. |

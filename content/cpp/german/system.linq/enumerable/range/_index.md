## Enumerable::Range(int32_t, int32_t) Methode

Erzeugt eine Sequenz von Ganzzahlen innerhalb eines angegebenen Bereichs.

```cpp
static SharedPtr<Collections::Generic::IEnumerable<int32_t>> System::Linq::Enumerable::Range(int32_t start, int32_t count)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| start | **int32_t** | Der Wert der ersten Ganzzahl in der Sequenz. |
| count | **int32_t** | Die Anzahl der zu erzeugenden aufeinanderfolgenden Ganzzahlen. |

### Rückgabewert

Ein IEnumerable von **int32_t**, das einen Bereich aufeinanderfolgender Ganzzahlen enthält.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [BitVector32](./bitvector32/) | Stellt einen einfachen leichten Bit-Vektor mit einfacher Ganzzahl- oder [Boolean](../system/boolean/)-Zugriff auf einen 32-Bit-Speicher bereit. |
| [NameValueCollection](./namevaluecollection/) | Sammlung von zugehörigen [String](../system/string/)-Schlüsseln und [String](../system/string/)-Werten, die entweder über den Schlüssel oder über den Index zugänglich sind. |
| [StringCollection](./stringcollection/) | Indizierte Liste von Zeichenketten. Objekte dieser Klasse sollten nur mit der Funktion [System::MakeObject()](../system/makeobject/) alloziert werden. Erzeugen Sie niemals eine Instanz dieses Typs auf dem Stack oder mit dem Operator new, da dies zu Laufzeitfehlern und/oder Assertionsfehlern führt. Wickeln Sie diese Klasse immer in einen Zeiger [System::SmartPtr](../system/smartptr/) ein und verwenden Sie diesen Zeiger, um sie als Argument an Funktionen zu übergeben. |
| [StringCollectionPtr](./stringcollectionptr/) | String-Sammlungspointer mit Zugriffsoperator. |
| [StringDictionary](./stringdictionary/) | [String](../system/string/)-zu-String-Wörterbuch. Objekte dieser Klasse sollten nur mit der Funktion [System::MakeObject()](../system/makeobject/) alloziert werden. Erzeugen Sie niemals eine Instanz dieses Typs auf dem Stack oder mit dem Operator new, da dies zu Laufzeitfehlern und/oder Assertionsfehlern führt. Packen Sie diese Klasse immer in einen Zeiger [System::SmartPtr](../system/smartptr/) ein und verwenden Sie diesen Zeiger, um sie als Argument an Funktionen zu übergeben. |

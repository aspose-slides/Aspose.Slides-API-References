## StringCollection::rbegin() Methode

Gibt einen Reverse-Iterator auf das erste Element des umgekehrten Containers zurück. Er entspricht dem letzten Element des nicht umgekehrten Containers. Ist der Container leer, ist der zurückgegebene Iterator gleich [rend()](../rend/).

```cpp
reverse_iterator System::Collections::Specialized::StringCollection::rbegin() noexcept
```

### Rückgabewert

Ein Iterator, der auf das letzte Element des Containers zeigt.

## StringCollection::cend() const Methode

Gibt einen Iterator zurück, der auf das Element zeigt, das dem letzten Element des Containers folgt. Dieses Element fungiert als Platzhalter; der Versuch, darauf zuzugreifen, führt zu undefiniertem Verhalten.

```cpp
const_iterator System::Collections::Specialized::StringCollection::cend() const noexcept
```

### Rückgabewert

Ein Iterator, der auf das theoretisch const-qualifizierte Element zeigt, das nach dem letzten Element des Containers platziert ist.

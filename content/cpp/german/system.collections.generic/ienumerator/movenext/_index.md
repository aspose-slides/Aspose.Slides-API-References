## IEnumerator::MoveNext() Methode


Bewegt den Enumerator zum nächsten Element. Wenn zuvor kein Element referenziert wurde, wird die Referenz auf das erste verfügbare Element gesetzt. Wenn das Ende des Containers erreicht wurde, geschieht nichts.

```cpp
virtual bool System::Collections::Generic::IEnumerator<T>::MoveNext()=0
```


### Rückgabewert

True wenn nach dem Aufruf ein Element verfügbar ist, sonst false.

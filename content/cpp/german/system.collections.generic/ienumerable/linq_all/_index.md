## IEnumerable::LINQ_All(std::function\<bool(T)>) Methode

Bestimmt, ob alle Elemente einer Sequenz eine Bedingung erfüllen.

```cpp
bool System::Collections::Generic::IEnumerable<T>::LINQ_All(std::function<bool(T)> predicate)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| predicate | std::function\<**bool**(T)> | Eine Funktion, um jedes Element auf eine Bedingung zu testen. |

### Rückgabewert

true wenn jedes Element der Quellsequenz den Test im angegebenen Prädikat besteht oder wenn die Sequenz leer ist; andernfalls false.

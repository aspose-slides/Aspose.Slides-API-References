## IEnumerable::begin() Methode

Ermittelt einen Iterator, der auf das erste Element (falls vorhanden) der Sammlung zeigt. Dieser Iterator kann nicht verwendet werden, um ein referenziertes Objekt zu ändern, weil [GetEnumerator()](../getenumerator/) ein Kopie-Objekt von T zurückgibt.

```cpp
iterator System::Collections::Generic::IEnumerable<T>::begin()
```

### Rückgabewert

Ein Iterator, der auf das erste Element (falls vorhanden) der Sammlung zeigt

## IEnumerable::begin() const Methode

Ermittelt einen Iterator, der auf das erste Element (falls vorhanden) der const-qualifizierten Instanz der Sammlung zeigt.

```cpp
const_iterator System::Collections::Generic::IEnumerable<T>::begin() const
```

### Rückgabewert

Ein Iterator, der auf das erste Element (falls vorhanden) der const-qualifizierten Instanz der Sammlung zeigt

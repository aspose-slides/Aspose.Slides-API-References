## IEnumerable::end() Methode


Liefert einen Iterator, der direkt nach dem letzten Element (falls vorhanden) der Sammlung zeigt. Dieser Iterator kann nicht verwendet werden, um ein referenziertes Objekt zu ändern, weil [GetEnumerator()](../getenumerator/) ein Kopie-Objekt von T zurückgibt.

```cpp
iterator System::Collections::Generic::IEnumerable<T>::end()
```


### Rückgabewert

Ein Iterator, der direkt nach dem letzten Element (falls vorhanden) der Sammlung zeigt

## IEnumerable::end() const Methode


Liefert einen Iterator, der direkt nach dem letzten Element (falls vorhanden) der const-qualifizierten Instanz der Sammlung zeigt.

```cpp
const_iterator System::Collections::Generic::IEnumerable<T>::end() const
```


### Rückgabewert

Ein Iterator, der direkt nach dem letzten Element (falls vorhanden) der const-qualifizierten Instanz der Sammlung zeigt

## SortedList::rend() Methode

Erhält einen Reverse-Iterator für ein nicht existierendes Element vor dem Beginn der Sammlung.

```cpp
reverse_iterator System::Collections::Generic::SortedList<TKey, TValue>::rend() noexcept
```

### Rückgabewert

Ein Reverse-Iterator, der auf das theoretische Element zeigt, das dem ersten Element in der Map vorausgeht (letztes im Reverse).

## SortedList::rend() const Methode

Erhält einen Reverse-Iterator für ein nicht existierendes Element vor dem Beginn der const-qualifizierten Sammlung.

```cpp
const_reverse_iterator System::Collections::Generic::SortedList<TKey, TValue>::rend() const noexcept
```

### Rückgabewert

Ein Reverse-Iterator, der auf das theoretische Element zeigt, das dem ersten Element in der const-qualifizierten Map vorausgeht (letztes im Reverse).

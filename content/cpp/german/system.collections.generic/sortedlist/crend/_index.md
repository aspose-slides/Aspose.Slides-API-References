## SortedList::crend() const Methode

Liefert einen Reverse-Iterator für ein nicht vorhandenes const-qualifiziertes Element vor dem Start der Sammlung zurück.

```cpp
const_reverse_iterator System::Collections::Generic::SortedList<TKey, TValue>::crend() const noexcept
```

### Rückgabewert

Ein Reverse-Iterator, der auf das theoretische const-qualifizierte Element zeigt, das dem ersten Element in der Map vorausgeht (letztes im Reverse).

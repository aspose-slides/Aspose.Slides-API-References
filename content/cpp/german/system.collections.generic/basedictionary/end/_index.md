## BaseDictionary::end() const Methode

Gibt einen Iterator zum KVPair-Wrapper für das Schlüssel-Wert-Element zurück, das dem letzten Element des Containers folgt. Implementiert im C#-Stil – der Iterator sollte das KVPair-Objekt mit der get_Key()- und get_Value()-Schnittstelle zurückgeben. Dieses Element fungiert als Platzhalter; ein Zugriff darauf führt zu undefiniertem Verhalten.

```cpp
const_iterator System::Collections::Generic::BaseDictionary<Map>::end() const noexcept
```

### Rückgabewert

Ein Iterator, der auf das theoretische Element zeigt, das nach dem Endelement der Sammlung platziert wäre.

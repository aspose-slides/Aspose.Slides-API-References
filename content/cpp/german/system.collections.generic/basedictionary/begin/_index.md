## BaseDictionary::begin() const Methode


Gibt einen Iterator zurück zum KVPair-Wrapper für das Schlüssel-Wert-Element des Containers. Implementiert im C#-Stil – der Iterator sollte das KVPair-Objekt mit der Schnittstelle get_Key() und get_Value() zurückgeben. Wenn der Container leer ist, ist der zurückgegebene Iterator gleich [end()](../../ienumerable/end/).

```cpp
const_iterator System::Collections::Generic::BaseDictionary<Map>::begin() const noexcept
```


### Rückgabewert

Ein Iterator, der auf das erste Element der Sammlung zeigt.

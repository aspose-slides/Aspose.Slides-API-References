## List::rend() Methode

Ermittelt einen Reverse-Iterator für ein nicht vorhandenes Element vor dem Beginn der Sammlung.

```cpp
reverse_iterator System::Collections::Generic::List<T>::rend() noexcept
```

### Rückgabewert

Ein Reverse-Iterator, der auf das theoretische Element zeigt, das dem ersten Element der Liste vorausgeht (letztes im Reverse).

### Rückgabewert

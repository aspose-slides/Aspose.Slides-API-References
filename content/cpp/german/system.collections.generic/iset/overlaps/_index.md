## ISet::Overlaps(IEnumerablePtr) Methode

Überprüft, ob das Set mit einem anderen Container überlappt.

```cpp
virtual bool System::Collections::Generic::ISet<T>::Overlaps(IEnumerablePtr other)=0
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| other | [IEnumerablePtr](../ienumerableptr/) | Container, gegen den geprüft wird. |

### Rückgabewert

Wahr, wenn das aktuelle Set und **other** mindestens ein gemeinsames Element haben, andernfalls falsch.

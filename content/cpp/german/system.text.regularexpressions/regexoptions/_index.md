## RegexOptions Enum

[Regex](../regex/) Optionen.

```cpp
enum class RegexOptions
```

### Values

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | 0 | Standardverhalten. |
| Compiled | 1 | Regex für bessere Leistung kompilieren. Standardmäßig immer durchgeführt. |
| CultureInvariant | 2 | Kulturunabhängige Übereinstimmung verwenden. Ignoriert. |
| ECMAScript | 4 | ECMAScript-Syntax verwenden. Ignoriert. |
| ExplicitCapture | 8 | Nur explizites Erfassen. Ignoriert. |
| IgnoreCase | 16 | Groß-/Kleinschreibung bei Übereinstimmung ignorieren. |
| IgnorePatternWhitespace | 32 | Leerzeichen im Muster ignorieren. Nicht unterstützt. |
| Multiline | 64 | Behandle '^' und '$' als Anfang und Ende einer Zeile, nicht des gesamten Strings. |
| RightToLeft | 128 | Übereinstimmung von rechts nach links. Nicht unterstützt. |
| Singleline | 256 | Lässt '.' jedes Zeichen ohne Ausnahmen matchen (normalerweise werden Zeilenumbrüche nicht gematcht). |

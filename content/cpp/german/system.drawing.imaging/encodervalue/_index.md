## EncoderValue Enum

Gibt den an einen JPEG- oder TIFF-Bild-Encoder übergebenen Parameterwert an.

```cpp
enum class EncoderValue
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| ColorTypeCMYK | 0 | Der CMYK-Farbraum. |
| ColorTypeYCCK | 1 | Der YCCK-Farbraum. |
| CompressionLZW | 2 | Die LZW-Komprimierungsmethode. |
| CompressionCCITT3 | 3 | Gibt die CCITT3-Komprimierungsmethode für ein TIFF-Bild an. |
| CompressionCCITT4 | 4 | Gibt die CCITT4-Komprimierungsmethode für ein TIFF-Bild an. |
| CompressionRle | 5 | Gibt die RLE-Komprimierungsmethode für ein TIFF-Bild an. |
| CompressionNone | 6 | Gibt an, dass für ein TIFF-Bild keine Kompression verwendet wird. |
| ScanMethodInterlaced | 7 | Interlaced-Modus. |
| ScanMethodNonInterlaced | 8 | Nicht-interlaced Modus. |
| VersionGif87 | 9 | Gibt Version 87 für ein TIFF-Bild an. |
| VersionGif89 | 10 | Gibt Version 89a für ein GIF-Bild an. |
| RenderProgressive | 11 | Progressiver Modus. |
| RenderNonProgressive | 12 | Nicht-progressiver Modus. |
| TransformRotate90 | 13 | Gibt eine verlustfreie 90-Grad-Drehung im Uhrzeigersinn für ein JPEG-Bild an. |
| TransformRotate180 | 14 | Gibt eine verlustfreie 180-Grad-Drehung für ein JPEG-Bild an. |
| TransformRotate270 | 15 | Gibt eine verlustfreie 270-Grad-Drehung im Uhrzeigersinn für ein JPEG-Bild an. |
| TransformFlipHorizontal | 16 | Gibt eine verlustfreie horizontale Spiegelung für ein JPEG-Bild an. |
| TransformFlipVertical | 17 | Gibt eine verlustfreie vertikale Spiegelung für ein JPEG-Bild an. |
| MultiFrame | 18 | Mehrframe-Kodierung. |
| LastFrame | 19 | Der letzte Frame eines Mehrframe-Bildes. |
| Flush | 20 | Das Encoder-Objekt soll geschlossen werden. |
| FrameDimensionTime | 21 | Gibt die Zeit-Frame-Dimension für ein GIF-Bild an. |
| FrameDimensionResolution | 22 | Die Auflösungs-Frame-Dimension. |
| FrameDimensionPage | 23 | Gibt die Seiten-Frame-Dimension für ein TIFF-Bild an. |

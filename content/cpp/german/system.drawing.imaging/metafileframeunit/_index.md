## MetafileFrameUnit Enum

Gibt die Einheit der Messung für das Rechteck an, das zum Skalieren und Positionieren einer Metadatei verwendet wird.

```cpp
enum class MetafileFrameUnit
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Pixel | n/a | Gibt einen Pixel als Maßeinheit an. |
| Point | n/a | Gibt den Punkt eines Druckers als Maßeinheit an. |
| Inch | n/a | Gibt einen Zoll als Maßeinheit an. |
| Document | n/a | Gibt 1/300 Zoll als Maßeinheit an. |
| Millimeter | n/a | Gibt einen Millimeter als Maßeinheit an. |
| GdiCompatible | n/a | Gibt .01 Millimeter als Maßeinheit an. Wird zur Kompatibilität mit GDI bereitgestellt. |

## ColorAdjustType Enum

Gibt an, welche Objekte Farbanpassungsinformationen verwenden.

```cpp
enum class ColorAdjustType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Default | 0 | Definiert Farbanpassungsinformationen, die von allen Objekten verwendet werden, die keine eigenen Farbanpassungsinformationen besitzen. |
| Bitmap | 1 | Definiert Farbanpassungsinformationen für [Bitmap](../../system.drawing/bitmap/)-Objekte. |
| Brush | 2 | Definiert Farbanpassungsinformationen für [Brush](../../system.drawing/brush/)-Objekte. |
| Pen | 3 | Definiert Farbanpassungsinformationen für [Pen](../../system.drawing/pen/)-Objekte. |
| Text | 4 | Definiert Farbanpassungsinformationen für Text. |
| Count | 5 | Gibt die Anzahl der angegebenen Typen an. |
| Any | 6 | Gibt die Anzahl der angegebenen Typen an. |

## EmfType enum

Specifiziert die Typen der Datensätze, die in einer EMF-Datei platziert werden.

```cpp
enum class EmfType
```

### Werte

| Name | Value | Description |
| --- | --- | --- |
| EmfOnly | n/a | [Windows](../../system.windows/) erweiterte Metadatei. Enthält GDI-Befehle. Metadateien dieses Typs werden als EMF-Datei bezeichnet. |
| EmfPlusOnly | n/a | [Windows](../../system.windows/) erweiterte Metadatei plus. Enthält GDI+-Befehle. Metadateien dieses Typs werden als EMF+-Datei bezeichnet. |
| EmfPlusDual | n/a | Dual [Windows](../../system.windows/) erweiterte Metadatei. Enthält äquivalente GDI- und GDI+-Befehle. Metadateien dieses Typs werden als EMF+-Datei bezeichnet. |

## ImageLockMode Enum

Gibt die Eigenschaften des gesperrten Bildbereichs an.

```cpp
enum class ImageLockMode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| ReadOnly | 1 | Der Bereich ist nur zum Lesen gesperrt. |
| WriteOnly | 2 | Der Bereich ist nur zum Schreiben gesperrt. |
| ReadWrite | n/a | Der Bereich ist sowohl zum Lesen als auch zum Schreiben gesperrt. |
| UserInputBuffer | 4 | Das Bild befindet sich in einem Benutzereingabepuffer, dessen Zugriff vom Benutzer gesteuert wird. |

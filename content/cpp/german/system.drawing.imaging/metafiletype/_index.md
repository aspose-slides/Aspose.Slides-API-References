## MetafileType enum

Stellt einen Typ einer Grafik-Metadatei dar.

```cpp
enum class MetafileType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Invalid | 0 | Ungültige Metadatei. |
| Wmf | 1 | Standard-WMF. |
| WmfPlaceable | 2 | Platzierbares [Metafile](../metafile/)-Format. |
| Emf | 3 | EMF (nicht EMF+). |
| EmfPlusOnly | 4 | EMF+ ohne duale, abwärtskompatible Datensätze. |
| EmfPlusDual | 5 | EMF+ mit dualen, abwärtskompatiblen Datensätzen. |

## CultureTypes Enum

Bitmasken-Einträge für Kulturkategorien.

```cpp
enum class CultureTypes : int32_t
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| NeutralCultures | 1 | Kultur, die für die Sprache spezifisch ist, aber nicht für Land oder Region. |
| SpecificCultures | 2 | Spezifisch für Land/Region. |
| InstalledWin32Cultures | 4 | In dem Betriebssystem installierte Kulturen. |
| AllCultures | n/a | Alle verfügbaren Kulturen. |
| UserCustomCulture | 8 | Benutzerdefinierte Kulturen. |
| ReplacementCultures | 16 | Benutzerdefinierte Ersetzungen für vorhandene Kulturen. |
| WindowsOnlyCultures | 32 | Veraltet. |
| FrameworkCultures | 64 | Veraltet. |

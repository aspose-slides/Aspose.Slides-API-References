## CultureInfo::ReadOnly(const CultureInfoPtr\&) Methode

Gibt eine schreibgeschützte Version der Kultur zurück.

```cpp
static CultureInfoPtr System::Globalization::CultureInfo::ReadOnly(const CultureInfoPtr &culture_info)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| culture_info | const [CultureInfoPtr](../../cultureinfoptr/)\& | Kultur, die als schreibgeschützt eingewickelt wird. |

### Rückgabewert

Wenn **culture_info** schreibgeschützt ist, wird dasselbe Objekt zurückgegeben, andernfalls wird eine schreibgeschützte Kopie erstellt.

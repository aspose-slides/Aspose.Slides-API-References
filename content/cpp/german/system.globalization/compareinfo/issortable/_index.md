## CompareInfo::IsSortable(char16_t) Methode

Überprüft, ob ein angegebenes Zeichen sortierbar ist.

```cpp
static bool System::Globalization::CompareInfo::IsSortable(char16_t ch)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| ch | char16_t | Unicode-Zeichen. |

### Rückgabewert

True, wenn **ch** sortierbar ist; sonst false.

## CompareInfo::IsSortable(const String\&) Methode

Überprüft, ob ein angegebener String sortierbar ist.

```cpp
static bool System::Globalization::CompareInfo::IsSortable(const String &text)
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| text | const [String](../../../system/string/)\& | Ein String. |

### Rückgabewert

True, wenn **text** nicht leer ist und alle Zeichen in **text** sortierbar sind; sonst false.

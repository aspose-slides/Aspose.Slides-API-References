## TimeSpanStyles Enum

Gibt die Formatoptionen an, die die Zeichenfolgenanalyse für Methoden anpassen, die eine Zeichenkettenrepräsentation eines Zeitintervalls in ein Timespan-Objekt konvertieren.

```cpp
enum class TimeSpanStyles : int32_t
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | 0 | Das von der inptut string dargestellte Zeitintervall wird nur dann als negatives Zeitintervall interpretiert, wenn ein negatives Vorzeichen vorhanden ist. |
| AssumeNegative | 1 | Das von der inptut string dargestellte Zeitintervall wird immer als negatives Zeitintervall interpretiert. |

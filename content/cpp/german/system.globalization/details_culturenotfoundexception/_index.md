## Details_CultureNotFoundException Klasse


CultureNotFoundException wird ausgelöst, wenn versucht wird, eine Kultur zu erstellen, die nicht verfügbar ist. Erstellen Sie niemals manuell Instanzen dieser Klasse. Verwenden Sie stattdessen die Klasse CultureNotFoundException. Wickeln Sie die Instanzen der Klasse CultureNotFoundException niemals in [System::SmartPtr](../../system/smartptr/) ein.

```cpp
class Details_CultureNotFoundException : public System::Details_ArgumentException
```

## WarningType Enum

Stellt einen Warnungstyp dar.

```cpp
enum class WarningType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| SourceFileCorruption | 0 | Ein Problem wurde im Quelldokument erkannt, das die Wahrscheinlichkeit stark erhöht, dass das Dokument nicht geöffnet werden kann, wenn es im Originalformat gespeichert wird. |
| DataLoss | 1 | Text/Diagramm/Bild oder andere Daten werden entweder nach dem Laden im Dokumentbaum oder nach dem Speichern im erstellten Dokument vollständig fehlen. |
| MajorFormattingLoss | 2 | Großer Formatierungsverlust. |
| MinorFormattingLoss | 3 | Geringer Formatierungsverlust. |
| CompatibilityIssue | 4 | Dies ist ein bekanntes Problem, das verhindert, dass das Dokument von bestimmten Benutzeragenten oder älteren Versionen von Benutzeragenten geöffnet wird. |
| UnexpectedContent | 99 | Einige Inhalte im Quelldokument konnten nicht erkannt werden (d. h. werden nicht unterstützt); dies kann zu Problemen führen oder zu Daten-/Formatierungsverlust führen. |

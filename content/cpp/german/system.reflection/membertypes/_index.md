## MemberTypes enum


Kennzeichnet jeden Typ von Mitglied.

```cpp
enum class MemberTypes
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Constructor | 1 | Gibt an, dass das Mitglied ein Konstruktor ist. |
| Event | 2 | Gibt an, dass das Mitglied ein Ereignis ist. |
| Field | 4 | Gibt an, dass das Mitglied ein Feld ist. |
| Method | 8 | Gibt an, dass das Mitglied eine Methode ist. |
| Property | 16 | Gibt an, dass das Mitglied eine Eigenschaft ist. |
| TypeInfo | 32 | Gibt an, dass das Mitglied ein Typ ist. |
| Custom | 64 | Gibt an, dass das Mitglied ein benutzerdefinierter Mitgliedstyp ist. |
| NestedType | 128 | Gibt an, dass das Mitglied ein verschachtelter Typ ist. |
| All | 191 | Gibt alle Mitgliedstypen an. |

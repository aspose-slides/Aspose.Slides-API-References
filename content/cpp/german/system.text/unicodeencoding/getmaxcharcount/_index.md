## UnicodeEncoding::GetMaxCharCount(int) Methode

Ermittelt die maximale Anzahl von Zeichen, die zum Dekodieren einer angegebenen Anzahl von Bytes benötigt werden.

```cpp
int System::Text::UnicodeEncoding::GetMaxCharCount(int byte_count) override
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| byte_count | int | Anzahl der zu dekodierenden Bytes. |

### Rückgabewert

Anzahl der Bytes, die ausreicht, um jede Zeichenfolge mit gegebener Zeichenlänge zu dekodieren.

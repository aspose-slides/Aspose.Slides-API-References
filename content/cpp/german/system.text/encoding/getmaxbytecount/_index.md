## Encoding::GetMaxByteCount(int) Methode

Ermittelt die maximale Anzahl von Bytes, die zum Codieren einer angegebenen Anzahl von Zeichen benötigt werden.

```cpp
virtual int System::Text::Encoding::GetMaxByteCount(int char_count)=0
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| char_count | int | Anzahl der zu codierenden Zeichen. |

### Rückgabewert

Anzahl der Bytes, die ausreicht, um jede Zeichenkette mit gegebener Zeichenlänge zu codieren.

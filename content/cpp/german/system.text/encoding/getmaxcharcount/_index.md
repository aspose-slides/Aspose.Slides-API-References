## Encoding::GetMaxCharCount(int) Methode

Ermittelt die maximale Anzahl von Zeichen, die zum Dekodieren einer angegebenen Anzahl von Bytes benötigt werden.

```cpp
virtual int System::Text::Encoding::GetMaxCharCount(int byte_count)=0
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| byte_count | int | Anzahl der zu dekodierenden Bytes. |

### Rückgabewert

Anzahl der Bytes, die ausreicht, um jede Zeichenkette mit gegebener Zeichenlänge zu dekodieren.

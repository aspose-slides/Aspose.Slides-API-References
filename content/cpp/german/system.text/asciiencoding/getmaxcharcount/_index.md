## ASCIIEncoding::GetMaxCharCount(int) Methode

Ermittelt die maximale Anzahl von Zeichen, die zum Dekodieren einer angegebenen Anzahl von Bytes erforderlich sind.

```cpp
int System::Text::ASCIIEncoding::GetMaxCharCount(int byteCount) override
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| byteCount | int | [Byte](../../../system/byte/) Anzahl. |

### Rückgabewert

Anzahl der Zeichen, die ausreicht, um jede beliebige Sequenz von **byteCount** Zeichen aufzunehmen.

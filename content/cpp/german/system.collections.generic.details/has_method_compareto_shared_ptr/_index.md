## has_method_compareto_shared_ptr struct

Überprüft, ob die Methode CompareTo(SharedPtr<T>) im angegebenen Typ existiert. Falls ja, erbt std::true_type, andernfalls erbt std::false_type. Kann in std::enable_if verwendet werden.

```cpp
template<typename T,typename Sfinae>class has_method_compareto_shared_ptr : public std::false_type
```

### Template-Parameter

| Parameter | Beschreibung |
| --- | --- |
| T | Typ, um das Vorhandensein der Equals-Methode zu prüfen. |
| Sfinae | Formaler Template-Parameter, damit SFINAE funktioniert. |

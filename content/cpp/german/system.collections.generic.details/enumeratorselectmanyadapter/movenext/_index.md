## EnumeratorSelectManyAdapter::MoveNext() Methode


Bewegt den Enumerator zum nächsten Element. Wenn zuvor kein Element referenziert wurde, wird die Referenz auf das erste verfügbare Element gesetzt. Wenn das Ende des Containers erreicht wurde, passiert nichts.

```cpp
bool System::Collections::Generic::Details::EnumeratorSelectManyAdapter<Source, Result>::MoveNext() override
```


### Rückgabewert

True, wenn ein Element nach dem Aufruf verfügbar ist, false andernfalls.

## EnumeratorOfTypeAdapter::MoveNext() Methode

Bewegt den Enumerator zum nächsten Element. Wenn vorher kein Element referenziert wurde, wird die Referenz auf das erste verfügbare Element gesetzt. Wenn das Ende des Containers erreicht ist, passiert nichts.

```cpp
bool System::Collections::Generic::Details::EnumeratorOfTypeAdapter<Source, Result>::MoveNext() override
```

### Rückgabewert

True, wenn nach dem Aufruf ein Element verfügbar ist, false andernfalls.

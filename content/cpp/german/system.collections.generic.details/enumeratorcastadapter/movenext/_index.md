## EnumeratorCastAdapter::MoveNext() Methode

Bewegt den Enumerator zum nächsten Element. Wenn vorher kein Element referenziert wurde, wird die Referenz auf das erste verfügbare Element gesetzt. Wenn das Ende des Containers erreicht wurde, wird nichts unternommen.

```cpp
bool System::Collections::Generic::Details::EnumeratorCastAdapter<Source, Result>::MoveNext() override
```

### Rückgabewert

True, wenn nach dem Aufruf ein Element verfügbar ist, sonst false.

## XmlDsigC14NTransform Klasse

Representiert die C14N XML-Kanonisierungstransformation für eine digitale Signatur ohne Kommentare. Objekte dieser Klasse sollten nur mit der [System::MakeObject()](../../system/makeobject/) Funktion allokiert werden. Erzeugen Sie niemals eine Instanz dieses Typs auf dem Stack oder mit operator new, da dies zu Laufzeitfehlern und/oder Assertionsfehlern führt. Wickeln Sie diese Klasse immer in einen [System::SmartPtr](../../system/smartptr/) Zeiger und verwenden Sie diesen Zeiger, um ihn als Argument an Funktionen zu übergeben.

```cpp
class XmlDsigC14NTransform : public System::Security::Cryptography::Xml::Transform
```

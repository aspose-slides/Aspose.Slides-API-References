## XmlDsigC14NWithCommentsTransform Klasse

Stellt die C14N XML-Kanonisierungstransformation für eine digitale Signatur mit Kommentaren dar. Objekte dieser Klasse sollten nur über die Funktion [System::MakeObject()](../../system/makeobject/) zugewiesen werden. Erstellen Sie niemals eine Instanz dieses Typs auf dem Stack oder mit dem Operator new, da dies zu Laufzeitfehlern und/oder Assertion-Fehlern führt. Wickeln Sie diese Klasse immer in einen [System::SmartPtr](../../system/smartptr/)-Zeiger ein und verwenden Sie diesen Zeiger, um ihn als Argument an Funktionen zu übergeben.

```cpp
class XmlDsigC14NWithCommentsTransform : public System::Security::Cryptography::Xml::XmlDsigC14NTransform
```

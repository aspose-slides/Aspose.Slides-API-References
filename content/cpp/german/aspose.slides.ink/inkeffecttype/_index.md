## InkEffectType enum

Specifiziert eine Menge vordefinierter visueller Effekte für die Tintenrenderung. Jeder Effekt entspricht einer Textur oder einem Bild, das ein stilisiertes Tintenbild simuliert. Diese Werte können verwendet werden, um den visuellen Stil digitaler Tintenstriche während der Renderung anzupassen.

```cpp
enum class InkEffectType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| NotDefined | -1 | Der Effekt ist nicht definiert oder unbekannt. Dieser Wert kann als Standard oder Fallback verwendet werden. |
| Bronze | 0 | Eine warme, bräunliche metallische Textur, die bronzefarbe Tinte nachahmt. |
| Galaxy | 1 | Eine bunte, schimmernde Textur, die einer Galaxie ähnelt, mit kosmischen Farbtönen. |
| Gold | 2 | Eine helle, metallische Goldtextur, die Tintenstrichen ein luxuriöses Aussehen verleiht. |
| Lava | 3 | Eine feurige Textur, die flüssiger Lava ähnelt, mit roten und orangefarbenen Tönen. |
| Ocean | 4 | Eine tiefblaue, flüssigkeitsähnliche Textur, die Meereswellen oder wasserbasierte Tinte nachahmt. |
| RainbowGlitter | 5 | Ein bunter, funkelnder Regenbogen-Glitzer-Effekt, der für festliche oder lebendige Tintenstriche verwendet wird. |
| RoseGold | 6 | Eine weiche pink-goldene Mischung, ähnlich wie Roségold, für elegante Tintenstriche. |
| Silver | 7 | Eine kühle, metallische Silbertextur, die klassische Silbertinte simuliert. |

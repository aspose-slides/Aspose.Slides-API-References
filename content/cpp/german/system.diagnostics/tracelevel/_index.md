## TraceLevel Enum

Gibt an, welche Nachrichten für die Klassen [System.Diagnostics.Debug](../debug/), [System.Diagnostics.Trace](../trace/) und System.Diagnostics.TraceSwitch ausgegeben werden.

```cpp
enum class TraceLevel
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Off | 0 | Gibt keine Tracing- und Debug-Nachrichten aus. |
| Error | 1 | Gibt Fehlermeldungen aus. |
| Warning | 2 | Gibt Warnungen und Fehlermeldungen aus. |
| Info | 3 | Gibt Informationsnachrichten, Warnungen und Fehlermeldungen aus. |
| Verbose | 4 | Gibt alle Debug- und Tracing-Nachrichten aus. |

## TraceEventType Enum

Identifiziert den Typ des Ereignisses, das die Trace ausgelöst hat.

```cpp
enum class TraceEventType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Critical | 1 | Fataler Fehler oder Anwendungsabsturz. |
| Error | 2 | Wiederherstellbarer Fehler. |
| Warning | 4 | Nichtkritisches Problem. |
| Information | 8 | Informationsnachricht. |
| Verbose | 16 | Debug-Trace. |
| Start | 256 | Beginn einer logischen Operation. |
| Stop | 512 | Ende einer logischen Operation. |
| Suspend | 1024 | Unterbrechung einer logischen Operation. |
| Resume | 2048 | Fortsetzung einer logischen Operation. |
| Transfer | 4096 | Änderung der Korrelation-Identität. |

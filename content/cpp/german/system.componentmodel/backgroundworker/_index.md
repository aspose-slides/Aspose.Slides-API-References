## BackgroundWorker Klasse


Objekte dieser Klasse sollten nur mithilfe der Funktion [System::MakeObject()](../../system/makeobject/) alloziert werden. Erstellen Sie niemals eine Instanz dieses Typs auf dem Stack oder mit dem Operator new, da dies zu Laufzeitfehlern und/oder Assert-Fehlern führt. Kapseln Sie diese Klasse immer in einen [System::SmartPtr](../../system/smartptr/)-Zeiger ein und benutzen Sie diesen Zeiger, um ihn als Argument an Funktionen zu übergeben.

```cpp
class BackgroundWorker : public System::ComponentModel::Component
```

## Details_Win32Exception Klasse


Wirft eine Ausnahme für einen Win32-Fehlercode. Erstellen Sie niemals manuell Instanzen dieser Klasse. Verwenden Sie stattdessen die Klasse Win32Exception. Umwickeln Sie niemals die Instanzen der Klasse Win32Exception in [System::SmartPtr](../../system/smartptr/).

```cpp
class Details_Win32Exception : public System::Details_ExceptionWithErrorCode<Details_SystemException>
```

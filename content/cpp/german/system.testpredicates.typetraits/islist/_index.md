## IsList typedef

Überprüft, ob der Typ eine [System::Collections::Generic::List](../../system.collections.generic/list/)-Spezialisierung ist. Falls ja, wird das Member value auf true gesetzt, andernfalls wird es auf false gesetzt.

```cpp
using System::TestPredicates::TypeTraits::IsList = typedef std::is_same<T, System::Collections::Generic::List<typename T::ValueType> >
```

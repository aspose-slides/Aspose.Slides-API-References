## defaultThreshold Feld


Standardwert für den Schwellenwert zur Erstellung des Lookup-Wörterbuchs.

```cpp
static const int System::Collections::ObjectModel::KeyedCollection<TKey, TItem>::defaultThreshold
```

## Bemerkungen


Setzt den Standard-Lookup-Wörterbuch-Erstellungs-Schwellenwert auf null, kann für konkrete Typen überschrieben werden.

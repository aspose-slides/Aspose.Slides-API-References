## IPPacketInformation Klasse

Stellt Informationen über das Paket dar. Objekte dieser Klasse sollten nur über die Funktion [System::MakeObject()](../../system/makeobject/) alloziert werden. Erstellen Sie niemals eine Instanz dieses Typs im Stack oder mittels operator new, da dies zu Laufzeitfehlern und/oder Assertion-Fehlern führen kann. Umwickeln Sie diese Klasse immer in einen [System::SmartPtr](../../system/smartptr/) Zeiger und verwenden Sie diesen Zeiger, um ihn als Argument an Funktionen zu übergeben.

```cpp
class IPPacketInformation : public System::Object
```

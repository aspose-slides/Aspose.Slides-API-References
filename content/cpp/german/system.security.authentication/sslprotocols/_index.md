## SslProtocols Enum

Enumeriert die kryptografischen Protokolle.

```cpp
enum class SslProtocols
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | 0 | Das Betriebssystem wählt ein zu verwendendes Protokoll aus. |
| Ssl2 | 12 | Das SSL 2.0-Protokoll. |
| Ssl3 | 48 | Das SSL 3.0-Protokoll. |
| Tls | 192 | Das TLS 1.0-Protokoll. |
| Default | 240 | Erlaubt nur die Verwendung des SSL 3.0- oder TLS 1.0-Protokolls. |
| Tls11 | 768 | Das TLS 1.1-Protokoll. |
| Tls12 | 3072 | Das TLS 1.2-Protokoll. |

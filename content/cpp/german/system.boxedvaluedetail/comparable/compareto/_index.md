## Comparable::CompareTo(T) Methode

Vergleicht das aktuelle Objekt mit dem angegebenen Objekt.

```cpp
int System::BoxedValueDetail::Comparable<T, Self>::CompareTo(T other) override
```

### Argumente

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| other | T | Das Objekt, das mit dem aktuellen Objekt verglichen werden soll |

### Rückgabewert

Eine negative Zahl, wenn das aktuelle Objekt im Sortierreihenfolge dem angegebenen Objekt vorausgeht; 0, wenn die verglichenen Objekte gleich sind; eine positive Zahl, wenn das aktuelle Objekt im Sortierreihenfolge dem angegebenen Objekt folgt

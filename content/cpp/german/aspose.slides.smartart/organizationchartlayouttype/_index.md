## OrganizationChartLayoutType enum

Stellt den Formatierungstyp der Kindknoten in einem Organigramm dar.

```cpp
enum class OrganizationChartLayoutType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Initial | 0 | Initialer Layouttyp |
| Standart | 1 | Platziert Kindknoten horizontal unter dem Elternknoten. |
| BothHanging | 2 | Platziert Kindknoten vertikal unter dem Elternknoten auf beiden Seiten, links und rechts. |
| LeftHanging | 3 | Platziert Kindknoten vertikal unter dem Elternknoten auf der linken Seite. |
| RightHanging | 4 | Platziert Kindknoten vertikal unter dem Elternknoten auf der rechten Seite. |

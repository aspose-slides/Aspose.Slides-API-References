## HeaderVariantInfo Klasse

Wird verwendet, um den Cookie-Namen und die Spezifikation abzugleichen. Objekte dieser Klasse sollten nur über die [System::MakeObject()](../../system/makeobject/) Funktion zugewiesen werden. Erstellen Sie niemals eine Instanz dieses Typs auf dem Stack oder mit dem Operator new, da dies Laufzeitfehler und/oder Assertion-Fehler zur Folge hat. Wickeln Sie diese Klasse immer in einen [System::SmartPtr](../../system/smartptr/) Zeiger und verwenden Sie diesen Zeiger, um ihn als Argument an Funktionen zu übergeben.

```cpp
class HeaderVariantInfo : public System::Object
```

## CookieCollection::get_IsOtherVersionSeen() Methode


Gibt einen Wert zurück, der angibt, ob die Sammlung ein Cookie mit einer Version enthält, die nicht gleich [Cookie::MaxSupportedVersion](../../cookie/maxsupportedversion/) ist.

```cpp
bool System::Net::CookieCollection::get_IsOtherVersionSeen()
```


### Rückgabewert

Wahr, wenn die Sammlung ein Cookie mit einer Version enthält, die nicht gleich [Cookie::MaxSupportedVersion](../../cookie/maxsupportedversion/) ist, andernfalls falsch.

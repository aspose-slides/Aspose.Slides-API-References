## CookieVariant enum

Enumeriert die Cookie-Spezifikationen.

```cpp
enum class CookieVariant
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Unknown | 0 | Die unbekannte Cookie-Spezifikation. |
| Plain | 1 | Die einfache Cookie-Spezifikation. |
| Rfc2109 | 2 | Die 'RFC 2109'-Cookie-Spezifikation. |
| Rfc2965 | 3 | Die 'RFC 2965'-Cookie-Spezifikation. |
| Default | 2 | Der Standardwert, der 'Rfc2109' entspricht. |

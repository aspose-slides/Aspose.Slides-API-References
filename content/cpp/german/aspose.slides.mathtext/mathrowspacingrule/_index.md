## MathRowSpacingRule enum

Der Typ des vertikalen Abstands zwischen Spalten in einer Matrix oder einem Array

```cpp
enum class MathRowSpacingRule
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| SingleLineGap | 0 | einzeiliger Abstand |
| OneAndAHalfLineGap | 1 | eineinhalbzeiliger Abstand |
| TwoLineGap | 2 | zweizeiliger Abstand |
| Exactly | 3 | Genau (abhängig vom Wert der RowSpacing-Eigenschaft, gemessen in Punkten) |
| Multiple | 4 | Mehrfach (abhängig vom Wert der RowSpacing-Eigenschaft, gemessen in Zeilen) |

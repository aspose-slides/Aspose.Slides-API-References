## MathPhantom::set_Transp(bool) Methode

Legt einen Wert fest, der angibt, ob das Phantom für klassenbasierte Abstandregeln transparent ist.

```cpp
void Aspose::Slides::MathText::MathPhantom::set_Transp(bool value) override
```

## Hinweise

Wenn **true**, beeinflussen Operatoren und Symbole im Phantom weiterhin den mathematischen Abstand um das Phantom herum (als ob es sichtbar wäre). Wenn **false**, wird die klassenbasierte Abstandsanpassung ignoriert. Entspricht dem OMML-Attribut **m:transp**.

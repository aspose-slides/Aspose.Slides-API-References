## MathPhantom::set_ZeroWidth(bool) Methode

Setzt einen Wert, der angibt, ob die Breite des Basiselements als null behandelt werden soll.

```cpp
void Aspose::Slides::MathText::MathPhantom::set_ZeroWidth(bool value) override
```

## Bemerkungen

Wenn **true**, reserviert das Phantom keinen horizontalen Platz für seine Basis. Entspricht dem OMML-Attribut **m:zeroWid**.

## MathPhantom::set_ZeroAsc(bool) method

Setzt einen Wert, der angibt, ob der Aufstieg (Höhe über der Grundlinie) des Basiselements als Null behandelt werden soll.

```cpp
void Aspose::Slides::MathText::MathPhantom::set_ZeroAsc(bool value) override
```

## Hinweise

Wenn **true**, erhöht das Phantom die Grundlinie der umgebenden mathematischen Zeile nicht. Entspricht dem OMML-Attribut **m:zeroAsc**.

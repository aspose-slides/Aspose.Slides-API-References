## IMathPhantom::set_Transp(bool) Methode


Setzt einen Wert, der angibt, ob das Phantom für klassenbasierte Abstandregeln transparent ist.

```cpp
virtual void Aspose::Slides::MathText::IMathPhantom::set_Transp(bool value)=0
```

## Hinweise


Wenn **true**, wirken Operatoren und Symbole innerhalb des Phantoms weiterhin auf den mathematischen Abstand um das Phantom (als ob es sichtbar wäre). Wenn **false**, wird die klassenbasierte Abstandsermittlung ignoriert. Entspricht dem OMML-Attribut **m:transp**.

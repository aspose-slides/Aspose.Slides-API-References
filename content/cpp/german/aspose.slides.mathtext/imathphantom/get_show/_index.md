## IMathPhantom::get_Show() Methode

Liefert einen Wert, der angibt, ob das Basiselement angezeigt wird.

```cpp
virtual bool Aspose::Slides::MathText::IMathPhantom::get_Show()=0
```

## Anmerkungen

Wenn **false**, ist das Basiselement ausgeblendet, kann jedoch basierend auf anderen Phantom-Einstellungen weiterhin Platz einnehmen. Entspricht dem OMML-Attribut **m:show**.

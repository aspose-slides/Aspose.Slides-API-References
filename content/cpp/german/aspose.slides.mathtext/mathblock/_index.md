## MathBlock Klasse

Specifiziert eine Instanz von mathematischem Text, die innerhalb eines [MathParagraph](../mathparagraph/) enthalten ist und in einer eigenen Zeile beginnt. Alle mathematischen Zonen, einschließlich Gleichungen, Ausdrücke, Arrays von Gleichungen oder Ausdrücken und Formeln, werden durch einen MathBlock dargestellt.

```cpp
class MathBlock : public Aspose::Slides::MathText::MathElementBase,
                  public Aspose::Slides::MathText::IMathBlock
```

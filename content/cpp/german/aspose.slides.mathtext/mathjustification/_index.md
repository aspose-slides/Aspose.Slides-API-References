## MathJustification enum

Gibt die Ausrichtung des mathematischen Absatzes an (eine Reihe von benachbarten Instanzen mathematischen Textes innerhalb desselben Absatzes)

```cpp
enum class MathJustification
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| LeftJustified | 1 | Mathematischen Text nach links ausrichten |
| RightJustified | 2 | Mathematischen Text nach rechts ausrichten |
| Centered | 3 | Mathematischen Text zentrieren |
| CenteredAsGroup | 4 | Zentriert die gesamte Gruppe von mathematischen Blöcken auf der Seite als einzelne Einheit. Das bedeutet, dass die Instanzen des mathematischen Textes relativ zueinander ausgerichtet werden können, aber die gesamte Gruppe des mathematischen Textes als Ganzes zentriert wird. |

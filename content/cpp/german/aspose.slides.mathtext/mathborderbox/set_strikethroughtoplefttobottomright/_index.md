## MathBorderBox::set_StrikethroughTopLeftToBottomRight(bool) Methode


Durchgestrichen von oben links nach unten rechts (Standardwert ist false). Gibt den ausgeblendeten oder angezeigten Zustand einer diagonal durchgestrichenen Linie von der oberen linken Ecke zur unteren rechten Ecke des Randkastens an.

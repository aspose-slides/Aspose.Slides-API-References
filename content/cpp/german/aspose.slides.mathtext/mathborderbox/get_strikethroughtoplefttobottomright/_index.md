## MathBorderBox::get_StrikethroughTopLeftToBottomRight() Methode


Durchgestrichen von oben links nach unten rechts (Standardwert ist false). Gibt den verborgenen oder angezeigten Zustand einer durchgestrichenen diagonalen Linie von der oberen linken Ecke zur unteren rechten Ecke des Randfelds an.

```cpp
bool Aspose::Slides::MathText::MathBorderBox::get_StrikethroughTopLeftToBottomRight() override
```

## Anmerkungen


Beispiel: 
```cpp
auto borderBox = System::MakeObject<MathBorderBox>(System::MakeObject<MathematicalText>(u"x"));
borderBox->set_StrikethroughTopLeftToBottomRight(true);
```

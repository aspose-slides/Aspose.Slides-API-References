## PrintRange enum


Указывает, какие страницы печатаются.

```cpp
enum class PrintRange
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| AllPages | 0 | Печатает все страницы. |
| Selection | 1 | Печатает выбранные страницы. |
| SomePages | 2 | Печатает страницы из указанного диапазона. |
| CurrentPage | 4194304 | Печатает текущую отображаемую страницу. |

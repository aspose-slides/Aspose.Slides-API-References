## SoapBindingStyle enum

Указывает тип действия в сервисе XML [Web](../../system.web/) на уровне иерархии классов, к которому применяется данное перечисление.

```cpp
enum class SoapBindingStyle
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Default | 0 | Тип действия по умолчанию. |
| Document | 1 | Сообщение ориентировано на документ. |
| Rpc | 2 | Сообщение содержит параметры для вызова процедуры или возвращаемые значения этой процедуры. |

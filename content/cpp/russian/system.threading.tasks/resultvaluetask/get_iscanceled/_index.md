## ResultValueTask::get_IsCanceled() const метод


Возвращает значение, указывающее, завершилась ли задача из-за отмены.

```cpp
bool System::Threading::Tasks::ResultValueTask<T>::get_IsCanceled() const
```


### Возвращаемое значение

bool True, если задача была отменена; в противном случае - false.
## Примечания



Возвращает false, если задача содержит прямое значение результата.

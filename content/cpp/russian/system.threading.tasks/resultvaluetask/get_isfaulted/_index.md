## ResultValueTask::get_IsFaulted() const метод


Получает значение, указывающее, завершилась ли задача из-за необработанного исключения.

```cpp
bool System::Threading::Tasks::ResultValueTask<T>::get_IsFaulted() const
```


### Возвращаемое значение

bool True если задача находится в состоянии ошибки; иначе false.
## Примечания

Возвращает false, если задача содержит прямое результирующее значение.

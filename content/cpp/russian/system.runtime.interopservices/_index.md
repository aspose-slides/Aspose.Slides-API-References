## Классы

| Класс | Описание |
| --- | --- |
| [Details_ExternalException](./details_externalexception/) | Базовый тип исключения для всех исключений COM-interop и исключений структурной обработки исключений (SEH). Никак не создавайте экземпляры этого класса вручную. Вместо этого используйте класс ExternalException. Никогда не оборачивайте экземпляры класса ExternalException в [System::SmartPtr](../system/smartptr/). |
| [Marshal](./marshal/) | Обеспечивает реализацию маршаллинга. Только для совместимости с транслированным кодом, поскольку управляемый код не поддерживается на стороне C++. Это статический тип без сервисов экземпляров. Вы никогда не должны создавать его экземпляры ни каким образом. |
| [MemoryMarshal](./memorymarshal/) | Обеспечивает реализацию маршаллинга памяти. Только для совместимости с транслированным кодом, поскольку управляемый код не поддерживается на стороне C++. Это статический тип без сервисов экземпляров. Вы никогда не должны создавать его экземпляры ни каким образом. |

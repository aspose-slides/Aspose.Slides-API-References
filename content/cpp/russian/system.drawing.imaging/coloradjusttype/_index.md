## ColorAdjustType перечисление

Указывает, какие объекты используют информацию о корректировке цвета.

```cpp
enum class ColorAdjustType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Default | 0 | Определяет информацию о корректировке цвета, используемую всеми объектами, у которых нет собственной информации о корректировке цвета. |
| Bitmap | 1 | Определяет информацию о корректировке цвета для объектов [Bitmap](../../system.drawing/bitmap/). |
| Brush | 2 | Определяет информацию о корректировке цвета для объектов [Brush](../../system.drawing/brush/). |
| Pen | 3 | Определяет информацию о корректировке цвета для объектов [Pen](../../system.drawing/pen/). |
| Text | 4 | Определяет информацию о корректировке цвета для текста. |
| Count | 5 | Указывает количество указанных типов. |
| Any | 6 | Указывает количество указанных типов. |

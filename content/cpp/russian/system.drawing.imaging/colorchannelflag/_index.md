## ColorChannelFlag перечисление

Указывает цветовой канал в цветовом пространстве CMYK.

```cpp
enum class ColorChannelFlag
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| ColorChannelC | 0 | Циановый цветовой канал. |
| ColorChannelM | 1 | Пурпурный цветовой канал. |
| ColorChannelY | 2 | Желтый цветовой канал. |
| ColorChannelK | 3 | Черный цветовой канал. |
| ColorChannelLast | 4 | Следует использовать последний выбранный канал. |

## ImageFormat::Equals(ImageFormatPtr) const метод


Определяет, равны ли форматы изображений, представленные текущим и указанным объектами.

```cpp
bool System::Drawing::Imaging::ImageFormat::Equals(ImageFormatPtr format) const
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| format | [ImageFormatPtr](../../imageformatptr/) | Объект формата изображения, с которым сравнивается текущий объект |

### Возвращаемое значение

True если текущий и указанный объекты представляют один и тот же формат изображения, иначе - false

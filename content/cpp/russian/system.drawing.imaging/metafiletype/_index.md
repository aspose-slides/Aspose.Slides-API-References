## MetafileType перечисление

Представляет тип графического метафайла.

```cpp
enum class MetafileType
```

### Значения

| Name | Value | Description |
| --- | --- | --- |
| Invalid | 0 | Недействительный метафайл. |
| Wmf | 1 | Стандартный WMF. |
| WmfPlaceable | 2 | Размещаемый формат [Metafile](../metafile/). |
| Emf | 3 | EMF (не EMF+). |
| EmfPlusOnly | 4 | EMF+ без двойных, устаревших записей. |
| EmfPlusDual | 5 | EMF+ с двойными, устаревшими записями. |

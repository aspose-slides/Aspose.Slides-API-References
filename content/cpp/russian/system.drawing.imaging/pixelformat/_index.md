## PixelFormat перечисление

Указывает формат цветовых данных пикселя.

```cpp
enum class PixelFormat
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Indexed | 65536 | Указывает, что данные пикселя содержат индексированные значения цветов, что означает, что они являются индексом цветов в системной таблице цветов. |
| Gdi | 131072 | Указывает, что данные пикселя содержат цвета GDI. |
| Alpha | 262144 | Указывает, что данные пикселя содержат альфа-значения, которые не предумножены. |
| PAlpha | 524288 | Указывает, что данные пикселя содержат предумноженные альфа-значения. |
| Extended | 1048576 | Зарезервировано. |
| Canonical | 2097152 | Указывает формат пикселя 32 бита на пиксель с 24-битной глубиной цвета и 8-битным альфа-каналом. |
| Undefined | 0 | Указывает, что формат пикселя не определён. |
| DontCare | 0 | Формат пикселя не указан. |
| Format1bppIndexed | n/a | Указывает, что формат пикселя — 1-битный индексированный цвет на пиксель. |
| Format4bppIndexed | n/a | Указывает, что формат пикселя — 4-битный индексированный цвет на пиксель. |
| Format8bppIndexed | n/a | Указывает, что формат пикселя — 8-битный индексированный цвет на пиксель. |
| Format16bppGrayScale | n/a | Указывает, что формат пикселя — 16 бит на пиксель. Цветовая информация задаёт 65536 оттенков серого. |
| Format16bppRgb555 | n/a | Указывает, что формат пикселя — 16 бит на пиксель с 5 битами для каждой из компонентов красного, зелёного и синего, а оставшийся бит не используется. |
| Format16bppRgb565 | n/a | Указывает, что формат пикселя — 16 бит на пиксель с 5 битами для красного, 6 битами для зелёного и 5 битами для синего компонентов. |
| Format16bppArgb1555 | n/a | Указывает, что формат пикселя — 16 бит на пиксель с 5 битами для каждой из компонентов красного, зелёного и синего и 1 битом для альфа. |
| Format24bppRgb | n/a | Указывает, что формат пикселя — 24 бита на пиксель с 8 битами для каждой из компонентов красного, зелёного и синего. |
| Format32bppRgb | n/a | Указывает, что формат пикселя — 32 бита на пиксель с 8 битами для каждой из компонентов красного, зелёного и синего, а оставшиеся 8 бит не используются. |
| Format32bppArgb | n/a | Указывает, что формат пикселя — 32 бита на пиксель с 8 битами для каждой из компонентов красного, зелёного и синего и 8 битами для альфа. |
| Format32bppPArgb | n/a | Указывает, что формат пикселя — 32 бита на пиксель с 8 битами для каждой из компонентов красного, зелёного и синего и 8 битами для альфа. Компоненты красного, зелёного и синего предумножены в соответствии со значением альфа-компоненты. |
| Format48bppRgb | n/a | Указывает, что формат пикселя — 48 бит на пиксель с 16 битами для каждой из компонентов красного, зелёного и синего. |
| Format64bppArgb | n/a | Указывает, что формат пикселя — 64 бита на пиксель с 16 битами для каждой из компонентов красного, зелёного и синего и 16 битами для альфа. |
| Format64bppPArgb | n/a | Указывает, что формат пикселя — 64 бита на пиксель с 16 битами для каждой из компонентов красного, зелёного и синего и 16 битами для альфа. Компоненты красного, зелёного и синего предумножены в соответствии со значением альфа-компоненты. |
| Format32bppCMYK | n/a | Указывает, что формат пикселя — 32 бита на пиксель с 8 битами для каждой из компонентов циан, маджента, жёлтого и ключевого. |
| Max | 16 | Максимальное значение этого перечисления. |

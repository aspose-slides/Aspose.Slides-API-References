## EmfType enum

Указывает типы записей, которые помещаются в файл EMF.

```cpp
enum class EmfType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| EmfOnly | n/a | [Windows](../../system.windows/) расширенный метафайл. Содержит команды GDI. Метафайлы этого типа называют файлом EMF. |
| EmfPlusOnly | n/a | [Windows](../../system.windows/) расширенный метафайл плюс. Содержит команды GDI+. Метафайлы этого типа называют файлом EMF+. |
| EmfPlusDual | n/a | Дуальный [Windows](../../system.windows/) расширенный метафайл. Содержит эквивалентные команды GDI и GDI+. Метафайлы этого типа называют файлом EMF+. |

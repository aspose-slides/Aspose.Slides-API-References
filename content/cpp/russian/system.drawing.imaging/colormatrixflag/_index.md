## ColorMatrixFlag перечисление


Указывает типы изображений и цветов, которые будут затронуты настройками коррекции цвета и градаций серого объекта [ImageAttributes](../imageattributes/).

```cpp
enum class ColorMatrixFlag
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Default | 0 | И цвета, и значения в градациях серого корректируются по цвету. |
| SkipGrays | 1 | Значения в градациях серого не корректируются по цвету. |
| AltGrays | 2 | Только значения в градациях серого корректируются по цвету. |

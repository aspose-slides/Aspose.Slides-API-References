## EventResetMode enum


Указывает, как сбрасывается состояние события.

```cpp
enum class EventResetMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| AutoReset | 0 | Состояние события сбрасывается автоматически при чтении. |
| ManualReset | 1 | Состояние события должно быть сброшено вручную. |

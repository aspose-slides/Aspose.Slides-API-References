## ThreadPool::SetMinThreads(int, int) метод

Устанавливает минимальное количество потоков, принадлежащих пулу.

```cpp
static bool System::Threading::ThreadPool::SetMinThreads(int wt, int cpt)
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| wt | int | Минимальное количество рабочих потоков. |
| cpt | int | Минимальное количество потоков [IO](../../../system.io/). |

### Возвращаемое значение

True, если параметры приняты, иначе false.

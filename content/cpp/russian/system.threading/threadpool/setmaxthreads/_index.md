## ThreadPool::SetMaxThreads(int, int) метод


Устанавливает количество потоков, принадлежащих пулу.

```cpp
static bool System::Threading::ThreadPool::SetMaxThreads(int wt, int cpt)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| wt | int | Максимальное количество рабочих потоков. |
| cpt | int | Максимальное количество [IO](../../../system.io/) потоков. |

### Возвращаемое значение

True если параметры приняты, false в противном случае.

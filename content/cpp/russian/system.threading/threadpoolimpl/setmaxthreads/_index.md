## ThreadPoolImpl::SetMaxThreads(int, int) метод

Устанавливает количество потоков, принадлежащих пулу.

```cpp
bool System::Threading::ThreadPoolImpl::SetMaxThreads(int wt, int cpt)
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| wt | int | Максимальное количество рабочих потоков. |
| cpt | int | Максимальное количество [IO](../../../system.io/) потоков. |

### Возвращаемое значение

True, если параметры были приняты, иначе false.

## ThreadPoolImpl::SetMinThreads(int, int) метод


Устанавливает минимальное количество потоков, принадлежащих пулу.

```cpp
bool System::Threading::ThreadPoolImpl::SetMinThreads(int wt, int cpt)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| wt | int | Минимальное количество рабочих потоков. |
| cpt | int | Минимальное количество [IO](../../../system.io/) потоков. |

### Возвращаемое значение

True, если параметры были приняты, иначе false.

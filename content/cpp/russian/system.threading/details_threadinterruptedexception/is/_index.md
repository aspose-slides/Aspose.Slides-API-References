
### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| target | const [System::TypeInfo](../../../system/typeinfo/)\& | [TypeInfo](../../../system/typeinfo/) структура, описывающая тип, с которым сравнивается текущий объект. |

### Возвращаемое значение

True, если объект имеет указанный тип или его подкласс, иначе false.

## Примечания

Проверьте, представляет ли объект экземпляр типа, описанного targetType. Аналог оператора C# 'is'.

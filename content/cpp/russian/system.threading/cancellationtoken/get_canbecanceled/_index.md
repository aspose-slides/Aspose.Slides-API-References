## CancellationToken::get_CanBeCanceled() const метод


Получает, может ли этот токен находиться в состоянии отмены.

```cpp
bool System::Threading::CancellationToken::get_CanBeCanceled() const
```


### Возвращаемое значение

true, если этот токен может находиться в состоянии отмены; иначе false.
## Примечания



Токены, созданные из [CancellationTokenSource](../../cancellationtokensource/), вернут true, тогда как токен None всегда возвращает false.

## XPathNodeIterator::MoveNext() метод

При переопределении в производном классе перемещает объект [XPathNavigator](../../xpathnavigator/), возвращённый [XPathNodeIterator::get_Current](../get_current/) методом, к следующему узлу в выбранном наборе узлов.

```cpp
virtual bool System::Xml::XPath::XPathNodeIterator::MoveNext()=0
```

### Возвращаемое значение

**true** если объект [XPathNavigator](../../xpathnavigator/) переместился к следующему узлу; **false** если больше нет выбранных узлов.

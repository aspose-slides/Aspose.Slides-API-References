## XmlCaseOrder перечисление

Указывает порядок сортировки заглавных и строчных букв.

```cpp
enum class XmlCaseOrder
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | 0 | Игнорировать регистр. |
| UpperFirst | 1 | Заглавные буквы сортируются перед строчными буквами. |
| LowerFirst | 2 | Строчные буквы сортируются перед заглавными буквами. |

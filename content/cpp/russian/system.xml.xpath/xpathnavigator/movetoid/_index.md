## XPathNavigator::MoveToId(String) метод

Когда переопределяется в производном классе, перемещается к узлу, у которого есть атрибут типа **ID**, значение которого соответствует указанному [String](../../../system/string/).

```cpp
virtual bool System::Xml::XPath::XPathNavigator::MoveToId(String id)=0
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| id | [String](../../../system/string/) | Объект [String](../../../system/string/), представляющий значение **ID** узла, к которому вы хотите переместиться. |

### Возвращаемое значение

**true**, если перемещение [XPathNavigator](../) выполнено успешно; в противном случае **false**. Если **false**, позиция навигатора остаётся неизменной.

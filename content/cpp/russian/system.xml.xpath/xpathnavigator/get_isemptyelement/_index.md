## XPathNavigator::get_IsEmptyElement() метод


При переопределении в производном классе получаем значение, которое указывает, является ли текущий узел пустым элементом без закрывающего тега.

```cpp
virtual bool System::Xml::XPath::XPathNavigator::get_IsEmptyElement()=0
```


### Возвращаемое значение

**true** если текущий узел является пустым элементом; в противном случае **false**.

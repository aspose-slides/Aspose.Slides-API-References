## XPathNavigator::get_HasChildren() метод

Возвращает значение, указывающее, имеет ли текущий узел дочерние узлы.

```cpp
virtual bool System::Xml::XPath::XPathNavigator::get_HasChildren()
```

### Возвращаемое значение

**true** если у текущего узла есть дочерние узлы; в противном случае **false**.

## XPathNavigator::GetNamespace(String) метод

Возвращает значение узла пространства имен, соответствующего указанному локальному имени.

```cpp
virtual String System::Xml::XPath::XPathNavigator::GetNamespace(String name)
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| name | [String](../../../system/string/) | Локальное имя узла пространства имен. |

### Возвращаемое значение

Объект [String](../../../system/string/), содержащий значение узла пространства имен; [String::Empty](../../../system/string/empty/), если соответствующий узел пространства имен не найден, или если [XPathNavigator](../) не находится на узле элемента.

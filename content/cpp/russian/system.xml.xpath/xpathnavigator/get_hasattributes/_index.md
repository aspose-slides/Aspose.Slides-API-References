## XPathNavigator::get_HasAttributes() метод

Возвращает значение, указывающее, имеет ли текущий узел атрибуты.

```cpp
virtual bool System::Xml::XPath::XPathNavigator::get_HasAttributes()
```

### Возвращаемое значение

**true** если текущий узел имеет атрибуты; возвращает **false**, если у текущего узла нет атрибутов, или если [XPathNavigator](../) не находится на узле-элементе.

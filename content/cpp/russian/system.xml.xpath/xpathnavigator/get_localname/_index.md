## XPathNavigator::get_LocalName() метод


Если переопределён в производном классе, получает [XPathNavigator::get_Name](../get_name/) текущего узла без префикса пространства имён.

```cpp
virtual String System::Xml::XPath::XPathNavigator::get_LocalName()=0
```


### Возвращаемое значение

Объект [String](../../../system/string/), содержащий локальное имя текущего узла, или [String::Empty](../../../system/string/empty/), если текущий узел не имеет имени (например, узлы текста или комментариев).

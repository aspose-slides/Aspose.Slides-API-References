## XPathNavigator::MoveToParent() метод

Когда переопределяется в производном классе, перемещает [XPathNavigator](../) к родительскому узлу текущего узла.

```cpp
virtual bool System::Xml::XPath::XPathNavigator::MoveToParent()=0
```

### Возвращаемое значение

**true** если [XPathNavigator](../) успешно перемещается к родительскому узлу текущего узла; в противном случае **false**. Если **false**, позиция [XPathNavigator](../) остаётся неизменной.

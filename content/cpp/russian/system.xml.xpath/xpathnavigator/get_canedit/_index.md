## XPathNavigator::get_CanEdit() метод

Возвращает значение, которое указывает, может ли [XPathNavigator](../) редактировать базовые XML-данные.

```cpp
virtual bool System::Xml::XPath::XPathNavigator::get_CanEdit()
```

### Возвращаемое значение

**true** если [XPathNavigator](../) может редактировать базовые XML-данные; в противном случае **false**.

## XPathNavigator::MoveToFirstChild() метод


При переопределении в производном классе перемещает [XPathNavigator](../) к первому дочернему узлу текущего узла.

```cpp
virtual bool System::Xml::XPath::XPathNavigator::MoveToFirstChild()=0
```


### Возвращаемое значение

**true** если [XPathNavigator](../) успешно перемещается к первому дочернему узлу текущего узла; иначе, **false**. Если **false**, позиция [XPathNavigator](../) остаётся неизменной.

## XPathNavigator::MoveToNextAttribute() метод

При переопределении в производном классе перемещает [XPathNavigator](../) к следующему атрибуту.

```cpp
virtual bool System::Xml::XPath::XPathNavigator::MoveToNextAttribute()=0
```

### Возвращаемое значение

**true**, если [XPathNavigator](../) успешно перемещается к следующему атрибуту; **false**, если больше нет атрибутов. Если **false**, положение [XPathNavigator](../) остаётся без изменений.

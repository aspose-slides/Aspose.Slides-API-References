## XPathItem::get_IsNode() метод

При переопределении в производном классе возвращает значение, указывающее, представляет ли элемент узел [XPath](../../) или атомарное значение.

```cpp
virtual bool System::Xml::XPath::XPathItem::get_IsNode()=0
```

### Возвращаемое значение

**true** если элемент представляет узел [XPath](../../); **false** если элемент представляет атомарное значение.

## Trendline::set_Backward(double) метод

Указывает количество категорий (или единиц на диаграмме разброса), на которое линия тренда простирается до данных ряда, для которого построен тренд. На диаграммах разброса и неразброса значение может быть любым неотрицательным. Запишите **double**.

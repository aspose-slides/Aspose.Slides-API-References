## Trendline::get_TextFrameForOverriding() метод

Может содержать форматированный текст с расширенным оформлением. Если это свойство не равно null, то значение этого форматированного текста переопределяет автоматически сгенерированный текст подписи данных. Автоматически сгенерированный текст подписи данных — это текст, управляемый свойствами ShowSeriesName, ShowValue, ... и форматируемый свойством TextFormatManager.TextFormat. Только для чтения [ITextFrame](../../../aspose.slides/itextframe/).

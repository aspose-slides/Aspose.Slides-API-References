## Trendline::set_Forward(double) метод

Указывает количество категорий (или единиц на диаграмме разброса), на которое трендовая линия продолжается после данных для серии, которая анализируется. На диаграммах разброса и неразбросных диаграммах значение должно быть любым неотрицательным значением. Запись **double**.

```cpp
void Aspose::Slides::Charts::Trendline::set_Forward(double value) override
```

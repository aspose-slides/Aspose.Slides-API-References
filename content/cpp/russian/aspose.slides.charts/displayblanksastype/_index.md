## DisplayBlanksAsType enum

Определяет, как будет отображаться недостающие данные.

```cpp
enum class DisplayBlanksAsType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Gap | 0 | Указывает, что пустые значения должны оставаться как пробел. |
| Span | 1 | Указывает, что пустые значения должны быть заполнены линией. |
| Zero | 2 | Указывает, что пустые значения должны рассматриваться как ноль. |

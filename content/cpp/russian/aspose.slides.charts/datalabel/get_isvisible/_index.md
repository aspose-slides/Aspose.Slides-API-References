## DataLabel::get_IsVisible() метод

False означает, что подпись данных не видима (и поэтому все флаги Show*-flags (ShowValue, ...) имеют значение false). Только для чтения **bool**.

```cpp
bool Aspose::Slides::Charts::DataLabel::get_IsVisible() override
```

## Примечания

Если подпись данных видима, её можно скрыть с помощью метода [Hide()](../hide/). Но если подпись данных не видима (IsVisible is false), её можно сделать видимой, установив флаги Show*-flags (ShowValue, ...) в состояние true.

## DataLabel::Hide() метод


Сделайте подпись данных скрытой, установив все флаги Show*-flags (ShowValue, ...) в состояние false. IsVisible будет false после этого.

```cpp
void Aspose::Slides::Charts::DataLabel::Hide() override
```

## Примечания


Если подпись данных не видима (IsVisible is false), вы можете сделать её видимой, установив флаги Show*-flags (ShowValue, ...) в состояние true.

## TickMarkType перечисление

Представляет тип деления шкалы для указанной оси.

```cpp
enum class TickMarkType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Cross | 0 | Указывает, что деления должны пересекать ось. |
| Inside | 1 | Указывает, что деления должны находиться внутри области построения. |
| None | 2 | Указывает, что делений не будет. |
| Outside | 3 | Указывает, что деления должны быть за пределами области построения. |

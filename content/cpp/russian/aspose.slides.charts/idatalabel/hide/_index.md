## IDataLabel::Hide() метод


Сделайте подпись данных скрытой, установив все флаги Show*- (ShowValue, ...) в состояние false. IsVisible будет false после этого.

```cpp
virtual void Aspose::Slides::Charts::IDataLabel::Hide()=0
```

## Примечания


Если подпись данных не видима (IsVisible is false), вы можете сделать её видимой, установив флаги Show*- (ShowValue, ...) в состояние true.

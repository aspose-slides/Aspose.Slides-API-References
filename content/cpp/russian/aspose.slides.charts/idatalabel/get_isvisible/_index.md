## IDataLabel::get_IsVisible() метод

False означает, что метка данных не видима (и поэтому все флаги Show*-flags (ShowValue, ...) имеют значение false). Только для чтения **bool**.

```cpp
virtual bool Aspose::Slides::Charts::IDataLabel::get_IsVisible()=0
```

## Примечания

Если метка данных видима, вы можете скрыть её с помощью метода [Hide()](../hide/). Но если метка данных не видима (IsVisible равно false), вы можете сделать её видимой, установив флаги Show*-flags (ShowValue, ...) в состояние true.

## ChartShapeType enum

Represents a shape of chart.

```cpp
enum class ChartShapeType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| NotDefined | -1 | [Chart](../chart/) форма не определена. |
| Box | 0 | Указывает, что диаграмма будет нарисована в виде коробки. |
| Cone | 1 | Указывает, что диаграмма будет нарисована в виде конуса, при этом основание конуса находится на полу, а вершина конуса — вверху маркера данных. |
| ConeToMax | 2 | Указывает, что диаграмма будет нарисована с усечёнными конусами так, чтобы вершина конуса соответствовала максимальному значению данных. |
| Cylinder | 3 | Указывает, что диаграмма будет нарисована в виде цилиндра. |
| Pyramid | 4 | Указывает, что диаграмма будет нарисована в виде прямоугольной пирамиды, при этом основание пирамиды находится на полу, а вершина пирамиды — вверху маркера данных. |
| PyramidToMaximum | 5 | Указывает, что диаграмма будет нарисована с усечёнными конусами так, чтобы вершина конуса соответствовала максимальному значению данных. |

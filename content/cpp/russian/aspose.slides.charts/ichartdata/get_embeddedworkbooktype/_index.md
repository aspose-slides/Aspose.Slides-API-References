## IChartData::get_EmbeddedWorkbookType() метод

Получает тип встроенной рабочей книги. Возвращает [WorkbookType::NotDefined](../../workbooktype/), если [IChartData::get_DataSourceType](../get_datasourcetype/) равен [ChartDataSourceType::ExternalWorkbook](../../chartdatasourcetype/). Только для чтения [WorkbookType](../../workbooktype/).

virtual WorkbookType Aspose::Slides::Charts::IChartData::get_EmbeddedWorkbookType()=0

## DataLabelCollection::get_IsVisible() метод


False означает, что подпись данных не видна по умолчанию (и поэтому все флаги Show* (ShowValue, ...) свойства DefaultDataLabelFormat имеют значение false). Только для чтения **bool**.

```cpp
bool Aspose::Slides::Charts::DataLabelCollection::get_IsVisible() override
```

## Примечания


Если подпись данных видна по умолчанию, вы можете сделать её скрытой по умолчанию с помощью метода [Hide()](../hide/). Но если подпись данных не видна по умолчанию (IsVisible is false) вы можете сделать подпись данных \"видимой 
по умолчанию\" установив флаги Show* (ShowValue, ...) свойства DefaultDataLabelFormat в состояние true.

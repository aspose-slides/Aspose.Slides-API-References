## IDoubleChartValue класс

Представляет значение double, которое может храниться в документе презентации pptx двумя способами: 1) в ячейке/ячейках рабочей книги, связанной с диаграммой; 2) как литеральное значение.

```cpp
class IDoubleChartValue : public Aspose::Slides::Charts::ISingleCellChartValue
```

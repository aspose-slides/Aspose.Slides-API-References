## DoubleChartValue::ToDouble() метод

Преобразует в **double**.

```cpp
double Aspose::Slides::Charts::DoubleChartValue::ToDouble() override
```

### Возвращаемое значение

Возвращает LiteralDouble, если DataSourceType равно DoubleLiterals. Если DataSourceType равно Worksheet, возвращает успешно преобразованное в double значение ячейки, иначе возвращает NaN.

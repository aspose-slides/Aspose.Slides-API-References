## LegendDataLabelPosition enum

Определяет положение меток данных.

```cpp
enum class LegendDataLabelPosition
```

### Значения

| Name | Value | Description |
| --- | --- | --- |
| NotDefined | -1 | Позиция не определена. |
| Bottom | 0 | Указывает, что метки данных должны отображаться ниже маркера данных. |
| BestFit | 1 | Указывает, что метки данных должны отображаться в наилучшей позиции. |
| Center | 2 | Указывает, что метки данных должны отображаться по центру маркера данных. |
| InsideBase | 3 | Указывает, что метки данных должны отображаться внутри основания маркера данных. |
| InsideEnd | 4 | Указывает, что метки данных должны отображаться внутри конца маркера данных. |
| Left | 5 | Указывает, что метки данных должны отображаться слева от маркера данных. |
| OutsideEnd | 6 | Указывает, что метки данных должны отображаться за пределами конца маркера данных. |
| Right | 7 | Указывает, что метки данных должны отображаться справа от маркера данных. |
| Top | 8 | Указывает, что метки данных должны отображаться выше маркера данных. |

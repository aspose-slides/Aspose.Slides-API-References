## IDataLabelCollection::get_IsVisible() метод


False означает, что подпись данных по умолчанию не видна (и поэтому все флаги Show*-flags (ShowValue, ...) свойства DefaultDataLabelFormat имеют значение false). Только для чтения **bool**.

```cpp
virtual bool Aspose::Slides::Charts::IDataLabelCollection::get_IsVisible()=0
```

## Примечания


Если подпись данных видима по умолчанию, вы можете сделать её скрытой по умолчанию с помощью метода [Hide()](../hide/). Но если подпись данных не видима по умолчанию (IsVisible is false) вы можете сделать подпись данных "видимая 
по умолчанию" установкой флагов Show*-flags (ShowValue, ...) свойства DefaultDataLabelFormat в состояние true.

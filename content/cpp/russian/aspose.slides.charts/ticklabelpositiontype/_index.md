## TickLabelPositionType перечисление


Представляет тип положения меток делений на указанной оси.

```cpp
enum class TickLabelPositionType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| High | 0 | Указывает, что подписи осей должны располагаться на верхнем конце перпендикулярной оси. |
| Low | 1 | Указывает, что подписи осей должны располагаться на нижнем конце перпендикулярной оси. |
| NextTo | 2 | Указывает, что подписи осей должны располагаться рядом с осью. |
| None | 3 | Указывает, что подписи осей не рисуются. |

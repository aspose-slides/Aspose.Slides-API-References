## Rotation3D::set_RotationY(uint16_t) метод

Устанавливает угол вращения вокруг оси Y, т.е. в направлении X для 3D-диаграмм (от 0 до 360 градусов). Свойство соответствует элементу 21.2.2.158 rotY (Y Rotation) в ECMA-376 и опции \"X Rotation\" в PowerPoint 2007+. Записывает **uint16_t**.

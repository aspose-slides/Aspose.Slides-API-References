## IChartTextBlockFormat::set_MarginTop(double) метод

Задает верхний отступ (points) в [TextFrame](../../../aspose.slides/textframe/). Изменение этого свойства может оказывать определённое влияние только на следующие части диаграммы: [DataLabel](../../datalabel/) и [DataLabelFormat](../../datalabelformat/) (полная поддержка в PowerPoint 2013; в PowerPoint 2007 нет эффекта при отображении). Запишите **double**.

```cpp
virtual void Aspose::Slides::Charts::IChartTextBlockFormat::set_MarginTop(double value)=0
```

## IChartTextBlockFormat::set_MarginLeft(double) метод

Устанавливает левый отступ (в пунктах) в [TextFrame](../../../aspose.slides/textframe/). Изменение этого свойства может оказывать определённое влияние только на следующие части диаграммы: [DataLabel](../../datalabel/) и [DataLabelFormat](../../datalabelformat/) (полная поддержка в PowerPoint 2013; в PowerPoint 2007 нет эффекта при рендеринге). Запишите **double**.

```cpp
virtual void Aspose::Slides::Charts::IChartTextBlockFormat::set_MarginLeft(double value)=0
```

## IChartTextBlockFormat::set_RotationAngle(float) метод

Указывает пользовательский угол поворота, применяемый к тексту внутри ограничивающего прямоугольника. Если он не указан, используется поворот сопутствующей фигуры. Если он указан, то применяется независимо от фигуры. То есть фигура может иметь поворот, а текст сам может иметь свой собственный поворот. Полученное значение визуального поворота текста суммируется из этого свойства и предопределенного вертикального типа в свойстве TextVerticalType. Записать **float**.

```cpp
virtual void Aspose::Slides::Charts::IChartTextBlockFormat::set_RotationAngle(float value)=0
```

## Примечания

Рассмотрим случай, когда у фигуры применён поворот на 90 градусов по часовой стрелке. Кроме того, к самому телу текста применяется поворот на -90 градусов против часовой стрелки. Тогда результирующая фигура будет выглядеть повернутой, но текст внутри неё будет выглядеть так, будто он не был повернут.

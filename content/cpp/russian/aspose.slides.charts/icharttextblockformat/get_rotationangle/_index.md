## IChartTextBlockFormat::get_RotationAngle() метод

Указывает пользовательский поворот, применяемый к тексту внутри ограничивающего прямоугольника. Если он не задан, используется поворот сопряжённой фигуры. Если он задан, то применяется независимо от фигуры. То есть фигура может иметь собственный поворот, добавленный к повороту самого текста. Получаемое значение визуального поворота текста формируется из этого свойства и предопределённого вертикального типа в свойстве TextVerticalType. Читать **float**.

```cpp
virtual float Aspose::Slides::Charts::IChartTextBlockFormat::get_RotationAngle()=0
```

## Примечания

Рассмотрим случай, когда у фигуры применяется поворот на 90 градусов по часовой стрелке. Кроме того, тело текста имеет поворот на -90 градусов против часовой стрелки. В результате фигура будет выглядеть повернутой, но текст внутри неё будет выглядеть так, будто его вовсе не вращали.

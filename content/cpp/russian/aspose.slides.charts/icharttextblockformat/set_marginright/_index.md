## IChartTextBlockFormat::set_MarginRight(double) метод


Устанавливает правый отступ (в точках) в [TextFrame](../../../aspose.slides/textframe/). Изменение этого свойства может оказывать определённое влияние только на следующие части диаграммы: [DataLabel](../../datalabel/) и [DataLabelFormat](../../datalabelformat/) (полная поддержка в PowerPoint 2013; в PowerPoint 2007 влияние на отрисовку отсутствует). Записывается **double**.

```cpp
virtual void Aspose::Slides::Charts::IChartTextBlockFormat::set_MarginRight(double value)=0
```

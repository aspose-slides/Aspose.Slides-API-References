## IChartTextBlockFormat::set_AutofitType(TextAutofitType) метод

Устанавливает режим автоматической подгонки текста. Изменение этого свойства может оказывать определённое влияние только для следующих частей диаграммы: [DataLabel](../../datalabel/) и [DataLabelFormat](../../datalabelformat/) (полная поддержка в PowerPoint 2013; в PowerPoint 2007 нет эффекта при рендеринге). Запишите [TextAutofitType](../../../aspose.slides/textautofittype/).

virtual void Aspose::Slides::Charts::IChartTextBlockFormat::set_AutofitType(TextAutofitType value)=0

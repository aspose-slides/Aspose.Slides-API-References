## IChartTextBlockFormat::set_MarginBottom(double) метод

Устанавливает нижний отступ (точки) в [TextFrame](../../../aspose.slides/textframe/). Изменение этого свойства может оказать определённое влияние только на следующие части диаграммы: [DataLabel](../../datalabel/) и [DataLabelFormat](../../datalabelformat/) (полная поддержка в PowerPoint 2013; в PowerPoint 2007 не оказывает эффекта при визуализации). Записать **double**.

```cpp
virtual void Aspose::Slides::Charts::IChartTextBlockFormat::set_MarginBottom(double value)=0
```

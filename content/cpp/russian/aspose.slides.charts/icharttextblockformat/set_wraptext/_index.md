## IChartTextBlockFormat::set_WrapText(NullableBool) метод


**True** если текст переносится по полям [TextFrame](../../../aspose.slides/textframe/)'s. Изменение этого свойства может оказывать определённое влияние только на следующие части диаграммы: [DataLabel](../../datalabel/) и [DataLabelFormat](../../datalabelformat/) (полная поддержка в PowerPoint 2007/2013). Запишите [NullableBool](../../../aspose.slides/nullablebool/).

virtual void Aspose::Slides::Charts::IChartTextBlockFormat::set_WrapText(NullableBool value)=0

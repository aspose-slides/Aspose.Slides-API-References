## IChartTextBlockFormat::get_MarginTop() метод

Возвращает верхний отступ (в пунктах) в [TextFrame](../../../aspose.slides/textframe/). Изменение этого свойства может оказывать определённое влияние только на следующие части диаграммы: [DataLabel](../../datalabel/) и [DataLabelFormat](../../datalabelformat/) (полная поддержка в PowerPoint 2013; в PowerPoint 2007 эффект при визуализации отсутствует). Читает **double**.

```cpp
virtual double Aspose::Slides::Charts::IChartTextBlockFormat::get_MarginTop()=0
```

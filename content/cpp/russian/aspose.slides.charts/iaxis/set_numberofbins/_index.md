## IAxis::set_NumberOfBins(uint32_t) метод


Указывает количество корзин, когда значение свойства AggregationType установлено в [AxisAggregationType::ByNumberOfBins](../../axisaggregationtype/). Применяется к категориальным осям. Используется только с сериями Histogram или HistogramPareto.

## Примечания

Если родитель этого объекта [DataLabelFormat](../) является коллекцией [DataLabelCollection](../../datalabelcollection/) подписей данных, то это свойство представляет формат по умолчанию для новых подписей данных в коллекции [DataLabelCollection](../../datalabelcollection/).

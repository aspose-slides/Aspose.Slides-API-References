## IChartCategoryCollection::get_UseCells() метод

Если true, то рабочий лист используется для хранения категорий (в этом случае поддерживаются многоуровневые категории). Если false, то рабочий лист НЕ используется для хранения значений (и в этом случае многоуровневые категории не поддерживаются). Читать **bool**.

```cpp
virtual bool Aspose::Slides::Charts::IChartCategoryCollection::get_UseCells()=0
```

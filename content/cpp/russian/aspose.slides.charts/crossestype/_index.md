## CrossesType enum

Определяет, где будет пересекаться ось.

```cpp
enum class CrossesType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| AxisCrossesAtZero | 0 | Ось категорий пересекает ось значений в нулевой точке (если возможно), либо в минимальном значении (если минимум больше нуля), либо в максимальном (если максимум меньше нуля). |
| Maximum | 1 | Ось пересекает в максимальном значении. |
| Custom | 2 | Пользовательское значение из свойства CrossAt |

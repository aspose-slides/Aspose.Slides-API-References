## IChartSeries::get_HasUpDownBars() метод

Определяет, содержит ли линейный или биржевой график вертикальные полосы. Это свойство относится не только к этой серии, но и ко всем сериям группы родительских серий — это проекция соответствующего свойства группы. Поэтому это свойство только для чтения. Используйте свойство ParentSeriesGroup для доступа к группе родительских серий. Используйте [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get_UpDownBars()->get(set)_HasUpDownBars() свойство чтения/записи для изменения значения. Используйте свойство [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get_UpDownBars() для форматирования вертикальных полос. Только для чтения **bool**.

```cpp
virtual bool Aspose::Slides::Charts::IChartSeries::get_HasUpDownBars()=0
```

## IChartSeries::get_DoughnutHoleSize() метод

Указывает размер отверстия в кольцевой диаграмме (может быть от 10 до 90 процентов от размера области построения). Это свойство относится не только к этой серии, но и ко всем сериям родительской группы серий — это проекция соответствующего свойства группы. Таким образом, это свойство только для чтения. Используйте ParentSeriesGroup свойство для доступа к родительской группе серий. Используйте [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get(set)_DoughnutHoleSize() свойство чтение/запись для изменения значения. Только для чтения **uint8_t**.

```cpp
virtual uint8_t Aspose::Slides::Charts::IChartSeries::get_DoughnutHoleSize()=0
```

## Примечания

Это проекция свойства [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get(set)_DoughnutHoleSize().

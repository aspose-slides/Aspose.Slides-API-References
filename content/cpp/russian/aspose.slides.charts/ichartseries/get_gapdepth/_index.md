## IChartSeries::get_GapDepth() метод


Возвращает расстояние, выраженное в процентах от ширины маркера, между рядами данных в 3D-диаграмме. Это свойство относится не только к этому ряду, но и ко всем рядам группы родительского ряда — это проекция соответствующего группового свойства. Поэтому это свойство только для чтения. Используйте свойство ParentSeriesGroup для доступа к группе родительского ряда. Используйте [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get(set)_GapDepth() свойство для чтения/записи, чтобы изменить значение. Только для чтения **int32_t**.

```cpp
virtual int32_t Aspose::Slides::Charts::IChartSeries::get_GapDepth()=0
```

## Примечания


Это проекция свойства [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get(set)_GapDepth().

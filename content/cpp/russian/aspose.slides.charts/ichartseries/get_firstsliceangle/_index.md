## IChartSeries::get_FirstSliceAngle() метод

Задает угол первого сегмента круговой или кольцевой диаграммы в градусах (по часовой стрелке от верхнего положения, от 0 до 360 градусов). Это свойство относится не только к этой серии, но и ко всем сериям группы родительской серии — это проекция соответствующего свойства группы. Поэтому это свойство только для чтения. Используйте свойство ParentSeriesGroup для доступа к группе родительской серии. Используйте [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get(set)_FirstSliceAngle() свойство чтения/записи для изменения значения. Только для чтения **uint16_t**.

```cpp
virtual uint16_t Aspose::Slides::Charts::IChartSeries::get_FirstSliceAngle()=0
```

## Примечания

Это проекция свойства [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get(set)_FirstSliceAngle().

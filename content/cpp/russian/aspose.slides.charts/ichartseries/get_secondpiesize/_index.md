## IChartSeries::get_SecondPieSize() метод

Указывает размер второго сектора или столбца в диаграмме «pie-of-pie» или «bar-of-pie» в процентах от размера первого сектора (может быть от 5 до 200 %). Это свойство относится не только к этой серии, но и ко всем сериям группы родительской серии — это проекция соответствующего свойства группы. Поэтому данное свойство является только для чтения. Используйте свойство ParentSeriesGroup для доступа к группе родительских серий. Используйте [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get(set)_SecondPieSize() свойство чтения/записи для изменения значения. Только для чтения **uint16_t**.

```cpp
virtual uint16_t Aspose::Slides::Charts::IChartSeries::get_SecondPieSize()=0
```

## Замечания

Это проекция свойства [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get(set)_SecondPieSize().

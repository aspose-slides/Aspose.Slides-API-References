## IChartSeries::get_PieSplitPosition() метод


Указывает значение, которое будет использоваться для определения, какие точки данных находятся во второй части круговой диаграммы или столбце в диаграмме «pie-of-pie» или «bar-of-pie». Используется вместе со свойством PieSplitBy. Это свойство относится не только к этой серии, но и ко всем сериям группы родительских серий — это проекция соответствующего свойства группы. Поэтому это свойство только для чтения. Используйте свойство ParentSeriesGroup для доступа к группе родительских серий. Используйте [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get(set)_PieSplitPosition() свойство чтения/записи для изменения значения. Только для чтения **double**.

```cpp
virtual double Aspose::Slides::Charts::IChartSeries::get_PieSplitPosition()=0
```

## Примечания


Это проекция свойства [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get(set)_PieSplitPosition().

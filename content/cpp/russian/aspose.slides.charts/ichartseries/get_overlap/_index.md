## IChartSeries::get_Overlap() метод

Указывает, насколько бары и колонки перекрываются на 2-D диаграммах, в процентах (от -100% до 100%). Это свойство относится не только к этой серии, но и ко всем сериям группы родительской серии. Это проекция соответствующего свойства в родительской группе серий, поэтому это свойство доступно только для чтения. Чтобы изменить значение, используйте [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get(set)_Overlap() свойство для чтения/записи. Только для чтения **int8_t**.

```cpp
virtual int8_t Aspose::Slides::Charts::IChartSeries::get_Overlap()=0
```

## Примечания

Overlap указывает степень перекрытия или расстояния между барами и колонками в процентах от их ширины:* -100%: Максимальное расстояние (бары полностью разделены). * 0%: Бары размещаются рядом без перекрытия и без расстояния. * 100%: Максимальное перекрытие (бары полностью перекрывают друг друга). Это проекция свойства [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get(set)_Overlap().

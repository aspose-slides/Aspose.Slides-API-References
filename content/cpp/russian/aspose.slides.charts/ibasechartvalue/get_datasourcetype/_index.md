## IBaseChartValue::get_DataSourceType() метод

Указывает, является ли свойство AsCell, AsLiteralString или AsLiteralDouble актуальным. Другими словами, указывает тип значения свойства Data. Это свойство только для чтения. Чтобы изменить значение этого свойства, вы можете использовать одно из свойств ChartDataPointCollection.DataSourceTypeFor<...>. Читайте [DataSourceType](../../datasourcetype/).

virtual Aspose::Slides::Charts::DataSourceType Aspose::Slides::Charts::IBaseChartValue::get_DataSourceType()=0

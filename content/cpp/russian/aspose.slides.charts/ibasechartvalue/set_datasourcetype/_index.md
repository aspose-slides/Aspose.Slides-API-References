## IBaseChartValue::set_DataSourceType(Aspose::Slides::Charts::DataSourceType) метод

Указывает, является ли свойство AsCell или AsLiteralString или AsLiteralDouble актуальным. Другими словами, он указывает тип значения свойства Data. Это свойство только для чтения. Для изменения значения этого свойства вы можете использовать одно из свойств ChartDataPointCollection.DataSourceTypeFor<...>. Запишите [DataSourceType](../../datasourcetype/).

virtual void Aspose::Slides::Charts::IBaseChartValue::set_DataSourceType(Aspose::Slides::Charts::DataSourceType value)=0

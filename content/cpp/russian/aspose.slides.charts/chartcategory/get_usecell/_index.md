## ChartCategory::get_UseCell() метод


Если true, то свойство AsCell актуально. Другими словами, рабочий лист используется для хранения категории (в этом случае поддерживается многоуровневая категория). Если false, то свойство AsLiteral актуально. Другими словами, рабочий лист НЕ используется для хранения категории (и в этом случае многоуровневые категории не поддерживаются). Только для чтения **bool**.

```cpp
bool Aspose::Slides::Charts::ChartCategory::get_UseCell() override
```

## Примечания


Для изменения значения этого свойства (для всех категорий в коллекции) задайте новое значение свойству ChartCategoryCollection::get(set)_UseCells.

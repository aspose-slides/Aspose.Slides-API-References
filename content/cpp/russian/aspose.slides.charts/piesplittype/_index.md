## Перечисление PieSplitType

Представляет тип точек разделения во второй круговой диаграмме или столбце на диаграмме типа pie-of-pie или bar-of-pie.

```cpp
enum class PieSplitType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Default | 0 | Указывает, что точки данных будут разделены с использованием механизма по умолчанию для этого типа диаграммы. |
| Custom | 1 | Указывает, что точки данных будут разделены между круговой диаграммой и второй диаграммой в соответствии со значениями Custom Split. |
| ByPercentage | 2 | Указывает, что точки данных будут разделены между круговой диаграммой и второй диаграммой, помещая точки с процентом меньше Split Position процентов во вторую диаграмму. |
| ByPos | 3 | Указывает, что точки данных будут разделены между круговой диаграммой и второй диаграммой, помещая последнее значение Split Position точек данных во вторую диаграмму. |
| ByValue | 4 | Указывает, что точки данных будут разделены между круговой диаграммой и второй диаграммой, помещая точки данных со значением меньше Split Position во вторую диаграмму. |

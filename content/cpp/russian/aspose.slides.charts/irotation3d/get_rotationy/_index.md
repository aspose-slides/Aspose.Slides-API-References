## IRotation3D::get_RotationY() метод

Возвращает угол поворота вокруг оси Y, т. е. в направлении X для 3D-диаграмм (от 0 до 360 градусов). Свойство соответствует элементу 21.2.2.158 rotY (Y Rotation) в ECMA-376 и параметру "X Rotation" в PowerPoint 2007+. Читать **uint16_t**.

```cpp
virtual uint16_t Aspose::Slides::Charts::IRotation3D::get_RotationY()=0
```

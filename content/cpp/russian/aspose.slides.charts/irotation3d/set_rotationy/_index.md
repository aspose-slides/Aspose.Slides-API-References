## IRotation3D::set_RotationY(uint16_t) метод

Устанавливает угол вращения вокруг оси Y, т.е. в направлении X для 3D-диаграмм (в диапазоне от 0 до 360 градусов). Свойство соответствует элементу 21.2.158 rotY (Поворот по оси Y) в ECMA-376 и опции \"X Rotation\" в PowerPoint 2007+. Записывает **uint16_t**.

virtual void Aspose::Slides::Charts::IRotation3D::set_RotationY(uint16_t value)=0

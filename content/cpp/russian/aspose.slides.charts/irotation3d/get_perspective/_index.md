## IRotation3D::get_Perspective() метод

Возвращает значение перспективы (угол поля зрения) для 3D-диаграмм (от 0 до 100). Игнорируется, если значение свойства RightAngleAxes равно true. Читается **uint8_t**.

virtual uint8_t Aspose::Slides::Charts::IRotation3D::get_Perspective()=0

## IRotation3D::set_RotationX(int8_t) метод

Устанавливает угол вращения вокруг оси X, т. е. в направлении Y для 3D-диаграмм (от -90 до 90 градусов). Свойство соответствует элементу 21.2.2.157 rotX (X Rotation) в ECMA-376 и параметру \"Y Rotation\" в PowerPoint 2007+. Записывайте **int8_t**.

```cpp
virtual void Aspose::Slides::Charts::IRotation3D::set_RotationX(int8_t value)=0
```

## IRotation3D::set_Perspective(uint8_t) метод

Устанавливает значение перспективы (угол поля зрения) для 3D-диаграмм (от 0 до 100). Игнорируется, если значение свойства RightAngleAxes равно true. Записывает **uint8_t**.

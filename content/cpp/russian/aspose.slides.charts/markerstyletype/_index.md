## MarkerStyleType enum

Определяет форму маркера на точке данных диаграммы.

```cpp
enum class MarkerStyleType
```

### Values

| Имя | Значение | Описание |
| --- | --- | --- |
| NotDefined | -1 |  |
| Circle | 0 | Указывает, что на каждой точке данных будет нарисован круг. |
| Dash | 1 | Указывает, что на каждой точке данных будет нарисована черта. |
| Diamond | 2 | Указывает, что на каждой точке данных будет нарисован ромб. |
| Dot | 3 | Указывает, что на каждой точке данных будет нарисована точка. |
| None | 4 | Указывает, что на каждой точке данных ничего не будет нарисовано. |
| Picture | 5 | Указывает, что на каждой точке данных будет нарисовано изображение. |
| Plus | 6 | Указывает, что на каждой точке данных будет нарисован плюс. |
| Square | 7 | Указывает, что на каждой точке данных будет нарисован квадрат. |
| Star | 8 | Указывает, что на каждой точке данных будет нарисована звезда. |
| Triangle | 9 | Указывает, что на каждой точке данных будет нарисован треугольник. |
| X | 10 | Указывает, что на каждой точке данных будет нарисована буква X. |

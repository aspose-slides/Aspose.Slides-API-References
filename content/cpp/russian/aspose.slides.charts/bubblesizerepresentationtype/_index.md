## BubbleSizeRepresentationType enum

Указывает возможные способы представления данных в виде размеров пузырчатой диаграммы.

```cpp
enum class BubbleSizeRepresentationType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Area | 0 | Указывает, что площадь пузырей должна быть пропорциональна значению размера пузыря. |
| Width | 1 | Указывает, что радиус пузырей должен быть пропорционален значению размера пузыря. |

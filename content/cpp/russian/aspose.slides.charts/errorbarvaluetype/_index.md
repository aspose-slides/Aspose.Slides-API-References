## ErrorBarValueType enum


Представляет тип значения полосы ошибок

```cpp
enum class ErrorBarValueType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Custom | 0 | Указывает, что длина линий ошибок будет определяться элементами Plus и Minus. |
| Fixed | 1 | Указывает, что длина линий ошибок будет фиксированным значением, определённым параметром Error Bar Value. |
| Percentage | 2 | Указывает, что длина линий ошибок будет составлять процент Error Bar Value от данных. |
| StandardDeviation | 3 | Указывает, что длина линий ошибок будет составлять стандартные отклонения Error Bar Value от данных. |
| StandardError | 4 | Указывает, что длина линий ошибок будет составлять стандартные ошибки Error Bar Value от данных. |

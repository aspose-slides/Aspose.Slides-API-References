## ChartCategoryCollection::set_UseCells(bool) метод

Если true, то рабочий лист используется для хранения категорий (в этом случае поддерживаются многоуровневые категории). Если false, то рабочий лист НЕ используется для хранения значений (и в этом случае не поддерживаются многоуровневые категории). Запишите **bool**.

void Aspose::Slides::Charts::ChartCategoryCollection::set_UseCells(bool value) override

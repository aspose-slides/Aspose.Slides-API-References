## ITrendline::set_Forward(double) метод

Указывает количество категорий (или единиц на диаграмме рассеяния), на которое трендовая линия продолжается после данных для ряда, который является трендируемым. На диаграммах рассеяния и нерассеяния значение должно быть любым неотрицательным. Запишите **double**.

```cpp
virtual void Aspose::Slides::Charts::ITrendline::set_Forward(double value)=0
```

## ITrendline::set_Backward(double) метод


Указывает количество категорий (или единиц на точечной диаграмме), на которое линия тренда расширяется до данных ряда, который тренируется. Для точечных и не точечных диаграмм значение должно быть любым неотрицательным значением. Записывается **double**.

virtual void Aspose::Slides::Charts::ITrendline::set_Backward(double value)=0

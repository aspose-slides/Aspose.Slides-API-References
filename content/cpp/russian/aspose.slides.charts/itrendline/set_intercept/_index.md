## ITrendline::set_Intercept(double) метод

Указывает значение, где трендовая линия пересекает ось y. Это свойство поддерживается только когда тип трендовой линии — exp, linear или poly. Записывается **double**.

virtual void Aspose::Slides::Charts::ITrendline::set_Intercept(double value)=0

## ITrendline::get_Backward() метод

Указывает количество категорий (или единиц на диаграмме разброса), на которое линия тренда продолжается до данных серии, для которой построен тренд. На диаграммах разброса и не-расброса значение должно быть любым неотрицательным значением. Чтение **double**.

```cpp
virtual double Aspose::Slides::Charts::ITrendline::get_Backward()=0
```

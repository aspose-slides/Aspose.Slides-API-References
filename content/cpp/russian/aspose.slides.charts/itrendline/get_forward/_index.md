## ITrendline::get_Forward() метод

Указывает количество категорий (или единиц на точечной диаграмме), на которое линия тренда расширяется после данных для серии, для которой построен тренд. На точечных и не точечных диаграммах значение должно быть любой неотрицательной величиной. Читайте **double**.

```cpp
virtual double Aspose::Slides::Charts::ITrendline::get_Forward()=0
```

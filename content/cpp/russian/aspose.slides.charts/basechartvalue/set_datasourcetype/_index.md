## BaseChartValue::set_DataSourceType(Aspose::Slides::Charts::DataSourceType) метод

Указывает, является ли свойство AsCell, AsCells, AsLiteralString или AsLiteralDouble актуальным в наследниках. Другими словами, указывает тип значения свойства Data. Запишите [Charts::DataSourceType](../../datasourcetype/).

```cpp
void Aspose::Slides::Charts::BaseChartValue::set_DataSourceType(Aspose::Slides::Charts::DataSourceType value) override
```

## Примечания

Для точек в [ChartDataPointCollection](../../chartdatapointcollection/) это свойство только для чтения. В этом случае для изменения значения этого свойства вы можете использовать одно из свойств ChartDataPointCollection.DataSourceTypeFor<...>.

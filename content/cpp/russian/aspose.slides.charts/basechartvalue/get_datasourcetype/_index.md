## BaseChartValue::get_DataSourceType() метод


Указывает, является ли свойство AsCell, AsCells, AsLiteralString или AsLiteralDouble фактическим в наследниках. Иными словами, указывает тип значения свойства Data. См. [Charts::DataSourceType](../../datasourcetype/).

```cpp
Aspose::Slides::Charts::DataSourceType Aspose::Slides::Charts::BaseChartValue::get_DataSourceType() override
```

## Примечания


Для точек в [ChartDataPointCollection](../../chartdatapointcollection/) это свойство только для чтения. В этом случае для изменения значения этого свойства вы можете использовать одно из свойств ChartDataPointCollection.DataSourceTypeFor<...>.

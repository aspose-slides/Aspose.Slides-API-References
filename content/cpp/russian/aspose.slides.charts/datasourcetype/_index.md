## DataSourceType перечисление


Типы источников данных.

```cpp
enum class DataSourceType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Worksheet | 0 | Используйте объекты CellXls для хранения XValue, YValue, Value или BubbleSize точки данных в worksheet. |
| StringLiterals | 1 | Используйте переменную строки с быстрым доступом для хранения XValue, YValue, Value или BubbleSize точки данных. Worksheet НЕ используется для хранения значений. |
| DoubleLiterals | 2 | Используйте переменную типа double с быстрым доступом для хранения XValue, YValue, Value или BubbleSize точки данных. Worksheet НЕ используется для хранения значений. |

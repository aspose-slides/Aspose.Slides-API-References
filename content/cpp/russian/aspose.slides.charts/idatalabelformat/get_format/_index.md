## IDataLabelFormat::get_Format() метод

Представляет формат подписи данных. Только для чтения [IFormat](../../iformat/).

```cpp
virtual System::SharedPtr<IFormat> Aspose::Slides::Charts::IDataLabelFormat::get_Format()=0
```

## Примечания

Если родительским объектом для этого [DataLabelFormat](../../datalabelformat/) является коллекция [DataLabelCollection](../../datalabelcollection/) подписей данных, то это свойство представляет формат по умолчанию для новых подписей данных в коллекции [DataLabelCollection](../../datalabelcollection/).

## ParentLabelLayoutType перечисление

Представляет макет меток данных категорий. Используется только с серией Treemap.

```cpp
enum class ParentLabelLayoutType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Banner | 0 | Указывает, что метки родительских категорий отображаются в виде баннера |
| None | 1 | Указывает, что метки родительских категорий не отображаются |
| Overlapping | 2 | Указывает, что метки родительских категорий отображаются перекрывающимися |

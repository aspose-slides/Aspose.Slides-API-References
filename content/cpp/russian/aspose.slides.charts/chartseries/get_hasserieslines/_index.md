## ChartSeries::get_HasSeriesLines() метод


Определяет, есть ли линии серии для этой серии и связанных с ней серий. Это свойство относится не только к этой серии, но и ко всем сериям родительской группы серий — это проекция соответствующего свойства группы. Таким образом, это свойство только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get(set)_HasSeriesLines() свойство чтения/записи для изменения значения. Используйте свойство ParentSeriesGroup.SeriesLinesFormat для форматирования линий серии. Толькo для чтения **bool**.

bool Aspose::Slides::Charts::ChartSeries::get_HasSeriesLines() override

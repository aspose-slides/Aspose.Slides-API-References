## ChartSeries::get_TrendLines() метод

Коллекция трендовых линий серии. Только для чтения [ITrendlineCollection](../../itrendlinecollection/).

```cpp
System::SharedPtr<ITrendlineCollection> Aspose::Slides::Charts::ChartSeries::get_TrendLines() override
```

## Замечания

TrendLines доступны (не null) для данных серий в неслоенных 2-D областных, столбчатых, линейных, столбцовых, графиках акций, xy (рассеяния) и пузырьковых диаграммах. Трендовая линия недоступна для данных серий в любом типе диаграммы, который является сложенным или 3-D. Trendlines также недоступны для радиальных, круговых, поверхностных или кольцевых диаграмм.

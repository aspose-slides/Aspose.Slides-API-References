## ChartSeries::get_FirstSliceAngle() метод


Задает угол первого сектора круговой или кольцевой диаграммы в градусах (по часовой стрелке от верха, от 0 до 360 градусов). Это свойство относится не только к этой серии, но ко всем сериям родительской группы серий — это проекция соответствующего свойства группы. Таким образом, это свойство только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get(set)_FirstSliceAngle() свойство чтения/записи для изменения значения. Только для чтения **uint16_t**.

```cpp
uint16_t Aspose::Slides::Charts::ChartSeries::get_FirstSliceAngle() override
```

## Примечания


Это проекция свойства [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get(set)_FirstSliceAngle().

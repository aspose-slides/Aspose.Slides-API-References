## ChartSeries::set_ShowMeanMarkers(bool) метод

Представляет маркеры среднего. True, если маркеры среднего отображаются на диаграмме BoxAndWhisker. Применяется только к диаграммам BoxAndWhisker. Записывается **bool**.

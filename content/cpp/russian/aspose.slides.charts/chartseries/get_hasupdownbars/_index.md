## ChartSeries::get_HasUpDownBars() метод


Определяет, имеет ли линейный или столбчатый график верхние/нижние полосы. Это свойство относится не только к этой серии, но и ко всем сериям группы родительской серии — это проекция соответствующего свойства группы. Поэтому это свойство только для чтения. Используйте свойство ParentSeriesGroup для доступа к группе родительской серии. Используйте [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get_UpDownBars()->get(set)_HasUpDownBars() свойство чтения/записи для изменения значения. Используйте [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get_UpDownBars() свойство для форматирования верхних/нижних полос. Только для чтения **bool**.

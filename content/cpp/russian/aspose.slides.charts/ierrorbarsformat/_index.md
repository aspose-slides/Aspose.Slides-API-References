## IErrorBarsFormat класс


Представляет линии ошибок серии диаграммы. Пользовательские значения ErrorBars находятся в [IChartDataPointCollection](../ichartdatapointcollection/) (в свойстве [IChartDataPoint::get_ErrorBarsCustomValues()](../ichartdatapoint/get_errorbarscustomvalues/)).

```cpp
class IErrorBarsFormat : public Aspose::Slides::Charts::IChartComponent
```

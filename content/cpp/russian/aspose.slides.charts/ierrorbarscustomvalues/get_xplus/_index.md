## IErrorBarsCustomValues::get_XPlus() метод


Указывает значение линии ошибок в положительном направлении. Доступно, если тип значения линий ошибок установлен в Custom и разрешён ErrorBarsXFormat. В остальных случаях это свойство возвращает null. Только для чтения [IDoubleChartValue](../../idoublechartvalue/).

## ErrorBarsFormat::set_ValueType(ErrorBarValueType) метод

Представляет возможные способы определения длины полос ошибок. В случае пользовательского типа значения для указания значения используйте свойство [IChartDataPoint::get_ErrorBarsCustomValues()](../../ichartdatapoint/get_errorbarscustomvalues/) конкретной точки данных в коллекции DataPoints серии. В случае типа значения Fixed, Percentage или StandardDeviation используйте свойство Value для указания значения.

Запишите [ErrorBarValueType](../../errorbarvaluetype/).

```cpp
void Aspose::Slides::Charts::ErrorBarsFormat::set_ValueType(ErrorBarValueType value) override
```

## ErrorBarsFormat::get_ValueType() метод

Представляет возможные способы определения длины полос погрешности. В случае пользовательского типа значения для указания значения используйте свойство [IChartDataPoint::get_ErrorBarsCustomValues()](../../ichartdatapoint/get_errorbarscustomvalues/) конкретной точки данных в коллекции DataPoints серии. В случае типа значения Fixed, Percentage или StandardDeviation используйте свойство Value для указания значения.

 Читать [ErrorBarValueType](../../errorbarvaluetype/).

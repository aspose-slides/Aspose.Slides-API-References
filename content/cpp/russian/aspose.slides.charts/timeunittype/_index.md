## TimeUnitType перечисление


Представляет базовую единицу для оси категорий

```cpp
enum class TimeUnitType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | 0 | Значения будут отображаться как есть. |
| Days | 1 | Указывает, что данные диаграммы будут отображаться в днях. |
| Months | 2 | Указывает, что данные диаграммы будут отображаться в месяцах. |
| Years | 3 | Указывает, что данные диаграммы будут отображаться в годах. |

## ICUEncoding::GetMaxByteCount(int) метод

Получает максимальное количество байтов, необходимых для кодирования указанного количества символов.

```cpp
int System::Text::ICUEncoding::GetMaxByteCount(int char_count) override
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| char_count | int | Number of characters to encode. |

### Возвращаемое значение

Количество байтов, достаточное для кодирования любой строки заданной длины в символах.

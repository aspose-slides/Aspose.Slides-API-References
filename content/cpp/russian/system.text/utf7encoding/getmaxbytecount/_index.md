## UTF7Encoding::GetMaxByteCount(int) метод


Возвращает максимальное количество байтов, необходимое для кодирования указанного количества символов.

```cpp
int System::Text::UTF7Encoding::GetMaxByteCount(int char_count) override
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| char_count | int | Количество символов для кодирования. |

### Возвращаемое значение

Количество байтов, достаточных для кодирования любой строки заданной длины.

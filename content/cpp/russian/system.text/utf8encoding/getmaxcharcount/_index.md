## UTF8Encoding::GetMaxCharCount(int) метод

Получить максимальное количество символов, необходимое для декодирования заданного количества байтов.

```cpp
int System::Text::UTF8Encoding::GetMaxCharCount(int byte_count) override
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| byte_count | int | Количество байтов для декодирования. |

### Возвращаемое значение

Количество байтов, достаточных для декодирования любой строки заданной длины.

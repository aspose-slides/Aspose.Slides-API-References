## UTF8Encoding::GetMaxByteCount(int) метод


Получает максимальное количество байтов, необходимое для кодирования заданного числа символов.

```cpp
int System::Text::UTF8Encoding::GetMaxByteCount(int char_count) override
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| char_count | int | Количество символов для кодирования. |

### Возвращаемое значение

Количество байтов, достаточное для кодирования любой строки заданной длины в символах.

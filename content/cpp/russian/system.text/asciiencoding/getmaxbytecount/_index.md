## ASCIIEncoding::GetMaxByteCount(int) метод

Получает максимальное количество байтов, которое может вместить строку с известным количеством символов.

```cpp
int System::Text::ASCIIEncoding::GetMaxByteCount(int char_count) override
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| char_count | int | Количество символов. |

### Возвращаемое значение

Количество символов, достаточное для размещения любой строки из **charCount** символов.

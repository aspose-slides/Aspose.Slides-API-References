## Encoding::get_IsBrowserDisplay() метод


Проверяет, можно ли использовать кодировку в браузере для отображения содержимого.

```cpp
virtual bool System::Text::Encoding::get_IsBrowserDisplay()
```


### Возвращаемое значение

True если кодировка может быть использована в браузере для отображения содержимого, false в противном случае.

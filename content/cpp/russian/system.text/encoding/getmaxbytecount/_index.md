## Encoding::GetMaxByteCount(int) метод


Получите максимальное количество байтов, необходимых для кодирования указанного числа символов.

```cpp
virtual int System::Text::Encoding::GetMaxByteCount(int char_count)=0
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| char_count | int | Количество символов для кодирования. |

### Возвращаемое значение

Количество байтов, достаточных для кодирования любой строки заданной длины в символах.

## Encoding::get_IsMailNewsDisplay() метод

Проверяет, может ли кодировка использоваться в почтовом клиенте для отображения содержимого.

```cpp
virtual bool System::Text::Encoding::get_IsMailNewsDisplay()
```

### Возвращаемое значение

True если кодировка может использоваться в почтовом клиенте для отображения содержимого, иначе false.

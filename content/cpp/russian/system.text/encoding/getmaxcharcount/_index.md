## Encoding::GetMaxCharCount(int) метод

Получить максимальное количество символов, необходимых для декодирования заданного количества байт.

```cpp
virtual int System::Text::Encoding::GetMaxCharCount(int byte_count)=0
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| byte_count | int | Количество байтов для декодирования. |

### Возвращаемое значение

Количество байтов, достаточное для декодирования любой строки заданной длины символов.

## UnicodeEncoding::GetMaxByteCount(int) метод

Получить максимальное количество байтов, необходимое для кодирования заданного количества символов.

```cpp
int System::Text::UnicodeEncoding::GetMaxByteCount(int char_count) override
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| char_count | int | Количество символов для кодирования. |

### Возвращаемое значение

Количество байтов, достаточное для кодирования любой строки заданной длины символов.

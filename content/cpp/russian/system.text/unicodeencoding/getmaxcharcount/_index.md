## UnicodeEncoding::GetMaxCharCount(int) метод

Получить максимальное количество символов, необходимое для декодирования указанного количества байтов.

```cpp
int System::Text::UnicodeEncoding::GetMaxCharCount(int byte_count) override
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| byte_count | int | Количество байтов для декодирования. |

### Возвращаемое значение

Количество байтов, достаточное для декодирования любой строки заданной длины символов.

## IEnumerator::MoveNext() метод


Перемещает перечислитель к следующему элементу. Если ранее ни один элемент не был ссылкой, устанавливает ссылку на первый доступный элемент. Если достигнут конец контейнера, ничего не делает.

```cpp
virtual bool System::Collections::IEnumerator::MoveNext()=0
```


### Возвращаемое значение

True если после вызова доступен какой-то элемент, false в противном случае.

## IEnumeratorImplValueType::MoveNext() метод

Перемещает перечислитель к следующему элементу. Если ранее ни один элемент не был выбран, устанавливает ссылку на первый доступный элемент. Если достигнут конец контейнера, ничего не делает.

```cpp
virtual bool System::Collections::IEnumeratorImplValueType<T>::MoveNext() override
```

### Возвращаемое значение

True, если после вызова доступен какой-либо элемент, в противном случае false.

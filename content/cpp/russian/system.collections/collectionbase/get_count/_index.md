## CollectionBase::get_Count() метод

Возвращает количество элементов, содержащихся в экземпляре коллекции. Этот метод не может быть переопределён.

```cpp
int32_t System::Collections::CollectionBase<T>::get_Count()
```

### Возвращаемое значение

Количество элементов, содержащихся в экземпляре коллекции. Получение значения этого метода является операцией O(1).

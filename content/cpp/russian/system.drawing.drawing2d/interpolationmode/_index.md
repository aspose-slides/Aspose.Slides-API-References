## InterpolationMode enum

Specifies an algorithm to use when images are rotated or scaled.

```cpp
enum class InterpolationMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Invalid | n/a | Эквивалентно [QualityMode::Invalid](../qualitymode/). |
| Default | n/a | Режим по умолчанию. |
| Low | n/a | Низкое качество. |
| High | n/a | Высокое качество. |
| Bilinear | n/a | Билинейная интерполяция. |
| Bicubic | n/a | Бикубическая интерполяция. |
| NearestNeighbor | n/a | Интерполяция ближайшего соседа. |
| HighQualityBilinear | n/a | Высококачественная билинейная интерполяция. |
| HighQualityBicubic | n/a | Высококачественная бикубическая интерполяция. |

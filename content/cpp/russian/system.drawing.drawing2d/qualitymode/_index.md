## Перечисление QualityMode

Указывает общую качество рендеринга.

```cpp
enum class QualityMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Invalid | -1 | Недопустимый режим качества. |
| Default | 0 | Режим качества по умолчанию. |
| Low | 1 | Низкое качество, более высокая скорость. |
| High | 2 | Высокое качество, более низкая скорость. |

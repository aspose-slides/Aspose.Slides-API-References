## LineCap перечисление

Указывает доступные стили окончаний, с которыми объект [Pen](../../system.drawing/pen/) может завершать линию.

```cpp
enum class LineCap
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Flat | 0 | Плоское окончание линии. |
| Square | 1 | Квадратное окончание линии. |
| Round | 2 | Круглое окончание линии. |
| Triangle | 3 | Треугольное окончание линии. |
| NoAnchor | 16 | Нет привязки. |
| SquareAnchor | 17 | Квадратное привязное окончание линии. |
| RoundAnchor | 18 | Круглое привязное окончание линии. |
| DiamondAnchor | 19 | Алмазное привязное окончание линии. |
| ArrowAnchor | 20 | Стрелочное привязное окончание линии. |
| AnchorMask | 240 | Маска, используемая для проверки, является ли окончание линии привязным. |
| Custom | 255 | Пользовательское окончание. |

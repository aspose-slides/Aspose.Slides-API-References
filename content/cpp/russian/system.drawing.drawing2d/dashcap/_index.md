## Перечисление DashCap


Указывает тип окончания, используемого на обоих концах штриха в пунктирной линии.

```cpp
enum class DashCap
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Flat | 0 | Квадратное окончание, которое делает оба конца штриха квадратными. |
| Round | 2 | Круглое окончание, которое скругляет оба конца штриха. |
| Triangle | 3 | Треугольное окончание, которое образует острые концы штриха. |

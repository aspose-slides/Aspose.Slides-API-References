## PixelOffsetMode enum

Указывает, как смещаются пиксели при их отображении.

```cpp
enum class PixelOffsetMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Invalid | n/a | Недопустимое значение. |
| Default | n/a | Режим по умолчанию. |
| HighSpeed | n/a | Режим низкого качества (высокая производительность). |
| HighQuality | n/a | Режим высокого качества (низкая производительность). |
| None | n/a | Без смещения пикселей. |
| Half | n/a | Пиксели смещаются на -0.5 единицы по горизонтали и вертикали для высокопроизводительного сглаживания. |

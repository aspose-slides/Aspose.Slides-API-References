## Перечисление CombineMode

Указывает, как объединяются области обрезки.

```cpp
enum class CombineMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Replace | 0 | Одна область обрезки заменяется другой. |
| Intersect | 1 | Две области обрезки объединяются путем взятия их пересечения. |
| Union | 2 | Две области обрезки объединяются путем взятия их объединения. |
| Xor | 3 | Две области обрезки объединяются, беря только площадь, ограниченную одной из областей, но не обеими. |
| Exclude | 4 | Две области обрезки объединяются, беря площадь первой области, которая не пересекается со второй. |
| Complement | 5 | Две области обрезки объединяются, беря площадь второй области, которая не пересекается с первой. |

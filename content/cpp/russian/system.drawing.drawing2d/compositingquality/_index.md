## CompositingQuality перечисление


Specifies the quality level to be used during compositing.

```cpp
enum class CompositingQuality
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Invalid | n/a | Неверное значение. |
| Default | n/a | Значение по умолчанию. |
| HighSpeed | n/a | Низкое качество, высокая скорость. |
| HighQuality | n/a | Высокое качество, низкая скорость. |
| GammaCorrected | n/a | Используется гамма-коррекция. |
| AssumeLinear | n/a | Предполагаются линейные значения. |

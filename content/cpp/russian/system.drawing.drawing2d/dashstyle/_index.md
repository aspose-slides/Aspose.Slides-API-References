## DashStyle enum

Указывает стиль пунктирной линии.

```cpp
enum class DashStyle
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Solid | 0 | Сплошная линия. |
| Dash | 1 | Пунктирная линия. |
| Dot | 2 | Линия, состоящая из точек. |
| DashDot | 3 | Линия тире-точка. |
| DashDotDot | 4 | Линия тире-точка-точка. |
| Custom | 5 | Пользовательский стиль пунктировки. |

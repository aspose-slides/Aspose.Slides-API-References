## StringCollection::begin() метод


Возвращает итератор на первый элемент контейнера. Если контейнер пуст, возвращаемый итератор будет равен [end()](../end/).

```cpp
iterator System::Collections::Specialized::StringCollection::begin() noexcept
```


### Возвращаемое значение

Итератор, указывающий на первый элемент контейнера.

## StringCollection::begin() const метод


Возвращает итератор на первый элемент константного контейнера. Если контейнер пуст, возвращаемый итератор будет равен [end()](../end/).

```cpp
const_iterator System::Collections::Specialized::StringCollection::begin() const noexcept
```


### Возвращаемое значение

Итератор, указывающий на первый элемент константного контейнера.

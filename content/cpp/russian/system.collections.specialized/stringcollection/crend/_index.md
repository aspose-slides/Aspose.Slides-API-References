## StringCollection::crend() const метод


Возвращает обратный итератор к элементу, следующему за последним элементом обратного контейнера. Он соответствует элементу, предшествующему первому элементу необратного контейнера. Этот элемент служит заполнителем; попытка доступа к нему приводит к неопределённому поведению.

```cpp
const_reverse_iterator System::Collections::Specialized::StringCollection::crend() const noexcept
```


### Возвращаемое значение

Итератор, указывающий на теоретический const-qualified элемент, предшествующий первому элементу контейнера.

## StringCollection::cbegin() const метод


Возвращает итератор, указывающий на первый элемент контейнера, объявленный const. Если контейнер пуст, возвращаемый итератор будет равен [cend()](../cend/).

```cpp
const_iterator System::Collections::Specialized::StringCollection::cbegin() const noexcept
```


### Возвращаемое значение

Итератор, указывающий на первый элемент контейнера, объявленный const.

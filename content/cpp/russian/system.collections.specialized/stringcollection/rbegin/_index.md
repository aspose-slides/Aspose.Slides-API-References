## StringCollection::rbegin() метод


Возвращает обратный итератор к первому элементу перевёрнутого контейнера. Он соответствует последнему элементу неперевёрнутого контейнера. Если контейнер пуст, возвращаемый итератор равен [rend()](../rend/).

```cpp
reverse_iterator System::Collections::Specialized::StringCollection::rbegin() noexcept
```


### Возвращаемое значение

Итератор, указывающий на последний элемент контейнера.

## StringCollection::rbegin() const метод


Возвращает обратный итератор к первому элементу перевёрнутого контейнера. Он соответствует последнему элементу неперевёрнутого контейнера. Если контейнер пуст, возвращаемый итератор равен [rend()](../rend/).

```cpp
const_reverse_iterator System::Collections::Specialized::StringCollection::rbegin() const noexcept
```


### Возвращаемое значение

Итератор, указывающий на последний элемент константного контейнера.

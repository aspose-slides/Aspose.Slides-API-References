## StringCollection::crbegin() const метод

Возвращает обратный итератор, указывающий на первый элемент обращённого контейнера. Он соответствует последнему элементу необращённого контейнера. Если контейнер пуст, возвращённый итератор равен [crend()](../crend/).

```cpp
const_reverse_iterator System::Collections::Specialized::StringCollection::crbegin() const noexcept
```

### Возвращаемое значение

Итератор, указывающий на последний элемент контейнера, имеющий квалификатор const.

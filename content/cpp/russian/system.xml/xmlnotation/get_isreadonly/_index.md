## XmlNotation::get_IsReadOnly() метод

Возвращает значение, указывающее, является ли узел только для чтения.

```cpp
bool System::Xml::XmlNotation::get_IsReadOnly() override
```

### Возвращаемое значение

**true** если узел только для чтения; иначе — **false**. Поскольку узлы [XmlNotation](../) только для чтения, этот метод всегда возвращает **true**.

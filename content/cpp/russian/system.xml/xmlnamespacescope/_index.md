## XmlNamespaceScope enum


Определяет область действия пространства имён.

```cpp
enum class XmlNamespaceScope
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| All | 0 | Все пространства имён, определённые в области действия текущего узла. Это включает пространство имён xmlns:xml, которое всегда объявляется неявно. Порядок возвращаемых пространств имён не определён. |
| ExcludeXml | 1 | Все пространства имён, определённые в области действия текущего узла, за исключением пространства имён xmlns:xml, которое всегда объявляется неявно. Порядок возвращаемых пространств имён не определён. |
| Local | 2 | Все пространства имён, определённые локально в текущем узле. |

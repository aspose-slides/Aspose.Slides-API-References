## XmlDocument::get_IsReadOnly() метод


Возвращает значение, указывающее, является ли текущий узел только для чтения.

```cpp
bool System::Xml::XmlDocument::get_IsReadOnly() override
```


### Возвращаемое значение

**true** если текущий узел только для чтения; в противном случае **false**. [XmlDocument](../) узлы всегда возвращают **false**.

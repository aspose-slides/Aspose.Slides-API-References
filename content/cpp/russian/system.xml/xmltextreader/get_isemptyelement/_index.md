## XmlTextReader::get_IsEmptyElement() метод


Возвращает значение, указывающее, является ли текущий узел пустым элементом (например, **<MyElement/>**).

```cpp
bool System::Xml::XmlTextReader::get_IsEmptyElement() override
```


### Возвращаемое значение

**true** если текущий узел является элементом ([XmlTextReader::get_NodeType](../get_nodetype/) значение равно [XmlNodeType::Element](../../xmlnodetype/)), который заканчивается на **/>**; иначе, **false**.

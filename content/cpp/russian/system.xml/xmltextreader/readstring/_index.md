## XmlTextReader::ReadString() метод


Считывает содержимое элемента или текстового узла как строку.

```cpp
String System::Xml::XmlTextReader::ReadString() override
```


### Возвращаемое значение

Содержимое элемента или текстового узла. Это может быть пустая строка, если считыватель находится не на элементе или текстовом узле, либо если в текущем контексте больше нет текстового содержимого для возврата. **Примечание:** Текстовый узел может быть как элементом, так и атрибутом.

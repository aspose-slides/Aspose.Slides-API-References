## XmlTextReader::get_CanResolveEntity() метод


Возвращает значение, указывающее, может ли данный считыватель разбирать и разрешать сущности.

```cpp
bool System::Xml::XmlTextReader::get_CanResolveEntity() override
```


### Возвращаемое значение

**true**, если считыватель может разбирать и разрешать сущности; в противном случае **false**. Класс [XmlTextReader](../) всегда возвращает **true**.

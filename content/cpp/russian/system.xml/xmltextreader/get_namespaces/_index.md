## XmlTextReader::get_Namespaces() метод


Возвращает значение, указывающее, нужно ли поддерживать пространства имён.

```cpp
bool System::Xml::XmlTextReader::get_Namespaces()
```


### Возвращаемое значение

**true** для поддержки пространств имён; иначе, **false**. По умолчанию **true**.

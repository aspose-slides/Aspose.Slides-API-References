## XmlTextReader::get_CanReadValueChunk() метод

Возвращает значение, указывающее, реализует ли [XmlTextReader](../) метод [XmlReader::ReadValueChunk](../../xmlreader/readvaluechunk/).

```cpp
bool System::Xml::XmlTextReader::get_CanReadValueChunk() override
```

### Возвращаемое значение

**true** если [XmlTextReader](../) реализует метод [XmlReader::ReadValueChunk](../../xmlreader/readvaluechunk/); иначе **false**. Класс [XmlTextReader](../) всегда возвращает **false**.

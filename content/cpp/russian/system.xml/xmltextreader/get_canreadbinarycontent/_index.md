## XmlTextReader::get_CanReadBinaryContent() метод


Возвращает значение, указывающее, реализует ли [XmlTextReader](../) методы чтения двоичного содержимого.

```cpp
bool System::Xml::XmlTextReader::get_CanReadBinaryContent() override
```


### Возвращаемое значение

**true** если методы чтения двоичного содержимого реализованы; иначе **false**. Класс [XmlTextReader](../) всегда возвращает **true**.

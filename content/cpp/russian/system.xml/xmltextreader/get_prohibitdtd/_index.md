## XmlTextReader::get_ProhibitDtd() метод


Возвращает значение, указывающее, разрешать ли обработку DTD.

```cpp
bool System::Xml::XmlTextReader::get_ProhibitDtd()
```


### Возвращаемое значение

**true** — запретить обработку DTD; иначе **false**. По умолчанию **false**.

Устарело
:   Этот метод устарел. Используйте [XmlTextReader::get_DtdProcessing](../get_dtdprocessing/) вместо него.

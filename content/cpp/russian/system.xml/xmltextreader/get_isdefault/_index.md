## XmlTextReader::get_IsDefault() метод

Возвращает значение, указывающее, является ли текущий узел атрибутом, сгенерированным из значения по умолчанию, определённого в DTD или схеме.

```cpp
bool System::Xml::XmlTextReader::get_IsDefault() override
```

### Возвращаемое значение

Этот метод всегда возвращает **false**. ([XmlTextReader](../) не раскрывает атрибуты по умолчанию.)

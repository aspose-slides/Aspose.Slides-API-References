## XmlNode::get_Prefix() метод


Возвращает префикс пространства имён этого узла.

```cpp
virtual String System::Xml::XmlNode::get_Prefix()
```


### Возвращаемое значение

Префикс пространства имён этого узла. Например, **Prefix** равно **bk** для элемента **<bk:book>**. Если префикса нет, этот метод возвращает [String::Empty](../../../system/string/empty/).

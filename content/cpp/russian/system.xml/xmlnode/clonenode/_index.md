## XmlNode::CloneNode(bool) метод

Создает дубликат узла, когда переопределяется в производном классе.

```cpp
virtual SharedPtr<XmlNode> System::Xml::XmlNode::CloneNode(bool deep)=0
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| deep | **bool** | **true** чтобы рекурсивно клонировать поддерево под указанным узлом; **false** чтобы клонировать только сам узел. |

### Возвращаемое значение

Клонированный узел.

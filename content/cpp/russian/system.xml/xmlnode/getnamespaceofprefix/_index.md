## XmlNode::GetNamespaceOfPrefix(String) метод

Ищет ближайшее объявление **xmlns** для указанного префикса, которое находится в области видимости текущего узла, и возвращает URI пространства имен в объявлении.

```cpp
virtual String System::Xml::XmlNode::GetNamespaceOfPrefix(String prefix)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| prefix | [String](../../../system/string/) | Префикс, чей URI пространства имен вы хотите найти. |

### Возвращаемое значение

URI пространства имен указанного префикса.

## XmlNode::GetPrefixOfNamespace(String) метод

Ищет ближайшее объявление **xmlns** для заданного URI пространства имён, которое находится в области действия текущего узла, и возвращает префикс, определённый в этом объявлении.

```cpp
virtual String System::Xml::XmlNode::GetPrefixOfNamespace(String namespaceURI)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| namespaceURI | [String](../../../system/string/) | URI пространства имён, префикс которого вы хотите найти. |

### Возвращаемое значение

Префикс для указанного URI пространства имён.

## NamespaceHandling enum

Указывает, следует ли удалять дублирующие объявления пространств имён в [XmlWriter](../xmlwriter/).

```cpp
enum class NamespaceHandling
```

### Values

| Имя | Значение | Описание |
| --- | --- | --- |
| Default | 0 | Указывает, что дублирующие объявления пространств имён не будут удалены. |
| OmitDuplicates | 1 | Указывает, что дублирующие объявления пространств имён будут удалены. Чтобы дублирующее пространство имён было удалено, префикс и пространство имён должны совпадать. |

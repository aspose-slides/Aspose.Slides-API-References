## XmlEntity::get_IsReadOnly() метод

Возвращает значение, указывающее, является ли узел только для чтения.

```cpp
bool System::Xml::XmlEntity::get_IsReadOnly() override
```

### Возвращаемое значение

**true** если узел только для чтения; в противном случае **false**. Поскольку узлы [XmlEntity](../) только для чтения, этот метод всегда возвращает **true**.

## XmlAttributeCollection::RemoveAt(int32_t) method

Удаляет атрибут, соответствующий указанному индексу, из коллекции.

```cpp
SharedPtr<XmlAttribute> System::Xml::XmlAttributeCollection::RemoveAt(int32_t i)
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| i | **int32_t** | Индекс узла, который нужно удалить. Первый узел имеет индекс 0. |

### Возвращаемое значение

Возвращает **nullptr**, если атрибут с указанным индексом отсутствует.

## XmlDocumentType::get_InternalSubset() метод

Возвращает значение определения типа документа (DTD) внутреннего подмножества в объявлении DOCTYPE.

```cpp
String System::Xml::XmlDocumentType::get_InternalSubset()
```

### Возвращаемое значение

Внутреннее подмножество DTD в объявлении DOCTYPE. Если внутреннее подмножество DTD отсутствует, возвращается [String::Empty](../../../system/string/empty/).

## XmlValidatingReader::get_HasValue() метод

Возвращает значение, указывающее, может ли текущий узел иметь [XmlValidatingReader::get_Value](../get_value/) отличное от [String::Empty](../../../system/string/empty/).

```cpp
bool System::Xml::XmlValidatingReader::get_HasValue() override
```

### Возвращаемое значение

**true** если узел, на котором в данный момент находится читатель, может иметь **Value**; иначе, **false**.

## XmlValidatingReader::get_SchemaType() метод


Возвращает объект типа схемы.

```cpp
SharedPtr<Object> System::Xml::XmlValidatingReader::get_SchemaType()
```


### Возвращаемое значение

XmlSchemaDatatype, XmlSchemaSimpleType, или XmlSchemaComplexType в зависимости от того, является ли значение узла встроенным типом XML [Schema](../../../system.xml.schema/) definition language (XSD) или пользовательским simpleType или complexType; **nullptr**, если у текущего узла нет типа схемы.

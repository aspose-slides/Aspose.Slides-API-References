## XmlValidatingReader::get_IsDefault() метод


Возвращает значение, указывающее, является ли текущий узел атрибутом, значение которого было сгенерировано из значения по умолчанию, определённого в определении типа документа (DTD) или схеме.

```cpp
bool System::Xml::XmlValidatingReader::get_IsDefault() override
```


### Возвращаемое значение

**true** если текущий узел является атрибутом, значение которого было сгенерировано из значения по умолчанию, определённого в DTD или схеме; **false** если значение атрибута было явно задано.

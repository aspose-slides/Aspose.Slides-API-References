## XmlValidatingReader::set_XmlResolver(const SharedPtr\<System::Xml::XmlResolver\>\&) метод

Устанавливает [XmlResolver](../../xmlresolver/), используемый для разрешения внешних ссылок на определения типов документов (DTD) и ссылки на расположения схем. [XmlResolver](../../xmlresolver/) также используется для обработки любых элементов import или include, найденных в схемах языка определения XML [Schema](../../../system.xml.schema/) (XSD).

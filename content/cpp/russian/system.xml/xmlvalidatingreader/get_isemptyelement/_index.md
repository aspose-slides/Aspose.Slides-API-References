## XmlValidatingReader::get_IsEmptyElement() метод

Возвращает значение, указывающее, является ли текущий узел пустым элементом (например, **<MyElement/>**).

```cpp
bool System::Xml::XmlValidatingReader::get_IsEmptyElement() override
```

### Возвращаемое значение

**true** если текущий узел является элементом (значение [XmlValidatingReader::get_NodeType](../get_nodetype/) равно [XmlNodeType::Element](../../xmlnodetype/)), который заканчивается **/>**; иначе **false**.

## XmlValidatingReader::MoveToElement() метод

Перемещает к элементу, который содержит текущий узел атрибута.

```cpp
bool System::Xml::XmlValidatingReader::MoveToElement() override
```

### Возвращаемое значение

**true** если читатель находится на атрибуте (читатель перемещается к элементу, которому принадлежит атрибут); **false** если читатель не находится на атрибуте (позиция читателя не меняется).

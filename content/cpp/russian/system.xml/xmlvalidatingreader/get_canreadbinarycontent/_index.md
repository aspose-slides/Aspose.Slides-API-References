## XmlValidatingReader::get_CanReadBinaryContent() метод


Возвращает значение, указывающее, реализует ли [XmlValidatingReader](../) методы чтения бинарного содержимого.

```cpp
bool System::Xml::XmlValidatingReader::get_CanReadBinaryContent() override
```


### Возвращаемое значение

**true** если методы чтения бинарного содержимого реализованы; иначе **false**. Класс [XmlValidatingReader](../) возвращает **true**.

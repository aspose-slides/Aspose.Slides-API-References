## XmlNodeReader::get_IsEmptyElement() метод

Возвращает значение, указывающее, является ли текущий узел пустым элементом (например, **<MyElement/>**).

```cpp
bool System::Xml::XmlNodeReader::get_IsEmptyElement() override
```

### Возвращаемое значение

**true** если текущий узел является элементом ([XmlNodeReader::get_NodeType](../get_nodetype/) равно [XmlNodeType::Element](../../xmlnodetype/)) и он заканчивается **/>**; иначе, **false**.

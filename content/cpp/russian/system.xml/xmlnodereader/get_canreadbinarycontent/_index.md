## XmlNodeReader::get_CanReadBinaryContent() method


Возвращает значение, указывающее, реализует ли [XmlNodeReader](../) методы чтения двоичного содержимого.

```cpp
bool System::Xml::XmlNodeReader::get_CanReadBinaryContent() override
```


### Возвращаемое значение

**true** если методы чтения двоичного содержимого реализованы; иначе **false**. Класс [XmlNodeReader](../) всегда возвращает **true**.

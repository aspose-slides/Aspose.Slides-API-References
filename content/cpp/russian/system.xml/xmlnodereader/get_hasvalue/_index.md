## XmlNodeReader::get_HasValue() метод

Возвращает значение, указывающее, может ли текущий узел иметь значение [XmlNodeReader::get_Value](../get_value/).

```cpp
bool System::Xml::XmlNodeReader::get_HasValue() override
```

### Возвращаемое значение

**true** если узел, на котором в данный момент находится считыватель, может иметь **Value**; иначе **false**.

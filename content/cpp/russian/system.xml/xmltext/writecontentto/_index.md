## XmlText::WriteContentTo(const SharedPtr\<XmlWriter\>\&) метод

Сохраняет всех дочерних элементов узла в указанный [XmlWriter](../../xmlwriter/). Узлы [XmlText](../) не имеют дочерних элементов, поэтому этот метод не оказывает действия.

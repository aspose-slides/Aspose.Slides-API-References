## XmlNodeChangedEventArgs::get_Action() метод

Возвращает значение, указывающее, какой тип события изменения узла происходит.

```cpp
XmlNodeChangedAction System::Xml::XmlNodeChangedEventArgs::get_Action()
```

### Возвращаемое значение

Одно из значений перечисления, описывающего событие изменения узла. Значение **Action** не различает, когда событие произошло (до или после). Вы можете создать отдельные обработчики событий для обработки обоих случаев.

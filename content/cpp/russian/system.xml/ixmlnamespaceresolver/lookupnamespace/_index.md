## IXmlNamespaceResolver::LookupNamespace(const String\&) метод

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| prefix | const [String](../../../system/string/)\& | Префикс, для которого необходимо найти URI пространства имен. |

## IXmlNamespaceResolver::LookupPrefix(const String\&) метод

Возвращает префикс, сопоставленный указанному URI пространства имён.

```cpp
virtual String System::Xml::IXmlNamespaceResolver::LookupPrefix(const String &namespaceName)=0
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| namespaceName | const [String](../../../system/string/)\& | URI пространства имён, префикс которого требуется найти. |

### Возвращаемое значение

Префикс, сопоставленный с пространством имён URI; **nullptr**, если пространство имён URI не сопоставлено префиксу.

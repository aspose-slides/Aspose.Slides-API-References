## XmlReader::MoveToNextAttribute() метод

При переопределении в производном классе перемещается к следующему атрибуту.

```cpp
virtual bool System::Xml::XmlReader::MoveToNextAttribute()=0
```

### Возвращаемое значение

**true** если есть следующий атрибут; **false** если больше атрибутов нет.

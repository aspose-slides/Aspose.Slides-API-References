## XmlReader::ReadOuterXml() метод

При переопределении в производном классе читает содержимое, включая разметку, представляющее этот узел и всех его дочерних элементов.

```cpp
virtual String System::Xml::XmlReader::ReadOuterXml()
```

### Возвращаемое значение

Если читатель расположен на узле элемента или атрибута, этот метод возвращает всё XML-содержимое, включая разметку, текущего узла и всех его дочерних элементов; в противном случае он возвращает пустую строку.

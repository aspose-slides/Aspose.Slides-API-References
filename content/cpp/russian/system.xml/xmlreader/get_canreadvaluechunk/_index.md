## XmlReader::get_CanReadValueChunk() метод

Возвращает значение, указывающее, реализует ли [XmlReader](../) метод [XmlReader::ReadValueChunk](../readvaluechunk/).

```cpp
virtual bool System::Xml::XmlReader::get_CanReadValueChunk()
```

### Возвращаемое значение

**true** если [XmlReader](../) реализует метод [XmlReader::ReadValueChunk](../readvaluechunk/); в противном случае **false**.

## XmlReader::ReadString() метод

При переопределении в производном классе читает содержимое элемента или текстового узла как строку. Однако рекомендуется использовать вместо этого метод [XmlReader::ReadElementContentAsString](../readelementcontentasstring/), поскольку он предоставляет более простой способ выполнения этой операции.

```cpp
virtual String System::Xml::XmlReader::ReadString()
```

### Возвращаемое значение

Содержимое элемента или пустая строка.

## XmlReader::get_HasAttributes() метод

Возвращает значение, указывающее, есть ли у текущего узла какие-либо атрибуты.

```cpp
virtual bool System::Xml::XmlReader::get_HasAttributes()
```

### Возвращаемое значение

**true** если у текущего узла есть атрибуты; в противном случае **false**.

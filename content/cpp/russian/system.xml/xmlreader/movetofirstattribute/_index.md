## XmlReader::MoveToFirstAttribute() метод

При переопределении в производном классе перемещается к первому атрибуту.

```cpp
virtual bool System::Xml::XmlReader::MoveToFirstAttribute()=0
```

### Возвращаемое значение

**true** если атрибут существует (читатель перемещается к первому атрибуту); в противном случае **false** (позиция читателя не меняется).

## XmlReader::get_XmlSpace() метод

virtual System::Xml::XmlSpace System::Xml::XmlReader::get_XmlSpace()

Одно из значений XmlSpace. Если область **xml:space** не существует, это значение по умолчанию равно [XmlSpace::None](../../xmlspace/).

## XmlReader::get_HasValue() method


При переопределении в производном классе возвращает значение, указывающее, может ли текущий узел иметь значение [XmlReader::get_Value](../get_value/).

```cpp
virtual bool System::Xml::XmlReader::get_HasValue()
```


### Возвращаемое значение

**true** если узел, на котором в данный момент находится читатель, может иметь **Value**; в противном случае **false**. Если **false**, у узла значение [String::Empty](../../../system/string/empty/).

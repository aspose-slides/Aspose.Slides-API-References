## XmlReader::get_NamespaceURI() метод

При переопределении в производном классе возвращает URI пространства имён (как определено в спецификации W3C Namespace) узла, на котором находится считыватель.

```cpp
virtual String System::Xml::XmlReader::get_NamespaceURI()=0
```

### Возвращаемое значение

URI пространства имён текущего узла; иначе пустая строка.

## XmlDeclaration::get_Standalone() метод

Возвращает значение атрибута standalone.

```cpp
String System::Xml::XmlDeclaration::get_Standalone()
```

### Возвращаемое значение

Допустимыми значениями являются **yes**, если все объявления сущностей, необходимые документу XML, находятся в документе, или **no**, если требуется внешняя декларация типа документа (DTD). Если атрибут standalone отсутствует в объявлении XML, этот метод возвращает [String::Empty](../../../system/string/empty/).

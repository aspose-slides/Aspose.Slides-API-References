## XmlAttribute::get_Specified() метод

Возвращает значение, указывающее, было ли значение атрибута явно установлено.

```cpp
virtual bool System::Xml::XmlAttribute::get_Specified()
```

### Возвращаемое значение

**true** если этот атрибут был явно задан со значением в исходном документе экземпляра; в противном случае **false**. Значение **false** указывает, что значение атрибута получено из DTD.

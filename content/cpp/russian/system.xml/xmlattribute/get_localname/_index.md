## XmlAttribute::get_LocalName() метод

Возвращает локальное имя узла.

```cpp
String System::Xml::XmlAttribute::get_LocalName() override
```

### Возвращаемое значение

Имя узла атрибута без префикса. В следующем примере **<book bk:genre= 'novel'>**, **LocalName** атрибута равно **genre**.

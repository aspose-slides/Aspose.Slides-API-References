## XmlElement::get_IsEmpty() метод


Возвращает формат тэга элемента.

```cpp
bool System::Xml::XmlElement::get_IsEmpty()
```


### Возвращаемое значение

**true** если элемент должен быть сериализован в коротком формате тэга "<tt><item/></tt>"; **false** для длинного формата "<tt><item></item></tt>".

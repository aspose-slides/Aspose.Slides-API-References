## XmlElement::set_IsEmpty(bool) метод

Устанавливает формат тега элемента.

```cpp
void System::Xml::XmlElement::set_IsEmpty(bool value)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | **bool** | Значение для установки. При установке этого значения, если оно установлено в **true**, дочерние элементы удаляются, и элемент сериализуется в коротком формате тега. Если оно установлено в **false**, значение меняется (независимо от того, содержит ли элемент данные); если элемент пуст, он сериализуется в длинном формате. |

## XmlElement::get_HasAttributes() метод


Возвращает значение **bool**, указывающее, имеет ли текущий узел какие-либо атрибуты.

```cpp
virtual bool System::Xml::XmlElement::get_HasAttributes()
```


### Возвращаемое значение

**true** если у текущего узла есть атрибуты; в противном случае **false**.

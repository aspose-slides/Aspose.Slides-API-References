## XmlNodeOrder enum

Описывает порядок узлов в документе относительно второго узла.

```cpp
enum class XmlNodeOrder
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Before | 0 | Текущий узел этого навигатора находится перед текущим узлом поставляемого навигатора. |
| After | 1 | Текущий узел этого навигатора находится после текущего узла поставляемого навигатора. |
| Same | 2 | Оба навигатора находятся на одном и том же узле. |
| Unknown | 3 | Позицию узлов нельзя определить в порядке документа относительно друг друга. Это может происходить, если два узла находятся в разных деревьях. |

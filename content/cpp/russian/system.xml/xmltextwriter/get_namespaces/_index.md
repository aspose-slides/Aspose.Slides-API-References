## XmlTextWriter::get_Namespaces() метод

Возвращает значение, указывающее, следует ли поддерживать пространства имён.

```cpp
bool System::Xml::XmlTextWriter::get_Namespaces()
```

### Возвращаемое значение

**true** для поддержки пространств имён; иначе, **false**. По умолчанию **true**.

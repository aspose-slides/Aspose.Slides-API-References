## XmlTextWriter::get_XmlSpace() метод

Возвращает XmlSpace, представляющий текущую область **xml:space**.

```cpp
System::Xml::XmlSpace System::Xml::XmlTextWriter::get_XmlSpace() override
```

### Возвращаемое значение

XmlSpace, представляющий текущую область **xml:space**.

| Значение | Описание |
| --- | --- |
| None | Это значение по умолчанию, если область `xml:space` не существует. |
| Default | Текущая область `xml:space="default"`. |
| Preserve | Текущая область `xml:space="preserve"`. |

## XmlTextWriter::get_Indentation() метод

Возвращает, сколько **IndentChars** записывать для каждого уровня иерархии, когда [XmlTextWriter::set_Formatting](../set_formatting/) установлен в [Formatting::Indented](../../formatting/).

```cpp
int32_t System::Xml::XmlTextWriter::get_Indentation()
```

### Возвращаемое значение

Количество **IndentChars** для каждого уровня. По умолчанию 2.

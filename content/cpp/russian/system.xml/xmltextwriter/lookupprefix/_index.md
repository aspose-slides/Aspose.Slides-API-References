## XmlTextWriter::LookupPrefix(String) метод


Возвращает ближайший префикс, определенный в текущей области пространства имен, для указанного URI пространства имен.

```cpp
String System::Xml::XmlTextWriter::LookupPrefix(String ns) override
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| ns | [String](../../../system/string/) | URI пространства имен, префикс которого вы хотите найти. |

### Возвращаемое значение

Соответствующий префикс. Или **nullptr**, если подходящий URI пространства имен не найден в текущей области.

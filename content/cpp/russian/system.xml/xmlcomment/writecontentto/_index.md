## XmlComment::WriteContentTo(const SharedPtr\<XmlWriter\>\&) метод

Сохраняет всех дочерних узлов узла в указанный [XmlWriter](../../xmlwriter/). Поскольку узлы комментариев не имеют дочерних элементов, этот метод не оказывает эффекта.

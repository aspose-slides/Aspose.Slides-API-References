## XmlCharacterData::Substring(int32_t, int32_t) метод


Получает подстроку полной строки по заданному диапазону.

```cpp
virtual String System::Xml::XmlCharacterData::Substring(int32_t offset, int32_t count)
```


### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| offset | **int32_t** | Позиция в строке, с которой начинается извлечение. Смещение, равное нулю, указывает, что начальная точка находится в начале данных. |
| count | **int32_t** | Количество символов для извлечения. |

### Возвращаемое значение

Подстрока, соответствующая указанному диапазону.

## Перечисление Formatting

Указывает параметры форматирования для [XmlTextWriter](../xmltextwriter/).

```cpp
enum class Formatting
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | 0 | Не применяется специальное форматирование. Это значение по умолчанию. |
| Indented | 1 | Отступает дочерние элементы в соответствии с настройками [XmlTextWriter::set_Indentation](../xmltextwriter/set_indentation/) и [XmlTextWriter::set_IndentChar](../xmltextwriter/set_indentchar/). |

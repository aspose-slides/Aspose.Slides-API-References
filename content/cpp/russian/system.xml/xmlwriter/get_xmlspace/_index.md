## XmlWriter::get_XmlSpace() метод


При переопределении в производном классе возвращает объект XmlSpace, представляющий текущий **xml:space** диапазон.

```cpp
virtual System::Xml::XmlSpace System::Xml::XmlWriter::get_XmlSpace()
```


### Возвращаемое значение

Объект XmlSpace, представляющий текущий **xml:space** диапазон.



| Value | Meaning |
| --- | --- |
| `None` | Это значение по умолчанию, если диапазон `xml:space` отсутствует. |
| `Default` | Текущий диапазон имеет значение `xml:space="default"`. |
| `Preserve` | Текущий диапазон имеет значение `xml:space="preserve"`. |

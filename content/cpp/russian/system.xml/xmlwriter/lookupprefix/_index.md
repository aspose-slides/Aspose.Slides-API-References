## XmlWriter::LookupPrefix(String) метод

При переопределении в производном классе возвращает ближайший префикс, определённый в текущей области видимости пространства имён, для указанного URI пространства имён.

```cpp
virtual String System::Xml::XmlWriter::LookupPrefix(String ns)=0
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| ns | [String](../../../system/string/) | URI пространства имён, префикс которого необходимо найти. |

### Возвращаемое значение

Соответствующий префикс или **nullptr**, если в текущей области видимости не найдено соответствующего URI пространства имён.

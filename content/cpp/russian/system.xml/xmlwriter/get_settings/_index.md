## XmlWriter::get_Settings() метод

Возвращает объект [XmlWriterSettings](../../xmlwritersettings/), используемый для создания этого экземпляра [XmlWriter](../).

```cpp
virtual SharedPtr<XmlWriterSettings> System::Xml::XmlWriter::get_Settings()
```

### Возвращаемое значение

Объект [XmlWriterSettings](../../xmlwritersettings/), используемый для создания этого экземпляра записывающего устройства. Если этот записывающий объект не был создан с помощью метода [XmlWriter::Create](../create/), этот метод возвращает **nullptr**.

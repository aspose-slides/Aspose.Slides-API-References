## XmlNamespaceManager::HasNamespace(String) метод

Возвращает значение, указывающее, определён ли указанный префикс для пространства имён в текущей добавленной области.

```cpp
virtual bool System::Xml::XmlNamespaceManager::HasNamespace(String prefix)
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| prefix | [String](../../../system/string/) | Префикс пространства имён, которое нужно найти. |

### Возвращаемое значение

**true**, если пространство имён определено; в противном случае **false**.

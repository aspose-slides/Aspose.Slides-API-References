## XmlNamespaceManager::RemoveNamespace(String, String) метод

Удаляет указанный пространство имён для заданного префикса.

```cpp
virtual void System::Xml::XmlNamespaceManager::RemoveNamespace(String prefix, String uri)
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| prefix | [String](../../../system/string/) | Префикс для пространства имён. |
| uri | [String](../../../system/string/) | Пространство имён, которое нужно удалить для данного префикса. Удаляемое пространство имён берётся из текущей области видимости. Пространства имён за пределами текущей области игнорируются. |

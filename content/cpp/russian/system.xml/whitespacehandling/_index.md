## WhitespaceHandling перечисление

Указывает, как следует обрабатывать пробелы.

```cpp
enum class WhitespaceHandling
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| All | 0 | Возвращать узлы **Whitespace** и **SignificantWhitespace**. Это значение по умолчанию. |
| Significant | 1 | Возвращать только узлы **SignificantWhitespace**. |
| None | 2 | Не возвращать ни **Whitespace**, ни **SignificantWhitespace** узлы. |

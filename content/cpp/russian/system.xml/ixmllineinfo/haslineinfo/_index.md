## IXmlLineInfo::HasLineInfo() метод


Возвращает значение, указывающее, может ли класс возвращать информацию о строках.

```cpp
virtual bool System::Xml::IXmlLineInfo::HasLineInfo()=0
```


### Возвращаемое значение

**true** если [IXmlLineInfo::get_LineNumber](../get_linenumber/) и [IXmlLineInfo::get_LinePosition](../get_lineposition/) могут быть предоставлены; в противном случае **false**.

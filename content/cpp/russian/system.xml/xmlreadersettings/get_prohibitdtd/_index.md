## XmlReaderSettings::get_ProhibitDtd() method

Возвращает значение, указывающее, следует ли запрещать обработку определений типов документов (DTD).

```cpp
bool System::Xml::XmlReaderSettings::get_ProhibitDtd()
```

### Возврат значения

**true** для запрета обработки DTD; иначе **false**. По умолчанию **true**.

Устарел
:   Этот метод устарел. Используйте [XmlTextReader::get_DtdProcessing](../../xmltextreader/get_dtdprocessing/) вместо этого.

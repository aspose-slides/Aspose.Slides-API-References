## XmlReaderSettings::get_CloseInput() метод


Возвращает значение, указывающее, следует ли закрывать основной поток или TextReader при закрытии читателя.

```cpp
bool System::Xml::XmlReaderSettings::get_CloseInput()
```


### Возвращаемое значение

**true** чтобы закрыть основной поток или TextReader при закрытии читателя; в противном случае **false**. По умолчанию **false**.

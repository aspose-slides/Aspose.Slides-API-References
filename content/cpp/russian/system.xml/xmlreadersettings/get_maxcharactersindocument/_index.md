## XmlReaderSettings::get_MaxCharactersInDocument() метод

Возвращает значение, указывающее максимальное допустимое количество символов в документе XML. Нулевое (0) значение означает отсутствие ограничений размера документа XML. Ненулевое значение задаёт максимальный размер в символах.

```cpp
int64_t System::Xml::XmlReaderSettings::get_MaxCharactersInDocument()
```

### Возвращаемое значение

Максимальное допустимое количество символов в документе XML. По умолчанию 0.

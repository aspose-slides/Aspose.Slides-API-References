## XmlReaderSettings::set_MaxCharactersInDocument(int64_t) метод

Устанавливает значение, указывающее максимальное допустимое количество символов в XML-документе. Значение ноль (0) означает отсутствие ограничений на размер XML-документа. Ненулевое значение задаёт максимальный размер в символах.

```cpp
void System::Xml::XmlReaderSettings::set_MaxCharactersInDocument(int64_t value)
```

| --- | --- | --- |

## XmlReaderSettings::get_MaxCharactersFromEntities() метод

Возвращает значение, указывающее максимальное допустимое количество символов в документе, получаемое в результате раскрытия сущностей.

```cpp
int64_t System::Xml::XmlReaderSettings::get_MaxCharactersFromEntities()
```

### Возвращаемое значение

Максимальное допустимое количество символов из раскрытых сущностей. По умолчанию — 0.

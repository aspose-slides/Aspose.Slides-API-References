## XmlReaderSettings::get_ValidationType() метод

Возвращает значение, указывающее, будет ли [XmlReader](../../xmlreader/) выполнять проверку или назначение типа при чтении.

```cpp
System::Xml::ValidationType System::Xml::XmlReaderSettings::get_ValidationType()
```

### Return Value

Одно из значений ValidationType, которое указывает, будет ли [XmlReader](../../xmlreader/) выполнять проверку или назначение типа при чтении. По умолчанию — [ValidationType::None](../../validationtype/).

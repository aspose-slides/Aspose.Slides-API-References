## XmlConvert::VerifyXmlChars(const String\&) метод

Возвращает переданную строку, если все символы и символы пар суррогатов в аргументе строки являются допустимыми XML-символами; в противном случае выбрасывается XmlException с информацией о первом обнаруженном недопустимом символе.

```cpp
static String System::Xml::XmlConvert::VerifyXmlChars(const String &content)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| content | const [String](../../../system/string/)\& | [String](../../../system/string/) содержащий символы для проверки. |

### Возвращаемое значение

Возвращает переданную строку, если все символы и символы пар суррогатов в аргументе строки являются допустимыми XML-символами; в противном случае выбрасывается XmlException с информацией о первом обнаруженном недопустимом символе.

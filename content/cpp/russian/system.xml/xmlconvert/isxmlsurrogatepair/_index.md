## XmlConvert::IsXmlSurrogatePair(char16_t, char16_t) метод

Проверяет, является ли переданная пара суррогатных символов допустимым символом XML.

```cpp
static bool System::Xml::XmlConvert::IsXmlSurrogatePair(char16_t lowChar, char16_t highChar)
```

### Arguments

| Parameter | Type | Description |
| --- | --- | --- |
| lowChar | char16_t | Суррогатный символ для проверки. |
| highChar | char16_t | Суррогатный символ для проверки. |

### Return Value

**true** если переданная пара суррогатных символов является допустимым символом XML; в противном случае **false**.

## XmlConvert::VerifyWhitespace(const String\&) метод


Возвращает переданный экземпляр строки, если все символы в аргументе строки являются допустимыми пробельными символами.

```cpp
static String System::Xml::XmlConvert::VerifyWhitespace(const String &content)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| content | const [String](../../../system/string/)\& | [String](../../../system/string/) для проверки. |

### Возвращаемое значение

Переданный экземпляр строки, если все символы в аргументе строки являются допустимыми пробельными символами, в противном случае **nullptr**.

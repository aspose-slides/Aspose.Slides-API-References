## XmlConvert::IsWhitespaceChar(char16_t) метод


Проверяет, является ли переданный символ допустимым пробельным символом XML.

```cpp
static bool System::Xml::XmlConvert::IsWhitespaceChar(char16_t ch)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| ch | char16_t | Символ для проверки. |

### Возвращаемое значение

**true**, если переданный символ является допустимым пробельным символом XML; в противном случае **false**.

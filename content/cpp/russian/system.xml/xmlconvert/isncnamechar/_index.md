## XmlConvert::IsNCNameChar(char16_t) метод

Проверяет, является ли переданный символ допустимым символом без двоеточия.

```cpp
static bool System::Xml::XmlConvert::IsNCNameChar(char16_t ch)
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| ch | char16_t | Символ, который необходимо проверить как символ без двоеточия. |

### Возвращаемое значение

**true** если символ является допустимым символом без двоеточия; в противном случае **false**.

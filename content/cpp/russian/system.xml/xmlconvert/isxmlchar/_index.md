## XmlConvert::IsXmlChar(char16_t) метод

Проверяет, является ли переданный символ допустимым символом XML.

```cpp
static bool System::Xml::XmlConvert::IsXmlChar(char16_t ch)
```

### Arguments

| Parameter | Type | Description |
| --- | --- | --- |
| ch | char16_t | Символ для проверки. |

### Return Value

**true** если переданный символ является допустимым символом XML; в противном случае **false**.

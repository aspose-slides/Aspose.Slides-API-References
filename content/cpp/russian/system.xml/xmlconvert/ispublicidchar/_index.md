## XmlConvert::IsPublicIdChar(char16_t) метод

Возвращает переданный символ, если символ в аргументе является допустимым символом публичного идентификатора, иначе **nullptr**.

```cpp
static bool System::Xml::XmlConvert::IsPublicIdChar(char16_t ch)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| ch | char16_t | [Char](../../../system/char/) объект для проверки. |

### Возвращаемое значение

Переданный символ, если он является допустимым символом публичного идентификатора, иначе **nullptr**.

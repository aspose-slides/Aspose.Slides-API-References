## XmlConvert::VerifyNCName(const String\&) метод


Проверяет, что имя является действительным **NCName** в соответствии с рекомендацией W3C Extended Markup Language. **NCName** — это имя, которое не может содержать двоеточие.

```cpp
static String System::Xml::XmlConvert::VerifyNCName(const String &name)
```


### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| name | const [String](../../../system/string/)\& | Имя для проверки. |

### Возвращаемое значение

Имя, если оно является действительным NCName.

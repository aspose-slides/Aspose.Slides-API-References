## XmlConvert::ToString(DateTimeOffset, const String\&) метод

Преобразует переданный [DateTimeOffset](../../../system/datetimeoffset/) в [String](../../../system/string/) в указанном формате.

```cpp

```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | [DateTimeOffset](../../../system/datetimeoffset/) | [DateTimeOffset](../../../system/datetimeoffset/) для преобразования. |
| format | const [String](../../../system/string/)\& | Формат, в который **s** преобразуется. Параметр формата может быть любой подсубстрокой рекомендации W3C для типа XML dateTime. Для получения дополнительной информации см. раздел [dateTime](https://www.w3.org/TR/xmlschema-2/#dateTime) спецификации XML [Schema](../../../system.xml.schema/). |

### Возвращаемое значение

[String](../../../system/string/) представление в указанном формате переданного [DateTimeOffset](../../../system/datetimeoffset/).

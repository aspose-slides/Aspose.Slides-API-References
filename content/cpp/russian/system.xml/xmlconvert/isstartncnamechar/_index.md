## XmlConvert::IsStartNCNameChar(char16_t) метод

Проверяет, является ли переданный символ допустимым символом начала имени.

```cpp
static bool System::Xml::XmlConvert::IsStartNCNameChar(char16_t ch)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| ch | char16_t | Символ для проверки. |

### Возвращаемое значение

**true** если символ является допустимым символом начала имени; иначе — **false**.

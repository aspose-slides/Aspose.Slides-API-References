## XmlWriterSettings::get_NewLineChars() метод

Возвращает строку символов, используемую для разрывов строк.

```cpp
String System::Xml::XmlWriterSettings::get_NewLineChars()
```

### Возвращаемое значение

Строка символов, используемая для разрывов строк. Её можно задать любым строковым значением. Однако, чтобы обеспечить корректный XML, следует указывать только допустимые пробельные символы, такие как пробелы, табуляции, возвраты каретки или перевод строки. По умолчанию используется **\r\n** (возврат каретки, новая строка).

## XmlWriterSettings::get_NamespaceHandling() метод


Возвращает значение, указывающее, должен ли [XmlWriter](../../xmlwriter/) удалять дублирующие объявления пространств имён при записи XML-контента. Поведение по умолчанию заключается в том, что писатель выводит все объявления пространств имён, присутствующие в разрешателе пространств имён писателя.

```cpp
System::Xml::NamespaceHandling System::Xml::XmlWriterSettings::get_NamespaceHandling()
```


### Возвращаемое значение

Перечисление NamespaceHandling, используемое для указания, следует ли удалять дублирующие объявления пространств имён в [XmlWriter](../../xmlwriter/).

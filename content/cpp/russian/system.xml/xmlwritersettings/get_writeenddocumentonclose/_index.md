## XmlWriterSettings::get_WriteEndDocumentOnClose() метод

Возвращает значение, которое указывает, будет ли [XmlWriter](../../xmlwriter/) добавлять закрывающие теги ко всем незакрытым тегам элементов, когда вызывается метод [XmlWriter::Close](../../xmlwriter/close/).

```cpp
bool System::Xml::XmlWriterSettings::get_WriteEndDocumentOnClose()
```

### Возвращаемое значение

**true** если все незакрытые теги элементов будут закрыты; в противном случае, **false**. Значение по умолчанию — **true**.

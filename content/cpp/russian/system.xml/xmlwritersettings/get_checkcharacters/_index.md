## XmlWriterSettings::get_CheckCharacters() метод

Возвращает значение, указывающее, должен ли XML writer проверять, чтобы все символы в документе соответствовали разделу \"2.2 Characters\" спецификации W3C [XML 1.0 Recommendation](https://www.w3.org/TR/REC-xml/#charsets).

```cpp
bool System::Xml::XmlWriterSettings::get_CheckCharacters()
```

### Возвращаемое значение

**true** — выполнить проверку символов; в противном случае **false**. По умолчанию **true**.

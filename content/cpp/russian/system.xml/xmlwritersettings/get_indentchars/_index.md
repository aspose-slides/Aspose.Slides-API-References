## XmlWriterSettings::get_IndentChars() метод


Возвращает строку символов, используемую для отступов. Этот параметр используется, когда значение [XmlWriterSettings::set_Indent](../set_indent/) установлено в **true**.

```cpp
String System::Xml::XmlWriterSettings::get_IndentChars()
```


### Возвращаемое значение

Строка символов, используемая для отступов. Ее можно установить в любое строковое значение. Однако, чтобы обеспечить корректный XML, следует указывать только допустимые пробельные символы, такие как пробелы, табуляции, возвраты каретки или символы перехода на новую строку. По умолчанию используется два пробела.

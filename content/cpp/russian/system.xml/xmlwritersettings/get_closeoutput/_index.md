## XmlWriterSettings::get_CloseOutput() метод

Возвращает значение, указывающее, должен ли [XmlWriter](../../xmlwriter/) также закрывать основной поток или TextWriter, когда вызывается метод [XmlWriter::Close](../../xmlwriter/close/).

```cpp
bool System::Xml::XmlWriterSettings::get_CloseOutput()
```

### Возвращаемое значение

**true** чтобы также закрывать основной поток или TextWriter; иначе **false**. Значение по умолчанию — **false**.

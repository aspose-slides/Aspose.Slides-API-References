## XmlWriterSettings::get_DoNotEscapeUriAttributes() метод


Возвращает значение, указывающее, не экранирует ли [XmlWriter](../../xmlwriter/) атрибуты URI.

```cpp
bool System::Xml::XmlWriterSettings::get_DoNotEscapeUriAttributes()
```


### Возвращаемое значение

**true** если [XmlWriter](../../xmlwriter/) не экранирует атрибуты URI; в противном случае **false**. По умолчанию **false**.

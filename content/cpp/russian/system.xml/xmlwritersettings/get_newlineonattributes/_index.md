## XmlWriterSettings::get_NewLineOnAttributes() метод


Возвращает значение, указывающее, следует ли писать атрибуты на новой строке.

```cpp
bool System::Xml::XmlWriterSettings::get_NewLineOnAttributes()
```


### Возвращаемое значение

**true** — писать атрибуты в отдельных строках; иначе, **false**. По умолчанию — **false**.

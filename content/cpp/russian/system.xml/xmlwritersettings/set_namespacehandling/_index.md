## XmlWriterSettings::set_NamespaceHandling(System::Xml::NamespaceHandling) метод

Устанавливает значение, указывающее, должна ли [XmlWriter](../../xmlwriter/) удалять дублирующие объявления пространств имён при записи XML-содержимого. Поведение по умолчанию заключается в том, что запись выводит все объявления пространств имён, присутствующие в разрешателе пространств имён записывающего объекта.

```cpp
void System::Xml::XmlWriterSettings::set_NamespaceHandling(System::Xml::NamespaceHandling value)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | [System::Xml::NamespaceHandling](../../namespacehandling/) | Значение value, которое необходимо установить. |

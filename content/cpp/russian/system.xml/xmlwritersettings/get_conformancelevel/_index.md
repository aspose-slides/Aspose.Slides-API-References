## XmlWriterSettings::get_ConformanceLevel() метод

Возвращает уровень соответствия, который XML-writer проверяет в выводе XML.
```cpp
System::Xml::ConformanceLevel System::Xml::XmlWriterSettings::get_ConformanceLevel()
```

### Возвращаемое значение

Одно из значений перечисления, которое указывает уровень соответствия (document, fragment, or automatic detection). По умолчанию [ConformanceLevel::Document](../../conformancelevel/).

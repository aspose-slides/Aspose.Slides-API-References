## Метод XmlWriterSettings::get_OmitXmlDeclaration()


Возвращает значение, указывающее, следует ли опустить объявление XML.

```cpp
bool System::Xml::XmlWriterSettings::get_OmitXmlDeclaration()
```


### Возвращаемое значение

**true** чтобы опустить объявление XML; иначе **false**. По умолчанию **false**, объявление XML записывается.

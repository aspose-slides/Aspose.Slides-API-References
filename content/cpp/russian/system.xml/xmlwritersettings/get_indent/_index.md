## XmlWriterSettings::get_Indent() метод

Возвращает значение, указывающее, следует ли делать отступы у элементов.

```cpp
bool System::Xml::XmlWriterSettings::get_Indent()
```

### Возвращаемое значение

**true** чтобы записывать отдельные элементы на новых строках и делать отступ; в противном случае **false**. По умолчанию **false**.

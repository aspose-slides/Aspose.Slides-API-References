## XmlWriterSettings::get_Encoding() метод

Возвращает тип используемой текстовой кодировки.

```cpp
SharedPtr<System::Text::Encoding> System::Xml::XmlWriterSettings::get_Encoding()
```

### Возвращаемое значение

Текстовая кодировка, которую следует использовать. По умолчанию используется Encoding::get_UTF8.

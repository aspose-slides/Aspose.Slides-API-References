## IEnumerator::MoveNext() метод

Перемещает перечислитель к следующему элементу. Если ранее ни один элемент не был указан, устанавливает ссылку на первый доступный элемент. Если достигнут конец контейнера, ничего не делает.

```cpp
virtual bool System::Collections::Generic::IEnumerator<T>::MoveNext()=0
```

### Возвращаемое значение

True, если после вызова доступен какой-то элемент, иначе false.

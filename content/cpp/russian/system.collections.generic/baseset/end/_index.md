## BaseSet::end() const метод

Получает итератор для несуществующего элемента, находящегося за пределами конца константно-квалифицированной коллекции.

```cpp
const_iterator System::Collections::Generic::BaseSet<T, SET_T>::end() const noexcept
```

### Возвращаемое значение

Итератор, указывающий на теоретический элемент, размещённый после конечного элемента в константно-квалифицированном экземпляре хеш-множества.

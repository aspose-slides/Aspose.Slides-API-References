## BaseSet::cend() const метод

Получает итератор для несуществующего константно-квалифицированного элемента, находящегося за пределами коллекции.

```cpp
const_iterator System::Collections::Generic::BaseSet<T, SET_T>::cend() const noexcept
```

### Возвращаемое значение

Итератор, указывающий на теоретический константно-квалифицированный элемент, расположенный после конечного элемента в хеш-множестве.

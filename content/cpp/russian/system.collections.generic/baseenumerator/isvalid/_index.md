## BaseEnumerator::IsValid() const метод

Проверяет, был ли вызван [MoveNext()](../movenext/) и не достигнут конец.

```cpp
bool System::Collections::Generic::BaseEnumerator<Container, Element>::IsValid() const
```

### Возвращаемое значение

false, если [MoveNext()](../movenext/) не был вызван вообще или достигнут конец контейнера; иначе true.

## Stack::Contains(const T\&) const метод

Проверяет, присутствует ли определённый элемент в контейнере; для сравнения используется оператор ==.

```cpp
virtual bool System::Collections::Generic::Stack<T>::Contains(const T &item) const
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| item | const T\& | Элемент, который нужно найти. |

### Возвращаемое значение

True если элемент найден, false в противном случае.

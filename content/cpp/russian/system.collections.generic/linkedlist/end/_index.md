## LinkedList::end() метод

Получает итератор для несуществующего элемента, находящегося за пределами конца коллекции.

```cpp
iterator System::Collections::Generic::LinkedList<T>::end() noexcept
```

### Возвращаемое значение

Итератор, указывающий на теоретический элемент, расположенный после конечного элемента в связном списке.

Получает итератор для несуществующего элемента, находящегося за пределами конца const-квалифицированной коллекции.

```cpp
const_iterator System::Collections::Generic::LinkedList<T>::end() const noexcept
```

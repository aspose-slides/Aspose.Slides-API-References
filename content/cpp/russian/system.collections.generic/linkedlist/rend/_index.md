## LinkedList::rend() метод

Получает обратный итератор для несуществующего элемента перед началом коллекции.

```cpp
reverse_iterator System::Collections::Generic::LinkedList<T>::rend() noexcept
```

### Возвращаемое значение

Обратный итератор, указывающий на теоретический элемент, предшествующий первому элементу в связанном списке (последний в обратном порядке).

## LinkedList::rend() const метод

Получает обратный итератор для несуществующего элемента перед началом константно-квалифицированной коллекции.

```cpp
const_reverse_iterator System::Collections::Generic::LinkedList<T>::rend() const noexcept
```

### Возвращаемое значение

Обратный итератор, указывающий на теоретический элемент, предшествующий первому элементу в константно-квалифицированном связанном списке (последний в обратном порядке).

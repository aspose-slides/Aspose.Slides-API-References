## LinkedList::cend() const метод

Получает итератор для несуществующего const-qualified элемента, расположенного за концом коллекции.

```cpp
const_iterator System::Collections::Generic::LinkedList<T>::cend() const noexcept
```

### Возвращаемое значение

Итератор, указывающий на теоретический const-qualified элемент, размещённый после конечного элемента в связанном списке.

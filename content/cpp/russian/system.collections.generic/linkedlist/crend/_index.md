## LinkedList::crend() const метод

Получает обратный итератор для несуществующего const-квалифицированного элемента до начала коллекции.

```cpp
const_reverse_iterator System::Collections::Generic::LinkedList<T>::crend() const noexcept
```

### Возвращаемое значение

Обратный итератор, указывающий на теоретический const-квалифицированный элемент, предшествующий первому элементу в связанном списке (последний в обратном порядке).

## ISet::IsSupersetOf(IEnumerablePtr) метод

Проверяет, является ли текущий набор надмножеством другого контейнера.

```cpp
virtual bool System::Collections::Generic::ISet<T>::IsSupersetOf(IEnumerablePtr other)=0
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| other | [IEnumerablePtr](../ienumerableptr/) | Подмножество для проверки. |

### Возвращаемое значение

True если все элементы в **other** присутствуют в наборе, иначе false.

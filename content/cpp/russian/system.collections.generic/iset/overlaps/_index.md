## ISet::Overlaps(IEnumerablePtr) метод

Проверяет, перекрывается ли набор с другим контейнером.

```cpp
virtual bool System::Collections::Generic::ISet<T>::Overlaps(IEnumerablePtr other)=0
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| other | [IEnumerablePtr](../ienumerableptr/) | Контейнер для проверки. |

### Возвращаемое значение

True, если текущий набор и **other** имеют хотя бы один общий элемент, false в противном случае.

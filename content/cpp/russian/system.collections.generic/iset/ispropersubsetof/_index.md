## ISet::IsProperSubsetOf(IEnumerablePtr) метод

Проверяет, является ли текущий набор строгим подмножеством другого контейнера.

```cpp
virtual bool System::Collections::Generic::ISet<T>::IsProperSubsetOf(IEnumerablePtr other)=0
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| other | [IEnumerablePtr](../ienumerableptr/) | Супермножество для проверки. |

### Возвращаемое значение

True если все элементы текущего набора присутствуют в **other** и **other** содержит больше элементов, чем текущий набор, false иначе.

## ISet::IsSubsetOf(IEnumerablePtr) метод

Проверяет, является ли текущий набор подмножеством другого контейнера.

```cpp
virtual bool System::Collections::Generic::ISet<T>::IsSubsetOf(IEnumerablePtr other)=0
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| other | [IEnumerablePtr](../ienumerableptr/) | Супермножество для проверки. |

### Возвращаемое значение

True, если все элементы текущего набора присутствуют в **other**, иначе false.

## ReverseEnumerator::IsValid() const метод

Проверяет, был ли [MoveNext()](../movenext/) вызван и конец не был достигнут.

```cpp
bool System::Collections::Generic::ReverseEnumerator<Container, Element>::IsValid() const
```

### Возвращаемое значение

false если [MoveNext()](../movenext/) не был вызван вообще или конец контейнера достигнут; true в противном случае.

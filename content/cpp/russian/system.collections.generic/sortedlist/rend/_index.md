## метод SortedList::rend()

Получает обратный итератор для несуществующего элемента перед началом коллекции.

```cpp
reverse_iterator System::Collections::Generic::SortedList<TKey, TValue>::rend() noexcept
```

### Возвращаемое значение

Обратный итератор, указывающий на теоретический элемент, предшествующий первому элементу в карте (последний в обратном порядке).

## метод SortedList::rend() const

Получает обратный итератор для несуществующего элемента перед началом константно-квалифицированной коллекции.

```cpp
const_reverse_iterator System::Collections::Generic::SortedList<TKey, TValue>::rend() const noexcept
```

### Возвращаемое значение

Обратный итератор, указывающий на теоретический элемент, предшествующий первому элементу в константно-квалифицированной карте (последний в обратном порядке).

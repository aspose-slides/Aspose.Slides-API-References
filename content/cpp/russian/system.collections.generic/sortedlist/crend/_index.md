## SortedList::crend() const метод


Получает обратный итератор для несуществующего const-qualified элемента перед началом коллекции.

```cpp
const_reverse_iterator System::Collections::Generic::SortedList<TKey, TValue>::crend() const noexcept
```


### Возвращаемое значение

Обратный итератор, указывающий на теоретический const-qualified элемент, предшествующий первому элементу в map (последний в обратном порядке).

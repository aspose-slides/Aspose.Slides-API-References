## IDictionary::TryGetValue(const TKey\&, TValue\&) const метод

Ищет значение и возвращает его, если найдено.

```cpp
virtual bool System::Collections::Generic::IDictionary<TKey, TValue>::TryGetValue(const TKey &key, TValue &value) const =0
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| key | const TKey\& | Ключ для поиска. |
| value | TValue\& | Ссылка для сохранения полученного значения. |

### Возвращаемое значение

True, если значение найдено и получено, false иначе.

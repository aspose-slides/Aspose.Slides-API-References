## IDictionary::get_IsSynchronized() const метод


Проверяет, является ли контейнер потокобезопасным.

```cpp
bool System::Collections::Generic::IDictionary<TKey, TValue>::get_IsSynchronized() const
```


### Возвращаемое значение

true, если методы доступа к контейнеру являются потокобезопасными, false в противном случае.

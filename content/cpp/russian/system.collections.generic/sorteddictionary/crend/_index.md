## SortedDictionary::crend() const метод


Возвращает обратный итератор для несуществующего константного элемента перед началом коллекции.

```cpp
const_reverse_iterator System::Collections::Generic::SortedDictionary<TKey, TValue>::crend() const noexcept
```


### Возвращаемое значение

Обратный итератор, указывающий на теоретический константный элемент, предшествующий первому элементу в отображении (последний в обратном порядке).

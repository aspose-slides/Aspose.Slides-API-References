## SortedDictionary::rend() метод

Возвращает обратный итератор для несуществующего элемента перед началом коллекции.

```cpp
reverse_iterator System::Collections::Generic::SortedDictionary<TKey, TValue>::rend() noexcept
```

### Возвращаемое значение

Обратный итератор, указывающий на теоретический элемент, предшествующий первому элементу в map (последний в обратном порядке).

## SortedDictionary::rend() const метод

Возвращает обратный итератор для несуществующего элемента перед началом константно-квалифицированной коллекции.

```cpp
const_reverse_iterator System::Collections::Generic::SortedDictionary<TKey, TValue>::rend() const noexcept
```

### Возвращаемое значение

Обратный итератор, указывающий на теоретический элемент, предшествующий первому элементу в константно-квалифицированной map (последний в обратном порядке).

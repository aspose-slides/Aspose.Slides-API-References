## BaseDictionary::cbegin() const метод

Возвращает итератор на первый элемент контейнера. Реализовано в стиле STL. Если контейнер пуст, возвращаемый итератор будет равен [end()](../../ienumerable/end/).

```cpp
stl_const_iterator System::Collections::Generic::BaseDictionary<Map>::cbegin() const noexcept
```

### Возвращаемое значение

Итератор, указывающий на первый элемент коллекции, квалифицированный как const.

## BaseDictionary::cend() const метод


Возвращает итератор к элементу, следующему за последним элементом контейнера. Реализовано в стиле STL. Этот элемент служит заполнительным; попытка доступа к нему приводит к неопределённому поведению.

```cpp
stl_const_iterator System::Collections::Generic::BaseDictionary<Map>::cend() const noexcept
```


### Возвращаемое значение

Итератор, указывающий на теоретический const-квалифицированный элемент, расположенный после конечного элемента коллекции.

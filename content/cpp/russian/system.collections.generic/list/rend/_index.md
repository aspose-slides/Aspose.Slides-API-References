## List::rend() метод

Получает обратный итератор для несуществующего элемента перед началом коллекции.

```cpp
reverse_iterator System::Collections::Generic::List<T>::rend() noexcept
```

### Возвращаемое значение

Обратный итератор, указывающий на теоретический элемент, предшествующий первому элементу списка (последний в обратном порядке).

## List::rend() const метод

Получает обратный итератор для несуществующего элемента перед началом константной коллекции.

```cpp
const_reverse_iterator System::Collections::Generic::List<T>::rend() const noexcept
```

### Возвращаемое значение

Обратный итератор, указывающий на теоретический элемент, предшествующий первому элементу константного списка (последний в обратном порядке).

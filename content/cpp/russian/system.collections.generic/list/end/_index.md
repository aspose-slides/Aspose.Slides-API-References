## List::end() метод

Получает итератор для несуществующего элемента, находящегося за пределами конца коллекции.

```cpp
iterator System::Collections::Generic::List<T>::end() noexcept
```

### Возвращаемое значение

Итератор, указывающий на теоретический элемент, расположенный после конечного элемента в списке.

Получает итератор для несуществующего элемента, находящегося за пределами конца константно-квалифицированной коллекции.

```cpp
const_iterator System::Collections::Generic::List<T>::end() const noexcept
```

### Возвращаемое значение

Итератор, указывающий на теоретический элемент, расположенный после конечного элемента в константно-квалифицированном экземпляре списка.

## List::TrueForAll(System::Predicate\<T\>) метод

Определяет, соответствует ли каждый элемент в коллекции условиям, определённым указанным предикатом.

```cpp
bool System::Collections::Generic::List<T>::TrueForAll(System::Predicate<T> match)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| match | [System::Predicate](../../../system/predicate/)\<T\> | Предикат для проверки элементов. |

### Возвращаемое значение

True если каждый элемент соответствует условиям, false в противном случае.

## List::LastIndexOf(const T&) const метод

Ищет указанный объект и возвращает нуль-ориентированный индекс последнего вхождения во весь список.

```cpp
int32_t System::Collections::Generic::List<T>::LastIndexOf(const T &item) const
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| item | const T\& | Объект, который необходимо найти в списке |

### Возвращаемое значение

Нуль-ориентированный индекс последнего вхождения item во весь [List](../), если найдено; иначе, -1.

## List::LastIndexOf(const T&, int32_t) const метод

Ищет указанный объект и возвращает нуль-ориентированный индекс последнего вхождения в диапазоне элементов [List](../), который простирается от первого элемента до указанного индекса.

```cpp
int32_t System::Collections::Generic::List<T>::LastIndexOf(const T &item, int32_t index) const
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| item | const T\& | Объект, который необходимо найти в списке |
| index | **int32_t** | Нуль-ориентированный начальный индекс обратного поиска. |

### Возвращаемое значение

Нуль-ориентированный индекс последнего вхождения item в диапазоне элементов [List](../), который простирается от первого элемента до index, если найдено; иначе, -1.

## List::LastIndexOf(const T&, int32_t, int32_t) const метод

Ищет указанный объект и возвращает нуль-ориентированный индекс последнего вхождения в диапазоне элементов [List](../), который содержит указанное количество элементов и завершается указанным индексом.

```cpp
int32_t System::Collections::Generic::List<T>::LastIndexOf(const T &item, int32_t index, int32_t count) const
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| item | const T\& | Объект, который необходимо найти в [List](../) |
| index | **int32_t** | Нуль-ориентированный начальный индекс обратного поиска. |
| count | **int32_t** | Количество элементов в разделе для поиска. |

### Возвращаемое значение

Нуль-ориентированный индекс последнего вхождения item в диапазоне элементов [List](../), который содержит count элементов и завершается index, если найдено; иначе, -1.

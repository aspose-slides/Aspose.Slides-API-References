## List::crend() const метод

Получает обратный итератор для несуществующего const-qualified элемента перед началом коллекции.

```cpp
const_reverse_iterator System::Collections::Generic::List<T>::crend() const noexcept
```

### Возвращаемое значение

Обратный итератор, указывающий на теоретический const-qualified элемент, предшествующий первому элементу списка (последний в обратном порядке).

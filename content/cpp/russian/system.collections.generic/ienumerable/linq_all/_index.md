## IEnumerable::LINQ_All(std::function\<bool(T)>) метод


Определяет, удовлетворяют ли все элементы последовательности условию.

```cpp
bool System::Collections::Generic::IEnumerable<T>::LINQ_All(std::function<bool(T)> predicate)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| predicate | std::function\<**bool**(T)> | Функция, проверяющая каждый элемент на соответствие условию. |

### Возвращаемое значение

true, если каждый элемент исходной последовательности проходит проверку указанного предиката, или если последовательность пуста; в противном случае — false.

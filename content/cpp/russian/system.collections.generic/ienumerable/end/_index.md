## IEnumerable::end() метод

Получает итератор, указывающий сразу за последним элементом (если он есть) коллекции. Этот итератор нельзя использовать для изменения ссылки на объект, потому что [GetEnumerator()](../getenumerator/) возвращает объект-копию типа T.

```cpp
iterator System::Collections::Generic::IEnumerable<T>::end()
```

### Возвращаемое значение

Итератор, указывающий сразу за последним элементом (если он есть) коллекции

## IEnumerable::end() const метод

Получает итератор, указывающий сразу за последним элементом (если он есть) константно-квалифицированного экземпляра коллекции.

```cpp
const_iterator System::Collections::Generic::IEnumerable<T>::end() const
```

### Возвращаемое значение

Итератор, указывающий сразу за последним элементом (если он есть) константно-квалифицированного экземпляра коллекции

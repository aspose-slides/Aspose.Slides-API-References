## Структуры

Если не установлен, использует оператор ==, [Object::Equals](../system/object/equals/) или T::Equals, в зависимости от доступности. |
| [EqualityComparerHashAdapter](./equalitycomparerhashadapter/) | Адаптер для использования [IEqualityComparer](./iequalitycomparer/) при хешировании.

## HttpCompletionOption перечисление

Указывает, когда операция [HttpClient](../httpclient/) должна быть завершена.

```cpp
enum class HttpCompletionOption
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| ResponseContentRead | 0 | Операция должна быть завершена после чтения содержимого ответа. |
| ResponseHeadersRead | 1 | Операция должна быть завершена после чтения заголовков ответа. |

## X509NameType перечисление

Тип имени, содержащегося в сертификате X.509, которое относится либо к эмитенту, либо к субъекту сертификата.

```cpp
enum class X509NameType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| SimpleName | 0 | Простое имя субъекта или эмитента. |
| EmailName | 1 | Адрес электронной почты. |
| UpnName | 2 | UPN имя. |
| DnsName | 3 | DNS имя. |
| DnsFromAlternativeName | 4 | Альтернативное DNS имя, эквивалентное DNS имени. |
| UrlName | 5 | URL. |

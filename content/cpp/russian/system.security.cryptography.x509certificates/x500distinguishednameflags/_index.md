## X500DistinguishedNameFlags перечисление

Правила форматирования отличительных имён сертификатов X509.

```cpp
enum class X500DistinguishedNameFlags
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | 0 | Нет специальных характеристик. |
| Reversed | 1 | Имя зарезервировано. |
| UseSemicolons | 16 | Использовать точки с запятой. |
| DoNotUsePlusSign | 32 | Имя не использует знак плюса. |
| DoNotUseQuotes | 64 | Отключает кавычки в имени. |
| UseCommas | 128 | Включает использование запятых. |
| UseNewLines | 256 | Включает использование новых строк. |
| UseUTF8Encoding | 4096 | Переключается с использования Unicode на кодировку UTF-8. |
| UseT61Encoding | 8192 | Переключается на кодировку T61. |
| ForceUTF8Encoding | 16384 | Принуждает использовать UTF-8 при кодировании определённых ключей X500. |

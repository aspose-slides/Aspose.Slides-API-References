## X509IncludeOption enum

Указывает, какие сертификаты в цепочке включать.

```cpp
enum class X509IncludeOption
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | 0 | Только сведения о цепочке. |
| ExcludeRoot | 1 | Вся цепочка без корневого сертификата. |
| EndCertOnly | 2 | Вся цепочка без конечного сертификата. |
| WholeChain | 3 | Вся цепочка. |

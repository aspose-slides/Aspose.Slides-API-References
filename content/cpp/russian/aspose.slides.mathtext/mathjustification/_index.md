## MathJustification enum

Specifies justification of the math paragraph (a series of adjacent instances of mathematical text within the same paragraph)

```cpp
enum class MathJustification
```

### Values

| Имя | Значение | Описание |
| --- | --- | --- |
| LeftJustified | 1 | Выравнивает математический текст по левому краю |
| RightJustified | 2 | Выравнивает математический текст по правому краю |
| Centered | 3 | Выравнивает математический текст по центру |
| CenteredAsGroup | 4 | Размещает в центре страницы всю группу блоков математики как единое целое. Это означает, что отдельные экземпляры математического текста могут быть выровнены относительно друг друга, но вся группа математического текста центрируется в целом. |

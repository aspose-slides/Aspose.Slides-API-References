## IMathPhantom::get_Show() метод

Получает значение, указывающее, отображается ли базовый элемент.

```cpp
virtual bool Aspose::Slides::MathText::IMathPhantom::get_Show()=0
```

## Примечания

Когда **false**, базовый элемент скрыт, но может по-прежнему занимать пространство в зависимости от других настроек phantom. Соответствует атрибуту OMML **m:show**.

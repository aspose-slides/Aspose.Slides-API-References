## IMathPhantom::get_Transp() метод


Возвращает значение, указывающее, является ли фантом прозрачным для правил интервалов, основанных на классах.

```cpp
virtual bool Aspose::Slides::MathText::IMathPhantom::get_Transp()=0
```

## Примечания


Когда **true**, операторы и символы внутри фантома по-прежнему влияют на математические интервалы вокруг фантома (как если бы он был видимым). Когда **false**, интервалы, основанные на классах, игнорируются. Соответствует атрибуту OMML **m:transp**.

## IMathPhantom::get_ZeroAsc() метод

Возвращает значение, указывающее, следует ли считать подъем (высоту выше базовой линии) базового элемента равным нулю.

```cpp
virtual bool Aspose::Slides::MathText::IMathPhantom::get_ZeroAsc()=0
```

## Примечания

Когда **true**, фантом не повышает базовую линию окружающей математической строки. Соответствует атрибуту OMML **m:zeroAsc**.

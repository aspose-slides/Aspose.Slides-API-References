## IMathPhantom::set_ZeroDesc(bool) метод

Устанавливает значение, указывающее, следует ли рассматривать спуск (глубину ниже базовой линии) базового элемента как ноль.

```cpp
virtual void Aspose::Slides::MathText::IMathPhantom::set_ZeroDesc(bool value)=0
```

## Примечания

Когда **true**, фантом не опускает базовую линию окружающей математической строки. Соответствует атрибуту OMML **m:zeroDesc**.

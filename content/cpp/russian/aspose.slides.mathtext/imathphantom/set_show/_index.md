## IMathPhantom::set_Show(bool) метод


Устанавливает значение, указывающее, отображается ли базовый элемент.

```cpp
virtual void Aspose::Slides::MathText::IMathPhantom::set_Show(bool value)=0
```

## Примечания


Когда **false**, базовый элемент скрыт, но может по-прежнему занимать место в зависимости от других настроек phantom. Соответствует атрибуту OMML **m:show**.

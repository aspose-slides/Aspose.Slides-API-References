## IMathPhantom::set_ZeroAsc(bool) метод

Устанавливает значение, указывающее, должно ли восхождение (высота над линией базиса) базового элемента рассматриваться как ноль.

```cpp
virtual void Aspose::Slides::MathText::IMathPhantom::set_ZeroAsc(bool value)=0
```

## Примечания

Когда **true**, фантом не поднимает базовую линию окружающей математической строки. Соответствует атрибуту OMML **m:zeroAsc**.

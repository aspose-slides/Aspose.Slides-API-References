## IMathPhantom::get_ZeroWidth() метод

Возвращает значение, указывающее, следует ли рассматривать ширину базового элемента как нулевую.

```cpp
virtual bool Aspose::Slides::MathText::IMathPhantom::get_ZeroWidth()=0
```

## Примечания

Когда **true**, фантом не резервирует горизонтальное пространство для своего базового элемента. Соответствует атрибуту OMML **m:zeroWid**.

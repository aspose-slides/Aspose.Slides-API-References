## IMathPhantom::set_ZeroWidth(bool) метод

Устанавливает значение, указывающее, следует ли рассматривать ширину базового элемента как нулевую.

```cpp
virtual void Aspose::Slides::MathText::IMathPhantom::set_ZeroWidth(bool value)=0
```

## Примечания

Когда **true**, фантом не резервирует горизонтальное пространство для своей базы. Соответствует атрибуту OMML **m:zeroWid**.

## MathBorderBox::set_StrikethroughTopLeftToBottomRight(bool) метод

Зачёркивание слева-вверх до справа-вниз (по умолчанию false). Указывает скрытое или отображаемое состояние диагональной линии перечёркивания от левого верхнего угла до правого нижнего угла рамки.

```cpp
void Aspose::Slides::MathText::MathBorderBox::set_StrikethroughTopLeftToBottomRight(bool value) override
```

## Примечания

Пример: 
```cpp
auto borderBox = System::MakeObject<MathBorderBox>(System::MakeObject<MathematicalText>(u"x"));
borderBox->set_StrikethroughTopLeftToBottomRight(true);
```

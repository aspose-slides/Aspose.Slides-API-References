## MathSpacingRules enum

Типы зазоров (горизонтальное расстояние) между столбцами матрицы

```cpp
enum class MathSpacingRules
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| SingleSpacingGap | 0 | Одинарный (1 em для столбцов или 1 строка для строк) |
| OneAndHalfSpacingGap | 1 | Полуторный (1.5 em для столбцов или 1.5 строк для строк) |
| DoubleSpacingGap | 2 | Двойной интервал (2 em для столбцов или 2 строки для строк) |
| Exactly | 3 | Точное значение. Для столбцов используется значение ColumnGap, измеряется в twips. Для строк используется значение RowGap, измеряется в twips. |
| Multiple | 4 | Множественный. Для столбцов используется значение ColumnGap, измеряется шагами 0.5 em. Для строк используется значение RowGap, измеряется в строках. |

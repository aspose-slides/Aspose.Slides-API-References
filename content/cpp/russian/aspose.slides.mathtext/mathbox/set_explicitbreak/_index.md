## MathBox::set_ExplicitBreak(uint8_t) метод


Явный разрыв указывает, существует ли разрыв строки в начале объекта Box, так что строка переносится в начале объекта Box. Указывает номер оператора на предыдущей строке математического текста, который будет использоваться в качестве точки выравнивания для текущей строки математического текста. Возможные значения: 1..255 По умолчанию: 0 (нет явного разрыва)

```cpp
void Aspose::Slides::MathText::MathBox::set_ExplicitBreak(uint8_t value) override
```

## Примечания


Пример: 
```cpp
auto box = System::MakeObject<MathematicalText>(u"==")->ToBox();
box->set_ExplicitBreak(1);
```

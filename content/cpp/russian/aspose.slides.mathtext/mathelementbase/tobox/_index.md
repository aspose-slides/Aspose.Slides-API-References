## MathElementBase::ToBox() метод

Помещает этот элемент в невизуальный контейнер (логическое объединение), который используется для группировки компонентов уравнения или другого фрагмента математического текста. Такой контейнер может (например) служить эмулятором оператора с точкой выравнивания или без неё, действовать как точка разрыва строки или группироваться так, чтобы запрещать разрывы строки внутри.

```cpp
System::SharedPtr<IMathBox> Aspose::Slides::MathText::MathElementBase::ToBox() override
```

### Возвращаемое значение

Логический контейнер, в который помещён данный элемент

## Примечания

Пример:
```cpp
auto box = System::MakeObject<MathematicalText>(u"x:=y")->ToBox();
```

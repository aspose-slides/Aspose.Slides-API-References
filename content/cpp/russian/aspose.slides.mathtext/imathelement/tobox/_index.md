## IMathElement::ToBox() метод


Размещает этот элемент в невизуальном контейнере (логической группе), который используется для группировки компонентов уравнения или другого экземпляра математического текста. Упакованный объект может (например) выступать в роли эмулятора оператора с точкой выравнивания или без неё, служить точкой разрыва строки или быть сгруппирован так, чтобы не допускать разрывов строк внутри.

```cpp
virtual System::SharedPtr<IMathBox> Aspose::Slides::MathText::IMathElement::ToBox()=0
```


### Возвращаемое значение

Логический контейнер, в котором помещён этот элемент
## Примечания



Пример: 
```cpp
auto box = System::MakeObject<MathematicalText>(u"x:=y")->ToBox();
```

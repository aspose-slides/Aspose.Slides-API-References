## MathMatrix::get_MinColumnWidth() метод

Минимальная ширина столбца в твипсах (1/20 части пункта) Промежуток (также именуемый \\u201CColumn Gap\\u201D или \\u201CGap Width\\u201D) добавляется к MinColumnWidth, чтобы определить общий Matrix [Column](../../../aspose.slides/column/) Spacing (расстояние между одинаковыми краями разных столбцов). По умолчанию: 0.

```cpp
uint32_t Aspose::Slides::MathText::MathMatrix::get_MinColumnWidth() override
```

## Примечания

Пример:
```cpp
auto matrix = System::MakeObject<MathMatrix>(2, 3);
matrix->set_MinColumnWidth(20);
```

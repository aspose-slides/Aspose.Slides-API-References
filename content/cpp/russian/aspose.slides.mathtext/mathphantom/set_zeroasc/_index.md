## MathPhantom::set_ZeroAsc(bool) метод

Устанавливает значение, указывающее, следует ли считать подъем (высоту над базовой линией) базового элемента равным нулю.

```cpp
void Aspose::Slides::MathText::MathPhantom::set_ZeroAsc(bool value) override
```

## Примечания

Когда **true**, фантом не поднимает базовую линию окружающей математической строки. Соответствует атрибуту OMML **m:zeroAsc**.

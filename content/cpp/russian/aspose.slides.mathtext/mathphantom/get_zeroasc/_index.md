## MathPhantom::get_ZeroAsc() метод

Возвращает значение, указывающее, следует ли считать подъем (высоту над базовой линией) базового элемента равным нулю.

```cpp
bool Aspose::Slides::MathText::MathPhantom::get_ZeroAsc() override
```

## Примечания

Когда **true**, фантом не поднимает базовую линию окружающей математической строки. Соответствует атрибуту OMML **m:zeroAsc**.

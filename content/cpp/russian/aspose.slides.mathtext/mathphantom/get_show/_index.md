## MathPhantom::get_Show() метод

Возвращает значение, указывающее, отображается ли базовый элемент.

```cpp
bool Aspose::Slides::MathText::MathPhantom::get_Show() override
```

## Примечания

Когда **false**, базовый элемент скрыт, но может по-прежнему занимать пространство в зависимости от других настроек phantom. Соответствует атрибуту OMML **m:show**.

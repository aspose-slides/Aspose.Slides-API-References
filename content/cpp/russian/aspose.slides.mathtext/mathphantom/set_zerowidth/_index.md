## MathPhantom::set_ZeroWidth(bool) метод


Устанавливает значение, указывающее, следует ли рассматривать ширину базового элемента как нулевую.

```cpp
void Aspose::Slides::MathText::MathPhantom::set_ZeroWidth(bool value) override
```

## Примечания


Когда **true**, фантом не резервирует горизонтальное пространство для своей основы. Соответствует атрибуту OMML **m:zeroWid**.

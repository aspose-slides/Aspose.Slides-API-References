## MathPhantom::get_Transp() метод

Возвращает значение, указывающее, является ли фантом прозрачным для правил интервального отступа, основанных на классах.

```cpp
bool Aspose::Slides::MathText::MathPhantom::get_Transp() override
```

## Примечания

Когда **true**, операторы и символы внутри фантома всё ещё влияют на математические интервалы вокруг фантома (как будто он видим). Когда **false**, интервалы, основанные на классах, игнорируются. Соответствует атрибуту OMML **m:transp**.

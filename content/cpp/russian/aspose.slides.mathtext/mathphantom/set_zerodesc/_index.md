## MathPhantom::set_ZeroDesc(bool) метод


Устанавливает значение, указывающее, следует ли рассматривать спуск (глубина ниже базовой линии) базового элемента как ноль.

```cpp
void Aspose::Slides::MathText::MathPhantom::set_ZeroDesc(bool value) override
```

## Примечания


Когда **true**, фантом не опускает базовую линию окружающей математической строки. Соответствует атрибуту OMML **m:zeroDesc**.

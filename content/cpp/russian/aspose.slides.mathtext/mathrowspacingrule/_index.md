## MathRowSpacingRule enum

Тип вертикального интервала между столбцами в матрице или массиве

```cpp
enum class MathRowSpacingRule
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| SingleLineGap | 0 | одиночный промежуток между строками |
| OneAndAHalfLineGap | 1 | промежуток в полтора строки |
| TwoLineGap | 2 | промежуток в две строки |
| Exactly | 3 | Точно (использует значение свойства RowSpacing, измеряется в пунктах) |
| Multiple | 4 | Множественный (использует значение свойства RowSpacing, измеряется в строках) |

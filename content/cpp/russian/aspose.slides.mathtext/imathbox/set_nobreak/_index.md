## IMathBox::set_NoBreak(bool) метод

Без переноса. Это свойство определяет свойство \"неразрывный\" у объектного бокса. Когда значение true, внутри бокса не могут происходить разрывы строк. Это может быть важно для эмуляторов операторов, состоящих более чем из одного бинарного оператора. Если этот элемент не указан, внутри бокса могут происходить разрывы. По умолчанию: true

```cpp
virtual void Aspose::Slides::MathText::IMathBox::set_NoBreak(bool value)=0
```

## Примечание

Пример: 
```cpp
auto box = System::MakeObject<MathematicalText>(u"**********")->ToBox();
box->set_NoBreak(false);
```

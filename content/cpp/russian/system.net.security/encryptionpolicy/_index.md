## EncryptionPolicy перечисление


Перечисляет политики шифрования.

```cpp
enum class EncryptionPolicy
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| RequireEncryption | 0 | Требовать шифрование и никогда не позволять использовать шифр «Null». |
| AllowNoEncryption | 1 | Предпочитать полное шифрование, но разрешать использовать шифр «Null», если сервер согласен. |
| NoEncryption | 2 | Разрешить отсутствие шифрования и запросить использование шифра «Null», если другая сторона может его обработать. |

## XmlSerializer::CanDeserialize(System::SharedPtr\<XmlReader\>) method

Проверяет, находится ли указанный читатель в состоянии, пригодном для десериализации.


### Параметры

| Параметр | Тип | Описание |

| xmlReader | [System::SharedPtr](../../../system/sharedptr/)\<[XmlReader](../../../system.xml/xmlreader/)\> | Читатель для проверки. |

true если xmlReader может быть десериализован, false в противном случае.

## Details_SecurityException::Is(const System::TypeInfo\&) const method

```cpp
bool System::Security::Details_SecurityException::Is(const System::TypeInfo &target) const override
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| target | const [System::TypeInfo](../../../system/typeinfo/)\& | структура [TypeInfo](../../../system/typeinfo/), описывающая тип, с которым сравнивается текущий объект. |

### Возвращаемое значение

True, если объект является типом с тегом или его подклассом, false в противном случае.

## Примечания

Проверьте, представляет ли объект экземпляр типа, описанного targetType. Аналог оператора C# 'is'.

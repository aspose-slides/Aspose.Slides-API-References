## RectangleF::Contains(float, float) метод

Определяет, находится ли указанная точка внутри прямоугольника, представленного текущим объектом.

```cpp
bool System::Drawing::RectangleF::Contains(float x, float y)
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| x | **float** | Координата X проверяемой точки |
| y | **float** | Координата Y проверяемой точки |

### Возвращаемое значение

True, если указанная точка находится внутри прямоугольника, представленного текущим объектом, иначе — false

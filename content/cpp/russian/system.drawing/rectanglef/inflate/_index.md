## RectangleF::Inflate(float, float) метод


Увеличивает ширину и высоту прямоугольника, представленного текущим объектом, сохраняя расположение геометрического центра прямоугольника. Ширина и высота увеличиваются в обоих направлениях на указанные значения.

```cpp
void System::Drawing::RectangleF::Inflate(float width, float height)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| width | **float** | Величина, на которую следует увеличить ширину прямоугольника в обоих направлениях |
| height | **float** | Величина, на которую следует увеличить высоту прямоугольника в обоих направлениях |

## RectangleF::Inflate(const SizeF\&) метод


Увеличивает ширину и высоту прямоугольника, представленного текущим объектом, сохраняя расположение геометрического центра прямоугольника. Ширина и высота увеличиваются в обоих направлениях на значения ширины и высоты, указанные соответствующими полями заданного объекта размера.

```cpp
void System::Drawing::RectangleF::Inflate(const SizeF &size)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| size | const [SizeF](../../sizef/)\& | [SizeF](../../sizef/) объект, определяющий величины, на которые следует увеличить ширину и высоту прямоугольника |

## RectangleF::Inflate(const RectangleF\&, float, float) метод


Увеличивает ширину и высоту прямоугольника, представленного указанным объектом, сохраняя расположение геометрического центра прямоугольника. Ширина и высота увеличиваются в обоих направлениях на указанные значения.

```cpp
static RectangleF System::Drawing::RectangleF::Inflate(const RectangleF &rect, float x, float y)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| rect | const [RectangleF](../)\& | Прямоугольник, подлежащий увеличению |
| x | **float** | Величина, на которую следует увеличить ширину прямоугольника в обоих направлениях |
| y | **float** | Величина, на которую следует увеличить высоту прямоугольника в обоих направлениях |

### Возвращаемое значение

Объект [RectangleF](../), представляющий увеличенный прямоугольник

## RectangleF::Offset(const PointF\&) метод

Смещает позицию прямоугольника, представленного текущим объектом, на указанные величины.

```cpp
void System::Drawing::RectangleF::Offset(const PointF &pos)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| pos | const [PointF](../../pointf/)\& | Объект [PointF](../../pointf/), который указывает величину, на которую смещается прямоугольник; значение координаты X точки указывает размер, который следует добавить к координате X верхнего левого угла прямоугольника, а значение координаты Y точки указывает размер, который следует добавить к координате Y верхнего левого угла прямоугольника |

Смещает позицию прямоугольника, представленного текущим объектом, на указанные величины.

void System::Drawing::RectangleF::Offset(float x, float y)

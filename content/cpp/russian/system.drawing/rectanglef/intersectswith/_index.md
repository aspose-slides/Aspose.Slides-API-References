## RectangleF::IntersectsWith(const RectangleF\&) метод

Определяет, пересекаются ли прямоугольники, представленные текущим объектом и указанным объектом.

```cpp
bool System::Drawing::RectangleF::IntersectsWith(const RectangleF &rect)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| rect | const [RectangleF](../)\& | Прямоугольник для проверки |

### Возвращаемое значение

True, если прямоугольники, представленные текущим объектом и **rect**, пересекаются, иначе — false

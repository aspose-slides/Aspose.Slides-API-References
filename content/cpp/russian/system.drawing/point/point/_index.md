## Point::Point() конструктор

Создает новый объект [Point](../) и инициализирует его координаты X и Y значениями 0.

```cpp
System::Drawing::Point::Point()
```

## Point::Point(int, int) конструктор

Создает новый объект [Point](../) и инициализирует его указанными значениями.

```cpp
System::Drawing::Point::Point(int x, int y)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| x | int | Значение координаты X |
| y | int | Значение координаты Y |

## Point::Point(const Size\&) конструктор

Создает новый объект [Point](../) и инициализирует его координаты X и Y значениями ширины и высоты соответствующего объекта [SizeF](../../sizef/).

```cpp
System::Drawing::Point::Point(const Size &size)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| size | const [Size](../../size/)\& | Объект [SizeF](../../sizef/), у которого значения ширины и высоты используются для инициализации координат X и Y объекта [Point](../), создаваемого |

## Point::Point(int) конструктор

Создает новый объект [Point](../) и инициализирует значение координаты X значением, сформированным из старших 16 бит указанного 32-битного целого числа, а значение координаты Y — значением, сформированным из младших 16 бит указанного 32-битного целого числа.

```cpp
System::Drawing::Point::Point(int dw)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| dw | int | 32-битное целое значение, старшие 16 бит которого указывают значение координаты X, а младшие 16 бит — значение координаты Y создаваемого объекта |

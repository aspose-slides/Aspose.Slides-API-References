## Point::Offset(int, int) метод

Смещает значение координат X и Y, представленных текущим объектом, на указанные значения.

```cpp
void System::Drawing::Point::Offset(int dx, int dy)
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| dx | int | Значение, на которое смещается координата X |
| dy | int | Значение, на которое смещается координата Y |

## Point::Offset(Point) метод

Смещает координаты X и Y, представленные текущим объектом, на значения координат X и Y, представленные указанным объектом [Point](../) соответственно.

```cpp
void System::Drawing::Point::Offset(Point point)
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| point | [Point](../) | Объект [Point](../), указывающий значения, на которые смещаются координаты X и Y |

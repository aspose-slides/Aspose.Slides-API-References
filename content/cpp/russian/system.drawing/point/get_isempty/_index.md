## Point::get_IsEmpty() const метод

Определяет, являются ли значения обеих координат X и Y равными 0.

```cpp
bool System::Drawing::Point::get_IsEmpty() const
```

### Возвращаемое значение

True если обе координаты X и Y, представленные текущим объектом, равны 0; иначе - false

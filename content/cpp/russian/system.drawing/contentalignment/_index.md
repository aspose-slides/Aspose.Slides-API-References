## ContentAlignment enum

Указывает выравнивание содержимого.

```cpp
enum class ContentAlignment
```

### Values

| Имя | Значение | Описание |
| --- | --- | --- |
| TopLeft | 1 | Выравнивается по вертикали вверху и по горизонтали слева. |
| TopCenter | 2 | Выравнивается по вертикали вверху и по горизонтали по центру. |
| TopRight | 4 | Выравнивается по вертикали вверху и по горизонтали справа. |
| MiddleLeft | 16 | Выравнивается по вертикали по центру и по горизонтали слева. |
| MiddleCenter | 32 | Выравнивается по вертикали по центру и по горизонтали по центру. |
| MiddleRight | 64 | Выравнивается по вертикали по центру и по горизонтали справа. |
| BottomLeft | 256 | Выравнивается по вертикали внизу и по горизонтали слева. |
| BottomCenter | 512 | Выравнивается по вертикали внизу и по горизонтали по центру. |
| BottomRight | 1024 | Выравнивается по вертикали внизу и по горизонтали справа. |

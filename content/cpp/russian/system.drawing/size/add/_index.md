## Size::Add(const Size\&, const Size\&) method

Возвращает новый объект [Size](../), который является суммой указанных объектов [Size](../), т.е. значение ширины которого равно сумме значений ширины указанных объектов, а значение высоты равно сумме значений высоты указанных объектов.

```cpp
static Size System::Drawing::Size::Add(const Size &size1, const Size &size2)
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| size1 | const [Size](../)\& | Первый операнд |
| size2 | const [Size](../)\& | Второй операнд |

### Возвращаемое значение

Объект [Size](../), который является суммой **size1** и **size2**

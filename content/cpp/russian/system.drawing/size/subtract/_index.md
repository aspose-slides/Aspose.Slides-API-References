## Size::Subtract(const Size\&, const Size\&) метод


Возвращает новый объект [Size](../), который является результатом вычитания **size2** из **size1**, то есть значение ширины равно результату вычитания ширины **size2's** из ширины **size1's**, а значение высоты равно результату вычитания высоты **size2's** из высоты **size1's**.

```cpp
static Size System::Drawing::Size::Subtract(const Size &size1, const Size &size2)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| size1 | const [Size](../)\& | Объект [Size](../), из которого происходит вычитание |
| size2 | const [Size](../)\& | Объект [Size](../), который вычитается |

### Возвращаемое значение

Объект [Size](../), который является результатом вычитания **size2** из **size1**

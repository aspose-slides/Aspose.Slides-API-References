## Size::Equals(const Size\&) const метод


Определяет, являются ли текущий объект и указанный объект равными, т.е. представляют одну и ту же пару значений ширины и высоты.

```cpp
bool System::Drawing::Size::Equals(const Size &size) const
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| size | const [Size](../)\& | Объект, с которым сравнивается текущий объект |

### Возвращаемое значение

True, если объекты равны, иначе - false

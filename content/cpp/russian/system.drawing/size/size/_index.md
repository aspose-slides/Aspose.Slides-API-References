## Size::Size(const Point\&) конструктор

Создаёт новый объект [Size](../) и инициализирует его значения ширины и высоты соответственно значениями координат X и Y указанной точки.

```cpp
System::Drawing::Size::Size(const Point &point)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| point | const [Point](../../point/)\& | Объект [Point](../../point/), чьи координаты X и Y используются для инициализации значений ширины и высоты создаваемого объекта [Size](../) |

## Size::Size(int, int) конструктор

Создаёт новый объект [Size](../) и инициализирует его указанным значением.

```cpp
System::Drawing::Size::Size(int width, int height)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| width | int | Значение ширины |
| height | int | Значение высоты |

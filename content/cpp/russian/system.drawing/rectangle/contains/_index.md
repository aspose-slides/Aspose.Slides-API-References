## Rectangle::Contains(int, int) const метод


Определяет, находится ли указанная точка внутри прямоугольника, представленного текущим объектом.

```cpp
bool System::Drawing::Rectangle::Contains(int x, int y) const
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| x | int | Координата X проверяемой точки |
| y | int | Координата Y проверяемой точки |

### Возвращаемое значение

True, если указанная точка находится внутри прямоугольника, представленного текущим объектом, иначе — false

## Rectangle::Equals(const Rectangle\&) const метод

Определяет, являются ли прямоугольники, представленные текущим объектом и указанным объектом, идентичными.

```cpp
bool System::Drawing::Rectangle::Equals(const Rectangle &rect) const
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| rect | const [Rectangle](../)\& | Объект [Rectangle](../), с которым сравнивается текущий объект |

### Возвращаемое значение

True, если прямоугольники, представленные текущим и указанным объектами, идентичны, иначе - false

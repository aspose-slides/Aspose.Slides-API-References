## Rectangle::IntersectsWith(const Rectangle\&) метод

Определяет, пересекаются ли прямоугольники, представленные текущим и указанным объектами.

```cpp
bool System::Drawing::Rectangle::IntersectsWith(const Rectangle &rect)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| rect | const [Rectangle](../)\& | Прямоугольник для проверки |

### Возвращаемое значение

True если прямоугольники, представленные текущим объектом и **rect**, пересекаются, иначе - false

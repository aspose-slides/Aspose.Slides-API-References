## Rectangle::Offset(const Point\&) метод

Смещает позицию прямоугольника, представленного текущим объектом, на заданные величины.

```cpp
void System::Drawing::Rectangle::Offset(const Point &pos)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| pos | const [Point](../../point/)\& | Объект [Point](../../point/), указывающий величину смещения прямоугольника; значение X координаты точки задаёт величину, добавляемую к X координате верхнего левого угла прямоугольника, а значение Y координаты точки задаёт величину, добавляемую к Y координате верхнего левого угла прямоугольника |

## Rectangle::Offset(int, int) метод

Смещает позицию прямоугольника, представленного текущим объектом, на заданные величины.

```cpp
void System::Drawing::Rectangle::Offset(int x, int y)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| x | int | Значение, добавляемое к текущей X координате прямоугольника |
| y | int | Значение, добавляемое к текущей Y координате прямоугольника |

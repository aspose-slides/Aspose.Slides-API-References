## Rectangle::Inflate(int, int) метод


Увеличивает ширину и высоту прямоугольника, представленного текущим объектом, сохраняет позицию геометрического центра прямоугольника. Ширина и высота увеличиваются в обоих направлениях на указанные величины.

```cpp
void System::Drawing::Rectangle::Inflate(int width, int height)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| width | int | Величина, на которую ширина прямоугольника будет увеличена в обоих направлениях |
| height | int | Величина, на которую высота прямоугольника будет увеличена в обоих направлениях |

## Rectangle::Inflate(const Size\&) метод


Увеличивает ширину и высоту прямоугольника, представленного текущим объектом, сохраняет позицию геометрического центра прямоугольника. Ширина и высота увеличиваются в обоих направлениях на величины, указанные значениями width и height указанного объекта size соответственно.

```cpp
void System::Drawing::Rectangle::Inflate(const Size &size)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| size | const [Size](../../size/)\& | Объект [Size](../../size/), определяющий величины, на которые следует увеличить ширину и высоту прямоугольника |

## Rectangle::Inflate(const Rectangle\&, int, int) метод


Увеличивает ширину и высоту прямоугольника, представленного указанным объектом, сохраняет позицию геометрического центра прямоугольника. Ширина и высота увеличиваются в обоих направлениях на указанные величины.

```cpp
static Rectangle System::Drawing::Rectangle::Inflate(const Rectangle &rect, int x, int y)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| rect | const [Rectangle](../)\& | Прямоугольник для увеличения |
| x | int | Величина, на которую ширина прямоугольника будет увеличена в обоих направлениях |
| y | int | Величина, на которую высота прямоугольника будет увеличена в обоих направлениях |

### Возвращаемое значение

Объект [Rectangle](../), представляющий увеличенный прямоугольник

## CopyPixelOperation enum

Specifies how the source color in a pixel copying operation is combined with the destination color to result in a final color.

```cpp
enum class CopyPixelOperation
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| NoMirrorBitmap | n/a | Битмап не зеркалируется. |
| Blackness | 66 | Область назначения заполняется цветом с индексом 0 в физической палитре. |
| NotSourceErase | 1114278 | Цвета источника и назначения объединяются оператором OR, после чего полученный цвет инвертируется. |
| NotSourceCopy | 3342344 | Область источника инвертируется, а затем копируется в назначение. |
| SourceErase | 4457256 | Инвертированные цвета области назначения комбинируются оператором AND с цветами области источника. |
| DestinationInvert | 5570569 | Область назначения инвертируется. |
| PatInvert | 5898313 | Цвета кисти, текущо выбранной в контексте устройства назначения, комбинируются оператором XOR с цветами назначения. |
| SourceInvert | 6684742 | Цвета областей источника и назначения комбинируются оператором XOR. |
| SourceAnd | 8913094 | Цвета областей источника и назначения комбинируются оператором AND. |
| MergePaint | 12255782 | Цвета инвертированной области источника объединяются оператором OR с цветами области назначения. |
| MergeCopy | 12583114 | Цвета области источника комбинируются оператором AND с цветами выбранной кисти контекста устройства назначения. |
| SourceCopy | 13369376 | Область источника копируется напрямую в область назначения. |
| SourcePaint | 15597702 | Цвета областей источника и назначения объединяются оператором OR. |
| PatCopy | 15728673 | Кисть, текущо выбранная в контексте устройства назначения, копируется в битмап назначения. |
| PatPaint | 16452105 | Цвета кисти, текущо выбранной в контексте устройства назначения, объединяются оператором OR с цветами инвертированной области источника. Результат этой операции объединяется оператором OR с цветами области назначения. |
| Whiteness | 16711778 | Область назначения заполняется цветом с индексом 1 в физической палитре. |
| CaptureBlt | 1073741824 | [Windows](../../system.windows/), находящиеся поверх окна приложения, включаются в результирующее изображение. |

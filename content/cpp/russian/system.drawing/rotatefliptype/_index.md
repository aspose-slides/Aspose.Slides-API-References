## Перечисление RotateFlipType

Указывает тип операции вращения и/или отражения.

```cpp
enum class RotateFlipType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| RotateNoneFlipNone | 0 | Нет вращения и нет отражения. |
| Rotate90FlipNone | 1 | Вращение на 90 градусов по часовой стрелке без отражения. |
| Rotate180FlipNone | 2 | Вращение на 180 градусов по часовой стрелке без отражения. |
| Rotate270FlipNone | 3 | Вращение на 270 градусов по часовой стрелке без отражения. |
| RotateNoneFlipX | 4 | Горизонтальное отражение без вращения. |
| Rotate90FlipX | 5 | Вращение на 90 градусов по часовой стрелке и горизонтальное отражение. |
| Rotate180FlipX | 6 | Вращение на 180 градусов по часовой стрелке и горизонтальное отражение. |
| Rotate270FlipX | 7 | Вращение на 270 градусов по часовой стрелке и горизонтальное отражение. |
| RotateNoneFlipY | n/a | Вертикальное отражение без вращения. |
| Rotate90FlipY | n/a | Вращение на 90 градусов по часовой стрелке и вертикальное отражение. |
| Rotate180FlipY | n/a | Вращение на 180 градусов по часовой стрелке и вертикальное отражение. |
| Rotate270FlipY | n/a | Вращение на 270 градусов по часовой стрелке и вертикальное отражение. |
| RotateNoneFlipXY | n/a | Горизонтальное и вертикальное отражение без вращения. |
| Rotate90FlipXY | n/a | Вращение на 90 градусов по часовой стрелке и горизонтальное и вертикальное отражение. |
| Rotate180FlipXY | n/a | Вращение на 180 градусов по часовой стрелке и горизонтальное и вертикальное отражение. |
| Rotate270FlipXY | n/a | Вращение на 270 градусов по часовой стрелке и горизонтальное и вертикальное отражение. |

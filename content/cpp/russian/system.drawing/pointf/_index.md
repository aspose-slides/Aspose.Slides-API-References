## PointF класс

Представляет пару координат X и Y точки на двумерной плоскости, выраженных числами с одинарной точностью с плавающей запятой. Этот тип следует выделять в стеке и передавать функциям по значению или по ссылке. Никогда не используйте класс [System::SmartPtr](../../system/smartptr/) для управления объектами этого типа.

```cpp
class PointF
```

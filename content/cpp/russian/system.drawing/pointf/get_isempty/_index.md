## PointF::get_IsEmpty() const метод

Определяет, равны ли значения обеих координат X и Y нулю.

```cpp
bool System::Drawing::PointF::get_IsEmpty() const
```

### Возвращаемое значение

True если обе координаты X и Y, представленные текущим объектом, равны 0; иначе — false

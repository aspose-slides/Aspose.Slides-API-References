## PointF::Subtract(const PointF\&, const SizeF\&) метод


Вычитает значения ширины и высоты указанного объекта [SizeF](../../sizef/) из значений координат X и Y указанного объекта [PointF](../) соответственно.

```cpp
static PointF System::Drawing::PointF::Subtract(const PointF &point, const SizeF &size)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| point | const [PointF](../)\& | Точка для преобразования |
| size | const [SizeF](../../sizef/)\& | Объект [SizeF](../../sizef/), задающий значения, которые нужно вычесть из координат **point** |

### Возвращаемое значение

Новый объект [PointF](../), у которого значение координаты X равно результату вычитания значения ширины **size** из значения координаты X **point**, а значение координаты Y равно результату вычитания значения высоты **size** из значения координаты Y **point**.

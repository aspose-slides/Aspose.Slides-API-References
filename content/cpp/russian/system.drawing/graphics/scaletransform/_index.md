## Graphics::ScaleTransform(float, float, Drawing2D::MatrixOrder) метод


Применяет указанный вектор масштабирования к матрице мирового преобразования текущего объекта.

```cpp
void System::Drawing::Graphics::ScaleTransform(float sx, float sy, Drawing2D::MatrixOrder order=Drawing2D::MatrixOrder::Prepend)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| sx | **float** | Значение, на которое масштабируется матрица по оси x |
| sy | **float** | Значение, на которое масштабируется матрица по оси y |
| order | [Drawing2D::MatrixOrder](../../../system.drawing.drawing2d/matrixorder/) | Порядок, в котором вектор масштабирования применяется к матрице |

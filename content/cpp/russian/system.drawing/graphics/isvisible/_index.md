## Graphics::IsVisible(Point) метод

Определяет, находится ли указанная точка внутри видимой области клипа текущего объекта [Graphics](../).

```cpp
bool System::Drawing::Graphics::IsVisible(Point point)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| point | [Point](../../point/) | Точка для проверки |

### Возвращаемое значение

True, если **point** находится внутри видимой области клипа текущего объекта [Graphics](../), иначе - false

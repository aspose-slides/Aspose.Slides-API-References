## Pen::ScaleTransform(float, float, Drawing2D::MatrixOrder) метод

Масштабирует локальное геометрическое преобразование с помощью указанных коэффициентов в заданном порядке.

```cpp
void System::Drawing::Pen::ScaleTransform(float sx, float sy, Drawing2D::MatrixOrder order=Drawing2D::MatrixOrder::Prepend)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| sx | **float** | Коэффициент масштабирования, по которому выполняется масштабирование в направлении оси X |
| sy | **float** | Коэффициент масштабирования, по которому выполняется масштабирование в направлении оси Y |
| order | [Drawing2D::MatrixOrder](../../../system.drawing.drawing2d/matrixorder/) | Указывает порядок выполнения операции |

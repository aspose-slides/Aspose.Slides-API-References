## StringDigitSubstitute enum

Указывает, как цифры в строке заменяются в соответствии с локалью или языком.

```cpp
enum class StringDigitSubstitute
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| User | 0 | Подстановка, определяемая пользователем. |
| None | 1 | Подстановка отключена. |
| National | 2 | Подменять цифры, соответствующие официальному национальному языку локали пользователя. |
| Traditional | 3 | Подменять цифры, соответствующие родному письму или языку пользователя, который может отличаться от официального национального языка локали пользователя. |

## SystemColors класс

Класс, предоставляющий набор заранее созданных [Color](../color/) объектов, представляющих цвета элементов отображения [Windows](../../system.windows/). Это статический тип без экземплярных служб. Вы никогда не должны создавать его экземпляры каким-либо способом.

```cpp
class SystemColors
```

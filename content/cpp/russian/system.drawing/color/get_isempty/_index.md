## Color::get_IsEmpty() const метод

Возвращает значение, указывающее, является ли текущий объект \"empty\", то есть не представляет никакого цвета.

```cpp
bool System::Drawing::Color::get_IsEmpty() const
```

### Возвращаемое значение

true, если текущий объект не представляет никакого цвета, иначе — false

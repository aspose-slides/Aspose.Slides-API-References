## Color::FromArgb(int) метод

Создает экземпляр класса [Color](../) , представляющего указанный цвет.

```cpp
static Color System::Drawing::Color::FromArgb(int argb)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| argb | int | 32-битное значение ARGB цвета, которое будет представлять создаваемый объект |

### Возвращаемое значение

Объект, представляющий указанный цвет.

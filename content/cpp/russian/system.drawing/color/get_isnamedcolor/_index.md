## Color::get_IsNamedColor() const method

Возвращает значение, определяющее, представляет ли структура [Color](../) именованный цвет или член перечисления KnownColor.

```cpp
bool System::Drawing::Color::get_IsNamedColor() const
```

### Возвращаемое значение

True если [Color](../) был создан с использованием метода FromName(String) или метода [FromKnownColor(KnownColor)](../fromknowncolor/); в противном случае возвращает false.

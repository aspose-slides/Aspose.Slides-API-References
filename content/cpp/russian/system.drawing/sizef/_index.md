## SizeF класс

Представляет пару значений одинарной точности с плавающей запятой, которые представляют ширину и высоту изображения. Этот тип следует выделять в стеке и передавать функциям по значению или по ссылке. Никогда не используйте [System::SmartPtr](../../system/smartptr/) класс для управления объектами этого типа.

```cpp
class SizeF
```

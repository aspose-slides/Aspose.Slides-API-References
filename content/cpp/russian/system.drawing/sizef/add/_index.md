## SizeF::Add(const SizeF\&, const SizeF\&) метод

Возвращает новый [SizeF](../) объект, который является суммой указанных объектов [SizeF](../), то есть значение ширины равно сумме значений ширины указанных объектов, а значение высоты равно сумме значений высоты указанных объектов.

```cpp
static SizeF System::Drawing::SizeF::Add(const SizeF &size1, const SizeF &size2)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| size1 | const [SizeF](../)\& | Первый операнд |
| size2 | const [SizeF](../)\& | Второй операнд |

### Возвращаемое значение

[SizeF](../) объект, который является суммой **size1** и **size2**

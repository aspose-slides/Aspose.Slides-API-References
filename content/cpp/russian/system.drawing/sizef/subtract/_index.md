## SizeF::Subtract(const SizeF\&, const SizeF\&) метод

Возвращает новый объект [SizeF](../), который является результатом вычитания **size2** из **size1**, т.е. значение ширины равно разности ширины **size2's** и ширины **size1's**, а значение высоты равно разности высоты **size2's** и высоты **size1's**.

```cpp
static SizeF System::Drawing::SizeF::Subtract(const SizeF &size1, const SizeF &size2)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| size1 | const [SizeF](../)\& | Объект [SizeF](../), из которого производится вычитание |
| size2 | const [SizeF](../)\& | Объект [SizeF](../), который вычитается |

### Возвращаемое значение

Объект [SizeF](../), который является результатом вычитания **size2** из **size1**.

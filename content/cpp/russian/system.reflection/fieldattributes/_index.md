## FieldAttributes перечисление

Отражённые атрибуты поля.

```cpp
enum class FieldAttributes
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| FieldAccessMask | 7 | Маска доступа члена. Используйте эту маску для получения информации о доступности. |
| PrivateScope | 0 | Члены, которые нельзя ссылаться. |
| Private | 1 | Приватные члены. |
| FamANDAssem | 2 | Приватные члены, ограниченные сборкой. |
| Assembly | 3 | Члены, ограниченные сборкой. |
| Family | 4 | Члены, доступные типу и его подтипам. |
| FamORAssem | 5 | Члены, доступные типу, подтипам и сборке. |
| Public | 6 | Члены, доступные всем. |
| Static | 16 | Статические члены в отличие от экземплярных. |
| InitOnly | 32 | Константные члены, которые могут быть только инициализированы, но не изменены. |
| Literal | 64 | Члены, являющиеся константами во время компиляции. |
| NotSerialized | 128 | Несериализуемые члены. |
| SpecialName | 512 | Специальное поле одного из нижеуказанных имён. |
| PinvokeImpl | 8192 | Реализация, перенаправляемая через межоперационную связку. |
| ReservedMask | 38144 | Зарезервированные флаги только для использования в режиме выполнения. |
| RTSpecialName | 1024 | Среде выполнения следует проверять кодировку имени. |
| HasFieldMarshal | 4096 | Присутствует информация о маршаллировании. |
| HasDefault | 32768 | Присутствует значение по умолчанию. |
| HasFieldRVA | 256 | Присутствует RVA. |

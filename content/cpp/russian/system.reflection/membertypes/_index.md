## MemberTypes перечисление


Отмечает каждый тип члена.

```cpp
enum class MemberTypes
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Constructor | 1 | Указывает, что элемент является конструктором. |
| Event | 2 | Указывает, что элемент является событием. |
| Field | 4 | Указывает, что элемент является полем. |
| Method | 8 | Указывает, что элемент является методом. |
| Property | 16 | Указывает, что элемент является свойством. |
| TypeInfo | 32 | Указывает, что элемент является типом. |
| Custom | 64 | Указывает, что элемент является пользовательским типом члена. |
| NestedType | 128 | Указывает, что элемент является вложенным типом. |
| All | 191 | Указывает все типы членов. |

## ReflectionTypeLoadException typedef

ReflectionTypeLoadException выбрасывается методом Module.GetTypes, если любые классы в модуле не удаётся загрузить. Никогда не оборачивайте экземпляры класса ReflectionTypeLoadException в [System::SmartPtr](../../system/smartptr/).

```cpp
using System::Reflection::ReflectionTypeLoadException = typedef System::ExceptionWrapper<Details_ReflectionTypeLoadException>
```

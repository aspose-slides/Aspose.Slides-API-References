## TargetInvocationException определение типа


TargetInvocationException генерируется методами, вызываемыми через рефлексию. Никогда не оборачивайте экземпляры класса TargetInvocationException в [System::SmartPtr](../../system/smartptr/).

```cpp
using System::Reflection::TargetInvocationException = typedef System::ExceptionWrapper<Details_TargetInvocationException>
```

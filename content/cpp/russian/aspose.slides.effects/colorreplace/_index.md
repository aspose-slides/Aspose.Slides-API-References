## Класс ColorReplace


Представляет эффект замены цвета. Все цвета эффекта заменяются на фиксированный цвет. Значения альфа остаются без изменений.

```cpp
class ColorReplace : public Aspose::Slides::Effects::ImageTransformOperation,
                     public Aspose::Slides::Effects::IColorReplace,
                     public Aspose::Slides::Effects::IVisualEffect
```

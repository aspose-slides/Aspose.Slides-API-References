## AlphaCeiling класс

Представляет эффект Alpha Ceiling. Значения альфа (непрозрачности), большие нуля, изменяются на 100 %. Другими словами, всё частично непрозрачное становится полностью непрозрачным.

```cpp
class AlphaCeiling : public Aspose::Slides::Effects::ImageTransformOperation,
                     public Aspose::Slides::Effects::IAlphaCeiling,
                     public Aspose::Slides::Effects::IVisualEffect
```

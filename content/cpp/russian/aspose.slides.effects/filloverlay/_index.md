## FillOverlay класс

Представляет эффект Fill Overlay. Fill overlay может использоваться для указания дополнительной заливки объекта и смешивания двух заливок вместе.

```cpp
class FillOverlay : public Aspose::Slides::Effects::ImageTransformOperation,
                    public Aspose::Slides::Effects::IFillOverlay,
                    public Aspose::Slides::Effects::IVisualEffect
```

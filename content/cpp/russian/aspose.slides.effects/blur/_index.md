## Blur класс

Представляет эффект [Blur](./), который применяется ко всей фигуре, включая её заливку. Все цветовые каналы, включая альфа-канал, затрагиваются.

```cpp
class Blur : public Aspose::Slides::Effects::ImageTransformOperation,
             public Aspose::Slides::Effects::IBlur,
             public Aspose::Slides::Effects::IVisualEffect
```

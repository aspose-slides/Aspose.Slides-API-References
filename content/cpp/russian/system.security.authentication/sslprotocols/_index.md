## SslProtocols перечисление

Перечисляет криптографические протоколы.

```cpp
enum class SslProtocols
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | 0 | Операционная система выбирает протокол для использования. |
| Ssl2 | 12 | Протокол SSL 2.0. |
| Ssl3 | 48 | Протокол SSL 3.0. |
| Tls | 192 | Протокол TLS 1.0. |
| Default | 240 | Разрешает использовать только протоколы SSL 3.0 или TLS 1.0. |
| Tls11 | 768 | Протокол TLS 1.1. |
| Tls12 | 3072 | Протокол TLS 1.2. |

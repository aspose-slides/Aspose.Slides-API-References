## HttpRequestCacheLevel enum

The enum describes cache settings for HTTP.

```cpp
enum class HttpRequestCacheLevel
```

### Values

| Имя | Значение | Описание |
| --- | --- | --- |
| Default | 0 | Удовлетворяет запрос к ресурсу, используя либо кэшированную копию ресурса, либо отправляя запрос к ресурсу на сервер. |
| BypassCache | 1 | Удовлетворяет запрос, используя сервер. |
| CacheOnly | 2 | Всегда использует кэш клиента для получения ресурса. |
| CacheIfAvailable | 3 | Удовлетворяет запрос к ресурсу из кэша, если ресурс доступен; иначе отправляет запрос на сервер. |
| Revalidate | 4 | Использует локальную копию ресурса, если метка времени клиента совпадает с меткой времени ресурса на сервере. В противном случае ресурс загружается с сервера. |
| Reload | 5 | Ресурс всегда загружается с сервера. |
| NoCacheNoStore | 6 | Никогда не удовлетворяет запрос, используя ресурсы из кэша, и не кэширует ресурсы. |
| CacheOrNextCacheOnly | 7 | Удовлетворяет запрос к ресурсу либо из кэша локального компьютера, либо из удалённого кэша в LAN. |
| Refresh | 8 | Удовлетворяет запрос, используя сервер или кэш, отличный от локального кэша. |

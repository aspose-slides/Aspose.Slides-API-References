```cpp
bool System::Web::Details_HttpException::Is(const System::TypeInfo &target) const override
```


### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| target | const [System::TypeInfo](../../../system/typeinfo/)\& | [TypeInfo](../../../system/typeinfo/) структура, описывающая тип, с которым проверяется текущий объект. |

### Возвращаемое значение

True, если объект имеет указанный тип или его подкласс, иначе false.
## Примечания


Проверьте, представляет ли объект экземпляр типа, описанного параметром targetType. Аналог оператора C# 'is'.

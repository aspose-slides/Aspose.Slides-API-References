## RSACryptoServiceProvider::get_UseMachineKeyStore() метод


Проверяет, сохраняется ли ключ в хранилище машины вместо хранилища пользователя.

```cpp
static bool System::Security::Cryptography::RSACryptoServiceProvider::get_UseMachineKeyStore()
```


### Возвращаемое значение

True, если ключ должен сохраняться в хранилище машины, false, если он должен сохраняться в хранилище пользователя.

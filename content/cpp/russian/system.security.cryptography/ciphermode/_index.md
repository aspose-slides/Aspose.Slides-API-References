## CipherMode enum

Режим блочного шифрования.

```cpp
enum class CipherMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| CBC | 1 | Цепочка блоков шифрования, которая объединяет текущий блок с предыдущим для повышения надёжности шифрования. |
| ECB | 2 | Режим электронного блокнота без влияния между блоками; обеспечивает более слабое шифрование. |
| OFB | 3 | Режим обратной связи вывода, который обрабатывает большие входные блоки небольшими частями. |
| CFB | 4 | Режим обратной связи шифра, который обрабатывает большие входные блоки небольшими частями. Правила преобразования отличаются от OFB. |
| CTS | 5 | Режим захвата текста шифра, ведёт себя как CBC за исключением двух последних блоков текста. |

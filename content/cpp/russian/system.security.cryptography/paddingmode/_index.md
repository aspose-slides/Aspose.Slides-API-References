## PaddingMode enum

Определяет, как обрабатывать сообщения, длина которых короче требуемого блока криптографической операции.

```cpp
enum class PaddingMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | 1 | Без заполнения. |
| PKCS7 | 2 | Использовать строку заполнения PKCS #7. |
| Zeros | 3 | Использовать нулевые байты. |
| ANSIX923 | 4 | Использовать строку заполнения ANSIX923. |
| ISO10126 | 5 | Использовать строку заполнения ISO10126. |

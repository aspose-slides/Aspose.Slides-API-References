## DSA::ExportParameters(bool) метод

Экспортирует все параметры.

```cpp
virtual DSAParameters System::Security::Cryptography::DSA::ExportParameters(bool include_private_parameters)=0
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| include_private_parameters | **bool** | True для экспорта как приватных, так и публичных параметров, false для экспорта только публичных параметров. |

### Возвращаемое значение

Экспортированная структура параметров.

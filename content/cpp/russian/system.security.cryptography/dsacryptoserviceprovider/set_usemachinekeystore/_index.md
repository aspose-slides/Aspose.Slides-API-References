## DSACryptoServiceProvider::set_UseMachineKeyStore(bool) метод

Определяет, сохраняется ли ключ в машинном хранилище вместо пользовательского хранилища.

```cpp
static void System::Security::Cryptography::DSACryptoServiceProvider::set_UseMachineKeyStore(bool value)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | **bool** | True, если ключ должен сохраняться в машинном хранилище, false, если он должен сохраняться в пользовательском хранилище. |

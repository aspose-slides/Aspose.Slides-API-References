## DSACryptoServiceProvider::get_UseMachineKeyStore() метод

Проверяет, сохраняется ли ключ в хранилище машины вместо пользовательского хранилища.

```cpp
static bool System::Security::Cryptography::DSACryptoServiceProvider::get_UseMachineKeyStore()
```

### Возвращаемое значение

True если ключ должен сохраняться в хранилище машины, false если он должен сохраняться в пользовательском хранилище.

## RSA::ExportParameters(bool) метод

Экспортирует все параметры.

```cpp
virtual RSAParameters System::Security::Cryptography::RSA::ExportParameters(bool include_private_parameters)=0
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| include_private_parameters | **bool** | True, если экспортировать как приватные, так и публичные параметры, false, если экспортировать только публичные параметры. |

### Возвращаемое значение

Структура экспортированных параметров.

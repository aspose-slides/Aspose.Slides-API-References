## EnumeratorSelectManyAdapter::MoveNext() метод


Перемещает перечислитель к следующему элементу. Если ранее не был ссылаться ни один элемент, устанавливает ссылку на первый доступный элемент. Если достигнут конец контейнера, ничего не делает.

```cpp
bool System::Collections::Generic::Details::EnumeratorSelectManyAdapter<Source, Result>::MoveNext() override
```


### Возвращаемое значение

True если после вызова доступен какой-либо элемент, иначе false.

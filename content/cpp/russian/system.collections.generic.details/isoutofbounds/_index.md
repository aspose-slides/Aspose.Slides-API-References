## System::Collections::Generic::Details::IsOutOfBounds(int, const Container\&) функция

Проверяет, находится ли индекс за пределами границ контейнера, исключая размер контейнера.

```cpp
template<typename Container> bool System::Collections::Generic::Details::IsOutOfBounds(int idx, const Container &container)
```

### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| Container | Container type. |

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| idx | int | Index. |
| container | const Container\& | Container to check index against. |

### Возвращаемое значение

True, если индекс находится за пределами границ контейнера, false в противном случае.

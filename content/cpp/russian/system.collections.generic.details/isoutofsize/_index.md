## System::Collections::Generic::Details::IsOutOfSize(int, const Container\&) функция

Проверяет, выходит ли индекс за пределы контейнера, включая размер контейнера.

```cpp
template<typename Container> bool System::Collections::Generic::Details::IsOutOfSize(int count, const Container &container)
```

### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| Container | Container type. |

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| count | int | Index. |
| container | const Container\& | Container to check index against. |

### Возвращаемое значение

True если индекс выходит за пределы контейнера, false в противном случае.

## EnumeratorSelectAdapter::MoveNext() метод


Перемещает перечислитель к следующему элементу. Если ранее ни один элемент не был выбран, устанавливает ссылку на первый доступный элемент. Если достигнут конец контейнера, ничего не делает.

```cpp
bool System::Collections::Generic::Details::EnumeratorSelectAdapter<Source, Result>::MoveNext() override
```


### Возвращаемое значение

True, если после вызова доступен какой-то элемент, иначе false.

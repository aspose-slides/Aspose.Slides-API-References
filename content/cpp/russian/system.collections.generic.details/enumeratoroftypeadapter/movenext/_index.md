## EnumeratorOfTypeAdapter::MoveNext() метод

Перемещает перечислитель к следующему элементу. Если ранее элемент не был выбран, устанавливает ссылку на первый доступный элемент. Если достигнут конец контейнера, ничего не делает.

```cpp
bool System::Collections::Generic::Details::EnumeratorOfTypeAdapter<Source, Result>::MoveNext() override
```

### Возвращаемое значение

True если после вызова доступен какой-либо элемент, иначе false.

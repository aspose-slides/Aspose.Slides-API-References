## CultureTypes перечисление

Элементы битовой маски категорий культур.

```cpp
enum class CultureTypes : int32_t
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| NeutralCultures | 1 | Культура, специфичная для языка, но не для страны или региона. |
| SpecificCultures | 2 | Специфична для страны/региона. |
| InstalledWin32Cultures | 4 | Культуры, установленные в ОС. |
| AllCultures | n/a | Все доступные культуры. |
| UserCustomCulture | 8 | Пользовательские культуры. |
| ReplacementCultures | 16 | Пользовательские замены для существующих культур. |
| WindowsOnlyCultures | 32 | Устарело. |
| FrameworkCultures | 64 | Устарело. |

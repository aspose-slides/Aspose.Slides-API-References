## NumberStyles перечисление

Стиль числа, разрешенный при разборе.

```cpp
enum class NumberStyles : int32_t
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | 0 | Нет недигитных символов. |
| AllowLeadingWhite | 1 | Разрешать начальные пробелы. |
| AllowTrailingWhite | 2 | Разрешать конечные пробелы. |
| AllowLeadingSign | 4 | Разрешать начальный знак. |
| Integer | n/a | Форматирование целого числа по умолчанию. |
| AllowTrailingSign | 8 | Разрешать конечный знак. |
| AllowParentheses | 16 | Разрешать скобки для отрицательных значений. |
| AllowDecimalPoint | 32 | Разрешать десятичную точку. |
| AllowThousands | 64 | Разрешать разделители групп. |
| Number | n/a | Составной числовой формат по умолчанию. |
| AllowExponent | 128 | Разрешать знак экспоненты. |
| Float | n/a | Формат числа с плавающей точкой по умолчанию. |
| AllowCurrencySymbol | 256 | Разрешать символ валюты. |
| Currency | n/a | Формат валюты по умолчанию. |
| Any | n/a | Разрешать любой спецификатор формата. |
| AllowHexSpecifier | 512 | Разрешать шестнадцатеричные числа. |
| HexNumber | n/a | Формат шестнадцатеричного числа по умолчанию. |

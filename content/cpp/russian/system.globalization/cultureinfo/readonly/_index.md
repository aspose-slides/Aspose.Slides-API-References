## CultureInfo::ReadOnly(const CultureInfoPtr\&) метод


Получает только для чтения версию культуры.

```cpp
static CultureInfoPtr System::Globalization::CultureInfo::ReadOnly(const CultureInfoPtr &culture_info)
```


### Аргументы

| Parameter | Type | Description |
| --- | --- | --- |
| culture_info | const [CultureInfoPtr](../../cultureinfoptr/)\& | Культура, которую нужно обернуть как только для чтения. |

### Возвращаемое значение

Если **culture_info** только для чтения, возвращает тот же объект, в противном случае создает копию только для чтения.

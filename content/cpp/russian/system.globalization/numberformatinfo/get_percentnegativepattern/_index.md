## NumberFormatInfo::get_PercentNegativePattern() const метод


Получает отрицательный шаблон процента.

```cpp
int System::Globalization::NumberFormatInfo::get_PercentNegativePattern() const
```


### Возвращаемое значение

Индекс шаблона в массиве {"-n %", "-n%", "-%n", "%-n", "%n-", "n-%", "n%-", "-% n", "n %-","% n-","% -n","n- %"}, где "n" — число, "%" — знак процента и "-" — знак минуса.

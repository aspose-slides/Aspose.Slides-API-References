## NumberFormatInfo::set_NumberGroupSizes(const ArrayPtr\<int\>\&) метод

Устанавливает количество цифр в группе.

```cpp
void System::Globalization::NumberFormatInfo::set_NumberGroupSizes(const ArrayPtr<int> &value)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | const [ArrayPtr](../../../system/arrayptr/)\<int\>\& | [Array](../../../system/array/) цифр в группе слева направо; каждый элемент должен быть от 1 до 9, последний может быть 0, что означает "объединить всё"; последний элемент повторяется. |

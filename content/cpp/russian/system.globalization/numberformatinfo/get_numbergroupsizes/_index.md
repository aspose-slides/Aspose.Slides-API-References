## NumberFormatInfo::get_NumberGroupSizes() const метод


Получает количество цифр в группе.

```cpp
ArrayPtr<int> System::Globalization::NumberFormatInfo::get_NumberGroupSizes() const
```


### Возвращаемое значение

[Array](../../../system/array/) цифр в группе, слева направо; каждый элемент должен быть от 1 до 9, последний может быть 0, что означает \"combine all\"; последний элемент повторяется.

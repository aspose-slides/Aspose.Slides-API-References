## NumberFormatInfo::set_PercentGroupSizes(const ArrayPtr\<int\>\&) метод

Устанавливает количество цифр в группе значений процента.

```cpp
void System::Globalization::NumberFormatInfo::set_PercentGroupSizes(const ArrayPtr<int> &value)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | const [ArrayPtr](../../../system/arrayptr/)\<int\>\& | [Array](../../../system/array/) количества цифр в группе слева направо; каждый элемент должен быть от 1 до 9, последний может быть 0, что означает \"объединить всё\"; последний элемент повторяется. |

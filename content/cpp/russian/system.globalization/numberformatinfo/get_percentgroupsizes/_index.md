## NumberFormatInfo::get_PercentGroupSizes() const метод

Получает количество цифр в каждой группе значения процента.

```cpp
ArrayPtr<int> System::Globalization::NumberFormatInfo::get_PercentGroupSizes() const
```

### Возвращаемое значение

[Array](../../../system/array/) цифр в группе, слева направо; каждый элемент должен быть от 1 до 9, последний может быть 0, что означает \"объединить все\"; последний элемент повторяется.

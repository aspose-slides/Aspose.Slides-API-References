## CompareInfo::IsPrefix(const String\&, const String\&, CompareOptions) const метод

Проверяет, начинается ли указанная строка с указанного префикса, используя указанные параметры сравнения.

```cpp
virtual bool System::Globalization::CompareInfo::IsPrefix(const String &source, const String &prefix, CompareOptions options) const
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| source | const [String](../../../system/string/)\& | Исходная строка. |
| prefix | const [String](../../../system/string/)\& | Строка префикса. |
| options | [CompareOptions](../../compareoptions/) | Параметры сравнения. |

### Возвращаемое значение

True if string starts with prefix; otherwise false.

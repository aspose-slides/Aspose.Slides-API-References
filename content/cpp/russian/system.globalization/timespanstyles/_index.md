## TimeSpanStyles enum

Specifies the formatting options that customize string parsing for methods that convert a string representation of a time interval into Timespan object.

```cpp
enum class TimeSpanStyles : int32_t
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | 0 | Временной интервал, представленный строкой inptut, интерпретируется как отрицательный временной интервал только если присутствует знак минуса. |
| AssumeNegative | 1 | Временной интервал, представленный строкой inptut, всегда интерпретируется как отрицательный временной интервал. |

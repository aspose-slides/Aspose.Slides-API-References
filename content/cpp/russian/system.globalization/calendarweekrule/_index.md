## CalendarWeekRule enum


Как определить первую неделю года.

```cpp
enum class CalendarWeekRule : int32_t
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| FirstDay | 0 | Первый день года считается началом первой недели года. |
| FirstFullWeek | 1 | Первая полная неделя года считается первой. |
| FirstFourDayWeek | 2 | Первая четырёхдневная неделя года считается первой. |

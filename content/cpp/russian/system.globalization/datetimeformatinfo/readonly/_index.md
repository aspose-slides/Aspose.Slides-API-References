## DateTimeFormatInfo::ReadOnly(const DateTimeFormatInfoPtr\&) method

Получает только читаемую версию форматтера.

```cpp
static DateTimeFormatInfoPtr System::Globalization::DateTimeFormatInfo::ReadOnly(const DateTimeFormatInfoPtr &info)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| info | const [DateTimeFormatInfoPtr](../../datetimeformatinfoptr/)\& | Форматтер, для которого требуется получить только читаемую версию. |

### Возвращаемое значение

Если **info** только для чтения, возвращает его, иначе создает его копию и отмечает её как только для чтения.

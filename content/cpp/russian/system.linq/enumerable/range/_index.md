## Enumerable::Range(int32_t, int32_t) метод


Генерирует последовательность целочисленных чисел в заданном диапазоне.

```cpp
static SharedPtr<Collections::Generic::IEnumerable<int32_t>> System::Linq::Enumerable::Range(int32_t start, int32_t count)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| start | **int32_t** | Значение первого целого числа в последовательности. |
| count | **int32_t** | Количество последовательных целых чисел для генерации. |

### Возвращаемое значение

IEnumerable из **int32_t**, содержащий диапазон последовательных целых чисел.

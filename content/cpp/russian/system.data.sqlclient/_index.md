## Классы

| Класс | Описание |
| --- | --- |
| [SqlConnectionStringBuilder](./sqlconnectionstringbuilder/) | Конструктор соединения на основе SQL. Объекты этого класса должны выделяться только с помощью функции [System::MakeObject()](../system/makeobject/). Никогда не создавайте экземпляр этого типа в стеке или с помощью оператора new, так как это приведёт к ошибкам выполнения и/или сбоям утверждений. Всегда обертывайте этот класс в указатель [System::SmartPtr](../system/smartptr/) и используйте этот указатель для передачи его в функции в качестве аргумента. |

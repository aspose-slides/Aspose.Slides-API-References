## Timer::set_AutoReset(bool) метод

Устанавливает таймер в режим автоматического сброса или отключает его.

```cpp
void System::Timers::Timer::set_AutoReset(bool autoreset)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| autoreset | **bool** | true — чтобы таймер вызывался в цикле, false — чтобы остановить его после первого вызова. |

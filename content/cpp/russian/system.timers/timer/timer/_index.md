## Timer::Timer() конструктор

Информация RTTI.

```cpp
System::Timers::Timer::Timer()
```

## Примечания

Создаёт остановленный таймер с неуказанным интервалом. 
## Timer::Timer(double) конструктор

Создаёт остановленный таймер с указанным интервалом.

```cpp
System::Timers::Timer::Timer(double interval)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| interval | **double** | Интервал между последовательными вызовами таймера или между запуском таймера и первым вызовом, в миллисекундах. |

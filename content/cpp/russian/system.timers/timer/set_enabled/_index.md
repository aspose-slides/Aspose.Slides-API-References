## Timer::set_Enabled(bool) метод

Запускает или останавливает таймер. Запуск таймера не перезапускает счёт времени, если таймер уже выполняется.

```cpp
void System::Timers::Timer::set_Enabled(bool value)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | **bool** | true для запуска таймера, false для остановки. |

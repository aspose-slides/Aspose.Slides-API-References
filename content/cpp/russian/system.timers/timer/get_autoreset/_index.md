## Timer::get_AutoReset() const метод

Проверяет, находится ли таймер в режиме автоматического сброса.

```cpp
bool System::Timers::Timer::get_AutoReset() const
```

### Возвращаемое значение

true если таймер вызывается в цикле, false если он останавливается после первого вызова.

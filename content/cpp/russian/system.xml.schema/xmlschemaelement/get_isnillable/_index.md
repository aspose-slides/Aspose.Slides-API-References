## XmlSchemaElement::get_IsNillable() метод


Возвращает информацию, указывающую, может ли **xsi:nil** встречаться в данных экземпляра. Указывает, можно ли присвоить элементу явное значение nil.

```cpp
bool System::Xml::Schema::XmlSchemaElement::get_IsNillable()
```


### Возвращаемое значение

Если nillable равно **true**, это позволяет экземпляру элемента иметь атрибут **nil**, установленный в **true**. Атрибут **nil** определён как часть пространства имён XML [Schema](../../) для экземпляров. По умолчанию **false**. Необязательно.

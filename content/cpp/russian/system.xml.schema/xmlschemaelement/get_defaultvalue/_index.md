## XmlSchemaElement::get_DefaultValue() метод


Возвращает значение по умолчанию элемента, если его содержимое является простым типом или содержимое элемента имеет значение **textOnly**.

```cpp
String System::Xml::Schema::XmlSchemaElement::get_DefaultValue()
```


### Возвращаемое значение

Значение по умолчанию для элемента. По умолчанию это **nullptr**. Опционально.

## XmlSchemaElement::get_IsAbstract() method

Возвращает информацию, указывающую, может ли элемент использоваться в документе-экземпляре.

```cpp
bool System::Xml::Schema::XmlSchemaElement::get_IsAbstract()
```

### Возвращаемое значение

Если **true**, элемент не может появляться в документе-экземпляре. По умолчанию **false**. Необязательно.

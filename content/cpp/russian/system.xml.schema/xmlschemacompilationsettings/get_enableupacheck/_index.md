## XmlSchemaCompilationSettings::get_EnableUpaCheck() метод

Возвращает значение, указывающее, должен ли [XmlSchemaSet](../../xmlschemaset/) проверять нарушения Unique Particle Attribution (UPA).

```cpp
bool System::Xml::Schema::XmlSchemaCompilationSettings::get_EnableUpaCheck()
```

### Возвращаемое значение

**true** если [XmlSchemaSet](../../xmlschemaset/) должен проверять нарушения Unique Particle Attribution (UPA); в противном случае, **false**. По умолчанию **true**.

## XmlSchemaParticle::get_MinOccursString() метод

Возвращает число в виде строкового значения. Минимальное количество раз, которое частица может появиться.

```cpp
String System::Xml::Schema::XmlSchemaParticle::get_MinOccursString()
```

### Возвращаемое значение

Число в виде строкового значения. [String::Empty](../../../system/string/empty/) указывает, что **MinOccurs** равен значению по умолчанию. Значение по умолчанию — **nullptr**.

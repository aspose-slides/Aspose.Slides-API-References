## XmlSchemaParticle::get_MaxOccursString() метод

Возвращает число в виде строкового значения. Максимальное количество раз, когда частица может появиться.

```cpp
String System::Xml::Schema::XmlSchemaParticle::get_MaxOccursString()
```

### Возвращаемое значение

Число в виде строкового значения. [String::Empty](../../../system/string/empty/) указывает, что **MaxOccurs** равен значению по умолчанию. Значение по умолчанию — **nullptr**.

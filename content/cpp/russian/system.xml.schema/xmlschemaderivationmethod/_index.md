## XmlSchemaDerivationMethod перечисление

Предоставляет различные методы предотвращения наследования.

```cpp
enum class XmlSchemaDerivationMethod
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Empty | 0 | Переопределяет метод наследования по умолчанию, позволяя любое наследование. |
| Substitution | 1 | Относится к наследованиям посредством **Substitution**. |
| Extension | 2 | Относится к наследованиям посредством **Extension**. |
| Restriction | 4 | Относится к наследованиям посредством **Restriction**. |
| List | 8 | Относится к наследованиям посредством **List**. |
| Union | 16 | Относится к наследованиям посредством **Union**. |
| All | 255 | **#all**. Относится ко всем методам наследования. |
| None | 256 | Принимает метод наследования по умолчанию. |

## XmlSchemaAnyAttribute::get_ProcessContents() метод

Возвращает информацию о том, как приложение или XML-процессор должен обрабатывать проверку XML-документов для атрибутов, указанных элементом **anyAttribute**.

```cpp
XmlSchemaContentProcessing System::Xml::Schema::XmlSchemaAnyAttribute::get_ProcessContents()
```

### Возвращаемое значение

Одно из значений XmlSchemaContentProcessing. Если атрибут **processContents** не указан, значение по умолчанию — **Strict**.

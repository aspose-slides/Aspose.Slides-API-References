## XmlSchemaAny::get_ProcessContents() метод

Возвращает информацию о том, как приложение или XML-процессор должно обрабатывать проверку XML-документов для элементов, указанных в элементе **any**.

```cpp
XmlSchemaContentProcessing System::Xml::Schema::XmlSchemaAny::get_ProcessContents()
```

### Возвращаемое значение

Одно из значений XmlSchemaContentProcessing. Если атрибут **processContents** не указан, по умолчанию используется **Strict**.

## XmlSchemaComplexType::get_IsMixed() метод


Возвращает информацию, определяющую, имеет ли сложный тип смешанную модель содержимого (разметка внутри содержимого).

```cpp
bool System::Xml::Schema::XmlSchemaComplexType::get_IsMixed() override
```


### Возвращаемое значение

**true**, если символьные данные могут появляться между дочерними элементами этого сложного типа; в противном случае **false**. По умолчанию **false**. Необязательно.

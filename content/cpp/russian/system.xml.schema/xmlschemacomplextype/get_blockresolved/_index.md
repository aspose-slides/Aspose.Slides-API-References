## XmlSchemaComplexType::get_BlockResolved() метод


Возвращает значение после того, как тип был скомпилирован в информационный набор после проверки схемы (infoset). Это значение указывает, как тип применяется, когда в документе экземпляра используется **xsi:type**.

```cpp
XmlSchemaDerivationMethod System::Xml::Schema::XmlSchemaComplexType::get_BlockResolved()
```


### Return Value

Значение информационного набора после проверки схемы. По умолчанию это значение **BlockDefault** на элементе **schema**.

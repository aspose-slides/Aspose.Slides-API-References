## XmlSchemaComplexType::get_IsAbstract() метод

Возвращает информацию, определяющую, может ли элемент **complexType** использоваться в документе экземпляра.

```cpp
bool System::Xml::Schema::XmlSchemaComplexType::get_IsAbstract()
```

### Возвращаемое значение

Если **true**, элемент не может использовать этот **complexType** элемент напрямую и должен использовать сложный тип, производный от этого **complexType** элемента. По умолчанию **false**. Необязательно.

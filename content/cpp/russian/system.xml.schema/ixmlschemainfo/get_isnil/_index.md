## IXmlSchemaInfo::get_IsNil() метод

Возвращает значение, указывающее, является ли значение этого проверенного XML-узла **nil**.

```cpp
virtual bool System::Xml::Schema::IXmlSchemaInfo::get_IsNil()=0
```

### Возвращаемое значение

**true** если значение для этого проверенного XML-узла равно **nil**; в противном случае, **false**.

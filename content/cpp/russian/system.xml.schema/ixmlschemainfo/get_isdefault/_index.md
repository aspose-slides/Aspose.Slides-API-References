## IXmlSchemaInfo::get_IsDefault() метод


Возвращает значение, указывающее, был ли этот проверенный XML-узел установлен в результате применения значения по умолчанию во время проверки схемы XML [Schema](../../) Definition Language (XSD).

```cpp
virtual bool System::Xml::Schema::IXmlSchemaInfo::get_IsDefault()=0
```


### Возвращаемое значение

**true** если этот проверенный XML-узел был установлен в результате применения значения по умолчанию во время проверки схемы; иначе, **false**.

## XmlSchemaFacet::get_IsFixed() метод


Возвращает информацию, указывающую, что этот фасет зафиксирован.

```cpp
virtual bool System::Xml::Schema::XmlSchemaFacet::get_IsFixed()
```


### Возвращаемое значение

Если **true**, значение зафиксировано; в противном случае **false**. По умолчанию **false**. Необязательно.

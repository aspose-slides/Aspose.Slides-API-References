## InferenceOption enum

Влияет на сведения о наличии и типе, выводимые классом [XmlSchemaInference](../) для элементов и атрибутов в документе XML.

```cpp
enum class InferenceOption
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Restricted | 0 | Указывает, что для конкретного элемента или атрибута должна быть выведена более ограничительная декларация схемы. |
| Relaxed | 1 | Указывает, что для конкретного элемента или атрибута должна быть выведена менее ограничительная декларация схемы. |

## XmlSchemaValidator::ValidateEndOfAttributes(const SharedPtr\<XmlSchemaInfo\>\&) метод

Проверяет, присутствуют ли все требуемые атрибуты в контексте элемента, и готовит объект [XmlSchemaValidator](../) для проверки дочернего содержимого элемента.

```cpp
void System::Xml::Schema::XmlSchemaValidator::ValidateEndOfAttributes(const SharedPtr<XmlSchemaInfo> &schemaInfo)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| schemaInfo | const [SharedPtr](../../../system/sharedptr/)\<[XmlSchemaInfo](../../xmlschemainfo/)\>\& | Объект [XmlSchemaInfo](../../xmlschemainfo/), свойства которого устанавливаются при успешной проверке того, что все требуемые атрибуты в контексте элемента присутствуют. Этот параметр может быть **nullptr**. |

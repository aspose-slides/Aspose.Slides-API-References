## XmlSchemaForm перечисление

Указывает, требуется ли, чтобы атрибуты или элементы были квалифицированы префиксом пространства имён.

```cpp
enum class XmlSchemaForm
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | 0 | Форма элемента и атрибута не указана в схеме. |
| Qualified | 1 | Элементы и атрибуты должны быть квалифицированы префиксом пространства имён. |
| Unqualified | 2 | Элементы и атрибуты не обязаны быть квалифицированы префиксом пространства имён. |

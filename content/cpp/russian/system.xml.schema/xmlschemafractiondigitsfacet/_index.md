## XmlSchemaFractionDigitsFacet класс


Указывает ограничение на количество цифр, которое может быть введено для дробного значения элемента simpleType. Значение fractionDigits должно быть положительным целым числом. Представляет собой фасет **fractionDigits** Всемирного [Web](../../system.web/) консорциума (W3C).

```cpp
class XmlSchemaFractionDigitsFacet : public System::Xml::Schema::XmlSchemaNumericFacet
```

## Замечания

Экземпляры этого класса должны создаваться только с помощью функции [System::MakeObject()](../../system/makeobject/). Никогда не создавайте экземпляры этого типа в стеку или с помощью оператора new, так как это приведёт к ошибкам выполнения и/или сбоям проверок. Всегда оборачивайте этот класс в указатель [System::SmartPtr](../../system/smartptr/) и используйте этот указатель для передачи его в функции в качестве аргумента.

## XmlSchema::get_ElementFormDefault() метод


Возвращает форму для элементов, объявленных в целевом пространстве имен схемы.

```cpp
XmlSchemaForm System::Xml::Schema::XmlSchema::get_ElementFormDefault()
```


### Возвращаемое значение

Значение XmlSchemaForm, указывающее, должны ли элементы из целевого пространства имен быть квалифицированы префиксом пространства имен. По умолчанию [XmlSchemaForm::None](../../xmlschemaform/).

## XmlSchema::get_AttributeFormDefault() метод

Возвращает форму атрибутов, объявленных в целевом пространстве имён схемы.

```cpp
XmlSchemaForm System::Xml::Schema::XmlSchema::get_AttributeFormDefault()
```

### Возвращаемое значение

Значение XmlSchemaForm, которое указывает, требуется ли квалифицировать атрибуты из целевого пространства имён с префиксом пространства имён. Значение по умолчанию — [XmlSchemaForm::None](../../xmlschemaform/).

## XmlSchemaType::get_IsMixed() метод


Возвращает значение, указывающее, имеет ли этот тип смешанную модель содержимого. Этот вызов допустим только для сложного типа.

```cpp
virtual bool System::Xml::Schema::XmlSchemaType::get_IsMixed()
```


### Возвращаемое значение

**true** если тип имеет смешанную модель содержимого; в противном случае **false**. По умолчанию **false**.

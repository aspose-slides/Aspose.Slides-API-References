## XmlAtomicValue::get_IsNode() метод

Возвращает значение, указывающее, является ли проверенный элемент XML или атрибут [XPath](../../../system.xml.xpath/) узлом или атомарным значением.

```cpp
bool System::Xml::Schema::XmlAtomicValue::get_IsNode() override
```

### Возвращаемое значение

**true** если проверенный элемент XML или атрибут является [XPath](../../../system.xml.xpath/) узлом; **false** если проверенный элемент XML или атрибут является атомарным значением.

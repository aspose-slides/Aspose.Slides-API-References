## XmlSchemaComplexContent::get_IsMixed() метод

Возвращает информацию, определяющую, имеет ли тип смешанную модель содержимого.

```cpp
bool System::Xml::Schema::XmlSchemaComplexContent::get_IsMixed()
```

### Возвращаемое значение

Если это значение **true**, символьные данные могут появляться между дочерними элементами сложного типа (модель смешанного содержимого). По умолчанию **false**. Необязательно.

## XmlSchemaAttribute::get_AttributeType() метод


Возвращает объект, основанный на значении [XmlSchemaAttribute::get_SchemaType](../get_schematype/) или [XmlSchemaAttribute::get_SchemaTypeName](../get_schematypename/) атрибута, который содержит посткомпиляционную интерпретацию значения **AttributeType**.

```cpp
SharedPtr<Object> System::Xml::Schema::XmlSchemaAttribute::get_AttributeType()
```


### Возвращаемое значение

Объект, который содержит посткомпиляционную интерпретацию значения **AttributeType**.

Устарело
:   Рекомендуется использовать метод get_AttributeSchemaType вместо него.

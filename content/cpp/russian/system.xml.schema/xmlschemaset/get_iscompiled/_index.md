## XmlSchemaSet::get_IsCompiled() метод


Возвращает значение, указывающее, были ли схемы языка определения XML [Schema](../../) (XSD) в [XmlSchemaSet](../) скомпилированы.

```cpp
bool System::Xml::Schema::XmlSchemaSet::get_IsCompiled()
```


### Возвращаемое значение

**true** если схемы в [XmlSchemaSet](../) были скомпилированы с момента последнего добавления или удаления схемы из [XmlSchemaSet](../); в противном случае, **false**.

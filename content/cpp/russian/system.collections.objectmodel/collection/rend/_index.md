## Collection::rend() метод

Получает обратный итератор для несуществующего элемента, находящегося перед началом коллекции.

```cpp
reverse_iterator System::Collections::ObjectModel::Collection<T>::rend() noexcept
```

### Возвращаемое значение

Обратный итератор, указывающий на теоретический элемент, предшествующий первому элементу в списке (последний в обратном порядке).

## Collection::rend() const метод

Получает обратный итератор для несуществующего элемента, находящегося перед началом const-квалифицированной коллекции.

```cpp
const_reverse_iterator System::Collections::ObjectModel::Collection<T>::rend() const noexcept
```

### Возвращаемое значение

Обратный итератор, указывающий на теоретический элемент, предшествующий первому элементу в const-квалифицированном списке (последний в обратном порядке).

## Collection::crend() const метод

Получает обратный итератор для несуществующего const-квалифицированного элемента перед началом коллекции.

```cpp
const_reverse_iterator System::Collections::ObjectModel::Collection<T>::crend() const noexcept
```

Обратный итератор, указывающий на теоретический const-квалифицированный элемент, предшествующий первому элементу в списке (последний в обратном порядке).

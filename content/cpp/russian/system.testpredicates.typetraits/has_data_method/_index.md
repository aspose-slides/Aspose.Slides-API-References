## has_data_method структура

Проверяет, имеет ли тип метод data(). Если да, наследует std::true_type, иначе наследует std::false_type.

```cpp
template<typename T,typename Enable>class has_data_method : public std::false_type
```

### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| T | Тип для проверки. |
| Enable | Формальный аргумент для работы SFINAE. |

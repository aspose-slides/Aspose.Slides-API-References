## System::CheckedCast(TFrom) функция


Определяет, попадает ли указанное значение в диапазон значений типа **TTo**, и если да, приводит его к типу **TTo**.

```cpp
template<typename TTo,typename TFrom> TTo System::CheckedCast(TFrom value)
```


### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| TTo | Тип, к которому будет приведено указанное значение |
| TFrom | Тип указанного значения |

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | TFrom | Значение для приведения |

### Возвращаемое значение

Значение типа **TTo**, эквивалентное **value**

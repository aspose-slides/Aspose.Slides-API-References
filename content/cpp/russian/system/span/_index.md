## Span класс


Представляет собой сплошной регион произвольной памяти, аналогичный C++20's std::span.

```cpp
template<typename T>class Span : public System::Details::SpanCore<T, Span<T>, Span<T>>
```


### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| T | Тип элементов в span. Этот класс предоставляет типобезопасный способ работы со сплошными последовательностями объектов. Его можно использовать для обёртывания массивов, стековых массивов или необработанных указателей, обеспечивая проверку границ. [Span](./) не владеет памятью, на которую он указывает, — это лишь представление существующей памяти. |
## Методы

| Метод | Описание |
| --- | --- |
| void [Clear](./clear/)() const | Очищает содержимое span, устанавливая все элементы в значение по умолчанию. |
| void [Fill](./fill/)(const T\&) const | Заполняет span указанным значением. |
| static [ThisType](./) [to_Span](./to_span/)(const typename BaseType::ArrayPtrT\&) | Преобразует массив в [Span](./). |

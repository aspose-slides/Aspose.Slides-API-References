## Uri::IsHexEncoding(const String\&, int32_t) method


Определяет, закодирован ли символ в указанной строке на указанной позиции в шестнадцатеричном виде.

```cpp
static bool System::Uri::IsHexEncoding(const String &pattern, int32_t index)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| pattern | const [String](../../string/)\& | Строка для проверки |
| index | **int32_t** | Позиция символа, начиная с нуля |

### Возвращаемое значение

True, если указанный символ закодирован в шестнадцатеричном виде, иначе — false

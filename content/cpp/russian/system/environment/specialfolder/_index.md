## SpecialFolder enum

Представляет специальные системные папки.

```cpp
enum class SpecialFolder
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Desktop | 0 | Логический рабочий стол. |
| Programs | 2 | Папка, содержащая группы программ пользователя. |
| Personal | 5 | Папка, служащая общим хранилищем документов. |
| MyDocuments | 5 | Каталог \"My Documents\". |
| Favorites | 6 | Папка, служащая общим хранилищем избранных элементов пользователя. |
| Startup | 7 | Папка, соответствующая группе программ «Startup» пользователя. |
| Recent | 8 | Папка, содержащая последние использованные документы пользователя. |
| SendTo | 9 | Папка, служащая контейнером для пунктов меню «Send To». |
| StartMenu | 11 | Папка, служащая контейнером для пунктов меню «Start». |
| MyMusic | 13 | Каталог \"My Music\". |
| DesktopDirectory | 16 | Папка, используемая для физического хранения элементов на рабочем столе. |
| MyComputer | 17 | Каталог \"My Computer\". |
| Templates | 21 | Папка, служащая общим контейнером для шаблонов документов. |
| ApplicationData | 26 | Папка, используемая для хранения данных приложения для текущего Roaming-пользователя. |
| LocalApplicationData | 28 | Папка, используемая для хранения данных приложения для текущего Non-roaming-пользователя. |
| InternetCache | 32 | Папка, используемая как хранилище временных файлов Интернета. |
| Cookies | 33 | Папка, используемая как хранилище файлов cookie Интернета. |
| History | 34 | Папка, используемая как хранилище элементов истории Интернета. |
| CommonApplicationData | 35 | Папка, используемая для хранения данных приложения, используемых всеми пользователями. |
| System | 37 | Системная папка. |
| ProgramFiles | 38 | Папка файлов программ. |
| MyPictures | 39 | Папка \"My picutres\". |
| UserProfile | 40 | Папка профиля пользователя. |
| CommonProgramFiles | 43 | Каталог, используемый как хранилище компонентов, общих для приложений. |

## Environment::get_Version() метод

Возвращает объект [Version](../../version/), который представляет информацию о версии среды выполнения общего языка. Номер версии, возвращаемый этим методом, скорее фиктивный и не означает, что все классы библиотеки работают в соответствии с возвращённой версией.

```cpp
static Version System::Environment::get_Version()
```

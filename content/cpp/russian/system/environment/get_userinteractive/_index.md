## Environment::get_UserInteractive() метод

Определяет, работает ли текущий процесс в интерактивном режиме пользователя.

```cpp
static bool System::Environment::get_UserInteractive()
```

### Возвращаемое значение

True если текущий процесс работает в интерактивном режиме пользователя, иначе — false

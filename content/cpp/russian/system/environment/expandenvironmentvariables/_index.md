## Environment::ExpandEnvironmentVariables(const String\&) метод

Заменяет имена переменных окружения, найденные в указанной строке, значениями этих переменных и возвращает полученную строку.

```cpp
static String System::Environment::ExpandEnvironmentVariables(const String &name)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| name | const [String](../../string/)\& | Строка, содержащая имена переменных окружения |

### Возвращаемое значение

Строка, полученная в результате замены имен переменных в **name** их значениями.

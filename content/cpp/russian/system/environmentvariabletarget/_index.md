## Перечисление EnvironmentVariableTarget

Указывает место расположения переменной среды.

```cpp
enum EnvironmentVariableTarget
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Process | 0 | [Environment](../environment/) блок, связанный с текущим процессом. |
| User | 1 | Ключ реестра HKEY_CURRENT_USER\[Environment](../environment/). |
| Machine | 2 | Ключ реестра HKEY_LOCAL_MACHINE\[System](../)\CurrentControlSet\Control\Session Manager\[Environment](../environment/). |

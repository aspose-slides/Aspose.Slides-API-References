## Boolean::TryParse(const String\&, bool\&) method

Преобразует указанную строку в значение типа bool.

```cpp
static bool System::Boolean::TryParse(const String &value, bool &result)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | const [String](../../string/)\& | Строка для преобразования. |
| result | **bool**\& | Ссылка на переменную типа bool, в которую помещается результат преобразования; результат равен true, если указанная строка равна "True", и false, если указанная строка равна "False". |

### Возвращаемое значение

True, если указанная строка равна "True" или "False", иначе - false.

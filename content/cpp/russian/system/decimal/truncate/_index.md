## Decimal::Truncate(const Decimal\&) метод


Возвращает объект [Decimal](../), представляющий значение, у которого целая часть равна целой части значения, представленного указанным объектом [Decimal](../), при этом все дробные цифры отбрасываются.

```cpp
static Decimal System::Decimal::Truncate(const Decimal &d)
```


### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| d | const [Decimal](../)\& | Десятичное число |

### Возвращаемое значение

Объект [Decimal](../), представляющий десятичное значение, у которого целая часть равна целой части указанного значения, а дробные цифры равны 0.

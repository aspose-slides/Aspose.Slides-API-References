## Decimal::Compare(const Decimal\&, const Decimal\&) метод


Определяет, меньше ли значение, представляемое первым объектом [Decimal](../), равно ли оно или больше значения, представляемого вторым объектом [Decimal](../).

```cpp
static int System::Decimal::Compare(const Decimal &d1, const Decimal &d2)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| d1 | const [Decimal](../)\& | Первый сравниваемый |
| d2 | const [Decimal](../)\& | Второй сравниваемый |

### Возвращаемое значение

- 1, если значение, представляемое **d1**, меньше значения, представляемого **d2**; 0, если значения равны; 1, если значение, представляемое **d1**, больше значения, представляемого **d2**

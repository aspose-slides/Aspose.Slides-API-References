## Decimal::operator<=(const Decimal\&) const метод


Определяет, является ли значение, представляемое текущим объектом, меньше или равно значению, представляемому указанным объектом.

```cpp
bool System::Decimal::operator<=(const Decimal &d) const
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| d | const [Decimal](../)\& | Объект [Decimal](../) для сравнения с текущим объектом |

### Возвращаемое значение

True, если значение, представляющее текущий объект, меньше или равно значению, представляемому **d**, иначе — false

## Decimal::CompareTo(const Decimal\&) const метод

Определяет, меньше ли значение, представленное текущим объектом, равно ли оно или больше значения, представленного указанным объектом.

```cpp
int System::Decimal::CompareTo(const Decimal &d) const
```

### Arguments

| Параметр | Тип | Описание |
| --- | --- | --- |
| d | const [Decimal](../)\& | Сравниваемое значение |

### Return Value

- 1 если значение, представленное текущим объектом, меньше значения, представленного **d**; 0 если значения равны; 1 если значение, представленное текущим объектом, больше значения, представленного **d**

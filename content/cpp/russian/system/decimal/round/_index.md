## Decimal::Round(const Decimal\&, MidpointRounding) метод

Округляет указанное значение до ближайшего целого числа. Параметр определяет поведение функции, если указанное значение находится на одинаковом расстоянии от двух ближайших чисел.

```cpp
static Decimal System::Decimal::Round(const Decimal &d, MidpointRounding mode=MidpointRounding::ToEven)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| d | const [Decimal](../)\& | Значение для округления |
| mode | [MidpointRounding](../../midpointrounding/) | Указывает, как выполнять округление, если **value** находится на одинаковом расстоянии от двух ближайших чисел. |

### Возвращаемое значение

**d** округленное до ближайшего целого значения

## Decimal::Round(const Decimal\&, int, MidpointRounding) метод

Округляет указанное значение до ближайшего значения с заданным числом знаков после запятой. Параметр определяет поведение функции, если указанное значение находится на одинаковом расстоянии от двух ближайших чисел.

```cpp
static Decimal System::Decimal::Round(const Decimal &d, int digits, MidpointRounding mode=MidpointRounding::ToEven)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| d | const [Decimal](../)\& | Значение для округления |
| digits | int | Количество знаков после запятой в округленном значении |
| mode | [MidpointRounding](../../midpointrounding/) | Указывает, как выполнять округление, если **value** находится на одинаковом расстоянии от двух ближайших чисел. |

### Возвращаемое значение

Число с указанным количеством знаков, ближайшее к **value**

## Decimal::operator>=(const Decimal\&) const метод

Определяет, является ли значение, представленное текущим объектом, больше или равно значению, представленному указанным объектом.

```cpp
bool System::Decimal::operator>=(const Decimal &d) const
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| d | const [Decimal](../)\& | Объект [Decimal](../) для сравнения текущего объекта |

### Возвращаемое значение

True если значение, представленное текущим объектом, больше или равно значению, представленному **d**, иначе - false

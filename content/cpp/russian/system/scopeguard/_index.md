## ScopeGuard структура

Сервисный класс, предоставляющий услуги по выполнению определённого объект-функции, когда экземпляр класса выходит из области видимости.

```cpp
template<typename F>class ScopeGuard
```

### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| F | Тип объект-функции, вызываемого экземплярами класса ScopedGuard |
## Методы

| Метод | Описание |
| --- | --- |
| void [Disable](./disable/)() | Отключает вызов охранника. |
|   [ScopeGuard](./scopeguard/)(F) | Конструирует экземпляр, настроенный для вызова указанного объект-функции. |
|   [~ScopeGuard](./~scopeguard/)() | Вызывает объект-функцию, переданный конструктору. |

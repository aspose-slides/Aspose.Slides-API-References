## SmartPtr::dynamic_pointer_cast() const метод

Преобразует указатель к другому типу, используя dynamic_cast для указываемого объекта.

```cpp
template<class Y> SmartPtr<Y> System::SmartPtr<T>::dynamic_pointer_cast() const
```

### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| Y | Целевой тип указываемого объекта. |

### Возвращаемое значение

Указатель изменённого типа, который всегда находится в режиме shared.

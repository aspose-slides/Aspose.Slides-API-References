## SmartPtr::ToObjectPtr() const метод

Преобразует любой тип указателя в указатель на [Object](../../object/). Не требует, чтобы тип Pointee_ был полным.

```cpp
SmartPtr<Object> System::SmartPtr<T>::ToObjectPtr() const
```

### Возвращаемое значение

Сырой указатель на объект, на который ссылаются (если он существует), или nullptr.

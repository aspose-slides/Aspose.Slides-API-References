## SmartPtr::const_pointer_cast() const метод

Преобразует указатель к другому типу, используя const_cast над объектом, на который он указывает.

```cpp
template<class Y> SmartPtr<Y> System::SmartPtr<T>::const_pointer_cast() const
```

### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| Y | Целевой тип указываемого объекта. |

### Возвращаемое значение

Указатель изменённого типа, который всегда находится в режиме shared.

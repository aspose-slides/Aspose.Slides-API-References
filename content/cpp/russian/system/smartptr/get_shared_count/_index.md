## SmartPtr::get_shared_count() const метод


Получает количество общих указателей, существующих для ссылочного объекта, включая текущий. Утверждает, что текущий указатель находится в режиме совместного использования.

```cpp
int System::SmartPtr<T>::get_shared_count() const
```


### Возвращаемое значение

Количество общих указателей, существующих для ссылочного объекта, если таковые имеются. Если указатель равен null, возвращает 0.

## SmartPtr::Is(const System::TypeInfo\&) const метод

Проверяет, является ли указанный объект определённого типа или его дочерним типом. Соответствует семантике оператора C# 'is'.

```cpp
bool System::SmartPtr<T>::Is(const System::TypeInfo &target) const
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | Указывает target тип для проверки. |

### Возвращаемое значение

Истина, если проверка в стиле C# 'is' положительна, и ложь в противном случае.

Реализация.

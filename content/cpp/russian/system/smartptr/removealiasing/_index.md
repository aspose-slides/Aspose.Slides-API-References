## SmartPtr::RemoveAliasing() const метод

Удаляет алиасинг (созданный конструктором алиасинга) из указателя, гарантируя, что он управляет (если shared) или отслеживает (если weak) тот же объект, на который указывает.

```cpp
SmartPtr_ System::SmartPtr<T>::RemoveAliasing() const
```

### Возвращаемое значение

Указатель на тот же объект, на который указывает данный указатель, который следит за временем жизни того же объекта.

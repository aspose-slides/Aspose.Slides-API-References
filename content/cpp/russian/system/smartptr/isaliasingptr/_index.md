## SmartPtr::IsAliasingPtr() const метод

Проверяет, указывает ли указатель на объект, отличный от владения (созданный конструктором алиасинга).

```cpp
bool System::SmartPtr<T>::IsAliasingPtr() const
```


### Возвращаемое значение

True если у указателя есть различие между указываемым и принадлежащим объектами, иначе false.

## SmartPtr::static_pointer_cast() const метод

Преобразует указатель к другому типу, используя static_cast над объектом, на который он указывает.

```cpp
template<class Y> SmartPtr<Y> System::SmartPtr<T>::static_pointer_cast() const
```

### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| Y | Тип целевого объекта, на который указывает указатель. |

### Возвращаемое значение

Указатель изменённого типа, который всегда находится в режиме shared.

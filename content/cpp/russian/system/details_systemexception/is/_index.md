## Подробности_SystemException::Is(const System::TypeInfo\&) const method




```cpp
bool System::Details_SystemException::Is(const System::TypeInfo &target) const override
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) структура, описывающая тип, с которым сравнивается текущий объект. |

### Возвращаемое значение

True если объект имеет указанный тип или его подкласс, false в противном случае.
## Примечания

Проверьте, представляет ли объект экземпляр типа, описанного targetType. Аналог оператора C# 'is' operator.

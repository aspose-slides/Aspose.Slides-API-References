## GC::GetTotalAllocatedBytes(bool) метод

Возвращает общее количество выделенных байтов.

```cpp
static int64_t System::GC::GetTotalAllocatedBytes(bool precise=false)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| precise | **bool** | Игнорируется |

### Возвращаемое значение

Количество байтов приватной памяти, в настоящее время выделенной текущим процессом.

Это заглушка реализации для метода .NET

## System::Greater(const ExpressionT\&, const ConstantT\&) функция

Реализует перевод относительного шаблона ‘>’.

```cpp
template<class ExpressionT,class ConstantT> bool System::Greater(const ExpressionT &left, const ConstantT &constant)
```

### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| ExpressionT | тип левого выражения. |
| ConstantT | тип константного выражения. |

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| left | const ExpressionT\& | выражение, которое будет проверяться. |
| constant | const ConstantT\& | выражение, которое будет сравниваться с левым. |

### Возвращаемое значение

true, если проверка типа успешна; false — в противном случае.

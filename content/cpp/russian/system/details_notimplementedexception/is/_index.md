## Details_NotImplementedException::Is(const System::TypeInfo\&) const method




```cpp
bool System::Details_NotImplementedException::Is(const System::TypeInfo &target) const override
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) структура, описывающая тип, против которого проверяется текущий объект. |

### Возвращаемое значение

True, если объект имеет указанный тип или его подкласс, иначе false.

## Замечания

Проверяет, представляет ли объект экземпляр типа, описанного параметром targetType. Аналог оператора C# 'is'.

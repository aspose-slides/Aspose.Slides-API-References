## Перечисление UriFormat

Указывает, как производится экранирование URI.

```cpp
enum class UriFormat
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| UriEscaped | 1 | Экранирование согласно RFC-2396. |
| Unescaped | 2 | Без экранирования. |
| SafeUnescaped | 3 | Экранируются только символы, имеющие зарезервированное значение в URI. |

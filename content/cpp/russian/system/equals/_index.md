## System::Equals(const TA\&, const TB\&) функция


Определяет равенство двух значений, применяя [operator==()](../operator_equal_equal/) к ним.

```cpp
template<typename TA,typename TB> bool System::Equals(const TA &a, const TB &b)
```


### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| TA | Тип первого сравниваемого |
| TB | Тип второго сравниваемого |

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| a | const TA\& | Первый сравниваемый |
| b | const TB\& | Второй сравниваемый |

### Возвращаемое значение

Булево значение, возвращаемое [operator==()](../operator_equal_equal/), применённое к **a** и **b**

## Details_IndexOutOfRangeException::Is(const System::TypeInfo\&) const метод

```cpp
bool System::Details_IndexOutOfRangeException::Is(const System::TypeInfo &target) const override
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) структура, описывающая тип, с которым проверяется текущий объект. |

### Возвращаемое значение

true, если объект имеет указанный тип или его подкласс; false — в противном случае.

## Примечания

Проверьте, представляет ли объект экземпляр типа, описанного в targetType. Аналог оператора C# 'is'.

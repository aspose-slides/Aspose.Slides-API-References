## System::MemberwiseClone(T *) функция

Выполняет побайтовое клонирование с использованием конструктора копирования.

```cpp
template<typename T> SmartPtr<Object> System::MemberwiseClone(T *ptr)
```

### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| T | Класс, который будет копировано-конструирован. Информация о подклассах будет утеряна. |

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| ptr | T * | Указатель на объект для клонирования. |

### Возвращаемое значение

Указатель на клонированный объект.

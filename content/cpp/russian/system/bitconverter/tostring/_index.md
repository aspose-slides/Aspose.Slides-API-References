## BitConverter::ToString(const ArrayPtr\<uint8_t\>\&, bool, const String\&) метод


Преобразует все значения указанного массива байтов в их шестнадцатеричное строковое представление. Регистр букв, используемых в шестнадцатеричной нотации, и разделитель, вставляемый между каждой парой соседних байтов, задаются соответствующими аргументами.

```cpp
static String System::BitConverter::ToString(const ArrayPtr<uint8_t> &value, bool uppercase=1, const String &separator=u"-")
```


### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | const [ArrayPtr](../../arrayptr/)\<**uint8_t**\>\& | [Array](../../array/) содержащий байты для преобразования |
| uppercase | **bool** | Указывает регистр букв, используемых в результирующем шестнадцатеричном представлении |
| separator | const [String](../../string/)\& | Строка, используемая в качестве разделителя, вставляемого между каждой парой соседних байтов в результирующей строке |

### Возвращаемое значение

[String](../../string/) содержащий шестнадцатеричное представление указанного массива байтов

| startIndex | int | Индекс в указанном массиве, с которого начинать преобразование |

### Возвращаемое значение

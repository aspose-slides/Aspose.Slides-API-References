```cpp
static double System::BitConverter::Int64BitsToDouble(int64_t value)
```


### Аргументы

| --- | --- | --- |
| value | **int64_t** | 64-битное целочисленное значение, бинарное представление которого равно представлению указанного значения |

Значение двойной точности с плавающей запятой

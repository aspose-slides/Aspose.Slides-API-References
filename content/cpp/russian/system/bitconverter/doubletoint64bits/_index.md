## BitConverter::DoubleToInt64Bits(double) метод


Возвращает 64-битное целое значение, двоичное представление которого равно двоичному представлению указанного значения двойной точности с плавающей запятой.

```cpp
static int64_t System::BitConverter::DoubleToInt64Bits(double value)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | **double** | Значение двойной точности с плавающей запятой |

### Возвращаемое значение

64-битное целое значение, двоичное представление которого равно представлению указанного значения

## SmartPtrInfo::operator!() const метод


Проверяет, не указывает ли объект info на ненулевой указатель.

```cpp
bool System::SmartPtrInfo::operator!() const
```


### Возвращаемое значение

False, если объект info не указывает ни на какой указатель или если целевой указатель равен null, true в противном случае.

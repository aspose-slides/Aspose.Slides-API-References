## IConvertible::ToSByte(System::SharedPtr\<System::IFormatProvider\>) method


Преобразует значение этого экземпляра в эквивалентное 8-битное целое со знаком, используя указанные сведения о форматировании, зависящие от культуры.

```cpp
virtual int8_t System::IConvertible::ToSByte(System::SharedPtr<System::IFormatProvider> provider)=0
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| provider | [System::SharedPtr](../../sharedptr/)\<[System::IFormatProvider](../../iformatprovider/)\> | Реализация интерфейса [System::IFormatProvider](../../iformatprovider/), которая предоставляет сведения о форматировании, зависящие от культуры. |

### Возвращаемое значение

8-битное целое со знаком, эквивалентное значению этого экземпляра.

## IConvertible::ToDecimal(System::SharedPtr\<System::IFormatProvider\>) метод

Преобразует значение этого экземпляра в эквивалентное [System::Decimal](../../decimal/) число, используя указанные сведения о формате, зависящие от культуры.

```cpp
virtual System::Decimal System::IConvertible::ToDecimal(System::SharedPtr<System::IFormatProvider> provider)=0
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| provider | [System::SharedPtr](../../sharedptr/)\<[System::IFormatProvider](../../iformatprovider/)\> | Реализация интерфейса [System::IFormatProvider](../../iformatprovider/), которая предоставляет сведения о формате, зависящие от культуры. |

### Возвращаемое значение

Число [System::Decimal](../../decimal/), эквивалентное значению этого экземпляра.

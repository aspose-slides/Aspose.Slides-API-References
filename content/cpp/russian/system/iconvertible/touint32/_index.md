## IConvertible::ToUInt32(System::SharedPtr\<System::IFormatProvider\>) метод

Преобразует значение данного экземпляра в эквивалентное 32-битное uint32_teger, используя указанные сведения о форматировании, специфичные для культуры.

```cpp
virtual uint32_t System::IConvertible::ToUInt32(System::SharedPtr<System::IFormatProvider> provider)=0
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| provider | [System::SharedPtr](../../sharedptr/)\<[System::IFormatProvider](../../iformatprovider/)\> | Реализация интерфейса [System::IFormatProvider](../../iformatprovider/), предоставляющая информацию о форматировании, специфичную для культуры. |

### Возвращаемое значение

32-битное беззнаковое целое значение, эквивалентное значению данного экземпляра.

## IConvertible::ToUInt16(System::SharedPtr\<System::IFormatProvider\>) method

Преобразует значение этого экземпляра в эквивалентное 16-битное uint32_teger, используя указанную культурно-специфичную информацию о форматировании.

```cpp
virtual uint16_t System::IConvertible::ToUInt16(System::SharedPtr<System::IFormatProvider> provider)=0
```

### Arguments

| Параметр | Тип | Описание |
| --- | --- | --- |
| provider | [System::SharedPtr](../../sharedptr/)\<[System::IFormatProvider](../../iformatprovider/)\> | [System::IFormatProvider](../../iformatprovider/) интерфейсная реализация, предоставляющая информацию о форматировании, специфичную для культуры. |

### Return Value

16-битное беззнаковое целочисленное значение, эквивалентное значению этого экземпляра.

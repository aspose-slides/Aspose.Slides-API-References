## IConvertible::ToInt64(System::SharedPtr\<System::IFormatProvider\>) метод


Преобразует значение данного экземпляра в эквивалентное 64-битное знаковое целое число, используя указанные сведения о форматировании, специфичные для культуры.

```cpp
virtual int64_t System::IConvertible::ToInt64(System::SharedPtr<System::IFormatProvider> provider)=0
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| provider | [System::SharedPtr](../../sharedptr/)\<[System::IFormatProvider](../../iformatprovider/)\> | Реализация [System::IFormatProvider](../../iformatprovider/) интерфейса, предоставляющая сведения о форматировании, специфичные для культуры. |

### Возвращаемое значение

64-битное знаковое целое, эквивалентное значению данного экземпляра.

## IConvertible::ToUInt64(System::SharedPtr\<System::IFormatProvider\>) метод


Преобразует значение этого экземпляра в эквивалентное 64-битное uint32_teger, используя указанные сведения о форматировании, зависящие от культуры.

```cpp
virtual uint64_t System::IConvertible::ToUInt64(System::SharedPtr<System::IFormatProvider> provider)=0
```


### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| provider | [System::SharedPtr](../../sharedptr/)\<[System::IFormatProvider](../../iformatprovider/)\> | Объект реализации интерфейса [System::IFormatProvider](../../iformatprovider/), который предоставляет сведения о форматировании, зависящие от культуры. |

### Возвращаемое значение

64-битное беззнаковое целое значение, эквивалентное значению этого экземпляра.

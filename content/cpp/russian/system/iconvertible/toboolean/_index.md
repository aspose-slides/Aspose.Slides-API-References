## IConvertible::ToBoolean(System::SharedPtr\<System::IFormatProvider\>) метод

Преобразует значение этого экземпляра в эквивалентное значение [Boolean](../../boolean/) с использованием указанной информации о форматировании, специфичной для культуры.

```cpp
virtual bool System::IConvertible::ToBoolean(System::SharedPtr<System::IFormatProvider> provider)=0
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| provider | [System::SharedPtr](../../sharedptr/)\<[System::IFormatProvider](../../iformatprovider/)\> | Реализация интерфейса [System::IFormatProvider](../../iformatprovider/), предоставляющая информацию о форматировании, специфичную для культуры. |

### Возвращаемое значение

Значение типа [Boolean](../../boolean/), эквивалентное значению этого экземпляра.

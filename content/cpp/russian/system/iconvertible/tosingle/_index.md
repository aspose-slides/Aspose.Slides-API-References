## IConvertible::ToSingle(System::SharedPtr\<System::IFormatProvider\>) метод

Преобразует значение этого экземпляра в эквивалентное число с одинарной точностью с плавающей точкой, используя указанные сведения о форматировании, зависящие от культуры.

```cpp
virtual float System::IConvertible::ToSingle(System::SharedPtr<System::IFormatProvider> provider)=0
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| provider | [System::SharedPtr](../../sharedptr/)\<[System::IFormatProvider](../../iformatprovider/)\> | Реализация интерфейса [System::IFormatProvider](../../iformatprovider/), которая поставляет сведения о форматировании, зависящие от культуры. |

### Возвращаемое значение

Число с одинарной точностью с плавающей точкой, эквивалентное значению этого экземпляра.

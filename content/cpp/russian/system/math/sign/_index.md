## Math::Sign(T) метод


Определяет знак указанного знакового целочисленного значения.

```cpp
template<typename T> static std::enable_if<std::is_integral<T>::value &&!std::is_unsigned<T>::value, int>::type System::Math::Sign(T value)
```


### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| T | Знаковый целочисленный тип |

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | T | Значение, знак которого определяется |

### Возвращаемое значение

- 1, если **value** меньше 0; 0, если **value** равно 0; 1, если **value** больше 0

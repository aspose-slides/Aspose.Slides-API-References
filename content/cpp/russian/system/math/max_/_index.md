## Math::Max_(float, float) метод


Возвращает наибольшее значение с одинарной точностью из двух указанных.

```cpp
float System::Math::Max_(float val1, float val2)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| val1 | **float** | Первый сравниваемый |
| val2 | **float** | Второй сравниваемый |

### Возвращаемое значение

Наибольшее значение из **val1** и **val2**; либо NaN, если один из операндов является NaN

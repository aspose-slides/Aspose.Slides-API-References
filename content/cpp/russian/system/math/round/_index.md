## Math::Round(double) метод


Округляет заданное значение до ближайшего целого числа.

```cpp
static double System::Math::Round(double a)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| a | **double** | Значение для округления |

### Возвращаемое значение

**a** округленное до ближайшего целого числа

## Math::Round(double, int) метод


Округляет заданное значение до ближайшего значения с указанным числом знаков после запятой.

```cpp
static double System::Math::Round(double value, int digits)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | **double** | Значение для округления |
| digits | int | Количество знаков после запятой в округленном значении |

### Возвращаемое значение

Число с указанным числом знаков, ближайшее к **value**

## Math::Round(double, MidpointRounding) метод


Округляет заданное значение до ближайшего целого числа. Параметр указывает поведение функции, если заданное значение находится на одинаковом расстоянии от двух ближайших чисел.

```cpp
static double System::Math::Round(double value, MidpointRounding mode)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | **double** | Значение для округления |
| mode | [MidpointRounding](../../midpointrounding/) | Указывает, как выполнять округление, если **value** находится на одинаковом расстоянии от двух ближайших чисел. |

### Возвращаемое значение

**value** округленное до ближайшего целого числа

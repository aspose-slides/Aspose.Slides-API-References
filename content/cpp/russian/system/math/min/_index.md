## Math::Min(T0, T1) метод

Возвращает наименьшее значение из двух указанных числовых значений.

```cpp
template<class T0,class T1,class> static auto System::Math::Min(T0 val1, T1 val2) -> decltype(val1+val2)
```

### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| T0 | Type of first value. |
| T1 | Type of second value. |

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| val1 | T0 | First value. |
| val2 | T1 | Second value. |

### Возвращаемое значение

Наименьшее значение из **val1** и **val2**.

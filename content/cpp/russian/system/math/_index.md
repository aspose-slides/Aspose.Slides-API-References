## Примечания



```cpp
#include "system/math.h"
#include <iostream>

int main()
{
  using namespace System;

  // Вывести абсолютные значения.
  for (int i = -1; i < 2; ++i)
  {
    std::cout << Math::Abs(i) << " ";
  }
  std::cout << std::endl;

  // Вывести синус PI/2 и косинус PI.
  std::cout << "sin(PI/2)=" << Math::Sin(Math::PI/2) << "; cos(PI)=" << Math::Cos(Math::PI) << std::endl;

  return 0;
}
/*
Этот пример кода выводит следующий результат:
1 0 1
sin(PI/2)=1; cos(PI)=-1
*/
```

## Math::Pow(double, double) метод

Возвращает указанное значение, возведённое в указанную степень.

```cpp
static double System::Math::Pow(double x, double y)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| x | **double** | Значение, которое нужно возвести в указанную степень |
| y | **double** | Степень, в которую нужно возвести указанное значение |

### Возвращаемое значение

**x**, возведённое в степень **y**

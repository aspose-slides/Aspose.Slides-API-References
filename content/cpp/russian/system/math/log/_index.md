## Math::Log(double) метод

Возвращает натуральный логарифм указанного значения.

```cpp
static double System::Math::Log(double d)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| d | **double** | Значение двойной точности с плавающей запятой |

### Возвращаемое значение

Натуральный логарифм **d**

## Math::Log(double, double) метод

Возвращает логарифм указанного значения в указанном основании.

```cpp
static double System::Math::Log(double a, double newBase)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| a | **double** | Значение двойной точности с плавающей запятой |
| newBase | **double** | Основание логарифма |

### Возвращаемое значение

Логарифм **a** в основании **newBase**

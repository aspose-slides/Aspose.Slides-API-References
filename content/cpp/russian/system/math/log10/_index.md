## Math::Log10(double) метод

Возвращает логарифм по основанию 10 указанного значения.

```cpp
static double System::Math::Log10(double d)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| d | **double** | Значение с плавающей запятой двойной точности |

### Возвращаемое значение

Логарифм по основанию 10 **d**

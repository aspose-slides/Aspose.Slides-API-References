## Math::Min_(float, float) метод


Возвращает наименьшее значение одинарной точности из двух указанных.

```cpp
float System::Math::Min_(float val1, float val2)
```


### Arguments

| Параметр | Тип | Описание |
| --- | --- | --- |
| val1 | **float** | Первый сравниваемый |
| val2 | **float** | Второй сравниваемый |

### Return Value

Наименьшее значение из **val1** и **val2**; или NaN, если один из операндов является NaN

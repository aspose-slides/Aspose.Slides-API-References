## Math::DivRem(int, int, int\&) метод

Вычисляет частное двух 32-битных целых чисел и остаток.

```cpp
static int System::Math::DivRem(int a, int b, int &result)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| a | int | Делимое |
| b | int | Делитель |
| result | int\& | Параметр-вывод, содержащий остаток от деления **a** на **b** |

### Возвращаемое значение

Квотирование **a** и **b**

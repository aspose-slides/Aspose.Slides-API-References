## Math::Max(T0, T1) метод


Возвращает наибольшее значение из двух указанных числовых.

```cpp
template<class T0,class T1,class> static auto System::Math::Max(T0 val1, T1 val2) -> decltype(val1+val2)
```


### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| T0 | Тип первого значения. |
| T1 | Тип второго значения. |

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| val1 | T0 | Первое значение. |
| val2 | T1 | Второе значение. |

### Возвращаемое значение

Наибольшее значение из **val1** и **val2**.

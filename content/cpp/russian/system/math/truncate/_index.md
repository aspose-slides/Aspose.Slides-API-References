## Math::Truncate(const Decimal\&) метод


Возвращает [Decimal](../../decimal/) объект, представляющий значение, у которого целая часть равна целой части значения, представленного указанным [Decimal](../../decimal/) объектом, при этом все дробные цифры отбрасываются.

```cpp
static Decimal System::Math::Truncate(const Decimal &d)
```


### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| d | const [Decimal](../../decimal/)\& | Десятичное число |

### Возвращаемое значение

[Decimal](../../decimal/) объект, представляющий десятичное значение, у которого целая часть равна целой части указанного значения, а все дробные цифры равны 0.

## Math::Truncate(double) метод


Возвращает значение типа double-precision floating point, у которого целая часть равна целой части указанного значения, при этом все дробные цифры отбрасываются.

```cpp
static double System::Math::Truncate(double d)
```


### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| d | **double** | Десятичное число |

### Возвращаемое значение

Значение типа double-precision floating point, у которого целая часть равна целой части указанного значения, а все дробные цифры равны 0.

## Math::Abs(T) метод


Возвращает абсолютное значение указанного значения.

```cpp
template<class T> static T System::Math::Abs(T value)
```


### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| T | Тип значения, принимаемого методом в качестве аргумента |

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | T | Значение арифметического типа |

### Возвращаемое значение

Абсолютное значение **value**

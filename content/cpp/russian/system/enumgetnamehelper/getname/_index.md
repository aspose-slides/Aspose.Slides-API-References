## EnumGetNameHelper::GetName(T) метод


Возвращает строковое название указанного значения константы перечисления.

```cpp
template<class T> static String System::EnumGetNameHelper<E, G, Guard>::GetName(T value)
```


### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| T | Тип значений перечисления, принимаемых методом; **T** должен быть преобразуем к **E** |

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | T | Значение константы перечисления, строковое название которого должно быть получено |

### Возвращаемое значение

Строковое название указанного значения константы перечисления

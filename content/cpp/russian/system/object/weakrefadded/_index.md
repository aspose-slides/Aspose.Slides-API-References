## Object::WeakRefAdded() метод


Увеличивает счётчик слабых ссылок. Не следует вызывать напрямую; вместо этого используйте умные указатели или ThisProtector.

```cpp
Detail::SmartPtrCounter * System::Object::WeakRefAdded()
```


### Возвращаемое значение

Указатель на структуру данных подсчёта ссылок, связанную с объектом.

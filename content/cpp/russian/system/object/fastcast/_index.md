## Object::FastCast(const Details::FastRttiBase\&, void **) const метод

Только для внутреннего использования.

```cpp
virtual bool System::Object::FastCast(const Details::FastRttiBase &helper, void **out_ptr) const
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| helper | const Details::FastRttiBase\& | FastRttiBase структура, специализированная с необходимым типом. |
| out_ptr | void ** | Указатель на требуемый тип, если преобразование доступно. |

### Возвращаемое значение

True, если класс поддерживает операции FastRTTI.

## Object::Is(const TypeInfo\&) const метод

Проверьте, представляет ли объект экземпляр типа, описанный targetType. Аналог оператора C# 'is'.

```cpp
virtual bool System::Object::Is(const TypeInfo &targetType) const
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| targetType | const [TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) структура, описывающая тип, против которого проверяется текущий объект. |

### Возвращаемое значение

True, если объект имеет помеченный тип или его подкласс, иначе false.

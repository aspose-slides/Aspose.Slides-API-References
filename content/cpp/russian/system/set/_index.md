## System::Set(ExpressionT\&, const ExpressionT\&) функция


Implements 'var' pattern translation.

```cpp
template<class ExpressionT> bool System::Set(ExpressionT &var, const ExpressionT &value)
```


### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| ExpressionT | тип переменной. |

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| var | ExpressionT\& | ссылка на переменную, которую необходимо инициализировать. |
| value | const ExpressionT\& | значение, которое будет присвоено переменной. |

### Возвращаемое значение

всегда true (этот шаблон всегда срабатывает).

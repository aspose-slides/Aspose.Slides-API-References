## Delegate< ReturnType(ArgumentTypes...)>::operator==(const Delegate\&) const метод


Сравнивает два объекта делегата, чтобы проверить, указывают ли они на одну и ту же сущность.

```cpp
bool System::Delegate<ReturnType(ArgumentTypes...)>::operator==(const Delegate &f) const
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| f | const [Delegate](../delegate/)\& | Объект делегата, с которым сравнивается текущий объект делегата |

### Возвращаемое значение

True, если оба делегата указывают на одну и ту же сущность, иначе - false

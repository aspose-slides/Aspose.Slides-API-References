## NullableUtils класс

Представляет статический класс C# [System.Nullable](../nullable/) (без аргументов типа). Невозможно использовать оригинальное имя из-за невозможности перегрузки шаблонов классов в C++. Поддерживает тип значения, которому можно присвоить null. Этот класс не может быть унаследован.

```cpp
class NullableUtils
```

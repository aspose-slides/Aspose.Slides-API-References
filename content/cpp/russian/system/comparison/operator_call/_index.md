## Comparison::operator()(T, T) метод

Вызывает вызываемый объект, на который указывает текущий объект.

```cpp
bool System::Comparison<T>::operator()(T a, T b)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| a | T | Первый сравниваемый |
| b | T | Второй сравниваемый |

### Возвращаемое значение

True, если **a** меньше **b**, иначе - true

```cpp
bool System::Details_UnauthorizedAccessException::Is(const System::TypeInfo &target) const override
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) структура, описывающая тип, против которого проверяется текущий объект. |

### Возвращаемое значение

True, если объект имеет указанный тип или его подкласс, иначе false.

## Примечания

Проверьте, представляет ли объект экземпляр типа, описанного targetType. Аналог оператора C# 'is'.

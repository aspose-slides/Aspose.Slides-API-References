## IAsyncResult::get_CompletedSynchronously() метод

Возвращает значение, указывающее, завершилась ли асинхронная операция синхронно.

```cpp
virtual bool System::IAsyncResult::get_CompletedSynchronously()=0
```

### Возвращаемое значение

True если асинхронная операция завершилась синхронно, иначе - false

## SmartPtrMode enum

[SmartPtr](../smartptr/) тип указателя: weak или shared. Определяет, учитывается ли указатель при принятии решения, удалять объект или нет.

```cpp
enum class SmartPtrMode : char
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Shared | 0 | Shared mode: указатель участвует в подсчёте ссылок. |
| Weak | 1 | Weak mode: указатель не участвует в подсчёте ссылок. |

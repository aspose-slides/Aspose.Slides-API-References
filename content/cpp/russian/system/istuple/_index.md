## System::IsTuple(const SharedPtr\<Object\>\&, int32_t) function

Проверяет, является ли объект кортежем (реализует интерфейс ITuple). Используется в реализации позиционных шаблонов.

```cpp
bool System::IsTuple(const SharedPtr<Object> &object, int32_t length)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| object | const [SharedPtr](../sharedptr/)\<[Object](../object/)\>\& | объект для проверки. |
| length | **int32_t** | ожидаемая длина кортежа |

### Возвращаемое значение

является ли объект кортежем и его длина равна ожидаемой

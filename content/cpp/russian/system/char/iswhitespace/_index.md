## Char::IsWhiteSpace(const char_t *, int) метод

Определяет, классифицируется ли символ по указанному индексу в заданном буфере символов как символ пробела.

```cpp
static bool System::Char::IsWhiteSpace(const char_t *str, int idx)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| str | const char_t * | Указатель на начало буфера символов |
| idx | int | Нулевая позиция в указанном буфере символа, который нужно проверить |

### Возвращаемое значение

True, если символ по указанному индексу является символом пробела, иначе — false

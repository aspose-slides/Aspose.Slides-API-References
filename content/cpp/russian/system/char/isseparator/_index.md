## Char::IsSeparator(const char_t *, int) method

Определяет, классифицируется ли символ по указанному индексу в заданном буфере символов как символ-разделитель.

```cpp
static bool System::Char::IsSeparator(const char_t *str, int idx)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| str | const char_t * | Указатель на начало буфера символов |
| idx | int | Нулевой индекс в указанном буфере символа для проверки |

### Возвращаемое значение

True если символ по указанному индексу является символом-разделителем, иначе - false

static bool System::Char::IsSeparator(char_t c)

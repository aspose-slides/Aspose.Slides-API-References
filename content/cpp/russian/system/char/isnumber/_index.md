## Char::IsNumber(const char_t *, int) метод

Определяет, классифицируется ли символ по указанному индексу в заданном буфере символов как число.

```cpp
static bool System::Char::IsNumber(const char_t *str, int idx)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| str | const char_t * | Указатель на начало буфера символов |
| idx | int | Нулевой индекс в заданном буфере символа, который нужно проверить |

### Возвращаемое значение

True если символ по указанному индексу является числом, иначе — false

## Char::IsNumber(char_t) метод

Определяет, классифицируется ли указанный символ как число.

```cpp
static bool System::Char::IsNumber(char_t c)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| c | char_t | Символ, который нужно проверить |

### Возвращаемое значение

True если указанный символ является числом, иначе — false

## Char::ToLowerInvariant(char_t) метод

Преобразует указанный символ к нижнему регистру.

```cpp
static char_t System::Char::ToLowerInvariant(char_t c)
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| c | char_t | Символ для преобразования |

### Возвращаемое значение

Указанный символ в нижнем регистре, если указанный символ является буквой верхнего регистра, в противном случае — указанный символ

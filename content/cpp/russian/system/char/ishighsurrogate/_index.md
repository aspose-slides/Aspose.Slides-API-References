## Char::IsHighSurrogate(const String\&, int) метод

Определяет, является ли символ в указанном индексе в заданной строке кодовой единицей UTF-16 высокого суррогата.

```cpp
static bool System::Char::IsHighSurrogate(const String &s, int index)
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| s | const [String](../../string/)\& | Строка |
| index | int | Индекс в указанной строке символа, который проверяется |

### Возвращаемое значение

True, если символ в указанном индексе является кодовой единицей UTF-16 высокого суррогата, иначе — false

## Char::IsHighSurrogate(const char_t *, int) метод

Определяет, является ли символ в указанном индексе в заданном буфере символов высоким суррогатом.

```cpp
static bool System::Char::IsHighSurrogate(const char_t *str, int idx)
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| str | const char_t * | Указатель на начало буфера символов |
| idx | int | Нулевой индекс в указанном буфере символа для проверки |

### Возвращаемое значение

True, если символ в указанном индексе является высоким суррогатом, иначе — false

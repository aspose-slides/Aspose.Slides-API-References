## Char::IsUpper(const String\&, int) метод

Определяет, является ли символ в указанном индексе заданной строки символом верхнего регистра.

```cpp
static bool System::Char::IsUpper(const String &str, int idx)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| str | const [String](../../string/)\& | Строка, содержащая char. |
| idx | int | Нулевой индекс в указанной string для проверки |

### Возвращаемое значение

True, если символ в указанном индексе является символом верхнего регистра, иначе — false

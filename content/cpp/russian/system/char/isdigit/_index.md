## Char::IsDigit(const char_t *, int) метод


Определяет, является ли символ по указанному индексу в заданном буфере символов десятичной цифрой.

```cpp
static bool System::Char::IsDigit(const char_t *str, int idx)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| str | const char_t * | Указатель на начало буфера символов |
| idx | int | Нулевой индекс в указанном буфере символа для проверки |

### Возвращаемое значение

True, если символ по указанному индексу является десятичной цифрой, иначе — false

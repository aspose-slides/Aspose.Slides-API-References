## Char::IsLetter(const char_t *, int) метод

Определяет, классифицируется ли символ по указанному индексу в заданном буфере символов как Unicode-буква.

```cpp
static bool System::Char::IsLetter(const char_t *str, int idx)
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| str | const char_t * | Указатель на начало буфера символов |
| idx | int | Ноль-базовый индекс в указанном буфере символа, который необходимо проверить |

### Возвращаемое значение

True если символ по указанному индексу является Unicode-буквой, иначе — false

## Char::IsLetter(char_t) метод

Определяет, классифицируется ли указанный символ как Unicode-буква.

```cpp
static bool System::Char::IsLetter(char_t c)
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| c | char_t | Символ для проверки |

### Возвращаемое значение

True если указанный символ является Unicode-буквой, иначе — false

## Char::IsSymbol(const char_t *, int) метод


Определяет, является ли символ по заданному индексу в указанном буфере символов символом-символом.

```cpp
static bool System::Char::IsSymbol(const char_t *str, int idx)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| str | const char_t * | Указатель на начало буфера символов |
| idx | int | Нулевой индекс в указанном буфере символа для проверки |

### Возвращаемое значение

True если символ по указанному индексу является символом-символом, иначе - false

## Char::IsSymbol(char_t) метод


Определяет, классифицируется ли указанный символ как символ-символ.

```cpp
static bool System::Char::IsSymbol(char_t c)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| c | char_t | Тестируемый символ |

### Возвращаемое значение

True если указанный символ является символом-символом, иначе - false

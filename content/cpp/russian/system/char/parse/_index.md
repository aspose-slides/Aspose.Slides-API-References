## Char::Parse(const String\&) метод

Преобразует первый и единственный символ указанной строки в значение типа char_t.

```cpp
static char_t System::Char::Parse(const String &value)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | const [String](../../string/)\& | Строка для преобразования. |

### Возвращаемое значение

Первый и единственный символ указанной строки, если она имеет ровно 1 символ.

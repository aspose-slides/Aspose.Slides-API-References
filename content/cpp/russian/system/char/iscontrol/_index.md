## Char::IsControl(const char_t *, int) метод


Определяет, является ли символ по указанному индексу в заданном буфере символов Unicode-символом управления.

```cpp
static bool System::Char::IsControl(const char_t *str, int idx)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| str | const char_t * | Указатель на начало буфера символов |
| idx | int | Нулевая позиция индекса в указанном буфере символа для проверки |

### Возвращаемое значение

True если символ по указанному индексу является Unicode-символом управления, иначе — false

## Char::IsControl(char_t) метод

static bool System::Char::IsControl(char_t c)

### Аргументы

| Параметр | Тип | Описание |

### Возвращаемое значение

True если указанный символ является Unicode-символом управления, иначе — false

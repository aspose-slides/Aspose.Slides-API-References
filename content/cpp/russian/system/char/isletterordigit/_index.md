## Char::IsLetterOrDigit(const char_t *, int) метод

Определяет, является ли символ по указанному индексу в заданном буфере символов буквой Unicode или десятичной цифрой.

```cpp
static bool System::Char::IsLetterOrDigit(const char_t *str, int idx)
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| str | const char_t * | Указатель на начало буфера символов |
| idx | int | Нулевой индекс в указанном буфере символа, который нужно проверить |

### Возвращаемое значение

true, если символ по указанному индексу является буквой Unicode или десятичной цифрой, в противном случае - false

Определяет, является ли указанный символ буквой Unicode или десятичной цифрой.

static bool System::Char::IsLetterOrDigit(char_t c)

### Параметры

| Параметр | Тип | Описание |

| c | char_t | Символ для проверки |

### Возвращаемое значение

true, если указанный символ является буквой Unicode или десятичной цифрой, в противном случае - false

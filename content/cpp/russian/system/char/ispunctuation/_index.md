## Char::IsPunctuation(const char_t *, int) метод

Определяет, является ли символ в указанном индексе в заданном буфере символов классифицированным как знак препинания.

```cpp
static bool System::Char::IsPunctuation(const char_t *str, int idx)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| str | const char_t * | Указатель на начало буфера символов |
| idx | int | Нулевой индекс в указанном буфере символа для проверки |

### Возвращаемое значение

True, если символ в указанном индексе является знаком препинания, иначе — false

### Возвращаемое значение

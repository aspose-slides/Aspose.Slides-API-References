## Char::GetUnicodeCategory(char_t) метод

Возвращает значение, представляющее категорию Unicode указанного символа.

```cpp
static Globalization::UnicodeCategory System::Char::GetUnicodeCategory(char_t ch)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| ch | char_t | Символ для классификации |

### Возвращаемое значение

Одно из значений перечисления UnicodeCategory, представляющее категорию Unicode, к которой относится указанный символ.

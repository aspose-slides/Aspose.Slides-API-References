## Char::IsAsciiWhiteSpace(char_t) метод

Определяет, является ли указанный символ классифицированным как символ ASCII-пробела.

```cpp
static constexpr bool System::Char::IsAsciiWhiteSpace(char_t c)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| c | char_t | Символ для тестирования |

### Возвращаемое значение

True, если указанный символ является символом ASCII-пробела, иначе — false

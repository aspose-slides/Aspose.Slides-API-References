## Char::IsLower(const char_t *, int) метод


Определяет, классифицируется ли символ по указанному индексу в заданном буфере символов как строчная буква.

```cpp
static bool System::Char::IsLower(const char_t *str, int idx)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| str | const char_t * | Указатель на начало буфера символов |
| idx | int | Индекс, начинающийся с нуля, в указанном буфере символа для проверки |

### Возвращаемое значение

True если символ по указанному индексу является строчной буквой, иначе — false

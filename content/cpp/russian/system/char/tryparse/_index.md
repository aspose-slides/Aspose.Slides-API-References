## Char::TryParse(const System::String\&, char_t\&) метод

Пытается преобразовать строку, состоящую из одного символа, в символ UTF-16. Функция успешно завершается только когда входная строка не null и имеет длину ровно один символ.

```cpp
static bool System::Char::TryParse(const System::String &s, char_t &result)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| s | const [System::String](../../string/)\& | [String](../../string/) для преобразования |
| result | char_t\& | Выходная переменная, содержащая результат преобразования, если преобразование успешно |

## Возвращаемое значение

True, если преобразование удалось, иначе - false

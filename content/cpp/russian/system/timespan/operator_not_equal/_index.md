## TimeSpan::operator!=(TimeSpan) const метод

Определяет, не равен ли интервал времени, представленный текущим объектом, интервалу времени, представленного указанным объектом.

```cpp
constexpr bool System::TimeSpan::operator!=(TimeSpan value) const
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | [TimeSpan](../) | Объект [TimeSpan](../) для сравнения с текущим объектом |

### Возвращаемое значение

True если текущий объект и указанный объект представляют разные интервалы времени, иначе - false

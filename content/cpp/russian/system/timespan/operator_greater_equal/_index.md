## TimeSpan::operator>=(TimeSpan) const method

Определяет, является ли интервал времени, представленный текущим объектом, длиннее или равен интервалу времени, представляемому указанным объектом.

```cpp
constexpr bool System::TimeSpan::operator>=(TimeSpan value) const
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | [TimeSpan](../) | Объект [TimeSpan](../), с которым сравнивается текущий объект |

### Возвращаемое значение

True если интервал времени, представленный текущим объектом, длиннее или равен интервалу времени, представляемому **value**, иначе — false

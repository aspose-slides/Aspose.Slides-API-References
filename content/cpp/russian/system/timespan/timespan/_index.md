## TimeSpan::TimeSpan() конструктор

Создаёт объект [TimeSpan](../), представляющий нулевой интервал времени.

```cpp
constexpr System::TimeSpan::TimeSpan()
```

## TimeSpan::TimeSpan(int64_t) конструктор

Создаёт экземпляр класса [TimeSpan](../), представляющий указанный интервал времени.

```cpp
constexpr System::TimeSpan::TimeSpan(int64_t ticks)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| ticks | **int64_t** | Интервал времени, который будет представлять создаваемый экземпляр, выраженный числом интервалов по 100 наносекунд. |

## TimeSpan::TimeSpan(int, int, int) конструктор

Создаёт экземпляр класса [TimeSpan](../), представляющий интервал времени, равный сумме указанного количества часов, минут и секунд.

```cpp
System::TimeSpan::TimeSpan(int hours, int minutes, int seconds)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| hours | int | Количество часов в компоненте часов интервала времени, который будет представлять создаваемый экземпляр |
| minutes | int | Количество минут в компоненте минут интервала времени, который будет представлять создаваемый экземпляр |
| seconds | int | Количество секунд в компоненте секунд интервала времени, который будет представлять создаваемый экземпляр |

## TimeSpan::TimeSpan(int, int, int, int, int) конструктор

Создаёт экземпляр класса [TimeSpan](../), представляющий интервал времени, равный сумме указанного количества дней, часов, минут, секунд и миллисекунд.

```cpp
System::TimeSpan::TimeSpan(int days, int hours, int minutes, int seconds, int milliseconds=0)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| days | int | Количество дней в компоненте дней интервала времени, который будет представлять создаваемый экземпляр |
| hours | int | Количество часов в компоненте часов интервала времени, который будет представлять создаваемый экземпляр |
| minutes | int | Количество минут в компоненте минут интервала времени, который будет представлять создаваемый экземпляр |
| seconds | int | Количество секунд в компоненте секунд интервала времени, который будет представлять создаваемый экземпляр |
| milliseconds | int | Количество миллисекунд в компоненте миллисекунд интервала времени, который будет представлять создаваемый экземпляр |

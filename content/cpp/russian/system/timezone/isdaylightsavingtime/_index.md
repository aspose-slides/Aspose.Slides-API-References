## TimeZone::IsDaylightSavingTime(DateTime) метод


Определяет, попадает ли значение даты и времени, представляемое указанным объектом [DateTime](../../datetime/), в диапазон летнего времени для часового пояса, представленного текущим объектом [TimeZone](../).

```cpp
virtual bool System::TimeZone::IsDaylightSavingTime(DateTime time)=0
```


### Возвращаемое значение

True если значение даты и времени, представляемое указанным объектом [DateTime](../../datetime/), попадает в диапазон летнего времени часового пояса, представленного текущим объектом [TimeZone](../), иначе - false

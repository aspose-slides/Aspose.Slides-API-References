## IComparable класс

Определяет метод, который сравнивает два объекта. Объекты этого класса должны выделяться только с помощью функции [System::MakeObject()](../makeobject/). Никогда не создавайте экземпляр этого типа в стеке или с помощью operator new, так как это приведёт к ошибкам во время выполнения и/или нарушениям утверждений. Всегда оберните этот класс в указатель [System::SmartPtr](../smartptr/) и используйте этот указатель для передачи его в функции в качестве аргумента.

```cpp
template<typename T>class IComparable : public virtual System::Object
```

### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| T | Тип объектов, с которыми сравнивается текущий объект |

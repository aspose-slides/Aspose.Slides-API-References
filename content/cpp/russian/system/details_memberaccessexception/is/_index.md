## Подробности_MemberAccessException::Is(const System::TypeInfo\&) const method




```cpp
bool System::Details_MemberAccessException::Is(const System::TypeInfo &target) const override
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) структура, описывающая тип, с которым сравнивается текущий объект. |

### Возвращаемое значение

True если объект является объектом указанного типа или его подклассом, false в противном случае.
## Примечания

Проверьте, представляет ли объект экземпляр типа, описанного targetType. Аналог оператора C# 'is'.

## Details_MethodAccessException::Is(const System::TypeInfo\&) const метод

```cpp
bool System::Details_MethodAccessException::Is(const System::TypeInfo &target) const override
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) структура, описывающая тип, против которого проверяется текущий объект. |

### Возвращаемое значение

True, если объект имеет помеченный тип или его подкласс, false в противном случае.

## Примечания

Проверяет, представляет ли объект экземпляр типа, описанного targetType. Аналог оператора C# 'is'.

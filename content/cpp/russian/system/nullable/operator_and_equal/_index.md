## Nullable::operator&=(bool) метод

Применяет [operator&=()](./) к значению, представленному текущим объектом, используя указанное значение в качестве аргумента справа.

```cpp
template<typename T1> std::enable_if<std::is_same<T1, bool>::value, Nullable<T>>::type System::Nullable<T>::operator&=(bool other)
```

### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| T1 | Параметр шаблона, позволяющий выполнить SFINAE. |

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| other | **bool** | Булево значение, которое используется в качестве правостороннего значения [operator&=()](./), примененного к значению, представленному текущим объектом. |

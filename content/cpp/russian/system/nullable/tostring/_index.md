## Nullable::ToString() const метод

Преобразует значение, представленное текущим объектом, в строку.

```cpp
String System::Nullable<T>::ToString() const
```

### Возвращаемое значение

Строковое представление значения, представленного текущим объектом, или пустая строка, если значение, представленное текущим объектом, равно null

## Nullable::GetValueOrDefault(T) метод

Возвращает значение, представляемое текущим объектом, или указанное значение, если значение, представляемое текущим объектом, равно null.

```cpp
T System::Nullable<T>::GetValueOrDefault(T default_value)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| default_value | T | Значение, возвращаемое методом, если значение, представляемое текущим объектом, равно null |

### Возвращаемое значение

Копия значения, представляемого текущим объектом, если оно не равно null, в противном случае возвращается указанное значение

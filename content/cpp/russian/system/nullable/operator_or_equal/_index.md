## Nullable::operator|=(bool) метод

Применяет [operator|=()](./) к значению, представленному текущим объектом, используя указанное значение в качестве аргумента-правой стороны.

```cpp
template<typename T1> std::enable_if<std::is_same<T1, bool>::value, Nullable<T>>::type System::Nullable<T>::operator|=(bool other)
```

### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| T1 | Параметр шаблона, необходимый для работы SFINAE. |

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| other | **bool** | Булево значение, которое используется в качестве правой части [operator|=()](./), применяемой к значению, представленному текущим объектом. |

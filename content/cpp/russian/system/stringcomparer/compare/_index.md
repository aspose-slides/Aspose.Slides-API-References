## StringComparer::Compare(args_type, args_type) const метод

Сравнивает две строки, используя текущие настройки.

```cpp
virtual int System::StringComparer::Compare(args_type x, args_type y) const override
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| x | [args_type](../args_type/) | Значение строки слева. |
| y | [args_type](../args_type/) | Значение строки справа. |

### Возвращаемое значение

Отрицательное значение, если **x** предшествует **y**, ноль, если они равны, и положительное значение в остальных случаях.

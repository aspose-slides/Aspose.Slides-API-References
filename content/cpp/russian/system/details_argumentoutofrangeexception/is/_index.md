## Details_ArgumentOutOfRangeException::Is(const System::TypeInfo\&) const метод

```cpp
bool System::Details_ArgumentOutOfRangeException::Is(const System::TypeInfo &target) const override
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) структура, описывающая тип, с которым проверяется текущий объект. |

### Возвращаемое значение

True если объект имеет указанный тип или его подкласс, false в противном случае.
## Примечания

Проверяет, представляет ли объект экземпляр типа, описанного targetType. Аналог оператора C# 'is' operator.

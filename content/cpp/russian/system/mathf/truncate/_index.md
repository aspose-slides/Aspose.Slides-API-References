## MathF::Truncate(float) метод

Возвращает значение с плавающей точкой одинарной точности, у которого целая часть равна целой части указанного значения, а все дробные знаки отбрасываются.

```cpp
static float System::MathF::Truncate(float x)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| x | **float** | Десятичное число |

### Возвращаемое значение

Значение с плавающей точкой одинарной точности, у которого целая часть равна целой части указанного значения, а все дробные знаки равны 0.

## MathF::Floor(float) метод


Возвращает наибольшее целое значение, которое меньше или равно указанному значению.

```cpp
static float System::MathF::Floor(float x)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| x | **float** | Число с плавающей точкой одинарной точности |

### Возвращаемое значение

Наибольшее целое значение, которое меньше или равно **d**

## MathF::Pow(float, float) метод

Возвращает указанное значение, возведённое в указанную степень.

```cpp
static float System::MathF::Pow(float x, float y)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| x | **float** | Значение, которое нужно возвести в указанную степень |
| y | **float** | Степень, в которую нужно возвести указанное значение |

### Возвращаемое значение

**x** возведён в степень **y**

## MathF::Round(float) метод


Округляет указанное значение до ближайшего целого числа.

```cpp
static float System::MathF::Round(float a)
```


### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| a | **float** | Значение для округления |

### Возвращаемое значение

**a** округлено до ближайшего целого значения

## MathF::Round(float, int) метод


Округляет указанное значение до ближайшего значения с заданным числом знаков после запятой.

```cpp
static float System::MathF::Round(float value, int digits)
```


### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | **float** | Значение для округления |
| digits | int | Количество знаков после запятой в округленном значении |

### Возвращаемое значение

Число с указанным числом знаков, ближайшее к **value**

## MathF::Round(float, MidpointRounding) метод


Округляет указанное значение до ближайшего целого числа. Параметр указывает поведение функции, если указанное значение одинаково близко к двум ближайшим числам.

```cpp
static float System::MathF::Round(float value, MidpointRounding mode)
```


### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | **float** | Значение для округления |
| mode | [MidpointRounding](../../midpointrounding/) | Указывает, как выполнять округление, если **value** одинаково близко к двум ближайшим числам. |

### Возвращаемое значение

**value** округлено до ближайшего целого значения

## MathF::Log(float) метод

Возвращает натуральный логарифм указанного значения.

```cpp
static float System::MathF::Log(float x)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| x | **float** | Значение с плавающей точкой одинарной точности |

### Возвращаемое значение

Натуральный логарифм **d**

## MathF::Log(float, float) метод

Возвращает логарифм указанного значения в указанном основании.

```cpp
static float System::MathF::Log(float a, float newBase)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| a | **float** | Значение с плавающей точкой одинарной точности |
| newBase | **float** | Основание логарифма |

### Возвращаемое значение

Логарифм **a** в основании **newBase**

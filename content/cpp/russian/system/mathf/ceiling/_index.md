## MathF::Ceiling(float) метод

Возвращает наименьшее целочисленное значение, которое больше или равно указанному значению.

```cpp
static float System::MathF::Ceiling(float a)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| a | **float** | Число с плавающей точкой одинарной точности |

### Возвращаемое значение

Наименьшее целочисленное значение, которое больше или равно **d**

## DateTimeOffset::Compare(const DateTimeOffset\&, const DateTimeOffset\&) метод

Сравнивает два [DateTimeOffset](../) объекта.

```cpp
static int System::DateTimeOffset::Compare(const DateTimeOffset &first, const DateTimeOffset &second)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| first | const [DateTimeOffset](../)\& | Первый объект для сравнения. |
| second | const [DateTimeOffset](../)\& | Второй объект для сравнения. |

### Возвращаемое значение

- 1 - если **first** раньше **second**, 1 - если **first** позже **second**, 0 - если **first** равно **second**.

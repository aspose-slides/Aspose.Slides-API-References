## DateTimeOffset::operator+(TimeSpan) const метод

Возвращает новый экземпляр класса [DateTimeOffset](../), представляющий значение даты и времени, которое является суммой значения, представленного текущим объектом, и указанного интервала времени.

```cpp
DateTimeOffset System::DateTimeOffset::operator+(TimeSpan value) const
```

### Arguments

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | [TimeSpan](../../timespan/) | Интервал времени, который добавляется к значению, представленному текущим объектом |

### Возвращаемое значение

Новый экземпляр класса [DateTimeOffset](../), представляющий значение даты и времени, которое является суммой значения, представленного текущим объектом, и **value**.

## DateTimeOffset::operator>=(const DateTimeOffset\&) const метод

Определяет, представляет ли текущий объект значение даты и времени, которое позже или совпадает со значением, представленным указанным объектом [DateTimeOffset](../).

```cpp
bool System::DateTimeOffset::operator>=(const DateTimeOffset &other) const
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| other | const [DateTimeOffset](../)\& | Объект [DateTimeOffset](../), с которым следует сравнить текущий объект |

### Возвращаемое значение

True, если значение даты и времени, представленное текущим объектом, позже или совпадает со значением, представленным **other**, иначе - false

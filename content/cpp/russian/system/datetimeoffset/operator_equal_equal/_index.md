## DateTimeOffset::operator==(const DateTimeOffset\&) const метод

Определяет, представляют ли текущий объект и указанный [DateTimeOffset](../) объект одно и то же значение даты и времени.

```cpp
bool System::DateTimeOffset::operator==(const DateTimeOffset &other) const
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| other | const [DateTimeOffset](../)\& | Объект [DateTimeOffset](../), с которым сравнивается текущий объект |

### Возвращаемое значение

True, если текущий объект и указанный объект представляют одинаковое значение даты и времени, иначе — false

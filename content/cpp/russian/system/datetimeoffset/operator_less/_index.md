## DateTimeOffset::operator<(const DateTimeOffset\&) const method

Определяет, представляет ли текущий объект значение даты и времени, которое предшествует значению, представленному указанным объектом [DateTimeOffset](../).

```cpp
bool System::DateTimeOffset::operator<(const DateTimeOffset &other) const
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| other | const [DateTimeOffset](../)\& | Объект [DateTimeOffset](../), с которым сравнивается текущий объект |

### Возвращаемое значение

True, если значение даты и времени, представленное текущим объектом, предшествует значению, представленному **other**, иначе — false

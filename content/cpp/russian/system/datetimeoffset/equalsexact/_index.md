## DateTimeOffset::EqualsExact(const DateTimeOffset\&) const method

Проверяет, представляют ли два [DateTimeOffset](../) объекта один и тот же момент времени и имеют одинаковый смещённый часовой пояс.

```cpp
bool System::DateTimeOffset::EqualsExact(const DateTimeOffset &other) const
```

### Аргументы

| Parameter | Type | Description |
| --- | --- | --- |
| other | const [DateTimeOffset](../)\& | [Object](../../object/) для сравнения. |

### Возвращаемое значение

true, если текущий объект точно равен **other**, иначе — false.

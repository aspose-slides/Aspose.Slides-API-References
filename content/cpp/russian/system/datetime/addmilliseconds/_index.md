## DateTime::AddMilliseconds(double) const метод

Возвращает новый экземпляр [DateTime](../) класса, представляющего значение даты и времени, которое является суммой значения, представленного текущим объектом, и указанного количества миллисекунд.

```cpp
DateTime System::DateTime::AddMilliseconds(double value) const
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | **double** | Количество миллисекунд для добавления; значение может быть положительным или отрицательным |

### Возвращаемое значение

Новый экземпляр [DateTime](../) класса, представляющего значение даты и времени, которое является суммой значения, представленного текущим объектом, и **value** миллисекунд

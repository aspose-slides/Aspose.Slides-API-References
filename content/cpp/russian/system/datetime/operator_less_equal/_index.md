## DateTime::operator<=(DateTime) const метод


Определяет, представляет ли текущий объект значение даты и времени, которое ранее или равно значению, представленному указанным объектом [DateTime](../).

```cpp
constexpr bool System::DateTime::operator<=(DateTime other) const
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| other | [DateTime](../) | [DateTime](../) объект, с которым сравнивается текущий объект |

### Возвращаемое значение

True если значение даты и времени, представленное текущим объектом, предшествует или равно значению, представленному **other**, иначе — false

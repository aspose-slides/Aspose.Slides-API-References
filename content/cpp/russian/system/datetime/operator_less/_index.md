## DateTime::operator<(DateTime) const метод


Определяет, представляет ли текущий объект значение даты и времени, которое раньше значения, представляемого указанным объектом [DateTime](../).

```cpp
constexpr bool System::DateTime::operator<(DateTime other) const
```


### Arguments

| Parameter | Type | Description |
| --- | --- | --- |
| other | [DateTime](../) | Объект [DateTime](../) для сравнения с текущим объектом |

### Return Value

True, если значение даты и времени, представляемое текущим объектом, ранее, чем значение, представляемое **other**, иначе — false

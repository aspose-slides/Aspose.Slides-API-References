## DateTime::AddTicks(int64_t) const метод

Возвращает новый экземпляр класса [DateTime](../), представляющий значение даты и времени, которое является суммой значения, представленного текущим объектом, и указанного количества 100-наносекундных интервалов.

```cpp
DateTime System::DateTime::AddTicks(int64_t value) const
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | **int64_t** | Количество 100-наносекундных интервалов для добавления; значение может быть положительным или отрицательным |

### Возвращаемое значение

Новый экземпляр класса [DateTime](../), представляющий значение даты и времени, которое является суммой значения, представленного текущим объектом, и **value** 100-наносекундных интервалов

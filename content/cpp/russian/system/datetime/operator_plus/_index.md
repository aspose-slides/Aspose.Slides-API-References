## DateTime::operator+(TimeSpan) const метод

Возвращает новый экземпляр класса [DateTime](../), который представляет значение даты и времени, являющееся суммой значения, представленного текущим объектом, и указанного временного интервала.

```cpp
DateTime System::DateTime::operator+(TimeSpan value) const
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | [TimeSpan](../../timespan/) | Временной интервал, добавляемый к значению, представленному текущим объектом |

### Возвращаемое значение

Новый экземпляр класса [DateTime](../), который представляет значение даты и времени, являющееся суммой значения, представленного текущим объектом, и **value**.

## DateTime::FromBinary(int64_t) метод


Десериализует значение даты и времени из заданного беззнакового 64-битного целого числа и устанавливает новый экземпляр класса [DateTime](../) в это значение.

```cpp
static DateTime System::DateTime::FromBinary(int64_t value)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | **int64_t** | 64-битное целое число, из которого десериализуется значение даты и времени |

### Возвращаемое значение

новый экземпляр класса [DateTime](../), представляющий значение даты и времени, десериализованное из указанного беззнакового 64-битного целого числа

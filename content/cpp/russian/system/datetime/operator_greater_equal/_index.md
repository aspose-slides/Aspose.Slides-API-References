## DateTime::operator>=(DateTime) const метод


Определяет, представляет ли текущий объект значение даты и времени, которое позже или равно значению, представляемому указанным [DateTime](../) объектом.

```cpp
constexpr bool System::DateTime::operator>=(DateTime other) const
```


### Аргументы

| Parameter | Type | Description |
| --- | --- | --- |
| other | [DateTime](../) | Объект [DateTime](../), с которым сравнивают текущий объект |

### Возвращаемое значение

True, если значение даты и времени, представленное текущим объектом, позже или равно значению, представленному **other**, в противном случае — false

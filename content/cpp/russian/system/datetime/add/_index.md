## DateTime::Add(TimeSpan) const метод

Возвращает новый экземпляр класса [DateTime](../), представляющий значение даты и времени, полученное в результате добавления указанного интервала времени к значению даты и времени, представленному текущим объектом.

```cpp
DateTime System::DateTime::Add(TimeSpan value) const
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | [TimeSpan](../../timespan/) | Интервал времени для добавления |

### Возвращаемое значение

Новый экземпляр класса [DateTime](../), представляющий значение даты и времени, полученное в результате добавления интервала времени **tm** к значению даты и времени, представленному текущим объектом.

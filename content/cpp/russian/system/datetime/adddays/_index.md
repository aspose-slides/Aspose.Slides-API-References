## DateTime::AddDays(double) const метод

Возвращает новый экземпляр класса [DateTime](../), представляющий значение даты и времени, которое является суммой значения, представленного текущим объектом, и указанного количества дней.

```cpp
DateTime System::DateTime::AddDays(double value) const
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | **double** | Количество дней для добавления; значение может быть положительным или отрицательным |

### Возвращаемое значение

Новый экземпляр класса [DateTime](../), представляющий значение даты и времени, которое является суммой значения, представленного текущим объектом, и **value** дней

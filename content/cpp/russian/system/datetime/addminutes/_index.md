```cpp
DateTime System::DateTime::AddMinutes(double value) const
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | **double** | Количество минут для добавления; значение может быть положительным или отрицательным |

### Возвращаемое значение

Новый экземпляр класса [DateTime](../) представляющий значение даты и времени, которое является суммой значения, представленного текущим объектом, и **value** минут

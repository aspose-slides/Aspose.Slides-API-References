## DateTime::AddSeconds(double) const метод

Возвращает новый экземпляр класса [DateTime](../), представляющий значение даты и времени, которое является суммой значения, представленного текущим объектом, и указанного количества секунд.

```cpp
DateTime System::DateTime::AddSeconds(double value) const
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | **double** | Количество секунд для добавления; значение может быть положительным или отрицательным |

### Возвращаемое значение

Новый экземпляр класса [DateTime](../), представляющий значение даты и времени, которое является суммой значения, представленного текущим объектом, и **value** секунд

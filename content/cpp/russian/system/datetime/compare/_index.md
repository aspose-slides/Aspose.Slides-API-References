## DateTime::Compare(DateTime, DateTime) метод


Сравнивает два значения, представленные указанными экземплярами класса [DateTime](../), и возвращает значение, указывающее относительное положение этих значений во времени.

```cpp
static constexpr int System::DateTime::Compare(DateTime t1, DateTime t2)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| t1 | [DateTime](../) | Первый сравниваемый |
| t2 | [DateTime](../) | Второй сравниваемый |

### Возвращаемое значение

Значение, меньшее 0, если **t1** раньше **t2**; 0, если **t1** равен **t2**; значение, большее 0, если **t1** позже **t2**.

## DateTime::CompareTo(DateTime) const метод

Сравнивает два значения даты и времени, представленные текущим объектом и указанным экземпляром класса [DateTime](../) и возвращает значение, указывающее относительное положение значений во временной шкале.

```cpp
constexpr int System::DateTime::CompareTo(DateTime value) const
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | [DateTime](../) | Экземпляр класса [DateTime](../) для сравнения с текущим объектом |

### Возвращаемое значение

Значение, меньшее 0, если текущий объект представляет значение, предшествующее тому, которое представлено **value**; 0, если значения, представленные обоими объектами, одинаковы; значение, большее 0, если значение, представленное текущим объектом, позже значения, представленного **value**.

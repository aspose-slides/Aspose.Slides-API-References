## DateTime::operator!=(DateTime) const метод

Определяет, представляют ли текущий объект и указанный [DateTime](../) объект различные значения даты и времени.

```cpp
constexpr bool System::DateTime::operator!=(DateTime other) const
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| other | [DateTime](../) | Объект [DateTime](../), с которым сравнивается текущий объект |

### Возвращаемое значение

True, если текущий объект и указанный объект представляют различные значения даты и времени, иначе — false

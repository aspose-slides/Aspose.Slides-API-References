## DateTime::IsDaylightSavingTime() const метод

Определяет, попадает ли значение даты и времени, представленное текущим объектом, в диапазон действия летнего времени для текущего часового пояса.

```cpp
bool System::DateTime::IsDaylightSavingTime() const
```

### Возвращаемое значение

True, если значение даты и времени, указанное текущим объектом, попадает в диапазон действия летнего времени, иначе - false

## DateTime::operator==(DateTime) const метод


Определяет, представляет ли текущий объект и указанный [DateTime](../) объект одно и то же значение даты и времени.

```cpp
constexpr bool System::DateTime::operator==(DateTime other) const
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| other | [DateTime](../) | [DateTime](../) объект для сравнения текущего объекта |

### Возвращаемое значение

True если текущий объект и указанный объект представляют одно и то же значение даты и времени, иначе - false

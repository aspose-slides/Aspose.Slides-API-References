## DateTime::AddYears(int) const метод


Возвращает новый экземпляр [DateTime](../) класса, представляющий значение даты и времени, равное тому, которое представлено текущим объектом, с увеличенным на указанное число компонентом года.

```cpp
DateTime System::DateTime::AddYears(int value) const
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | int | Количество лет для добавления; значение может быть положительным или отрицательным |

### Возвращаемое значение

Новый экземпляр [DateTime](../) класса, представляющий значение даты и времени, равное тому, которое представлено текущим объектом, с увеличенным на **value** компонентом года.

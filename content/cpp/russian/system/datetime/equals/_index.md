## DateTime::Equals(DateTime) const метод

Определяет, представляет ли указанный экземпляр класса [DateTime](../) то же значение даты и времени, что и текущий объект.

```cpp
constexpr bool System::DateTime::Equals(DateTime other) const
```

### Аргументы

| Parameter | Type | Description |
| --- | --- | --- |
| other | [DateTime](../) | Экземпляр класса [DateTime](../) для сравнения с текущим объектом |

### Возвращаемое значение

True, если **other** и текущий объект представляют одно и то же значение, иначе - false

## DateTime::Equals(DateTime, DateTime) метод

Определяет, представляют ли указанные экземпляры класса [DateTime](../) одинаковое значение даты и времени.

```cpp
static constexpr bool System::DateTime::Equals(DateTime t1, DateTime t2)
```

### Аргументы

| Parameter | Type | Description |
| --- | --- | --- |
| t1 | [DateTime](../) | Первый операнд |
| t2 | [DateTime](../) | Второй операнд |

### Возвращаемое значение

True, если **t1** и **t2** представляют одно и то же значение, иначе - false

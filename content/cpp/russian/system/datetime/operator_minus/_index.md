## DateTime::operator-(TimeSpan) const метод

Возвращает новый экземпляр класса [DateTime](../), представляющий значение даты и времени, которое является результатом вычитания указанного интервала времени из значения, представляемого текущим объектом.

```cpp
DateTime System::DateTime::operator-(TimeSpan value) const
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | [TimeSpan](../../timespan/) | Интервал времени для вычитания |

### Возвращаемое значение

Новый экземпляр класса [DateTime](../), представляющий значение даты и времени, которое является результатом вычитания **value** из значения, представляемого текущим объектом.

## DateTime::operator-(DateTime) const метод

Возвращает экземпляр класса [TimeSpan](../../timespan/), который представляет интервал времени между значениями даты и времени, представленными текущим объектом и указанным объектом.

```cpp
constexpr TimeSpan System::DateTime::operator-(DateTime value) const
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | [DateTime](../) | Экземпляр класса [DateTime](../), обозначающий один конец вычисляемого интервала |

### Возвращаемое значение

Экземпляр класса [TimeSpan](../../timespan/), представляющий интервал времени между значениями даты и времени, представленными текущим объектом и **value**.

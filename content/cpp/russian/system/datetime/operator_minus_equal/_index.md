## DateTime::operator-=(TimeSpan) метод

Устанавливает текущий объект в значение даты и времени, которое является результатом вычитания указанного промежутка времени из значения даты и времени, представленного текущим объектом.

```cpp
DateTime & System::DateTime::operator-=(TimeSpan value)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | [TimeSpan](../../timespan/) | Промежуток времени для вычитания |

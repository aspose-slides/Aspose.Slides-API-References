## DateTime::Subtract(TimeSpan) const метод

Возвращает новый экземпляр класса [DateTime](../), представляющий значение даты и времени, которое является результатом вычитания указанного временного интервала из значения, представленного текущим объектом.

```cpp
DateTime System::DateTime::Subtract(TimeSpan duration) const
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| duration | [TimeSpan](../../timespan/) | Временной интервал для вычитания |

### Возвращаемое значение

Новый экземпляр класса [DateTime](../), представляющий значение даты и времени, которое является результатом вычитания **duration** из значения, представленного текущим объектом.

## DateTime::Subtract(DateTime) const метод

Возвращает экземпляр класса [TimeSpan](../../timespan/), представляющий временной интервал между значениями даты и времени, представленными текущим объектом и указанным объектом.

```cpp
constexpr TimeSpan System::DateTime::Subtract(DateTime value) const
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | [DateTime](../) | Экземпляр класса [DateTime](../), обозначающий один конец интервала, который нужно вычислить |

### Возвращаемое значение

Экземпляр класса [TimeSpan](../../timespan/), представляющий временной интервал между значениями даты и времени, представленными текущим объектом и **value**.

## Details_NotSupportedException::Is(const System::TypeInfo\&) const method




```cpp
bool System::Details_NotSupportedException::Is(const System::TypeInfo &target) const override
```


### Аргументы

| Parameter | Type | Description |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) структура, описывающая тип, с которым сравнивается текущий объект. |

### Возвращаемое значение

True, если объект имеет помеченный тип или его подкласс, иначе false.

## Замечания

Проверьте, представляет ли объект экземпляр типа, описанного targetType. Аналог оператора C# 'is'.

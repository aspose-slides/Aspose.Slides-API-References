## Details_OverflowException::Is(const System::TypeInfo\&) const method




```cpp
bool System::Details_OverflowException::Is(const System::TypeInfo &target) const override
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) структура, описывающая тип, с которым проверяется текущий объект. |

### Возвращаемое значение

True, если объект имеет помеченный тип или его подкласс, иначе — false.
## Примечания


Check if object represents an instance of type described by targetType. Analog of C# 'is' operator.

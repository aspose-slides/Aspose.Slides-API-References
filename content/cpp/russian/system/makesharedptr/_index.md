## System::MakeSharedPtr(X *) функция


Преобразует необработанный указатель в умный указатель.

```cpp
template<class X> SmartPtr<X> System::MakeSharedPtr(X *p)
```


### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| X | Тип указателя. |

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| p | X * | Необработанный указатель на объект. |

### Возвращаемое значение

Разделяемый умный указатель на объект.

## System::MakeSharedPtr(const X *) функция


Преобразует необработанный указатель в умный указатель. Перегрузка для const указателей. Полезно, например, при использовании переменной 'this' в методах C#, переводимых как const.

```cpp
template<class X> SmartPtr<X> System::MakeSharedPtr(const X *p)
```


### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| X | Тип указателя. |

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| p | const X * | Необработанный указатель на объект. |

### Возвращаемое значение

Разделяемый умный указатель на объект.

## TestToolsExt::IsTest(const char *, const char *, const char *) метод


Проверяет, существует ли тестовый метод.

```cpp
static bool System::TestToolsExt::IsTest(const char *name_space, const char *class_name, const char *method_name)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| name_space | const char * | Пространство имён для поиска. |
| class_name | const char * | Класс для поиска. |
| method_name | const char * | Метод для поиска. |

### Возвращаемое значение

True, если тестовый метод зарегистрирован, иначе false.

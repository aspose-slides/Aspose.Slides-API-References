## TestToolsExt::GetNamespace(const char *, const char *, std::string\&) метод

Получает пространство имён указанного теста.

```cpp
static bool System::TestToolsExt::GetNamespace(const char *class_name, const char *method_name, std::string &name_space)
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| class_name | const char * | Класс для поиска. |
| method_name | const char * | Метод для поиска. |
| name_space | std::string\& | Переменная для помещения имени пространства имён, если найдено. |

### Возвращаемое значение

Истина, если метод теста найден, иначе ложь.

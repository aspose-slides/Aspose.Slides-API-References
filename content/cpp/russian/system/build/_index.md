## System::Build(Args\&&...) функция

Построить объект с прямым владением.

```cpp
template<typename T,typename...> Details::ObjectBuilder<T> System::Build(Args &&... args)
```

### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| T | Тип объекта для построения |
| Args | Типы аргументов для конструктора объекта |

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| args | Args\&&... | Аргументы, передаваемые конструктору объекта |

### Возвращаемое значение

ObjectBuilder, настроенный для прямого создания объекта

## Примечания

[Object](../object/) построение должно быть завершено вызовом [Get()](../get/)

## IsWeakPtr struct

Класс-трейт для проверки, является ли конкретный класс специализацией [System::WeakPtr](../weakptr/). Не проверяет, находится ли экземпляр действительно в слабом режиме.

```cpp
template<class T>class IsWeakPtr : public System::detail::is_a<T, System::WeakPtr>
```

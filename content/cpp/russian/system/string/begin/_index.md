## String::begin() const метод

Возвращает указатель на начало фактического буфера строки. Никогда не переаллочирует ничего. Не гарантирует, что буфер будет нуль-терминированным.

```cpp
const UChar * System::String::begin() const
```

### Возвращаемое значение

Указатель на фактический буфер строки, нуль-терминированный или нет.

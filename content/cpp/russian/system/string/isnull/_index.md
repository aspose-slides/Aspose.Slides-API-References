## String::IsNull() const метод

Проверяет, считается ли строка нулевой. [String](../) считается нулевой только если она создана через конструктор [String()](../string/), перемещена, скопирована или назначена из нулевой строки или был вызван метод [reset()](../reset/).

```cpp
bool System::String::IsNull() const
```

### Возвращаемое значение

true если строка считается нулевой, false в противном случае.

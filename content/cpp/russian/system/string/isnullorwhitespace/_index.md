## String::IsNullOrWhiteSpace(const String\&) метод

Указывает, является ли заданная строка null, пустой или состоит только из пробельных символов.

```cpp
static bool System::String::IsNullOrWhiteSpace(const String &str)
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| str | const [String](../)\& | [String](../) для проверки. |

### Возвращаемое значение

true, если string равен null или пуст или состоит исключительно из пробельных символов, false в противном случае.

## String::rbegin() const метод

Возвращает обратный итератор к последнему символу (если он есть) фактического буфера строки.

```cpp
reverse_iterator System::String::rbegin() const
```

### Возвращаемое значение

Обратный итератор к последнему символу фактического буфера строки (если он есть) или такой же, как [rend()](../rend/).

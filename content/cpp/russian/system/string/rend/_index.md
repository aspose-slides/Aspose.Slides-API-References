## String::rend() const метод


Возвращает обратный итератор к позиции перед первым символом (если он существует) фактического буфера строки.

```cpp
reverse_iterator System::String::rend() const
```


### Возвращаемое значение

Обратный итератор к элементу, предшествующему первому символу фактического буфера строки (если он существует), или к элементу сразу перед выделенным буфером.

## String::end() const метод

Возвращает указатель на конец фактического буфера строки. Никогда не переаллоцирует ничего. Не гарантирует, что буфер будет нуль-терминированным.

```cpp
const UChar * System::String::end() const
```

### Возвращаемое значение

Указатель на первый элемент после фактического буфера строки, нуль-терминированный или нет.

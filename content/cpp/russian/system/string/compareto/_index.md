## String::CompareTo(const String\&) const method

Сравнивает две строки в стиле 'less-equals-more'. Использует текущую культуру.

```cpp
int System::String::CompareTo(const String &str) const
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| str | const [String](../)\& | [String](../) для сравнения с этим. |

## Возвращаемое значение

Отрицательное значение, если это считается меньше, чем str, ноль, если они совпадают, положительное значение, если это больше, чем str.

## String::Remove(int32_t, int32_t) const метод


Извлекает всё, кроме подстроки, из текущей строки.

```cpp
String System::String::Remove(int32_t startIndex, int32_t count=INT32_MAX) const
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| startIndex | **int32_t** | Индекс первого символа, который не следует извлекать. |
| count | **int32_t** | Количество символов, которые не следует извлекать. |

### Возвращаемое значение

[String](../) вокруг указанной подстроки.

## String::operator[](int) const метод

Получает символ в указанной позиции.

```cpp
char_t System::String::operator[](int index) const
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| index | int | Индекс, из которого получается символ. |

### Возвращаемое значение

Символ в указанной позиции или 0xffff, если выходит за пределы.

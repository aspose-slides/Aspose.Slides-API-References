## WeakPtr::get_weak() const метод


Получает ссылочный объект. Утверждает, что указатель находится в режиме слабой ссылки.

```cpp
Object * System::WeakPtr<T>::get_weak() const
```


### Возвращаемое значение

Необработанный указатель на ссылочный объект (если назначен и не истёк) или nullptr.

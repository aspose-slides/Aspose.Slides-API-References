## WeakPtr::operator=(Q\&&) метод

Присваивает значение слабому указателю. Вызывает конкретный оператор присваивания SmartPtr_.

```cpp
template<typename Q> WeakPtr & System::WeakPtr<T>::operator=(Q &&value)
```

### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| Q | Тип аргумента, поддерживаемый [System::SmartPtr](../../smartptr/) операторами присваивания. |

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | Q\&& | Указатель, из которого копировать значение объекта. |

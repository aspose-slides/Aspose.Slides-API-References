```cpp
bool System::Details_UriFormatException::Is(const System::TypeInfo &target) const override
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) структура, описывающая тип, с которым сравнивается текущий объект. |

### Возвращаемое значение

True, если объект имеет указанный тип или его подкласс, иначе — false.

## Примечания

Проверьте, представляет ли объект экземпляр типа, описанного параметром targetType. Аналог оператора C# 'is' operator.

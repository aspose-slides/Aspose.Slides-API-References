## TimeZoneInfo::IsDaylightSavingTime(DateTime) const метод


Проверяет, попадает ли указанная дата и время в диапазон перехода на летнее время.

```cpp
bool System::TimeZoneInfo::IsDaylightSavingTime(DateTime date_time) const
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| date_time | [DateTime](../../datetime/) | Дата и время. |

### Возвращаемое значение

True если date_time находится в периоде летнего времени.

## BoxedValue::is() const метод

Определяет, является ли тип упакованного значения, представленного текущим объектом, **V**.

```cpp
template<class V> bool System::BoxedValue<T>::is() const
```

### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| V | Тип, с которым сравнивается тип упакованного значения, представленного текущим объектом |

### Возвращаемое значение

true, если тип упакованного значения, представленного текущим объектом, **V**, иначе — false

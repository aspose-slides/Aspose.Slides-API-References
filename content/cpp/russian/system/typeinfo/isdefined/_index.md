## TypeInfo::IsDefined(const TypeInfo\&, bool) const метод

НЕ РЕАЛИЗОВАНО. Указывает, применён ли один или несколько атрибутов указанного типа или его производных типов к этому члену.

```cpp
bool System::TypeInfo::IsDefined(const TypeInfo &attributeType, bool inherit) const
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| attributeType | const [TypeInfo](../)\& | Тип пользовательского атрибута для поиска. Поиск включает производные типы. |
| inherit | **bool** | true, чтобы искать в цепочке наследования этого члена атрибуты; иначе false. Этот параметр игнорируется для свойств и событий. |

### Возвращаемое значение

true, если один или несколько экземпляров attributeType или любого из его производных типов применён к этому члену; иначе false.

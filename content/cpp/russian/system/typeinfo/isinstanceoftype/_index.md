```cpp
bool System::TypeInfo::IsInstanceOfType(const SharedPtr<Object> &obj) const
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| obj | const [SharedPtr](../../sharedptr/)\<[Object](../../object/)\>\& | Объект, с которым сравнивается текущий тип |

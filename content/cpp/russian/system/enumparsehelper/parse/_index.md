## EnumParseHelper::Parse(const String\&, bool) метод

Преобразует заданную строку в соответствующее значение константы перечисления.

```cpp
static E System::EnumParseHelper<E, G, Guard>::Parse(const String &str, bool ignoreCase=false)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| str | const [String](../../string/)\& | [String](../../string/) который интерпретируется как содержащий имя константы перечисления |
| ignoreCase | **bool** | Указывает, следует ли игнорировать регистр при интерпретации строки |

### Возвращаемое значение

Константа перечисления, имя которой указано в **str**

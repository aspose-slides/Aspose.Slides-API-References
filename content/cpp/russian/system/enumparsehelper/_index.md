## EnumParseHelper struct

Вспомогательный класс, предоставляющий возможность преобразования строкового представления перечисления в соответствующее значение перечисления.

```cpp
template<class E,class G,class Guard>class EnumParseHelper
```

### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| E | Тип перечисления, с которым работают методы класса |
| G | Второй формальный аргумент [System::Enum](../enum/), который будет использоваться. |
| Guard | Формальный параметр шаблона для выбора доступного алгоритма разбора. |

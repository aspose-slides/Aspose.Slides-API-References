## Array::GetLongLength(int) метод

Возвращает количество элементов в указанном измерении как 64-битное целое число.

```cpp
int64_t System::Array<T>::GetLongLength(int dimension)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| dimension | int | Измерение, длину которого необходимо вернуть |

### Возвращаемое значение

64-битное целое, представляющее количество элементов в указанном измерении

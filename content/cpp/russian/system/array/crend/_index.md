## Array::crend() const метод

Возвращает обратный итератор к элементу, следующему за последним элементом перевёрнутого контейнера. Он соответствует элементу, предшествующему первому элементу неперевёрнутого контейнера. Этот элемент служит заполнительным, попытка доступа к нему приводит к неопределённому поведению.

```cpp
const_reverse_iterator System::Array<T>::crend() const noexcept
```

### Возвращаемое значение

Итератор, указывающий на теоретически const-квалифицированный элемент, предшествующий первому элементу контейнера.

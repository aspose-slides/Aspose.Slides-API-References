## Array::end() метод

Возвращает итератор к элементу, следующему за последним элементом контейнера. Этот элемент служит заполнителем; попытка доступа к нему приводит к неопределённому поведению.

```cpp
iterator System::Array<T>::end() noexcept
```

### Возвращаемое значение

Итератор, указывающий на теоретический элемент, расположенный после последнего элемента контейнера.

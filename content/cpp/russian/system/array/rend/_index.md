## Array::rend() метод

Возвращает обратный итератор, указывающий на элемент, следующий за последним элементом обратного контейнера. Это соответствует элементу, предшествующему первому элементу необратного контейнера. Этот элемент служит заполнительным, попытка доступа к нему приводит к неопределенному поведению.

```cpp
reverse_iterator System::Array<T>::rend() noexcept
```

### Возвращаемое значение

Итератор, указывающий на теоретический элемент, предшествующий первому элементу контейнера.

## Array::rend() const метод

Возвращает обратный итератор, указывающий на элемент, следующий за последним элементом обратного контейнера. Это соответствует элементу, предшествующему первому элементу необратного контейнера. Этот элемент служит заполнительным, попытка доступа к нему приводит к неопределенному поведению.

```cpp
const_reverse_iterator System::Array<T>::rend() const noexcept
```

### Возвращаемое значение

Итератор, указывающий на теоретический элемент, предшествующий первому элементу константного контейнера.

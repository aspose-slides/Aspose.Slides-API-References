## Array::cbegin() const метод


Возвращает итератор к первому элементу, квалифицированному как const, контейнера. Если контейнер пуст, возвращаемый итератор будет равен [cend()](../cend/).

```cpp
const_iterator System::Array<T>::cbegin() const noexcept
```


### Возвращаемое значение

Итератор, указывающий на первый элемент, квалифицированный как const, контейнера.

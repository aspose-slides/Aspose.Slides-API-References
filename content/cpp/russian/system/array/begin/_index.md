## Array::begin() метод

Возвращает итератор на первый элемент контейнера. Если контейнер пуст, возвращённый итератор будет равен [end()](../end/).

```cpp
iterator System::Array<T>::begin() noexcept
```

### Возвращаемое значение

Итератор, указывающий на первый элемент контейнера.

## Array::begin() const метод

Возвращает итератор на первый элемент контейнера, объявленного const. Если контейнер пуст, возвращённый итератор будет равен [end()](../end/).

```cpp
const_iterator System::Array<T>::begin() const noexcept
```

### Возвращаемое значение

Итератор, указывающий на первый элемент контейнера, объявленного const.

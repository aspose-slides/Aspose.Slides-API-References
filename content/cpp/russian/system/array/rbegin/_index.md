## Array::rbegin() метод

Возвращает обратный итератор, указывающий на первый элемент перевёрнутого контейнера. Он соответствует последнему элементу неперевёрнутого контейнера. Если контейнер пуст, возвращаемый итератор равен [rend()](../rend/).

```cpp
reverse_iterator System::Array<T>::rbegin() noexcept
```

### Возвращаемое значение

Итератор, указывающий на последний элемент контейнера.

## Array::rbegin() const метод

Возвращает обратный итератор, указывающий на первый элемент перевёрнутого контейнера. Он соответствует последнему элементу неперевёрнутого контейнера. Если контейнер пуст, возвращаемый итератор равен [rend()](../rend/).

```cpp
const_reverse_iterator System::Array<T>::rbegin() const noexcept
```

### Возвращаемое значение

Итератор, указывающий на последний элемент контейнера, объявленного const.

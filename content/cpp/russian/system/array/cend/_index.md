## Array::cend() const метод

Возвращает итератор к элементу, следующему за последним элементом контейнера. Этот элемент служит заполняющим элементом; попытка доступа к нему приводит к неопределённому поведению.

```cpp
const_iterator System::Array<T>::cend() const noexcept
```

### Возвращаемое значение

Итератор, указывающий на теоретический const-квалифицированный элемент, расположенный после последнего элемента контейнера.

## Array::crbegin() const метод

Возвращает обратный итератор к первому элементу обратного контейнера. Он соответствует последнему элементу обычного контейнера. Если контейнер пуст, возвращаемый итератор равен [crend()](../crend/).

```cpp
const_reverse_iterator System::Array<T>::crbegin() const noexcept
```

### Возвращаемое значение

Итератор, указывающий на последний const-квалифицированный элемент контейнера.

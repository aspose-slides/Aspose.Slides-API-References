## System::operator/(const T\&, const Decimal\&) функция

Возвращает новый экземпляр класса [Decimal](../decimal/), представляющий значение, являющееся результатом деления указанного значения и значения, представленного указанным объектом [Decimal](../decimal/).

```cpp
template<typename T,typename _> Decimal System::operator/(const T &x, const Decimal &d)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| x | const T\& | Значение для деления |
| d | const [Decimal](../decimal/)\& | Объект [Decimal](../decimal/), представляющий делитель |

### Возвращаемое значение

Новый экземпляр класса [Decimal](../decimal/), представляющий значение, являющееся результатом деления **x** на значение, представляемое **d**.

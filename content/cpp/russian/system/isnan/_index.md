## System::IsNaN(const T\&) функция

Определяет, является ли указанное значение значением Not-A-Number.

```cpp
template<typename T> bool System::IsNaN(const T &value)
```

### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| T | Тип значения, проверяемого функцией |

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | const T\& | Значение для проверки |

### Возвращаемое значение

True, если **value** является NaN-значением, иначе - false

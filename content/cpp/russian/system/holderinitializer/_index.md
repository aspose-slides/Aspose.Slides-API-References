## Структура HolderInitializer

Этот класс используется для получения постоянной ссылки на экземпляр объекта, независимо от того, является ли он lvalue или rvalue. Чтобы получить такую ссылку, используйте метод 'HoldIfTemporary', у которого есть три перегрузки. Две из них принимают rvalue в качестве параметра и просто возвращают ссылку на него. Третья, наоборот, принимает lvalue в качестве параметра, делает копию указателя, а затем возвращает ссылку на эту копию. Кроме того, в классе есть метод 'Hold', который безусловно удерживает переданное значение (используется для копирования значений локальных переменных в стеке или их дочерних ссылок).

```cpp
template<typename T,bool>class HolderInitializer
```

### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| T | Тип объекта, который необходимо удерживать. |
| R | True, если T является типом ссылки (специализация [SmartPtr](../smartptr/) или тип [System::String](../string/)), и удержание временных ссылок действительно требуется, false — в противном случае. |

## HolderInitializer::Hold(const T\&) метод

Копирует переданное lvalue в holder, затем возвращает ссылку на holder. Вызывающий код должен использовать этот метод, чтобы безусловно удерживать переданное значение.

```cpp
const T & System::HolderInitializer<T, R>::Hold(const T &value)
```

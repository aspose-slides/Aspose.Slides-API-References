## ObjectExt::UnboxStringSafe(const SmartPtr\<Object\>\&) метод

Извлекает строку из упакованного значения.

```cpp
static String System::ObjectExt::UnboxStringSafe(const SmartPtr<Object> &obj)
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| obj | const [SmartPtr](../../smartptr/)\<[Object](../../object/)\>\& | Упакованное строковое значение. |

### Возвращаемое значение

Если **obj** является упакованной строкой, возвращает разупакованное значение, иначе возвращает пустую строку.

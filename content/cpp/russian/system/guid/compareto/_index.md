## Guid::CompareTo(const Guid\&) const метод

Выполняет арифметическое сравнение GUID, представленных текущим и указанными объектами.

```cpp
int System::Guid::CompareTo(const Guid &g) const
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| g | const [Guid](../)\& | Операнд сравнения |

### Возвращаемое значение

- 1 если GUID, представленный текущим объектом, меньше GUID, представленного **g**; 0 если GUID равны; 1 если GUID, представленный текущим объектом, больше GUID, представленного **g**

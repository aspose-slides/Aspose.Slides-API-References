## Enum::HasFlag(E, E) метод

Определяет, установлены ли указанные биты в битовом представлении заданного значения перечисления.

```cpp
static bool System::Enum<E, Guard>::HasFlag(E value, E mask)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | E | Значение перечисления для проверки |
| mask | E | Маска для проверки битов значения |

### Возвращаемое значение

True, если биты, установленные в **mask**, также установлены в **value**, иначе — false

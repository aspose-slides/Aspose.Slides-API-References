## Enum::Compare(E, T) метод


Выполняет арифметическое сравнение значений указанных констант перечисления.

```cpp
template<typename T,typename _> static int System::Enum<E, Guard>::Compare(E a, T b)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| a | E | Первый сравниваемый |
| b | T | Второй сравниваемый |

### Возвращаемое значение

- 1 если **a** меньше **b**; 0 если значения равны; 1 если **a** больше **b**

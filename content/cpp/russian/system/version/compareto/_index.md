## Version::CompareTo(const Version\&) const метод

Сравнивает версии, представленные текущим объектом и указанным объектом.

```cpp
int System::Version::CompareTo(const Version &ver) const
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| ver | const [Version](../)\& | Объект [Version](../) для сравнения с текущим объектом |

### Возвращаемое значение

- 1 если номер версии, представленной текущим объектом, меньше того, что представлен **ver**; 0 если номера версий равны; 1 если номер версии, представленной текущим объектом, больше значения, представленного **ver**

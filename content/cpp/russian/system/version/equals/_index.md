## Version::Equals(const Version\&) const метод

Определяет, равны ли номера версий, представленные текущим и указанным объектами.

```cpp
bool System::Version::Equals(const Version &ver) const
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| ver | const [Version](../)\& | Объект [Version](../), с которым сравнивается текущий объект |

### Возвращаемое значение

True, если номера версий, представленные текущим объектом и указанным объектом, равны, иначе — false

## Version::Version(int, int, int, int) конструктор

Создаёт экземпляр, представляющий указанные значения major, minor, build и revision.

```cpp
System::Version::Version(int major, int minor, int build, int revision)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| major | int | Главный номер версии |
| minor | int | Номер второстепенной версии |
| build | int | Номер сборки |
| revision | int | Номер ревизии |

## ValueType класс

Базовый класс для типов значений с [Object](../object/) наследованием, сокращённым ради производительности. Этот тип должен быть выделен в стеке и передаваться в функции по значению или по ссылке. Никогда не используйте класс [System::SmartPtr](../smartptr/) для управления объектами этого типа.

```cpp
class ValueType
```

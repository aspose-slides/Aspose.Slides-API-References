Поддерживает сравнение значимых типов и ссылочных типов.

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| o | const [SharedPtr](../../sharedptr/)\<[Object](../../object/)\>\& | Объект ссылочного типа для сравнения. |

True, если объекты считаются равными, иначе — false.

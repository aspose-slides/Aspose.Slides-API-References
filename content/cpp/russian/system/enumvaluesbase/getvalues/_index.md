## EnumValuesBase::GetValues(const TypeInfo\&) метод

Возвращает массив, содержащий все значения указанного типа перечисления.

```cpp
static ArrayPtr<int64_t> System::EnumValuesBase::GetValues(const TypeInfo &type)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| type | const [TypeInfo](../../typeinfo/)\& | Объект [TypeInfo](../../typeinfo/), представляющий тип перечисления, значения которого необходимо вернуть |

Массив, содержащий значения всех констант перечисления типа, указанного параметром **type**.

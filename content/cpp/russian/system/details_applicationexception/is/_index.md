## Подробности_ApplicationException::Is(const System::TypeInfo\&) const метод


```cpp
bool System::Details_ApplicationException::Is(const System::TypeInfo &target) const override
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | [TypeInfo](../../typeinfo/) структура, описывающая тип, с которым проверяется текущий объект. |

### Возвращаемое значение

True если объект имеет указанный тип или его подкласс, иначе false.

## Замечания

Проверьте, представляет ли объект экземпляр типа, описанного targetType. Аналог оператора C# 'is'.

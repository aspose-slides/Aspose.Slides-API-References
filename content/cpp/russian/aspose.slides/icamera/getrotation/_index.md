## ICamera::GetRotation() метод

Вращение определяется с использованием координаты широты, координаты долготы и оборота вокруг оси, представляющего широту и долготу. первый элемент массива возвращаемого значения — широта, второй — долгота, третий — оборот. Возвращает null, если вращение не определено.

```cpp
virtual System::ArrayPtr<float> Aspose::Slides::ICamera::GetRotation()=0
```

### Возвращаемое значение

Массив значений вращения типа **float**[].

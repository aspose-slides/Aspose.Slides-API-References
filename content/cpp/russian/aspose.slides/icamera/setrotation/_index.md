## ICamera::SetRotation(float, float, float) метод


Вращение определяется с помощью координаты широты, координаты долготы и оборота вокруг оси как координат широты и долготы. Если значение любой координаты равно std::numeric_limits<float>::quiet_NaN(), всё вращение не определено.

```cpp
virtual void Aspose::Slides::ICamera::SetRotation(float latitude, float longitude, float revolution)=0
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| latitude | **float** | Значение широты **float** |
| longitude | **float** | Значение долготы **float** |
| revolution | **float** | Значение оборота **float** |

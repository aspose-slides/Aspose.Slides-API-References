## ParagraphFormat::set_SpaceBefore(float) метод

Устанавливает размер пространства перед первой строкой абзаца без наследования. Положительное значение задаёт процент от размера шрифта, который должен составлять пробел. Отрицательное значение задаёт размер пробела в пунктах. Записывается **float**.

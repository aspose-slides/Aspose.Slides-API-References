## ShapeThumbnailBounds enum

Перечисление типов границ миниатюры фигуры.

```cpp
enum class ShapeThumbnailBounds
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Slide | 0 | [Shape](../shape/) миниатюра будет иметь размер, равный размеру слайда. [Shape](../shape/) позиция будет сохранена. |
| Shape | 1 | [Shape](../shape/) миниатюра будет иметь размер, равный прямоугольнику границ фигуры с учётом настроек контура фигуры. |
| Appearance | 2 | [Shape](../shape/) миниатюра будет иметь размер, равный внешнему виду фигуры (в границах слайда). Возможны случаи, когда внешний вид фигуры не помещается в границы фигуры. Например, вращение, срез угла острого угла, 3D-effects, и т.п. |

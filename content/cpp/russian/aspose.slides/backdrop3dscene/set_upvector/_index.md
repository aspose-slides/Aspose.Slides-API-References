## Backdrop3DScene::set_UpVector(System::ArrayPtr\<float\>) метод

Устанавливает вектор, представляющий направление вверх. Точнее говоря, этот атрибут определяет вектор, представляющий направление вверх относительно плоскости заднего фона. Вектор представлен массивом из 3 значений типа float, определяющих координаты X, Y и Z. Записать **float**[].

## NormalViewProperties::set_PreferSingleView(bool) метод

Указывает, предпочитает ли пользователь видеть одноконтентный регион на весь экран вместо стандартного вида с тремя контентными регионами. Если включено, приложение может выбрать отображение одного из контентных регионов во всём окне. Записать **bool**.

void Aspose::Slides::NormalViewProperties::set_PreferSingleView(bool value) override

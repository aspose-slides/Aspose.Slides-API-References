## NormalViewProperties::get_PreferSingleView() метод

Указывает, предпочитает ли пользователь видеть область с одним содержимым на весь экран вместо стандартного обычного просмотра с тремя областями содержимого. Если включено, приложение может выбрать отображение одной из областей содержимого во всём окне. Читает **bool**.

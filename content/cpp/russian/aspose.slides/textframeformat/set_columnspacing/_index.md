## TextFrameFormat::set_ColumnSpacing(double) метод

Устанавливает расстояние между текстовыми колонками в области текста (в пунктах). Это должно применяться только когда присутствует более 1 колонки. Это значение должно быть положительным числом. В противном случае значение будет установлено в ноль. Записать **double**.

```cpp
void Aspose::Slides::TextFrameFormat::set_ColumnSpacing(double value) override
```

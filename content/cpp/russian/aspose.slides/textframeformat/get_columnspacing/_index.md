## TextFrameFormat::get_ColumnSpacing() метод

Возвращает расстояние между колонками текста в текстовой области (в пунктах). Это применяется только когда присутствует более одной колонки. Это значение должно быть положительным числом. В противном случае значение будет установлено в ноль. Читает **double**.

```cpp
double Aspose::Slides::TextFrameFormat::get_ColumnSpacing() override
```

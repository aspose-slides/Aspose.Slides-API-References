## TextFrameFormat::set_RotationAngle(float) метод

Указывает пользовательский угол поворота, применяемый к тексту внутри ограничивающего прямоугольника. Если он не указан, используется поворот соответствующей фигуры. Если указан, то он применяется независимо от фигуры. То есть к фигуре может быть применён поворот, а к самому тексту — свой поворот. Получаемое значение визуального поворота текста суммируется из этого свойства и предопределённого вертикального типа в свойстве TextVerticalType. Записывается **float**.

```cpp
void Aspose::Slides::TextFrameFormat::set_RotationAngle(float value) override
```

## Примечания

Рассмотрим случай, когда к фигуре применён поворот на 90 градусов по часовой стрелке. Кроме того, тело текста имеет поворот на -90 градусов против часовой стрелки. В результате фигура будет выглядеть повернутой, но текст внутри неё будет казаться неповернутым.

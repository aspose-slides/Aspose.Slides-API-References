## IShapeStyle::get_FillStyleIndex() метод

Возвращает индекс столбца заливки формы в матрицах стилей. 0 означает отсутствие заливки, положительное значение - индекс в стилях заливки темы, отрицательное значение - индекс в стилях фона темы. Читает **int16_t**.

```cpp
virtual int16_t Aspose::Slides::IShapeStyle::get_FillStyleIndex()=0
```

## IShapeStyle::set_FillStyleIndex(int16_t) метод

Устанавливает индекс столбца заполнения фигуры в матрицах стилей. 0 означает отсутствие заполнения, положительное значение — индекс в стилях заполнения темы, отрицательное значение — индекс в стилях фона темы. Запишите **int16_t**.

virtual void Aspose::Slides::IShapeStyle::set_FillStyleIndex(int16_t value)=0

## ShapeFrame::ShapeFrame(float, float, float, float, NullableBool, NullableBool, float) конструктор


Создает свойства новой рамки формы.

```cpp
Aspose::Slides::ShapeFrame::ShapeFrame(float x, float y, float width, float height, NullableBool flipH, NullableBool flipV, float rotationAngle)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| x | **float** | Координата X рамки. |
| y | **float** | Координата Y рамки. |
| width | **float** | Ширина рамки. |
| height | **float** | Высота рамки. |
| flipH | [NullableBool](../../nullablebool/) | True, если рамка отражена горизонтально. |
| flipV | [NullableBool](../../nullablebool/) | True, если рамка отражена вертикально. |
| rotationAngle | **float** | Количество градусов, на которое рамка повернута. |

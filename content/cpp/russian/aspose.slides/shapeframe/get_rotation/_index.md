## ShapeFrame::get_Rotation() метод


Возвращает количество градусов, на которое кадр вращён вокруг оси z. Положительное значение указывает на вращение по часовой стрелке; отрицательное значение указывает на вращение против часовой стрелки. Только для чтения **float**.

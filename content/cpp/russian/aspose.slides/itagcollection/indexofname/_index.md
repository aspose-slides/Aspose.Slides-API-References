## ITagCollection::IndexOfName(System::String) метод

Возвращает нуль-основанный индекс указанного ключа в коллекции.

```cpp
virtual int32_t Aspose::Slides::ITagCollection::IndexOfName(System::String name)=0
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| name | [System::String](../../../system/string/) | Имя для поиска в коллекции. |

### Возвращаемое значение

Нуль-основанный индекс ключа, если ключ найден в коллекции; в противном случае -1.

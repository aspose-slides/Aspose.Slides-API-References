## BulletFormat::set_Height(float) method


Устанавливает высоту маркера абзаца без наследования. Значение std::numeric_limits<float>::quiet_NaN() указывает, что маркер наследует высоту из первой части абзаца. Записывается **float**.

```cpp
void Aspose::Slides::BulletFormat::set_Height(float value) override
```

## Примечания


Отрицательное значение высоты означает, что высота задаётся в пунктах, а положительное значение — в процентах от окружающего текста.

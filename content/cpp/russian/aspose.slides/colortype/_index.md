## перечисление ColorType


Представляет различные режимы цвета.

```cpp
enum class ColorType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| NotDefined | -1 | Цвет не определён вовсе. |
| RGB | 0 | Стандартный 24-битный цвет RGB. |
| RGBPercentage | 1 | Цвет RGB высокого разрешения. |
| HSL | 2 | Цвет HSL высокого разрешения. |
| Scheme | 3 | Цвет Scheme. |
| System | 4 | [System](../../system/) цвет. |
| Preset | 5 | Цвет Preset. |

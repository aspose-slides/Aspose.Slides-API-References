## IShapeCollection::InsertGroupShape(int32_t) метод

Создает новую пустую групповую фигуру и вставляет её в коллекцию фигур по указанному индексу. Кадр группы автоматически подстраивается под любые добавленные в неё фигуры.

```cpp
virtual System::SharedPtr<IGroupShape> Aspose::Slides::IShapeCollection::InsertGroupShape(int32_t index)=0
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| index | **int32_t** | Нулевой индекс, по которому будет вставлена групповая фигура. |

### Возвращаемое значение

Недавно созданный [IGroupShape](../../igroupshape/).

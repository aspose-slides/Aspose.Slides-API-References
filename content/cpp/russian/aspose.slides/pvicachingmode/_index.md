## PVICachingMode enum

Представляет различные уровни кэширования для эффективных объектов PVI.

```cpp
enum class PVICachingMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| NoCaching | 0 | Классический режим с созданием каждого экземпляра. |
| NotDefinedOnly | 1 | Использовать кэширование эффективных значений для форматов NotDefined, классический режим для остальных. |
| Full | 2 | Использовать кэширование эффективных значений для всех форматов. |

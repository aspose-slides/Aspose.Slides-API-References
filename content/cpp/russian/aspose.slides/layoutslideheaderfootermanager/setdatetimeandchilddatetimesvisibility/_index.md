## LayoutSlideHeaderFooterManager::SetDateTimeAndChildDateTimesVisibility(bool) method


Изменяет видимость плейсхолдера даты и времени на слайде шаблона и всех вложенных плейсхолдеров даты и времени. Вложенные плейсхолдеры означают плейсхолдеры, содержащиеся на зависимых слайдах. Зависимые слайды используют и зависят от слайда шаблона.

```cpp
void Aspose::Slides::LayoutSlideHeaderFooterManager::SetDateTimeAndChildDateTimesVisibility(bool isVisible) override
```


### Arguments

| Parameter | Type | Description |
| --- | --- | --- |
| isVisible | **bool** | true - делает плейсхолдеры даты и времени видимыми, иначе - скрывает их. |

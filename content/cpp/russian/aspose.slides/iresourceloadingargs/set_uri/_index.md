## IResourceLoadingArgs::set_Uri(System::String) метод

URI ресурса, который используется для загрузки, если [IResourceLoadingCallback::ResourceLoading()](../../iresourceloadingcallback/resourceloading/) возвращает [ResourceLoadingAction::Default](../../resourceloadingaction/). Изначально он установлен на оригинальный URI ресурса, но может быть переопределён на любое значение.

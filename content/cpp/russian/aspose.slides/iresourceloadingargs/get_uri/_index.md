## IResourceLoadingArgs::get_Uri() метод


URI ресурса, который используется для загрузки, если [IResourceLoadingCallback::ResourceLoading()](../../iresourceloadingcallback/resourceloading/) возвращает [ResourceLoadingAction::Default](../../resourceloadingaction/). Изначально он установлен на исходный URI ресурса, но может быть переопределён на любое значение.

virtual System::String Aspose::Slides::IResourceLoadingArgs::get_Uri()=0

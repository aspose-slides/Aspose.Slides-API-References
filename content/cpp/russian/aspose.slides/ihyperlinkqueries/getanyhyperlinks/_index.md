## IHyperlinkQueries::GetAnyHyperlinks() метод

Получить все [IHyperlinkContainer](../../ihyperlinkcontainer/) подобъекты, которые содержат не null HyperlinkMouseOver. С помощью данного объекта [IHyperlinkContainer](../../ihyperlinkcontainer/) вы можете управлять его гиперссылкой (чтение, обновление или удаление). Смотрите интерфейс [IHyperlinkContainer](../../ihyperlinkcontainer/).

```cpp
virtual System::SharedPtr<System::Collections::Generic::IList<System::SharedPtr<IHyperlinkContainer>>> Aspose::Slides::IHyperlinkQueries::GetAnyHyperlinks()=0
```

### Возвращаемое значение

Все [IHyperlinkContainer](../../ihyperlinkcontainer/) подобъекты, которые содержат не null HyperlinkMouseOver

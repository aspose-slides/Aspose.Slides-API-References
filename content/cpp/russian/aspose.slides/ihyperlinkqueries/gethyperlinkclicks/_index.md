## IHyperlinkQueries::GetHyperlinkClicks() метод

Получить все [IHyperlinkContainer](../../ihyperlinkcontainer/) подобъекты, которые содержат не null HyperlinkClick. С помощью заданного [IHyperlinkContainer](../../ihyperlinkcontainer/) объекта вы можете управлять его гиперссылкой (чтение, обновление или удаление). См. [IHyperlinkContainer](../../ihyperlinkcontainer/) интерфейс.

```cpp
virtual System::SharedPtr<System::Collections::Generic::IList<System::SharedPtr<IHyperlinkContainer>>> Aspose::Slides::IHyperlinkQueries::GetHyperlinkClicks()=0
```

### Возвращаемое значение

Все [IHyperlinkContainer](../../ihyperlinkcontainer/) подобъекты, которые содержат не null HyperlinkClick

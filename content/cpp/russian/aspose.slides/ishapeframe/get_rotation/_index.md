## IShapeFrame::get_Rotation() метод


Возвращает количество градусов, на которое кадр повернут вокруг оси z. Положительное значение указывает на вращение по часовой стрелке; отрицательное значение указывает на вращение против часовой стрелки. Только для чтения **float**.

```cpp
virtual float Aspose::Slides::IShapeFrame::get_Rotation()=0
```

## ShapeStyle::set_FillStyleIndex(int16_t) метод

Устанавливает индекс столбца заполнения фигуры в матрицах стилей. 0 означает отсутствие заполнения, положительное значение - индекс в стилях заполнения темы, отрицательное значение - индекс в стилях фона темы. Записывать **int16_t**.

void Aspose::Slides::ShapeStyle::set_FillStyleIndex(int16_t value) override

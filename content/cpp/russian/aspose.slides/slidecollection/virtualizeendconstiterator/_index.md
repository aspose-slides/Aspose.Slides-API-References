## SlideCollection::virtualizeEndConstIterator() const метод

Получает итератор, указывающий непосредственно после последнего элемента (если он есть) const-квалифицированного экземпляра коллекции.

```cpp
virtualized_iterator * Aspose::Slides::SlideCollection::virtualizeEndConstIterator() const override
```

### Возврат значения

Итератор, указывающий непосредственно после последнего элемента (если он есть) const-квалифицированного экземпляра коллекции. Предоставляет реализацию const-итератора, указывающего на конец контейнера.

Новый объект итератора.

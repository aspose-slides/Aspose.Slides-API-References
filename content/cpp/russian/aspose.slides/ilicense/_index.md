## Примечания

В этом примере будет выполнена попытка найти файл лицензии с именем MyLicense.lic в папке, содержащей компонент, в папке, содержащей вызывающую сборку, в папке входной сборки и затем во встроенных ресурсах вызывающей сборки. 
```cpp
auto license = MakeObject<License>();
license->SetLicense(u"MyLicense.lic");
```

## RowCollection::virtualizeEndConstIterator() const метод

Возвращает итератор, указывающий сразу после последнего элемента (если он есть) константно-квалифицированного экземпляра коллекции.


### Возвращаемое значение

Итератор, указывающий сразу после последнего элемента (если он есть) константно-квалифицированного экземпляра коллекции. Предоставляет реализацию константного итератора до конца контейнера.

Ново созданный объект итератора.

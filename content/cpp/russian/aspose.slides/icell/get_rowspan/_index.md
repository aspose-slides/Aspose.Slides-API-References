## ICell::get_RowSpan() метод


Возвращает количество строк, которые охватывает объединённая ячейка. Это используется в сочетании с атрибутом vMerge у других ячеек для указания начальной ячейки горизонтального объединения. Только для чтения **int32_t**.

```cpp
virtual int32_t Aspose::Slides::ICell::get_RowSpan()=0
```

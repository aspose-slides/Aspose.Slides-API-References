## IBackdrop3DScene::set_UpVector(System::ArrayPtr\<float\>) метод

Задает вектор, представляющий направление вверх. Точнее, этот атрибут определяет вектор, представляющий направление вверх относительно поверхности плоскости заднего фона. Вектор задаётся массивом из 3 значений типа **float**, которые определяют координаты X, Y и Z. Записать **float**[].

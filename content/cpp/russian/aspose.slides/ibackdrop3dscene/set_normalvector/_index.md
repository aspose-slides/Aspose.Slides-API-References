## IBackdrop3DScene::set_NormalVector(System::ArrayPtr\<float\>) метод

Устанавливает вектор нормали. Точнее, этот атрибут определяет вектор, нормальный к плоскости заднего фона. Вектор представляется массивом из 3 значений типа float, определяющих координаты X, Y и Z. Записать **float**[].

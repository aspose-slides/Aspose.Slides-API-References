## FontStyleType enum

Определяет стили форматирования, которые можно применить к тексту.

```cpp
enum class FontStyleType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Regular | 0 | Обычный текст. |
| Bold | 1 | Жирный текст. |
| Italic | 2 | Курсивный текст. |
| Underline | 4 | Подчёркнутый текст. |
| Strikeout | 8 | Текст, зачёркнутый горизонтальной линией. |

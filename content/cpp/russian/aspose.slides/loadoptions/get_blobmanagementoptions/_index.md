## LoadOptions::get_BlobManagementOptions() метод

Представляет параметры, которые могут использоваться для управления поведением обработки бинарных крупных объектов (BLOBs), например использованием временных файлов или максимальным количеством байт BLOBs в памяти. Эти параметры предназначены для настройки оптимального соотношения производительность/потребление памяти для конкретной среды или требований.

```cpp
System::SharedPtr<IBlobManagementOptions> Aspose::Slides::LoadOptions::get_BlobManagementOptions() override
```

## Примечания

Бинарный крупный объект (BLOB) — это бинарные данные, хранящиеся как единый объект, то есть BLOB может быть аудио, видео или самой презентацией.

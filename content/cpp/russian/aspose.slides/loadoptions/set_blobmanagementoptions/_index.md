## LoadOptions::set_BlobManagementOptions(System::SharedPtr\<IBlobManagementOptions\>) метод

Представляет параметры, которые могут использоваться для управления поведением обработки Binary Large Objects (BLOBs), например использованием временных файлов или максимальным количеством байт BLOBs в памяти. Эти параметры предназначены для настройки оптимального соотношения производительность/потребление памяти для конкретной среды или требований.

```cpp
void Aspose::Slides::LoadOptions::set_BlobManagementOptions(System::SharedPtr<IBlobManagementOptions> value) override
```

## Примечания

Binary Large Object (BLOB) — это бинарные данные, хранящиеся как единый объект, т.е. BLOB может быть аудио, видео или самой презентацией.

## LoadOptions::get_OnlyLoadDocumentProperties() метод


Этот параметр имеет смысл, если файл презентации защищён паролем. Значение true означает, что из зашифрованного файла презентации должны быть загружены только свойства документа, а пароль должен быть проигнорирован. Значение false означает, что вся зашифрованная презентация должна быть загружена с использованием правильного пароля. Если презентация не зашифрована, то значение свойства всегда игнорируется. Если свойства документа зашифрованного файла не являются публичными и значение свойства равно true, то свойства документа нельзя загрузить, и будет выброшено исключение. Чтение **bool**.

## IGlobalLayoutSlideCollection класс

Представляет коллекцию всех макетных слайдов в презентации. Расширяет интерфейс [ILayoutSlideCollection](../ilayoutslidecollection/) методами для добавления/клонирования макетных слайдов в контексте объединения отдельных коллекций макетных слайдов мастера.

```cpp
class IGlobalLayoutSlideCollection : public virtual Aspose::Slides::ILayoutSlideCollection
```

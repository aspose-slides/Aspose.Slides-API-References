## ITextSearchOptions::set_IncludeNotes(bool) метод

Установите **true**, чтобы включить текст, содержащийся в заметках слайдов, при выполнении поиска текста, замены или подсветки. По умолчанию значение **false**.

virtual void Aspose::Slides::ITextSearchOptions::set_IncludeNotes(bool value)=0

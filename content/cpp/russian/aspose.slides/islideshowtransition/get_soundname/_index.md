## ISlideShowTransition::get_SoundName() метод

Указывает читаемое человеком имя звука перехода. Свойство [ISlideShowTransition::set_Sound](../set_sound/) должно быть присвоено для получения или установки имени звука. Читает [System::String](../../../system/string/).

```cpp
virtual System::String Aspose::Slides::ISlideShowTransition::get_SoundName()=0
```

## Примечания

Это имя появляется в пользовательском интерфейсе PowerPoint при ручной настройке звука перехода.

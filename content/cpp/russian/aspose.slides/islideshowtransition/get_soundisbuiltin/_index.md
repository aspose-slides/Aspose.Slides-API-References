## ISlideShowTransition::get_SoundIsBuiltIn() метод

Указывает, является ли данный звук встроенным. Если этот атрибут установлен в true, генерирующее приложение получает уведомление о необходимости проверить атрибут name, указанный для этого звука в его списке встроенных звуков, и при необходимости может отобразить пользовательское имя или пользовательский интерфейс. Возвращает **bool**.

```cpp
virtual bool Aspose::Slides::ISlideShowTransition::get_SoundIsBuiltIn()=0
```

## ISlideShowTransition::set_SoundName(System::String) метод


Указывает человекочитаемое имя для звука перехода. [ISlideShowTransition::set_Sound](../set_sound/) должен быть присвоен для получения или установки имени звука. Записывает [System::String](../../../system/string/).

```cpp
virtual void Aspose::Slides::ISlideShowTransition::set_SoundName(System::String value)=0
```

## Примечания


Это имя отображается в пользовательском интерфейсе PowerPoint при ручной настройке звука перехода.

## ISlideShowTransition::set_AdvanceAfterTime(uint32_t) метод

Указывает время в миллисекундах, после которого должен начаться переход. Этот параметр может использоваться вместе с атрибутом advClick. Если этот атрибут не указан, считается, что автоматическое продвижение не будет происходить. Записывает **uint32_t**.

```cpp
virtual void Aspose::Slides::ISlideShowTransition::set_AdvanceAfterTime(uint32_t value)=0
```

## ISlideShowTransition::set_Duration(int32_t) метод


Устанавливает продолжительность эффекта перехода слайда в миллисекундах. Запишите **int32_t**.

```cpp
virtual void Aspose::Slides::ISlideShowTransition::set_Duration(int32_t value)=0
```

## Примечания


Соответствует атрибуту **p14:dur** элемента **p:transition** в схеме PresentationML. Если не задано, продолжительность определяется автоматически на основе свойства [Speed](../) и типа перехода.

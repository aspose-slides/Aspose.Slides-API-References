## ISlideShowTransition::set_SoundIsBuiltIn(bool) метод

Указывает, является ли этот звук встроенным. Если этот атрибут установлен в true, то приложение-генератор получает сигнал проверить атрибут name, указанный для этого звука, в его списке встроенных звуков и при необходимости отобразить пользовательское имя или пользовательский интерфейс. Записывает **bool**.

```cpp
virtual void Aspose::Slides::ISlideShowTransition::set_SoundIsBuiltIn(bool value)=0
```

## ISlideShowTransition::get_Duration() метод

Получает продолжительность эффекта перехода слайда в миллисекундах. Читает **int32_t**.

```cpp
virtual int32_t Aspose::Slides::ISlideShowTransition::get_Duration()=0
```

## Примечания

Соответствует атрибуту **p14:dur** элемента **p:transition** в схеме PresentationML. Если не установлен, продолжительность определяется автоматически на основе свойства [Speed](../) и типа перехода.

## IBlobManagementOptions::set_IsTemporaryFilesAllowed(bool) метод

Это свойство определяет, могут ли быть созданы временные файлы при работе с BLOBs, что значительно снижает потребление памяти, но требует разрешения на создание файлов.

```cpp
virtual void Aspose::Slides::IBlobManagementOptions::set_IsTemporaryFilesAllowed(bool value)=0
```

## Примечания

Все файлы будут удалены после завершения работы с презентацией.

## IBlobManagementOptions::get_IsTemporaryFilesAllowed() метод

Это свойство определяет, можно ли создавать временные файлы при работе с BLOB, что значительно уменьшает потребление памяти, но требует разрешения на создание файлов.

```cpp
virtual bool Aspose::Slides::IBlobManagementOptions::get_IsTemporaryFilesAllowed()=0
```

## Примечания

Все файлы будут удалены после завершения работы с презентацией.

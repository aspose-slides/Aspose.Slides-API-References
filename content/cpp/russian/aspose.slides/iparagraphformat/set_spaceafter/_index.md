## IParagraphFormat::set_SpaceAfter(float) метод


Устанавливает количество пространства после последней строки в абзаце без наследования. Положительное значение задаёт процент от размера шрифта, который должен составлять пробел. Отрицательное значение задаёт размер пробела в пунктах. Записывается **float**.

virtual void Aspose::Slides::IParagraphFormat::set_SpaceAfter(float value)=0

## IParagraphFormat::get_SpaceAfter() метод

Возвращает количество пространства после последней строки в абзаце без наследования. Положительное значение задаёт процент от размера шрифта, который должен составлять пробел. Отрицательное значение указывает размер пробела в пунктах. Читает **float**.

```cpp
virtual float Aspose::Slides::IParagraphFormat::get_SpaceAfter()=0
```

## IParagraphFormat::set_SpaceBefore(float) метод

Устанавливает количество пространства перед первой строкой в абзаце без наследования. Положительное значение задает процент от размера шрифта, который должен составлять пробел. Отрицательное значение задает размер пробела в пунктах. Записывается **float**.

virtual void Aspose::Slides::IParagraphFormat::set_SpaceBefore(float value)=0

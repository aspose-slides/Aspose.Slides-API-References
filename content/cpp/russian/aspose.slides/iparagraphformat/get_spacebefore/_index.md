## IParagraphFormat::get_SpaceBefore() метод

Возвращает количество пространства перед первой строкой в абзаце без наследования. Положительное значение указывает процент от размера шрифта, который должен занимать пробел. Отрицательное значение указывает размер пробела в пунктах. Возвращаемый тип **float**.

```cpp
virtual float Aspose::Slides::IParagraphFormat::get_SpaceBefore()=0
```

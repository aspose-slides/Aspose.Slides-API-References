## Presentation::get_LayoutSlides() метод

Возвращает список всех шаблонных слайдов, определённых в презентации. Только для чтения [IGlobalLayoutSlideCollection](../../igloballayoutslidecollection/).

```cpp
System::SharedPtr<IGlobalLayoutSlideCollection> Aspose::Slides::Presentation::get_LayoutSlides() override
```

## Примечания

Вы можете получить доступ к альтернативному API для добавления/вставки/удаления/клонирования шаблонных слайдов, используя свойство [IMasterSlide::get_LayoutSlides](../../imasterslide/get_layoutslides/).

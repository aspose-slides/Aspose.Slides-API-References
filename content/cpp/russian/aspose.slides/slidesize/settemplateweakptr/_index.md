## SlideSize::SetTemplateWeakPtr(uint32_t) метод

Устанавливает n-й аргумент шаблона как слабый указатель (вместо shared). Позволяет переключать указатели в контейнерах в режим слабых указателей.

```cpp
void Aspose::Slides::SlideSize::SetTemplateWeakPtr(uint32_t argument) override
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| argument | **uint32_t** | Index of template argument to treat as weak poiner. |

## IHyperlink::get_TargetSlide() метод

Если HyperlinkEx указывает на конкретный слайд, возвращает этот слайд. Если свойство становится не null, то свойство ExternalUrl становится null. Только для чтения [ISlide](../../islide/).

```cpp
virtual System::SharedPtr<ISlide> Aspose::Slides::IHyperlink::get_TargetSlide()=0
```

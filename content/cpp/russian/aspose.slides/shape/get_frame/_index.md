## Примечания

Значение каждого свойства возвращённого [IShapeFrame](../../ishapeframe/) экземпляра не является неопределённым (не NaN или NotDefined). Значение каждого свойства назначенного [IShapeFrame](../../ishapeframe/) экземпляра должно быть не неопределённым (не NaN или NotDefined). Вы можете задавать неопределённые значения для свойств экземпляра RawFrame.

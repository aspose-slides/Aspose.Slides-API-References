## Shape::set_Frame(System::SharedPtr\<IShapeFrame\>) метод


Устанавливает свойства рамки фигуры. Запишите [IShapeFrame](../../ishapeframe/).

```cpp
void Aspose::Slides::Shape::set_Frame(System::SharedPtr<IShapeFrame> value) override
```

## Примечания


Значение каждого свойства возвращённого [IShapeFrame](../../ishapeframe/) экземпляра не является неопределённым (не NaN и не NotDefined). Значение каждого свойства назначенного [IShapeFrame](../../ishapeframe/) экземпляра должно быть определённым (не NaN и не NotDefined). Вы можете задавать неопределённые значения свойствам экземпляра RawFrame.

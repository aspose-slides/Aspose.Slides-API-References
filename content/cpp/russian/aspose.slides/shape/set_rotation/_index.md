## Shape::set_Rotation(float) метод

Устанавливает число градусов, на которое указанная shape вращается вокруг оси z. Положительное значение указывает на вращение по часовой стрелке; отрицательное значение указывает на вращение против часовой стрелки. Запишите **float**.

```cpp
void Aspose::Slides::Shape::set_Rotation(float value) override
```

## Примечания

Возвращаемое значение всегда определено (не является std::numeric_limits<float>::quiet_NaN()). Присваиваемое значение должно быть определено (не является std::numeric_limits<float>::quiet_NaN()). Вы можете устанавливать неопределённые значения для свойств экземпляра RawFrame.

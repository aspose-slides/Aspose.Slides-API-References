## Shape::set_Y(float) метод


Задаёт координату y верхнего левого угла формы, измеряемую в пунктах. Запишите **float**.

```cpp
void Aspose::Slides::Shape::set_Y(float value) override
```

## Примечания


Возвращаемое значение всегда определено и никогда не [std::numeric_limits<float>::quiet_NaN()](../). Назначаемое значение также должно быть определено; назначайте [std::numeric_limits<float>::quiet_NaN()](../) только свойствам экземпляра **RawFrame**.

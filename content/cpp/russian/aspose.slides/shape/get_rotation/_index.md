## Shape::get_Rotation() метод

Возвращает количество градусов, на которое указанная фигура повёрнута вокруг оси z. Положительное значение указывает на вращение по часовой стрелке; отрицательное значение указывает на вращение против часовой стрелки. Читается **float**.

```cpp
float Aspose::Slides::Shape::get_Rotation() override
```

## Примечания

Возвращаемое значение всегда определено (не является std::numeric_limits<float>::quiet_NaN()). Присваиваемое значение должно быть определено (не std::numeric_limits<float>::quiet_NaN()). Вы можете задавать неопределённые значения для свойств экземпляра RawFrame.

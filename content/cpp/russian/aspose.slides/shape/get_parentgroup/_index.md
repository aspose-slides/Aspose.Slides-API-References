## Shape::get_ParentGroup() метод

Возвращает родительский объект [GroupShape](../../groupshape/), если фигура сгруппирована. В противном случае возвращает null. Только для чтения [IGroupShape](../../igroupshape/).

```cpp
System::SharedPtr<IGroupShape> Aspose::Slides::Shape::get_ParentGroup() override
```

## DocumentProperties::set_HyperlinksChanged(bool) метод

Указывает, что одна или несколько гиперссылок в этой части были обновлены исключительно в этой части производителем. Следующий производитель, открывающий этот документ, должен обновить отношения гиперссылок новыми гиперссылками, указанными в этой части. Записать **bool**.

```cpp
void Aspose::Slides::DocumentProperties::set_HyperlinksChanged(bool value) override
```

## DocumentProperties::get_ScaleCrop() метод

Указывает режим отображения миниатюры документа. Установите этот элемент в **true**, чтобы включить масштабирование миниатюры документа для отображения. Установите этот элемент в **false**, чтобы включить обрезку миниатюры документа, показывая только те части, которые подходят для отображения. Чтение **bool**.

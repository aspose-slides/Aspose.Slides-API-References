## DocumentProperties::set_ScaleCrop(bool) метод


Указывает режим отображения миниатюры документа. Установите этот элемент в **true**, чтобы включить масштабирование миниатюры документа под дисплей. Установите этот элемент в **false**, чтобы включить обрезку миниатюры документа, показывая только те разделы, которые помещаются в дисплей. Запишите **bool**.

void Aspose::Slides::DocumentProperties::set_ScaleCrop(bool value) override

## IProtectionManager::get_IsEncrypted() метод

Возвращает значение, указывающее, зашифровано ли данный экземпляр. Только для чтения **bool**.

```cpp
virtual bool Aspose::Slides::IProtectionManager::get_IsEncrypted()=0
```

## Примечания

**true** если презентация была загружена из зашифрованного файла или [Encrypt(String)](../encrypt/) метод был вызван; в противном случае **false**.

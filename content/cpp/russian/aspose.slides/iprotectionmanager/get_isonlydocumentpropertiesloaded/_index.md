## IProtectionManager::get_IsOnlyDocumentPropertiesLoaded() метод

Это свойство имеет смысл, если файл презентации защищен паролем и свойства документа этого файла являются публичными. Значение true означает, что из зашифрованного файла презентации загружаются только свойства документа без использования пароля. Значение false означает, что полностью зашифрованная презентация загружается с использованием правильного пароля, а не только свойства документа. Если презентация не зашифрована, то значение свойства всегда равно false. Если свойства документа зашифрованного файла не являются публичными, то значение свойства всегда равно false. Если PresentationEx.EncryptDocumentProperties равно true, то значение свойства IsOnlyDocumentPropertiesLoaded всегда равно false. Только для чтения **bool**.

```cpp
virtual bool Aspose::Slides::IProtectionManager::get_IsOnlyDocumentPropertiesLoaded()=0
```

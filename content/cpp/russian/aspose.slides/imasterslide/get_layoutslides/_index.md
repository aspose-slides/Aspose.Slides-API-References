## IMasterSlide::get_LayoutSlides() метод

Возвращает коллекцию дочерних слайдов макета для этого главного слайда. Только для чтения [IMasterLayoutSlideCollection](../../imasterlayoutslidecollection/).

```cpp
virtual System::SharedPtr<IMasterLayoutSlideCollection> Aspose::Slides::IMasterSlide::get_LayoutSlides()=0
```

## Примечания

Вы можете получить доступ к альтернативному API для добавления/вставки/удаления/клонирования слайдов макета, используя свойство [IPresentation::get_LayoutSlides()](../../ipresentation/get_layoutslides/).

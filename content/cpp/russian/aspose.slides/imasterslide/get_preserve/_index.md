## IMasterSlide::get_Preserve() метод

Определяет, будет ли соответствующий мастер удалён, когда удалены все слайды, следующие за этим мастером. Примечание: [Aspose.Slides](../../) никогда не удалит неиспользуемый мастер сам по себе; чтобы действительно удалить неиспользуемые мастера, вызовите [IMasterSlideCollection::RemoveUnused](../../imasterslidecollection/removeunused/). Чтение **bool**.

```cpp
virtual bool Aspose::Slides::IMasterSlide::get_Preserve()=0
```

## IMasterSlide::set_Preserve(bool) метод


Определяет, будет ли соответствующий мастер удалён, когда все слайды, следующие за этим мастером, удалены. Примечание: [Aspose.Slides](../../) никогда сам не удалит неиспользуемый мастер, чтобы действительно удалить неиспользуемые мастера, вызовите [IMasterSlideCollection::RemoveUnused](../../imasterslidecollection/removeunused/) Write **bool**.

virtual void Aspose::Slides::IMasterSlide::set_Preserve(bool value)=0

## TextAutofitType enum

Представляет режим автоадаптации текста.

```cpp
enum class TextAutofitType : int8_t
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| NotDefined | -1 | Не определено. |
| None | 0 | Без автоадаптации. |
| Normal | 1 | Обычная автоадаптация. Размер шрифта и межстрочный интервал будут уменьшены, чтобы поместиться в форму. |
| Shape | 2 | [Shape](../shape/) автоадаптация. Размер [Shape](../shape/) будет изменён, чтобы поместить текст. |

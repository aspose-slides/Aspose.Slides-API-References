## FillType перечисление

Указывает тип внутренней заливки различных визуальных объектов.

```cpp
enum class FillType : int8_t
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| NotDefined | -1 | Тип заливки не определён. |
| NoFill | 0 | Заливка не применяется. |
| Solid | 1 | Заполнено сплошным цветом. |
| Gradient | 2 | Заливка градиентная. |
| Pattern | 3 | Для заполнения объекта используется повторяющийся узор. |
| Picture | 4 | Для заполнения объекта используется одно изображение. |
| Group | 5 | Визуальный объект наследует свойства заливки от группы. |

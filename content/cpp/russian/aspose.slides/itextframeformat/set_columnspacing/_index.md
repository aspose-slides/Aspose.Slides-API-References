## ITextFrameFormat::set_ColumnSpacing(double) метод

Устанавливает расстояние между текстовыми колонками в области текста (в пунктах). Это должно применяться только когда присутствует более чем 1 колонка. Это значение должно быть положительным числом. В противном случае значение будет установлено в ноль. Записывается **double**.

```cpp
virtual void Aspose::Slides::ITextFrameFormat::set_ColumnSpacing(double value)=0
```

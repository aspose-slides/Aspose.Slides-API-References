## ITextFrameFormat::get_ColumnCount() метод

Возвращает число столбцов в области текста. Это значение должно быть положительным числом. В противном случае значение будет установлено в ноль. Значение 0 означает неопределённое значение. Читает **int32_t**.

virtual int32_t Aspose::Slides::ITextFrameFormat::get_ColumnCount()=0

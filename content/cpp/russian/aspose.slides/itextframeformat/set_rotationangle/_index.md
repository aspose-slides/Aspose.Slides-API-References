## ITextFrameFormat::set_RotationAngle(float) метод

Указывает пользовательский угол поворота, применяемый к тексту внутри ограничивающего прямоугольника. Если он не указан, используется поворот соответствующей фигуры. Если указан, он применяется независимо от фигуры. То есть фигура может иметь собственный поворот, дополнительно к повороту самого текста. Получаемое значение визуального поворота текста учитывается из этого свойства и заранее определённого вертикального типа в свойстве TextVerticalType. Запишите **float**.

```cpp
virtual void Aspose::Slides::ITextFrameFormat::set_RotationAngle(float value)=0
```

## Примечания

Рассмотрите случай, когда у фигуры применяется поворот на 90 градусов по часовой стрелке. Кроме того, сам текстовый блок имеет поворот на -90 градусов против часовой стрелки. В результате фигура будет выглядеть повернутой, но текст внутри неё будет казаться ненаправленным, как будто его не вращали.

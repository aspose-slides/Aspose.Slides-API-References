## ITextFrameFormat::get_ColumnSpacing() метод

Возвращает расстояние между текстовыми колонками в текстовой области (в пунктах). Это применимо только тогда, когда присутствует более одной колонки. Значение должно быть положительным числом. В противном случае значение будет установлено в ноль. Читать **double**.

```cpp
virtual double Aspose::Slides::ITextFrameFormat::get_ColumnSpacing()=0
```

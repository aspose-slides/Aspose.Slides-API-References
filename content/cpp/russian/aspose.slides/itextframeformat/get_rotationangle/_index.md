## ITextFrameFormat::get_RotationAngle() метод


Указывает пользовательский угол вращения, применяемый к тексту внутри ограничивающего прямоугольника. Если он не указан, используется вращение сопутствующей формы. Если он указан, то он применяется независимо от формы. То есть форма может иметь применённое вращение, дополнительно к вращению, применяемому к самому тексту. Полученное значение визуального вращения текста суммируется из этого свойства и предопределённого вертикального типа в свойстве TextVerticalType. Читать **float**.

```cpp
virtual float Aspose::Slides::ITextFrameFormat::get_RotationAngle()=0
```

## Примечания


Рассмотрим случай, когда к форме применено вращение на 90 градусов по часовой стрелке. В дополнение к этому к самому текстовому блоку применяется вращение на -90 градусов против часовой стрелки. Тогда результирующая форма будет выглядеть повернутой, но текст внутри неё будет казаться неповоротным.

## ITextFrameFormat::set_ColumnCount(int32_t) метод

Задаёт количество столбцов в текстовой области. Это значение должно быть положительным числом. В противном случае значение будет установлено в ноль. Значение 0 означает неопределённое значение. Записывается **int32_t**.

virtual void Aspose::Slides::ITextFrameFormat::set_ColumnCount(int32_t value)=0

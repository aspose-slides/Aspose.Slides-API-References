## Tab::CompareTo(System::SharedPtr\<ITab\>) метод

Сравнивает текущий экземпляр с другим объектом того же типа.

```cpp
int32_t Aspose::Slides::Tab::CompareTo(System::SharedPtr<ITab> obj) override
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| obj | [System::SharedPtr](../../../system/sharedptr/)\<[ITab](../../itab/)\> | Объект для сравнения с этим экземпляром. |

### Возвращаемое значение

32-битное целое число, указывающее относительный порядок сравниваемых объектов. Возвращаемое значение имеет следующие значения: * < 0 - Этот экземпляр меньше, чем obj.
* = 0 - Этот экземпляр равен obj.
* > 0 - Этот экземпляр больше, чем obj.

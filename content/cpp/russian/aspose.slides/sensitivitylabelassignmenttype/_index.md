## SensitivityLabelAssignmentType перечисление

Представляет метод назначения для метки чувствительности.

```cpp
enum class SensitivityLabelAssignmentType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Standard | 0 | Используется для любой метки чувствительности, которая не была применена пользователем напрямую. Это включает любые метки по умолчанию, автоматически применяемые метки. |
| Privileged | 1 | Используется для любой метки чувствительности, которая была применена пользователем напрямую. Это включает любые вручную применённые метки чувствительности, а также рекомендованное или обязательное маркирование или любую функцию, где пользователь решает, какую метку чувствительности применять. |

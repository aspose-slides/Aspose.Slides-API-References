## IBulletFormat::set_Height(float) метод

Устанавливает высоту маркера абзаца без наследования. Значение std::numeric_limits<float>::quiet_NaN() определяет, что маркер наследует высоту от первой части абзаца. Запишите **float**.

```cpp
virtual void Aspose::Slides::IBulletFormat::set_Height(float value)=0
```

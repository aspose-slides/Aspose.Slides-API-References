## PresentationHeaderFooterManager::SetAllSlideNumbersVisibility(bool) метод


Изменяет видимость всех заполнителей номеров страниц, включая слайды-шаблоны, слайды-раскладки, обычные слайды, шаблоны заметок, слайды заметок и шаблоны раздаточных материалов.

```cpp
void Aspose::Slides::PresentationHeaderFooterManager::SetAllSlideNumbersVisibility(bool isVisible) override
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| isVisible | **bool** | true - делает заполнители номеров страниц видимыми, иначе - скрывает их. |

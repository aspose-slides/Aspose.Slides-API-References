## EmbeddingLevel enum

Представляет лицензированные права на внедрение шрифта.

```cpp
enum class EmbeddingLevel : uint16_t
```

### Значения

| Name | Value | Description |
| --- | --- | --- |
| Installable | 0 | [Fonts](../fonts/) с этой настройкой указывает, что они могут быть внедрены и постоянно установлены на удалённой системе приложением. Пользователь удалённой системы получает такие же права, обязательства и лицензии на этот шрифт, как у оригинального покупателя, и подпадает под те же условия лицензионного соглашения конечного пользователя, авторского права, патента на дизайн и/или товарного знака, что и оригинальный покупатель. |
| Restricted | 2 | [Fonts](../fonts/) с установленным только этим битом не должны изменяться, внедряться или передаваться каким-либо образом без предварительного получения разрешения от законного владельца. |
| PreviewPrint | 4 | Когда этот бит установлен, шрифт может быть внедрён и временно загружен на удалённую систему. Документы, содержащие шрифты Preview & Print, должны открываться «только для чтения»; вносимые изменения в документ недоступны. |
| Editable | 8 | Когда этот бит установлен, шрифт может быть внедрён, но должен устанавливаться только временно на других системах. В отличие от шрифтов Preview & Print, документы с шрифтами Editable могут открываться для чтения, редактирование разрешено, и изменения могут сохраняться. |
| NoSubsetting | 256 | Когда этот бит установлен, шрифт не может быть подсечён перед внедрением. Применяются и другие ограничения внедрения, указанные в битах 0-3 и 9. |
| BitmapOnly | 512 | Когда этот бит установлен, могут быть внедрены только растровые изображения, содержащиеся в шрифте. Контурные данные внедрять нельзя. Если в шрифте нет растровых изображений, шрифт считается не внедряемым, и службы внедрения завершатся неудачей. |

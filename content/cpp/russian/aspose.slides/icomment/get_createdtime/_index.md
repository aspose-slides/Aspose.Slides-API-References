## IComment::get_CreatedTime() метод


Возвращает время создания комментария. Установка этого свойства в значение [DateTime::MinValue](../../../system/datetime/minvalue/) означает, что время комментария не задано. См. [System::DateTime](../../../system/datetime/).

```cpp
virtual System::DateTime Aspose::Slides::IComment::get_CreatedTime()=0
```

## Примечания


[Comment](../../comment/) время является необязательным параметром.

## IComment::set_CreatedTime(System::DateTime) метод

Устанавливает время создания комментария. Установка этого свойства в значение [DateTime::MinValue](../../../system/datetime/minvalue/) означает, что время комментария не задано. Запишите [System::DateTime](../../../system/datetime/).

```cpp
virtual void Aspose::Slides::IComment::set_CreatedTime(System::DateTime value)=0
```

## Примечания

[Comment](../../comment/) время является необязательным параметром.

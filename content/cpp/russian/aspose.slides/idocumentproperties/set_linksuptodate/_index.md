## IDocumentProperties::set_LinksUpToDate(bool) метод

Указывает, являются ли гиперссылки в документе актуальными. Установите этот элемент в **true**, чтобы указать, что гиперссылки обновлены. Установите этот элемент в **false**, чтобы указать, что гиперссылки устарели. Запишите **bool**.

virtual void Aspose::Slides::IDocumentProperties::set_LinksUpToDate(bool value)=0

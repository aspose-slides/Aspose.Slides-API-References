## IDocumentProperties::set_ScaleCrop(bool) метод


Указывает режим отображения миниатюры документа. Установите этот элемент в **true**, чтобы включить масштабирование миниатюры документа под дисплей. Установите этот элемент в **false**, чтобы включить обрезку миниатюры документа, отображая только те части, которые помещаются на дисплей. Запишите **bool**.

```cpp
virtual void Aspose::Slides::IDocumentProperties::set_ScaleCrop(bool value)=0
```

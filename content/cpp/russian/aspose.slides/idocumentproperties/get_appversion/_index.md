## IDocumentProperties::get_AppVersion() метод

Возвращает версию приложения. Только для чтения [System::String](../../../system/string/).

```cpp
virtual System::String Aspose::Slides::IDocumentProperties::get_AppVersion()=0
```

## Примечания

Содержимое этого элемента должно быть в форме XX.YYYY, где X и Y представляют числовые значения; в противном случае документ считается не соответствующим требованиям. [Aspose.Slides](../../) представляет её версию в формате XX.YYZZ, где:
XX - основной номер версии YY - вспомогательный номер версии ZZ - номер патча
Например, значение 23.0105 означает версию [Aspose.Slides](../../) 23.1.5.

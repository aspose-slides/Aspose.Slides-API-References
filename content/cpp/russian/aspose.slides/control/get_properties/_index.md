
## Примечания

Примечание: [Aspose.Slides](../../) поддерживает только свойства ActiveX на основе XML. Если свойства хранятся в бинарном формате, это свойство вернет null.

Только для чтения [IControlPropertiesCollection](../../icontrolpropertiescollection/).

## ProtectionManager::get_IsEncrypted() метод


Возвращает значение, указывающее, зашифровано ли этот экземпляр. Только для чтения **bool**.

```cpp
bool Aspose::Slides::ProtectionManager::get_IsEncrypted() override
```

## Примечания


**true** если презентация была загружена из зашифрованного файла или [Encrypt(String)](../../iprotectionmanager/encrypt/) метод был вызван ; в противном случае **false**.

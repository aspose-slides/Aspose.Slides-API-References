## IVideo::get_BinaryData() метод

Возвращает копию данных аудио. В случае большого количества данных рекомендуется использовать метод [IVideo::GetStream](../getstream/), чтобы предотвратить ненужную загрузку данных видео в память или даже OutOfMemoryException. Только для чтения **uint8_t**[].

```cpp
virtual System::ArrayPtr<uint8_t> Aspose::Slides::IVideo::get_BinaryData()=0
```

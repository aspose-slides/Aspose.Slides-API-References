## ILoadOptions::get_BlobManagementOptions() метод


Представляет параметры, которые могут использоваться для управления поведением обработки Binary Large Objects (BLOBs), например использованием временных файлов или максимальным количеством байт BLOBs в памяти. Эти параметры предназначены для установки лучшего соотношения производительности/потребления памяти для конкретной среды или требований.

```cpp
virtual System::SharedPtr<IBlobManagementOptions> Aspose::Slides::ILoadOptions::get_BlobManagementOptions()=0
```

## Замечания


Binary Large Object (BLOB) — это бинарные данные, хранящиеся как единый объект, то есть BLOB может быть аудио, видео или самой презентацией.

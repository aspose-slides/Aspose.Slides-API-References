## ILoadOptions::set_BlobManagementOptions(System::SharedPtr\<IBlobManagementOptions\>) метод

Представляет параметры, которые могут использоваться для управления поведением обработки Binary Large Objects (BLOBs), таких как использование временных файлов или максимальное количество байт BLOB в памяти. Эти параметры предназначены для настройки оптимального соотношения производительность/потребление памяти для конкретной среды или требований.

```cpp
virtual void Aspose::Slides::ILoadOptions::set_BlobManagementOptions(System::SharedPtr<IBlobManagementOptions> value)=0
```

## Примечания

Binary Large Object (BLOB) — это двоичные данные, хранящиеся как единый объект, т.е. BLOB может представлять собой аудио, видео или саму презентацию.

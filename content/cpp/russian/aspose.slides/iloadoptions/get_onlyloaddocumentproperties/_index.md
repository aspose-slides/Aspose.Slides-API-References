## ILoadOptions::get_OnlyLoadDocumentProperties() метод

Это свойство имеет смысл, если файл презентации защищён паролем. Значение true означает, что из зашифрованного файла презентации должны быть загружены только свойства документа, а пароль игнорируется. Значение false означает, что вся зашифрованная презентация должна быть загружена с использованием правильного пароля. Если презентация не зашифрована, то значение свойства всегда игнорируется. Если свойства документа зашифрованного файла не публичны и значение свойства равно true, то свойства документа загрузить нельзя, и будет выброшено исключение. Возвращает **bool**.

```cpp
virtual bool Aspose::Slides::ILoadOptions::get_OnlyLoadDocumentProperties()=0
```

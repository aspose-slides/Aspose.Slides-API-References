## ParagraphCollection::get_IsReadOnly() метод

Получает значение, указывающее, является ли [ICollection](../../../system.collections.generic/icollection/) только для чтения. Только для чтения **bool**.

```cpp
bool Aspose::Slides::ParagraphCollection::get_IsReadOnly()
```

### Возвращаемое значение

true если [ICollection](../../../system.collections.generic/icollection/) только для чтения; в противном случае false.

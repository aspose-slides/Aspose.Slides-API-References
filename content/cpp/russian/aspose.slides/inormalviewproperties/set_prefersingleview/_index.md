## INormalViewProperties::set_PreferSingleView(bool) метод

Указывает, предпочитает ли пользователь видеть одноредовый однообластьный регион на весь экран вместо стандартного обычного представления с тремя областями содержимого. Если включено, приложение может выбрать отображение одной из областей содержимого во всём окне. Запись **bool**.

```cpp
virtual void Aspose::Slides::INormalViewProperties::set_PreferSingleView(bool value)=0
```

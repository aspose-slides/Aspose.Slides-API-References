## INormalViewProperties::get_PreferSingleView() метод

Указывает, предпочитает ли пользователь видеть область содержимого, занимающую всё окно, вместо стандартного обычного вида с тремя областями содержимого. Если включено, приложение может выбрать отображение одной из областей содержимого во всём окне. Чтение **bool**.

```cpp
virtual bool Aspose::Slides::INormalViewProperties::get_PreferSingleView()=0
```

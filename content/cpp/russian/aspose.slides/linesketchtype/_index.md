## LineSketchType перечисление

Представляет тип наброска или эффект, назначенный фигуре.

```cpp
enum class LineSketchType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| NotDefined | -1 | Указывает, что эффект Sketch фигуры не определён. |
| None | 0 | Указывает, что у фигуры нет эффекта Sketch. Это эквивалентно тому, что свойство пусто. |
| Curved | 1 | Указывает, что у фигуры есть эффект Curved, который преобразует каждое ребро фигуры в одну большую плавную кривую. |
| Freehand | 2 | Указывает, что у фигуры есть эффект Freehand, который наиболее точно имитирует нарисованную вручную неровную линию. |
| Scribble | 3 | Указывает, что у фигуры есть эффект Scribble, который имеет преувеличенную осцилляцию, будто нарисован преднамеренно неаккуратно. |

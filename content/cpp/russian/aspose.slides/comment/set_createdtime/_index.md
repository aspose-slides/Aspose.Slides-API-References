## Comment::set_CreatedTime(System::DateTime) метод

Устанавливает время создания комментария. Установка этого свойства в [DateTime::MinValue](../../../system/datetime/minvalue/) означает, что время комментария не задано. Запишите [System::DateTime](../../../system/datetime/).

```cpp
void Aspose::Slides::Comment::set_CreatedTime(System::DateTime value) override
```

## Примечания

[Comment](../) время является необязательным параметром.

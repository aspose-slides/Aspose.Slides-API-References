## BlobManagementOptions::set_MaxBlobsBytesInMemory(uint64_t) метод

Определяет максимальный общий размер (в байтах), который все BLOB могут занимать в памяти. По умолчанию все BLOB загружаются в память; только когда достигается этот предел, применяются альтернативные механизмы (например, временные файлы). Хранение BLOB в памяти максимизирует производительность, но может привести к большому использованию памяти. Используйте это свойство, чтобы адаптировать поведение к вашей среде или требованиям.

```cpp
void Aspose::Slides::BlobManagementOptions::set_MaxBlobsBytesInMemory(uint64_t value) override
```

## Примечания

Это значение игнорируется, если [BlobManagementOptions::set_IsTemporaryFilesAllowed](../set_istemporaryfilesallowed/) установлено в false, поскольку память тогда является единственным доступным местом хранения, и ограничение использования BLOB в памяти не имеет эффекта.

Значение по умолчанию равно 629,145,600 байт (600 MB).

Вы можете установить это свойство в ноль, но небольшое минимальное количество памяти всё равно будет зарезервировано.

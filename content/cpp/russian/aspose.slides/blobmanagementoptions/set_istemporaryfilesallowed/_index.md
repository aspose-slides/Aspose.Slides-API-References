## BlobManagementOptions::set_IsTemporaryFilesAllowed(bool) метод


Это свойство определяет, могут ли создаваться временные файлы при работе с BLOBs, что значительно уменьшает потребление памяти, но требует разрешения на создание файлов.

```cpp
void Aspose::Slides::BlobManagementOptions::set_IsTemporaryFilesAllowed(bool value) override
```

## Примечания

Все файлы будут удалены после завершения работы с презентацией.

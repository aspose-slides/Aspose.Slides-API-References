## ShapeCollection::AddAudioFrameLinked(float, float, float, float, System::String) метод

Создает новый аудиофрейм, связанный с внешним аудиофайлом, и добавляет его в конец коллекции фигур.

```cpp
System::SharedPtr<IAudioFrame> Aspose::Slides::ShapeCollection::AddAudioFrameLinked(float x, float y, float width, float height, System::String fname) override
```

### Параметры

| Parameter | Type | Description |
| --- | --- | --- |
| x | **float** | Координата x нового аудиофрейма в пунктах. |
| y | **float** | Координата y нового аудиофрейма в пунктах. |
| width | **float** | Ширина нового аудиофрейма в пунктах. |
| height | **float** | Высота нового аудиофрейма в пунктах. |
| fname | [System::String](../../../system/string/) | Путь или имя внешнего аудиофайла для ссылки. |

### Возвращаемое значение

Новое созданное [IAudioFrame](../../iaudioframe/).

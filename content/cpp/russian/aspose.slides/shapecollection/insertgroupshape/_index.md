## ShapeCollection::InsertGroupShape(int32_t) метод

Создаёт новую пустую группу фигур и вставляет её в коллекцию фигур по указанному индексу. Рамка группы автоматически подстроится под любые добавленные в неё фигуры.

```cpp
System::SharedPtr<IGroupShape> Aspose::Slides::ShapeCollection::InsertGroupShape(int32_t index) override
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| index | **int32_t** | Нулевой индекс, по которому будет вставлена группа фигур. |

### Возвращаемое значение

Новосозданный [IGroupShape](../../igroupshape/).

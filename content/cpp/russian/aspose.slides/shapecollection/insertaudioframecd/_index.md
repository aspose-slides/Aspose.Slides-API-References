## ShapeCollection::InsertAudioFrameCD(int32_t, float, float, float, float) метод

Создает новый аудио-кадр, связанный с дорожкой CD, и вставляет его в коллекцию фигур по указанному индексу.

```cpp
System::SharedPtr<IAudioFrame> Aspose::Slides::ShapeCollection::InsertAudioFrameCD(int32_t index, float x, float y, float width, float height) override
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| index | **int32_t** | Нулевой (отсчёт с нуля) индекс, по которому следует вставить аудио-кадр. |
| x | **float** | Координата x нового аудио-кадра в пунктах. |
| y | **float** | Координата y нового аудио-кадра в пунктах. |
| width | **float** | Ширина нового аудио-кадра в пунктах. |
| height | **float** | Высота нового аудио-кадра в пунктах. |

### Возвращаемое значение

Созданный [IAudioFrame](../../iaudioframe/).

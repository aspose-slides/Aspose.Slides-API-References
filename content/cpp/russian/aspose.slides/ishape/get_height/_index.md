## IShape::get_Height() метод

Получает высоту фигуры, измеренную в пунктах. Читает **float**.

```cpp
virtual float Aspose::Slides::IShape::get_Height()=0
```

## Замечания

Возвращаемое значение всегда определено и никогда не [std::numeric_limits<float>::quiet_NaN()](../). Присваиваемое значение также должно быть определено; присваивайте [std::numeric_limits<float>::quiet_NaN()](../) только свойствам экземпляра **RawFrame**.

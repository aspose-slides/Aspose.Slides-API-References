## IShape::get_ParentGroup() метод

Возвращает родительский объект [GroupShape](../../groupshape/), если фигура находится в группе. В противном случае возвращает null. Только для чтения [IGroupShape](../../igroupshape/).

```cpp
virtual System::SharedPtr<IGroupShape> Aspose::Slides::IShape::get_ParentGroup()=0
```

## Примечания

Свойство [IShape::get_IsGrouped](../get_isgrouped/) определяет, находится ли фигура в группе.

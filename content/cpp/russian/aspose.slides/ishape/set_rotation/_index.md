## IShape::set_Rotation(float) method

Устанавливает количество градусов, на которое указанная фигура вращается вокруг оси z. Положительное значение указывает на вращение по часовой стрелке; отрицательное значение указывает на вращение против часовой стрелки. Запись **float**.

```cpp
virtual void Aspose::Slides::IShape::set_Rotation(float value)=0
```

## Примечания

Возвращаемое значение всегда определено (не является std::numeric_limits<float>::quiet_NaN()). Присваиваемое значение должно быть определено (не std::numeric_limits<float>::quiet_NaN()). Вы можете задавать неопределённые значения для свойств экземпляра RawFrame.

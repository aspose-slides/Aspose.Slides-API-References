## IShape::set_X(float) метод


Задает координату x верхнего левого угла фигуры, измеряемую в пунктах. Запишите **float**.

```cpp
virtual void Aspose::Slides::IShape::set_X(float value)=0
```

## Примечания


Возвращаемое значение всегда определено и никогда не [std::numeric_limits<float>::quiet_NaN()](../). Присваиваемое значение также должно быть определено; назначайте [std::numeric_limits<float>::quiet_NaN()](../) только свойствам экземпляра **RawFrame**.

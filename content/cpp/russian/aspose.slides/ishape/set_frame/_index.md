## IShape::set_Frame(System::SharedPtr\<IShapeFrame\>) метод

Устанавливает свойства рамки фигуры. Запишите [IShapeFrame](../../ishapeframe/).

```cpp
virtual void Aspose::Slides::IShape::set_Frame(System::SharedPtr<IShapeFrame> value)=0
```

## Примечания

Значение каждого свойства возвращённого [IShapeFrame](../../ishapeframe/) экземпляра не является неопределённым (не NaN и не NotDefined). Значение каждого свойства назначенного [IShapeFrame](../../ishapeframe/) экземпляра должно быть не неопределённым (не должно быть NaN или NotDefined). Вы можете задавать неопределённые значения для свойств экземпляра RawFrame.

## IShape::get_Rotation() метод

Возвращает количество градусов, на которое указанный объект вращается вокруг оси z. Положительное значение указывает на вращение по часовой стрелке; отрицательное значение указывает на вращение против часовой стрелки. Читать **float**.

```cpp
virtual float Aspose::Slides::IShape::get_Rotation()=0
```

## Примечания

Возвращаемое значение всегда определено (не является std::numeric_limits<float>::quiet_NaN()). Присваиваемое значение должно быть определено (не std::numeric_limits<float>::quiet_NaN()). Вы можете задавать неопределённые значения для свойств экземпляра RawFrame.

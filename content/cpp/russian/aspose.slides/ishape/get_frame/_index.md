```cpp
virtual System::SharedPtr<IShapeFrame> Aspose::Slides::IShape::get_Frame()=0
```

## Примечания

Значение каждого свойства возвращённого экземпляра [IShapeFrame](../../ishapeframe/) не является неопределённым (не NaN и не NotDefined). Значение каждого свойства назначенного экземпляра [IShapeFrame](../../ishapeframe/) должно быть не неопределённым (не NaN и не NotDefined). Вы можете задавать неопределённые значения для свойств экземпляра RawFrame.

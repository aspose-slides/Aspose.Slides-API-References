## WarningType enum

Представляет тип предупреждения.

```cpp
enum class WarningType
```

### Значения

| Name | Value | Description |
| --- | --- | --- |
| SourceFileCorruption | 0 | Обнаружена проблема в исходном документе, из-за которой очень вероятно, что документ не сможет быть открыт, если его сохранить в его оригинальном формате. |
| DataLoss | 1 | Текст/диаграмма/изображение или другие данные будут полностью отсутствовать либо в дереве документа после загрузки, либо в созданном документе после сохранения. |
| MajorFormattingLoss | 2 | Значительная потеря форматирования. |
| MinorFormattingLoss | 3 | Незначительная потеря форматирования. |
| CompatibilityIssue | 4 | Это известная проблема, которая помешает открытию документа некоторыми пользовательскими агентами или предыдущими версиями пользовательских агентов. |
| UnexpectedContent | 99 | Некоторые элементы в исходном документе не распознаны (т.е. не поддерживаются); это может вызвать проблемы или привести к потере данных/форматирования. |

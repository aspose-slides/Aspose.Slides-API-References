## Классы

| Класс | Описание |
| --- | --- |
| [IKnownIssueWarningInfo](./iknownissuewarninginfo/) | Представляет предупреждение о известной проблеме, которая не будет исправлена в ближайшем будущем. |
| [INotImplementedWarningInfo](./inotimplementedwarninginfo/) | Представляет предупреждение о известной не реализованной функции, которая не будет реализована в ближайшем будущем. |
| [IObsoletePresLockingBehaviorWarningInfo](./iobsoletepreslockingbehaviorwarninginfo/) | Это предупреждение указывает на использование устаревшего поведения блокировки презентации. |
| [IPresentationSignedWarningInfo](./ipresentationsignedwarninginfo/) | Это предупреждение указывает, что читаемая презентация имеет подпись, и эта подпись будет удалена во время обработки. |
| [IWarningCallback](./iwarningcallback/) | Интерфейс для классов, получающих предупреждение. |
| [IWarningInfo](./iwarninginfo/) | Представляет базовый интерфейс для всех предупреждений. |

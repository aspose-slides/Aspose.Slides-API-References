## BasicSTDIOStreamWrapper::WriteByte(uint8_t) метод

Если режим обёртывания бинарный, записывает в поток указанное беззнаковое 8-битное целое значение, иначе преобразует его в тип char_type и затем записывает результат в поток.

```cpp
virtual void System::IO::BasicSTDIOStreamWrapper<T, typename>::WriteByte(uint8_t value) override
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | **uint8_t** | Значение для записи |

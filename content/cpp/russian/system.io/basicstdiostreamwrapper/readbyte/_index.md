## BasicSTDIOStreamWrapper::ReadByte() метод

Если режим обертывания бинарный, читает один байт из хранилища последнего декодированного символа, иначе читает один символ из потока и преобразует его к типу **uint8_t**.

```cpp
virtual int System::IO::BasicSTDIOStreamWrapper<T, typename>::ReadByte() override
```

### Возвращаемое значение

32-битное целочисленное значение, эквивалентное результату чтения

## TextReader::Peek() метод

Читает один символ из потока, не изменяя курсор чтения потока.

```cpp
virtual int System::IO::TextReader::Peek()
```

### Возвращаемое значение

Считываемый символ закодирован в кодировке UTF-16; если символ представлен двумя кодовыми точками в кодировке UTF-16, возвращается только старший суррогат; если символ не считан, возвращается -1.

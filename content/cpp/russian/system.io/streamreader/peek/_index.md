## StreamReader::Peek() метод

Считывает один символ из потока, не изменяя курсор чтения потока.

```cpp
virtual int System::IO::StreamReader::Peek() override
```

### Возвращаемое значение

Символ, закодированный в кодировке UTF-16; если считываемый символ представлен двумя кодовыми точками в кодировке UTF-16, возвращается только старший суррогат; если символ не был считан, возвращается -1

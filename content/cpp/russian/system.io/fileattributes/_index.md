## FileAttributes enum

Представляет атрибуты каталога или файла.

```cpp
enum class FileAttributes
```

### Values

| Имя | Значение | Описание |
| --- | --- | --- |
| Archive | 32 | Элемент файловой системы является кандидатом на удаление или резервное копирование. |
| Compressed | 2048 | Элемент файловой системы сжат. |
| Device | 64 | Не используется. |
| Directory | 16 | Элемент файловой системы является каталогом. |
| Encrypted | 16384 | Элемент файловой системы зашифрован. |
| Hidden | 2 | Элемент файловой системы скрыт. |
| IntegrityStream | 32768 | Элемент файловой системы поддерживает целостность данных. |
| Normal | 128 | Элемент файловой системы — файл без специальных атрибутов. |
| NotContentIndexed | 8192 | Элемент файловой системы не будет индексироваться службой индексирования содержимого. |
| NoScrubData | 131072 | Элемент файловой системы не включён в сканирование целостности данных. |
| Offline | 4096 | Элемент файловой системы находится в офлайн-режиме, и его данные недоступны сразу. |
| ReadOnly | 1 | Элемент файловой системы доступен только для чтения. |
| ReparsePoint | 1024 | Элемент файловой системы — файл, содержащий точку повторной обработки. |
| SparseFile | 512 | Элемент файловой системы является разрежённым файлом. |
| System | 4 | Элемент файловой системы является системным файлом. |
| Temporary | 256 | Элемент файловой системы является временным файлом. |
| Invalid | 0 | Недопустимое значение. |

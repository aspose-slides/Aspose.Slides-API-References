## Path::HasExtension(const String\&) метод

Определяет, указывает ли указанный путь на файл с расширением.

```cpp
static bool System::IO::Path::HasExtension(const String &path)
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| path | const [String](../../../system/string/)\& | Путь для проверки |

### Возвращаемое значение

true если **path** указывает на файл с расширением, иначе — false

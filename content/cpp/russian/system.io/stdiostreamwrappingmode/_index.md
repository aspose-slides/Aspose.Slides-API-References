## STDIOStreamWrappingMode перечисление

Определяет режим операций ввода/вывода, которые обертки будут выполнять над потоками, похожими на std::iostreams.

```cpp
enum class STDIOStreamWrappingMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Binary | 0 | Режим, позволяющий операциям ввода декодировать данные потока типа char_type в байты и кодировать байты в данные типа char_type для операций вывода. |
| Conversion | 1 | Режим, позволяющий операциям ввода преобразовывать данные потока из типа char_type в тип **uint8_t** и наоборот для операций вывода. |

## Stream::get_CanTimeout() const метод

Возвращает значение, определяющее, может ли текущий поток завершаться по тайм-ауту.

```cpp

```

### Возвращаемое значение

Истина, если текущий поток может завершаться по тайм-ауту, иначе - ложь.

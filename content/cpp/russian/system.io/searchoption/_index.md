## SearchOption enum

Указывает, что поиск должен выполняться только в текущем каталоге или в текущем каталоге и во всех его подкаталогах.

```cpp
enum class SearchOption
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| AllDirectories | 0 | Поиск в текущем каталоге и во всех его подкаталогах. |
| TopDirectoryOnly | 1 | Поиск только в текущем каталоге. |

## File::OpenRead(const String\&) метод

Открывает указанный файл только для чтения, в режиме 'Open' с общим доступом для чтения.

```cpp
static FileStreamPtr System::IO::File::OpenRead(const String &path)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| path | const [String](../../../system/string/)\& | Путь к файлу, который нужно открыть |

### Возвращаемое значение

Объект [FileStream](../../filestream/), связанный с открытым файлом

## File::Exists(const String\&) метод

Определяет, указывает ли указанный путь на существующий файл.

```cpp
static bool System::IO::File::Exists(const String &path)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| path | const [String](../../../system/string/)\& | Путь для проверки |

### Возвращаемое значение

True если **path** указывает на существующий файл, иначе - false

## FileSystemInfo::get_Exists() метод

Определяет, существует ли сущность, на которую указывает путь, представленный текущим объектом.

```cpp
virtual bool System::IO::FileSystemInfo::get_Exists()=0
```

### Возвращаемое значение

True если сущность, на которую указывает путь, представленный текущим объектом, существует, иначе - false

## BasicSTDOStreamWrapper::ReadByte() метод

Если режим обёртывания бинарный, читает один байт из хранилища последнего декодированного символа, иначе читает один символ из потока и преобразует его в тип **uint8_t**. Не поддерживается!

```cpp
virtual int System::IO::BasicSTDOStreamWrapper<T, typename>::ReadByte() override
```

### Возвращаемое значение

32-битное целочисленное значение, эквивалентное результату чтения

## BasicSTDOStreamWrapper::WriteByte(uint8_t) метод

Если режим обёртки бинарный, записывает в поток указанное беззнаковое 8-bit целое значение, в противном случае преобразует его к типу char_type и затем записывает результат в поток.

```cpp
virtual void System::IO::BasicSTDOStreamWrapper<T, typename>::WriteByte(uint8_t value) override
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | **uint8_t** | Значение для записи |

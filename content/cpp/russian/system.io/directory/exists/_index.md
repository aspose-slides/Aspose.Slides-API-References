## Directory::Exists(const String\&) метод

Определяет, указывает ли указанный путь на существующий каталог.

```cpp
static bool System::IO::Directory::Exists(const String &path)
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| path | const [String](../../../system/string/)\& | Путь для проверки |

### Возвращаемое значение

True, если указанный путь указывает на существующий каталог, иначе — false

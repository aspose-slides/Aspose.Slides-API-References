## Directory::Move(const String\&, const String\&) метод

Перемещает указанный объект в новое место. Если перемещаемый объект является каталогом, он перемещается со всем его содержимым.

```cpp
static void System::IO::Directory::Move(const String &sourceDirName, const String &destDirName)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| sourceDirName | const [String](../../../system/string/)\& | Каталог или файл для перемещения |
| destDirName | const [String](../../../system/string/)\& | Новое расположение **sourceDirName** |

```cpp
bool System::IO::Details_DirectoryNotFoundException::Is(const System::TypeInfo &target) const override
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| target | const [System::TypeInfo](../../../system/typeinfo/)\& | [TypeInfo](../../../system/typeinfo/) структура, описывающая тип, против которого проверяется текущий объект. |

## Возвращаемое значение

True, если объект имеет указанный тип или его подкласс, false в противном случае.

## Примечания

Проверьте, представляет ли объект экземпляр типа, описанного targetType. Аналог оператора C# 'is'.

## FileMode enum

Указывает, как файл должен быть открыт.

```cpp
enum class FileMode
```

### Values

| Имя | Значение | Описание |
| --- | --- | --- |
| CreateNew | 1 | Создать новый файл. Если файл уже существует, генерируется исключение. |
| Create | 2 | Создать новый файл. Если файл уже существует, он будет перезаписан. |
| Open | 3 | Открыть существующий файл. Если файл не существует, генерируется исключение. |
| OpenOrCreate | 4 | Открыть существующий файл или создать новый, если он не существует. |
| Truncate | 5 | Открыть существующий файл и обрезать его, чтобы он стал пустым. Если файл не существует, генерируется исключение. |
| Append | 6 | Открыть существующий файл и переместить указатель в конец, или создать новый, если он не существует. |

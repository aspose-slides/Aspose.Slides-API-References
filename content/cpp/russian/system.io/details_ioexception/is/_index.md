## Details_IOException::Is(const System::TypeInfo\&) const метод




```cpp
bool System::IO::Details_IOException::Is(const System::TypeInfo &target) const override
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| target | const [System::TypeInfo](../../../system/typeinfo/)\& | [TypeInfo](../../../system/typeinfo/) структура, описывающая тип, против которого проверяется текущий объект. |

### Возвращаемое значение

True, если объект имеет помеченный тип или его подкласс, иначе false.
## Примечания


Check if object represents an instance of type described by targetType. Analog of C# 'is' operator.

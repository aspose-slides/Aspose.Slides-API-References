## FileInfo::AppendText() метод

Открывает файл, представленный текущим объектом, для записи текста в кодировке UTF-8, в режиме «Append» без совместного доступа.

```cpp
StreamWriterPtr System::IO::FileInfo::AppendText()
```

### Возвращаемое значение

Умный указатель на объект [StreamWriter](../../streamwriter/), связанный с открытым файлом

## FileInfo::OpenText() метод


Открывает существующий файл по пути, представляемому текущим объектом, для чтения текста с использованием кодировки UTF-8 без совместного доступа.

```cpp
StreamReaderPtr System::IO::FileInfo::OpenText()
```


### Возвращаемое значение

Разделяемый указатель на объект [StreamWriter](../../streamwriter/), связанный с открытым файлом

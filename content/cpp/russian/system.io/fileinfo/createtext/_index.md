## FileInfo::CreateText() метод

Создаёт файл в месте, указанном путём, представляемым текущим объектом, и открывает его для записи текста с использованием кодировки UTF-8 без совместного доступа.

```cpp
StreamWriterPtr System::IO::FileInfo::CreateText()
```

### Возвращаемое значение

Умный указатель на объект [StreamWriter](../../streamwriter/), связанный с созданным файлом

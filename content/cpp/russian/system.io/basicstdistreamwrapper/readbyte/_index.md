## BasicSTDIStreamWrapper::ReadByte() метод

Если режим обёртки бинарный, считывает один байт из хранилища последнего декодированного символа, иначе считывает один символ из потока и преобразует его в тип **uint8_t**.

```cpp
virtual int System::IO::BasicSTDIStreamWrapper<T, typename>::ReadByte() override
```

### Возвращаемое значение

32-битное целочисленное значение, эквивалентное результату чтения

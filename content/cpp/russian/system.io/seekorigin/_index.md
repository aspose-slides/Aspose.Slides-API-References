## SeekOrigin перечисление

Указывает позицию отсчёта в потоке, относительно которой задаётся позиция, к которой нужно переместиться.

```cpp
enum class SeekOrigin
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Begin | 0 | Начало потока. |
| Current | 1 | Текущее положение в потоке. |
| End | 2 | Конец потока. |

## FileAccess enum

Указывает тип доступа при открытии файла.

```cpp
enum class FileAccess
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Read | 1 | Доступ только для чтения. |
| Write | 2 | Доступ только для записи. |
| ReadWrite | n/a | Доступ для чтения и записи. |

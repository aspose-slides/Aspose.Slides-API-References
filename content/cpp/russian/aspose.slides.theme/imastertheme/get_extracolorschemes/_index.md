## IMasterTheme::get_ExtraColorSchemes() метод

Возвращает коллекцию дополнительных схем цветов. Эти схемы не влияют на внешний вид презентации, их можно выбрать в качестве основной схемы цветов для слайда. Только для чтения [IExtraColorSchemeCollection](../../iextracolorschemecollection/).

## IOverrideThemeManager::get_IsOverrideThemeEnabled() метод


Определяет, переопределяет ли [OverrideTheme](../../overridetheme/) унаследованную эффективную тему или нет. Чтобы включить [OverrideTheme](../../overridetheme/) для переопределения, используйте методы OverrideTheme.Init*(). Чтобы отключить [OverrideTheme](../../overridetheme/) от переопределения, используйте метод [get_OverrideTheme()](../get_overridetheme/)->Clear(). Только для чтения **bool**.

```cpp
virtual bool Aspose::Slides::Theme::IOverrideThemeManager::get_IsOverrideThemeEnabled()=0
```

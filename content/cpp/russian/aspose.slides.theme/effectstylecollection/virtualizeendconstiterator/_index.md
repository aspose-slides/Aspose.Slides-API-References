## EffectStyleCollection::virtualizeEndConstIterator() const метод

Получает итератор, указывающий сразу после последнего элемента (при наличии) константно-квалифицированного экземпляра коллекции.

Итератор, указывающий сразу после последнего элемента (при наличии) константно-квалифицированного экземпляра коллекции. Предоставляет реализацию константного итератора для конца контейнера.

Новый созданный объект итератора.

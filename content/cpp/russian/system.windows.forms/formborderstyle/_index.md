## FormBorderStyle enum

Определяет, как рисуется граница.

```cpp
enum class FormBorderStyle
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | 0 | Нет границы. |
| FixedSingle | 1 | [Single](../../system/single/) граница. |
| Fixed3D | 2 | Граница в виде 3D. |
| FixedDialog | 3 | Граница в стиле диалога. |
| Sizable | 4 | Изменяемая граница. |
| FixedToolWindow | 5 | Неподвижная граница окна инструмента. |
| SizableToolWindow | 6 | Изменяемая граница окна инструмента. |

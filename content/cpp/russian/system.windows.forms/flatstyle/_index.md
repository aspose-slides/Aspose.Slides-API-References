## FlatStyle перечисление

Определяет, отображается ли элемент управления плоско или трехмерно.

```cpp
enum class FlatStyle
```

### Values

| Имя | Значение | Описание |
| --- | --- | --- |
| Flat | 0 | [Control](../control/) рисуется плоско. |
| Popup | 1 | [Control](../control/) рисуется плоско, если только мышь не переместится над ним, тогда он рисуется в 3D-подобном виде. |
| Standard | 2 | [Control](../control/) рисуется в 3D-подобном виде. |
| System | 3 | Стиль управляется ОС. |

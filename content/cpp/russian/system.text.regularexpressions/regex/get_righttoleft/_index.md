## Regex::get_RightToLeft() метод

Проверяет, выполняется ли сопоставление в режиме справа налево.

```cpp
bool System::Text::RegularExpressions::Regex::get_RightToLeft()
```

### Возвращаемое значение

True если сопоставление выполняется справа налево, false если оно выполняется слева направо.

## IXsltContextVariable::get_IsParam() метод


Возвращает значение, указывающее, является ли переменная параметром Extensible Stylesheet Language Transformations (XSLT). Это может быть параметр таблицы стилей или шаблона.

```cpp
virtual bool System::Xml::Xsl::IXsltContextVariable::get_IsParam()=0
```


### Возвращаемое значение

**true** если переменная является параметром XSLT; в противном случае, **false**.

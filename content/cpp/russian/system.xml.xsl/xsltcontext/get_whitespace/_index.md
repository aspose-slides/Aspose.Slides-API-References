## XsltContext::get_Whitespace() метод


При переопределении в производном классе возвращает значение, указывающее, следует ли включать узлы пробельных символов в вывод.

```cpp
virtual bool System::Xml::Xsl::XsltContext::get_Whitespace()=0
```


### Возвращаемое значение

**true** — проверять узлы пробельных символов в исходном документе для включения в вывод; **false** — не оценивать узлы пробельных символов. По умолчанию **true**.

## XsltSettings::get_EnableDocumentFunction() метод

Возвращает значение, указывающее, следует ли включить поддержку функции XSLT **document()**.

```cpp
bool System::Xml::Xsl::XsltSettings::get_EnableDocumentFunction()
```

### Возвращаемое значение

**true** — поддержка функции XSLT **document()**; **false** — иначе. По умолчанию **false**.

## XsltSettings::get_Default() метод

Возвращает объект [XsltSettings](../) с настройками по умолчанию. Поддержка функции XSLT **document()** и встроенных скриптовых блоков отключена.

```cpp
static SharedPtr<XsltSettings> System::Xml::Xsl::XsltSettings::get_Default()
```

### Возвращаемое значение

Объект [XsltSettings](../) с параметрами [XsltSettings::set_EnableDocumentFunction](../set_enabledocumentfunction/) и [XsltSettings::set_EnableScript](../set_enablescript/), установленными в **false**.

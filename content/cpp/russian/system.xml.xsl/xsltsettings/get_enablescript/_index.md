## XsltSettings::get_EnableScript() метод


Возвращает значение, указывающее, следует ли включать поддержку встроенных блоков скриптов.

```cpp
bool System::Xml::Xsl::XsltSettings::get_EnableScript()
```


### Возвращаемое значение

**true** — поддержка блоков скриптов в стилевых таблицах XSLT; иначе **false**. По умолчанию **false**.

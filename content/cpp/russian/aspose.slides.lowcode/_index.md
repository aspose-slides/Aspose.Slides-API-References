## Классы

| Класс | Описание |
| --- | --- |
| [Collect](./collect/) | Представляет группу методов, предназначенных для сбора объектных моделей разных типов из [Presentation](../aspose.slides/presentation/). |
| [Compress](./compress/) | Представляет группу методов, предназначенных для сжатия [Presentation](../aspose.slides/presentation/). |
| [Convert](./convert/) | Представляет группу методов, предназначенных для преобразования [Presentation](../aspose.slides/presentation/). |
| [ForEach](./foreach/) | Представляет группу методов, предназначенных для перебора различных [Presentation](../aspose.slides/presentation/) объектных моделей. Эти методы могут быть полезны, если вам нужно перебрать и изменить форматирование или содержимое некоторых [Presentation](../aspose.slides/presentation/)' элементов, например изменить форматирование каждой части. |
| [Merger](./merger/) | Представляет группу методов для объединения презентаций PowerPoint одинакового формата в один файл. |

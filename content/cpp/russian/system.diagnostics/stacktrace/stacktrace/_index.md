## StackTrace::StackTrace() конструктор


Создает трассировку стека, описывающую текущее состояние стека.

```cpp
System::Diagnostics::StackTrace::StackTrace()
```

## StackTrace::StackTrace(bool) конструктор


Создает трассировку стека, описывающую текущее состояние стека.

```cpp
System::Diagnostics::StackTrace::StackTrace(bool isFileInfoNeeded)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| isFileInfoNeeded | **bool** | true, чтобы захватить имя файла, номер строки и номер столбца; иначе — false |

## StackTrace::StackTrace(const StackTrace\&) конструктор


Нет копирования.

```cpp
System::Diagnostics::StackTrace::StackTrace(const StackTrace &)=delete
```

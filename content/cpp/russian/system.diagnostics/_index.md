## Структуры

| Структура | Описание |
| --- | --- |
| [Debug](./debug/) | Коллекция методов отладки, позволяющая отправлять информацию об отладке зарегистрированным слушателям. Все функции вывода работают только в [Debug](./debug/). Это статический тип без сервисов экземпляра. Вы никогда не должны создавать его экземпляры каким-либо образом. |
| [Debugger](./debugger/) | [Debugger](./debugger/) интерфейс. Это статический тип без сервисов экземпляра. Вы никогда не должны создавать его экземпляры каким-либо образом. |
| [Trace](./trace/) | Предоставляет интерфейс доступа к трассировке отладчика (если есть). Работает только в режиме [Debug](./debug/). Это статический тип без сервисов экземпляра. Вы никогда не должны создавать его экземпляры каким-либо образом. |

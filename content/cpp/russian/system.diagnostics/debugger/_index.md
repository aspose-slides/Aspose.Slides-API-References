## Структура Debugger

[Debugger](./) интерфейс. Это статический тип без сервисов экземпляра. Вы никогда не должны создавать его экземпляры какими бы ни были способами.

```cpp
class Debugger
```

## Методы

| Метод | Описание |
| --- | --- |
| static void [Break](./break/)() | Инициирует прерывание отладчика (если подключён). |
| static **bool** [get_IsAttached](./get_isattached/)() | Проверяет, подключён ли отладчик к текущему экземпляру приложения. |

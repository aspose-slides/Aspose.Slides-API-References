## TraceLevel перечисление

Указывает, какие сообщения выводить для [System.Diagnostics.Debug](../debug/), [System.Diagnostics.Trace](../trace/) и классов System.Diagnostics.TraceSwitch.

```cpp
enum class TraceLevel
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Off | 0 | Не выводит сообщения трассировки и отладки. |
| Error | 1 | Выводит сообщения обработки ошибок. |
| Warning | 2 | Выводит предупреждения и сообщения обработки ошибок. |
| Info | 3 | Выводит информационные сообщения, предупреждения и сообщения обработки ошибок. |
| Verbose | 4 | Выводит все сообщения отладки и трассировки. |

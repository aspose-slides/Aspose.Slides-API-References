## ISVGOptions::set_DisableLineEndCropping(bool) метод

SVG 1.1 отсутствует возможность задавать отступы для маркеров. [Aspose.Slides](../../../aspose.slides/) движок записи SVG имеет обходное решение этой проблемы: он обрезает конец линии со стрелкой, так что линия не перекрывает маркеры. Эта опция отключает такое поведение. Записывайте **bool**.

```cpp
virtual void Aspose::Slides::Export::ISVGOptions::set_DisableLineEndCropping(bool value)=0
```

## ISVGOptions::get_DisableLineEndCropping() метод

SVG 1.1 отсутствует возможность задавать отступы для маркеров. [Aspose.Slides](../../../aspose.slides/) движок записи SVG имеет обходное решение этой проблемы: он обрезает конец линии со стрелкой, поэтому линия не перекрывает маркеры. Эта опция отключает такое поведение. Читать **bool**.

```cpp
virtual bool Aspose::Slides::Export::ISVGOptions::get_DisableLineEndCropping()=0
```

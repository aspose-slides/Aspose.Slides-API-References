## ISVGOptions::set_DeletePicturesCroppedAreas(bool) метод


Логический флаг указывает, остаются ли обрезанные части частью документа. Если значение true, обрезанные части будут удалены, если false, они будут сериализованы в документе (что может привести к увеличению размера файла) Записать **bool**.

virtual void Aspose::Slides::Export::ISVGOptions::set_DeletePicturesCroppedAreas(bool value)=0

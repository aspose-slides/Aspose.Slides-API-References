## SvgExternalFontsHandling перечисление

Представляет способ обработки внешних шрифтов, используемых для рисования текста.

```cpp
enum class SvgExternalFontsHandling
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| AddLinksToFontFiles | 0 | Добавить ссылки на отдельные файлы шрифтов в секцию style SVG-файла. |
| Embed | 1 | Сохранить данные шрифтов непосредственно в SVG-файл. Пожалуйста, проверьте все лицензионные соглашения на внешние шрифты перед использованием этой опции. |
| Vectorize | 2 | Сохранить весь текст с использованием внешних шрифтов как графику. |

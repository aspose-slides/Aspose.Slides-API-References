## Conformance перечисление

Specifies the conformance class to which the PresentationML document conforms.

```cpp
enum class Conformance
```

### Значения

| Name | Value | Description |
| --- | --- | --- |
| Ecma376_2006 | 0 | Указывает, что документ соответствует ECMA376:2006. |
| Iso29500_2008_Transitional | 1 | Указывает, что документ соответствует переходному классу соответствия ISO/IEC 29500:2008. |
| Iso29500_2008_Strict | 2 | Указывает, что документ соответствует строгому классу соответствия ISO/IEC 29500:2008. |

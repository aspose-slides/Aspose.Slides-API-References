## CompressionLevel перечисление

Указывает уровни сжатия ZIP для файлов OpenXML. Более высокие уровни обеспечивают лучшее сжатие за счет более медленной обработки.

```cpp
enum class CompressionLevel
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | 0 | Сжатие не применяется. Файлы сохраняются без изменений. |
| Level1 | 1 | Самое быстрое сжатие с наименьшим коэффициентом сжатия. |
| Level2 | 2 | Быстрее сжатие с немного лучшим коэффициентом сжатия, чем [CompressionLevel::Level1](./). |
| Level3 | 3 | Обеспечивает лучшее сжатие, чем [CompressionLevel::Level2](./), с умеренным влиянием на производительность. |
| Level4 | 4 | Обеспечивает лучшее сжатие, чем [CompressionLevel::Level3](./). |
| Level5 | 5 | Обеспечивает улучшенное сжатие по сравнению с [CompressionLevel::Level4](./) с дополнительным временем обработки. |
| Level6 | 6 | Стандартное сжатие, предлагающее хороший баланс между скоростью сжатия и размером файла. Уровень сжатия по умолчанию. |
| Level7 | 7 | Обеспечивает более высокое сжатие, чем [CompressionLevel::Level6](./), но с более медленной обработкой. |
| Level8 | 8 | Обеспечивает более высокое сжатие, чем [CompressionLevel::Level7](./). |
| Level9 | 9 | Максимальное сжатие. Создаёт наименьший размер файла при самой медленной скорости обработки. |

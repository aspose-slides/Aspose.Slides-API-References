## NotesCommentsLayoutingOptions::get_ShowCommentsByNoAuthor() метод

Получает видимость комментариев без автора. Если значение true, то комментарии будут отображаться. (Применяется только если комментарии отображаются.)

```cpp
bool Aspose::Slides::Export::NotesCommentsLayoutingOptions::get_ShowCommentsByNoAuthor()
```

## Примечания

Значение по умолчанию — **false**.

## NotesCommentsLayoutingOptions::set_ShowCommentsByNoAuthor(bool) метод

Устанавливает видимость комментариев без автора. Если true, комментарии будут отображаться. (Применяется только если комментарии отображаются).

```cpp
void Aspose::Slides::Export::NotesCommentsLayoutingOptions::set_ShowCommentsByNoAuthor(bool value)
```

## Примечания

Значение по умолчанию **false**.

## ISwfOptions::set_LogoImageBytes(System::ArrayPtr\<uint8_t\>) метод

Изображение, которое будет отображаться в качестве логотипа в правом верхнем углу просмотрщика.

Изображение должно быть PNG размером 32×64 пикселей, иначе логотип может отображаться некорректно.

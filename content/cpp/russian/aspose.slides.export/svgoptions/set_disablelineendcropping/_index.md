## SVGOptions::set_DisableLineEndCropping(bool) метод

В SVG 1.1 отсутствует возможность задавать отступы для маркеров. [Aspose.Slides](../../../aspose.slides/) движок записи SVG имеет обходное решение этой проблемы: он обрезает конец линии со стрелкой, поэтому линия не пересекает маркеры. Эта опция отключает такое поведение. Записывается **bool**.

```cpp
void Aspose::Slides::Export::SVGOptions::set_DisableLineEndCropping(bool value) override
```

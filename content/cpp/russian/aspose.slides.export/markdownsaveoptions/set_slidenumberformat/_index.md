## MarkdownSaveOptions::set_SlideNumberFormat(System::String) метод

Устанавливает строку формата, используемую для заголовков номеров слайдов в выводе Markdown. Формат должен включать заполнитель \"{0}\", который будет заменён индексом слайда при экспорте. Пример: \"# Slide {0}\" будет генерировать \"# Slide 1\", \"# Slide 2\", и т.д.

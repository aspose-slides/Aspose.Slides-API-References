## SvgEvent enum

Представляет варианты для SVG-формы.

```cpp
enum class SvgEvent
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| OnFocusIn | 0 | Происходит, когда элемент получает фокус, например, когда текст выделяется. |
| OnFocusOut | 1 | Происходит, когда элемент теряет фокус, например, когда текст перестаёт быть выделенным. |
| OnActivate | 2 | Происходит, когда элемент активируется, например, через щелчок мышью или нажатие клавиши. |
| OnClick | 3 | Происходит, когда кнопка указательного устройства щёлкает над элементом. |
| OnMouseDown | 4 | Происходит, когда кнопка указательного устройства нажата над элементом. |
| OnMouseUp | 5 | Происходит, когда кнопка указательного устройства отпущена над элементом. |
| OnMouseOver | 6 | Происходит, когда указательное устройство перемещается на элемент. |
| OnMouseMove | 7 | Происходит, когда указательное устройство перемещается, находясь над элементом. |
| OnMouseOut | 8 | Происходит, когда указательное устройство перемещается от элемента. |
| OnLoad | 9 | Происходит, когда пользовательский агент полностью проанализировал элемент, его потомков и все связанные ресурсы, необходимые для его отображения. |
| OnUnload | 10 | Происходит, когда реализация [DOM](../../aspose.slides.dom/) удаляет документ из окна или фрейма. Применимо только к самым внешним элементам svg. |
| OnAbort | 11 | Происходит, когда загрузка страницы останавливается до того, как элемент полностью загрузился. |
| OnError | 12 | Происходит, когда элемент не загружается корректно или когда возникает ошибка во время выполнения скрипта. |
| OnResize | 13 | Происходит, когда представление документа изменяется в размере. Применимо только к самым внешним элементам svg. |
| OnScroll | 14 | Происходит, когда представление документа смещается вдоль оси X, Y или обеих осей. Применимо только к самым внешним элементам svg. |
| OnZoom | 15 | Происходит, когда уровень масштабирования представления документа изменяется. Применимо только к самым внешним элементам svg. |
| OnBegin | 16 | Происходит, когда элемент анимации начинается. |
| OnEnd | 17 | Происходит, когда элемент анимации заканчивается. |
| OnRepeat | 18 | Происходит, когда элемент анимации повторяется. |

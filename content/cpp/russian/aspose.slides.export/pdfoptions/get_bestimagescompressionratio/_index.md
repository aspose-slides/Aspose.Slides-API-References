## PdfOptions::get_BestImagesCompressionRatio() метод


Указывает, следует ли автоматически выбирать наиболее эффективное сжатие (вместо стандартного) для каждого изображения. Если установлено в **bool**.true, для каждого изображения в презентации будет выбран наиболее подходящий алгоритм сжатия, что приведёт к уменьшению размера получаемого PDF-документа.

```cpp
bool Aspose::Slides::Export::PdfOptions::get_BestImagesCompressionRatio() override
```

## Примечания


Выбор оптимального коэффициента сжатия изображений требует значительных вычислительных ресурсов и дополнительного объёма ОЗУ, и по умолчанию эта опция имеет значение **bool**.false.

По умолчанию **bool**.false.

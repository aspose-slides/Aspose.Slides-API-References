## PdfOptions::get_EmbedTrueTypeFontsForASCII() метод

Определяет, будет ли [Aspose.Slides](../../../aspose.slides/) встраивать общие шрифты для текста ASCII (диапазон кодов 33..127). [Fonts](../../../aspose.slides/fonts/) для кодов символов больше 127 всегда встраиваются. Список общих шрифтов включает базовые 14 шрифтов PDF и дополнительные шрифты, указанные пользователем. Читается **bool**.

```cpp
bool Aspose::Slides::Export::PdfOptions::get_EmbedTrueTypeFontsForASCII() override
```

## Примечания

По умолчанию **true**.

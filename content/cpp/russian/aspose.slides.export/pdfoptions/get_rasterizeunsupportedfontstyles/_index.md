## PdfOptions::get_RasterizeUnsupportedFontStyles() метод

Указывает, следует ли растрировать текст как растровое изображение и сохранять его в PDF, когда шрифт не поддерживает полужирное начертание. Такой подход может улучшить качество текста в создаваемом PDF для некоторых шрифтов. Читает **bool**.

```cpp
bool Aspose::Slides::Export::PdfOptions::get_RasterizeUnsupportedFontStyles() override
```

## Примечания

По умолчанию **false**.

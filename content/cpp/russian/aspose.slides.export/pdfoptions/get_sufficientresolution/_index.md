## PdfOptions::get_SufficientResolution() метод


Возвращает значение, определяющее разрешение изображений внутри PDF-документа.

```cpp
float Aspose::Slides::Export::PdfOptions::get_SufficientResolution() override
```

## Примечания


Свойство влияет на размер файла, время экспорта и качество изображения.

Значение по умолчанию — **96**.

Эффект этого параметра зависит от нескольких факторов. Алгоритм пытается получить лучший размер выходного изображения в соответствии со значением свойства, размером исходного изображения и размерами кадра изображения. Использование похожих значений свойства может дать тот же результат. Рекомендуется использовать шаг 16 или 32 для заметного эффекта.

Читать **float**.

## PdfOptions::set_EmbedTrueTypeFontsForASCII(bool) метод

Определяет, будет ли [Aspose.Slides](../../../aspose.slides/) встраивать общие шрифты для текста ASCII (диапазон кодов 33..127). [Fonts](../../../aspose.slides/fonts/) для кодов символов больше 127 всегда встраиваются. Список общих шрифтов включает базовые 14 шрифтов PDF и дополнительные пользовательские шрифты. Записать **bool**.

```cpp
void Aspose::Slides::Export::PdfOptions::set_EmbedTrueTypeFontsForASCII(bool value) override
```

## Примечания

По умолчанию **true**.

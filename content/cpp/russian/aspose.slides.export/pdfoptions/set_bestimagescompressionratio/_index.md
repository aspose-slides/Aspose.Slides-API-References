## PdfOptions::set_BestImagesCompressionRatio(bool) метод

Указывает, должно ли автоматически выбираться наиболее эффективное сжатие (вместо используемого по умолчанию) для каждого изображения. Если установлено значение **bool**.true, для каждого изображения в презентации будет выбран наиболее подходящий алгоритм сжатия, что приведёт к уменьшению размера получаемого PDF-документа.

```cpp
void Aspose::Slides::Export::PdfOptions::set_BestImagesCompressionRatio(bool value) override
```
## Примечания

Выбор лучшего коэффициента сжатия изображений требует значительных вычислительных ресурсов и дополнительного объёма оперативной памяти, и эта опция по умолчанию имеет значение **bool**.false.

По умолчанию **bool**.false.

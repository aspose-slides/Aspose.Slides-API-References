## PdfOptions::set_RasterizeUnsupportedFontStyles(bool) метод

Указывает, следует ли растеризовать текст как bitmap и сохранить в PDF, когда шрифт не поддерживает полужирное начертание. Такой подход может повысить качество текста в получаемом PDF для некоторых шрифтов. Записать **bool**.

```cpp
void Aspose::Slides::Export::PdfOptions::set_RasterizeUnsupportedFontStyles(bool value) override
```

## Примечания

По умолчанию **false**.

## IPdfOptions::get_RasterizeUnsupportedFontStyles() метод


Указывает, следует ли растеризовать текст как bitmap и сохранить в PDF, когда шрифт не поддерживает полужирное начертание. Этот подход может улучшить качество текста в получаемом PDF для некоторых шрифтов. Читать **bool**.

```cpp
virtual bool Aspose::Slides::Export::IPdfOptions::get_RasterizeUnsupportedFontStyles()=0
```

## Примечания


По умолчанию **false**.

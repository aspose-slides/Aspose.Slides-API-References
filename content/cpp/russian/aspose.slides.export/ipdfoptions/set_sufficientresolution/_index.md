## IPdfOptions::set_SufficientResolution(float) метод

Устанавливает значение, определяющее разрешение изображений внутри PDF-документа.

```cpp
virtual void Aspose::Slides::Export::IPdfOptions::set_SufficientResolution(float value)=0
```

## Примечания

Свойство влияет на размер файла, время экспорта и качество изображения.

Значение по умолчанию — **96**.

Эффект этого параметра зависит от нескольких факторов. Алгоритм пытается получить оптимальный размер выходного изображения в соответствии со значением свойства, размером исходного изображения и размером кадра изображения. Использование схожих значений свойства может дать одинаковый результат. Рекомендуется использовать шаг 16 или 32, чтобы увидеть заметный эффект.

Записать **float**.

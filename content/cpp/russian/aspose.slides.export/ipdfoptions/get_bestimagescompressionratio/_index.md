## IPdfOptions::get_BestImagesCompressionRatio() метод

Указывает, следует ли автоматически выбирать наиболее эффективное сжатие (вместо сжатия по умолчанию) для каждого изображения. Если установлено **bool**.true, для каждого изображения в презентации будет выбран наиболее подходящий алгоритм сжатия, что приведёт к меньшему размеру получаемого PDF-документа.

```cpp
virtual bool Aspose::Slides::Export::IPdfOptions::get_BestImagesCompressionRatio()=0
```

## Примечания

Выбор лучшего коэффициента сжатия изображения является ресурсоёмким и требует дополнительного объёма ОЗУ, и эта опция по умолчанию **bool**.false.

По умолчанию **bool**.false.

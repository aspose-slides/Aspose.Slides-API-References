## IPdfOptions::set_EmbedTrueTypeFontsForASCII(bool) метод


True, чтобы встраивать шрифты TrueType для символов ASCII 32-127. [Fonts](../../../aspose.slides/fonts/) для кодов символов, превышающих 127, всегда встраиваются. Запишите **bool**.

```cpp
virtual void Aspose::Slides::Export::IPdfOptions::set_EmbedTrueTypeFontsForASCII(bool value)=0
```

## Примечания


По умолчанию **true**.

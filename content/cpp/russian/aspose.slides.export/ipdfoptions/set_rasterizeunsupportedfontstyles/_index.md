## IPdfOptions::set_RasterizeUnsupportedFontStyles(bool) метод

Указывает, следует ли растировать текст как растровое изображение и сохранять его в PDF, когда шрифт не поддерживает полужирное начертание. Такой подход может улучшить качество текста в результирующем PDF для некоторых шрифтов. Записывает **bool**.

```cpp
virtual void Aspose::Slides::Export::IPdfOptions::set_RasterizeUnsupportedFontStyles(bool value)=0
```

## Замечания


По умолчанию **false**.

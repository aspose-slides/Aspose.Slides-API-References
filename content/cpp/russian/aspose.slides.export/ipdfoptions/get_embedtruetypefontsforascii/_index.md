## IPdfOptions::get_EmbedTrueTypeFontsForASCII() метод

True, чтобы встраивать шрифты TrueType для символов ASCII 32-127. [Fonts](../../../aspose.slides/fonts/) для кодов символов больше 127 всегда встраиваются. Читать **bool**.

```cpp
virtual bool Aspose::Slides::Export::IPdfOptions::get_EmbedTrueTypeFontsForASCII()=0
```

## Примечания

По умолчанию **true**.

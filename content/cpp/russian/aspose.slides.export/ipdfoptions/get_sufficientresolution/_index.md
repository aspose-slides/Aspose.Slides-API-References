## IPdfOptions::get_SufficientResolution() метод

Возвращает значение, определяющее разрешение изображений в документе PDF.

```cpp
virtual float Aspose::Slides::Export::IPdfOptions::get_SufficientResolution()=0
```

## Примечания

Свойство влияет на размер файла, время экспорта и качество изображения.

Значение по умолчанию — **96**.

Эффект этого параметра зависит от нескольких факторов. Алгоритм пытается получить оптимальный размер выходного изображения в соответствии со значением свойства, размером исходного изображения и размером кадра изображения. Использование похожих значений свойства может дать тот же результат. Рекомендуется использовать шаг 16 или 32, чтобы увидеть заметный эффект.

Читать **float**.

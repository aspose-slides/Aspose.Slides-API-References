## IPdfOptions::set_BestImagesCompressionRatio(bool) метод


Указывает, следует ли автоматически выбирать наиболее эффективное сжатие (вместо используемого по умолчанию) для каждого изображения. Если установлено значение **bool**.true, для каждого изображения в презентации будет выбран наиболее подходящий алгоритм сжатия, что приведёт к уменьшению размера получаемого PDF-документа.

```cpp
virtual void Aspose::Slides::Export::IPdfOptions::set_BestImagesCompressionRatio(bool value)=0
```

## Примечания


Выбор лучшего коэффициента сжатия изображений требует значительных вычислительных ресурсов и дополнительного объёма ОЗУ, и эта опция по умолчанию имеет значение **bool**.false.

По умолчанию **bool**.false.

## PdfAccessPermissions enum

Содержит набор флагов, определяющих, какие разрешения доступа следует предоставить при открытии документа пользователем.

```cpp
enum class PdfAccessPermissions
```

### Values

| Имя | Значение | Описание |
| --- | --- | --- |
| None | 0 | Указывает, что у пользователя нет разрешений доступа. |
| PrintDocument | 4 | Указывает, может ли пользователь печатать документ (возможно не в наивысшем качестве, в зависимости от того, установлен ли также бит [PdfAccessPermissions::HighQualityPrint](./)). |
| ModifyContent | 8 | Указывает, может ли пользователь изменять содержимое документа операциями, отличными от тех, которые контролируются битами [PdfAccessPermissions::AddOrModifyFields](./), [PdfAccessPermissions::FillExistingFields](./), [PdfAccessPermissions::AssembleDocument](./). |
| CopyTextAndGraphics | 16 | Указывает, может ли пользователь копировать или иным образом извлекать текст и графику из документа операциями, отличными от тех, что контролируются битом [PdfAccessPermissions::ExtractTextAndGraphics](./). |
| AddOrModifyFields | 32 | Указывает, может ли пользователь добавлять или изменять текстовые аннотации, заполнять интерактивные поля формы, и, если также установлен бит [PdfAccessPermissions::ModifyContent](./), создавать или изменять интерактивные поля формы (включая поля подписи). |
| FillExistingFields | 256 | Указывает, может ли пользователь заполнять существующие интерактивные поля формы (включая поля подписи), даже если бит [PdfAccessPermissions::AddOrModifyFields](./) не установлен. |
| ExtractTextAndGraphics | 512 | Указывает, может ли пользователь извлекать текст и графику в целях обеспечения доступности для пользователей с ограниченными возможностями или для иных целей. |
| AssembleDocument | 1024 | Указывает, может ли пользователь собирать документ (вставлять, вращать или удалять страницы и создавать закладки или миниатюры), даже если бит [PdfAccessPermissions::ModifyContent](./) не установлен. |
| HighQualityPrint | 2048 | Указывает, может ли пользователь печатать документ в представление, из которого можно создать точную цифровую копию содержимого PDF. Когда этот бит не установлен (и бит [PdfAccessPermissions::PrintDocument](./) установлен), печать ограничивается низкоуровневым представлением внешнего вида, возможно с пониженным качеством. |

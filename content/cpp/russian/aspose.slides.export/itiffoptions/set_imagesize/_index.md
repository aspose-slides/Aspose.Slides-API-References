## ITiffOptions::set_ImageSize(System::Drawing::Size) метод

Указывает размер генерируемого TIFF-изображения. Значение по умолчанию — 0x0, что означает, что размеры генерируемого изображения будут вычисляться на основе значения размера слайда презентации. Запишите [System::Drawing::Size](../../../system.drawing/size/).

virtual void Aspose::Slides::Export::ITiffOptions::set_ImageSize(System::Drawing::Size value)=0

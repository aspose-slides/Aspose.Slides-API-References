## TextInheritanceLimit перечисление

Управляет глубиной наследования свойств текста.

```cpp
enum class TextInheritanceLimit
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| All | 0 | Наследовать все свойства текста. |
| TextBox | 1 | Наследовать только стиль [TextFrame](../../aspose.slides/textframe/). |
| ParagraphOnly | 2 | Использовать только свойства, определённые для абзаца. |

## EmbedFontCharacters enum

Представляет правило, используемое для добавления нового встраиваемого шрифта в [IPresentation](../../aspose.slides/ipresentation/)

```cpp
enum class EmbedFontCharacters
```

### Values

| Имя | Значение | Описание |
| --- | --- | --- |
| OnlyUsed | 0 | Встраивать только те символы, которые использованы в презентации (лучше всего для уменьшения размера файла). |
| All | 1 | Встраивать все символы (лучшее для редактирования другими людьми). |

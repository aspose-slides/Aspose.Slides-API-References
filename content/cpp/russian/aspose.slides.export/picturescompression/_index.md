## PicturesCompression перечисление

Представляет уровень сжатия изображений

```cpp
enum class PicturesCompression
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Dpi330 | 0 | Хорошее качество для дисплеев высокого разрешения (HD) |
| Dpi220 | 1 | Отличное качество на большинстве принтеров и экранов |
| Dpi150 | 2 | Хорошо подходит для веб-страниц и проекторов |
| Dpi96 | 3 | Минимизировать размер документа для обмена |
| Dpi72 | 4 | Уровень сжатия по умолчанию |
| DocumentResolution | 5 | Использовать разрешение документа — изображение не будет сжиматься и будет использовано в документе как есть |

## NotesPositions enum

Представляет правило отображения заметок в экспортируемом документе

```cpp
enum class NotesPositions
```

### Values

| Имя | Значение | Описание |
| --- | --- | --- |
| None | 0 | Указывает, что заметки не должны отображаться вовсе. |
| BottomFull | 1 | Указывает, что заметки должны полностью отображаться с использованием дополнительных страниц по мере необходимости. |
| BottomTruncated | 2 | Указывает, что заметки должны отображаться только на одной странице. |

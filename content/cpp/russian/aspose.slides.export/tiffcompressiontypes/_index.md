## TiffCompressionTypes enum

Предоставляет параметры, управляющие тем, как презентация сжимается в формате TIFF.

```cpp
enum class TiffCompressionTypes
```

### Values

| Имя | Значение | Описание |
| --- | --- | --- |
| Default | 0 | Указывает схему сжатия по умолчанию (LZW). |
| None | 1 | Указывает отсутствие сжатия. |
| CCITT3 | 2 | Указывает схему сжатия CCITT3. |
| CCITT4 | 3 | Указывает схему сжатия CCITT4. |
| LZW | 4 | Указывает схему сжатия LZW (По умолчанию). |
| RLE | 5 | Указывает схему сжатия RLE. |

## TcpListener::Pending() метод

Возвращает значение, указывающее, есть ли ожидающие запросы на соединение.

```cpp
bool System::Net::Sockets::TcpListener::Pending()
```

### Возвращаемое значение

True, если присутствуют ожидающие запросы на соединение, в противном случае false.

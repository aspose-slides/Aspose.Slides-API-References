## Details_SocketException класс

Представляет исключение, выбрасываемое при возникновении ошибки сокета. Никогда не создавайте экземпляры этого класса вручную. Вместо этого используйте класс SocketException. Никогда не оборачивайте экземпляры класса SocketException в [System::SmartPtr](../../system/smartptr/).

```cpp
class Details_SocketException : public System::Details_Exception
```

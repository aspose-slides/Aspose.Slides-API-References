| Pup | 12 | Протокол PARC Universal Packet. |

| IPv6 | 41 | Протокол Интернета версии 6. |
| IPv6RoutingHeader | 43 | Заголовок IPv6 Routing. |
| IPv6FragmentHeader | 44 | Заголовок IPv6 Fragment. |
| IPSecEncapsulatingSecurityPayload | 50 | Заголовок IPv6 Encapsulating [Security](../../system.net.security/) Payload. |

| SpxII | 1257 | Протокол Sequenced Packet Exchange версии 2. |
| Unknown | n/a | Неизвестный протокол. |

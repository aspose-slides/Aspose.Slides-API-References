## SocketType enum

Перечисляет типы сокетов.

```cpp
enum class SocketType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Stream | 1 | Тип, поддерживающий надёжные, двунаправленные, основанные на соединении потоки байтов без дублирования данных и без сохранения границ. |
| Dgram | 2 | Тип, поддерживающий датаграммы, которые являются безсоединительными, ненадёжными сообщениями фиксированной максимальной длины. |
| Raw | 3 | Тип, предоставляющий доступ к базовому транспортному протоколу. |
| Rdm | 4 | Тип, поддерживающий безсоединительные, ориентированные на сообщения, надёжно доставляемые сообщения и сохраняющий границы сообщений в данных. |
| Seqpacket | 5 | Тип, предоставляющий ориентированный на соединение и надёжный двунаправленный перенос упорядоченных потоков байтов через сеть. |
| Unknown | n/a | Неизвестный тип. |

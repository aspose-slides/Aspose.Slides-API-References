## SocketOptionLevel перечисление

Определяет уровни параметров сокета для класса '[Socket](../socket/)'.

```cpp
enum class SocketOptionLevel
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Socket | 65535 | Указанные параметры сокета применяются ко всем сокетам. |
| IP | n/a | Указанные параметры сокета применяются к IP-сокетам. |
| IPv6 | n/a | Указанные параметры сокета применяются к IPv6-сокетам. |
| Tcp | n/a | Указанные параметры сокета применяются к TCP-сокетам. |
| Udp | n/a | Указанные параметры сокета применяются к UDP-сокетам. |

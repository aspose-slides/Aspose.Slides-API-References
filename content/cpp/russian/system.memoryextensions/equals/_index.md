```cpp
bool System::MemoryExtensions::Equals(const ReadOnlySpan<char16_t> &span, const ReadOnlySpan<char16_t> &other, StringComparison comparisonType)
```


### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| span | const [ReadOnlySpan](../../system/readonlyspan/)\<char16_t\>\& | Первый сравниваемый диапазон |
| other | const [ReadOnlySpan](../../system/readonlyspan/)\<char16_t\>\& | Второй сравниваемый диапазон |
| comparisonType | [StringComparison](../../system/stringcomparison/) | Тип сравнения строк, который следует использовать |

### Возвращаемое значение

true, если диапазоны равны, иначе false

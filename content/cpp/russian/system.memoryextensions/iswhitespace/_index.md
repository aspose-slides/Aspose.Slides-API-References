## System::MemoryExtensions::IsWhiteSpace(const ReadOnlySpan\<char16_t\>\&) функция

Проверяет, состоит ли весь диапазон только из символов пробела.

```cpp
bool System::MemoryExtensions::IsWhiteSpace(const ReadOnlySpan<char16_t> &span)
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| span | const [ReadOnlySpan](../../system/readonlyspan/)\<char16_t\>\& | Диапазон, который необходимо проверить на наличие только пробельных символов |

### Возвращаемое значение

true если все символы в диапазоне являются пробельными, иначе false

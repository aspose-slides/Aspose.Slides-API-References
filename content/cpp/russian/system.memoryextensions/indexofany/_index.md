## System::MemoryExtensions::IndexOfAny(const ReadOnlySpan\<T\>\&, const T\&, const T\&) function

Находит индекс первого вхождения любого из двух указанных значений в ReadOnlySpan<T>

```cpp
template<typename T> int32_t System::MemoryExtensions::IndexOfAny(const ReadOnlySpan<T> &span, const T &value0, const T &value1)
```

### Параметры шаблона

| Параметр | Описание |
| --- | --- |
| T | Тип элементов в диапазоне |

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| span | const [ReadOnlySpan](../../system/readonlyspan/)\<T\>\& | Диапазон, в котором выполняется поиск |
| value0 | const T\& | Первое значение для поиска |
| value1 | const T\& | Второе значение для поиска |

### Возвращаемое значение

Нулевой-based индекс первого вхождения или -1, если значение не найдено

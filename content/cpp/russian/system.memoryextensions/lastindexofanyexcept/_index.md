## System::MemoryExtensions::LastIndexOfAnyExcept(const ReadOnlySpan\<T\>\&, const T\&, const T\&, const T\&) функция


Находит последнее вхождение любого элемента, кроме трёх указанных значений, в пределах span.

```cpp
template<typename T> int32_t System::MemoryExtensions::LastIndexOfAnyExcept(const ReadOnlySpan<T> &span, const T &value0, const T &value1, const T &value2)
```


### Параметры шаблона

| Parameter | Description |
| --- | --- |
| T | Тип элементов в span |

### Аргументы

| Parameter | Type | Description |
| --- | --- | --- |
| span | const [ReadOnlySpan](../../system/readonlyspan/)\<T\>\& | Span, в котором выполняется поиск |
| value0 | const T\& | Первое значение, которое исключается |
| value1 | const T\& | Второе значение, которое исключается |
| value2 | const T\& | Третье значение, которое исключается |

### Возвращаемое значение

Нулевой индекс последнего не исключённого элемента, или -1 если не найдено

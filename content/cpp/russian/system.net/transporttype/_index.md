## TransportType enum

Определяет тип транспорта, разрешённый для сокета.

```cpp
enum class TransportType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Udp | 1 | Тип транспорта UDP. |
| Connectionless | n/a | Тип транспорта без соединения. |
| Tcp | 2 | Тип транспорта TCP. |
| ConnectionOriented | n/a | Транспорт ориентирован на соединение. |
| All | 3 | Все типы транспорта. |

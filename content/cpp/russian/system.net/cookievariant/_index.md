## CookieVariant enum

Перечисляет спецификации файлов cookie.

```cpp
enum class CookieVariant
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Unknown | 0 | Неизвестная спецификация файлов cookie. |
| Plain | 1 | Простая спецификация файлов cookie. |
| Rfc2109 | 2 | Спецификация файлов cookie 'RFC 2109'. |
| Rfc2965 | 3 | Спецификация файлов cookie 'RFC 2965'. |
| Default | 2 | Значение по умолчанию, равное 'Rfc2109'. |

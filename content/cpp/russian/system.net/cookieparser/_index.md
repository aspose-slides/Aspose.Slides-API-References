## CookieParser класс

Used to parse a cookie header and create an instance of the [Cookie](../cookie/) класс. Objects of this класс should only be allocated using [System::MakeObject()](../../system/makeobject/) функция. Never create instance of this type on stack or using operator new, as it will result in runtime errors and/or assertion faults. Always wrap this класс into [System::SmartPtr](../../system/smartptr/) указатель and use this pointer to pass it to functions as argument.

```cpp
class CookieParser : public System::Object
```

## CookieContainer::IsLocalDomain(String) метод

Проверяет, является ли указанный домен localhost.

```cpp
bool System::Net::CookieContainer::IsLocalDomain(String host)
```

### Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| host | [String](../../../system/string/) | Домен, который необходимо проверить. |

### Возвращаемое значение

true, когда указанный домен является localhost, иначе false.

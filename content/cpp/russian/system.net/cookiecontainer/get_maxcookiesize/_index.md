## CookieContainer::get_MaxCookieSize() метод


Получает максимальный размер cookie.

```cpp
int32_t System::Net::CookieContainer::get_MaxCookieSize()
```


### Возвращаемое значение

Максимальный размер cookie.
## Примечания



Новый cookie будет усечён, если его размер превышает максимальный размер cookie.

## IPAddress::IsLoopback(System::SharedPtr\<IPAddress\>) метод

Возвращает значение, указывающее, является ли указанный адрес адресом обратной связи.

```cpp
static bool System::Net::IPAddress::IsLoopback(System::SharedPtr<IPAddress> address)
```

### Аргументы

| Параметр | Тип | Описание |
| --- | --- | --- |
| address | [System::SharedPtr](../../../system/sharedptr/)\<[IPAddress](../)\> | Адрес для проверки. |

### Возвращаемое значение

True, если указанный адрес является адресом обратной связи, иначе false.

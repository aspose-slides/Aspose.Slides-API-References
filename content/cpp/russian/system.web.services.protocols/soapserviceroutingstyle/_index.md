## SoapServiceRoutingStyle перечисление

Перечисляет варианты того, как SOAP-сообщение направляется к XML [Web](../../system.web/) сервису.

```cpp
enum class SoapServiceRoutingStyle
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| SoapAction | 0 | SOAP-сообщение направляется на основе атрибута 'SOAPAction'. |
| RequestElement | 1 | SOAP-сообщение направляется на основе первого дочернего элемента после элемента 'Body'. |

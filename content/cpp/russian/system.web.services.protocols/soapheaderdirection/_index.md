## SoapHeaderDirection enum

Перечисляет направления заголовка SOAP.

```cpp
enum class SoapHeaderDirection
```

### Значения

| Name | Value | Description |
| --- | --- | --- |
| In | 1 | Получатель заголовка SOAP — XML [Web](../../system.web/) сервис. |
| InOut | 3 | Получателями заголовка SOAP являются как XML [Web](../../system.web/) сервис, так и клиент XML [Web](../../system.web/) сервиса. |
| Out | 2 | Получатель заголовка SOAP — клиент XML [Web](../../system.web/) сервиса. |
| Fault | 4 | Заголовок SOAP отправляется клиенту XML [Web](../../system.web/) сервиса, когда метод XML [Web](../../system.web/) сервиса генерирует исключение. |

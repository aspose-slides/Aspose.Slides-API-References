## TextRenderingHint перечисление

Указывает качество рендеринга текста.

```cpp
enum class TextRenderingHint
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| SystemDefault | 0 | Glyph с системным указателем рендеринга по умолчанию. |
| SingleBitPerPixelGridFit | 1 | Glyph bitmap с хинтингом. |
| SingleBitPerPixel | 2 | Glyph bitmap без хинтинга. |
| AntiAliasGridFit | 3 | Сглаживание с хинтингом. |
| AntiAlias | 4 | Glyph anti-alias bitmap без хинтинга. |
| ClearTypeGridFit | 5 | Glyph CT bitmap с хинтингом. |

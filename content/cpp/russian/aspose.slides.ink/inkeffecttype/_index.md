## InkEffectType enum

Указывает набор предопределённых визуальных эффектов для отрисовки чернил. Каждый эффект соответствует текстуре или изображению, имитирующему стилизованный вид чернил. Эти значения можно использовать для настройки визуального стиля цифровых штрихов чернил во время отрисовки.

```cpp
enum class InkEffectType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| NotDefined | -1 | Эффект не определён или неизвестен. Это значение можно использовать как значение по умолчанию или резервное. |
| Bronze | 0 | Тёплая, коричневато-металлическая текстура, напоминающая бронзовые чернила. |
| Galaxy | 1 | Яркая, мерцающая текстура, напоминающая галактику, с космическими тонами. |
| Gold | 2 | Светлая, металлическая золотая текстура, придающая штрихам чернил роскошный вид. |
| Lava | 3 | Огненная текстура, напоминающая расплавленную лаву, с красными и оранжевыми оттенками. |
| Ocean | 4 | Глубокая синяя, жидкостная текстура, имитирующая океанские волны или чернила на водной основе. |
| RainbowGlitter | 5 | Яркий, искристый радужный блестящий эффект, используемый для праздничных или живых штрихов чернил. |
| RoseGold | 6 | Мягкая смесь розового и золотого, похожая на розовое золото, для элегантных штрихов чернил. |
| Silver | 7 | Холодная, металлическая серебряная текстура, имитирующая классические серебряные чернила. |

## InkBrush::get_InkEffect() метод

Получает тип эффекта чернил (например, Galaxy, Gold, Silver), определяющий визуальный стиль штриха чернила. Значение извлекается из свойства кисти "inkEffects". Если указанный эффект не распознан, возвращается [InkEffectType::NotDefined](../../inkeffecttype/).

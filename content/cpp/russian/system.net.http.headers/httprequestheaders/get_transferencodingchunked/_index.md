## HttpRequestHeaders::get_TransferEncodingChunked() метод

Возвращает значение, указывающее, содержит ли значение заголовка 'Transfer-Encoding' строку 'Chunked'.

```cpp
Nullable<bool> System::Net::Http::Headers::HttpRequestHeaders::get_TransferEncodingChunked()
```

### Возвращаемое значение

True, когда значение заголовка 'Transfer-Encoding' содержит 'Chunked', иначе false.

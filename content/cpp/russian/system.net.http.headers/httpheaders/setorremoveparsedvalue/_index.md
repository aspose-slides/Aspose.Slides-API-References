## HttpHeaders::SetOrRemoveParsedValue(String, System::SharedPtr\<Object\>) метод

Получает заголовок по указанному имени и задаёт или удаляет его значение. Значение заголовка будет удалено, когда параметр 'value' равен nullptr, иначе будет установлено разобранное значение.

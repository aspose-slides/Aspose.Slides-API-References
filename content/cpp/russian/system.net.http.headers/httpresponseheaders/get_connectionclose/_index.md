## HttpResponseHeaders::get_ConnectionClose() метод


Возвращает значение, указывающее, содержит ли значение заголовка 'Connection' строку 'Close'.

```cpp
Nullable<bool> System::Net::Http::Headers::HttpResponseHeaders::get_ConnectionClose()
```


### Возвращаемое значение

True, когда значение заголовка 'Connection' содержит 'Close', иначе false.

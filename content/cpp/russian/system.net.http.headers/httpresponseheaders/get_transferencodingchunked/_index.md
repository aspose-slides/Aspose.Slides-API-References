## HttpResponseHeaders::get_TransferEncodingChunked() метод


Получает значение, которое указывает, содержит ли значение заголовка 'Transfer-Encoding' параметр 'Chunked'.

```cpp
Nullable<bool> System::Net::Http::Headers::HttpResponseHeaders::get_TransferEncodingChunked()
```


### Возвращаемое значение

True, если значение заголовка 'Transfer-Encoding' содержит 'Chunked', иначе false.

## SlideShowTransition::get_SoundName() метод


Указывает человекочитаемое имя для звука перехода. Значение [ISlideShowTransition::set_Sound](../../../aspose.slides/islideshowtransition/set_sound/) должно быть присвоено для получения или установки имени звука. Читает [System::String](../../../system/string/).

```cpp
System::String Aspose::Slides::SlideShow::SlideShowTransition::get_SoundName() override
```

## Примечания


Это имя отображается в пользовательском интерфейсе PowerPoint при ручной настройке звука перехода.

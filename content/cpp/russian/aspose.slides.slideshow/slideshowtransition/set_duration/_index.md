## SlideShowTransition::set_Duration(int32_t) метод

Устанавливает длительность эффекта перехода слайда в миллисекундах. Запишите **int32_t**.

```cpp
void Aspose::Slides::SlideShow::SlideShowTransition::set_Duration(int32_t value) override
```

## Примечания

Соответствует атрибуту **p14:dur** элемента **p:transition** в схеме PresentationML. Если не задано, длительность определяется автоматически на основе свойства [Speed](../) и типа перехода.

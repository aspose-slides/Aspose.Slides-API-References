## SlideShowTransition::set_SoundIsBuiltIn(bool) метод

Указывает, является ли этот звук встроенным. Если этот атрибут установлен в true, то приложение-генератор получает уведомление проверить атрибут name, указанный для этого звука в его списке встроенных звуков, и при необходимости может отобразить пользовательское имя или пользовательский интерфейс. Записывает **bool**.

void Aspose::Slides::SlideShow::SlideShowTransition::set_SoundIsBuiltIn(bool value) override

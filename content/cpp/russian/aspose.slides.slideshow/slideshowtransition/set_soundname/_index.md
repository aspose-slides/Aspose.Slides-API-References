## SlideShowTransition::set_SoundName(System::String) метод

Указывает человекочитаемое имя звука перехода. [ISlideShowTransition::set_Sound](../../../aspose.slides/islideshowtransition/set_sound/) должен быть назначен для получения или установки имени звука. Записывает [System::String](../../../system/string/).

```cpp
void Aspose::Slides::SlideShow::SlideShowTransition::set_SoundName(System::String value) override
```

## Примечания

Это имя отображается в пользовательском интерфейсе PowerPoint при ручной настройке звука перехода.

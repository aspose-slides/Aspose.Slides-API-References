## SlideShowTransition::set_AdvanceAfterTime(uint32_t) метод


Указывает время в миллисекундах, после которого должно начаться переключение. Этот параметр может использоваться вместе с атрибутом advClick. Если этот атрибут не указан, считается, что автоматическое переключение не произойдёт. Запись **uint32_t**.

```cpp
void Aspose::Slides::SlideShow::SlideShowTransition::set_AdvanceAfterTime(uint32_t value) override
```

## TransitionMorphType перечисление

Представляет тип morph transition.

```cpp
enum class TransitionMorphType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| ByObject | 0 | Morph transition будет выполнен с учётом фигур как неделимых объектов. |
| ByWord | 1 | Morph transition будет выполнен с переносом текста по словам, где это возможно. |
| ByChar | 2 | Morph transition будет выполнен с переносом текста по символам, где это возможно. |

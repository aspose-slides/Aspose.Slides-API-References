## TrendlineType 枚举

表示趋势线的类型

```cpp
enum class TrendlineType
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Exponential | 0 | 指定趋势线应为指数曲线的形式。 |
| Linear | 1 | 指定趋势线应为直线的形式。 |
| Logarithmic | 2 | 指定趋势线应为对数曲线的形式，其中 log 是自然对数。 |
| MovingAverage | 3 | 指定趋势线应为周期为 Period 的移动平均线。 |
| Polynomial | 4 | 指定趋势线应为阶数为 Order 的多项式曲线的形式。 |
| Power | 5 | 指定趋势线应为幂曲线的形式。 |

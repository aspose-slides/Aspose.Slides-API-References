## DataLabelCollection::Hide() 方法


通过将 DefaultDataLabelFormat 属性的所有 Show*-flags（ShowValue，...）设置为 false 状态，使数据标签默认隐藏。此后 IsVisible 将为 false。

```cpp
void Aspose::Slides::Charts::DataLabelCollection::Hide() override
```

## 备注


如果数据标签默认不可见（IsVisible 为 false），可以通过将 DefaultDataLabelFormat 属性的 Show*-flags（ShowValue，...）设置为 true 状态，使数据标签 "visible 
by default"。

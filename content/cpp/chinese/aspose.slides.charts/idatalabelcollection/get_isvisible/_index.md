## IDataLabelCollection::get_IsVisible() 方法

False 表示数据标签默认不可见（因此 DefaultDataLabelFormat 属性的所有 Show*-flags（ShowValue，...）均为 false）。只读 **bool**。

```cpp
virtual bool Aspose::Slides::Charts::IDataLabelCollection::get_IsVisible()=0
```

## 备注

如果数据标签默认可见，您可以使用 [Hide()](../hide/) 方法将其默认隐藏。但如果数据标签默认不可见（IsVisible 为 false），您可以通过将 DefaultDataLabelFormat 属性的 Show*-flags（ShowValue，...）设置为 true 状态，使数据标签 “visible 
by default”。

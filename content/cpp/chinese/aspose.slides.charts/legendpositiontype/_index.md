## LegendPositionType 枚举

确定图表中图例的位置。

```cpp
enum class LegendPositionType
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Bottom | 0 | 指定图例应绘制在图表的底部。 |
| Left | 1 | 指定图例应绘制在图表的左侧。 |
| Right | 2 | 指定图例应绘制在图表的右侧。 |
| Top | 3 | 指定图例应绘制在图表的顶部。 |
| TopRight | 4 | 指定图例应绘制在图表的右上角。 |

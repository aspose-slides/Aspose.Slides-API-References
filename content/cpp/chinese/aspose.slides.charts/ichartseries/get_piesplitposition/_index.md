## IChartSeries::get_PieSplitPosition() 方法

指定一个值，用于确定哪些数据点位于饼图或条形图的第二个饼或条中（用于 pie-of-pie 或 bar-of-pie 图表）。该属性与 PieSplitBy 属性一起使用。此属性不仅属于此系列，还属于父系列组的所有系列——它是相应组属性的投影。因此此属性为只读。使用 ParentSeriesGroup 属性访问父系列组。使用 [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get(set)_PieSplitPosition() 读/写属性来更改值。只读 **double**。

```cpp
virtual double Aspose::Slides::Charts::IChartSeries::get_PieSplitPosition()=0
```

## 备注

这是属性 [get_ParentSeriesGroup()](../get_parentseriesgroup/)->get(set)_PieSplitPosition() 的投影。

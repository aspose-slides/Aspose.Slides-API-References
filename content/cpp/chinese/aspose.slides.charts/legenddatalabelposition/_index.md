## LegendDataLabelPosition 枚举

确定数据标签的位置。

```cpp
enum class LegendDataLabelPosition
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| NotDefined | -1 | 位置未定义。 |
| Bottom | 0 | 指定数据标签应显示在数据标记下方。 |
| BestFit | 1 | 指定数据标签应显示在最佳位置。 |
| Center | 2 | 指定数据标签应居中显示在数据标记上。 |
| InsideBase | 3 | 指定数据标签应显示在数据标记基部内部。 |
| InsideEnd | 4 | 指定数据标签应显示在数据标记末端内部。 |
| Left | 5 | 指定数据标签应显示在数据标记左侧。 |
| OutsideEnd | 6 | 指定数据标签应显示在数据标记末端外部。 |
| Right | 7 | 指定数据标签应显示在数据标记右侧。 |
| Top | 8 | 指定数据标签应显示在数据标记上方。 |

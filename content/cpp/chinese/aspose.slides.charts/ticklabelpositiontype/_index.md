## TickLabelPositionType 枚举

表示指定坐标轴上刻度标签的位置类型。

```cpp
enum class TickLabelPositionType
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| High | 0 | 指定轴标签应位于垂直坐标轴的高端。 |
| Low | 1 | 指定轴标签应位于垂直坐标轴的低端。 |
| NextTo | 2 | 指定轴标签应紧邻坐标轴。 |
| None | 3 | 指定不绘制轴标签。 |

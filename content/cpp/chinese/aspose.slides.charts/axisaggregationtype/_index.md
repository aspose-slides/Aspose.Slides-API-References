## AxisAggregationType 枚举

表示分类轴的聚合类型。仅用于 Histogram 或 HistogramPareto 系列。

```cpp
enum class AxisAggregationType
```

### Values

| 名称 | 值 | 描述 |
| --- | --- | --- |
| ByCategory | 0 | 按类别聚合 |
| Automatic | 1 | 自动聚合类别。 |
| ByBinWidth | 2 | 按箱宽度聚合类别。 Property IAxis::get(set)_BinWidth must be setted. |
| ByNumberOfBins | 3 | 按箱数量聚合类别。 Property IAxis::get(set)_NumberOfBins must be setted. |

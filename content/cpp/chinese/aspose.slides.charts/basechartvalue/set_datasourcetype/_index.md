## BaseChartValue::set_DataSourceType(Aspose::Slides::Charts::DataSourceType) 方法

指定在派生类中 AsCell、AsCells、AsLiteralString 或 AsLiteralDouble 属性是否实际存在。换句话说，它指定 Data 属性的值的类型。写入 [Charts::DataSourceType](../../datasourcetype/)。

```cpp
void Aspose::Slides::Charts::BaseChartValue::set_DataSourceType(Aspose::Slides::Charts::DataSourceType value) override
```

## 备注

对于 [ChartDataPointCollection](../../chartdatapointcollection/) 中的点，此属性为只读。在这种情况下，要更改此属性的值，可以使用 ChartDataPointCollection.DataSourceTypeFor<...> 属性之一。

## DataLabel::get_IsVisible() 方法

False 表示数据标签不可见（因此所有 Show*-flags（ShowValue，...）均为 false）。只读 **bool**.

```cpp
bool Aspose::Slides::Charts::DataLabel::get_IsVisible() override
```

## 备注

如果数据标签可见，您可以使用 [Hide()](../hide/) 方法将其隐藏。但如果数据标签不可见（IsVisible 为 false），您可以通过将 Show*-flags（ShowValue，...）设置为 true 状态来使数据标签可见。

## DataLabel::Hide() 方法

通过将所有 Show*-flags (ShowValue, ...) 设置为 false 状态，使数据标签隐藏。IsVisible 将为 false。

```cpp
void Aspose::Slides::Charts::DataLabel::Hide() override
```

## 备注

如果数据标签不可见（IsVisible 为 false），可以通过将 Show*-flags (ShowValue, ...) 设置为 true 状态使数据标签可见。

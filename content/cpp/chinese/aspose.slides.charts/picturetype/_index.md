## PictureType 枚举

确定条形图片填充的模式。

```cpp
enum class PictureType
```

### 取值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| NotDefined | -1 | 未定义。 |
| Stack | 0 | 图片的尺寸设置为在最长的堆叠条形中最多重复 15 次。 |
| StackScale | 1 | 图片的尺寸设置为指定的单位数，并在条形的长度上重复。 |
| Stretch | 2 | 图片被拉伸到堆叠条形的完整长度。 |

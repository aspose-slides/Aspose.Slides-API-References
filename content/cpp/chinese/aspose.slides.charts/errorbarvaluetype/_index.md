## ErrorBarValueType 枚举


表示误差线值的类型

```cpp
enum class ErrorBarValueType
```

### 取值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Custom | 0 | 指定误差线的长度应由 Plus 和 Minus 元素确定。 |
| Fixed | 1 | 指定误差线的长度应为由 Error Bar Value 确定的固定值。 |
| Percentage | 2 | 指定误差线的长度应为数据的 Error Bar Value 百分比。 |
| StandardDeviation | 3 | 指定误差线的长度应为数据的 Error Bar Value 标准差。 |
| StandardError | 4 | 指定误差线的长度应为数据的 Error Bar Value 标准误差。 |

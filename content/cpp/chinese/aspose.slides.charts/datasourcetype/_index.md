## DataSourceType 枚举


数据源类型。

```cpp
enum class DataSourceType
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Worksheet | 0 | 使用 CellXls 对象来存储数据点的 XValue、YValue、Value 或 BubbleSize 于工作表中。 |
| StringLiterals | 1 | 使用快速访问的字符串变量来存储数据点的 XValue、YValue、Value 或 BubbleSize。Worksheet 未用于存储值。 |
| DoubleLiterals | 2 | 使用快速访问的 double 变量来存储数据点的 XValue、YValue、Value 或 BubbleSize。Worksheet 未用于存储值。 |

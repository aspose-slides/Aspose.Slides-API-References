## TickMarkType 枚举

表示指定坐标轴的刻度线类型。

```cpp
enum class TickMarkType
```

### 取值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Cross | 0 | 指定刻度线应穿过坐标轴。 |
| Inside | 1 | 指定刻度线应位于绘图区域内部。 |
| None | 2 | 指定不应有刻度线。 |
| Outside | 3 | 指定刻度线应位于绘图区域外部。 |

## Image::IsAlphaPixelFormat(Imaging::PixelFormat) 方法


### 参数

| Parameter | Type | Description |
| --- | --- | --- |
| pixfmt | [Imaging::PixelFormat](../../../system.drawing.imaging/pixelformat/) | 用于测试的像素格式 |

### 返回值

如果 **pixfmt** 包含 alpha 信息则返回 true，否则返回 false

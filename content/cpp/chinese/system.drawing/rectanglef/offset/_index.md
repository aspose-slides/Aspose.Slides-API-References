## RectangleF::Offset(const PointF\&) 方法

将当前对象表示的矩形的位置按指定的量进行偏移。

```cpp
void System::Drawing::RectangleF::Offset(const PointF &pos)
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| pos | const [PointF](../../pointf/)\& | [PointF](../../pointf/) 对象，指定矩形偏移的量；点的 X 坐标值指定要添加到矩形左上角 X 坐标的量，点的 Y 坐标指定要添加到矩形左上角 Y 坐标的量 |

## RectangleF::Offset(float, float) 方法

将当前对象表示的矩形的位置按指定的量进行偏移。

```cpp
void System::Drawing::RectangleF::Offset(float x, float y)
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| x | **float** | 要添加到矩形当前 X 坐标的值 |
| y | **float** | 要添加到矩形当前 Y 坐标的值 |

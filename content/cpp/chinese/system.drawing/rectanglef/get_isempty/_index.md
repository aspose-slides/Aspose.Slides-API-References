## RectangleF::get_IsEmpty() const 方法

```cpp
bool System::Drawing::RectangleF::get_IsEmpty() const
```

如果当前对象表示的矩形的左上角 X 和 Y 坐标以及其宽度和高度的值为 0，则返回 True，否则返回 false

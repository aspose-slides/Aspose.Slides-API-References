## StringDigitSubstitute 枚举

指定根据区域设置或语言对字符串中的数字进行替换的方式。

```cpp
enum class StringDigitSubstitute
```

### 取值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| User | 0 | 用户定义的替换。 |
| None | 1 | 替换已禁用。 |
| National | 2 | 替换对应用户区域设置的官方国家语言的数字。 |
| Traditional | 3 | 替换对应用户本土脚本或语言的数字，该语言可能与用户区域设置的官方国家语言不同。 |

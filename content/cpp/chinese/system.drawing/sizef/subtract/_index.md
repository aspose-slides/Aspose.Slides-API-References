## SizeF::Subtract(const SizeF\&, const SizeF\&) method

返回一个新的 [SizeF](../) 对象，该对象是从 **size1** 中减去 **size2** 的结果，即其宽度值为 **size2's** 宽度值从 **size1's** 宽度值减去后的结果，高度值为 **size2's** 高度值从 **size1's** 高度值减去后的结果。

```cpp
static SizeF System::Drawing::SizeF::Subtract(const SizeF &size1, const SizeF &size2)
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| size1 | const [SizeF](../)\& | [SizeF](../) 对象，用于减去 |
| size2 | const [SizeF](../)\& | [SizeF](../) 对象，用于减去 |

### 返回值

[SizeF](../) 对象，即从 **size1** 中减去 **size2** 的结果。

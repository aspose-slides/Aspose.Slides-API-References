## SizeF::Add(const SizeF\&, const SizeF\&) 方法

返回一个新的 [SizeF](../) 对象，该对象是指定 [SizeF](../) 对象的和，即其宽度值等于指定对象的宽度值之和，且高度值等于指定对象的高度值之和。

```cpp
static SizeF System::Drawing::SizeF::Add(const SizeF &size1, const SizeF &size2)
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| size1 | const [SizeF](../)\& | 第一个操作数 |
| size2 | const [SizeF](../)\& | 第二个操作数 |

### 返回值

[SizeF](../) 对象，它是 **size1** 和 **size2** 的和

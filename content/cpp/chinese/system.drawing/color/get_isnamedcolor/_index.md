## Color::get_IsNamedColor() const method


返回一个值，用于确定 [Color](../) 结构是否表示命名颜色或 KnownColor 枚举的成员。

```cpp
bool System::Drawing::Color::get_IsNamedColor() const
```


### 返回值

如果 [Color](../) 是使用 FromName(String) 方法或 [FromKnownColor(KnownColor)](../fromknowncolor/) 方法创建的，则返回 true；否则返回 false。

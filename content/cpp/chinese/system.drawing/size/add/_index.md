## Size::Add(const Size\&, const Size\&) 方法

返回一个新的 [Size](../) 对象，该对象是指定 [Size](../) 对象的和，即其宽度值等于指定对象的宽度值之和，高度值等于指定对象的高度值之和。

```cpp
static Size System::Drawing::Size::Add(const Size &size1, const Size &size2)
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| size1 | const [Size](../)\& | 第一个操作数 |
| size2 | const [Size](../)\& | 第二个操作数 |

### 返回值

[Size](../) 对象，它是 **size1** 和 **size2** 的和

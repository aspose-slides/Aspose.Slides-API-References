## Size::Subtract(const Size\&, const Size\&) 方法

返回一个新的 [Size](../) 对象，该对象是 **size2** 从 **size1** 减去的结果，即其宽度值是 **size1** 的宽度值减去 **size2** 的宽度值的结果，高度值是 **size1** 的高度值减去 **size2** 的高度值的结果。

```cpp
static Size System::Drawing::Size::Subtract(const Size &size1, const Size &size2)
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| size1 | const [Size](../)\& | 用于减去的 [Size](../) 对象 |
| size2 | const [Size](../)\& | 用于减去的 [Size](../) 对象 |

### 返回值

从 **size1** 中减去 **size2** 的结果 [Size](../) 对象

## RotateFlipType 枚举

指定旋转和/或翻转操作的类型。

```cpp
enum class RotateFlipType
```

### Values

| 名称 | 值 | 描述 |
| --- | --- | --- |
| RotateNoneFlipNone | 0 | 没有旋转，也没有翻转。 |
| Rotate90FlipNone | 1 | 顺时针旋转 90 度，且不进行翻转。 |
| Rotate180FlipNone | 2 | 顺时针旋转 180 度，且不进行翻转。 |
| Rotate270FlipNone | 3 | 顺时针旋转 270 度，且不进行翻转。 |
| RotateNoneFlipX | 4 | 水平翻转，且没有旋转。 |
| Rotate90FlipX | 5 | 顺时针旋转 90 度，并水平翻转。 |
| Rotate180FlipX | 6 | 顺时针旋转 180 度，并水平翻转。 |
| Rotate270FlipX | 7 | 顺时针旋转 270 度，并水平翻转。 |
| RotateNoneFlipY | n/a | 垂直翻转，且没有旋转。 |
| Rotate90FlipY | n/a | 顺时针旋转 90 度，并垂直翻转。 |
| Rotate180FlipY | n/a | 顺时针旋转 180 度，并垂直翻转。 |
| Rotate270FlipY | n/a | 顺时针旋转 270 度，并垂直翻转。 |
| RotateNoneFlipXY | n/a | 水平和垂直翻转，且没有旋转。 |
| Rotate90FlipXY | n/a | 顺时针旋转 90 度，并水平和垂直翻转。 |
| Rotate180FlipXY | n/a | 顺时针旋转 180 度，并水平和垂直翻转。 |
| Rotate270FlipXY | n/a | 顺时针旋转 270 度，并水平和垂直翻转。 |

## SystemColors 类

类提供一组预创建的 [Color](../color/) 对象，表示 [Windows](../../system.windows/) 显示元素的颜色。这是一个静态类型，不提供实例服务。您绝不应以任何方式创建其实例。

```cpp
class SystemColors
```

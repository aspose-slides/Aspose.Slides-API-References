## Rectangle::Offset(const Point\&) 方法


将当前对象表示的矩形的位置按指定量进行偏移。

```cpp
void System::Drawing::Rectangle::Offset(const Point &pos)
```


### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| pos | const [Point](../../point/)\& | 指定矩形偏移量的 [Point](../../point/) 对象；点的 X 坐标值指定要添加到矩形左上角 X 坐标的量，点的 Y 坐标指定要添加到矩形左上角 Y 坐标的量 |

## Rectangle::Offset(int, int) 方法


将当前对象表示的矩形的位置按指定量进行偏移。



### 参数

| 参数 | 类型 | 描述 |

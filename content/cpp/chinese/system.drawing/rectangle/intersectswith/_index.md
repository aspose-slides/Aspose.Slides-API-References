## Rectangle::IntersectsWith(const Rectangle\&) 方法

确定当前对象和指定对象表示的矩形是否相交。

```cpp
bool System::Drawing::Rectangle::IntersectsWith(const Rectangle &rect)
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| rect | const [Rectangle](../)\& | 要检查的矩形 |

### 返回值

如果当前对象和 **rect** 表示的矩形相交，则返回 true；否则返回 false

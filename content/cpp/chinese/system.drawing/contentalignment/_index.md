## ContentAlignment 枚举

指定内容对齐方式。

```cpp
enum class ContentAlignment
```

### Values

| 名称 | 值 | 描述 |
| --- | --- | --- |
| TopLeft | 1 | 在垂直方向上对齐到顶部，水平方向上对齐到左侧。 |
| TopCenter | 2 | 在垂直方向上对齐到顶部，水平方向上对齐到中心。 |
| TopRight | 4 | 在垂直方向上对齐到顶部，水平方向上对齐到右侧。 |
| MiddleLeft | 16 | 在垂直方向上对齐到中间，水平方向上对齐到左侧。 |
| MiddleCenter | 32 | 在垂直方向上对齐到中间，水平方向上对齐到中心。 |
| MiddleRight | 64 | 在垂直方向上对齐到中间，水平方向上对齐到右侧。 |
| BottomLeft | 256 | 在垂直方向上对齐到底部，水平方向上对齐到左侧。 |
| BottomCenter | 512 | 在垂直方向上对齐到底部，水平方向上对齐到中心。 |
| BottomRight | 1024 | 在垂直方向上对齐到底部，水平方向上对齐到右侧。 |

## StringFormatFlags 枚举

指定文本字符串的显示和布局信息。

```cpp
enum class StringFormatFlags
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| DirectionRightToLeft | 1 | 从右到左的文本方向。 |
| DirectionVertical | 2 | [Text](../../system.drawing.text/) 垂直对齐。 |
| FitBlackBox | 4 | 字符的部分可以超出布局框。 |
| DisplayFormatControl | 32 | 控制字符被显示。 |
| NoFontFallback | 1024 | [Font](../font/) 回退已禁用。 |
| MeasureTrailingSpaces | 2048 | 在每行末尾插入尾随空格。 |
| NoWrap | 4096 | [Text](../../system.drawing.text/) 自动换行已禁用。 |
| LineLimit | 8192 | 整行在格式矩形中布局。 |
| NoClip | 16384 | 允许显示字形的超出部分。 |

## Point::Offset(int, int) 方法

```cpp

```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| dx | int | 用于偏移 X 坐标值的值 |
| dy | int | 用于偏移 Y 坐标值的值 |

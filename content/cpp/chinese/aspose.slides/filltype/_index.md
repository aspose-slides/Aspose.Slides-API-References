## FillType 枚举

Specifies the interior fill type of various visual objects.

```cpp
enum class FillType : int8_t
```

### 值

| 名称 | 值 | 说明 |
| --- | --- | --- |
| NotDefined | -1 | 填充类型未定义。 |
| NoFill | 0 | 未应用填充。 |
| Solid | 1 | 使用纯色填充。 |
| Gradient | 2 | 填充为渐变。 |
| Pattern | 3 | 使用重复图案填充对象。 |
| Picture | 4 | 使用单张图片填充对象。 |
| Group | 5 | 可视对象从组继承填充属性。 |

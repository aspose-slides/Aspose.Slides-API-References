## PathFillModeType 枚举


指定应如何填充路径

```cpp
enum class PathFillModeType : uint8_t
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| None | 0 | 无路径填充 这指定相应的路径不应有填充。 |
| Normal | 1 | 普通路径填充 这指定相应的路径应使用普通阴影颜色应用于它的填充。 |
| Lighten | 2 | 变亮路径填充 这指定相应的路径应使用轻度阴影颜色应用于它的填充。 |
| LightenLess | 3 | 轻微变亮路径填充 这指定相应的路径应使用稍微浅一点的阴影颜色应用于它的填充。 |
| Darken | 4 | 加深路径填充 这指定相应的路径应使用更深的阴影颜色应用于它的填充。 |
| DarkenLess | 5 | 轻微加深路径填充 这指定相应的路径应使用稍微深一点的阴影颜色应用于它的填充。 |

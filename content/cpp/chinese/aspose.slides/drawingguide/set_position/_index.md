## DrawingGuide::set_Position(float) 方法

设置绘图指南相对于幻灯片左上角的垂直位置（以点为单位）。写入 **float**。

```cpp
void Aspose::Slides::DrawingGuide::set_Position(float value) override
```

## 备注

典型的取值范围是水平指南从零到幻灯片高度，垂直指南从零到幻灯片宽度。

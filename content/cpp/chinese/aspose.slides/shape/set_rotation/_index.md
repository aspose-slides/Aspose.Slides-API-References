## Shape::set_Rotation(float) 方法

设置指定形状绕 z 轴旋转的度数。正值表示顺时针旋转；负值表示逆时针旋转。写入 **float**。

```cpp
void Aspose::Slides::Shape::set_Rotation(float value) override
```

## 备注

返回值始终已定义（不是 std::numeric_limits<float>::quiet_NaN()）。分配的值必须已定义（不是 std::numeric_limits<float>::quiet_NaN()）。您可以为 RawFrame 实例属性设置未定义的值。

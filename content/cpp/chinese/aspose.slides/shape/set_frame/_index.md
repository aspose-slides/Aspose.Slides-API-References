## Shape::set_Frame(System::SharedPtr\<IShapeFrame\>) 方法

设置形状框架的属性。写入 [IShapeFrame](../../ishapeframe/)。

```cpp
void Aspose::Slides::Shape::set_Frame(System::SharedPtr<IShapeFrame> value) override
```

## 备注

返回的 [IShapeFrame](../../ishapeframe/) 实例的每个属性的值均不是未定义的（不是 NaN 或 NotDefined）。分配的 [IShapeFrame](../../ishapeframe/) 实例的每个属性的值必须不是未定义的（必须不是 NaN 或 NotDefined）。可以为 RawFrame 实例的属性设置未定义的值。

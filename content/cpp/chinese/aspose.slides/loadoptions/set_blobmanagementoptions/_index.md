## LoadOptions::set_BlobManagementOptions(System::SharedPtr\<IBlobManagementOptions\>) 方法

表示可用于管理二进制大对象（BLOB）处理行为的选项，例如使用临时文件或在内存中限制 BLOB 的最大字节数。这些选项旨在为特定环境或需求设置最佳的性能/内存消耗比。

```cpp
void Aspose::Slides::LoadOptions::set_BlobManagementOptions(System::SharedPtr<IBlobManagementOptions> value) override
```

## 备注

二进制大对象（BLOB）是作为单个实体存储的二进制数据——即 BLOB 本身可以是音频、视频或演示文稿本身。

## LineSketchType 枚举

表示已分配给形状的草图类型或效果。

```cpp
enum class LineSketchType
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| NotDefined | -1 | 指定形状的 Sketch 效果为未定义。 |
| None | 0 | 指定形状没有 Sketch 效果。这相当于此属性为空。 |
| Curved | 1 | 指定形状具有 Curved 效果，该效果会将形状的每条边转换为一个大的柔和曲线。 |
| Freehand | 2 | 指定形状具有 Freehand 效果，该效果最接近不完美的手绘线条。 |
| Scribble | 3 | 指定形状具有 Scribble 效果，该效果具有夸张的摇摆，仿佛故意绘制得凌乱。 |

## IProtectionManager::set_EncryptDocumentProperties(bool) 方法

此属性在演示文稿受密码保护时才有意义。如果 true 则文档属性在演示文件中被加密。如果 false 则文档属性是公开的，而演示文稿则被加密。写入 **bool**。

```cpp
virtual void Aspose::Slides::IProtectionManager::set_EncryptDocumentProperties(bool value)=0
```

## TextAutofitType 枚举

表示文本自动适应模式。

```cpp
enum class TextAutofitType : int8_t
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| NotDefined | -1 | 未定义。 |
| None | 0 | 无自动适应。 |
| Normal | 1 | 正常自动适应。字体大小和行间距将被缩小以适应形状。 |
| Shape | 2 | [Shape](../shape/) 自动适应。[Shape](../shape/) 大小将被更改以适应文本。 |

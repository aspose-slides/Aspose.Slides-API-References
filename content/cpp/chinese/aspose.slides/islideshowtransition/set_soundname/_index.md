## ISlideShowTransition::set_SoundName(System::String) method

指定用于过渡音效的人类可读名称。 必须为 [ISlideShowTransition::set_Sound](../set_sound/) 赋值才能获取或设置音效名称。写入 [System::String](../../../system/string/)。

```cpp
virtual void Aspose::Slides::ISlideShowTransition::set_SoundName(System::String value)=0
```

## 备注

当手动配置过渡音效时，此名称会出现在 PowerPoint 用户界面中。

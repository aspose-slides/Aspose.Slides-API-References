## ISlideShowTransition::set_SoundIsBuiltIn(bool) 方法

指定此声音是否为内置声音。如果将此属性设置为 true，则生成应用程序会被提醒检查此声音在其内置声音列表中指定的 name 属性，并可根据需要显示自定义名称或 UI。写入 **bool**。

virtual void Aspose::Slides::ISlideShowTransition::set_SoundIsBuiltIn(bool value)=0

## ISlideShowTransition::set_Duration(int32_t) 方法

设置幻灯片切换效果的持续时间（单位为毫秒）。写入 **int32_t**。

```cpp
virtual void Aspose::Slides::ISlideShowTransition::set_Duration(int32_t value)=0
```

## 备注

对应 PresentationML 架构中 **p:transition** 元素的 **p14:dur** 属性。如果未设置，则持续时间会根据 [Speed](../) 属性和过渡类型自动确定。

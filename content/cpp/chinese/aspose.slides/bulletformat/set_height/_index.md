## BulletFormat::set_Height(float) 方法


设置段落的项目符号高度，不进行继承。值 std::numeric_limits<float>::quiet_NaN() 表示项目符号从段落的第一段继承高度。写入 **float**.

```cpp
void Aspose::Slides::BulletFormat::set_Height(float value) override
```

## 备注


负高度值表示高度以点给出，正值表示高度是周围文本的百分比。

## BulletFormat::get_Height() 方法

返回段落的项目符号高度，且不继承。Value std::numeric_limits<float>::quiet_NaN() determines that bullet inherits height from the first portion in the paragraph. Read **float**.

```cpp
float Aspose::Slides::BulletFormat::get_Height() override
```

## 备注

负高度值表示以点为单位给出的高度，正值表示高度是周围文本的百分比。

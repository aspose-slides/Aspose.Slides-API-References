## PathCommandType enum

表示图形路径命令

```cpp
enum class PathCommandType : uint8_t
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Close | 0 | 关闭几何路径 |
| MoveTo | 1 | 设置几何路径的下一个点位置 |
| LineTo | 2 | 向几何路径追加线段 |
| ArcTo | 3 | 向几何路径追加指定的弧线。 |
| QuadBezierTo | 4 | 向几何路径追加二次贝塞尔曲线 |
| CubicBezierTo | 5 | 向几何路径追加三次贝塞尔曲线 |

## TextVerticalType 枚举

确定文本的垂直书写模式。

```cpp
enum class TextVerticalType : int8_t
```

### 取值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| NotDefined | -1 | 未定义。 |
| Horizontal | 0 | 水平文本。 |
| Vertical | 1 | 垂直文本。 |
| Vertical270 | 2 | 垂直 270 度文本。 |
| WordArtVertical | 3 | WordArt 垂直文本。 |
| EastAsianVertical | 4 | 东亚垂直文本。 |
| MongolianVertical | 5 | 蒙古垂直文本。 |
| WordArtVerticalRightToLeft | 6 | WordArt 垂直从右到左文本。 |

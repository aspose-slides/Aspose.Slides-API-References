## ColorType 枚举

表示不同的颜色模式。

```cpp
enum class ColorType
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| NotDefined | -1 | 颜色根本未定义。 |
| RGB | 0 | 标准 24 位 RGB 颜色。 |
| RGBPercentage | 1 | 高分辨率 RGB 颜色。 |
| HSL | 2 | 高分辨率 HSL 颜色。 |
| Scheme | 3 | 方案颜色。 |
| System | 4 | [System](../../system/) 颜色。 |
| Preset | 5 | 预设颜色。 |

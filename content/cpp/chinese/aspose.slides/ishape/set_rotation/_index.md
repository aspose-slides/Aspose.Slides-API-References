## IShape::set_Rotation(float) 方法

设置指定形状绕 z 轴旋转的角度（以度为单位）。正值表示顺时针旋转；负值表示逆时针旋转。写入 **float**。

```cpp
virtual void Aspose::Slides::IShape::set_Rotation(float value)=0
```

## 备注

返回值始终是已定义的（不是 std::numeric_limits<float>::quiet_NaN()）。分配的值必须是已定义的（不是 std::numeric_limits<float>::quiet_NaN()）。可以为 RawFrame 实例属性设置未定义的值。

## System::LEqual(const ExpressionT&, const ConstantT&) 函数

```cpp
template<class ExpressionT,class ConstantT> bool System::LEqual(const ExpressionT &left, const ConstantT &constant)
```

### 模板参数

| 参数 | 描述 |
| --- | --- |
| ExpressionT | 左表达式类型。 |
| ConstantT | 常量表达式的类型。 |

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| left | const ExpressionT& | 将被检查的表达式。 |
| constant | const ConstantT& | 将与左侧表达式比较的表达式。 |

### 返回值

true if type check is successfull, false otherwise.

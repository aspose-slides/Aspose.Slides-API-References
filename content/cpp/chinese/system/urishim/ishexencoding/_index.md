## UriShim::IsHexEncoding(const String\&, int32_t) 方法

检查给定模式是否为转义的十六进制形式。

```cpp
static bool System::UriShim::IsHexEncoding(const String &pattern, int32_t index)
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| pattern | const [String](../../string/)\& | [String](../../string/) 读取转义的十六进制形式的来源。 |
| index | **int32_t** | 给定字符串中的偏移量。 |

### 返回值

如果在给定位置存在转义的十六进制模式则返回 true，否则返回 false。

## SmartPtrMode 枚举


[SmartPtr](../smartptr/) 指针类型：弱或共享。定义在决定是否删除对象时指针是否计入引用计数。

```cpp
enum class SmartPtrMode : char
```

### 值

| Name | Value | Description |
| --- | --- | --- |
| Shared | 0 | 共享模式：指针参与引用计数。 |
| Weak | 1 | 弱模式：指针不参与引用计数。 |

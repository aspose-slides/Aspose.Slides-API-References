## DateTime::AddMilliseconds(double) const 方法

返回一个新的 [DateTime](../) 类实例，表示日期和时间值，该值等于当前对象表示的值与指定的毫秒数之和。

```cpp
DateTime System::DateTime::AddMilliseconds(double value) const
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| value | **double** | 要添加的毫秒数；该值可以是正数或负数 |

### 返回值

一个新的 [DateTime](../) 类实例，表示日期和时间值，该值等于当前对象表示的值与 **value** 毫秒之和

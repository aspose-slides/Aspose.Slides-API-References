## DateTime::AddTicks(int64_t) const 方法

返回一个新的 [DateTime](../) 类实例，表示日期和时间值，该值是当前对象表示的值与指定的 100 纳秒间隔数之和。

```cpp
DateTime System::DateTime::AddTicks(int64_t value) const
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| value | **int64_t** | 要添加的 100 纳秒间隔的数量；value 可以是正数或负数 |

### 返回值

一个新的 [DateTime](../) 类实例，表示日期和时间值，其等于当前对象表示的值与 **value** 个 100 纳秒间隔的和

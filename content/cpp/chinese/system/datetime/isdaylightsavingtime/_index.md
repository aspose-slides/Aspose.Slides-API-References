## DateTime::IsDaylightSavingTime() const 方法

确定当前对象表示的日期和时间值是否位于当前时区的夏令时范围内。

```cpp
bool System::DateTime::IsDaylightSavingTime() const
```

### 返回值

如果当前对象指定的日期和时间值位于夏令时范围内，则为 true；否则为 false

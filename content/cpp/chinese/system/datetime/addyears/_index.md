## DateTime::AddYears(int) const 方法

返回一个新的 [DateTime](../) 类实例，该实例表示的日期时间值等同于当前对象表示的值，但年份组件已增加指定的数量。

```cpp
DateTime System::DateTime::AddYears(int value) const
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| value | int | 要添加的年数；该值可以为正或负 |

### 返回值

返回一个新的 [DateTime](../) 类实例，该实例表示的日期时间值等同于当前对象表示的值，但年份组件已增加 **value**。

## DateTime::Equals(DateTime) const 方法


确定指定的 [DateTime](../) 类实例是否代表与当前对象相同的日期和时间值。

```cpp
constexpr bool System::DateTime::Equals(DateTime other) const
```


### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| other | [DateTime](../) | 与当前对象比较的 [DateTime](../) 类实例 |

### 返回值

如果 **other** 和当前对象表示相同的值则为 true，否则为 false

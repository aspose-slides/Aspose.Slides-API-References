## DateTime::Add(TimeSpan) const 方法

返回一个新的 [DateTime](../) 类实例，表示将指定的时间跨度添加到当前对象表示的日期和时间值后得到的日期和时间值。

```cpp
DateTime System::DateTime::Add(TimeSpan value) const
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| value | [TimeSpan](../../timespan/) | 要添加的时间间隔 |

### 返回值

一个新的 [DateTime](../) 类实例，表示将 **tm** 时间跨度添加到当前对象表示的日期和时间值后得到的日期和时间值

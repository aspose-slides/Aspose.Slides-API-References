## Char::ToUpperInvariant(char_t) 方法

将指定字符转换为大写。

```cpp
static char_t System::Char::ToUpperInvariant(char_t c)
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| c | char_t | 要转换的字符 |

### 返回值

如果指定字符是小写字母，则返回其大写形式；否则返回原字符。

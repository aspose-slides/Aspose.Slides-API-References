## Char::Parse(const String\&) 方法


将指定字符串的第一个且唯一的字符转换为 char_t 值。

```cpp
static char_t System::Char::Parse(const String &value)
```


### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| value | const [String](../../string/)\& | 要转换的字符串。 |

### 返回值

如果指定字符串恰好为 1 个字符，则返回该字符串的第一个且唯一的字符。

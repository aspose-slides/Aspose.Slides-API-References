## Char::IsPunctuation(const char_t *, int) 方法

确定在指定字符缓冲区的指定索引处的字符是否被分类为标点符号。

```cpp
static bool System::Char::IsPunctuation(const char_t *str, int idx)
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| str | const char_t * | 指向字符缓冲区起始位置的指针 |
| idx | int | 缓冲区中要测试的字符的零基索引 |

### 返回值

如果指定索引处的字符是标点符号，则为 true；否则为 false

## Char::IsPunctuation(char_t) 方法

确定指定字符是否被分类为标点符号。

```cpp
static bool System::Char::IsPunctuation(char_t c)
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| c | char_t | 要测试的字符 |

### 返回值

如果指定字符是标点符号，则为 true；否则为 false

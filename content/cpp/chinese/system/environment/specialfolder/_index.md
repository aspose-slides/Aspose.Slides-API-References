## SpecialFolder 枚举

表示系统特殊文件夹。

```cpp
enum class SpecialFolder
```

### 值

| 名称 | 数值 | 描述 |
| --- | --- | --- |
| Desktop | 0 | 逻辑 Desktop。 |
| Programs | 2 | 包含用户程序组的文件夹。 |
| Personal | 5 | 用作文档公共存储库的文件夹。 |
| MyDocuments | 5 | "My Documents" 目录。 |
| Favorites | 6 | 用作用户收藏项目公共存储库的文件夹。 |
| Startup | 7 | 对应用户启动程序组的文件夹。 |
| Recent | 8 | 包含用户最近使用文档的文件夹。 |
| SendTo | 9 | 用作“发送到”菜单项容器的文件夹。 |
| StartMenu | 11 | 用作“开始”菜单项容器的文件夹。 |
| MyMusic | 13 | "My Music" 目录。 |
| DesktopDirectory | 16 | 用于在桌面上实际存储项目的文件夹。 |
| MyComputer | 17 | "My Computer" 目录。 |
| Templates | 21 | 用作文档模板公共容器的文件夹。 |
| ApplicationData | 26 | 用于为当前漫游用户存储特定于应用程序的数据的文件夹。 |
| LocalApplicationData | 28 | 用于为当前非漫游用户存储特定于应用程序的数据的文件夹。 |
| InternetCache | 32 | 用于存放临时 Internet 文件的存储空间。 |
| Cookies | 33 | 用于存放 Internet Cookie 的存储空间。 |
| History | 34 | 用于存放 Internet 历史项目的存储空间。 |
| CommonApplicationData | 35 | 用于存放所有用户使用的特定于应用程序的数据的文件夹。 |
| System | 37 | 系统文件夹。 |
| ProgramFiles | 38 | 程序文件夹。 |
| MyPictures | 39 | "My picutres" 文件夹。 |
| UserProfile | 40 | 用户的配置文件夹。 |
| CommonProgramFiles | 43 | 用作共享组件存储的目录。 |

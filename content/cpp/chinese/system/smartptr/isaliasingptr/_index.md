## SmartPtr::IsAliasingPtr() const 方法

检查指针是否指向了不同于拥有对象的另一个对象（由别名构造函数创建）。

```cpp
bool System::SmartPtr<T>::IsAliasingPtr() const
```

### 返回值

如果指针指向的对象与拥有的对象不同则返回 true，否则返回 false。

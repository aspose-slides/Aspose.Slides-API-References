## SmartPtr::static_pointer_cast() const 方法

使用 static_cast 将指针转换为指向对象的不同类型。

```cpp
template<class Y> SmartPtr<Y> System::SmartPtr<T>::static_pointer_cast() const
```

### 模板参数

| 参数 | 描述 |
| --- | --- |
| Y | 指向对象的目标类型。 |

### 返回值

更改类型后的指针，始终为共享模式。

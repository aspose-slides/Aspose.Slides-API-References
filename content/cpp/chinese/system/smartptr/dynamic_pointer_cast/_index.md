## SmartPtr::dynamic_pointer_cast() const 方法

使用 dynamic_cast 将指针转换为指向对象的不同类型。

```cpp
template<class Y> SmartPtr<Y> System::SmartPtr<T>::dynamic_pointer_cast() const
```

### 模板参数

| 参数 | 描述 |
| --- | --- |
| Y | 指向对象的目标类型。 |

### 返回值

指针的类型已更改，始终处于共享模式。

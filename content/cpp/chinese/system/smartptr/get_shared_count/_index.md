## SmartPtr::get_shared_count() const 方法

获取引用对象上现有的共享指针数量，包括当前的指针。断言当前指针处于共享模式。

```cpp
int System::SmartPtr<T>::get_shared_count() const
```

### 返回值

引用对象上现有的共享指针数量（如果有的话）。如果指针为 null，则返回 0。

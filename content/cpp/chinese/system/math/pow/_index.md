## Math::Pow(double, double) 方法

返回指定值的指定次幂。

```cpp
static double System::Math::Pow(double x, double y)
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| x | **double** | 要提升到指定幂的值 |
| y | **double** | 要将指定值提升到的幂 |

### 返回值

**x** 提升到 **y** 的幂

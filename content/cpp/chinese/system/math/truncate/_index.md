## Math::Truncate(const Decimal\&) 方法

返回表示一个值的 [Decimal](../../decimal/) 对象，该值的整数部分等于由指定的 [Decimal](../../decimal/) 对象表示的值的整数部分，并且所有小数位均被截除。

```cpp
static Decimal System::Math::Truncate(const Decimal &d)
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| d | const [Decimal](../../decimal/)\& | 一个十进制数 |

### 返回值

一个 [Decimal](../../decimal/) 对象，表示一个十进制值，其整数部分等于指定值的整数部分，且所有小数位均为 0。

## Math::Truncate(double) 方法

返回一个双精度浮点值，其整数部分等于指定值的整数部分，并且所有小数位均被截除。

```cpp
static double System::Math::Truncate(double d)
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| d | **double** | 一个十进制数 |

### 返回值

一个双精度浮点值，其整数部分等于指定值的整数部分，且所有小数位均为 0。

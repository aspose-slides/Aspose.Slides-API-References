## Uri::IsHexEncoding(const String\&, int32_t) 方法

确定指定字符串中位于指定位置的字符是否为十六进制编码。

```cpp
static bool System::Uri::IsHexEncoding(const String &pattern, int32_t index)
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| pattern | const [String](../../string/)\& | 要检查的字符串 |
| index | **int32_t** | 要测试的字符的零基位置 |

### 返回值

如果指定字符是十六进制编码，则为 True，否则为 false

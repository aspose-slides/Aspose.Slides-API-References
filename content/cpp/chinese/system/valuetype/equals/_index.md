## ValueType::Equals(const SharedPtr\<Object\>\&) 方法

支持值类型和引用类型的比较。

```cpp
static bool System::ValueType::Equals(const SharedPtr<Object> &o)
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| o | const [SharedPtr](../../sharedptr/)\<[Object](../../object/)\>\& | 要比较的引用类型对象。 |

### 返回值

如果对象被认为相等则返回 True，否则返回 false。

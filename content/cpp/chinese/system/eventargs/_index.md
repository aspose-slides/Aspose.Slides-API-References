## EventArgs 类


这是表示在事件触发时传递给事件订阅者的上下文的基类。此类的对象只能使用 [System::MakeObject()](../makeobject/) 函数分配。绝不要在栈上或使用 new 操作符创建此类型的实例，因为这会导致运行时错误和/或断言故障。始终将此类包装成 [System::SmartPtr](../smartptr/) 指针，并使用该指针将其作为参数传递给函数。

```cpp
class EventArgs : public System::Object
```

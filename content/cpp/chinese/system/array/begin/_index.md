## Array::begin() 方法

返回一个指向容器中第一个元素的迭代器。如果容器为空，返回的迭代器将等于 [end()](../end/)。

```cpp
iterator System::Array<T>::begin() noexcept
```

### 返回值

指向容器中第一个元素的迭代器。

## Array::begin() const 方法

返回一个指向 const 限定容器中第一个元素的迭代器。如果容器为空，返回的迭代器将等于 [end()](../end/)。

```cpp
const_iterator System::Array<T>::begin() const noexcept
```

### 返回值

指向 const 限定容器中第一个元素的迭代器。

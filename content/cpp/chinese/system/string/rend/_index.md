## String::rend() const 方法

返回反向迭代器，指向实际字符串缓冲区的第一个字符之前的位置（如果有）。

```cpp
reverse_iterator System::String::rend() const
```

### 返回值

反向迭代器，指向实际字符串缓冲区的第一个字符之前的元素（如果有），或者指向已分配缓冲区之前的元素。

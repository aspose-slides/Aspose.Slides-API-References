
### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| target | const [System::TypeInfo](../../typeinfo/)\& | 用于检查的 [TypeInfo](../../typeinfo/)。 |

### 返回值

如果 target 是 [String](../) 类的 [TypeInfo](../../typeinfo/) 则返回 true，否则返回 false。

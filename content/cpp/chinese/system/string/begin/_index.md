## String::begin() const 方法

返回指向实际字符串缓冲区开头的指针。永不重新分配任何内容。不保证缓冲区以 null 结尾。

```cpp
const UChar * System::String::begin() const
```

### 返回值

指向实际字符串缓冲区的指针，无论是否以 null 结尾。

## String::end() const 方法

返回指向实际字符串缓冲区末尾的指针。永不重新分配任何内容。不能保证缓冲区为 null 结尾。

```cpp
const UChar * System::String::end() const
```

### 返回值

指向实际字符串缓冲区之后的第一个元素的指针，无论是否为 null 结尾。

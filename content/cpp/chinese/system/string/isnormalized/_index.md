## String::IsNormalized(System::Text::NormalizationForm) const 方法

检查 Unicode 字符串是否已使用指定的标准化形式规范化。

```cpp
bool System::String::IsNormalized(System::Text::NormalizationForm form) const
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| form | [System::Text::NormalizationForm](../../../system.text/normalizationform/) | 标准化形式，详细信息请参见 [System::Text::NormalizationForm](../../../system.text/normalizationform/)。 |

### 返回值

如果字符串已被规范化则返回 true，否则返回 false。

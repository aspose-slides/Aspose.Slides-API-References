## String::IsNull() const 方法

检查字符串是否被视为 null。[String](../) 为 null，且仅当它通过 [String()](../string/) 构造函数构造、移动、复制或从 null 字符串赋值，或者调用了 [reset()](../reset/) 方法时才为 null。

```cpp
bool System::String::IsNull() const
```

### 返回值

如果字符串被视为 null，则为 true；否则为 false。

## String::rbegin() const 方法

返回指向实际字符串缓冲区中最后一个字符（如果有）的反向迭代器。

```cpp
reverse_iterator System::String::rbegin() const
```

### 返回值

指向实际字符串缓冲区中最后一个字符（如果有）的反向迭代器，或与 [rend()](../rend/) 相同。

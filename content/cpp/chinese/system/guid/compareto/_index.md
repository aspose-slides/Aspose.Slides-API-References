## Guid::CompareTo(const Guid\&) const 方法

对当前对象和指定对象所表示的 GUID 进行算术比较。

```cpp
int System::Guid::CompareTo(const Guid &g) const
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| g | const [Guid](../)\& | 比较对象 |

### 返回值

- 1 表示当前对象表示的 GUID 小于 **g** 表示的 GUID；0 表示 GUID 相等；1 表示当前对象表示的 GUID 大于 **g** 表示的 GUID

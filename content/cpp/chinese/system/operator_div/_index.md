## System::operator/(const T\&, const Decimal\&) 函数

返回一个新的 [Decimal](../decimal/) 类实例，该实例表示指定值与指定 [Decimal](../decimal/) 对象所代表的值相除的结果。

```cpp
template<typename T,typename _> Decimal System::operator/(const T &x, const Decimal &d)
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| x | const T\& | 要除的值 |
| d | const [Decimal](../decimal/)\& | 表示除数的 [Decimal](../decimal/) 对象 |

### 返回值

一个新的 [Decimal](../decimal/) 类实例，它表示 **x** 除以 **d** 所代表的值的结果。

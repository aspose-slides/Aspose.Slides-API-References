## UriFormat 枚举

指定 URI 的转义方式。

```cpp
enum class UriFormat
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| UriEscaped | 1 | 按照 RFC-2396 进行转义。 |
| Unescaped | 2 | 不进行转义。 |
| SafeUnescaped | 3 | 仅转义在 URI 中具有保留含义的字符。 |

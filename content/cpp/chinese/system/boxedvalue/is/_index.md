## BoxedValue::is() const 方法

确定当前对象表示的装箱值的类型是否为 **V**。

```cpp
template<class V> bool System::BoxedValue<T>::is() const
```

### 模板参数

| 参数 | 描述 |
| --- | --- |
| V | 与当前对象表示的装箱值的类型进行比较的类型 |

### 返回值

如果当前对象表示的装箱值的类型是 **V**，则返回 true；否则返回 false

## TypeCode 枚举

表示对象的类型。

```cpp
enum class TypeCode
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Empty | 0 | 空引用。 |
| Object | 1 | 表示任何未由其他 TypeCode 明确表示的引用或值类型的通用类型。 |
| DBNull | 2 | 数据库空（列）值。 |
| Boolean | 3 | 表示 [Boolean](../boolean/) 值为 true 或 false 的简单类型。 |
| Char | 4 | 表示无符号 16 位整数且值在 0 到 65535 之间的整数类型。 |
| SByte | 5 | 表示有符号 8 位整数且值在 -128 到 127 之间的整数类型。 |
| Byte | 6 | 表示无符号 8 位整数且值在 0 到 255 之间的整数类型。 |
| Int16 | 7 | 表示有符号 16 位整数且值在 -32768 到 32767 之间的整数类型。 |
| UInt16 | 8 | 表示无符号 16 位整数且值在 0 到 65535 之间的整数类型。 |
| Int32 | 9 | 表示有符号 32 位整数且值在 -2147483648 到 2147483647 之间的整数类型。 |
| UInt32 | 10 | 表示无符号 32 位整数且值在 0 到 4294967295 之间的整数类型。 |
| Int64 | 11 | 表示有符号 64 位整数且值在 -9223372036854775808 到 9223372036854775807 之间的整数类型。 |
| UInt64 | 12 | 表示无符号 64 位整数且值在 0 到 18446744073709551615 之间的整数类型。 |
| Single | 13 | 表示值范围约为 1.5 x 10 -45 到 3.4 x 10 38，精度为 7 位数字的单精度浮点类型。 |
| Double | 14 | 表示值范围约为 5.0 x 10 -324 到 1.7 x 10 308，精度为 15-16 位数字的双精度浮点类型。 |
| Decimal | 15 | 表示值范围从 1.0 x 10 -28 到约 7.9 x 10 28，具有 28-29 位有效数字的十进制简单类型。 |
| DateTime | 16 | 表示日期和时间值的类型。 |
| String | 18 | 表示 Unicode 字符串的密封类类型。 |

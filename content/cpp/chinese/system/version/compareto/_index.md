## Version::CompareTo(const Version\&) const 方法

Compares the versions represented by the current object and the specified object.

```cpp
int System::Version::CompareTo(const Version &ver) const
```

### 参数

| Parameter | Type | Description |
| --- | --- | --- |
| ver | const [Version](../)\& | 一个 [Version](../) 对象，用于与当前对象比较 |

### 返回值

- 1 如果当前对象表示的版本号小于 **ver** 表示的版本号； 0 如果版本号相等； 1 如果当前对象表示的版本号大于 **ver** 表示的值

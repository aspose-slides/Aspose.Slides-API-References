## System::Set(ExpressionT\&, const ExpressionT\&) 函数


实现 'var' 模式翻译。

```cpp
template<class ExpressionT> bool System::Set(ExpressionT &var, const ExpressionT &value)
```


### 模板参数

| 参数 | 描述 |
| --- | --- |
| ExpressionT | 变量类型。 |

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| var | ExpressionT\& | 对要初始化的变量的引用。 |
| value | const ExpressionT\& | 要分配给变量的值。 |

### 返回值

始终为 true（此模式始终命中）。

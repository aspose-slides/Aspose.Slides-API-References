## Boolean::Parse(const String\&) 方法


将指定的字符串转换为 bool 类型的值。

```cpp
static bool System::Boolean::Parse(const String &value)
```


### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| value | const [String](../../string/)\& | 要转换的字符串。 |

### 返回值

转换结果被放入的 bool 变量；如果指定的字符串等于 "True"，结果为 true；如果指定的字符串等于 "False"，结果为 false。

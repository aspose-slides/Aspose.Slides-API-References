## System::IsNaN(const T\&) 函数

确定指定的值是否为非数字（NaN）值。

```cpp
template<typename T> bool System::IsNaN(const T &value)
```

### 模板参数

| 参数 | 描述 |
| --- | --- |
| T | 函数检查的值的类型 |

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| value | const T\& | 要检查的值 |

### 返回值

如果 **value** 是 NaN 值则返回 true，否则返回 false

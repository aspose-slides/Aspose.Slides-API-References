## StringComparison 枚举

Defines string comparison style.

```cpp
enum class StringComparison
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| CurrentCulture | 0 | 使用当前文化。 |
| CurrentCultureIgnoreCase | 1 | 使用当前文化，但忽略大小写。 |
| InvariantCulture | 2 | 使用不变文化。 |
| InvariantCultureIgnoreCase | 3 | 使用不变文化，但忽略大小写。 |
| Ordinal | 4 | 使用序数排序规则。 |
| OrdinalIgnoreCase | 5 | 使用序数排序规则，但忽略大小写。 |

## Decimal::Compare(const Decimal&, const Decimal&) 方法

确定第一个 [Decimal](../) 对象所表示的值是小于、等于还是大于第二个 [Decimal](../) 对象所表示的值。

```cpp
static int System::Decimal::Compare(const Decimal &d1, const Decimal &d2)
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| d1 | const [Decimal](../)\& | 第一个比较数 |
| d2 | const [Decimal](../)\& | 第二个比较数 |

### 返回值

- 1 表示 **d1** 所表示的值小于 **d2** 所表示的值；0 表示值相等；1 表示 **d1** 所表示的值大于 **d2** 所表示的值

## Decimal::Truncate(const Decimal\&) 方法

返回 [Decimal](../) 对象，表示一个值，其整数部分等于由指定 [Decimal](../) 对象表示的值的整数部分，并且所有小数位均被舍弃。

```cpp
static Decimal System::Decimal::Truncate(const Decimal &d)
```


### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| d | const [Decimal](../)\& | 一个十进制数 |

### 返回值

返回一个 [Decimal](../) 对象，表示一个十进制值，其整数部分等于指定值的整数部分，且所有小数位均为 0。

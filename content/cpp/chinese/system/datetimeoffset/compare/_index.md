## DateTimeOffset::Compare(const DateTimeOffset\&, const DateTimeOffset\&) 方法

比较两个[DateTimeOffset](../)对象。

```cpp
static int System::DateTimeOffset::Compare(const DateTimeOffset &first, const DateTimeOffset &second)
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| first | const [DateTimeOffset](../)\& | 要比较的第一个对象。 |
| second | const [DateTimeOffset](../)\& | 要比较的第二个对象。 |

### 返回值

- 1 - 如果 **first** 早于 **second**，1 - 如果 **first** 晚于 **second**，0 - 如果 **first** 等于 **second**。

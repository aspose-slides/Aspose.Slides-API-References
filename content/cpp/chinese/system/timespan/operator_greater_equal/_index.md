## TimeSpan::operator>=(TimeSpan) const 方法

确定当前对象表示的时间间隔是否长于或等于指定对象表示的时间间隔。

```cpp
constexpr bool System::TimeSpan::operator>=(TimeSpan value) const
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| value | [TimeSpan](../) | 与当前对象比较的 [TimeSpan](../) 对象 |

### 返回值

如果当前对象表示的时间间隔长于或等于 **value** 表示的时间间隔，则返回 True；否则返回 false

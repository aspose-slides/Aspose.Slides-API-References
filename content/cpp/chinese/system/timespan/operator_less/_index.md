## TimeSpan::operator<(TimeSpan) const method

```cpp
constexpr bool System::TimeSpan::operator<(TimeSpan value) const
```


### 参数

| 参数 | 类型 | 描述 |

| value | [TimeSpan](../) | 用于将当前对象与 [TimeSpan](../) 对象进行比较的对象 |

### 返回值

如果当前对象表示的时间间隔比 **value** 表示的时间间隔更短，则为 true，否则为 false

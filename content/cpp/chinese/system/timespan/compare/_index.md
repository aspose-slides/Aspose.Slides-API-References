## TimeSpan::Compare(TimeSpan, TimeSpan) 方法

比较两个 [TimeSpan](../) 对象。

```cpp
static constexpr int System::TimeSpan::Compare(TimeSpan t1, TimeSpan t2)
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| t1 | [TimeSpan](../) | 第一个比较数 |
| t2 | [TimeSpan](../) | 第二个比较数 |

### 返回值

- 1 表示 **t1** 比 **t2** 短；0 表示 **t1** 与 **t2** 相等；1 表示 **t1** 比 **t2** 长

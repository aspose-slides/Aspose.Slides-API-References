```cpp
TimeSpan System::TimeSpan::operator-(TimeSpan value) const
```

### 返回值

返回一个新的 [TimeSpan](../) 类实例，表示一个时间间隔，该间隔是从当前对象表示的时间间隔中减去指定对象表示的时间间隔的结果。

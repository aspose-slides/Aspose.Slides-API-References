## SoapParameterStyle 枚举

枚举 SOAP 消息中的参数格式。

```cpp
enum class SoapParameterStyle
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Default | 0 | 如果未将 '[SoapDocumentServiceAttribute](../soapdocumentserviceattribute/)' 应用于类，则默认值为 'Wrapped'。 |
| Bare | 1 | 参数放置在跟随 'Body' 元素的 XML 元素中。 |
| Wrapped | 2 | 参数封装在跟随 'Body' 元素的单个 XML 元素中。 |

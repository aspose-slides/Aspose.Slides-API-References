## XmlSchemaElement::get_IsNillable() 方法


返回指示 **xsi:nil** 是否可以出现在实例数据中的信息。指示是否可以将显式的 nil 值分配给该元素。

```cpp
bool System::Xml::Schema::XmlSchemaElement::get_IsNillable()
```


### 返回值

如果 nillable 为 **true**，则允许该元素的实例将 **nil** 属性设置为 **true**。**nil** 属性是作为 XML [Schema](../../) 命名空间的一部分为实例定义的。默认值为 **false**。可选。

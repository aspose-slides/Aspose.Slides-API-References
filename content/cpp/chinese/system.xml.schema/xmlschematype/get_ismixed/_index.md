## XmlSchemaType::get_IsMixed() 方法


返回一个值，指示此类型是否具有混合内容模型。此调用仅在复杂类型中有效。

```cpp
virtual bool System::Xml::Schema::XmlSchemaType::get_IsMixed()
```


### 返回值

**true** 如果该类型具有混合内容模型；否则为 **false**。默认值为 **false**。

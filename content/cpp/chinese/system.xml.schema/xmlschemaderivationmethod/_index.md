## XmlSchemaDerivationMethod 枚举

提供防止派生的不同方法。

```cpp
enum class XmlSchemaDerivationMethod
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Empty | 0 | 覆盖默认的派生方法以允许任何派生。 |
| Substitution | 1 | 指代通过 **Substitution** 的派生。 |
| Extension | 2 | 指代通过 **Extension** 的派生。 |
| Restriction | 4 | 指代通过 **Restriction** 的派生。 |
| List | 8 | 指代通过 **List** 的派生。 |
| Union | 16 | 指代通过 **Union** 的派生。 |
| All | 255 | **#all**. 指代所有派生方法。 |
| None | 256 | 接受默认的派生方法。 |

## XmlSchema::get_BlockDefault() 方法


返回 **blockDefault** 属性，该属性设置模式中 **targetNamespace** 下元素和复合类型的 **block** 属性的默认值。

```cpp
XmlSchemaDerivationMethod System::Xml::Schema::XmlSchema::get_BlockDefault()
```


### 返回值

一个 XmlSchemaDerivationMethod 值，表示用于防止派生的不同方法。默认值为 [XmlSchemaDerivationMethod::None](../../xmlschemaderivationmethod/)。

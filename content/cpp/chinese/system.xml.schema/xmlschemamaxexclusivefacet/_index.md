## XmlSchemaMaxExclusiveFacet 类

表示 XML [Schema](../) 中的 **maxExclusive** 元素，符合万维网 [Web](../../system.web/) 联盟 (W3C) 的规格。此类可用于对 **simpleType** 元素的最大值进行限制。元素值必须小于 **maxExclusive** 元素的值。

```cpp
class XmlSchemaMaxExclusiveFacet : public System::Xml::Schema::XmlSchemaFacet
```

## 备注

此 class 的对象应仅使用 [System::MakeObject()](../../system/makeobject/) 函数分配。切勿在栈上或使用 operator new 创建此类型的实例，因为这会导致运行时错误和/或断言故障。始终将此 class 包装在 [System::SmartPtr](../../system/smartptr/) 指针中，并使用该指针将其作为参数传递给函数。

## XmlSchemaTotalDigitsFacet 类

表示来自 XML [Schema](../) 的 **totalDigits** facet，依据万维网 [Web](../../system.web/) 联盟 (W3C) 的规定。此类可用于指定 **simpleType** 元素值可以输入的数字位数的限制。**totalDigits** 的值必须是正整数。

```cpp
class XmlSchemaTotalDigitsFacet : public System::Xml::Schema::XmlSchemaNumericFacet
```

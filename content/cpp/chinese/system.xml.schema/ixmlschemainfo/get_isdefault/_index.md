## IXmlSchemaInfo::get_IsDefault() 方法

返回一个值，指示在 XML [Schema](../../) 定义语言 (XSD) 模式验证期间应用默认值后，此验证的 XML 节点是否被设置为该默认值的结果。

```cpp
virtual bool System::Xml::Schema::IXmlSchemaInfo::get_IsDefault()=0
```

### 返回值

**true** 如果在模式验证期间应用默认值后，此验证的 XML 节点被设置为该默认值的结果；否则为 **false**。

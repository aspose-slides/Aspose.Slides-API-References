## XmlSchemaComplexContent::get_IsMixed() 方法


返回确定该类型是否具有混合内容模型的信息。

```cpp
bool System::Xml::Schema::XmlSchemaComplexContent::get_IsMixed()
```


### 返回值

如果此值为 **true**，则允许在复杂类型的子元素之间出现字符数据（混合内容模型）。默认值为 **false**。可选。

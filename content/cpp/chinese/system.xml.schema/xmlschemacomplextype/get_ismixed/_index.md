## XmlSchemaComplexType::get_IsMixed() 方法

返回信息，用于确定复合类型是否具有混合内容模型（内容中包含标记）。

```cpp
bool System::Xml::Schema::XmlSchemaComplexType::get_IsMixed() override
```

### 返回值

**true**，如果字符数据可以出现在此复合类型的子元素之间；否则为 **false**。默认值为 **false**。可选。

## XmlSchemaForm 枚举


指示属性或元素是否需要使用命名空间前缀进行限定。

```cpp
enum class XmlSchemaForm
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| None | 0 | 在模式中未指定元素和属性的形式。 |
| Qualified | 1 | 元素和属性必须使用命名空间前缀进行限定。 |
| Unqualified | 2 | 元素和属性不需要使用命名空间前缀进行限定。 |

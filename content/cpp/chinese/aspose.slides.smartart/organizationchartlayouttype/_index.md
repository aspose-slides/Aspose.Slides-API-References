## OrganizationChartLayoutType 枚举

表示组织结构图中子节点的格式类型

```cpp
enum class OrganizationChartLayoutType
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Initial | 0 | 初始布局类型 |
| Standart | 1 | 将子节点水平放置在父节点下方。 |
| BothHanging | 2 | 将子节点垂直放置在父节点下方的左侧和右侧。 |
| LeftHanging | 3 | 将子节点垂直放置在父节点左侧下方。 |
| RightHanging | 4 | 将子节点垂直放置在父节点右侧下方。 |

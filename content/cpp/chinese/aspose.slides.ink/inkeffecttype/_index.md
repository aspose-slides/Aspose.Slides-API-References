## InkEffectType 枚举

指定一组预定义的可视效果用于墨水渲染。每种效果对应一种纹理或图像，模拟风格化的墨水外观。这些值可用于在渲染期间自定义数字墨水笔画的视觉风格。

```cpp
enum class InkEffectType
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| NotDefined | -1 | 效果未定义或未知。此值可用作默认或回退。 |
| Bronze | 0 | 温暖的棕色金属纹理，类似青铜墨水。 |
| Galaxy | 1 | 多彩的闪烁纹理，类似星系，带有宇宙色调。 |
| Gold | 2 | 明亮的金属金色纹理，使墨水笔画呈现奢华外观。 |
| Lava | 3 | 炽热的纹理，类似熔岩，带有红色和橙色色调。 |
| Ocean | 4 | 深蓝色的流体纹理，模拟海浪或水基墨水。 |
| RainbowGlitter | 5 | 多彩的闪亮彩虹闪粉效果，用于节日或鲜艳的墨水笔画。 |
| RoseGold | 6 | 柔和的粉金色混合，类似玫瑰金，用于优雅的墨水笔画。 |
| Silver | 7 | 冷色金属银纹理，模拟经典银色墨水。 |

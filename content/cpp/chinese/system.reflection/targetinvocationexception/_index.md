## TargetInvocationException 类型定义

TargetInvocationException 由通过反射调用的方法抛出。永远不要将 TargetInvocationException 类实例包装到 [System::SmartPtr](../../system/smartptr/) 中。

```cpp
using System::Reflection::TargetInvocationException = typedef System::ExceptionWrapper<Details_TargetInvocationException>
```

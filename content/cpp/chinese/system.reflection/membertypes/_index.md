## MemberTypes 枚举


标记每种成员类型。

```cpp
enum class MemberTypes
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Constructor | 1 | 指定该成员是 constructor。 |
| Event | 2 | 指定该成员是 event。 |
| Field | 4 | 指定该成员是 field。 |
| Method | 8 | 指定该成员是 method。 |
| Property | 16 | 指定该成员是 property。 |
| TypeInfo | 32 | 指定该成员是 type。 |
| Custom | 64 | 指定该成员是 custom member type。 |
| NestedType | 128 | 指定该成员是 nested type。 |
| All | 191 | 指定所有成员类型。 |

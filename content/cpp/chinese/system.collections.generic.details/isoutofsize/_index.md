## System::Collections::Generic::Details::IsOutOfSize(int, const Container\&) 函数

检查索引是否超出容器边界，包括容器大小。

```cpp
template<typename Container> bool System::Collections::Generic::Details::IsOutOfSize(int count, const Container &container)
```

### 模板参数

| 参数 | 描述 |
| --- | --- |
| Container | 容器类型。 |

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| count | int | 索引。 |
| container | const Container\& | 用于检查索引的容器。 |

### 返回值

如果索引超出容器边界则返回 true，否则返回 false。

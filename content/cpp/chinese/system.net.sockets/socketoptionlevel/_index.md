## SocketOptionLevel 枚举

定义 [Socket](../socket/) 类的套接字选项级别。

```cpp
enum class SocketOptionLevel
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Socket | 65535 | 指定的套接字选项适用于所有套接字。 |
| IP | n/a | 指定的套接字选项适用于 IP 套接字。 |
| IPv6 | n/a | 指定的套接字选项适用于 IPv6 套接字。 |
| Tcp | n/a | 指定的套接字选项适用于 TCP 套接字。 |
| Udp | n/a | 指定的套接字选项适用于 UDP 套接字。 |

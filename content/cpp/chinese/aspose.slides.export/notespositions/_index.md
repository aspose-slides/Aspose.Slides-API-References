## NotesPositions enum

表示在导出文档中渲染备注的规则

```cpp
enum class NotesPositions
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| None | 0 | 指定不应显示备注。 |
| BottomFull | 1 | 指定应使用额外页面完整显示备注，视需要而定。 |
| BottomTruncated | 2 | 指定应仅在一页中显示备注。 |

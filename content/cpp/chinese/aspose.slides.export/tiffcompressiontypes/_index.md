## TiffCompressionTypes 枚举

提供用于控制演示文稿在 TIFF 格式中压缩方式的选项。

```cpp
enum class TiffCompressionTypes
```

### Values

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Default | 0 | 指定默认的压缩方案 (LZW)。 |
| None | 1 | 指定不进行压缩。 |
| CCITT3 | 2 | 指定 CCITT3 压缩方案。 |
| CCITT4 | 3 | 指定 CCITT4 压缩方案。 |
| LZW | 4 | 指定 LZW 压缩方案 (默认)。 |
| RLE | 5 | 指定 RLE 压缩方案。 |

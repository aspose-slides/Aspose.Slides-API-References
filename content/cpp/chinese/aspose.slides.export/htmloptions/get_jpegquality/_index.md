## HtmlOptions::get_JpegQuality() method


返回一个确定 PDF 文档内 JPEG 图像质量的值。读取 **uint8_t**。

```cpp
uint8_t Aspose::Slides::Export::HtmlOptions::get_JpegQuality() override
```

## 备注


仅当文档包含 JPEG 图像时有效。

使用此属性在将文档保存为 PDF 格式时获取或设置文档中图像的质量。该值范围为 0 到 100，其中 0 表示最差质量但最大压缩，100 表示最佳质量但最小压缩。

默认值为 **95**。

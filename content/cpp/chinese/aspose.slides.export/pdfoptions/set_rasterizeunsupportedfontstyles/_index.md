## PdfOptions::set_RasterizeUnsupportedFontStyles(bool) 方法


指示当字体不支持粗体样式时，文本是否应以位图形式光栅化并保存为 PDF。此方法可以提升某些字体生成的 PDF 中文本的质量。写入 **bool**.

```cpp
void Aspose::Slides::Export::PdfOptions::set_RasterizeUnsupportedFontStyles(bool value) override
```

## 备注


默认是 **false**.

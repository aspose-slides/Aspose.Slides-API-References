## PdfOptions::set_EmbedTrueTypeFontsForASCII(bool) 方法

确定 [Aspose.Slides](../../../aspose.slides/) 是否会为 ASCII（33..127 代码范围）文本嵌入常用字体。对于大于 127 的字符代码，[Fonts](../../../aspose.slides/fonts/) 总是会被嵌入。常用字体列表包括 PDF 的基本 14 种字体以及用户指定的其他字体。写入 **bool**。

```cpp
void Aspose::Slides::Export::PdfOptions::set_EmbedTrueTypeFontsForASCII(bool value) override
```

## 备注

默认值为 **true**。

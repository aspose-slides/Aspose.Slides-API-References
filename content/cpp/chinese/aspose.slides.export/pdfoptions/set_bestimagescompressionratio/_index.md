## PdfOptions::set_BestImagesCompressionRatio(bool) 方法

指示是否应自动为每个图像选择最有效的压缩（而不是默认的压缩）。如果设置为 **bool**.true，则演示文稿中的每个图像将选择最合适的压缩算法，从而导致生成的 PDF 文档体积更小。

```cpp
void Aspose::Slides::Export::PdfOptions::set_BestImagesCompressionRatio(bool value) override
```

## 备注

最佳图像压缩比例的选择计算量大且会占用额外的内存，并且此选项默认是 **bool**.false。

默认是 **bool**.false。

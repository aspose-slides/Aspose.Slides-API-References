## ISVGOptions::set_DisableLineEndCropping(bool) 方法

SVG 1.1 缺少为标记定义插入的能力。 [Aspose.Slides](../../../aspose.slides/) SVG 写入引擎为该问题提供了变通方法：它会裁剪带箭头的线段末端，从而使线条不会与标记重叠。 此选项可关闭此行为。 写入 **bool**。

```cpp
virtual void Aspose::Slides::Export::ISVGOptions::set_DisableLineEndCropping(bool value)=0
```

## PicturesCompression 枚举

表示图片压缩级别

```cpp
enum class PicturesCompression
```

### 取值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Dpi330 | 0 | 适用于高清（HD）显示器的良好质量 |
| Dpi220 | 1 | 大多数打印机和屏幕上的卓越质量 |
| Dpi150 | 2 | 适用于网页和投影仪的良好质量 |
| Dpi96 | 3 | 最小化文档大小以便共享 |
| Dpi72 | 4 | 默认压缩级别 |
| DocumentResolution | 5 | 使用文档分辨率——图片将不会被压缩，并按原样在文档中使用 |

## IPdfOptions::set_RasterizeUnsupportedFontStyles(bool) 方法

指示当字体不支持粗体样式时，文本是否应以位图方式光栅化并保存为 PDF。此方法可以提升某些字体生成的 PDF 中文本的质量。写入 **bool**。

```cpp
virtual void Aspose::Slides::Export::IPdfOptions::set_RasterizeUnsupportedFontStyles(bool value)=0
```

## 备注

默认值为 **false**。

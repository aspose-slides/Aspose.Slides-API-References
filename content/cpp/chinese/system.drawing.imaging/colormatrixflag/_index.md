## ColorMatrixFlag 枚举


指定将受到 [ImageAttributes](../imageattributes/) 对象的颜色和灰度调整设置影响的图像和颜色类型。

```cpp
enum class ColorMatrixFlag
```

### Values

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Default | 0 | 颜色和灰度值均已进行颜色调整。 |
| SkipGrays | 1 | 灰度值未进行颜色调整。 |
| AltGrays | 2 | 仅灰度值已进行颜色调整。 |

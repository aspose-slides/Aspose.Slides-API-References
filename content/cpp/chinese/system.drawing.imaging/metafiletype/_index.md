## MetafileType 枚举

表示图形元文件的类型。

```cpp
enum class MetafileType
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Invalid | 0 | 无效的元文件。 |
| Wmf | 1 | 标准 WMF。 |
| WmfPlaceable | 2 | 可放置的 [Metafile](../metafile/) 格式。 |
| Emf | 3 | EMF（非 EMF+）。 |
| EmfPlusOnly | 4 | EMF+，不含双重、下层记录。 |
| EmfPlusDual | 5 | EMF+，含双重、下层记录。 |

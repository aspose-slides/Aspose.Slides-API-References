## EmfType 枚举

指定放置在 EMF 文件中的记录类型。

```cpp
enum class EmfType
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| EmfOnly | n/a | [Windows](../../system.windows/) 增强型元文件。包含 GDI 命令。此类型的元文件被称为 EMF 文件。 |
| EmfPlusOnly | n/a | [Windows](../../system.windows/) 增强型元文件 plus。包含 GDI+ 命令。此类型的元文件被称为 EMF+ 文件。 |
| EmfPlusDual | n/a | 双重 [Windows](../../system.windows/) 增强型元文件。包含等效的 GDI 和 GDI+ 命令。此类型的元文件被称为 EMF+ 文件。 |

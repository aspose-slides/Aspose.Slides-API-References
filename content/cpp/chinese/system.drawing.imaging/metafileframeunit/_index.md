## MetafileFrameUnit 枚举

指定用于设置和定位元文件的矩形的测量单位。

```cpp
enum class MetafileFrameUnit
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Pixel | n/a | 指定像素为计量单位。 |
| Point | n/a | 指定打印机的点为计量单位。 |
| Inch | n/a | 指定英寸为计量单位。 |
| Document | n/a | 指定每英寸 1/300 为计量单位。 |
| Millimeter | n/a | 指定毫米为计量单位。 |
| GdiCompatible | n/a | 指定 0.01 毫米为计量单位。提供与 GDI 的兼容性。 |

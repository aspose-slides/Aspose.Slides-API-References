## Details_Win32Exception 类

抛出与 Win32 错误代码对应的异常。不要手动创建此类的实例。请改用 Win32Exception 类。不要将 Win32Exception 类实例包装到 [System::SmartPtr](../../system/smartptr/) 中。

```cpp
class Details_Win32Exception : public System::Details_ExceptionWithErrorCode<Details_SystemException>
```

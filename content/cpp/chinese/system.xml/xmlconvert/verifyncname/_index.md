## XmlConvert::VerifyNCName(const String\&) 方法

验证名称是否符合 W3C 扩展标记语言（XML）推荐的有效 NCName。NCName 是一种不能包含冒号的名称。

```cpp
static String System::Xml::XmlConvert::VerifyNCName(const String &name)
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| name | const [String](../../../system/string/)\& | 要验证的名称。 |

### 返回值

如果是有效的 NCName，则返回该名称。

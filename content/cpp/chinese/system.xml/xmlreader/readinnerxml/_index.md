## XmlReader::ReadInnerXml() 方法

当在派生类中被重写时，读取所有内容，包括标记，作为字符串。

```cpp
virtual String System::Xml::XmlReader::ReadInnerXml()
```

### 返回值

当前节点中的所有 XML 内容，包括标记。如果当前节点没有子节点，则返回空字符串。如果当前节点既不是元素也不是属性，则返回空字符串。

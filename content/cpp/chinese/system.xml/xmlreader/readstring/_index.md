## XmlReader::ReadString() 方法

在派生类中重写时，读取元素或文本节点的内容并作为字符串返回。然而，建议改用 [XmlReader::ReadElementContentAsString](../readelementcontentasstring/) 方法，因为它提供了更直接的处理方式。

```cpp
virtual String System::Xml::XmlReader::ReadString()
```

### 返回值

元素的内容或空字符串。

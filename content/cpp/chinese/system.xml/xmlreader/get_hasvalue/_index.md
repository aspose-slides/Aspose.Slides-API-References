## XmlReader::get_HasValue() 方法


在派生类中被重写时，获取一个值，指示当前节点是否可以具有 [XmlReader::get_Value](../get_value/) 值。

```cpp
virtual bool System::Xml::XmlReader::get_HasValue()
```


### 返回值

**true** 如果读取器当前所在的节点可以具有 **Value**，则为 **true**；否则为 **false**。如果 **false**，则节点的值为 [String::Empty](../../../system/string/empty/)。

## XmlReader::get_IsDefault() 方法

在派生类中覆盖时，获取一个值，指示当前节点是否为从 DTD 或模式中定义的默认值生成的属性。

```cpp
virtual bool System::Xml::XmlReader::get_IsDefault()
```

### 返回值

**true** 如果当前节点是其值由 DTD 或模式中定义的默认值生成的属性；**false** 如果属性值是显式设置的。

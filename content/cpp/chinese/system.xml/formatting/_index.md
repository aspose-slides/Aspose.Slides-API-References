## 格式化枚举

指定 [XmlTextWriter](../xmltextwriter/) 的格式设置选项。

```cpp
enum class Formatting
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| None | 0 | 未应用任何特殊格式。此为默认设置。 |
| Indented | 1 | 根据 [XmlTextWriter::set_Indentation](../xmltextwriter/set_indentation/) 和 [XmlTextWriter::set_IndentChar](../xmltextwriter/set_indentchar/) 设置，使子元素缩进。 |

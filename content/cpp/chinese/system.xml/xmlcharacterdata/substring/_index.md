## XmlCharacterData::Substring(int32_t, int32_t) 方法

检索指定范围内完整字符串的子字符串。

```cpp
virtual String System::Xml::XmlCharacterData::Substring(int32_t offset, int32_t count)
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| offset | **int32_t** | 字符串中开始检索的位置。偏移量为零表示起始点位于数据的开头。 |
| count | **int32_t** | 要检索的字符数。 |

### 返回值

对应指定范围的子字符串。

## XmlNodeOrder 枚举

描述节点相对于第二个节点的文档顺序。

```cpp
enum class XmlNodeOrder
```

### 取值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Before | 0 | 此导航器的当前节点位于提供的导航器的当前节点之前。 |
| After | 1 | 此导航器的当前节点位于提供的导航器的当前节点之后。 |
| Same | 2 | 两个导航器位于同一节点上。 |
| Unknown | 3 | 节点位置无法相对于彼此在文档顺序中确定。如果两个节点位于不同的树中，可能会出现此情况。 |

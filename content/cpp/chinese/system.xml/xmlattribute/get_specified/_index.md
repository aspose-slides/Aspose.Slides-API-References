## XmlAttribute::get_Specified() 方法

返回一个值，指示属性值是否被显式设置。

```cpp
virtual bool System::Xml::XmlAttribute::get_Specified()
```

### 返回值

**true** 如果此属性在原始实例文档中被显式赋值；否则为 **false**。**false** 表示属性值来自 DTD。

## XmlElement::set_IsEmpty(bool) 方法

设置元素的标签格式。

```cpp
void System::Xml::XmlElement::set_IsEmpty(bool value)
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| value | **bool** | 要设置的值。设置此值时，如果设为 **true**，则会移除元素的子节点，并以短标签格式序列化元素。如果设为 **false**，则会更改该值（无论元素是否有内容）；如果元素为空，则以长格式序列化。 |

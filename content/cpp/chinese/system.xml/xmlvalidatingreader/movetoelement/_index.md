## XmlValidatingReader::MoveToElement() 方法

将移动到包含当前属性节点的元素。

```cpp
bool System::Xml::XmlValidatingReader::MoveToElement() override
```

### 返回值

**true** 如果读取器定位在属性上（读取器将移动到拥有该属性的元素）；**false** 如果读取器未定位在属性上（读取器的位置不变）。

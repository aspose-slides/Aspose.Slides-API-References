## XmlReaderSettings::get_MaxCharactersInDocument() 方法


返回一个值，指示 XML 文档中允许的最大字符数。值为零 (0) 表示对 XML 文档的大小没有限制。非零值指定最大大小（以字符为单位）。

```cpp
int64_t System::Xml::XmlReaderSettings::get_MaxCharactersInDocument()
```


### 返回值

XML 文档中允许的最大字符数。默认值为 0。

## IXmlLineInfo::HasLineInfo() 方法


返回一个值，指示该类是否能够返回行信息。

```cpp
virtual bool System::Xml::IXmlLineInfo::HasLineInfo()=0
```


### 返回值

如果可以提供 [IXmlLineInfo::get_LineNumber](../get_linenumber/) 和 [IXmlLineInfo::get_LinePosition](../get_lineposition/)，则为 **true**；否则为 **false**。

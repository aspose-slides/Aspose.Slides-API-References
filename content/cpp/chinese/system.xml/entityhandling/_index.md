## EntityHandling 枚举

指定 [XmlTextReader](../xmltextreader/) 或 [XmlValidatingReader](../xmlvalidatingreader/) 如何处理实体。

```cpp
enum class EntityHandling
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| ExpandEntities | 1 | 展开所有实体并返回展开后的节点。 |
| ExpandCharEntities | 2 | 展开字符实体并将通用实体作为 [XmlNodeType::EntityReference](../xmlnodetype/) 节点返回。 |

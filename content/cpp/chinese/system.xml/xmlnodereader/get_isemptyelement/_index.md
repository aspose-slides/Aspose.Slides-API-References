## XmlNodeReader::get_IsEmptyElement() 方法


返回一个值，指示当前节点是否为空元素（例如，**<MyElement/>**）。

```cpp
bool System::Xml::XmlNodeReader::get_IsEmptyElement() override
```


### 返回值

**true** 如果当前节点是元素（[XmlNodeReader::get_NodeType](../get_nodetype/) 等于 [XmlNodeType::Element](../../xmlnodetype/)）并且以 **/>** 结束；否则，**false**。

## XmlWriterSettings::get_WriteEndDocumentOnClose() 方法


返回一个值，指示在调用 [XmlWriter::Close](../../xmlwriter/close/) 方法时，[XmlWriter](../../xmlwriter/) 是否会向所有未闭合的元素标签添加结束标签。

```cpp
bool System::Xml::XmlWriterSettings::get_WriteEndDocumentOnClose()
```


### 返回值

**true** 如果所有未闭合的元素标签都将被关闭；否则为 **false**。默认值为 **true**。

## XmlWriterSettings::get_IndentChars() 方法

返回用于缩进的字符字符串。当 [XmlWriterSettings::set_Indent](../set_indent/) 值设置为 **true** 时使用此设置。

```cpp
String System::Xml::XmlWriterSettings::get_IndentChars()
```

### 返回值

用于缩进的字符字符串。可以设置为任意字符串值。但是，为了确保有效的 XML，您应仅指定有效的空白字符，例如空格、制表符、回车或换行。默认值是两个空格。

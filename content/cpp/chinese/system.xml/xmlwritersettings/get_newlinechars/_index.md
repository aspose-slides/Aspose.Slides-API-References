## XmlWriterSettings::get_NewLineChars() 方法

返回用于换行的字符字符串。

```cpp
String System::Xml::XmlWriterSettings::get_NewLineChars()
```

### 返回值

用于换行的字符字符串。它可以设置为任意字符串值。然而，为了确保有效的 XML，您应仅指定有效的空白字符，例如空格字符、制表符、回车符或换行符。默认值是 **\r\n**（回车，换行）。

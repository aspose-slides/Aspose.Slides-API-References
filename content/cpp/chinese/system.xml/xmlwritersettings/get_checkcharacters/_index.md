## XmlWriterSettings::get_CheckCharacters() 方法

返回一个值，指示 XML 编写器是否应检查以确保文档中的所有字符符合 W3C [XML 1.0 Recommendation](https://www.w3.org/TR/REC-xml/#charsets) 的 \"2.2 Characters\" 部分。

```cpp
bool System::Xml::XmlWriterSettings::get_CheckCharacters()
```

### 返回值

**true** 表示进行字符检查；否则为 **false**。默认是 **true**。

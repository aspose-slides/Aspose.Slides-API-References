## XmlWriterSettings::get_Indent() 方法


返回一个指示是否对元素进行缩进的值。

```cpp
bool System::Xml::XmlWriterSettings::get_Indent()
```


### 返回值

**true** 表示将各个元素写在新行并进行缩进；否则为 **false**。默认值为 **false**。

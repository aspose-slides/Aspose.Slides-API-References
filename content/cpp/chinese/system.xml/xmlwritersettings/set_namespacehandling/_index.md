## XmlWriterSettings::set_NamespaceHandling(System::Xml::NamespaceHandling) 方法

设置一个值，以指示在写入 XML 内容时 [XmlWriter](../../xmlwriter/) 是否应删除重复的命名空间声明。默认行为是写入器输出写入器的命名空间解析器中存在的所有命名空间声明。

```cpp
void System::Xml::XmlWriterSettings::set_NamespaceHandling(System::Xml::NamespaceHandling value)
```

| value | [System::Xml::NamespaceHandling](../../namespacehandling/) | 要设置的值。 |

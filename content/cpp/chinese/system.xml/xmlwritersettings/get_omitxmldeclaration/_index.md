## XmlWriterSettings::get_OmitXmlDeclaration() 方法

返回一个值，指示是否省略 XML 声明。

```cpp
bool System::Xml::XmlWriterSettings::get_OmitXmlDeclaration()
```

### 返回值

**true** 以省略 XML 声明；否则为 **false**。默认是 **false**，会写入 XML 声明。

## XmlDocumentType::get_IsReadOnly() 方法

返回一个值，指示该节点是否为只读。

```cpp
bool System::Xml::XmlDocumentType::get_IsReadOnly() override
```

### 返回值

**true** 如果节点是只读；否则为 **false**。由于 DocumentType 节点是只读的，此方法始终返回 **true**。

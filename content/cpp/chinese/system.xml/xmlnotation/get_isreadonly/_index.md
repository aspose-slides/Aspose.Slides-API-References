## XmlNotation::get_IsReadOnly() 方法


返回一个指示节点是否为只读的值。

```cpp
bool System::Xml::XmlNotation::get_IsReadOnly() override
```


### 返回值

**true** 如果节点为只读；否则为 **false**。因为 [XmlNotation](../) 节点是只读的，此方法始终返回 **true**。

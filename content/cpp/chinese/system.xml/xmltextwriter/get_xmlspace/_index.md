## XmlTextWriter::get_XmlSpace() 方法


返回一个 XmlSpace，表示当前的 **xml:space** 范围。

```cpp
System::Xml::XmlSpace System::Xml::XmlTextWriter::get_XmlSpace() override
```


### 返回值

返回一个 XmlSpace，表示当前的 **xml:space** 范围。



| 值 | 含义 |
| --- | --- |
| None | 如果不存在 `xml:space` 范围，则为默认值。 |
| Default | 当前范围是 `xml:space="default"`。 |
| Preserve | 当前范围是 `xml:space="preserve"`。 |

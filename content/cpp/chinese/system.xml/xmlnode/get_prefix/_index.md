## XmlNode::get_Prefix() 方法


返回此节点的命名空间前缀。

```cpp
virtual String System::Xml::XmlNode::get_Prefix()
```


### 返回值

此节点的命名空间前缀。示例，**Prefix** 为 **bk**，对应元素 **<bk:book>**。如果没有前缀，此方法返回 [String::Empty](../../../system/string/empty/)。

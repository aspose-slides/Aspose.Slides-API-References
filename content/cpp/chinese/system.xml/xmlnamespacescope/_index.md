## XmlNamespaceScope 枚举

定义命名空间范围。

```cpp
enum class XmlNamespaceScope
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| All | 0 | 在当前节点范围内定义的所有命名空间。其中包括始终隐式声明的 xmlns:xml 命名空间。返回的命名空间顺序未定义。 |
| ExcludeXml | 1 | 在当前节点范围内定义的所有命名空间，排除始终隐式声明的 xmlns:xml 命名空间。返回的命名空间顺序未定义。 |
| Local | 2 | 在当前节点本地定义的所有命名空间。 |

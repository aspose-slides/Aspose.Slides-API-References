## System::MemoryExtensions::IsWhiteSpace(const ReadOnlySpan\<char16_t\>\&) 函数

检查整个 span 是否仅由空白字符组成。

```cpp
bool System::MemoryExtensions::IsWhiteSpace(const ReadOnlySpan<char16_t> &span)
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| span | const [ReadOnlySpan](../../system/readonlyspan/)\<char16_t\>\& | 要检查空白的 span |

### 返回值

如果 span 中的所有字符都是空白字符则返回 true，否则返回 false

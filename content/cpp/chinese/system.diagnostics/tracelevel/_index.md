## TraceLevel 枚举

指定为 [System.Diagnostics.Debug](../debug/)、[System.Diagnostics.Trace](../trace/) 和 System.Diagnostics.TraceSwitch 类输出哪些消息。

```cpp
enum class TraceLevel
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Off | 0 | 不输出任何跟踪和调试消息。 |
| Error | 1 | 输出错误处理消息。 |
| Warning | 2 | 输出警告和错误处理消息。 |
| Info | 3 | 输出信息性消息、警告和错误处理消息。 |
| Verbose | 4 | 输出所有调试和跟踪消息。 |

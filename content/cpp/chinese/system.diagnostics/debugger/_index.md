## Debugger 结构体

[Debugger](./) 接口. 这是一个没有实例服务的静态类型。您不应以任何方式创建它的实例。

```cpp
class Debugger
```

## 方法

| 方法 | 描述 |
| --- | --- |
| static void [Break](./break/)() | 启动调试器中断（如果已附加）。 |
| static **bool** [get_IsAttached](./get_isattached/)() | 检查调试器是否已附加到当前应用程序实例。 |

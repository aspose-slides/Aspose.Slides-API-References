## SmoothingMode 枚举

指定对直线、曲线和已填充区域的边缘应用何种平滑（抗锯齿）类型。

```cpp
enum class SmoothingMode
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Invalid | n/a | 无效值。 |
| Default | n/a | 默认模式，不进行抗锯齿。 |
| HighSpeed | n/a | 不进行抗锯齿，高速。 |
| HighQuality | n/a | 抗锯齿渲染，低速。 |
| None | n/a | 不进行抗锯齿。 |
| AntiAlias | n/a | 抗锯齿渲染。 |

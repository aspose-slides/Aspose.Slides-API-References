## HatchBrush 类

表示具有填充样式、前景色和背景色的矩形画笔。此类的对象只能通过 [System::MakeObject()](../../system/makeobject/) 函数分配。禁止在栈上或使用 new 运算符创建此类型的实例，否则会导致运行时错误和/或断言故障。始终将此类包装在 [System::SmartPtr](../../system/smartptr/) 指针中，并使用该指针作为参数传递给函数。

```cpp
class HatchBrush : public System::Drawing::Brush
```

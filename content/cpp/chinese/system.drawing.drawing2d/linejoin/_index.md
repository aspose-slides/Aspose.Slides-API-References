## LineJoin 枚举

指定后续直线或曲线的连接方式。

```cpp
enum class LineJoin
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Miter | 0 | 产生锐角或被裁剪的角的斜接连接，取决于斜接长度是否超过斜接限制。 |
| Bevel | 1 | 产生对角角的斜角连接。 |
| Round | 2 | 在直线之间产生平滑圆弧的圆形连接。 |
| MiterClipped | 3 | 产生锐角或斜角的斜接连接，取决于斜接长度是否超过斜接限制。 |

## PixelOffsetMode 枚举


指定在渲染时像素的偏移方式。

```cpp
enum class PixelOffsetMode
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Invalid | n/a | 无效值。 |
| Default | n/a | 默认模式。 |
| HighSpeed | n/a | 低质量（高性能）模式。 |
| HighQuality | n/a | 高质量（性能较低）模式。 |
| None | n/a | 无像素偏移。 |
| Half | n/a | 像素在水平和垂直方向上均偏移 -0.5 单位，以实现高性能抗锯齿。 |

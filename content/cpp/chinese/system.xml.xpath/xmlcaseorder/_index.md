## XmlCaseOrder 枚举

指定大写字母和小写字母的排序顺序。

```cpp
enum class XmlCaseOrder
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| None | 0 | 忽略大小写。 |
| UpperFirst | 1 | 大写字母排在小写字母之前。 |
| LowerFirst | 2 | 小写字母排在大写字母之前。 |

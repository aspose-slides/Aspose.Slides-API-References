## XPathNavigator::MoveToId(String) 方法

当在派生类中被重写时，移动到具有 **ID** 类型属性且其值匹配指定的 [String](../../../system/string/) 的节点。

```cpp
virtual bool System::Xml::XPath::XPathNavigator::MoveToId(String id)=0
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| id | [String](../../../system/string/) | 表示要移动的节点的 **ID** 值的 [String](../../../system/string/)。 |

### 返回值

**true** 表示 [XPathNavigator](../) 成功移动；否则为 **false**。如果 **false**，导航器的位置保持不变。

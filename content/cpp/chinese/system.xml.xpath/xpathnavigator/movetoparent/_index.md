## XPathNavigator::MoveToParent() method

当在派生类中重写时，将 [XPathNavigator](../) 移动到当前节点的父节点。

```cpp
virtual bool System::Xml::XPath::XPathNavigator::MoveToParent()=0
```

### 返回值

**true** 如果 [XPathNavigator](../) 成功移动到当前节点的父节点；否则为 **false**。如果为 **false**，则 [XPathNavigator](../) 的位置保持不变。

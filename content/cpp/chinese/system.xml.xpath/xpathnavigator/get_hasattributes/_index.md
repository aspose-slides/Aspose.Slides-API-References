## XPathNavigator::get_HasAttributes() 方法

返回一个值，指示当前节点是否具有任何属性。

```cpp
virtual bool System::Xml::XPath::XPathNavigator::get_HasAttributes()
```

### 返回值

**true** 如果当前节点具有属性；返回 **false** 如果当前节点没有属性，或者 [XPathNavigator](../) 未定位在元素节点上。

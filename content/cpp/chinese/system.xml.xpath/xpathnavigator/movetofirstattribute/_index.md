## XPathNavigator::MoveToFirstAttribute() 方法

当在派生类中被重写时，将 [XPathNavigator](../) 移动到当前节点的第一个属性。

```cpp
virtual bool System::Xml::XPath::XPathNavigator::MoveToFirstAttribute()=0
```

### 返回值

**true** 如果 [XPathNavigator](../) 成功移动到当前节点的第一个属性；否则为 **false**。如果 **false**，[XPathNavigator](../) 的位置保持不变。

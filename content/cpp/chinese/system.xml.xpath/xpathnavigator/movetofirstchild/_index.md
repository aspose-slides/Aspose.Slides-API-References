## XPathNavigator::MoveToFirstChild() 方法

当在派生类中被重写时，将 [XPathNavigator](../) 移动到当前节点的第一个子节点。

```cpp
virtual bool System::Xml::XPath::XPathNavigator::MoveToFirstChild()=0
```

### 返回值

**true** 如果 [XPathNavigator](../) 成功移动到当前节点的第一个子节点；否则为 **false**。如果为 **false**，[XPathNavigator](../) 的位置保持不变。

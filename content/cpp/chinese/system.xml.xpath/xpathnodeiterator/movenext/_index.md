## XPathNodeIterator::MoveNext() 方法

当在派生类中覆盖时，将 [XPathNavigator](../../xpathnavigator/) 对象（由 [XPathNodeIterator::get_Current](../get_current/) 方法返回）移动到所选节点集中的下一个节点。

```cpp
virtual bool System::Xml::XPath::XPathNodeIterator::MoveNext()=0
```

### 返回值

**true** 如果 [XPathNavigator](../../xpathnavigator/) 对象已移动到下一个节点；**false** 如果没有更多的选定节点。

## 类

| 类 | 描述 |
| --- | --- |
| [Soap](./soap/) | 包含 SOAP 的字符串常量。 |
| [Soap12](./soap12/) | 包含 SOAP 版本 1.2 的字符串常量。 |
| [WebServiceAttribute](./webserviceattribute/) | 为 XML [Web](../system.web/) 服务添加额外信息。此类的对象只能使用 [System::MakeObject()](../system/makeobject/) 函数分配。切勿在栈上或使用 operator new 创建此类型的实例，因为这会导致运行时错误和/或断言失败。始终将此类包装为 [System::SmartPtr](../system/smartptr/) 指针，并使用该指针将其作为参数传递给函数。 |
| [WebServiceBindingAttribute](./webservicebindingattribute/) | 用于声明绑定，以定义 XML [Web](../system.web/) 服务的一个或多个方法。此类的对象只能使用 [System::MakeObject()](../system/makeobject/) 函数分配。切勿在栈上或使用 operator new 创建此类型的实例，因为这会导致运行时错误和/或断言失败。始终将此类包装为 [System::SmartPtr](../system/smartptr/) 指针，并使用该指针将其作为参数传递给函数。 |

## Timer::set_Enabled(bool) 方法

启动或停止计时器。如果计时器已经在运行，启动计时器不会重新计时。

```cpp
void System::Timers::Timer::set_Enabled(bool value)
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| value | **bool** | true 表示启动计时器，false 表示停止。 |

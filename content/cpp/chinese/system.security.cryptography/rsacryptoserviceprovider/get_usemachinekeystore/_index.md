## RSACryptoServiceProvider::get_UseMachineKeyStore() 方法

检查密钥是否持久化在机器存储区而不是用户存储区。

```cpp
static bool System::Security::Cryptography::RSACryptoServiceProvider::get_UseMachineKeyStore()
```

### 返回值

如果密钥应持久化在机器存储区则返回 true；如果应持久化在用户存储区则返回 false。

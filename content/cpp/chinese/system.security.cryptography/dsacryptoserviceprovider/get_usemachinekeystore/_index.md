## DSACryptoServiceProvider::get_UseMachineKeyStore() 方法

检查密钥是否在机器存储区而不是用户存储区中持久化。

```cpp
static bool System::Security::Cryptography::DSACryptoServiceProvider::get_UseMachineKeyStore()
```

### 返回值

如果密钥应持久化在机器存储区则返回 True，如果应持久化在用户存储区则返回 false。

## XsltSettings::get_EnableDocumentFunction() 方法

返回一个值，指示是否启用对 XSLT **document()** 函数的支持。

```cpp
bool System::Xml::Xsl::XsltSettings::get_EnableDocumentFunction()
```

### 返回值

**true** 表示支持 XSLT **document()** 函数；否则为 **false**。默认值为 **false**。

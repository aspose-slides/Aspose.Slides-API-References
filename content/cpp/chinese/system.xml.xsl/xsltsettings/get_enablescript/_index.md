## XsltSettings::get_EnableScript() 方法

返回一个值，指示是否启用对嵌入式脚本块的支持。

```cpp
bool System::Xml::Xsl::XsltSettings::get_EnableScript()
```

### 返回值

**true** 表示在 XSLT 样式表中支持脚本块；否则为 **false**。默认值为 **false**。

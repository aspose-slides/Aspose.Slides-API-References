## IXsltContextVariable::get_IsParam() 方法

返回一个值，指示该变量是否为可扩展样式表语言转换 (XSLT) 参数。这可以是样式表或模板的参数。

```cpp
virtual bool System::Xml::Xsl::IXsltContextVariable::get_IsParam()=0
```

### 返回值

**true** 如果该变量是 XSLT 参数；否则为 **false**。

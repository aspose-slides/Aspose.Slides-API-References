## List::rend() 方法

获取指向集合起始位置之前的一个不存在的元素的逆向迭代器。

```cpp
reverse_iterator System::Collections::Generic::List<T>::rend() noexcept
```

### 返回值

一个指向列表中第一个元素之前的理论元素的逆向迭代器（逆向中的最后一个）。

## List::rend() const 方法

获取指向常量限定集合起始位置之前的一个不存在的元素的逆向迭代器。

```cpp
const_reverse_iterator System::Collections::Generic::List<T>::rend() const noexcept
```

### 返回值

一个指向常量限定列表中第一个元素之前的理论元素的逆向迭代器（逆向中的最后一个）。

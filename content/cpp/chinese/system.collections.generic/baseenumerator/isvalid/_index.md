## BaseEnumerator::IsValid() const 方法

检查是否已调用 [MoveNext()](../movenext/) 且未到达结束。

```cpp
bool System::Collections::Generic::BaseEnumerator<Container, Element>::IsValid() const
```

### 返回值

如果 [MoveNext()](../movenext/) 根本未被调用或已到达容器末尾，则为 false；否则为 true。

## SortedDictionary::crend() const 方法

获取集合起始位置之前的一个不存在的 const 限定元素的反向迭代器。

```cpp
const_reverse_iterator System::Collections::Generic::SortedDictionary<TKey, TValue>::crend() const noexcept
```

### 返回值

一个指向理论上位于映射中第一个元素之前的 const 限定元素的反向迭代器（在反向中是最后一个）。

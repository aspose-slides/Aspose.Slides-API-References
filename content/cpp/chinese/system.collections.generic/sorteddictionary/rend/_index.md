## SortedDictionary::rend() 方法

获取集合开始之前的一个不存在的元素的反向迭代器。

```cpp
reverse_iterator System::Collections::Generic::SortedDictionary<TKey, TValue>::rend() noexcept
```


### 返回值

指向映射中第一个元素之前的理论元素的反向迭代器（逆向中的最后一个）。

## SortedDictionary::rend() const 方法

获取 const 限定集合开始之前的一个不存在的元素的反向迭代器。

```cpp
const_reverse_iterator System::Collections::Generic::SortedDictionary<TKey, TValue>::rend() const noexcept
```


### 返回值

指向 const 限定映射中第一个元素之前的理论元素的反向迭代器（逆向中的最后一个）。

## IEnumerable::begin() 方法

获取指向集合中第一个元素（如果有）的迭代器。此迭代器不能用于修改所引用的对象，因为 [GetEnumerator()](../getenumerator/) 返回 T 的复制对象。

```cpp
iterator System::Collections::Generic::IEnumerable<T>::begin()
```

### 返回值

指向集合中第一个元素（如果有）的迭代器

## IEnumerable::begin() const 方法

获取指向集合的 const 限定实例中第一个元素（如果有）的迭代器。

```cpp
const_iterator System::Collections::Generic::IEnumerable<T>::begin() const
```

### 返回值

指向集合的 const 限定实例中第一个元素（如果有）的迭代器

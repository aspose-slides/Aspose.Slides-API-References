## IEnumerable::end() 方法

获取指向集合中最后一个元素（如果有）之后的迭代器。此迭代器不能用于修改引用的对象，因为 [GetEnumerator()](../getenumerator/) 返回 T 的拷贝对象。

```cpp
iterator System::Collections::Generic::IEnumerable<T>::end()
```

### 返回值

指向集合中最后一个元素（如果有）之后的迭代器

## IEnumerable::end() const 方法

获取指向集合的 const 限定实例中最后一个元素（如果有）之后的迭代器。

```cpp
const_iterator System::Collections::Generic::IEnumerable<T>::end() const
```

### 返回值

指向集合的 const 限定实例中最后一个元素（如果有）之后的迭代器

## BaseDictionary::cbegin() const 方法


返回一个迭代器，指向容器的第一个元素。采用 STL 风格实现。如果容器为空，返回的迭代器将等于 [end()](../../ienumerable/end/)。

```cpp
stl_const_iterator System::Collections::Generic::BaseDictionary<Map>::cbegin() const noexcept
```


### 返回值

一个迭代器，指向集合中第一个 const 限定的元素。

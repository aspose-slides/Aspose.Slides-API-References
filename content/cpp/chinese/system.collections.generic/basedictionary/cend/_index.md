## BaseDictionary::cend() const 方法

返回一个迭代器，指向容器中最后一个元素之后的元素。已实现为 STL 风格。该元素充当占位符；尝试访问它会导致未定义行为。

```cpp
stl_const_iterator System::Collections::Generic::BaseDictionary<Map>::cend() const noexcept
```

### 返回值

返回指向理论上 const 限定的元素的迭代器，该元素位于集合的结束元素之后。

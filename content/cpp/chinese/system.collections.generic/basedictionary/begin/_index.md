## BaseDictionary::begin() const 方法

返回指向容器中键值元素的 KVPair-wrapper 的迭代器。采用 C# 风格实现——迭代器应返回具有 get_Key() 和 get_Value() 接口的 KVPair-object。如果容器为空，返回的迭代器将等于 [end()](../../ienumerable/end/)。

```cpp
const_iterator System::Collections::Generic::BaseDictionary<Map>::begin() const noexcept
```

### 返回值

指向集合中第一个元素的迭代器。

## LinkedList::Find(const T\&) const 方法

```cpp
SharedPtr<LinkedListNode<T>> System::Collections::Generic::LinkedList<T>::Find(const T &element) const
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| element | const T\& | 要查找的元素。 |

### 返回值

如果找到 **element**，则返回节点，否则返回 nullptr。

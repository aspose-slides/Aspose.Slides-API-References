## ReverseEnumerator::IsValid() const 方法


检查是否已经调用了 [MoveNext()](../movenext/) 且未达到结束。

```cpp
bool System::Collections::Generic::ReverseEnumerator<Container, Element>::IsValid() const
```


### 返回值

如果 [MoveNext()](../movenext/) 完全未被调用或已到达容器末尾，则返回 false；否则返回 true。

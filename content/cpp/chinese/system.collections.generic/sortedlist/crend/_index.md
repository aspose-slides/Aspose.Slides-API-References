## SortedList::crend() const method

获取一个指向集合起始位置之前的、不存在的 const 限定元素的反向迭代器。

```cpp
const_reverse_iterator System::Collections::Generic::SortedList<TKey, TValue>::crend() const noexcept
```

### 返回值

返回一个指向理论上位于映射中第一个元素之前（即反向的最后一个）的 const 限定元素的反向迭代器。

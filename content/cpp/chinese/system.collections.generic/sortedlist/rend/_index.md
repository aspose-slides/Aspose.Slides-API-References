## SortedList::rend() 方法

获取指向集合开始之前的不存在元素的反向迭代器。

```cpp
reverse_iterator System::Collections::Generic::SortedList<TKey, TValue>::rend() noexcept
```

### 返回值

一个指向映射中第一个元素之前的理论元素的反向迭代器（在反向中为最后一个）。

## SortedList::rend() const 方法

获取指向 const 限定集合开始之前的不存在元素的反向迭代器。

```cpp
const_reverse_iterator System::Collections::Generic::SortedList<TKey, TValue>::rend() const noexcept
```

### 返回值

一个指向 const 限定映射中第一个元素之前的理论元素的反向迭代器（在反向中为最后一个）。

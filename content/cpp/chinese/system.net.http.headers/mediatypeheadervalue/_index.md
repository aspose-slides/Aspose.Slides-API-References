## MediaTypeHeaderValue 类

代表 `Content-Type` 头部值中的 MIME 类型。此类的对象只能通过 [System::MakeObject()](../../system/makeobject/) 函数分配。不要在栈上或使用 `new` 运算符创建此类型的实例，因为这会导致运行时错误和/或断言故障。始终将此类包装在 [System::SmartPtr](../../system/smartptr/) 指针中，并使用该指针将其作为参数传递给函数。

```cpp
class MediaTypeHeaderValue : public virtual System::ICloneable
```

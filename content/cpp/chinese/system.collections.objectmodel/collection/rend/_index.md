## Collection::rend() 方法

获取指向集合起始位置之前的不存在元素的逆向迭代器。

```cpp
reverse_iterator System::Collections::ObjectModel::Collection<T>::rend() noexcept
```

### 返回值

返回指向列表中首个元素之前的理论元素的逆向迭代器（即逆向的最后一个）。

## Collection::rend() const 方法

获取 const 限定的集合起始位置之前的不存在元素的逆向迭代器。

```cpp
const_reverse_iterator System::Collections::ObjectModel::Collection<T>::rend() const noexcept
```

### 返回值

返回指向 const 限定列表中首个元素之前的理论元素的逆向迭代器（即逆向的最后一个）。

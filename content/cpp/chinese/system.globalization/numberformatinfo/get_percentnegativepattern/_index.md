## NumberFormatInfo::get_PercentNegativePattern() const 方法

获取百分比负数模式。

```cpp
int System::Globalization::NumberFormatInfo::get_PercentNegativePattern() const
```

### 返回值

数组中模式的索引 {\"-n %\", \"-n%\", \"-%n\", \"%-n\", \"%n-\", \"n-%\", \"n%-\", \"-% n\", \"n %-\", \"% n-\", \"% -n\", \"n- %\"}，其中 \"n\" 表示数字，\"%\" 表示百分号，\"-\" 表示负号。

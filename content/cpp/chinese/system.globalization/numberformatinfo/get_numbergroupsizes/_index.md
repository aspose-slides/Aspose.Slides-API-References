## NumberFormatInfo::get_NumberGroupSizes() const 方法

获取每组的数字位数。

```cpp
ArrayPtr<int> System::Globalization::NumberFormatInfo::get_NumberGroupSizes() const
```

### 返回值

[Array](../../../system/array/) 表示每组的数字位数，从左到右；每个元素必须是1到9，最后一个可以是0，表示 \"全部合并\"；最后一个元素重复。

## NumberFormatInfo::get_CurrencyNegativePattern() const 方法

获取货币负数模式。

```cpp
int System::Globalization::NumberFormatInfo::get_CurrencyNegativePattern() const
```

### 返回值

数组中模式的索引，数组为 {"($n)", "-$n", "$-n", "$n-", "(n$)", "-n$", "n-$", "n$-", "-n $", "-$ n", "n $-", "$ n-", "$ -n", "n- $", "($ n)", "(n $)"}, 其中 "n" 是数字，"-" 是负号，"$" 是货币符号。

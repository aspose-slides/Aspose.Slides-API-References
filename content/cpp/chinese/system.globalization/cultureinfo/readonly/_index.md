## CultureInfo::ReadOnly(const CultureInfoPtr\&) 方法

获取文化的只读版本。

```cpp
static CultureInfoPtr System::Globalization::CultureInfo::ReadOnly(const CultureInfoPtr &culture_info)
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| culture_info | const [CultureInfoPtr](../../cultureinfoptr/)\& | 要包装为只读的文化。 |

### 返回值

如果 **culture_info** 为只读，则返回相同对象；否则创建只读副本。

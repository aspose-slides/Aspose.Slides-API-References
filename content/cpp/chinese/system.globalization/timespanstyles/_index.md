## TimeSpanStyles 枚举

指定用于自定义字符串解析的格式选项，这些方法将时间间隔的字符串表示转换为 Timespan 对象。

```cpp
enum class TimeSpanStyles : int32_t
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| None | 0 | 仅当输入字符串包含负号时，才将其表示的时间间隔解释为负时间间隔。 |
| AssumeNegative | 1 | 始终将输入字符串表示的时间间隔解释为负时间间隔。 |

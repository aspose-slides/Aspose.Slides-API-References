## CultureTypes 枚举

Culture 类别位掩码条目。

```cpp
enum class CultureTypes : int32_t
```

### 值

| Name | Value | Description |
| --- | --- | --- |
| NeutralCultures | 1 | 特定于语言但不针对国家或地区的 Culture。 |
| SpecificCultures | 2 | 特定于国家/地区。 |
| InstalledWin32Cultures | 4 | 已在操作系统中安装的 Cultures。 |
| AllCultures | n/a | 所有可用的 cultures。 |
| UserCustomCulture | 8 | 用户自定义 cultures。 |
| ReplacementCultures | 16 | 用户自定义的现有 cultures 的替代项。 |
| WindowsOnlyCultures | 32 | 已废弃。 |
| FrameworkCultures | 64 | 已废弃。 |

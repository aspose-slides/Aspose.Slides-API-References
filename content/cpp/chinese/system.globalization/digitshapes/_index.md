## DigitShapes 枚举

指定数字的显示方式。

```cpp
enum class DigitShapes : int32_t
```

### 取值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Context | 0 | 数字形状取决于前面的文本。 |
| None | 1 | 数字形状未改变。 |
| NativeNational | 2 | ASCII 数字 0..9 被本地数字替换。 |

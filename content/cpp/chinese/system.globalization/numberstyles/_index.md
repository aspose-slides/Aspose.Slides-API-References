## NumberStyles 枚举

解析时允许的数字样式。

```cpp
enum class NumberStyles : int32_t
```

### 值

| Name | Value | Description |
| --- | --- | --- |
| None | 0 | 没有非数字符号。 |
| AllowLeadingWhite | 1 | 允许前导空白字符。 |
| AllowTrailingWhite | 2 | 允许尾随空白字符。 |
| AllowLeadingSign | 4 | 允许前导符号。 |
| Integer | n/a | 默认整数格式化。 |
| AllowTrailingSign | 8 | 允许尾随符号。 |
| AllowParentheses | 16 | 允许对负值使用括号。 |
| AllowDecimalPoint | 32 | 允许小数点。 |
| AllowThousands | 64 | 允许千位分隔符。 |
| Number | n/a | 默认复合数字格式。 |
| AllowExponent | 128 | 允许指数符号。 |
| Float | n/a | 默认浮点数格式。 |
| AllowCurrencySymbol | 256 | 允许货币符号。 |
| Currency | n/a | 默认货币格式。 |
| Any | n/a | 允许任何格式说明符。 |
| AllowHexSpecifier | 512 | 允许十六进制数。 |
| HexNumber | n/a | 默认十六进制数字格式。 |

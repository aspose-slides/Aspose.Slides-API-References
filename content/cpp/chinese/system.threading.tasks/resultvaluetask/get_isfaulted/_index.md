## ResultValueTask::get_IsFaulted() const 方法

获取一个指示任务是否因未处理的异常而完成的值。

```cpp
bool System::Threading::Tasks::ResultValueTask<T>::get_IsFaulted() const
```

### 返回值

bool 如果任务出现故障则为 True；否则为 false。
## 备注

如果任务包含直接结果值，则返回 false。

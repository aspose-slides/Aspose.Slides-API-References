## ResultValueTask::get_IsCanceled() const 方法

获取一个值，指示任务是否因被取消而完成。

```cpp
bool System::Threading::Tasks::ResultValueTask<T>::get_IsCanceled() const
```

### 返回值

bool 为 True 如果任务被取消；否则为 false.
## 备注

如果任务包含直接结果值，则返回 false。

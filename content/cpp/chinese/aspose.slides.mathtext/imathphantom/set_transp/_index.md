## IMathPhantom::set_Transp(bool) 方法

设置一个值，以指示该幻影在基于类的间距规则中是否透明。

```cpp
virtual void Aspose::Slides::MathText::IMathPhantom::set_Transp(bool value)=0
```

## 备注

当 **true** 时，幻影内部的运算符和符号仍会影响幻影周围的数学间距（如同可见）。当 **false** 时，基于类的间距将被忽略。对应于 OMML 属性 **m:transp**。

## IMathPhantom 类

表示一个幻影数学对象 (<m:phant>)，它影响其子元素的布局但不一定显示它。幻影可以隐藏其基础表达式，同时保留宽度、高度或深度，以对齐公式或预留空间。可见性和几何行为受诸如 Show、ZeroWid、ZeroAsc、ZeroDesc 和 Transp 等属性控制。

```cpp
class IMathPhantom : public virtual Aspose::Slides::MathText::IMathElement
```

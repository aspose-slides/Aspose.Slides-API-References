## MathJustification enum

指定数学段落的对齐方式（同一段落中相邻的数学文本实例的系列）

```cpp
enum class MathJustification
```

### 取值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| LeftJustified | 1 | 将数学文本左对齐 |
| RightJustified | 2 | 将数学文本右对齐 |
| Centered | 3 | 将数学文本居中 |
| CenteredAsGroup | 4 | 在页面上将整组数学块整体居中，视为一个单元。这意味着数学文本的实例可以相互对齐，但整个数学文本组整体居中。 |

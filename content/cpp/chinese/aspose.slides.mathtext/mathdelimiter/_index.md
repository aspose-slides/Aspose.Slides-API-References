## MathDelimiter 类

指定分隔符对象，由起始和结束字符组成（例如括号、花括号、方括号和竖线），以及内部的一个或多个数学元素，这些元素由指定字符分隔。示例: (\\uD835\\uDC652); [\\uD835\\uDC652|\\uD835\\uDC662]

```cpp
class MathDelimiter : public Aspose::Slides::MathText::MathElementBase,
                      public Aspose::Slides::MathText::IMathDelimiter,
                      public Aspose::Slides::MathText::IHasControlCharacterProperties
```

## MathRowSpacingRule enum

矩阵或数组中列之间的垂直间距类型

```cpp
enum class MathRowSpacingRule
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| SingleLineGap | 0 | 单行间距 |
| OneAndAHalfLineGap | 1 | 一行半间距 |
| TwoLineGap | 2 | 两行间距 |
| Exactly | 3 | 精确（依赖 RowSpacing 属性的值，以点为单位） |
| Multiple | 4 | 多重（依赖 RowSpacing 属性的值，以行为单位） |

## MathPhantom::set_ZeroAsc(bool) 方法

设置一个值，指示是否应将基元素的上升（基线以上的高度）视为零。

```cpp
void Aspose::Slides::MathText::MathPhantom::set_ZeroAsc(bool value) override
```

## 备注

当 **true** 时，phantom 不会提升周围数学行的基线。对应 OMML 属性 **m:zeroAsc**。

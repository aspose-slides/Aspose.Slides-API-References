## MathPhantom::set_Transp(bool) 方法


设置一个值，指示针对基于类的间距规则，幻影是否透明。

```cpp
void Aspose::Slides::MathText::MathPhantom::set_Transp(bool value) override
```

## 备注


当 **true** 时，幻影内部的运算符和符号仍会影响幻影周围的数学间距（如同可见）。当 **false** 时，基于类的间距被忽略。对应 OMML 属性 **m:transp**.

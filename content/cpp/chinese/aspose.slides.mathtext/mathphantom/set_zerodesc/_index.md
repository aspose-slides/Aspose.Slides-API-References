## MathPhantom::set_ZeroDesc(bool) 方法

设置一个值，用于指示是否应将基元元素的下降（基线以下的深度）视为零。

```cpp
void Aspose::Slides::MathText::MathPhantom::set_ZeroDesc(bool value) override
```

## 备注

当 **true** 时，幻影不会降低周围数学行的基线。对应于 OMML 属性 **m:zeroDesc**。

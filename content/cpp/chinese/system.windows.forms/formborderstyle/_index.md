## FormBorderStyle 枚举

定义边框的绘制方式。

```cpp
enum class FormBorderStyle
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| None | 0 | 完全没有边框。 |
| FixedSingle | 1 | [Single](../../system/single/) 边框。 |
| Fixed3D | 2 | 类似3D的边框。 |
| FixedDialog | 3 | 对话框样式的边框。 |
| Sizable | 4 | 可调整大小的边框。 |
| FixedToolWindow | 5 | 不可调整大小的工具窗口边框。 |
| SizableToolWindow | 6 | 可调整大小的工具窗口边框。 |

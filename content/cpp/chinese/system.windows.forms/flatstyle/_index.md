## FlatStyle 枚举

定义控件是以平面方式绘制还是三维方式绘制。

```cpp
enum class FlatStyle
```

### 取值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Flat | 0 | [Control](../control/) 以平面方式绘制。 |
| Popup | 1 | [Control](../control/) 以平面方式绘制，除非当鼠标移到其上时以 3D 方式绘制。 |
| Standard | 2 | [Control](../control/) 以 3D 方式绘制。 |
| System | 3 | 样式由操作系统控制。 |

## AutoSizeMode 枚举

定义自动大小调整的工作方式。

```cpp
enum class AutoSizeMode
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| GrowAndShrink | 0 | [Control](../control/) 会根据需要增长和收缩。它不能被手动调整大小。 |
| GrowOnly | 1 | [Control](../control/) 会根据需要增长，但不能低于最小尺寸收缩。它可以手动调整大小。 |

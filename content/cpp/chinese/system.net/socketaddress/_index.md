## SocketAddress 类

Used to store serialized information about the [EndPoint](../endpoint/) 类 instances. Objects of this 类 should only be allocated using [System::MakeObject()](../../system/makeobject/) 函数. Never create instance of this type on stack or using operator new, as it will result in runtime errors and/or assertion faults. Always wrap this 类 into [System::SmartPtr](../../system/smartptr/) pointer and use this pointer to pass it to functions as argument.

```cpp
class SocketAddress : public System::Object
```

## IPHostEntry 类

Represents information about an internet host address. Objects of this class should only be allocated using [System::MakeObject()](../../system/makeobject/) 函数. Never create instance of this type on stack or using operator new, as it will result in runtime errors and/or assertion faults. Always wrap this class into [System::SmartPtr](../../system/smartptr/) pointer and use this pointer to pass it to functions as argument.

```cpp
class IPHostEntry : public System::Object
```

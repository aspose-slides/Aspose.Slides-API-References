## TransitionMorphType 枚举


表示一种变形过渡类型。

```cpp
enum class TransitionMorphType
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| ByObject | 0 | Morph 过渡将在将形状视为不可分割的对象时执行。 |
| ByWord | 1 | Morph 过渡将在可能的情况下通过按单词传输文本来执行。 |
| ByChar | 2 | Morph 过渡将在可能的情况下通过按字符传输文本来执行。 |

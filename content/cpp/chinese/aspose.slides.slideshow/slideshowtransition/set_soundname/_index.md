## SlideShowTransition::set_SoundName(System::String) 方法

指定过渡效果声音的可读名称。必须为 [ISlideShowTransition::set_Sound](../../../aspose.slides/islideshowtransition/set_sound/) 赋值以获取或设置声音名称。写入 [System::String](../../../system/string/)。

```cpp
void Aspose::Slides::SlideShow::SlideShowTransition::set_SoundName(System::String value) override
```

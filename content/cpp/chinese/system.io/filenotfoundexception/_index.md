## FileNotFoundException 类型定义


当尝试访问磁盘上不存在的文件时抛出的异常。永远不要将 FileNotFoundException 类 实例包装到 [System::SmartPtr](../../system/smartptr/) 中。

```cpp
using System::IO::FileNotFoundException = typedef System::ExceptionWrapper<Details_FileNotFoundException>
```

## STDIOStreamPositionPreference 枚举

确定在流中哪一个位置更适合作为通用的读取和写入位置，当 std::basic_iostream 及其派生类在创建包装器时具有不同的读取和写入位置时。

```cpp
enum class STDIOStreamPositionPreference
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Zero | 0 | Zero 位置将作为读取和写入位置。 |
| ReadPosition | 1 | gptr 位置将作为读取和写入位置。 |
| WritePosition | 2 | pptr 位置将作为读取和写入位置。 |

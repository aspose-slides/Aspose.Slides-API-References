## Path::HasExtension(const String\&) 方法

确定指定的路径是否引用了带扩展名的文件。

```cpp
static bool System::IO::Path::HasExtension(const String &path)
```

### 参数

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| path | const [String](../../../system/string/)\& | 要检查的路径 |

### 返回值

如果 **path** 引用的是带扩展名的文件，则返回 True；否则返回 false

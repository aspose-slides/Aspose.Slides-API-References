## SearchOption 枚举

指定仅在当前目录中执行搜索，或在当前目录及其所有子目录中执行搜索。

```cpp
enum class SearchOption
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| AllDirectories | 0 | 在当前目录及其所有子目录中搜索。 |
| TopDirectoryOnly | 1 | 仅在当前目录中搜索。 |

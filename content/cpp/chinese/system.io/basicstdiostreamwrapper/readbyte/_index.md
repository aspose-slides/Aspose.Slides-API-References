## BasicSTDIOStreamWrapper::ReadByte() 方法


如果封装模式为二进制，则从上一次解码的字符存储中读取单个字节；否则从流中读取单个字符并将其转换为 **uint8_t** 类型。

```cpp
virtual int System::IO::BasicSTDIOStreamWrapper<T, typename>::ReadByte() override
```


### 返回值

相当于读取结果的 32 位整数值

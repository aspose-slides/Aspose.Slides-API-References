## FileAttributes 枚举

表示目录或文件的属性。

```cpp
enum class FileAttributes
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Archive | 32 | 该文件系统实体是删除或备份的候选项。 |
| Compressed | 2048 | 该文件系统实体已压缩。 |
| Device | 64 | 未使用。 |
| Directory | 16 | 该文件系统实体是目录。 |
| Encrypted | 16384 | 该文件系统实体已加密。 |
| Hidden | 2 | 该文件系统实体已隐藏。 |
| IntegrityStream | 32768 | 该文件系统实体包含数据完整性支持。 |
| Normal | 128 | 该文件系统实体是没有特殊属性的文件。 |
| NotContentIndexed | 8192 | 该文件系统实体不会被内容索引服务索引。 |
| NoScrubData | 131072 | 该文件系统实体未包含在数据完整性扫描中。 |
| Offline | 4096 | 该文件系统实体离线，其数据无法立即获取。 |
| ReadOnly | 1 | 该文件系统实体是只读的。 |
| ReparsePoint | 1024 | 该文件系统实体是包含重新解析点的文件。 |
| SparseFile | 512 | 该文件系统实体是稀疏文件。 |
| System | 4 | 该文件系统实体是系统文件。 |
| Temporary | 256 | 该文件系统实体是临时文件。 |
| Invalid | 0 | 无效的值。 |

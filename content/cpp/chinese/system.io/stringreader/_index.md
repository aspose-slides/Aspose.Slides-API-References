## StringReader 类


表示从字符串读取字符的读取器。该类的对象只能使用 [System::MakeObject()](../../system/makeobject/) 函数分配。永不在栈上或使用 new 运算符创建此类型的实例，因为这会导致运行时错误和/或断言故障。始终将此类包装为 [System::SmartPtr](../../system/smartptr/) 指针，并使用该指针将其作为参数传递给函数。

```cpp
class StringReader : public System::IO::TextReader
```

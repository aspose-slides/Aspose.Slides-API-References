## StringCollection::cbegin() const 方法

返回一个迭代器，指向容器中第一个 const 限定的元素。如果容器为空，返回的迭代器将等于 [cend()](../cend/)。

```cpp
const_iterator System::Collections::Specialized::StringCollection::cbegin() const noexcept
```

### 返回值

一个指向容器中第一个 const 限定的元素的迭代器。

## StringCollection::crbegin() const 方法

返回一个反向迭代器，指向已翻转容器的第一个元素。它对应于未翻转容器的最后一个元素。如果容器为空，返回的迭代器等于 [crend()](../crend/)。

```cpp
const_reverse_iterator System::Collections::Specialized::StringCollection::crbegin() const noexcept
```

### 返回值

一个指向容器中最后的 const 限定元素的迭代器。

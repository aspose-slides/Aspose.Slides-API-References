## StringCollection::cend() const 方法

返回一个迭代器，指向容器中最后一个元素之后的元素。该元素充当占位符；尝试访问它会导致未定义行为。

```cpp
const_iterator System::Collections::Specialized::StringCollection::cend() const noexcept
```

### 返回值

一个迭代器，指向放置在容器最后一个元素之后的理论上 const 限定的元素。

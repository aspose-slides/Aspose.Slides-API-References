## StringCollection::crend() const 方法

返回一个反向迭代器，指向倒置容器中最后一个元素之后的元素。它对应于未倒置容器中第一个元素之前的元素。该元素充当占位符，尝试访问它会导致未定义行为。

```cpp
const_reverse_iterator System::Collections::Specialized::StringCollection::crend() const noexcept
```

### 返回值

一个迭代器，指向容器中第一个元素之前的理论 const 限定元素。

## StringCollection::rbegin() 方法


返回一个指向反转容器中第一个元素的逆向迭代器。它对应于未反转容器的最后一个元素。如果容器为空，返回的迭代器等于 [rend()](../rend/)。

```cpp
reverse_iterator System::Collections::Specialized::StringCollection::rbegin() noexcept
```


### 返回值

指向容器最后一个元素的迭代器。

## StringCollection::rbegin() const 方法


返回一个指向反转容器中第一个元素的逆向迭代器。它对应于未反转容器的最后一个元素。如果容器为空，返回的迭代器等于 [rend()](../rend/)。

```cpp
const_reverse_iterator System::Collections::Specialized::StringCollection::rbegin() const noexcept
```


### 返回值

指向 const 限定容器的最后一个元素的迭代器。

## IChartSeries.HasSeriesLines property

Определяет наличие строк серии для этой серии и родственных серий. Это свойство не только этой серии, но и всех серий родительской серии группа - это проекция соответствующего группового свойства. Таким образом, это свойство доступно только для чтения. Использовать свойство ParentSeriesGroup для доступа к родительской группе серий. Использовать свойство ParentSeriesGroup.HasSeriesLines для чтения/записи для значения изменения. Использовать свойство ParentSeriesGroup.SeriesLinesFormat для форматирования строк серий. Только для чтенияBoolean .

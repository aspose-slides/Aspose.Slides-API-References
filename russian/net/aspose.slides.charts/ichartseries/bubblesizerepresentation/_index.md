## IChartSeries.BubbleSizeRepresentation property

Указывает, как значения размера пузырьков представляются на пузырьковой диаграмме. Это свойство не только этой серии, но и всех серий родительской серии группа - это проекция соответствующего группового свойства. Таким образом, это свойство доступно только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство ParentSeriesGroup.BubbleSizeRepresentation для чтения/записи для значения изменения.

public BubbleSizeRepresentationType BubbleSizeRepresentation { get; }

## IChartSeries.IsColorVaried property

Указывает, что каждый маркер данных в серии имеет свой цвет. Это свойство не только этой серии, но и всех серий родительской серии группа - это проекция соответствующего группового свойства. Таким образом, это свойство доступно только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство ParentSeriesGroup.IsColorVaried для чтения/записи для значения изменения. Только для чтенияBoolean .

```csharp
public bool IsColorVaried { get; }
```

## IChartSeries.SecondPieSize property

Задает размер второй круговой диаграммы или столбца круговой диаграммы или столбчатой диаграммы в виде процент от размера первого круга (может быть от 5 до 200 процентов). Это свойство не только этой серии, но и всех серий родительской серии группа - это проекция соответствующего группового свойства. Итак, это свойство доступно только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство чтения/записи ParentSeriesGroup.SecondPieSize для значения изменения. Только для чтенияUInt16.

```csharp
public ushort SecondPieSize { get; }
```

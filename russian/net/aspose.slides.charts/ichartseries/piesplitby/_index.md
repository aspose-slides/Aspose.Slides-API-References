## IChartSeries.PieSplitBy property

Указывает, как определить, какие точки данных находятся во второй круговой диаграмме или столбце на круговой или столбчатой диаграмме . Это свойство не только этой серии, но и всех серий родительской серии группа - это проекция соответствующего группового свойства. Итак, это свойство доступно только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство ParentSeriesGroup.PieSplitBy для чтения/записи для значения изменения. Только для чтения[`PieSplitType`](../../piesplittype).

```csharp
public PieSplitType PieSplitBy { get; }
```

### Примечания

1) Это проекция свойства ParentSeriesGroup.PieSplitBy. 2) Если значением свойства является PieSplitType.Custom, вы можете определить пользовательское разделение информации с помощью свойства ParentSeriesGroup.PieSplitCustomPoints.

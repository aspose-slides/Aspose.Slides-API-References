## IChartSeries.Overlap property

Указывает, сколько баров и столбцов должно перекрываться на двумерных графиках (от -100 до 100). Это свойство не только этой серии, но и всей серии родительской серии группа - это проекция соответствующего группового свойства. Таким образом, это свойство доступно только для чтения. Использовать свойство ParentSeriesGroup для доступа к родительской группе серий. Использовать свойство ParentSeriesGroup.Overlap для чтения/записи для значения изменения. Только для чтенияSByte .

```csharp
public sbyte Overlap { get; }
```

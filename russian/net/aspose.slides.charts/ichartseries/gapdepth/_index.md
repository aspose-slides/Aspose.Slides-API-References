## IChartSeries.GapDepth property

Возвращает или задает расстояние в процентах от ширины маркера между рядами данных на трехмерной диаграмме. Это свойство не только этого ряда, но и всех рядов родительского ряда группа - это проекция соответствующей группы имущество. Таким образом, это свойство доступно только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство ParentSeriesGroup.GapDepth для чтения/записи для значения изменения. Только для чтенияInt32 .

public int GapDepth { get; }

## ChartSeries.GapWidth property

Определяет расстояние между кластерами столбцов или столбцов в процентах от ширины столбца или столбца. Это свойство не только этой серии, но и всех серий родительской серии. группа - это проекция соответствующего группового свойства. Таким образом, это свойство доступно только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство ParentSeriesGroup.GapWidth для чтения/записи для значения изменения. Только для чтенияInt32 .

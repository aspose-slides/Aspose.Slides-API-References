## ChartSeries.HasSeriesLines property

Определяет, есть ли строки серии для этой серии и родственных серий. Это свойство не только этой серии, но и всех серий родительской серии группа - это проекция соответствующего группового свойства. Итак, это свойство доступно только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство чтения/записи ParentSeriesGroup.HasSeriesLines для значения изменения. Используйте свойство ParentSeriesGroup.SeriesLinesFormat для форматирования строк серии. Только чтениеBoolean.

```csharp
public bool HasSeriesLines { get; }
```

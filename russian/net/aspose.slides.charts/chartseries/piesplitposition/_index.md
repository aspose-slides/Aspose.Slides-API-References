## ChartSeries.PieSplitPosition property

Определяет значение, которое должно использоваться для определения того, какие точки данных находятся на второй круговой диаграмме или столбце на круговой или столбчатой диаграмме. Используется вместе со свойством PieSplitBy. Это свойство не только этой серии, но и всей серии родительской серии группа - это проекция соответствующего группового свойства. Таким образом, это свойство доступно только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство ParentSeriesGroup.PieSplitPosition для чтения/записи для значения изменения. Только для чтенияDouble .

public double PieSplitPosition { get; }

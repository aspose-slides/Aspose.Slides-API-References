## ChartSeries.BubbleSizeScale property

Указывает масштабный коэффициент для пузырьковой диаграммы (может быть между 0 и 300 процентами от размера по умолчанию). Это свойство не только этой серии, но и всех серий родительской серии группа - это проекция соответствующей группы имущество. Таким образом, это свойство доступно только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство ParentSeriesGroup.BubbleSizeScale для чтения/записи для значения изменения.

```csharp
public int BubbleSizeScale { get; }
```

## ChartSeries.BubbleSizeRepresentation property

Указывает, как значения размера пузырьков представлены на пузырьковой диаграмме. Это свойство не только этой серии, но и всех серий родительской серии группа - это проекция соответствующего группового свойства. Итак, это свойство доступно только для чтения. Используйте свойство ParentSeriesGroup для доступа к родительской группе серий. Используйте свойство чтения/записи ParentSeriesGroup.BubbleSizeRepresentation для значения изменения.

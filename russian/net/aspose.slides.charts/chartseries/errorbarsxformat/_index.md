## ChartSeries.ErrorBarsXFormat property

Представляет ErrorBars серии с направлением X.  ErrorBars с направлением X доступны для серий типа area, bar, Scatter и Bubble. Для любых других типов диаграмм это свойство возвращает значение null (включая трехмерные диаграммы). В случае пользовательских значений используйте коллекцию DataPoints для указания значения (с[`ErrorBarsCustomValues`](../../ichartdatapoint/errorbarscustomvalues) свойство).

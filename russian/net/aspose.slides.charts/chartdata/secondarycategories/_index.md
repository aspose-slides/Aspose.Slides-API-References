## ChartData.SecondaryCategories property

Получает вторичные категории, если[`UseSecondaryCategories`](../usesecondarycategories) свойство имеет значение true. Только для чтения[`IChartCategoryCollection`](../../ichartcategorycollection) .

```csharp
public IChartCategoryCollection SecondaryCategories { get; }
```

### Примечания

Если[`UseSecondaryCategories`](../usesecondarycategories) свойство ложно, то это`SecondaryCategories` свойство возвращает null и данные в[`Categories`](../categories) используется как для первичной , так и для вторичной серии. Если[`UseSecondaryCategories`](../usesecondarycategories) свойство истинно, тогда данные в this`SecondaryCategories` свойство используется для вторичной серии и данных в[`Categories`](../categories) свойство используется для основной серии.

### Примеры

Пример. Какие категории относятся к рядам - ChartData.Categories или ChartData.SecondaryCategories?

```csharp
if (series.PlotOnSecondAxis && series.Chart.ChartData.UseSecondaryCategories)
{
    // связанные категории: series.Chart.ChartData.SecondaryCategories
}
else
{
    // связанные категории: series.Chart.ChartData.Categories
}
```

## IChartData.SetRange method

Установить диапазон данных диаграммы. Серии и категории будут обновляться на основе нового диапазона данных. Если количество рядов в диапазоне данных превышает количество рядов в данных диаграммы, то в список будут добавлены дополнительные ряды того же типа , что и последний ряд в текущей коллекции. конец коллекции.

```csharp
public void SetRange(string formula)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| formula | String | Формула диапазона данных ячеек. Например:"Лист1!$A$1:$C$4" , "SomeSheetName!A1:B100", "Лист1!$A$1:$B$5;Лист1!$D$1:$D$5". |

### Исключения

| исключение | условие |
| --- | --- |
| ArgumentNullException | Формула имеет значение null. |
| ArgumentException | Формула имеет неверный формат. |

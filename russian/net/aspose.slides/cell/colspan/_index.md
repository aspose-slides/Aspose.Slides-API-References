## Cell.ColSpan property

Возвращает количество столбцов сетки в таблице grid родительской таблицы, которые должны охватывать текущую ячейку. Это свойство позволяет ячейкам выглядеть объединенными, поскольку они охватывают вертикальные границы других ячеек в таблице. Только для чтенияInt32.

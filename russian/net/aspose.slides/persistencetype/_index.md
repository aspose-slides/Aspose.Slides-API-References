## PersistenceType enumeration

Указывает метод, используемый для хранения свойств элемента управления ActiveX.

```csharp
public enum PersistenceType
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| NotDefined | `-1` | Идентификатор сохраняемости не указан. |
| PersistPropertyBag | `0` | Указывает, что элемент управления ActiveX сохраняется с использованием сохраняемости на основе набора свойств. Постоянство на основе набора свойств хранит элемент управления ActiveX с помощью набора имен и пар значений, которые определяют данные, сохраняемые элементом управления ActiveX. |
| PersistStream | `1` | Указывает, что элемент управления ActiveX сохраняется с использованием сохраняемости на основе потока , которая не поддерживает инициализацию элемента управления ActiveX до состояния по умолчанию. |
| PersistStreamInit | `2` | Указывает, что элемент управления ActiveX сохраняется с использованием сохраняемости на основе потока , которая поддерживает инициализацию элемента управления ActiveX до состояния по умолчанию. |
| PersistStorage | `3` | Указывает, что элемент управления ActiveX сохраняется с использованием сохраняемости на основе хранилища. |

## IFontFallBackRule.Remove method

Удаляет первое вхождение определенного шрифта FallBack из списка.

```csharp
public void Remove(string fontName)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| fontName | String | Имя шрифта для удаления из списка. |

### Примеры

```csharp
[C#]
```

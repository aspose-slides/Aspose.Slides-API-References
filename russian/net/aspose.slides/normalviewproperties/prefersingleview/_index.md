## NormalViewProperties.PreferSingleView property

Указывает, предпочитает ли пользователь видеть полноэкранную область с одним содержимым по сравнению с стандартным обычным видом с тремя областями содержимого. Если включено, приложение может выбрать для отображения одной из областей содержимого во всем окне. Чтение/записьBoolean.

public bool PreferSingleView { get; set; }

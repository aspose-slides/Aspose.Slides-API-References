## IProtectionManager.CheckWriteProtection method

Определяет, защищена ли презентация паролем для изменения.

```csharp
public bool CheckWriteProtection(string password)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| password | String | Пароль для проверки. |

### Возвращаемое значение

Истинно, если пароль действителен; в противном случае ложно.

### Примечания

1. Вы должны проверить[`IsWriteProtected`](../iswriteprotected) перед вызовом этого метода. 2. Если пароль нулевой или пустой, этот метод возвращает false.

### Примеры

```csharp
[C#]
using (var presentation = new Presentation(presentationFilePath))
{
    var isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("my_password");
}
```

## ProtectionManager.IsOnlyDocumentPropertiesLoaded property

Это свойство имеет смысл, если файл презентации защищен паролем и документ свойства этого файла общедоступны. Значение true означает, что только свойства документа загружаются из зашифрованного файла презентации без использования пароля. Значение false означает, вся зашифрованная презентация загружается с использованием правильного пароля , загружаются не только свойства документа. Если презентация не зашифрована, значение свойства всегда равно false. Если свойства документа зашифрованного файла не являются общедоступными, значение свойства всегда false. Если Presentation.EncryptDocumentProperties имеет значение true, то значение свойства IsOnlyDocumentPropertiesLoaded всегда равно false. Только для чтенияBoolean .

```csharp
public bool IsOnlyDocumentPropertiesLoaded { get; }
```

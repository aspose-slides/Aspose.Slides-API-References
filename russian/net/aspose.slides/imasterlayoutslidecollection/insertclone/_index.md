```csharp
public ILayoutSlide InsertClone(int index, ILayoutSlide sourceLayout)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| index | Int32 | Указатель нового слайда. |
| sourceLayout | ILayoutSlide | Сдвиньте, чтобы клонировать. |

### Возвращаемое значение

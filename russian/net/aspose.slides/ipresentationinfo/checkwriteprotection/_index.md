## IPresentationInfo.CheckWriteProtection method

Проверяет правильность изменяемого пароля для защищенной от записи презентации.

```csharp
public bool CheckWriteProtection(string password)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| password | String | Пароль для проверки. |

### Возвращаемое значение

Истинно, если презентация защищена от записи и пароль правильный. False в противном случае.

### Исключения

| исключение | условие |
| --- | --- |
| InvalidOperationException |  |

### Примечания

1. Вы должны проверить[`IsWriteProtected`](../iswriteprotected) перед вызовом этого метода. 2. Если пароль нулевой или пустой, этот метод возвращает false.

### Примеры

```csharp
[C#]
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationFilePath);
if (info.IsWriteProtected == NullableBool.True)
{
    bool isWriteProtectedByPassword = info.CheckWriteProtection("my_password");
}
```

## PdfOptions.BestImagesCompressionRatio property

Указывает, должно ли автоматически выбираться наиболее эффективное сжатие (вместо стандартного) для каждого изображения . Если установлено значениеBoolean.true, для каждого изображения в презентации будет выбран наиболее подходящий алгоритм сжатия , что приведет к меньшему размеру результирующего PDF-документ.  Выбор наилучшего коэффициента сжатия изображения требует больших вычислительных ресурсов и требует дополнительного объема ОЗУ, и этот параметр равенBoolean.false по умолчанию.

```csharp
public bool BestImagesCompressionRatio { get; set; }
```

### Примечания

По умолчанию:Boolean.false.

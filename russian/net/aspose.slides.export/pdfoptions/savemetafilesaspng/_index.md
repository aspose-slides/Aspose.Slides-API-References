## PdfOptions.SaveMetafilesAsPng property

Значение true, чтобы преобразовать все метафайлы, используемые в презентации, в изображения PNG. Чтение/записьBoolean.

```csharp
public bool SaveMetafilesAsPng { get; set; }
```

### Примечания

По умолчанию: **true** . Документ Pdf может содержать векторную графику и растровые изображения. Если для параметра SaveMetafilesAsPng установлено значение true, то исходный метафайл изображение преобразуется в формат Png и сохраняется в формате Pdf как растр изображение. Если для параметра SaveMetafilesAsPng установлено значение false, исходный файл Metafile преобразуется в векторную графику в формате Pdf. У каждого подхода есть преимущества и недостатки. Например, если Metafile конвертируется в PNG, , то возможна некоторая потеря качества при результирующем масштабировании документа. Если Metafile конвертируется в векторную графику Pdf, то возможны проблемы с производительностью в инструменте просмотра Pdf.

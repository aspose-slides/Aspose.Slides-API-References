## PdfOptions.SaveMetafilesAsPng property

Значение true для преобразования всех метафайлов, используемых в презентации, в изображения PNG. Чтение/записьBoolean .

```csharp
public bool SaveMetafilesAsPng { get; set; }
```

### Примечания

По умолчанию **истинный**. Документ Pdf может содержать векторную графику и растровые изображения. Если для параметра SaveMetafilesAsPng установлено значение true, исходное изображение метафайла преобразуется в формат Png и сохраняется в Pdf как растровое изображение . Если для параметра SaveMetafilesAsPng установлено значение false, исходный метафайл преобразуется в векторную графику в формате Pdf. Каждый подход имеет свои преимущества и недостатки. Например, если метафайл конвертируется в PNG, , то возможна некоторая потеря качества при масштабировании результирующего документа . Если метафайл конвертируется в векторную графику Pdf, , то возможны проблемы с производительностью в инструменте просмотра Pdf.

## SVGOptions.DisableLineEndCropping property

В SVG 1.1 отсутствует возможность определять вставки для маркеров. Механизм записи Aspose.Slides SVG имеет обходной путь для этой проблемы: он обрезает конец строки стрелкой, поэтому линия не перекрывает маркеры. Этот параметр отключает такое поведение. Чтение/записьBoolean .

```csharp
public bool DisableLineEndCropping { get; set; }
```

## MathBox.NoBreak property

Нет break Это свойство определяет свойство «нерушимости» в поле объекта. При значении true внутри поля не может быть разрывов строк. Это может быть важно для эмуляторов операторов, которые состоят из более чем одного бинарного оператора. Если этот элемент не указан, внутри блока могут возникать разрывы. По умолчанию: true

```csharp
public bool NoBreak { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("*****"));
box.NoBreak = false;
```

## MathBox.NoBreak property

Без разрыва Это свойство задает свойство "нерушимости" в поле объекта. Если задано значение true, внутри блока не может быть разрывов строк. Это может быть важно для эмуляторов операторов, которые состоят из более чем одного бинарного оператора. Когда этот элемент не указан, внутри блока могут возникать разрывы. По умолчанию:true

```csharp
public bool NoBreak { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
MathBox box = new MathBox(new MathematicalText("*****"));
box.NoBreak = false;
```

## MathMatrix.MinColumnWidth property

Минимальная ширина столбца в твипах (1/20 пункта) Интервал между столбцами (также называемый «Пробел между столбцами» или «Ширина зазора») добавляется к MinColumnWidth, чтобы определить общий интервал между столбцами матрицы (расстояние между одинаковые ребра разных столбцов). По умолчанию: 0.

```csharp
public uint MinColumnWidth { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.MinColumnWidth = 20;
```

## MathMatrix.RowGap property

Значение интервала по вертикали между строками матрицы; Если для RowGapRule установлено значение 3 ("Точно"), то единица интерпретируется как твипы (1/20 точки) Если для RowGapRule установлено значение 4 ( "Несколько"), то единица измерения интерпретируется как половина строки. По умолчанию: 0

```csharp
public uint RowGap { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.RowGapRule = MathSpacingRules.Exactly;
matrix.RowGap = 20;
```

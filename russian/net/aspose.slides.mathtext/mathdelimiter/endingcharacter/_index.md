## MathDelimiter.EndingCharacter property

Символ окончания разделителя определяет конечный или закрывающий символ разделителя. Математические разделители заключают в себя символы, такие как скобки, квадратные и фигурные скобки. Значение по умолчанию: ')'.

```csharp
public char EndingCharacter { get; set; }
```

Пример:

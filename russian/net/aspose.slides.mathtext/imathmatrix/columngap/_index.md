## IMathMatrix.ColumnGap property

Значение горизонтального интервала между столбцами матрицы; Если для ColumnGapRule установлено значение 3 ("Точно"), то единица интерпретируется как твипы (1/20 точки) Если для ColumnGapRule установлено значение 4 ( "Multiple"), то единица интерпретируется как число с шагом 0,5 em. В других случаях игнорируется. По умолчанию: 0

```csharp
public uint ColumnGap { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
IMathMatrix matrix = new MathMatrix(2, 3);
matrix.ColumnGapRule = MathSpacingRules.Exactly;
matrix.ColumnGap = 20;
```

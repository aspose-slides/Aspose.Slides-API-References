## IMathFraction interface

Определяет объект дроби, состоящий из числителя и знаменателя, разделенных чертой дроби. Полоса дроби может быть горизонтальной или диагональной, в зависимости от свойств дроби. Объект дроби также используется для представления функции стека, которая помещает один элемент над другим, без дробной черты.

public interface IMathFraction : IMathElement

## MathGroupingCharacter.VerticalJustification property

Вертикальное выравнивание группового символа. Задает выравнивание объекта относительно базовой линии. Например, когда групповой символ находится над объектом, Вертикальное выравнивание по верху означает, что верхняя часть объекта падает на базовую линию; когда для параметра VerticalJustification установлено значение Bottom, нижняя часть объекта находится на baseline По умолчанию: Bottom для Position=Top и Top для Position=Bottom

```csharp
public MathTopBotPositions VerticalJustification { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
MathGroupingCharacter groupingCharacter = new MathGroupingCharacter(new MathematicalText("abc"));
groupingCharacter.VerticalJustification = MathTopBotPositions.Top;
```

## MathArray.RowSpacing property

Интервал между строками массива Используется только тогда, когда RowSpacingRule имеет значение 3. Именно в этом случае единицей измерения являются точки или Несколько, и в этом случае единицей измерения являются полулинии. По умолчанию: 0

```csharp
public uint RowSpacing { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
IMathArray mathArray = new MathArray(new MathematicalText("item1"));
mathArray.RowSpacingRule = MathRowSpacingRule.Exactly;
mathArray.RowSpacing = 10;
```

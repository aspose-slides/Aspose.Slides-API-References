## IMathBox.ExplicitBreak property

Явный разрыв указывает, есть ли разрыв строки в начале объекта Box, такой, чтобы строка переносилась в начале блока объект. Определяет номер оператора в предыдущей строке математического текста, который должен использоваться в качестве точки выравнивания для текущей строки математического текста возможно значения:1..255 По умолчанию:0 (без явного разрыва)

```csharp
public byte ExplicitBreak { get; set; }
```

### Примеры

Пример:

```csharp
[C#]
IMathBox box = new MathematicalText("==").ToBox();
box.ExplicitBreak = 1;
```

## PdfAccessPermissions enumeration

Contient un ensemble d'indicateurs spécifiant les autorisations d'accès à accorder lorsque le document est ouvert avec un accès utilisateur .

```csharp
[Flags]
public enum PdfAccessPermissions
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| None | `0` | Spécifie qu'un utilisateur n'a pas d'autorisations d'accès. |
| PrintDocument | `4` | Spécifie si un utilisateur peut imprimer le document (éventuellement pas au niveau de qualité le plus élevé, selon si bitHighQualityPrint est également défini). |
| ModifyContent | `8` | Spécifie si un utilisateur peut modifier le contenu du document par des opérations autres que celles contrôlées par des bitsAddOrModifyFields ,FillExistingFields ,AssembleDocument . |
| CopyTextAndGraphics | `10` | Spécifie si un utilisateur peut copier ou extraire du texte et des graphiques du document par des opérations autres que celles contrôlées par bitExtractTextAndGraphics . |
| AddOrModifyFields | `20` | Spécifie si un utilisateur peut ajouter ou modifier des annotations de texte, remplir des champs de formulaire interactifs et, si bit ModifyContentest également défini, créé ou modifié des champs de formulaire interactifs (y compris les champs de signature ). |
| FillExistingFields | `100` | Spécifie si un utilisateur peut remplir les champs de formulaires interactifs existants (y compris les champs de signature), même if bitAddOrModifyFields est clair. |
| ExtractTextAndGraphics | `200` | Spécifie si un utilisateur peut extraire du texte et des graphiques pour faciliter l'accessibilité aux utilisateurs handicapés ou à d'autres fins. |

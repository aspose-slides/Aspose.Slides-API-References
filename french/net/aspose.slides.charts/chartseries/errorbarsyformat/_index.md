## ChartSeries.ErrorBarsYFormat property

Représente les ErrorBars des séries avec déréction Y.  Les barres d'erreur avec la direction Y sont disponibles pour les séries de type zone, barre, ligne, dispersion et bulle. Pour tous les autres types de graphique, cette propriété renvoie null (y compris les graphiques 3D). En cas de valeurs personnalisées, utilisez la collection DataPoints pour spécifier value (avec[`ErrorBarsCustomValues`](../../ichartdatapoint/errorbarscustomvalues) propriété).

## IDataLabelCollection.Hide method

Faites en sorte que l'étiquette de données soit masquée par défaut en définissant tous les indicateurs Show* (ShowValue, ...) de la propriété DefaultDataLabelFormat sur l'état faux. IsVisible sera faux après cela.

```csharp
public void Hide()
```

### Remarques

Si l'étiquette de données n'est pas visible par défaut (IsVisible est faux), vous pouvez rendre l'étiquette de données "visible par défaut" en définissant les drapeaux Show* (ShowValue, ...) de la propriété DefaultDataLabelFormat sur l'état vrai.

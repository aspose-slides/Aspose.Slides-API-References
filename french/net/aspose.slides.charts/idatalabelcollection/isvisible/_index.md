## IDataLabelCollection.IsVisible property

False signifie que l'étiquette de données n'est pas visible par défaut (et donc tous les indicateurs Show* (ShowValue, ...) de la propriété DefaultDataLabelFormat sont faux). Lecture seuleBoolean .

```csharp
public bool IsVisible { get; }
```

### Remarques

Si l'étiquette de données est visible par défaut, vous pouvez la masquer par défaut avec la méthode Hide (). Mais si l'étiquette de données n'est pas visible par défaut (IsVisible est faux), vous pouvez rendre l'étiquette de données "visible par défaut" avec le paramètre Afficher *-flags (ShowValue, ...) de la propriété DefaultDataLabelFormat à l'état vrai.

## ShapeCollection.InsertSummaryZoomFrame method

Crée un nouvel objet Summary Zoom et l'insère dans une collection à l'index spécifié.

```csharp
public ISummaryZoomFrame InsertSummaryZoomFrame(int index, float x, float y, float width, 
    float height)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| index | Int32 | Index de base zéro auquel le cadre de zoom de section doit être inséré. |
| x | Single | Coordonnée X d'un nouveau cadre de zoom de sectionSingle. |
| y | Single | Coordonnée Y d'un nouveau cadre de zoom de sectionSingle. |
| width | Single | Largeur d'un nouveau cadre de zoom de sectionSingle. |
| height | Single | Hauteur d'un nouveau cadre de zoom de sectionSingle. |

### Return_Value

Objet zoom récapitulatif créé[`ISummaryZoomFrame`](../../isummaryzoomframe).

### Exceptions

| exception | condition |
| --- | --- |
| [PptxEditException](../../pptxeditexception) | Il n'y a pas de section dans la présentation ou la diapositive cible n'appartient à aucune section. |

### Remarques

Cette méthode crée un nouveau zoom récapitulatif et y place une collection d'objets pour toutes les sections de cette présentation.

### Exemples

Cet exemple illustre la création et l'insertion d'un objet Summary Zoom à l'index spécifié d'une collection (en supposant qu'il existe au moins deux sections dans la présentation "Presentation.pptx") :

```csharp
[C#]
using (Presentation pres = new Presentation("Presentation.pptx"))
{
    ISummaryZoomFrame zoomFrame = pres.Slides[0].Shapes.InsertSummaryZoomFrame(2, 150, 20, 50, 50);
}
```

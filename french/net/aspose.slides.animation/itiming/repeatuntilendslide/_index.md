## ITiming.RepeatUntilEndSlide property

Cet attribut spécifie si l'effet se répétera jusqu'à la fin de la diapositive. Lecture/écritureBoolean .

```csharp
public bool RepeatUntilEndSlide { get; set; }
```

### Exemples

```csharp
[C#]
using (Presentation presentation = new Presentation("demo.pptx"))
{
    // Récupère la séquence d'effets pour la première diapositive
    ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;

    // Récupère le premier effet de la séquence principale.
    IEffect effect = effectsSequence[0];

    // Changer l'effet Timing/Repeat en "Jusqu'à la fin de la diapositive"
    effect.Timing.RepeatUntilEndSlide = true;
}
```
